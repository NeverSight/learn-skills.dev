문제 해결, 의사 결정, 아키텍처 및 디자인 패턴, 제한 및 할당량, 보안, 구성, 통합 및 코딩 패턴을 포함한 Azure DNS 개발에 대한 전문 지식입니다. Azure DNS 애플리케이션을 빌드, 디버깅 또는 최적화할 때 사용합니다. Azure Traffic Manager(azure-traffic-manager 사용), Azure Front Door(azure-front-door 사용), Azure Virtual Network(azure-virtual-network 사용), Azure Private Link(azure-private-link 사용)에는 해당되지 않습니다.
