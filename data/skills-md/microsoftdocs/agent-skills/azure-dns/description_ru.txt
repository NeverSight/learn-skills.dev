Экспертные знания для разработки Azure DNS, включая устранение неполадок, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, а также интеграцию и шаблоны кодирования. Используйте при создании, отладке или оптимизации приложений Azure DNS. Не для диспетчера трафика Azure (используйте azure-traffic-manager), Azure Front Door (используйте azure-front-door), виртуальной сети Azure (используйте azure-virtual-network), Azure Private Link (используйте azure-private-link).
