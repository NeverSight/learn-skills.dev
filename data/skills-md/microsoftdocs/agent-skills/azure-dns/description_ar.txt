معرفة الخبراء بتطوير Azure DNS بما في ذلك استكشاف الأخطاء وإصلاحها واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز. يُستخدم عند إنشاء تطبيقات Azure DNS أو تصحيح أخطائها أو تحسينها. ليس لـ Azure Traffic Manager (استخدم azure-traffic-manager)، وAzure Front Door (استخدم azure-front-door)، وAzure Virtual Network (استخدم azure-virtual-network)، وAzure Private Link (استخدم رابط azure-private).
