Conoscenza approfondita dello sviluppo DNS di Azure, tra cui risoluzione dei problemi, processo decisionale, modelli di architettura e progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni DNS di Azure. Non per Gestione traffico di Azure (usare azure-traffic-manager), Porta anteriore di Azure (usare azure-front-door), Rete virtuale di Azure (usare azure-virtual-network), Collegamento privato di Azure (usare azure-private-link).
