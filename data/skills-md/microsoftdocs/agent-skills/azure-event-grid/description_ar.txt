معرفة الخبراء بتطوير Azure Event Grid بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Event Grid أو تصحيح أخطائها أو تحسينها. ليس لـ Azure Event Hubs (استخدم azure-event-hubs)، وAzure Service Bus (استخدم azure-service-bus)، وAzure Notification Hubs (استخدم azure-notification-hubs)، وAzure Logic Apps (استخدم azure-logic-apps).
