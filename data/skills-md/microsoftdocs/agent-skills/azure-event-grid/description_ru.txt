Экспертные знания для разработки Azure Event Grid, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure Event Grid. Не для Центров событий Azure (используйте azure-event-hubs), Azure Service Bus (используйте azure-service-bus), Центров уведомлений Azure (используйте azure-notification-hubs), Azure Logic Apps (используйте azure-logic-apps).
