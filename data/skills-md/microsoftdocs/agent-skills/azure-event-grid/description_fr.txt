Connaissance approfondie du développement d'Azure Event Grid, y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation d’applications Azure Event Grid. Pas pour Azure Event Hubs (utilisez azure-event-hubs), Azure Service Bus (utilisez azure-service-bus), Azure Notification Hubs (utilisez azure-notification-hubs), Azure Logic Apps (utilisez azure-logic-apps).
