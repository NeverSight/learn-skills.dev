문제 해결, 모범 사례, 의사 결정, 아키텍처 및 디자인 패턴, 제한 및 할당량, 보안, 구성, 통합 및 코딩 패턴, 배포를 포함한 Azure Event Grid 개발에 대한 전문 지식입니다. Azure Event Grid 애플리케이션을 빌드, 디버깅 또는 최적화할 때 사용합니다. Azure Event Hubs(azure-event-hubs 사용), Azure Service Bus(azure-service-bus 사용), Azure 알림 허브(azure-notification-hubs 사용), Azure Logic Apps(azure-logic-apps 사용)에는 해당되지 않습니다.
