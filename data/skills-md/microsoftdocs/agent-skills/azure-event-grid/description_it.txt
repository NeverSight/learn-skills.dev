Conoscenza approfondita dello sviluppo di Griglia di eventi di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, architettura e modelli di progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni di Griglia di eventi di Azure. Non per Hub eventi di Azure (usare azure-event-hubs), Bus di servizio di Azure (usare azure-service-bus), Hub di notifica di Azure (usare azure-notification-hubs), App per la logica di Azure (usare azure-logic-apps).
