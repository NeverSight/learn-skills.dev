Azure Database for PostgreSQL 開発に関する専門知識 (トラブルシューティング、ベスト プラクティス、意思決定、アーキテクチャと設計パターン、制限とクォータ、セキュリティ、構成、統合とコーディング パターン、デプロイなど)。 Azure Database for PostgreSQL アプリケーションを構築、デバッグ、最適化するときに使用します。 Azure SQL Database (azure-sql-database を使用)、Azure SQL Managed Instance (azure-sql-managed-instance を使用)、Azure Virtual Machines 上の SQL Server (azure-sql-virtual-machines を使用)、Azure Cosmos DB (azure-cosmos-db を使用) には適用されません。
