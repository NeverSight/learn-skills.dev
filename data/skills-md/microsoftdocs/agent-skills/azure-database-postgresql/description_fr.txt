Connaissance approfondie du développement d'Azure Database pour PostgreSQL, y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation des applications Azure Database pour PostgreSQL. Pas pour Azure SQL Database (utilisez azure-sql-database), Azure SQL Managed Instance (utilisez azure-sql-managed-instance), SQL Server sur les machines virtuelles Azure (utilisez azure-sql-virtual-machines), Azure Cosmos DB (utilisez azure-cosmos-db).
