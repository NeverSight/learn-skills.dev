Expertenwissen für die Entwicklung von Azure Database for PostgreSQL, einschließlich Fehlerbehebung, Best Practices, Entscheidungsfindung, Architektur- und Entwurfsmuster, Grenzwerte und Kontingente, Sicherheit, Konfiguration, Integrationen und Codierungsmuster sowie Bereitstellung. Wird zum Erstellen, Debuggen oder Optimieren von Azure Database for PostgreSQL-Anwendungen verwendet. Nicht für Azure SQL-Datenbank (verwenden Sie azure-sql-database), Azure SQL Managed Instance (verwenden Sie azure-sql-managed-instance), SQL Server auf Azure Virtual Machines (verwenden Sie azure-sql-virtual-machines), Azure Cosmos DB (verwenden Sie azure-cosmos-db).
