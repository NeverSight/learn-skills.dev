Экспертные знания по разработке базы данных Azure для PostgreSQL, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, настройку, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации базы данных Azure для приложений PostgreSQL. Не для базы данных SQL Azure (используйте azure-sql-database), Управляемого экземпляра Azure SQL (используйте azure-sql-managed-instance), SQL Server на виртуальных машинах Azure (используйте azure-sql-virtual-machines), Azure Cosmos DB (используйте azure-cosmos-db).
