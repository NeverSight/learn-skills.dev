Conoscenza approfondita dello sviluppo del database di Azure per PostgreSQL, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, architettura e modelli di progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da usare durante la creazione, il debug o l'ottimizzazione del database di Azure per applicazioni PostgreSQL. Non per il database SQL di Azure (usare azure-sql-database), Istanza gestita SQL di Azure (usare azure-sql-built-instance), SQL Server in macchine virtuali di Azure (usare azure-sql-virtual-machines), Azure Cosmos DB (usare azure-cosmos-db).
