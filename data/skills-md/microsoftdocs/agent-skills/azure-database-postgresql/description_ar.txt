معرفة الخبراء بقاعدة بيانات Azure لتطوير PostgreSQL بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء قاعدة بيانات Azure أو تصحيح الأخطاء أو تحسينها لتطبيقات PostgreSQL. ليس لقاعدة بيانات Azure SQL (استخدم قاعدة بيانات azure-sql)، ومثيل Azure SQL المُدار (استخدم مثيل azure-sql-managed)، وSQL Server على أجهزة Azure الظاهرية (استخدم azure-sql-virtual-machines)، وAzure Cosmos DB (استخدم azure-cosmos-db).
