Azure Database for PostgreSQL 開發的專業知識，包括故障排除、最佳實踐、決策、體系結構和設計模式、限制和配額、安全性、配置、整合和編碼模式以及部署。在建置、偵錯或最佳化 Azure Database for PostgreSQL 應用程式時使用。不適用於 Azure SQL 資料庫（使用 azure-sql-database）、Azure SQL 託管執行個體（使用 azure-sql-託管執行個體）、Azure 虛擬機器上的 SQL Server（使用 azure-sql-virtual-machines）、Azure Cosmos DB（使用 azure-cosmos-db）。
