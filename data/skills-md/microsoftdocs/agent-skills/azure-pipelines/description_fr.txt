Connaissance approfondie du développement d'Azure Pipelines, y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. Pas pour Azure DevOps (utilisez azure-devops), Azure Boards (utilisez azure-boards), Azure Repos (utilisez azure-repos), Azure Test Plans (utilisez azure-test-plans).
