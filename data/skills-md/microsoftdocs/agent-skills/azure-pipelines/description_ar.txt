معرفة الخبراء بتطوير Azure Pipelines بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Pipelines أو تصحيح أخطائها أو تحسينها. ليس لـ Azure DevOps (استخدم azure-devops)، ولوحات Azure (استخدم لوحات azure)، وAzure Repos (استخدم azure-repos)، وخطط اختبار Azure (استخدم خطط اختبار azure).
