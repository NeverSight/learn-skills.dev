Экспертные знания для разработки Azure Pipelines, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure Pipelines. Не для Azure DevOps (используйте azure-devops), Azure Boards (используйте azure-boards), Azure Repos (используйте azure-repos), планов тестирования Azure (используйте azure-test-plans).
