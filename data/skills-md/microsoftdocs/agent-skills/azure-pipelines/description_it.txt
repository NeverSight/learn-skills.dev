Conoscenza approfondita dello sviluppo di Azure Pipelines, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, architettura e modelli di progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni Azure Pipelines. Non per Azure DevOps (usare azure-devops), Azure Boards (usare azure-boards), Azure Repos (usare azure-repos), Azure Test Plans (usare azure-test-plans).
