معرفة الخبراء بالتطوير المحلي لـ Azure بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure المحلية أو تصحيح أخطائها أو تحسينها. ليس لـ Azure AI Foundry Local (استخدم azure-ai-foundry-local)، وAzure Stack Edge (استخدم azure-stack-edge)، وأجهزة Azure الافتراضية (استخدم الأجهزة الافتراضية azure)، وخدمة Azure Kubernetes (AKS) (استخدم خدمة azure-kubernetes).
