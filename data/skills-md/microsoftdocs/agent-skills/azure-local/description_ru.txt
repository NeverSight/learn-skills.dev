Экспертные знания для локальной разработки Azure, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации локальных приложений Azure. Не для Azure AI Foundry Local (используйте azure-ai-foundry-local), Azure Stack Edge (используйте azure-stack-edge), виртуальных машин Azure (используйте azure-virtual-machines), Azure Kubernetes Service (AKS) (используйте azure-kubernetes-service).
