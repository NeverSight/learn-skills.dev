معرفة الخبراء لتطوير سياسة Azure بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Policy أو تصحيح أخطائها أو تحسينها. ليس لمخططات Azure (استخدم مخططات azure)، والتحكم في الوصول المستند إلى دور Azure (استخدم azure-rbac)، وAzure Resource Manager (استخدم azure-resource-manager)، وAzure Security (استخدم azure-security).
