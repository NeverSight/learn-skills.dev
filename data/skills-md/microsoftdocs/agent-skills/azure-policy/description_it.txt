Conoscenza approfondita dello sviluppo di criteri di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da usare durante la creazione, il debug o l'ottimizzazione delle applicazioni di criteri di Azure. Non per Azure Blueprints (usare azure-blueprints), controllo degli accessi in base al ruolo di Azure (usare azure-rbac), Azure Resource Manager (usare azure-resource-manager), Sicurezza di Azure (usare azure-security).
