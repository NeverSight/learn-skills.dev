Экспертные знания для разработки политик Azure, включая устранение неполадок, лучшие практики, принятие решений, безопасность, настройку, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure Policy. Не для Azure Blueprints (используйте Azure-Blueprints), управления доступом на основе ролей Azure (используйте azure-rbac), Azure Resource Manager (используйте azure-resource-manager), Azure Security (используйте azure-security).
