Connaissance approfondie du développement d'Azure Policy, y compris le dépannage, les meilleures pratiques, la prise de décision, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation des applications Azure Policy. Pas pour Azure Blueprints (utilisez azure-blueprints), le contrôle d'accès basé sur le rôle Azure (utilisez azure-rbac), Azure Resource Manager (utilisez azure-resource-manager), Azure Security (utilisez azure-security).
