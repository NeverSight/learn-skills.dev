إرشادات الخبراء لتصميم حلول Azure باستخدام Azure Architecture. يغطي البنى المرجعية، وأفكار الحلول، وأنماط التصميم، واختيارات التكنولوجيا، وأنماط الهندسة المعمارية، وأفضل الممارسات، والأنماط المضادة، وأمثلة لأعباء العمل، وأدلة الترحيل. يُستخدم عند تحديد أنماط البنية أو اختيار خدمات Azure أو تنفيذ حلول جاهزة للإنتاج.
