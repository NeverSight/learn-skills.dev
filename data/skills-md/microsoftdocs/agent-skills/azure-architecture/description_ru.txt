Экспертное руководство по проектированию решений Azure с использованием архитектуры Azure. Охватывает эталонные архитектуры, идеи решений, шаблоны проектирования, выбор технологий, стили архитектуры, лучшие практики, антишаблоны, примеры рабочих нагрузок и руководства по миграции. Используйте при выборе шаблонов архитектуры, выборе служб Azure или реализации готовых к работе решений.
