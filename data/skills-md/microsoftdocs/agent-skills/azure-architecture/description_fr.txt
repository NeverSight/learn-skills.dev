Conseils d’experts pour la conception de solutions Azure à l’aide de l’architecture Azure. Couvre les architectures de référence, les idées de solutions, les modèles de conception, les choix technologiques, les styles d'architecture, les meilleures pratiques, les anti-modèles, les exemples de charges de travail et les guides de migration. À utiliser lors de la sélection de modèles d'architecture, du choix des services Azure ou de la mise en œuvre de solutions prêtes pour la production.
