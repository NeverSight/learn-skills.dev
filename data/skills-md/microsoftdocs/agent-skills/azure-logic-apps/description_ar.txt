معرفة الخبراء بتطوير تطبيقات Azure Logic بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Logic Apps أو تصحيح أخطائها أو تحسينها. ليس لوظائف Azure (استخدم وظائف azure)، وإدارة Azure API (استخدم إدارة api-azure)، وناقل خدمة Azure (استخدم ناقل خدمة azure)، وشبكة أحداث Azure (استخدم شبكة أحداث azure).
