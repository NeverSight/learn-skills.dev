Экспертные знания для разработки Azure Logic Apps, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure Logic Apps. Не для функций Azure (используйте функции azure), управления API Azure (используйте azure-api-management), служебной шины Azure (используйте azure-service-bus), Azure Event Grid (используйте azure-event-grid).
