Connaissance approfondie du développement d'Azure Logic Apps, y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation d’applications Azure Logic Apps. Pas pour Azure Functions (utilisez azure-functions), Azure API Management (utilisez azure-api-management), Azure Service Bus (utilisez azure-service-bus), Azure Event Grid (utilisez azure-event-grid).
