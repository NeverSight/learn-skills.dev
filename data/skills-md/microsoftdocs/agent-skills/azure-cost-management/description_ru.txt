Экспертные знания для разработки Azure Cost Management, включая устранение неполадок, лучшие практики, принятие решений, ограничения и квоты, безопасность, настройку, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure Cost Management. Не для Azure Advisor (используйте azure-advisor), Azure Monitor (используйте azure-monitor), Azure Quotas (используйте azure-quotas), Azure Impact Reporting (используйте azure-impact-reporting).
