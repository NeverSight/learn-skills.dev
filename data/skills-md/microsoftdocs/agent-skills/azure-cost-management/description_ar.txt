معرفة الخبراء بتطوير إدارة تكلفة Azure بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Cost Management أو تصحيح أخطائها أو تحسينها. ليس لـ Azure Advisor (استخدم azure-advisor)، وAzure Monitor (استخدم azure-monitor)، وAzure Quotas (استخدم حصص azure)، وتقارير تأثير Azure (استخدم تقارير تأثير azure).
