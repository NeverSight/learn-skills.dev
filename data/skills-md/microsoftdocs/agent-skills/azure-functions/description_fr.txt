Connaissance approfondie du développement d'Azure Functions, y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation d’applications Azure Functions. Pas pour Azure App Service (utilisez azure-app-service), Azure Logic Apps (utilisez azure-logic-apps), Azure Container Apps (utilisez azure-container-apps), Azure Kubernetes Service (AKS) (utilisez azure-kubernetes-service).
