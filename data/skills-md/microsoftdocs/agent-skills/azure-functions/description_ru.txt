Экспертные знания для разработки функций Azure, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений функций Azure. Не для Службы приложений Azure (используйте azure-app-service), Azure Logic Apps (используйте azure-logic-apps), Azure Container Apps (используйте azure-container-apps), Azure Kubernetes Service (AKS) (используйте azure-kubernetes-service).
