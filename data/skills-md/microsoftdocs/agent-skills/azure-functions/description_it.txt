Conoscenza approfondita dello sviluppo di Funzioni di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, architettura e modelli di progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da usare durante la creazione, il debug o l'ottimizzazione delle applicazioni Funzioni di Azure. Non per Servizio app di Azure (usare azure-app-service), App per la logica di Azure (usare azure-logic-apps), App contenitore di Azure (usare azure-container-apps), Servizio Azure Kubernetes (AKS) (usare azure-kubernetes-service).
