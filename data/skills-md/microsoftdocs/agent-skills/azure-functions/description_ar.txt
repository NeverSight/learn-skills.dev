معرفة الخبراء لتطوير وظائف Azure بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Functions أو تصحيح أخطائها أو تحسينها. ليس لخدمة Azure App (استخدم azure-app-service)، وتطبيقات Azure Logic (استخدم azure-logic-apps)، وتطبيقات Azure Container (استخدم تطبيقات azure-container)، وخدمة Azure Kubernetes (AKS) (استخدم خدمة azure-kubernetes).
