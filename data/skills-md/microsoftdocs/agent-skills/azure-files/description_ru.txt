Экспертные знания для разработки файлов Azure, включая лучшие практики, принятие решений, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure Files. Не для хранилища BLOB-объектов Azure (используйте azure-blob-storage), файлов Azure NetApp (используйте azure-netapp-files), хранилища таблиц Azure (используйте azure-table-storage), хранилища очередей Azure (используйте azure-queue-storage).
