Conoscenza approfondita dello sviluppo di File di Azure, tra cui procedure consigliate, processo decisionale, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni File di Azure. Non per archiviazione BLOB di Azure (usare azure-blob-storage), Azure NetApp Files (usare azure-netapp-files), archiviazione tabelle di Azure (usare azure-table-storage), archiviazione code di Azure (usare azure-queue-storage).
