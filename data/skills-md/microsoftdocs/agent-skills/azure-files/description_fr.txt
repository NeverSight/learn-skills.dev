Connaissance approfondie du développement d'Azure Files, y compris les meilleures pratiques, la prise de décision, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation d’applications Azure Files. Pas pour Azure Blob Storage (utilisez azure-blob-storage), Azure NetApp Files (utilisez azure-netapp-files), Azure Table Storage (utilisez azure-table-storage), Azure Queue Storage (utilisez azure-queue-storage).
