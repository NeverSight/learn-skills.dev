معرفة الخبراء بتطوير ملفات Azure بما في ذلك أفضل الممارسات واتخاذ القرار والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات ملفات Azure أو تصحيح أخطائها أو تحسينها. ليس مخصصًا لتخزين Azure Blob (استخدم azure-blob-storage)، وملفات Azure NetApp (استخدم ملفات azure-netapp)، وتخزين جدول Azure (استخدم تخزين جدول azure)، وتخزين قائمة انتظار Azure (استخدم تخزين قائمة انتظار azure).
