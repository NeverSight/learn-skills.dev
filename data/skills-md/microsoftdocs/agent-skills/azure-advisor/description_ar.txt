معرفة الخبراء بتطوير Azure Advisor بما في ذلك أفضل الممارسات واتخاذ القرار والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز. يُستخدم عند إنشاء تطبيقات Azure Advisor أو تصحيح أخطائها أو تحسينها. ليس لإدارة تكلفة Azure (استخدم إدارة تكلفة azure)، وAzure Monitor (استخدم azure-monitor)، وسياسة Azure (استخدم سياسة azure)، وAzure Service Health (استخدم azure-service-health).
