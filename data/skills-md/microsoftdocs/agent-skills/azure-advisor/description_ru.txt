Экспертные знания для разработки Azure Advisor, включая лучшие практики, принятие решений, ограничения и квоты, безопасность, конфигурацию, а также интеграцию и шаблоны кодирования. Используйте при создании, отладке или оптимизации приложений Azure Advisor. Не для управления затратами Azure (используйте azure-cost-management), Azure Monitor (используйте azure-monitor), Политики Azure (используйте azure-policy), работоспособности служб Azure (используйте azure-service-health).
