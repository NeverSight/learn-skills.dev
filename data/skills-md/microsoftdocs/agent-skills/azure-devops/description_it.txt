Conoscenza approfondita dello sviluppo di Azure DevOps, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, architettura e modelli di progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni Azure DevOps. Non per Azure Boards (usare azure-boards), Azure Pipelines (usare azure-pipelines), Azure Repos (usare azure-repos), Azure Test Plans (usare azure-test-plans).
