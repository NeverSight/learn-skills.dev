Экспертные знания для разработки Azure DevOps, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure DevOps. Не для плат Azure (используйте azure-boards), Azure Pipelines (используйте azure-pipelines), Azure Repos (используйте azure-repos), планов тестирования Azure (используйте azure-test-plans).
