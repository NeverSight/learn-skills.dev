معرفة الخبراء بتطوير Azure DevOps بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure DevOps أو تصحيح أخطائها أو تحسينها. ليس للوحات Azure (استخدم لوحات azure)، وخطوط أنابيب Azure (استخدم خطوط أنابيب azure)، وAzure Repos (استخدم azure-repos)، وخطط اختبار Azure (استخدم خطط اختبار azure).
