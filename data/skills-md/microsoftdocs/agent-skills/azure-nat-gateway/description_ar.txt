معرفة الخبراء بتطوير Azure NAT Gateway بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والتكوين والنشر. يُستخدم عند إنشاء تطبيقات Azure NAT Gateway أو تصحيح أخطائها أو تحسينها. ليس لجدار الحماية Azure (استخدم جدار الحماية azure)، وAzure Load Balancer (استخدم azure-load-balancer)، وAzure Virtual Network (استخدم azure-virtual-network)، وAzure Virtual Network Manager (استخدم azure-virtual-network-manager).
