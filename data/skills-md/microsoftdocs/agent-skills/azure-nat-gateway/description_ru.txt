Экспертные знания по разработке шлюза Azure NAT, включая устранение неполадок, рекомендации, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, настройку и развертывание. Используйте при создании, отладке или оптимизации приложений Azure NAT Gateway. Не для брандмауэра Azure (используйте azure-firewall), Azure Load Balancer (используйте azure-load-balancer), виртуальной сети Azure (используйте azure-virtual-network), Azure Virtual Network Manager (используйте azure-virtual-network-manager).
