문제 해결, 모범 사례, 의사 결정, 아키텍처 및 디자인 패턴, 제한 및 할당량, 구성 및 배포를 포함한 Azure NAT 게이트웨이 개발에 대한 전문 지식입니다. Azure NAT Gateway 애플리케이션을 빌드, 디버깅 또는 최적화할 때 사용합니다. Azure Firewall(azure-firewall 사용), Azure Load Balancer(azure-load-balancer 사용), Azure Virtual Network(azure-virtual-network 사용), Azure Virtual Network Manager(azure-virtual-network-manager 사용)에는 해당되지 않습니다.
