Экспертные знания для разработки служб искусственного интеллекта Azure, включая устранение неполадок, лучшие практики, принятие решений, ограничения и квоты, безопасность, настройку, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений служб ИИ Azure. Не для машинного обучения Azure (используйте azure-machine-learning), Azure AI Search (используйте azure-cognitive-search), Azure AI Speech (используйте azure-speech), Azure AI Custom Vision (используйте azure-custom-vision).
