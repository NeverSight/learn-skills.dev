Conoscenza approfondita dello sviluppo della sicurezza di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni di sicurezza di Azure. Non per Azure Defender for Cloud (usare azure-defender-for-cloud), Protezione DDos di Azure (usare azure-ddos-protection), Firewall di Azure (usare azure-firewall), Firewall applicazione Web di Azure (usare azure-web-application-firewall).
