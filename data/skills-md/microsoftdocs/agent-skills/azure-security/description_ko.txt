문제 해결, 모범 사례, 의사 결정, 제한 및 할당량, 보안, 구성, 통합 및 코딩 패턴, 배포를 포함한 Azure 보안 개발에 대한 전문 지식입니다. Azure Security 애플리케이션을 빌드, 디버깅 또는 최적화할 때 사용합니다.
