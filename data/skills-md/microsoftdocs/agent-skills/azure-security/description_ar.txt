معرفة الخبراء بتطوير Azure Security بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Security أو تصحيح أخطائها أو تحسينها. ليس لـ Azure Defender For Cloud (استخدم azure-defender-for-cloud)، وحماية Azure DDos (استخدم حماية azure-ddos)، وجدار حماية Azure (استخدم جدار الحماية azure)، وجدار حماية تطبيق الويب Azure (استخدم جدار حماية تطبيق الويب azure).
