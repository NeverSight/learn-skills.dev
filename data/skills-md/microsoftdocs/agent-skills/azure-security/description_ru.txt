Экспертные знания для разработки безопасности Azure, включая устранение неполадок, лучшие практики, принятие решений, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений безопасности Azure. Не для Защитника Azure для облака (используйте azure-defender-for-cloud), Azure DDos Protection (используйте azure-ddos-protection), Брандмауэра Azure (используйте azure-firewall), Брандмауэра веб-приложения Azure (используйте azure-web-application-firewall).
