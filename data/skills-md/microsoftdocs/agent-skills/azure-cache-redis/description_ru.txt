Экспертные знания по разработке Azure Cache для Redis, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, безопасность, настройку, шаблоны интеграции и кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений кэша Azure для Redis. Не для Azure Managed Redis (используйте azure-managed-redis), Azure Cosmos DB (используйте azure-cosmos-db), хранилища таблиц Azure (используйте azure-table-storage).
