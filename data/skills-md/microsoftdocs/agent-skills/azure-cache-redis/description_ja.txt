トラブルシューティング、ベスト プラクティス、意思決定、アーキテクチャと設計パターン、セキュリティ、構成、統合とコーディング パターン、デプロイなどの Azure Cache for Redis 開発に関する専門知識。 Azure Cache for Redis アプリケーションを構築、デバッグ、または最適化するときに使用します。 Azure Managed Redis (azure-managed-redis を使用)、Azure Cosmos DB (azure-cosmos-db を使用)、Azure Table Storage (azure-table-storage を使用) には適用されません。
