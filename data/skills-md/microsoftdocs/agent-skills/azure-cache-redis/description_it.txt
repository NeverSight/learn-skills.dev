Conoscenza approfondita dello sviluppo di Cache Redis di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, modelli di architettura e progettazione, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da usare durante la creazione, il debug o l'ottimizzazione delle applicazioni Cache Redis di Azure. Non per Azure Managed Redis (usare azure-activated-redis), Azure Cosmos DB (usare azure-cosmos-db), Azure Table Storage (usare azure-table-storage).
