Connaissance approfondie du développement d'Azure Cache pour Redis, y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation des applications Azure Cache pour Redis. Pas pour Azure Managed Redis (utilisez azure-managed-redis), Azure Cosmos DB (utilisez azure-cosmos-db), Azure Table Storage (utilisez azure-table-storage).
