معرفة الخبراء بتطوير Azure Cache for Redis بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء Azure Cache لتطبيقات Redis أو تصحيح الأخطاء أو تحسينها. ليس لـ Azure Managed Redis (استخدم azure-managed-redis)، وAzure Cosmos DB (استخدم azure-cosmos-db)، وAzure Table Storage (استخدم تخزين جدول azure).
