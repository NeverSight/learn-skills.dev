معرفة الخبراء بتطوير Azure Portal بما في ذلك استكشاف الأخطاء وإصلاحها والحدود والحصص والأمان والتكوين. يُستخدم عند إنشاء تطبيقات Azure Portal أو تصحيح أخطائها أو تحسينها. ليس لـ Azure Cloud Shell (استخدم azure-cloud-shell)، وAzure Resource Manager (استخدم azure-resource-manager)، وAzure Policy (استخدم azure-policy)، وAzure Monitor (استخدم azure-monitor).
