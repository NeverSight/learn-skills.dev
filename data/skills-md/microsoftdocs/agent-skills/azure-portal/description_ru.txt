Экспертные знания по разработке портала Azure, включая устранение неполадок, ограничения и квоты, безопасность и настройку. Используйте при создании, отладке или оптимизации приложений портала Azure. Не для Azure Cloud Shell (используйте azure-cloud-shell), Azure Resource Manager (используйте azure-resource-manager), Политики Azure (используйте azure-policy), Azure Monitor (используйте azure-monitor).
