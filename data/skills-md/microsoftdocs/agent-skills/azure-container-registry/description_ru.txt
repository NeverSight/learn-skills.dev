Экспертные знания для разработки реестра контейнеров Azure, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений реестра контейнеров Azure. Не для приложений-контейнеров Azure (используйте azure-container-apps), экземпляров контейнеров Azure (используйте azure-container-instances), службы Azure Kubernetes (AKS) (используйте azure-kubernetes-service), Azure Red Hat OpenShift (используйте azure-redhat-openshift).
