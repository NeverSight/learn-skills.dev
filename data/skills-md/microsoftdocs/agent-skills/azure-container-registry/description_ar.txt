معرفة الخبراء بتطوير Azure Container Registry بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Container Registry أو تصحيح أخطائها أو تحسينها. ليس لتطبيقات Azure Container (استخدم azure-container-apps)، ومثيلات حاوية Azure (استخدم مثيلات حاوية azure)، وخدمة Azure Kubernetes (AKS) (استخدم خدمة azure-kubernetes)، وAzure Red Hat OpenShift (استخدم azure-redhat-openshift).
