Connaissance approfondie du développement d'Azure Container Registry, y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation d’applications Azure Container Registry. Pas pour Azure Container Apps (utilisez azure-container-apps), Azure Container Instances (utilisez azure-container-instances), Azure Kubernetes Service (AKS) (utilisez azure-kubernetes-service), Azure Red Hat OpenShift (utilisez azure-redhat-openshift).
