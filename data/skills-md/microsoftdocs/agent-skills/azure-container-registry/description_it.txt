Conoscenza approfondita dello sviluppo del Registro contenitori di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, modelli di architettura e progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni Registro contenitori di Azure. Non per app contenitore di Azure (usare azure-container-apps), istanze di contenitore di Azure (usare azure-container-instances), servizio Azure Kubernetes (AKS) (usare azure-kubernetes-service), Azure Red Hat OpenShift (usare azure-redhat-openshift).
