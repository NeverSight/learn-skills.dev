Экспертное руководство по проектированию, оценке и оптимизации рабочих нагрузок Azure с помощью Azure Well Architected. Охватывает контрольные списки проверки дизайна, рекомендации, принципы проектирования, компромиссы, руководства по обслуживанию, шаблоны рабочей нагрузки и вопросы для оценки. Используйте при разработке новых решений, анализе существующих рабочих нагрузок или применении принципов Well-Architected.
