Fachkundige Anleitung zum Entwerfen, Bewerten und Optimieren von Azure-Workloads mithilfe von Azure Well Architected. Behandelt Design-Review-Checklisten, Empfehlungen, Design-Prinzipien, Kompromisse, Service-Leitfäden, Arbeitslastmuster und Bewertungsfragen. Verwenden Sie es, wenn Sie neue Lösungen entwerfen, bestehende Workloads überprüfen oder Well-Architected-Prinzipien anwenden.
