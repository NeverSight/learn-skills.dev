Экспертные знания для разработки Microsoft Foundry, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, настройку, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Microsoft Foundry. Не для Azure AI Foundry Local (используйте azure-ai-foundry-local), Azure Foundry Classic (используйте azure-foundry-classic), Машинное обучение Azure (используйте azure-machine-learning), Azure Databricks (используйте azure-databricks).
