معرفة الخبراء بتطوير Microsoft Foundry بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Microsoft Foundry أو تصحيح أخطائها أو تحسينها. ليس لـ Azure AI Foundry Local (استخدم azure-ai-foundry-local)، وAzure Foundry Classic (استخدم azure-foundry-classic)، وAzure Machine Learning (استخدم التعلم الآلي azure)، وAzure Databricks (استخدم azure-databricks).
