SAP HANA on Azure Large Instances 開発に関する専門知識 (トラブルシューティング、ベスト プラクティス、意思決定、アーキテクチャと設計パターン、制限とクォータ、セキュリティ、構成、統合とコーディング パターン、デプロイメントなど)。 Azure Large Instances アプリケーションで SAP HANA を構築、デバッグ、または最適化するときに使用します。 Azure Large Instances (azure-large-instances を使用)、Azure Virtual Machines (azure-virtual-machines を使用)、Azure Virtual Machine Scale Sets (azure-vm-scalesets を使用)、Azure Baremetal Infrastructure (azure-baremetal-infrastructor を使用) には適用されません。
