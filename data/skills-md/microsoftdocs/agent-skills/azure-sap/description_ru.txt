Экспертные знания SAP HANA по разработке крупных экземпляров Azure, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации SAP HANA в приложениях крупных экземпляров Azure. Не для крупных экземпляров Azure (используйте azure-large-instances), виртуальных машин Azure (используйте azure-virtual-machines), масштабируемых наборов виртуальных машин Azure (используйте azure-vm-scalesets), базовой инфраструктуры Azure (используйте azure-baremetal-infrastructure).
