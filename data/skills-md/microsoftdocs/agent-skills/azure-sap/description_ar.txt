معرفة الخبراء لـ SAP HANA بشأن تطوير مثيلات Azure الكبيرة بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء SAP HANA أو تصحيح الأخطاء أو تحسينه على تطبيقات Azure Large Instances. ليس لمثيلات Azure الكبيرة (استخدم مثيلات azure الكبيرة)، وأجهزة Azure الافتراضية (استخدم أجهزة azure الافتراضية)، ومجموعات مقياس الجهاز الظاهري لـ Azure (استخدم مجموعات مقياس azure-vm)، والبنية الأساسية لـ Azure Baremetal (استخدم البنية الأساسية المعدنية اللازوردية).
