Connaissance approfondie du développement de SAP HANA sur Azure Large Instances, y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation de SAP HANA sur des applications Azure Large Instances. Pas pour les grandes instances Azure (utilisez Azure-large-instances), les machines virtuelles Azure (utilisez les machines virtuelles Azure), les groupes de machines virtuelles identiques Azure (utilisez les Azure-VM-scalesets), l'infrastructure Azure Baremetal (utilisez l'infrastructure Azure-baremetal).
