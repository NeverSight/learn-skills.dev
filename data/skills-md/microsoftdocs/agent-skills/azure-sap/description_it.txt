Conoscenza approfondita di SAP HANA sullo sviluppo di istanze Large di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, modelli di architettura e progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione di SAP HANA su applicazioni Istanze Large di Azure. Non per istanze large di Azure (usare azure-large-instances), macchine virtuali di Azure (usare azure-virtual-machines), set di scalabilità di macchine virtuali di Azure (usare azure-vm-scalesets), infrastruttura baremetal di Azure (usare azure-baremetal-infrastructure).
