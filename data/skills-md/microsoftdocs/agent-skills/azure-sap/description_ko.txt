문제 해결, 모범 사례, 의사 결정, 아키텍처 및 디자인 패턴, 제한 및 할당량, 보안, 구성, 통합 및 코딩 패턴, 배포를 포함한 Azure 대규모 인스턴스 개발의 SAP HANA에 대한 전문 지식입니다. Azure 대규모 인스턴스 애플리케이션에서 SAP HANA를 빌드, 디버깅 또는 최적화할 때 사용합니다. Azure 대규모 인스턴스(azure-large-instances 사용), Azure Virtual Machines(azure-virtual-machines 사용), Azure Virtual Machine Scale Set(azure-vm-scalesets 사용), Azure Baremetal 인프라(azure-baremetal-infrastructure 사용)에는 해당되지 않습니다.
