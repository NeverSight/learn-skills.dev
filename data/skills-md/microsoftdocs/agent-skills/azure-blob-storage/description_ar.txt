معرفة الخبراء بتطوير Azure Blob Storage بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Blob Storage أو تصحيح الأخطاء أو تحسينها. ليس لملفات Azure (استخدم ملفات azure)، وتخزين جدول Azure (استخدم تخزين جدول azure)، وتخزين قائمة انتظار Azure (استخدم تخزين قائمة انتظار azure).
