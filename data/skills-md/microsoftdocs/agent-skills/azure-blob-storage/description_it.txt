Conoscenza approfondita dello sviluppo di Archiviazione BLOB di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da usare durante la creazione, il debug o l'ottimizzazione delle applicazioni di archiviazione BLOB di Azure. Non per File di Azure (usare azure-files), Archiviazione tabelle di Azure (usare azure-table-storage), Archiviazione code di Azure (usare azure-queue-storage).
