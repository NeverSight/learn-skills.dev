Экспертные знания по разработке хранилища BLOB-объектов Azure, включая устранение неполадок, рекомендации, принятие решений, ограничения и квоты, безопасность, настройку, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений хранилища BLOB-объектов Azure. Не для файлов Azure (используйте файлы azure), хранилища таблиц Azure (используйте хранилище таблиц azure), хранилища очередей Azure (используйте хранилище azure-queue).
