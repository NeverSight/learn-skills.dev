معرفة الخبراء بتطوير Azure Table Storage بما في ذلك أفضل الممارسات وأنماط التصميم والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز. يُستخدم عند إنشاء تطبيقات Azure Table Storage أو تصحيح أخطائها أو تحسينها. ليس لـ Azure Cosmos DB (استخدم azure-cosmos-db)، وAzure Blob Storage (استخدم azure-blob-storage)، وAzure Queue Storage (استخدم azure-queue-storage)، وملفات Azure (استخدم ملفات azure).
