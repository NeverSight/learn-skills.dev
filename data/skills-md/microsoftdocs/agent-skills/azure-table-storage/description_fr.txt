Connaissance approfondie du développement d'Azure Table Storage, y compris les meilleures pratiques, les modèles d'architecture et de conception, les limites et les quotas, la sécurité, la configuration, les intégrations et les modèles de codage. À utiliser lors de la création, du débogage ou de l’optimisation d’applications Azure Table Storage. Pas pour Azure Cosmos DB (utilisez azure-cosmos-db), Azure Blob Storage (utilisez azure-blob-storage), Azure Queue Storage (utilisez azure-queue-storage), Azure Files (utilisez azure-files).
