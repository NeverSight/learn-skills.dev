Conoscenza approfondita dello sviluppo di Archiviazione tabelle di Azure, tra cui best practice, architettura e modelli di progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica. Da utilizzare durante la creazione, il debug o l'ottimizzazione di applicazioni di archiviazione tabelle di Azure. Non per Azure Cosmos DB (usare azure-cosmos-db), Archiviazione BLOB di Azure (usare azure-blob-storage), Archiviazione code di Azure (usare azure-queue-storage), File di Azure (usare azure-files).
