Экспертные знания по разработке хранилища таблиц Azure, включая лучшие практики, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, а также интеграцию и шаблоны кодирования. Используйте при создании, отладке или оптимизации приложений хранилища таблиц Azure. Не для Azure Cosmos DB (используйте azure-cosmos-db), хранилища BLOB-объектов Azure (используйте azure-blob-storage), хранилища очередей Azure (используйте azure-queue-storage), файлов Azure (используйте azure-files).
