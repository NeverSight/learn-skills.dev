Экспертные знания для разработки приложений-контейнеров Azure, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure Container Apps. Не для Службы Azure Kubernetes (AKS) (используйте azure-kubernetes-service), Экземпляров контейнеров Azure (используйте azure-container-instances), Службы приложений Azure (используйте azure-app-service), Приложений Azure Spring (используйте azure-spring-apps).
