Conoscenza approfondita dello sviluppo di app per contenitori di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, architettura e modelli di progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da usare durante la creazione, il debug o l'ottimizzazione delle applicazioni delle app contenitore di Azure. Non per il servizio Azure Kubernetes (AKS) (usare azure-kubernetes-service), Istanze di contenitore di Azure (usare azure-container-instances), Servizio app di Azure (usare azure-app-service), App Azure Spring (usare azure-spring-apps).
