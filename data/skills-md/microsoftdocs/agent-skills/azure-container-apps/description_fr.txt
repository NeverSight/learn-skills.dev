Connaissance approfondie du développement d'Azure Container Apps, y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation des applications Azure Container Apps. Pas pour Azure Kubernetes Service (AKS) (utilisez azure-kubernetes-service), Azure Container Instances (utilisez azure-container-instances), Azure App Service (utilisez azure-app-service), Azure Spring Apps (utilisez azure-spring-apps).
