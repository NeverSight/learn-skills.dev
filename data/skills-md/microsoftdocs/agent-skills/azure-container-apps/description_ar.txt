معرفة الخبراء بتطوير Azure Container Apps بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Container Apps أو تصحيح أخطائها أو تحسينها. ليس لخدمة Azure Kubernetes (AKS) (استخدم خدمة azure-kubernetes)، ومثيلات حاوية Azure (استخدم مثيلات حاوية azure)، وخدمة Azure App (استخدم خدمة تطبيق azure)، وتطبيقات Azure Spring (استخدم تطبيقات azure-spring).
