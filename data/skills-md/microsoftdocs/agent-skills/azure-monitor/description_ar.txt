معرفة الخبراء بتطوير Azure Monitor بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Monitor أو تصحيح أخطائها أو تحسينها. ليس لـ Azure Managed Grafana (استخدم azure-managed-grafana)، وAzure Network Watcher (استخدم azure-network-watcher)، وAzure Service Health (استخدم azure-service-health)، وAzure Defender For Cloud (استخدم azure-defender-for-cloud).
