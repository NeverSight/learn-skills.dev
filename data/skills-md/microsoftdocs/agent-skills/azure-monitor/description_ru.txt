Экспертные знания для разработки Azure Monitor, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure Monitor. Не для Azure Managed Grafana (используйте azure-managed-grafana), Azure Network Watcher (используйте azure-network-watcher), Azure Service Health (используйте azure-service-health), Azure Defender для облака (используйте azure-defender-for-cloud).
