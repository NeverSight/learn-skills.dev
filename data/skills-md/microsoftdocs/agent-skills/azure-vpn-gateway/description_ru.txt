Экспертные знания для разработки VPN-шлюзов Azure, включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, конфигурацию, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure VPN Gateway. Не для виртуальной сети Azure (используйте azure-virtual-network), виртуальной глобальной сети Azure (используйте azure-virtual-wan), Azure ExpressRoute (используйте azure-expressroute), шлюза приложений Azure (используйте azure-application-gateway).
