Conoscenza approfondita dello sviluppo del gateway VPN di Azure, tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, modelli di architettura e progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni gateway VPN di Azure. Non per rete virtuale di Azure (usare azure-virtual-network), WAN virtuale di Azure (usare azure-virtual-wan), Azure ExpressRoute (usare azure-expressroute), Gateway applicazione di Azure (usare azure-application-gateway).
