معرفة الخبراء بتطوير Azure VPN Gateway بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure VPN Gateway أو تصحيح أخطائها أو تحسينها. ليس لشبكة Azure الافتراضية (استخدم azure-virtual-network)، وAzure Virtual WAN (استخدم azure-virtual-wan)، وAzure ExpressRoute (استخدم azure-expressroute)، وبوابة تطبيق Azure (استخدم بوابة تطبيق azure).
