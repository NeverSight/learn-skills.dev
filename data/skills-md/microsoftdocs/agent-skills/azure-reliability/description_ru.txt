Экспертные знания для разработки систем обеспечения надежности Azure, включая лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, а также развертывание. Используйте при создании, отладке или оптимизации приложений обеспечения надежности Azure. Не для устойчивости Azure (используйте azure-resiliency), Azure Monitor (используйте azure-monitor), работоспособности служб Azure (используйте azure-service-health), Azure Site Recovery (используйте azure-site-recovery).
