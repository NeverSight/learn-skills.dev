Conoscenza approfondita dello sviluppo dell'affidabilità di Azure, incluse procedure consigliate, processo decisionale, architettura e modelli di progettazione, limiti e quote e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni di affidabilità di Azure. Non per la resilienza di Azure (usare azure-resiliency), Monitoraggio di Azure (usare azure-monitor), Integrità dei servizi di Azure (usare azure-service-health), Azure Site Recovery (usare azure-site-recovery).
