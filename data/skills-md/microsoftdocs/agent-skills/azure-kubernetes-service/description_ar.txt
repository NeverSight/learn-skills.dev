معرفة الخبراء بتطوير خدمة Azure Kubernetes (AKS) بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار وأنماط الهندسة والتصميم والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Kubernetes Service (AKS) أو تصحيح أخطائها أو تحسينها. ليس لـ Azure Kubernetes Service Edge Essentials (استخدم azure-aks-edge-essentials)، وAzure Container Apps (استخدم azure-container-apps)، ومثيلات حاوية Azure (استخدم مثيلات حاوية azure)، وAzure Red Hat OpenShift (استخدم azure-redhat-openshift).
