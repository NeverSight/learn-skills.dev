Azure Kubernetes Service (AKS) 開発に関する専門知識 (トラブルシューティング、ベスト プラクティス、意思決定、アーキテクチャと設計パターン、制限とクォータ、セキュリティ、構成、統合とコーディング パターン、デプロイなど)。 Azure Kubernetes Service (AKS) アプリケーションを構築、デバッグ、または最適化するときに使用します。 Azure Kubernetes Service Edge Essentials (azure-aks-edge-essentials を使用)、Azure Container Apps (azure-container-apps を使用)、Azure Container Instances (azure-container-instances を使用)、Azure Red Hat OpenShift (azure-redhat-openshift を使用) には適用されません。
