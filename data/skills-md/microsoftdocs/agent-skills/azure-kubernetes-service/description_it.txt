Conoscenza approfondita dello sviluppo del servizio Azure Kubernetes (AKS), tra cui risoluzione dei problemi, procedure consigliate, processo decisionale, modelli di architettura e progettazione, limiti e quote, sicurezza, configurazione, integrazioni e modelli di codifica e distribuzione. Da utilizzare durante la creazione, il debug o l'ottimizzazione delle applicazioni del servizio Azure Kubernetes (AKS). Non per Azure Kubernetes Service Edge Essentials (usare azure-aks-edge-essentials), App contenitore di Azure (usare azure-container-apps), Istanze di contenitore di Azure (usare azure-container-instances), Azure Red Hat OpenShift (usare azure-redhat-openshift).
