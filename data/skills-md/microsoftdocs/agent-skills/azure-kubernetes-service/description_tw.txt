Azure Kubernetes 服務 (AKS) 開發的專業知識，包括故障排除、最佳實踐、決策、體系結構和設計模式、限制和配額、安全性、配置、整合和編碼模式以及部署。在建置、偵錯或最佳化 Azure Kubernetes 服務 (AKS) 應用程式時使用。不適用於 Azure Kubernetes Service Edge Essentials（使用 azure-aks-edge-essentials）、Azure 容器應用程式（使用 azure-container-apps）、Azure 容器實例（使用 azure-container-instances）、Azure Red HatShift（使用 azurepift-openshift）。
