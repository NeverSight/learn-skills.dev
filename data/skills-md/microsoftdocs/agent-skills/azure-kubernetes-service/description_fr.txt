Connaissance approfondie du développement d'Azure Kubernetes Service (AKS), y compris le dépannage, les meilleures pratiques, la prise de décision, les modèles d'architecture et de conception, les limites et quotas, la sécurité, la configuration, les intégrations et les modèles de codage, ainsi que le déploiement. À utiliser lors de la création, du débogage ou de l’optimisation d’applications Azure Kubernetes Service (AKS). Pas pour Azure Kubernetes Service Edge Essentials (utilisez azure-aks-edge-essentials), Azure Container Apps (utilisez azure-container-apps), Azure Container Instances (utilisez azure-container-instances), Azure Red Hat OpenShift (utilisez azure-redhat-openshift).
