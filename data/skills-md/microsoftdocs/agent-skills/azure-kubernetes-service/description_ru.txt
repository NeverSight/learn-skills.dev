Экспертные знания для разработки Azure Kubernetes Service (AKS), включая устранение неполадок, лучшие практики, принятие решений, шаблоны архитектуры и проектирования, ограничения и квоты, безопасность, настройку, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений службы Azure Kubernetes (AKS). Не для Azure Kubernetes Service Edge Essentials (используйте azure-aks-edge-essentials), приложений-контейнеров Azure (используйте azure-container-apps), экземпляров контейнеров Azure (используйте azure-container-instances), Azure Red Hat OpenShift (используйте azure-redhat-openshift).
