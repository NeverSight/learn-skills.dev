Экспертные знания для разработки Azure Key Vault, включая устранение неполадок, рекомендации, принятие решений, ограничения и квоты, безопасность, настройку, интеграцию и шаблоны кодирования, а также развертывание. Используйте при создании, отладке или оптимизации приложений Azure Key Vault. Не для Azure Cloud Hsm (используйте azure-cloud-hsm), Azure Dedicated HSM (используйте azure-dedicated-hsm), Azure Payment Hsm (используйте azure-pay-hsm), Azure Attestation (используйте azure-attestation).
