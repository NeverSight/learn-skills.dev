معرفة الخبراء بتطوير Azure Key Vault بما في ذلك استكشاف الأخطاء وإصلاحها وأفضل الممارسات واتخاذ القرار والحدود والحصص والأمان والتكوين والتكامل وأنماط الترميز والنشر. يُستخدم عند إنشاء تطبيقات Azure Key Vault أو تصحيح أخطائها أو تحسينها. ليس لـ Azure Cloud Hsm (استخدم azure-cloud-hsm)، وAzure Dedicated HSM (استخدم azure-dedicated-hsm)، وAzure Payment Hsm (استخدم azure- Payment-hsm)، وAzure Attestation (استخدم azure-attestation).
