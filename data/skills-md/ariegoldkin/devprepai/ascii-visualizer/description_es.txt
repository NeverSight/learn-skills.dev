Crea hermosas visualizaciones artísticas ASCII para planos, arquitecturas, flujos de trabajo y datos. Esta habilidad debe usarse al explicar la arquitectura del sistema, crear planes de implementación, mostrar flujos de trabajo, visualizar comparaciones o documentar estructuras de archivos. NO para resaltado de sintaxis de código o tablas de rebajas. Al usuario le encanta explícitamente el arte ASCII: utilícelo generosamente para la comunicación visual.
