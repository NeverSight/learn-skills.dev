Crée de superbes visualisations artistiques ASCII pour les plans, les architectures, les flux de travail et les données. Cette compétence doit être utilisée pour expliquer l'architecture du système, créer des plans de mise en œuvre, afficher des flux de travail, visualiser des comparaisons ou documenter les structures de fichiers. PAS pour la coloration syntaxique du code ou les tableaux de démarques. L'utilisateur aime explicitement l'art ASCII - utilisez-le généreusement pour la communication visuelle.
