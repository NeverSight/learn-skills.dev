Генерируйте обычные сообщения о фиксации git в соответствии с форматом соглашения Angular. Используйте этот навык при создании коммитов, написании сообщений о коммитах или просмотре истории git. Триггеры включают «git commit», «commit message», «changelog» или запросы на изменения системы контроля версий.
