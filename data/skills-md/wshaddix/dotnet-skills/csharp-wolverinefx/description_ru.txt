Создавайте .NET-приложения с помощью WolverineFX для обмена сообщениями, HTTP-сервисов и источников событий. Используйте при реализации обработчиков команд, обработчиков сообщений, конечных точек HTTP с WolverineFx.HTTP, шаблонов исходящих сообщений транзакций, источников событий с помощью Marten, архитектур CQRS, каскадных сообщений, пакетной обработки сообщений или настройки транспортных средств, таких как RabbitMQ, Azure Service Bus или Amazon SQS.
