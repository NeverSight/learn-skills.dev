Créez des applications .NET avec WolverineFX pour la messagerie, les services HTTP et la recherche d'événements. À utiliser lors de l'implémentation de gestionnaires de commandes, de gestionnaires de messages, de points de terminaison HTTP avec WolverineFx.HTTP, de modèles de boîte d'envoi transactionnels, de sourcing d'événements avec Marten, d'architectures CQRS, de messages en cascade, de traitement de messages par lots ou de configuration de transports comme RabbitMQ, Azure Service Bus ou Amazon SQS.
