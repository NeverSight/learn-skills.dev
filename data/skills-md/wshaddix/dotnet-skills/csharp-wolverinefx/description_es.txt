Cree aplicaciones .NET con WolverineFX para mensajería, servicios HTTP y abastecimiento de eventos. Úselo al implementar controladores de comandos, controladores de mensajes, puntos finales HTTP con WolverineFx.HTTP, patrones de bandeja de salida transaccional, abastecimiento de eventos con Marten, arquitecturas CQRS, mensajes en cascada, procesamiento de mensajes por lotes o configuración de transportes como RabbitMQ, Azure Service Bus o Amazon SQS.
