Crea applicazioni .NET con WolverineFX per messaggistica, servizi HTTP e sourcing di eventi. Da utilizzare durante l'implementazione di gestori di comandi, gestori di messaggi, endpoint HTTP con WolverineFx.HTTP, modelli di posta in uscita transazionale, sourcing di eventi con Marten, architetture CQRS, messaggi a cascata, elaborazione di messaggi batch o configurazione di trasporti come RabbitMQ, bus di servizio di Azure o Amazon SQS.
