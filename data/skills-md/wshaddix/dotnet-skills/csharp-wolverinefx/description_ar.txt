أنشئ تطبيقات .NET باستخدام WolverineFX للمراسلة وخدمات HTTP ومصادر الأحداث. يُستخدم عند تنفيذ معالجات الأوامر، أو معالجات الرسائل، أو نقاط نهاية HTTP مع WolverineFx.HTTP، أو أنماط صندوق الصادر للمعاملات، أو مصادر الأحداث مع Marten، أو بنيات CQRS، أو الرسائل المتتالية، أو معالجة الرسائل المجمعة، أو تكوين وسائل النقل مثل RabbitMQ، أو Azure Service Bus، أو Amazon SQS.
