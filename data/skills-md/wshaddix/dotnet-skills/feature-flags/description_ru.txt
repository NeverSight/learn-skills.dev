Шаблоны Microsoft.FeatureManagement для переключения функций, постепенного развертывания и A/B-тестирования в приложениях ASP.NET Core Razor Pages. Используйте при реализации переключения функций в приложениях ASP.NET Core, настройке постепенного развертывания функций или настройке сценариев A/B-тестирования с помощью флагов функций.
