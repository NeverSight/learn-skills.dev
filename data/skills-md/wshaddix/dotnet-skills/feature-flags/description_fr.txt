Modèles Microsoft.FeatureManagement pour les basculements de fonctionnalités, les déploiements progressifs et les tests A/B dans les applications ASP.NET Core Razor Pages. À utiliser lors de l'implémentation de bascules de fonctionnalités dans les applications ASP.NET Core, de la configuration de déploiements progressifs de fonctionnalités ou de la configuration de scénarios de tests A/B avec des indicateurs de fonctionnalités.
