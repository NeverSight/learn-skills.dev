Modelli Microsoft.FeatureManagement per attivazioni/disattivazioni di funzionalità, implementazioni graduali e test A/B nelle applicazioni ASP.NET Core Razor Pages. Utilizzare quando si implementano attivazioni/disattivazioni di funzionalità nelle applicazioni ASP.NET Core, si impostano distribuzioni graduali di funzionalità o si configurano scenari di test A/B con flag di funzionalità.
