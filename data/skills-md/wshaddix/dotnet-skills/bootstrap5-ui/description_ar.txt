أنماط Bootstrap 5.3 على مستوى الإنتاج لإنشاء واجهات مستخدم ويب حديثة وسريعة الاستجابة بتنسيقي HTML وASP.NET Core Razor Pages/Views. يُستخدم عند إنشاء أو تصميم صفحات الويب، أو التخطيطات، أو التنقل، أو النماذج، أو البطاقات، أو الوسائط، أو الجداول، أو أي مكون لواجهة المستخدم مع Bootstrap 5. يغطي نظام الشبكة، ونقاط التوقف المستجيبة، وفئات الأدوات المساعدة، وأوضاع الألوان (غامقة/فاتحة)، وإمكانية الوصول، والتكامل مع مساعدي علامات ASP.NET Core وبناء جملة Razor. قم بتشغيل أي مهمة تتضمن فئات Bootstrap CSS، أو تخطيطات HTML سريعة الاستجابة، أو تصميم واجهة مستخدم Razor Page، أو تصميم الواجهة الأمامية لتطبيقات الويب .NET.
