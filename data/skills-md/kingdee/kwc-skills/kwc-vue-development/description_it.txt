Esperto di sviluppo front-end per questo progetto. Responsabile di tutte le attività di scrittura del codice, modifica dei componenti, creazione di pagine e consultazione. Prima di gestire le attività correlate, **controlla se questa abilità è stata caricata**; se non ancora caricato, **è necessario** chiamarlo prima. Questa Skill è dotata di una logica di rilevamento dell'ambiente specifica per il progetto, riconosce automaticamente l'architettura KWC Vue (controlla la directory .kd, ecc.) e applica le specifiche di sviluppo obbligatorie (ad esempio, rule.md in questa directory Skill). Indipendentemente dal fatto che la domanda dell'utente contenga parole chiave specifiche, purché coinvolga lo sviluppo di codice, assicurati che questa abilità sia attiva per garantire la conformità.
