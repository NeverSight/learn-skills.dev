L'esperto di sviluppo front-end per questo progetto. Responsabile di tutte le attività di scrittura del codice, modifica dei componenti, creazione di pagine e consulenza. Prima di gestire le attività correlate, **controlla se questa abilità è caricata**; se non ancora caricato, **è necessario** richiamarlo prima. Questa Skill dispone di una logica di rilevamento dell'ambiente specifica del progetto integrata, che identifica automaticamente l'architettura KWC React (controllando la directory .kd, ecc.) e applica standard di sviluppo obbligatori (ad esempio, rule.md in questa directory Skill). Indipendentemente dal fatto che la domanda dell'utente contenga parole chiave specifiche, purché coinvolga lo sviluppo di codice, assicurati che questa abilità sia attiva per garantire la conformità.
