Automatice las interacciones del navegador web utilizando lenguaje natural mediante comandos CLI. Se utiliza cuando el usuario solicita navegar por sitios web, navegar por páginas web, extraer datos de sitios web, tomar capturas de pantalla, completar formularios, hacer clic en botones o interactuar con aplicaciones web. Admite sesiones remotas de Browserbase con resolución automática de CAPTCHA, modo oculto anti-bot y proxies residenciales, ideal para raspar sitios web protegidos, evitar la detección de bots e interactuar con páginas con mucho JavaScript.
