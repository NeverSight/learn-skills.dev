Rufen Sie Webseiten ab und geben Sie deren Inhalte, Header und Metadaten mit der Browserbase Fetch API zurück. Verwenden Sie diese Option, wenn der Benutzer Seiteninhalte ohne eine vollständige Browsersitzung abrufen möchte – ideal zum Scrapen statischer Seiten, Überprüfen von HTTP-Antworten oder Abrufen der Seitenquelle. Unterstützt Proxys, Umleitungskontrolle und unsichere SSL-Umgehung.
