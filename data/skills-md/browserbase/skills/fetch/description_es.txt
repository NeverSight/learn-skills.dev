Obtenga páginas web y devuelva su contenido, encabezados y metadatos utilizando la API Browserbase Fetch. Úselo cuando el usuario desee recuperar el contenido de la página sin una sesión completa del navegador; ideal para extraer páginas estáticas, verificar respuestas HTTP u obtener el origen de la página. Admite servidores proxy, control de redireccionamiento y derivación de SSL inseguro.
