Definición de modelo independiente del lenguaje y generador de código RPC a partir de especificaciones YAML. Úselo para generar datos con seguridad de tipos (Rust, Python, TypeScript, Java, Swift) a partir de una única fuente de verdad, definir interfaces de servicio RPC con tipos de solicitud/respuesta, crear definiciones de tipos en varios idiomas para microservicios, trabajar con archivos de especificaciones .yaml existentes en el proyecto o crear nuevas definiciones de modelos, tipos de enumeración o métodos de servicio.
