Definizione del modello indipendente dal linguaggio e generatore di codice RPC dalle specifiche YAML. Utilizzare per generare dati indipendenti dai tipi (Rust, Python, TypeScript, Java, Swift) da un'unica fonte attendibile, definire interfacce di servizio RPC con tipi di richiesta/risposta, creare definizioni di tipi multilingue per microservizi, lavorare con file di specifiche .yaml esistenti nel progetto o creare nuove definizioni di modello, tipi di enumerazione o metodi di servizio.
