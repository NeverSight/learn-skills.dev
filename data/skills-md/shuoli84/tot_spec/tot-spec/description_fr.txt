Définition de modèle indépendant du langage et générateur de code RPC à partir des spécifications YAML. À utiliser pour générer des données de type sécurisé (Rust, Python, TypeScript, Java, Swift) à partir d'une source unique de vérité, définir des interfaces de service RPC avec des types de requête/réponse, créer des définitions de types multilingues pour les microservices, travailler avec des fichiers de spécifications .yaml existants dans le projet ou créer de nouvelles définitions de modèle, des types d'énumération ou des méthodes de service.
