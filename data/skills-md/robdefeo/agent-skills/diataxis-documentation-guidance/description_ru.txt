Используйте этот навык для классификации, определения объема, разделения, составления черновиков и проверки документации с помощью платформы Diataxis. Триггеры включают выбор режима учебника/инструкций/справки/объяснения, разрешение страниц в смешанном режиме, переписывание документов в целевой режим и проверку черновиков с помощью контрольного списка.
