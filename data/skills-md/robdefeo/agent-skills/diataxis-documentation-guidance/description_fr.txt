Utilisez cette compétence pour classer, définir la portée, diviser, rédiger et réviser la documentation avec le framework Diataxis. Les déclencheurs incluent le choix du mode tutoriel/comment faire/référence/explication, la résolution des pages en mode mixte, la réécriture des documents en mode cible et la validation des brouillons avec une liste de contrôle.
