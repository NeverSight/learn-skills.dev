Создавайте изображения с помощью моделей Pruna P-Image через интерфейс командной строки inference.sh. Модели: P-Image, P-Image-LoRA, P-Image-Edit, P-Image-Edit-LoRA. Возможности: преобразование текста в изображение, редактирование изображений, стили LoRA, компоновка нескольких изображений, быстрый вывод. Pruna оптимизирует модели для скорости без потери качества. Триггеры: pruna, p-image, pruna image, быстрое создание изображения, оптимизированный поток, pruna ai, p-изображение, быстрое изображение ai, экономичное создание изображения, дешевое создание изображения
