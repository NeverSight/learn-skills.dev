Crea file di logo animati (Lottie JSON, GIF, MP4) da immagini di logo statiche. Questa abilità deve essere utilizzata quando gli utenti forniscono un'immagine del logo (PNG/SVG/JPG) e richiedono qualsiasi tipo di animazione del logo, grafica in movimento, effetto logo animato, animazione della forma d'onda, logo che rimbalza, logo rotante, logo pulsante, logo oscillante o chiedono di "animare il mio logo" o "far muovere il mio logo". Genera file di animazione autonomi (non artefatti React/HTML). Genera Lottie JSON con rendering GIF/MP4 automatico, convalida del loop perfetta e modelli di motion design professionali.
