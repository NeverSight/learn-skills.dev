Feishu IM Message Reader Tool User Guide, covering conversation message retrieval, thread reply reading, cross-conversation message search, and image/file resource download.

**Use this Skill when**:
(1) You need to fetch history messages from group or one-on-one chats
(2) You need to read replies inside a thread
(3) You need to search messages across conversations (by keyword, sender, time, etc.)
(4) Messages contain images, files, audio, or video that you want to download
(5) Users mention "chat history", "message", "what was said in the group", "thread reply", "search message", "image", "file download"
(6) You need to filter messages by time range or paginate to load more
