Комплексный обзор запроса на включение GitHub с использованием интерфейса командной строки gh. Просматривает PR-разногласия по связанным проблемам, соглашениям о репо и контексту кодовой базы. Проверяет наличие ошибок, угроз безопасности, согласованность стиля и соответствие целям. Запускает доступные команды lint, проверки типа и тестирования. Создает приоритетный отчет с вердиктом «одобрить/блокировать». Используйте этот навык всякий раз, когда пользователь просит «рассмотреть PR», «проверить PR».
