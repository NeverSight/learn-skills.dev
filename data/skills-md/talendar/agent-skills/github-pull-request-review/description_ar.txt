مراجعة شاملة لطلبات سحب GitHub باستخدام gh CLI. تختلف مراجعات العلاقات العامة مقابل المشكلات المرتبطة واتفاقيات الريبو وسياق قاعدة التعليمات البرمجية. التحقق من الأخطاء والمخاطر الأمنية واتساق الأسلوب ومواءمة الأهداف. يقوم بتشغيل أوامر الوبر والتحقق من النوع والاختبار المتاحة. يُنتج تقريرًا محددًا حسب الأولوية مع حكم بالموافقة/الحظر. استخدم هذه المهارة عندما يطلب المستخدم "مراجعة العلاقات العامة"، "التحقق من العلاقات العامة".
