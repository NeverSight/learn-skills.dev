Examen complet des demandes d'extraction GitHub à l'aide de la CLI gh. Examine les différences de relations publiques par rapport aux problèmes liés, aux conventions de dépôt et au contexte de la base de code. Vérifie les bogues, les risques de sécurité, la cohérence du style et l’alignement des objectifs. Exécute les commandes lint, de vérification de type et de test disponibles. Produit un rapport hiérarchisé avec un verdict d’approbation/blocage. Utilisez cette compétence chaque fois que l'utilisateur demande « examiner un PR », « vérifier le PR »
