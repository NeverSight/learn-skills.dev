Находите и редактируйте символы в коде с помощью ast-grep (на основе Tree-Sitter). Используйте при поиске определений функций, определений классов, использования символов, переименования символов или замены кода на JavaScript, TypeScript, Python, Go, Rust и других языках. НЕ для Clojure — вместо этого используйте символы clj.
