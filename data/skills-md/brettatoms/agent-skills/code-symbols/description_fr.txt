Recherchez et modifiez des symboles dans le code à l'aide d'ast-grep (basé sur un tree-sitter). À utiliser pour rechercher des définitions de fonctions, des définitions de classes, des utilisations de symboles, renommer des symboles ou remplacer du code dans JavaScript, TypeScript, Python, Go, Rust et d'autres langages. PAS pour Clojure - utilisez plutôt les symboles clj.
