يقوم بإنشاء صور AI باستخدام وكيل BlockEden OpenAI (DALL-E 3 / DALL-E 2 / gpt-image-1). استخدم هذه المهارة عندما يريد المستخدم إنشاء أو إنشاء أو رسم أو إنشاء أو إنتاج صورة أو صورة أو رسم توضيحي أو عمل فني أو مرئي - حتى لو لم يقول "DALL-E" أو "BlockEden". قم بتشغيل عبارات مثل "إنشاء صورة لـ"، أو "إنشاء صورة لـ"، أو "اصنع لي رسمًا توضيحيًا"، أو "ارسم مشهدًا"، أو "تصور هذا"، أو "أريد صورة"، أو "أنشئ بناءً على هذه الصورة/عنوان URL". يتطلب تعيين متغير البيئة BLOCKEDEN_ACCESS_KEY.
