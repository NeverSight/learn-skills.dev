Помощник по написанию ИИ для китайских научных статей. По умолчанию обсуждается перед выполнением, автоматически создается и поддерживается контекст плана, поддерживается написание де-ИИ, обзоры литературы, диаграммы Python и настройка среды Miniconda, по умолчанию используется доставка Markdown/простого текста и предоставляются рекомендации по миграции в Word.
