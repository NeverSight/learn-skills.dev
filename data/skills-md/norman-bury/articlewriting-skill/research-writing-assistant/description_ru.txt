Помощник по написанию ИИ для китайских исследовательских работ. По умолчанию обсуждается перед действием, автоматически создается и поддерживается контекст плана, поддерживается написание де-ИИ, обзор литературы, диаграммы Python и настройка среды Miniconda, по умолчанию доставка Markdown/простого текста с инструкциями по миграции Word.
