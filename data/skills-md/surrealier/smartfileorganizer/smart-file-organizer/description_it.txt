Analizza il contenuto dei file in un albero di directory e rinomina in modo intelligente i file in base a ciò che contengono, riorganizzandoli facoltativamente in una struttura di cartelle logica. Utilizza questa abilità ogni volta che l'utente desidera ripulire nomi di file disordinati, organizzare file scaricati, ordinare documenti per contenuto, rinominare file che hanno nomi senza significato (come IMG_20240101.jpg o document(3).pdf) o riordinare qualsiasi cartella in cui i nomi di file non riflettono il loro contenuto effettivo. Si attiva anche quando gli utenti menzionano la ridenominazione in blocco, la pulizia dei file, l'organizzazione basata sui contenuti, la ristrutturazione delle cartelle, la raccolta di file per argomento o scopo (ad esempio, "이력 관련 자료 모아줘", "raccogliere file di set di dati", "일회성 자료 판단해줘") o il filtraggio e il raggruppamento di file per categoria.
