قم بتحليل محتويات الملف في شجرة الدليل وإعادة تسمية الملفات بذكاء بناءً على ما تحتويه، وإعادة تنظيمها اختياريًا في بنية مجلد منطقي. استخدم هذه المهارة عندما يريد المستخدم تنظيف أسماء الملفات الفوضوية، أو تنظيم الملفات التي تم تنزيلها، أو فرز المستندات حسب المحتوى، أو إعادة تسمية الملفات التي لها أسماء لا معنى لها (مثل IMG_20240101.jpg أو document(3).pdf)، أو ترتيب أي مجلد لا تعكس فيه أسماء الملفات محتواها الفعلي. يتم تشغيله أيضًا عندما يذكر المستخدمون إعادة التسمية المجمعة، أو تنظيف الملفات، أو التنظيم القائم على المحتوى، أو إعادة هيكلة المجلد، أو جمع الملفات حسب الموضوع أو الغرض (على سبيل المثال، "الملفات المجمعة"، أو "تجميع ملفات مجموعة البيانات"، أو "تجميع ملفات البيانات"، أو تصفية الملفات وتجميعها حسب الفئة.
