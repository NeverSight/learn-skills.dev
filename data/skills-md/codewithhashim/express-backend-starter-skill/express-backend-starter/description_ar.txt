يوجه وكلاء الذكاء الاصطناعي لدعم وتطوير الواجهات الخلفية لـ Node.js + Express على مستوى الإنتاج من خلال بنية مجلد نظيفة وقابلة للتطوير وأفضل ممارسات الصناعة. يُستخدم عندما يطلب المستخدم "إنشاء واجهة خلفية"، أو "إعداد واجهة برمجة التطبيقات"، أو "دعم مشروع عقدة"، أو "إنشاء خادم سريع"، أو "إنشاء REST API"، أو "بنية مجلد الواجهة الخلفية"، أو "لوحة API النموذجية"، أو بدء مشروع الواجهة الخلفية Node.js/Express جديد. يغطي JS وTS.
