À utiliser lorsque l'utilisateur demande de « créer un backend », « configurer une API », « échafauder un projet Node », « créer un serveur Express », « créer une API REST », « une structure de dossiers backend », « un passe-partout API » ou démarre un nouveau projet backend Node.js/Express. Couvre JS et TS.
