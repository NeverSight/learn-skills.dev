Directus ヘッドレス CMS を使用してバックエンドと API を構築します。ユーザーが、ヘッドレス CMS の作成、コーディングなしのコンテンツ API の構築、バックエンド管理パネルのセットアップ、データベースからの REST または GraphQL API の作成、ロールと権限によるコンテンツの管理、自動生成された API によるデータ プラットフォームの構築、または従来の CMS のヘッドレス ソリューションへの置き換えを要求する場合に使用します。データ モデリング、自動生成された REST/GraphQL API、ロール/権限、フロー (自動化)、ファイル ストレージ、SDK 統合について説明します。
