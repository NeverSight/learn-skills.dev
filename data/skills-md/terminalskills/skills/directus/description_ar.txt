أنشئ واجهات خلفية وواجهات برمجة التطبيقات باستخدام نظام إدارة المحتوى المباشر Directus. يُستخدم عندما يطلب المستخدم إنشاء نظام إدارة محتوى بدون رأس، أو إنشاء واجهة برمجة تطبيقات للمحتوى بدون تعليمات برمجية، أو إعداد لوحة إدارة خلفية، أو إنشاء واجهات برمجة تطبيقات REST أو GraphQL من قاعدة بيانات، أو إدارة المحتوى باستخدام الأدوار والأذونات، أو إنشاء نظام أساسي للبيانات باستخدام واجهات برمجة التطبيقات التي تم إنشاؤها تلقائيًا، أو استبدال نظام إدارة المحتوى التقليدي بحل بدون رأس. يغطي نمذجة البيانات، وواجهات برمجة تطبيقات REST/GraphQL التي يتم إنشاؤها تلقائيًا، والأدوار/الأذونات، والتدفقات (الأتمتة)، وتخزين الملفات، وتكامل SDK.
