Créez des backends et des API avec le CMS sans tête Directus. À utiliser lorsqu'un utilisateur demande de créer un CMS sans tête, de créer une API de contenu sans codage, de configurer un panneau d'administration back-end, de créer des API REST ou GraphQL à partir d'une base de données, de gérer le contenu avec des rôles et des autorisations, de créer une plate-forme de données avec des API générées automatiquement ou de remplacer le CMS traditionnel par une solution sans tête. Couvre la modélisation des données, les API REST/GraphQL générées automatiquement, les rôles/autorisations, les flux (automatisation), le stockage de fichiers et l'intégration du SDK.
