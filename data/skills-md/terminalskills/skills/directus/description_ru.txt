Создавайте серверные части и API с помощью headless CMS Directus. Используйте, когда пользователь просит создать автономную CMS, создать контентный API без кодирования, настроить внутреннюю панель администратора, создать REST или GraphQL API из базы данных, управлять контентом с помощью ролей и разрешений, создать платформу данных с автоматически генерируемыми API или заменить традиционную CMS автономным решением. Охватывает моделирование данных, автоматически создаваемые API-интерфейсы REST/GraphQL, роли/разрешения, потоки (автоматизация), хранилище файлов и интеграцию SDK.
