Помогает создавать сквозные типобезопасные API с использованием tRPC без генерации кода. Используйте при создании типобезопасных мостов клиент-сервер, определении процедур с проверкой Zod, интеграции с Next.js или React Query или настройке подписок WebSocket. Триггерные слова: trpc, типобезопасный API, процедуры, ответный запрос, сквозные типы.
