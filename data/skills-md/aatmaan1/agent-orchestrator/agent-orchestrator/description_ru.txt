Навыки метаагента для организации сложных задач с помощью автономных субагентов. Разбивает макрозадачи на подзадачи, создает специализированные субагенты с динамически генерируемыми файлами SKILL.md, координирует обмен данными на основе файлов, консолидирует результаты и распускает агентов после завершения.

ОБЯЗАТЕЛЬНЫЕ ТРИГГЕРЫ: оркестрация, мультиагентность, декомпозиция задачи, создание агентов, субагенты, параллельные агенты, координация агентов, разбивка задач, метаагент, фабрика агентов, делегирование задач.
