Automatisiert den Git-Commit- und Push-Workflow mit beschreibenden Commit-Nachrichten. Analysiert Änderungen, generiert herkömmliche Commit-Nachrichten, stellt Dateien bereit, führt Commits durch und überträgt sie an die Remote-Adresse. Verwenden Sie diese Option, wenn der Benutzer Änderungen festschreiben, Code übertragen oder Arbeit in Git speichern möchte.
