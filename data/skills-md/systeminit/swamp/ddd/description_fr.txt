Conseils de conception pilotée par domaine pour les bases de code TypeScript/Deno. Appliquer lors de l'écriture ou de la modification de code pour choisir les éléments constitutifs DDD appropriés (entités, objets de valeur, agrégats, services de domaine, référentiels) et maintenir un langage omniprésent. Se déclenche sur toutes les modifications de code pour garantir la cohérence du modèle de domaine.
