Guía de diseño controlado por dominio para bases de código TypeScript/Deno. Aplicar al escribir o modificar código para elegir los bloques de construcción DDD apropiados (entidades, objetos de valor, agregados, servicios de dominio, repositorios) y mantener un lenguaje ubicuo. Activa todos los cambios de código para garantizar la coherencia del modelo de dominio.
