Orquestador Get Shit Done (GSD). Ejecuta todo el proceso del proyecto desde la idea hasta la implementación: configuración → PRD → lista de tareas → implementación. Úselo cuando el usuario quiera construir algo de un extremo a otro, diga "vamos a GSD", "construyamos esto desde cero", "hagamos algo hecho" o quiera ejecutar el flujo de trabajo de desarrollo completo. Coordina las habilidades gsdl-setup-project, gsdl-create-prd, gsdl-create-plan y gsdl-execute-plan. Genera subagentes por tarea principal durante la implementación para preservar el contexto. Acepta un nombre de proyecto opcional o una URL de origen: /gsdl [nombre-proyecto] O /gsdl [linear|notion|slite] [url] O /gsdl [url].
