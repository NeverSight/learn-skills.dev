Get Shit Done (GSD)-Orchestrator. Führt die gesamte Projektpipeline von der Idee bis zur Implementierung durch: Setup → PRD → Aufgabenliste → Implementierung. Verwenden Sie diese Option, wenn der Benutzer etwas End-to-End erstellen möchte, sagt „Lass uns GSD machen“, „das von Grund auf neu erstellen“, „Mach alles fertig“ oder den gesamten Entwicklungsworkflow ausführen möchte. Koordiniert die Fähigkeiten „gsdl-setup-project“, „gsdl-create-prd“, „gsdl-create-plan“ und „gsdl-execute-plan“. Erzeugt während der Implementierung Subagenten pro übergeordneter Aufgabe, um den Kontext beizubehalten. Akzeptiert einen optionalen Projektnamen oder eine Quell-URL: /gsdl [Projektname] ODER /gsdl [linear|notion|slite] [URL] ODER /gsdl [URL].
