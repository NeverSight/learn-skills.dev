Orchestratore Get Shit Done (GSD). Esegue l'intera pipeline del progetto dall'idea all'implementazione: configurazione → PRD → elenco attività → implementazione. Da utilizzare quando l'utente desidera creare qualcosa end-to-end, dice "facciamo GSD", "costruiscilo da zero", "fai un lavoro" o desidera eseguire il flusso di lavoro di sviluppo completo. Coordina le competenze gsdl-setup-project, gsdl-create-prd, gsdl-create-plan e gsdl-execute-plan. Genera agenti secondari per attività principale durante l'implementazione per preservare il contesto. Accetta un nome di progetto o un URL di origine facoltativo: /gsdl [nome-progetto] O /gsdl [lineare|nozione|slite] [url] O /gsdl [url].
