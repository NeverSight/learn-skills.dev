Администрируйте системы Windows 11 с помощью PowerShell 7.x. Включает перевод команд из bash/Linux, использование менеджера пакетов (winget, scoop, npm, Chocolatey), настройку PATH и управление средой для настройки нескольких устройств.

Используйте при настройке среды администрирования Windows, написании сценариев автоматизации PowerShell, трансляции команд bash в PowerShell, настройке переменных PATH и среды или устранении ошибок «команда не найдена», «Get-Content не распознан», «winget не работает».
