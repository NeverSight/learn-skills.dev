معالج تحسين تكوين OpenClaw. يستخدم سير عمل "ملاحظات اختبار المستوى 0 (اختياري) + أول 3 جولات تحسين + إعداد ملحق الجولة الرابعة": طبقة التوصية الأساسية (مع بحث الويب وأوضاع الأذونات)، وطبقة تحسين القناة (تحسين فرع Discord/Feishu/Telegram)، وطبقة توصية المهارات (قابلة للتنفيذ وقابلة للتخطي)، ودليل إعداد القناة الجديد (قابل للتخطي). يساعد المستخدمين على إكمال تهيئة OpenClaw بسرعة أو تحسين التكوين. يتم تشغيله عندما يقول المستخدمون "تحسين openclaw"، أو "معالج التكوين"، أو "تهيئة openclaw"، أو "التجربة سيئة للغاية، أو يريدون تنظيف التكوين بنقرة واحدة"، وما إلى ذلك.
