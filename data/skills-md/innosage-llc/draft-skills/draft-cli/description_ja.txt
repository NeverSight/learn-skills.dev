@innosage/draft-cli を使用して、「ドラフト」ページとドキュメントを管理および操作します。ユーザーが「ドラフト ページ」、「ドラフト ドキュメント」、または「ドラフト ワークスペース」(例: 「'Founder Sync' という名前のドラフト ページ」、「ドラフト ワークスペースにあるすべてのページ」、「ドラフト CLI」など) のコンテンツの読み取り、作成、一覧表示、パッチ、または追加を明示的に要求する場合には、このスキルを使用します。これは、ローカル デーモンを介してドラフト PWA (draft.innosage.co) に接続し、生きているドキュメントを読み取りまたは変更します。 「ドラフト」が動詞として使用される一般的な執筆タスク (例: 「電子メールの下書き」、「回答の下書き」) や、名前に「ドラフト」が含まれるローカル マークダウン/テキスト ファイル (例: 「draft.md」、「investor_update_draft.md」) を参照する場合には、このスキルを使用しないでください。実際の InnoSage Draft Web アプリケーションまたは Draft CLI ツールと対話する場合にのみ使用してください。トリガーされた場合は、常に「最初に接続」の動作パターンに従います。つまり、他のコマンドの前にステータスを確認し、接続されていない場合はデーモンを開始します。
