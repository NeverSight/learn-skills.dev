Administre e interactúe con páginas y documentos "Borrador" utilizando @innosage/draft-cli. Utilice esta habilidad siempre que el usuario solicite explícitamente leer, crear, enumerar, parchear o agregar contenido a una "página borrador", "documento borrador" o su "espacio de trabajo borrador" (por ejemplo, "mi página borrador llamada 'Founder Sync'", "todas las páginas que tengo en mi espacio de trabajo borrador", "CLI borrador"). Esto se conecta al Draft PWA (draft.innosage.co) a través de un demonio local para leer o modificar documentos vivos. NO utilice esta habilidad para tareas de escritura generalizadas en las que se utilice "borrador" como verbo (por ejemplo, "redactar un correo electrónico", "redactar una respuesta") o cuando se refiera a archivos de texto o rebajas locales con "borrador" en el nombre (por ejemplo, "draft.md", "investor_update_draft.md"). Utilícelo únicamente cuando interactúe con la aplicación web InnoSage Draft o la herramienta Draft CLI real. Cuando se active, SIEMPRE siga el patrón operativo "Conexión primero": verifique el estado antes de cualquier otro comando e inicie el demonio si no está conectado.
