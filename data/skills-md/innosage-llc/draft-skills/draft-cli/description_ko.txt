@innosage/draft-cli를 사용하여 "초안" 페이지와 문서를 관리하고 상호 작용하세요. 사용자가 "초안 페이지", "초안 문서" 또는 "초안 작업 영역"(예: "'Founder Sync'라는 이름의 초안 페이지", "초안 작업 영역에 있는 모든 페이지", "CLI 초안")에 콘텐츠 읽기, 만들기, 나열, 패치 또는 추가를 명시적으로 요청할 때마다 이 기술을 사용하세요. 이는 로컬 데몬을 통해 Draft PWA(draft.innosage.co)에 연결하여 살아있는 문서를 읽거나 수정합니다. "draft"가 동사로 사용되는 일반 쓰기 작업(예: "draft an email", "draft a response")이나 이름에 "draft"가 포함된 로컬 마크다운/텍스트 파일을 참조할 때(예: "draft.md", "investor_update_draft.md") 이 기술을 사용하지 마십시오. 실제 InnoSage Draft 웹 애플리케이션 또는 Draft CLI 도구와 상호 작용할 때만 사용하십시오. 트리거되면 항상 "Connection First" 작동 패턴을 따릅니다. 즉, 다른 명령을 실행하기 전에 상태를 확인하고 연결되지 않은 경우 데몬을 시작합니다.
