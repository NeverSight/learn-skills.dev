عندما يريد المستخدم إنشاء مغناطيس رئيسي أو تخطيطه أو تحسينه لالتقاط البريد الإلكتروني أو إنشاء عملاء محتملين. استخدمه أيضًا عندما يذكر المستخدم "مغناطيس رئيسي"، أو "محتوى مسور"، أو "ترقية المحتوى"، أو "قابل للتنزيل"، أو "كتاب إلكتروني"، أو "ورقة الغش"، أو "قائمة المراجعة"، أو "تنزيل القالب"، أو "الاشتراك"، أو "الهدية الترويجية"، أو "تنزيل PDF"، أو "مكتبة الموارد"، أو "عرض المحتوى"، أو "محتوى التقاط البريد الإلكتروني"، أو "قالب الفكرة"، أو "قالب جدول البيانات"، أو "ما الذي يجب أن أعطيه لرسائل البريد الإلكتروني". استخدم هذا لتخطيط ما سيتم إنشاؤه وكيفية توزيعه. للحصول على أدوات تفاعلية كمغناطيس رئيسي، راجع استراتيجية الأداة المجانية. لكتابة المحتوى الفعلي، راجع كتابة الإعلانات. لمعرفة تسلسل البريد الإلكتروني بعد الالتقاط، راجع تسلسل البريد الإلكتروني.
