قم بتحليل نشاط مساهمة التعليمات البرمجية لمستخدم GitHub في نافذة زمنية محددة بالتوقيت العالمي المنسق (UTC) وقم بإنشاء تقرير تخفيض السعر الذي يركز على المساهمات العملية (العلاقات العامة المدمجة المؤلفة، والعلاقات العامة المدمجة من قبل المستخدم المستهدف، والالتزامات المؤلفة) عبر المستودعات العامة والخاصة التي يمكن الوصول إليها. يُستخدم عندما يطلب المستخدمون تقييم مساهمة هندسية حقيقية خلال فترة معينة ويريدون روابط الأدلة.
