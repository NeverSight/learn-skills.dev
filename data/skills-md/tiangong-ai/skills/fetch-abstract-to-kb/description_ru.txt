Извлекайте кандидатов DOI из «журналов» PostgreSQL и пакетно записывайте «абстракты» по DOI. Используйте, когда вам нужен рабочий процесс обратной засыпки только для БД: (1) выберите самые новые строки, где doi не пуст, автор не пуст и абстракт пуст (ограничение по умолчанию 10), затем (2) запишите подготовленные абстрактные значения обратно в пакете по DOI.
