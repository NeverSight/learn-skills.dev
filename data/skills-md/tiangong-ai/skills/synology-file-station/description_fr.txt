Utilisez Synology DSM File Station via WebAPI pour les principaux flux de travail de fichiers, notamment la liste, la recherche, la création de dossiers, le renommage, la copie/déplacement, la suppression, le téléchargement/téléchargement et l'extraction d'archives. À utiliser lorsque les tâches nécessitent des opérations de fichier NAS scriptées avec l'adresse de service, le nom d'utilisateur et le mot de passe chargés à partir de variables d'environnement. Remarque : la compression est temporairement indisponible dans cette compétence.
