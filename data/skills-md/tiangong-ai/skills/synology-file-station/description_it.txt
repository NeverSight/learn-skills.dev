Gestisci Synology DSM File Station tramite WebAPI per i principali flussi di lavoro dei file tra cui elenco, ricerca, creazione di cartelle, ridenominazione, copia/sposta, eliminazione, caricamento/download ed estrazione di archivi. Da utilizzare quando le attività richiedono operazioni su file NAS con script con indirizzo del servizio, nome utente e password caricati da variabili di ambiente. Nota: la compressione non è temporaneamente disponibile in questa abilità.
