Betreiben Sie die Synology DSM File Station über WebAPI für wichtige Datei-Workflows, einschließlich Auflisten, Suchen, Erstellen von Ordnern, Umbenennen, Kopieren/Verschieben, Löschen, Hochladen/Herunterladen und Archivieren. Verwenden Sie diese Option, wenn für Aufgaben Skript-NAS-Dateivorgänge erforderlich sind, bei denen Dienstadresse, Benutzername und Kennwort aus Umgebungsvariablen geladen werden. Hinweis: „Komprimieren“ ist in diesem Skill vorübergehend nicht verfügbar.
