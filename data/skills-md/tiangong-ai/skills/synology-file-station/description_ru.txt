Используйте Synology DSM File Station через WebAPI для основных рабочих процессов с файлами, включая составление списка, поиск, создание папок, переименование, копирование/перемещение, удаление, загрузку/загрузку и извлечение архива. Используйте, когда задачам требуются операции с файлами NAS по сценарию с адресом службы, именем пользователя и паролем, загруженными из переменных среды. Примечание: сжатие в этом умении временно недоступно.
