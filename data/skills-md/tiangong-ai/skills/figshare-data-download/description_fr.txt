Téléchargez les fichiers Figshare via un flux de travail uniquement sur navigateur (pas de sondage web_fetch/curl). À utiliser lorsqu'un utilisateur fournit un DOI Figshare ou une URL de téléchargement et a besoin d'un chemin de récupération fiable qui ouvre toujours la page dans le navigateur, déclenche le téléchargement réel, puis copie le fichier téléchargé dans un chemin cible.
