Scarica i file Figshare tramite un flusso di lavoro solo tramite browser (nessun sondaggio web_fetch/curl). Da utilizzare quando un utente fornisce un DOI Figshare o un URL ndownloader e necessita di un percorso di recupero affidabile che apra sempre la pagina nel browser, attivi il download reale, quindi copi il file scaricato in un percorso di destinazione.
