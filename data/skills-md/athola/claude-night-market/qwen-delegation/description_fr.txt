Workflow de délégation Qwen CLI implémentant le noyau de délégation pour les modèles Qwen d'Alibaba. À utiliser lorsque le noyau de délégation a sélectionné Qwen, nécessite les grandes capacités contextuelles de Qwen, traitement par lots requis. Ne pas utiliser pour décider quel modèle utiliser (utilisez d'abord le noyau de délégation), qwen CLI n'est pas installé.
