Профессиональный инструмент для создания индивидуальных контрольных списков текущих функций на основе требований пользователя. Специально используется для проверки качества требований, генерируя «английские модульные тесты» для проверки полноты, ясности и последовательности требований. Триггерные слова: контрольный список спецификаций, контрольный список, проверка требований, проверка качества, контрольный список, проверка требований, проверка качества, проверка спецификации.
