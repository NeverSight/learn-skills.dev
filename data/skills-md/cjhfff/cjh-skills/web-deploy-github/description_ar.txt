قم بإنشاء مواقع ويب ثابتة ذات صفحة واحدة ونشرها على صفحات GitHub باستخدام سير عمل مستقل. يُستخدم عند إنشاء مواقع المحفظة أو صفحات السيرة الذاتية أو الصفحات المقصودة أو أي مشروع ويب ثابت يحتاج إلى نشر صفحات GitHub. يتعامل مع سير العمل الكامل بدءًا من تهيئة المشروع وحتى النشر المباشر باستخدام أتمتة GitHub Actions.
