Создавайте и развертывайте одностраничные статические веб-сайты на страницах GitHub с помощью автономного рабочего процесса. Используйте при создании сайтов-портфолио, страниц резюме, целевых страниц или любого статического веб-проекта, требующего развертывания GitHub Pages. Управляет полным рабочим процессом от инициализации проекта до живого развертывания с помощью автоматизации GitHub Actions.
