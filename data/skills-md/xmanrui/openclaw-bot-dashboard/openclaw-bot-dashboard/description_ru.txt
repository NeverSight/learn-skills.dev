Запускайте веб-интерфейс OpenClaw Dashboard и управляйте им для мониторинга всех ваших ботов, агентов, моделей и сеансов. Автоматически определяет ОС, проверяет наличие обновлений, устанавливает зависимости и запускает сервер разработки. Поддерживает macOS, Windows и Linux с автоматическим обнаружением git и методами резервной загрузки. Триггеры включают в себя «打开机器人大盘», «打开 OpenClaw-bot-review», «打开Openclaw大盘», «打开 обзор бота», «открыть панель управления openclaw», «открыть обзор OpenClaw-bot», «открыть панель управления ботом», «открыть обзор бота», «запустить обзор бота».
