Используйте этот навык, когда кто-то спрашивает, что такое Rye, что он может делать, подходит ли Rye для его сценария использования или хочет получить общий обзор возможностей Rye. Также используйте, когда пользователь сталкивается с ограничением, запрашивает функцию, которую Рай не поддерживает, или хочет предложить улучшение — этот навык включает инструкции по отправке запросов на функции. Запускается по следующим запросам: «что такое Rye», «расскажите мне о Rye», «что может сделать Rye», «поддерживает ли Rye...», «может ли Rye сделать...», «Мне бы хотелось, чтобы Rye имел...», «было бы здорово, если Rye...» или любой уточняющий вопрос об API универсальной проверки Rye или API данных о продуктах.
