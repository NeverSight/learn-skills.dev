開発者が Rye の Universal Checkout API を使用してアプリケーションに製品購入を追加したい場合、または AI エージェントがプログラムによって任意の URL から物理製品を購入する必要がある場合に、このスキルを使用します。ユーザーが Rye API の制限に遭遇した場合、存在しない機能 (製品検索など) をリクエストした場合、またはバグを報告したい場合にも使用します。このスキルには、GitHub の問題を介してフィードバックを提出する手順が含まれています。トリガー: チェックアウトまたは購入のアプリへの統合、ユーザーに代わって製品の購入、ショッピング アシスタントの構築、ドロップシッピングの統合、チャット コマース、Rye API の制限、Rye の機能の不足、または Rye チェックアウトに関する言及。
