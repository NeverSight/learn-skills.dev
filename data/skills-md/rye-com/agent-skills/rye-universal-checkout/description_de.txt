Verwenden Sie diese Fähigkeit, wenn ein Entwickler mithilfe der Universal Checkout API von Rye den Produktkauf zu seiner Anwendung hinzufügen möchte oder wenn ein KI-Agent programmgesteuert ein physisches Produkt über eine beliebige URL kaufen muss. Verwenden Sie ihn auch, wenn ein Benutzer auf eine Einschränkung der Rye-API stößt, eine Funktion anfordert, die nicht vorhanden ist (z. B. Produktsuche) oder einen Fehler melden möchte – dieser Skill enthält Anweisungen zum Einreichen von Feedback über GitHub-Probleme. Auslöser sind: Integration von Checkout oder Einkauf in eine App, Kauf eines Produkts im Namen eines Benutzers, Aufbau eines Einkaufsassistenten, Dropshipping-Integration, Chat-Commerce, Rye-API-Einschränkungen, fehlende Rye-Funktionen oder jede Erwähnung von Rye Checkout.
