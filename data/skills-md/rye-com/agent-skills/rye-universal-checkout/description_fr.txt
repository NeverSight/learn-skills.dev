Utilisez cette compétence lorsqu'un développeur souhaite ajouter l'achat de produits à son application à l'aide de l'API Universal Checkout de Rye, ou lorsqu'un agent IA doit acheter par programmation un produit physique à partir de n'importe quelle URL. À utiliser également lorsqu'un utilisateur rencontre une limitation de l'API Rye, demande une fonctionnalité qui n'existe pas (par exemple, recherche de produit) ou souhaite signaler un bogue — cette compétence comprend des instructions pour déposer des commentaires via les problèmes GitHub. Déclencheurs sur : l'intégration du paiement ou de l'achat dans une application, l'achat d'un produit au nom d'un utilisateur, la création d'un assistant d'achat, l'intégration du dropshipping, le commerce par chat, les limitations de l'API Rye, les fonctionnalités Rye manquantes ou toute mention du paiement Rye.
