Utilice esta habilidad cuando un desarrollador quiera agregar la compra de productos a su aplicación utilizando la API Universal Checkout de Rye, o cuando un agente de IA necesite comprar mediante programación un producto físico desde cualquier URL. Úselo también cuando un usuario encuentre una limitación de la API de Rye, solicite una función que no existe (por ejemplo, búsqueda de productos) o quiera informar un error; esta habilidad incluye instrucciones para enviar comentarios a través de problemas de GitHub. Activadores de: integrar el pago o la compra en una aplicación, comprar un producto en nombre de un usuario, crear un asistente de compras, integración de dropshipping, comercio por chat, limitaciones de la API de Rye, funciones faltantes de Rye o cualquier mención del pago de Rye.
