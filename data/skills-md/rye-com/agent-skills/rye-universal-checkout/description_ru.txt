Используйте этот навык, когда разработчик хочет добавить покупку продуктов в свое приложение с помощью API универсальной проверки Rye или когда агенту ИИ необходимо программно купить физический продукт по любому URL-адресу. Также используйте, когда пользователь сталкивается с ограничением Rye API, запрашивает несуществующую функцию (например, поиск продукта) или хочет сообщить об ошибке — этот навык включает инструкции по отправке отзывов через проблемы GitHub. Срабатывает при: интеграции оформления заказа или покупки в приложение, покупке продукта от имени пользователя, создании помощника по покупкам, интеграции прямой поставки, торговле в чате, ограничениях API Rye, отсутствующих функциях Rye или любом упоминании о проверке Rye.
