استخدم هذه المهارة عندما يريد أحد المطورين إضافة شراء منتج إلى تطبيقه باستخدام واجهة برمجة التطبيقات Universal Checkout API الخاصة بـ Rye، أو عندما يحتاج وكيل الذكاء الاصطناعي إلى شراء منتج مادي برمجيًا من أي عنوان URL. تُستخدم أيضًا عندما يواجه المستخدم قيودًا في واجهة برمجة تطبيقات Rye، أو يطلب ميزة غير موجودة (على سبيل المثال، البحث عن منتج)، أو يريد الإبلاغ عن خطأ - تتضمن هذه المهارة إرشادات لتقديم التعليقات عبر مشكلات GitHub. المشغلات عند: دمج الدفع أو الشراء في تطبيق، أو شراء منتج نيابة عن مستخدم، أو إنشاء مساعد تسوق، أو تكامل دروبشيبينغ، أو تجارة الدردشة، أو قيود Rye API، أو ميزات Rye المفقودة، أو أي ذكر لـ Rye checkout.
