Blender-Automatisierung mit Geometrieerstellung, Materialien, Modifikatoren und Mixamo-Animations-Retargeting.

Kernfunktionen: WebSocket-basierte Echtzeitsteuerung, automatische Knochenkartierung mit UI-Überprüfung, zweiphasiger Bestätigungsworkflow, Qualitätsbewertung, Unterstützung mehrerer Projekte, umfassende CLI-Befehle.

Anwendungsfälle: Erstellen Sie 3D-Primitive (Würfel, Kugel, Zylinder usw.), manipulieren Sie Objekte (transformieren, duplizieren, löschen), verwalten Sie Materialien und Modifikatoren, richten Sie Mixamo-Animationen auf benutzerdefinierte Rigs mit Fuzzy-Bone-Matching um.
