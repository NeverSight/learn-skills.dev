Автоматизация Blender с созданием геометрии, материалов, модификаторов и перенацеливанием анимации Mixamo.

Основные функции: управление в реальном времени на основе WebSocket, автоматическое картирование костей с просмотром пользовательского интерфейса, двухэтапный рабочий процесс подтверждения, оценка качества, поддержка нескольких проектов, комплексные команды CLI.

Варианты использования: создавайте 3D-примитивы (куб, сфера, цилиндр и т. д.), манипулируйте объектами (трансформируйте, дублируйте, удаляйте), управляйте материалами и модификаторами, перенацеливайте анимацию Mixamo на пользовательские установки с нечетким сопоставлением костей.
