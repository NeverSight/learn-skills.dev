أتمتة الخلاط مع إنشاء الأشكال الهندسية والمواد والمعدلات وإعادة استهداف الرسوم المتحركة لـ Mixamo.

الميزات الأساسية: التحكم في الوقت الفعلي المستند إلى WebSocket، ورسم خرائط العظام تلقائيًا مع مراجعة واجهة المستخدم، وسير عمل التأكيد على مرحلتين، وتقييم الجودة، ودعم المشاريع المتعددة، وأوامر CLI الشاملة.

حالات الاستخدام: إنشاء عناصر أولية ثلاثية الأبعاد (مكعب، كرة، أسطوانة، وما إلى ذلك)، ومعالجة الكائنات (التحويل، والتكرار، والحذف)، وإدارة المواد والمعدلات، وإعادة توجيه الرسوم المتحركة لـ Mixamo إلى منصات مخصصة باستخدام مطابقة العظام الغامضة.
