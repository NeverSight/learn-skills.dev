Unity エディターの制御と自動化、WebSocket ベースのリアルタイム通信。

機能: ゲームオブジェクト制御、変換操作、コンポーネント管理、シーン管理、SQLite データベース統合、GUID ベースの永続性、マルチシーン同期、元に戻す/やり直しを伴うコマンド パターン、メニュー実行、ScriptableObject 管理、配列/リスト操作、すべてのフィールド タイプのサポート、マテリアル/レンダリング、プレハブ システム、アセット データベース、アニメーション、物理学、コンソール ロギング、EditorPrefs 管理、エディター オートメーション、ビルド パイプライン、照明、カメラ、オーディオ、ナビゲーション、パーティクル、タイムライン、UI ツールキット、プロファイラー、テスト ランナー。

プロトコル: WebSocket 経由の JSON-RPC 2.0 (ポート 9500 ～ 9600)。 500 以上のコマンド、25 のカテゴリ。リアルタイムの双方向通信。

セキュリティ: 多層防御 (パストラバーサル保護、コマンドインジェクション防御、JSON インジェクション防止、SQL インジェクション防止、トランザクションの安全性)。ローカルホストのみの接続。クロスプラットフォーム (Windows、macOS、Linux)。
