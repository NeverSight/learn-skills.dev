Управление и автоматизация Unity Editor, общение в реальном времени на основе WebSocket.

Функции: Управление игровыми объектами, Манипулирование преобразованиями, Управление сценой, Интеграция базы данных SQLite, Сохранение на основе GUID, Синхронизация нескольких сцен, Шаблон команды с отменой/повтором действия, Выполнение меню, Управление ScriptableObject, Манипулирование массивами/списками, Поддержка всех типов полей, Материал/рендеринг, Сборная система, База данных активов, Анимация, Физика, Ведение журнала консоли, Управление EditorPrefs, Автоматизация редактора, Конвейер сборки, Освещение, Камера, Аудио, Навигация, Частицы, Временная шкала, набор инструментов пользовательского интерфейса, профилировщик, средство выполнения тестов.

Протокол: JSON-RPC 2.0 через WebSocket (порт 9500–9600). 500+ команд, 25 категорий. Двусторонняя связь в режиме реального времени.

Безопасность: глубокая защита (защита от обхода пути, защита от внедрения команд, предотвращение внедрения JSON, предотвращение внедрения SQL, безопасность транзакций). Соединения только для локального хоста. Кроссплатформенность (Windows, macOS, Linux).
