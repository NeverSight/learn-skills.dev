Unity Editor 제어 및 자동화, WebSocket 기반 실시간 통신.

기능: 게임 객체 제어, 변환 조작, 구성 요소 관리, 장면 관리, SQLite 데이터베이스 통합, GUID 기반 지속성, 다중 장면 동기화, 실행 취소/다시 실행을 통한 명령 패턴, 메뉴 실행, 스크립터블 객체 관리, 배열/목록 조작, 모든 필드 유형 지원, 재질/렌더링, 프리팹 시스템, 자산 데이터베이스, 애니메이션, 물리, 콘솔 로깅, EditorPrefs 관리, 편집기 자동화, 빌드 파이프라인, 조명, 카메라, 오디오, 탐색, 입자, 타임라인, UI 툴킷, 프로파일러, 테스트 러너.

프로토콜: WebSocket(포트 9500-9600)을 통한 JSON-RPC 2.0. 500개 이상의 명령, 25개 카테고리. 실시간 양방향 통신.

보안: 심층 방어(경로 통과 보호, 명령 주입 방어, JSON 주입 방지, SQL 주입 방지, 트랜잭션 안전). 로컬호스트 전용 연결. 크로스 플랫폼(Windows, macOS, Linux).
