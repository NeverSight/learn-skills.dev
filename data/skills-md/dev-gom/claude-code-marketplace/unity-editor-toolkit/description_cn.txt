Unity编辑器控制和自动化，基于WebSocket的实时通信。

功能：游戏对象控制、变换操作、组件管理、场景管理、SQLite 数据库集成、基于 GUID 的持久性、多场景同步、带撤消/重做的命令模式、菜单执行、ScriptableObject 管理、数组/列表操作、所有字段类型支持、材质/渲染、预制系统、资产数据库、动画、物理、控制台日志记录、EditorPrefs 管理、编辑器自动化、构建管道、灯光、相机、音频、导航、粒子、时间轴、UI 工具包、分析器、测试运行器。

协议：基于 WebSocket 的 JSON-RPC 2.0（端口 9500-9600）。 500 多个命令，25 个类别。实时双向通信。

安全：深度防御（路径遍历防护、命令注入防护、JSON注入防护、SQL注入防护、事务安全）。仅本地主机连接。跨平台（Windows、macOS、Linux）。
