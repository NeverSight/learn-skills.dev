Führen Sie Unity Test Framework-Tests über die Befehlszeile aus und analysieren Sie sie. Dieser Skill automatisiert die Testausführung für Unity-Projekte, indem er den Unity-Editor erkennt, Testparameter konfiguriert (EditMode/PlayMode), Tests über CLI ausführt, XML-Ergebnisse analysiert und detaillierte Fehlerberichte generiert. Verwenden Sie dies, wenn Sie Unity-Tests ausführen, die Spiellogik validieren oder Testfehler debuggen.
