تنفيذ وتحليل اختبارات Unity Test Framework من سطر الأوامر. تعمل هذه المهارة على أتمتة تنفيذ الاختبار لمشاريع Unity من خلال الكشف عن Unity Editor، وتكوين معلمات الاختبار (EditMode/PlayMode)، وتشغيل الاختبارات عبر CLI، وتحليل نتائج XML، وإنشاء تقارير فشل مفصلة. استخدم هذا عند تشغيل اختبارات Unity، أو التحقق من صحة منطق اللعبة، أو تصحيح أخطاء الاختبار.
