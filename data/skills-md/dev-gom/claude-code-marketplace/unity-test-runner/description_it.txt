Esegui e analizza i test Unity Test Framework dalla riga di comando. Questa competenza automatizza l'esecuzione dei test per i progetti Unity rilevando l'editor Unity, configurando i parametri di test (EditMode/PlayMode), eseguendo test tramite CLI, analizzando i risultati XML e generando report dettagliati sugli errori. Utilizzalo durante l'esecuzione di test Unity, la convalida della logica di gioco o il debug degli errori dei test.
