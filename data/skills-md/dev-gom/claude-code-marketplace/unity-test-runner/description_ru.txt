Выполняйте и анализируйте тесты Unity Test Framework из командной строки. Этот навык автоматизирует выполнение тестов для проектов Unity путем обнаружения редактора Unity, настройки параметров теста (EditMode/PlayMode), запуска тестов через CLI, анализа результатов XML и создания подробных отчетов об ошибках. Используйте это при запуске тестов Unity, проверке игровой логики или отладке сбоев тестов.
