Ejecute y analice pruebas de Unity Test Framework desde la línea de comandos. Esta habilidad automatiza la ejecución de pruebas para proyectos de Unity detectando el Editor de Unity, configurando parámetros de prueba (EditMode/PlayMode), ejecutando pruebas a través de CLI, analizando resultados XML y generando informes de fallas detallados. Úselo cuando ejecute pruebas de Unity, valide la lógica del juego o depure fallas de pruebas.
