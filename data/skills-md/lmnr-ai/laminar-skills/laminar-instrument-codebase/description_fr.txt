Instrumentez une base de code existante avec le traçage Laminar : choisissez les fonctions à observer, initialisez Laminar correctement, ajoutez des balises/métadonnées/ID de session et vérifiez les traces. À utiliser lorsqu'un utilisateur demande d'ajouter le traçage Laminar, des fonctions d'instrument ou d'intégrer Laminar à une base de code TS/JS ou Python.
