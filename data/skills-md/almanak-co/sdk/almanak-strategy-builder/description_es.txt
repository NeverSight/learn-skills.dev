Cree, pruebe e implemente estrategias comerciales de DeFi utilizando el SDK de Almanak. Utilice SIEMPRE esta habilidad cuando el usuario mencione almanaque, estrategia DeFi, estrategia comercial, cultivo de rendimiento, provisión de liquidez, intercambio de tokens, endeudamiento, préstamos, perpetuas, apuestas, depósito en bóveda, tokens puente, backtesting, comercio en papel o ejecución en cadena. Úselo para escribir archivos Strategy.py, componer intents (Swap, LP, Borrow, Supply, Perp, Bridge, Stake, Vault, Prediction), trabajar con parámetros de estrategia config.json, ejecutar comandos CLI de almanak strat o almanak gateway, o depurar la ejecución de estrategias en bifurcaciones de Anvil. NO lo utilice para el desarrollo general de contratos inteligentes, código Solidity o componentes internos del SDK que no sean estratégicos.
