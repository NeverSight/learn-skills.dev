Используйте, когда вам нужно применить передовые методы функциональной обработки исключений в Java, включая замену чрезмерного использования исключений типами «Необязательный» и «VAVR Any», разработку иерархий типов ошибок с использованием запечатанных классов и перечислений, реализацию монадических конвейеров композиции ошибок, установление шаблонов потока функционального управления и резервирование исключений только для действительно исключительных сбоев на уровне системы.
