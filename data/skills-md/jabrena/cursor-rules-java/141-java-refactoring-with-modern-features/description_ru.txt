Используйте, когда вам нужно реорганизовать код Java для внедрения современных функций Java (Java 8+), включая миграцию анонимных классов в лямбда-выражения, замену циклов Iterator на Stream API, принятие опционального для нулевой безопасности, переключение с устаревшего Date/Calendar на java.time, использование методов фабрики коллекций, переход на CompletableFuture для асинхронных операций, применение текстовых блоков, вывод переменных или использование функций Java 25, таких как гибкие тела конструкторов и объявления импорта модулей.
