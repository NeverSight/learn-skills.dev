Используйте, когда вам нужно добавить или настроить плагины Maven в вашем pom.xml — включая инструменты качества (enforcer, Surefire, Failsafe, jacoco,pitest, Spotbugs, pmd), сканирование безопасности (OWASP), форматирование кода (Spotless), управление версиями, отслеживание информации о сборке и сравнительное тестирование (JMH) — с помощью консультативного модульного пошагового подхода, который добавляет только то, что вам действительно нужно.
