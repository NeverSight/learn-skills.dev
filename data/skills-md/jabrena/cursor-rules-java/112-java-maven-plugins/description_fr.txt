À utiliser lorsque vous devez ajouter ou configurer des plugins Maven dans votre pom.xml — y compris des outils de qualité (enforcer, surefire, failsafe, jacoco, pitest, spotbugs, pmd), analyse de sécurité (OWASP), formatage de code (Spotless), gestion des versions, suivi des informations de construction et analyse comparative (JMH) — via une approche consultative et modulaire étape par étape qui n'ajoute que ce dont vous avez réellement besoin. Une partie du projet skills-for-java
