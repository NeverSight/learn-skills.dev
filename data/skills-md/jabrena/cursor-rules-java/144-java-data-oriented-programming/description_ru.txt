Используйте, когда вам нужно применить лучшие практики ориентированного на данные программирования на Java, включая отделение кода (поведения) от структур данных с помощью записей, разработку неизменяемых данных с помощью чистых функций преобразования, сохранение данных плоскими и денормализованными с помощью ссылок на основе идентификаторов, начиная с преобразования универсальных структур данных в определенные типы при необходимости, обеспечения целостности данных с помощью чистых функций проверки и создания гибких универсальных уровней доступа к данным.
