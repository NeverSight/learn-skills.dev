À utiliser lorsque vous devez appliquer les meilleures pratiques de programmation orientées données en Java, notamment séparer le code (comportement) des structures de données à l'aide d'enregistrements, concevoir des données immuables avec des fonctions de transformation pures, conserver les données plates et dénormalisées avec des références basées sur l'ID, commencer par des structures de données génériques se convertissant en types spécifiques si nécessaire, garantir l'intégrité des données grâce à des fonctions de validation pures et créer des couches d'accès aux données génériques flexibles. Une partie du projet skills-for-java
