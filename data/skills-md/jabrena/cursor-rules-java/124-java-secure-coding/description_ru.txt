Используйте, когда вам нужно применить передовые методы безопасного кодирования Java, включая проверку ненадежных входных данных, защиту от атак путем внедрения с помощью параметризованных запросов, минимизацию поверхности атаки с помощью минимальных привилегий, применение надежных криптографических алгоритмов, безопасную обработку исключений без раскрытия конфиденциальных данных, управление секретами во время выполнения, избежание небезопасной десериализации и кодирование вывода для предотвращения XSS.
