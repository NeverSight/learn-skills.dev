مهارة أفضل الممارسات الشاملة لـ GSAP v3 والتي تغطي الأساسيات والمكونات الإضافية والميزات/الأدوات المفيدة. يُستخدم عندما يذكر المستخدم GSAP، أو GreenSock، أو gsap.to، أو gsap.timeline، أو ScrollTrigger، أو ScrollSmoother، أو Flip، أو SplitText، أو MorphSVG، أو DrawSVG، أو Draggable، أو useGSAP، أو يطلب إنشاء الرسوم المتحركة أو الحركة أو التفاعلات المستندة إلى التمرير أو مراجعتها أو تحسينها.
