Da utilizzare quando si lavora con progetti Vite, vite.config.ts, HMR, plugin Vite, modalità libreria, build SSR o migrazione da Vite 5/6 a Vite 8. Si attiva quando l'utente menziona Vite, vite.config, vitest (lato configurazione), HMR, Rolldown, plugin Rollup o chiede di configurare, eseguire il debug o ottimizzare un progetto basato su Vite (Vue, React, Svelte, Nuxt, SvelteKit).
