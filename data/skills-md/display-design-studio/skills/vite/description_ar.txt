أداة إنشاء Vite 8 (التي تعمل بنظام Rolldown) - التكوين، وواجهة برمجة تطبيقات البرنامج المساعد، وSSR، وترحيل Rolldown. يُستخدم عند العمل مع مشاريع Vite، أو vite.config.ts، أو HMR، أو مكونات Vite الإضافية، أو وضع المكتبة، أو إنشاءات SSR، أو الترحيل من Vite 5/6 إلى Vite 8. يتم تشغيله عندما يذكر المستخدم Vite، أو vite.config، أو vitest (جانب التكوين)، أو HMR، أو Rolldown، أو البرنامج الإضافي Rollup، أو يطلب تكوين مشروع يستند إلى Vite أو تصحيحه أو تحسينه (Vue، React، Svelte، Nuxt، سفلتكيت).
