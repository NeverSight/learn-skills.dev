Инструмент сборки Vite 8 (на базе Rolldown) — настройка, API плагинов, SSR и миграция Rolldown. Используйте при работе с проектами Vite, vite.config.ts, HMR, плагинами Vite, режимом библиотеки, сборками SSR или переходом с Vite 5/6 на Vite 8. Срабатывает, когда пользователь упоминает Vite, vite.config, vitest (сторона конфигурации), HMR, Rolldown, плагин Rollup или запрашивает настройку, отладку или оптимизацию проекта на основе Vite (Vue, React, Svelte, Nuxt, SvelteKit).
