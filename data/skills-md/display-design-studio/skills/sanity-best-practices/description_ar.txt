أفضل ممارسات تطوير Sanity لتصميم المخطط، واستعلامات GROQ، وTypeGen، والتحرير المرئي، والصور، والنص المحمول، وبنية الاستوديو، والترجمة، والترحيلات، وتكاملات إطار العمل مثل Next.js، وNuxt، وAstro، وRemix، وSvelteKit، وAngular، وHydrogen، وApp SDK. استخدم هذه المهارة عند العمل مع مخططات Sanity، أو DefineType أو DefineField، أو GROQ أو DefinQuery، أو نمذجة المحتوى، أو إعدادات العرض التقديمي أو المعاينة، أو عمليات تكامل الواجهة الأمامية التي تدعم Sanity، أو عند مراجعة قاعدة تعليمات Sanity وإصلاحها.
