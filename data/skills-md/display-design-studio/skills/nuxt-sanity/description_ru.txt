Рекомендации по интеграции модулей @nuxtjs/sanity для приложений Nuxt 3, подключенных к Sanity CMS. Охватывает useSanityQuery, useLazySanityQuery, useSanity, SanityImage, SanityContent (Portable Text), визуальное редактирование/предварительный просмотр в реальном времени с помощью stega, TypeScript typegen, именованных клиентов и маршрутов сервера Nitro. Также рассматривается начальная архитектура с кэшированными конечными точками Nitro, компонентами предварительного просмотра, кэшированием Netlify CDN и аннулированием тегов кэша. Используйте, когда пользователь упоминает @nuxtjs/sanity, nuxt-sanity, useSanityQuery, SanityImage в Nuxt или создает приложение Nuxt, которое извлекает данные из Sanity.
