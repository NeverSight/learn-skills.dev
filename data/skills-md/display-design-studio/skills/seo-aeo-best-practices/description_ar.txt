أفضل ممارسات تحسين محركات البحث (SEO) وAEO للبيانات الوصفية، وOpen Graph، وخرائط المواقع، وrobots.txt، وhreflang، والبيانات المنظمة JSON-LD، وEEAT، والمحتوى المحسّن لمحركات البحث وأسطح إجابات الذكاء الاصطناعي. استخدم هذه المهارة عند تنفيذ تحسين محركات البحث للصفحة، أو تحسين محركات البحث الفنية، أو ترميز المخطط، أو تحسين محركات البحث الدولية، أو الاستعداد للنظرة العامة على الذكاء الاصطناعي، أو تحسين المحتوى لـ Google، وChatGPT، وPerplexity، والمساعدين المشابهين.
