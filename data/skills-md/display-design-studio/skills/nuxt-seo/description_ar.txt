أفضل الممارسات الكاملة لمجموعة @nuxtjs/seo لـ Nuxt 3 - تغطي @nuxtjs/sitemap، وnuxt-og-image، وnuxt-schema-org، وnuxt-seo-utils، وnuxt-link-checker، و@nuxtjs/robots، وuseSeoMeta، وuseHead، وbreadcrumbs، وعناوين URL الأساسية، وfavicon، وog:image، والبيانات المنظمة، و llms.txt. يُستخدم عندما يذكر المستخدم خريطة الموقع، أو og image، أو schema.org، أو useSeoMeta، أو useHead، أو breadcrumbs، أو canonical، أو link checker، أو robots.txt، أو noindex، أو blockAiBots، أو llms.txt، أو أي وحدة @nuxtjs/seo في مشروع Nuxt.
