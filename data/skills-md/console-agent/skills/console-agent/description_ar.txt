أنشئ وكلاء الذكاء الاصطناعي باستخدام console.agent() - jQuery الخاص بوكلاء الذكاء الاصطناعي. قم بإسقاط console.agent(...) في أي مكان في التعليمات البرمجية الخاصة بك لسير العمل الوكيل مع بساطة console.log(). يُستخدم عند إضافة إمكانات وكيل الذكاء الاصطناعي، أو تصحيح الأخطاء باستخدام الذكاء الاصطناعي، أو تدقيق الأمان، أو التسجيل الذكي، أو تحليل وقت التشغيل.
