Erstellen, ziehen, pushen und verwalten Sie Versionen von Tapcart-Blöcken und globalen Komponenten mithilfe der Tapcart-CLI. Verwenden Sie diese Option, wenn der Benutzer einen Block oder eine Komponente erstellen, Blöcke aus dem Dashboard abrufen, Änderungen übertragen, einen Entwicklungsserver starten, Versionen auflisten oder eine Version live schalten möchte.
