Configure y administre un proyecto CLI de Tapcart. Úselo cuando el usuario quiera comenzar con el desarrollo de Tapcart por primera vez o administrar un proyecto existente, incluida la autenticación, las dependencias, el linting, los registros y el servidor de desarrollo de diseño. El usuario solo necesita proporcionar su ID de aplicación para la configuración inicial; el agente se encarga de todo lo demás.
