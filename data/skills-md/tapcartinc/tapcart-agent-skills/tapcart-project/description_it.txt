Configura e gestisci un progetto CLI Tapcart. Da utilizzare quando l'utente desidera iniziare con lo sviluppo di Tapcart per la prima volta o gestire un progetto esistente, inclusi autenticazione, dipendenze, linting, registri e server di sviluppo del layout. L'utente deve solo fornire il proprio ID app per la prima configurazione: l'agente gestisce tutto il resto.
