Настройте проект Tapcart CLI и управляйте им. Используйте, когда пользователь хочет впервые начать разработку Tapcart или управлять существующим проектом, включая аутентификацию, зависимости, анализ, журналы и сервер разработки макета. Пользователю необходимо указать свой идентификатор приложения только для первоначальной настройки — все остальное сделает агент.
