Agent Farm CLI – das Tool zum Spawnen von Buildern, zum Verwalten von Tower, Arbeitsbereichen und Cron-Aufgaben. Konsultieren Sie diesen Skill IMMER, BEVOR Sie einen „af“-Befehl ausführen, um die genaue Syntax zu erhalten. Dies verhindert, dass Sie Zeit damit verschwenden, nicht vorhandene Flags zu erraten. Verwenden Sie dies immer dann, wenn Sie einen Builder starten, den Status überprüfen, Nachrichten senden, Arbeitsbäume bereinigen, Tower verwalten oder Cron-Aufgaben ausführen müssen. Wenn Sie im Begriff sind, „af“ gefolgt von etwas einzugeben, schauen Sie zuerst hier nach.
