CLI di gestione dei progetti Codev: comandi init, adopt, update e doctor. Controlla questa abilità prima di eseguire qualsiasi comando `codev` (eccetto `consult`, `porch` o `af` che hanno le proprie abilità). Da utilizzare durante l'impostazione di nuovi progetti, l'aggiunta di codev a repository esistenti, l'aggiornamento di file framework o la diagnosi di dipendenze mancanti.
