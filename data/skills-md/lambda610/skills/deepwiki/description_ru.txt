Генератор документации по локальной кодовой базе на базе искусственного интеллекта. Используется для: (1) глубокого анализа кода с извлечением бизнес-логики, (2) комплексной документации API с сигнатурами, (3) архитектурных диаграмм и сопоставления потоков данных, (4) идентификации шаблонов кода и принятия проектных решений. Не для: только простые списки файлов или комментарии к коду.
