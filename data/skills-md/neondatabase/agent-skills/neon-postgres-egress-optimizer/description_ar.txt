تشخيص وإصلاح خروج Postgres المفرط (نقل بيانات الشبكة) في قاعدة التعليمات البرمجية. يُستخدم عندما يذكر المستخدم ارتفاع فواتير قاعدة البيانات، أو تكاليف نقل البيانات غير المتوقعة، أو رسوم نقل الشبكة، أو ارتفاعات الخروج، أو "لماذا ترتفع فاتورة النيون الخاصة بي"، أو "قفزة تكاليف قاعدة البيانات"، أو تحسين SELECT *، أو جلب الاستعلام الزائد، أو تقليل تكاليف النيون، أو تحسين استخدام قاعدة البيانات، أو يريد تقليل البيانات المرسلة من قاعدة البيانات الخاصة به إلى تطبيقه. استخدمه أيضًا عند مراجعة أنماط الاستعلام لتحقيق كفاءة التكلفة، حتى إذا لم يذكر المستخدم صراحةً الخروج أو نقل البيانات.
