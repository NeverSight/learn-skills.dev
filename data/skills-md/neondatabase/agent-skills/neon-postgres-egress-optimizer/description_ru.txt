Диагностика и устранение чрезмерного выхода Postgres (передача данных по сети) в базе кода. Используйте, когда пользователь упоминает высокие счета за базу данных, непредвиденные расходы на передачу данных, плату за передачу данных по сети, скачки исходящего трафика, «почему мой счет за Neon такой высокий», «расходы на базу данных выросли», оптимизацию SELECT *, чрезмерную выборку запросов, снижение затрат на Neon, оптимизацию использования базы данных или желание уменьшить объем данных, отправляемых из базы данных в приложение. Также используйте при проверке шаблонов запросов для обеспечения экономической эффективности, даже если пользователь явно не упоминает исходящий трафик или передачу данных.
