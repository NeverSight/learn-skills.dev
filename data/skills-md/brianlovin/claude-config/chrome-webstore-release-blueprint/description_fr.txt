Guidez un utilisateur de bout en bout dans la configuration de l'automatisation de la publication de l'API du Chrome Web Store dans n'importe quel référentiel. À utiliser lorsqu'on vous demande de guider quelqu'un dans la configuration des informations d'identification OAuth/CWS, la création de jetons d'actualisation, la configuration des secrets locaux/CI, l'automatisation de la publication basée sur la version et les vérifications de l'état de soumission.
