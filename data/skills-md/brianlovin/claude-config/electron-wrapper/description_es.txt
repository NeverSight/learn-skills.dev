Combine una aplicación web de Bun en una aplicación de escritorio de Electron con administración de ventanas nativa, actualizaciones automáticas, firma de código y distribución de CI/CD. Úselo cuando el usuario desee crear una aplicación de escritorio nativa a partir de un servidor web existente basado en Bun, empaquetarla para macOS/Windows, configurar la actualización automática o manejar inquietudes de Electron UX, como regiones de arrastre y semáforos. Úselo también al cortar lanzamientos o etiquetar versiones para aplicaciones de Electron.
