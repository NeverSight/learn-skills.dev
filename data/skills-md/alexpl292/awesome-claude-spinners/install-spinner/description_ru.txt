Установите тематические пакеты спиннеров для Claude Code. Получает пакеты спиннеров непосредственно из репозитория Awesome-Claude-Spinners и устанавливает их в настройки пользователя. Используйте, когда пользователь хочет настроить свой счетчик Claude Code, установить глаголы счетчика или изменить темы счетчика.
