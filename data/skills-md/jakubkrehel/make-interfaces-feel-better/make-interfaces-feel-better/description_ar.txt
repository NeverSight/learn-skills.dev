مبادئ هندسة التصميم لجعل الواجهات مصقولة. يُستخدم عند إنشاء مكونات واجهة المستخدم، أو مراجعة كود الواجهة الأمامية، أو تنفيذ الرسوم المتحركة، أو حالات التمرير، أو الظلال، أو الحدود، أو الطباعة، أو التفاعلات الدقيقة، أو رسوم الدخول/الخروج المتحركة، أو أي عمل تفصيلي مرئي. مشغلات لتلميع واجهة المستخدم، وتفاصيل التصميم، "تجعل الأمر يبدو أفضل"، "يشعر بالراحة"، الرسوم المتحركة المتداخلة، نصف قطر الحدود، المحاذاة البصرية، تجانس الخط، الأرقام الجدولية، مخططات الصور، ظلال المربع.
