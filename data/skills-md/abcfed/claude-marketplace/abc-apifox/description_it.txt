Strumento di query sulla documentazione API di ABC Medical Cloud. Legge ed esegue query sul documento delle specifiche OpenAPI dell'API ABC (oltre 5000 endpoint), supporta la ricerca per modulo, percorso e metodo e risolve automaticamente i riferimenti $ref. Utilizza una struttura di cache divisa in moduli per query veloci. Casi d'uso: (1) Definizioni di endpoint API di query (2) Ricerca di endpoint funzionali specifici (3) Definizioni di schemi di query (4) Visualizzazione di statistiche di endpoint
