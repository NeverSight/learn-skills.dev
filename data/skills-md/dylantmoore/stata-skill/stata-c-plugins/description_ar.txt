قم بتطوير مكونات إضافية C/C++ عالية الأداء لـ Stata باستخدام stplugin.h SDK. يُستخدم عندما يطلب المستخدم إنشاء مكون إضافي لـ Stata، أو كتابة كود C/C++ لـ Stata، أو تسريع أمر Stata باستخدام C، أو إنشاء مكونات إضافية لـ Stata عبر الأنظمة الأساسية، أو ترجمة/نقل حزمة Python أو R إلى Stata. يغطي دورة الحياة الكاملة: إعداد SDK، وتدفق البيانات، وأمان الذاكرة، وأغلفة .ado مع الحفظ/الدمج، والتجميع عبر الأنظمة الأساسية، وتحسين الأداء (pthreads، والمؤشرات التي تم فرزها مسبقًا، وXorShift RNG)، وتصحيح الأخطاء، والتوزيع عبر تثبيت الشبكة. يتضمن أيضًا سير عمل ترجمة لنقل حزم Python/R إلى Stata - تغليف واجهات C++ الخلفية الموجودة عندما تكون متاحة، أو كتابة C من البداية عندما لا يكون ذلك متاحًا.
