Разрабатывайте высокопроизводительные плагины C/C++ для Stata, используя stplugin.h SDK. Используйте, когда пользователь просит создать плагин Stata, написать код C/C++ для Stata, ускорить выполнение команды Stata с помощью C, создать кросс-платформенные плагины Stata или перевести/портировать пакет Python или R в Stata. Охватывает полный жизненный цикл: настройку SDK, поток данных, безопасность памяти, оболочки .ado с сохранением/слиянием, кроссплатформенную компиляцию, оптимизацию производительности (pthreads, предварительно отсортированные индексы, XorShift RNG), отладку и распространение через сетевую установку. Также включает рабочий процесс перевода для переноса пакетов Python/R в Stata — перенос существующих серверных частей C++, если они доступны, или написание C с нуля, если они недоступны.
