Entwickeln Sie leistungsstarke C/C++-Plugins für Stata mit dem stplugin.h SDK. Verwenden Sie diese Option, wenn der Benutzer ein Stata-Plugin erstellen, C/C++-Code für Stata schreiben, einen Stata-Befehl mit C beschleunigen, plattformübergreifende Stata-Plugins erstellen oder ein Python- oder R-Paket in Stata übersetzen/portieren möchte. Deckt den gesamten Lebenszyklus ab: SDK-Einrichtung, Datenfluss, Speichersicherheit, .ado-Wrapper mit Erhaltung/Zusammenführung, plattformübergreifende Kompilierung, Leistungsoptimierung (Pthreads, vorsortierte Indizes, XorShift RNG), Debugging und Verteilung über die Netzinstallation. Enthält außerdem einen Übersetzungsworkflow für die Portierung von Python/R-Paketen nach Stata – das Umschließen vorhandener C++-Backends, wenn verfügbar, oder das Schreiben von C von Grund auf, wenn nicht.
