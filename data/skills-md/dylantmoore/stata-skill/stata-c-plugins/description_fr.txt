Développez des plugins C/C++ hautes performances pour Stata à l'aide du SDK stplugin.h. À utiliser lorsque l'utilisateur demande de créer un plugin Stata, d'écrire du code C/C++ pour Stata, d'accélérer une commande Stata avec C, de créer des plugins Stata multiplateformes ou de traduire/porter un package Python ou R dans Stata. Couvre le cycle de vie complet : configuration du SDK, flux de données, sécurité de la mémoire, wrappers .ado avec préservation/fusion, compilation multiplateforme, optimisation des performances (pthreads, index pré-triés, XorShift RNG), débogage et distribution via l'installation réseau. Inclut également un flux de travail de traduction pour le portage des packages Python/R vers Stata – en encapsulant les backends C++ existants lorsqu'ils sont disponibles, ou en écrivant du C à partir de zéro lorsqu'ils ne sont pas disponibles.
