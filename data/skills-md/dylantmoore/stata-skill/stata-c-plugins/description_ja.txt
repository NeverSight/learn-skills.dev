stplugin.h SDK を使用して、Stata 用の高性能 C/C++ プラグインを開発します。ユーザーが Stata プラグインの作成、Stata 用の C/C++ コードの作成、C による Stata コマンドの高速化、クロスプラットフォーム Stata プラグインの構築、または Python または R パッケージの Stata への変換/移植を要求する場合に使用します。 SDK セットアップ、データ フロー、メモリ安全性、保持/マージを備えた .ado ラッパー、クロスプラットフォーム コンパイル、パフォーマンスの最適化 (pthread、事前にソートされたインデックス、XorShift RNG)、デバッグ、ネット インストールによる配布など、ライフサイクル全体をカバーします。 Python/R パッケージを Stata に移植するための変換ワークフローも含まれています。利用可能な場合は既存の C++ バックエンドをラップし、利用できない場合は C を最初から作成します。
