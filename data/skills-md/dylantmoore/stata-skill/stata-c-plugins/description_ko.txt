stplugin.h SDK를 사용하여 Stata용 고성능 C/C++ 플러그인을 개발하세요. 사용자가 Stata 플러그인 생성, Stata용 C/C++ 코드 작성, C를 사용하여 Stata 명령 가속화, 크로스 플랫폼 Stata 플러그인 구축, Python 또는 R 패키지를 Stata로 변환/포팅하도록 요청할 때 사용합니다. SDK 설정, 데이터 흐름, 메모리 안전, 보존/병합이 포함된 .ado 래퍼, 크로스 플랫폼 컴파일, 성능 최적화(pthread, 사전 정렬된 인덱스, XorShift RNG), 디버깅 및 넷 설치를 통한 배포 등 전체 수명주기를 다룹니다. 또한 Python/R 패키지를 Stata로 포팅하기 위한 번역 워크플로가 포함되어 있습니다. 사용 가능한 경우 기존 C++ 백엔드를 래핑하고, 사용 가능하지 않은 경우 처음부터 C를 작성합니다.
