Desarrolle complementos C/C++ de alto rendimiento para Stata utilizando el SDK stplugin.h. Úselo cuando el usuario solicite crear un complemento de Stata, escribir código C/C++ para Stata, acelerar un comando de Stata con C, crear complementos de Stata multiplataforma o traducir/portar un paquete Python o R a Stata. Cubre el ciclo de vida completo: configuración de SDK, flujo de datos, seguridad de la memoria, contenedores .ado con preservación/fusión, compilación multiplataforma, optimización del rendimiento (pthreads, índices preordenados, XorShift RNG), depuración y distribución a través de instalación en red. También incluye un flujo de trabajo de traducción para portar paquetes de Python/R a Stata: empaquetar los backends de C++ existentes cuando estén disponibles o escribir C desde cero cuando no esté disponible.
