استخدم هذه المهارة عندما يريد المستخدم تخزين أسرار Goldsky أو ​​إدارتها أو العمل معها - وهي كائنات بيانات الاعتماد المسماة التي تستخدمها أحواض خطوط الأنابيب. يتضمن ذلك: إنشاء سر جديد من سلسلة اتصال أو بيانات الاعتماد، وإدراج الأسرار الموجودة أو فحصها، وتحديث بيانات الاعتماد أو تدويرها بعد تغيير كلمة المرور، وحذف الأسرار التي لم تعد هناك حاجة إليها. تشغيل أي استعلام حيث يذكر المستخدم "سر Goldsky"، أو يريد تخزين بيانات اعتماد قاعدة البيانات لخط أنابيب بشكل آمن، أو يعمل مع مصادقة الحوض لـ PostgreSQL، أو Neon، أو Supabase، أو ClickHouse، أو Kafka، أو S3، أو Elasticsearch، أو DynamoDB، أو SQS، أو OpenSearch، أو webhooks.
