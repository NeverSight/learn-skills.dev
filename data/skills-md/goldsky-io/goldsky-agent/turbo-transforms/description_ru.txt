Написание SQL, TypeScript и преобразований динамических таблиц для конвейеров Goldsky Turbo. Используйте этот навык для: декодирования журналов событий EVM с помощью _gs_log_decode (требуется ABI) или ввода транзакций с помощью _gs_tx_decode, фильтрации и преобразования данных блокчейна в SQL, объединения нескольких декодированных типов событий в одну таблицу с помощью UNION ALL, написания преобразований TypeScript/WASM с использованием сигнатуры функции вызова (данных), настройки таблиц динамического поиска для фильтрации переводов по списку кошельков, который вы обновляете во время выполнения (dynamic_table_check), связывания SQL и Совместные действия TypeScript или отладка нулевых значений в декодированных полях. Для полной структуры конвейера YAML вместо этого используйте /turbo-pipelines. Для построения всего конвейера от начала до конца используйте вместо этого /turbo-builder.
