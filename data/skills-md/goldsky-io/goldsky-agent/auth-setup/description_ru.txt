Настройте аутентификацию Goldsky CLI и конфигурацию проекта. Используйте этот навык, когда пользователю необходимо: установить CLI goldsky (какая официальная команда установки?), запустить вход в goldsky (в том числе, когда браузер открывается, но «аутентификация не удалась»), запустить список проектов goldsky и увидеть «не авторизован» или «не авторизован», переключиться между проектами Goldsky, проверить, в каком проекте они в данный момент аутентифицированы, или исправить «несанкционированные» ошибки при запуске турбо-команд goldsky. Также используйте, чтобы «познакомить меня с настройкой goldsky CLI с нуля в первый раз». Если какой-либо другой навык Голдски выявляет ошибку авторизации, сначала перенаправьте сюда.
