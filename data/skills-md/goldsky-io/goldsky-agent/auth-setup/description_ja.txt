Goldsky CLI 認証とプロジェクト構成をセットアップします。このスキルは、ユーザーが以下を行う必要がある場合に使用します: goldsky CLI のインストール (公式のインストール コマンドは何ですか?)、goldsky ログインの実行 (ブラウザが開くが「認証に失敗した」場合を含む)、goldsky プロジェクト リストを実行して「ログインしていません」または「未承認」を確認する、Goldsky プロジェクトを切り替える、現在認証されているプロジェクトを確認する、または goldsky Turbo コマンド実行時の「未承認」エラーを修正します。 「初めて goldsky CLI を最初からセットアップする手順を説明する」にも使用します。他の Goldsky スキルで認証エラーが発生した場合は、最初にここにリダイレクトしてください。
