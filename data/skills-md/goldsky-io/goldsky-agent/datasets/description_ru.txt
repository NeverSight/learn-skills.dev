Используйте этот навык, когда пользователю необходимо найти или проверить имена наборов данных блокчейна Goldsky, префиксы цепочек, типы или версии наборов данных. Запускается по таким вопросам, как «Какое имя набора данных для X?», «Какой префикс Голдски использует для цепочки Y?», «Какую версию мне следует использовать для Z?» или «Какие наборы данных доступны для Solana/Stellar/Arbitrum/etc?». Также используйте для вопросов о наборах данных, специфичных для цепочки (например, префикс полигона или matic, наборы данных баланса stellarnet, имена наборов данных передачи токенов solana). НЕ задавайте вопросы о командах CLI, настройке конвейера или общей архитектуре Goldsky, если только основной вопрос не касается поиска правильного имени набора данных или префикса цепочки.
