Utilice esta habilidad cuando el usuario necesite buscar o verificar nombres de conjuntos de datos, prefijos de cadena, tipos de conjuntos de datos o versiones de Goldsky blockchain. Activa preguntas como "¿cuál es el nombre del conjunto de datos para X?", "¿qué prefijo usa Goldsky para la cadena Y?", "¿qué versión debo usar para Z?" o "¿qué conjuntos de datos están disponibles para Solana/Stellar/Arbitrum/etc?". Úselo también para preguntas sobre conjuntos de datos específicos de la cadena (por ejemplo, prefijo polígono versus mático, conjuntos de datos de equilibrio de Stellarnet, nombres de conjuntos de datos de transferencia de tokens de Solana). NO active preguntas sobre comandos CLI, configuración de canalización o arquitectura general de Goldsky a menos que la pregunta principal sea encontrar el nombre del conjunto de datos o el prefijo de cadena correcto.
