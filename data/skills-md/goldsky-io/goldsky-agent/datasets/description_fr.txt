Utilisez cette compétence lorsque l'utilisateur a besoin de rechercher ou de vérifier les noms des ensembles de données de la blockchain Goldsky, les préfixes de chaîne, les types d'ensembles de données ou les versions. Se déclenche sur des questions telles que « quel est le nom de l'ensemble de données pour X ? », « quel préfixe Goldsky utilise-t-il pour la chaîne Y ? », « quelle version dois-je utiliser pour Z ? » ou « quels ensembles de données sont disponibles pour Solana/Stellar/Arbitrum/etc ?'. À utiliser également pour les questions sur les ensembles de données spécifiques à la chaîne (par exemple, préfixe polygone vs matic, ensembles de données d'équilibre stellarnet, noms d'ensembles de données de transfert de jetons Solana). Ne déclenchez PAS de questions sur les commandes CLI, la configuration du pipeline ou l'architecture générale de Goldsky, sauf si la question principale concerne la recherche du bon nom d'ensemble de données ou du bon préfixe de chaîne.
