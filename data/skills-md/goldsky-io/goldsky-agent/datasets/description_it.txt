Utilizza questa competenza quando l'utente ha bisogno di cercare o verificare i nomi dei set di dati della blockchain Goldsky, i prefissi delle catene, i tipi di set di dati o le versioni. Si attivano domande come "qual è il nome del set di dati per X?", "quale prefisso usa Goldsky per la catena Y?", "quale versione dovrei usare per Z?" o "quali set di dati sono disponibili per Solana/Stellar/Arbitrum/ecc?". Utilizzare anche per domande sui set di dati specifici della catena (ad esempio, prefisso poligono vs matic, set di dati di bilancio stellarnet, nomi di set di dati di trasferimento di token solana). NON attivare per domande sui comandi CLI, sulla configurazione della pipeline o sull'architettura generale di Goldsky, a meno che la domanda principale non riguardi la ricerca del nome del set di dati o del prefisso della catena corretti.
