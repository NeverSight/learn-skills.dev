Utilice esta habilidad cuando el usuario quiera construir, crear o configurar una nueva tubería Goldsky Turbo desde cero. Se activa cuando alguien describe los datos que desea mover, especificando una fuente (cadena, conjunto de datos, contrato) y un destino (postgres, clickhouse, kafka, s3, webhook), o solicita que se le guíe durante la creación del proceso. También activadores de frases como "ayúdame a construir", "Quiero indexar", "configurar una canalización" o "seguir la cadena X en Y". Cubre el flujo de trabajo completo: recopilación de requisitos, selección de conjuntos de datos, generación de YAML, validación e implementación. NO lo use para depurar canalizaciones existentes (use /turbo-doctor), búsquedas de sintaxis YAML (use /turbo-pipelines) o preguntas sobre campos específicos/opciones de configuración sin intención de compilar.
