Utilizza questa competenza quando l'utente desidera costruire, creare o impostare una nuova pipeline Goldsky Turbo da zero. Si attiva quando qualcuno descrive i dati che desidera spostare, specificando un'origine (catena, set di dati, contratto) e una destinazione (postgres, clickhouse, kafka, s3, webhook), o chiede di essere guidato attraverso la creazione della pipeline. Si attivano anche frasi come "aiutami a costruire", "voglio indicizzare", "impostare una pipeline" o "tracciare X sulla catena Y". Copre l'intero flusso di lavoro: raccolta dei requisiti, selezione di set di dati, generazione di YAML, convalida e distribuzione. NON utilizzare per il debug di pipeline esistenti (utilizzare /turbo-doctor), ricerche di sintassi YAML (utilizzare /turbo-pipelines) o domande su campi/opzioni di configurazione specifici senza intenzione di creare.
