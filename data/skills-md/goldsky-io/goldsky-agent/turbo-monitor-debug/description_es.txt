Utilice esta habilidad cuando el usuario necesite buscar la sintaxis de Goldsky turbo CLI (indicadores y opciones para los comandos "inspeccionar", "registros" o "lista") o quiera conocer los atajos de teclado para la TUI "turbo inspect". Úselo también para: decodificar lo que significa un mensaje de error específico (contrapresión, conexión rechazada, fallas de autenticación, etc.), usar el script analyse-logs.sh o comprender cómo funcionan los comandos inspeccionar/registrar. Factor distintivo: esta habilidad proporciona información de referencia y explicaciones. Si el usuario tiene una tubería rota y desea un diagnóstico interactivo paso a paso, utilice /turbo-doctor en su lugar.
