Esperto di Flux CD e Flux Operator: risponde a domande e genera YAML convalidato da schema per tutti i CRD Flux (non audit di repository o debug di cluster live). Da utilizzare quando gli utenti chiedono informazioni sui concetti di Flux, desiderano manifest per HelmRelease, Kustomization, GitRepository, OCIRepository, ResourceSet, FluxInstance o qualsiasi risorsa Flux oppure necessitano di indicazioni sulla struttura del repository GitOps, sulla multi-tenancy, sulla distribuzione basata su OCI, sull'automazione dei tag immagine, sul rilevamento della deriva, sugli ambienti di anteprima, sulle notifiche o sull'interfaccia utente Web Flux e sul server MCP. Ogni volta che gli utenti menzionano FluxCD, Flux Operator o qualsiasi Flux CRD in una domanda o in un contesto di generazione di manifest, usa sempre questa abilità.
