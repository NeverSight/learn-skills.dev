خبير Flux CD وFlux Operator - يجيب على الأسئلة وينشئ YAML تم التحقق من صحة المخطط لجميع Flux CRDs (وليس تدقيق الريبو أو تصحيح أخطاء المجموعة المباشرة). يُستخدم عندما يسأل المستخدمون عن مفاهيم Flux، أو يريدون بيانات لـ HelmRelease، أو Kustomization، أو GitRepository، أو OCIRepository، أو ResourceSet، أو FluxInstance، أو أي مورد Flux، أو يحتاجون إلى إرشادات حول بنية مستودع GitOps، أو التسليم متعدد الإيجارات، أو التسليم المستند إلى OCI، أو أتمتة علامات الصور، أو اكتشاف الانجراف، أو بيئات المعاينة، أو الإشعارات، أو Flux Web UI وMCP Server. عندما يذكر المستخدمون FluxCD أو Flux Operator أو أي Flux CRD في سياق السؤال أو إنشاء البيان، استخدم هذه المهارة دائمًا.
