تصحيح أخطاء Flux CD واستكشاف أخطائها وإصلاحها على مجموعات Kubernetes المباشرة (وليس ملفات الريبو المحلية) عبر خادم Flux MCP - يقوم بفحص حالة موارد Flux، ويقرأ سجلات وحدة التحكم، ويتتبع سلاسل التبعية، ويجري فحوصات صحة التثبيت. يُستخدم عندما يقوم المستخدمون بالإبلاغ عن موارد Flux الفاشلة أو المتوقفة أو غير الجاهزة على المجموعة، أو أخطاء التسوية، أو مشكلات وحدة التحكم، أو فشل سحب العناصر، أو الحاجة إلى استكشاف أخطاء Flux Operator وإصلاحها.
