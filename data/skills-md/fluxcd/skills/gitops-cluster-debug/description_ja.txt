Flux MCP サーバー経由でライブ Kubernetes クラスター (ローカル リポジトリ ファイルではない) 上で Flux CD をデバッグおよびトラブルシューティングします。これは、Flux リソースのステータスを検査し、コントローラー ログを読み取り、依存関係チェーンをトレースし、インストールの健全性チェックを実行します。クラスター上の Flux リソースの失敗、スタック、または準備ができていないこと、リコンシリエーション エラー、コントローラーの問題、アーティファクトのプルの失敗をユーザーが報告する場合、またはライブ クラスターの Flux Operator のトラブルシューティングが必要な場合に使用します。
