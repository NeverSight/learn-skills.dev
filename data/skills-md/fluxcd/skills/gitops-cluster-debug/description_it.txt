Debug e risoluzione dei problemi del CD Flux su cluster Kubernetes live (non file repository locali) tramite il server Flux MCP: ispeziona lo stato delle risorse Flux, legge i log del controller, traccia le catene di dipendenze ed esegue controlli di integrità dell'installazione. Da utilizzare quando gli utenti segnalano risorse Flux non riuscite, bloccate o non pronte su un cluster, errori di riconciliazione, problemi del controller, errori di pull degli artefatti o necessitano della risoluzione dei problemi dell'operatore Flux del cluster in tempo reale.
