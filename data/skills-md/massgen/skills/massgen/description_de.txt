Rufen Sie das Multiagentensystem von MassGen für allgemeine Aufgaben, Evaluierung, Planung oder Spezifikationserstellung auf. Verwenden Sie es immer dann, wenn Sie möchten, dass mehrere KI-Agenten ein Problem angehen, eine externe Sicht auf Ihre Arbeit benötigen, einen gründlich verfeinerten Plan oder einen genau spezifizierten Satz von Anforderungen benötigen. Perfekt für: Schreiben, Codegenerierung, Recherche, Design, Analyse, Pre-PR-Überprüfung, komplexe Projektplanung, Feature-Spezifikation, Architekturentscheidungen oder jede Aufgabe, bei der Multi-Agent-Iteration bessere Ergebnisse liefert als die alleinige Arbeit.
