Interagisci con il motore di simulazione API basato su regole mock-loom tramite strumenti MCP. Configura aree di lavoro, integrazioni, pacchetti, contratti e scenari di endpoint, origini dati, invia richieste fittizie ed esegui il debug dello stato dell'entità con sequenza temporale/rollback. Da utilizzare quando si lavora con mock-loom, mocking API, simulazione API, server mock, importazione di percorsi da OpenAPI/Postman/cURL, rollback di entità, log di traffico o quando l'utente chiede di creare, testare, ispezionare o gestire endpoint fittizi.
