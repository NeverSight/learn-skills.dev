تفاعل مع محرك محاكاة API القائم على القواعد الوهمية عبر أدوات MCP. قم بإعداد مساحات العمل وعمليات التكامل والحزم وعقود نقطة النهاية والسيناريوهات ومصادر البيانات وإرسال طلبات وهمية وتصحيح حالة الكيان باستخدام المخطط الزمني/التراجع. يُستخدم عند العمل مع النول الوهمي، أو محاكاة واجهة برمجة التطبيقات، أو محاكاة واجهة برمجة التطبيقات، أو الخادم الوهمي، أو استيراد المسار من OpenAPI/Postman/cURL، أو التراجع عن الكيان، أو سجلات حركة المرور، أو عندما يطلب المستخدم إنشاء نقاط نهاية وهمية أو اختبارها أو فحصها أو إدارتها.
