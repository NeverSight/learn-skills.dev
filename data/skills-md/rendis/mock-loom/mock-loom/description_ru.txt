Взаимодействуйте с механизмом Mocking Engine на основе правил API с помощью инструментов MCP. Настраивайте рабочие области, интеграцию, пакеты, контракты и сценарии конечных точек, источники данных, отправляйте макетные запросы и отлаживайте состояние объекта с помощью временной шкалы и отката. Используйте при работе с макетом, имитацией API, имитацией API, макетным сервером, импортом маршрутов из OpenAPI/Postman/cURL, откатом объекта, журналами трафика или когда пользователь запрашивает создание, тестирование, проверку или управление макетными конечными точками.
