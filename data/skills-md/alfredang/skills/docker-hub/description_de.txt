Erstellen Sie Docker-Images und übertragen Sie sie an Docker Hub (tertiaryinfotech). Generiert bei fehlender Docker-Datei automatisch eine Docker-Datei, indem der Projekttyp erkannt wird. Verwenden Sie diese Option, wenn Sie Docker-Images übertragen, „Push to Docker Hub“, „Docker Push“ oder eine beliebige Docker Hub-Bereitstellungsaufgabe ausführen.
