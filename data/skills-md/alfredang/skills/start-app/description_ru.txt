Автоматическое определение типа проекта и запуск любого приложения на локальном хосте. Находит доступный порт без конфликтов, определяет правильную команду запуска и запускает сервер разработки. Используйте при запуске «запустить приложение», «запустить приложение», «запустить сервер» или любой локальной задачи сервера разработки.
