Запросите исторические данные фондового рынка (акции A, акции HK, ETF, индексы) через протокол MCP. Используйте этот навык, когда пользователь хочет получить исторические ежедневные данные K-line, финансовые отчеты, рыночные индикаторы, результаты проверки акций или любые другие пакетные данные из базы данных акций. Требуется действительный токен запроса MCP, приобретенный на платформе управления.
