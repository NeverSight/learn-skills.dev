Interroga i dati storici del mercato azionario (azioni A, azioni HK, ETF, indici) tramite il protocollo MCP. Utilizzare questa funzionalità quando l'utente desidera recuperare dati storici giornalieri della linea K, rendiconti finanziari, indicatori di mercato, risultati dello screening delle azioni o qualsiasi altro dato batch dal database delle azioni. Richiede un token di query MCP valido acquistato dalla piattaforma di gestione.
