Взаимодействуйте с API платформы данных Scout. Используйте при работе с коллекциями, таблицами, документами, агентами, хранилищем на диске, синхронизацией или использованием Scout. Поддерживает операции CRUD с коллекциями/таблицами/документами, управление агентами, загрузку/выгрузку файлов, настройку синхронизации данных из различных источников (Интернет, Notion, Google Drive и т. д.) и мониторинг использования.
