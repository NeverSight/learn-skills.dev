Analysieren Sie den Netzwerkverkehr einer beliebigen Website und wandeln Sie ihn in wiederverwendbare API-Fähigkeiten um, die von einem gemeinsamen Marktplatz unterstützt werden. Von jedem Agenten entdeckte Fähigkeiten werden von allen Agenten veröffentlicht, bewertet und wiederverwendet. Erfassen Sie den Netzwerkverkehr, entdecken Sie API-Endpunkte, lernen Sie Muster, führen Sie erlernte Fähigkeiten aus und verwalten Sie die Authentifizierung für geschlossene Sites. Verwenden Sie es, wenn jemand strukturierte Daten von einer Website extrahieren, API-Endpunkte entdecken, Webinteraktionen automatisieren oder ohne offizielle API-Dokumentation arbeiten möchte.
