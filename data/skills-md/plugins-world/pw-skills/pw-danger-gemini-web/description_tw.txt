透過逆向工程的 Gemini Web API 進行文字和圖像生成。

核心能力:
- 文字產生: 使用 Gemini 模型產生文字回應
- 圖像生成: 從文字提示產生圖像並儲存到本機
- 視覺輸入: 支援參考影像進行圖生圖或圖生文
- 多輪對話: 透過 sessionId 保持上下文連續性

使用時機:
- 需要產生高品質 AI 影像時 (封面圖、配圖、插畫等)
- 需要 Gemini 模型進行文字產生時
- 需要基於參考影像產生變體或描述時
- 作為其他技能 (pw-cover-image、pw-redbook-image) 的圖像生成後端

不適用場景:
- 需要官方 API 支援和穩定性保證的生產環境
- 對 API 變更敏感的關鍵業務流程
- 需要批量高頻呼叫的場景 (可能觸發限流)

重要提醒:
- 首次使用需要用戶同意免責聲明
- 需要 Google 帳號認證 (自動開啟瀏覽器登入)
- 在中國大陸需要設定代理訪問
- 這是非官方 API, 可能隨時失效
