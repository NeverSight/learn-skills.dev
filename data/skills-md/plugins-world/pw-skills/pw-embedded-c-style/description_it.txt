Assistente di stile codice C incorporato, basato sulle specifiche del codice del progetto didattico 51 microcontrollore. Per impostazione predefinita, viene utilizzata la denominazione snake_case e la denominazione camel case è facoltativa. Utilizzato per creare strutture di progetto conformi alle specifiche di sviluppo integrate, ottimizzare lo stile del codice e fornire modelli di driver hardware. Adatto per lo sviluppo di 51 microcontrollori, STM32 e altri progetti C integrati.
