Встроенный помощник по стилю кода C, основанный на спецификациях кода учебного проекта 51 микроконтроллера. По умолчанию используется именование Snake_case, а именование в верблюжьем регистре не является обязательным. Используется для создания структур проекта, соответствующих спецификациям разработки встроенных систем, оптимизации стиля кода и предоставления шаблонов драйверов оборудования. Подходит для 51 микроконтроллера, STM32 и других встроенных проектов C.
