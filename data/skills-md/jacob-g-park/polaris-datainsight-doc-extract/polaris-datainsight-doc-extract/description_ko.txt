Polaris AI DataInsight Doc Extract API를 사용하여 Office 문서(DOCX, PPTX, XLSX, HWP, HWPX)에서 구조화된 데이터를 추출합니다. 사용자가 문서 파일에서 텍스트, 표, 차트, 이미지 또는 도형을 구문 분석, 분석 또는 추출하려는 경우에 사용합니다. 사용자가 "DataInsight" 또는 "Polaris"를 명시적으로 언급하지 않더라도 Word, PowerPoint, Excel, HWP 또는 HWPX 파일에서 콘텐츠 추출을 언급하거나, 문서 구조를 구문 분석하거나, RAG 파이프라인용 문서 데이터를 변환해야 하거나, Office 형식 문서에서 테이블, 차트 또는 텍스트를 읽는 것에 대해 요청할 때마다 이 기술을 호출합니다.
