استخرج البيانات المنظمة من مستندات Office (DOCX وPPTX وXLSX وHWP وHWPX) باستخدام Polaris AI DataInsight Doc Extract API. يُستخدم عندما يريد المستخدم تحليل أو تحليل أو استخراج النص أو الجداول أو المخططات أو الصور أو الأشكال من ملفات المستندات. قم باستدعاء هذه المهارة عندما يذكر المستخدم استخراج محتوى من ملفات Word أو PowerPoint أو Excel أو HWP أو HWPX، أو يريد تحليل بنية المستند، أو يحتاج إلى تحويل بيانات المستند لخطوط أنابيب RAG، أو يسأل عن قراءة الجداول أو المخططات أو النص من مستندات بتنسيق Office - حتى لو لم يذكروا صراحة "DataInsight" أو "Polaris".
