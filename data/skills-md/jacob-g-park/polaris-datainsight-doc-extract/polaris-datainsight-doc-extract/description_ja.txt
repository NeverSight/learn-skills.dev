Polaris AI DataInsight Doc Extract API を使用して、Office ドキュメント (DOCX、PPTX、XLSX、HWP、HWPX) から構造化データを抽出します。ユーザーがドキュメント ファイルからテキスト、表、グラフ、画像、または図形を解析、分析、または抽出する場合に使用します。ユーザーが Word、PowerPoint、Excel、HWP、または HWPX ファイルからコンテンツを抽出することに言及したとき、ドキュメント構造を解析したいとき、RAG パイプライン用にドキュメント データを変換する必要があるとき、または Office 形式のドキュメントからの表、グラフ、テキストの読み取りについて尋ねたときは、「DataInsight」または「Polaris」について明示的に言及していなくても、このスキルを呼び出します。
