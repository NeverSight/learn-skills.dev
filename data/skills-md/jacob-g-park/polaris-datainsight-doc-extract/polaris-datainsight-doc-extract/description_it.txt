Estrai dati strutturati da documenti Office (DOCX, PPTX, XLSX, HWP, HWPX) utilizzando l'API Polaris AI DataInsight Doc Extract. Utilizzare quando l'utente desidera analizzare, analizzare o estrarre testo, tabelle, grafici, immagini o forme da file di documenti. Richiama questa funzionalità ogni volta che l'utente menziona l'estrazione di contenuto da file Word, PowerPoint, Excel, HWP o HWPX, desidera analizzare la struttura del documento, deve convertire i dati del documento per pipeline RAG o chiede informazioni sulla lettura di tabelle, grafici o testo da documenti in formato Office, anche se non menzionano esplicitamente "DataInsight" o "Polaris".
