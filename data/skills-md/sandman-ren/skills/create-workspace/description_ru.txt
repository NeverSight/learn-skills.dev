Создавайте и управляйте рабочими пространствами разработки с несколькими репозиториями для сложных задач, включающих несколько репозиториев, зависимостей и параллельную работу агентов. Также управляет разрешениями модульных агентов — добавляйте, удаляйте и просматривайте модули разрешений для GitHub, Docker, Node.js, Python, Rust, Go, Playwright и других. Используйте, когда пользователь хочет настроить рабочую область, добавить репозитории/рабочие деревья/депы в существующую рабочую область, управлять разрешениями рабочей области или инициализировать проект, охватывающий несколько репозиториев. Запускается по таким запросам, как «создать рабочую область», «настроить проект с несколькими репозиториями», «добавить репозиторий в рабочую область», «создать рабочее дерево», «добавить разрешения GitHub» или «настроить разрешения рабочей области».
