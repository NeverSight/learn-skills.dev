Guida esperta sullo sviluppo di Progressive Web App (PWA): installabilità, addetti all'assistenza, funzionalità offline e UX nativa. Utilizza questa competenza ogni volta che l'utente menziona PWA, service lavoratore, manifest dell'app Web, manifest.json, app Web installabile offline-first, aggiunta alla schermata iniziale, beforeinstallprompt, shell dell'app, casella di lavoro, strategie di cache o notifiche push per le app Web. Si attivano anche quando l'utente desidera che il proprio sito Web funzioni offline, sembri un'app nativa o sia installabile, anche se non utilizza esplicitamente il termine "PWA". Attiva frasi come "fallo funzionare senza Internet", "gli utenti possono installarlo", "modalità offline", "icona della schermata iniziale" o "perché la mia app sembra un sito Web".
