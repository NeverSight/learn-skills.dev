漸進式 Web 應用程式 (PWA) 開發的專家指南 - 可安裝性、服務工作者、離線功能和原生體驗 UX。每當使用者提及 PWA、Service Worker、Web 應用程式清單、manifest.json、離線優先、可安裝的 Web 應用程式、新增至主畫面、安裝提示之前、應用程式 shell、工作箱、快取原則或 Web 應用程式的推播通知時，請使用此技能。當用戶想讓他們的網站離線工作、感覺像本機應用程式或可安裝時也會觸發 - 即使他們沒有明確使用術語「PWA」。觸發諸如“讓它在沒有互聯網的情況下工作”、“用戶可以安裝這個”、“離線模式”、“主螢幕圖標”或“為什麼我的應用程式感覺像一個網站”等短語。
