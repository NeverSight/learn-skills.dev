Orientación experta sobre el desarrollo de aplicaciones web progresivas (PWA): instalabilidad, trabajadores de servicios, funcionalidad fuera de línea y experiencia de usuario nativa. Utilice esta habilidad siempre que el usuario mencione PWA, trabajador de servicio, manifiesto de aplicación web, manifest.json, aplicación web instalable primero sin conexión, agregar a la pantalla de inicio, aviso previo a la instalación, shell de aplicación, cuadro de trabajo, estrategias de caché o notificaciones push para aplicaciones web. También se activa cuando el usuario quiere que su sitio web funcione sin conexión, se sienta como una aplicación nativa o sea instalable, incluso si no usa el término "PWA" explícitamente. Active frases como "hacer que funcione sin Internet", "pueden los usuarios instalar esto", "modo sin conexión", "ícono de la pantalla de inicio" o "por qué mi aplicación parece un sitio web".
