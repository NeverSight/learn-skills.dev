Conseils d'experts sur le développement de Progressive Web App (PWA) : installabilité, techniciens de service, fonctionnalités hors ligne et UX native. Utilisez cette compétence chaque fois que l'utilisateur mentionne PWA, service worker, manifeste d'application Web, manifest.json, application Web installable hors ligne d'abord, ajout à l'écran d'accueil, invite avant l'installation, shell d'application, boîte de travail, stratégies de cache ou notifications push pour les applications Web. Déclenchez également lorsque l'utilisateur souhaite que son site Web fonctionne hors ligne, ressemble à une application native ou soit installable - même s'il n'utilise pas explicitement le terme « PWA ». Déclenchez des expressions telles que « faire fonctionner sans Internet », « les utilisateurs peuvent-ils installer ceci », « mode hors ligne », « icône de l'écran d'accueil » ou « pourquoi mon application ressemble-t-elle à un site Web ».
