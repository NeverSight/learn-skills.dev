プログレッシブ Web アプリ (PWA) 開発に関する専門家によるガイダンス - インストール性、サービス ワーカー、オフライン機能、ネイティブ感覚の UX。ユーザーが PWA、Service Worker、Web アプリ マニフェスト、manifest.json、オフラインファースト、インストール可能な Web アプリ、ホーム画面への追加、インストールプロンプト前、アプリ シェル、ワークボックス、キャッシュ戦略、または Web アプリのプッシュ通知に言及するたびに、このスキルを使用します。また、「PWA」という用語を明示的に使用していない場合でも、ユーザーが Web サイトをオフラインで動作させたい、ネイティブ アプリのように感じさせたい、またはインストール可能にしたい場合にもトリガーされます。 「インターネットがなくても機能するようにする」、「ユーザーはこれをインストールできますか」、「オフライン モード」、「ホーム画面のアイコン」、または「なぜ私のアプリが Web サイトのように見えるのか」などのフレーズをトリガーします。
