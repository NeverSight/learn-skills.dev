إرشادات الخبراء حول تطوير تطبيق الويب التقدمي (PWA) - إمكانية التثبيت، والعاملين في الخدمة، والوظائف دون اتصال بالإنترنت، والشعور الأصلي بتجربة المستخدم. استخدم هذه المهارة عندما يذكر المستخدم PWA، أو عامل الخدمة، أو بيان تطبيق الويب، أو Manifest.json، أو تطبيق الويب غير المتصل أولاً، أو تطبيق الويب القابل للتثبيت، أو الإضافة إلى الشاشة الرئيسية، أو قبل مطالبة التثبيت، أو غلاف التطبيق، أو صندوق العمل، أو إستراتيجيات ذاكرة التخزين المؤقت، أو دفع الإشعارات لتطبيقات الويب. يتم تشغيله أيضًا عندما يريد المستخدم جعل موقع الويب الخاص به يعمل دون اتصال بالإنترنت، أو يبدو وكأنه تطبيق أصلي، أو يكون قابلاً للتثبيت - حتى إذا لم يستخدم مصطلح "PWA" بشكل صريح. قم بتشغيل عبارات مثل "اجعله يعمل بدون الإنترنت"، أو "هل يمكن للمستخدمين تثبيت هذا"، أو "وضع عدم الاتصال"، أو "رمز الشاشة الرئيسية"، أو "لماذا يبدو تطبيقي وكأنه موقع ويب".
