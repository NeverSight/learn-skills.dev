Plattformübergreifendes Toolkit zur Betriebssystemautomatisierung und Bildschirmsteuerung. Verwenden Sie es, wenn Benutzer Screenshots, Maus-/Tastatursteuerung, visuelle Erkennung, Fensterverwaltung, Browser-Automatisierung oder Desktop-Automatisierungsaufgaben benötigen. Unterstützt macOS 12+ und Windows 10+. Verwendet unter macOS AppleScript, pyautogui und OpenCV. Verwendet unter Windows pywinauto, pyautogui und OpenCV (kein Hammerspoon-Äquivalent).
