Tencent Docs bietet vollständige Betriebsfunktionen für Tencent Docs. Nutzen Sie diese Fähigkeit, wenn Benutzer Tencent Docs bedienen müssen, einschließlich: (1) Erstellen verschiedener Online-Dokumente (Smart Docs, Word, Excel, Folien, Mindmaps, Flussdiagramme); (2) Abfragen und Durchsuchen von Dokumentbereichen und Dateien; (3) Verwaltung von Raumknoten und Ordnerstrukturen; (4) Lesen des Dokumentinhalts; (5) Bearbeiten von Smart Sheets; (6) Bearbeiten von Smart Docs.
