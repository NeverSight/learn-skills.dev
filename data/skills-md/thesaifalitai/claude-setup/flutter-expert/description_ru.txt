Продвинутый специалист по архитектуре Flutter. Используйте для решений по архитектуре Flutter, а НЕ для стандартной реализации Flutter (для этого используйте flutter-dev). Вызов для: чистой архитектуры Flutter, структуры проекта с приоритетом функций, расширенных шаблонов Riverpod (Notifier/AsyncNotifier/family), расширенного блока (компромиссы Cubit и Bloc, BlocObserver), профилирования производительности Flutter (DevTools, компиляция шейдеров, jank), пользовательского RenderObject, каналов платформы/MethodChannel, многомодульного монорепозитория, разработки плагинов Flutter, isolates/compute(), build_runner генерация кода, get_it/injectable DI, миграция обновлений Flutter 3.x.
