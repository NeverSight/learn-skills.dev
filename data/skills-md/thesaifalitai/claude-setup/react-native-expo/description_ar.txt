خبير متخصص في سير العمل المُدار في React Native وExpo (SDK 52+). قم دائمًا بتشغيل المشروعات التي يديرها Expo: جهاز توجيه Expo، وبناء/إرسال EAS، وتحديثات OTA (تحديثات Expo)، ووحدات Expo (الكاميرا، والإشعارات، والموقع)، والتوافق مع Expo Go، وتقديم متجر التطبيقات عبر EAS. مشغلات أيضًا لـ: إنشاء تطبيق جوال، وإنشاء تطبيق لنظام التشغيل iOS/Android، ومشروع المعرض، وعلامات تبويب/مكدسات Expo Router، ودفع الإشعارات مع إشعارات المعرض، والربط العميق، وشاشات البداية، وNativeWind، والرسوم المتحركة (Reanimated/Skia). بالنسبة إلى مشاريع React Native CLI العارية بدون Expo، استخدم React-Native-expert بدلاً من ذلك.
