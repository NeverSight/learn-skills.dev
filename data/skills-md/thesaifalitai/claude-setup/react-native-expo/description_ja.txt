React Native および Expo マネージド ワークフロー スペシャリスト (SDK 52+)。 Expo 管理プロジェクトでは常にトリガーします: Expo Router、EAS ビルド/送信、OTA アップデート (expo アップデート)、Expo モジュール (カメラ、通知、ロケーション)、Expo Go 互換性、EAS 経由のアプリストア送信。また、次のトリガーも発生します: モバイル アプリの構築、iOS/Android 用アプリの作成、expo プロジェクト、Expo Router タブ/スタック、expo 通知付きプッシュ通知、ディープ リンク、スプラッシュ スクリーン、NativeWind、アニメーション (Reanimated/Skia)。 Expo のない裸の React Native CLI プロジェクトの場合は、代わりにreact-native-expertを使用してください。
