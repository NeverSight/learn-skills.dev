Эксперт по управляемым рабочим процессам React Native и Expo (SDK 52+). ВСЕГДА триггер для проектов, управляемых Expo: Expo Router, сборка/отправка EAS, OTA-обновления (expo-обновления), модули Expo (камера, уведомления, местоположение), совместимость Expo Go, отправка в магазин приложений через EAS. Также триггеры для: создания мобильного приложения, создания приложения для iOS/Android, выставочного проекта, вкладок/стеков Expo Router, push-уведомлений с выставочными уведомлениями, глубоких ссылок, заставок, NativeWind, анимации (Reanimated/Skia). Для простых проектов React Native CLI без Expo используйте вместо этого act-native-expert.
