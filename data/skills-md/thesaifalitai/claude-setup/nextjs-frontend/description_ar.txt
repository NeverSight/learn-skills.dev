مهارة تطوير الواجهة الأمامية للخبراء Next.js وReact.js وVue.js. قم دائمًا بتشغيل أي مهمة تتضمن Next.js (جهاز توجيه التطبيقات، جهاز توجيه الصفحات)، React.js، Vue.js، الواجهة الأمامية لـ TypeScript، مكونات الخادم، مكونات العميل، SSR، SSG، ISR، تحسين SEO، واجهة برمجة التطبيقات للبيانات الوصفية، إدارة حالة Zustand/Redux/Pinia، جلب بيانات React Query/SWR، معالجة النماذج (React Hook Form + Zod)، التوجيه، البرامج الوسيطة، تحسين الصورة، الأداء (Core Web Vitals)، أو التدويل (i18n)، أو النشر إلى Vercel/Netlify. مشغلات أيضًا لـ: "إنشاء موقع ويب"، "إنشاء تطبيق ويب"، "مشروع Next.js"، "صفحة التفاعل"، "مكون Vue"، "هندسة الواجهة الأمامية"، "الصفحة المقصودة"، "واجهة مستخدم لوحة المعلومات".
