Spécialiste de l’implémentation du code des composants de l’interface utilisateur. Déclenchement lors de l'ÉCRITURE ou de la FIXATION du code de l'interface utilisateur — utilitaires CSS Tailwind, composants shadcn/ui, primitives de l'interface utilisateur Radix, variantes de composants CVA, animations Framer Motion, variables CSS pour le mode sombre, code d'accessibilité WCAG (étiquettes ARIA, gestion du focus, navigation au clavier), mises en page réactives, chargeurs de squelette, états vides, histoires de Storybook, Figma-to-code. Déclenche également pour : corriger ce composant, ajouter le mode sombre, le rendre accessible, écrire un composant Button, styliser ce formulaire, ajouter des animations, corriger le changement de disposition. Pour la PLANIFICATION de la conception (choix des couleurs, des styles, des associations de polices pour un nouveau projet), utilisez plutôt ui-ux-pro-max.
