Специалист по внедрению кода компонентов пользовательского интерфейса. Запуск при НАПИСАНИИ или ИСПРАВЛЕНИИ кода пользовательского интерфейса — утилиты Tailwind CSS, компоненты shadcn/ui, примитивы пользовательского интерфейса Radix, варианты компонентов CVA, анимация Framer Motion, переменные CSS для темного режима, код доступности WCAG (метки ARIA, управление фокусом, навигация с помощью клавиатуры), адаптивные макеты, скелетные загрузчики, пустые состояния, истории из сборника рассказов, преобразование Figma в код. Также триггеры для: исправить этот компонент, добавить темный режим, сделать его доступным, написать компонент «Кнопка», стилизовать эту форму, добавить анимацию, исправить сдвиг макета. Для ПЛАНИРОВАНИЯ дизайна (выбора цветов, стилей, сочетаний шрифтов для нового проекта) вместо этого используйте ui-ux-pro-max.
