قم بإنشاء الوثائق الحية للخدمات الصغيرة والحفاظ عليها من خلال تحليل حالة الكود الحالية وسجل Git عند الحاجة. يتم تشغيله عندما يطلب المستخدمون توثيق خدمة ما، أو تحديث المستندات بعد التغييرات، أو مقارنة الفروع، أو إنشاء سجلات التغيير، أو تدقيق حداثة المستندات، أو مزامنة المستندات مع الريبو المركزي، أو ترحيل المستندات القديمة، أو إنشاء مستندات التدفق، أو ADRs، أو نظرة عامة على النظام. عبارات التشغيل: "document [service]"، و"document-branch"، و"document-commits"، و"sync"، و"audit"، و"migrate-legacy"، و"document-flow"، و"create-adr"، و"system-overview"، و"documenta"، و"qué cambió"، و"actualizar docs"، و"gener Changelog"، و"release Notes"، "المستندات الحية".
