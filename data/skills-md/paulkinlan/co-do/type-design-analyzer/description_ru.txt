Анализирует качество шрифтового дизайна, уделяя особое внимание инкапсуляции, инвариантному выражению,
полезность и правоприменение. Предоставляет количественные рейтинги (1–10) для каждого измерения.

Триггеры: при добавлении новых типов, проверке дизайна шрифтов в PR, рефакторинге типов.

Примеры:
- «Просмотр конструкции типа UserAccount» -> анализирует инкапсуляцию типов и инварианты.
- «Анализ шрифтового дизайна в этом PR» -> просмотр всех новых добавленных типов.
- «Проверьте, есть ли у этого типа сильные инварианты» -> оценивает соблюдение инвариантов
- «Как мне улучшить этот тип?» -> предоставляет действенные предложения по дизайну шрифтов
