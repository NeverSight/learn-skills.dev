Lista de verificación de auditoría de seguridad para aplicaciones web. Úselo al revisar, auditar o reforzar la postura de seguridad de una aplicación web. Cubre limitación de velocidad, encabezados de autenticación, bloqueo de IP, CORS, middleware de seguridad, validación de entrada, límites de carga de archivos, uso de ORM y hash de contraseñas. Se activa en solicitudes como "revisar seguridad", "reforzar esta aplicación", "auditoría de seguridad", "verificar vulnerabilidades" o al crear/revisar puntos finales de API.
