Git 分岐戦略、競合解決、リベース ワークフロー、モノリポジトリ パターン、およびチェリーピック、バイセクト、reflog リカバリなどの高度な操作。 「git」、「分岐戦略」、「マージ競合」、「リベース」、「チェリーピック」、「モノリポ git」、「git bisect」、「git reflog」、「スカッシュ コミット」でアクティブ化します。 GitHub Actions CI/CD (github-actions-pipeline-builder を使用)、GitHub API/webhook、または PR レビュー プロセス (code-review-checklist を使用) には適用されません。
