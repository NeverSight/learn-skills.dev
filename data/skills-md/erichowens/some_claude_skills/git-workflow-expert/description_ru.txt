Стратегии ветвления Git, разрешение конфликтов, рабочие процессы перебазирования, шаблоны монорепозиториев и расширенные операции, такие как выбор вишни, разделение пополам и восстановление повторного журнала. Активируйте «git», «стратегию ветвления», «конфликт слияния», «перебазирование», «вишневый выбор», «монорепо git», «git bisect», «git reflog», «сквош-фиксации». НЕ для GitHub Actions CI/CD (используйте github-actions-pipeline-builder), API/веб-перехватчиков GitHub или процессов PR-проверки (используйте контрольный список проверки кода).
