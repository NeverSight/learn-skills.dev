Git 分支策略、冲突解决、rebase 工作流程、monorepo 模式以及高级操作（如cherry-pick、bisect 和 reflog 恢复）。在“git”、“分支策略”、“合并冲突”、“rebase”、“cherry-pick”、“monorepo git”、“git bisect”、“git reflog”、“squash commits”上激活。不适用于 GitHub Actions CI/CD（使用 github-actions-pipeline-builder）、GitHub API/webhooks 或 PR 审核流程（使用 code-review-checklist）。
