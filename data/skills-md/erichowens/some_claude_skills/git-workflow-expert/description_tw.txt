Git 分支策略、衝突解決、rebase 工作流程、monorepo 模式以及進階操作（如cherry-pick、bisect 和 reflog 復原）。在「git」、「分支策略」、「合併衝突」、「rebase」、「cherry-pick」、「monorepo git」、「git bisect」、「git reflog」、「squash commits」上啟動。不適用於 GitHub Actions CI/CD（使用 github-actions-pipeline-builder）、GitHub API/webhooks 或 PR 審核流程（使用 code-review-checklist）。
