Bewertet die Fähigkeiten von Claude Agent anhand von 10 Qualitätsachsen mit Buchstabennoten (A+ bis F) und spezifischen Verbesserungsempfehlungen. Verwenden Sie diese Funktion, wenn Sie eine Fertigkeit prüfen, Fertigkeiten vergleichen, Verbesserungen priorisieren oder eine Qualitätskontrolle für eine Fertigkeitsbibliothek durchführen. Aktivieren Sie „Fähigkeit bewerten“, „Fähigkeit bewerten“, „Fähigkeitsqualität“, „Fähigkeitsprüfung“, „Fähigkeitsüberprüfung“, „Fähigkeit bewerten“. NICHT zum Erstellen von Skills (verwenden Sie Skill-Architect), zum Bewerten der Codequalität oder zum Bewerten von Nicht-Skill-Dokumenten.
