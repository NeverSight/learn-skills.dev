Разбивает описания проблем на естественном языке на подзадачи, подходящие для узлов DAG. Точка входа мета-DAG. Определяет этапы, зависимости, возможности распараллеливания и расплывчатые/плюрипотентные узлы, которые еще невозможно указать. Использует метанавыки предметной области, если они доступны. Активируйте «Декомпозицию задачи», «Разбиение задачи», «План рабочего процесса», «Каковы шаги», «Подзадачи», «Разбивка задачи». НЕ для выполнения декомпозированных задач (используйте dag-runtime), построения структуры DAG (используйте dag-planner) или сопоставления навыков с узлами (используйте dag-skills-matcher).
