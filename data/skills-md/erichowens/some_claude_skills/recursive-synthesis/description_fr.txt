Orchestrez la synthèse collaborative de documents multi-agents à travers 6 phases : divergence, synthèse, commentaire, consolidation, vérification de la réalité, fusion finale. Produit des documents fondateurs faisant autorité à partir de contributions multiperspectives complexes. À utiliser pour les documents constitutionnels, les décisions d'architecture, les chartes organisationnelles ou tout document nécessitant une synthèse multiperspective rigoureuse. S'active sur « document de synthèse », « paternité multi-agents », « synthèse collaborative », « document fondateur », « document d'architecture », « synthèse récursive », « document constitutionnel », « document multi-perspectives ». PAS pour la rédaction simple de documents, les tâches à auteur unique, les résumés rapides ou les documents qui ne nécessitent pas d'examen contradictoire.
