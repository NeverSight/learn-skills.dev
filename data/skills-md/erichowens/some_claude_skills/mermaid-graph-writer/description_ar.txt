يكتب مخططات دقيقة ومنظمة بشكل جيد لحورية البحر لأي احتياجات تصورية. يُستخدم عند إنشاء مخططات انسيابية، أو مخططات تسلسلية، أو أجهزة الحالة، أو نماذج التقارير الإلكترونية، أو المخططات الزمنية، أو الخرائط الذهنية، أو مخططات جانت، أو أي نوع آخر من المخططات التي تدعم حورية البحر. التنشيط على "حورية البحر"، "الرسم التخطيطي"، "المخطط الانسيابي"، "مخطط التسلسل"، "مخطط الحالة"، "مخطط ER"، "تصور"، "رسم الرسم البياني". ليس لعرض/تصدير حورية البحر إلى صور (استخدم عارض الرسم البياني لحورية البحر)، أو فن ASCII، أو أدوات التصميم المستندة إلى واجهة المستخدم الرسومية.
