Используйте при отработке раундов HM или при калибровке глубины истории для целевого уровня. Активируйте на «раунде менеджера по найму», «экране HM», «уровне персонала», «масштабе воздействия». НЕ для собеседований по кодированию, проектирования систем, раундов по поведению/ценностям или написания резюме.
