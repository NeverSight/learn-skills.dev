Web とオフラインの両方のコンテキストで人魚図を SVG、PNG、PDF にレンダリングします。クライアント側の遅延読み込み、SSR トレードオフ、CLI バッチ エクスポート、ネイティブ Rust レンダリングについて説明します。図を画像にエクスポートする場合、Web ページでの Mermaid レンダリング パフォーマンスを最適化する場合、CI/CD 図パイプラインを設定する場合、またはレンダリング ライブラリを選択する場合に使用します。 「マーメイドのレンダリング」、「ダイアグラムのエクスポート」、「マーメイドからpng」、「マーメイドからsvg」、「マーメイドのパフォーマンス」、「ダイアグラムのエクスポート」でアクティブ化します。 Mermaid 構文 (mermaid-graph-writer を使用)、一般的な画像処理、または非 Mermaid ダイアグラム形式の作成には使用できません。
