Создавайте рабочие файлы Dockerfile, конфигурации Docker-Compose, многоэтапные сборки и оптимизацию контейнеров. Активируйте «Dockerfile», «docker», «docker-compose», «контейнер», «многоэтапную сборку», «образ докера», «оптимизацию контейнера», «безопасность докера». НЕ для оркестрации Kubernetes, облачных контейнерных сервисов (ECS, Cloud Run) или конвейеров CI/CD (используйте github-actions-pipeline-builder).
