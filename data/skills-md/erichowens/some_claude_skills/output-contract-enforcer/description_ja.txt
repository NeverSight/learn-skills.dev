DAG ノードの出力が、ダウンストリーム ノードに渡す前に、宣言された JSON スキーマと一致することを検証します。マルチエージェント DAG の信頼性を高める接着剤。出力のコンプライアンス準拠のチェック、説明から検証スキーマの生成、またはノード間のコントラクトの不一致のデバッグを行う場合に使用します。 「出力の検証」、「出力コントラクト」、「スキーマ検証」、「コントラクトの不一致」、「出力が一致しません」の場合にアクティブ化します。コンテンツ品質評価 (dag-quality を使用)、スキル評価 (skill-grader を使用)、または DAG コンテキスト外での一般的な JSON スキーマ作業には使用できません。
