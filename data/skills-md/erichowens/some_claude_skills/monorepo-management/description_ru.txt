Инструменты Monorepo, оркестровка задач и архитектура рабочей области для репозиториев JavaScript/TypeScript. Используйте при настройке рабочих пространств Turborepo, Nx, pnpm или рабочих пространств npm; проектирование границ упаковки; настройка удаленного кэширования; оптимизация CI для затронутых пакетов; управление версиями с помощью наборов изменений; или распутывание циклических зависимостей. Активируйте в «monorepo», «turborepo», «nx», «рабочей области pnpm», «конвейере задач», «удаленном кеше», «наборах изменений», «CODEOWNERS», «циклической зависимости», «затронутых пакетах», «рабочей области». НЕ для подмодулей git или стратегий федерации с несколькими репозиториями, монорепозиториев без JavaScript (Bazel, Pants, Buck) или настройки репозитория с одним пакетом.
