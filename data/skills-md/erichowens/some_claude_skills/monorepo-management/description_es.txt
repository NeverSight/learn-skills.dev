Herramientas Monorepo, orquestación de tareas y arquitectura de espacio de trabajo para repositorios de JavaScript/TypeScript. Úselo al configurar espacios de trabajo Turborepo, Nx, pnpm o npm; diseñar límites de paquetes; configurar el almacenamiento en caché remoto; optimizar CI para los paquetes afectados; gestionar el control de versiones con conjuntos de cambios; o desenredar dependencias circulares. Activar en "monorepo", "turborepo", "nx", "pnpm workspace", "task pipeline", "remote cache", "changesets", "CODEOWNERS", "circular dependency", "affected packages", "workspace". NO para submódulos de git o estrategias de federación de múltiples repositorios, monorepositorios que no sean de JavaScript (Bazel, Pants, Buck) o configuración de repositorio de paquete único.
