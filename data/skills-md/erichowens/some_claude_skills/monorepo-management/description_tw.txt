用於 JavaScript/TypeScript 儲存庫的 Monorepo 工具、任務編排和工作區架構。在設定 Turborepo、Nx、pnpm 工作區或 npm 工作區時使用；設計包邊界；配置遠端快取；針對受影響的套件最佳化 CI；使用變更集管理版本控制；或解開循環相依性。在「monorepo」、「turborepo」、「nx」、「pnpm 工作區」、「任務管道」、「遠端快取」、「變更集」、「程式碼擁有者」、「循環依賴」、「受影響的包」、「工作區」上啟動。不適用於 git 子模組或多儲存庫聯合策略、非 JavaScript monorepos（Bazel、Pants、Buck）或單包儲存庫設定。
