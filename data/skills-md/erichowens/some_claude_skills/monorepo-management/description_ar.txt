أدوات Monorepo وتنسيق المهام وهندسة مساحة العمل لمستودعات JavaScript/TypeScript. يُستخدم عند إعداد مساحات عمل Turborepo، أو Nx، أو pnpm، أو مساحات عمل npm؛ وتصميم حدود الحزمة؛ تكوين التخزين المؤقت عن بعد؛ تحسين CI للحزم المتأثرة؛ إدارة الإصدارات باستخدام Changesets؛ أو فك التبعيات الدائرية. التنشيط على "monorepo"، "turborepo"، "nx"، "مساحة عمل pnpm"، "خط أنابيب المهام"، "ذاكرة التخزين المؤقت البعيدة"، "مجموعات التغيير"، "مالكو الرموز"، "التبعية الدائرية"، "الحزم المتأثرة"، "مساحة العمل". ليس لوحدات git الفرعية أو استراتيجيات اتحاد الريبو المتعدد، أو monorepos غير JavaScript (Bazel، Pants، Buck)، أو إعداد مستودع الحزمة الواحدة.
