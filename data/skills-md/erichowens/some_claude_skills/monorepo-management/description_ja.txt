Monorepo ツール、タスク オーケストレーション、JavaScript/TypeScript リポジトリのワークスペース アーキテクチャ。 Turborepo、Nx、pnpm ワークスペース、または npm ワークスペースをセットアップするときに使用します。パッケージ境界の設計。リモート キャッシングの構成。影響を受けるパッケージの CI を最適化します。変更セットによるバージョン管理。または循環依存関係を解きます。 「monorepo」、「turborepo」、「nx」、「pnpm ワークスペース」、「タスク パイプライン」、「リモート キャッシュ」、「変更セット」、「CODEOWNERS」、「循環依存関係」、「影響を受けるパッケージ」、「ワークスペース」でアクティブ化します。 git サブモジュールやマルチリポジトリフェデレーション戦略、非 JavaScript モノリポジトリ (Bazel、Pants、Buck)、または単一パッケージのリポジトリ設定には使用できません。
