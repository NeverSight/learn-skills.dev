用于 JavaScript/TypeScript 存储库的 Monorepo 工具、任务编排和工作区架构。在设置 Turborepo、Nx、pnpm 工作区或 npm 工作区时使用；设计包边界；配置远程缓存；针对受影响的包优化 CI；使用变更集管理版本控制；或解开循环依赖。在“monorepo”、“turborepo”、“nx”、“pnpm 工作区”、“任务管道”、“远程缓存”、“变更集”、“代码所有者”、“循环依赖”、“受影响的包”、“工作区”上激活。不适用于 git 子模块或多存储库联合策略、非 JavaScript monorepos（Bazel、Pants、Buck）或单包存储库设置。
