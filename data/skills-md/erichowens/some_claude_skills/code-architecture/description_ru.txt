Шаблоны архитектуры приложений и организация кода — чистая архитектура, шестиугольная архитектура, структура на основе функций, инверсия зависимостей, границы модулей, принципы SOLID в масштабе модуля. Активируйте «чистую архитектуру», «шестиугольную архитектуру», «функциональную структуру», «инверсию зависимостей», «границы модуля», «шаблон архитектуры», «структуру папок», «жирный контроллер», «круговую зависимость», «контейнер IoC». НЕ для декомпозиции микросервисов (используйте шаблоны микросервисов) или проектирования схемы базы данных (используйте шаблоны проектирования баз данных).
