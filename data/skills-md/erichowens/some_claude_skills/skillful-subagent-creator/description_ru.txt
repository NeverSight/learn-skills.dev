Создает субагентов Claude, обладающих тщательно подобранными навыками для решения проблем в рабочих процессах DAG. Используйте при разработке специализированного субагента, выборе навыков для субагента, написании приглашения субагента из 4 разделов или подключении субагентов к группам DAG оркестрации. Активируйте «Создать субагент», «Субагент с навыками», «Агент-специалист», «Агент DAG», «Управление агентами», «Агент с навыками». НЕ для создания самих навыков (используйте мастер-архитектор), общего использования кода Клода или подсказок одного агента без навыков.
