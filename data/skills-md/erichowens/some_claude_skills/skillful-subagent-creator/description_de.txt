Erstellt Claude-Subagenten, die mit kuratierten Fähigkeiten ausgestattet sind, um Probleme innerhalb von DAG-Workflows zu lösen. Verwenden Sie diese Option, wenn Sie einen speziellen Subagenten entwerfen, Fähigkeiten für einen Subagenten auswählen, die 4-teilige Subagenten-Eingabeaufforderung schreiben oder Subagenten in Orchestrierungs-DAGs vernetzen. Aktivieren Sie „Subagent erstellen“, „Subagent mit Fertigkeiten“, „Spezialagent“, „Agent-DAG“, „Agenten orchestrieren“, „mit Fertigkeiten ausgestatteter Agent“. NICHT für die Erstellung von Fertigkeiten selbst (verwenden Sie „skill-architect“), für die allgemeine Verwendung von Claude Code oder für die Eingabeaufforderung durch einen einzelnen Agenten ohne Fertigkeiten.
