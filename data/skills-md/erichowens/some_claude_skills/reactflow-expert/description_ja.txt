カスタム ノード、ELKjs 自動レイアウト、Zustand 状態管理、WebSocket 経由のライブ状態更新を備えた ReactFlow v12 を使用して DAG ビジュアライゼーションを構築します。ワークフロー視覚化ダッシュボードの実装、カスタム エージェント ノード コンポーネントの作成、ELK レイアウト アルゴリズムの統合、または React コンポーネントへの実行状態の接続時に使用します。 「ReactFlow」、「ワークフロー可視化」、「DAG可視化」、「ELKjs」、「カスタムノード」、「ノードベースエディタ」、「グラフ可視化」で有効化します。マーメイド ダイアグラムの作成 (マーメイド グラフ ライターを使用)、一般的な React 開発、または静的ダイアグラムのレンダリングには使用できません。
