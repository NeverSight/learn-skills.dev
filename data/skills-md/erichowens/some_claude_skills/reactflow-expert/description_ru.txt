Создает визуализации DAG с использованием ReactFlow v12 с настраиваемыми узлами, автоматическим макетом ELKjs, управлением состоянием Zustand и обновлением состояния в реальном времени через WebSocket. Используйте при реализации информационных панелей визуализации рабочих процессов, создании пользовательских компонентов узла агента, интеграции алгоритмов макета ELK или подключении состояния выполнения к компонентам React. Активируйте «ReactFlow», «визуализацию рабочего процесса», «визуализацию DAG», «ELKjs», «пользовательские узлы», «редактор на основе узлов», «визуализацию графа». НЕ для написания диаграмм Mermaid (используйте mermaid-graph-writer), общей разработки React или рендеринга статических диаграмм.
