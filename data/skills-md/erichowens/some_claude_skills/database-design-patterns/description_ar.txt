تصميم المخطط وقرارات التطبيع واستراتيجيات الفهرسة وسلامة الترحيل لقواعد البيانات العلائقية. يُستخدم عند تصميم الجداول، أو الاختيار بين التطبيع مقابل إلغاء التطبيع، أو تحديد أنواع الفهرس، أو التخطيط لعمليات الترحيل بدون توقف، أو إعداد تجمع الاتصالات. التنشيط على "تصميم المخطط"، "التطبيع"، "إلغاء التطبيع"، "المفتاح الخارجي"، "استراتيجية الفهرس"، "الترحيل"، "تجمع الاتصالات"، "المفتاح المركب"، "المفتاح البديل"، "الحذف الناعم"، "الارتباط متعدد الأشكال". ليس لضبط الاستعلام الخاص بـ PostgreSQL أو تحليل الشرح (استخدم تحسين postgresql)، أو NoSQL أو تصميم قاعدة بيانات المستندات، أو توفير قاعدة البيانات السحابية.
