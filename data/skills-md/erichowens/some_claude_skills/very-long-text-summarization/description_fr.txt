Résume des textes très longs (livres, manuels, biographies, bases de code) en utilisant une extraction hiérarchique multi-passes avec des armées modèles bon marché. Produit des cartes de connaissances structurées, pas seulement des résumés. À utiliser lors du traitement de documents de plus de 50 pages, de manuels professionnels, de biographies de carrière ou de tout texte trop volumineux pour une seule fenêtre contextuelle. Activez sur « résumer le livre », « résumer le manuel », « document long », « extraire les connaissances », « distiller le texte », « biographie professionnelle ». PAS pour le résumé de texte court (&lt;10 pages), le résumé de chat en temps réel ou la documentation de code (utilisez un rédacteur technique).
