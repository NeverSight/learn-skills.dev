قم بدمج ذاكرة Nebula المستمرة في التطبيقات باستخدام مجموعات SDK الرسمية لـ Python أو JavaScript/TypeScript أو REST API أو MCP. يُستخدم عندما: (1) إنشاء تطبيقات تقوم بتخزين الذكريات والبحث فيها (روبوتات الدردشة، وRAG، وقواعد المعرفة)، (2) إعداد تكامل MCP لمساعدي الذكاء الاصطناعي (Claude Code، Cursor، VS Code، Claude Desktop، Windsurf)، (3) العمل مع مجموعات Nebula، أو المحادثات، أو تحميلات المستندات، أو البحث الدلالي، أو تصفية البيانات الوصفية، (4) استيراد التعليمات البرمجية nebula أو @nebula-ai/sdk، (5) إشارات المستخدم Nebula Memory API أو Trynebula.ai.
