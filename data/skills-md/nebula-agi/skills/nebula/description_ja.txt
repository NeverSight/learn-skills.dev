公式の Python または JavaScript/TypeScript SDK、REST API、または MCP サーバーを使用して、Nebula 永続メモリをアプリケーションに統合します。次の場合に使用します: (1) メモリを保存および検索するアプリの構築 (チャットボット、RAG、ナレッジ ベース)、(2) AI アシスタント用の MCP 統合のセットアップ (Claude Code、Cursor、VS Code、Claude Desktop、Windsurf)、(3) Nebula コレクション、会話、ドキュメントのアップロード、セマンティック検索、またはメタデータ フィルタリングの操作、(4) コード インポート nebula または @nebula-ai/sdk、(5) ユーザーのメンションNebula メモリ API または trynebula.ai。
