Integre la memoria persistente de Nebula en aplicaciones utilizando los SDK oficiales de Python o JavaScript/TypeScript, la API REST o el servidor MCP. Úselo cuando: (1) cree aplicaciones que almacenen y busquen recuerdos (chatbots, RAG, bases de conocimiento), (2) configure la integración MCP para asistentes de IA (Claude Code, Cursor, VS Code, Claude Desktop, Windsurf), (3) trabaje con colecciones, conversaciones, carga de documentos, búsqueda semántica o filtrado de metadatos de Nebula, (4) importe de código nebula o @nebula-ai/sdk, (5) el usuario mencione la API de memoria de Nebula o trynebula.ai.
