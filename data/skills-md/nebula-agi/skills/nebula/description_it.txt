Integra la memoria persistente Nebula nelle applicazioni utilizzando gli SDK Python o JavaScript/TypeScript ufficiali, l'API REST o il server MCP. Utilizzare quando: (1) creare app che archiviano e ricercano ricordi (chatbot, RAG, basi di conoscenza), (2) impostare l'integrazione MCP per gli assistenti AI (Claude Code, Cursor, VS Code, Claude Desktop, Windsurf), (3) lavorare con raccolte Nebula, conversazioni, caricamenti di documenti, ricerca semantica o filtro dei metadati, (4) il codice importa nebula o @nebula-ai/sdk, (5) l'utente menziona l'API di memoria Nebula o trynebula.ai.
