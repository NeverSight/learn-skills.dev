Быстрый доступ к актуальной документации библиотеки с помощью MCP. Используйте этот навык, когда вам нужно обратиться к официальной документации по библиотекам, платформам или API. Использует сервер context7 MCP для получения текущей документации для React, Next.js, Vue, MongoDB, Supabase и сотен других библиотек. Дополняет агент ведения документации.
