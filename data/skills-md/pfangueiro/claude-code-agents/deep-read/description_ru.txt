Комплексный механизм чтения кодовой базы. Систематически считывает фактический исходный код построчно с помощью 6-фазного протокола — определение области действия, структурное сопоставление, отслеживание выполнения, глубокое чтение, синтез шаблонов и структурированные отчеты. Исходный код — источник истины. Используйте, когда нужно по-настоящему понять, как работает код, а не только то, что утверждает документация.
