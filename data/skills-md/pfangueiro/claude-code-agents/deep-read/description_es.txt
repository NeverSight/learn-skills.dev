Motor completo de lectura de base de código. Lee sistemáticamente el código fuente real línea por línea a través de un protocolo de 6 fases: alcance, mapeo estructural, seguimiento de ejecución, lectura profunda, síntesis de patrones e informes estructurados. El código fuente es la fuente de la verdad. Úselo cuando necesite comprender realmente cómo funciona el código, no solo lo que afirma la documentación.
