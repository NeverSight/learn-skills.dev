Meilleures pratiques et modèles de workflow Git. Utilisez cette compétence lorsque vous travaillez avec des opérations git, créez des commits, gérez des branches, gérez des demandes d'extraction ou établissez des flux de travail git d'équipe. Fournit des conseils sur les messages de validation, les stratégies de branchement et les modèles de collaboration.
