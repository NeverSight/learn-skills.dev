Механизм глубокого анализа первопричин. Систематически исследует ошибки, сбои, неожиданное поведение и проблемы с производительностью с помощью 8-фазного диагностического протокола. Использует структурированное мышление (MCP с последовательным мышлением), многопроходной анализ кодовой базы, криминалистика git, проверку гипотез на основе фактических данных и метод «5 почему». Никогда не переходите сразу к решению проблемы — всегда сначала доказывает первопричину.
