Комплексные рекомендации по UI/UX для создания компонентов React/Next.js с помощью Ant Design, диаграмм shadcn/ui и единообразного стиля. Используйте при создании форм, таблиц, модальных окон, карточек или любого компонента пользовательского интерфейса. Обеспечивает цветовую палитру, типографику, интервалы (8 пикселей/12 пикселей/16 пикселей/24 пикселей), анимацию и шаблоны компонентов, специфичные для приложения.
