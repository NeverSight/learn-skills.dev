Motore di esecuzione delle attività orchestrate. Decompone qualsiasi obiettivo in piccole attività atomiche, pianifica le dipendenze, seleziona l'agente/strumento/server MCP giusto per ciascuno, viene eseguito in batch paralleli in modo ottimale e tiene traccia di tutto. Da utilizzare quando viene assegnato un obiettivo complesso in più passaggi che trae vantaggio dalla scomposizione strutturata e dall'utilizzo completo dello strumento.
