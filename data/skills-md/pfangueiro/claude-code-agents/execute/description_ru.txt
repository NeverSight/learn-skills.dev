Организованный механизм выполнения задач. Разбивает любую цель на небольшие атомарные задачи, планирует зависимости, выбирает для каждой подходящий агент/инструмент/сервер MCP, выполняет оптимально параллельные пакеты и отслеживает все. Используйте, когда задана сложная, многоэтапная цель, которая выигрывает от структурированной декомпозиции и полного использования инструментов.
