Motor de ejecución de tareas orquestadas. Descompone cualquier objetivo en pequeñas tareas atómicas, planifica las dependencias, selecciona el agente/herramienta/servidor MCP adecuado para cada uno, lo ejecuta en lotes paralelos de manera óptima y realiza un seguimiento de todo. Úselo cuando se le asigne un objetivo complejo de varios pasos que se beneficie de la descomposición estructurada y la utilización completa de la herramienta.
