Analyse approfondie des problèmes de code et localisation des causes profondes. Utilisé pour analyser le comportement anormal du code, les problèmes de flux d'exécution, les erreurs de configuration et d'autres problèmes techniques. Lorsque les utilisateurs doivent : 1) analyser le flux d'exécution du code pour trouver la cause première, 2) comprendre la logique du code complexe et les chaînes d'appels, 3) localiser la cause d'un comportement anormal, 4) résoudre les problèmes de configuration ou d'état. Mots déclencheurs : localisation du problème, analyse du problème, analyse du problème, analyse du code, analyse des causes profondes.
