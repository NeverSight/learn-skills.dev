Глубокий анализ проблем с кодом и локализация первопричин. Используется для анализа аномального поведения кода, проблем с потоком выполнения, ошибок конфигурации и других технических проблем. Когда пользователям необходимо: 1) проанализировать поток выполнения кода, чтобы найти основную причину проблемы, 2) понять сложную логику кода и цепочки вызовов, 3) определить причину ненормального поведения, 4) устранить проблемы с конфигурацией или состоянием. Триггерные слова: локализация проблемы, анализ проблемы, анализ проблемы, анализ кода, анализ первопричин.
