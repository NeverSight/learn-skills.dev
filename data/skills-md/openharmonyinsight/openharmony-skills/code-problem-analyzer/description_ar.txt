تحليل مشكلة التعليمات البرمجية العميقة وتوطين السبب الجذري. يُستخدم لتحليل سلوك التعليمات البرمجية غير الطبيعي ومشكلات تدفق التنفيذ وأخطاء التكوين والمشكلات الفنية الأخرى. عندما يحتاج المستخدمون إلى: 1) تحليل تدفق تنفيذ التعليمات البرمجية للعثور على السبب الجذري، 2) فهم منطق التعليمات البرمجية المعقدة وسلاسل الاتصال، 3) تحديد سبب السلوك غير الطبيعي، 4) استكشاف مشكلات التكوين أو الحالة وإصلاحها. الكلمات المحفزة: توطين المشكلة، تحليل المشكلة، تحليل المشكلة، تحليل الكود، تحليل السبب الجذري.
