OpenHarmony/Chromium Deep Code Audit-Fähigkeiten – konzentriert sich auf Thread-Sicherheit, Lifetime-Management und Logikfehler

Basierend auf umfassender Programmiererfahrung auf Systemebene prüfen Sie den C/C++-Quellcode eingehend auf Folgendes:
- Thread-Sicherheitsprobleme (Datenrennen, Deadlocks, sequenzübergreifender Zugriff)
- Lebenslange Probleme (UAF, baumelnde Zeiger, WeakPtr-Missbrauch)
- OpenHarmony/Chromium-spezifische Musterrisiken
- Sicherheitsrisiken höherer Ordnung, die von statischen Analysetools nicht erkannt werden können

Kernfunktionen:
- Extrem strenger Rezensionsstil
- Einhaltung von 14 strengen benutzerdefinierten Regeln
- Generierung strukturierter JSON- und Markdown-Berichte
- Problemtriage nach Teamverantwortung
