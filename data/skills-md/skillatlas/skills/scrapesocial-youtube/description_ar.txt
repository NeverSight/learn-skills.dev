يُستخدم عندما يريد المستخدم إجراء بحث على YouTube أو إرشادات سير العمل لإنشاء عملاء محتملين أو اكتشاف المؤثرين أو مراقبة العلامة التجارية أو تحليل المنافسين أو تحليلات المحتوى أو بحث الاتجاهات أو تحليل الجمهور، بما في ذلك فحص القناة أو مقاطع فيديو القناة أو مجموعة الأفلام القصيرة أو استرجاع تفاصيل الفيديو أو استخراج النص أو التعليق أو البحث عن الكلمات الرئيسية أو علامة التصنيف أو البحث عن السراويل القصيرة الرائجة أو تحليل قائمة التشغيل أو استرجاع منشورات المجتمع.
