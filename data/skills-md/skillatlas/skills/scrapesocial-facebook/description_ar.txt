يُستخدم عندما يريد المستخدم إجراء بحث على Facebook أو إرشادات سير العمل لإنشاء عملاء محتملين أو اكتشاف المؤثرين أو مراقبة العلامة التجارية أو تحليل المنافسين أو تحليلات المحتوى أو بحث الاتجاهات أو تحليل الجمهور، بما في ذلك عمليات تدقيق الصفحة أو الملف الشخصي أو استخراج المنشورات أو البكرات أو الصور أو التعليقات، ومراقبة المجموعة، واسترجاع النصوص، وتحليل إعلانات Facebook أو المعلنين.
