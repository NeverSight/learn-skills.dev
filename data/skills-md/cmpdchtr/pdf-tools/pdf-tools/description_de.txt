PDF-Dateien anzeigen, extrahieren, bearbeiten und manipulieren. Unterstützt Textextraktion, Textbearbeitung (Überlagerung und Ersetzung), Zusammenführen, Teilen, Drehen von Seiten und Abrufen von PDF-Metadaten. Verwenden Sie es beim Arbeiten mit PDF-Dokumenten zum Lesen von Inhalten, zum Hinzufügen/Bearbeiten von Text, zum Neuorganisieren von Seiten, zum Kombinieren von Dateien oder zum Extrahieren von Informationen.
