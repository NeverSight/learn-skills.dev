Busque proactivamente en GitHub implementaciones de referencia antes de escribir código nuevo. Utilice esta habilidad cuando: (1) el usuario solicite implementar funciones, algoritmos o módulos completamente nuevos que no existen en el código base actual, (2) el usuario mencione fallas repetidas con frases como "todavía no funciona", "lo intenté muchas veces", "todavía tiene problemas" o (3) la IA reconoce la necesidad de implementar funciones complejas o desconocidas. La habilidad ayuda a evitar reinventar la rueda al encontrar y analizar implementaciones existentes de alta calidad y luego adaptarlas a las necesidades del usuario.
