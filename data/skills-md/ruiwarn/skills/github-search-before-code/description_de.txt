Durchsuchen Sie GitHub proaktiv nach Referenzimplementierungen, bevor Sie neuen Code schreiben. Verwenden Sie diese Fähigkeit, wenn: (1) der Benutzer die Implementierung völlig neuer Funktionen, Algorithmen oder Module anfordert, die in der aktuellen Codebasis nicht vorhanden sind, (2) der Benutzer wiederholte Fehler mit Ausdrücken wie „funktioniert immer noch nicht“, „viele Male versucht“, „immer noch Probleme“ erwähnt oder (3) die KI die Notwendigkeit erkennt, unbekannte oder komplexe Funktionen zu implementieren. Die Fähigkeit trägt dazu bei, das Rad nicht neu erfinden zu müssen, indem sie vorhandene hochwertige Implementierungen findet, analysiert und sie dann an die Bedürfnisse der Benutzer anpasst.
