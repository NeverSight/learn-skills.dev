ابحث بشكل استباقي في GitHub عن التطبيقات المرجعية قبل كتابة التعليمات البرمجية الجديدة. استخدم هذه المهارة عندما: (1) يطلب المستخدم تنفيذ وظائف أو خوارزميات أو وحدات نمطية جديدة تمامًا غير موجودة في قاعدة التعليمات البرمجية الحالية، (2) يذكر المستخدم حالات الفشل المتكررة بعبارات مثل "لا يزال لا يعمل"، أو "حاول عدة مرات"، أو "لا تزال تواجه مشكلات"، أو (3) يتعرف الذكاء الاصطناعي على الحاجة إلى تنفيذ ميزات غير مألوفة أو معقدة. تساعد هذه المهارة على تجنب إعادة اختراع العجلة من خلال البحث عن التطبيقات الحالية عالية الجودة وتحليلها، ثم تكييفها مع احتياجات المستخدم.
