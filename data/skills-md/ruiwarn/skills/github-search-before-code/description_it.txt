Cerca in modo proattivo GitHub per implementazioni di riferimento prima di scrivere nuovo codice. Utilizza questa competenza quando: (1) l'utente richiede l'implementazione di funzionalità, algoritmi o moduli completamente nuovi che non esistono nella codebase corrente, (2) l'utente menziona ripetuti errori con frasi come "ancora non funziona", "ho provato molte volte", "ha ancora problemi" o (3) l'intelligenza artificiale riconosce la necessità di implementare funzionalità non familiari o complesse. Questa competenza aiuta a evitare di reinventare la ruota trovando e analizzando le implementazioni esistenti di alta qualità, quindi adattandole alle esigenze degli utenti.
