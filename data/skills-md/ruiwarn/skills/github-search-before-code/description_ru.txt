Прежде чем писать новый код, активно выполняйте поиск эталонных реализаций на GitHub. Используйте этот навык, когда: (1) пользователь запрашивает реализацию совершенно новых функций, алгоритмов или модулей, которых нет в текущей кодовой базе, (2) пользователь упоминает повторяющиеся сбои с такими фразами, как «все еще не работает», «пробовал много раз», «все еще есть проблемы» или (3) ИИ признает необходимость реализации незнакомых или сложных функций. Этот навык помогает избежать изобретения велосипеда путем поиска и анализа существующих высококачественных реализаций, а затем их адаптации к потребностям пользователей.
