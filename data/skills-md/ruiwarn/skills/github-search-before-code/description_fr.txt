Recherchez de manière proactive sur GitHub des implémentations de référence avant d'écrire du nouveau code. Utilisez cette compétence lorsque : (1) l'utilisateur demande l'implémentation de fonctionnalités, d'algorithmes ou de modules complètement nouveaux qui n'existent pas dans la base de code actuelle, (2) l'utilisateur mentionne des échecs répétés avec des expressions telles que "ne fonctionne toujours pas", "essayé plusieurs fois", "a toujours des problèmes", ou (3) l'IA reconnaît la nécessité d'implémenter des fonctionnalités inconnues ou complexes. Cette compétence permet d'éviter de réinventer la roue en recherchant et en analysant les implémentations existantes de haute qualité, puis en les adaptant aux besoins des utilisateurs.
