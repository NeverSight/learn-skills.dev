Автономная разработка и отладка в текущем рабочем дереве со строгой безопасностью git. Срабатывает, когда пользователь говорит «перейти к автоматической разработке» или просит запустить автоматическую разработку. Используйте для кодирования, тестирования, MCP/навыков (например, драматург), проверки облаков (Firestore/Storage/GCP), просмотра веб-страниц или развертывания действий GitHub для текущей ветки. Политика по умолчанию блокирует передачу данных dev/main и между ветвями, за узким исключением для явного обслуживания Skills-Hub на Main и проекта передачи PR. Запрашивайте необходимые учетные данные или значения сеанса посредством ввода команды.
