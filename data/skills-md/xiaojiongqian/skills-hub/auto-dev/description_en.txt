Autonomous development and debugging in the current worktree with strict git safety. Trigger when user says "go auto-dev" or asks to start auto-dev. Use for coding, testing, MCP/skills (e.g., Playwright), cloud inspection (Firestore/Storage/GCP), web browsing, or GitHub Actions deploys for the current branch. Default policy blocks dev/main and cross-branch pushes, with a narrow exception for explicit `skills-hub` maintenance on `main` and draft PR handoff. Prompt for required credentials or session values via command input.
