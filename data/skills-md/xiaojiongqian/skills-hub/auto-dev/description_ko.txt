엄격한 Git 안전성을 바탕으로 현재 작업 트리에서 자율적으로 개발 및 디버깅합니다. 사용자가 "go auto-dev"라고 말하거나 자동 개발을 시작하라고 요청할 때 트리거됩니다. 코딩, 테스트, MCP/기술(예: Playwright), 클라우드 검사(Firestore/Storage/GCP), 웹 브라우징 또는 현재 분기에 대한 GitHub Actions 배포에 사용합니다. 기본 정책은 'main'에 대한 명시적인 'skills-hub' 유지 관리 및 초안 PR 전달에 대한 제한된 예외를 제외하고 개발/기본 및 교차 지점 푸시를 차단합니다. 명령 입력을 통해 필수 자격 증명이나 세션 값을 묻는 메시지를 표시합니다.
