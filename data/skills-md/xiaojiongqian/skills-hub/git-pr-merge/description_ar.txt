قم بمراجعة طلب سحب GitHub والتحقق من صحته ودمجه في الفرع المستهدف باستخدام `gh` و`git`. يُستخدم عندما يطلب المستخدم دمج العلاقات العامة من طرف إلى طرف، وحذف الفرع المصدر اختياريًا، واستخدام شجرة عمل مؤقتة اختياريًا. تشغيل المراجعة الذاتية أولاً؛ في حالة العثور على مشكلات محظورة، أو تعارضات لم يتم حلها، أو عمليات فحص فاشلة، أو مستندات/اختبارات غير كافية، قم بالتعليق على PR وتوقف بدلاً من الدمج.
