Examinez, validez et fusionnez une demande d'extraction GitHub dans une branche cible avec « gh » et « git ». À utiliser lorsque l'utilisateur demande de fusionner un PR de bout en bout, de supprimer éventuellement la branche source et éventuellement d'utiliser un arbre de travail temporaire. Exécutez d’abord un examen autonome ; si des problèmes de blocage, des conflits non résolus, des vérifications échouées ou des documents/tests insuffisants, commentez le PR et arrêtez-vous au lieu de fusionner.
