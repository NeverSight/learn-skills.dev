Просмотрите, проверьте и объедините запрос на включение GitHub в целевую ветку с помощью `gh` и `git`. Используйте, когда пользователь запрашивает сквозное объединение PR, при необходимости удалить исходную ветку и при необходимости использовать временное рабочее дерево. Сначала запустите автономную проверку; если обнаружены проблемы с блокировкой, неразрешенные конфликты, неудачные проверки или недостаточное количество документации/тестов, прокомментируйте PR и остановитесь, а не объединяйтесь.
