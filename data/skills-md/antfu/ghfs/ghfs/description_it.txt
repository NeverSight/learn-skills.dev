Gestisce i file mirror locali di ghfs in `.ghfs/`, in particolare traducendo le istruzioni dell'utente in operazioni di esecuzione valide (`execute.md`, `execute.yml` o modifiche per elemento), eseguendo `ghfsexecu`/`ghfs sync` e convalidando le modifiche batch di problemi/PR. Da utilizzare quando le attività implicano la modifica di problemi/PR tramite artefatti `.ghfs`, la riconciliazione dello stato di sincronizzazione o l'applicazione di operazioni GitHub in coda.
