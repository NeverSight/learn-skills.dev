Guía de expertos para la validación del esquema Zod v4 en TypeScript. Úselo al diseñar esquemas, migrar desde Zod 3, manejar errores de validación, generar esquemas JSON/OpenAPI, usar códecs/transformaciones o integrar con React Hook Form, tRPC, Hono o Next.js. Cubre todas las API de Zod v4, incluidos formatos de cadena de nivel superior, estrictoObjeto/looseObject, metadatos, registros, tipos de marca y esquemas recursivos.
