使用 AI SDK React hook 构建聊天 UI 的专家指南。
使用场合：(1) 使用 useChat 钩子构建聊天机器人，
(2) 通过服务器/客户端执行来实现工具 UI，
(3) 处理消息持久化和流恢复，
(4) 使用 React 组件创建生成式 UI，
(5) 与 Next.js/Node/Fastify/Nest 后端集成，
(6) 使用useObject流式传输结构化数据。
触发器：useChat、useObject、useCompletion、聊天 UI、聊天机器人、生成式 UI、消息持久性、@ai-sdk/react、UIMessage、工具部件、流式 UI、toUIMessageStreamResponse。
