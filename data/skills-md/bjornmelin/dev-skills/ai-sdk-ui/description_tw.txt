使用 AI SDK React hook 建立聊天 UI 的專家指南。
使用場合：(1) 使用 useChat 鉤子建立聊天機器人，
(2) 透過伺服器/客戶端執行來實現工具 UI，
(3) 處理訊息持久化和流恢復，
(4) 使用 React 元件建立生成式 UI，
(5) 與 Next.js/Node/Fastify/Nest 後端集成，
(6) 使用useObject串流結構化資料。
觸發器：useChat、useObject、useCompletion、聊天 UI、聊天機器人、生成式 UI、訊息持久性、@ai-sdk/react、UIMessage、工具部件、串流 UI、toUIMessageStreamResponse。
