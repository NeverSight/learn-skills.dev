AI SDK React フックを使用してチャット UI を構築するための専門家によるガイダンス。
次の場合に使用します: (1) useChat フックを使用してチャットボットを構築する
(2) サーバー/クライアント実行によるツール UI の実装、
(3) メッセージの永続化とストリームの再開の処理、
(4) React コンポーネントを使用した生成 UI の作成、
(5) Next.js/Node/Fastify/Nest バックエンドとの統合、
(6) useObject を使用して構造化データをストリーミングします。
トリガー: useChat、useObject、useCompletion、チャット UI、チャットボット、生成 UI、メッセージ永続性、@ai-sdk/react、UIMessage、ツール パーツ、ストリーミング UI、toUIMessageStreamResponse。
