Автоматически классифицируйте помеченные репозитории GitHub в списки GitHub с помощью анализа ИИ. Поддерживает полный пакетный режим и выборочный режим (определенные репозитории или последние N звезд). Используйте, когда пользователь хочет упорядочить, классифицировать, отсортировать или очистить свои звезды GitHub или управлять списками GitHub. Триггерные фразы включают «организовать мои звезды», «классифицировать звезды», «整理 звезды», «звезды 分类», «gh-star-list».
