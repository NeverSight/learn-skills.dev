Classifica automaticamente i repository stellati GitHub negli elenchi GitHub utilizzando l'analisi AI. Supporta la modalità batch completa e la modalità selettiva (repository specifici o ultime N stelle). Da utilizzare quando l'utente desidera organizzare, classificare, ordinare o ripulire le proprie stelle GitHub o gestire elenchi GitHub. Le frasi trigger includono "organizza le mie stelle", "categorizza stelle", "stelle 整理", "stelle 分类", "gh-star-list".
