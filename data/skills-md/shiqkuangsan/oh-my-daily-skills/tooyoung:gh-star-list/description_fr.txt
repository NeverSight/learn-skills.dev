Classez automatiquement les référentiels favoris GitHub dans les listes GitHub à l'aide de l'analyse IA. Prend en charge le mode batch complet et le mode sélectif (repos spécifiques ou dernières étoiles N). À utiliser lorsque l'utilisateur souhaite organiser, catégoriser, trier ou nettoyer ses étoiles GitHub, ou gérer les listes GitHub. Les phrases de déclenchement incluent « organiser mes étoiles », « catégoriser les étoiles », « 整理 étoiles », « étoiles 分类 », « gh-star-list ».
