قم بتصنيف مستودعات GitHub المميزة بنجمة تلقائيًا إلى قوائم GitHub باستخدام تحليل الذكاء الاصطناعي. يدعم الوضع الدفعي الكامل والوضع الانتقائي (اتفاقيات إعادة الشراء المحددة أو أحدث نجوم N). يُستخدم عندما يريد المستخدم تنظيم نجوم GitHub أو تصنيفها أو فرزها أو تنظيفها، أو إدارة قوائم GitHub. تتضمن عبارات التشغيل "تنظيم نجومي"، و"تصنيف النجوم"، و"نجوم 整理"، و"نجوم 分类"، و"قائمة نجوم gh".
