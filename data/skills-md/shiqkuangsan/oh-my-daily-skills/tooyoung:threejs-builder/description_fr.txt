Crée des applications Web Three.js simples avec configuration de scène, éclairage, géométries, matériaux, animations et rendu réactif. À utiliser pour « Créer une scène/une application/une vitrine threejs » ou lorsque l'utilisateur souhaite du contenu Web 3D. Prend en charge les modules ES, les API modernes Three.js r170+. Déclencheur : scène 3D, WebGL, threejs, 三维展示, vitrine 3D, 3D interactive
