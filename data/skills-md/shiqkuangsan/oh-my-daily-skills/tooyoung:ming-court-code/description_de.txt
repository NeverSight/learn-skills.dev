Ming Court Code – strukturierter Entwicklungsworkflow, inspiriert vom Gerichtssystem der Ming-Dynastie. Drei adaptive Modi: mündlicher Beschluss (schnelle Ausführung), Gerichtsdebatte (strukturierter Plan), morgendliche Versammlung (Parallel mit mehreren Agenten). Auslöser: 明廷典, Ming Court, 开发流程, Dev-Workflow-Protokoll, 朝廷模式
