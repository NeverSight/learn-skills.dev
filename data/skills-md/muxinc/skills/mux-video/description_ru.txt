Комплексное руководство по созданию видеоприложений с помощью Mux, первой для разработчиков платформы видеоинфраструктуры. Этот навык охватывает потоковое видео, прямую трансляцию, интеграцию плееров, аналитику с помощью Mux Data и рабочие процессы на базе искусственного интеллекта. Независимо от того, создаете ли вы платформу видео по запросу, приложение для потоковой передачи в реальном времени или интегрируете видео в существующий продукт, в этой документации представлены шаблоны и примеры кода, необходимые для быстрой доставки.
