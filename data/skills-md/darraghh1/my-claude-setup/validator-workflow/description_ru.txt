Рабочий процесс проверки на уровне фазы для агентов проверки. Управляет загрузкой правил проекта, чтением фазовых файлов, запуском/проверкой кода, условной проверкой (модульные тесты + проверка типов + тесты E2E/DB на основе типа фазы), определением вердикта и отчетом оркестратору. Призовите этот навык в качестве своего первого действия, не вызываемого пользователем.
