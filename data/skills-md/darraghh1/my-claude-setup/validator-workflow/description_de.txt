Validierungsworkflow auf Phasenebene für Validatoragenten. Behandelt das Laden von Projektregeln, das Lesen von Phasendateien, das Ausführen von /code-review, die bedingte Überprüfung (Komponententests + Typprüfung + E2E/DB-Tests basierend auf dem Phasentyp), die Festlegung des Urteils und die Berichterstattung an den Orchestrator. Rufen Sie diesen Skill als Ihre erste Aktion auf – nicht vom Benutzer aufrufbar.
