Рабочий процесс реализации поэтапного уровня для агентов-строителей. Управляет чтением фазовых файлов, поиском ссылок, вызовом навыков предметной области, реализацией всех шагов и запуском проверки (тесты + проверка типов). Предварительно загружается в агенты компоновщика с помощью поля навыков: не вызывается пользователем.
