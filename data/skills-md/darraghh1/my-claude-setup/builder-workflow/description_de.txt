Implementierungsworkflow auf Phasenebene für Builder-Agenten. Behandelt das Lesen von Phasendateien, das Suchen von Referenzen, das Aufrufen von Domänenkenntnissen, das Implementieren aller Schritte und das Durchführen von Überprüfungen (Tests + Typprüfung). Über das Feld „skills:“ in Builder-Agents vorinstalliert – nicht vom Benutzer aufrufbar.
