Esta habilidad debe usarse cuando los usuarios necesitan trabajar con Expo SDK y Expo Router para crear aplicaciones React Native. Proporciona orientación integral sobre patrones de navegación, manejo de medios (cámara, imagen, video, audio), almacenamiento de datos (SQLite, sistema de archivos, SecureStore), autenticación, características del dispositivo (ubicación, notificaciones, datos biométricos) e integraciones de terceros (Stripe, vinculación, uso compartido).
