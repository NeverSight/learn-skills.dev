Этот навык следует использовать, когда пользователям необходимо работать с Expo SDK и Expo Router для создания приложений React Native. Он предоставляет комплексные рекомендации по шаблонам навигации, обработке мультимедиа (камера, изображение, видео, аудио), хранению данных (SQLite, файловая система, SecureStore), аутентификации, функциям устройства (местоположение, уведомления, биометрия) и сторонней интеграции (Stripe, связывание, совместное использование).
