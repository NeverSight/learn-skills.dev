Diese Fähigkeit sollte verwendet werden, wenn Benutzer zum Erstellen von React Native-Anwendungen mit Expo SDK und Expo Router arbeiten müssen. Es bietet umfassende Anleitungen zu Navigationsmustern, Medienhandhabung (Kamera, Bild, Video, Audio), Datenspeicherung (SQLite, Dateisystem, SecureStore), Authentifizierung, Gerätefunktionen (Standort, Benachrichtigungen, Biometrie) und Integrationen von Drittanbietern (Stripe, Verknüpfung, Freigabe).
