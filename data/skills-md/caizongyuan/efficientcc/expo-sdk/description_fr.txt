Cette compétence doit être utilisée lorsque les utilisateurs doivent travailler avec Expo SDK et Expo Router pour créer des applications React Native. Il fournit des conseils complets sur les modèles de navigation, la gestion des médias (caméra, image, vidéo, audio), le stockage de données (SQLite, système de fichiers, SecureStore), l'authentification, les fonctionnalités de l'appareil (localisation, notifications, biométrie) et les intégrations tierces (Stripe, liaison, partage).
