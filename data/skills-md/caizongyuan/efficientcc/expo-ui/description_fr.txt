Cette compétence doit être utilisée lorsque les utilisateurs doivent travailler avec Expo UI (@expo/ui) pour intégrer des composants SwiftUI dans des applications React Native. Il fournit des conseils complets sur l'installation, le composant hôte, les primitives SwiftUI, les mises en page, les modificateurs, la navigation par onglets natifs et les fonctionnalités d'aperçu v10.
