Этот навык следует использовать, когда пользователям необходимо работать с пользовательским интерфейсом Expo (@expo/ui) для интеграции компонентов SwiftUI в приложения React Native. Он содержит подробные инструкции по установке, компоненту хоста, примитивам SwiftUI, макетам, модификаторам, встроенной навигации по вкладкам и функциям предварительного просмотра версии 10.
