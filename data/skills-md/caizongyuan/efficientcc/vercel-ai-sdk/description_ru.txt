Этот навык следует использовать, когда пользователям необходимо работать с Vercel AI SDK для создания приложений на базе искусственного интеллекта. Он предоставляет исчерпывающие рекомендации по основным API (generateText,streamText), компонентам пользовательского интерфейса (useChat, useCompletion), вызову инструментов, генерации структурированных данных, управлению поставщиками, протоколам потоковой передачи и расширенным функциям, таким как промежуточное программное обеспечение и настраиваемые поставщики.
