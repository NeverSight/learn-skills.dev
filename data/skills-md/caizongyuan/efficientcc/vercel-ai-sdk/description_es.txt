Esta habilidad debe usarse cuando los usuarios necesitan trabajar con Vercel AI SDK para crear aplicaciones impulsadas por AI. Proporciona orientación completa sobre las API principales (generateText, streamText), componentes de la interfaz de usuario (useChat, useCompletion), llamadas de herramientas, generación de datos estructurados, gestión de proveedores, protocolos de transmisión y funciones avanzadas como middleware y proveedores personalizados.
