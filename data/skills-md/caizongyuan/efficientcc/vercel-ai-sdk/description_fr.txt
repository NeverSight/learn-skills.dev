Cette compétence doit être utilisée lorsque les utilisateurs doivent travailler avec le SDK Vercel AI pour créer des applications basées sur l'IA. Il fournit des conseils complets sur les API de base (generateText, streamText), les composants d'interface utilisateur (useChat, useCompletion), les appels d'outils, la génération de données structurées, la gestion des fournisseurs, les protocoles de streaming et les fonctionnalités avancées telles que les middlewares et les fournisseurs personnalisés.
