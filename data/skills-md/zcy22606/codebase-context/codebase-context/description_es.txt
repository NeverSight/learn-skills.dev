Analice la base de código de un proyecto desde cero y genere archivos contextuales fáciles de usar para los agentes (CLAUDE.md, documentos de arquitectura, documentos de pila tecnológica, configuraciones específicas de herramientas para 7 herramientas de IA). Úselo cuando (1) inicialice el contexto de IA para un proyecto existente, (2) el usuario diga "generar el contexto del proyecto", "prepare el proyecto para la IA", "inicialice el contexto de la IA", (3) incorpore un nuevo repositorio para el desarrollo asistido por IA, (4) el equipo quiere un contexto de IA consistente entre todos los desarrolladores.
