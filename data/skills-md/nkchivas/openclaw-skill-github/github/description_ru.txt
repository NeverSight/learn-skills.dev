Операции GitHub через CLI `gh`: проблемы, запросы на запросы, CI-прогоны, проверка кода, запросы API. Используйте при: (1) проверке статуса PR или CI, (2) создании/комментировании проблем, (3) составлении списка/фильтрации PR или проблем, (4) просмотре журналов выполнения. НЕ для: сложных взаимодействий с веб-интерфейсом, требующих ручных операций браузера (используйте инструменты браузера, если они доступны), массовых операций во многих репозиториях (скрипт с gh API) или когда gh auth не настроен.
