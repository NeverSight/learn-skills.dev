Operazioni GitHub tramite CLI `gh`: problemi, PR, esecuzioni CI, revisione del codice, query API. Da utilizzare quando: (1) si controlla lo stato PR o CI, (2) si creano/commentano problemi, (3) si elencano/filtrano PR o problemi, (4) si visualizzano i registri di esecuzione. NON per: interazioni complesse dell'interfaccia utente Web che richiedono flussi manuali del browser (utilizzare gli strumenti del browser quando disponibili), operazioni in blocco su più repository (script con gh api) o quando gh auth non è configurato.
