Obtenez la météo et les prévisions actuelles via wttr.in ou Open-Meteo. À utiliser lorsque : l'utilisateur pose des questions sur la météo, la température ou les prévisions pour n'importe quel endroit. PAS pour : les données météorologiques historiques, les alertes de temps violent ou les analyses météorologiques détaillées. Aucune clé API nécessaire.
