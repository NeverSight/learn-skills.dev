Framework di blog statico Valaxy creato con spazi di lavoro Vue 3, Vite e pnpm. Da utilizzare quando si lavora con progetti Valaxy: (1) creazione o configurazione di siti blog Valaxy (site.config.ts, valaxy.config.ts), (2) scrittura di post di blog con frontmatter, (3) sviluppo di temi Valaxy (valaxy-theme-*), (4) sviluppo di componenti aggiuntivi Valaxy (valaxy-addon-*), (5) personalizzazione di componenti/layout/stili, (6) utilizzo del framework principale Valaxy monorepo, (7) eseguire il debug delle build Valaxy o del server di sviluppo, (8) distribuire i siti Valaxy. Si attiva su: valaxy.config.ts, site.config.ts, theme.config.ts, defineValaxyConfig, defineSiteConfig, defineThemeConfig, defineTheme, defineAddon, valaxy-theme-*, valaxy-addon-*, pagine/posts/, ValaxyMain, useSiteConfig, useFrontmatter, #valaxy/config.
