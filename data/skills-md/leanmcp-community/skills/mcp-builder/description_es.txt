Cree servidores MCP (Protocolo de contexto modelo) utilizando el @modelcontextprotocol/sdk oficial. Utilice esta habilidad cuando los usuarios soliciten crear un servidor MCP, herramientas MCP, recursos MCP o quieran exponer API/funcionalidad a agentes de IA. Esta habilidad cubre la implementación básica de MCP con transporte HTTP Streamable, administración de sesiones, herramientas, recursos e indicaciones.
