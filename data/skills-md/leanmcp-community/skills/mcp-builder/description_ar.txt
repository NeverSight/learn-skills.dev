أنشئ خوادم MCP (بروتوكول سياق النموذج) باستخدام @modelcontextprotocol/sdk الرسمي. استخدم هذه المهارة عندما يطلب المستخدمون إنشاء خادم MCP، أو أدوات MCP، أو موارد MCP، أو عندما يريدون كشف واجهات برمجة التطبيقات/الوظائف لوكلاء الذكاء الاصطناعي. تغطي هذه المهارة تنفيذ Vanilla MCP مع نقل HTTP القابل للتدفق وإدارة الجلسة والأدوات والموارد والمطالبات.
