Создайте серверы MCP (Model Context Protocol), используя официальный @modelcontextprotocol/sdk. Используйте этот навык, когда пользователи просят создать сервер MCP, инструменты MCP, ресурсы MCP или хотят предоставить API/функции агентам ИИ. Этот навык охватывает реализацию стандартного MCP с потоковым HTTP-транспортом, управление сеансами, инструменты, ресурсы и подсказки.
