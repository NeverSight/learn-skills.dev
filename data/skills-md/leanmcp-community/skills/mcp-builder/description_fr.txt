Créez des serveurs MCP (Model Context Protocol) à l'aide du @modelcontextprotocol/sdk officiel. Utilisez cette compétence lorsque les utilisateurs demandent à créer un serveur MCP, des outils MCP, des ressources MCP ou souhaitent exposer des API/fonctionnalités aux agents IA. Cette compétence couvre la mise en œuvre de MCP Vanilla avec le transport HTTP Streamable, la gestion de session, les outils, les ressources et les invites.
