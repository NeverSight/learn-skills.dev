Costruisci server MCP (Model Context Protocol) utilizzando il @modelcontextprotocol/sdk ufficiale. Utilizza questa competenza quando gli utenti chiedono di creare un server MCP, strumenti MCP, risorse MCP o desiderano esporre API/funzionalità agli agenti AI. Questa competenza copre l'implementazione MCP vanilla con trasporto HTTP streaming, gestione delle sessioni, strumenti, risorse e istruzioni.
