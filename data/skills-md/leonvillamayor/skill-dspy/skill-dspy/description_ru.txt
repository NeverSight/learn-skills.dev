Экспертное руководство по созданию программ искусственного интеллекта с помощью DSPy — декларативной среды для программирования LM с автоматической оптимизацией подсказок. Используйте этот навык ПРОАКТИВНО всякий раз, когда: импортируете dspy, используете сигнатуры/модули/оптимизаторы, строите RAG/агентные/многоузловые конвейеры, оптимизируете с помощью BootstrapFewShot/MIPROv2/COPRO/SIMBA/GEPA/BetterTogether, пишете dspy.ChainOfThought/ReAct/Predict/CodeAct, оцениваете с помощью dspy.Evaluate, используете dspy.Refine/BestOfN для обеспечения качества вывода, настройки любого поставщика LM (OpenAI/Anthropic/Gemini/Ollama/модели рассуждения), сохранения/загрузки скомпилированных программ, интеграции инструментов MCP (stdio или HTTP), потоковой передачи, асинхронных модулей, отслеживания использования токенов или отладки с помощью Inspect_history. Охватывает: конфигурацию LM, сигнатуры, модули, оптимизаторы, оценку, уточнение/BestOfN, инструменты, адаптеры, потоковую передачу, асинхронность, обратные вызовы, внедрения и сохранение/загрузку.
