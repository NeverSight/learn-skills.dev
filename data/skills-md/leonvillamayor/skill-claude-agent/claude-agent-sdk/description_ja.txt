Claude Agent SDK (Python および TypeScript) を使用して AI エージェントを構築するためのエキスパート ガイド。
次のような場合には、このスキルを積極的に使用してください。
- `claude_agent_sdk` または `@anthropic-ai/claude-agent-sdk` を使用した AI エージェントの構築または構成
- `query()`、`ClaudeAgentOptions`、`AgentDefinition`、フック、セッション、サブエージェント、MCP、またはカスタム ツールの操作
- ユーザーは、タスクを自動化する方法、パイプラインを構築する方法、またはクロードとマルチエージェント ワークフローを調整する方法を尋ねます。
- 権限制御、「canUseTool」コールバック、またはツール制限の実装
- MCP 経由で外部サービス (データベース、API、ブラウザ) を Claude エージェントに接続する
- エージェントの動作、ストリーミング、セッション、またはフックのデバッグまたは最適化
- 「クロード エージェント」、「エージェント SDK」、「エージェント ループ」、または自律型クロード ワークフローに関する言及
完全な SDK をカバーします: クエリ API、組み込みツール、フックのライフサイクル、セッション管理、サブエージェント、
MCP サーバー、カスタム インプロセス ツール、権限モード、ユーザー入力フロー、構造化出力、
思考構成、エフォートレベル、V2 プレビュー API、セッション検出、エラー処理、認証。
