Экспертное руководство по созданию агентов ИИ с помощью Claude Agent SDK (Python и TypeScript).
Используйте этот навык ПРОАКТИВНО всякий раз, когда:
- Создание или настройка агентов ИИ с помощью `claude_agent_sdk` или `@anthropic-ai/claude-agent-sdk`
- Работа с `query()`, `ClaudeAgentOptions`, `AgentDefinition`, хуками, сеансами, субагентами, MCP или пользовательскими инструментами.
- Пользователь спрашивает, как автоматизировать задачи, построить конвейеры или организовать многоагентные рабочие процессы с помощью Claude.
- Реализация контроля разрешений, обратных вызовов canUseTool или ограничений инструментов.
- Подключение внешних сервисов (баз данных, API, браузеров) к агенту Claude через MCP.
- Отладка или оптимизация поведения агента, потоковой передачи, сеансов или перехватчиков.
- Любое упоминание об «агенте Claude», «агентском SDK», «агентном цикле» или автономных рабочих процессах Claude.
Охватывает полный пакет SDK: API запросов, встроенные инструменты, жизненный цикл перехватчиков, управление сеансами, субагенты,
Серверы MCP, пользовательские внутрипроцессные инструменты, режимы разрешений, потоки пользовательского ввода, структурированные выходные данные,
конфигурация мышления, уровни усилий, API предварительного просмотра V2, обнаружение сеансов, обработка ошибок и аутентификация.
