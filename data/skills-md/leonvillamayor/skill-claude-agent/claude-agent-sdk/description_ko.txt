Claude Agent SDK(Python 및 TypeScript)를 사용하여 AI 에이전트를 구축하기 위한 전문가 가이드입니다.
다음과 같은 경우에 이 기술을 적극적으로 사용하십시오.
- `claude_agent_sdk` 또는 `@anthropic-ai/claude-agent-sdk`를 사용하여 AI 에이전트 구축 또는 구성
- `query()`, `ClaudeAgentOptions`, `AgentDefinition`, 후크, 세션, 하위 에이전트, MCP 또는 사용자 정의 도구 작업
- 사용자가 Claude를 사용하여 작업을 자동화하고, 파이프라인을 구축하고, 다중 에이전트 워크플로를 조정하는 방법을 묻습니다.
- 권한 제어, `canUseTool` 콜백 또는 도구 제한 구현
- MCP를 통해 외부 서비스(데이터베이스, API, 브라우저)를 Claude 에이전트에 연결
- 에이전트 동작, 스트리밍, 세션 또는 후크 디버깅 또는 최적화
- "Claude 에이전트", "에이전트 SDK", "에이전트 루프" 또는 자율 Claude 워크플로에 대한 언급
전체 SDK 포함: 쿼리 API, 내장 도구, 후크 수명 주기, 세션 관리, 하위 에이전트,
MCP 서버, 사용자 정의 프로세스 내 도구, 권한 모드, 사용자 입력 흐름, 구조화된 출력,
사고 구성, 노력 수준, V2 미리보기 API, 세션 검색, 오류 처리 및 인증.
