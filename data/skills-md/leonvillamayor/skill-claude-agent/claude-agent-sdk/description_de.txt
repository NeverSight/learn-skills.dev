Expertenhandbuch zum Erstellen von KI-Agenten mit dem Claude Agent SDK (Python & TypeScript).
Nutzen Sie diese Fähigkeit PROAKTIV, wann immer:
- Erstellen oder Konfigurieren von KI-Agenten mit „claude_agent_sdk“ oder „@anthropic-ai/claude-agent-sdk“.
- Arbeiten mit „query()“, „ClaudeAgentOptions“, „AgentDefinition“, Hooks, Sitzungen, Subagenten, MCP oder benutzerdefinierten Tools
– Der Benutzer fragt, wie er mit Claude Aufgaben automatisieren, Pipelines erstellen oder Multi-Agent-Workflows orchestrieren kann
- Implementierung von Berechtigungskontrollen, „canUseTool“-Rückrufen oder Tool-Einschränkungen
- Anbindung externer Dienste (Datenbanken, APIs, Browser) an einen Claude-Agenten über MCP
- Debuggen oder Optimieren von Agentenverhalten, Streaming, Sitzungen oder Hooks
– Jegliche Erwähnung von „Claude Agent“, „Agent SDK“, „Agentic Loop“ oder autonomen Claude-Workflows
Deckt das gesamte SDK ab: Abfrage-API, integrierte Tools, Hooks-Lebenszyklus, Sitzungsverwaltung, Subagenten,
MCP-Server, benutzerdefinierte In-Process-Tools, Berechtigungsmodi, Benutzereingabeflüsse, strukturierte Ausgaben,
Denkkonfiguration, Aufwandsstufen, V2-Vorschau-API, Sitzungserkennung, Fehlerbehandlung und Authentifizierung.
