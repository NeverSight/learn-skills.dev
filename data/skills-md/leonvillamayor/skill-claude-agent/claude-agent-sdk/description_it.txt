Guida esperta per la creazione di agenti AI con Claude Agent SDK (Python e TypeScript).
Usa questa abilità in modo PROATTIVO ogni volta che:
- Costruire o configurare agenti AI con `claude_agent_sdk` o `@anthropic-ai/claude-agent-sdk`
- Lavorare con `query()`, `ClaudeAgentOptions`, `AgentDefinition`, hook, sessioni, subagenti, MCP o strumenti personalizzati
- L'utente chiede come automatizzare le attività, creare pipeline o orchestrare flussi di lavoro multi-agente con Claude
- Implementazione di controlli dei permessi, callback "canUseTool" o restrizioni sugli strumenti
- Connessione di servizi esterni (database, API, browser) a un agente Claude tramite MCP
- Debug o ottimizzazione del comportamento, dello streaming, delle sessioni o degli hook dell'agente
- Qualsiasi menzione di "agente Claude", "SDK agente", "loop agenti" o flussi di lavoro Claude autonomi
Copre l'intero SDK: API di query, strumenti integrati, ciclo di vita degli hook, gestione delle sessioni, agenti secondari,
Server MCP, strumenti in-process personalizzati, modalità di autorizzazione, flussi di input dell'utente, output strutturati,
configurazione di pensiero, livelli di impegno, API di anteprima V2, rilevamento della sessione, gestione degli errori e autenticazione.
