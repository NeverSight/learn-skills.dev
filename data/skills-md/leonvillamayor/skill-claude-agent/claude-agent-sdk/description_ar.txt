دليل الخبراء لبناء وكلاء الذكاء الاصطناعي باستخدام Claude Agent SDK (Python & TypeScript).
استخدم هذه المهارة بشكل استباقي كلما:
- إنشاء وكلاء الذكاء الاصطناعي أو تكوينهم باستخدام `clude_agent_sdk` أو `@anthropic-ai/claude-agent-sdk`
- العمل مع `query()` أو `ClaudeAgentOptions` أو `AgentDefinition` أو الخطافات أو الجلسات أو الوكلاء الفرعيين أو MCP أو الأدوات المخصصة
- يسأل المستخدم عن كيفية أتمتة المهام أو بناء خطوط الأنابيب أو تنسيق سير العمل متعدد الوكلاء مع Claude
- تنفيذ عناصر التحكم في الأذونات، أو عمليات رد الاتصال "canUseTool"، أو قيود الأداة
- ربط الخدمات الخارجية (قواعد البيانات، واجهات برمجة التطبيقات، المتصفحات) بوكيل كلود عبر MCP
- تصحيح أو تحسين سلوك الوكيل أو التدفق أو الجلسات أو الخطافات
- أي ذكر لـ "وكيل Claude"، أو "agent SDK"، أو "agentic Loop"، أو سير عمل Claude المستقل
يغطي SDK الكامل: واجهة برمجة تطبيقات الاستعلام، والأدوات المضمنة، ودورة حياة الخطافات، وإدارة الجلسة، والوكلاء الفرعيين،
خوادم MCP، والأدوات المخصصة قيد التشغيل، وأوضاع الأذونات، وتدفقات إدخال المستخدم، والمخرجات المنظمة،
تكوين التفكير ومستويات الجهد وواجهة برمجة تطبيقات معاينة V2 واكتشاف الجلسة ومعالجة الأخطاء والمصادقة.
