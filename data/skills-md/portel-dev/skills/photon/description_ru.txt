Создайте Photon MCP — однофайловые серверы TypeScript MCP. Используется для создания фотонов с аннотациями @format (таблица, диаграмма: полоса), фотонов с отслеживанием состояния с использованием this.memory для постоянного хранения, фотонов с аннотациями @readOnly/@destructive, пользовательского интерфейса с использованием тегов @ui/шаблонов HTML, API-интерфейсов упаковки фотонов (Stripe, Payments), фотонов планировщика задач с cron, диаграмм русалок для фотонной архитектуры, редактирования файлов .photon.ts. НЕ запускайте триггер для общего TypeScript или нефотонного MCP.
