Créez des MCP Photon – des serveurs TypeScript MCP à fichier unique. À utiliser pour créer des photons avec des annotations @format (tableau, graphique: barre), des photons avec état utilisant this.memory pour le stockage persistant, des photons avec des annotations @readOnly/@destructive, une interface utilisateur personnalisée utilisant des balises @ui/modèles HTML, des API d'encapsulation de photons (Stripe, paiements), des photons du planificateur de tâches avec cron, des diagrammes de sirène pour l'architecture de photons, l'édition de fichiers .photon.ts. NE PAS déclencher pour TypeScript général ou MCP non photonique.
