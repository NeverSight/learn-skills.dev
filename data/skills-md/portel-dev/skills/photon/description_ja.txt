Photon MCP を構築する — 単一ファイルの TypeScript MCP サーバー。 @format アノテーション (table、chart:bar) を使用したフォトン、永続ストレージに this.memory を使用したステートフル フォトン、@readOnly/@destructive アノテーションを使用したフォトン、@ui タグ/HTML テンプレートを使用したカスタム UI、フォトン ラッピング API (ストライプ、支払い)、cron を使用したタスク スケジューラ フォトン、フォトン アーキテクチャ用のマーメイド ダイアグラム、.photon.ts ファイルの編集に使用します。一般的な TypeScript または非フォトン MCP に対してトリガーしないでください。
