Costruisci MCP Photon: server MCP TypeScript a file singolo. Utilizzare per creare fotoni con annotazioni @format (tabella, grafico:bar), fotoni con stato utilizzando this.memory per l'archiviazione persistente, fotoni con annotazioni @readOnly/@destructiva, interfaccia utente personalizzata utilizzando tag @ui/modelli HTML, API di wrapper di fotoni (Stripe, pagamenti), fotoni di pianificazione delle attività con cron, diagrammi a sirena per l'architettura di fotoni, modifica di file .photon.ts. NON attivare per TypeScript generale o MCP non fotonico.
