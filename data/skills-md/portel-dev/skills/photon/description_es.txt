Cree Photon MCP: servidores MCP TypeScript de un solo archivo. Úselo para crear fotones con anotaciones @format (tabla, gráfico:barra), fotones con estado usando this.memory para almacenamiento persistente, fotones con anotaciones @readOnly/@destructivas, interfaz de usuario personalizada usando etiquetas @ui/plantillas HTML, API de ajuste de fotones (Stripe, pagos), fotones del programador de tareas con cron, diagramas de sirena para arquitectura de fotones, edición de archivos .photon.ts. NO dispare para TypeScript general o MCP sin fotones.
