Authentifizieren Sie sich mit einem beliebigen OAuth 2.1 MCP-Server. Verwaltet Erkennung, PKCE, dynamische Client-Registrierung, lokalen Callback-Server, Token-Austausch und authentifizierte Anfragen. Verwenden Sie diese Option, wenn Sie eine Verbindung zu OAuth-geschützten MCP-Servern herstellen, MCP-Tools testen oder Authentifizierungsabläufe debuggen. Akzeptiert eine MCP-Server-URL als Argument.
