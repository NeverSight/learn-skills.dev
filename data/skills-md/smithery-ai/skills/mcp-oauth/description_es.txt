Autentíquese con cualquier servidor MCP OAuth 2.1. Maneja descubrimiento, PKCE, registro dinámico de clientes, servidor de devolución de llamadas local, intercambio de tokens y solicitudes autenticadas. Úselo al conectarse a servidores MCP protegidos por OAuth, probar herramientas MCP o depurar flujos de autenticación. Acepta una URL del servidor MCP como argumento.
