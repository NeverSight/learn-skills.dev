قم بإنشاء تصورات بيانات بجودة الإنتاج بما في ذلك المخططات ولوحات المعلومات والرسوم البيانية. يُستخدم عندما يطلب المستخدم تصور البيانات، أو إنشاء مخططات، أو إنشاء لوحات معلومات، أو إنشاء مخططات معلوماتية، أو رسم إحصائيات، أو تحويل مجموعات البيانات إلى تمثيلات مرئية. يدعم عناصر React/Recharts والصور الثابتة (PNG/PDF عبر Python) وHTML التفاعلي. تتضمن المشغلات "تصور هذه البيانات"، أو "إنشاء مخطط"، أو "إنشاء لوحة معلومات"، أو "إنشاء رسم بياني"، أو "تخطيط هذا"، أو "مخطط معلومات بياني"، أو أي طلب لتمثيل البيانات بشكل مرئي.
