Habilidades para el desarrollo impulsado por rompecabezas (PDD). Consulte esta habilidad cuando un proyecto utilice PDD o el operador mencione PDD o desarrollo basado en rompecabezas. Utilice esta habilidad cuando: (1) Divida una característica grande en entregables incrementales, (2) Escriba comentarios de código auxiliar de @todo para marcar el código no implementado, (3) Elija un rompecabezas @todo existente para implementarlo, (4) Decida si concluir una tarea con un código auxiliar o seguir trabajando.
