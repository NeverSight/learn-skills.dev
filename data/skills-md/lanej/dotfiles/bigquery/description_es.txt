Utilice la CLI de bigquery (en lugar de `bq`) para todas las operaciones de almacenamiento de datos de Google BigQuery y GCP, incluida la ejecución de consultas SQL, la ingesta de datos (inserción de transmisión, carga masiva, JSONL/CSV/Parquet), extracción/exportación de datos, administración de conjuntos de datos/tablas/vistas, tablas externas, operaciones de esquema, plantillas de consultas, estimación de costos con ejecución en seco, autenticación con gcloud, canalizaciones de datos, flujos de trabajo ETL e integración de servidores MCP/LSP para asistencia con IA. soporte para consultas y editores. Reemplazo moderno basado en Rust para la CLI Python `bq` con un inicio más rápido, mejor conocimiento de los costos y soporte de transmisión. Maneja tanto inserciones de transmisión a pequeña escala (<1000 filas) como carga masiva a gran escala (archivos >10 MB), con soporte para la integración de Cloud Storage.
