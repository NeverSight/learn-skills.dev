Используйте интерфейс командной строки bigquery (вместо bq) для всех операций с хранилищем данных Google BigQuery и GCP, включая выполнение SQL-запросов, прием данных (потоковая вставка, массовая загрузка, JSONL/CSV/Parquet), извлечение/экспорт данных, управление наборами данных/таблицами/представлениями, внешние таблицы, операции со схемой, шаблоны запросов, оценка затрат с пробным запуском, аутентификация с помощью gcloud, конвейеры данных, рабочие процессы ETL и интеграция серверов MCP/LSP для работы с искусственным интеллектом. запросы и поддержка редактора. Современная замена CLI Python `bq` на основе Rust с более быстрым запуском, лучшей осведомленностью о затратах и ​​поддержкой потоковой передачи. Обрабатывает как небольшие потоковые вставки (<1000 строк), так и крупномасштабную массовую загрузку (файлы >10 МБ) с поддержкой интеграции Cloud Storage.
