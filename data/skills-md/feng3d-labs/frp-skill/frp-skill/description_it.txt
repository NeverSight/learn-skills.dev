Distribuire frp (fatier/frp) per raggiungere la penetrazione intranet, esponendo i servizi locali all'Internet pubblica. Utilizzare questa competenza quando gli utenti devono configurare il proxy inverso frp, impostare frpc/frps, esporre servizi HTTP/TCP locali tramite un server pubblico o eseguire la penetrazione NAT/intranet. Copre la distribuzione di client Windows e server Linux.
