Conseiller en analyse de trading crypto. Se déclenche lorsque les utilisateurs demandent si un jeton vaut la peine d'être échangé, si une position peut être prise ou pour une analyse des tendances. Fournit des sources de données de grande valeur et des principes de trading professionnels. Les phrases de déclenchement incluent « dois-je trader », « puis-je long/short », « est-ce une bonne configuration », « recommandation de trading », « puis-je entrer ici », « quel est le jeu », etc.
