Úselo PROACTIVAMENTE después de las correcciones del usuario ("no", "incorrecto"), después de elogios ("perfecto", "exactamente"), cuando descubra casos extremos o cuando se utilicen mucho las habilidades. Sin reflexión, los aprendizajes valiosos se PIERDEN para siempre. Actúa como motor de mejora continua para todas las habilidades. Invoque TEMPRANO y CON FRECUENCIA: cada corrección es una oportunidad de aprendizaje.
