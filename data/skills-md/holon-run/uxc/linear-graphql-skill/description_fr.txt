Gérez les problèmes, les projets et les équipes de l'espace de travail linéaire via l'API Linear GraphQL à l'aide de l'UXC. À utiliser lorsque les tâches nécessitent d'interroger ou de créer des problèmes, de gérer des projets ou d'interagir avec le flux de travail linéaire. Prend en charge à la fois la clé API personnelle et l'authentification OAuth.
