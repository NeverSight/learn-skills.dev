Используйте Etherscan MCP через UXC для проверки баланса адресов, анализа держателей токенов, проверки транзакций и задач поиска контрактов. Используйте, когда задачам требуются инструменты Etherscan MCP для исследования цепочки с проверкой схемы в первую очередь, аутентификацией по ключу на предъявителя и обработкой первого чтения с учетом уровня.
