Этот навык следует использовать, когда пользователь просит «написать текст», «создать заголовок», «написать слоган», «улучшить мой текст», «написать копию целевой страницы», «создать копию объявления», «написать тему электронного письма», «написать описание продукта», «написать вступительную часть информационного бюллетеня», «написать копию страницы с ценами», «написать копию домашней страницы» или упоминает копирайтинг, убедительный текст, маркетинговый текст или текст для конверсии. Использует структуру маркетинговых примеров Гарри Драй для создания четких, запоминающихся текстов с высокой конверсией.
