Cette compétence doit être utilisée lorsque l'utilisateur demande de « diagnostiquer le problème », « décomposer le problème », « construire une arborescence logique », « quelle est la cause de cela », ou mentionne le diagnostic du problème, les arbres logiques, la décomposition MECE ou la résolution structurée de problèmes. Il s’agit du POINT D’ENTRÉE pour l’analyse des problèmes – à utiliser lorsqu’aucun diagnostic n’existe encore.
