Этот навык следует использовать, когда пользователь просит «создать пользовательский поток», «составить навигацию», «спроектировать путь пользователя», «создать проводной поток», «спланировать переходы между экранами», «составить карту потока приложения», «спроектировать навигацию» или упоминает пользовательский поток, проводной поток, карту навигации, путь пользователя, переходы между экранами или исследование дизайна перед интерфейсом. Создает схемы пользовательских потоков и потоки операций для цифровых продуктов.
