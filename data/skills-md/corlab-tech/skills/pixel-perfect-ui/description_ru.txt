Автономный цикл реализации пользовательского интерфейса с точностью до пикселя для Next.js/React с использованием Figma MCP и Playwright. Преобразует проекты Figma в готовые к производству компоненты с итеративной визуальной проверкой. **АВТО-ТРИГГЕРЫ** по ЛЮБОМУ запросу для реализации проектов Figma, включая: «реализовать эту Figma», «создать эту страницу/компонент из Figma», «создать на основе дизайна Figma», «реализовать дизайн», «построить этот блок», «создать компонент на основе дизайна». Используется для: (1) создания страниц/компонентов из Figma, (2) идеальной до пикселя точности, (3) адаптивных макетов, (4) преобразования токенов дизайна.
