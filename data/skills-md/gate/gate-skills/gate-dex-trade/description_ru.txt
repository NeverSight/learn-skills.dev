Комплексные навыки торговли Gate DEX. Поддерживает двойные режимы MCP и OpenAPI: вызовы в режиме MCP через службу шлюза-кошелька (требуется аутентификация), вызовы в режиме OpenAPI напрямую через AK/SK. Используйте, когда пользователи упоминают обмен, обмен, покупку, продажу, котировку, торговлю. Автоматически выбирайте наиболее подходящий метод вызова в зависимости от среды.
