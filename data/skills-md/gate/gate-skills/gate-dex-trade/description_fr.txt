Compétence complète de trading Gate DEX. Prend en charge les modes doubles MCP et OpenAPI : appels en mode MCP via le service gate-wallet (nécessite une authentification), appels en mode OpenAPI directement via AK/SK. À utiliser lorsque les utilisateurs mentionnent échanger, échanger, acheter, vendre, coter, échanger. Sélectionnez automatiquement la méthode d'appel la plus appropriée en fonction de l'environnement.
