بوابة التداول الفوري ومهارة عمليات الحساب. استخدم هذه المهارة عندما يطلب المستخدم شراء/بيع العملات المشفرة، أو التحقق من قيمة الحساب، أو إلغاء/تعديل الأوامر الفورية، أو وضع خطط شراء/بيع مشروطة، أو التحقق من عمليات التعبئة، أو إجراء مقايضات عملة مقابل عملة في التداول الفوري على البوابة. تتضمن عبارات التحفيز "شراء عملة"، أو "بيع عملة"، أو "مراقبة السوق"، أو "إلغاء الطلب"، أو "تعديل الطلب"، أو "سعر التعادل"، أو "إعادة التوازن"، أو "التداول الفوري"، أو "الشراء/البيع"، أو أي طلب يجمع بين تنفيذ الأوامر الفورية وفحص الحساب.
