وظيفة تحليل السوق في Gate Exchange - السيولة، والزخم، والتصفية، ومراجحة التمويل، والأساس، ومخاطر التلاعب، وشرح دفتر الأوامر، ومحاكاة الانزلاق. يُستخدم عندما يسأل المستخدم عن السيولة، أو العمق، أو الانزلاق، أو ضغط الشراء/البيع، أو التصفية، أو تحكيم معدل التمويل، أو الأساس/القسط، أو مخاطر التلاعب، أو شرح دفتر الطلبات، أو محاكاة الانزلاق (على سبيل المثال، انزلاق شراء السوق X $). عبارات التحفيز: السيولة، العمق، الانزلاق، الزخم، ضغط الشراء/البيع، التصفية، الضغط، معدل التمويل، المراجحة، الأساس، العلاوة، التلاعب، دفتر الطلبات، السبريد، محاكاة الانزلاق.
