Комплексные навыки кошелька Gate DEX. Поддержка унифицированной точки входа: вход в систему аутентификации, запросы активов, выполнение передачи, взаимодействие с DApp, командная строка CLI для пяти основных модулей. Используйте, когда пользователи упоминают вход в систему, проверку баланса, перевод, взаимодействие с DApp, подписание, шлюз-кошелек, CLI, командную строку, openapi-swap и другие операции, связанные с кошельком. Маршрутизация к определенным справочным файлам операций посредством маршрутизации подфункций.
