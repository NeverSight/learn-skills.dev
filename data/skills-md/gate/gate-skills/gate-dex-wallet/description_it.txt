Abilità di portafoglio completa Gate DEX. Punto di ingresso unificato che supporta: accesso di autenticazione, query sulle risorse, esecuzione del trasferimento, interazioni DApp, riga di comando CLI per cinque moduli principali. Da utilizzare quando gli utenti menzionano accesso, controllo del saldo, trasferimento, interazione DApp, firma, gate-wallet, CLI, riga di comando, openapi-swap e altre operazioni relative al portafoglio. Instradamento a file di riferimento dell'operazione specifici tramite l'instradamento delle sottofunzioni.
