Comparaison de pièces. Utilisez cette compétence chaque fois que l'utilisateur demande à comparer deux pièces ou plus. Les phrases de déclenchement incluent : comparer, contre, contre, ce qui est mieux, différence. Outils MCP : info_marketsnapshot_get_market_snapshot, info_coin_get_coin_info par pièce (ou lot/recherche si disponible).
