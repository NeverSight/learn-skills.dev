Виртуальный хостинг для нескольких сайтов HTTPS с автоматизацией Let's Encrypt SSL. Каждый домен получает изолированный экземпляр VirtualServer, поддерживает Express/Socket.IO/пользовательские обработчики, локальный режим разработки HTTP с портами на основе CRC32, автоматические перенаправления www и управление сертификатами SNI.
