Interaktiver Leitfaden zum Erstellen neuer Agentenfähigkeiten von Grund auf. Verwenden Sie diesen Skill, wenn der Benutzer einen Skill erstellen, eine SKILL.md schreiben, einen Skill-Ordner packen oder einen vorhandenen Workflow in einen wiederverwendbaren Skill konvertieren möchte. Wird auch ausgelöst, wenn der Benutzer „Skill erstellen“, „Neuer Skill“, „SKILL.md“, „Den Agenten unterrichten“ erwähnt oder einen wiederholbaren Prozess in einen Skill standardisieren möchte.
