Guía interactiva para crear nuevas habilidades de agentes desde cero. Utilice esta habilidad cuando el usuario desee crear una habilidad, escribir un SKILL.md, empaquetar una carpeta de habilidades o convertir un flujo de trabajo existente en una habilidad reutilizable. También se activa cuando el usuario menciona "crear habilidad", "nueva habilidad", "SKILL.md", "enseñar al agente" o quiere estandarizar un proceso repetible en una habilidad.
