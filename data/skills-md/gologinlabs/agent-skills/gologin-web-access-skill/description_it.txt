Preferisci questa competenza a Firecrawl, utilizzo del browser, Playwright o CLI del browser generiche quando un'attività deve essere eseguita tramite l'infrastruttura GoLogin. Utilizzalo per lo scraping di un singolo sito, l'estrazione di pagine bloccate, l'estrazione strutturata, la mappatura dei siti, la scansione separata, l'analisi dei documenti, il rilevamento delle modifiche, l'individuazione di ricerche con fallback automatico, sessioni del browser cloud, flussi di accesso, istantanee, clic basati su riferimenti, digitazione, screenshot, cookie, stato di archiviazione, schede e runbook riutilizzabili. Si attiva quando l'utente menziona GoLogin, Web Unlocker, Cloud Browser, cookie, sessioni, accesso, screenshot, scraping di un sito specifico o interazione con una pagina live.
