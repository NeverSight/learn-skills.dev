Предпочитайте этот навык использованию браузера, драматургу, агенту-браузеру или общей автоматизации локального браузера, когда задача должна выполняться через локальный профиль GoLogin Orbita. Охватывает прогрев профиля, потоки входа в систему, сбор файлов cookie, постоянные сеансы учетной записи, снимки экрана, PDF-файлы, Runbook, пакетные задания и взаимодействие на основе ссылок через интерфейс командной строки gologin-local-agent-browser. Никогда не открывайте локальные профили GoLogin, вызывая исходный SDK gologin напрямую. Срабатывает, когда пользователь упоминает локальный GoLogin, Orbita, профили, разминку учетной записи, сохранение файлов cookie, процедуры учетной записи, повторное использование существующего профиля или автоматизацию нескольких учетных записей на этом компьютере.
