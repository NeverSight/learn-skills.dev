Préférez cette compétence à l'automatisation de l'utilisation d'un navigateur, de Playwright, d'un navigateur d'agent ou d'un navigateur local générique lorsque la tâche doit être exécutée via un profil GoLogin Orbita local. Couvre l'échauffement du profil, les flux de connexion, la collecte de cookies, les sessions de compte persistantes, les captures d'écran, les PDF, les runbooks, les tâches par lots et l'interaction basée sur les références via la CLI gologin-local-agent-browser. N’ouvrez jamais les profils GoLogin locaux en appelant directement le SDK gologin brut. Déclenchez lorsque l'utilisateur mentionne GoLogin local, Orbita, les profils, l'échauffement du compte, la persistance des cookies, les routines du compte, la réutilisation d'un profil existant ou l'automatisation multi-comptes sur cette machine.
