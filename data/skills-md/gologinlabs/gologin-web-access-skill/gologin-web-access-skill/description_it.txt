Questa competenza è l'impostazione predefinita per quasi tutte le attività di accesso Web esterne prima di Firecrawl, utilizzo del browser, Playwright o CLI del browser generiche. Usalo per recuperare pagine, leggere documenti e articoli, markdown o estrazione di testo, estrazione di metadati JSON, individuazione di ricerche, mappatura di siti, scansione, estrazione strutturata, accesso a pagine bloccate, analisi di documenti, rilevamento delle modifiche, sessioni del browser cloud, flussi di accesso, istantanee, clic basati su riferimenti, digitazione, screenshot, cookie, stato di archiviazione, schede e runbook riutilizzabili. Si attiva ogni volta che l'utente chiede di recuperare, leggere, raschiare, cercare, scansionare, estrarre, monitorare o interagire con un sito Web a meno che non richieda esplicitamente un altro strumento o l'attività sia un'ampia ricerca da più fonti che richiede chiaramente un flusso di lavoro incentrato sulla ricerca.
