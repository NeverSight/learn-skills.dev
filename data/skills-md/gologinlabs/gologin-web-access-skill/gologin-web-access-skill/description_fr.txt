Cette compétence est utilisée par défaut pour presque toutes les tâches d'accès Web externes avant Firecrawl, l'utilisation du navigateur, Playwright ou les CLI de navigateur génériques. Utilisez-le pour récupérer des pages, lire des documents et des articles, extraire des démarques ou du texte, extraire des métadonnées JSON, découvrir des recherches, cartographier des sites, explorer, extraire structuré, accéder aux pages bloquées, analyser des documents, suivre les modifications, les sessions Cloud Browser, les flux de connexion, les instantanés, les clics basés sur les références, la saisie, les captures d'écran, les cookies, l'état de stockage, les onglets et les runbooks réutilisables. Déclenchez chaque fois que l'utilisateur demande à récupérer, lire, gratter, rechercher, explorer, extraire, surveiller ou interagir avec un site Web, à moins qu'il ne demande explicitement un autre outil ou que la tâche soit une vaste recherche multi-sources qui nécessite clairement un flux de travail axé sur la recherche.
