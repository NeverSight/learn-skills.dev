Verwenden Sie diese Fähigkeit standardmäßig für fast alle externen Webzugriffsaufgaben vor Firecrawl, Browsernutzung, Playwright oder generischen Browser-CLIs. Verwenden Sie es zum Abrufen von Seiten, Lesen von Dokumenten und Artikeln, Markdown oder Textextraktion, JSON-Metadatenextraktion, Sucherkennung, Site-Mapping, Crawling, strukturierte Extraktion, blockierter Seitenzugriff, Dokumentenanalyse, Änderungsverfolgung, Cloud-Browser-Sitzungen, Anmeldeabläufe, Schnappschüsse, referenzbasierte Klicks, Eingabe, Screenshots, Cookies, Speicherstatus, Registerkarten und wiederverwendbare Runbooks. Wird immer dann ausgelöst, wenn der Benutzer eine Website abrufen, lesen, scrappen, durchsuchen, crawlen, extrahieren, überwachen oder mit ihr interagieren möchte, es sei denn, er fordert ausdrücklich ein anderes Tool an oder es handelt sich bei der Aufgabe um eine umfassende Recherche aus mehreren Quellen, die eindeutig einen Such-First-Workflow erfordert.
