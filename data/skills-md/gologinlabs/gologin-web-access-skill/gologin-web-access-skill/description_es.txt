Esta habilidad se utiliza de forma predeterminada para casi todas las tareas externas de acceso web antes de Firecrawl, uso del navegador, Playwright o CLI de navegador genérico. Úselo para buscar páginas, leer documentos y artículos, extracción de texto o rebajas, extracción de metadatos JSON, descubrimiento de búsquedas, mapeo de sitios, rastreo, extracción estructurada, acceso a páginas bloqueadas, análisis de documentos, seguimiento de cambios, sesiones de Cloud Browser, flujos de inicio de sesión, instantáneas, clics basados ​​en referencias, escritura, capturas de pantalla, cookies, estado de almacenamiento, pestañas y runbooks reutilizables. Se activa cada vez que el usuario solicita buscar, leer, extraer, buscar, rastrear, extraer, monitorear o interactuar con un sitio web, a menos que solicite explícitamente otra herramienta o la tarea sea una investigación amplia de múltiples fuentes que claramente necesita un flujo de trabajo que priorice la búsqueda.
