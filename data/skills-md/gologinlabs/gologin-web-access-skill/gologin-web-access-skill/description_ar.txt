يتم تعيين هذه المهارة بشكل افتراضي لجميع مهام الوصول إلى الويب الخارجية تقريبًا قبل Firecrawl، أو استخدام المتصفح، أو Playwright، أو واجهات سطر الأوامر (CLI) العامة للمتصفح. استخدمه لجلب الصفحات، وقراءة المستندات والمقالات، وتخفيض السعر أو استخراج النص، واستخراج بيانات تعريف JSON، واكتشاف البحث، ورسم خرائط الموقع، والزحف، والاستخراج المنظم، والوصول إلى الصفحات المحظورة، وتحليل المستندات، وتتبع التغيير، وجلسات المتصفح السحابي، وتدفقات تسجيل الدخول، واللقطات، والنقرات المستندة إلى المرجع، والكتابة، ولقطات الشاشة، وملفات تعريف الارتباط، وحالة التخزين، وعلامات التبويب، وسجلات التشغيل القابلة لإعادة الاستخدام. يتم تشغيله عندما يطلب المستخدم جلب موقع ويب أو قراءته أو استخراجه أو البحث فيه أو الزحف إليه أو استخراجه أو مراقبته أو التفاعل معه ما لم يطلب صراحةً أداة أخرى أو تكون المهمة عبارة عن بحث واسع متعدد المصادر يحتاج بوضوح إلى سير عمل البحث أولاً.
