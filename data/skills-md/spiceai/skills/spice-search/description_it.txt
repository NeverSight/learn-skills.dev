Cerca dati utilizzando la somiglianza vettoriale, parole chiave full-text o metodi ibridi con Reciprocal Rank Fusion (RRF). Da utilizzare durante l'impostazione degli incorporamenti per la ricerca, la configurazione dell'indicizzazione del testo completo, la scrittura di query SQL vector_search/text_search/rrf, l'utilizzo dell'API HTTP /v1/search o la configurazione di motori vettoriali come S3 Vectors.
