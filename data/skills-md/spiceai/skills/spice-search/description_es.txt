Busque datos utilizando similitud de vectores, palabras clave de texto completo o métodos híbridos con Reciprocal Rank Fusion (RRF). Úselo al configurar incrustaciones para búsqueda, configurar la indexación de texto completo, escribir consultas SQL vector_search/text_search/rrf, usar la API HTTP /v1/search o configurar motores vectoriales como S3 Vectors.
