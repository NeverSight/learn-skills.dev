ابحث عن البيانات باستخدام تشابه المتجهات أو الكلمات الأساسية للنص الكامل أو الطرق المختلطة باستخدام Reciprocal Rank Fusion (RRF). يُستخدم عند إعداد التضمينات للبحث، أو تكوين فهرسة النص الكامل، أو كتابة استعلامات SQL Vector_search/text_search/rrf، باستخدام /v1/search HTTP API، أو تكوين محركات المتجهات مثل S3 Vectors.
