[ЧТО] Пособие по графическому мышлению (CDO)
[КАК] Классифицирует входные данные по сложности, направляет к соответствующему рабочему процессу (Быстрый/Базовый/Глубокий/Эпический)
[КОГДА] Используется для проектирования, архитектуры и комплексной декомпозиции проблем, требующих многоаспектного анализа.
[ПОЧЕМУ] Предоставляет повторяемую схему мышления вместо специального мозгового штурма.

Триггеры: «думать», «проектировать», «архитектор», «сложность», «cdo», «график», «мультиагент», «глубокий анализ».
