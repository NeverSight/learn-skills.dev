مركز تصميم تجربة المستخدم: خط أنابيب سلكي مكون من 7 خطوات + تصميم المنتج (design-os) + النماذج الأولية المرئية (قلم رصاص) + أنماط تجربة المستخدم (lev-ref). يُستخدم للإطارات السلكية، أو التدفقات، أو IA، أو JTBD، أو تصميم المنتج، أو تصميم CLI، أو تجربة المستخدم الوكيلة، أو أنظمة التصميم، أو بنية المكونات.
