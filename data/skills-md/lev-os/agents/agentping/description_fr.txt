Protocole d'interaction agent-humain pour les agents IA. À utiliser lorsque les agents ont besoin d'approbations humaines, de sélections, d'orientations de recherche ou de commentaires asynchrones. Déclencheurs sur « l'humain dans la boucle », « l'approbation », « l'interaction de l'agent », « obtenir une contribution humaine » ou « attendre une réponse ».
