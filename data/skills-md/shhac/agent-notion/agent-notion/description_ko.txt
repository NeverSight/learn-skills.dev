인간과 LLM을 위한 Notion CLI. 다음과 같은 경우에 사용하세요:
- 제목으로 Notion 페이지 또는 데이터베이스 검색
- 필터 및 정렬을 사용하여 데이터베이스 행 쿼리
- 페이지 속성 및 콘텐츠 읽기(마크다운 또는 구조화된 블록)
- 페이지 생성, 업데이트 또는 보관
- 블록 콘텐츠 읽기, 추가, 업데이트, 삭제 또는 교체
- 페이지에 댓글 나열 또는 추가(페이지 수준 또는 특정 텍스트의 인라인)
- 페이지 또는 전체 작업 공간을 마크다운/HTML로 내보내기
- 백링크(특정 페이지로 연결되는 페이지) 찾기
- 페이지의 버전 기록 스냅샷 보기
- 최근 작업공간이나 페이지 활동 확인
- Notion AI와 채팅(메시지 보내기, 스레드 나열, 응답 스트리밍)
- 사용 가능한 Notion AI 모델 나열
- Notion OAuth, 통합 토큰 또는 데스크톱 세션 인증 관리
