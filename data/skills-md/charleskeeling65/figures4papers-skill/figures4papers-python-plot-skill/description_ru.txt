Используйте при создании или рефакторинге сценариев рисунков Python в стиле ChenLiu-1996/figures4papers, включая исходные папки assets/output иfig_*. Триггер для таких запросов, как «нарисовать фигуру на бумаге», «воспроизвести фигуру из сценария», «добавить новый модуль фигуры» или «пакетное выполнение сценариев построения графиков с согласованными настройками вывода качества публикации».
