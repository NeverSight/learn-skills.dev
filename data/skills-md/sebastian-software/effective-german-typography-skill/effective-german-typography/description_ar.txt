قم بتطبيق قواعد الطباعة الألمانية الصحيحة عند كتابة النص الألماني أو مراجعته. استخدمه عندما يُطلب منك "إصلاح الطباعة" أو "مراجعة الطباعة" أو كتابة محتوى باللغة الألمانية أو تحرير النص الألماني في Markdown أو HTML أو JSX أو النص العادي. يغطي علامات الاقتباس والشرطات والمسافات ذات الوحدات وعلامات الحذف والفواصل العليا والمتغيرات الإقليمية (ألمانيا والنمسا وسويسرا).
