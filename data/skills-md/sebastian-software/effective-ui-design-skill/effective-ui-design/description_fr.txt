Directives de conception d’interface utilisateur professionnelle pour des interfaces accessibles et bien structurées. À utiliser lorsque : vous écrivez du code de composant CSS, HTML ou frontend ; concevoir des sites Web, des applications, des tableaux de bord ou toute interface Web ; revoir ou améliorer les conceptions d’interface utilisateur existantes ; prendre des décisions concernant la couleur, la typographie, la mise en page, l'espacement, les boutons ou les formes ; assurer l’accessibilité aux WCAG 2.1 AA ; créer des systèmes de conception ou des bibliothèques de composants. Couvre les palettes de couleurs OKLCH, la grille d'espacement de 8 points, la typographie fluide avec clamp(), les requêtes de conteneur, la sous-grille, les modèles de validation de formulaire, la hiérarchie des boutons, le mode sombre, le mouvement réduit, les balises méta SEO et Core Web Vitals.
