Trasforma suggerimenti o idee in istruzioni per piani di assemblaggio/costruzione 3D come LEGO Minecraft tramite l'API Craftsman Agent (router OneKey o server locale). Da utilizzare quando si generano piani di costruzione, elenchi di inventario o immagini di assemblaggio passo passo per LEGO/Minecraft da testo o immagini di riferimento o quando si collegano i client agli endpoint di Craftsman Agent.
