قم بتحويل المطالبات أو الأفكار إلى تعليمات خطط التجميع/البناء ثلاثية الأبعاد مثل LEGO Minecraft عبر Craftsman Agent API (OneKey Router أو الخادم المحلي). يُستخدم عند إنشاء خطط البناء أو قوائم المخزون أو صور التجميع خطوة بخطوة لـ LEGO/Minecraft من النصوص أو الصور المرجعية، أو عند توصيل العملاء بنقاط نهاية Craftsman Agent.
