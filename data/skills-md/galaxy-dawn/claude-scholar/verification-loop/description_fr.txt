Cette compétence doit être utilisée lorsque l'utilisateur demande de « vérifier le code », « exécuter la vérification », « vérifier la qualité », « valider les modifications » ou avant de créer un PR. Fournit une vérification complète comprenant la construction, la vérification de type, les peluches, les tests, l'analyse de sécurité et l'examen des différences.
