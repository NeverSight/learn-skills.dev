僅在需要工廠/註冊表模式的 ML 專案中建立新的可註冊元件時使用。

✅ 在下列情況下使用：
- 建立一個新的Dataset類別（需要@register_dataset）
- 建立一個新的模型類別（需要@register_model）
- 使用 __init__.py 工廠建立一個新的模組目錄
- 從頭開始初始化新的 ML 專案結構
- 新增新的元件類型（Augmentation、CollateFunction、Metrics）

❌請勿在下列情況下使用：
- 修改現有的函數或方法
- 修復現有程式碼中的錯誤
- 新增輔助函數或實用程式
- 重構而不添加新的可註冊元件
- 對單一檔案進行簡單的程式碼更改
- 修改設定檔
- 閱讀或理解現有程式碼

關鍵指標：任務是否需要@register_*裝飾器或Factory模式？如果不是，請跳過該技能。
