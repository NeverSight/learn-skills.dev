Этот навык следует использовать, когда пользователь просит «создать git-коммит», «управлять ветвями», «следовать рабочему процессу git», «использовать обычные коммиты», «обрабатывать конфликты слияния» или спрашивает о стратегиях ветвления git, лучших методах управления версиями, рабочих процессах запросов на включение. Предоставляет комплексное руководство по рабочему процессу Git для совместной работы в команде.
