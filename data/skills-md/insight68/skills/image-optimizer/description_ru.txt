Комплексный набор инструментов для оптимизации изображений для повышения производительности в Интернете. Преобразует изображения в формат WebP, создает миниатюры с указанием пропорций, удаляет белый фон с помощью интеллектуального сглаживания краев и обрабатывает изображения в параллельных пакетах. Используйте его, когда Клоду необходимо оптимизировать изображения для работы в Интернете, создавать адаптивные миниатюры, удалять фон с фотографий продуктов или пакетно обрабатывать каталоги изображений. Поддерживает режимы прозрачности, суперсэмплинг для сглаживания краев, пользовательские размеры и несколько соотношений сторон (16:9, 4:3, 21:9, 1:1, 3:2).
