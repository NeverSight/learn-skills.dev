Kit completo di strumenti di ottimizzazione delle immagini per le prestazioni web. Converte le immagini in WebP, genera miniature con proporzioni, rimuove gli sfondi bianchi con smussamento intelligente dei bordi ed elabora le immagini in batch paralleli. Utilizzalo quando Claude ha bisogno di ottimizzare le immagini per le prestazioni web, generare miniature reattive, rimuovere sfondi dalle foto dei prodotti o elaborare in batch directory di immagini. Supporta modalità di trasparenza, supercampionamento per bordi smussati, dimensioni personalizzate e proporzioni multiple (16:9, 4:3, 21:9, 1:1, 3:2).
