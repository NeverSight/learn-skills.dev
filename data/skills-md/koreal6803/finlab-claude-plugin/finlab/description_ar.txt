دليل شامل لحزمة التداول الكمي FinLab. يُستخدم عند العمل مع استراتيجيات التداول، أو الاختبار الخلفي، أو بيانات الأسهم، أو FinLabDataFrame، أو تحليل العوامل، أو اختيار الأسهم، أو عندما يذكر المستخدم FinLab، أو التداول، أو التداول الكمي، أو تحليل سوق الأوراق المالية. يتضمن الوصول إلى البيانات، وتطوير الإستراتيجية، واختبار سير العمل، وأفضل الممارسات.
