Examen itératif du code et discussion de planification entre Claude Code et Codex CLI (par défaut, gpt-5.4 avec un effort de raisonnement xélevé, tous deux configurables). Orchestre un débat automatique au cours duquel les deux agents discutent des résultats, des décisions d'architecture ou des plans de mise en œuvre jusqu'à parvenir à un consensus. Codex CLI fonctionne en mode LECTURE SEULE — il ne modifie jamais les fichiers. Prend en charge le mode plan : lorsque Claude Code a un plan prêt, invoquez cette compétence pour que le Codex évalue et itère sur le plan avant sa mise en œuvre, produisant ainsi un plan consensuel mis à jour. À utiliser lorsque l'utilisateur demande à réviser avec le codex, analyser avec le codex, discuter avec le codex, itérer avec le codex, consulter le codex, demander au codex, réviser le plan avec le codex, valider le plan avec le codex ou toute demande impliquant la CLI du Codex pour la révision du code, la révision de l'architecture, la discussion de planification ou l'analyse collaborative du code, de la conception ou de la stratégie de mise en œuvre.
