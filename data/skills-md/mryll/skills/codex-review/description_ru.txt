Итеративный анализ кода и обсуждение планирования между Claude Code и Codex CLI (по умолчанию gpt-5.4 с высокой степенью аргументации, оба настраиваемые). Организует автоматические двусторонние дебаты, в ходе которых оба агента обсуждают выводы, архитектурные решения или планы реализации до достижения консенсуса. Codex CLI работает в режиме ТОЛЬКО ЧТЕНИЕ — он никогда не изменяет файлы. Поддерживает режим плана: когда у Клода Кода есть готовый план, используйте этот навык, чтобы Кодекс оценил и выполнил итерацию плана перед его реализацией, создав обновленный согласованный план. Используйте, когда пользователь просит просмотреть с помощью Кодекса, проанализировать с помощью Кодекса, обсудить с Кодексом, выполнить итерацию с Кодексом, проконсультироваться с Кодексом, задать вопрос Кодексу, просмотреть план с помощью Кодекса, проверить план с помощью Кодекса или любой запрос, включающий интерфейс командной строки Кодекса для проверки кода, обзора архитектуры, обсуждения планирования или совместного анализа кода, проекта или стратегии реализации.
