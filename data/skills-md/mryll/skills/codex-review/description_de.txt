Iterative Codeüberprüfung und Planungsdiskussion zwischen Claude Code und Codex CLI (standardmäßig gpt-5.4 mit xhohem Argumentationsaufwand, beide konfigurierbar). Orchestriert eine automatische Hin- und Her-Debatte, bei der beide Agenten Ergebnisse, Architekturentscheidungen oder Implementierungspläne diskutieren, bis ein Konsens erzielt wird. Codex CLI arbeitet im READ-ONLY-Modus – es ändert niemals Dateien. Unterstützt den Planmodus: Wenn Claude Code einen Plan bereit hat, rufen Sie diese Fähigkeit auf, damit Codex den Plan vor der Implementierung auswertet und iteriert und so einen aktualisierten Konsensplan erstellt. Verwenden Sie diese Option, wenn der Benutzer eine Überprüfung mit Codex, eine Analyse mit Codex, eine Diskussion mit Codex, eine Iteration mit Codex, eine Rücksprache mit Codex, eine Anfrage bei Codex, eine Überprüfung des Plans mit Codex, eine Validierung des Plans mit Codex oder eine andere Anforderung an Codex CLI für eine Codeüberprüfung, Architekturüberprüfung, Planungsdiskussion oder gemeinsame Analyse von Code, Design oder Implementierungsstrategie anfordert.
