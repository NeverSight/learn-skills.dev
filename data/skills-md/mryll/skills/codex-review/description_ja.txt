Claude Code と Codex CLI の間での反復的なコード レビューと計画に関するディスカッション (デフォルトは xhigh 推論機能を備えた gpt-5.4 で、両方とも構成可能)。双方のエージェントが合意に達するまで調査結果、アーキテクチャの決定、または実装計画について議論する、自動的な往復討論を調整します。 Codex CLI は読み取り専用モードで動作し、ファイルを変更することはありません。計画モードのサポート: Claude Code で計画の準備ができたら、このスキルを呼び出して、実装前に Codex に計画を評価して反復させ、更新されたコンセンサス計画を生成します。ユーザーが、コーデックスによるレビュー、コーデックスによる分析、コーデックスによるディスカッション、コーデックスによる反復、コーデックスによる相談、コーデックスによる質問、コーデックスによる計画のレビュー、コーデックスによる計画の検証、またはコード レビュー、アーキテクチャ レビュー、計画の議論、またはコード、設計、または実装戦略の共同分析のための Codex CLI に関連するリクエストを要求する場合に使用します。
