Создавайте портативные навыки S-уровня с помощью подробных рекомендаций по обеспечению качества. Этот навык следует использовать при создании новых навыков, улучшении существующих навыков, проверке качества навыков или изучении передового опыта в области формирования навыков. Охватывает полный жизненный цикл от определения варианта использования до тестирования, распространения и итерации. НЕ используйте для общих задач по кодированию, написанию документации, не связанной с навыками, или для создания приложений.
