Standard di architettura del progetto Node.js per lo sviluppo assistito dall'intelligenza artificiale. Applica la suddivisione dei file (<400 righe), l'esternalizzazione della configurazione, la modularizzazione del percorso e i dashboard di amministrazione. Da utilizzare durante la creazione di nuovi progetti Node.js, il refactoring di codebase a file singolo di grandi dimensioni o quando la finestra di contesto AI viene utilizzata da file di grandi dimensioni. Copre giochi H5 (Canvas/Phaser/Matter.js), strumenti di dati (crawler/scrapers), piattaforme di contenuti, dashboard di monitoraggio, servizi API e librerie SDK.
