Стандарты архитектуры проекта Node.js для разработки с использованием искусственного интеллекта. Обеспечивает разделение файлов (<400 строк), экстернализацию конфигурации, модульность маршрутов и панели администрирования. Используйте при создании новых проектов Node.js, рефакторинге больших однофайловых кодовых баз или когда окно контекста AI используется файлами слишком большого размера. Охватывает игры H5 (Canvas/Phaser/Matter.js), инструменты обработки данных (сканеры/скребки), контентные платформы, панели мониторинга, службы API и библиотеки SDK.
