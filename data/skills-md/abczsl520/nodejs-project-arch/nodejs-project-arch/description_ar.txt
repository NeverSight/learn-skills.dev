معايير بنية مشروع Node.js للتطوير بمساعدة الذكاء الاصطناعي. يفرض تقسيم الملفات (<400 سطر)، وإضفاء الطابع الخارجي على التكوين، ونمطية المسار، ولوحات معلومات المسؤول. يُستخدم عند إنشاء مشاريع Node.js جديدة، أو إعادة بناء قواعد تعليمات برمجية كبيرة ذات ملف واحد، أو عندما يتم استهلاك نافذة سياق الذكاء الاصطناعي بواسطة ملفات كبيرة الحجم. يغطي ألعاب H5 (Canvas/Phaser/Matter.js)، وأدوات البيانات (برامج الزحف/المكشطات)، ومنصات المحتوى، ولوحات معلومات المراقبة، وخدمات API، ومكتبات SDK.
