Normes d'architecture de projet Node.js pour le développement assisté par l'IA. Applique le fractionnement des fichiers (<400 lignes), l'externalisation de la configuration, la modularisation des itinéraires et les tableaux de bord d'administration. À utiliser lors de la création de nouveaux projets Node.js, de la refactorisation de bases de code volumineuses à fichier unique ou lorsque la fenêtre contextuelle de l'IA est consommée par des fichiers surdimensionnés. Couvre les jeux H5 (Canvas/Phaser/Matter.js), les outils de données (crawlers/scrapers), les plateformes de contenu, les tableaux de bord de surveillance, les services API et les bibliothèques SDK.
