Auditoría integral de errores para proyectos web Node.js. Actívelo cuando el usuario solicite auditar, revisar, verificar errores, encontrar vulnerabilidades o realizar una revisión de seguridad/calidad en un proyecto. Funciona analizando el código real del proyecto para crear matrices de verificación específicas del proyecto y luego verificando exhaustivamente cada elemento, sin ejecutar una lista de verificación genérica. Admite juegos, herramientas de datos, aplicaciones WeChat, servicios API, bots y paneles.
