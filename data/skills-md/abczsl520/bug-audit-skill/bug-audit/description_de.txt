Umfassende Fehlerprüfung für Node.js-Webprojekte. Aktivieren Sie diese Option, wenn der Benutzer ein Projekt prüfen, überprüfen, Fehler überprüfen, Schwachstellen finden oder eine Sicherheits-/Qualitätsprüfung durchführen möchte. Funktioniert, indem der tatsächliche Code des Projekts zerlegt wird, um projektspezifische Prüfmatrizen zu erstellen, und anschließend jedes Element umfassend überprüft wird – nicht durch die Ausführung einer generischen Checkliste. Unterstützt Spiele, Datentools, WeChat-Apps, API-Dienste, Bots und Dashboards.
