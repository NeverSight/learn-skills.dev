Комплексный аудит ошибок веб-проектов Node.js. Активируйте, когда пользователь запрашивает аудит, проверку, проверку ошибок, поиск уязвимостей или проверку безопасности/качества проекта. Работает путем анализа фактического кода проекта для построения проверочных матриц для конкретного проекта, а затем исчерпывающей проверки каждого элемента, а не путем запуска общего контрольного списка. Поддерживает игры, инструменты обработки данных, приложения WeChat, службы API, ботов и информационные панели.
