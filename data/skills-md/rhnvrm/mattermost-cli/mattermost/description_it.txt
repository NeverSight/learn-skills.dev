Leggi ed esegui ricerche nella chat di Mattermost utilizzando la CLI `mm`. Utilizza questa abilità ogni volta che l'utente menziona Mattermost, messaggi di chat, chat di gruppo, messaggi non letti, messaggi diretti, cronologia del canale, menzioni o vuole aggiornarsi su ciò che è successo nella chat. Si attiva anche quando l'utente chiede informazioni sui messaggi di persone specifiche, sull'attività del canale, sulla ricerca di qualcosa che qualcuno ha detto o sul controllo delle notifiche. Per impostazione predefinita, la CLI genera JSON compatibile con l'agente con ID thread, rilevamento bot e riferimenti canale, senza necessità di analisi.
