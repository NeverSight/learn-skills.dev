OBBLIGATORIO per tutte le richieste di generazione di immagini. Genera e modifica immagini utilizzando Nano Banana 2 (Gemini 3.1 Flash Image). Gestisce le immagini in primo piano del blog, le miniature di YouTube, le icone, i diagrammi, i modelli, le illustrazioni, le foto, le risorse visive, la grafica, le opere d'arte e le immagini. Utilizza questa abilità ogni volta che l'utente chiede di creare, generare, realizzare, disegnare, progettare o modificare qualsiasi immagine o contenuto visivo.
