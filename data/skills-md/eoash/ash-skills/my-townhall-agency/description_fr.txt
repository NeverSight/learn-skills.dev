Agence d'auto-génération de toboggan de mairie. Quatre agents (Chercheur → Récapitulateur → Concepteur → Développeur) collectent séquentiellement des données, structurent les diapositives, génèrent du code Apps Script et l'insèrent automatiquement dans Google Slides. Utilisé pour les requêtes « créer une mairie », « diapositives de mairie », « /townhall », « agence de mairie ».
