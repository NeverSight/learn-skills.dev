**Générateur de documents commerciaux coréens** : crée automatiquement les documents que les employés de bureau coréens écrivent quotidiennement : rapports hebdomadaires, journaux de travail, brouillons, demandes d'approbation, notes de frais, rapports de voyage, procès-verbaux de réunion, etc. Il applique la structure "introduction-développement-tour-conclusion", le format de ligne d'approbation et le ton commercial coréen standard.
- DÉCLENCHEURS OBLIGATOIRES : rapport, brouillon, demande d'approbation, note de frais, rapport de voyage, carnet de travail, rapport hebdomadaire, rapport mensuel, approbation, document coréen, document commercial, lettre officielle, rapport de travail
