ينشئ تصميمات مفاهيمية توضح استراتيجية التصميم ونهجه. يُستخدم عندما يقول المستخدم "نموذج بالحجم الطبيعي"، أو "إطار سلكي"، أو "مفهوم"، أو "لوحة مزاجية"، أو "اتجاه مرئي"، أو "خيارات التصميم"، أو "استكشاف التصميمات"، أو "تصميم واجهة المستخدم"، أو "النموذج الأولي"، أو "صمم هذا"، أو "أرني التصميمات"، أو "إنشاء تصميم"، أو "الشكل والمظهر"، أو "أشكال التصميم المختلفة". الاستفادة من رؤى البحث وملخصات التصميم لتطوير مفاهيم واجهة المستخدم ولوحات المزاج والنماذج الأولية التفاعلية. لا تستخدمه عندما يحتاج المستخدم إلى فهم المستخدمين أولاً (استخدم بحث التصميم)، أو عندما يقوم بمراجعة منتج تم تنفيذه (استخدم design-qa)، أو يكتب تعليمات برمجية.
