Самообслуживание OpenClaw с собеседованием. Использует поток устройства CLI для аутентификации в браузере, создает машинный ключ агента пользователя, проводит каскадное интервью души для определения голоса/персонажа/целевого рынка, записывает проект души и файл режима обнаружения и выводит блок env. Используйте, когда: «/unclawg-internet», «настройте меня», «подключитесь к openclaw», «начните», «подключитесь ко мне», «зарегистрируйтесь в openclaw», «Я хочу получить одобрение»
