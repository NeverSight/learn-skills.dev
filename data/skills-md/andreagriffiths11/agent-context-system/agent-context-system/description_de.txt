Ein persistentes, nur lokales Speichersystem für KI-Codierungsagenten. Zwei Dateien, eine Idee – AGENTS.md (festgeschrieben, freigegeben) + .agents.local.md (gitignored, persönlich). Agenten lesen beides zu Beginn der Sitzung, aktualisieren den Notizblock am Ende der Sitzung und fördern im Laufe der Zeit stabile Muster. Funktioniert mit Claude Code, Cursor, Copilot und Windsurf. Bereit für Subagenten. Keine Plugins, keine Infrastruktur, keine Hintergrundprozesse.
