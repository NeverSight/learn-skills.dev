Постоянная локальная система памяти для агентов кодирования ИИ. Два файла, одна идея — AGENTS.md (фиксированный, общий) + .agents.local.md (gitignored, личный). Агенты читают как в начале сеанса, обновляют блокнот в конце сеанса, так и поддерживают стабильные шаблоны с течением времени. Работает в Claude Code, Cursor, Copilot, Windsurf. Готов к субагенту. Никаких плагинов, никакой инфраструктуры, никаких фоновых процессов.
