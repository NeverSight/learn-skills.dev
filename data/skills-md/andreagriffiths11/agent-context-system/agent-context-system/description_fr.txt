Un système de mémoire persistante uniquement locale pour les agents de codage d'IA. Deux fichiers, une idée — AGENTS.md (engagé, partagé) + .agents.local.md (gitignored, personnel). Les agents lisent les deux au début de la session, mettent à jour le bloc-notes à la fin de la session et promeuvent des modèles stables au fil du temps. Fonctionne avec Claude Code, Cursor, Copilot, Windsurf. Prêt pour les sous-agents. Pas de plugins, pas d'infrastructure, pas de processus en arrière-plan.
