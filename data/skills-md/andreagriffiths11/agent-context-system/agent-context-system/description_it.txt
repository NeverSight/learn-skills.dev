Un sistema di memoria persistente solo locale per agenti di codifica AI. Due file, un'idea: AGENTS.md (impegnato, condiviso) + .agents.local.md (gitignored, personale). Gli agenti leggono entrambi all'inizio della sessione, aggiornano gli appunti alla fine della sessione e promuovono modelli stabili nel tempo. Funziona con Claude Code, Cursor, Copilot, Windsurf. Predisposto per subagent. Nessun plugin, nessuna infrastruttura, nessun processo in background.
