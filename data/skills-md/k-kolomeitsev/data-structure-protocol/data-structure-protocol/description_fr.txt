Créez et parcourez DSP (Data Structure Protocol) – mémoire structurelle à long terme basée sur des graphiques des bases de code pour les agents LLM. Stocke les entités (modules, fonctions), leurs dépendances (importations), l'API publique (partagée/exportations) et les raisons de chaque connexion. À utiliser lorsque : (1) le projet a un répertoire .dsp/, (2) l'utilisateur demande de configurer la structure du projet DSP ou d'amorçage, (3) la création/modification/suppression de fichiers de code dans un projet suivi par DSP, (4) la navigation dans la structure du projet, la compréhension des dépendances ou la recherche de modules, (5) l'utilisateur mentionne DSP, dsp-cli, .dsp ou le mappage de structure.
