Erstellen und navigieren Sie DSP (Data Structure Protocol) – graphbasiertes langfristiges Strukturgedächtnis von Codebasen für LLM-Agenten. Speichert Entitäten (Module, Funktionen), ihre Abhängigkeiten (Importe), öffentliche API (gemeinsam genutzt/Exporte) und Gründe für jede Verbindung. Wird verwendet, wenn: (1) das Projekt über ein .dsp/-Verzeichnis verfügt, (2) der Benutzer die Einrichtung einer DSP- oder Bootstrap-Projektstruktur anfordert, (3) Codedateien in einem DSP-verfolgten Projekt erstellt/ändert/löscht, (4) in der Projektstruktur navigiert, Abhängigkeiten versteht oder Module findet, (5) der Benutzer DSP, dsp-cli, .dsp oder Strukturzuordnung erwähnt.
