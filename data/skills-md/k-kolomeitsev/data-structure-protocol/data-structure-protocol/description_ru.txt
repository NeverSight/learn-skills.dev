Создавайте и управляйте DSP (протоколом структуры данных) — долговременной структурной памятью кодовых баз на основе графов для агентов LLM. Хранит сущности (модули, функции), их зависимости (импорт), общедоступный API (общий/экспорт) и причины каждого соединения. Используйте, когда: (1) проект имеет каталог .dsp/, (2) пользователь запрашивает настройку DSP или структуру проекта начальной загрузки, (3) создание/изменение/удаление файлов кода в проекте, отслеживаемом DSP, (4) навигация по структуре проекта, понимание зависимостей или поиск модулей, (5) пользователь упоминает DSP, dsp-cli, .dsp или сопоставление структуры.
