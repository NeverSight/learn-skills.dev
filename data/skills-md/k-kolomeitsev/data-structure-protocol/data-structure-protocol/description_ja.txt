DSP (データ構造プロトコル) を構築して操作します。これは、LLM エージェントのコードベースのグラフベースの長期構造メモリです。エンティティ (モジュール、関数)、その依存関係 (インポート)、パブリック API (共有/エクスポート)、およびすべての接続の理由を保存します。 (1) プロジェクトに .dsp/ ディレクトリがある場合、(2) ユーザーが DSP またはブートストラップ プロジェクト構造のセットアップを要求する場合、(3) DSP 追跡プロジェクトでコード ファイルを作成/変更/削除する場合、(4) プロジェクト構造をナビゲートする、依存関係を理解する、またはモジュールを検索する場合、(5) ユーザーが DSP、dsp-cli、.dsp、または構造マッピングについて言及する場合に使用します。
