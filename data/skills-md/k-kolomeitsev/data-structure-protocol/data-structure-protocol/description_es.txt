Cree y navegue por DSP (Protocolo de estructura de datos): memoria estructural a largo plazo de bases de código basada en gráficos para agentes LLM. Almacena entidades (módulos, funciones), sus dependencias (importaciones), API pública (compartida/exportaciones) y motivos de cada conexión. Úselo cuando: (1) el proyecto tiene un directorio .dsp/, (2) el usuario solicita configurar DSP o la estructura del proyecto de arranque, (3) crear/modificar/eliminar archivos de código en un proyecto rastreado por DSP, (4) navegar por la estructura del proyecto, comprender dependencias o encontrar módulos, (5) el usuario menciona DSP, dsp-cli, .dsp o mapeo de estructura.
