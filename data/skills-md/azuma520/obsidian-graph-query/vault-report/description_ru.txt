Создавайте комплексные отчеты в виде графиков знаний для хранилищ Obsidian. Объединяет несколько графических запросов (статистика хранилища, концентраторы, мосты, потерянные данные) для создания структурированного аналитического отчета из 4 модулей со сводными таблицами данных. Требуется установленный навык запроса обсидиановых графиков. Используется для: отчета о хранилище, отчета о знаниях, анализа хранилища, состояния хранилища, отчета о графике знаний, 知識圖譜報告, хранилища 分析.
