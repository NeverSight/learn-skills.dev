Questa competenza dovrebbe essere utilizzata quando si lavora con memory-lancedb-pro, un plug-in MCP di memoria a lungo termine di livello produttivo per agenti AI OpenClaw. Utilizzare durante l'installazione, la configurazione o l'utilizzo di qualsiasi funzionalità di memory-lancedb-pro tra cui Smart Extraction, recupero ibrido, gestione del ciclo di vita della memoria, isolamento multi-ambito, governance di auto-miglioramento o qualsiasi strumento di memoria MCP (memory_recall, memory_store, memory_forget, memory_update, memory_stats, memory_list, self_improvement_log, self_improvement_extract_skill, auto_miglioramento_revisione).
