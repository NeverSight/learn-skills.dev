Cette compétence doit être utilisée lorsque vous travaillez avec memory-lancedb-pro, un plugin MCP de mémoire à long terme de qualité production pour les agents OpenClaw AI. À utiliser lors de l'installation, de la configuration ou de l'utilisation de toute fonctionnalité de memory-lancedb-pro, notamment l'extraction intelligente, la récupération hybride, la gestion du cycle de vie de la mémoire, l'isolation multi-portée, la gouvernance d'auto-amélioration ou tout autre outil de mémoire MCP (memory_recall, memory_store, memory_forget, memory_update, memory_stats, memory_list, self_improvement_log, self_improvement_extract_skill, self_improvement_review).
