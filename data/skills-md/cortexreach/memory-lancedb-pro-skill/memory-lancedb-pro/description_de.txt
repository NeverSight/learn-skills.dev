Diese Fähigkeit sollte bei der Arbeit mit „memory-lancedb-pro“, einem Langzeitspeicher-MCP-Plugin der Produktionsqualität für OpenClaw AI-Agenten, verwendet werden. Wird bei der Installation, Konfiguration oder Verwendung einer beliebigen Funktion von Memory-lancedb-pro verwendet, einschließlich Smart Extraction, Hybrid Retrieval, Memory Lifecycle Management, Multi-Scope-Isolation, Self-Improvement Governance oder beliebigen MCP-Speichertools (memory_recall, memory_store, memory_forget, memory_update, memory_stats, memory_list, self_improvement_log, self_improvement_extract_skill, self_improvement_review).
