قم بعرض تخفيض السعر كصور جميلة باستخدام واجهة سطر الأوامر لإطار الحبر. استخدم هذه المهارة عندما يريد المستخدم تحويل النص أو تخفيض السعر إلى صورة مرئية، أو إنشاء رسومات الوسائط الاجتماعية (التغريدات، أو منشورات Instagram، أو بطاقات LinkedIn، أو بطاقات الاقتباس، أو الميمات)، أو إنشاء صور مقتطف التعليمات البرمجية، أو إنشاء بطاقات مرئية أو ملصقات من المحتوى، أو إنتاج أي صورة من النص/تخفيض السعر. يتم تشغيله أيضًا عندما يذكر المستخدم "inkframe"، أو يطلب "عرض تخفيض السعر"، أو "خفض السعر إلى صورة"، أو "نص إلى صورة"، أو يريد جعل المحتوى قابلاً للمشاركة بشكل مرئي. حتى لو قالوا فقط "اجعل هذا يبدو جميلًا كصورة" أو "أنشئ رسمًا لهذا" - استخدم هذه المهارة.
