Procese las rebajas como hermosas imágenes con estilo utilizando la CLI de inkframe. Utilice esta habilidad siempre que el usuario desee convertir texto o rebajas en una imagen visual, crear gráficos de redes sociales (tweets, publicaciones de Instagram, tarjetas de LinkedIn, tarjetas de citas, memes), generar imágenes de fragmentos de código, crear tarjetas visuales o carteles a partir del contenido, o producir cualquier imagen a partir de texto/rebajas. También se activa cuando el usuario menciona "inkframe", solicita "renderizar rebaja", "rebaja de imagen", "texto a imagen" o quiere que el contenido se pueda compartir visualmente. Incluso si solo te dicen "haz que esto se vea bien como imagen" o "crea un gráfico para esto", usa esta habilidad.
