Obtenga contenido de texto legible desde cualquier URL utilizando el servicio defuddle.md. Utilice esta habilidad siempre que necesite extraer contenido de URL que sean difíciles de recuperar directamente, como publicaciones de x.com/twitter, páginas detrás del procesamiento de JavaScript o sitios que bloquean los buscadores estándar. Úselo también como respaldo general cuando WebFetch falle o devuelva resultados inutilizables. Activar: leer tweets/publicaciones X, obtener contenido de artículos de dominios complicados, "tomar el texto de este enlace" o cualquier URL donde la búsqueda directa no sea confiable.
