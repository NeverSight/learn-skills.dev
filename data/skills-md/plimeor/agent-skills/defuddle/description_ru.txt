Получайте читаемый текстовый контент с любого URL-адреса с помощью сервиса defuddle.md. Используйте этот навык всякий раз, когда вам нужно извлечь контент из URL-адресов, которые сложно получить напрямую — например, из сообщений x.com/twitter, страниц с отрисовкой JavaScript или сайтов, которые блокируют стандартные сборщики. Также используйте его в качестве общего запасного варианта, когда WebFetch дает сбой или возвращает непригодные для использования результаты. Запуск по: чтению твитов/X-сообщений, получению контента статьи из сложных доменов, «взятию текста по этой ссылке» или любому URL-адресу, где прямое получение ненадежно.
