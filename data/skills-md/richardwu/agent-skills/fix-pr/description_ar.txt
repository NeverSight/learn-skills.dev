قم بإصلاح المشكلات المتعلقة بالعلاقات العامة الحالية: يقوم روبوت العنوان (مثل Claude Code أو CodeRabbit أو GHA المخصص) بمراجعة التعليقات وإصلاح عمليات فحص CI الفاشلة. استخدمه عند مطالبتك بإصلاح العلاقات العامة، أو إصلاح تعليقات المراجعة، أو إصلاح CI، أو إصلاح عمليات التحقق. المشغلات على: إصلاح العلاقات العامة، إصلاح المراجعة، إصلاح CI، إصلاح الشيكات، إصلاح الشيكات الفاشلة.
