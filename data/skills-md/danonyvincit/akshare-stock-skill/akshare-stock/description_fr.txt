Utilisez la bibliothèque AKShare pour récupérer et analyser les données du marché chinois des actions A. AKShare est une interface de données financières Python gratuite et open source qui ne nécessite aucune inscription ni jeton. Il prend en charge : les cotations boursières en temps réel, les lignes K historiques, les états financiers, les mesures de valorisation, les flux de capitaux, les indicateurs techniques, les tâches planifiées, etc. Utilisez cette compétence lorsque l'utilisateur a besoin d'obtenir les prix des actions A, d'analyser les fondamentaux d'actions individuelles, d'interroger des données financières, de calculer des indicateurs techniques ou d'analyser les flux de capitaux.
