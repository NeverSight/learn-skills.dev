Use the AKShare library to fetch and analyze China A-share market data. AKShare is a free, open-source Python financial-data interface that requires no registration or token. It supports: real-time stock quotes, historical K-lines, financial statements, valuation metrics, capital flows, technical indicators, scheduled tasks, etc. Use this skill when the user needs to obtain A-share prices, analyze individual-stock fundamentals, query financial data, calculate technical indicators, or analyze capital flows.
