Configurez et gérez un système de banque de mémoire pour la continuité du contexte entre les sessions entre les agents de codage d'IA. À utiliser lorsque l'utilisateur mentionne « banque de mémoire » avec une intention d'action : configuration, installation, initialisation, initialisation, mise à jour, actualisation, synchronisation, statut, vérification, lecture, affichage, révision, affichage ou équivalents dans n'importe quelle langue (par exemple, turc : kur, kurulum, güncelle, durumu, oku ; allemand : einrichten, aktualisieren ; espagnol : configurar, actualizar ; français : installer, mettre à jour). Prend en charge Claude Code, Cursor, Windsurf, Cline, GitHub Copilot, Roo Code, Aider, Antigravity et OpenAI Codex.
