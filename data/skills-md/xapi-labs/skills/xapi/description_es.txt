Utilice la CLI de xapi para acceder a datos externos en tiempo real: perfiles, tweets y líneas de tiempo de Twitter/X, precios y metadatos de tokens criptográficos, búsqueda web, noticias y procesamiento de textos con IA (resumen, reescritura, chat, incrustaciones). Active esta habilidad cada vez que el usuario quiera buscar un usuario de Twitter, obtener detalles de un tweet, verificar los precios de las criptomonedas, buscar en la web o en noticias, generar incrustaciones, resumir o reescribir texto, o llamar a cualquier API de terceros a través de xapi. Utilice también esta habilidad cuando el usuario mencione xapi, pregunte sobre las capacidades o API disponibles, o quiera descubrir qué servicios externos son accesibles.
