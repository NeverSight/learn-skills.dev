Utilisez xapi CLI pour accéder aux données externes en temps réel : profils Twitter/X, tweets et chronologies, prix et métadonnées des jetons cryptographiques, recherche sur le Web, actualités et traitement de texte IA (résumer, réécrire, discuter, intégrations). Déclenchez cette compétence chaque fois que l'utilisateur souhaite rechercher un utilisateur de Twitter, obtenir des détails sur un tweet, vérifier les prix des cryptomonnaies, rechercher sur le Web ou dans les actualités, générer des intégrations, résumer ou réécrire du texte, ou appeler une API tierce via xapi. Utilisez également cette compétence lorsque l'utilisateur mentionne xapi, pose des questions sur les fonctionnalités ou API disponibles ou souhaite découvrir quels services externes sont accessibles.
