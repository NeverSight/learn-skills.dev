Utilizza la CLI xapi per accedere a dati esterni in tempo reale: profili Twitter/X, tweet e sequenze temporali, prezzi e metadati dei token crittografici, ricerca web, notizie ed elaborazione di testi AI (riepilogo, riscrittura, chat, incorporamenti). Attiva questa abilità ogni volta che l'utente desidera cercare un utente Twitter, ottenere dettagli sui tweet, controllare i prezzi delle criptovalute, cercare sul Web o nelle notizie, generare incorporamenti, riassumere o riscrivere testo o chiamare qualsiasi API di terze parti tramite xapi. Utilizza questa abilità anche quando l'utente menziona xapi, chiede informazioni sulle funzionalità o API disponibili o desidera scoprire quali servizi esterni sono accessibili.
