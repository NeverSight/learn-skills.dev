xapi CLI を使用して、リアルタイムの外部データ (Twitter/X プロファイル、ツイート、タイムライン、暗号トークンの価格とメタデータ、Web 検索、ニュース、AI テキスト処理 (要約、書き換え、チャット、埋め込み)) にアクセスします。ユーザーが Twitter ユーザーの検索、ツイートの詳細の取得、仮想通貨価格の確認、Web またはニュースの検索、埋め込みの生成、テキストの要約または書き換え、または xapi を介したサードパーティ API の呼び出しを行うときはいつでも、このスキルをトリガーします。また、ユーザーが xapi について言及したり、利用可能な機能や API について尋ねたり、アクセス可能な外部サービスを確認したい場合にも、このスキルを使用します。
