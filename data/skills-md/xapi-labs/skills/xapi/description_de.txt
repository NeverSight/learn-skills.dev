Verwenden Sie die xapi-CLI, um in Echtzeit auf externe Daten zuzugreifen – Twitter/X-Profile, Tweets und Zeitleisten, Krypto-Token-Preise und Metadaten, Websuche, Nachrichten und KI-Textverarbeitung (Zusammenfassen, Umschreiben, Chat, Einbettungen). Lösen Sie diesen Skill immer dann aus, wenn der Benutzer einen Twitter-Benutzer suchen, Tweet-Details abrufen, Kryptopreise überprüfen, das Web oder Nachrichten durchsuchen, Einbettungen generieren, Text zusammenfassen oder umschreiben oder eine Drittanbieter-API über xapi aufrufen möchte. Verwenden Sie diese Fähigkeit auch, wenn der Benutzer xapi erwähnt, nach verfügbaren Funktionen oder APIs fragt oder herausfinden möchte, auf welche externen Dienste zugegriffen werden kann.
