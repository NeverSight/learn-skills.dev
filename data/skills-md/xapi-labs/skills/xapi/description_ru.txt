Используйте интерфейс командной строки xapi для доступа к внешним данным в режиме реального времени — профилям Twitter/X, твитам и временным графикам, ценам и метаданным крипто-токенов, веб-поиску, новостям и обработке текста с помощью ИИ (обобщение, переписывание, общение в чате, встраивание). Активируйте этот навык всякий раз, когда пользователь хочет найти пользователя Твиттера, получить подробную информацию о твите, проверить цены на криптовалюту, выполнить поиск в Интернете или новостях, сгенерировать встраивания, обобщить или переписать текст или вызвать любой сторонний API через xapi. Также используйте этот навык, когда пользователь упоминает xapi, спрашивает о доступных возможностях или API или хочет узнать, какие внешние сервисы доступны.
