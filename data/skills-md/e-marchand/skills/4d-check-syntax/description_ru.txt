Проверьте синтаксис и скомпилируйте 4D-проект, чтобы найти ошибки и проблемы с типом. Используйте этот навык, когда пользователь хочет скомпилировать, проверить синтаксис или проверить 4D-проект. Создает метод _compile, если он отсутствует, затем запускает его с помощьюtool4d. Возвращает ошибки компиляции в формате JSON для удобства анализа.
