Esegui un metodo 4D utilizzando lo strumento da riga di comando tool4d. Utilizza questa competenza quando l'utente desidera eseguire, testare o eseguire un metodo di progetto 4D. Trova automaticamente tool4d.app nelle posizioni standard (estensioni VS Code, Antigravity). Supporta l'esecuzione di qualsiasi metodo di progetto con la modalità --dataless per eseguire test senza file di dati.
