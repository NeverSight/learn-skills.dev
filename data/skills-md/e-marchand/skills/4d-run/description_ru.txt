Запустите 4D-метод с помощью инструмента командной строкиtool4d. Используйте этот навык, когда пользователь хочет выполнить, протестировать или запустить метод 4D-проекта. Автоматически находитtool4d.app в стандартных местах (расширения VS Code, Антигравитация). Поддерживает запуск любого метода проекта в режиме --dataless для тестирования без файла данных.
