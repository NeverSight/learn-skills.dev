Ejecute un método 4D utilizando la herramienta de línea de comandos tool4d. Utilice esta habilidad cuando el usuario quiera ejecutar, probar o ejecutar un método de proyecto 4D. Encuentra automáticamente tool4d.app en ubicaciones estándar (extensiones de VS Code, Antigravity). Admite la ejecución de cualquier método de proyecto con el modo --dataless para realizar pruebas sin archivo de datos.
