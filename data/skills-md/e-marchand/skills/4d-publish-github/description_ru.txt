Опубликуйте 4D-проект на GitHub с помощью gh CLI. Используйте этот навык, когда пользователь хочет опубликовать, отправить или поделиться 4D-проектом на GitHub. Создает удаленный репозиторий, инициализирует git, добавляет рабочие процессы CI/CD для сборки и выпуска. Поддерживает рабочие процессы выпуска по тегу или выпуска при создании.
