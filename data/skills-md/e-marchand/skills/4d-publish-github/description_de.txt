Veröffentlichen Sie ein 4D-Projekt auf GitHub mit der gh CLI. Verwenden Sie diesen Skill, wenn der Benutzer ein 4D-Projekt auf GitHub veröffentlichen, pushen oder teilen möchte. Erstellt ein Remote-Repository, initialisiert Git und fügt CI/CD-Workflows zum Erstellen und Freigeben hinzu. Unterstützt Release-on-Tag- oder Release-on-Create-Workflows.
