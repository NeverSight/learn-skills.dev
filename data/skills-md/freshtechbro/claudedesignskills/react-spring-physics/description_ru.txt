Библиотека анимации на основе физики, объединяющая React Spring (динамика пружины, интеграция жестов, анимация с частотой 60 кадров в секунду) и Popmotion (утилиты низкоуровневой компонуемой анимации, реактивные потоки). Используйте при создании плавной, естественной анимации пользовательского интерфейса, интерфейсов, управляемых жестами, физического моделирования или подпружиненных взаимодействий. Запускается при выполнении задач, включающих хуки React Spring, физику Spring, прокрутку по инерции, движение на основе физики, композицию анимации или естественные движения пользовательского интерфейса. Альтернативный физический подход к motion-framer для более физически точной анимации.
