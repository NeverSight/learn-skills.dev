React Spring(스프링 역학, 제스처 통합, 60fps 애니메이션)과 Popmotion(낮은 수준의 구성 가능한 애니메이션 유틸리티, 반응 스트림)을 결합한 물리 기반 애니메이션 라이브러리입니다. 유연하고 자연스러운 느낌의 UI 애니메이션, 제스처 기반 인터페이스, 물리 시뮬레이션 또는 스프링 로드 상호 작용을 구축할 때 사용합니다. React Spring 후크, 스프링 물리학, 관성 스크롤링, 물리학 기반 모션, 애니메이션 구성 또는 자연스러운 UI 움직임과 관련된 작업을 트리거합니다. 보다 물리적으로 정확한 애니메이션을 위한 모션 프레이머에 대한 대체 물리 접근 방식입니다.
