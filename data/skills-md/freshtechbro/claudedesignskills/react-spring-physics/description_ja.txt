React Spring (スプリング ダイナミクス、ジェスチャー統合、60fps アニメーション) と Popmotion (低レベルのコンポーザブル アニメーション ユーティリティ、リアクティブ ストリーム) を組み合わせた物理ベースのアニメーション ライブラリ。滑らかで自然な感触の UI アニメーション、ジェスチャー駆動のインターフェイス、物理シミュレーション、またはスプリング式インタラクションを構築するときに使用します。 React Spring フック、スプリング物理学、慣性スクロール、物理ベースのモーション、アニメーション合成、または自然な UI の動きを含むタスクをトリガーします。より物理的に正確なアニメーションを実現するモーション フレーマーへの代替物理アプローチ。
