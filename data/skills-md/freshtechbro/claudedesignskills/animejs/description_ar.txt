محرك رسوم متحركة JavaScript متعدد الاستخدامات لكائنات DOM وCSS وSVG وJavaScript. يُستخدم عند إنشاء رسوم متحركة قائمة على المخطط الزمني، أو تأثيرات متداخلة، أو تحويل SVG، أو تسلسلات إطارات رئيسية، أو رسوم متحركة معقدة مصممة. تشغيل المهام التي تتضمن Anime.js، أو الرسوم المتحركة للمخطط الزمني، أو التسلسلات المتدرجة، أو الرسوم المتحركة لمسار SVG، أو التحويل، أو تصميم الرقصات المتحركة متعددة الخطوات. بديل لـ GSAP للرسوم المتحركة ذات SVG الثقيلة والمشاريع المستقلة عن React.
