Alibaba Cloud Yunyxiao DevOps プラットフォーム統合スキル。コード管理、パイプライン、プロジェクト管理、組織管理、成果物リポジトリ、アプリケーション配信、テスト管理などの完全な DevOps 機能を提供します。ユーザーが Yunyxiao プラットフォームと対話する必要がある場合に使用されます。これには、(1) コード リポジトリの操作 (ブランチ、ファイル、マージ リクエスト) (2) パイプラインの作成と実行 (3) プロジェクトと作業項目の管理 (4) 組織メンバーの管理 (5) アーティファクト リポジトリのクエリ (6) アプリケーションのデプロイとリリース (7) テスト ケースの管理が含まれます。トリガーワード: Yunyxiao、Codeup、Flow、Projex、AppStack、DevOps。
