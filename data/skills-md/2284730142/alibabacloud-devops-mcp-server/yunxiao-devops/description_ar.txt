مهارات تكامل منصة Alibaba Cloud Effect DevOps. يوفر إمكانات DevOps كاملة بما في ذلك إدارة التعليمات البرمجية وخطوط الأنابيب وإدارة المشاريع وإدارة المؤسسة ومستودعات العناصر وتسليم التطبيقات وإدارة الاختبار. يُستخدم عندما يحتاج المستخدمون إلى التفاعل مع منصة Effect، بما في ذلك: (1) عمليات مستودع التعليمات البرمجية (الفروع والملفات وطلبات الدمج) (2) إنشاء خطوط الأنابيب وتنفيذها (3) إدارة المشروع وعناصر العمل (4) إدارة أعضاء المؤسسة (5) استعلامات مستودع العناصر (6) نشر التطبيق وإصداره (7) إدارة حالة الاختبار. الكلمات المحفزة: Effect، Codeup، Flow، Projex، AppStack، DevOps.
