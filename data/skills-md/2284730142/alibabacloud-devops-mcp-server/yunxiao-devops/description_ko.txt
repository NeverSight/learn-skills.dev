Alibaba Cloud Yunyxiao DevOps 플랫폼 통합 기술. 코드 관리, 파이프라인, 프로젝트 관리, 조직 관리, 아티팩트 저장소, 애플리케이션 제공 및 테스트 관리를 포함한 완전한 DevOps 기능을 제공합니다. 사용자가 다음을 포함하여 Yunyxiao 플랫폼과 상호 작용해야 할 때 사용됩니다.
