Capacità di integrazione della piattaforma DevOps Cloud Efficiency di Alibaba Cloud. Fornisce funzionalità DevOps complete tra cui gestione del codice, pipeline, gestione dei progetti, gestione dell'organizzazione, repository di artefatti, distribuzione delle applicazioni e gestione dei test. Da utilizzare quando si interagisce con la piattaforma Cloud Efficiency, coprendo: (1) operazioni del repository di codice (rami, file, richieste di unione), (2) creazione ed esecuzione di pipeline, (3) gestione di progetti ed elementi di lavoro, (4) gestione dei membri dell'organizzazione, (5) query del repository di artefatti, (6) distribuzione e rilascio dell'applicazione, (7) gestione dei test case. Parole chiave: Cloud Efficiency, Codeup, Flow, Projex, AppStack, DevOps.
