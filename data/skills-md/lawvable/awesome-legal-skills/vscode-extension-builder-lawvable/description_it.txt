Crea estensioni VS Code da zero o converti le app JS/React/Vue esistenti. Supporta comandi, visualizzazioni Web (React/Vue), editor personalizzati, visualizzazioni ad albero e integrazione di agenti AI tramite IPC file-bridge. Da utilizzare quando l'utente desidera creare un'estensione VS Code, convertire un'app Web in un'estensione, aggiungere visualizzazioni Web o interfacce utente personalizzate a VS Code, implementare visualizzazioni ad albero, creare editor di file personalizzati, integrare con agenti AI o creare pacchetti/pubblicare estensioni (.vsix).
