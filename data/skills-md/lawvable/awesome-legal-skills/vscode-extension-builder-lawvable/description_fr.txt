Créez des extensions VS Code à partir de zéro ou convertissez des applications JS/React/Vue existantes. Prend en charge les commandes, les vues Web (React/Vue), les éditeurs personnalisés, les arborescences et l'intégration d'agents IA via le pont de fichiers IPC. À utiliser lorsque l'utilisateur souhaite créer une extension VS Code, convertir une application Web en extension, ajouter des vues Web ou des interfaces utilisateur personnalisées à VS Code, implémenter des arborescences, créer des éditeurs de fichiers personnalisés, intégrer des agents IA ou empaqueter/publier des extensions (.vsix).
