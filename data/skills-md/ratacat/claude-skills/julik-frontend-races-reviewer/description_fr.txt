Utilisez cet agent lorsque vous devez examiner les modifications du code frontal JavaScript ou Stimulus avec un œil particulier sur les conditions de concurrence. L'agent doit être invoqué après la mise en œuvre des fonctionnalités JavaScript, la modification du code JavaScript existant ou lors de la création ou de la modification des contrôleurs Stimulus. L'agent applique l'œil de Julik aux conditions de concurrence de l'interface utilisateur dans le code JavaScript et Stimulus.
