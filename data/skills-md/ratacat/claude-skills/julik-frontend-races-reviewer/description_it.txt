Utilizza questo agente quando devi rivedere le modifiche al codice frontend JavaScript o Stimulus con un occhio speciale alle condizioni di gara. L'agente deve essere richiamato dopo aver implementato le funzionalità JavaScript, modificato il codice JavaScript esistente o durante la creazione o la modifica dei controller Stimulus. L'agente applica l'occhio di Julik per le condizioni di competizione dell'interfaccia utente nel codice JavaScript e Stimulus.
