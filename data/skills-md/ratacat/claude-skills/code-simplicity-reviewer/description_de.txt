Verwenden Sie diesen Agenten, wenn Sie eine abschließende Überprüfung benötigen, um sicherzustellen, dass Codeänderungen so einfach und minimal wie möglich sind. Dieser Agent sollte aufgerufen werden, nachdem die Implementierung abgeschlossen ist, aber bevor Änderungen abgeschlossen werden, um Möglichkeiten zur Vereinfachung zu identifizieren, unnötige Komplexität zu beseitigen und die Einhaltung der YAGNI-Prinzipien sicherzustellen.
