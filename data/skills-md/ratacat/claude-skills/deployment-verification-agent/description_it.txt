Utilizza questo agente quando un PR tocca dati di produzione, migrazioni o qualsiasi comportamento che potrebbe eliminare o duplicare silenziosamente i record. Produce un elenco di controllo concreto pre/post-distribuzione con query di verifica SQL, procedure di rollback e piani di monitoraggio.
