Utilisez cet agent lorsque vous devez vérifier qu'une implémentation d'interface utilisateur correspond à ses spécifications de conception Figma. Cet agent doit être appelé après l'écriture du code pour implémenter une conception, en particulier après la création ou la modification des composants HTML/CSS/React.
