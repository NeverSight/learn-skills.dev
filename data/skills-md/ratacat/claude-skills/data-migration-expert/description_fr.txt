Utilisez cet agent lors de l'examen des PR qui concernent les migrations de bases de données, les remplissages de données ou tout code transformant les données de production. Cet agent valide les mappages d'ID par rapport à la réalité de production, vérifie les valeurs échangées, vérifie la sécurité de la restauration et garantit l'intégrité des données lors des modifications de schéma. Indispensable pour toute migration impliquant des mappages d’ID, des changements de noms de colonnes ou des transformations de données. <exemple>Contexte : l'utilisateur dispose d'un PR avec des migrations de bases de données qui impliquent des mappages d'ID. utilisateur : "Examinez ce PR qui migre d'action_id vers action_module_name" assistant : "J'utiliserai l'agent data-migration-expert pour valider les mappages d'ID et la sécurité de la migration" <commentary>Étant donné que le PR implique des mappages d'ID et une migration de données, utilisez l'expert en migration de données pour vérifier que les mappages correspondent à la production et vérifier les valeurs échangées.</commentary></example> <example>Contexte : l'utilisateur dispose d'une migration qui transforme les valeurs d'énumération. utilisateur : « Cette migration convertit les entiers de statut en énumérations de chaînes » Assistant : « Laissez-moi avoir l'ex de migration de données...
