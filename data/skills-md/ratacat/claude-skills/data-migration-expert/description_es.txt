Utilice este agente cuando revise relaciones públicas que afecten a migraciones de bases de datos, reposiciones de datos o cualquier código que transforme datos de producción. Este agente valida las asignaciones de ID con la realidad de producción, busca valores intercambiados, verifica la seguridad de reversión y garantiza la integridad de los datos durante los cambios de esquema.
