Комплексный обзор кода в стиле обжарки. Анализирует безопасность, архитектуру, сложность, соответствие TDD, безопасность типов, обработку ошибок, именование, мертвый код, производительность, зависимости, дизайн API, качество интерфейса и гигиену git. Предоставляет результаты в виде занимательных обжарок с практическими исправлениями. Используйте /roast для проверки вашей кодовой базы. Поддерживает --уровень (нежный/острый/дикий/сенсей).
