Costruisci pipeline automatizzate per dati di imaging medico: conversione del formato (da DICOM a NIfTI, PNG, HDF5), elaborazione batch, preparazione di set di dati ML, esportazione di dati di ricerca e flussi di lavoro ETL di imaging. Da utilizzare durante la conversione di DICOM in altri formati, la preparazione di set di dati di imaging per l'apprendimento automatico, la creazione di pipeline di dati di ricerca, l'elaborazione batch di immagini mediche, l'estrazione di funzionalità di imaging o l'automazione dei flussi di lavoro di imaging.
