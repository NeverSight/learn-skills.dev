Lavora con file di imaging medico DICOM a livello di codice utilizzando pydicom, dcmtk e strumenti correlati. Leggi, scrivi, modifica e convalida tag DICOM, dati pixel e metadati. Da utilizzare durante la lettura di file DICOM, l'estrazione di metadati, la modifica di tag, l'utilizzo di dati pixel, la conversione di sintassi di trasferimento, la convalida della conformità DICOM o lo scripting di operazioni DICOM in blocco.
