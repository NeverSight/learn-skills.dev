العمل مع ملفات التصوير الطبي DICOM برمجيًا باستخدام pydicom وdcmtk والأدوات ذات الصلة. قراءة وكتابة وتعديل والتحقق من صحة علامات DICOM وبيانات البكسل والبيانات التعريفية. يُستخدم عند قراءة ملفات DICOM، أو استخراج بيانات التعريف، أو تعديل العلامات، أو العمل مع بيانات البكسل، أو تحويل صيغ النقل، أو التحقق من توافق DICOM، أو البرمجة النصية لعمليات DICOM المجمعة.
