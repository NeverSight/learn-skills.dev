Trabaje con archivos de imágenes médicas DICOM mediante programación utilizando pydicom, dcmtk y herramientas relacionadas. Lea, escriba, modifique y valide etiquetas DICOM, datos de píxeles y metadatos. Úselo al leer archivos DICOM, extraer metadatos, modificar etiquetas, trabajar con datos de píxeles, convertir sintaxis de transferencia, validar la conformidad DICOM o programar operaciones DICOM masivas.
