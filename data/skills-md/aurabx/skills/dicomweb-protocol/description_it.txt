Crea client e integrazioni DICOMweb utilizzando i protocolli WADO-RS (recupero), STOW-RS (negozio) e QIDO-RS (query). Gestisce la codifica MIME multiparte, la negoziazione del contenuto, i parametri di query e l'autenticazione. Da utilizzare durante la creazione di client DICOMweb, l'interrogazione di un server DICOMweb, il caricamento di DICOM tramite STOW-RS, il recupero di immagini tramite WADO-RS o l'integrazione con qualsiasi sistema compatibile con DICOMweb tra cui Aurabox, Orthanc, dcm4chee o API Google Cloud Healthcare.
