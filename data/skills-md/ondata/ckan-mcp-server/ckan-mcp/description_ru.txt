Сервер MCP для изучения порталов открытых данных на основе CKAN (dati.gov.it, data.gov, data.gov.uk, open.canada.ca, demo.ckan.org и любой другой экземпляр CKAN). Также охватывает data.europa.eu через REST API (не CKAN). Используйте этот навык всякий раз, когда пользователь: спрашивает об открытых данных, общедоступных наборах данных или порталах данных; упоминает страну, регион или город в связи с данными или статистикой; спрашивает о прозрачности правительства, публичных отчетах или официальных публикациях; спрашивает «где я могу найти данные по X», «есть ли наборы данных по Y» или «какие данные публикует организация Z»; необходимо искать, фильтровать, исследовать или анализировать любой каталог открытых данных; или упоминает известный портал по имени или URL-адресу.
