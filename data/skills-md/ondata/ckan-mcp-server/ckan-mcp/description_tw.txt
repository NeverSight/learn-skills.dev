MCP 伺服器，用於探索基於 CKAN 的開放資料入口網站（dati.gov.it、data.gov、data.gov.uk、open.canada.ca、demo.ckan.org 和任何其他 CKAN 實例）。也透過其 REST API（不是 CKAN）覆蓋 data.europa.eu。當使用者出現以下情況時，請使用此技能： 詢問開放資料、公共資料集或資料入口網站；在資料或統計資料中提及國家、地區或城市；詢問政府透明度、公共記錄或官方出版物；詢問「在哪裡可以找到有關 X 的資料」、「是否有有關 Y 的資料集」或「組織 Z 發佈了哪些資料」；需要搜尋、過濾資料、分析任何網站或「開放」目錄或「開放」。
