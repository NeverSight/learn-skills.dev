CKAN ベースのオープン データ ポータル (dati.gov.it、data.gov、data.gov.uk、open.canada.ca、demo.ckan.org、およびその他の CKAN インスタンス) を探索するための MCP サーバー。 REST API (CKAN ではなく) を介した data.europa.eu もカバーします。ユーザーがオープン データ、公開データセット、またはデータ ポータルについて質問するときは常にこのスキルを使用します。データまたは統計に関連して国、地域、または都市に言及する。政府の透明性、公記録、または公式出版物について質問する。 「X のデータはどこで見つけられますか」、「Y に関するデータセットはありますか」、または「組織 Z が公開しているデータは何ですか」と尋ねます。オープンデータカタログを検索、フィルタリング、探索、または分析する必要がある。または、既知のポータルを名前または URL で言及します。
