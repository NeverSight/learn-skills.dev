MCP 服务器，用于探索基于 CKAN 的开放数据门户（dati.gov.it、data.gov、data.gov.uk、open.canada.ca、demo.ckan.org 和任何其他 CKAN 实例）。还通过其 REST API（不是 CKAN）覆盖 data.europa.eu。当用户出现以下情况时，请使用此技能： 询问开放数据、公共数据集或数据门户；在数据或统计数据中提及国家、地区或城市；询问政府透明度、公共记录或官方出版物；询问“在哪里可以找到有关 X 的数据”、“是否有有关 Y 的数据集”或“组织 Z 发布了哪些数据”；需要搜索、过滤、探索或分析任何开放数据目录；或通过名称或 URL 提及已知门户。
