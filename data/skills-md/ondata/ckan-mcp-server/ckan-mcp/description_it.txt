Server MCP per esplorare portali di dati aperti basati su CKAN (dati.gov.it, data.gov, data.gov.uk, open.canada.ca, demo.ckan.org e qualsiasi altra istanza CKAN). Copre anche data.europa.eu tramite la sua API REST (non CKAN). Utilizza questa competenza ogni volta che l'utente: chiede informazioni su dati aperti, set di dati pubblici o portali di dati; menziona un paese, una regione o una città in relazione a dati o statistiche; chiede informazioni sulla trasparenza del governo, sui registri pubblici o sulle pubblicazioni ufficiali; chiede "dove posso trovare dati su X", "ci sono set di dati su Y" o "quali dati pubblica l'organizzazione Z"; ha bisogno di cercare, filtrare, esplorare o analizzare qualsiasi catalogo di dati aperti; o menziona un portale conosciuto per nome o URL.
