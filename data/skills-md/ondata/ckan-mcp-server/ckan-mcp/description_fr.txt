Serveur MCP pour explorer les portails de données ouvertes basés sur CKAN (dati.gov.it, data.gov, data.gov.uk, open.canada.ca, demo.ckan.org et toute autre instance de CKAN). Couvre également data.europa.eu via son API REST (et non CKAN). Utilisez cette compétence chaque fois que l'utilisateur : pose des questions sur les données ouvertes, les ensembles de données publics ou les portails de données ; mentionne un pays, une région ou une ville en relation avec des données ou des statistiques ; des questions sur la transparence du gouvernement, les archives publiques ou les publications officielles ; demande « où puis-je trouver des données sur X », « existe-t-il des ensembles de données sur Y » ou « quelles données l'organisation Z publie-t-elle » ; doit rechercher, filtrer, explorer ou analyser n'importe quel catalogue de données ouvertes ; ou mentionne un portail connu par son nom ou son URL.
