خادم MCP لاستكشاف بوابات البيانات المفتوحة المستندة إلى CKAN (dati.gov.it وdata.gov وdata.gov.uk وopen.canada.ca وdemo.ckan.org وأي مثيل CKAN آخر). يغطي أيضًا data.europa.eu عبر REST API (وليس CKAN). استخدم هذه المهارة عندما يسأل المستخدم عن البيانات المفتوحة، أو مجموعات البيانات العامة، أو بوابات البيانات؛ يذكر بلدًا أو منطقة أو مدينة فيما يتعلق بالبيانات أو الإحصائيات؛ يسأل عن الشفافية الحكومية، والسجلات العامة، أو المنشورات الرسمية؛ يسأل "أين يمكنني العثور على بيانات حول X"، أو "هل توجد مجموعات بيانات حول Y"، أو "ما هي البيانات التي تنشرها المؤسسة Z"؛ يحتاج إلى البحث أو التصفية أو الاستكشاف أو التحليل في أي كتالوج للبيانات المفتوحة؛ أو يذكر بوابة معروفة بالاسم أو عنوان URL.
