Generieren oder aktualisieren Sie Laravel-Plugin-REST-APIs in AureusERP für jedes Modul. Wird verwendet, wenn Sie neue API-Ressourcen implementieren oder vorhandene Endpunkte umgestalten, um sie an gemeinsame Projektkonventionen für Routen, Controller, Anforderungen, Ressourcen, Richtlinien, Spatie QueryBuilder-Filterung/Einbindung/Sortierung, Soft-Delete-Wiederherstellung/Erzwingung von Endpunkten und Scribe-Attribute/Dokumentation anzupassen.
