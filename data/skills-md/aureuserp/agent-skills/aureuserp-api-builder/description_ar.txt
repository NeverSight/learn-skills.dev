قم بإنشاء أو تحديث واجهات برمجة التطبيقات REST API الخاصة ببرنامج Laravel في AureusERP لأي وحدة نمطية. يُستخدم عند تنفيذ موارد واجهة برمجة التطبيقات الجديدة أو إعادة بناء نقاط النهاية الحالية لمطابقة اصطلاحات المشروع المشتركة للمسارات ووحدات التحكم والطلبات والموارد والسياسات وتصفية/تتضمن/فرز Spatie QueryBuilder ونقاط نهاية الاستعادة/فرض الحذف الناعم وسمات/وثائق Scribe.
