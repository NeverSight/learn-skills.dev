Cree y refactorice pruebas de funciones de Pest para las API del complemento AureusERP con ayudas reutilizables, configuración de autenticación de permisos de políticas, cargas útiles de fábrica, afirmaciones JSON estables y estrategia de arranque de instalación de complementos. Úselo al crear o actualizar pruebas de API en `plugins/*/*/tests`. Esta habilidad está estructurada para admitir una futura pista de casos de prueba de Filament sin cambiar el comportamiento actual de la API.
