قم ببناء وإعادة بناء اختبارات ميزات Pest لواجهات برمجة تطبيقات المكونات الإضافية AureusERP مع مساعدين قابلين لإعادة الاستخدام، وإعداد مصادقة أذونات السياسة، والحمولات النافعة من المصنع، وتأكيدات JSON المستقرة، واستراتيجية التمهيد لتثبيت المكونات الإضافية. يُستخدم عند إنشاء اختبارات واجهة برمجة التطبيقات (API) أو تحديثها ضمن "plugins/*/*/tests". تم تصميم هذه المهارة لدعم مسار حالة اختبار Filament في المستقبل دون تغيير سلوك واجهة برمجة التطبيقات الحالية.
