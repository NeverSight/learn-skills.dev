Создавайте и реорганизуйте тесты функций Pest для API-интерфейсов плагинов AureusERP с помощью многоразовых помощников, настройкой аутентификации на основе политик, заводскими полезными нагрузками, стабильными утверждениями JSON и стратегией начальной загрузки при установке плагина. Используйте при создании или обновлении тестов API в разделе «plugins/*/*/tests». Этот навык структурирован для поддержки будущих тестовых сценариев Filament без изменения текущего поведения API.
