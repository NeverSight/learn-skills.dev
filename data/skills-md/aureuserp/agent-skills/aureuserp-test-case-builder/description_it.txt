Crea e rifattorizza i test delle funzionalità Pest per le API dei plugin AureusERP con helper riutilizzabili, configurazione di autenticazione delle policy, payload di fabbrica, asserzioni JSON stabili e strategia di bootstrap per l'installazione dei plugin. Da utilizzare durante la creazione o l'aggiornamento dei test API in `plugins/*/*/tests`. Questa competenza è strutturata per supportare un futuro percorso di test di Filament senza modificare il comportamento attuale dell'API.
