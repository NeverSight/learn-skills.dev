Créez et refactorisez des tests de fonctionnalités Pest pour les API du plugin AureusERP avec des assistants réutilisables, une configuration d'authentification par autorisation de politique, des charges utiles d'abord en usine, des assertions JSON stables et une stratégie d'amorçage d'installation de plugin. À utiliser lors de la création ou de la mise à jour de tests API sous `plugins/*/*/tests`. Cette compétence est structurée pour prendre en charge un futur scénario de test Filament sans modifier le comportement actuel de l'API.
