قم بخدمة الأصول الثابتة في Litestar باستخدام `create_static_files_router()` ومسارات التثبيت الواضحة ووضع HTML وسياسة التحكم في ذاكرة التخزين المؤقت وعكس عنوان URL وأنظمة الملفات المخصصة وحدود الدليل الآمنة للارتباط الرمزي. يُستخدم عند الكشف عن JS أو CSS أو الصور أو العناصر القابلة للتنزيل أو HTML الثابت من الأدلة الخاضعة للتحكم. لا تستخدمه لنقل التحميل، أو نشر ملف قابل للكتابة بواسطة المستخدم، أو نقاط النهاية العشوائية لتنزيل الملفات التي تحتاج إلى منطق تفويض لكل طلب.
