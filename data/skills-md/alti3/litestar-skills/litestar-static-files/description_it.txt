Servi risorse statiche in Litestar con `create_static_files_router()`, percorsi di montaggio espliciti, modalità HTML, policy di controllo della cache, inversione URL, file system personalizzati e limiti di directory sicuri per i collegamenti simbolici. Da utilizzare quando si espongono JS, CSS, immagini, artefatti scaricabili o HTML statico da directory controllate. Non utilizzare per l'inserimento di caricamenti, la pubblicazione di file scrivibili dall'utente o endpoint di download di file arbitrari che richiedono una logica di autorizzazione per richiesta.
