Обслуживайте статические ресурсы в Litestar с помощью `create_static_files_router()`, явных путей монтирования, режима HTML, политики управления кэшем, обращения URL-адресов, пользовательских файловых систем и границ каталогов, безопасных для символических ссылок. Используйте при предоставлении доступа к JS, CSS, изображениям, загружаемым артефактам или статическому HTML из контролируемых каталогов. Не используйте для приема загрузки, публикации файлов, записываемых пользователем, или произвольных конечных точек загрузки файлов, которым требуется логика авторизации для каждого запроса.
