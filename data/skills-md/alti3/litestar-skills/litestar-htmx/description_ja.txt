「HTMXPlugin」、「HTMXRequest」、「request.htmx」、「HTMXTemplate」、および「HXLocation」、「PushUrl」、「ReplaceUrl」、「Reswap」、「Retarget」、「TriggerEvent」、「ClientRedirect」などの HTMX 応答クラスを使用して、HTMX 駆動の Litestar フローを構築します。 「ClientRefresh」および「HXStopPolling」。サーバーレンダリングによる部分更新、プログレッシブ拡張、ポーリング、リダイレクト、または履歴を認識した HTML インタラクションを実装する場合に使用します。 HTML または HTMX ヘッダーを交換しない SPA 専用の JSON API には使用しないでください。
