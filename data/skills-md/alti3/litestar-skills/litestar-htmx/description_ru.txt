Создавайте управляемые HTMX потоки Litestar с помощью `HTMXPlugin`, `HTMXRequest`, `request.htmx`, `HTMXTemplate` и классов ответов HTMX, таких как `HXLocation`, `PushUrl`, `ReplaceUrl`, `Reswap`, `Retarget`, `TriggerEvent`, `ClientRedirect`, `ClientRefresh` и `HXStopPolling`. Используйте при реализации частичных обновлений, отображаемых на сервере, прогрессивного улучшения, опросов, перенаправлений или взаимодействий HTML с учетом истории. Не используйте API JSON только для SPA, которые не обмениваются заголовками HTML или HTMX.
