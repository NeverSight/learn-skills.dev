أنشئ تدفقات Litestar تعتمد على HTMX باستخدام فئات استجابة `HTMXPlugin` و`HTMXRequest` وrequest.htmx` و`HTMXTemplate` وHTMX مثل `HXLocation` و`PushUrl` و`ReplaceUrl` و`Reswap` و`Retarget` و`TriggerEvent` و`ClientRedirect`، "ClientRefresh"، و"HXStopPolling". يُستخدم عند تنفيذ التحديثات الجزئية المقدمة من الخادم، أو التحسين التدريجي، أو الاستقصاء، أو عمليات إعادة التوجيه، أو تفاعلات HTML المدركة للتاريخ. لا تستخدمه مع واجهات برمجة تطبيقات JSON الخاصة بـ SPA فقط والتي لا تتبادل رؤوس HTML أو HTMX.
