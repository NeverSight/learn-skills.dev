Cree flujos Litestar controlados por HTMX con `HTMXPlugin`, `HTMXRequest`, `request.htmx`, `HTMXTemplate` y clases de respuesta HTMX como `HXLocation`, `PushUrl`, `ReplaceUrl`, `Reswap`, `Retarget`, `TriggerEvent`, `ClientRedirect`, `ClientRefresh` y `HXStopPolling`. Utilícelo al implementar actualizaciones parciales renderizadas por el servidor, mejoras progresivas, sondeos, redireccionamientos o interacciones HTML con reconocimiento del historial. No lo utilice para API JSON solo para SPA que no intercambian encabezados HTML o HTMX.
