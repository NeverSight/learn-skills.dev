قم بتكوين قوالب Litestar باستخدام `TemplateConfig`، ومحركات Jinja/Mako/MiniJinja، واستجابات `Template` للملف أو السلسلة، والسياق المدرك للطلب وCSRF، واستدعاءات القالب، وتكامل المحرك المخصص. يُستخدم عند تنفيذ أو إصلاح HTML المقدم من الخادم في Litestar. لا تستخدم لخدمة الأصول الثابتة أو نقاط نهاية JSON API النقية.
