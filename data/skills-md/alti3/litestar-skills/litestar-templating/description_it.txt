Configura i modelli Litestar con `TemplateConfig`, motori Jinja/Mako/MiniJinja, risposte `Template` di file o stringhe, contesto compatibile con richieste e CSRF, richiamabili di modelli e integrazione di motori personalizzati. Da utilizzare durante l'implementazione o la correzione dell'HTML reso dal server in Litestar. Non utilizzare per la gestione di asset statici o endpoint API JSON puri.
