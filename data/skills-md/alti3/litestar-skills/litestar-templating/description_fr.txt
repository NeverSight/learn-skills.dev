Configurez les modèles Litestar avec `TemplateConfig`, les moteurs Jinja/Mako/MiniJinja, les réponses `Template` de fichier ou de chaîne, le contexte de requête et CSRF, les appelables de modèles et l'intégration de moteur personnalisé. À utiliser lors de l'implémentation ou de la correction du code HTML rendu par le serveur dans Litestar. Ne pas utiliser pour la diffusion d'actifs statiques ou pour les points de terminaison d'API JSON purs.
