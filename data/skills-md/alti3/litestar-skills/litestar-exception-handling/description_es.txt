Implemente el manejo de excepciones Litestar con HTTPException, subclases de excepciones integradas, controladores de excepciones personalizados, anulaciones en capas, asignaciones de códigos de estado y contratos de error de API estables. Úselo al traducir fallas de dominio, errores de validación, fallas de middleware/dependencia o errores HTTP generados por el enrutador en respuestas deterministas de Litestar. No lo utilice para el diseño de políticas de autenticación o autorización que pertenezcan a las capas de seguridad.
