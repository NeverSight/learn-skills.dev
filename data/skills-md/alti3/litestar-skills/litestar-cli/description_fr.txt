Utilisez Litestar CLI pour la découverte automatique d'applications, le ciblage explicite d'applications, les commandes opérationnelles, les workflows de génération de schémas et les points d'extension CLI. À utiliser lors de la normalisation des flux de travail de commandes locales/dev/ops autour de Litestar. Ne pas utiliser pour les décisions de conception d’applications d’exécution qui appartiennent au code de l’application.
