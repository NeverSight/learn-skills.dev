Используйте классы данных Python в качестве транспортных моделей Litestar с явной типизацией, значениями по умолчанию и взаимодействием DTO. Используйте при моделировании контрактов запроса/ответа с помощью упрощенных типизированных объектов. Не используйте, если системы моделей на основе плагинов (Pydantic/msgspec/attrs/ORM) лучше подходят для этой задачи.
