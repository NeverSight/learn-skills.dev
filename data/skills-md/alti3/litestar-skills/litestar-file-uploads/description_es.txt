Maneje las cargas de archivos multiparte de Litestar de forma segura con `UploadFile`, modelos multiparte escritos, entradas de archivos múltiples, límites de cuerpo de solicitud y patrones de procesamiento seguros para el almacenamiento. Úselo cuando acepte archivos cargados únicos o múltiples en cuerpos de solicitud, especialmente puntos finales de datos de formularios de varias partes. No lo utilice para análisis de solicitudes genéricas, entrega de activos estáticos o comportamiento de respuesta de descarga.
