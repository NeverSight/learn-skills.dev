Gestisci i caricamenti di file in più parti Litestar in modo sicuro con "UploadFile", modelli multiparte digitati, input multi-file, limiti del corpo della richiesta e modelli di elaborazione sicuri per l'archiviazione. Da utilizzare quando si accettano file caricati singoli o multipli nei corpi della richiesta, in particolare negli endpoint dati modulo multiparte. Non utilizzare per l'analisi di richieste generiche, la pubblicazione di risorse statiche o il comportamento di risposta al download.
