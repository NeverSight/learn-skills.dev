Utilice esta habilidad cada vez que el usuario quiera analizar datos, crear gráficos o crear visualizaciones de datos. Esto incluye: análisis de ventas, modelos financieros, análisis de cohortes, análisis de embudo, resultados de pruebas A/B, seguimiento de KPI, informes de datos, desgloses de ingresos, análisis de retención de usuarios, análisis de tasa de conversión, resumen CSV y creación de paneles. También se activa cuando: el usuario dice 分析这组数据, 做个图表, 数据可视化, 销售分析, 漏斗分析, 留存分析,做个数据报表. Si es necesario analizar o visualizar datos, utilice esta habilidad.
