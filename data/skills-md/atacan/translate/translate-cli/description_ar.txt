دليل المستخدم النهائي لتشغيل وتكوين واجهة سطر الأوامر "ترجمة" عبر مدخلات النص/stdin/الملف/glob، واختيار الموفر، والإعدادات المسبقة، وقوالب المطالبة المخصصة، وإعدادات TOML. يُستخدم عندما يطلب المستخدمون بناء الأوامر، أو تحديثات التكوين ("تكوين الترجمة"/"إعدادات الترجمة المسبقة")، أو إعداد الموفر، أو التحقق من التشغيل التجريبي، أو استكشاف أخطاء سلوك الترجمة وإصلاحها.
