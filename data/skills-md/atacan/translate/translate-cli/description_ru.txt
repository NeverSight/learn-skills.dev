Руководство для конечного пользователя по запуску и настройке CLI `translate` для входных данных text/stdin/file/glob, выбора поставщика, предустановок, пользовательских шаблонов подсказок и настроек TOML. Используйте, когда пользователи запрашивают конструкцию команды, обновления конфигурации («перевести конфигурацию»/«пресеты перевода»), настройку поставщика, проверку пробного запуска или устранение неполадок в поведении перевода.
