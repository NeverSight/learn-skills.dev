Используйте этот навык при работе с QuestDB — высокопроизводительной базой данных временных рядов. Срабатывает при любом упоминании QuestDB, SQL временных рядов с SAMPLE BY, LATEST ON, ASOF JOIN, приеме ILP или клиентских библиотеках questdb Python/Go/Java/Rust/.NET. Также срабатывает при написании запросов Grafana к QuestDB, создании материализованных представлений для сводных данных временных рядов, работе со книгой заказов или данными финансового рынка в QuestDB, а также при любом SQL, который включает назначенные временные метки или таблицы, разделенные по времени. QuestDB расширяет SQL уникальными ключевыми словами временных рядов — стандартные шаблоны PostgreSQL или MySQL не работают. Всегда читайте этот навык перед написанием QuestDB SQL, чтобы избежать галлюцинаций с неправильным синтаксисом.
