高性能時系列データベースである QuestDB を使用する場合は常に、このスキルを使用してください。 QuestDB、SAMPLE BY、LATEST ON、ASOF JOIN を使用した時系列 SQL、ILP 取り込み、または QuestDB Python/Go/Java/Rust/.NET クライアント ライブラリに関する言及をトリガーします。また、QuestDB に対する Grafana クエリを作成するとき、時系列ロールアップのマテリアライズド ビューを作成するとき、QuestDB 内のオーダーブックや金融市場データを操作するとき、または指定されたタイムスタンプや時間分割テーブルを含む SQL を操作するときにもトリガーされます。 QuestDB は、独自の時系列キーワードを使用して SQL を拡張します。標準の PostgreSQL または MySQL パターンは失敗します。間違った構文の幻覚を避けるために、QuestDB SQL を作成する前に必ずこのスキルをお読みください。
