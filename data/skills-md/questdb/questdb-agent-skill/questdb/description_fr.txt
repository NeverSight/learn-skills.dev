Utilisez cette compétence lorsque vous travaillez avec QuestDB, une base de données de séries chronologiques hautes performances. Déclenchez toute mention de QuestDB, de séries temporelles SQL avec SAMPLE BY, LATEST ON, ASOF JOIN, l'ingestion ILP ou les bibliothèques clientes questdb Python/Go/Java/Rust/.NET. Déclenchez également lors de l'écriture de requêtes Grafana sur QuestDB, de la création de vues matérialisées pour les cumuls de séries chronologiques, de l'utilisation du carnet d'ordres ou des données du marché financier dans QuestDB, ou de tout SQL impliquant des horodatages désignés ou des tables partitionnées dans le temps. QuestDB étend SQL avec des mots-clés de séries chronologiques uniques : les modèles PostgreSQL ou MySQL standard échoueront. Lisez toujours cette compétence avant d'écrire QuestDB SQL pour éviter d'halluciner une syntaxe incorrecte.
