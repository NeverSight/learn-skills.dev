Utilice esta habilidad siempre que trabaje con QuestDB, una base de datos de series temporales de alto rendimiento. Se activa ante cualquier mención de QuestDB, SQL de serie temporal con SAMPLE BY, LATEST ON, ASOF JOIN, ingestión de ILP o las bibliotecas cliente questdb Python/Go/Java/Rust/.NET. También se activa al escribir consultas de Grafana en QuestDB, crear vistas materializadas para resúmenes de series temporales, trabajar con libros de pedidos o datos del mercado financiero en QuestDB, o cualquier SQL que involucre marcas de tiempo designadas o tablas particionadas en el tiempo. QuestDB amplía SQL con palabras clave únicas de series temporales: los patrones estándar de PostgreSQL o MySQL fallarán. Lea siempre esta habilidad antes de escribir QuestDB SQL para evitar alucinaciones con una sintaxis incorrecta.
