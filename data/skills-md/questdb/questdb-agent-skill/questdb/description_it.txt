Utilizza questa competenza ogni volta che lavori con QuestDB, un database di serie temporali ad alte prestazioni. Attivazione su qualsiasi menzione di QuestDB, SQL di serie temporali con SAMPLE BY, LATEST ON, ASOF JOIN, acquisizione ILP o le librerie client questdb Python/Go/Java/Rust/.NET. Si attiva anche quando si scrivono query Grafana su QuestDB, si creano viste materializzate per i rollup di serie temporali, si lavora con il registro degli ordini o i dati del mercato finanziario in QuestDB o qualsiasi SQL che coinvolge timestamp designati o tabelle partizionate nel tempo. QuestDB estende SQL con parole chiave di serie temporali univoche: i modelli PostgreSQL o MySQL standard falliranno. Leggi sempre questa competenza prima di scrivere QuestDB SQL per evitare allucinazioni di sintassi errata.
