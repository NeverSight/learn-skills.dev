استخدم هذه المهارة عند العمل مع QuestDB — وهي قاعدة بيانات سلاسل زمنية عالية الأداء. قم بالتشغيل عند أي ذكر لـ QuestDB، أو SQL للسلاسل الزمنية مع SAMPLE BY، أو LATEST ON، أو ASOF JOIN، أو استيعاب ILP، أو مكتبات عملاء Questdb Python/Go/Java/Rust/.NET. يتم التشغيل أيضًا عند كتابة استعلامات Grafana مقابل QuestDB، أو إنشاء طرق عرض مادية لمجموعات السلاسل الزمنية، أو العمل مع دفتر الطلبات أو بيانات السوق المالية في QuestDB، أو أي SQL يتضمن طوابع زمنية معينة أو جداول مقسمة زمنيًا. يقوم QuestDB بتوسيع SQL بكلمات رئيسية فريدة من نوعها للسلاسل الزمنية - ستفشل أنماط PostgreSQL أو MySQL القياسية. اقرأ دائمًا هذه المهارة قبل كتابة QuestDB SQL لتجنب الهلوسة ببناء جملة غير صحيح.
