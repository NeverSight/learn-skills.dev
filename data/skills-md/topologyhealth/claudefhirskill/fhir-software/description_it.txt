Assistente allo sviluppo software FHIR (Fast Healthcare Interoperability Resources) completo. Da utilizzare quando si lavora con API FHIR, implementazioni o scambio di dati sanitari. Supporta FHIR R4, R4B, R5, guide di implementazione (IG), creazione di FHIR Shorthand (FSH), SUSHI, GoFSH, convalida, terminologia e SMART su FHIR. Ideale per creare server, client, validatori FHIR, autori IG o applicazioni sanitarie che necessitano di elaborare risorse FHIR.
