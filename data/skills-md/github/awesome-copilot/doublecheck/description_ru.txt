Трехуровневый конвейер проверки выходных данных ИИ. Извлекает поддающиеся проверке утверждения, находит подтверждающие или противоречащие источники с помощью веб-поиска, проводит состязательную проверку моделей галлюцинаций и создает структурированный отчет о проверке со ссылками на источники для проверки людьми.
