Crée des cas de test d'intégration pour les artefacts d'accès aux données .NET lors des migrations de bases de données Oracle vers PostgreSQL. Génère des tests xUnit indépendants de la base de données avec des données de départ déterministes qui valident la cohérence du comportement sur les deux systèmes de base de données. À utiliser lors de la création de tests d'intégration pour un projet migré, de la génération d'une couverture de test pour les couches d'accès aux données ou de l'écriture de tests de validation de migration Oracle vers PostgreSQL.
