Crea casos de prueba de integración para artefactos de acceso a datos .NET durante las migraciones de bases de datos de Oracle a PostgreSQL. Genera pruebas xUnit independientes de la base de datos con datos iniciales deterministas que validan la coherencia del comportamiento en ambos sistemas de bases de datos. Úselo al crear pruebas de integración para un proyecto migrado, generar cobertura de prueba para capas de acceso a datos o escribir pruebas de validación de migración de Oracle a PostgreSQL.
