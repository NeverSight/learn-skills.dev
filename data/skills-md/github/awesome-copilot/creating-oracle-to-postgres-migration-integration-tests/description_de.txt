Erstellt Integrationstestfälle für .NET-Datenzugriffsartefakte während Oracle-zu-PostgreSQL-Datenbankmigrationen. Erzeugt DB-agnostische xUnit-Tests mit deterministischen Startdaten, die die Verhaltenskonsistenz in beiden Datenbanksystemen validieren. Verwenden Sie diese Option, wenn Sie Integrationstests für ein migriertes Projekt erstellen, Testabdeckung für Datenzugriffsebenen generieren oder Validierungstests für die Oracle-zu-PostgreSQL-Migration schreiben.
