إنشاء حالات اختبار التكامل لعناصر الوصول إلى بيانات .NET أثناء عمليات ترحيل قاعدة بيانات Oracle إلى PostgreSQL. يُنشئ اختبارات xUnit غير المحددة لقاعدة البيانات باستخدام بيانات أولية حتمية تتحقق من صحة اتساق السلوك عبر كلا نظامي قاعدة البيانات. يُستخدم عند إنشاء اختبارات تكامل لمشروع تم ترحيله، أو إنشاء تغطية اختبار لطبقات الوصول إلى البيانات، أو كتابة اختبارات التحقق من صحة ترحيل Oracle-to-PostgreSQL.
