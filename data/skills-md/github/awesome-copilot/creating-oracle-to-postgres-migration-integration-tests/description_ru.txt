Создает тестовые сценарии интеграции для артефактов доступа к данным .NET во время миграции базы данных Oracle в PostgreSQL. Генерирует независимые от БД тесты xUnit с детерминированными исходными данными, которые проверяют согласованность поведения в обеих системах баз данных. Используйте при создании интеграционных тестов для перенесенного проекта, создании тестового покрытия для уровней доступа к данным или написании проверочных тестов миграции с Oracle на PostgreSQL.
