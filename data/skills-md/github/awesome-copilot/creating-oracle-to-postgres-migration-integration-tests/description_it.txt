Crea casi di test di integrazione per gli artefatti di accesso ai dati .NET durante le migrazioni di database da Oracle a PostgreSQL. Genera test xUnit indipendenti dal DB con dati seed deterministici che convalidano la coerenza del comportamento tra entrambi i sistemi di database. Da utilizzare durante la creazione di test di integrazione per un progetto migrato, la generazione di copertura di test per i livelli di accesso ai dati o la scrittura di test di convalida della migrazione da Oracle a PostgreSQL.
