Erkennt alle Projekte in einer .NET-Lösung, klassifiziert jedes für die Migrationsberechtigung von Oracle nach PostgreSQL und erstellt einen dauerhaften Master-Migrationsplan. Verwenden Sie diese Option, wenn Sie eine Oracle-zu-PostgreSQL-Migration mit mehreren Projekten starten, ein Migrationsinventar erstellen oder bewerten, welche .NET-Projekte Oracle-Abhängigkeiten enthalten.
