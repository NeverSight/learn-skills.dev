Обнаруживает все проекты в решении .NET, классифицирует каждый из них на соответствие критериям миграции с Oracle на PostgreSQL и создает постоянный генеральный план миграции. Используйте при запуске многопроектной миграции Oracle на PostgreSQL, создании инвентаризации миграции или оценке того, какие проекты .NET содержат зависимости Oracle.
