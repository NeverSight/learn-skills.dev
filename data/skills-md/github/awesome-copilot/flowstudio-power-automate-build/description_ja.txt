FlowStudio MCP サーバーを使用して、Power Automate クラウド フローを構築、足場、展開します。フローの作成、新しいフローの構築、フロー定義のデプロイ、Power Automate ワークフローのスキャフォールディング、フロー JSON の構築、既存のフローのアクションの更新、フロー定義へのパッチ適用、フローへのアクションの追加、接続の接続、またはワークフロー定義の最初からの生成を行うように求められたら、このスキルを読み込みます。 FlowStudio MCP サブスクリプションが必要です — https://mcp.flowstudio.app を参照してください
