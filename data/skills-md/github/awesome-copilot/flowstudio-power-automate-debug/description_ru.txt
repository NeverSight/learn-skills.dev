Отладка неудачных облачных потоков Power Automate с помощью сервера FlowStudio MCP. Загрузите этот навык, когда его попросят: отладить поток, исследовать неудачный запуск, почему происходит сбой этого потока, проверить выходные данные действий, найти основную причину ошибки потока, исправить нарушенный поток Power Automate, диагностировать тайм-аут, отследить DynamicOperationRequestFailure, проверить ошибки аутентификации соединителя, прочитать сведения об ошибках во время выполнения или устранить сбои выражений. Требуется подписка FlowStudio MCP — см. https://mcp.flowstudio.app.
