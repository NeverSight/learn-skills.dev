Eseguire il debug dei flussi cloud Power Automate non riusciti utilizzando il server MCP FlowStudio. Carica questa competenza quando viene richiesto di: eseguire il debug di un flusso, esaminare un'esecuzione non riuscita, perché questo flusso non funziona, ispezionare gli output dell'azione, trovare la causa principale di un errore di flusso, correggere un flusso Power Automate interrotto, diagnosticare un timeout, tracciare un DynamicOperationRequestFailure, controllare gli errori di autenticazione del connettore, leggere i dettagli dell'errore da un'esecuzione o risolvere gli errori di espressione. Richiede un abbonamento MCP FlowStudio: vedere https://mcp.flowstudio.app
