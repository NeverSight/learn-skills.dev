Crée un plan de test d'intégration pour les artefacts d'accès aux données .NET lors des migrations de bases de données Oracle vers PostgreSQL. Analyse un seul projet pour identifier les référentiels, les DAO et les couches de service qui interagissent avec la base de données, puis produit un plan de test structuré. À utiliser lors de la planification de la couverture des tests d'intégration pour un projet migré, de l'identification des méthodes d'accès aux données nécessitant des tests ou de la préparation de la validation de la migration d'Oracle vers PostgreSQL.
