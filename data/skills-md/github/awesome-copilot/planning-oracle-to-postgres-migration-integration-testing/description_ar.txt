إنشاء خطة اختبار تكامل لعناصر الوصول إلى بيانات .NET أثناء عمليات ترحيل قاعدة بيانات Oracle إلى PostgreSQL. يقوم بتحليل مشروع واحد لتحديد المستودعات والمنظمات اللامركزية المستقلة وطبقات الخدمة التي تتفاعل مع قاعدة البيانات، ثم ينتج خطة اختبار منظمة. يُستخدم عند التخطيط لتغطية اختبار التكامل لمشروع تم ترحيله، أو تحديد طرق الوصول إلى البيانات التي تحتاج إلى اختبارات، أو التحضير للتحقق من صحة ترحيل Oracle-to-PostgreSQL.
