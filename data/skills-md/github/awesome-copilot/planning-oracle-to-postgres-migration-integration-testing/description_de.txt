Erstellt einen Integrationstestplan für .NET-Datenzugriffsartefakte während Oracle-zu-PostgreSQL-Datenbankmigrationen. Analysiert ein einzelnes Projekt, um Repositorys, DAOs und Serviceschichten zu identifizieren, die mit der Datenbank interagieren, und erstellt dann einen strukturierten Testplan. Verwenden Sie diese Option, wenn Sie die Integrationstestabdeckung für ein migriertes Projekt planen, ermitteln, welche Datenzugriffsmethoden getestet werden müssen, oder sich auf die Validierung der Oracle-zu-PostgreSQL-Migration vorbereiten.
