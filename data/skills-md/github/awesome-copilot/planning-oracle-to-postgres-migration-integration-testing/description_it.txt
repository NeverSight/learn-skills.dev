Crea un piano di test di integrazione per gli artefatti di accesso ai dati .NET durante le migrazioni del database da Oracle a PostgreSQL. Analizza un singolo progetto per identificare repository, DAO e livelli di servizio che interagiscono con il database, quindi produce un piano di test strutturato. Da utilizzare quando si pianifica la copertura dei test di integrazione per un progetto migrato, si identifica quali metodi di accesso ai dati necessitano di test o si prepara la convalida della migrazione da Oracle a PostgreSQL.
