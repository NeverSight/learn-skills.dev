Identifica los riesgos de migración de Oracle a PostgreSQL mediante una referencia cruzada del código con diferencias de comportamiento conocidas (cadenas vacías, refcursores, coerción de tipos, clasificación, marcas de tiempo, transacciones concurrentes, etc.).
