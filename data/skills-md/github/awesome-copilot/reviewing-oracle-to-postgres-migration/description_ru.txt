Идентифицирует риски миграции с Oracle на PostgreSQL путем перекрестных ссылок кода на известные различия в поведении (пустые строки, рекурсоры, приведение типов, сортировка, временные метки, параллельные транзакции и т. д.). Используйте при планировании миграции базы данных, проверке артефактов миграции или проверке того, что интеграционные тесты охватывают различия между Oracle и PostgreSQL.
