Migrieren Sie Feldwerte aus GitHub Project V2-Feldern oder Repo-Labels in Problemfelder auf Organisationsebene. Verwenden Sie diese Fähigkeit, wenn Benutzer Metadaten (Priorität, Status, Daten, Text, Zahlen) in großen Mengen aus Projektfeldern oder -bezeichnungen in Problemfelder kopieren müssen oder wenn sie nach der Migration, Übertragung oder dem Kopieren von Projektfelddaten oder -bezeichnungen in Problemfelder fragen.
