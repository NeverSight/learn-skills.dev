Migre valores de campo desde campos de GitHub Project V2 o etiquetas de repositorio a campos de problemas a nivel de organización. Utilice esta habilidad cuando los usuarios necesiten copiar metadatos de forma masiva (prioridad, estado, fechas, texto, números) desde campos o etiquetas de proyecto a campos de emisión, o cuando pregunten sobre la migración, transferencia o copia de datos o etiquetas de campos de proyecto a campos de emisión.
