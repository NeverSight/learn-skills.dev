قم بترحيل قيم الحقول من حقول GitHub Project V2 أو تسميات الريبو إلى حقول المشكلات على مستوى المؤسسة. استخدم هذه المهارة عندما يحتاج المستخدمون إلى نسخ بيانات تعريف مجمعة (الأولوية، الحالة، التواريخ، النص، الأرقام) من حقول المشروع أو التسميات إلى حقول الإصدار، أو عندما يسألون عن ترحيل أو نقل أو نسخ بيانات حقل المشروع أو التسميات إلى حقول الإصدار.
