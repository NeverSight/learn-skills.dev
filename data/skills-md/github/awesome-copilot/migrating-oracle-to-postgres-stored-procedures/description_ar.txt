ترحيل إجراءات Oracle PL/SQL المخزنة إلى PostgreSQL PL/pgSQL. يترجم بناء الجملة الخاص بـ Oracle، ويحافظ على توقيعات الطريقة والمعلمات المرتبطة بالنوع، ويعزز orafce حيثما كان ذلك مناسبًا، ويطبق COLLATE "C" لفرز النص المتوافق مع Oracle. يُستخدم عند تحويل إجراءات أو وظائف Oracle المخزنة إلى مكافئات PostgreSQL أثناء ترحيل قاعدة البيانات.
