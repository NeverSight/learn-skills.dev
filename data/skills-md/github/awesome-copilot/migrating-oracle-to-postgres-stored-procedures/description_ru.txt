Переносит хранимые процедуры Oracle PL/SQL в PostgreSQL PL/pgSQL. Преобразует специфичный для Oracle синтаксис, сохраняет сигнатуры методов и параметры, привязанные к типу, при необходимости использует orafce и применяет COLLATE "C" для сортировки текста, совместимой с Oracle. Используйте при преобразовании хранимых процедур или функций Oracle в эквиваленты PostgreSQL во время миграции базы данных.
