Migriert gespeicherte Oracle PL/SQL-Prozeduren nach PostgreSQL PL/pgSQL. Übersetzt Oracle-spezifische Syntax, behält Methodensignaturen und typverankerte Parameter bei, nutzt gegebenenfalls Orafce und wendet COLLATE „C“ für Oracle-kompatible Textsortierung an. Verwenden Sie diese Option, wenn Sie während einer Datenbankmigration gespeicherte Oracle-Prozeduren oder -Funktionen in PostgreSQL-Äquivalente konvertieren.
