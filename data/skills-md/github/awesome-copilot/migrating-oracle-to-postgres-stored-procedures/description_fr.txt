Migre les procédures stockées Oracle PL/SQL vers PostgreSQL PL/pgSQL. Traduit la syntaxe spécifique à Oracle, préserve les signatures de méthode et les paramètres ancrés par type, exploite orafce le cas échéant et applique COLLATE "C" pour le tri de texte compatible Oracle. À utiliser lors de la conversion de procédures ou de fonctions stockées Oracle en équivalents PostgreSQL lors d'une migration de base de données.
