Migra procedimientos almacenados de Oracle PL/SQL a PostgreSQL PL/pgSQL. Traduce la sintaxis específica de Oracle, conserva firmas de métodos y parámetros de tipo anclado, aprovecha orafce cuando corresponde y aplica COLLATE "C" para la clasificación de texto compatible con Oracle. Úselo al convertir funciones o procedimientos almacenados de Oracle a equivalentes de PostgreSQL durante una migración de base de datos.
