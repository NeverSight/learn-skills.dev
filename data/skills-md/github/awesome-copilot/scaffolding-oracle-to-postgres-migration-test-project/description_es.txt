Desarrolla un proyecto de prueba de integración de xUnit para validar el comportamiento de migración de bases de datos de Oracle a PostgreSQL en soluciones .NET. Crea el proyecto de prueba, la clase base de reversión de transacciones y el administrador de datos iniciales. Úselo al configurar la infraestructura de prueba antes de escribir pruebas de integración de migración o cuando se necesita un proyecto de prueba para la validación de Oracle a PostgreSQL.
