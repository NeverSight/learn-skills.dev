يدعم مشروع اختبار تكامل xUnit للتحقق من صحة سلوك ترحيل قاعدة بيانات Oracle إلى PostgreSQL في حلول .NET. إنشاء مشروع اختبار، والفئة الأساسية لاستعادة المعاملات، ومدير البيانات الأولية. يُستخدم عند إعداد البنية الأساسية للاختبار قبل كتابة اختبارات تكامل الترحيل، أو عند الحاجة إلى مشروع اختبار للتحقق من صحة Oracle-to-PostgreSQL.
