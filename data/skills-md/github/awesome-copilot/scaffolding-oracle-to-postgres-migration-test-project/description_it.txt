Supporta un progetto di test di integrazione xUnit per la convalida del comportamento di migrazione del database da Oracle a PostgreSQL nelle soluzioni .NET. Crea il progetto di test, la classe base di rollback delle transazioni e il gestore dei dati seed. Da utilizzare quando si configura l'infrastruttura di test prima di scrivere test di integrazione della migrazione o quando è necessario un progetto di test per la convalida da Oracle a PostgreSQL.
