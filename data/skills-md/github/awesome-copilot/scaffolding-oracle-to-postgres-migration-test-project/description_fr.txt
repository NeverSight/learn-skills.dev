Élabore un projet de test d'intégration xUnit pour valider le comportement de migration de base de données Oracle vers PostgreSQL dans les solutions .NET. Crée le projet de test, la classe de base d'annulation de transaction et le gestionnaire de données de départ. À utiliser lors de la configuration de l'infrastructure de test avant d'écrire des tests d'intégration de migration, ou lorsqu'un projet de test est nécessaire pour la validation Oracle vers PostgreSQL.
