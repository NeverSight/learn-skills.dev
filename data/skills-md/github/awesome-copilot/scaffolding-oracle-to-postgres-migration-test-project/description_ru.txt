Формирует проект тестирования интеграции xUnit для проверки поведения миграции базы данных Oracle в PostgreSQL в решениях .NET. Создает тестовый проект, базовый класс отката транзакций и диспетчер начальных данных. Используйте при настройке тестовой инфраструктуры перед написанием интеграционных тестов миграции или когда тестовый проект необходим для проверки Oracle на PostgreSQL.
