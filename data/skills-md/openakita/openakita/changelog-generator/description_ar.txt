يقوم تلقائيًا بإنشاء سجلات التغيير التي تواجه المستخدم من التزامات git من خلال تحليل سجل الالتزام وتصنيف التغييرات وتحويل الالتزامات الفنية إلى ملاحظات إصدار واضحة وسهلة الاستخدام. استخدم هذه المهارة عندما يذكر المستخدم سجلات التغيير، أو ملاحظات الإصدار، أو تحديثات الإصدار، أو "ما تغير"، أو تحديثات المنتج، أو أوصاف متجر التطبيقات، أو عندما يحتاج إلى تلخيص أعمال التطوير الأخيرة للجماهير غير التقنية.
