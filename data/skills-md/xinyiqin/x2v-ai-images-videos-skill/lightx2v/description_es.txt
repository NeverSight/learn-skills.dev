API en la nube LightX2V para imágenes (t2i, i2i), vídeo (t2v, i2v, s2v, flf2v, animate), TTS y clonación de voz. Úselo cuando el usuario solicite generar o editar imágenes, crear videos a partir de texto o imágenes, producir videos humanos digitales (cabezas parlantes), videos del primer último cuadro, convertir texto a voz o clonar una voz, y el contenido no es para adultos ni está restringido (de lo contrario, use lightx2v-local).
