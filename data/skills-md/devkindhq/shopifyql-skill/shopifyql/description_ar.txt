كتابة وتصحيح وشرح استعلامات ShopifyQL وتعبيرات لغة الاستعلام الخاصة بـ Shopify. استخدم هذه المهارة عندما يريد المستخدم الاستعلام عن بيانات تحليلات Shopify، أو بناء شرائح العملاء، أو كتابة ShopifyQL للتقارير، أو استكشاف بيانات المبيعات/الطلبات/المنتجات عبر Shopify Admin API، أو تصحيح خطأ ShopifyQL، أو فهم الجداول/الأبعاد/المقاييس المتاحة، أو ترجمة سؤال عمل إلى استعلام Shopify. مشغلات أيضًا لـ: "ShopifyQL"، أو "استعلام تحليلات Shopify"، أو "مرشح شريحة العملاء"، أو "شريحة Shopify"، أو "العرض من المبيعات"، أو "GROUP BY في Shopify"، أو "الاستعلام عن تقرير Shopify"، أو أي ذكر لجداول ShopifyQL مثل "المبيعات"، أو "الجلسات"، أو "الطلبات".
