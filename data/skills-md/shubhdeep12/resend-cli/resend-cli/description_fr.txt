À utiliser lorsque l'utilisateur souhaite exécuter des opérations de renvoi à partir de la ligne de commande (envoyer un e-mail, lister les e-mails, gérer les domaines, les contacts, les modèles, les webhooks) via la CLI de renvoi. Préférez cela lorsque la tâche est basée sur CLI/terminal ou lorsque vous avez besoin d’une sortie JSON scriptable.
