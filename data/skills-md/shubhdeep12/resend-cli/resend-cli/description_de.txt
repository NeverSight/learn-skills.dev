Verwenden Sie diese Option, wenn der Benutzer Resend-Vorgänge über die Befehlszeile (E-Mail senden, E-Mails auflisten, Domänen, Kontakte, Vorlagen, Webhooks verwalten) über die Resend-CLI ausführen möchte. Bevorzugen Sie dies, wenn die Aufgabe CLI-/terminalbasiert ist oder wenn Sie eine skriptfähige JSON-Ausgabe benötigen.
