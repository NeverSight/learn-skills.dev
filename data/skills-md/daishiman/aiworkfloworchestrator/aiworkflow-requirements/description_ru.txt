Навыки поиска, ссылки и обновления основной спецификации AIWorkflowOrchestrator из `references/`.  
Он поэтапно считывает только минимально необходимые документы, начиная с карты ресурсов/быстрой справки/карты тем/ключевых слов.  
Использование: проверка требований, проверка контракта на проектирование/API/IPC, управление пользовательским интерфейсом/состоянием/решения по безопасности, рабочий процесс задач, извлеченные уроки, синхронизация открытых задач.  
Особенно безопасно обрабатывает тайм-аут вызова, обход настроек, жизненный цикл навыков, глобальную навигацию и перенаправление Центра навыков / Рабочего пространства / Агента / Создателя навыков.

Якоря:  
• Разработка на основе спецификаций / применение: синхронизация основных спецификаций / цель: сохранение соответствия спецификации реализации.  
• Прогрессивное раскрытие/применить: начать чтение карты ресурсов/цель: предотвратить пропуски с минимальным количеством ссылок.

Триггер:  
проверка спецификации, обновление спецификации, синхронизация рабочих процессов задач, синхронизация извлеченных уроков, отражение спецификации пользовательского интерфейса, проверка контракта API/IPC, проверка требований безопасности, SafeInvoke, таймаут, обход настроек, жизненный цикл навыков, Центр навыков, Рабочая область, Агент, Создатель навыков, navContract, GlobalNavStrip, MobileNavBar, SkillManagementPanel, реформа линейного бюджета, разделение спецификаций, разделение семейства, сегментирование созданного индекса
