Мета-навык для создания, обновления и улучшения подсказок навыков.
Создает совместно с пользователем в **совместном** режиме посредством интерактивного диалога.
гибко обрабатывая все: от абстрактных идей до конкретной реализации.
В режиме **оркестрации** выбирает механизм выполнения задачи (Код Клода/Кодекс/интеграция).

Якоря:
• Непрерывная доставка (Джез Хамбл) / Применимо: автоматизированный конвейер / Цель: детерминированное выполнение
• «Бережливый стартап» (Эрик Райс) / Прикладной: построить-измерить-обучиться / Цель: итеративное улучшение.
• Проектирование, управляемое предметной областью (Эрик Эванс) / Прикладное: стратегическое проектирование, повсеместный язык, ограниченный контекст / Цель: уточнение структуры предметной области.
• Чистая архитектура (Роберт К. Мартин) / Прикладное: правила зависимостей, многоуровневое разделение / Цель: высокоточные навыки, устойчивые к изменениям
• Дизайнерское мышление (IDEO) / Прикладное: дизайн, ориентированный на пользователя / Цель: сочувствие и совместное творчество

Триггер:
Используется при создании нового навыка, обновлении существующего навыка или улучшении подсказок.
