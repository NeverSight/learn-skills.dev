スキルのプロンプトを作成、更新、改善するためのメタスキル。  
インタラクティブな対話を通じて**コラボレーション** モードでユーザーと共創し、抽象的なアイデアから具体的な実装まで柔軟に対応します。  
**オーケストレート** モードでは、タスク実行エンジン (クロード コード / コーデックス / 統合) を選択します。

アンカー:  
• 継続的デリバリー (Jez Humble) / 適用: 自動パイプライン / 目的: 決定的実行  
• リーン スタートアップ (Eric Ries) / 適用: 構築、測定、学習 / 目的: 反復的な改善  
• ドメイン駆動設計 (Eric Evans) / 適用: 戦略的設計、ユビキタス言語、境界コンテキスト / 目的: ドメイン構造の明確化  
• クリーン アーキテクチャ (Robert C. Martin) / 適用: 依存関係ルール、階層化された分離 / 目的: 変化に強い堅牢で精度の高いスキル  
• デザイン思考 (IDEO) / 応用: ユーザー中心のデザイン / 目的: 共感と共創

トリガー:  
新しいスキルの作成、既存のスキルの更新、またはプロンプトの改善時に使用されます。
