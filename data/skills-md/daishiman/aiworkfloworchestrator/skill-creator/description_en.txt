A meta-skill for creating, updating, and improving prompts.  
Co-creates interactively with the user in **collaborative** mode from abstract ideas to concrete implementation, flexibly adapting to any stage.  
In **orchestrate** mode, selects the task execution engine (Claude Code / Codex / integrations).

Anchors:  
• Continuous Delivery (Jez Humble) / Applied: automated pipeline / Purpose: deterministic execution  
• The Lean Startup (Eric Ries) / Applied: Build-Measure-Learn / Purpose: iterative improvement  
• Domain-Driven Design (Eric Evans) / Applied: strategic design, ubiquitous language, bounded context / Purpose: clarity of domain structure  
• Clean Architecture (Robert C. Martin) / Applied: dependency rules, layered separation / Purpose: robust, high-accuracy skills resilient to change  
• Design Thinking (IDEO) / Applied: user-centered design / Purpose: empathy and co-creation

Trigger:  
Use when creating a new skill, updating an existing skill, or improving prompts.
