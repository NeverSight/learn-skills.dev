مهارة وصفية لإنشاء المطالبات الخاصة بالمهارات وتحديثها وتحسينها.  
شارك في الإنشاء مع المستخدم في الوضع **التعاوني**، والتعامل بمرونة مع كل شيء بدءًا من الأفكار المجردة وحتى التنفيذ الملموس.  
في وضع **التنسيق**، يحدد محرك تنفيذ المهمة (Claude Code / Codex / التكامل).

المراسي:  
• التسليم المستمر (Jez Humble) / التطبيقي: خط الأنابيب الآلي / الغرض: التنفيذ الحتمي  
• The Lean Startup (إريك ريس) / التطبيقي: البناء والقياس والتعلم / الغرض: التحسين التكراري  
• التصميم المبني على المجال (إيريك إيفانز) / التطبيقي: التصميم الاستراتيجي، اللغة واسعة الانتشار، السياق المحدود / الغرض: توضيح بنية المجال  
• الهندسة المعمارية النظيفة (روبرت سي. مارتن) / التطبيقية: قواعد التبعية، والفصل الطبقي / الغرض: مهارات قوية وعالية الدقة مقاومة للتغيير  
• التفكير التصميمي (IDEO) / التطبيقي: التصميم الذي يركز على المستخدم / الغرض: التعاطف والإبداع المشترك

الزناد:  
يُستخدم عند إنشاء مهارات جديدة، أو تحديث المهارات الحالية، أو تحسين المطالبات.
