Una metahabilidad para crear, actualizar y mejorar indicaciones.  
En el modo **colaborativo**, cocrea con el usuario mientras interactúa, manejando de manera flexible todo, desde ideas abstractas hasta implementaciones concretas.  
En modo **orquestar** selecciona el motor de ejecución de tareas (Claude Code / Codex / integración).

Anclas:  
• Entrega continua (Jez Humble) / Aplicado: canalización automatizada / Propósito: ejecución determinista  
• El Lean Startup (Eric Ries) / Aplicado: Construir-Medir-Aprender / Propósito: mejora iterativa  
• Diseño impulsado por el dominio (Eric Evans) / Aplicado: diseño estratégico, lenguaje ubicuo, contexto limitado / Propósito: aclarar la estructura del dominio  
• Arquitectura limpia (Robert C. Martin) / Aplicado: reglas de dependencia, diseño de separación en capas / Propósito: habilidades de alta precisión resistentes al cambio  
• Design Thinking (IDEO) / Aplicado: diseño centrado en el usuario / Propósito: empatía y co-creación

Desencadenante:  
Se utiliza al crear una nueva habilidad, actualizar una habilidad existente o mejorar una indicación.
