Используйте при разработке полных систем GraphRAG, которые сочетают извлечение графов с рассуждениями LLM. Вызывается, когда пользователь упоминает систему GraphRAG, стек технологий, Neo4j с LLM, граф LangChain, граф знаний LlamaIndex, обнаружение сообщества для RAG, гибридную интеграцию векторов символов, производственный GraphRAG или граф RAG для конкретной предметной области. Обеспечивает проектирование архитектуры, выбор технологий и рекомендации по настройке домена.
