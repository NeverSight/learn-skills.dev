Úselo al diseñar sistemas GraphRAG completos que combinen la recuperación de gráficos con el razonamiento LLM. Invocar cuando el usuario mencione el sistema GraphRAG, la pila de tecnología, Neo4j con LLM, el gráfico LangChain, el gráfico de conocimiento LlamaIndex, la detección de comunidad para RAG, la integración híbrida de símbolo-vector, el GraphRAG de producción o el gráfico RAG de dominio específico. Proporciona diseño de arquitectura, selección de tecnología y orientación para la personalización de dominios.
