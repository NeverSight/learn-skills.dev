يُستخدم عند تصميم أنظمة GraphRAG الكاملة التي تجمع بين استرجاع الرسم البياني واستدلال LLM. يتم الاستدعاء عندما يذكر المستخدم نظام GraphRAG، أو مكدس التكنولوجيا، أو Neo4j مع LLM، أو الرسم البياني LangChain، أو الرسم البياني للمعرفة LlamaIndex، أو اكتشاف المجتمع لـ RAG، أو تكامل ناقل الرمز المختلط، أو GraphRAG للإنتاج، أو الرسم البياني الخاص بالمجال RAG. يوفر التصميم المعماري واختيار التكنولوجيا وإرشادات تخصيص المجال.
