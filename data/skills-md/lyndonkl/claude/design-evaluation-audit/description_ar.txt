يُستخدم عند التقييم المنهجي للتصميمات الحالية للمواءمة المعرفية، أو إجراء مراجعات أو انتقادات للتصميم، أو تشخيص مشكلات قابلية الاستخدام، أو ضمان الجودة قبل الإطلاق، أو الاختيار بين بدائل التصميم ذات المعايير الموضوعية. يتم الاستدعاء عندما يذكر المستخدم مراجعة التصميم، أو نقد التصميم، أو تقييم التصميم، أو تصور التدقيق، أو قائمة المراجعة، أو التقييم المعرفي، أو تقييم سهولة الاستخدام.
