Создавайте и обновляйте привязки взаимодействия тонкой/родной платформы для Android в проектах .NET MAUI и .NET для Android. Помогает создавать оболочки Java/Kotlin, настраивать проекты Gradle, разрешать зависимости Maven, создавать привязки C# и интегрировать собственные библиотеки Android с использованием подхода Native Library Interop (NLI). Используйте, когда вас спрашивают о привязках Android, интеграции AAR/JAR, взаимодействии Kotlin, зависимостях Maven или соединении собственных SDK Android с .NET.
