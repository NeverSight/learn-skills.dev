Créez et mettez à jour des liaisons d'interopérabilité de plate-forme slim/native pour Android dans les projets .NET MAUI et .NET pour Android. Guide dans la création de wrappers Java/Kotlin, la configuration de projets Gradle, la résolution des dépendances Maven, la génération de liaisons C# et l'intégration de bibliothèques Android natives à l'aide de l'approche Native Library Interop (NLI). À utiliser lorsque vous êtes interrogé sur les liaisons Android, l'intégration AAR/JAR, l'interopérabilité Kotlin, les dépendances Maven ou le pont entre les SDK Android natifs et .NET.
