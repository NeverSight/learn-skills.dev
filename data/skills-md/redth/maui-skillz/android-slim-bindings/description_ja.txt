.NET MAUI および .NET for Android プロジェクトで Android 用のスリム/ネイティブ プラットフォーム相互運用バインディングを作成および更新します。 Java/Kotlin ラッパーの作成、Gradle プロジェクトの構成、Maven 依存関係の解決、C# バインディングの生成、Native Library Interop (NLI) アプローチを使用したネイティブ Android ライブラリの統合について説明します。 Android バインディング、AAR/JAR 統合、Kotlin 相互運用性、Maven 依存関係、またはネイティブ Android SDK から .NET へのブリッジについて質問された場合に使用します。
