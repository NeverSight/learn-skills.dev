.NET MAUI および .NET for iOS プロジェクトで iOS 用のスリム/ネイティブ プラットフォーム相互運用バインディングを作成および更新します。 Swift/Objective-C ラッパーの作成、Xcode プロジェクトの構成、C# API 定義の生成、Native Library Interop (NLI) アプローチを使用したネイティブ iOS ライブラリの統合について説明します。 iOS バインディング、xcframework 統合、Swift 相互運用性、Objective Sharpie、またはネイティブ iOS SDK の .NET へのブリッジについて質問された場合に使用します。
