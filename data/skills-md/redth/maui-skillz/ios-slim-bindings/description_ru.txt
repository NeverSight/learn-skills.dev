Создавайте и обновляйте привязки взаимодействия тонкой/собственной платформы для iOS в проектах .NET MAUI и .NET для iOS. Помогает создавать оболочки Swift/Objective-C, настраивать проекты Xcode, генерировать определения API C# и интегрировать собственные библиотеки iOS с использованием подхода Native Library Interop (NLI). Используйте, когда вас спрашивают о привязках iOS, интеграции xcframework, взаимодействии Swift, Objective Sharpie или подключении собственных SDK iOS к .NET.
