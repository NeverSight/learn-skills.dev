Créez et mettez à jour des liaisons d'interopérabilité de plate-forme slim/native pour iOS dans les projets .NET MAUI et .NET pour iOS. Guide dans la création de wrappers Swift/Objective-C, la configuration de projets Xcode, la génération de définitions d'API C# et l'intégration de bibliothèques iOS natives à l'aide de l'approche Native Library Interop (NLI). À utiliser lorsque vous êtes interrogé sur les liaisons iOS, l'intégration de xcframework, l'interopérabilité Swift, Objective Sharpie ou le pont entre les SDK iOS natifs et .NET.
