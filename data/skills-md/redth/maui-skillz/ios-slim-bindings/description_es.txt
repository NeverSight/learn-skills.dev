Cree y actualice enlaces de interoperabilidad de plataformas nativas/esbeltas para iOS en proyectos .NET MAUI y .NET para iOS. Guía a través de la creación de contenedores Swift/Objective-C, la configuración de proyectos Xcode, la generación de definiciones de API de C# y la integración de bibliotecas nativas de iOS utilizando el enfoque Native Library Interop (NLI). Úselo cuando se le pregunte sobre enlaces de iOS, integración de xcframework, interoperabilidad Swift, Objective Sharpie o cómo conectar SDK nativos de iOS a .NET.
