Используйте, когда пользователь упоминает настройку, инициализацию, новые проекты, файлы рабочего процесса, настройку CI, настройку LSP, настройку MCP, настройку кодекса, настройку открытого кода, настройку 2-агентов, координацию PM, настройку курсора, жгут памяти, интеграцию claude-mem, межсессионную память, правила локализации, правила адаптации. НЕ загружайте для: работы по внедрению, проверки, проверки сборки или развертывания.
