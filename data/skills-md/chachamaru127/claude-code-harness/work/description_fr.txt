Implémentez les tâches dans Plans.md. Renseignez-vous sur la portée et décidez automatiquement, d'une tâche à toutes les tâches. À utiliser lorsque l'utilisateur mentionne « /work », exécute le plan, implémente des tâches, crée des fonctionnalités, travaille sur des tâches, « tout faire », « implémenter », « 実装して », « 全部やって », « ここだけ ». Ne chargez PAS pour : la planification, les révisions, la configuration, le déploiement ou le breezing (exécution en équipe).
