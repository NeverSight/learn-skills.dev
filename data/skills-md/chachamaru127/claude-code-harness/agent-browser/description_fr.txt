Utiliser lorsque les utilisateurs demandent à naviguer sur des sites Web, remplissez formulaires, prendre des captures d'écran, extraire des données Web, tester des applications Web ou automatiser les flux de travail du navigateur. Les phrases de déclenchement incluent « aller sur [url] », « cliquer sur », « remplir le formulaire », « prendre une capture d'écran », « gratter », « automatiser », « tester le site Web », « se connecter » ou toute demande d'interaction du navigateur. Ne chargez PAS pour : le partage d'URL, l'intégration de liens, les fichiers image de capture d'écran.
