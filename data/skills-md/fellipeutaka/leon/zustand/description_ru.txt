Реализуйте глобальное управление состоянием для приложений React/TypeScript. Используйте при создании, изменении или отладке хранилищ Zustand, реализации шаблонов управления состоянием (срезы, сохранение, инструменты разработчика, погружение), настройке Zustand с помощью Next.js/SSR, написании тестов для хранилищ Zustand или работе с типизацией TypeScript для Zustand (создание с карри, StateCreator, мутаторы промежуточного программного обеспечения).
