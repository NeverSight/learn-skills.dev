Implementieren Sie eine globale Statusverwaltung für React/TypeScript-Anwendungen. Wird zum Erstellen, Ändern oder Debuggen von Zustandsspeichern, zum Implementieren von Zustandsverwaltungsmustern (Slices, Persist, Devtools, Immer), zum Einrichten von Zustand mit Next.js/SSR, zum Schreiben von Tests für Zustandsspeicher oder zum Arbeiten mit TypeScript-Typisierung für Zustand (Curried Create, StateCreator, Middleware-Mutatoren) verwendet.
