تنفيذ إدارة الحالة العالمية لتطبيقات React/TypeScript. يُستخدم عند إنشاء متاجر Zustand أو تعديلها أو تصحيح أخطاء متاجر Zustand، أو تنفيذ أنماط إدارة الحالة (الشرائح، أو الاستمرار، أو أدوات التطوير، أو immer)، أو إعداد Zustand باستخدام Next.js/SSR، أو كتابة اختبارات لمتاجر Zustand، أو العمل مع كتابة TypeScript لـ Zustand (الإنشاء المتقن، أو StateCreator، أو محولات البرامج الوسيطة).
