Configurez et optimisez les référentiels pour les agents de codage IA. Crée un fichier AGENTS.md minimal,
Lien symbolique CLAUDE.md, docs/REQUIREMENTS.md, docs/BUSINESS-RULES.md, boucles de rétroaction,
et application déterministe (hooks Claude Code, plugins OpenCode). À utiliser lorsque l'utilisateur
souhaite créer un dépôt compatible avec l'IA, configurez AGENTS.md/CLAUDE.md, documentez
exigences/règles métier pour l'IA, ajoutez des hooks de pré-validation pour les flux de travail d'IA, ou
optimiser la structure de la base de code pour les agents de codage.
