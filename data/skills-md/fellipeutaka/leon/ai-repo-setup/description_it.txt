Configura e ottimizza i repository per gli agenti di codifica AI. Crea AGENTS.md minimo,
CLAUDE.md collegamento simbolico, docs/REQUIREMENTS.md, docs/BUSINESS-RULES.md, cicli di feedback,
e applicazione deterministica (hook Claude Code, plugin OpenCode). Utilizzare quando utente
vuole creare un repository compatibile con l'intelligenza artificiale, impostare AGENTS.md/CLAUDE.md, documento
requisiti/regole aziendali per l'intelligenza artificiale, aggiungere hook di pre-impegno per i flussi di lavoro dell'intelligenza artificiale o
ottimizzare la struttura della base di codice per gli agenti di codifica.
