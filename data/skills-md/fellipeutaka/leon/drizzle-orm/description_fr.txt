Drizzle ORM – ORM TypeScript léger et sécurisé pour les bases de données SQL. Déclaration de schéma, requêtes CRUD, jointures, relations, migrations avec Drizzle Kit et amorçage de base de données.
À utiliser lors de la définition de schémas de base de données, de l'écriture de requêtes (sélection/insertion/mise à jour/suppression), de la configuration des migrations, de la configuration de drizzle.config.ts, de l'établissement de connexions à la base de données,
valider les données avec drizzle-zod/valibot, ou travailler avec PostgreSQL, MySQL, SQLite, Turso, Bun SQL, Neon, Supabase, PGlite,
Expo SQLite, Cloudflare D1, PlanetScale, SingleStore, MSSQL, CockroachDB.
