Drizzle ORM — SQL 데이터베이스를 위한 유형 안전하고 가벼운 TypeScript ORM입니다. 스키마 선언, CRUD 쿼리, 조인, 관계, Drizzle Kit를 사용한 마이그레이션 및 데이터베이스 시드.
데이터베이스 스키마 정의, 쿼리 작성(선택/삽입/업데이트/삭제), 마이그레이션 설정, drizzle.config.ts 구성, 데이터베이스 연결 설정,
drizzle-zod/valibot을 사용하여 데이터 유효성을 검사하거나 PostgreSQL, MySQL, SQLite, Turso, Bun SQL, Neon, Supabase, PGlite를 사용하여 작업
엑스포 SQLite, Cloudflare D1, PlanetScale, SingleStore, MSSQL, CockroachDB.
