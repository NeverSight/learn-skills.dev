Drizzle ORM – typsicheres, leichtes TypeScript ORM für SQL-Datenbanken. Schemadeklaration, CRUD-Abfragen, Verknüpfungen, Beziehungen, Migrationen mit Drizzle Kit und Datenbank-Seeding.
Verwendung beim Definieren von Datenbankschemata, Schreiben von Abfragen (Auswählen/Einfügen/Aktualisieren/Löschen), Einrichten von Migrationen, Konfigurieren von drizzle.config.ts, Einrichten von Datenbankverbindungen,
Validieren von Daten mit Drizzle-Zod/Valibot oder Arbeiten mit PostgreSQL, MySQL, SQLite, Turso, Bun SQL, Neon, Supabase, PGlite,
Expo SQLite, Cloudflare D1, PlanetScale, SingleStore, MSSQL, CockroachDB.
