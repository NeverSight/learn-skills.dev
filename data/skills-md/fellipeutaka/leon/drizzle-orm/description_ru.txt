Drizzle ORM — типобезопасная и легкая ORM TypeScript для баз данных SQL. Объявление схемы, запросы CRUD, объединения, отношения, миграции с помощью Drizzle Kit и заполнение базы данных.
Используйте при определении схем базы данных, написании запросов (выбор/вставка/обновление/удаление), настройке миграции, настройке drizzle.config.ts, установлении подключений к базе данных,
проверка данных с помощью drizzle-zod/valibot или работа с PostgreSQL, MySQL, SQLite, Turso, Bun SQL, Neon, Supabase, PGlite,
Expo SQLite, Cloudflare D1, PlanetScale, SingleStore, MSSQL, CockroachDB.
