Drizzle ORM: ORM TypeScript ligero y seguro para tipos para bases de datos SQL. Declaración de esquemas, consultas CRUD, uniones, relaciones, migraciones con Drizzle Kit y siembra de bases de datos.
Úselo al definir esquemas de bases de datos, escribir consultas (seleccionar/insertar/actualizar/eliminar), configurar migraciones, configurar drizzle.config.ts, establecer conexiones de bases de datos,
validar datos con drizzle-zod/valibot, o trabajar con PostgreSQL, MySQL, SQLite, Turso, Bun SQL, Neon, Supabase, PGlite,
Expo SQLite, Cloudflare D1, PlanetScale, SingleStore, MSSQL, CockroachDB.
