Prisma ORM — مجموعة أدوات قاعدة بيانات آمنة للنوع لـ TypeScript/Node.js. نهج المخطط الأول مع لغة Prisma Schema، عميل النوع الآمن الذي يتم إنشاؤه تلقائيًا، وعمليات الترحيل عبر Prisma Migrate.
يُستخدم عند تحديد نماذج قاعدة البيانات في schema.prisma، وكتابة استعلامات CRUD باستخدام Prisma Client، وإعداد العلاقات (1-1، 1-n، m-n)،
تشغيل عمليات الترحيل (ترحيل prisma dev/deploy)، وتكوين prisma.config.ts، باستخدام SQL/TypedSQL الخام، وإنشاء ملحقات العميل،
أو العمل مع PostgreSQL، وMySQL، وSQLite، وSQL Server، وMongoDB، وCockroachDB.
