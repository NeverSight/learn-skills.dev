Prisma ORM — boîte à outils de base de données de type sécurisé pour TypeScript/Node.js. Approche basée sur le schéma avec Prisma Schema Language, client de type sécurisé généré automatiquement, migrations via Prisma Migrate.
À utiliser lors de la définition de modèles de base de données dans schema.prisma, de l'écriture de requêtes CRUD avec Prisma Client, de la configuration des relations (1-1, 1-n, m-n),
exécuter des migrations (prisma migrate dev/deploy), configurer prisma.config.ts, utiliser du SQL brut/TypedSQL, créer des extensions client,
ou travailler avec PostgreSQL, MySQL, SQLite, SQL Server, MongoDB, CockroachDB.
