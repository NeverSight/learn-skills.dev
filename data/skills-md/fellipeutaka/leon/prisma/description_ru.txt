Prisma ORM — типобезопасный набор инструментов для работы с базами данных для TypeScript/Node.js. Подход на основе схемы с использованием языка Prisma Schema, автоматически создаваемый типобезопасный клиент, миграция с помощью Prisma Migrate.
Используйте при определении моделей базы данных в Schema.prisma, написании CRUD-запросов с помощью Prisma Client, настройке отношений (1-1, 1-n, m-n),
выполнение миграций (prismamigration dev/deploy), настройка prisma.config.ts, использование необработанного SQL/TypedSQL, создание клиентских расширений,
или работа с PostgreSQL, MySQL, SQLite, SQL Server, MongoDB, CockroachDB.
