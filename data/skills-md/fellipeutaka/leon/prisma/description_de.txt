Prisma ORM – typsicheres Datenbank-Toolkit für TypeScript/Node.js. Schema-First-Ansatz mit Prisma Schema Language, automatisch generierter typsicherer Client, Migrationen über Prisma Migrate.
Verwendung beim Definieren von Datenbankmodellen in schema.prisma, beim Schreiben von CRUD-Abfragen mit Prisma Client, beim Einrichten von Beziehungen (1-1, 1-n, m-n),
Ausführen von Migrationen (prisma migrate dev/deploy), Konfigurieren von prisma.config.ts, Verwenden von Raw SQL/TypedSQL, Erstellen von Client-Erweiterungen,
oder mit PostgreSQL, MySQL, SQLite, SQL Server, MongoDB, CockroachDB arbeiten.
