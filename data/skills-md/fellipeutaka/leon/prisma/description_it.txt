Prisma ORM: toolkit di database indipendente dai tipi per TypeScript/Node.js. Approccio basato sullo schema con Prisma Schema Language, client indipendente dai tipi generato automaticamente, migrazioni tramite Prisma Migrate.
Utilizzare quando si definiscono modelli di database in schema.prisma, si scrivono query CRUD con Prisma Client, si impostano relazioni (1-1, 1-n, m-n),
eseguire migrazioni (prisma migrate dev/deploy), configurare prisma.config.ts, utilizzare SQL/TypedSQL non elaborato, creare estensioni client,
o lavorare con PostgreSQL, MySQL, SQLite, SQL Server, MongoDB, CockroachDB.
