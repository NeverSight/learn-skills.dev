Prisma ORM: kit de herramientas de base de datos con seguridad de tipos para TypeScript/Node.js. Enfoque de esquema primero con Prisma Schema Language, cliente de tipo seguro generado automáticamente, migraciones a través de Prisma Migrate.
Úselo al definir modelos de bases de datos en esquema.prisma, escribir consultas CRUD con Prisma Client, configurar relaciones (1-1, 1-n, m-n),
ejecutar migraciones (prisma migra dev/deploy), configurar prisma.config.ts, usar SQL sin formato/TypedSQL, crear extensiones de cliente,
o trabajar con PostgreSQL, MySQL, SQLite, SQL Server, MongoDB, CockroachDB.
