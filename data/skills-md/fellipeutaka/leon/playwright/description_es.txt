Escriba, depure y mantenga pruebas integrales de Playwright para aplicaciones web. Úselo cuando trabaje con archivos de prueba de Playwright, configure playwright.config.ts, escriba automatización del navegador, depure pruebas E2E inestables, configure autenticación para pruebas, burla/interceptación de API, pruebas de regresión visual, pruebas de accesibilidad o integración de CI/CD para pruebas de navegador. Activadores: Dramaturgo, prueba E2E, de un extremo a otro, prueba de navegador, @playwright/test, playwright.config, modelo de objetos de página, dispositivo de prueba, instantánea visual, visor de seguimiento.
