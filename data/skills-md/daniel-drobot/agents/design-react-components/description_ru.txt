Навыки разработки React для просмотра снимков экрана пользовательского интерфейса и разделения их на модульные, безголовые, нестилизованные компоненты React в соответствии с принципами пользовательского интерфейса, управляемого компонентами (CDUI). Используйте, когда вам нужно спроектировать пользовательский интерфейс на основе визуального эталона перед началом реализации.
