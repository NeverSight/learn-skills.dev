Compétence en ingénierie React pour examiner les captures d'écran de l'interface utilisateur et les diviser en composants React modulaires, sans tête et sans style, selon les principes de l'interface utilisateur pilotée par les composants (CDUI). À utiliser lorsque vous devez concevoir une interface utilisateur à partir d'une référence visuelle avant de commencer l'implémentation.
