CLI d'automatisation du navigateur avec intégration Nstbrowser pour les agents IA. À utiliser lorsque l'utilisateur a besoin d'une empreinte digitale de navigateur avancée, d'une gestion de profil, d'une configuration de proxy, d'opérations par lots sur plusieurs profils de navigateur ou d'une pagination basée sur le curseur pour de grands ensembles de données. Les déclencheurs incluent les requêtes « utiliser le profil NST », « configurer le proxy pour le profil », « gérer les profils de navigateur », « mettre à jour les profils par lots », « démarrer plusieurs navigateurs », « lister les profils avec pagination » ou toute tâche nécessitant les fonctionnalités anti-détection de Nstbrowser.
