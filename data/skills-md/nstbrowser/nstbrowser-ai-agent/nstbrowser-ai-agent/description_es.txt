CLI de automatización del navegador con integración de Nstbrowser para agentes de IA. Úselo cuando el usuario necesite huellas digitales avanzadas del navegador, administración de perfiles, configuración de proxy, operaciones por lotes en múltiples perfiles del navegador o paginación basada en cursor para grandes conjuntos de datos. Los activadores incluyen solicitudes para "usar perfil NST", "configurar proxy para perfil", "administrar perfiles de navegador", "actualizar perfiles por lotes", "iniciar varios navegadores", "enumerar perfiles con paginación" o cualquier tarea que requiera las funciones antidetección de Nstbrowser.
