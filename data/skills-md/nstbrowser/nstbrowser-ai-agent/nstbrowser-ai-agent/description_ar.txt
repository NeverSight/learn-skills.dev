أتمتة المتصفح CLI مع تكامل Nstbrowser لوكلاء الذكاء الاصطناعي. يُستخدم عندما يحتاج المستخدم إلى بصمات المتصفح المتقدمة، أو إدارة ملف التعريف، أو تكوين الوكيل، أو العمليات المجمعة على ملفات تعريف متصفح متعددة، أو ترقيم الصفحات المستند إلى المؤشر لمجموعات البيانات الكبيرة. تتضمن المشغلات طلبات "استخدام ملف تعريف NST"، أو "تكوين الوكيل للملف الشخصي"، أو "إدارة ملفات تعريف المتصفح"، أو "ملفات تعريف التحديث المجمعة"، أو "بدء متصفحات متعددة"، أو "سرد ملفات التعريف مع ترقيم الصفحات"، أو أي مهمة تتطلب ميزات مكافحة الاكتشاف في Nstbrowser.
