قم بتنفيذ الصفقات وتحليل الأسواق وإدارة المحافظ باستخدام أداة سطر الأوامر apcacli لواجهة برمجة تطبيقات التداول الخاصة بـ Alpaca. يُستخدم عندما يريد المستخدم تداول الأسهم أو صناديق الاستثمار المتداولة أو الخيارات أو العملات المشفرة أو التحقق من بيانات السوق أو عرض المراكز أو إدارة الأوامر أو تحليل محفظته. استخدم أيضًا عندما يذكرون "شراء" أو "بيع" أو "تداول" أو "بيانات السوق" أو "سعر السهم" أو "المحفظة" أو "رصيد الحساب" أو "تداول الألبكة".
