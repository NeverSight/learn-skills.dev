Используйте полный интерфейс командной строки OpenAI Codex в качестве менеджера проекта. Включает: обслуживание базы знаний (автоматическое отслеживание последних функций Кодекса), выполнение задач (быстрое проектирование → запуск → мониторинг → контроль качества → итерация → отчет), управление конфигурацией (флаги функций/модели/навыки/MCP). Используйте tmux для интерактивного TUI с перехватчиками уведомлений + панельным монитором для асинхронного пробуждения. НЕ для: простого однострочного редактирования (используйте редактирование), чтения файлов (используйте чтение), быстрых вопросов и ответов (отвечайте напрямую).
