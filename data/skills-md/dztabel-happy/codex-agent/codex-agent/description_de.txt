Betrieb der vollständigen OpenAI Codex CLI als Projektmanager. Beinhaltet: Wissensdatenbankpflege (automatische Verfolgung der neuesten Codex-Funktionen), Aufgabenausführung (promptes Design → Ausführung → Überwachung → Qualitätssicherung → Iteration → Berichterstellung), Konfigurationsmanagement (Funktionsflags / Modelle / Fähigkeiten / MCP). Verwendet tmux für eine interaktive TUI; Asynchrones Aufwecken über Benachrichtigungs-Hooks + Pane-Monitor. NICHT für: einfache einzeilige Bearbeitungen (Bearbeiten verwenden), Dateilesen (Lesen verwenden), schnelle Fragen und Antworten (direkt antworten).
