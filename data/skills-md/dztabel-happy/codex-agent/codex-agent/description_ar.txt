قم بتشغيل OpenAI Codex CLI الكامل كمدير للمشروع. يتضمن: صيانة قاعدة المعرفة (التتبع التلقائي لأحدث ميزات الدستور الغذائي)، وتنفيذ المهام (التصميم الفوري ← التشغيل ← المراقبة ← ضمان الجودة ← التكرار ← التقرير)، وإدارة التكوين (أعلام الميزات / النماذج / المهارات / MCP). استخدم tmux لقيادة TUI التفاعلية؛ استخدم خطافات الإخطار + شاشة الجزء لعمليات التنبيه غير المتزامنة. ليس من أجل: تعديلات سطر واحد بسيطة (استخدم التحرير)، وقراءة الملفات (استخدم القراءة)، والأسئلة والأجوبة السريعة (الإجابة مباشرة).
