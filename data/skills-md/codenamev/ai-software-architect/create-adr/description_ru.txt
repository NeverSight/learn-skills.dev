Создает НОВУЮ запись архитектурного решения (ADR), документирующую конкретное архитектурное решение. Используйте, когда пользователь запрашивает «Создать ADR для [темы]», «Документировать решение по [теме]», «Написать ADR для [выбор]» или при документировании выбора технологий, шаблонов или архитектурных подходов. НЕ используйте для обзоров (используйте обзор архитектуры или обзор специалиста), проверки существующих ADR (используйте статус архитектуры) или общей документации.
