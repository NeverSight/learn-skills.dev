Crea un NUEVO Registro de Decisión Arquitectónica (ADR) que documenta una decisión arquitectónica específica. Úselo cuando el usuario solicite "Crear ADR para [tema]", "Documentar decisión sobre [tema]", "Escribir ADR para [elección]" o cuando documente opciones tecnológicas, patrones o enfoques arquitectónicos. NO lo use para revisiones (use revisión de arquitectura o revisión de especialistas), verifique ADR existentes (use estado de arquitectura) o documentación general.
