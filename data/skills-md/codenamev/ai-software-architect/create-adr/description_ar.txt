إنشاء سجل قرار معماري جديد (ADR) لتوثيق قرار معماري محدد. يُستخدم عندما يطلب المستخدم "إنشاء ADR لـ [الموضوع]"، أو "توثيق القرار حول [الموضوع]"، أو "كتابة ADR لـ [الاختيار]"، أو عند توثيق اختيارات التكنولوجيا أو الأنماط أو الأساليب المعمارية. لا تستخدم للمراجعات (استخدم مراجعة البنية أو المراجعة المتخصصة)، أو التحقق من التفاعلات الضارة الحالية (استخدم حالة البنية)، أو الوثائق العامة.
