Effectuer un examen complet de l'architecture multiperspective en utilisant TOUS les membres de l'équipe d'architecture. À utiliser lorsque l'utilisateur demande « Démarrer la révision de l'architecture », « Révision complète de l'architecture », « Réviser l'architecture pour la version X.Y.Z », « Effectuer une révision complète » ou lorsqu'il souhaite une évaluation sous plusieurs perspectives. Ne PAS utiliser pour les examens par un seul spécialiste (utilisez plutôt l'examen par un spécialiste) ou pour les vérifications de statut (utilisez plutôt le statut d'architecture).
