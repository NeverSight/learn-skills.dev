Проводит комплексный многоаспектный анализ архитектуры с участием ВСЕХ членов архитектурной команды. Используйте, когда пользователь запрашивает «Начать проверку архитектуры», «Полную проверку архитектуры», «Просмотреть архитектуру для версии X.Y.Z», «Провести комплексную проверку» или когда ему нужна оценка с нескольких точек зрения. НЕ используйте для обзоров одного специалиста (вместо этого используйте обзор специалиста) или для проверки статуса (вместо этого используйте статус архитектуры).
