Sistema de diseño interno de Casper Studios para generar una interfaz de usuario SaaS consistente y de nivel de producción. Utilice esta habilidad siempre que genere código de interfaz de usuario para herramientas internas, aplicaciones de cliente, paneles, POC, prototipos o cualquier interfaz visual, incluso maquetas o artefactos rápidos. Aplíquelo cada vez que la salida sea un componente, página o diseño de React. Si el usuario menciona "nuestro sistema de diseño", "estilo Casper", "coincide con nuestra apariencia" o le pide que cree cualquier tipo de aplicación o interfaz, utilice esta habilidad. También se activa al rediseñar o tematizar la interfaz de usuario existente para que coincida con el lenguaje visual de Casper. Esta habilidad tiene prioridad sobre la orientación genérica de diseño de interfaz.
