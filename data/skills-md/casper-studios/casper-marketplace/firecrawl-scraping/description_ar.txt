تجريف صفحة الويب وموقع الويب باستخدام Firecrawl API. استخدم هذه المهارة عند نسخ مقالات الويب، أو منشورات المدونات، أو صفحات الوثائق، أو المحتوى المحمي بنظام حظر الاشتراك غير المدفوع، أو مواقع JavaScript الثقيلة. يتم تشغيل طلبات استخراج مواقع الويب، أو استخراج محتوى المقالة، أو تحويل الصفحات إلى تخفيض السعر، أو التعامل مع الحماية ضد الروبوتات.
