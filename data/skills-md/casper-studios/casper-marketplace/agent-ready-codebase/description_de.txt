Diese Fähigkeit sollte verwendet werden, wenn eine Codebasis auf die Bereitschaft von KI-Agenten überprüft wird oder wenn Verbesserungen angeleitet werden, damit eine Codebasis gut mit Agenten-Codierungstools funktioniert. Dies gilt, wenn Benutzer Testabdeckung, Dateistruktur, Typsystemnutzung, Geschwindigkeit der Entwicklungsumgebung oder automatisierte Durchsetzung bewerten möchten – die fünf Säulen, die bestimmen, wie effektiv Codierungsagenten in einem Projekt arbeiten können. Wird ausgelöst bei „Meine Codebasis prüfen“, „Diesen Agenten vorbereiten“, „Für KI-Agenten verbessern“, „Agentenfreundlich“ oder Fragen dazu, warum Agenten mit einer Codebasis Probleme haben.
