Pulisce i file di codice rimuovendo il codice morto, semplificando la struttura ed eliminando la ridondanza senza modificare il comportamento. Da utilizzare quando (1) si pulisce il codice dopo l'implementazione, (2) si riduce la complessità di un file o di una funzione, (3) si rimuove il codice morto e le importazioni inutilizzate, (4) si semplifica il flusso di controllo e le espressioni, (5) si applicano idiomi linguistici moderni a modelli legacy o (6) si migliora la leggibilità di codice denso o contorto.
