Bereinigt Codedateien, indem toter Code entfernt, die Struktur vereinfacht und Redundanz beseitigt wird, ohne das Verhalten zu ändern. Wird verwendet, wenn (1) Code nach der Implementierung bereinigt wird, (2) die Komplexität in einer Datei oder Funktion reduziert wird, (3) toter Code und ungenutzte Importe entfernt werden, (4) Kontrollfluss und Ausdrücke vereinfacht werden, (5) moderne Sprachidiome auf ältere Muster angewendet werden oder (6) die Lesbarkeit von dichtem oder kompliziertem Code verbessert wird.
