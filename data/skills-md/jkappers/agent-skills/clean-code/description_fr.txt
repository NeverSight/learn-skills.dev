Nettoie les fichiers de code en supprimant le code mort, en simplifiant la structure et en éliminant la redondance sans modifier le comportement. À utiliser pour (1) nettoyer le code après l'implémentation, (2) réduire la complexité d'un fichier ou d'une fonction, (3) supprimer le code mort et les importations inutilisées, (4) simplifier le flux de contrôle et les expressions, (5) appliquer des idiomes de langage moderne à des modèles hérités ou (6) améliorer la lisibilité d'un code dense ou alambiqué.
