Limpia archivos de código eliminando código inactivo, simplificando la estructura y eliminando la redundancia sin cambiar el comportamiento. Úselo cuando (1) limpie el código después de la implementación, (2) reduzca la complejidad de un archivo o función, (3) elimine el código inactivo y las importaciones no utilizadas, (4) simplifique el flujo de control y las expresiones, (5) aplique modismos del lenguaje moderno a patrones heredados o (6) mejore la legibilidad de código denso o complicado.
