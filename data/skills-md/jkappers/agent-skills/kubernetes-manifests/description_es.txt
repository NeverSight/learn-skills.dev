Escriba y revise manifiestos de Kubernetes listos para producción con seguridad, gestión de recursos y mejores prácticas operativas. Úselo al (1) crear nuevos manifiestos de implementación, servicio o ingreso de Kubernetes, (2) revisar manifiestos existentes para determinar la preparación para la producción, (3) agregar sondas de estado, contextos de seguridad o límites de recursos, (4) configurar la estructura base/superpuesta de Kustomize, (5) contener aplicaciones .NET o React para Kubernetes, o (6) configurar PodDisruptionBudgets, distribución de topología o escalado automático.
