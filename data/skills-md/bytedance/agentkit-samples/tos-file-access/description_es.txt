Cargue archivos o directorios en Volcano Engine TOS (Torch Object Storage) y descargue archivos desde URL. Utilice esta habilidad cuando (1) cargue archivos o directorios generados por el Agente (como videos, imágenes, informes, carpetas de salida) a TOS para compartirlos, (2) descargue archivos de las URL antes del procesamiento del Agente.
