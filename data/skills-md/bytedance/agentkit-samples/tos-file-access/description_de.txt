Laden Sie Dateien oder Verzeichnisse in Volcano Engine TOS (Torch Object Storage) hoch und laden Sie Dateien von URLs herunter. Verwenden Sie diese Fähigkeit, wenn (1) vom Agenten generierte Dateien oder Verzeichnisse (wie Videos, Bilder, Berichte, Ausgabeordner) zum Teilen in TOS hochladen, (2) Dateien von URLs herunterladen, bevor der Agent sie verarbeitet.
