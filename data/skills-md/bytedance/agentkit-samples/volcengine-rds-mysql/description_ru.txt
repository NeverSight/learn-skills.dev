Используйте сервер RDS MySQL MCP Volcano Engine, чтобы помочь пользователям управлять экземплярами RDS MySQL, выполнять операции с базами данных, управлять учетными записями и выполнять задачи обслуживания. Вы можете напрямую запустить скрипт uv run ./scripts/call_rds_mysql.py, чтобы получить результаты в реальном времени.
