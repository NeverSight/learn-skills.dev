Verwenden Sie den Volcano Engine RDS MySQL MCP Server, um Benutzern bei der Durchführung von RDS MySQL-bezogenen Instanzverwaltungs-, Datenbankoperationen-, Kontoverwaltungs- und Betriebs- und Wartungsaufgaben zu helfen. Sie können das Skript uv run ./scripts/call_rds_mysql.py direkt ausführen, um Echtzeitergebnisse zu erhalten.
