Запустите локальный сервер Microcks с помощью Docker Compose для макетирования и тестирования API. Используйте, когда разработчику необходимо запустить Microcks локально, чтобы имитировать REST API из спецификаций OpenAPI. Microcks — это инструмент CNCF с открытым исходным кодом для моделирования и тестирования API.
