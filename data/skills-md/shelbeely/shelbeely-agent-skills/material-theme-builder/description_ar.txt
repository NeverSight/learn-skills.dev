قم بإنشاء سمات ألوان 3 لتصميم المواد برمجيًا من لون مصدر باستخدام @material/material-color-utilities، وهو نفس المحرك الذي يقوم بتشغيل Material Theme Builder. استخدم هذا عندما يطلب المستخدم إنشاء لوحة ألوان M3، أو إنشاء سمة مخصصة من لون العلامة التجارية، أو تصدير رموز M3 إلى تكوينات CSS أو JSON أو إطار العمل.
