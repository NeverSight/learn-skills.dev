Создавайте цветовые темы Material Design 3 программно из исходного цвета с помощью @material/material-color-utilities, того же движка, который используется в Material Theme Builder. Используйте это, когда пользователь просит создать цветовую палитру M3, создать собственную тему на основе цвета бренда или экспортировать токены M3 в конфигурации CSS, JSON или платформы.
