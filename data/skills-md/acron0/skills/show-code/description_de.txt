Öffnen Sie Codedateien in einem tmux-Bereich in der entsprechenden Zeile mit dem bevorzugten Editor des Benutzers. Verwenden Sie diese Option, wenn der Benutzer sagt: „Zeige mir den Code“, „öffne dies in einem Editor“, „Lass mich diesen Code sehen“ oder möchte den besprochenen Code in einem Editor anzeigen.
