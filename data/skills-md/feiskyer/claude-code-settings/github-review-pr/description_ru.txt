Просмотрите запросы на включение GitHub с подробным многоаспектным анализом кода с использованием параллельных субагентов. Используйте этот навык всякий раз, когда пользователь хочет просмотреть PR, запрашивает проверку кода в запросе на включение, упоминает «просмотреть PR», «проверить этот PR», «посмотреть запрос на включение» или ссылается на номер PR или URL-адрес PR GitHub. Также срабатывает, когда пользователь хочет получить отзыв об изменениях кода, хочет утвердить или запросить изменения в PR или просит просмотреть чей-то вклад.
