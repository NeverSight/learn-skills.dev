قم بمراجعة طلبات سحب GitHub من خلال تحليل التعليمات البرمجية التفصيلي متعدد وجهات النظر باستخدام وكلاء فرعيين متوازيين. استخدم هذه المهارة عندما يريد المستخدم مراجعة العلاقات العامة، أو يطلب مراجعة التعليمات البرمجية في طلب سحب، أو يذكر "مراجعة العلاقات العامة"، أو "التحقق من هذه العلاقات العامة"، أو "النظر في طلب السحب"، أو يشير إلى رقم العلاقات العامة أو عنوان URL للعلاقات العامة على GitHub. يتم تشغيله أيضًا عندما يريد المستخدم الحصول على تعليقات حول تغييرات التعليمات البرمجية، أو يريد الموافقة على تغييرات أو طلبها على العلاقات العامة، أو يطلب مراجعة مساهمة شخص ما.
