قم بإنشاء أوامر شرطة مائلة مخصصة لـ Claude Code باستخدام البنية المناسبة والواجهة الأمامية وأفضل الممارسات. استخدم هذه المهارة عندما يريد المستخدم إنشاء أمر جديد، أو إضافة أمر شرطة مائلة، أو إنشاء أمر مخصص، أو ذكر "إنشاء أمر"، أو "أمر جديد"، أو "إضافة أمر"، أو "إنشاء أمر" لكلود كود. يتم تشغيله أيضًا عندما يريد المستخدم تحويل سير العمل إلى أمر قابل لإعادة الاستخدام.
