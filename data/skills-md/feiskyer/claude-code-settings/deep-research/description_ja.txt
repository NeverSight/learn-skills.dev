深い調査のためのマルチエージェント オーケストレーション ワークフロー: 研究目標を並列サブ目標に分割し、非対話モード (`claude -p`) でクロード コードを使用してサブプロセスを実行します。最初に Web とデータ収集にインストールされたスキルを使用し、次に MCP ツールを使用します。サブ結果をスクリプトで集計し、章ごとに絞り込み、「最終レポート ファイル パス + 主要な調査結果/推奨事項の概要」を提供します。用途: 体系的な Web/データ調査、競合他社/業界分析、バッチ リンク/データセット セグメントの取得、長文執筆と証拠の統合、またはユーザーが「深い調査/広範な調査/マルチエージェントの並行調査/マルチプロセスの調査」に言及した場合。
