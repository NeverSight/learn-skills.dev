Workflow d'orchestration multi-agents pour une recherche approfondie : diviser un objectif de recherche en sous-objectifs parallèles, exécuter des sous-processus avec Claude Code en mode non interactif (`claude -p`) ; utiliser les compétences installées d'abord pour le Web et la collecte de données, puis pour les outils MCP ; regroupez les sous-résultats avec des scripts, affinez par chapitres et fournissez « le chemin du fichier du rapport final + le résumé des principales conclusions/recommandations ». À utiliser pour : la recherche systématique sur le Web/les données, l'analyse des concurrents/du secteur, la récupération de liens par lots/de segments d'ensembles de données, la rédaction de longs formulaires et l'intégration de preuves, ou lorsque l'utilisateur mentionne "recherche approfondie/recherche étendue/recherche parallèle multi-agents/recherche multi-processus".
