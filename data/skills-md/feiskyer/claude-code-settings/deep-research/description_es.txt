Flujo de trabajo de orquestación de múltiples agentes para una investigación profunda: divida un objetivo de investigación en subobjetivos paralelos, ejecute subprocesos con Claude Code en modo no interactivo (`claude -p`); utilice primero las habilidades instaladas para la web y la recopilación de datos, luego las herramientas MCP; agregue subresultados con guiones, refine por capítulos y entregue "ruta del archivo del informe final + resumen de hallazgos clave/recomendaciones". Úselo para: investigación sistemática de datos/web, análisis de competidores/industria, recuperación de enlaces por lotes/segmentos de conjuntos de datos, redacción de formato largo e integración de evidencia, o cuando el usuario menciona "investigación profunda/investigación amplia/investigación paralela de múltiples agentes/investigación de múltiples procesos".
