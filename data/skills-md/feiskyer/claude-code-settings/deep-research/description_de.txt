Multi-Agent-Orchestrierungs-Workflow für tiefgreifende Forschung: Unterteilen Sie ein Forschungsziel in parallele Unterziele, führen Sie Unterprozesse mit Claude Code im nicht-interaktiven Modus aus („claude -p“); Verwenden Sie installierte Fähigkeiten zuerst für die Web- und Datenerfassung und dann für MCP-Tools. Aggregieren Sie Unterergebnisse mit Skripten, verfeinern Sie sie nach Kapiteln und liefern Sie „Dateipfad des Abschlussberichts + Zusammenfassung der wichtigsten Ergebnisse/Empfehlungen“. Verwendung für: systematische Web-/Datenrecherche, Wettbewerber-/Branchenanalyse, Batch-Link-/Datensatzsegmentabruf, langes Schreiben und Beweisintegration oder wenn der Benutzer „tiefgreifende Recherche/umfassende Recherche/parallele Recherche mit mehreren Agenten/recherche mit mehreren Prozessen“ erwähnt.
