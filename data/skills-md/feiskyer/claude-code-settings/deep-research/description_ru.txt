Рабочий процесс многоагентной оркестрации для глубоких исследований: разбейте цель исследования на параллельные подцели, запустите подпроцессы с помощью Claude Code в неинтерактивном режиме (`claude -p`); сначала используйте приобретенные навыки для работы в сети и сбора данных, а затем инструменты MCP; агрегировать промежуточные результаты с помощью сценариев, уточнять по главам и предоставлять «путь к файлу окончательного отчета + сводку основных выводов/рекомендаций». Используется для: систематического исследования сети/данных, анализа конкурентов/отрасли, пакетного извлечения ссылок/сегментов набора данных, написания длинных форм и интеграции доказательств, а также когда пользователь упоминает «глубокое исследование/широкое исследование/многоагентное параллельное исследование/многопроцессное исследование».
