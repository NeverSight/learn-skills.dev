ترجمة المقالات والنصوص التقنية باللغة الإنجليزية أو اليابانية إلى اللغة الصينية الطبيعية بطلاقة. استخدم هذه المهارة عندما يريد المستخدم ترجمة نص إلى الصينية، أو يطلب ترجمة صينية، أو يذكر "ترجمة إلى الصينية"، أو "翻译"، أو يقدم محتوى تقنيًا باللغة الإنجليزية/اليابانية للترجمة، أو يريد تحويل أي نص إلى اللغة الصينية. يتم تشغيله أيضًا عندما يلصق المستخدم نصًا ويطلب ترجمته، أو يشير إلى ملف لترجمته إلى اللغة الصينية.
