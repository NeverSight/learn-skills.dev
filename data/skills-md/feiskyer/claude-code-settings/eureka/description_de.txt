Erfassen Sie technische Durchbrüche und wandeln Sie sie in umsetzbare, wiederverwendbare Dokumentation um. Verwenden Sie diese Fähigkeit, wenn der Benutzer einen wichtigen technischen Einblick gewonnen hat, ein schwieriges Problem gelöst hat, eine nicht offensichtliche Lösung entdeckt hat oder einen Durchbruch dokumentieren möchte. Wird auch ausgelöst, wenn der Benutzer „Eureka“, „Durchbruch“, „diese Erkenntnis dokumentieren“, „diese Entdeckung erfassen“ erwähnt oder einen technischen Erfolg in wiederverwendbares Wissen umwandeln möchte.
