التقط الإنجازات التقنية وقم بتحويلها إلى وثائق قابلة للتنفيذ وقابلة لإعادة الاستخدام. استخدم هذه المهارة عندما يحقق المستخدم رؤية فنية مهمة، أو يحل مشكلة صعبة، أو يكتشف حلاً غير واضح، أو يريد توثيق لحظة اختراق. يتم تشغيله أيضًا عندما يذكر المستخدم "eureka" أو "اختراق" أو "توثيق هذه الرؤية" أو "التقاط هذا الاكتشاف" أو يريد تحويل فوز تقني إلى معرفة قابلة لإعادة الاستخدام.
