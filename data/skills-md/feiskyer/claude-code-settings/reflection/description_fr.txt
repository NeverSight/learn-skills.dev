Analysez les sessions de développement, capturez les apprentissages et améliorez les instructions de Claude Code. À utiliser lorsque l'utilisateur souhaite réfléchir à une session, améliorer CLAUDE.md, extraire des enseignements ou optimiser la collaboration IA-humain. Prend en charge deux modes : rapide (par défaut) se concentre sur les améliorations de CLAUDE.md, le mode profond effectue une analyse complète de session avec capture d'apprentissage.
