سير عمل منسق متعدد المثيلات (متعدد الوكلاء) للبحث العميق: قم بتقسيم هدف البحث إلى أهداف فرعية متوازية، وتشغيل العمليات الفرعية باستخدام Codex CLI (`codex exec`) داخل صندوق الحماية الافتراضي "workspace-write`"؛ تفضيل المهارات المثبتة لمهام الويب والتحصيل، والرجوع إلى أدوات MCP؛ تجميع النتائج الفرعية عبر البرامج النصية، وتنقيحها حسب الفصل، وتقديم "مسار ملف التقرير النهائي + ملخص النتائج/التوصيات الرئيسية". يُستخدم من أجل: البحث المنهجي على الويب/البيانات، أو تحليل المنافسين/الصناعة، أو استرجاع أجزاء الارتباط/مجموعة البيانات المجمعة، أو الكتابة الطويلة مع تكامل الأدلة، أو عندما يذكر المستخدم "بحث عميق / بحث واسع / بحث متوازي متعدد الوكلاء / بحث متعدد العمليات".
