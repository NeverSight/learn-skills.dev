Руководство по работе с компонентами React системы проектирования Syntax компании Cambly. Используйте при реализации функций пользовательского интерфейса с помощью компонентов Syntax, преобразовании существующего пользовательского интерфейса для использования системы дизайна, исправлении проблем со стилем или доступностью компонентов или при проверке кода, использующего Syntax.
