Guía para exponer los puntos finales del producto PostHog como herramientas MCP. Úselo al crear puntos finales de API nuevos o actualizarlos, agregar definiciones de herramientas MCP, crear andamios de configuraciones YAML o escribir serializadores con buenas descripciones. Cubre el proceso completo desde el serializador de Django hasta el controlador de herramientas TypeScript generado.
