Guida per esporre gli endpoint del prodotto PostHog come strumenti MCP. Da utilizzare durante la creazione di endpoint API nuovi o l'aggiornamento, l'aggiunta di definizioni di strumenti MCP, lo scaffolding di configurazioni YAML o la scrittura di serializzatori con descrizioni valide. Copre l'intera pipeline dal serializzatore Django al gestore dello strumento TypeScript generato.
