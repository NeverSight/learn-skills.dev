Leitfaden zum Offenlegen von PostHog-Produktendpunkten als MCP-Tools. Verwenden Sie es, wenn Sie neue API-Endpunkte erstellen oder aktualisieren, MCP-Tool-Definitionen hinzufügen, YAML-Konfigurationen gerüsten oder Serialisierer mit guten Beschreibungen schreiben. Deckt die gesamte Pipeline vom Django-Serializer bis zum generierten TypeScript-Tool-Handler ab.
