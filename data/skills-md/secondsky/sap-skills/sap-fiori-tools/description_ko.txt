VS Code 및 SAP Business Application Studio용 SAP Fiori 도구 확장을 사용하여 SAP Fiori 애플리케이션을 개발합니다.
사용 시기: Fiori Elements 또는 Freestyle SAPUI5 애플리케이션 생성, 목록 보고서용 페이지 편집기 구성
또는 개체 페이지, 주석 및 서비스 모델러 작업, ABAP 또는 Cloud Foundry에 배포 설정,
적응 프로젝트 생성, 안내 개발 사용, 모의 데이터 또는 라이브 데이터로 미리 보기, 구성
SAP Fiori 런치패드 또는 Project Accelerator/Joule과 함께 AI 기반 생성을 사용합니다.
기술: SAP Fiori Elements, SAPUI5, OData V2/V4, CAP, SAP BTP, ABAP, Cloud Foundry, fiori-mcp-server(AI 지원 생성을 위한 MCP 도구).
