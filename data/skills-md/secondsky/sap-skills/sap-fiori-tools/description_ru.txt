Разрабатывает приложения SAP Fiori с использованием расширений инструментов SAP Fiori для VS Code и SAP Business Application Studio.
Используйте при: создании приложений Fiori Elements или Freestyle SAPUI5, настройке редактора страниц для отчета по списку.
или Страница объекта, работа с аннотациями и Service Modeler, настройка развертывания в ABAP или Cloud Foundry,
создание проектов адаптации, использование управляемой разработки, предварительный просмотр с использованием макетов или реальных данных, настройка
Панель запуска SAP Fiori или использование генерации на основе искусственного интеллекта с помощью Project Accelerator/Joule.
Технологии: SAP Fiori Elements, SAPUI5, OData V2/V4, CAP, SAP BTP, ABAP, Cloud Foundry, fiori-mcp-server (инструменты MCP для генерации с помощью искусственного интеллекта).
