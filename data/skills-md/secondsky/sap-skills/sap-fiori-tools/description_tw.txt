使用 VS Code 和 SAP Business Application Studio 的 SAP Fiori 工具擴充開發 SAP Fiori 應用程式。
使用場合：產生 Fiori Elements 或 Freestyle SAPUI5 應用程式、為清單報表設定頁面編輯器
或物件頁面，使用註解和服務建模器，設定到 ABAP 或 Cloud Foundry 的部署，
建立適應項目，使用引導開發，使用模擬數據或即時數據預覽，配置
SAP Fiori 啟動板，或使用 AI 驅動的產生與 Project Accelerator/Joule。
技術：SAP Fiori Elements、SAPUI5、OData V2/V4、CAP、SAP BTP、ABAP、Cloud Foundry、fiori-mcp-server（用於 AI 輔助產生的 MCP 工具）。
