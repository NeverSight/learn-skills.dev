使用 VS Code 和 SAP Business Application Studio 的 SAP Fiori 工具扩展开发 SAP Fiori 应用程序。
使用场合：生成 Fiori Elements 或 Freestyle SAPUI5 应用程序、为列表报表配置页面编辑器
或对象页面，使用注释和服务建模器，设置到 ABAP 或 Cloud Foundry 的部署，
创建适应项目，使用引导开发，使用模拟数据或实时数据预览，配置
SAP Fiori 启动板，或使用 AI 驱动的生成与 Project Accelerator/Joule。
技术：SAP Fiori Elements、SAPUI5、OData V2/V4、CAP、SAP BTP、ABAP、Cloud Foundry、fiori-mcp-server（用于 AI 辅助生成的 MCP 工具）。
