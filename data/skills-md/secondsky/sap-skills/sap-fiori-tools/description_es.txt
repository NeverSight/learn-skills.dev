Desarrolla aplicaciones SAP Fiori utilizando extensiones de herramientas SAP Fiori para VS Code y SAP Business Application Studio.
Úselo cuando: genere aplicaciones Fiori Elements o Freestyle SAPUI5, configure el editor de páginas para el informe de lista
o página de objetos, trabajar con anotaciones y Service Modeler, configurar la implementación en ABAP o Cloud Foundry,
crear proyectos de adaptación, usar desarrollo guiado, obtener una vista previa con datos simulados o datos en vivo, configurar
Plataforma de lanzamiento de SAP Fiori o uso de generación impulsada por IA con Project Accelerator/Joule.
Tecnologías: SAP Fiori Elements, SAPUI5, OData V2/V4, CAP, SAP BTP, ABAP, Cloud Foundry, fiori-mcp-server (herramientas MCP para generación asistida por IA).
