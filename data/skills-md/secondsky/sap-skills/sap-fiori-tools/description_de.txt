Entwickelt SAP-Fiori-Anwendungen mithilfe von SAP-Fiori-Tools-Erweiterungen für VS Code und SAP Business Application Studio.
Verwendung bei: Generieren von Fiori Elements- oder Freestyle SAPUI5-Anwendungen, Konfigurieren des Seiteneditors für Listenberichte
oder Objektseite, Arbeiten mit Annotationen und Service Modeler, Einrichten der Bereitstellung für ABAP oder Cloud Foundry,
Anpassungsprojekte erstellen, Guided Development verwenden, Vorschau mit Scheindaten oder Live-Daten anzeigen, konfigurieren
SAP Fiori Launchpad oder Nutzung der KI-gestützten Generierung mit Project Accelerator/Joule.
Technologien: SAP Fiori Elements, SAPUI5, OData V2/V4, CAP, SAP BTP, ABAP, Cloud Foundry, fiori-mcp-server (MCP-Tools für KI-gestützte Generierung).
