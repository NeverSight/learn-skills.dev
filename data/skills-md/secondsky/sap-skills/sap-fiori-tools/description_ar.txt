تطوير تطبيقات SAP Fiori باستخدام ملحقات أدوات SAP Fiori لـ VS Code وSAP Business Application Studio.
يُستخدم عندما: إنشاء تطبيقات Fiori Elements أو Freestyle SAPUI5، وتكوين محرر الصفحة لتقرير القائمة
أو صفحة الكائن، والعمل مع التعليقات التوضيحية ومصمم الخدمة، وإعداد النشر إلى ABAP أو Cloud Foundry،
إنشاء مشاريع التكيف، باستخدام التطوير الموجه، والمعاينة باستخدام بيانات وهمية أو بيانات حية، والتكوين
لوحة تشغيل SAP Fiori، أو استخدام الجيل المدعوم بالذكاء الاصطناعي مع Project Accelerator/Joule.
التقنيات: SAP Fiori Elements، وSAPUI5، وOData V2/V4، وCAP، وSAP BTP، وABAP، وCloud Foundry، وfiori-mcp-server (أدوات MCP للتوليد بمساعدة الذكاء الاصطناعي).
