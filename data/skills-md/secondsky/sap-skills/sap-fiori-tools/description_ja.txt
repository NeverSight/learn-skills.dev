VS Code および SAP Business Application Studio 用の SAP Fiori ツール拡張機能を使用して、SAP Fiori アプリケーションを開発します。
次の場合に使用します: Fiori Elements または Freestyle SAPUI5 アプリケーションの生成、リスト レポート用のページ エディターの構成
またはオブジェクト ページ、アノテーションと Service Modeler の操作、ABAP または Cloud Foundry へのデプロイメントのセットアップ、
適応プロジェクトの作成、ガイド付き開発の使用、モックデータまたはライブデータによるプレビュー、構成
SAP Fiori ラウンチパッド、または Project Accelerator/Joule による AI を利用した生成を使用します。
テクノロジー: SAP Fiori Elements、SAPUI5、OData V2/V4、CAP、SAP BTP、ABAP、Cloud Foundry、fiori-mcp-server (AI 支援生成用の MCP ツール)。
