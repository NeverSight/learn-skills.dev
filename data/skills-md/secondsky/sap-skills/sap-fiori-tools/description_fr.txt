Développe des applications SAP Fiori à l'aide des extensions d'outils SAP Fiori pour VS Code et SAP Business Application Studio.
À utiliser lorsque : génération d'applications Fiori Elements ou Freestyle SAPUI5, configuration de l'éditeur de page pour le rapport de liste
ou une page d'objet, travaillant avec des annotations et Service Modeler, configurant le déploiement sur ABAP ou Cloud Foundry,
créer des projets d'adaptation, utiliser le développement guidé, prévisualiser avec des données fictives ou des données en direct, configurer
La barre de lancement SAP Fiori ou l'utilisation de la génération basée sur l'IA avec Project Accelerator/Joule.
Technologies : SAP Fiori Elements, SAPUI5, OData V2/V4, CAP, SAP BTP, ABAP, Cloud Foundry, fiori-mcp-server (outils MCP pour la génération assistée par IA).
