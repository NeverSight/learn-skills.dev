Sviluppa applicazioni SAP Fiori utilizzando le estensioni degli strumenti SAP Fiori per VS Code e SAP Business Application Studio.
Utilizzare quando: si generano applicazioni Fiori Elements o Freestyle SAPUI5, si configura Page Editor per List Report
o Pagina oggetto, lavorare con annotazioni e Service Modeler, impostare la distribuzione su ABAP o Cloud Foundry,
creazione di progetti di adattamento, utilizzo dello sviluppo guidato, anteprima con dati simulati o dati live, configurazione
SAP Fiori Launchpad o utilizzando la generazione basata sull'intelligenza artificiale con Project Accelerator/Joule.
Tecnologie: SAP Fiori Elements, SAPUI5, OData V2/V4, CAP, SAP BTP, ABAP, Cloud Foundry, fiori-mcp-server (strumenti MCP per la generazione assistita da AI).
