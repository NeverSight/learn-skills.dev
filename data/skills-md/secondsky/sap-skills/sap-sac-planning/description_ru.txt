Этот навык следует использовать при разработке приложений планирования SAP Analytics Cloud (SAC), включая создание историй с поддержкой планирования, приложений дизайнера аналитики с функциями планирования, действий с данными, множественных действий, управления версиями и рабочих процессов планирования. Используйте при создании моделей планирования, реализации форм ввода данных, настройке распространения/распределения/распределения, настройке блокировки данных, построении процессов планирования на основе календаря с рабочими процессами утверждения, написании сценариев JavaScript для автоматизации планирования, использовании API getPlanning(), API PlanningModel или API DataSource для сценариев планирования, устранении проблем с производительностью планирования, интеграции прогнозного прогнозирования в рабочие процессы планирования, реализации бесшовного планирования с SAP Datasphere, настройке живых подключений BPC для BW на Интеграция HANA, построение деревьев стоимостных драйверов для анализа «что если» или отладка действий с данными с помощью трассировки.
