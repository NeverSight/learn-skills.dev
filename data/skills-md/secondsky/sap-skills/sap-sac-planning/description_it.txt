Questa competenza dovrebbe essere utilizzata durante lo sviluppo di applicazioni di pianificazione SAP Analytics Cloud (SAC), inclusa la creazione di storie abilitate alla pianificazione, applicazioni di progettazione di analisi con funzionalità di pianificazione, azioni dati, azioni multiple, gestione delle versioni e flussi di lavoro di pianificazione. Utilizzare durante la creazione di modelli di pianificazione, l'implementazione di moduli di immissione dati, la configurazione di diffusione/distribuzione/allocazione, l'impostazione del blocco dei dati, la creazione di processi di pianificazione basati su calendario con flussi di lavoro di approvazione, la scrittura di script JavaScript per l'automazione della pianificazione, l'utilizzo dell'API getPlanning(), l'API PlanningModel o l'API DataSource per scenari di pianificazione, la risoluzione dei problemi di prestazioni di pianificazione, l'integrazione della previsione predittiva nei flussi di lavoro di pianificazione, l'implementazione di Seamless Planning con SAP Datasphere, la configurazione di connessioni live BPC per l'integrazione di BW su HANA, la creazione di alberi di driver di valore per analisi what-if o debug di azioni sui dati con tracciamento.
