Esta habilidad se debe utilizar al desarrollar aplicaciones de planificación de SAP Analytics Cloud (SAC), incluida la creación de historias habilitadas para la planificación, aplicaciones de diseño de análisis con funcionalidad de planificación, acciones de datos, acciones múltiples, gestión de versiones y flujos de trabajo de planificación. Úselo al crear modelos de planificación, implementar formularios de entrada de datos, configurar la distribución/distribución/asignación, configurar el bloqueo de datos, crear procesos de planificación basados en calendario con flujos de trabajo de aprobación, escribir scripts JavaScript para la automatización de la planificación, usar la API getPlanning(), la API PlanningModel o la API DataSource para escenarios de planificación, solucionar problemas de rendimiento de la planificación, integrar pronósticos predictivos en flujos de trabajo de planificación, implementar Seamless Planning con SAP Datasphere, configurar conexiones en vivo de BPC para BW en la integración de HANA, crear árboles de impulsores de valor para qué pasaría si análisis o depuración de acciones de datos con seguimiento.
