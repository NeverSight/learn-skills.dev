يجب استخدام هذه المهارة عند تطوير تطبيقات التخطيط SAP Analytics Cloud (SAC)، بما في ذلك إنشاء قصص ممكّنة للتخطيط، وتطبيقات مصمم التحليلات مع وظائف التخطيط، وإجراءات البيانات، والإجراءات المتعددة، وإدارة الإصدارات، وتخطيط سير العمل. يُستخدم عند إنشاء نماذج التخطيط، وتنفيذ نماذج إدخال البيانات، وتكوين النشر/التوزيع/التخصيص، وإعداد قفل البيانات، وبناء عمليات التخطيط المستندة إلى التقويم مع سير عمل الموافقة، وكتابة برامج JavaScript النصية لأتمتة التخطيط، واستخدام getPlanning() API، أو PlanningModel API، أو DataSource API لسيناريوهات التخطيط، واستكشاف مشكلات أداء التخطيط وإصلاحها، ودمج التنبؤ التنبؤي في تخطيط سير العمل، وتنفيذ التخطيط السلس مع SAP Datasphere، وتكوين اتصالات BPC المباشرة لـ BW على تكامل HANA، أو بناء أشجار محركات القيمة لتحليل ماذا لو، أو تصحيح أخطاء إجراءات البيانات من خلال التتبع.
