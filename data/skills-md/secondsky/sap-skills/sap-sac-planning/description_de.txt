Diese Fähigkeit sollte bei der Entwicklung von SAP Analytics Cloud (SAC)-Planungsanwendungen verwendet werden, einschließlich der Erstellung planungsfähiger Storys, Analytics-Designer-Anwendungen mit Planungsfunktionalität, Datenaktionen, Multiaktionen, Versionsverwaltung und Planungsworkflows. Verwendung beim Erstellen von Planungsmodellen, Implementieren von Dateneingabeformularen, Konfigurieren von Verteilung/Verteilung/Zuordnung, Einrichten von Datensperren, Erstellen kalenderbasierter Planungsprozesse mit Genehmigungsworkflows, Schreiben von JavaScript-Skripts für die Planungsautomatisierung, Verwenden der getPlanning()-API, PlanningModel-API oder DataSource-API für Planungsszenarien, Beheben von Planungsleistungsproblemen, Integrieren von Vorhersageprognosen in Planungsworkflows, Implementieren von Seamless Planning mit SAP Datasphere, Konfigurieren von BPC-Live-Verbindungen für die BW-on-HANA-Integration, Erstellen von Werttreiberbäumen für Was-wäre-wenn-Analyse oder Debuggen von Datenaktionen mit Ablaufverfolgung.
