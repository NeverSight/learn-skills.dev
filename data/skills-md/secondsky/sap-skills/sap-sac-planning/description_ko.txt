이 기술은 계획 지원 스토리 구축, 계획 기능이 포함된 분석 디자이너 애플리케이션, 데이터 작업, 다중 작업, 버전 관리, 계획 워크플로 등 SAP Analytics Cloud(SAC) 계획 애플리케이션을 개발할 때 사용해야 합니다. 계획 모델 생성, 데이터 입력 양식 구현, 분산/배포/할당 구성, 데이터 잠금 설정, 승인 워크플로를 사용하여 달력 기반 계획 프로세스 구축, 계획 자동화를 위한 JavaScript 스크립트 작성, 계획 시나리오를 위한 getPlanning() API, PlanningModel API 또는 DataSource API 사용, 계획 성능 문제 해결, 예측 예측을 계획 워크플로에 통합, SAP Datasphere를 통한 원활한 계획 구현, BW on HANA 통합을 위한 BPC 라이브 연결 구성, what-if 분석을 위한 가치 동인 트리 구축 또는 디버깅 시 사용합니다. 추적을 통한 데이터 작업.
