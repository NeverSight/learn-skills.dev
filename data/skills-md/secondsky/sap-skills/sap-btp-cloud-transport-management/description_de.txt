Umfassende Kenntnisse für den SAP Cloud Transport Management Service auf SAP BTP. Verwendung beim Einrichten von Transportlandschaften, Konfigurieren von Transportknoten und -routen, Verwalten von Importwarteschlangen, Bereitstellen von MTAs in Cloud Foundry-Umgebungen, Integrieren in CI/CD-Pipelines, Konfigurieren von ABAP-Umgebungstransporten, Beheben von Bereitstellungsfehlern oder Implementieren von Änderungsmanagement-Workflows. Deckt Berechtigungen, Abonnements, Rollensammlungen, Serviceinstanzen, Ziele und API-Integrationen ab.
