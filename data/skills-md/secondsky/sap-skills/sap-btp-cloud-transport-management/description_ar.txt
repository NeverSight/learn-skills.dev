مهارة شاملة لخدمة SAP Cloud Transport Management على SAP BTP. يُستخدم عند إعداد مناظر النقل الطبيعية، أو تكوين عقد النقل والمسارات، أو إدارة قوائم انتظار الاستيراد، أو نشر MTAs عبر بيئات Cloud Foundry، أو التكامل مع خطوط أنابيب CI/CD، أو تكوين عمليات نقل بيئة ABAP، أو استكشاف أخطاء النشر وإصلاحها، أو تنفيذ سير عمل إدارة التغيير. يغطي الاستحقاقات والاشتراكات ومجموعات الأدوار ومثيلات الخدمة والوجهات وتكاملات واجهة برمجة التطبيقات.
