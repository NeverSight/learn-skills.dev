Questa competenza fornisce una guida completa per il servizio SAP Cloud Logging su SAP BTP.
Da utilizzare durante la configurazione delle istanze Cloud Logging, configurando l'importazione di log da Cloud Foundry
o runtime Kyma, implementando l'osservabilità di OpenTelemetry, analizzando log/metriche/tracce
in OpenSearch Dashboards, configurazione dell'autenticazione SAML, gestione dei certificati o
risoluzione dei problemi di importazione. Copre i piani di servizio (sviluppo/standard/grandi), tutti e 4
metodi di creazione dell'istanza (BTP Cockpit, CF CLI, BTP CLI, Service Operator), tutti 4
metodi di acquisizione (Cloud Foundry, Kyma, OpenTelemetry, API JSON) e best practice per la sicurezza.
