このスキルは、SAP BTP 上の SAP Cloud Logging サービスに関する包括的なガイダンスを提供します。
Cloud Logging インスタンスのセットアップ時、Cloud Foundry からのログ取り込みの構成時に使用します。
または Kyma ランタイム、OpenTelemetry オブザーバビリティの実装、ログ/メトリクス/トレースの分析
OpenSearch ダッシュボードで、SAML 認証の構成、証明書の管理、または
取り込みの問題のトラブルシューティング。サービス プラン (開発/標準/大規模) の 4 つすべてをカバー
インスタンス作成方法 (BTP Cockpit、CF CLI、BTP CLI、Service Operator)、すべて 4
取り込み方法 (Cloud Foundry、Kyma、OpenTelemetry、JSON API)、およびセキュリティのベスト プラクティス。
