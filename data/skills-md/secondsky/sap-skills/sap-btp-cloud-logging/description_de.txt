Dieser Skill bietet umfassende Anleitung für den SAP Cloud Logging-Service auf SAP BTP.
Verwenden Sie diese Option, wenn Sie Cloud Logging-Instanzen einrichten und die Protokollaufnahme von Cloud Foundry konfigurieren
oder Kyma-Laufzeiten, Implementierung der OpenTelemetry-Beobachtbarkeit, Analyse von Protokollen/Metriken/Traces
in OpenSearch-Dashboards, SAML-Authentifizierung konfigurieren, Zertifikate verwalten oder
Fehlerbehebung bei Aufnahmeproblemen. Deckt alle 4 Servicepläne (Dev/Standard/Large) ab
Instanzerstellungsmethoden (BTP Cockpit, CF CLI, BTP CLI, Service Operator), alle 4
Aufnahmemethoden (Cloud Foundry, Kyma, OpenTelemetry, JSON API) und Best Practices für die Sicherheit.
