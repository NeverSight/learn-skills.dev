Esta habilidad proporciona una guía integral para el servicio SAP Cloud Logging en SAP BTP.
Úselo al configurar instancias de Cloud Logging o configurar la ingesta de registros desde Cloud Foundry
o tiempos de ejecución de Kyma, implementando la observabilidad de OpenTelemetry, analizando registros/métricas/rastreos
en OpenSearch Dashboards, configurar la autenticación SAML, administrar certificados o
solucionar problemas de ingestión. Cubre planes de servicio (desarrollador/estándar/grande), los 4
métodos de creación de instancias (BTP Cockpit, CF CLI, BTP CLI, Service Operador), los 4
métodos de ingesta (Cloud Foundry, Kyma, OpenTelemetry, JSON API) y mejores prácticas de seguridad.
