Cette compétence fournit des conseils complets pour le service SAP Cloud Logging sur SAP BTP.
À utiliser lors de la configuration des instances Cloud Logging et de la configuration de l'ingestion des journaux à partir de Cloud Foundry
ou les runtimes Kyma, implémentant l'observabilité OpenTelemetry, analysant les journaux/métriques/traces
dans les tableaux de bord OpenSearch, la configuration de l'authentification SAML, la gestion des certificats ou
dépannage des problèmes d’ingestion. Couvre les plans de service (dev/standard/large), tous les 4
méthodes de création d'instances (BTP Cockpit, CF CLI, BTP CLI, Service Operator), toutes les 4
méthodes d'ingestion (Cloud Foundry, Kyma, OpenTelemetry, API JSON) et bonnes pratiques de sécurité.
