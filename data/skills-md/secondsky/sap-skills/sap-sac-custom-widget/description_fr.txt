Compétence en développement de widgets personnalisés SAP Analytics Cloud (SAC). À utiliser lors de la création de visualisations personnalisées, de composants interactifs, de l'extension de SAC avec des composants Web ou de la création de modules complémentaires de widget pour personnaliser les widgets intégrés. Couvre la configuration des métadonnées JSON, les composants Web JavaScript, les fonctions de cycle de vie, la liaison de données avec les flux, les panneaux de style, les panneaux de création, les définitions de propriétés/événements/méthodes, les types personnalisés, les types de données API de script, l'intégration de bibliothèques tierces, les options d'hébergement, la sécurité, l'optimisation des performances et le débogage. Inclut la fonctionnalité Widget Add-On (QRC Q4 2023+) pour étendre les widgets intégrés sans créer à partir de zéro. Fournit des modèles pour les widgets de base, les graphiques liés aux données, les panneaux de style et les cartes KPI. Prend en charge l'expérience d'histoire optimisée et le concepteur d'analyses. Empêche les erreurs courantes : fonctions de cycle de vie manquantes, schéma JSON incorrect, avertissements d'intégrité, échecs CORS, incompatibilités de types de propriétés, problèmes de liaison de données et anti-modèles de performances.
