مهارة تطوير عناصر واجهة المستخدم المخصصة لـ SAP Analytics Cloud (SAC). يُستخدم عند إنشاء تصورات مخصصة، أو مكونات تفاعلية، أو توسيع SAC بمكونات الويب، أو إنشاء وظائف إضافية لعناصر واجهة المستخدم لتخصيص عناصر واجهة المستخدم المضمنة. يغطي تكوين بيانات تعريف JSON، ومكونات الويب JavaScript، ووظائف دورة الحياة، وربط البيانات بالخلاصات، ولوحات التصميم، ولوحات الإنشاء، وتعريفات الخاصية/الحدث/الأسلوب، والأنواع المخصصة، وأنواع بيانات واجهة برمجة التطبيقات النصية، وتكامل مكتبة الطرف الثالث، وخيارات الاستضافة، والأمان، وتحسين الأداء، وتصحيح الأخطاء. يتضمن ميزة Widget Add-On (QRC Q4 2023+) لتوسيع الأدوات المدمجة دون الإنشاء من الصفر. يوفر قوالب لعناصر واجهة المستخدم الأساسية والمخططات المرتبطة بالبيانات ولوحات التصميم وبطاقات مؤشرات الأداء الرئيسية. يدعم تجربة القصة المحسنة ومصمم التحليلات. يمنع الأخطاء الشائعة: فقدان وظائف دورة الحياة، ومخطط JSON غير الصحيح، وتحذيرات التكامل، وفشل CORS، وعدم تطابق نوع الخاصية، ومشكلات ربط البيانات، وأنماط الأداء المضادة.

الكلمات الرئيسية: سحابة تحليلات sap، عنصر واجهة المستخدم المخصص sac، تطوير عنصر واجهة المستخدم المخصص، كيس مكون الويب، عنصر واجهة مستخدم بيانات تعريف json، وظائف دورة حياة عنصر واجهة المستخدم، onCustomWidgetBeforeUpdate، onCustomWidgetAfterUpdate، onCustomWidgetResize، onCustomWidgetDestroy، ربط بيانات الكيس، ربط بيانات عنصر واجهة المستخدم، خلاصات dataBindings، getDataBinding، getResultSet، عنصر واجهة مستخدم لوحة التصميم، عنصر واجهة مستخدم لوحة الإنشاء، أساليب أحداث خصائص عنصر واجهة المستخدم، حدث PropertiesChanged، عنصر واجهة المستخدم المخصص لـ SubmitEvent، تكامل sac echarts، تكامل sac d3js، كيس مكتبة الطرف الثالث، كيس استضافة عناصر واجهة المستخدم، عنصر واجهة مستخدم sac المستضاف، عنصر واجهة مستخدم تجزئة التكامل، تكامل sha256، عناصر أمان عناصر واجهة المستخدم، تصحيح أخطاء عناصر واجهة المستخدم، كيس التصور المخصص، أداة مصمم تحليلات sac، أداة تجربة القصة المحسنة، واجهة برمجة التطبيقات لأداة sac، الوظيفة الإضافية لعنصر واجهة المستخدم، أداة واجهة برمجة تطبيقات البرنامج النصي sac، تعداد الأنواع المخصصة، اختيار MemberInfo ResultMemberInfo، مسؤول تثبيت القطعة، تحسين أداء sac، مكون ويب الظل dom، تخصيص تلميح أداة sac، ملحق منطقة قطعة الأرض
