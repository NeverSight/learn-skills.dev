Competenza nello sviluppo di widget personalizzati SAP Analytics Cloud (SAC). Da utilizzare quando si creano visualizzazioni personalizzate, componenti interattivi, si estende SAC con componenti Web o si creano componenti aggiuntivi widget per personalizzare i widget integrati. Copre la configurazione dei metadati JSON, i componenti Web JavaScript, le funzioni del ciclo di vita, l'associazione dati con feed, pannelli di stile, pannelli di creazione, definizioni di proprietà/eventi/metodi, tipi personalizzati, tipi di dati API script, integrazione di librerie di terze parti, opzioni di hosting, sicurezza, ottimizzazione delle prestazioni e debug. Include la funzionalità Widget Add-On (QRC Q4 2023+) per estendere i widget integrati senza crearli da zero. Fornisce modelli per widget di base, grafici associati a dati, pannelli di stile e schede KPI. Supporta l'esperienza della storia ottimizzata e il designer di analisi. Previene gli errori comuni: funzioni del ciclo di vita mancanti, schema JSON errato, avvisi di integrità, errori CORS, mancata corrispondenza dei tipi di proprietà, problemi di associazione dei dati e anti-pattern delle prestazioni.
