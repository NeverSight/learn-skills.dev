Навыки разработки пользовательских виджетов SAP Analytics Cloud (SAC). Используйте при создании пользовательских визуализаций, интерактивных компонентов, расширении SAC веб-компонентами или создании надстроек виджетов для настройки встроенных виджетов. Охватывает конфигурацию метаданных JSON, веб-компоненты JavaScript, функции жизненного цикла, привязку данных к каналам, панели стилей, панели компоновщика, определения свойств/событий/методов, пользовательские типы, типы данных API сценариев, интеграцию сторонних библиотек, параметры хостинга, безопасность, оптимизацию производительности и отладку. Включает функцию надстройки виджетов (QRC Q4 2023+) для расширения встроенных виджетов без создания с нуля. Предоставляет шаблоны для базовых виджетов, диаграмм с привязкой к данным, панелей стилей и карточек ключевых показателей эффективности. Поддерживает Optimized Story Experience и Analytics Designer. Предотвращает распространенные ошибки: отсутствие функций жизненного цикла, неправильная схема JSON, предупреждения целостности, сбои CORS, несоответствие типов свойств, проблемы привязки данных и антишаблоны производительности.
