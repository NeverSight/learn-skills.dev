SAP Analytics Cloud (SAC) カスタム ウィジェット開発スキル。カスタム ビジュアライゼーション、対話型コンポーネントの構築、Web コンポーネントによる SAC の拡張、または組み込みウィジェットをカスタマイズするためのウィジェット アドオンの作成時に使用します。 JSON メタデータ構成、JavaScript Web コンポーネント、ライフサイクル関数、フィードとのデータ バインディング、スタイル パネル、ビルダー パネル、プロパティ/イベント/メソッド定義、カスタム タイプ、スクリプト API データ タイプ、サードパーティ ライブラリの統合、ホスティング オプション、セキュリティ、パフォーマンスの最適化、デバッグについて説明します。組み込みウィジェットを最初から作成せずに拡張できるウィジェット アドオン機能 (QRC Q4 2023+) が含まれています。基本的なウィジェット、データバインドされたグラフ、スタイル パネル、KPI カードのテンプレートを提供します。最適化されたストーリー エクスペリエンスと分析デザイナーをサポートします。ライフサイクル関数の欠落、不正な JSON スキーマ、整合性警告、CORS エラー、プロパティ タイプの不一致、データ バインディングの問題、パフォーマンスのアンチパターンなどの一般的なエラーを防止します。
