Этот навык обеспечивает комплексные знания для SAP Service Manager на платформе бизнес-технологий SAP (BTP). Его следует использовать при управлении экземплярами служб, привязками, брокерами и платформами в Cloud Foundry, Kyma, Kubernetes и других средах. Используйте при предоставлении услуг через SMCTL CLI, BTP CLI или API-интерфейсы REST, настройке аутентификации OAuth2, работе с оператором службы SAP BTP в Kubernetes, устранении проблем с потреблением служб или реализации межсредового управления службами.

Ключевые слова: SAP Service Manager, BTP, экземпляры сервисов, привязки сервисов, SMCTL, сервисный брокер, OSBAPI, Cloud Foundry, Kyma, Kubernetes, сервис-менеджер, сервис-оператор-доступ, субаккаунт-администратор, OAuth2, X.509, рынок услуг, планы обслуживания, ограничение скорости, cf create-service, btp create Services/instance, ServiceInstance CRD, ServiceBinding CRD
