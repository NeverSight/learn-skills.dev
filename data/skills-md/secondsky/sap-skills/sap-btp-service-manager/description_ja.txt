このスキルは、SAP Business Technology Platform (BTP) 上の SAP Service Manager に関する包括的な知識を提供します。これは、Cloud Foundry、Kyma、Kubernetes、その他の環境全体でサービス インスタンス、バインディング、ブローカー、プラットフォームを管理するときに使用する必要があります。 SMCTL CLI、BTP CLI、または REST API を介したサービスのプロビジョニング、OAuth2 認証の構成、Kubernetes での SAP BTP Service Operator の操作、サービス消費の問題のトラブルシューティング、またはクロス環境サービス管理の実装を行う場合に使用します。
