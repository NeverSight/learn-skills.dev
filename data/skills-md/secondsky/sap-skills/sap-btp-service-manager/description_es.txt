Esta habilidad proporciona conocimientos integrales para SAP Service Manager en SAP Business Technology Platform (BTP). Debe usarse al administrar instancias de servicio, enlaces, corredores y plataformas en Cloud Foundry, Kyma, Kubernetes y otros entornos. Úselo al aprovisionar servicios a través de SMCTL CLI, BTP CLI o API REST, configurar la autenticación OAuth2, trabajar con el operador de servicios SAP BTP en Kubernetes, solucionar problemas de consumo de servicios o implementar la gestión de servicios entre entornos.

Palabras clave: SAP Service Manager, BTP, instancias de servicio, enlaces de servicios, SMCTL, intermediario de servicios, OSBAPI, Cloud Foundry, Kyma, Kubernetes, administrador de servicios, acceso de operador de servicios, administrador de subcuenta, OAuth2, X.509, mercado de servicios, planes de servicio, limitación de velocidad, cf create-service, btp create servicios/instancia, ServiceInstance CRD, ServiceBinding CRD
