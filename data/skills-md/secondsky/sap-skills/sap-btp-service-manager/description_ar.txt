توفر هذه المهارة معرفة شاملة لمدير خدمة SAP على منصة SAP Business Technology Platform (BTP). ويجب استخدامه عند إدارة مثيلات الخدمة والارتباطات والوسطاء والأنظمة الأساسية عبر Cloud Foundry وKyma وKubernetes والبيئات الأخرى. يُستخدم عند توفير الخدمات عبر SMCTL CLI أو BTP CLI أو REST APIs، أو تكوين مصادقة OAuth2، أو العمل مع مشغل خدمة SAP BTP في Kubernetes، أو استكشاف مشكلات استهلاك الخدمة وإصلاحها، أو تنفيذ إدارة الخدمات عبر البيئات.

الكلمات الرئيسية: مدير خدمة SAP، BTP، مثيلات الخدمة، روابط الخدمة، SMCTL، وسيط الخدمة، OSBAPI، Cloud Foundry، Kyma، Kubernetes، مدير الخدمة، الوصول إلى مشغل الخدمة، مسؤول الحساب الفرعي، OAuth2، X.509، سوق الخدمات، خطط الخدمة، تحديد المعدل، إنشاء خدمة cf، إنشاء خدمات/مثيل btp، ServiceInstance CRD، ServiceBinding CRD
