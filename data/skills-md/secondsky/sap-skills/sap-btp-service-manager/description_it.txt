Questa competenza fornisce una conoscenza completa per SAP Service Manager su SAP Business Technology Platform (BTP). Dovrebbe essere utilizzato durante la gestione di istanze di servizi, associazioni, broker e piattaforme in Cloud Foundry, Kyma, Kubernetes e altri ambienti. Da utilizzare durante il provisioning di servizi tramite CLI SMCTL, CLI BTP o API REST, configurazione dell'autenticazione OAuth2, collaborazione con SAP BTP Service Operator in Kubernetes, risoluzione dei problemi di consumo dei servizi o implementazione della gestione dei servizi tra ambienti.

Parole chiave: SAP Service Manager, BTP, istanze di servizio, associazioni di servizi, SMCTL, broker di servizi, OSBAPI, Cloud Foundry, Kyma, Kubernetes, gestore di servizi, accesso operatore di servizio, amministratore di account secondario, OAuth2, X.509, mercato di servizi, piani di servizio, limitazione di velocità, cf create-service, btp create services/instance, ServiceInstance CRD, ServiceBinding CRD
