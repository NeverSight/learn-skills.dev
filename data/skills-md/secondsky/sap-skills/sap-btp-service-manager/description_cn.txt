此技能为 SAP 业务技术平台 (BTP) 上的 SAP Service Manager 提供全面的知识。它应该在跨 Cloud Foundry、Kyma、Kubernetes 和其他环境管理服务实例、绑定、代理和平台时使用。在通过 SMCTL CLI、BTP CLI 或 REST API 配置服务、配置 OAuth2 身份验证、与 Kubernetes 中的 SAP BTP Service Operator 合作、解决服务消耗问题或实施跨环境服务管理时使用。
