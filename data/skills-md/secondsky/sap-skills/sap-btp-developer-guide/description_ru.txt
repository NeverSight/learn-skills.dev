Разрабатывает бизнес-приложения на платформе SAP Business Technology Platform (BTP) с использованием CAP (Node.js/Java) или ABAP Cloud. 

Используйте при: создании облачных приложений на SAP BTP, развертывании в среде выполнения Cloud Foundry или Kyma, интеграции с SAP HANA Cloud, внедрении пользовательского интерфейса SAP Fiori, подключении к удаленным системам SAP, создании мультитенантных приложений SaaS, расширении SAP S/4HANA или SuccessFactors, настройке конвейеров CI/CD, реализации наблюдаемости или следовании передовым практикам разработки SAP.

Ключевые слова: SAP BTP, Платформа бизнес-технологий, CAP, Модель программирования облачных приложений, ABAP Cloud, Cloud Foundry, Kyma, SAP HANA Cloud, SAP Fiori, SAPUI5, CI/CD, наблюдаемость, мультитенантность, SaaS, среда SAP BTP ABAP, SAP Business Application Studio, SAP Cloud SDK, SAP Integration Suite, SAP Event Mesh, SAP Connectivity Service, SAP Destination Service, XSUAA, OAuth, OpenID Connect, OData, CDS, Core Data Services, ABAP CDS, Модель программирования приложений ABAP RESTful, RAP, разработка ABAP, разработка SAP BTP
