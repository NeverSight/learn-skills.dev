SAP Analytics Cloud Analytics Designer および最適化されたストーリー エクスペリエンスのための包括的な SAC スクリプト スキル。このスキルは、ユーザーが「SAC スクリプトの作成」、「Analytics Designer のデバッグ」、「SAC パフォーマンスの最適化」、「SAC での操作の計画」、「SAC でのデータのフィルター」、「DataSource API の使用」、「チャート スクリプト」、「テーブル操作」、「SAC イベント ハンドラー」、「バージョン管理」、「データ ロック」を要求する場合、または SAC ウィジェット、計画モデル、または分析アプリケーションを操作する場合に使用する必要があります。
