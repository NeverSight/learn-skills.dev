Competenza SAP BTP Cloud Integration Automation Service (CIAS) per flussi di lavoro di integrazione guidata.
Utilizzare per: impostare abbonamenti CIAS, configurare destinazioni, assegnare ruoli (CIASIntegrationAdministrator, CIASIntegrationExpert, CIASIntegrationMonitor), pianificare scenari di integrazione, lavorare con le attività di Posta in arrivo, monitorare l'esecuzione di scenari, risolvere errori CIAS, creare istanze OAuth2, configurare provider di identità per CIAS, comprendere l'architettura di sicurezza CIAS o integrare prodotti SAP (S/4HANA, SuccessFactors, servizi BTP, SAP Costruire, IBP).
