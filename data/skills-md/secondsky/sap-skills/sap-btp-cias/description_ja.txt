ガイド付き統合ワークフローのための SAP BTP Cloud Integration Automation Service (CIAS) スキル。
次の場合に使用します: CIAS サブスクリプションの設定、宛先の構成、ロールの割り当て (CIASIntegrationAdministrator、CIASIntegrationExpert、CIASIntegrationMonitor)、統合シナリオの計画、受信トレイ タスクの操作、シナリオの実行の監視、CIAS エラーのトラブルシューティング、OAuth2 インスタンスの作成、CIAS の ID プロバイダーの構成、CIAS セキュリティ アーキテクチャの理解、または SAP 製品の統合 (S/4HANA、SuccessFactors、BTP サービス、 SAP ビルド、IBP)。
