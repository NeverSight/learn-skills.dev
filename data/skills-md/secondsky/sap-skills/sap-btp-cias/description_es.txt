Habilidad de SAP BTP Cloud Integration Automation Service (CIAS) para flujos de trabajo de integración guiados.
Úselo cuando: configurar suscripciones CIAS, configurar destinos, asignar roles (CIASIntegrationAdministrator, CIASIntegrationExpert, CIASIntegrationMonitor), planificar escenarios de integración, trabajar con tareas de Mi bandeja de entrada, monitorear la ejecución de escenarios, solucionar errores de CIAS, crear instancias OAuth2, configurar proveedores de identidad para CIAS, comprender la arquitectura de seguridad de CIAS o integrar productos SAP (S/4HANA, SuccessFactors, servicios BTP, Construcción SAP, IBP).
