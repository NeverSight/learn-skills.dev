SAP BTP Cloud Integration Automation Service (CIAS)-Skill für geführte Integrationsworkflows.
Verwendung bei: Einrichten von CIAS-Abonnements, Konfigurieren von Zielen, Zuweisen von Rollen (CIASIntegrationAdministrator, CIASIntegrationExpert, CIASIntegrationMonitor), Planen von Integrationsszenarien, Arbeiten mit „Mein Posteingang“-Aufgaben, Überwachen der Szenarioausführung, Beheben von CIAS-Fehlern, Erstellen von OAuth2-Instanzen, Konfigurieren von Identitätsanbietern für CIAS, Verstehen der CIAS-Sicherheitsarchitektur oder Integrieren von SAP-Produkten (S/4HANA, SuccessFactors, BTP-Dienste, SAP Build, IBP).
