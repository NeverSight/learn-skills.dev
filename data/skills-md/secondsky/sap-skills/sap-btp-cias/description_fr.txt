Compétence SAP BTP Cloud Integration Automation Service (CIAS) pour les flux de travail d'intégration guidés.
À utiliser lorsque : la configuration d'abonnements CIAS, la configuration de destinations, l'attribution de rôles (CIASIntegrationAdministrator, CIASIntegrationExpert, CIASIntegrationMonitor), la planification de scénarios d'intégration, l'utilisation de tâches Ma boîte de réception, la surveillance de l'exécution de scénarios, le dépannage d'erreurs CIAS, la création d'instances OAuth2, la configuration de fournisseurs d'identité pour CIAS, la compréhension de l'architecture de sécurité CIAS ou l'intégration de produits SAP (S/4HANA, SuccessFactors, services BTP, SAP Build, PBI).
