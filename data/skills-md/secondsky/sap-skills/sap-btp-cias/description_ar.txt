مهارة SAP BTP Cloud Integration Automation Service (CIAS) لسير عمل التكامل الموجه.
يُستخدم عندما: إعداد اشتراكات CIAS، أو تكوين الوجهات، أو تعيين الأدوار (CIASIntegrationAdministrator، أو CIASIntegrationExpert، أو CIASIntegrationMonitor)، أو تخطيط سيناريوهات التكامل، أو العمل مع مهام علبة الوارد الخاصة بي، أو مراقبة تنفيذ السيناريو، أو استكشاف أخطاء CIAS وإصلاحها، أو إنشاء مثيلات OAuth2، أو تكوين موفري الهوية لـ CIAS، أو فهم بنية أمان CIAS، أو تكامل منتجات SAP (S/4HANA، SuccessFactors، خدمات BTP، SAP Build، IBP).
