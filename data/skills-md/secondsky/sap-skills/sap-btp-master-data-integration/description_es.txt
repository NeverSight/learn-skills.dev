Configura e integra el servicio SAP Master Data Integration (MDI) en SAP Business Technology Platform. Úselo al configurar inquilinos MDI, conectar aplicaciones (S/4HANA, SuccessFactors, Ariba, Fieldglass, etc.), configurar modelos de distribución, API SOAP para socios comerciales, extensibilidad o solucionar problemas de replicación de datos maestros. Cubre la integración del modelo de dominio único, la orquestación de datos empresariales, la autenticación de clientes (OAuth2, mTLS) y las configuraciones de seguridad.
