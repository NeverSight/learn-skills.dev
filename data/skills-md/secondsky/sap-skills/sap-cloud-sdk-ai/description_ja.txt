SAP Cloud SDK for AI を JavaScript/TypeScript および Java アプリケーションに統合します。 SAP AI コア、Generative AI Hub、または Orchestration Service を使用してアプリケーションを構築する場合に使用します。チャットの完了、埋め込み、ストリーミング、関数呼び出し、コンテンツ フィルタリング、データ マスキング、ドキュメント グラウンディング、プロンプト レジストリ、LangChain/Spring AI 統合をカバーします。 SAP BTP を介して、OpenAI GPT-4o、Llama、Gemini、Amazon Nova、およびその他の基盤モデルをサポートします。
