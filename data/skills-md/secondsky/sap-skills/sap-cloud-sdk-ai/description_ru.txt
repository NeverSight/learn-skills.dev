Интегрирует SAP Cloud SDK для AI в приложения JavaScript/TypeScript и Java. Используйте при создании приложений с помощью SAP AI Core, Generative AI Hub или Orchestration Service. Охватывает завершение чата, встраивание, потоковую передачу, вызов функций, фильтрацию контента, маскирование данных, обоснование документов, реестр подсказок и интеграцию LangChain/Spring AI. Поддерживает OpenAI GPT-4o, Llama, Gemini, Amazon Nova и другие базовые модели через SAP BTP.
