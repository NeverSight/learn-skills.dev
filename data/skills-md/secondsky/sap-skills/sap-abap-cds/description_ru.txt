Комплексный справочник SAP ABAP CDS (Core Data Services) для моделирования данных, разработки представлений и семантического обогащения. Используйте при создании представлений CDS или сущностей представлений в ABAP, определении моделей данных с аннотациями (@AbapCatalog, @AccessControl, @EndUserText, @Semantics, @UI, @Consumption, @ObjectModel), работе с ассоциациями и кардинальностью, реализации входных параметров, использовании встроенных функций (строковые, числовые, дата/время), написании CASE-выражений и условной логики, реализации контроля доступа с помощью DCL (язык управления данными), обработке. Типы данных CURR/QUAN со ссылочными полями, устранение ошибок CDS (SD_CDS_ENTITY105), запрос представлений CDS из ABAP или отображение данных с помощью SALV IDA. Охватывает ABAP 7.4+ через ABAP Cloud с проверенными на производстве шаблонами.
