Questa competenza fornisce una conoscenza completa della connettività SAP BTP, inclusi il servizio di destinazione, il servizio di connettività, il connettore cloud, il proxy di connettività e il proxy trasparente per Kubernetes. Dovrebbe essere utilizzato durante la configurazione delle destinazioni, l'impostazione della connettività cloud-on-premise, l'implementazione della propagazione principale, la distribuzione dei proxy di connettività negli ambienti Kubernetes/Kyma o la risoluzione dei problemi di connettività.

Utilizza questa abilità quando:
- Creazione o configurazione destinazioni SAP BTP (HTTP, RFC, LDAP, MAIL, TCP)
- Configurazione del connettore cloud per la connettività in sede
- Implementazione dei flussi di autenticazione OAuth per le destinazioni
- Configurazione della propagazione principale o della propagazione dell'utente
- Distribuzione del proxy di connettività o del proxy trasparente in Kubernetes
- Risoluzione dei problemi relativi agli errori di connettività (405, 407, 503)
- Impostazione della disponibilità elevata per Cloud Connector
- Configurazione multitenancy per destinazioni

Parole chiave: SAP BTP, connettività, servizio di destinazione, connettore cloud, proxy di connettività, proxy trasparente, Kyma, Kubernetes, OAuth, propagazione principale, RFC, LDAP, on-premise, connettività ibrida, canali di servizio, SOCKS5, proxy inverso, tunnel
