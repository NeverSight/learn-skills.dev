このスキルは、Destination Service、Connectivity Service、Cloud Connector、Connectivity Proxy、Kubernetes の透過プロキシなど、SAP BTP Connectivity に関する包括的な知識を提供します。これは、宛先の構成、クラウドからオンプレミスへの接続のセットアップ、プリンシパル伝播の実装、Kubernetes/Kyma 環境での接続プロキシの展開、または接続の問題のトラブルシューティングを行うときに使用する必要があります。

このスキルは次の場合に使用します。
- SAP BTP 宛先 (HTTP、RFC、LDAP、MAIL、TCP) の作成または構成
- オンプレミス接続用の Cloud Connector のセットアップ
- 宛先のOAuth認証フローの実装
- プリンシパル伝播またはユーザー伝播の構成
- Kubernetes での接続プロキシまたは透過プロキシのデプロイ
- 接続エラーのトラブルシューティング (405、407、503)
- Cloud Connector の高可用性のセットアップ
- 宛先のマルチテナンシーの構成

キーワード: SAP BTP、接続、宛先サービス、クラウド コネクタ、接続プロキシ、透過的プロキシ、Kyma、Kubernetes、OAuth、プリンシパル伝播、RFC、LDAP、オンプレミス、ハイブリッド接続、サービス チャネル、SOCKS5、リバース プロキシ、トンネル
