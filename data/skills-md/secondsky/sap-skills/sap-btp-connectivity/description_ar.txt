توفر هذه المهارة معرفة شاملة لـ SAP BTP Connectivity، بما في ذلك خدمة الوجهة، وخدمة الاتصال، وموصل السحابة، ووكيل الاتصال، والوكيل الشفاف لـ Kubernetes. ويجب استخدامه عند تكوين الوجهات، أو إعداد اتصال السحابة المحلية، أو تنفيذ النشر الأساسي، أو نشر وكلاء الاتصال في بيئات Kubernetes/Kyma، أو استكشاف مشكلات الاتصال وإصلاحها.

استخدم هذه المهارة عندما:
- إنشاء أو تكوين وجهات SAP BTP (HTTP، RFC، LDAP، MAIL، TCP)
- إعداد Cloud Connector للاتصال الداخلي
- تنفيذ تدفقات مصادقة OAuth للوجهات
- تكوين النشر الرئيسي أو نشر المستخدم
- نشر وكيل الاتصال أو الوكيل الشفاف في Kubernetes
- استكشاف أخطاء الاتصال وإصلاحها (405، 407، 503)
- إعداد التوفر العالي لـ Cloud Connector
- تكوين multitenancy للوجهات

الكلمات الرئيسية: SAP BTP، الاتصال، خدمة الوجهة، موصل السحابة، وكيل الاتصال، الوكيل الشفاف، Kyma، Kubernetes، OAuth، النشر الرئيسي، RFC، LDAP، محلي، اتصال هجين، قنوات الخدمة، SOCKS5، الوكيل العكسي، النفق
