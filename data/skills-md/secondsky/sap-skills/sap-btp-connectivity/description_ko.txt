이 기술은 대상 서비스, 연결 서비스, 클라우드 커넥터, 연결 프록시 및 Kubernetes용 투명 프록시를 포함하여 SAP BTP 연결에 대한 포괄적인 지식을 제공합니다. 대상 구성, 클라우드-온-프레미스 연결 설정, 주체 전파 구현, Kubernetes/Kyma 환경에 연결 프록시 배포 또는 연결 문제 해결 시 사용해야 합니다.

다음과 같은 경우에 이 기술을 사용하십시오.
- SAP BTP 대상(HTTP, RFC, LDAP, MAIL, TCP) 생성 또는 구성
- 온프레미스 연결을 위한 클라우드 커넥터 설정
- 대상에 대한 OAuth 인증 흐름 구현
- 주체 전파 또는 사용자 전파 구성
- Kubernetes에 연결 프록시 또는 투명 프록시 배포
- 연결 오류 문제 해결(405, 407, 503)
- Cloud Connector에 대한 고가용성 설정
- 대상에 대한 다중 테넌시 구성

키워드: SAP BTP, 연결, 대상 서비스, 클라우드 커넥터, 연결 프록시, 투명 프록시, Kyma, Kubernetes, OAuth, 주체 전파, RFC, LDAP, 온프레미스, 하이브리드 연결, 서비스 채널, SOCKS5, 역방향 프록시, 터널
