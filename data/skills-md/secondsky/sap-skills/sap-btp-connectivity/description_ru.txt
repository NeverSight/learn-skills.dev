Этот навык дает всесторонние знания о подключении SAP BTP, включая службу назначения, службу подключения, облачный соединитель, прокси-сервер подключения и прозрачный прокси-сервер для Kubernetes. Его следует использовать при настройке мест назначения, настройке подключения между облаком и локальной средой, реализации распространения принципала, развертывании прокси-серверов подключения в средах Kubernetes/Kyma или устранении проблем с подключением.

Используйте этот навык, когда:
- Создание или настройка назначений SAP BTP (HTTP, RFC, LDAP, MAIL, TCP)
- Настройка Cloud Connector для локального подключения.
- Реализация потоков аутентификации OAuth для пунктов назначения.
- Настройка распространения принципала или распространения пользователя.
— Развертывание прокси-сервера подключения или прозрачного прокси-сервера в Kubernetes.
- Устранение ошибок подключения (405, 407, 503)
- Настройка высокой доступности для Cloud Connector.
- Настройка мультиарендности для пунктов назначения.

Ключевые слова: SAP BTP, подключение, служба назначения, облачный соединитель, прокси-сервер подключения, прозрачный прокси, Kyma, Kubernetes, OAuth, основное распространение, RFC, LDAP, локально, гибридное подключение, каналы обслуживания, SOCKS5, обратный прокси-сервер, туннель.
