Unterstützt die SAP HANA Developer CLI (hana-cli) für die Datenbankentwicklung und -verwaltung.
Verwendung bei: Installation von Hana-Cli, Herstellen einer Verbindung zu SAP HANA-Datenbanken, Überprüfen von Datenbankobjekten
(Tabellen, Ansichten, Prozeduren, Funktionen), HDI-Container verwalten, SQL-Abfragen ausführen,
Konvertieren von Metadaten in CDS/EDMX/OpenAPI-Formate, Verwalten von SAP HANA Cloud-Instanzen,
Arbeiten mit der BTP-CLI-Integration oder Fehlerbehebung bei Hana-Cli-Befehlen.
Deckt ab: 91 Befehle, über 17 Ausgabeformate, HDI-Containerverwaltung, Cloud-Betrieb.
