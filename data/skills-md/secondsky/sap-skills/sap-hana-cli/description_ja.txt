データベースの開発と管理のために SAP HANA Developer CLI (hana-cli) を支援します。
使用場面: hana-cli のインストール、SAP HANA データベースへの接続、データベース オブジェクトの検査
(テーブル、ビュー、プロシージャ、関数)、HDI コンテナの管理、SQL クエリの実行、
メタデータの CDS/EDMX/OpenAPI 形式への変換、SAP HANA Cloud インスタンスの管理、
BTP CLI 統合の使用、または hana-cli コマンドのトラブルシューティング。
対象範囲: 91 個のコマンド、17 個以上の出力形式、HDI コンテナー管理、クラウド操作。
