Помогает использовать SAP HANA Developer CLI (hana-cli) для разработки и администрирования баз данных.
Используйте при: установке hana-cli, подключении к базам данных SAP HANA, проверке объектов базы данных.
(таблицы, представления, процедуры, функции), управление HDI-контейнерами, выполнение SQL-запросов,
преобразование метаданных в форматы CDS/EDMX/OpenAPI, управление экземплярами SAP HANA Cloud,
работа с интеграцией BTP CLI или устранение неполадок команд hana-cli.
Охватывает: 91 команду, более 17 форматов вывода, управление HDI-контейнерами, облачные операции.
