Aide SAP HANA Developer CLI (hana-cli) pour le développement et l'administration de bases de données.
À utiliser lors de : l'installation de hana-cli, la connexion aux bases de données SAP HANA, l'inspection des objets de base de données
(tables, vues, procédures, fonctions), gestion des conteneurs HDI, exécution de requêtes SQL,
conversion des métadonnées aux formats CDS/EDMX/OpenAPI, gestion des instances SAP HANA Cloud,
travailler avec l'intégration BTP CLI ou dépanner les commandes hana-cli.
Couvre : 91 commandes, plus de 17 formats de sortie, la gestion des conteneurs HDI, les opérations cloud.
