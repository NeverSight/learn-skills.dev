이 기술은 Code-OSS를 기반으로 구축된 SAP BTP의 클라우드 기반 IDE인 SAP Business Application Studio(BAS)에 대한 포괄적인 지침을 제공합니다. BAS 구독 설정, 개발 공간 생성, 외부 시스템에 연결, MTA 애플리케이션 배포, 연결 문제 해결, Git 리포지토리 관리, 런타임 버전 구성 또는 레이아웃 편집기 사용 시 사용합니다.
