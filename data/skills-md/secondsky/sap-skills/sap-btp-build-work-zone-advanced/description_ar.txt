تطوير وإدارة SAP Build Work Zone، الإصدار المتقدم من حلول أماكن العمل الرقمية. يُستخدم عند إنشاء مساحات عمل وصفحات عمل ومواقع تعاونية، وتطوير بطاقات تكامل واجهة المستخدم في SAP Business Application Studio، وإنشاء حزم المحتوى وقوالب مساحة العمل، والتكامل مع Microsoft 365/Teams/SharePoint/Google Drive، وتكوين برامج الدردشة الآلية وخطافات الويب، وتنفيذ توفير مستخدم SCIM API، وإعداد سجلات أعمال OData، وإدارة السمات والعلامات التجارية، وتكوين الوصول المستند إلى الدور والتسجيل الموحد (SSO)، واستكشاف مشكلات النشر وإصلاحها، أو العمل مع وحدة التحكم الإدارية.

الكلمات الرئيسية: الإصدار المتقدم من SAP Build Work Zone، مكان العمل الرقمي، بطاقات تكامل واجهة المستخدم، حزم المحتوى، قوالب مساحة العمل، SAP Business Application Studio، SAP Conversational AI، SCIM API، OData، تكامل Microsoft Teams، SSO، السمات، وحدة التحكم الإدارية
