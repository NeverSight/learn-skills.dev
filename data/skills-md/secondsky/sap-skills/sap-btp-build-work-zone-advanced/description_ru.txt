Разрабатывает и администрирует SAP Build Work Zone, расширенную версию решений для цифровых рабочих мест. Используйте при создании рабочих пространств, рабочих страниц и сайтов для совместной работы, разработке карт интеграции пользовательского интерфейса в SAP Business Application Studio, создании пакетов контента и шаблонов рабочих пространств, интеграции с Microsoft 365/Teams/SharePoint/Google Drive, настройке чат-ботов и веб-перехватчиков, реализации подготовки пользователей SCIM API, настройке бизнес-записей OData, управлении темами и брендингом, настройке ролевого доступа и единого входа, устранении проблем с развертыванием или работе с Консоль администрирования.

Ключевые слова: расширенная версия SAP Build Work Zone, цифровое рабочее место, карты интеграции пользовательского интерфейса, пакеты контента, шаблоны рабочих пространств, SAP Business Application Studio, SAP Conversational AI, SCIM API, OData, интеграция Microsoft Teams, SSO, темы, консоль администрирования.
