Desarrolla y administra SAP Build Work Zone, soluciones de lugar de trabajo digital de edición avanzada. Úselo al crear espacios de trabajo, páginas de trabajo y sitios colaborativos, desarrollar tarjetas de integración de UI en SAP Business Application Studio, crear paquetes de contenido y plantillas de espacios de trabajo, integrar con Microsoft 365/Teams/SharePoint/Google Drive, configurar chatbots y webhooks, implementar el aprovisionamiento de usuarios de API SCIM, configurar registros comerciales de OData, administrar temas y marcas, configurar el acceso basado en roles y SSO, solucionar problemas de implementación o trabajar con la Consola de administración.

Palabras clave: edición avanzada de SAP Build Work Zone, lugar de trabajo digital, tarjetas de integración de UI, paquetes de contenido, plantillas de espacio de trabajo, SAP Business Application Studio, SAP Conversational AI, SCIM API, OData, integración de Microsoft Teams, SSO, tematización, consola de administración
