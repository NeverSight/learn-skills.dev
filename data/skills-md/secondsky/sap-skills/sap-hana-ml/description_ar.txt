مهارة تطوير SAP HANA Machine Learning Python Client (hana-ml).

يُستخدم عندما: إنشاء حلول تعلم الآلة باستخدام التعلم الآلي داخل قاعدة بيانات SAP HANA
استخدام مكتبة Python hana-ml لخوارزميات PAL/APL وعمليات DataFrame،
AutoML واستمرارية النموذج والتصور.

الكلمات المفتاحية: hana-ml، SAP HANA، التعلم الآلي، PAL، APL، التحليلات التنبؤية،
HANA DataFrame، ConnectionContext، التصنيف، الانحدار، التجميع،
السلاسل الزمنية، ARIMA، تعزيز التدرج، AutoML، SHAP، تخزين النماذج
