此技能為遵循 SAP API 樣式指南標準記錄 SAP API 提供了全面的指導。
在建立或檢視 REST、OData、Java、JavaScript、.NET 或 C/C++ API 的 API 文件時應使用它。

此技能涵蓋命名約定、文件註釋、OpenAPI 規範、品質檢查表、棄用政策、
和手冊文件模板。它確保與 SAP API Business Hub 標準和行業最佳實踐的一致性。

關鍵字：SAP API、REST、OData、OpenAPI、Swagger、Javadoc、JSDoc、XML 文件、API Business Hub、API 命名、
API 棄用、x-sap-stateInfo、實體資料模型、EDM、文件標籤、API 品質、API 模板
