توفر هذه المهارة إرشادات شاملة لتوثيق واجهات برمجة تطبيقات SAP وفقًا لمعايير دليل أنماط SAP API.
ويجب استخدامه عند إنشاء أو مراجعة وثائق API لواجهات برمجة تطبيقات REST أو OData أو Java أو JavaScript أو .NET أو C/C++.

تغطي المهارة اصطلاحات التسمية، وتعليقات التوثيق، ومواصفات OpenAPI، وقوائم التحقق من الجودة، وسياسات الإهمال،
وقوالب التوثيق اليدوية. فهو يضمن الاتساق مع معايير SAP API Business Hub وأفضل ممارسات الصناعة.

الكلمات الرئيسية: SAP API، REST، OData، OpenAPI، Swagger، Javadoc، JSDoc، وثائق XML، API Business Hub، تسمية API،
إهمال واجهة برمجة التطبيقات (API)، x-sap-stateInfo، نموذج بيانات الكيان، EDM، علامات التوثيق، جودة واجهة برمجة التطبيقات (API)، قوالب واجهة برمجة التطبيقات (API)
