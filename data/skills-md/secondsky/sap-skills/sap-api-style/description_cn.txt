此技能为遵循 SAP API 样式指南标准记录 SAP API 提供了全面的指导。
在创建或查看 REST、OData、Java、JavaScript、.NET 或 C/C++ API 的 API 文档时应使用它。

该技能涵盖命名约定、文档注释、OpenAPI 规范、质量检查表、弃用政策、
和手册文档模板。它确保与 SAP API Business Hub 标准和行业最佳实践的一致性。
