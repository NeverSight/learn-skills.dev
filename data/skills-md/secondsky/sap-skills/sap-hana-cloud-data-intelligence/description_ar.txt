تطوير خطوط معالجة البيانات وعمليات التكامل وسيناريوهات التعلم الآلي في SAP Data Intelligence Cloud. يُستخدم عند إنشاء الرسوم البيانية/خطوط الأنابيب مع المشغلين، أو دمج أنظمة ABAP/S4HANA، أو إنشاء تدفقات النسخ المتماثل، أو تطوير سيناريوهات تعلم الآلة باستخدام JupyterLab، أو استخدام وظائف لغة تحويل البيانات. يغطي مشغلي Gen1/Gen2 والمحركات الفرعية (Python وNode.js وC++) ومشغلي البيانات المنظمة وكائنات المستودع.
