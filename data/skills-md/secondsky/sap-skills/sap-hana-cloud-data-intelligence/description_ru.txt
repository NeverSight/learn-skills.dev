Разрабатывает конвейеры обработки данных, сценарии интеграции и машинного обучения в SAP Data Intelligence Cloud. Используйте при построении графиков/конвейеров с помощью операторов, интеграции систем ABAP/S4HANA, создании потоков репликации, разработке сценариев машинного обучения с помощью JupyterLab или использовании функций языка преобразования данных. Охватывает операторы Gen1/Gen2, подсистемы (Python, Node.js, C++), операторы структурированных данных и объекты репозитория.
