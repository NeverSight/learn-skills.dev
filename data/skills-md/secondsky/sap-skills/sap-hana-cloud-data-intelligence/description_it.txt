Sviluppa pipeline di elaborazione dati, integrazioni e scenari di machine learning in SAP Data Intelligence Cloud. Da utilizzare durante la creazione di grafici/pipeline con operatori, l'integrazione di sistemi ABAP/S4HANA, la creazione di flussi di replica, lo sviluppo di scenari ML con JupyterLab o l'utilizzo delle funzioni Data Transformation Language. Copre operatori Gen1/Gen2, motori secondari (Python, Node.js, C++), operatori di dati strutturati e oggetti repository.
