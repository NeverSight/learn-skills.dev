UI5 ツール CLI (@ui5/cli) を使用して SAPUI5/OpenUI5 プロジェクトを管理します。 UI5 プロジェクトの初期化、ui5.yaml または ui5-workspace.yaml ファイルの構成、UI5 アプリケーションまたはライブラリのビルド、HTTP/2 サポートによる開発サーバーの実行、カスタム ビルド タスクまたはサーバー ミドルウェアの作成、ワークスペース/モノリポジトリ セットアップの管理、UI5 CLI エラーのトラブルシューティング、UI5 CLI バージョン間の移行、またはビルド パフォーマンスの最適化を行う場合に使用します。完全な構成と拡張性のガイダンスにより、OpenUI5 と SAPUI5 の両方のフレームワークをサポートします。
