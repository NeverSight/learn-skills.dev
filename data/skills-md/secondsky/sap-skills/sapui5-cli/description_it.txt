Gestisce progetti SAPUI5/OpenUI5 utilizzando la CLI degli strumenti UI5 (@ui5/cli). Da utilizzare durante l'inizializzazione di progetti UI5, la configurazione di file ui5.yaml o ui5-workspace.yaml, la creazione di applicazioni o librerie UI5, l'esecuzione di server di sviluppo con supporto HTTP/2, la creazione di attività di build personalizzate o middleware del server, la gestione delle configurazioni dell'area di lavoro/monorepo, la risoluzione dei problemi degli errori della CLI UI5, la migrazione tra versioni della CLI UI5 o l'ottimizzazione delle prestazioni della build. Supporta i framework OpenUI5 e SAPUI5 con guida completa alla configurazione e all'estensibilità.
