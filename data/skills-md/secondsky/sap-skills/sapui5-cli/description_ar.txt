يدير مشاريع SAPUI5/OpenUI5 باستخدام UI5 Tooling CLI (@ui5/cli). يُستخدم عند تهيئة مشاريع UI5، أو تكوين ملفات ui5.yaml أو ui5-workspace.yaml، أو إنشاء تطبيقات أو مكتبات UI5، أو تشغيل خوادم التطوير بدعم HTTP/2، أو إنشاء مهام بناء مخصصة أو برامج وسيطة للخادم، أو إدارة إعدادات مساحة العمل/monorepo، أو استكشاف أخطاء UI5 CLI وإصلاحها، أو الترحيل بين إصدارات UI5 CLI، أو تحسين أداء البناء. يدعم كلاً من أطر عمل OpenUI5 وSAPUI5 مع إرشادات التكوين وقابلية التوسعة الكاملة.
