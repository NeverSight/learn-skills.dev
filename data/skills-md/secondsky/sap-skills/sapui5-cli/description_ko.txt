UI5 Tooling CLI(@ui5/cli)를 사용하여 SAPUI5/OpenUI5 프로젝트를 관리합니다. UI5 프로젝트 초기화, ui5.yaml 또는 ui5-workspace.yaml 파일 구성, UI5 애플리케이션 또는 라이브러리 빌드, HTTP/2 지원으로 개발 서버 실행, 사용자 정의 빌드 작업 또는 서버 미들웨어 생성, 작업 공간/모노레포 설정 관리, UI5 CLI 오류 문제 해결, UI5 CLI 버전 간 마이그레이션 또는 빌드 성능 최적화 시 사용합니다. 완전한 구성 및 확장성 지침을 통해 OpenUI5 및 SAPUI5 프레임워크를 모두 지원합니다.
