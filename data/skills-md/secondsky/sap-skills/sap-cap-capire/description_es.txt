Habilidad de desarrollo del modelo de programación de aplicaciones (CAP) en la nube de SAP utilizando la documentación de Capire.
Úselo cuando: cree aplicaciones CAP, defina modelos CDS, implemente servicios, trabaje con
Bases de datos SAP HANA/SQLite/PostgreSQL, implementando en SAP BTP Cloud Foundry o Kyma, implementando
UI de Fiori, autorización de manejo, tenencia múltiple o mensajería. Cubre la sintaxis CDL/CQL/CSN,
Tiempos de ejecución de Node.js y Java, controladores de eventos, servicios OData y complementos CAP.
