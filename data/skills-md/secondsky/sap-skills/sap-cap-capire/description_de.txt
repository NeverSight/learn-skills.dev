Kenntnisse in der Entwicklung des SAP Cloud Application Programming Model (CAP) mithilfe der Capire-Dokumentation.
Verwendung bei: Erstellen von CAP-Anwendungen, Definieren von CDS-Modellen, Implementieren von Diensten, Arbeiten mit
SAP HANA/SQLite/PostgreSQL-Datenbanken, Bereitstellung in SAP BTP Cloud Foundry oder Kyma, Implementierung
Fiori-Benutzeroberflächen, Handhabung von Autorisierung, Mandantenfähigkeit oder Messaging. Behandelt die CDL/CQL/CSN-Syntax,
Node.js- und Java-Laufzeiten, Event-Handler, OData-Dienste und CAP-Plugins.
