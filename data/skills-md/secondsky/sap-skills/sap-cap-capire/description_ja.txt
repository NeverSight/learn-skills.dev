Capire ドキュメントを使用した SAP Cloud Application Programming Model (CAP) 開発スキル。

SAP HANA/SQLite/PostgreSQL データベース、SAP BTP Cloud Foundry または Kyma へのデプロイ、実装
Fiori UI、承認、マルチテナンシー、またはメッセージングの処理。 CDL/CQL/CSN 構文をカバーします。
Node.js および Java ランタイム、イベント ハンドラー、OData サービス、CAP プラグイン。
