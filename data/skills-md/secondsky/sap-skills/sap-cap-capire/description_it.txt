Capacità di sviluppo di SAP Cloud Application Programming Model (CAP) utilizzando la documentazione Capire.
Utilizzare quando: creare applicazioni CAP, definire modelli CDS, implementare servizi, lavorare con
Database SAP HANA/SQLite/PostgreSQL, distribuzione su SAP BTP Cloud Foundry o Kyma, implementazione
Fiori UI, gestione dell'autorizzazione, multi-tenancy o messaggistica. Copre la sintassi CDL/CQL/CSN,
Runtime Node.js e Java, gestori di eventi, servizi OData e plug-in CAP.
