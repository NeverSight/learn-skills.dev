مهارة تطوير نموذج برمجة تطبيقات SAP السحابية (CAP) باستخدام وثائق Capire.
يُستخدم عندما: إنشاء تطبيقات CAP، وتحديد نماذج CDS، وتنفيذ الخدمات، والعمل معها
قواعد بيانات SAP HANA/SQLite/PostgreSQL، والنشر إلى SAP BTP Cloud Foundry أو Kyma، وتنفيذها
واجهات مستخدم Fiori، أو التعامل مع التفويض، أو تعدد الإيجارات، أو المراسلة. يغطي بناء جملة CDL/CQL/CSN،
أوقات تشغيل Node.js وJava ومعالجات الأحداث وخدمات OData ومكونات CAP الإضافية.
