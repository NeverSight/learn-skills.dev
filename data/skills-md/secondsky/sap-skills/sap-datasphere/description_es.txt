Complemento completo para el desarrollo de SAP Datasphere con 3 agentes especializados, 5 comandos de barra diagonal y ganchos de validación. Úselo al crear almacenes de datos en SAP BTP, crear modelos analíticos, configurar flujos de datos y flujos de replicación, configurar conexiones a SAP y sistemas de terceros, administrar espacios y usuarios, implementar controles de acceso a datos, utilizar la CLI de datasphere, crear productos de datos para el mercado o monitorear tareas de integración de datos. Cubre Data Builder (vistas gráficas/SQL, tablas locales/remotas, flujos de transformación), Business Builder (entidades comerciales, modelos de consumo), modelos analíticos (dimensiones, medidas, jerarquías), más de 40 tipos de conexión (SAP S/4HANA, BW/4HANA, HANA Cloud, AWS, Azure, GCP, Kafka, HTTP genérico), replicación en tiempo real, cadenas de tareas, transporte de contenido, automatización CLI, gobernanza de catálogos y mercado de datos. Incluye características de 2025: conexiones HTTP genéricas, tareas de API REST en cadenas de tareas, integración de SAP Business Data Cloud. Palabras clave: sap datasphere, nube de almacén de datos, dwc, generador de datos, creador de negocios, modelo analítico, vista gráfica, vista sql, flujo de transformación, flujo de replicación, flujo de datos, cadena de tareas, tabla remota, tabla local, almacén de datos sap btp, conexión de esfera de datos, espacio de esfera de datos, control de acceso a datos, nodo de cómputo elástico, integración de nube de análisis de sap, CLI de esfera de datos, productos de datos, mercado de datos, catálogo, gobernanza
