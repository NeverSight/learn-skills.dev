مكون إضافي شامل لتطوير SAP Datasphere مع 3 وكلاء متخصصين، و5 أوامر شرطة مائلة، وخطافات التحقق من الصحة. يُستخدم عند إنشاء مستودعات البيانات على SAP BTP، أو إنشاء نماذج تحليلية، أو تكوين تدفقات البيانات وتدفقات النسخ المتماثل، أو إعداد الاتصالات بأنظمة SAP والجهات الخارجية، أو إدارة المساحات والمستخدمين، أو تنفيذ عناصر التحكم في الوصول إلى البيانات، أو استخدام واجهة سطر الأوامر لقاعدة البيانات، أو إنشاء منتجات بيانات للسوق، أو مراقبة مهام تكامل البيانات. يغطي منشئ البيانات (طرق العرض الرسومية/SQL، والجداول المحلية/البعيدة، وتدفقات التحويل)، ومنشئ الأعمال (الكيانات التجارية، ونماذج الاستهلاك)، والنماذج التحليلية (الأبعاد، والمقاييس، والتسلسلات الهرمية)، وأكثر من 40 نوعًا من أنواع الاتصال (SAP S/4HANA، وBW/4HANA، وHANA Cloud، وAWS، وAzure، وGCP، وKafka، وGenic HTTP)، والنسخ المتماثل في الوقت الفعلي، وسلاسل المهام، ونقل المحتوى، وأتمتة CLI، وإدارة الكتالوج، وسوق البيانات. يتضمن ميزات 2025: اتصالات HTTP العامة ومهام REST API في سلاسل المهام وتكامل SAP Business Data Cloud. الكلمات الرئيسية: sap datasphere، سحابة مستودع البيانات، dwc، منشئ البيانات، منشئ الأعمال، النموذج التحليلي، العرض الرسومي، عرض sql، تدفق التحويل، تدفق النسخ المتماثل، تدفق البيانات، سلسلة المهام، الجدول البعيد، الجدول المحلي، مستودع بيانات sap btp، اتصال قاعدة البيانات، مساحة قاعدة البيانات، التحكم في الوصول إلى البيانات، عقدة حساب مرنة، تكامل سحابة تحليلات sap، datasphere cli، منتجات البيانات، سوق البيانات، الكتالوج، الإدارة
