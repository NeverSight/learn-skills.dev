Комплексный плагин для разработки SAP Datasphere с 3 специализированными агентами, 5 слэш-командами и перехватчиками проверки. Используйте при построении хранилищ данных на SAP BTP, создании аналитических моделей, настройке потоков данных и потоков репликации, настройке подключений к SAP и сторонним системам, управлении пространствами и пользователями, реализации контроля доступа к данным, использовании интерфейса командной строки данных, создании продуктов данных для рынка или мониторинге задач интеграции данных. Охватывает Data Builder (графические представления/представления SQL, локальные/удаленные таблицы, потоки преобразования), Business Builder (бизнес-сущности, модели потребления), аналитические модели (измерения, меры, иерархии), более 40 типов соединений (SAP S/4HANA, BW/4HANA, HANA Cloud, AWS, Azure, GCP, Kafka, Generic HTTP), репликацию в реальном времени, цепочки задач, транспорт контента, автоматизацию CLI, управление каталогами и рынок данных. Включает функции 2025: общие HTTP-соединения, задачи REST API в цепочках задач, интеграцию с SAP Business Data Cloud. Ключевые слова: сфера данных SAP, облако хранилища данных, dwc, построитель данных, построитель бизнеса, аналитическая модель, графическое представление, представление sql, поток преобразования, поток репликации, поток данных, цепочка задач, удаленная таблица, локальная таблица, хранилище данных SAP btp, подключение к сфере данных, пространство сферы данных, контроль доступа к данным, эластичный вычислительный узел, интеграция облака SAP Analytics, datasphere cli, продукты данных, рынок данных, каталог, управление
