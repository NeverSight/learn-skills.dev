Esta habilidad proporciona una guía integral para el desarrollo, la configuración y las operaciones del servicio de programación de trabajos SAP BTP.
Debe utilizarse al crear, gestionar o solucionar problemas de trabajos programados en SAP Business Technology Platform.

La habilidad cubre la configuración del servicio, el uso de la API REST, los tipos y formatos de programación, la autenticación OAuth 2.0, la tenencia múltiple,
Tareas de Cloud Foundry, integración del tiempo de ejecución de Kyma y monitoreo con SAP Cloud ALM y Alert Notification Service.

Palabras clave: SAP BTP, programación de trabajos, programador de trabajos, cron, programación, trabajos recurrentes, trabajos únicos, tareas de Cloud Foundry,
Tareas CF, Kyma, OAuth 2.0, XSUAA, @sap/jobs-client, API REST, trabajos asincrónicos, punto final de acción, registros de ejecución,
SAP Cloud ALM, servicio de notificación de alertas, multiinquilino, consciente de inquilinos, BC-CP-CF-JBS
