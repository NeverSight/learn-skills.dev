このスキルは、SAP BTP Job Scheduling Service の開発、構成、運用に関する包括的なガイダンスを提供します。
これは、SAP Business Technology Platform でスケジュールされたジョブを作成、管理、またはトラブルシューティングするときに使用する必要があります。

このスキルは、サービスのセットアップ、REST API の使用法、スケジュールのタイプと形式、OAuth 2.0 認証、マルチテナント、
Cloud Foundry タスク、Kyma ランタイム統合、SAP Cloud ALM およびアラート通知サービスによるモニタリング。

CF タスク、Kyma、OAuth 2.0、XSUAA、@sap/jobs-client、REST API、非同期ジョブ、アクション エンドポイント、実行ログ、
SAP Cloud ALM、アラート通知サービス、マルチテナンシー、テナント対応、BC-CP-CF-JBS
