توفر هذه المهارة إرشادات شاملة لتطوير خدمة جدولة الوظائف SAP BTP وتكوينها وعملياتها.
ويجب استخدامه عند إنشاء المهام المجدولة أو إدارتها أو استكشاف الأخطاء وإصلاحها على SAP Business Technology Platform.

تغطي المهارة إعداد الخدمة، واستخدام REST API، وأنواع وتنسيقات الجدول الزمني، ومصادقة OAuth 2.0، والتعددية،
مهام Cloud Foundry وتكامل وقت تشغيل Kyma والمراقبة باستخدام SAP Cloud ALM وخدمة تنبيه التنبيهات.

الكلمات الرئيسية: SAP BTP، جدولة الوظائف، جدولة الوظائف، cron، الجدول الزمني، الوظائف المتكررة، وظائف لمرة واحدة، مهام Cloud Foundry،
مهام CF، Kyma، OAuth 2.0، XSUAA، @sap/jobs-client، REST API، المهام غير المتزامنة، نقطة نهاية الإجراء، سجلات التشغيل،
SAP Cloud ALM، خدمة إعلام التنبيه، تعدد الإيجارات، علم المستأجر، BC-CP-CF-JBS
