Этот навык обеспечивает комплексное руководство по разработке, настройке и эксплуатации службы планирования заданий SAP BTP.
Его следует использовать при создании, управлении и устранении неполадок запланированных заданий на платформе SAP Business Technology Platform.

Навык охватывает настройку служб, использование REST API, типы и форматы расписаний, аутентификацию OAuth 2.0, многопользовательскую эксплуатацию,
Задачи Cloud Foundry, интеграция среды выполнения Kyma и мониторинг с помощью SAP Cloud ALM и службы оповещений.

Ключевые слова: SAP BTP, планирование заданий, планировщик заданий, cron, расписание, повторяющиеся задания, разовые задания, задачи Cloud Foundry,
Задачи CF, Kyma, OAuth 2.0, XSUAA, @sap/jobs-client, REST API, асинхронные задания, конечная точка действия, журналы выполнения,
SAP Cloud ALM, служба оповещений, многопользовательская организация, с учетом арендаторов, BC-CP-CF-JBS
