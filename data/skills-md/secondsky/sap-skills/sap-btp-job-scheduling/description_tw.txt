此技能為 SAP BTP Job Scheduling Service 開發、設定和操作提供全面的指導。
在 SAP Business Technology Platform 上建立、管理計劃作業或對其進行故障排除時應使用它。

此技能涵蓋服務設定、REST API 使用、計劃類型和格式、OAuth 2.0 身份驗證、多租戶、
Cloud Foundry 任務、Kyma 執行時期整合以及使用 SAP Cloud ALM 和警報通知服務進行監控。

關鍵字：SAP BTP、作業排程、jobscheduler、cron、計畫、重複作業、一次性作業、Cloud Foundry 任務、
CF 任務、Kyma、OAuth 2.0、XSUAA、@sap/jobs-client、REST API、非同步作業、操作端點、執行日誌、
SAP Cloud ALM、警報通知服務、多租戶、租戶感知、BC-CP-CF-JBS
