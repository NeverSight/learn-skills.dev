xAI Grok モデルを利用した x_search ツールを使用して、Twitter コンテンツをインテリジェントに検索します。 Fast/Reasoning デュアルエンジン動的ルーティングの先駆者であり、正確なトラフィック分割のための SOCKS5 プロキシ (WARP) をサポートし、公式 X API (1,000 呼び出しあたり約 2.8 ドルでテスト) と比較してコストを削減しながら、より強力なセマンティック理解とよりスマートな結果を提供します。このスキルは、ユーザーがツイートを検索したり、世論を監視したり、トレンドのトピックを分析したり、リアルタイムのツイート データを取得したりする必要がある場合に使用します。
