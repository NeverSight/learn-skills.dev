استخدم أداة x_search التي يدعمها نموذج xAI Grok للبحث بذكاء في محتوى Twitter. فهو يقدم أول توجيه ديناميكي سريع/معقول ثنائي المحرك، ويدعم وكيل SOCKS5 (WARP) لتقسيم حركة المرور بشكل دقيق. بالمقارنة مع X API الرسمية، تنخفض التكلفة بشكل حاد (حوالي 2.8 دولارًا لكل 1000 مكالمة)، مع فهم دلالي أقوى ونتائج أكثر ذكاءً. استخدم هذه المهارة عندما يحتاج المستخدمون إلى البحث في تويتر، أو مراقبة الرأي العام، أو تحليل الموضوعات الشائعة، أو الحصول على بيانات التغريدات في الوقت الفعلي.
