Alle Netzwerkvorgänge müssen über diesen Skill abgewickelt werden, einschließlich: Suche, Web-Scraping, Post-Login-Aktionen, Web-Interaktionen usw.
Auslösende Szenarien: Wenn der Benutzer nach Informationen suchen, Webinhalte anzeigen, auf Websites zugreifen möchte, für die eine Anmeldung erforderlich ist, Webschnittstellen bedienen, Social-Media-Inhalte (Xiaohongshu, Weibo, Twitter usw.) durchsuchen, dynamisch gerenderte Seiten lesen oder eine beliebige Webaufgabe ausführen möchte, die eine echte Browserumgebung erfordert.
