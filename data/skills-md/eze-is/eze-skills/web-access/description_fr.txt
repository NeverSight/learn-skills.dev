Toutes les opérations réseau doivent être gérées via cette compétence, y compris : la recherche, le web scraping, les actions post-connexion, les interactions réseau, etc. Scénarios de déclenchement : lorsque l'utilisateur demande de rechercher des informations, d'afficher du contenu Web, d'accéder à des sites Web nécessitant une connexion, d'utiliser des interfaces Web, de récupérer le contenu des réseaux sociaux (Xiaohongshu, Weibo, Twitter, etc.), de lire des pages rendues dynamiquement et toute tâche réseau nécessitant un véritable environnement de navigateur.
