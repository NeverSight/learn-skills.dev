Tutte le operazioni di rete devono essere gestite tramite questa abilità, tra cui: ricerca, web scraping, azioni post-accesso, interazioni di rete, ecc. Scenari di attivazione: quando l'utente richiede di cercare informazioni, visualizzare contenuti web, accedere a siti Web che richiedono l'accesso, utilizzare interfacce web, eseguire lo scraping di contenuti di social media (Xiaohongshu, Weibo, Twitter, ecc.), leggere pagine visualizzate in modo dinamico e qualsiasi attività di rete che richieda un ambiente browser reale.
