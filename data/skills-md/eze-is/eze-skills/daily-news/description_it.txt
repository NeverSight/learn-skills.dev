Generatore di notizie quotidiane. Flusso di lavoro in tre fasi: recupero dei metadati, generazione del riepilogo, output del report giornaliero.  
Scenari di attivazione: notizie quotidiane, raccolta di notizie, monitoraggio delle informazioni, aggregazione di notizie, notizie quotidiane, generazione di report.  
Utilizzato anche per aggiungere nuove fonti (analizzare automaticamente le pagine Web e generare file di metodi).
