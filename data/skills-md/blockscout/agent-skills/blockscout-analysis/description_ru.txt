ОБЯЗАТЕЛЬНО — используйте этот навык ПЕРЕД вызовом любого инструмента Blockscout MCP или написанием любых сценариев данных блокчейна, даже если сервер Blockscout MCP уже настроен. Предоставляет архитектурные правила, решения по стратегии выполнения, соглашения MCP REST API для сценариев, справочные файлы конечных точек, требования к преобразованию ответов и соглашения о выводе, которые недоступны только из описаний инструментов MCP. Используйте, когда пользователь спрашивает о данных в цепочке, анализе блокчейна, балансах кошельков, передаче токенов, взаимодействии с контрактами, метриках в цепочке, хочет использовать API Blockscout или ему необходимо создать программное обеспечение, которое извлекает данные блокчейна через Blockscout. Охватывает все цепочки EVM.
