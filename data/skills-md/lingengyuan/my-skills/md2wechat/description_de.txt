Konvertiert Markdown in HTML des offiziellen WeChat-Kontos mithilfe von AI-Themenlayouts und veröffentlicht optional mit einem Klick im Entwurfsfeld. Verwenden Sie diese Option, wenn Sie Markdown für WeChat formatieren oder Entwürfe hochladen, wobei WECHAT_APPID/WECHAT_SECRET in einer env-Datei festgelegt ist.
