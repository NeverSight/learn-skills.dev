Автоматизируйте git commit и push. Используйте, когда пользователь говорит «синхронизировать с github», «зафиксировать и отправить», «отправить мои изменения», «сохранить в git», «отправить мою работу» или вызывает /sync_to_github. Стабилизирует все изменения, генерирует сообщение о фиксации, фиксирует и отправляет на удаленный компьютер.
