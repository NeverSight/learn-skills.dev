Automatisieren Sie Git Commit und Push. Wird verwendet, wenn der Benutzer „Mit Github synchronisieren“, „Festschreiben und Pushen“, „Meine Änderungen übertragen“, „In Git speichern“, „Meine Arbeit senden“ sagt oder /sync_to_github aufruft. Stellt alle Änderungen bereit, generiert eine Commit-Nachricht, führt einen Commit durch und sendet sie an die Remote-Adresse.
