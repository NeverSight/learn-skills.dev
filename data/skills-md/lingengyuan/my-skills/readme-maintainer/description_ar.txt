قم بتحديث أو إعادة كتابة ملفات README الخاصة بالمشروع مباشرةً باستخدام نموذج الجلسة الحالية (Codex/Claude Code) باستخدام حقائق المستودع المحلي، دون استدعاء واجهات برمجة تطبيقات إنشاء README الخارجية. يُستخدم عندما يطلب المستخدمون "تحديث الملف README"، أو "إعادة كتابة الملف readme.md"، أو "إنشاء وثائق المشروع"، أو "تحديث أقسام الاستخدام/التثبيت"، أو "محاذاة الملف README مع أحدث تغييرات التعليمات البرمجية" لمستودع الريبو المحلي.
