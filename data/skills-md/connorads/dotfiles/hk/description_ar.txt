قم بإعداد وصيانة hk git Hook manager في أي مستودع. يُستخدم عند إضافة خطافات ما قبل الالتزام، أو تكوين وحدات الفحص، أو إعداد أتمتة جودة التعليمات البرمجية، أو العمل مع hk.pkl، أو الحفاظ على تكوينات الخطاف الموجودة. يتم تشغيلها عند تنفيذ المهام التي تتضمن hk، أو git Hooks، أو فحوصات ما قبل الالتزام، أو التحقق من صحة الالتزام، أو مسارات الفحص.
