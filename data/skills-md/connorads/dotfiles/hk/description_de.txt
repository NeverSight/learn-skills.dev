Richten Sie den hk git Hook Manager in einem beliebigen Repository ein und verwalten Sie ihn. Verwenden Sie diese Option, wenn Sie Pre-Commit-Hooks hinzufügen, Linters konfigurieren, die Automatisierung der Codequalität einrichten, mit hk.pkl arbeiten oder vorhandene Hook-Konfigurationen verwalten. Wird bei Aufgaben ausgelöst, die hk, Git-Hooks, Pre-Commit-Prüfungen, Commit-Msg-Validierung oder Linting-Pipelines umfassen.
