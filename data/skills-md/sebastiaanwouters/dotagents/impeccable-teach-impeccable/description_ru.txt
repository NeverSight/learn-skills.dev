Эквивалент impeccable.style /teach-impecable только для навыков. Однократная настройка, которая собирает контекст дизайна для вашего проекта и сохраняет его в файле конфигурации AI. Запустите один раз, чтобы установить постоянные рекомендации по проектированию. Используйте для задач дизайна интерфейса и пользовательского интерфейса.
