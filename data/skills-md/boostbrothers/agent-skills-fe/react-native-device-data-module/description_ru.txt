Кроссплатформенный собственный модуль доступа к хранилищу, обеспечивающий унифицированный интерфейс для iOS UserDefaults и Android SharedPreferences. Используйте при работе с данными устройства, UserDefaults, SharedPreferences, собственным хранилищем, хранилищем устройства или сопоставлением ключей платформы.
