Plattformübergreifendes natives Speicherzugriffsmodul, das eine einheitliche Schnittstelle zu iOS UserDefaults und Android SharedPreferences bietet. Verwenden Sie diese Option, wenn Sie mit Gerätedaten, UserDefaults, SharedPreferences, nativem Speicher, Gerätespeicher oder Plattformschlüsselzuordnung arbeiten.
