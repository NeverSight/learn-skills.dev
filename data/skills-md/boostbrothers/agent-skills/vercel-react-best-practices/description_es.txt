Pautas de optimización del rendimiento de React, React Native y Next.js de Vercel Engineering. Esta habilidad debe usarse al escribir, revisar o refactorizar código React/React Native/Next.js para garantizar patrones de rendimiento óptimos. Incluye soluciones específicas del marco (Next.js, Vite, CRA) y alternativas de biblioteca (SWR vs React Query, Jotai). Activadores de tareas que involucran componentes de React, páginas Next.js, aplicaciones React Native, recuperación de datos, optimización de paquetes o mejoras de rendimiento.
