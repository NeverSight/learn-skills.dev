Рекомендации по оптимизации производительности React, React Native и Next.js от Vercel Engineering. Этот навык следует использовать при написании, проверке или рефакторинге кода React/React Native/Next.js, чтобы обеспечить оптимальные шаблоны производительности. Включает решения для конкретной платформы (Next.js, Vite, CRA) и альтернативные библиотеки (SWR и React Query, Jotai). Запускается при выполнении задач, включающих компоненты React, страницы Next.js, приложения React Native, получение данных, оптимизацию пакетов или повышение производительности.
