Cómo utilizar la CLI de OSMO para gestionar recursos informáticos en la nube para el desarrollo de robótica. Utilice esta habilidad siempre que el usuario pregunte sobre los recursos, nodos, grupos, GPU o capacidad de procesamiento disponibles en OSMO, incluso si no dice "OSMO" explícitamente. Úselo también cuando pregunten qué pueden ejecutar, si tienen cuota, quieren verificar su perfil o acceso al grupo, quieren enviar un flujo de trabajo (SDG, capacitación de RL o personalizado), quieren verificar el estado o los registros de un flujo de trabajo en ejecución/completado, listar o explorar envíos de flujo de trabajo recientes, quieren entender qué hace o está configurado para hacer un flujo de trabajo específico, o quieren crear una aplicación OSMO a partir de un flujo de trabajo.
