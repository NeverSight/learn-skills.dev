كيفية استخدام OSMO CLI لإدارة موارد الحوسبة السحابية لتطوير الروبوتات. استخدم هذه المهارة عندما يسأل المستخدم عن الموارد المتاحة أو العقد أو المجمعات أو وحدات معالجة الرسومات أو سعة الحوسبة على OSMO - حتى لو لم يذكر "OSMO" بشكل صريح. استخدمه أيضًا عندما يسألون عما يمكنهم تشغيله، سواء كان لديهم حصة، أو يريدون التحقق من ملفهم الشخصي أو الوصول إلى المجمع، أو يريدون إرسال سير عمل (SDG، أو تدريب RL، أو مخصص)، أو يريدون التحقق من حالة أو سجلات سير عمل قيد التشغيل/مكتمل، أو قائمة أو تصفح عمليات إرسال سير العمل الأخيرة، أو يريدون فهم ما يفعله سير عمل معين أو تم تكوينه للقيام به، أو يريدون إنشاء تطبيق OSMO من سير عمل.
