So verwenden Sie die OSMO-CLI zum Verwalten von Cloud-Computing-Ressourcen für die Robotikentwicklung. Verwenden Sie diesen Skill immer dann, wenn der Benutzer nach verfügbaren Ressourcen, Knoten, Pools, GPUs oder Rechenkapazität auf OSMO fragt – auch wenn er nicht explizit „OSMO“ sagt. Verwenden Sie es auch, wenn sie fragen, was sie ausführen können, ob sie ein Kontingent haben, ihr Profil oder ihren Poolzugriff überprüfen möchten, einen Workflow (SDG, RL-Training oder benutzerdefiniert) übermitteln möchten, den Status oder Protokolle eines laufenden/abgeschlossenen Workflows überprüfen möchten, kürzlich eingereichte Workflows auflisten oder durchsuchen möchten, verstehen möchten, was ein bestimmter Workflow tut oder wozu er konfiguriert ist, oder eine OSMO-App aus einem Workflow erstellen möchten.
