قم بإنشاء صور معاينة اجتماعية ديناميكية Open Graph (OG) لمشاريع Next.js App Router باستخدام مكتبة next/og وImageResponse. استخدمه عندما يُطلب منك: إنشاء صور OG، أو إصلاح معاينات المشاركة الاجتماعية، أو إضافة og: دعم الصورة، أو إعداد opengraph-image.tsx، أو تكوين البيانات التعريفية للمشاركة الاجتماعية، أو إصلاح صور OG التي لا تظهر الشعار، أو إصلاح النص المعطل في صور OG، أو دعم مجالات متعددة في بيانات تعريف OG. يتعامل مع عمليات النشر متعددة النطاقات (مثل خدمة Vercel wemoova.com + wemoova.cl)، ومراوغات عرض Satori، وتكوين الأيقونة المفضلة، وحدود طول العنوان/الوصف.
