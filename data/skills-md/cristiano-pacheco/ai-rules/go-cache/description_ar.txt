قم بإنشاء تطبيقات Go Cache باتباع اصطلاحات البنية المعيارية لـ GO. استخدم هذه المهارة دائمًا عندما يطلب المستخدم إنشاء ذاكرة تخزين مؤقت، أو إضافة طبقة ذاكرة تخزين مؤقت لـ Redis، أو تخزين مؤقت للبيانات قصيرة العمر باستخدام TTL، أو تنفيذ تخزين محدد للمعدل، أو تخزين مؤقت لـ OTP، أو تخزين مؤقت للجلسة، أو تخزين حالة OAuth، أو أي ذاكرة تخزين مؤقت للمجال في الوحدات الداخلية/الوحدات/<module>/cache/. قم بالاستدعاء بشكل استباقي عندما يذكر المستخدم التخزين المؤقت، أو التخزين المدعوم من Redis، أو انتهاء صلاحية TTL، أو البيانات المؤقتة - حتى لو لم يذكر "ذاكرة التخزين المؤقت" بشكل صريح.
