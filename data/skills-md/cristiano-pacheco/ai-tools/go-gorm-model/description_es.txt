Genere modelos Go GORM siguiendo las convenciones de arquitectura modular de Go. Úselo al crear o actualizar modelos de persistencia en internal/modules/<module>/model/, incluida la asignación de tablas, tipos de punteros que aceptan valores NULL, etiquetas de índice, tipos específicos de PostgreSQL y marcas de tiempo. Utilice siempre esta habilidad cuando se le solicite crear un modelo, agregar una estructura GORM, asignar una tabla de base de datos o generar archivos de modelo.
