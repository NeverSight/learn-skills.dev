Genere implementaciones de enrutadores Chi siguiendo las convenciones de arquitectura modular de Go (enrutador Chi a partir del paquete bricks, Fx DI con interfaz chi.Route, puntos finales REST). Utilice siempre esta habilidad al crear o modificar el registro de ruta HTTP en internal/modules/<module>/http/chi/router/, incluido cualquier archivo de enrutador nuevo, rutas CRUD (GET, POST, PUT, DELETE), puntos finales de acciones personalizadas, API versionadas, grupos de rutas con middleware o cableado de enrutadores en fx.go de un módulo.
