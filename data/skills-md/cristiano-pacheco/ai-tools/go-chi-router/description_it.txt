Genera implementazioni di router Chi seguendo le convenzioni dell'architettura modulare Go (router Chi dal pacchetto bricks, Fx DI con interfaccia chi.Route, endpoint REST). Utilizza sempre questa competenza quando crei o modifichi la registrazione del percorso HTTP in internal/modules/<module>/http/chi/router/, inclusi eventuali nuovi file del router, percorsi CRUD (GET, POST, PUT, DELETE), endpoint di azioni personalizzate, API con versione, gruppi di percorsi con middleware o cablaggio di router nel fx.go di un modulo.
