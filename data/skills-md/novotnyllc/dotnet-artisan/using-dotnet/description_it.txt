Rileva l'intento .NET per qualsiasi richiesta C#, ASP.NET Core, EF Core, Blazor, MAUI, Uno Platform, WPF, WinUI, SignalR, gRPC, xUnit, NuGet o MSBuild dalle parole chiave del prompt e dai segnali del repository (file .sln, .csproj, global.json, .cs). Prima competenza da invocare per tutto il lavoro .NET: carica standard di codifica specifici della versione e instrada alle competenze del dominio tramite [skill:dotnet-advisor] prima di qualsiasi pianificazione o implementazione. Non utilizzare per attività chiaramente non .NET (Python, JavaScript, Go, Rust, Java).
