Détecte l'intention .NET pour toute requête C#, ASP.NET Core, EF Core, Blazor, MAUI, Uno Platform, WPF, WinUI, SignalR, gRPC, xUnit, NuGet ou MSBuild à partir de mots-clés d'invite et de signaux de référentiel (fichiers .sln, .csproj, global.json, .cs). Première compétence à invoquer pour tout travail .NET : charge les normes de codage spécifiques à la version et les itinéraires vers les compétences de domaine via [skill: dotnet-advisor] avant toute planification ou implémentation. Ne pas utiliser pour des tâches clairement non-.NET (Python, JavaScript, Go, Rust, Java).
