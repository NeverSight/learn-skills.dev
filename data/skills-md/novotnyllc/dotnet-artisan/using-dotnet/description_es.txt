Detecta la intención de .NET para cualquier solicitud de C#, ASP.NET Core, EF Core, Blazor, MAUI, Uno Platform, WPF, WinUI, SignalR, gRPC, xUnit, NuGet o MSBuild a partir de palabras clave y señales de repositorio (archivos .sln, .csproj, global.json, .cs). Primera habilidad a invocar para todo el trabajo .NET: carga estándares de codificación específicos de la versión y rutas a habilidades de dominio a través de [skill:dotnet-advisor] antes de cualquier planificación o implementación. No lo utilice para tareas que claramente no sean .NET (Python, JavaScript, Go, Rust, Java).
