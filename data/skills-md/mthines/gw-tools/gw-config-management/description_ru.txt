Настраивайте и оптимизируйте инструменты gw для различных типов проектов и потребностей команды. Используйте при настройке gw для новых проектов, настройке файлов автоматического копирования, устранении проблем с конфигурацией или настройке gw для Next.js, API-интерфейсов Node.js, монорепозиториев или React SPA. Запускается по .gw/config.json, шаблонам автоматического копирования, файлам среды или командам инициализации gw.
