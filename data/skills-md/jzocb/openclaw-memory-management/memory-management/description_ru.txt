Управляйте памятью агента ИИ с помощью системы приоритетов P0/P1/P2 и автоматического архивирования.
Используйте при: настройке управления памятью, очистке MEMORY.md, создании
файлы уроков, настройку автоматического архивирования cron или проверку структуры памяти.
Снижает использование токенов на 70-80%, сохраняя при этом отзыв посредством семантического поиска.
