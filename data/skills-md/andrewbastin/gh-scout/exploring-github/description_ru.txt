Исследуйте и изучайте репозитории GitHub с помощью интерфейса командной строки gh. Используйте, когда вас просят просмотреть репозитории, прочитать файлы, выполнить поиск кода, проанализировать проблемы/запросы/коммиты/релизы или разобраться в базах кода на GitHub. Заменяет инструменты GitHub на основе MCP с нулевыми зависимостями, кроме gh.
