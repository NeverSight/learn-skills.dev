Rilascia una nuova versione della CLI CodeMie. Da utilizzare quando l'utente dice "rilascio", "crea rilascio", "pubblica versione", "versione bump e rilascio" o desidera pubblicare una nuova versione su npm. Gestisce il bumping della versione, il git tagging, il push e la creazione di versioni di GitHub con rilevamento automatico della versione dai commit convenzionali.
