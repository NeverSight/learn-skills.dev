Veröffentlichen Sie eine neue Version von CodeMie CLI. Verwenden Sie diese Option, wenn der Benutzer „Release“, „Release erstellen“, „Version veröffentlichen“, „Bump-Version und Release“ sagt oder eine neue Version auf npm veröffentlichen möchte. Verwaltet Versions-Bumping, Git-Tagging, Pushing und GitHub-Release-Erstellung mit automatischer Versionserkennung anhand herkömmlicher Commits.
