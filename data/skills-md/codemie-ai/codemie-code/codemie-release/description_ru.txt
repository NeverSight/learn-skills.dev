Выпустите новую версию CodeMie CLI. Используйте, когда пользователь говорит «выпустить», «создать выпуск», «опубликовать версию», «обновить версию и выпустить» или хочет опубликовать новую версию в npm. Управляет обновлением версий, тегами git, отправкой и созданием выпусков GitHub с автоматическим определением версий на основе обычных коммитов.
