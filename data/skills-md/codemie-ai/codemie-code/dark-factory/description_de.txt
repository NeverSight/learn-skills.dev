Diese Fähigkeit sollte verwendet werden, wenn der Benutzer „ein Jira-Ticket an Dark Factory delegieren“, „mit der Arbeit an einem EPMCDME-Ticket als Fabrik beginnen“, „EPMCDME-Ticket implementieren“, „Implementierung beginnen“ oder „Aufgabe autonom implementieren“ möchte oder eine strukturierte technische Führung für die Implementierung eines Jira-Tickets wünscht. Zum Start ist eine gültige EPMCDME-XXXXX-Ticket-ID ERFORDERLICH. Wenn kein Ticket bereitgestellt wird, wird der Skill blockiert und der Benutzer wird aufgefordert, zuerst ein Ticket zu erstellen.
