Swarm-safe рабочий процесс контрольной точки git для проверки, подготовки, фиксации и, при необходимости, отправки работы в любой репозиторий. Используйте, когда пользователь говорит «контрольная точка», «сохранить», «зафиксировать», «подготовить и зафиксировать» или хочет получить безопасный, повторяемый снимок.
