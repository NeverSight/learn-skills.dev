Utilisez la CLI Memrise pour les cours communautaires, y compris la configuration de l'authentification, la recherche de cours/niveau/mot, les recherches de pool et l'ajout d'éléments. À utiliser lorsque les requêtes mentionnent les commandes `memcli` ou le package npm `memrise-cli` (qui installe le binaire `memcli`), en particulier pour la construction de commandes, les indicateurs de dépannage et la sortie JSON conviviale pour les scripts.
