Opere la CLI de Memrise para cursos comunitarios, incluida la configuración de autenticación, búsqueda de cursos/niveles/palabras, búsquedas en grupos y adición de elementos. Úselo cuando las solicitudes mencionen comandos `memcli` o el paquete npm `memrise-cli` (que instala el binario `memcli`), especialmente para la construcción de comandos, resolución de problemas y salida JSON compatible con scripts.
