Интегрируйте OpenClaw с DroidClaw для автоматизации устройств Android с помощью размещенных или локальных развертываний. Используйте, когда пользователи просят настроить поля ключа/ссылки на сервер DroidClaw Android API, самостоятельно разместить DroidClaw, подключить устройства через Tailscale, запустить/остановить цели DroidClaw, проверить сеансы/шаги устройства или предоставить OpenClaw API управления DroidClaw.
