Integre OpenClaw con DroidClaw para la automatización de dispositivos Android mediante implementaciones alojadas o autohospedadas. Úselo cuando los usuarios soliciten configurar los campos de enlace del servidor/clave API de Android de DroidClaw, autohospedar DroidClaw, conectar dispositivos a través de Tailscale, ejecutar/detener objetivos de DroidClaw, inspeccionar sesiones/pasos del dispositivo o exponer las API de control de DroidClaw a OpenClaw.
