Integra OpenClaw con DroidClaw per l'automazione dei dispositivi Android utilizzando distribuzioni ospitate o self-hosted. Da utilizzare quando gli utenti chiedono di configurare i campi di collegamento chiave/server API Android DroidClaw, ospitare autonomamente DroidClaw, connettere dispositivi su Tailscale, eseguire/arrestare obiettivi DroidClaw, ispezionare sessioni/passaggi del dispositivo o esporre le API di controllo DroidClaw a OpenClaw.
