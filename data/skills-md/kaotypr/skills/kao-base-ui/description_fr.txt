Guide pour créer une interface utilisateur avec Base UI React (@base-ui/react), une bibliothèque de composants sans tête et accessible utilisant des modèles de composants composés. Utilisez cette compétence chaque fois que l'utilisateur crée ou modifie une interface utilisateur dans React, y compris les formulaires et la validation, la navigation et les menus, les modaux et les superpositions, les contrôles de sélection, les notifications toast, les accordéons, les onglets ou tout autre composant d'interface utilisateur interactif. Déclenchez également lorsque l'utilisateur mentionne @base-ui/react, Base UI, les composants sans tête, la migration depuis Radix UI ou pose des questions sur les modèles de composants accessibles. Même si l'utilisateur ne mentionne pas explicitement Base UI, utilisez cette compétence chaque fois qu'il crée des composants React UI, construit un système de conception ou travaille sur l'expérience utilisateur front-end.
