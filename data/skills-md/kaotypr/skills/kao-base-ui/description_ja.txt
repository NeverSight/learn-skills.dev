複合コンポーネント パターンを使用したヘッドレスでアクセス可能なコンポーネント ライブラリである Base UI React (@base-ui/react) を使用して UI を構築するためのガイド。このスキルは、ユーザーが React でフォームと検証、ナビゲーションとメニュー、モーダルとオーバーレイ、選択コントロール、トースト通知、アコーディオン、タブ、または対話型 UI コンポーネントを含むユーザー インターフェイスを構築または変更するときに常に使用します。また、ユーザーが @base-ui/react、Base UI、ヘッドレス コンポーネント、Radix UI からの移行について言及した場合、またはアクセス可能なコンポーネント パターンについて質問した場合にもトリガーされます。ユーザーが Base UI について明示的に言及していない場合でも、React UI コンポーネントを作成するとき、デザイン システムを構築するとき、またはフロントエンドのユーザー エクスペリエンスに取り組むときは常にこのスキルを使用してください。
