Guía para crear una interfaz de usuario con Base UI React (@base-ui/react), una biblioteca de componentes accesible y sin cabeza que utiliza patrones de componentes compuestos. Utilice esta habilidad siempre que el usuario esté creando o modificando cualquier interfaz de usuario en React, incluidos formularios y validación, navegación y menús, modales y superposiciones, controles de selección, notificaciones de brindis, acordeones, pestañas o cualquier componente interactivo de la interfaz de usuario. También se activa cuando el usuario menciona @base-ui/react, Base UI, componentes sin cabeza, migración desde Radix UI o pregunta sobre patrones de componentes accesibles. Incluso si el usuario no menciona explícitamente la interfaz de usuario base, utilice esta habilidad siempre que esté creando componentes de React UI, construyendo un sistema de diseño o trabajando en la experiencia del usuario frontend.
