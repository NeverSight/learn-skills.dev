Guida per la creazione dell'interfaccia utente con Base UI React (@base-ui/react), una libreria di componenti accessibile e headless che utilizza modelli di componenti composti. Utilizza questa competenza ogni volta che l'utente crea o modifica qualsiasi interfaccia utente in React, inclusi moduli e convalida, navigazione e menu, modali e sovrapposizioni, controlli di selezione, notifiche pop-up, accordion, schede o qualsiasi componente dell'interfaccia utente interattiva. Si attiva anche quando l'utente menziona @base-ui/react, Base UI, componenti headless, migrazione da Radix UI o chiede informazioni sui modelli di componenti accessibili. Anche se l'utente non menziona esplicitamente l'interfaccia utente di base, utilizza questa competenza ogni volta che crea componenti dell'interfaccia utente di React, costruisce un sistema di progettazione o lavora sull'esperienza utente frontend.
