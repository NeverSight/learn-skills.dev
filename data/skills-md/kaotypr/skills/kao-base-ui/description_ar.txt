دليل لبناء واجهة المستخدم باستخدام Base UI React (@base-ui/react)، وهي مكتبة مكونات مقطوعة الرأس ويمكن الوصول إليها باستخدام أنماط المكونات المركبة. استخدم هذه المهارة عندما يقوم المستخدم بإنشاء أو تعديل أي واجهة مستخدم في React، بما في ذلك النماذج والتحقق من الصحة، والتنقل والقوائم، والنماذج والتراكبات، وعناصر التحكم في التحديد، وإشعارات الخبز المحمص، والأكورديون، وعلامات التبويب، أو أي مكون تفاعلي لواجهة المستخدم. يتم التشغيل أيضًا عندما يذكر المستخدم @base-ui/react، أو Base UI، أو المكونات بدون رأس، أو الترحيل من Radix UI، أو يسأل عن أنماط المكونات التي يمكن الوصول إليها. حتى لو لم يذكر المستخدم واجهة المستخدم الأساسية صراحةً، فاستخدم هذه المهارة عندما يقوم بإنشاء مكونات React UI، أو بناء نظام تصميم، أو العمل على تجربة مستخدم الواجهة الأمامية.
