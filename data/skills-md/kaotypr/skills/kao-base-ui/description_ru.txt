Руководство по созданию пользовательского интерфейса с помощью Base UI React (@base-ui/react), безголовой, доступной библиотеки компонентов с использованием шаблонов составных компонентов. Используйте этот навык всякий раз, когда пользователь создает или изменяет любой пользовательский интерфейс в React, включая формы и проверку, навигацию и меню, модальные окна и наложения, элементы управления выбором, всплывающие уведомления, аккордеоны, вкладки или любой интерактивный компонент пользовательского интерфейса. Также срабатывает, когда пользователь упоминает @base-ui/react, Base UI, автономные компоненты, переход с пользовательского интерфейса Radix или спрашивает о шаблонах доступных компонентов. Даже если пользователь явно не упоминает базовый пользовательский интерфейс, используйте этот навык всякий раз, когда он создает компоненты пользовательского интерфейса React, создает систему дизайна или работает над пользовательским интерфейсом.
