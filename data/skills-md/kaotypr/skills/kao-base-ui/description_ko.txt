복합 구성 요소 패턴을 사용하는 헤드리스 액세스 가능 구성 요소 라이브러리인 Base UI React(@base-ui/react)를 사용하여 UI를 구축하기 위한 가이드입니다. 사용자가 양식 및 유효성 검사, 탐색 및 메뉴, 모달 및 오버레이, 선택 컨트롤, 토스트 알림, 아코디언, 탭 또는 대화형 UI 구성 요소를 포함하여 React에서 사용자 인터페이스를 구축하거나 수정할 때마다 이 기술을 사용하세요. 또한 사용자가 @base-ui/react, Base UI, 헤드리스 구성 요소를 언급하거나 Radix UI에서 마이그레이션하거나 액세스 가능한 구성 요소 패턴에 대해 묻는 경우에도 트리거됩니다. 사용자가 Base UI를 명시적으로 언급하지 않더라도 React UI 구성 요소를 만들거나 디자인 시스템을 구축하거나 프런트엔드 사용자 경험 작업을 할 때마다 이 기술을 사용하세요.
