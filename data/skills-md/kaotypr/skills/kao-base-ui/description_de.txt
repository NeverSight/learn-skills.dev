Leitfaden zum Erstellen einer Benutzeroberfläche mit Base UI React (@base-ui/react), einer kopflosen, zugänglichen Komponentenbibliothek, die zusammengesetzte Komponentenmuster verwendet. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine Benutzeroberfläche in React erstellt oder ändert, einschließlich Formulare und Validierung, Navigation und Menüs, Modalitäten und Überlagerungen, Auswahlsteuerelemente, Toastbenachrichtigungen, Akkordeons, Registerkarten oder andere interaktive UI-Komponenten. Wird auch ausgelöst, wenn der Benutzer @base-ui/react, Base UI, Headless-Komponenten, die Migration von der Radix-Benutzeroberfläche erwähnt oder nach zugänglichen Komponentenmustern fragt. Auch wenn der Benutzer Base UI nicht explizit erwähnt, verwenden Sie diese Fähigkeit immer dann, wenn er React UI-Komponenten erstellt, ein Designsystem erstellt oder an der Frontend-Benutzererfahrung arbeitet.
