Recherche de code sémantique à l'aide des intégrations ML et de l'analyse AST. Remplace les outils de recherche intégrés pour l'exploration de code basée sur l'intention. À utiliser lorsque l'utilisateur demande à trouver du code en décrivant ce qu'il fait, à comprendre les relations entre les codes ou à explorer une base de code sémantiquement.
