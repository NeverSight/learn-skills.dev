GoldRush x402 — доступ к данным блокчейна с оплатой за запрос с использованием протокола x402 (требуется оплата HTTP 402). Используйте этот навык всякий раз, когда пользователь создает ИИ-агент, которому нужны данные блокчейна без ключей API, он хочет осуществлять микроплатежи на основе кошелька для данных внутри цепочки, нуждается в автономном доступе или доступе без учетной записи к API GoldRush, упоминает протокол x402 или хочет получить доступ к данным блокчейна без регистрации/без регистрации. Это правильный навык для автономных агентов, бессерверных приложений и создания прототипов без предварительной подготовки. Предоставляет доступ к более чем 60 конечным точкам Foundational API через прозрачный обратный прокси-сервер с платежами в стейблкоинах на базе. Если пользователю нужен традиционный ключ API с ежемесячной оплатой, используйте вместо него goldrush-foundational-api. Если пользователю нужна потоковая передача данных в реальном времени через WebSocket, вместо этого используйте goldrush-streaming-api.
