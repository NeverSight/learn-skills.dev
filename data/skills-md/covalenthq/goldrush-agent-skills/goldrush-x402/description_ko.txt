GoldRush x402 — x402 프로토콜을 사용하는 요청당 지불 블록체인 데이터 액세스(HTTP 402 지불 필요) 사용자가 API 키 없이 블록체인 데이터가 필요한 AI 에이전트를 구축할 때, 온체인 데이터에 대한 지갑 기반 소액 결제를 원할 때, GoldRush API에 대한 자율적 또는 비계정 액세스가 필요하거나 x402 프로토콜을 언급할 때, 또는 가입 없음/온보딩 없음 블록체인 데이터 액세스를 원할 때마다 이 기술을 사용하십시오. 이는 자율 에이전트, 서버리스 애플리케이션 및 온보딩 없이 프로토타입을 제작하는 데 적합한 기술입니다. Base에서 스테이블코인 결제가 가능한 투명한 역방향 프록시를 통해 60개 이상의 기본 API 엔드포인트에 대한 액세스를 제공합니다. 사용자에게 월별 청구가 포함된 기존 API 키가 필요한 경우 대신 goldrush-foundational-api를 사용하세요. 사용자가 WebSocket을 통한 실시간 스트리밍 데이터가 필요한 경우 대신 goldrush-streaming-api를 사용하세요.
