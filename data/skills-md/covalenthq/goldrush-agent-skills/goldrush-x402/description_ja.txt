GoldRush x402 — x402 プロトコルを使用したペイパーリクエストのブロックチェーン データ アクセス (HTTP 402 支払いが必要)。 API キーなしでブロックチェーン データが必要な場合、オンチェーン データに対してウォレット ベースのマイクロペイメントが必要な場合、GoldRush API への自律アクセスまたはアカウントなしのアクセスが必要な場合、x402 プロトコルに言及している場合、またはサインアップ/オンボーディングなしのブロックチェーン データ アクセスを必要とする AI エージェントをユーザーが構築する場合は、必ずこのスキルを使用してください。これは、自律エージェント、サーバーレス アプリケーション、オンボーディングなしのプロトタイピングに適したスキルです。 Base でのステーブルコイン支払いによる透過的なリバース プロキシを介して、60 以上の基礎 API エンドポイントへのアクセスを提供します。ユーザーが月額課金の従来の API キーを必要とする場合は、代わりに goldrush-foundational-api を使用してください。ユーザーが WebSocket 経由でリアルタイムのストリーミング データを必要とする場合は、代わりに goldrush-streaming-api を使用してください。
