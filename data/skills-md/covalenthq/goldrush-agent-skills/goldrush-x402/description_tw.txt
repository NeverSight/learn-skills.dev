GoldRush x402 — 使用 x402 協定按請求付費的區塊鏈資料存取（需要 HTTP 402 付款）。当用户构建的 AI 代理需要没有 API 密钥的区块链数据、需要基于钱包的微支付链上数据、需要自主或无账户访问 GoldRush API、提及 x402 协议或需要无需注册/无需登录的区块链数据访问时，请使用此技能。這是自治代理、無伺服器應用程式和無需入門的原型設計的正確技能。透過透明的反向代理，透過 Base 上的穩定幣支付，提供對 60 多個基礎 API 端點的存取。如果使用者需要按月計費的傳統 API 金鑰，請改用 goldrush-foundational-api。如果使用者需要透過 WebSocket 取得即時串流數據，請使用 goldrush-streaming-api。
