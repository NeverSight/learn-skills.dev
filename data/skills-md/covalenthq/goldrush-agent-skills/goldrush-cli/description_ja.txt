GoldRush CLI — Claude Desktop および Claude Code の MCP サポートを備えたターミナルファーストのブロックチェーン データ ツール。ユーザーがコマンドラインからブロックチェーンデータをクエリしたいとき、ターミナルでDEXペアやウォレットアクティビティをストリーミングしたいとき、GoldRushをMCPツールプロバイダーとして設定したいとき、またはコードを書かずに素早い一回限りのクエリを実行したいとき（例：「ウォレット残高を確認する」、「ガソリン価格はいくらか」、「トークンを検索する」など）には、このスキルを使用します。また、ユーザーが「goldrush」CLI コマンド、「npx @covalenthq/goldrush-cli」、または GoldRush との MCP 統合について言及する場合にもこれを使用します。 CLI は、端末からのアドホック ブロックチェーン ルックアップの最速パスです。ユーザーがアプリケーションでプログラムによる API アクセスを必要とする場合は、代わりに goldrush-foundational-api または goldrush-streaming-api を使用してください。ユーザーが API キーを使用せずにペイパーリクエストアクセスを必要とする場合は、代わりに goldrush-x402 を使用してください。
