GoldRush CLI: strumento di dati blockchain terminale con supporto MCP per Claude Desktop e Claude Code. Utilizza questa abilità ogni volta che l'utente desidera interrogare i dati blockchain dalla riga di comando, trasmettere in streaming coppie DEX o attività del portafoglio in un terminale, impostare GoldRush come fornitore di strumenti MCP o eseguire query rapide una tantum senza scrivere codice (ad esempio, "controlla il saldo del portafoglio", "qual è il prezzo del gas", "cerca un token"). Utilizzalo anche quando l'utente menziona i comandi CLI "goldrush", "npx @covalenthq/goldrush-cli" o l'integrazione MCP con GoldRush. La CLI è il percorso più veloce per le ricerche blockchain ad hoc dal terminale. Se l'utente necessita dell'accesso API programmatico in un'applicazione, utilizza invece goldrush-foundational-api o goldrush-streaming-api. Se l'utente necessita di un accesso con pagamento per richiesta senza una chiave API, utilizza invece goldrush-x402.
