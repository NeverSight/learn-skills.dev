GoldRush CLI: herramienta de datos blockchain de terminal con soporte MCP para Claude Desktop y Claude Code. Utilice esta habilidad siempre que el usuario quiera consultar datos de blockchain desde la línea de comando, transmitir pares DEX o actividad de billetera en una terminal, configurar GoldRush como proveedor de herramientas MCP o ejecutar consultas rápidas únicas sin escribir código (por ejemplo, "verificar el saldo de una billetera", "cuál es el precio del gas", "buscar un token"). Utilícelo también cuando el usuario mencione los comandos CLI 'goldrush', 'npx @covalenthq/goldrush-cli' o la integración de MCP con GoldRush. La CLI es la ruta más rápida para búsquedas ad hoc de blockchain desde la terminal. Si el usuario necesita acceso API programático en una aplicación, use goldrush-foundational-api o goldrush-streaming-api en su lugar. Si el usuario necesita acceso de pago por solicitud sin una clave API, utilice goldrush-x402 en su lugar.
