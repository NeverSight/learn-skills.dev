GoldRush CLI – Terminal-First-Blockchain-Datentool mit MCP-Unterstützung für Claude Desktop und Claude Code. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer Blockchain-Daten über die Befehlszeile abfragen, DEX-Paare oder Wallet-Aktivitäten in einem Terminal streamen, GoldRush als MCP-Tool-Anbieter einrichten oder schnelle einmalige Abfragen ausführen möchte, ohne Code schreiben zu müssen (z. B. „Wallet-Kontostand prüfen“, „Wie hoch ist der Benzinpreis“, „Nach einem Token suchen“). Verwenden Sie dies auch, wenn der Benutzer „goldrush“-CLI-Befehle, „npx @covalenthq/goldrush-cli“ oder die MCP-Integration mit GoldRush erwähnt. Die CLI ist der schnellste Weg für Ad-hoc-Blockchain-Suchen vom Terminal aus. Wenn der Benutzer programmgesteuerten API-Zugriff in einer Anwendung benötigt, verwenden Sie stattdessen goldrush-foundational-api oder goldrush-streaming-api. Wenn der Benutzer einen Pay-per-Request-Zugriff ohne API-Schlüssel benötigt, verwenden Sie stattdessen goldrush-x402.
