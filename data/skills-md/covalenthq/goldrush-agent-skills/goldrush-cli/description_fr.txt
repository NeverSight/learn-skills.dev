GoldRush CLI — outil de données blockchain premier terminal avec prise en charge MCP pour Claude Desktop et Claude Code. Utilisez cette compétence chaque fois que l'utilisateur souhaite interroger des données de blockchain à partir de la ligne de commande, diffuser des paires DEX ou l'activité du portefeuille dans un terminal, configurer GoldRush en tant que fournisseur d'outils MCP ou exécuter des requêtes ponctuelles rapides sans écrire de code (par exemple, « vérifier le solde d'un portefeuille », « quel est le prix de l'essence », « rechercher un jeton »). Utilisez-le également lorsque l'utilisateur mentionne les commandes CLI « goldrush », « npx @covalenthq/goldrush-cli » ou l'intégration MCP avec GoldRush. La CLI est le chemin le plus rapide pour les recherches ad hoc de blockchain à partir du terminal. Si l'utilisateur a besoin d'un accès programmatique à l'API dans une application, utilisez plutôt goldrush-foundational-api ou goldrush-streaming-api. Si l'utilisateur a besoin d'un accès au paiement par demande sans clé API, utilisez plutôt goldrush-x402.
