GoldRush CLI — 终端优先的区块链数据工具，支持 Claude Desktop 和 Claude Code 的 MCP。每当用户想要从命令行查询区块链数据、在终端中流式传输 DEX 对或钱包活动、将 GoldRush 设置为 MCP 工具提供商或在不编写代码的情况下运行快速一次性查询（例如，“检查钱包余额”、“汽油价格是多少”、“搜索代币”）时，请使用此技能。当用户提到“goldrush”CLI 命令、“npx @covalenthq/goldrush-cli”或 MCP 与 GoldRush 集成时，也可以使用此命令。 CLI 是从终端进行临时区块链查找的最快路径。如果用户需要在应用程序中进行编程 API 访问，请改用 goldrush-foundational-api 或 goldrush-streaming-api。如果用户需要按请求付费访问而不需要 API 密钥，请改用 goldrush-x402。
