GoldRush CLI — 終端優先的區塊鏈資料工具，支援 Claude Desktop 和 Claude Code 的 MCP。每當用戶想要從命令列查詢區塊鏈資料、在終端中串流 DEX 對或錢包活動、將 GoldRush 設定為 MCP 工具提供者或在不編寫程式碼的情況下執行快速一次性查詢（例如，「檢查錢包餘額」、「汽油價格是多少」、「搜尋代幣」）時，請使用此技能。當使用者提到「goldrush」CLI 指令、「npx @covalenthq/goldrush-cli」或 MCP 與 GoldRush 整合時，也可以使用此指令。 CLI 是從終端進行臨時區塊鏈查找的最快路徑。如果用戶需要在應用程式中進行程式設計 API 訪問，請改用 goldrush-foundational-api 或 goldrush-streaming-api。如果使用者需要按請求付費存取而不需要 API 金鑰，請改用 goldrush-x402。
