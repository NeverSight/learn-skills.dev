GoldRush CLI — Claude Desktop 및 Claude Code에 대한 MCP를 지원하는 터미널 우선 블록체인 데이터 도구입니다. 사용자가 명령줄에서 블록체인 데이터를 쿼리하거나, 터미널에서 DEX 쌍 또는 지갑 활동을 스트리밍하거나, GoldRush를 MCP 도구 공급자로 설정하거나, 코드를 작성하지 않고 빠른 일회성 쿼리(예: '지갑 잔액 확인', '가스 가격은 얼마입니까', '토큰 검색')를 실행하려고 할 때마다 이 기술을 사용하세요. 또한 사용자가 'goldrush' CLI 명령, 'npx @covalenthq/goldrush-cli' 또는 GoldRush와의 MCP 통합을 언급하는 경우에도 이를 사용하십시오. CLI는 터미널에서 임시 블록체인 조회를 위한 가장 빠른 경로입니다. 사용자가 애플리케이션에서 프로그래밍 방식의 API 액세스가 필요한 경우 대신 goldrush-foundational-api 또는 goldrush-streaming-api를 사용하세요. 사용자가 API 키 없이 요청당 지불 액세스가 필요한 경우 대신 goldrush-x402를 사용하세요.
