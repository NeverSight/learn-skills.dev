GoldRush CLI — инструмент обработки данных блокчейна, ориентированный на терминал, с поддержкой MCP для Claude Desktop и Claude Code. Используйте этот навык всякий раз, когда пользователь хочет запросить данные блокчейна из командной строки, транслировать пары DEX или активность кошелька в терминале, настроить GoldRush в качестве поставщика инструментов MCP или выполнить быстрые одноразовые запросы без написания кода (например, «проверить баланс кошелька», «какая цена на газ», «поиск токена»). Также используйте это, когда пользователь упоминает команды CLI «goldrush», «npx @coвалентhq/goldrush-cli» или интеграцию MCP с GoldRush. CLI — это самый быстрый путь для специального поиска блокчейна с терминала. Если пользователю необходим программный доступ к API в приложении, используйте вместо него goldrush-foundational-api или goldrush-streaming-api. Если пользователю нужен доступ с оплатой за запрос без ключа API, вместо этого используйте goldrush-x402.
