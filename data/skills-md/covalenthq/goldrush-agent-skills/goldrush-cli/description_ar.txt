GoldRush CLI - أداة بيانات blockchain الأولى مع دعم MCP لـ Claude Desktop وClaude Code. استخدم هذه المهارة عندما يريد المستخدم الاستعلام عن بيانات blockchain من سطر الأوامر، أو دفق أزواج DEX أو نشاط المحفظة في محطة طرفية، أو إعداد GoldRush كموفر أداة MCP، أو تشغيل استعلامات سريعة لمرة واحدة دون كتابة التعليمات البرمجية (على سبيل المثال، "التحقق من رصيد المحفظة"، "ما هو سعر الغاز"، "البحث عن رمز مميز"). استخدم هذا أيضًا عندما يذكر المستخدم أوامر CLI "goldrush" أو "npx @covalenthq/goldrush-cli" أو تكامل MCP مع GoldRush. يعد CLI هو أسرع مسار لعمليات البحث المخصصة عن blockchain من المحطة. إذا كان المستخدم يحتاج إلى الوصول إلى واجهة برمجة التطبيقات (API) في أحد التطبيقات، فاستخدم goldrush-foundational-api أو goldrush-streaming-api بدلاً من ذلك. إذا كان المستخدم يحتاج إلى الوصول إلى الدفع لكل طلب بدون مفتاح API، فاستخدم goldrush-x402 بدلاً من ذلك.
