API fundamental GoldRush: API REST para datos de blockchain históricos y casi en tiempo real en más de 100 cadenas. Utilice esta habilidad siempre que el usuario necesite saldos de tokens de billetera, historial de transacciones, tenencias de NFT, precios de tokens, aprobaciones de tokens, actividad entre cadenas, datos de bloques, seguimiento del valor de la cartera o cualquier consulta de datos en cadena a través de REST. Esta es la habilidad predeterminada para búsquedas de datos de blockchain, paneles de cartera, herramientas fiscales, verificaciones de cumplimiento, exploradores de bloques y cualquier aplicación que obtenga datos de cadena históricos o actuales. Si el usuario necesita transmisión en tiempo real o datos push de WebSocket, use goldrush-streaming-api en su lugar. Si el usuario necesita acceso de pago por solicitud sin una clave API, utilice goldrush-x402 en su lugar.
