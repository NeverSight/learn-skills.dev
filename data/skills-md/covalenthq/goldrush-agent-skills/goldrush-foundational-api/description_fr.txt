API fondamentale GoldRush — API REST pour les données blockchain historiques et en temps quasi réel sur plus de 100 chaînes. Utilisez cette compétence chaque fois que l'utilisateur a besoin des soldes de jetons de portefeuille, de l'historique des transactions, des avoirs NFT, des prix des jetons, des approbations de jetons, de l'activité inter-chaînes, des données de bloc, du suivi de la valeur du portefeuille ou de toute requête de données en chaîne via REST. Il s'agit de la compétence par défaut pour les recherches de données blockchain, les tableaux de bord de portefeuille, les outils fiscaux, les contrôles de conformité, les explorateurs de blocs et toute application qui récupère les données de chaîne historiques ou actuelles. Si l'utilisateur a besoin d'un streaming en temps réel ou de données push WebSocket, utilisez plutôt goldrush-streaming-api. Si l'utilisateur a besoin d'un accès au paiement par demande sans clé API, utilisez plutôt goldrush-x402.
