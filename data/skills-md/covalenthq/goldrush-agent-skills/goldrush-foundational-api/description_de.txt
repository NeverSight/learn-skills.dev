GoldRush Foundational API – REST-API für historische und nahezu Echtzeit-Blockchain-Daten über 100 Ketten. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer Wallet-Token-Guthaben, Transaktionsverlauf, NFT-Bestände, Token-Preise, Token-Genehmigungen, kettenübergreifende Aktivitäten, Blockdaten, Portfoliowertverfolgung oder eine beliebige On-Chain-Datenabfrage über REST benötigt. Dies ist die Standardfähigkeit für Blockchain-Datensuchen, Portfolio-Dashboards, Steuertools, Compliance-Prüfungen, Block-Explorer und jede Anwendung, die historische oder aktuelle Kettendaten abruft. Wenn der Benutzer Echtzeit-Streaming oder WebSocket-Push-Daten benötigt, verwenden Sie stattdessen goldrush-streaming-api. Wenn der Benutzer einen Pay-per-Request-Zugriff ohne API-Schlüssel benötigt, verwenden Sie stattdessen goldrush-x402.
