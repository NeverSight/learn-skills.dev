GoldRush Foundational API — 100 以上のチェーンにわたる履歴およびほぼリアルタイムのブロックチェーン データ用の REST API。ユーザーがウォレットトークン残高、取引履歴、NFT保有量、トークン価格、トークン承認、クロスチェーンアクティビティ、ブロックデータ、ポートフォリオ価値追跡、またはREST経由のオンチェーンデータクエリを必要とするときはいつでも、このスキルを使用します。これは、ブロックチェーン データ ルックアップ、ポートフォリオ ダッシュボード、税務ツール、コンプライアンス チェック、ブロック エクスプローラー、および過去または現在のチェーン データを取得するアプリケーションのデフォルト スキルです。ユーザーがリアルタイム ストリーミングまたは WebSocket プッシュ データを必要とする場合は、代わりに goldrush-streaming-api を使用してください。ユーザーが API キーを使用せずにペイパーリクエストアクセスを必要とする場合は、代わりに goldrush-x402 を使用してください。
