GoldRush 기초 API — 100개 이상의 체인에 걸쳐 과거 및 실시간에 가까운 블록체인 데이터를 위한 REST API입니다. 사용자가 지갑 토큰 잔액, 거래 내역, NFT 보유, 토큰 가격, 토큰 승인, 크로스체인 활동, 블록 데이터, 포트폴리오 가치 추적 또는 REST를 통한 온체인 데이터 쿼리가 필요할 때마다 이 기술을 사용하세요. 이는 블록체인 데이터 조회, 포트폴리오 대시보드, 세금 도구, 규정 준수 확인, 블록 탐색기 및 과거 또는 현재 체인 데이터를 가져오는 모든 애플리케이션을 위한 기본 기술입니다. 사용자에게 실시간 스트리밍 또는 WebSocket 푸시 데이터가 필요한 경우 대신 goldrush-streaming-api를 사용하세요. 사용자가 API 키 없이 요청당 지불 액세스가 필요한 경우 대신 goldrush-x402를 사용하세요.
