GoldRush Foundational API — REST API لبيانات blockchain التاريخية وفي الوقت الفعلي تقريبًا عبر أكثر من 100 سلسلة. استخدم هذه المهارة عندما يحتاج المستخدم إلى أرصدة الرموز المميزة للمحفظة، أو سجل المعاملات، أو مقتنيات NFT، أو أسعار الرموز المميزة، أو الموافقات الرمزية، أو النشاط عبر السلسلة، أو بيانات الكتلة، أو تتبع قيمة المحفظة، أو أي استعلام بيانات على السلسلة عبر REST. هذه هي المهارة الافتراضية لعمليات البحث عن بيانات blockchain، ولوحات معلومات المحفظة، والأدوات الضريبية، وفحوصات الامتثال، ومستكشفات الكتل، وأي تطبيق يجلب بيانات السلسلة التاريخية أو الحالية. إذا كان المستخدم يحتاج إلى البث في الوقت الفعلي أو دفع بيانات WebSocket، فاستخدم goldrush-streaming-api بدلاً من ذلك. إذا كان المستخدم يحتاج إلى الوصول إلى الدفع لكل طلب بدون مفتاح API، فاستخدم goldrush-x402 بدلاً من ذلك.
