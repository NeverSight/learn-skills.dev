GoldRush Foundational API — REST API для исторических данных и данных блокчейна, близких к реальному времени, в более чем 100 цепочках. Используйте этот навык всякий раз, когда пользователю нужны балансы токенов кошелька, история транзакций, запасы NFT, цены токенов, утверждения токенов, межсетевая активность, данные блоков, отслеживание стоимости портфеля или любой запрос данных в цепочке через REST. Это навык по умолчанию для поиска данных блокчейна, информационных панелей портфеля, налоговых инструментов, проверок соответствия, обозревателей блоков и любого приложения, которое извлекает исторические или текущие данные цепочки. Если пользователю нужна потоковая передача в реальном времени или push-данные WebSocket, вместо этого используйте goldrush-streaming-api. Если пользователю нужен доступ с оплатой за запрос без ключа API, вместо этого используйте goldrush-x402.
