GoldRush Foundational API: API REST per dati blockchain storici e quasi in tempo reale su oltre 100 catene. Utilizza questa abilità ogni volta che l'utente ha bisogno dei saldi dei token del portafoglio, della cronologia delle transazioni, delle partecipazioni NFT, dei prezzi dei token, delle approvazioni dei token, dell'attività cross-chain, dei dati sui blocchi, del monitoraggio del valore del portafoglio o di qualsiasi query di dati sulla catena tramite REST. Questa è la competenza predefinita per le ricerche di dati blockchain, dashboard di portafoglio, strumenti fiscali, controlli di conformità, block explorer e qualsiasi applicazione che recuperi dati storici o attuali della catena. Se l'utente necessita di streaming in tempo reale o dati push WebSocket, utilizza invece goldrush-streaming-api. Se l'utente necessita di un accesso con pagamento per richiesta senza una chiave API, utilizza invece goldrush-x402.
