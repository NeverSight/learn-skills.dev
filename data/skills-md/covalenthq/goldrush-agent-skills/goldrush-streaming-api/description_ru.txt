GoldRush Streaming API — данные блокчейна в реальном времени через подписки GraphQL через WebSocket. Используйте этот навык всякий раз, когда пользователю нужны потоки цен в реальном времени (свечи OHLCV), мониторинг пар DEX в реальном времени (новые пары, обновления ликвидности), потоковая передача активности кошелька, декодированные события обмена/перевода, поиск токенов, анализ PnL трейдера или любое событие блокчейна с задержкой менее секунды. Это правильный навык для торговых ботов, живых информационных панелей, систем оповещения, копи-трейдинга, снайпинга DEX и аналитики в реальном времени. Также охватывает одноразовые запросы GraphQL для обнаружения токенов и анализа прибыльности. Если пользователю нужны исторические данные, пакетные запросы или результаты REST с разбивкой на страницы, вместо этого используйте goldrush-foundational-api. Если пользователю нужен доступ с оплатой за запрос без ключа API, вместо этого используйте goldrush-x402.
