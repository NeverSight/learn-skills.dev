GoldRush ストリーミング API — WebSocket 経由の GraphQL サブスクリプションを介したリアルタイムのブロックチェーン データ。ユーザーがライブ価格フィード (OHLCV キャンドル)、リアルタイムの DEX ペア監視 (新規ペア、流動性更新)、ウォレットアクティビティストリーミング、デコードされたスワップ/転送イベント、トークン検索、トレーダー損益分析、または 1 秒未満のレイテンシーブロックチェーンイベントプッシュを必要とする場合は常に、このスキルを使用します。これは、取引ボット、ライブ ダッシュボード、アラート システム、コピー取引、DEX スナイピング、およびリアルタイム分析に適したスキルです。トークン検出と収益性分析のためのワンタイム GraphQL クエリもカバーします。ユーザーが履歴データ、バッチ クエリ、またはページ分割された REST 結果を必要とする場合は、代わりに goldrush-foundational-api を使用してください。ユーザーが API キーを使用せずにペイパーリクエストアクセスを必要とする場合は、代わりに goldrush-x402 を使用してください。
