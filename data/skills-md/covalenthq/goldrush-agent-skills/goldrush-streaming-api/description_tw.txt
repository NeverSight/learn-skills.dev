GoldRush Streaming API — 透過 WebSocket 上的 GraphQL 訂閱提供即時區塊鏈資料。每當用戶需要即時價格資訊（OHLCV 蠟燭）、即時 DEX 貨幣對監控（新貨幣對、流動性更新）、錢包活動串流、解碼交換/轉移事件、代幣搜尋、交易者損益分析或任何亞秒延遲區塊鏈事件推送時，請使用此技能。這是交易機器人、即時儀表板、警報系統、複製交易、DEX 狙擊和即時分析的正確技能。也涵蓋用於代幣發現和盈利能力分析的一次性 GraphQL 查詢。如果使用者需要歷史資料、批次查詢或分頁 REST 結果，請改用 goldrush-foundational-api。如果使用者需要按請求付費存取而不需要 API 金鑰，請改用 goldrush-x402。
