API GoldRush Streaming: datos de blockchain en tiempo real a través de suscripciones GraphQL a través de WebSocket. Utilice esta habilidad siempre que el usuario necesite información de precios en vivo (velas OHLCV), monitoreo de pares DEX en tiempo real (nuevos pares, actualizaciones de liquidez), transmisión de actividad de billetera, eventos de intercambio/transferencia decodificados, búsqueda de tokens, análisis PnL de comerciantes o cualquier impulso de evento de blockchain de latencia inferior a un segundo. Esta es la habilidad adecuada para bots comerciales, paneles de control en vivo, sistemas de alerta, comercio de copias, francotiradores DEX y análisis en tiempo real. También cubre consultas GraphQL únicas para el descubrimiento de tokens y el análisis de rentabilidad. Si el usuario necesita datos históricos, consultas por lotes o resultados REST paginados, utilice goldrush-foundational-api en su lugar. Si el usuario necesita acceso de pago por solicitud sin una clave API, utilice goldrush-x402 en su lugar.
