GoldRush Streaming API — 通过 WebSocket 上的 GraphQL 订阅提供实时区块链数据。每当用户需要实时价格信息（OHLCV 蜡烛）、实时 DEX 货币对监控（新货币对、流动性更新）、钱包活动流、解码交换/转移事件、代币搜索、交易者损益分析或任何亚秒级延迟区块链事件推送时，请使用此技能。这是交易机器人、实时仪表板、警报系统、复制交易、DEX 狙击和实时分析的正确技能。还涵盖用于代币发现和盈利能力分析的一次性 GraphQL 查询。如果用户需要历史数据、批量查询或分页 REST 结果，请改用 goldrush-foundational-api。如果用户需要按请求付费访问而不需要 API 密钥，请改用 goldrush-x402。
