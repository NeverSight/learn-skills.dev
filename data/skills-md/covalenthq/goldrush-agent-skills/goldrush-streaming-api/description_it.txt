GoldRush Streaming API: dati blockchain in tempo reale tramite abbonamenti GraphQL su WebSocket. Utilizza questa abilità ogni volta che l'utente ha bisogno di feed sui prezzi in tempo reale (candele OHLCV), monitoraggio delle coppie DEX in tempo reale (nuove coppie, aggiornamenti di liquidità), streaming dell'attività del portafoglio, eventi di swap/trasferimento decodificati, ricerca di token, analisi PnL dei trader o qualsiasi push di eventi blockchain con latenza inferiore al secondo. Questa è la competenza giusta per il trading di bot, dashboard live, sistemi di avviso, copy trading, sniping DEX e analisi in tempo reale. Copre anche le query GraphQL una tantum per la scoperta di token e l'analisi della redditività. Se l'utente ha bisogno di dati storici, query batch o risultati REST impaginati, utilizza invece goldrush-foundational-api. Se l'utente necessita di un accesso con pagamento per richiesta senza una chiave API, utilizza invece goldrush-x402.
