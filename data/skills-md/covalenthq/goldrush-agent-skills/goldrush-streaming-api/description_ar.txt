GoldRush Streaming API — بيانات blockchain في الوقت الفعلي عبر اشتراكات GraphQL عبر WebSocket. استخدم هذه المهارة عندما يحتاج المستخدم إلى موجزات الأسعار المباشرة (شموع OHLCV)، ومراقبة زوج DEX في الوقت الفعلي (أزواج جديدة، وتحديثات السيولة)، وتدفق نشاط المحفظة، وأحداث المبادلة/التحويل التي تم فك تشفيرها، والبحث عن الرمز المميز، وتحليل PnL للمتداول، أو أي حدث دفع لحدث blockchain بزمن انتقال أقل من الثانية. هذه هي المهارة المناسبة لروبوتات التداول ولوحات المعلومات المباشرة وأنظمة التنبيه ونسخ التداول وقنص DEX والتحليلات في الوقت الفعلي. يغطي أيضًا استعلامات GraphQL لمرة واحدة لاكتشاف الرمز المميز وتحليل الربحية. إذا كان المستخدم يحتاج إلى بيانات تاريخية، أو استعلامات مجمعة، أو نتائج REST مرقّمة، فاستخدم goldrush-foundational-api بدلاً من ذلك. إذا كان المستخدم يحتاج إلى الوصول إلى الدفع لكل طلب بدون مفتاح API، فاستخدم goldrush-x402 بدلاً من ذلك.
