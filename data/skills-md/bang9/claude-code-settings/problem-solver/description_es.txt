Flujo de trabajo sistemático de resolución de problemas para corrección de errores e implementación de funciones. Úselo al depurar problemas, corregir errores, implementar funciones o cuando el usuario describe comportamientos inesperados, errores o necesita un enfoque metódico para los cambios de código. Garantiza una verificación reproducible y valida que las soluciones sean soluciones fundamentales en lugar de soluciones alternativas.
