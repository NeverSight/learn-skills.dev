سير عمل منهجي لحل المشكلات لإصلاح الأخطاء وتنفيذ الميزات. يُستخدم عند تصحيح الأخطاء، أو إصلاح الأخطاء، أو تنفيذ الميزات، أو عندما يصف المستخدم سلوكًا غير متوقع، أو أخطاء، أو يحتاج إلى أسلوب منهجي لتغيير التعليمات البرمجية. يضمن أن التحقق القابل للتكرار والتحقق من صحة الحلول عبارة عن إصلاحات أساسية وليست حلول بديلة.
