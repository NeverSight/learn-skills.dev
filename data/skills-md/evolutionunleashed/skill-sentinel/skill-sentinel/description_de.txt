Sicherheitsscanner und Bedrohungsanalysator für KI-Agentenfähigkeiten. Aktivieren Sie diesen Skill immer dann, wenn dem Arbeitsbereich ein neuer Skill hinzugefügt wird, wenn der Benutzer einen Skill aus einer externen Quelle importiert oder installiert, wenn er aufgefordert wird, einen vorhandenen Skill auf Sicherheit zu prüfen oder zu überprüfen, oder wenn der Benutzer das Scannen, Überprüfen, Überprüfen oder Validieren eines Skills vor der Verwendung erwähnt. Wird auch ausgelöst, wenn der Benutzer für eine SKILL.md-Datei oder ein Skill-Verzeichnis auf Skill-Sicherheit, Erkennung bösartiger Codes, sofortiges Injektionsscannen oder Lieferkettensicherheit verweist. Dieser Skill liest alle Dateien in einem Ziel-Skill-Ordner und erstellt eine strukturierte Bedrohungsbewertung mit Schweregradbewertungen und umsetzbaren Empfehlungen.
