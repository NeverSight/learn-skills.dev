الماسح الضوئي الأمني ​​ومحلل التهديدات لمهارات وكيل الذكاء الاصطناعي. قم بتنشيط هذه المهارة كلما تمت إضافة مهارة جديدة إلى مساحة العمل، عندما يقوم المستخدم باستيراد أو تثبيت مهارة من مصدر خارجي، عندما يُطلب منه تدقيق أو مراجعة مهارة موجودة للسلامة، أو عندما يذكر المستخدم مسح المهارة أو فحصها أو فحصها أو التحقق من صحتها قبل الاستخدام. يتم التشغيل أيضًا عندما يشير المستخدم إلى أمان المهارات، أو اكتشاف التعليمات البرمجية الضارة، أو المسح السريع للحقن، أو سلامة سلسلة التوريد لأي ملف SKILL.md أو دليل المهارات. تقرأ هذه المهارة جميع الملفات الموجودة في مجلد المهارات المستهدفة وتنتج تقييمًا منظمًا للتهديدات مع تقييمات الخطورة وتوصيات قابلة للتنفيذ.
