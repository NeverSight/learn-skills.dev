Escáner de seguridad y analizador de amenazas para las habilidades de los agentes de IA. Active esta habilidad siempre que se agregue una nueva habilidad al espacio de trabajo, cuando el usuario importe o instale una habilidad desde una fuente externa, cuando se le solicite auditar o revisar una habilidad existente por motivos de seguridad, o cuando el usuario mencione escanear, examinar, verificar o validar una habilidad antes de usarla. También se activa cuando el usuario hace referencia a la seguridad de habilidades, la detección de códigos maliciosos, el escaneo de inyección rápida o la seguridad de la cadena de suministro para cualquier archivo SKILL.md o directorio de habilidades. Esta habilidad lee todos los archivos dentro de una carpeta de habilidades de destino y produce una evaluación de amenazas estructurada con clasificaciones de gravedad y recomendaciones prácticas.
