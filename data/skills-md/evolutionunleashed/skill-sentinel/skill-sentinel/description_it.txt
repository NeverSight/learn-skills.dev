Scanner di sicurezza e analizzatore di minacce per le competenze degli agenti IA. Attiva questa competenza ogni volta che una nuova competenza viene aggiunta all'area di lavoro, quando l'utente importa o installa una competenza da una fonte esterna, quando viene chiesto di verificare o rivedere una competenza esistente per motivi di sicurezza o quando l'utente menziona la scansione, il controllo, il controllo o la convalida di una competenza prima dell'uso. Si attiva anche quando l'utente fa riferimento alla sicurezza delle competenze, al rilevamento di codici dannosi, alla scansione di prompt injection o alla sicurezza della catena di fornitura per qualsiasi file SKILL.md o directory delle competenze. Questa competenza legge tutti i file all'interno di una cartella della competenza di destinazione e produce una valutazione strutturata della minaccia con valutazioni di gravità e consigli attuabili.
