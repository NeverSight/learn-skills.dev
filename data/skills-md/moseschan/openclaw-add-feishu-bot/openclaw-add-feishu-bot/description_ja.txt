新しい Feishu ボット アカウントを openclaw に追加し、それを新しいエージェントに接続します。ユーザーが Feishu ボットを追加する場合、openclaw に新しい Feishu アプリを登録する場合、2 つ目 (または 3 つ目など) の Feishu アカウントを設定する場合、または openclaw で新しい Feishu アプリ ID とアプリ シークレットのペアを設定する場合は、このスキルを使用します。 「add feishu bot」、「新增飞书 bot」、「add feishu account」、「register feishu app」、「接入飞书」などのフレーズ、またはユーザーが Feishu appId + appSecret のペアを提供し、それを openclaw に接続したいときにトリガーします。
