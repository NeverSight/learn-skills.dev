Erstellen und prüfen Sie MECE-Problembäume auf Beratungsniveau durch strukturierte Zerlegung. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine MECE-Analyse, Problembäume, eine Ursachenzerlegung, eine treiberbasierte Problemstrukturierung oder eine strategische Aufschlüsselung komplexer Fragen anfordert. Wird auch ausgelöst, wenn der Benutzer einen vorhandenen Baum auf Überlappungen, Lücken oder strukturelle Probleme prüfen muss. Deckt sowohl den Entwurfsmodus (neue Bäume von Grund auf) als auch den Überprüfungsmodus (Verbesserung vorhandener Bäume) ab.
