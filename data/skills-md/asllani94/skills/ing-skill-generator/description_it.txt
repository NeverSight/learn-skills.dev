Competenze esperte per generare competenze GitHub Copilot da repository di documentazione interni a ING. Utilizza questa competenza quando ti viene chiesto di creare una competenza da qualsiasi repository di documentazione come codice ING, generare una competenza di knowledge base per un framework ING, convertire la documentazione dello strumento ING in una competenza Copilot o trasformare qualsiasi documento/cartella in un file di competenza esperto. Si attiva anche quando l'utente menziona "abilità da documenti", "genera abilità", "crea abilità da repository" o fa riferimento a framework interni a ING come Baker, Merak, Kingsroad o simili. Include un quadro di valutazione, agenti di valutazione e strumenti di benchmark per testare le competenze generate.
