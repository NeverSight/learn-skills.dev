Expertenfähigkeit zum Generieren von GitHub Copilot-Kenntnissen aus ING-internen Dokumentationsrepositorys. Verwenden Sie diesen Skill, wenn Sie aufgefordert werden, einen Skill aus einem beliebigen ING-Dokumentations-als-Code-Repository zu erstellen, einen Wissensdatenbank-Skill für ein ING-Framework zu generieren, die ING-Tool-Dokumentation in einen Copilot-Skill umzuwandeln oder beliebige Dokumente/Ordner in eine Experten-Skill-Datei umzuwandeln. Wird auch ausgelöst, wenn der Benutzer „Skill aus Dokumenten“, „Skill generieren“, „Skill aus Repo erstellen“ erwähnt oder auf ING-interne Frameworks wie Baker, Merak, Kingsroad oder ähnliches verweist. Beinhaltet einen Bewertungsrahmen, Bewertungsagenten und Benchmark-Tools zum Testen der generierten Fähigkeiten.
