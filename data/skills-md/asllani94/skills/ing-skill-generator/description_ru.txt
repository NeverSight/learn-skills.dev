Экспертные навыки создания навыков GitHub Copilot из внутренних репозиториев документации ING. Используйте этот навык, когда вас попросят создать навык из любого репозитория документации ING в виде кода, создать навык базы знаний для платформы ING, преобразовать документацию инструмента ING в навык второго пилота или превратить любую папку docs/ в файл экспертного навыка. Также срабатывает, когда пользователь упоминает «навык из документации», «создать навык», «создать навык из репозитория» или ссылается на внутренние структуры ING, такие как Baker, Merak, Kingsroad или аналогичные. Включает систему оценки, агенты оценивания и контрольные инструменты для тестирования сформированных навыков.
