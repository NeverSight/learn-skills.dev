Habilidad experta para generar habilidades de GitHub Copilot a partir de repositorios de documentación internos de ING. Utilice esta habilidad cuando se le solicite crear una habilidad a partir de cualquier repositorio de documentación como código de ING, generar una habilidad de base de conocimientos para un marco de ING, convertir la documentación de la herramienta ING en una habilidad Copilot o convertir cualquier documento/carpeta en un archivo de habilidad experto. También se activa cuando el usuario menciona "habilidad de documentos", "generar habilidad", "crear habilidad desde repositorio" o hace referencia a marcos internos de ING como Baker, Merak, Kingsroad o similares. Incluye un marco de evaluación, agentes de calificación y herramientas de referencia para evaluar las habilidades generadas.
