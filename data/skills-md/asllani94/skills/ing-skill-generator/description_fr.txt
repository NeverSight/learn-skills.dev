Compétence d'expert pour générer des compétences GitHub Copilot à partir des référentiels de documentation internes d'ING. Utilisez cette compétence lorsqu'on vous demande de créer une compétence à partir de n'importe quel référentiel de documentation en tant que code ING, de générer une compétence de base de connaissances pour un framework ING, de convertir la documentation de l'outil ING en compétence Copilot ou de transformer n'importe quel document/dossier en un fichier de compétence expert. Se déclenche également lorsque l'utilisateur mentionne « compétence à partir de documents », « générer une compétence », « créer une compétence à partir d'un dépôt » ou fait référence à des frameworks internes à ING comme Baker, Merak, Kingsroad ou similaire. Comprend un cadre d'évaluation, des agents de notation et des outils de référence pour tester les compétences générées.
