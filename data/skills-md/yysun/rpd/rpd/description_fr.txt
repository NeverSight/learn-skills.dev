Utilisez cette compétence lorsque (1) l'intention de l'utilisateur est d'architecturer, de concevoir ou de développer un logiciel, et (2) le message contient n'importe quel mot-clé de commande RPD n'importe où (insensible à la casse), tel que RPD, REQ, AP, AR, AT, SS, CC, DF, DD, ET, TT, CR, GC, WT, !!. Le(s) mot(s)-clé(s) détecté(s) déclenchent les règles de flux de travail/comportements d'étape correspondants ci-dessous. Ignorez les mots-clés qui apparaissent uniquement à l'intérieur de blocs de code isolés ou de code en ligne, sauf si l'utilisateur indique explicitement qu'il invoque le mot-clé.
