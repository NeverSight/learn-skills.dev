Converti una base di codice esistente nella directory di lavoro corrente in una directory di attività ShinkaEvolve eseguendo uno snapshot del codice pertinente, aggiungendo blocchi di evoluzione e generando `evaluate.py` più file runner/config Shinka. Da utilizzare quando l'utente desidera ottimizzare il codice esistente con Shinka invece di creare un'attività nuova di zecca da una descrizione in linguaggio naturale.
