Exécutez les tâches ShinkaEvolve existantes avec la CLI `shinka_run` à partir d'un répertoire de tâches (`evaluate.py` + `initial.<ext>`). À utiliser lorsqu'un agent doit lancer une évolution asynchrone qui s'exécute rapidement avec `--results_dir` requis, le nombre de générations et les remplacements de mots clés à espace de noms strict.
