Запускайте существующие задачи ShinkaEvolve с помощью CLI `shinka_run` из каталога задач (`evaluate.py` + `initial.<ext>`). Используйте, когда агенту необходимо запустить асинхронную эволюцию, которая выполняется быстро с обязательным `--results_dir`, счетчиком поколений и строгими переопределениями ключевых слов в пространстве имен.
