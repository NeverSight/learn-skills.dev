Создайте заголовок и описание запроса на включение, проанализировав разницу текущей ветки с базовой веткой. Используйте этот навык всякий раз, когда пользователь хочет создать описание PR, заполнить шаблон PR, написать детали PR, сгенерировать запрос на включение, открыть запрос на включение, обновить описание PR или спросить что-либо, связанное с описаниями, названиями или шаблонами запросов на включение — даже если он просто говорит «написать PR», «опишите мои изменения», «открыть запрос на включение» или «обновить PR».
