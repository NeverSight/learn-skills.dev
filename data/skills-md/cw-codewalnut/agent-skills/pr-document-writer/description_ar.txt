قم بإنشاء عنوان ووصف طلب السحب من خلال تحليل فرق الفرع الحالي مقابل الفرع الأساسي. استخدم هذه المهارة عندما يريد المستخدم إنشاء وصف العلاقات العامة، أو ملء قالب العلاقات العامة، أو كتابة تفاصيل العلاقات العامة، أو إنشاء طلب سحب، أو فتح العلاقات العامة، أو تحديث وصف العلاقات العامة، أو طرح أي شيء متعلق بأوصاف طلبات السحب، أو العناوين، أو القوالب - حتى لو قالوا فقط "اكتب علاقات عامة" أو "صف تغييراتي" أو "افتح طلب سحب" أو "قم بتحديث العلاقات العامة".
