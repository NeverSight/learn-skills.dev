الاستعلام عن البيانات والتقييمات الخاصة بالملكية عبر واجهات برمجة تطبيقات المكافآت. يُستخدم عندما يسأل المستخدم عن مقاييس التوقيع المساحي، أو معدلات المكافآت، أو المدققين، أو مقدمي الخدمة، أو تقييمات DeFi/البنية التحتية. مشغلات على "بيانات التخزين"، و"معدل المكافأة"، و"واجهة برمجة تطبيقات مكافآت التخزين"، و"مقاييس التحقق"، و"بيانات الموفر"، و"تقييمات التمويل اللامركزي"، و"تقييمات البنية التحتية".
