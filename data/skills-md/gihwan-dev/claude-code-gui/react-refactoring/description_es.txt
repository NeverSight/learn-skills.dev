Habilidades críticas de análisis y ejecución para la refactorización de código React/TypeScript.  
Cuando el usuario proporciona una ruta de componente y sus problemas, cada problema se revisa críticamente desde una perspectiva de "código limpio", se aplica el pensamiento secuencial para analizar paso a paso y se crea y ejecuta un plan de refactorización.  

Activador: se utiliza cuando palabras clave como "refactorización", "quiero mejorar", "cambio de estructura de código", "separación de ganchos", "cohesión", "acoplamiento", etc. se mencionan junto con la ruta de un componente de React.
