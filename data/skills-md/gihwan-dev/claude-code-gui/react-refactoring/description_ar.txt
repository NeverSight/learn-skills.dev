مهارات التحليل النقدي والتنفيذ لإعادة بناء كود React/TypeScript.  
عندما يوفر المستخدم مسار مكون ومشاكله، تتم مراجعة كل مشكلة بشكل نقدي من منظور "الكود النظيف"، ويتم تطبيق التفكير المتسلسل لتحليل خطوة بخطوة، ويتم إنشاء خطة إعادة البناء وتنفيذها.  

المشغل: يُستخدم عندما يتم ذكر كلمات رئيسية مثل "إعادة البناء"، و"الرغبة في التحسين"، و"تغيير بنية التعليمات البرمجية"، و"فصل الخطاف"، و"التماسك"، و"الاقتران"، وما إلى ذلك، جنبًا إلى جنب مع مسار مكون React.
