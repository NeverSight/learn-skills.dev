Compétences critiques en matière d'analyse et d'exécution pour la refactorisation de code React/TypeScript.  
Lorsque l'utilisateur fournit un chemin de composant et ses problèmes, chaque problème est examiné de manière critique du point de vue du « code propre », la pensée séquentielle est appliquée pour analyser étape par étape, et un plan de refactoring est créé et exécuté.  

Déclencheur : utilisé lorsque des mots-clés tels que « refactoring », « vouloir améliorer », « changement de structure de code », « séparation de hook », « cohésion », « couplage », etc. sont mentionnés avec un chemin de composant React.
