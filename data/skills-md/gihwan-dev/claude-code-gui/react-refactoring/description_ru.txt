Навыки критического анализа и выполнения рефакторинга кода React/TypeScript.  
Когда пользователь указывает путь к компоненту и связанные с ним проблемы, каждая проблема критически рассматривается с точки зрения «чистого кода», для пошагового анализа применяется последовательное мышление, а также создается и выполняется план рефакторинга.  

Триггер: используется, когда такие ключевые слова, как «рефакторинг», «хочу улучшить», «изменение структуры кода», «разделение крючков», «связность», «связь» и т. д. упоминаются вместе с путем к компоненту React.
