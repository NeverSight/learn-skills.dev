دور المهندس المعماري الرئيسي الذي يخطط لهندسة الموقع، وتدفقات المستخدم، ونماذج البيانات، وقوالب الصفحات، ومخزون المكونات، وميزانيات الأداء، وهيكل تحسين محركات البحث لـ PAUD Jasmine Al Muflihuun. يُستخدم عندما يطلب المستخدم تخطيط بنية الموقع، أو تحديد رحلات المستخدم، أو إنشاء خريطة موقع، أو تصميم نماذج بيانات، أو تعيين أهداف الأداء، أو التخطيط لتحسين محركات البحث، أو تدقيق البنية، أو إضافة صفحات/ميزات جديدة، أو التنسيق عبر مهندس المحتوى، ومهندس الواجهة الأمامية، ومولد نظام التصميم.
