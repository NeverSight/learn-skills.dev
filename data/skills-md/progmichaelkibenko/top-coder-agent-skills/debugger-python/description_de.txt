Debuggen Sie Python-Anwendungen zur Laufzeit mithilfe von DAP-Haltepunkten und Variablenprüfung. Verwenden Sie diese Option, wenn der Benutzer einen Laufzeitfehler in Python, einen stillen Fehler, unerwartete Variablenwerte, einen falschen Datenfluss meldet oder wenn das Drucken-Debuggen unzureichend ist. Verwenden Sie es auch proaktiv, wenn Sie auf einen Fehler stoßen, den Sie nicht allein durch statische Analyse diagnostizieren können.
