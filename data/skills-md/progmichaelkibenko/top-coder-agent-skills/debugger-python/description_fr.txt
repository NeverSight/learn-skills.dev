Déboguez les applications Python au moment de l'exécution à l'aide des points d'arrêt DAP et de l'inspection des variables. À utiliser lorsque l'utilisateur signale un bogue d'exécution dans Python, une panne silencieuse, des valeurs de variables inattendues, un flux de données incorrect ou lorsque le débogage d'impression est insuffisant. Utilisez également de manière proactive lorsque vous rencontrez un bug que vous ne pouvez pas diagnostiquer à partir de la seule analyse statique.
