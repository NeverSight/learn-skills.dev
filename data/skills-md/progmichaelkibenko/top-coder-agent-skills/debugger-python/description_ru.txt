Отлаживайте приложения Python во время выполнения, используя точки останова DAP и проверку переменных. Используйте, когда пользователь сообщает об ошибке во время выполнения Python, тихом сбое, неожиданных значениях переменных, неправильном потоке данных или когда отладки печати недостаточно. Также используйте его активно, если вы столкнулись с ошибкой, которую невозможно диагностировать только с помощью статического анализа.
