تصحيح أخطاء تطبيقات Python في وقت التشغيل باستخدام نقاط توقف DAP والفحص المتغير. يُستخدم عندما يبلغ المستخدم عن خطأ في وقت التشغيل في Python، أو فشل صامت، أو قيم متغيرة غير متوقعة، أو تدفق بيانات غير صحيح، أو عندما يكون تصحيح أخطاء الطباعة غير كافٍ. استخدم أيضًا بشكل استباقي عندما تواجه خطأً لا يمكنك تشخيصه من خلال التحليل الثابت وحده.
