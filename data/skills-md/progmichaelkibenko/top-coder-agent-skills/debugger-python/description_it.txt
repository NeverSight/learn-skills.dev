Eseguire il debug delle applicazioni Python in fase di runtime utilizzando i punti di interruzione DAP e l'ispezione delle variabili. Da utilizzare quando l'utente segnala un bug di runtime in Python, un errore silenzioso, valori di variabili imprevisti, flusso di dati errato o quando il debug della stampa è insufficiente. Utilizzalo anche in modo proattivo quando riscontri un bug che non puoi diagnosticare solo mediante l'analisi statica.
