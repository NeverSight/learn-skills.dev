Взаимодействуйте с пулами ликвидности Hydrex на Base. Используйте, когда пользователь хочет заблокировать HYDX для получения права голоса, проверить количество голосов для калибровочного голосования, проголосовать за стратегии пула ликвидности, просмотреть информацию о пуле, проверить вес голосов, принять участие в управлении Hydrex, внести одностороннюю ликвидность в автоматически управляемые хранилища для получения доходности Hydrex, потребовать вознаграждения oHYDX от поощрительных кампаний или использовать oHYDX в veHYDX. Использует Bankr для выполнения транзакций.
