Créez une nouvelle tâche avec une prime crypto sur OpenAnt. À utiliser lorsque l'agent ou l'utilisateur souhaite publier un emploi, créer une prime, embaucher quelqu'un, publier un travail ou utiliser l'IA pour analyser une description de tâche. Couvre "créer une tâche", "publier une prime", "embaucher quelqu'un pour", "J'ai besoin de quelqu'un pour", "publier un emploi". Le séquestre de financement est inclus par défaut.
