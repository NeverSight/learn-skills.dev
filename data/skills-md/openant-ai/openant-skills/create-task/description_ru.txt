Создайте новую задачу с крипто-баунти на OpenAnt. Используйте, когда агент или пользователь хочет опубликовать вакансию, назначить вознаграждение, нанять кого-то, опубликовать работу или использовать ИИ для анализа описания задачи. Обложки «создать задачу», «объявить награду», «нанять кого-то», «мне нужно, чтобы кто-то», «опубликовать вакансию». Эскроу финансирования включен по умолчанию.
