قم بإنشاء تقارير تحليلية عميقة لمستودعات GitHub Trending، مما يوفر رؤى حول الشعبية الشائعة والتطبيقات الفنية والأطر واتجاهات التطبيق المحتملة. يُستخدم عندما يحتاج المستخدم إلى تحليل GitHub Trending اليومي الآلي، أو تقارير عن مستودعات الاتجاه مع تعمق تقني، أو رؤى حول اتجاهات التكنولوجيا الناشئة، أو تقارير يومية مجدولة لمراقبة تغييرات النظام البيئي لـ GitHub.
