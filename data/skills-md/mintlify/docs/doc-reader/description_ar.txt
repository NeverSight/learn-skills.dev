قراءة الوثائق الخارجية والتنقل فيها بكفاءة. يتم الاستدعاء عندما تتطلب المهمة التحقق من كيفية عمل وظيفة معينة أو نقطة نهاية أو خيار تكوين؛ عندما يشير المستخدم إلى واجهة API أو SDK أو مكتبة أو أداة خارجية بالاسم؛ عند ذكر أي عنوان URL لمحرر المستندات أو موقع التوثيق؛ أو عند تنفيذ شيء يعتمد على خدمة أو حزمة خارجية.
