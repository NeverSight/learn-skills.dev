Запускайте и автоматизируйте VS Code (Code OSS) с помощью агента-браузера по протоколу Chrome DevTools. Используйте, когда вам нужно взаимодействовать с пользовательским интерфейсом VS Code, автоматизировать панель чата, тестировать функции пользовательского интерфейса или делать снимки экрана VS Code. Триггеры включают «автоматизировать VS Code», «взаимодействовать с чатом», «проверить пользовательский интерфейс», «сделать снимок экрана», «запустить Code OSS с отладкой».
