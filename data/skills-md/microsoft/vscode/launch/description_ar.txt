قم بتشغيل وأتمتة VS Code (Code OSS) باستخدام متصفح الوكيل عبر بروتوكول Chrome DevTools. استخدمه عندما تحتاج إلى التفاعل مع واجهة مستخدم VS Code، أو أتمتة لوحة الدردشة، أو اختبار ميزات واجهة المستخدم، أو التقاط لقطات شاشة لـ VS Code. تتضمن المشغلات "أتمتة رمز VS"، و"التفاعل مع الدردشة"، و"اختبار واجهة المستخدم"، و"التقاط لقطة شاشة"، و"تشغيل Code OSS مع تصحيح الأخطاء".
