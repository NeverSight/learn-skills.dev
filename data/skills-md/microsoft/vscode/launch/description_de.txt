Starten und automatisieren Sie VS-Code (Code OSS) mithilfe eines Agent-Browsers über das Chrome DevTools-Protokoll. Verwenden Sie es, wenn Sie mit der VS Code-Benutzeroberfläche interagieren, das Chat-Panel automatisieren, UI-Funktionen testen oder Screenshots von VS Code erstellen müssen. Zu den Auslösern gehören „VS-Code automatisieren“, „Mit Chat interagieren“, „Benutzeroberfläche testen“, „Screenshot erstellen“ und „Code OSS mit Debugging starten“.
