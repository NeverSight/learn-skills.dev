Inicie y automatice VS Code (Code OSS) utilizando el navegador-agente a través del protocolo Chrome DevTools. Úselo cuando necesite interactuar con la interfaz de usuario de VS Code, automatizar el panel de chat, probar funciones de la interfaz de usuario o tomar capturas de pantalla de VS Code. Los activadores incluyen "automatizar VS Code", "interactuar con el chat", "probar la interfaz de usuario", "tomar una captura de pantalla", "iniciar Code OSS con depuración".
