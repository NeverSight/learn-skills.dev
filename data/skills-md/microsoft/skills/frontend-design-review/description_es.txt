Revise y cree interfaces frontales distintivas y de calidad de producción con alta calidad de diseño y cumplimiento del sistema de diseño. Evalúa utilizando tres pilares: conocimiento para la acción sin fricciones, artesanía de calidad y construcción confiable. USO PARA: Revisiones de relaciones públicas, revisiones de diseño, auditorías de accesibilidad, comprobaciones de cumplimiento del sistema de diseño, diseño de interfaz creativo, revisión de código de interfaz de usuario, revisiones de componentes, comprobaciones de diseño responsivo, pruebas de temas y creación de una interfaz de usuario memorable. NO UTILIZAR PARA: Revisiones de API de backend, revisiones de esquemas de bases de datos, trabajo de infraestructura o DevOps, lógica empresarial pura sin interfaz de usuario o código que no sea de frontend.
