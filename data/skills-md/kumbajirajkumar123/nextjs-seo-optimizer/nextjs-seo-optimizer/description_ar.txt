مهارة تحسين محركات البحث Next.js الشاملة للبيانات الوصفية، وخرائط الموقع، وrobots.txt، والبيانات المنظمة، وعناصر الويب الحيوية الأساسية. يُستخدم عند إنشاء تطبيقات Next.js أو تدقيقها لتحسين محركات البحث، أو تنفيذ واجهة برمجة تطبيقات البيانات التعريفية، أو إنشاء خرائط الموقع، أو إضافة علامات Open Graph، أو إنشاء بيانات JSON-LD المنظمة، أو تحسين محركات البحث، أو تحسين تصنيفات البحث. يعمل مع جهاز توجيه التطبيقات Next.js 13+ وجهاز توجيه الصفحات.
