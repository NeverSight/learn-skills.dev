Compétence complète d'optimisation SEO Next.js pour les métadonnées, les plans de site, les robots.txt, les données structurées et les Core Web Vitals. À utiliser lors de la création ou de l'audit d'applications Next.js pour le référencement, de la mise en œuvre d'une API de métadonnées, de la génération de plans de site, de l'ajout de balises Open Graph, de la création de données structurées JSON-LD, de l'optimisation pour les moteurs de recherche ou de l'amélioration des classements de recherche. Fonctionne avec le routeur d'applications Next.js 13+ et le routeur de pages.
