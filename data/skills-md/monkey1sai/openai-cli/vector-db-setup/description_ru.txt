Настраивает базы данных векторов для семантического поиска, включая Pinecone, Chroma, pgvector и Qdrant с встроенной генерацией и поиском по сходству. Используйте, когда пользователи запрашивают «базу данных векторов», «семантический поиск», «хранилище вложений», «настройку шишки» или «поиск по сходству».
