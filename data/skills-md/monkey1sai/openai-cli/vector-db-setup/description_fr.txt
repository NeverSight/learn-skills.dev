Configure des bases de données vectorielles pour la recherche sémantique, notamment Pinecone, Chroma, pgvector et Qdrant avec génération d'intégration et recherche de similarité. À utiliser lorsque les utilisateurs demandent une « base de données vectorielle », une « recherche sémantique », un « stockage d'intégrations », une « configuration Pinecone » ou une « recherche de similarité ».
