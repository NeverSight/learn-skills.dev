Создает пользовательские перехватчики React для распространенных шаблонов, включая выборку данных, формы, аутентификацию, локальное хранилище, устранение дребезга и многое другое. Используйте, когда пользователи запрашивают «создать собственный хук», «Хук React для», «хук useX» или «многоразовый хук».
