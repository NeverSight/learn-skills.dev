Applique des seuils de qualité minimum dans CI, notamment la couverture du code, le peluchage, la vérification de type et l'analyse de sécurité. Fournit les vérifications requises, les règles de relations publiques et l'application automatisée. À utiliser pour les « portes de qualité », les « contrôles CI », la « qualité du code » ou les « exigences PR ».
