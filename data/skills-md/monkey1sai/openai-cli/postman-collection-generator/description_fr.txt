Génère des fichiers JSON de collection Postman à partir d'Express, Next.js, Fastify, Hono ou d'autres routes API. Analyse les définitions de route, extrait les points de terminaison, les méthodes, les paramètres et crée des collections importables. À utiliser lorsque les utilisateurs demandent « générer une collection Postman », « exporter vers Postman », « créer un fichier Postman » ou « importer Postman ».
