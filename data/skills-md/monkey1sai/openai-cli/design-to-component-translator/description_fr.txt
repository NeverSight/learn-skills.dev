Convertit les spécifications Figma/conception en composants d'interface utilisateur prêts pour la production avec un espacement, une typographie, des jetons de couleur, des règles réactives et des états d'interaction précis (survol, focus, désactivé, actif). Génère du code Tailwind/shadcn avec le mappage des jetons du système de conception. À utiliser pour traduire « Figma en code », « spécifications de conception en composants » ou « implémenter un système de conception ».
