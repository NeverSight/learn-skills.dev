يُنشئ سجلات التغيير وملاحظات الإصدار من التزامات git وعناوين العلاقات العامة ومراجع المشكلات. ينظم التغييرات حسب نوع التأثير (الأعطال، الميزات، الإصلاحات، التحسينات)، والتنسيقات وفقًا لمعيار Keep a Changelog، ويوفر علامات الإصدار واقتراحات الإصدار الدلالي. يُستخدم عندما يطلب المستخدمون "إنشاء سجل التغيير" أو "كتابة ملاحظات الإصدار" أو "تغييرات إصدار المستند" أو "التحضير للإصدار".
