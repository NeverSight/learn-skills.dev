Создает интеграционные тесты для конечных точек API с потоками базы данных, включая настройку тестовых средств, исправления, настройку/демонтаж, заполнение базы данных и стратегии, ориентированные на CI. Используйте для «интеграционного тестирования», «тестов API», «тестов базы данных» или «тестового оборудования».
