Automatiza lanzamientos y publicación de paquetes con conjuntos de cambios o lanzamientos semánticos. Maneja el control de versiones, la generación de registros de cambios, etiquetas git y notas de la versión. Úselo para "automatización de lanzamiento", "control de versiones semántica", "publicación de paquetes" o "generación de registros de cambios".
