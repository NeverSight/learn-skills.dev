Автоматизирует выпуски и публикацию пакетов с помощью наборов изменений или семантического выпуска. Управляет версиями, генерацией журнала изменений, тегами git и примечаниями к выпуску. Используйте для «автоматизации выпуска», «семантического управления версиями», «публикации пакетов» или «создания журнала изменений».
