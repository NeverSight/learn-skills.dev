Реализует TanStack Query (React Query) для управления состоянием сервера с помощью кэширования, мутаций, оптимистичных обновлений и бесконечных запросов. Используйте, когда пользователи запрашивают «реагирующий запрос», «запрос tanstack», «извлечение данных», «управление кешем» или «состояние сервера».
