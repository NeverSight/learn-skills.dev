Implémente des systèmes complets de thématiques en mode sombre/clair à l'aide de variables CSS, du mode sombre Tailwind, du contexte React et de la détection des préférences système. À utiliser lorsque les utilisateurs demandent « ajouter un mode sombre », « basculer le thème », « thème sombre », « changer de mode clair » ou « jeu de couleurs ».
