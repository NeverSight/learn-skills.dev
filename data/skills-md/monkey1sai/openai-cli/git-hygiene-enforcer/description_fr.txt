Établit des garde-fous de flux de travail Git, y compris les validations conventionnelles, les hooks de messages de validation, les conventions de dénomination des branches, les modèles PR et les processus de révision de code. Fournit des configurations de hook, des modèles de flux de travail et des instructions de contournement d’urgence. À utiliser lorsque les utilisateurs demandent « configurer des hooks git », « appliquer les conventions de validation », « ajouter des modèles PR » ou « normaliser le flux de travail git ».
