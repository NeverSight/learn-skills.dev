Устанавливает ограничения рабочего процесса git, включая обычные коммиты, перехватчики сообщений о коммитах, соглашения об именах ветвей, шаблоны PR и процессы проверки кода. Содержит конфигурации перехватчиков, шаблоны рабочих процессов и инструкции по аварийному обходу. Используйте, когда пользователи запрашивают «настроить git-хуки», «обеспечить соблюдение соглашений о фиксации», «добавить шаблоны PR» или «стандартизировать рабочий процесс git».
