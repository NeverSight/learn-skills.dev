Создает конечные точки CRUD REST API с проверкой запроса, типами TypeScript, согласованными форматами ответов, обработкой ошибок и документацией. Включает обработчики маршрутов, схемы проверки (Zod/Joi), типизированные ответы и примеры использования. Используйте при создании «REST API», «конечных точек CRUD», «маршрутов API» или «конечных точек серверной части».
