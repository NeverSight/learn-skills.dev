Génère des points de terminaison de l'API CRUD REST avec validation des demandes, types TypeScript, formats de réponse cohérents, gestion des erreurs et documentation. Comprend des gestionnaires de routes, des schémas de validation (Zod/Joi), des réponses saisies et des exemples d'utilisation. À utiliser lors de la création de « API REST », de « points de terminaison CRUD », de « routes API » ou de « points de terminaison backend ».
