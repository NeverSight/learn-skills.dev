Erzeugt umfassende API-Dokumentation im Markdown-, HTML- oder Docusaurus-Format aus Express, Next.js, Fastify oder anderen API-Routen. Erstellt Endpunktreferenzen, Anforderungs-/Antwortbeispiele, Authentifizierungshandbücher und Fehlerdokumentation. Wird verwendet, wenn Benutzer „API-Dokumente generieren“, „API-Dokumentation“, „Endpunktdokumentation“ oder „API-Referenz“ anfordern.
