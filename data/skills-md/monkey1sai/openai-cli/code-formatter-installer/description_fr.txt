Installe et configure Prettier, ESLint, EditorConfig et d'autres outils de qualité de code pour appliquer un style de code cohérent au sein de l'équipe. Génère des fichiers de configuration, des scripts npm, des recommandations de paramètres d'éditeur et des suggestions d'intégration CI. À utiliser lorsque les utilisateurs demandent « une configuration plus jolie », « ajouter eslint », « configurer le formatage du code » ou « appliquer le style de code ».
