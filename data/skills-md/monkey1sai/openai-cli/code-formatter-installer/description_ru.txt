Устанавливает и настраивает Prettier, ESLint, EditorConfig и другие инструменты качества кода, чтобы обеспечить единообразный стиль кода для всей команды. Создает файлы конфигурации, сценарии npm, рекомендации по настройкам редактора и предложения по интеграции CI. Используйте, когда пользователи запрашивают «настроить красивее», «добавить eslint», «настроить форматирование кода» или «применить стиль кода».
