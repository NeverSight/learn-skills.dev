Создает комплексные планы системной архитектуры для функций и продуктов, включая разбивку компонентов, диаграммы потоков данных, границы системы, контракты API и соображения масштабирования. Используется для «проектирования системы», «планирования архитектуры», «проектирования функций» или «технических спецификаций».
