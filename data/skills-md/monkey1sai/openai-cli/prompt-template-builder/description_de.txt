Erstellt wiederverwendbare Eingabeaufforderungsvorlagen mit strengen Ausgabeverträgen, Stilregeln, Beispielen für wenige Aufnahmen und Do/Don't-Richtlinien. Stellt System-/Benutzer-Eingabeaufforderungsdateien, variable Platzhalter, Anweisungen zur Ausgabeformatierung und Qualitätskriterien bereit. Wird beim Erstellen von „Prompt-Vorlagen“, „LLM-Prompts“, „KI-System-Prompts“ oder „Prompt-Engineering“ verwendet.
