Создает многократно используемые шаблоны подсказок со строгими контрактами вывода, правилами стиля, небольшими примерами и рекомендациями «что делать/не делать». Предоставляет файлы системных/пользовательских подсказок, переменные заполнители, инструкции по форматированию вывода и критерии качества. Используйте при создании «шаблонов подсказок», «подсказок LLM», «подсказок системы искусственного интеллекта» или «подсказок для проектирования».
