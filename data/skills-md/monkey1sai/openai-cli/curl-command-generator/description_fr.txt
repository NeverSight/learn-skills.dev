Génère des commandes cURL prêtes à l'emploi à partir d'Express, Next.js, Fastify ou d'autres routes API. Crée des commandes copier-coller avec les en-têtes, l'authentification et les corps de requête appropriés. À utiliser lorsque les utilisateurs demandent « générer des commandes curl », « exemples de curl », « api curl » ou « tests d'API en ligne de commande ».
