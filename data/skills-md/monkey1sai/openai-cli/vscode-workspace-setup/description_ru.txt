Настраивает рабочие пространства VS Code с оптимальными настройками, расширениями, задачами и средствами отладки для обеспечения согласованности команды. Используйте, когда пользователи запрашивают «настройку VS Code», «настройки рабочей области», «конфигурацию команды VS Code», «конфигурацию редактора» или «настройку контейнера разработки».
