Защищает получателей веб-перехватчиков с помощью проверки подписи, обработки повторных попыток, дедупликации, ключей идемпотентности и ответов об ошибках. Предоставляет код проверки, стратегию дедупликации хранилища, книгу операций для инцидентов. Используйте при реализации «веб-перехватчиков», «безопасности веб-перехватчиков», «приемников событий» или «сторонних интеграций».
