Реализует тестирование контрактов API для обеспечения совместимости поставщика и потребителя с использованием Pact или аналогичных инструментов. Предотвращает критические изменения с помощью спецификаций контракта и двунаправленной проверки. Используйте для «контрактного тестирования», «контрактов API», «Пакта» или «контрактов, управляемых потребителем».
