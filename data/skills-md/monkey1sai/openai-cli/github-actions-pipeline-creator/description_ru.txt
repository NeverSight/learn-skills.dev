Создает комплексные рабочие процессы CI/CD GitHub Actions для анализа, тестирования, сборки и развертывания. Включает стратегии кэширования, построение матриц, обработку артефактов и диагностику сбоев. Используйте для «Действий GitHub», «Конвейера CI», «автоматизации рабочих процессов» или «непрерывной интеграции».
