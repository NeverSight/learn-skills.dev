يطبق نماذج وأدراج يمكن الوصول إليها مع مصيدة التركيز، وESC للإغلاق، وقفل التمرير، وعرض البوابة، وسمات ARIA. يتضمن تطبيقات نموذجية لحالات الاستخدام الشائعة مثل نماذج التحرير والتأكيدات وطرق عرض التفاصيل. يُستخدم عند إنشاء "النماذج" أو "مربعات الحوار" أو "الأدراج" أو "الأشرطة الجانبية" أو "التراكبات".
