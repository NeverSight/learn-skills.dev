Génère des spécifications OpenAPI 3.0/3.1 à partir des routes Express, Next.js, Fastify, Hono ou NestJS. Crée des spécifications complètes avec des schémas, des exemples et de la documentation qui peuvent être importés dans Postman, Insomnia ou utilisés avec l'interface utilisateur Swagger. À utiliser lorsque les utilisateurs demandent « générer openapi », « créer une spécification swagger », « documentation openapi » ou « spécification API ».
