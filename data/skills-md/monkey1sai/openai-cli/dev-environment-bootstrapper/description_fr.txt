Standardise la configuration de l'environnement de développement sur toutes les machines en générant des configurations de version d'outil (Node, Python, Ruby), des configurations de gestionnaire de packages (pnpm, Volta, asdf, mise), des modèles de variables d'environnement et des scripts de configuration avec documentation d'intégration. À utiliser lorsque les utilisateurs ont besoin de « configurer un environnement de développement », de « standardiser les outils », de « configurer les gestionnaires de versions » ou de « créer des scripts d'intégration ».
