Создает полные макеты страниц и оболочки для общих шаблонов (панель мониторинга, аутентификация, настройки, страницы CRUD) с единообразной навигацией, компонентами макета, структурой маршрутизации и заполнителями управления состоянием. Используйте при создании «новой страницы», «макета информационной панели», «страниц аутентификации» или «структуры панели администратора».
