توحيد حالات تجربة المستخدم لتدفقات جلب البيانات (التحميل، والخطأ، والفارغ، والنجاح) وأنماط واجهة المستخدم المعقدة متعددة العرض. يوفر توصيات بشأن نهج آلة الحالة، وتحميل الهياكل العظمية، وحدود الأخطاء، ومكونات الحالة الفارغة. يُستخدم عند تنفيذ "حالات التحميل" أو "معالجة الأخطاء" أو "الحالات الفارغة" أو "أجهزة الحالة".
