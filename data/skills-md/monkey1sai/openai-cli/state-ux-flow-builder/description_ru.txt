Стандартизирует состояния пользовательского интерфейса для потоков получения данных (загрузка, ошибка, пусто, успех) и сложных шаблонов пользовательского интерфейса с несколькими представлениями. Предоставляет рекомендации по использованию конечного автомата, скелеты загрузки, границы ошибок и пустые компоненты состояния. Используйте при реализации «состояний загрузки», «обработки ошибок», «пустых состояний» или «конечных автоматов».
