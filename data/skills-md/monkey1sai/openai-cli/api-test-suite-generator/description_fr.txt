Génère des suites de tests d'API complètes à l'aide de Jest, Vitest ou Supertest à partir d'Express, Next.js, Fastify ou d'autres routes API. Crée des tests d'intégration, des tests contractuels et une couverture des cas extrêmes. À utiliser lorsque les utilisateurs demandent « générer des tests API », « créer des tests de point de terminaison », « suite de tests API » ou « tests d'intégration pour API ».
