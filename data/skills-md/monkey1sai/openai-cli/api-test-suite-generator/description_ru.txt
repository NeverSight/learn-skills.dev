Создает комплексные наборы тестов API с использованием Jest, Vitest или Supertest из Express, Next.js, Fastify или других маршрутов API. Создает интеграционные тесты, контрактные тесты и покрытие крайних случаев. Используйте, когда пользователи запрашивают «создать тесты API», «создать тесты конечных точек», «набор тестов API» или «тесты интеграции для API».
