Génère des référentiels de démarrage complets pour diverses piles technologiques (Next.js, Vite, Nest, FastAPI, etc.) avec les meilleures pratiques, la structure des dossiers, le code de base, les configurations, les scripts et la documentation de configuration. À utiliser lorsque les utilisateurs demandent à « échafauder un projet », « créer un dépôt de démarrage », « initialiser un nouveau projet » ou spécifier une pile technologique pour commencer.
