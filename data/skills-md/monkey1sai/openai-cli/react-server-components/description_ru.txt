Реализует шаблоны серверных компонентов React (RSC) с помощью Next.js App Router, включая выборку данных, потоковую передачу, выборочную гидратацию и композицию сервер/клиент. Используйте, когда пользователи запрашивают «серверные компоненты», «RSC», «маршрутизатор приложений Next.js», «рендеринг на стороне сервера» или «потоковую передачу».
