Создает комплексные шаблоны документации для проектов с открытым исходным кодом и внутренних проектов, включая README, CONTRIBUTING, SECURITY, CODE_OF_CONDUCT, LICENSE и другие стандартные документы с предлагаемыми разделами и рекомендациями. Используйте, когда пользователи запрашивают «создать документацию проекта», «добавить документацию OSS», «настроить стандартную документацию» или «подготовить ее к открытому исходному коду».
