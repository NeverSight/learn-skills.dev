Compétence de configuration qui connecte les notifications toast natives du système d’exploitation aux hooks Claude Code. Basé sur node-notifier, il prend en charge Windows (SnoreToast), macOS (Notification Center) et Linux (libnotify). Il affiche des toasts dans trois situations : achèvement de la tâche, demandes d'autorisation et questions. Déclencheurs : « configurer les notifications de fin de tâche », « Je souhaite recevoir les notifications de Claude », « notification de hook », « faites-moi savoir quand c'est terminé », « configurer les notifications toast »
