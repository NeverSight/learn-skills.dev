Setting skill that connects OS-native toast notifications to Claude Code hooks. Based on node-notifier, it supports Windows (SnoreToast), macOS (Notification Center), and Linux (libnotify). It shows toasts in three situations: task completion, permission requests, and questions. Triggers: “set up task completion notifications”, “I want to get Claude notifications”, “hook notification”, “let me know when it’s done”, “configure toast notifications”
