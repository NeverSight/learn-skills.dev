Навык настройки, который связывает встроенные в ОС всплывающие уведомления с Claude Code. Созданный на основе узла-уведомителя, он поддерживает Windows (SnoreToast), macOS (Центр уведомлений) и Linux (libnotify). Он показывает всплывающие уведомления в трех случаях: завершение задания, запросы разрешения и вопросы. Триггеры: «установить оповещение о завершении», «Я хочу получать уведомления Клода», «уведомление о перехвате», «сообщить мне, когда закончите», «включить всплывающие уведомления».
