Навык интерфейса, который превращает любое информационное пространство в рабочее пространство вики на базе git и управляет им. Независимо от содержимого или формата документа, он инициализирует рабочую область как рабочую область git и динамически генерирует AGENTS.md, чтобы помочь агентам эффективно читать и работать. Для проектов с исходным кодом он связывает существующие вики как подмодули git для предоставления контекста агента. Триггеры: «создать вики», «создать рабочую область», «превратить эту папку в вики», «связать подмодуль», «подключить вики», «проанализировать содержимое вики», «обновить вики», «прочитать вики», «агент-вики», «агент вики», «добавить подмодуль», «инициализировать вики»
