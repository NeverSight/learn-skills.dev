Interface skill to create and manage any information space as a git-based wiki workspace. Regardless of document content or format, it initializes the workspace as a git workspace and dynamically generates AGENTS.md to guide agents in reading and working efficiently. In source-code projects, it connects existing wikis as git submodules to provide agent context. Triggers: "create wiki", "create workspace", "make this folder a wiki", "connect submodule", "connect wiki", "analyze wiki content", "update wiki", "read wiki", "agent-wiki", "agent wiki", "add submodule", "initialize wiki"
