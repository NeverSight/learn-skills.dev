Compétence d'interface pour transformer n'importe quel espace d'informations et le gérer comme un espace de travail wiki basé sur git. Quel que soit le contenu ou le format du document, il initialise la zone de travail en tant qu'espace de travail git et crée dynamiquement AGENTS.md pour guider les agents dans la lecture et le travail efficaces. Pour les projets de code source, il relie les wikis existants en tant que sous-modules git pour fournir le contexte de l'agent.
