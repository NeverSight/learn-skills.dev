Навык, который мгновенно генерирует предварительный просмотр экрана в чистом формате HTML/CSS/JS из документов с требованиями, файлов дизайна экрана, заметок с идеями и т. д. Принимает файлы .pdf/.txt/.md или папку screen-design/ в качестве входных данных, создает HTML-файл для каждого экрана и позволяет поделиться им по ссылке. Всякий раз, когда вы говорите «создать предварительный просмотр», «покажите мне экран», «сделать его HTML», «демонстрация клиента», «создать прототип», «демонстрационная страница», «нужно поделиться экраном» или «макет, готовый к просмотру», используется этот навык. Также используйте его для таких обновлений, как «обновить предварительный просмотр», «редактировать экран» или «справочный документ изменен».
