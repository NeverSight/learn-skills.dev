A skill that instantly generates a pure HTML/CSS/JS screen preview from requirement documents, wireframes, idea memos, etc. It accepts .pdf/.txt/.md files or a screen-design/ folder as input, creates an HTML file per screen, and lets you share it via a link. Whenever you say “create a preview”, “screen preview”, “make it HTML”, “client-facing screen”, “build a prototype”, “demo page”, “need to share the screen”, or “review wireframe”, this skill is always used. It is also used when updating existing outputs, such as “update preview”, “revise the screen”, or “the reference doc changed”.
