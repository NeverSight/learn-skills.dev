Eine Fähigkeit, die sofort eine reine HTML/CSS/JS-Bildschirmvorschau aus Anforderungsdokumenten, Wireframes, Ideennotizen usw. generiert. Sie akzeptiert .pdf/.txt/.md-Dateien oder einen Screen-Design/-Ordner als Eingabe, erstellt eine HTML-Datei für jeden Bildschirm und ermöglicht die Freigabe über Links. Immer wenn Sie sagen „Vorschau erstellen“, „Bildschirm zeigen“, „in HTML umwandeln“, „Client-Demo“, „Prototyp erstellen“, „Demoseite“, „Ich muss den Bildschirm freigeben“ oder „Wireframes überprüfen“, muss diese Fähigkeit verwendet werden. Verwenden Sie es auch, wenn Sie vorhandene Ausgaben aktualisieren, z. B. „Vorschau aktualisieren“, „Bildschirm reparieren“ oder „Referenzdokument geändert“.
