Una guía para usar componentes shadcn-svelte en un proyecto SvelteKit. Los usuarios siempre deben usar esta habilidad cuando crean pantallas de interfaz de usuario, no están seguros de qué componente usar o necesitan instalar o configurar inicialmente componentes shadcn-svelte. Siempre que se mencionen elementos de la interfaz de usuario como botones, modales, tarjetas, barras laterales, formularios o tablas, haga referencia activamente a esta habilidad.
