anime.js v4 を使用して JavaScript アニメーションを作成する場合、これには DOM 要素アニメーション、タイムライン、スタッガー効果、スクロール トリガー アニメーション、SVG アニメーション、テキスト アニメーション、ドラッグ可能な要素、レイアウト遷移、または Web サイト上のモーション/遷移効果が含まれます。ユーザーが「アニメーション」、「アニメーション ライブラリ」、「anime.js」、「スタッガー」、「タイムライン アニメーション」、「スクロール アニメーション」、「テキスト アニメーション」、「ドラッグ可能」、または「モーション エフェクト」と言う場合にも使用します。
