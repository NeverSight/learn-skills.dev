Interaktiver Kurs zum Erlernen von Claude Code von Grund auf bis zum Power-User. Verwenden Sie diese Option, wenn der Benutzer „Lernen“, „Tutorial“, „Claude-Code lernen“, „Claude-Code lernen“, „Wie man Claude-Code verwendet“, „Erste Schritte mit Claude-Code“, „Grundlagen von Claude-Code“ sagt oder fragt, wie man Claude-Code verwendet. Bietet eine progressive, praktische Lernerfahrung, die alle Claude Code-Funktionen von den Grundlagen bis hin zu Power-User-Workflows abdeckt.
