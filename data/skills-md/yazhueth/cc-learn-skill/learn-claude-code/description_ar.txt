دورة تفاعلية لتعلم كلود كود من الصفر إلى المستخدم المتميز. يُستخدم عندما يقول المستخدم "تعلم"، أو "البرنامج التعليمي"، أو "تعلم كود كلود"، أو "علمني كود كلود"، أو "كيفية استخدام كود كلود"، أو "البدء باستخدام كود كلود"، أو "أساسيات كود كلود"، أو يطلب معرفة كيفية استخدام كود كلود. يوفر تجربة تعليمية عملية تقدمية تغطي جميع ميزات Claude Code بدءًا من الأساسيات وحتى سير عمل المستخدم المتميز.
