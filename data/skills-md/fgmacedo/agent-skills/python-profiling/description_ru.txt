Структурированный рабочий процесс для профилирования и оптимизации кода Python — горячие точки ЦП, использование памяти и проверка на основе тестов. Используйте этот навык всякий раз, когда пользователь просит ускорить код Python, уменьшить использование памяти, повысить производительность профиля, найти узкие места, оптимизировать горячие пути, уменьшить выделение ресурсов или выяснить, почему что-то работает медленно. Также срабатывает, когда пользователь упоминает инструменты профилирования (cProfile, line_profiler, py-spy, memray, Tracemalloc) или просит настроить тесты. Даже случайные запросы вроде «это медленно» или «можно ускорить?» должны активировать этот навык.
