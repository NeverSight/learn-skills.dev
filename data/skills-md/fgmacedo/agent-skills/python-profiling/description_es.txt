Flujo de trabajo estructurado para crear perfiles y optimizar el código Python: puntos de acceso de CPU, uso de memoria y validación basada en puntos de referencia. Utilice esta habilidad siempre que el usuario solicite acelerar el código Python, reducir el uso de memoria, perfilar el rendimiento, encontrar cuellos de botella, optimizar rutas activas, reducir las asignaciones o investigar por qué algo es lento. También se activa cuando el usuario menciona herramientas de creación de perfiles (cProfile, line_profiler, py-spy, memray, tracemalloc) o solicita configurar puntos de referencia. Incluso solicitudes casuales como "esto es lento" o "¿podemos acelerar esto?" deberían activar esta habilidad.
