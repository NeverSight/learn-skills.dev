Superviser et gérer une instance interne de Claude Code exécutée dans tmux. Utilisez cette compétence lorsque vous devez déléguer le travail de mise en œuvre à un Claude intérieur tout en vous concentrant sur la planification des tâches, le suivi des progrès et les tests d'acceptation de bout en bout. Idéal pour les tâches de longue durée qui autrement épuiseraient une seule fenêtre contextuelle de Claude.
