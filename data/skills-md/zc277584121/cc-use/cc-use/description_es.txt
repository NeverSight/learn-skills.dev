Supervise y administre una instancia interna de Claude Code que se ejecuta en tmux. Utilice esta habilidad cuando necesite delegar el trabajo de implementación a un Claude interno mientras se concentra en la planificación de tareas, el seguimiento del progreso y las pruebas de aceptación de un extremo a otro. Ideal para tareas de larga duración que, de otro modo, agotarían la ventana de contexto de una única Claude.
