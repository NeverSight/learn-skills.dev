Modelli di gestione della valuta per la libreria di denaro Dinero.js. Da utilizzare quando si lavora con più valute, si effettuano conversioni tra valute, si definiscono valute personalizzate, si archiviano valori monetari in database o si integra con servizi di pagamento come Stripe o PayPal. Trigger sulla conversione di valuta, progettazione di schemi di database per denaro o integrazione dell'API di pagamento con oggetti Dinero.
