Создавайте персонализированные отчеты о профилях пользователей, анализируя историю чатов с помощью ИИ. Поддерживает анализ записей чата с помощью различных инструментов программирования искусственного интеллекта, включая Claude Code и OpenCode. Этот навык делает упор на гибкий анализ на основе искусственного интеллекта, а не на жесткие шаблоны.
