الترحيل بمساعدة الذكاء الاصطناعي وترقية حزم npm. يتعامل مع التغييرات الفاصلة بين الإصدارات الرئيسية، وتنظيف الإهمال في الإصدارات الثانوية، واعتماد واجهات برمجة التطبيقات والأنماط الجديدة، والترقيات المستندة إلى الأمان من تدقيق npm، واستبدال التبعية الكاملة (تبديل حزمة بأخرى، على سبيل المثال moment → dayjs). يحلل سجلات التغيير، واختلافات git، والمستندات، ويفحص قاعدة التعليمات البرمجية الخاصة بك للاستخدام الفعلي، ويراجع المراجع للعثور على ما يتأثر، وينشئ إصلاحات التعليمات البرمجية المستهدفة أو تعديلات التعليمات البرمجية، ويتحقق من خلال مجموعة الاختبار الخاصة بك. استخدم هذه المهارة عندما يذكر المستخدم ترقية حزم npm أو ترحيلها أو تحديثها، أو التعامل مع التغييرات العاجلة، أو إصلاح تحذيرات الإهمال، أو استبدال التبعية ببديل، أو اعتماد واجهات برمجة التطبيقات الجديدة من تحديث الحزمة، أو تشغيل إصلاح تدقيق npm مع تغييرات التعليمات البرمجية، أو مقارنة ما تغير بين إصدارات الحزمة. تتضمن عبارات التشغيل ما يلي: "ترقية axios إلى الإصدار 2"، و"الترحيل إلى Express 5"، و"استبدال اللحظة بـ dayjs"، و"إصلاح تحذيرات الإهمال"، و"تدقيق npm يقول أنه معرض للخطر"، و"اعتماد واجهة برمجة التطبيقات الجديدة"، و"ما تغير بين الإصدارين X وY"، و"مبادلة lodash بمجموعة أدوات es"، و"ساعدني في ترقية تبعياتي"، و"تنظيف المكالمات المهملة".
