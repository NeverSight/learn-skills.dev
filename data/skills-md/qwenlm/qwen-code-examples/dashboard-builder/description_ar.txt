أنشئ تطبيقات لوحة معلومات متكاملة باستخدام React/Next.js + shadcn/ui + Tailwind CSS + Recharts + Express مع مصادر بيانات قابلة للتخصيص. يُستخدم عند إنشاء لوحات معلومات تصور البيانات، أو واجهات ذكاء الأعمال، أو أنظمة المراقبة، أو شاشات عرض مؤشرات الأداء الرئيسية، أو منصات التحليلات، أو أي مشروع يتطلب تصور البيانات باستخدام مكونات واجهة المستخدم الحديثة والتأثيرات الديناميكية.
