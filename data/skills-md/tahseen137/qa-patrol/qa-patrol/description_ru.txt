Автоматизированное тестирование качества веб-приложений с использованием локальной автоматизации браузера. Работает полностью на вашем компьютере — никаких данных, никаких облачных сервисов и внешних серверов. Уровень 1 (дымовые тесты) требует только URL-адреса. Уровень 2 (тестирование аутентификации/платежей) использует дополнительные переменные среды для тестовых учетных данных. Уровень 3 (статический анализ, проверка БД) опционально считывает локальные файлы и подключается к базе данных, предоставленной пользователем. Поддерживает аутентификацию Supabase/Firebase, платежи Stripe, React Native Web, Next.js и SPA.
