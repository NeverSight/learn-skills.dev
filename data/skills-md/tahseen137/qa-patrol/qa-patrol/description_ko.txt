로컬 브라우저 자동화를 사용하여 웹 앱에 대한 자동화된 QA 테스트. 데이터가 남지 않고, 클라우드 서비스도 없고, 외부 서버도 없이 전적으로 컴퓨터에서 실행됩니다. 레벨 1(스모크 테스트)에는 URL만 필요합니다. 레벨 2(인증/결제 테스트)에서는 테스트 자격 증명에 선택적 환경 변수를 사용합니다. 레벨 3(정적 분석, DB 검사)은 선택적으로 로컬 파일을 읽고 사용자가 제공한 데이터베이스에 연결합니다. Supabase/Firebase 인증, Stripe 결제, React Native Web, Next.js 및 SPA를 지원합니다.
