Automatisierte QS-Tests für Web-Apps mithilfe lokaler Browser-Automatisierung. Läuft vollständig auf Ihrem Computer – keine Datenverluste, keine Cloud-Dienste, keine externen Server. Stufe 1 (Rauchtests) benötigt nur eine URL. Stufe 2 (Authentifizierungs-/Zahlungstests) verwendet optionale Umgebungsvariablen für Testanmeldeinformationen. Level 3 (statische Analyse, DB-Prüfungen) liest optional lokale Dateien und stellt eine Verbindung zu einer vom Benutzer bereitgestellten Datenbank her. Unterstützt Supabase/Firebase-Authentifizierung, Stripe-Zahlungen, React Native Web, Next.js und SPAs.
