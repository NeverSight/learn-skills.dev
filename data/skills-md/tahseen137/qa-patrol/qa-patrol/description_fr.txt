Tests d'assurance qualité automatisés pour les applications Web à l'aide de l'automatisation du navigateur local. Fonctionne entièrement sur votre machine : aucune donnée ne part, aucun service cloud, aucun serveur externe. Le niveau 1 (tests de fumée) n'a besoin que d'une URL. Le niveau 2 (test d'authentification/paiement) utilise des variables d'environnement facultatives pour les informations d'identification de test. Le niveau 3 (analyse statique, vérifications de base de données) lit éventuellement les fichiers locaux et se connecte à une base de données fournie par l'utilisateur. Prend en charge l'authentification Supabase/Firebase, les paiements Stripe, React Native Web, Next.js et SPA.
