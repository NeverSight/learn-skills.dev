Test QA automatizzati per app Web utilizzando l'automazione del browser locale. Funziona interamente sul tuo computer: nessun dato, nessun servizio cloud, nessun server esterno. Il livello 1 (test del fumo) necessita solo di un URL. Il livello 2 (test di autenticazione/pagamento) utilizza variabili env opzionali per le credenziali di test. Il livello 3 (analisi statica, controlli DB) facoltativamente legge i file locali e si connette a un database fornito dall'utente. Supporta l'autenticazione Supabase/Firebase, i pagamenti Stripe, React Native Web, Next.js e SPA.
