Pruebas de control de calidad automatizadas para aplicaciones web mediante la automatización del navegador local. Se ejecuta completamente en su máquina: no hay salidas de datos, ni servicios en la nube, ni servidores externos. El nivel 1 (pruebas de humo) solo necesita una URL. El nivel 2 (prueba de autenticación/pago) utiliza variables de entorno opcionales para las credenciales de prueba. El nivel 3 (análisis estático, comprobaciones de bases de datos) opcionalmente lee archivos locales y se conecta a una base de datos proporcionada por el usuario. Admite autenticación de Supabase/Firebase, pagos Stripe, React Native Web, Next.js y SPA.
