Dashboard dell'agente in tempo reale per OpenClaw. Monitora le attività attive, lo stato del lavoro cron, i problemi e le attività da qualsiasi luogo. Tre livelli di configurazione: (1) Rendering canvas a configurazione zero all'interno di OpenClaw, (2) Pagine GitHub con polling di 30 secondi: configurazione gratuita in 2 minuti, (3) Supabase Realtime + Vercel per aggiornamenti istantanei del websocket. Tutti i dati rimangono sul tuo computer. Protetto da PIN. Nessun servizio esterno richiesto per il Livello 1. Il Livello 3 richiede SUPABASE_URL e SUPABASE_ANON_KEY (non è necessaria alcuna chiave service_role). Vengono inviati solo i dati sullo stato operativo (nomi delle attività, stato cron, timestamp), mai credenziali, chiavi API o contenuto dei file.
