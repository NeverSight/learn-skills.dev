OpenClaw のリアルタイム エージェント ダッシュボード。アクティブなタスク、cron ジョブの健全性、問題、アクション アイテムをどこからでも監視します。 3 つのセットアップ層: (1) OpenClaw 内でのゼロ構成キャンバス レンダリング、(2) 30 秒のポーリングを備えた GitHub Pages (無料、2 分間のセットアップ)、(3) WebSocket の即時更新のための Supabase Realtime + Vercel。すべてのデータはマシン上に残ります。 PIN で保護されています。 Tier 1 には外部サービスは必要ありません。Tier 3 には SUPABASE_URL と SUPABASE_ANON_KEY が必要です (service_role キーは必要ありません)。操作ステータス データ (タスク名、cron ステータス、タイムスタンプ) のみがプッシュされます。資格情報、API キー、ファイルの内容はプッシュされません。
