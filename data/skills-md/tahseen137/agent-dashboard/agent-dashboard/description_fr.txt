Tableau de bord d'agent en temps réel pour OpenClaw. Surveillez les tâches actives, l’état des tâches cron, les problèmes et les actions à entreprendre où que vous soyez. Trois niveaux de configuration : (1) Rendu de canevas sans configuration dans OpenClaw, (2) Pages GitHub avec interrogation de 30 secondes — configuration gratuite en 2 minutes, (3) Supabase Realtime + Vercel pour des mises à jour instantanées de Websocket. Toutes les données restent sur votre machine. Protégé par code PIN. Aucun service externe requis pour le niveau 1. Le niveau 3 nécessite SUPABASE_URL et SUPABASE_ANON_KEY (aucune clé service_role n'est nécessaire). Seules les données d’état opérationnel sont transmises (noms de tâches, statut cron, horodatages) – jamais les informations d’identification, les clés API ou le contenu des fichiers.
