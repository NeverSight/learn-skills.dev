Панель управления агента в режиме реального времени для OpenClaw. Отслеживайте активные задачи, состояние заданий cron, проблемы и действия из любой точки мира. Три уровня настройки: (1) рендеринг холста без настройки внутри OpenClaw, (2) страницы GitHub с 30-секундным опросом — бесплатная, 2-минутная установка, (3) Supabase Realtime + Vercel для мгновенных обновлений через веб-сокет. Все данные остаются на вашем компьютере. PIN-защищен. Для уровня 1 внешние службы не требуются. Для уровня 3 требуются SUPABASE_URL и SUPABASE_ANON_KEY (ключ service_role не требуется). Передаются только данные о рабочем состоянии (имена задач, статус cron, метки времени) — никогда учетные данные, ключи API или содержимое файла.
