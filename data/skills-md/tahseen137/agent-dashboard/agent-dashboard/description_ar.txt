لوحة تحكم الوكيل في الوقت الفعلي لـ OpenClaw. مراقبة المهام النشطة، وصحة وظيفة cron، والمشكلات، وعناصر العمل من أي مكان. ثلاثة مستويات إعداد: (1) عرض قماش بدون تكوين داخل OpenClaw، (2) صفحات GitHub مع استقصاء مدته 30 ثانية - إعداد مجاني لمدة دقيقتين، (3) Supabase Realtime + Vercel لتحديثات websocket الفورية. تبقى جميع البيانات على جهازك. محمي برقم التعريف الشخصي. لا توجد خدمات خارجية مطلوبة للمستوى 1. ويتطلب المستوى 3 SUPABASE_URL وSUPABASE_ANON_KEY (لا يلزم وجود مفتاح_دور_الخدمة). يتم دفع بيانات الحالة التشغيلية فقط (أسماء المهام، وحالة cron، والطوابع الزمنية) - ولا يتم دفع بيانات الاعتماد أو مفاتيح API أو محتويات الملف مطلقًا.
