Panel de agentes en tiempo real para OpenClaw. Supervise las tareas activas, el estado del trabajo cron, los problemas y los elementos de acción desde cualquier lugar. Tres niveles de configuración: (1) renderizado de lienzo de configuración cero dentro de OpenClaw, (2) páginas de GitHub con sondeo de 30 segundos: configuración gratuita de 2 minutos, (3) Supabase Realtime + Vercel para actualizaciones instantáneas de websocket. Todos los datos permanecen en su máquina. Protegido por PIN. No se requieren servicios externos para el Nivel 1. El Nivel 3 requiere SUPABASE_URL y SUPABASE_ANON_KEY (no se necesita clave service_role). Solo se envían datos de estado operativo (nombres de tareas, estado de cron, marcas de tiempo), nunca credenciales, claves API o contenidos de archivos.
