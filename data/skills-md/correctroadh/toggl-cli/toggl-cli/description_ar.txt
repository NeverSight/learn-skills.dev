إدارة Toggl تتبع إدخالات الوقت والمشاريع والعلامات والعملاء والمهام ومساحات العمل والمؤسسات عبر `toggle` CLI. يتميز بالتخزين المؤقت لاستجابة HTTP لتحسين الأداء وتقليل مكالمات API. يُستخدم لطلبات تتبع الوقت مثل البدء/الإيقاف/المتابعة/القائمة/التحرير/الحذف، والملف الشخصي/التفضيلات، وإدارة الموارد، حتى عندما لا يذكر المستخدم Toggl صراحةً.
