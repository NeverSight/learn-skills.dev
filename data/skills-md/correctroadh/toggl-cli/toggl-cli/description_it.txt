Gestisci Toggl Tieni traccia di voci di tempo, progetti, tag, clienti, attività, spazi di lavoro e organizzazioni tramite la CLI "toggl". Dispone di memorizzazione nella cache della risposta HTTP per prestazioni migliorate e chiamate API ridotte. Utilizzare per richieste di monitoraggio del tempo come avvio/arresto/continua/elenco/modifica/eliminazione, profilo/preferenze e gestione delle risorse, anche quando l'utente non menziona esplicitamente Toggl.
