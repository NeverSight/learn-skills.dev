新しいテクノロジーを探索する JavaScript 開発者向けの学習支援スキル。ユーザーが新しいプログラミング言語 (Elixir、Swift、Go、Rust、Python など)、新しいツールやプラットフォーム (Kubernetes、Docker、Terraform、Redis、Kafka など)、または新しいアーキテクチャのアプローチやパラダイム (コンテナー化、マイクロサービス、オブジェクト プーリング、分散システム、イベント ソーシング、CQRS、DDD など) を学習、理解、または探索したい場合に使用します。トリガー: 「教えてください」、「説明してください」、「X はどのように機能しますか」、「JavaScript 開発者のための X」、「X を学びます」、「X とは何ですか」、「X と JS を比較します」、「X を理解したいです」、「X がどのように機能するかを教えてください」、「学習を手伝ってください」、「JS から X への橋渡し」、「Y の X に相当するものは何ですか」、または JS 開発者の観点から見たなじみのないテクノロジーに関する質問。プロジェクト コードは生成しません。説明、比較、図、洞察を提供します。学習の進行状況をセッション全体にわたって永続メモリとしてトレッカーに保存します。
