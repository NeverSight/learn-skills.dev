Guide de configuration TanStack (Routeur/Requête/Formulaire/Table) pour les projets Vite + React. Couvre la configuration du plugin Vite, l'enregistrement main.tsx, la configuration DevTools et les paramètres de l'éditeur. À utiliser lors de la configuration ou de la révision de la configuration TanStack dans un projet Vite. Déclencheurs sur : vite-tanstack, configuration tanstack, configuration du routeur tanstack, configuration des requêtes tanstack, configuration du formulaire tanstack, configuration de la table tanstack.
