Guida esperta per l'implementazione di NuxtUI v4.1+: una libreria dell'interfaccia utente open source di oltre 100 componenti personalizzabili creata con Tailwind CSS e Reka UI. Da utilizzare quando l'utente menziona i componenti NuxtUI (UButton, UInput, UCard e così via), chiede informazioni sull'implementazione delle funzionalità dell'interfaccia utente, necessita di esempi di componenti, lavora con moduli/layout/navigazione o richiede la personalizzazione di temi/stili. Suggerire in modo proattivo durante la creazione dell'interfaccia utente per garantire una progettazione coerente. (progetto)
