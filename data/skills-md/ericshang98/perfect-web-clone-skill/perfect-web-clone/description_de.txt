Klonen Sie jede Webseite in pixelgenauen, produktionsbereiten Code. Extrahiert die vollständige Seitenstruktur mit Playwright, segmentiert Inhalte intelligent nach drei Grundprinzipien (gegenseitige Exklusivität, vollständige Abdeckung, Größenkontrolle) und generiert parallel React/Next.js/Vue-Komponenten mithilfe von Subagenten. Verwenden Sie diese Option, wenn der Benutzer eine Website klonen, ein Seitendesign replizieren, eine URL in Code konvertieren, eine Webseite neu erstellen oder das Layout einer Website kopieren möchte.
