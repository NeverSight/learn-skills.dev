Клонируйте любую веб-страницу в идеальный, готовый к использованию код с точностью до пикселя. Извлекает полную структуру страницы с помощью Playwright, разумно разбивает контент на фрагменты в соответствии с тремя основными принципами (взаимная эксклюзивность, полный охват, контроль размера) и параллельно генерирует компоненты React/Next.js/Vue с помощью подагентов. Используйте, когда пользователь хочет клонировать веб-сайт, скопировать дизайн страницы, преобразовать URL-адрес в код, перестроить веб-страницу или скопировать макет веб-сайта.
