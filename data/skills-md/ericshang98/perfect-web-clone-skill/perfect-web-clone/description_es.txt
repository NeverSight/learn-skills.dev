Clona cualquier página web en un código perfecto en píxeles y listo para producción. Extrae la estructura completa de la página utilizando Playwright, fragmenta el contenido de forma inteligente siguiendo tres principios básicos (exclusividad mutua, cobertura completa, control de tamaño) y genera componentes React/Next.js/Vue en paralelo utilizando subagentes. Úselo cuando el usuario quiera clonar un sitio web, replicar el diseño de una página, convertir una URL en código, reconstruir una página web o copiar el diseño de un sitio web.
