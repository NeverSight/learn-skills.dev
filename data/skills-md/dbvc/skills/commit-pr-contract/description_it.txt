Genera messaggi di commit (titolo + corpo) e descrizioni PR in cinese, separando chiaramente la prospettiva di commit di "evidenza/tracciabilità a lungo termine" dalla prospettiva di PR "aiutare il revisore a comprendere e rivedere". Applicabile quando l'utente richiede "scrivi messaggio di commit / scrivi descrizione PR / genera commit + PR in base alle modifiche" e deve seguire il formato "M-xxx(tipo): titolo" e la struttura PR in stile contratto in quattro parti; può dedurre rischi e rivedere l'attenzione in base alle descrizioni delle modifiche/differenze di codice e chiedere chiarimenti se necessario.
