REST API для оптимизированного обмена токенов (включая создание исполняемых транзакций), котирования свопов и ценообразования с использованием агрегатора SushiSwap.
Используйте этот навык, когда пользователь хочет: - Получить котировку обмена между двумя токенами в более чем 40 сетях evm - Сгенерировать исполняемые данные о транзакциях обмена - Получить цены токенов для конкретной сети или токена - Получить метаданные токена - Обнаружить поддерживаемые источники ликвидности AMM - Интегрировать логику обмена или ценообразования SushiSwap через HTTP/REST (а не Javascript API SushiSwap)
