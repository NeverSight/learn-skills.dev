API REST pour l'échange optimisé de jetons (y compris la génération de transactions exécutables), les devis d'échange et la tarification à l'aide de SushiSwap Aggregator.
Utilisez cette compétence lorsque l'utilisateur souhaite : - Obtenir un devis de swap entre deux jetons sur plus de 40 réseaux evm - Générer des données de transaction de swap exécutables - Récupérer les prix des jetons pour un réseau ou un jeton spécifique - Récupérer les métadonnées du jeton - Découvrir les sources de liquidité AMM prises en charge - Intégrer la logique d'échange ou de tarification SushiSwap via HTTP/REST (et non l'API Javascript SushiSwap)
