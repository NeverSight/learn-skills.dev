REST API لمبادلة الرموز المميزة (بما في ذلك إنشاء المعاملات القابلة للتنفيذ)، وتسعير المبادلة، والتسعير باستخدام SushiSwap Aggregator.
استخدم هذه المهارة عندما يريد المستخدم: - الحصول على عرض أسعار مبادلة بين رمزين مميزين على أكثر من 40 شبكة evm - إنشاء بيانات معاملات مبادلة قابلة للتنفيذ - جلب أسعار الرمز المميز لشبكة أو رمز مميز معين - استرداد البيانات التعريفية للرمز المميز - اكتشاف مصادر سيولة AMM المدعومة - دمج منطق المبادلة أو التسعير في SushiSwap عبر HTTP/REST (وليس SushiSwap Javascript API)
