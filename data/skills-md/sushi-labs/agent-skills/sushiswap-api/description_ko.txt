SushiSwap Aggregator를 사용하여 최적화된 토큰 교환(실행 가능한 트랜잭션 생성 포함), 교환 견적 및 가격 책정을 위한 REST API입니다.
사용자가 원할 때 이 기술을 사용하십시오: - 40개 이상의 evm 네트워크에서 두 토큰 사이의 스왑 견적 받기 - 실행 가능한 스왑 거래 데이터 생성 - 특정 네트워크 또는 토큰에 대한 토큰 가격 가져오기 - 토큰 메타데이터 검색 - 지원되는 AMM 유동성 소스 검색 - HTTP/REST를 통해 SushiSwap 스왑 또는 가격 책정 로직 통합(SushiSwap Javascript API 아님)
