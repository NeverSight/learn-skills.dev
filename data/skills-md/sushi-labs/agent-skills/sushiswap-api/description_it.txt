API REST per scambio di token ottimizzato (inclusa la generazione di transazioni eseguibili), quotazione di scambio e determinazione dei prezzi utilizzando SushiSwap Aggregator.
Utilizza questa competenza quando l'utente desidera: - Ottenere una quotazione di swap tra due token su oltre 40 reti evm - Generare dati di transazioni di swap eseguibili - Recuperare i prezzi dei token per una rete o un token specifico - Recuperare i metadati dei token - Scoprire le fonti di liquidità AMM supportate - Integrare lo swapping di SushiSwap o la logica dei prezzi tramite HTTP/REST (e non l'API Javascript SushiSwap)
