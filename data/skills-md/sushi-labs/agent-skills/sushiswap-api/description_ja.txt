SushiSwap Aggregator を使用した、最適化されたトークン スワッピング (実行可能なトランザクションの生成を含む)、スワップ クォート、および価格設定のための REST API。
ユーザーが次のことを行う場合、このスキルを使用します。 - 40 以上の evm ネットワーク上の 2 つのトークン間のスワップ相場を取得 - 実行可能なスワップ トランザクション データを生成 - 特定のネットワークまたはトークンのトークン価格を取得 - トークン メタデータを取得 - サポートされている AMM 流動性ソースを検出 - HTTP/REST (SushiSwap JavaScript API ではなく) 経由で SushiSwap スワップまたは価格設定ロジックを統合
