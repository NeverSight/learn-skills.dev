API REST para intercambio de tokens optimizado (incluida la generación de transacciones ejecutables), cotizaciones de intercambio y precios utilizando SushiSwap Aggregator.
Utilice esta habilidad cuando el usuario desee: - Obtener una cotización de intercambio entre dos tokens en más de 40 redes evm - Generar datos de transacciones de intercambio ejecutables - Obtener precios de tokens para una red o token específico - Recuperar metadatos de tokens - Descubrir fuentes de liquidez de AMM admitidas - Integrar la lógica de precios o intercambio de SushiSwap a través de HTTP/REST (y no la API de Javascript de SushiSwap)
