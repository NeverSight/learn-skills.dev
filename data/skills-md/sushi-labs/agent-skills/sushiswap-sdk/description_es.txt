SDK de TypeScript para interactuar con SushiSwap Aggregator y primitivas relacionadas. Este SDK es un contenedor escrito sobre la API SushiSwap, que proporciona ayudas ergonómicas para cantidades de tokens, precios, cotizaciones y generación de transacciones de intercambio.
UTILICE ESTA HABILIDAD CUANDO: - Cree aplicaciones TypeScript o JavaScript - Quiere primitivas de token, cantidad y fracción fuertemente tipadas - Necesita solicitar cotizaciones de intercambio o transacciones de intercambio ejecutables a través de código - Quiere aritmética, formato y comparaciones más seguras sin errores de punto flotante - Prefiere la integración basada en SDK a las solicitudes HTTP sin formato
