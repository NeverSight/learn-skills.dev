TypeScript SDK для взаимодействия с агрегатором SushiSwap и связанными с ним примитивами. Этот SDK представляет собой типизированную оболочку API SushiSwap, предоставляющую эргономичные помощники для определения количества токенов, цен, котировок и генерации транзакций свопа.
ИСПОЛЬЗУЙТЕ ЭТОТ НАВЫК, КОГДА: - Создание приложений TypeScript или JavaScript. - Вам нужны строго типизированные примитивы токенов, сумм и дробей. - Вам необходимо запрашивать котировки обмена или исполняемые транзакции обмена с помощью кода. - Вам нужна более безопасная арифметика, форматирование и сравнение без ошибок с плавающей запятой. - Вы предпочитаете интеграцию на основе SDK, а не необработанным HTTP-запросам.
