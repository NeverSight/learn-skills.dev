SDK TypeScript pour interagir avec SushiSwap Aggregator et les primitives associées. Ce SDK est un wrapper typé sur l'API SushiSwap, fournissant des aides ergonomiques pour les montants de jetons, les prix, les cotations et la génération de transactions de swap.
UTILISEZ CETTE COMPÉTENCE LORSQUE : - Création d'applications TypeScript ou JavaScript - Vous souhaitez des primitives de jeton, de montant et de fraction fortement typées - Vous devez demander des cotations de swap ou des transactions de swap exécutables via du code - Vous souhaitez une arithmétique, un formatage et des comparaisons plus sûrs sans erreurs à virgule flottante - Vous préférez l'intégration basée sur le SDK aux requêtes HTTP brutes
