SushiSwap Aggregator 및 관련 기본 요소와 상호작용하기 위한 TypeScript SDK입니다. 이 SDK는 SushiSwap API에 대한 형식화된 래퍼로, 토큰 금액, 가격, 견적 및 스왑 거래 생성을 위한 인체공학적 도우미를 제공합니다.
이 기술을 사용하는 경우: - TypeScript 또는 JavaScript 애플리케이션 구축 - 강력한 유형의 토큰, 금액 및 분수 프리미티브를 원하는 경우 - 코드를 통해 스왑 견적 또는 실행 가능한 스왑 트랜잭션을 요청해야 하는 경우 - 부동 소수점 오류 없이 더 안전한 산술, 형식 지정 및 비교를 원하는 경우 - 원시 HTTP 요청보다 SDK 기반 통합을 선호하는 경우
