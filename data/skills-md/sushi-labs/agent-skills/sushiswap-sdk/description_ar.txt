TypeScript SDK للتفاعل مع SushiSwap Aggregator والأوليات ذات الصلة. إن SDK هذا عبارة عن غلاف مكتوب فوق SushiSwap API، مما يوفر مساعدين مريحين لمبالغ الرمز المميز والأسعار وعروض الأسعار وإنشاء معاملات المبادلة.
استخدم هذه المهارة عندما: - إنشاء تطبيقات TypeScript أو JavaScript - إذا كنت تريد الرموز المميزة والمبلغ والأجزاء الأولية المكتوبة بقوة - تحتاج إلى طلب عروض أسعار المبادلة أو معاملات المبادلة القابلة للتنفيذ عبر التعليمات البرمجية - تريد عمليات حسابية وتنسيق ومقارنات أكثر أمانًا بدون أخطاء الفاصلة العائمة - تفضل التكامل المستند إلى SDK على طلبات HTTP الأولية
