SushiSwap Aggregator および関連するプリミティブと対話するための TypeScript SDK。この SDK は SushiSwap API の型付きラッパーであり、トークンの量、価格、相場、およびスワップ トランザクションの生成のための人間工学に基づいたヘルパーを提供します。
このスキルは次の場合に使用してください: - TypeScript または JavaScript アプリケーションを構築する - 厳密に型指定されたトークン、金額、分数プリミティブが必要 - スワップ クォートまたは実行可能なスワップ トランザクションをコード経由でリクエストする必要がある - 浮動小数点エラーを発生させずに、より安全な算術演算、書式設定、および比較が必要な場合 - 生の HTTP リクエストよりも SDK ベースの統合を好む場合
