TypeScript SDK per interagire con SushiSwap Aggregator e le relative primitive. Questo SDK è un wrapper digitato sull'API SushiSwap, che fornisce aiutanti ergonomici per importi di token, prezzi, quotazioni e generazione di transazioni di swap.
UTILIZZA QUESTA COMPETENZA QUANDO: - Costruisci applicazioni TypeScript o JavaScript - Desideri primitive di token, importi e frazioni fortemente tipizzate - Devi richiedere virgolette di scambio o transazioni di scambio eseguibili tramite codice - Desideri operazioni aritmetiche, formattazioni e confronti più sicuri senza errori in virgola mobile - Preferisci l'integrazione basata su SDK rispetto alle richieste HTTP non elaborate
