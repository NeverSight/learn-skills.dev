TypeScript SDK für die Interaktion mit dem SushiSwap Aggregator und zugehörigen Grundelementen. Dieses SDK ist ein typisierter Wrapper für die SushiSwap-API und bietet ergonomische Helfer für Token-Beträge, Preise, Angebote und die Generierung von Swap-Transaktionen.
VERWENDEN SIE DIESE FÄHIGKEIT, WENN: - TypeScript- oder JavaScript-Anwendungen erstellt werden - Sie stark typisierte Token-, Betrags- und Bruchprimitive wünschen - Sie Swap-Anführungszeichen oder ausführbare Swap-Transaktionen über Code anfordern müssen - Sie sicherere Arithmetik, Formatierung und Vergleiche ohne Gleitkommafehler wünschen - Sie eine SDK-basierte Integration gegenüber reinen HTTP-Anfragen bevorzugen
