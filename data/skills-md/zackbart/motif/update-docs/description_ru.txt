Сканирует проектную документацию на наличие устаревших данных и неточностей, а затем обновляет ее, чтобы отразить текущую кодовую базу. Использует субагент сканирования только для чтения для оценки документов и субагент записи для применения утвержденных изменений. При необходимости область действия может быть ограничена определенной областью с помощью аргументов.
