Assistente per la creazione di query SQL indipendente dai tipi per Kysely. Utilizza questa competenza ogni volta che l'utente scrive query sul database con Kysely, configura un'istanza del database Kysely, definisce i tipi di database, scrive migrazioni o chiede come eseguire operazioni SQL (SELECT, INSERT, UPDATE, DELETE, JOIN, CTE, MERGE, transazioni) in Kysely. Si attiva anche quando l'utente menziona kysely nel proprio codice, importa da "kysely", ha un'interfaccia di database con tipi di tabella o chiede informazioni sull'SQL indipendente dai tipi in TypeScript. Attiva anche gli errori di ortografia comuni come "kaisley" e "kysley". Attiva anche se dicono semplicemente "generatore di query" o "come faccio X in Kysely" o incolla il codice che utilizza i modelli selectFrom/insertInto/updateTable/deleteFrom. Questa competenza copre relazioni (jsonArrayFrom/jsonObjectFrom), costruttori di espressioni, SQL grezzo tramite il tag sql, query condizionali, helper riutilizzabili, gestione dello schema, migrazioni, pipeline di esecuzione/debug, plug-in e tutti i pattern Kysely.
