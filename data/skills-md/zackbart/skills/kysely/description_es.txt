Asistente de creación de consultas SQL con seguridad de tipos para Kysely. Utilice esta habilidad siempre que el usuario escriba consultas de bases de datos con Kysely, configure una instancia de base de datos de Kysely, defina tipos de bases de datos, escriba migraciones o pregunte cómo realizar operaciones SQL (SELECCIONAR, INSERTAR, ACTUALIZAR, ELIMINAR, UNIR, CTE, FUSIONAR, transacciones) en Kysely. También se activa cuando el usuario menciona kysely en su código, importa desde 'kysely', tiene una interfaz de base de datos con tipos de tablas o pregunta sobre SQL con seguridad de tipos en TypeScript. También active errores ortográficos comunes como "kaisley" y "kysley". Actívelo incluso si solo dicen "generador de consultas" o "cómo hago X en kysely" o pegue el código que usa los patrones selectFrom/insertInto/updateTable/deleteFrom. Esta habilidad cubre relaciones (jsonArrayFrom/jsonObjectFrom), creadores de expresiones, SQL sin formato a través de la etiqueta sql, consultas condicionales, ayudas reutilizables, gestión de esquemas, migraciones, canalización/depuración de ejecución, complementos y todos los patrones de Kysely.
