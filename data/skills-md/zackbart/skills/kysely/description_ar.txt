مساعد منشئ استعلام SQL الآمن لـ Kysely. استخدم هذه المهارة عندما يقوم المستخدم بكتابة استعلامات قاعدة البيانات باستخدام Kysely، أو إعداد مثيل قاعدة بيانات Kysely، أو تحديد أنواع قواعد البيانات، أو كتابة عمليات الترحيل، أو السؤال عن كيفية إجراء عمليات SQL (SELECT، INSERT، UPDATE، DELETE، JOIN، CTE، MERGE، المعاملات) في Kysely. يتم تشغيله أيضًا عندما يذكر المستخدم kysely في التعليمات البرمجية الخاصة به، أو يستورد من "kysely"، أو لديه واجهة قاعدة بيانات مع أنواع الجداول، أو يسأل عن SQL الآمن للنوع في TypeScript. قم أيضًا بتشغيل الأخطاء الإملائية الشائعة مثل "kaisley" و"kysley". قم بالتنشيط حتى إذا قالوا فقط "منشئ الاستعلام" أو "كيف أفعل X في kysely" أو الصق الكود الذي يستخدم أنماط SelectFrom/insertInto/updateTable/deleteFrom. تغطي هذه المهارة العلاقات (jsonArrayFrom/jsonObjectFrom)، ومنشئي التعبير، وSQL الخام عبر علامة SQL، والاستعلامات المشروطة، والمساعدين القابلين لإعادة الاستخدام، وإدارة المخطط، والترحيلات، وخط أنابيب التنفيذ/تصحيح الأخطاء، والمكونات الإضافية، وجميع أنماط Kysely.
