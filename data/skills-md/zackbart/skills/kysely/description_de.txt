Typsicherer SQL-Abfrageerstellungsassistent für Kysely. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer Datenbankabfragen mit Kysely schreibt, eine Kysely-Datenbankinstanz einrichtet, Datenbanktypen definiert, Migrationen schreibt oder fragt, wie SQL-Operationen (SELECT, INSERT, UPDATE, DELETE, JOIN, CTE, MERGE, Transaktionen) in Kysely ausgeführt werden. Wird auch ausgelöst, wenn der Benutzer Kysely in seinem Code erwähnt, aus „Kysely“ importiert, über eine Datenbankschnittstelle mit Tabellentypen verfügt oder nach typsicherem SQL in TypeScript fragt. Wird auch bei häufigen Rechtschreibfehlern wie „kaisley“ und „kysley“ ausgelöst. Aktivieren Sie die Funktion, auch wenn sie nur „Query Builder“ oder „Wie mache ich X in Kysely“ sagen, oder fügen Sie Code ein, der die Muster „selectFrom/insertInto/updateTable/deleteFrom“ verwendet. Diese Fertigkeit umfasst Beziehungen (jsonArrayFrom/jsonObjectFrom), Ausdrucksgeneratoren, Roh-SQL über das SQL-Tag, bedingte Abfragen, wiederverwendbare Helfer, Schemaverwaltung, Migrationen, Ausführungspipeline/Debugging, Plugins und alle Kysely-Muster.
