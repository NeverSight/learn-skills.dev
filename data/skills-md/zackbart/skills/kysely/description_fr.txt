Assistant de création de requêtes SQL de type sécurisé pour Kysely. Utilisez cette compétence chaque fois que l'utilisateur écrit des requêtes de base de données avec Kysely, configure une instance de base de données Kysely, définit des types de bases de données, écrit des migrations ou demande comment effectuer des opérations SQL (SELECT, INSERT, UPDATE, DELETE, JOIN, CTE, MERGE, transactions) dans Kysely. Déclenchez également lorsque l'utilisateur mentionne kysely dans son code, importe depuis « kysely », dispose d'une interface de base de données avec des types de table ou pose des questions sur le SQL de type sécurisé dans TypeScript. Déclenchez également les fautes d'orthographe courantes telles que "kaisley" et "kysley". Activez même s'ils disent simplement "générateur de requêtes" ou "comment puis-je faire X dans kysely" ou collez le code qui utilise les modèles selectFrom/insertInto/updateTable/deleteFrom. Cette compétence couvre les relations (jsonArrayFrom/jsonObjectFrom), les générateurs d'expressions, le SQL brut via la balise sql, les requêtes conditionnelles, les assistants réutilisables, la gestion des schémas, les migrations, le pipeline d'exécution/débogage, les plugins et tous les modèles Kysely.
