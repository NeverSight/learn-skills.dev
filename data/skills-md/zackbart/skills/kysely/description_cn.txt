Kysely 的类型安全 SQL 查询生成器助手。每当用户使用 Kysely 编写数据库查询、设置 Kysely 数据库实例、定义数据库类型、编写迁移或询问如何在 Kysely 中执行 SQL 操作（SELECT、INSERT、UPDATE、DELETE、JOIN、CTE、MERGE、事务）时，请使用此技能。当用户在代码中提及 kysely、从“kysely”导入、具有带表类型的数据库接口或询问 TypeScript 中的类型安全 SQL 时也会触发。还会触发常见的拼写错误，例如“kaisley”和“kysley”。即使他们只是说“查询构建器”或“我如何在 kysely 中执行 X”或粘贴使用 selectFrom/insertInto/updateTable/deleteFrom 模式的代码，也可以激活。该技能涵盖关系 (jsonArrayFrom/jsonObjectFrom)、表达式构建器、通过 sql 标签的原始 SQL、条件查询、可重用帮助器、模式管理、迁移、执行管道/调试、插件和所有 Kysely 模式。
