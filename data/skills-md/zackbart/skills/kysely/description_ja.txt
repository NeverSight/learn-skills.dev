Kysely 用のタイプセーフな SQL クエリ ビルダー アシスタント。ユーザーが Kysely でデータベース クエリを作成する場合、Kysely データベース インスタンスをセットアップする場合、データベース タイプを定義する場合、移行を作成する場合、または Kysely で SQL 操作 (SELECT、INSERT、UPDATE、DELETE、JOIN、CTE、MERGE、トランザクション) を実行する方法を尋ねる場合には、このスキルを使用してください。また、ユーザーがコード内で kysely に言及した場合、「kysely」からインポートした場合、テーブル型のデータベース インターフェイスがある場合、または TypeScript でのタイプ セーフ SQL について質問した場合にもトリガーされます。 「kaisley」や「kysley」などの一般的なスペルミスでもトリガーされます。 「クエリ ビルダー」または「kysely で X を実行するにはどうすればよいですか」というだけの場合でもアクティブ化するか、selectFrom/insertInto/updateTable/deleteFrom パターンを使用するコードを貼り付けます。このスキルは、関係 (jsonArrayFrom/jsonObjectFrom)、式ビルダー、sql タグを介した生の SQL、条件付きクエリ、再利用可能なヘルパー、スキーマ管理、移行、実行パイプライン/デバッグ、プラグイン、およびすべての Kysely パターンをカバーします。
