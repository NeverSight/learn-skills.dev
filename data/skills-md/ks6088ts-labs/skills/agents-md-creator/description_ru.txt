Навыки создания инструкций "AGENTS.md" для кодирования ИИ-агентов. Используйте это, если хотите создать файл, объединяющий контекст и инструкции для работы AI-агента в вашем проекте. Он запускается такими запросами, как «Создать AGENTS.md», «Создать инструкции для ИИ-агентов» и «Создать README для агентов». Это открытый стандартный формат, который может обычно использоваться несколькими агентами искусственного интеллекта, такими как OpenAI Codex, Claude Code, GitHub Copilot и Cursor.
