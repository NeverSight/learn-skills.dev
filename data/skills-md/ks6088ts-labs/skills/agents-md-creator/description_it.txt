Competenze per la creazione di istruzioni "AGENTS.md" per agenti di codifica AI. Utilizzalo quando desideri creare un file che aggrega il contesto e le istruzioni con cui l'agente AI può lavorare nel tuo progetto. Viene attivato da richieste come "Crea AGENTS.md", "Crea istruzioni per agenti AI" e "Crea README per agenti". È un formato standard aperto che può essere utilizzato comunemente da più agenti AI, come OpenAI Codex, Claude Code, GitHub Copilot e Cursor.
