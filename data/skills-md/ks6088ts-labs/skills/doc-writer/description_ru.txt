Навыки, необходимые для написания документов. Используйте, когда пользователи хотят создавать, редактировать или обновлять документы на основе PRD. Этот навык поддерживает написание технической документации, проектной документации, руководств пользователя, документов API и т. д. со ссылкой на спецификации требований к продукту, перечисленные в docs/prd.md. Запускается при упоминании задачи документирования, например «Написать документ», «Создать спецификацию» или «Создать руководство».
