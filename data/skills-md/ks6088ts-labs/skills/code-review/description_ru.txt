Навыки проведения code review. Используется, когда пользователь запрашивает проверку кода с точки зрения качества кода, безопасности, тестирования, производительности или архитектуры. Запускается при упоминании задачи проверки кода, например «проверка кода», «проверка кода», «проверка PR», «проверка этого кода» и т. д. Мы расставляем приоритеты и указываем на уязвимости безопасности, логические ошибки, проблемы с качеством тестирования и производительностью, а также предоставляем конкретные предложения по улучшению.
