IDEAS (навык адаптера IDE) — подключитесь к расширению VS Code адаптера IDE через WebSocket, чтобы использовать функции VS Code из CLI. Используйте этот навык, когда пользователь запрашивает: поиск/замену текста в базе кода, поиск определений символов или ссылок, получение диагностики/ошибок/предупреждений, составление списка символов документа, поиск файлов по имени, просмотр истории коммитов git, сравнение версий файлов (diff), откат файлов к предыдущему коммиту git или доступ к локальной истории сохранений VS Code (снимки между коммитами). Триггеры: «найти все варианты использования», «перейти к определению», «показать ошибки», «история git», «текущая разница с HEAD», «откат к фиксации», «локальная история», «найти файлы с именами», «ИДЕИ», «Адаптер IDE». Требуется: расширение адаптера IDE, работающее в VS Code (порт по умолчанию 7200).
