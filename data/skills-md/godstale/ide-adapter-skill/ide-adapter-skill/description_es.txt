IDEAS (IDE Adapter Skill): conéctese a la extensión IDE Adapter VS Code a través de WebSocket para usar las funciones de VS Code desde CLI. Utilice esta habilidad cuando el usuario solicite: buscar/reemplazar texto en una base de código, encontrar definiciones o referencias de símbolos, obtener diagnósticos/errores/advertencias, enumerar símbolos de documentos, buscar archivos por nombre, ver el historial de confirmaciones de git, comparar versiones de archivos (diff), revertir archivos a una confirmación de git anterior o acceder al historial de guardado local de VS Code (instantáneas entre confirmaciones). Activadores: "buscar todos los usos de", "ir a la definición", "mostrar errores en", "historial de git de", "diff actual vs HEAD", "revertir para confirmar", "historial local", "buscar archivos con nombre", "IDEAS", "Adaptador IDE". Requiere: Extensión del adaptador IDE ejecutándose en VS Code (puerto predeterminado 7200).
