IDÉES (IDE Adapter Skill) - Connectez-vous à l'extension IDE Adapter VS Code via WebSocket pour utiliser les fonctionnalités VS Code de CLI. Utilisez cette compétence lorsque l'utilisateur demande : rechercher/remplacer du texte dans une base de code, trouver des définitions ou des références de symboles, obtenir des diagnostics/erreurs/avertissements, répertorier les symboles de documents, rechercher des fichiers par nom, afficher l'historique des validations git, comparer les versions de fichiers (diff), restaurer des fichiers vers une validation git précédente ou accéder à l'historique de sauvegarde local de VS Code (instantanés entre les validations). Déclencheurs : "trouver toutes les utilisations de", "aller à la définition", "afficher les erreurs dans", "l'historique git de", "diff current vs HEAD", "revenir en arrière pour valider", "historique local", "trouver les fichiers nommés", "IDEAS", "Adaptateur IDE". Nécessite : extension de l'adaptateur IDE exécutée dans VS Code (port par défaut 7200).
