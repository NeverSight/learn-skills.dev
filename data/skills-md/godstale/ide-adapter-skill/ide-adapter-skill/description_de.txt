IDEAS (IDE-Adapter-Skill) – Stellen Sie über WebSocket eine Verbindung zur IDE-Adapter-VS-Code-Erweiterung her, um VS-Code-Funktionen von CLI zu nutzen. Verwenden Sie diese Fähigkeit, wenn der Benutzer Folgendes wünscht: Text in einer Codebasis suchen/ersetzen, Symboldefinitionen oder Referenzen finden, Diagnosen/Fehler/Warnungen abrufen, Dokumentsymbole auflisten, Dateien nach Namen durchsuchen, Git-Commit-Verlauf anzeigen, Dateiversionen vergleichen (Diff), Dateien auf einen früheren Git-Commit zurücksetzen oder auf den lokalen Speicherverlauf von VS Code zugreifen (Schnappschüsse zwischen Commits). Auslöser: „Alle Verwendungen von finden“, „zur Definition gehen“, „Fehler anzeigen in“, „Git-Verlauf von“, „Aktuell vs. HEAD unterscheiden“, „Zurück zum Festschreiben“, „Lokaler Verlauf“, „Dateien mit Namen suchen“, „IDEEN“, „IDE-Adapter“. Erfordert: IDE-Adaptererweiterung, die in VS Code ausgeführt wird (Standardport 7200).
