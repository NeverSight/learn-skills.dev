IDEAS（IDE 适配器技能）- 通过 WebSocket 连接到 IDE 适配器 VS Code 扩展，以使用 CLI 中的 VS Code 功能。当用户要求执行以下操作时，请使用此技能：搜索/替换代码库中的文本、查找符号定义或引用、获取诊断/错误/警告、列出文档符号、按名称搜索文件、查看 git 提交历史记录、比较文件版本（差异）、将文件回滚到以前的 git 提交或访问 VS Code 本地保存历史记录（提交之间的快照）。触发器：“查找所有用法”、“转到定义”、“显示错误”、“git 历史记录”、“比较当前与 HEAD”、“回滚到提交”、“本地历史记录”、“查找名为”的文件、“IDEAS”、“IDE 适配器”。需要：在 VS Code 中运行的 IDE 适配器扩展（默认端口 7200）。
