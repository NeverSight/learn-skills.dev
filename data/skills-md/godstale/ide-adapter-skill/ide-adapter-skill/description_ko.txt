IDEAS(IDE 어댑터 기술) - WebSocket을 통해 IDE 어댑터 VS Code 확장에 연결하여 CLI에서 VS Code 기능을 사용합니다. 사용자가 코드베이스에서 텍스트 검색/바꾸기, 기호 정의 또는 참조 찾기, 진단/오류/경고 가져오기, 문서 기호 나열, 이름으로 파일 검색, git 커밋 기록 보기, 파일 버전 비교(diff), 이전 git 커밋으로 파일 롤백 또는 VS Code 로컬 저장 기록 액세스(커밋 간 스냅샷)를 요청할 때 이 기술을 사용합니다. 트리거: "모든 사용법 찾기", "정의로 이동", "오류 표시", "git 기록", "현재와 HEAD 비교", "커밋으로 롤백", "로컬 기록", "이름이 지정된 파일 찾기", "IDEAS", "IDE 어댑터". 필요 사항: VS Code에서 실행되는 IDE 어댑터 확장(기본 포트 7200)
