IDEAS (IDE アダプター スキル) - WebSocket 経由で IDE アダプター VS Code 拡張機能に接続し、CLI から VS Code 機能を使用します。このスキルは、コードベース内のテキストの検索/置換、シンボル定義または参照の検索、診断/エラー/警告の取得、ドキュメント シンボルの一覧表示、ファイルの名前による検索、git コミット履歴の表示、ファイル バージョンの比較 (差分)、ファイルを以前の git コミットにロールバックする、または VS Code のローカル保存履歴 (コミット間のスナップショット) にアクセスすることをユーザーが要求した場合に使用します。トリガー: 「すべての使用箇所を検索」、「定義に移動」、「エラーを表示」、「git 履歴を表示」、「現在と HEAD を比較」、「コミットにロールバック」、「ローカル履歴」、「名前付きファイルを検索」、「IDEAS」、「IDE アダプター」。必要なもの: VS Code で実行される IDE アダプター拡張機能 (デフォルトのポート 7200)。
