IDEAS (مهارة محول IDE) - اتصل بامتداد IDE محول VS Code عبر WebSocket لاستخدام ميزات VS Code من CLI. استخدم هذه المهارة عندما يطلب المستخدم ما يلي: البحث/استبدال النص في قاعدة التعليمات البرمجية، أو العثور على تعريفات الرموز أو المراجع، أو الحصول على التشخيصات/الأخطاء/التحذيرات، أو إدراج رموز المستندات، أو البحث في الملفات حسب الاسم، أو عرض سجل التزام git، أو مقارنة إصدارات الملفات (الفرق)، أو استرجاع الملفات إلى التزام git سابق، أو الوصول إلى سجل الحفظ المحلي لـ VS Code (لقطات بين الالتزامات). المشغلات: "البحث عن جميع استخدامات"، "الانتقال إلى التعريف"، "إظهار الأخطاء في"، "سجل git"، "فرق التيار مقابل الرأس"، "التراجع للالتزام"، "السجل المحلي"، "البحث عن الملفات المسماة"، "IDEAS"، "محول IDE". يتطلب: ملحق محول IDE يعمل في VS Code (المنفذ الافتراضي 7200).
