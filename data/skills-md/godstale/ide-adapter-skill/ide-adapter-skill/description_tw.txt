IDEAS（IDE 適配器技能）- 透過 WebSocket 連接到 IDE 適配器 VS Code 擴展，以使用 CLI 中的 VS Code 功能。當使用者要求執行以下操作時，請使用此技能：搜尋/取代程式碼庫中的文字、尋找符號定義或引用、取得診斷/錯誤/警告、列出文件符號、按名稱搜尋檔案、查看 git 提交歷史記錄、比較文件版本（差異）、將文件回滾到先前的 git 提交或存取 VS Code 本地記錄（提交之間的快照）。觸發器：「尋找所有用法」、「前往定義」、「顯示錯誤」、「git 歷史記錄」、「比較目前與 HEAD」、「回滾到提交」、「本地歷史記錄」、「尋找名為」的檔案、「IDEAS」、「IDE 適配器」。需要：在 VS Code 中執行的 IDE 適配器擴充（預設連接埠 7200）。
