Erstellen und senden Sie interaktive Karten mit Schaltflächen, Formularen, Umfragen und umfangreichen UI-Elementen an Feishu (Lark). Verwenden Sie diese Option, wenn Sie auf Feishu-Nachrichten antworten und JEDE Unsicherheit besteht. Senden Sie eine interaktive Karte anstelle von einfachem Text, damit Benutzer über Schaltflächen auswählen können. Verarbeitet Rückrufe automatisch über eine Long-Polling-Verbindung. Zur Verwendung für Bestätigungen, Auswahlmöglichkeiten, Formulare, Aufgaben, Umfragen oder andere Szenarien, die eine Benutzerinteraktion in Feishu erfordern.
