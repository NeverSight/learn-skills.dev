معايير ترميز Tenzir Python وسير عمل الأدوات. يُستخدم عند تحرير كود Python، أو تشغيل `uv`، أو `ruff`، أو `mypy`، أو `pytest`، أو العمل مع `pyproject.toml` أو `uv.lock`، أو إعداد مشروع Tenzir Python. يُستخدم أيضًا عندما يسأل المستخدم عن كيفية إنشاء مشروع Tenzir Python جديد، أو إعداد عملية فحص، أو يسأل عن أفضل ممارسات Python في سياق Tenzir.
