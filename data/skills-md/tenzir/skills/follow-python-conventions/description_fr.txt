Normes de codage Tenzir Python et flux de travail d'outillage. À utiliser lors de l'édition de code Python, de l'exécution de `uv`, `ruff`, `mypy` ou `pytest`, de l'utilisation de `pyproject.toml` ou `uv.lock`, ou de la configuration d'un projet Tenzir Python. À utiliser également lorsque l'utilisateur demande comment structurer un nouveau projet Tenzir Python, configurer le linting ou pose des questions sur les meilleures pratiques Python dans le contexte Tenzir.
