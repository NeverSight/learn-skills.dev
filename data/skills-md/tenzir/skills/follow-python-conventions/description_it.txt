Standard di codifica Tenzir Python e flusso di lavoro degli strumenti. Da utilizzare quando si modifica il codice Python, si esegue `uv`, `ruff`, `mypy` o `pytest`, si lavora con `pyproject.toml` o `uv.lock` o si imposta un progetto Tenzir Python. Da utilizzare anche quando l'utente chiede come strutturare un nuovo progetto Tenzir Python, impostare il linting o chiedere informazioni sulle migliori pratiche Python nel contesto Tenzir.
