Coordine las actualizaciones de docs.tenzir.com junto con los cambios de código. Úselo al preparar el proceso de pago `.docs/`, crear una rama de documentos coincidente, abrir una solicitud de extracción `tenzir/docs` o vincular documentos y códigos PR. Úselo también cuando el usuario mencione "actualizar los documentos", "docs PR", ".docs/", o cuando un cambio de código afecte el comportamiento del usuario que debería reflejarse en docs.tenzir.com.
