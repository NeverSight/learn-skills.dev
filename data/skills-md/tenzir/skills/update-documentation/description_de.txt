Koordinieren Sie Aktualisierungen von docs.tenzir.com sowie Codeänderungen. Verwenden Sie diese Option, wenn Sie das Auschecken von „.docs/“ vorbereiten, einen passenden Dokumentzweig erstellen, eine „tenzir/docs“-Pull-Anfrage öffnen oder Dokumente und Code-PRs verknüpfen. Verwenden Sie es auch, wenn der Benutzer „Update the docs“, „docs PR“, „.docs/“ erwähnt oder wenn eine Codeänderung Auswirkungen auf das Benutzerverhalten hat, das sich auf docs.tenzir.com widerspiegeln sollte.
