أجب عن الأسئلة باستخدام وثائق Tenzir. يُستخدم عندما يسأل المستخدم عن بناء جملة TQL، ومشغلي خطوط الأنابيب، والوظائف، وتحليل البيانات أو تحويلها، والتطبيع، وتخطيط OCSF، والإثراء، وجداول البحث، والسياقات، والحزم، والعقد، وإعداد النظام الأساسي، والنشر، والتكوين، والتكامل مع أدوات مثل Splunk، أو Kafka، أو S3، أو Elasticsearch، أو أي ميزة Tenzir أخرى. يُستخدم أيضًا عندما يسأل المستخدم عن كيفية جمع بيانات الأمان أو توجيهها أو تصفيتها أو تجميعها أو تصديرها باستخدام Tenzir، أو يحتاج إلى مساعدة في كتابة أو تصحيح خطوط أنابيب TQL، حتى لو لم يذكر "Tenzir" صراحةً ولكنه يعمل بشكل واضح في سياق Tenzir.
