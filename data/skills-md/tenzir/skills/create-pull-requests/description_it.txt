Spedisci modifiche tramite richiesta pull nei progetti Tenzir. Da utilizzare quando l'utente desidera aprire un PR, inviare un ramo, aggiungere o aggiornare voci del registro delle modifiche, creare una richiesta pull per la revisione o quando dice "spedisci questo", "apri un PR", "invia questo per la revisione" o "invia e crea una richiesta pull". Utilizzare anche per modifiche successive a un PR esistente.
