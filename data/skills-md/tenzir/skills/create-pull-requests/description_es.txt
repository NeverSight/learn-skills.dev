Enviar cambios mediante pull request en proyectos de Tenzir. Úselo cuando el usuario quiera abrir un PR, enviar una rama, agregar o actualizar entradas del registro de cambios, crear una solicitud de extracción para revisión o cuando diga "enviar esto", "abrir un PR", "enviar esto para revisión" o "enviar y crear una solicitud de extracción". Úselo también para cambios de seguimiento en un RP existente.
