Expédiez les modifications via une pull request dans les projets Tenzir. À utiliser lorsque l'utilisateur souhaite ouvrir un PR, pousser une branche, ajouter ou mettre à jour des entrées du journal des modifications, créer une demande d'extraction pour révision, ou lorsqu'il dit « envoyer ceci », « ouvrir un PR », « envoyer ceci pour révision » ou « pousser et créer une demande d'extraction ». À utiliser également pour le suivi des modifications apportées à un PR existant.
