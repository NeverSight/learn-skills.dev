Versenden Sie Änderungen per Pull-Request in Tenzir-Projekten. Verwenden Sie diese Option, wenn der Benutzer eine PR öffnen, einen Zweig pushen, Änderungsprotokolleinträge hinzufügen oder aktualisieren, eine Pull-Anfrage zur Überprüfung erstellen möchte oder wenn er sagen möchte: „Versenden“, „Öffnen Sie eine PR“, „Senden Sie dies zur Überprüfung“ oder „Push und erstellen Sie eine Pull-Anfrage“. Auch für nachträgliche Änderungen an einer bestehenden PR verwenden.
