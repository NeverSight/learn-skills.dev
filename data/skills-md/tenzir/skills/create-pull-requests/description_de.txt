Versenden Sie Änderungen per Pull-Request in Tenzir-Projekten. Verwenden Sie diese Option, wenn der Benutzer eine PR öffnen, einen Zweig pushen, Änderungsprotokolleinträge hinzufügen oder aktualisieren, eine Pull-Anfrage zur Überprüfung erstellen möchte oder wenn er „dies versenden“, „eine PR öffnen“, „dies zur Überprüfung senden“ oder „pushen und eine Pull-Anfrage erstellen“ sagt. Auch für nachträgliche Änderungen an einer bestehenden PR verwenden.
