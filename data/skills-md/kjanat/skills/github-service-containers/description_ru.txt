Настройте контейнеры служб Docker (Redis, PostgreSQL и т. д.) в качестве дополнительных служб в рабочих процессах GitHub Actions для интеграционного тестирования. Используйте при добавлении баз данных, кэшей или очередей сообщений в рабочие процессы CI, а также при отладке сети сервисного контейнера и проверках работоспособности.
