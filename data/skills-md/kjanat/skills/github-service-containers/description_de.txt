Konfigurieren Sie Docker-Dienstcontainer (Redis, PostgreSQL usw.) als Sidecar-Dienste in GitHub Actions-Workflows für Integrationstests. Verwenden Sie diese Option, wenn Sie Datenbanken, Caches oder Nachrichtenwarteschlangen zu CI-Workflows hinzufügen oder Netzwerk- und Integritätsprüfungen für Service-Container debuggen.
