قرر كيفية تنفيذ تغييرات وقت التشغيل وواجهة برمجة التطبيقات (API) في openai-agents-python قبل تحرير التعليمات البرمجية. يُستخدم عندما تقوم مهمة بتغيير واجهات برمجة التطبيقات المصدرة، أو سلوك وقت التشغيل، أو الحالة التسلسلية، أو الاختبارات، أو المستندات، وتحتاج إلى اختيار حدود التوافق، سواء كانت الحشوات أو عمليات الترحيل مضمونة، وعندما يمكن إعادة كتابة الواجهات غير المُصدرة مباشرةً.
