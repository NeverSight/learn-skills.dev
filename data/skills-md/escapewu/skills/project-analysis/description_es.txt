Herramienta de análisis de proyectos para analizar la arquitectura del sistema y el flujo de datos del módulo de una base de código. Utilice esta habilidad cuando los usuarios necesiten comprender la estructura del proyecto, generar diagramas de arquitectura, analizar el flujo de datos entre módulos o crear diagramas de secuencia. Admite la generación de gráficos visuales en la sintaxis de Mermaid. Escenarios aplicables: (1) Clasificación de la arquitectura del proyecto, (2) Análisis de dependencia del módulo, (3) Seguimiento del flujo de datos, (4) Incorporación de proyectos de nuevos miembros, (5) Generación de documentación técnica
