Инструмент анализа проекта для анализа системной архитектуры и потока данных модулей кодовой базы. Используйте этот навык, когда пользователям необходимо понять структуру проекта, создать диаграммы архитектуры, проанализировать поток данных между модулями или создать диаграммы последовательности. Поддерживает вывод визуальных диаграмм в синтаксисе Mermaid. Применимые сценарии: (1) Сортировка архитектуры проекта, (2) Анализ зависимостей модулей, (3) Отслеживание потока данных, (4) Регистрация новых участников проекта, (5) Генерация технической документации.
