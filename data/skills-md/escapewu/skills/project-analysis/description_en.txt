Project analysis tool for analyzing the system architecture and module data flow of a codebase. Use this skill when users need to understand project structure, generate architecture diagrams, analyze data flow between modules, or create sequence diagrams. Supports outputting visual charts in Mermaid syntax. Applicable scenarios: (1) Project architecture sorting, (2) Module dependency analysis, (3) Data flow tracking, (4) New member project onboarding, (5) Technical documentation generation
