ينسق تطوير الميزات المستندة إلى المواصفات في Linear - يكتشف تلقائيًا إطار التطوير المثبت (Spec-Kit، BMAD METHOD، Superpowers) ويتبع سير العمل الأصلي أثناء مزامنة العناصر مع Linear عبر MCP (المشاريع والمستندات والمعالم الرئيسية والقضايا والتعليقات). يدعم إرسال الوكيل المستقل: يولد وكلاء فرعيين لـ Claude لتنفيذ المشكلات الخطية بأقل قدر من المشاركة البشرية.
استخدم هذه المهارة عندما يريد المستخدم: تخطيط ميزة جديدة أو إطلاقها، أو تقسيم المتطلبات إلى مشكلات خطية، أو تشغيل سير عمل تطوير منظم يتم تتبعه في Linear، أو مزامنة المواصفات أو الخطط مع Linear، أو إدارة دورة حياة الميزة من الفكرة إلى الاكتمال، أو الحصول على تقرير حالة المشروع من Linear، أو إرسال الوكلاء لتنفيذ المهام، أو إعداد مشروع Linear جديد مع الوكلاء، أو أتمتة التطوير باستخدام وكلاء الذكاء الاصطناعي.
