Armar una nueva "habilidad inicial" de tipo integración: un SKILL.md con recursos de referencia y entradas de avisos.jsonl que enseñan a un agente a generar código de integración funcional y comprobable desde cero.
- Úselo cuando: el desarrollador desea crear una nueva habilidad para un SDK o integración de marco que los agentes deberían poder implementar y probar.
- Pregunte primero: ¿Es esta una habilidad de integración (generación de código + pruebas) o una habilidad de herramienta (contenedor CLI sin pruebas)? Utilice esta habilidad únicamente para habilidades de tipo integración.
