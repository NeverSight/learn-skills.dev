Cette compétence est utilisée pour automatiser l'envoi de messages WeChat via AppleScript sur macOS. Il est invoqué lorsque l'utilisateur doit « envoyer un message WeChat », « envoyer un message à un ami WeChat », « envoyer automatiquement sur WeChat », « envoyer des messages dans un groupe WeChat » ou mentionner « message WeChat » ou « automatisation WeChat ».
