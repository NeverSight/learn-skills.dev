Cette compétence est utilisée pour automatiser l'envoi de messages WeChat sur macOS via AppleScript. Utilisez cette compétence lorsque l'utilisateur a besoin de « envoyer un message WeChat », « d'envoyer un message à un ami WeChat », « d'envoyer automatiquement sur WeChat », « d'envoyer un message dans un groupe WeChat » ou de mentionner « message WeChat » ou « automatisation WeChat ».
