Dieser Skill wird verwendet, um das Senden von WeChat-Nachrichten über AppleScript unter macOS zu automatisieren. Es wird aufgerufen, wenn der Benutzer „eine WeChat-Nachricht senden“, „einem WeChat-Freund eine Nachricht senden“, „automatisch auf WeChat senden“, „Nachrichten in einer WeChat-Gruppe senden“ muss oder „WeChat-Nachricht“ oder „WeChat-Automatisierung“ erwähnt.
