Genera un CLAUDE.md (o AGENTS.md) specifico per il progetto con standard di codifica, guide di stile e best practice. Si attiva su qualsiasi richiesta per impostare un progetto, inizializzare standard di codifica, creare o aggiornare CLAUDE.md o AGENTS.md, configurare le istruzioni dell'agente, aggiungere regole di qualità del codice o avviare una nuova codebase. Usa questa abilità all'inizio di ogni nuovo progetto.
