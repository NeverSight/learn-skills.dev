Создайте файл CLAUDE.md (или AGENTS.md) для конкретного проекта, содержащий стандарты кодирования, руководства по стилю и лучшие практики. Запускается при любом запросе на настройку проекта, инициализацию стандартов кодирования, создание или обновление CLAUDE.md или AGENTS.md, настройку инструкций агента, добавление правил качества кода или запуск новой базы кода. Используйте этот навык в начале каждого нового проекта.
