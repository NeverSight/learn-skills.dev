Развертывайте приложения на платформе Temps с автоматическим обнаружением платформы, созданием файлов Dockerfile и оркестровкой контейнеров. Поддерживает приложения Next.js, Vite, React, Node.js, Python, Go, Rust, Java и C#. Используйте, когда пользователь хочет: (1) развернуть свое приложение в Temps, (2) настроить CI/CD с помощью Temps, (3) настроить параметры развертывания, (4) создать файл Dockerfile для Temps, (5) развернуть контейнерное приложение, (6) настроить автоматическое развертывание из Git. Триггеры: «развертывание во временные файлы», «развертывание временных файлов», «отправка во временные файлы», «контейнеризация для временных файлов», «временные файлы ci/cd».
