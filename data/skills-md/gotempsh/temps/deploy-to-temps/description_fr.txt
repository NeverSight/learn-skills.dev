Déployez des applications sur la plateforme Temps avec la détection automatique du framework, la génération Dockerfile et l'orchestration des conteneurs. Prend en charge les applications Next.js, Vite, React, Node.js, Python, Go, Rust, Java et C#. À utiliser lorsque l'utilisateur souhaite : (1) déployer son application sur Temps, (2) configurer CI/CD avec Temps, (3) configurer les paramètres de déploiement, (4) créer un fichier Docker pour Temps, (5) déployer une application conteneurisée, (6) configurer des déploiements automatiques à partir de Git. Déclencheurs : "déployer vers temps", "déploiement temps", "push vers temps", "conteneuriser pour temps", "temps ci/cd".
