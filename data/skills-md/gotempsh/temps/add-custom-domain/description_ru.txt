Настройте персональные домены для развертываний Temps с помощью автоматических сертификатов SSL/TLS с помощью Let's Encrypt. Поддерживает вызовы HTTP-01 и DNS-01, домены с подстановочными знаками и интеграцию Cloudflare DNS. Используйте, когда пользователь хочет: (1) добавить собственный домен в свое приложение Temps, (2) настроить SSL-сертификаты, (3) настроить DNS для Temps, (4) добавить домен с подстановочными знаками, (5) настроить HTTPS для их развертывания, (6) настроить Cloudflare с помощью Temps. Триггеры: «пользовательский домен», «добавить домен», «ssl-сертификат», «настройка https», «домен с подстановочными знаками», «конфигурация DNS», «температура Cloudflare».
