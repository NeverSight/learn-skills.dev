Best practices for building and improving CLIs that work for both humans and AI agents. Activate when building, designing, reviewing, refactoring, or improving any new or existing command-line tool or CLI application. Keywords - CLI, command line, terminal, argparse, commander, cobra, clap, oclif, typer, click, flags, subcommands, exit codes, help text, error messages. NOT for TUI/full-screen apps, GUI apps, or REST API design.
