Mejores prácticas para crear y mejorar CLI que funcionen tanto para humanos como para agentes de IA. Actívelo al crear, diseñar, revisar, refactorizar o mejorar cualquier herramienta de línea de comandos o aplicación CLI nueva o existente. Palabras clave: CLI, línea de comando, terminal, argparse, Commander, cobra, clap, oclif, typer, click, banderas, subcomandos, códigos de salida, texto de ayuda, mensajes de error. NO para aplicaciones TUI/de pantalla completa, aplicaciones GUI o diseño de API REST.
