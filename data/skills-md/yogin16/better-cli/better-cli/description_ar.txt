أفضل الممارسات لبناء وتحسين CLIs التي تناسب كل من البشر ووكلاء الذكاء الاصطناعي. يتم التنشيط عند إنشاء أو تصميم أو مراجعة أو إعادة هيكلة أو تحسين أي أداة سطر أوامر جديدة أو موجودة أو تطبيق CLI. الكلمات الرئيسية - CLI، سطر الأوامر، المحطة الطرفية، argparse، القائد، الكوبرا، التصفيق، oclif، الطابعة، النقر، الأعلام، الأوامر الفرعية، رموز الخروج، نص المساعدة، رسائل الخطأ. ليس لتطبيقات TUI/ملء الشاشة، أو تطبيقات واجهة المستخدم الرسومية، أو تصميم REST API.
