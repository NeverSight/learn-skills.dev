Best practice per creare e migliorare CLI che funzionino sia per gli esseri umani che per gli agenti IA. Attivalo durante la creazione, la progettazione, la revisione, il refactoring o il miglioramento di qualsiasi strumento da riga di comando o applicazione CLI nuova o esistente. Parole chiave: CLI, riga di comando, terminale, argparse, comandante, cobra, clap, oclif, typer, clic, flag, sottocomandi, codici di uscita, testo della guida, messaggi di errore. NON per app TUI/a schermo intero, app GUI o progettazione API REST.
