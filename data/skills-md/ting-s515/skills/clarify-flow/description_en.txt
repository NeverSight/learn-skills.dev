When a user provides a process description document and requests organizing or optimizing the flow, this skill must be loaded. Rewrite vague, colloquial process descriptions into precise, structured formats so that every step, decision branch, and loop is clearly traceable and ready for conversion into a Mermaid flowchart. Triggering scenarios include but are not limited to: “Help me organize this flow,” “structure this description,” “optimize the process documentation,” “clarify flow.”
