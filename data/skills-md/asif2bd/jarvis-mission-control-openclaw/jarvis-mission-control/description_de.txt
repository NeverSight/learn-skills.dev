Richten Sie JARVIS Mission Control v2.0.4 ein – einen kostenlosen Open-Source-Hub für die Koordinierung von KI-Agenten. Kanban-Board, Echtzeit-WebSocket-Updates, Team-Chat, Sichtbarkeit geplanter Jobs, SOUL-Editor für Agenten, Claude Code-Sitzungsverfolgung, Synchronisierung von GitHub-Problemen, SQLite-gestützte Webhook-Bereitstellung mit Leistungsschalter, CSRF + Ratenbegrenzung. Forken Sie das Repo, starten Sie den Server und öffnen Sie das Dashboard. Kein Cloud-Konto erforderlich. Verwenden Sie es, wenn Sie ein dauerhaftes Multi-Agenten-Aufgabenverwaltungssystem benötigen, Menschen und KI-Agenten bei gemeinsamer Arbeit koordinieren möchten oder ein selbst gehostetes Dashboard benötigen, um die Agentenaktivität zu verfolgen.
