Сократите использование токенов OpenClaw и затраты на API за счет интеллектуальной маршрутизации моделей, оптимизации пульса, отслеживания бюджета и встроенных функций 2026.2.15 (обрезка сеансов, ограничения размера начальной загрузки, выравнивание TTL кэша). Используйте, когда стоимость токенов высока, превышены ограничения скорости API или когда размещается несколько агентов в большом масштабе. Четыре исполняемых сценария (context_optimizer, model_router, heartbeat_optimizer, token_tracker) предназначены только для локального использования — никаких сетевых запросов, никаких вызовов подпроцессов, никаких модификаций системы. Справочные файлы (PROVIDERS.md, config-patches.json) документируют дополнительные стратегии работы с несколькими поставщиками, для которых требуются внешние ключи API и доступ к сети, если вы решите их использовать. Подробную информацию смотрите на сайте SECURITY.md.
