استخدم هذه المهارة عندما يحتاج المستخدم إلى تخطيط محتوى تحسين محركات البحث، أو إجراء بحث عن الكلمات الرئيسية، أو إنشاء تقويم محتوى، أو تعيين هدف البحث لأنواع الصفحات، أو إنشاء استراتيجية ربط داخلية. هذه هي مهارة الإستراتيجية والتخطيط - لكتابة المحتوى، راجع محتوى تحسين محركات البحث، وللتنفيذ الفني، راجع تقنية تحسين محركات البحث، وللتدقيق، راجع تدقيق محركات البحث.
