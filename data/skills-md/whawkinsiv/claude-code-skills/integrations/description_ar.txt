استخدم هذه المهارة عندما يحتاج المستخدم إلى الاتصال بخدمات الجهات الخارجية، أو إعداد واجهات برمجة التطبيقات، أو إضافة OAuth، أو تكوين خطافات الويب، أو دمج أدوات مثل Slack، أو Zapier، أو موفري البريد الإلكتروني، أو معالجات الدفع. يغطي أنماط تكامل واجهة برمجة التطبيقات (API)، وتدفقات المصادقة، ومعالجة خطاف الويب، وبناء عمليات التكامل التي يمكن للمؤسسين غير التقنيين الحفاظ عليها.
