استخدم هذه المهارة عندما يحتاج المستخدم إلى تقليل معدل فقدان العملاء، أو تحسين الاحتفاظ بهم، أو فهم سبب مغادرة العملاء، أو إنشاء حملات استرداد الأموال، أو زيادة القيمة الدائمة للعميل، أو خلق الثبات في منتجهم. يغطي تحليل المخلفات، وتحديد المخاطر، وتكتيكات الاحتفاظ، وتسلسلات استعادة الربح، وإيرادات التوسعة لـ SaaS التي تم تشغيلها.
