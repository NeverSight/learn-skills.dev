استخدم هذه المهارة عندما يحتاج المستخدم إلى جمع تعليقات المستخدمين، أو تشغيل الاستطلاعات، أو إعداد NPS، أو تحديد أولويات طلبات الميزات، أو إغلاق حلقة التعليقات، أو تحويل مدخلات العميل إلى قرارات المنتج. يغطي أساليب جمع التعليقات، وتصميم الاستطلاع، وأطر تحديد الأولويات، وبناء ثقافة المنتج القائمة على التعليقات.
