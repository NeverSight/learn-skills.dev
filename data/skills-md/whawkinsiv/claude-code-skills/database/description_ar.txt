استخدم هذه المهارة عندما يحتاج المستخدم إلى اختيار قاعدة بيانات، أو تصميم مخطط، أو إعداد Supabase أو قاعدة بيانات أخرى، أو كتابة الاستعلامات، أو التعامل مع عمليات الترحيل، أو إصلاح المشكلات المتعلقة بالبيانات. يغطي اختيار قاعدة البيانات، وتصميم المخطط، وأمان مستوى الصف، وعمليات الترحيل، والأنماط الشائعة لتطبيقات SaaS التي أنشأها مؤسسون غير تقنيين.
