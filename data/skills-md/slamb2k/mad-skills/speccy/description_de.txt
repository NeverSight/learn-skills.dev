Tiefgreifende Interviewfähigkeiten zur Erstellung umfassender Spezifikationen. Überprüft vorhandenen Code und Dokumente und befragt den Benutzer dann in mehreren Runden gezielter Fragen zu technischer Implementierung, UI/UX, Bedenken und Kompromissen. Erstellt eine strukturierte Spezifikation in specs/. Verwenden Sie diese Option, wenn Sie eine neue Funktion, ein neues System oder eine größere Änderung starten, für die eine Spezifikation erforderlich ist.
