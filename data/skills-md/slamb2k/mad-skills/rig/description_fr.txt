Amorcez de manière idempotente n'importe quel référentiel avec des outils de développement, des hooks et des flux de travail standard. À utiliser pour commencer à travailler sur un nouveau dépôt, pour intégrer un projet existant ou pour s'assurer qu'un dépôt dispose d'une configuration CI/CD appropriée. Configure : les hooks git (lefthook), les modèles de messages de validation, les modèles PR et les actions GitHub pour lint/format/type-check/build. Invite l’utilisateur à confirmer avant les modifications. Déclencheurs : "bootstrap repo", "setup hooks", "configure CI", "rig", "standardize repo".
