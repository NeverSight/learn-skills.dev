Analiza los temas de tendencia de Weibo (微博) y genera ideas creativas de productos basadas en palabras clave de búsqueda populares. Obtiene datos de tendencias en tiempo real de la API de Weibo, investiga información general y conocimientos de los usuarios, evalúa el potencial de desarrollo de productos utilizando un sistema de puntuación de 100 puntos y crea un panel HTML interactivo. Úselo al analizar las tendencias de las redes sociales chinas, identificar oportunidades de productos a partir de temas de tendencia o generar conceptos de productos creativos impulsados ​​por el mercado.
