Web スクレイピングとデータ抽出にはスクレイピングを使用します。ターゲット サイトの特性に基づいて最適な Fetcher を自動的に選択し、Python スクリプトを生成して実行してタスクを完了します。次の場合に使用します。
(1) Web コンテンツまたはデータのスクレイピング/クロール (スクレイピング、クロール、ページの取得、データの抽出)
(2) Cloudflare/WAF などのアンチスクレイピング保護をバイパスする必要がある
(3) ログイン後の保護されたページのスクレイピング
(4) 既存のHTMLを解析して構造化データを抽出する
(5) ユーザーは URL を提供し、ページのコンテンツまたは特定の要素をリクエストします。
(6) 複数ページを一括収集する
