Utilizza lo scrapling per il web scraping e l'estrazione dei dati. Seleziona automaticamente il miglior Fetcher in base alle caratteristiche del sito di destinazione, genera ed esegue uno script Python per completare l'attività. Utilizzare quando:
(1) Scraping/scansione di pagine Web o dati (scrape, scansione, recupero di pagine, estrazione di dati)
(2) Bypassare le protezioni anti-bot come Cloudflare/WAF
(3) Eliminazione delle pagine protette dopo l'accesso
(4) Analisi dell'HTML esistente per estrarre dati strutturati
(5) L'utente fornisce un URL e richiede il contenuto della pagina o elementi specifici
(6) Raccolta batch su più pagine
