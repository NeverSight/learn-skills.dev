웹 스크래핑 및 데이터 추출에 스크래핑을 사용합니다. 대상 사이트의 특성에 따라 최적의 Fetcher를 자동으로 선택하고 Python 스크립트를 생성 및 실행하여 작업을 완료합니다. 다음과 같은 경우에 사용하세요:
(1) 웹 콘텐츠 또는 데이터 스크래핑/크롤링(스크래핑, 크롤링, 페이지 가져오기, 데이터 추출)
(2) Cloudflare/WAF와 같은 안티봇 보호 우회
(3) 로그인 후 보호된 페이지 긁기
(4) 기존 HTML을 구문 분석하여 구조화된 데이터 추출
(5) 사용자는 URL을 제공하고 페이지 콘텐츠 또는 특정 요소를 요청합니다.
(6) 여러 페이지를 일괄 수집
