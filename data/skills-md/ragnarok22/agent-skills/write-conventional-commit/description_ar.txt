قم بإنشاء رسائل التزام Git التي تتوافق مع الالتزامات التقليدية 1.0.0، بما في ذلك تنسيق النوع/النطاق/الوصف، والنص الاختياري، والتذييلات على شكل المقطع الدعائي، وإشارة تغيير التغيير الصريحة. يُستخدم عندما يطلب المستخدمون صياغة رسائل التزام، أو تنفيذ التغييرات الحالية، أو إعادة كتابة رسالة التزام بتنسيق تقليدي، أو فرض معايير الالتزام التقليدية في الريبو.
