قم بالرد على تعليقات مراجعة العلاقات العامة المضمنة بشأن طلبات سحب GitHub باستخدام واجهة برمجة تطبيقات GitHub. استخدم هذه المهارة عندما تحتاج إلى الرد على تعليقات المراجعة الفردية على العلاقات العامة، أو الاعتراف بالتعليقات، أو وضع علامة على التعليقات على أنها تم حلها عن طريق نشر ردود مباشرة على سلاسل التعليقات.
