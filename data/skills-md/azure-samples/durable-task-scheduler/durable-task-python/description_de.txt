Erstellen Sie dauerhafte, fehlertolerante Arbeitsabläufe in Python mithilfe des Durable Task SDK mit Azure Durable Task Scheduler. Verwenden Sie diese Option, wenn Sie Orchestrierungen, Aktivitäten, Entitäten erstellen oder Muster wie Funktionsverkettung, Fan-Out/Fan-In, menschliche Interaktion oder zustandsbehaftete Agenten implementieren. Gilt für jede Python-Anwendung, die eine dauerhafte Ausführung, Zustandspersistenz oder verteilte Transaktionen ohne Azure Functions-Abhängigkeit erfordert.
