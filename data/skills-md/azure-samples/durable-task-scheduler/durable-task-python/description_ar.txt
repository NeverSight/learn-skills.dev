أنشئ مسارات عمل متينة ومتسامحة مع الأخطاء في Python باستخدام أداة SDK للمهام المتينة مع برنامج جدولة المهام المتين من Azure. يُستخدم عند إنشاء تنسيقات، أو أنشطة، أو كيانات، أو تنفيذ أنماط مثل تسلسل الوظائف، أو التوزيع الموسع/المروحة الداخلية، أو التفاعل البشري، أو الوكلاء ذوي الحالة. ينطبق على أي تطبيق Python يتطلب تنفيذًا متينًا أو ثبات الحالة أو المعاملات الموزعة دون تبعية Azure Functions.
