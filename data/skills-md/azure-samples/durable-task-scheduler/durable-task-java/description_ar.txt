أنشئ مسارات عمل متينة ومتسامحة مع الأخطاء في Java باستخدام أداة SDK للمهام المتينة مع برنامج جدولة المهام المتين من Azure. يُستخدم عند إنشاء تنسيقات أو أنشطة أو تنفيذ أنماط مثل تسلسل الوظائف أو التوزيع الموسع/المروحة الداخلية أو التفاعل البشري أو المراقبة. ينطبق على أي تطبيق Java يتطلب التنفيذ الدائم أو استمرار الحالة أو المعاملات الموزعة دون تبعية Azure Functions.
