Erstellen Sie dauerhafte, fehlertolerante Arbeitsabläufe in Java mithilfe des Durable Task SDK mit Azure Durable Task Scheduler. Verwenden Sie diese Option, wenn Sie Orchestrierungen und Aktivitäten erstellen oder Muster wie Funktionsverkettung, Fan-Out/Fan-In, menschliche Interaktion oder Überwachung implementieren. Gilt für jede Java-Anwendung, die eine dauerhafte Ausführung, Zustandspersistenz oder verteilte Transaktionen ohne Abhängigkeit von Azure Functions erfordert.
