Cree flujos de trabajo duraderos y tolerantes a errores en Java mediante el SDK de Durable Task con Azure Durable Task Scheduler. Úselo al crear orquestaciones, actividades o implementar patrones como encadenamiento de funciones, distribución/entrada, interacción humana o monitoreo. Se aplica a cualquier aplicación Java que requiera ejecución duradera, persistencia de estado o transacciones distribuidas sin dependencia de Azure Functions.
