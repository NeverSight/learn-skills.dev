Crea flussi di lavoro durevoli e con tolleranza agli errori in Java usando Sustainable Task SDK con l'Utilità di pianificazione durevole di Azure. Da utilizzare durante la creazione di orchestrazioni, attività o l'implementazione di modelli come concatenamento di funzioni, fan-out/fan-in, interazione umana o monitoraggio. Si applica a qualsiasi applicazione Java che richiede esecuzione durevole, persistenza dello stato o transazioni distribuite senza dipendenza da Funzioni di Azure.
