Erstellen Sie dauerhafte, fehlertolerante Arbeitsabläufe mithilfe von Azure Durable Functions mit .NET-isoliertem Worker und dauerhaftem Taskplaner-Backend. Verwenden Sie diese Option, wenn Sie serverlose Orchestrierungen, Aktivitäten, Entitäten erstellen oder Muster wie Funktionsverkettung, Fan-Out/Fan-In, asynchrone HTTP-APIs, menschliche Interaktion, Überwachung oder zustandsbehaftete Aggregatoren implementieren. Gilt für Azure Functions-Apps, die dauerhafte Ausführung, Statuspersistenz oder verteilte Koordination mit integrierten HTTP-Verwaltungs-APIs und Azure-Integration erfordern.
