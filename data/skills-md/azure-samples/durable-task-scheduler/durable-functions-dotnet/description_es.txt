Cree flujos de trabajo duraderos y tolerantes a errores mediante Azure Durable Functions con el trabajador aislado de .NET y el backend del Programador de tareas duradero. Utilícelo al crear orquestaciones, actividades y entidades sin servidor o al implementar patrones como encadenamiento de funciones, distribución en abanico/entrada, API HTTP asíncronas, interacción humana, supervisión o agregadores con estado. Se aplica a aplicaciones de Azure Functions que requieren ejecución duradera, persistencia de estado o coordinación distribuida con API de administración HTTP integradas e integración de Azure.
