Crea flussi di lavoro durevoli e con tolleranza agli errori utilizzando Funzioni durevoli di Azure con il lavoratore isolato .NET e il backend dell'Utilità di pianificazione durevole. Da utilizzare durante la creazione di orchestrazioni, attività ed entità serverless o l'implementazione di modelli come concatenamento di funzioni, fan-out/fan-in, API HTTP asincrone, interazione umana, monitoraggio o aggregatori con stato. Si applica alle app Funzioni di Azure che richiedono esecuzione durevole, persistenza dello stato o coordinamento distribuito con API di gestione HTTP integrate e integrazione di Azure.
