أنشئ مسارات عمل متينة ومتسامحة مع الأخطاء باستخدام وظائف Azure المتينة مع عامل .NET المعزول والواجهة الخلفية لجدولة المهام المتينة. يُستخدم عند إنشاء تنسيقات أو أنشطة أو كيانات أو أنماط تنفيذ بدون خادم مثل تسلسل الوظائف، أو التوزيع الموسع/المروحة الداخلية، أو واجهات برمجة تطبيقات HTTP غير المتزامنة، أو التفاعل البشري، أو المراقبة، أو المجمعات ذات الحالة. ينطبق على تطبيقات Azure Functions التي تتطلب تنفيذًا متينًا أو استمرارية الحالة أو تنسيقًا موزعًا مع واجهات برمجة تطبيقات إدارة HTTP المضمنة وتكامل Azure.
