Synchronisez la configuration du serveur MCP entre VS Code (.vscode/mcp.json) et Copilot CLI (~/.copilot/mcp-config.json). À utiliser lors de la première configuration de Copilot CLI, lorsque des serveurs MCP sont ajoutés ou modifiés dans le référentiel, ou lorsqu'un utilisateur signale des outils MCP manquants dans CLI.
