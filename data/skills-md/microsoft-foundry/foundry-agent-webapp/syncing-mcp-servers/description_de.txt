Synchronisieren Sie die MCP-Serverkonfiguration zwischen VS Code (.vscode/mcp.json) und Copilot CLI (~/.copilot/mcp-config.json). Verwenden Sie diese Option, wenn Sie die Copilot-CLI zum ersten Mal einrichten, wenn MCP-Server im Repository hinzugefügt oder geändert werden oder wenn ein Benutzer fehlende MCP-Tools in der CLI meldet.
