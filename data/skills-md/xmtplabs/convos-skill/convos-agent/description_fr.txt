Cette compétence doit être utilisée lorsqu'un agent IA doit rejoindre, créer ou participer à des conversations de messagerie cryptées Convos.
Couvre le cycle de vie complet : installation de la CLI, participation ou création de conversations, exécution en mode agent avec streaming en temps réel, envoi de messages/réponses/réactions/pièces jointes, lecture de l'historique, gestion des profils et administration des groupes.
Comprend des principes comportementaux pour être un véritable membre d'un groupe, le protocole de service d'agent ndjson, un modèle de script de pont et une référence de commande CLI.
À utiliser lorsque : l'utilisateur dit "rejoindre cette conversation Convos", "créer un groupe Convos", "envoyer un message sur Convos", "configurer le mode agent", "rejoindre via un lien d'invitation" ou toute tâche impliquant la messagerie CLI ou XMTP des convos.
Fonctionne avec n'importe quel agent capable d'exécuter des commandes shell.
