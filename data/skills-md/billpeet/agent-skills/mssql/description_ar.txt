الاستعلام عن قواعد بيانات Microsoft SQL Server وإدارتها عبر mssql CLI. يُستخدم عند تشغيل استعلامات SQL أو فحص مخططات الجدول أو إدارة اتصالات الخادم. تشغيل عبارات مثل "استعلام SQL"، و"قاعدة البيانات"، و"مخطط الجدول"، و"تشغيل الاستعلام"، و"SQL Server"، و"mssql"، و"التحقق من قاعدة البيانات"، و"البحث في قاعدة البيانات".
