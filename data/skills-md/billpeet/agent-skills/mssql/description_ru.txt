Запрашивает базы данных Microsoft SQL Server и управляет ими через интерфейс командной строки mssql. Используйте при выполнении SQL-запросов, проверке схем таблиц или управлении соединениями с сервером. Запускается по таким фразам, как «SQL-запрос», «база данных», «схема таблицы», «выполнить запрос», «SQL Server», «mssql», «проверить базу данных», «поиск в базе данных».
