Interroge et gère les bases de données Microsoft SQL Server via la CLI mssql. À utiliser lors de l'exécution de requêtes SQL, de l'inspection des schémas de table ou de la gestion des connexions au serveur. Se déclenche sur des expressions telles que « requête SQL », « base de données », « schéma de table », « exécuter une requête », « SQL Server », « mssql », « vérifier la base de données », « rechercher dans la base de données ».
