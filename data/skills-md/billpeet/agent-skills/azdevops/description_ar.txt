يدير مشاريع Azure DevOps ومستودعات إعادة الشراء والفروع وطلبات السحب وخطوط الأنابيب وعناصر العمل عبر azdevops CLI. يُستخدم عند البحث عن عناصر العمل أو إنشائها أو تحديثها أو إدارة العلاقات العامة أو تشغيل خطوط الأنابيب أو إدراج مستودعات/فروع. مشغلات لعبارات مثل "Azure DevOps"، و"عنصر العمل"، و"طلب السحب"، و"خط الأنابيب"، و"إنشاء خطأ"، و"قائمة إعادة الشراء"، و"إنشاء المشغل".
