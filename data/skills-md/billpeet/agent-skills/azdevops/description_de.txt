Verwaltet Azure DevOps-Projekte, Repos, Branches, Pull Requests, Pipelines und Arbeitselemente über die azdevops-CLI. Wird zum Suchen, Erstellen und Aktualisieren von Arbeitselementen, zum Verwalten von PRs, zum Auslösen von Pipelines oder zum Auflisten von Repos/Branches verwendet. Löst Ausdrücke wie „Azure DevOps“, „Arbeitselement“, „Pull-Anforderung“, „Pipeline“, „Fehler erstellen“, „Repos auflisten“ und „Build auslösen“ aus.
