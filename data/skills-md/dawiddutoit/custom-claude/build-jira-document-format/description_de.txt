Erstellt komplexe ADF-Dokumente (Atlassian Document Format) mit erweiterten Mustern, Vorlagen und speziellen Buildern. Verwenden Sie es, wenn Sie anspruchsvolle Jira-Beschreibungen, Epics mit Struktur, Rich-Content-Dokumente erstellen oder Builder-Muster für Fluent-APIs implementieren. Auslöseschlüsselwörter: „ADF-Vorlage“, „Dokument-Builder“, „komplexe Beschreibung“, „fließende API“, „epische Struktur“, „formatiertes Problem“, „Rich Content“, „Builder-Muster“, „EpicBuilder“, „IssueBuilder“, „Dokumentvorlage“, „verschachtelter Inhalt“, „erweiterte Formatierung“. Funktioniert mit Python 3.10+, dem Modul jira_tool.formatter und der Jira REST API v3.
