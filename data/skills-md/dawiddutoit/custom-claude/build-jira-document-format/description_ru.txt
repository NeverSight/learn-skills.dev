Создает сложные документы ADF (Atlassian Document Format) с использованием расширенных шаблонов, шаблонов и специализированных конструкторов. Используйте при создании сложных описаний Jira, эпиков со структурой, документов с богатым содержимым или реализации шаблонов компоновщика для гибких API. Ключевые слова-триггеры: «шаблон ADF», «построитель документов», «сложное описание», «свободный API», «эпическая структура», «форматированная проблема», «богатый контент», «шаблон построителя», «EpicBuilder», «IssueBuilder», «шаблон документа», «вложенный контент», «расширенное форматирование». Работает с Python 3.10+, модулем jira_tool.formatter и Jira REST API v3.
