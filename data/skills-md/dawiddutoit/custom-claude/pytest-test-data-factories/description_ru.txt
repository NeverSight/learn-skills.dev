Шаблоны для создания тестовых данных многократного использования: фабричные функции для объектов предметной области, шаблоны компоновщика для сложных объектов, фабрики приспособлений, параметризованные тестовые данные, наследование тестовых данных и типобезопасные генераторы тестовых данных. Используйте при повторном создании тестовых данных, создании объектов с множеством дополнительных полей или создании реалистичных наборов тестовых данных.

Используйте, когда: пишете тесты, которым нужны объекты предметной области, создаете приспособления для нескольких тестов, генерируете тестовые данные со значениями по умолчанию, создаете сложные объекты с необязательными полями или параметризируете тесты с различными данными.
