Modelli per la creazione di dati di test riutilizzabili: funzioni factory per oggetti di dominio, modelli di builder per oggetti complessi, factory di dispositivi, dati di test parametrizzati, ereditarietà dei dati di test e generatori di dati di test indipendenti dai tipi. Da utilizzare quando si creano dati di test ripetutamente, si creano oggetti con molti campi opzionali o si generano set di dati di test realistici.

Da utilizzare quando: scrivere test che necessitano di oggetti di dominio, creare strutture per più test, generare dati di test con valori predefiniti, creare oggetti complessi con campi opzionali o parametrizzare test con vari dati.
