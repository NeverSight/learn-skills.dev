Crea repositorio siguiendo Arquitectura Limpia con Protocolo en capa de dominio e Implementación
en la capa de infraestructura. Úselo al agregar una nueva capa de acceso a datos, crear interacción con la base de datos,
implementar persistencia o necesita almacenar/recuperar modelos de dominio. Hace cumplir el protocolo/patrón ABC
con ServiceResult, ciclo de vida de ManagedResource y separación de capas adecuada. Activadores en "crear
repositorio para X", "implementar acceso a datos para Y", "agregar capa de persistencia" o "almacenar/recuperar
modelo de dominio".
