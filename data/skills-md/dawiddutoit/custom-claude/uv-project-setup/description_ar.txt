إعداد pyproject.toml وإدارة مجموعات التبعية وتثبيت إصدارات Python.
يُستخدم عند بدء مشاريع جديدة، أو تكوين بيئات التطوير، أو توحيد المعايير
هيكل المشروع مع الأشعة فوق البنفسجية. يغطي "uv init"، و"uv add"، و"uv python pin"، والأحرف الأولية
سقالات المشروع مع تنظيم التبعية المناسب.
