Инициализировать и настраивать новые проекты Python с помощью uv, включая создание проектов,
настройка pyproject.toml, управление группами зависимостей и закрепление версий Python.
Используйте при запуске новых проектов, настройке сред разработки или стандартизации.
структура проекта с ув. Охватывает `uv init`, `uv add`, `uv python pin` и начальный
каркас проекта с правильной организацией зависимостей.
