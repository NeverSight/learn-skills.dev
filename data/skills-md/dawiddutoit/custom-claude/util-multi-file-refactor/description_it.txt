Ottimizza i flussi di lavoro di refactoring multi-file applicando un utilizzo MultiEdit efficiente in termini di token (risparmio del 30-50%), coordinando le modifiche tra file, convalidando importazioni/riferimenti ed eseguendo controlli di qualità. Da utilizzare durante il refactoring di pattern nella codebase, la ridenominazione di funzioni/classi/variabili, l'estrazione di codice comune, lo spostamento di funzionalità o l'aggiornamento di pattern in più file. Impedisce la modifica sequenziale degli anti-pattern.
