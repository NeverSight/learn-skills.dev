Verwenden Sie diese Option, wenn Sie Muster in der gesamten Codebasis umgestalten, Funktionen/Klassen/Variablen umbenennen, allgemeinen Code extrahieren, Funktionen verschieben oder Muster in mehreren Dateien aktualisieren.
