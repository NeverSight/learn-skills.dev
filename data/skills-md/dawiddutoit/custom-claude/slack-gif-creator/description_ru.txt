Slack», «создать анимированный смайлик», «создать GIF-файл для Slack» или «анимировать это для Slack». Обеспечивает
Утилиты GIFBuilder, управление кадрами, уменьшение цветовой палитры и инструменты проверки, обеспечивающие
GIF-файлы соответствуют требованиям Slack. Работает с Python Pillow (PIL) для генерации кадров и цвета.
