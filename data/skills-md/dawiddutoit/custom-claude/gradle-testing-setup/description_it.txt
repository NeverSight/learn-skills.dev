Configura test completi in Gradle inclusi JUnit 5, TestContainers,
separazione dei test (unità vs integrazione) e copertura del codice con JaCoCo.
Utilizzare quando viene richiesto di "configurare JUnit 5", "configurare TestContainers", "test di integrazione separati",
o "aggiungi copertura codice".
Funziona con build.gradle.kts, set di sorgenti di test e configurazioni CI/CD.
