قم بإجراء مراجعة ذاتية منهجية لتغييرات التعليمات البرمجية قبل الالتزامات باستخدام قائمة التحقق المنظمة. التحقق من صحة حدود البنية وجودة الكود وتغطية الاختبار والوثائق والأنماط المضادة الخاصة بالمشروع. استخدمه قبل الالتزام أو إنشاء العلاقات العامة أو عندما يقول المستخدم "مراجعة تغييراتي" أو "المراجعة الذاتية" أو "التحقق من الكود الخاص بي". يتكيف مع مشاريع Python وJavaScript وTypeScript وGo وRust.
