Настраивает Gradle с проектами Spring Boot, включая настройку плагинов, создание загрузочного JAR,
многоуровневые JAR-файлы для оптимизации Docker и многомодульные конфигурации Spring Boot.
Используйте, когда вас просят «настроить Spring Boot с Gradle», «создать исполняемые файлы JAR»,
«настроить уровни Docker» или «настроить микросервисы Spring Boot».
