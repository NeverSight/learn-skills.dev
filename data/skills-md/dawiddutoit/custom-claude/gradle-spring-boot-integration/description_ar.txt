يقوم بتكوين Gradle باستخدام مشاريع Spring Boot بما في ذلك إعداد المكونات الإضافية، وإنشاء JAR قابل للتمهيد،
ملفات JAR ذات طبقات لتحسين Docker وتكوينات Spring Boot متعددة الوحدات.
يُستخدم عندما يُطلب منك "إعداد Spring Boot باستخدام Gradle"، و"إنشاء ملفات JAR قابلة للتنفيذ"،
"تكوين طبقات Docker" أو "إعداد خدمات Spring Boot الصغيرة".
