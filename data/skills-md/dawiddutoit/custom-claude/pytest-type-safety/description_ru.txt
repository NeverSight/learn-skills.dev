Обеспечьте безопасность типов во всех тестах: аннотации типов для тестовых функций и приспособлений, макеты шаблонов автоспецификации, конфигурация mypy для тестов, типобезопасные утверждения, общая обработка типов в макетах. Включает типизацию на основе протокола и шаблоны приведения для сложных сценариев.

Используйте, когда: пишете типобезопасные тесты, настраиваете mypy для тестовых файлов, используете протоколы для макетов, обеспечиваете проверку типов в CI/CD или отлаживаете проблемы тестов, связанные с типами.
