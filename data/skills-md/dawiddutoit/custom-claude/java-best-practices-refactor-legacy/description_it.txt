Effettua il refactoring del codice Java legacy secondo modelli e best practice moderni. Da utilizzare durante la modernizzazione del vecchio codice Java, la conversione alle funzionalità Java 8+, il refactoring di applicazioni legacy, l'applicazione di modelli di progettazione, il miglioramento della gestione degli errori, l'estrazione di metodi/classi, la conversione in flussi/opzionali/record o la migrazione da vecchie versioni Java. Funziona con codice precedente a Java 8, Java procedurale, framework legacy e modelli obsoleti.
