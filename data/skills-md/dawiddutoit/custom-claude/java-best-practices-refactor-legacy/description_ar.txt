يعيد بناء كود Java القديم ليتوافق مع الأنماط الحديثة وأفضل الممارسات. يُستخدم عند تحديث كود Java القديم، أو التحويل إلى ميزات Java 8+، أو إعادة بناء التطبيقات القديمة، أو تطبيق أنماط التصميم، أو تحسين معالجة الأخطاء، أو استخراج الأساليب/الفئات، أو التحويل إلى التدفقات/اختياري/السجلات، أو الترحيل من إصدارات Java القديمة. يعمل مع تعليمات برمجية ما قبل Java 8، وJava الإجرائية، والأطر القديمة، والأنماط القديمة.
