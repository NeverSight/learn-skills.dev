Impedisci a OutOfMemoryError di tag di parametri illimitati implementando i limiti di cardinalità.
Da utilizzare quando si aggiungono tag alle metriche Micrometer, si normalizza URI o ID in categorie delimitate,
monitorare l'esplosione delle metriche o evitare campi ad alta cardinalità come ID utente e ID richiesta.
Fondamentale per i microservizi di produzione in GKE con backend di monitoraggio sensibili ai costi.
