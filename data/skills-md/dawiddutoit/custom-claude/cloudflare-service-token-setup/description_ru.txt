Создает и управляет токенами службы Cloudflare Access для автоматизированной инфраструктуры.
проверка и нечеловеческий доступ. Используйте при настройке автоматизации, скрипты проверки,
системы мониторинга или необходимо протестировать сервисы без Google OAuth. Запускается при «создании
токен службы", "доступ к автоматизации настройки", "проверка без OAuth", "автоматический мониторинг",
или «служебный токен для тестирования». Работает с аутентификацией Cloudflare Access Service, учетными данными .env.
хранилище и скрипт cf-service-token.sh для тестирования и управления.
