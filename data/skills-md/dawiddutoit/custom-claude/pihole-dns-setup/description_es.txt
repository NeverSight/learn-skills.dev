Configura los registros DNS locales de Pi-hole para habilitar la resolución de red local de *.temet.ai
dominios. Úselo cuando necesite configurar DNS, agregar entradas DNS locales, configurar DNS Pi-hole,
solucionar problemas de resolución de DNS o hacer que los servicios se resuelvan localmente. Se activa en "configurar DNS",
"configurar DNS Pi-hole", "agregar registro DNS", "DNS no se resuelve", "configuración de DNS local" o
"¿Por qué no puedo acceder a [servicio].temet.ai localmente?". Funciona con el contenedor Pi-hole, domains.toml,
docker-compose.yml y administrar-dominios.sh.
