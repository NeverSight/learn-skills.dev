يقوم بتكوين سجلات DNS المحلية Pi-hole لتمكين تحليل الشبكة المحلية لـ *.temet.ai
المجالات. استخدمه عندما تحتاج إلى إعداد DNS، وإضافة إدخالات DNS المحلية، وتكوين Pi-hole DNS،
استكشاف أخطاء تحليل DNS وإصلاحها، أو جعل الخدمات تحل محليًا. المشغلات على "إعداد DNS"،
"تكوين Pi-hole DNS"، أو "إضافة سجل DNS"، أو "لا يتم حل DNS"، أو "إعداد DNS المحلي"، أو
"لماذا لا يمكنني الوصول إلى [الخدمة].temet.ai محليًا". يعمل مع حاوية Pi-hole، domains.toml،
docker-compose.yml وmanage-domains.sh.
