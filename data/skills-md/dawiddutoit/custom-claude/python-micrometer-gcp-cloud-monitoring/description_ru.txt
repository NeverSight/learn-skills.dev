Экспортирует метрики Micrometer в GCP Cloud Monitoring (Stackdriver) для развертываний GKE. Используйте при настройке экспорта метрик в средах Kubernetes, настройке идентификации рабочей нагрузки, управлении префиксами метрик и метками ресурсов или при сопоставлении метрик со службами GCP. Критически важно для микросервисов на базе GKE с централизованным наблюдением в Google Cloud.
