Cree administradores de contexto asíncronos para la gestión de recursos utilizando el decorador @asynccontextmanager.
Úselo al administrar recursos asíncronos, sesiones de bases de datos, identificadores de archivos, conexiones o cualquier
recurso que requiere limpieza automática. Funciona con patrones asíncronos/en espera de Python. Análisis
Archivos .py con declaraciones async with, sugerencias de tipo AsyncIterator y patrones try/finally.
