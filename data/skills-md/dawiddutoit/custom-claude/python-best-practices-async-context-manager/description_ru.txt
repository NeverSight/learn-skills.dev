Создайте асинхронные контекстные менеджеры для управления ресурсами с помощью декоратора @asynccontextmanager.
Используйте при управлении асинхронными ресурсами, сеансами базы данных, дескрипторами файлов, соединениями и другими
ресурс, требующий автоматической очистки. Работает с шаблонами async/await Python. Анализы
Файлы .py с операторами async with, подсказками типа AsyncIterator и шаблонами try/finally.
