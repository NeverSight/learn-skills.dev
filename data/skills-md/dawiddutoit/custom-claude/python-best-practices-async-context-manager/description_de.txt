Erstellen Sie asynchrone Kontextmanager für die Ressourcenverwaltung mit dem @asynccontextmanager-Dekorator.
Wird zum Verwalten von asynchronen Ressourcen, Datenbanksitzungen, Dateihandles, Verbindungen usw. verwendet
Ressource, die eine automatische Bereinigung erfordert. Funktioniert mit Python-Async/Await-Mustern. Analysen
.py-Dateien mit async with-Anweisungen, AsyncIterator-Typhinweisen und try/finally-Mustern.
