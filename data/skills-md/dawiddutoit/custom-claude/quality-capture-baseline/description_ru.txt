Собирает базовые показатели качества (тесты, покрытие, ошибки типов, анализ, мертвый код) путем запуска
вентили качества и сохранение результатов в памяти для обнаружения регрессии. Используйте при запуске функции,
перед рефакторингом или после серьезных изменений для установления базовой линии. Триггеры по «захвату базовой линии»,
«установить базовый уровень» или ПРОАКТИВНО в начале любой функции/работы по рефакторингу. Работает с pytest
выходные данные, ошибки пирайта, предупреждения, результаты поиска и сервер MCP в памяти для хранения базовых данных.
