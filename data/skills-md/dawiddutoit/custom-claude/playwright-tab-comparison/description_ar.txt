افتح علامات تبويب متعددة في المتصفح وقارن المحتوى أو السلوك أو المظهر المرئي عبر الصفحات. يُستخدم عند مقارنة التدريج مقابل الإنتاج، أو متغيرات صفحة اختبار A/B، أو اختبار الاتساق عبر المستعرضات، أو التحقق من تكافؤ المحتوى، أو تحليل الاختلافات بين عناوين URL المتعددة. تتضمن عبارات التشغيل "مقارنة هذه الصفحات"، أو "فتح علامات تبويب متعددة ومقارنتها"، أو "اختبار الاتساق عبر"، أو "اختبار A/B"، أو "التدريج مقابل الإنتاج"، أو "مقارنة الإصدارات". يعمل مع أدوات أتمتة متصفح Playwright MCP لسير العمل متعدد علامات التبويب.
