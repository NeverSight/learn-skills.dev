Предоставляет методы взаимодействия с Home Assistant REST API для обнаружения объектов, состояния
извлечение и вызовы служб, включая аутентификацию, шаблоны сущностей, вызовы служб,
и государственные запросы. Используйте при работе с интеграциями Home Assistant, управлении устройствами,
запрос данных датчиков или автоматизация операций Home Assistant.
