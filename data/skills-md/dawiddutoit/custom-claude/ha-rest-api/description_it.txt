Fornisce metodi per interagire con l'API REST di Home Assistant per il rilevamento di entità, stato
recupero e chiamate di servizio tra cui autenticazione, modelli di entità, chiamate di servizio,
e domande di stato. Da utilizzare quando si lavora con le integrazioni di Home Assistant, si controllano i dispositivi,
interrogare i dati dei sensori o automatizzare le operazioni di Home Assistant.
