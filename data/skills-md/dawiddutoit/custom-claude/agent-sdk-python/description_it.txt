Implementa agenti AI pronti per la produzione utilizzando l'SDK Python ufficiale per flussi di lavoro autonomi.
Da utilizzare durante l'implementazione di flussi di lavoro dell'agente, integrazione SDK, query one-shot, interattive
conversazioni, strumenti MCP personalizzati, controllo dei permessi, caricamento del contesto CLAUDE.md, errore
gestione, hook o delega dell'agente. Funziona con file .py e architetture di agenti.
Termini di attivazione: "Agent SDK", "query()", "ClaudeSDKClient", "@tool decorator",
"setting_sources", "autorizzazioni agente", "hook SDK".
