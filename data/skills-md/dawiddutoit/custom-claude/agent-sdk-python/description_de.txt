Implementiert produktionsbereite KI-Agenten mithilfe des offiziellen Python SDK für autonome Arbeitsabläufe.
Verwendung bei der Implementierung von Agent-Workflows, SDK-Integration, einmaligen Abfragen und interaktiv
Konversationen, benutzerdefinierte MCP-Tools, Berechtigungskontrolle, Laden des CLAUDE.md-Kontexts, Fehler
Handling, Hooks oder Agentendelegation. Funktioniert mit .py-Dateien und Agent-Architekturen.
Auslöserbegriffe: „Agent SDK“, „query()“, „ClaudeSDKClient“, „@tool decorator“,
„setting_sources“, „Agent-Berechtigungen“, „SDK-Hooks“.
