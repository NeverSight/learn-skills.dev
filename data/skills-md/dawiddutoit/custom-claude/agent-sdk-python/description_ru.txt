Реализует готовые к использованию агенты искусственного интеллекта с использованием официального Python SDK для автономных рабочих процессов.
Используйте при реализации рабочих процессов агентов, интеграции SDK, одноразовых запросов, интерактивных
разговоры, пользовательские инструменты MCP, контроль разрешений, загрузка контекста CLAUDE.md, ошибка
обработка, перехваты или делегирование агента. Работает с файлами .py и архитектурами агентов.
Триггерные термины: «Агент SDK», «query()», «ClaudeSDKClient», «декоратор @tool»,
«setting_sources», «разрешения агента», «перехватчики SDK».
