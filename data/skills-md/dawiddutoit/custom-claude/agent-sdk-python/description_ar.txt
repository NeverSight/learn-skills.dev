تنفيذ وكلاء الذكاء الاصطناعي الجاهزين للإنتاج باستخدام Python SDK الرسمي لسير العمل المستقل.
يُستخدم عند تنفيذ سير عمل الوكيل، وتكامل SDK، والاستعلامات الفردية، والتفاعلية
المحادثات، وأدوات MCP المخصصة، والتحكم في الأذونات، وتحميل سياق CLAUDE.md، والخطأ
المناولة أو الخطافات أو تفويض الوكيل. يعمل مع ملفات .py وبنيات الوكيل.
مصطلحات التشغيل: "Agent SDK"، "query()"، "ClaudeSDKClient"، "@tool Decorator"،
"setting_sources"، "أذونات الوكيل"، "خطافات SDK".
