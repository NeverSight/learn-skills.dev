Создавайте заводские шаблоны креплений для настраиваемых тестовых настроек с различными вариациями.
Используйте при создании многоразовых тестовых приспособлений с несколькими конфигурациями, создавая
параметризуемые макеты или реализация построителей тестовых данных. Работает с pytest
приспособления, макеты объектов и тестовые утилиты. Включает настройку СУХОГО теста во время
сохранение гибкости для крайних случаев.
