Cree patrones de accesorios de fábrica para una configuración de prueba personalizable con variaciones.
Úselo al construir dispositivos de prueba reutilizables con múltiples configuraciones, creando
simulacros parametrizables o implementar creadores de datos de prueba. Funciona con pytest
accesorios, objetos simulados y utilidades de prueba. Habilita la configuración de prueba DRY mientras
manteniendo la flexibilidad para los casos extremos.
