يقوم بإعداد اختبارات غير متزامنة باستخدام تركيبات ونماذج مناسبة باستخدام أنماط pytest-asyncio. يُستخدم عند اختبار وظائف غير متزامنة، أو إنشاء تركيبات غير متزامنة، أو الاستهزاء بالخدمات غير المتزامنة، أو التعامل مع مديري السياق غير المتزامن. يغطي @pytest_asyncio.fixture وAsyncMock مع التأثير الجانبي وتركيبات المولد غير المتزامن (العائد) واختبار مديري السياق غير المتزامن. يعمل مع أنماط Python غير المتزامنة/الانتظار، وpytest-asyncio، وunitest.mock.AsyncMock.
