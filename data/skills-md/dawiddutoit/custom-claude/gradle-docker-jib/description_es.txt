Configura y crea imágenes Docker optimizadas utilizando el complemento Jib de Google sin Docker Daemon o Dockerfile.
Úselo al contener microservicios Spring Boot, crear imágenes de múltiples arquitecturas,
o empujando a los registros. Activadores en "configurar Jib", "construir imagen de Docker", "contenerizar"
con Jib", o "imagen de múltiples arcos". Funciona con build.gradle.kts e incluye el complemento Jib
instalación, configuración de múltiples módulos e integración CI/CD.
