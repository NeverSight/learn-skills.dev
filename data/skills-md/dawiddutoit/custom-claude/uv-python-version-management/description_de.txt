Verwaltet Python-Versionen mit UV. Erfahren Sie, wie Sie verfügbare Python-Versionen ermitteln, bestimmte Versionen installieren, projektspezifische Versionen mithilfe von .python-version-Dateien anheften und Tools mit verschiedenen Python-Versionen ausführen. Verwenden Sie es, wenn Sie neue Projekte einrichten, sicherstellen, dass das Team dieselbe Python-Version verwendet, mehrere Versionen testen oder ältere Projekte mit älteren Python-Anforderungen verwalten.
