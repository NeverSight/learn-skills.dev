Gestisce le versioni Python con uv. Scopri come scoprire le versioni Python disponibili, installare versioni specifiche, aggiungere versioni specifiche del progetto utilizzando file .python-version ed eseguire strumenti con diverse versioni Python. Da utilizzare durante l'impostazione di nuovi progetti, per garantire che il team utilizzi la stessa versione di Python, per i test su più versioni o per la gestione di progetti legacy con requisiti Python precedenti.
