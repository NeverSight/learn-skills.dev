Gestiona versiones de Python con uv. Aprenda a descubrir versiones de Python disponibles, instalar versiones específicas, anclar versiones específicas del proyecto usando archivos .python-version y ejecutar herramientas con diferentes versiones de Python. Úselo al configurar nuevos proyectos, garantizar que el equipo utilice la misma versión de Python, realizar pruebas en varias versiones o administrar proyectos heredados con requisitos de Python más antiguos.
