Управляет версиями Python с помощью uv. Узнайте, как находить доступные версии Python, устанавливать определенные версии, закреплять версии для конкретного проекта с помощью файлов версии .python и запускать инструменты с различными версиями Python. Используйте при настройке новых проектов, обеспечении использования командой одной и той же версии Python, тестировании нескольких версий или управлении устаревшими проектами с более старыми требованиями Python.
