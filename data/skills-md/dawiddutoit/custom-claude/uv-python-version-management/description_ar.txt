يدير إصدارات بايثون باستخدام الأشعة فوق البنفسجية. تعرف على كيفية اكتشاف إصدارات Python المتاحة، وتثبيت إصدارات معينة، وتثبيت الإصدارات الخاصة بالمشروع باستخدام ملفات إصدار .python، وتشغيل الأدوات باستخدام إصدارات Python المختلفة. يُستخدم عند إعداد مشاريع جديدة، أو التأكد من أن الفريق يستخدم نفس إصدار Python، أو الاختبار عبر إصدارات متعددة، أو إدارة المشاريع القديمة باستخدام متطلبات Python الأقدم.
