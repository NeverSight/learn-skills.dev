يقوم بإعداد وتكوين مجموعات Google Kubernetes Engine (GKE) لاستخدامها في الإنتاج. يُستخدم عند إنشاء مجموعات GKE جديدة، أو الاختيار بين أوضاع Autopilot مقابل Standard، أو تكوين الشبكات (VPC الأصلية، أو المجموعات الخاصة)، أو إعداد مجموعات العقد، أو تخطيط بنية المجموعة لخدمات Spring Boot الصغيرة. يتضمن القرارات الإقليمية مقابل قرارات المناطق، وتشديد الأمن، وتوجيهات توفير الموارد.
