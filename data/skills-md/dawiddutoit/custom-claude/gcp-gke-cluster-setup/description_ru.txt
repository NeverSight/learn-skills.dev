Устанавливает и настраивает кластеры Google Kubernetes Engine (GKE) для производственного использования. Используйте при создании новых кластеров GKE, выборе между режимами «Автопилот» и «Стандарт», настройке сети (собственный VPC, частные кластеры), настройке пулов узлов или планировании архитектуры кластера для микросервисов Spring Boot. Включает региональные и зональные решения, усиление безопасности и рекомендации по предоставлению ресурсов.
