Fornisce una guida all'implementazione dettagliata per la creazione di comandi CQRS o gestori di query
seguendo i modelli di progetto con ServiceResult, inserimento delle dipendenze e gestore
registrazione. Da utilizzare durante l'implementazione di nuovi casi d'uso, l'aggiunta di funzionalità, la creazione di API
endpoint o creare la logica del livello di applicazione.
