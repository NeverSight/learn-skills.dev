Создает браузерные редакторы 2D-макетов с бэкэндом FastAPI и ванильным интерфейсом JS + SVG. Используйте при создании: (1) Визуальных редакторов для 2D-аранжировки (списки вырезов, планы этажей, диаграммы), (2) Интерфейсы перетаскивания с несколькими контейнерами/листами, (3) Однофайловые пользовательские интерфейсы браузера, обслуживаемые из Python, (4) Редактирование положения в реальном времени с проверкой. Триггеры: «редактор макета», «перетаскивание между», «визуальный редактор», «пользовательский интерфейс браузера для редактирования», «редактор SVG».
