Gestisce le dipendenze del progetto Python con uv. Scopri come aggiungere, rimuovere e aggiornare le dipendenze, organizzarle in gruppi (sviluppo, test, lint, documenti), aggiungere versioni, gestire i conflitti e gestire i file di blocco per installazioni riproducibili in tutti gli ambienti. Da utilizzare quando si aggiungono o si aggiornano pacchetti, si organizzano dipendenze di sviluppo, si risolvono conflitti di versione o si gestiscono file di blocco nelle pipeline CI/CD.
