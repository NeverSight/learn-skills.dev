Extraiga datos estructurados de múltiples páginas web usando Playwright con ética incorporada
prácticas de rastreo que incluyen limitación de velocidad, cumplimiento de robots.txt y monitoreo de errores.
Úselo cuando se le solicite "extraer datos de", "extraer información de páginas", "recopilar datos
desde el sitio", "rastrear varias páginas" o al recopilar datos estructurados de sitios web.
Admite paginación, extracción de varias páginas, agregación de datos y exportación a CSV/JSON/Markdown.
Funciona con las herramientas browser_navigate, browser_evaluate, browser_wait_for y browser_snapshot.
