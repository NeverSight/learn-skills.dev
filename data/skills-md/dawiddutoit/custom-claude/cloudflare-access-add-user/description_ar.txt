يضيف مستخدمين جدد إلى مصادقة Cloudflare Access عن طريق تحديث ACCESS_ALLOWED_EMAIL بتنسيق .env
ومزامنة السياسات مع جميع الخدمات المحمية. استخدمه عندما تحتاج إلى منح حق الوصول إلى ملف جديد
المستخدم، أو إضافة شخص ما إلى الشبكة، أو مشاركة الوصول إلى الخدمة، أو تحديث رسائل البريد الإلكتروني المسموح بها. المشغلات
في "إضافة مستخدم للوصول"، "منح الوصول إلى [البريد الإلكتروني]"، "إضافة [بريد إلكتروني] إلى cloudflare"، "مشاركة"
الوصول باستخدام"، أو "السماح لـ [البريد الإلكتروني] بالمصادقة"، أو "تحديث المستخدمين الذين يمكنهم الوصول". يعمل مع .env،
تحديث-الوصول-emails.sh، وسياسات الوصول إلى Cloudflare لـ pihole، وjaeger، وlangfuse،
خدمات الرش، هكتار، وtemet.ai.
