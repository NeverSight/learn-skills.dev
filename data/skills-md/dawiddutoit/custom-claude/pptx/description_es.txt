Creación, edición y análisis completos de presentaciones de PowerPoint mediante manipulación OOXML
incluyendo diapositivas, diseños, notas del orador, comentarios y formato. Úselo cuando se le solicite "crear
una presentación", "editar este PowerPoint", "agregar diapositivas a .pptx", "extraer texto de presentación",
o "analizar la estructura de la diapositiva". Proporciona acceso OOXML sin formato para formato avanzado, python-pptx
para la generación programática de diapositivas y Markitdown para la extracción de texto. Funciona con archivos .pptx
a través de extracción de archivos ZIP y manipulación XML para flujos de trabajo de presentación profesionales.
