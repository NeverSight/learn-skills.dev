Applicazioni Java/Spring Boot strumentali con metriche di osservabilità. Da utilizzare quando si aggiungono parametri ai microservizi,
integrazione con sistemi di monitoraggio (Prometheus, Cloud Monitoring), gestione della cardinalità metrica o implementazione
Monitoraggio SLI/SLO. Funziona con Spring Boot Actuator, GCP Cloud Monitoring e metriche dimensionali.
