Инструментируйте приложения Java/Spring Boot с помощью показателей наблюдаемости. Используйте при добавлении метрик в микросервисы,
интеграция с системами мониторинга (Prometheus, Cloud Monitoring), управление кардинальностью метрик или внедрение
SLI/SLO мониторинг. Работает с Spring Boot Actuator, облачным мониторингом GCP и размерными метриками.
