يوفر ترحيلًا منظمًا لتنسيق البيانات وإصداره لتغييرات المخطط المتوافقة مع الإصدارات السابقة.
يستخدم عند "إضافة إصدار بيانات"، "ترحيل تنسيق البيانات"، "تنسيق ملف متوافق مع الإصدارات السابقة"،
"ترقية الإصدار 1 إلى الإصدار 2"، أو "ترحيل المخطط"، أو "دعم تنسيق الملف القديم"، أو "ترحيل بنية JSON".
يغطي استراتيجيات ترقيم الإصدارات (الدلالية، المنقط، عدد صحيح، على أساس التاريخ)، والتوافق مع الإصدارات السابقة
الأنماط (التحويل التلقائي، الترحيل الصريح، التنسيق المزدوج)، توقيت الترقية التلقائية، إهمال الحقل،
قوائم مراجعة الاختبار، والمزالق الشائعة مثل الاحتفاظ الجزئي بالبيانات. يعمل مع JSON، YAML،
مخططات قاعدة البيانات وملفات التكوين في Python وJavaScript وGo وRust.
