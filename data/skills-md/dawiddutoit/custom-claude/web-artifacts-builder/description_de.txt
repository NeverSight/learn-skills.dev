Erstellt aufwändige mehrkomponentige claude.ai-HTML-Artefakte mit React 18, TypeScript, Vite,
Tailwind CSS und shadcn/ui gebündelt in Einzeldateien. Verwenden Sie diese Option, wenn Sie zum „Erstellen“ aufgefordert werden
komplexes Artefakt“, „React-Artefakt mit Routing erstellen“, „shadcn/ui-Komponenten verwenden“, „erstellen
Stateful-Web-App-Artefakt“ oder „Mehrkomponenten-Frontend erstellen“. Bietet Initialisierung
Skripte, Entwicklungsworkflow und Paketbündelung, um anspruchsvolle Einzel-HTML-Artefakte zu erstellen
mit staatlicher Verwaltung und modernen Komponenten. Funktioniert mit dem React/TypeScript-Stack für claude.ai
Artefakte, die eine Komplexität erfordern, die über einfaches HTML/JSX hinausgeht.
