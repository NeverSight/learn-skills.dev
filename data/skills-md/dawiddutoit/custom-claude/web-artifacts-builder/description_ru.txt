Создает сложные многокомпонентные HTML-артефакты claude.ai с использованием React 18, TypeScript, Vite,
Tailwind CSS и shadcn/ui объединены в отдельные файлы. Используйте, когда вас попросят «создать
сложный артефакт", "создать артефакт React с маршрутизацией", "использовать компоненты shadcn/ui", "создать
артефакт веб-приложения с сохранением состояния» или «создать многокомпонентный интерфейс». Обеспечивает инициализацию
сценарии, рабочий процесс разработки и объединение пакетов для создания сложных HTML-артефактов.
с госуправлением и современными компонентами. Работает со стеком React/TypeScript для claude.ai.
артефакты, требующие сложности, выходящей за рамки простого HTML/JSX.
