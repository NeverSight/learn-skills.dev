Tailwind CSS y shadcn/ui agrupados en entregables de un solo archivo. Úselo cuando se le solicite "crear
artefacto complejo", "construir artefacto React con enrutamiento", "usar componentes shadcn/ui", "crear
artefacto de aplicación web con estado" o "construir una interfaz de múltiples componentes". Proporciona inicialización
secuencias de comandos, flujo de trabajo de desarrollo y agrupación de paquetes para crear sofisticados artefactos HTML únicos
con gestión estatal y componentes modernos. Funciona con la pila React/TypeScript para claude.ai
artefactos que requieren complejidad más allá del simple HTML/JSX.
