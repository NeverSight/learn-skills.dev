Configura un nuevo túnel Cloudflare desde cero para un acceso remoto seguro a la red doméstica
servicios. Úselo al crear un nuevo túnel, reemplazar un token de túnel vencido o configurar
Active el acceso remoto para una nueva instalación. Activadores en "configurar túnel cloudflare", "crear
túnel", "nuevo token de túnel", "configurar acceso remoto", "túnel de cloudflare desde cero",
o "túnel no configurado". Funciona con .env, docker-compose.yml y Cloudflare Zero Trust.
