Richtet von Grund auf einen neuen Cloudflare-Tunnel für den sicheren Fernzugriff auf das Heimnetzwerk ein
Dienstleistungen. Verwenden Sie diese Option, wenn Sie einen neuen Tunnel erstellen, ein abgelaufenes Tunnel-Token ersetzen oder eine Einstellung vornehmen
Richten Sie den Fernzugriff für eine Neuinstallation ein. Löst auf „Cloudflare-Tunnel einrichten“, „erstellen“ aus
Tunnel“, „Neues Tunnel-Token“, „Remotezugriff konfigurieren“, „Cloudflare-Tunnel von Grund auf“,
oder „Tunnel nicht konfiguriert“. Funktioniert mit .env, docker-compose.yml und Cloudflare Zero Trust.
