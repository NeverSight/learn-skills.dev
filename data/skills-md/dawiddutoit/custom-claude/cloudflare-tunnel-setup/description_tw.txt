Sets up a new Cloudflare Tunnel from scratch for secure remote access to home network
服務。 Use when creating a new tunnel, replacing an expired tunnel token, or setting
全新安裝的遠端存取。觸發「設定 cloudflare 隧道」、「創建
tunnel", "new tunnel token", "configure remote access", "cloudflare tunnel from scratch",
或“隧道未配置”。 Works with .env, docker-compose.yml, and Cloudflare Zero Trust.
