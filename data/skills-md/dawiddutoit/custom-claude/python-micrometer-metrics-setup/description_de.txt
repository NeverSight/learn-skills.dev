Richtet Mikrometermetriken in einem Spring Boot-Dienst von Grund auf ein, einschließlich Abhängigkeiten, Aktuatorkonfiguration und Registrierungsanpassung. Verwenden Sie diese Option, wenn Sie einen neuen Microservice starten, Metrik-Export-Backends konfigurieren, Prometheus-Scraping aktivieren oder automatisch konfigurierte JVM-Metriken verwenden. Grundlegende Kenntnisse zur Metrikinstrumentierung in Java/Spring Boot-Anwendungen.
