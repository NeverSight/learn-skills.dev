Configura uv (gestore di pacchetti Python basato su Rust) nelle pipeline CI/CD.
Da utilizzare durante la configurazione di flussi di lavoro GitHub Actions, CI/CD GitLab, build Docker o test di matrici tra versioni Python.
Include modelli per l'ottimizzazione della cache, file di lock congelati, build in più fasi e pubblicazione PyPI con pubblicazione affidabile.
Copre l'azione setup-uv delle azioni GitHub, le build di produzione/sviluppo multifase di Docker e i modelli di distribuzione.
