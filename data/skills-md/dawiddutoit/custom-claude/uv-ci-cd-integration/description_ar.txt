قم بإعداد uv (مدير حزم Python المستند إلى Rust) في خطوط أنابيب CI/CD.
يُستخدم عند تكوين سير عمل GitHub Actions، أو GitLab CI/CD، أو إنشاءات Docker، أو اختبار المصفوفة عبر إصدارات Python.
يتضمن أنماطًا لتحسين ذاكرة التخزين المؤقت وملفات القفل المجمدة والإنشاءات متعددة المراحل ونشر PyPI مع النشر الموثوق.
يغطي إجراءات إعداد GitHub Actions، وبنيات الإنتاج/التطوير متعددة المراحل لـ Docker، وأنماط النشر.
