Настройте uv (менеджер пакетов Python на основе Rust) в конвейерах CI/CD.
Используйте при настройке рабочих процессов GitHub Actions, GitLab CI/CD, сборок Docker или матричного тестирования версий Python.
Включает шаблоны для оптимизации кэша, замороженные файлы блокировки, многоэтапные сборки и публикацию PyPI с доверенной публикацией.
Охватывает действие GitHub Actions setup-uv, многоэтапные сборки/разработки Docker и шаблоны развертывания.
