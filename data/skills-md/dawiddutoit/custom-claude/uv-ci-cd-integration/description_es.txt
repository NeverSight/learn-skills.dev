Configure uv (administrador de paquetes Python basado en Rust) en canalizaciones de CI/CD.
Úselo al configurar flujos de trabajo de GitHub Actions, GitLab CI/CD, compilaciones de Docker o pruebas matriciales en todas las versiones de Python.
Incluye patrones para optimización de caché, archivos de bloqueo congelados, compilaciones de varias etapas y publicación PyPI con publicación confiable.
Cubre la acción setup-uv de GitHub Actions, las compilaciones de producción/desarrollo de varias etapas de Docker y los patrones de implementación.
