Выполняет проверку двойного типа с помощью Pyright (быстрая, разработка) и Mypy (тщательная, CI/CD). Используйте при проверке типов перед фиксацией, отладке ошибок типов или проверке аннотаций типов. Объясняет, почему необходимы оба инструмента, различия в конфигурации и когда использовать каждый из них. Работает с файлами Python .py, pyproject.toml, включен строгий режим.
