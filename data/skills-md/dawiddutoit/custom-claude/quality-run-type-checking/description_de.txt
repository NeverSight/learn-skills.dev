Führt eine doppelte Typprüfung mit pyright (schnell, Entwicklung) und mypy (gründlich, CI/CD) durch. Wird zum Überprüfen von Typen, vor Festschreibungen, zum Debuggen von Typfehlern oder zum Validieren von Typanmerkungen verwendet. Erklärt, warum beide Tools erforderlich sind, welche Konfigurationsunterschiede es gibt und wann die einzelnen Tools verwendet werden sollten. Funktioniert mit Python .py-Dateien, pyproject.toml, strikter Modus aktiviert.
