تشغيل فحص النوع المزدوج باستخدام pyright (سريع، تطوير) وmypy (شامل، CI/CD). يُستخدم عند التحقق من الأنواع، أو قبل الالتزامات، أو تصحيح أخطاء النوع، أو التحقق من صحة التعليقات التوضيحية للنوع. يشرح سبب الحاجة إلى كلتا الأداتين، واختلافات التكوين، ومتى يتم استخدام كل منهما. يعمل مع ملفات Python .py وpyproject.toml مع تمكين الوضع الصارم.
