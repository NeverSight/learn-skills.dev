Предоставляет полные шаблоны для тестирования асинхронного кода Python с помощью pytest: конфигурация pytest-asyncio, использование AsyncMock, асинхронные приспособления, тестирование FastAPI с AsyncClient, тестирование асинхронных производителей/потребителей Kafka, цикл событий и шаблоны очистки.

Используйте при: тестировании асинхронных функций, вариантов асинхронного использования, конечных точек FastAPI, асинхронных операций с базой данных, асинхронных клиентов Kafka или любых шаблонов кода асинхронности/ожидания.
