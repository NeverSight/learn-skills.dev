يوفر أنماطًا كاملة لاختبار كود Python غير المتزامن مع pytest: تكوين pytest-asyncio، واستخدام AsyncMock، والتركيبات غير المتزامنة، واختبار FastAPI مع AsyncClient، واختبار منتجي/مستهلكي Kafka غير المتزامنين، وحلقة الأحداث وأنماط التنظيف.

يُستخدم عندما: اختبار وظائف غير متزامنة، أو حالات استخدام غير متزامنة، أو نقاط نهاية FastAPI، أو عمليات قاعدة البيانات غير المتزامنة، أو عملاء Kafka غير المتزامنة، أو أي أنماط تعليمات برمجية غير متزامنة/انتظار.
