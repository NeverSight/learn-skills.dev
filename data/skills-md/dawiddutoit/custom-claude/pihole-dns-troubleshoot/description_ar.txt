لا يتم حل Pi-hole محليًا، وتكوين DNS لجهاز التوجيه، والتحقق من سجل DNS.

لا يتم حل DNS، أو تحتاج إلى التحقق من سجلات DNS Pi-hole. مشغلات على "DNS لا يعمل"،
"لا يمكن حل المجال"، أو "لم يتم العثور على المجال"، أو "مشكلة DNS Pi-hole"، أو "إصلاح DNS"، أو
"استكشاف أخطاء تحليل DNS وإصلاحها". يعمل مع Pi-hole DNS وdocker-compose.yml FTLCONF
المتغيرات، وحل DNS للشبكة المحلية.
