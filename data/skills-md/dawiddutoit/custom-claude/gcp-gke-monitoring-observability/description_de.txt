Konfiguriert umfassende Protokollierung, Metriken, verteilte Ablaufverfolgung und Warnungen für
GKE-Anwendungen. Zum Einrichten von Cloud Logging und zum Erstellen von Dashboards in der Cloud
Überwachung, Instrumentierung von Spring Boot mit Metriken, Konfiguration von Fehlerwarnungen
Raten oder Ressourcennutzung oder die Implementierung verteilter Ablaufverfolgung mit Cloud Trace.
Beinhaltet Prometheus-Integration, strukturierte Protokollierungsmuster und Beobachtbarkeit
Best Practices für Microservices.
