تحليل نصوص جلسة Claude Code (ملفات JSONL) للكشف عن محتوى نافذة السياق، وأنماط استخدام الرمز المميز،
وعمليات اتخاذ القرار باستخدام أداة view_session_context.py. يُستخدم عند تصحيح سلوك كلود،
التحقيق في أنماط الرمز المميز، أو تفويض وكيل التتبع، أو تحليل استنفاد السياق. المشغلات قيد التشغيل
"لماذا فعل كلود X"، أو "تحليل الجلسة"، أو "التحقق من سجلات الجلسة"، أو "استنفاد نافذة السياق"، أو
"تفويض وكيل المسار".
