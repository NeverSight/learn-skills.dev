Анализирует расшифровки сеансов Claude Code (файлы JSONL), чтобы выявить содержимое контекстного окна, шаблоны использования токенов,
и процессы принятия решений с использованием инструмента view_session_context.py. Используйте при отладке поведения Клода,
исследование шаблонов токенов, отслеживание делегирования агентов или анализ исчерпания контекста. Запускается по
«почему Клод сделал X», «проанализировать сеанс», «проверить журналы сеанса», «исчерпание контекстного окна» или
«отслеживать делегацию агента».
