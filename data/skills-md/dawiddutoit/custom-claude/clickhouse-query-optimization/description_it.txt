Ottimizza le query ClickHouse per velocità ed efficienza. Aiuta con la progettazione di chiavi primarie, indici sparsi, indici di salto dei dati (minmax, set, filtro Bloom, ngrambf_v1), strategie di partizionamento, proiezioni, ottimizzazione PREWHERE, funzioni approssimative e profilazione delle query con EXPLAIN. Da utilizzare durante la scrittura di query ClickHouse, la progettazione di schemi di tabelle, l'analisi di query lente o l'implementazione di aggregazioni analitiche. Funziona con carichi di lavoro OLAP colonnari.
