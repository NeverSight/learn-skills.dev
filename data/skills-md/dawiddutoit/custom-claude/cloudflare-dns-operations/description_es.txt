registros, administración de configuraciones de zona y actualizaciones dinámicas de DNS a través de la API de Cloudflare. Usar cuando
necesita administración manual de registros DNS, actualizaciones dinámicas de DNS, configuración de configuración de zona o
operaciones fuera del sistema de gestión de dominio. Activadores en "agregar registro DNS", "actualizar DNS",
"eliminar registro DNS", "DNS dinámico", "API de Cloudflare" o "administración manual de DNS". obras
con Cloudflare API v4, scripts auxiliares cf-dns.sh y cf-settings.sh y curl directo
