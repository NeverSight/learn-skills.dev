ينفذ عمليات Cloudflare DNS منخفضة المستوى بما في ذلك إضافة DNS وتحديثه وحذفه
السجلات وإدارة إعدادات المنطقة وتحديثات DNS الديناميكية عبر Cloudflare API. استخدم متى
تحتاج إلى إدارة سجلات DNS اليدوية، أو تحديثات DNS الديناميكية، أو تكوين إعدادات المنطقة، أو
العمليات خارج نظام إدارة المجال. مشغلات على "إضافة سجل DNS"، "تحديث DNS"،
"حذف سجل DNS" أو "DNS الديناميكي" أو "Cloudflare API" أو "إدارة DNS اليدوية". يعمل
مع Cloudflare API v4، وcf-dns.sh، وcf-settings.sh، والنصوص البرمجية المساعدة، والضفيرة المباشرة
مكالمات API.
