Выполняет низкоуровневые операции Cloudflare DNS, включая добавление, обновление и удаление DNS.
записи, управление настройками зоны и динамические обновления DNS через API Cloudflare. Используйте, когда
требуется ручное управление записями DNS, динамические обновления DNS, настройка параметров зоны или
операции вне системы управления доменом. Срабатывает при «добавлении записи DNS», «обновлении DNS»,
«удалить запись DNS», «динамический DNS», «Cloudflare API» или «управление DNS вручную». Работает
с Cloudflare API v4, вспомогательными скриптами cf-dns.sh и cf-settings.sh и прямым завихрением
API-вызовы.
