Führt Cloudflare-DNS-Vorgänge auf niedriger Ebene durch, einschließlich Hinzufügen, Aktualisieren und Löschen von DNS
Datensätze, Verwaltung von Zoneneinstellungen und dynamische DNS-Updates über die Cloudflare-API. Verwenden Sie wann
Sie benötigen eine manuelle DNS-Eintragsverwaltung, dynamische DNS-Updates, die Konfiguration von Zoneneinstellungen oder
Vorgänge außerhalb des Domänenverwaltungssystems. Löst auf „DNS-Eintrag hinzufügen“, „DNS aktualisieren“ aus.
„DNS-Eintrag löschen“, „dynamisches DNS“, „Cloudflare API“ oder „manuelle DNS-Verwaltung“. Funktioniert
mit Cloudflare API v4, den Hilfsskripten cf-dns.sh und cf-settings.sh und direktem Curl
API-Aufrufe.
