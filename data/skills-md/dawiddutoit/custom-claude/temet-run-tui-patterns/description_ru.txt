Создает компоненты TUI, запускаемые temet, в соответствии с шаблонами проекта для мониторинга демонов, управления агентами и интеграции возможностей наблюдения. Используйте при реализации информационных панелей агентов, отображений состояния, средств просмотра журналов, многоагентных макетов, интеграции IPC и инструментов OpenTelemetry в текстовых виджетах. Специально для архитектуры и требований temet-run.
