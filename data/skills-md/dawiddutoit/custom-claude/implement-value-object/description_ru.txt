Создает неизменяемые объекты значений домена, используя шаблон замороженного класса данных с проверкой.
Используйте при реализации объектов значений домена, создании неизменяемых структур данных или
добавление проверки значений. Охватывает @dataclass(frozen=True), object.__setattr__()
шаблон в __post_init__, фабричные методы (from_string, from_dict, from_content),
и проверка в замороженном контексте. Триггеры «создать объект значения для X», «реализовать
неизменяемое значение Y», «добавить проверку к значению Z» или «построить объект значения».
