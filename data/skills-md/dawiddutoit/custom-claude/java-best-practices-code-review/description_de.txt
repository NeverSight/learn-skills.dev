Analysiert Java-Code anhand von Best Practices der Branche und bewertet Designprinzipien
einschließlich SOLID, Ausnahmebehandlung, Thread-Sicherheit und Ressourcenverwaltung. Rezensionen
Namenskonventionen, Stream-API-Nutzung, optionale Muster und allgemeine Codequalität.
Verwendung beim Überprüfen von Java-Dateien, Überprüfen der Codequalität, Bewerten der Ausnahmebehandlung,
oder die Prüfung des Ressourcenmanagements.
