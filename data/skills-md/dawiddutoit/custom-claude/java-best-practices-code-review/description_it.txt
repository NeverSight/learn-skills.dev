Analizza il codice Java rispetto alle migliori pratiche del settore e valuta i principi di progettazione
inclusi SOLID, gestione delle eccezioni, sicurezza dei thread e gestione delle risorse. Recensioni
convenzioni di denominazione, utilizzo dell'API Stream, modelli facoltativi e qualità generale del codice.
Da utilizzare durante la revisione di file Java, il controllo della qualità del codice, la valutazione della gestione delle eccezioni,
o controllare la gestione delle risorse.
