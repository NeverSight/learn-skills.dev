يقوم بإنشاء وتعديل وإدارة الوكلاء الفرعيين لـ Claude Code عن طريق كتابة ملفات الوكيل باستخدام YAML frontmatter،
مطالبات النظام وتكوينات الأداة. استخدمه عندما تحتاج إلى "إنشاء وكيل"، "تعديل وكيل"،
"إعداد متخصص"، أو "أحتاج إلى وكيل لـ [المهمة]"، أو "وكيل للتعامل مع [المجال]"، أو "تكوين
أدوات الوكيل". يغطي تنسيق ملف الوكيل، وواجهة YAML، ومطالبات النظام، وقيود الأداة،
تكامل MCP واختيار النموذج والاختبار.
