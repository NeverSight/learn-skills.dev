Complete de forma segura formularios web en Chrome con la aprobación y verificación del usuario en cada paso.
Úselo cuando se le solicite "completar este formulario", "completar el formulario por mí", "completar el registro",
"enviar esta solicitud", o cuando el usuario necesita automatización de formularios con controles de seguridad.
Funciona con pestañas del navegador Chrome a través de herramientas MCP (tabs_context_mcp, read_page, form_input).
Aplica flujos de trabajo de permisos, verificación de campos y nunca completa datos confidenciales.
