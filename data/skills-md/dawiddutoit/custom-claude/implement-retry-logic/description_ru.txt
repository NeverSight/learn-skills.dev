Реализует логику повтора с экспоненциальной задержкой и джиттером для асинхронных операций. Используйте при реализации
вызовы внешних служб, операции с базой данных, сетевые запросы или любые операции, которые
может временно выйти из строя. Соблюдает шаблоны проекта для упаковки и настройки ServiceResult.
модели инъекций и устойчивости. Триггеры по «добавить логику повтора для X», «реализовать экспоненциальную
откат», «сделать Z устойчивым» или «обработать временные ошибки».
