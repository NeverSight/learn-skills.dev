Implementa lógica de reintento con retroceso exponencial y fluctuación para operaciones asíncronas. Usar al implementar
llamadas de servicio externo, operaciones de bases de datos, solicitudes de red o cualquier operación que
puede fallar transitoriamente. Sigue los patrones del proyecto para el ajuste y la configuración de ServiceResult.
patrones de inyección y resiliencia. Activadores en "agregar lógica de reintento para X", "implementar exponencial
retroceder", "hacer que Z sea resistente" o "manejar errores transitorios".
