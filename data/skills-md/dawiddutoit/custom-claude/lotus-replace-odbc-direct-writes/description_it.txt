Sostituisce le scritture dirette del database ODBC e le connessioni back-end con modelli di integrazione adeguati basati su API.
Da utilizzare durante la migrazione del codice che modifica direttamente i database Lotus Notes tramite ODBC, convertendo il backend
connessioni alle API HTTP, implementando modelli di richiesta/risposta adeguati o garantendo la sicurezza
sincronizzazione dei dati.
