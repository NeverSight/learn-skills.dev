Предоставляет подробное руководство по добавлению служб в контейнер внедрения зависимостей с помощью
Шаблоны библиотеки зависимостей, включая Singleton, Factory и Dependency
поставщиков, шаблоны переопределения для тестирования и обнаружение циклических зависимостей. Используйте, когда
создание нового сервиса, добавление зависимости к контейнеру, отладка циклической зависимости
ошибки или компоненты проводки для впрыска.
