Proporciona una guía completa para agregar servicios al contenedor de inyección de dependencia usando
patrones de biblioteca de inyector de dependencia que incluyen Singleton vs Factory vs Dependency
proveedores, anulación de patrones para pruebas y detección de dependencia circular. Usar cuando
crear un nuevo servicio, agregar dependencia al contenedor, depurar dependencia circular
errores o cableado de componentes para inyección.
