Configura i webhook GitHub per la distribuzione automatica verificando i segreti e controllando l'endpoint
accessibilità e guida alla configurazione. Utilizzare quando è necessario aggiungere un nuovo repository per
eseguire la distribuzione automatica, configurare i webhook GitHub, impostare la distribuzione continua o risolvere i problemi
problemi con i webhook. Si attiva su "configura webhook", "aggiungi webhook per [repo]", "configure
distribuzione automatica", "distribuzione GitHub" o "webhook non funzionante". Funziona con config/hooks.json,
.env (WEBHOOK_SECRET) e endpoint webhook.temet.ai.
