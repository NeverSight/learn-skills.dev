Konfiguriert GitHub-Webhooks für die automatische Bereitstellung durch Überprüfung von Geheimnissen und Überprüfung des Endpunkts
Zugänglichkeit und Führungskonfiguration. Verwenden Sie diese Option, wenn Sie ein neues Repo hinzufügen müssen
Automatische Bereitstellung, Konfiguration von GitHub-Webhooks, Einrichtung einer kontinuierlichen Bereitstellung oder Fehlerbehebung
Webhook-Probleme. Löst auf „Webhook einrichten“, „Webhook für [Repo] hinzufügen“, „konfigurieren“ aus
Auto-Deploy“, „GitHub-Bereitstellung“ oder „Webhook funktioniert nicht“. Funktioniert mit config/hooks.json,
.env (WEBHOOK_SECRET) und webhook.temet.ai-Endpunkt.
