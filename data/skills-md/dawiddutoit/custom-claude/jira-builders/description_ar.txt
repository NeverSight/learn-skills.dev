أدلة لاستخدام jira-tool CLI بشكل صحيح لإنشاء تذاكر Jira وإدارتها بتنسيق غني. يُستخدم عند العمل مع تذاكر Jira أو الملاحم أو الصادرات. مشغلات عند "إنشاء تذكرة Jira"، أو "البحث في Jira"، أو "الحصول على تذكرة Jira"، أو "تصدير بيانات Jira"، أو "قائمة الملاحم"، أو أي عمليات Jira API. يمنع الأخطاء الشائعة مثل محاولة استيراد وحدة Python jira_tool أو استخدام الضفيرة دون داع. يعمل مع متغيرات الأوامر والبيئة الخاصة بـ jira-tool CLI (JIRA_BASE_URL، JIRA_USERNAME، JIRA_API_TOKEN).
