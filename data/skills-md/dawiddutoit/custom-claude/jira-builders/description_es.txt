Guías para usar correctamente la CLI de jira-tool para crear y administrar tickets de Jira con formato enriquecido. Úselo cuando trabaje con tickets, epopeyas o exportaciones de Jira. Activadores para "crear ticket de Jira", "buscar en Jira", "obtener ticket de Jira", "exportar datos de Jira", "enumerar epopeyas" o cualquier operación de la API de Jira. Evita errores comunes como intentar importar el módulo Python jira_tool o usar curl innecesariamente. Funciona con el comando CLI de jira-tool y las variables de entorno (JIRA_BASE_URL, JIRA_USERNAME, JIRA_API_TOKEN).
