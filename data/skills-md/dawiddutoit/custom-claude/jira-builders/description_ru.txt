Руководства по правильному использованию интерфейса командной строки jira-tool для создания билетов Jira с расширенным форматированием и управления ими. Используйте при работе с заявками, эпиками или экспортами Jira. Триггеры: «создать билет Jira», «поиск в Jira», «получить билет Jira», «экспортировать данные Jira», «составить список эпиков» или любые операции API Jira. Предотвращает распространенные ошибки, такие как попытка импортировать модуль Python jira_tool или ненужное использование curl. Работает с командой CLI jira-tool и переменными среды (JIRA_BASE_URL, JIRA_USERNAME, JIRA_API_TOKEN).
