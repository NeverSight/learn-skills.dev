Guide per utilizzare correttamente la CLI di jira-tool per creare e gestire ticket Jira con formattazione avanzata. Da utilizzare quando si lavora con ticket Jira, epiche o esportazioni. Si attiva su "crea ticket Jira", "cerca Jira", "ottieni ticket Jira", "esporta dati Jira", "elenca epics" o qualsiasi operazione API Jira. Previene errori comuni come il tentativo di importare il modulo jira_tool Python o l'utilizzo di curl inutilmente. Funziona con il comando CLI di jira-tool e le variabili di ambiente (JIRA_BASE_URL, JIRA_USERNAME, JIRA_API_TOKEN).
