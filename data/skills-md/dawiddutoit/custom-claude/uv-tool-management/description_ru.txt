Управляет инструментами Python CLI с помощью uv. Узнайте, когда использовать uvx для временного выполнения, установку инструмента uv для постоянных инструментов и как различать зависимости инструментов и зависимости проекта. Включает управление версиями, составление списков, очистку и сценарии для запуска инструментов с конкретными версиями Python.
