Gestiona las herramientas CLI de Python con uv. Aprenda cuándo usar uvx para ejecución temporal, instalación de herramientas uv para herramientas persistentes y cómo diferenciar entre dependencias de herramientas y dependencias de proyectos. Incluye administración de versiones, listado, limpieza y escenarios para ejecutar herramientas con versiones específicas de Python.
