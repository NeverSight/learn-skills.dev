Erstellt vollständige Spring Boot-Dienste gemäß Best Practices und mehrschichtiger Architektur. Verwenden Sie es beim Erstellen von REST-APIs, beim Erstellen von Spring-Diensten, beim Einrichten von Controllern, beim Implementieren von Repositorys, beim Hinzufügen von Spring Boot-Anwendungen, beim Konfigurieren der Abhängigkeitsinjektion, beim Erstellen von RESTful-Endpunkten, beim Hinzufügen von OpenAPI-Dokumentation oder beim Erstellen von Mikrodiensten. Funktioniert mit Spring Boot 2.x und 3.x, Spring Data JPA, Spring Web und Spring Security.
