Esegue controlli di sicurezza completi del codice Java rispetto alla Top 10 OWASP e alle migliori pratiche. Da utilizzare durante il controllo della sicurezza, il controllo delle vulnerabilità, l'analisi dei rischi di SQL injection, la prevenzione degli attacchi XSS, la revisione dell'autenticazione/autorizzazione, il rilevamento dell'esposizione di dati sensibili, il controllo delle vulnerabilità delle dipendenze, la garanzia della conformità OWASP o il rafforzamento delle applicazioni Java. Funziona con applicazioni Web Java, API REST, applicazioni Spring e qualsiasi codebase Java.
