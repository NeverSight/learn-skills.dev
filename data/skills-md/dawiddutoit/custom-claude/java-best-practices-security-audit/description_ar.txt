إجراء عمليات تدقيق أمنية شاملة لتعليمات Java البرمجية مقابل أفضل 10 ممارسات OWASP وأفضلها. يُستخدم عند تدقيق الأمان، أو التحقق من الثغرات الأمنية، أو تحليل مخاطر حقن SQL، أو منع هجمات XSS، أو مراجعة المصادقة/الترخيص، أو اكتشاف تعرض البيانات الحساسة، أو التحقق من ثغرات التبعية، أو ضمان امتثال OWASP، أو تقوية تطبيقات Java. يعمل مع تطبيقات الويب Java وREST APIs وتطبيقات Spring وأي قاعدة تعليمات برمجية لـ Java.
