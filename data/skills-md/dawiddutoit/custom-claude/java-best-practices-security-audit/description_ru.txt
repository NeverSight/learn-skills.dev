Выполняет комплексные проверки безопасности кода Java на предмет соответствия 10 лучшим практикам OWASP. Используйте при аудите безопасности, проверке уязвимостей, анализе рисков SQL-инъекций, предотвращении XSS-атак, проверке аутентификации/авторизации, обнаружении раскрытия конфиденциальных данных, проверке уязвимостей зависимостей, обеспечении соответствия требованиям OWASP или усилении защиты приложений Java. Работает с веб-приложениями Java, REST API, приложениями Spring и любой кодовой базой Java.
