Создает списки распилов для деревообработки на основе проектов мебели OpenSCAD с использованием библиотеки woodworkers-lib.
библиотека. Автоматизирует извлечение размеров панелей из вывода ECHO для мебели, шкафов,
шкафы и стеллажи. Используйте при проектировании мебели с панелями фанеры/МДФ,
создание списков вырезов для фрезерования с ЧПУ или ручной резки или подготовка данных для листа
инструменты оптимизации. Триггеры «создать список вырезов», «извлечь размеры панели»,
«список вырезов мебели», «выход ECHO для деревообработки» или при работе с planeLeft/planeRight/
Модули planeTop/planeBottom/planeFront/planeBack. Работает с файлами .scad, используя
библиотека woodworkers-lib.
