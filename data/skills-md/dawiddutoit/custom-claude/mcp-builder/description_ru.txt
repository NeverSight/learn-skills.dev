Создает серверы протокола контекста модели (MCP), позволяющие LLM взаимодействовать с внешними сервисами.
с помощью хорошо продуманных инструментов, ресурсов и подсказок. Используйте, когда вас просят «создать сервер MCP»,
«создать интеграцию MCP», «внедрить инструменты MCP», «подключиться к внешнему API через MCP» или
«разработать сервер MCP SDK». Проводит этапы исследования, проектирования, внедрения и тестирования.
для серверов Python (FastMCP) или TypeScript (MCP SDK). Охватывает наименование инструментов, проектирование схемы,
обработка ошибок, тестирование и оптимизация рабочих процессов. Работает с протоколом MCP, платформой FastMCP,
и MCP TypeScript SDK.
