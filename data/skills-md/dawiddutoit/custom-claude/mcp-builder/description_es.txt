Crea servidores Model Context Protocol (MCP) que permiten a los LLM interactuar con servicios externos
a través de herramientas, recursos e indicaciones bien diseñados. Úselo cuando se le solicite "construir un servidor MCP",
"crear integración MCP", "implementar herramientas MCP", "conectarse a una API externa a través de MCP" o
"desarrollar el servidor MCP SDK". Guía a través de las fases de investigación, diseño, implementación y prueba.
para servidores Python (FastMCP) o TypeScript (MCP SDK). Cubre la denominación de herramientas, el diseño de esquemas,
manejo de errores, pruebas y optimización del flujo de trabajo. Funciona con protocolo MCP, marco FastMCP,
y SDK de MCP TypeScript.
