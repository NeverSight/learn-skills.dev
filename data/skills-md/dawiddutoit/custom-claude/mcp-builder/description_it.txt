Crea server MCP (Model Context Protocol) che consentono ai LLM di interagire con servizi esterni
attraverso strumenti, risorse e istruzioni ben progettati. Utilizzare quando viene richiesto di "costruire un server MCP",
"creare integrazione MCP", "implementare strumenti MCP", "connettersi ad API esterne tramite MCP" o
"sviluppare il server SDK MCP". Guida attraverso le fasi di ricerca, progettazione, implementazione e test
per server Python (FastMCP) o TypeScript (MCP SDK). Copre la denominazione degli strumenti, la progettazione dello schema,
gestione degli errori, test e ottimizzazione del flusso di lavoro. Funziona con il protocollo MCP, il framework FastMCP,
e SDK MCP TypeScript.
