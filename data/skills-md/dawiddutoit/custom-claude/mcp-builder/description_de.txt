Erstellt Model Context Protocol (MCP)-Server, die es LLMs ermöglichen, mit externen Diensten zu interagieren
durch gut gestaltete Tools, Ressourcen und Eingabeaufforderungen. Wird verwendet, wenn Sie aufgefordert werden, „einen MCP-Server zu erstellen“.
„MCP-Integration erstellen“, „MCP-Tools implementieren“, „über MCP mit externer API verbinden“ oder
„MCP SDK-Server entwickeln“. Leitet durch die Forschungs-, Design-, Implementierungs- und Testphasen
für Python- (FastMCP) oder TypeScript- (MCP SDK) Server. Behandelt Tool-Benennung, Schema-Design,
Fehlerbehandlung, Tests und Workflow-Optimierung. Funktioniert mit MCP-Protokoll, FastMCP-Framework,
und MCP TypeScript SDK.
