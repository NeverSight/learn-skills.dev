يقوم بإنشاء خوادم بروتوكول السياق النموذجي (MCP) لتمكين LLMs من التفاعل مع الخدمات الخارجية
من خلال الأدوات والموارد والمطالبات المصممة جيدًا. يُستخدم عندما يُطلب منك "إنشاء خادم MCP"،
"إنشاء تكامل MCP" أو "تنفيذ أدوات MCP" أو "الاتصال بواجهة برمجة التطبيقات الخارجية عبر MCP" أو
"تطوير خادم MCP SDK". أدلة خلال مراحل البحث والتصميم والتنفيذ والاختبار
لخوادم Python (FastMCP) أو TypeScript (MCP SDK). يغطي تسمية الأداة، وتصميم المخطط،
معالجة الأخطاء والاختبار وتحسين سير العمل. يعمل مع بروتوكول MCP، إطار عمل FastMCP،
وMCP TypeScript SDK.
