Guidare il posizionamento del test nello strato piramidale corretto del test (unità/integrazione/e2e).
Da utilizzare quando si creano nuovi file di test, si decide il livello di test, si organizza la struttura del test,
o determinare l'ambito dell'apparecchio. Analizza modelli fittizi, dipendenze e ambito del test
per consigliare il corretto posizionamento dei livelli. Funziona con file di test pytest (test_*.py).
