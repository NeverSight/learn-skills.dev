Genera casi di test JUnit 5 completi per il codice Java con derisione e copertura adeguate.
Da utilizzare durante la scrittura di test, la creazione di test unitari, la generazione di casi di test, l'aggiunta di test JUnit,
impostare mock Mockito, scrivere test parametrizzati o ottenere la copertura dei test.
Funziona con file .java, genera classi di test seguendo le convenzioni di denominazione,
include casi limite, condizioni al contorno e utilizza il modello Arrange-Act-Assert.
