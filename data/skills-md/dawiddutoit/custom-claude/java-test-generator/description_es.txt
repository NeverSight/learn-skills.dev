Genere casos de prueba completos de JUnit 5 para código Java con simulación y cobertura adecuadas.
Úselo al escribir pruebas, crear pruebas unitarias, generar casos de prueba, agregar pruebas JUnit,
configurar simulacros de Mockito, escribir pruebas parametrizadas o lograr cobertura de prueba.
Funciona con archivos .java, genera clases de prueba siguiendo convenciones de nomenclatura,
incluye casos extremos, condiciones de contorno y utiliza el patrón Arrange-Act-Assert.
