قم بإنشاء حالات اختبار JUnit 5 شاملة لتعليمات Java البرمجية مع الاستهزاء والتغطية المناسبة.
يُستخدم عند كتابة الاختبارات، وإنشاء اختبارات الوحدة، وإنشاء حالات الاختبار، وإضافة اختبارات JUnit،
إعداد نماذج Mockito، أو كتابة اختبارات ذات معلمات، أو تحقيق تغطية الاختبار.
يعمل مع ملفات .java، وينشئ فئات اختبار تتبع اصطلاحات التسمية،
يتضمن حالات الحافة، وشروط الحدود، ويستخدم نمط Arrange-Act-Assert.
