Estrae componenti Svelte riutilizzabili dal codice esistente per migliorare la modularità e ridurre la duplicazione.
Da utilizzare durante il refactoring di file Svelte di grandi dimensioni, la creazione di componenti dell'interfaccia utente condivisi o il miglioramento della riusabilità del codice.
Si attiva su "estrai componente", "refactoring file Svelte", "crea componente riutilizzabile", "troppo codice
in un file" o "componente suddiviso". Funziona con file .svelte utilizzando le rune Svelte 5 ($state, $props,
$derived), interfacce prop appropriate e modelli di organizzazione dei file dei componenti.
