Тестирует пользовательские метрики Micrometer в модульных и интеграционных тестах с использованием SimpleMeterRegistry. Используйте при написании тестов для сервисов, которые записывают метрики, проверяют значения метрик после операций, тестируют процентили и гистограммы или утверждают поведение метрик без полного контекста Spring. Необходим для обеспечения точной записи показателей в бизнес-логике.
