تشغيل عملية فحص وتنسيق ruff مع تفاصيل التكوين والإصلاحات الشائعة. يُستخدم عند التحقق من نمط التعليمات البرمجية، أو تنسيق الملفات، أو إصلاح انتهاكات الوبر، أو قبل الالتزامات. يشرح فحص ruff مقابل تنسيق ruff، وعلامة --fix، والتكوين في pyproject.toml، وأنماط الانتهاك الشائعة. يعمل مع ملفات Python .py، ويستهدف Python 3.12.
