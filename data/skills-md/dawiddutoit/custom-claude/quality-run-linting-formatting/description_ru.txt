Выполняет анализ и форматирование Ruff с подробной информацией о конфигурации и типичными исправлениями. Используйте при проверке стиля кода, форматировании файлов, исправлении нарушений lint или перед фиксацией. Объясняется проверка ruff и формат ruff, флаг --fix, конфигурация в pyproject.toml и распространенные шаблоны нарушений. Работает с файлами Python .py, предназначен для Python 3.12.
