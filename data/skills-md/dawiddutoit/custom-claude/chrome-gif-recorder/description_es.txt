Registra los flujos de trabajo del navegador como GIF anotados para documentación y tutoriales. Úselo al crear documentación visual de las interacciones del navegador, grabar tutoriales paso a paso o capturar flujos de trabajo para tickets de soporte. Se activan frases como "grabar este flujo de trabajo", "crear un tutorial GIF", "documentar este proceso como GIF", "muéstrame cómo hacer X como GIF" o cuando el usuario necesita documentación visual paso a paso. Funciona con la automatización del navegador Chrome a través de herramientas MCP (tabs_context_mcp, gif_creator, computadora).
