Записывает рабочие процессы браузера в виде GIF-файлов с аннотациями для документации и учебных пособий. Используйте при создании визуальной документации по взаимодействиям с браузером, записи пошаговых руководств или рабочих процессов для заявок в службу поддержки. Запускается по таким фразам, как «записать этот рабочий процесс», «создать учебное пособие в формате GIF», «задокументировать этот процесс в формате GIF», «покажите мне, как сделать X в формате GIF» или когда пользователю нужна визуальная пошаговая документация. Работает с автоматизацией браузера Chrome с помощью инструментов MCP (tabs_context_mcp, gif_creator, компьютер).
