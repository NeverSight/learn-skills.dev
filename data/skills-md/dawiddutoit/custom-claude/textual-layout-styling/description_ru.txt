Стиль и макет Текстовые виджеты с использованием синтаксиса, подобного CSS, для адаптивного дизайна.
Используйте при реализации адаптивных макетов, стилей CSS, цветовых схем, интервалов, размеров,
выравнивание, макеты сетки, контейнеры в стиле flexbox и настройка темы.
Охватывает псевдоязык CSS и систему тем Textual.
