النمط والتخطيط عناصر واجهة مستخدم نصية تستخدم بناء جملة يشبه CSS للتصميم سريع الاستجابة.
يُستخدم عند تنفيذ التخطيطات سريعة الاستجابة، وأنماط CSS، وأنظمة الألوان، والمسافات، والتحجيم،
المحاذاة، وتخطيطات الشبكة، والحاويات المشابهة للمربع المرن، وتخصيص السمات.
يغطي لغة CSS الزائفة ونظام السمات الخاص بـ Textual.
