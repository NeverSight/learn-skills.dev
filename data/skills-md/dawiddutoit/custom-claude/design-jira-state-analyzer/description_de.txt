Entwirft und implementiert Zustandsübergangsanalysesysteme zur Verfolgung der in verschiedenen Staaten verbrachten Zeit. Zur Verwendung bei der Analyse von Workflows mit Statusänderungen (Jira, GitHub PRs, Bereitstellungen, Support-Tickets usw.). Behandelt die Grundlagen von Zustandsmaschinen, zeitliche Berechnungen, die Erkennung von Engpässen und Geschäftsmetriken. Auslöseschlüsselwörter: „Zustandsanalyse“, „Dauerverfolgung“, „Workflow-Metriken“, „Engpass“, „Zykluszeit“, „Zustandsübergänge“, „Zeit im Status“, „wie lange“, „Zustandsdauer“, „Workflow-Leistung“, „Zustandsmaschine“, „Änderungsprotokollanalyse“, „SLA-Verfolgung“, „Prozessmetriken“.
