Diagnostica y resuelve sistemáticamente problemas comunes de GKE, incluidas fallas de pods,
problemas de red, errores de conexión de bases de datos y problemas de Pub/Sub. Usar cuando las vainas
están atascados en Pendiente, CrashLoopBackOff, ImagePullBackOff, experimentan fallas de DNS,
Tiempos de espera de conexión de Cloud SQL o problemas de procesamiento de mensajes de Pub/Sub. Proporciona
flujos de trabajo de depuración sistemática y patrones de solución para aplicaciones Spring Boot.
