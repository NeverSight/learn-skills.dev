Automatiza los flujos de trabajo del navegador Chrome para probar aplicaciones web, depurar con registros de consola/red,
extraer datos, completar formularios e interactuar con aplicaciones web autenticadas (Google Docs,
Gmail, noción). Úselo al probar aplicaciones web locales, depurar problemas de interfaz, automatizar la entrada de datos,
extraer contenido web o trabajar con servicios autenticados. Activadores en "probar mi aplicación web",
"comprobar la consola", "rellenar este formulario", "extraer datos de [URL]", "automatizar [tarea del navegador]",
"abrir [aplicación autenticada]" o depurar problemas de aplicaciones web. Funciona con Chrome a través de Claude en
Extensión de Chrome (herramientas MCP: navegar, hacer clic, form_input, read_console_messages, read_network_requests,
tabs_context_mcp, gif_creator).
