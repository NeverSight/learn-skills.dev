Automatizza i flussi di lavoro del browser Chrome per testare app web, eseguire il debug con log di console/rete,
estrarre dati, compilare moduli e interagire con applicazioni web autenticate (Google Docs,
Gmail, Nozione). Da utilizzare durante il test di app Web locali, il debug di problemi frontend, l'automazione dell'immissione di dati,
raschiare contenuti web o lavorare con servizi autenticati. Trigger su "prova la mia app Web",
"controlla la console", "compila questo modulo", "estrai dati da [URL]", "automatizza [attività del browser]",
"apri [app autenticata]" o esegui il debug dei problemi dell'applicazione web. Funziona con Chrome tramite Claude in
Estensione Chrome (strumenti MCP: navigazione, clic, form_input, read_console_messages, read_network_requests,
tabs_context_mcp, gif_creator).
