Автоматизирует рабочие процессы браузера Chrome для тестирования веб-приложений, отладки с помощью журналов консоли/сети.
извлечение данных, заполнение форм и взаимодействие с аутентифицированными веб-приложениями (Документы Google,
Gmail, Понятие). Используйте при тестировании локальных веб-приложений, отладке проблем внешнего интерфейса, автоматизации ввода данных,
очистка веб-контента или работа с аутентифицированными сервисами. Триггеры «проверить мое веб-приложение»,
«проверьте консоль», «заполните эту форму», «извлеките данные из [URL]», «автоматизируйте [задачу браузера]»,
«открыть [приложение с аутентификацией]» или отладка проблем веб-приложения. Работает с Chrome через Клода в
Расширение Chrome (инструменты MCP: навигация, щелчок, form_input, read_console_messages, read_network_requests,
tabs_context_mcp, gif_creator).
