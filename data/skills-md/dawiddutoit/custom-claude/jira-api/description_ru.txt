Содержит подробный справочник по документации и передовым практикам Atlassian Jira REST API v3.
Используйте, когда задаете вопросы о конечных точках API Jira, аутентификации, форматах запросов/ответов, запросах JQL,
Формат документов Atlassian (ADF), веб-перехватчики, обработка ошибок, ограничение скорости и шаблоны использования API.
Триггерные термины: «Конечная точка API Jira», «Как использовать API», «Запрос JQL», «Формат ADF»,
«Аутентификация API», «Запрос API», «Полезная нагрузка веб-перехватчика», «Jira REST API», «Настраиваемые поля»,
«Ограничение скорости API», «Ошибка API», «развернуть поля», «нумерация страниц».
Работает с Jira Cloud REST API v3, Python JiraClient и содержимым в формате ADF.
