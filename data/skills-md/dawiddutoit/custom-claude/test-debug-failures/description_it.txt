Debug dei test basato sull'evidenza che impone l'analisi sistematica delle cause alla radice.
Da utilizzare quando i test falliscono, si verificano errori pytest, la suite di test non viene superata,
eseguire il debug degli errori dei test o correggere i test non funzionanti. Previene l'ipotesi basata su presupposti
correzioni applicando la corretta sequenza diagnostica. Funziona con Python (.py),
File di test JavaScript/TypeScript (.js/.ts), Go, Rust. Supporta pytest,
jest, vitest, mocha, go test, cargo test e altri framework.
