Отладка тестов на основе фактических данных, обеспечивающая систематический анализ первопричин.
Используйте, когда тесты терпят неудачу, возникают ошибки pytest, набор тестов не проходит,
отладка сбоев тестов или исправление неработающих тестов. Предотвращает основанное на предположениях
исправления путем обеспечения правильной диагностической последовательности. Работает с Python (.py),
Тестовые файлы JavaScript/TypeScript (.js/.ts), Go, Rust. Поддерживает pytest,
jest, vitest, mocha, go test, Cargo Test и другие фреймворки.
