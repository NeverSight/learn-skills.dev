توفير موارد Google Cloud Platform وإدارتها باستخدام Terraform. يُستخدم عند العمل مع موضوعات/اشتراكات Pub/Sub ومجموعات GKE،
Cloud SQL، أو أدوار IAM، أو Cloud Storage، أو VPCs، أو حسابات الخدمة، أو أي خدمة GCP. يغطي تكوين المزود،
أنماط الموارد والمصادقة وأفضل الممارسات الخاصة ببرنامج Google Cloud Platform.
