Visuelle Inspektion von Live-Websites, um Designprobleme zu finden und zu beheben. Verwenden Sie es, wenn Sie das Layout/Design der Benutzeroberfläche überprüfen, das responsive Design visuell überprüfen, visuelle Inkonsistenzen erkennen oder CSS-/Barrierefreiheitsprobleme auf Quellcodeebene diagnostizieren. Nicht für automatisierte E2E-Tests.
