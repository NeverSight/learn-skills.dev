Analysez une base de code et un contexte commercial pour découvrir des opportunités de référencement programmatique, en identifiant les types de pages à générer, les actifs de données existants et les intentions de recherche pouvant être mises en correspondance à grande échelle. À utiliser lors du démarrage d'un nouveau projet pSEO, lorsque l'utilisateur n'est pas sûr de ce qu'il doit créer par programme ou lors de l'exploration des données structurées existantes dans la base de code ou dans l'entreprise qui pourraient alimenter des pages évolutives.
