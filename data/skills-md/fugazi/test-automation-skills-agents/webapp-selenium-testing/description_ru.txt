Набор инструментов для автоматизации браузера с использованием Selenium WebDriver с Java и JUnit 5. Используйте для создания, отладки или запуска тестов Selenium, реализации объектной модели страницы, обработки явных ожиданий, создания снимков экрана или настройки тестовых проектов Maven. Поддерживает Chrome, Firefox и Edge.
