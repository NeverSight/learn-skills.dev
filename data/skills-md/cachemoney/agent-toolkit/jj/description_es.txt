Jujutsu (jj): el sistema de control de versiones compatible con Git. Actívelo SÓLO cuando haya un directorio .jj/ en el proyecto o cuando se mencione explícitamente jj/jujutsu. NO lo active para repositorios de git simples sin .jj/. Úselo para cualquier operación de VCS en proyectos administrados por jj: confirmar, enviar, extraer, bifurcar, marcar, rebase, aplastar, fusionar, diferenciar, registrar, estado, copia de trabajo, cambiar ID, revset, conjunto de archivos, plantilla, configuración, espacios de trabajo.
