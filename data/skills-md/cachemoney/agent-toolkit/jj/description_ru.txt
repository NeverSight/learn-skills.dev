Jujutsu (jj) — Git-совместимая система контроля версий. Активируйте ТОЛЬКО при наличии в проекте каталога .jj/ или при явном упоминании jj/jujutsu. НЕ активируйте простые репозитории git без .jj/. Используйте для любых операций VCS в проектах, управляемых jj: фиксация, отправка, извлечение, ветвление, закладка, перебазирование, сжатие, слияние, сравнение, журнал, статус, рабочая копия, идентификатор изменения, набор версий, набор файлов, шаблон, конфигурация, рабочие области.
