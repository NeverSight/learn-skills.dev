Руководство по технической коммуникации для разработчиков программного обеспечения. Охватывает структуру электронной почты, этикет командного обмена сообщениями, повестки дня встреч и адаптацию сообщений для технической и нетехнической аудитории. Используйте при составлении профессиональных сообщений, подготовке сообщений для совещаний или улучшении письменного общения.
