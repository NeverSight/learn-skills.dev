استخدم Memberstack CLI لإدارة حسابات Memberstack من الجهاز. يغطي المصادقة والتطبيقات والأعضاء والخطط والأسعار والحقول المخصصة وجداول البيانات والسجلات والأذونات وموفري المصادقة وSSO والمستخدمين. قم بتشغيل هذه المهارة عندما يريد المستخدم التفاعل مع Memberstack - بما في ذلك إدارة الأعضاء، أو الخطط، أو الأسعار، أو الحقول المخصصة، أو جداول/سجلات البيانات، أو الأذونات، أو موفري المصادقة، أو تطبيقات SSO، أو المصادقة مع Memberstack. يتم تشغيله أيضًا عندما يذكر المستخدم "memberstack"، أو "memberstack-cli"، أو إدارة العضوية، أو عمليات بيانات الأعضاء عبر سطر الأوامر.
