Экспертное руководство по проектированию, созданию и документированию готовых к использованию REST API. Используйте, когда пользователь запрашивает создание API, проектирование конечных точек, обработку ошибок, реализацию аутентификации, написание документации API, версию API, структурирование ответов или проверку существующего кода API на предмет передового опыта.
