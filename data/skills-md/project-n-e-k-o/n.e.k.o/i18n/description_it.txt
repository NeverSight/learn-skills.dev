Toolkit i18n (internazionalizzazione) per progetti che utilizzano i18next. Fornisce tre funzioni principali: (1) i18n-check: rileva il testo cinese codificato nei file HTML/JS, (2) i18n-fix: sostituisce il testo codificato con marcatori i18n, (3) i18n-sync: allinea le chiavi di traduzione in più lingue (zh-CN, en, ja, ko, zh-TW). Da utilizzare quando si lavora su attività di internazionalizzazione, si rilevano stringhe non tradotte o si sincronizzano file locali.
