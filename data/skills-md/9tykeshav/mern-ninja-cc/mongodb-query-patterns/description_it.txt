Da utilizzare durante la scrittura di QUALSIASI query Mongoose (.find, .findOne, .findById, .aggregate, .populate), l'aggiunta di operazioni di database a servizi o controller, il collegamento di dati tra servizi, la creazione di endpoint che leggono o scrivono su MongoDB o la revisione del codice che concatena le chiamate di servizio. TRIGGER soprattutto quando si sta per scrivere un nuovo findById o passare un ID al quale potrebbe invece essere passato un documento.
