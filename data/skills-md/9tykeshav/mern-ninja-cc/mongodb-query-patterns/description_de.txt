Verwenden Sie diese Option, wenn Sie JEDE Mongoose-Abfrage schreiben (.find, .findOne, .findById, .aggregate, .populate), Datenbankoperationen zu Diensten oder Controllern hinzufügen, Daten zwischen Diensten verbinden, Endpunkte erstellen, die MongoDB lesen oder schreiben, oder Code überprüfen, der Dienstaufrufe verkettet. TRIGGER, insbesondere wenn eine neue findById geschrieben oder eine ID übergeben werden soll, an der stattdessen ein Dokument übergeben werden könnte.
