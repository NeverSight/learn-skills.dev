Úselo al escribir CUALQUIER consulta de Mongoose (.find, .findOne, .findById, .aggregate, .populate), agregar operaciones de base de datos a servicios o controladores, cablear datos entre servicios, crear puntos finales que leen o escriben en MongoDB o revisar código que encadena llamadas de servicio. TRIGGER, especialmente cuando está a punto de escribir un nuevo findById o pasar una identificación donde se podría pasar un documento.
