v1.0.21 — 관용적인 Go 코드 작성, 코드 검토, 오류 처리, 테스트, 동시성, 보안 및 프로그램 설계를 위한 Go 프로그래밍 언어 기술입니다. 사용자가 'Go 모범 사례'를 명시적으로 언급하지 않더라도 Go 코드를 작성, 검토, 디버깅하거나 질문할 때 사용하세요. 또한 Go PR 검토, Go 테스트 디버깅, Go 오류 수정, Go API 설계, 보안에 민감한 코드 구현, 사용자 입력 처리, 인증, 세션, 암호화, Google AIP 표준을 사용하여 리소스 지향 gRPC API 구축, golangci-lint 구성, slog를 사용하여 구조화된 로깅 설정 또는 Go 관용구 및 패턴에 대한 질문이 있는 경우에도 사용하세요. 테이블 기반 테스트, 오류 래핑, 고루틴 패턴, 인터페이스 디자인, 제네릭, 반복자, Go 1.26까지의 stdlib 패턴, OWASP 보안 관행, 페이지 매김, 필터링, 순서 지정, 필드 마스크 및 리소스 이름을 위한 einride/aip-go를 사용하는 Google AIP(API 개선 제안)를 다룹니다.
