v1.0.21 — 慣用的な Go コードの作成、コード レビュー、エラー処理、テスト、同時実行性、セキュリティ、およびプログラム設計のための Go プログラミング言語スキル。ユーザーが「Go のベスト プラクティス」について明示的に言及していない場合でも、Go コードを作成、レビュー、デバッグ、または質問するときに使用します。また、次のような場合にも使用します。Go PR のレビュー、Go テストのデバッグ、Go エラーの修正、Go API の設計、セキュリティ重視のコードの実装、ユーザー入力、認証、セッション、暗号化の処理、Google AIP 標準を使用したリソース指向の gRPC API の構築、golangci-lint の構成、slog を使用した構造化ログの設定、または Go のイディオムとパターンに関する質問。テーブル駆動テスト、エラー ラッピング、Goroutine パターン、インターフェイス設計、ジェネリックス、イテレータ、Go 1.26 までの stdlib パターン、OWASP セキュリティ プラクティス、およびページネーション、フィルタリング、順序付け、フィールド マスク、リソース名のための einride/aip-go を使用した Google AIP (API Improvement Proposals) をカバーします。
