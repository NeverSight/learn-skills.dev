v1.0.21: competenze nel linguaggio di programmazione Go per la scrittura di codice Go idiomatico, revisione del codice, gestione degli errori, test, concorrenza, sicurezza e progettazione di programmi. Da utilizzare durante la scrittura, la revisione, il debug o la richiesta di informazioni sul codice Go, anche se l'utente non menziona esplicitamente le "migliori pratiche Go". Da utilizzare anche per: rivedere i PR Go, eseguire il debug dei test Go, correggere errori Go, progettare API Go, implementare codice sensibile alla sicurezza, gestire input utente, autenticazione, sessioni, crittografia, creare API gRPC orientate alle risorse con standard AIP di Google, configurare golangci-lint, impostare la registrazione strutturata con slog o qualsiasi domanda sugli idiomi e modelli Go. Copre test guidati da tabelle, avvolgimento degli errori, modelli goroutine, progettazione dell'interfaccia, generici, iteratori, modelli stdlib fino a Go 1.26, pratiche di sicurezza OWASP e Google AIP (API Improvement Proposals) con einride/aip-go per impaginazione, filtraggio, ordinamento, maschere di campo e nomi di risorse.
