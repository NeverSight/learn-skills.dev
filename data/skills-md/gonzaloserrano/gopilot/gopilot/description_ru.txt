v1.0.21 — Навыки языка программирования Go для написания идиоматического кода Go, проверки кода, обработки ошибок, тестирования, параллелизма, безопасности и проектирования программ. Используйте при написании, проверке, отладке или вопросах о коде Go — даже если пользователь явно не упоминает «лучшие практики Go». Также используйте при просмотре отчетов Go PR, отладке тестов Go, исправлении ошибок Go, разработке API Go, реализации чувствительного к безопасности кода, обработке пользовательского ввода, аутентификации, сеансов, криптографии, создании ресурсно-ориентированных API-интерфейсов gRPC со стандартами Google AIP, настройке golangci-lint, настройке структурированного журналирования с помощью slog или при возникновении любых вопросов об идиомах и шаблонах Go. Охватывает тесты на основе таблиц, перенос ошибок, шаблоны горутин, дизайн интерфейса, универсальные шаблоны, итераторы, шаблоны stdlib до Go 1.26, методы обеспечения безопасности OWASP и Google AIP (предложения по улучшению API) с einride/aip-go для нумерации страниц, фильтрации, упорядочивания, масок полей и имен ресурсов.
