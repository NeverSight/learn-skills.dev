Da utilizzare quando l'utente richiede diagrammi, diagrammi di flusso, grafici di architettura o visualizzazioni. Utilizzalo anche in modo proattivo quando spieghi sistemi con più di 3 componenti, flussi di dati complessi o relazioni che traggono vantaggio dalla rappresentazione visiva. Genera file XML .drawio ed esporta localmente in PNG/SVG/PDF utilizzando la CLI desktop nativa draw.io.
