Guide des opérations de la base de connaissances RemNote. Utilisez l'outil de ligne de commande remnote-bridge pour lire, modifier et rechercher du contenu dans la base de connaissances RemNote. Utilisez cette compétence lorsque les utilisateurs doivent manipuler la base de connaissances RemNote, notamment lire des notes, modifier du contenu, rechercher, parcourir la structure de la base de connaissances, créer des flashcards, modifier les propriétés du document, etc. Les mots-clés déclencheurs incluent : RemNote, base de connaissances, notes, Rem, flashcards, documents, lecture-rem, modification-rem, lecture-arborescence, modification-arborescence, recherche, lecture-globe, lecture-contexte. Même si l'utilisateur ne mentionne pas explicitement les commandes CLI, tant que l'intention est d'opérer sur des données dans RemNote, cette compétence doit être déclenchée.
