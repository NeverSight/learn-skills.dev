Guía de funcionamiento de la base de conocimientos de RemNote. Utilice la herramienta CLI remnote-bridge para leer, editar y buscar contenido en la base de conocimientos de RemNote. Utilice esta habilidad cuando los usuarios necesiten manipular la base de conocimientos de RemNote, incluida la lectura de notas, la edición de contenido, la búsqueda, la exploración de la estructura de conocimientos, la creación de tarjetas didácticas, la modificación de propiedades de documentos, etc. Las palabras clave desencadenantes incluyen: RemNote, base de conocimientos, notas, Rem, tarjetas didácticas, documentos, lectura-rem, edición-rem, lectura-árbol, edición-árbol, búsqueda, lectura-globo, lectura-contexto. Incluso si el usuario no menciona explícitamente los comandos CLI, siempre que la intención sea operar con datos dentro de RemNote, esta habilidad debe activarse.
