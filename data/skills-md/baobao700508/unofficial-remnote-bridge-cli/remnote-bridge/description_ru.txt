Руководство по использованию базы знаний RemNote. Используйте инструмент командной строки remnote-bridge для чтения, редактирования и поиска содержимого в базе знаний RemNote. Используйте этот навык, когда пользователю необходимо работать с базой знаний RemNote, включая чтение заметок, редактирование содержимого, поиск, просмотр структуры базы знаний, создание карточек, изменение свойств документа и т. д. Ключевые слова-триггеры включают в себя: RemNote, база знаний, заметки, Rem, карточки, документы, чтение-rem, редактирование-rem, дерево чтения, дерево редактирования, поиск, глобус чтения, контекст чтения. Даже если пользователь явно не упоминает команды CLI, если его целью является работа с данными в RemNote, этот навык должен быть активирован.
