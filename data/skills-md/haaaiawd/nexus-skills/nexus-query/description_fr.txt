Requêtes précises et instantanées sur la structure du code pour un développement actif - répondez « qui dépend de cette interface avant de la refactoriser », « combien de modules se cassent si je change cela », « quel est le rayon d'impact réel de ce changement de fonctionnalité », « quel module est le véritable point chaud de couplage élevé dans cette base de code héritée ». Indispensable avant tout changement d'interface, tâche de refactorisation continue, estimation du travail de sprint ou lors de la navigation dans des bases de code héritées inconnues ou volumineuses. Nécessite Python 3.10+ et Shell. Utilisez plutôt nexus-mapper lors de la création d’une base de connaissances .nexus-map/ complète.
