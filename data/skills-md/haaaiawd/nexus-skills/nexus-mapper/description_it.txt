Genera una base di conoscenza .nexus-map/ persistente che consente a qualsiasi sessione di intelligenza artificiale di comprendere immediatamente l'architettura, i sistemi, le dipendenze e gli hotspot di modifica di una base di codice. Utilizzalo quando inizi a lavorare su un repository sconosciuto, esegui l'onboarding con un contesto assistito dall'intelligenza artificiale, ti prepari per un'importante iniziativa di refactoring o abilita sessioni affidabili di intelligenza artificiale con avvio a freddo in un team. Produce INDEX.md, Systems.md, concept_model.json, git_forensics.md e altro ancora. Richiede l'esecuzione della shell e Python 3.10+. Per query di file ad hoc o analisi di impatto immediato durante lo sviluppo attivo, utilizza invece nexus-query.
