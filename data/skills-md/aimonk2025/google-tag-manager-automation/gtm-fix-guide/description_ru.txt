Простое руководство по исправлению неудачных тестов GTM. Используйте, когда тесты не пройдены и разработчику нужны пошаговые инструкции для устранения каждой проблемы. Читает gtm-test-results.json и пишет руководство по исправлению на простом английском языке — для этого не требуется никаких знаний GTM. Никаких вызовов API. Триггер — «руководство по исправлению», «как исправить неудачные тесты», «неудачные тесты», «неудачные тесты gtm», «исправить отслеживание», «что сломалось».
