Интегрируйте шифрование CipherStash с Amazon DynamoDB с помощью @cipherstash/stack/dynamodb. Охватывает вспомогательную функцию EncryptedDynamoDB для шифрования элементов перед PutItem и расшифровки после GetItem, массового шифрования/дешифрования для BatchWrite и BatchGet, запросов с зашифрованными ключами разделов и сортировки с помощью атрибутов HMAC, шифрования вложенных объектов, ведения журнала аудита и соглашений об именовании атрибутов DynamoDB (__source/__hmac). Используйте при добавлении шифрования в проект DynamoDB, шифровании элементов перед записью, расшифровке элементов после чтения или запросе зашифрованных атрибутов DynamoDB.
