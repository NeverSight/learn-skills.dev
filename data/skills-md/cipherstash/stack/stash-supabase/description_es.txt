Integre el cifrado CipherStash con Supabase usando @cipherstash/stack/supabase. Cubre el contenedor encryptedSupabase, cifrado/descifrado transparente al insertar/actualizar/seleccionar, filtros de consulta cifrados (eq, like, ilike, gt/gte/lt/lte, in, or, match), cifrado con reconocimiento de identidad y la API completa del generador de consultas.
