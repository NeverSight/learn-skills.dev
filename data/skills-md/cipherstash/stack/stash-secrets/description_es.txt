Administre secretos cifrados con @cipherstash/stack. Cubre la API de secretos para almacenar, recuperar, enumerar y eliminar secretos cifrados de un extremo a otro, la CLI oculta para la gestión de secretos basada en terminales, el aislamiento basado en el entorno y la recuperación masiva de secretos. Úselo al implementar la administración de secretos, almacenar claves API o URL de bases de datos, o trabajar con la API o CLI de CipherStash Secrets.
