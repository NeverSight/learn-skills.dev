Управляйте зашифрованными секретами с помощью @cipherstash/stack. Охватывает API-интерфейс Secrets для хранения, получения, перечисления и удаления секретов со сквозным шифрованием, интерфейс командной строки Stash для управления секретами на основе терминала, изоляции на основе среды и массового извлечения секретов. Используйте при реализации управления секретами, хранении ключей API или URL-адресов базы данных, а также при работе с API CipherStash Secrets или CLI.
