Реализуйте шифрование на уровне поля с помощью @cipherstash/stack. Охватывает определение схемы, операции шифрования/дешифрования, шифрование с возможностью поиска (равенство, произвольный текст, диапазон, JSON), массовые операции, операции с моделями, шифрование с учетом идентификационных данных с помощью LockContext, многопользовательские наборы ключей и полную систему типов TypeScript. Используйте при добавлении шифрования в проект, определении зашифрованных схем или работе с API шифрования CipherStash.
