Integra il gateway di pagamento Transbank Webpay Plus utilizzando la sua API REST (HTTP). Utilizza questa competenza ogni volta che l'utente ha bisogno di implementare pagamenti Webpay Plus, creare transazioni di pagamento, confermare pagamenti, rimborsare/stornare transazioni, controllare lo stato della transazione o acquisire pagamenti differiti tramite l'API Webpay Plus. Utilizza questa capacità anche quando l'utente menziona Transbank, Webpay, elaborazione dei pagamenti cilena, integrazione dei pagamenti per il Cile o necessita di credenziali di prova per l'ambiente di integrazione Webpay Plus.
