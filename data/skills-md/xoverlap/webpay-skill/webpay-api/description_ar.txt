قم بدمج بوابة الدفع Transbank Webpay Plus باستخدام REST API (HTTP). استخدم هذه المهارة عندما يحتاج المستخدم إلى تنفيذ مدفوعات Webpay Plus، أو إنشاء معاملات دفع، أو تأكيد المدفوعات، أو استرداد/عكس المعاملات، أو التحقق من حالة المعاملة، أو التقاط المدفوعات المؤجلة عبر Webpay Plus API. استخدم هذه المهارة أيضًا عندما يذكر المستخدم Transbank، أو Webpay، أو معالجة الدفع التشيلية، أو تكامل الدفع في تشيلي، أو عندما يحتاج إلى بيانات اعتماد اختبار لبيئة تكامل Webpay Plus.
