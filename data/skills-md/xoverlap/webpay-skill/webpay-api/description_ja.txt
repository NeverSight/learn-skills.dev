REST API (HTTP) を使用して、Transbank Webpay Plus 支払いゲートウェイを統合します。ユーザーが Webpay Plus 支払いの実装、支払いトランザクションの作成、支払いの確認、トランザクションの返金/取り消し、トランザクション ステータスの確認、または Webpay Plus API 経由の後払い支払いの取得を行う必要がある場合は、このスキルを常に使用します。ユーザーが Transbank、Webpay、チリの支払い処理、チリの支払い統合について言及する場合、または Webpay Plus 統合環境のテスト資格情報が必要な場合にも、このスキルを使用します。
