Интегрируйте платежный шлюз Transbank Webpay Plus, используя его REST API (HTTP). Используйте этот навык всякий раз, когда пользователю необходимо реализовать платежи Webpay Plus, создать платежные транзакции, подтвердить платежи, вернуть/отменить транзакции, проверить статус транзакции или получить отложенные платежи через API Webpay Plus. Также используйте этот навык, когда пользователь упоминает Transbank, Webpay, обработку платежей в Чили, интеграцию платежей для Чили или ему нужны тестовые учетные данные для среды интеграции Webpay Plus.
