Integra la pasarela de pagos Transbank Webpay Plus utilizando su API REST (HTTP). Utilice esta habilidad siempre que el usuario necesite implementar pagos de Webpay Plus, crear transacciones de pago, confirmar pagos, reembolsar/revertir transacciones, verificar el estado de las transacciones o capturar pagos diferidos a través de la API de Webpay Plus. Utilice también esta habilidad cuando el usuario mencione Transbank, Webpay, procesamiento de pagos en Chile, integración de pagos para Chile o necesite credenciales de prueba para el entorno de integración de Webpay Plus.
