Erstellen Sie Karten, Gebiete, Zonen und große Umgebungen in Roblox Studio über MCP. Verwenden Sie diese Option, wenn der Benutzer eine Karte, eine Kampfarena, eine Lobby, einen Hub, eine Stadt, ein Dorf, ein offenes Gebiet, eine Spielwelt, ein Leveldesign oder eine beliebige Umgebung erstellen möchte, die sich über mehrere Zonen oder Regionen erstreckt. Wird ausgelöst durch: „マップを作って“, „エリアを作って“, „フィールドを作りたい“, „eine Karte erstellen“, „eine Arena erstellen“, „eine Lobby erstellen“, „ein Level entwerfen“, „eine Hub-Welt erstellen“, „eine Stadt erstellen“ oder jede Anfrage bezüglich Raumaufteilung, Zonenplanung oder Mehrbereichsumgebungen. Bereichsgrenze: Gilt für Mehrzonen-Layouts. Für ein einzelnes Objekt, eine Requisite oder einen Rauminnenraum → verwenden Sie stattdessen „building-3d-objects“.
