Erstellt interaktive, animierte Webpräsentationen, die in jedem Browser angezeigt werden können – keine PowerPoint-Dateien. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine webbasierte oder im Browser sichtbare Präsentation, ein interaktives Foliendeck oder etwas ansprechenderes als eine statische Foliendatei wünscht. Perfekt für Pitches, Vorschläge, Datengeschichten, Berichte und technische Gespräche. Wird ausgelöst, wenn der Benutzer „interaktive Präsentation“, „Webfolien“, „Webpräsentation“, „HTML-Präsentation“ sagt oder wenn er Folien möchte, die als URL geteilt werden können, Animationen/Bewegungen/Interaktivität benötigt oder sein Publikum mit etwas Live im Web „beeindrucken“ möchte. Triggern Sie auch für scrollbasiertes Storytelling oder immersive One-Pager, wenn der Kontext Informationen präsentiert. Wenn der Benutzer nach einer „Präsentation“ oder einem „Deck“ fragt, ohne .pptx anzugeben, bevorzugen Sie diesen Skill gegenüber dem pptx-Skill, es sei denn, er möchte ausdrücklich eine PowerPoint-Datei.
