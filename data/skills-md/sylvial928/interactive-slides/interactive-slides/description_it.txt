Crea presentazioni Web interattive e animate visualizzabili in qualsiasi browser, non in file PowerPoint. Utilizza questa capacità quando l'utente desidera una presentazione basata sul Web o visualizzabile tramite browser, una presentazione interattiva o qualcosa di più coinvolgente di un file di diapositive statico. Perfetto per presentazioni, proposte, storie di dati, report e colloqui tecnici. Si attiva quando l'utente dice "presentazione interattiva", "diapositive web", "presentazione web", "presentazione HTML" o quando desidera diapositive che possono essere condivise come URL, ha bisogno di animazioni/movimento/interattività o vuole "stupire" il pubblico con qualcosa dal vivo sul web. Attiva anche la narrazione basata su scorrimento o pagine immersive di una sola pagina quando il contesto presenta informazioni. Se l'utente richiede una "presentazione" o una "composizione" senza specificare .pptx, preferisci questa competenza rispetto alla competenza pptx a meno che non desideri esplicitamente un file PowerPoint.
