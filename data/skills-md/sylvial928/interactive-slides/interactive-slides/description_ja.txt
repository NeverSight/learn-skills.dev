PowerPoint ファイルではなく、どのブラウザでも表示できるインタラクティブなアニメーション Web プレゼンテーションを作成します。このスキルは、ユーザーが Web ベースまたはブラウザーで表示可能なプレゼンテーション、インタラクティブなスライドデッキ、または静的なスライド ファイルよりも魅力的なものを必要とする場合に使用します。ピッチ、提案、データストーリー、レポート、技術的な話に最適です。ユーザーが「インタラクティブ プレゼンテーション」、「Web スライド」、「Web プレゼンテーション」、「HTML プレゼンテーション」と発言したとき、または URL として共有できるスライドが必要な場合、アニメーション/モーション/インタラクティブ性が必要な場合、または Web 上のライブで視聴者を「驚かせたい」場合にトリガーします。また、コンテキストが情報を提示しているときに、スクロールベースのストーリーテリングや没入型の概要資料をトリガーします。ユーザーが .pptx を指定せずに「プレゼンテーション」または「デッキ」を要求した場合は、明示的に PowerPoint ファイルが必要でない限り、pptx スキルよりもこのスキルを優先します。
