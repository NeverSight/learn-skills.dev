Assistant de dépannage interactif pour les 5 principaux problèmes GitHub rencontrés par les nouveaux utilisateurs. Diagnostique les échecs d'authentification, la confusion Git vs GitHub, les problèmes de synchronisation local-distant, les conflits de fusion et les validations accidentelles de données sensibles. Fournit des solutions étape par étape avec l’exécution de commandes. À utiliser lorsque les utilisateurs mentionnent des erreurs GitHub, des problèmes de mot de passe, des problèmes de synchronisation, des conflits de fusion ou la validation accidentelle de secrets.
