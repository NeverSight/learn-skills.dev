KIRIKAE local development proxy environment management skill. Operate the Admin API (localhost:4000) via curl to list, add, switch, rename, update, and delete environments.  
Usage timing: (1) when you want to switch the proxy destination of the development environment, (2) when you want to register a new port, (3) when you want to change the label name of an environment, (4) when you want to check registered environments, (5) when you want to delete an environment.  
Trigger keywords: kirikae, switch, proxy switch, environment switch, port register, port switch
