Habilidad de gestión del entorno proxy de desarrollo local de KIRIKAE. Opere la API de administración (localhost:4000) a través de curl para enumerar, agregar, cambiar, cambiar el nombre, actualizar y eliminar entornos.  
Tiempo de uso: (1) cuando desea cambiar el destino del proxy de un entorno de desarrollo (2) cuando desea registrar un nuevo puerto (3) cuando desea cambiar el nombre de la etiqueta de un entorno (4) cuando desea verificar los entornos registrados (5) cuando desea eliminar un entorno  
Palabras clave de activación: kirikae, conmutador, conmutador de proxy, conmutador de entorno, registro de puerto, conmutador de puerto
