Систематически улучшайте код с помощью структурированных циклов анализа-изменения-оценки. Адаптировано из ALMA (автоматическое метаобучение конструкций памяти для агентных систем). Используйте при проверке качества кода, оптимизации реализаций, отладке постоянных проблем или развитии проекта посредством нескольких циклов улучшения. Заменяет ситуативное «попробуй и исправь» дисциплинированным размышлением, отслеживанием вариантов и принципиальным выбором того, что изменить дальше.
