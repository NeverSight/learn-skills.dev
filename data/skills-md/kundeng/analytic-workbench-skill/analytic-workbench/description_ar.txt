استخدم دائمًا هذه المهارة لسير العمل التحليلي البشري في الحلقة: عمليات تشغيل الدفاتر الاستكشافية، وعمليات المسح التجريبية التي تديرها Hydra، وDAGs للتحليل المستند إلى هاملتون، وحلقات المراجعة/الموافقة، وطبقات إمكانية التكرار الاختيارية من المستوى 3 مثل DVC أو Kedro. استخدمه عندما يريد المستخدم إعداد مسار تحليل أو تشغيله، أو اختيار طبقة، أو مقارنة المعلمات الفائقة، أو إنشاء جدول مقارنة، أو مراجعة المخرجات قبل الموافقة عليها، أو تشغيل المرحلة التالية، أو تحديد كيفية توافق marimo أو Hydra أو Hamilton أو DVC أو Kedro أو Dagster أو Prefect معًا، أو تطوير تحليل من العمل الاستكشافي إلى طاولة عمل منظمة. قم بتشغيل عبارات مثل خط أنابيب التحليل، والتحليل القابل للتكرار، والإنسان في الحلقة، والمرحلة التالية، واكتساح التجربة، ومقارنة المعلمات الفائقة، وجدول المقارنة، ماريمو، هيدرا، هاملتون، DVC، Kedro، Dagster، Prefect، طاولة العمل التحليلية.
