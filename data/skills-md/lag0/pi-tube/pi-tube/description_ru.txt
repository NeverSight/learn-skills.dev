Официальный навык CLI pi-tube для детерминированных рабочих процессов транскрипции мультимедиа.

ИСПОЛЬЗОВАТЬ ДЛЯ:
- YouTube/Instagram/прямой URL-адрес/транскрипция локального файла через CLI
- Детерминированная генерация артефактов Markdown/JSON.
- Готовность поставщика и потоки настройки

Должен быть предварительно установлен. См. правила/install.md для установки и настройки и правила/security.md для обработки вывода.
