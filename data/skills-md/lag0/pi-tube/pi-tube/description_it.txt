Competenza CLI pi-tube ufficiale per flussi di lavoro di trascrizione multimediale deterministici.

UTILIZZARE PER:
- YouTube/Instagram/URL diretto/trascrizione file locale tramite CLI
- Generazione di artefatti deterministici Markdown/JSON
- Preparazione del provider e flussi di configurazione

Deve essere preinstallato. Vedere regole/install.md per l'installazione e la configurazione e regole/security.md per la gestione dell'output.
