Конвейер сквозного клонирования голоса с использованием qwen-tts на Apple Silicon. Охватывает сбор данных (yt-dlp), очистку звука (ASR с parakeet-tdt-0.6b-v3), подготовку данных, тонкую настройку LoRA, генерацию и оценку с помощью проверки разборчивости ASR (локально, без API) + Gemini-as-judge (проверка qwen-tts). Используйте, когда пользователь хочет клонировать голос, обучить модель TTS, подготовить данные для обучения голосу, оценить качество сгенерированной речи или запустить полный конвейер сбора → очистки → подготовки → обучения → генерации → оценки.
