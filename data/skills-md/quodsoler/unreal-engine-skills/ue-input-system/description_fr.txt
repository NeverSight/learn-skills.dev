Utilisez cette compétence lors de la mise en œuvre de la saisie du joueur avec le système de saisie améliorée d'Unreal Engine. À utiliser également lorsque l'utilisateur mentionne « Enhanced Entrée », « entrée », « action d'entrée », « InputAction », « contexte de mappage », « InputMappingContext », « liaison d'entrée », « liaison de touches », « déclencheur d'entrée », « modificateur d'entrée », « manette de jeu » ou « clavier ». Couvre ETriggerEvent, les déclencheurs intégrés (Hold, Tap, Pulse, ChordAction, Combo), les modificateurs intégrés (DeadZone, Scalar, Negate, SwizzleAxis) et la création de déclencheurs/modificateurs personnalisés. Voir references/input-action-reference.md pour le catalogue complet. Pour les modes de saisie de l'interface utilisateur, voir ue-ui-umg-slate.
