Unreal Engine の拡張入力システムを使用してプレーヤー入力を実装する場合は、このスキルを使用します。ユーザーが「拡張入力」、「入力」、「入力アクション」、「InputAction」、「マッピング コンテキスト」、「InputMappingContext」、「入力バインディング」、「キー バインド」、「入力トリガー」、「入力修飾子」、「ゲームパッド」、または「キーボード」について言及する場合にも使用します。 ETriggerEvent、組み込みトリガー (Hold、Tap、Pulse、ChordAction、Combo)、組み込みモディファイアー (DeadZone、Scalar、Negate、SwizzleAxis)、およびカスタム トリガー/モディファイアーのオーサリングをカバーします。完全なカタログについては、references/input-action-reference.md を参照してください。 UI 入力モードについては、ue-ui-umg-slate を参照してください。
