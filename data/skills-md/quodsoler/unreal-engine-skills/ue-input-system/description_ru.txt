Используйте этот навык при реализации ввода данных игроком с помощью системы расширенного ввода Unreal Engine. Также используйте, когда пользователь упоминает «Расширенный ввод», «ввод», «действие ввода», «InputAction», «контекст сопоставления», «InputMappingContext», «привязку ввода», «привязку клавиш», «триггер ввода», «модификатор ввода», «геймпад» или «клавиатуру». Охватывает ETriggerEvent, встроенные триггеры (Hold, Tap, Pulse, ChordAction, Combo), встроенные модификаторы (DeadZone, Scalar, Negate, SwizzleAxis) и разработку пользовательских триггеров/модификаторов. Полный каталог см. в references/input-action-reference.md. Режимы ввода пользовательского интерфейса см. в разделе ue-ui-umg-slate.
