在使用虛幻引擎的增強輸入系統實現玩家輸入時使用此技能。當使用者提及「增強輸入」、「輸入」、「輸入操作」、「InputAction」、「映射上下文」、「InputMappingContext」、「輸入綁定」、「鍵綁定」、「輸入觸發器」、「輸入修飾符」、「遊戲手把」或「鍵盤」時也可使用。涵蓋 ETriggerEvent、內建觸發器（Hold、Tap、Pulse、ChordAction、Combo）、內建修改器（DeadZone、Scalar、Negate、SwizzleAxis）以及自訂觸發器/修改器創作。完整目錄請參考references/input-action-reference.md。對於 UI 輸入模式，請參閱 ue-ui-umg-slate。
