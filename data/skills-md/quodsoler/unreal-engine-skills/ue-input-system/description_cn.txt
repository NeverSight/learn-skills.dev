在使用虚幻引擎的增强输入系统实现玩家输入时使用此技能。当用户提及“增强输入”、“输入”、“输入操作”、“InputAction”、“映射上下文”、“InputMappingContext”、“输入绑定”、“键绑定”、“输入触发器”、“输入修饰符”、“游戏手柄”或“键盘”时也可使用。涵盖 ETriggerEvent、内置触发器（Hold、Tap、Pulse、ChordAction、Combo）、内置修改器（DeadZone、Scalar、Negate、SwizzleAxis）以及自定义触发器/修改器创作。完整目录请参见references/input-action-reference.md。对于 UI 输入模式，请参阅 ue-ui-umg-slate。
