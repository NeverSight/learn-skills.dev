Usa questa abilità quando implementi l'input del giocatore con il sistema Enhanced Input di Unreal Engine. Da utilizzare anche quando l'utente menziona "Enhanced Input", "input", "azione di input", "InputAction", "contesto di mappatura", "InputMappingContext", "associazione di input", "associazione di tasti", "trigger di input", "modificatore di input", "gamepad" o "tastiera". Copre ETriggerEvent, trigger integrati (Hold, Tap, Pulse, ChordAction, Combo), modificatori integrati (DeadZone, Scalar, Negate, SwizzleAxis) e creazione di trigger/modificatori personalizzati. Vedi references/input-action-reference.md per il catalogo completo. Per le modalità di input dell'interfaccia utente, vedere ue-ui-umg-slate.
