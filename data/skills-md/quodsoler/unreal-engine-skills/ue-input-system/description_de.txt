Verwenden Sie diese Fähigkeit, wenn Sie Spielereingaben mit dem Enhanced Input-System der Unreal Engine implementieren. Wird auch verwendet, wenn der Benutzer „Erweiterte Eingabe“, „Eingabe“, „Eingabeaktion“, „Eingabeaktion“, „Zuordnungskontext“, „InputMappingContext“, „Eingabebindung“, „Tastenbindung“, „Eingabeauslöser“, „Eingabemodifikator“, „Gamepad“ oder „Tastatur“ erwähnt. Behandelt ETriggerEvent, integrierte Trigger (Hold, Tap, Pulse, ChordAction, Combo), integrierte Modifikatoren (DeadZone, Scalar, Negate, SwizzleAxis) und die Erstellung benutzerdefinierter Trigger/Modifikatoren. Den vollständigen Katalog finden Sie unter references/input-action-reference.md. Informationen zu UI-Eingabemodi finden Sie unter ue-ui-umg-slate.
