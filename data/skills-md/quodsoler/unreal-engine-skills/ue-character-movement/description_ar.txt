استخدم هذه المهارة عند العمل مع حركة الشخصية، أو CharacterMovementComponent، أو CMC، أو أوضاع الحركة، أو المشي، أو السقوط، أو السباحة، أو الطيران، أو الحركة المخصصة، أو التنبؤ بالشبكة، أو FSavedMove، أو حركة الجذر، أو اكتشاف الأرضية، أو التقدم، أو فيزياء الشخصية. استخدم أيضًا لـ "PhysWalking" أو "PhysCustom" أو "LaunchCharacter" أو "WalkableFloor" أو "النسخ المتماثل للحركة". راجع المراجع/ للحصول على أنماط تمديد CMC وتفاصيل خطوط أنابيب الحركة.
