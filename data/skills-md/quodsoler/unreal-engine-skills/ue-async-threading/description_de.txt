Verwenden Sie diese Fähigkeit, wenn Sie mit asynchronen Vorgängen, Threading, paralleler Ausführung oder Parallelität der Unreal Engine arbeiten. Wird auch verwendet, wenn der Benutzer „FRunnable“, „FAsyncTask“, „TaskGraph“, „UE::Tasks“, „ParallelFor“, „TFuture“, „TPromise“, „Async()“, „Thread Safety“, „FCriticalSection“, „FRWLock“, „Background Thread“, „Game Thread Dispatch“ oder „Thread Pool“ erwähnt. Informationen zu asynchronen Netzwerken (RPCs, Replikation) finden Sie unter ue-networking-replication. Informationen zum Asset-Streaming finden Sie unter ue-data-assets-tables.
