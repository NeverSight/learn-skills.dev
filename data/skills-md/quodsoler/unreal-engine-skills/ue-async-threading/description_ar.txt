استخدم هذه المهارة عند العمل مع عمليات المزامنة أو الترابط أو التنفيذ المتوازي أو التزامن في Unreal Engine. يُستخدم أيضًا عندما يذكر المستخدم "FRunnable"، أو "FAsyncTask"، أو "TaskGraph"، أو "UE::Tasks"، أو "ParallelFor"، أو "TFuture"، أو "TPromise"، أو "Async()"، أو "سلامة سلسلة المحادثات"، أو "FCriticalSection"، أو "FRWLock"، أو "سلسلة محادثات الخلفية"، أو "إرسال سلسلة رسائل اللعبة"، أو "تجمع مؤشرات الترابط". بالنسبة لمزامنة الشبكات (RPCs، النسخ المتماثل)، راجع ue-networking-replication. بالنسبة لتدفق الأصول، راجع جداول أصول بيانات ue.
