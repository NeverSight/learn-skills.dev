Verwenden Sie diese Fähigkeit, wenn Sie mit den Gameplay-Framework-Klassen der Unreal Engine arbeiten: GameMode, GameState, PlayerController, PlayerState, Pawn, Character oder GameInstance. Wird auch verwendet, wenn der Benutzer „Gameplay-Framework“, „Spielregeln“, „Spielerverwaltung“, „Spielablauf“ oder „Spieler-Spawnen“ erwähnt. Die vollständige Autoritäts-/Präsenzmatrix finden Sie unter references/framework-class-map.md. Informationen zu Netzwerk/Replikation finden Sie unter ue-networking-replication. Informationen zur Eingabeeinrichtung finden Sie unter ue-input-system.
