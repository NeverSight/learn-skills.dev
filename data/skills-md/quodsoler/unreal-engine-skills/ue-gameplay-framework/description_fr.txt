Utilisez cette compétence lorsque vous travaillez avec les classes du framework de gameplay d'Unreal Engine : GameMode, GameState, PlayerController, PlayerState, Pawn, Character ou GameInstance. À utiliser également lorsque l'utilisateur mentionne « cadre de jeu », « règles du jeu », « gestion des joueurs », « déroulement du match » ou « reproduction des joueurs ». Voir references/framework-class-map.md pour la matrice complète d'autorité/présence. Pour la mise en réseau/réplication, voir ue-networking-replication. Pour la configuration des entrées, voir ue-input-system.
