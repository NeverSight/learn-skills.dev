Когда пользователь хочет создать или обновить контекстный документ своего проекта Unreal Engine. Используйте, когда пользователь говорит «контекст проекта», «настроить контекст», «контекст UE», «настроить проект» или хочет избежать повторения настройки проекта в задачах разработки UE. Создает `.agents/ue-project-context.md`, на который ссылаются все остальные навыки UE. См. нижний колонтитул по соответствующим навыкам, чтобы узнать о навыках, которые зависят от этого контекста.
