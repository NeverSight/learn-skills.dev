يُستخدم عند كتابة اختبارات الأتمتة أو الاختبارات الوظيفية أو أي اختبار في Unreal Engine. يُستخدم أيضًا عندما يسأل المستخدم عن "UE_LOG"، أو التسجيل، أو فئات السجل، أو التأكيد، أو التحقق، أو التأكد، أو التحقق، أو DrawDebug، أو رسم التصحيح، أو أمر وحدة التحكم، أو التوصيف، أو Unreal Insights، أو أوامر stat، أو تقنيات تصحيح الأخطاء. راجع ue-module-build-system لإعداد وحدة الاختبار، وأسس ue-cpp للتعرف على أنماط تسجيل C++ العامة.
