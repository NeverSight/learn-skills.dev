دليل لتشغيل أحمال عمل ML/AI على مجموعات HPC التابعة لـ Alliance Canada (المعروفة سابقًا باسم Compute Canada). استخدم هذه المهارة عندما يذكر المستخدم Alliance Canada، أو Compute Canada، أو Narval، أو Cedar، أو Graham، أو Trillium، أو Niagara، أو Fir، أو Nibi، أو Rorqual، أو Killarney، أو تقديم مهمة Slurm على مجموعات كندية، أو sbatch للتدريب على تعلم الآلة، أو وظائف GPU على HPC، أو virtualenv على المجموعات، أو أي شيء متعلق بتشغيل التعلم العميق على البنية التحتية للحوسبة المشتركة. يُستخدم أيضًا عندما يسأل المستخدم عن CUDA على المجموعات، أو التدريب الموزع باستخدام DeepSpeed/PyTorch DDP على Slurm، أو إدارة مجموعات البيانات على أنظمة الملفات المتوازية، أو HuggingFace على HPC، أو Apache Arrow أو Parquet على المجموعات، أو أفضل ممارسات ML للحوسبة المشتركة، أو Apptainer، أو Singularity، أو الحاويات على HPC، أو vLLM، أو خدمة الاستدلال على HPC، أو استدلال LLM على المجموعات.
