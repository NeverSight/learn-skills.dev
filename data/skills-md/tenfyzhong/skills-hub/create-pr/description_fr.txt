Créez une Pull Request à partir de la branche actuelle. Synchronise le principal/maître en amont, met à jour la branche actuelle, gère intelligemment les télécommandes fork et génère le titre/la description du PR à partir des différences et des validations. Respecte les modèles de relations publiques s'ils sont présents. REQUIS Doit être dans un référentiel git avec gh CLI disponible.
