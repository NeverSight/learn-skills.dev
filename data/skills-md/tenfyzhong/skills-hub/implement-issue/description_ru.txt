Комплексная реализация задачи GitHub. Принимает номер проблемы или URL-адрес, синхронизируется с последней версией main/master, создает ветку функции, анализирует проблему, тщательно изучает базу кода и реализует решение. ТРЕБУЕТСЯ Должен находиться в репозитории git с доступным интерфейсом командной строки gh.
