Compétence complète de révision du code PR pour les référentiels git. À utiliser lors de l'examen d'un PR GitHub/GitLab en fournissant un lien PR. Analyse les modifications par rapport à la base de fusion, explique ce que fait le PR, fournit des directives de révision, identifie les problèmes triés par gravité, évalue la couverture des tests et soulève des questions incertaines. REQUIS Doit être dans un référentiel git avec gh CLI disponible.
