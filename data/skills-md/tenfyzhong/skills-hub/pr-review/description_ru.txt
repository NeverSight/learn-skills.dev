Комплексные навыки проверки PR-кода для репозиториев git. Используйте при просмотре PR GitHub/GitLab, предоставив ссылку на PR. Анализирует изменения по сравнению с базой слияний, объясняет, что делает PR, предоставляет рекомендации по проверке, определяет проблемы, отсортированные по серьезности, оценивает покрытие тестированием и поднимает неопределенные вопросы. ТРЕБУЕТСЯ Должен находиться в репозитории git с доступным интерфейсом командной строки gh.
