Habilidad integral de revisión de código de relaciones públicas para repositorios de git. Úselo al revisar un PR de GitHub/GitLab proporcionando un enlace de PR. Analiza los cambios en comparación con la base de fusión, explica lo que hace el RP, proporciona pautas de revisión, identifica problemas ordenados por gravedad, evalúa la cobertura de las pruebas y plantea preguntas inciertas. REQUIERE Debe estar en un repositorio git con gh CLI disponible.
