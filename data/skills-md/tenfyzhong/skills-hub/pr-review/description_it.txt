Capacità completa di revisione del codice PR per i repository git. Da utilizzare durante la revisione di un PR GitHub/GitLab fornendo un collegamento PR. Analizza le modifiche rispetto alla base di fusione, spiega cosa fa il PR, fornisce linee guida per la revisione, identifica i problemi ordinati per gravità, valuta la copertura dei test e solleva domande incerte. RICHIEDE Deve essere in un repository git con gh CLI disponibile.
