Umfassende Fähigkeiten zur PR-Codeüberprüfung für Git-Repositories. Verwenden Sie diese Option, wenn Sie eine GitHub/GitLab-PR überprüfen, indem Sie einen PR-Link bereitstellen. Analysiert Änderungen anhand der Zusammenführungsbasis, erklärt, was der PR tut, stellt Überprüfungsrichtlinien bereit, identifiziert Probleme nach Schweregrad sortiert, bewertet die Testabdeckung und wirft unsichere Fragen auf. ERFORDERLICH Muss sich in einem Git-Repository mit verfügbarer GH-CLI befinden.
