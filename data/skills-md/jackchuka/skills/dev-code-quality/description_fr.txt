Analyse systématique de la qualité de la base de code pour identifier les opportunités de duplication, de redondance et d’amélioration. À utiliser lors de l'examen de l'architecture d'un dépôt, de la recherche de cibles de refactorisation ou de l'évaluation de l'état du code. Déclencheurs : "analyser le dépôt", "trouver une duplication de code", "suggérer des améliorations", "révision de la qualité du code", "y a-t-il du code redondant", "plan de refactoring", "révision de l'architecture".
