Créez des versions GitHub sur plusieurs référentiels OSS. Vérifie l'état de la version via « gh oss-watch releases », analyse les validations inédites pour recommander des modifications semver, confirme avec l'utilisateur, puis crée des versions. Déclencheurs : "créer des versions", "libérer mes packages", "vérifier les versions", "versions oss", "versions gh oss-watch", "/oss-release", "état de la version", "versions de remplacement"
