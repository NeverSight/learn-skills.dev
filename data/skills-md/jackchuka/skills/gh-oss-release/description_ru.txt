Создавайте выпуски GitHub в нескольких репозиториях OSS. Проверяет статус выпуска с помощью `gh oss-watch Releases`, анализирует невыпущенные коммиты, чтобы рекомендовать повторные изменения, подтверждает это с пользователем, затем создает релизы. Триггеры: «создать выпуски», «выпустить мои пакеты», «проверить наличие выпусков», «выпуски oss», «gh oss-watch Releases», «/oss-release», «статус выпуска», «увеличить версии»
