Esamina e riorganizza le impostazioni delle autorizzazioni di Claude Code in tutti i file di configurazione (impostazioni globali.json, impostazioni del progetto.local.json, copie dotfiles). Identifica ridondanza, autorizzazioni fuori posto e mancanza di organizzazione di lettura/scrittura. Produce un layout pulito in cui le impostazioni globali sono la fonte della verità e i file locali del progetto contengono solo sostituzioni specifiche del progetto. Utilizza questa competenza ogni volta che l'utente menziona la revisione delle autorizzazioni, la pulizia delle impostazioni, il controllo degli strumenti consentiti, la riorganizzazione di settings.json o la domanda "quali autorizzazioni ho". Da utilizzare anche quando si aggiungono nuovi server o strumenti MCP e si desidera decidere cosa pre-autorizzare. Trigger: "permessi di revisione", "impostazioni di controllo", "pulizia settings.json", "controllo delle autorizzazioni", "/permissions-audit".
