قم بمراجعة وإعادة تنظيم إعدادات أذونات Claude Code عبر جميع ملفات التكوين (الإعدادات العامة.json، إعدادات المشروع.local.json، نسخ dotfiles). يحدد التكرار والأذونات في غير محلها ونقص تنظيم القراءة/الكتابة. يُنتج تخطيطًا نظيفًا حيث تكون الإعدادات العامة هي مصدر الحقيقة وتحتوي ملفات المشروع المحلية فقط على تجاوزات خاصة بالمشروع. استخدم هذه المهارة عندما يذكر المستخدم مراجعة الأذونات، أو تنظيف الإعدادات، أو تدقيق الأدوات المسموح بها، أو إعادة تنظيم settings.json، أو السؤال "ما هي الأذونات التي أملكها". استخدمه أيضًا عند إضافة خوادم أو أدوات MCP جديدة والرغبة في تحديد ما سيتم السماح به مسبقًا. المشغلات: "مراجعة الأذونات"، "إعدادات التدقيق"، "تنظيف settings.json"، "تدقيق الأذونات"، "/permissions-audit".
