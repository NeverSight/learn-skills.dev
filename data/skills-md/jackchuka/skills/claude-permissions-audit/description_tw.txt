檢查並重新組織所有設定檔（全域 settings.json、專案 settings.local.json、dotfiles 副本）中的 Claude Code 權限設定。識別冗餘、錯誤的權限以及缺乏讀取/寫入組織。產生一個乾淨的佈局，其中全域設定是事實來源，專案本機檔案僅包含專案特定的覆蓋。每當使用者提到審查權限、清理設定、審核允許的工具、重新組織 settings.json 或詢問「我有什麼權限」時，請使用此技能。在新增的 MCP 伺服器或工具並想要決定預先允許哪些內容時也可使用。觸發器：「審查權限」、「審核設定」、「清理settings.json」、「權限審核」、「/permissions-audit」。
