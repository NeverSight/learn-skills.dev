Crea commit git seguendo la specifica Conventional Commits v1.0.0 (conventionalcommits.org). Da utilizzare quando l'utente chiede di confermare le modifiche, dice "/conventional-commit" o desidera un messaggio di commit ben strutturato. Si attiva su richieste come "commit this", "commit my change", "crea un commit" o qualsiasi flusso di lavoro git commit. Analizza le modifiche graduali/non graduali e produce messaggi di commit conformi con tipo, ambito, descrizione, corpo e piè di pagina corretti.
