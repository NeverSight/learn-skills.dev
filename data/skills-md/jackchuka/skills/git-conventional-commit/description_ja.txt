Conventional Commits v1.0.0 仕様 (conventionalcommits.org) に従って git コミットを作成します。ユーザーが変更のコミットを要求する場合、「/conventional-commit」と言う場合、または適切に構造化されたコミット メッセージが必要な場合に使用します。 「これをコミット」、「変更をコミット」、「コミットを作成」、または任意の git commit ワークフローなどのリクエストでトリガーされます。ステージングされた/ステージングされていない変更を分析し、適切なタイプ、スコープ、説明、本文、およびフッターを含む準拠したコミット メッセージを生成します。
