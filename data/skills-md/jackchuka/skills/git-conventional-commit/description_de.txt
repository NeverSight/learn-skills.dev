Erstellen Sie Git-Commits gemäß der Spezifikation „Conventional Commits v1.0.0“ (conventionalcommits.org). Verwenden Sie diese Option, wenn der Benutzer nach einem Commit für Änderungen fragt, „/conventional-commit“ sagt oder eine gut strukturierte Commit-Nachricht wünscht. Wird bei Anfragen wie „dieses festschreiben“, „meine Änderungen festschreiben“, „einen Commit erstellen“ oder einem beliebigen Git-Commit-Workflow ausgelöst. Analysiert bereitgestellte/nicht bereitgestellte Änderungen und erstellt konforme Commit-Nachrichten mit dem richtigen Typ, Umfang, Beschreibung, Text und Fußzeilen.
