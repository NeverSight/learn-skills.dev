Cree confirmaciones de git siguiendo la especificación de confirmaciones convencionales v1.0.0 (conventionalcommits.org). Úselo cuando el usuario solicite confirmar cambios, diga "/conventional-commit" o quiera un mensaje de confirmación bien estructurado. Activadores de solicitudes como "confirmar esto", "confirmar mis cambios", "crear una confirmación" o cualquier flujo de trabajo de confirmación de git. Analiza los cambios preparados/no preparados y produce mensajes de confirmación compatibles con el tipo, alcance, descripción, cuerpo y pies de página adecuados.
