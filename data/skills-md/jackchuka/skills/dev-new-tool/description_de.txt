End-to-End-Workflow, um eine neue Tool-Idee von der Forschung zum funktionierenden MVP zu bringen. Verwenden Sie diese Option, wenn der Benutzer eine Idee für ein CLI-Tool, eine Bibliothek oder ein kleines Projekt hat und vom Konzept zur ersten Implementierung übergehen möchte. Löst aus bei „Ich habe eine Idee für ein Tool“, „Erstellen Sie eine neue CLI“, „Erstellen Sie ein neues Projekt“, „Starten Sie ein neues Tool“, „Ich möchte erstellen“, „Lassen Sie uns ein neues erstellen“, „Erstellen Sie eine neue CLI“, „CLI erstellen“.
