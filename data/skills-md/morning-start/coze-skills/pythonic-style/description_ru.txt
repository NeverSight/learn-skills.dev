Оптимизация стиля кода Python и руководство по идиомам Pythonic; на основе полного содержания «One Python Craftsman» и концепции «Friendly Python», охватывающей именование переменных, поток управления, типы данных, типы контейнеров, проектирование функций, обработку исключений, декораторы, файловые операции и принципы SOLID; предоставление удобных для пользователя и удобных в обслуживании шаблонов проектирования, контрольных списков для проверки и более 140 практических шаблонов.
