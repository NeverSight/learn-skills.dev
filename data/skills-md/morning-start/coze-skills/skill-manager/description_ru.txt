Расширенная система управления навыками, поддерживающая управление полным жизненным циклом навыков, стандарты документации, интеграцию/разделение навыков и оркестровку рабочих процессов; предоставляет простые/сложные стандарты документации по навыкам для обеспечения качества и удобства обслуживания.
