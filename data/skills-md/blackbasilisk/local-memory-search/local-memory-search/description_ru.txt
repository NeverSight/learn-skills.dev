Локальный автономный семантический поиск по файлам памяти OpenClaw (MEMORY.md и Memory/*.md) с использованием вложений Python + FAISS (без онлайн-LLM/API). Используйте, когда Дэйв запрашивает «поиск в памяти» без оплаты облачных вычислений, хочет семантический вызов по заметкам или хочет поделиться многоразовым навыком семантического поиска в локальной памяти.
