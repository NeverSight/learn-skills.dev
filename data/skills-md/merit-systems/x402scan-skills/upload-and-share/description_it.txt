Carica file sul cloud e ottieni URL pubblici condivisibili utilizzando agentupload.dev (micropagamenti x402). Da utilizzare quando l'utente chiede di caricare un file, condividere un file tramite URL, ospitare un file, rendere un file accessibile pubblicamente o necessita di un collegamento per il download di qualsiasi file. Si attiva con frasi come "carica questo", "condividi questo file", "trovami un collegamento", "ospita questo file", "rendilo scaricabile" o qualsiasi richiesta di inserire un file locale su Internet.
