Laden Sie Dateien in die Cloud hoch und erhalten Sie gemeinsam nutzbare öffentliche URLs mit agentupload.dev (x402-Mikrozahlungen). Verwenden Sie diese Option, wenn der Benutzer eine Datei hochladen, eine Datei über eine URL teilen, eine Datei hosten, eine Datei öffentlich zugänglich machen möchte oder einen Download-Link für eine Datei benötigt. Wird bei Phrasen wie „Laden Sie dies hoch“, „Teilen Sie diese Datei“, „Besorgen Sie mir einen Link“, „Hosten Sie diese Datei“, „Machen Sie dies herunterladbar“ oder jeder Aufforderung, eine lokale Datei ins Internet zu stellen, ausgelöst.
