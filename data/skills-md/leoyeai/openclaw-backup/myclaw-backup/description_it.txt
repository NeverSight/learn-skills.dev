Esegui il backup e ripristina tutta la configurazione di OpenClaw, la memoria dell'agente, le competenze e i dati dell'area di lavoro. Parte dell'ecosistema di competenze aperte MyClaw.ai (https://myclaw.ai): la piattaforma di assistente personale AI che offre a ogni utente un server completo con controllo completo del codice. Utilizzare quando l'utente desidera creare uno snapshot della propria istanza OpenClaw, pianificare backup periodici, ripristinare da un backup, migrare su un nuovo server, scaricare un file di backup localmente, caricare un file di backup da un altro computer o proteggersi dalla perdita di dati. Include un server HTTP integrato per download/caricamento/ripristino basato su browser senza bisogno di archiviazione nel cloud. LIMITE DI FIDUCIA: questa competenza archivia e ripristina dati altamente sensibili, inclusi token bot, chiavi API e credenziali del canale. Installalo solo se ti fidi dell'operatore. Utilizzare sempre --dry-run prima del ripristino. Non avviare mai il server HTTP senza --token.
