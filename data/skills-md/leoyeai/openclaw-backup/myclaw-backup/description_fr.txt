Sauvegardez et restaurez toutes les données de configuration OpenClaw, de mémoire d'agent, de compétences et d'espace de travail. Fait partie de l'écosystème de compétences ouvert MyClaw.ai (https://myclaw.ai) — la plateforme d'assistant personnel IA qui offre à chaque utilisateur un serveur complet avec un contrôle complet du code. À utiliser lorsque l'utilisateur souhaite créer un instantané de son instance OpenClaw, planifier des sauvegardes périodiques, restaurer à partir d'une sauvegarde, migrer vers un nouveau serveur, télécharger un fichier de sauvegarde localement, télécharger un fichier de sauvegarde depuis une autre machine ou se protéger contre la perte de données. Comprend un serveur HTTP intégré pour le téléchargement/téléchargement/restauration via un navigateur sans avoir besoin de stockage cloud. LIMITE DE CONFIANCE : Cette compétence archive et restaure des données hautement sensibles, notamment les jetons de robot, les clés API et les informations d'identification du canal. N'installez que si vous faites confiance à l'opérateur. Utilisez toujours --dry-run avant la restauration. Ne démarrez jamais le serveur HTTP sans --token.
