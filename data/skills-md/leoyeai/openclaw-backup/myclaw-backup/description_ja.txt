すべての OpenClaw 設定、エージェント メモリ、スキル、およびワークスペース データをバックアップおよび復元します。 MyClaw.ai (https://myclaw.ai) オープン スキル エコシステムの一部です。これは、すべてのユーザーに完全なコード制御を備えた完全なサーバーを提供する AI パーソナル アシスタント プラットフォームです。ユーザーが OpenClaw インスタンスのスナップショットの作成、定期的なバックアップのスケジュール、バックアップからの復元、新しいサーバーへの移行、バックアップ ファイルのローカル ダウンロード、別のマシンからのバックアップ ファイルのアップロード、またはデータ損失からの保護を行う場合に使用します。クラウド ストレージを必要とせず、ブラウザベースのダウンロード/アップロード/復元を行うための HTTP サーバーが組み込まれています。信頼境界: このスキルは、ボット トークン、API キー、チャネル認証情報などの機密性の高いデータをアーカイブおよび復元します。オペレーターを信頼できる場合にのみインストールしてください。復元する前に必ず --dry-run を使用してください。 --token を指定せずに HTTP サーバーを起動しないでください。
