Sichern und wiederherstellen Sie alle OpenClaw-Konfigurations-, Agentenspeicher-, Fertigkeiten- und Arbeitsbereichsdaten. Teil des offenen Kompetenz-Ökosystems MyClaw.ai (https://myclaw.ai) – der KI-Plattform für persönliche Assistenten, die jedem Benutzer einen vollständigen Server mit vollständiger Codekontrolle bietet. Verwenden Sie diese Option, wenn der Benutzer einen Snapshot seiner OpenClaw-Instanz erstellen, regelmäßige Backups planen, aus einem Backup wiederherstellen, auf einen neuen Server migrieren, eine Backup-Datei lokal herunterladen, eine Backup-Datei von einem anderen Computer hochladen oder sich vor Datenverlust schützen möchte. Enthält einen integrierten HTTP-Server für browserbasiertes Herunterladen/Hochladen/Wiederherstellen, ohne dass Cloud-Speicher erforderlich ist. VERTRAUENSGRENZE: Dieser Skill archiviert und stellt hochsensible Daten einschließlich Bot-Tokens, API-Schlüssel und Kanalanmeldeinformationen wieder her. Installieren Sie nur, wenn Sie dem Betreiber vertrauen. Verwenden Sie vor der Wiederherstellung immer --dry-run. Starten Sie den HTTP-Server niemals ohne --token.
