Сделайте резервную копию и восстановите всю конфигурацию OpenClaw, память агента, навыки и данные рабочего пространства. Часть экосистемы открытых навыков MyClaw.ai (https://myclaw.ai) — платформы личного помощника с искусственным интеллектом, которая предоставляет каждому пользователю полноценный сервер с полным контролем над кодом. Используйте, когда пользователь хочет создать снимок своего экземпляра OpenClaw, запланировать периодическое резервное копирование, восстановить из резервной копии, перейти на новый сервер, загрузить файл резервной копии локально, загрузить файл резервной копии с другого компьютера или защитить от потери данных. Включает встроенный HTTP-сервер для загрузки/выгрузки/восстановления через браузер без использования облачного хранилища. ГРАНИЦА ДОВЕРИЯ: этот навык архивирует и восстанавливает высококонфиденциальные данные, включая токены ботов, ключи API и учетные данные канала. Устанавливайте только в том случае, если вы доверяете оператору. Всегда используйте --dry-run перед восстановлением. Никогда не запускайте HTTP-сервер без --token.
