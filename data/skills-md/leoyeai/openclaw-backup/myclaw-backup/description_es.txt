Realice una copia de seguridad y restaure toda la configuración de OpenClaw, la memoria del agente, las habilidades y los datos del espacio de trabajo. Parte del ecosistema de habilidades abiertas MyClaw.ai (https://myclaw.ai): la plataforma de asistente personal de IA que brinda a cada usuario un servidor completo con control total del código. Úselo cuando el usuario desee crear una instantánea de su instancia de OpenClaw, programar copias de seguridad periódicas, restaurar desde una copia de seguridad, migrar a un nuevo servidor, descargar un archivo de copia de seguridad localmente, cargar un archivo de copia de seguridad desde otra máquina o proteger contra la pérdida de datos. Incluye un servidor HTTP integrado para descarga/carga/restauración basada en navegador sin necesidad de almacenamiento en la nube. LÍMITE DE CONFIANZA: esta habilidad archiva y restaura datos altamente confidenciales, incluidos tokens de bot, claves API y credenciales de canal. Instálelo solo si confía en el operador. Utilice siempre --dry-run antes de restaurar. Nunca inicie el servidor HTTP sin un --token.
