Esamina le richieste pull di GitHub utilizzando l'estensione gh-pr-review. Da utilizzare quando viene richiesto di rivedere un PR, aggiungere commenti di revisione in linea, richiedere modifiche, approvare o commentare una richiesta pull. Gestisce l'intero ciclo di vita della revisione: avvia, aggiungi commenti in linea, visualizza l'anteprima e invia.
