Erstellen oder aktualisieren Sie GitHub-Pull-Requests mithilfe des für das Repository erforderlichen Workflows und der Vorlagenkonformität. Wird verwendet, wenn Sie aufgefordert werden, eine PR zu erstellen/öffnen/aktualisieren, damit der Assistent „.github/pull_request_template.md“ liest, jeden Vorlagenabschnitt ausfüllt, die Markdown-Struktur genau beibehält und fehlende Daten als „N/A“ oder „Keine“ markiert, anstatt Abschnitte zu überspringen.
