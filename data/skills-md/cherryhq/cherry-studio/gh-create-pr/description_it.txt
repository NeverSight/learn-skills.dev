Crea o aggiorna le richieste pull di GitHub utilizzando il flusso di lavoro richiesto dal repository e la conformità del modello. Da utilizzare quando viene richiesto di creare/aprire/aggiornare un PR in modo che l'assistente legga `.github/pull_request_template.md`, riempia ogni sezione del modello, preservi esattamente la struttura di markdown e contrassegni i dati mancanti come N/A o Nessuno invece di saltare le sezioni.
