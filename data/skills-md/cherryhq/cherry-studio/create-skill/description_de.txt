Erstellen Sie einen neuen Skill im aktuellen Repository. Verwenden Sie diese Option, wenn der Benutzer einen neuen Skill erstellen/hinzufügen möchte oder erwähnt, dass er einen Skill von Grund auf neu erstellen möchte. Dieser Skill folgt dem in .agents/skills/README.md definierten Workflow und hilft beim Aufbau, der Validierung und Synchronisierung neuer Skills.
