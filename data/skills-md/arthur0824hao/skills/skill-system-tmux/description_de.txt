Tmux-Ausführungsunterstützung für lang laufende und persistente Befehle. Laden Sie diesen Skill, wenn Sie: (1) Befehle ausführen müssen, von denen erwartet wird, dass sie das Tool-Timeout (über 60 Sekunden) überschreiten, (2) persistente Server oder Entwicklungsprozesse starten, (3) TUI/interaktive Anwendungen ausführen, (4) parallele isolierte Aufgaben in separaten Sitzungen ausführen, (5) Remote-Befehle über SSH ausführen, die die Trennung überstehen müssen. Auslöserphrasen: „im Hintergrund ausführen“, „Server starten“, „lange Laufzeit“, „tmux-Sitzung“, „Weiterlaufen“, „persistenter Prozess“, „Entwicklungsserver“, „Trainingsskript“, „Git Clone Large Repo“, „Docker Build“.
