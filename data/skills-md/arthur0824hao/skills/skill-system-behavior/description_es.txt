Motor de especificación de comportamiento estilo BDD para el sistema de habilidades. Úselo cuando: (1) defina el comportamiento de una nueva habilidad antes de la implementación, (2) valide una especificación con respecto al esquema, (3) genere un contrato de comportamiento (Mermaid DAG) a partir de una especificación, (4) ejecute pruebas de aceptación estructural con respecto a una habilidad creada. Flujo de trabajo: Especificaciones → Prueba → Desarrollar → Contrato.
