Механизм спецификации поведения в стиле BDD для системы навыков. Используйте, когда: (1) определяете поведение нового навыка перед реализацией, (2) проверяете спецификацию на соответствие схеме, (3) создаете контракт поведения (Mermaid DAG) на основе спецификации, (4) выполняете структурные приемочные тесты для встроенного навыка. Рабочий процесс: Спецификация → Тестирование → Разработка → Контракт.
