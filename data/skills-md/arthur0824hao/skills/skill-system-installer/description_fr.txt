Installez et mettez à jour les compétences dans le système de compétences à partir d'une liste organisée ou d'un chemin de dépôt GitHub. À utiliser lorsqu'un utilisateur demande de répertorier les compétences installables, d'installer une compétence organisée, de mettre à jour les compétences existantes ou d'installer une compétence à partir d'un autre dépôt (y compris les dépôts privés).
