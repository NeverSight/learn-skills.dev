Installieren und aktualisieren Sie Fertigkeiten aus einer kuratierten Liste oder einem GitHub-Repo-Pfad im Fertigkeitssystem. Verwenden Sie diese Option, wenn ein Benutzer darum bittet, installierbare Fertigkeiten aufzulisten, eine kuratierte Fertigkeit zu installieren, vorhandene Fertigkeiten zu aktualisieren oder eine Fertigkeit aus einem anderen Repository (einschließlich privater Repositorys) zu installieren.
