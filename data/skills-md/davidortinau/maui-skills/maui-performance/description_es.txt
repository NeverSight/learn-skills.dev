Guía de optimización del rendimiento para aplicaciones .NET MAUI que cubren creación de perfiles, enlaces compilados, eficiencia del diseño, optimización de imágenes, diccionarios de recursos, tiempo de inicio, recorte y configuración NativeAOT. USO PARA: "optimización del rendimiento", "inicio lento", "rendimiento de la aplicación", "enlaces compilados", "optimización del diseño", "optimización de la imagen", "recorte", "NativeAOT", "creación de perfiles MAUI", "reducir el tamaño de la aplicación", "tiempo de inicio". NO UTILIZAR PARA: sintaxis de enlace de datos (use maui-data-binding), migración de API obsoleta (use maui-current-apis) o configuración de pruebas unitarias (use maui-unit-testing).
