プロファイリング、コンパイルされたバインディング、レイアウト効率、画像の最適化、リソース ディクショナリ、起動時間、トリミング、NativeAOT 構成をカバーする .NET MAUI アプリのパフォーマンス最適化ガイダンス。用途: 「パフォーマンスの最適化」、「起動が遅い」、「アプリのパフォーマンス」、「コンパイルされたバインディング」、「レイアウトの最適化」、「画像の最適化」、「トリミング」、「NativeAOT」、「MAUI のプロファイリング」、「アプリのサイズの削減」、「起動時間」。次の場合には使用しないでください: データ バインディング構文 (maui-data-binding を使用)、非推奨の API 移行 (maui-current-apis を使用)、単体テスト設定 (maui-unit-testing を使用)。
