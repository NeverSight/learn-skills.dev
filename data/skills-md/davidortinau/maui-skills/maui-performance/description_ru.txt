Руководство по оптимизации производительности для приложений .NET MAUI, охватывающее профилирование, скомпилированные привязки, эффективность макета, оптимизацию изображений, словари ресурсов, время запуска, обрезку и конфигурацию NativeAOT. ИСПОЛЬЗОВАНИЕ ДЛЯ: «оптимизация производительности», «медленный запуск», «производительность приложения», «скомпилированные привязки», «оптимизация макета», «оптимизация изображения», «обрезка», «NativeAOT», «профилирование MAUI», «уменьшение размера приложения», «время запуска». НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: синтаксиса привязки данных (используйте maui-data-binding), устаревшей миграции API (используйте maui-current-apis) или настройки модульного тестирования (используйте maui-unit-testing).
