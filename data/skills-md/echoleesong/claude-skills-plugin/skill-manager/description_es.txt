Gestor del ciclo de vida de habilidades. Se utiliza para escanear por lotes el directorio de habilidades, verificar las actualizaciones de GitHub, realizar actualizaciones de versión, administrar el inventario de habilidades y administrar la capa de experiencia (promoción/extracción). Utilice esta herramienta cuando los usuarios necesiten "buscar actualizaciones", "enumerar habilidades", "eliminar una habilidad", "administrar habilidades", "promover experiencia" o "extraer experiencia".
