Responsabile del ciclo di vita delle competenze. Utilizzato per la scansione in batch della directory delle competenze, il controllo degli aggiornamenti di GitHub, l'esecuzione degli aggiornamenti della versione, la gestione dell'inventario delle competenze e la gestione del livello di esperienza (promozione/estrazione). Utilizza questo strumento quando l'utente deve "controllare gli aggiornamenti", "elencare le competenze", "eliminare una competenza", "gestire le competenze", "promuovere l'esperienza" o "promuovere l'esperienza".
