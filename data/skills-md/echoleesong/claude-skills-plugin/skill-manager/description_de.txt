Skill-Lifecycle-Manager. Wird zum Batch-Scannen des Skills-Verzeichnisses, zum Überprüfen von GitHub-Updates, zum Durchführen von Versions-Upgrades, zum Verwalten des Skills-Inventars und zum Verwalten der Erfahrungsschicht (Heraufstufen/Pull) verwendet. Verwenden Sie dieses Tool, wenn der Benutzer „nach Updates suchen“, „Skills auflisten“, „einen Skill löschen“, „Skills verwalten“, „Erfahrung fördern“ oder „Erfahrung abrufen“ muss.
