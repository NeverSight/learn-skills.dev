Менеджер жизненного цикла навыков. Используется для пакетного сканирования каталога навыков, проверки обновлений GitHub, выполнения обновлений версий, управления инвентаризацией навыков и управления уровнем опыта (продвижение/извлечение). Используйте этот инструмент, когда пользователям нужно «проверить наличие обновлений», «перечислить навыки», «удалить навык», «управлять навыками», «продвигать опыт» или «извлекать опыт».
