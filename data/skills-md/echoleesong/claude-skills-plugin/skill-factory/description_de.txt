Automatisierte Fabrik, die GitHub-Repositories in standardisierte KI-Fähigkeiten umwandelt. Verwenden Sie dieses Tool, wenn ein Benutzer eine GitHub-URL angibt und einen Skill „verpacken“, „verpacken“ oder „erstellen“ möchte. Es ruft automatisch Repository-Metadaten ab, generiert eine Standardverzeichnisstruktur und fügt die für die Lebenszyklusverwaltung erforderlichen Erweiterungsmetadaten ein.
