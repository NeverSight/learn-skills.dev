Usine automatisée qui transforme les référentiels GitHub en compétences d'IA standardisées. Utilisez cet outil lorsqu'un utilisateur fournit une URL GitHub et souhaite « empaqueter », « envelopper » ou « créer une compétence ». Il récupère automatiquement les métadonnées du référentiel, génère une structure de répertoires standard et injecte les métadonnées d'extension requises pour la gestion du cycle de vie.
