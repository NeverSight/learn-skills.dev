Automated Factory: преобразует репозитории GitHub в стандартизированные навыки искусственного интеллекта. Используйте этот инструмент, когда пользователь предоставляет URL-адрес GitHub и хочет «упаковать», «обернуть» или «создать навык». Он автоматически извлекает метаданные репозитория, генерирует стандартную структуру каталогов и внедряет метаданные расширений, необходимые для управления жизненным циклом.
