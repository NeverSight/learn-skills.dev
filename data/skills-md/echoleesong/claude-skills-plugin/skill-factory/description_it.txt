Fabbrica automatizzata che converte i repository GitHub in competenze AI standardizzate. Utilizza questo strumento quando un utente fornisce un URL GitHub e richiede di "confezionare", "incapsulare" o "creare una competenza". Recupera automaticamente i metadati del repository, genera una struttura di directory standard e inserisce i metadati estesi necessari per la gestione del ciclo di vita.
