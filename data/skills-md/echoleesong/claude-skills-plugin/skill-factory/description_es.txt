Fábrica automatizada que convierte los repositorios de GitHub en habilidades de IA estandarizadas. Utilice esta herramienta cuando un usuario proporcione una URL de GitHub y quiera "empaquetar", "envolver" o "crear una habilidad". Obtiene automáticamente los metadatos del repositorio, genera una estructura de directorio estándar e inyecta los metadatos de extensión necesarios para la gestión del ciclo de vida.
