Aide les utilisateurs à découvrir et à installer les fonctionnalités de l’écosystème de compétences d’agent ouvert. À utiliser lorsque les utilisateurs demandent « comment faire X » pour des tâches spécialisées, demandent « trouver une compétence pour X », souhaitent étendre les capacités de l'agent ou ont besoin d'aide dans des domaines spécifiques (tests, conception, déploiement, etc.).
