خريطة بيئة OpenClaw لوكلاء الترميز (Claude Code، Cursor، Windsurf، وما إلى ذلك). يُستخدم عندما يحتاج وكيل الترميز إلى التنقل أو تعديل تثبيت OpenClaw - حيث توجد التكوينات والسجلات ووظائف cron والجلسات والمهارات ومساحات العمل والمستندات. يتم تشغيل أسئلة مثل "أين يتم تخزين وظائف cron؟"، أو "أين توجد سجلات الجلسة؟"، أو "كيف تعمل مساحة العمل؟"، أو "أين تكوين OpenClaw؟"، أو أي مهمة تتطلب فهم بنية نظام ملفات OpenClaw.
