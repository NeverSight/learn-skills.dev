Carte d'environnement OpenClaw pour les agents de codage (Claude Code, Cursor, Windsurf, etc.). À utiliser lorsqu'un agent de codage doit naviguer ou modifier une installation OpenClaw — où se trouvent la configuration, les journaux, les tâches cron, les sessions, les compétences, les espaces de travail et les documents. Se déclenche sur des questions telles que « où sont stockées les tâches cron ? », « où sont les journaux de session ? », « comment fonctionne l'espace de travail ? », « où est la configuration OpenClaw ? » ou toute tâche nécessitant une compréhension de la structure du système de fichiers OpenClaw.
