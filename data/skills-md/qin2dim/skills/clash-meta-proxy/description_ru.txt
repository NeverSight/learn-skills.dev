Обновите конфигурацию прокси-сервера Clash-Meta. Используйте при обновлении прокси-узлов для службы clash-meta, редактировании config.yaml с использованием новых прокси из .proxies.yaml или обеспечении согласованности групп прокси. Используйте предоставленный скрипт Python с uv для автоматического резервного копирования, обновления прокси и проверки групп прокси.
