Справочник по парсингу веб-страниц для этого проекта. Обеспечивает выбор сборщика (Fetcher/DynamicFetcher/StealthyFetcher), шаблоны селектора, управление сеансами, обход защиты от ботов, адаптивные/самовосстанавливающиеся селекторы, структуру Spider для сканирования, ротацию прокси, команды CLI и сервер MCP для интеграции ИИ. Используйте при парсинге веб-страниц, работе с защищенными сайтами или построении конвейеров извлечения данных.
