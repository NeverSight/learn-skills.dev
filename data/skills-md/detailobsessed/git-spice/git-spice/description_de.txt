Verwenden Sie git-spice, um gestapelte Branches und GitLab/GitHub-Merge-Anfragen zu verwalten. Behandelt die Einrichtung, den täglichen Arbeitsablauf, Bereitstellungsregeln, MR-Beschreibungen, das Schließen von Problemen, das Ändern und das Zusammenführen von Stapeln von unten nach oben. Wird aufgerufen, wenn mit gestapelten PRs/MRs gearbeitet wird oder wenn der Benutzer gs, git-spice oder gestapelte Branches erwähnt.
