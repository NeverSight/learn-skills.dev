Utilice git-spice para administrar ramas apiladas y solicitudes de fusión de GitLab/GitHub. Cubre la configuración, el flujo de trabajo diario, las reglas de preparación, las descripciones de MR, el cierre de problemas, la modificación y la fusión de pilas de abajo hacia arriba. Invocar cuando se trabaja con PR/MR apilados o cada vez que el usuario menciona gs, git-spice o ramas apiladas.
