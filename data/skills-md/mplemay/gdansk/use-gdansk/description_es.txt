Guía completa de implementación y depuración de Gdansk para las interfaces de usuario de aplicaciones MCP. Úselo al agregar o reparar las interfaces de usuario de las herramientas Amber y FastMCP, conectar `@amber.tool(..., page=...)` a las entradas React `views/apps/**/page.tsx` o `page.jsx`, habilitar SSR, configurar metadatos o `cache_html`, agregar el complemento PostCSS, montar bajo FastAPI o diagnosticar errores de tiempo de ejecución y empaquetado de Gdansk.
