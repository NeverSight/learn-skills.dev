Guida completa all'implementazione e al debug di Danzica per le interfacce utente delle app MCP. Da utilizzare quando si aggiungono o si correggono le UI degli strumenti Amber e FastMCP, si collega `@amber.tool(..., page=...)` a React `views/apps/**/page.tsx` o voci `page.jsx`, si abilita SSR, si configura metadata o `cache_html`, si aggiunge il plugin PostCSS, si monta sotto FastAPI o si diagnosticano errori di runtime e di raggruppamento di Gdansk.
