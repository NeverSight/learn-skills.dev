دليل شامل لتطبيق gdansk وتصحيح الأخطاء لواجهات مستخدم تطبيق MCP. يُستخدم عند إضافة أو إصلاح واجهات مستخدم أداة Amber وFastMCP، أو توصيل `@amber.tool(..., page=...)` إلى React `views/apps/**/page.tsx` أو إدخالات `page.jsx`، أو تمكين SSR، أو تكوين البيانات التعريفية أو `cache_html`، أو إضافة مكون PostCSS الإضافي، أو التثبيت ضمن FastAPI، أو تشخيص أخطاء تجميع gdansk وأخطاء وقت التشغيل.
