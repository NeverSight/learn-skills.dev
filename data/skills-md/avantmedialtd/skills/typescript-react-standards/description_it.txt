Standard di sviluppo supponenti TypeScript e React di Avant Media. Da utilizzare per l'impalcatura di nuovi componenti, la revisione del codice, la scrittura di interfacce o tipi TypeScript, l'impostazione della struttura del progetto, la creazione di hook React o il lavoro su qualsiasi base di codice TypeScript/React. Da utilizzare anche quando l'utente chiede informazioni su best practice, modelli o convenzioni per progetti TypeScript o React, anche se non menzionano esplicitamente gli "standard".
