Genere un gráfico de curva de pago de opciones interactivas con controles de parámetros dinámicos. Utilice esta habilidad siempre que el usuario comparta una captura de pantalla de la posición de las opciones, describa una estrategia de opciones o solicite visualizar cómo una operación de opciones genera o pierde dinero. Los desencadenantes incluyen: cualquier mención de mariposa, extensión (vertical/calendario/diagonal/relación), horcajadas, estrangulamiento, cóndor, llamada cubierta, posición protectora, cóndor de hierro o cualquier estructura de opciones de múltiples patas. También se activa cuando un usuario pega precios de ejercicio, primas, fechas de vencimiento o dice cosas como "muéstrame el pago", "dibuja la curva de pérdidas y ganancias", "cómo se ve esta operación" o carga una captura de pantalla de un corredor (IBKR, TastyTrade, Robinhood, etc.). Utilice siempre esta habilidad incluso si el usuario solo proporciona información parcial: extraiga lo que pueda y use los valores predeterminados para el resto.
