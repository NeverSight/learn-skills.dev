قم بإنشاء مخطط منحنى عائد الخيارات التفاعلية باستخدام عناصر تحكم المعلمات الديناميكية. استخدم هذه المهارة عندما يقوم المستخدم بمشاركة لقطة شاشة لموضع الخيارات، أو يصف استراتيجية الخيارات، أو يطلب تصور كيف تحقق تجارة الخيارات المال أو تخسره. تتضمن المشغلات: أي ذكر للفراشة، أو الانتشار (عمودي/تقويم/قطري/نسبة)، أو امتطاء، أو خنق، أو كوندور، أو نداء مغطى، أو وضع وقائي، أو كوندور حديدي، أو أي هيكل خيارات متعدد الأرجل. يتم تشغيله أيضًا عندما يلصق المستخدم أسعار التنفيذ، أو العلاوات، أو تواريخ انتهاء الصلاحية، أو يقول أشياء مثل "أرني العائد"، أو "ارسم منحنى الربح والخسارة"، أو "كيف تبدو هذه التجارة"، أو يقوم بتحميل لقطة شاشة من وسيط (IBKR، TastyTrade، Robinhood، وما إلى ذلك). استخدم هذه المهارة دائمًا حتى لو كان المستخدم يقدم معلومات جزئية فقط - استخرج ما تستطيع واستخدم الإعدادات الافتراضية للباقي.
