Erstellen Sie ein interaktives Auszahlungskurvendiagramm für Optionen mit dynamischen Parametersteuerungen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer einen Screenshot einer Optionsposition teilt, eine Optionsstrategie beschreibt oder visualisieren möchte, wie ein Optionshandel Geld einbringt oder verliert. Zu den Auslösern gehören: jede Erwähnung von Butterfly, Spread (vertikal/kalender/diagonal/ratio), Straddle, Strangle, Condor, Covered Call, Protective Put, Iron Condor oder einer beliebigen mehrbeinigen Optionsstruktur. Wird auch ausgelöst, wenn ein Benutzer Ausübungspreise, Prämien oder Ablaufdaten einfügt oder Dinge wie „Zeig mir die Auszahlung“, „Zeichne die Gewinn- und Verlustkurve“, „Wie sieht dieser Handel aus“ sagt oder einen Screenshot von einem Broker (IBKR, TastyTrade, Robinhood usw.) hochlädt. Verwenden Sie diese Fähigkeit immer, auch wenn der Benutzer nur Teilinformationen bereitstellt – extrahieren Sie, was Sie können, und verwenden Sie für den Rest die Standardeinstellungen.
