Sistema de diseño y pautas para la interfaz de usuario generativa incorporada de Claude: la herramienta show_widget que presenta widgets HTML/SVG interactivos en línea en las conversaciones de claude.ai. Esta habilidad proporciona el sistema de diseño Anthropic "Imagine" completo para que Claude produzca widgets de alta calidad sin necesidad de llamar primero a read_me. Utilice esta habilidad siempre que el usuario solicite visualizar datos, crear un gráfico interactivo, crear un panel, representar un diagrama, dibujar un diagrama de flujo, mostrar una maqueta, crear una explicación interactiva o producir cualquier contenido visual más allá del texto sin formato o rebajas. Los activadores incluyen: "muéstrame", "visualizar", "dibujar", "gráfico", "panel", "diagrama", "diagrama de flujo", "widget", "interactivo", "maqueta", "ilustrar", "explicar cómo funciona X" (con elementos visuales), o cualquier solicitud de salida visual/interactiva. También se activa cuando el usuario desea mostrar datos financieros visualmente, crear cuadrículas de comparación o crear herramientas con controles deslizantes, alternadores o pantallas de actualización en vivo.
