Designsystem und Richtlinien für Claudes integrierte generative Benutzeroberfläche – das show_widget-Tool, das interaktive HTML/SVG-Widgets inline in claude.ai-Konversationen rendert. Dieser Skill stellt das komplette Anthropic-Designsystem „Imagine“ bereit, sodass Claude hochwertige Widgets erstellt, ohne zuerst read_me aufrufen zu müssen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer Daten visualisieren, ein interaktives Diagramm erstellen, ein Dashboard erstellen, ein Diagramm rendern, ein Flussdiagramm zeichnen, ein Modell zeigen, eine interaktive Erklärung erstellen oder visuelle Inhalte über reinen Text oder Markdown hinaus erstellen möchte. Zu den Auslösern gehören: „zeigen“, „visualisieren“, „zeichnen“, „Diagramm“, „Dashboard“, „Diagramm“, „Flussdiagramm“, „Widget“, „interaktiv“, „Mockup“, „illustrieren“, „erklären, wie X funktioniert“ (mit Visual) oder jede Anfrage nach visueller/interaktiver Ausgabe. Wird auch ausgelöst, wenn der Benutzer Finanzdaten visuell anzeigen, Vergleichsraster erstellen oder Tools mit Schiebereglern, Umschaltern oder Live-Aktualisierungsanzeigen erstellen möchte.
