Sistema di progettazione e linee guida per l'interfaccia utente generativa integrata di Claude: lo strumento show_widget che rende i widget HTML/SVG interattivi in ​​linea nelle conversazioni di claude.ai. Questa competenza fornisce il sistema di progettazione antropico completo "Imagine" in modo che Claude produca widget di alta qualità senza dover prima chiamare read_me. Utilizza questa competenza ogni volta che l'utente chiede di visualizzare dati, creare un grafico interattivo, costruire un dashboard, eseguire il rendering di un diagramma, disegnare un diagramma di flusso, mostrare un modello, creare una spiegazione interattiva o produrre qualsiasi contenuto visivo oltre il semplice testo o markdown. I trigger includono: "mostrami", "visualizza", "disegna", "grafico", "cruscotto", "diagramma", "diagramma di flusso", "widget", "interattivo", "mockup", "illustra", "spiega come funziona X" (con visualizzazione) o qualsiasi richiesta di output visivo/interattivo. Si attiva anche quando l'utente desidera visualizzare visivamente i dati finanziari, creare griglie di confronto o creare strumenti con dispositivi di scorrimento, levette o visualizzazioni di aggiornamento in tempo reale.
