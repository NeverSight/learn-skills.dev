Claude の組み込み生成 UI のシステムとガイドラインを設計します。これは、claude.ai の会話でインタラクティブな HTML/SVG ウィジェットをインラインでレンダリングする show_widget ツールです。このスキルは完全な Anthropic "Imagine" デザイン システムを提供するため、Claude は最初に read_me を呼び出す必要なく高品質のウィジェットを作成できます。このスキルは、ユーザーがデータの視覚化、インタラクティブなチャートの作成、ダッシュボードの構築、図のレンダリング、フローチャートの描画、モックアップの表示、インタラクティブな説明の作成、またはプレーン テキストやマークダウンを超えたビジュアル コンテンツの作成を要求するたびに使用します。トリガーには、「見せる」、「視覚化」、「描画」、「チャート」、「ダッシュボード」、「ダイアグラム」、「フローチャート」、「ウィジェット」、「インタラクティブ」、「モックアップ」、「イラスト」、「X の仕組みの説明」 (ビジュアルを使用)、またはビジュアル/インタラクティブ出力のリクエストが含まれます。また、ユーザーが財務データを視覚的に表示したり、比較グリッドを作成したり、スライダー、トグル、またはライブ更新表示を備えたツールを構築したりする場合にもトリガーされます。
