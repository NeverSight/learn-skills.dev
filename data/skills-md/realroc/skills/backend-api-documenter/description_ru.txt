Автоматически создавайте полную документацию по серверному API в формате AGENTS.md. Используйте, когда пользователь запрашивает: (1) документирование конечных точек внутреннего API, (2) обновление спецификаций внутреннего API после изменений кода, (3) создание или обновление backend/AGENTS.md с полной документацией по API, включая схемы запросов/ответов, бизнес-правила и сведения об аутентификации, (4) создание документации API из файлов маршрутов FastAPI.
