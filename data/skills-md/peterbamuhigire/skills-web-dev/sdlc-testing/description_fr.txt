Générer une documentation de test et de qualité pour les projets SDLC. Conforme à la norme BS ISO/IEC/IEEE 29119-3:2013 (remplace IEEE 829:2008 et BS 7925-2:1998). Couvre le plan de test logiciel, les spécifications des scénarios de test (avec les champs normatifs 29119-3), le plan V&V, le rapport de test de validation, le rapport d'incident, le rapport d'achèvement des tests et les rapports d'examen par les pairs. À utiliser lors de l'établissement d'une stratégie de test, de la création d'une documentation de test ou de la validation de la qualité.
