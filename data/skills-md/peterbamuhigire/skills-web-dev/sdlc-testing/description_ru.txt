Создание документации по тестированию и качеству для проектов SDLC. Соответствует BS ISO/IEC/IEEE 29119-3:2013 (заменяет IEEE 829:2008 и BS 7925-2:1998). Охватывает план тестирования программного обеспечения, спецификации тестовых наборов (с нормативными полями 29119-3), план V&V, отчет о проверочном тестировании, отчет об инцидентах, отчет о завершении тестирования и отчеты о коллегиальной проверке. Используйте при разработке стратегии тестирования, создании тестовой документации или проведении проверки качества.
