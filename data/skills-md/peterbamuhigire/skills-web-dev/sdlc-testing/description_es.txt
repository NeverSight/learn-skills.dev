Generar documentación de pruebas y calidad para proyectos SDLC. Cumple con BS ISO/IEC/IEEE 29119-3:2013 (reemplaza a IEEE 829:2008 y BS 7925-2:1998). Cubre el plan de prueba de software, especificaciones de casos de prueba (con campos normativos 29119-3), plan de V&V, informe de prueba de validación, informe de incidente, informe de finalización de prueba e informes de revisión por pares. Úselo al establecer una estrategia de prueba, crear documentación de prueba o realizar una validación de calidad.
