Охватывает документ проектирования системы (SDD), техническую спецификацию, документ управления интерфейсом (ICD), документ проектирования базы данных, стандарты документации кода и документацию API. Используйте при переводе требований в техническую архитектуру и руководстве командами разработчиков.
