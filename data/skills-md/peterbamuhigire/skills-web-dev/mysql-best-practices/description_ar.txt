أفضل ممارسات MySQL 8.x لتطبيقات SaaS عالية الأداء والآمنة. يُستخدم عند تصميم مخططات قاعدة البيانات، أو كتابة الاستعلامات، أو تحسين الأداء، أو تنفيذ عزل المستأجرين المتعددين، أو تكوين الخوادم، أو إعداد النسخ المتماثل، أو تعزيز الأمان، أو استكشاف أخطاء الاستعلامات البطيئة وإصلاحها. يغطي الفهرسة، وشرح التحليل، والمعاملات، والقفل، وSQL المتقدمة (CTEs، ووظائف النافذة)، والنسخ الاحتياطي، وضبط الخادم.
