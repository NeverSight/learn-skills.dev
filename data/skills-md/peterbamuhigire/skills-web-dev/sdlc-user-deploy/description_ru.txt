Создание документации пользователя и развертывания для проектов SDLC. Включает Руководство пользователя программного обеспечения (SUM), Руководство по эксплуатации/развертыванию, учебные материалы, примечания к выпуску, руководство по техническому обслуживанию и файл README. Используйте при подготовке программного обеспечения для конечных пользователей, системных администраторов и рабочих групп, которые будут использовать, развертывать и обслуживать программное обеспечение.
