Generieren Sie Benutzer- und Bereitstellungsdokumentation für SDLC-Projekte. Umfasst Software-Benutzerhandbuch (SUM), Betriebs-/Bereitstellungshandbuch, Schulungsmaterialien, Versionshinweise, Wartungshandbuch und README-Datei. Zur Verwendung bei der Vorbereitung von Software für Endbenutzer, Systemadministratoren und Betriebsteams, die die Software verwenden, bereitstellen und warten.
