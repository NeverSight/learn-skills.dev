Genera documentazione utente e distribuzione per progetti SDLC. Comprende il Manuale utente del software (SUM), il Manuale operativo/di distribuzione, i materiali di formazione, le note sulla versione, il Manuale di manutenzione e il file README. Da utilizzare durante la preparazione del software per utenti finali, amministratori di sistema e team operativi che utilizzeranno, distribuiranno e manterranno il software.
