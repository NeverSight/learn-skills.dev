Sistema di doppia autenticazione (Sessione + JWT) con controllo degli accessi basato sui ruoli (RBAC) per applicazioni multi-tenant. Da utilizzare quando si implementa l'autenticazione sicura nell'interfaccia utente Web e nei client API/mobili, con autorizzazioni con ambito franchising/tenant. Funziona su più linguaggi: PHP, Node.js, Python, Go, ecc.
