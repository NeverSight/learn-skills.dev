Système de double authentification (Session + JWT) avec contrôle d'accès basé sur les rôles (RBAC) pour les applications multi-tenants. À utiliser lors de la mise en œuvre d'une authentification sécurisée sur l'interface utilisateur Web et les clients API/mobiles, avec des autorisations au niveau de la franchise/du locataire. Fonctionne dans tous les langages : PHP, Node.js, Python, Go, etc.
