Sistema de autenticación dual (Session + JWT) con control de acceso basado en roles (RBAC) para aplicaciones multiinquilino. Úselo al implementar autenticación segura en la interfaz de usuario web y en clientes móviles/API, con permisos específicos de franquicia/inquilino. Funciona en todos los idiomas: PHP, Node.js, Python, Go, etc.
