Система двойной аутентификации (сеанс + JWT) с контролем доступа на основе ролей (RBAC) для мультитенантных приложений. Используйте при реализации безопасной аутентификации через веб-интерфейс и API/мобильные клиенты с разрешениями на уровне франшизы/арендатора. Работает на разных языках: PHP, Node.js, Python, Go и т. д.
