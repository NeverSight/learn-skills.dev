Дополнительные биометрические ворота (отпечаток пальца/лицо) при запуске приложения Android с использованием AndroidX Biometric API. Охватывает утилиту BiometricHelper, интеграцию с заставкой, переключение настроек с проверкой, хранилище EncryptedSharedPreferences и плавный откат. Используйте при добавлении биометрической аутентификации в любое приложение Android.
