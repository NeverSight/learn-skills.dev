Optionales biometrisches Tor (Fingerabdruck/Gesicht) beim Start der Android-App mithilfe der AndroidX Biometric API. Behandelt das BiometricHelper-Dienstprogramm, die Integration des Begrüßungsbildschirms, das Umschalten der Einstellungen mit Überprüfung, den EncryptedSharedPreferences-Speicher und einen eleganten Fallback. Verwenden Sie diese Option, wenn Sie einer Android-App eine biometrische Authentifizierung hinzufügen.
