Проведите комплексный аудит статуса реализации любого программного проекта. Создает структурированную документацию в формате docs/implementation/review-{date}/ с анализом пробелов, аудитом схемы, статусом интеграции, планом завершения и приоритетными действиями. Используйте, когда пользователь хочет точно знать, на каком этапе находится проект и что осталось для его завершения.
