Realice una auditoría integral del estado de implementación de cualquier proyecto de software. Produce documentación estructurada en docs/implementation/review-{date}/ con análisis de brechas, auditoría de esquema, estado de integración, plan de finalización y elementos de acción priorizados. Se utiliza cuando el usuario quiere saber exactamente en qué punto se encuentra un proyecto y qué queda para terminarlo.
