Создание документации по планированию и управлению для проектов SDLC. Охватывает видение и объем проекта, SDP, SCMP, план обеспечения качества, план рисков, SRS и технико-экономическое обоснование. Используйте при запуске нового проекта, управлении проектом или определении базовых показателей планирования до начала разработки.
