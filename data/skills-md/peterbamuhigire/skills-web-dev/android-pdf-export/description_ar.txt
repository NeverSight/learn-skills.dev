نظام تصدير PDF أصلي لنظام Android باستخدام PdfDocument API (بدون تبعيات). مولد يستند إلى Canvas قابل لإعادة الاستخدام مع ترويسة ذات علامة تجارية وجداول بيانات وبطاقات تلخيصية والمشاركة عبر Intent. يُستخدم عند إضافة تصدير PDF إلى أي شاشة تطبيق Android - التقارير أو الفواتير أو طرق عرض التفاصيل أو القوائم.
