Встроенная система экспорта PDF-файлов для Android с использованием PdfDocument API (нулевые зависимости). Многоразовый генератор на основе Canvas с фирменными бланками, таблицами данных, сводными карточками и возможностью поделиться через намерение. Используйте при добавлении экспорта PDF на любой экран приложения Android — отчеты, счета, подробные представления или списки.
