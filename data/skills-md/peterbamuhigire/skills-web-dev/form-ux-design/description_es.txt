Patrones UX/UI multiplataforma para web (Bootstrap 5/Tabler) y Android (Jetpack Compose). Cubre la anatomía del campo, la validación, los estados de error, los asistentes de varios pasos, la accesibilidad, las entradas táctiles y los flujos de trabajo de envío. Úselo al crear cualquier formulario: registro, ingreso de datos, configuración, pago, búsqueda, filtros o formularios clínicos. Complemento para las habilidades de diseño de interfaz de usuario de aplicaciones web, diseño de interfaz de usuario para atención médica, diseño de interfaz de usuario posventa y jetpack-compose-ui.
