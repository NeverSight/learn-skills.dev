Кроссплатформенные шаблоны UX/UI для Интернета (Bootstrap 5/Tabler) и Android (Jetpack Compose). Охватывает анатомию поля, проверку, состояния ошибок, многошаговые мастера, специальные возможности, сенсорный ввод и рабочие процессы отправки. Используйте при построении любой формы — регистрации, ввода данных, настроек, оформления заказа, поиска, фильтров или клинических форм. Сопутствует навыкам проектирования пользовательского интерфейса веб-приложений, дизайна пользовательского интерфейса для здравоохранения, дизайна пользовательского интерфейса для продаж и создания пользовательского интерфейса для реактивных ранцев.
