أنماط UX/UI عبر الأنظمة الأساسية للويب (Bootstrap 5/Tabler) وAndroid (Jetpack Compose). يغطي التشريح الميداني، والتحقق من الصحة، وحالات الخطأ، والمعالجات متعددة الخطوات، وإمكانية الوصول، والمدخلات سهلة اللمس، وسير عمل الإرسال. يُستخدم عند إنشاء أي نموذج — التسجيل أو إدخال البيانات أو الإعدادات أو الخروج أو البحث أو المرشحات أو النماذج السريرية. مصاحب لمهارات تصميم webapp-gui، وتصميم واجهة المستخدم للرعاية الصحية، وتصميم واجهة المستخدم لنقاط البيع، ومهارات jetpack-compose-ui.
