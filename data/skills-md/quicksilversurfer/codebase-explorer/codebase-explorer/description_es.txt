Un método estructurado para explorar y comprender bases de código desconocidas. Utilice esta habilidad siempre que el usuario solicite explorar, comprender, auditar, revisar o navegar por una base de código en la que no ha trabajado antes. También se activa cuando el usuario dice cosas como "ayúdame a comprender este código", "qué hace este repositorio", "guíame a través de este código base", "explora este proyecto", "heredé este código", "incorpórame a este sistema" o pregunta sobre la arquitectura, estructura o estado de un código base existente. Se activa incluso si simplemente dejan caer una ruta de repositorio y dicen "¿qué es esto?". Esta habilidad produce una comprensión estructurada: no resúmenes, sino mapas navegables de cómo funciona realmente el código.
