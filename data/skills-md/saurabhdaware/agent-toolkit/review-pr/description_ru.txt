Комплексный оркестратор PR-обзоров. Извлекает различия GitHub PR через URL-адреса .diff для обобщения изменений кода, запускает проверку CI с использованием навыка review-ci и, когда CI предоставляет URL-адрес предварительного просмотра, запускает review-ui для этого URL-адреса и соответствующих сценариев. Используйте, когда пользователю нужен целостный обзор запроса на извлечение GitHub, включая код, конвейеры CI и (если возможно) развернутый пользовательский интерфейс.
