Orquestador de revisión de relaciones públicas de principio a fin. Obtiene diferencias de relaciones públicas de GitHub a través de URL .diff para resumir los cambios de código, activa la revisión de CI usando la habilidad review-ci y, cuando CI expone una URL de vista previa, ejecuta review-ui contra esa URL y escenarios relevantes. Úselo cuando el usuario desee una revisión integral de una solicitud de extracción de GitHub, incluido el código, las canalizaciones de CI y (si es posible) la interfaz de usuario implementada.
