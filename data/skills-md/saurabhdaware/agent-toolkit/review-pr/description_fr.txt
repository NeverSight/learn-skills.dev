Orchestrateur de revue de relations publiques de bout en bout. Récupère les différences GitHub PR via les URL .diff pour résumer les modifications de code, déclenche la révision de CI à l'aide de la compétence review-ci et, lorsque CI expose une URL d'aperçu, exécute review-ui sur cette URL et les scénarios pertinents. À utiliser lorsque l'utilisateur souhaite un examen holistique d'une pull request GitHub, y compris le code, les pipelines CI et (si possible) l'interface utilisateur déployée.
