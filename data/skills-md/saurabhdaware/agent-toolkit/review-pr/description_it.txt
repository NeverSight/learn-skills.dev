Orchestratore di revisione PR end-to-end. Recupera le differenze PR di GitHub tramite URL .diff per riepilogare le modifiche al codice, attiva la revisione CI utilizzando la capacità review-ci e, quando CI espone un URL di anteprima, esegue review-ui rispetto a tale URL e agli scenari pertinenti. Da utilizzare quando l'utente desidera una revisione olistica di una richiesta pull GitHub che includa codice, pipeline CI e (se possibile) l'interfaccia utente distribuita.
