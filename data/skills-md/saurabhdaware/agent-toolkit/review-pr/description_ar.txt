منسق مراجعة العلاقات العامة من البداية إلى النهاية. جلب اختلافات GitHub PR عبر عناوين URL ذات الامتداد .diff لتلخيص تغييرات التعليمات البرمجية، وتشغيل مراجعة CI باستخدام مهارة review-ci، وعندما يكشف CI عن عنوان URL للمعاينة، يقوم بتشغيل واجهة مستخدم المراجعة مقابل عنوان URL هذا والسيناريوهات ذات الصلة. يُستخدم عندما يريد المستخدم إجراء مراجعة شاملة لطلب سحب GitHub بما في ذلك التعليمات البرمجية وخطوط أنابيب CI و(إن أمكن) واجهة المستخدم المنشورة.
