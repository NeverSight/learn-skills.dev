Déploie des applications sur TrueFoundry. Gère les services HTTP uniques, les travailleurs asynchrones/de file d'attente, les projets multiservices et l'application de manifeste déclaratif. Prend en charge `tfy apply`, `tfy déployer`, la traduction docker-compose et les pipelines CI/CD. À utiliser lors du déploiement d'applications, de l'application de manifestes, de l'expédition de services ou de l'orchestration de déploiements multiservices.
