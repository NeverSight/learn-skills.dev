ينشر التطبيقات إلى TrueFoundry. يتعامل مع خدمات HTTP الفردية، والعمال غير المتزامنين/قائمة الانتظار، والمشاريع متعددة الخدمات، ويتم تطبيق البيان التعريفي. يدعم تطبيق tfy، ونشر tfy، وترجمة إنشاء عامل الإرساء، وخطوط أنابيب CI/CD. يُستخدم عند نشر التطبيقات أو تطبيق البيانات أو خدمات الشحن أو تنسيق عمليات نشر الخدمات المتعددة.
