Déploie les composants d'infrastructure via des graphiques Helm sur TrueFoundry. Prend en charge tout graphique Helm OCI public ou privé, y compris les bases de données (Postgres, MongoDB, Redis), les courtiers de messages (Kafka, RabbitMQ) et les bases de données vectorielles (Qdrant, Milvus). Utilise les manifestes YAML avec « tfy apply ». À utiliser lors de l'installation de graphiques Helm ou du déploiement d'une infrastructure sur TrueFoundry.
