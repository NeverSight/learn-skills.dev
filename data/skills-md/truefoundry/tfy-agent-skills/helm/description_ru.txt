Развертывает компоненты инфраструктуры с помощью диаграмм Helm в TrueFoundry. Поддерживает любые общедоступные и частные диаграммы OCI Helm, включая базы данных (Postgres, MongoDB, Redis), брокеры сообщений (Kafka, RabbitMQ) и векторные базы данных (Qdrant, Milvus). Использует манифесты YAML с tfy apply. Используйте при установке диаграмм Helm или развертывании инфраструктуры в TrueFoundry.
