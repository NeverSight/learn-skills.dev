Платформа пользовательского интерфейса XAF Blazor — настройка BlazorApplication в Program.cs, службы AddXaf/AddXafBlazor, потокобезопасность InvokeAsync (критически важно для сервера Blazor), шаблон действий асинхронного контроллера, редакторы, специфичные для Blazor, встраивание пользовательских компонентов Razor в качестве ViewItems с использованием IComponentContentHolder, взаимодействие JavaScript через IJSRuntime, настройка макета подробного представления (вкладки/группы), программная навигация, обработка ошибок с помощью UserFriendlyException, конфигурация SignalR. Используйте при создании или настройке серверных приложений XAF Blazor.
