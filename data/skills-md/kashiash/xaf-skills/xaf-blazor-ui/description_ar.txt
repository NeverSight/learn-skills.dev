منصة XAF Blazor UI - إعداد BlazorApplication في Program.cs، خدمات AddXaf/AddXafBlazor، أمان سلسلة InvocAsync (أمر بالغ الأهمية لخادم Blazor)، نمط إجراءات وحدة التحكم غير المتزامنة، محررات خاصة بـ Blazor، تضمين مكونات Razor المخصصة كعناصر عرض باستخدام IComponentContentHolder، إمكانية التشغيل المتداخل لـ JavaScript عبر IJSRuntime، تخصيص تخطيط عرض التفاصيل (علامات التبويب/المجموعات)، برمجي التنقل ومعالجة الأخطاء باستخدام UserFriendlyException وتكوين SignalR. يُستخدم عند إنشاء تطبيقات XAF Blazor Server أو تخصيصها.
