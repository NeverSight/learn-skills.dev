Модели бизнес-объектов XAF Entity Framework Core — базовый класс BaseObject, интерфейсы IXafEntityObject/IObjectSpaceLink/IOptimisticLock, виртуальные свойства, настройка DbContext с помощью IDbContextFactory, отношения один-ко-многим/многие-ко-многим/один-к-одному через свойства навигации, миграции EF Core в XAF, вычисляемые столбцы, ObservableCollection. Используйте при определении бизнес-объектов с помощью EF Core ORM в DevExpress XAF.
