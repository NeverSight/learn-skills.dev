Piattaforma UI XAF WinForms: configurazione WinApplication, barra degli strumenti Ribbon o Standard, editor specifici di WinForms (XtraGrid, controlli DevExpress), personalizzazione del layout della vista di dettaglio tramite Layout Manager, controlli WinForms personalizzati incorporati nelle viste XAF, operatori in background per aggiornamenti dell'interfaccia utente thread-safe, personalizzazione della schermata iniziale, navigazione WinForms (NavigationFrame), stampa/anteprima in WinForms, distribuzione ClickOnce/MSI. Da utilizzare durante la creazione o la personalizzazione di applicazioni XAF WinForms.
