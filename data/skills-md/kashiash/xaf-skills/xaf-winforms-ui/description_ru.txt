Платформа пользовательского интерфейса XAF WinForms — настройка WinApplication, панель инструментов «Лента» и «Стандартная», редакторы для WinForms (XtraGrid, элементы управления DevExpress), настройка макета подробного представления с помощью Layout Manager, пользовательские элементы управления WinForms, встроенные в представления XAF, фоновые рабочие процессы для потокобезопасных обновлений пользовательского интерфейса, настройка экрана-заставки, навигация WinForms (NavigationFrame), печать/предварительный просмотр в WinForms, развертывание ClickOnce/MSI. Используйте при создании или настройке приложений XAF WinForms.
