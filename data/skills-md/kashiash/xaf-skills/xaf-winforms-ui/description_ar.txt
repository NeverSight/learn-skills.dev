منصة XAF WinForms UI - إعداد WinApplication، شريط الأدوات Ribbon مقابل Standard، محررات خاصة بـ WinForms (عناصر تحكم XtraGrid، DevExpress)، تخصيص تخطيط عرض التفاصيل عبر Layout Manager، عناصر تحكم WinForms المخصصة المضمنة في طرق عرض XAF، عمال الخلفية لتحديثات واجهة المستخدم الآمنة للخيط، تخصيص شاشة البداية، التنقل WinForms (NavigationFrame)، الطباعة/المعاينة في WinForms، نشر ClickOnce/MSI. يُستخدم عند إنشاء تطبيقات XAF WinForms أو تخصيصها.
