Неперсистентные объекты XAF — атрибут [DomainComponent], базовый класс NonPersistentBaseObject, регистрация AddNonPersistent(), события ObjectsGetting/ObjectByKeyGetting с DynamicCollection, PopupWindowShowAction для диалоговых окон ввода с использованием непостоянных объектов, связывание с постоянными объектами, шаблоны обновления, RemoveFromModifiedObjects, распространенные ошибки. Используйте при создании пользовательских источников данных, оболочек API, форм ввода или вычисляемых представлений без сохранения базы данных в DevExpress XAF.
