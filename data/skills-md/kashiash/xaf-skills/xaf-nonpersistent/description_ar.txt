كائنات XAF NonPersistent - سمة [DomainComponent]، الفئة الأساسية NonPersistentBaseObject، تسجيل AddNonPersistent()، أحداث ObjectsGetting/ObjectByKeyGetting مع DynamicCollection، PopupWindowShowAction لمربعات حوار الإدخال باستخدام الكائنات غير المستمرة، والربط بالكائنات المستمرة، وأنماط التحديث، وRemoveFromModifiedObjects، والمزالق الشائعة. يُستخدم عند إنشاء مصادر بيانات مخصصة أو أغلفة API أو نماذج الإدخال أو طرق العرض المحسوبة دون استمرار قاعدة البيانات في DevExpress XAF.
