Éditeurs de propriétés et éditeurs de liste intégrés XAF - mappage des types d'éditeur par type de données, constantes EditorAliases, attribut [EditorAlias], [ModelDefault] pour DisplayFormat/EditMask, ObjectPropertyEditor pour les sous-formulaires en ligne, types d'éditeur de liste (GridListEditor, DxGridListEditor, TreeListEditor, ChartListEditor), personnalisation GridListEditor WinForms, DxGridListEditor Personnalisation Blazor, propriétés IModelListView/IModelColumn. À utiliser lorsque vous travaillez avec les éditeurs XAF intégrés ou lorsque vous personnalisez les vues de grille/liste.
