أرسل الملفات والصور والرسائل الصوتية عبر Feishu Lark API باستخدام العملية الإلزامية المكونة من خطوتين. استخدمه عند الحاجة إلى إرسال ملفات أو صور أو رسائل صوتية إلى مستخدمي أو مجموعات Feishu. تفرض هذه المهارة سير العمل المطلوب - قم أولاً بالتحميل للحصول على file_key، أو image_key، أو audio_key، ثم أرسل رسالة بالمفتاح. يدعم واجهة برمجة تطبيقات الملفات للمستندات، وواجهة برمجة تطبيقات الصور للصور، وواجهة برمجة تطبيقات الملفات مع نوع الصوت للرسائل الصوتية.
