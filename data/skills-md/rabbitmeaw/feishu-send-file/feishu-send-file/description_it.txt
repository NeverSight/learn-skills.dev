Invia file, immagini e messaggi audio tramite l'API Feishu Lark utilizzando la procedura obbligatoria in due passaggi. Utilizzare quando è necessario inviare file, immagini o messaggi vocali a utenti o gruppi Feishu. Questa abilità applica il flusso di lavoro richiesto: prima carica per ottenere file_key, image_key o audio_key, quindi invia il messaggio con la chiave. Supporta API file per documenti, API immagini per immagini e API file con tipo audio per messaggi vocali.
