Envíe archivos, imágenes y mensajes de audio a través de la API Feishu Lark mediante el proceso obligatorio de dos pasos. Úselo cuando necesite enviar archivos, imágenes o mensajes de voz a usuarios o grupos de Feishu. Esta habilidad aplica el flujo de trabajo requerido: primero cargue para obtener file_key, image_key o audio_key, luego envíe el mensaje con la clave. Admite API de archivos para documentos, API de imágenes para imágenes y API de archivos con tipo de audio para mensajes de voz.
