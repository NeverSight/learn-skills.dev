Отправляйте файлы, изображения и аудиосообщения через API Feishu Lark, используя обязательный двухэтапный процесс. Используйте, когда необходимо отправить файлы, изображения или голосовые сообщения пользователям или группам Feishu. Этот навык обеспечивает необходимый рабочий процесс — сначала загрузите файл, чтобы получить file_key, image_key или audio_key, а затем отправьте сообщение с ключом. Поддерживает файловый API для документов, API изображений для изображений и API файлов со звуковым типом для голосовых сообщений.
