Senden Sie Dateien, Bilder und Audionachrichten über die Feishu Lark API mit dem obligatorischen zweistufigen Prozess. Verwenden Sie diese Option, wenn Sie Dateien, Bilder oder Sprachnachrichten an Feishu-Benutzer oder -Gruppen senden müssen. Dieser Skill erzwingt den erforderlichen Workflow – zuerst hochladen, um file_key, image_key oder audio_key zu erhalten, und dann eine Nachricht mit dem Schlüssel senden. Unterstützt die Datei-API für Dokumente, die Bild-API für Bilder und die Datei-API mit Audiotyp für Sprachnachrichten.
