Преобразует устаревший код Java в современные идиомы. Используйте при миграции версий Java, преобразовании подробных POJO/DTO в записи, потоков платформы в виртуальные потоки, циклов for в потоки, анонимных классов в лямбда-выражения или когда база кода ориентирована на Java 11–25, но использует более старые шаблоны. С учетом версии (Java 8-25), с учетом особенностей платформы (Spring Boot, Quarkus, Micronaut), автоматически определяет версию из pom.xml или build.gradle.
