Trasforma il codice Java legacy in idiomi moderni. Da utilizzare durante la migrazione di versioni Java, la conversione di POJO/DTO dettagliati in record, i thread della piattaforma in thread virtuali, i cicli for in flussi, le classi anonime in lambda o quando una codebase ha come target Java 11-25 ma utilizza modelli precedenti. Riconoscimento della versione (Java 8-25), framework-smart (Spring Boot, Quarkus, Micronaut), rileva automaticamente la versione da pom.xml o build.gradle.
