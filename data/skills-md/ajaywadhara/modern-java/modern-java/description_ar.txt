يحول كود Java القديم إلى التعابير الحديثة. يُستخدم عند ترحيل إصدارات Java، أو تحويل POJOs/DTOs المطولة إلى سجلات، أو سلاسل رسائل النظام الأساسي إلى سلاسل رسائل افتراضية، أو حلقات for إلى التدفقات، أو فئات مجهولة إلى lambdas، أو عندما تستهدف قاعدة التعليمات البرمجية Java 11-25 ولكنها تستخدم أنماطًا أقدم. مدرك للإصدار (Java 8-25)، إطار عمل ذكي (Spring Boot، Quarkus، Micronaut)، يكتشف الإصدار من pom.xml أو build.gradle تلقائيًا.
