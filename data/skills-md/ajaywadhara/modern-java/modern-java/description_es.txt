Transforma el código Java heredado en modismos modernos. Úselo al migrar versiones de Java, convertir POJO/DTO detallados en registros, subprocesos de plataforma en subprocesos virtuales, bucles for a secuencias, clases anónimas en lambdas o cuando una base de código apunta a Java 11-25 pero utiliza patrones más antiguos. Consciente de la versión (Java 8-25), inteligente con el marco (Spring Boot, Quarkus, Micronaut), detecta la versión de pom.xml o build.gradle automáticamente.
