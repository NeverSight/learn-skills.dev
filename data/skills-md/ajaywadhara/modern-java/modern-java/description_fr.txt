Transforme le code Java existant en idiomes modernes. À utiliser lors de la migration de versions Java, de la conversion de POJO/DTO verbeux en enregistrements, de threads de plate-forme en threads virtuels, de boucles for en flux, de classes anonymes en lambdas ou lorsqu'une base de code cible Java 11-25 mais utilise des modèles plus anciens. Prise en compte des versions (Java 8-25), framework-smart (Spring Boot, Quarkus, Micronaut), détecte automatiquement la version de pom.xml ou build.gradle.
