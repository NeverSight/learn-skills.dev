Wandelt alten Java-Code in moderne Redewendungen um. Wird verwendet, wenn Sie Java-Versionen migrieren, ausführliche POJOs/DTOs in Datensätze, Plattform-Threads in virtuelle Threads, For-Schleifen in Streams, anonyme Klassen in Lambdas konvertieren oder wenn eine Codebasis auf Java 11–25 abzielt, aber ältere Muster verwendet. Versionsbewusst (Java 8–25), Framework-intelligent (Spring Boot, Quarkus, Micronaut), erkennt die Version automatisch aus pom.xml oder build.gradle.
