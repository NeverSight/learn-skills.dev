يوفر إرشادات الاستخدام لمكونات Instructure UI (InstUI) React. يُستخدم عند العمل مع حزم @instructure/ui، وعندما يُسأل عن كيفية استخدام مكونات InstUI، أو المكون الذي يجب اختياره، أو كيفية تطبيق الدعائم أو الإرشادات أو السمات أو قواعد إمكانية الوصول. يغطي AiInformation، تنبيه، AppNav، Avatar، Badge، Billboard، Breadcrumb، Button، Byline، Calendar، Checkbox، CheckboxGroup، CloseButton، ColorContrast، ColorIndicator، ColorMixer، ColorPicker، ColorPreset، CondensedButton، contextView، DataPermissionLevels، DateInput، DateInput2، DateTimeInput، DrawerLayout، Drilldown، Editable، FileDrop، Flex، FormField، FormFieldGroup، Grid، Heading، IconButton، Img، InlineList، InPlaceEdit، InstUISettingsProvider، Link، List، القائمة، Metric، MetricGroup، Modal، NutritionFacts، NumberInput، Overlay، Pages، ترقيم الصفحات، Pill، Popover، ProgressBar، ProgressCircle، RadioInput، RadioInputGroup، RangeInput، التقييم، الاستجابة، التحديد، SideNavBar، SimpleSelect، SourceCodeEditor، Spinner، الجدول، علامات التبويب، العلامة، النص، TextArea، TextInput، TimeSelect، ToggleButton، ToggleDetails، ToggleGroup، Tooltip، TopNavBar، Tray، TreeBrowser، TruncateText، View.
