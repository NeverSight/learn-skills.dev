Содержит руководство по использованию компонентов React Instructure UI (InstUI). Используйте при работе с пакетами @instructure/ui, когда вас спрашивают, как использовать компоненты InstUI, какой компонент выбрать или как применять реквизиты, рекомендации, темы или правила доступности.
