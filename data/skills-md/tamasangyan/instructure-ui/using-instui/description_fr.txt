Fournit des conseils d’utilisation pour les composants React d’Instructure UI (InstUI). À utiliser lorsque vous travaillez avec des packages @instructure/ui, lorsqu'on lui demande comment utiliser les composants InstUI, quel composant choisir ou comment appliquer des accessoires, des directives, des thèmes ou des règles d'accessibilité. Couvre AiInformation, Alerte, AppNav, Avatar, Badge, Panneau d'affichage, Fil d'Ariane, Bouton, Byline, Calendrier, Case à cocher, CheckboxGroup, CloseButton, ColorContrast, ColorIndicator, ColorMixer, ColorPicker, ColorPreset, CondensedButton, ContextView, DataPermissionLevels, DateInput, DateInput2, DateTimeInput, DrawerLayout, Drilldown, Editable, FileDrop, Flex, FormField, FormFieldGroup, Grille, Titre, IconButton, Img, InlineList, InPlaceEdit, InstUISettingsProvider, Lien, Liste, Menu, Métrique, MetricGroup, Modal, NutritionFacts, NumberInput, Superposition, Pages, Pagination, Pill, Popover, ProgressBar, ProgressCircle, RadioInput, RadioInputGroup, RangeInput, Note, Responsive, Select, SideNavBar, SimpleSelect, SourceCodeEditor, Spinner, Tableau, Onglets, Balise, Texte, TextArea, TextInput, TimeSelect, ToggleButton, ToggleDetails, ToggleGroup, Tooltip, TopNavBar, Tray, TreeBrowser, TruncateText, View.
