Спецификация реализации интерфейса API OpenLark Project (краткий справочник). Для добавления/рефакторинга API-интерфейсов открытой платформы Feishu: определите путь к диску, реализуйте Body/Response + Builder(Request), выровняйте константы/перечисления конечных точек, завершите экспорт mod.rs и уточните сигнатуру метода для «вызова серверных API» и соглашений о передаче RequestOption. Ключевые слова-триггеры: интерфейс API, файл API, API Feishu, добавление API, вызов API на стороне сервера.
