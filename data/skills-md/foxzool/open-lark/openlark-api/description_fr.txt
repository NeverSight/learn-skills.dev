Spécification d'implémentation de l'interface API du projet OpenLark (référence rapide). Pour ajouter/refactoriser les API Feishu Open Platform : déterminez le chemin du disque, implémentez Body/Response + Builder (Request), alignez les constantes/énumérations de point de terminaison, terminez les exportations mod.rs et clarifiez la signature de la méthode pour « appeler les API côté serveur » et les conventions de passage de RequestOption.
