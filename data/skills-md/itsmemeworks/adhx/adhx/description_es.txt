Obtenga publicaciones de X/Twitter como JSON limpio y compatible con LLM a través de la API ADHX. Convierte cualquier enlace de x.com, twitter.com o adhx.com en datos estructurados con el contenido completo del artículo, información del autor y métricas de participación. Úselo cuando un usuario comparte un enlace X/Twitter y desea leer, analizar o resumir la publicación.
