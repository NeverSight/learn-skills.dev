Capacité de recherche Web utilisant l'API DuckDuckGo par défaut, avec l'API Tavily de secours lorsque TRAVILY_TOKEN est fourni. À utiliser lorsque vous devez rechercher sur le Web des informations actuelles, rechercher des sujets, trouver des URL ou collecter des données à partir de sources en ligne. Prend en charge les requêtes d'actualités, d'informations techniques, de documents de recherche, de données d'entreprise et de contenu Web général.
