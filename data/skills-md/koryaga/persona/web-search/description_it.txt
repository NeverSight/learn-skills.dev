Funzionalità di ricerca Web che utilizza l'API DuckDuckGo per impostazione predefinita, con fallback dell'API Tavily quando viene fornito TRAVILY_TOKEN. Utilizzare quando è necessario cercare sul Web informazioni attuali, ricercare argomenti, trovare URL o raccogliere dati da fonti online. Supporta query per notizie, informazioni tecniche, documenti di ricerca, dati aziendali e contenuti Web generali.
