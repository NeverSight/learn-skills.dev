Этот навык следует использовать, когда пользователь запрашивает «инициализацию автоматической памяти», «создание CLAUDE.md», «настройку памяти проекта» или запускает команду /auto-memory:init. Анализирует структуру кодовой базы и генерирует файлы CLAUDE.md, используя точный формат шаблона с АВТОМАТИЧЕСКИ УПРАВЛЯЕМЫМИ маркерами.
