Aktualisieren Sie Java-Code von älteren Redewendungen auf moderne Äquivalente. Sucht nach älteren Mustern (vor Java 10 bis Java 25) in allen Sprachfunktionen, Sammlungen, Zeichenfolgen, Streams, Parallelität, Eingabe/Ausgabe, Fehlerbehandlung, Datum/Uhrzeit, Sicherheit, Tools und Unternehmens-APIs und schlägt dann moderne Ersetzungen mit Vorher/Nachher-Beispielen vor. Verwenden Sie es, wenn Sie Java-Codebasen modernisieren, Pull-Requests auf veraltete Muster überprüfen oder von Java EE nach Jakarta EE migrieren. Nicht für Nicht-Java-Sprachen, Build-Tool-Konfiguration oder Framework-spezifische Anwendungslogik verwenden.
