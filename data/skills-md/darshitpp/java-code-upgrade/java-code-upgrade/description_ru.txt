Обновите код Java со старых идиом до современных эквивалентов. Сканирует устаревшие шаблоны (от версий до Java 10 до Java 25) в функциях языка, коллекциях, строках, потоках, параллелизме, вводе/выводе, обработке ошибок, дате/времени, безопасности, инструментах и ​​корпоративных API, а затем предлагает современные замены с примерами до/после. Используйте при модернизации баз кода Java, проверке запросов на включение устаревших шаблонов или переходе с Java EE на Jakarta EE. Не используйте для языков, отличных от Java, конфигурации инструмента сборки или логики приложения, специфичной для платформы.
