Java コードを古い表現から最新の表現にアップグレードします。言語機能、コレクション、文字列、ストリーム、同時実行性、入出力、エラー処理、日付/時刻、セキュリティ、ツール、エンタープライズ API にわたってレガシー パターン (Java 10 より前から Java 25 まで) をスキャンし、前後の例を使用して最新の置き換えを提案します。 Java コードベースを最新化する場合、古いパターンのプル リクエストを確認する場合、または Java EE から Jakarta EE に移行する場合に使用します。 Java 以外の言語、ビルド ツール構成、またはフレームワーク固有のアプリケーション ロジックには使用しないでください。
