Actualice el código Java de modismos antiguos a equivalentes modernos. Busca patrones heredados (pre-Java 10 a Java 25) en características de lenguaje, colecciones, cadenas, secuencias, simultaneidad, entrada/salida, manejo de errores, fecha/hora, seguridad, herramientas y API empresariales, luego sugiere reemplazos modernos con ejemplos de antes/después. Úselo al modernizar bases de código Java, revisar solicitudes de extracción en busca de patrones obsoletos o migrar de Java EE a Jakarta EE. No lo utilice para lenguajes que no sean Java, configuración de herramientas de compilación o lógica de aplicación específica del marco.
