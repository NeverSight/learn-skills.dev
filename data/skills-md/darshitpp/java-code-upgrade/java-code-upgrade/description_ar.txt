ترقية كود Java من التعابير القديمة إلى مرادفاتها الحديثة. يبحث عن الأنماط القديمة (ما قبل Java 10 إلى Java 25) عبر ميزات اللغة والمجموعات والسلاسل والتدفقات والتزامن والإدخال/الإخراج ومعالجة الأخطاء والتاريخ/الوقت والأمان والأدوات وواجهات برمجة تطبيقات المؤسسة، ثم يقترح بدائل حديثة بأمثلة قبل/بعد. يُستخدم عند تحديث قواعد تعليمات Java البرمجية، أو مراجعة طلبات السحب للأنماط القديمة، أو الترحيل من Java EE إلى Jakarta EE. لا تستخدمه للغات غير Java، أو تكوين أداة البناء، أو منطق التطبيق الخاص بإطار العمل.
