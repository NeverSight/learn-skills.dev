قم بتحليل ملفات سير عمل GitHub Actions بحثًا عن الثغرات الأمنية، والأخطاء الدلالية، وانتهاكات أفضل الممارسات، ومشكلات المخطط، ومشكلات النمط باستخدام 46 قاعدة عبر 6 فئات (المخطط، والأمان، والدلالي، وأفضل الممارسات، والأسلوب، وactionlint). يسجل سير العمل على مقياس من 0 إلى 100 مع درجات الحروف (A+ إلى F). يُنشئ توصيات إصلاح مفصلة مع أمثلة YAML قبل/بعد. استخدم هذه المهارة بشكل استباقي عندما يشارك المستخدم سير عمل GitHub Actions للمراجعة، أو يلصق CI/CD YAML، أو يسأل عن أفضل ممارسات GitHub Actions أو تشديد الأمان، أو يريد تحسين الوضع الأمني ​​لسير العمل، أو يذكر .github/workflows، أو يناقش تكوين خط أنابيب CI/CD - حتى لو لم يقول صراحة "التحقق من الصحة" أو "التحليل".
