قم بتحليل ملفات Docker Compose بحثًا عن الثغرات الأمنية، والأخطاء الدلالية، وانتهاكات أفضل الممارسات، ومشكلات المخطط، ومشكلات النمط باستخدام 52 قاعدة عبر 5 فئات (الأمان، والدلالات، والمخطط، وأفضل الممارسات، والأسلوب). الدرجات قم بتأليف الملفات على مقياس من 0 إلى 100 مع درجات الحروف (A+ إلى F). يُنشئ توصيات إصلاح مفصلة مع أمثلة YAML قبل/بعد. استخدم هذه المهارة بشكل استباقي عندما يقوم المستخدم بمشاركة docker-compose.yml أو compose.yaml للمراجعة، أو لصق Docker Compose YAML، أو يسأل عن أفضل ممارسات Docker Compose أو تشديد الأمان، أو يريد تحسين الوضع الأمني ​​لتنسيق الحاوية، أو يذكر إنشاء عامل إرساء أو إنشاء ملفات، أو يسأل عن إعدادات Docker متعددة الحاويات - حتى لو لم يقولوا صراحةً "التحقق من الصحة" أو "التحليل".
