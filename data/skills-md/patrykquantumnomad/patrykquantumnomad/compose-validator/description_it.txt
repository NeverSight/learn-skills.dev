Analizza Docker Componi file per vulnerabilità di sicurezza, errori semantici, violazioni di best practice, problemi di schema e problemi di stile utilizzando 52 regole in 5 categorie (sicurezza, semantica, schema, best practice, stile). Punteggi Componi file su una scala da 0 a 100 con voti in lettere (da A+ a F). Genera consigli dettagliati sulle correzioni con esempi YAML prima/dopo. Utilizza questa competenza in modo PROATTIVO ogni volta che un utente condivide un docker-compose.yml o compose.yaml per la revisione, incolla Docker Compose YAML, chiede informazioni sulle best practice o sul rafforzamento della sicurezza di Docker Compose, desidera migliorare il livello di sicurezza dell'orchestrazione dei contenitori, menziona docker-compose o compose file o chiede informazioni sulle configurazioni Docker multi-contenitore, anche se non dice esplicitamente "convalida" o "analizza".
