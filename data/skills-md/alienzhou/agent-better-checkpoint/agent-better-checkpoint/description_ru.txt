Заменяет непрозрачные контрольные точки платформы на прозрачные, запрашиваемые фиксации Git с использованием формата обычных коммитов с трейлерами Git. Вы ДОЛЖНЫ следовать этому навыку всякий раз, когда вносите изменения в код — фиксируйте изменения после каждого значимого редактирования.
