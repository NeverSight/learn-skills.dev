يقوم تلقائيًا بإنشاء نقاط تفتيش Git الدلالية أثناء جلسات تشفير الذكاء الاصطناعي. يستبدل نقاط تفتيش النظام الأساسي غير الشفافة بالتزامات Git الشفافة والقابلة للاستعلام باستخدام تنسيق الالتزامات التقليدية مع Git Trailers. يجب عليك اتباع هذه المهارة كلما قمت بإجراء تغييرات على التعليمات البرمجية - قم بالالتزام بعد كل تعديل ذي معنى.
