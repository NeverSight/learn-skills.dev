(Norme industrielle : agent parallèle) Cas d'utilisation principal : travail pouvant être partitionné en sous-tâches indépendantes exécutées simultanément sur plusieurs agents. Modèle d'exécution multi-agent parallèle. À utiliser lorsque : le travail peut être divisé en tâches indépendantes que N agents peuvent exécuter simultanément sur des arbres de travail. Comprend le routage (séquentiel ou parallèle), la vérification de fusion et les boucles de correction.
