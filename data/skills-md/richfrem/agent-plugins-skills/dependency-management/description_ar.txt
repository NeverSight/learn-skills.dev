تبعية بايثون وإدارة البيئة للخدمات المتعددة أو الواجهات الخلفية لبيثون مونوريبو. يُستخدم عندما: (1) إضافة حزمة Python أو ترقيتها أو إزالتها، (2) الاستجابة لـ Dependabot أو تنبيهات الثغرات الأمنية (GHSA/CVE)، (3) إنشاء خدمة جديدة تحتاج إلى ملفات المتطلبات الخاصة بها، (4) تصحيح أخطاء تثبيت النقاط أو مشكلات بناء Docker المتعلقة بالتبعيات، (5) مراجعة شجرة التبعية أو تدقيقها، (6) تشغيل ترجمة النقاط. يفرض سير عمل الملف المقفل والترجمة الهرمية للتبعية المتدرجة.
