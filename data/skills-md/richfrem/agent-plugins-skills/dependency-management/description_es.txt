Gestión del entorno y dependencia de Python para backends de Python multiservicio o monorepo. Úselo cuando: (1) agregue, actualice o elimine un paquete de Python, (2) responda a Dependabot o alertas de vulnerabilidad de seguridad (GHSA/CVE), (3) cree un nuevo servicio que necesita sus propios archivos de requisitos, (4) depure fallas de instalación de pip o problemas de compilación de Docker relacionados con dependencias, (5) revise o audite el árbol de dependencias, (6) ejecute pip-compile. Aplica el flujo de trabajo de archivos bloqueados de compilación pip y la jerarquía de dependencia por niveles.
