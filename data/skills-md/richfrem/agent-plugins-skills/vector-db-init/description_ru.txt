Интерактивно инициализирует плагин Vector DB. Устанавливает необходимые зависимости pip (chromadb, оболочки langchain) и настраивает вектор_профили.json для подключений к собственному серверу Python. Запустите это, прежде чем пытаться использовать навыки векторного поиска или векторного запуска db.
