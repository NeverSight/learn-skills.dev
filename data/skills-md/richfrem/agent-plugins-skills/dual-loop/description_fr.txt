(Norme industrielle : Agent séquentiel / Agent en tant qu'outil) Cas d'utilisation principal : déléguer une tâche bien définie à un agent de travail, vérifier son exécution et répéter si nécessaire. Modèle de délégation d’agent interne/externe. À utiliser lorsque : le travail doit être délégué d'un contrôleur stratégique (Outer Loop) à un exécuteur tactique (Inner Loop) via des paquets de stratégie, avec des boucles de vérification et de correction.
