(Branchenstandard: Sequential Agent / Agent as a Tool) Primärer Anwendungsfall: Delegieren einer genau definierten Aufgabe an einen Worker-Agenten, Überprüfen ihrer Ausführung und Wiederholen bei Bedarf. Delegationsmuster für innere/äußere Agenten. Verwenden Sie, wenn: Arbeit über Strategiepakete mit Überprüfungs- und Korrekturschleifen von einem strategischen Controller (Outer Loop) an einen taktischen Ausführenden (Inner Loop) delegiert werden muss.
