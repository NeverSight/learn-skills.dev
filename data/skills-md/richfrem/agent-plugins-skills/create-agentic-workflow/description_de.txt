Gerüst für GitHub-Agentendateien aus einem vorhandenen Agenten-Skill. Erzeugt IDE/UI-Agenten (aufrufbar über GitHub Copilot Chat per Slash-Befehl) und/oder autonome CI/CD-Agenten (GitHub Actions Quality Gates mit Kill Switch). Wird zum Konvertieren eines Skills in einen GitHub-nativen Agenten verwendet.
