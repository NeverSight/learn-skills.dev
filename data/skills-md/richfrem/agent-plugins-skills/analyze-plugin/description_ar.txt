قم بتحليل المكونات الإضافية للوكيل ومهاراته بشكل منهجي لاستخراج أنماط التصميم والقرارات المعمارية والتقنيات القابلة لإعادة الاستخدام. قم بتشغيل "تحليل هذا المكون الإضافي"، أو "استخراج الأنماط من"، أو "مراجعة بنية المكون الإضافي"، أو "استخراج الدروس المستفادة من"، أو "ما الأنماط التي يستخدمها هذا المكون الإضافي"، أو عند فحص أي مكون إضافي أو مجموعة مهارات لفهم تصميمه.
