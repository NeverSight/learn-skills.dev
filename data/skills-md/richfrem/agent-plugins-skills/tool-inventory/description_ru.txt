Менеджер инвентаризации инструментов и агент обнаружения (Библиотекарь). Автоматически вызывается, когда задачи включают регистрацию инструментов, поиск сценариев, аудит покрытия или ведение реестра инструментов. Сочетает семантический поиск ChromaDB с протоколом обнаружения Search → Bind → Execute. V2 включает ограничения L4/L5 для предотвращения галлюцинаций.
