Gestionnaire d'inventaire d'outils et agent de découverte (The Librarian). Invoqué automatiquement lorsque les tâches impliquent l'enregistrement d'outils, la recherche de scripts, l'audit de la couverture ou la maintenance du registre des outils. Combine la recherche sémantique ChromaDB avec le protocole de découverte Rechercher → Lier → Exécuter. La V2 inclut des contraintes L4/L5 pour éviter les hallucinations.
