(معيار الصناعة: وكيل الحلقة / وكيل واحد) حالة الاستخدام الأساسي: البحث القائم بذاته، وإنشاء المحتوى، والاستكشاف حيث لا يلزم التفويض الداخلي. حلقة البحث والمعرفة الموجهة ذاتيًا. يُستخدم عندما: بدء جلسة (توجيه)، أو إجراء بحث (توليف)، أو إغلاق جلسة (ختم، استمرار، بأثر رجعي). يضمن بقاء المعرفة عبر جلسات الوكيل المعزولة.
