(Branchenstandard: Loop Agent / Single Agent) Primärer Anwendungsfall: Eigenständige Forschung, Inhaltsgenerierung und Erkundung, bei der keine interne Delegation erforderlich ist. Selbstgesteuerte Forschungs- und Wissenserfassungsschleife. Verwenden Sie diese Option, wenn Sie eine Sitzung starten (Orientierung), eine Recherche durchführen (Synthese) oder eine Sitzung schließen (Siegeln, Persistieren, Retrospektive). Stellt sicher, dass das Wissen über isolierte Agentensitzungen hinweg erhalten bleibt.
