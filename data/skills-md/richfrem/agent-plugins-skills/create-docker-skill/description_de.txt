Interaktives Initialisierungsskript, das einen kompatiblen Agenten-Skill generiert, der Pre-Flight-Umgebungsprüfungen, ein Gerüst für die Unterprozessausführung und eine Sicherheitsüberschreibungskonfiguration enthält. Verwenden Sie diese Option, wenn Sie neue Workflow-Routinen erstellen, die von externen Containerlaufzeiten abhängen (z. B. Docker, Nextflow, HPC).
