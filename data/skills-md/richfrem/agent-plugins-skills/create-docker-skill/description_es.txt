Script de inicialización interactivo que genera una habilidad de agente compatible que contiene comprobaciones ambientales previas al vuelo, andamios de ejecución de subprocesos y una configuración de anulación de seguridad. Úselo al crear nuevas rutinas de flujo de trabajo que dependen de tiempos de ejecución en contenedores externos (por ejemplo, Docker, Nextflow, HPC).
