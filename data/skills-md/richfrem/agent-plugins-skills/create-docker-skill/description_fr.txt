Script d'initialisation interactif qui génère une compétence d'agent conforme contenant des vérifications d'environnement avant le vol, un échafaudage d'exécution de sous-processus et une configuration de remplacement de sécurité. À utiliser lors de la création de nouvelles routines de flux de travail qui dépendent d'environnements d'exécution conteneurisés externes (par exemple, Docker, Nextflow, HPC).
