Sistema de memoria escalonada para la continuidad cognitiva entre las sesiones del agente. Administra el caché activo (contexto de sesión cargado al arrancar) y el almacenamiento profundo (cargado bajo demanda). Úselo cuando: (1) inicie una sesión y cargue el contexto, (2) decida qué recordar y qué olvidar, (3) promover/degradar el conocimiento entre niveles, (4) el usuario dice "recordar esto" o pregunta sobre el historial del proyecto.
