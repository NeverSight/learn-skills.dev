Создайте традиционный детерминированный рабочий процесс CI/CD GitHub Actions. Используйте это при создании конвейеров сборки, тестирования, развертывания, проверки, выпуска или сканирования безопасности. Это отличается от агентских рабочих процессов — во время выполнения не задействован искусственный интеллект.
