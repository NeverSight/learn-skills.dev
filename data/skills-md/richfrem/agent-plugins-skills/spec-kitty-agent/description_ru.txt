Комбинированный агент Spec-Kitty: механизм синхронизации + рабочий процесс разработки на основе спецификаций. Автоматически вызывается для жизненного цикла функции (Указать → План → Задачи → Реализация → Обзор → Объединить) и синхронизации конфигурации агента. Предварительное условие: установлен spec-kitty-cli.
