Agente combinato Spec-Kitty: motore di sincronizzazione + flusso di lavoro di sviluppo basato sulle specifiche. Richiamato automaticamente per il ciclo di vita delle funzionalità (Specifica → Piano → Attività → Implementa → Revisione → Unisci) e la sincronizzazione della configurazione dell'agente. Prerequisito: spec-kitty-cli installato.
