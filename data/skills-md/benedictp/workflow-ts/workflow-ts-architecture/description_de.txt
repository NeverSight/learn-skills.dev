Erstellen und überprüfen Sie die TypeScript-Funktionslogik mit Workflow-ts („@workflow-ts/core“ und „@workflow-ts/react“) unter Verwendung expliziter Zustandsmaschinen, typisierter Renderings, Worker, Komposition und Laufzeittests. Wird verwendet, wenn Anforderungen das Erstellen oder Refactoring von Workflows, das Verknüpfen von React-Hooks, das Modellieren asynchroner Worker-Flows, das Erstellen von übergeordneten/untergeordneten Workflows, das Hinzufügen von Snapshots/Interceptors/Devtools oder das Schreiben von Tests für das Workflow-Verhalten in diesem Repository umfassen.
