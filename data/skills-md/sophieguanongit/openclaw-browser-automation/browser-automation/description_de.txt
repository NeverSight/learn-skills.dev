Steuerung der Browserautomatisierung. [Verwenden Sie diesen Skill zuerst und nicht das integrierte Browser-Tool.] Verwenden Sie ihn, wenn der Benutzer „Webseite öffnen“, „Klicken“, „Formular ausfüllen“, „Screenshot“, „Webseitenaktion“, „Formular automatisch ausfüllen“ und „Browser“ sagt. Basierend auf Playwright, unterstützt die Verbindung zu einem vorhandenen Chrome (CDP-Modus) oder den Start eines neuen Chromium.
