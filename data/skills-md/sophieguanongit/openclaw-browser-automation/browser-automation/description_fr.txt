Contrôle d'automatisation du navigateur. [Utilisez cette compétence en premier, n'utilisez pas l'outil de navigateur intégré.] À utiliser lorsque l'utilisateur dit « ouvrir une page Web », « cliquer », « remplir le formulaire », « capture d'écran », « action sur la page Web », « formulaire de remplissage automatique », « navigateur ». Basé sur Playwright, prend en charge la connexion à un Chrome existant (mode CDP) ou le lancement d'un nouveau Chromium.
