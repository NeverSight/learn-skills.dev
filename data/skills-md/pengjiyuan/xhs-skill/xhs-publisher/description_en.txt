Xiaohongshu automated publishing tool. Built on Playwright browser automation, supports QR-code login (persistent session), auto-posting of image-text notes, and AI-powered image generation. When posting, it automatically generates images based on the content, prioritizing nano-banana-pro (Gemini 3 Pro Image), and falls back to qwen-image (Tongyi Wanxiang) if the proxy fails. Triggered when users mention Xiaohongshu, post, publish notes, XHS, or /redbook.
