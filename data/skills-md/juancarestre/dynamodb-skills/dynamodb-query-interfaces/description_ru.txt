Создает независимые от языка интерфейсы репозитория (сигнатуры методов, типы ввода/вывода и модели сущностей) из документа проекта таблицы DynamoDB. Используйте, когда у пользователя есть файл дизайна таблицы .md (на основе навыков dynamodb-table-design) и он хочет создать скелет кода для своего уровня доступа к данным. Это шаг 3 трехэтапного конвейера: шаблоны доступа -> дизайн таблиц -> интерфейсы запросов.
