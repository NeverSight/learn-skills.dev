Erzeugt sprachunabhängige Repository-Schnittstellen (Methodensignaturen, Eingabe-/Ausgabetypen und Entitätsmodelle) aus einem DynamoDB-Tabellenentwurfsdokument. Verwenden Sie diese Option, wenn der Benutzer über eine Tabellenentwurfs-.md-Datei (aus dem Skill Dynamodb-table-design) verfügt und das Codegerüst für seine Datenzugriffsschicht generieren möchte. Dies ist Schritt 3 einer dreistufigen Pipeline: Zugriffsmuster -> Tabellendesign -> Abfrageschnittstellen.
