Entwirft eine DynamoDB-Einzeltabellenstruktur aus einem dokumentierten Satz von Zugriffsmustern. Verwenden Sie diese Option, wenn der Benutzer über eine .md-Datei mit Zugriffsmustern (aus dem Skill dynamodb-access-patterns) verfügt und Partitionsschlüssel, Sortierschlüssel, GSIs und zusammengesetzte Schlüssel entwerfen und jedes Zugriffsmuster einer konkreten DynamoDB-Abfrageoperation zuordnen möchte. Dies ist Schritt 2 einer dreistufigen Pipeline: Zugriffsmuster -> Tabellendesign -> Abfrageschnittstellen.
