Suppression des signes de texte généré par l'IA en ukrainien. À utiliser pour éditer ou vérifier le texte afin qu'il paraisse naturel en ukrainien. Détecte et corrige : les russismes, les anglicismes, l'officialisme, les pléonasmes, le vocabulaire de l'IA, l'accentuation excessive, le langage promotionnel, les filigranes Unicode cachés, les modèles statistiques d'IA (faible perplexité, faible rafale).

Déclencheurs : humaniser le texte ukrainien, rendre le texte ukrainien, supprimer l'IA, corriger le surzhyk, modifier en ukrainien, nature ukrainien, supprimer les modèles d'IA de l'ukrainien, corriger les calques russes, corriger les russismes, la bureaucratie.
