Herramienta de gestión de implementación de Yanqianyun. Se utiliza para consultar las versiones de Helm Chart en ChartMuseum, administrar las configuraciones de implementación del repositorio de GitOps, modificar las variables de entorno y chartVersion. Utilice esta habilidad cuando el usuario necesite: (1) consultar las versiones de gráficos disponibles para un servicio, (2) implementar o actualizar servicios en diferentes entornos (prueba/preproducción/stag), (3) modificar configuraciones de implementación o variables de entorno, (4) ver el estado de implementación actual.
