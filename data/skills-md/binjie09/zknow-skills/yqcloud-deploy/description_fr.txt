Outil de gestion du déploiement Yanqianyun. Utilisé pour interroger les versions de Helm Chart dans ChartMuseum, gérer les configurations de déploiement du référentiel GitOps, modifier les variables d'environnement et chartVersion. Utilisez cette compétence lorsque l'utilisateur a besoin de : (1) interroger les versions de graphiques disponibles pour un service, (2) déployer ou mettre à jour des services dans différents environnements (test/preprod/stag), (3) modifier les configurations de déploiement ou les variables d'environnement, (4) vérifier l'état actuel du déploiement.
