Yanqianyun Deployment Management Tool. Used to query Helm Chart versions in ChartMuseum, manage GitOps repository deployment configurations, modify environment variables and chartVersion. Use this skill when the user needs to: (1) query available chart versions for a service, (2) deploy or update a service to different environments (test/preprod/stag), (3) modify deployment configurations or environment variables, (4) check the current deployment status.
