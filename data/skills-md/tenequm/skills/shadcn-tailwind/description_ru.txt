Создавайте пользовательские интерфейсы с помощью Tailwind CSS v4 и shadcn/ui. Охватывает переменные CSS с цветами OKLCH, варианты компонентов с CVA, адаптивный дизайн, темный режим и функции Tailwind v4.2. Поддерживает примитивы пользовательского интерфейса Radix и базового пользовательского интерфейса, CLI 3.0 и визуальные стили. Используйте при создании интерфейсов с помощью Tailwind, стилизации компонентов shadcn/ui, реализации тем или работе с CSS, ориентированным на утилиты. Триггеры по попутному ветру, shadcn, служебным классам, переменным CSS, OKLCH, стилю компонентов, темам, темному режиму, основанию пользовательского интерфейса.
