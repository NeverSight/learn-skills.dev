أنشئ مدفوعات عبر الإنترنت باستخدام بروتوكول x402 المفتوح. يُستخدم عند تطوير واجهات برمجة التطبيقات المدفوعة، أو المحتوى المحمي بنظام حظر الاشتراك غير المدفوع، أو تدفقات دفع وكيل الذكاء الاصطناعي، أو أي خدمة تستخدم الدفع HTTP 402 المطلوبة للمدفوعات الصغيرة على السلسلة. يغطي أدوات تطوير البرامج الخاصة بـ TypeScript وPython وGo عبر شبكات EVM (Base وMegaETH وMonad) وSolana وStellar وAptos مع عمليات نقل HTTP وMCP وA2A. يدعم أنظمة الدفع الدقيقة وحتى (على أساس الاستخدام).
