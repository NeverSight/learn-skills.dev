Создавайте собственные интернет-платежи с помощью открытого протокола x402. Используйте при разработке платных API, контента с платным доступом, потоков платежей AI-агента или любой службы, использующей HTTP 402 «Требуется оплата» для микроплатежей в цепочке. Охватывает SDK TypeScript, Python и Go в сетях EVM (Base, MegaETH, Monad), Solana, Stellar и Aptos с транспортами HTTP, MCP и A2A. Поддерживает точные и максимальные схемы оплаты (на основе использования).
