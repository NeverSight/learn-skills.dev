أنشئ تطبيقات React آمنة من حيث النوع باستخدام TanStack Query (جلب البيانات، والتخزين المؤقت، والطفرات)، وجهاز التوجيه (التوجيه المستند إلى الملف، ومعلمات البحث، وأدوات التحميل)، والبدء (SSR، ووظائف الخادم، والبرامج الوسيطة). يُستخدم عند العمل مع استعلام التفاعل، أو جلب البيانات، أو حالة الخادم، أو التوجيه، أو معلمات البحث، أو أدوات التحميل، أو SSR، أو وظائف الخادم، أو React الكامل. المشغلات على tanstack، استعلام التفاعل، عميل الاستعلام، useQuery، useMutation، validateQueries، جهاز توجيه tanstack، التوجيه المستند إلى الملف، معلمات البحث، أداة تحميل المسار، بدء tanstack، createServerFn، وظائف الخادم، SSR.
