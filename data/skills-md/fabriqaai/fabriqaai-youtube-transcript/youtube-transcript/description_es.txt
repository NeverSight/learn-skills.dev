Recupere transcripciones de YouTube sin MCP utilizando una herramienta CLI de Python. Úselo cuando un usuario solicita el texto de la transcripción completa o los idiomas de transcripción disponibles desde una URL de YouTube o una ID de video. Para resúmenes, puntos clave o citas, utilice el resumen de IA (prefiera un subagente cuando esté disponible) para guardar tokens del contexto principal.
