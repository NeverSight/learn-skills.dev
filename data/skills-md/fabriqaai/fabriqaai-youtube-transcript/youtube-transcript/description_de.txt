Rufen Sie YouTube-Transkripte ohne MCP mit einem Python-CLI-Tool ab. Verwenden Sie diese Option, wenn ein Benutzer den vollständigen Transkripttext oder verfügbare Transkriptsprachen über eine YouTube-URL oder Video-ID anfordert. Verwenden Sie für Zusammenfassungen, wichtige Punkte oder Zitate die KI-Zusammenfassung (bevorzugen Sie einen Unteragenten, sofern verfügbar), um Hauptkontext-Tokens zu speichern.
