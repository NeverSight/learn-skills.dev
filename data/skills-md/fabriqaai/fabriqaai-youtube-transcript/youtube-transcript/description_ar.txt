استرجع نصوص YouTube بدون MCP باستخدام أداة Python CLI. يُستخدم عندما يطلب المستخدم نص النص الكامل أو لغات النص المتوفرة من عنوان URL أو معرف الفيديو على YouTube. بالنسبة للملخصات أو النقاط الرئيسية أو علامات الاقتباس، استخدم تلخيص الذكاء الاصطناعي (يفضل وكيلًا فرعيًا عند توفره) لحفظ الرموز المميزة للسياق الرئيسي.
