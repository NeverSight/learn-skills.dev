Récupérez les transcriptions YouTube sans MCP à l'aide d'un outil Python CLI. À utiliser lorsqu'un utilisateur demande le texte complet de la transcription ou les langues de transcription disponibles à partir d'une URL YouTube ou d'un identifiant vidéo. Pour les résumés, les points clés ou les citations, utilisez le résumé IA (préférez un sous-agent lorsqu'il est disponible) pour enregistrer les jetons du contexte principal.
