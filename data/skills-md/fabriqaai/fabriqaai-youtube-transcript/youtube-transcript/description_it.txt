Recupera le trascrizioni di YouTube senza MCP utilizzando uno strumento CLI Python. Da utilizzare quando un utente richiede il testo della trascrizione completa o le lingue di trascrizione disponibili da un URL di YouTube o un ID video. Per riepiloghi, punti chiave o citazioni, utilizza il riepilogo AI (preferisci un agente secondario quando disponibile) per salvare i token del contesto principale.
