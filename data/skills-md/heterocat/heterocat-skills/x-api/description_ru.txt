Используйте X (Twitter) API v2 для получения и публикации твитов, мультимедиа, информации о пользователях и других данных. Используется для: получения последних твитов конкретного пользователя, поиска твитов, получения информации о пользователе и публикации твитов в виде обычного текста, изображений или видео. Для операций выборки требуется X_BEARER_TOKEN; для публикации требуется набор из четырех частей OAuth 1.0a: X_API_KEY, X_API_SECRET, X_ACCESS_TOKEN, X_ACCESS_TOKEN_SECRET. Срабатывает, когда пользователю необходимо получить или опубликовать твиты в Twitter/X или загрузить медиафайлы (изображения/видео).
