X (Twitter) API v2 を使用して、ツイート、メディア、ユーザー情報などを取得および投稿します。使用例: 指定したユーザーの最新のツイートの取得、ツイートの検索、ユーザー情報の取得、プレーンテキスト/画像/ビデオのツイートの投稿。読み取り操作には X_BEARER_TOKEN が必要です。書き込み操作には、OAuth 1.0a の 4 つのセット (X_API_KEY、X_API_SECRET、X_ACCESS_TOKEN、X_ACCESS_TOKEN_SECRET) が必要です。ユーザーが Twitter/X でツイートを取得または投稿するか、メディア ファイル (画像/ビデオ) をアップロードする必要があるときにトリガーされます。
