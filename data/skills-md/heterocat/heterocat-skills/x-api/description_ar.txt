استخدم X (Twitter) API v2 لجلب ونشر التغريدات والوسائط ومعلومات المستخدم وما إلى ذلك. من أجل: استرداد أحدث التغريدات لمستخدم محدد، والبحث في التغريدات، والحصول على معلومات المستخدم، ونشر تغريدات بنص عادي/صورة/فيديو. تتطلب عمليات القراءة X_BEARER_TOKEN؛ تحتاج عمليات الكتابة إلى مجموعة OAuth 1.0a المكونة من أربع قطع: X_API_KEY وX_API_SECRET وX_ACCESS_TOKEN وX_ACCESS_TOKEN_SECRET. يتم تشغيله عندما يحتاج المستخدم إلى جلب التغريدات أو نشرها على Twitter/X أو تحميل ملفات الوسائط (الصور/مقاطع الفيديو).
