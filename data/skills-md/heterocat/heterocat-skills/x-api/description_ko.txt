X(Twitter) API v2를 사용하여 트윗, 미디어, 사용자 정보 및 기타 데이터를 가져오고 게시합니다. 용도: 특정 사용자의 최신 트윗 검색, 트윗 검색, 사용자 정보 가져오기, 일반 텍스트/이미지/비디오 트윗 게시. 가져오기 작업에는 X_BEARER_TOKEN이 필요합니다. 게시 작업에는 X_API_KEY, X_API_SECRET, X_ACCESS_TOKEN, X_ACCESS_TOKEN_SECRET의 OAuth 1.0a 4개 세트가 필요합니다. 사용자가 Twitter/X에서 트윗을 받거나 게시해야 하거나 미디어 파일(이미지/비디오)을 업로드해야 할 때 트리거됩니다.
