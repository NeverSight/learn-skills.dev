Use X (Twitter) API v2 to fetch and post tweets, media, user info, etc. Use cases: retrieve a specified user’s latest tweets, search tweets, get user info, post plain-text/image/video tweets. Read operations require X_BEARER_TOKEN; write operations require OAuth 1.0a four-piece set: X_API_KEY, X_API_SECRET, X_ACCESS_TOKEN, X_ACCESS_TOKEN_SECRET. Triggered when the user needs to fetch or post tweets on Twitter/X or upload media files (images/videos).
