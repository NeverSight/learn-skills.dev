Utilizza l'API X (Twitter) v2 per recuperare e pubblicare tweet, contenuti multimediali, informazioni sull'utente, ecc. Utilizzato per: recuperare gli ultimi tweet di un utente specificato, cercare tweet, recuperare informazioni sull'utente, pubblicare tweet di testo semplice/immagine/video. Il recupero richiede X_BEARER_TOKEN; la pubblicazione richiede il set di quattro elementi OAuth 1.0a: X_API_KEY, X_API_SECRET, X_ACCESS_TOKEN, X_ACCESS_TOKEN_SECRET. Si attiva quando l'utente deve ricevere o pubblicare tweet su Twitter/X o caricare file multimediali (immagini/video).
