Verwenden Sie die X (Twitter) API v2, um Tweets, Medien, Benutzerinformationen usw. abzurufen und zu veröffentlichen. Anwendungsfälle: Abrufen der neuesten Tweets eines bestimmten Benutzers, Durchsuchen von Tweets, Abrufen von Benutzerinformationen, Veröffentlichen von Klartext-/Bild-/Video-Tweets. Lesevorgänge erfordern X_BEARER_TOKEN; Für Schreibvorgänge ist OAuth 1.0 ein vierteiliger Satz erforderlich: X_API_KEY, X_API_SECRET, X_ACCESS_TOKEN, X_ACCESS_TOKEN_SECRET. Wird ausgelöst, wenn der Benutzer Tweets auf Twitter/X abrufen oder posten oder Mediendateien (Bilder/Videos) hochladen muss.
