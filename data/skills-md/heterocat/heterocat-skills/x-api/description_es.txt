Utilice la API X (Twitter) v2 para buscar y publicar tweets, medios, información de usuario y otros datos. Se utiliza para: recuperar los últimos tweets de un usuario específico, buscar tweets, obtener información del usuario, publicar tweets de texto sin formato, imágenes o videos. La recuperación requiere X_BEARER_TOKEN; la publicación requiere el conjunto de cuatro piezas de OAuth 1.0a: X_API_KEY, X_API_SECRET, X_ACCESS_TOKEN, X_ACCESS_TOKEN_SECRET. Se activa cuando el usuario necesita buscar o publicar tweets en Twitter/X o cargar archivos multimedia (imágenes/videos).
