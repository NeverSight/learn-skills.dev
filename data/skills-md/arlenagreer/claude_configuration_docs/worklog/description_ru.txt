Отслеживайте оплачиваемые часы для клиентов. Этот навык следует использовать, когда пользователь запрашивает регистрацию рабочего времени, запись времени, потраченного на клиентские проекты, просмотр записей рабочего журнала, подсчет общего количества оплачиваемых часов или подведение итогов недавней работы. Автоматически запрашивает недостающую информацию (клиент, часы работы, описание) и проверяет имена клиентов по базе данных клиентов навыка выставления счетов.
