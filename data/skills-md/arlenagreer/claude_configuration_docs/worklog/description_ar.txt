تتبع الساعات القابلة للفوترة للعملاء. يجب استخدام هذه المهارة عندما يطلب المستخدم تسجيل ساعات العمل، أو تسجيل الوقت المنقضي في مشاريع العميل، أو عرض إدخالات سجل العمل، أو حساب إجمالي الساعات القابلة للفوترة، أو تلخيص العمل الأخير. يطالب تلقائيًا بالمعلومات المفقودة (العميل، والساعات، والوصف) ويتحقق من صحة أسماء العملاء مقابل قاعدة بيانات العملاء الخاصة بمهارة الفاتورة.
