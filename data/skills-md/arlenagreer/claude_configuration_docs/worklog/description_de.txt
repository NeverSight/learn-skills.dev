Verfolgen Sie abrechnungsfähige Stunden für Kunden. Diese Fähigkeit sollte verwendet werden, wenn der Benutzer Arbeitsstunden protokollieren, für Kundenprojekte aufgewendete Zeit aufzeichnen, Arbeitsprotokolleinträge anzeigen, die gesamten abrechnungsfähigen Stunden berechnen oder die letzten Arbeiten zusammenfassen möchte.
