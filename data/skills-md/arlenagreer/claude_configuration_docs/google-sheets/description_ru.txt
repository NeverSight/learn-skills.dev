Управляйте Google Таблицами с помощью комплексных операций с электронными таблицами, включая чтение и запись значений ячеек, формул, форматирование, управление листами и пакетные операции. Используйте для операций с данными электронных таблиц, диапазонов ячеек, формул, форматирования, пакетных обновлений и рабочих процессов анализа данных. Совместно использует токен OAuth с электронной почтой, календарем, контактами, диском и документами.
