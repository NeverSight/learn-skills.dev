Gestisci progetti e attività OmniFocus a livello di codice. Questa competenza deve essere utilizzata durante la creazione, la lettura, l'aggiornamento o l'elenco di attività e progetti di OmniFocus. Supporta tutte le versioni di OmniFocus tramite il rilevamento del metodo di automazione intelligente (Omni Automation → AppleScript → SQLite fallback di sola lettura). Utilizzare per flussi di lavoro di gestione delle attività, query di progetto, monitoraggio del completamento delle attività e operazioni sui dati OmniFocus.
