Управляйте проектами и задачами OmniFocus программным способом. Этот навык следует использовать при создании, чтении, обновлении или составлении списка задач и проектов OmniFocus. Поддерживает все версии OmniFocus благодаря интеллектуальному обнаружению методов автоматизации (Omni Automation → AppleScript → резервный вариант SQLite только для чтения). Используйте для рабочих процессов управления задачами, запросов к проектам, отслеживания выполнения задач и операций с данными OmniFocus.
