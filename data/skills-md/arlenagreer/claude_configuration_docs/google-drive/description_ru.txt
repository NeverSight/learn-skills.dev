Управляйте файлами и папками Google Диска с помощью полных операций CRUD с помощью сценариев Ruby. Используйте для операций с хранилищем файлов, организации папок, разрешений на общий доступ, поиска на Диске и загрузки/скачивания файлов любого типа. Беспрепятственно работает с Google-таблицами и Google-документами для полной интеграции с Google Workspace.
