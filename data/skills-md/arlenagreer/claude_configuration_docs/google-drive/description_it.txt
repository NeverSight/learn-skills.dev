Gestisci file e cartelle di Google Drive con operazioni CRUD complete tramite script Ruby. Utilizzare per operazioni di archiviazione di file, organizzazione di cartelle, autorizzazioni di condivisione, ricerca su Drive e caricamento/download di file di qualsiasi tipo. Funziona perfettamente con le competenze di Fogli Google e Documenti Google per un'integrazione completa di Google Workspace.
