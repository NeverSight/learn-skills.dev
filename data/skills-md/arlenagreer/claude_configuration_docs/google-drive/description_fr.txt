Gérez les fichiers et dossiers Google Drive avec des opérations CRUD complètes via des scripts Ruby. À utiliser pour les opérations de stockage de fichiers, l'organisation des dossiers, les autorisations de partage, la recherche dans Drive et le téléchargement/téléchargement de fichiers de tout type. Fonctionne de manière transparente avec les compétences Google Sheets et Google Docs pour une intégration complète de Google Workspace.
