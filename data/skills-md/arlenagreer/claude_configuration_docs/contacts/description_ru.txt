Управляйте контактами Google с помощью полных операций CRUD с помощью скриптов Ruby. Этот навык следует использовать при поиске, просмотре, создании, обновлении или удалении контактов в Google Контактах. Поддерживает поиск контактов, пакетные операции и комплексное управление полями контактов, включая имена, адреса электронной почты, телефоны, адреса, дни рождения, организации и заметки.
