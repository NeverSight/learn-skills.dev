Envoyez des messages texte via l'application Apple Messages avec recherche automatique de contacts et prise en charge des pièces jointes. Cette compétence doit être utilisée lors de l'envoi de SMS/iMessage aux contacts. Prend en charge la recherche de destinataire basée sur le nom via l'intégration de Google Contacts, les pièces jointes d'images/fichiers et gère les invites de destinataires/messages manquants. CRITIQUE - Les messages sont TOUJOURS envoyés individuellement à chaque destinataire, JAMAIS sous forme de messages de groupe. EXIGE le format téléphonique E.164 (+1XXXXXXXXXX) pour une livraison fiable. Intègre le guide de style d'écriture d'Arlen pour un message authentique.
