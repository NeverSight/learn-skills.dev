自動連絡先検索と添付ファイルのサポートを備えた Apple メッセージ アプリ経由でテキスト メッセージを送信します。このスキルは、SMS/iMessage を連絡先に送信するときに使用する必要があります。 Google コンタクトの統合による名前ベースの受信者の検索、画像/ファイルの添付をサポートし、受信者/メッセージの欠落プロンプトを処理します。重要 - メッセージは常に各受信者に個別に送信され、グループ メッセージとして送信されることはありません。信頼性の高い配信のためには、E.164 電話形式 (+1XXXXXXXXXX) が必要です。本格的なメッセージングのための Arlen の文章スタイル ガイドを統合します。
