Envíe mensajes de texto a través de la aplicación Apple Messages con búsqueda automática de contactos y soporte para archivos adjuntos. Esta habilidad debe usarse al enviar SMS/iMessage a contactos. Admite la búsqueda de destinatarios basada en el nombre a través de la integración de Contactos de Google, archivos adjuntos de imágenes/archivos y maneja indicaciones de destinatarios/mensajes faltantes. CRÍTICO: los mensajes SIEMPRE se envían individualmente a cada destinatario, NUNCA como mensajes grupales. REQUIERE formato de teléfono E.164 (+1XXXXXXXXXX) para una entrega confiable. Integra la guía de estilo de escritura de Arlen para mensajes auténticos.
