Отправляйте текстовые сообщения через приложение Apple Messages с автоматическим поиском контактов и поддержкой вложений. Этот навык следует использовать при отправке SMS/iMessage контактам. Поддерживает поиск получателей по имени через интеграцию с Google Контактами, вложения изображений/файлов и обрабатывает запросы на отсутствие получателей/сообщений. ВАЖНО. Сообщения ВСЕГДА отправляются индивидуально каждому получателю, НИКОГДА как групповые сообщения. Для надежной доставки ТРЕБУЕТСЯ телефонный формат E.164 (+1XXXXXXXXXX). Интегрирует руководство по стилю письма Арлена для достоверного обмена сообщениями.
