Web テスト、フォーム入力、スクリーンショット、データ抽出のためのブラウザー操作を自動化します。ユーザーが Web サイトの移動、Web ページの操作、フォームの入力、スクリーンショットの撮影、Web アプリケーションのテスト、Web ページからの情報の抽出、Web アプリケーションのデバッグ、ブラウザー セッションのビデオとしての記録、API リクエストのモックまたはインターセプト、ブラウザー Cookie/localStorage の管理、Playwright テスト コードの生成、実行トレースのキャプチャ、または複数のブラウザー セッションの同時実行を行う必要がある場合に使用します。
