Automatizza le interazioni del browser per test web, compilazione di moduli, screenshot ed estrazione di dati. Utilizzare quando l'utente deve navigare nei siti Web, interagire con le pagine Web, compilare moduli, acquisire screenshot, testare applicazioni Web, estrarre informazioni da pagine Web, eseguire il debug di app Web, registrare sessioni del browser come video, simulare o intercettare richieste API, gestire cookie/localStorage del browser, generare codice di test Playwright, acquisire tracce di esecuzione o eseguire più sessioni del browser contemporaneamente.
