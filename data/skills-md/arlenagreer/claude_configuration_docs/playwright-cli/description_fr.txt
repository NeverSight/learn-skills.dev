Automatise les interactions du navigateur pour les tests Web, le remplissage de formulaires, les captures d'écran et l'extraction de données. À utiliser lorsque l'utilisateur a besoin de naviguer sur des sites Web, d'interagir avec des pages Web, de remplir des formulaires, de prendre des captures d'écran, de tester des applications Web, d'extraire des informations de pages Web, de déboguer des applications Web, d'enregistrer des sessions de navigateur sous forme de vidéo, de simuler ou d'intercepter des requêtes API, de gérer les cookies/localStorage du navigateur, de générer du code de test Playwright, de capturer des traces d'exécution ou d'exécuter plusieurs sessions de navigateur simultanément.
