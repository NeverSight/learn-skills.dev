Gestisci i file di documentazione in un vault Obsidian tramite la CLI ufficiale di Obsidian (v1.12.4+) o l'accesso diretto ai file. Da utilizzare durante la creazione di elementi di sessione, lo spostamento di documenti di pianificazione nel deposito, l'aggiornamento degli stati del ciclo di vita dei documenti (attivo/elaborato/archiviato), la ricerca nel deposito per tag o contenuto, l'automazione dell'archiviazione di vecchi documenti, la lettura/aggiunta di note giornaliere, la gestione di proprietà e tag o l'integrazione con flussi di lavoro di avvio attività/conclusione attività.
