Guía para configurar contenedores de desarrollo VS Code seguros para agentes de codificación. Úselo al crear o reforzar un DevContainer para aislar Claude Code u otros agentes de codificación, configurar servidores proxy de socket Docker, manejar vectores de escape IPC de VS Code, configurar el soporte de git worktree o verificar controles de seguridad. Cubre el modelo de amenazas, la arquitectura de defensa de tres capas, la configuración de Node.js/pnpm y las pruebas de verificación.
