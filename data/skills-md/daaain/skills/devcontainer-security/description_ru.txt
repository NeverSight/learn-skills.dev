Руководство по настройке защищенных контейнеров разработки VS Code для агентов кодирования. Используйте при создании или усилении защиты DevContainer для изолированной среды Claude Code или других агентов кодирования, настройке прокси-серверов сокетов Docker, обработке escape-векторов VS Code IPC, настройке поддержки рабочего дерева git или проверке элементов управления безопасностью. Охватывает модель угроз, трехуровневую архитектуру защиты, настройку Node.js/pnpm и проверочное тестирование.
