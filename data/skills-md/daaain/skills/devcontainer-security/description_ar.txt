دليل لإعداد حاويات تطوير VS Code الآمنة لوكلاء الترميز. يُستخدم عند إنشاء DevContainer أو تقويته لوضع الحماية Claude Code أو عوامل الترميز الأخرى، أو تكوين وكلاء مأخذ توصيل Docker، أو التعامل مع متجهات الهروب VS Code IPC، أو إعداد دعم شجرة عمل git، أو التحقق من عناصر التحكم في الأمان. يغطي نموذج التهديد، وبنية الدفاع ثلاثية الطبقات، وإعداد Node.js/pnpm، واختبار التحقق.
