Guida per la configurazione di contenitori di sviluppo VS Code protetti per agenti di codifica. Da utilizzare durante la creazione o il rafforzamento di un DevContainer per il sandboxing di Claude Code o altri agenti di codifica, la configurazione dei proxy socket Docker, la gestione dei vettori di escape IPC VS Code, l'impostazione del supporto git worktree o la verifica dei controlli di sicurezza. Copre il modello delle minacce, l'architettura di difesa a tre livelli, la configurazione di Node.js/pnpm e i test di verifica.
