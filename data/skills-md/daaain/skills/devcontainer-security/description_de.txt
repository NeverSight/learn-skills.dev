Leitfaden zum Einrichten gesicherter VS Code-Entwicklungscontainer für Codierungsagenten. Verwenden Sie diese Option, wenn Sie einen DevContainer erstellen oder härten, um Claude Code oder andere Codierungsagenten zu sandboxen, Docker-Socket-Proxys konfigurieren, VS Code IPC-Escape-Vektoren verarbeiten, Git-Worktree-Unterstützung einrichten oder Sicherheitskontrollen überprüfen. Behandelt das Bedrohungsmodell, die dreischichtige Verteidigungsarchitektur, die Node.js/pnpm-Einrichtung und Verifizierungstests.
