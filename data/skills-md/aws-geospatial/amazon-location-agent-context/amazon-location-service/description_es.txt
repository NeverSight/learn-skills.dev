Integra las API de Amazon Location Service para aplicaciones de AWS. Utilice esta habilidad cuando los usuarios quieran agregar mapas (MapLibre interactivo o imágenes estáticas); geocodificar direcciones en coordenadas o invertir geocodificar coordenadas en direcciones; calcular rutas, tiempos de viaje o áreas de servicio; encuentre lugares y negocios mediante búsqueda de texto, búsqueda cercana o sugerencias de autocompletar; recuperar información detallada del lugar, incluidos horarios, contactos y direcciones; monitorear los límites geográficos con geocercas; o rastrear ubicaciones de dispositivos. Cubre la autenticación, la integración de SDK y todas las capacidades del servicio de ubicación de Amazon.
