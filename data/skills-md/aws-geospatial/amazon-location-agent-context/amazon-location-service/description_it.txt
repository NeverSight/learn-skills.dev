Integra le API di Amazon Location Service per le applicazioni AWS. Utilizzare questa abilità quando gli utenti desiderano aggiungere mappe (MapLibre interattive o immagini statiche); geocodificare gli indirizzi in coordinate o invertire le coordinate geocodifica in indirizzi; calcolare percorsi, tempi di percorrenza o aree di servizio; trovare luoghi e attività commerciali tramite la ricerca testuale, la ricerca nelle vicinanze o i suggerimenti di completamento automatico; recuperare informazioni dettagliate sul luogo inclusi orari, contatti e indirizzi; monitorare i confini geografici con geofence; o tenere traccia delle posizioni dei dispositivi. Copre l'autenticazione, l'integrazione dell'SDK e tutte le funzionalità di Amazon Location Service.
