يدمج واجهات برمجة تطبيقات Amazon Location Service لتطبيقات AWS. استخدم هذه المهارة عندما يريد المستخدمون إضافة خرائط (MapLibre التفاعلية أو الصور الثابتة)؛ عناوين الترميز الجغرافي إلى الإحداثيات أو عكس إحداثيات الترميز الجغرافي إلى العناوين؛ حساب الطرق، وأوقات السفر، أو مناطق الخدمة؛ ابحث عن الأماكن والشركات من خلال البحث النصي، أو البحث القريب، أو اقتراحات الإكمال التلقائي؛ استرداد معلومات المكان التفصيلية بما في ذلك ساعات العمل وجهات الاتصال والعناوين؛ مراقبة الحدود الجغرافية باستخدام السياج الجغرافي؛ أو تتبع مواقع الجهاز. يغطي المصادقة وتكامل SDK وجميع إمكانيات Amazon Location Service.
