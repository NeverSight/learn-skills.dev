Создавайте проверяемые болевые гипотезы на основе контекстного файла компании (ICP, случаи выигрыша, знания о продукте) и вводимых пользователем данных. Быстро, ключи API не нужны — чистое рассуждение. Выводит набор гипотез с углами поиска, которые непосредственно определяют запросы при построении списка. Находится между построением контекста и построением списков. Запускается по: «построить гипотезы», «набор гипотез», «больные гипотезы», «определить гипотезы», «какие болевые точки», «углы кампании», «углы поиска», «уточнить гипотезы».
