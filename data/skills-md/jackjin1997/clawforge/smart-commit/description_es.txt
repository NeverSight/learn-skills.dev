Generación de mensajes de confirmación de git impulsada por IA. Utiliza LLM para resumir cambios y crear mensajes de confirmación significativos. Se activa cuando el usuario desea confirmar cambios, modificar/aplastar confirmaciones o necesita LLM para resumir los cambios de los mensajes de confirmación. Activadores chinos: 提交, 提交代码, 提交更改, 提交修改, 要提交, commit, 提交信息, git 提交, 生成提交信息.
