Generazione di messaggi Git Commit basata sull'intelligenza artificiale. Utilizza LLM per riepilogare le modifiche e creare messaggi di commit significativi. Si attiva quando l'utente desidera confermare le modifiche, modificare/eliminare i commit o necessita che LLM riepiloghi le modifiche per i messaggi di commit. Trigger cinesi: 提交, 提交代码, 提交更改, 提交修改, 要提交, commit, 提交信息, git 提交, 生成提交信息.
