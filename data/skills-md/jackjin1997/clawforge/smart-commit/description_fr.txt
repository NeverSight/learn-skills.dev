Génération de messages de validation git basée sur l'IA. Utilise LLM pour résumer les modifications et créer des messages de validation significatifs. Se déclenche lorsque l'utilisateur souhaite valider les modifications, modifier/écraser les validations ou a besoin de LLM pour résumer les modifications pour les messages de validation. Déclencheurs chinois : 提交, 提交代码, 提交更改, 提交修改, 要提交, commit, 提交信息, git 提交, 生成提交信息.
