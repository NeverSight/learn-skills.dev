Генерация сообщений о фиксации git на базе искусственного интеллекта. Использует LLM для обобщения изменений и создания содержательных сообщений о фиксации. Срабатывает, когда пользователь хочет зафиксировать изменения, изменить/сжать коммиты или ему нужен LLM для суммирования изменений для сообщений о фиксации. Китайские триггеры: 提交, 提交代码, 提交更改, 提交修改, 要提交, commit, 提交信息, git 提交, 生成提交信息.
