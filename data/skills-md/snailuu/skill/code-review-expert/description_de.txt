Codeüberprüfung durch Experten mit der Linse eines leitenden Ingenieurs. Wird ausgelöst, wenn der Benutzer „Codeüberprüfung“, „Code überprüfen“, „Codeüberprüfung“, „Änderungen überprüfen“ oder „Codequalität prüfen“ sagt. Gilt nicht für das Schreiben von neuem Code, das Beheben von Fehlern, Refactoring und andere Entwicklungsvorgänge.
