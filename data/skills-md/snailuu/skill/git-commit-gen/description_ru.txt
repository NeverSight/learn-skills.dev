Автоматически генерировать стандартизированные сообщения коммитов на китайском языке на основе статуса git и различий git, поддерживая формат обычных коммитов и включая имена модулей. Срабатывает, когда пользователь говорит «создать фиксацию», «сгенерировать сообщение о фиксации», «сообщение о фиксации», «помогите мне написать фиксацию», «информацию о фиксации» или «git commit». Не применимо для просмотра журнала git, переключения ветвей, разрешения конфликтов слияния или других общих операций git.
