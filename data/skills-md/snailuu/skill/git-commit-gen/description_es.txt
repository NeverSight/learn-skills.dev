Genere automáticamente mensajes de confirmación chinos estandarizados basados ​​en el estado de git y git diff, admitiendo el formato de confirmaciones convencionales con nombres de módulos. Se activa cuando el usuario dice "generar confirmación", "generar mensaje de confirmación", "mensaje de confirmación", "ayúdame a escribir la confirmación", "información de confirmación" o "git commit". No aplicable para: ver el registro de git, cambiar de rama, resolver conflictos de fusión u otras operaciones generales de git.
