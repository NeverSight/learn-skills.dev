Genera automaticamente messaggi di commit cinesi standardizzati in base allo stato git e al diff git, supportando il formato Commit convenzionale con i nomi dei moduli. Si attiva quando l'utente dice "genera commit", "genera messaggio di commit", "messaggio di commit", "aiutami a scrivere commit", "informazioni sul commit" o "git commit". Non applicabile per: visualizzazione del log git, cambio di ramo, risoluzione di conflitti di unione o altre operazioni git generali.
