Générez automatiquement des messages de validation chinois standardisés basés sur le statut git et git diff, prenant en charge le format Conventional Commits et incluant le nom du module. Déclenché lorsque l'utilisateur dit « générer un commit », « générer un message de commit », « un message de commit », « aidez-moi à écrire un commit », « des informations sur le commit » ou « git commit ». Ne s'applique pas à : l'affichage du journal git, le changement de branche, la résolution de conflits de fusion ou d'autres opérations git générales.
