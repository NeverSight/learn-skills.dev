Aggiungi file/cartella all'area locale di git ignore (.git/info/exclude) senza modificare il file .gitignore. Si attiva quando l'utente dice "ignora file", "ignora locale", "git esclude", "non impegnare", "escludi file", "aggiungi da ignorare". Non applicabile per: modifica di .gitignore, git rm e altre operazioni generali di git.
