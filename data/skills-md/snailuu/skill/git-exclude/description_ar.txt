أضف ملفات/مجلدات إلى منطقة تجاهل git المحلية (.git/info/exclude) دون تعديل ملف .gitignore. يتم تشغيله عندما يقول المستخدم "تجاهل الملف"، أو "التجاهل المحلي"، أو "استبعاد git"، أو "عدم الالتزام"، أو "استبعاد ملف"، أو "إضافة إلى التجاهل". غير قابل للتطبيق: تعديل .gitignore أو git rm أو عمليات git العامة الأخرى.
