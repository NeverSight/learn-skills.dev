Ajoutez un fichier/dossier à la zone git ignore locale (.git/info/exclude) sans modifier le fichier .gitignore. Déclenché lorsque l'utilisateur dit "ignorer le fichier", "ignorer local", "git exclure", "ne pas valider", "exclure le fichier", "ajouter pour ignorer". Non applicable pour : la modification de .gitignore, git rm et d'autres opérations générales de git.
