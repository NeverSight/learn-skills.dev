Fügen Sie eine Datei/einen Ordner zum lokalen Git-Ignorierbereich (.git/info/exclude) hinzu, ohne die .gitignore-Datei zu ändern. Wird ausgelöst, wenn der Benutzer „Datei ignorieren“, „Lokal ignorieren“, „Git-Ausschluss“, „Nicht festschreiben“, „Datei ausschließen“ oder „Zum Ignorieren hinzufügen“ sagt. Gilt nicht für: Ändern von .gitignore, git rm und anderen allgemeinen Git-Vorgängen.
