Agregue archivos/carpetas al área local de ignorar de git (.git/info/exclude) sin modificar el archivo .gitignore. Se activa cuando el usuario dice "ignorar archivo", "ignorar local", "git excluir", "no confirmar", "excluir archivo" o "agregar para ignorar". No se aplica a: modificar .gitignore, git rm u otras operaciones generales de git.
