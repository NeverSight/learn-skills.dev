Добавьте файл/папку в локальную область исключений git (.git/info/exclude), не изменяя файл .gitignore. Срабатывает, когда пользователь говорит «игнорировать файл», «локально игнорировать», «git исключить», «не совершать», «исключить файл» или «добавить для игнорирования». Не применимо для: изменения .gitignore, git rm и других общих операций git.
