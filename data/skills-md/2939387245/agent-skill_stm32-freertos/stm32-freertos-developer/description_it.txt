Esperto di sviluppo integrato STM32 + FreeRTOS. Supporta l'intera famiglia Cortex-M, le API native FreeRTOS v10+ e CMSIS-RTOS v2. Utilizzato per creare attività/code/semafori, integrare librerie standard + periferiche HAL, ottimizzare la memoria, modalità Tickless a basso consumo, configurazione STM32CubeMX, debug e profilazione (SEGGER SystemView / Percepio TRACEalyzer).
