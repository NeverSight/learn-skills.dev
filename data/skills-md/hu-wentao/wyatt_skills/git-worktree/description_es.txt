Cree un árbol de trabajo de git para el repositorio actual en el mismo nivel de directorio que la raíz del proyecto. Esta habilidad automatiza la creación de ramas, la denominación de directorios según el formato proyecto-T-rama y la configuración inicial del proyecto (por ejemplo, instalación de dependencias). Úselo cuando el usuario quiera trabajar en una nueva función o corregirla sin cambiar su espacio de trabajo actual.
