قم بإنشاء شجرة عمل git للمستودع الحالي على نفس مستوى الدليل مثل جذر المشروع. تعمل هذه المهارة على أتمتة إنشاء الفرع، وتسمية الدليل وفقًا لتنسيق فرع المشروع T، وإعداد المشروع الأولي (على سبيل المثال، تثبيت التبعية). استخدم هذا عندما يريد المستخدم العمل على ميزة جديدة أو إصلاحها دون تبديل مساحة العمل الحالية.
