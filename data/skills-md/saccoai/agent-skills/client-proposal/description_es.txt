Generar una propuesta de proyecto profesional a partir de una auditoría de un sitio web. Analiza el sitio actual del cliente potencial, identifica problemas y produce una propuesta estructurada con alcance, entregables, recomendaciones técnicas y cronograma por fases. Úselo como herramienta de ventas o para determinar el alcance de las interacciones con los clientes.
