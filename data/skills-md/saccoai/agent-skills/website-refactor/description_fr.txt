Workflow de refactorisation de site Web de bout en bout à l’aide d’équipes d’agents. Orchestre les compétences d’extraction de contenu, d’audit Web et de migration SEO en tant que coéquipiers spécialisés aux côtés d’un concepteur. À utiliser lorsque l'utilisateur souhaite reconstruire, reconcevoir ou migrer un site Web existant.
