Устанавливает PowerShell Universal в изолированной среде, что позволяет проводить изолированное тестирование и разработку пользовательских компонентов, не затрагивая глобальную конфигурацию системы. Этот навык позволяет настроить локальный экземпляр PowerShell Universal с минимальной конфигурацией, идеально подходящий для разработки и экспериментирования.
