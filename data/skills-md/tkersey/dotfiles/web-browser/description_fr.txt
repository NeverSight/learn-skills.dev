À utiliser lorsque les tâches nécessitent une automatisation Web dans un navigateur réel dans Chrome/Chromium via CDP : ouvrez ou parcourez des URL, cliquez/saisissez/sélectionnez dans des formulaires, exécutez la page JS, attendez des sélecteurs, récupérez du contenu structuré, capturez des captures d'écran, validez les flux d'interface utilisateur ou exécutez des contrôles de latence mesurés du navigateur Web (`bench:eval`, `bench:all`) pour les régressions de performances.
