Úselo cuando las tareas necesiten automatización web del navegador real en Chrome/Chromium a través de CDP: abra o navegue por URL, haga clic/escriba/seleccione formularios, ejecute JS de página, espere selectores, extraiga contenido estructurado, capture capturas de pantalla, valide flujos de interfaz de usuario o ejecute comprobaciones medidas de latencia del navegador web (`bench:eval`, `bench:all`) para regresiones de rendimiento.
