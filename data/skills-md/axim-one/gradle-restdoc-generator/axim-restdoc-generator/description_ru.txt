Автоматически создавайте документацию REST API из контроллеров Spring Boot с помощью плагина Axim Gradle RestDoc Generator. Используйте при настройке DSL restMetaGenerator, написании тегов Javadoc (@response, @group, @auth, @error, @header), создании спецификаций OpenAPI 3.0.3, spec-bundle.json, сканировании кода ошибок, синхронизации Postman или устранении неполадок плагина gradle-restdoc-generator.
