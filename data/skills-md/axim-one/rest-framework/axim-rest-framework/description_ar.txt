قم ببناء واجهات برمجة تطبيقات Spring Boot REST باستخدام إطار عمل Axim REST. يُستخدم عند إنشاء كيانات أو مستودعات أو خدمات أو وحدات تحكم أو معالجة الأخطاء أو ترقيم الصفحات باستخدام إطار عمل axim-rest (Spring Boot + MyBatis). يغطي @XEntity، و@XRepository، وIXRepository، واشتقاق الاستعلام، والحفظ/التعديل/upsert، وXPagination، وXPage، ورموز الخطأ، واستثناءات i18n، وعميل REST التعريفي.
