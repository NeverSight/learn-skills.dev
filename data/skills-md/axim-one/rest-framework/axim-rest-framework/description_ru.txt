Создавайте REST API Spring Boot с помощью Axim REST Framework. Используйте при создании сущностей, репозиториев, сервисов, контроллеров, обработки ошибок или разбиения на страницы с помощью axim-rest-framework (Spring Boot + MyBatis). Охватывает @XEntity, @XRepository, IXRepository, деривацию запросов, сохранение/изменение/обновление, XPagination, XPage, коды ошибок, исключения i18n и декларативный клиент REST.
