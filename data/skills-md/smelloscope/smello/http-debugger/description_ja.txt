Smello によってキャプチャされた HTTP リクエストをデバッグします。ユーザーがトラフィックの検査、API 呼び出しのデバッグ、失敗したリクエストのトラブルシューティング、応答本文の分析、またはコードがどのようなリクエストを行っているかを理解することを要求する場合に使用します。ユーザーが http://localhost:5110/#<uuid> や http://localhost:5111/#<uuid> などの Smello ダッシュボード URL を貼り付けるときにも使用します。ハッシュの後の UUID をリクエスト ID として抽出します。 Google Cloud ライブラリからの gRPC 呼び出しをサポートします。実行中の Smello サーバーが必要です。
