Déboguer les requêtes HTTP capturées par Smello. À utiliser lorsque l'utilisateur demande à inspecter le trafic, à déboguer les appels d'API, à résoudre les requêtes ayant échoué, à analyser les corps de réponse ou à comprendre les requêtes émises par leur code. À utiliser également lorsque l'utilisateur colle une URL de tableau de bord Smello comme http://localhost:5110/#<uuid> ou http://localhost:5111/#<uuid> — extrayez l'UUID après le hachage comme ID de demande. Prend en charge les appels gRPC à partir des bibliothèques Google Cloud. Nécessite un serveur Smello en cours d'exécution.
