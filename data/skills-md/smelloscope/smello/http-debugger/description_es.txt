Depurar solicitudes HTTP capturadas por Smello. Úselo cuando el usuario solicite inspeccionar el tráfico, depurar llamadas API, solucionar problemas de solicitudes fallidas, analizar cuerpos de respuesta o comprender qué solicitudes realiza su código. Úselo también cuando el usuario pegue una URL del panel de Smello como http://localhost:5110/#<uuid> o http://localhost:5111/#<uuid>: extraiga el UUID después del hash como ID de solicitud. Admite llamadas gRPC desde bibliotecas de Google Cloud. Requiere un servidor Smello en ejecución.
