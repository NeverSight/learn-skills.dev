Debug delle richieste HTTP catturate da Smello. Da utilizzare quando l'utente chiede di ispezionare il traffico, eseguire il debug delle chiamate API, risolvere i problemi relativi alle richieste non riuscite, analizzare i corpi delle risposte o comprendere quali richieste sta effettuando il codice. Utilizzalo anche quando l'utente incolla un URL del dashboard Smello come http://localhost:5110/#<uuid> o http://localhost:5111/#<uuid>: estrai l'UUID dopo l'hash come ID della richiesta. Supporta le chiamate gRPC dalle librerie Google Cloud. Richiede un server Smello in esecuzione.
