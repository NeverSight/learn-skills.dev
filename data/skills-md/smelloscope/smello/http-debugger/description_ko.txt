Smello에서 캡처한 HTTP 요청을 디버그합니다. 사용자가 트래픽 검사, API 호출 디버그, 실패한 요청 문제 해결, 응답 본문 분석 또는 코드에서 수행하는 요청 이해를 요청할 때 사용합니다. 또한 사용자가 http://localhost:5110/#<uuid> 또는 http://localhost:5111/#<uuid>와 같은 Smello 대시보드 URL을 붙여 넣을 때 사용합니다. 해시 뒤의 UUID를 요청 ID로 추출합니다. Google Cloud 라이브러리의 gRPC 호출을 지원합니다. 실행 중인 Smello 서버가 필요합니다.
