تصحيح طلبات HTTP التي تم التقاطها بواسطة Smello. يُستخدم عندما يطلب المستخدم فحص حركة المرور، أو تصحيح أخطاء استدعاءات واجهة برمجة التطبيقات (API)، أو استكشاف أخطاء الطلبات الفاشلة وإصلاحها، أو تحليل أجسام الاستجابة، أو فهم الطلبات التي يقدمها الكود الخاص به. يُستخدم أيضًا عندما يقوم المستخدم بلصق عنوان URL للوحة تحكم Smello مثل http://localhost:5110/#<uuid> أو http://localhost:5111/#<uuid> - قم باستخراج UUID بعد التجزئة كمعرف الطلب. يدعم مكالمات gRPC من مكتبات Google Cloud. يتطلب خادم Smello قيد التشغيل.
