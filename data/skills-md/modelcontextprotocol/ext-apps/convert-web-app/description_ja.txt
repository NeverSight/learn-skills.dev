このスキルは、ユーザーが「Web アプリに MCP アプリのサポートを追加する」、「Web アプリをハイブリッド MCP アプリに変える」、「Web ページも MCP アプリとして動作させる」、「既存の UI を MCP アプリとしてラップする」、「iframe 埋め込みを MCP アプリに変換する」、「SPA を MCP アプリに変える」、またはスタンドアロンで動作させたまま既存の Web アプリケーションに MCP アプリのサポートを追加する必要がある場合に使用する必要があります。既存の Web アプリを分析し、サーバー側ツールとリソース登録を使用してハイブリッド Web + MCP アプリを作成するためのガイダンスを提供します。
