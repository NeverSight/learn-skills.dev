Этот навык следует использовать, когда пользователь просит «добавить приложение на мой сервер MCP», «добавить пользовательский интерфейс на мой сервер MCP», «добавить представление к моему инструменту MCP», «обогатить инструменты MCP пользовательским интерфейсом», «добавить интерактивный пользовательский интерфейс на существующий сервер», «добавить приложения MCP на мой сервер» или ему необходимо добавить возможности интерактивного пользовательского интерфейса к существующему серверу MCP, на котором уже есть инструменты. Содержит рекомендации по анализу существующих инструментов и добавлению ресурсов пользовательского интерфейса приложений MCP.
