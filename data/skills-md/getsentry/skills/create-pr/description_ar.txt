استخدم دائمًا هذه المهارة عند إنشاء طلبات السحب، ولا تقم أبدًا بإنشاء علاقات عامة مباشرة بدونها. يتبع اصطلاحات Sentry لعناوين العلاقات العامة والأوصاف ومراجع المشكلات. قم بتشغيل أي إنشاء علاقات عامة، أو فتح علاقات عامة، أو إرسال علاقات عامة، أو إجراء علاقات عامة، أو دفع وإنشاء علاقات عامة، أو إعداد التغييرات لمهمة المراجعة.
