Examen du code de sécurité pour les vulnérabilités. À utiliser lorsqu'on vous demande « examen de sécurité », « trouver des vulnérabilités », « vérifier les problèmes de sécurité », « auditer la sécurité », « révision OWASP » ou examiner le code pour les problèmes d'injection, XSS, d'authentification, d'autorisation et de cryptographie. Fournit un examen systématique avec des rapports basés sur la confiance.
