Создавайте структурированные документы планирования проекта для новых программных проектов, функций или инициатив. Используйте при запуске нового проекта, когда вам необходимо создать спецификации, планы реализации и разбивку задач, которые агенты кодирования ИИ могут использовать для написания кода. Результаты включают спецификацию проекта, этапы реализации и список задач текущих/следующих шагов.
