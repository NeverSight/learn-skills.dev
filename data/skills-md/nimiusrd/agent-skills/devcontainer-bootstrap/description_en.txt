Bootstrap to get Dev Containers up and running or updated in the shortest time. It auto-detects the stack (Node/Python/Rust) and applies templates or performs a safe update. If a .devcontainer already exists, it backs it up before merging. Chat options: stack (auto|node|python|rust), packageManager (npm|pnpm|yarn), mode (safe|overwrite), includeTools (true|false), addCI (true|false).
