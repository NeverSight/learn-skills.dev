Bootstrap per rendere operativi o aggiornati i Dev Containers nel più breve tempo possibile. Rileva automaticamente lo stack (Node/Python/Rust) e applica modelli o esegue un aggiornamento sicuro. Se un .devcontainer esiste già, ne esegue il backup prima di unirlo. Opzioni chat: stack (auto|node|python|rust), packageManager (npm|pnpm|yarn), modalità (safe|overwrite), includeTools (true|false), addCI (true|false).
