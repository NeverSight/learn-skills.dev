Bootstrap pour que les conteneurs de développement soient opérationnels ou mis à jour dans les plus brefs délais. Il détecte automatiquement la pile (Node/Python/Rust) et applique des modèles ou effectue une mise à jour sécurisée. Si un .devcontainer existe déjà, il le sauvegarde avant de le fusionner. Options de discussion : pile (auto|node|python|rust), packageManager (npm|pnpm|yarn), mode (safe|overwrite), includeTools (true|false), addCI (true|false).
