Bootstrap para configurar o actualizar rápidamente Dev Containers. Detecta automáticamente la pila (Node/Python/Rust) y aplica plantillas o realiza actualizaciones seguras. Si ya existe un .devcontainer, realiza una copia de seguridad antes de fusionarlo. Opciones de chat: pila (auto|nodo|python|rust), packageManager (npm|pnpm|yarn), modo (seguro|sobrescribir), includeTools (verdadero|falso), addCI (verdadero|falso).
