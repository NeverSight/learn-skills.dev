Bootstrap для максимально быстрой установки/обновления Dev-контейнеров. Он автоматически определяет стек (node/python/rust) и применяет шаблоны или выполняет безопасные обновления. Если .devcontainer уже существует, он создает его резервную копию перед объединением. Параметры чата: стек (auto|node|python|rust), packageManager (npm|pnpm|yarn), режим (безопасный|перезаписать), includeTools (true|false), addCI (true|false).
