Nettoyez et organisez les scripts et les dépendances dans package.json. À utiliser pour des requêtes telles que « Je souhaite nettoyer package.json », « Je souhaite réorganiser les scripts » ou « Je souhaite supprimer les dépendances inutilisées ». Cible spécifiquement : (1) supprimer ou fusionner des scripts redondants (alias/pass-through), (2) rendre explicites les hooks implicites du cycle de vie (pré/post), (3) unifier les conventions de dénomination des scripts, (4) détecter et supprimer les packages de dépendances inutilisés, (5) mettre à jour la documentation et le CI lorsque les noms de script changent, (6) régénérer les fichiers de verrouillage après la suppression des dépendances.
