動的サーバーレンダリング UI 用の Symfony UX LiveComponent。 JavaScript を記述せずに、AJAX、リアルタイム フォーム、データ バインディング、ライブ検証、またはリアクティブ UI 経由で再レンダリングするインタラクティブ コンポーネントを構築する場合に使用します。トリガー - ライブ コンポーネント、AsLiveComponent、LiveProp、LiveAction、データ モデル、リアルタイム フォーム、動的 UI、AJAX コンポーネント、リアクティブ PHP、双方向バインディング、サーバー再レンダリング、ライブ検索、ライブ フィルター、ライブ検証、ComponentWithFormTrait、エミット、LiveListener、ポーリング、遅延、遅延コンポーネント、データ読み込み、書き込み可能プロップ、URL バインディング、コンポーネント通信。また、ユーザーが JavaScript を記述せずにユーザー入力に基づいて自身を更新するコンポーネントを必要とする場合、または PHP で Vue/React のような反応性を必要とする場合にもトリガーします。
