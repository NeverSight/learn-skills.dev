用于动态服务器渲染 UI 的 Symfony UX LiveComponent。在构建通过 AJAX、实时表单、数据绑定、实时验证或反应式 UI 重新渲染而无需编写 JavaScript 的交互式组件时使用。触发器 - 实时组件、AsLiveComponent、LiveProp、LiveAction、数据模型、实时表单、动态 UI、AJAX 组件、反应式 PHP、双向绑定、服务器重新渲染、实时搜索、实时过滤器、实时验证、ComponentWithFormTrait、emit、LiveListener、轮询、延迟、惰性组件、数据加载、可写属性、URL 绑定、组件通信。当用户想要一个无需编写 JavaScript 即可根据用户输入进行自我更新的组件，或者想要在 PHP 中实现类似 Vue/React 的反应性时也会触发。
