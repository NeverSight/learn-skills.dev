동적 서버 렌더링 UI를 위한 Symfony UX LiveComponent. JavaScript를 작성하지 않고 AJAX, 실시간 양식, 데이터 바인딩, 실시간 유효성 검사 또는 반응형 UI를 통해 다시 렌더링하는 대화형 구성 요소를 구축할 때 사용합니다. 트리거 - 라이브 구성 요소, AsLiveComponent, LiveProp, LiveAction, 데이터 모델, 실시간 양식, 동적 UI, AJAX 구성 요소, 반응형 PHP, 양방향 바인딩, 서버 다시 렌더링, 라이브 검색, 라이브 필터, 라이브 유효성 검사, ComponentWithFormTrait, 내보내기, LiveListener, 폴링, 지연, 지연 구성 요소, 데이터 로딩, 쓰기 가능한 속성, URL 바인딩, 구성 요소 통신. 또한 사용자가 JavaScript를 작성하지 않고 사용자 입력을 기반으로 자체 업데이트되는 구성 요소를 원하거나 PHP에서 Vue/React와 유사한 반응성을 원하는 경우에도 트리거됩니다.
