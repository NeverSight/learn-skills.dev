Symfony UX LiveComponent für dynamische, servergerenderte Benutzeroberfläche. Verwenden Sie diese Option, wenn Sie interaktive Komponenten erstellen, die über AJAX, Echtzeitformulare, Datenbindung, Live-Validierung oder reaktive Benutzeroberfläche neu gerendert werden, ohne JavaScript zu schreiben. Auslöser – Live-Komponente, AsLiveComponent, LiveProp, LiveAction, Datenmodell, Echtzeitformular, dynamische Benutzeroberfläche, AJAX-Komponente, reaktives PHP, bidirektionale Bindung, Server-Re-Rendering, Live-Suche, Live-Filter, Live-Validierung, ComponentWithFormTrait, Emit, LiveListener, Polling, Defer, Lazy Component, Datenladen, beschreibbare Requisite, URL-Bindung, Komponentenkommunikation. Wird auch ausgelöst, wenn der Benutzer eine Komponente wünscht, die sich basierend auf Benutzereingaben aktualisiert, ohne JavaScript zu schreiben, oder eine Vue/React-ähnliche Reaktivität in PHP wünscht.
