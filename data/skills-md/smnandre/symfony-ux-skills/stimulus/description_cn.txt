Symfony UX 的 Stimulus JS 框架。在使用数据属性构建客户端交互性、创建用于 DOM 操作的控制器、处理用户事件、管理组件状态或与 Symfony 的 StimulusBundle 和 AssetMapper 集成时使用。触发器 - 刺激控制器、数据控制器、数据操作、数据目标、前端交互性、JavaScript 行为、Symfony UX 前端、切换、下拉、模态 JS、选项卡 JS、剪贴板、图表控制器、日期选择器、自动完成 JS、惰性控制器、stimulusFetch、插座、键盘快捷键、全局事件侦听器。当用户想要将 JavaScript 行为添加到服务器呈现的 HTML、包装第三方 JS 库或构建不需要服务器往返的仅客户端交互时也会触发。
