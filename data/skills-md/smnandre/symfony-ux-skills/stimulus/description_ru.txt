Фреймворк Stimulus JS для Symfony UX. Используйте при создании интерактивности на стороне клиента с атрибутами данных, создании контроллеров для манипулирования DOM, обработки пользовательских событий, управления состоянием компонентов или интеграции с StimulusBundle и AssetMapper от Symfony. Триггеры - контроллер стимулов, контроллер данных, действие данных, цель данных, интерактивность интерфейса, поведение JavaScript, интерфейс Symfony UX, переключение, раскрывающийся список, модальный JS, вкладки JS, буфер обмена, контроллер диаграмм, указатель даты, автозаполнение JS, ленивый контроллер, стимулFetch, розетки, сочетание клавиш, глобальный прослушиватель событий. Также срабатывает, когда пользователь хочет добавить поведение JavaScript в HTML, отображаемый на сервере, обернуть стороннюю библиотеку JS или создать взаимодействия только с клиентом, которые не требуют двустороннего обращения к серверу.
