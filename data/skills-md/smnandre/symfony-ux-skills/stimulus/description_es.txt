Marco Stimulus JS para Symfony UX. Úselo al crear interactividad del lado del cliente con atributos de datos, crear controladores para la manipulación DOM, manejar eventos de usuario, administrar el estado de los componentes o integrar con StimulusBundle y AssetMapper de Symfony. Activadores: controlador de estímulo, controlador de datos, acción de datos, objetivo de datos, interactividad de interfaz, comportamiento de JavaScript, interfaz de Symfony UX, alternar, menú desplegable, JS modal, pestañas JS, portapapeles, controlador de gráficos, selector de fechas, JS de autocompletar, controlador diferido, estímuloFetch, salidas, método abreviado de teclado, detector de eventos globales. También se activa cuando el usuario desea agregar un comportamiento de JavaScript al HTML renderizado por el servidor, empaquetar una biblioteca JS de terceros o crear interacciones solo para el cliente que no necesitan un viaje de ida y vuelta al servidor.
