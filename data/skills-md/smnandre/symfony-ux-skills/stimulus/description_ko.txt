Symfony UX용 Stimulus JS 프레임워크. 데이터 속성을 사용하여 클라이언트 측 상호 작용을 구축하고, DOM 조작을 위한 컨트롤러를 생성하고, 사용자 이벤트를 처리하고, 구성 요소 상태를 관리하거나 Symfony의 StimulusBundle 및 AssetMapper와 통합할 때 사용합니다. 트리거 - 자극 컨트롤러, 데이터 컨트롤러, 데이터 액션, 데이터 타겟, 프런트엔드 상호 작용, JavaScript 동작, Symfony UX 프런트엔드, 토글, 드롭다운, 모달 JS, 탭 JS, 클립보드, 차트 컨트롤러, 날짜 선택기, 자동 완성 JS, 게으른 컨트롤러, 자극 가져오기, 아웃렛, 키보드 단축키, 전역 이벤트 리스너. 또한 사용자가 서버 렌더링 HTML에 JavaScript 동작을 추가하거나, 타사 JS 라이브러리를 래핑하거나, 서버 왕복이 필요하지 않은 클라이언트 전용 상호 작용을 구축하려는 경우에도 트리거됩니다.
