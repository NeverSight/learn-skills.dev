Symfony UX 用の Stimulus JS フレームワーク。データ属性を使用したクライアント側の対話性の構築、DOM 操作用のコントローラーの作成、ユーザー イベントの処理、コンポーネントの状態の管理、または Symfony の StimulusBundle および AssetMapper との統合を行う場合に使用します。トリガー - スティミュラス コントローラー、データ コントローラー、データ アクション、データ ターゲット、フロントエンドの対話性、JavaScript の動作、Symfony UX フロントエンド、トグル、ドロップダウン、モーダル JS、タブ JS、クリップボード、チャート コントローラー、日付ピッカー、オートコンプリート JS、遅延コントローラー、stimulusFetch、アウトレット、キーボード ショートカット、グローバル イベント リスナー。また、ユーザーがサーバーでレンダリングされた HTML に JavaScript の動作を追加する場合、サードパーティの JS ライブラリをラップする場合、またはサーバーのラウンドトリップを必要としないクライアントのみのインタラクションを構築する場合にもトリガーします。
