Symfony UX 的 Stimulus JS 框架。在使用資料屬性建立用戶端互動性、建立用於 DOM 操作的控制器、處理使用者事件、管理元件狀態或與 Symfony 的 StimulusBundle 和 AssetMapper 整合時使用。觸發器 - 刺激控制器、資料控制器、資料操作、資料目標、前端互動性、JavaScript 行為、Symfony UX 前端、切換、下拉、模態 JS、選項卡 JS、剪貼簿、圖表控制器、日期選擇器、自動完成 JS、惰性控制器、stimulusFetch、插座、快捷快捷鍵、全域事件偵錯器鍵。當使用者想要將 JavaScript 行為新增至伺服器呈現的 HTML、包裝第三方 JS 程式庫或建置不需要伺服器往返的僅客戶端互動時也會觸發。
