Symfony UX TwigComponent لعناصر واجهة المستخدم القابلة لإعادة الاستخدام. يُستخدم عند إنشاء قوالب Twig قابلة لإعادة الاستخدام مع فئات دعم PHP، أو تكوين المكونات، أو الدعائم، أو الفتحات/الكتل، أو الخصائص المحسوبة، أو المكونات المجهولة. المشغلات - مكون twig، AsTwigComponent، قالب قابل لإعادة الاستخدام، دعائم المكون، كتل twig، فتحات المكونات، مكون مجهول، مكون Symfony UX، مكون HTML، مكتبة المكونات، مكون نظام التصميم، مجموعة واجهة المستخدم، زر قابل لإعادة الاستخدام، بطاقة قابلة لإعادة الاستخدام، PreMount، PostMount، طريقة التثبيت. يمكنك أيضًا طرح أي سؤال حول إنشاء جزء قابل لإعادة الاستخدام من واجهة المستخدم في Symfony، حتى لو لم يذكر المستخدم TwigComponent بالاسم.
