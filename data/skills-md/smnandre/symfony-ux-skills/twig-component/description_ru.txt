Symfony UX TwigComponent для многократного использования элементов пользовательского интерфейса. Используйте при создании многоразовых шаблонов Twig с классами поддержки PHP, композицией компонентов, реквизитами, слотами/блоками, вычисляемыми свойствами или анонимными компонентами. Триггеры — компонент ветки, AsTwigComponent, многоразовый шаблон, реквизиты компонента, блоки веток, слоты компонентов, анонимный компонент, компонент Symfony UX, компонент HTML, библиотека компонентов, компонент системы дизайна, набор пользовательского интерфейса, многоразовая кнопка, многоразовая карта, PreMount, PostMount, метод монтирования. Также вызывает любой вопрос о создании многократно используемого элемента пользовательского интерфейса в Symfony, даже если пользователь не упоминает TwigComponent по имени.
