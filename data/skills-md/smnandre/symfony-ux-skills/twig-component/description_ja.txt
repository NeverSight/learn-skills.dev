再利用可能な UI 要素用の Symfony UX TwigComponent。 PHP バッキング クラス、コンポーネント構成、プロップ、スロット/ブロック、計算プロパティ、または匿名コンポーネントを使用して再利用可能な Twig テンプレートを作成する場合に使用します。トリガー - 小枝コンポーネント、AsTwigComponent、再利用可能なテンプレート、コンポーネント プロップ、小枝ブロック、コンポーネント スロット、匿名コンポーネント、Symfony UX コンポーネント、HTML コンポーネント、コンポーネント ライブラリ、デザイン システム コンポーネント、UI キット、再利用可能なボタン、再利用可能なカード、PreMount、PostMount、マウント メソッド。また、ユーザーが TwigComponent について名前を挙げていない場合でも、Symfony での再利用可能な UI の構築に関する質問もトリガーします。
