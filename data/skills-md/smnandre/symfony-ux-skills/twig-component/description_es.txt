Symfony UX TwigComponent para elementos de interfaz de usuario reutilizables. Úselo al crear plantillas Twig reutilizables con clases de respaldo PHP, composición de componentes, accesorios, ranuras/bloques, propiedades calculadas o componentes anónimos. Activadores: componente twig, AsTwigComponent, plantilla reutilizable, accesorios de componentes, bloques twig, ranuras de componentes, componente anónimo, componente Symfony UX, componente HTML, biblioteca de componentes, componente del sistema de diseño, kit de interfaz de usuario, botón reutilizable, tarjeta reutilizable, PreMount, PostMount, método de montaje. También activa cualquier pregunta sobre la creación de una interfaz de usuario reutilizable en Symfony, incluso si el usuario no menciona TwigComponent por su nombre.
