Hotwire Turbo для Symfony UX. Используйте при создании навигации в стиле SPA без JS, частичного обновления страниц с помощью фреймов, обновлений в реальном времени с помощью потоков или интеграции с Mercure для трансляций. Триггеры — турбо-привод, турбо-фрейм, турбо-поток, частичное обновление страницы, ощущение SPA, навигация ajax, обновление в реальном времени, трансляция Mercure, Symfony UX Turbo, встроенное редактирование, раздел отложенной загрузки, фрейм нумерации страниц, модальное окно с сервера, поток флэш-сообщений, многосекционное обновление, TurboStreamResponse, twig:Turbo:Stream, data-turbo, турбо-поток-источник, SSE. Также срабатывает, когда пользователь хочет обновить часть страницы без полной перезагрузки или хочет получать обновления от сервера к браузеру в реальном времени.
