Hotwire Turbo für Symfony UX. Verwenden Sie diese Option, wenn Sie eine SPA-ähnliche Navigation ohne JS, teilweise Seitenaktualisierungen mit Frames, Echtzeitaktualisierungen mit Streams oder die Integration mit Mercure für Übertragungen erstellen. Auslöser – Turbo-Laufwerk, Turbo-Frame, Turbo-Stream, teilweise Seitenaktualisierung, SPA-Feeling, Ajax-Navigation, Echtzeit-Update, Mercure-Broadcast, Symfony UX Turbo, Inline-Edit, Lazy-Load-Abschnitt, Paginierungs-Frame, Modal vom Server, Flash-Nachrichtenstream, Multi-Section-Update, TurboStreamResponse, twig:Turbo:Stream, data-turbo, turbo-stream-source, SSE. Wird auch ausgelöst, wenn der Benutzer einen Teil einer Seite ohne ein vollständiges Neuladen aktualisieren möchte oder Server-zu-Browser-Updates in Echtzeit wünscht.
