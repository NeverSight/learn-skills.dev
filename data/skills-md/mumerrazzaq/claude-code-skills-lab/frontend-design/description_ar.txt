قم بإنشاء واجهات أمامية على مستوى الإنتاج مع تصميم منهجي لواجهة المستخدم/تجربة المستخدم. يُستخدم عند إنشاء مكونات الويب أو الصفحات أو التطبيقات. يغطي أنظمة التصميم، وحالات المكونات، وإمكانية الوصول (WCAG 2.2)، والتصميم سريع الاستجابة، واتفاقيات الصناعة. يقيم واجهة المستخدم الحالية مقابل أفضل الممارسات. يفرض التفكير التصميمي قبل التنفيذ. يُنشئ كودًا مميزًا يتجنب جماليات الذكاء الاصطناعي العامة مع اتباع مبادئ تجربة المستخدم.
