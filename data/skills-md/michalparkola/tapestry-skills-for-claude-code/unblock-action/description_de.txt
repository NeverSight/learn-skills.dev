Helfen Sie dem Benutzer, eine vage oder festgefahrene Aktion zu entsperren, indem Sie die beabsichtigte Ausgabe klarstellen, sie auf heute beschränken und die konkrete nächste Aktion identifizieren. Verwenden Sie diese Option, wenn der Benutzer „Entsperren“, „Entfernen“ oder „Ich stecke fest“ sagt oder eine vage Aufgabe präsentiert, die er nicht starten kann.
