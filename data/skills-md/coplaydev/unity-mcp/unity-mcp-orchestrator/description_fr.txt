Orchestrez Unity Editor via les outils et ressources MCP (Model Context Protocol). À utiliser lorsque vous travaillez avec des projets Unity via MCP pour Unity : création/modification de GameObjects, édition de scripts, gestion de scènes, exécution de tests ou toute automatisation de l'éditeur Unity. Fournit les meilleures pratiques, les schémas d'outils et les modèles de flux de travail pour une intégration efficace de Unity-MCP.
