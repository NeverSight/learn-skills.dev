Orchestra Unity Editor tramite strumenti e risorse MCP (Model Context Protocol). Da utilizzare quando si lavora con progetti Unity tramite MCP per Unity: creazione/modifica di GameObject, modifica di script, gestione di scene, esecuzione di test o qualsiasi automazione di Unity Editor. Fornisce best practice, schemi di strumenti e modelli di flusso di lavoro per un'integrazione efficace Unity-MCP.
