Orchestrieren Sie Unity Editor über MCP-Tools und -Ressourcen (Model Context Protocol). Verwenden Sie es, wenn Sie mit Unity-Projekten über MCP für Unity arbeiten – GameObjects erstellen/ändern, Skripts bearbeiten, Szenen verwalten, Tests ausführen oder jede Unity-Editor-Automatisierung durchführen. Bietet Best Practices, Tool-Schemata und Workflow-Muster für eine effektive Unity-MCP-Integration.
