تنسيق محرر الوحدة عبر أدوات وموارد MCP (بروتوكول السياق النموذجي). يُستخدم عند العمل مع مشاريع Unity من خلال MCP for Unity - إنشاء/تعديل كائنات GameObjects، أو تحرير البرامج النصية، أو إدارة المشاهد، أو تشغيل الاختبارات، أو أي أتمتة لمحرر Unity. يوفر أفضل الممارسات ومخططات الأدوات وأنماط سير العمل لتحقيق التكامل الفعال مع Unity-MCP.
