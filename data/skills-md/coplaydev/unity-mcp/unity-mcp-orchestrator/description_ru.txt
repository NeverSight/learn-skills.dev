Управляйте редактором Unity с помощью инструментов и ресурсов MCP (Model Context Protocol). Используйте при работе с проектами Unity через MCP для Unity — создание/изменение игровых объектов, редактирование сценариев, управление сценами, выполнение тестов или любая автоматизация редактора Unity. Предоставляет лучшие практики, схемы инструментов и шаблоны рабочих процессов для эффективной интеграции Unity-MCP.
