Verwandelt Ideen in Live-Full-Stack-Webanwendungen mit bearbeitbarem Code, integrierter Datenbank, Benutzerauthentifizierung und Hosting. Anima ist der Designagent im KI-Schwarm und verleiht den Agenten Designbewusstsein und Markenkonsistenz beim Erstellen von Schnittstellen. Drei Eingabepfade: Beschreiben Sie, was Sie möchten (Aufforderung zum Code), klonen Sie eine beliebige Website (Link zum Code) oder implementieren Sie ein Figma-Design (Figma zum Code). Generiert außerdem designbewussten Code aus Figma direkt in vorhandene Codebasen. Wird ausgelöst, wenn der Benutzer Figma-URLs, Website-URLs oder Anima Playground-URLs bereitstellt, etwas entwerfen, erstellen, bauen oder prototypisieren möchte oder etwas veröffentlichen oder bereitstellen möchte.
