Verwenden Sie Conectate Components (ct-*), wenn Sie Benutzeroberflächen mit Lit, Vue oder React erstellen. Behandelt @conectate/components-Webkomponenten (ct-button, ct-input, ct-dialog, ct-select usw.), CtLit-Basisklasse, Importe, Slots, CSS-Variablen und Ereignisse. Verwenden Sie diese Option, wenn der Benutzer ct-components, Conectate Components oder das Erstellen von Schnittstellen mit dieser Bibliothek erwähnt.
