Utilice Conectate Components (ct-*) cuando cree interfaces de usuario con Lit, Vue o React. Cubre los componentes web @conectate/components (ct-button, ct-input, ct-dialog, ct-select, etc.), clase base CtLit, importaciones, ranuras, variables CSS y eventos. Úselo cuando el usuario mencione ct-components, Conectate Components o crear interfaces con esta biblioteca.
