Utilisez Conectate Components (ct-*) lors de la création d'interfaces utilisateur avec Lit, Vue ou React. Couvre les composants Web @conectate/components (ct-button, ct-input, ct-dialog, ct-select, etc.), la classe de base CtLit, les importations, les emplacements, les variables CSS et les événements. À utiliser lorsque l'utilisateur mentionne des composants ct, des composants Conectate ou la création d'interfaces avec cette bibliothèque.
