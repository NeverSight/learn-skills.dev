Utilizza Conectate Components (ct-*) quando crei interfacce utente con Lit, Vue o React. Copre i componenti web @conectate/components (ct-button, ct-input, ct-dialog, ct-select, ecc.), classe base CtLit, importazioni, slot, variabili CSS ed eventi. Da utilizzare quando l'utente menziona ct-components, Conectate Components o la creazione di interfacce con questa libreria.
