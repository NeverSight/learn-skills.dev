Используйте компоненты Conectate (ct-*) при создании пользовательских интерфейсов с помощью Lit, Vue или React. Охватывает веб-компоненты @conectate/comComponents (ct-button, ct-input, ct-dialog, ct-select и т. д.), базовый класс CtLit, импорт, слоты, переменные CSS и события. Используйте, когда пользователь упоминает ct-компоненты, компоненты Conectate или создает интерфейсы с помощью этой библиотеки.
