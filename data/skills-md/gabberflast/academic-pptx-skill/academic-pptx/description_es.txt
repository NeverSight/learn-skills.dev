Utilice esta habilidad siempre que el usuario quiera crear o mejorar una presentación para un contexto académico: artículos de conferencias, charlas en seminarios, defensas de tesis, sesiones informativas sobre subvenciones, reuniones de laboratorio, conferencias invitadas o cualquier presentación en la que la audiencia evalúe el razonamiento y la evidencia. Los desencadenantes incluyen: 'charla de conferencia', 'diapositivas de seminario', 'defensa de tesis', 'presentación de investigación', 'presentación académica', 'presentación académica'. También se activa cuando el usuario solicita "hacer diapositivas" en combinación con contenido académico (por ejemplo, "hacer diapositivas para mi artículo sobre X", "crear una presentación para la defensa de mi tesis", "crear una presentación para mi propuesta de subvención"). Esta habilidad gobierna las decisiones de CONTENIDO y ESTRUCTURA. Para el trabajo técnico de crear o editar el archivo .pptx en sí, lea también pptx SKILL.md.
