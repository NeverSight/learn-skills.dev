سمة مسؤول Django Unfold - إنشاء واجهات إدارة Django الحديثة وتكوينها وتحسينها باستخدام Unfold. يُستخدم عند العمل مع: (1) تخصيص واجهة مستخدم Django admin أو تخصيصها، (2) Unfold ModelAdmin، أو السطور، أو الإجراءات، أو المرشحات، أو عناصر واجهة المستخدم، أو أدوات الديكور، (3) مكونات لوحة تحكم المسؤول وبطاقات KPI، (4) التنقل عبر الشريط الجانبي، أو علامات التبويب، أو الحقول الشرطية، (5) أي ذكر لـ "unfold" أو "django-unfold" أو "unfold admin". يغطي مجموعة ميزات Unfold الكاملة: تكوين الموقع، ونظام الإجراءات، وديكورات العرض، وأنواع المرشحات، وتجاوزات عناصر واجهة المستخدم، والمتغيرات المضمنة، ومكونات لوحة المعلومات، ومجموعات البيانات، والأقسام، والسمات، وتكاملات الجهات الخارجية.
