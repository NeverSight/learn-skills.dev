Tema di amministrazione Django Unfold: crea, configura e migliora le moderne interfacce di amministrazione Django con Unfold. Da utilizzare quando si lavora con: (1) personalizzazione o temi dell'interfaccia utente dell'amministratore di Django, (2) Unfold ModelAdmin, inline, azioni, filtri, widget o decoratori, (3) componenti del dashboard di amministrazione e schede KPI, (4) navigazione della barra laterale, schede o campi condizionali, (5) qualsiasi menzione di "unfold", "django-unfold" o "unfold admin". Copre l'intero set di funzionalità di Unfold: configurazione del sito, sistema di azioni, decoratori di visualizzazione, tipi di filtri, sostituzioni di widget, varianti in linea, componenti del dashboard, set di dati, sezioni, temi e integrazioni di terze parti.
