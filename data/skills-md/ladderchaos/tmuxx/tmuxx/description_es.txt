CLI determinista tmuxx + TUI interactiva con información sobre la actividad a nivel de panel. Úselo para la orquestación de tmux, la ejecución de tareas del árbol de trabajo y la visibilidad en tiempo real de qué agentes se están ejecutando, inactivos o bloqueados según la entrada del usuario. CLI con salida JSON para agentes; TUI con indicadores visuales de estado para humanos.
