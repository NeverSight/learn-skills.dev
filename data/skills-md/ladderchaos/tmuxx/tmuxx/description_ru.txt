Детерминированный интерфейс командной строки tmuxx + интерактивный TUI с аналитикой активности на уровне панели. Используйте для оркестровки tmux, выполнения задач рабочего дерева и отслеживания в реальном времени того, какие агенты работают, простаивают или блокируются при вводе данных пользователем. CLI с выводом JSON для агентов; TUI с визуальными индикаторами состояния для человека.
