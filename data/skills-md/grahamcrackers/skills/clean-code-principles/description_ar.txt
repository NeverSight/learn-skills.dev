مبادئ التعليمات البرمجية النظيفة لقواعد تعليمات TypeScript وReact القابلة للقراءة والصيانة. يغطي التسمية والوظائف والتجريد والتكوين ومعالجة الأخطاء والتعليقات وروائح التعليمات البرمجية. يُستخدم عند كتابة تعليمات برمجية جديدة، أو إعادة البناء، أو مراجعة جودة التعليمات البرمجية، أو عندما يسأل المستخدم عن التعليمات البرمجية النظيفة، أو سهولة القراءة، أو إمكانية الصيانة.
