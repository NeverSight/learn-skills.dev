Patrones de React Aria Components para crear una interfaz de usuario accesible y sin estilo con una arquitectura basada en composición. Cubre la estructura de los componentes, el estilo con Tailwind y CSS, los accesorios de renderizado, las colecciones, los formularios, las selecciones, las superposiciones y la función de arrastrar y soltar. Úselo al crear componentes accesibles, usar componentes reaccionaria, crear sistemas de diseño o cuando el usuario pregunte sobre React Aria, primitivas de interfaz de usuario accesibles o bibliotecas de componentes sin cabeza.
