Modèles React Aria Components pour créer une interface utilisateur accessible et sans style avec une architecture basée sur la composition. Couvre la structure des composants, le style avec Tailwind et CSS, les accessoires de rendu, les collections, les formulaires, les sélections, les superpositions et le glisser-déposer. À utiliser lors de la création de composants accessibles, de l'utilisation de composants React-aria, de la création de systèmes de conception ou lorsque l'utilisateur pose des questions sur React Aria, les primitives d'interface utilisateur accessibles ou les bibliothèques de composants sans tête.
