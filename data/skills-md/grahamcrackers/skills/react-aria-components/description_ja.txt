React Aria Components パターンを使用して、コンポジションベースのアーキテクチャでアクセシブルでスタイルのない UI を構築します。コンポーネントの構造、Tailwind と CSS によるスタイル設定、レンダリング プロップ、コレクション、フォーム、選択、オーバーレイ、ドラッグ アンド ドロップについて説明します。アクセシブルなコンポーネントを構築する場合、react-aria-components を使用する場合、デザイン システムを作成する場合、またはユーザーが React Aria、アクセシブルな UI プリミティブ、またはヘッドレス コンポーネント ライブラリについて質問する場合に使用します。
