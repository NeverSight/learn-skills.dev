Шаблоны React Aria Components для создания доступного, нестилизованного пользовательского интерфейса с архитектурой на основе композиции. Охватывает структуру компонентов, стили с помощью Tailwind и CSS, рендеринг реквизитов, коллекций, форм, выделений, наложений и перетаскивания. Используйте при создании доступных компонентов, использовании компонентов реакции-aria, создании систем дизайна или когда пользователь спрашивает о React Aria, доступных примитивах пользовательского интерфейса или библиотеках безголовых компонентов.
