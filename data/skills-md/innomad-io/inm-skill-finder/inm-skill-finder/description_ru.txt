Найдите и установите навыки Claude Code из нескольких источников (таблица лидеров Skills.sh, репозитории GitHub, такие как Awesome-Claude-Skills и anthropics/skills). Используйте, когда пользователь хочет найти, обнаружить, просмотреть или установить навыки, плагины или расширения Claude Code. Поддерживает неанглийские ключевые слова посредством автоматического перевода.
