Paystack Terminal API — Paystack POS 端末を使用して対面支払いエクスペリエンスを構築します。イベント (請求書/トランザクション) の送信、端末ステータスの確認、デバイスの使用開始/使用停止、端末の詳細の管理。このスキルは、Paystack POS 端末の統合、端末への支払いイベントの送信、対面支払いの処理、端末の可用性の確認、端末デバイスのアクティブ化または非アクティブ化、または POS 端末群の管理を行うときに常に使用します。 /terminal エンドポイント、terminal_id、シリアル番号、またはイベントベースの端末通信への参照が表示された場合にも使用します。
