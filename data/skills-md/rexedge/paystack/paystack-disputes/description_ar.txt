Paystack Disputes API - إدارة نزاعات المعاملات (رد المبالغ المدفوعة) على التكامل الخاص بك. سرد النزاعات وجلبها وتحديثها وحلها وتصديرها. تقديم الأدلة، وتحميل المستندات الداعمة، والتعامل مع سير عمل الحل. استخدم هذه المهارة عند التعامل مع عمليات رد المبالغ المدفوعة، أو إدارة النزاعات، أو مطالبات الاحتيال، أو نزاعات استرداد الأموال، أو تقديم دليل على المعاملات المتنازع عليها، أو مراقبة حالة النزاع. استخدمه أيضًا عندما ترى إشارات إلى نقطة نهاية /dispute، أو حالات النزاع (في انتظار تعليقات التاجر، أو في انتظار المراجعة، أو الحل)، أو عندما تحتاج إلى تحميل ملفات أدلة النزاع.
