API Paystack Transaction Splits: cree, actualice y administre divisiones de pagos entre múltiples subcuentas. Configure divisiones porcentuales o de cantidad fija con tipos de portadores flexibles (subcuenta, cuenta, totalmente proporcional, todo). Agregue o elimine subcuentas de grupos divididos. Utilice esta habilidad siempre que implemente el reparto de ingresos entre varias partes, estructuras de comisiones del mercado, distribución de pagos entre múltiples proveedores o cualquier flujo que divida un pago único entre múltiples destinatarios. Úselo también cuando vea referencias a split_code, códigos con prefijo SPL_, tipo_portador o el punto final /split.
