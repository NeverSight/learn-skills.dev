واجهة برمجة تطبيقات Paystack Transaction Splits API - إنشاء وتحديث وإدارة تقسيمات الدفع متعددة الأطراف عبر الحسابات الفرعية. قم بتكوين تقسيمات النسبة المئوية أو المبلغ الثابت باستخدام أنواع حاملة مرنة (حساب فرعي، حساب، الكل متناسب، الكل). إضافة أو إزالة الحسابات الفرعية من المجموعات المقسمة. استخدم هذه المهارة عند تنفيذ مشاركة الإيرادات بين أطراف متعددة، أو هياكل عمولات السوق، أو توزيع مدفوعات متعددة البائعين، أو أي تدفق يقسم دفعة واحدة بين عدة مستلمين. استخدمه أيضًا عندما ترى مراجع لـ Split_code، أو رموز SPL_ البادئة، أو Bearer_type، أو نقطة النهاية /split.
