Paystack Transaction Splits API – Erstellen, aktualisieren und verwalten Sie Mehrparteien-Zahlungsaufteilungen über Unterkonten hinweg. Konfigurieren Sie prozentuale oder pauschale Aufteilungen mit flexiblen Inhabertypen (Unterkonto, Konto, All-Proportional, Alle). Unterkonten zu geteilten Gruppen hinzufügen oder daraus entfernen. Nutzen Sie diese Fähigkeit, wenn Sie eine Umsatzaufteilung zwischen mehreren Parteien, Provisionsstrukturen auf dem Markt, eine Zahlungsverteilung über mehrere Anbieter oder einen beliebigen Ablauf implementieren, der eine einzelne Zahlung auf mehrere Empfänger aufteilt. Verwenden Sie es auch, wenn Sie Verweise auf Split_code, SPL_-Präfixcodes, Bearer_Type oder den /split-Endpunkt sehen.
