Paystack-Integrations-API – Verwalten Sie Einstellungen auf Integrationsebene wie Zeitüberschreitungen bei Zahlungssitzungen. Verwenden Sie diese Fähigkeit, wenn Sie Paystack-Integrationseinstellungen konfigurieren, die Zeitüberschreitungsdauer für Zahlungssitzungen anpassen oder Admin-Panels erstellen, die das Verhalten von Zahlungssitzungen steuern. Verwenden Sie diese Option auch, wenn Sie Verweise auf den Endpunkt /integration/ payment_session_timeout sehen.
