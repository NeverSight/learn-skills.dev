API интеграции Paystack — управляйте настройками уровня интеграции, такими как тайм-ауты платежных сеансов. Используйте этот навык при настройке параметров интеграции Paystack, настройке продолжительности тайм-аута платежного сеанса или создании панелей администратора, которые контролируют поведение платежного сеанса. Также используйте, когда видите ссылки на конечную точку /integration/pay_session_timeout.
