API de integración de Paystack: administre configuraciones a nivel de integración, como tiempos de espera de sesiones de pago. Utilice esta habilidad siempre que configure los ajustes de integración de Paystack, ajuste la duración del tiempo de espera de las sesiones de pago o cree paneles de administración que controlen el comportamiento de las sesiones de pago. Utilícelo también cuando vea referencias al punto final /integration/paid_session_timeout.
