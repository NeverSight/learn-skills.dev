Paystack Subaccounts API – Erstellen und verwalten Sie Unterkonten für geteilte Zahlungen zwischen Ihrem Hauptkonto und Unterhändlern. Konfigurieren Sie prozentuale Gebühren, Abwicklungsbanken, Abwicklungspläne und Kontaktdaten. Nutzen Sie diese Fähigkeit, wenn Sie einen Marktplatz aufbauen, geteilte Zahlungen implementieren, Unterhändler einbinden, Auszahlungspläne konfigurieren (automatisch/wöchentlich/monatlich/manuell) oder Zahlungsströme mehrerer Anbieter verwalten. Verwenden Sie es auch, wenn Sie Verweise auf subaccount_code, ACCT_-Präfixcodes, prozentuale Gebühr oder den /subaccount-Endpunkt sehen.
