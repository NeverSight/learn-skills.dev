واجهة برمجة تطبيقات Paystack Subaccounts API - إنشاء وإدارة حسابات فرعية لتقسيم المدفوعات بين حسابك الرئيسي والتجار الفرعيين. قم بتكوين نسبة الرسوم وبنوك التسوية وجداول التسوية وتفاصيل الاتصال. استخدم هذه المهارة عند إنشاء سوق، أو تنفيذ مدفوعات مقسمة، أو تأهيل التجار الفرعيين، أو تكوين جداول الدفع (تلقائي/أسبوعي/شهري/يدوي)، أو إدارة تدفقات الدفع متعددة البائعين. استخدمه أيضًا عندما ترى إشارات إلى رمز_الحساب_الفرعي، أو رموز ACCT_ البادئة، أو النسبة المئوية_للشحن، أو نقطة نهاية الحساب/الحساب الفرعي.
