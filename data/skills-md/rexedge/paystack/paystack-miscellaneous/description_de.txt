Paystack Miscellaneous API – unterstützt Endpunkte zum Abrufen von Banklisten, Länderdaten und Adressüberprüfungsstatus. Nutzen Sie diesen Skill immer dann, wenn Sie Bankleitzahlen für Überweisungen oder Kontoauflösungen benötigen, unterstützte Länder und deren Währungen auflisten, Staaten zur Adressverifizierung (AVS) abrufen, Bankschlüssel für DVA-Anbieter suchen oder Banken nach Zahlungsmöglichkeiten filtern. Verwenden Sie diese Option auch, wenn Sie Verweise auf /bank-Endpunkt, Bankcode-Suchen, /country-Endpunkt oder /address_verification/states sehen.
