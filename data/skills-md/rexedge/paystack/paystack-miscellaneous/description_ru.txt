Разное API Paystack — поддержка конечных точек для получения списков банков, данных о стране и состояний проверки адреса. Используйте этот навык всякий раз, когда вам нужны банковские коды для переводов или разрешения счетов, перечисления поддерживаемых стран и их валют, получения штатов для проверки адреса (AVS), поиска банковских сертификатов для поставщиков DVA или фильтрации банков по платежным возможностям. Также используйте, когда вы видите ссылки на конечную точку /bank, поиск по коду банка, конечную точку /country или /address_verification/states.
