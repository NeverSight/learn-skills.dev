API miscelánea de Paystack: puntos finales compatibles para obtener listas de bancos, datos de países y estados de verificación de direcciones. Utilice esta habilidad siempre que necesite códigos bancarios para transferencias o resolución de cuentas, enumerar los países admitidos y sus monedas, buscar estados para verificación de direcciones (AVS), buscar slugs bancarios para proveedores de DVA o filtrar bancos por capacidades de pago. Utilícelo también cuando vea referencias a /punto final del banco, búsquedas de código_banco, /punto final del país o /verificación_dirección/estados.
