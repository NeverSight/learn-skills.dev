Paystack Settlements API – Auszahlungen von Paystack auf Ihr Bankkonto anzeigen und verfolgen. Listen Sie Abrechnungen mit Statusfiltern auf, zeigen Sie einzelne Abrechnungstransaktionen an und überwachen Sie die Abwicklungszeitpläne. Verwenden Sie diese Fähigkeit, wenn Sie Auszahlungen verfolgen, Bankeinlagen mit Paystack-Transaktionen abgleichen, den Abrechnungsstatus überwachen (Erfolg/Verarbeitung/ausstehend/fehlgeschlagen), Abrechnungen nach Unterkonto filtern oder Tools für den Finanzabgleich erstellen. Verwenden Sie diese Option auch, wenn Sie Verweise auf den Endpunkt /settlement sehen oder wissen möchten, wann Gelder auf Ihrem Bankkonto eingehen.
