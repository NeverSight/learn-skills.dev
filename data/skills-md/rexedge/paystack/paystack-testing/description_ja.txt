ペイスタック テスト ガイド — テスト モード キーを使用したペイスタック統合のための包括的なテスト戦略。テスト カード番号、銀行口座テスト データ、Webhook テスト、トランザクション シミュレーション、転送テスト、エンドツーエンド テスト パターンをカバーします。このスキルは、Paystack 支払いのテストの作成、テスト モードでのトランザクションのシミュレーション、Webhook ハンドラーのテスト、Paystack API 応答のモック、公開前の支払いフローの検証、または失敗したテスト トランザクションのデバッグを行うときに常に使用します。 sk_test_、pk_test_、テスト カード番号、またはペイスタック テスト モードへの参照が表示された場合にも使用します。
