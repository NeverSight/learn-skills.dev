Paystack Verification API – KYC-Verifizierungstools zur Auflösung von Bankkonten, zur Validierung des Kontoinhabers und zum Nachschlagen von Karten-BIN-Informationen. Verwenden Sie diese Fähigkeit, wenn Sie Bankkontodaten vor Überweisungen überprüfen, Namen von Kontoinhabern bestätigen, Kundenidentität auf Compliance überprüfen, Kartenmarke/-typ/Bank aus BIN nachschlagen oder KYC-Abläufe implementieren. Verwenden Sie diese Option auch, wenn Sie Verweise auf die Endpunkte /bank/resolve, /bank/validate, /decision/bin sehen oder Kontonummern Namen zuordnen müssen.
