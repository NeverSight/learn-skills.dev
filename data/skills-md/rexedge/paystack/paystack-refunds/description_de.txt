Paystack Refunds API – Erstellen Sie vollständige oder teilweise Rückerstattungen, wiederholen Sie fehlgeschlagene Rückerstattungen mit den Bankdaten des Kunden, listen Sie Rückerstattungsdatensätze auf und rufen Sie sie ab. Verwenden Sie diesen Skill immer dann, wenn Sie Rückerstattungsfunktionen implementieren, Rückerstattungsanfragen von Kunden bearbeiten, Teilrückerstattungen abwickeln, Rückerstattungen wiederholen, die Ihrer Aufmerksamkeit bedürfen, oder den Rückerstattungsstatus verfolgen. Verwenden Sie diese Option auch, wenn Sie Verweise auf Webhook-Ereignisse „refund.processed“, „refund.failed“, „refund.pending“ oder den Endpunkt „/refund“ sehen.
