Paystack Customers API — 作成、リスト、フェッチ、更新、ID の検証、ホワイトリスト/ブラックリスト、承認の管理、および口座振替操作を行います。このスキルは、Paystack での顧客レコードの作成または管理、BVN または銀行口座による顧客 ID の検証、リスク アクションによる顧客のホワイトリストまたはブラックリストへの登録、保存されたカード認証の無効化、口座振替義務の設定、または顧客管理ダッシュボードの構築を行うときに常に使用します。 customer_code、CUS_ プレフィックス付きコード、risk_action、または /customer/authorization エンドポイントへの参照が表示された場合にも使用します。
