Paystack Charge API – Initiieren Sie direkte Belastungen über Karte, Bankkonto, USSD, mobiles Geld, QR, EFT, Capitec Pay und Banküberweisungskanäle. Bewältigen Sie die mehrstufige Authentifizierung mit den Endpunkten „PIN senden“, „OTP senden“, „Telefon senden“, „Geburtstag senden“ und „Adresse senden“. Überprüfen Sie den Status der ausstehenden Ladung. Nutzen Sie diese Fähigkeit, wenn Sie einen benutzerdefinierten Zahlungsfluss erstellen, der eine direkte Kanalsteuerung anstelle des Standard-Checkouts erfordert, Banklastschrift- oder USSD-Zahlungen implementieren, PIN-/OTP-Überprüfungsabläufe für Karten abwickeln, mobile Geldzahlungen in Ghana/Kenia verarbeiten oder den Status einer ausstehenden Belastung überprüfen. Verwenden Sie diese Option auch, wenn Sie im Paystack-Code Verweise auf „/charge“, „submit_pin“, „submit_otp“ oder „send_birthday“ sehen.
