API Paystack Transactions: inizializza pagamenti, verifica transazioni, elenca/recupera la cronologia delle transazioni, addebita autorizzazioni salvate, visualizza sequenze temporali, ottieni totali, esporta dati ed esegui addebiti parziali. Utilizza questa competenza ogni volta che crei un flusso di pagamento, verifichi lo stato del pagamento, ricarichi la carta salvata di un cliente abituale, estragga report o analisi delle transazioni, esporti CSV delle transazioni o gestisca qualsiasi endpoint Paystack correlato alle transazioni. Utilizzalo anche quando vedi riferimenti a /transaction/initialize, /transaction/verify, warranty_url, access_code o charge_authorization nelle integrazioni Paystack.
