Paystack Transactions API – Zahlungen initialisieren, Transaktionen überprüfen, Transaktionsverlauf auflisten/abrufen, gespeicherte Autorisierungen belasten, Zeitpläne anzeigen, Gesamtbeträge abrufen, Daten exportieren und Teilabbuchungen durchführen. Verwenden Sie diese Fähigkeit, wenn Sie einen Checkout-Ablauf erstellen, den Zahlungsstatus überprüfen, die gespeicherte Karte eines wiederkehrenden Kunden aufladen, Transaktionsberichte oder Analysen abrufen, Transaktions-CSVs exportieren oder einen transaktionsbezogenen Paystack-Endpunkt verwalten. Verwenden Sie es auch, wenn Sie in Paystack-Integrationen Verweise auf /transaction/initialize, /transaction/verify,authorization_url, access_code oder charge_authorization sehen.
