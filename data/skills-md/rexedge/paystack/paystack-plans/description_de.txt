Paystack Plans API – Erstellen, Auflisten, Abrufen und Aktualisieren von Abonnementplänen in Intervallen (täglich, wöchentlich, monatlich, vierteljährlich, halbjährlich, jährlich). Verwenden Sie diese Fähigkeit, wenn Sie Preisstufen oder wiederkehrende Abrechnungspläne auf Paystack erstellen, Abonnementintervalle einrichten, Rechnungslimits konfigurieren, Planwährungen verwalten oder eine SaaS-Preisseite erstellen. Verwenden Sie es auch, wenn Sie Verweise auf plan_code, PLN_-Präfixcodes oder den /plan-Endpunkt sehen.
