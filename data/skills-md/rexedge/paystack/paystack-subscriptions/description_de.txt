Paystack Subscriptions API – Abonnements erstellen, auflisten, abrufen, aktivieren, deaktivieren und Kartenaktualisierungslinks verwalten. Verwenden Sie diese Fähigkeit, wenn Sie eine wiederkehrende Abrechnung implementieren, Kunden Pläne abonnieren, die automatische Verlängerung aktivieren oder deaktivieren, Kartenaktualisierungslinks für ablaufende Karten erstellen oder Ereignisse im Abonnementlebenszyklus verwalten. Verwenden Sie es auch, wenn Sie Verweise auf subscription_code, SUB_-Präfixcodes, email_token oder den /subscription-Endpunkt sehen.
