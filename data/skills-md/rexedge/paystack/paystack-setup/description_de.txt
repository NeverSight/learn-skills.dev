Richten Sie den Paystack-API-Client, Umgebungsvariablen und TypeScript-Helfer für die serverseitige Zahlungsintegration ein. Verwenden Sie diese Fähigkeit immer dann, wenn Sie eine neue Paystack-Integration starten, API-Schlüssel konfigurieren, einen wiederverwendbaren Abruf-Wrapper für die Paystack-REST-API erstellen oder die Grundlage für eine beliebige Paystack-Funktion einrichten. Verwenden Sie es auch, wenn Sie Fehler im Zusammenhang mit fehlendem PAYSTACK_SECRET_KEY oder Authentifizierungsfehlern sehen oder wenn Sie die Basis-URL, das Antwortformat, die Paginierung, die Währungen oder die Umrechnung von Betragsuntereinheiten von Paystack verstehen müssen.
