Configurez le client API Paystack, les variables d'environnement et les assistants TypeScript pour l'intégration des paiements côté serveur. Utilisez cette compétence chaque fois que vous démarrez une nouvelle intégration Paystack, configurez des clés API, créez un wrapper de récupération réutilisable pour l'API REST Paystack ou configurez les bases de toute fonctionnalité Paystack. À utiliser également lorsque vous voyez des erreurs liées à l'absence de PAYSTACK_SECRET_KEY, à des échecs d'authentification ou si vous avez besoin de comprendre l'URL de base, le format de réponse, la pagination, les devises ou la conversion de sous-unités de montant de Paystack.
