Configura il client API Paystack, le variabili di ambiente e gli helper TypeScript per l'integrazione dei pagamenti lato server. Utilizza questa competenza ogni volta che avvii una nuova integrazione Paystack, configuri le chiavi API, crei un wrapper di recupero riutilizzabile per l'API REST Paystack o configuri le basi per qualsiasi funzionalità Paystack. Da utilizzare anche quando vengono visualizzati errori relativi alla mancanza di PAYSTACK_SECRET_KEY, errori di autenticazione o necessità di comprendere l'URL di base di Paystack, il formato di risposta, l'impaginazione, le valute o la conversione delle subunità dell'importo.
