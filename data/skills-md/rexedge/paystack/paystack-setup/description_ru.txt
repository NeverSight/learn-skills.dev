Настройте клиент API Paystack, переменные среды и помощники TypeScript для интеграции платежей на стороне сервера. Используйте этот навык всякий раз, когда начинаете новую интеграцию Paystack, настраиваете ключи API, создаете многоразовую оболочку выборки для Paystack REST API или настраиваете основу для любой функции Paystack. Также используйте, когда вы видите ошибки, связанные с отсутствием PAYSTACK_SECRET_KEY, ошибками аутентификации или необходимостью понять базовый URL-адрес Paystack, формат ответа, нумерацию страниц, валюты или преобразование субъединиц суммы.
