Configure el cliente API Paystack, las variables de entorno y los asistentes de TypeScript para la integración de pagos del lado del servidor. Utilice esta habilidad siempre que inicie una nueva integración de Paystack, configure claves API, cree un contenedor de recuperación reutilizable para la API REST de Paystack o establezca las bases para cualquier función de Paystack. Úselo también cuando vea errores relacionados con la falta de PAYSTACK_SECRET_KEY, fallas de autenticación o necesite comprender la URL base de Paystack, el formato de respuesta, la paginación, las monedas o la conversión de subunidades de cantidad.
