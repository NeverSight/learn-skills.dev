قم بإعداد عميل Paystack API ومتغيرات البيئة ومساعدي TypeScript لتكامل الدفع من جانب الخادم. استخدم هذه المهارة عند بدء تكامل Paystack جديد، أو تكوين مفاتيح API، أو إنشاء غلاف جلب قابل لإعادة الاستخدام لـ Paystack REST API، أو إعداد الأساس لأي ميزة Paystack. استخدمه أيضًا عندما ترى أخطاء تتعلق بفقدان PAYSTACK_SECRET_KEY، أو فشل المصادقة، أو تحتاج إلى فهم عنوان URL الأساسي لـ Paystack، أو تنسيق الاستجابة، أو ترقيم الصفحات، أو العملات، أو مبلغ تحويل الوحدة الفرعية.
