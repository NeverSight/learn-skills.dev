Genere una tarjeta de resumen de información médica concisa para un acceso rápido en caso de emergencia. Utilice esta habilidad cuando el usuario esté viajando, preparándose para una visita al médico, en caso de emergencia o solicite "información de emergencia", "tarjeta médica", "información de primeros auxilios". Extraiga datos clave (alergias, medicamentos, afecciones, implantes) y admita salida multiformato (JSON, texto, código QR) para primeros auxilios o atención médica rápida.
