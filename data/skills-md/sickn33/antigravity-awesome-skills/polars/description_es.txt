Biblioteca DataFrame rápida en memoria para conjuntos de datos que caben en la RAM. Úselo cuando pandas sea demasiado lento pero los datos aún quepan en la memoria. Evaluación diferida, ejecución paralela, backend Apache Arrow. Lo mejor para conjuntos de datos de 1 a 100 GB, canalizaciones ETL y reemplazo de pandas más rápido. Para datos de mayor tamaño que la RAM, utilice dask o vaex.
