Bibliothèque DataFrame en mémoire rapide pour les ensembles de données qui tiennent dans la RAM. À utiliser lorsque les pandas sont trop lents mais que les données tiennent toujours en mémoire. Évaluation paresseuse, exécution parallèle, backend Apache Arrow. Idéal pour les ensembles de données de 1 à 100 Go, les pipelines ETL et le remplacement plus rapide des pandas. Pour les données supérieures à la RAM, utilisez dask ou vaex.
