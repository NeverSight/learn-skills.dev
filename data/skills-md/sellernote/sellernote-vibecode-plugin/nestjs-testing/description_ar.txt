كتابة اختبار الواجهة الخلفية لـ NestJS وفقًا لاتفاقيات Sellernote. يُستخدم عند إنشاء اختبارات الوحدة أو اختبارات التكامل أو اختبارات e2e لخدمات NestJS أو وحدات التحكم أو المستودعات أو الوحدات النمطية. تشغيل المهام التي تتضمن كتابة الاختبار، أو ملفات المواصفات، أو السخرية من الدعابة، أو اختبار الخدمة، أو اختبار وحدة التحكم، أو اختبار المستودع، أو اختبار e2e، أو تغطية الاختبار، أو أي عمل اختبار خلفي في مشروع NestJS. استخدمه أيضًا عندما يُطلب منك كتابة اختبارات، أو إضافة اختبارات وحدة، أو إنشاء اختبارات e2e، أو إضافة ملفات المواصفات، أو التحقق من جودة الاختبار لبنية Sellernote المكونة من 3 طبقات (وحدة التحكم → الخدمة → المستودع).
