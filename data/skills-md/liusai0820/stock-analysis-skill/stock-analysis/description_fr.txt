Compétence d’analyse intelligente des stocks. Saisissez le code d'action (actions A/actions de Hong Kong/actions américaines), complétez automatiquement :
1. Obtenez des cotations en temps réel + des données historiques en chandeliers
2. Calculer les indicateurs techniques (MA/MACD/RSI/volume/bias)
3. Score global (sur 100) + signaux d'achat/vente
4. Recherchez les dernières nouvelles et le sentiment du marché
5. Analyse complète de l'IA, tableau de bord de décision de sortie

Scénarios de déclenchement : l'utilisateur fournit le code boursier pour l'analyse, demande comment se porte un certain titre, demande une analyse de marché, etc.
Exemple d'entrée : "Analyser TSLA PLTR" "Comment va 600519" "Vérifiez HK00700 pour moi"
