Compétence d’analyse des stocks basée sur l’IA. Saisissez un code d'action (actions A/actions de Hong Kong/actions américaines), et automatiquement :

1. Récupère les cotations en temps réel + les données historiques des chandeliers  
2. Calcule les indicateurs techniques (MA/MACD/RSI/volume/bias ratio)  
3. Donne une note composite (sur 100) + signaux d'achat/vente  
4. Recherche les dernières nouvelles et sentiments  
5. Utilise l'IA pour tout synthétiser et génère un tableau de bord de décision  

Scénarios de déclenchement : l'utilisateur fournit un code boursier à des fins d'analyse, demande comment se porte un certain titre, demande une révision du graphique, etc.  
Exemples d'entrées : « Analyser TSLA PLTR » « Comment va le 600519 ? » "Vérifiez HK00700 pour moi"
