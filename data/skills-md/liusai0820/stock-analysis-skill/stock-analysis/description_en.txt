AI-powered stock analysis tool. Input any stock code (A-shares/HK-shares/US-shares) and it automatically:
1. Pulls real-time quotes + historical candlestick data
2. Calculates technical indicators (MA/MACD/RSI/volume/bias)
3. Gives a composite score (0-100) + buy/sell signals
4. Searches the latest news and headlines
5. AI synthesizes everything into a decision dashboard

Trigger: user supplies a ticker for analysis, asks “how’s this stock?” or requests a chart review, etc.
Sample input: “analyze TSLA PLTR” “how’s 600519?” “check HK00700 for me”
