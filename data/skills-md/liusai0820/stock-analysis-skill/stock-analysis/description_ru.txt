Навыки анализа акций с помощью искусственного интеллекта. Введите код акции (доля A/доля Гонконга/доля США), и она автоматически:
1. Получает котировки в реальном времени + исторические данные K-line.  
2. Рассчитывает технические индикаторы (MA/MACD/RSI/объем/смещение).  
3. Дает совокупный балл (из 100) + сигналы покупки/продажи.  
4. Ищет последние новости и настроения  
5. ИИ синтезирует все и выводит панель принятия решений  

Сценарии триггера: пользователь предоставляет код для анализа, спрашивает «как дела с акциями», запрашивает обзор графика и т. д.  
Пример входных данных: «Анализ TSLA PLTR», «Как дела с 600519», «Проверьте HK00700 для меня»
