Навыки анализа акций с помощью искусственного интеллекта. Введите код акции (акции A/акции Гонконга/акции США), и он автоматически:

1. Получает котировки в реальном времени + исторические данные свечей.  
2. Рассчитывает технические индикаторы (MA/MACD/RSI/объем/соотношение смещения).  
3. Дает совокупный балл (из 100) + сигналы покупки/продажи.  
4. Ищет последние новости и настроения  
5. Использует ИИ для синтеза всего и выводит панель принятия решений.  

Сценарии триггера: пользователь предоставляет код акции для анализа, спрашивает, как обстоят дела с определенной акцией, запрашивает обзор графика и т. д.  
Пример входных данных: «Анализ TSLA PLTR», «Как 600519?» «Проверьте для меня HK00700»
