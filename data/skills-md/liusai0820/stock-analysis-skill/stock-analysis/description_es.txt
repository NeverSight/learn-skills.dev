Habilidad de análisis de acciones impulsada por IA. Ingrese un código de acciones (acciones A/acciones de Hong Kong/acciones de EE. UU.) y automáticamente:

1. Obtiene cotizaciones en tiempo real + datos históricos de velas japonesas  
2. Calcula indicadores técnicos (relación MA/MACD/RSI/volumen/sesgo)  
3. Da una puntuación compuesta (sobre 100) + señales de compra/venta.  
4. Busca las últimas noticias y sentimientos.  
5. Utiliza IA para sintetizar todo y genera un panel de decisiones.  

Escenarios desencadenantes: el usuario proporciona un código de acción para su análisis, pregunta cómo le está yendo a una determinada acción, solicita una revisión del gráfico, etc.  
Entradas de muestra: "Analizar TSLA PLTR" "¿Cómo está 600519?" “Comprueba HK00700 por mí”
