Sync the commits from the current development branch (usually dev) to the main branch via git rebase, push to the remote, then switch back to the original branch. Trigger this skill when the user mentions phrases like “sync to main,” “rebase to main,” “update main,” “push to main,” or “merge dev into main.” Even if the user does not explicitly say “rebase,” as long as the intent is to synchronize the development branch content to main, this skill should be activated.
