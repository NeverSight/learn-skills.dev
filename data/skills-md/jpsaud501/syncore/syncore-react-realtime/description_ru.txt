Шаблоны React для Syncore, включая SyncoreProvider, поставщиков оболочек платформы, useQuery, useMutation, useAction, useQueries, пропуск семантики, обработку состояния загрузки и безопасное использование перехватчиков. Используйте при подключении Syncore к приложениям React или при отладке проблем с выводом перехватчиков и жизненным циклом подписки.
