Boucle d'ingénierie composée - 5 étapes pour réduire les retouches. À utiliser lorsque : (1) vous démarrez un nouveau travail (brainstorm) ; (2) la planification de la mise en œuvre (plan); (3) écrire du code (travail); (4) évaluer les résultats (examen) ; (5) capturer des leçons (composé). PAS pour : le rythme quotidien (à utiliser quotidiennement) ; gestion des tâches uniquement (utiliser la tâche).
