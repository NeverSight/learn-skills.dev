Gestion des tâches indépendante du backend. À utiliser dans les cas suivants : (1) création, liste, mise à jour ou archivage de tâches ; (2) analyser l'état des tâches ; (3) configuration du backend des tâches. PAS pour : la population de mémoire (utiliser la mémoire) ; gestion de session (utiliser la session mémoire).
