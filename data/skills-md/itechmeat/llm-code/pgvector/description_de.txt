pgvector Postgres-Erweiterung. Behandelt Vektortypen, Distanzoperatoren, Indizierung (HNSW/IVFFlat) und Client-Bibliotheksnutzung. Wird zum Speichern von Vektoren in PostgreSQL, zum Durchführen von Suchen nach nächsten Nachbarn oder zum Konfigurieren von HNSW/IVFFlat-Indizes verwendet. Schlüsselwörter: pgvector, PostgreSQL, Vektorsuche, HNSW, IVFFlat.
