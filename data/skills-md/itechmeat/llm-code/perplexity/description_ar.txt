دمج Perplexity API لاستجابات الذكاء الاصطناعي والبحث على شبكة الإنترنت. يغطي نماذج السونار، وواجهة برمجة تطبيقات البحث، واستخدام SDK (Python/TypeScript)، والبث، والمخرجات المنظمة، والمرشحات، ومرفقات الوسائط، والبحث الاحترافي، والمطالبة. الكلمات الرئيسية: الحيرة، السونار، السونار المحترف، السونار المنطقي المحترف، البحث العميق بالسونار، واجهة برمجة التطبيقات للبحث على الويب، LLM المؤرض، إكمال الدردشة، الحيرة SDK، مرفقات الصور، تحليل PDF.
