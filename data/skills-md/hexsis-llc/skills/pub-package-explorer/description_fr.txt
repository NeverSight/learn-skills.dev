Recherchez et lisez le code source des packages Dart ou Flutter à partir des dépendances du projet ou de pub.dev. À utiliser lorsqu'on vous demande d'inspecter les détails d'implémentation du package, de tracer le code de dépendance, de localiser les fichiers du package en résolvant `.dart_tool/package_config.json` ou d'inspecter un package qui n'est pas actuellement installé en le décompressant avec `dart pub unpack`.
