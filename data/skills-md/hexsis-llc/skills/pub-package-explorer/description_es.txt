Busque y lea el código fuente de los paquetes Dart o Flutter desde las dependencias del proyecto o pub.dev. Úselo cuando se le solicite inspeccionar los detalles de implementación del paquete, rastrear el código de dependencia, ubicar archivos del paquete resolviendo `.dart_tool/package_config.json` o inspeccionar un paquete que no está instalado actualmente descomprimiéndolo con `dart pub unpack`.
