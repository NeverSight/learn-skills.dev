Zeichnen Sie iPhoneClaw-Aktionsskripte auf, registrieren Sie sie und rufen Sie sie auf, um VLM-Tokens zu reduzieren und Automatisierungen wiederholbar zu machen. Verwenden Sie diese Option, wenn Sie möchten, dass das Modell/der Agent „run_script(name=..., vars=...)“ aufruft, wenn Sie „action_scripts/registry.json“ aktualisieren, wenn Sie Skripte aus „runs/*/events.jsonl“ exportieren oder wenn Sie Skripte remote über „python -m iphoneclaw ctl run-script“ auslösen, während der Worker angehalten ist.
