Integre datos de Excel (.xlsx) en los scripts codegen de Playwright reemplazando los valores codificados con búsquedas basadas en configuración, cargando datos con polares y validando cada paso con Playwright MCP (inicie MCP desde el repositorio si no se está ejecutando). Úselo para actualizaciones de automatización de Playwright que requieren datos respaldados por Excel, centralización de config.yaml o validación/informes de MCP.
