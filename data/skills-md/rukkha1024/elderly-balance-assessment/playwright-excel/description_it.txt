Integra i dati Excel (.xlsx) negli script codegen di Playwright sostituendo i valori hardcoded con ricerche guidate dalla configurazione, caricando i dati con polari e convalidando ogni passaggio con Playwright MCP (avvia MCP dal repository se non è in esecuzione). Utilizzare per gli aggiornamenti di automazione di Playwright che richiedono dati supportati da Excel, centralizzazione config.yaml o convalida/report MCP.
