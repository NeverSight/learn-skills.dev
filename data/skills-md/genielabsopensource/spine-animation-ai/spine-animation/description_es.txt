Cree animaciones esqueléticas 2D de Spine a partir de recursos de personajes preexistentes (PNG de partes del cuerpo separadas, hoja de sprites de atlas o una imagen de personaje completa). Utilice esta habilidad siempre que el usuario desee animar un personaje 2D, crear Spine JSON a partir de recursos artísticos existentes, equipar un personaje con huesos, crear animaciones para caminar/inactivo/correr/atacar, producir una vista previa interactiva de Spine Web Player o generar archivos de exportación compatibles con Spine (.json + .atlas + .png). También se activa cuando el usuario menciona "animación de columna", "rigging 2D", "animación esquelética", "animación de huesos", "animación de recorte", "animar este personaje", "hacer este paseo", "crear un ciclo de paseo" o carga partes separadas del cuerpo del personaje y quiere animarlas. Esta habilidad maneja todo el proceso: análisis de activos, manipulación de esqueletos, fotogramas clave de animación, exportación de Spine JSON y vista previa interactiva de HTML5.
