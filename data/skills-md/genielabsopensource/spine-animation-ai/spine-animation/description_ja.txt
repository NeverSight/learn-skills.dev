既存のキャラクター アセット (分割されたボディパーツ PNG、アトラス スプライトシート、または完全なキャラクター イメージ) から Spine 2D スケルトン アニメーションを作成します。ユーザーが 2D キャラクターのアニメート、既存のアート アセットからの Spine JSON の作成、ボーンを使用したキャラクターのリグ、歩行/アイドル/走行/攻撃アニメーションの構築、インタラクティブな Spine Web Player プレビューの作成、または Spine 互換のエクスポート ファイル (.json + .atlas + .png) の生成を行う場合は、このスキルをいつでも使用できます。また、ユーザーが「脊椎アニメーション」、「2D リギング」、「スケルトン アニメーション」、「ボーン アニメーション」、「カットアウト アニメーション」、「このキャラクターをアニメートする」、「これを歩かせる」、「歩行サイクルを作成する」と言及した場合、または分離されたキャラクターのボディ パーツをアップロードしてそれらをアニメーション化したい場合にもトリガーされます。このスキルは、アセット分析、スケルトン リギング、アニメーション キーフレーム、Spine JSON エクスポート、およびインタラクティブな HTML5 プレビューといったパイプライン全体を処理します。
