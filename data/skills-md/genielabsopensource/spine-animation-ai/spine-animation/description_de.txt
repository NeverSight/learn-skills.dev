Erstellen Sie Spine-2D-Skelettanimationen aus bereits vorhandenen Charakter-Assets (separierte Körperteil-PNGs, Atlas-Spritesheet oder ein vollständiges Charakterbild). Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine 2D-Figur animieren, Spine JSON aus vorhandenen Kunstelementen erstellen, eine Figur mit Knochen ausrüsten, Geh-/Leerlauf-/Lauf-/Angriffsanimationen erstellen, eine interaktive Spine Web Player-Vorschau erstellen oder Spine-kompatible Exportdateien (.json + .atlas + .png) generieren möchte. Wird auch ausgelöst, wenn der Benutzer „Wirbelsäulenanimation“, „2D-Rigging“, „Skelettanimation“, „Knochenanimation“, „Ausschnittanimation“, „Diesen Charakter animieren“, „Diesen Spaziergang machen“, „Gehzyklus erstellen“ erwähnt oder separate Körperteile des Charakters hochlädt und diese animieren möchte. Dieser Skill kümmert sich um die gesamte Pipeline: Asset-Analyse, Skelett-Rigging, Animations-Keyframing, Spine-JSON-Export und interaktive HTML5-Vorschau.
