قم بإنشاء رسوم متحركة هيكلية ثنائية الأبعاد للعمود الفقري من أصول الشخصيات الموجودة مسبقًا (أجزاء PNG المنفصلة لأجزاء الجسم، أو ورقة الأطلس المتحركة، أو صورة شخصية كاملة). استخدم هذه المهارة عندما يريد المستخدم تحريك شخصية ثنائية الأبعاد، أو إنشاء Spine JSON من الأصول الفنية الموجودة، أو تجهيز شخصية بالعظام، أو إنشاء رسوم متحركة للمشي/التباطؤ/الجري/الهجوم، أو إنتاج معاينة Spine Web Player تفاعلية، أو إنشاء ملفات تصدير متوافقة مع Spine (.json + .atlas + .png). يتم تشغيله أيضًا عندما يذكر المستخدم "الرسوم المتحركة للعمود الفقري"، أو "التجهيز ثنائي الأبعاد"، أو "الرسوم المتحركة للهيكل العظمي"، أو "الرسوم المتحركة للعظام"، أو "الرسوم المتحركة المقطوعة"، أو "تحريك هذه الشخصية"، أو "إجراء هذه المسيرة"، أو "إنشاء دورة سير"، أو تحميل أجزاء منفصلة من جسم الشخصية ويريد تحريكها. تتعامل هذه المهارة مع المسار الكامل: تحليل الأصول، والتزوير الهيكلي، والإطارات الرئيسية للرسوم المتحركة، وتصدير Spine JSON، ومعاينة HTML5 التفاعلية.
