قم بكتابة التعليمات البرمجية ودعمها وشرحها وإعادة تصميمها باستخدام الحزمة lorisleiva/laravel-actions. استخدم هذه المهارة عندما يعمل المستخدم مع Laravel Actions، أو يريد إنشاء فئة إجراء، أو تحويل وحدة تحكم/وظيفة/مستمع/أمر إلى إجراء، أو يسأل عن كيفية استخدام سمة AsAction، أو إرسال إجراء كمهمة، أو استخدام إجراء كوحدة تحكم أو مستمع، أو إعداد التحقق/التفويض في إجراء، أو اختبار أو إجراءات وهمية، أو يسأل أي شيء عن نمط حزمة إجراءات laravel.
