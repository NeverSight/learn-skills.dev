كتابة المكونات الإضافية ودعمها وشرحها وتصحيح أخطائها للوحة ألعاب Pelican. استخدم هذه المهارة عندما يريد المستخدم إنشاء مكون إضافي لـ Pelican، أو إضافة وظائف إلى لوحة Pelican، أو إنشاء مورد أو صفحة FilamentPHP لـ Pelican، أو تسجيل أذونات مخصصة، أو إضافة إعدادات المكونات الإضافية، أو كتابة مسارات المكونات الإضافية، أو طرح أي شيء حول تطوير البرنامج الإضافي Pelican - حتى لو قالوا فقط "أريد إضافة شيء ما إلى Pelican" أو "كيف يمكنني توسيع Pelican".
