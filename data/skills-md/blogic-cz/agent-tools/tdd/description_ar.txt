قم بتحميل هذه المهارة عندما: تنفيذ ميزات جديدة باستخدام TDD، أو يذكر المستخدم "TDD"، أو "الاختبار أولاً"، أو "الأحمر والأخضر-refactor"، أو "الاختبار الفاشل"، أو عند إنشاء خدمات التأثير التي تحتاج إلى اختبار شامل. يغطي منهجية TDD، ودورة Red-Green-Refactor، واختبار خدمة التأثير باستخدام طبقات وهمية، وسير عمل تطوير الاختبار أولاً.
