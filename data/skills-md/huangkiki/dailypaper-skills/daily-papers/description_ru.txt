Основная запись в одно предложение для ежедневных рекомендаций по газетам. Используется, когда пользователь говорит «рекомендации по бумагам на сегодня», «рекомендации по бумагам за последние 3 дня», «рекомендации по бумагам за прошедшую неделю», «последние статьи за 3 дня» или «покажите мне статьи на этой неделе».

Внутренняя цепочка получения документов, создания рекомендаций и заметок на бумаге за один раз — никаких ручных действий для пользователя.
