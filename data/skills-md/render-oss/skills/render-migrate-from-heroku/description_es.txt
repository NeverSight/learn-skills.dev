Migre de Heroku a Render leyendo archivos de proyecto locales y generando servicios de Render equivalentes. Desencadenantes: cualquier mención de migrar de Heroku, pasar de Heroku, migración de Heroku a Render o cambiar de Heroku. Lee Procfile, archivos de dependencia y configuración de aplicaciones del repositorio local. Opcionalmente, utiliza Heroku MCP para enriquecer con variables de configuración en vivo, detalles de complementos y tamaños de dinamómetro. Utiliza Render MCP o Blueprint YAML para crear servicios.
