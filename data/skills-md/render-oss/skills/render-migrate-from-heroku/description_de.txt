Migrieren Sie von Heroku zu Render, indem Sie lokale Projektdateien lesen und entsprechende Render-Dienste generieren. Auslöser: jede Erwähnung einer Migration von Heroku, eines Wechsels von Heroku, einer Heroku-zu-Render-Migration oder eines Wechsels von Heroku. Liest Procfile, Abhängigkeitsdateien und App-Konfiguration aus dem lokalen Repo. Verwendet optional Heroku MCP zur Anreicherung mit Live-Konfigurationsvariablen, Add-on-Details und Dyno-Größen. Verwendet Render MCP oder Blueprint YAML, um Dienste zu erstellen.
