قم بإعداد وتطوير واختبار ونشر سير عمل العرض. يغطي السقالات لأول مرة (عبر سطر الأوامر أو الدليل)، وأنماط المهام (إعادة المحاولة، والمهام الفرعية، والتوزيع)، والتطوير المحلي، ونشر لوحة المعلومات، واستكشاف الأخطاء وإصلاحها. يُستخدم عندما يريد المستخدم إعداد Render Workflows لأول مرة، أو تثبيت Render Workflows SDK (Python أو TypeScript)، أو دعم خدمة سير العمل، أو إضافة المهام أو تعديلها، أو الاختبار محليًا، أو النشر إلى Render.
