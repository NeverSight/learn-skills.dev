Принципы проектирования для создания API-интерфейсов HAL+JSON, которые агенты могут исследовать и использовать в детерминированных клиентах. Использование при проектировании, создании, проверке или улучшении API с гиперссылками — охватывает точки входа, отношения ссылок, CURIE, отношения версий, документацию отношений, устаревание, встроенные ресурсы и постепенное раскрытие информации.
