JavaScript/TypeScript プロジェクトでテストを作成、構成、実行するための包括的な Vitest テスト フレームワーク スキル。クロードが次のことを行う必要がある場合に使用します: 新規または既存のプロジェクト (Vite または非 Vite) で Vitest をセットアップする、Vitest API を使用してテストを作成または変更する、特定のシナリオ (カバレッジ、ブラウザ テスト、モックなど) に合わせて Vitest を構成する、Jest または古い Vitest バージョンから移行する、テストの失敗または構成の問題をデバッグする、高度なテスト パターン (ワークスペース、ブラウザ モード、スナップショット、モッキング) を実装する。
