Compétence complète du cadre de test Vitest pour écrire, configurer et exécuter des tests dans des projets JavaScript/TypeScript. À utiliser lorsque Claude a besoin de : configurer Vitest dans un projet nouveau ou existant (Vite ou non-Vite), écrire ou modifier des tests à l'aide des API Vitest, configurer Vitest pour des scénarios spécifiques (couverture, tests de navigateur, moquerie, etc.), migrer à partir de Jest ou d'anciennes versions de Vitest, déboguer des échecs de test ou des problèmes de configuration, implémenter des modèles de tests avancés (espace de travail, mode navigateur, instantanés, moquerie).
