Habilidad integral del marco de pruebas de Vitest para escribir, configurar y ejecutar pruebas en proyectos de JavaScript/TypeScript. Úselo cuando Claude necesite: configurar Vitest en un proyecto nuevo o existente (Vite o no Vite), escribir o modificar pruebas usando las API de Vitest, configurar Vitest para escenarios específicos (cobertura, pruebas de navegador, burlas, etc.), migrar desde Jest o versiones anteriores de Vitest, depurar fallas de pruebas o problemas de configuración, implementar patrones de prueba avanzados (espacio de trabajo, modo de navegador, instantáneas, burlas).
