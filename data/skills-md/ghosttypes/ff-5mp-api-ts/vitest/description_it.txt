Competenze complete del framework di test Vitest per la scrittura, la configurazione e l'esecuzione di test in progetti JavaScript/TypeScript. Utilizzare quando Claude deve: configurare Vitest in un progetto nuovo o esistente (Vite o non Vite), scrivere o modificare test utilizzando le API Vitest, configurare Vitest per scenari specifici (copertura, test del browser, mocking, ecc.), migrare da Jest o versioni precedenti di Vitest, eseguire il debug di errori di test o problemi di configurazione, implementare modelli di test avanzati (area di lavoro, modalità browser, istantanee, mocking).
