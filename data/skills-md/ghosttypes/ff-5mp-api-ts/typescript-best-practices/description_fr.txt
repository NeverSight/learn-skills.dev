Meilleures pratiques et modèles TypeScript pour écrire du code de type sécurisé et maintenable. À utiliser lorsque vous travaillez avec des fichiers TypeScript, configurez tsconfig, définissez des interfaces/types, implémentez la gestion des erreurs, écrivez des génériques ou configurez des modèles de communication sécurisés. Comprend des modèles de syndicats discriminés, de gardes de types, de types de services publics, etc.
