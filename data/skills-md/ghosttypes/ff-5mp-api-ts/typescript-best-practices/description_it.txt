Best practice e modelli TypeScript per la scrittura di codice indipendente dai tipi e gestibile. Da utilizzare quando si lavora con file TypeScript, si configura tsconfig, si definiscono interfacce/tipi, si implementa la gestione degli errori, si scrivono generici o si impostano modelli di comunicazione indipendenti dai tipi. Include modelli per sindacati discriminati, tipi di protezione, tipi di servizi pubblici e altro ancora.
