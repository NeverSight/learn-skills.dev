Best Practices und Muster für TypeScript zum Schreiben von typsicherem, wartbarem Code. Verwenden Sie es, wenn Sie mit TypeScript-Dateien arbeiten, tsconfig konfigurieren, Schnittstellen/Typen definieren, Fehlerbehandlung implementieren, Generika schreiben oder typsichere Kommunikationsmuster einrichten. Enthält Muster für diskriminierte Gewerkschaften, Typwächter, Versorgungstypen und mehr.
