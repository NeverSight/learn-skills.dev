Mejores prácticas y patrones de TypeScript para escribir código mantenible y con seguridad de escritura. Úselo cuando trabaje con archivos TypeScript, configure tsconfig, defina interfaces/tipos, implemente manejo de errores, escriba genéricos o configure patrones de comunicación con seguridad de tipos. Incluye patrones para sindicatos discriminados, tipos de protección, tipos de servicios públicos y más.
