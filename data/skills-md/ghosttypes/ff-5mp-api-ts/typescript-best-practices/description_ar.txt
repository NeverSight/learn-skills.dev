أفضل ممارسات وأنماط TypeScript لكتابة تعليمات برمجية آمنة وقابلة للصيانة. يُستخدم عند العمل مع ملفات TypeScript، أو تكوين tsconfig، أو تحديد الواجهات/الأنواع، أو تنفيذ معالجة الأخطاء، أو كتابة الأدوية العامة، أو إعداد أنماط الاتصال الآمنة للنوع. يتضمن أنماطًا للنقابات التمييزية، وأنواع الحراسة، وأنواع المرافق، والمزيد.
