Abilità di creazione interattiva di sub-agenti per Claude Code. Da utilizzare quando l'utente desidera creare un agente secondario personalizzato o menziona la necessità di un agente specializzato per attività specifiche. Questa competenza guida l'intero processo di creazione di un subagente, inclusa la progettazione dell'identificatore, la generazione di prompt di sistema, la selezione di competenze/contesto e la scrittura di file agente formattati correttamente in .claude/agents.
