Interaktive Fähigkeit zur Erstellung von Subagenten für Claude Code. Verwenden Sie diese Option, wenn der Benutzer einen benutzerdefinierten Subagenten erstellen möchte oder erwähnt, dass er für bestimmte Aufgaben einen spezialisierten Agenten benötigt. Dieser Skill leitet den gesamten Prozess der Subagentenerstellung, einschließlich des Identifikatordesigns, der Generierung von Systemaufforderungen, der Skill-/Kontextauswahl und dem Schreiben ordnungsgemäß formatierter Agentendateien in .claude/agents.
