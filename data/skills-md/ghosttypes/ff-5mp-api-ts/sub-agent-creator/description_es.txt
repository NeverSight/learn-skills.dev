Habilidad interactiva de creación de subagentes para Claude Code. Úselo cuando el usuario quiera crear un subagente personalizado o mencione que necesita un agente especializado para tareas específicas. Esta habilidad guía todo el proceso de creación de subagentes, incluido el diseño de identificadores, la generación de mensajes del sistema, la selección de habilidades/contexto y la escritura de archivos de agentes con el formato adecuado en .claude/agents.
