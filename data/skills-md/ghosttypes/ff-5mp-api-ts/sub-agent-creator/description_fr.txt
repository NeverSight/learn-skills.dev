Compétence interactive de création de sous-agents pour Claude Code. À utiliser lorsque l'utilisateur souhaite créer un sous-agent personnalisé ou mentionne avoir besoin d'un agent spécialisé pour des tâches spécifiques. Cette compétence guide l'ensemble du processus de création de sous-agents, y compris la conception des identifiants, la génération des invites système, la sélection des compétences/contextes et l'écriture de fichiers d'agent correctement formatés dans .claude/agents.
