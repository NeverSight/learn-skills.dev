Навык создания интерактивного субагента для Claude Code. Используйте, когда пользователь хочет создать собственный субагент или упоминает, что ему нужен специализированный агент для конкретных задач. Этот навык управляет всем процессом создания субагента, включая разработку идентификатора, генерацию системных подсказок, выбор навыка/контекста и запись правильно отформатированных файлов агента в .claude/agents.
