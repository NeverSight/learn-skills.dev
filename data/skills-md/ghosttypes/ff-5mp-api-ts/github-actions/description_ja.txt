包括的な GitHub Actions ワークフロー オーサリング スキル。 Claude は、最新のベスト プラクティス、構文、セキュリティ パターンを使用して実稼働グレードの CI/CD ワークフローを作成できるようになります。 GitHub Actions ワークフローの作成または変更、CI/CD パイプラインの設計、ワークフロー トリガー/イベント/スケジュールの構成、セキュリティ パターン (シークレット、OIDC、証明書) の実装、カスタム アクションの作成、他の CI システムからの移行、失敗したワークフローのトラブルシューティング、または GitHub Actions 自動化タスクの場合に使用します。公式 GitHub ドキュメントから抽出された完全なリファレンス ドキュメントと、検証用のソース URL が含まれています。
