مهارة تأليف سير عمل GitHub الشاملة. يمكّن Claude من كتابة سير عمل CI/CD على مستوى الإنتاج باستخدام أحدث أفضل الممارسات وبناء الجملة وأنماط الأمان. يُستخدم عند إنشاء أو تعديل مسارات عمل GitHub Actions، أو تصميم خطوط أنابيب CI/CD، أو تكوين مشغلات/أحداث/جداول سير العمل، أو تنفيذ أنماط الأمان (الأسرار، OIDC، الشهادات)، أو إنشاء إجراءات مخصصة، أو الترحيل من أنظمة CI الأخرى، أو استكشاف أخطاء سير العمل الفاشلة وإصلاحها، أو أي مهمة أتمتة لـ GitHub Actions. يتضمن وثائق مرجعية كاملة مستخرجة من مستندات GitHub الرسمية مع عناوين URL المصدر للتحقق منها.
