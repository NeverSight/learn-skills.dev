Umfassende Fähigkeiten zur Erstellung von GitHub Actions-Workflows. Ermöglicht Claude das Schreiben von CI/CD-Workflows in Produktionsqualität mit den neuesten Best Practices, Syntax und Sicherheitsmustern. Verwendung beim Erstellen oder Ändern von GitHub Actions-Workflows, beim Entwerfen von CI/CD-Pipelines, beim Konfigurieren von Workflow-Triggern/Ereignissen/Zeitplänen, beim Implementieren von Sicherheitsmustern (Geheimnisse, OIDC, Attestierungen), beim Erstellen benutzerdefinierter Aktionen, beim Migrieren von anderen CI-Systemen, bei der Fehlerbehebung bei fehlgeschlagenen Workflows oder bei jeder GitHub Actions-Automatisierungsaufgabe. Enthält eine vollständige Referenzdokumentation, die aus offiziellen GitHub-Dokumenten extrahiert wurde, mit Quell-URLs zur Überprüfung.
