Compétence complète de création de flux de travail GitHub Actions. Permet à Claude d'écrire des flux de travail CI/CD de qualité production avec les dernières meilleures pratiques, syntaxes et modèles de sécurité. À utiliser lors de la création ou de la modification de flux de travail GitHub Actions, de la conception de pipelines CI/CD, de la configuration de déclencheurs/événements/planifications de flux de travail, de la mise en œuvre de modèles de sécurité (secrets, OIDC, attestations), de la création d'actions personnalisées, de la migration à partir d'autres systèmes CI, du dépannage des flux de travail ayant échoué ou de toute tâche d'automatisation des actions GitHub. Comprend une documentation de référence complète extraite des documents officiels GitHub avec les URL sources pour vérification.
