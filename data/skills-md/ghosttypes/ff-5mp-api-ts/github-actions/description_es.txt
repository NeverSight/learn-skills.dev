Habilidad integral de creación de flujos de trabajo de GitHub Actions. Permite a Claude escribir flujos de trabajo de CI/CD de nivel de producción con las mejores prácticas, sintaxis y patrones de seguridad más recientes. Úselo al crear o modificar flujos de trabajo de GitHub Actions, diseñar canalizaciones de CI/CD, configurar activadores/eventos/programaciones de flujos de trabajo, implementar patrones de seguridad (secretos, OIDC, atestados), crear acciones personalizadas, migrar desde otros sistemas de CI, solucionar problemas de flujos de trabajo fallidos o cualquier tarea de automatización de GitHub Actions. Incluye documentación de referencia completa extraída de documentos oficiales de GitHub con URL de origen para verificación.
