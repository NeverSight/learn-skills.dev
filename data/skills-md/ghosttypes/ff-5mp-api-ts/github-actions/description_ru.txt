Комплексные навыки разработки рабочих процессов GitHub Actions. Позволяет Клоду писать рабочие процессы CI/CD производственного уровня с использованием новейших лучших практик, синтаксиса и шаблонов безопасности. Используйте при создании или изменении рабочих процессов GitHub Actions, проектировании конвейеров CI/CD, настройке триггеров/событий/расписаний рабочих процессов, реализации шаблонов безопасности (секреты, OIDC, аттестации), создании настраиваемых действий, миграции из других систем CI, устранении неполадок в рабочих процессах или любых задачах автоматизации GitHub Actions. Включает полную справочную документацию, извлеченную из официальных документов GitHub, с исходными URL-адресами для проверки.
