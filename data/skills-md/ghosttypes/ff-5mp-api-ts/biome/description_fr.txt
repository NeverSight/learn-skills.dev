Intégration complète de Biome (biomejs.dev) pour le développement professionnel TypeScript/JavaScript. À utiliser pour le peluchage, le formatage, la qualité du code et l’intégration parfaite de Biome dans les bases de code. Couvre l'installation, la configuration, la migration depuis ESLint/Prettier, toutes les règles linter, les options de formatage, l'utilisation de la CLI, l'intégration de l'éditeur, la configuration monorepo et l'intégration CI/CD. À utiliser lorsque vous travaillez avec les outils Biome, configurez biome.json, configurez le peluchage/formattage, migrez des projets, déboguez des problèmes Biome ou implémentez des flux de travail Biome prêts pour la production.
