ترحيل المشاريع من npm إلى pnpm بما في ذلك تحويل ملف القفل وإعداد مساحة العمل وتحديثات CI/CD واستكشاف الأخطاء وإصلاحها. يُستخدم عند تحويل حزمة واحدة من npm إلى pnpm، أو ترحيل مساحات عمل npm أو monorepos، أو تحديث خطوط أنابيب CI/CD لـ pnpm، أو استكشاف المشكلات وإصلاحها بعد ترحيل npm إلى pnpm، أو تحويل package-lock.json إلى pnpm-lock.yaml.
