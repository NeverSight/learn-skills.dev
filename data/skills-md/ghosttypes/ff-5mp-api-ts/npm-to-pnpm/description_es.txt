Migre proyectos de npm a pnpm, incluida la conversión de archivos de bloqueo, la configuración del espacio de trabajo, las actualizaciones de CI/CD y la resolución de problemas. Úselo al convertir un solo paquete de npm a pnpm, migrar espacios de trabajo de npm o monorepos, actualizar canalizaciones de CI/CD para pnpm, solucionar problemas después de la migración de npm a pnpm o convertir package-lock.json a pnpm-lock.yaml.
