Migra progetti da npm a pnpm, inclusa la conversione dei file di blocco, la configurazione dell'area di lavoro, gli aggiornamenti CI/CD e la risoluzione dei problemi. Da utilizzare durante la conversione di un singolo pacchetto da npm a pnpm, la migrazione di spazi di lavoro npm o monorepos, l'aggiornamento di pipeline CI/CD per pnpm, la risoluzione dei problemi dopo la migrazione da npm a pnpm o la conversione di package-lock.json in pnpm-lock.yaml.
