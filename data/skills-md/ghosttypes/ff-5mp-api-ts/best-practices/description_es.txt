Mejores prácticas universales de ingeniería de software para cualquier lenguaje o proyecto. Úselo al programar, refactorizar código, diseñar sistemas, revisar código o garantizar una calidad lista para producción. Cubre principios fundamentales (SOLID, DRY, KISS, YAGNI), patrones arquitectónicos (SoC, SSOT, LoD, CQS, DbC) y confiabilidad operativa (ETC, PoLP, CoC, Idempotencia). Aplíquelo al crear nuevas funciones, limpiar bases de código, mejorar la capacidad de mantenimiento o garantizar que el código siga los estándares de la industria.
