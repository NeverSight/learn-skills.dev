أفضل الممارسات العالمية لهندسة البرمجيات لأي لغة أو مشروع. يُستخدم عند البرمجة، أو إعادة بناء التعليمات البرمجية، أو تصميم الأنظمة، أو مراجعة التعليمات البرمجية، أو ضمان جودة جاهزة للإنتاج. يغطي المبادئ الأساسية (SOLID، DRY، KISS، YAGNI)، والأنماط المعمارية (SoC، SSOT، LoD، CQS، DbC)، والموثوقية التشغيلية (ETC، PoLP، CoC، Idempotency). يتم تطبيقه عند إنشاء ميزات جديدة، أو تنظيف قواعد التعليمات البرمجية، أو تحسين إمكانية الصيانة، أو التأكد من أن التعليمات البرمجية تتبع معايير الصناعة.
