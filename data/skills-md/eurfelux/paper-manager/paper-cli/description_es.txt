Guía para utilizar la herramienta CLI `paper`: un sistema local de gestión de trabajos académicos con búsqueda vectorial impulsada por IA. Utilice esta habilidad siempre que el usuario quiera administrar artículos académicos, crear bases de conocimientos, agregar archivos (PDF, TXT, MD, TEX, etc.) a una base de conocimientos, buscar artículos semánticamente, configurar modelos de incrustación o administrar metadatos y notas de literatura. También se activa cuando el usuario menciona CLI "en papel", bases de conocimiento para investigación, gestión de literatura o desea consultar su colección de artículos. Even if the user just says something like "add this PDF", "add this text file", or "search my papers" in a project that uses paper-manager, this skill should activate.
