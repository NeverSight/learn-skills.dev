Guida per l'utilizzo dello strumento CLI "paper": un sistema locale di gestione dei documenti accademici con ricerca vettoriale basata sull'intelligenza artificiale. Utilizza questa competenza ogni volta che l'utente desidera gestire documenti accademici, creare basi di conoscenza, aggiungere file (PDF, TXT, MD, TEX, ecc.) a una base di conoscenza, cercare documenti semanticamente, configurare modelli di incorporamento o gestire metadati e note della letteratura. Si attiva anche quando l'utente menziona CLI "cartacea", basi di conoscenza per la ricerca, gestione della letteratura o desidera interrogare la propria raccolta di documenti. Anche se l'utente dice semplicemente qualcosa come "aggiungi questo PDF", "aggiungi questo file di testo" o "cerca i miei documenti" in un progetto che utilizza Paper Manager, questa abilità dovrebbe attivarsi.
