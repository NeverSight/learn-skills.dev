AI 기반 벡터 검색 기능을 갖춘 로컬 학술 논문 관리 시스템인 'paper' CLI 도구 사용 가이드입니다. 사용자가 학술 논문 관리, 지식 베이스 생성, 지식 베이스에 파일(PDF, TXT, MD, TEX 등) 추가, 의미론적 논문 검색, 임베딩 모델 구성, 문헌 메타데이터 및 메모 관리를 원할 때마다 이 기술을 사용하세요. 또한 사용자가 "종이" CLI, 연구용 지식 기반, 문헌 관리를 언급하거나 종이 컬렉션을 쿼리하려는 경우에도 트리거됩니다. paper-manager를 사용하는 프로젝트에서 사용자가 "이 PDF 추가", "이 텍스트 파일 추가", "내 논문 검색"과 같은 말만 해도 이 스킬이 활성화됩니다.
