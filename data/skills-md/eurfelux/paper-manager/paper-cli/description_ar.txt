دليل لاستخدام أداة CLI "الورقية" - نظام محلي لإدارة الأوراق الأكاديمية مع بحث متجه مدعوم بالذكاء الاصطناعي. استخدم هذه المهارة عندما يريد المستخدم إدارة الأوراق الأكاديمية، أو إنشاء قواعد معرفية، أو إضافة ملفات (PDF، TXT، MD، TEX، وما إلى ذلك) إلى قاعدة المعرفة، أو البحث في الأوراق بشكل دلالي، أو تكوين نماذج التضمين، أو إدارة البيانات التعريفية والملاحظات الأدبية. يتم تشغيله أيضًا عندما يذكر المستخدم واجهة سطر الأوامر "الورقية"، أو قواعد المعرفة للبحث، أو إدارة الأدبيات، أو يريد الاستعلام عن مجموعته الورقية. حتى لو قال المستخدم شيئًا مثل "أضف ملف PDF هذا" أو "أضف هذا الملف النصي" أو "ابحث في أوراقي" في مشروع يستخدم مدير الورق، فيجب تنشيط هذه المهارة.
