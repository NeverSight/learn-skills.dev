このスキルを使用して、会話全体の思い出を保存、呼び出し、整理します。トリガー: 「これを覚えておいてください」、「これを保存してください」、「これをメモしてください」、「何について話し合ったか...」、「メモを確認してください」、「覚えていますか」、「思い出してください」。また、ユーザーが以前の作業を再開したり、過去の決定を参照したりしているように見える場合、または将来のセッションのために本当に保存する価値のあるものを発見した場合にも、積極的に使用してください。このスキルはコードに限定されるものではありません。ビジネス上の意思決定、個人的なメモ、会議の要約、調査、プロジェクト管理、クリエイティブな仕事、顧客履歴など、あらゆるものに使用できます。
