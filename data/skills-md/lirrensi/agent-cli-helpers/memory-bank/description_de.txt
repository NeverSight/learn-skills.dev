Verwenden Sie es auch proaktiv, wenn der Benutzer scheinbar frühere Arbeiten fortsetzt, auf frühere Entscheidungen verweist oder wenn Sie etwas entdecken, das es wirklich wert ist, für zukünftige Sitzungen aufbewahrt zu werden. Diese Fähigkeit ist NICHT auf Code beschränkt – verwenden Sie sie für Geschäftsentscheidungen, persönliche Notizen, Besprechungsrückblicke, Recherche, Projektmanagement, kreative Arbeit, Kundenhistorie usw.
