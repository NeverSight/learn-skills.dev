Utilisez cette compétence pour enregistrer, rappeler ou organiser des souvenirs au fil des conversations. Déclenchez : "souvenez-vous de ceci", "enregistrez ceci", "notez ceci", "de quoi avons-nous discuté de...", "vérifiez vos notes", "vous en souvenez-vous", "rappelez-vous". Utilisez également de manière proactive lorsque l'utilisateur semble reprendre un travail précédent, faire référence à des décisions passées ou lorsque vous découvrez quelque chose qui mérite vraiment d'être préservé pour les sessions futures. Cette compétence ne se limite PAS au code : utilisez-la pour les décisions commerciales, les notes personnelles, les récapitulatifs de réunions, la recherche, la gestion de projet, le travail créatif, l'historique des clients, etc.
