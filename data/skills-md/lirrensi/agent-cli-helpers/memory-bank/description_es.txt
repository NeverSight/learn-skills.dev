Utilice esta habilidad para guardar, recordar u organizar recuerdos de conversaciones. Activar: 'recuerda esto', 'guarda esto', 'nota esto', 'sobre qué discutimos...', 'revisa tus notas', 'te acuerdas', 'recuerda'. Úselo también de manera proactiva cuando el usuario parezca estar reanudando un trabajo anterior, haciendo referencia a decisiones pasadas o cuando descubra algo que realmente vale la pena conservar para sesiones futuras. Esta habilidad NO se limita al código: úsela para decisiones comerciales, notas personales, resúmenes de reuniones, investigaciones, gestión de proyectos, trabajo creativo, historial de clientes, cualquier cosa.
