Отправляйте уведомления на рабочий стол с терминала. Используйте этот навык, когда пользователь хочет отображать системные уведомления, передавать выходные данные команд в уведомления или получать оповещения после завершения длительных операций. Поддерживает кроссплатформенность (Windows Toast, macOS osascript, отправка уведомлений Linux).
