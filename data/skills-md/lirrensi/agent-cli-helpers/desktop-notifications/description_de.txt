Senden Sie Desktop-Benachrichtigungen vom Terminal. Verwenden Sie diesen Skill, wenn der Benutzer Systembenachrichtigungen anzeigen, Befehlsausgaben an Benachrichtigungen weiterleiten oder Warnungen erhalten möchte, nachdem lang andauernde Vorgänge abgeschlossen sind. Unterstützt plattformübergreifend (Windows Toast, macOS Osascript, Linux Notify-Send).
