Machen Sie Screenshots vom Terminal mit dem Screenshot-CLI-Tool. Verwenden Sie diese Fähigkeit, wenn der Benutzer den Bildschirm oder einen Bereich erfassen, Screenshots in Dateien speichern oder die Screenshot-Erfassung in Skripts integrieren möchte. Unterstützt plattformübergreifend (Windows/macOS/Linux) mit automatischem Fallback auf native Tools.
