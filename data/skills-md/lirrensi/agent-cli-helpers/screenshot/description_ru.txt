Сделайте снимки экрана с терминала с помощью инструмента CLI для снимков экрана. Используйте этот навык, когда пользователь хочет сделать снимок экрана или области, сохранить снимки экрана в файлы или интегрировать снимок экрана в сценарии. Поддерживает кроссплатформенность (Windows/macOS/Linux) с автоматическим возвратом к собственным инструментам.
