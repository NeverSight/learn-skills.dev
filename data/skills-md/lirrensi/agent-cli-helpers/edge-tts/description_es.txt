Genere scripts TTS (Text-to-Speech) completos y listos para producción y herramientas CLI utilizando edge-tts, el motor de voz neuronal de Microsoft con más de 400 voces con sonido natural. Utilice esta habilidad siempre que el usuario desee pronunciar texto en voz alta desde la terminal/PowerShell, enumerar o buscar voces, seleccionar voces por nombre/idioma/género, controlar la velocidad/volumen/tono, guardar la voz en archivos MP3/audio o canalizar texto en un comando de voz. Activa CUALQUIER solicitud que involucre TTS, síntesis de voz, comando de decir, función de hablar, salida de voz, edge-tts o audio de guiones, incluso pequeños fragmentos. Siempre verifique primero la instalación y prefiera la instalación de la herramienta uv para la configuración.
