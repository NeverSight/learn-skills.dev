Genera script TTS (Text-to-Speech) completi e pronti per la produzione e strumenti CLI utilizzando edge-tts, il motore vocale neurale di Microsoft con oltre 400 voci dal suono naturale. Utilizza questa abilità ogni volta che l'utente desidera pronunciare il testo ad alta voce dal terminale/PowerShell, elencare o cercare voci, selezionare voci per nome/lingua/genere, controllare velocità/volume/tono, salvare la voce in file MP3/audio o inserire testo in un comando pronuncia. Attiva QUALSIASI richiesta che coinvolga TTS, sintesi vocale, comando pronuncia, funzione parla, output vocale, edge-tts o audio da script, anche piccoli frammenti. Controlla sempre prima l'installazione e preferisci l'installazione dello strumento uv per la configurazione.
