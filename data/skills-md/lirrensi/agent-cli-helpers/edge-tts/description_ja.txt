Edge-tts (400 以上の自然な音声を備えた Microsoft のニューラル音声エンジン) を使用して、本番環境に対応した完全な TTS (Text-to-Speech) スクリプトと CLI ツールを生成します。ユーザーが端末/PowerShell からテキストを読み上げたり、音声をリストまたは検索したり、名前/言語/性別で音声を選択したり、速度/音量/ピッチを制御したり、音声を MP3/オーディオ ファイルに保存したり、テキストを音声コマンドにパイプしたりする場合は、このスキルをいつでも使用できます。 TTS、音声合成、say コマンド、speak 関数、音声出力、エッジ TTS、またはスクリプトからのオーディオを含むあらゆるリクエスト (小さなスニペットも含む) をトリガーします。常に最初にインストールを確認し、セットアップには UV ツールのインストールを優先してください。
