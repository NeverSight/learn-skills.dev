Создавайте полные, готовые к использованию сценарии TTS (преобразование текста в речь) и инструменты CLI с помощью Edge-tts — нейронного голосового движка Microsoft с более чем 400 естественно звучащими голосами. Используйте этот навык всякий раз, когда пользователь хочет произнести текст вслух с терминала/PowerShell, составить список или выполнить поиск голосов, выбрать голоса по имени/языку/полу, контролировать скорость/громкость/тональность, сохранить речь в MP3/аудиофайлы или передать текст в команду произнесения. Триггер для ЛЮБОГО запроса, включающего TTS, синтез речи, скажем, команду, функцию речи, голосовой вывод, Edge-TTS или звук из сценариев — даже небольшие фрагменты. Всегда сначала проверяйте установку и отдавайте предпочтение установке uv-инструмента для настройки.
