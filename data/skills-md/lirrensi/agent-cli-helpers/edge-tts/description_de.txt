Generieren Sie vollständige, produktionsbereite TTS-Skripte (Text-to-Speech) und CLI-Tools mit Edge-Tts – der neuronalen Sprach-Engine von Microsoft mit über 400 natürlich klingenden Stimmen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer Text vom Terminal/PowerShell aus vorsprechen, Stimmen auflisten oder suchen, Stimmen nach Name/Sprache/Geschlecht auswählen, Geschwindigkeit/Lautstärke/Tonhöhe steuern, Sprache in MP3-/Audiodateien speichern oder Text in einen Sprechbefehl einspeisen möchte. Trigger für JEDE Anfrage, die TTS, Sprachsynthese, Sagbefehl, Sprechfunktion, Sprachausgabe, Edge-TTS oder Audio aus Skripten betrifft – sogar kleine Schnipsel. Überprüfen Sie immer zuerst die Installation und bevorzugen Sie für die Einrichtung die Installation des UV-Tools.
