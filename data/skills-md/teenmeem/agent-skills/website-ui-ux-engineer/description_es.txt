Utilice esta habilidad cuando el usuario quiera diseñar o crear cualquier sitio web o interfaz de usuario web.

DISPARADORES: se activan cuando la solicitud involucra:
- Páginas de destino, sitios de marketing, carteras, interfaces SaaS
- Paneles de control, paneles de administración, visualizaciones de datos.
- Aplicaciones web, sitios de varias páginas, bibliotecas de componentes.
- Frases como: "crear un sitio web", "crear una página de destino", "diseñar una interfaz de usuario",
  "crear una aplicación web", "crear un portafolio", "rediseñar esta página"
- Cualquier tarea donde el entregable principal sea una interfaz visual renderizada por el navegador.

COMPORTAMIENTO: esta habilidad produce resultados web completos y de calidad de producción:
- Diseños responsivos en dispositivos móviles, tabletas y computadoras de escritorio
- HTML5 semántico con accesibilidad (ARIA, contraste, navegación por teclado)
- CSS moderno a través de Tailwind o Vanilla: incluye animaciones y transiciones.
- Componentes interactivos de React o Vanilla JS.
- Sistema de diseño consistente: tipografía, fichas de color, espaciado, iconografía.
- Cero marcadores de posición o TODO: todas las secciones completamente implementadas

NO active para: lógica de backend, API REST/GraphQL, esquemas de bases de datos,
Herramientas CLI, scripts o cualquier tarea en la que el resultado no sea la interfaz de usuario del navegador.
PREFIERA a las habilidades de codificación genéricas siempre que se espere un resultado web visual.
