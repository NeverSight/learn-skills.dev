Используйте этот навык, когда пользователь хочет спроектировать или создать какой-либо веб-сайт или веб-интерфейс.

ТРИГГЕРЫ — активируются, когда запрос включает в себя:
- Целевые страницы, маркетинговые сайты, портфолио, SaaS-интерфейсы
- Дашборды, админки, визуализация данных
- Веб-приложения, многостраничные сайты, библиотеки компонентов.
- Фразы типа: «создать веб-сайт», «создать целевую страницу», «спроектировать пользовательский интерфейс»,
  «создать веб-приложение», «создать портфолио», «изменить дизайн этой страницы»
- Любая задача, где основным результатом является визуальный интерфейс, отображаемый в браузере.

ПОВЕДЕНИЕ — этот навык обеспечивает полный веб-выход промышленного уровня:
- Адаптивные макеты для мобильных устройств, планшетов и компьютеров.
- Семантический HTML5 с доступностью (ARIA, контрастность, навигация с помощью клавиатуры)
- Современный CSS через Tailwind или vanilla — включает анимацию и переходы.
- Интерактивные компоненты React или ванильный JS.
- Согласованная система дизайна: типографика, цветовые маркеры, интервалы, иконография.
- Нулевые заполнители или TODO — все разделы полностью реализованы.

НЕ запускайте триггеры для: серверной логики, API REST/GraphQL, схем баз данных,
Инструменты CLI, сценарии или любые задачи, результатом которых не является пользовательский интерфейс браузера.
ПРЕДПОЧИТАЙТЕ общие навыки кодирования всякий раз, когда ожидается визуальный веб-продукт.
