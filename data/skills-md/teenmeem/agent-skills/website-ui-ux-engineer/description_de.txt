Verwenden Sie diese Fähigkeit, wenn der Benutzer eine Website oder Web-Benutzeroberfläche entwerfen oder erstellen möchte.

TRIGGERS – aktivieren, wenn die Anfrage Folgendes betrifft:
- Landingpages, Marketingseiten, Portfolios, SaaS-Frontends
- Dashboards, Admin-Panels, Datenvisualisierungen
- Web-Apps, mehrseitige Websites, Komponentenbibliotheken
- Sätze wie: „eine Website erstellen“, „eine Zielseite erstellen“, „eine Benutzeroberfläche entwerfen“,
  „Eine Web-App erstellen“, „ein Portfolio erstellen“, „Diese Seite neu gestalten“
– Jede Aufgabe, bei der das primäre Ergebnis eine visuelle, vom Browser gerenderte Schnittstelle ist

VERHALTEN – Diese Fähigkeit erzeugt eine vollständige Webausgabe in Produktionsqualität:
- Responsive Layouts für Mobilgeräte, Tablets und Desktops
- Semantisches HTML5 mit Barrierefreiheit (ARIA, Kontrast, Tastaturnavigation)
- Modernes CSS über Tailwind oder Vanilla – beinhaltet Animationen und Übergänge
- Interactive React oder Vanilla JS-Komponenten
- Konsistentes Designsystem: Typografie, Farbtoken, Abstände, Ikonografie
- Keine Platzhalter oder TODOs – alle Abschnitte vollständig implementiert

NICHT auslösen für: Backend-Logik, REST/GraphQL-APIs, Datenbankschemata,
CLI-Tools, Skripte oder jede Aufgabe, bei der keine Browser-Benutzeroberfläche die Ausgabe ist.
Bevorzugen Sie generische Programmierkenntnisse, wenn ein visuelles Webergebnis erwartet wird.
