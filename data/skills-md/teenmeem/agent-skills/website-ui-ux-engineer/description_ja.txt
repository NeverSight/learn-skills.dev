ユーザーが Web サイトまたは Web UI を設計または構築する場合は、このスキルを使用します。

TRIGGERS — リクエストに以下が含まれる場合にアクティブ化されます。
- ランディング ページ、マーケティング サイト、ポートフォリオ、SaaS フロントエンド
- ダッシュボード、管理パネル、データの視覚化
- Web アプリ、複数ページのサイト、コンポーネント ライブラリ
- 「Web サイトを構築する」、「ランディング ページを作成する」、「UI をデザインする」などのフレーズ
  「Web アプリを作成する」、「ポートフォリオを作成する」、「このページを再デザインする」
- 主な成果物がブラウザでレンダリングされた視覚的なインターフェイスであるタスク

BEHAVIOR — このスキルは、完全な運用グレードの Web 出力を生成します。
- モバイル、タブレット、デスクトップにわたるレスポンシブなレイアウト
- アクセシビリティを備えたセマンティック HTML5 (ARIA、コントラスト、キーボード ナビゲーション)
- Tailwind またはバニラによる最新の CSS — アニメーションとトランジションを含む
- インタラクティブな React またはバニラ JS コンポーネント
- 一貫したデザイン システム: タイポグラフィー、カラー トークン、間隔、図像
- プレースホルダーや TODO はゼロ — すべてのセクションが完全に実装されています

バックエンド ロジック、REST/GraphQL API、データベース スキーマ、
CLI ツール、スクリプト、またはブラウザー UI を出力しないタスク。
ビジュアルな Web 成果物が期待される場合は、一般的なコーディング スキルよりも優先されます。
