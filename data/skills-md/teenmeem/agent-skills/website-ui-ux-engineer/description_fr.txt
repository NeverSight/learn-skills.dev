Utilisez cette compétence lorsque l'utilisateur souhaite concevoir ou créer un site Web ou une interface utilisateur Web.

DÉCLENCHEURS – s'activent lorsque la demande implique :
- Pages de destination, sites marketing, portefeuilles, frontends SaaS
- Tableaux de bord, panneaux d'administration, visualisations de données
- Applications Web, sites multipages, bibliothèques de composants
- Des expressions telles que : "créer un site Web", "créer une page de destination", "concevoir une interface utilisateur",
  "créer une application Web", "créer un portfolio", "repenser cette page"
- Toute tâche dont le livrable principal est une interface visuelle rendue par un navigateur

COMPORTEMENT – cette compétence produit une sortie Web complète de qualité production :
- Mises en page réactives sur mobile, tablette et ordinateur de bureau
- HTML5 sémantique avec accessibilité (ARIA, contraste, navigation clavier)
- CSS moderne via Tailwind ou Vanilla - inclut des animations et des transitions
- Composants Interactive React ou Vanilla JS
- Système de conception cohérent : typographie, jetons de couleur, espacement, iconographie
- Zéro espace réservé ou TODO - toutes les sections entièrement mises en œuvre

NE PAS déclencher pour : la logique backend, les API REST/GraphQL, les schémas de base de données,
Outils CLI, scripts ou toute tâche pour laquelle aucune interface utilisateur de navigateur n'est la sortie.
PRÉFÉREZ les compétences de codage génériques chaque fois qu'un livrable Web visuel est attendu.
