استخدم هذه المهارة عندما يريد المستخدم تصميم أو بناء أي موقع ويب أو واجهة مستخدم ويب.

TRIGGERS — يتم تفعيلها عندما يتضمن الطلب ما يلي:
- الصفحات المقصودة، مواقع التسويق، المحافظ، الواجهات الأمامية SaaS
- لوحات المعلومات ولوحات الإدارة وتصورات البيانات
- تطبيقات الويب والمواقع متعددة الصفحات ومكتبات المكونات
- عبارات مثل: "إنشاء موقع ويب"، "إنشاء صفحة مقصودة"، "تصميم واجهة مستخدم"،
  "إنشاء تطبيق ويب"، "إنشاء محفظة"، "إعادة تصميم هذه الصفحة"
- أي مهمة حيث يكون التسليم الأساسي عبارة عن واجهة مرئية يعرضها المتصفح

السلوك — تنتج هذه المهارة مخرجات ويب كاملة على مستوى الإنتاج:
- تخطيطات سريعة الاستجابة عبر الهاتف المحمول والكمبيوتر اللوحي وسطح المكتب
- HTML5 الدلالي مع إمكانية الوصول (ARIA، والتباين، والتنقل في لوحة المفاتيح)
- CSS حديث عبر Tailwind أو Vanilla - يتضمن رسومًا متحركة وانتقالات
- مكونات التفاعلية React أو Vanilla JS
- نظام تصميم متناسق: الطباعة، الرموز الملونة، المسافات، الأيقونية
- لا توجد عناصر نائبة أو مهام - تم تنفيذ جميع الأقسام بالكامل

لا تقم بتشغيل: منطق الواجهة الخلفية، وواجهات برمجة تطبيقات REST/GraphQL، ومخططات قاعدة البيانات،
أدوات CLI أو البرامج النصية أو أي مهمة لا تكون فيها واجهة مستخدم المتصفح هي المخرجات.
تفضل على مهارات الترميز العامة عندما يكون من المتوقع تسليم الويب المرئي.
