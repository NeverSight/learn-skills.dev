当用户想要设计或构建任何网站或 Web UI 时，请使用此技能。

TRIGGERS——当请求涉及以下内容时激活：
- 登陆页面、营销网站、产品组合、SaaS 前端
- 仪表板、管理面板、数据可视化
- Web 应用程序、多页面网站、组件库
- 类似这样的短语：“构建网站”、“创建登陆页面”、“设计 UI”、
  “制作网络应用程序”、“构建作品集”、“重新设计此页面”
- 主要交付成果是可视化、浏览器渲染界面的任何任务

行为 — 此技能可生成完整的、生产级的 Web 输出：
- 跨移动设备、平板电脑和桌面的响应式布局
- 具有辅助功能的语义 HTML5（ARIA、对比度、键盘导航）
- 通过 Tailwind 或 vanilla 实现的现代 CSS — 包括动画和过渡
- 交互式 React 或 vanilla JS 组件
- 一致的设计系统：版式、颜色标记、间距、图像
- 零占位符或 TODO — 所有部分均已完全实施

不要触发：后端逻辑、REST/GraphQL API、数据库模式、
CLI 工具、脚本或任何没有浏览器 UI 输出的任务。
每当需要视觉网络交付时，优先考虑通用编码技能。
