사용자가 웹 사이트나 웹 UI를 디자인하거나 구축하려는 경우 이 기술을 사용하세요.

TRIGGERS — 요청에 다음이 포함될 때 활성화됩니다.
- 랜딩 페이지, 마케팅 사이트, 포트폴리오, SaaS 프런트엔드
- 대시보드, 관리 패널, 데이터 시각화
- 웹 앱, 다중 페이지 사이트, 구성 요소 라이브러리
- 다음과 같은 문구: "웹사이트 구축", "랜딩 페이지 생성", "UI 디자인",
  "웹 앱 만들기", "포트폴리오 구축", "이 페이지 디자인 변경"
- 기본 결과물이 시각적인 브라우저 렌더링 인터페이스인 모든 작업

행동 — 이 기술은 완전한 프로덕션 수준의 웹 출력을 생성합니다.
- 모바일, 태블릿, 데스크톱 전반에 걸친 반응형 레이아웃
- 접근성을 갖춘 시맨틱 HTML5(ARIA, 대비, 키보드 탐색)
- Tailwind 또는 바닐라를 통한 최신 CSS - 애니메이션 및 전환 포함
- Interactive React 또는 바닐라 JS 구성요소
- 일관된 디자인 시스템: 타이포그래피, 색상 토큰, 간격, 아이콘
- 제로 자리 표시자 또는 TODO — 모든 섹션이 완전히 구현되었습니다.

백엔드 로직, REST/GraphQL API, 데이터베이스 스키마 등에 대해 트리거하지 마세요.
CLI 도구, 스크립트 또는 브라우저 UI가 출력되지 않는 모든 작업.
시각적인 웹 결과물이 필요할 때마다 일반 코딩 기술보다 선호됩니다.
