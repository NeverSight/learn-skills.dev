Utilizza questa competenza quando l'utente desidera progettare o creare qualsiasi sito Web o interfaccia utente Web.

TRIGGER — si attivano quando la richiesta coinvolge:
- Pagine di destinazione, siti di marketing, portfolio, frontend SaaS
- Dashboard, pannelli di amministrazione, visualizzazioni di dati
- App Web, siti multipagina, librerie di componenti
- Frasi come: "costruisci un sito web", "crea una landing page", "progetta un'interfaccia utente",
  "crea un'app Web", "crea un portfolio", "ridisegna questa pagina"
- Qualsiasi attività in cui il risultato finale principale è un'interfaccia visiva resa dal browser

COMPORTAMENTO: questa abilità produce output web completi e di livello produttivo:
- Layout reattivi su dispositivi mobili, tablet e desktop
- HTML5 semantico con accessibilità (ARIA, contrasto, navigazione da tastiera)
- CSS moderno tramite Tailwind o Vanilla: include animazioni e transizioni
- Componenti Interactive React o Vanilla JS
- Sistema di progettazione coerente: tipografia, gettoni di colore, spaziatura, iconografia
- Zero segnaposto o TODO: tutte le sezioni completamente implementate

NON eseguire l'attivazione per: logica backend, API REST/GraphQL, schemi di database,
Strumenti CLI, script o qualsiasi attività in cui l'output non è l'interfaccia utente del browser.
PREFERISCI rispetto a competenze di codifica generiche ogni volta che è previsto un risultato web visivo.
