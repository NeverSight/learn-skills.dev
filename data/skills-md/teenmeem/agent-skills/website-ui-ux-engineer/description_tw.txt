當使用者想要設計或建立任何網站或 Web UI 時，請使用此技能。

TRIGGERS－當請求涉及以下內容時啟動：
- 登陸頁面、行銷網站、產品組合、SaaS 前端
- 儀表板、管理面板、資料視覺化
- Web 應用程式、多頁面網站、元件庫
- 類似這樣的短語：「建立網站」、「建立登陸頁面」、「設計 UI」、
  “製作網頁應用程式”、“建立作品集”、“重新設計此頁面”
- 主要交付成果是視覺化、瀏覽器渲染介面的任何任務

行為 — 此技能可產生完整的、生產級的 Web 輸出：
- 跨行動裝置、平板電腦和桌面的響應式佈局
- 具有輔助功能的語意 HTML5（ARIA、對比、鍵盤導覽）
- 透過 Tailwind 或 vanilla 實現的現代 CSS — 包括動畫和過渡
- 互動式 React 或 vanilla JS 元件
- 一致的設計系統：版面、顏色標記、間距、影像
- 零佔位符或 TODO — 所有部分已完全實施

不要觸發：後端邏輯、REST/GraphQL API、資料庫模式、
CLI 工具、腳本或任何沒有瀏覽器 UI 輸出的任務。
每當需要視覺網路交付時，優先考慮通用編碼技能。
