Рабочий процесс для развертывания стратегических документов (например, планов финансирования, обзоров архитектуры) в репозиторий с контролем версий, создания презентации и синхронизации всех артефактов с GitHub и Google Drive. Используйте, когда пользователь предоставляет доктринальный документ и запрашивает его обработку, развертывание или представление.
