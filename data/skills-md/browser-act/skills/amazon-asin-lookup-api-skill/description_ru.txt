Этот навык помогает пользователям извлекать структурированные сведения о продукте из Amazon, используя определенный ASIN (стандартный идентификационный номер Amazon). Используйте этот навык, когда пользователь запрашивает информацию о продукте Amazon по ASIN, ищет название и цену продукта Amazon с помощью ASIN, извлекает рейтинги продуктов Amazon и количество отзывов для определенного ASIN, проверяет наличие продуктов Amazon и текущую цену, получает описание и функции продуктов Amazon через ASIN, пополняет каталог продуктов данными Amazon с помощью ASIN, отслеживает изменения цен на продукты Amazon для конкретных ASIN, получает информацию о бренде продукта Amazon и информации о материалах, получает изображения и характеристики продуктов Amazon по ASIN, проверяет Amazon ASIN и получает метаданные продукта.
