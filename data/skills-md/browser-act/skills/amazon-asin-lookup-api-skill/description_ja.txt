このスキルは、ユーザーが特定の ASIN (Amazon 標準識別番号) を使用して Amazon から構造化された商品の詳細を抽出するのに役立ちます。 ASIN による Amazon 製品の詳細の取得、ASIN を使用した Amazon 製品のタイトルと価格の検索、特定の ASIN の Amazon 製品の評価とレビュー数の抽出、Amazon 製品の在庫状況と現在の価格の確認、ASIN を介した Amazon 製品の説明と機能の取得、ASIN を使用した Amazon データによる製品カタログの強化、特定の ASIN の Amazon 製品の価格変更の監視、Amazon 製品のブランドと素材情報の取得、ASIN による Amazon 製品の画像と仕様の取得、Amazon ASIN の検証、および製品メタデータの取得をユーザーが要求する場合、このスキルを使用します。
