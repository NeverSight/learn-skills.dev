Esta habilidad ayuda a los usuarios a extraer detalles estructurados del producto de Amazon utilizando un ASIN (Número de identificación estándar de Amazon) específico. Utilice esta habilidad cuando el usuario solicite obtener detalles del producto de Amazon por ASIN, buscar el título y el precio del producto de Amazon usando ASIN, extraer calificaciones de productos de Amazon y el recuento de reseñas de un ASIN específico, verificar la disponibilidad y el precio actual del producto de Amazon, obtener la descripción y las características del producto de Amazon a través de ASIN, enriquecer el catálogo de productos con datos de Amazon usando ASIN, monitorear los cambios de precios de productos de Amazon para ASIN específicos, recuperar información sobre la marca y el material del producto de Amazon, buscar imágenes y especificaciones de productos de Amazon por ASIN, validar ASIN de Amazon y obtener metadatos de productos.
