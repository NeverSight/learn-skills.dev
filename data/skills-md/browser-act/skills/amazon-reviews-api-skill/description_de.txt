Dieser Skill hilft Benutzern, automatisch Amazon-Produktbewertungen über die Amazon Reviews API zu extrahieren. Der Agent sollte diese Fähigkeit proaktiv anwenden, wenn Benutzer Bedürfnisse äußern, wie z. B. Bewertungen für ein Amazon-Produkt mit der ASIN B07TS6R1SF zu erhalten, Kundenfeedback für einen bestimmten Amazon-Artikel zu analysieren, Bewertungen und Kommentare für ein Konkurrenzprodukt zu erhalten, die Stimmung aktueller Amazon-Bewertungen zu verfolgen, verifizierte Kaufbewertungen zur Qualitätsbewertung zu extrahieren, Benutzererfahrungen von Amazon-Produktseiten zusammenzufassen, die Produktleistung anhand von Kundenbewertungen zu überwachen, Rezensentenprofile und Links für Marktforschung zu sammeln, Bewertungstitel und Beschreibungen für die Inhaltsanalyse zu sammeln, Amazon-Bewertungen ohne Anmeldung zu extrahieren.
