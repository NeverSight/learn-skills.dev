Этот навык помогает пользователям автоматически извлекать обзоры продуктов Amazon через API Amazon Reviews. Агент должен активно применять этот навык, когда пользователи выражают такие потребности, как получение отзывов о продукте Amazon с помощью ASIN B07TS6R1SF, анализ отзывов клиентов по конкретному товару Amazon, получение оценок и комментариев к конкурентному продукту, отслеживание настроений в отношении недавних обзоров Amazon, извлечение проверенных отзывов о покупках для оценки качества, обобщение пользовательского опыта со страниц продуктов Amazon, мониторинг эффективности продуктов с помощью отзывов клиентов, сбор профилей рецензентов и ссылок для исследования рынка, сбор заголовков и описаний обзоров для анализа контента, сбор отзывов Amazon без необходимости входа в систему.
