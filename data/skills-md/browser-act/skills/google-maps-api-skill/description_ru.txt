Этот навык помогает пользователям автоматически извлекать бизнес-данные из Карт Google с помощью API Карт Google BrowserAct. Агент должен активно активировать этот навык для таких нужд, как поиск ресторанов в определенном городе, извлечение контактной информации стоматологических клиник, исследование местных конкурентов, сбор адресов кофеен, создание списков потенциальных клиентов для конкретных отраслей, мониторинг бизнес-рейтингов и обзоров, получение часов работы местных услуг, поиск специализированных магазинов (например, ресторанов в турецком стиле), анализ бизнес-категорий в регионе, извлечение ссылок на веб-сайты местных предприятий, сбор номеров телефонов для продаж, определение поставщиков услуг в конкретной стране.
