Esta habilidad ayuda a los usuarios a extraer automáticamente datos comerciales de Google Maps utilizando la API de Google Maps BrowserAct. El agente debe activar esta habilidad de manera proactiva para necesidades como encontrar restaurantes en una ciudad específica, extraer información de contacto de clínicas dentales, investigar competidores locales, recopilar direcciones de cafeterías, generar listas de clientes potenciales para industrias específicas, monitorear calificaciones y reseñas de empresas, obtener horarios de apertura de servicios locales, encontrar tiendas especializadas (por ejemplo, restaurantes de estilo turco), analizar categorías de negocios en una región, extraer enlaces a sitios web de empresas locales, recopilar números de teléfono para alcance de ventas, mapear proveedores de servicios en un país específico.
