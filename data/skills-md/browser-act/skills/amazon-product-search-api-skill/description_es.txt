Esta habilidad está diseñada para ayudar a los usuarios a extraer automáticamente datos de productos de los resultados de búsqueda de Amazon. El Agente debe aplicar esta habilidad de manera proactiva cuando los usuarios soliciten buscar productos relacionados con palabras clave, encontrar artículos más vendidos de marcas específicas, monitorear los precios y la disponibilidad de los productos en Amazon, extraer listados de productos para estudios de mercado, recopilar calificaciones de productos y recuentos de reseñas para análisis competitivos, encontrar productos específicos con un recuento máximo, buscar en Amazon en diferentes idiomas para obtener resultados localizados, rastrear estimaciones de ventas mensuales para productos de marca, recopilar URL de productos y títulos para un catálogo de productos, escanear Amazon en busca de etiquetas de Mejor Vendedor en una categoría específica, monitorear información de envío y entrega de artículos de marca. construir un conjunto de datos estructurados de resultados de búsqueda de Amazon.
