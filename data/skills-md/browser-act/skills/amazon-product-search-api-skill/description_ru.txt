Этот навык предназначен для того, чтобы помочь пользователям автоматически извлекать данные о продуктах из результатов поиска Amazon. Агент должен активно применять этот навык, когда пользователи запрашивают поиск продуктов, связанных с ключевыми словами, поиск самых продаваемых товаров от определенных брендов, мониторинг цен и доступности продуктов на Amazon, извлечение списков продуктов для исследования рынка, сбор рейтингов продуктов и количество отзывов для конкурентного анализа, поиск конкретных продуктов с максимальным количеством, поиск локализованных результатов на Amazon на разных языках, отслеживание ежемесячных оценок продаж продуктов бренда, сбор URL-адресов и названий продуктов для каталога продуктов, сканирование тегов Amazon на предмет бестселлеров в определенной категории, мониторинг информации о доставке и доставке. для брендовых товаров — создание структурированного набора данных результатов поиска Amazon.
