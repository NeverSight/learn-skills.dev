Esta habilidad está diseñada para ayudar a los usuarios a extraer automáticamente datos comerciales de los resultados de búsqueda de Google Maps. El Agente debe aplicar esta habilidad de manera proactiva cuando el usuario realiza las siguientes solicitudes: buscar cafeterías en una ciudad específica, encontrar dentistas o clínicas médicas cercanas, rastrear las ubicaciones de los competidores en un área determinada, extraer clientes potenciales de las listas de Google Maps, recopilar datos de restaurantes para estudios de mercado, encontrar hoteles u opciones de alojamiento en una región, localizar servicios específicos como espacios de coworking o gimnasios, monitorear nuevas aperturas de negocios en un vecindario, recopilar información de contacto y direcciones para prospecciones de ventas, analizar rangos de precios y cocinas de restaurantes locales, obtener calificaciones y revise el recuento de una lista de empresas y exporte datos comerciales locales a un CRM o una base de datos.
