Этот навык предназначен для того, чтобы помочь пользователям автоматически извлекать бизнес-данные из результатов поиска Google Maps. Агент должен активно применять этот навык, когда пользователь делает следующие запросы: поиск кофеен в определенном городе, поиск стоматологов или медицинских клиник поблизости, отслеживание местоположений конкурентов в определенном районе, извлечение бизнес-лидов из списков Google Maps, сбор данных о ресторанах для исследования рынка, поиск отелей или вариантов размещения в регионе, поиск конкретных услуг, таких как коворкинги или спортивные залы, мониторинг новых бизнес-открытий в районе, сбор контактной информации и адресов для поиска продаж, анализ ценового диапазона и кухни местных закусочных, получение рейтинги и количество отзывов для списка предприятий, экспорт данных о местных предприятиях в CRM или базу данных.
