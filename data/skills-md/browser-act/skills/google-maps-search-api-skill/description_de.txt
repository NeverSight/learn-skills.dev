Dieser Skill soll Benutzern dabei helfen, Geschäftsdaten automatisch aus Google Maps-Suchergebnissen zu extrahieren. Der Agent sollte diese Fähigkeit proaktiv anwenden, wenn der Benutzer die folgenden Anfragen stellt: Suche nach Cafés in einer bestimmten Stadt, Suche nach Zahnärzten oder medizinischen Kliniken in der Nähe, Verfolgung der Standorte von Mitbewerbern in einem bestimmten Gebiet, Extraktion von Geschäftskontakten aus Google Maps-Listen, Sammeln von Restaurantdaten für Marktforschung, Finden von Hotels oder Unterkunftsmöglichkeiten in einer Region, Lokalisieren bestimmter Dienstleistungen wie Coworking Spaces oder Fitnessstudios, Überwachen neuer Geschäftseröffnungen in einer Nachbarschaft, Sammeln von Kontaktinformationen und Adressen zur Verkaufsförderung, Analysieren von Preisspannen und Küchen lokaler Restaurants, Einholen von Bewertungen usw Überprüfung zählt für eine Liste von Unternehmen und exportiert lokale Geschäftsdaten in ein CRM oder eine Datenbank.
