Cette compétence aide les utilisateurs à extraire des listes de produits structurées d'Amazon, y compris les titres, les ASIN, les prix, les notes et les spécifications. Utilisez cette compétence lorsque les utilisateurs souhaitent rechercher des produits sur Amazon, trouver les produits de marque les plus vendus, suivre les changements de prix des articles, obtenir une liste de catégories avec des notes élevées, comparer différents produits de marque sur Amazon, extraire des données sur les produits Amazon pour des études de marché, rechercher des produits dans une langue ou un marché spécifique, analyser les prix des concurrents pour les mots-clés, trouver des produits en vedette pour les termes de recherche, obtenir des spécifications techniques telles que le matériau ou la couleur pour les listes de produits.
