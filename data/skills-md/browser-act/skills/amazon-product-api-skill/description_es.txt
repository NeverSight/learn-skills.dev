Esta habilidad ayuda a los usuarios a extraer listados de productos estructurados de Amazon, incluidos títulos, ASIN, precios, calificaciones y especificaciones. Utilice esta habilidad cuando los usuarios quieran buscar productos en Amazon, encontrar los productos de marca más vendidos, realizar un seguimiento de los cambios de precios de los artículos, obtener una lista de categorías con calificaciones altas, comparar productos de diferentes marcas en Amazon, extraer datos de productos de Amazon para estudios de mercado, buscar productos en un idioma o mercado específico, analizar los precios de la competencia para palabras clave, encontrar productos destacados para términos de búsqueda, obtener especificaciones técnicas como material o color para listas de productos.
