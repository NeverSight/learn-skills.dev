Этот навык помогает пользователям извлекать структурированные списки продуктов из Amazon, включая названия, ASIN, цены, рейтинги и характеристики. Используйте этот навык, когда пользователи хотят искать товары на Amazon, находить продукты самых продаваемых брендов, отслеживать изменения цен на товары, получать список категорий с высокими рейтингами, сравнивать продукты различных брендов на Amazon, извлекать данные о продуктах Amazon для исследования рынка, искать продукты на определенном языке или рынке, анализировать цены конкурентов по ключевым словам, находить избранные продукты по поисковым запросам, получать технические характеристики, такие как материал или цвет, для списков продуктов.
