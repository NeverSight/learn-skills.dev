Mit dieser Fähigkeit können Benutzer strukturierte Produktlisten von Amazon extrahieren, einschließlich Titel, ASINs, Preise, Bewertungen und Spezifikationen. Verwenden Sie diese Fähigkeit, wenn Benutzer auf Amazon nach Produkten suchen, die meistverkauften Markenprodukte finden, Preisänderungen für Artikel verfolgen, eine Liste von Kategorien mit hohen Bewertungen erhalten, verschiedene Markenprodukte auf Amazon vergleichen, Amazon-Produktdaten für Marktforschung extrahieren, nach Produkten in einer bestimmten Sprache oder auf einem bestimmten Marktplatz suchen, die Preise von Mitbewerbern nach Schlüsselwörtern analysieren, vorgestellte Produkte nach Suchbegriffen finden und technische Spezifikationen wie Material oder Farbe für Produktlisten abrufen möchten.
