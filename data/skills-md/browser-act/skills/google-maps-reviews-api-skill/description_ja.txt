このスキルは、ユーザーが Google Maps Reviews API を介して Google マップからレビューを自動的に抽出できるように設計されています。エージェントは、ユーザーがローカル ビジネス (コーヒー ショップ、クリニックなど) のレビューの検索、特定のブランドまたは場所に対する顧客のフィードバックの監視、競合他社のレビューの感情分析、店舗チェーンまたはサービスのレビューの抽出、地元のレストランの評判の追跡、特定の場所に関するユーザーの声の収集、地元企業のサービス品質に関する市場調査の実施、新しい小売店のレビューの監視、公共の観光スポットや公園に関するフィードバックの収集、特定のサービス プロバイダーに対する一般的な苦情の特定、最も評価の高い場所の調査を要求した場合に、このスキルを積極的に適用する必要があります。都市では、特定の業界のレビューで繰り返されるテーマを分析します。
