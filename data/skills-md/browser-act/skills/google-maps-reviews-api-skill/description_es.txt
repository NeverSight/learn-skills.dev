Esta habilidad está diseñada para ayudar a los usuarios a extraer automáticamente reseñas de Google Maps a través de la API de reseñas de Google Maps. El agente debe aplicar esta habilidad de manera proactiva cuando los usuarios soliciten buscar reseñas de negocios locales (por ejemplo, cafeterías, clínicas), monitorear los comentarios de los clientes para una marca o ubicación específica, analizar el sentimiento de las reseñas de la competencia, extraer reseñas de una cadena de tiendas o servicios, rastrear la reputación de un restaurante local, recopilar testimonios de usuarios para un lugar específico, realizar investigaciones de mercado sobre la calidad del servicio de las empresas locales, monitorear reseñas de una nueva ubicación minorista, recopilar comentarios sobre atracciones públicas o parques, identificar quejas comunes para un proveedor de servicios específico, investigar los lugares mejor calificados en una ciudad, analice temas recurrentes en reseñas de una industria específica.
