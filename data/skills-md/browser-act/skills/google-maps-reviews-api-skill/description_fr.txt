Cette compétence est conçue pour aider les utilisateurs à extraire automatiquement les avis de Google Maps via l'API Google Maps Reviews. L'agent doit appliquer cette compétence de manière proactive lorsque les utilisateurs demandent des avis sur des entreprises locales (par exemple, des cafés, des cliniques), surveillent les commentaires des clients pour une marque ou un emplacement spécifique, analysent le sentiment des avis sur les concurrents, extraient les avis sur une chaîne de magasins ou de services, suivent la réputation d'un restaurant local, recueillent des témoignages d'utilisateurs pour un lieu spécifique, mènent des études de marché sur la qualité du service des entreprises locales, surveillent les avis sur un nouveau point de vente au détail, recueillent des commentaires sur les attractions publiques ou les parcs, identifient les plaintes courantes pour un fournisseur de services spécifique, recherchent les endroits les mieux notés dans un ville, analyser les thèmes récurrents dans les revues pour un secteur spécifique.
