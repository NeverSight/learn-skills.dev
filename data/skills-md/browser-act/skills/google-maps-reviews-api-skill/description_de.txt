Dieser Skill soll Benutzern dabei helfen, automatisch Bewertungen aus Google Maps über die Google Maps Reviews API zu extrahieren. Agenten sollten diese Fähigkeit proaktiv anwenden, wenn Benutzer Bewertungen für lokale Unternehmen (z. B. Cafés, Kliniken) suchen, Kundenfeedback für eine bestimmte Marke oder einen bestimmten Standort überwachen, die Stimmung von Bewertungen für Wettbewerber analysieren, Bewertungen für eine Kette von Geschäften oder Dienstleistungen extrahieren, den Ruf eines lokalen Restaurants verfolgen, Benutzerstimmen für einen bestimmten Veranstaltungsort sammeln, Marktforschung zur Servicequalität lokaler Unternehmen durchführen, Bewertungen für einen neuen Einzelhandelsstandort überwachen, Feedback zu öffentlichen Attraktionen oder Parks sammeln, häufige Beschwerden für einen bestimmten Dienstleister identifizieren und die am besten bewerteten Orte recherchieren Analysieren Sie in einer Stadt wiederkehrende Themen in Bewertungen für eine bestimmte Branche.
