Этот навык предназначен для того, чтобы помочь пользователям автоматически извлекать отзывы из Карт Google через API отзывов Google Maps. Агенту следует активно применять этот навык, когда пользователи запрашивают отзывы о местных предприятиях (например, кафе, клиниках), отслеживают отзывы клиентов о конкретном бренде или месте, анализируют настроение отзывов о конкурентах, извлекают отзывы о сети магазинов или услуг, отслеживают репутацию местного ресторана, собирают отзывы пользователей о конкретном заведении, проводят исследование рынка качества обслуживания местных предприятий, отслеживают отзывы о новой торговой точке, собирают отзывы об общественных достопримечательностях или парках, выявляют распространенные жалобы на конкретного поставщика услуг, изучают места с лучшим рейтингом в городе, анализируйте повторяющиеся темы в обзорах по конкретной отрасли.
