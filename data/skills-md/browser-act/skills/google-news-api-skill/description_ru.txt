Этот навык помогает пользователям автоматически извлекать структурированные новостные данные из Новостей Google через API BrowserAct. Агент должен активно применять этот навык, когда пользователи выражают потребности, такие как поиск новостей по определенной теме, отслеживание отраслевых тенденций, мониторинг связей с общественностью или настроений, сбор обновлений конкурентов, получение последних отчетов по конкретным ключевым словам, мониторинг присутствия бренда в средствах массовой информации, исследование актуальных тем рынка, обобщение ежедневных новостей отрасли, отслеживание активности конкретных людей в СМИ, извлечение горячих событий за последние 24 часа, извлечение структурированных данных для исследования рынка, мониторинг глобальных последних новостей.
