Contexte personnel pour les agents IA via Mio. Charge les préférences utilisateur, la pile technologique, les conventions de codage et le contexte du projet au début de la session afin que vous puissiez prendre des décisions éclairées sans demander à nouveau. À utiliser lorsque l'un de ces outils Mio MCP est disponible : get_context, list_categories, list_projects, set_active_project, log_conversation, update_profile. Se déclenche sur : (1) le début de la session pour charger le contexte utilisateur avant de travailler, (2) les tâches de code pour lesquelles les conventions utilisateur sont importantes, (3) le travail à l'échelle du projet où le contexte du projet est nécessaire, (4) la fin des conversations de fond pour enregistrer les points à retenir, (5) lorsque l'utilisateur exprime une préférence nouvelle ou modifiée pour suggérer une mise à jour de profil.
