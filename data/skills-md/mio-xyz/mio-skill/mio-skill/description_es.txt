Contexto personal para agentes de IA a través de Mio. Carga las preferencias del usuario, la pila tecnológica, las convenciones de codificación y el contexto del proyecto al inicio de la sesión para que pueda tomar decisiones informadas sin volver a preguntar. Úselo cuando cualquiera de estas herramientas de Mio MCP esté disponible: get_context, list_categories, list_projects, set_active_project, log_conversation, update_profile. Se activa en: (1) el inicio de la sesión para cargar el contexto del usuario antes de trabajar, (2) tareas de código donde las convenciones del usuario importan, (3) trabajo dentro del alcance del proyecto donde se necesita el contexto del proyecto, (4) fin de conversaciones sustantivas para registrar conclusiones, (5) cuando el usuario expresa una preferencia nueva o modificada para sugerir una actualización del perfil.
