Персональный контекст для агентов ИИ через Mio. Загружает пользовательские настройки, технологический стек, соглашения по кодированию и контекст проекта в начале сеанса, чтобы вы могли принимать обоснованные решения без повторного запроса. Используйте, когда доступен любой из этих инструментов Mio MCP: get_context, list_categories, list_projects, set_active_project, log_conversation, update_profile. Срабатывает при: (1) начале сеанса для загрузки пользовательского контекста перед работой, (2) задачах кода, для которых важны пользовательские соглашения, (3) работе в рамках проекта, где необходим контекст проекта, (4) завершении предметного разговора для регистрации выводов, (5) когда пользователь выражает новое или измененное предпочтение, чтобы предложить обновление профиля.
