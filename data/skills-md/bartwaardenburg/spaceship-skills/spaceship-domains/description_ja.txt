Spaceship レジストラー API を介してドメインを管理します。ドメインの登録、DNS レコードの構成、連絡先と WHOIS プライバシーの管理、ドメインの転送、ネームサーバーのセットアップ、SellerHub でのドメインの販売を行います。 「ドメインの管理」、「ドメインの登録」、「DNS の構成」、「DNS レコードの追加」、「ネームサーバーのセットアップ」、「ドメインの可用性の確認」、「ドメインの移管」、「ドメインの販売」、「電子メール DNS のセットアップ」、「WHOIS プライバシーの管理」、または Spaceship ドメイン レジストラーの操作を求められた場合に使用します。 Spaceship MCP サーバーを構成する必要があります。
