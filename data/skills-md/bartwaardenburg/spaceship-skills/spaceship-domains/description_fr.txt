Gère les domaines via l'API du registraire Spaceship : enregistre les domaines, configure les enregistrements DNS, gère les contacts et la confidentialité WHOIS, transfère les domaines, configure les serveurs de noms et vend des domaines sur SellerHub. À utiliser lorsqu'on vous demande de « gérer les domaines », « enregistrer un domaine », « configurer le DNS », « ajouter des enregistrements DNS », « configurer des serveurs de noms », « vérifier la disponibilité du domaine », « transférer un domaine », « vendre un domaine », « configurer le DNS de courrier électronique », « gérer la confidentialité WHOIS » ou toute opération de registraire de domaine Spaceship. Nécessite la configuration du serveur Spaceship MCP.
