Da utilizzare quando si aggiungono, si rinominano o si esegue il debug dei file shader Stride SDSL (`*.sdsl`) in un progetto Stride. Applica il corretto flusso di lavoro di registrazione dello shader tra le cartelle delle risorse ".sdpkg", gli elementi dello shader collegati ".csproj", i file delle chiavi dello shader generati (`*.sdsl.cs") e le destinazioni forzate di ricostruzione delle risorse Stride prima del debug dello shader di runtime, in modo che gli agenti non diagnostichino erroneamente gli errori del database/modulo delle risorse obsoleti come errori del codice shader.
