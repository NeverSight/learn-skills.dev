À utiliser lors de l'ajout, du renommage ou du débogage de fichiers de shader Stride SDSL (`*.sdsl`) dans un projet Stride. Applique le flux de travail d'enregistrement de shader correct dans les dossiers d'actifs `.sdpkg`, les éléments de shader liés `.csproj`, les fichiers de clés de shader générés (`*.sdsl.cs`) et les cibles de reconstruction d'actifs Stride forcées avant le débogage du shader d'exécution, afin que les agents ne diagnostiquent pas à tort les erreurs de base de données/module d'actifs obsolètes comme des erreurs de code de shader.
