Configura il debug per i progetti del nodo vvvv gamma C#: genera configurazioni VS Code launch.json e task.json, collega il debugger all'esecuzione di vvvv, configura i profili di debug di Visual Studio e utilizza le migliori pratiche di debug. Da utilizzare durante la configurazione di un debugger per vvvv, la creazione di configurazioni di avvio, il collegamento al processo vvvv o la risoluzione dei problemi relativi ai punti di interruzione nei nodi C#. Supporta più configurazioni di avvio per diversi scenari/patch di test.
