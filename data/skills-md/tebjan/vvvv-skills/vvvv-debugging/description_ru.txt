Настройте отладку для проектов узлов vvvv gamma C#: создайте конфигурации VS Code launch.json и Tasks.json, подключите отладчик к запуску vvvv, настройте профили отладки Visual Studio и используйте рекомендации по отладке. Используйте при настройке отладчика для vvvv, создании конфигураций запуска, подключении к процессу vvvv или устранении неполадок точек останова на узлах C#. Поддерживает несколько конфигураций запуска для различных тестовых сценариев/исправлений.
