Copre l'avvio di vvvv gamma dalla riga di comando o in modo programmatico: avvio normale, apertura di patch .vl specifiche, argomenti della riga di comando, repository di pacchetti e percorsi chiave del filesystem (directory di installazione, dati utente, schizzi, esportazioni, pacchetti). Da utilizzare quando si avvia vvvv, si configurano gli argomenti di avvio, si impostano i repository di pacchetti o si cercano le directory dei dati di vvvv.
