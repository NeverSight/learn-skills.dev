Comprehensive Feishu/Lark workspace agent. Handles messages, docs, bitable (multi-dimensional tables), calendar, and tasks via Feishu Open API.

Trigger when user says anything involving:
- Feishu messages: send message, send Feishu, check message, reply message, search message, view group chat
- Feishu docs: create doc, view doc, read doc, Feishu doc
- Bitable/multi-dimensional tables: query record, add record, update record, delete record
- Calendar: Feishu calendar, check schedule, create event, add calendar
- Tasks: Feishu task, create task, view task, complete task
- User: look up Feishu user, find contact
