フェイシュ/ラークの総合ワークスペースエージェント。 Feishu Open API を介して、メッセージ、ドキュメント、ビットテーブル (多次元テーブル)、カレンダー、タスクを処理します。

ユーザーが次のようなことを言ったときにトリガーされます。
- フェイシュメッセージ：メッセージの送信、フェイシュの送信、メッセージの確認、メッセージの返信、メッセージの検索、グループチャットの表示
- フェイシュドキュメント: ドキュメントの作成、ドキュメントの表示、ドキュメントの読み取り、フェイシュドキュメント
- Bitable/多次元テーブル: レコードのクエリ、レコードの追加、レコードの更新、レコードの削除
- カレンダー：フェイシュカレンダー、スケジュールの確認、イベントの作成、カレンダーの追加
- タスク: フェイシュタスク、タスク作成、タスク表示、タスク完了
- ユーザー: Feishu ユーザーを検索し、連絡先を見つけます
