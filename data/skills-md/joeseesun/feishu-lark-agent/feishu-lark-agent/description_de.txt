Umfassender Feishu/Lark-Workspace-Agent. Verarbeitet Nachrichten, Dokumente, Bitables (mehrdimensionale Tabellen), Kalender und Aufgaben über die Feishu Open API.

Wird ausgelöst, wenn der Benutzer Folgendes sagt:
- Feishu-Nachrichten: Nachricht senden, Feishu senden, Nachricht prüfen, Nachricht beantworten, Nachricht suchen, Gruppenchat anzeigen
- Feishu-Dokumente: Dokument erstellen, Dokument anzeigen, Dokument lesen, Feishu-Dokument
- Bitbare/mehrdimensionale Tabellen: Datensatz abfragen, Datensatz hinzufügen, Datensatz aktualisieren, Datensatz löschen
- Kalender: Feishu-Kalender, Zeitplan prüfen, Ereignis erstellen, Kalender hinzufügen
- Aufgaben: Feishu-Aufgabe, Aufgabe erstellen, Aufgabe anzeigen, Aufgabe abschließen
- Benutzer: Feishu-Benutzer suchen, Kontakt finden
