Agent d'espace de travail complet Feishu/Lark. Gère les messages, les documents, les fichiers bitables (tableaux multidimensionnels), le calendrier et les tâches via l'API ouverte Feishu.

Déclenchez lorsque l'utilisateur dit quelque chose impliquant :
- Messages Feishu : envoyer un message, envoyer Feishu, vérifier le message, répondre au message, rechercher un message, afficher le chat de groupe
- Documents Feishu : créer un document, afficher un document, lire un document, un document Feishu
- Tables bitables/multidimensionnelles : interroger un enregistrement, ajouter un enregistrement, mettre à jour un enregistrement, supprimer un enregistrement
- Calendrier : calendrier Feishu, vérifier le calendrier, créer un événement, ajouter un calendrier
- Tâches : tâche Feishu, créer une tâche, afficher la tâche, terminer la tâche
- Utilisateur : recherchez l'utilisateur Feishu, trouvez le contact
