포괄적인 Feishu/Lark 작업공간 에이전트. Feishu Open API를 통해 메시지, 문서, Bitable(다차원 테이블), 달력 및 작업을 처리합니다.

사용자가 다음과 관련된 내용을 말하면 트리거됩니다.
- 페이슈 메시지: 메시지 보내기, 페이슈 보내기, 메시지 확인, 메시지 답장, 메시지 검색, 그룹 채팅 보기
- Feishu 문서: 문서 만들기, 문서 보기, 문서 읽기, Feishu 문서
- 비트 가능/다차원 테이블: 레코드 조회, 레코드 추가, 레코드 업데이트, 레코드 삭제
- 캘린더 : 페이슈 캘린더, 일정 확인, 이벤트 생성, 캘린더 추가
- 작업: Feishu 작업, 작업 생성, 작업 보기, 작업 완료
- 사용자 : Feishu 사용자 조회, 연락처 찾기
