Комплексный агент рабочего пространства Feishu/Lark. Обрабатывает сообщения, документы, побитовые (многомерные таблицы), календарь и задачи через Feishu Open API.

Триггер, когда пользователь говорит что-либо, связанное с:
- Сообщения Фейшу: отправлять сообщения, отправлять Фейшу, проверять сообщения, отвечать на сообщения, искать сообщения, просматривать групповые чаты.
- Документы Feishu: создавайте документы, просматривайте документы, читайте документы, документы Feishu.
- Битовые/многомерные таблицы: запросы к записям, добавление записей, обновление записей, удаление записей.
- Календарь: календарь Фейшу, расписание проверки, создание расписания, добавление календаря.
- Задачи: задачи Фейшу, создание задач, просмотр задач, выполнение задач.
- Пользователи: запрашивайте пользователей Feishu, находите контакты.
