YouTubeビデオ検索、ビデオダウンローダー、字幕ダウンローダー。高度な検索のための YouTube Data API v3 と、ビデオ/オーディオ/字幕をダウンロードするための yt-dlp を統合します。
主な機能: サイト全体のキーワード検索、チャンネルの閲覧、時間/視聴数/関連性による並べ替え、ビデオのダウンロード、音声の抽出 (MP3)、字幕のダウンロード (中国語/英語)、ビデオの詳細の表示。
トリガー シナリオ (YouTube 関連の操作にはこのスキルを使用します):
- 検索: 「YouTube を検索」、「YouTube 検索」、「YouTube 動画を検索」、「[チャンネル] の最新動画を検索」、「[名前] の YouTube を検索」、「[名前] の最近の YouTube アップロード」、「XXX についての YouTube」
- チャンネルの参照: 「チャンネルを参照」、「[チャンネル] の最新のビデオを表示」、「[名前] が最近アップロードしたもの」、「[名前] の YouTube アップデート」、「[チャンネル] のビデオ リストを表示」
- ビデオのダウンロード: 「YouTube をダウンロード」、「このビデオをダウンロード」、「YouTube ビデオをダウンロード」、「このビデオを保存」
- オーディオのダウンロード: 「オーディオの抽出」、「オーディオのダウンロード」、「YouTube to MP3」、「オーディオのみ」
- 字幕のダウンロード: 「字幕をダウンロード」、「字幕を抽出」、「字幕を取得」、「YouTube 字幕をダウンロード」、「中国語字幕」、「英語字幕」、「テキストを抽出」、「ビデオからテキストへ」、「ビデオテキストを取得」
- ビデオ情報: 「ビデオの詳細」、「ビデオ情報」、「このビデオの長さ」
キーワード一致: ユーザーメッセージに「YouTube」、「油管」、「YT」、「yt」に加えて、検索/ダウンロード/字幕/チャンネル/ビデオなどのアクションワードが含まれる場合にトリガーされます。
