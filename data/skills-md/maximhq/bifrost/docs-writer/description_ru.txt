Написание, обновление и просмотр документации Mintlify MDX для функций Bifrost. Исследует полную базу кода (пользовательский интерфейс, серверную часть Go, схему конфигурации), проверяет примеры config.json, размещает заполнители снимков экрана и представляет схемы для утверждения перед написанием. Вызывается с помощью /docs-writer <имя-функции>, /docs-writer update <doc-path> или /docs-writer review <doc-path>.
