Automatisation du navigateur à l'aide de Playwright MCP. Naviguez sur des sites Web, remplissez des formulaires, cliquez sur des éléments, prenez des captures d'écran et extrayez des données. À utiliser pour la navigation sur le Web, la soumission de formulaires, le web scraping ou les tests d'interface utilisateur. PAS pour le contenu statique (utilisez curl/wget).
