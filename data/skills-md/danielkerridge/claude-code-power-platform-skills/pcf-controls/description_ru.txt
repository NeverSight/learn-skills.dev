Используйте при создании, развертывании или использовании элементов управления PowerApps Component Framework (PCF). Охватывает элементы управления полями и наборами данных, виртуальные элементы управления React, жизненный цикл элемента управления, конфигурацию манифеста, упаковку решения и развертывание. Запускается по: «pcf», «пользовательский элемент управления», «структура компонентов», «управление набором данных», «виртуальный контроль», «pac pcf», «pcf init», «pcf push», «ControlManifest», «StandardControl», «ReactControl», «updateView», «getOutputs».
