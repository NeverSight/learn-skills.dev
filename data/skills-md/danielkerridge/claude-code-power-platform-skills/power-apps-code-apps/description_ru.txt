Используйте при создании, создании шаблонов, отладке или развертывании приложений кода Microsoft Power Apps с использованием React, Vue или TypeScript в VS Code. Обеспечивает инициализацию проекта через pac CLI, Dataverse и доступ к данным соединителя через @microsoft/power-apps SDK, конфигурацию power.config.json, конвейеры сборки Vite и развертывание в средах Power Platform. Запускается по: «Power Apps», «Code App», «Pac Code», «Dataverse», «Power Platform App», «Vibe Coding», «Scaffold Power App», «Развертывание кода приложения».
