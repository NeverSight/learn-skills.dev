VS Code で React、Vue、または TypeScript を使用して Microsoft Power Apps コード アプリを構築、スキャフォールディング、デバッグ、またはデプロイするときに使用します。 pac CLI を介したプロジェクトの初期化、@microsoft/power-apps SDK を介した Dataverse とコネクタのデータ アクセス、power.config.json 構成、Vite ビルド パイプライン、および Power Platform 環境へのデプロイメントを処理します。トリガー: 「パワー アプリ」、「コード アプリ」、「パック コード」、「データバース」、「パワー プラットフォーム アプリ」、「バイブ コーディング」、「スキャフォールド パワー アプリ」、「コード アプリのデプロイ」。
