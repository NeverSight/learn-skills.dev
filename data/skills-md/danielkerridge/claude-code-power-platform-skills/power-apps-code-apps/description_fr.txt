À utiliser lors de la création, de l'échafaudage, du débogage ou du déploiement d'applications de code Microsoft Power Apps à l'aide de React, Vue ou TypeScript dans VS Code. Gère l'initialisation du projet via pac CLI, Dataverse et l'accès aux données du connecteur via le SDK @microsoft/power-apps, la configuration power.config.json, les pipelines de build Vite et le déploiement dans les environnements Power Platform.
