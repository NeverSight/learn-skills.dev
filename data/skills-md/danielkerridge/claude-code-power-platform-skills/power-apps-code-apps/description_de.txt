Zum Erstellen, Gerüstbauen, Debuggen oder Bereitstellen von Microsoft Power Apps Code Apps mithilfe von React, Vue oder TypeScript in VS Code. Verwaltet die Projektinitialisierung über pac CLI, Dataverse und Connector-Datenzugriff über das @microsoft/power-apps SDK, die power.config.json-Konfiguration, Vite-Build-Pipelines und die Bereitstellung in Power Platform-Umgebungen. Löst aus: „Power Apps“, „Code App“, „Pac Code“, „Dataverse“, „Power Platform App“, „Vibe Coding“, „Scaffold Power App“, „Deploy Code App“.
