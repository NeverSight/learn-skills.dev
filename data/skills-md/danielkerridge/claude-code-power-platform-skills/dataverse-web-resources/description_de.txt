Zur Verwendung beim Erstellen, Bereitstellen oder Verwalten von Dataverse-Webressourcen für modellgesteuerte Apps. Behandelt JavaScript-Formularskripte (OnLoad-, OnSave-, OnChange-Ereignisse), HTML-Dashboard-Seiten, CSS-Stil, Bildressourcen, Navigation/Seitenbereiche, Anpassung der Multifunktionsleiste/Befehlsleiste, Geschäftsprozessfluss-Client-API und Bereitstellung über die Web-API.
