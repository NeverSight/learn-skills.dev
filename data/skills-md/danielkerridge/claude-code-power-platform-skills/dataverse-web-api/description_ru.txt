Используйте при программном создании, изменении или запросе схемы и метаданных Dataverse через веб-API (OData v4.0). Охватывает определения таблиц/столбцов/связей, решение ALM, построение форм и представлений XML, состав модулей приложений, глобальные наборы параметров, бизнес-правила, регистрацию пользовательских API и публикацию.
