Modifica e gestisci le casseforti di Obsidian: crea, leggi, aggiorna ed elimina note, gestisci proprietà/frontmatter, gestisci collegamenti e backlink, lavora con tag, attività, note giornaliere, modelli, basi e altro ancora. Utilizza la CLI Obsidian per operazioni di sicurezza quando disponibile, con la modifica diretta dei file come fallback. Usa questa abilità ogni volta che l'utente menziona Obsidian, vault, note della knowledge base con wikilink, frontmatter/proprietà su file markdown, note giornaliere o qualsiasi attività che coinvolga un vault Obsidian, anche se dice semplicemente "le mie note" o fa riferimento a una cartella che assomiglia a un vault (ha la directory .obsidian/).
