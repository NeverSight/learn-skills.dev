ابحث عن أيقونات Microsoft Fabric / Azure SVG الرسمية واحضرها لاستخدامها في المخططات والوثائق وواجهة المستخدم. استخدمها عند إنشاء مخططات معمارية (HTML أو PPTX أو أي تنسيق مرئي) تحتاج إلى أيقونات لمفاهيم النسيج مثل Lakehouse وData Warehouse وPipline وEventhouse وPower BI وOneLake وDataflow وNotebook والنموذج الدلالي وقاعدة بيانات KQL وعناصر النسيج الأخرى. يُستخدم أيضًا عندما يطلب المستخدم أيقونات النظام الأساسي للبيانات Microsoft Fabric أو Azure.
