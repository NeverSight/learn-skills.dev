Suchen und rufen Sie offizielle Microsoft Fabric/Azure SVG-Symbole zur Verwendung in Diagrammen, Dokumentation und Benutzeroberfläche ab. Verwenden Sie diese Funktion beim Erstellen von Architekturdiagrammen (HTML, PPTX oder ein beliebiges visuelles Format), die Symbole für Fabric-Konzepte wie Lakehouse, Data Warehouse, Pipeline, Eventhouse, Power BI, OneLake, Dataflow, Notebook, Semantic Model, KQL Database und andere Fabric-Elemente benötigen. Verwenden Sie es auch, wenn der Benutzer nach Symbolen für Microsoft Fabric oder Azure-Datenplattform fragt.
