Mobile-first UX هو منفذ أفضل ممارسات UX لتطبيقات الويب. يغطي أهداف اللمس والتصميم سريع الاستجابة وإيماءات الهاتف المحمول وتحسين الأداء وإمكانية الوصول. يُستخدم عند إنشاء مكونات واجهة مستخدم الهاتف المحمول أو تحريرها، أو تنفيذ تخطيطات سريعة الاستجابة، أو التعامل مع تفاعلات اللمس، أو تحسين الأجهزة المحمولة، أو العمل مع تكوينات إطار العرض، ونقاط التوقف، والأنماط الخاصة بالهواتف المحمولة.
