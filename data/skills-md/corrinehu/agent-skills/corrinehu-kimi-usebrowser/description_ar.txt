استكشاف أخطاء أتمتة المتصفح وإصلاحها وأفضل الممارسات لـ Kimi CLI باستخدام متصفح الوكيل. يُستخدم عند مواجهة مشكلات تتعلق بأتمتة المتصفح، أو انتهاء المهلة، أو تنزيلات متصفح Playwright، أو عند الحاجة إلى استخدام نظام Chrome بدلاً من تنزيل Chromium. يغطي المخاطر الشائعة مثل مهلة التثبيت لأول مرة، ومشكلات الوكيل، وتكوين المسار القابل للتنفيذ.
