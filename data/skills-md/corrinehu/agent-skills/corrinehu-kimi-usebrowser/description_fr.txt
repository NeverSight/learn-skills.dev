Dépannage de l'automatisation du navigateur et bonnes pratiques pour Kimi CLI à l'aide du navigateur d'agent. À utiliser lorsque vous rencontrez des problèmes avec l'automatisation du navigateur, les délais d'attente, les téléchargements du navigateur Playwright ou lorsque vous devez utiliser le système Chrome au lieu de télécharger Chromium. Couvre les pièges courants tels que les délais d'attente pour la première installation, les problèmes de proxy et la configuration du chemin d'accès exécutable.
