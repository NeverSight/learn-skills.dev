Solución de problemas de automatización del navegador y mejores prácticas para Kimi CLI utilizando agente-navegador. Úselo cuando tenga problemas con la automatización del navegador, los tiempos de espera, las descargas del navegador Playwright o cuando necesite usar el sistema Chrome en lugar de descargar Chromium. Cubre errores comunes como tiempos de espera de instalación por primera vez, problemas de proxy y configuración de rutas ejecutables.
