Diagnostique fallas de conexión y emparejamiento del nodo OpenClaw para aplicaciones complementarias de Android, iOS y macOS. Úselo cuando falla el código QR/de configuración/la conexión manual, el Wi-Fi local funciona pero el VPS/tailnet no, o los errores mencionan el emparejamiento requerido, no autorizado, el token de arranque no válido o vencido, gateway.bind, gateway.remote.url, Tailscale o plugins.entries.device-pair.config.publicUrl.
