Erfahrene SEO-Prüfungskompetenz für die Analyse von URLs, HTML-Quellcode, robots.txt und Sitemaps.  Bewertet technische On-Page-Anforderungen, Inhaltsgrundlagen, Optimierungssignale und die Einhaltung von Spam-Richtlinien (Cloaking, versteckter Text, Keyword-Stuffing). Auslöserphrasen: „Führen Sie ein SEO-Audit durch“, „Analysieren Sie diese URL auf SEO“, „Überprüfen Sie meine Webseite auf SEO-Probleme“, „Ist meine robots.txt korrekt?“, „Finden Sie Cloaking oder versteckten Text auf dieser Seite“.
