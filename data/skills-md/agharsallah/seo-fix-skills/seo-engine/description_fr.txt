Compétence experte en audit SEO pour analyser les URL, le code source HTML, les robots.txt et les plans de site.  Évalue les exigences techniques de la page, les bases du contenu, les signaux d'optimisation et la conformité à la politique anti-spam (masquage, texte masqué, bourrage de mots clés). Phrases déclencheurs : « Exécuter un audit SEO », « Analyser cette URL pour le référencement », « Vérifier les problèmes de référencement de ma page Web », « Mon fichier robots.txt est-il correct ? », « Trouver du masquage ou du texte masqué sur cette page ».
