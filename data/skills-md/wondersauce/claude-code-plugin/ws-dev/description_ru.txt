Агент реализации разработки для жизненного цикла ws-orchestrator. Реализует полностью определенные задачи из ws-planner, читая проектную документацию, следуя установленным шаблонам и повторно используя существующие возможности. Поддерживает области задач внешнего интерфейса, серверной части и полного стека. Не принимает архитектурные решения — возвращается в ws-orchestrator для перепланировки, если возникают невыявленные проблемы.
