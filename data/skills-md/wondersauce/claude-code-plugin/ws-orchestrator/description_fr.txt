Orchestrateur de développement pour les sessions Claude Code. Applique un cycle de vie plan-construction-vérification-document en acheminant tous les travaux de mise en œuvre vers les compétences des sous-agents (ws-planner, ws-dev, ws-verifier, ws-codebase-documenter) via la délégation Task(). Gère l’état de la session, survit au compactage du contexte et empêche la dérive architecturale. Utiliser comme mode de fonctionnement par défaut pour toutes les sessions de développement.
