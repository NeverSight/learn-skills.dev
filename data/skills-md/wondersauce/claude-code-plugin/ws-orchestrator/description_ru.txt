Оркестратор разработки для сессий Claude Code. Обеспечивает жизненный цикл «план-сборка-проверка-документ», перенаправляя всю работу по реализации навыкам субагента (ws-planner, ws-dev, ws-verifier, ws-codebase-documenter) через делегирование Task(). Управляет состоянием сеанса, выдерживает сжатие контекста и предотвращает сдвиг архитектуры. Используйте в качестве режима работы по умолчанию для всех сеансов разработки.
