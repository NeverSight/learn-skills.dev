Планировщик разработки для жизненного цикла ws-orchestrator. Учитывая описание задачи и проектную документацию, разрабатывает полностью определенные, структурированные планы развития. Определяет, что создавать, как структурировать это в соответствии с существующими шаблонами, как разложить на подзадачи и какие ограничения применять. Возвращает массивы определений задач, которые ws-dev может выполнять без принятия архитектурных решений.
