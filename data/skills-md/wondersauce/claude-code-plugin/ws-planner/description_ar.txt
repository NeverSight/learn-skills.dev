مخطط التطوير لدورة حياة ws-orchestrator. نظرًا لوصف المهمة ووثائق المشروع، يتم إنتاج خطط تطوير محددة ومنظمة بالكامل. يحدد ما سيتم بناؤه، وكيفية هيكلته وفقًا للأنماط الموجودة، وكيفية تقسيمه إلى مهام فرعية، وما هي القيود المطبقة. إرجاع صفائف تعريف المهمة التي يمكن لـ ws-dev تنفيذها دون اتخاذ قرارات معمارية.
