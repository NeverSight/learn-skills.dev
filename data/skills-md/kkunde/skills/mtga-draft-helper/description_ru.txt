Помощник по драфту MTG Arena, использующий данные 17Lands и руководства по драфту для конкретного набора. Используйте, когда пользователь просит помочь с драфтом MTG Arena, проанализировать выбор драфта, оценить карты для ограниченного набора, получить рейтинги карт 17Lands, прочитать журнал Arena Player.log о текущем состоянии драфта, предложить, какую карту выбрать, показать изображения карт, построить или предложить колоду после составления или что-либо, связанное с составлением драфта Magic: The Gathering Arena и ограниченными форматами. Включает руководства по архетипам для конкретных выпусков и списки уровней выбора (например, «Затмение Лорвина» / ECL, «Через тропы знамений» / «Человек-паук» / OM1). Срабатывает при: драфте, 17lands, MTGA, выборе, пакете, ограниченном, запечатанном, предложении колоды, помощнике по драфту, рейтинге карт, GIHWR, проценте побед, ECL, Lorwyn Eclipsed, OM1, Through the Omenpaths, Человек-паук.
