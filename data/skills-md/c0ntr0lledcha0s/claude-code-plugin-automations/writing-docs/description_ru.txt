Эксперт по написанию высококачественной документации для кода, API и проектов. Автоматически вызывается при создании строк документации, создании файлов README, написании документации API, добавлении комментариев к коду или создании любой технической документации. Предоставляет шаблоны для конкретного языка и лучшие практики для эффективного написания документации.
