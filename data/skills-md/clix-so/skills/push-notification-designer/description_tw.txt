設計和實施本地推播通知活動，以提高用戶對行動應用程式的參與度。每當用戶提到本地推播通知、應用程式內通知、用戶參與活動、保留訊息、重新參與推動或想要向其應用程式新增計畫通知時，請使用此技能。當使用者詢問通知計時策略、入職通知流程或鍛鍊/習慣/目標提醒系統時也會觸發。涵蓋 iOS (Swift/UNUserNotificationCenter)、Android (Kotlin/WorkManager+NotificationManager)、Flutter (flutter_local_notifications) 和 React Native (notifee/expo-notifications)。
