تصميم وتنفيذ حملات إعلام الدفع المحلية لتعزيز مشاركة المستخدم في تطبيقات الهاتف المحمول. استخدم هذه المهارة عندما يذكر المستخدم إشعارات الدفع المحلية، أو الإشعارات داخل التطبيق، أو حملات مشاركة المستخدم، أو رسائل الاحتفاظ، أو تنبيهات إعادة المشاركة، أو يريد إضافة إشعارات مجدولة إلى تطبيقه. يتم التشغيل أيضًا عندما يسأل المستخدم عن إستراتيجيات توقيت الإشعارات، أو تدفقات الإشعارات، أو أنظمة تذكير التمرين/العادات/الأهداف. يغطي iOS (Swift/UNUserNotificationCenter)، وAndroid (Kotlin/WorkManager+NotificationManager)، وFlutter (flutter_local_notifications)، وReact Native (إشعارات/إشعارات المعرض).
