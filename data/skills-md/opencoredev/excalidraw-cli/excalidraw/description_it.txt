Controlla un canvas Excalidraw live tramite la CLI "excalidraw". Utilizzare quando un agente deve (1) disegnare o disporre diagrammi su un'area di disegno live, (2) creare/aggiornare/eliminare singoli elementi, (3) creare batch di diagrammi complessi da JSON, (4) ispezionare l'area di disegno con descrizione o screenshot, (5) esportare/importare file .excalidraw, PNG o SVG, (6) salvare/ripristinare istantanee dell'area di disegno, (7) allineare, distribuire, raggruppare, bloccare o duplicare elementi, (8) convertire i diagrammi della sirena in Excalidraw elementi, (9) condividono diagrammi come URL excalidraw.com crittografati. Richiede un server Canvas in esecuzione (http://localhost:3000 predefinito).
