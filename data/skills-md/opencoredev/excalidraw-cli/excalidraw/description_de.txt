Steuern Sie eine Live-Excalidraw-Leinwand über die „excalidraw“-CLI. Wird verwendet, wenn ein Agent (1) Diagramme auf einer Live-Leinwand zeichnen oder anordnen, (2) einzelne Elemente erstellen/aktualisieren/löschen, (3) komplexe Diagramme aus JSON stapelweise erstellen, (4) die Leinwand mit Beschreibung oder Screenshot überprüfen, (5) .excalidraw-Dateien, PNG oder SVG exportieren/importieren, (6) Leinwand-Schnappschüsse speichern/wiederherstellen, (7) Elemente ausrichten, verteilen, gruppieren, sperren oder duplizieren muss, (8) Mermaid-Diagramme in konvertieren muss Excalidraw-Elemente (9) teilen Diagramme als verschlüsselte excalidraw.com-URLs. Erfordert einen laufenden Canvas-Server (Standard http://localhost:3000).
