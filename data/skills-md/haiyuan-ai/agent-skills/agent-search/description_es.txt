Búsqueda web de información actual o externa.

Activador: verbos de búsqueda ("buscar", "buscar", "查", "搜", "google"), consultas en tiempo real ("últimas", "noticias", "更新", "进展"), investigación/comparación ("调研", "对比", "vs"), nombres propios (personas, empresas, productos).

Saltar: operaciones locales, uso explícito de otra herramienta de búsqueda y conocimientos generales.

Salida: JSON a través de `--json`.
