Ricerca sul Web per informazioni attuali o esterne.

Trigger: verbi di ricerca ("cerca", "cerca", "查", "搜", "google"), query in tempo reale ("ultime", "notizie", "更新", "进展"), ricerca/confronto ("调研", "对比", "vs"), nomi propri (persone, aziende, prodotti).

Salta: operazioni locali, uso esplicito di un altro strumento di ricerca e conoscenze generali.

Output: JSON tramite `--json`.
