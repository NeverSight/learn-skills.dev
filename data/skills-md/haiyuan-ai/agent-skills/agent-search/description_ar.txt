البحث في الويب عن المعلومات الحالية أو الخارجية.

المشغل: أفعال البحث ("بحث"، "بحث"، "查"، "搜"، "google")، الاستعلامات في الوقت الفعلي ("الأحدث"، "أخبار"، "更新"، "进展")، البحث/المقارنة ("调研"، "对比"، "vs")، أسماء الأعلام (الأشخاص والشركات والمنتجات).

التخطي: العمليات المحلية، والاستخدام الصريح لأداة بحث أخرى، والمعرفة العامة.

الإخراج: JSON عبر `--json`.
