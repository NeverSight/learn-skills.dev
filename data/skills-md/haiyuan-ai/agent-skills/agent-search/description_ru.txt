Веб-поиск текущей или внешней информации.

Триггер: поисковые глаголы («поиск», «поиск», «查», «搜», «google»), запросы в реальном времени («последние», «новости», «更新», «进展»), исследование/сравнение («调研», «对比», «vs»), имена собственные (люди, компании, продукты).

Пропустить: локальные операции, явное использование другого инструмента поиска и общие знания.

Вывод: JSON через `--json`.
