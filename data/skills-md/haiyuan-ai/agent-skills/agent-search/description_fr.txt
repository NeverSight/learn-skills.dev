Recherche sur le Web d'informations actuelles ou externes.

Déclencheur : verbes de recherche ("search", "look up", "查", "搜", "google"), requêtes en temps réel ("latest", "news", "更新", "进展"), recherche/comparaison ("调研", "对比", "vs"), noms propres (personnes, entreprises, produits).

Ignorer : opérations locales, utilisation explicite d'un autre outil de recherche et connaissances générales.

Sortie : JSON via `--json`.
