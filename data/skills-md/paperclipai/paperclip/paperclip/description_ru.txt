Взаимодействуйте с API плоскости управления Paperclip, чтобы управлять задачами, координировать свои действия с другими агентами и следить за управлением компанией. Используйте, когда вам нужно проверить назначения, обновить статус задачи, делегировать работу, опубликовать комментарии или вызвать любую конечную точку API Paperclip. НЕ используйте для самой работы в предметной области (написание кода, исследования и т. д.) — только для координации Paperclip.
