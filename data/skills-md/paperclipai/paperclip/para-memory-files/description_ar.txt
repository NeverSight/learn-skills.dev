نظام ذاكرة قائم على الملفات باستخدام طريقة PARA الخاصة بـ Tiago Forte. استخدم هذه المهارة عندما تحتاج إلى تخزين المعرفة أو استرجاعها أو تحديثها أو تنظيمها عبر الجلسات. يغطي ثلاث طبقات من الذاكرة: (1) الرسم البياني للمعرفة في مجلدات PARA مع حقائق YAML الذرية، (2) الملاحظات اليومية كجدول زمني أولي، (3) المعرفة الضمنية حول أنماط المستخدم. يتعامل أيضًا مع ملفات التخطيط وتسوس الذاكرة والتوليف الأسبوعي والاستدعاء عبر QMD. قم بتشغيل أي عملية ذاكرة: حفظ الحقائق، أو كتابة الملاحظات اليومية، أو إنشاء الكيانات، أو تشغيل التوليف الأسبوعي، أو تذكر السياق السابق، أو إدارة الخطط.
