Sistema de memoria basado en archivos utilizando el método PARA de Tiago Forte. Utilice esta habilidad siempre que necesite almacenar, recuperar, actualizar u organizar conocimientos entre sesiones. Cubre tres capas de memoria: (1) Gráfico de conocimiento en carpetas PARA con datos atómicos de YAML, (2) Notas diarias como línea de tiempo sin procesar, (3) Conocimiento tácito sobre patrones de usuario. También maneja archivos de planificación, deterioro de la memoria, síntesis semanal y recuperación mediante qmd. Activa cualquier operación de memoria: guardar datos, escribir notas diarias, crear entidades, ejecutar síntesis semanales, recordar contextos pasados ​​o gestionar planes.
