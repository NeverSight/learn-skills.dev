Dateibasiertes Speichersystem mit der PARA-Methode von Tiago Forte. Nutzen Sie diese Fähigkeit immer dann, wenn Sie Wissen sitzungsübergreifend speichern, abrufen, aktualisieren oder organisieren müssen. Deckt drei Speicherebenen ab: (1) Wissensdiagramm in PARA-Ordnern mit atomaren YAML-Fakten, (2) Tägliche Notizen als Rohzeitleiste, (3) Implizites Wissen über Benutzermuster. Behandelt außerdem Planungsdateien, Speicherabbau, wöchentliche Synthese und den Abruf über qmd. Lösen Sie jeden Speichervorgang aus: Speichern von Fakten, Schreiben täglicher Notizen, Erstellen von Entitäten, Ausführen wöchentlicher Synthesen, Abrufen vergangener Kontexte oder Verwalten von Plänen.
