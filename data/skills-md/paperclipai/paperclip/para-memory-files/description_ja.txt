Tiago Forte の PARA メソッドを使用したファイルベースのメモリ システム。セッション全体で知識を保存、取得、更新、または整理する必要がある場合は、このスキルをいつでも使用できます。 3 つのメモリ層をカバーします: (1) アトミック YAML ファクトを含む PARA フォルダー内のナレッジ グラフ、(2) 生のタイムラインとしての毎日のメモ、(3) ユーザー パターンに関する暗黙知。また、計画ファイル、メモリ減衰、毎週の合成、および qmd によるリコールも処理します。ファクトの保存、毎日のメモの作成、エンティティの作成、毎週の合成の実行、過去のコンテキストの呼び出し、または計画の管理など、あらゆるメモリ操作をトリガーします。
