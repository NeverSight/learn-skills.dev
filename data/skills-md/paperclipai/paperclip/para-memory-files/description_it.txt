Sistema di memoria basato su file che utilizza il metodo PARA di Tiago Forte. Utilizza questa abilità ogni volta che devi archiviare, recuperare, aggiornare o organizzare la conoscenza tra le sessioni. Copre tre livelli di memoria: (1) Grafico della conoscenza nelle cartelle PARA con fatti YAML atomici, (2) Note giornaliere come sequenza temporale grezza, (3) Conoscenza tacita dei modelli degli utenti. Gestisce anche file di pianificazione, decadimento della memoria, sintesi settimanale e richiamo tramite qmd. Attiva qualsiasi operazione di memoria: salvataggio di fatti, scrittura di appunti giornalieri, creazione di entità, esecuzione di sintesi settimanali, richiamo di contesti passati o gestione di piani.
