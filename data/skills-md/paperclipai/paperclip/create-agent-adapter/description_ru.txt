Техническое руководство по созданию нового адаптера агента Paperclip. Используйте при создании нового пакета адаптера, добавлении поддержки нового инструмента кодирования AI (например, нового агента CLI, агента на основе API или пользовательского процесса) или при изменении системы адаптера. Охватывает необходимые интерфейсы, структуру модуля, точки регистрации и соглашения, полученные из существующих адаптеров claude-local и codex-local.
