Guía técnica para la creación de un nuevo adaptador de agente Paperclip. Úselo cuando cree un nuevo paquete de adaptador, agregue soporte para una nueva herramienta de codificación de IA (por ejemplo, un nuevo agente CLI, un agente basado en API o un proceso personalizado) o cuando modifique el sistema de adaptador. Cubre las interfaces requeridas, la estructura del módulo, los puntos de registro y las convenciones derivadas de los adaptadores claude-local y codex-local existentes.
