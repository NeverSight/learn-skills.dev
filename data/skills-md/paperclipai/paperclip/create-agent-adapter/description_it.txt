Guida tecnica per la creazione di un nuovo adattatore dell'agente Paperclip. Da utilizzare quando si crea un nuovo pacchetto adattatore, si aggiunge il supporto per un nuovo strumento di codifica AI (ad esempio un nuovo agente CLI, un agente basato su API o un processo personalizzato) o quando si modifica il sistema adattatore. Copre le interfacce richieste, la struttura dei moduli, i punti di registrazione e le convenzioni derivate dagli adattatori claude-local e codex-local esistenti.
