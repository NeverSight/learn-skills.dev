Cree y administre catálogos de marketing, páginas de destino y embudos de varios pasos con su agente de inteligencia artificial. Cree catálogos a partir de esquemas JSON, publíquelos al instante, ejecute pruebas A/B con variantes ponderadas y realice un seguimiento de los análisis de visitantes, todo a través de la conversación.
Úselo cuando: (1) cree o actualice un catálogo/embudo/página de destino, (2) verifique análisis como visitantes, conversiones y tasas de abandono, (3) ejecute pruebas A/B en diferentes versiones del catálogo, (4) direccione a los visitantes a la variante correcta del catálogo con sugerencias en lenguaje natural, (5) administre claves API para el acceso del equipo, (6) cargue videos para catálogos, (7) vea recorridos de visitantes individuales, (8) revise distribuciones de respuestas para campos de formulario, (9) cree sandboxes para editar catálogos de forma segura sin afectar la producción, (10) Usar el inspector de elementos para obtener referencias exactas de los componentes para los agentes de IA, (11) Agregar ganchos de secuencias de comandos para un comportamiento dinámico como llamadas API, enrutamiento condicional y estado entre páginas, (12) Cargar y comprimir imágenes para una carga rápida.
Activadores: embudo de catálogo, kit de catálogo, creador de embudos, página de destino, captura de clientes potenciales, creación de catálogo, análisis de catálogo, embudo de conversión, creador de formularios, prueba ab, API de catálogo, enrutamiento ai, enrutamiento de variantes, enrutamiento de sugerencias, sandbox, inspector de elementos, devtools, ganchos, secuencias de comandos, on_change, on_enter, on_submit, carga de imágenes, compresión de imágenes, webp
