AI エージェントを使用して、マーケティング カタログ、ランディング ページ、マルチステップ ファネルを構築および管理します。 JSON スキーマからカタログを作成し、即座に公開し、重み付けされたバリアントを使用して A/B テストを実行し、訪問者の分析を追跡します。これらはすべて会話を通じて行われます。
次の場合に使用します: (1) カタログ/ファネル/ランディング ページの作成または更新、(2) 訪問者、コンバージョン、離脱率などの分析の確認、(3) さまざまなカタログ バージョンでの A/B テストの実行、(4) 自然言語ヒントを使用して訪問者を適切なカタログ バリアントに AI ルーティング、(5) チーム アクセスの API キーの管理、(6) カタログのビデオのアップロード、(7) 個々の訪問者のジャーニーの表示、(8) フォーム フィールドの応答分布の確認、(9) の作成運用に影響を与えることなくカタログを安全に編集するためのサンドボックス、(10) AI エージェントの正確なコンポーネント参照を取得するための要素インスペクターの使用、(11) API 呼び出し、条件付きルーティング、クロスページ状態などの動的な動作のためのスクリプト フックの追加、(12) 高速読み込みのための画像のアップロードと圧縮。
