使用 AI 代理构建和管理营销目录、登陆页面和多步骤渠道。从 JSON 模式创建目录，立即发布它们，使用加权变体运行 A/B 测试，并跟踪访客分析 - 所有这些都通过对话进行。
在以下情况下使用：(1) 创建或更新目录/渠道/登陆页面，(2) 检查访客、转化和流失率等分析，(3) 在不同的目录版本上运行 A/B 测试，(4) 通过自然语言提示将访客路由到正确的目录变体，(5) 管理团队访问的 API 密钥，(6) 上传目录视频，(7) 查看个人访客旅程，(8) 查看表单字段的响应分布，(9) 创建沙箱以安全地编辑目录而不影响生产，(10) 使用元素检查器获取 AI 代理的准确组件引用，(11) 为 API 调用、条件路由和跨页面状态等动态行为添加脚本挂钩，(12) 上传和压缩图像以实现快速加载。
