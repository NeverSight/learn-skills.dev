OfficeX의 Farm TikTok 앱(슬러그: video-farm)용 REST API 클라이언트입니다. 키워드 또는 채널 검색을 통해 우승한 TikTok 콘텐츠를 발견하고, Gemini를 사용한 AI 필터를 사용하고, 중복 제거된 재게시를 자원 게시자에게 예약하고, 출판 증명을 추적합니다. 또한 사용자가 검색을 건너뛰고 자신의 비디오/콘텐츠를 예약 파이프라인에 직접 삽입하는 수동 모드(BYO 콘텐츠)도 지원합니다. 다음 경우에 사용하십시오: (1) 키워드 또는 채널 스크랩으로 콘텐츠 검색 작업 생성, (2) 검색된 비디오 결과 검토, 승인 또는 거부, (3) 자원봉사자를 위한 AI 생성 캡션/지시로 승인된 비디오 예약, (4) 콘텐츠 달력 보기 또는 관리, (5) 자원봉사자의 게시물 증명 제출 또는 확인, (6) NocoDB 스프레드시트 보기 확인, (7) 모든 TikTok 테마 페이지 성장 또는 콘텐츠 파밍 워크플로, (8) 일정/캘린더 파이프라인에 자신의 비디오를 삽입합니다(수동 모드). 트리거: 틱톡 팜, 일일 틱톡, 틱톡 팜, 틱톡 콘텐츠, 테마 페이지, 틱톡 리포스트, 틱톡 일정, 콘텐츠 농업, 틱톡 성장, 자원 봉사 게시물, 틱톡 달력, 틱톡 증명, 매뉴얼 콘텐츠, 바이 비디오.
