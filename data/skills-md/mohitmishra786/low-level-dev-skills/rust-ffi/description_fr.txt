Compétence Rust FFI pour l’interopérabilité C. À utiliser lors de l'appel de bibliothèques C depuis Rust, de la génération de liaisons Rust avec bindgen, de l'exportation de fonctions Rust vers C avec cbindgen, de l'écriture de wrappers sécurisés autour de FFI non sécurisés ou de la liaison des bibliothèques système et fournisseur. S'active sur les requêtes concernant bindgen, cbindgen, extern "C", FFI non sécurisé, les liaisons Rust C, la liaison C à partir de Rust ou les caisses système.
