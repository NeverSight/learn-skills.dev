Навыки Rust FFI для совместимости с C. Используйте при вызове библиотек C из Rust, создании привязок Rust с помощьюbindgen, экспорте функций Rust в C с помощью cbindgen, написании безопасных оболочек для небезопасных FFI или связывании системных и вендорных библиотек. Активируется по запросам о привязках Bindgen, cbindgen, extern "C", небезопасных FFI, привязках Rust C, связывании C с Rust или системных крейтах.
