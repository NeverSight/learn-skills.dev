Competenza FFI Rust per l'interoperabilità C. Da utilizzare quando si chiamano librerie C da Rust, si generano collegamenti Rust con bindgen, si esportano funzioni Rust in C con cbindgen, si scrivono wrapper sicuri attorno a FFI non sicuri o si collegano librerie di sistema e fornitori. Si attiva su query su bindgen, cbindgen, extern "C", FFI non sicuro, collegamenti Rust C, collegamento di C da Rust o casse di sistema.
