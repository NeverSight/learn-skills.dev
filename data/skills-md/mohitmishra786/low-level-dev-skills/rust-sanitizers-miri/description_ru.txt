Дезинфицирующие средства от ржавчины и навыки Мири для проверки безопасности памяти. Используйте при запуске AddressSanitizer или ThreadSanitizer в коде Rust, интерпретации отчетов дезинфицирующего средства, использовании Miri для обнаружения неопределенного поведения в небезопасном Rust или проверке правильности небезопасного кода. Активируется при запросах о Rust ASan, Rust Tsan, Miri, RUSTFLAGS Sanitize, Cargo Miri, небезопасном Rust UB или интерпретации выходных данных Rust Sanitizer.
