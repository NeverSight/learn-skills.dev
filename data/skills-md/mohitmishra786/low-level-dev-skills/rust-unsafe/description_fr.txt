Compétence de code dangereux pour la programmation de systèmes. À utiliser lors de l'écriture ou de la révision de Rust non sécurisé, pour comprendre quelles opérations nécessitent un caractère non sécurisé, pour implémenter des abstractions sécurisées sur du code non sécurisé, pour auditer des blocs non sécurisés ou pour comprendre les pointeurs bruts, la transmutation et l'externe. S'active sur les requêtes concernant Rust non sécurisé, les pointeurs bruts, la transmutation, les blocs non sécurisés, l'écriture de wrappers sécurisés, UnsafeCell, l'implément de trait non sécurisé ou l'audit de code non sécurisé.
