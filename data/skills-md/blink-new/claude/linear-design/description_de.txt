Erstellen und prüfen Sie eine Benutzeroberfläche in Linear-/Vercel-/Notion-Qualität. Behandelt Farbsysteme, Bewegung, progressive Offenlegung, Keyboard-First-Design, Layout-Architektur, Oberflächenhöhe, Informationsdichte, Mikrointeraktionen und Zustandsbehandlung. Verwenden Sie es beim Erstellen von UI-Komponenten, beim Überprüfen der Designqualität, beim Erstellen von Seiten oder wenn der Benutzer eine Designüberprüfung anfordert.
