استخدمها عند تصميم واجهات مستخدم الويب أو مراجعتها أو تنفيذها وتحتاج إلى قواعد "يجب/ينبغي/أبدًا" ملموسة لإمكانية الوصول أو أنماط التفاعل أو النماذج أو التخطيط أو الرسوم المتحركة أو الأداء أو المحتوى أو قرارات التصميم المرئي. استخدمه أيضًا عندما يُطلب منك "مراجعة واجهة المستخدم الخاصة بي" أو "التحقق من إمكانية الوصول" أو "تصميم التدقيق".
