دليل لاستخدام @guiiai/logg - مسجل منظم خفيف الوزن لـ Node.js والمتصفح. استخدم هذه المهارة عندما يقوم المستخدم بالاستيراد من '@guiiai/logg'، أو يذكر السجل، أو يحتاج إلى تسجيل منظم باستخدام JSON أو مخرجات جميلة، أو يريد سجلات معرفة المتصل مع ارتباطات تشعبية قابلة للنقر، أو يحتاج إلى أدوات تسجيل فرعية ذات حقول ثابتة، أو يسأل عن بدائل خفيفة الوزن لـ Winston أو Pino أو Bunyan. استخدمه أيضًا عند إعداد تسجيل التطبيق باستخدام مستويات السجل أو التنسيقات المخصصة أو استخراج الأخطاء.
