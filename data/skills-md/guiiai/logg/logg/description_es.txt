Guía para usar @guiiai/logg: un registrador estructurado liviano para Node.js y el navegador. Utilice esta habilidad siempre que el usuario importe desde '@guiiai/logg', mencione logg, necesite un registro estructurado con JSON o una salida bonita, quiera registros sensibles a las llamadas con hipervínculos en los que se pueda hacer clic, necesite registradores secundarios con campos persistentes o pregunte sobre alternativas ligeras a winston, pino o bunyan. Utilícelo también al configurar el registro de aplicaciones con niveles de registro, formateadores personalizados o extracción de errores.
