يحلل استعلامات SQL لمشكلات الأداء: الفهارس المفقودة، وأنماط N+1، والصلات دون المستوى الأمثل، وعمليات فحص الجدول بالكامل. يفسر مخرجات EXPLAIN، ويكتشف الأنماط المضادة، ويوصي بالفهارس، ويعيد كتابة الاستعلامات مع شرح تفصيلي لكل تحسين. مشغلات على: "تحسين هذا الاستعلام"، "الاستعلام البطيء"، "أداء الاستعلام"، "إضافة فهارس"، "توصية الفهرس"، "شرح الخطة"، "خطة الاستعلام"، "استعلام N+1"، "إصلاح هذا SQL"، "أداء SQL"، "تحسين SQL"، "لماذا هذا الاستعلام بطيء"، "استراتيجية الفهرس". استخدم هذه المهارة عند إعطائك استعلام SQL يحتاج إلى تحليل الأداء أو تحسينه.
