Analysiert SQL-Abfragen auf Leistungsprobleme: fehlende Indizes, N+1-Muster, suboptimale Verknüpfungen, vollständige Tabellenscans. Interpretiert die EXPLAIN-Ausgabe, erkennt Anti-Patterns, empfiehlt Indizes und schreibt Abfragen mit detaillierten Erläuterungen zu jeder Optimierung neu. Löst aus bei: „Diese Abfrage optimieren“, „Abfrage langsam“, „Abfrageleistung“, „Indizes hinzufügen“, „Indexempfehlung“, „Plan erklären“, „Abfrageplan“, „N+1-Abfrage“, „Diese SQL beheben“, „SQL-Leistung“, „SQL optimieren“, „Warum ist diese Abfrage langsam“, „Indexstrategie“. Verwenden Sie diese Fähigkeit, wenn Sie eine SQL-Abfrage erhalten, die eine Leistungsanalyse oder -optimierung erfordert.
