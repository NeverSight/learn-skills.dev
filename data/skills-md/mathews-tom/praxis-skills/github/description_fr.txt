Opérations CLI GitHub via `gh` pour les problèmes, les demandes d'extraction, les CI/Actions, les versions, les dépôts, la recherche, l'essentiel et l'API REST/GraphQL. Sortie structurée avec `--json` et `--jq` pour l'analyse. Couvre `gh issue create/list/view/edit/close`, `gh pr create/review/merge/checks`, `gh run list/view/rerun/watch`, `gh release create`, `gh search repos/issues/prs/code`, `gh api` pour les requêtes REST et GraphQL et `gh gist`. Inclut la gestion des erreurs pour HTTP 401/403/404/422/429, le dépannage de l'étendue et la gestion des limites de débit. Phrases déclencheurs : "créer un problème", "signaler un bug", "ouvrir un ticket", "soumettre un PR", "lancer une pull request", "vérifier le pipeline", "afficher les résultats des tests", "échec du CI", "pourquoi le CI a-t-il échoué", "vérifier l'état du CI", "fusionner un PR", "gérer les versions", "interroger l'API GitHub", "rechercher des référentiels", "trier les workflows", "automatiser les opérations GitHub". Se déclenche également lorsque l'utilisateur colle une URL GitHub.
