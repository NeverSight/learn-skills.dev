Аудит и улучшение документации API для конечных точек FastAPI и REST. Выявляет отсутствующие описания, неполные коды ответов, отсутствующие примеры и генерирует расширенные строки документации, примеры моделей Pydantic и улучшения спецификаций OpenAPI. Срабатывает при: «создать документацию API», «документировать этот API», «OpenAPI для», «добавить примеры», «улучшить строки документации», «аудит документации API», «документация FastAPI», «конечные точки документа», «ссылка на API», «документы Swagger», «документация REST API», «документация конечной точки», «документация ответа». Используйте этот навык, когда конечным точкам API требуется документация или аудит документации.
