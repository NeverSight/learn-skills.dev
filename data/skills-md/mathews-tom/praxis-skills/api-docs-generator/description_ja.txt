FastAPI および REST エンドポイントの API ドキュメントを監査し、強化します。欠落している説明、不完全な応答コード、欠落しているサンプルを特定し、拡張された docstring、Pydantic モデルのサンプル、および OpenAPI 仕様の改善を生成します。トリガー: 「API ドキュメントの生成」、「この API のドキュメント化」、「OpenAPI for」、「例の追加」、「ドキュメント文字列の改善」、「API ドキュメント監査」、「FastAPI ドキュメント」、「エンドポイントのドキュメント化」、「API リファレンス」、「swagger ドキュメント」、「REST API ドキュメント」、「エンドポイント ドキュメント」、「応答ドキュメント」。 API エンドポイントにドキュメントまたはドキュメント監査が必要な場合は、このスキルを使用します。
