使用 Remotion (React) 建立生產級動態圖形和影片。每當使用者需要品牌影片內容、產品簡報、資料驅動影片產生或具有音訊同步的動態圖形、Web 字體、TailwindCSS 樣式或媒體嵌入時即可使用。涵蓋：行銷影片、產品發布、資料視覺化、社群媒體內容、大規模個人化影片、帶有畫外音的解說影片、動畫圖表、透過 Three.js 製作的 3D 場景。需要 Node.js 和 Claude Code 環境。觸發條件：「創建 Remotion 視訊」、「React 視訊」、「動態圖形」、「品牌視訊」、「產品示範影片」、「遠端」、「有音訊的視訊」、「TailwindCSS 視訊」、「資料驅動的視訊產生」、「大規模的畫外音」、「內建視訊」。對於數學動畫、演算法視覺化或無頭容器渲染，請改用概念到影片 (Manim)。
