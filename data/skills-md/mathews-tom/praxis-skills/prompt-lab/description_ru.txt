Систематическая разработка подсказок LLM: анализирует существующие подсказки на предмет режимов сбоя, генерирует структурированные варианты (прямые, пошаговые, цепочки мыслей), разрабатывает критерии оценки с взвешенными критериями и создает наборы тестовых примеров для сравнения производительности подсказок. Запускается по: «быстрое проектирование», «быстрая лабораторная работа», «генерация вариантов подсказки», «подсказки для A/B-тестирования», «оценка подсказки», «оптимизировать подсказку», «написать лучшую подсказку», «подсказка по проектированию», «подсказка по итерации», «несколько примеров», «подсказка по цепочке мыслей», «подсказка о режимах отказа», «улучшить эту подсказку». Используйте этот навык при разработке, улучшении или оценке предложений LLM. НЕ для оценки навыков Claude Code или файлов SKILL.md — вместо этого используйте оценщик навыков.
