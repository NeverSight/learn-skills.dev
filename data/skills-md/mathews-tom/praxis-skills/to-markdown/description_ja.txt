あらゆるファイルまたは URL をクリーンな Markdown に変換します。 PDF、Word (DOCX)、Excel (XLSX)、PowerPoint (PPTX)、HTML、画像 (EXIF + OCR)、音声 (文字起こし)、CSV、JSON、XML、YouTube URL、EPub などを処理します。出力は、LLM パイプライン、ナレッジ ベース、ドキュメント取り込みワークフロー用に最適化されています。ファイルをマークダウンに変換する、ドキュメントからテキストを抽出する、URL をマークダウンにスクレイピングする、PDF を読み取り可能なテキストに変換する、「このファイルのコンテンツを取得する」、RAG 用にドキュメントを取り込む、LLM 用にファイルを準備する、または「md / マークダウンに変換する」というユーザーが必要なときにこのスキルを使用します。トリガー: 「マークダウンに変換」、「PDF からテキストを抽出」、「これをマークダウンに変換」、「この URL をスクレイピング」、「ファイルを md に」、「ドキュメントの取り込み」、「この PDF を読む」、「内容を取得」、「このドキュメントを解析」、「RAG 用に取り込み」。
