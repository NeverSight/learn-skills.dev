Конвертируйте любой файл или URL-адрес в чистую Markdown. Обрабатывает PDF, Word (DOCX), Excel (XLSX), PowerPoint (PPTX), HTML, изображения (EXIF + OCR), аудио (транскрипция), CSV, JSON, XML, URL-адреса YouTube, EPub и многое другое. Вывод оптимизирован для конвейеров LLM, баз знаний и рабочих процессов приема документов. Используйте этот навык всякий раз, когда пользователь хочет: преобразовать файл в уценку, извлечь текст из документа, очистить URL-адрес для уценки, превратить PDF в читаемый текст, «получить содержимое этого файла», вставить документы для RAG, подготовить файлы для LLM или сказать «конвертировать в md/markdown». Запуск по: «преобразовать в уценку», «извлечь текст из PDF», «превратить это в уценку», «очистить этот URL», «файл в md», «прием документа», «прочитать этот PDF», «получить содержимое», «проанализировать этот документ», «загрузить для RAG».
