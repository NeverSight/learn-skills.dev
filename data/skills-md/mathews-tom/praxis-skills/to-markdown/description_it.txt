Utilizza questa abilità ogni volta che l'utente desidera: convertire un file in markdown, estrarre testo da un documento, raschiare un URL per markdown, trasformare un PDF in testo leggibile, "ottenere il contenuto di questo file", importare documenti per RAG, preparare file per un LLM o dice "converti in md / markdown".
