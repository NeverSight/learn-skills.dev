Web コンテンツの取得と、curl および WebFetch による URL 取得 — Fetch MCP サーバー (fetch_html、fetch_json、fetch_markdown、fetch_txt) を置き換えます。特定の URL が提供され、ユーザーがそのコンテンツを必要とする場合は、このスキルを使用します。 HTTP GET/POST、jq による JSON API の使用、HTML 取得、マークダウン変換、プレーン テキスト抽出、認証されたリクエスト、リダイレクト、Cookie、タイムアウトをカバーします。トリガーフレーズ: 「この URL を取得する」、「ページ コンテンツを取得する」、「HTML をダウンロードする」、「この API を呼び出す」、「このエンドポイントをカールする」、「JSON を取得する」、「マークダウンを取得する」、「Web コンテンツを取得する」、「このエンドポイントにヒットする」、「このページを取得する」、「この URL を読み取る」、「API からデータを取得する」、「HTTP リクエストを作成する」、「ページ コンテンツを抽出する」、「記事テキストを取得する」。 URL のない Web 検索には使用できません。その場合は tavily を使用してください。
