Recupero dei contenuti Web e recupero degli URL tramite curl e WebFetch: sostituisce il server Fetch MCP (fetch_html, fetch_json, fetch_markdown, fetch_txt). Utilizza questa abilità quando viene fornito un URL specifico e l'utente ne desidera il contenuto. Copre HTTP GET/POST, consumo di API JSON con jq, recupero HTML, conversione markdown, estrazione di testo semplice, richieste autenticate, reindirizzamenti, cookie e timeout. Frasi di attivazione: "recupera questo URL", "recupera il contenuto della pagina", "scarica HTML", "chiama questa API", "arriccia questo endpoint", "prendi il JSON", "recupera markdown da", "recupera contenuto web", "raggiungi questo endpoint", "raschia questa pagina", "leggi questo URL", "estrai dati dall'API", "effettua una richiesta HTTP", "estrai il contenuto della pagina", "ottieni il testo dell'articolo". NON per ricerche web senza URL: usa tavily per questo.
