透過curl和WebFetch進行Web內容取得和URL檢索－取代Fetch MCP伺服器（fetch_html、fetch_json、fetch_markdown、fetch_txt）。當提供了特定 URL 並且使用者想要其內容時，請使用此技能。涵蓋 HTTP GET/POST、使用 jq 的 JSON API 使用、HTML 檢索、Markdown 轉換、純文字擷取、經過驗證的請求、重新導向、cookie 和逾時。觸發短語：「取得此 URL」、「取得頁面內容」、「下載 HTML」、「呼叫此 API」、「捲曲此端點」、「取得 JSON」、「從中取得 markdown」、「檢索網頁內容」、「點擊此端點」、「抓取此頁面」、「讀取此 URL」、「從 API 中提取資料」、「發出 HTTP 要求文章、擷取此頁面」、「從 API 擷取文字」、「擷取內容」不適用於沒有 URL 的網路搜尋 — 請使用 tavilly。
