Извлечение веб-контента и получение URL-адресов с помощью Curl и WebFetch — заменяет сервер Fetch MCP (fetch_html, fetch_json, fetch_markdown, fetch_txt). Используйте этот навык, когда указан конкретный URL-адрес и пользователю нужен его контент. Охватывает HTTP GET/POST, использование JSON API с помощью jq, извлечение HTML, преобразование уценки, извлечение простого текста, аутентифицированные запросы, перенаправления, файлы cookie и тайм-ауты. Триггерные фразы: «получить этот URL-адрес», «получить содержимое страницы», «загрузить HTML», «вызвать этот API», «свернуть эту конечную точку», «захватить JSON», «получить уценку из», «получить веб-контент», «достичь этой конечной точки», «очистить эту страницу», «прочитать этот URL», «извлечь данные из API», «сделать HTTP-запрос», «извлечь содержимое страницы», «получить текст статьи». НЕ для поиска в Интернете без URL-адреса — для этого используйте tavily.
