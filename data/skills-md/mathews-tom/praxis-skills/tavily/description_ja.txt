Tavily API を介した AI に最適化された Web 検索。特定の URL が指定されておらず、ユーザーが最新情報、最近のイベント、技術文書、またはニュースを Web で検索する必要がある場合に、このスキルを使用します。トリガー: 「Web を検索する」、「オンラインでこれを調べる」、「最近の情報を見つける」、「検索する」、「これを Google で検索する」、「Web 検索」、「調べる」、「現在のデータを見つける」、「Web で何について述べているか」、「オンラインで調査する」。既知の URL を取得するためではありません。そのためには Web-fetch を使用してください。 Tavily は、検索優先の目的に最適化されたクリーンな AI 対応スニペットを返します。
