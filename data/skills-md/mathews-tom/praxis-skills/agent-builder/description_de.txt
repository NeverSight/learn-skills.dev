Erstellen Sie KI-Agenten und automatisieren Sie Claude Code programmgesteuert mit dem Claude Agent SDK und dem Headless-CLI-Modus. Verwenden Sie diese Fähigkeit, wenn Sie einen Agenten erstellen, einen Claude-Agenten erstellen, einen Bot erstellen, mit dem Agenten-SDK arbeiten, Claude im Headless-Modus ausführen, programmgesteuerten Agentencode schreiben, mit Claude automatisieren, einen MCP-Server-Builder erstellen oder Claude programmgesteuert abfragen müssen. Behandelt das Python SDK, die Headless-Schnittstelle claude -p, die Erstellung benutzerdefinierter Tools mit SDK-MCP-Servern, Hooks für deterministische Steuerung, Sitzungsverwaltung und CLI-Flag-Referenz. Die Authentifizierung verwendet die vorhandene ~/.claude/config – es sind keine API-Schlüssel erforderlich.
