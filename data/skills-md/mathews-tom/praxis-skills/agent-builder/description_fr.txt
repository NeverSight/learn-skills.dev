Créez des agents IA et automatisez Claude Code par programmation à l'aide du SDK Claude Agent et du mode CLI sans tête. Utilisez cette compétence lorsque vous devez créer un agent, créer un agent Claude, créer un bot, travailler avec le SDK de l'agent, exécuter Claude en mode sans tête, écrire du code d'agent programmatique, automatiser avec Claude, créer un générateur de serveur MCP ou interroger Claude par programmation. Couvre le SDK Python, l'interface sans tête claude -p, la création d'outils personnalisés avec les serveurs SDK MCP, les hooks pour le contrôle déterministe, la gestion de session et la référence des indicateurs CLI. L'authentification utilise la configuration ~/.claude/ existante — aucune clé API n'est requise.
