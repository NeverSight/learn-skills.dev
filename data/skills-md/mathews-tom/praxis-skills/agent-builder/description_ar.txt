قم ببناء وكلاء الذكاء الاصطناعي وأتمتة Claude Code برمجيًا باستخدام Claude Agent SDK ووضع CLI بدون رأس. استخدم هذه المهارة عندما تحتاج إلى إنشاء وكيل، أو إنشاء وكيل Claude، أو إنشاء روبوت، أو العمل مع الوكيل SDK، أو تشغيل Claude في وضع بدون رأس، أو كتابة كود وكيل برمجي، أو التشغيل الآلي باستخدام Claude، أو إنشاء منشئ خادم MCP، أو الاستعلام عن Claude برمجيًا. يغطي Python SDK، وواجهة claude -p بدون رأس، وإنشاء أداة مخصصة باستخدام خوادم SDK MCP، وخطافات للتحكم الحتمي، وإدارة الجلسة، ومرجع إشارة CLI. تستخدم المصادقة التكوين ~/.claude/ config الموجود — ولا يلزم وجود مفاتيح API.
