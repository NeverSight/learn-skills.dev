Создавайте агенты искусственного интеллекта и автоматизируйте Claude Code программно с помощью Claude Agent SDK и режима headless CLI. Используйте этот навык, когда вам нужно создать агент, создать агент Claude, создать бота, работать с SDK агента, запустить Claude в автономном режиме, написать программный код агента, автоматизировать с помощью Claude, создать построитель сервера MCP или программно запросить Claude. Охватывает Python SDK, безголовый интерфейс claude -p, создание пользовательских инструментов с помощью серверов SDK MCP, перехватчики для детерминированного управления, управления сеансами и ссылку на флаги CLI. Для аутентификации используется существующая конфигурация ~/.claude/ — ключи API не требуются.
