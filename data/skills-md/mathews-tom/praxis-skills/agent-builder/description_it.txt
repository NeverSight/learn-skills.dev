Crea agenti AI e automatizza Claude Code in modo programmatico utilizzando Claude Agent SDK e la modalità CLI headless. Utilizza questa competenza quando devi creare un agente, creare un agente Claude, creare un bot, lavorare con l'SDK dell'agente, eseguire Claude in modalità headless, scrivere codice agente programmatico, automatizzare con Claude, creare un generatore di server MCP o eseguire query su Claude a livello di codice. Copre l'SDK Python, l'interfaccia headless claude -p, la creazione di strumenti personalizzati con server MCP SDK, hook per il controllo deterministico, gestione delle sessioni e riferimento ai flag CLI. L'autenticazione utilizza ~/.claude/config esistente: non sono richieste chiavi API.
