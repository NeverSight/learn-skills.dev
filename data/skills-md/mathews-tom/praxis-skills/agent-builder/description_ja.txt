Claude Agent SDK とヘッドレス CLI モードを使用して、AI エージェントを構築し、Claude Code をプログラムで自動化します。このスキルは、エージェントの構築、クロード エージェントの作成、ボットの作成、エージェント SDK の操作、ヘッドレス モードでのクロードの実行、プログラムによるエージェント コードの作成、クロードによる自動化、MCP サーバー ビルダーの作成、プログラムによるクロードのクエリを行う必要がある場合に使用します。 Python SDK、claude -p ヘッドレス インターフェイス、SDK MCP サーバーを使用したカスタム ツールの作成、決定論的制御用のフック、セッション管理、および CLI フラグ リファレンスについて説明します。認証には既存の ~/.claude/ config が使用されます。API キーは必要ありません。
