Opérations sur les fichiers et les répertoires à l'aide des outils intégrés de Claude Code - remplace le serveur Filesystem MCP. Mappe les 11 outils MCP vers leurs équivalents natifs : lecture, écriture, édition, Glob, Grep et Bash. Couvre la lecture de fichiers avec des plages de lignes, les lectures parallèles, la recherche de fichiers basée sur des modèles, la recherche de contenu regex, la liste de répertoires, la traversée d'arborescence, le déplacement/copie/renommage et l'inspection des métadonnées. Phrases de déclenchement : "lire ce fichier", "écrire dans un fichier", "créer un fichier", "modifier un fichier", "trouver les fichiers correspondants", "rechercher du texte dans les fichiers", "lister le répertoire", "afficher l'arborescence des répertoires", "déplacer le fichier", "renommer le fichier", "copier le fichier", "informations sur le fichier", "trouver tous les fichiers Python", "rechercher la base de code". Utilisez cette compétence lors de l'exécution d'opérations sur les fichiers, de la navigation dans les bases de code ou de la gestion des répertoires.
