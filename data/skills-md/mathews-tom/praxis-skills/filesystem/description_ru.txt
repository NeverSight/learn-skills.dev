Операции с файлами и каталогами с использованием встроенных инструментов Claude Code — заменяет сервер Filesystem MCP. Сопоставляет все 11 инструментов MCP с собственными эквивалентами: Read, Write, Edit, Glob, Grep и Bash. Охватывает чтение файлов с диапазонами строк, параллельное чтение, поиск файлов по шаблону, поиск содержимого по регулярным выражениям, листинг каталогов, обход дерева, перемещение/копирование/переименование и проверку метаданных. Триггерные фразы: «прочитать этот файл», «записать в файл», «создать файл», «редактировать файл», «найти соответствующие файлы», «поиск текста в файлах», «составить список каталогов», «показать дерево каталогов», «переместить файл», «переименовать файл», «копировать файл», «информация о файле», «найти все файлы Python», «найти кодовую базу». Используйте этот навык при выполнении файловых операций, навигации по базам кода или управлении каталогами.
