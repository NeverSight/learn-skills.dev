クロード コードの組み込みツールを使用したファイルとディレクトリの操作 — ファイルシステム MCP サーバーを置き換えます。 11 個の MCP ツールすべてをネイティブの同等ツール (Read、Write、Edit、Glob、Grep、Bash) にマップします。行範囲によるファイル読み取り、並列読み取り、パターンベースのファイル検索、正規表現コンテンツ検索、ディレクトリリスト、ツリートラバーサル、移動/コピー/名前変更、メタデータ検査をカバーします。トリガーフレーズ: 「このファイルを読み取る」、「ファイルに書き込む」、「ファイルを作成する」、「ファイルを編集する」、「一致するファイルを検索する」、「ファイル内のテキストを検索する」、「ディレクトリをリストする」、「ディレクトリツリーを表示する」、「ファイルを移動する」、「ファイルの名前を変更する」、「ファイルをコピーする」、「ファイル情報」、「すべての Python ファイルを検索する」、「コードベースを検索する」。このスキルは、ファイル操作の実行、コードベースの移動、またはディレクトリの管理を行うときに使用します。
