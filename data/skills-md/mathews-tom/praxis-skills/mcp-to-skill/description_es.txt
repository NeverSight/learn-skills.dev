Convierta servidores MCP (Protocolo de contexto modelo) en habilidades bajo demanda para reducir drásticamente el uso de ventanas de contexto activas. Los esquemas de herramientas MCP consumen tokens en cada turno, ya sea que se usen o no; Las habilidades se cargan sólo cuando es necesario. Analiza las definiciones de herramientas MCP, clasifica cada herramienta por estrategia de reemplazo y genera un paquete de habilidades completo. Utilice esta habilidad siempre que el usuario mencione: convertir MCP, MCP en habilidad, reducir el tamaño del contexto, demasiadas herramientas, aumento del token de herramientas, reducir los MCP, ventana de contexto llena, reemplazar MCP, migración de MCP. También se activa cuando el usuario tiene muchos servidores MCP activos y desea optimizar el uso del contexto, pregunta sobre la reducción de la sobrecarga de tokens de las definiciones de herramientas o dice que mi contexto es demasiado grande o se está quedando sin contexto cuando los MCP están activos.
