Converti i server MCP (Model Context Protocol) in competenze on-demand per ridurre drasticamente l'utilizzo della finestra di contesto attiva. Gli schemi degli strumenti MCP consumano token ad ogni turno, indipendentemente dal fatto che vengano utilizzati o meno; caricare le competenze solo quando necessario. Analizza le definizioni degli strumenti MCP, classifica ciascuno strumento in base alla strategia di sostituzione e genera un pacchetto di competenze completo. Utilizza questa abilità ogni volta che l'utente menziona: convertire MCP, MCP in abilità, ridurre le dimensioni del contesto, troppi strumenti, gonfiare i token degli strumenti, ridurre gli MCP, finestra del contesto piena, sostituire MCP, migrazione MCP. Si attiva anche quando l'utente ha molti server MCP attivi e desidera ottimizzare l'utilizzo del contesto, chiede di ridurre il sovraccarico dei token dalle definizioni degli strumenti o dice che il mio contesto è troppo grande o sta esaurendo il contesto quando gli MCP sono attivi.
