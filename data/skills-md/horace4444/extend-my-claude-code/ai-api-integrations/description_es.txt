Conecte aplicaciones, scripts y servicios backend a las API de modelos de IA (OpenAI, Anthropic Claude, Google Gemini/Vertex AI, xAI Grok), Supabase (base de datos PostgreSQL con búsqueda vectorial) y Clerk (autenticación). Úselo cuando cree funciones impulsadas por IA que requieran (1) integración de modelos de IA para generación de texto, traducción, incrustaciones o generación de imágenes, (2) operaciones de base de datos de Supabase con búsqueda semántica de pgvector, (3) autenticación de usuarios administrativos y administración de sesiones, (4) combinación de salidas de IA con almacenamiento de bases de datos, (5) selección de modelos con costos optimizados e ingeniería rápida, (6) mejores prácticas para implementaciones de producción que eviten antipatrones comunes.
