قم بتوصيل التطبيقات والبرامج النصية وخدمات الواجهة الخلفية بواجهات برمجة التطبيقات لنماذج الذكاء الاصطناعي (OpenAI وAnthropic Claude وGoogle Gemini/Vertex AI وxAI Grok) وSupabase (قاعدة بيانات PostgreSQL مع بحث المتجهات) وClerk (المصادقة). يُستخدم عند إنشاء ميزات مدعومة بالذكاء الاصطناعي تتطلب (1) تكامل نموذج الذكاء الاصطناعي لإنشاء النص أو الترجمة أو التضمين أو إنشاء الصور، (2) عمليات قاعدة بيانات Supabase مع البحث الدلالي pgvector، (3) مصادقة مستخدم Clerk وإدارة الجلسة، (4) الجمع بين مخرجات الذكاء الاصطناعي وتخزين قاعدة البيانات، (5) اختيار النموذج المحسّن من حيث التكلفة والهندسة السريعة، (6) أفضل الممارسات لعمليات نشر الإنتاج مع تجنب الأنماط المضادة الشائعة.
