Connetti applicazioni, script e servizi backend alle API del modello AI (OpenAI, Anthropic Claude, Google Gemini/Vertex AI, xAI Grok), Supabase (database PostgreSQL con ricerca vettoriale) e Clerk (autenticazione). Da utilizzare durante la creazione di funzionalità basate sull'intelligenza artificiale che richiedono (1) integrazione del modello AI per la generazione di testo, traduzione, incorporamenti o generazione di immagini, (2) operazioni di database Supabase con ricerca semantica pgvector, (3) autenticazione degli utenti e gestione delle sessioni, (4) combinazione di output AI con archiviazione di database, (5) selezione di modelli a costi ottimizzati e ingegneria tempestiva, (6) best practice per distribuzioni di produzione che evitano anti-pattern comuni.
