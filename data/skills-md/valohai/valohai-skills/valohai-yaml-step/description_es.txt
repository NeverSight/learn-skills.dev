Cree definiciones de pasos valohai.yaml para proyectos de ML. Utilice esta habilidad cuando un usuario quiera definir un nuevo paso en valohai.yaml, configurar imágenes de Docker, configurar comandos, definir parámetros/entradas para un paso o crear un valohai.yaml completo desde cero para su proyecto de ML. Se activa al mencionar valohai.yaml, definición de pasos, configuración de YAML, selección de imágenes de Docker o creación de pasos de Valohai.
