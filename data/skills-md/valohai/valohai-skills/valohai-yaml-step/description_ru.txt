Создайте определения шагов valohai.yaml для проектов машинного обучения. Используйте этот навык, когда пользователь хочет определить новый шаг в valohai.yaml, настроить образы Docker, настроить команды, определить параметры/входные данные для шага или создать полный файл valohai.yaml с нуля для своего проекта машинного обучения. Срабатывает при упоминании valohai.yaml, определении шага, конфигурации YAML, выборе образа Docker или создании шагов Valohai.
