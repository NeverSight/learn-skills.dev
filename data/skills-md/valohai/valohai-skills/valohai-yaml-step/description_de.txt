Erstellen Sie valohai.yaml-Schrittdefinitionen für ML-Projekte. Verwenden Sie diese Fähigkeit, wenn ein Benutzer einen neuen Schritt in valohai.yaml definieren, Docker-Images konfigurieren, Befehle einrichten, Parameter/Eingaben für einen Schritt definieren oder eine vollständige valohai.yaml von Grund auf für sein ML-Projekt erstellen möchte. Wird bei Erwähnungen von valohai.yaml, Schrittdefinition, YAML-Konfiguration, Docker-Image-Auswahl oder beim Erstellen von Valohai-Schritten ausgelöst.
