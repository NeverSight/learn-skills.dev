Migra il caricamento dei dati e il salvataggio dei modelli nel codice ML per utilizzare il sistema di input/output di Valohai. Utilizzare questa competenza quando un utente desidera configurare gli input di dati dall'archiviazione cloud (S3, BLOB di Azure, GCS), salvare gli output del modello in Valohai, sostituire i percorsi di file codificati, rimuovere il codice SDK boto3/cloud o configurare il sistema di I/O dei file Valohai. Si attiva quando vengono menzionati input, output, caricamento dei dati, salvataggio del modello, S3, archiviazione nel cloud, percorsi di file o migrazione dei dati Valohai.
