Перенесите загрузку данных и сохранение модели в коде ML, чтобы использовать систему ввода-вывода Valohai. Используйте этот навык, когда пользователь хочет настроить входные данные из облачного хранилища (S3, Azure Blob, GCS), сохранить выходные данные модели в Valohai, заменить жестко запрограммированные пути к файлам, удалить код boto3/cloud SDK или настроить систему файлового ввода-вывода Valohai. Срабатывает при упоминании входных и выходных данных, загрузке данных, сохранении модели, S3, облачном хранилище, путях к файлам или миграции данных Valohai.
