Migrieren Sie das Laden von Daten und das Speichern von Modellen in ML-Code, um das Eingabe-/Ausgabesystem von Valohai zu verwenden. Verwenden Sie diese Fähigkeit, wenn ein Benutzer Dateneingaben aus dem Cloud-Speicher (S3, Azure Blob, GCS) konfigurieren, Modellausgaben in Valohai speichern, hartcodierte Dateipfade ersetzen, Boto3/Cloud-SDK-Code entfernen oder das Valohai-Datei-E/A-System einrichten möchte. Wird bei Erwähnungen von Eingaben, Ausgaben, Datenladen, Modellspeicherung, S3, Cloud-Speicher, Dateipfaden oder Valohai-Datenmigration ausgelöst.
