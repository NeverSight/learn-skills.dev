Migre la carga de datos y el guardado de modelos en código ML para utilizar el sistema de entrada/salida de Valohai. Utilice esta habilidad cuando un usuario quiera configurar entradas de datos desde el almacenamiento en la nube (S3, Azure Blob, GCS), guardar las salidas del modelo en Valohai, reemplazar rutas de archivos codificadas, eliminar el código SDK de boto3/cloud o configurar el sistema de E/S de archivos de Valohai. Se activa cuando se mencionan entradas, salidas, carga de datos, guardado de modelos, S3, almacenamiento en la nube, rutas de archivos o migración de datos de Valohai.
