Migrez le chargement des données et la sauvegarde du modèle dans le code ML pour utiliser le système d'entrée/sortie de Valohai. Utilisez cette compétence lorsqu'un utilisateur souhaite configurer les entrées de données à partir du stockage cloud (S3, Azure Blob, GCS), enregistrer les sorties du modèle sur Valohai, remplacer les chemins de fichiers codés en dur, supprimer le code du SDK boto3/cloud ou configurer le système d'E/S de fichiers Valohai. Se déclenche sur les mentions d'entrées, de sorties, de chargement de données, d'enregistrement de modèle, de S3, de stockage cloud, de chemins de fichiers ou de migration de données Valohai.
