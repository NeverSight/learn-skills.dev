Créez un projet Valohai, liez-le à un répertoire local et exécutez des exécutions ou des pipelines à l'aide de la CLI Valohai. Utilisez cette compétence lorsqu'un utilisateur souhaite configurer un nouveau projet Valohai, lier un projet existant, exécuter une exécution d'étape, exécuter un pipeline, consulter les journaux d'exécution ou gérer son projet Valohai via la CLI. Se déclenche sur les mentions de création de projet vh, d'exécution d'exécution vh, d'exécution de pipeline vh, d'exécution sur Valohai ou de commandes CLI Valohai.
