أنشئ مشروع Valohai، واربطه بدليل محلي، وقم بتشغيل عمليات التنفيذ أو خطوط الأنابيب باستخدام Valohai CLI. استخدم هذه المهارة عندما يريد المستخدم إعداد مشروع Valohai جديد، أو ربط مشروع حالي، أو تشغيل تنفيذ خطوة، أو تشغيل خط أنابيب، أو مشاهدة سجلات التنفيذ، أو إدارة مشروع Valohai الخاص به عبر واجهة سطر الأوامر (CLI). المشغلات عند الإشارة إلى إنشاء مشروع vh، أو تشغيل تنفيذ vh، أو تشغيل خط أنابيب vh، أو التشغيل على أوامر Valohai، أو Valohai CLI.
