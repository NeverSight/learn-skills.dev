Создайте проект Valohai, свяжите его с локальным каталогом и запустите исполнения или конвейеры с помощью интерфейса командной строки Valohai. Используйте этот навык, когда пользователь хочет настроить новый проект Valohai, связать существующий проект, запустить пошаговое выполнение, запустить конвейер, просмотреть журналы выполнения или управлять своим проектом Valohai через интерфейс командной строки. Запускается при упоминании создания проекта vh, запуска выполнения vh, запуска конвейера vh, запуска на Valohai или команд Valohai CLI.
