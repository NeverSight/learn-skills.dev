Erstellen Sie ein Valohai-Projekt, verknüpfen Sie es mit einem lokalen Verzeichnis und führen Sie Ausführungen oder Pipelines mit der Valohai-CLI aus. Verwenden Sie diese Fähigkeit, wenn ein Benutzer ein neues Valohai-Projekt einrichten, ein vorhandenes Projekt verknüpfen, eine Schrittausführung ausführen, eine Pipeline ausführen, Ausführungsprotokolle überwachen oder sein Valohai-Projekt über die CLI verwalten möchte. Wird bei Erwähnungen von „vh project create“, „vh execution run“, „vhpipeline run“, „run on Valohai“ oder „Valohai CLI“-Befehlen ausgelöst.
