Crea un progetto Valohai, collegalo a una directory locale ed esegui esecuzioni o pipeline utilizzando la CLI Valohai. Utilizza questa competenza quando un utente desidera impostare un nuovo progetto Valohai, collegare un progetto esistente, eseguire un'esecuzione di passaggi, eseguire una pipeline, guardare i log di esecuzione o gestire il proprio progetto Valohai tramite la CLI. Si attiva quando si menziona la creazione del progetto vh, l'esecuzione dell'esecuzione vh, l'esecuzione della pipeline vh, l'esecuzione su Valohai o i comandi CLI di Valohai.
