Реализуйте возможности создания изображений ИИ с помощью z-ai-web-dev-sdk. Используйте этот навык, когда пользователю необходимо создавать изображения на основе текстовых описаний, генерировать визуальный контент, создавать иллюстрации, проектировать ресурсы или создавать приложения с помощью создания изображений с помощью искусственного интеллекта. Поддерживает несколько размеров изображений и возвращает изображения в кодировке Base64. Также включает инструмент CLI для быстрого создания изображений.
