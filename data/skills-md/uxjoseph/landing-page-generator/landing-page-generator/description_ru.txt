Навык автоматического создания подробной страницы (целевой страницы) на корейском языке. Берет информацию о продукте/услуге и создает высококонверсионную страницу с подробностями из 13 разделов. Использует API Gemini для создания изображений по разделам и выводит конечный результат в формате PNG/PDF.

Используйте, когда: запросы на страницы с подробными сведениями, целевые страницы, страницы с описанием продукта или страницы продаж.
