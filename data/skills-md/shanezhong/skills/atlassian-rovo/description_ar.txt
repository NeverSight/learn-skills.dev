يدير Atlassian Jira وConfluence عبر خادم Rovo MCP. يتعامل مع إعداد MCP ومصادقة OAuth واستكشاف الأخطاء وإصلاحها. يدير إدارة مشاريع الوكلاء: خطط التقاء وملاحم Jira مع تذاكر الأطفال وتنسيق فريق الوكلاء واستئناف العمل المتقطع من ولاية Jira. يدعم تحميل الصور/المرفقات إلى صفحات Confluence عبر REST API. يقرأ ويكتب تعليقات صفحة التقاء (التذييل، المضمنة، سلاسل الرد). إنشاء فروع git مرتبطة بتذاكر Jira (GitHub وBitbucket). استخدم هذه المهارة عندما يذكر المستخدم Jira أو Confluence أو Atlassian أو التذاكر أو الملاحم أو سباقات السرعة أو لوحات المشروع أو صفحات wiki أو مساحات Confluence. يتم التشغيل أيضًا عندما يريد المستخدم التخطيط لمشروع، أو تقسيم العمل إلى مهام، أو تتبع التقدم، أو استئناف العمل المتقطع، أو تحميل الصور إلى صفحات wiki، أو إدارة التعليقات على صفحات Confluence، أو إنشاء فروع git مرتبطة بالتذاكر - حتى لو لم يذكروا Atlassian بالاسم.
