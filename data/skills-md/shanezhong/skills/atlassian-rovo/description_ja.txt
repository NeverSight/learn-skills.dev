Rovo MCP サーバー経由で Atlassian Jira と Confluence を管理します。 MCP セットアップ、OAuth 認証、トラブルシューティングを処理します。エージェント プロジェクト管理を実行します: Confluence プラン、子チケットを含む Jira Epic、エージェント チームの調整、Jira 状態からの中断された作業の再開。 REST API を介した Confluence ページへの画像/添付ファイルのアップロードをサポートします。 Confluence ページのコメント (フッター、インライン、返信スレッド) の読み取りと書き込みを行います。 Jira チケット (GitHub および Bitbucket) にリンクされた Git ブランチを作成します。ユーザーが Jira、Confluence、Atlassian、チケット、エピック、スプリント、プロジェクト ボード、Wiki ページ、または Confluence スペースについて言及するたびに、このスキルを使用します。また、ユーザーがプロジェクトの計画、作業をタスクに分割、進捗状況の追跡、中断された作業の再開、Wiki ページへの画像のアップロード、Confluence ページのコメントの管理、またはチケットにリンクされた Git ブランチの作成を希望している場合にもトリガーします。たとえユーザーが Atlassian の名前を挙げていなくてもです。
