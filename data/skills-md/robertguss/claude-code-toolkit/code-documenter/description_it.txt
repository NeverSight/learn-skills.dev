Generatore di documentazione esperta per progetti di codifica. Analizza le basi di codice per creare una documentazione approfondita e completa per sviluppatori e utenti. Supporta aggiornamenti incrementali, documentazione multi-pubblico, record di decisioni sull'architettura e monitoraggio dello stato della documentazione. Funziona con qualsiasi tipo di progetto (API, CLI, app Web, librerie). Utilizzalo quando devi documentare un nuovo progetto, aggiornare i documenti dopo aver aggiunto funzionalità o creare documentazione completa per versioni open source.
