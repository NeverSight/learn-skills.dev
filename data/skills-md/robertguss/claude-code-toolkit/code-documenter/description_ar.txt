مولد وثائق الخبراء لمشاريع الترميز. يحلل قواعد التعليمات البرمجية لإنشاء وثائق شاملة وشاملة للمطورين والمستخدمين. يدعم التحديثات المتزايدة، والتوثيق متعدد الجماهير، وسجلات قرارات الهندسة المعمارية، وتتبع صحة التوثيق. يعمل مع أي نوع مشروع (واجهات برمجة التطبيقات، وواجهات سطر الأوامر، وتطبيقات الويب، والمكتبات). يُستخدم عندما تحتاج إلى توثيق مشروع جديد، أو تحديث المستندات بعد إضافة الميزات، أو إنشاء وثائق شاملة للإصدارات مفتوحة المصدر.
