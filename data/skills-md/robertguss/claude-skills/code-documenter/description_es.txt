Generador de documentación experto para proyectos de codificación. Analiza bases de código para crear documentación completa y exhaustiva para desarrolladores y usuarios. Admite actualizaciones incrementales, documentación para múltiples audiencias, registros de decisiones de arquitectura y seguimiento del estado de la documentación. Funciona con cualquier tipo de proyecto (API, CLI, aplicaciones web, bibliotecas). Úselo cuando necesite documentar un nuevo proyecto, actualizar documentos después de agregar funciones o crear documentación completa para versiones de código abierto.
