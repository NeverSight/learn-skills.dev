Решение для совместной отладки искусственного интеллекта. Превратите console.log в HTTP-запросы для сбора логов; после того, как пользователь завершит работу, ИИ сможет просматривать и анализировать их самостоятельно — скриншоты или копирование консоли не требуются. Поддерживает код Клода, OpenCode, курсор.
