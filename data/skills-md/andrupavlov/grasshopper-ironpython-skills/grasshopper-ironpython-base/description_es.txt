Escriba scripts de IronPython 2.7 para componentes de Rhino Grasshopper. Cubre restricciones de sintaxis, API de Rhino.Geometry y Grasshopper, operaciones de DataTree, estado sc.sticky, manejo de errores y estructura de componentes. Utilícelo al crear o editar archivos .py en el directorio Gh/, al escribir nodos de Grasshopper Python o al trabajar con la geometría de Rhino.
