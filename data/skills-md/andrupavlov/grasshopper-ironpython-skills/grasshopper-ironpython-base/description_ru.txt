Напишите сценарии IronPython 2.7 для компонентов Rhino Grasshopper. Охватывает синтаксические ограничения, API Rhino.Geometry и Grasshopper, операции DataTree, состояние sc.sticky, обработку ошибок и структуру компонентов. Используйте при создании или редактировании файлов .py в каталоге Gh/, написании узлов Grasshopper Python или работе с геометрией Rhino.
