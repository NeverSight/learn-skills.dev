Scrivi script IronPython 2.7 per i componenti di Rhino Grasshopper. Copre i vincoli di sintassi, le API di Rhino.Geometry e Grasshopper, le operazioni di DataTree, lo stato sc.sticky, la gestione degli errori e la struttura dei componenti. Da utilizzare quando si creano o si modificano file .py nella directory Gh/, si scrivono nodi Grasshopper Python o si lavora con la geometria di Rhino.
