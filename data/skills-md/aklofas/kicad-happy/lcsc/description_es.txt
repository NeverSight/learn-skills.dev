Busque componentes electrónicos en LCSC Electronics: busque piezas por número LCSC (Cxxxxx) o MPN, verifique stock/precios, descargue hojas de datos, analice especificaciones. Empresa hermana de JLCPCB, misma biblioteca de piezas. Sincronice y mantenga un directorio de hojas de datos local para un proyecto KiCad. No se necesita clave API: utiliza la API gratuita de la comunidad jlcsearch. Utilice esta habilidad cuando el usuario mencione LCSC, biblioteca de piezas de JLCPCB, piezas de ensamblaje de JLCPCB, abastecimiento de producción, números de pieza de Cxxxxx, necesite encontrar equivalentes de LCSC para piezas, esté preparando una lista de materiales para el ensamblaje de JLCPCB o quiera descargar hojas de datos y LCSC esté disponible. Para tablas de referencias cruzadas de paquetes y flujo de trabajo de BOM, consulte la habilidad `bom`.
