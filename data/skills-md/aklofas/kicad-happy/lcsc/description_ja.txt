LCSC Electronics で電子部品を検索 — LCSC 番号 (Cxxxxx) または MPN で部品を検索し、在庫/価格を確認し、データシートをダウンロードし、仕様を分析します。 JLCPCBの姉妹会社であり、同じ部品ライブラリです。 KiCad プロジェクトのローカル データシート ディレクトリを同期して維持します。 API キーは必要ありません。無料の jlcsearch コミュニティ API を使用します。このスキルは、ユーザーが LCSC、JLCPCB 部品ライブラリ、JLCPCB アセンブリ部品、生産調達、Cxxxxx 部品番号に言及した場合、部品の LCSC 相当品を見つける必要がある場合、JLCPCB アセンブリの BOM を準備している場合、またはデータシートをダウンロードしたい場合に使用します。LCSC は利用可能です。パッケージ相互参照テーブルと BOM ワークフローについては、「bom」スキルを参照してください。
