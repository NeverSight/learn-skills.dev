Найдите в DigiKey электронные компоненты и загрузите технические описания — основной источник заказов на прототипы и предпочтительный метод на основе API для получения технических описаний. Находите детали по ключевому слову или номеру детали, проверяйте цены/наличие на складе, загружайте таблицы данных напрямую через API, анализируйте спецификации. Синхронизируйте и поддерживайте локальный каталог таблиц данных для проекта KiCad — извлекайте компоненты из схем, загружайте все недостающие таблицы данных и поддерживайте их в актуальном состоянии. Используйте с KiCad для создания спецификации и выбора деталей. Используйте этот навык, когда пользователь спрашивает об электронных компонентах, спецификациях деталей, технических характеристиках, габаритах, ценах, наличии на складе или ему необходимо загрузить/прочитать техническое описание — даже если он не упоминает «DigiKey» по имени. Также используйте, когда пользователь говорит «синхронизировать таблицы данных», «загрузить таблицы данных для моей платы/проекта» или упоминает каталог таблиц данных. DigiKey является дистрибьютором по умолчанию для прототипирования и предпочтительным источником таблиц, поскольку его API возвращает прямые ссылки на PDF-файлы без очистки веб-страниц. Для получения информации о таблицах перекрестных ссылок пакетов и рабочем процессе спецификации см. навык bom.
