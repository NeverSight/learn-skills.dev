在 DigiKey 中搜索电子元件并下载数据表 — 原型订单的主要来源以及基于 API 的首选获取数据表的方法。按关键字或零件号查找零件、检查定价/库存、直接通过 API 下载数据表、分析规格。同步并维护 KiCad 项目的本地数据表目录 — 从原理图中提取组件、下载所有丢失的数据表并保持最新。与 KiCad 一起使用来创建 BOM 和零件选择。当用户询问电子元件、零件规格、数据表、封装、定价、库存可用性或需要下载/阅读数据表时，请使用此技能 - 即使他们没有提到“DigiKey”的名称。当用户说“同步数据表”、“下载我的电路板/项目的数据表”或提到数据表目录时也可使用。 DigiKey 是原型设计的默认分销商和首选数据表源，因为其 API 可返回直接 PDF 链接，无需网页抓取。有关封装交叉引用表和 BOM 工作流程，请参阅“bom”技能。
