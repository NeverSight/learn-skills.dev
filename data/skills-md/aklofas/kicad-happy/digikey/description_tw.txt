在 DigiKey 中搜尋電子元件並下載資料表 — 原型訂單的主要來源以及基於 API 的首選取得資料表的方法。按關鍵字或零件號碼尋找零件、檢查定價/庫存、直接透過 API 下載資料表、分析規格。同步並維護 KiCad 專案的本機資料表目錄 — 從原理圖中擷取元件、下載所有遺失的資料表並保持最新。與 KiCad 一起使用來創建 BOM 和零件選擇。當使用者詢問電子元件、零件規格、資料表、封裝、定價、庫存可用性或需要下載/閱讀資料表時，請使用此技能 - 即使他們沒有提到「DigiKey」的名稱。當使用者說「同步資料表」、「下載我的電路板/專案的資料表」或提到資料表目錄時也可使用。 DigiKey 是原型設計的預設分銷商和首選資料表來源，因為其 API 可返回直接 PDF 鏈接，無需網頁抓取。有關封裝交叉引用表和 BOM 工作流程，請參閱「bom」技能。
