Cerca componenti elettronici su DigiKey e scarica le schede tecniche: fonte primaria per ordini di prototipi e metodo preferito basato su API per il recupero delle schede tecniche. Trova ricambi per parola chiave o codice, controlla prezzi/disponibilità, scarica le schede tecniche direttamente tramite API, analizza le specifiche. Sincronizza e mantieni una directory dei fogli dati locali per un progetto KiCad: estrai componenti dagli schemi, scarica tutti i fogli dati mancanti e mantienili aggiornati. Utilizzare con KiCad per la creazione di distinte base e la selezione delle parti. Utilizza questa capacità quando l'utente chiede informazioni su componenti elettronici, specifiche delle parti, schede tecniche, ingombri, prezzi, disponibilità in magazzino o ha bisogno di scaricare/leggere una scheda tecnica, anche se non menziona "DigiKey" per nome. Da utilizzare anche quando l'utente dice "sincronizza fogli dati", "scarica fogli dati per la mia scheda/progetto" o menziona una directory di fogli dati. DigiKey è il distributore predefinito per la prototipazione e la fonte di schede tecniche preferita perché la sua API restituisce collegamenti PDF diretti senza web scraping. Per le tabelle dei riferimenti incrociati dei pacchetti e il flusso di lavoro della distinta base, consulta la competenza "bom".
