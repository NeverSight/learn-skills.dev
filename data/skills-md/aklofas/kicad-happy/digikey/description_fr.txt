Recherchez des composants électroniques dans DigiKey et téléchargez des fiches techniques : source principale pour les commandes de prototypes et méthode préférée basée sur l'API pour récupérer les fiches techniques. Recherchez des pièces par mot-clé ou numéro de pièce, vérifiez les prix/stocks, téléchargez les fiches techniques directement via l'API, analysez les spécifications. Synchronisez et gérez un répertoire de fiches techniques local pour un projet KiCad : extrayez les composants des schémas, téléchargez toutes les fiches techniques manquantes et tenez-les à jour. À utiliser avec KiCad pour la création de nomenclatures et la sélection de pièces. Utilisez cette compétence lorsque l'utilisateur pose des questions sur les composants électroniques, les spécifications des pièces, les fiches techniques, les empreintes, les prix, la disponibilité des stocks ou lorsqu'il a besoin de télécharger/lire une fiche technique, même s'il ne mentionne pas « DigiKey » par son nom. À utiliser également lorsque l'utilisateur dit « synchroniser les fiches techniques », « télécharger les fiches techniques pour ma carte/projet » ou mentionne un répertoire de fiches techniques. DigiKey est le distributeur par défaut pour le prototypage et la source de fiches techniques préférée car son API renvoie des liens PDF directs sans web scraping. Pour les tables de références croisées des packages et le workflow de nomenclature, consultez la compétence « bom ».
