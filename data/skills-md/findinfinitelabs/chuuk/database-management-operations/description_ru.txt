Специализированные операции с базами данных для данных чуукского языка, включая управление словарями, коллекциями фраз, парами переводов и лингвистическими метаданными. Поддерживает Azure CosmoDB с API MongoDB и локальную базу данных MongoDB. Используйте при работе с базами данных чуукского языка, управлении данными перевода или выполнении операций базы данных с наборами лингвистических данных.
