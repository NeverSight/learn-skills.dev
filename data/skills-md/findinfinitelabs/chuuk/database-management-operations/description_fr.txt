Opérations de base de données spécialisées pour les données en langue chuukese, y compris la gestion des dictionnaires, les collections de phrases, les paires de traduction et les métadonnées linguistiques. Prend en charge Azure Cosmos DB avec l'API MongoDB et MongoDB local. À utiliser lorsque vous travaillez avec des bases de données en langue chuukese, gérez des données de traduction ou effectuez des opérations de base de données sur des ensembles de données linguistiques.
