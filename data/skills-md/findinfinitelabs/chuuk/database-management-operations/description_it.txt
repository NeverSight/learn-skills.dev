Operazioni di database specializzate per dati in lingua chuukese, tra cui gestione di dizionari, raccolte di frasi, coppie di traduzioni e metadati linguistici. Supporta Azure Cosmos DB con API MongoDB e MongoDB locale. Da utilizzare quando si lavora con database in lingua chuukese, si gestiscono dati di traduzione o si eseguono operazioni di database su set di dati linguistici.
