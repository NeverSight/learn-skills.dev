عمليات قاعدة البيانات المتخصصة لبيانات لغة تشوكيز بما في ذلك إدارة القاموس ومجموعات العبارات وأزواج الترجمة والبيانات الوصفية اللغوية. يدعم Azure Cosmos DB مع MongoDB API وMongoDB المحلي. يُستخدم عند العمل مع قواعد بيانات لغة Chuukese، أو إدارة بيانات الترجمة، أو تنفيذ عمليات قاعدة البيانات على مجموعات البيانات اللغوية.
