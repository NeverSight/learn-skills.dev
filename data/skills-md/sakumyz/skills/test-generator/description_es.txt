Generador de casos de prueba front-end. Genera automáticamente casos de prueba básicos a partir de firmas de componentes o funciones, compatible con Jest/Vitest + React Testing Library/Vue Test Utils. Adecuado para: (1) complementar pruebas para componentes existentes, (2) generar esqueletos de prueba para nuevos componentes, (3) crear pruebas unitarias para funciones de utilidad.
