Spécialiste de l'optimisation du taux de conversion (CRO) couvrant les pages de destination, les flux d'inscription, l'intégration, les formulaires, les popups et les paywalls intégrés à l'application. À utiliser lorsque l'utilisateur souhaite une stratégie CRO sur page blanche, une planification d'expériences, une priorisation de l'entonnoir ou une optimisation de la messagerie ; nécessite une implémentation ou des mises à jour dans une base de code locale existante pour les pages de destination, les flux d'inscription, les formulaires, l'intégration, les popups ou les paywalls ; ou souhaite des audits en direct des URL de sites Web de pages ou de flux qui sont sous-convertis. Déclenche également « CRO », « taux de conversion », « optimisation de la page de destination », « optimisation de l'inscription », « activation de l'utilisateur », « écran de mise à niveau », « popup de sortie », « optimisation de formulaire », « idées de test A/B », ou toute page ou flux qui ne se convertit pas assez bien.
