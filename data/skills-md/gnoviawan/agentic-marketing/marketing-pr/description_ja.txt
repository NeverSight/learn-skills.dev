デジタル PR およびアウトリーチのスペシャリスト。メディア関係、プレス リリース、ジャーナリストのアウトリーチ、HARO/Connectively、リンク構築のためのデジタル PR キャンペーン、危機コミュニケーション、ブランドの評判管理、ソート リーダーシップの配置をカバーします。ユーザーが白紙の PR 戦略、メディア報道、プレスリリース、ジャーナリストのピッチング、PR によるバックリンクの構築、危機対応、またはライブ Web サイト、ニュースルーム、検索フットプリント、プロフィールの存在の監査を希望する場合に使用します。また、メディア キット、プレス ページ、広報担当者の準備、評判管理、またはリポジトリに裏付けされた PR アセットの実装のトリガーにもなります。
