Richten Sie über Inlay einen MCP-Server (Model Context Protocol) für jede Website ein.
Ermöglicht KI-Agenten das programmgesteuerte Durchsuchen, Durchsuchen und Interagieren mit Ihrer Website.
Verwenden Sie diese Option, wenn Sie MCP-Unterstützung hinzufügen, den AI-Agent-Zugriff aktivieren oder Inlay einrichten.
