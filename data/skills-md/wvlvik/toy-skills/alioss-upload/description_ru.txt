Загрузите файлы и каталоги в Alibaba Cloud OSS с помощью Python SDK v2. Поддерживает загрузку одного файла, загрузку нескольких изображений с автоматическим переименованием, пакетную загрузку каталогов и возобновляемую загрузку больших файлов. Используйте, когда пользователи запрашивают «загрузить в OSS», «阿里云OSS上传», «загрузить изображения», «批量上传图片» или любую задачу загрузки файла, связанную с Alibaba Cloud OSS.
