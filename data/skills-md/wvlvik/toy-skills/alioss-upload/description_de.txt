Laden Sie Dateien und Verzeichnisse mit Python SDK v2 in Alibaba Cloud OSS hoch. Unterstützt das Hochladen einzelner Dateien, das Hochladen mehrerer Bilder mit automatischer Umbenennung, das Hochladen von Batch-Verzeichnissen und das fortsetzbare Hochladen großer Dateien. Wird verwendet, wenn Benutzer „Auf OSS hochladen“, „阿里云OSS上传“, „Bilder hochladen“, „批量上传图片“ oder jede Datei-Upload-Aufgabe mit Alibaba Cloud OSS anfordern.
