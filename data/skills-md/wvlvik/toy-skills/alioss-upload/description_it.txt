Carica file e directory su Alibaba Cloud OSS utilizzando Python SDK v2. Supporta il caricamento di file singoli, il caricamento di più immagini con ridenominazione automatica, il caricamento di directory in batch e il caricamento ripristinabile di file di grandi dimensioni. Da utilizzare quando gli utenti richiedono "caricamento su OSS", "阿里云OSS上传", "caricamento immagini", "批量上传图片" o qualsiasi attività di caricamento di file che coinvolga Alibaba Cloud OSS.
