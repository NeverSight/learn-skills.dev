قم بتقسيم الفروع كبيرة الحجم أو ذات الاهتمامات المختلطة إلى مجموعات علاقات عامة أصغر وقابلة للمراجعة مع مراجع الأمان واختيار الهيكل وعمليات تدقيق التكافؤ وتسلسل الدمج. يُستخدم عندما يكون PR أو الفرع كبيرًا جدًا، أو يصعب مراجعته، أو مختلطًا عبر الاهتمامات، أو عرضة للصراع، أو يحتاج إلى التحلل دون فقدان صافي التغييرات.
