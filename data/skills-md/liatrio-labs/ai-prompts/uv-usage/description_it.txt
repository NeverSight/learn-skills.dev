Fornisce una guida concisa per l'utilizzo di uv (gestore di pacchetti Python), inclusi flussi di lavoro del progetto, comandi compatibili con pip, gestione delle versioni di Python e dipendenze di script inline PEP 723. Da utilizzare quando gli utenti menzionano uv, uv run, dipendenze inline, PEP 723 o gestione delle dipendenze/progetti Python.
