Crea messaggi di commit convenzionale ed esegui commit git sicuri dalle modifiche dell'albero di lavoro. Da utilizzare quando gli utenti chiedono di creare un commit, scrivere un messaggio di commit convenzionale, suddividere modifiche generali in più commit, organizzare solo parti di file, eseguire il pre-commit prima del commit o eseguire una rapida revisione della qualità in fase di commit con limiti di gravità.
