Erstellen Sie herkömmliche Commit-Nachrichten und führen Sie sichere Git-Commits aus Änderungen im Arbeitsbaum aus. Verwenden Sie diese Option, wenn Benutzer einen Commit erstellen, eine herkömmliche Commit-Nachricht schreiben, umfassende Änderungen in mehrere Commits aufteilen, nur Teile von Dateien bereitstellen, vor dem Commit einen Pre-Commit ausführen oder eine schnelle Qualitätsprüfung zur Commit-Zeit mit Schweregrad-Leitlinien durchführen möchten.
