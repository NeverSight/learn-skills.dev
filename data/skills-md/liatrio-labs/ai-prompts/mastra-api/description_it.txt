Interagisci con l'API del server di sviluppo Mastra per il debug degli agenti, la visualizzazione dei thread di conversazione, l'elenco/ispezione di strumenti e flussi di lavoro, l'accesso ai dati di osservabilità e la gestione delle risorse Mastra. Da utilizzare quando si lavora con agenti Mastra ed è necessario controllare lo stato di runtime, eseguire il debug degli errori dell'agente, visualizzare la cronologia dei thread, vedere strumenti/flussi di lavoro disponibili o accedere agli endpoint API su localhost:4111/api o porte simili.
