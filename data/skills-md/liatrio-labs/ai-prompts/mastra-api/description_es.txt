Interactúe con la API del servidor de desarrollo de Mastra para depurar agentes, ver hilos de conversación, enumerar/inspeccionar herramientas y flujos de trabajo, acceder a datos de observabilidad y administrar recursos de Mastra. Úselo cuando trabaje con agentes Mastra y necesite inspeccionar el estado del tiempo de ejecución, depurar errores del agente, ver el historial de subprocesos, ver herramientas/flujos de trabajo disponibles o acceder a puntos finales de API en localhost:4111/api o puertos similares.
