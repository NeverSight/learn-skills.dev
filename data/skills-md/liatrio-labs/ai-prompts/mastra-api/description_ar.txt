تفاعل مع واجهة برمجة تطبيقات خادم تطوير Mastra لوكلاء تصحيح الأخطاء، وعرض سلاسل المحادثات، وإدراج/فحص الأدوات وسير العمل، والوصول إلى بيانات إمكانية المراقبة، وإدارة موارد Mastra. يُستخدم عند العمل مع وكلاء Mastra وتحتاج إلى فحص حالة وقت التشغيل، أو تصحيح أخطاء الوكيل، أو عرض محفوظات سلسلة الرسائل، أو الاطلاع على الأدوات/سير العمل المتاحة، أو الوصول إلى نقاط نهاية واجهة برمجة التطبيقات على المضيف المحلي:4111/api أو منافذ مماثلة.
