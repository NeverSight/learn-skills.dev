مراقبة صحة النظام عبر Glances REST API (وحدة المعالجة المركزية والذاكرة والقرص والشبكة وأجهزة الاستشعار والحاويات والعمليات). يُستخدم عندما يطلب المستخدم "التحقق من اللمحات"، أو "إحصائيات النظام"، أو "استخدام وحدة المعالجة المركزية"، أو "استخدام الذاكرة"، أو "حالة اللمحات"، أو "حالة الخادم عبر اللمحات"، أو "مساحة القرص"، أو "حركة مرور الشبكة"، أو "إحصائيات الحاوية"، أو "قائمة العمليات"، أو "درجات حرارة المستشعر"، أو يذكر اللمحات/مراقبة النظام.
