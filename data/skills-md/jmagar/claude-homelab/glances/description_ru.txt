Отслеживайте состояние системы через Glances REST API (ЦП, память, диск, сеть, датчики, контейнеры, процессы). Используйте, когда пользователь запрашивает «проверить обзоры», «статистику системы», «использование процессора», «использование памяти», «состояние просмотров», «работоспособность сервера с помощью просмотров», «пространство на диске», «сетевой трафик», «статистику контейнера», «список процессов», «температуру датчиков» или упоминает Glances/мониторинг системы.
