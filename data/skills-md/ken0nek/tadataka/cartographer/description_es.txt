Descubrimiento de elementos de UI de iOS y Android y mapeo de navegación mediante CLI agente-dispositivo. Utilice esta habilidad siempre que el usuario quiera: encontrar localizadores/identificadores de elementos en una pantalla móvil, obtener información de accesibilidad para la automatización XCUITest o Espresso, tomar una instantánea de los elementos de la interfaz de usuario de una pantalla, navegar a una pantalla específica y capturar sus elementos, mapear los flujos de navegación entre pantallas, explorar la topología de la pantalla de una aplicación, auditar la cobertura del identificador de accesibilidad o cualquier cosa relacionada con "qué elementos hay en esta pantalla" o "cómo llego a esa pantalla". Activadores de: localizadores, identificadores, elementos, instantánea, mapa de pantalla, gráfico de navegación, auditoría de accesibilidad, exploración de aplicaciones, catálogo de interfaz de usuario, elementos XCUITest, jerarquía de vistas, localizadores de Appium, elementos de automatización de pruebas, descubrimiento de pantalla, catálogo de elementos, cartógrafo, elementos de Android, localizadores de espresso, nombre del paquete, instantánea de Android. Incluso si no mencionan al Cartógrafo por su nombre, use esta habilidad para cualquier tarea de descubrimiento de elementos de la interfaz de usuario móvil.
