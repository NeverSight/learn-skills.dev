قم بتيسير محادثة منظمة بين 3 أصدقاء بين المنتج والتطوير وضمان الجودة لتحديد الوظائف قبل التنفيذ، ثم استخرج المتطلبات الوظيفية الجاهزة للتنفيذ من المناقشة.
استخدم هذه المهارة عندما يذكر المستخدم 3 أصدقاء، أو تحسين الأعمال المتراكمة باستخدام المنتج/المطور/ضمان الجودة، أو تحديد المتطلبات قبل البرمجة، أو توضيح النطاق ومعايير القبول، أو مراجعة نص مناقشة المتطلبات، أو تحويل مناقشة الميزة إلى متطلبات نمط قصة المستخدم. استخدمه أيضًا عندما يطلب المستخدم تحديد أوجه الغموض أو الافتراضات أو قواعد العمل أو الأسئلة المفتوحة قبل بدء التطوير.
