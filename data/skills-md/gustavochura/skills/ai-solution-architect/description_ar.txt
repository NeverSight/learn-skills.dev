مهندس الحلول الرقمية المرتكزة على IA. طريقتان: (1) تحليل مستودعات الكود الموجود وشرح بنيته لأي جمهور، بما في ذلك الأشخاص الذين ليس لديهم معرفة تقنية. (2) تفكيك البنية الكاملة للأنظمة الجديدة التي تستخدم LLMs أو RAG أو الوكلاء أو الضبط الدقيق. استخدام هذه المهارة عند ذكر المستخدم: هندسة IA، تصميم نظام LLM، مهارات الهندسة المعمارية، هندسة RAG، مكدس التكنولوجيا لـ IA، قاعدة بيانات المتجهات، مخطط الهندسة، مكونات النظام، التضمين، الاسترجاع، خط أنابيب البيانات، MLOps، LLMOps، التركيزات التقييمية، RAG مقابل الضبط الدقيق، تصميم حل ذكاء اصطناعي، شرح المستودع، شرح الكود، تحليل المشروع، ما هو هذا المستودع، كيف يعمل هذا النظام، شرح هذا المشروع، أو أي تغيير في "ما هي المكونات المطلوبة" أو "الموضحة كوظيفة" هذا". يتم التنشيط عندما يحتاج المستخدم إلى كود، أو README، أو بنية الأرشيف، أو ذكر مستودع GitHub للتحليل. وأيضًا عندما أرغب في تصميم معماري جديد.
