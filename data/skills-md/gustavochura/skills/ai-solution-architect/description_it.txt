Architettura di soluzioni digitali basate su IA. Due modalità: (1) ANALIZZA i repository o il codice esistente e spiega la tua architettura per qualsiasi pubblico, incluse le persone senza conoscenza tecnica. (2) DISEGNA l'architettura completa di nuovi sistemi che utilizzano LLM, RAG, agenti o fine-tuning. Usa questa competenza quando l'utente menziona: architettura IA, progettazione di sistema con LLM, strutture architettoniche, architettura RAG, stack tecnologico per IA, database vettoriale, diagramma di architettura, componenti di sistema, embedding, recupero, pipeline di dati, MLOps, LLMOps, valutazione punti, RAG vs fine-tuning, progettazione di soluzioni intelligenza artificiale, spiegare repository, spiegare codice, analizzare progetto, cosa fa questo repository, come funziona questo sistema, spiegare questo progetto, o qualsiasi variazione di "quello componente necessario" o "spiegare come funziona questo". Attivarlo quando l'utente visualizza un codice, un file README, una struttura di file o menziona un repository GitHub per l'analisi. Anche quando vuoi disegnare un'architettura nuova.
