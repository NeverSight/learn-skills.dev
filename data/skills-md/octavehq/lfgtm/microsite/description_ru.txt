Персонализированный конструктор микросайтов ABM, который генерирует автономные целевые страницы HTML с использованием интеллекта GTM. Используйте, когда пользователь говорит «микросайт для [компании]», «персонализированная целевая страница», «страница ABM» или запрашивает персонализированную веб-страницу для целевой учетной записи.
