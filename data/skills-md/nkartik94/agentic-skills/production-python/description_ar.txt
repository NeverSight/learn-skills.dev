اصطلاحات ترميز Python على مستوى الإنتاج للوحدات والفئات والوظائف ونماذج Pydantic ونماذج SQLAlchemy والاختبارات. استخدم هذه المهارة عند كتابة ملفات Python جديدة، أو تعديل كود Python الحالي، أو إنشاء نماذج بيانات، أو إعداد هيكل المشروع، أو مراجعة كود Python للحصول على أفضل الممارسات. يتم تشغيله في أي مهمة تتضمن ملفات مصدر Python (.py)، أو ملفات الاختبار، أو دفاتر الملاحظات، أو دعامات المشروع.
