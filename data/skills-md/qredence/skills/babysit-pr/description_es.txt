Cuida una solicitud de extracción de GitHub sondeando continuamente las comprobaciones de CI, los comentarios de revisión y el estado de fusionabilidad hasta que el PR esté listo para fusionarse o cerrarse. Diagnostica fallas, reintenta fallas irregulares hasta 3 veces, corrige automáticamente problemas relacionados con sucursales y se detiene solo cuando se requiere ayuda del usuario. Úselo cuando se le solicite monitorear un PR, observar CI, manejar comentarios de revisión o realizar un seguimiento de fallas en un PR abierto.
