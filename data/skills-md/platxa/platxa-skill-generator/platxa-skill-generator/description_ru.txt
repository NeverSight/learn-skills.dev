Автономный создатель навыков для Claude Code CLI. Использует многоэтапный организованный рабочий процесс с субагентами инструмента задач для исследования предметной области, проектирования архитектуры, создания контента и проверки качества. Создает готовые к использованию навыки в соответствии со спецификацией «Навыки агентов» Anthropic.
