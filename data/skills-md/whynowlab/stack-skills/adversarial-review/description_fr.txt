Tests de résistance de Devil's Advocate pour le code, l'architecture, les relations publiques et les décisions. Fait apparaître les défauts cachés grâce à une analyse contradictoire structurée avec une profondeur métacognitive. À utiliser pour des examens à enjeux élevés, des choix de tests de résistance ou lorsque l'utilisateur souhaite que des problèmes soient détectés délibérément. PAS pour la révision de routine du code (utilisez Engineering:code-review).
