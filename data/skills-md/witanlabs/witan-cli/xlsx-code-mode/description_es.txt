Utilice esta habilidad cada vez que necesite leer, explorar, comprender o modificar un archivo de Excel (.xlsx, .xlsm). No puede leer archivos .xlsx con cat, head o herramientas normales de lectura de archivos; esta es la única forma de inspeccionarlos. Se activa cuando usted o el usuario necesitan abrir, mirar o explorar un libro de trabajo; averigüe qué hojas tiene o dónde se encuentran datos específicos; leer celdas, filas, columnas o rangos; buscar valores, etiquetas o patrones; rastrear dependencias de fórmulas o comprender cómo se calcula una celda; ejecutar escenarios hipotéticos cambiando las entradas y leyendo las salidas recalculadas; o editar celdas, filas, columnas y hojas. Se activa cuando el usuario hace referencia a un archivo de hoja de cálculo por nombre o ruta, incluso de manera casual (por ejemplo, "verificar el xlsx", "qué hay en report.xlsx"), y también cuando necesita inspeccionar un libro usted mismo como parte de una tarea más grande. La herramienta ejecuta JavaScript en un espacio aislado en el lado del servidor del libro de trabajo a través de `witan xlsx exec`".
