Utilizza questa competenza ogni volta che è necessario leggere, esplorare, comprendere o modificare un file Excel (.xlsx, .xlsm). Non puoi leggere file .xlsx con cat, head o normali strumenti di lettura file: questo è l'unico modo per esaminarli. Si attiva quando tu o l'utente dovete aprire, guardare o esplorare una cartella di lavoro; scoprire quali fogli ha o dove risiedono dati specifici; leggere celle, righe, colonne o intervalli; cercare valori, etichette o modelli; tracciare le dipendenze delle formule o comprendere come viene calcolata una cella; eseguire scenari ipotetici modificando gli input e leggendo gli output ricalcolati; o modificare celle, righe, colonne e fogli. Si attiva quando l'utente fa riferimento a un file di foglio di calcolo per nome o percorso, anche casualmente (ad esempio "controlla xlsx", "cosa c'è in report.xlsx"), e anche quando è necessario ispezionare personalmente una cartella di lavoro come parte di un'attività più ampia. Lo strumento esegue JavaScript in modalità sandbox sul lato server della cartella di lavoro tramite `witan xlsx exec`."
