Excel ファイル (.xlsx、.xlsm) を読み取り、探索、理解、または変更する必要がある場合には、このスキルをいつでも使用してください。 cat、head、または通常のファイル読み取りツールでは .xlsx ファイルを読み取ることはできません。これがファイルを検査する唯一の方法です。あなたまたはユーザーがワークブックを開いたり、閲覧したり、探索したりする必要があるときにトリガーします。どのようなシートがあるのか​​、または特定のデータがどこに存在するのかを調べます。セル、行、列、または範囲を読み取ります。値、ラベル、またはパターンを検索します。数式の依存関係を追跡したり、セルの計算方法を理解したりできます。入力を変更し、再計算された出力を読み取ることで、仮定のシナリオを実行します。またはセル、行、列、シートを編集します。ユーザーがスプレッドシート ファイルを名前またはパスで参照したとき (たとえば、「xlsx を確認する」、「report.xlsx の内容を確認する」) など、カジュアルな場合や、より大きなタスクの一部としてワークブックを自分で検査する必要があるときにもトリガーされます。このツールは、「witan xlsx exec」を介して、ワークブックのサーバー側に対してサンドボックス化された JavaScript を実行します。
