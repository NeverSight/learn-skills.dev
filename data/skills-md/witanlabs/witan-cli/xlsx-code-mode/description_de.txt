Nutzen Sie diese Fähigkeit immer dann, wenn eine Excel-Datei (.xlsx, .xlsm) gelesen, untersucht, verstanden oder geändert werden muss. Sie können .xlsx-Dateien nicht mit cat, head oder normalen Dateilesetools lesen – dies ist die einzige Möglichkeit, sie zu überprüfen. Wird ausgelöst, wenn Sie oder der Benutzer eine Arbeitsmappe öffnen, ansehen oder erkunden müssen; Finden Sie heraus, welche Blätter es hat oder wo bestimmte Daten gespeichert sind; Zellen, Zeilen, Spalten oder Bereiche lesen; nach Werten, Bezeichnungen oder Mustern suchen; Formelabhängigkeiten nachverfolgen oder verstehen, wie eine Zelle berechnet wird; Führen Sie Was-wäre-wenn-Szenarien durch, indem Sie Eingaben ändern und neu berechnete Ausgaben lesen. oder bearbeiten Sie Zellen, Zeilen, Spalten und Blätter. Wird ausgelöst, wenn der Benutzer anhand des Namens oder Pfads auf eine Tabellenkalkulationsdatei verweist – auch beiläufig (z. B. „Überprüfen Sie die XLSX-Datei“, „Was ist in report.xlsx“) – und auch, wenn Sie eine Arbeitsmappe im Rahmen einer größeren Aufgabe selbst überprüfen müssen. Das Tool führt über „witan xlsx exec“ Sandbox-JavaScript auf der Arbeitsmappenserverseite aus.
