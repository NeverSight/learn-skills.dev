Excel 파일(.xlsx, .xlsm)을 읽고, 탐색하고, 이해하거나 수정해야 할 때마다 이 기술을 사용하세요. cat, head 또는 일반 파일 읽기 도구로는 .xlsx 파일을 읽을 수 없습니다. 이것이 파일을 검사하는 유일한 방법입니다. 귀하 또는 사용자가 통합 문서를 열거나 보거나 탐색해야 할 때 트리거됩니다. 어떤 시트가 있는지 또는 특정 데이터가 어디에 있는지 알아보세요. 셀, 행, 열 또는 범위를 읽습니다. 값, 레이블 또는 패턴을 검색합니다. 수식 종속성을 추적하거나 셀 계산 방법을 이해합니다. 입력을 변경하고 다시 계산된 출력을 읽어 가정 시나리오를 실행합니다. 또는 셀, 행, 열, 시트를 편집할 수 있습니다. 사용자가 이름이나 경로로 스프레드시트 파일을 참조할 때(예: 'xlsx 확인', 'report.xlsx 내용' 등), 대규모 작업의 일부로 통합 문서를 직접 검사해야 할 때에도 트리거됩니다. 이 도구는 `witan xlsx exec`를 통해 통합 문서 서버 측에 대해 샌드박스 JavaScript를 실행합니다."
