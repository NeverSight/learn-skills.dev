استخدم هذه المهارة في أي وقت يلزم فيه قراءة ملف Excel (.xlsx، .xlsm) أو استكشافه أو فهمه أو تعديله. لا يمكنك قراءة ملفات .xlsx باستخدام أدوات قراءة الملفات cat أو head أو العادية — فهذه هي الطريقة الوحيدة لفحصها. يتم تشغيله عندما تحتاج أنت أو المستخدم إلى فتح مصنف أو النظر إليه أو استكشافه؛ تعرف على الأوراق التي تحتوي عليها أو مكان وجود بيانات محددة؛ قراءة الخلايا أو الصفوف أو الأعمدة أو النطاقات؛ البحث عن القيم أو التسميات أو الأنماط؛ تتبع تبعيات الصيغة أو فهم كيفية حساب الخلية؛ تشغيل سيناريوهات "ماذا لو" عن طريق تغيير المدخلات وقراءة المخرجات المعاد حسابها؛ أو تحرير الخلايا والصفوف والأعمدة والأوراق. يتم تشغيله عندما يشير المستخدم إلى ملف جدول بيانات حسب الاسم أو المسار - حتى بشكل عرضي (على سبيل المثال، "التحقق من xlsx"، "ما هو موجود في التقرير.xlsx") - وأيضًا عندما تحتاج إلى فحص مصنف بنفسك كجزء من مهمة أكبر. تقوم الأداة بتشغيل جافا سكريبت في وضع الحماية على جانب خادم المصنف عبر `witan xlsx exec`."
