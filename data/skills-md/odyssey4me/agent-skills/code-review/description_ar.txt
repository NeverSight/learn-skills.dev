قم بمراجعة تغييرات PRs وMRs وGerrit مع التركيز على الأمان وقابلية الصيانة والملاءمة المعمارية. يستفيد من مهارات github أو gitlab أو gerrit بناءً على سياق المستودع. استخدمه عند مطالبتك بمراجعة الكود الخاص بي، أو التحقق من العلاقات العامة هذه، أو مراجعة طلب سحب، أو إلقاء نظرة على طلب دمج، أو مراجعة مجموعة تصحيحات، أو تقديم تعليقات حول مراجعة الكود.
