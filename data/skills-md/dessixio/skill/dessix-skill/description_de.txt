Greifen Sie auf einen lokalen Dessix-Desktop-Arbeitsbereich zu, indem Sie die Electron MCP-Brücke direkt von Node.js aus (Socket/Pipe) aufrufen, ohne MCP stdio JSON-RPC zu verwenden. Wird verwendet, wenn ein Agent Dessix-Tools programmgesteuert lesen oder aufrufen, lokale Automatisierungen erstellen oder Inhalte von Aktions-/Szenenaufforderungen aus Dessix-Blöcken abrufen muss.
