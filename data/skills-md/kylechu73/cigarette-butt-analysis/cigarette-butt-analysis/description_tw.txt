靜態值雪茄煙蒂定量分析技能（v1.8）。  
建立在三大支柱（資產緩衝、低營運維護、資產變現邏輯）和用於系統篩選的 22 點事實查核系統之上。  

主要更新：
1.新增了事實查核第22項「資產結構健康測試」（如果非流動金融資產>30%，則執行核心T級安全邊際；>50%則觸發徹底拒絕）。  
2. 新增「實現-路徑完整性測試」（如果三個路徑都不合格，評級上限鎖定為C）。  
3. 增加了最低持有報酬檢查（股息殖利率低於無風險利率會觸發警告風險）。  
4.保留v1.7規則「最新資料優先」、「多期同時計算」、「股息削減高基數調整」。  

使用案例：當使用者上傳財務資訊（PDF 或文字表）以及股票代碼/名稱時觸發；涵蓋港股/A股/美股；自動獲取帶有來源 URL 的最新市場數據；輸出完整的 Markdown 報告：執行摘要、商業模式分析、管理和治理審查、三支柱評估、子類型（A 高止/B 套利/C 停損事件驅動）、事實查證、行動方案/止損
