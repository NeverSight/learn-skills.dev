Набор инструментов для количественного анализа окурков статических значений (v1.8).  
Систематическая оценка основана на трех столпах (подушка активов, низкий уровень операционного расхода, логика реализации активов) и наборе проверок фактов из 22 пунктов.  
Ключевые обновления:  
(1) Новая проверка фактов № 22 «Проверка состояния активов и структуры» (если внеоборотные финансовые активы> 30 %, обязателен базовый запас безопасности T-уровня; > 50 % вызывает полный отказ).  
(2) Новый «Тест на полноту пути реализации» (если ни один из трех путей не соответствует требованиям, потолок рейтинга фиксируется на уровне C).  
(3) Новая проверка минимальной доходности холдинга (дивидендная доходность ниже отметки «ПРЕДУПРЕЖДЕНИЕ-Риск» безрисковой ставки).  
(4) Сохраняются правила версии 1.7: «сначала самые последние данные», «одновременный расчет за несколько периодов», «коррекция высокой базы с сокращением дивидендов».  

Вариант использования: срабатывает, когда пользователь загружает финансовые отчеты (PDF или текстовую таблицу) плюс тикер/имя; охватывает акции Гонконга/А/США; автоматически извлекает последние рыночные данные с исходными URL-адресами; выводит полный отчет Markdown: краткое изложение, анализ бизнес-модели, обзор управления и управления, трехкомпонентную оценку, подтип (арбитраж с высоким делением / B / C, управляемый событиями), результаты проверки фактов, план действий (вход / тейк-профит / стоп-лосс / горизонт), оповещения о рисках.
