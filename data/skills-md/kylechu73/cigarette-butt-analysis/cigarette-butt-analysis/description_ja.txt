静的値シガーバット定量分析ツールキット (v1.8)。  
3 つの柱 (資産クッション、低運用コスト、資産実現ロジック) と 22 項目のファクトチェック検証スイートに基づく体系的な分析。  
コアアップデート:  
(1) ファクトチェック項目 22 を追加:「資産構造の健全性テスト」(非流動金融資産が 30% を超える場合、コア T 層安全マージンを計算します。50% を超えると自動拒否権が発動されます)。  
(2) 「実現パス完全性テスト」を追加しました (3 つのパスのいずれも適格でない場合、C 層での評価上限がロックされます)。  
(3) 最小保有リターンチェックを追加しました (リスクフリーレートを下回る配当利回りは警告リスクを引き起こします)。  
(4) v1.7のルール「最新データ優先」「複数期間同時計算」「減配高ベース補正」を踏襲。  

ユースケース: ユーザーが財務諸表 (PDF またはテキスト表) とティッカー/名前をアップロードするとトリガーされます。香港/オーストラリア/米国株をサポート。ログに記録されたソース URL を含む最新の市場データの必須のライブフェッチ。完全なマークダウンレポートを出力します: エグゼクティブサマリー、ビジネスモデル分析、経営およびガバナンス分析、3 本柱評価、サブタイプ (A 高配当 / B アービトラージ / C イベントドリブン)、ファクトチェック検証、アクションプラン (エントリー / テイクプロフィット / ストップロス / リターン期間の計算)、リスク警告。
