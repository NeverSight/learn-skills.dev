سير عمل مكدس الواجهة الأمامية لـ Next.js App Router: مكونات الخادم أولاً، خادم Prisma فقط (وقت تشغيل العقدة)، Tailwind + shadcn/ui، TanStack Query، React Hook Form + Zod، واصطلاحات العمل ضمن الواجهة الأمامية/. يُستخدم عند تنفيذ أو إعادة هيكلة واجهة المستخدم أو المسارات أو جلب البيانات أو الطفرات أو النماذج في هذه المكدس.
