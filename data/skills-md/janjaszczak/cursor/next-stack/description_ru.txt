Рабочий процесс стека внешнего интерфейса Next.js App Router: сначала компоненты сервера, только сервер Prisma (среда выполнения узла), Tailwind + shadcn/ui, TanStack Query, React Hook Form + Zod и соглашения для работы под интерфейсом/. Используйте при реализации или рефакторинге пользовательского интерфейса, маршрутов, выборки данных, мутаций или форм в этом стеке.
