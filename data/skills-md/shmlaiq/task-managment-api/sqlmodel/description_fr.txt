SQLModel - Pydantic + SQLAlchemy combinés. Cette compétence doit être utilisée lors de la création d'applications FastAPI avec base de données, de la création de modèles ORM, de la validation des données avec persistance de la base de données ou de tout projet Python nécessitant à la fois une validation et un ORM. Déclencheurs sur « créer un modèle de base de données », « ajouter SQLModel », « base de données avec FastAPI », « modèle ORM » ou tout développement lié à SQLModel.
