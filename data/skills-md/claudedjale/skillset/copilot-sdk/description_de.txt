Erstellen Sie mit dem Copilot SDK Anwendungen, die von GitHub Copilot unterstützt werden. Verwenden Sie diese Option, wenn Sie programmatische Integrationen mit Copilot über Node.js/TypeScript, Python, Go oder .NET erstellen. Behandelt Sitzungsverwaltung, benutzerdefinierte Tools, Streaming, Hooks, MCP-Server, BYOK-Anbieter, Sitzungspersistenz, benutzerdefinierte Agenten, Fähigkeiten und Bereitstellungsmuster. Erfordert die Installation der GitHub Copilot-CLI und ein GitHub Copilot-Abonnement (es sei denn, Sie verwenden BYOK).
