Crea applicazioni basate su GitHub Copilot utilizzando Copilot SDK. Da utilizzare durante la creazione di integrazioni programmatiche con Copilot su Node.js/TypeScript, Python, Go o .NET. Copre gestione delle sessioni, strumenti personalizzati, streaming, hook, server MCP, provider BYOK, persistenza della sessione, agenti personalizzati, competenze e modelli di distribuzione. Richiede l'installazione della CLI GitHub Copilot e un abbonamento a GitHub Copilot (a meno che non si utilizzi BYOK).
