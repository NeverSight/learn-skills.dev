أنشئ تطبيقات مدعومة بواسطة GitHub Copilot باستخدام Copilot SDK. يُستخدم عند إنشاء عمليات تكامل برمجية مع Copilot عبر Node.js/TypeScript أو Python أو Go أو .NET. يغطي إدارة الجلسة، والأدوات المخصصة، والتدفق، والخطافات، وخوادم MCP، وموفري BYOK، واستمرارية الجلسة، والوكلاء المخصصين، والمهارات، وأنماط النشر. يتطلب تثبيت GitHub Copilot CLI واشتراك GitHub Copilot (ما لم تستخدم BYOK).
