Cree aplicaciones impulsadas por GitHub Copilot utilizando el SDK de Copilot. Úselo al crear integraciones programáticas con Copilot en Node.js/TypeScript, Python, Go o .NET. Cubre administración de sesiones, herramientas personalizadas, transmisión, enlaces, servidores MCP, proveedores BYOK, persistencia de sesiones, agentes personalizados, habilidades y patrones de implementación. Requiere la CLI de GitHub Copilot instalada y una suscripción a GitHub Copilot (a menos que use BYOK).
