Создавайте приложения на базе GitHub Copilot, используя Copilot SDK. Используйте при создании программной интеграции с Copilot через Node.js/TypeScript, Python, Go или .NET. Охватывает управление сеансами, специальные инструменты, потоковую передачу, перехватчики, серверы MCP, поставщиков BYOK, сохранение сеансов, настраиваемых агентов, навыки и шаблоны развертывания. Требуется установленный GitHub Copilot CLI и подписка GitHub Copilot (если не используется BYOK).
