Créez des applications optimisées par GitHub Copilot à l'aide du SDK Copilot. À utiliser lors de la création d'intégrations programmatiques avec Copilot sur Node.js/TypeScript, Python, Go ou .NET. Couvre la gestion des sessions, les outils personnalisés, le streaming, les hooks, les serveurs MCP, les fournisseurs BYOK, la persistance des sessions, les agents personnalisés, les compétences et les modèles de déploiement. Nécessite l'installation de GitHub Copilot CLI et un abonnement GitHub Copilot (sauf si vous utilisez BYOK).
