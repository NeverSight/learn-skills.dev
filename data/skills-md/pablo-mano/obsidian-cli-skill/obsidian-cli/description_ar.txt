استخدم هذه المهارة عندما يريد المستخدم أن يتفاعل Claude مباشرة مع قبو Obsidian الخاص به - قراءة ملاحظة أو ملاحظة يومية، أو كتابة محتوى أو إلحاقه، أو البحث في محتويات القبو، أو عد الملاحظات أو إدراجها، أو إدارة المهام، أو نقل الملفات أو إعادة تسميتها، أو العثور على الملاحظات المعزولة أو الروابط المعطلة. بدون هذه المهارة، لن يكون لدى Claude أي طريقة للوصول إلى بيانات المخزن أو تنفيذ عمليات المخزن. تعامل مع أي طلب يتضمن "اذهب إلى خزنتي وافعل X" كمحفز - يطلب المستخدم من كلود التصرف، وليس الشرح. قم أيضًا بتشغيل أتمتة القبو أو البرمجة النصية لـ CLI أو سير العمل المستند إلى cron والذي يتضمن Obsidian، أو إدارة سجل المزامنة، أو الاستعلام عن القواعد، أو استعادة إصدارات الملفات عبر السجل، أو إدارة الإشارات المرجعية، أو تشغيل JavaScript مقابل Obsidian API. انتقل إلى الأسئلة المفاهيمية البحتة: كيفية عمل واجهة المستخدم الرسومية الخاصة بـ Obsidian، أو التنقل في قوائم الإعدادات، أو تثبيت السمات أو المكونات الإضافية عبر واجهة المستخدم، أو تعارضات مزامنة iCloud/الطرف الثالث، أو بناء جملة استعلام Dataview العام، أو اختصارات لوحة المفاتيح، أو تحليل ملفات vault باستخدام نصوص برمجية خارجية - أي شيء يحتاج فيه المستخدم إلى شرح بدلاً من قيام Claude بعملية قبو.
