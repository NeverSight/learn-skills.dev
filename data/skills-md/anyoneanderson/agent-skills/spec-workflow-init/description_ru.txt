Генератор рабочего процесса разработки. Создайте файл Issue-to-pr-workflow.md для вашего проекта.

Создает рабочий процесс разработки конкретного проекта посредством интерактивного диалога,
охватывающую стратегию филиала, контроль качества, стиль разработки и дополнительные определения субагентов.

Английские триггеры: «Создать рабочий процесс», «Создать рабочий процесс разработки», «Настроить поток от проблемы к PR».
日本語トリガー: 「ワークフローを生成」「開発フローを作成」「Выпуск для PRフローを設定」
