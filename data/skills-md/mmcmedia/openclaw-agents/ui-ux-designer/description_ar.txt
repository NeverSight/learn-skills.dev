تصميم وبناء واجهات المستخدم ولوحات المعلومات وتطبيقات الويب مع التركيز على تجربة المستخدم والتسلسل الهرمي المرئي والتصميم التفاعلي. يُستخدم عندما تحتاج شركة McKinzie إلى إنشاء أو إعادة تصميم لوحات المعلومات أو الصفحات المقصودة أو الأدوات أو أي واجهة مرئية. يتعامل مع استراتيجية التصميم والتنفيذ (HTML/CSS/JS).
