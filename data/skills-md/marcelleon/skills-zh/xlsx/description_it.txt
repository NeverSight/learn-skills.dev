Utilizzare questa capacità quando l'attività dell'utente è incentrata su un file di foglio di calcolo come input o output principale. Copre la lettura, la pulizia, la riparazione, la modellazione, il calcolo delle formule, la formattazione, la creazione di grafici, l'aggiornamento dei modelli e la conversione del formato per i file .xlsx/.xlsm/.csv/.tsv. Attivalo ogni volta che l'utente menziona il percorso di un file di foglio di calcolo e richiede l'elaborazione o la produzione di una tabella. Non attivare se il risultato finale principale non è un foglio di lavoro (ad esempio, pipeline Word/HTML/database/integrazione API Fogli Google).
