Esta habilidad debe usarse siempre que la tarea de un usuario se centre en un archivo de hoja de cálculo como entrada o salida principal. Cubre lectura, limpieza, reparación, modelado, cálculo de fórmulas, formato, gráficos, actualizaciones de plantillas y conversión de formato para archivos .xlsx/.xlsm/.csv/.tsv. Actívelo cada vez que el usuario mencione la ruta de un archivo de hoja de cálculo y solicite el procesamiento o la generación de una hoja de cálculo. No activar si el entregable principal no es una hoja de cálculo (por ejemplo, integración de Word/HTML/canal de base de datos/API de Google Sheets).
