Направляйте пользователей через структурированный рабочий процесс для совместного редактирования документов. Используйте это, когда пользователь хочет написать документы, предложения, технические спецификации, записи решений или аналогичный структурированный контент. Этот рабочий процесс помогает пользователям эффективно передавать контекст, уточнять контент посредством итерации и проверять, подходит ли документ читателям. Запускайте его всякий раз, когда пользователь упоминает написание документа, создание предложения, составление спецификации или аналогичные задачи по документированию.
