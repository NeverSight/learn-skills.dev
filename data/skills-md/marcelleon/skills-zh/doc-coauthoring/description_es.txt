Guíe a los usuarios a través de un flujo de trabajo estructurado para ser coautores de documentos. Úselo cuando el usuario quiera escribir un documento, propuesta, especificación técnica, registro de decisiones o contenido estructurado similar. Este flujo de trabajo ayuda a los usuarios a transmitir contexto de manera eficiente, refinar el contenido mediante iteración y verificar que el documento funcione para los lectores. Actívelo cada vez que el usuario mencione escribir un documento, crear una propuesta, redactar una especificación o tareas de documentación similares.
