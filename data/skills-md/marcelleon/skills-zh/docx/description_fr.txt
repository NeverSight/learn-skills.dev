Cette compétence doit être utilisée chaque fois qu'un utilisateur doit créer, lire, modifier ou convertir des documents Word (.docx). Les scénarios de déclenchement incluent : les mentions de Word/docx, la sortie de rapport/mémo/lettre formelle, la table des matières et les en-têtes/pieds de page, les commentaires ou le mode de révision, le remplacement du contenu du document, l'insertion ou le remplacement d'images, l'extraction de contenu structuré à partir de docx, etc. Si le livrable cible est un fichier .docx, cette compétence doit être déclenchée préférentiellement. Ne pas utiliser pour les fichiers PDF, les feuilles de calcul, l'API Google Docs ou les tâches de programmation non liées.
