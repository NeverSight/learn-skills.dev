This skill must be used whenever a user needs to create, read, edit, or convert Word documents (.docx). Triggering scenarios include: mentions of Word/docx, report/memo/formal letter output, table of contents and headers/footers, comments or revision mode, replacing document content, inserting or replacing images, extracting structured content from docx, etc. If the target deliverable is a .docx file, this skill should be triggered preferentially. Do not use for PDFs, spreadsheets, Google Docs API, or unrelated programming tasks.
