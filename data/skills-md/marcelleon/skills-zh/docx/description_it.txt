Questa competenza deve essere utilizzata ogni volta che un utente deve creare, leggere, modificare o convertire documenti Word (.docx). Gli scenari di attivazione includono: menzioni di Word/docx, output di report/memo/lettere formali, sommario e intestazioni/piè di pagina, modalità commenti o revisione, sostituzione del contenuto del documento, inserimento o sostituzione di immagini, estrazione di contenuto strutturato da docx, ecc. Se il risultato finale è un file .docx, questa abilità dovrebbe essere attivata preferibilmente. Non utilizzare per PDF, fogli di calcolo, API Google Docs o attività di programmazione non correlate.
