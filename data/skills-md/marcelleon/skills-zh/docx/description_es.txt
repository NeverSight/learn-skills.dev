Esta habilidad debe usarse siempre que un usuario necesite crear, leer, editar o convertir documentos de Word (.docx). Los escenarios desencadenantes incluyen: menciones de Word/docx, producción de informes/memorandos/cartas formales, tablas de contenido y encabezados/pies de página, comentarios o modo de revisión, reemplazo del contenido del documento, inserción o reemplazo de imágenes, extracción de contenido estructurado de docx, etc. Si el entregable es un .docx, se debe priorizar esta habilidad. No lo utilice para archivos PDF, hojas de cálculo, API de Google Docs ni tareas de programación no relacionadas.
