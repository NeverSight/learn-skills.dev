Siempre que la tarea implique entrada o salida de PDF, se debe utilizar esta habilidad. Incluye: leer/extraer texto y tablas, fusionar/dividir, rotar páginas, agregar marcas de agua, crear nuevos archivos PDF, completar formularios, cifrar/descifrar, extraer imágenes, OCR para archivos escaneados, etc. Debe activarse cada vez que el usuario mencione un nombre de archivo `.pdf` o solicite la entrega en formato PDF.
