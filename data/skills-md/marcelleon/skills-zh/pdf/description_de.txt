Immer wenn eine Aufgabe die Eingabe oder Ausgabe von PDFs beinhaltet, sollte dieser Fähigkeit Vorrang eingeräumt werden. Es umfasst: Lesen/Extrahieren von Text und Tabellen, Zusammenführen/Teilen, Drehen von Seiten, Hinzufügen von Wasserzeichen, Erstellen neuer PDFs, Ausfüllen von Formularen, Ver-/Entschlüsselung, Extrahieren von Bildern, OCR für gescannte Dokumente usw. Es sollte immer dann ausgelöst werden, wenn der Benutzer einen „.pdf“-Dateinamen erwähnt oder eine PDF-Lieferung anfordert.
