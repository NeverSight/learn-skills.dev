Planifique nuevos productos o características de ingeniería en trabajos listos para la implementación para agentes de codificación. Utilícelo cuando el usuario solicite definir el alcance de una nueva función, dividir el trabajo en fases o tickets, evaluar compensaciones, definir criterios de aceptación, secuenciar la entrega o preparar un plan de ejecución que los agentes puedan implementar de forma segura.
