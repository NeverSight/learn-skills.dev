Créez des documents structurés à partir de conversations, de résumés ou de contenus dans des formats ouverts (markdown, PDF, texte). À utiliser lorsque l'utilisateur demande la création de documents, la génération de rapports, l'exportation de contenu, les résumés de conversations ou la documentation structurée. Les déclencheurs incluent « créer un document », « créer un rapport », « résumer cette conversation », « exporter au format PDF/markdown » ou toute demande de formalisation du contenu dans un document. Fonctionne indépendamment ou s'intègre aux compétences d'assistant de conception pour un rendu visuel soigné.
