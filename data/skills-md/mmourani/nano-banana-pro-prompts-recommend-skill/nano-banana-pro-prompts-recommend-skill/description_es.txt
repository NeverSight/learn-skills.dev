Recomiende mensajes adecuados entre más de 10,000 mensajes de generación de imágenes de Nano Banana Pro según las necesidades del usuario.
Optimizado para Nano Banana Pro (Gemini), pero las indicaciones también funcionan con Nano Banana 2, Seedream 5.0,
GPT Image 1.5, Midjourney, DALL-E, Flux, Stable Diffusion y cualquier modelo de IA de conversión de texto a imagen.

Utilice esta habilidad cuando los usuarios quieran:
- Genera imágenes con IA (cualquier modelo: Nano Banana Pro, Gemini, GPT Image, Seedream, etc.)
- Encuentre indicaciones y plantillas de indicaciones comprobadas para la generación de imágenes de IA
- Obtenga recomendaciones rápidas para casos de uso específicos (retratos, productos, redes sociales, carteles, etc.)
- Crear ilustraciones para artículos, vídeos, podcasts o contenido de marketing.
- Explore una biblioteca de mensajes seleccionada con imágenes de muestra
- Traducir y comprender técnicas rápidas.

También disponible: habilidad "ai-image-prompts": una versión independiente del modelo de esta biblioteca para la generación universal de imágenes.
