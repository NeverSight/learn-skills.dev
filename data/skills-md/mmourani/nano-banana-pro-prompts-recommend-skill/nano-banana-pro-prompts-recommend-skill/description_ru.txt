Рекомендуйте подходящие подсказки из более чем 10 000 подсказок для создания изображений Nano Banana Pro в зависимости от потребностей пользователя.
Оптимизировано для Nano Banana Pro (Gemini), но подсказки также работают с Nano Banana 2, Seedream 5.0,
GPT Image 1.5, Midjourney, DALL-E, Flux, Stable Diffusion и любая модель искусственного интеллекта для преобразования текста в изображение.

Используйте этот навык, когда пользователи хотят:
- Генерируйте изображения с помощью ИИ (любая модель — Nano Banana Pro, Gemini, GPT Image, Seedream и т. д.)
- Найдите проверенные подсказки для создания изображений с помощью ИИ и шаблоны подсказок.
- Получайте быстрые рекомендации для конкретных случаев использования (портреты, продукты, социальные сети, плакаты и т. д.)
- Создавайте иллюстрации для статей, видео, подкастов или маркетингового контента.
- Просмотрите тщательно подобранную библиотеку подсказок с образцами изображений.
- Перевести и понять методы подсказок

Также доступен: навык «ai-image-prompts» — независимая от модели версия этой библиотеки для универсальной генерации изображений.
