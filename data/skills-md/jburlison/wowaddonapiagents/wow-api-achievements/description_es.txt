Referencia completa para las API de criterios de logros, estadísticas y logros minoristas de WoW. Cubre C_AchievementInfo, funciones de logros globales (GetAchievementInfo, GetAchievementCriteriaInfo, GetStatistic, GetComparisonStatistic), categorías de logros, logros de gremio, enlaces de logros, logros rastreados, notificaciones de brindis y eventos relacionados con logros. Úselo cuando trabaje con seguimiento de logros, visualización de logros, estadísticas, criterios de logros, logros de gremio o detección de finalización de logros.
