Полный справочник по API достижений, статистики и критериев достижений WoW. Охватывает C_AchievementInfo, глобальные функции достижений (GetAchievementInfo, GetAchievementCriteriaInfo, GetStatistic, GetComparisonStatistic), категории достижений, достижения гильдии, ссылки на достижения, отслеживаемые достижения, всплывающие уведомления и события, связанные с достижениями. Используйте при работе с отслеживанием достижений, отображением достижений, статистикой, критериями достижений, достижениями гильдии или обнаружением выполнения достижений.
