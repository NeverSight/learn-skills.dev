مرجع كامل لواجهات برمجة تطبيقات WoW لإنجاز البيع بالتجزئة والإحصائيات ومعايير الإنجاز. يغطي C_AchievementInfo، ووظائف الإنجاز العالمية (GetAchievementInfo، وGetAchievementCriteriaInfo، وGetStatistic، وGetComparisonStatistic)، وفئات الإنجاز، وإنجازات النقابة، وروابط الإنجاز، والإنجازات المتعقبة، وإشعارات الخبز المحمص، والأحداث المتعلقة بالإنجاز. يُستخدم عند العمل مع تتبع الإنجاز أو عرض الإنجاز أو الإحصائيات أو معايير الإنجاز أو إنجازات الجماعة أو اكتشاف اكتمال الإنجاز.
