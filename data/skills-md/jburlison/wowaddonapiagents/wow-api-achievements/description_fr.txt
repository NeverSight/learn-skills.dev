Référence complète pour les API de réussite, de statistiques et de critères de réussite de WoW Retail. Couvre C_AchievementInfo, les fonctions de réussite globales (GetAchievementInfo, GetAchievementCriteriaInfo, GetStatistic, GetComparisonStatistic), les catégories de réussite, les réalisations de guilde, les liens de réussite, les réalisations suivies, les notifications toast et les événements liés aux réalisations. À utiliser lorsque vous travaillez avec le suivi des réalisations, l'affichage des réalisations, les statistiques, les critères de réussite, les réalisations de guilde ou la détection d'achèvement des réalisations.
