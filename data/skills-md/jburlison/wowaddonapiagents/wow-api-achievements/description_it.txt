Riferimento completo per le API WoW Retail Achievement, Statistics e Achievement Criteria. Copre C_AchievementInfo, funzioni di conseguimento globali (GetAchievementInfo, GetAchievementCriteriaInfo, GetStatistic, GetComparisonStatistic), categorie di risultati, risultati di gilda, collegamenti a risultati, risultati monitorati, notifiche di avviso popup ed eventi relativi ai risultati. Da utilizzare quando si lavora con il monitoraggio degli obiettivi, la visualizzazione degli obiettivi, le statistiche, i criteri degli obiettivi, gli obiettivi della gilda o il rilevamento del completamento degli obiettivi.
