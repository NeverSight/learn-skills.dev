الذاكرة المستمرة للوكيل أولاً لتخزين الملاحظات التي يمكن قراءتها بواسطة الإنسان واسترجاعها والحفاظ عليها عبر المحادثات. يجب على الوكيل اتخاذ القرارات المهمة والعمل المنجز بشكل استباقي؛ يمكن للمستخدمين أيضًا أن يطلبوا تذكر العمل السابق أو حفظه أو تدوينه أو استرجاعه أو تذكيره أو مراجعته (على سبيل المثال، "تذكر هذا"، "احفظ هذا"، "لاحظ هذا"، "ذكّرني بـ..."، "ماذا قررنا بشأن..."، "راجع ملاحظاتك"، "تنظيف الذكريات").
