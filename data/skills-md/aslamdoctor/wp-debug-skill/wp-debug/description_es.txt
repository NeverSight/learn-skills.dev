Inicie y detenga el modo de depuración de WordPress desde dentro de una carpeta de tema o complemento. Habilita de forma segura WP_DEBUG, WP_DEBUG_LOG, SCRIPT_DEBUG y SAVEQUERIES en wp-config.php, sigue debug.log en tiempo real y restaura limpiamente la configuración original cuando termina. Úselo cuando el usuario diga "iniciar la depuración", "depurar WordPress", "habilitar el modo de depuración", "detener la depuración" o "desactivar la depuración". Diseñado para directorios de trabajo dentro de wp-content/themes/<theme> o wp-content/plugins/<plugin>.
