Démarrez et arrêtez le mode de débogage de WordPress depuis un dossier de thème ou de plugin. Active en toute sécurité WP_DEBUG, WP_DEBUG_LOG, SCRIPT_DEBUG et SAVEQUERIES dans wp-config.php, termine debug.log en temps réel et restaure proprement la configuration d'origine une fois terminé. À utiliser lorsque l'utilisateur dit « démarrer le débogage », « déboguer WordPress », « activer le mode débogage », « arrêter le débogage » ou « désactiver le débogage ». Conçu pour les répertoires de travail dans wp-content/themes/<theme> ou wp-content/plugins/<plugin>.
