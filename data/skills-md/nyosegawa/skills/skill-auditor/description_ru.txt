Анализирует стенограммы сеансов Claude Code для оценки состояния портфеля навыков — ошибок маршрутизации, конкуренции внимания между описаниями и пробелов в охвате. Создает интерактивный отчет в формате HTML с картами здоровья для каждого навыка, матрицей конкуренции, анализом бюджета внимания и практическими исправлениями. В отличие от инструмента создания навыков, который оптимизирует отдельные навыки изолированно, аудит навыков оптимизирует портфолио как систему, выявляя кражу внимания между навыками и каскадные риски. Используйте, когда пользователь говорит: «Проверить мои навыки», «Аудит навыков», «Запустить аудит навыков», «Анализ маршрутизации навыков», «Проверить конкуренцию навыков», «Состояние портфолио», «スキル監査», «スキルの精度を分析», «スキルルーティング分析».
