Verwenden Sie diese Fähigkeit, wenn Sie mit dem Station-Hintergrundjob-Framework erstellen. Dazu gehören das Erstellen von Signalen (Hintergrundjobs), das Definieren von Broadcasts (DAG-Workflows), das Konfigurieren von Adaptern (SQLite, PostgreSQL, MySQL, Redis), das Einrichten von Runnern, das Schreiben von Abonnenten und das Konfigurieren des Station-Dashboards. Station ist ein TypeScript-First-Framework für typsichere Hintergrundjobs mit Zod-Validierung.
