Destillieren Sie Wissen aus beliebigen Quellen – Blogbeiträge, Artikel, Dokumentation, GitHub-Repos, Videotranskripte, Bücher, Papiere – in eine gut strukturierte Agentenkompetenz. Verwenden Sie diese Option, wenn der Benutzer eine URL, einen Artikel, ein Repo oder einen Wissensbestand teilt und möchte, dass dieser in eine wiederverwendbare Fähigkeit umgewandelt wird. Auslöser bei: „Daraus eine Fertigkeit erstellen“, „Dies in eine Fertigkeit destillieren“, „Eine Fertigkeit aus diesem Artikel erstellen“, „Dieses Repo in eine Fertigkeit umwandeln“, „Muster daraus extrahieren“, „In eine Fertigkeit umwandeln“. Dieser Skill ergänzt den Skill „Skill-Creator“ – der Skill-Creator kümmert sich um die Mechanik (Frontmatter, Verpackung, Init-Skripte), dieser Skill kümmert sich um den Destillationsprozess.
