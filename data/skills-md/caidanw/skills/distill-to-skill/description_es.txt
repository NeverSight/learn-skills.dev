Obtenga conocimiento de cualquier fuente (publicaciones de blogs, artículos, documentación, repositorios de GitHub, transcripciones de videos, libros, artículos) en una habilidad de agente bien estructurada. Úselo cuando el usuario comparte una URL, un artículo, un repositorio o un conjunto de conocimientos y quiere que se convierta en una habilidad reutilizable. Activadores de: "crear una habilidad a partir de esto", "destilar esto en una habilidad", "crear una habilidad a partir de este artículo", "convertir este repositorio en una habilidad", "extraer patrones de", "convertir en una habilidad". Esta habilidad complementa la habilidad "creador de habilidades": el creador de habilidades maneja la mecánica (frontmatter, empaquetado, scripts de inicio), esta habilidad maneja el proceso de destilación.
