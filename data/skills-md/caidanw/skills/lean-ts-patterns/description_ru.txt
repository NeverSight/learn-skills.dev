Шаблоны для создания легковесных инструментов и библиотек TypeScript без каких-либо зависимостей. Используйте при создании новых инструментов, библиотек или утилит CLI с нуля. Используйте при рефакторинге существующих проектов TypeScript для удаления ненужных зависимостей. Используйте при написании оболочек выборки, анализаторов CLI, средств ведения журнала, слияния конфигураций, строковых утилит или любого кода инфраструктуры, который должен быть компактным и автономным. Запускается по: «создать CLI», «написать средство ведения журнала», «получить оболочку», «удалить зависимости», «облегченный», «нулевая глубина», «встроенная утилита», «рефакторинг для упрощения».
