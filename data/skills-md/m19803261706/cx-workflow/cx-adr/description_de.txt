CX-Workflow – Architekturentscheidungsdatensatz. Wird ausgelöst, wenn Benutzer „Architekturentscheidung“, „Technologieauswahl“, „ADR“, „Warum X wählen“ oder „Architekturlösungsvergleich“ erwähnen (normalerweise automatisch von cx-design aufgerufen). Zeichnet Architekturentscheidungsdatensätze auf und speichert sie lokal unter .claude/cx/features/{dev_id}-{feature}/adr.md. Kann von cx-design manuell oder automatisch ausgelöst werden, wenn eine Architekturentscheidung erkannt wird.
