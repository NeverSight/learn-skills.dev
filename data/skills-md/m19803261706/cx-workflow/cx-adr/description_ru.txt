Рабочий процесс CX — запись решения по архитектуре. Срабатывает, когда пользователи упоминают «решение по архитектуре», «выбор технологии», «ADR», «почему выбирают X» или «сравнение архитектур» (обычно вызывается автоматически cx-design). Записывает записи решений по архитектуре и сохраняет их в локальном файле .claude/cx/features/{dev_id}-{feature}/adr.md. Может запускаться вручную или автоматически с помощью cx-design при обнаружении архитектурного решения.
