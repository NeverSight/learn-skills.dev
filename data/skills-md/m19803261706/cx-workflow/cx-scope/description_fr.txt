Flux de travail CX – Discussion sur le plan de projet. Déclenché lorsque les utilisateurs mentionnent « plan », « plan global », « portée du projet », « portée », « discussion du projet » ou « proposition de fonctionnalités ». Des conversations à plusieurs tours explorent le projet ou le plan de fonctionnalités, enregistrent les résultats dans le fichier local .claude/cx/features/{dev_id}-{feature}/scope.md, éventuellement synchronisent avec le problème GitHub (basé sur le mode config.github_sync).
