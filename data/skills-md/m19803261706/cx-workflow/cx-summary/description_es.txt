Flujo de trabajo CX: publicación resumida y circuito cerrado. Se activa de forma manual o automática cuando cx-exec se completa por completo. Genera un documento de resumen, sincronización de GitHub (basada en config.github_sync), actualiza de forma inteligente la especificación CLAUDE.md y activa una notificación de finalización. Solo se ejecuta cuando el usuario invoca explícitamente /cx-summary o cuando cx-exec está completamente completado. No disparar automáticamente.
