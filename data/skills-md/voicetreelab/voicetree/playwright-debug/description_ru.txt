Этот навык следует использовать, когда пользователь просит «отладить электронное приложение», «подключить драматурга к VoiceTree», «сделать снимки экрана работающего приложения», «взаимодействовать с живым пользовательским интерфейсом», «проверить работающее приложение» или «тестировать элементы пользовательского интерфейса в реальном времени». Содержит пошаговые инструкции по подключению Playwright MCP к работающему приложению Electron для отладки и автоматизации в реальном времени.
