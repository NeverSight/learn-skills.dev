Gestisci dotfile e segreti con chezmoi. Utilizzare per inizializzare chezmoi, aggiungere/modificare file di configurazione, creare modelli per configurazioni su più macchine, crittografare file sensibili, integrare gestori di password (1Password, Bitwarden, pass, ecc.), eseguire script durante l'applicazione e sincronizzare dotfile tra macchine. Trigger sulla gestione dei file dot, comandi chezmoi, modelli di configurazione o segreti nei file di configurazione.
