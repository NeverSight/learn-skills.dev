Richten Sie Conductor (conductor.build) für ein Repository ein. Erstellt „ductor.json“, richtet Skripte für .env-Symlinking ein und konfiguriert Dev-Server-Ausführungsbefehle. Verwenden Sie diese Option, wenn der Benutzer „Conductor einrichten“, „Conductor konfigurieren“, „conductor.json hinzufügen“, „Conductor-Konfiguration erstellen“ auffordert oder Conductor-Arbeitsbereiche für ein neues oder vorhandenes Repository erwähnt.
