Configure Conductor (conductor.build) para un repositorio. Crea conductor.json, configura scripts para enlaces simbólicos .env y configura comandos de ejecución del servidor de desarrollo. Úselo cuando el usuario solicite "configurar Conductor", "configurar Conductor", "agregar conductor.json", "crear configuración de conductor" o mencione espacios de trabajo de Conductor para un repositorio nuevo o existente.
