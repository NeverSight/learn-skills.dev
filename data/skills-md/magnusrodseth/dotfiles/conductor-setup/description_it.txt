Configura Conductor (conductor.build) per un repository. Crea conduttore.json, imposta script per il collegamento simbolico .env e configura i comandi di esecuzione del server di sviluppo. Da utilizzare quando l'utente chiede di "impostare Conductor", "configurare Conductor", "aggiungere conduttore.json", "creare la configurazione di Conductor" o menziona le aree di lavoro di Conductor per un repository nuovo o esistente.
