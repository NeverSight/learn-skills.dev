Échafaudez un projet de robot SDK Microsoft Teams dans TypeScript qui intègre des agents dans Teams. Utilisez cette compétence chaque fois que l'utilisateur souhaite créer un bot Teams, échafauder un projet Teams, créer un agent Teams, intégrer un agent IA dans Microsoft Teams ou configurer un nouveau bot à l'aide du SDK Teams. Déclenchez également lorsque l'utilisateur mentionne « teams2 », « teams CLI », « tunnel de développement pour Teams » ou souhaite enregistrer un bot auprès d'Azure AD pour Teams. Même si l'utilisateur dit simplement quelque chose comme « Je veux créer un bot pour Teams » ou « Mettre en place un projet Teams », cette compétence doit être utilisée.
