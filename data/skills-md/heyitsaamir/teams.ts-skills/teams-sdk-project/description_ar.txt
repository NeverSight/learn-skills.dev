قم بدعم مشروع روبوت Microsoft Teams SDK في TypeScript الذي يجلب الوكلاء إلى Teams. استخدم هذه المهارة عندما يريد المستخدم إنشاء روبوت Teams، أو دعم مشروع Teams، أو إنشاء وكيل Teams، أو إحضار وكيل AI إلى Microsoft Teams، أو إعداد روبوت جديد باستخدام Teams SDK. يتم التشغيل أيضًا عندما يذكر المستخدم "teams2"، أو "teams CLI"، أو "dev Tunnel for Teams"، أو عندما يريد تسجيل روبوت في Azure AD for Teams. حتى لو قال المستخدم شيئًا مثل "أريد إنشاء روبوت لـ Teams" أو "إعداد مشروع Teams"، فيجب استخدام هذه المهارة.
