Erstellen Sie ein Gerüst für ein Microsoft Teams SDK-Bot-Projekt in TypeScript, das Agenten in Teams integriert. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer einen Teams-Bot erstellen, ein Teams-Projekt aufbauen, einen Teams-Agenten erstellen, einen KI-Agenten in Microsoft Teams integrieren oder einen neuen Bot mithilfe des Teams SDK einrichten möchte. Wird auch ausgelöst, wenn der Benutzer „teams2“, „teams CLI“, „dev tunnel for Teams“ erwähnt oder einen Bot bei Azure AD für Teams registrieren möchte. Auch wenn der Benutzer nur etwas sagt wie „Ich möchte einen Bot für Teams erstellen“ oder „ein Teams-Projekt einrichten“, sollte dieser Skill genutzt werden.
