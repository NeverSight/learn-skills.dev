Создайте проект бота Microsoft Teams SDK на TypeScript, который переносит агентов в Teams. Используйте этот навык всякий раз, когда пользователь хочет создать бота Teams, подготовить проект Teams, создать агент Teams, перенести агент AI в Microsoft Teams или настроить нового бота с помощью Teams SDK. Также срабатывает, когда пользователь упоминает «teams2», «CLI Teams», «туннель разработки для Teams» или хочет зарегистрировать бота в Azure AD для Teams. Даже если пользователь просто говорит что-то вроде «Я хочу создать бота для Teams» или «настроить проект Teams», этот навык следует использовать.
