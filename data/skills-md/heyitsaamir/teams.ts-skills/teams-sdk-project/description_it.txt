Impalcatura di un progetto bot Microsoft Teams SDK in TypeScript che porta gli agenti in Teams. Utilizza questa competenza ogni volta che l'utente desidera creare un bot di Teams, supportare un progetto di Teams, creare un agente di Teams, portare un agente AI in Microsoft Teams o configurare un nuovo bot utilizzando l'SDK di Teams. Si attiva anche quando l'utente menziona "teams2", "teams CLI", "dev tunnel for Teams" o vuole registrare un bot con Azure AD for Teams. Anche se l'utente dice semplicemente qualcosa come "Voglio creare un bot per Teams" o "impostare un progetto Teams", questa abilità dovrebbe essere utilizzata.
