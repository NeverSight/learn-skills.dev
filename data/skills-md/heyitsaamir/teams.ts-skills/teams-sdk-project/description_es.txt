Cree un proyecto de bot del SDK de Microsoft Teams en TypeScript que incorpore agentes a Teams. Utilice esta habilidad siempre que el usuario quiera crear un bot de Teams, desarrollar un proyecto de Teams, crear un agente de Teams, incorporar un agente de IA a Microsoft Teams o configurar un nuevo bot mediante el SDK de Teams. También se activa cuando el usuario menciona "teams2", "teams CLI", "túnel de desarrollo para Teams" o desea registrar un bot con Azure AD para Teams. Incluso si el usuario simplemente dice algo como "Quiero crear un bot para Teams" o "configurar un proyecto de Teams", se debe utilizar esta habilidad.
