Управляйте устройствами умного дома Xiaomi/Mijia, напрямую вызывая команды CLI mijiaAPI через облачный API Xiaomi. Поддерживает обнаружение устройств, управление включением/выключением, регулировку яркости, настройку цвета и многое другое. Используйте этот навык, когда пользователям нужно управлять интеллектуальными устройствами Xiaomi (такими как настольные лампы, лампочки, розетки и т. д.), получать списки устройств или проверять состояние устройств.
