Administre y controle los dispositivos domésticos inteligentes Xiaomi/Mijia invocando directamente los comandos CLI de mijiaAPI a través de la API en la nube de Xiaomi. Admite descubrimiento de dispositivos, control de encendido/apagado, ajuste de brillo, configuración de color y más. Utilice esta habilidad cuando los usuarios necesiten controlar dispositivos inteligentes Xiaomi (como lámparas de escritorio, bombillas, enchufes, etc.), recuperar listas de dispositivos o verificar el estado del dispositivo.
