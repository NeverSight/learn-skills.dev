Convierta cualquier servidor MCP o especificación OpenAPI en una CLI. Utilice esta habilidad cuando el usuario quiera interactuar con un servidor MCP o OpenAPI/REST API a través de la línea de comandos, descubrir herramientas/puntos finales disponibles, llamar a operaciones de API o generar una nueva habilidad desde una API. Los desencadenantes incluyen "mcp2cli", "llamar a este servidor MCP", "usar esta API", "enumerar herramientas de", "crear una habilidad para esta API" o cualquier tarea que implique la invocación de herramientas MCP o llamadas a puntos finales de OpenAPI sin escribir código.
