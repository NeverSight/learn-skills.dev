あらゆる MCP サーバーまたは OpenAPI 仕様を CLI に変換します。このスキルは、ユーザーがコマンド ライン経由で MCP サーバーまたは OpenAPI/REST API と対話したり、利用可能なツール/エンドポイントを検出したり、API オペレーションを呼び出したり、API から新しいスキルを生成したりする場合に使用します。トリガーには、「mcp2cli」、「この MCP サーバーを呼び出す」、「この API を使用する」、「ツールをリストする」、「この API のスキルを作成する」、またはコードを記述せずに MCP ツールの呼び出しや OpenAPI エンドポイント呼び出しを伴うタスクが含まれます。
