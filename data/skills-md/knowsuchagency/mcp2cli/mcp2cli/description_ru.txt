Превратите любой сервер MCP или спецификацию OpenAPI в интерфейс командной строки. Используйте этот навык, когда пользователь хочет взаимодействовать с сервером MCP или API OpenAPI/REST через командную строку, обнаруживать доступные инструменты/конечные точки, вызывать операции API или генерировать новый навык из API. Триггеры включают «mcp2cli», «вызвать этот сервер MCP», «использовать этот API», «составить список инструментов», «создать навык для этого API» или любую задачу, включающую вызов инструмента MCP или вызовы конечной точки OpenAPI без написания кода.
