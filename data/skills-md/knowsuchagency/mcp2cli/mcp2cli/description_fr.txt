Transformez n'importe quel serveur MCP ou spécification OpenAPI en CLI. Utilisez cette compétence lorsque l'utilisateur souhaite interagir avec un serveur MCP ou une API OpenAPI/REST via une ligne de commande, découvrir les outils/points de terminaison disponibles, appeler des opérations d'API ou générer une nouvelle compétence à partir d'une API. Les déclencheurs incluent « mcp2cli », « appeler ce serveur MCP », « utiliser cette API », « lister les outils de », « créer une compétence pour cette API » ou toute tâche impliquant l'invocation d'un outil MCP ou des appels de point de terminaison OpenAPI sans écrire de code.
