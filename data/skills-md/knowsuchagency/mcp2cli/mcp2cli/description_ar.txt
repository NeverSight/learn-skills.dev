قم بتحويل أي خادم MCP أو مواصفات OpenAPI إلى واجهة سطر الأوامر (CLI). استخدم هذه المهارة عندما يريد المستخدم التفاعل مع خادم MCP أو OpenAPI/REST API عبر سطر الأوامر، أو اكتشاف الأدوات/نقاط النهاية المتاحة، أو استدعاء عمليات واجهة برمجة التطبيقات، أو إنشاء مهارة جديدة من واجهة برمجة التطبيقات. تتضمن المشغلات "mcp2cli"، أو "استدعاء خادم MCP هذا"، أو "استخدام واجهة برمجة التطبيقات هذه"، أو "إدراج الأدوات من"، أو "إنشاء مهارة لواجهة برمجة التطبيقات هذه"، أو أي مهمة تتضمن استدعاء أداة MCP أو استدعاءات نقطة نهاية OpenAPI دون كتابة تعليمات برمجية.
