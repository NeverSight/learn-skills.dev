Verwandeln Sie jeden MCP-Server oder jede OpenAPI-Spezifikation in eine CLI. Verwenden Sie diesen Skill, wenn der Benutzer über die Befehlszeile mit einem MCP-Server oder einer OpenAPI/REST-API interagieren, verfügbare Tools/Endpunkte entdecken, API-Vorgänge aufrufen oder einen neuen Skill aus einer API generieren möchte. Zu den Auslösern gehören „mcp2cli“, „diesen MCP-Server aufrufen“, „diese API verwenden“, „Tools auflisten von“, „einen Skill für diese API erstellen“ oder jede Aufgabe, die MCP-Tool-Aufrufe oder OpenAPI-Endpunktaufrufe beinhaltet, ohne Code zu schreiben.
