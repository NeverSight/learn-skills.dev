Trasforma qualsiasi server MCP o specifica OpenAPI in una CLI. Utilizza questa competenza quando l'utente desidera interagire con un server MCP o un'API OpenAPI/REST tramite riga di comando, scoprire strumenti/endpoint disponibili, chiamare operazioni API o generare una nuova competenza da un'API. I trigger includono "mcp2cli", "chiama questo server MCP", "usa questa API", "elenca gli strumenti da", "crea una competenza per questa API" o qualsiasi attività che coinvolga l'invocazione dello strumento MCP o le chiamate all'endpoint OpenAPI senza scrivere codice.
