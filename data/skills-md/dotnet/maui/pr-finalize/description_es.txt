Úselo cuando se le solicite "finalizar PR", "verificar la descripción de PR", "revisar el mensaje de confirmación", antes de fusionar cualquier PR o cuando la implementación de PR cambió durante la revisión. NO lo use para extraer lecciones (use learn-from-pr), escribir pruebas (use write-tests-agent) o investigar fallas de compilación (use pr-build-status).
