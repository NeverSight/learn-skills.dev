Finalizza qualsiasi PR per l'unione verificando l'implementazione della corrispondenza titolo/descrizione ED eseguendo la revisione del codice per le migliori pratiche. Da utilizzare quando viene richiesto di "finalizzare il PR", "controllare la descrizione del PR", "revisionare il messaggio di commit", prima di unire qualsiasi PR o quando l'implementazione del PR è cambiata durante la revisione. NON utilizzare per estrarre lezioni (utilizzare learn-from-pr), scrivere test (utilizzare write-tests-agent) o indagare sugli errori di compilazione (utilizzare pr-build-status).
