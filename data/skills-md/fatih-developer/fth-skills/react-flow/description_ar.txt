تحليل وإصلاح وترحيل ودعم قواعد التعليمات البرمجية @xyflow/react. يُستخدم عندما يطلب المستخدمون تصحيح سلوك React Flow، أو إصلاح أسلاك حالة العقدة/الحافة، أو تحسين أمان النوع أو الأداء، أو ترقية واجهات برمجة تطبيقات React Flow القديمة، أو الحفاظ على توافق الرسم البياني المستمر، أو إنشاء مشغل React Flow كامل من البداية.
