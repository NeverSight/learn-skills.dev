Analyse en profondeur un projet logiciel existant et génère 3 à 4 rapports complets dans le dossier docs/analyze/. Déclenchez lorsque des expressions telles que « analyser le projet », « effectuer une analyse du code », « examen du projet », « rapport d'architecture », « dette technique », « évaluation du projet », « analyse de la base de code » se produisent. Fonctionne lorsqu'un répertoire de projet ou une URL de dépôt est fourni. Si des points de terminaison de l'API sont détectés, il génère automatiquement un 4ème rapport.
