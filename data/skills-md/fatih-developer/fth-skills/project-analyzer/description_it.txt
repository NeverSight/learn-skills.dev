Analizza in modo approfondito un progetto software esistente e genera 3-4 report completi nella cartella docs/analyze/. Si attiva quando si verificano frasi come "analizzare il progetto", "eseguire analisi del codice", "revisione del progetto", "rapporto sull'architettura", "debito tecnico", "valutazione del progetto", "analisi della base del codice". Funziona quando viene fornita una directory di progetto o un URL di repository. Se vengono rilevati endpoint API, genera automaticamente un quarto report.
