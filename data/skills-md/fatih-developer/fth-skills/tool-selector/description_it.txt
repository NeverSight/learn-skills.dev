Prima di eseguire un'attività, analizzare il set di strumenti disponibili (ricerca Web, esecuzione di codice, lettura/scrittura di file, chiamate API, query di database, strumenti di memoria), selezionare gli strumenti ottimali, pianificare l'ordine di esecuzione ed evitare chiamate di strumenti non necessarie. Si attiva su attività in più fasi, "quale strumento dovrei utilizzare", "pianifica gli strumenti" o ogni volta che potrebbero essere applicati più strumenti.
