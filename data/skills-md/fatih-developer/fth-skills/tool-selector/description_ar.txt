قبل تنفيذ مهمة، قم بتحليل مجموعة الأدوات المتاحة (بحث الويب، وتنفيذ التعليمات البرمجية، وقراءة/كتابة الملفات، واستدعاءات واجهة برمجة التطبيقات، واستعلامات قاعدة البيانات، وأدوات الذاكرة)، وحدد الأدوات المثالية، وتخطيط ترتيب التنفيذ، ومنع استدعاءات الأدوات غير الضرورية. يتم تشغيله عند تنفيذ مهام متعددة الخطوات، أو "ما هي الأداة التي يجب أن أستخدمها"، أو "تخطيط الأدوات"، أو عندما يمكن تطبيق أدوات متعددة.
