عند فشل إحدى الخطوات أثناء مهمة وكيل، قم بتصنيف الخطأ (عابر أو تكوين أو منطق أو دائم)، وقم بتطبيق استراتيجية الاسترداد الصحيحة، وقم بالتصعيد إلى المستخدم عند استنفاد جميع الاستراتيجيات. يتم تشغيله عند ظهور رسائل الخطأ، أو الاستثناءات، أو عمليات التتبع، أو "فشل"، أو "لا يعمل"، أو "إعادة المحاولة"، أو عند فشل خطوتين متتاليتين.
