Analizza i dati finanziari dai file Excel/PDF caricati e genera report interattivi con grafici di tendenza sparkline. Supporta l'output in PDF, DOCX, Markdown e HTML.
Da utilizzare quando: (1) l'utente carica Excel/CSV/PDF con dati finanziari, (2) l'utente richiede analisi finanziarie o report aziendali, (3) l'utente desidera report visivi dai rendiconti finanziari, (4) l'utente menziona ticker azionario + analisi finanziaria, (5) l'utente condivide un collegamento a un foglio/documento Feishu con dati finanziari.
