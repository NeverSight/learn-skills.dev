Анализируйте финансовые данные из загруженных файлов Excel/PDF и создавайте интерактивные отчеты со спарклайн-диаграммами тенденций. Поддерживает вывод в PDF, DOCX, Markdown и HTML.
Используйте, когда: (1) Пользователь загружает Excel/CSV/PDF с финансовыми данными, (2) Пользователь запрашивает финансовый анализ или отчет компании, (3) Пользователь хочет визуальные отчеты из финансовой отчетности, (4) Пользователь упоминает биржевой тикер + финансовый анализ, (5) Пользователь делится ссылкой на лист/документ Feishu с финансовыми данными.
