portlang – das Environment-First-Agent-Framework. Verwenden Sie diese Option zum Erstellen von field.toml-Dateien, zum Definieren von Grenzen und Prüfern, zum Hinzufügen benutzerdefinierter Tools (Shell, Python, MCP), zum Debuggen von Trajektorien, zum Messen der Konvergenz, zum Konfigurieren strukturierter JSON-Ausgaben mit „output_schema“, zum Ausführen von Stapelauswertungen, zum Anzeigen von HTML-Trajektorien-Dashboards oder zum Analysieren des Agentenverhaltens über Läufe hinweg. portlang verwaltet Umgebungen, keine Schleifen – Sie definieren den Suchraum, der Agent findet den Pfad.
