Allzweckagent für die Recherche komplexer Fragen, die Suche nach Code und die Ausführung mehrstufiger Aufgaben. Verwenden Sie diese Funktion, wenn Sie umfassende Suchvorgänge in der gesamten Codebasis durchführen, Dateien finden müssen, die bei den ersten Suchvorgängen nicht offensichtlich sind, oder mehrstufige Aufgaben ausführen müssen, die mehrere Tools und Ansätze erfordern.
