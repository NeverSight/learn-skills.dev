استخدم هذه المهارة للعمل الجوي أو المناخي القائم على الملفات مع هيكل مشروع حقيقي بدلاً من إجابة لمرة واحدة. قم بتشغيله لـ NetCDF أو GRIB أو إعادة التحليل أو ERA5 أو CMIP أو WRF أو مخرجات النموذج أو EOF أو PCA أو علم المناخ أو الشذوذ أو الانحدار أو المركب أو أي تشخيصات جوية أخرى عندما تتضمن المهمة التخطيط أو التشخيص أو الإحصائيات أو إعادة الشبكة أو الحساب القابل لإعادة الاستخدام أو فصل الحساب مقابل قطعة الأرض أو أرقام درجة النشر أو مراجعة المخرجات الفعلية. استخدمه أيضًا لملفات أشكال الغلاف الجوي الموجودة أو تصحيحات الأشكال الصغيرة فقط داخل مشروع الغلاف الجوي أو المناخ الحالي عندما تحتاج التسميات أو المستويات أو أشرطة الألوان أو وسائل الإيضاح أو أحرف اللوحة أو التخطيط أو الصادرات إلى التثبيت دون تغيير الطريقة العلمية. لا تستخدمه للتنبؤات، أو ملخصات الأدبيات، أو الكتابة النقية، أو التخطيط العام خارج علوم الغلاف الجوي، أو عمل التطبيق أو واجهة المستخدم، أو النشر، أو التنزيلات البسيطة بخطوة واحدة.
