Cree, edite y mejore aplicaciones Retool utilizando ToolScript (formato RSX). Utilice esta habilidad siempre que el usuario mencione retool, toolscript, RSX, herramienta interna, panel de administración, aplicación de administración de datos, aplicación CRUD o quiera crear cualquier aplicación Retool, ya sea desde cero, modificando una existente o mejorándola con patrones de producción. También activadores para: remodelar importación, remodelar zip, remodelar tabla, remodelar modal, renovar formulario, renovar filtro, renovar panel, remodelar chat, crear aplicación interna, panel de administración, editor de datos.
