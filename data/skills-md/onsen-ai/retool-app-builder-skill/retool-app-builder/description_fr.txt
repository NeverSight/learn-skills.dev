Créez, modifiez et améliorez des applications Retool à l'aide de ToolScript (format RSX). Utilisez cette compétence chaque fois que l'utilisateur mentionne Retool, ToolScript, RSX, un outil interne, un panneau d'administration, une application de gestion de données, une application CRUD ou souhaite créer une application Retool, que ce soit à partir de zéro, en modifiant une application existante ou en l'améliorant avec des modèles de production. Déclenche également pour : l'importation de réoutil, le zip de réoutil, la table de réoutil, le modal de réoutil, le formulaire de réoutil, le filtre de réoutil, le tableau de bord de réoutil, le chat de réoutil, la création d'une application interne, le tableau de bord d'administration, l'éditeur de données.
