قم بتحليل المطالبات الأولية، وتحديد النوايا والفجوات، ومطابقة مكونات ECC (المهارات/الأوامر/الوكلاء/الخطافات)، وإخراج موجه محسّن جاهز لللصق. الدور الاستشاري فقط - لا ينفذ المهمة نفسها أبدًا. يتم تشغيله عندما: يقول المستخدم "تحسين المطالبة"، أو "تحسين المطالبة"، أو "كيفية كتابة مطالبة لـ"، أو "مساعدتي في المطالبة"، أو "إعادة كتابة هذه المطالبة"، أو يطلب صراحة تحسين جودة المطالبة. يتم تشغيلها أيضًا في المرادفات الصينية: "优化prompt"، "改进prompt"، "怎么写prompt"، "帮我优化这个指令". لا تقم بالتشغيل عندما: يريد المستخدم تنفيذ المهمة مباشرة، أو يقول "فقط افعلها" / "直接做". لا تقم بالتشغيل عندما يقول المستخدم "优化代码"، "优化性能"، "تحسين الأداء"، "تحسين هذا الرمز" - فهذه مهام إعادة هيكلة/أداء، وليست مهام تحسين سريعة.
