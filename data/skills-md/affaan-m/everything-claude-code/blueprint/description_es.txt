Convierta un objetivo de una sola línea en un plan de construcción paso a paso para proyectos de ingeniería de múltiples sesiones y múltiples agentes. Cada paso tiene un resumen de contexto autónomo para que un nuevo agente pueda ejecutarlo en frío. Incluye puerta de revisión adversaria, gráfico de dependencia, detección de pasos paralelos, catálogo antipatrón y protocolo de mutación del plan. DISPARAR cuando: el usuario solicita un plan, plano o hoja de ruta para una tarea compleja de relaciones públicas múltiples, o describe un trabajo que necesita múltiples sesiones. NO DISPARAR cuando: la tarea se puede completar en un solo PR o en menos de 3 llamadas a herramientas, o el usuario dice "simplemente hazlo".
