Générez des hooks React personnalisés prêts pour la production pour les modèles d'interface utilisateur courants. À utiliser lorsque l'utilisateur a besoin de hooks pour la gestion de l'état, les interactions DOM, les opérations asynchrones ou la conception réactive dans les applications React. Couvre : useToggle, useDebounce, useThrottle, useFetch, useAsync, useLocalStorage, useInterval, useKeyPress, useHover, useMediaQuery, useOnClickOutside, useOnScreen, useScrollToEnd, useScrolledPastThreshold, useWindowSize.
