Automatisation du navigateur anti-détection pour les agents IA. À utiliser lorsque l'utilisateur a besoin d'une navigation Web furtive, d'un grattage indétectable, d'une usurpation d'empreintes digitales, d'une rotation de proxy ou d'une automatisation du navigateur axée sur la confidentialité. Les déclencheurs incluent « grattage furtif », « anti-détection », « contournement des empreintes digitales », « camofox », « camoufox », « navigateur indétectable », « évasion de robots » ou toute tâche de navigateur nécessitant l'évasion des systèmes de détection de robots.
