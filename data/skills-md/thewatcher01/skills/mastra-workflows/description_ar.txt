أنشئ سير عمل DAG باستخدام Mastra 1.x (@mastra/core/workflows). يُستخدم عند تنفيذ تنسيق سير العمل باستخدام الرسوم البيانية غير الحلقية الموجهة، أو تسلسل الخطوات، أو التنفيذ المتوازي، أو التفرع، أو الحلقات، أو معالجة الأخطاء، أو التعليق/الاستئناف، أو الدفق. يتم تشغيل المهام التي تتضمن createStep، أو createWorkflow، أو .then()، أو .parallel()، أو .branch()، أو .foreach()، أو .dountil()، أو .dowhile()، أو .map()، أو إدارة حالة سير العمل، أو تكوين خطوة Mastra.
