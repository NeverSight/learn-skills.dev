Neuroergonomía autónoma + auditoría de UI a11y y habilidad de iteración. Combina los estándares WCAG 2.2 AA, principios de ciencia cognitiva (Fitts Law, CLT, Gestalt) y un bucle de captura de pantalla de Chrome DevTools MCP para una entrega lista para producción. Activadores de neuro ergo, a11y, auditoría de accesibilidad, carga cognitiva, revisión de UX, WCAG, ergonomía, accesibilidad.
