Автономный нейроэргономический + a11y аудит пользовательского интерфейса и навыки итерации. Сочетает в себе стандарты WCAG 2.2 AA, принципы когнитивной науки (закон Фиттса, CLT, гештальт) и цикл создания снимков экрана MCP Chrome DevTools для готовой к работе поставки. Триггеры по нейроэргономике, a11y, аудиту доступности, когнитивной нагрузке, обзору UX, WCAG, эргономии, доступности.
