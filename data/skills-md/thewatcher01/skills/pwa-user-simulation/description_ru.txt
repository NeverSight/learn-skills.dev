Моделируйте реальные действия пользователя через PWA, используя MCP для автоматизации браузера (Playwright MCP + Chrome DevTools MCP). Сочетает в себе SXO (оптимизацию поискового опыта), нейроэргономическую эвристику, проверки WCAG 2.2 AA и анализ основных веб-показателей с помощью структурированных снимков доступности и визуальных снимков экрана. Запускается по: симуляции пользователя, пути пользователя, тесту PWA, аудиту SXO, пошаговому руководству UX, автоматизации браузера, исследованию e2e, симуляции пользователя, тестированию приложения, парковке как у пользователя, тестированию потока, проверке парковок.
