Simule viajes de usuarios reales a través de una PWA utilizando MCP de automatización del navegador (Playwright MCP + Chrome DevTools MCP). Combina SXO (Optimización de la experiencia de búsqueda), heurística neuroergonómica, comprobaciones WCAG 2.2 AA y análisis Core Web Vitals a través de instantáneas de accesibilidad estructuradas y capturas de pantalla visuales. Activadores de: simular usuario, recorrido del usuario, prueba de PWA, auditoría SXO, tutorial de UX, automatización del navegador, exploración e2e, simulación de usuario, probar la aplicación, explorar como usuario, probar el flujo, verificar el recorrido.
