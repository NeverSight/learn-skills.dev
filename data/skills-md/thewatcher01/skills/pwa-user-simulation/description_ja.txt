ブラウザ自動化 MCP (Playwright MCP + Chrome DevTools MCP) を使用して、PWA を介した実際のユーザー ジャーニーをシミュレートします。 SXO (検索エクスペリエンス最適化)、神経人間工学ヒューリスティック、WCAG 2.2 AA チェック、および構造化されたアクセシビリティ スナップショットとビジュアル スクリーンショットによる Core Web Vitals 分析を組み合わせます。トリガー: ユーザーのシミュレーション、ユーザー ジャーニー、PWA テスト、SXO 監査、UX ウォークスルー、ブラウザー自動化、e2e 探索、ユーザー シミュレーション、アプリのテスト、実用化のパルクール、フローのテスター、パルクールの検証。
