Insegna la progettazione della tassonomia degli agenti, gli ecosistemi delle competenze, la stratificazione delle istruzioni, i grafici di trasferimento in base ad antropiche/competenze, OpenAI Agents SDK e gli standard degli agenti personalizzati GitHub Copilot (VS Code 1.106+).
