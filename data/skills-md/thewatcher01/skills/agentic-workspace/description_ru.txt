Настраивайте и обслуживайте агентские архитектуры SOTA в рабочих пространствах VS Code с несколькими проектами. Используйте при: настройке агентов, архитектуре агентов, многопроектной рабочей области, иерархии агентов, разработке контекста, проектировании подсказок, экосистеме навыков, AGENTS.md, инструкциях второго пилота, передаче обслуживания агентов, шаблоне оркестратора, прогрессивном раскрытии информации, структуре агента рабочей области. Обучает проектированию таксономии агентов, экосистемам навыков, многоуровневому распределению инструкций, графам передачи управления в соответствии с антропными данными/навыками, OpenAI Agents SDK и стандартам пользовательских агентов GitHub Copilot (VS Code 1.106+).
