قم بإنشاء الرسوم البيانية باستخدام تنسيق Draw.io XML. يُستخدم عندما يطلب المستخدم إنشاء رسوم تخطيطية، أو مخططات انسيابية، أو مخططات معمارية، أو بنية RTL الدقيقة، أو مخططات حالة FSM، أو تحلل DATAFLOW، أو طوبولوجيا النظام، أو الإطارات السلكية، أو أي رسومات مرئية. يولد ملفات XML `.drawio`. يُفضل على excalidraw لمخططات FPGA/HLS بسبب التنسيق الأبسط والنص الذي يتم توسيطه تلقائيًا وXML الموحد لجميع أنواع المخططات بما في ذلك RTL microarch.
