Создавайте диаграммы, используя XML-формат Draw.io. Используйте, когда пользователь запрашивает создание диаграмм, блок-схем, архитектурных диаграмм, микроархитектуры RTL, диаграмм состояний FSM, декомпозиции DATAFLOW, топологии системы, каркасов или любых визуальных рисунков. Генерирует XML-файлы `.drawio`. Предпочтителен для диаграмм FPGA/HLS по сравнению с excalidraw из-за более простого формата, автоцентрирования текста и единообразного XML для всех типов диаграмм, включая микроархи RTL.
