Cree diagramas utilizando el formato XML de Draw.io. Úselo cuando el usuario solicite crear diagramas, diagramas de flujo, diagramas de arquitectura, microarquitectura RTL, diagramas de estado FSM, descomposición de DATAFLOW, topología del sistema, estructuras alámbricas o cualquier dibujo visual. Genera archivos XML `.drawio`. Preferido sobre excalidraw para diagramas FPGA/HLS debido a su formato más simple, texto autocentrado y XML uniforme para todos los tipos de diagramas, incluido el microarco RTL.
