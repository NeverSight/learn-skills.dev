قم بإعداد حوكمة تصميم Fragments Cloud في مشروعك. يقوم بتثبيت @fragments-sdk/govern، ويرشد تكوين مفتاح واجهة برمجة التطبيقات، وينشئ تكوينًا للإدارة، ويقوم بتشغيل الفحص الأول، ويضيف CI/CD بشكل اختياري. يُستخدم عندما يريد المستخدم إضافة إدارة التصميم، أو الاتصال بـ Fragments Cloud، أو تشغيل عمليات فحص التصميم، أو إعداد إدارة Fragments في مشروعه.
