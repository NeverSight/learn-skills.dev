Richten Sie Fragments Cloud-Design-Governance in Ihrem Projekt ein. Installiert @fragments-sdk/govern, leitet die API-Schlüsselkonfiguration, erstellt eine Governance-Konfiguration, führt Ihre erste Prüfung durch und fügt optional CI/CD hinzu. Verwenden Sie diese Option, wenn der Benutzer Design-Governance hinzufügen, eine Verbindung zu Fragments Cloud herstellen, Designprüfungen durchführen oder Fragments-Governance in seinem Projekt einrichten möchte.
