Configurez la gouvernance de conception Fragments Cloud dans votre projet. Installe @fragments-sdk/govern, guide la configuration de la clé API, crée une configuration de gouvernance, exécute votre première vérification et ajoute éventuellement CI/CD. À utiliser lorsque l'utilisateur souhaite ajouter une gouvernance de conception, se connecter à Fragments Cloud, exécuter des vérifications de conception ou configurer la gouvernance Fragments dans son projet.
