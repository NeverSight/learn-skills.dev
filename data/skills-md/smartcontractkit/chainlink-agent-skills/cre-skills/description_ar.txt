تمكين المطورين من تعلم واستخدام Chainlink Runtime Environment (CRE) بسرعة من خلال الرجوع إلى مستندات CRE التي تمت تصفيتها. يتم التشغيل عندما يريد المستخدم الإعداد، أو إنشاء سير عمل CRE (في TypeScript أو Golang أو اللغات الأخرى المدعومة)، أو توجيه سير العمل، أو مساعدة CRE CLI و/أو SDK، أو مشورة عمليات وقت التشغيل، أو اختيار القدرة
