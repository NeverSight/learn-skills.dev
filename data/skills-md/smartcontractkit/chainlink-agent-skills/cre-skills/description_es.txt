Permita que los desarrolladores aprendan y utilicen Chainlink Runtime Environment (CRE) rápidamente haciendo referencia a documentos CRE filtrados. Se activa cuando el usuario desea incorporación, generación de flujo de trabajo CRE (en TypeScript o Golang u otros lenguajes compatibles), guía de flujo de trabajo, ayuda CRE CLI y/o SDK, asesoramiento sobre operaciones en tiempo de ejecución o selección de capacidades.
