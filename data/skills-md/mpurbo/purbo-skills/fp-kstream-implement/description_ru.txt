Реализация топологии Kafka Streams с использованием архитектуры функционального ядра/императивной оболочки. Используйте этот навык при реализации топологии Kafka Streams в коде. Триггеры: «реализовать эту топологию», «написать код KStream», «создать процессор», «локально настроить kafka», «написать тесты для топологии», «настройка докера Kafka», «реализация KStream», «код потокового процессора» или любой запрос на запись, тестирование или запуск кода приложения Kafka Streams. НЕ используйте для дизайна, мозгового штурма или архитектуры — используйте для них fp-kstream-design.
