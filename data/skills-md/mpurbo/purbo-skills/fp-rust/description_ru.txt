Функциональное программирование на Rust — аргументированные рекомендации по написанию идиоматического, ориентированного на FP кода Rust. Используйте этот навык при написании, проверке, рефакторинге или сборке кода Rust. Охватывает: архитектуру функционального ядра/императивной оболочки, шаблоны, ориентированные на неизменность, чистые функции, алгебраические типы данных, конвейерно-ориентированное программирование с итераторами и цепочками результатов, обработку ошибок как данных, отказ от зависимостей вместо внедрения зависимостей, проектирование на основе типов (анализ не проверяется, состояние типа) и параллелизм посредством передачи сообщений. Также используйте, когда пользователь спрашивает о структуре проекта Rust, шаблонах обработки ошибок Rust, конвейерах итераторов Rust, настройке Cargo.toml для нового проекта Rust или проверке кода Rust. Применяйте эти рекомендации, даже если пользователь явно не упоминает «функциональное программирование» — это стиль кодирования по умолчанию для всей работы на Rust.
