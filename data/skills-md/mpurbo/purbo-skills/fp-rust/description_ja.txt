Rust での関数型プログラミング — 慣用的な FP ファーストの Rust コードを記述するための独自のガイドライン。 Rust コードを作成、レビュー、リファクタリング、またはスキャフォールディングするときは常に、このスキルを使用してください。対象範囲: 関数コア/命令型シェル アーキテクチャ、不変性優先パターン、純粋関数、代数データ型、イテレータと結果チェーンを使用したパイプライン指向プログラミング、データとしてのエラー処理、依存関係注入による依存関係の拒否、型駆動設計 (解析は検証しない、typestate)、メッセージ パッシングによる同時実行性。また、ユーザーが Rust プロジェクトの構造、Rust エラー処理パターン、Rust イテレーター パイプライン、新しい Rust プロジェクトの Cargo.toml セットアップ、または Rust コード レビューについて質問するときにも使用します。ユーザーが「関数型プログラミング」について明示的に言及していない場合でも、これらのガイドラインを適用してください。これは、すべての Rust 作業のデフォルトのコーディング スタイルです。
