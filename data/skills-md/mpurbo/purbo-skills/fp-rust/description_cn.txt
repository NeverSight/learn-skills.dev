Rust 函数式编程 — 用于编写惯用的、函数优先的 Rust 代码的固执己见的指南。每当编写、审查、重构或搭建 Rust 代码时，请使用此技能。涵盖：功能核心/命令式 shell 架构、不变性优先模式、纯函数、代数数据类型、使用迭代器和结果链的面向管道编程、数据错误处理、依赖注入的依赖拒绝、类型驱动设计（解析不验证、类型状态）以及通过消息传递实现的并发。当用户询问 Rust 项目结构、Rust 错误处理模式、Rust 迭代器管道、新 Rust 项目的 Cargo.toml 设置或 Rust 代码审查时也可以使用。即使用户没有明确提及“函数式编程”，也要应用这些指南——这是所有 Rust 工作的默认编码风格。
