Rust의 함수형 프로그래밍 — 관용적인 FP 우선 Rust 코드 작성을 위한 독선적인 지침입니다. Rust 코드를 작성, 검토, 리팩토링 또는 스캐폴딩할 때마다 이 기술을 사용하세요. 다루는 내용: 기능적 코어/명령형 셸 아키텍처, 불변성 우선 패턴, 순수 함수, 대수 데이터 유형, 반복자와 결과 체인을 사용한 파이프라인 지향 프로그래밍, 데이터로 오류 처리, 종속성 주입에 대한 종속성 거부, 유형 기반 설계(구문 분석이 아닌 유효성 검사, 유형 상태) 및 메시지 전달을 통한 동시성. 또한 사용자가 Rust 프로젝트 구조, Rust 오류 처리 패턴, Rust 반복자 파이프라인, 새로운 Rust 프로젝트를 위한 Cargo.toml 설정 또는 Rust 코드 검토에 관해 질문할 때에도 사용하세요. 사용자가 "함수형 프로그래밍"을 명시적으로 언급하지 않더라도 이 지침을 적용하세요. 이는 모든 Rust 작업의 기본 코딩 스타일입니다.
