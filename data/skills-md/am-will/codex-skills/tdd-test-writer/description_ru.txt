Сначала пишет неудачные тесты для разработки через тестирование и передает строгий контракт на реализацию, который требует, чтобы агенты проходили эти тесты, не ослабляя их. Используйте, когда пользователи запрашивают рабочие процессы «сначала тестирование», КРАСНЫЕ/ЗЕЛЕНЫЕ циклы или задачи по контролю поведения с помощью автоматических тестов.
