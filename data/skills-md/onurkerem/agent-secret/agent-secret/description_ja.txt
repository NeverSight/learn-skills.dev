OS キーチェーンを使用した安全なシークレット管理。使用目的:
- .env ファイルにシークレットを設定/挿入する
- 値を公開せずに構成ステータスを確認する
- .env ファイルに保存されているシークレットまたはキーをリストします。

ユーザーが次の場合に自動的にトリガーされます。
- .env ファイルへのシークレット/キー/トークンの追加、設定、または注入について言及
- サービスの API キーについて言及 (Stripe、OpenAI、AWS、Supabase、Firebase など)
- 環境変数または .env ファイルを構成するように求められます
- 安全な秘密の保管について質問します。
- 資格情報、パスワード、またはトークンについての言及
