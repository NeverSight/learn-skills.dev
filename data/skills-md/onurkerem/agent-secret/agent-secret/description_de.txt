Sichere Geheimnisverwaltung mit dem Betriebssystem-Schlüsselbund. Verwendung für:
- Geheimnisse in .env-Dateien festlegen/einfügen
- Überprüfen Sie den Konfigurationsstatus, ohne Werte preiszugeben
- Listen Sie gespeicherte Geheimnisse oder Schlüssel in .env-Dateien auf

AUTOMATISCH AUSLÖSEN, wenn der Benutzer:
– Erwähnt das Hinzufügen, Festlegen oder Einfügen von Geheimnissen/Schlüsseln/Tokens in .env-Dateien
– Erwähnt API-Schlüssel für Dienste (Stripe, OpenAI, AWS, Supabase, Firebase usw.)
– Fordert zur Konfiguration von Umgebungsvariablen oder .env-Dateien auf
- Fragt nach sicherer Geheimspeicherung
– Erwähnt Anmeldeinformationen, Passwörter oder Token
