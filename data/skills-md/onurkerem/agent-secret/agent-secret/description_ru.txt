Безопасное управление секретами с помощью цепочки ключей ОС. Используйте для:
- Установить/внедрить секреты в файлы .env.
- Проверьте состояние конфигурации, не раскрывая значения.
- Список сохраненных секретов или ключей в файлах .env.

АВТОМАТИЧЕСКИ ЗАПУСКАЕТСЯ, когда пользователь:
- Упоминает добавление, настройку или внедрение секретов/ключей/токенов в файлы .env.
- Упоминает ключи API для сервисов (Stripe, OpenAI, AWS, Supabase, Firebase и т. д.).
- Просит настроить переменные среды или файлы .env.
- Спрашивает о безопасном секретном хранилище.
- Упоминает учетные данные, пароли или токены.
