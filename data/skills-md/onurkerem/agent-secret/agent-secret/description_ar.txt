إدارة سرية آمنة باستخدام سلسلة مفاتيح نظام التشغيل. استخدم لـ:
- تعيين/حقن الأسرار في ملفات .env
- التحقق من حالة التكوين دون تعريض القيم
- قائمة الأسرار أو المفاتيح المخزنة في ملفات .env

يتم التشغيل تلقائيًا عندما يقوم المستخدم بما يلي:
- يذكر إضافة أو إعداد أو إدخال الأسرار/المفاتيح/الرموز المميزة في ملفات .env
- يذكر مفاتيح API للخدمات (Stripe وOpenAI وAWS وSupabase وFirebase وما إلى ذلك)
- يطلب تكوين متغيرات البيئة أو ملفات .env
- يسأل عن التخزين السري الآمن
- يذكر بيانات الاعتماد أو كلمات المرور أو الرموز المميزة
