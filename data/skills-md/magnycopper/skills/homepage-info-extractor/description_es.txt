Extraiga con precisión datos específicos (por ejemplo, nombre, historial, información de contacto, personal) de sitios web oficiales institucionales o corporativos y genere un informe Excel estandarizado y aplanado. Adecuado para tareas profesionales de extracción de información de sitios web o de sitios oficiales que requieren una estricta fidelidad al texto fuente, fuentes XPath/localizador verificables y cumplimiento de una lógica de campo específica.
