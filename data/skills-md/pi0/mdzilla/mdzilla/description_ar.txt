تصفح الوثائق وابحث عنها وقم بتصديرها من أي مصدر باستخدام mdzilla CLI. يُستخدم عندما يحتاج Claude إلى: (1) قراءة الوثائق أو تصفحها لمكتبة أو إطار عمل أو مشروع، (2) جلب المستندات من مستودعات GitHub أو حزم npm أو مواقع الويب، (3) تصدير الوثائق إلى ملفات تخفيض السعر المسطحة، (4) البحث عن مراجع واجهة برمجة التطبيقات أو الأدلة أو أمثلة الاستخدام من مواقع المستندات. المشغلات: يطلب المستخدم "قراءة المستندات"، أو "التحقق من الوثائق"، أو "استعراض المستندات بحثًا عن X"، أو "جلب المستندات من"، أو "تصدير المستندات"، أو الإشارة إلى عنوان URL لموقع المستندات، أو مستندات حزمة npm، أو مستندات GitHub repo.
