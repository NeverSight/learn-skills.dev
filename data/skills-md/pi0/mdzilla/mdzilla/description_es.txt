Explore, busque y exporte documentación desde cualquier fuente utilizando la CLI de mdzilla. Úselo cuando Claude necesite: (1) leer o buscar documentación para una biblioteca, marco o proyecto, (2) obtener documentos de repositorios de GitHub, paquetes npm o sitios web, (3) exportar documentación a archivos planos de rebajas, (4) buscar referencias de API, guías o ejemplos de uso en sitios de documentos. Activadores: el usuario solicita "leer los documentos", "verificar la documentación", "buscar documentos para X", "obtener documentos de", "exportar documentos" o hace referencia a la URL de un sitio de documentos, documentos de paquete npm o documentos de repositorio de GitHub.
