mdzilla CLI を使用して、任意のソースからドキュメントを参照、検索、エクスポートします。クロードが次のことを行う必要がある場合に使用します: (1) ライブラリ、フレームワーク、またはプロジェクトのドキュメントを読むまたは参照する、(2) GitHub リポジトリ、npm パッケージ、または Web サイトからドキュメントを取得する、(3) ドキュメントをフラット マークダウン ファイルにエクスポートする、(4) ドキュメント サイトから API リファレンス、ガイド、または使用例を検索する。トリガー: ユーザーが「ドキュメントを読む」、「ドキュメントをチェックする」、「X のドキュメントを参照する」、「ドキュメントを取得する」、「ドキュメントをエクスポートする」を要求するか、ドキュメント サイト URL、npm パッケージ ドキュメント、または GitHub リポジトリ ドキュメントを参照します。
