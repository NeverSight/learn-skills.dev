Sfoglia, cerca ed esporta la documentazione da qualsiasi fonte utilizzando la CLI di mdzilla. Da utilizzare quando Claude ha bisogno di: (1) Leggere o sfogliare la documentazione per una libreria, un framework o un progetto, (2) Recuperare documenti da repository GitHub, pacchetti npm o siti Web, (3) Esportare documentazione in file markdown flat, (4) Cercare riferimenti API, guide o esempi di utilizzo da siti di documentazione. Trigger: l'utente chiede di "leggere i documenti", "controllare la documentazione", "sfogliare documenti per X", "recuperare documenti da", "esportare documenti" o fare riferimento all'URL di un sito di documenti, ai documenti del pacchetto npm o ai documenti del repository GitHub.
