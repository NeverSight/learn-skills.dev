Parcourez, recherchez et exportez de la documentation à partir de n'importe quelle source à l'aide de la CLI mdzilla. À utiliser lorsque Claude a besoin de : (1) Lire ou parcourir la documentation d'une bibliothèque, d'un framework ou d'un projet, (2) Récupérer des documents à partir de dépôts GitHub, de packages npm ou de sites Web, (3) Exporter la documentation vers des fichiers de démarques plates, (4) Rechercher des références d'API, des guides ou des exemples d'utilisation à partir de sites de documentation. Déclencheurs : l'utilisateur demande de "lire la documentation", de "vérifier la documentation", de "parcourir la documentation pour X", de "récupérer la documentation depuis", "d'exporter la documentation" ou de faire référence à l'URL d'un site de documentation, à la documentation d'un package npm ou à la documentation du dépôt GitHub.
