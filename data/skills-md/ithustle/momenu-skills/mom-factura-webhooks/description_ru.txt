Внедрите подтверждение оплаты на основе веб-перехватчика для API Mom Factura. Вебхуки работают как для Bank Reference, так и для E-kwanza, отправляя два последовательных события (платеж.подтвержден + счет-фактура.создан). Используйте при построении потоков подтверждения платежей, получении уведомлений веб-перехватчика или обработке переходов состояния заказа из ОТКРЫТОГО в ОПЛАЧЕННОЕ. Конечные точки опроса состояния доступны в качестве резерва.
