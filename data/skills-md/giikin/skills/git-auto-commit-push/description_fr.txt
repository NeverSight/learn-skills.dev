Git prépare, valide et pousse automatiquement. Utilisez cette compétence lorsque l'utilisateur demande de valider du code, de pousser du code, de valider, de pousser, de valider et de pousser, de valider automatiquement, de stage-commit-push ou toute formulation impliquant des opérations git commit/push. Cela s'applique également lorsque l'utilisateur dit « validez-le », « poussez le code », « aidez-moi à valider », « poussez-le vers le haut », « validez uniquement » ou « validez simplement », etc.
