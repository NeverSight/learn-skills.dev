Generieren Sie professionelle PDF-Geschäftsdokumente (Rechnungen, Quittungen), indem Sie die DocGen Engine API aufrufen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine Rechnung, Quittung oder ein beliebiges Geschäftsdokument als PDF erstellen möchte – auch wenn er Dinge wie „buatkanrechnung“, „buat kwitansi“, „faktur generieren“, „Ich benötige eine Rechnungs-PDF“, „eine Quittung für meinen Kunden erstellen“ oder „tolong bikinrechnung buat klien“ sagt. Verwenden Sie dies auch, wenn der Benutzer Rohdaten (eine Liste von Artikeln, Kundennamen, Beträge) bereitstellt und ein formatiertes Dokument daraus erstellen möchte. Wenn der Benutzer ein PDF hochlädt und es neu generieren oder neu formatieren möchte, nutzen Sie diese Fähigkeit ebenfalls.
