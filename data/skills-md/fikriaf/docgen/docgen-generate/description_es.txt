Genere documentos comerciales PDF profesionales (facturas, recibos) llamando a la API de DocGen Engine. Utilice esta habilidad siempre que el usuario quiera crear una factura, un recibo o cualquier documento comercial en formato PDF, incluso si dice cosas como "factura buatkan", "buat kwitansi", "generar faktur", "Necesito una factura en PDF", "crear un recibo para mi cliente" o "factura tolong bikin buat klien". Utilícelo también cuando el usuario proporcione datos sin procesar (una lista de artículos, nombre del cliente, cantidades) y quiera obtener un documento formateado. Si el usuario carga un PDF y quiere regenerarlo o reformatearlo, utilice esta habilidad también.
