Überprüfen und verfeinern Sie Tests, die während einer Codierungssitzung erstellt wurden, kritisch. Verwenden Sie diese Fähigkeit, wenn der Benutzer Tests überprüfen, bereinigen, verfeinern, testen, die Testqualität verbessern oder nutzlose/geringwertige Tests entfernen möchte. Verwenden Sie es auch, wenn der Benutzer nach einer Codierungssitzung „Meine Tests überprüfen“, „Tests bereinigen“ sagt oder die Testqualität erwähnt.
