Revisar y perfeccionar críticamente las pruebas generadas durante una sesión de codificación. Utilice esta habilidad cuando el usuario solicite revisar pruebas, limpiar pruebas, refinar pruebas, podar pruebas, mejorar la calidad de las pruebas o eliminar pruebas inútiles o de bajo valor. Utilícelo también cuando el usuario diga "revisar mis pruebas", "limpiar pruebas" o mencione la calidad de la prueba después de una sesión de codificación.
