قم بمراجعة وتحسين الاختبارات التي تم إنشاؤها أثناء جلسة البرمجة بشكل نقدي. استخدم هذه المهارة عندما يطلب المستخدم مراجعة الاختبارات، أو تنظيف الاختبارات، أو تحسين الاختبارات، أو تقليم الاختبارات، أو تحسين جودة الاختبار، أو إزالة الاختبارات عديمة الفائدة/منخفضة القيمة. يُستخدم أيضًا عندما يقول المستخدم "مراجعة اختباراتي"، أو "تنظيف الاختبارات"، أو يذكر جودة الاختبار بعد جلسة البرمجة.
