Examinez de manière critique et affinez les tests générés lors d’une session de codage. Utilisez cette compétence lorsque l'utilisateur demande de réviser des tests, de nettoyer des tests, d'affiner des tests, d'élaguer des tests, d'améliorer la qualité des tests ou de supprimer des tests inutiles/de faible valeur. À utiliser également lorsque l'utilisateur dit « revoir mes tests », « nettoyer les tests » ou mentionne la qualité des tests après une session de codage.
