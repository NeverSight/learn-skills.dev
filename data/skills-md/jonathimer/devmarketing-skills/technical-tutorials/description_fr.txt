Lorsque l'utilisateur souhaite créer des didacticiels techniques étape par étape, des démarrages rapides ou des procédures pas à pas de code. Les expressions de déclenchement incluent « didacticiel », « démarrage rapide », « guide de démarrage », « procédure pas à pas », « étape par étape », « comment guider », « guide pratique » ou « didacticiel de code ».
