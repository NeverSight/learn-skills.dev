Создавайте SDK, которые любят использовать разработчики: API, которые кажутся естественными, сообщения об ошибках, которые помогают, и возможности, которые уменьшают трение. Этот навык охватывает создание SDK, которые способствуют внедрению благодаря исключительному опыту разработчиков, а не агрессивному маркетингу.
