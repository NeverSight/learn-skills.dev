Biblioteca de primitivas Rust CESR para el protocolo KERI. Se activa automáticamente cuando se trabaja con importaciones de cesride, rasgos de materia/indexador, tipos primitivos de CESR (Verfer, Diger, Signer, Salter, Siger, Cigar), serialización de Serder/Sadder o codificación/decodificación de Rust CESR. Cubre la API completa: construcción primitiva, operaciones criptográficas, serialización SAD, lógica de umbral y manejo de errores. Se remite a cesr/spec/acdc para la teoría del protocolo; se centra en los detalles específicos de Rust API.
