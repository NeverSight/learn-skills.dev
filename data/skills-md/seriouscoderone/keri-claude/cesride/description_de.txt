Rust CESR-Primitivbibliothek für das KERI-Protokoll. Wird automatisch aktiviert, wenn mit Cesride-Importen, Matter/Indexer-Merkmalen, CESR-Primitivtypen (Verfer, Diger, Signer, Salter, Siger, Cigar), Serder/Sadder-Serialisierung oder Rust CESR-Kodierung/Dekodierung gearbeitet wird. Deckt die gesamte API ab: primitive Konstruktion, kryptografische Operationen, SAD-Serialisierung, Schwellenwertlogik und Fehlerbehandlung. Verweist auf cesr/spec/acdc für Protokolltheorie; konzentriert sich auf die Besonderheiten der Rust-API.
