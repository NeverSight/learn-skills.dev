KERIプロトコル用のRust CESRプリミティブライブラリ。 cesride インポート、Matter/Indexer トレイト、CESR プリミティブ タイプ (Verfer、Diger、Signer、Salter、Siger、Cigar)、Serder/Sadder シリアル化、または Rust CESR エンコード/デコードを操作するときに自動的にアクティブになります。完全な API (プリミティブ構造、暗号操作、SAD シリアル化、しきい値ロジック、エラー処理) をカバーします。プロトコル理論については cesr/spec/acdc に従います。 Rust API の詳細に焦点を当てています。
