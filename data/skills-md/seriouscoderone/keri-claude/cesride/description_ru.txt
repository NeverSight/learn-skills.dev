Библиотека примитивов Rust CESR для протокола KERI. Автоматически активируется при работе с импортом cesride, характеристиками Matter/Indexer, примитивными типами CESR (Verfer, Diger, Signer, Salter, Siger, Cigar), сериализацией Serder/Sadder или кодированием/декодированием Rust CESR. Охватывает полный API: примитивную конструкцию, криптографические операции, сериализацию SAD, пороговую логику и обработку ошибок. Относится к cesr/spec/acdc для теории протоколов; фокусируется на специфике Rust API.
