مكتبة Rust CESR الأولية لبروتوكول KERI. يتم التنشيط تلقائيًا عند العمل مع عمليات استيراد cecride، أو سمات Matter/Indexer، أو الأنواع البدائية لـ CESR (Verfer، أو Diger، أو Signer، أو Salter، أو Siger، أو Cigar)، أو تسلسل Serder/Sadder، أو تشفير/فك تشفير Rust CESR. يغطي واجهة برمجة التطبيقات (API) الكاملة: البناء البدائي، وعمليات التشفير، وتسلسل SAD، ومنطق العتبة، ومعالجة الأخطاء. يؤجل إلى cesr/spec/acdc لنظرية البروتوكول؛ يركز على تفاصيل Rust API.
