Cree interfaces de usuario accesibles con HTML semántico, atributos ARIA adecuados, navegación por teclado y compatibilidad con lectores de pantalla. Utilice esta habilidad al crear o modificar etiquetas HTML, componentes de interfaz de usuario, formularios o elementos interactivos. Al implementar la navegación por teclado, la gestión del foco o el orden de tabulación. Al agregar texto alternativo para imágenes o etiquetas para entradas de formulario. Al garantizar relaciones de contraste de color suficientes. Al trabajar con menús de navegación, modales, botones o contenido dinámico. Al estructurar páginas con la jerarquía de encabezados adecuada (h1-h6). Al probar cambios en la interfaz de usuario para la compatibilidad del lector de pantalla.
