أنشئ واجهات مستخدم يمكن الوصول إليها باستخدام HTML الدلالي وسمات ARIA المناسبة والتنقل عبر لوحة المفاتيح والتوافق مع قارئ الشاشة. استخدم هذه المهارة عند إنشاء أو تعديل علامات HTML أو مكونات واجهة المستخدم أو النماذج أو العناصر التفاعلية. عند تنفيذ التنقل عبر لوحة المفاتيح أو إدارة التركيز أو ترتيب علامات التبويب. عند إضافة نص بديل للصور أو تسميات لمدخلات النموذج. عند ضمان نسب تباين الألوان الكافية. عند العمل باستخدام قوائم التنقل أو الوسائط أو الأزرار أو المحتوى الديناميكي. عند تنظيم الصفحات باستخدام التسلسل الهرمي للعناوين المناسبة (h1-h6). عند اختبار تغييرات واجهة المستخدم لتوافق قارئ الشاشة.
