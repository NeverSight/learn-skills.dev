Utilice esta habilidad al crear o refactorizar React, Vue, Svelte u otros componentes del marco. Al definir accesorios de componentes, interfaces o API. Al gestionar el estado de los componentes o elevar el estado a los componentes principales. Al crear elementos de interfaz de usuario reutilizables como botones, tarjetas, formularios, modales o listas. Al implementar patrones de composición de componentes. Cuando se trabaja con archivos de componentes (.jsx, .tsx, .vue, .svelte). Al documentar el uso de componentes o crear bibliotecas de componentes.
