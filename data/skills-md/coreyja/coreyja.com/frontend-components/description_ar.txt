تصميم مكونات واجهة مستخدم قابلة لإعادة الاستخدام وقابلة للتركيب بمسؤولية واحدة وواجهات واضحة وإدارة سليمة للحالة. استخدم هذه المهارة عند إنشاء React أو Vue أو Svelte أو مكونات إطار العمل الأخرى أو إعادة هيكلتها. عند تحديد دعائم المكونات أو الواجهات أو واجهات برمجة التطبيقات. عند إدارة حالة المكون أو رفع الحالة إلى المكونات الأصلية. عند إنشاء عناصر واجهة مستخدم قابلة لإعادة الاستخدام مثل الأزرار أو البطاقات أو النماذج أو الوسائط أو القوائم. عند تنفيذ أنماط تكوين المكونات. عند العمل مع ملفات المكونات (.jsx، .tsx، .vue، .svelte). عند توثيق استخدام المكونات أو إنشاء مكتبات المكونات.
