単一責任、明確なインターフェイス、および適切な状態管理を備えた、再利用可能で構成可能な UI コンポーネントを設計します。このスキルは、React、Vue、Svelte、またはその他のフレームワーク コンポーネントを作成またはリファクタリングするときに使用します。コンポーネントのプロパティ、インターフェイス、または API を定義するとき。コンポーネントの状態を管理する場合、または状態を親コンポーネントまで引き上げる場合。ボタン、カード、フォーム、モーダル、リストなどの再利用可能な UI 要素を構築する場合。コンポーネント構成パターンを実装する場合。コンポーネント ファイル (.jsx、.tsx、.vue、.svelte) を使用する場合。コンポーネントの使用法を文書化するとき、またはコンポーネント ライブラリを作成するとき。
