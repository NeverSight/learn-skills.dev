Utilice esta habilidad al escribir consultas SQL, métodos de consulta ORM o códigos de acceso a bases de datos. Al implementar una lógica de recuperación de datos que recupera o actualiza registros de la base de datos. Al optimizar consultas para evitar problemas N+1 mediante cargas ansiosas o uniones. Cuando se utilizan cláusulas WHERE, operaciones JOIN o declaraciones ORDER BY. Al implementar transacciones para operaciones de bases de datos relacionadas. Cuando se trabaja con métodos de creación de consultas, SQL sin formato u interfaces de consulta ORM. Al configurar tiempos de espera de consultas o implementar el almacenamiento en caché de resultados de consultas.
