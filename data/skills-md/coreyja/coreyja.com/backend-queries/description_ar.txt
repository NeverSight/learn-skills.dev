كتابة استعلامات قاعدة بيانات آمنة وفعالة باستخدام عبارات ذات معلمات، والفهرسة المناسبة، وتقنيات تحسين الاستعلام. استخدم هذه المهارة عند كتابة استعلامات SQL أو أساليب استعلام ORM أو رمز الوصول إلى قاعدة البيانات. عند تنفيذ منطق جلب البيانات الذي يقوم باسترداد أو تحديث سجلات قاعدة البيانات. عند تحسين الاستعلامات لمنع مشاكل N+1 من خلال التحميل أو الانضمامات المتلهفة. عند استخدام عبارات WHERE، أو عمليات JOIN، أو عبارات ORDER BY. عند تنفيذ المعاملات لعمليات قاعدة البيانات ذات الصلة. عند العمل باستخدام أساليب إنشاء الاستعلامات أو واجهات استعلام SQL الأولية أو ORM. عند تعيين مهلات الاستعلام أو تنفيذ التخزين المؤقت لنتائج الاستعلام.
