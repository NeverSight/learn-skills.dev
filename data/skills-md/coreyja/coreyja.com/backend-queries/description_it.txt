Utilizzare questa competenza durante la scrittura di query SQL, metodi di query ORM o codice di accesso al database. Quando si implementa la logica di recupero dei dati che recupera o aggiorna i record del database. Quando si ottimizzano le query per evitare problemi N+1 tramite caricamento rapido o join. Quando si utilizzano clausole WHERE, operazioni JOIN o istruzioni ORDER BY. Quando si implementano transazioni per operazioni di database correlate. Quando si lavora con metodi di creazione di query, interfacce di query SQL non elaborate o ORM.
