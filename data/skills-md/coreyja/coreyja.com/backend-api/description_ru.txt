Используйте этот навык при создании или изменении маршрутов API, контроллеров или обработчиков во внутреннем коде. При реализации логики конечной точки, которая обрабатывает HTTP-запросы (GET, POST, PUT, PATCH, DELETE). При работе с управлением версиями API, параметрами запроса или форматированием ответа. При определении структур URL-адресов для конечных точек на основе ресурсов. При настройке ограничения скорости или обработки кода состояния HTTP. При работе с такими файлами, как определения маршрутов, файлы контроллеров, модули обработчиков API или файлы конфигурации веб-платформы.
