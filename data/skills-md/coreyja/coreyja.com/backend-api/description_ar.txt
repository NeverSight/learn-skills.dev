قم بتصميم وتنفيذ نقاط نهاية RESTful API باتباع مبادئ REST واصطلاحات التسمية المتسقة وطرق HTTP المناسبة. استخدم هذه المهارة عند إنشاء أو تعديل مسارات API أو وحدات التحكم أو المعالجات في كود الواجهة الخلفية. عند تنفيذ منطق نقطة النهاية الذي يتعامل مع طلبات HTTP (GET، POST، PUT، PATCH، DELETE). عند العمل مع إصدار واجهة برمجة التطبيقات (API)، أو معلمات الاستعلام، أو تنسيق الاستجابة. عند تحديد بنيات URL لنقاط النهاية المستندة إلى الموارد. عند إعداد تحديد المعدل أو التعامل مع رمز حالة HTTP. عند العمل مع ملفات مثل تعريفات المسار، أو ملفات وحدة التحكم، أو وحدات معالج API، أو ملفات تكوين إطار عمل الويب.
