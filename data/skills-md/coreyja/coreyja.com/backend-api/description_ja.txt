REST の原則、一貫した命名規則、適切な HTTP メソッドに従って、RESTful API エンドポイントを設計および実装します。このスキルは、バックエンド コードで API ルート、コントローラー、またはハンドラーを作成または変更するときに使用します。 HTTP リクエスト (GET、POST、PUT、PATCH、DELETE) を処理するエンドポイント ロジックを実装する場合。 API のバージョン管理、クエリ パラメーター、または応答の書式設定を使用する場合。リソースベースのエンドポイントの URL 構造を定義する場合。レート制限または HTTP ステータス コードの処理を設定する場合。ルート定義、コントローラー ファイル、API ハンドラー モジュール、Web フレームワーク構成ファイルなどのファイルを操作する場合。
