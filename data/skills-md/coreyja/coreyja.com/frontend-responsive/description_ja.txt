モバイルファースト開発、流動的なレイアウト、および相対単位を使用して、モバイル、タブレット、デスクトップの画面サイズ全体でシームレスに機能するレスポンシブなレイアウトを構築します。このスキルは、さまざまな画面サイズに適応する必要があるレイアウトまたは UI コンポーネントを実装するときに使用します。ブレークポイント、メディア クエリ、または応答性の高いユーティリティを設定するとき。固定ピクセルの代わりに相対単位 (rem、em、%) を使用する場合。タッチ フレンドリーなタップ ターゲットを確保し、モバイル パフォーマンスを最適化する場合。複数のデバイス サイズにわたって UI の変更をテストする場合。レスポンシブなタイポグラフィ、画像、またはナビゲーション パターンを実装する場合。
