تعريف نماذج قاعدة البيانات بتسمية واضحة، وأنواع البيانات المناسبة، والقيود المناسبة، وتعريفات العلاقات. استخدم هذه المهارة عند إنشاء أو تعديل ملفات نموذج ORM أو تعريفات كيان قاعدة البيانات أو مخططات بنية البيانات. عند تحديد علاقات الجدول والمفاتيح الخارجية والسلوكيات المتتالية. عند تنفيذ قواعد التحقق من صحة النموذج أو الطوابع الزمنية أو الفهارس. عند العمل مع ملفات النماذج في أطر عمل مثل SQLAlchemy، أو Sequelize، أو ActiveRecord، أو SQLx، أو Prisma، أو ORMs المشابهة. عند اختيار أنواع البيانات المناسبة للأعمدة أو إنشاء قيود على تكامل البيانات (NOT NULL، UNIQUE، المفاتيح الخارجية).
