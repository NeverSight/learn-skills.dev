حافظ على تنسيق التعليمات البرمجية المتسق واصطلاحات التسمية والبنية عبر قاعدة التعليمات البرمجية بأكملها باستخدام أدوات تلقائية ومعايير واضحة. استخدم هذه المهارة عند كتابة أي كود بأي لغة أو إطار عمل. عند تسمية المتغيرات أو الوظائف أو الفئات أو الملفات. عند تنسيق التعليمات البرمجية مع المسافة البادئة المناسبة وفواصل الأسطر. عند إعادة هيكلة التعليمات البرمجية لإزالة التكرار أو استخراج وظائف قابلة لإعادة الاستخدام. عند تشغيل الفلاتر أو المنسقات لضمان الاتساق. عند إزالة التعليمات البرمجية الميتة أو الواردات غير المستخدمة أو الكتل التي تم التعليق عليها. عند تطبيق مبادئ DRY لتجنب تكرار التعليمات البرمجية. تنطبق هذه المهارة عالميًا على جميع مهام البرمجة.
