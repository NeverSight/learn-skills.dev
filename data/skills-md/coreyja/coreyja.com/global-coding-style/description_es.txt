Mantenga un formato de código, convenciones de nomenclatura y una estructura coherentes en todo el código base con herramientas automatizadas y estándares claros. Utilice esta habilidad al escribir cualquier código en cualquier lenguaje o marco. Al nombrar variables, funciones, clases o archivos. Al formatear código con sangría y saltos de línea adecuados. Al refactorizar código para eliminar duplicaciones o extraer funciones reutilizables. Al ejecutar linters o formateadores para garantizar la coherencia. Al eliminar código inactivo, importaciones no utilizadas o bloques comentados. Al aplicar principios DRY para evitar la duplicación de código. Esta habilidad se aplica universalmente a todas las tareas de programación.
