قم بإنشاء وإدارة عمليات ترحيل قاعدة البيانات باستخدام طرق التراجع المناسبة والتغييرات المركزة واعتبارات النشر بدون توقف. استخدم هذه المهارة عند إنشاء ملفات ترحيل قاعدة بيانات جديدة، أو تعديل مخططات الجدول، أو إضافة أعمدة أو إزالتها، أو إنشاء فهارس أو إسقاطها، أو إدارة التحكم في إصدار قاعدة البيانات. عند العمل مع أدلة الترحيل أو ملفات تعريف المخطط أو البرامج النصية لتغيير قاعدة البيانات. عند تنفيذ تغييرات قاعدة البيانات المتوافقة مع الإصدارات السابقة لعمليات نشر الإنتاج. عند فصل تغييرات المخطط عن عمليات ترحيل البيانات.
