Crea e gestisci le migrazioni dei database con metodi di rollback adeguati, modifiche mirate e considerazioni sulla distribuzione senza tempi di inattività. Utilizzare questa competenza durante la creazione di nuovi file di migrazione del database, la modifica degli schemi di tabelle, l'aggiunta o la rimozione di colonne, la creazione o l'eliminazione di indici o la gestione del controllo della versione del database. Quando si lavora con directory di migrazione, file di definizione dello schema o script di modifica del database. Quando si implementano modifiche al database compatibili con le versioni precedenti per le distribuzioni di produzione. Quando si separano le modifiche dello schema dalle migrazioni dei dati.
