Encuentra ideas de productos, negocios o proyectos viables en cualquier dominio generando candidatos en lotes, validando rigurosamente cada uno de ellos frente a competidores reales y datos de mercado, y eliminando ideas débiles con razones documentadas. Úselo cuando el usuario solicite realizar una lluvia de ideas, encontrar brechas en el mercado, identificar oportunidades comerciales, descubrir qué construir, proponer ideas para proyectos paralelos o encontrar nichos desatendidos. Funciona para SaaS, código abierto, servidores MCP, aplicaciones, herramientas, API o cualquier categoría de producto.
