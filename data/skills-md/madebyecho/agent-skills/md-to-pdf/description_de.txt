Konvertiert Markdown-Dateien in professionell gestaltete PDF-Dokumente. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine PDF-Datei erstellen, Markdown als PDF exportieren, .md in .pdf konvertieren oder eine druckbare Version eines Dokuments erstellen möchte. Wird auch ausgelöst, wenn der Benutzer statusfarbene Tabellen oder einen formatierten Dokumentationsexport wünscht oder Dinge sagt wie „Erstelle dies als PDF“, „Ich benötige eine PDF-Version“ oder „Exportiere dieses Dokument“. Unterstützt automatische Ausrichtung, optionale schlüsselwortbasierte Zellfärbung und elegantes Engine-Fallback.
