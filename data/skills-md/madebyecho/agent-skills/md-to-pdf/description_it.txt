Converte file di markdown in documenti PDF dallo stile professionale. Utilizza questa abilità ogni volta che l'utente chiede di generare un PDF, esportare markdown come PDF, convertire .md in .pdf o creare una versione stampabile di un documento. Si attiva anche quando l'utente desidera tabelle con colore di stato, esportazione di documentazione con stile o dice cose come "crea un PDF", "Ho bisogno di una versione PDF", "esporta questo documento". Supporta l'orientamento automatico, la colorazione opzionale delle celle basata su parole chiave e il grazioso fallback del motore.
