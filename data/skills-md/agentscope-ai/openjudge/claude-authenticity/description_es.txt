También extrae indicaciones del sistema inyectadas de proveedores que anulan la identidad de Claude. Totalmente autónomo: copie el código siguiente y ejecútelo, sin paquetes adicionales más allá de httpx. Úselo cuando el usuario desee verificar una clave o punto final de API de Claude, verificar si un servicio Claude de terceros es auténtico, auditar proveedores de API para verificar la autenticidad de Claude, probar varios modelos en paralelo o descubrir qué indicador del sistema ha inyectado un proveedor.
