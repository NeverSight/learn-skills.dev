Создайте безголовые витрины магазинов Shopify с помощью водорода и кислорода. Используйте этот навык для создания пользовательских витрин на основе React, использования компонентов Hydrogen, развертывания на хостинге Oxygen, работы с API Storefront и создания высокопроизводительного опыта электронной коммерции. Также рассматривается использование собственного стека с пользовательскими платформами.
