Realice una investigación de base de código profunda y respaldada por evidencia con subinvestigaciones paralelas y produzca una nota estructurada con citas de archivos/líneas. Utilícelo siempre que el usuario solicite investigar, analizar, rastrear, depurar, auditar o comprender cómo se conecta el código, incluidas solicitudes como "analizar este repositorio", "rastrear este flujo", "dónde se maneja", investigaciones de causa raíz, inmersiones profundas en la arquitectura, comparaciones de comportamiento y mapeo de implementación, incluso si el usuario nunca dice "investigar".
