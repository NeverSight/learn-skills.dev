Bootstrap проверяет Amp с высоким уровнем сигнала на основе объединенной истории запросов на извлечение. Используйте это всякий раз, когда пользователь просит создать `.agents/checks`, сделать вывод о правилах командной проверки на основе запросов на участие или предложить проверки на основе истории проверок.
