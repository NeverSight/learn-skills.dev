Installa e configura la CLI case.dev per flussi di lavoro di intelligenza artificiale legali tra cui archivi di documenti, OCR, trascrizione e ricerca. Da utilizzare quando l'utente menziona "case.dev", "casedev", deve autenticarsi con case.dev, eseguire la diagnostica, impostare obiettivi di messa a fuoco, elencare percorsi API, tenere traccia dei lavori o effettuare chiamate API non elaborate. Competenza gateway per tutte le competenze case.dev.
