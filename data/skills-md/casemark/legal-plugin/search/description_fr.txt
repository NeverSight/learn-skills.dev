Recherche sur le Web, les bases de données juridiques, la jurisprudence, les brevets et la base de connaissances case.dev via la CLI casedev. À utiliser lorsque l'utilisateur mentionne « recherche », « recherche juridique », « recherche de cas », « jurisprudence », « recherche de brevets », « recherche sur le Web », « récupération d'URL », « récupération sur le Web », « compétences juridiques » ou a besoin de rechercher des sujets juridiques, de trouver des cas similaires ou de récupérer du contenu Web.
