Delega le attività di codifica agli agenti Codex, Claude Code o Pi tramite un processo in background. Utilizzare quando: (1) creare/creare nuove funzionalità o app, (2) rivedere PR (spawn in directory temporanea), (3) eseguire il refactoring di codebase di grandi dimensioni, (4) codifica iterativa che richiede l'esplorazione di file. NON per: semplici correzioni di una riga (basta modificare), lettura del codice (utilizzare lo strumento di lettura), richieste di cablaggio ACP legate a thread in chat (ad esempio generare/eseguire Codex o Codice Claude in un thread Discord; utilizzare sessions_spawn con runtime:"acp") o qualsiasi lavoro nell'area di lavoro ~/clawd (non generare mai agenti qui). Richiede uno strumento bash che supporti pty:true.
