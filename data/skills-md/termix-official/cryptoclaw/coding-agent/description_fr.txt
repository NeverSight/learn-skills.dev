Déléguez les tâches de codage aux agents Codex, Claude Code ou Pi via un processus en arrière-plan. À utiliser lorsque : (1) la création/la création de nouvelles fonctionnalités ou applications, (2) la révision des PR (apparition dans le répertoire temporaire), (3) la refactorisation de bases de code volumineuses, (4) le codage itératif nécessitant une exploration de fichiers. PAS pour : de simples correctifs d'une seule ligne (il suffit de modifier), la lecture de code (utiliser l'outil de lecture), les demandes d'exploitation ACP liées au thread dans le chat (par exemple, générer/exécuter Codex ou Claude Code dans un fil Discord ; utilisez sessions_spawn avec runtime : "acp"), ou tout travail dans l'espace de travail ~/clawd (ne jamais générer d'agents ici). Nécessite un outil bash prenant en charge pty:true.
