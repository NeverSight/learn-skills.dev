Delegieren Sie Codierungsaufgaben über einen Hintergrundprozess an Codex-, Claude Code- oder Pi-Agenten. Wird verwendet, wenn: (1) neue Funktionen oder Apps erstellt/erstellt werden, (2) PRs überprüft werden (spawn im temporären Verzeichnis), (3) große Codebasen umgestaltet werden, (4) iteratives Codieren, das eine Dateierkundung erfordert. NICHT für: einfache einzeilige Korrekturen (einfach bearbeiten), Code lesen (Lesetool verwenden), Thread-gebundene ACP-Harness-Anfragen im Chat (zum Beispiel Codex oder Claude Code in einem Discord-Thread erzeugen/ausführen; session_spawn mit runtime:"acp verwenden") oder jede Arbeit im ~/clawd-Arbeitsbereich (hier niemals Agenten erzeugen). Erfordert ein Bash-Tool, das pty:true unterstützt.
