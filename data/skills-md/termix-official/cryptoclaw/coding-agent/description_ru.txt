Делегируйте задачи кодирования агентам Codex, Claude Code или Pi с помощью фонового процесса. Используйте при: (1) создании/создании новых функций или приложений, (2) просмотре PR (появляются во временном каталоге), (3) рефакторинге больших кодовых баз, (4) итеративном кодировании, требующем исследования файлов. НЕ для: простых однострочных исправлений (просто отредактируйте), чтения кода (используйте инструмент чтения), запросов на использование ACP с привязкой к потоку в чате (например, создайте/запустите Codex или Claude Code в потоке Discord; используйте session_spawn с runtime:"acp") или любую работу в рабочей области ~/clawd (никогда не запускайте агентов здесь). Требуется инструмент bash, поддерживающий pty:true.
