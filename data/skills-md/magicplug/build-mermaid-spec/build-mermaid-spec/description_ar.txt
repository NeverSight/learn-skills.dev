يُستخدم عندما يريد المستخدم إنشاء رسم تخطيطي لحورية البحر، أو تصور تدفق النظام، أو يقول "إنشاء رسم تخطيطي"، أو "حورية البحر"، أو "تصور التدفق"، أو "رسم هذا الرسم التخطيطي". استخدمه أيضًا عند وجود مستند تصميم أو خطة تنفيذ في docs/plans/ ويطلب المستخدم رسم تخطيطي له. يُستخدم عندما تكون المتطلبات غامضة وتحتاج إلى توضيح قبل البناء - "أريد البناء"، "التخطيط"، "التصميم"، "دعونا نصمم معماريًا".
