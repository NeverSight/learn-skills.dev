Úselo cuando un usuario quiera generar un diagrama de sirena, visualizar el flujo de un sistema o diga "crear un diagrama", "sirena", "visualizar el flujo", "diagramar esto". Utilícelo también cuando exista un documento de diseño o un plan de implementación en docs/plans/ y el usuario solicite diagramarlo. Úselo cuando los requisitos sean vagos y necesiten una aclaración antes de construir: "Quiero construir", "planificar", "diseñar", "vamos a diseñar".
