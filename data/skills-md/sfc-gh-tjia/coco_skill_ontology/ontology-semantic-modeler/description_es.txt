Genera modelos semánticos de Snowflake Cortex Analyst a partir de archivos de ontología OWL asignados a tablas existentes. Analiza clases y relaciones OWL, crea tablas de metadatos, vistas abstractas y modelo semántico YAML con consultas verificadas para recorrido de jerarquía. Incluye visualización de ontologías. Úselo cuando: construir un modelo semántico de ontología, OWL a modelo semántico, asignar ontología a tablas, generar vistas de ontología, vista semántica de ontología, unir ontología con tablas de datos.
