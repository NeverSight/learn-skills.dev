Compétence pour analyser le code existant (sans supposer DDD) et proposer un modèle de domaine (agrégats, entités locales, objets de valeur, services de domaine) avec pseudocode. Propose une conception qui inclut des propriétés, des méthodes, des invariants et des limites d'agrégation. Langue cible : indépendante de la langue. Déclencheur : activé par les demandes d'analyse du code existant et d'extraction d'un modèle de domaine telles que « Extraire un modèle de domaine du code existant », « Je souhaite repenser ce code avec DDD », « Proposer un modèle de domaine », « Rechercher les agrégats », « Migrer vers DDD », « Analyser le code existant et créer un modèle de domaine » ou « Transformer le code existant en DDD ».
