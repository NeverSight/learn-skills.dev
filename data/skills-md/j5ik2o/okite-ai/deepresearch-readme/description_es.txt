Admite la creación, mejora y revisión de README.md para GitHub OSS. Organiza la información a lo largo de los ejes Qué/Por qué/Cómo iniciar/Ayuda/Mantenedores y proporciona un Inicio rápido que funciona en el menor tiempo posible y una ruta a los documentos. Se utiliza al crear un nuevo archivo README, reorganizar un archivo README existente, revisar un archivo README o decidir dividir la documentación. Activador: solicitudes relacionadas con el archivo README, como "Crear un archivo README", "Mejorar el archivo README", "Agregar un inicio rápido", "plantilla README para GitHub", "Revisar el archivo README".
