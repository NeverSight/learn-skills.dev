GitHub OSS の README.md の作成、改善、レビューをサポートします。 「何を/なぜ/どのように開始するか/ヘルプ/メンテナ」の軸に沿って情報を整理し、ユーザーが最短時間で実行できるようにするクイックスタートとドキュメントへのパスを構造化します。新しい README の作成、既存の README の再編成、README のレビュー、またはドキュメントを分割するかどうかを決定するときに使用します。 「README の作成」、「README の改善」、「クイックスタートの追加」、「GitHub README テンプレート」、「README のレビュー」などの README 関連のリクエストによってトリガーされます。
