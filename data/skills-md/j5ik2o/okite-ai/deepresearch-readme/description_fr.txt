Aide à créer, améliorer et réviser README.md pour GitHub OSS. Organise les informations selon les axes Quoi/Pourquoi/Comment démarrer/Aide/Mainteneurs, en structurant un démarrage rapide qui fonctionne immédiatement et des liens vers des documents. À utiliser lors de l'écriture d'un nouveau README, de la restructuration d'un fichier existant, de la révision d'un README ou de la décision de diviser la documentation. Déclencheurs : « Créer un README », « Améliorer le README », « Ajouter un démarrage rapide », « Modèle GitHub README », « Réviser le README » et d'autres demandes liées au README.
