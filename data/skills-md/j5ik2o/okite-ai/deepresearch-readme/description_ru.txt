Помогает в создании, улучшении и проверке README.md для GitHub OSS. Организует информацию по осям «Что/Почему/Как начать/Помощь/Сопровождающие» и предоставляет быстрый старт, который работает в кратчайшие сроки, а также путь к документации. Используйте при создании нового README, реструктуризации существующего README, просмотре README или решении разделить документацию. Триггер: запросы, связанные с README, такие как «Создать README», «Улучшить README», «Добавить быстрый старт», «Шаблон README GitHub», «Просмотреть README».
