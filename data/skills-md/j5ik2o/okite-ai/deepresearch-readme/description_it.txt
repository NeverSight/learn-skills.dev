Supporta la creazione, il miglioramento e la revisione di README.md per GitHub OSS. Organizza le informazioni lungo gli assi Cosa/Perché/Come avviare/Aiuto/Manutentori e fornisce un avvio rapido che funziona immediatamente e percorsi per i documenti. Utilizzato quando si crea un nuovo README, si riorganizza uno esistente, si rivede un README o si decide se dividere la documentazione. Trigger: richieste relative al README come "Crea un README", "Migliora il README", "Aggiungi un Quickstart", "Modello README di GitHub" o "Rivedi il mio README".
