Unterstützt das Erstellen, Verbessern und Überprüfen von README.md-Dateien für Open-Source-Projekte auf GitHub. Organisiert Informationen rund um die Achsen Was/Warum/Starten/Hilfe/Betreuer und bietet einen Schnellstart, der Benutzer schnell einsatzbereit macht und sie zur Dokumentation führt. Verwenden Sie diese Option, wenn Sie eine neue README-Datei erstellen, eine vorhandene README-Datei umstrukturieren, eine README-Datei überprüfen oder entscheiden, ob die Dokumentation aufgeteilt werden soll. Auslöser: „Erstellen Sie eine README-Datei“, „Verbessern Sie die README-Datei“, „Fügen Sie einen Schnellstart hinzu“, „GitHub README-Vorlage“, „Überprüfen Sie meine README-Datei“ oder alle anderen README-bezogenen Anfragen.
