A migration skill that moves the skill directory from `.claude/skills/{name}` to `.agents/skills/{name}` and creates symbolic links from both `.claude/skills/{name}` and `.codex/skills/{name}`. This directory structure centralizes the actual skill files in `.agents/skills/` and makes them accessible from both Claude Code and Codex CLI. The skill name is required as an argument.  
Triggers: activated by requests to change skill locations such as "move skill to .agents", "migrate skill", "create symbolic link", "migrate skill", or "unify skill placement".
