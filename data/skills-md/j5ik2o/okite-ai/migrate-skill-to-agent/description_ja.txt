スキル ディレクトリを `.claude/skills/{name}` から `.agents/skills/{name}` に移動し、`.claude/skills/{name}` と `.codex/skills/{name}` の両方からシンボリック リンクを作成する移行スキル。実際のスキル ファイルを `.agents/skills/` に集中させ、Claude Code と Codex CLI の両方からアクセスできるディレクトリ構造を有効にします。引数としてスキル名が必要です。 「スキルを .agents に移動」、「スキルを移行」、「シンボリック リンクを作成」、「スキルを移行」、または「スキルの配置を統合」などのスキルの場所を変更するリクエストによってトリガーされます。
