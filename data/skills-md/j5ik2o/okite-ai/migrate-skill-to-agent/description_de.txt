Ein Migrations-Skill, der das Skill-Verzeichnis von „.claude/skills/{name}“ nach „.agents/skills/{name}“ verschiebt und symbolische Links sowohl von „.claude/skills/{name}“ als auch von „.codex/skills/{name}“ erstellt. Diese Verzeichnisstruktur zentralisiert den eigentlichen Skill in „.agents/skills/“ und macht ihn sowohl über Claude Code als auch über die Codex-CLI zugänglich. Als Argument ist der Skillname erforderlich. Auslöser: „Skill nach .agents verschieben“, „Skill migrieren“, „Symbolischen Link erstellen“, „Skill migrieren“, „Skill-Platzierung vereinheitlichen“ und andere Änderungsanfragen zur Skill-Platzierung.
