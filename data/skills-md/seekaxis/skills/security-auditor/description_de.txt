Überprüfen Sie die Agentenfähigkeiten (SKILL.md und gebündelte Skripte) proaktiv auf Sicherheitsrisiken, einschließlich böswilliger Installationsprogramme, verschleierter Payloads, Exfiltration von Anmeldeinformationen und Angriffe auf die Lieferkette. Verwenden Sie diese Option, wenn Sie einen Agenten-Skill installieren, überprüfen oder testen oder wenn der Benutzer darum bittet, einen Skill auf Sicherheit zu überprüfen.
