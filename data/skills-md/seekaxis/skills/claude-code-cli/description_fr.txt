Exécutez Claude Code par programmation à partir de scripts, de pipelines CI/CD et d'autres agents IA. Couvre le mode d'impression CLI (`claude -p`), le SDK d'agent (Python et TypeScript), la sortie structurée/streaming, la gestion de session, les sous-agents, les équipes d'agents et les modèles d'automatisation courants. À utiliser lorsque l'utilisateur souhaite : (1) automatiser des tâches avec Claude Code, (2) intégrer Claude Code dans des scripts ou des pipelines, (3) utiliser le SDK Agent de Python ou TypeScript, (4) orchestrer plusieurs sessions Claude Code, (5) obtenir une sortie JSON structurée à partir de Claude Code ou (6) créer des agents au-dessus de Claude Code.
