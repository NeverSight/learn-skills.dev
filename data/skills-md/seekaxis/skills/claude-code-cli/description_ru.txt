Запускайте Claude Code программно из сценариев, конвейеров CI/CD и других агентов ИИ. Охватывает режим печати CLI (`claude -p`), Agent SDK (Python и TypeScript), структурированный/потоковый вывод, управление сеансами, субагенты, команды агентов и общие шаблоны автоматизации. Используйте, когда пользователь хочет: (1) автоматизировать задачи с помощью Claude Code, (2) интегрировать Claude Code в сценарии или конвейеры, (3) использовать Agent SDK из Python или TypeScript, (4) организовать несколько сеансов Claude Code, (5) получить структурированный вывод JSON из Claude Code или (6) построить агенты поверх Claude Code.
