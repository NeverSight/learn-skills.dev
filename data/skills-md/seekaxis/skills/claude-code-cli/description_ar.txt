قم بتشغيل Claude Code برمجيًا من البرامج النصية وخطوط أنابيب CI/CD ووكلاء الذكاء الاصطناعي الآخرين. يغطي وضع الطباعة CLI (`clude -p`)، وAgent SDK (Python وTypeScript)، والمخرجات المنظمة/المتدفقة، وإدارة الجلسة، والوكلاء الفرعيين، وفرق الوكلاء، وأنماط الأتمتة الشائعة. يُستخدم عندما يريد المستخدم: (1) أتمتة المهام باستخدام Claude Code، (2) دمج Claude Code في البرامج النصية أو خطوط الأنابيب، (3) استخدام Agent SDK من Python أو TypeScript، (4) تنسيق جلسات Claude Code المتعددة، (5) الحصول على مخرجات JSON منظمة من Claude Code، أو (6) إنشاء وكلاء فوق Claude Code.
