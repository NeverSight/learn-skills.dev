从脚本、CI/CD 管道和其他 AI 代理以编程方式运行 Claude Code。涵盖 CLI 打印模式（“claude -p”）、代理 SDK（Python 和 TypeScript）、结构化/流式输出、会话管理、子代理、代理团队和常见自动化模式。当用户想要执行以下操作时使用：(1) 使用 Claude Code 自动执行任务，(2) 将 Claude Code 集成到脚本或管道中，(3) 使用 Python 或 TypeScript 中的代理 SDK，(4) 编排多个 Claude Code 会话，(5) 从 Claude Code 获取结构化 JSON 输出，或 (6) 在 Claude Code 之上构建代理。
