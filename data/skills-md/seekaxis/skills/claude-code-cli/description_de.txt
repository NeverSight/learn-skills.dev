Führen Sie Claude Code programmgesteuert über Skripte, CI/CD-Pipelines und andere KI-Agenten aus. Behandelt den CLI-Druckmodus („claude -p“), das Agent SDK (Python und TypeScript), strukturierte/Streaming-Ausgabe, Sitzungsverwaltung, Subagenten, Agententeams und allgemeine Automatisierungsmuster. Verwenden Sie diese Option, wenn der Benutzer Folgendes möchte: (1) Aufgaben mit Claude Code automatisieren, (2) Claude Code in Skripte oder Pipelines integrieren, (3) das Agent SDK von Python oder TypeScript verwenden, (4) mehrere Claude Code-Sitzungen orchestrieren, (5) eine strukturierte JSON-Ausgabe von Claude Code erhalten oder (6) Agenten auf Claude Code erstellen möchte.
