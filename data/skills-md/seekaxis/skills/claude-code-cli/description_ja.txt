スクリプト、CI/CD パイプライン、その他の AI エージェントからプログラムでクロード コードを実行します。 CLI 印刷モード (「claude -p」)、エージェント SDK (Python および TypeScript)、構造化/ストリーミング出力、セッション管理、サブエージェント、エージェント チーム、一般的な自動化パターンについて説明します。ユーザーが次のことを行う場合に使用します: (1) クロード コードでタスクを自動化する、(2) クロード コードをスクリプトまたはパイプラインに統合する、(3) Python または TypeScript からエージェント SDK を使用する、(4) 複数のクロード コード セッションをオーケストレーションする、(5) クロード コードから構造化された JSON 出力を取得する、または (6) クロード コード上にエージェントを構築する。
