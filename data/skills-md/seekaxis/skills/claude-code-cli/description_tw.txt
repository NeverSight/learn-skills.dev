從腳本、CI/CD 管道和其他 AI 代理程式以程式設計方式執行 Claude Code。涵蓋 CLI 列印模式（“claude -p”）、代理程式 SDK（Python 和 TypeScript）、結構化/串流輸出、會話管理、子代理程式、代理團隊和常見自動化模式。當使用者想要執行以下操作時使用：(1) 使用 Claude Code 自動執行任務，(2) 將 Claude Code 整合到腳本或管道中，(3) 使用 Python 或 TypeScript 中的代理 SDK，(4) 編排多個 Claude Code 會話，(5) 從 Claude Code 取得結構化 JSON 輸出，或 6de 代理程式。
