Fait fonctionner Notion via la CLI « notion » ou « notion-cli-agent » avec des commandes optimisées pour l'agent pour l'inspection de schéma, l'interrogation de base de données, l'édition de pages, l'importation/exportation Markdown, les opérations par lots, la validation, la sauvegarde et les invites d'aide de l'IA. À utiliser lorsque l'utilisateur souhaite gérer Notion depuis le terminal via cette CLI au lieu d'utiliser directement l'API Notion.
