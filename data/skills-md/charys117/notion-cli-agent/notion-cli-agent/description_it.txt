Gestisce Notion tramite la CLI `notion` o `notion-cli-agent` con comandi ottimizzati per l'agente per l'ispezione dello schema, l'interrogazione del database, la modifica delle pagine, l'importazione/esportazione di Markdown, le operazioni batch, la convalida, il backup e i prompt dell'helper AI. Da utilizzare quando l'utente desidera gestire Notion dal terminale tramite questa CLI invece di utilizzare direttamente l'API Notion.
