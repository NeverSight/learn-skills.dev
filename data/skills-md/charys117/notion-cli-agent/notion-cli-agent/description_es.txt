Opera Notion a través de la CLI `notion` o `notion-cli-agent` con comandos optimizados para el agente para inspección de esquemas, consultas de bases de datos, edición de páginas, importación/exportación de Markdown, operaciones por lotes, validación, copias de seguridad y mensajes auxiliares de IA. Úselo cuando el usuario quiera administrar Notion desde la terminal a través de esta CLI en lugar de usar la API de Notion directamente.
