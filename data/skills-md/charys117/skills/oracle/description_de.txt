Bereiten Sie ein „Fragen Sie einen Experten“-Übergabepaket vor: eine einfügbare prompt.md plus eine context.zip mit den mindestens erforderlichen Repo-Dateien für eine fundierte Zweitmeinung von einem externen Assistenten (z. B. ChatGPT Pro / GPT-5.2 Pro). Nur verwenden, wenn der Benutzer ausdrücklich darum bittet, „Oracle zu verwenden“, „ChatGPT Pro zu fragen“, „einen Experten zu fragen“ oder „eine zweite Meinung einzuholen“. Fügen Sie niemals Geheimnisse (.env, Schlüssel, Anmeldeinformationen) in das Paket ein.
