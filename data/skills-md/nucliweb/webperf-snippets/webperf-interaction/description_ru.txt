Интеллектуальный анализ производительности взаимодействия с автоматизированными рабочими процессами для отладки INP, исследования зависаний прокрутки и блокировки основного потока. Включает деревья решений, которые автоматически запускают атрибуцию сценариев при обнаружении длинных кадров, разбивают фазы задержки ввода и коррелируют изменения макета с взаимодействиями. Содержит рабочие процессы для полного аудита взаимодействия, анализа влияния сторонних сценариев и отладки производительности анимации. Интеграция между навыками с Core Web Vitals (корреляция INP/CLS) и загрузкой (анализ выполнения скриптов). Используйте, когда пользователь спрашивает о медленном взаимодействии, прерывистой прокрутке, не отвечающих страницах или оптимизации INP. Совместим с Chrome DevTools MCP.
