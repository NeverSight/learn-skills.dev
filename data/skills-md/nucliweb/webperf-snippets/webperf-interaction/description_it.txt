Analisi intelligente delle prestazioni dell'interazione con flussi di lavoro automatizzati per il debug INP, l'indagine sugli scroll jank e il blocco del thread principale. Include alberi decisionali che eseguono automaticamente l'attribuzione degli script quando vengono rilevati frame lunghi, suddividono le fasi di latenza dell'input e correlano i cambiamenti di layout con le interazioni. Dispone di flussi di lavoro per il controllo completo dell'interazione, l'analisi dell'impatto degli script di terze parti e il debug delle prestazioni dell'animazione. Integrazione di competenze incrociate con Core Web Vitals (correlazione INP/CLS) e Loading (analisi dell'esecuzione degli script). Da utilizzare quando l'utente chiede informazioni su interazioni lente, scorrimento lento, pagine che non rispondono o ottimizzazione INP. Compatibile con Chrome DevTools MCP.
