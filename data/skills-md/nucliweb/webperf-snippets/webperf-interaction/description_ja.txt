INP デバッグ、スクロール ジャンク調査、メイン スレッド ブロッキングのための自動ワークフローによるインテリジェントなインタラクション パフォーマンス分析。長いフレームが検出されたときにスクリプト アトリビューションを自動的に実行し、入力レイテンシ フェーズを分析し、レイアウトのシフトとインタラクションを関連付けるデシジョン ツリーが含まれています。完全なインタラクション監査、サードパーティ製スクリプトの影響分析、アニメーション パフォーマンスのデバッグのためのワークフローを備えています。 Core Web Vitals (INP/CLS 相関) およびローディング (スクリプト実行分析) とのクロススキル統合。インタラクションの遅さ、スクロールが不安定、ページが応答しない、または INP の最適化についてユーザーが尋ねた場合に使用します。 Chrome DevTools MCP と互換性があります。
