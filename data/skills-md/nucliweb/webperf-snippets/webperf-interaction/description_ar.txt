تحليل أداء التفاعل الذكي مع سير العمل الآلي لتصحيح أخطاء INP، والتحقيق في الملفات غير المرغوب فيها، وحظر الخيط الرئيسي. تتضمن أشجار القرار التي تقوم تلقائيًا بتشغيل إسناد البرنامج النصي عند اكتشاف إطارات طويلة، وتقسيم مراحل زمن وصول الإدخال، وربط تغييرات التخطيط بالتفاعلات. يتميز بسير العمل لتدقيق التفاعل الكامل، وتحليل تأثير البرنامج النصي لجهة خارجية، وتصحيح أخطاء أداء الرسوم المتحركة. التكامل عبر المهارات مع Core Web Vitals (ارتباط INP/CLS) والتحميل (تحليل تنفيذ البرنامج النصي). يُستخدم عندما يسأل المستخدم عن التفاعلات البطيئة، أو التمرير غير المرغوب فيه، أو الصفحات غير المستجيبة، أو تحسين INP. متوافق مع Chrome DevTools MCP.
