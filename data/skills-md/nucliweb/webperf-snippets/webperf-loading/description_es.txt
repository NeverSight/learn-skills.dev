Análisis inteligente del rendimiento de carga con flujos de trabajo automatizados para la investigación de TTFB (DNS/conexión/avería del servidor), detección de bloqueo de renderizado, análisis profundo del rendimiento del script (atribución propia frente a terceros), optimización de fuentes y validación de sugerencias de recursos. Incluye árboles de decisión que analizan automáticamente las subpartes de TTFB cuando son lentas, detectan antipatrones de carga de scripts (conflictos asíncronos/aplazados/precargados), identifican recursos que bloquean el procesamiento y validan el uso de sugerencias de recursos. Incluye flujos de trabajo para una auditoría de carga completa (6 fases), investigación del rendimiento del backend y optimización de prioridades. Integración entre habilidades con Core Web Vitals (carga de recursos LCP), Interaction (bloqueo de ejecución de script) y Media (estrategia de carga diferida). Úselo cuando el usuario pregunte sobre TTFB, FCP, bloqueo de renderizado, carga lenta, rendimiento de fuentes, optimización de scripts o sugerencias de recursos. Compatible con Chrome DevTools MCP.
