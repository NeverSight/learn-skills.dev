تحليل أداء التحميل الذكي مع سير العمل الآلي لتحقيق TTFB (نظام أسماء النطاقات/الاتصال/انهيار الخادم)، واكتشاف حظر العرض، والتعمق في أداء البرنامج النصي (إسناد الطرف الأول مقابل إسناد الطرف الثالث)، وتحسين الخط، والتحقق من صحة تلميحات الموارد. تتضمن أشجار القرار التي تحلل تلقائيًا أجزاء TTFB الفرعية عندما تكون بطيئة، وتكتشف الأنماط المضادة لتحميل البرنامج النصي (تعارضات المزامنة/التأجيل/التحميل المسبق)، وتحديد موارد حظر العرض، والتحقق من صحة استخدام تلميحات الموارد. يتميز بسير العمل لتدقيق التحميل الكامل (6 مراحل)، والتحقيق في أداء الواجهة الخلفية، وتحسين الأولويات. التكامل عبر المهارات مع Core Web Vitals (تحميل موارد LCP)، والتفاعل (حظر تنفيذ البرنامج النصي)، والوسائط (استراتيجية التحميل البطيء). يُستخدم عندما يسأل المستخدم عن TTFB، أو FCP، أو حظر العرض، أو التحميل البطيء، أو أداء الخط، أو تحسين البرنامج النصي، أو تلميحات الموارد. متوافق مع Chrome DevTools MCP.
