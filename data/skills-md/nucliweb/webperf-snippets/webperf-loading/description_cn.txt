智能加载性能分析，具有用于 TTFB 调查（DNS/连接/服务器故障）、渲染阻塞检测、脚本性能深入研究（第一与第三方归因）、字体优化和资源提示验证的自动化工作流程。包括决策树，可在速度缓慢时自动分析 TTFB 子部分、检测脚本加载反模式（异步/延迟/预加载冲突）、识别渲染阻塞资源并验证资源提示使用情况。具有完整加载审核（6 个阶段）、后端性能调查和优先级优化的工作流程。与 Core Web Vitals（LCP 资源加载）、Interaction（脚本执行阻止）和 Media（延迟加载策略）的跨技能集成。当用户询问 TTFB、FCP、渲染阻塞、加载缓慢、字体性能、脚本优化或资源提示时使用。与 Chrome DevTools MCP 兼容。
