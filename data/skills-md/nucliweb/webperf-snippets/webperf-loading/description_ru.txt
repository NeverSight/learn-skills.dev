Интеллектуальный анализ производительности загрузки с автоматизированными рабочими процессами для исследования TTFB (сбой DNS/соединения/сервера), обнаружения блокировки рендеринга, глубокого изучения производительности скриптов (первая или сторонняя атрибуция), оптимизации шрифтов и проверки подсказок ресурсов. Включает деревья решений, которые автоматически анализируют подчасти TTFB при медленной работе, обнаруживают антишаблоны загрузки скриптов (конфликты асинхронности, отсрочки и предварительной загрузки), определяют ресурсы, блокирующие рендеринг, и проверяют использование подсказок ресурсов. Содержит рабочие процессы для полного аудита загрузки (6 этапов), исследования производительности серверной части и оптимизации приоритетов. Межквалифицированная интеграция с Core Web Vitals (загрузка ресурсов LCP), Interaction (блокировка выполнения сценариев) и Media (стратегия отложенной загрузки). Используйте, когда пользователь спрашивает о TTFB, FCP, блокировке рендеринга, медленной загрузке, производительности шрифтов, оптимизации сценариев или подсказках по ресурсам. Совместим с Chrome DevTools MCP.
