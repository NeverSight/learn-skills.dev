智慧型載入效能分析，具有 TTFB 調查（DNS/連接/伺服器故障）、渲染阻塞偵測、腳本效能深入研究（第一與第三方歸因）、字體優化和資源提示驗證的自動化工作流程。包括決策樹，可在速度緩慢時自動分析 TTFB 子部分、偵測腳本載入反模式（非同步/延遲/預先載入衝突）、識別渲染阻塞資源並驗證資源提示使用情況。具有完整加載審核（6 個階段）、後端效能調查和優先優化的工作流程。與 Core Web Vitals（LCP 資源載入）、Interaction（腳本執行封鎖）和 Media（延遲載入策略）的跨技能整合。當使用者詢問 TTFB、FCP、渲染阻塞、載入緩慢、字體效能、腳本優化或資源提示時使用。與 Chrome DevTools MCP 相容。
