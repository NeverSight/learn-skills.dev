Интеллектуальный анализ качества сети с адаптивными стратегиями загрузки. Определяет тип соединения (2G/3G/4G), пропускную способность, RTT и режим сохранения данных, а затем автоматически запускает соответствующие рабочие процессы оптимизации. Включает деревья решений, которые рекомендуют сжатие изображений для медленных соединений, критическое встраивание CSS для высокого RTT и оптимизацию сохранения данных (отключить автозапуск, снизить качество). Содержит бюджеты производительности с учетом подключения (500 КБ для 2G, 1,5 МБ для 3G, 3 МБ для 4G+) и руководства по реализации адаптивной загрузки. Интеграция между навыками с загрузкой (влияние TTFB), мультимедиа (адаптивные изображения) и основными веб-показателями (влияние соединения на LCP/INP). Используйте, когда пользователь спрашивает о медленных соединениях, мобильной оптимизации, поддержке сохранения данных или стратегиях адаптивной загрузки. Совместим с Chrome DevTools MCP.
