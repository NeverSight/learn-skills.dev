تحليل ذكي لجودة الشبكة مع استراتيجيات التحميل التكيفية. يكتشف نوع الاتصال (2g/3g/4g)، وعرض النطاق الترددي، وRTT، ووضع حفظ البيانات، ثم يقوم تلقائيًا بتشغيل سير عمل التحسين المناسب. يتضمن أشجار القرار التي توصي بضغط الصور للاتصالات البطيئة، وتضمين CSS المهم لـ RTT العالية، وتحسينات حفظ البيانات (تعطيل التشغيل التلقائي، وتقليل الجودة). تتميز بميزانيات أداء مدركة للاتصال (500 كيلو بايت لشبكة 2g، و1.5 ميجا بايت لشبكة 3g، و3 ميجا بايت لشبكة 4g+) وأدلة تنفيذ التحميل التكيفية. التكامل عبر المهارات مع التحميل (تأثير TTFB)، والوسائط (الصور المستجيبة)، وعناصر الويب الأساسية (تأثير الاتصال على LCP/INP). يُستخدم عندما يسأل المستخدم عن الاتصالات البطيئة، أو تحسين الهاتف المحمول، أو دعم حفظ البيانات، أو إستراتيجيات التحميل التكيفية. متوافق مع Chrome DevTools MCP.
