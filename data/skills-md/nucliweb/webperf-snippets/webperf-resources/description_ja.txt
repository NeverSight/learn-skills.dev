適応負荷戦略によるインテリジェントなネットワーク品質分析。接続タイプ (2g/3g/4g)、帯域幅、RTT、およびデータ保存モードを検出し、適切な最適化ワークフローを自動的にトリガーします。低速接続に対する画像圧縮、高い RTT に対する重要な CSS インライン化、および保存データの最適化 (自動再生の無効化、品質の低下) を推奨するデシジョン ツリーが含まれています。接続を意識したパフォーマンス バジェット (2g の場合は 500KB、3g の場合は 1.5MB、4g+ の場合は 3MB) とアダプティブ ローディングの実装ガイドが特徴です。読み込み (TTFB への影響)、メディア (レスポンシブな画像)、およびコア Web バイタル (LCP/INP への接続への影響) とのクロススキル統合。ユーザーが低速接続、モバイルの最適化、セーブデータのサポート、またはアダプティブロード戦略について尋ねた場合に使用します。 Chrome DevTools MCP と互換性があります。
