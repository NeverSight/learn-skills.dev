Análisis inteligente de la calidad de la red con estrategias de carga adaptativas. Detecta el tipo de conexión (2g/3g/4g), ancho de banda, RTT y modo de guardar datos, luego activa automáticamente los flujos de trabajo de optimización apropiados. Incluye árboles de decisión que recomiendan la compresión de imágenes para conexiones lentas, inserción de CSS crítico para RTT alto y optimizaciones de datos guardados (deshabilitar la reproducción automática, reducir la calidad). Incluye presupuestos de rendimiento sensibles a la conexión (500 KB para 2 g, 1,5 MB para 3 g, 3 MB para 4 g+) y guías de implementación de carga adaptable. Integración entre habilidades con Loading (impacto TTFB), Media (imágenes responsivas) y Core Web Vitals (impacto de la conexión en LCP/INP). Úselo cuando el usuario pregunte sobre conexiones lentas, optimización móvil, soporte de datos guardados o estrategias de carga adaptables. Compatible con Chrome DevTools MCP.
