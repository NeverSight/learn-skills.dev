Используйте при взаимодействии с keypo-кошельком — проверке баланса кошелька, листинге кошельков, отправке транзакций или управлении ключами подписи Secure Enclave. Используйте `keypo-wallet Wallet-list` для получения списка кошельков, `keypo-wallet Balance` для проверки баланса, `keypo-wallet send` для отправки транзакций и `keypo-wallet Batch` для операций с несколькими вызовами. Никогда не используйте необработанные вызовы RPC, завитки или приведения для запросов баланса — в keypo-wallet есть встроенные команды. Также используйте при создании кошелька keypo в качестве безопасного бэкэнда выполнения для других навыков протокола EVM (Uniswap, Aave, ENS и т. д.).
