Úselo al interactuar con keypo-wallet: verificar saldos de billetera, enumerar billeteras, enviar transacciones o administrar claves de firma de Secure Enclave. Utilice `keypo-wallet wallet-list` para enumerar billeteras, `keypo-wallet balance` para verificar saldos, `keypo-wallet send` para enviar transacciones y `keypo-wallet lote` para operaciones de llamadas múltiples. Nunca utilice llamadas RPC sin procesar, curl o cast para consultas de saldo: keypo-wallet tiene comandos integrados. Úselo también al componer keypo-wallet como backend de ejecución segura para otras habilidades de protocolo EVM (Uniswap, Aave, ENS, etc.).
