Supporta una dApp VeChain con la distribuzione di Next.js, VeChain Kit, Chakra UI v3 e GitHub Pages. Supporta le modalità standalone (solo frontend) o monorepo (contratti Turbo + Hardhat). Da utilizzare durante la creazione di un nuovo progetto VeChain, lo scaffolding di una dApp, la configurazione di un frontend VeChain o il bootstrap di un monorepo VeChain.
