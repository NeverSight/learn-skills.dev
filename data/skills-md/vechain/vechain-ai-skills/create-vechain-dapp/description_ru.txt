Создайте децентрализованное приложение VeChain с помощью Next.js, VeChain Kit, Chakra UI v3 и развертывания страниц GitHub. Поддерживает автономный режим (только для внешнего интерфейса) или монорепозиторий (контракты Turbo + Hardhat). Используйте при создании нового проекта VeChain, создании децентрализованного приложения, настройке внешнего интерфейса VeChain или начальной загрузке монорепозитория VeChain.
