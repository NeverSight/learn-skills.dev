الكتابة اليدوية وتصحيح الأخطاء وتحويل وتحسين تظليل Unity ShaderLab/HLSL عبر URP وHDRP وRender Pipeline المدمج مع أهداف عبر الأنظمة الأساسية (الهاتف المحمول والكمبيوتر الشخصي ووحدة التحكم وWebGL). يُستخدم لتأليف التظليل، أو ترحيل خطوط الأنابيب، أو BRDF أو أعمال الإضاءة، أو ضبط أداء التظليل، أو ميزة العارض المخصص أو تكامل المرور المخصص، واستكشاف أخطاء التظليل وإصلاحها.
