Intégration du portefeuille d'agents FluxA via l'API REST. Permet aux agents d'effectuer des paiements x402 pour les API payantes, d'envoyer des paiements USDC vers n'importe quel portefeuille et de créer des liens de paiement pour recevoir des paiements. À utiliser lorsque l'utilisateur pose des questions sur les paiements cryptés, x402, les transferts USDC, les liens de paiement ou l'interaction avec le portefeuille d'agent FluxA via des appels API directs.
