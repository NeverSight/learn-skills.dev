FluxA Agent Wallet-Integration über REST API. Ermöglicht es Agenten, x402-Zahlungen für kostenpflichtige APIs zu tätigen, USDC-Auszahlungen an jede Wallet zu senden und Zahlungslinks zu erstellen, um Zahlungen zu empfangen. Verwenden Sie diese Option, wenn der Benutzer nach Krypto-Zahlungen, x402, USDC-Überweisungen, Zahlungslinks oder der Interaktion mit dem FluxA Agent Wallet über direkte API-Aufrufe fragt.
