Интеграция кошелька агента FluxA через REST API. Позволяет агентам совершать платежи x402 за платные API, отправлять выплаты в долларах США на любой кошелек и создавать платежные ссылки для получения платежей. Используйте, когда пользователь спрашивает о криптовалютных платежах, x402, переводах USDC, платежных ссылках или взаимодействии с кошельком агента FluxA через прямые вызовы API.
