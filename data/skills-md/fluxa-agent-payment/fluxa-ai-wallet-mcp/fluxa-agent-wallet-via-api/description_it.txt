Integrazione del portafoglio FluxA Agent tramite API REST. Consente agli agenti di effettuare pagamenti x402 per API a pagamento, inviare pagamenti USDC a qualsiasi portafoglio e creare collegamenti di pagamento per ricevere pagamenti. Da utilizzare quando l'utente richiede informazioni su pagamenti crittografici, x402, trasferimenti USDC, collegamenti di pagamento o interazione con FluxA Agent Wallet tramite chiamate API dirette.
