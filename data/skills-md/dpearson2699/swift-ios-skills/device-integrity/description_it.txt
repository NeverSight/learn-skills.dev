Verificare la legittimità del dispositivo e l'integrità dell'app utilizzando DeviceCheck (bit per dispositivo DCDevice) e App Attest (generazione di chiavi DCAppAttestService, flussi di attestazione e asserzione). Da utilizzare quando si implementa la prevenzione delle frodi, si rilevano dispositivi compromessi, si convalida l'autenticità delle app con i server Apple, si proteggono endpoint API sensibili con richieste attestate o si aggiunge la verifica del dispositivo all'architettura backend.
