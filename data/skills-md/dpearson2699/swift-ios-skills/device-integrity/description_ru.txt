Проверьте легитимность устройства и целостность приложения с помощью DeviceCheck (биты DCDevice для каждого устройства) и App Attest (потоки генерации ключей DCAppAttestService, аттестации и утверждения). Используйте при реализации предотвращения мошенничества, обнаружении скомпрометированных устройств, проверке подлинности приложений на серверах Apple, защите конфиденциальных конечных точек API с помощью подтвержденных запросов или добавлении проверки устройств в вашу серверную архитектуру.
