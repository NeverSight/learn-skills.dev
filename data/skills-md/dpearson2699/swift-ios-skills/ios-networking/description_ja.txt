URLSession と async/await、構造化された同時実行性、および最新の Swift パターンを使用して、iOS/macOS アプリのネットワーク コードを構築、レビュー、または改善します。 REST API の操作、ファイルのダウンロード、データのアップロード、WebSocket 接続、ページネーション、再試行ロジック、リクエスト ミドルウェア、キャッシュ、バックグラウンド転送、またはネットワーク到達可能性の監視を行う場合に使用します。 Swift アプリでの HTTP リクエスト、API クライアント、ネットワーク エラー処理、またはデータの取得に関連するタスクのトリガー。
