Apple の AlarmKit フレームワーク (iOS 26 以降 / iPadOS 26 以降) を使用して、アラームおよびカウントダウン タイマー機能を実装します。アラームとタイマーをスケジュールするための AlarmManager、ロック画面とダイナミック アイランド UI のための AlarmAttributes と AlarmPresentation、停止/スヌーズ アクション用の AlarmButton、承認フロー、アラーム状態の監視、およびライブ アクティビティの統合をカバーします。目覚ましアラーム、システム UI を備えたカウントダウン タイマー、またはロック画面やダイナミック アイランドに表示されるアラーム スタイルの通知を作成するときに使用します。
