Appliquez des modèles de langage et des idiomes Swift modernes pour le code non concurrent et non SwiftUI. Couvre les expressions if/switch (Swift 5.9+), les lancements typés (Swift 6+), les générateurs de résultats, les wrappers de propriétés, les types opaques et existentiels (certains ou tous), les modèles de garde, Ne jamais taper, les générateurs Regex (Swift 5.7+), les meilleures pratiques codables (CodingKeys, décodage personnalisé, conteneurs imbriqués), les API de collection modernes (count(where:), contient(where:), remplaçant()), FormatStyle (.formatted() sur les dates, les nombres, les mesures), et les modèles d'interpolation de chaînes. À utiliser lors de l'écriture de code Swift de base impliquant des génériques, des protocoles, des énumérations, des fermetures ou des fonctionnalités de langage moderne.
