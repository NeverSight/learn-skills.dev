Implemente funciones de visión por computadora que incluyen reconocimiento de texto (OCR), detección de rostros, escaneo de códigos de barras, segmentación de imágenes, seguimiento de objetos y escaneo de documentos en aplicaciones de iOS. Cubre tanto la moderna API Vision nativa de Swift (iOS 16+) como los patrones VNRequest heredados, VisionKit DataScannerViewController para escaneo de cámara en vivo y VNCoreMLRequest para inferencia de modelos personalizados. Úselo al agregar OCR, escaneo de códigos de barras, detección de rostros o inferencia de modelo Core ML personalizado con Vision.
