テキスト認識 (OCR)、顔検出、バーコード スキャン、画像セグメンテーション、オブジェクト追跡、ドキュメント スキャンなどのコンピューター ビジョン機能を iOS アプリに実装します。最新の Swift ネイティブ Vision API (iOS 16 以降) と従来の VNRequest パターン、ライブ カメラ スキャン用の VisionKit DataScannerViewController、およびカスタム モデル推論用の VNCoreMLRequest の両方をカバーします。 OCR、バーコード スキャン、顔検出、または Vision によるカスタム Core ML モデル推論を追加する場合に使用します。
