Внедряйте функции компьютерного зрения, включая распознавание текста (OCR), распознавание лиц, сканирование штрих-кодов, сегментацию изображений, отслеживание объектов и сканирование документов в приложениях iOS. Охватывает как современный Swift-родной Vision API (iOS 16+), так и устаревшие шаблоны VNRequest, VisionKit DataScannerViewController для сканирования с камеры в реальном времени и VNCoreMLRequest для вывода пользовательских моделей. Используйте при добавлении оптического распознавания символов, сканирования штрих-кода, распознавания лиц или вывода пользовательской модели Core ML с помощью Vision.
