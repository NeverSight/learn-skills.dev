قم بإنشاء أحداث التقويم والتذكيرات وقراءتها وإدارتها باستخدام EventKit وEventKitUI. يُستخدم عند إضافة أحداث إلى تقويم المستخدم، أو إنشاء تذكيرات، أو تعيين قواعد التكرار، أو طلب الوصول إلى التقويم أو التذكيرات، أو تقديم محرري الأحداث، أو اختيار التقويمات، أو التعامل مع التنبيهات، أو ملاحظة تغييرات التقويم، أو العمل مع طرق عرض EKEventStore أو EKEvent أو EKReminder أو EKCalendar أو EKRecurrenceRule أو EKEventEditViewController أو EKCalendarChooser أو EventKitUI.
