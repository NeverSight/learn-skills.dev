iOS セキュリティのベスト プラクティスには、キーチェーン サービス、CryptoKit 暗号化、Face ID および Touch ID による生体認証、Secure Enclave キー ストレージ、LAContext 構成、App Transport Security (ATS)、証明書の固定、データ保護クラス、セキュア コーディング パターン、暗号化の実装が含まれます。アプリのセキュリティ機能の実装、プライバシー マニフェストの監査、アプリ トランスポート セキュリティの構成、キーチェーン アクセスの保護、生体認証の追加、または CryptoKit による機密データの暗号化を行うときに使用します。
