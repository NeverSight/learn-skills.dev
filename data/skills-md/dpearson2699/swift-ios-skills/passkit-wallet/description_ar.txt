قم بدمج مدفوعات Apple Pay وبطاقات Wallet باستخدام PassKit. يُستخدم عند إضافة أزرار Apple Pay، أو إنشاء طلبات الدفع، أو التعامل مع تفويض الدفع، أو إضافة تصاريح إلى Wallet، أو تكوين إمكانات التاجر، أو إدارة حقول الشحن والاتصال، أو العمل مع تدفقات الدفع PKPaymentRequest أو PKPaymentAuthorizationController أو PKPaymentButton أو PKPass أو PKAddPassesViewController أو PKPassLibrary أو Apple Pay.
