قم بالمسح الضوئي والاتصال والتواصل مع الأجهزة الطرفية التي تعمل بتقنية Bluetooth منخفضة الطاقة ونشر الخدمات الطرفية المحلية باستخدام Core Bluetooth. يُستخدم عند تنفيذ الأدوار المركزية أو الطرفية لـ BLE، واكتشاف الخدمات والخصائص، وقراءة وكتابة القيم المميزة، والاشتراك في الإشعارات، وتكوين أوضاع BLE الخلفية، واستعادة الحالة بعد إعادة تشغيل التطبيق، أو العمل مع CBCentralManager، أو CBPeripheral، أو CBPeripheralManager، أو CBService، أو CBCharacteristic، أو CBUUID، أو سير عمل Bluetooth منخفض الطاقة.
