Внедряйте, проверяйте или улучшайте сохранение данных с помощью SwiftData. Используйте при определении классов @Model с помощью @Attribute, @Relationship, @Transient, @Unique или @Index; при запросе с помощью @Query, #Predicate, FetchDescriptor или SortDescriptor; при настройке ModelContainer и ModelContext для SwiftUI или фоновой работы с @ModelActor; при планировании миграции схемы с помощью VersionedSchema и SchemaMigrationPlan; при настройке синхронизации CloudKit с ModelConfiguration; или при сосуществовании с Core Data или миграции из него.
