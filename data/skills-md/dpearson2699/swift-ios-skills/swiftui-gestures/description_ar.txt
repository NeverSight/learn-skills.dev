تنفيذ أو مراجعة أو تحسين التعامل مع إيماءات SwiftUI. يُستخدم عند إضافة إيماءات النقر أو الضغط لفترة طويلة أو السحب أو التكبير أو التدوير، أو إنشاء الإيماءات بشكل متزامن/متسلسل/حصريًا، أو إدارة الحالة العابرة باستخدام @GestureState، أو حل تعارضات إيماءات الوالدين/الطفل باستخدام إيماءات ذات أولوية عالية أو إيماءات متزامنة، أو إنشاء توافقات مخصصة لبروتوكول الإيماءات، أو الترحيل من MagnificationGesture/RotationGesture المهملة إلى تكبير الإيماءة/تدوير الإيماءة.
