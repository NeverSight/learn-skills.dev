Apple로 로그인(ASAuthorizationAppleIDProvider, ASAuthorizationController, ASAuthorizationAppleIDCredential), 자격 증명 상태 확인, ID 토큰 유효성 검사, OAuth 및 타사 인증 흐름을 위한 ASWebAuthenticationSession, 자동 완성 자격 증명 제안을 위한 ASAuthorizationPasswordProvider, LAContext를 사용한 생체 인식 인증을 포함한 iOS 인증 패턴을 구현합니다. Apple로 로그인 구현, Apple ID 자격 증명 처리, OAuth 로그인 흐름 구축, 비밀번호 자동 완성 통합, 자격 증명 취소 상태 확인 또는 서버 측 ID 토큰 검증 시 사용합니다.
