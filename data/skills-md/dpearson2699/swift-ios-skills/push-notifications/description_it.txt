Implementa, rivedi o esegui il debug delle notifiche push nelle app iOS/macOS: notifiche locali, notifiche remote (APN), notifiche avanzate, azioni di notifica, push silenziosi ed estensioni di contenuti/servizi di notifica. Da utilizzare quando si lavora con UNUserNotificationCenter, si registra per notifiche remote, si gestiscono payload di notifiche, si impostano categorie e azioni di notifica, si crea contenuto di notifica avanzato o si esegue il debug della consegna delle notifiche. Attiva qualsiasi attività che coinvolga avvisi, badge, suoni, push in background o autorizzazioni di notifica utente nelle app Swift.
