Транскрибируйте речь в текст с помощью платформы Speech. Используйте при реализации транскрипции живого микрофона с помощью AVAudioEngine, распознавании предварительно записанных аудиофайлов, настройке распознавания на устройстве или на сервере, обработке потоков авторизации или использовании нового SpeechAnalyzer API (iOS 26+) для современного асинхронного/ожидающего преобразования речи в текст.
