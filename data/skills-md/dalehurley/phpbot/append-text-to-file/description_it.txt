Aggiungi o aggiungi nuovo contenuto testuale a un file esistente. Utilizza questa abilità quando l'utente chiede di aggiornare un file, aggiungere contenuto, aggiungere testo, aggiungere un versetto, aggiungere una sezione o estendere un documento con materiale aggiuntivo. Supporta qualsiasi formato di file basato su testo, inclusi .txt, .md, .py, .json e altri.
