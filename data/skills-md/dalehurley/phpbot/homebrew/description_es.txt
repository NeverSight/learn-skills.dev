Instale, administre y busque paquetes de software en macOS usando Homebrew. Utilice esta habilidad cuando el usuario solicite instalar software, aplicaciones, herramientas CLI, utilidades de desarrollador, lenguajes de programación, bases de datos o cualquier paquete en una Mac. Admite fórmulas (herramientas CLI) y barriles (aplicaciones GUI). También puede buscar, actualizar, mejorar, desinstalar y diagnosticar problemas de Homebrew.
