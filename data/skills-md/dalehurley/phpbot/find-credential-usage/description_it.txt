Cerca nell'intero sistema per trovare tutte le istanze in cui è stata utilizzata o archiviata una chiave API, un token o una credenziale specifica. Utilizza questa capacità quando l'utente chiede di individuare, controllare o tenere traccia della posizione in cui viene visualizzata una credenziale tra file, variabili di ambiente, file di configurazione, registri o cronologia della shell. Aiuta a identificare i rischi per la sicurezza e l'esposizione delle credenziali.
