Busque en todo su sistema para encontrar todas las instancias en las que se haya utilizado o almacenado una clave, token o credencial API específica. Utilice esta habilidad cuando el usuario solicite ubicar, auditar o rastrear dónde aparece una credencial en archivos, variables de entorno, archivos de configuración, registros o historial de shell. Ayuda a identificar riesgos de seguridad y exposición de credenciales.
