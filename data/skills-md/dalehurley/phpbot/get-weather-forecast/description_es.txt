Recupere las condiciones meteorológicas actuales y los pronósticos de varios días para cualquier ubicación utilizando la API wttr.in. Utilice esta habilidad cuando el usuario solicite información meteorológica, pronóstico del tiempo, condiciones actuales, temperatura o actualizaciones del tiempo para una ciudad o ubicación específica. Proporciona datos meteorológicos detallados que incluyen temperatura, viento, precipitación y visibilidad.
