Получайте текущие погодные условия и многодневные прогнозы для любого места с помощью API wttr.in. Используйте этот навык, когда пользователь запрашивает информацию о погоде, прогноз погоды, текущие условия, температуру или обновления погоды для определенного города или места. Предоставляет подробные данные о погоде, включая температуру, ветер, осадки и видимость.
