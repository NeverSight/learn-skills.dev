Analizzare e riepilogare i file di configurazione di PHP Composer (composer.json). Utilizza questa capacità quando l'utente chiede di riepilogare, rivedere, analizzare o comprendere un file compositer.json. Estrae e spiega metadati, dipendenze, requisiti, script e impostazioni di configurazione del progetto in un formato chiaro e strutturato.
