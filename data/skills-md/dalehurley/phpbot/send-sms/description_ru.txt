Отправляйте текстовые сообщения SMS на любой номер телефона с помощью API Twilio. Используйте этот навык, когда пользователь просит отправить текстовое сообщение, SMS, текстовое сообщение или уведомить кого-либо по телефону. Поддерживает настраиваемое содержимое сообщения и любой номер получателя в формате E.164.
