Converti qualsiasi documento o file in Markdown utilizzando la libreria MarkItDown di Microsoft. Utilizza questa capacità quando l'utente chiede di convertire un file in markdown, estrarre testo da un documento, trasformare un file PDF/Word/Excel/PowerPoint/HTML/immagine/audio in markdown o analizzare il contenuto del documento. Supporta PDF, DOCX, PPTX, XLSX, HTML, immagini (con OCR), audio, CSV, JSON, XML, ZIP, EPub e altro.
