Konvertieren Sie jedes Dokument oder jede Datei mithilfe der MarkItDown-Bibliothek von Microsoft in Markdown. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine Datei in Markdown konvertieren, Text aus einem Dokument extrahieren, eine PDF-/Word-/Excel-/PowerPoint-/HTML-/Bild-/Audiodatei in Markdown umwandeln oder Dokumentinhalte analysieren möchte. Unterstützt PDF, DOCX, PPTX, XLSX, HTML, Bilder (mit OCR), Audio, CSV, JSON, XML, ZIP, EPub und mehr.
