Aggiorna o modifica file di testo (poesie, documenti, note, ecc.) in qualsiasi directory. Utilizza questa abilità quando l'utente chiede di modificare, aggiornare, modificare, rivedere o riscrivere un file di testo, una poesia, un documento o un contenuto simile. Supporta l'aggiunta di nuovi contenuti, l'espansione del testo esistente o la sostituzione del contenuto dei file preservando la posizione originale.
