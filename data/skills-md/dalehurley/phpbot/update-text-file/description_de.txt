Aktualisieren oder ändern Sie Textdateien (Gedichte, Dokumente, Notizen usw.) in jedem Verzeichnis. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine Textdatei, ein Gedicht, ein Dokument oder einen ähnlichen Inhalt bearbeiten, aktualisieren, modifizieren, überarbeiten oder neu schreiben möchte. Unterstützt das Hinzufügen neuer Inhalte, das Erweitern vorhandener Texte oder das Ersetzen von Dateiinhalten unter Beibehaltung des ursprünglichen Speicherorts.
