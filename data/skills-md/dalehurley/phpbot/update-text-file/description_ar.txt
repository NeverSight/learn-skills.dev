تحديث أو تعديل الملفات النصية (قصائد، وثائق، ملاحظات، الخ) في أي دليل. استخدم هذه المهارة عندما يطلب المستخدم تحرير أو تحديث أو تعديل أو مراجعة أو إعادة كتابة ملف نصي أو قصيدة أو مستند أو محتوى مشابه. يدعم إضافة محتوى جديد، أو توسيع النص الموجود، أو استبدال محتويات الملف مع الحفاظ على الموقع الأصلي.
