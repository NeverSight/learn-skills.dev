Actualiza o modifica archivos de texto (poemas, documentos, notas, etc.) en cualquier directorio. Utilice esta habilidad cuando el usuario solicite editar, actualizar, modificar, revisar o reescribir un archivo de texto, poema, documento o contenido similar. Admite agregar contenido nuevo, expandir texto existente o reemplazar el contenido del archivo conservando la ubicación original.
