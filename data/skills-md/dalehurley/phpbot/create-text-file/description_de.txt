Erstellen und schreiben Sie Textinhalte in eine Datei in einem angegebenen Verzeichnis. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine Datei erstellen, Inhalte in eine Datei schreiben, Text speichern, ein Dokument erstellen oder ein Gedicht, eine Geschichte, ein Skript oder einen anderen textbasierten Inhalt auf die Festplatte schreiben möchte. Unterstützt jedes Textformat und benutzerdefinierte Dateipfade.
