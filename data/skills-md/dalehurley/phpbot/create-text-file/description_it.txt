Crea e scrivi contenuto di testo in un file in una directory specificata. Utilizza questa capacità quando l'utente chiede di creare un file, scrivere contenuto in un file, salvare testo, generare un documento o scrivere una poesia, una storia, una sceneggiatura o qualsiasi contenuto basato su testo su disco. Supporta qualsiasi formato di testo e percorsi di file personalizzati.
