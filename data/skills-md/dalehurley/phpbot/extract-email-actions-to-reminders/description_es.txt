Analice correos electrónicos recientes para identificar elementos de acción y cree automáticamente recordatorios priorizados en la aplicación Recordatorios. Utilice esta habilidad cuando el usuario solicite convertir correos electrónicos en tareas pendientes, crear recordatorios de correo electrónico, organizar tareas de correo electrónico o crear una lista de tareas a partir de mensajes de la bandeja de entrada. Busca palabras clave de acción como "pagar", "revisión", "urgente", "fecha límite", "factura" y "aprobación" para extraer de forma inteligente elementos procesables.
