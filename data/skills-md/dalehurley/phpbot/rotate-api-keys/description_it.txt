Sostituisci le vecchie chiavi API con quelle nuove su più file di configurazione, archivi chiavi e log. Utilizza questa abilità quando l'utente chiede di ruotare, sostituire, aggiornare o rinfrescare le chiavi API nel proprio sistema. Supporta file .env, archivi chiavi JSON e file di registro. Crea automaticamente i backup prima di apportare modifiche.
