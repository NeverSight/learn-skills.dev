Remplacez les anciennes clés API par de nouvelles dans plusieurs fichiers de configuration, magasins de clés et journaux. Utilisez cette compétence lorsque l'utilisateur demande à faire pivoter, remplacer, mettre à jour ou actualiser les clés API sur son système. Prend en charge les fichiers .env, les magasins de clés JSON et les fichiers journaux. Crée automatiquement des sauvegardes avant d'apporter des modifications.
