Замените старые ключи API новыми в нескольких файлах конфигурации, хранилищах ключей и журналах. Используйте этот навык, когда пользователь запрашивает ротацию, замену, обновление или обновление ключей API в своей системе. Поддерживает файлы .env, хранилища ключей JSON и файлы журналов. Автоматически создает резервные копии перед внесением изменений.
