قم بمراجعة رمز التزامن Java للتأكد من سلامة سلسلة المحادثات وظروف السباق والحالات المسدودة والأنماط الحديثة (Virtual Threads وCompletableFuture وAsync). يُستخدم عندما يسأل المستخدم "التحقق من سلامة سلسلة الرسائل"، أو "مراجعة التزامن"، أو "مراجعة التعليمات البرمجية غير المتزامنة"، أو عند مراجعة التعليمات البرمجية متعددة الخيوط.
