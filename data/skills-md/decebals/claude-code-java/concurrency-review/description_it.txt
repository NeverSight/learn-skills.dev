Esamina il codice di concorrenza Java per la sicurezza dei thread, le condizioni di competizione, i deadlock e i modelli moderni (thread virtuali, CompletableFuture, @Async). Da utilizzare quando l'utente chiede "verifica la sicurezza del thread", "revisione della concorrenza", "revisione del codice asincrono" o durante la revisione del codice multi-thread.
