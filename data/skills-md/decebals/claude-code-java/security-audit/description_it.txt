Elenco di controllo della sicurezza Java che copre OWASP Top 10, convalida dell'input, prevenzione dell'iniezione e codifica sicura. Funziona con Spring, Quarkus, Jakarta EE e Java semplice. Da utilizzare durante la revisione della sicurezza del codice, prima dei rilasci o quando l'utente richiede informazioni sulle vulnerabilità.
