Контрольный список безопасности Java, охватывающий 10 лучших OWASP, проверку ввода, предотвращение внедрения и безопасное кодирование. Работает со Spring, Quarkus, Jakarta EE и простой Java. Используйте при проверке безопасности кода, перед выпуском или когда пользователь спрашивает об уязвимостях.
