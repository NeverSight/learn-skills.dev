Analizza l'architettura del progetto Java a livello macro: struttura del pacchetto, limiti dei moduli, direzione delle dipendenze e stratificazione. Da utilizzare quando l'utente chiede "revisiona l'architettura", "controlla la struttura", "l'organizzazione del pacchetto" o quando valuta se una codebase segue i principi dell'architettura pulita.
