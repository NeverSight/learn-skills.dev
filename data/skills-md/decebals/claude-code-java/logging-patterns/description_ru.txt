Лучшие практики ведения журнала Java с использованием SLF4J, структурированного ведения журнала (JSON) и MDC для отслеживания запросов. Включает удобные для искусственного интеллекта форматы журналов для отладки кода Claude. Используйте, когда пользователь спрашивает о ведении журнала, отладке потока приложения или анализе журналов.
