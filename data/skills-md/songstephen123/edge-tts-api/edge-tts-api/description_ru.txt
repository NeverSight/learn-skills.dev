Используйте этот навык всякий раз, когда вы хотите развернуть локальную службу API TTS с помощью Microsoft Edge TTS (совершенно бесплатно, ключ API не требуется). Триггеры включают в себя: «развертывание службы tts», «сервер API tts», «самостоятельный tts», «пограничную службу tts» или запросы на настройку API преобразования текста в речь для Feishu, DingTalk или других приложений.
