Integrationsleitfaden für das Bloque SDK – ein TypeScript SDK für programmierbare Finanzkonten, Karten mit Ausgabenkontrolle und Multi-Asset-Übertragungen. Wird verwendet, wenn der Benutzer „Bloque integrieren“, „eine Karte erstellen“, „Ausgabenkontrollen einrichten“, „Karten-Webhooks verwalten“, „Gelder überweisen“, „Taschen erstellen“, „MCC-Routing einrichten“, „Guthaben zwischen beliebigen Medien teilen“, „gleiches Guthaben für Taschen/Karten/Polygon/Bank“ oder eine beliebige Fintech-Funktion auf der Bloque-Plattform erstellen möchte.
