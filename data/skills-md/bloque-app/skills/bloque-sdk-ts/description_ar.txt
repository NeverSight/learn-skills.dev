دليل التكامل لـ Bloque SDK — TypeScript SDK للحسابات المالية القابلة للبرمجة، والبطاقات التي تحتوي على ضوابط الإنفاق، والتحويلات متعددة الأصول. يُستخدم عندما يطلب المستخدم "دمج Bloque"، أو "إنشاء بطاقة"، أو "إعداد ضوابط الإنفاق"، أو "التعامل مع خطافات الويب للبطاقة"، أو "تحويل الأموال"، أو "إنشاء جيوب"، أو "إعداد توجيه مركز عملائي"، أو "مشاركة الأرصدة بين أي وسائط"، أو "نفس الرصيد عبر الجيوب/البطاقات/المضلع/البنك"، أو إنشاء أي ميزة للتكنولوجيا المالية على منصة Bloque.
