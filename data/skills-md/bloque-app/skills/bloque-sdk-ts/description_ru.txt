Руководство по интеграции Bloque SDK — TypeScript SDK для программируемых финансовых счетов, карт с контролем расходов и переводов нескольких активов. Используйте, когда пользователь просит «интегрировать Bloque», «создать карту», ​​«настроить контроль расходов», «обработать веб-перехватчики карт», «перевести средства», «создать карманы», «настроить маршрутизацию MCC», «распределить балансы между любыми средами», «одинаковый баланс между карманами/картами/Polygon/банком» или создать любую финтех-функцию на платформе Bloque.
