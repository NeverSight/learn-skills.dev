Guida per l'interrogazione dei database tramite il server DBHub MCP. Utilizza questa competenza ogni volta che devi esplorare schemi di database, ispezionare tabelle o eseguire query SQL tramite gli strumenti MCP di DBHub (search_objects,execute_sql). Si attiva su qualsiasi attività di query del database, esplorazione dello schema, recupero dei dati o esecuzione SQL tramite MCP, anche se l'utente dice semplicemente "controlla il database" o "trovami dei dati". Questa abilità ti assicura di seguire il corretto flusso di lavoro esplora prima invece di indovinare le strutture delle tabelle.
