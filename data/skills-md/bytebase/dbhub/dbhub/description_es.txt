Guía para consultar bases de datos a través del servidor DBHub MCP. Utilice esta habilidad siempre que necesite explorar esquemas de bases de datos, inspeccionar tablas o ejecutar consultas SQL a través de las herramientas MCP de DBHub (search_objects, ejecutar_sql). Se activa en cualquier tarea de consulta de base de datos, exploración de esquemas, recuperación de datos o ejecución de SQL a través de MCP, incluso si el usuario simplemente dice "revise la base de datos" o "búsqueme algunos datos". Esta habilidad garantiza que siga el flujo de trabajo correcto de exploración primero en lugar de adivinar las estructuras de las tablas.
