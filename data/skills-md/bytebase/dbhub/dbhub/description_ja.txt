DBHub MCP サーバーを介してデータベースにクエリを実行するためのガイド。 DBHub の MCP ツール (search_objects、execute_sql) を介してデータベース スキーマの調査、テーブルの検査、または SQL クエリの実行が必要な場合は、このスキルをいつでも使用してください。ユーザーが単に「データベースをチェックして」または「データを見つけて」と言った場合でも、MCP を介したデータベース クエリ タスク、スキーマ探索、データ取得、または SQL 実行でアクティブ化されます。このスキルにより、テーブル構造を推測するのではなく、正しい探索優先ワークフローに確実に従うことができます。
