DBHub MCP 서버를 통한 데이터베이스 쿼리 가이드입니다. DBHub의 MCP 도구(search_objects,execute_sql)를 통해 데이터베이스 스키마를 탐색하거나, 테이블을 검사하거나, SQL 쿼리를 실행해야 할 때마다 이 기술을 사용하세요. 사용자가 "데이터베이스를 확인하세요" 또는 "데이터를 좀 찾아주세요"라고만 말하는 경우에도 MCP를 통해 모든 데이터베이스 쿼리 작업, 스키마 탐색, 데이터 검색 또는 SQL 실행을 활성화합니다. 이 기술을 사용하면 테이블 구조를 추측하는 대신 올바른 탐색 우선 작업 흐름을 따를 수 있습니다.
