Guide pour interroger les bases de données via le serveur DBHub MCP. Utilisez cette compétence chaque fois que vous avez besoin d'explorer des schémas de base de données, d'inspecter des tables ou d'exécuter des requêtes SQL via les outils MCP de DBHub (search_objects,execute_sql). S'active sur n'importe quelle tâche de requête de base de données, d'exploration de schéma, de récupération de données ou d'exécution SQL via MCP, même si l'utilisateur dit simplement « vérifier la base de données » ou « trouver des données ». Cette compétence garantit que vous suivez le bon flux de travail d'exploration en premier au lieu de deviner les structures des tableaux.
