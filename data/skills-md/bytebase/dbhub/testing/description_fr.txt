Exécutez et dépannez des tests pour DBHub, y compris des tests unitaires, des tests d'intégration avec Testcontainers et des tests spécifiques à la base de données. À utiliser lorsqu'on vous demande d'exécuter des tests, de corriger les échecs des tests, de déboguer les tests d'intégration, de résoudre les problèmes de Docker/conteneur de base de données ou d'ajouter de nouveaux tests. À utiliser également pour vérifier le bon fonctionnement des modifications de code ou lorsque les échecs des tests CI nécessitent une enquête.
