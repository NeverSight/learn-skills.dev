تبسيط كود بايثون المعقد للغاية. يُستخدم عندما يطلب المستخدم التبسيط، أو إعادة البناء، أو التنظيف، أو جعل القراءة أكثر سهولة، أو تقليل التعقيد، أو تحسين جودة التعليمات البرمجية، أو العثور على روائح التعليمات البرمجية، أو اكتشاف التكرارات، أو تحليل الاقتران في تعليمات Python البرمجية. مشغلات لطلبات مثل "تبسيط هذا الرمز"، أو "هذا معقد للغاية"، أو "جعل هذا أكثر قابلية للقراءة"، أو "إعادة صياغة هذا"، أو "تنظيف هذا"، أو "العثور على مشكلات"، أو "تحليل قاعدة التعليمات البرمجية هذه"، أو عند مراجعة التعليمات البرمجية التي تعرض أنماطًا مضادة للتعقيد. بالنسبة للتحليل الخاص بـ Django، استخدم مهارة Django-simplifier بدلاً من ذلك.
