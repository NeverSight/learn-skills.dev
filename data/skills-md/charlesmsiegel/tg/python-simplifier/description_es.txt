Simplifique el código Python demasiado complejo. Úselo cuando el usuario solicite simplificar, refactorizar, limpiar, hacerlo más legible, reducir la complejidad, mejorar la calidad del código, encontrar olores de código, detectar duplicados o analizar el acoplamiento en el código Python. Se activan solicitudes como "simplificar este código", "esto es demasiado complejo", "hacerlo más legible", "refactorizar esto", "limpiar esto", "encontrar problemas", "analizar esta base de código" o al revisar código que exhibe antipatrones de complejidad. Para análisis específicos de Django, utilice la habilidad django-simplifier en su lugar.
