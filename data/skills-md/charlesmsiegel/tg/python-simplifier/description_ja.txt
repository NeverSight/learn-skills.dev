複雑すぎる Python コードを簡素化します。ユーザーが、Python コードの単純化、リファクタリング、クリーンアップ、読みやすさの向上、複雑さの軽減、コードの品質の向上、コードの匂いの検出、重複の検出、または結合の分析を要求する場合に使用します。 「このコードを単純化する」、「これは複雑すぎる」、「これを読みやすくする」、「これをリファクタリングする」、「これをクリーンアップする」、「問題を見つける」、「このコードベースを分析する」などのリクエスト、または複雑さのアンチパターンを示すコードをレビューするときにトリガーされます。 Django 固有の分析の場合は、代わりに django-simplifier スキルを使用してください。
