Django-Entwicklungsstandards für die Tellurium Games World of Darkness-Anwendung. Verwendung bei: Erstellen/Bearbeiten von Modellen (Charaktere, Elemente, Orte), Implementieren von CRUD-Ansichten, Erstellen von Formularen, Erstellen von Vorlagen, Konfigurieren von URLs, Arbeiten mit Berechtigungen, Caching, Migrationen, Tests, Validierung, Bereitstellung oder Verwaltungsbefehlen. Löst aus: Modellvererbungsmuster, polymorphe Modelle, Ansichtsmixins, spiellinienspezifischer Code (Vampir/Werwolf/Magier/Geist/Changeling/Dämon/Mama/Jäger), TG-Kartenkomponenten, XP/Freebie-Transaktionen, ST-Berechtigungen, Charaktervorlagen, Frontend-Styling.
