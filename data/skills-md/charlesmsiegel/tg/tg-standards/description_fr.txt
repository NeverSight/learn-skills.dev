Normes de développement Django pour l'application Tellurium Games World of Darkness. À utiliser lors de : la création/modification de modèles (personnages, éléments, emplacements), la mise en œuvre de vues CRUD, la création de formulaires, la création de modèles, la configuration d'URL, l'utilisation d'autorisations, la mise en cache, les migrations, les tests, la validation, le déploiement ou les commandes de gestion. Déclencheurs sur : les modèles d'héritage de modèles, les modèles polymorphes, les mixins de vues, le code spécifique à la ligne de jeu (vampire/loup-garou/mage/wraith/changeling/démon/momie/hunter), les composants de la carte tg, les transactions XP/freebie, les autorisations ST, les modèles de personnages, le style frontal.
