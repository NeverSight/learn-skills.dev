تبسيط كود Django المعقد للغاية واكتشاف الأنماط المضادة الخاصة بـ Django. يُستخدم عندما يطلب المستخدم تبسيط تعليمات Django البرمجية أو إعادة هيكلتها أو تحسينها، أو العثور على استعلامات N+1، أو اكتشاف أنماط Django المضادة، أو تحسين مجموعات الاستعلام، أو تحليل بنية مشروع Django. يتم تشغيل الطلبات التي تتضمن نماذج Django، أو طرق العرض، أو QuerySets، أو تحسين ORM، أو المتسلسلات، أو النماذج، أو الإشارات، أو البرامج الوسيطة، أو أفضل ممارسات Django. لتحليل بايثون العام، استخدم مهارة تبسيط بايثون بدلاً من ذلك.
