Simplifiez le code Django trop complexe et détectez les anti-modèles spécifiques à Django. À utiliser lorsque l'utilisateur demande de simplifier, refactoriser ou améliorer le code Django, rechercher des requêtes N+1, détecter les anti-modèles Django, optimiser les QuerySets ou analyser l'architecture du projet Django. Se déclenche sur des requêtes impliquant des modèles Django, des vues, des QuerySets, l'optimisation ORM, des sérialiseurs, des formulaires, des signaux, des middlewares ou les meilleures pratiques Django. Pour une analyse Python générale, utilisez plutôt la compétence python-simplifier.
