Работа смарт-кошелька Pillar осуществляется в двух режимах: передача через браузер (pillar.ts) открывает интерфейс Pillar для подписи пользователя, а прямая подпись агента (pillar-direct.ts) подписывает локально с помощью пары ключей secp256k1 и отправляет данные непосредственно в API Pillar (браузер не требуется, спонсируется Gas). Поддерживает отправку/поставку/повышение/размотку sBTC, программы DCA, стекирование, управление ключами, создание кошелька и запросы позиции.
