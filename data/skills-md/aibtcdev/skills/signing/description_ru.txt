Подписание и проверка сообщений — подписание структурированных данных Clarity SIP-018 (проверяемое в цепочке), подписание простых текстовых сообщений Stacks (совместимо с SIWS), подписание биткойн-сообщений (BIP-137 для устаревшего/обернутого SegWit, BIP-322 для собственного SegWit bc1q и Taproot bc1p), BIP-340 подписание Шнорра для Taproot multisig и подписание событий Nostr с использованием Получение ключа NIP-06. Для всех подписей требуется разблокированный кошелек; операции хеширования и проверки этого не делают.
