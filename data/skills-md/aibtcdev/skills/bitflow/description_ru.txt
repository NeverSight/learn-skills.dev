Bitflow DEX on Stacks — унифицированное ранжирование маршрутов SDK и котировок HODLMM, обмен токенов, данные рыночных тикеров, проверка корзин HODLMM и управление ликвидностью, анализ влияния на цену и автоматизация Keeper для запланированных заказов. Все операции выполняются только в основной сети. Во время бета-тестирования для общедоступных маршрутов не требуется ключ API. Для операций записи требуется разблокированный кошелек.
