Bitflow DEX on Stacks: clasificación de ruta unificada entre rutas SDK y cotizaciones de HODLMM, intercambios de tokens, datos de cotización de mercado, inspección de contenedores de HODLMM y gestión de liquidez, análisis de impacto de precios y automatización de Keeper para órdenes programadas. Todas las operaciones son únicamente para la red principal. No se requiere clave API para rutas públicas durante la versión beta. Las operaciones de escritura requieren una billetera desbloqueada.
