Bitflow DEX sur Stacks : classement des itinéraires unifiés sur les itinéraires SDK et les cotations HODLMM, échanges de jetons, données de ticker de marché, inspection des bacs HODLMM et gestion des liquidités, analyse de l'impact des prix et automatisation Keeper pour les commandes planifiées. Toutes les opérations concernent uniquement le réseau principal. Aucune clé API requise pour les routes publiques pendant la version bêta. Les opérations d'écriture nécessitent un portefeuille déverrouillé.
