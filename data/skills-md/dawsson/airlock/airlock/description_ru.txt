Локальный сервер обновлений Expo OTA. Airlock — это библиотека Hono, которую вы подключаете к существующему API для обслуживания манифестов протокола Expo Updates v1. Используйте этот навык при работе с сервером шлюза, интерфейсом командной строки, адаптерами или при интеграции шлюза в приложение Expo.
