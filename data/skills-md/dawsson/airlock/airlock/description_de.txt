Selbstgehosteter Expo OTA-Update-Server. Airlock ist eine Hono-Bibliothek, die Sie auf Ihrer vorhandenen API bereitstellen, um Manifeste des Expo Updates-Protokolls v1 bereitzustellen. Nutzen Sie diesen Skill, wenn Sie am Airlock-Server, der CLI, den Adaptern arbeiten oder Airlock in eine Expo-App integrieren.
