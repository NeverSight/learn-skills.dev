Комплексная библиотека компонентов пользовательского интерфейса на основе Tailwind CSS и shadcn/ui. Используйте этот навык, когда пользователь просит создать или стилизовать пользовательский интерфейс с помощью библиотеки @kenshinx/ui или когда вам нужен доступ к его компонентам React (например, кнопкам, формам, диалогам).
