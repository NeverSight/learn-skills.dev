Bibliothèque complète de composants d'interface utilisateur basée sur Tailwind CSS et shadcn/ui. Utilisez cette compétence lorsque l'utilisateur demande à créer ou à styliser une interface utilisateur à l'aide de la bibliothèque @kenshinx/ui, ou lorsque vous avez besoin d'accéder à ses composants React (par exemple, bouton, formulaire, boîte de dialogue).
