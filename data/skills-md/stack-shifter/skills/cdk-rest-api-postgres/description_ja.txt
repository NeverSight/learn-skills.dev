最初にターゲット リポジトリの CDK パターンを使用し、デフォルト データストアとして AWS Lambda ハンドラーと Drizzle 経由の Postgres を使用して、AWS 上で REST API を設計および実装します。ユーザーが API Gateway REST エンドポイント、Lambda ハンドラー、Cognito 認証、リクエスト モデル、再利用可能な CDK コンストラクト、Drizzle スキーマ、SQL リポジトリ、またはデータベースにバックアップされた CRUD ルートを追加または変更する場合は常に、「エンドポイント」、「ハンドラー」、「テーブル」、「リポジトリ」、または「一部の CDK ワイヤリング」のみを要求する場合でも、これを使用します。
