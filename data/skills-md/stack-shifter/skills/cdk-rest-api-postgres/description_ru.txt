Разрабатывает и реализует REST API на AWS, сначала используя шаблоны CDK целевого репозитория, с обработчиками AWS Lambda и Postgres через Drizzle в качестве хранилища данных по умолчанию. Используйте это всякий раз, когда пользователь хочет добавить или изменить конечные точки REST API-шлюза, обработчики Lambda, аутентификацию Cognito, модели запросов, повторно используемые конструкции CDK, схему Drizzle, репозитории SQL или маршруты CRUD на основе базы данных, даже если они запрашивают только «конечную точку», «обработчик», «таблицу», «репозиторий» или «какую-то проводку CDK».
