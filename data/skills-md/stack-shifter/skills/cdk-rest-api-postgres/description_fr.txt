Conçoit et implémente des API REST sur AWS en utilisant d'abord les modèles CDK du référentiel cible, avec les gestionnaires AWS Lambda et Postgres via Drizzle comme banque de données par défaut. Utilisez-le chaque fois que l'utilisateur souhaite ajouter ou modifier des points de terminaison REST d'API Gateway, des gestionnaires Lambda, l'authentification Cognito, des modèles de requête, des constructions CDK réutilisables, un schéma Drizzle, des référentiels SQL ou des routes CRUD basées sur une base de données, même s'ils ne demandent que « un point de terminaison », « un gestionnaire », « une table », « un référentiel » ou « un câblage CDK ».
