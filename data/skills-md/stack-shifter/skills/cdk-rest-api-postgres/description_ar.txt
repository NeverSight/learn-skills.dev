تصميم وتنفيذ واجهات برمجة تطبيقات REST على AWS باستخدام أنماط CDK الخاصة بالمستودع الهدف أولاً، مع معالجات AWS Lambda وPostgres عبر Drizzle كمخزن البيانات الافتراضي. استخدم هذا عندما يريد المستخدم إضافة أو تعديل نقاط نهاية API Gateway REST أو معالجات Lambda أو مصادقة Cognito أو نماذج الطلب أو بنيات CDK القابلة لإعادة الاستخدام أو مخطط Drizzle أو مستودعات SQL أو مسارات CRUD المدعومة بقاعدة البيانات، حتى لو كانوا يطلبون فقط "نقطة نهاية" أو "معالج" أو "جدول" أو "مستودع" أو "بعض أسلاك CDK".
