Diseña e implementa API REST en AWS utilizando primero los patrones CDK del repositorio de destino, con los controladores AWS Lambda y Postgres a través de Drizzle como almacén de datos predeterminado. Utilice esto siempre que el usuario desee agregar o modificar puntos finales REST de API Gateway, controladores Lambda, autenticación Cognito, modelos de solicitud, construcciones CDK reutilizables, esquema Drizzle, repositorios SQL o rutas CRUD respaldadas por bases de datos, incluso si solo solicitan "un punto final", "un controlador", "una tabla", "un repositorio" o "algunos cables CDK".
