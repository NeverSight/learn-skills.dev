Progetta e implementa le API REST su AWS utilizzando prima i modelli CDK del repository di destinazione, con gestori AWS Lambda e Postgres tramite Drizzle come archivio dati predefinito. Utilizzalo ogni volta che l'utente desidera aggiungere o modificare endpoint REST API Gateway, gestori Lambda, autenticazione Cognito, modelli di richiesta, costrutti CDK riutilizzabili, schema Drizzle, repository SQL o percorsi CRUD supportati da database, anche se richiedono solo "un endpoint", "un gestore", "una tabella", "un repository" o "alcuni collegamenti CDK".
