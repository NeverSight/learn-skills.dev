Entwirft und implementiert REST-APIs auf AWS, wobei zunächst die CDK-Muster des Ziel-Repositorys verwendet werden, mit AWS Lambda-Handlern und Postgres über Drizzle als Standarddatenspeicher. Verwenden Sie dies immer dann, wenn der Benutzer API Gateway-REST-Endpunkte, Lambda-Handler, Cognito-Authentifizierung, Anforderungsmodelle, wiederverwendbare CDK-Konstrukte, Drizzle-Schema, SQL-Repositorys oder datenbankgestützte CRUD-Routen hinzufügen oder ändern möchte, auch wenn er nur nach „einem Endpunkt“, „einem Handler“, „einer Tabelle“, „einem Repository“ oder „etwas CDK-Verkabelung“ fragt.
