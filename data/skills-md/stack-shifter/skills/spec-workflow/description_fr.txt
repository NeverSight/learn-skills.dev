Pilote un flux de travail d'exécution par étapes axé sur les spécifications pour le travail sur les fonctionnalités. À utiliser lorsque l'utilisateur souhaite une spécification de fonctionnalité, un plan de mise en œuvre ou une boucle itérative « tâche suivante uniquement » où les exigences restent verrouillées dans `docs/specs/*.spec.md` et l'exécution réside dans un ou plusieurs fichiers `docs/specs/plans/*.plan.md` mutables.
