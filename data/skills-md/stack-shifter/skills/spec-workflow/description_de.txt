Fördert einen spezifikationsorientierten, phasenweisen Ausführungsworkflow für die Feature-Arbeit. Verwenden Sie diese Option, wenn der Benutzer eine Funktionsspezifikation, einen Implementierungsplan oder eine iterative „Nur nächste Aufgabe“-Schleife wünscht, bei der Anforderungen in „docs/specs/*.spec.md“ gesperrt bleiben und die Ausführung in einer oder mehreren veränderlichen „docs/specs/plans/*.plan.md“-Dateien erfolgt.
