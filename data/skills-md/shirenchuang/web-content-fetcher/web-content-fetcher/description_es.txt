Extraer el contenido principal de la página web. Admite una cadena alternativa de tres niveles: Jina Reader → Scrapling+html2text → web_fetch, que devuelve automáticamente el texto del cuerpo limpio con formato Markdown y al mismo tiempo conserva los encabezados, enlaces, URL de imágenes y estructuras de listas. Puede leer artículos de la cuenta oficial de WeChat (escenarios que Jina no puede manejar). Activador: el usuario desea capturar el contenido principal de una URL, leer un artículo, extraer el contenido de una página web, etc.
