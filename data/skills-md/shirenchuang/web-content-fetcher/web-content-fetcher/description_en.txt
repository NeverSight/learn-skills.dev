Web page body content extraction. Supports three-level fallback strategies: Jina Reader / Scrapling+html2text / web_fetch, automatically returns clean Markdown-formatted body, preserving titles, links, image URLs, and list structures. Can read WeChat Official Account articles (scenarios where Jina fails). Trigger condition: when the user wants to grab the body content of a URL, read an article, or extract web page content, etc.
