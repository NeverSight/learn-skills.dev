学術データのクエリと分析には AMiner Open Platform API を使用します。ユーザーが学者のプロフィール、論文の詳細、機関データ、雑誌の内容、または特許情報を検索する必要がある場合に、このスキルを使用します。トリガーシナリオ: AMiner への言及、学術データのクエリ、論文/学者/機関/ジャーナル/特許の検索、学術 Q&A 検索、引用分析、研究機関分析、学者のポートレート、論文引用チェーン、ジャーナル投稿分析など。6 つの組み合わせたワークフロー (学者プロフィール、論文詳細調査、組織分析、会場論文、論文 QA 検索、特許分析) と 28 人全員への直接呼び出しをサポートAPI。ユーザーが単に「○○という学者を調べて」とか「○○に関する論文を探して」と言った場合でも、このスキルを積極的に使ってください。
