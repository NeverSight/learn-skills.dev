Используйте API открытой платформы AMiner для запросов и анализа академических данных. Используйте этот навык, когда пользователям нужно найти профили ученых, сведения о статьях, данные об учреждениях, содержание журналов или патентную информацию. Сценарии запуска: упоминания AMiner, запросы академических данных, поиск статей/ученых/институтов/журналов/патентов, академический поиск вопросов и ответов, анализ цитирования, анализ научно-исследовательских учреждений, портреты ученых, цепочки цитирования статей, анализ материалов журналов и т. д. Поддерживает 6 комбинированных рабочих процессов (профиль ученого, глубокое погружение в статьи, организационный анализ, доклады с места проведения, поиск документов по обеспечению качества, патентный анализ) и прямые звонки всем 28 лицам API. Даже если пользователь просто говорит: «Найди ученого XXX» или «найди статьи о XXX», активно используйте этот навык.
