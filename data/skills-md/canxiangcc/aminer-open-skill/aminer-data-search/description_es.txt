Utilice la API de AMiner Open Platform para consultas y análisis de datos académicos. Utilice esta habilidad cuando los usuarios necesiten buscar perfiles de académicos, detalles de artículos, datos de instituciones, contenido de revistas o información de patentes. Escenarios desencadenantes: menciones de AMiner, consultas de datos académicos, búsqueda de artículos/académicos/instituciones/revistas/patentes, búsqueda de preguntas y respuestas académicas, análisis de citas, análisis de instituciones de investigación, retratos de académicos, cadenas de citas de artículos, análisis de envío de revistas, etc. Admite 6 flujos de trabajo combinados (perfil académico, análisis profundo de papel, análisis de organización, artículos de lugar, búsqueda de control de calidad de papel, análisis de patentes) y llamadas directas a las 28 API individuales. Incluso si el usuario simplemente dice "buscar académico XXX" o "buscar artículos sobre XXX", utilice esta habilidad de manera proactiva.
