Utilizza l'API AMiner Open Platform per query e analisi di dati accademici. Utilizza questa capacità quando gli utenti hanno bisogno di cercare profili di studiosi, dettagli di articoli, dati di istituti, contenuti di riviste o informazioni sui brevetti. Scenari di attivazione: menzioni di AMiner, query di dati accademici, ricerca di documenti/studiosi/istituzioni/riviste/brevetti, ricerca di domande e risposte accademiche, analisi di citazioni, analisi di istituti di ricerca, ritratti di studiosi, catene di citazioni di documenti, analisi di invio di riviste, ecc. Supporta 6 flussi di lavoro combinati (Profilo Scholar, Paper Deep Dive, Org Analysis, Venue Papers, Paper QA Search, Patent Analysis) e chiamate dirette a tutte le 28 API individuali. Anche se l'utente dice semplicemente "cerca studioso XXX" o "trova documenti su XXX", utilizza questa abilità in modo proattivo.
