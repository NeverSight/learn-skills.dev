استخدم AMiner Open Platform API للاستعلامات والتحليلات المتعلقة بالبيانات الأكاديمية. استخدم هذه المهارة عندما يحتاج المستخدمون إلى البحث عن الملفات الشخصية للباحثين، أو تفاصيل الأوراق، أو بيانات المؤسسة، أو محتوى المجلة، أو معلومات براءات الاختراع. سيناريوهات التشغيل: ذكر AMiner، استعلامات البيانات الأكاديمية، البحث عن الأوراق البحثية/الباحثين/المؤسسات/المجلات/براءات الاختراع، البحث الأكاديمي في الأسئلة والأجوبة، تحليل الاستشهادات، تحليل المؤسسات البحثية، صور الباحثين، سلاسل الاستشهادات الورقية، تحليل تقديم المجلات، وما إلى ذلك. يدعم 6 مسارات عمل مشتركة (ملف تعريف الباحث العلمي، والغوص العميق للورق، والتحليل التنظيمي، وأوراق المكان، وبحث ضمان الجودة الورقية، وتحليل براءات الاختراع) والمكالمات المباشرة لجميع الأفراد البالغ عددهم 28 فردًا واجهات برمجة التطبيقات. حتى لو قال المستخدم ببساطة "ابحث عن الباحث XXX" أو "ابحث عن أوراق بحثية حول XXX"، استخدم هذه المهارة بشكل استباقي.
