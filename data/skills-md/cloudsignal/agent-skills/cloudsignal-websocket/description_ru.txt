Создайте контекст React MQTT производственного уровня для уведомлений CloudSignal в реальном времени через WebSocket. Поддерживает Clerk, Supabase, Auth0, Firebase и пользовательских поставщиков аутентификации OIDC. Используйте при реализации уведомлений в реальном времени, обновлений в режиме реального времени, отслеживания хода выполнения заданий или обмена сообщениями через WebSocket с CloudSignal.
