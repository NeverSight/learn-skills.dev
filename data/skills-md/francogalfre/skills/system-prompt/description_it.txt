Guide per la scrittura di prompt di sistema e prompt utente di livello produttivo per SDK AI (Vercel AI SDK, Anthropic, OpenAI), agenti e chatbot. Utilizzare durante la scrittura o il refactoring del campo di sistema in generateText, streamText, generateObject; progettazione delle istruzioni dell'agente o del chatbot; debug di output incoerenti o allucinanti; ottimizzare le richieste in termini di costi o qualità; implementare chiamate di strumenti, output strutturato (Zod/generateObject) o agenti multi-step. Applicare quando l'utente desidera che un'intelligenza artificiale si comporti in un modo specifico, limiti gli output o migliori la coerenza.
