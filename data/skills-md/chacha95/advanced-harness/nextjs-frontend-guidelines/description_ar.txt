Next.js 15 إرشادات تطوير الواجهة الأمامية لتطبيق YGS (영영사) React 19/TypeScript. الأنماط الحديثة بما في ذلك جهاز توجيه التطبيقات، ومكونات الخادم/العميل، ومكونات shadcn/ui، وTailwind CSS 4، والمصادقة متعددة الأساليب (Firebase/Kakao/JWT)، وأنماط لوحة تحكم المشرف، والترجمة الكورية. يُستخدم عند إنشاء مكونات أو صفحات أو مسارات واجهة برمجة التطبيقات أو جلب البيانات أو التصميم أو العمل باستخدام كود الواجهة الأمامية.
