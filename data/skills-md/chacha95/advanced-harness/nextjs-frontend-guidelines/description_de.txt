Next.js 15 Frontend-Entwicklungsrichtlinien für die YGS (neue) React 19/TypeScript-Anwendung. Moderne Muster, einschließlich App Router, Server-/Client-Komponenten, Shadcn/UI-Komponenten, Tailwind CSS 4, Authentifizierung mit mehreren Methoden (Firebase/Kakao/JWT), Admin-Dashboard-Muster und koreanische Lokalisierung. Wird zum Erstellen von Komponenten, Seiten, API-Routen, zum Abrufen von Daten, zum Gestalten oder zum Arbeiten mit Frontend-Code verwendet.
