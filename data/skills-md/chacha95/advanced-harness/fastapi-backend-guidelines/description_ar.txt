إرشادات تطوير الواجهة الخلفية لـ FastAPI لتطبيقات Python غير المتزامنة. تصميم يستند إلى المجال باستخدام موجهات FastAPI، وSQLModel/SQLAlchemy ORM، ونمط المستودع، وطبقة الخدمة، وأنماط غير متزامنة/انتظار، والتحقق من صحة Pydantic، ومعالجة الأخطاء. يُستخدم عند إنشاء واجهات برمجة التطبيقات أو المسارات أو الخدمات أو المستودعات أو العمل باستخدام كود الواجهة الخلفية.
