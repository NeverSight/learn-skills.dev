一种工作流程技能，可分析 CSS、Tailwind 设置和现有项目的组件，以了解设计标记、直观地改进 UI，并通过 Figma MCP 服务器将其发送到 Figma。它不仅仅是增强 UI，还确保改进与项目的设计系统保持一致。它提出了新的颜色和间距，无需硬编码，保持在现有的标记和命名约定范围内。在以下情况下使用它：“查看 UI 并将其发送到 Figma”、“改进此页面的设计并将其反映在 Figma 中”、“分析 UI 并在 Figma 中创建设计方案”，或者每当需要改进 Web 应用程序 UI 并将设计移交给 Figma 时，尤其是当用户说“ui-to-figma”、“将 UI 发送到 Figma”或“改进设计并发送到 Figma”时。先决条件：Playwright（网络应用程序测试技能）和 Figma MCP 服务器必须可用。
