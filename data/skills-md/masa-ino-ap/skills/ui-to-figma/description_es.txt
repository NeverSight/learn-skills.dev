Una habilidad de flujo de trabajo que analiza CSS, la configuración de Tailwind y los componentes de un proyecto existente para comprender los tokens de diseño, mejora visualmente la interfaz de usuario y la envía a Figma a través del servidor Figma MCP. En lugar de simplemente mejorar la interfaz de usuario, garantiza que las mejoras se alineen con el sistema de diseño del proyecto. Propone nuevos colores y espacios sin codificación, manteniéndose dentro de los tokens y convenciones de nomenclatura existentes. Úselo en situaciones como: "Mire la interfaz de usuario y envíela a Figma", "Mejore el diseño de esta página y reflejela en Figma", "Analice la interfaz de usuario y cree una propuesta de diseño en Figma" o cuando sea necesario mejorar la interfaz de usuario de la aplicación web y transferir el diseño a Figma, especialmente cuando el usuario dice "ui-to-figma", "enviar UI a Figma" o "mejorar el diseño y enviar a Figma". Requisitos previos: Dramaturgo (habilidad de prueba de aplicaciones web) y el servidor Figma MCP deben estar disponibles.
