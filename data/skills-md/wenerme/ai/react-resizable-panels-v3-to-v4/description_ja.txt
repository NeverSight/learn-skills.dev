React-resizable-panels を v3 から v4 にアップグレードする場合、v4 のタイプ/ランタイム エラー (PanelGroup がエクスポートされていない、無効な方向プロパティ、レイアウトが壊れている) を修正する場合、新しいグループ/セパレーター API およびパーセンテージ文字列構文に適応する場合、または名前変更されたデータ属性の CSS セレクターを更新する場合に使用します。 「react-resizable-panels v4」、「PanelGroup が見つかりません」、「PanelResizeHandle が見つかりません」、「アップグレード後に壊れたパネル レイアウト」でトリガーされます。
