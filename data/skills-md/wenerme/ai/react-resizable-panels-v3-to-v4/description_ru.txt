Используйте при обновлении панелей с изменяемым размером реагирования с версии 3 до версии 4, исправлении ошибок типа/времени выполнения версии 4 (PanelGroup не экспортируется, недопустимое свойство направления, нарушенный макет), адаптации к новому API группы/разделителя и синтаксису процентных строк или обновлении селекторов CSS для переименованных атрибутов данных. Триггеры по «реагировать на изменяемые размеры панелей v4», «PanelGroup не найдена», «PanelResizeHandle не найдена», «макет панели нарушен после обновления».
