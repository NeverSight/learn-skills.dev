Маршрутизатор источника данных для доступа к данным китайского финансового рынка. Направляет пользователей к соответствующим навыкам работы с данными (akshare-data для акций/облигаций/фондов/макро, tianqin-data для фьючерсов). Предоставляет подробную таблицу ресурсов данных, показывающую, какие данные предоставляет каждый навык и какие API следует вызывать.
