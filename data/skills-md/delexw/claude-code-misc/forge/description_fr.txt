Forgez l’implémentation à partir de tickets JIRA en utilisant la génération dynamique de compétences. Rassemble le contexte dans une compétence chargée paresseusement, puis l'invoque pour exécution. À utiliser lorsqu'on vous donne une URL de ticket JIRA (format : https://[domain].atlassian.net/browse/[TICKET-ID]) pour traiter de bout en bout.
