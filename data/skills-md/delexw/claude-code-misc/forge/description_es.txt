Forje la implementación a partir de tickets JIRA mediante la generación dinámica de habilidades. Reúne el contexto en una habilidad con carga diferida y luego la invoca para su ejecución. Úselo cuando se le proporcione una URL de ticket JIRA (formato: https://[dominio].atlassian.net/browse/[TICKET-ID]) para procesar de un extremo a otro.
