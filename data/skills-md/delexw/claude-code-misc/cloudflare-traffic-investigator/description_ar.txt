التحقيق في حالات الشذوذ في حركة المرور والارتفاعات وتدهور الخدمة على المجالات المحمية بواسطة Cloudflare. يستخدم أدوات Cloudflare MCP لتحليلات GraphQL، وتحليل بصمات الأصابع JA4، وتسجيل أمان الروبوتات/WAF، والإبلاغ عن الحوادث. استخدم هذه المهارة عندما يتم ذكر ارتفاع حركة المرور، أو التحميل الزائد للخدمة، أو 429 خطأ، أو أحداث انقطاع الدائرة، أو تحليلات Cloudflare، أو مشكلات أداء المجال - حتى لو لم يذكر المستخدم "ارتفاع حركة المرور" صراحةً. يتم تشغيله أيضًا عندما يُطلب منك التحقق من بيانات Cloudflare لأي مجال.
