Investigue anomalías de tráfico, picos y degradación del servicio en dominios protegidos por Cloudflare. Utiliza herramientas MCP de Cloudflare para análisis GraphQL, análisis de huellas dactilares JA4, puntuación de seguridad de bot/WAF e informes de incidentes. Utilice esta habilidad siempre que se mencionen picos de tráfico, sobrecargas de servicios, errores 429, eventos de disyuntores, análisis de Cloudflare o problemas de rendimiento del dominio, incluso si el usuario no dice explícitamente "pico de tráfico". También se activa cuando se le solicita verificar los datos de Cloudflare para cualquier dominio.
