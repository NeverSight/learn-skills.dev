Untersuchen Sie Verkehrsanomalien, Spitzen und Dienstverschlechterungen auf durch Cloudflare geschützten Domänen. Verwendet Cloudflare MCP-Tools für GraphQL-Analysen, JA4-Fingerabdruckanalyse, Bot/WAF-Sicherheitsbewertung und Vorfallberichte. Verwenden Sie diesen Skill immer dann, wenn Verkehrsspitzen, Dienstüberlastungen, 429-Fehler, Leistungsschalterereignisse, Cloudflare-Analysen oder Probleme mit der Domänenleistung erwähnt werden – auch wenn der Benutzer nicht ausdrücklich „Verkehrsspitze“ sagt. Wird auch ausgelöst, wenn Sie aufgefordert werden, Cloudflare-Daten für eine beliebige Domain zu überprüfen.
