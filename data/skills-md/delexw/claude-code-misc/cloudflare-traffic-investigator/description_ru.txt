Исследуйте аномалии, скачки и ухудшение качества трафика в доменах, защищенных Cloudflare. Использует инструменты Cloudflare MCP для аналитики GraphQL, анализа отпечатков пальцев JA4, оценки безопасности ботов/WAF и отчетности об инцидентах. Используйте этот навык всякий раз, когда упоминаются скачки трафика, перегрузки службы, ошибки 429, события автоматического выключения, аналитика Cloudflare или проблемы с производительностью домена — даже если пользователь явно не говорит «всплеск трафика». Также срабатывает, когда вас просят проверить данные Cloudflare для любого домена.
