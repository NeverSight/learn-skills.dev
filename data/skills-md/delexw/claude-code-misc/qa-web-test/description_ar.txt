مهارة اختبار ضمان الجودة على الويب باستخدام أدوات Chrome DevTools MCP لاختبار الانحدار المرئي، والتحقق من صحة نقاط التوقف سريعة الاستجابة، وتصحيح أخطاء تخطيط CSS. استخدم هذه المهارة عندما يطلب المستخدم "اختبار صفحة"، أو "التحقق من نقاط التوقف"، أو "التحقق من التخطيط المستجيب"، أو "ضمان جودة هذه الصفحة"، أو "اختبار CSS في إطارات عرض مختلفة"، أو "التحقق من وجود أخطاء في التخطيط"، أو "التحقق من الإصلاح"، أو يريد فحص صفحة ويب بشكل مرئي عند عرض إطار عرض محدد. يتم تشغيله أيضًا عندما يقدم المستخدم عنوان URL ويطلب التقاط لقطات شاشة أو مقارنة التخطيطات أو فحص أبعاد العناصر. يعمل مع أي جلسة متصفح متصلة بـ Chrome DevTools MCP على المضيف المحلي أو البيئات المرحلية.
