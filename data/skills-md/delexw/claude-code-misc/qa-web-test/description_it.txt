Competenze di test web QA utilizzando gli strumenti MCP di Chrome DevTools per test di regressione visiva, convalida dei punti di interruzione reattivi e debug del layout CSS. Utilizza questa abilità ogni volta che l'utente chiede di "testare una pagina", "controllare i punti di interruzione", "verificare il layout reattivo", "fare il QA di questa pagina", "testare CSS in diverse finestre", "controllare bug di layout", "verificare la correzione" o desidera ispezionare visivamente una pagina web con larghezze di finestra specifiche. Si attiva anche quando l'utente fornisce un URL e chiede di acquisire screenshot, confrontare layout o controllare le dimensioni degli elementi. Funziona con qualsiasi sessione del browser Chrome DevTools connessa a MCP su ambienti host locali o di staging.
