Compétence en matière de tests Web d'assurance qualité à l'aide des outils Chrome DevTools MCP pour les tests de régression visuelle, la validation des points d'arrêt réactifs et le débogage de la mise en page CSS. Utilisez cette compétence chaque fois que l'utilisateur demande de « tester une page », « vérifier les points d'arrêt », « vérifier la mise en page réactive », « contrôler la qualité de cette page », « tester le CSS dans différentes fenêtres », « vérifier les bogues de mise en page », « vérifier le correctif » ou souhaite inspecter visuellement une page Web à des largeurs de fenêtre spécifiques. Se déclenche également lorsque l'utilisateur fournit une URL et demande à prendre des captures d'écran, à comparer les mises en page ou à inspecter les dimensions des éléments. Fonctionne avec n'importe quelle session de navigateur connectée à Chrome DevTools MCP sur un hôte local ou un environnement de test.
