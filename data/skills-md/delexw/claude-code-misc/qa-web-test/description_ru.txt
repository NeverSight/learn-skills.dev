Навыки веб-тестирования QA с использованием инструментов Chrome DevTools MCP для визуального регрессионного тестирования, адаптивной проверки точек останова и отладки макета CSS. Используйте этот навык всякий раз, когда пользователь просит «проверить страницу», «проверить точки останова», «проверить адаптивный макет», «проверить качество этой страницы», «проверить CSS в разных окнах просмотра», «проверить ошибки макета», «проверить исправление» или хочет визуально проверить веб-страницу при определенной ширине области просмотра. Также срабатывает, когда пользователь предоставляет URL-адрес и просит сделать снимки экрана, сравнить макеты или проверить размеры элемента. Работает с любым сеансом браузера, подключенным к Chrome DevTools MCP, на локальном хосте или в промежуточной среде.
