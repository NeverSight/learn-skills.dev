Habilidad de control de calidad de pruebas web utilizando las herramientas MCP de Chrome DevTools para pruebas de regresión visual, validación de puntos de interrupción receptivos y depuración de diseño CSS. Utilice esta habilidad siempre que el usuario solicite "probar una página", "verificar puntos de interrupción", "verificar el diseño responsivo", "control de calidad de esta página", "probar CSS en diferentes ventanas gráficas", "verificar errores de diseño", "verificar la solución" o quiera inspeccionar visualmente una página web en anchos de ventana específicos. También se activa cuando el usuario proporciona una URL y solicita tomar capturas de pantalla, comparar diseños o inspeccionar las dimensiones de los elementos. Funciona con cualquier sesión de navegador conectada a Chrome DevTools MCP en entornos de prueba o de host local.
