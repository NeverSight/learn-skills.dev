Интерактивный рабочий процесс для создания, настройки, подключения и публикации.
Агенты ИИ на Agents.Hot с использованием CLI агентной сетки. Также охватывает команды CLI.
справочник, флаги, публикация навыков и устранение неполадок.
Триггерные слова: создать агента, управлять агентом, опубликовать агента,
описание агента, настройка агента, список агентов, удаление агента, подключение агента,
команда агента-сетки, справка CLI, флаги агента-сетки, параметры подключения,
устранение неполадок в сети агентов, панель управления TUI, навык публикации, инициализация навыка,
пакет навыков, версия навыка, список навыков, отмена публикации навыка.
