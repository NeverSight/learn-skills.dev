Flujo de trabajo interactivo para crear, configurar, conectar y publicar
Agentes de IA en Agents.Hot usando la CLI de malla de agente. También cubre el comando CLI
referencia, banderas, publicación de habilidades y solución de problemas.
Palabras desencadenantes: crear agente, administrar agente, publicar agente,
descripción del agente, configuración del agente, enumerar agentes, eliminar agente, conectar agente,
comando de malla de agente, ayuda de CLI, indicadores de malla de agente, opciones de conexión,
solución de problemas de malla de agente, panel TUI, habilidad de publicación, inicio de habilidad,
paquete de habilidades, versión de habilidades, lista de habilidades, habilidad para no publicar.
