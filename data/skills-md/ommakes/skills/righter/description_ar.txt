قم بتطبيق مبادئ كتابة محتوى تجربة المستخدم لمراجعة نسخة واجهة المستخدم الحالية أو كتابة نسخة واجهة مستخدم جديدة من البداية. استخدم هذه المهارة عندما يطلب منك شخص ما: مراجعة أو تدقيق أو نقد أو تحسين نص واجهة المستخدم أو رسائل الخطأ أو تسميات الأزرار أو تلميحات الأدوات أو الحالات الفارغة أو نسخة الإعداد أو النص المساعد للنموذج أو أي نسخة لواجهة البرنامج. يتم تشغيله أيضًا عندما يطلب منك شخص ما كتابة نسخة جديدة من واجهة المستخدم، أو تسمية زر، أو صياغة رسالة خطأ، أو كتابة نموذج، أو إنشاء أي نص داخل المنتج. إذا كان الطلب يتضمن كلمات تظهر داخل البرنامج - استخدم هذه المهارة.
