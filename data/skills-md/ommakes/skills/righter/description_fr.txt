Appliquez les principes de rédaction de contenu UX pour examiner la copie d'interface utilisateur existante ou rédiger une nouvelle copie d'interface utilisateur à partir de zéro. Utilisez cette compétence chaque fois que quelqu'un vous demande de : réviser, auditer, critiquer ou améliorer le texte de l'interface utilisateur, les messages d'erreur, les étiquettes de boutons, les info-bulles, les états vides, la copie d'intégration, le texte d'aide au formulaire ou toute copie d'interface logicielle. Déclenchez également lorsque quelqu'un vous demande d'écrire une nouvelle copie de l'interface utilisateur, d'étiqueter un bouton, de rédiger un message d'erreur, d'écrire un modal ou de créer un texte intégré au produit. Si la demande implique des mots qui apparaissent dans le logiciel, utilisez cette compétence.
