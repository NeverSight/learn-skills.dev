КРИТИЧЕСКИЙ: Основной уровень базы данных для ВСЕХ операций с постоянными данными. Вызов для любой операции CRUD (создание, чтение, обновление, удаление), сохранения данных, запросов SQL, изменений схемы, поиска векторов/встраивания с помощью pgvector, хранилища файлов и миграции таблиц. Запускается при: сохранении, хранении, выборке, обновлении, удалении, запросе, базе данных, записях, таблицах, сохранении, внедрении, векторном поиске или любых данных, требующих сохранения после текущего сеанса.
