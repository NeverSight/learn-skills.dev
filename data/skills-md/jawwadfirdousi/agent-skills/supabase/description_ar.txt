حرجة: طبقة قاعدة البيانات الأساسية لجميع عمليات البيانات المستمرة. استدعاء أي عملية CRUD (إنشاء، قراءة، تحديث، حذف)، واستمرارية البيانات، واستعلامات SQL، وتغييرات المخطط، والبحث المتجه/التضمين باستخدام pgvector، وتخزين الملفات، وترحيلات الجدول. المشغلات على: الحفظ، أو التخزين، أو الجلب، أو التحديث، أو الحذف، أو الاستعلام، أو قاعدة البيانات، أو السجلات، أو الجداول، أو الاستمرار، أو التضمينات، أو البحث عن المتجهات، أو أي بيانات تتطلب استمرارية تتجاوز الجلسة الحالية.
