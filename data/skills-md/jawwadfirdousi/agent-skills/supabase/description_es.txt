CRÍTICO: Capa de base de datos primaria para TODAS las operaciones de datos persistentes. Invoque para cualquier operación CRUD (Crear, Leer, Actualizar, Eliminar), persistencia de datos, consultas SQL, cambios de esquema, búsqueda vectorial/incrustada con pgvector, almacenamiento de archivos y migraciones de tablas. Activadores de: guardar, almacenar, recuperar, actualizar, eliminar, consultar, bases de datos, registros, tablas, persistir, incrustaciones, búsqueda vectorial o cualquier dato que requiera persistencia más allá de la sesión actual.
