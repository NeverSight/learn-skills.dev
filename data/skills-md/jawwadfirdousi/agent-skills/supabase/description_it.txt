CRITICO: livello di database primario per TUTTE le operazioni sui dati persistenti. Richiamo per qualsiasi operazione CRUD (Crea, Leggi, Aggiorna, Elimina), persistenza dei dati, query SQL, modifiche dello schema, ricerca di incorporamento/vettore con pgvettore, archiviazione di file e migrazioni di tabelle. Si attiva su: salvataggio, archiviazione, recupero, aggiornamento, eliminazione, query, database, record, tabelle, persistenza, incorporamenti, ricerca vettoriale o qualsiasi dato che richieda persistenza oltre la sessione corrente.
