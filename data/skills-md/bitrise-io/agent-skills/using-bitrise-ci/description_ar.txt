- التخطيط لإعداد Bitrise CI أو تحليله
- تشغيل الإصدارات أو فحصها أو استكشاف الأخطاء وإصلاحها
- العمل مع ملفات bitrise.yml:
  - تصميم خطوط الأنابيب أو سير العمل أو حزم الخطوات أو تكوينات الخطوات
  - إصلاح الازدواجية أو تحسين بنية سير العمل
  - التحقق من صحة أو شرح تكوينات Bitrise
- إدارة مساحات العمل أو المشاريع أو التطبيقات أو المجموعات أو الأدوار
- العمل مع أدوات Bitrise CLI أو API أو MCP
