Funziona con Bitrise CI. **UTILIZZA SEMPRE PRIMA QUESTA COMPETENZA per qualsiasi attività relativa a Bitrise CI**, anche in modalità Pianificazione. Questa abilità fornisce conoscenze essenziali su come:
- Pianifica una configurazione CI Bitrise o analizzane una
- Attiva, controlla o risolvi i problemi delle build
- Lavora con file bitrise.yml:
  - Progetta pipeline, flussi di lavoro, pacchetti di passaggi o configurazioni di passaggi
  - Correggi la duplicazione o ottimizza la struttura del flusso di lavoro
  - Convalidare o spiegare le configurazioni di Bitrise
- Gestisci spazi di lavoro, progetti, app, gruppi o ruoli
- Lavora con gli strumenti Bitrise CLI, API o MCP
