Используйте, когда пользователь хочет читать/искать/отправлять/анализировать сообщения Telegram через интерфейс командной строки tgcli. Запуск по запросам об истории канала/чата, дайджестам/новостям, упоминаниям, файлам, темам, контактам, группам, тегам, загрузкам мультимедиа и состоянию архива/синхронизации. Также охватывает администратора группы (переименование, участники, ссылки для приглашения, присоединение/выход). Для кнопок редактирования/удаления/реакций/встроенных кнопок используйте вместо этого Telegram-MCP.
