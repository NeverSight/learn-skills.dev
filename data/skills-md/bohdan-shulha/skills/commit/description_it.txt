Crea ed esegui commit Git nel formato Commit convenzionali con un corpo richiesto. Da utilizzare quando l'utente chiede di confermare modifiche, scrivere messaggi di commit o preparare un commit pulito. Esegui sempre prima una protezione di sicurezza pre-commit, interrompi i file di registro o le estensioni binarie ad alto rischio non tracciate, quindi esegui lo stage con `git add . -A`.
