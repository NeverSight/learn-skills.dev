Crea ed esegui commit Git con un oggetto in stile JIRA e un corpo richiesti. Da utilizzare quando l'utente chiede di confermare modifiche, scrivere messaggi di commit o preparare un commit pulito e si aspetta un oggetto con prefisso ticket. Esegui sempre prima una protezione di sicurezza pre-commit, interrompi i file di registro o le estensioni binarie ad alto rischio non tracciate, quindi esegui lo stage con `git add . -A`.
