Cree, edite y mantenga la automatización `justfile` para cualquier base de código utilizando el ejecutor de comandos `just`. Se activa siempre que una solicitud implique crear un nuevo `justfile`, modificar/actualizar un `justfile` existente, agregar o cambiar recetas, refactorizar la estructura de `justfile` o automatizar comandos del proyecto a través de `just`.
