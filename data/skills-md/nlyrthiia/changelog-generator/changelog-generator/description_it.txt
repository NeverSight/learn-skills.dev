Genera log delle modifiche dalla cronologia dei commit git in più formati (Mantieni un log delle modifiche, Log delle modifiche convenzionale, raggruppato per data). Da utilizzare quando l'utente chiede di "generare il registro delle modifiche", "creare note di rilascio", "riepilogare i commit", "cosa è cambiato dall'ultimo rilascio" o necessita della creazione o dell'aggiornamento di un file CHANGELOG.md. Si attiva su qualsiasi registro delle modifiche, note di rilascio o attività di riepilogo del commit.
