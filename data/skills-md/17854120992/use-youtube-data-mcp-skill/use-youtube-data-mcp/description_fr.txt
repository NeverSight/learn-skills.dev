Configurez et utilisez le MCP de données YouTube hébergé de bout en bout avec une intervention minimale de l'utilisateur. À utiliser lorsque les utilisateurs souhaitent que l'agent vérifie Node.js et `npx`, configure la configuration du serveur MCP (Windows/macOS, Cursor/Codex/OpenClaw/OpenCode), demande la clé API au moment de la configuration, exécute la découverte des fonctionnalités post-installation (`tools/list` et `get_patch_notes`), puis recommande fortement les compétences d'assistance et la configuration Python pour les flux de travail locaux complets de documents et de feuilles de calcul.
