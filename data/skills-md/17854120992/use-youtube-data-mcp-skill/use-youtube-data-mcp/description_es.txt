Configure y utilice el MCP de datos de YouTube alojado de un extremo a otro con una mínima intervención del usuario. Úselo cuando los usuarios quieran que el agente verifique Node.js y `npx`, configure la configuración del servidor MCP (Windows/macOS, Cursor/Codex/OpenClaw/OpenCode), solicite la clave API en el momento de la configuración, ejecute el descubrimiento de capacidades posteriores a la instalación (`tools/list` y `get_patch_notes`) y luego recomiende encarecidamente la habilidad de ayuda y la configuración de Python para flujos de trabajo de hojas de cálculo y documentos locales completos.
