Configura e utilizza end-to-end l'MCP dati di YouTube ospitato con un input minimo da parte dell'utente. Da utilizzare quando gli utenti desiderano che l'agente verifichi Node.js e `npx`, configuri la configurazione del server MCP (Windows/macOS, Cursor/Codex/OpenClaw/OpenCode), richieda la chiave API al momento della configurazione, esegua il rilevamento delle funzionalità post-installazione (`tools/list` e `get_patch_notes`) e quindi consiglia vivamente la competenza di supporto e la configurazione di Python per flussi di lavoro completi di documenti locali e fogli di calcolo.
