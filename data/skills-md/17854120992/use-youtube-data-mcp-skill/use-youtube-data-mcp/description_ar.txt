قم بتكوين واستخدام YouTube Data MCP المستضاف من البداية إلى النهاية مع الحد الأدنى من إدخال المستخدم. يُستخدم عندما يريد المستخدمون من الوكيل التحقق من Node.js و`npx`، وتكوين تكوين خادم MCP (Windows/macOS، Cursor/Codex/OpenClaw/OpenCode)، وطلب مفتاح واجهة برمجة التطبيقات في وقت الإعداد، وتشغيل اكتشاف إمكانية ما بعد التثبيت (`tools/list` و`get_patch_notes`)، ثم يوصي بشدة بمهارة المساعد وإعداد Python لسير عمل المستندات وجداول البيانات المحلية الكاملة.
