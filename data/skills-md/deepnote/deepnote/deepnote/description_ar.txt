العمل مع ملفات مشروع Deepnote (.deepnote). استخدمه عند إنشاء ملفات .deepnote أو تحريرها أو فهمها - مشاريع دفتر الملاحظات المستندة إلى YAML والتي تحتوي على كود Python واستعلامات SQL وتخفيض السعر والمرئيات وعناصر واجهة المستخدم. يغطي بنية الملف، وأنواع الكتل، وتكامل قاعدة البيانات، واللقطات، واستخدام واجهة سطر الأوامر (CLI).
