Consulta la API pública del Régimen de Transparencia del BCRA. Permite buscar y comparar productos financieros de entidades argentinas: cajas de ahorro, plazos fijos, préstamos (personales, hipotecarios, prendas), tarjetas de crédito y paquetes de productos. Usa cuando el usuario pregunta sobre tasas, comisiones, productos bancarios argentinos, o quiera comparar ofertas entre bancos.
