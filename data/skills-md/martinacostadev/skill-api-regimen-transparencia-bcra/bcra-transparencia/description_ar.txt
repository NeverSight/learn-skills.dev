راجع API العامة لنظام الشفافية في BCRA. اسمح بالبحث عن المنتجات المالية للأشخاص في الأرجنتين ومقارنتها: صناديق الائتمان، وصناديق الاستثمار العقاري، والقروض (الشخصية، والرهن العقاري، والمقرضون)، وبطاقات الائتمان، وحزم المنتجات. الولايات المتحدة الأمريكية عندما يسأل المستخدم عن الأسعار والعمولات والمنتجات المصرفية الأرجنتينية، أو يرغب في مقارنة العروض بين البنوك.
