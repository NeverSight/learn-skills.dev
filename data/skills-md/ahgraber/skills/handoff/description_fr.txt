À utiliser lorsque le travail en cours doit se poursuivre dans une nouvelle conversation, session ou agent sans historique partagé. Déclencheurs : « transfert », « préparer le contexte pour la prochaine conversation », « résumer pour une nouvelle session », « continuer ceci dans une autre conversation ». Pas pour les mises à jour du statut du même chat ou la documentation du référentiel.
