À utiliser lorsque l'utilisateur demande une relecture ou une édition légère tout en préservant le libellé, le ton et l'ordre d'origine. Déclencheurs : « relecture », « correction de fautes de frappe », « grammaire uniquement », « copie, modification uniquement », « réussite SPAG ». Pas pour les réécritures, les changements de ton, les modifications structurelles ou les mises à niveau de style ; utilisez de la bonne prose pour ces demandes.
