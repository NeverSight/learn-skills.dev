Используйте при составлении сообщения об обычной фиксации на основе текущих изменений Git. Триггеры: «написать сообщение о фиксации», «черновик фиксации из поэтапных изменений», «создать обычную фиксацию», «сообщение для git commit -F -». Не для создания коммитов, внесения изменений в историю или полной проверки кода.
