Verwenden Sie diese Option, wenn Sie eine konventionelle Commit-Nachricht aus aktuell bereitgestellten Git-Änderungen erstellen. Auslöser: „Commit-Nachricht schreiben“, „Commit aus bereitgestellten Änderungen entwerfen“, „konventionelles Commit generieren“, „Nachricht für Git-Commit -F -“. Nicht zum Erstellen von Commits, zum Ändern des Verlaufs oder zur vollständigen Codeüberprüfung.
