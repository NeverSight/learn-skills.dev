Verwenden Sie diese Option, wenn Sie Codeänderungen vor dem Festschreiben oder Zusammenführen überprüfen – gestaffelte Unterschiede, Zweigunterschiede oder PR-Bereitschaftsprüfungen. Auslöser: „Meinen Code überprüfen“, „Diese Änderungen überprüfen“, „Diesen Zweig überprüfen“, „Feedback zu meinem Diff geben“, „Ist diese PR bereit?“. Nicht für die Implementierung von Funktionen, die Ausführung von Testsuiten oder reine Formatierungsdurchläufe.
