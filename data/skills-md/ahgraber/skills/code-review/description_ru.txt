Используйте при проверке изменений кода перед их фиксацией или слиянием — поэтапные различия, различия ветвей или проверки готовности к PR. Триггеры: «проверь мой код», «проверь эти изменения», «проверь эту ветку», «оставь отзыв о моих различиях», «готов ли этот PR?». Не для реализации функций, запуска наборов тестов или проходов только для форматирования.
