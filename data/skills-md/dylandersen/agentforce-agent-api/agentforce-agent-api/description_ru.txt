Интегрирует API агента Salesforce Agentforce в веб-приложения с использованием потока учетных данных клиента OAuth. Охватывает создание сеанса, синхронный обмен сообщениями, кэширование токенов и правильные форматы конечной точки/тела API. Используйте при создании интерфейсов чата с Agentforce, настройке аутентификации API агента, отладке ошибок API агента или подключении внешних приложений к агентам Salesforce.
