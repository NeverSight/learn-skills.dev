يدمج Salesforce Agentforce Agent API في تطبيقات الويب باستخدام تدفق بيانات اعتماد عميل OAuth. يغطي إنشاء الجلسة، والرسائل المتزامنة، والتخزين المؤقت للرمز المميز، وتنسيقات نقطة نهاية/نص واجهة برمجة التطبيقات الصحيحة. يُستخدم عند إنشاء واجهات الدردشة باستخدام Agentforce، أو إعداد مصادقة Agent API، أو تصحيح أخطاء Agent API، أو توصيل التطبيقات الخارجية بوكلاء Salesforce.
