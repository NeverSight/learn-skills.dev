Integra la API del agente Salesforce Agentforce en aplicaciones web mediante el flujo de credenciales de cliente OAuth. Cubre la creación de sesiones, la mensajería sincrónica, el almacenamiento en caché de tokens y los formatos correctos de cuerpo/punto final de API. Úselo al crear interfaces de chat con Agentforce, configurar la autenticación de API del agente, depurar errores de API del agente o conectar aplicaciones externas a agentes de Salesforce.
