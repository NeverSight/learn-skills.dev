Integra l'API Salesforce Agentforce Agent nelle applicazioni Web utilizzando il flusso delle credenziali client OAuth. Riguarda la creazione di sessioni, la messaggistica sincrona, la memorizzazione nella cache dei token e i formati corretti di endpoint/corpo dell'API. Da utilizzare durante la creazione di interfacce di chat con Agentforce, la configurazione dell'autenticazione dell'API dell'agente, il debug degli errori dell'API dell'agente o la connessione di app esterne agli agenti Salesforce.
