Устраняйте конфликты слияния/перебазирования Git с помощью явного отслеживания намерений и проверки после разрешения. Используйте, когда возникают конфликты и перед интеграцией требуется разрешение с сохранением намерений; не используйте для проектирования рабочих процессов CI или реализации поведения приложений.
