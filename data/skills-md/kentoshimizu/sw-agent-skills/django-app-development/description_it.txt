Flusso di lavoro di sviluppo dell'applicazione Django per modifiche alle app di livello produttivo. Da utilizzare quando il codice specifico di Django (modelli, visualizzazioni, serializzatori/moduli, routing, impostazioni, middleware, flussi di autenticazione) deve essere implementato o rivisto con il comportamento di runtime del framework nell'ambito; non utilizzare per la governance dell'architettura a livello di repository o per la politica di gestione dei rilasci.
