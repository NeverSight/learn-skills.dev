Рабочий процесс разработки приложений Django для внесения изменений в приложения производственного уровня. Используйте, когда код, специфичный для Django (модели, представления, сериализаторы/формы, маршрутизация, настройки, промежуточное программное обеспечение, потоки аутентификации) должен быть реализован или изменен с учетом поведения среды выполнения платформы; не используйте для управления архитектурой всего репозитория или политики управления выпусками.
