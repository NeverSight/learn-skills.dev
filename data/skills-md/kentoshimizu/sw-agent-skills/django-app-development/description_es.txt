Flujo de trabajo de desarrollo de aplicaciones Django para cambios de aplicaciones de nivel de producción. Úselo cuando se deba implementar o revisar código específico de Django (modelos, vistas, serializadores/formularios, enrutamiento, configuraciones, middleware, flujos de autenticación) teniendo en cuenta el comportamiento del tiempo de ejecución del marco; no lo utilice para la gobernanza de la arquitectura de todo el repositorio ni para la política de gestión de versiones.
