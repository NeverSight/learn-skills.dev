Flux de travail de développement d'applications Django pour les modifications d'applications de niveau production. À utiliser lorsque le code spécifique à Django (modèles, vues, sérialiseurs/formulaires, routage, paramètres, middleware, flux d'authentification) doit être implémenté ou révisé avec le comportement d'exécution du framework dans le cadre ; ne pas utiliser pour la gouvernance de l'architecture à l'échelle du référentiel ou pour la politique de gestion des versions.
