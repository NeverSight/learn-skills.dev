Рабочий процесс проектирования физической базы данных для структуры хранилища, разделения, настроек механизма и поведения производительности с учетом аппаратного обеспечения. Используйте, когда устойчивая эффективность рабочей нагрузки зависит от физической организации данных; не используйте для задач концептуального моделирования.
