Рабочий процесс создания Runbook для четких и исполняемых рабочих процедур для дежурных операторов. Используйте, когда действия по реагированию на инциденты, откату, аварийному переключению или обслуживанию должны быть задокументированы перед производственным использованием; не используйте для проектирования логики производственных функций.
