Flux de travail de capture de preuves visuelles pour des captures d'écran reproductibles dans les tâches d'ingénierie et d'assurance qualité. À utiliser lorsque les rapports de bogues, les contrôles de régression ou les preuves d'implémentation nécessitent des artefacts de capture d'écran stables ; ne pas utiliser pour l’extraction de sources de conception ou la documentation d’architecture.
