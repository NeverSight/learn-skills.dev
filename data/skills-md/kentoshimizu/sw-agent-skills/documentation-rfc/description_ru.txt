Автор документов RFC для предлагаемых технических изменений с четкой формулировкой проблемы, вариантами, компромиссами и путями решения. Используйте, когда перед реализацией необходимо согласование и рассмотрение предложения; не используйте для документов, предназначенных только для реализации после принятия решения.
