Standard di stile, revisione e refactoring per schemi SQL, migrazione e artefatti di query. Si attiva quando vengono creati, modificati o rivisti gli elementi di schema/query/migrazione `.sql` e devono essere applicate regole di qualità specifiche di SQL (denominazione, sicurezza della migrazione, chiarezza delle query e intento prestazionale). Non utilizzare per modifiche al codice solo ORM senza elementi SQL o per documenti di modellazione dati non SQL. Utilizzare insieme alle guide di stile specifiche della lingua solo quando vengono modificati sia il codice SQL che quello dell'applicazione.
