Стандарты стиля, проверки и рефакторинга для схемы SQL, миграции и артефактов запросов. Запускается, когда артефакты схемы/запроса/миграции `.sql` создаются, изменяются или проверяются, а правила качества, специфичные для SQL (именование, безопасность миграции, ясность запроса и производительность), должны быть соблюдены. Не используйте для изменений кода только для ORM без артефактов SQL или для документов моделирования данных, отличных от SQL. Используйте вместе с руководствами по стилю для конкретного языка только в том случае, если изменены как SQL, так и код приложения.
