معايير التصميم والمراجعة وإعادة البناء لمخطط SQL والترحيل وعناصر الاستعلام. يتم التشغيل عند إنشاء عناصر المخطط/الاستعلام/الترحيل `.sql` أو تغييرها أو مراجعتها ويجب فرض قواعد الجودة الخاصة بـ SQL (التسمية وأمان الترحيل ووضوح الاستعلام ونية الأداء). لا تستخدمه لتغييرات التعليمات البرمجية الخاصة بـ ORM فقط دون عناصر SQL، أو لمستندات نمذجة البيانات غير SQL. استخدمه مع أدلة الأنماط الخاصة باللغة فقط عندما يتم تعديل كل من SQL ورمز التطبيق.
