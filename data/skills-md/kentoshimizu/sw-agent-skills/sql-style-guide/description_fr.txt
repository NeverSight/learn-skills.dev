Normes de style, de révision et de refactorisation pour les artefacts de schéma SQL, de migration et de requête. Se déclenche lorsque des artefacts de schéma/requête/migration `.sql` sont créés, modifiés ou révisés et que des règles de qualité spécifiques à SQL (dénomination, sécurité de la migration, clarté des requêtes et intention de performances) doivent être appliquées. Ne pas utiliser pour les modifications de code ORM uniquement sans artefacts SQL, ni pour les documents de modélisation de données non SQL. À utiliser avec les guides de style spécifiques au langage uniquement lorsque SQL et le code de l'application sont modifiés.
