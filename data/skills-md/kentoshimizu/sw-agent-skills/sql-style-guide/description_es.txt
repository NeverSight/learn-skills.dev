Estándares de estilo, revisión y refactorización para artefactos de consulta, migración y esquema SQL. Se activa cuando se crean, cambian o revisan artefactos de esquema/consulta/migración `.sql` y se deben aplicar reglas de calidad específicas de SQL (denominación, seguridad de la migración, claridad de las consultas e intención de rendimiento). No lo utilice para cambios de código solo ORM sin artefactos SQL ni para documentos de modelado de datos que no sean SQL. Úselo junto con guías de estilo específicas del idioma solo cuando se modifiquen tanto SQL como el código de la aplicación.
