Flujo de trabajo de seguridad para arquitectura de autenticación, ciclo de vida de credenciales y garantía de sesión/token. Úselo cuando se requieran decisiones de inicio de sesión, prueba de identidad, MFA o seguridad de la sesión; no lo utilice para el diseño de políticas de autorización ni para ajustes de calidad no relacionados con la seguridad.
