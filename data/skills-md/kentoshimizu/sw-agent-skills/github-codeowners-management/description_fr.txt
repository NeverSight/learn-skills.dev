Gouvernez les règles CODEOWNERS afin que le routage des révisions reflète la propriété réelle et les limites de risque sur GitHub. À utiliser lorsque le mappage de propriété du référentiel ou les règles de réviseur obligatoires doivent être définis, mis à jour ou audités ; ne pas utiliser pour une architecture d'exécution non GitHub ou une conception de couche de données.
