Gobierne las reglas de CODEOWNERS para que el enrutamiento de revisión refleje la propiedad real y los límites de riesgo en GitHub. Úselo cuando se deban definir, actualizar o auditar la asignación de propiedad del repositorio o las reglas obligatorias del revisor; no lo utilice para arquitectura de tiempo de ejecución o diseño de capa de datos que no sean de GitHub.
