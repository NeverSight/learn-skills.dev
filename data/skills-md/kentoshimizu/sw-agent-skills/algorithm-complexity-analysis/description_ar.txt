تحليل الخوارزميات المرشحة لتعقيد الوقت/المكان، وحدود قابلية التوسع، وملاءمة ميزانية الموارد (وحدة المعالجة المركزية، والذاكرة، والإدخال/الإخراج، والتزامن). يُستخدم عندما تعتمد الجدوى على نمو المدخلات أو قيود الكمون/الذاكرة وتكون الحدود الكمية مطلوبة قبل التنفيذ؛ لا تستخدم لمخطط الثبات أو قرارات طوبولوجيا النشر.
