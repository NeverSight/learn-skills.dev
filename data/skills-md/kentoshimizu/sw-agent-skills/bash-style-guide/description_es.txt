Estándares de estilo, revisión y refactorización para secuencias de comandos de shell Bash. Se activa cuando se crean, modifican o revisan archivos `.sh`, archivos con `#!/usr/bin/env bash` o `#!/bin/bash`, o bloques de flujo de trabajo de CI con `shell: bash` y se deben aplicar controles de calidad específicos de Bash (seguridad de citas, manejo de errores, portabilidad, legibilidad). No lo utilice para POSIX `sh` genérico, PowerShell o reglas de estilo de aplicación específicas del idioma. En solicitudes de extracción en varios idiomas, ejecútelas junto con otras habilidades de `*-style-guide` aplicables.
