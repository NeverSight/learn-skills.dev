Standard di stile, revisione e refactoring per lo scripting della shell Bash. Si attiva quando i file ".sh", i file con "#!/usr/bin/env bash" o "#!/bin/bash" o i blocchi del flusso di lavoro CI con "shell: bash" vengono creati, modificati o revisionati e devono essere applicati controlli di qualità specifici di Bash (quotazione di sicurezza, gestione degli errori, portabilità, leggibilità). Non utilizzare per POSIX `sh` generico, PowerShell o regole di stile dell'applicazione specifiche della lingua. Nelle richieste pull multilingue, eseguile insieme ad altre competenze `*-style-guide` applicabili.
