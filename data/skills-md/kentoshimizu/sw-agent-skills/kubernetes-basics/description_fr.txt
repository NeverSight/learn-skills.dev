Workflow de base de Kubernetes pour le déploiement de charges de travail, la découverte de services et les primitives de cluster de base. À utiliser lorsque les équipes ont besoin de décisions concrètes en matière d'espace de noms, de charge de travail, de service et de sonde avant le déploiement ; ne pas utiliser pour la conception de contrats API ou la priorisation des exigences.
