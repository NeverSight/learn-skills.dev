Рабочий процесс по основам Kubernetes для развертывания рабочих нагрузок, обнаружения сервисов и базовых примитивов кластера. Используйте, когда перед развертыванием командам нужны конкретные решения по пространству имен/рабочей нагрузке/сервису/зонду; не используйте для разработки контрактов API или определения приоритетов требований.
