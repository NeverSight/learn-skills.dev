Стандарты стиля, проверки и рефакторинга для баз кода JavaScript с явными контрактами времени выполнения, безопасным асинхронным поведением и поддерживаемыми границами модулей. Используйте, когда артефакты `.js`, `.jsx`, `.mjs` или `.cjs` создаются, изменяются или проверяются, и необходимо соблюдать правила качества, специфичные для JS. Не используйте для кода, принадлежащего `.ts`, `.tsx`, `.d.ts` или `tsconfig*.json`. Для общих файлов конфигурации JS/TS запустите вместе с `typescript-style-guide`, когда артефакты JS и TS изменяются в одном запросе на включение.
