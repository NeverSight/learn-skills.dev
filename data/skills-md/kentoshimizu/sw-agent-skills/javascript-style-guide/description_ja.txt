明示的な実行時契約、安全な非同期動作、保守可能なモジュール境界を備えた JavaScript コードベースのスタイル、レビュー、およびリファクタリング標準。 「.js」、「.jsx」、「.mjs」、または「.cjs」アーティファクトが作成、変更、またはレビューされ、JS 固有の品質ルールを適用する必要がある場合に使用します。 `.ts`、`.tsx`、`.d.ts`、または `tsconfig*.json` が所有するコードには使用しないでください。共有 JS/TS 構成ファイルの場合、JS アーティファクトと TS アーティファクトの両方が同じプル リクエスト内で変更されるときに、`typescript-style-guide` とともに実行します。
