معايير التصميم والمراجعة وإعادة البناء لقواعد تعليمات JavaScript مع عقود تشغيل واضحة وسلوك غير متزامن آمن وحدود وحدة يمكن صيانتها. يُستخدم عند إنشاء عناصر `.js` أو `.jsx` أو `.mjs` أو `.cjs` أو تغييرها أو مراجعتها ويجب تطبيق قواعد الجودة الخاصة بـ JS. لا تستخدم التعليمات البرمجية المملوكة لـ `.ts` أو `.tsx` أو `.d.ts` أو `tsconfig*.json`. بالنسبة لملفات تكوين JS/TS المشتركة، يمكنك تشغيلها مع "typescript-style-guide" عندما تتغير عناصر JS وTS في نفس طلب السحب.
