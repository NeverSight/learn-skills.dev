명시적인 런타임 계약, 안전한 비동기 동작 및 유지 관리 가능한 모듈 경계를 갖춘 JavaScript 코드베이스의 스타일, 검토 및 리팩토링 표준입니다. `.js`, `.jsx`, `.mjs` 또는 `.cjs` 아티팩트가 생성, 변경 또는 검토되고 JS별 품질 규칙을 적용해야 하는 경우 사용합니다. `.ts`, `.tsx`, `.d.ts` 또는 `tsconfig*.json` 소유 코드에는 사용하지 마세요. 공유 JS/TS 구성 파일의 경우 동일한 풀 요청에서 JS 및 TS 아티팩트가 모두 변경되면 'typescript-style-guide'와 함께 실행하세요.
