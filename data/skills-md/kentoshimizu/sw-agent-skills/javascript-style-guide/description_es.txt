Estándares de estilo, revisión y refactorización para bases de código JavaScript con contratos de ejecución explícitos, comportamiento asíncrono seguro y límites de módulo mantenibles. Úselo cuando se crean, cambian o revisan artefactos `.js`, `.jsx`, `.mjs` o `.cjs` y se deben aplicar reglas de calidad específicas de JS. No lo utilice para código propiedad de `.ts`, `.tsx`, `.d.ts` o `tsconfig*.json`. Para archivos de configuración JS/TS compartidos, ejecútelo junto con `typescript-style-guide` cuando los artefactos JS y TS cambien en la misma solicitud de extracción.
