TypeScript コードベースのスタイル、レビュー、およびリファクタリングの標準。 `.ts`、`.tsx`、`.d.ts`、または `tsconfig*.json` ファイルが作成、変更、またはレビューされ、TS 固有の品質ルール (タイプ セーフティ、絞り込み、非同期/エラー処理、モジュール境界) を適用する必要があるときにトリガーされます。 TypeScript の型チェック範囲外の JavaScript 専用モジュールには使用しないでください。混合 JS/TS リポジトリでは、このスキルは「tsconfig」スコープのモジュールを所有します。共有 JS/TS 構成ファイル (例: 「package.json」、共有 lint/test 構成) の場合、JS アーティファクトと TS アーティファクトの両方が同じプル リクエスト内で変更されるときに、「javascript-style-guide」 と一緒に実行します。
