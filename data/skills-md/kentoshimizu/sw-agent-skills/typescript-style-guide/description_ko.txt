TypeScript 코드베이스에 대한 스타일, 검토 및 리팩토링 표준입니다. `.ts`, `.tsx`, `.d.ts` 또는 `tsconfig*.json` 파일이 생성, 변경 또는 검토될 때 트리거되며 TS 관련 품질 규칙(유형 안전성, 축소, 비동기/오류 처리, 모듈 경계)을 시행해야 합니다. TypeScript 유형 검사 범위를 벗어나는 JavaScript 전용 모듈에는 사용하지 마십시오. 혼합 JS/TS 리포지토리에서 이 기술은 `tsconfig` 범위의 모듈을 소유합니다. 공유 JS/TS 구성 파일(예: `package.json`, 공유 lint/test config)의 경우 동일한 풀 요청에서 JS와 TS 아티팩트가 모두 변경되면 `javascript-style-guide`와 함께 실행하세요.
