معايير التصميم والمراجعة وإعادة البناء لقواعد تعليمات TypeScript. يتم تشغيله عند إنشاء ملفات `.ts` أو `.tsx` أو `.d.ts` أو `tsconfig*.json` أو تغييرها أو مراجعتها ويجب فرض قواعد الجودة الخاصة بـ TS (أمان النوع، والتضييق، ومعالجة الأخطاء/غير المتزامنة، وحدود الوحدة النمطية). لا تستخدمه مع وحدات JavaScript فقط التي تقع خارج نطاق التحقق من النوع TypeScript. في مستودعات JS/TS المختلطة، تمتلك هذه المهارة وحدات ذات نطاق `tsconfig`. بالنسبة لملفات تكوين JS/TS المشتركة (على سبيل المثال، `package.json`، وتكوين Lint/test المشترك)، يتم تشغيلها مع `javascript-style-guide` عندما تتغير عناصر JS وTS في نفس طلب السحب.
