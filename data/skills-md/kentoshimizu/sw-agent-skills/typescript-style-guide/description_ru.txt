Стандарты стиля, проверки и рефакторинга для кодовых баз TypeScript. Запускается, когда файлы `.ts`, `.tsx`, `.d.ts` или `tsconfig*.json` создаются, изменяются или проверяются, а правила качества, специфичные для TS (безопасность типов, сужение, асинхронная обработка/обработка ошибок, границы модулей), должны быть соблюдены. Не используйте для модулей только JavaScript, которые находятся за пределами области проверки типов TypeScript. В смешанных репозиториях JS/TS этот навык владеет модулями с областью действия `tsconfig`. Для общих файлов конфигурации JS/TS (например, package.json, общая конфигурация lint/test) запускайте вместе с javascript-style-guide, когда артефакты JS и TS изменяются в одном запросе на извлечение.
