تصميم عقد REST/OpenAPI موجه نحو الموارد لدلالات URI/الأسلوب، والتناقض، وترقيم الصفحات، وسلوك حالة HTTP. يتم التشغيل عند اكتشاف اختلافات عقد REST في المسارات أو الأساليب أو المخططات أو دلالات الحالة، أو عندما لا يزال اختيار نقل الطلب والاستجابة دون حل قبل التنفيذ. لا تستخدم لتأليف مخطط GraphQL، أو تغييرات تصنيف الأخطاء فقط، أو إدارة دورة حياة الإصدار، أو تنفيذ اختبار العقد بين المستهلك والموفر.
