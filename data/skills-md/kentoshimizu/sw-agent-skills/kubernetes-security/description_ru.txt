Рабочий процесс безопасности Kubernetes для усиления защиты кластера, изоляции рабочей нагрузки и применения политик. Используйте, когда RBAC, сетевая политика, безопасность модуля, обработка секретов или контроль доступа должны быть определены до раскрытия рабочей среды; не используйте для разработки контрактов API или определения приоритетов требований.
