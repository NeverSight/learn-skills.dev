Рабочий процесс создания пользовательских историй для превращения приоритетных требований в реализуемые и тестируемые фрагменты доставки. Используйте, когда утвержденная работа по требованию должна быть разложена для выполнения спринта; не используйте для первоначального обнаружения требований или определения приоритетов на уровне выпуска.
