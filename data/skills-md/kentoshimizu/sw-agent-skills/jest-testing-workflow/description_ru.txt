Рабочий процесс проверки Jest для кодовых баз JavaScript/TypeScript с детерминированными тестами, явными фиктивными границами и действенными доказательствами CI. Используйте, когда изменения JS/TS требуют исполняемого доказательства Jest перед слиянием; не используйте для E2E на уровне браузера или разработки политики, не зависящей от языка.
