Рабочий процесс инцидентов безопасности для сортировки, локализации, уничтожения и обработки доказательств. Используйте, когда предполагаемые или подтвержденные инциденты безопасности требуют скоординированных ответных действий; не используйте для превентивного моделирования угроз или регулярной обработки журнала уязвимостей.
