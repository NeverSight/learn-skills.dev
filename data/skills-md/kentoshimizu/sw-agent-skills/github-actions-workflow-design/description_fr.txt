Concevez et gérez des workflows GitHub Actions avec une portée de déclenchement explicite, des limites de sécurité et une orchestration fiable des tâches. À utiliser lorsque l'automatisation de GitHub (tâches de build/test/release, déclencheurs, autorisations, mise en cache, concurrence, environnements) doit être créée ou révisée ; ne pas utiliser pour une architecture d'exécution non GitHub ou une conception de couche de données.
