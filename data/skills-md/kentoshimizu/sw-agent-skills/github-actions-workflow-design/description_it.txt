Progetta e mantieni flussi di lavoro GitHub Actions con ambito di trigger esplicito, limiti di sicurezza e orchestrazione affidabile dei lavori. Da utilizzare quando è necessario creare o modificare l'automazione GitHub (processi di creazione/test/rilascio, trigger, autorizzazioni, memorizzazione nella cache, concorrenza, ambienti); non utilizzare per un'architettura runtime non GitHub o per la progettazione del livello dati.
