Diseñe y mantenga flujos de trabajo de GitHub Actions con un alcance de activación explícito, límites de seguridad y una orquestación de trabajos confiable. Úselo cuando se deba crear o revisar la automatización de GitHub (trabajos de compilación/prueba/lanzamiento, activadores, permisos, almacenamiento en caché, concurrencia, entornos); no lo utilice para arquitectura de tiempo de ejecución o diseño de capa de datos que no sean de GitHub.
