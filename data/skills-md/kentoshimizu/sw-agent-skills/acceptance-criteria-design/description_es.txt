Diseñe criterios de aceptación ejecutables para los requisitos aprobados mediante la conversión de objetivos/especificaciones en comprobaciones binarias de aprobación/rechazo con resultados observables. Úselo cuando la transferencia de implementación, la validación de control de calidad o las decisiones de lanzamiento necesiten criterios comprobables para líneas base estables `REQ-*`/`NFR-*`; no lo utilice para el descubrimiento de requisitos, la priorización o el corte de sprints.
