Flujo de trabajo de desarrollo de servicios FastAPI para API de Python de nivel de producción. Úselo cuando se deba implementar o revisar el comportamiento del servicio FastAPI (operaciones de ruta, inyección de dependencia, modelos de solicitud/respuesta, validación, ejecución asíncrona, exposición a OpenAPI); no lo utilice para la gobernanza de la arquitectura de todo el repositorio ni para la política de gestión de versiones.
