Workflow de développement de services FastAPI pour les API Python de qualité production. À utiliser lorsque le comportement du service FastAPI (opérations de chemin, injection de dépendances, modèles de requête/réponse, validation, exécution asynchrone, exposition OpenAPI) doit être implémenté ou révisé ; ne pas utiliser pour la gouvernance de l'architecture à l'échelle du référentiel ou pour la politique de gestion des versions.
