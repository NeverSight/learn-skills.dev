Рабочий процесс разработки сервисов FastAPI для API Python промышленного уровня. Используйте, когда поведение службы FastAPI (операции пути, внедрение зависимостей, модели запроса/ответа, проверка, асинхронное выполнение, раскрытие OpenAPI) должно быть реализовано или пересмотрено; не используйте для управления архитектурой всего репозитория или политики управления выпусками.
