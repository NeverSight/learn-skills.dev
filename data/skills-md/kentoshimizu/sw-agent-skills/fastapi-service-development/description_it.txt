Flusso di lavoro di sviluppo del servizio FastAPI per API Python di livello produttivo. Utilizzare quando il comportamento del servizio FastAPI (operazioni sul percorso, inserimento di dipendenze, modelli di richiesta/risposta, convalida, esecuzione asincrona, esposizione OpenAPI) deve essere implementato o rivisto; non utilizzare per la governance dell'architettura a livello di repository o per la politica di gestione dei rilasci.
