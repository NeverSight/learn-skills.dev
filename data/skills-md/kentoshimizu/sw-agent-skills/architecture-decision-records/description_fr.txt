Flux de travail d'enregistrement des décisions d'architecture pour capturer les décisions techniques, les alternatives, les compromis et les déclencheurs de révision au fil du temps. À utiliser lorsque les décisions affectent matériellement le comportement du système, les coûts ou le flux de travail de l'équipe ; ne pas utiliser pour des choix d’implémentation triviaux.
