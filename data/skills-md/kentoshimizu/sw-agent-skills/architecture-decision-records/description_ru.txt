Рабочий процесс записи решений по архитектуре, позволяющий фиксировать технические решения, альтернативы, компромиссы и причины изменений с течением времени. Используйте, когда решения существенно влияют на поведение системы, стоимость или рабочий процесс команды; не используйте для тривиальных вариантов реализации.
