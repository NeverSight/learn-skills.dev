Workflow de sécurité pour les décisions de codage sécurisées par défaut et la prévention des vulnérabilités lors des modifications de mise en œuvre. À utiliser lorsque les chemins de code d'application traitent des entrées non fiables, des données sensibles ou des opérations privilégiées ; ne pas utiliser pour une gouvernance basée uniquement sur les politiques sans impact sur le code.
