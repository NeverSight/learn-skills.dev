Рабочий процесс показателей наблюдаемости для разработки модели показателей, согласованный с работоспособностью сервиса и влиянием на бизнес. Используйте, когда команды определяют или пересматривают метрики обслуживания/SLI для принятия надежных решений о работоспособности и емкости; не используйте для логики реализации бизнес-функций.
