Рабочий процесс мониторинга отклонения MLOps для обнаружения отклонения данных, отклонения концепции и ухудшения качества с помощью действенных правил реагирования. Используйте, когда производственные системы машинного обучения нуждаются в пороговых значениях обнаружения отклонений и эскалации владения; не используйте для принятия решений по исследованию модели-архитектуры.
