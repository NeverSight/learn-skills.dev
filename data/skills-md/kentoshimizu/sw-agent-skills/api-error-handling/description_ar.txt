تصميم عقد فشل واجهة برمجة التطبيقات (API) لتعيين الحالة، وأكواد الأخطاء الثابتة، ودلالات إمكانية إعادة المحاولة، وحمولات الأخطاء التي يمكن تتبعها عبر عمليات النقل المتزامنة وغير المتزامنة. يتم تشغيله عندما يتغير سلوك الخطأ، أو إعادة محاولة الدلالات، أو حقول قابلية تصحيح الأخطاء في المواصفات أو المصدر، أو عندما تظل هذه القواعد ضمنية. لا تستخدم لنمذجة الموارد/المخطط الكامل أو تعريف سياسة قناة الإصدار.
