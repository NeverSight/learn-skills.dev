Разработка контракта сбоя API для сопоставления состояний, стабильных кодов ошибок, семантики повторных попыток и отслеживаемых полезных данных ошибок при синхронном и асинхронном транспорте. Запускается, когда поведение ошибки, семантика повторных попыток или поля отладки изменяются в спецификациях или исходном коде или когда эти правила остаются неявными. Не используйте для полного моделирования ресурсов/схем или определения политики канала версий.
