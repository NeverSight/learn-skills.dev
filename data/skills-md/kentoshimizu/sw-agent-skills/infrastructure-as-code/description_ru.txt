Декларативный рабочий процесс изменения инфраструктуры для проверенных планом операций, обеспечения государственной безопасности и предотвращения сноса. Используйте, когда изменения инфраструктуры должны быть представлены декларативно с проверяемыми планами и контролируемым применением; не используйте для разработки контрактов API или определения приоритетов требований.
