Flusso di lavoro dichiarativo di modifica dell'infrastruttura per operazioni di revisione del piano, sicurezza dello stato e prevenzione della deriva. Da utilizzare quando le modifiche infrastrutturali devono essere rappresentate in modo dichiarativo con piani rivedibili e applicazione controllata; non utilizzare per la progettazione del contratto API o per la definizione delle priorità dei requisiti.
