Flux de travail déclaratif de modification de l'infrastructure pour les opérations révisées par plan, la sécurité de l'État et la prévention de la dérive. À utiliser lorsque les modifications de l'infrastructure doivent être représentées de manière déclarative avec des plans révisables et une application contrôlée ; ne pas utiliser pour la conception de contrats API ou la priorisation des exigences.
