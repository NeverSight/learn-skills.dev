Рабочий процесс нагрузочного тестирования производительности для реалистичного моделирования рабочей нагрузки, обнаружения узких мест и анализа поведения при насыщении. Используйте, когда системам требуется проверка пропускной способности/задержки перед развертыванием; не использовать в случае неисполнения функциональных решений о приемке.
