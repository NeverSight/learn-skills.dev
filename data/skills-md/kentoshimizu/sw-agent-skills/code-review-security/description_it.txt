Esegui una revisione del codice incentrata sulla sicurezza quando le modifiche superano i limiti di attendibilità o possono influire sull'autenticazione, sull'autorizzazione, sulla convalida dell'input, sulla gestione dei segreti o sull'esposizione di dati sensibili. Utilizzare per decisioni di unione che richiedono risultati di sicurezza espliciti; non utilizzare per ambiti di revisione non esclusivamente legati alla sicurezza.
