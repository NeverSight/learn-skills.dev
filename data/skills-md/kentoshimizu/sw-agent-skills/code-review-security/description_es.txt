Ejecute una revisión de código centrada en la seguridad cuando los cambios crucen los límites de confianza o puedan afectar la autenticación, la autorización, la validación de entradas, el manejo de secretos o la exposición de datos confidenciales. Úselo para decisiones de fusión que requieran hallazgos de seguridad explícitos; no lo utilice para un ámbito de revisión que no sea únicamente de seguridad.
