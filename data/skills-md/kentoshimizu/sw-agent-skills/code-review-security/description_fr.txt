Exécutez une révision du code axée sur la sécurité lorsque les modifications dépassent les limites de confiance ou peuvent affecter l'authentification, l'autorisation, la validation des entrées, la gestion des secrets ou l'exposition des données sensibles. À utiliser pour les décisions de fusion nécessitant des résultats de sécurité explicites ; ne pas utiliser pour une portée d’examen autre que la sécurité.
