قم بإجراء مراجعة التعليمات البرمجية التي تركز على الأمان عندما تتجاوز التغييرات حدود الثقة أو قد تؤثر على المصادقة أو الترخيص أو التحقق من صحة الإدخال أو معالجة الأسرار أو الكشف عن البيانات الحساسة. يُستخدم لقرارات الدمج التي تتطلب نتائج أمنية واضحة؛ لا تستخدم لنطاق المراجعة غير المتعلق بالأمان فقط.
