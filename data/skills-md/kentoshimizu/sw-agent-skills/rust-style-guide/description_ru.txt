Стандарты стиля, проверки и рефакторинга для кодовых баз Rust. Запускается, когда файлы `.rs`, `Cargo.toml` или `Cargo.lock` создаются, изменяются или проверяются, а также при необходимости соблюдения специфичных для Rust правил качества (ясность владения/заимствования, обработка ошибок, дизайн признаков/API, границы ящиков). Не используйте соглашения о наборе инструментов C/C++, изменения только в оболочке или проблемы со стилем среды выполнения, не связанные с Rust, если также не изменяются артефакты Rust. В многоязычных запросах на включение используйте другие применимые навыки `*-style-guide`.
