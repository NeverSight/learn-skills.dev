Estándares de estilo, revisión y refactorización para bases de código de Rust. Se activa cuando se crean, cambian o revisan archivos `.rs`, `Cargo.toml` o `Cargo.lock` y se deben aplicar reglas de calidad específicas de Rust (claridad de propiedad/préstamo, manejo de errores, diseño de rasgos/API, límites de cajas). No lo use para convenciones de cadena de herramientas de C/C++, cambios solo de shell o inquietudes de estilo de tiempo de ejecución que no sean de Rust, a menos que también se cambien los artefactos de Rust. En solicitudes de extracción en varios idiomas, ejecútelas junto con otras habilidades de `*-style-guide` aplicables.
