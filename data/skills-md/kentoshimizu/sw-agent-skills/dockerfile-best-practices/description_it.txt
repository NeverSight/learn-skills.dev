Progetta file Docker per build di immagini sicure, deterministiche ed efficienti con una superficie di attacco minima e dipendenze riproducibili. Da utilizzare quando il comportamento di creazione dell'immagine, la strategia dei livelli e il rafforzamento del runtime richiedono decisioni esplicite; non utilizzare per problemi relativi allo stile del codice di runtime non contenitore.
