Diseñe Dockerfiles para compilaciones de imágenes seguras, deterministas y eficientes con una superficie de ataque mínima y dependencias reproducibles. Úselo cuando el comportamiento de creación de imágenes, la estrategia de capas y el endurecimiento del tiempo de ejecución necesiten decisiones explícitas; no lo utilice para cuestiones de estilo de código de tiempo de ejecución que no sean contenedores.
