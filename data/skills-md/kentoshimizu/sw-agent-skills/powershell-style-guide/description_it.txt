Standard di stile, revisione e refactoring per gli script di PowerShell. Si attiva quando vengono creati, modificati o revisionati i file ".ps1", ".psm1", ".psd1" o i blocchi del flusso di lavoro CI con "shell: pwsh" o "shell: powershell" e devono essere applicati controlli di qualità specifici di PowerShell (gestione degli errori, convalida dei parametri, leggibilità, sicurezza operativa). Non utilizzare per Bash, POSIX generico `sh` o regole di stile dell'applicazione specifiche della lingua. Nelle richieste pull multilingue, eseguile insieme ad altre competenze `*-style-guide` applicabili.
