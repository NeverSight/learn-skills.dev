Estándares de estilo, revisión y refactorización para secuencias de comandos de PowerShell. Se activa cuando se crean, modifican o revisan archivos `.ps1`, `.psm1`, `.psd1` o bloques de flujo de trabajo de CI con `shell: pwsh` o `shell: powershell` y se deben aplicar controles de calidad específicos de PowerShell (manejo de errores, validación de parámetros, legibilidad, seguridad operativa). No lo use para Bash, POSIX `sh` genérico o reglas de estilo de aplicación específicas del idioma. En solicitudes de extracción en varios idiomas, ejecútelas junto con otras habilidades de `*-style-guide` aplicables.
