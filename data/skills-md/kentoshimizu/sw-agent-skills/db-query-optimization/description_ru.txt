Рабочий процесс оптимизации запросов для сокращения задержек и затрат ресурсов за счет перезаписи с учетом плана и улучшения путей доступа. Используйте, когда поведение запроса горячего пути является узким местом; не используйте для изменения концептуальной схемы без подтверждения рабочей нагрузки.
