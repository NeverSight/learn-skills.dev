اختبار العقود بين المستهلك وموفر الخدمة وتصميم بوابة الإصدار لمصفوفات التوافق وتأكيدات العقد وقواعد حظر CI عبر الإصدارات وعمليات النقل. يتم تشغيله عند اكتشاف اختلافات في عقد واجهة برمجة التطبيقات (API) ولكن دليل التوافق مفقود أو قديم (اجتياز اختبار العقد، أو تعيين تأثير المستهلك، أو تحديث مصفوفة التوافق)، أو عندما يجب حماية العديد من المستهلكين أو الإصدارات أو وسائل النقل بواسطة بوابات آلية. لا تستخدم لتصميم واجهة API للمرور الأول.
