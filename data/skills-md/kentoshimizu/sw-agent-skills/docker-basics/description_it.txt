Progettare ed esaminare le nozioni di base sul runtime del contenitore per un'esecuzione locale/di servizio riproducibile utilizzando Docker. Da utilizzare quando il comportamento di creazione/esecuzione del contenitore, la rete, i volumi e l'isolamento del runtime richiedono decisioni esplicite; non utilizzare per attività di definizione delle priorità del contratto API o dei requisiti.
