Concevoir et réviser les bases de l'exécution des conteneurs pour une exécution locale/de service reproductible à l'aide de Docker. À utiliser lorsque le comportement de création/d'exécution du conteneur, la mise en réseau, les volumes et l'isolation d'exécution nécessitent des décisions explicites ; ne pas utiliser pour les tâches de contrat API ou de priorisation des exigences.
