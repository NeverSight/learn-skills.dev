مرجع شامل لتسجيل الدخول إلى LINE (OAuth 2.1) - تدفق رمز التفويض، وPKCE، وإدارة الرمز المميز، والتحقق من رمز المعرف JWT، وملفات تعريف المستخدمين، وربط الروبوت، وتصميم زر تسجيل الدخول. يجب استخدام هذه المهارة عندما يطلب المستخدم "تنفيذ تسجيل الدخول إلى LINE"، أو "إضافة تسجيل دخول باستخدام LINE"، أو "إعداد تدفق ترخيص OAuth"، أو "التحقق من رمز معرف"، أو "تحديث رمز وصول"، أو "ربط روبوت لتسجيل الدخول"، أو "تصميم زر تسجيل الدخول"، أو يذكر تسجيل دخول LINE، OAuth 2.1، PKCE، تدفق رمز التفويض، التحقق من رمز JWT للمعرف، تحديث/إلغاء الرمز المميز، استرجاع ملف تعريف المستخدم، الروبوت الربط أو تسجيل الدخول SSO أو مصادقة LIFF أو مصادقة تطبيق LINE MINI. استخدم هذه المهارة دائمًا عندما يذكر المستخدم مصادقة LINE، أو تسجيل الدخول الاجتماعي باستخدام LINE، أو تدفقات OAuth التي تتضمن LINE، حتى لو لم يذكر صراحة "تسجيل الدخول إلى LINE".
