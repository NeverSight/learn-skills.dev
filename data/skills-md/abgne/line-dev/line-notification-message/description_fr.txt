Référence pour les messages de notification LINE : messagerie basée sur un numéro de téléphone destinée aux non-amis, type de modèle (mises en page prédéfinies) et type flexible (message Flex personnalisé), webhooks de fin de livraison, flux de consentement de l'utilisateur, authentification SMS et facturation. À utiliser lorsque l'utilisateur demande « d'envoyer un message de notification via un numéro de téléphone », de « hacher un numéro de téléphone pour PNP », de « gérer le webhook de fin de livraison », de « configurer un modèle ou une notification flexible », de « vérifier le nombre de messages de notification » ou mentionne les messages de notification LINE, le push PNP, la livraison basée sur le numéro de téléphone, le numéro de téléphone haché, le modèle de notification, l'événement de fin de livraison, les états de consentement ou l'authentification par SMS pour les notifications. Utilisez toujours cette compétence chaque fois que l'utilisateur mentionne des messages de notification LINE, PNP ou des messages LINE par téléphone à des non-amis, même s'ils ne disent pas explicitement « message de notification ».
