مرجع لرسائل إشعارات LINE - المراسلة المستندة إلى رقم الهاتف لغير الأصدقاء، ونوع القالب (تخطيطات معدة مسبقًا) والنوع المرن (رسالة مرنة مخصصة)، وخطافات الويب الخاصة بإكمال التسليم، وتدفق موافقة المستخدم، ومصادقة الرسائل القصيرة، والفوترة. يُستخدم عندما يطلب المستخدم "إرسال رسالة إشعار عبر رقم الهاتف"، أو "تجزئة رقم هاتف لـ PNP"، أو "التعامل مع خطاف ويب لإكمال التسليم"، أو "إعداد القالب أو الإشعار المرن"، أو "التحقق من عدد رسائل الإشعارات"، أو يذكر رسائل إشعار LINE، أو دفع PNP، أو التسليم المستند إلى رقم الهاتف، أو رقم الهاتف المجزأ، أو قالب الإشعار، أو حدث إكمال التسليم، أو حالات الموافقة، أو مصادقة الرسائل القصيرة للإشعارات. استخدم هذه المهارة دائمًا عندما يذكر المستخدم رسائل إشعار LINE، أو PNP، أو رسائل LINE المستندة إلى الهاتف إلى غير الأصدقاء، حتى لو لم يذكر "رسالة إشعار" صراحةً.
