Справочник по уведомительным сообщениям LINE — обмен сообщениями на основе номера телефона для лиц, не являющихся друзьями, тип шаблона (предварительные макеты) и гибкий тип (настраиваемое гибкое сообщение), веб-перехватчики завершения доставки, поток согласия пользователя, аутентификация по SMS и выставление счетов. Используйте, когда пользователь просит «отправить сообщение с уведомлением по номеру телефона», «хешировать номер телефона для PNP», «обработать веб-перехватчик завершения доставки», «настроить шаблон или гибкое уведомление», «проверить количество сообщений с уведомлением» или упоминает сообщения уведомлений LINE, push-уведомление PNP, доставку на основе номера телефона, хешированный номер телефона, шаблон уведомления, событие завершения доставки, состояния согласия или аутентификацию по SMS для уведомлений. Всегда используйте этот навык всякий раз, когда пользователь упоминает сообщения уведомлений LINE, PNP или сообщения LINE по телефону тем, кто не является другом, даже если они явно не говорят «сообщение уведомления».
