LINE Messaging API の包括的なリファレンス — Webhook の設定、メッセージ送信、フレックス メッセージのデザイン、リッチ メニューの管理、対象ユーザーのターゲティング、インサイト、クーポン、チャネル アクセス トークン。このスキルは、ユーザーが「LINE Botの構築」、「Webhookの設定」、「プッシュメッセージの送信」、「フレックスメッセージのデザイン」、「リッチメニューの作成」、「オーディエンスターゲティングの管理」、「メッセージングのインサイトの取得」、「クーポンキャンペーンの作成」、「Webhook署名検証のデバッグ」を要求する場合、またはLINE Messaging API、LINE OAチャットボット、リプライ/プッシュ/マルチキャスト/ナローキャスト/ブロードキャスト、フレックス メッセージ JSON、リッチ メニュー、グループ チャット ボット、チャネル アクセス トークン、または URL スキーム。ユーザーが「Messaging API」と明示的に言わなくても、LINE ボット、チャットボット、LINE OA、またはメッセージング関連の LINE 統合について言及するときは常にこのスキルを使用してください。
