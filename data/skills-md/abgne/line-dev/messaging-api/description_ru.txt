Комплексное руководство по API LINE Messaging — настройка веб-перехватчика, отправка сообщений, дизайн гибких сообщений, управление расширенным меню, таргетинг на аудиторию, аналитика, купоны и токены доступа к каналу. Этот навык следует использовать, когда пользователь просит «создать бота LINE», «настроить вебхук», «отправить push-сообщение», «создать гибкое сообщение», «создать расширенное меню», «управлять таргетингом на аудиторию», «получить информацию об обмене сообщениями», «создать купонную кампанию», «отладить проверку подписи веб-перехватчика» или упоминает LINE Messaging API, чат-бот LINE OA, ответ/push/многоадресную/узкую рассылку/широковещательную рассылку, Flex Сообщение JSON, расширенное меню, групповой чат-бот, токен доступа к каналу или схемы URL-адресов. Всегда используйте этот навык всякий раз, когда пользователь упоминает ботов LINE, чат-ботов, LINE OA или любую интеграцию LINE, связанную с обменом сообщениями, даже если они явно не говорят «API обмена сообщениями».
