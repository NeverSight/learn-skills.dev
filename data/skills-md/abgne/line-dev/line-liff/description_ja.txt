LINE フロントエンド フレームワーク (LIFF) SDK の包括的なリファレンス — 認証、メッセージング、QR スキャン、パーマネント リンク、プラグイン可能な SDK、LIFF プラグイン開発を使用して LINE 内に Web アプリを構築します。このスキルは、ユーザーが「LIFFアプリを構築する」、「liff.init()を初期化する」、「LIFFからメッセージを送信する」、「シェアターゲットピッカーを使用する」、「LIFFでQRコードをスキャンする」、「パーマネントリンクを作成する」、「LIFFプラグインを開発する」、またはLIFF SDK、LINEフロントエンドフレームワーク、CDN/npm統合、プラガブルSDKについて言及する場合に使用する必要があります。ツリーシェイキング、LIFFからLIFFへの移行、LIFFブラウザと外部ブラウザ、エンドポイントURL設定、またはLIFFからのサーバーサイドIDトークン検証。ユーザーが「LIFF SDK」と明示的に言わなくても、LIFF、LINE内のWebアプリ、LINE Front-end Frameworkについて言及するときは必ずこのスキルを使用してください。
