LINE 前端框架 (LIFF) SDK 的綜合參考 — 在 LINE 內建立 Web 應用程序，包括身份驗證、訊息傳遞、QR 掃描、永久連結、可插入 SDK 和 LIFF 插件開發。當使用者要求「建構 LIFF 應用程式」、「初始化 liff.init()」、「從 LIFF 發送訊息」、「使用共享目標選擇器」、「在 LIFF 中掃描二維碼」、「建立永久連結」、「開發 LIFF 外掛程式」或提到 LIFF SDK、LINE 前端框架、CDN/npm 樹狀圖、瀏覽器與外部瀏覽器、端點 URL 配置或來自 LIFF 的伺服器端 ID 令牌驗證。每當使用者提及 LIFF、LINE 內的 Web 應用程式或 LINE 前端框架時，請務必使用此技能，即使他們沒有明確指出「LIFF SDK」。
