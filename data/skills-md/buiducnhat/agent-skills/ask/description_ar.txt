التوضيح المنظم وجمع المتطلبات من خلال الحوار المركز. يُستخدم عندما تكون المهمة غامضة أو غير محددة أو تتطلب إدخال المستخدم قبل اتخاذ أي إجراء. لا تخطط أو تنفذ أي شيء، اطرح الأسئلة فقط لجمع المعلومات المطلوبة. يتم تشغيلها على: "اسألني"، أو "طرح أسئلة حول"، أو "توضيح المتطلبات"، أو "جمع المتطلبات"، أو "أريدك أن تسأل"، أو عندما يريد المستخدم صراحةً عقد جلسة أسئلة وأجوبة قبل بدء العمل.
