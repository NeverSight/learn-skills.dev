إنشاء مهارات جديدة وتعديل وتحسين المهارات الموجودة وقياس الأداء المهاري. يُستخدم عندما يريد المستخدمون إنشاء مهارة من البداية، أو تحديث مهارة موجودة أو تحسينها، أو تجميع مهارة للتوزيع، أو تكرار جودة المهارة. قم بتشغيل هذه المهارة عندما يقول المستخدم "إنشاء مهارة"، أو "بناء مهارة"، أو "اصنع مهارة لـ X"، أو "تحديث هذه المهارة"، أو "تحسين هذه المهارة"، أو "حزم هذه المهارة"، أو يذكر رغبته في توسيع قدرات Claude من خلال مسارات عمل أو أدوات متخصصة.
