تصميم وإنشاء وتدقيق وتحسين مهارات وكيل كلود من خلال الكشف التدريجي على مستوى الخبراء. يُستخدم عند بناء مهارات جديدة، أو مراجعة المهارات الموجودة، أو تصحيح فشل التنشيط، أو تشفير خبرة المجال، أو تصميم المهارات لاستهلاك الوكيل الفرعي. التنشيط على "إنشاء مهارة"، "تحسين المهارة"، "تدقيق المهارات"، "مراجعة المهارات"، "تصحيح أخطاء التنشيط"، "shibboleth"، "الكشف التدريجي"، "وصف المهارة". ليس لميزات Claude Code العامة، أو تصحيح الأخطاء في وقت التشغيل، أو البرمجة غير المهارية، أو تنفيذ خادم MCP.
