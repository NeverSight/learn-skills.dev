Современные веб-приложения с аутентичной эстетикой Windows 3.1. Сплошные строки заголовка темно-синего цвета, навигация в Менеджере программ, скошенные границы, элементы управления в одном окне. Экстраполирует Win31 на чат-ботов с искусственным интеллектом (парадигма Cue Card), мобильные пользовательские интерфейсы (карманные вычисления). Активируйте в «Windows 3.1», «win31», «Диспетчере программ», «Рабочий стол в стиле ретро», «Эстетика 90-х», «Скошенный». НЕ для Windows 95 (используйте windows-95-web-designer - есть градиенты, меню «Пуск»), паровая волна/синтетическая волна, macOS, плоский дизайн.
