Désinfectez le contenu MDX pour les builds Docusaurus. Corrige les crochets angulaires non échappés (<, >, &lt;=, &gt;=), la syntaxe du modèle Liquid/Nunjucks ({{ }}), les génériques TypeScript (Promise&lt;T&gt;) et les cas extrêmes de backtick de code en ligne. À utiliser lorsque les hooks de pré-validation échouent lors de la validation bracket ou Liquid, ou lorsque les erreurs de build MDX/JSX font référence à des jetons inattendus. PAS pour le peluchage général ou l’édition de prose.
