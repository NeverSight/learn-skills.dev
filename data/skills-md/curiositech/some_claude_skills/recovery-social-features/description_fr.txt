Fonctionnalités sociales axées sur la confidentialité pour les applications de récupération : sponsors, groupes, messagerie, connexions entre amis. À utiliser pour les systèmes de sponsor/filleul, les groupes basés sur des réunions, le soutien par les pairs et la messagerie sécurisée. Activez sur « parrain », « filleul », « groupe de rétablissement », « partenaire de responsabilité », « réseau sobre », « groupe de réunion », « soutien par les pairs ». PAS pour les modèles généraux de médias sociaux (utilisez les réseaux sociaux standards), les fonctionnalités de rencontres ou les profils publics.
