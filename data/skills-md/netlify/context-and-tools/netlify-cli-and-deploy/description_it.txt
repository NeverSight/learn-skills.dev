Guida per l'utilizzo della CLI Netlify e la distribuzione dei siti. Da utilizzare durante l'installazione della CLI, il collegamento di siti, la distribuzione (basata su Git o manuale), la gestione delle variabili di ambiente o l'esecuzione dello sviluppo locale. Copre netlify dev, netlify deploy, flussi di lavoro Git e non Git e gestione delle variabili di ambiente.
