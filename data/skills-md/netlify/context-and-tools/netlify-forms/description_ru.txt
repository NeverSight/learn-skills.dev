Руководство по использованию Netlify Forms для обработки HTML-форм. Используйте при добавлении контактных форм, форм обратной связи, форм загрузки файлов или любой формы, которая должна быть собрана Netlify. Охватывает атрибут data-netlify, фильтрацию спама, отправку AJAX, загрузку файлов, уведомления и API отправки.
