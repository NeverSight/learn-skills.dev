Guía para usar Netlify Forms para el manejo de formularios HTML. Úselo al agregar formularios de contacto, formularios de comentarios, formularios de carga de archivos o cualquier formulario que Netlify deba recopilar. Cubre el atributo data-netlify, el filtrado de spam, los envíos AJAX, la carga de archivos, las notificaciones y la API de envíos.
