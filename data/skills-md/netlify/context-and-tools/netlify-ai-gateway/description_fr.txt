Guide d'utilisation de Netlify AI Gateway pour accéder aux modèles d'IA. À utiliser lors de l'ajout de fonctionnalités d'IA (génération de texte, génération d'images, intégrations) à un projet Netlify. Couvre les fournisseurs pris en charge (OpenAI, Anthropic, Google), la configuration du SDK via le remplacement de l'URL de base, les variables d'environnement et l'utilisation de Netlify Functions.
