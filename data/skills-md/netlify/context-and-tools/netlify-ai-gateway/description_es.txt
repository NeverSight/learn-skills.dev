Guía para usar Netlify AI Gateway para acceder a modelos de IA. Úselo al agregar capacidades de IA (generación de texto, generación de imágenes, incrustaciones) a un proyecto de Netlify. Cubre proveedores compatibles (OpenAI, Anthropic, Google), configuración de SDK mediante anulación de URL base, variables de entorno y uso de Netlify Functions.
