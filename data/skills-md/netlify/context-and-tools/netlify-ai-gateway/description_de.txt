Leitfaden zur Verwendung von Netlify AI Gateway für den Zugriff auf KI-Modelle. Verwenden Sie diese Option, wenn Sie einem Netlify-Projekt KI-Funktionen (Textgenerierung, Bildgenerierung, Einbettungen) hinzufügen. Deckt unterstützte Anbieter (OpenAI, Anthropic, Google), SDK-Konfiguration über Basis-URL-Überschreibung, Umgebungsvariablen und Nutzung von Netlify-Funktionen ab.
