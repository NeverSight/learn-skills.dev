Руководство по использованию Netlify AI Gateway для доступа к моделям ИИ. Используйте при добавлении возможностей искусственного интеллекта (генерация текста, генерация изображений, встраивание) в проект Netlify. Охватывает поддерживаемых поставщиков (OpenAI, Anthropic, Google), конфигурацию SDK через переопределение базового URL-адреса, переменные среды и использование функций Netlify.
