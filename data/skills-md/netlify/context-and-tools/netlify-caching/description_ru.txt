Руководство по управлению кэшированием в CDN Netlify. Используйте при настройке заголовков кэша, настройке устаревших данных при повторной проверке, реализации очистки кэша по требованию или понимании поведения кэширования Netlify CDN. Охватывает Cache-Control, Netlify-CDN-Cache-Control, теги кэша, устойчивый кэш и шаблоны кэширования, специфичные для платформы.
