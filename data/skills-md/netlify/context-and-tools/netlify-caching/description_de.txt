Leitfaden zur Steuerung des Cachings im CDN von Netlify. Verwenden Sie diese Option, wenn Sie Cache-Header konfigurieren, Stale-While-Revalidate einrichten, eine bedarfsgesteuerte Cache-Löschung implementieren oder das CDN-Caching-Verhalten von Netlify verstehen. Behandelt Cache-Control, Netlify-CDN-Cache-Control, Cache-Tags, dauerhaften Cache und Framework-spezifische Caching-Muster.
