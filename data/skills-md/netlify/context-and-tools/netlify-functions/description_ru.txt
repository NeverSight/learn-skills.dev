Руководство по написанию бессерверных функций Netlify. Используйте при создании конечных точек API, фоновой обработки, запланированных задач или любой серверной логики с использованием функций Netlify. Охватывает современный синтаксис (экспорт по умолчанию + конфигурация), TypeScript, маршрутизацию путей, фоновые функции, запланированные функции, потоковую передачу и маршрутизацию методов.
