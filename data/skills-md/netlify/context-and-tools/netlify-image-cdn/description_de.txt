Leitfaden zur Verwendung von Netlify Image CDN zur Bildoptimierung und -transformation. Verwenden Sie diese Option, wenn Sie optimierte Bilder bereitstellen, reaktionsfähiges Bild-Markup erstellen, vom Benutzer hochgeladene Bild-Pipelines einrichten oder Bildtransformationen konfigurieren. Behandelt den /.netlify/images-Endpunkt, Abfrageparameter, Remote-Image-Zulassungslisten, saubere URL-Umschreibungen und das Verfassen von Uploads mit Funktionen + Blobs.
