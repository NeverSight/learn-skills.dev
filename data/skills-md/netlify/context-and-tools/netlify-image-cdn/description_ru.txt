Руководство по использованию Netlify Image CDN для оптимизации и преобразования изображений. Используйте при показе оптимизированных изображений, создании адаптивной разметки изображений, настройке конвейеров загружаемых пользователем изображений или настройке преобразований изображений. Охватывает конечную точку /.netlify/images, параметры запроса, удаленный список разрешенных изображений, чистую перезапись URL-адресов и составление загрузок с помощью функций + больших двоичных объектов.
