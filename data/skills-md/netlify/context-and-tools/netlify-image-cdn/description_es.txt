Guía para usar Netlify Image CDN para optimización y transformación de imágenes. Úselo cuando proporcione imágenes optimizadas, cree marcas de imágenes responsivas, configure canalizaciones de imágenes cargadas por el usuario o configure transformaciones de imágenes. Cubre el punto final /.netlify/images, los parámetros de consulta, la lista de imágenes permitidas remotas, la reescritura de URL limpia y la composición de cargas con Functions + Blobs.
