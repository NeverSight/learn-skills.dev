دليل لاستخدام Netlify Image CDN لتحسين الصورة وتحويلها. يُستخدم عند عرض الصور المحسنة، أو إنشاء علامات صور سريعة الاستجابة، أو إعداد مسارات الصور المحملة بواسطة المستخدم، أو تكوين تحويلات الصور. يغطي نقطة النهاية /.netlify/images، ومعلمات الاستعلام، والقائمة المسموح بها للصور عن بعد، وإعادة كتابة عناوين URL النظيفة، وإنشاء التحميلات باستخدام Functions + Blobs.
