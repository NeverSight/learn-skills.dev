Guide d'utilisation de Netlify DB (Gestion Neon Postgres). À utiliser lorsque le projet a besoin d'une base de données relationnelle, d'un stockage de données structuré, de requêtes SQL ou de données qui augmenteront au fil du temps. Couvre le provisionnement, le SQL brut via @netlify/neon, l'intégration de Drizzle ORM, les migrations et le déploiement de branchements en version préliminaire. Indique également quand utiliser Netlify Blobs à la place.
