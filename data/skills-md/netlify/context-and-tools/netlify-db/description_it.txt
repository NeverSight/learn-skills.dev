Guida all'utilizzo di Netlify DB (Neon Postgres gestito). Da utilizzare quando il progetto necessita di un database relazionale, archiviazione di dati strutturati, query SQL o dati che cresceranno nel tempo. Copre il provisioning, l'SQL raw tramite @netlify/neon, l'integrazione di Drizzle ORM, le migrazioni e la distribuzione delle ramificazioni in anteprima. Copre anche quando utilizzare invece i BLOB di Netlify.
