Руководство по использованию Netlify DB (управляемый Neon Postgres). Используйте, когда проекту требуется реляционная база данных, структурированное хранилище данных, SQL-запросы или данные, которые со временем будут расти. Охватывает подготовку, необработанный SQL через @netlify/neon, интеграцию Drizzle ORM, миграцию и развертывание предварительных ветвей. Также рассказывается, когда вместо этого следует использовать Netlify Blobs.
