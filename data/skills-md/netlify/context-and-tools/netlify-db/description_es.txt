Guía para usar Netlify DB (Neon Postgres administrado). Úselo cuando el proyecto necesite una base de datos relacional, almacenamiento de datos estructurados, consultas SQL o datos que crecerán con el tiempo. Cubre el aprovisionamiento, SQL sin formato a través de @netlify/neon, la integración de Drizzle ORM, las migraciones y la implementación de ramificaciones de vista previa. También cubre cuándo usar Netlify Blobs en su lugar.
