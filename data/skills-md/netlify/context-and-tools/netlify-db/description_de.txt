Leitfaden zur Verwendung von Netlify DB (verwaltetes Neon Postgres). Verwenden Sie diese Option, wenn das Projekt eine relationale Datenbank, strukturierte Datenspeicherung, SQL-Abfragen oder Daten benötigt, die mit der Zeit wachsen. Deckt Bereitstellung, Raw-SQL über @netlify/neon, Drizzle ORM-Integration, Migrationen und Deploy-Preview-Branching ab. Behandelt auch, wann stattdessen Netlify Blobs verwendet werden sollten.
