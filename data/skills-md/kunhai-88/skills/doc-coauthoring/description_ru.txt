Направляйте пользователей через структурированные рабочие процессы для совместной работы над документами. Используется, когда пользователь хочет написать документ, предложение, техническую спецификацию, документ решения или аналогичный структурированный контент. Этот рабочий процесс помогает пользователям эффективно доставлять контекст, уточнять контент посредством итерации и проверять, насколько документы эффективны для читателей. Триггерные слова: написание документов, создание предложений, составление спецификаций или аналогичные задачи с документами.
