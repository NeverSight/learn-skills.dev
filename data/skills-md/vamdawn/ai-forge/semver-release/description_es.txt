Flujo de trabajo de lanzamiento de versión automatizado. Analiza el historial de confirmaciones de git para inferir la versión semántica, detecta automáticamente archivos de versión en todos los ecosistemas, actualiza CHANGELOG multilingües, crea confirmaciones y etiquetas de git. Úselo cuando: (1) el usuario dice "liberar", "publicar versión", "versión mejorada", (2) el usuario invoca el comando /liberar, (3) se prepara para lanzar una nueva versión.
