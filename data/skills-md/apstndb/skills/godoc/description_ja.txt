`go doc` を使用して Go パッケージのドキュメント、型、関数、メソッド、およびソース コードを効率的に検索する方法。
Go コードを操作し、標準ライブラリ パッケージであっても、API を理解する必要がある場合は常にこのスキルを使用してください。
サードパーティの依存関係、または現在のプロジェクト独自のパッケージ。 Go のタイプ、インターフェイス、
関数シグネチャ、メソッド セット、定数、または変数。次の場合は、ソース ファイルを直接読むよりも「go doc」を優先します。
「go doc」はノイズなしで正確なドキュメントと署名を提供するため、Go API が何をするのかを理解する必要があります。
ユーザーが Go ドキュメントや godoc について質問したり、Go パッケージのパブリック API を探索したい場合にも、このスキルを使用します。
