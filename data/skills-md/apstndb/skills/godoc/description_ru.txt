Как использовать go doc для эффективного поиска документации, типов, функций, методов и исходного кода пакетов Go.
Используйте этот навык всякий раз, когда работаете с кодом Go, и вам необходимо понимать API — будь то стандартный пакет библиотеки,
сторонняя зависимость или собственные пакеты текущего проекта. Используйте этот навык при изучении типов Go, интерфейсов,
сигнатуры функций, наборы методов, константы или переменные. Предпочитайте `go doc` чтению исходных файлов напрямую, когда вы
нужно понимать, что делает Go API, потому что go doc предоставляет вам именно документацию и подписи без шума.
Также используйте этот навык, когда пользователь спрашивает о документации Go, godoc или хочет изучить общедоступный API пакета Go.
