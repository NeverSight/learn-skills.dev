Cómo utilizar `go doc` para buscar documentación, tipos, funciones, métodos y código fuente del paquete Go de manera eficiente.
Utilice esta habilidad siempre que trabaje con código Go y necesite comprender una API, ya sea un paquete de biblioteca estándar,
una dependencia de terceros o los paquetes propios del proyecto actual. Activa esta habilidad al investigar tipos de Go, interfaces,
firmas de funciones, conjuntos de métodos, constantes o variables. Prefiere `go doc` a leer archivos fuente directamente cuando
Necesito entender qué hace una API Go, porque "go doc" le brinda exactamente la documentación y las firmas sin ruido.
Utilice también esta habilidad cuando el usuario pregunte sobre la documentación de Go, godoc o quiera explorar la API pública de un paquete de Go.
