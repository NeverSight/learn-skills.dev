Scrivi ed esegui valutazioni LLM utilizzando la libreria evalite TypeScript (basata su Vitest). Utilizzare quando si creano file .eval.ts, si scrivono valutatori, si tracciano chiamate LLM, si confrontano varianti di modelli, si configura evalite, si eseguono valutazioni in CI o quando l'utente menziona "evalite", "evals", "valutazione LLM", "output LLM punteggio" o "eval runner". Si attiva anche quando vedi importazioni da "evalite", "evalite/traces", "evalite/ai-sdk", "evalite/config", "evalite/runner" o "autoevals".
