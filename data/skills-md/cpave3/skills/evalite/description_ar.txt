قم بكتابة وتشغيل تقييمات LLM باستخدام مكتبة TypeScript evalite (المبنية على Vitest). يُستخدم عند إنشاء ملفات .eval.ts، أو كتابة الهدافين، أو تتبع مكالمات LLM، أو مقارنة متغيرات النماذج، أو تكوين evalite، أو تشغيل التقييمات في CI، أو عندما يذكر المستخدم "evalite"، أو "evals"، أو "تقييم LLM"، أو "تسجيل مخرجات LLM"، أو "eval runner". يتم تشغيله أيضًا عندما ترى عمليات استيراد من "evalite"، أو "evalite/traces"، أو "evalite/ai-sdk"، أو "evalite/config"، أو "evalite/runner"، أو "autoevals".
