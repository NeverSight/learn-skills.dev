Erstellen Sie neue Fähigkeiten, ändern und verbessern Sie bestehende Fähigkeiten und messen Sie die Fähigkeitsleistung. Verwenden Sie diese Option, wenn Benutzer einen neuen Skill erstellen (oder einen vorhandenen Skill aktualisieren) möchten, der Claudes Fähigkeiten um Spezialwissen, Arbeitsabläufe oder Tool-Integrationen erweitert. Verwenden Sie dies auch, wenn Benutzer Evaluierungen durchführen möchten, um eine Fertigkeit zu testen, die Fertigkeitsleistung zu vergleichen, die Beschreibung einer Fertigkeit für eine bessere Auslösegenauigkeit zu optimieren oder wenn sie sagen: „Daraus in eine Fertigkeit umwandeln“. Lösen Sie jede Erwähnung von Fähigkeiten zum Erstellen, Schreiben, Aufbauen, Verbessern oder Testen aus.
