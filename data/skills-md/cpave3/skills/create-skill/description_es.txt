Cree nuevas habilidades, modifique y mejore las existentes y mida el desempeño de las habilidades. Úselo cuando los usuarios quieran crear una nueva habilidad (o actualizar una habilidad existente) que amplíe las capacidades de Claude con conocimientos especializados, flujos de trabajo o integraciones de herramientas. Utilícelo también cuando los usuarios quieran ejecutar evaluaciones para probar una habilidad, comparar el rendimiento de una habilidad, optimizar la descripción de una habilidad para mejorar la precisión de activación o cuando digan "convierta esto en una habilidad". Activa cualquier mención de creación, escritura, desarrollo, mejora o prueba de habilidades.
