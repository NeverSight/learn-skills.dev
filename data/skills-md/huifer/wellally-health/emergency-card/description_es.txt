Genere una tarjeta de resumen de información médica concisa para un acceso rápido en emergencias. Utilice esta habilidad cuando el usuario necesite viajar, prepararse para una visita médica, enfrentar una emergencia o solicitar "información de emergencia", "tarjeta médica" o "información de primeros auxilios". Extraiga datos clave (alergias, medicamentos, afecciones agudas, implantes), admita salida multiformato (JSON, texto, código QR) para primeros auxilios o atención rápida.
