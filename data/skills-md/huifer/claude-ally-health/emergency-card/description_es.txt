Genera una tarjeta resumen de información médica de acceso rápido para emergencias. Utilice esta habilidad cuando el usuario esté viajando, preparándose para una visita médica, en caso de emergencia o solicitando "información de emergencia", "tarjeta médica" o "información de primeros auxilios". Extraiga detalles clave (alergias, medicamentos, afecciones, implantes), admita múltiples formatos de salida (JSON, texto, código QR) para primeros auxilios o atención médica rápida.
