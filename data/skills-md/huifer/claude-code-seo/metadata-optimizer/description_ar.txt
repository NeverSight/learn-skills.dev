تحليل البيانات الوصفية وتحسينها لمشاريع Next.js، بما في ذلك العنوان والوصف والرسم البياني المفتوح وبطاقات تويتر. اكتشاف جهاز توجيه التطبيقات أو جهاز توجيه الصفحات تلقائيًا، وتقديم توصيات طويلة، وتحسين الكلمات الرئيسية، وإرشادات أفضل الممارسات. يدعم تحليل تحسين محركات البحث ثنائي اللغة باللغتين الصينية والإنجليزية.
