Проанализируйте темы и ключевые слова на основе содержания проекта, чтобы разработать стратегию работы с гостевым блогом. Предоставьте синтаксис расширенного поиска Google, список источников гостевых блогов (читайте из resources/guest-blog-sources.md) и шаблон электронного письма (читайте из templates/guest-blog-pitch.md). Автоматически оценивайте качество гостевого блога (DA, релевантность, трафик) и предоставляйте персонализированные рекомендации по отправке с приоритетом.
