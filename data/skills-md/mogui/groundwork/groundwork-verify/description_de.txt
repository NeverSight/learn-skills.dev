Verwenden Sie diese Option, wenn der Benutzer überprüfen möchte, ob der implementierte Code seinen Spezifikationen entspricht, Integrationstests aus Feature-Dateien generieren oder prüfen möchte, ob der Code noch bestehende Szenarien erfüllt. Trigger, nachdem die Implementierung eine Funktion abgeschlossen hat. Wird auch verwendet, wenn der Benutzer fragt: „Tut der Code das, was wir angegeben haben?“ oder „Tests aus den Feature-Dateien generieren“.
