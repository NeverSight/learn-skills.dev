À utiliser lorsque l'utilisateur souhaite valider que le code implémenté correspond à ses spécifications, générer des tests d'intégration à partir de fichiers de fonctionnalités ou vérifier si le code satisfait toujours aux scénarios existants. Déclenchement après que la mise en œuvre ait terminé une fonctionnalité. À utiliser également lorsque l'utilisateur demande « le code fait-il ce que nous avons spécifié ? » ou "générer des tests à partir des fichiers de fonctionnalités".
