Fügen Sie die Auflösung und Anzeige von .skr-Domänennamen zu Solana Mobile React Native-Apps hinzu. Wird verwendet, wenn der Benutzer die Integration der .skr-Domänenvalidierung, die Auflösung von .skr-Domänen in Wallet-Adressen, die Anzeige von .skr-Namen anstelle von Adressen, das Hinzufügen einer Rückwärtssuche von Adressen zu Domänen oder die Implementierung von .skr-Namensfunktionen überall dort anfordert, wo Wallet-Adressen angezeigt werden (Profile, Freundeslisten, Transaktionsverlauf usw.).
