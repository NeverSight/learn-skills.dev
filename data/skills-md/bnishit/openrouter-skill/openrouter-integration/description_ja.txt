OpenRouter の OpenAI 互換 API を通じて、アプリを 300 以上の AI モデルに接続します。モデル検出、画像生成、マルチモーダル チャット (テキスト、画像、PDF)、正確な世代ごとのコスト追跡、フォールバックを使用したプロバイダー ルーティング、安全制限付きのツール呼び出し、構造化出力検証、フリーモデル フィルタリング、スターター テンプレート (Next.js、Express)、アセット ワークフロー (アイコン、OG イメージ)、プロダクション プレイブック、検証スクリプトなどです。エージェントが OpenRouter の使用法を追加またはデバッグする必要がある場合、モデル ピッカー、プロキシ `/api/v1/models`、`/api/v1/models/user`、`/api/v1/providers`、または `/api/v1/generation` を構築する必要がある場合に使用し、画像または PDF コンテンツを `/api/v1/chat/completions` に送信し、OpenRouter を介して画像を生成し、アイコンまたは OG イメージを構築する必要があります。フロー、OpenRouter 応答の解析、事後の生成コストの検査、Next.js または Express サーバー ルートの追加、構造化出力の検証、スモーク テストの実行、コーディング前に現在のドキュメントを OpenRouter に対して検証する、またはモデルまたはプロバイダーのフォールバックをサーバーまたは UI に配線します。
