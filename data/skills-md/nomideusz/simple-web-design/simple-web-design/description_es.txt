Aplique principios de diseño web simples y probados para criticar, auditar, mejorar o crear UI y UX de sitios web. Active esta habilidad cada vez que un usuario solicite comentarios sobre el diseño, una revisión de UX, una crítica de la página de destino o ayuda para diseñar cualquier interfaz web, incluso cuando se exprese casualmente como "¿se ve bien?", "¿qué le pasa a mi sitio?", "¿cómo puedo mejorar la UX?", "revisar mi página de inicio", "hacer que mi aplicación se sienta más limpia" o "ayúdame a diseñar una página". También se aplica cuando el usuario proporciona código HTML/CSS o datos de imagen para revisión del diseño. Esta habilidad se basa en el marco de "Diseño web simple" de Anthony Hobday: 15 principios probados en batalla que priorizan el contenido, la claridad y la usabilidad a largo plazo sobre la complejidad visual y la búsqueda de tendencias.
