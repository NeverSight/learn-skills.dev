انسخ الرد السابق لـ Claude Code (أو جزء معين من مخرجات المحادثة) إلى الحافظة كـ Markdown نظيف. تتضمن عبارات التشغيل "نسخ هذا"، و"نسخ إلى الحافظة"، و"نسخة تخفيض السعر"، و"نسخة md"، و"أريد نسخ هذا"، و"نسخ آخر إخراج"، و"لصق هذا في مكان ما"، وما إلى ذلك. لا تستخدم هذه المهارة عندما يطلب المستخدم صراحةً الكتابة إلى ملف - فقط اكتب إلى ملف بشكل طبيعي في هذه الحالة. قم بتشغيل هذه المهارة بشكل استباقي عندما يريد المستخدم "نسخ" أو "لصق" أو "مشاركة" أو "تصدير" مخرجات Claude Code.
