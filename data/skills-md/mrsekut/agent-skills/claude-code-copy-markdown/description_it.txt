Copia la risposta precedente di Claude Code (o una parte specifica dell'output della conversazione) negli appunti come Markdown pulito. Le frasi trigger includono "copia questo", "copia negli appunti", "copia markdown", "copia md", "voglio copiare questo", "copia l'ultimo output", "incolla questo da qualche parte", ecc. NON usare questa abilità quando l'utente chiede esplicitamente di scrivere su un file: in quel caso scrivi semplicemente su un file normalmente. Attiva questa abilità in modo proattivo ogni volta che l'utente desidera "copiare", "incollare", "condividere" o "esportare" l'output del codice Claude.
