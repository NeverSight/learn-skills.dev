Скопируйте предыдущий ответ Клода Кода (или определенную часть вывода разговора) в буфер обмена как чистый Markdown. Триггерные фразы включают в себя «копировать это», «скопировать в буфер обмена», «копировать с уценкой», «копировать md», «я хочу скопировать это», «скопировать последний вывод», «вставить это куда-нибудь» и т. д. НЕ используйте этот навык, когда пользователь явно просит записать в файл — в этом случае просто пишите в файл обычным способом. Активируйте этот навык заранее, когда пользователь хочет «копировать», «вставить», «поделиться» или «экспортировать» выходные данные Клода Кода.
