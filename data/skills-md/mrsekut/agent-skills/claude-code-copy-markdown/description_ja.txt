Claude Code の以前の応答 (または会話出力の特定の部分) をクリーンな Markdown としてクリップボードにコピーします。トリガー フレーズには、「これをコピー」、「クリップボードにコピー」、「マークダウン コピー」、「MD コピー」、「これをコピーしたい」、「最後の出力をコピー」、「これをどこかに貼り付け」などが含まれます。ユーザーがファイルへの書き込みを明示的に要求した場合は、このスキルを使用しないでください。その場合は、通常どおりファイルに書き込むだけです。ユーザーがクロード コードの出力を「コピー」、「貼り付け」、「共有」、または「エクスポート」したい場合は常に、このスキルを積極的にトリガーします。
