Automatisez la soumission de nouvelles extensions Chrome au Chrome Web Store à l'aide de playwright-mcp. Lit le fichier manifest.json et le code source, génère le texte de la liste des magasins dans n'importe quelle langue, crée des images de magasin et remplit tous les formulaires requis via l'automatisation du navigateur. À utiliser lors de la soumission d'une nouvelle extension Chrome au Chrome Web Store, de la publication d'une extension Chrome ou du remplissage des formulaires du tableau de bord des développeurs du Chrome Web Store.
