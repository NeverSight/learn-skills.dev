Автоматизируйте отправку новых расширений Chrome в Интернет-магазин Chrome с помощью playwright-mcp. Считывает файл манифеста.json и исходный код, генерирует текст списка магазинов на любом языке, создает изображения магазинов и заполняет все необходимые формы с помощью автоматизации браузера. Используйте при отправке нового расширения Chrome в Интернет-магазин Chrome, публикации расширения Chrome или заполнении форм панели инструментов разработчика Интернет-магазина Chrome.
