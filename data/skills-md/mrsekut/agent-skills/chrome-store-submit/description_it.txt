Automatizza l'invio di nuove estensioni Chrome al Chrome Web Store utilizzando playwright-mcp. Legge manifest.json e il codice sorgente, genera testo della scheda dello Store in qualsiasi lingua, crea immagini del negozio e compila tutti i moduli richiesti tramite l'automazione del browser. Da utilizzare quando si invia una nuova estensione di Chrome al Chrome Web Store, si pubblica un'estensione di Chrome o si compilano i moduli della dashboard per sviluppatori di Chrome Web Store.
