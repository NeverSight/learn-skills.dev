Automatice el envío de nuevas extensiones de Chrome a Chrome Web Store usando playwright-mcp. Lee manifest.json y el código fuente, genera texto de listado de tiendas en cualquier idioma, crea imágenes de tiendas y completa todos los formularios necesarios a través de la automatización del navegador. Úselo al enviar una nueva extensión de Chrome a Chrome Web Store, publicar una extensión de Chrome o completar formularios del panel de desarrollador de Chrome Web Store.
