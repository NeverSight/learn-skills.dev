أتمتة إرسال ملحق Chrome الجديد إلى سوق Chrome الإلكتروني باستخدام playwright-mcp. يقرأ ملفmani.json والكود المصدري، وينشئ نص قائمة المتجر بأي لغة، وينشئ صور المتجر، ويملأ جميع النماذج المطلوبة عبر أتمتة المتصفح. يُستخدم عند إرسال ملحق Chrome جديد إلى سوق Chrome الإلكتروني، أو نشر ملحق Chrome، أو ملء نماذج لوحة تحكم مطوري سوق Chrome الإلكتروني.
