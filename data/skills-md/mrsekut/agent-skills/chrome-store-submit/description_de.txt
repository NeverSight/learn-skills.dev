Automatisieren Sie die Übermittlung neuer Chrome-Erweiterungen an den Chrome Web Store mit playwright-mcp. Liest manifest.json und Quellcode, generiert Store-Eintragstext in jeder Sprache, erstellt Store-Bilder und füllt alle erforderlichen Formulare über die Browser-Automatisierung aus. Verwenden Sie diese Option, wenn Sie eine neue Chrome-Erweiterung an den Chrome Web Store senden, eine Chrome-Erweiterung veröffentlichen oder Entwickler-Dashboard-Formulare für den Chrome Web Store ausfüllen.
