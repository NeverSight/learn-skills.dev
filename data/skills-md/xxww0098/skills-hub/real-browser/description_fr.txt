Lancez un vrai Chrome avec l'état de connexion de l'utilisateur (profil cloné), exposant CDP sur un port (9851 par défaut). Prend en charge le mode sans tête (par défaut) ou avec tête, les mises à jour de profil incrémentielles, la journalisation personnalisable, la sortie lisible par machine et les arguments Chrome supplémentaires via des variables d'environnement. Nécessite la compétence « agent-browser » pour toutes les commandes du navigateur.
