قم بتشغيل Chrome حقيقي بحالة تسجيل دخول المستخدم (ملف تعريف مستنسخ)، وكشف CDP على منفذ (9851 افتراضي). يدعم الوضع بدون رأس (افتراضي) أو الوضع الرأسي، وتحديثات الملف الشخصي المتزايدة، والتسجيل القابل للتخصيص، والمخرجات المقروءة آليًا، ووسائط Chrome الإضافية عبر متغيرات البيئة. يتطلب مهارة "agent-browser" لجميع أوامر المتصفح.
