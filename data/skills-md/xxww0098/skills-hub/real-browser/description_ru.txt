Запустите настоящий Chrome с состоянием входа пользователя (клонированный профиль), выставив CDP на порту (по умолчанию 9851). Поддерживает безголовый (по умолчанию) или возглавляемый режим, дополнительные обновления профиля, настраиваемое ведение журнала, машиночитаемый вывод и дополнительные аргументы Chrome через переменные среды. Требуется навык `agent-browser` для всех команд браузера.
