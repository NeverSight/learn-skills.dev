Recherchez des produits sur les plateformes de commerce électronique chinoises : Taobao, Tmall, XHS (小红书). Zéro configuration : aucune clé API nécessaire. Propulsé par la base de données de produits unifiée Shopme. À utiliser lorsque l'utilisateur demande à trouver des produits, à obtenir des informations sur les produits par URL ou ID, à rechercher des fournisseurs chinois ou à comparer les prix.
