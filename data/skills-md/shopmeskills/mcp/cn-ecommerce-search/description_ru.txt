Ищите товары на китайских платформах электронной коммерции: Taobao, Tmall, XHS (小红书). Нулевая конфигурация — ключи API не нужны. Работает на базе единой базы данных продуктов Shopme. Используйте, когда пользователь просит найти продукты, получить информацию о продукте по URL-адресу или идентификатору, выполнить поиск китайских поставщиков или сравнить цены.
