Busque productos en plataformas de comercio electrónico chinas: Taobao, Tmall, XHS (小红书). Configuración cero: no se necesitan claves API. Desarrollado por la base de datos de productos unificada de Shopme. Úselo cuando el usuario solicite buscar productos, obtener información del producto por URL o ID, buscar proveedores chinos o comparar precios.
