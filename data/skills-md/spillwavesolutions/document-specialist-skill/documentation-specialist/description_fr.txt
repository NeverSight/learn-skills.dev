Cette compétence doit être utilisée lors de la création de documentation logicielle professionnelle (SRS, PRD, OpenAPI,
manuels d'utilisation, tutoriels, runbooks) à partir de modèles (greenfield) ou de documentation de rétro-ingénierie
à partir de code existant comme Spring Boot ou FastAPI (brownfield). Gère également les audits/examens de la documentation,
conversion de format (Markdown, DOCX, PDF) et génération de diagrammes (C4, Mermaid, PlantUML, ER, séquence).
À utiliser lorsqu'on vous demande de « créer de la documentation », « documenter mon code », « écrire SRS », « générer un PRD » ou « spécialiste de la documentation ».
