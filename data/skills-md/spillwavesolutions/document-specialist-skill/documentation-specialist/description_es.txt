Esta habilidad debe utilizarse al crear documentación de software profesional (SRS, PRD, OpenAPI,
manuales de usuario, tutoriales, runbooks) a partir de plantillas (greenfield) o documentación de ingeniería inversa
a partir de código existente como Spring Boot o FastAPI (brownfield). También maneja auditorías/revisiones de documentación,
conversión de formatos (Markdown, DOCX, PDF) y generación de diagramas (C4, Mermaid, PlantUML, ER, secuencia).
Úselo cuando se le solicite "crear documentación", "documentar mi código", "escribir SRS", "generar PRD" o "especialista en documentación".
