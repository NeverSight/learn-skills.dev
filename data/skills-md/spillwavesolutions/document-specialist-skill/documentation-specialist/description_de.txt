Diese Fähigkeit sollte bei der Erstellung professioneller Softwaredokumentation (SRS, PRD, OpenAPI,
Benutzerhandbücher, Tutorials, Runbooks) aus Vorlagen (Greenfield) oder Reverse-Engineering-Dokumentation
aus vorhandenem Code wie Spring Boot oder FastAPI (Brownfield). Führt auch Dokumentationsprüfungen/-überprüfungen durch,
Formatkonvertierung (Markdown, DOCX, PDF) und Diagrammerstellung (C4, Mermaid, PlantUML, ER, Sequenz).
Verwenden Sie diese Option, wenn Sie nach „Dokumentation erstellen“, „Meinen Code dokumentieren“, „SRS schreiben“, „PRD generieren“ oder „Dokumentationsspezialist“ gefragt werden.
