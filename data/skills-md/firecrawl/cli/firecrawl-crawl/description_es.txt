Extraiga contenido de forma masiva de un sitio web completo o de una sección del sitio. Utilice esta habilidad cuando el usuario quiera rastrear un sitio, extraer todas las páginas de una sección de documentos, extraer de forma masiva varias páginas siguiendo enlaces, o decir "rastrear", "obtener todas las páginas", "extraer todo en /docs", "extraer en masa" o necesite contenido de muchas páginas en el mismo sitio. Maneja límites de profundidad, filtrado de rutas y extracción simultánea.
