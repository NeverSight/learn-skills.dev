Extrahieren Sie Inhalte in großen Mengen aus einer gesamten Website oder einem Website-Bereich. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine Website crawlen, alle Seiten aus einem Dokumentabschnitt extrahieren, mehrere Seiten nach Links in großen Mengen durchsuchen möchte oder „Crawlen“, „Alle Seiten abrufen“, „Alles unter /docs extrahieren“, „Massenextrahieren“ sagen möchte oder Inhalte von vielen Seiten derselben Website benötigt. Behandelt Tiefenbeschränkungen, Pfadfilterung und gleichzeitige Extraktion.
