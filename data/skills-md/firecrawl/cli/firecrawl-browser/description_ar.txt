أتمتة المتصفح السحابي للصفحات التي تتطلب التفاعل - النقرات، وملء النماذج، وتسجيل الدخول، وترقيم الصفحات، والتمرير اللانهائي. استخدم هذه المهارة عندما يحتاج المستخدم إلى التفاعل مع صفحة ويب، أو تسجيل الدخول إلى موقع، أو النقر على الأزرار، أو ملء النماذج، أو التنقل في التدفقات متعددة الخطوات، أو التعامل مع ترقيم الصفحات، أو عندما تفشل عملية المسح العادية لأن المحتوى يتطلب تفاعل JavaScript. يتم تشغيلها عند "النقر" أو "ملء النموذج" أو "تسجيل الدخول إلى" أو "مرقّم الصفحات" أو "التمرير اللانهائي" أو "التفاعل مع الصفحة" أو "فشل الاستخراج". يوفر جلسات Chromium عن بعد مع ملفات تعريف مستمرة.
