インタラクションが必要なページのクラウド ブラウザ自動化 (クリック、フォーム入力、ログイン、ページネーション、無限スクロール)。このスキルは、ユーザーが Web ページの操作、サイトへのログイン、ボタンのクリック、フォームへの入力、複数ステップのフローの移動、ページネーションの処理を行う必要がある場合、またはコンテンツに JavaScript インタラクションが必要なために通常のスクレイピングが失敗する場合に使用します。 「クリック」、「フォームへの記入」、「ログイン」、「ページ分割」、「無限スクロール」、「ページの操作」、または「スクレイピングの失敗」でトリガーされます。永続的なプロファイルを使用したリモート Chromium セッションを提供します。
