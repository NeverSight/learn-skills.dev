Cloud-Browser-Automatisierung für Seiten, die Interaktion erfordern – Klicks, Ausfüllen von Formularen, Anmeldung, Paginierung, unendliches Scrollen. Verwenden Sie diese Fähigkeit, wenn der Benutzer mit einer Webseite interagieren, sich bei einer Website anmelden, auf Schaltflächen klicken, Formulare ausfüllen, in mehrstufigen Abläufen navigieren, Paginierung verwalten muss oder wenn das reguläre Scraping fehlschlägt, weil der Inhalt eine JavaScript-Interaktion erfordert. Wird ausgelöst bei „Klick“, „Formular ausfüllen“, „Anmelden bei“, „Paginierung“, „unendliches Scrollen“, „Mit der Seite interagieren“ oder „Scrape fehlgeschlagen“. Bietet Remote-Chromium-Sitzungen mit dauerhaften Profilen.
