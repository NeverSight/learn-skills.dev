Scarica un intero sito web come file locali: markdown, screenshot o più formati per pagina. Utilizza questa abilità quando l'utente desidera salvare un sito localmente, scaricare documentazione per l'utilizzo offline, salvare in blocco pagine come file o dice "scarica il sito", "salva come file locale", "copia offline", "scarica tutti i documenti" o "salva per riferimento". Combina la mappatura e lo scraping del sito in directory locali organizzate.
