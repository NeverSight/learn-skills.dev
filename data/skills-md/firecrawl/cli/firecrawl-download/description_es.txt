Descargue un sitio web completo como archivos locales: rebajas, capturas de pantalla o múltiples formatos por página. Utilice esta habilidad cuando el usuario quiera guardar un sitio localmente, descargar documentación para usar sin conexión, guardar páginas en masa como archivos o decir "descargar el sitio", "guardar como archivos locales", "copiar sin conexión", "descargar todos los documentos" o "guardar como referencia". Combina mapeo de sitios y raspado en directorios locales organizados.
