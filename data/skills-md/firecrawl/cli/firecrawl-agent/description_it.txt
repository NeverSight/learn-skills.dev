Estrazione autonoma dei dati basata sull'intelligenza artificiale che naviga in siti complessi e restituisce JSON strutturato. Utilizza questa competenza quando l'utente desidera dati strutturati da siti Web, deve estrarre livelli di prezzo, elenchi di prodotti, voci di directory o qualsiasi dato come JSON con uno schema. Si attiva su "estrai dati strutturati", "ottieni tutti i prodotti", "estrai informazioni sui prezzi", "estrai come JSON" o quando l'utente fornisce uno schema JSON per i dati del sito web. Più potente del semplice scraping per l'estrazione strutturata di più pagine.
