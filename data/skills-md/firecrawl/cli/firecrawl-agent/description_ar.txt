استخراج البيانات بشكل مستقل مدعوم بالذكاء الاصطناعي والذي يتنقل في المواقع المعقدة ويعيد JSON المنظم. استخدم هذه المهارة عندما يريد المستخدم بيانات منظمة من مواقع الويب، أو يحتاج إلى استخراج طبقات التسعير، أو قوائم المنتجات، أو إدخالات الدليل، أو أي بيانات بتنسيق JSON مع مخطط. يتم تشغيلها عند "استخراج البيانات المنظمة"، أو "الحصول على جميع المنتجات"، أو "سحب معلومات التسعير"، أو "استخراج كـ JSON"، أو عندما يقدم المستخدم مخطط JSON لبيانات موقع الويب. أقوى من عملية الكشط البسيطة للاستخراج المنظم متعدد الصفحات.
