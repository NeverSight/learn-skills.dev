Extraction de données autonome basée sur l'IA qui parcourt des sites complexes et renvoie du JSON structuré. Utilisez cette compétence lorsque l'utilisateur souhaite des données structurées à partir de sites Web, doit extraire des niveaux de tarification, des listes de produits, des entrées de répertoire ou toute autre donnée au format JSON avec un schéma. Déclencheurs sur « extraire des données structurées », « obtenir tous les produits », « extraire des informations sur les prix », « extraire en JSON » ou lorsque l'utilisateur fournit un schéma JSON pour les données du site Web. Plus puissant que le simple scraping pour une extraction structurée de plusieurs pages.
