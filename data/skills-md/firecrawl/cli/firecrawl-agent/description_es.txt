Extracción de datos autónoma impulsada por IA que navega por sitios complejos y devuelve JSON estructurado. Utilice esta habilidad cuando el usuario quiera datos estructurados de sitios web, necesite extraer niveles de precios, listados de productos, entradas de directorio o cualquier dato como JSON con un esquema. Se activan al "extraer datos estructurados", "obtener todos los productos", "extraer información de precios", "extraer como JSON" o cuando el usuario proporciona un esquema JSON para los datos del sitio web. Más potente que el simple raspado para extracción estructurada de varias páginas.
