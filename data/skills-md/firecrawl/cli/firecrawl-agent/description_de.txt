KI-gestützte autonome Datenextraktion, die durch komplexe Websites navigiert und strukturiertes JSON zurückgibt. Verwenden Sie diese Fähigkeit, wenn der Benutzer strukturierte Daten von Websites wünscht, Preisstufen, Produktlisten, Verzeichniseinträge oder andere Daten als JSON mit einem Schema extrahieren muss. Wird ausgelöst bei „Strukturierte Daten extrahieren“, „Alle Produkte abrufen“, „Preisinformationen abrufen“, „Als JSON extrahieren“ oder wenn der Benutzer ein JSON-Schema für Website-Daten bereitstellt. Leistungsstärker als einfaches Scraping für mehrseitige strukturierte Extraktion.
