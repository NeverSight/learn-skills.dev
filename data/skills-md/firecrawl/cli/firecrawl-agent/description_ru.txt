Автономное извлечение данных на базе искусственного интеллекта, позволяющее перемещаться по сложным сайтам и возвращать структурированный JSON. Используйте этот навык, когда пользователю нужны структурированные данные с веб-сайтов, ему необходимо извлечь ценовые уровни, списки продуктов, записи каталогов или любые данные в формате JSON со схемой. Срабатывает при «извлечении структурированных данных», «получении всех продуктов», «извлечении информации о ценах», «извлечении в формате JSON» или когда пользователь предоставляет схему JSON для данных веб-сайта. Более мощный, чем простой парсинг, для многостраничного структурированного извлечения.
