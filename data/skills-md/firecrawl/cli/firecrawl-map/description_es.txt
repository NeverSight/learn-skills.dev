Descubra y enumere todas las URL de un sitio web, con filtrado de búsqueda opcional. Utilice esta habilidad cuando el usuario quiera encontrar una página específica en un sitio grande, enumerar todas las URL, ver la estructura del sitio, encontrar dónde está algo en un dominio o decir "mapear el sitio", "buscar la URL", "en qué páginas se encuentran" o "enumerar todas las páginas". Esencial cuando el usuario sabe qué sitio pero no qué página exacta.
