Découvrez et répertoriez toutes les URL d'un site Web, avec un filtrage de recherche facultatif. Utilisez cette compétence lorsque l'utilisateur souhaite rechercher une page spécifique sur un grand site, répertorier toutes les URL, voir la structure du site, trouver où se trouve quelque chose sur un domaine ou dire « mapper le site », « trouver l'URL de », « sur quelles pages se trouvent » ou « répertorier toutes les pages ». Indispensable lorsque l’utilisateur sait quel site mais pas quelle page exacte.
