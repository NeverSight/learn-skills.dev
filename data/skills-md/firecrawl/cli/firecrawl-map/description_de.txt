Entdecken und listen Sie alle URLs auf einer Website auf, mit optionaler Suchfilterung. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine bestimmte Seite auf einer großen Site finden, alle URLs auflisten, die Site-Struktur anzeigen, herausfinden möchte, wo sich etwas in einer Domain befindet, oder sagen möchte: „Site zuordnen“, „URL suchen“, „Auf welchen Seiten befinden sich“ oder „Alle Seiten auflisten“. Unverzichtbar, wenn der Benutzer weiß, um welche Seite es sich handelt, aber nicht genau, welche Seite.
