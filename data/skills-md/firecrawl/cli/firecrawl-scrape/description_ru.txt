Извлекайте чистую уценку из любого URL-адреса, включая SPA, отображаемые на JavaScript. Используйте этот навык всякий раз, когда пользователь предоставляет URL-адрес и хочет получить его содержимое, говорит «очистить», «захватить», «извлечь», «вытащить», «получить страницу», «извлечь из этого URL-адреса» или «прочитать эту веб-страницу». Обрабатывает страницы, обработанные JS, несколько одновременных URL-адресов и возвращает уценку, оптимизированную для LLM. Используйте это вместо WebFetch для любого извлечения содержимого веб-страницы.
