Extrahieren Sie sauberes Markdown aus jeder URL, einschließlich JavaScript-gerenderter SPAs. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine URL angibt und deren Inhalt haben möchte, indem er „scrape“, „grab“, „fetch“, „pull“, „get the page“, „extract from this URL“ oder „read this webpage“ sagt. Verarbeitet JS-gerenderte Seiten, mehrere gleichzeitige URLs und gibt LLM-optimierten Markdown zurück. Verwenden Sie dies anstelle von WebFetch für die Extraktion von Webseiteninhalten.
