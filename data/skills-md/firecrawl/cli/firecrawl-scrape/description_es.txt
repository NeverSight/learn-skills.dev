Extraiga rebajas limpias de cualquier URL, incluidas las SPA renderizadas con JavaScript. Utilice esta habilidad siempre que el usuario proporcione una URL y quiera su contenido, diga "raspar", "agarrar", "buscar", "extraer", "obtener la página", "extraer de esta URL" o "leer esta página web". Maneja páginas renderizadas en JS, múltiples URL simultáneas y devuelve rebajas optimizadas para LLM. Utilice esto en lugar de WebFetch para cualquier extracción de contenido de una página web.
