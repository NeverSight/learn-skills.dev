Scraping Web, recherche, exploration et automatisation du navigateur via la CLI Firecrawl. Utilisez cette compétence chaque fois que l'utilisateur souhaite effectuer une recherche sur le Web, trouver des articles, rechercher un sujet, rechercher quelque chose en ligne, gratter une page Web, récupérer le contenu d'une URL, extraire des données d'un site Web, explorer la documentation, télécharger un site ou interagir avec des pages nécessitant des clics ou des connexions. Utilisez également lorsqu'ils disent « récupérer cette page », « extraire le contenu de », « obtenir la page sur https:// » ou faire du scraping de références sur des sites Web externes. Cela permet une recherche Web en temps réel avec extraction de contenu de page complète et automatisation du navigateur cloud – des capacités au-delà de ce que Claude peut faire de manière native avec les outils intégrés.
