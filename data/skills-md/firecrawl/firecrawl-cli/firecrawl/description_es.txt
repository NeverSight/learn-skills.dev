Web scraping, búsqueda, rastreo y automatización del navegador a través de Firecrawl CLI. Utilice esta habilidad siempre que el usuario quiera buscar en la web, encontrar artículos, investigar un tema, buscar algo en línea, extraer una página web, obtener contenido de una URL, extraer datos de un sitio web, rastrear documentación, descargar un sitio o interactuar con páginas que necesitan clics o inicios de sesión. Utilícelo también cuando diga "buscar esta página", "extraer el contenido de", "obtener la página en https://" o hacer referencia a sitios web externos de raspado. Esto proporciona búsqueda web en tiempo real con extracción de contenido de página completa y automatización del navegador en la nube, capacidades que van más allá de lo que Claude puede hacer de forma nativa con herramientas integradas.
