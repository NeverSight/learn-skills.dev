Web-Scraping, Suche, Crawling und Browser-Automatisierung über die Firecrawl-CLI. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer das Web durchsuchen, Artikel finden, ein Thema recherchieren, etwas online nachschlagen, eine Webseite durchsuchen, Inhalte von einer URL abrufen, Daten von einer Website extrahieren, Dokumentation crawlen, eine Website herunterladen oder mit Seiten interagieren möchte, die Klicks oder Anmeldungen erfordern. Verwenden Sie es auch, wenn sie sagen „Diese Seite abrufen“, „Inhalt abrufen von“, „Seite unter https:// abrufen“ oder wenn Sie auf das Scraping externer Websites verweisen. Dies ermöglicht eine Echtzeit-Websuche mit vollständiger Seiteninhaltsextraktion und Cloud-Browser-Automatisierung – Funktionen, die über das hinausgehen, was Claude mit integrierten Tools nativ leisten kann.
