Firecrawl CLI を介した Web スクレイピング、検索、クローリング、ブラウザ自動化。ユーザーが Web の検索、記事の検索、トピックの調査、オンラインでの検索、Web ページのスクレイピング、URL からのコンテンツの取得、Web サイトからのデータの抽出、ドキュメントのクロール、サイトのダウンロード、またはクリックやログインが必要なページの操作を行う場合には、このスキルを使用します。 「このページを取得する」、「コンテンツを取得する」、「https:// でページを取得する」、または外部 Web サイトのスクレイピングを参照する場合にも使用します。これにより、フルページ コンテンツ抽出とクラウド ブラウザ自動化を備えたリアルタイム Web 検索が提供されます。これは、Claude が組み込みツールでネイティブに実行できる機能を超えています。ローカル ファイル操作、git コマンド、デプロイメント、またはコード編集タスクに対してトリガーしないでください。
