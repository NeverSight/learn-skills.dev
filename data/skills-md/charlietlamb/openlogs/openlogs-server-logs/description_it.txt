Recupera e controlla i log recenti del server locale nei repository che utilizzano openlogs o la "vecchia" CLI. Da utilizzare quando un utente chiede cosa è successo nel server, desidera un output recente del server di sviluppo, ha bisogno di errori di avvio o analisi dello stack oppure ti chiede di controllare i log di backend da "openlogs tail", log specifici dei comandi o ".openlogs/latest.txt".
