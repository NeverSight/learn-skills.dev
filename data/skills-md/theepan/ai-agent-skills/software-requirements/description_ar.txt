كتابة وثائق متطلبات البرامج الاحترافية (SRS، PRD، BRD، FRD، قصص المستخدم، حالات الاستخدام). يُستخدم عندما يطلب المستخدم كتابة المتطلبات، أو إنشاء SRS أو PRD، أو تحويل وصف الميزة إلى قصص مستخدم، أو إنشاء معايير القبول، أو مراجعة مستند المتطلبات الحالي، أو تحويل فكرة غير رسمية إلى متطلبات منظمة. يتم تشغيله أيضًا عندما يذكر المستخدم "معايير القبول"، أو "المتطلبات الوظيفية"، أو "المتطلبات غير الوظيفية"، أو "مصفوفة تتبع المتطلبات".
