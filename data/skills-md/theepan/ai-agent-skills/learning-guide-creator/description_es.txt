Cree guías de aprendizaje integrales (.md) de aproximadamente 50 páginas respaldadas por investigaciones sobre cualquier tema, escritas para profesionales técnicos experimentados. Utilice esta habilidad siempre que el usuario solicite crear una guía de aprendizaje, una guía de estudio, un documento de investigación, un documento de orientación, un documento informativo o cualquier solicitud para investigar un tema y producir un documento educativo de Markdown. Activa frases como "crear una guía sobre...", "Necesito entender...", "investigar y escribir sobre...", "ayúdame a ponerme al día sobre...", "escribe una guía de aprendizaje para...", "crea materiales de aprendizaje sobre...", "infórmame sobre...", "qué necesito saber sobre...", "enséñame sobre..." o "Quiero aprender sobre...". Esta habilidad combina una investigación web exhaustiva (de 10 a 15 búsquedas como mínimo con web_fetch para contenido completo) con escritura estructurada para producir guías densas y prácticas a nivel de ingeniero con marcos de decisión, pensamiento arquitectónico, mapas de ecosistemas, escenarios del mundo real y análisis de compensaciones. La salida es siempre Markdown (.md), nunca .docx. Sin fragmentos de código: se centra en conceptos, arquitectura, compensaciones, marcos de decisión y casos de uso del mundo real. Active siempre esta habilidad incluso para solicitudes de aprendizaje casuales: el usuario prefiere documentos completos a explicaciones conversacionales.
