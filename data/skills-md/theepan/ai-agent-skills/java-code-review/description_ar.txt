قم بمراجعة كود مصدر Java بحثًا عن الأخطاء والثغرات الأمنية ومشكلات الأداء ومشكلات التزامن ومشكلات جودة التعليمات البرمجية التي تم إنشاؤها بواسطة الذكاء الاصطناعي ومخاطر ترخيص التبعية وانتهاكات أفضل الممارسات. يُستخدم عندما يطلب المستخدم مراجعة كود Java، أو تدقيق ملفات Java، أو العثور على أخطاء في Java، أو التحقق من جودة كود Java، أو اكتشاف تباطؤ الذكاء الاصطناعي، أو التحقق من تراخيص المكتبة، أو إجراء مراجعة التعليمات البرمجية على ملفات .java. يقبل التعليمات البرمجية المقدمة مباشرة، كمسارات ملفات محلية، أو كمسارات دليل، أو كمخرجات فرق موحدة.
