قم بمراجعة ملفات مواصفات OpenAPI (Swagger) للتأكد من توافقها مع معيار OpenAPI 3.1 وتصميم نظام الأمان واصطلاحات التسمية وجودة المخطط وأفضل ممارسات تصميم واجهة برمجة التطبيقات. يُستخدم عندما يطلب المستخدم مراجعة مواصفات واجهة برمجة التطبيقات (API)، أو التحقق من ملف OpenAPI، أو تدقيق تعريف Swagger، أو التحقق من صحة مستند OAS، أو تحسين تصميم واجهة برمجة التطبيقات (API). يقبل ملفات YAML أو JSON، أو مسارات الملفات، أو محتوى المواصفات المضمنة.
