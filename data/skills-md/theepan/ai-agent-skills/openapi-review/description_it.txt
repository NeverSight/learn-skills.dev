Esamina i file delle specifiche OpenAPI (Swagger) per verificarne la conformità allo standard OpenAPI 3.1, alla progettazione dello schema di sicurezza, alle convenzioni di denominazione, alla qualità dello schema e alle best practice per la progettazione dell'API. Da utilizzare quando un utente chiede di rivedere una specifica API, controllare un file OpenAPI, controllare una definizione Swagger, convalidare un documento OAS o migliorare la progettazione dell'API. Accetta file YAML o JSON, percorsi di file o contenuti delle specifiche in linea.
