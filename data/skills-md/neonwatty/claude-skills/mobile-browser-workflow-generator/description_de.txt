Generiert, erstellt oder aktualisiert mobile Browser-Workflowdateien zum Testen von Web-Apps mit der Chrome-Mobile-Viewport-Emulation. Verwenden Sie dies, wenn der Benutzer „Workflows für mobile Browser generieren“, „Workflows für mobile Browser erstellen“, „Workflows für mobile Browser aktualisieren“, „Workflows für mobile Browser iterieren“ oder „Chrome Mobile-Tests“ sagt. Untersucht die Codebasis der Web-App, um alle benutzerorientierten Funktionen zu entdecken, und erstellt dann mobilspezifische Workflows mit iOS-HIG-Konformitätsprüfungen für den mobilen Chrome-Ansichtsbereich (393 x 852 iPhone 15 Pro-Abmessungen).
