Идиомы Rust и шаблоны проектирования для написания идиоматического кода. Используйте, когда: (1) Написание нового кода Rust, (2) Проверка кода Rust, (3) Решение проблем с проверкой заимствований, (4) Проектирование API-интерфейсов Rust. Запускается по: проблемам владения Rust, ошибкам времени жизни, вопросам дизайна API, «как идиоматически реализовать X в Rust».
