Idiomes Rust et modèles de conception pour l'écriture de code idiomatique. À utiliser lorsque : (1) écriture d'un nouveau code Rust, (2) révision du code Rust, (3) résolution des problèmes du vérificateur d'emprunt, (4) conception d'API Rust. Déclencheurs sur : problèmes de propriété de Rust, erreurs de durée de vie, questions de conception d'API, "comment faire X de manière idiomatique dans Rust".
