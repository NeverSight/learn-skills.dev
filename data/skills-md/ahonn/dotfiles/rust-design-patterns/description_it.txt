Idiomi Rust e modelli di progettazione per scrivere codice idiomatico. Da utilizzare quando: (1) scrivere nuovo codice Rust, (2) rivedere il codice Rust, (3) risolvere problemi relativi al controllo del prestito, (4) progettare le API Rust. Si attiva su: problemi di proprietà di Rust, errori di durata, domande sulla progettazione dell'API, "come eseguire X in modo idiomatico in Rust".
