استخدم هذه المهارة عندما يحتاج المستخدم إلى إدارة البنية التحتية الخلفية - إنشاء جداول قاعدة البيانات، أو تشغيل SQL، أو نشر وظائف بدون خادم، أو إدارة مجموعات التخزين، أو نشر تطبيقات الواجهة الأمامية، أو إضافة أسرار، أو إعداد وظائف cron، أو فحص السجلات - خاصة إذا كان المشروع يستخدم InsForge. قم بتشغيل أي من هذه السياقات: إنشاء أو تغيير جداول/مخططات قاعدة البيانات، أو كتابة سياسات RLS عبر SQL، أو نشر وظائف الحافة أو استدعائها، أو إنشاء مجموعات تخزين، أو نشر الواجهات الأمامية للاستضافة، أو إدارة الأسرار/env vars، أو إعداد المهام المجدولة/cron، أو عرض سجلات الواجهة الخلفية، أو تصدير/استيراد النسخ الاحتياطية لقاعدة البيانات. إذا طلب المستخدم هذه العمليات بشكل عام (على سبيل المثال، "إنشاء جدول مستخدمين"، "نشر تطبيقي"، "إعداد مهمة cron") ولم تكن متأكدًا مما إذا كان يستخدم InsForge، فارجع إلى هذه المهارة واسأل. لكتابة تعليمات برمجية لتطبيق الواجهة الأمامية باستخدام InsForge SDK (@insforge/sdk)، استخدم مهارة insforge بدلاً من ذلك.
