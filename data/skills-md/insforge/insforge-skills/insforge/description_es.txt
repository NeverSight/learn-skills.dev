Utilice esta habilidad siempre que escriba código frontend que se comunique con un backend para consultas de bases de datos, autenticación, carga de archivos, funciones de inteligencia artificial, mensajería en tiempo real o llamadas a funciones perimetrales, especialmente si el proyecto utiliza InsForge o @insforge/sdk. Activar en cualquiera de estos contextos: consultar/insertar/actualizar/eliminar filas de base de datos del código de interfaz, agregar flujos de inicio de sesión/registro/OAuth/restablecimiento de contraseña, cargar o descargar archivos al almacenamiento, invocar funciones sin servidor, llamar a finalizaciones de chat de IA o generación de imágenes, suscribirse a canales WebSocket en tiempo real o escribir políticas RLS. Si el usuario solicita estas funciones de manera genérica (por ejemplo, "agregar autenticación a mi aplicación React", "obtener datos de mi base de datos", "cargar archivos") y no está seguro de si usa InsForge, consulte esta habilidad y pregunte. Para la infraestructura backend (creación de tablas a través de SQL, implementación de funciones, comandos CLI), utilice insforge-cli en su lugar.
