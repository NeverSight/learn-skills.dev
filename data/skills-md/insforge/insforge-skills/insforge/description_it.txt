Utilizza questa competenza ogni volta che scrivi codice frontend che comunica con un backend per query di database, autenticazione, caricamenti di file, funzionalità di intelligenza artificiale, messaggistica in tempo reale o chiamate di funzioni edge, soprattutto se il progetto utilizza InsForge o @insforge/sdk. Attivazione in uno qualsiasi di questi contesti: query/inserimento/aggiornamento/eliminazione di righe di database dal codice frontend, aggiunta di flussi di accesso/iscrizione/OAuth/reimpostazione della password, caricamento o download di file nell'archivio, richiamo di funzioni serverless, chiamata di completamenti di chat AI o generazione di immagini, sottoscrizione a canali WebSocket in tempo reale o scrittura di policy RLS. Se l'utente richiede queste funzionalità in modo generico (ad esempio, "aggiungi autorizzazione alla mia app React", "recupera dati dal mio database", "carica file") e non sei sicuro che utilizzi InsForge, consulta questa competenza e chiedi. Per l'infrastruttura backend (creazione di tabelle tramite SQL, distribuzione di funzioni, comandi CLI), utilizzare invece insforge-cli.
