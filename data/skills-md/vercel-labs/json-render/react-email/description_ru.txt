Средство рендеринга электронной почты React для json-render, которое преобразует спецификации JSON в HTML или текстовые электронные письма с помощью @react-email/comComponents и @react-email/render. Используйте при работе с @json-render/react-email, создании транзакционных или маркетинговых электронных писем из JSON, создании каталогов электронной почты, рендеринге спецификаций электронной почты, сгенерированных искусственным интеллектом, или когда пользователь упоминает ответную электронную почту, электронную почту в формате HTML или транзакционную электронную почту.
