@react-email/components と @react-email/render を使用して JSON 仕様を HTML またはプレーンテキスト電子メールに変換する json-render 用の React Email レンダラー。 @json-render/react-email を操作する場合、JSON からトランザクション電子メールまたはマーケティング電子メールを構築する場合、電子メール カタログを作成する場合、AI によって生成された電子メール仕様をレンダリングする場合、またはユーザーが反応電子メール、HTML 電子メール、またはトランザクション電子メールに言及する場合に使用します。
