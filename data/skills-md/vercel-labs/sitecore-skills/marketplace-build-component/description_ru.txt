Создает компоненты пользовательского интерфейса с использованием системы дизайна Blok для приложений Sitecore Marketplace. Используйте, когда пользователь хочет создать пользовательский интерфейс, добавить компоненты, создать макет страницы или работать с компонентами Blok/shadcn в приложении Marketplace.
