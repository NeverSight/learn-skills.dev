Chrome DevTools プロトコル経由でエージェント ブラウザを使用して Electron デスクトップ アプリ (VS Code、Slack、Discord、Figma、Notion、Spotify など) を自動化します。ユーザーが Electron アプリとの対話、デスクトップ アプリの自動化、実行中のアプリへの接続、ネイティブ アプリの制御、または Electron アプリケーションのテストを行う必要がある場合に使用します。トリガーには、「Slack アプリの自動化」、「VS Code の制御」、「Discord アプリとの対話」、「この Electron アプリのテスト」、「デスクトップ アプリへの接続」、またはネイティブ Electron アプリケーションの自動化を必要とするタスクが含まれます。
