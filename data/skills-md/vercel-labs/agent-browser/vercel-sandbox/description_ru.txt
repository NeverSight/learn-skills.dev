Запустите агент-браузер + Chrome внутри микроVM Vercel Sandbox для автоматизации браузера из любого приложения, развернутого Vercel. Используйте, когда пользователю нужна автоматизация браузера в приложении Vercel (Next.js, SvelteKit, Nuxt, Remix, Astro и т. д.), он хочет запускать Chrome без ограничений без ограничений по двоичному размеру, нуждается в постоянных сеансах браузера между командами или хочет иметь временные изолированные среды браузера. Триггеры включают «браузер Vercel Sandbox», «microVM Chrome», «агент-браузер в песочнице», «автоматизация браузера на Vercel» или любую задачу, требующую Chrome в песочнице Vercel.
