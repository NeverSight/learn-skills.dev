Exécutez agent-browser + Chrome dans les microVM Vercel Sandbox pour l'automatisation du navigateur à partir de n'importe quelle application déployée par Vercel. À utiliser lorsque l'utilisateur a besoin d'automatiser le navigateur dans une application Vercel (Next.js, SvelteKit, Nuxt, Remix, Astro, etc.), souhaite exécuter Chrome sans tête sans limite de taille binaire, a besoin de sessions de navigateur persistantes entre les commandes ou souhaite des environnements de navigateur isolés éphémères. Les déclencheurs incluent « Navigateur Vercel Sandbox », « microVM Chrome », « Agent-navigateur dans sandbox », « automatisation du navigateur sur Vercel » ou toute tâche nécessitant Chrome dans un Vercel Sandbox.
