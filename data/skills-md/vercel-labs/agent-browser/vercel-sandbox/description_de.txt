Führen Sie agent-browser + Chrome in Vercel Sandbox-Mikro-VMs aus, um die Browser-Automatisierung von jeder von Vercel bereitgestellten App aus zu ermöglichen. Verwenden Sie diese Option, wenn der Benutzer eine Browserautomatisierung in einer Vercel-App (Next.js, SvelteKit, Nuxt, Remix, Astro usw.) benötigt, Headless Chrome ohne binäre Größenbeschränkungen ausführen möchte, dauerhafte Browsersitzungen über Befehle hinweg benötigt oder kurzlebige isolierte Browserumgebungen wünscht. Zu den Auslösern gehören „Vercel Sandbox-Browser“, „microVM Chrome“, „Agent-Browser in Sandbox“, „Browserautomatisierung auf Vercel“ oder jede Aufgabe, die Chrome in einer Vercel-Sandbox erfordert.
