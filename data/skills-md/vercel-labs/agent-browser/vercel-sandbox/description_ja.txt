Vercel サンドボックス microVM 内でエージェント ブラウザ + Chrome を実行し、Vercel がデプロイされたアプリからブラウザを自動化します。ユーザーが Vercel アプリ (Next.js、SvelteKit、Nuxt、Remix、Astro など) でブラウザーの自動化が必要な場合、バイナリ サイズ制限なしでヘッドレス Chrome を実行したい場合、コマンド間で永続的なブラウザー セッションが必要な場合、または一時的な分離されたブラウザー環境が必要な場合に使用します。トリガーには、「Vercel Sandbox ブラウザ」、「microVM Chrome」、「サンドボックス内のエージェント ブラウザ」、「Vercel 上のブラウザ自動化」、または Vercel サンドボックスで Chrome を必要とするタスクが含まれます。
