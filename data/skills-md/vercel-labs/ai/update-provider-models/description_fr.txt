Ajoutez de nouveaux ID de modèle ou supprimez les ID de modèle obsolètes pour les fournisseurs de SDK AI existants. À utiliser lors de l'ajout d'un modèle à un fournisseur, de la suppression d'un modèle obsolète ou du traitement d'une liste de modifications de modèle à partir d'un problème. Déclencheurs sur « ajouter un modèle », « supprimer un modèle », « nouvel ID de modèle », « modèle obsolète », « mettre à jour les ID de modèle ».
