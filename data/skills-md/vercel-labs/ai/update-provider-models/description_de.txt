Fügen Sie neue Modell-IDs für bestehende AI SDK-Anbieter hinzu oder entfernen Sie veraltete. Verwenden Sie diese Option, wenn Sie einem Anbieter ein Modell hinzufügen, ein veraltetes Modell entfernen oder eine Liste von Modelländerungen aus einem Problem verarbeiten. Löst auf „Modell hinzufügen“, „Modell entfernen“, „neue Modell-ID“, „veraltetes Modell“, „Modell-IDs aktualisieren“ aus.
