Создает минимальный самообслуживаемый файл CLAUDE.md для проектов посредством автоматического обнаружения и собеседования с разработчиком. Охватывает индивидуальность проекта, правила «делать/не делать», ловушки и самообслуживание. Используйте при запуске нового проекта или добавлении поддержки Claude Code в существующий.
