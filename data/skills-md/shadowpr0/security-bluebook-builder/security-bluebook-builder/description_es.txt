Cree o perfeccione una política de seguridad normativa y concisa ("Libro Azul") para aplicaciones confidenciales. Úselo cuando los usuarios necesiten un modelo de amenaza, reglas de clasificación de datos, política de autenticación/sesión, requisitos de registro y auditoría, expectativas de retención/eliminación, respuesta a incidentes o puertas de seguridad para aplicaciones que manejan PII/PHI/datos financieros.
