DÉCLENCHEUR SUR CHAQUE tâche de développement — cette compétence doit s'activer avant de planifier, implémenter, déboguer, refactoriser, réviser le code, répondre à des questions sur le code ou préparer une validation. Lisez d'abord AGENTS.md et .agent/docs, puis continuez. Utilisez cette compétence lorsque : vous créez ou révisez un plan, écrivez ou modifiez du code, corrigez un bogue, explorez une base de code, effectuez une révision de code, refactorisez ou validez des modifications. Si le projet a (ou devrait avoir) un AGENTS.md, invoquez cette compétence, même en mode plan, même en lisant uniquement du code. Maintient la documentation de Context Brain, l'initialise dans les nouveaux projets, applique les trois lois (réciprocité, création, immuabilité) et détecte les lacunes dans la documentation au moment de la validation.
