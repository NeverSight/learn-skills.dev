Comment utiliser la CLI Paw Mail (paw-mail). Utilisez cette compétence lorsque : (1) vous devez exécuter une commande CLI (auth, comptes, e-mails, synchronisation, threads, configuration), (2) quelqu'un vous demande comment utiliser ou appeler la CLI de messagerie, (3) vous devez vérifier les indicateurs ou sous-commandes CLI.
