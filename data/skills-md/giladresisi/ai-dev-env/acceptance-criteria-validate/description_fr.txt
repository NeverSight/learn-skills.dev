À utiliser lorsqu'un agent exécutant un plan de mise en œuvre prétend avoir terminé, pour valider que tous les critères d'acceptation ont été effectivement remplis. Localise les critères d'acceptation dans le fichier de plan, accept_criteria.md, ou dans la demande elle-même, puis étudie la base de code et fait apparaître un verdict de réussite/échec par critère.
