Utilisez la gog CLI (v0.10.0) pour effectuer des opérations Google Docs (créer, lire, écrire, insérer, rechercher et remplacer, exporter). Déclenché par des expressions telles que « Créer un document », « Écrire dans Docs », « Lire le document », « Exporter le document », « Copier le document », « Remplacer le texte », etc.
