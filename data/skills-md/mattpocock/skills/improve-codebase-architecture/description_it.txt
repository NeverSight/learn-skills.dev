Esplora una base di codice per trovare opportunità di miglioramento dell'architettura, concentrandosi sul rendere la base di codice più testabile approfondendo moduli poco profondi. Da utilizzare quando l'utente desidera migliorare l'architettura, trovare opportunità di refactoring, consolidare moduli strettamente collegati o rendere una base di codice più navigabile tramite intelligenza artificiale.
