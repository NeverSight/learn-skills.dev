تصميم وتنفيذ وإعادة بناء مسارات العمل والأنشطة المؤقتة عالية الجودة في Python. يُستخدم عند إنشاء تطبيقات Temporal Python أو تغييرها؛ عند تصميم الأنشطة (deps القابلة للحقن، معلمات/نتيجة Pydantic، منطق الأعمال الخارجي)؛ عند تنفيذ سير العمل باستخدام المدخلات الديناميكية أو الاستمرار كجديد؛ أو عند تكوين الموثوقية (المهلات، أو إعادة المحاولة، أو العجز)، أو الحتمية، أو الاختبار. يغطي الحدود والعقود المكتوبة والتاريخ المحدود والكشف التدريجي عبر الموارد.
