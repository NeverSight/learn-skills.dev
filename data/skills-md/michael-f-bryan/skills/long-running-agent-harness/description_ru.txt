Планирует и структурирует крупномасштабную работу для агентов ИИ на протяжении многих сеансов. Человек и искусственный интеллект работают над созданием проектной документации; запустите субагент инициализатора один раз, чтобы создать список функций, модуль Runbook и невыполненную работу в _working/; затем несколько раз запускайте субагент кодирования, пока все функции не будут пройдены. На контрольных точках (например, в конце группы рабочих подразделений) делайте паузу для проверки человеком, повторно запускайте инициализатор, а затем продолжайте. Приглашения передаются субагентам при создании (без копирования в .cursor/rules). Используйте при планировании работы агента в нескольких сеансах, долгосрочном кодировании проекта или переключении между сеансами кодирования.
