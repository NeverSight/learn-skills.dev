Planifica y estructura el trabajo a gran escala para los agentes de IA en muchas sesiones. Los humanos y la IA iteran para producir un documento de diseño; ejecute un subagente Inicializador una vez para crear una lista de funciones, un runbook y un trabajo pendiente en _working/; luego ejecute repetidamente un subagente de codificación hasta que pasen todas las funciones. En los hitos (por ejemplo, el final de un grupo de unidades de trabajo), haga una pausa para el control humano, vuelva a ejecutar el Inicializador y luego continúe. Las indicaciones se pasan a los subagentes cuando se generan (no se copian en .cursor/rules). Utilícelo al planificar el trabajo del agente con varias sesiones, la codificación a largo plazo a partir de un diseño o el traspaso entre sesiones de codificación.
