Espere a que finalice la ejecución de GitHub Actions con una salida de terminal mínima y un código de salida confiable. Úselo cuando un agente debe esperar a que pase CI (por ejemplo, después de enviar, después de abrir un PR o al verificar una ejecución específica). Prefiera gh run watch con --exit-status y --compact para evitar inundar el contexto con resultados de la encuesta.
