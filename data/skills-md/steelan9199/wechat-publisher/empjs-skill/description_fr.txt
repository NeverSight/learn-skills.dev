Utilisez le langage naturel pour appeler l'outil CLI @empjs/skill afin de gérer les compétences de l'agent AI.  
Utilisez cette compétence lorsque l'utilisateur a besoin de :  
1. Installer/ajouter un package de compétences (installer/ajouter)  
2. Lister les compétences installées (list/ls)  
3. Supprimer/désinstaller une compétence (remove/rm/uninstall)  
4. Afficher les plates-formes d'agents AI prises en charge (agents/liste-agents)  
5. Gérez les compétences avec la commande eskill
