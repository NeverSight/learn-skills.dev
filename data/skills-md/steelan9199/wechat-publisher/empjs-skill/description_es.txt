Utilice lenguaje natural para llamar a la herramienta CLI @empjs/skill para administrar las habilidades del agente AI.
Utilice esta habilidad cuando el usuario necesite:
1. Instalar/agregar un paquete de habilidades (instalar/agregar)
2. Listar las habilidades instaladas (list/ls)
3. Eliminar/desinstalar una habilidad (eliminar/rm/uninstall)
4. Ver plataformas de agentes AI compatibles (agentes/agentes de lista)
5. Utilice el comando eskill para gestionar habilidades.
