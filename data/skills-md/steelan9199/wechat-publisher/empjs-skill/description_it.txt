Utilizza il linguaggio naturale per richiamare lo strumento CLI @empjs/skill per gestire le competenze dell'agente AI.  
Utilizza questa abilità quando l'utente ha bisogno di:  
1. Installa/aggiungi pacchetti di competenze (installa/aggiungi)  
2. Elenca le competenze installate (list/ls)  
3. Rimuovere/disinstallare competenze (rimuovi/rm/disinstalla)  
4. Visualizza le piattaforme AI Agent supportate (agenti/list-agents)  
5. Utilizzare il comando eskill per gestire le competenze
