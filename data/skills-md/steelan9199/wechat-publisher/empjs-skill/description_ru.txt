Используйте естественный язык для вызова инструмента CLI @empjs/skill для управления навыками агента ИИ.  
Используйте этот навык, когда пользователю необходимо:  
1. Установить/добавить пакеты навыков (установить/добавить)  
2. Перечислите установленные навыки (list/ls)  
3. Удаление/деинсталляция навыков (remove/rm/uninstall)  
4. Просмотрите поддерживаемые платформы AI Agent (агенты/списки агентов).  
5. Управляйте навыками с помощью команды eskill.
