Используйте Chrome DevTools MCP для получения и анализа данных веб-страницы. Используйте этот навык, когда пользователю необходимо очистить содержимое веб-страницы, проанализировать структуру веб-страницы, извлечь данные, сделать снимки экрана, отслеживать сетевые запросы или выполнить задачи автоматизации веб-страницы.
