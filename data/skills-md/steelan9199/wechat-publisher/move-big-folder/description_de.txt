Verschieben Sie große Ordner vom Laufwerk C auf ein anderes Laufwerk (D/E/F usw.), behalten Sie den ursprünglichen Pfad über symbolische Links bei und geben Sie Speicherplatz auf dem Laufwerk C frei. Verwenden Sie diesen Skill, wenn der Benutzer „Laufwerk C voll“, „Laufwerk C rot“, „Großen Ordner verschieben“, „Speicherplatz auf Laufwerk C freigeben“ oder „Symlink-Migration“ erwähnt.
