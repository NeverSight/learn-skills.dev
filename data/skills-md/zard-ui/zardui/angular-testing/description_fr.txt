Écrivez des tests unitaires et d'intégration pour les applications Angular v21+ à l'aide de Jest avec @testing-library/angular, en vous concentrant sur les tests centrés sur l'utilisateur, le modèle AAA et les modèles angulaires modernes (composants autonomes, signaux). À utiliser pour tester les composants, les services et les interactions HTTP avec les globaux Jest et les comparateurs DOM de la bibliothèque de tests.
