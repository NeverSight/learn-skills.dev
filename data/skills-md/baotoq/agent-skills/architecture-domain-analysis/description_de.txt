Identifiziert Subdomänen und schlägt begrenzte Kontexte in jeder Codebasis gemäß DDD Strategic Design vor. Verwenden Sie diese Funktion, wenn Sie Domänengrenzen analysieren, geschäftliche Subdomänen identifizieren, den Domänenzusammenhalt bewerten, begrenzte Kontexte abbilden oder wenn der Benutzer Fragen zum strategischen DDD-Design, zur Domänenanalyse oder zur Klassifizierung von Subdomänen stellt.
