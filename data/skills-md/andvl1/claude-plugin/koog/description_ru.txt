Платформа JetBrains Koog AI Agent (Kotlin) — используется для создания агентов AI с вызовом инструментов, интеграции LLM через OpenRouter/OpenAI/Anthropic/Google/DeepSeek и рабочих процессов на базе AI. Используйте при реализации агентов искусственного интеллекта, вызовов LLM, шаблонов вызова инструментов или интеграции поставщиков LLM в проекты Kotlin.
