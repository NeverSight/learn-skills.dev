Обрабатывайте неструктурированные внешние входные данные (расшифровки встреч, журналы разговоров, вставленные документы) в структурированные объекты базовой памяти. Извлекает объекты, ищет существующие совпадения, предлагает новые объекты с одобрением, создает заметки с наблюдениями и связями, а также фиксирует действия.
