Управление жизненным циклом схемы для базовой памяти: обнаружение несхемированных заметок, вывод схем, создание и редактирование определений схем, проверка заметок и обнаружение отклонений. Используйте при работе со структурированными типами заметок («Задача», «Человек», «Собрание» и т. д.) для обеспечения единообразия в графике знаний.
