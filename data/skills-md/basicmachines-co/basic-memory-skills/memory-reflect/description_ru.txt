Размышления о памяти во время сна: просматривайте недавние разговоры и ежедневные заметки, извлекайте идеи и закрепляйте их в долговременной памяти. Используйте при запуске cron, Heartbeat или явного запроса, чтобы отразить недавнюю активность. Запускается в фоновом режиме для улучшения качества памяти с течением времени.
