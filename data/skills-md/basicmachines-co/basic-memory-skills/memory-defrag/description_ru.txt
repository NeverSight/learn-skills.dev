Дефрагментируйте и реорганизуйте файлы памяти агента: разделяйте раздутые файлы, объединяйте дубликаты, удаляйте устаревшую информацию и реструктурируйте иерархию памяти. Используйте, когда файлы памяти стали громоздкими, содержат избыточность или нуждаются в реорганизации. Запускать периодически (еженедельно) или по требованию.
