Desfragmente y reorganice los archivos de memoria del agente: divida archivos inflados, combine duplicados, elimine información obsoleta y reestructure la jerarquía de la memoria. Úselo cuando los archivos de memoria se hayan vuelto difíciles de manejar, contengan redundancias o necesiten reorganización. Ejecutar periódicamente (semanalmente) o bajo demanda.
