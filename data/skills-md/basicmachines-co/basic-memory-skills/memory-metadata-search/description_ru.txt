Поиск структурированных метаданных для базовой памяти: запросы к заметкам по настраиваемым полям заголовка с использованием фильтров равенства, диапазона, массива и вложенных фильтров. Используйте при поиске заметок по статусу, приоритету, достоверности или любому настраиваемому полю YAML, а не по произвольному текстовому содержимому.
