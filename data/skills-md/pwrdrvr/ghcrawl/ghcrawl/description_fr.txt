Utilisez la CLI ghcrawl locale pour inspecter les clusters en double et les résumés de problèmes/PR à partir de l'ensemble de données ghcrawl existant, et actualisez un dépôt uniquement lorsque l'utilisateur le demande explicitement. À utiliser lorsqu'un utilisateur souhaite trier les problèmes ou les PR associés, inspecter les clusters sémantiques ou exécuter le pipeline d'actualisation par étapes de ghcrawl.
