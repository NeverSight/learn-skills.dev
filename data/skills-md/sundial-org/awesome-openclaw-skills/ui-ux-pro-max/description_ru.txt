Аналитика UI/UX-дизайна и руководство по реализации для создания безупречных интерфейсов. Используйте, когда пользователь запрашивает дизайн пользовательского интерфейса, потоки пользовательского интерфейса, информационную архитектуру, направление визуального стиля, системы дизайна/токены, спецификации компонентов, копирование/микрокопирование, доступность или для создания/критики/уточнения пользовательского интерфейса внешнего интерфейса (HTML/CSS/JS, React, Next.js, Vue, Svelte, Tailwind). Включает рабочие процессы для (1) создания новых макетов и стилей пользовательского интерфейса, (2) улучшения существующего пользовательского интерфейса/UX, (3) создания токенов системы дизайна и рекомендаций по компонентам и (4) превращения рекомендаций UX в конкретные изменения кода.
