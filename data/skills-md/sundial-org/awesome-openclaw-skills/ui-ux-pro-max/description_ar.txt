ذكاء تصميم واجهة المستخدم/تجربة المستخدم وإرشادات التنفيذ لبناء واجهات مصقولة. يُستخدم عندما يطلب المستخدم تصميم واجهة المستخدم، أو تدفقات تجربة المستخدم، أو هندسة المعلومات، أو اتجاه النمط المرئي، أو أنظمة التصميم/الرموز المميزة، أو مواصفات المكونات، أو النسخ/النسخ المصغر، أو إمكانية الوصول، أو لإنشاء/نقد/تحسين واجهة المستخدم الأمامية (HTML/CSS/JS، React، Next.js، Vue، Svelte، Tailwind). يتضمن سير العمل من أجل (1) إنشاء تخطيطات وتصميمات جديدة لواجهة المستخدم، (2) تحسين واجهة المستخدم/تجربة المستخدم الحالية، (3) إنتاج الرموز المميزة لنظام التصميم وإرشادات المكونات، و(4) تحويل توصيات تجربة المستخدم إلى تغييرات ملموسة في التعليمات البرمجية.
