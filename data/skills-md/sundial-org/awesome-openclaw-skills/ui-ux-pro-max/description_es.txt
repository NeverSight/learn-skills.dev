Inteligencia de diseño UI/UX y guía de implementación para crear interfaces pulidas. Úselo cuando el usuario solicite diseño de UI, flujos de UX, arquitectura de información, dirección de estilo visual, sistemas/tokens de diseño, especificaciones de componentes, copia/microcopia, accesibilidad o para generar/criticar/refinar la interfaz de usuario (HTML/CSS/JS, React, Next.js, Vue, Svelte, Tailwind). Incluye flujos de trabajo para (1) generar nuevos diseños y estilos de UI, (2) mejorar UI/UX existente, (3) producir tokens de sistema de diseño y pautas de componentes, y (4) convertir recomendaciones de UX en cambios de código concretos.
