洗練されたインターフェイスを構築するための UI/UX 設計インテリジェンスと実装ガイダンス。ユーザーが UI デザイン、UX フロー、情報アーキテクチャ、ビジュアル スタイルの方向性、デザイン システム/トークン、コンポーネント仕様、コピー/マイクロコピー、アクセシビリティを要求する場合、またはフロントエンド UI (HTML/CSS/JS、React、Next.js、Vue、Svelte、Tailwind) を生成/批評/改良する場合に使用します。 (1) 新しい UI レイアウトとスタイルの生成、(2) 既存の UI/UX の改善、(3) デザイン システム トークンとコンポーネント ガイドラインの作成、(4) UX の推奨事項を具体的なコード変更に変換するためのワークフローが含まれます。
