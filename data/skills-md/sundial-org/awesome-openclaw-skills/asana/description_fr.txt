Intégrez Asana à Clawdbot via l'API REST d'Asana. À utiliser lorsque vous devez répertorier/rechercher/créer/mettre à jour des tâches/projets/espaces de travail Asana, ou configurer Asana OAuth (octroi de code d’autorisation) pour une intégration personnelle uniquement locale (OOB/collage manuel de code).
