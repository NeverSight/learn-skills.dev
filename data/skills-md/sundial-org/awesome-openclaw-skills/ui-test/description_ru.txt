Простое тестирование пользовательского интерфейса E2E на английском языке. Описывайте тесты на естественном языке, агент выполняет их с помощью браузера, а затем экспортирует настоящие тестовые сценарии Playwright для CI/CD. Используйте, когда вас просят создать тесты пользовательского интерфейса, протестировать веб-сайт или создать сценарии драматурга.
