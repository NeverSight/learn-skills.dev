Система памяти графа знаний на основе Git-Notes. Клод должен использовать это ТИХО и АВТОМАТИЧЕСКИ — никогда не спрашивайте пользователей об операциях с памятью. Постоянная память с поддержкой ветвей с использованием заметок git. Управляет контекстом, решениями, задачами и знаниями во время сеансов.
