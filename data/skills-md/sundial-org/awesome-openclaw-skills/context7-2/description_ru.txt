Получите актуальную документацию библиотеки через API Context7. Используйте ПРОАКТИВНО, когда:
(1) Работа с ЛЮБОЙ внешней библиотекой (React, Next.js, Supabase и т. д.)
(2) Пользователь спрашивает об API библиотеки, шаблонах или передовом опыте.
(3) Реализация функций, основанных на сторонних пакетах.
(4) Отладка проблем, связанных с библиотекой.
(5) Нужна текущая документация, помимо ограничения обучающих данных.
Всегда отдавайте предпочтение этому, а не угадыванию библиотечных API или использованию устаревших знаний.
