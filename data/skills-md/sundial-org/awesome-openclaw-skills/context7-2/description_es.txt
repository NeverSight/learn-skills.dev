Obtenga documentación de biblioteca actualizada a través de la API Context7. Úselo PROACTIVAMENTE cuando:
(1) Trabajar con CUALQUIER biblioteca externa (React, Next.js, Supabase, etc.)
(2) El usuario pregunta sobre las API, los patrones o las mejores prácticas de la biblioteca
(3) Implementar funciones que dependen de paquetes de terceros
(4) Depuración de problemas específicos de la biblioteca
(5) Necesita documentación actualizada más allá del límite de datos de entrenamiento
Prefiera siempre esto a adivinar las API de la biblioteca o utilizar conocimientos obsoletos.
