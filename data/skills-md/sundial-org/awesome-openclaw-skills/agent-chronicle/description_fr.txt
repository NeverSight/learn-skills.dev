Journalisation en perspective de l'IA : documentez les expériences, les émotions et les apprentissages quotidiens du point de vue de l'agent. À utiliser lorsqu'on vous pose des questions sur le journal, les entrées de journal, l'auto-réflexion ou la documentation des expériences d'IA. Crée des entrées quotidiennes structurées capturant les projets, les victoires, les frustrations, les apprentissages et les états émotionnels.
