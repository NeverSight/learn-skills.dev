Используйте браузер. Используйте облачный API для запуска облачных браузеров для Clawdbot и выполнения автономных задач браузера. Основное использование — создание сеансов браузера с профилями (постоянные логины/файлы cookie), которыми Clawdbot может управлять. Вторичное использование — запуск субагентов задач для быстрой автономной автоматизации браузера. Документы на docs.browser-use.com и docs.cloud.browser-use.com.
