Diese Fähigkeit sollte verwendet werden, wenn Benutzer im Internet nach Informationen suchen, aktuelle Inhalte finden, Nachrichtenartikel nachschlagen, nach Bildern suchen oder Videos finden müssen. Es verwendet die Such-API von DuckDuckGo, um Ergebnisse in einer sauberen, formatierten Ausgabe (Text, Markdown oder JSON) zurückzugeben. Zur Recherche, zur Überprüfung von Fakten, zur Suche nach aktuellen Informationen oder zum Sammeln von Webressourcen.
