Questa abilità dovrebbe essere utilizzata quando gli utenti hanno bisogno di cercare informazioni sul Web, trovare contenuti attuali, cercare articoli di notizie, cercare immagini o trovare video. Utilizza l'API di ricerca di DuckDuckGo per restituire risultati in un output pulito e formattato (testo, markdown o JSON). Utilizzare per ricerche, verifiche dei fatti, ricerca di informazioni recenti o raccolta di risorse Web.
