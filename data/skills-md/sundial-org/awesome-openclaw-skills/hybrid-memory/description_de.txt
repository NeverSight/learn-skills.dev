Hybride Speicherstrategie, die den integrierten Vektorspeicher von OpenClaw mit dem zeitlichen Wissensgraphen von Graphiti kombiniert. Verwenden Sie diese Funktion, wenn Sie vergangene Kontexte abrufen, zeitliche Fragen beantworten („Wann ist X passiert?“) oder Speicherdateien durchsuchen müssen. Bietet einen Entscheidungsrahmen für die Verwendung von „memory_search“ im Vergleich zu „Graphit“.
