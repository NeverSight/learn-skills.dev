Strategia di memoria ibrida che combina la memoria vettoriale integrata di OpenClaw con il grafico della conoscenza temporale di Graphiti. Da utilizzare quando è necessario richiamare il contesto passato, rispondere a domande temporali ("quando è successo X?") o cercare file di memoria. Fornisce un quadro decisionale su quando utilizzare memory_search rispetto a Graphiti.
