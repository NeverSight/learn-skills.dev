Estrategia de memoria híbrida que combina la memoria vectorial incorporada de OpenClaw con el gráfico de conocimiento temporal Graphiti. Úselo cuando necesite recordar contexto pasado, responder preguntas temporales ("¿cuándo sucedió X?") o buscar archivos de memoria. Proporciona un marco de decisión sobre cuándo utilizar Memory_search frente a Graphiti.
