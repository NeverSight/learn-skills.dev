Стратегия гибридной памяти, сочетающая встроенную векторную память OpenClaw с временным графом знаний Graphiti. Используйте, когда вам нужно вспомнить прошлый контекст, ответить на временные вопросы («когда произошло X?») или выполнить поиск в файлах памяти. Предоставляет основу для принятия решений, когда использовать Memory_search или Graphiti.
