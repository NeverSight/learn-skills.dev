Stratégie de mémoire hybride combinant la mémoire vectorielle intégrée d'OpenClaw avec le graphe de connaissances temporelles Graphiti. À utiliser lorsque vous avez besoin de rappeler le contexte passé, de répondre à des questions temporelles (« quand X s'est-il produit ? ») ou de rechercher des fichiers mémoire. Fournit un cadre de décision pour savoir quand utiliser memory_search vs Graphiti.
