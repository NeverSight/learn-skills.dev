Guía las actualizaciones de proyectos PHP utilizando los comandos de Composer. Úselo para ayudar a los usuarios a actualizar paquetes PHP, comprender conflictos de dependencia, interpretar la salida de "composer obsoleto", usar "composerWhy-not" para diagnosticar restricciones de versión, usar "composerWhy" para rastrear dependencias, usar "composerbump" para reforzar las restricciones de versión después de la actualización, planificar rutas de actualización seguras o resolver conflictos de versiones de paquetes en Composer.json.
