يرشد ترقيات مشروع PHP باستخدام أوامر الملحن. يُستخدم عند مساعدة المستخدمين على ترقية حزم PHP، أو فهم تعارضات التبعيات، أو تفسير مخرجات "المؤلف القديم"، أو استخدام "لماذا لا يوجد الملحن" لتشخيص قيود الإصدار، أو استخدام "لماذا الملحن" لتتبع التبعيات، أو استخدام "تضخيم الملحن" لتشديد قيود الإصدار بعد الترقية، أو التخطيط لمسارات ترقية آمنة، أو حل تعارضات إصدار الحزمة في ملف Composer.json.
