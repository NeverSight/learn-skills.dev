Управляет обновлением PHP-проектов с помощью команд Composer. Используйте, когда помогаете пользователям обновлять пакеты PHP, понимать конфликты зависимостей, интерпретировать вывод `composer outdated`, использовать `composerWhy-No` для диагностики ограничений версии, использовать `composerWhy` для отслеживания зависимостей, использовать ComposerBump` для ужесточения ограничений версии после обновления, планировать безопасные пути обновления или разрешать конфликты версий пакетов в композиторе.json.
