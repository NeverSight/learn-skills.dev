قم ببناء قواعد Rector PHP التي تحول كود PHP عبر AST. يُستخدم عندما يُطلب منك إنشاء أو تعديل أو شرح قواعد Rector لتحويلات كود PHP. تستخدم قواعد Rector تحليل النوع PHP-Parser AST وPHPStan. يتم تشغيل طلبات مثل "كتابة قاعدة Rector لـ..." أو "إنشاء قاعدة Rector لـ..." أو "إضافة قاعدة Rector لـ..." أو عند العمل في مشروع يستند إلى rector-src أو طلب تنفيذ منطق تحويل التعليمات البرمجية.
