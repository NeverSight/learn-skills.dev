AST 経由で PHP コードを変換する Rector PHP ルールを構築します。 PHP コード変換の Rector ルールを作成、変更、または説明するように求められた場合に使用します。 Rector ルールは、PHP-Parser AST および PHPStan の型分析を使用します。 「... に Rector ルールを書き込む」、「... の Rector ルールを作成する」、「... の Rector ルールを追加する」などのリクエストでトリガーされます。または、rector-src またはlector ベースのプロジェクトで作業しており、コード変換ロジックの実装を求められた場合にトリガーされます。
