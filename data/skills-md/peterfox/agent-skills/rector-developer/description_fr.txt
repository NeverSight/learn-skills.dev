Créez des règles PHP Rector qui transforment le code PHP via AST. À utiliser lorsqu'on vous demande de créer, modifier ou expliquer les règles de Rector pour les transformations de code PHP. Les règles du recteur utilisent l'analyse de type PHP-Parser AST et PHPStan. Se déclenche sur des requêtes telles que "écrire une règle Rector pour...", "créer une règle Rector qui...", "ajouter une règle Rector pour...", ou lorsque vous travaillez dans un projet rector-src ou basé sur rector et qu'il est demandé d'implémenter une logique de transformation de code.
