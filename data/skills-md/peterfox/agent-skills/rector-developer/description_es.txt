Cree reglas PHP de Rector que transformen el código PHP a través de AST. Úselo cuando se le solicite crear, modificar o explicar las reglas de Rector para transformaciones de código PHP. Las reglas de Rector utilizan el análisis de tipo PHP-Parser AST y PHPStan. Se activa en solicitudes como "escribir una regla de Rector para...", "crear una regla de Rector que...", "agregar una regla de Rector para...", o cuando se trabaja en un proyecto rector-src o basado en rector y se le solicita implementar lógica de transformación de código.
