Erstellen Sie Rector-PHP-Regeln, die PHP-Code über AST umwandeln. Verwenden Sie diese Option, wenn Sie aufgefordert werden, Rector-Regeln für PHP-Codetransformationen zu erstellen, zu ändern oder zu erläutern. Rector-Regeln verwenden die Typanalyse PHP-Parser AST und PHPStan. Wird bei Anfragen wie „Schreiben Sie eine Rector-Regel für …“, „Erstellen Sie eine Rector-Regel, die ...“ oder „Fügen Sie eine Rector-Regel für …“ aus oder wenn Sie in einem Rector-SRC- oder Rector-basierten Projekt arbeiten und aufgefordert werden, Codetransformationslogik zu implementieren.
