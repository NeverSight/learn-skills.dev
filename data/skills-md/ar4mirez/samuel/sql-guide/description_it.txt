Guardrail, modelli e best practice del linguaggio SQL per lo sviluppo assistito dall'intelligenza artificiale.
Da utilizzare quando si lavora con file SQL (.sql), migrazioni di database o quando l'utente menziona SQL/database.
Fornisce modelli di ottimizzazione delle query, linee guida per l'indicizzazione, gestione delle transazioni,
e standard di migrazione specifici per gli standard di codifica di questo progetto.
