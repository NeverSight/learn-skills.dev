Ограждения, шаблоны и лучшие практики языка SQL для разработки с помощью ИИ.
Используйте при работе с файлами SQL (.sql), миграции баз данных или когда пользователь упоминает SQL/базы данных.
Предоставляет шаблоны оптимизации запросов, рекомендации по индексированию, управление транзакциями,
и стандарты миграции, специфичные для стандартов кодирования этого проекта.
