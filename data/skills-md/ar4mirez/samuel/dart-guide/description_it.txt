Guardrail, modelli e best practice del linguaggio Dart per lo sviluppo assistito dall'intelligenza artificiale.
Da utilizzare quando si lavora con file Dart (.dart), pubspec.yaml o quando l'utente menziona Dart/Flutter.
Fornisce modelli di sicurezza nulli, linee guida di asincrona/attesa, convenzioni di integrazione Flutter,
e standard di test specifici per gli standard di codifica di questo progetto.
