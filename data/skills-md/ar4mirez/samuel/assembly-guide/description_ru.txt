Ограждения, шаблоны и лучшие практики языка ассемблера для разработки с помощью ИИ.
Используйте при работе с файлами сборки (.asm, .s, .S) или когда пользователь упоминает Assembly/x86/ARM.
Содержит рекомендации по соглашению о вызовах, шаблоны использования регистров и методы отладки.
специально для стандартов кодирования этого проекта.
