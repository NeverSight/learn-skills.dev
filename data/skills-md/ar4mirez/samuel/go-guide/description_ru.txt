Языковые барьеры Go, шаблоны и лучшие практики для разработки с помощью ИИ.
Используйте при работе с файлами Go (.go), go.mod или когда пользователь упоминает Go/Golang.
Предоставляет соглашения о структуре проекта, шаблоны обработки ошибок, стандарты тестирования,
и рекомендации по параллелизму, специфичные для стандартов кодирования этого проекта.
