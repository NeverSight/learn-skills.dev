Ограждения, шаблоны и лучшие практики сценариев Shell/Bash для разработки с помощью ИИ.
Используйте при работе со сценариями оболочки (.sh, .bash), Makefiles или когда пользователь упоминает Bash/Shell.
Содержит рекомендации по обеспечению соответствия POSIX, шаблоны обработки ошибок и правила ShellCheck.
специально для стандартов кодирования этого проекта.
