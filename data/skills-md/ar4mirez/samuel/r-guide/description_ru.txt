Ограждения, шаблоны и лучшие практики языка R для разработки с помощью ИИ.
Используйте при работе с файлами R (.R, .Rmd, .qmd), DESCRIPTION или когда пользователь упоминает R/RStudio.
Содержит соглашения tidyverse, шаблоны векторизации, рекомендации по R Markdown,
и стандарты тестирования, соответствующие стандартам кодирования этого проекта.
