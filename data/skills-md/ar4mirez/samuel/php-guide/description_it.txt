Guardrail, modelli e best practice del linguaggio PHP per lo sviluppo assistito dall'intelligenza artificiale.
Da utilizzare quando si lavora con file PHP (.php), compositore.json o quando l'utente menziona PHP.
Fornisce modelli di dichiarazione del tipo, convenzioni del compositore, standard PSR,
e linee guida di test specifiche per gli standard di codifica di questo progetto.
