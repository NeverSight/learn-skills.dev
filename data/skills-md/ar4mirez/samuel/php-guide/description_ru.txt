Ограждения языка PHP, шаблоны и лучшие практики для разработки с помощью ИИ.
Используйте при работе с файлами PHP (.php), композитором.json или когда пользователь упоминает PHP.
Предоставляет шаблоны объявления типов, соглашения Composer, стандарты PSR,
и рекомендации по тестированию, соответствующие стандартам кодирования этого проекта.
