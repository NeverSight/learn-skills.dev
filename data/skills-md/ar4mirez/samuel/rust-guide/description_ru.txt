Ограждения, шаблоны и лучшие практики Rust для разработки с помощью ИИ.
Используйте при работе с файлами Rust (.rs), Cargo.toml или когда пользователь упоминает Rust.
Обеспечивает соглашения о собственности, обработку ошибок с помощью Result/Option, стандарты тестирования,
и рекомендации по небезопасному коду.
