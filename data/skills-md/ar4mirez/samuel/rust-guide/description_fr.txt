Garde-corps, modèles et bonnes pratiques contre la rouille pour le développement assisté par l'IA.
À utiliser lorsque vous travaillez avec des fichiers Rust (.rs), Cargo.toml ou lorsque l'utilisateur mentionne Rust.
Fournit les conventions de propriété, la gestion des erreurs avec Résultat/Option, les normes de test,
et les directives de code dangereux.
