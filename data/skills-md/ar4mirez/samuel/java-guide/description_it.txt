Guardrail, modelli e best practice del linguaggio Java per lo sviluppo assistito dall'intelligenza artificiale.
Da utilizzare quando si lavora con file Java (.java), pom.xml, build.gradle o quando l'utente menziona Java.
Fornisce modelli Java moderni (record, classi sigillate, flussi), standard di test,
e convenzioni primaverili specifiche per gli standard di codifica di questo progetto.
