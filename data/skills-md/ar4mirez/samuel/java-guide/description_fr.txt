Garde-fous, modèles et bonnes pratiques du langage Java pour le développement assisté par l'IA.
À utiliser lorsque vous travaillez avec des fichiers Java (.java), pom.xml, build.gradle ou lorsque l'utilisateur mentionne Java.
Fournit des modèles Java modernes (enregistrements, classes scellées, flux), des normes de test,
et les conventions Spring spécifiques aux normes de codage de ce projet.
