Ограждения, шаблоны и лучшие практики языка Java для разработки с помощью ИИ.
Используйте при работе с файлами Java (.java), pom.xml, build.gradle или когда пользователь упоминает Java.
Предоставляет современные шаблоны Java (записи, запечатанные классы, потоки), стандарты тестирования,
и соглашения Spring, специфичные для стандартов кодирования этого проекта.
