Ограждения, шаблоны и лучшие практики Gin Framework для разработки с помощью ИИ.
Используйте при работе с проектами Gin или когда пользователь упоминает структуру Gin.
Предоставляет шаблоны промежуточного программного обеспечения, маршрутизацию, проверку, обработку ошибок и рекомендации по REST API.
