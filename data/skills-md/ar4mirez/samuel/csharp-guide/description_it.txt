Guardrail, modelli e best practice del linguaggio C# per lo sviluppo assistito dall'intelligenza artificiale.
Da utilizzare quando si lavora con file C# (.cs, .csx), .csproj, .sln o quando l'utente menziona C#/.NET.
Fornisce tipi di riferimento nullable, modelli asincroni/await, linee guida LINQ,
e standard di test specifici per gli standard di codifica di questo progetto.
