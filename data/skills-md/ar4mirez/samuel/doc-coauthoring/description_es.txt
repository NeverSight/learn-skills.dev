Guíe a los usuarios a través de un flujo de trabajo estructurado para la documentación de coautoría. Usar cuando
El usuario desea escribir documentación, propuestas, especificaciones técnicas, documentos de decisión o
contenido estructurado similar. Este flujo de trabajo ayuda a los usuarios a transferir contexto,
refine el contenido mediante iteración y verifique que el documento funcione para los lectores. Activar cuando
El usuario menciona escribir documentos, crear propuestas, redactar especificaciones o tareas similares.
