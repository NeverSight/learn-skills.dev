Führen Sie Benutzer durch einen strukturierten Workflow für die gemeinsame Erstellung von Dokumentationen. Verwenden Sie wann
Der Benutzer möchte Dokumentationen, Vorschläge, technische Spezifikationen, Entscheidungsdokumente usw. schreiben
ähnlich strukturierte Inhalte. Dieser Workflow hilft Benutzern, Kontext effizient zu übertragen,
Verfeinern Sie den Inhalt durch Iteration und überprüfen Sie, ob das Dokument für die Leser funktioniert. Auslösen, wenn
Der Benutzer erwähnt das Schreiben von Dokumenten, das Erstellen von Vorschlägen, das Entwerfen von Spezifikationen oder ähnliche Aufgaben.
