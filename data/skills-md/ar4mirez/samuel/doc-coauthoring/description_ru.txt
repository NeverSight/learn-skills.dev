Направляйте пользователей через структурированный рабочий процесс для совместного редактирования документации. Используйте, когда
пользователь хочет написать документацию, предложения, технические спецификации, документы для принятия решений или
схожий структурированный контент. Этот рабочий процесс помогает пользователям эффективно передавать контекст,
уточняйте контент посредством итерации и проверяйте, работает ли документ для читателей. Запустить, когда
пользователь упоминает написание документации, создание предложений, составление спецификаций или подобные задачи.
