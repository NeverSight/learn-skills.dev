Guardrail, modelli e best practice del linguaggio Ruby per lo sviluppo assistito dall'intelligenza artificiale.
Da utilizzare quando si lavora con file Ruby (.rb), Gemfile, Rakefile o quando l'utente menziona Ruby.
Fornisce modelli di blocco/proc, linee guida per la metaprogrammazione, convenzioni bundler,
e standard di test specifici per gli standard di codifica di questo progetto.
