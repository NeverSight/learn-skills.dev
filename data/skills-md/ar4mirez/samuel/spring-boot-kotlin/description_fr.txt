Spring Boot avec les garde-fous, les modèles et les meilleures pratiques du framework Kotlin.
À utiliser lorsque vous travaillez avec des projets Spring Boot Kotlin ou lorsque l'utilisateur mentionne Spring Boot avec Kotlin.
Fournit des modèles de coroutine, WebFlux, JPA, la sécurité et des directives idiomatiques Kotlin-Spring.
