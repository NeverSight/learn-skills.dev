Ограждения HTML и CSS, шаблоны и лучшие практики для разработки с помощью ИИ.
Используйте при работе с файлами HTML/CSS (.html, .css, .scss, .less) или когда пользователь упоминает HTML/CSS.
Предоставляет семантические рекомендации HTML, стандарты доступности, современные шаблоны CSS,
и соглашения об адаптивном дизайне, специфичные для стандартов кодирования этого проекта.
