Garde-corps, modèles et bonnes pratiques HTML et CSS pour le développement assisté par l'IA.
À utiliser lorsque vous travaillez avec des fichiers HTML/CSS (.html, .css, .scss, .less) ou lorsque l'utilisateur mentionne HTML/CSS.
Fournit des directives HTML sémantiques, des normes d'accessibilité, des modèles CSS modernes,
et les conventions de conception réactive spécifiques aux normes de codage de ce projet.
