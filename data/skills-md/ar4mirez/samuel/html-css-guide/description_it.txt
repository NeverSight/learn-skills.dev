Guardrail, modelli e best practice HTML e CSS per lo sviluppo assistito dall'intelligenza artificiale.
Da utilizzare quando si lavora con file HTML/CSS (.html, .css, .scss, .less) o quando l'utente menziona HTML/CSS.
Fornisce linee guida semantiche HTML, standard di accessibilità, modelli CSS moderni,
e convenzioni di progettazione reattiva specifiche per gli standard di codifica di questo progetto.
