Generieren Sie eine Referenzvorlage mit vollständigen Eigenschaftsbeschreibungen für eine einzelne Terraform AWS-Ressource. Wenn der Benutzer einen Ressourcennamen angibt (z. B. aws_s3_bucket, aws_lambda_function), wird eine Vorlage ausgegeben, die die genaue Attributliste basierend auf dem Provider-Schema und Erklärungen basierend auf der offiziellen AWS-Dokumentation enthält. Wird für Anfragen wie „Erstelle eine Vorlage für aws_xxx“ oder „Zeige mir alle Eigenschaften von aws_xxx“ verwendet.
