قم بإنشاء قالب مرجعي يحتوي على أوصاف كاملة للخصائص لمورد Terraform AWS واحد. عندما يحدد المستخدم اسم المورد (على سبيل المثال، aws_s3_bucket، aws_lambda_function)، قم بإخراج قالب يحتوي على قائمة السمات الدقيقة بناءً على مخطط الموفر والتفسيرات المستندة إلى وثائق AWS الرسمية. يُستخدم لطلبات مثل "إنشاء قالب لـ aws_xxx" أو "إظهار جميع خصائص aws_xxx."
