Générez un modèle de référence avec des descriptions complètes des propriétés pour une seule ressource Terraform AWS. Lorsque l'utilisateur spécifie un nom de ressource (par exemple, aws_s3_bucket, aws_lambda_function), génère un modèle contenant la liste d'attributs exacte basée sur le schéma du fournisseur et des explications basées sur la documentation officielle d'AWS. Utilisé pour les requêtes telles que « créer un modèle pour aws_xxx » ou « me montrer toutes les propriétés de aws_xxx ».
