Genera un modello di riferimento con tutte le descrizioni delle proprietà per una singola risorsa AWS Terraform. Quando un utente specifica un nome di risorsa (ad esempio aws_s3_bucket, aws_lambda_function), restituisce un modello con un elenco accurato di attributi basato sullo schema del provider e spiegazioni basate sulla documentazione ufficiale di AWS. Utilizzato per richieste quali "Crea un modello per aws_xxx" e "Dimmi tutte le proprietà di aws_xxx".
