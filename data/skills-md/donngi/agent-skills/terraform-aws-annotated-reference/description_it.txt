Genera un modello di riferimento con le descrizioni complete delle proprietà per una singola risorsa AWS Terraform. Quando l'utente specifica un nome di risorsa (ad esempio, aws_s3_bucket, aws_lambda_function), genera un modello contenente l'elenco esatto degli attributi in base allo schema del provider e spiegazioni basate sulla documentazione ufficiale di AWS. Utilizzato per richieste come "crea un modello per aws_xxx" o "mostrami tutte le proprietà di aws_xxx".
