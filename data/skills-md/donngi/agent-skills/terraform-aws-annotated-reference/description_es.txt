Genere una plantilla de referencia con descripciones completas de las propiedades para un único recurso de Terraform AWS. Cuando el usuario especifica un nombre de recurso (por ejemplo, aws_s3_bucket, aws_lambda_function), genera una plantilla que contiene la lista de atributos exacta según el esquema del proveedor y explicaciones basadas en la documentación oficial de AWS. Se utiliza para solicitudes como "crear una plantilla para aws_xxx" o "muéstrame todas las propiedades de aws_xxx".
