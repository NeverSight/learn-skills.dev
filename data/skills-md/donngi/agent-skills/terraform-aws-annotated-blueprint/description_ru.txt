Создайте шаблон Terraform AWS с подробными пояснениями для каждого свойства. Когда пользователь вводит обзор инфраструктуры, которую он хочет построить, с помощью команды «/terraform-annotated-aws-blueprint», создается шаблон, включающий точный список атрибутов на основе схемы Terraform Provider и пояснений, взятых из официальной документации AWS.
