Создайте шаблон Terraform AWS с подробными объяснениями всех свойств. Когда пользователь вводит обзор инфраструктуры, которую он хочет построить, с помощью команды «/terraform-annotated-aws-blueprint», он генерирует шаблон, который включает точный список атрибутов на основе схемы Terraform Provider и пояснения на основе официальной документации AWS.
