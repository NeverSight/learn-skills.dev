محرر ومدير مشروع لخادم الألعاب CCN2 (serverccn2/). استخدم هذه المهارة عندما يريد المستخدم: إدارة تكوينات الخادم، أو إنشاء بيئات النشر أو تعديلها، أو مراجعة بنية الخادم، أو إنشاء أو تحديث الوثائق الفنية للخادم، أو تحرير أفكار ميزات الخادم قبل البرمجة، أو التحقق من الاتساق بين مستندات تصميم الخادم والتعليمات البرمجية المصدر، أو إدارة خصائص الخادم عبر البيئات، أو تدقيق التكوين لجاهزية الإنتاج، أو التخطيط لوحدات/ميزات الخادم الجديدة. يتم التشغيل أيضًا عندما يذكر المستخدم: "تكوين الخادم"، أو "خصائص الخادم"، أو "إنشاء بيئة"، أو "نشر التكوين"، أو "هندسة الخادم"، أو "مستند تكنولوجيا الخادم"، أو "فحص الخادم"، أو "مراجعة التكوين"، أو "إعداد البيئة"، أو "dao_type"، أو "db_prefix_key"، أو "configByMode"، أو أي طلب يتضمن إدارة مشروع serverccn2/. تفرض هذه المهارة سير عمل التصميم أولاً: يتم تحديث الوثائق دائمًا قبل كتابة التعليمات البرمجية. يتم التحقق من صحة كل مخرجات أمر تلقائيًا عبر "validate_result" لضمان صحتها قبل الثقة في النتائج.
