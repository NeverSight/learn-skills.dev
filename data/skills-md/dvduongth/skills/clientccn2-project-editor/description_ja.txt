CCN2 ゲーム クライアント (clientccn2/) の編集者およびプロジェクト マネージャー。ユーザーが必要なときにこのスキルを使用します: クライアント モジュールの管理、新しいゲーム アクションやエフェクトの追加、クライアント側の機能の計画または設計、クライアントの技術ドキュメントの生成または更新、コーディング前のクライアント機能のアイデアの編集、クライアント コードとサーバー/GDD 間の一貫性の確認、新しいシーンまたは UI コンポーネントの追加、アクション キュー システムの管理、イベント バスの操作、クライアント アーキテクチャ (レガシー→新規) のレビューまたはリファクタリング、構成ローダーの管理、または新しい UI パネルの計画。ユーザーが「クライアント コード」、「クライアント モジュール」、「クライアント アーキテクチャ」、「アクション キュー」、「ActionQueue」、「BaseAction」、「EventBus」、「EventKeys」、「BaseModule」、「GameHUD」、「SceneGame」、「BaseScene」、「BaseGUI」、「ゲーム ロジック」、「クライアント スキャン」、「クライアント テクニカル ドキュメント」、 「Cocos2d-x」、「cocos client」、「client config」、「ConfigManager」、「ServiceContainer」、「AppContext」、「NavigationCoordinator」、「gv.bus」、「gv.signalMgr」、または clientccn2/ プロジェクト管理に関連するリクエスト。このスキルは、設計優先のワークフローを強制します。つまり、コードが記述される前にドキュメントが常に更新されます。すべてのコマンド出力は「validate_result」によって自動検証され、結果を信頼する前に正確性が保証されます。
