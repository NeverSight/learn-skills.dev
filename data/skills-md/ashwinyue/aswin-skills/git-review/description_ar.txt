أداة مراجعة الكود المحلي للفحص الذاتي قبل الدفع. يتم تشغيله عندما يطلب المستخدمون "مراجعة الكود الخاص بي"، أو "التحقق من تغييرات الكود"، أو "مراجعة هذا الالتزام"، أو "مراجعة هذا"، أو "مراجعة الكود"، أو "مراجعة git"، أو "مساعدتي في التحقق من الكود". يدعم مراجعة التغييرات غير المرحلية، والمرحلية ولكن غير الملتزم بها، والملتزمة ولكن غير المدفوعة، ويخرج تقرير مراجعة Markdown مع النتيجة.
