Herramienta de revisión de código local para autoverificación antes de enviar. Se activa cuando el usuario dice "revisar mi código", "verificar cambios de código", "revisar esta confirmación", "revisar esto", "revisar código", "revisión de git", "ayúdame a verificar el código". Admite la revisión de cambios no preparados, preparados pero no comprometidos y comprometidos pero no enviados, y genera un informe de revisión de Markdown puntuado.
