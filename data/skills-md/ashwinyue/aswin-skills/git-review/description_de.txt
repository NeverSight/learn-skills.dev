Lokales Code-Review-Tool zur Selbstprüfung vor dem Push. Wird ausgelöst, wenn der Benutzer „Überprüfe meinen Code“, „Überprüfe Codeänderungen“, „Überprüfe diesen Commit“, „Überprüfe dies“, „Codeüberprüfung“, „Git-Überprüfung“ oder „Hilf mir, meinen Code zu überprüfen“ sagt. Unterstützt die Überprüfung von nicht bereitgestellten, bereitgestellten, aber nicht festgeschriebenen und festgeschriebenen, aber nicht übertragenen Änderungen und gibt einen Markdown-Überprüfungsbericht mit einer Bewertung aus.
