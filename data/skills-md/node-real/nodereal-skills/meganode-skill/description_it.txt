Fornisce API dell'infrastruttura blockchain NodeReal MegaNode per oltre 25 catene tra cui BSC, Ethereum, opBNB, Optimism, Polygon, Arbitrum e Klaytn. Copre endpoint JSON-RPC standard, API avanzate (metodi nr_ per saldi token ERC-20, partecipazioni NFT, trasferimenti di risorse), transazioni gasless MegaFuel tramite paymaster BEP-322, protezione MEV Direct Route, API di debug/traccia, abbonamenti WebSocket, livello di consenso ETH Beacon Chain, monitoraggio dell'utilizzo dell'API del portale, API Marketplace (NFTScan, Contracts API, SPACE ID, Greenfield, Staking BNB, PancakeSwap, zkSync), catene non EVM (Aptos, NEAR, Avalanche) e autenticazione JWT. Da utilizzare durante la creazione di dApp blockchain con NodeReal, l'interrogazione di dati token o NFT, l'impostazione dell'infrastruttura RPC, la configurazione di transazioni gasless, la protezione contro MEV, il tracciamento delle transazioni, la verifica di contratti intelligenti, la risoluzione di domini .bnb o il monitoraggio dei validatori e dell'utilizzo delle API.
