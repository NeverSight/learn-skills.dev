BSC、Ethereum、opBNB、Optimism、Polygon、Arbitrum、Klaytn を含む 25 以上のチェーンに NodeReal MegaNode ブロックチェーン インフラストラクチャ API を提供します。標準の JSON-RPC エンドポイント、拡張 API (ERC-20 トークン残高、NFT 保有、資産移転の多数のメソッド)、BEP-322 ペイマスターを介した MegaFuel ガスレス トランザクション、ダイレクト ルート MEV 保護、デバッグ/トレース API、WebSocket サブスクリプション、ETH ビーコン チェーン コンセンサス レイヤー、ポータル API 使用状況モニタリング、API マーケットプレイス (NFTScan、契約 API、SPACE ID、グリーンフィールド、 BNB ステーキング、PancakeSwap、zkSync）、非 EVM チェーン（Aptos、NEAR、Avalanche）、および JWT 認証。 NodeReal を使用したブロックチェーン dApp の構築、トークンまたは NFT データのクエリ、RPC インフラストラクチャのセットアップ、ガスレス トランザクションの構成、MEV からの保護、トランザクションの追跡、スマート コントラクトの検証、.bnb ドメインの解決、またはバリデータと API の使用状況の監視を行うときに使用します。
