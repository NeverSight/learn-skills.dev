Fournit des API d'infrastructure blockchain NodeReal MegaNode pour plus de 25 chaînes, dont BSC, Ethereum, opBNB, Optimism, Polygon, Arbitrum et Klaytn. Couvre les points de terminaison JSON-RPC standard, les API améliorées (nr_ méthodes pour les soldes de jetons ERC-20, les avoirs NFT, les transferts d'actifs), les transactions sans gaz MegaFuel via le paymaster BEP-322, la protection Direct Route MEV, les API Debug/Trace, les abonnements WebSocket, la couche de consensus ETH Beacon Chain, la surveillance de l'utilisation des API du portail, le marché des API (NFTScan, API des contrats, SPACE ID, Greenfield, BNB Staking, PancakeSwap, zkSync), les chaînes non-EVM (Aptos, NEAR, Avalanche) et l'authentification JWT. À utiliser lors de la création de dApps blockchain avec NodeReal, de l'interrogation de données de jeton ou NFT, de la configuration de l'infrastructure RPC, de la configuration des transactions sans gaz, de la protection contre le MEV, du suivi des transactions, de la vérification des contrats intelligents, de la résolution des domaines .bnb ou de la surveillance des validateurs et de l'utilisation des API.
