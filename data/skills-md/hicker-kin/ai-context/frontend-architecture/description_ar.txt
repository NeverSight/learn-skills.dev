يوصي ويدعم مكدس تكنولوجيا الواجهة الأمامية للمشاريع الجديدة. يُستخدم عند بدء مشروع واجهة أمامية جديد، أو تحديد إطار عمل، أو إعداد تطبيق ويب من البداية، أو عندما يسأل المستخدم عن مجموعة تقنيات الواجهة الأمامية، أو السقالات، أو اختيار البنية. يغطي React/Next.js/Astro/Vue وTypeScript وTailwind CSS وإدارة الحالة وجلب البيانات ومكونات واجهة المستخدم والاختبار وإعداد monorepo.
