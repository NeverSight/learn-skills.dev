Recomienda y estructura la pila de tecnología frontend para nuevos proyectos. Úselo al iniciar un nuevo proyecto frontend, seleccionar un marco, configurar una aplicación web desde cero o cuando el usuario pregunte sobre la pila tecnológica, el andamiaje o la selección de arquitectura del frontend. Cubre React/Next.js/Astro/Vue, TypeScript, Tailwind CSS, gestión de estado, obtención de datos, componentes de interfaz de usuario, pruebas y configuración de monorepo.
