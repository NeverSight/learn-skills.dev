Meilleures pratiques C/C++/CAPL pour les systèmes embarqués automobiles. Cette compétence doit être utilisée lors de l'écriture, de la révision ou de la refactorisation de code C/C++ intégré ou de scripts CAPL ciblant les calculateurs automobiles, conformément aux directives MISRA, AUTOSAR, ISO 26262 et ISO 21434. Se déclenche sur des tâches impliquant un micrologiciel intégré, la communication CAN/CAN FD/LIN/Ethernet, les protocoles TCP/UDP/DoIP/SOME-IP, la programmation RTOS, le code critique pour la sécurité, la cybersécurité, les diagnostics (UDS), l'automatisation des tests CAPL ou l'intégration de la chaîne d'outils d'étalonnage.
