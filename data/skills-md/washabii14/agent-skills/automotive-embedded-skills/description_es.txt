Mejores prácticas de C/C++/CAPL para sistemas integrados de automoción. Esta habilidad debe usarse al escribir, revisar o refactorizar código C/C++ integrado o scripts CAPL destinados a ECU de automóviles, siguiendo las pautas MISRA, AUTOSAR, ISO 26262 e ISO 21434. Activa tareas que involucran firmware integrado, comunicación CAN/CAN FD/LIN/Ethernet, protocolos TCP/UDP/DoIP/SOME-IP, programación RTOS, código crítico para la seguridad, ciberseguridad, diagnóstico (UDS), automatización de pruebas CAPL o integración de cadena de herramientas de calibración.
