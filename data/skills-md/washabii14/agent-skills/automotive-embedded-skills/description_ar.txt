أفضل ممارسات C/C++/CAPL للأنظمة المدمجة في السيارات. يجب استخدام هذه المهارة عند كتابة أو مراجعة أو إعادة بناء كود C/C++ المضمن أو نصوص CAPL التي تستهدف وحدات التحكم الإلكترونية في السيارات، باتباع إرشادات MISRA وAUTOSAR وISO 26262 وISO 21434. تشغيل المهام التي تتضمن البرامج الثابتة المضمنة، أو اتصالات CAN/CAN FD/LIN/Ethernet، أو بروتوكولات TCP/UDP/DoIP/SOME-IP، أو برمجة RTOS، أو التعليمات البرمجية الحرجة للسلامة، أو الأمن السيبراني، أو التشخيص (UDS)، أو أتمتة اختبار CAPL، أو تكامل سلسلة أدوات المعايرة.
