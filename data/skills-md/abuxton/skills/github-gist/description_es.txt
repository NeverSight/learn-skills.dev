Cree, administre y organice GitHub Gists usando la CLI de gh. Utilice esta habilidad siempre que un usuario desee crear una esencia, compartir fragmentos de código, administrar esencias existentes, agregar archivos a una esencia, enumerar o buscar esencias, editar contenido o descripciones de esencia, clonar una esencia o eliminar una esencia. Cada esencia creada o modificada debe incluir un README.md que documente su propósito y contenido.
