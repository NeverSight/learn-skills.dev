Crea, gestisci e organizza GitHub Gists utilizzando la CLI gh. Utilizza questa competenza ogni volta che un utente desidera creare un contenuto, condividere frammenti di codice, gestire i contenuti esistenti, aggiungere file a un contenuto, elencare o cercare contenuti, modificare il contenuto o le descrizioni del contenuto, clonare un contenuto o eliminare un contenuto. Ogni sostanza creata o modificata deve includere un file README.md che ne documenti lo scopo e il contenuto.
