Создавайте, управляйте и систематизируйте GitHub Gists с помощью gh CLI. Используйте этот навык всякий раз, когда пользователь хочет создать суть, поделиться фрагментами кода, управлять существующими сутью, добавить файлы в суть, составить список или выполнить поиск по сути, отредактировать содержимое или описание сущности, клонировать суть или удалить суть. Каждый созданный или измененный gist должен включать README.md, документирующий его назначение и содержимое.
