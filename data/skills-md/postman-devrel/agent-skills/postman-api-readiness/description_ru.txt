Анализируйте любой API на совместимость с агентом ИИ. Сканирует спецификации OpenAPI по 8 компонентам (48 проверок), оценивает готовность агента от 0 до 100 и предоставляет приоритетные рекомендации по исправлению ситуации. Используйте этот навык, когда пользователь спрашивает: «Готов ли мой агент API?», «Сканируйте мой API», «Проанализируйте мою спецификацию OpenAPI», «Что не так с моим API для агентов ИИ?», «Насколько удобен для агентов мой API?» или хочет улучшить свой API для использования ИИ. Работает с локальными спецификациями и коллекциями Postman через MCP.
