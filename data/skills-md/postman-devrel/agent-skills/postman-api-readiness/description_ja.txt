AI エージェントの互換性について API を分析します。 OpenAPI 仕様を 8 つの柱 (48 チェック) にわたってスキャンし、エージェントの対応状況を 0 ～ 100 でスコア付けし、優先順位付けされた修正の推奨事項を提供します。このスキルは、ユーザーが「私の API エージェント対応ですか?」、「私の API をスキャンしてください」、「OpenAPI 仕様を分析してください」、「AI エージェント用の API の何が問題になっていますか?」、「私の API はどの程度エージェントに適していますか?」と尋ねた場合、または AI 利用のために API を改善したい場合に使用します。 MCP を介してローカル仕様および Postman コレクションで動作します。
