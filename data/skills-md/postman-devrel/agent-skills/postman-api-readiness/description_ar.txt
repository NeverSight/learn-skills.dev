قم بتحليل أي واجهة برمجة تطبيقات (API) للتأكد من توافقها مع وكيل الذكاء الاصطناعي. يقوم بمسح مواصفات OpenAPI عبر 8 ركائز (48 عملية فحص)، ويسجل جاهزية الوكيل من 0 إلى 100، ويقدم توصيات الإصلاح ذات الأولوية. استخدم هذه المهارة عندما يسأل المستخدم: "هل وكيل واجهة برمجة التطبيقات الخاص بي جاهز؟"، أو "فحص واجهة برمجة التطبيقات الخاصة بي"، أو "تحليل مواصفات OpenAPI الخاصة بي"، أو "ما الخطأ في واجهة برمجة التطبيقات الخاصة بي لوكلاء الذكاء الاصطناعي؟"، أو "ما مدى ملاءمة واجهة برمجة التطبيقات الخاصة بي؟"، أو يريد تحسين واجهة برمجة التطبيقات الخاصة به لاستهلاك الذكاء الاصطناعي. يعمل مع المواصفات المحلية ومجموعات Postman عبر MCP.
