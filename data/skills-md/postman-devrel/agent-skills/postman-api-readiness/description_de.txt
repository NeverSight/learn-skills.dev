Analysieren Sie jede API auf Kompatibilität mit KI-Agenten. Scannt OpenAPI-Spezifikationen über 8 Säulen hinweg (48 Prüfungen), bewertet die Agentenbereitschaft mit 0–100 und bietet priorisierte Korrekturempfehlungen. Verwenden Sie diesen Skill, wenn der Benutzer fragt: „Ist mein API-Agent bereit?“, „Meine API scannen“, „Meine OpenAPI-Spezifikation analysieren“, „Was stimmt mit meiner API für KI-Agenten nicht?“, „Wie agentenfreundlich ist meine API?“ oder seine API für die KI-Verwendung verbessern möchte. Funktioniert mit lokalen Spezifikationen und Postman-Sammlungen über MCP.
