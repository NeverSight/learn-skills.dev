Gestion complète du cycle de vie des API via Postman. Synchronisez les spécifications OpenAPI avec les collections, générez du code client tapé, exécutez des tests d'API, créez des serveurs fictifs, publiez de la documentation, auditez la sécurité par rapport à l'OWASP Top 10 et découvrez les API dans les espaces de travail. Nécessite le serveur Postman MCP. Utilisez cette compétence lorsque l'utilisateur mentionne Postman, les collections d'API, les spécifications de synchronisation, la génération de SDK, l'exécution de tests d'API, la création de simulations, la documentation d'API ou les audits de sécurité d'API. Déclencheurs sur des tâches impliquant des workflows de développement d'API, la gestion des collections ou toute opération liée à Postman.
