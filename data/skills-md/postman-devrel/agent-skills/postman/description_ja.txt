Postman による完全な API ライフサイクル管理。 OpenAPI 仕様をコレクションに同期し、型指定されたクライアント コードを生成し、API テストを実行し、モック サーバーを作成し、ドキュメントを公開し、OWASP Top 10 に対するセキュリティを監査し、ワークスペース全体の API を検出します。 Postman MCP サーバーが必要です。このスキルは、ユーザーが Postman、API コレクション、仕様の同期、SDK の生成、API テストの実行、モックの作成、API ドキュメント、または API セキュリティ監査について言及した場合に使用します。 API 開発ワークフロー、コレクション管理、または Postman 関連の操作に関連するタスクをトリガーします。
