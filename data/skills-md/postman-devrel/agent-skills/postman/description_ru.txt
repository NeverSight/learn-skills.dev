Полное управление жизненным циклом API через Postman. Синхронизируйте спецификации OpenAPI с коллекциями, генерируйте типизированный клиентский код, запускайте тесты API, создавайте макеты серверов, публикуйте документацию, проверяйте безопасность на соответствие топ-10 OWASP и обнаруживайте API в рабочих областях. Требуется сервер Postman MCP. Используйте этот навык, когда пользователь упоминает Postman, коллекции API, спецификации синхронизации, создание SDK, запуск тестов API, создание макетов, документацию API или аудит безопасности API. Запускается при выполнении задач, связанных с рабочими процессами разработки API, управлением коллекциями или любыми операциями, связанными с Postman.
