Gestione completa del ciclo di vita dell'API tramite Postman. Sincronizza le specifiche OpenAPI con le raccolte, genera codice client digitato, esegui test API, crea server fittizi, pubblica documentazione, controlla la sicurezza rispetto alla Top 10 OWASP e scopri le API negli spazi di lavoro. Richiede il server Postman MCP. Utilizza questa competenza quando l'utente menziona Postman, raccolte API, specifiche di sincronizzazione, generazione di SDK, esecuzione di test API, creazione di mock, documentazione API o controlli di sicurezza API. Attiva attività che coinvolgono flussi di lavoro di sviluppo API, gestione delle raccolte o qualsiasi operazione correlata a Postman.
