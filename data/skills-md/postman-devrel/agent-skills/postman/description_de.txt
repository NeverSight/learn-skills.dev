Vollständiges API-Lebenszyklusmanagement durch Postman. Synchronisieren Sie OpenAPI-Spezifikationen mit Sammlungen, generieren Sie typisierten Client-Code, führen Sie API-Tests aus, erstellen Sie Scheinserver, veröffentlichen Sie Dokumentation, prüfen Sie die Sicherheit anhand der OWASP Top 10 und entdecken Sie APIs in allen Arbeitsbereichen. Erfordert den Postman MCP-Server. Verwenden Sie diese Fähigkeit, wenn der Benutzer Postman, API-Sammlungen, das Synchronisieren von Spezifikationen, das Generieren von SDKs, das Ausführen von API-Tests, das Erstellen von Mocks, API-Dokumentation oder API-Sicherheitsüberprüfungen erwähnt. Wird bei Aufgaben ausgelöst, die API-Entwicklungsworkflows, Sammlungsverwaltung oder alle Postman-bezogenen Vorgänge umfassen.
