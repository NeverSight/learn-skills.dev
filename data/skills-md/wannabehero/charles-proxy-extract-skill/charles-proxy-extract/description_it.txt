Estrae i dati di richiesta e risposta HTTP/HTTPS dai file di sessione Charles Proxy (formato .chlsj), inclusi URL, metodi, codici di stato, intestazioni, corpi di richiesta e corpi di risposta. Da utilizzare durante l'analisi del traffico di rete acquisito dalle sessioni di debug di Charles Proxy, l'ispezione delle chiamate API, il debug delle richieste HTTP o l'esame dei log proxy.
