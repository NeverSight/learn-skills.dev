Навык отражения опыта: анализируйте диалог, чтобы извлечь воспоминания и классифицировать их по типам: пары дилемма-стратегия → опыт/, понимание проекта/технические исследования → знания/. Поддерживает пользовательские запросы (например, «записать поток данных») для автоматического определения типа. Сценарии срабатывания: после выполнения задачи (рекомендуется), после решения сложной задачи, когда пользователь просит обобщить опыт, когда пользователь просит зафиксировать понимание проекта.
