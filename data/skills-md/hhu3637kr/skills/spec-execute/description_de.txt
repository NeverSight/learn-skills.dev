Befolgen Sie plan.md strikt, um neue Funktionen zu entwickeln. Wird vom Rollenspezifikationsausführer aufgerufen. Auslöserbedingungen: (1) Der Rollen-Spec-Executor muss Code gemäß plan.md implementieren, (2) der Benutzer sagt „Spec ausführen“ / „Implementierung starten“, (3) im spec/-Verzeichnis ist eine plan.md vorhanden, aber keine summary.md. Hinweis: Ab Version 2.0 wurden Pfad B (Agententeams) und Testschritte entfernt. spec-execute hat nur eine Aufgabe: Code gemäß plan.md implementieren, summary.md erstellen und archivieren. Wenn das Verzeichnis bereits summary.md enthält, verwenden Sie spec-update.
