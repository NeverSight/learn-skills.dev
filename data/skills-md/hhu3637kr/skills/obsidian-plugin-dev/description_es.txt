Guía de desarrollo de complementos de Obsidian, que proporciona plantillas de inicialización de proyectos, archivos de configuración, soluciones a problemas comunes y ejemplos de uso de API. Úselo al desarrollar complementos de Obsidian, resolver problemas de desarrollo de complementos de Obsidian o consultar el uso de la API de Obsidian. Escenarios desencadenantes: (1) Crear un nuevo proyecto de complemento de Obsidian, (2) Configurar el entorno de compilación esbuild/TypeScript, (3) Resolver problemas de carga del módulo Node.js, (4) Usar la API de Obsidian (operaciones de archivos, frontmatter, componentes de UI, registro de comandos, escucha de eventos), (5) Depurar complementos de Obsidian.
