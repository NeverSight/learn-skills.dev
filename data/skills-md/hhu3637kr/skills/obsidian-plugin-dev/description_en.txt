Obsidian Plugin Development Guide, providing project initialization templates, configuration files, solutions to common issues, and API usage examples. Use it when developing Obsidian plugins, solving development problems, or querying Obsidian API usage. Trigger scenarios: (1) creating a new Obsidian plugin project, (2) configuring esbuild/TypeScript build environment, (3) resolving Node.js module loading issues, (4) using Obsidian API (file operations, frontmatter, UI components, command registration, event listening), (5) debugging Obsidian plugins.
