Запустите рабочий процесс групп агентов разработки на основе спецификаций. Используйте его, когда пользователь приступает к новой задаче разработки и ему необходимо создать структурированную многоролевую команду для совместной работы. Соответствует спецификации; Spec-start запускает команду, spec-end завершает ее. Условия триггера: (1) пользователь говорит «начать новую задачу»/«создать команду разработчиков»/«запустить рабочий процесс», (2) требуется полный цикл разработки, основанный на спецификациях (исследовать → написать → внедрить → протестировать → подвести итоги), (3) пользователь хочет управлять задачей разработки с помощью архитектуры команд агентов. Примечание: этот навык предполагает, что проект уже инициализирован с помощью spec-init. Если каталог spec/ не существует, сначала вызовите spec-init.
