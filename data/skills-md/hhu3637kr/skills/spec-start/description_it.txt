Avvia un flusso di lavoro di sviluppo basato sulle specifiche Agent Teams. Da utilizzare quando l'utente avvia una nuova attività di sviluppo e deve creare un team di collaborazione strutturato con più ruoli. Abbinato a spec-end: spec-start dà il via alla squadra, spec-end la conclude. Condizioni di attivazione: (1) l'utente dice "avvia una nuova attività"/"crea team di sviluppo"/"avvia flusso di lavoro", (2) è richiesto un ciclo di sviluppo completo basato sulle specifiche (esplora→scrivi→implement→test→wrap-up), (3) l'utente desidera gestire un'attività di sviluppo tramite l'architettura Agent Teams. Nota: questa abilità presuppone che il progetto abbia già eseguito spec-init; se la directory spec/ non esiste, chiamare prima spec-init.
