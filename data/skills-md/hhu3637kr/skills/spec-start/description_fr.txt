Lancez le workflow Agent Teams pour le développement basé sur les spécifications. Utilisez-le lorsqu'un utilisateur démarre une nouvelle tâche de développement et doit créer une équipe collaborative structurée et multirôle. Associé à spec-end, spec-start initie l'équipe et spec-end la ferme. Conditions de déclenchement : (1) l'utilisateur dit « démarrer une nouvelle tâche » / « créer une équipe de développement » / « lancer le workflow », (2) un processus de développement complet basé sur les spécifications est requis (explorer → écrire → implémenter → tester → fermer), (3) l'utilisateur souhaite gérer une tâche de développement avec l'architecture Agent Teams. Remarque : cette compétence suppose que le projet a déjà été initialisé avec spec-init ; si le répertoire spec/ n'existe pas, appelez d'abord spec-init.
