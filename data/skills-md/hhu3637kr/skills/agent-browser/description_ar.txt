أتمتة المتصفح بدون رأس لوكلاء الذكاء الاصطناعي باستخدام CLI لمتصفح الوكيل. يُستخدم عندما يحتاج Claude إلى أتمتة تصفح الويب، أو استخراج بيانات الويب، أو التفاعل مع صفحات الويب، أو ملء النماذج، أو التقاط لقطات الشاشة، أو تنفيذ أي مهام تعتمد على المتصفح. يدعم استهداف العناصر المستندة إلى المرجع وإدارة الجلسة ومحددات المواقع الدلالية.
