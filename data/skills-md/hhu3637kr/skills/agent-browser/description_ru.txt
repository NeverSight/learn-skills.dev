Безголовая автоматизация браузера для агентов ИИ с использованием интерфейса командной строки агент-браузер. Используйте, когда Клоду необходимо автоматизировать просмотр веб-страниц, собирать веб-данные, взаимодействовать с веб-страницами, заполнять формы, делать снимки экрана или выполнять любые задачи в браузере. Поддерживает таргетинг элементов на основе ссылок, управление сеансами и семантические локаторы.
