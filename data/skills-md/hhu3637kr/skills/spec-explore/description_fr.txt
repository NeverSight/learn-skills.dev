Collecte et exploration d'informations avant la création de spécifications. Appelé par le rôle spec-explorer. Conditions de déclenchement : (1) le rôle spec-explorer doit collecter des informations de base avant de créer la spécification, (2) l'expérience historique doit être récupérée (exp-search), (3) la base de code du projet, la documentation externe ou les bibliothèques tierces doivent être explorées, (4) TeamLead informe spec-explorer de commencer le travail.
