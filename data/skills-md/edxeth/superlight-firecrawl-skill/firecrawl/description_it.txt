Raschia ed esegue la scansione delle pagine Web, convertendole in markdown pulito o JSON strutturato per il consumo LLM. Utilizzare quando è necessario estrarre contenuti da URL, eseguire la scansione di interi siti Web, mappare la struttura del sito, eseguire ricerche sul Web con lo scraping o estrarre dati strutturati dalle pagine. Ideale per web scraping, scansione di siti, rilevamento di URL e conversione di contenuti Web in formati pronti per LLM.
