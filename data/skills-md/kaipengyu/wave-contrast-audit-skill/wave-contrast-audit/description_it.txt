Controlla una pagina web live per individuare eventuali errori di contrasto dei colori WCAG utilizzando axe-core iniettato tramite Playwright, quindi mappa ogni coppia di colori difettosa sulla sua esatta regola CSS e applica correzioni conformi. Da utilizzare quando un report WAVE mostra errori di contrasto o quando viene richiesto di risolvere i problemi di contrasto di accessibilità su una pagina distribuita.
