Aiuta nella scrittura del codice Mojo che interagisce con Python utilizzando la sintassi e le convenzioni attuali. Usa questa abilità in aggiunta alla sintassi mojo quando scrivi codice Mojo che interagisce con Python, chiama librerie Python da Mojo o espone tipi/funzioni Mojo a Python. Da utilizzare anche quando l'utente desidera creare moduli di estensione Python in Mojo, eseguire il wrap delle strutture Mojo per l'utilizzo di Python o convertire tra tipi Python e Mojo.
