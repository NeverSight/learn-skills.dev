Ayuda a escribir código Mojo que interopera con Python utilizando la sintaxis y las convenciones actuales. Utilice esta habilidad además de la sintaxis de mojo cuando escriba código Mojo que interactúe con Python, llame a bibliotecas de Python desde Mojo o exponga tipos/funciones de Mojo a Python. Úselo también cuando el usuario quiera crear módulos de extensión de Python en Mojo, empaquetar estructuras de Mojo para el consumo de Python o convertir entre tipos de Python y Mojo.
