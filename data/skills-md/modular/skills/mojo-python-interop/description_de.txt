Hilft beim Schreiben von Mojo-Code, der mit Python unter Verwendung aktueller Syntax und Konventionen interagiert. Verwenden Sie diese Fähigkeit zusätzlich zur Mojo-Syntax, wenn Sie Mojo-Code schreiben, der mit Python interagiert, Python-Bibliotheken von Mojo aus aufruft oder Mojo-Typen/-Funktionen für Python verfügbar macht. Wird auch verwendet, wenn der Benutzer Python-Erweiterungsmodule in Mojo erstellen, Mojo-Strukturen für die Python-Verwendung umschließen oder zwischen Python- und Mojo-Typen konvertieren möchte.
