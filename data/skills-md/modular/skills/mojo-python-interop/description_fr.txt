Aide à écrire du code Mojo qui interagit avec Python en utilisant la syntaxe et les conventions actuelles. Utilisez cette compétence en plus de la syntaxe Mojo lors de l'écriture de code Mojo qui interagit avec Python, appelle les bibliothèques Python à partir de Mojo ou expose les types/fonctions Mojo à Python. À utiliser également lorsque l'utilisateur souhaite créer des modules d'extension Python dans Mojo, envelopper des structures Mojo pour la consommation Python ou effectuer une conversion entre les types Python et Mojo.
