Competenza nel framework Laravel per scrivere, rivedere e testare applicazioni PHP create con Laravel. Da utilizzare ogni volta che si lavora su modelli specifici di Laravel: controller, FormRequest, modelli Eloquent, classi di servizi/azioni, modelli Blade, lavori, eventi, policy, osservatori o comandi Artisan. Si applica anche ai frontend Inertia (React o Vue) incorporati all'interno di un progetto Laravel: copre useForm, usePage, navigazione dei collegamenti, ricaricamenti parziali, denominazione dei componenti della pagina (PostsIndexPage) e convenzioni wrapper shadcn. Si attiva ogni volta che qualcuno menziona Laravel, Eloquent, Artisan, Blade, FormRequest, Inertia all'interno di Laravel, query N+1 in un'app Laravel, convalida del controller in linea, dove la logica aziendale dovrebbe risiedere in un controller Laravel o chiede di seguire le migliori pratiche in una base di codice Laravel. Non si applica ai semplici script PHP senza Laravel, Rails, Django, Next.js autonomo o framework non Laravel.
