Laravel 框架技能，用於編寫、審查和測試使用 Laravel 建立的 PHP 應用程式。每當處理 Laravel 特定模式時使用：控制器、FormRequests、Eloquent 模型、服務/作業類別、Blade 範本、作業、事件、策略、觀察者或 Artisan 指令。也適用於嵌入 Laravel 專案中的 Inertia 前端（React 或 Vue）—涵蓋 useForm、usePage、連結導覽、部分重新載入、頁面元件命名 (PostsIndexPage) 和 shadcn 包裝器約定。每當有人提及 Laravel、Eloquent、Artisan、Blade、FormRequest、Laravel 中的 Inertia、Laravel 應用程式中的 N+1 查詢、內聯控制器驗證（其中業務邏輯應位於 Laravel 控制器中）或要求遵循 Laravel 程式碼庫中的最佳實踐時都會觸發。不適用於沒有 Laravel、Rails、Django、獨立 Next.js 或非 Laravel 框架的純 PHP 腳本。
