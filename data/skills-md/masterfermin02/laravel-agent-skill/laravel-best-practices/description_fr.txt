Compétence du framework Laravel pour écrire, réviser et tester des applications PHP créées avec Laravel. À utiliser lorsque vous travaillez sur des modèles spécifiques à Laravel : contrôleurs, FormRequests, modèles Eloquent, classes de service/action, modèles Blade, tâches, événements, politiques, observateurs ou commandes Artisan. S'applique également aux interfaces Inertia (React ou Vue) intégrées dans un projet Laravel - couvrant useForm, usePage, la navigation par lien, les rechargements partiels, la dénomination des composants de page (PostsIndexPage) et les conventions de wrapper shadcn. Déclenchez chaque fois que quelqu'un mentionne Laravel, Eloquent, Artisan, Blade, FormRequest, Inertia dans Laravel, les requêtes N+1 dans une application Laravel, la validation du contrôleur en ligne, où la logique métier devrait vivre dans un contrôleur Laravel, ou demande de suivre les meilleures pratiques dans une base de code Laravel. Ne s'applique pas aux scripts PHP simples sans Laravel, Rails, Django, Next.js autonome ou les frameworks non-Laravel.
