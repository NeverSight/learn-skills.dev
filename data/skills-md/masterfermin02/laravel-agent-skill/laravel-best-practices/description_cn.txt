Laravel 框架技能，用于编写、审查和测试使用 Laravel 构建的 PHP 应用程序。每当处理 Laravel 特定模式时使用：控制器、FormRequests、Eloquent 模型、服务/操作类、Blade 模板、作业、事件、策略、观察者或 Artisan 命令。也适用于嵌入 Laravel 项目中的 Inertia 前端（React 或 Vue）——涵盖 useForm、usePage、链接导航、部分重新加载、页面组件命名 (PostsIndexPage) 和 shadcn 包装器约定。每当有人提及 Laravel、Eloquent、Artisan、Blade、FormRequest、Laravel 中的 Inertia、Laravel 应用程序中的 N+1 查询、内联控制器验证（其中业务逻辑应位于 Laravel 控制器中）或要求遵循 Laravel 代码库中的最佳实践时都会触发。不适用于没有 Laravel、Rails、Django、独立 Next.js 或非 Laravel 框架的纯 PHP 脚本。
