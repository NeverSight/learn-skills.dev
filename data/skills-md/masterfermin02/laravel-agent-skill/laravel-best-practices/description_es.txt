Habilidad del marco Laravel para escribir, revisar y probar aplicaciones PHP creadas con Laravel. Úselo siempre que trabaje en patrones específicos de Laravel: controladores, FormRequests, modelos Eloquent, clases de servicio/acción, plantillas Blade, trabajos, eventos, políticas, observadores o comandos Artisan. También se aplica a las interfaces de Inertia (React o Vue) integradas dentro de un proyecto de Laravel, que cubren useForm, usePage, navegación de enlaces, recargas parciales, denominación de componentes de página (PostsIndexPage) y convenciones de contenedor shadcn. Se activa cada vez que alguien menciona Laravel, Eloquent, Artisan, Blade, FormRequest, Inertia dentro de Laravel, consultas N+1 en una aplicación de Laravel, validación de controlador en línea, dónde la lógica de negocios debe residir en un controlador de Laravel o solicita seguir las mejores prácticas en un código base de Laravel. No se aplica a scripts PHP simples sin Laravel, Rails, Django, Next.js independiente o marcos que no sean Laravel.
