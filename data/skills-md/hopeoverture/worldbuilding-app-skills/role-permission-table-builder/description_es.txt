Genera matrices de permisos integrales basadas en roles en formato Markdown o SQL para páginas, componentes y patrones de acceso a datos. Esta habilidad debe utilizarse al diseñar sistemas de autorización, documentar permisos, crear tablas RBAC o planificar el control de acceso. Úselo para RBAC, permisos de roles, control de acceso, matriz de autorización, asignación de permisos o políticas de seguridad.
