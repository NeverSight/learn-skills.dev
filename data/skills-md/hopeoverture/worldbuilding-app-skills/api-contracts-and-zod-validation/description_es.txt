Genere esquemas Zod y tipos TypeScript para formularios, rutas API y acciones del servidor con validación en tiempo de ejecución. Utilice esta habilidad al crear contratos API, validar cargas útiles de solicitud/respuesta, generar esquemas de formulario, agregar validación de entrada a acciones del servidor o controladores de ruta, o garantizar la seguridad de tipos entre los límites cliente-servidor. Los términos desencadenantes incluyen zod, esquema, validación, contrato API, validación de formulario, inferencia de tipos, validación en tiempo de ejecución, análisis, safeParse, validación de entrada, validación de solicitudes y validación de acciones del servidor.
