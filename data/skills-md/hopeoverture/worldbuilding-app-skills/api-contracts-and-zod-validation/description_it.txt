Genera schemi Zod e tipi TypeScript per moduli, percorsi API e azioni server con convalida runtime. Utilizza questa competenza durante la creazione di contratti API, la convalida di payload di richiesta/risposta, la generazione di schemi di moduli, l'aggiunta di convalida dell'input alle azioni del server o ai gestori di routing o la garanzia della sicurezza dei tipi oltre i confini client-server. I termini del trigger includono zod, schema, convalida, contratto API, convalida del modulo, inferenza del tipo, convalida del runtime, analisi, safeParse, convalida dell'input, convalida della richiesta, convalida dell'azione del server.
