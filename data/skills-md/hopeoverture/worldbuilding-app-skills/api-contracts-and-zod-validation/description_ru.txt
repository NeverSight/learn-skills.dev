Создавайте схемы Zod и типы TypeScript для форм, маршрутов API и действий сервера с проверкой во время выполнения. Используйте этот навык при создании контрактов API, проверке полезных данных запроса/ответа, создании схем форм, добавлении проверки ввода в действия сервера или обработчики маршрутов, а также при обеспечении безопасности типов на границах клиент-сервер. Термины триггера включают zod, схему, проверку, контракт API, проверку формы, вывод типа, проверку во время выполнения, синтаксический анализ, SafeParse, проверку ввода, проверку запроса, проверку действия сервера.
