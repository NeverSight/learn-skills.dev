Cette compétence doit être utilisée lorsque l'utilisateur demande à échafauder, créer ou initialiser une application Next.js complète avec une pile technologique moderne comprenant Next.js 16, React 19, TypeScript, Tailwind CSS v4, shadcn/ui, Supabase auth, Prisma ORM et une configuration de test complète. Utilisez-le pour créer des modèles de démarrage prêts pour la production avec une authentification, des itinéraires protégés, des formulaires et des exemples de fonctionnalités. Échafaudage de termes de déclenchement, création d'une application nextjs, initialisation de fullstack, modèle de démarrage, passe-partout, configuration de nextjs, modèle de production, configuration full-stack, supabase nextjs, prisma nextjs.
