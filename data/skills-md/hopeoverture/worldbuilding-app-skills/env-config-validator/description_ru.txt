Проверка файлов конфигурации среды в локальной, промежуточной и производственной средах. Убедитесь, что необходимые секреты, URL-адреса базы данных, ключи API и общедоступные переменные правильно определены и установлены. Используйте этот навык при настройке сред, проверке конфигурации, проверке недостающих секретов, аудите переменных среды, обеспечении правильного определения области действия общедоступных и частных переменных или устранении неполадок среды. Термины триггера включают env, переменные среды, секреты, конфигурацию, файл .env, проверку среды, отсутствующие переменные, проверку конфигурации, NEXT_PUBLIC, переменные среды, URL-адрес базы данных, ключи API.
