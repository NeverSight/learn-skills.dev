Valide los archivos de configuración del entorno en entornos locales, provisionales y de producción. Asegúrese de que los secretos requeridos, las URL de bases de datos, las claves API y las variables públicas tengan el alcance y la configuración adecuados. Utilice esta habilidad al configurar entornos, validar la configuración, comprobar si faltan secretos, auditar variables de entorno, garantizar el alcance adecuado de las variables públicas y privadas o solucionar problemas de entorno. Los términos de activación incluyen env, variables de entorno, secretos, configuración, archivo .env, validación de entorno, variables faltantes, verificación de configuración, NEXT_PUBLIC, env vars, URL de base de datos, claves API.
