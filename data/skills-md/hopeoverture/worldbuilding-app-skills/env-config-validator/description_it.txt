Convalida i file di configurazione dell'ambiente negli ambienti locali, di gestione temporanea e di produzione. Assicurati che i segreti richiesti, gli URL dei database, le chiavi API e le variabili pubbliche abbiano un ambito e siano impostati correttamente. Utilizza questa competenza durante l'impostazione degli ambienti, la convalida della configurazione, il controllo dei segreti mancanti, il controllo delle variabili di ambiente, la garanzia dell'ambito corretto delle variabili pubbliche e private o la risoluzione dei problemi dell'ambiente. I termini di attivazione includono env, variabili di ambiente, segreti, configurazione, file .env, convalida dell'ambiente, variabili mancanti, controllo di configurazione, NEXT_PUBLIC, variabili env, URL del database, chiavi API.
