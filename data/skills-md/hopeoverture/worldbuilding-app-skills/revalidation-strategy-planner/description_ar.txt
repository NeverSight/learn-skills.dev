يقوم بتقييم مسارات Next.js وإخراج إعدادات إعادة التحقق المثالية أو علامات التخزين المؤقت لـ ISR أو تكوينات SSR أو أنماط التدفق. يجب استخدام هذه المهارة عند تحسين استراتيجيات التخزين المؤقت لـ Next.js، أو تكوين التجديد الثابت التزايدي، أو التخطيط لإبطال ذاكرة التخزين المؤقت، أو الاختيار بين SSR/ISR/SSG. يُستخدم للتخزين المؤقت لـ Next.js، أو إعادة التحقق، أو ISR، أو علامات ذاكرة التخزين المؤقت، أو إعادة التحقق عند الطلب، أو استراتيجيات العرض.
