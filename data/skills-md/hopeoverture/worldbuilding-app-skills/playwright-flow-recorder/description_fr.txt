Crée des scripts de test Playwright à partir de descriptions de flux d'utilisateurs en langage naturel. Cette compétence doit être utilisée lors de la génération de tests E2E à partir de scénarios, de la conversion de user stories en code de test, de l'enregistrement des flux d'utilisateurs, de la création de scripts de test à partir de descriptions telles que « l'utilisateur s'inscrit et crée un projet » ou de la traduction des critères d'acceptation en tests exécutables. Les termes déclencheurs incluent le test du dramaturge, le flux e2e, le scénario utilisateur, le test à partir de la description, le flux d'enregistrement, le parcours utilisateur, la génération de scripts de test, le test d'acceptation, le test de comportement, le test de user story.
