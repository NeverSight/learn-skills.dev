Crea script di test del drammaturgo dalle descrizioni del flusso utente in linguaggio naturale. Questa competenza dovrebbe essere utilizzata durante la generazione di test E2E da scenari, la conversione di storie utente in codice di test, la registrazione di flussi di utenti, la creazione di script di test da descrizioni come "l'utente si iscrive e crea un progetto" o la traduzione dei criteri di accettazione in test eseguibili. I termini di attivazione includono test del drammaturgo, flusso e2e, scenario utente, test dalla descrizione, flusso di record, percorso dell'utente, generazione di script di test, test di accettazione, test di comportamento, test della storia dell'utente.
