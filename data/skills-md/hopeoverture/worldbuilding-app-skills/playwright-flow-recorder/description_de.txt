Erstellt Playwright-Testskripte aus Benutzerablaufbeschreibungen in natürlicher Sprache. Diese Fähigkeit sollte beim Generieren von E2E-Tests aus Szenarien, beim Konvertieren von User Stories in Testcode, beim Aufzeichnen von Benutzerflüssen, beim Erstellen von Testskripten aus Beschreibungen wie „Benutzer meldet sich an und erstellt Projekt“ oder beim Übersetzen von Akzeptanzkriterien in ausführbare Tests verwendet werden. Zu den Auslöserbegriffen gehören Dramatikertest, E2E-Flow, Benutzerszenario, Test aus Beschreibung, Datensatzfluss, Benutzerreise, Testskriptgenerierung, Akzeptanztest, Verhaltenstest, User Story-Test.
