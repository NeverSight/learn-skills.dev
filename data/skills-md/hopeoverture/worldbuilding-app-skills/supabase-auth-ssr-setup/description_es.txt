Esta habilidad se debe utilizar al configurar Supabase Auth para la representación del lado del servidor con Next.js App Router, incluido el manejo seguro de cookies, protección de middleware, protecciones de ruta, utilidades de autenticación y flujo de cierre de sesión. Aplicar al configurar la autenticación SSR, agregar rutas protegidas, implementar la autenticación de middleware, configurar sesiones seguras o crear flujos de inicio y cierre de sesión con Supabase.
