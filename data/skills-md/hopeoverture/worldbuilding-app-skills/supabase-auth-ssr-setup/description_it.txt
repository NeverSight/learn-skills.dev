Questa competenza deve essere utilizzata durante la configurazione di Supabase Auth per il rendering lato server con Next.js App Router, inclusa la gestione sicura dei cookie, la protezione del middleware, le protezioni del percorso, le utilità di autenticazione e il flusso di disconnessione. Applicare durante la configurazione dell'autenticazione SSR, l'aggiunta di percorsi protetti, l'implementazione dell'autenticazione middleware, la configurazione di sessioni sicure o la creazione di flussi di accesso/disconnessione con Supabase.
