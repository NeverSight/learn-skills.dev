يجب استخدام هذه المهارة عند تكوين Supabase Auth للعرض من جانب الخادم باستخدام Next.js App Router، بما في ذلك التعامل الآمن مع ملفات تعريف الارتباط وحماية البرامج الوسيطة وحراس المسار وأدوات المصادقة المساعدة وتدفق تسجيل الخروج. يتم تطبيقه عند إعداد مصادقة SSR، أو إضافة مسارات محمية، أو تنفيذ مصادقة البرامج الوسيطة، أو تكوين جلسات آمنة، أو إنشاء تدفقات تسجيل الدخول/الخروج باستخدام Supabase.
