Этот навык следует использовать при настройке Supabase Auth для рендеринга на стороне сервера с помощью Next.js App Router, включая безопасную обработку файлов cookie, защиту промежуточного программного обеспечения, защиту маршрутов, утилиты аутентификации и поток выхода из системы. Применяется при настройке аутентификации SSR, добавлении защищенных маршрутов, реализации аутентификации промежуточного программного обеспечения, настройке безопасных сеансов или построении потоков входа и выхода из системы с помощью Supabase.
