Analysez les routes et recommandez l'utilisation d'actions serveur ou de routes API en fonction de modèles de cas d'utilisation, notamment l'authentification, la revalidation, les appels d'API externes et les exigences du client. Utilisez cette compétence pour décider entre les actions du serveur et les routes API, pour optimiser la récupération de données Next.js, pour refactoriser les routes, pour analyser l'architecture des routes ou pour choisir le bon modèle de mutation de données. Les termes de déclenchement incluent les actions du serveur, les routes API, le gestionnaire de routes, la mutation des données, la revalidation, le flux d'authentification, l'API externe, la récupération côté client, l'optimisation des routes, les modèles Next.js.
