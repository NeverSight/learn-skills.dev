Analizza i percorsi e consiglia se utilizzare azioni server o percorsi API in base a modelli di casi d'uso tra cui autenticazione, riconvalida, chiamate API esterne e requisiti del client. Utilizza questa competenza per decidere tra azioni server e percorsi API, ottimizzare il recupero dei dati Next.js, eseguire il refactoring dei percorsi, analizzare l'architettura dei percorsi o scegliere il giusto modello di mutazione dei dati. I termini di attivazione includono azioni del server, percorsi API, gestore di percorsi, mutazione dei dati, riconvalida, flusso di autenticazione, API esterna, recupero lato client, ottimizzazione del percorso, modelli Next.js.
