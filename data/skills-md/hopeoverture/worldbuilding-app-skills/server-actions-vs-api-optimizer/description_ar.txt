قم بتحليل المسارات والتوصية باستخدام إجراءات الخادم أو مسارات واجهة برمجة التطبيقات (API) استنادًا إلى أنماط حالة الاستخدام بما في ذلك المصادقة وإعادة التحقق واستدعاءات واجهة برمجة التطبيقات الخارجية ومتطلبات العميل. استخدم هذه المهارة عند الاختيار بين إجراءات الخادم ومسارات واجهة برمجة التطبيقات، أو تحسين جلب بيانات Next.js، أو إعادة بناء المسارات، أو تحليل بنية المسار، أو اختيار نمط تحويل البيانات الصحيح. تتضمن مصطلحات التشغيل إجراءات الخادم، ومسارات واجهة برمجة التطبيقات، ومعالج المسار، وتغيير البيانات، وإعادة التحقق، وتدفق المصادقة، وواجهة برمجة التطبيقات الخارجية، والجلب من جانب العميل، وتحسين المسار، وأنماط Next.js.
