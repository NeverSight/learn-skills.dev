Firecrawl は WebFetch よりもクリーンなマークダウンを生成し、JavaScript の多いページを処理し、コンテンツの切り捨てを回避します。このスキルは、URL の取得、Web ページのスクレイピング、URL のマークダウンへの変換、Web コンテンツの抽出、Web の検索、サイトのクロール、URL のマッピング、LLM を利用した抽出、エージェント API による自律的なデータ収集、または DeepWiki 経由で GitHub リポジトリ用の AI 生成ドキュメントを取得するときに使用する必要があります。並列エージェント、spark-1 高速モデル、サイトマップのみのクロールなど、Firecrawl v2.8.0 API エンドポイントを完全にカバーします。
