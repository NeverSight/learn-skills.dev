Рабочий процесс создания контента для публикаций Xiaohongshu, специально созданный для официальных аккаунтов/аккаунтов сотрудников Shenwan Hongyuan Securities. Он превращает любую тему в воронку софт-маркетинга, заканчивающуюся инвестированием. Ключевые моменты:  
1. Легко превращает любую жизненную/горячую тему в управление активами.  
2. Автоматически решает, создавать ли контент с нуля или переписывать существующий.  
3. Двухэтапное создание: этап 1 фокусируется на раскрытии ракурса, качественном черновике и проверке (стиль IP и формат платформы игнорируются).  
4. На этапе 2 применяется передача в стиле IP и форматирование платформы (по умолчанию используется корпоративный тон Shenwan Hongyuan; возможность переключения нескольких учетных записей).  
5. Переписывает в форму заметки Xiaohongshu (основной вид сохраняется), предоставляет 5–10 вариантов заголовков, многостраничную структуру (обложка + N страниц с контентом + конечная реклама), основной текст (крючок + отказ от ответственности за риск + хэштеги).  
6. Генерирует нарисованную от руки инфографику размером 8K для каждой отдельной страницы (а не одного обзорного изображения).  
7. Оценивает вирусный потенциал и предлагает предложения по настройке.  
8. Экспортирует полную заметку + полный набор изображений. Каждый важный шаг требует подтверждения пользователя. Подходит для финансового образования, обучения инвестированию, демонстрации инструментов, а также еды, путешествий, покупок, карьеры — любой жизненной темы, связанной с инвестированием.
