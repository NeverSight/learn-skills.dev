Questa competenza deve essere utilizzata quando gli utenti devono formattare, pulire, eseguire l'linting o convalidare i file Markdown utilizzando lo strumento da riga di comando markdownlint-cli2. Utilizza questa competenza per attività che coinvolgono controlli di qualità dei file Markdown (incluso Quarto Markdown `.qmd`), correzioni automatiche di formattazione, applicazione di regole di stile Markdown o identificazione di problemi di sintassi Markdown.
