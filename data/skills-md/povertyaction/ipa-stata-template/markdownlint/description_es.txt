Esta habilidad debe usarse cuando los usuarios necesitan formatear, limpiar, eliminar pelusas o validar archivos Markdown usando la herramienta de línea de comandos markdownlint-cli2. Utilice esta habilidad para tareas que impliquen comprobaciones de calidad de archivos de Markdown (incluido Quarto Markdown `.qmd`), correcciones automáticas de formato, aplicación de reglas de estilo de Markdown o identificación de problemas de sintaxis de Markdown.
