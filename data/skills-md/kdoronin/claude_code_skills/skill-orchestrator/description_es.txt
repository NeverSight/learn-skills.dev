Esta habilidad organiza tareas de programación analizando las habilidades disponibles de Claude Code y creando planes de ejecución. Debe usarse cuando se trabaja en cualquier tarea de codificación que pueda beneficiarse de múltiples habilidades especializadas. La habilidad admite dos modos de ejecución seleccionados por el usuario: manual (Claude ejecuta con referencias explícitas de habilidades) o delegado (tareas enviadas a subagentes con habilidades).
