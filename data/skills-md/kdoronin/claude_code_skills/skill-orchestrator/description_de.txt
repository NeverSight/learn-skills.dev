Diese Fähigkeit orchestriert Programmieraufgaben, indem sie verfügbare Claude-Code-Kenntnisse analysiert und Ausführungspläne erstellt. Es sollte bei der Arbeit an Codierungsaufgaben verwendet werden, die von mehreren Spezialkenntnissen profitieren könnten. Der Skill unterstützt zwei Ausführungsmodi, die vom Benutzer ausgewählt werden können: manuell (Claude führt die Ausführung mit expliziten Skill-Referenzen aus) oder delegiert (Aufgaben werden an Unteragenten mit Skills gesendet).
