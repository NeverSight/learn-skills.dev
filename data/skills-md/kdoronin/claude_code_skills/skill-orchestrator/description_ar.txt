تقوم هذه المهارة بتنسيق مهام البرمجة من خلال تحليل مهارات Claude Code المتاحة وإنشاء خطط التنفيذ. ويجب استخدامه عند العمل في أي مهمة برمجية يمكن أن تستفيد من مهارات متخصصة متعددة. تدعم المهارة وضعي تنفيذ يتم تحديدهما من قبل المستخدم - يدوي (ينفذ كلود بمراجع مهارة صريحة) أو مفوض (يتم إرسال المهام إلى الوكلاء الفرعيين ذوي المهارات).
