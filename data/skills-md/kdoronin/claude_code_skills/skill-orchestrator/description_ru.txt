Этот навык координирует задачи программирования, анализируя доступные навыки Claude Code и создавая планы выполнения. Его следует использовать при работе над любой задачей кодирования, для решения которой можно использовать несколько специализированных навыков. Навык поддерживает два режима выполнения, выбранных пользователем: вручную (Клод выполняет с явными ссылками на навыки) или делегированный (задачи отправляются субагентам с навыками).
