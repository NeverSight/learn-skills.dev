Универсальная интеграция Strava API для управления фитнес-данными. Используйте при работе с занятиями Strava, профилями спортсменов, сегментами, маршрутами, клубами или любыми данными отслеживания фитнеса. Запускается по запросам на получение/создание/обновление действий, анализ статистики тренировок, экспорт маршрутов, исследование сегментов или программное взаимодействие с данными Strava.
