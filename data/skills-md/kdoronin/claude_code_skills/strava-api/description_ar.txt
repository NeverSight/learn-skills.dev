تكامل Universal Strava API لإدارة بيانات اللياقة البدنية. يُستخدم عند العمل مع أنشطة Strava أو ملفات تعريف الرياضيين أو القطاعات أو المسارات أو الأندية أو أي بيانات لتتبع اللياقة البدنية. يتم تشغيل طلبات الحصول على/إنشاء/تحديث الأنشطة، أو تحليل إحصائيات التدريب، أو طرق التصدير، أو استكشاف الشرائح، أو التفاعل مع بيانات Strava برمجيًا.
