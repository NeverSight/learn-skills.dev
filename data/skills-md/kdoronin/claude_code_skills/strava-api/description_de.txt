Universelle Strava-API-Integration für die Fitnessdatenverwaltung. Zur Verwendung bei der Arbeit mit Strava-Aktivitäten, Athletenprofilen, Segmenten, Routen, Vereinen oder anderen Fitness-Tracking-Daten. Wird bei Anfragen zum Abrufen/Erstellen/Aktualisieren von Aktivitäten, Analysieren von Trainingsstatistiken, Exportieren von Routen, Erkunden von Segmenten oder programmgesteuerter Interaktion mit Strava-Daten ausgelöst.
