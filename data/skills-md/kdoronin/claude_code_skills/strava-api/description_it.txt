Integrazione API Strava universale per la gestione dei dati di fitness. Da utilizzare quando si lavora con attività Strava, profili di atleti, segmenti, percorsi, club o qualsiasi dato di monitoraggio del fitness. Attiva richieste per ottenere/creare/aggiornare attività, analizzare statistiche di allenamento, esportare percorsi, esplorare segmenti o interagire con i dati Strava in modo programmatico.
