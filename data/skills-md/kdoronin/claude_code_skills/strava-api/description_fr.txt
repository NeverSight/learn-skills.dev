Intégration universelle de l'API Strava pour la gestion des données de fitness. À utiliser lorsque vous travaillez avec des activités Strava, des profils d'athlètes, des segments, des itinéraires, des clubs ou toute autre donnée de suivi de la condition physique. Se déclenche sur des demandes pour obtenir/créer/mettre à jour des activités, analyser les statistiques d'entraînement, exporter des itinéraires, explorer des segments ou interagir avec les données Strava par programmation.
