Exploitez les bases, tables, champs, vues, enregistrements, requêtes de lecture SQL et flux de travail d'application/d'automatisation associés avec un processus de lecture avant écriture sécurisé. À utiliser chaque fois que l'entrée de l'utilisateur mentionne Cuppy, Teable, teable CLI ou fait référence à des ID de style Teable (bseXXX, tblXXX, fldXXX, recXXX, viw/viwXXX). Se déclenche également lorsque l'utilisateur souhaite : interroger des enregistrements, créer/mettre à jour des tables, gérer des champs, créer des tableaux de bord ou des applications Web, créer des automatisations, importer/exporter des données, générer des graphiques, exécuter des scripts dans un bac à sable, rechercher/appeler des API Teable, déclencher le remplissage de l'IA ou effectuer toute opération de base de données sur Teable - même s'ils ne disent pas explicitement "Teable" mais travaillent clairement avec une base Teable ou la CLI teable.
