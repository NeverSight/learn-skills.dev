قم بتشغيل قواعد Teaable والجداول والحقول وطرق العرض والسجلات واستعلامات قراءة SQL ومسارات عمل التطبيق/الأتمتة ذات الصلة من خلال عملية قراءة قبل الكتابة آمنة. يُستخدم عندما يذكر إدخال المستخدم Cuppy أو Teaable أو Teaable CLI أو يشير إلى معرفات نمط Teaable (bseXXX، tblXXX، fldXXX، recXXX، viw/viwXXX). يتم التشغيل أيضًا عندما يريد المستخدم: الاستعلام عن السجلات، أو إنشاء/تحديث الجداول، أو إدارة الحقول، أو إنشاء لوحات المعلومات أو تطبيقات الويب، أو إنشاء عمليات التشغيل الآلي، أو استيراد/تصدير البيانات، أو إنشاء المخططات، أو تنفيذ البرامج النصية في وضع الحماية، أو البحث/الاتصال بواجهات برمجة تطبيقات Teable، أو تشغيل تعبئة الذكاء الاصطناعي، أو تنفيذ أي عملية قاعدة بيانات على Teable - حتى لو لم يقولوا صراحة "Teable" ولكنهم يعملون بشكل واضح مع قاعدة Teable أو واجهة سطر الأوامر القابلة للتشغيل.
