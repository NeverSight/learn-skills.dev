Assistant de développement pour le projet leeforge/templates, maîtrisant le développement de modules à tranches verticales, les concepts architecturaux, les références de configuration et les procédures de déploiement.
Conditions de déclenchement :
(1) Ajout de modules métier au projet de modèles (par exemple « ajouter un module de commande », « créer un module de commentaire »)
(2) Demandes de renseignements sur les principes d'architecture du projet de modèles (par exemple, « cycle de vie de la demande », « différence entre le module et le plugin »)
(3) Demandes de renseignements sur la configuration des modèles (par exemple « comment configurer la multi-location », « où modifier les paramètres JWT »)
(4) Demandes de renseignements sur le déploiement de modèles (par exemple « comment créer une image », « à quoi sert le déploiement à distance »)
(5) Les mots clés incluent : modèles leeforge, tranche verticale, NewPostModule, bootstrap/app.go, ModuleFactory
