Guía para usar @moeru/eventa: una biblioteca de eventos compatible con el transporte que impulsa flujos de transmisión y RPC ergonómicos. Utilice esta habilidad siempre que el usuario importe desde '@moeru/eventa', mencione eventa, necesite comunicación de eventos entre procesos/procesos (Electron IPC, Web Workers, WebSocket, BroadcastChannel, EventEmitter, EventTarget, Worker Threads), quiera definir eventos de tipo seguro con patrones de invocación RPC, necesite transmisión RPC (transmisión de servidor, transmisión de cliente o bidireccional) o pregunte sobre abstracciones de eventos independientes del transporte. Úselo también cuando el usuario analice alternativas a birpc o async-call-rpc.
