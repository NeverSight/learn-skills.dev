Guía para usar xsAI: un SDK de IA extrapequeño para OpenAI y API compatibles con OpenAI. Utilice esta habilidad siempre que el usuario importe desde 'xsai' o cualquier paquete '@xsai/*', mencione xsai, necesite llamar a API compatibles con OpenAI con un tamaño de paquete mínimo, quiera generarText/streamText/generateObject/streamObject/llamada a herramientas, necesite incrustaciones, síntesis de voz, generación de imágenes o transcripción a través de OpenAI API, o pregunte sobre alternativas ligeras al Vercel AI SDK (paquete ai). Utilícelo también cuando el usuario necesite resultados estructurados con esquema estándar (Zod, Valibot, ArkType) a través de puntos finales compatibles con OpenAI.
