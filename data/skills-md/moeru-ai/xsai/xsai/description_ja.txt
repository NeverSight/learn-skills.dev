xsAI の使用ガイド — OpenAI および OpenAI 互換 API 用の超小型 AI SDK。ユーザーが「xsai」または任意の「@xsai/*」パッケージからインポートする場合、xsai について言及する場合、最小限のバンドル サイズで OpenAI 互換 API を呼び出す必要がある場合、generateText/streamText/generateObject/streamObject/tool 呼び出しを必要とする場合、OpenAI API を介した埋め込み、音声合成、画像生成、またはトランスクリプションが必要な場合、または Vercel AI SDK (ai パッケージ) の軽量代替手段について質問する場合は、必ずこのスキルを使用してください。ユーザーが OpenAI 互換エンドポイント経由で標準スキーマ (Zod、Valibot、ArkType) を使用した構造化された出力を必要とする場合にも使用します。
