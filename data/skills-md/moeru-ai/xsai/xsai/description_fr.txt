Guide d'utilisation de xsAI — un SDK IA extra-petit pour les API compatibles OpenAI et OpenAI. Utilisez cette compétence chaque fois que l'utilisateur importe depuis 'xsai' ou n'importe quel package '@xsai/*', mentionne xsai, doit appeler des API compatibles OpenAI avec une taille de bundle minimale, souhaite générer des appels generateText/streamText/generateObject/streamObject/tool, a besoin d'intégrations, de synthèse vocale, de génération d'images ou de transcription via l'API OpenAI, ou pose des questions sur des alternatives légères au SDK Vercel AI (paquet ai). À utiliser également lorsque l'utilisateur a besoin d'une sortie structurée avec un schéma standard (Zod, Valibot, ArkType) via des points de terminaison compatibles OpenAI.
