Guida per l'utilizzo di xsAI: un SDK AI extra piccolo per OpenAI e API compatibili con OpenAI. Utilizza questa competenza ogni volta che l'utente importa da "xsai" o da qualsiasi pacchetto "@xsai/*", menziona xsai, deve chiamare API compatibili con OpenAI con dimensioni minime del bundle, desidera generareText/streamText/generateObject/streamObject/tool ​​call, necessita di incorporamenti, sintesi vocale, generazione di immagini o trascrizione tramite API OpenAI o chiede alternative leggere all'SDK Vercel AI (pacchetto ai). Da utilizzare anche quando l'utente necessita di output strutturato con schema standard (Zod, Valibot, ArkType) tramite endpoint compatibili con OpenAI.
