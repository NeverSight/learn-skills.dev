Руководство по использованию xsAI — очень маленького AI SDK для OpenAI и OpenAI-совместимых API. Используйте этот навык всякий раз, когда пользователь импортирует из пакета «xsai» или любого пакета «@xsai/*», упоминает xsai, ему необходимо вызвать API-интерфейсы, совместимые с OpenAI, с минимальным размером пакета, он хочет вызвать GenerateText/streamText/generateObject/streamObject/tool, нуждается во внедрении, синтезе речи, генерации изображений или транскрипции через API OpenAI или спрашивает об облегченных альтернативах Vercel AI SDK (пакет ai). Также используйте, когда пользователю нужен структурированный вывод со стандартной схемой (Zod, Valibot, ArkType) через конечные точки, совместимые с OpenAI.
