دليل لاستخدام xsAI - مجموعة SDK صغيرة للغاية تعمل بالذكاء الاصطناعي لواجهات برمجة التطبيقات المتوافقة مع OpenAI وOpenAI. استخدم هذه المهارة عندما يقوم المستخدم بالاستيراد من حزمة 'xsai' أو أي حزمة '@xsai/*'، أو ذكر xsai، أو يحتاج إلى استدعاء واجهات برمجة التطبيقات المتوافقة مع OpenAI مع الحد الأدنى لحجم الحزمة، أو يريد إنشاء نص/streamText/generateObject/streamObject/استدعاء الأدوات، أو يحتاج إلى عمليات التضمين، أو تركيب الكلام، أو إنشاء الصور، أو النسخ عبر OpenAI API، أو يسأل عن بدائل خفيفة الوزن لحزمة Vercel AI SDK (حزمة AI). يُستخدم أيضًا عندما يحتاج المستخدم إلى مخرجات منظمة باستخدام Standard Schema (Zod وValibot وArkType) عبر نقاط النهاية المتوافقة مع OpenAI.
