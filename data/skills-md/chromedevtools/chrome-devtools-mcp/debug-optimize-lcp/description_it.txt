Guida il debug e l'ottimizzazione del Largest Contentful Paint (LCP) utilizzando gli strumenti MCP di Chrome DevTools. Utilizza questa competenza ogni volta che l'utente chiede informazioni sulle prestazioni LCP, sui caricamenti lenti delle pagine, sull'ottimizzazione dei Core Web Vitals o vuole capire perché i contenuti principali della loro pagina impiegano troppo tempo per essere visualizzati. Da utilizzare anche quando l'utente menziona "il disegno con il contenuto più grande", "velocità di caricamento della pagina", "CWV" o desidera migliorare la velocità di rendering dell'immagine hero o del contenuto principale.
