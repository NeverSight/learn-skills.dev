Guide pour écrire des messages de commit en suivant Conventional Commits. Utilisez-le lorsque vous allez réaliser un commit, créer des messages descriptifs, standardiser l'historique du projet ou automatiser les versions et les journaux de modifications. Inclut les types de commit, les portées, les modifications avec rupture, le format et les meilleures pratiques. Activer les phrases comme "faire un commit", "mensaje de commit", "conventional commits" ou "écrire un commit sémantique".
