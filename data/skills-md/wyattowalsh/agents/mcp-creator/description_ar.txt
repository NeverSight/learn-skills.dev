قم ببناء خوادم MCP جاهزة للإنتاج باستخدام FastMCP v3. يرشد البحث والسقالات والأداة/الموارد/التنفيذ الفوري والاختبار والنشر. يستهدف FastMCP 3.0.0rc2 مع الموفرين والتحويلات والبرمجيات الوسيطة وOAuth والتكوين. يُستخدم عند إنشاء خوادم MCP، أو دمج واجهات برمجة التطبيقات عبر MCP، أو تحويل مواصفات OpenAPI أو تطبيقات FastAPI، أو استكشاف مشكلات FastMCP وإصلاحها. ليس لإنشاء واجهات برمجة تطبيقات REST أو أدوات CLI أو عمليات تكامل غير تابعة لـ MCP.
