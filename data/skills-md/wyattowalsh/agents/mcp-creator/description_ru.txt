Создавайте готовые к работе серверы MCP с помощью FastMCP v3. Направляет исследования, разработку, быстрое внедрение, тестирование и развертывание инструментов/ресурсов/быстрых действий. Ориентирован на FastMCP 3.0.0rc2 с поставщиками, преобразованиями, промежуточным программным обеспечением, OAuth и композицией. Используйте при создании серверов MCP, интеграции API через MCP, преобразовании спецификаций OpenAPI или приложений FastAPI или устранении неполадок FastMCP. НЕ для создания REST API, инструментов CLI или интеграций без MCP.
