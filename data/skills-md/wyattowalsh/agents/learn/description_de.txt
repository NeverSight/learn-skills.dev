Erfassen Sie Korrekturen, Erkenntnisse und Muster als wiederverwendbares Projektwissen. Leitet Erkenntnisse an die richtige Anweisungsdatei weiter. Wendet Kaizen an: kleine Verbesserungen, Fehlervermeidung, Standardarbeit. Wird automatisch aufgerufen, wenn ein Korrekturmuster mehr als dreimal erkannt wird. Verwenden Sie es auch manuell, wenn Claude wiederholt einen Fehler macht, ein nicht offensichtliches Problem entdeckt oder wenn Sie eine Workflow-Präferenz beibehalten möchten.
