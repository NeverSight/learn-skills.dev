Crea interfacce frontend di livello produttivo con i moderni React, Tailwind CSS e shadcn/ui. Cinque modalità: progetti di scaffold, creazione di componenti, configurazione di temi e token di progettazione, stili di refactoring e basi di codici di controllo. Codifica le migliori pratiche per i componenti server React 19 (dipendenti dal framework), TailwindCSS v4 CSS-first config, shadcn/ui con primitive Radix, CSS moderni (query sui contenitori, :has(), transizioni di visualizzazione, animazioni guidate da scorrimento), tipografia Monaspace e Vite 6. Sostituisce le competenze di progettazione frontend. Da utilizzare durante la creazione, lo styling, la creazione di temi o il miglioramento di qualsiasi progetto frontend. NON per API backend, progettazione di database, DevOps, framework di test, librerie di gestione dello stato, routing o configurazione completa del framework SSR.
