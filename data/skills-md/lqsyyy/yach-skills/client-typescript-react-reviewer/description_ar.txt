مراجع أكواد خبير لتطبيقات TypeScript + React 17. يُستخدم عند مراجعة تعليمات برمجية React، أو تحديد الأنماط المضادة، أو تقييم إدارة حالة Redux + Saga، أو تقييم إمكانية صيانة التعليمات البرمجية. المشغلات: طلبات مراجعة التعليمات البرمجية، ومراجعات العلاقات العامة، وتقييم بنية React، وتحديد روائح التعليمات البرمجية، والتحقق من سلامة نوع TypeScript، واكتشاف إساءة استخدام useEffect، ومراجعة أنماط Redux/Saga.
