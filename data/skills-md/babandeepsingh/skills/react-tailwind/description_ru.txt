Упрямый помощник для установки и настройки Tailwind CSS в проектах React, а затем обновления компонентов с использованием служебных классов Tailwind и токенов семантического проектирования. Используйте при работе с React (CRA, Vite, Next.js, Remix, обычный SPA), и вам нужно руководство по настройке Tailwind, выбору зависимостей и порядку установки, настройке, дизайну семантической темы и общим рецептам пользовательского интерфейса (формы, таблицы данных, навигация, модальные окна, макет).
