Assistente supponente per l'installazione e la configurazione di Tailwind CSS nei progetti React, quindi il rinnovamento dei componenti utilizzando classi di utilità Tailwind e token di progettazione semantica. Utilizzalo quando lavori su React (CRA, Vite, Next.js, Remix, SPA semplice) e hai bisogno di indicazioni sulla configurazione di Tailwind, sulla selezione delle dipendenze e sull'ordine di installazione, sulla configurazione, sulla progettazione del tema semantico e sulle ricette comuni dell'interfaccia utente (moduli, tabelle dati, navigazione, modali, layout).
