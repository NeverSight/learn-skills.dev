Strumento di analisi del completamento delle funzionalità di Codebase. Analizza lo stato di implementazione di vari moduli funzionali in un progetto, l'esistenza di file core, l'adozione dell'architettura e genera un report di analisi JSON strutturato. Adatto per il monitoraggio dell'avanzamento del progetto, l'inventario delle funzionalità, la valutazione del debito tecnico e altri scenari.
