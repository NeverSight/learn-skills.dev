Questa competenza dovrebbe essere utilizzata ogni volta che l'utente desidera testare il comportamento di una competenza, analizzare come utilizza l'API Claude, ispezionare input/output dagli script o eseguire controlli di sicurezza e di revisione del codice rispetto agli script delle competenze. Anche per frasi casuali come "testare la mia abilità", "analizzare questa abilità", "controllare gli script delle abilità", "esaminare le abilità per problemi di sicurezza", "cosa fa effettivamente questa abilità quando viene eseguita", "ispezionare le chiamate API dall'abilità", "eseguire una competenza alla perfezione", "controllare la mia abilità per bug o vulnerabilità". Si attiva anche quando l'utente ti mostra uno SKILL.md e ti chiede di valutarlo, criticarlo o sottoporlo a stress test.
