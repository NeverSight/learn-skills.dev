Dieser Skill sollte immer dann verwendet werden, wenn der Benutzer das Verhalten eines Skills testen, analysieren möchte, wie er die Claude-API verwendet, Ein-/Ausgaben von Skripten überprüfen oder Sicherheits- und Codeüberprüfungsprüfungen für Skill-Skripte durchführen möchte. Selbst für umgangssprachliche Formulierungen wie „meinen Skill testen“, „diesen Skill analysieren“, „Skill-Skripte prüfen“, „Skill auf Sicherheitsprobleme überprüfen“, „Was macht dieser Skill tatsächlich, wenn er ausgeführt wird“, „API-Aufrufe vom Skill prüfen“, „einen Skill auf Herz und Nieren prüfen“, „meinen Skill auf Fehler oder Schwachstellen prüfen“. Wird auch ausgelöst, wenn der Benutzer Ihnen ein SKILL.md zeigt und Sie auffordert, es zu bewerten, zu kritisieren oder einem Stresstest zu unterziehen.
