Этот навык следует использовать всякий раз, когда пользователь хочет протестировать поведение навыка, проанализировать, как он использует API Claude, проверить входные/выходные данные сценариев или запустить аудит безопасности и проверки кода на основе сценариев навыков. Даже для случайных фраз, таких как «проверить мой навык», «проанализировать этот навык», «проверить сценарии навыков», «проверить навык на предмет проблем безопасности», «что на самом деле делает этот навык, когда он работает», «проверить вызовы API от навыка», «проверить навык», «проверить мой навык на наличие ошибок или уязвимостей». Также срабатывает, когда пользователь показывает вам SKILL.md и просит вас оценить, покритиковать или провести стресс-тестирование.
