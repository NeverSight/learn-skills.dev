قم بإنشاء مستند شامل لمتطلبات العمل (BRD) أو مستند المواصفات الوظيفية (FSD) من متطلبات المشروع أو PRDs أو الملخصات أو المقابلات التفاعلية. استخدم هذه المهارة عندما يذكر المستخدم BRD، أو متطلبات العمل، أو المواصفات الوظيفية، أو FSD، أو مواصفات البرنامج، أو مواصفات الميزة، أو مواصفات النظام، أو مستند المتطلبات الوظيفية، أو يريد توثيق احتياجات العمل أو كيفية تصرف النظام. يتم تشغيله أيضًا عندما يطلب المستخدمون "تحديد" ميزة ما، أو كتابة المتطلبات، أو إنشاء خطة مفصلة لأصحاب المصلحة أو المطورين - حتى لو لم يستخدموا المصطلحات الدقيقة بشكل صريح.
