Genere un Documento de requisitos comerciales (BRD) o un Documento de especificación funcional (FSD) completo a partir de los requisitos del proyecto, PRD, resúmenes o entrevistas interactivas. Utilice esta habilidad siempre que el usuario mencione BRD, requisitos comerciales, especificaciones funcionales, FSD, especificación de software, especificación de características, especificación del sistema, documento de requisitos funcionales o quiera documentar las necesidades comerciales o cómo debe comportarse un sistema. También se activa cuando los usuarios solicitan "especificar" una característica, redactar requisitos o crear un plan detallado para las partes interesadas o los desarrolladores, incluso si no usan los términos exactos explícitamente.
