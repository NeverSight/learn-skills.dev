تصميم وتحسين مخططات قاعدة البيانات على مستوى الإنتاج لقواعد بيانات SQL وNoSQL. يُستخدم عند إنشاء قواعد بيانات جديدة، أو تصميم الجداول/المجموعات، أو تعريف العلاقات، أو استراتيجيات الفهرسة، أو عمليات الترحيل، أو الإيجارات المتعددة، أو تعزيز الأمان، أو تحسين الأداء. يتعامل مع PostgreSQL وMySQL وMongoDB باستخدام لغة SQL الأصلية - بدون ORM.
