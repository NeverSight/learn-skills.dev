Progetta e ottimizza schemi di database di livello produttivo per database SQL e NoSQL. Da utilizzare durante la creazione di nuovi database, la progettazione di tabelle/raccolte, la definizione di relazioni, strategie di indicizzazione, migrazioni, multi-tenancy, rafforzamento della sicurezza o ottimizzazione delle prestazioni. Gestisce PostgreSQL, MySQL, MongoDB con SQL nativo, senza ORM.
