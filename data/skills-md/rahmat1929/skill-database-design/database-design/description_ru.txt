Проектируйте и оптимизируйте схемы баз данных производственного уровня для баз данных SQL и NoSQL. Используйте при создании новых баз данных, проектировании таблиц/коллекций, определении связей, стратегиях индексирования, миграции, мультитенантности, усилении безопасности или оптимизации производительности. Обрабатывает PostgreSQL, MySQL, MongoDB с помощью собственного SQL — без ORM.
