إرشادات الخبراء لتصميم وتنفيذ وترحيل وتصحيح استمرارية SwiftData في تطبيقات Swift وSwiftUI. يُستخدم عند العمل مع مخططات @Model، أو قواعد @Relationship/@Attribute، أو الوصول إلى بيانات الاستعلام أو FetchDescriptor، أو تكوين ModelContainer/ModelContext، أو مزامنة CloudKit، أو واجهات برمجة تطبيقات SchemaMigrationPlan/history، أو عزل التزامن ModelActor، أو البيانات الأساسية لاعتماد/تعايش SwiftData.
