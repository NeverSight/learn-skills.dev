Conseils d'experts pour la conception, la mise en œuvre, la migration et le débogage de la persistance SwiftData dans les applications Swift et SwiftUI. À utiliser lorsque vous travaillez avec des schémas @Model, des règles @Relationship/@Attribute, l'accès aux données Query ou FetchDescriptor, la configuration ModelContainer/ModelContext, la synchronisation CloudKit, les API SchemaMigrationPlan/history, l'isolation simultanée ModelActor ou l'adoption/coexistence de Core Data vers SwiftData.
