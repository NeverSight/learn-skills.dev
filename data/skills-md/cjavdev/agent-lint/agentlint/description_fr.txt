Auditez n’importe quel site Web pour la convivialité de l’IA/des agents à l’aide d’AgentLint. Exécutez `npx @cjavdev/agent-lint` avec une URL pour analyser un site selon 17 règles dans 5 catégories (transport, structure, jetons, découvrabilité, agent), obtenez un AgentScore de 0 à 100 avec une note par lettre et recevez un plan de remédiation prioritaire. Utilisez cette compétence pour : auditer un site pour l'état de préparation à l'IA, vérifier si un site prend en charge llms.txt ou markdown, améliorer le score de convivialité des agents d'un site Web, corriger les violations d'AgentLint ou comprendre ce qui rend un site convivial pour l'IA. Phrases de déclenchement : « exécuter agentlint », « auditer le site pour l'IA », « vérifier la convivialité de l'agent », « analyse agentlint », « audit convivial pour l'IA », « vérifier llms.txt », « préparation de l'agent ».
