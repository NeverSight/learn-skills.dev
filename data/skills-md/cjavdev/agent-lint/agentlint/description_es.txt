Audite cualquier sitio web en busca de compatibilidad con IA/agentes utilizando AgentLint. Ejecute `npx @cjavdev/agent-lint` con una URL para escanear un sitio a través de 17 reglas en 5 categorías (transporte, estructura, tokens, capacidad de descubrimiento, agente), obtenga un AgentScore de 0 a 100 con calificación con letras y reciba un plan de remediación priorizado. Utilice esta habilidad cuando: audite un sitio para determinar si está preparado para la IA, compruebe si un sitio tiene llms.txt o compatibilidad con rebajas, mejore la puntuación de facilidad de uso de los agentes de un sitio web, corrija infracciones de AgentLint o comprenda qué hace que un sitio sea compatible con la IA. Frases de activación: 'ejecutar agentlint', 'auditar el sitio para AI', 'verificar la compatibilidad con el agente', 'escaneo de agentlint', 'auditoría compatible con AI', "verificar llms.txt", "preparación del agente".
