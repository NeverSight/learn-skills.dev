Überprüfen Sie jede Website mit AgentLint auf KI-/Agentenfreundlichkeit. Führen Sie „npx @cjavdev/agent-lint“ mit einer URL aus, um eine Site nach 17 Regeln in 5 Kategorien (Transport, Struktur, Token, Auffindbarkeit, Agent) zu scannen, einen AgentScore von 0–100 mit Buchstabennote zu erhalten und einen priorisierten Behebungsplan zu erhalten. Verwenden Sie diese Fähigkeit, wenn Sie: eine Website auf KI-Bereitschaft prüfen, überprüfen, ob eine Website über llms.txt- oder Markdown-Unterstützung verfügt, den Agentenfreundlichkeitswert einer Website verbessern, AgentLint-Verstöße beheben oder verstehen, was eine Website KI-freundlich macht. Auslöserphrasen: „Agentlint ausführen“, „Site auf KI prüfen“, „Agentenfreundlichkeit prüfen“, „Agentlint-Scan“, „KI-freundliche Prüfung“, „llms.txt prüfen“, „Agentenbereitschaft“.
