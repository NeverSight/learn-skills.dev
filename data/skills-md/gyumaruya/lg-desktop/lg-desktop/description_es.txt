Escritorio virtual Docker con automatización GUI. Úselo cuando se le solicite interactuar con aplicaciones de escritorio, automatizar flujos de trabajo de GUI, probar la interfaz de usuario u operar software que requiera una pantalla. Inspección de texto basada en OCR en lugar de capturas de pantalla (97 % menos tokens). Subcomandos: arriba, abajo, estado, actualizar, inspeccionar, hacer clic, escribir, clave, captura de pantalla, buscar y hacer clic, esperar, afirmar, ejecutar, copiar desde, copiar a.
