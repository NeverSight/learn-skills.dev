Vollständige FDK-Verwaltung für macOS und Windows – Installation, Upgrade, Downgrade und Deinstallation. Verwenden Sie diese Option, wenn der Benutzer das Freshworks Development Kit installieren/konfigurieren/aktualisieren/downgraden/deinstallieren muss, FDK auf einem neuen Computer einrichten muss, nach der FDK-Installation auf Mac oder Windows fragt, die FDK-Version ändern möchte oder auf FDK/CLI-Setup-Probleme stößt.
