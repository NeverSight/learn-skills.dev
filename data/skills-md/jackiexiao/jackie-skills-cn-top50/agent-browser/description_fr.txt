CLI d’automatisation du navigateur d’agent AI. À utiliser lorsque l'utilisateur a besoin d'interagir avec des sites Web : parcourir des pages, remplir des formulaires, cliquer sur des boutons, prendre des captures d'écran, extraire des données, tester des applications Web ou automatiser n'importe quelle tâche du navigateur. Les déclencheurs incluent « ouvrir un site Web », « remplir un formulaire », « cliquer sur un bouton », « prendre une capture d'écran », « extraire les données de la page », « tester cette application Web », « se connecter au site », « automatiser les actions du navigateur » ou toute tâche nécessitant une interaction Web par programmation.
