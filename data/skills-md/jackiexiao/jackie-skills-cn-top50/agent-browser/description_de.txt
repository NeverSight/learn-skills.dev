KI-Agent-Browser-Automatisierungs-CLI. Verwenden Sie es, wenn der Benutzer mit Websites interagieren muss – zum Navigieren durch Seiten, Ausfüllen von Formularen, Klicken auf Schaltflächen, Erstellen von Screenshots, Extrahieren von Daten, Testen von Web-Apps oder Automatisieren einer beliebigen Browseraufgabe. Zu den Auslösern gehören „Website öffnen“, „Formular ausfüllen“, „Schaltfläche anklicken“, „Screenshot erstellen“, „Daten von Seite entfernen“, „Diese Web-App testen“, „Auf der Website anmelden“, „Browseraktionen automatisieren“ oder jede andere Aufgabe, die eine programmatische Webinteraktion erfordert.
