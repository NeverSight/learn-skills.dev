CLI di automazione del browser dell'agente AI. Da utilizzare quando l'utente deve interagire con i siti Web: navigare tra le pagine, compilare moduli, fare clic su pulsanti, acquisire schermate, estrarre dati, testare app Web o automatizzare qualsiasi attività del browser. I trigger includono "apri sito Web", "compila modulo", "fai clic sul pulsante", "acquisisci screenshot", "rimuovi dati dalla pagina", "prova questa app Web", "accedi al sito", "automatizza le azioni del browser" o qualsiasi attività che richieda un'interazione Web programmatica.
