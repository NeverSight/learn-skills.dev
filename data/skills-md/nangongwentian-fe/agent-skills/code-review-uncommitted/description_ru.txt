Выполните многомерную проверку кода незафиксированных изменений в git, включая соответствие спецификации проекта, сканирование ошибок, соответствие комментариям кода, а также проверку рациональности инкапсуляции компонентов/архитектуры, а также отфильтруйте ложные срабатывания с помощью оценки достоверности. Срабатывает, когда пользователь запрашивает просмотр незафиксированного кода, просмотр текущих изменений или использует /code-review-uncommited.
