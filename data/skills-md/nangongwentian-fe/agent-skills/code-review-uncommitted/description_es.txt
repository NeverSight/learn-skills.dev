Realice una revisión de código multidimensional de cambios no confirmados en Git, que abarque el cumplimiento de las especificaciones del proyecto, el escaneo de errores, el cumplimiento de comentarios, el diseño de arquitectura/encapsulación de componentes y filtre los falsos positivos con una puntuación de confianza. Se activa cuando el usuario solicita una revisión de código no confirmado, revisa cambios actuales o usa /code-review-uncommitted.
