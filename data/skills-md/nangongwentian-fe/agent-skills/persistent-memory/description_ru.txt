Универсальный протокол межсессионной памяти. Позволяет всем инструментам кодирования AI использовать единую систему памяти. Совместим с Claude Code/Cursor/Aider/Cline/Codex/Trae/OpenCode. Возможности: интеллектуальная категоризация/распад FSRS/ежемесячное сжатие/многоуровневый поиск. Триггеры: пользователь говорит «запомни»; спрашивает «до»; обнаруживает конфиденциальную информацию; сеанс заканчивается.
