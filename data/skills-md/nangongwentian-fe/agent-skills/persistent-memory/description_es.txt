Protocolo universal de memoria entre sesiones: permite que todas las herramientas de codificación de IA compartan un sistema de memoria unificado. Compatible con Claude Code/Cursor/Aider/Cline/Codex/Trae/OpenCode. Características: categorización inteligente/desintegración FSRS/compresión mensual/recuperación multicapa. Desencadenantes: el usuario dice "recordar"; pregunta "antes"; detecta información confidencial; finaliza la sesión.
