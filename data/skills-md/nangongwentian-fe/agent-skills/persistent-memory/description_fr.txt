Protocole de mémoire universel inter-sessions. Permet à tous les outils de codage d’IA de partager un système de mémoire unifié. Fonctionne avec Claude Code / Cursor / Aider / Cline / Codex / Trae / OpenCode. Caractéristiques : catégorisation intelligente/décroissance FSRS/compression mensuelle/récupération multicouche. Déclencheurs : l'utilisateur dit « se souvenir » ; demande « avant » ; détecte les informations sensibles ; la séance se termine.
