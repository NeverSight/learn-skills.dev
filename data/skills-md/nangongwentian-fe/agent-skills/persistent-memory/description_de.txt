Universelles Cross-Session-Memory-Protokoll. Ermöglicht allen KI-Codierungstools die gemeinsame Nutzung eines einheitlichen Speichersystems. Funktioniert mit Claude Code / Cursor / Aider / Cline / Codex / Trae / OpenCode. Funktionen: intelligente Kategorisierung / FSRS-Abfall / monatliche Komprimierung / mehrschichtiger Abruf. Auslöser: Benutzer sagt „merken“; fragt „vorher“; erkennt vertrauliche Informationen; Sitzung endet.
