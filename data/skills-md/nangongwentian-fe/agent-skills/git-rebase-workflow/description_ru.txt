Процесс синхронизации ветки Git Rebase, используемый для перебазирования текущей функциональной ветки на последнюю целевую ветку (например, главную/основную), сохраняя историю коммитов чистой. Подходит для сценариев, когда ветвь функции находится позади целевой ветки и необходимо синхронизировать последнюю версию кода.
