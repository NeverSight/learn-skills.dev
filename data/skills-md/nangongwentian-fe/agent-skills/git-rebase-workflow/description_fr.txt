Processus de synchronisation de branche Git Rebase, utilisé pour rebaser la branche de fonctionnalité actuelle sur la dernière branche cible (telle que master/main), en gardant l'historique des validations propre. Convient aux scénarios dans lesquels la branche de fonctionnalité se trouve derrière la branche cible et doit synchroniser le dernier code.
