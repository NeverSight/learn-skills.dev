Создавайте готовые диаграммы Helm для развертывания приложений Kubernetes с помощью шаблонов, управления значениями, зависимостей диаграмм, перехватчиков и тестирования. Охватывает структуру диаграммы, синтаксис шаблонов Go, дизайн значений.yaml, репозитории диаграмм, управление версиями и лучшие практики для поддерживаемых и повторно используемых диаграмм. Используйте при упаковке приложения Kubernetes для повторяемых развертываний, параметризации манифестов для нескольких сред, управлении сложными многокомпонентными приложениями с зависимостями или стандартизации методов развертывания с возможностью отката версий между командами.
