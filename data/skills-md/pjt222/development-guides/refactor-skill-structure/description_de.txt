Refaktorieren Sie eine überlange oder schlecht strukturierte SKILL.md, indem Sie Beispiele in references/EXAMPLES.md extrahieren, zusammengesetzte Prozeduren aufteilen und Abschnitte für eine progressive Offenlegung neu organisieren. Verwenden Sie diese Option, wenn ein Skill die CI-Begrenzung von 500 Zeilen überschreitet, wenn Codeblöcke den Skill-Körper dominieren, wenn ein Verfahrensschritt mehrere unabhängige Vorgänge enthält oder nachdem eine Inhaltsaktualisierung den Skill über die Zeilenbeschränkung hinausgeschoben hat.
