Configure un entorno de desarrollo de Kubernetes local utilizando kind, k3d o minikube para un desarrollo rápido de bucle interno. Cubre la creación de clústeres, la configuración de ingreso, la configuración del registro local y la integración con herramientas de desarrollo como Skaffold y Tilt para flujos de trabajo de reconstrucción y reimplementación automática. Úselo cuando necesite un entorno de Kubernetes local para el desarrollo, pruebe manifiestos o gráficos de Helm antes de la implementación de producción, desee ciclos rápidos de reconstrucción y reimplementación automática o aprenda Kubernetes sin costos de nube.
