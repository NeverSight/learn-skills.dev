قم بإعداد بيئة تطوير Kubernetes محلية باستخدام النوع أو k3d أو minikube للتطوير السريع للحلقة الداخلية. يغطي إنشاء المجموعة وتكوين الدخول وإعداد التسجيل المحلي والتكامل مع أدوات التطوير مثل Skaffold وTilt لإعادة البناء التلقائي وإعادة نشر سير العمل. استخدمه عند الحاجة إلى بيئة Kubernetes محلية للتطوير، أو اختبار البيانات أو مخططات Helm قبل نشر الإنتاج، أو الرغبة في دورات إعادة البناء وإعادة النشر التلقائية السريعة، أو تعلم Kubernetes بدون تكاليف سحابية.
