Create general-purpose Dockerfiles for Node.js, Python, Go, Rust, and Java projects. Охватывает выбор базового образа, установку зависимостей, разрешения пользователя, шаблоны копирования, сравнение ENTRYPOINT и CMD и .dockerignore. Используйте при первом контейнеризации приложения, создании согласованной среды сборки и выполнения, подготовке приложения к облачному развертыванию или Docker Compose, а также при отсутствии существующего файла Dockerfile в проекте.
