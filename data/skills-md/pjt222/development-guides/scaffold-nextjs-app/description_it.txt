Supporta una nuova applicazione Next.js con App Router, TypeScript e impostazioni predefinite moderne. Copre la creazione del progetto, la struttura delle directory, l'impostazione del routing e la configurazione iniziale. Da utilizzare quando si avvia un nuovo progetto di applicazione Web, si crea un frontend basato su React con rendering lato server, si crea un'applicazione full-stack con percorsi API o si configura un progetto Web TypeScript da zero.
