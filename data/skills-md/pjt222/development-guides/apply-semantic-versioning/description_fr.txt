Appliquez le versioning sémantique (SemVer 2.0.0) pour déterminer la version correcte en fonction de l'analyse des modifications. Couvre la classification majeur/mineur/correctif, les identifiants de pré-version, les métadonnées de construction et la détection des modifications avec rupture. À utiliser lors de la préparation d'une nouvelle version pour déterminer le numéro de version correct, après avoir fusionné les modifications avant le marquage, pour évaluer si une modification constitue une modification avec rupture, pour ajouter des identifiants de version préliminaire ou pour résoudre un désaccord sur le changement de version approprié.
