Exploración intuitiva de IA para abordar bases de código, problemas o sistemas desconocidos sin ideas preconcebidas. Adapta el protocolo Coordinate Remote Viewing a la investigación de IA: enfriamiento (suposiciones claras), recopilación de datos por etapas (señales sin procesar → dimensionales → analíticas), gestión de AOL (separación de observaciones de etiquetas prematuras) y revisión estructurada. Úselo cuando se investiga una base de código desconocida con una arquitectura desconocida, se depura un problema donde las hipótesis prematuras podrían inducir a error, se explora un dominio con contexto limitado o cuando los intentos anteriores se han desviado por suposiciones y la "mente de principiante" sería más productiva.
