Überprüfen Sie die Softwarearchitektur auf Kopplung, Kohäsion, SOLID-Prinzipien, API-Design, Skalierbarkeit und technische Schulden. Umfasst die Bewertung auf Systemebene, die Überprüfung von Architekturentscheidungsdatensätzen und Verbesserungsempfehlungen. Verwenden Sie es, wenn Sie eine vorgeschlagene Architektur vor der Implementierung bewerten, ein vorhandenes System auf Skalierbarkeit oder Sicherheit bewerten, ADRs überprüfen, eine technische Schuldenbewertung durchführen oder die Bereitschaft für eine erhebliche Skalierung bewerten.
