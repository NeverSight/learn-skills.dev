Analysez une base de code arbitraire pour identifier les fonctions, les API et les sources de données pouvant être exposées en tant qu'outils MCP, en produisant un document de spécification d'outil. À utiliser lors de la planification d'un serveur MCP pour un projet existant, de l'audit d'une base de code avant de l'encapsuler en tant que surface d'outil accessible à l'IA, de comparaison de ce qu'une base de code peut faire par rapport à ce qui est déjà exposé via MCP ou de génération d'une spécification d'outil à transmettre au scaffold-mcp-server.
