Analizzare una base di codice arbitraria per identificare funzioni, API e origini dati adatte all'esposizione come strumenti MCP, producendo un documento di specifiche dello strumento. Da utilizzare quando si pianifica un server MCP per un progetto esistente, si controlla una base di codice prima di inserirla come superficie di uno strumento accessibile dall'intelligenza artificiale, si confronta ciò che una base di codice può fare rispetto a ciò che è già esposto tramite MCP o si genera una specifica dello strumento da trasferire a scaffold-mcp-server.
