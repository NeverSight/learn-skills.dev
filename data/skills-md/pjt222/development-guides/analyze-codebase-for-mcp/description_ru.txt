Проанализируйте произвольную базу кода, чтобы определить функции, API и источники данных, подходящие для использования в качестве инструментов MCP, и создайте документ со спецификацией инструмента. Используйте при планировании сервера MCP для существующего проекта, аудите базы кода перед ее обертыванием в качестве поверхности инструмента, доступной для искусственного интеллекта, сравнении возможностей базы кода с тем, что уже доступно через MCP, или при создании спецификации инструмента для передачи на scaffold-mcp-server.
