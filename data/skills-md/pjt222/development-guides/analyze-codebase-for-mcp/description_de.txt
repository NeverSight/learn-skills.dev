Analysieren Sie eine beliebige Codebasis, um Funktionen, APIs und Datenquellen zu identifizieren, die für die Bereitstellung als MCP-Tools geeignet sind, und erstellen Sie ein Tool-Spezifikationsdokument. Verwenden Sie diese Option, wenn Sie einen MCP-Server für ein bestehendes Projekt planen, eine Codebasis prüfen, bevor Sie sie als KI-zugängliche Tool-Oberfläche umschließen, vergleichen, was eine Codebasis kann, mit dem, was bereits über MCP verfügbar gemacht wird, oder eine Tool-Spezifikation erstellen, die an scaffold-mcp-server übergeben werden soll.
