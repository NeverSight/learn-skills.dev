Claude Code および Claude Desktop 用に MCP (Model Context Protocol) サーバーを構成します。 mcptools のセットアップ、Hugging Face の統合、WSL パスの処理、およびマルチクライアント構成について説明します。 mcptools 経由で R に接続するように Claude Code を設定する場合、MCP サーバーを使用して Claude Desktop を構成する場合、Hugging Face またはその他のリモート MCP サーバーを追加する場合、またはクライアントとサーバー間の MCP 接続のトラブルシューティングを行う場合に使用します。
