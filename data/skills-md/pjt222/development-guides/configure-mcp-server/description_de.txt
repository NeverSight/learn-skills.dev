Konfigurieren Sie MCP-Server (Model Context Protocol) für Claude Code und Claude Desktop. Behandelt die mcptools-Einrichtung, die Hugging Face-Integration, die WSL-Pfadbehandlung und die Multi-Client-Konfiguration. Verwenden Sie diese Option, wenn Sie Claude Code einrichten, um über mcptools eine Verbindung zu R herzustellen, Claude Desktop mit MCP-Servern konfigurieren, Hugging Face oder andere Remote-MCP-Server hinzufügen oder Probleme bei der MCP-Konnektivität zwischen Clients und Servern beheben.
