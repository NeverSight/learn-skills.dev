Configurez les serveurs MCP (Model Context Protocol) pour Claude Code et Claude Desktop. Couvre la configuration de mcptools, l'intégration de Hugging Face, la gestion des chemins WSL et la configuration multi-client. À utiliser lors de la configuration de Claude Code pour se connecter à R via mcptools, de la configuration de Claude Desktop avec des serveurs MCP, de l'ajout de Hugging Face ou d'autres serveurs MCP distants, ou du dépannage de la connectivité MCP entre les clients et les serveurs.
