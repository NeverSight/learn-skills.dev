قم بتكوين خوادم MCP (بروتوكول السياق النموذجي) لـ Claude Code وClaude Desktop. يغطي إعداد mcptools، وتكامل Hugging Face، ومعالجة مسار WSL، والتكوين متعدد العملاء. استخدمه عند إعداد Claude Code للاتصال بـ R عبر mcptools، أو تكوين Claude Desktop مع خوادم MCP، أو إضافة Hugging Face أو خوادم MCP بعيدة أخرى، أو استكشاف أخطاء اتصال MCP وإصلاحها بين العملاء والخوادم.
