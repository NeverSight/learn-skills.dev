Configurez GitHub Actions CI/CD pour régénérer automatiquement les diagrammes de flux de travail putior lors du push. Couvre la création de flux de travail YAML, le script R pour la génération de diagrammes avec des marqueurs sentinelles, la validation automatique des diagrammes mis à jour et l'intégration de sentinelle README pour les mises à jour de diagrammes sur place. À utiliser lorsque les diagrammes de workflow doivent toujours refléter l'état actuel du code, lorsque plusieurs contributeurs peuvent modifier le code affectant le workflow, ou lors du remplacement de la régénération manuelle des diagrammes par un pipeline CI/CD automatisé.
