Настройте GitHub Actions CI/CD для автоматического восстановления диаграмм рабочих процессов при отправке. Охватывает создание рабочего процесса YAML, сценарий R для создания диаграмм с контрольными маркерами, автоматическую фиксацию обновленных диаграмм и интеграцию дозорных README для обновлений диаграмм на месте. Используйте, когда диаграммы рабочих процессов всегда должны отражать текущее состояние кода, когда несколько участников могут изменить код, влияющий на рабочий процесс, или при замене ручной регенерации диаграммы автоматизированным конвейером CI/CD.
