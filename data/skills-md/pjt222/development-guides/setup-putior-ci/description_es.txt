Configure GitHub Actions CI/CD para regenerar automáticamente diagramas de flujo de trabajo superiores al presionar. Cubre la creación de flujos de trabajo YAML, el script R para la generación de diagramas con marcadores centinela, la confirmación automática de diagramas actualizados y la integración de README centinela para actualizaciones de diagramas in situ. Úselo cuando los diagramas de flujo de trabajo siempre deben reflejar el estado actual del código, cuando varios contribuyentes pueden cambiar el código que afecta el flujo de trabajo o cuando se reemplaza la regeneración manual de diagramas con una canalización de CI/CD automatizada.
