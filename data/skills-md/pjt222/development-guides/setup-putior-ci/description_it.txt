Configura GitHub Actions CI/CD per rigenerare automaticamente i diagrammi del flusso di lavoro putior durante il push. Copre la creazione YAML del flusso di lavoro, lo script R per la generazione di diagrammi con marcatori sentinella, il commit automatico dei diagrammi aggiornati e l'integrazione sentinella README per gli aggiornamenti dei diagrammi sul posto. Da utilizzare quando i diagrammi del flusso di lavoro devono sempre riflettere lo stato corrente del codice, quando più contributori possono modificare il codice che influisce sul flusso di lavoro o quando si sostituisce la rigenerazione manuale del diagramma con una pipeline CI/CD automatizzata.
