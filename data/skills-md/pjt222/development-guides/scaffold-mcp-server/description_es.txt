Cree un nuevo servidor MCP a partir de las especificaciones de la herramienta utilizando el SDK oficial (TypeScript o Python), incluida la configuración de transporte, los controladores de herramientas y el arnés de prueba. Úselo cuando tenga una especificación de herramienta y necesite un servidor que funcione, cuando inicie un nuevo proyecto de servidor MCP y desee una estructura correcta desde el principio, cuando migre una integración de herramienta existente al protocolo MCP o cuando cree un prototipo de una superficie de herramienta para probar con Claude Code antes de la implementación completa.
