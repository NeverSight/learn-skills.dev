قم ببناء خادم MCP جديد من مواصفات الأداة باستخدام SDK الرسمي (TypeScript أو Python)، بما في ذلك تكوين النقل ومعالجات الأدوات وأدوات الاختبار. استخدمه عندما تكون لديك مواصفات أداة وتحتاج إلى خادم عامل، أو عند بدء مشروع خادم MCP جديد وتريد البنية الصحيحة من البداية، أو عند ترحيل تكامل أداة حالية إلى بروتوكول MCP، أو عند إنشاء نموذج أولي لسطح أداة لاختباره باستخدام Claude Code قبل التنفيذ الكامل.
