Élaborez un nouveau serveur MCP à partir des spécifications d'outils à l'aide du SDK officiel (TypeScript ou Python), y compris la configuration du transport, les gestionnaires d'outils et le harnais de test. À utiliser lorsque vous disposez d'une spécification d'outil et avez besoin d'un serveur fonctionnel, lorsque vous démarrez un nouveau projet de serveur MCP et souhaitez une structure correcte dès le départ, lors de la migration d'une intégration d'outil existante vers le protocole MCP, ou lors du prototypage d'une surface d'outil à tester avec Claude Code avant la mise en œuvre complète.
