Impalcatura di un nuovo server MCP dalle specifiche dello strumento utilizzando l'SDK ufficiale (TypeScript o Python), inclusa la configurazione del trasporto, i gestori degli strumenti e il test cablaggio. Da utilizzare quando si dispone di una specifica dello strumento e si necessita di un server funzionante, quando si avvia un nuovo progetto di server MCP e si desidera una struttura corretta fin dall'inizio, quando si esegue la migrazione di un'integrazione di uno strumento esistente al protocollo MCP o quando si prototipa una superficie di uno strumento da testare con Claude Code prima dell'implementazione completa.
