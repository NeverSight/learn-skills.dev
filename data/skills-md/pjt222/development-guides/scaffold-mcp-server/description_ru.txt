Создайте новый сервер MCP на основе спецификаций инструментов с помощью официального SDK (TypeScript или Python), включая конфигурацию транспорта, обработчики инструментов и средства тестирования. Используйте его, когда у вас есть спецификация инструмента и вам нужен работающий сервер, при запуске нового проекта сервера MCP и вам нужна правильная структура с самого начала, при переносе существующей интеграции инструмента в протокол MCP или при создании прототипа поверхности инструмента для тестирования с помощью Claude Code перед полной реализацией.
