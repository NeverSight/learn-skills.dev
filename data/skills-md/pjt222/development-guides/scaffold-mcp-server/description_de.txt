Erstellen Sie mithilfe des offiziellen SDK (TypeScript oder Python) einen neuen MCP-Server aus Tool-Spezifikationen, einschließlich Transportkonfiguration, Tool-Handlern und Testumgebung. Verwenden Sie diese Option, wenn Sie über eine Tool-Spezifikation verfügen und einen funktionierenden Server benötigen, wenn Sie ein neues MCP-Serverprojekt starten und von Anfang an eine korrekte Struktur wünschen, wenn Sie eine vorhandene Tool-Integration auf das MCP-Protokoll migrieren oder wenn Sie einen Prototyp einer Tool-Oberfläche erstellen, um diese vor der vollständigen Implementierung mit Claude Code zu testen.
