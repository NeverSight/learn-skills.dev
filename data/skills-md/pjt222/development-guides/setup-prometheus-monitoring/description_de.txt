Konfigurieren Sie Prometheus für die Erfassung von Zeitreihenmetriken, einschließlich Scrape-Konfigurationen, Serviceerkennung, Aufzeichnungsregeln und Föderationsmuster für Multi-Cluster-Bereitstellungen. Verwenden Sie es, wenn Sie eine zentralisierte Metrikerfassung für Microservices einrichten, Zeitreihenüberwachung für Anwendungen und Infrastruktur implementieren, eine Grundlage für SLO/SLI-Verfolgung und -Warnung schaffen oder von älteren Überwachungslösungen auf einen modernen Observability-Stack migrieren.
