Создайте новый файл состава группы, следуя шаблону группы агента-альманаха и соглашениям реестра. Охватывает определение целей команды, выбор участников, выбор схемы координации, проект декомпозиции задач, машиночитаемый блок конфигурации, интеграцию реестра и автоматизацию README. Используйте при определении рабочего процесса с несколькими агентами, составлении агентов для сложного процесса проверки или создании скоординированной группы для повторяющихся задач совместной работы.
