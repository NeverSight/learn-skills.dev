Überprüfen Sie eine SKILL.md-Datei auf Konformität mit dem Agentskills.io-Standard. Überprüft YAML-Frontmatter-Felder, erforderliche Abschnitte, Zeilenanzahlbeschränkungen, Verfahrensschrittformat und Registrierungssynchronisierung. Verwenden Sie diese Option, wenn ein neuer Skill vor der Zusammenführung eine Formatvalidierung erfordert, ein vorhandener Skill geändert wurde und eine erneute Validierung erfordert, eine Stapelprüfung aller Skills in einer Domäne durchgeführt wird oder die Skill-Übermittlung eines Mitwirkenden in einer Pull-Anfrage überprüft wird.
