Внедрите непрерывную доставку GitOps с помощью Argo CD или Flux с шаблоном «приложения из приложений», политиками автоматической синхронизации, обнаружением отклонений и продвижением в нескольких средах. Управляйте развертываниями Kubernetes декларативно из Git с помощью автоматической сверки. Используйте при реализации декларативного управления инфраструктурой, переходе от императивных команд kubectl к развертываниям на основе Git, настройке рабочих процессов продвижения в нескольких средах, принудительном внедрении шлюзов проверки кода для производства или выполнении требований аудита и соответствия требованиям.
