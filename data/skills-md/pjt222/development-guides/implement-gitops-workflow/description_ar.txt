قم بتنفيذ التسليم المستمر لـ GitOps باستخدام Argo CD أو Flux مع نمط تطبيق التطبيقات وسياسات المزامنة التلقائية واكتشاف الانجراف والترويج للبيئات المتعددة. قم بإدارة عمليات نشر Kubernetes بشكل تصريحي من Git باستخدام التسوية التلقائية. يُستخدم عند تنفيذ إدارة البنية التحتية التعريفية، أو الترحيل من أوامر kubectl الحتمية إلى عمليات النشر المستندة إلى Git، أو إعداد سير عمل تعزيز البيئات المتعددة، أو فرض بوابات مراجعة التعليمات البرمجية للإنتاج، أو تلبية متطلبات التدقيق والامتثال.
