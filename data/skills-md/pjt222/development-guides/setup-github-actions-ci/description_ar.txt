قم بتكوين GitHub Actions CI/CD لحزم R بما في ذلك فحص R CMD على منصات متعددة، وتقارير تغطية الاختبار، ونشر موقع pkgdown. يستخدم r-lib/actions لسير العمل القياسي. يُستخدم عند إعداد CI/CD لحزمة R جديدة، أو إضافة اختبار متعدد الأنظمة الأساسية إلى حزمة موجودة، أو تكوين نشر موقع pkgdown الآلي، أو إضافة تقارير تغطية التعليمات البرمجية إلى مستودع.
