Konfigurieren Sie GitHub Actions CI/CD für R-Pakete, einschließlich R-CMD-Prüfung auf mehreren Plattformen, Berichten zur Testabdeckung und Bereitstellung der pkgdown-Site. Verwendet r-lib/actions für Standard-Workflows. Verwenden Sie diese Option, wenn Sie CI/CD für ein neues R-Paket einrichten, plattformübergreifende Tests zu einem vorhandenen Paket hinzufügen, die automatisierte Bereitstellung von pkgdown-Sites konfigurieren oder Codeabdeckungsberichte zu einem Repository hinzufügen.
