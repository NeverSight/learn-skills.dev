Configurez les packages GitHub Actions CI/CD pour R, y compris la vérification R CMD sur plusieurs plates-formes, les rapports de couverture de test et le déploiement du site pkgdown. Utilise r-lib/actions pour les flux de travail standard. À utiliser lors de la configuration de CI/CD pour un nouveau package R, de l'ajout de tests multiplateformes à un package existant, de la configuration du déploiement automatisé du site pkgdown ou de l'ajout de rapports de couverture de code à un référentiel.
