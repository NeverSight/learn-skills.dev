Configure GitHub Actions CI/CD para paquetes de R, incluida la verificación de R CMD en múltiples plataformas, informes de cobertura de prueba e implementación de sitios pkgdown. Utiliza r-lib/actions para flujos de trabajo estándar. Úselo al configurar CI/CD para un nuevo paquete R, agregar pruebas multiplataforma a un paquete existente, configurar la implementación automatizada del sitio pkgdown o agregar informes de cobertura de código a un repositorio.
