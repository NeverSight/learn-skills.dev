Настройте GitHub Actions CI/CD для пакетов R, включая проверку R CMD на нескольких платформах, отчеты о покрытии тестов и развертывание сайта pkgdown. Использует r-lib/actions для стандартных рабочих процессов. Используйте при настройке CI/CD для нового пакета R, добавлении многоплатформенного тестирования в существующий пакет, настройке автоматического развертывания сайта pkgdown или добавлении отчетов о покрытии кода в репозиторий.
