Configura CI/CD di GitHub Actions per i pacchetti R, incluso il controllo CMD R su più piattaforme, report sulla copertura dei test e distribuzione del sito pkgdown. Utilizza r-lib/actions per flussi di lavoro standard. Da utilizzare durante la configurazione di CI/CD per un nuovo pacchetto R, l'aggiunta di test multipiattaforma a un pacchetto esistente, la configurazione della distribuzione automatizzata del sito pkgdown o l'aggiunta di report sulla copertura del codice a un repository.
