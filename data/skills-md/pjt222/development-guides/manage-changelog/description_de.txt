Führen Sie ein Änderungsprotokoll gemäß dem Format „Änderungsprotokoll beibehalten“. Deckt die Kategorisierung von Einträgen (Hinzugefügt, Geändert, Veraltet, Entfernt, Behoben, Sicherheit), die Verwaltung von Versionsabschnitten und die Nachverfolgung unveröffentlichter Einträge ab. Verwenden Sie diese Option, wenn Sie ein neues Projekt starten, das ein Änderungsprotokoll benötigt, Einträge hinzufügen, nachdem Funktionen oder Korrekturen abgeschlossen wurden, eine Veröffentlichung vorbereiten, indem Sie unveröffentlichte Einträge in einen versionierten Abschnitt hochstufen, oder ein Freiform-Änderungsprotokoll in das Format „Änderungsprotokoll beibehalten“ konvertieren.
