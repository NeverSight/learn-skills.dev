Controlla le dipendenze del progetto per individuare eventuali versioni obsolete, vulnerabilità della sicurezza e problemi di compatibilità. Copre l'analisi dei file di blocco, la pianificazione del percorso di aggiornamento e la valutazione delle modifiche di rilievo. Da utilizzare prima di un rilascio per garantire che le dipendenze siano aggiornate e sicure, durante le revisioni periodiche di manutenzione, dopo aver ricevuto un avviso di sicurezza, quando si esegue l'aggiornamento a una nuova versione linguistica, prima dell'invio a CRAN o npm o quando si eredita un progetto per valutarne l'integrità delle dipendenze.
