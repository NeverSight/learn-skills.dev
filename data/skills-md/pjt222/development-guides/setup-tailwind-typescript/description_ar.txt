قم بتكوين Tailwind CSS باستخدام TypeScript في مشروع Next.js أو React. يغطي التثبيت والتكوين وملحقات السمات المخصصة وأنماط المكونات وأدوات التصميم الآمنة للنوع. يُستخدم عند إضافة Tailwind CSS إلى مشروع TypeScript موجود، أو تخصيص سمة Tailwind لنظام تصميم المشروع، أو إعداد أنماط تصميم المكونات الآمنة للنوع، أو تكوين مكونات وإضافات Tailwind الإضافية.
