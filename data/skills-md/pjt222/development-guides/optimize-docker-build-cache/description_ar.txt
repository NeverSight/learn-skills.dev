قم بتحسين أوقات إنشاء Docker باستخدام التخزين المؤقت للطبقة، والإنشاءات متعددة المراحل، وميزات BuildKit، وأنماط النسخ التبعية أولاً. ينطبق على مشاريع R وNode.js وPython. يُستخدم عندما تكون إصدارات Docker بطيئة بسبب عمليات تثبيت الحزم المتكررة، أو عندما تقوم عمليات إعادة البناء بإعادة تثبيت جميع التبعيات عند كل تغيير في التعليمات البرمجية، أو عندما تكون أحجام الصور كبيرة بشكل غير ضروري، أو عندما تكون عمليات إنشاء خط أنابيب CI/CD بمثابة عنق الزجاجة.
