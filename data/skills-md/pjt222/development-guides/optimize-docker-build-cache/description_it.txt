Ottimizza i tempi di creazione di Docker utilizzando la memorizzazione nella cache dei livelli, build in più fasi, funzionalità BuildKit e modelli di copia con prima dipendenza. Applicabile ai progetti R, Node.js e Python. Da utilizzare quando le build Docker sono lente a causa di installazioni ripetute di pacchetti, quando le ricostruzioni reinstallano tutte le dipendenze a ogni modifica del codice, quando le dimensioni delle immagini sono inutilmente grandi o quando le build della pipeline CI/CD rappresentano un collo di bottiglia.
