قم بتكوين سجلات صور الحاوية بما في ذلك GitHub Container Registry (ghcr.io) وDocker Hub وHarbour باستخدام المسح التلقائي للصور واستراتيجيات وضع العلامات وسياسات الاحتفاظ وتكامل CI/CD لتوزيع الصور بشكل آمن. يُستخدم عند إعداد سجل حاوية خاص، أو الترحيل من Docker Hub إلى السجلات المستضافة ذاتيًا، أو تنفيذ فحص الثغرات الأمنية في خطوط أنابيب CI/CD، أو إدارة الصور متعددة البنى التحتية، أو فرض توقيع الصور، أو تكوين سياسات التنظيف والاحتفاظ التلقائية.
