Configure registros de imágenes de contenedores, incluidos GitHub Container Registry (ghcr.io), Docker Hub y Harbor, con escaneo de imágenes automatizado, estrategias de etiquetado, políticas de retención e integración de CI/CD para una distribución segura de imágenes. Úselo al configurar un registro de contenedor privado, migrar de Docker Hub a registros autohospedados, implementar escaneo de vulnerabilidades en canalizaciones de CI/CD, administrar imágenes de múltiples arquitecturas, aplicar la firma de imágenes o configurar políticas automáticas de limpieza y retención.
