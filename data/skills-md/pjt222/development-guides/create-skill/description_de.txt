Erstellen Sie eine neue SKILL.md-Datei gemäß dem offenen Agent Skills-Standard (agentskills.io). Behandelt das Frontmatter-Schema, die Abschnittsstruktur, das Schreiben effektiver Verfahren mit erwarteten/bei Fehlerpaaren, Validierungschecklisten, Querverweise und Registrierungsintegration. Verwenden Sie es, wenn Sie ein wiederholbares Verfahren für Agenten kodifizieren, der Kompetenzbibliothek eine neue Funktion hinzufügen, einen Leitfaden oder ein Runbook in ein für Agenten nutzbares Format konvertieren oder einen Workflow über Projekte oder Teams hinweg standardisieren.
