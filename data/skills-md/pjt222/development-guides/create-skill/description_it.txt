Crea un nuovo file SKILL.md seguendo lo standard aperto Agent Skills (agentskills.io). Copre lo schema del frontmatter, la struttura delle sezioni, la scrittura di procedure efficaci con coppie di errori previsti/on, elenchi di controllo di convalida, riferimenti incrociati e integrazione del registro. Da utilizzare durante la codifica di una procedura ripetibile per gli agenti, l'aggiunta di una nuova funzionalità alla libreria delle competenze, la conversione di una guida o un runbook in un formato utilizzabile dagli agenti o la standardizzazione di un flusso di lavoro tra progetti o team.
