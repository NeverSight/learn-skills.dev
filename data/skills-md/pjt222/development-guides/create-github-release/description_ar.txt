قم بإنشاء إصدار GitHub باستخدام العلامات المناسبة وملاحظات الإصدار وعناصر البناء الاختيارية. يغطي الإصدارات الدلالية وإنشاء سجل التغيير واستخدام GitHub CLI. يُستخدم عند وضع علامة على إصدار ثابت من البرنامج للتوزيع، أو نشر إصدار مكتبة أو تطبيق جديد، أو إنشاء ملاحظات الإصدار لأصحاب المصلحة، أو توزيع عناصر البناء (الثنائيات، وكرات القطران).
