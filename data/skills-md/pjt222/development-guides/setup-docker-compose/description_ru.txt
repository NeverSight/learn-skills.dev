Настройте Docker Compose для многоконтейнерных сред разработки R. Охватывает определения служб, монтирование томов, сети, переменные среды, а также конфигурации разработки и производства. Используйте при запуске R вместе с другими службами (базами данных, API), настройке воспроизводимой среды разработки R, оркестрации серверного контейнера MCP на основе R или управлении переменными среды и монтированием томов для проектов R.
