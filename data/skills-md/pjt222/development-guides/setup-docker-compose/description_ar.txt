قم بتكوين Docker Compose لبيئات تطوير R متعددة الحاويات. يغطي تعريفات الخدمة، ووحدات التخزين، والشبكات، ومتغيرات البيئة، وتكوينات التطوير مقابل الإنتاج. يُستخدم عند تشغيل R جنبًا إلى جنب مع خدمات أخرى (قواعد البيانات وواجهات برمجة التطبيقات)، أو إعداد بيئة تطوير R قابلة للتكرار، أو تنسيق حاوية خادم MCP المستندة إلى R، أو إدارة متغيرات البيئة ووحدات التخزين لمشروعات R.
