Extrayez la valeur maximale du code existant : optimisation des performances, affinement de la surface de l'API et élimination des poids morts. L’art de transformer le code de base en or grâce à l’identification et à l’amplification systématiques de modèles porteurs de valeur. À utiliser pour optimiser une base de code fonctionnelle mais lente, affiner une surface d'API qui a accumulé des éléments inutiles, réduire la taille du bundle ou l'empreinte mémoire, ou préparer le code pour une version open source - lorsque le code fonctionne correctement mais ne brille pas et nécessite un peaufinage plutôt qu'une réécriture complète.
