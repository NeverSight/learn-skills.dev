Estrai il massimo valore dal codice esistente: ottimizzazione delle prestazioni, perfezionamento della superficie API ed eliminazione dei pesi morti. L'arte di trasformare il codice di base in oro attraverso l'identificazione sistematica e l'amplificazione di modelli portatori di valore. Da utilizzare quando si ottimizza una base di codice funzionante ma lenta, si perfeziona una superficie API che ha accumulato residui, si riducono le dimensioni del bundle o l'ingombro della memoria o si prepara il codice per il rilascio open source, quando il codice funziona correttamente ma non brilla e necessita di miglioramenti anziché di una riscrittura completa.
