قم بنشر وتكوين بوابة API (Kong أو Traefik) للتعامل مع إدارة حركة مرور API والمصادقة وتحديد المعدل وتحويل الطلب/الاستجابة والتوجيه. يغطي تكوين البرنامج المساعد، والخدمات الأولية، وإدارة المستهلك، والتكامل مع البنية التحتية الحالية. يُستخدم عندما تحتاج خدمات خلفية متعددة إلى نقطة نهاية موحدة لواجهة برمجة التطبيقات (API)، أو عندما تكون المصادقة المركزية أو تحديد المعدل مطلوبة، أو عند تنفيذ إصدار واجهة برمجة التطبيقات (API)، أو عند الحاجة إلى تحليلات تفصيلية وموازنة التحميل للخدمات الصغيرة.
