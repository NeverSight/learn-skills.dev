Distribuisci e configura un gateway API (Kong o Traefik) per gestire la gestione del traffico API, l'autenticazione, la limitazione della velocità, la trasformazione di richiesta/risposta e il routing. Copre la configurazione dei plug-in, i servizi upstream, la gestione dei consumatori e l'integrazione con l'infrastruttura esistente. Da utilizzare quando più servizi backend necessitano di un endpoint API unificato, quando è richiesta l'autenticazione centralizzata o la limitazione della velocità, quando si implementa il controllo delle versioni API o quando sono necessari analisi dettagliate e bilanciamento del carico per i microservizi.
