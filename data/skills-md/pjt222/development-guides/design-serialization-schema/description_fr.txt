Concevez des schémas de sérialisation à l'aide de JSON Schema, des définitions de Protocol Buffer ou d'Apache Avro. Couvre la gestion des versions de schéma, la compatibilité ascendante, les règles de validation et les stratégies d'évolution pour les formats de données de longue durée. À utiliser lors de la définition d'un nouveau contrat API ou d'un nouveau format d'échange de données, de l'ajout de champs à un schéma existant sans interrompre les consommateurs, de la migration entre les versions de schéma, du choix entre des systèmes de schéma ou de la documentation des règles de validation des données pour une application automatisée.
