إنشاء وإدارة طلبات السحب باستخدام GitHub CLI. يغطي إعداد الفرع، وكتابة عناوين وأوصاف العلاقات العامة، وإنشاء علاقات عامة، والتعامل مع تعليقات المراجعة، وسير عمل الدمج/التنظيف. يُستخدم عند اقتراح تغييرات من ميزة أو فرع إصلاح للمراجعة، أو دمج العمل المكتمل في الفرع الرئيسي، أو طلب مراجعة التعليمات البرمجية من المتعاونين، أو توثيق غرض ونطاق مجموعة من التغييرات.
