Создавайте запросы на включение и управляйте ими с помощью GitHub CLI. Охватывает подготовку филиалов, написание заголовков и описаний PR, создание PR, обработку отзывов по результатам проверки и рабочие процессы слияния/очистки. Используйте, когда предлагаете изменения из ветки функции или исправления на рассмотрение, объединяете завершенную работу с основной веткой, запрашиваете проверку кода у соавторов или документируете цель и объем набора изменений.
