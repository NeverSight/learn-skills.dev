Cree y administre solicitudes de extracción utilizando GitHub CLI. Cubre la preparación de sucursales, la redacción de títulos y descripciones de relaciones públicas, la creación de relaciones públicas, el manejo de comentarios de revisión y los flujos de trabajo de fusión/limpieza. Úselo al proponer cambios de una característica o rama de corrección para su revisión, fusionar trabajo completado en la rama principal, solicitar revisión de código a colaboradores o documentar el propósito y alcance de un conjunto de cambios.
