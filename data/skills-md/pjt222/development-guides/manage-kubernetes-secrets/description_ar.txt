تنفيذ إدارة الأسرار الآمنة في Kubernetes باستخدام SealedSecrets لـ GitOps، ومشغل الأسرار الخارجية لمديري الأسرار السحابية، واستراتيجيات التناوب. يمكنك التعامل مع شهادات TLS ومفاتيح API وبيانات الاعتماد باستخدام التشفير أثناء الراحة وعناصر تحكم RBAC. يُستخدم عند تخزين التكوين الحساس لتطبيقات Kubernetes، أو تنفيذ GitOps حيث يجب التحكم في الأسرار في الإصدار، أو التكامل مع AWS Secrets Manager أو Azure Key Vault، أو تدوير بيانات الاعتماد دون توقف، أو الترحيل من أسرار النص العادي إلى الحلول المشفرة.
