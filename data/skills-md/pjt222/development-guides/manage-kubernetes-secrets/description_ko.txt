GitOps용 SealedSecrets, 클라우드 비밀 관리자용 외부 비밀 연산자 및 순환 전략을 사용하여 Kubernetes에서 보안 비밀 관리를 구현합니다. 미사용 암호화 및 RBAC 제어를 통해 TLS 인증서, API 키 및 자격 증명을 처리합니다. Kubernetes 애플리케이션에 대한 중요한 구성을 저장하고, 비밀을 버전 관리해야 하는 GitOps를 구현하고, AWS Secrets Manager 또는 Azure Key Vault와 통합하고, 가동 중지 시간 없이 자격 증명을 교체하거나 일반 텍스트 비밀에서 암호화된 솔루션으로 마이그레이션할 때 사용합니다.
