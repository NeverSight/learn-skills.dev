GitOps 用の SealedSecrets、クラウド シークレット マネージャー用の外部シークレット オペレーター、およびローテーション戦略を使用して、Kubernetes に安全なシークレット管理を実装します。保存時の暗号化と RBAC 制御を使用して、TLS 証明書、API キー、資格情報を処理します。 Kubernetes アプリケーションの機密構成を保存する場合、シークレットをバージョン管理する必要がある GitOps を実装する場合、AWS Secrets Manager または Azure Key Vault と統合する場合、ダウンタイムなしで認証情報をローテーションする場合、またはプレーンテキストのシークレットから暗号化されたソリューションに移行する場合に使用します。
