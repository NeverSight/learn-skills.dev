قم بحل تعارضات الدمج وإعادة الأساس باستخدام إستراتيجيات الاسترداد الآمنة. يغطي تحديد مصادر الصراع، وقراءة علامات الصراع، واختيار استراتيجيات الحل، ومواصلة العمليات أو إحباطها بأمان. يُستخدم عند الإبلاغ عن تعارضات في دمج git، أو rebase، أو Cherry-pick، أو stash pop، أو عندما يؤدي سحب git إلى تغييرات متعارضة، أو عندما تحتاج إلى إحباط عملية دمج أو إعادة تأسيس فاشلة بأمان وإعادة تشغيلها.
