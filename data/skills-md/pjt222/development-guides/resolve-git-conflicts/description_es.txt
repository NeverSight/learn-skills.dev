Resuelva conflictos de fusión y rebase con estrategias de recuperación segura. Cubre la identificación de fuentes de conflictos, la lectura de marcadores de conflictos, la elección de estrategias de resolución y la continuación o cancelación de operaciones de forma segura. Úselo cuando un git merge, rebase, cherry-pick o stash pop informe conflictos, cuando un git pull resulte en cambios conflictivos o cuando necesite cancelar y reiniciar de forma segura una operación fallida de merge o rebase.
