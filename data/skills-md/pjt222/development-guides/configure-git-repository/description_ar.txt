قم بتكوين مستودع Git باستخدام .gitignore المناسب، وإستراتيجية الفرع، واتفاقيات الالتزام، والخطافات، والإعداد عن بعد. يغطي الإعداد الأولي والأنماط الشائعة لمشاريع R وNode.js وPython. يُستخدم عند تهيئة التحكم في الإصدار لمشروع جديد، أو إضافة .gitignore للغة أو إطار عمل محدد، أو إعداد حماية الفروع واتفاقياتها، أو تكوين خطافات الالتزام.
