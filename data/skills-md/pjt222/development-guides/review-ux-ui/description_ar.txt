قم بمراجعة تجربة المستخدم وتصميم الواجهة باستخدام أساليب Nielsen الاستدلالية وإرشادات إمكانية الوصول إلى WCAG 2.1 ومراجعة لوحة المفاتيح وقارئ الشاشة وتحليل تدفق المستخدم وتقييم الحمل المعرفي وتقييم سهولة استخدام النموذج. يُستخدم عند إجراء مراجعة لقابلية الاستخدام قبل الإصدار، أو تقييم امتثال إمكانية الوصول إلى WCAG 2.1، أو تقييم تدفقات المستخدم من أجل الكفاءة، أو مراجعة تصميم النموذج، أو إجراء تقييم إرشادي للواجهة الحالية.
