Diagnostica e risolvi i problemi di connessione del server MCP tra Claude Code, Claude Desktop e i server MCP. Copre l'analisi degli argomenti di Windows, gli errori di autenticazione, i problemi di trasporto e il debug specifico della piattaforma. Da utilizzare quando Claude Code o Claude Desktop non riescono a connettersi a un server MCP, quando gli strumenti MCP non vengono visualizzati nelle sessioni, in caso di errori "impossibile collegare il server", quando una connessione funzionante viene interrotta o quando si configura MCP su una nuova macchina.
