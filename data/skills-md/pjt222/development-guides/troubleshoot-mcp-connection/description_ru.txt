Диагностика и устранение проблем с подключением сервера MCP между Claude Code, Claude Desktop и серверами MCP. Охватывает анализ аргументов Windows, ошибки аутентификации, проблемы транспорта и отладку для конкретной платформы. Используйте, когда Claude Code или Claude Desktop не удается подключиться к серверу MCP, когда инструменты MCP не отображаются в сеансах, при ошибках «невозможно подключить сервер», когда рабочее соединение остановлено или при настройке MCP на новом компьютере.
