Diagnostiquer et résoudre les problèmes de connexion au serveur MCP entre Claude Code, Claude Desktop et les serveurs MCP. Couvre l'analyse des arguments Windows, les échecs d'authentification, les problèmes de transport et le débogage spécifique à la plate-forme. À utiliser lorsque Claude Code ou Claude Desktop ne parvient pas à se connecter à un serveur MCP, lorsque les outils MCP n'apparaissent pas dans les sessions, en cas d'erreurs « Impossible de connecter le serveur », lorsqu'une connexion fonctionnelle s'est arrêtée ou lors de la configuration de MCP sur une nouvelle machine.
