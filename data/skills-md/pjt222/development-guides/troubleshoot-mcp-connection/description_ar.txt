تشخيص وإصلاح مشكلات اتصال خادم MCP بين خوادم Claude Code وClaude Desktop وخوادم MCP. يغطي تحليل وسيطة Windows وفشل المصادقة ومشكلات النقل وتصحيح الأخطاء الخاصة بالنظام الأساسي. يُستخدم عندما يفشل Claude Code أو Claude Desktop في الاتصال بخادم MCP، أو عندما لا تظهر أدوات MCP في الجلسات، أو عند ظهور أخطاء "لا يمكن إرفاق الخادم"، أو عند توقف اتصال العمل، أو عند إعداد MCP على جهاز جديد.
