Diagnostizieren und beheben Sie MCP-Server-Verbindungsprobleme zwischen Claude Code, Claude Desktop und MCP-Servern. Behandelt das Parsen von Windows-Argumenten, Authentifizierungsfehler, Transportprobleme und plattformspezifisches Debugging. Verwenden Sie diese Option, wenn Claude Code oder Claude Desktop keine Verbindung zu einem MCP-Server herstellen kann, wenn MCP-Tools nicht in Sitzungen angezeigt werden, bei Fehlern „Server kann nicht angeschlossen werden“, wenn eine funktionierende Verbindung unterbrochen wurde oder wenn MCP auf einem neuen Computer eingerichtet wird.
