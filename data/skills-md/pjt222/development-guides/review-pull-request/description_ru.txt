Просмотрите сквозной запрос на включение с помощью GitHub CLI. Охватывает анализ различий, просмотр истории коммитов, проверку проверки CI/CD, обратную связь на уровне серьезности (блокировка/предложение/гнида/похвала) и отправку на проверку GH PR. Используйте, когда запрос на включение назначен на проверку, выполнение самопроверки перед запросом вклада других, проведение второй проверки после рассмотрения отзывов или аудит объединенного PR для оценки качества после слияния.
