قم بمراجعة طلب السحب من البداية إلى النهاية باستخدام GitHub CLI. يغطي تحليل الفرق، ومراجعة سجل الالتزام، والتحقق من فحص CI/CD، والتعليقات على مستوى الخطورة (الحظر/الاقتراح/النيت/الثناء)، وتقديم مراجعة gh pr. يُستخدم عند تعيين طلب سحب للمراجعة، أو إجراء مراجعة ذاتية قبل طلب مدخلات الآخرين، أو إجراء مراجعة ثانية بعد معالجة الملاحظات، أو تدقيق العلاقات العامة المدمجة لتقييم الجودة بعد الدمج.
