Revise una solicitud de extracción de un extremo a otro mediante la CLI de GitHub. Cubre análisis de diferencias, revisión del historial de confirmaciones, verificación de verificación de CI/CD, comentarios de nivel de gravedad (bloqueo/sugerencia/nit/elogio) y envío de revisión de gh pr. Úselo cuando se asigna una solicitud de extracción para revisión, se realiza una autoevaluación antes de solicitar la opinión de otros, se realiza una segunda revisión después de abordar los comentarios o se audita un RP fusionado para una evaluación de calidad posterior a la fusión.
