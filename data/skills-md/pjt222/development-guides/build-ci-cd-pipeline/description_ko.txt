매트릭스 빌드, 종속성 캐싱, 아티팩트 관리 및 비밀 처리 기능이 포함된 GitHub Actions를 사용하여 다단계 CI/CD 파이프라인을 설계하고 구현합니다. 병렬 실행 및 조건부 논리를 사용하여 린팅, 테스트, 구축 및 배포 단계를 포괄하는 워크플로를 만듭니다. 새 프로젝트에 대한 자동화된 테스트 및 배포를 설정하고, Jenkins 또는 CircleCI에서 GitHub Actions로 마이그레이션하고, 플랫폼 전반에 걸쳐 매트릭스 빌드를 구현하고, 빌드 캐싱을 추가하거나, 보안 검색 및 품질 게이트가 포함된 다단계 파이프라인을 생성할 때 사용합니다.
