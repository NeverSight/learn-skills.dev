قم بتصميم وتنفيذ خطوط أنابيب CI/CD متعددة المراحل باستخدام GitHub Actions مع إنشاءات المصفوفة والتخزين المؤقت للتبعية وإدارة العناصر والمعالجة السرية. قم بإنشاء مهام سير عمل تمتد عبر مراحل الفحص والاختبار والبناء والنشر من خلال التنفيذ المتوازي والمنطق الشرطي. يُستخدم عند إعداد الاختبار والنشر الآلي لمشروع جديد، أو الترحيل من Jenkins أو CircleCI إلى GitHub Actions، أو تنفيذ إنشاءات المصفوفة عبر الأنظمة الأساسية، أو إضافة التخزين المؤقت للإنشاء، أو إنشاء خطوط أنابيب متعددة المراحل مع فحص الأمان وبوابات الجودة.
