マトリックス ビルド、依存関係のキャッシュ、アーティファクト管理、シークレット処理を備えた GitHub Actions を使用して、マルチステージ CI/CD パイプラインを設計および実装します。並列実行と条件付きロジックを使用して、lint、テスト、構築、展開の各段階にわたるワークフローを作成します。新しいプロジェクトの自動テストとデプロイメントの設定、Jenkins または CircleCI から GitHub Actions への移行、プラットフォーム間でのマトリックス ビルドの実装、ビルド キャッシュの追加、またはセキュリティ スキャンと品質ゲートを備えたマルチステージ パイプラインの作成時に使用します。
