Progetta e implementa pipeline CI/CD multifase utilizzando GitHub Actions con build di matrici, memorizzazione nella cache delle dipendenze, gestione degli artefatti e gestione dei segreti. Crea flussi di lavoro che abbracciano fasi di linting, test, creazione e distribuzione con esecuzione parallela e logica condizionale. Da utilizzare quando si configurano test e distribuzioni automatizzati per un nuovo progetto, si esegue la migrazione da Jenkins o CircleCI a GitHub Actions, si implementano build di matrici su piattaforme, si aggiunge il caching delle build o si creano pipeline multifase con scansione di sicurezza e controlli di qualità.
