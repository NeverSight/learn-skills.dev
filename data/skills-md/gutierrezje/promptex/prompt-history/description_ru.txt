Извлекает и обрабатывает историю запросов ИИ в уценку, готовую для PR, с помощью PromptEx (pmtx). Считывает журналы сеансов инструмента искусственного интеллекта, сопоставляет запросы с областью действия git и отображает выходные данные по категориям для описаний запросов на включение. Используйте, когда пользователь спрашивает об истории запросов, хочет задокументировать свои аргументы в пользу PR с помощью ИИ или просит извлечь или суммировать свой сеанс.
