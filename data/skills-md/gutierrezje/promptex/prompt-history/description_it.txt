Estrae e cura la cronologia dei prompt dell'IA in un markdown pronto per il PR utilizzando PromptEx (pmtx). Legge i log delle sessioni dello strumento AI, correla i prompt all'ambito git ed esegue il rendering dell'output classificato per le descrizioni delle richieste pull. Da utilizzare quando l'utente chiede informazioni sulla cronologia dei prompt, desidera documentare il proprio ragionamento assistito dall'intelligenza artificiale per un PR o chiede di estrarre o riepilogare la propria sessione.
