التحليل التنافسي المنهجي لتحديد موضع المنتج وتمكين المبيعات والتخطيط الاستراتيجي. يُستخدم عندما يريد المستخدم تحليل المنافسين، أو إنشاء بطاقات قتالية، أو إنشاء صفحات مقارنة، أو فهم وضع السوق، أو البحث في المشهد التنافسي. يتم تشغيله أيضًا على: "تحليل المنافسين" أو "المشهد التنافسي" أو "بطاقة المعركة" أو "تحليل الفوز/الخسارة" أو "وضع السوق" أو "كيف نقارن بـ" أو "ما تفعله [الشركة]".
