Итеративно проверяет и исправляет проблемы с качеством навыков Claude Code, пока они не будут соответствовать стандартам. Запускает автоматические циклы проверки исправлений с помощью агента проверки навыков. Используйте для устранения проблем с качеством навыков, улучшения описаний навыков, запуска автоматических циклов проверки навыков или итеративного уточнения навыков. Запускается по «исправить мой навык», «улучшить качество навыка», «цикл улучшения навыка». НЕ для разовых проверок — используйте напрямую /skill-reviewer.
