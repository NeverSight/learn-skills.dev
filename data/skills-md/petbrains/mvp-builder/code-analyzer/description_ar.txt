تحليل شامل لقاعدة التعليمات البرمجية لبناء نموذج عقلي لهيكل المشروع والتبعيات وسياق التنفيذ. يُستخدم عند الحاجة إلى: (1) فهم بنية المشروع قبل المراجعة أو التوثيق، (2) البحث عن التبعيات والوحدات النمطية المشتركة، (3) تحديد موقع علامات التنفيذ (AICODE-*)، (4) إعداد السياق للمراجعة أو إنشاء الذاكرة أو إنشاء الوكيل. المشغلات: تحليل التعليمات البرمجية، وتحميل سياق التعليمات البرمجية، ومسح قاعدة التعليمات البرمجية، وفهم بنية المشروع.
