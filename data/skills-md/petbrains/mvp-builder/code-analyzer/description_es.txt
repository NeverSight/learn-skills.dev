Análisis completo de la base de código para construir un modelo mental de la estructura del proyecto, las dependencias y el contexto de implementación. Úselo cuando sea necesario: (1) comprender la arquitectura del proyecto antes de la revisión o documentación, (2) encontrar dependencias y módulos compartidos, (3) localizar marcadores de implementación (AICODE-*), (4) preparar el contexto para la revisión, generación de memoria o creación de agentes. Activadores: analizar código, cargar el contexto del código, escanear la base del código, comprender la estructura del proyecto.
