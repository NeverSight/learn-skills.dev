Комплексный анализ кодовой базы для построения мысленной модели структуры проекта, зависимостей и контекста реализации. Используйте, когда необходимо: (1) понять архитектуру проекта перед проверкой или документированием, (2) найти зависимости и общие модули, (3) найти маркеры реализации (AICODE-*), (4) подготовить контекст для проверки, создания памяти или создания агента. Запускается при: анализе кода, загрузке контекста кода, сканировании базы кода, понимании структуры проекта.
