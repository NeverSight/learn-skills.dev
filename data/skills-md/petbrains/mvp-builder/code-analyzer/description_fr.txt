Analyse complète de la base de code pour créer un modèle mental de la structure du projet, des dépendances et du contexte de mise en œuvre. À utiliser lorsque vous avez besoin de : (1) Comprendre l'architecture du projet avant la révision ou la documentation, (2) Rechercher les dépendances et les modules partagés, (3) Localiser les marqueurs d'implémentation (AICODE-*), (4) Préparer le contexte pour la révision, la génération de mémoire ou la création d'agent. Déclencheurs sur : analyser le code, charger le contexte du code, analyser la base de code, comprendre la structure du projet.
