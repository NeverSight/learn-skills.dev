Получение актуальной библиотечной документации с помощью инструментов Context7 MCP. Процесс ДУМАТЬ → РЕШИТЬ → ВЫБРАТЬ → ПРИМЕНИТЬ. Используйте при получении документации библиотеки, преобразовании имен пакетов в идентификаторы, получении руководств по реализации, изучении ссылок на API. Предоставляет стратегию разрешения пакетов, оценку оценки доверия, масштабирование токенов (3–20 тыс.), шаблоны выбора тем.
