Capture la interfaz de usuario y envíela a Figma como capas de diseño editables usando generate_figma_design. Úselo cuando alguien diga "enviar a Figma", "enviar a Figma", "capturar interfaz de usuario", "código en lienzo", "sincronizar con Figma", "actualizar archivo Figma" o quiere enviar la interfaz de usuario implementada a Figma para revisión del diseñador. También se activa después de completar la creación de una función cuando el flujo de trabajo incluye el viaje de ida y vuelta de Figma (Figma → código → Figma). Cubre solo la dirección del código a Figma: para leer desde Figma, use las herramientas get_metadata o get_screenshot directamente.
