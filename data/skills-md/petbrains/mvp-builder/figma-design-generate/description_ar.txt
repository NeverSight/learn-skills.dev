التقط واجهة المستخدم وأرسلها إلى Figma كطبقات تصميم قابلة للتحرير باستخدام generator_figma_design. يُستخدم عندما يقول شخص ما "ادفع إلى Figma"، أو "أرسل إلى Figma"، أو "التقاط واجهة المستخدم"، أو "رمز إلى اللوحة القماشية"، أو "المزامنة مع Figma"، أو "تحديث ملف Figma"، أو يريد إرسال واجهة المستخدم المنفذة إلى Figma لمراجعة المصمم. يتم التشغيل أيضًا بعد إكمال إنشاء الميزة عندما يتضمن سير العمل رحلة ذهاب وإياب Figma (Figma → code → Figma). يغطي فقط اتجاه الكود إلى Figma - للقراءة من Figma، استخدم أدوات get_metadata أو get_screenshot مباشرةً.
