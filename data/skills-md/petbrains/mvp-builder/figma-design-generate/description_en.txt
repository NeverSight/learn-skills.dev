Capture UI and send it to Figma as editable design layers using generate_figma_design. Use when someone says "push to Figma", "send to Figma", "capture UI", "code to canvas", "sync with Figma", "update Figma file", or wants to send implemented UI to Figma for designer review. Also trigger after completing a feature build when the workflow includes Figma roundtrip (Figma → code → Figma). Covers only the code-to-Figma direction — for reading from Figma, use get_metadata or get_screenshot tools directly.
