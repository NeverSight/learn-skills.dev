Capturez l'interface utilisateur et envoyez-la à Figma sous forme de couches de conception modifiables à l'aide de generate_figma_design. À utiliser lorsque quelqu'un dit « pousser vers Figma », « envoyer à Figma », « capturer l'interface utilisateur », « coder sur le canevas », « synchroniser avec Figma », « mettre à jour le fichier Figma » ou souhaite envoyer l'interface utilisateur implémentée à Figma pour examen par le concepteur. Déclenchez également après avoir terminé la création d'une fonctionnalité lorsque le flux de travail inclut un aller-retour Figma (Figma → code → Figma). Couvre uniquement la direction code vers Figma – pour lire à partir de Figma, utilisez directement les outils get_metadata ou get_screenshot.
