Simplifique y refine el código para mejorar la claridad, la coherencia y la facilidad de mantenimiento, preservando al mismo tiempo toda la funcionalidad.  
De forma predeterminada, céntrese en el código modificado más recientemente, a menos que se indique lo contrario.

Condiciones de activación:
- Después de completar la implementación de una función y necesitar limpiar el código  
- Cuando se considera que el código es demasiado complejo o difícil de entender.  
- Al refactorizar el código existente para mejorar la legibilidad  
- Cuando el usuario menciona "simplificar", "simplificar", "limpiar" o "refactorizar"
