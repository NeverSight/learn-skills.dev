Упростите и уточните код, чтобы повысить ясность, согласованность и удобство сопровождения, сохраняя при этом всю функциональность.  
По умолчанию фокусируется на самом последнем измененном коде, если не указано иное.

Условия триггера:
- После завершения реализации функции и необходимости очистить код.  
- Когда код оказывается слишком сложным или трудным для понимания.  
- При рефакторинге существующего кода для улучшения читаемости.  
– Когда пользователь упоминает «упростить», «упростить», «очистить» или «рефакторить».
