Auslösebedingungen:

- Wenn sich herausstellt, dass der Code zu komplex oder schwer verständlich ist  
- Beim Refactoring von vorhandenem Code zur Verbesserung der Lesbarkeit  
- Wenn der Benutzer „vereinfachen“, „vereinfachen“, „bereinigen“ oder „umgestalten“ erwähnt
