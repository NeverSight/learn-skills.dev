Maîtrise de l'AWS CLI pour les opérations S3, EC2, Lambda, IAM et de services communs. À utiliser lorsque l'utilisateur demande de « télécharger sur S3 », « lancer EC2 », « déployer Lambda », « configurer AWS », « profils AWS », « vérifier les ressources AWS » ou toute tâche de ligne de commande AWS.
