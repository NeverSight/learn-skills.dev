Maîtrise de GitHub CLI (gh) pour les problèmes, les relations publiques, les versions, les actions, l'essentiel et la gestion des dépôts. À utiliser lorsque l'utilisateur demande de « créer un PR », « lister les problèmes », « vérifier l'état du CI », « créer une version », « afficher les exécutions de flux de travail », « créer un résumé », « cloner un dépôt », « créer un dépôt » ou toute opération GitHub à partir de la ligne de commande.
