إتقان GitHub CLI (gh) للقضايا والعلاقات العامة والإصدارات والإجراءات والجوهر وإدارة الريبو. يُستخدم عندما يطلب المستخدم "إنشاء علاقات عامة"، أو "قائمة المشكلات"، أو "التحقق من حالة CI"، أو "إنشاء إصدار"، أو "عرض عمليات تشغيل سير العمل"، أو "إنشاء جوهر"، أو "استنساخ الريبو"، أو "تفرع الريبو"، أو أي عمليات GitHub من سطر الأوامر.
