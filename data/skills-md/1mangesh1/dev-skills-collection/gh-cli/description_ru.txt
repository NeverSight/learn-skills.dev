GitHub CLI (gh) — знание проблем, PR, релизов, действий, сути и управления репозиториями. Используйте, когда пользователь просит «создать PR», «составить список проблем», «проверить статус CI», «создать выпуск», «просмотреть выполнение рабочего процесса», «создать суть», «клонировать репозиторий», «разветвить репо» или любые операции GitHub из командной строки.
