Используйте, когда пользователь запрашивает «перебазировать ветку», «выбрать вишневую фиксацию», «найти ошибку с помощью биссектрисы», «восстановить потерянную фиксацию», «сжать коммиты», «исправить историю git», «интерактивную перезагрузку» или любые расширенные задачи git.
