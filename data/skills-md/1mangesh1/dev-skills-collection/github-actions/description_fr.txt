GitHub Actions Maîtrise CI/CD pour les workflows, les constructions matricielles, la mise en cache, les secrets et les actions réutilisables. À utiliser lorsque l'utilisateur demande de « configurer CI », « créer un workflow », « ajouter des actions GitHub », « des constructions matricielles », des « dépendances de cache », « déployer avec des actions », des « workflows réutilisables » ou toute tâche de pipeline CI/CD.
