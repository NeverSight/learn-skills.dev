Настройка перехватчиков Git с предварительной фиксацией, хаски и lint-staged для автоматического контроля качества кода. Используйте, когда пользователь запрашивает «настроить перехватчики перед фиксацией», «добавить хаски», «линт при фиксации», «форматировать перед фиксацией», «запустить тесты при нажатии», «настроить перехватчики git» или автоматизировать проверку качества кода.
