Манипулирование JSON и YAML с помощью инструментов командной строки jq и yq. Используйте, когда пользователь запрашивает «анализ JSON», «преобразование YAML», «извлечение из JSON», «фильтрацию массива JSON», «преобразование YAML в JSON», «запрос JSON» или манипулирование структурированными данными из командной строки.
