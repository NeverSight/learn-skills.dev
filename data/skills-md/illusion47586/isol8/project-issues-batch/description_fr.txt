Traitez tous les problèmes liés à un projet GitHub et implémentez-les en tant que PR distincts. À utiliser lorsque l'utilisateur fournit une URL de projet GitHub (github.com/owner/repo/projects/N ou github.com/orgs/NAME/projects/N) et souhaite implémenter tous les problèmes de ce projet. Chaque numéro devient son propre PR. Gère les dépendances à l'aide de PR en cascade où les problèmes dépendants bifurquent de la branche PR de leur dépendance.
