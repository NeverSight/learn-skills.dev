Elabora tutti i problemi collegati da un progetto GitHub e implementali come PR separati. Da utilizzare quando l'utente fornisce l'URL di un progetto GitHub (github.com/owner/repo/projects/N o github.com/orgs/NAME/projects/N) e desidera implementare tutti i problemi in quel progetto. Ogni numero diventa il proprio PR. Gestisce le dipendenze utilizzando PR a cascata in cui i problemi dipendenti si diramano dal ramo PR della loro dipendenza.
