Обработайте все связанные проблемы из проекта GitHub и реализуйте их как отдельные запросы на отправку заявок. Используйте, когда пользователь предоставляет URL-адрес проекта GitHub (github.com/owner/repo/projects/N или github.com/orgs/NAME/projects/N) и хочет реализовать все задачи в этом проекте. Каждый выпуск становится отдельным пиаром. Обрабатывает зависимости с помощью каскадных PR, когда зависимые проблемы разветвляются от ветки PR своей зависимости.
