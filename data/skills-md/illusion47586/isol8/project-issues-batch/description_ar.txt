معالجة جميع المشكلات المرتبطة من مشروع GitHub وتنفيذها كممثلين رئيسيين منفصلين. يُستخدم عندما يقدم المستخدم عنوان URL لمشروع GitHub (github.com/owner/repo/projects/N أو github.com/orgs/NAME/projects/N) ويريد تنفيذ جميع المشكلات في هذا المشروع. كل قضية تصبح العلاقات العامة الخاصة بها. يتعامل مع التبعيات باستخدام العلاقات العامة المتتالية حيث تتفرع المشكلات التابعة إلى فرع العلاقات العامة الخاص بالتبعية.
