Senden Sie unter macOS über die WeChat-Desktop-App eine Textnachricht an einen bestimmten Kontakt. Wird verwendet, wenn der Benutzer den Namen des Kontakts und den Nachrichteninhalt angibt, z. B. „Senden Sie einen WeChat an Zhang San: xxx“, „Öffnen Sie WeChat und senden Sie einen Satz an jemanden“ oder „Senden Sie den angegebenen Text über WeChat.“ Vor der Ausführung muss WeChat Desktop installiert werden und dem aktuellen Terminal oder Codex müssen Zugriffsberechtigungen erteilt werden. Führen Sie während der Ausführung nur einen exakten Abgleich durch: Suchen Sie zuerst nach dem Kontakt und überprüfen Sie dann, ob der Titel der aktuellen Sitzung genau mit dem Ziel übereinstimmt. Wenn der Abgleich fehlschlägt, beenden Sie den Vorgang, um zu vermeiden, dass er an die falsche Person gesendet wird.
