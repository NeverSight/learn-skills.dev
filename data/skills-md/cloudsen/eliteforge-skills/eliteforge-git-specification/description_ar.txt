التنفيذ الموحد لمواصفات حوكمة CuiCan Workshop Git، التي تغطي نموذج الفرع ورسائل الالتزام وسير عمل MR واستراتيجيات الإصدار والإصلاح العاجل. يُستخدم عندما يذكر المستخدمون معايير Git، أو تسمية الفروع، أو خيارات إعادة التأسيس/الدمج، أو تدفق إرسال MR، أو متطلبات الاسكواش، أو التحقق من صحة الرسالة، أو دمج الإصدار، أو تسمية الإصلاح العاجل، أو تجنب التعارض.
