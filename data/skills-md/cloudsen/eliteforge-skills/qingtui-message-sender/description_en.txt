On macOS, use the desktop version of QingTui to send a text message to a specified contact. Use it when the user provides the contact name and message content, e.g. “Open QingTui and send to Zhang San: xxx” or “Send a message to someone via QingTui.” The QingTui desktop client must be installed and the current terminal or Codex granted Accessibility permissions before running; the script takes a screenshot and uses OCR to verify before sending, so Screen Recording permission is usually also required. During execution it performs exact matching only: first search for the contact, then verify that the current session title exactly matches the target; if matching fails it stops to avoid sending to the wrong person.
