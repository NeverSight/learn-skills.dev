Verwenden Sie unter macOS die Desktop-Version von QingTui, um eine Textnachricht an einen bestimmten Kontakt zu senden. Verwenden Sie es, wenn der Benutzer den Kontaktnamen und den Nachrichteninhalt bereitstellt, z. B. „Öffne QingTui und sende an Zhang San: xxx“ oder „Sende eine Nachricht an jemanden über QingTui.“ Der QingTui-Desktop-Client muss installiert sein und dem aktuellen Terminal oder Codex müssen Barrierefreiheitsberechtigungen erteilt werden, bevor er ausgeführt werden kann. Das Skript erstellt einen Screenshot und verwendet OCR zur Überprüfung vor dem Senden. Daher ist in der Regel auch eine Bildschirmaufzeichnungsberechtigung erforderlich. Während der Ausführung wird nur ein exakter Abgleich durchgeführt: Suchen Sie zuerst nach dem Kontakt und überprüfen Sie dann, ob der Titel der aktuellen Sitzung genau mit dem Ziel übereinstimmt. Wenn der Abgleich fehlschlägt, wird der Vorgang abgebrochen, um zu vermeiden, dass er an die falsche Person gesendet wird.
