استخدمه دائمًا عند تشخيص مشكلات Ably - عدم وصول الرسائل، أو عدم تحديث التواجد، أو فشل الاتصال، أو أخطاء المصادقة (40101، 40142، 40160)، أو مشكلات دورة حياة القناة، أو مشكلات الإعلام الدفعي، أو مشكلات غرف الدردشة، أو مشكلات أعضاء Spaces/المؤشر، أو أي رمز خطأ Ably. يوفر أوامر Ably CLI لمراقبة حركة المرور المباشرة وفحص الحالة ومحاكاة العملاء. ليس لبناء عمليات تكامل جديدة بمهارة (استخدم مهارة الاستخدام بدلاً من ذلك).
