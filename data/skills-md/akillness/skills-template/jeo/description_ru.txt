JEO — интегрированный навык оркестрации агентов искусственного интеллекта. Планируйте с помощью ralph+plannotator, выполняйте с помощью команды/bmad, проверяйте поведение браузера с помощью агента-браузера, применяйте обратную связь пользовательского интерфейса с помощью агентирования (аннотации), автоматическую очистку рабочих деревьев после завершения. Поддерживает Claude, Codex, Gemini CLI и OpenCode. Установите: ralph, omc, omx, ohmg, bmad, plannotator, агент-браузер, агентацию.
