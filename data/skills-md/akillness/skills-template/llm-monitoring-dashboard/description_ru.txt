Автоматически создает страницу мониторинга использования LLM на панели администратора PM. Отслеживание токенов/стоимостей/задержек на основе Tokuin CLI + система ранжирования пользователей + отслеживание неактивных пользователей + аналитика PM на основе данных + глобальный поиск Cmd+K + детализация навигации по каждому пользователю. Поддерживает OpenAI/Anthropic/Gemini/OpenRouter.
