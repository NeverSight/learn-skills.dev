Адаптируйте единое мастер-бриф для авторов в версии для конкретных платформ для TikTok, Instagram Reels и YouTube Shorts с правильными характеристиками, безопасными зонами и собственными инструкциями для платформы. Этот навык следует использовать при преобразовании брифа для нескольких платформ, адаптации брифа кампании из TikTok в Reels, создании брифа по контенту для конкретной платформы из основного брифа, корректировке спецификаций видео для разных платформ коротких видео, создании версий одного брифа для TikTok, Reels и Shorts, форматировании результатов креатора по платформам, разделении брифа кампании на версии для платформ или создании адаптированных для платформы брифов для многоплатформенной кампании креатора. Информацию о написании основного задания см. в разделе Content-brief-Builder. Сведения о создании концепций контента перед брифингом см. в разделе «Генератор концепций-контента-создателя». Информацию о проверке отправленного контента на соответствие брифу см. в разделе «Проверка соответствия содержимого брифу».
