Analysieren Sie Rohtext aus einem Instagram- oder TikTok-Story-Insights-Screenshot und formatieren Sie ihn in eine saubere, tabellenkalkulationsfähige Zeile mit beschrifteten Feldern. Diese Fähigkeit sollte verwendet werden, wenn Story-Metriken aus einem Screenshot analysiert, Story-Einblicke für eine Tabelle formatiert werden, Metriken aus einem eingefügten Story-Screenshot extrahiert werden, Story-Analysedaten bereinigt werden, Story-Insights-Text in strukturierte Daten umgewandelt werden, ein Story-Performance-Screenshot in eine Zeile für den Tracker umgewandelt wird, Story-Metriken in einer Tabelle protokolliert werden, Story-Screenshot-Daten normalisiert werden, Zahlen aus einer Story-Insights-Paste gezogen werden, Story-Metriken aus Ersteller-Screenshots organisiert werden, ein Stapel von Story-Screenshots in Zeilen verarbeitet wird und ein Story-Screenshot erstellt wird Story-Metrik-Tracker aus Screenshots oder Eingabe von Story-Daten aus einem Screenshot in ein Blatt. Informationen zum Normalisieren von Metriken aus mehreren Quellen in einer einheitlichen Tabelle finden Sie unter metrics-normalization-formatter. Informationen zur Berechnung der Engagement-Raten und zum Vergleich mit Benchmarks finden Sie unter Engagement-Rate-Calculator-Benchmarker.
