Создайте полную структуру ключевых показателей эффективности для маркетинговой кампании авторов на основе бизнес-цели. Этот навык следует использовать при настройке ключевых показателей эффективности для кампании влияния, построении плана измерений перед запуском кампании, сопоставлении бизнес-целей с маркетинговыми показателями создателей, определении первичных и вторичных ключевых показателей эффективности для программы авторов, создании системы показателей для кампании по повышению осведомленности или конверсии, установке контрольных показателей измерения по уровням создателей, построении структуры информационной панели KPI для отчетов влиятельных лиц или определении критериев успеха перед активацией авторов. Чтобы рассчитать рентабельность инвестиций после завершения кампании, см. калькулятор рентабельности кампании. Для установки числовых целевых показателей производительности см. раздел Performance-benchmark-setter. Информацию об отслеживании соблюдения требований к публикациям авторов см. в разделе «Отслеживание соответствия публикаций авторов».
