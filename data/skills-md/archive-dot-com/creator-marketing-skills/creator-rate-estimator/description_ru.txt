Оцените справедливые рыночные ставки для партнерских отношений с создателями на основе платформы, количества подписчиков, уровня вовлеченности, ниши и типа результата. Этот навык следует использовать при оценке рейтинга влиятельных лиц, расчете цен для авторов, построении прейскуранта для кампании, проверке справедливости ставки создателя, сравнении затрат влиятельных лиц на разных платформах, составлении бюджета кампании для авторов, оценке прейскуранта создателя, выяснении того, сколько платить влиятельному лицу, сравнении ставок создателя с рыночными данными или оценке того, не завышает ли автор завышенную цену. Для согласования ставок после оценки см. Руководство по переговорам по ставкам. Полную информацию о проверке авторов, помимо цен, см. в таблице показателей проверки авторов.
