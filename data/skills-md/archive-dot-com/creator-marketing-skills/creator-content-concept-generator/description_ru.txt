Создайте 5–10 индивидуальных концепций контента для партнерства с авторами, которые будут соответствовать стилю автора, а не продиктованы брендом. Этот навык следует использовать при мозговом штурме идей контента для кампании авторов, создании концепций контента для влиятельных лиц, разработке идей для результатов авторов, разработке ракурсов контента для партнерства с влиятельными лицами, разработке концепций видео для TikTok, Reels или YouTube Shorts, создании привязок к контенту для совместной работы авторов, создании концептуального меню для краткого описания автора, генерации идей контента для спонсируемого поста или планировании того, что создатель должен на самом деле сделать. Для написания полного описания контента после выбора концепций см. Content-brief-Builder. Для создания краткого описания кампании, которое предшествует планированию контента, см. генератор краткого обзора кампании. Информацию об адаптации одной концепции к различным платформам см. в разделе «Многоплатформенный-формат-адаптер».
