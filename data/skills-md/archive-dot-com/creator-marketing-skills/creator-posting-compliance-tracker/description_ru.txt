Укажите авторов, заключивших контракт, с указанием требуемого и фактического количества публикаций и получите таблицу соответствия с классификациями статусов и примечаниями для эскалации для авторов, не соответствующих требованиям. Этот навык следует использовать при проверке того, какие авторы выполнили свои обязательства по публикации, аудите завершения результатов в рамках кампании, отслеживании контрактных и фактических публикаций для программы влиятельных лиц, определении того, какие авторы отстают от результатов, построении отчета о соответствии для создателей кампании, пометке просроченных или отсутствующих публикаций авторов, проверке статуса публикации для всех авторов в кампании, создании примечаний по эскалации для влиятельных лиц, не соответствующих требованиям, или создании отслеживания результатов для кампании авторов. Чтобы проверить, соответствует ли конкретный фрагмент контента кратким требованиям, см. раздел «Проверка соответствия содержания и краткого обзора». Чтобы узнать, как преследовать конкретного автора по поводу позднего результата, см. раздел Universal-Creator-Follow-Up-Chaser. Чтобы создать полный отчет о кампании с показателями рентабельности инвестиций, см. калькулятор рентабельности инвестиций в кампанию.
