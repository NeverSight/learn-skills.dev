Geben Sie vertragliche Ersteller mit ihren erforderlichen und tatsächlichen Beitragszahlen ein und erhalten Sie eine Compliance-Tabelle mit Statusklassifizierungen und Eskalationshinweisen für nicht konforme Ersteller. Diese Fähigkeit sollte verwendet werden, wenn überprüft wird, welche Ersteller ihren Veröffentlichungsverpflichtungen nachgekommen sind, die Fertigstellung von Lieferungen in einer Kampagne überprüft wird, vertraglich vereinbarte und tatsächliche Beiträge für ein Influencer-Programm verfolgt werden, ermittelt wird, welche Ersteller mit ihren Lieferungen im Rückstand sind, ein Compliance-Bericht für Kampagnenersteller erstellt wird, überfällige oder fehlende Beiträge von Erstellern markiert werden, der Veröffentlichungsstatus aller Ersteller in einer Kampagne überprüft wird, Eskalationsnotizen für nicht konforme Influencer erstellt werden oder ein Lieferungs-Tracker für eine Erstellerkampagne erstellt wird. Informationen zur Überprüfung, ob ein bestimmter Inhalt den Kurzanforderungen entspricht, finden Sie unter content-to-brief-compliance-checker. Informationen zum Verfolgen eines bestimmten Erstellers wegen einer verspäteten Lieferung finden Sie unter universal-creator-follow-up-chaser. Informationen zum Erstellen eines vollständigen Kampagnenberichts mit ROI-Kennzahlen finden Sie im Kampagnen-Roi-Rechner.
