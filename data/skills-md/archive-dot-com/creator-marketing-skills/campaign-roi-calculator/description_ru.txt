Рассчитайте рентабельность инвестиций в кампанию влиятельных лиц и создайте готовое для руководства краткое изложение на основе необработанных данных об эффективности. Этот навык следует использовать при расчете рентабельности инвестиций в кампанию для авторов, составлении отчета об эффективности кампании для руководства, преобразовании необработанных показателей влиятельных лиц в сводную информацию, расчете рентабельности инвестиций в рекламу и EMV по CPM CPE для программы для авторов, суммировании расходов на кампанию по сравнению с доходами для встречи с заинтересованными сторонами, доказательстве рентабельности инвестиций в маркетинг влияния для директора по маркетингу или вице-президента, создания отчета по результатам кампании с финансовыми показателями или сравнения эффективности канала влиятельного лица с платными социальными сетями. Чтобы установить целевые показатели KPI перед запуском кампании, см. раздел Performance-benchmark-setter. Информацию об отслеживании соблюдения требований к публикациям авторов см. в разделе «Отслеживание соответствия публикаций авторов». Полный отчет по итогам кампании с качественным анализом можно найти в системе показателей после кампании. Для создания ссылок UTM для включения атрибуции см. utm-parameter-builder.
