Преобразует необработанные данные об эффективности кампании в структурированный документ ежеквартального обзора бизнеса с победами и промахами, анализом тенденций и рекомендуемыми изменениями на следующий квартал. Этот навык следует использовать при построении QBR для программы для авторов, написании ежеквартального маркетингового обзора влиятельных лиц, обобщении нескольких кампаний в одном квартальном отчете, представлении результатов программы для авторов руководству в конце квартала, анализе ежеквартальных тенденций в эффективности работы влиятельных лиц, подготовке краткого обзора маркетинговой программы для авторов на уровне совета директоров, определении того, что сработало, а что исключить из стратегии авторов, построении стратегического плана для кампаний для авторов в следующем квартале или анализе полной эффективности реестра за квартал. Анализ рентабельности инвестиций для одной кампании см. в разделе «Калькулятор рентабельности инвестиций в кампанию». Еженедельные обновления статуса см. в дайджесте кампании-status-dashboard-digest. Информацию об индивидуальной оценке авторов см. в оценочной карточке автора после кампании.
