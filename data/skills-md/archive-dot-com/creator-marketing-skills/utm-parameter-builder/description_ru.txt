Создавайте правильно отформатированные параметры UTM и полные URL-адреса отслеживания для маркетинговых кампаний авторов. Этот навык следует использовать при создании ссылок UTM для влиятельных лиц, создании URL-адресов отслеживания для кампании авторов, создании строк UTM для нескольких авторов, настройке ссылок отслеживания кампании, форматировании параметров UTM для партнерских отношений с влиятельными лицами, создании ссылок атрибуции для публикаций авторов, создании уникальных URL-адресов отслеживания для каждого автора, пакетном создании ссылок UTM для списка авторов или настройке отслеживания ссылок перед запуском кампании. Чтобы измерить результаты кампании после запуска, см. «Калькулятор рентабельности кампании». Информацию о нормализации метрик из нескольких источников см. в разделе metrics-normalization-formatter.
