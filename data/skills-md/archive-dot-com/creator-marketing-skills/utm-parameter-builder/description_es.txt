Genere parámetros UTM con el formato correcto y URL de seguimiento completas para campañas de marketing de creadores. Esta habilidad debe usarse al crear enlaces UTM para personas influyentes, crear URL de seguimiento para una campaña de creadores, generar cadenas UTM para múltiples creadores, configurar enlaces de seguimiento de campañas, formatear parámetros UTM para asociaciones de personas influyentes, crear enlaces de atribución para publicaciones de creadores, crear URL de seguimiento únicas por creador, generar enlaces UTM por lotes para una lista de creadores o configurar el seguimiento de enlaces antes del lanzamiento de una campaña. Para medir los resultados de la campaña después del lanzamiento, consulte la calculadora de ROI de campaña. Para normalizar métricas de múltiples fuentes, consulte metrics-normalization-formatter.
