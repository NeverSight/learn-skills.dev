Создайте последовательность действий в несколько касаний, чтобы связаться с авторами по электронной почте, Instagram DM, TikTok DM, электронной почте YouTube или X DM. Этот навык следует использовать при написании информационных сообщений для авторов, построении последовательности работы с общественностью, составлении DM влиятельных лиц, создании графика последующих действий для авторов, написании холодных обращений к влиятельным лицам, создании презентационных сообщений о партнерстве, построении последовательности набора авторов, написании совместных личных сообщений, составлении информационных писем авторам, планировании мультисенсорной работы с авторами или создании кампании по работе с авторами. Чтобы сначала создать файл контекста бренда, см. Brand-context. Чтобы написать отдельные краткие описания контента для отправки после согласия создателя, см. Конструктор контентных брифов. Информацию о проверке авторов перед обращением см. в таблице показателей проверки авторов.
