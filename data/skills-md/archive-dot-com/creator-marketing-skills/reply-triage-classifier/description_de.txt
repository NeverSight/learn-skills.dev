Klassifizieren Sie einen Stapel von Influencer-Antworten in umsetzbare Kategorien (interessiert, verhandelbar, abgelehnt, benötigt Informationen, nicht mehr verfügbar) und generieren Sie für jede einen Vorschlag für die nächste Aktion. Diese Fähigkeit sollte verwendet werden, wenn Sie die Antworten von Erstellern selektieren, Influencer-Antworten klassifizieren, Outreach-Antworten sortieren, Ersteller-DMs kategorisieren, Batch-Antworten überprüfen, den Posteingang von Influencern verarbeiten, Follow-ups von Erstellern priorisieren, Outreach-Ergebnisse organisieren, überprüfen, wer auf eine Kampagne geantwortet hat, herausfinden, an welche Creators man sich wenden soll, oder einen chaotischen Outreach-Thread bereinigen. Informationen zum Verfassen der ersten Outreach-Nachrichten finden Sie unter outreach-writer. Informationen zum Aushandeln von Tarifen mit Erstellern finden Sie im Rate-Negotiation-Playbook.
