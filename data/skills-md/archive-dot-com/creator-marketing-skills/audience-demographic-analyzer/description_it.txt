Analizza i dati demografici del pubblico di un influencer per determinare se i suoi follower corrispondono al tuo cliente target, con un chiaro verdetto superato/fallito. Questa abilità dovrebbe essere utilizzata quando si valuta l'adattamento del pubblico, si controllano i dati demografici dell'influencer, si analizzano i dati del pubblico, si esamina la suddivisione del pubblico, si valuta l'allineamento demografico, si esamina il pubblico di un influencer, si determina se i follower di un creatore corrispondono alla demo target, si esamina un'esportazione della piattaforma o uno screenshot delle statistiche, si incollano le statistiche dell'influencer, si valuta la qualità del pubblico, si decide se il pubblico di un influencer è adatto, si controlla se questo creatore vale la pena, si esegue un rapporto sul pubblico, si confronta il pubblico del creatore o si valuta la sovrapposizione del pubblico con la demo target. Per la valutazione generale dei creatori oltre ai dati demografici, consulta la scheda di valutazione dei creatori. Per trovare nuovi creatori, consulta Creator-Discovery.
