قم بتحليل التركيبة السكانية لجمهور المؤثر لتحديد ما إذا كان متابعوهم يتطابقون مع عميلك المستهدف، مع حكم واضح بالنجاح/الفشل. يجب استخدام هذه المهارة عند تقييم مدى ملاءمة الجمهور، أو التحقق من التركيبة السكانية للمؤثرين، أو تحليل بيانات الجمهور، أو مراجعة توزيع الجمهور، أو تقييم التوافق الديموغرافي، أو فحص جمهور المؤثر، أو تحديد ما إذا كان متابعو منشئ المحتوى يتطابقون مع العرض التوضيحي المستهدف، أو مراجعة تصدير النظام الأساسي أو لقطة شاشة للإحصائيات، أو لصق إحصائيات المؤثر، أو تصنيف جودة الجمهور، أو تحديد ما إذا كان جمهور المؤثر مناسبًا أم لا، أو التحقق مما إذا كان هذا المنشئ يستحق ذلك، أو تشغيل تقرير الجمهور، أو مقارنة جماهير المبدعين، أو تقييم تداخل الجمهور مع العرض التوضيحي المستهدف. للاطلاع على التدقيق الشامل لمنشئي المحتوى بما يتجاوز التركيبة السكانية، راجع بطاقة أداء منشئي المحتوى. للعثور على منشئي محتوى جدد، راجع اكتشاف المبدعين.
