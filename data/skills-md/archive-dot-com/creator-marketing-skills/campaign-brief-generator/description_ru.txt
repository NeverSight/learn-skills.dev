Создайте полное, готовое к отправке краткое описание кампании для авторов, используя несколько исходных данных — продукта, платформ, результатов, сообщений и аудитории. Этот навык следует использовать при написании краткого описания кампании, составлении краткого описания влиятельного лица, составления краткого описания для авторов, создания краткого описания для партнерства, создания краткого описания для запуска продукта, составления краткого описания кампании для нового запуска, запуска новой кампании для авторов, планирования результатов и направления контента или подготовки любого документа, который отправляется создателям - даже если пользователь не называет его «кратким». Если пользователю сначала нужен контекст бренда, см. контекст бренда. Если после брифа пользователю нужны концепции контента, см. генератор концепций-контента-создателя. Если пользователю нужны информационно-пропагандистские сообщения, см. создатель-информационно-последовательный-генератор. Если пользователю необходимо проверить содержимое на соответствие брифу, см. средство проверки соответствия содержимого брифу.
