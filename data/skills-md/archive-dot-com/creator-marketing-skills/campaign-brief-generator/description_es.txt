Genere un resumen de campaña para creadores completo y listo para enviar a partir de algunos aportes: producto, plataformas, entregables, mensajes y audiencia. Esta habilidad debe usarse al escribir un resumen de campaña, crear un resumen de influencia, redactar un resumen de creador, generar un resumen de asociación, crear un resumen de campaña para un nuevo lanzamiento, iniciar una nueva campaña de creador, planificar entregables y dirección de contenido, o preparar cualquier documento que se envíe a los creadores, incluso si el usuario no lo llama "resumen". Si el usuario necesita primero el contexto de la marca, consulte contexto de marca. Si el usuario necesita conceptos de contenido después del resumen, consulte creador-contenido-generador de conceptos. Si el usuario necesita mensajes de divulgación, consulte creador-generador-de-secuencia-de-difusión. Si el usuario necesita comparar el contenido con el informe, consulte el verificador de cumplimiento de contenido a informe.
