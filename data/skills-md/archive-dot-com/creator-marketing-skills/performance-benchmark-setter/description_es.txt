Genere puntos de referencia de KPI realistas para una campaña de influencers antes del lanzamiento según la industria, la plataforma, el nivel de creador y el presupuesto. Esta habilidad debe usarse al establecer expectativas de rendimiento para una campaña de creador, estimar puntos de referencia de participación y conversión de alcance antes del lanzamiento, crear objetivos de KPI para un programa de influencia, pronosticar el rendimiento de la campaña por nivel y plataforma de creador, establecer objetivos de EMV y ROAS para un resumen de campaña, definir cómo se ve bien para una próxima activación de creador, calibrar las expectativas para una campaña paga o de obsequios en Instagram, TikTok o YouTube, o crear un marco de referencia para medir el éxito de la campaña. Para calcular el ROI una vez finalizada una campaña, consulte calculadora de ROI de campaña. Para calcular las tasas de participación a partir de datos de publicaciones reales, consulte la calculadora de tasa de participación y el marcador de referencia. Para crear un marco de KPI completo vinculado a los objetivos comerciales, consulte Campaign-goal-to-kpi-framework-builder.
