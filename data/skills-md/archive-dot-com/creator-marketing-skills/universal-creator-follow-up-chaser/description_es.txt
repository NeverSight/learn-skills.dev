Genere una secuencia de seguimiento personalizada para cualquier escenario de búsqueda de creadores: información faltante, contrato sin firmar, contenido retrasado, métricas faltantes o configuración de lista blanca incompleta. Esta habilidad se debe utilizar al buscar una respuesta de un creador, escribir un mensaje de seguimiento a un influencer, presionar a un creador sobre una entrega tardía, hacer un seguimiento de un contrato no firmado, solicitar métricas de campaña faltantes, buscar listas blancas o configuración de acceso a anuncios, escalar a un creador que no responde, escribir un recordatorio a un creador que se hizo fantasma, crear una cadencia de seguimiento para elementos vencidos, redactar un empujón cortés pero firme a un influencer o administrar la comunicación con los creadores cuando se cumplen las fechas límite. resbalón. Para escribir mensajes de divulgación iniciales, consulte creador-alcance-secuencia-generador. Para clasificar y clasificar las respuestas de los creadores, consulte clasificador de clasificación de respuestas.
