Generieren Sie eine personalisierte Folgesequenz für jedes Szenario der Erstellerverfolgung – fehlende Informationen, nicht unterzeichneter Vertrag, verspäteter Inhalt, fehlende Metriken oder unvollständige Einrichtung der Whitelist. Diese Fähigkeit sollte verwendet werden, wenn Sie einen Ersteller um eine Antwort bitten, eine Folgenachricht an einen Influencer schreiben, einen Ersteller auf eine verspätete Lieferung aufmerksam machen, einem nicht unterzeichneten Vertrag nachgehen, fehlende Kampagnenmetriken anfordern, Whitelisting oder die Einrichtung des Anzeigenzugriffs verfolgen, einen nicht reagierenden Ersteller eskalieren, eine Erinnerung an einen Ersteller schreiben, der gespenstisch war, einen Folgerhythmus für überfällige Elemente erstellen, einen höflichen, aber entschiedenen Anstoß für einen Influencer verfassen oder verwalten Kommunikation mit den Erstellern, wenn Fristen verschoben werden. Informationen zum Verfassen erster Outreach-Nachrichten finden Sie unter Creator-Outreach-Sequence-Generator. Informationen zum Klassifizieren und Triagieren von Erstellerantworten finden Sie unter „reply-triage-classifier“. Informationen zum Aushandeln von Tarifen, nachdem ein Ersteller geantwortet hat, finden Sie unter Creator-Negotiation-Assistant.
