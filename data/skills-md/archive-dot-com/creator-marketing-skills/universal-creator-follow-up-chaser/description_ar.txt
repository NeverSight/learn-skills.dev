قم بإنشاء تسلسل متابعة مخصص لأي سيناريو يلاحق منشئ المحتوى - معلومات مفقودة، أو عقد غير موقع، أو محتوى متأخر، أو مقاييس مفقودة، أو إعداد القائمة البيضاء غير المكتمل. يجب استخدام هذه المهارة عند مطاردة منشئ المحتوى للحصول على رد، أو كتابة رسالة متابعة إلى أحد المؤثرين، أو تنبيه منشئ المحتوى بشأن التسليم المتأخر، أو متابعة عقد غير موقع، أو طلب مقاييس الحملة المفقودة، أو مطاردة القائمة البيضاء أو إعداد الوصول إلى الإعلانات، أو تصعيد منشئ محتوى غير مستجيب، أو كتابة تذكير لمنشئ محتوى مخفي، أو بناء إيقاع متابعة للعناصر المتأخرة، أو صياغة دفعة مهذبة ولكن حازمة لأحد المؤثرين، أو إدارة اتصالات المبدعين عندما زلة المواعيد النهائية. لكتابة رسائل التوعية الأولية، راجع Creator-Outreach-Sequence-generator. لتصنيف ردود المنشئ وفرزها، راجع مصنف الرد الفرز. للتفاوض على الأسعار بعد استجابة منشئ المحتوى، راجع مساعد تفاوض منشئ المحتوى.
