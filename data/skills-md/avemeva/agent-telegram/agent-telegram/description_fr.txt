Telegram CLI pour les agents IA. À utiliser lorsque l'utilisateur a besoin d'interagir avec Telegram : lire des messages, envoyer des messages, rechercher des discussions, télécharger des médias, surveiller des conversations ou automatiser n'importe quelle tâche Telegram. Se déclenche sur les demandes de « vérifier mes messages », « envoyer un message », « rechercher Telegram », « lire non lu », « écouter discuter », « télécharger depuis Telegram » ou toute tâche nécessitant une interaction programmatique avec Telegram via la CLI « agent-telegram ».
