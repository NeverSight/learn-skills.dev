Telegram CLI для агентов ИИ. Используйте, когда пользователю необходимо взаимодействовать с Telegram — читать сообщения, отправлять сообщения, искать в чатах, загружать мультимедиа, отслеживать разговоры или автоматизировать любую задачу Telegram. Запускается по запросам «проверить мои сообщения», «отправить сообщение», «поиск в Telegram», «прочитать непрочитанное», «прослушать чат», «загрузить из Telegram» или любую задачу, требующую программного взаимодействия с Telegram через CLI `agent-telegram`.
