Telegram CLI لوكلاء الذكاء الاصطناعي. يُستخدم عندما يحتاج المستخدم إلى التفاعل مع Telegram - قراءة الرسائل، أو إرسال الرسائل، أو البحث في الدردشات، أو تنزيل الوسائط، أو مراقبة المحادثات، أو تشغيل أي مهمة Telegram تلقائيًا. يتم تشغيل طلبات "التحقق من رسائلي"، أو "إرسال رسالة"، أو "البحث في Telegram"، أو "القراءة غير المقروءة"، أو "الاستماع للدردشة"، أو "التنزيل من Telegram"، أو أي مهمة تتطلب تفاعل Telegram برمجيًا عبر واجهة سطر الأوامر "agent-telegram".
