Telegram-CLI für KI-Agenten. Verwenden Sie es, wenn der Benutzer mit Telegram interagieren muss – Nachrichten lesen, Nachrichten senden, Chats durchsuchen, Medien herunterladen, Konversationen überwachen oder beliebige Telegram-Aufgaben automatisieren. Wird bei Anfragen zum „Überprüfen meiner Nachrichten“, „Nachricht senden“, „Telegram durchsuchen“, „Ungelesenes lesen“, „Chat anhören“, „Herunterladen von Telegram“ oder bei jeder Aufgabe ausgelöst, die eine programmatische Telegram-Interaktion über die „Agent-Telegram“-CLI erfordert.
