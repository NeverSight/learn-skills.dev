CLI de Telegram para agentes de IA. Úselo cuando el usuario necesite interactuar con Telegram: leer mensajes, enviar mensajes, buscar chats, descargar medios, monitorear conversaciones o automatizar cualquier tarea de Telegram. Activa solicitudes para "revisar mis mensajes", "enviar un mensaje", "buscar en Telegram", "leer no leído", "escuchar el chat", "descargar desde Telegram" o cualquier tarea que requiera interacción programática de Telegram a través de la CLI `agent-telegram`.
