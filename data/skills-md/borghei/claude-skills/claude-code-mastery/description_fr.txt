Cette compétence doit être utilisée lorsque l'utilisateur demande « optimiser CLAUDE.md », « créer une nouvelle compétence », « écrire un agent personnalisé », « configurer les hooks », « gérer la fenêtre contextuelle », « configurer les serveurs MCP », « élaborer un package de compétences », « analyser le budget des jetons », « créer des sous-agents », « configurer les autorisations », « configurer des arbres de travail » ou « intégrer Claude Code avec des éditeurs ». À utiliser pour la maîtrise de Claude Code CLI, la création de compétences, l'ingénierie de contexte, l'automatisation des hooks, la création de sous-agents et l'optimisation du flux de travail de développement.
