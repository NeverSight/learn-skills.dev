Этот навык следует использовать, когда пользователь просит «оптимизировать CLAUDE.md», «создать новый навык», «написать собственный агент», «настроить перехватчики», «управлять контекстным окном», «настроить серверы MCP», «собрать пакет навыков», «проанализировать бюджет токенов», «создать субагенты», «настроить разрешения», «настроить рабочие деревья» или «интегрировать Claude Code с редакторами». Используйте для освоения Claude Code CLI, разработки навыков, разработки контекста, автоматизации перехватчиков, создания субагентов и оптимизации рабочего процесса разработки.
