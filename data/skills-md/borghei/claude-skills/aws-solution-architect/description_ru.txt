Создавайте архитектуры AWS для стартапов, используя бессерверные шаблоны и шаблоны IaC. Используйте его, когда вас просят спроектировать бессерверную архитектуру, создать шаблоны CloudFormation, оптимизировать затраты на AWS, настроить конвейеры CI/CD или перейти на AWS. Охватывает Lambda, API Gateway, DynamoDB, ECS, Aurora и оптимизацию затрат.
