يراجع تصميمات REST API للتأكد من الجودة والاتساق وكسر التغييرات. تعمل مواصفات Lints OpenAPI على إنشاء بطاقات أداء لواجهة برمجة التطبيقات (API)، وتكتشف التغييرات المعطلة بين الإصدارات. يُستخدم عند تصميم واجهات برمجة التطبيقات، أو مراجعة عقود واجهة برمجة التطبيقات، أو إدارة إصدارات واجهة برمجة التطبيقات.
