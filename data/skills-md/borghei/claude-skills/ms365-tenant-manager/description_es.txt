Administración de inquilinos de Microsoft 365 para administradores globales. Automatice la configuración de inquilinos de M365, las tareas administrativas de Office 365, la administración de usuarios de Azure AD, la configuración de Exchange Online, la administración de Teams y las políticas de seguridad. Genere scripts de PowerShell para operaciones masivas, políticas de acceso condicional, administración de licencias e informes de cumplimiento. Úselo para el administrador de inquilinos de M365, el administrador de Office 365, los usuarios de Azure AD, el administrador global, la configuración de inquilinos o la automatización de Microsoft 365.
