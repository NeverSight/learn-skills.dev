Администрирование клиентов Microsoft 365 для глобальных администраторов. Автоматизируйте настройку клиента M365, задачи администрирования Office 365, управление пользователями Azure AD, настройку Exchange Online, администрирование Teams и политики безопасности. Создавайте сценарии PowerShell для массовых операций, политик условного доступа, управления лицензиями и отчетов о соответствии требованиям. Используйте для менеджера клиентов M365, администратора Office 365, пользователей Azure AD, глобального администратора, конфигурации клиента или автоматизации Microsoft 365.
