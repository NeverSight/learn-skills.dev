Administration des locataires Microsoft 365 pour les administrateurs globaux. Automatisez la configuration des locataires M365, les tâches d'administration d'Office 365, la gestion des utilisateurs Azure AD, la configuration d'Exchange Online, l'administration de Teams et les politiques de sécurité. Générez des scripts PowerShell pour les opérations en masse, les politiques d'accès conditionnel, la gestion des licences et les rapports de conformité. À utiliser pour le gestionnaire de locataires M365, l'administrateur Office 365, les utilisateurs Azure AD, l'administrateur global, la configuration de locataire ou l'automatisation Microsoft 365.
