Automatisierung der Codeüberprüfung für TypeScript, JavaScript, Python, Go, Swift, Kotlin. Analysiert PRs auf Komplexität und Risiko, überprüft die Codequalität auf SOLID-Verstöße und Code-Smells und erstellt Überprüfungsberichte. Wird zum Überprüfen von Pull-Anfragen, zum Analysieren der Codequalität, zum Identifizieren von Problemen und zum Erstellen von Überprüfungschecklisten verwendet.
