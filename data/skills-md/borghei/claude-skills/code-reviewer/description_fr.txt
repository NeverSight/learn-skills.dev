Automatisation de la révision de code pour TypeScript, JavaScript, Python, Go, Swift, Kotlin. Analyse les PR pour en déterminer la complexité et les risques, vérifie la qualité du code pour détecter les violations SOLID et les odeurs de code, génère des rapports d'examen. À utiliser lors de l'examen des demandes d'extraction, de l'analyse de la qualité du code, de l'identification des problèmes et de la génération de listes de contrôle de révision.
