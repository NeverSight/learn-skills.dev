Automatización de revisión de código para TypeScript, JavaScript, Python, Go, Swift, Kotlin. Analiza las relaciones públicas en busca de complejidad y riesgo, verifica la calidad del código en busca de violaciones SÓLIDAS y olores de código, genera informes de revisión. Úselo al revisar solicitudes de extracción, analizar la calidad del código, identificar problemas y generar listas de verificación de revisión.
