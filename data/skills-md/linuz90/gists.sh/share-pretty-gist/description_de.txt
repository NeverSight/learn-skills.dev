Teilen Sie Text, Code, Berichte oder andere Inhalte extern über einen wunderschön gerenderten GitHub Gist. Verwenden Sie diese Fähigkeit IMMER, wenn Sie einen Kerninhalt erstellen – unabhängig davon, ob der Benutzer „Kerninhalt erstellen“, „Kerninhalt erstellen“, „Als Kerninhalt teilen“, „Teilen Sie dies“, „Teilen Sie diesen Code“ oder eine Variation mit dem Wort „Kerninhalt“ sagt. Dies ist die Standardmethode zum Erstellen von Gists. Verwenden Sie es auch proaktiv, wenn Sie Inhalte teilen, und ein Kerninhalt wäre das beste Format.
