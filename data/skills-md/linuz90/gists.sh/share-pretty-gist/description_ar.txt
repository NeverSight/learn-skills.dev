قم بمشاركة النص أو التعليمات البرمجية أو التقارير أو أي محتوى آخر خارجيًا عبر GitHub Gist الذي تم تقديمه بشكل جميل. استخدم هذه المهارة دائمًا عند إنشاء جوهر - سواء قال المستخدم "أنشئ جوهرًا" أو "اصنع جوهرًا" أو "شارك كجوهر" أو "شارك هذا" أو "شارك هذا الرمز" أو أي شكل آخر يتضمن كلمة "جوهر". هذه هي الطريقة الافتراضية لإنشاء Gists. استخدم أيضًا بشكل استباقي عند مشاركة المحتوى وسيكون Gist هو التنسيق الأفضل.
