Cree interfaces frontend de nivel de producción siguiendo el sistema de diseño e identidad de marca Dayuse. Esta habilidad debe usarse cuando el usuario solicita "crear una página", "crear un componente", "diseñar una interfaz de usuario", "crear una interfaz", "crear una página de destino", "crear un panel" o cualquier tarea de desarrollo de interfaz para Dayuse. Aplica la paleta de colores oficial de Dayuse, la tipografía (Manrope + MaisonNeue), las convenciones de radio de borde, el sistema de sombras y los patrones de animación.
