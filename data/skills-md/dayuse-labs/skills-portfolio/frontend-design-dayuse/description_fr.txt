Créez des interfaces frontales de qualité production en suivant l'identité de la marque et le système de conception Dayuse. Cette compétence doit être utilisée lorsque l'utilisateur demande de « créer une page », « créer un composant », « concevoir une interface utilisateur », « créer une interface », « créer une page de destination », « créer un tableau de bord » ou toute tâche de développement front-end pour Dayuse. Applique la palette de couleurs officielle Dayuse, la typographie (Manrope + MaisonNeue), les conventions de rayon de bordure, le système d'ombres et les motifs d'animation.
