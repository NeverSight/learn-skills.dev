Verwenden Sie diese Option, wenn Sie SDKs für mehrere Sprachen aus einer einzigen OpenAPI-Spezifikation oder mehrere SDK-Varianten aus verschiedenen Quellen generieren. Behandelt die Multi-Target-Konfiguration von Workflow.yaml, gen.yaml pro Sprache und die Monorepo-Struktur. Löst auf „mehrere SDKs“, „mehrsprachiges SDK“, „SDK für jede Sprache“, „mehrere Ziel-SDK“, „SDK-Monorepo“ und „SDKs generieren für“ aus.
