Используйте при создании SDK для нескольких языков из одной спецификации OpenAPI или нескольких вариантов SDK из разных источников. Охватывает многоцелевую конфигурацию workflow.yaml, gen.yaml для каждого языка, структуру монорепо. Триггеры по «несколько SDK», «многоязычный SDK», «SDK для каждого языка», «многоцелевой SDK», «монорепозиторий SDK», «создать SDK для».
