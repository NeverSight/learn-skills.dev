Genera lista de materiales (BOM) a partir de descripciones de proyectos para proyectos Arduino/ESP32/RP2040. Úselo cuando el usuario necesite listas de componentes, listas de compra de piezas, estimaciones de costos o pregunte "qué piezas necesito". Genera listas de materiales formateadas con números de piezas, cantidades, proveedores (DigiKey, Mouser, Amazon, AliExpress) y advertencias de compatibilidad. Ejecute scripts/generate_bom.py para exportar xlsx/csv.
