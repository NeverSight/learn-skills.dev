swarm-tools モノリポジトリのバージョン バンプと npm リリースを処理します (opencode-swarm-plugin、
クロードコード-swarm-plugin、swarm-mail、swarm-queue)。次の場合に使用します: 変更セットの作成、バンピング
バージョン、リリースの準備、リリース ステータスの確認、公開失敗のデバッグ、検証
npm パッケージ、またはリリース PR のマージ。
