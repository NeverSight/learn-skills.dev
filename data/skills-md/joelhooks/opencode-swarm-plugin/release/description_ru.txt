Обрабатывает обновления версий и выпуски npm для монорепозитория swarm-tools (opencode-swarm-plugin,
claude-code-swarm-plugin, swarm-mail, swarm-queue). Используйте при: создании наборов изменений, обновлении
версии, подготовка выпусков, проверка статуса выпуска, отладка ошибок публикации, проверка
пакеты npm или объединение PR-релизов.
Триггеры: «выпуск», «публикация», «набор изменений», «обновление версии», «отправка», «новая версия»,
«создать выпуск», «проверить npm», «проверить публикацию», «/release»
