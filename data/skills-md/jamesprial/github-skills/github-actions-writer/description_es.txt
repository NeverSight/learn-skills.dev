Esta habilidad debe usarse cuando los usuarios necesitan crear, modificar, optimizar o solucionar problemas de flujos de trabajo de CI/CD de GitHub Actions. Úselo cuando los usuarios pregunten sobre la automatización de compilaciones, pruebas, implementaciones o cualquier tarea relacionada con GitHub Actions. Los desencadenantes incluyen solicitudes como "crear un flujo de trabajo de CI", "implementar en AWS", "automatizar pruebas", "configurar acciones de GitHub" o al depurar fallas en el flujo de trabajo.
