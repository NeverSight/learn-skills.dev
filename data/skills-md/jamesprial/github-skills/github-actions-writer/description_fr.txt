Cette compétence doit être utilisée lorsque les utilisateurs doivent créer, modifier, optimiser ou dépanner les flux de travail CI/CD GitHub Actions. À utiliser lorsque les utilisateurs posent des questions sur l'automatisation des builds, des tests, des déploiements ou de toute tâche liée aux actions GitHub. Les déclencheurs incluent des requêtes telles que « créer un workflow CI », « déployer sur AWS », « automatiser les tests », « configurer les actions GitHub » ou lors du débogage des échecs du workflow.
