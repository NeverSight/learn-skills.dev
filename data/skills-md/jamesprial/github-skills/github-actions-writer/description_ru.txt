Этот навык следует использовать, когда пользователям необходимо создавать, изменять, оптимизировать или устранять неполадки рабочих процессов GitHub Actions CI/CD. Используйте, когда пользователи спрашивают об автоматизации сборок, тестов, развертываний или любых задачах, связанных с GitHub Actions. Триггеры включают такие запросы, как «создать рабочий процесс CI», «развернуть на AWS», «автоматизировать тестирование», «настроить действия GitHub» или при отладке сбоев рабочего процесса.
