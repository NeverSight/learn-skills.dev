Cree y mantenga aplicaciones de línea de comandos Bash con el generador Bashly.
Úselo cuando los usuarios soliciten crear un proyecto Bashly, diseñar árboles de comandos, definir
banderas/argumentos/opciones en `bashly.yml`, genera scripts Bash desde Bashly
config, escriba parciales de comando/lib en la carpeta fuente de Bashly, o itere en
CLI existentes basadas en Bashly desde la idea hasta el script completo (por ejemplo: "esto es
un proyecto bashly" o "ayúdame a crear un script bash usando bashly").
