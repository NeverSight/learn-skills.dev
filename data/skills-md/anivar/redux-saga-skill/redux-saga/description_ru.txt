Лучшие практики, шаблоны и рекомендации по API Redux-Saga для создания, тестирования и отладки промежуточного программного обеспечения с побочными эффектами на основе генератора в приложениях Redux. Охватывает создателей эффектов, модель ветвления, каналы, тестирование с помощью redux-saga-test-plan, параллелизм, отмену и современную интеграцию Redux Toolkit. Базовая версия: redux-saga 1.4.2. Срабатывает при: файлах саги, импорте redux-saga, промежуточном программном обеспечении на основе генератора, упоминаниях «saga», «takeEvery», «takeLatest», «модели ветки» или «каналов».
