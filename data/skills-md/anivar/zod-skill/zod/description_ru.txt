Лучшие практики, шаблоны и рекомендации по API Zod v4 для проверки схемы, синтаксического анализа, обработки ошибок и вывода типов в приложениях TypeScript. Охватывает SafeParse, композицию объектов, уточнения, преобразования, кодеки, фирменные типы и миграцию v3 → v4. Базовый уровень: зод ^4.0.0. Срабатывает при: импорте zod, z.object, z.string, z.infer, SafeParse, упоминаниях «zod», «проверки схемы», «zod v4» или «z.enum».
