Рекомендации, шаблоны и рекомендации по API MSW (Mock Service Worker) v2 для имитации API в тестах и ​​разработке JavaScript/TypeScript. Охватывает проектирование обработчиков, настройку сервера, построение ответов, шаблоны тестирования, GraphQL и миграцию с версии 1 на версию 2. Базовый уровень: msw ^2.0.0. Срабатывает при: импорте msw, http.get, http.post, HttpResponse, setupServer, setupWorker,graphql.query, упоминаниях «msw», «mock service worker», «api-mock» или «msw v2».
