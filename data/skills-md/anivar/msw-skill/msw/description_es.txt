Mejores prácticas, patrones y guía de API de MSW (Mock Service Worker) v2 para la simulación de API en pruebas y desarrollo de JavaScript/TypeScript. Cubre el diseño de controladores, configuración del servidor, construcción de respuestas, patrones de prueba, GraphQL y migración de v1 a v2. Línea de base: msw ^2.0.0. Activadores de: importaciones de msw, http.get, http.post, HttpResponse, setupServer, setupWorker, graphql.query, menciones de "msw", "trabajador de servicio simulado", "api mocking" o "msw v2".
