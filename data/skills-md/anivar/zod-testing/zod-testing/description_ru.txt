Тестирование шаблонов для схем Zod с использованием Jest и Vitest. Охватывает тестирование правильности схемы, генерацию ложных данных, шаблоны утверждения ошибок, интеграционное тестирование с обработчиками API и формами, тестирование моментальных снимков с помощью z.toJSONSchema() и тестирование на основе свойств. Базовый уровень: зод ^4.0.0. Срабатывает при: тестовых файлах для схем Zod, импорте zod-schema-faker, упоминаниях «тестовой схемы», «теста схемы», «zod макета», «zod test» или шаблонов тестирования схемы.
