Лучшие практики Jest, шаблоны и рекомендации по API для тестирования JavaScript/TypeScript. Охватывает макетирование дизайна, асинхронное тестирование, сопоставление, макеты таймера, снимки, макет модуля, настройку и оптимизацию CI. Базовый уровень: только ^29.0.0 / ^30.0.0. Срабатывает при: импорте jest, описании, it, тесте, ожидании, jest.fn, jest.mock, jest.spyOn, упоминаниях «jest», «unit test», «test suite» или «mock».
