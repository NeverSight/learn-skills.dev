Genere, edite y componga imágenes utilizando el modelo de imagen Gemini 3 Pro de Google (Nano Banana Pro). Utilice esta habilidad cuando el usuario solicite crear imágenes, generar elementos visuales, editar fotografías, componer varias imágenes, crear logotipos, miniaturas, infografías, fotografías de productos o cualquier tarea de generación de imágenes. Admite conversión de texto a imagen, edición de imágenes, composición de múltiples imágenes (hasta 14 imágenes), refinamiento iterativo, control de relación de aspecto y generación de imágenes basada en la Búsqueda de Google para visualización de datos en tiempo real.
