Этот навык следует использовать, когда пользователь запрашивает «зашифровать резервную копию», «расшифровать файл .bep», «CLI для резервного копирования биткойнов», «кошелек резервного копирования» или ему необходимо зашифровать/расшифровать файлы резервных копий BSV с помощью CLI для резервного копирования биткойнов.
