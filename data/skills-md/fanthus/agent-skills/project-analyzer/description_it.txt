Utilizza questa competenza quando gli utenti desiderano comprendere una nuova base di codice, analizzare la struttura del progetto, identificare i punti di ingresso, comprendere i livelli e l'architettura del progetto, trovare file di configurazione, mappare le dipendenze, partecipare a un progetto sconosciuto o quando fanno domande come "aiutami a capire questo progetto", "qual è l'architettura", "da dove comincio a leggere questo codice", "spiega questa base di codice" o "analizza questo repository". Si attiva anche quando gli utenti caricano directory di progetto o chiedono informazioni sull'organizzazione del progetto.
