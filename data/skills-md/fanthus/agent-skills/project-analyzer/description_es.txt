Comprender y analizar rápidamente la arquitectura, estructura y organización de proyectos de software. Utilice esta habilidad cuando los usuarios quieran comprender una nueva base de código, analizar la estructura del proyecto, identificar puntos de entrada, comprender las capas y la arquitectura del proyecto, encontrar archivos de configuración, asignar dependencias, incorporarse a un proyecto desconocido o cuando hagan preguntas como "ayúdenme a comprender este proyecto", "cuál es la arquitectura", "dónde empiezo a leer este código", "explicar este código base" o "analizar este repositorio". También se activa cuando los usuarios cargan directorios de proyectos o preguntan sobre la organización del proyecto.
