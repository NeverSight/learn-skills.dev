Comprenez et analysez rapidement l’architecture, la structure et l’organisation des projets logiciels. Utilisez cette compétence lorsque les utilisateurs souhaitent comprendre une nouvelle base de code, analyser la structure du projet, identifier les points d'entrée, comprendre les couches et l'architecture du projet, rechercher des fichiers de configuration, cartographier les dépendances, se lancer dans un projet inconnu, ou lorsqu'ils posent des questions telles que « aidez-moi à comprendre ce projet », « quelle est l'architecture », « par où commencer à lire ce code », « expliquer cette base de code » ou « analyser ce référentiel ». Se déclenche également lorsque les utilisateurs téléchargent des répertoires de projet ou posent des questions sur l'organisation du projet.
