Verstehen und analysieren Sie schnell die Architektur, Struktur und Organisation von Softwareprojekten. Verwenden Sie diese Fähigkeit, wenn Benutzer eine neue Codebasis verstehen, die Projektstruktur analysieren, Einstiegspunkte identifizieren, Projektebenen und -architektur verstehen, Konfigurationsdateien finden, Abhängigkeiten zuordnen, in ein unbekanntes Projekt einsteigen möchten oder wenn sie Fragen stellen wie „Hilf mir, dieses Projekt zu verstehen“, „Was ist die Architektur“, „Wo fange ich an, diesen Code zu lesen“, „Erkläre diese Codebasis“ oder „Analysiere dieses Repository“. Wird auch ausgelöst, wenn Benutzer Projektverzeichnisse hochladen oder nach der Projektorganisation fragen.
