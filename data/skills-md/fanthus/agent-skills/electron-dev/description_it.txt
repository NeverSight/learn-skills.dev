Crea applicazioni desktop multipiattaforma con Electron utilizzando le migliori pratiche per sicurezza, prestazioni ed esperienza utente. Utilizza questa competenza quando sviluppi strumenti di sistema (file manager, strumenti per screenshot, app di produttività) o quando lavori con progetti Electron. I trigger includono richieste di creazione di app Electron, implementazione di operazioni sui file, funzionalità della barra delle applicazioni, gestione delle finestre, comunicazione IPC o ottimizzazione delle prestazioni di Electron. Supporta framework JavaScript, React e Vue vanilla con modelli di codice completi che incorporano best practice per sicurezza e prestazioni direttamente nei commenti.
