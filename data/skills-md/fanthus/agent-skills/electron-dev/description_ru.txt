Создавайте кроссплатформенные настольные приложения с помощью Electron, используя лучшие практики в области безопасности, производительности и удобства использования. Используйте этот навык при разработке системных инструментов (файловых менеджеров, инструментов создания снимков экрана, приложений для повышения производительности) или при работе с проектами Electron. Триггеры включают запросы на создание приложений Electron, реализацию файловых операций, функциональность панели задач, управление окнами, связь IPC или оптимизацию производительности Electron. Поддерживает базовые платформы JavaScript, React и Vue с комплексными шаблонами кода, которые внедряют лучшие практики безопасности и производительности непосредственно в комментарии.
