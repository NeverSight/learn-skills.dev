Cree, enriquezca, organice y realice un seguimiento de tickets con calidad de producción optimizados para que los ejecute un desarrollador o agente de IA sin ambigüedad. Independiente de la plataforma (Linear, Jira, GitHub Issues, Notion, Asana). Utilice esta habilidad siempre que el usuario mencione tickets, problemas, trabajos pendientes, planificación de sprints o gestión de proyectos, incluso si solo dice "crear una tarea", "qué sigue", "terminé con X" o "dividir esta función en tickets". Se activa cualquier señal relacionada con el flujo de trabajo de PM: iniciar, completar o planificar elementos de trabajo.
