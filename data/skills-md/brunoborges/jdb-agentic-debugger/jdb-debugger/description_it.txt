Eseguire il debug delle applicazioni Java in tempo reale utilizzando JDB (Java Debugger CLI). Collegati a JVM in esecuzione o avviane di nuove in JDB, imposta punti di interruzione, scorri il codice, ispeziona le variabili, analizza i thread e diagnostica le eccezioni. Da utilizzare durante il debug di programmi Java, l'analisi del comportamento di runtime o la diagnosi di problemi di produzione su JVM con JDWP abilitato.
