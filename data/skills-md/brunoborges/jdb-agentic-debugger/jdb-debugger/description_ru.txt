Отлаживайте приложения Java в режиме реального времени с помощью JDB (CLI Java Debugger). Подключайтесь к работающим JVM или запускайте новые под JDB, устанавливайте точки останова, пошагово просматривайте код, проверяйте переменные, анализируйте потоки и диагностируйте исключения. Используйте при отладке программ Java, изучении поведения во время выполнения или диагностике производственных проблем на JVM с включенным JDWP.
