À utiliser lorsque vous travaillez avec des opérations Supabase CLI, y compris la configuration de développement local, les migrations de bases de données, les fonctions Edge, le stockage, les secrets, la génération de type ou toute commande Supabase. Déclenchez des expressions telles que "projet supabase", "migration supabase", "déployer une fonction Edge", "pull schema", "login supabase", "lien supabase", "démarrage supabase", "générer des types à partir de supabase" ou toute tâche impliquant la CLI de la plateforme Supabase. Utilisez cette compétence même si l'utilisateur mentionne simplement Supabase en parallèle d'une autre tâche.
