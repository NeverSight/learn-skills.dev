Интерфейс Python для PROJ (библиотека картографических проекций и преобразований координат). Обрабатывает преобразования между различными системами координат (CRS) и выполняет геодезические расчеты (расстояние, площадь на эллипсоидах). Используйте для преобразований координат, преобразований CRS, геодезических расчетов, проекций UTM, преобразований координат GPS, вычислений эллипсоидных расстояний и операций с пространственной системой отсчета.
