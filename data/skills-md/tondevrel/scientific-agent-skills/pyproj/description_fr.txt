Interface Python vers PROJ (bibliothèque de projections cartographiques et de transformations de coordonnées). Gère les transformations entre différents systèmes de coordonnées de référence (CRS) et effectue des calculs géodésiques (distance, surface sur les ellipsoïdes). À utiliser pour les transformations de coordonnées, les conversions CRS, les calculs géodésiques, les projections UTM, les conversions de coordonnées GPS, les calculs de distance ellipsoïdale et les opérations du système de référence spatiale.
