Interfaccia Python per PROJ (libreria di proiezioni cartografiche e trasformazioni di coordinate). Gestisce le trasformazioni tra diversi sistemi di riferimento di coordinate (CRS) ed esegue calcoli geodetici (distanza, area sugli ellissoidi). Utilizzare per trasformazioni di coordinate, conversioni CRS, calcoli geodetici, proiezioni UTM, conversioni di coordinate GPS, calcoli di distanza ellissoidale e operazioni del sistema di riferimento spaziale.
