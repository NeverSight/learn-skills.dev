Interfaz Python para PROJ (biblioteca de proyecciones cartográficas y transformaciones de coordenadas). Maneja transformaciones entre diferentes Sistemas de Referencia de Coordenadas (CRS) y realiza cálculos geodésicos (distancia, área en elipsoides). Úselo para transformaciones de coordenadas, conversiones CRS, cálculos geodésicos, proyecciones UTM, conversiones de coordenadas GPS, cálculos de distancias elipsoidales y operaciones de sistemas de referencia espacial.
