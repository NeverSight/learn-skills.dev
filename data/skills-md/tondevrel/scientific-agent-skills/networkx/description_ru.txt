Пакет Python для создания, управления и изучения структуры, динамики и функций сложных сетей. Поддерживает различные типы графов (направленные, ненаправленные, мультиграфы) и имеет обширную библиотеку стандартных алгоритмов графов. Используется для сетевого анализа, теории графов, анализа социальных сетей, биологических сетей, инфраструктурных сетей, поиска пути, измерения центральности, обнаружения сообществ, графовых алгоритмов, кратчайших путей, PageRank, анализа связности и оптимизации маршрутизации.
