Libreria DataFrame incredibilmente veloce scritta in Rust. Dispone di un motore di query multi-thread, valutazione pigra e utilizzo efficiente della memoria tramite Apache Arrow. Progettato per l'elaborazione dati ad alte prestazioni su una singola macchina. Utilizzare per set di dati di grandi dimensioni (1 GB-100 GB+), trasformazioni rapide dei dati, elaborazione Parquet/CSV, pipeline di query complesse, operazioni efficienti in termini di memoria e quando la velocità è fondamentale (10-100 volte più veloce dei panda).
