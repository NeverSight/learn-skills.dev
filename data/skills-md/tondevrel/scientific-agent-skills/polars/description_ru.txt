Невероятно быстрая библиотека DataFrame, написанная на Rust. Имеет многопоточный механизм запросов, отложенную оценку и эффективное использование памяти через Apache Arrow. Предназначен для высокопроизводительной обработки данных на одной машине. Используйте для больших наборов данных (1–100 ГБ и более), быстрого преобразования данных, обработки Parquet/CSV, сложных конвейеров запросов, операций с эффективным использованием памяти, а также когда скорость имеет решающее значение (в 10–100 раз быстрее, чем у pandas).
