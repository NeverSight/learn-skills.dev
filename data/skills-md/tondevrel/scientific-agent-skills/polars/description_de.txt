Blitzschnelle DataFrame-Bibliothek, geschrieben in Rust. Verfügt über eine Multithread-Abfrage-Engine, verzögerte Auswertung und effiziente Speichernutzung über Apache Arrow. Entwickelt für die Hochleistungsdatenverarbeitung auf einer einzigen Maschine. Verwendung für große Datensätze (1 GB–100 GB+), schnelle Datentransformationen, Parquet/CSV-Verarbeitung, komplexe Abfragepipelines, speichereffiziente Vorgänge und wenn Geschwindigkeit entscheidend ist (10–100 x schneller als Pandas).
