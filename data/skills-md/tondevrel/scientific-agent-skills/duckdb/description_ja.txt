分析用インプロセス SQL データベース管理システム。高速分析クエリ (OLAP) 用に設計されています。 Python のデータ エコシステム (Pandas、NumPy、Arrow、Polars) との高い相互運用性。取り込みステップなしでファイル (CSV、Parquet、JSON) を直接クエリすることをサポートします。 Pandas/Polars データに対する複雑な SQL クエリ、大規模な Parquet/CSV ファイルの直接クエリ、さまざまなソースからのデータの結合、分析パイプライン、Excel には大きすぎるローカル データセット、中間データ ストレージ、ML 用の特徴量エンジニアリングに使用します。
