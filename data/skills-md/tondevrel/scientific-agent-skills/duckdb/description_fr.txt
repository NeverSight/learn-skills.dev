Un système de gestion de base de données SQL analytique en cours. Conçu pour les requêtes analytiques rapides (OLAP). Hautement interopérable avec l'écosystème de données Python (Pandas, NumPy, Arrow, Polars). Prend en charge l'interrogation de fichiers (CSV, Parquet, JSON) directement sans étape d'ingestion. À utiliser pour les requêtes SQL complexes sur les données Pandas/Polars, l'interrogation directe de gros fichiers Parquet/CSV, la jonction de données provenant de différentes sources, les pipelines analytiques, les ensembles de données locaux trop volumineux pour Excel, le stockage de données intermédiaire et l'ingénierie de fonctionnalités pour le ML.
