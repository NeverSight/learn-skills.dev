نظام إدارة قاعدة بيانات SQL تحليلي قيد التشغيل. مصممة للاستعلامات التحليلية السريعة (OLAP). قابلة للتشغيل المتبادل بشكل كبير مع النظام البيئي لبيانات Python (Pandas، NumPy، Arrow، Polars). يدعم الاستعلام عن الملفات (CSV، Parquet، JSON) مباشرة بدون خطوة استيعاب. يُستخدم لاستعلامات SQL المعقدة بشأن بيانات Pandas/Polars، والاستعلام عن ملفات Parquet/CSV الكبيرة مباشرة، وربط البيانات من مصادر مختلفة، وخطوط الأنابيب التحليلية، ومجموعات البيانات المحلية الكبيرة جدًا بالنسبة لـ Excel، وتخزين البيانات المتوسطة وهندسة الميزات لتعلم الآلة.
