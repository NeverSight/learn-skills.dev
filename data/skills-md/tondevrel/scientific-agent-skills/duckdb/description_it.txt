Un sistema di gestione di database SQL analitico in-process. Progettato per query analitiche veloci (OLAP). Altamente interoperabile con l'ecosistema di dati di Python (Pandas, NumPy, Arrow, Polars). Supporta l'esecuzione di query di file (CSV, Parquet, JSON) direttamente senza una fase di acquisizione. Utilizzare per query SQL complesse su dati Pandas/Polars, interrogando direttamente file Parquet/CSV di grandi dimensioni, unendo dati da diverse origini, pipeline analitiche, set di dati locali troppo grandi per Excel, archiviazione di dati intermedi e ingegneria delle funzionalità per ML.
