Ein analytisches In-Process-SQL-Datenbankverwaltungssystem. Entwickelt für schnelle analytische Abfragen (OLAP). Hohe Interoperabilität mit dem Datenökosystem von Python (Pandas, NumPy, Arrow, Polars). Unterstützt die direkte Abfrage von Dateien (CSV, Parquet, JSON) ohne Aufnahmeschritt. Verwendung für komplexe SQL-Abfragen zu Pandas/Polars-Daten, direkte Abfrage großer Parquet/CSV-Dateien, Zusammenführung von Daten aus verschiedenen Quellen, analytische Pipelines, lokale Datensätze, die für Excel zu groß sind, Zwischendatenspeicherung und Feature-Engineering für ML.
