Un sistema analítico de gestión de bases de datos SQL en proceso. Diseñado para consultas analíticas rápidas (OLAP). Altamente interoperable con el ecosistema de datos de Python (Pandas, NumPy, Arrow, Polars). Admite la consulta de archivos (CSV, Parquet, JSON) directamente sin un paso de ingesta. Úselo para consultas SQL complejas sobre datos de Pandas/Polars, consulta directa de archivos Parquet/CSV de gran tamaño, unión de datos de diferentes fuentes, canales analíticos, conjuntos de datos locales demasiado grandes para Excel, almacenamiento de datos intermedio e ingeniería de funciones para ML.
