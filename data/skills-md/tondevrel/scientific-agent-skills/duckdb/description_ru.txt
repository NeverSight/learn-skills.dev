Аналитическая внутрипроцессная система управления базами данных SQL. Предназначен для быстрых аналитических запросов (OLAP). Высокая совместимость с экосистемой данных Python (Pandas, NumPy, Arrow, Polars). Поддерживает запросы к файлам (CSV, Parquet, JSON) напрямую, без этапа приема. Используйте для сложных SQL-запросов к данным Pandas/Polars, прямых запросов к большим файлам Parquet/CSV, объединения данных из разных источников, аналитических конвейеров, локальных наборов данных, слишком больших для Excel, промежуточного хранения данных и проектирования функций для ML.
