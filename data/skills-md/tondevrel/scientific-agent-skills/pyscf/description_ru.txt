Подробное руководство по PySCF — платформе химического моделирования на основе Python. Используйте для расчетов квантовой химии ab initio, включая Хартри-Фока, DFT, MP2, CCSD, оптимизацию геометрии, возбужденные состояния и молекулярные свойства. Стандартная библиотека для расчетов электронной структуры.
