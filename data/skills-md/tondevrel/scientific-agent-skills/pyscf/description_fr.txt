Guide complet pour PySCF - Simulations of Chemistry Framework basées sur Python. À utiliser pour les calculs de chimie quantique ab initio, notamment Hartree-Fock, DFT, MP2, CCSD, l'optimisation géométrique, les états excités et les propriétés moléculaires. Bibliothèque standard de l'industrie pour les calculs de structure électronique.
