حزمة بايثون مفيدة للكيمياء (الكيمياء الفيزيائية/التحليلية/غير العضوية بشكل أساسي). وتشمل الميزات موازنة التفاعلات الكيميائية، والحركية الكيميائية (تكامل ODE)، والتوازن الكيميائي، وحسابات القوة الأيونية، والتعامل مع الوحدة. يُستخدم عند التعامل مع المعادلات الكيميائية، أو موازنة التفاعل، أو النمذجة الحركية، أو حسابات التوازن، أو الأنواع، أو حسابات الأس الهيدروجيني، أو القوة الأيونية، أو معاملات النشاط، أو تحليل الصيغة الكيميائية.
