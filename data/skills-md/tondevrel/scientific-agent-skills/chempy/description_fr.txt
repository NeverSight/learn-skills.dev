Un package Python utile pour la chimie (principalement chimie physique/analytique/inorganique). Les fonctionnalités incluent l'équilibrage des réactions chimiques, la cinétique chimique (intégration ODE), les équilibres chimiques, les calculs de force ionique et la manipulation des unités. À utiliser lorsque vous travaillez avec des équations chimiques, l'équilibrage des réactions, la modélisation cinétique, les calculs d'équilibre, la spéciation, les calculs de pH, la force ionique, les coefficients d'activité ou l'analyse de formules chimiques.
