Une bibliothèque graphique interactive de haut niveau pour Python. Idéal pour les visualisations Web, les tracés 3D et les tableaux de bord interactifs complexes. Construit sur plotly.js, il permet aux utilisateurs de zoomer, de faire un panoramique et de survoler des points de données dans un environnement basé sur un navigateur. À utiliser pour les graphiques interactifs, les applications Web, les blocs-notes Jupyter, la visualisation de données 3D, les cartes géographiques, les graphiques financiers, les animations, l'analyse de séries chronologiques et la création de tableaux de bord prêts pour la production avec Dash.
