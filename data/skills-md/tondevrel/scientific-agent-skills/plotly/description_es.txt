Una biblioteca de gráficos interactivos de alto nivel para Python. Ideal para visualizaciones basadas en web, gráficos 3D y paneles interactivos complejos. Construido sobre plotly.js, permite a los usuarios hacer zoom, desplazarse y desplazarse sobre puntos de datos en un entorno basado en navegador. Úselo para gráficos interactivos, aplicaciones web, cuadernos Jupyter, visualización de datos 3D, mapas geográficos, gráficos financieros, animaciones, análisis de series temporales y creación de paneles listos para producción con Dash.
