Une bibliothèque de visualisation de données Python basée sur matplotlib. Il fournit une interface de haut niveau pour dessiner des graphiques statistiques attrayants et informatifs. Idéal pour explorer les relations entre les variables et visualiser les distributions. Utilisation pour la visualisation de données statistiques, l'analyse exploratoire de données (EDA), les tracés de relations, les diagrammes de distribution, les comparaisons catégorielles, la visualisation de régression, les cartes thermiques, les cartes de cluster et la création de graphiques statistiques de qualité publication à partir de Pandas DataFrames.
