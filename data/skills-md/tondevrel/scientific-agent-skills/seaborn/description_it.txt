Una libreria di visualizzazione dati Python basata su matplotlib. Fornisce un'interfaccia di alto livello per disegnare grafici statistici accattivanti e informativi. Ottimo per esplorare le relazioni tra variabili e visualizzare le distribuzioni. Utilizzare per la visualizzazione di dati statistici, l'analisi esplorativa dei dati (EDA), grafici di relazione, grafici di distribuzione, confronti categoriali, visualizzazione di regressione, mappe di calore, mappe di cluster e creazione di grafici statistici di qualità di pubblicazione da Pandas DataFrames.
