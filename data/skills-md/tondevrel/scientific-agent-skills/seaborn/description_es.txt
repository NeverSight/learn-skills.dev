Una biblioteca de visualización de datos de Python basada en matplotlib. Proporciona una interfaz de alto nivel para dibujar gráficos estadísticos atractivos e informativos. Excelente para explorar relaciones entre variables y visualizar distribuciones. Úselo para visualización de datos estadísticos, análisis de datos exploratorios (EDA), gráficos de relaciones, gráficos de distribución, comparaciones categóricas, visualización de regresión, mapas de calor, mapas de conglomerados y creación de gráficos estadísticos con calidad de publicación a partir de Pandas DataFrames.
