Библиотека визуализации данных Python, основанная на matplotlib. Он предоставляет высокоуровневый интерфейс для рисования привлекательных и информативных статистических графиков. Отлично подходит для изучения связей между переменными и визуализации распределений. Используйте для визуализации статистических данных, исследовательского анализа данных (EDA), графиков отношений, графиков распределения, категориальных сравнений, визуализации регрессии, тепловых карт, кластерных карт и создания статистической графики публикационного качества из Pandas DataFrames.
