La biblioteca fundamental para crear visualizaciones estáticas, animadas e interactivas en Python. Altamente personalizable y el estándar de la industria para figuras con calidad de publicación. Úselo para trazado 2D, visualización de datos científicos, mapas de calor, contornos, campos vectoriales, figuras de paneles múltiples, trazados con formato LaTeX, herramientas de visualización personalizadas y trazado desde matrices NumPy o Pandas DataFrames.
