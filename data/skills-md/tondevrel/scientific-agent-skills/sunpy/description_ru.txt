Разработанный сообществом бесплатный пакет программного обеспечения с открытым исходным кодом для физики Солнца. Предоставляет инструменты для поиска и загрузки данных, преобразования координат, характерные для физики Солнца, а также мощную обработку изображений с помощью объекта «Карта». Используйте при работе с солнечными данными, солнечными изображениями (EUV, магнитограммы, белый свет), солнечными координатами (гелиопроекционные, гелиографические), поиском данных Фидо, солнечными временными рядами, дифференциальным вращением, подгонкой конечностей или многоинструментальным солнечным анализом (AIA, HMI, GOES).
