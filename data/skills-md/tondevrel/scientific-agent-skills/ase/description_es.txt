Entorno de simulación atómica: un conjunto de herramientas para configurar, manipular, ejecutar, visualizar y analizar simulaciones atomísticas. Actúa como una interfaz universal entre Python y numerosos códigos de dinámica molecular y química cuántica. Úselo para construir estructuras atómicas, optimización de geometría, simulaciones de dinámica molecular, búsquedas de estados de transición (NEB), conversión de formatos de archivos (CIF, XYZ, POSCAR, PDB), cálculos de propiedades electrónicas (DOS, estructuras de bandas) y automatización de flujos de trabajo de simulación con códigos DFT/MD como VASP, GPAW, Quantum ESPRESSO, LAMMPS.
