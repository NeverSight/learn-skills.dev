Projet open source pour faciliter le travail avec les données géospatiales en python. Étend les types de données utilisés par les pandas pour permettre des opérations spatiales sur les types géométriques. Construit sur Shapely, Fiona et Pyproj. Utilisation pour lire et écrire des formats spatiaux (Shapefile, GeoJSON, GeoPackage, KML), effectuer des jointures spatiales, des transformations de systèmes de coordonnées (reprojection), des analyses géométriques (tampons, centroïdes, coques convexes), une cartographie thématique (cartes Choroplèthe), le calcul de relations spatiales (contient, chevauchements, touches, à l'intérieur), l'utilisation de données OpenStreetMap ou de données vectorielles dérivées de satellites.
