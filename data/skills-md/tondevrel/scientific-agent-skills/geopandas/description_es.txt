Proyecto de código abierto para facilitar el trabajo con datos geoespaciales en Python. Extiende los tipos de datos utilizados por pandas para permitir operaciones espaciales en tipos geométricos. Construido sobre Shapely, Fiona y Pyproj. Úselo para leer y escribir formatos espaciales (Shapefile, GeoJSON, GeoPackage, KML), realizar uniones espaciales, transformaciones de sistemas de coordenadas (reproyección), análisis geométrico (búferes, centroides, cascos convexos), mapeo temático (mapas de coropletas), calcular relaciones espaciales (contiene, superposiciones, toques, dentro), trabajar con datos de OpenStreetMap o datos vectoriales derivados de satélites.
