Eine chemische Toolbox, die die vielen Sprachen chemischer Daten beherrscht. Unterstützt über 110 Formate und bietet Tools für die Konvertierung, 3D-Strukturgenerierung, molekulare Suche (SMARTS) und Kraftfeldberechnungen. Verwendung für die Konvertierung chemischer Dateiformate (SDF, PDB, SMILES, CIF, Gaussian), 3D-Koordinatengenerierung aus 2D-Strukturen, Substruktursuche mit SMARTS-Mustern, molekulare Docking-Vorbereitung, Kraftfeldminimierung (UFF, GAFF, MMFF94), molekulare Fingerabdrücke und Tanimoto-Koeffizienten sowie Stapelverarbeitung chemischer Datenbanken.
