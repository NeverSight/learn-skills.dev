Набор химических инструментов, предназначенный для использования многих языков химических данных. Поддерживает более 110 форматов и предоставляет инструменты для преобразования, создания 3D-структур, молекулярного поиска (SMARTS) и расчета силового поля. Используется для преобразования химических форматов файлов (SDF, PDB, SMILES, CIF, Gaussian), генерации 3D-координат из 2D-структур, поиска субструктур с помощью шаблонов SMARTS, подготовки молекулярной стыковки, минимизации силового поля (UFF, GAFF, MMFF94), молекулярных отпечатков пальцев и коэффициентов Танимото, а также пакетной обработки химических баз данных.
