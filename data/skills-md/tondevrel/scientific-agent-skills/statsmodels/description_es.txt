Modelado estadístico avanzado y prueba de hipótesis. Complementario al módulo de estadísticas de SciPy, proporciona clases y funciones para la estimación de muchos modelos estadísticos diferentes, así como para realizar pruebas estadísticas y exploración de datos estadísticos. Úselo para regresión lineal, GLM, análisis de series temporales, ANOVA, análisis de supervivencia, inferencia causal y prueba de hipótesis estadísticas. Cargue cuando trabaje con OLS, WLS, regresión logística, regresión de Poisson, ARIMA, SARIMAX, diagnóstico estadístico, valores p, intervalos de confianza o análisis estadístico estilo R.
