Manipulation et analyse d'objets géométriques planaires. Basé sur la bibliothèque GEOS largement déployée. Fournit des structures de données pour les points, les courbes et les surfaces, ainsi que des algorithmes standardisés pour les opérations géométriques. À utiliser pour les opérations de géométrie 2D, les relations spatiales, les opérations de théorie des ensembles (intersection, union, différence), les requêtes de point dans le polygone, les calculs géométriques (surface, distance, centroïde), la mise en mémoire tampon, la simplification des géométries, le référencement linéaire et le nettoyage des géométries invalides. Indispensable pour les opérations SIG, l'analyse spatiale et les calculs géométriques.
