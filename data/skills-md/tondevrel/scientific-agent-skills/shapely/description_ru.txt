Манипулирование и анализ плоских геометрических объектов. На основе широко распространенной библиотеки GEOS. Предоставляет структуры данных для точек, кривых и поверхностей, а также стандартизированные алгоритмы для геометрических операций. Используется для операций с двумерной геометрией, пространственных отношений, теоретико-множественных операций (пересечение, объединение, разность), запросов «точка в многоугольнике», геометрических вычислений (площадь, расстояние, центроид), буферизации, упрощения геометрии, линейных ссылок и очистки недопустимой геометрии. Необходим для операций ГИС, пространственного анализа и геометрических вычислений.
