Guía completa para MDAnalysis: la biblioteca de Python para analizar trayectorias de dinámica molecular. Úselo para carga de trayectoria, cálculos RMSD/RMSF, análisis de distancia/ángulo/diédrico, selecciones de átomos, enlaces de hidrógeno, área de superficie accesible a solventes, análisis de estructura de proteínas, análisis de membrana e integración con Biopython. Esencial para el análisis de simulación MD.
