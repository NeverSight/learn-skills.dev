Полное руководство по MDAnaанализу — библиотеке Python для анализа траекторий молекулярной динамики. Используется для загрузки траектории, расчетов RMSD/RMSF, анализа расстояний/углов/двугранников, выбора атомов, водородных связей, площади поверхности, доступной растворителю, анализа структуры белка, мембранного анализа и интеграции с Biopython. Необходим для анализа моделирования МД.
