قم بدمج منصة Mem0 في تطبيقات الذكاء الاصطناعي للذاكرة المستمرة والتخصيص والبحث الدلالي. استخدم هذه المهارة عندما يذكر المستخدم "mem0"، أو "طبقة الذاكرة"، أو "تذكر تفضيلات المستخدم"، أو "السياق المستمر"، أو "التخصيص"، أو عندما يحتاج إلى إضافة ذاكرة طويلة المدى إلى برامج الدردشة الآلية أو الوكلاء أو تطبيقات الذكاء الاصطناعي. يغطي أدوات تطوير البرامج الخاصة بـ Python وTypeScript، وعمليات تكامل الإطارات (LangChain، وCrewAI، وVercel AI SDK، وOpenAI Agents SDK، وPipcat)، وواجهة برمجة تطبيقات النظام الأساسي الكاملة. يُستخدم حتى عندما لا يقول المستخدم "mem0" بشكل صريح ولكنه يصف الحاجة إلى ذاكرة المحادثة، أو الاحتفاظ بسياق المستخدم، أو استرجاع المعرفة عبر الجلسات.
