Para usuarios de plataformas (desarrolladores que utilizan modelos de flujo de trabajo en acomo para desarrollo). Guía el uso estándar de acomo CLI y API públicas (recuperación de modelos, operaciones de proceso, autenticación). La implementación interna de acomo en sí no está cubierta. Consulte esto cuando utilice la CLI de acomo, opere flujos de trabajo o procesos o verifique las definiciones de modelos.
