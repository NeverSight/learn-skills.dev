Utilizza questa capacità ogni volta che Claude ha bisogno di recuperare, leggere, estrarre o analizzare contenuto da un URL web. Converte le pagine Web in un markdown pulito ed efficiente in termini di token utilizzando il servizio markdown.new invece di recuperare HTML non elaborato. Si attiva quando l'utente fornisce un URL e desidera che il suo contenuto venga riepilogato, citato, analizzato, confrontato, estratto o elaborato. Si attiva anche quando Claude ha bisogno di leggere documentazione, post di blog, articoli, wiki, note di rilascio, log delle modifiche o qualsiasi contenuto di testo ospitato sul web. Anche se l'utente incolla semplicemente un URL senza istruzioni, utilizza questa abilità. NON utilizzare per file binari, pagine autenticate o endpoint API che restituiscono JSON/XML.
