Verwenden Sie diese Fähigkeit immer dann, wenn Claude Inhalte von einer Web-URL abrufen, lesen, extrahieren oder analysieren muss. Konvertiert Webseiten mithilfe des markdown.new-Dienstes in sauberes, tokeneffizientes Markdown, anstatt rohes HTML abzurufen. Wird ausgelöst, wenn der Benutzer eine URL angibt und möchte, dass deren Inhalt zusammengefasst, zitiert, analysiert, verglichen, extrahiert oder verarbeitet wird. Wird auch ausgelöst, wenn Claude Dokumentation, Blogbeiträge, Artikel, Wikis, Versionshinweise, Änderungsprotokolle oder andere im Internet gehostete Textinhalte lesen muss. Auch wenn der Benutzer nur eine URL ohne Anweisungen einfügt, nutzen Sie diese Fähigkeit. NICHT für Binärdateien, authentifizierte Seiten oder API-Endpunkte verwenden, die JSON/XML zurückgeben.
