Utilice esta habilidad siempre que Claude necesite buscar, leer, extraer o analizar contenido de una URL web. Convierte páginas web en rebajas limpias y con token eficiente utilizando el servicio markdown.new en lugar de buscar HTML sin formato. Se activa cuando el usuario proporciona una URL y desea resumir, citar, analizar, comparar, extraer o procesar su contenido. También se activa cuando Claude necesita leer documentación, publicaciones de blogs, artículos, wikis, notas de la versión, registros de cambios o cualquier contenido de texto alojado en la web. Incluso si el usuario simplemente pega una URL sin instrucciones, utilice esta habilidad. NO lo use para archivos binarios, páginas autenticadas o puntos finales API que devuelvan JSON/XML.
