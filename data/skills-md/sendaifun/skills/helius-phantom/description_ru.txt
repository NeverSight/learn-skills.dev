Создавайте интерфейсные приложения Solana с помощью Phantom Connect SDK и инфраструктуры Helius. Охватывает интеграцию React, React Native и браузерного SDK, подписание транзакций через Helius Sender, проксирование ключей API, шлюзование токенов, чеканку NFT, криптоплатежи, обновления в реальном времени и безопасную архитектуру внешнего интерфейса.
