البنية التحتية لـ Quicknode blockchain لـ Solana - نقاط نهاية RPC، DAS API (معيار الأصول الرقمية) لـ NFTs والأصول المضغوطة، تدفق Yellowstone gRPC، واجهة برمجة تطبيقات رسوم الأولوية، التدفقات (خطوط أنابيب البيانات في الوقت الفعلي)، Webhooks، تكامل Metis Jupiter Swap، تخزين IPFS، متجر القيمة الرئيسية، Admin API، وx402 الدفع لكل طلب RPC. يدعم أكثر من 80 سلسلة بما في ذلك Ethereum وPolygon وArbitrum وBase والمزيد. يُستخدم عند إعداد البنية الأساسية لـ Solana RPC، أو الاستعلام عن NFTs/الرموز المميزة/الأصول المضغوطة عبر DAS API، أو إنشاء تدفقات gRPC في الوقت الفعلي، أو تكوين خطوط أنابيب البيانات، أو تقدير رسوم الأولوية، أو دمج مقايضات Jupiter عبر Metis. المشغلات عند الإشارة إلى Quicknode، أو qn_methods، أو DAS API، أو getAssetsByOwner، أو searchAssets، أو Yellowstone، أو gRPC، أو Geyser، أو Streams، أو IPFS، أو Key-Value Store، أو qnLib، أو Metis، أو x402، أو Quicknode RPC.
