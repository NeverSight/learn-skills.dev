Programmgesteuerte Screenshot-Erfassung unter macOS. Finden Sie Fenster-IDs mit Swift CGWindowListCopyWindowInfo, steuern Sie Anwendungsfenster über AppleScript (Zoom, Scrollen, Auswählen) und erfassen Sie mit Screencapture. Verwenden Sie es, wenn Sie Screenshots automatisieren, Anwendungsfenster zur Dokumentation erfassen oder visuelle Multi-Shot-Workflows erstellen.
