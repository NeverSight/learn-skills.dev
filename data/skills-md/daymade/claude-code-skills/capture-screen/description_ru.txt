Программный снимок экрана в macOS. Находите идентификаторы окон с помощью Swift CGWindowListCopyWindowInfo, управляйте окнами приложений с помощью AppleScript (масштабирование, прокрутка, выбор) и делайте захват с помощью снимков экрана. Используйте при автоматизации снимков экрана, захвате окон приложений для документации или создании многоэтапных визуальных рабочих процессов.
