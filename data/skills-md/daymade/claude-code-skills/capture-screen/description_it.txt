Acquisizione di screenshot programmatica su macOS. Trova gli ID delle finestre con Swift CGWindowListCopyWindowInfo, controlla le finestre delle applicazioni tramite AppleScript (zoom, scorrimento, selezione) e cattura con la cattura dello schermo. Da utilizzare per automatizzare screenshot, acquisire finestre di applicazioni per la documentazione o creare flussi di lavoro visivi multi-scatto.
