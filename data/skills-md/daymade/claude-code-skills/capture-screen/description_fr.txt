Capture d'écran programmatique sur macOS. Recherchez les ID de fenêtre avec Swift CGWindowListCopyWindowInfo, contrôlez les fenêtres d'application via AppleScript (zoom, défilement, sélection) et capturez avec capture d'écran. À utiliser pour automatiser des captures d'écran, capturer des fenêtres d'application à des fins de documentation ou créer des flux de travail visuels multi-prises.
