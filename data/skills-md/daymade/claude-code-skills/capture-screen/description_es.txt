Captura de pantalla programática en macOS. Encuentre ID de ventanas con Swift CGWindowListCopyWindowInfo, controle las ventanas de aplicaciones mediante AppleScript (zoom, desplazamiento, selección) y capture con captura de pantalla. Úselo para automatizar capturas de pantalla, capturar ventanas de aplicaciones para documentación o crear flujos de trabajo visuales de múltiples tomas.
