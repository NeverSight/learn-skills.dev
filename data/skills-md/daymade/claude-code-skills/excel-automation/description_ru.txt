Создавайте, анализируйте и управляйте файлами Excel в macOS. Профессиональное форматирование с помощью openpyxl, сложный анализ xlsm с помощью stdlib zipfile+xml для финансовых моделей инвестиционного банка и управление окном Excel через AppleScript. Используйте при создании форматированных отчетов Excel, анализе финансовых моделей, с которыми openpyxl не может справиться, или автоматизации Excel в macOS.
