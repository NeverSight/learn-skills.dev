Collectez des données financières réelles sur toute entreprise américaine cotée en bourse à partir de sources publiques gratuites (yfinance). Sortie structurée JSON consommable par compétences financières en aval (modélisation DCF, analyse des comps, revue des bénéfices). Gère les données de marché (prix, actions, bêta), les données financières historiques (compte de résultat, flux de trésorerie, bilan), les entrées WACC et les estimations des analystes. À utiliser lorsque les utilisateurs demandent à collecter des données pour le téléscripteur, à obtenir des données financières pour une entreprise, à extraire des données de marché, à rassembler des entrées DCF ou à toute tâche nécessitant des données financières structurées avant analyse. Se déclenche également sur les données financières, les données d'entreprise, les données boursières.
