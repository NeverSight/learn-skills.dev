Raccogli dati finanziari reali per qualsiasi società statunitense quotata in borsa da fonti pubbliche gratuite (yfinance). Output JSON strutturato utilizzabile dalle competenze finanziarie a valle (modellazione DCF, analisi comps, revisione degli utili). Gestisce dati di mercato (prezzo, azioni, beta), dati finanziari storici (conto economico, flusso di cassa, stato patrimoniale), input WACC e stime degli analisti. Da utilizzare quando gli utenti richiedono di raccogliere dati per ticker, ottenere dati finanziari per l'azienda, estrarre dati di mercato, raccogliere input DCF o qualsiasi attività che richieda dati finanziari strutturati prima dell'analisi. Si attiva anche su dati finanziari, dati aziendali, dati azionari.
