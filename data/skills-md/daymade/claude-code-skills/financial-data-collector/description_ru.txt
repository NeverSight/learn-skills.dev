Собирайте реальные финансовые данные любой публичной компании США из бесплатных общедоступных источников (yfinance). Вывод структурированных расходных материалов JSON с учетом последующих финансовых навыков (моделирование DCF, конкурентный анализ, обзор доходов). Обрабатывает рыночные данные (цена, акции, бета), исторические финансовые показатели (отчет о прибылях и убытках, поток денежных средств, баланс), входные данные WACC и оценки аналитиков. Используйте, когда пользователи запрашивают сбор данных для тикера, получение финансовых показателей компании, получение рыночных данных, сбор входных данных DCF или любую задачу, требующую структурированных финансовых данных перед анализом. Также срабатывает по финансовым данным, данным компании, данным по акциям.
