Recopile datos financieros reales de cualquier empresa que cotice en bolsa en EE. UU. de fuentes públicas gratuitas (yfinance). Salida de consumibles JSON estructurados según las habilidades financieras posteriores (modelado DCF, análisis de comparaciones, revisión de ganancias). Maneja datos de mercado (precio, acciones, beta), estados financieros históricos (estado de resultados, flujo de efectivo, balance), entradas de WACC y estimaciones de analistas. Utilícelo cuando los usuarios soliciten recopilar datos para el teletipo, obtener datos financieros de la empresa, extraer datos de mercado, recopilar entradas de DCF o cualquier tarea que requiera datos financieros estructurados antes del análisis. También se activa sobre datos financieros, datos de empresas, datos bursátiles.
