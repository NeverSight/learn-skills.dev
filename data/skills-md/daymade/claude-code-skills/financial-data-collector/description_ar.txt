قم بجمع البيانات المالية الحقيقية لأي شركة أمريكية مساهمة عامة من مصادر عامة مجانية (yfinance). مخرجات JSON قابلة للاستهلاك من خلال المهارات المالية النهائية (نمذجة DCF، تحليل الشركات، مراجعة الأرباح). يتعامل مع بيانات السوق (السعر، الأسهم، بيتا)، البيانات المالية التاريخية (بيان الدخل، التدفق النقدي، الميزانية العمومية)، مدخلات المتوسط ​​المرجح لتكلفة رأس المال (WACC)، وتقديرات المحللين. يُستخدم عندما يطلب المستخدمون جمع البيانات للمؤشر، أو الحصول على البيانات المالية للشركة، أو سحب بيانات السوق، أو جمع مدخلات التدفقات النقدية المخصومة، أو أي مهمة تتطلب بيانات مالية منظمة قبل التحليل. يتم تشغيله أيضًا على البيانات المالية وبيانات الشركة وبيانات الأسهم.
