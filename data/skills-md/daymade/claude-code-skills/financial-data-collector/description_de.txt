Sammeln Sie echte Finanzdaten für jedes börsennotierte US-Unternehmen aus kostenlosen öffentlichen Quellen (yfinance). Ausgabe strukturierter JSON-Verbrauchsmaterialien für nachgelagerte Finanzkompetenzen (DCF-Modellierung, Comps-Analyse, Ertragsüberprüfung). Verarbeitet Marktdaten (Preis, Aktien, Beta), historische Finanzdaten (Gewinn- und Verlustrechnung, Cashflow, Bilanz), WACC-Eingaben und Analystenschätzungen. Wird verwendet, wenn Benutzer das Sammeln von Daten für den Ticker, das Abrufen von Finanzdaten für das Unternehmen, das Abrufen von Marktdaten, das Sammeln von DCF-Eingaben oder andere Aufgaben anfordern, die strukturierte Finanzdaten vor der Analyse erfordern. Triggert auch auf Finanzdaten, Unternehmensdaten und Aktiendaten.
