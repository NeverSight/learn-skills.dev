تحليل المنتج المتوازي متعدد المسارات مع قياس حساب وقت الاختبار عبر النماذج. إنشاء وكلاء متوازيين (فرق وكلاء Claude Code + Codex CLI) لاستكشاف المنتج من وجهات نظر متعددة، ثم تجميع النتائج في خطط تحسين قابلة للتنفيذ. يمكن استدعاء تحليل المنافسين لقياس الأداء التنافسي. يُستخدم عند "تدقيق المنتج"، أو "المراجعة الذاتية"، أو "التحقق من المنتج"، أو "تحليل منتجنا"، أو "تدقيق تجربة المستخدم"، أو "التحقق من المنتج".
