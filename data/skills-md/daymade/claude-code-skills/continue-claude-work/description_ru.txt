Восстановите действенный контекст из локальных артефактов сеанса `.claude` и продолжайте прерванную работу без запуска `claude --resume`. Этот навык следует использовать, когда пользователь предоставляет идентификатор сеанса Claude, просит продолжить предыдущую работу с локальной историей или хочет проверить файлы `.claude` перед возобновлением реализации.
