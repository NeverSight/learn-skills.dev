Recupera il contesto utilizzabile dagli artefatti della sessione `.claude` locale e continua il lavoro interrotto senza eseguire `claude --resume`. Questa abilità dovrebbe essere utilizzata quando l'utente fornisce un ID di sessione Claude, chiede di continuare il lavoro precedente dalla cronologia locale o desidera ispezionare i file ".claude" prima di riprendere l'implementazione.
