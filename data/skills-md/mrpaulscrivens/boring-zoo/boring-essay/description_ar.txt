اكتب المقالات والمقالات الصغيرة باستخدام صوت World Code الخاص بالمستخدم ونظرته للعالم.
استخدم هذه المهارة عندما يطلب المستخدم كتابة مقال، أو مقال صغير، أو نموذج قصير
النثر، أو الكتابة الكثيفة البصيرة حول موضوع ما. يتم تشغيله أيضًا عندما يقول المستخدم
"اكتب عن X"، "مقالة عن Y"، "مقال صغير عن Z"، "اكتب شيئًا عن"،
"استكشاف هذه الفكرة"، أو أي طلب إنشاء محتوى يبدو كذلك
عبر المجالات، النثر برأيه. إذا ذكر المستخدم المقالات، والكتابة عنها
المفاهيم، أو إنشاء محتوى قائم على المعرفة من أي نوع، استخدم هذه المهارة.
