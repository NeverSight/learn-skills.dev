Lorsque l'utilisateur souhaite écrire, réécrire ou améliorer tout type de copie marketing. Cela inclut la copie de page (page d'accueil, page de destination, page de vente, prix, fonctionnalité, à propos), la copie abrégée (biographies, slogans, propositions de valeur, one-liners, pitchs d'ascenseur), la copie publicitaire (annonces sociales, annonces de recherche, annonces display) et la microcopie (CTA, texte des boutons, étiquettes de formulaire, texte de notification). À utiliser lorsque l'utilisateur dit quelque chose comme « écrire une copie pour », « améliorer cette copie », « réécrire ceci », « aide pour le titre », « copie CTA », « proposition de valeur », « slogan », « bio », « copie publicitaire », « section héros », « au-dessus de la ligne de flottaison », « cette copie est faible », « rendre cela plus convaincant », « aidez-moi à décrire mon produit », « écrire ma biographie », « bio sur les réseaux sociaux » ou « à propos de moi ». World Code intégré : utilise votre voix, votre point culminant, votre méthode et votre croisement pour rédiger une copie qui vous appartient sans aucun doute. Pour une copie par e-mail, voir ennuyeux-email-sequence. Pour une copie contextuelle, voir ennuyeux-popup-cro. Pour l'édition ligne par ligne d'une copie existante, voir édition de copie ennuyeuse.
