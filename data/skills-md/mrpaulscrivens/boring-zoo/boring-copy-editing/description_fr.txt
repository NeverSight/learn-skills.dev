Lorsque l'utilisateur souhaite modifier, réviser ou améliorer une copie marketing existante. À utiliser lorsque l'utilisateur dit quelque chose comme « modifier cette copie », « revoir ma copie », « copier les commentaires », « relire », « peaufiner ceci », « améliorer cela », « resserrer cela », « cela se lit mal », « nettoyer ce texte », « trop verbeux », « affiner le message », « cela ne me ressemble pas », « quelque chose ne va pas avec ça », « rendre cela plus percutant », « c'est ennuyeux », ou "pouvez-vous réparer cette copie". Gère tout, du peaufinage rapide d'une phrase à la restructuration en profondeur d'une page complète. World Code intégré : modifications par rapport à vos règles vocales et à la cohérence du World Code. Pour rédiger une nouvelle copie à partir de zéro, voir rédaction ennuyeuse.
