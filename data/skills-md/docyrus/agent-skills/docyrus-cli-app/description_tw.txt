使用 Docyrus CLI (`docyrus`) 從終端與 Docyrus 平台互動。當使用者要求進行身份驗證、列出應用程式、查詢/管理資料記錄（「ds」）、管理開發應用程式資料來源架構物件（「studio」）、發送 API 請求、切換環境/租用戶/帳戶或透過「docyrus」命令列工具發現 OpenAPI 規範時使用。在涉及 docyrus CLI 命令、基於終端的 Docyrus 操作、「docyrus ds list」、「docyrus studio」、「docyrus discovery」、「docyrus auth」、「docyrus env」或基於 shell 的 Docyrus 工作流程的任務上觸發。
