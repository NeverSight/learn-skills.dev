Используйте Docyrus CLI («docyrus») для взаимодействия с платформой Docyrus из терминала. Используйте, когда пользователь запрашивает аутентификацию, список приложений, запрос или управление записями данных («ds»), управление объектами схемы источника данных приложения разработки («студия»), отправку запросов API, переключение сред/клиентов/учетных записей или обнаружение спецификаций OpenAPI с помощью инструмента командной строки docyrus. Запускается при выполнении задач, включающих команды docyrus CLI, операции Docyrus на базе терминала, docyrus ds list, docyrus studio, docyrus Discover, docyrus auth, docyrus env или рабочие процессы Docyrus на основе оболочки.
