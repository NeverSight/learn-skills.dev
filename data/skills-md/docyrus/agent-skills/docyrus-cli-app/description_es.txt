Utilice la CLI de Docyrus (`docyrus`) para interactuar con la plataforma Docyrus desde la terminal. Úselo cuando el usuario solicite autenticarse, enumerar aplicaciones, consultar/administrar registros de datos (`ds`), administrar objetos de esquema de fuente de datos de aplicaciones de desarrollo (`studio`), enviar solicitudes de API, cambiar entornos/inquilinos/cuentas o descubrir especificaciones de OpenAPI a través de la herramienta de línea de comandos `docyrus`. Activadores de tareas que involucran comandos CLI de docyrus, operaciones de Docyrus basadas en terminal, `docyrus ds list`, `docyrus studio`, `docyrus discover`, `docyrus auth`, `docyrus env` o flujos de trabajo de Docyrus basados ​​en shell.
