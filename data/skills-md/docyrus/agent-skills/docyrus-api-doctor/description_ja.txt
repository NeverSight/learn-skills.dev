Docyrus API の変更を行った後に実行して、バグ、パフォーマンスの問題、コード品質の問題を検出します。 Docyrus コレクション フック (.list、.get、.create、.update、.delete)、直接 RestApiClient 呼び出し、フィルター/計算/式/childQueries/ピボットを使用したクエリ ペイロード、または Docyrus データ ソースとの TanStack Query 統合を使用するコードを実装または変更するときに使用します。 Docyrus API ロジック、データのフェッチ、ミューテーション、クエリ ペイロードの構築に関連するタスクをトリガーします。
