قم بإعداد @personize/signal — وهو محرك إشعارات ذكي يقرر ما إذا كان، وماذا، ومتى، وكيفية إعلام كل شخص باستخدام تخصيص الذاكرة والإدارة. يرشدك خلال ربط مصادر الأحداث وتكوين قنوات التسليم وإعداد قواعد الإدارة واختبار محرك القرار. استخدم هذه المهارة عندما يريد المستخدم إنشاء إشعارات ذكية، أو تنبيهات مدعومة بالذكاء الاصطناعي، أو منع إجهاد الإشعارات، أو ملخصات يومية/أسبوعية، أو رسائل مخصصة، أو توجيه ذكي للإشعارات. يتم تشغيله أيضًا عندما يذكرون @personize/signal، أو تسجيل الإشعارات، أو ساعات الهدوء، أو إلغاء البيانات المكررة، أو توجيه القناة (البريد الإلكتروني مقابل Slack مقابل داخل التطبيق مقابل الرسائل القصيرة)، أو يريدون إشعارات تعرف متى تظل صامتة.
