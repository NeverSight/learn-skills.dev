يدير المبادئ التوجيهية والسياسات وأفضل الممارسات التنظيمية كمتغيرات حوكمة يمكن لجميع وكلاء الذكاء الاصطناعي الوصول إليها عبر SmartContext. استخدم هذه المهارة عندما يريد المستخدم إنشاء أو تحديث أو إدارة الإرشادات أو صوت العلامة التجارية أو سياسات الامتثال أو أدلة التشغيل أو برامج المقارنات الدولية أو أدلة قواعد المبيعات أو قواعد النغمة أو أي قواعد تنظيمية. يتم تشغيله أيضًا عندما يذكر المستخدم SmartGuidelines، أو متغيرات الإدارة، أو مزامنة GitOps للسياسات، أو مشاركة معرفة الفريق، أو قواعد وكيل الذكاء الاصطناعي، أو عندما يريدون أن تتبع جميع أدوات الذكاء الاصطناعي الخاصة بهم نفس السياسات. حتى لو قالوا فقط "إعداد القواعد" أو "إضافة سياسة"، فهذه هي المهارة الصحيحة.
