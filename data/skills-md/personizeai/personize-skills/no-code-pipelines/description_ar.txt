يُنشئ ملفات JSON لسير عمل n8n قابلة للاستيراد والتي تقوم بمزامنة البيانات بين Personalize وأكثر من 400 تطبيق. يُنتج مسارات عمل جاهزة للاستيراد لمزامنة الدُفعات، واستيعاب خطاف الويب، وإثراء الذكاء الاصطناعي لكل سجل، وتصدير البيانات - دون الحاجة إلى تعليمات برمجية. استخدم هذه المهارة عندما يريد المستخدم عمليات تكامل بدون تعليمات برمجية، أو سير عمل مرئي، أو أتمتة n8n، أو توصيل Personalize بـ HubSpot، أو Salesforce، أو Google Sheets، أو Slack، أو Postgres، أو أي تطبيق بدون كتابة التعليمات البرمجية. يتم تشغيله أيضًا عندما يذكرون "أتمتة سير العمل"، أو "المزامنة المجدولة بدون تعليمات برمجية"، أو "خط الأنابيب المرئي"، أو "الاتصال بـ [التطبيق]" ولا يريدون كتابة TypeScript.
