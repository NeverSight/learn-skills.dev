Personalize と 400 以上のアプリの間でデータを同期する、インポート可能な n8n ワークフロー JSON ファイルを生成します。バッチ同期、Webhook 取り込み、レコードごとの AI エンリッチメント、およびデータ エクスポートのための、すぐにインポートできるワークフローを生成します。コードは必要ありません。ユーザーがコードなしの統合、ビジュアル ワークフロー、n8n オートメーションを必要とする場合、またはコードを記述せずに Personalize を HubSpot、Salesforce、Google Sheets、Slack、Postgres などのアプリに接続する場合には、このスキルを使用します。また、ユーザーが「ワークフローの自動化」、「コードなしのスケジュールされた同期」、「ビジュアル パイプライン」、または「パーソナライズを [アプリ] に接続」について言及し、TypeScript を記述したくない場合にもトリガーされます。
