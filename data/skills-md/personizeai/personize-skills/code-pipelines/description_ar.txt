إنشاء ونشر وتكرار خطوط أنابيب وكيل الذكاء الاصطناعي الجاهزة للإنتاج باستخدام Trigger.dev وPersonize SDK. يتعامل مع دورة الحياة الكاملة: إجراء مقابلة مع المستخدم حول ما يريده، وتصميم المخطط والحوكمة، وكتابة رمز خط الأنابيب، ونشره، ومراقبة النتائج، والتكرار بناءً على التعليقات. يُنشئ مهام TypeScript للتسلسلات الصادرة، ومعالجة العملاء المتوقعين الواردين، ومعالجات الرد التحادثي، ومسارات الإثراء، ومراقبة إشارات الحساب - كل ذلك مدعوم بتخصيص الذاكرة وسياق الذكاء الاصطناعي والحوكمة. استخدم هذه المهارة عندما يريد شخص ما إنشاء وكيل ذكاء اصطناعي، أو سير عمل آلي، أو تسلسل بريد إلكتروني، أو حملة تنقيط، أو تواصل بارد، أو إثراء العملاء المحتملين، أو معالج الرد، أو مراقبة الحساب، أو أتمتة CRM، أو ملخص يومي، أو أي مسار دائم - سواء كانوا يقدمون المواصفات الفنية أو يصفون ما يريدون بلغة واضحة. قم أيضًا بتشغيل Trigger.dev، ومهام الخلفية، ومتابعات الجدولة الذاتية، وأتمتة GTM، أو "أنشئ لي وكيلًا..."، أو "أريد التشغيل التلقائي...".
