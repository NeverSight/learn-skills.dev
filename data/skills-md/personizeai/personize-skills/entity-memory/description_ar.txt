يقوم بتخزين واسترداد الذاكرة الدائمة حول السجلات - جهات الاتصال والشركات والموظفين والأعضاء والمزيد. يتعامل مع الحفظ (الفردي والدفعي مع استخراج الذكاء الاصطناعي لكل خاصية)، والاستدعاء الدلالي، وملخصات الكيان، وتصدير البيانات. استخدم هذه المهارة عندما يريد المستخدم تخزين البيانات، أو مزامنة السجلات من CRM أو قاعدة البيانات، أو الاستعلام أو ذاكرة البحث، أو تذكر ما هو معروف عن شخص أو شركة، أو تجميع السياق للتخصيص، أو استيراد بيانات CSV أو جداول البيانات، أو القيام بأي شيء يتضمن أساليب ذاكرة Personalize SDK (الحفظ، الاستدعاء، SmartRecall، SmartDigest، البحث، memorizeBatch). يُستخدم أيضًا عندما يذكر المستخدم جهات الاتصال أو العملاء المتوقعين أو الحسابات أو بيانات العميل أو خصائص الكيان.
