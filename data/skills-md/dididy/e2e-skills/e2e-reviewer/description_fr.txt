À utiliser lors de la révision, de l'audit ou de l'amélioration des spécifications de test E2E pour Playwright, Cypress ou Puppeteer : analyse de code statique des fichiers de test existants, sans diagnostic des échecs d'exécution. Déclencheurs sur "réviser mes tests", "auditer la qualité des tests", "trouver les tests faibles", "mes tests réussissent toujours mais manquent des bugs", "les tests réussissent CI mais manquent les régressions", "améliorer les tests de dramaturge", "améliorer les tests de cyprès", "vérifier les lacunes de couverture des tests". Détecte 14 anti-modèles : incompatibilité d'assertion de nom, Then manquant, avalement d'erreurs, assertions toujours transmises, pièges booléens, contournement conditionnel, requêtes DOM brutes, tests de rendu uniquement, scénarios en double, noms trompeurs, assertions trop larges, inversion de sujet, délais d'attente codés en dur, modèles floconneux (sélecteurs de position, simulations manquantes, courses d'animation) et violations YAGNI dans les objets de page.
