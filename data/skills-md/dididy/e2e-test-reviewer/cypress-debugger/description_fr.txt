À utiliser lorsque les tests Cypress ont réellement échoué et que vous devez diagnostiquer les échecs d'exécution - à partir de fichiers de rapport mochawesome ou JUnit, locaux ou CI. Déclencheurs sur "tests de débogage Cypress", "pourquoi les tests Cypress ont-ils échoué", "échec de Cypress CI", "échecs de test de cyprès feuilletés", "cypress a expiré pour réessayer", "les tests de cyprès réussissent localement mais échouent dans CI", "analyser cyprès/rapports". Classifie les échecs d'exécution en causes profondes (et non en analyse de code statique) et suggère des correctifs concrets.
