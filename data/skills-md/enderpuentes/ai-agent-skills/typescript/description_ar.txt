استخدم TypeScript لجافا سكريبت الآمن للنوع: الأنواع، والواجهات، والأدوية العامة، والتضييق، وtsconfig، والوحدات النمطية، والوضع الصارم. يُستخدم عند كتابة أو مراجعة TypeScript، أو تكوين tsconfig.json، أو تعريف الأنواع أو الواجهات، أو الأدوية العامة، أو استنتاج النوع، أو عندما يذكر المستخدم TypeScript، أو TS، أو التحقق من النوع.
