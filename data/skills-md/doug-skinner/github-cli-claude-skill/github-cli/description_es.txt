Ejecute comandos de GitHub CLI (gh) para administración de repositorios, problemas, solicitudes de extracción, flujos de trabajo y lanzamientos. Úselo cuando el usuario quiera interactuar con repositorios de GitHub, crear/administrar problemas o relaciones públicas, activar flujos de trabajo o realizar cualquier operación de GitHub. Cubre tanto los flujos de trabajo de desarrolladores individuales como los patrones de colaboración en equipo.
