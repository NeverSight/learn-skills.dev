Exécutez les commandes GitHub CLI (gh) pour la gestion du référentiel, les problèmes, les demandes d'extraction, les flux de travail et les versions. À utiliser lorsque l'utilisateur souhaite interagir avec les référentiels GitHub, créer/gérer des problèmes ou des PR, déclencher des flux de travail ou effectuer des opérations GitHub. Couvre à la fois les flux de travail des développeurs solo et les modèles de collaboration en équipe.
