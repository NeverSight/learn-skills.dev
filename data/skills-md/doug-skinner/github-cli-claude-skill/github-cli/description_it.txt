Esegui i comandi GitHub CLI (gh) per la gestione del repository, i problemi, le richieste pull, i flussi di lavoro e le versioni. Da utilizzare quando l'utente desidera interagire con i repository GitHub, creare/gestire problemi o PR, attivare flussi di lavoro o eseguire qualsiasi operazione GitHub. Copre sia i flussi di lavoro degli sviluppatori singoli che i modelli di collaborazione del team.
