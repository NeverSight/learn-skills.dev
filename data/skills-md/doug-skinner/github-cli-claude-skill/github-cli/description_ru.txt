Выполняйте команды GitHub CLI (gh) для управления репозиторием, проблемами, запросами на включение, рабочими процессами и выпусками. Используйте, когда пользователь хочет взаимодействовать с репозиториями GitHub, создавать задачи или запросы на обработку или управлять ими, запускать рабочие процессы или выполнять любые операции GitHub. Охватывает как рабочие процессы индивидуальных разработчиков, так и шаблоны командной работы.
