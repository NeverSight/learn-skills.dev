قم بتنفيذ أوامر GitHub CLI (gh) لإدارة المستودعات والمشكلات وطلبات السحب ومسارات العمل والإصدارات. يُستخدم عندما يريد المستخدم التفاعل مع مستودعات GitHub، أو إنشاء/إدارة المشكلات أو العلاقات العامة، أو تشغيل سير العمل، أو تنفيذ أي عمليات على GitHub. يغطي كلاً من سير عمل المطور الفردي وأنماط التعاون الجماعي.
