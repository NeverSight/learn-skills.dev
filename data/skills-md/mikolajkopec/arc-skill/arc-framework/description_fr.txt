À utiliser lors de l'écriture de code avec @arcote.tech/arc, de la création de commandes, d'événements, de vues, d'écouteurs, d'itinéraires ou de l'utilisation d'ArcContext, de types ArcElement, de hooks React (useQuery, useCommands), de composants de formulaire ou d'un serveur hôte Bun. À utiliser lorsque l'utilisateur mentionne le framework ARC, arcote ou narzedziadlatworcow.
