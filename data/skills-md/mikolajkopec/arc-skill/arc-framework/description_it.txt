Base di conoscenza del framework ARC per la creazione di applicazioni basate sul modello di business con CQRS e Event Sourcing. Da utilizzare quando si scrive codice con @arcote.tech/arc, si creano comandi, eventi, visualizzazioni, ascoltatori, percorsi o si lavora con ArcContext, tipi ArcElement, hook React (useQuery, useCommands), componenti modulo o server host Bun.
