Используйте при создании транзакций Cardano с помощью MeshJS SDK. Охватывает API MeshTxBuilder для отправки ADA, создания NFT и токенов, расходов по сценариям Plutus, ставок, голосования по управлению, регистрации DRep и шаблонов с несколькими подписями. Включает правильный порядок методов, выбор монет, расчет комиссий и устранение распространенных ошибок транзакций Cardano.
