أتمتة المتصفح لوكلاء الذكاء الاصطناعي. يُستخدم عندما يحتاج المستخدم إلى التفاعل مع مواقع الويب، أو التنقل بين الصفحات، أو ملء النماذج، أو النقر على الأزرار، أو التقاط لقطات شاشة، أو استخراج البيانات، أو اختبار تطبيقات الويب، أو أتمتة أي مهمة في المتصفح. تتضمن المشغلات "فتح موقع ويب"، أو "ملء نموذج"، أو "النقر فوق زر"، أو "التقاط لقطة شاشة"، أو "استخراج البيانات"، أو "اختبار تطبيق الويب هذا"، أو "تسجيل الدخول إلى موقع"، أو أي مهمة تتطلب تفاعلًا برمجيًا على الويب.
