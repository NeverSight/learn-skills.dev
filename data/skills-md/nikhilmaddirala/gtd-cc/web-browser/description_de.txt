Browserautomatisierung für KI-Agenten. Verwenden Sie es, wenn der Benutzer mit Websites interagieren, durch Seiten navigieren, Formulare ausfüllen, auf Schaltflächen klicken, Screenshots erstellen, Daten extrahieren, Web-Apps testen oder eine beliebige Browseraufgabe automatisieren muss. Zu den Auslösern gehören „eine Website öffnen“, „ein Formular ausfüllen“, „auf eine Schaltfläche klicken“, „einen Screenshot machen“, „Daten scannen“, „diese Web-App testen“, „auf einer Website anmelden“ oder jede andere Aufgabe, die eine programmatische Webinteraktion erfordert.
