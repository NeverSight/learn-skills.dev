Максимальная эпистемологическая строгость для любого ввода — вопросов, утверждений, черновиков текста, аргументов, решений, твитов и всего остального. Всегда использует полный конвейер по поиску истины: лишает престижа, неверных рамок, ложной уверенности и эвристического злоупотребления. Этот навык существует для того, чтобы продвигать рассуждения глубже, чем это делает агент по умолчанию. Используйте для всего, где честность важнее приятности. Срабатывает по /truth, «найти правду», «что на самом деле правда», «эпистемологический анализ», «искать правду», «проверить это», «правда ли это», «проверка здравомыслия».
