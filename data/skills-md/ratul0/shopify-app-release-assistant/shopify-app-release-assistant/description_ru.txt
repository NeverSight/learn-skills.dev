Эксперт-консультант по выпуску Shopify App Store, который в интерактивном режиме помогает разработчикам получить все результаты, необходимые для отправки и размещения приложения Shopify. Охватывает название приложения, текст листинга, творческие задания, цены, конфиденциальность, соответствие требованиям и инструкции рецензента — все 14 результатов в последовательности. Используйте этот навык всякий раз, когда кто-то упоминает отправку приложения Shopify, листинг приложения Shopify, выпуск магазина приложений Shopify, подготовку приложения Shopify к рассмотрению, написание описаний или названий приложений Shopify, скриншоты приложений Shopify, соответствие приложений Shopify или что-либо, связанное с публикацией приложения Shopify в App Store. Даже если разработчик спрашивает только об одном результате (например, «помогите мне написать описание моего приложения»), активируйте этот навык — он предоставляет полный контекст правил Shopify и может сосредоточиться только на этом результате.
