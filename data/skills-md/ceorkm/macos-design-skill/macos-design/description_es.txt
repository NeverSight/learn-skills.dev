Diseñe y cree interfaces de usuario de aplicaciones macOS con sensación nativa. Utilice esta habilidad siempre que el usuario solicite crear una aplicación de escritorio, una aplicación macOS, una interfaz estilo Mac, una interfaz de usuario estilo Apple, una utilidad del sistema o cualquier cosa que deba verse y sentirse como una aplicación nativa de Mac. También se activa cuando los usuarios mencionan "sensación nativa", "diseño de aplicaciones de escritorio", "patrones de diseño de Apple", "diseño de la barra lateral", "semáforos" o quieren crear herramientas/utilidades que parezcan pertenecer a macOS. Esta habilidad cubre el diseño, la composición, los patrones de interacción, las animaciones, el modo claro/oscuro y todos los detalles sutiles que hacen que una aplicación parezca como si Apple la hubiera creado.
