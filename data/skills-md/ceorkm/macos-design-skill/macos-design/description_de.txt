Entwerfen und erstellen Sie native Benutzeroberflächen für macOS-Anwendungen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine Desktop-App, eine macOS-App, eine Benutzeroberfläche im Mac-Stil, eine Benutzeroberfläche im Apple-Stil, ein Systemdienstprogramm oder etwas anderes erstellen möchte, das wie eine native Mac-Anwendung aussehen und sich anfühlen soll. Wird auch ausgelöst, wenn Benutzer „natives Feeling“, „Desktop-App-Design“, „Apple-Designmuster“, „Sidebar-Layout“, „Ampeln“ erwähnen oder Tools/Dienstprogramme erstellen möchten, die zu macOS gehören. Diese Fähigkeit umfasst Layout, Komposition, Interaktionsmuster, Animationen, Hell-/Dunkelmodus und alle subtilen Details, die einer App das Gefühl geben, von Apple erstellt zu werden.
