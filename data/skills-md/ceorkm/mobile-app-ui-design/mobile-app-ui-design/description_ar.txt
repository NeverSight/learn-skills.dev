تصميم شاشات وتدفقات ومكونات تطبيقات الهاتف المحمول عالية الجودة لواجهة المستخدم/تجربة المستخدم. استخدم هذه المهارة عندما يطلب المستخدم تصميم شاشة تطبيق جوال، أو إنشاء نماذج بالحجم الطبيعي للتطبيق، أو إنشاء مكونات واجهة مستخدم للجوال، أو تحسين تصميم تطبيق جوال موجود، أو إنشاء تدفقات تأهيل، أو تصميم التنقل عبر الجوال، أو يطلب أي عمل لواجهة الهاتف المحمول أولاً. يتم تشغيله أيضًا عندما يذكر المستخدم تصميم التطبيق، أو واجهة مستخدم الهاتف المحمول، أو تجربة مستخدم الهاتف المحمول، أو تصميم الشاشة، أو نماذج التطبيقات، أو الإطارات السلكية، أو يريد إنشاء واجهات نمط React Native / Flutter / SwiftUI كنماذج أولية مرئية. حتى لو قال المستخدم فقط "صمم تطبيقًا" أو "اجعل هذه الشاشة تبدو أفضل"، فاستخدم هذه المهارة.
