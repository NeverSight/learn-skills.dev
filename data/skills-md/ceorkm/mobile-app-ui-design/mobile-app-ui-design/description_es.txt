Diseñe pantallas, flujos y componentes UI/UX de aplicaciones móviles de alta calidad. Utilice esta habilidad siempre que el usuario solicite diseñar la pantalla de una aplicación móvil, crear maquetas de aplicaciones, crear componentes de interfaz de usuario móvil, mejorar el diseño de una aplicación móvil existente, crear flujos de incorporación, diseñar navegación móvil o solicitar cualquier trabajo de interfaz móvil. También se activa cuando el usuario menciona el diseño de aplicaciones, UI móvil, UX móvil, diseño de pantalla, maquetas de aplicaciones, estructuras alámbricas o quiere crear interfaces de estilo React Native/Flutter/SwiftUI como prototipos visuales. Incluso si el usuario simplemente dice "diseñar una aplicación" o "hacer que esta pantalla se vea mejor", utilice esta habilidad.
