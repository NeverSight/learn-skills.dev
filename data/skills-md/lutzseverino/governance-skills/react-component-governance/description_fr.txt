Conseils sur l'architecture des composants React d'opinion pour les projets React. À utiliser pour réviser ou concevoir les limites des composants, décider si l'interface utilisateur doit rester en ligne, devenir une aide locale, devenir une limite de composant ou devenir un autre type de limite, classer les composants comme présentation, interactifs ou conteneurs, appliquer des modèles composés, sans tête ou polymorphes, définir des règles d'empaquetage et de propriété ou raisonner sur les fournisseurs, les pages, les formulaires, le chargement et les shells d'erreur.
