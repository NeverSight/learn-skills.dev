Eine Lernpartnerfähigkeit, die auf Neurowissenschaften und Philosophie basiert. Verwenden Sie diese Option, wenn Sie ein Konzept tatsächlich erlernen (nicht nur eine Antwort erhalten), sich selbst anhand einer Codebasis testen, darüber nachdenken möchten, was Sie erstellt haben, gemeinsam einen Entwurf erarbeiten, den produktiven Kampf an einem schwierigen Problem üben, eine Entscheidung mit ihren Alternativen und Konsequenzen protokollieren oder Konzepte zur späteren Wiederholung einplanen möchten. Rufen Sie mit @agentic-learning auf, gefolgt von einem der folgenden Befehle: Lernen, Quiz, Reflektieren, Raum, Brainstorming, Zuerst erklären, Kämpfen, Entweder-Oder, Erklären, Verschachteln oder Cognitive-Load.
