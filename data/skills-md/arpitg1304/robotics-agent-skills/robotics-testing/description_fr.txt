Stratégies, modèles et outils de test pour les logiciels de robotique. Utilisez cette compétence lors de l'écriture de tests unitaires, de tests d'intégration, de tests de simulation ou de tests de matériel dans la boucle pour les systèmes robotiques. Déclenchez chaque fois que l'utilisateur mentionne le test des nœuds ROS, pytest avec ROS, launch_testing, les tests de simulation, CI/CD pour la robotique, les montages de test pour les capteurs, le matériel fictif, la relecture déterministe, les tests de régression pour les comportements des robots ou la validation des pipelines de perception/planification/contrôle. Couvre également les tests basés sur les propriétés pour la cinématique, les tests fuzz pour les gestionnaires de messages et les tests de fichiers d'or pour les trajectoires.
