اختبار الاستراتيجيات والأنماط والأدوات الخاصة ببرامج الروبوتات. استخدم هذه المهارة عند كتابة اختبارات الوحدة، أو اختبارات التكامل، أو اختبارات المحاكاة، أو اختبارات الأجهزة داخل الحلقة لأنظمة الروبوت. يتم التشغيل عندما يذكر المستخدم اختبار عقد ROS، أو pytest مع ROS، أو Launch_testing، أو اختبار المحاكاة، أو CI/CD للروبوتات، أو تركيبات الاختبار لأجهزة الاستشعار، أو الأجهزة الوهمية، أو إعادة التشغيل الحتمية، أو اختبار الانحدار لسلوكيات الروبوت، أو التحقق من صحة خطوط أنابيب الإدراك/التخطيط/التحكم. يغطي أيضًا الاختبار القائم على الخصائص لعلم الحركة، واختبار الزغب لمعالجات الرسائل، واختبار الملف الذهبي للمسارات.
