يتم تطبيق مبادئ تصميم البرمجيات التأسيسية خصيصًا لتطوير وحدة الروبوتات. استخدم هذه المهارة عند تصميم وحدات برامج الروبوت، أو هيكلة قواعد التعليمات البرمجية، أو اتخاذ قرارات الهندسة المعمارية، أو مراجعة كود الروبوتات، أو إنشاء مكتبات الروبوتات القابلة لإعادة الاستخدام. يتم التشغيل عندما يذكر المستخدم مبادئ SOLID للروبوتات، أو برامج الروبوتات المعيارية، أو الهندسة المعمارية النظيفة للروبوتات، أو حقن التبعية في الروبوتات، أو تصميم الواجهة للأجهزة، أو قيود التصميم في الوقت الفعلي، أو استراتيجيات معالجة الأخطاء للروبوتات، أو إدارة التكوين، أو فصل الاهتمامات في التحكم في تخطيط الإدراك، أو قابلية تكوين سلوكيات الروبوت، أو أي مناقشة حول براعة البرامج في سياق الروبوتات. قم أيضًا بتشغيل مراجعات التعليمات البرمجية الخاصة بكود الروبوتات، أو إعادة بناء برامج الروبوت، أو تصميم واجهات برمجة التطبيقات لمكتبات الروبوتات.
