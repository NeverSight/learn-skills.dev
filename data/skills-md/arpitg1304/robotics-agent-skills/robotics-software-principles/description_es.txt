Principios fundamentales de diseño de software aplicados específicamente al desarrollo de módulos de robótica. Utilice esta habilidad al diseñar módulos de software para robots, estructurar bases de código, tomar decisiones de arquitectura, revisar código de robótica o crear bibliotecas de robótica reutilizables. Se activa cada vez que el usuario menciona principios SOLID para robots, software de robótica modular, arquitectura limpia para robots, inyección de dependencia en robótica, diseño de interfaz para hardware, restricciones de diseño en tiempo real, estrategias de manejo de errores para robots, gestión de configuración, separación de preocupaciones en percepción-planificación-control, componibilidad de comportamientos de robots o cualquier discusión sobre la artesanía del software en un contexto de robótica. También activa revisiones de código de robótica, refactorización de software de robots o diseño de API para bibliotecas de robótica.
