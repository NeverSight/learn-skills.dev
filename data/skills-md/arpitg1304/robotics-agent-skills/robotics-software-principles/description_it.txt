Principi fondamentali di progettazione del software applicati specificamente allo sviluppo di moduli di robotica. Utilizza questa competenza durante la progettazione di moduli software per robot, la strutturazione di basi di codice, la presa di decisioni sull'architettura, la revisione del codice di robotica o la creazione di librerie di robotica riutilizzabili. Si attiva ogni volta che l'utente menziona i principi SOLID per i robot, il software di robotica modulare, l'architettura pulita per i robot, l'inserimento delle dipendenze nella robotica, la progettazione dell'interfaccia per l'hardware, i vincoli di progettazione in tempo reale, le strategie di gestione degli errori per i robot, la gestione della configurazione, la separazione delle preoccupazioni nella percezione-pianificazione-controllo, la componibilità dei comportamenti dei robot o qualsiasi discussione sulla creazione di software in un contesto di robotica. Attiva anche la revisione del codice di robotica, il refactoring del software del robot o la progettazione di API per le librerie di robotica.
