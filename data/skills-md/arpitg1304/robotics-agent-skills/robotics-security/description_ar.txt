التشديد الأمني ​​وأفضل الممارسات للأنظمة الروبوتية، التي تغطي أمان SROS2 DDS، وتجزئة الشبكة، وإدارة الأسرار، والتمهيد الآمن، وتقاطع السلامة المادية عبر الإنترنت. استخدم هذه المهارة عند تأمين اتصالات ROS2، أو تكوين تشفير DDS والتحكم في الوصول، أو تقوية أجهزة الكمبيوتر الآلية الموجودة على متن الروبوت، أو إدارة الشهادات وبيانات الاعتماد، أو إعداد تجزئة الشبكة لأساطيل الروبوتات، أو معالجة التحديات الأمنية الفريدة حيث تصبح الثغرات السيبرانية مخاطر على السلامة المادية. يتم التشغيل عندما يذكر المستخدم SROS2، أو أمان DDS، أو أمان الروبوت، أو تقوية الروبوت، أو تشفير ROS2، أو التحكم في الوصول إلى ROS2، أو أمان شبكة الروبوت، أو النشر الآمن للروبوت، أو شهادات الروبوت، أو إنشاء مخزن المفاتيح، أو جدار حماية الروبوت، أو أمان التوقف الإلكتروني، أو عزل وحدة التحكم بالسلامة، أو IEC 62443 للروبوتات.
