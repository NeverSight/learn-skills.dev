Patrones de arquitectura, principios de diseño y recetas probadas para crear software de robótica robusto. Utilice esta habilidad al diseñar arquitecturas de software de robots, elegir entre marcos de comportamiento, estructurar canales de percepción, planificación y control, implementar máquinas de estado, diseñar sistemas de seguridad o diseñar sistemas de múltiples robots. Se activa cada vez que el usuario menciona árboles de comportamiento, máquinas de estados finitos, arquitectura de subsunción, fusión de sensores, seguridad de robots, perros guardianes, latidos, degradación elegante, capas de abstracción de hardware, restricciones en tiempo real o arquitectura de software para robots. También se aplica a la transferencia de simulación a real, los gemelos digitales y la gestión de flotas de robots.
