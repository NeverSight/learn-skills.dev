أنماط الهندسة المعمارية، ومبادئ التصميم، والوصفات المثبتة لبناء برامج الروبوتات القوية. استخدم هذه المهارة عند تصميم بنيات برمجيات الروبوت، أو الاختيار بين الأطر السلوكية، أو هيكلة خطوط أنابيب الإدراك والتخطيط والتحكم، أو تنفيذ أجهزة الحالة، أو تصميم أنظمة السلامة، أو تصميم أنظمة متعددة الروبوتات. يتم التشغيل عندما يذكر المستخدم أشجار السلوك، أو أجهزة الحالة المحدودة، أو بنية الغمر، أو دمج أجهزة الاستشعار، أو سلامة الروبوتات، أو هيئات المراقبة، أو نبضات القلب، أو التدهور السلس، أو طبقات تجريد الأجهزة، أو قيود الوقت الفعلي، أو بنية البرامج للروبوتات. ينطبق أيضًا على النقل من بطاقة sim إلى حقيقية، والتوائم الرقمية، وإدارة أسطول الروبوتات.
