Alph.ai Social Media Twitter(X) API — мониторинг учетных записей X, очистка твитов (публикации/ретвиты/ответы/цитаты), горячие списки наблюдения, управление подписками, конфигурация мониторинга, нажатие WebSocket в реальном времени, перевод, поиск данных пользователя X, поиск по ключевым словам твитов, список твитов пользователей, поиск токенов мемов твитов и т. д.
