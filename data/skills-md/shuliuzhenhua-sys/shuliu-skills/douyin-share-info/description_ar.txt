استخدم TikHub Douyin Web API الذي تم استدعاؤه عبر رابط مشاركة Douyin لجلب المعلومات الأساسية للعمل، واستخراج عناوين URL لتنزيل الغلاف والصوت والفيديو دائمًا (مع أخذ قائمة url الأولى). استخدم هذا عندما يذكر المستخدم "رابط مشاركة تحليل Douyin" أو "احصل على عنوان URL لمعلومات الفيديو/الغلاف/الصوت/التشغيل من رابط المشاركة".
