Utilisez une API Web TikHub Douyin appelée via un lien de partage Douyin pour récupérer les informations de base sur la publication et toujours extraire les URL de téléchargement de couverture, audio et vidéo (en prenant la première url_list). Utilisez-le lorsque l'utilisateur mentionne « 解析抖音分享链接 » ou « 根据分享链接取视频信息/封面/音频/播放地址 ».
