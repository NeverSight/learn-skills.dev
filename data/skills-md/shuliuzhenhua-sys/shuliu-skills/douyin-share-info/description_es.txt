Utilice una llamada a la API web de TikHub Douyin a través de un enlace para compartir de Douyin para obtener la información básica del trabajo y extraer siempre las URL de descarga de portada, audio y video (tomando la primera lista de URL). Utilícelo cuando el usuario mencione "Análisis del enlace para compartir de Douyin" u "obtener información del vídeo/portada/audio/URL de reproducción desde el enlace para compartir".
