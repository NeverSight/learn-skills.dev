Configurez le runtime @kuzenbo/theme, les thèmes prédéfinis et l'utilisation des jetons sémantiques pour les applications Kuzenbo. À utiliser lorsque les tâches mentionnent la configuration de ThemeProvider, l'amorçage/l'hydratation du thème, le comportement clair-obscur, les remplacements de jetons, la sélection de thème prédéfini, les jetons z-index/curseur ou le câblage des jetons de couleur.
