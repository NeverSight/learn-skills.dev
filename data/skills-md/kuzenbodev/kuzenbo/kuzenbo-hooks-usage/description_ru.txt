Выберите и внедрите стабильные перехватчики React из @kuzenbo/hooks для состояния, API браузера, хранилища, времени, области просмотра и шаблонов взаимодействия. Используйте, когда в задачах упоминается выбор перехватчика, замена специальных эффектов/прослушивателей, адаптивное поведение, потоки буфера обмена/полноэкранного режима или поведение React только для клиента.
