أنشئ واجهات React وأعد تصميمها باستخدام @kuzenbo/core و@kuzenbo/theme. يُستخدم عندما تشير المهام إلى مكونات Kuzenbo، أو عمليات استيراد المسار الفرعي (@kuzenbo/core/ui/*)، أو اختيار المكونات، أو أنماط التركيب، أو الرموز المميزة للحجم، أو سلوك إمكانية الوصول، أو الترحيل من واجهة المستخدم المخصصة إلى عناصر Kuzenbo الأولية.
