Extrae la fórmula esencial (El Uno) de libros, conceptos o ideas utilizando el pensamiento de primeros principios y luego la falsifica identificando condiciones límite. Genera una tarjeta HTML visual. Úselo para una extracción rápida de esencias de fórmula única. Para un análisis profundo de libros de varias rondas, utilice ljg-xray-book en su lugar. NO lo use para trabajos académicos (use ljg-xray-paper).
