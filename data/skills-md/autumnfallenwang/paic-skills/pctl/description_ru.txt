pctl CLI (v0.6.3) — PAIC Control, унифицированный интерфейс командной строки для тестирования для PingOne Advanced Identity Cloud (AIC). Управляет профилями подключения, генерацией/декодированием/проверкой токенов JWT, тестированием пути аутентификации, управлением локальным стеком ELK (потоковое вещание журналов Elasticsearch + Kibana), поиском по историческим журналам и отслеживанием изменений конфигурации. Содержит краткие описания среды, рецепты рабочих процессов ELK и ошибки.
