pctl CLI (v0.6.3): PAIC Control, una CLI di test unificata per PingOne Advanced Identity Cloud (AIC). Gestisce profili di connessione, generazione/decodifica/convalida di token JWT, test del percorso di autenticazione, gestione dello stack ELK locale (Elasticsearch + streaming di log Kibana), ricerca di log storici e monitoraggio delle modifiche alla configurazione. Contiene scorciatoie per l'ambiente, ricette per il flusso di lavoro ELK e trucchi.
