Автоматизация Chrome из CLI с помощью инструмента Rodney. Управляйте автономным Chrome для парсинга веб-страниц, тестирования, создания снимков экрана, заполнения форм, навигации и проверки доступности. Используйте, когда пользователь хочет автоматизировать взаимодействие с браузером, очистить веб-сайты, сделать снимки экрана, протестировать веб-страницы, заполнить формы или проверить доступность. Поддерживает постоянные сеансы Chrome, многоэтапные рабочие процессы и интеграцию сценариев оболочки.
