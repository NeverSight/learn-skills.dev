Automatización de Chrome desde CLI usando la herramienta Rodney. Utilice Chrome sin cabeza para realizar pruebas web, realizar pruebas, realizar capturas de pantalla, completar formularios, navegar y comprobar la accesibilidad. Úselo cuando el usuario desee automatizar las interacciones del navegador, extraer sitios web, tomar capturas de pantalla, probar páginas web, completar formularios o verificar la accesibilidad. Admite sesiones persistentes de Chrome, flujos de trabajo de varios pasos e integración de scripts de shell.
