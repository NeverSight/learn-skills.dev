Automazione di Chrome dalla CLI utilizzando lo strumento Rodney. Utilizza Chrome headless per web scraping, test, screenshot, compilazione di moduli, navigazione e controlli di accessibilità. Utilizzare quando l'utente desidera automatizzare le interazioni del browser, raschiare siti Web, acquisire screenshot, testare pagine Web, compilare moduli o verificare l'accessibilità. Supporta sessioni Chrome persistenti, flussi di lavoro in più passaggi e integrazione di scripting della shell.
