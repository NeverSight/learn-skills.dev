أفضل ممارسات تطوير .NET الشاملة من خبراء الصناعة. يُستخدم عند كتابة تطبيقات .NET أو مراجعتها أو تصميمها. المشغلات على البنية النظيفة، أو CQRS، أو تحسين Entity Framework Core، أو ضبط الأداء، أو تصميم واجهة برمجة التطبيقات، أو الأنماط غير المتزامنة، أو نشر الإنتاج. يوفر إرشادات سياقية بناءً على حجم المشروع — بدءًا من واجهات برمجة التطبيقات البسيطة وحتى أنظمة المؤسسة.
