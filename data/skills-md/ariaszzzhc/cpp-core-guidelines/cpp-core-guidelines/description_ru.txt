Основные рекомендации C++ от Страуструпа и Саттера по написанию современного, безопасного и эффективного кода C++. Используйте при написании, проверке или рефакторинге кода C++ для обеспечения безопасности типов, безопасности ресурсов и производительности. Запускает задачи кода C++, включая проектирование интерфейса, управление ресурсами (память, дескрипторы, блокировки), проектирование классов, обработку ошибок, параллелизм, шаблоны и общую проверку кода. Используйте для таких вопросов, как «просмотрите этот код C++», «как мне следует управлять ресурсами в C++», «лучшие практики для интерфейсов C++» или при написании любого кода C++, который должен соответствовать современным стандартам (C++11/14/17/20).
