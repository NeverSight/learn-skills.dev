Directives de base C++ de Stroustrup et Sutter pour écrire du code C++ moderne, sûr et efficace. À utiliser lors de l'écriture, de la révision ou de la refactorisation du code C++ pour garantir la sécurité des types, la sécurité des ressources et les performances. Déclencheurs sur les tâches de code C++, notamment la conception d'interface, la gestion des ressources (mémoire, handles, verrous), la conception de classes, la gestion des erreurs, la concurrence, les modèles et la révision générale du code. À utiliser pour des questions telles que « réviser ce code C++ », « comment dois-je gérer les ressources en C++ », « meilleures pratiques pour les interfaces C++ » ou lors de l'écriture de tout code C++ qui doit suivre les normes modernes (C++11/14/17/20).
