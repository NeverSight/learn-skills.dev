Directrices básicas de C++ de Stroustrup y Sutter para escribir código C++ moderno, seguro y eficiente. Úselo al escribir, revisar o refactorizar código C++ para garantizar la seguridad de tipos, la seguridad de recursos y el rendimiento. Activadores de tareas de código C++ que incluyen diseño de interfaz, administración de recursos (memoria, identificadores, bloqueos), diseño de clases, manejo de errores, simultaneidad, plantillas y revisión general de código. Úselo para preguntas como "revisar este código C++", "cómo debo administrar los recursos en C++", "mejores prácticas para interfaces C++" o al escribir cualquier código C++ que deba seguir los estándares modernos (C++14/11/17/20).
