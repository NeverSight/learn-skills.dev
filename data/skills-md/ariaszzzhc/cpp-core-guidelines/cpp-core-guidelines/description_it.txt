Linee guida C++ Core di Stroustrup e Sutter per scrivere codice C++ moderno, sicuro ed efficiente. Da utilizzare durante la scrittura, la revisione o il refactoring del codice C++ per garantire l'indipendenza dei tipi, l'incolumità delle risorse e le prestazioni. Attiva attività di codice C++ tra cui progettazione dell'interfaccia, gestione delle risorse (memoria, handle, blocchi), progettazione di classi, gestione degli errori, concorrenza, modelli e revisione generale del codice. Utilizzare per domande come "rivedere questo codice C++", "come dovrei gestire le risorse in C++", "procedure consigliate per le interfacce C++" o quando si scrive codice C++ che dovrebbe seguire gli standard moderni (C++11/14/17/20).
