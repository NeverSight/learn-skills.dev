المبادئ التوجيهية الأساسية لـ C++ من Stroustrup وSutter لكتابة كود C++ حديث وآمن وفعال. يُستخدم عند كتابة تعليمات برمجية C++ أو مراجعتها أو إعادة هيكلتها لضمان سلامة النوع وسلامة الموارد والأداء. تشغيل مهام التعليمات البرمجية لـ C++ بما في ذلك تصميم الواجهة وإدارة الموارد (الذاكرة والمقابض والأقفال) وتصميم الفئة ومعالجة الأخطاء والتزامن والقوالب ومراجعة التعليمات البرمجية العامة. استخدمه لأسئلة مثل "مراجعة كود C++ هذا" أو "كيف يمكنني إدارة الموارد في C++" أو "أفضل الممارسات لواجهات C++" أو عند كتابة أي كود C++ يجب أن يتبع المعايير الحديثة (C++11/14/17/20).
