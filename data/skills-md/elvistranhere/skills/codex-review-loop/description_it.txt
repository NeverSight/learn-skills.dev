Revisione automatizzata del codice e loop di correzione utilizzando la CLI di OpenAI Codex come revisore e Claude Code come riparatore. Da utilizzare quando l'utente dice "revisione del codice", "ciclo di revisione", "revisione e correzione automatica", "revisione del mio codice" o desidera eseguire una revisione iterativa del codice su un ramo di base finché tutti i problemi non vengono risolti. Chiama la revisione exec del codex per rivedere la differenza, Claude Code corregge i risultati, quindi riesamina fino al raggiungimento dei round puliti o massimi.
