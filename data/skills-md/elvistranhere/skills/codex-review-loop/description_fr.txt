Révision automatisée du code et correction de la boucle en utilisant OpenAI Codex CLI en tant que réviseur et Claude Code en tant que réparateur. À utiliser lorsque l'utilisateur dit « révision du codex », « boucle de révision », « révision et correction automatiques », « réviser mon code » ou souhaite exécuter une révision itérative du code sur une branche de base jusqu'à ce que tous les problèmes soient résolus. Appelle la révision exécutive du codex pour examiner la différence, Claude Code corrige les résultats, puis réexamine jusqu'à ce que les tours propres ou maximum soient atteints.
