Автоматизированный цикл проверки кода и исправлений с использованием интерфейса командной строки OpenAI Codex в качестве проверяющего и Claude Code в качестве исправления. Используйте, когда пользователь говорит «проверка кодекса», «цикл проверки», «автоматическая проверка и исправление», «проверка моего кода» или хочет запустить итеративную проверку кода для базовой ветки, пока все проблемы не будут решены. Вызывает проверку исполнительного директора кодекса для проверки различий, Клод Код исправляет результаты, затем повторяет проверку до тех пор, пока не будет достигнуто чистое или максимальное количество раундов.
