مراجعة التعليمات البرمجية تلقائيًا وحلقة الإصلاح باستخدام OpenAI Codex CLI كمراجع وكلود كود كمصلح. يُستخدم عندما يقول المستخدم "مراجعة المخطوطة"، أو "حلقة المراجعة"، أو "المراجعة والإصلاح التلقائي"، أو "مراجعة الكود الخاص بي"، أو عندما يريد تشغيل مراجعة التعليمات البرمجية التكرارية مقابل فرع أساسي حتى يتم حل جميع المشكلات. يستدعي مراجعة Codex exec لمراجعة الفرق، ويقوم Claude Code بإصلاح النتائج، ثم يعيد المراجعة حتى يتم الوصول إلى الجولات النظيفة أو الحد الأقصى.
