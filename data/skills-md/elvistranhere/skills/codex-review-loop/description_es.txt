Revisión de código automatizada y bucle de corrección utilizando OpenAI Codex CLI como revisor y Claude Code como reparador. Úselo cuando el usuario diga "revisión de códice", "bucle de revisión", "revisión y corrección automática", "revisar mi código" o quiera ejecutar una revisión de código iterativa en una rama base hasta que se resuelvan todos los problemas. Llama a la revisión ejecutiva del Codex para revisar la diferencia, Claude Code corrige los hallazgos y luego vuelve a revisar hasta alcanzar rondas limpias o máximas.
