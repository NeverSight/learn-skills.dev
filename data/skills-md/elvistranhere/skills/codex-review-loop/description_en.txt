Automated code review and fix loop using OpenAI Codex CLI as reviewer and Claude Code as fixer. Use when the user says 'codex review', 'review loop', 'auto review and fix', 'review my code', or wants to run iterative code review against a base branch until all issues are resolved. Calls codex exec review to review the diff, Claude Code fixes the findings, then re-reviews until clean or max rounds reached.
