Utilisez cette compétence lors de la mise en œuvre d'un traitement de file d'attente durable avec la file d'attente @valentinkolb/sync : boucles de consommateur d'envoi/recv/streaming, livraison au moins une fois avec ack/nack/touch, délai d'expiration de visibilité basé sur le bail, messages retardés, comportement DLQ, mise en file d'attente idempotente et espaces de clés multi-locataires. À utiliser également pour choisir entre la file d'attente (distribution du travail), le sujet (distribution d'événements) et le travail (exécution durable avec état).
