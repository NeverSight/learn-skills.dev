Генерирует YAML-код проверки проводки для планов станков. Помогает агентам доказать, что функции правильно интегрированы — команды зарегистрированы, конечные точки установлены, модули экспортированы, компоненты отрисованы. Используйте при написании полей истины/артефактов/связей для этапов плана ткацкого станка.
