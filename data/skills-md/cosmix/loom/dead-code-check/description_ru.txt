Создает конфигурации обнаружения мертвого кода для проверки плана станка. Предоставляет специфичные для языка команды, шаблоны сбоев и шаблоны игнорирования для Rust, TypeScript, Python, Go и JavaScript. Используйте при добавлении проверок качества кода в критерии приемки или поля истинности в планах ткацких станков. Обнаружение мертвого кода обнаруживает неполную проводку, идентифицируя код, который существует, но никогда не вызывается.
