Erzeugt Konfigurationen zur Erkennung toter Codes zur Überprüfung des Webstuhlplans. Bietet sprachspezifische Befehle, Fehlermuster und Ignoriermuster für Rust, TypeScript, Python, Go und JavaScript. Verwenden Sie diese Option, wenn Sie Codequalitätsprüfungen zu Akzeptanzkriterien- oder Wahrheitsfeldern in Webstuhlplänen hinzufügen. Die Erkennung von totem Code erkennt unvollständige Verbindungen, indem sie Code identifiziert, der vorhanden ist, aber nie aufgerufen wird.
