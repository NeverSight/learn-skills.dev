Vollständiger Testworkflow von der Site-Erkennung bis zur Testautomatisierung mit Qase-Integration. Verwenden Sie diese Option, wenn der Benutzer eine End-to-End-Testeinrichtung für eine Website wünscht. Löst auf „Diese Website testen“, „Testsuite erstellen für“, „Hilf mir beim Testen“, „Vollständiger Testworkflow“ und „Testen automatisieren für“ aus.
