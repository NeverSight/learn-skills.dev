Строгая проверка кода в соответствии с принципами «Чистый код», «Чистая архитектура» и «Прагматичный программист». Используйте при: (1) проверке кода или запросах на включение, (2) обнаружении запахов кода или проблемах с качеством, (3) аудите архитектурных решений, (4) подготовке кода к слиянию, (5) рефакторинге существующего кода или (6) проверке соблюдения принципов SOLID, DRY, YAGNI, KISS. Имеет систему опросов 3+4+2 для калибровки строгости от L1 (лабораторный) до L5 (критический). Также срабатывает по следующим запросам: «хорош ли этот код?», «проверить качество кода», «готов к слиянию?», «технический долг», «запах кода», «лучшие практики», «очистить код», «проверка рефакторинга».
