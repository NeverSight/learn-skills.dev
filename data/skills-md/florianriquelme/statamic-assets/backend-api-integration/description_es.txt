Genere integraciones de API basadas en Saloon listas para producción para proyectos de Laravel, incluidas clases de conector, clases de solicitud, DTO de solo lectura, métodos de capa de servicio con manejo integral de errores y conjuntos completos de pruebas de Pest. Utilice esta habilidad al integrar API externas usando Saloon, crear controladores de solicitudes/respuestas de API, agregar nuevos puntos finales de API a servicios existentes, crear envoltorios de clientes de API para servicios de terceros o configurar la comunicación de API con pruebas y manejo de errores adecuados. Esta habilidad se aplica cuando se trabaja en archivos en app/Services/Api/, app/DataTransferObjects/Api/, se crean conectores y solicitudes de Saloon, se implementan métodos de servicio API o se escriben pruebas de integración de API.
