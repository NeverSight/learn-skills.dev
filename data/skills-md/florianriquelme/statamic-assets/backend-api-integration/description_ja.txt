コネクタ クラス、リクエスト クラス、読み取り専用 DTO、包括的なエラー処理を備えたサービス層メソッド、および完全な Pest テスト スイートを含む、Laravel プロジェクト向けの本番環境に対応した Saloon ベースの API 統合を生成します。このスキルは、Saloon を使用した外部 API の統合、API リクエスト/レスポンス ハンドラーの作成、既存のサービスへの新しい API エンドポイントの追加、サードパーティ サービス用の API クライアント ラッパーの構築、または適切なエラー処理とテストによる API 通信の設定を行うときに使用します。このスキルは、app/Services/Api/、app/DataTransferObjects/Api/ のファイルを操作する場合、Saloon コネクタとリクエストを作成する場合、API サービス メソッドを実装する場合、または API 統合テストを作成する場合に適用されます。
