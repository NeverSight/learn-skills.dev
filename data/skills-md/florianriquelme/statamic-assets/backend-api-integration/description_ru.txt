Создавайте готовые к использованию API-интеграции на базе Saloon для проектов Laravel, включая классы коннекторов, классы запросов, DTO только для чтения, методы сервисного уровня с комплексной обработкой ошибок и полные наборы тестов Pest. Используйте этот навык при интеграции внешних API с помощью Saloon, создании обработчиков запросов/ответов API, добавлении новых конечных точек API к существующим сервисам, создании оболочек клиентов API для сторонних сервисов или настройке связи API с правильной обработкой ошибок и тестированием. Этот навык применяется при работе с файлами в app/Services/Api/, app/DataTransferObjects/Api/, создании соединителей и запросов Saloon, реализации методов обслуживания API или написании интеграционных тестов API.
