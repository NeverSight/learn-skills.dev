أنشئ عمليات تكامل لواجهة برمجة التطبيقات (API) قائمة على الصالون جاهزة للإنتاج لمشروعات Laravel، بما في ذلك فئات الموصلات وفئات الطلب وDTOs للقراءة فقط وطرق طبقة الخدمة مع معالجة شاملة للأخطاء ومجموعات اختبار Pest الكاملة. استخدم هذه المهارة عند دمج واجهات برمجة التطبيقات الخارجية باستخدام Saloon، أو إنشاء معالجات طلب/استجابة واجهة برمجة التطبيقات، أو إضافة نقاط نهاية جديدة لواجهة برمجة التطبيقات إلى الخدمات الحالية، أو إنشاء أغلفة عميل واجهة برمجة التطبيقات لخدمات الجهات الخارجية، أو إعداد اتصال واجهة برمجة التطبيقات مع معالجة الأخطاء واختبارها بشكل مناسب. تنطبق هذه المهارة عند العمل على الملفات في app/Services/Api/، أو app/DataTransferObjects/Api/، أو إنشاء موصلات وطلبات Saloon، أو تنفيذ أساليب خدمة API، أو كتابة اختبارات تكامل API.
