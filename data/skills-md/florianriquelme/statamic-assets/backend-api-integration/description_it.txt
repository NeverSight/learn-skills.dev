Genera integrazioni API basate su Saloon pronte per la produzione per progetti Laravel, tra cui classi di connettori, classi di richiesta, DTO di sola lettura, metodi del livello di servizio con gestione completa degli errori e suite complete di test Pest. Utilizza questa competenza quando integri API esterne utilizzando Saloon, crei gestori di richiesta/risposta API, aggiungi nuovi endpoint API a servizi esistenti, crei wrapper client API per servizi di terze parti o imposti la comunicazione API con gestione e test degli errori adeguati. Questa competenza si applica quando si lavora su file in app/Services/Api/, app/DataTransferObjects/Api/, si creano connettori e richieste Saloon, si implementano metodi di servizio API o si scrivono test di integrazione API.
