TRIGGER 언제: 코드가 React-native-theme-transition을 가져오고, 사용자가 createThemeTransition/ThemeTransitionProvider/useTheme로 작업하고, 애니메이션, 시스템 테마 따르기, 테마 지속성, React Navigation 또는 expo-router 테마 통합을 통한 다크 모드 토글을 구현하거나, 전환 문제(고정 오버레이, 플래시, 터치 차단)를 디버깅합니다. 일반 React Native 스타일, 애니메이션이 아닌 테마 전환 또는 웹 전용 테마인 경우에는 트리거하지 마세요.
