Modalità oscura animata e transizioni di temi per React Native tramite sovrapposizione di screenshot. ATTIVA quando: il codice importa la transizione del tema react-native, l'utente lavora con createThemeTransition/ThemeTransitionProvider/useTheme, implementa l'attivazione/disattivazione della modalità oscura con l'animazione, il seguito del tema di sistema, la persistenza del tema, l'integrazione di React Navigation o del tema expo-router o il debug di problemi di transizione (overlay bloccato, flash, blocco del tocco). NON SI ATTIVA in caso di: stile generale di React Native, cambio di temi non animati o temi solo web.
