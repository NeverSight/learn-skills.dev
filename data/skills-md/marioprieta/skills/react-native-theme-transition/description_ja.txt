スクリーンショット オーバーレイを介した React Native のアニメーション ダーク モードとテーマの遷移。トリガーされる場合: コードが反応ネイティブ テーマ トランジションをインポートする場合、ユーザーが createThemeTransition/ThemeTransitionProvider/useTheme を操作する場合、アニメーションによるダーク モードの切り替えを実装する場合、システム テーマのフォロー、テーマの永続化、React Navigation または expo-router テーマの統合、またはトランジションの問題 (スタック オーバーレイ、フラッシュ、タッチ ブロック) をデバッグする場合。一般的な React Native スタイル、アニメーション以外のテーマの切り替え、Web のみのテーマの場合はトリガーしないでください。
