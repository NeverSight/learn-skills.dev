الوضع المظلم المتحرك وانتقالات السمات لـ React Native عبر تراكب لقطة الشاشة. يتم تشغيله عندما: يستورد الكود رد فعل انتقال السمة الأصلي، ويعمل المستخدم مع createThemeTransition/ThemeTransitionProvider/useTheme، وينفذ تبديل الوضع المظلم مع الرسوم المتحركة، أو اتباع سمة النظام، أو استمرار السمة، أو React Navigation أو تكامل سمة جهاز توجيه المعرض، أو تصحيح مشكلات الانتقال (تراكب عالق، فلاش، حظر اللمس). لا يتم تشغيله عندما: تصميم React Native العام، أو تبديل السمات غير المتحركة، أو استخدام السمات على الويب فقط.
