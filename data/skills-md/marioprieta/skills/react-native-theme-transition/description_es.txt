Modo oscuro animado y transiciones de temas para React Native mediante superposición de captura de pantalla. DISPARAR cuando: el código importa la transición del tema nativo de reacción, el usuario trabaja con createThemeTransition/ThemeTransitionProvider/useTheme, implementa la alternancia del modo oscuro con animación, seguimiento del tema del sistema, persistencia del tema, integración del tema React Navigation o expo-router, o depura problemas de transición (superposición atascada, flash, bloqueo táctil). NO DISPARAR cuando: estilo general de React Native, cambio de tema no animado o tematización solo web.
