Анимированный темный режим и переходы тем для React Native посредством наложения скриншотов. ВЫЗЫВАЕТСЯ, когда: код импортирует реакцию-native-theme-transition, пользователь работает с createThemeTransition/ThemeTransitionProvider/useTheme, реализует переключение темного режима с анимацией, отслеживание системной темы, сохранение темы, интеграцию темы React Navigation или expo-router или отлаживает проблемы перехода (зависание наложения, Flash, блокировка касания). НЕ ЗАПУСКАЕТСЯ при: общем стиле React Native, переключении неанимированных тем или темах, доступных только в Интернете.
