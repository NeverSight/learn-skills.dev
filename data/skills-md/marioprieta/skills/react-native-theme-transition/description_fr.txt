Mode sombre animé et transitions de thème pour React Native via superposition de capture d'écran. DÉCLENCHEMENT lorsque : le code importe la transition de thème natif, l'utilisateur travaille avec createThemeTransition/ThemeTransitionProvider/useTheme, implémente le basculement en mode sombre avec animation, suivi du thème système, persistance du thème, navigation React ou intégration de thème expo-routeur, ou débogue les problèmes de transition (superposition bloquée, flash, blocage tactile). NE PAS DÉCLENCHER lorsque : un style React Native général, un changement de thème non animé ou un thème Web uniquement.
