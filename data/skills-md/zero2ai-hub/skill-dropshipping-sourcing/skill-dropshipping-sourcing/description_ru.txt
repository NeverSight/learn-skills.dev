Запросите CJ Dropshipping API v2.0 для поиска продуктов и получения сведений для создания каталога. Используйте для поиска по ключевым словам CJ, получения записей о продуктах (SPU/SKU, изображения, категории, варианты/цвета, если они доступны), обновления токенов доступа и создания нормализованных выходных данных JSON для автоматизации каталога прямой поставки.
