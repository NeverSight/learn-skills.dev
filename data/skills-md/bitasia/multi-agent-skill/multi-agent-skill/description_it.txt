Capacità di ingegneria del flusso di lavoro multi-agente. Utilizzare durante la progettazione, la creazione, la revisione o il debug
sistemi multiagente. Copre schemi tipizzati, schemi di azioni, contratti MCP, modelli di orchestrazione,
modalità di guasto e strategie di test. Pattern TypeScript/Zod indipendenti dal framework con cui funzionano
qualsiasi implementazione multi-agente (Claude Code, LangChain, CrewAI, personalizzato).
