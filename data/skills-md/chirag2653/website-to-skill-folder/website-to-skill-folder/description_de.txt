Wandelt jede Website in einen installierbaren KI-Skill-Ordner um, indem alle Seiten gecrawlt und als durchsuchbarer Markdown mit Frontmatter-Zusammenfassungen verpackt werden. Verwenden Sie diese Option, wenn Sie dazu aufgefordert werden: Erstellen Sie einen Website-Such-Skill, indizieren Sie eine Website für KI-Agenten, kratzen Sie eine Website in eine durchsuchbare Wissensdatenbank, verwandeln Sie eine Website in einen offline durchsuchbaren Skill oder machen Sie Website-Inhalte für KI-Agenten ohne Live-Browsing verfügbar. Gibt einen installierbaren Skill-Ordner sowie den genauen npx-Befehl aus, um ihn jedem KI-Agenten hinzuzufügen.
