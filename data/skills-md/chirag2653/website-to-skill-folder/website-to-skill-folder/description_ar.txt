يحول أي موقع ويب إلى مجلد مهارات الذكاء الاصطناعي قابل للتثبيت عن طريق الزحف إلى جميع الصفحات وتعبئتها كتخفيض قابل للبحث مع ملخصات أساسية. استخدمه عندما يُطلب منك: إنشاء مهارة بحث في موقع ويب، أو فهرسة موقع ويب لوكلاء الذكاء الاصطناعي، أو تحويل موقع إلى قاعدة معرفية قابلة للبحث، أو تحويل موقع ويب إلى مهارة قابلة للبحث في وضع عدم الاتصال، أو إتاحة محتوى موقع الويب لعملاء الذكاء الاصطناعي دون التصفح المباشر. يُخرج مجلد مهارات قابل للتثبيت بالإضافة إلى أمر npx الدقيق لإضافته إلى أي وكيل AI.
