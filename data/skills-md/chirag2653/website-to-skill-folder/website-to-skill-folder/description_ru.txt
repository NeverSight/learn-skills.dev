Преобразует любой веб-сайт в устанавливаемую папку навыков искусственного интеллекта, сканируя все страницы и упаковывая их в уценку с возможностью поиска с краткими описаниями. Используйте, когда вас попросят: создать навык поиска по веб-сайту, проиндексировать веб-сайт для агентов ИИ, внести сайт в базу знаний с возможностью поиска, превратить веб-сайт в навык с возможностью автономного поиска или сделать контент веб-сайта доступным для агентов ИИ без просмотра в режиме реального времени. Выводит папку устанавливаемых навыков, а также точную команду npx, чтобы добавить ее к любому агенту ИИ.
