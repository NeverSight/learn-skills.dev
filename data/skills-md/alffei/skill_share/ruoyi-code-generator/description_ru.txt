Сгенерируйте CRUD-код на основе спецификаций платформы RuoYi. Используйте этот навык, когда пользователю необходимо сгенерировать внутренний код Java (объект, Mapper, Сервис, Контроллер), внешний код Vue (страница списка, оболочка API), XML-файлы сопоставления MyBatis или SQL инициализации меню для таблицы данных.  
Применимые сценарии: создание новых модулей, добавление бизнес-функций, генерация кода из табличных структур и т. д.
