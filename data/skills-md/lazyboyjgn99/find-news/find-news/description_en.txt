Multi-engine search API tool that supports general web search and news search.
Use cases:
- When the user asks to search web pages, news, or social-media content
- When you need to search Google/Baidu/Bing/Yahoo/DuckDuckGo
- When you need to search WeChat Official Accounts, YouTube, GitHub, Reddit, Bilibili
- When you need to obtain the full content of search results (crawl_results > 0)
- When you need the latest news and trending topics
