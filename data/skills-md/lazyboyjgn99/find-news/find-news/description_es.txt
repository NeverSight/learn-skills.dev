Herramienta API de búsqueda multimotor que admite búsquedas web generales y búsqueda de noticias.
Casos de uso:
- Los usuarios solicitan búsquedas de páginas web, noticias o contenido de redes sociales.
- Búsquedas necesarias desde Google/Baidu/Bing/Yahoo/DuckDuckGo
- Búsquedas necesarias para cuentas oficiales de WeChat, YouTube, GitHub, Reddit, Bilibili
- Se requiere el contenido completo de los resultados de búsqueda (crawl_results > 0)
- Seguimiento de las últimas noticias y temas de actualidad.
