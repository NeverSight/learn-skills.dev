Boîte à outils complète de scraping, d'exploration et d'extraction de données optimisée par l'API Firecrawl.
Fournit des scripts pour le scraping d'une seule page (scrape.py), la recherche sur le Web (search.py),
Découverte d'URL (map.py), exploration de plusieurs pages (crawl.py), extraction de données structurées (extract.py),
et collecte de données autonome (agent.py).
À utiliser lorsque vous devez : (1) extraire le contenu de pages Web, (2) rechercher et gratter le Web,
(3) découvrir des URL sur des sites Web, (4) explorer plusieurs pages, (5) extraire des données structurées avec des schémas JSON,
ou (6) collecter des données de manière autonome depuis n'importe où sur le Web.
Nécessite la variable d'environnement FIRECRAWL_API_KEY.
