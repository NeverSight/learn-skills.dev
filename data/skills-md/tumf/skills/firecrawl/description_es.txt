Completo kit de herramientas de extracción de datos, rastreo y raspado web impulsado por la API Firecrawl.
Proporciona scripts para raspado de una sola página (scrape.py), búsqueda web (search.py),
Descubrimiento de URL (map.py), rastreo de varias páginas (crawl.py), extracción de datos estructurados (extract.py),
y recopilación de datos autónoma (agent.py).
Úselo cuando necesite: (1) extraer contenido de páginas web, (2) buscar y raspar la web,
(3) descubrir URL en sitios web, (4) rastrear varias páginas, (5) extraer datos estructurados con esquemas JSON,
o (6) recopilar datos de forma autónoma desde cualquier lugar de la web.
Requiere la variable de entorno FIRECRAWL_API_KEY.
