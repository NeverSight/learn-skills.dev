Toolkit completo di web scraping, scansione ed estrazione dei dati basato sull'API Firecrawl.
Fornisce script per lo scraping di una singola pagina (scrape.py), la ricerca web (search.py),
Rilevamento URL (map.py), scansione di più pagine (crawl.py), estrazione di dati strutturati (extract.py),
e raccolta dati autonoma (agent.py).
Utilizzare quando è necessario: (1) estrarre contenuti dalle pagine Web, (2) cercare e analizzare il Web,
(3) scoprire URL su siti Web, (4) eseguire la scansione di più pagine, (5) estrarre dati strutturati con schemi JSON,
o (6) raccogliere autonomamente dati da qualsiasi punto del web.
Richiede la variabile di ambiente FIRECRAWL_API_KEY.
