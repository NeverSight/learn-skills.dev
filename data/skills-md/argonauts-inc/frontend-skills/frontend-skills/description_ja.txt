Argonauts Inc. フロントエンド開発スキル。 Next.js App Router または React SPA (Vite) を使用してプロジェクトを最初から構築する場合、Tailwind CSS v4 + shadcn/ui を使用してスタイルを設定する場合、および render-as-you-fetch データ取得パターンを実装する場合に使用されます。新しいフロントエンド プロジェクトの作成、コンポーネントの実装、データのフェッチ、状態管理、およびスタイル設定に関連するすべてのタスクに適用されます。新しい React/Next.js プロジェクトの構築、shadcn/ui によるコンポーネントの実装、データ フェッチのセットアップ、状態管理ソリューションの選択、Tailwind CSS v4 の構成の場合に使用します。
