Argonauts Inc. 프론트엔드 개발 기술. Next.js App Router 또는 React SPA(Vite)를 사용하여 처음부터 프로젝트를 빌드하고, Tailwind CSS v4 + shadcn/ui를 사용하여 스타일을 지정하고, 가져온 만큼 렌더링 데이터 가져오기 패턴을 구현하는 데 사용됩니다. 새로운 프런트엔드 프로젝트 생성, 구성 요소 구현, 데이터 가져오기, 상태 관리 및 스타일링과 관련된 모든 작업에 적용됩니다. 사용 시기: 새 React/Next.js 프로젝트 빌드, shadcn/ui로 구성요소 구현, 데이터 가져오기 설정, 상태 관리 솔루션 선택, Tailwind CSS v4 구성.
