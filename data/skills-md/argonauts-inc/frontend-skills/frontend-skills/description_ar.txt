Argonauts Inc. مهارات تطوير الواجهة الأمامية. يُستخدم عند إنشاء المشاريع من البداية باستخدام Next.js App Router أو React SPA (Vite)، والتصميم باستخدام Tailwind CSS v4 + shadcn/ui، وتنفيذ أنماط جلب البيانات التي يتم عرضها أثناء جلبها. ينطبق على جميع المهام المتعلقة بإنشاء مشاريع واجهة أمامية جديدة وتنفيذ المكونات وجلب البيانات وإدارة الحالة والتصميم. يُستخدم عندما: إنشاء مشاريع React/Next.js جديدة، وتنفيذ المكونات باستخدام shadcn/ui، وإعداد جلب البيانات، واختيار حلول إدارة الحالة، وتكوين Tailwind CSS v4.
