Argonauts Inc. Frontend-Entwicklungsfähigkeiten. Wird beim Erstellen von Projekten von Grund auf mit Next.js App Router oder React SPA (Vite), beim Stylen mit Tailwind CSS v4 + shadcn/ui und beim Implementieren von Render-as-you-fetch-Datenabrufmustern verwendet. Gilt für alle Aufgaben im Zusammenhang mit der Erstellung neuer Frontend-Projekte, der Implementierung von Komponenten, dem Abrufen von Daten, der Statusverwaltung und dem Styling. Verwendung bei: Erstellen neuer React/Next.js-Projekte, Implementieren von Komponenten mit shadcn/ui, Einrichten des Datenabrufs, Auswählen von Statusverwaltungslösungen, Konfigurieren von Tailwind CSS v4.
