Примените методы карманного справочника по конвейерам данных (Джеймс Денсмор). Охватывает инфраструктуру (гл. 1–2: склады, озера, облако), шаблоны (гл. 3: ETL, ELT, CDC), прием БД (гл. 4: MySQL, PostgreSQL, MongoDB, полный/инкрементный), прием файлов (гл. 5: CSV, JSON, облачное хранилище), прием API (гл. 6: REST, нумерация страниц, ограничение скорости), потоковую передачу (гл. 7: Kafka, Kinesis, управление событиями), хранилище (глава 8: Redshift, BigQuery, Snowflake), преобразования (глава 9: SQL, Python, dbt), проверка (глава 10: большие надежды, проверки схемы), оркестровка (глава 11: воздушный поток, DAG, планирование), мониторинг (глава 12: соглашения об уровне обслуживания, оповещения), лучшие практики (глава 13: идемпотентность, обратное заполнение, обработка ошибок). Триггер по «конвейеру данных», «ETL», «ELT», «приему данных», «Воздушному потоку», «dbt», «хранилищу данных», «потоковой передаче Kafka», «CDC», «оркестрации данных».
