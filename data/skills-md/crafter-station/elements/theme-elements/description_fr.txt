Installez les composants du sélecteur de thème à partir du registre Elements. À utiliser lorsque l'utilisateur a besoin de bascules en mode sombre, de sélecteurs de thème, de boutons clair/sombre ou de listes déroulantes de thèmes. Déclenche le « sélecteur de thème », le « mode sombre », le « mode clair », le « basculement de thème », le « bouton de thème », le « changement de thème », le « basculement du mode sombre ».
