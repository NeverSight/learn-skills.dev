Установите звуковые эффекты CC0 из реестра Elements. Используйте, когда пользователю нужны звуки пользовательского интерфейса (щелчок, хлопок, уведомление), звуки перехода (свист, свист), кинематографические звуки (бум, подъем, сбой) или звук для создания видео. Запускается по «звуковому эффекту», «SFX», «аудио», «звуку уведомления», «звуку щелчка», «свисту», «звуку пользовательского интерфейса», «звуку для видео», «воспроизведению звука», «useSound».
