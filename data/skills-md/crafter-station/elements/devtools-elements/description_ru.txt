Установите компоненты инструмента разработчика из реестра Elements. Используйте, когда пользователю нужны средства просмотра JSON, отображения ответов API, средства просмотра различий кода, средства визуализации вывода CLI, редакторы окружения, границы ошибок, тестеры веб-перехватчиков или средства просмотра схемы. Триггеры по «инструментам разработчика», «просмотру json», «ответу API», «тестеру веб-перехватчика», «редактору окружения», «разнице кода», «просмотру схемы», «границе ошибки», «выводу CLI».
