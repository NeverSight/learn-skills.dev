Installieren Sie GitHub-Anzeigekomponenten aus der Elements-Registrierung. Verwenden Sie diese Option, wenn der Benutzer GitHub-Repo-Karten, Profilkarten, Beitragskalender, Sternzähler, Sprachaufschlüsselungen oder Aktivitätsanzeigen benötigt. Löst auf „Github-Komponente“, „Github-Karte“, „Repo-Karte“, „Beitragskalender“, „Github-Sterne“, „Github-Profil“, „Github-Aktivität“, „Github-Sprachen“, „Stern-Schaltfläche“ aus.
