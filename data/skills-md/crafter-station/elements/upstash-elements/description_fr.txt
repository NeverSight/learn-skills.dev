Installez les composants sans serveur Upstash à partir du registre Elements. À utiliser lorsque l'utilisateur a besoin d'affichages de limite de débit, de compteurs Redis, de classements, de cartes d'état de file d'attente ou de badges de cache. Déclencheurs sur "upstash", "limite de débit", "compteur redis", "composant de classement", "carte de file d'attente", "badge de cache", "redis sans serveur".
