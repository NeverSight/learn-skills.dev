Configure la capa de intención jerárquica (archivos AGENTS.md) para las bases de código. Úselo al inicializar un nuevo proyecto, agregar infraestructura de contexto a un repositorio existente, el usuario solicita configurar AGENTS.md, agregar una capa de intención, hacer que los agentes comprendan la base del código o crear un andamiaje para la documentación del proyecto compatible con IA.
