Configurez une couche d'intention hiérarchique (fichiers AGENTS.md) pour les bases de code. À utiliser lors de l'initialisation d'un nouveau projet, de l'ajout d'une infrastructure de contexte à un dépôt existant, de l'utilisateur demande de configurer AGENTS.md, d'ajouter une couche d'intention, de faire comprendre aux agents la base de code ou d'élaborer une documentation de projet adaptée à l'IA.
