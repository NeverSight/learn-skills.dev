Testa visivamente le recenti modifiche git nel browser utilizzando Chrome DevTools MCP. Da utilizzare quando l'utente desidera "e2e audit", "test visivo delle modifiche recenti", "test dell'interfaccia utente", "controllo degli ultimi commit nel browser", "test di regressione delle mie modifiche", "test di fumo dell'interfaccia utente", "QA del mio lavoro recente", "QA visivo" o invoca /e2e-audit. Analizza N commit recenti, mappa i file modificati sui percorsi dell'interfaccia utente, naviga in ciascuno di essi in Chrome, interagisce con le funzionalità modificate, acquisisce screenshot, controlla la console e la rete e produce un rapporto strutturato sui problemi con screenshot organizzati.
