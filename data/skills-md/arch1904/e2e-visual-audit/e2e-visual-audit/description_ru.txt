Визуально проверьте последние изменения git в браузере с помощью Chrome DevTools MCP. Используйте, когда пользователь хочет «аудит e2e», «визуальную проверку последних изменений», «проверку пользовательского интерфейса», «проверку последних коммитов в браузере», «регрессионное тестирование моих изменений», «проверку пользовательского интерфейса», «проверку качества моей недавней работы», «визуальный контроль качества» или вызывает /e2e-audit. Анализирует N недавних коммитов, сопоставляет измененные файлы с маршрутами пользовательского интерфейса, перемещается по каждому из них в Chrome, взаимодействует с измененными функциями, делает снимки экрана, проверяет консоль и сеть и создает структурированный отчет о проблемах с организованными снимками экрана.
