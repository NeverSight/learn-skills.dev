Pruebe visualmente los cambios recientes de git en el navegador usando Chrome DevTools MCP. Úselo cuando el usuario quiera realizar una "auditoría e2e", "probar visualmente los cambios recientes", "probar la interfaz de usuario", "verificar las últimas confirmaciones en el navegador", "probar mis cambios con regresión", "probar la interfaz de usuario con humo", "control de calidad de mi trabajo reciente", "control de calidad visual" o invoca /e2e-audit. Analiza N confirmaciones recientes, asigna archivos modificados a rutas de UI, navega cada uno en Chrome, interactúa con funciones modificadas, toma capturas de pantalla, verifica la consola y la red y produce un informe de problemas estructurado con capturas de pantalla organizadas.
