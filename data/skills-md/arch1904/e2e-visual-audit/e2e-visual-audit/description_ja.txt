Chrome DevTools MCP を使用して、ブラウザで最近の git の変更を視覚的にテストします。ユーザーが「e2e 監査」、「最近の変更のビジュアル テスト」、「UI のテスト」、「ブラウザでの最後のコミットの確認」、「変更の回帰テスト」、「UI のスモーク テスト」、「最近の作業の QA」、「ビジュアル QA」を実行する場合、または /e2e-audit を呼び出す場合に使用します。最近の N 件のコミットを分析し、変更されたファイルを UI ルートにマッピングし、Chrome で各ファイルに移動し、変更された機能を操作し、スクリーンショットを撮り、コンソールとネットワークをチェックし、整理されたスクリーンショットを含む構造化された問題レポートを作成します。
