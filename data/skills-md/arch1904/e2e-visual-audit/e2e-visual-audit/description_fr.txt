Testez visuellement les modifications récentes de git dans le navigateur à l'aide de Chrome DevTools MCP. À utiliser lorsque l'utilisateur souhaite effectuer un "audit e2e", "un test visuel des modifications récentes", "un test de l'interface utilisateur", "une vérification des dernières validations dans le navigateur", un "test de régression de mes modifications", un "test de fumée de l'interface utilisateur", un contrôle qualité de mon travail récent, un contrôle qualité visuel ou invoque /e2e-audit. Analyse N commits récents, mappe les fichiers modifiés aux itinéraires de l'interface utilisateur, navigue dans chacun d'eux dans Chrome, interagit avec les fonctionnalités modifiées, prend des captures d'écran, vérifie la console et le réseau et produit un rapport de problèmes structuré avec des captures d'écran organisées.
