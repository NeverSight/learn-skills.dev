Chrome DevTools MCP를 사용하여 브라우저에서 최근 Git 변경 사항을 시각적으로 테스트합니다. 사용자가 "e2e 감사", "최근 변경 사항 시각적 테스트", "UI 테스트", "브라우저에서 마지막 커밋 확인", "내 변경 사항 회귀 테스트", "UI 스모크 테스트", "최근 작업 QA", "시각적 QA"를 원할 때 사용하거나 /e2e-audit를 호출합니다. N개의 최근 커밋을 분석하고, 변경된 파일을 UI 경로에 매핑하고, Chrome에서 각 파일을 탐색하고, 변경된 기능과 상호작용하고, 스크린샷을 찍고, 콘솔과 네트워크를 확인하고, 정리된 스크린샷으로 구조화된 문제 보고서를 생성합니다.
