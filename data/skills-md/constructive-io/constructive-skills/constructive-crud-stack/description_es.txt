Cree acciones CRUD como tarjetas Stack (paneles deslizables estilo iOS) para cualquier CRM constructivo. Cubre el patrón de activación de la tarjeta Stack, la estructura CardComponent, la API de la tarjeta (close/push/setTitle), useCardReady para carga diferida y confirmación-eliminación apilada. Para formas dinámicas que hacen introspección _meta en tiempo de ejecución, consulte la habilidad de metaformas constructivas.
