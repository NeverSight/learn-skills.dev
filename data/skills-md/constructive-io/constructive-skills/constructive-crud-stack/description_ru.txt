Создавайте действия CRUD в виде карточек стека (выдвижных панелей в стиле iOS) для любой Constructive CRM. Охватывает шаблон триггера карты стека, структуру CardComponent, API карты (close/push/setTitle), useCardReady для отложенной загрузки и составное подтверждение-удаление. Для получения информации о динамических формах, которые анализируют _meta во время выполнения, см. навык конструктивных мета-форм.
