Создайте историю, которая заставит инвесторов сказать: «Я хочу войти». Овладейте повествовательной структурой, которую Рид Хоффман, Sequoia и ведущие основатели используют для привлечения капитала. Используйте, когда: **Подготовка к сбору средств** для развития вашей истории перед колодой; **Встречи с инвесторами** для структурирования бесед, которые приносят конверсию; **Практикуйтесь**, чтобы уточнить сюжетную линию; **Холодное общение** для написания убедительных электронных писем инвесторам; **Поворот**, чтобы переосмыслить историю после изменений.
