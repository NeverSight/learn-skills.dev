Проверьте готовность платить перед запуском с помощью проверенных методологий исследования цен. Объедините методы Ван Вестендорпа, Габора-Грейнджера и поведенческие методы, чтобы найти оптимальную цену. Используйте, когда: **После проверки решения** для проверки готовности платить; **Перед запуском**, чтобы установить первоначальную цену; **Изменение цен** для тестирования новых цен; **Новые сегменты**, чтобы понять чувствительность цен по сегментам; **Конкурентное позиционирование** для установления цен на альтернативы
