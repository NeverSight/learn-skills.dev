Comprenez pourquoi les clients achètent réellement en découvrant le « travail » pour lequel ils embauchent votre produit. À utiliser dans les cas suivants : **Comprendre la motivation des clients** au-delà des données démographiques et des demandes de fonctionnalités ; **Trouver l'adéquation produit-marché** en identifiant les progrès réels recherchés par les clients ; **Découvrir pourquoi les clients passent** (ou non) d'une solution à l'autre ; **Identifier la véritable concurrence** qui n'est pas évidente dans les catégories industrielles ; **Créer des messages marketing** qui correspondent aux véritables difficultés des clients
