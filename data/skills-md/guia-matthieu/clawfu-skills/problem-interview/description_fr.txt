Validez que le problème que vous souhaitez résoudre est réel, douloureux et mérite d’être résolu avant de construire quoi que ce soit. Maîtrisez l'approche structurée de Cindy Alvarez pour les entretiens de découverte de problèmes. À utiliser lorsque : **Avant les entretiens de solution** pour confirmer que le problème existe ; **Découverte précoce des clients** pour comprendre l'espace du problème ; **Pivoter** pour trouver de nouveaux problèmes qui méritent d'être résolus ; **Expansion du marché** pour comprendre les problèmes dans de nouveaux segments ; **Priorité des fonctionnalités** pour valider les problèmes les plus importants
