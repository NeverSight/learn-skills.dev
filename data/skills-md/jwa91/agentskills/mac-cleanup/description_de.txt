Interaktive macOS-Systembereinigung für jede Entwicklungsmaschine. Gibt Speicherplatz frei, indem Caches, Paketmanager, ungenutzte Apps, veraltete Entwicklungsartefakte und mehr gelöscht werden. Erkennt, was installiert ist, anstatt ein bestimmtes Setup anzunehmen. Konsultieren Sie immer den Benutzer, bevor Sie etwas löschen. Verwenden Sie es, wenn der Benutzer Folgendes wünscht: Bereinigen Sie seinen Mac, geben Sie Speicherplatz frei, entfernen Sie nicht verwendete Apps, bereinigen Sie Caches, bereinigen Sie Entwicklerartefakte oder führen Sie eine Wartungsaufgabe für den Speicherplatz aus.
