Nettoyage interactif du système macOS pour n'importe quelle machine de développement. Libère de l'espace disque en éliminant les caches, les gestionnaires de packages, les applications inutilisées, les artefacts de développement obsolètes, etc. Découvre ce qui est installé plutôt que de supposer une configuration spécifique. Consulte toujours l’utilisateur avant de supprimer quoi que ce soit. À utiliser lorsque l'utilisateur demande : nettoyer son Mac, libérer de l'espace disque, supprimer les applications inutilisées, élaguer les caches, nettoyer les artefacts de développeur ou toute tâche de maintenance de l'espace disque.
