Sicherheits-Anti-Pattern für SQL-Injection-Schwachstellen (CWE-89). Wird zum Generieren oder Überprüfen von Code verwendet, der Datenbankabfragen erstellt, SQL-Anweisungen erstellt oder Benutzereingaben bei Datenbankvorgängen verarbeitet. Erkennt Zeichenfolgenverkettungen in Abfragen und empfiehlt parametrisierte Abfragen.
