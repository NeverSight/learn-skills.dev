Sicherheits-Anti-Pattern für den Debug-Modus in der Produktion (CWE-215). Wird zum Generieren oder Überprüfen von Code verwendet, der Anwendungseinstellungen, Bereitstellungskonfigurationen oder Fehlerbehandlung konfiguriert. Erkennt hartcodierte Debug-Flags und reine Entwicklungsfunktionen in der Produktion.
