Anti-modello di sicurezza per le vulnerabilità di attraversamento del percorso (CWE-22). Da utilizzare durante la generazione o la revisione del codice che gestisce i percorsi dei file, legge o scrive file in base all'input dell'utente o fornisce contenuto statico. Rileva l'unione dell'input dell'utente a percorsi senza un'adeguata sanificazione o convalida.
