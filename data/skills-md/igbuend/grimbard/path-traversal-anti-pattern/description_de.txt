Sicherheits-Anti-Pattern für Path-Traversal-Schwachstellen (CWE-22). Wird zum Generieren oder Überprüfen von Code verwendet, der Dateipfade verarbeitet, Dateien basierend auf Benutzereingaben liest oder schreibt oder statische Inhalte bereitstellt. Erkennt die Verknüpfung von Benutzereingaben mit Pfaden ohne ordnungsgemäße Bereinigung oder Validierung.
