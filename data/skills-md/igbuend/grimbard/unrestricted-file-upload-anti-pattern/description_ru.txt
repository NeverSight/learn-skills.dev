Антишаблон безопасности для уязвимостей неограниченной загрузки файлов (CWE-434). Используйте при создании или проверке кода, который обрабатывает загрузку файлов, обрабатывает отправленные пользователем файлы или сохраняет загруженный контент. Обнаруживает отсутствующее расширение, тип MIME и проверку размера.
