Recherchez sur le Web et lisez le contenu des pages sans clés API. À utiliser lorsque vous devez effectuer une recherche via DuckDuckGo/Brave/Google (multipage), extraire du texte lisible à partir d'URL, naviguer de manière interactive avec un navigateur visible persistant (avec onglets, clic, capture d'écran, recherche de texte), télécharger des fichiers/PDF ou ignorer les bannières de cookies. Prend en charge la sortie JSON/markdown/texte. Propulsé par Playwright + Chromium.
