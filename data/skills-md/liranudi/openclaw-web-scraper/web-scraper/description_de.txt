Durchsuchen Sie das Web und lesen Sie Seiteninhalte ohne API-Schlüssel. Verwenden Sie diese Option, wenn Sie über DuckDuckGo/Brave/Google (mehrseitig) suchen, lesbaren Text aus URLs extrahieren, interaktiv mit einem dauerhaft sichtbaren Browser surfen (mit Tabs, Klick, Screenshot, Textsuche), Dateien/PDFs herunterladen oder Cookie-Banner schließen müssen. Unterstützt JSON/Markdown/Textausgabe. Unterstützt von Playwright + Chromium.
