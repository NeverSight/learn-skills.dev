Выполняйте поиск в Интернете и читайте содержимое страниц без ключей API. Используйте, когда вам нужно выполнять поиск через DuckDuckGo/Brave/Google (многостраничный), извлекать читаемый текст из URL-адресов, просматривать в интерактивном режиме с помощью постоянно видимого браузера (с вкладками, щелчком, снимком экрана, поиском текста), загружать файлы/PDF-файлы или отклонять баннеры cookie. Поддерживает вывод JSON/markdown/текста. Работает на платформе Playwright + Chromium.
