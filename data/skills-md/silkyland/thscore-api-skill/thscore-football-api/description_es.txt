Guía de desarrollo completa para integrar la API THScore Football. Cubre autenticación, selección de servidor regional, todos los puntos finales de fútbol (datos en vivo, estadísticas, probabilidades, animación en vivo, base de datos histórica), tipos de TypeScript, limitación de velocidad, almacenamiento en caché, manejo de errores e incrustación de iFrame/WebView. Activa cualquier tarea que involucre resultados de fútbol en vivo, horarios, clasificaciones, probabilidades, eventos, alineaciones, perfiles de equipos/jugadores o incrustaciones de animación en vivo.
