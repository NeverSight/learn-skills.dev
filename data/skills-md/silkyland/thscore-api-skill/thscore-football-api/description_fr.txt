Guide de développement complet pour l’intégration de l’API THScore Football. Couvre l'authentification, la sélection du serveur régional, tous les points de terminaison du football (données en direct, statistiques, cotes, animation en direct, base de données historique), les types TypeScript, la limitation de débit, la mise en cache, la gestion des erreurs et l'intégration iFrame/WebView. Déclenche toute tâche impliquant les résultats de football, les calendriers, les classements, les cotes, les événements, les compositions, les profils d'équipe/joueur ou l'intégration d'animations en direct.
