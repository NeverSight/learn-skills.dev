دليل التطوير الشامل لدمج THScore Football API. يغطي المصادقة، واختيار الخادم الإقليمي، وجميع نقاط نهاية كرة القدم (البيانات المباشرة، والإحصائيات، والاحتمالات، والرسوم المتحركة المباشرة، وقاعدة البيانات التاريخية)، وأنواع TypeScript، وتحديد المعدل، والتخزين المؤقت، ومعالجة الأخطاء، وتضمين iFrame/WebView. يتم تشغيلها في أي مهمة تتضمن النتائج المباشرة لكرة القدم، أو الجداول الزمنية، أو الترتيب، أو الاحتمالات، أو الأحداث، أو التشكيلات، أو الملفات الشخصية للفريق/اللاعب، أو تضمين الرسوم المتحركة المباشرة.
