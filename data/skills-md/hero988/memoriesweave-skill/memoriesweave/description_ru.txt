Создает коллекции воспоминаний фотографий в MemoriesWeave и управляет ими через REST API. Используйте, когда пользователь упоминает MemoriesWeave, коллекции воспоминаний, фотообои, фотокниги или хочет создавать/редактировать цифровые воспоминания или печатать продукты с помощью API MemoriesWeave. Не используйте для общего редактирования фотографий, создания изображений или задач, не связанных с MemoriesWeave.
