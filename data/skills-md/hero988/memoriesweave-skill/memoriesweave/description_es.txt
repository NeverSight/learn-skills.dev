Crea y administra colecciones de recuerdos fotográficos en MemoriesWeave a través de su API REST. Úselo cuando el usuario mencione MemoriesWeave, colecciones de recuerdos, fondos de pantalla de fotografías, libros de fotografías o desee crear/editar recuerdos digitales o imprimir productos utilizando la API de MemoriesWeave. No lo utilice para edición general de fotografías, generación de imágenes o tareas que no sean de MemoriesWeave.
