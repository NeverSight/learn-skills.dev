Récupérez TOUTES les conversations PR non résolues - fils de discussion, commentaires du corps de révision et commentaires PR réguliers - puis générez un plan /tdd strict avec des validations atomiques qui corrigent et répondent à chacun. Utilisez cette compétence chaque fois que l'utilisateur souhaite résoudre, traiter ou corriger les commentaires, les fils de discussion, les remarques ou les suggestions d'un réviseur (humain ou robot comme CodeRabbit, Copilot, etc.). Se déclenche sur des expressions telles que « résoudre les fils de discussion PR », « corriger les commentaires PR », « adresser les commentaires de révision », « résoudre les conversations ».
