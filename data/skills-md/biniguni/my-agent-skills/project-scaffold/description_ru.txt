Автоматически генерирует стандартные структуры каталогов для различных типов проектов (анализ данных, машинное/глубокое обучение, статистическое моделирование и агенты искусственного интеллекта). Используйте этот навык всякий раз, когда пользователь хочет начать новый проект и ему необходимо быстро создать стандартизированную структуру папок для своей конкретной задачи.
