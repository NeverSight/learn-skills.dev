Экспертный агент по созданию документов Word для macOS с использованием Python-docx и AppleScript. Создает и редактирует потрясающие профессиональные документы высочайшего качества. Используйте, когда: (1) Создание новых документов Word с нуля с помощью Python-docx, (2) Редактирование или перепроектирование существующих файлов .docx, (3) Создание документов с индивидуальным дизайном (обложки, стилизованные таблицы, выноски, цитаты, боковые панели, панели KPI), (4) Редактирование документов в реальном времени с помощью AppleScript IPC (текст, шрифты, абзацы, поиск/замена, поля, TOC), (5) Обновление полей, TOC или экспорт в PDF через AppleScript в macOS, (6) Создание изображений AI для обложек и содержимого документов, (7) Любая задача, требующая создания кода Python-docx с использованием лучших практик проектирования.
