Комплексный набор инструментов для построения технических диаграмм для архитекторов решений, специалистов по предпродажной подготовке и разработчиков. Создает диаграммы архитектуры Azure (более 700 официальных значков Microsoft), последовательности бизнес-процессов (дорожки для плавания, рабочие процессы), диаграммы ERD (схемы баз данных), временные рамки проекта, каркасные схемы пользовательского интерфейса и диаграммы топологии сети. Идеально подходит для предложений, документации и обзоров архитектуры. Также генерирует диаграммы на основе шаблонов Bicep, Terraform и ARM.
