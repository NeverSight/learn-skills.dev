Capture experiencia automáticamente y desarrolle habilidades de forma segura. Desencadenantes: (1) el comando/acción falla → registrar ERRORS.md (2) el usuario corrige (“incorrecto”/“debería ser”) → registrar LEARNINGS.md (3) el usuario necesita una capacidad inexistente → registrar FEATURE_REQUESTS.md (4) error de API/herramienta externa → registrar ERRORS.md (5) conocimiento propio desactualizado/incorrecto → registrar LEARNINGS.md (6) mejores prácticas encontradas → registrar LEARNINGS.md (7) cada tarea completada → revisar el proceso, registrar LEARNINGS.md si es una experiencia nueva. Deduplicación: omítala si no tiene experiencia nueva o ya está cubierta. Agregue la entrada JSONL a .learnings/CHANGELOG.md en cada escritura. Ascienda a AGENTS.md/TOOLS.md/SOUL.md después de ≥3 repeticiones. Consulte el texto completo para obtener más detalles.
