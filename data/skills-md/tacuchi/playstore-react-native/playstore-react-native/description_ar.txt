قم بتوجيه ومراجعة الإعداد خطوة بخطوة لمشروع React Native لإصدار متجر Google Play. يغطي إنشاء ملف تخزين المفاتيح وتكوين التوقيع وبناء ProGuard/R8 وAAB. يُستخدم عندما يريد المستخدم نشر تطبيق React Native أو إصداره أو نشره أو إعداده لمتجر Play Store، أو عندما يذكر "react-native build-android" أو "bundleRelease React Native" أو "Play Store React Native". ملحوظة: بالنسبة للمشروعات التي يديرها Expo، استخدم "eas build" بدلاً من ذلك.
