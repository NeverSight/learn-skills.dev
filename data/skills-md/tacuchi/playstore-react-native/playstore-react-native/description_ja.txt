Google Play ストア リリースに向けた React Native プロジェクトの段階的な準備をガイドし、確認します。キーストアの生成、署名構成、ProGuard/R8、および AAB ビルドについて説明します。ユーザーが Play ストア用の React Native アプリを公開、リリース、デプロイ、または準備したい場合、または「react-native build-android」、「bundleRelease、reactnative」、または「play ストア リアクト ネイティブ」について言及する場合に使用します。注: Expo 管理プロジェクトの場合は、代わりに「eas build」を使用してください。
