Guidez et passez en revue la préparation étape par étape d'un projet React Native pour la version Google Play Store. Couvre la génération de magasins de clés, la configuration de signature, ProGuard/R8 et la version AAB. À utiliser lorsque l'utilisateur souhaite publier, publier, déployer ou préparer une application React Native pour Play Store, ou lorsqu'il mentionne « react-native build-android », « bundleRelease react native » ou « Play Store react native ». Remarque : pour les projets gérés par Expo, utilisez plutôt "eas build".
