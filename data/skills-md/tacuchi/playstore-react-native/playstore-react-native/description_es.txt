Guíe y revise la preparación paso a paso de un proyecto React Native para el lanzamiento de Google Play Store. Cubre la generación de almacenes de claves, configuración de firmas, ProGuard/R8 y compilación de AAB. Úselo cuando el usuario quiera publicar, lanzar, implementar o preparar una aplicación React Native para Play Store, o cuando mencione "react-native build-android", "bundleRelease reaccionar nativo" o "play store reaccionar nativo". Nota: para proyectos administrados por Expo, utilice "eas build" en su lugar.
