Руководство и обзор пошаговой подготовки проекта React Native к выпуску в Google Play Store. Охватывает создание хранилища ключей, конфигурацию подписи, ProGuard/R8 и сборку AAB. Используйте, когда пользователь хочет опубликовать, выпустить, развернуть или подготовить приложение React Native для Play Store или когда он упоминает «react-native build-android», «bundleRelease React Native» или «Play Store React Native». Примечание. Для проектов, управляемых Expo, вместо этого используйте «eas build».
