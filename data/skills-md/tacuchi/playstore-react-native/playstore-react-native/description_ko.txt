Google Play 스토어 릴리스를 위한 React Native 프로젝트의 단계별 준비를 안내하고 검토하세요. 키 저장소 생성, 서명 구성, ProGuard/R8 및 AAB 빌드를 다룹니다. 사용자가 Play Store용 React Native 앱을 게시, 출시, 배포 또는 준비하려고 하거나 "react-native build-android", "bundleRelease React Native" 또는 "play store React Native"를 언급할 때 사용하세요. 참고: Expo 관리 프로젝트의 경우 대신 "eas build"를 사용하세요.
