Guida e rivedi la preparazione passo passo di un progetto React Native per il rilascio su Google Play Store. Copre la generazione del keystore, la configurazione della firma, ProGuard/R8 e la build AAB. Da utilizzare quando l'utente desidera pubblicare, rilasciare, distribuire o preparare un'app React Native per Play Store o quando menziona "react-native build-android", "bundleRelease react native" o "play store react native". Nota: per i progetti gestiti da Expo, utilizzare invece "eas build".
