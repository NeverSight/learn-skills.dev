Anleitung und Überprüfung der schrittweisen Vorbereitung eines React Native-Projekts für die Veröffentlichung im Google Play Store. Behandelt die Keystore-Generierung, die Signierungskonfiguration, ProGuard/R8 und den AAB-Build. Verwenden Sie diese Option, wenn der Benutzer eine React Native-App für den Play Store veröffentlichen, freigeben, bereitstellen oder vorbereiten möchte oder wenn er „react-native build-android“, „bundleRelease React Native“ oder „Play Store React Native“ erwähnt. Hinweis: Verwenden Sie für von Expo verwaltete Projekte stattdessen „eas build“.
