Génère des fichiers de documentation EventCatalog (services, événements, commandes, requêtes, domaines, flux, canaux, conteneurs) avec le contenu, la structure des dossiers et les meilleures pratiques corrects. À utiliser lorsque l'utilisateur demande de « documenter un service », « créer des fichiers EventCatalog », « ajouter un événement au catalogue », « documenter mon architecture », « générer une documentation de catalogue », « créer une documentation pour mon microservice » ou « documenter une base de données ».
