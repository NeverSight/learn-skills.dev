Создает файлы документации EventCatalog (службы, события, команды, запросы, домены, потоки, каналы, контейнеры) с правильным оформлением, структурой папок и рекомендациями. Используйте, когда пользователь просит «задокументировать службу», «создать файлы EventCatalog», «добавить событие в каталог», «документировать мою архитектуру», «создать документацию каталога», «создать документацию для моего микросервиса» или «документировать базу данных».
