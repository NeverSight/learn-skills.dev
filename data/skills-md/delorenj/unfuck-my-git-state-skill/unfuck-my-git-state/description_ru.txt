Диагностика и восстановление нарушенного состояния Git и метаданных рабочего дерева с помощью поэтапного процесса восстановления с низким уровнем риска. Используйте, когда Git сообщает об отсоединенном или противоречивом состоянии HEAD, фантомных блокировках рабочего дерева, потерянных записях рабочего дерева, отсутствующих ссылках, 0000000000000000000000000000000000000000 хешей или операциях ветвей завершается с ошибкой, например, уже извлечено, неизвестная ревизия, недопустимое имя объекта или невозможно заблокировать ссылку.
