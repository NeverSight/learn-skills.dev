Diagnostique y recupere metadatos del árbol de trabajo y estados de Git dañados con un flujo de recuperación por etapas y de bajo riesgo. Úselo cuando Git informe un estado HEAD desconectado o contradictorio, bloqueos de árbol de trabajo fantasma, entradas de árbol de trabajo huérfanos, referencias faltantes, 0000000000000000000000000000000000000000 hashes o las operaciones de rama fallan con errores como ya desprotegido, revisión desconocida, nombre de objeto no válido o no se puede bloquear la referencia.
