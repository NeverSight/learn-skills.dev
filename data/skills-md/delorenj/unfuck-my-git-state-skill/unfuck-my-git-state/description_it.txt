Diagnostica e ripristina lo stato Git e i metadati dell'albero di lavoro danneggiati con un flusso di ripristino graduale e a basso rischio. Da utilizzare quando Git segnala uno stato HEAD scollegato o contraddittorio, blocchi dell'albero di lavoro fantasma, voci dell'albero di lavoro orfane, riferimenti mancanti, 0000000000000000000000000000000000000000 hash o operazioni di ramo falliscono con errori come già estratto, revisione sconosciuta, nome oggetto non valido o impossibile bloccare il riferimento.
