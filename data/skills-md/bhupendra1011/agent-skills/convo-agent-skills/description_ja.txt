Agora SDK と Next.js を使用して、リアルタイムの会話型 AI アプリケーションを構築します。音声エージェント、ビデオ エージェント、AI エージェントのライフサイクル、ライブ トランスクリプト/字幕、RTM メッセージング/チャット、アバター統合 (HeyGen/Anam/Akool)、設定の永続性、MCP ツールの呼び出しをカバーします。ユーザーとエージェントのみ。ホスト制御はありません。次のベストプラクティスのスキルと併用してください。ユーザーが会話型 AI アプリ、音声/ビデオ エージェント、Agora とのリアルタイム通信を構築する場合、または AI エージェント機能を Next.js アプリに追加する場合にアクティブ化します。
