Cree aplicaciones de IA conversacionales en tiempo real utilizando Agora SDK y Next.js. Cubre agentes de voz, agentes de video, ciclo de vida del agente de IA, transcripción/subtítulos en vivo, mensajería/chat RTM, integración de avatar (HeyGen/Anam/Akool), persistencia de configuraciones y llamadas a herramientas MCP. Sólo usuario y agente; sin controles de host. Úselo junto con la siguiente habilidad de mejores prácticas. Se activa cuando el usuario desea crear una aplicación de IA conversacional, un agente de voz/video, comunicación en tiempo real con Agora o agregar funciones de agente de IA a una aplicación Next.js.
