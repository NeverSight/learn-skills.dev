Créez des applications d'IA conversationnelle en temps réel à l'aide des SDK Agora et Next.js. Couvre les agents vocaux, les agents vidéo, le cycle de vie des agents IA, la transcription/sous-titres en direct, la messagerie/chat RTM, l'intégration d'avatar (HeyGen/Anam/Akool), la persistance des paramètres et les appels d'outils MCP. Utilisateur et agent uniquement ; aucun contrôle de l'hôte. À utiliser avec les compétences suivantes en matière de meilleures pratiques. S'active lorsque l'utilisateur souhaite créer une application d'IA conversationnelle, un agent vocal/vidéo, une communication en temps réel avec Agora ou ajouter des fonctionnalités d'agent d'IA à une application Next.js.
