Охватывает голосовые агенты, видеоагенты, жизненный цикл агента искусственного интеллекта, расшифровку/субтитры в реальном времени, обмен сообщениями/чатом RTM, интеграцию аватаров (HeyGen/Anam/Akool), сохранение настроек и вызов инструмента MCP. Только пользователь и агент; нет контроля со стороны хоста. Используйте наряду с передовыми навыками. Активируется, когда пользователь хочет создать диалоговое приложение AI, голосовой/видеоагент, связь в реальном времени с Agora или добавить функции агента AI в приложение Next.js.
