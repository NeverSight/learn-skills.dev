قم بتطوير سمات WordPress وفقًا لمعايير ترميز WordPress الرسمية ومتطلبات مراجعة السمات ومعايير إمكانية الوصول (WCAG 2.1 AA) وأفضل الممارسات الحديثة لتطوير الويب. استخدم هذه المهارة عندما يريد المستخدم إنشاء سمة WordPress أو دعمها أو تطويرها - بما في ذلك سمات الكتلة (FSE) أو السمات الكلاسيكية أو السمات الفرعية أو السمات المختلطة. يتم تشغيله أيضًا عندما يذكر المستخدم "موضوع WordPress"، أو "موضوع WP"، أو "موضوع الحظر"، أو "موضوع FSE"، أو "theme.json"، أو "أجزاء القالب"، أو "أشكال النمط المختلفة"، أو "السمة الفرعية"، أو يريد إنشاء تصميم/تخطيط مخصص لـ WordPress. حتى لو قال المستخدم فقط "قم ببناء سمة لـ X" أو "أحتاج إلى سمة"، استخدم هذه المهارة.
