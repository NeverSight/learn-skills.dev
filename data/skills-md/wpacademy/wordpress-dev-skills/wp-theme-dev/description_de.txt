Entwickeln Sie WordPress-Themes gemäß den offiziellen WordPress-Codierungsstandards, Anforderungen an die Themenüberprüfung, Barrierefreiheitsstandards (WCAG 2.1 AA) und modernen Best Practices für die Webentwicklung. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer ein WordPress-Theme erstellen, ein Gerüst erstellen oder entwickeln möchte – einschließlich Block-Themes (FSE), klassischen Themes, untergeordneten Themes oder Hybrid-Themes. Wird auch ausgelöst, wenn der Benutzer „WordPress-Theme“, „WP-Theme“, „Block-Theme“, „FSE-Theme“, „theme.json“, „Vorlagenteile“, „Stilvariationen“, „untergeordnetes Theme“ erwähnt oder ein benutzerdefiniertes Design/Layout für WordPress erstellen möchte. Auch wenn der Benutzer nur sagt „Erstelle mir ein Theme für X“ oder „Ich brauche ein Theme“, nutzen Sie diese Fähigkeit.
