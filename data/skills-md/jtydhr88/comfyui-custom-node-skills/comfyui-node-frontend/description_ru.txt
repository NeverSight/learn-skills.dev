Расширения JavaScript для интерфейса ComfyUI — хуки, виджеты, вкладки боковой панели, команды, настройки, всплывающие уведомления, диалоговые окна. Используйте при добавлении функций пользовательского интерфейса к пользовательским узлам, создании пользовательских виджетов или расширении интерфейса ComfyUI.
