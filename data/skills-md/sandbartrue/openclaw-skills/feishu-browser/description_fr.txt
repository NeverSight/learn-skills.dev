Simulez des opérations en personne réelle sur la version Web de Feishu via un navigateur. Cela inclut l'envoi/la lecture de messages de discussion de groupe, le @mentionnement de personnes, la création de documents et la définition d'autorisations sur les documents. Utilisez cette compétence lorsque des opérations doivent être effectuées sur la page Web Feishu à l'aide du compte WALI.
