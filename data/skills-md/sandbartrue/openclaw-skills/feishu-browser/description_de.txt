Simulieren Sie reale Vorgänge in der Feishu-Webversion über einen Browser. Dazu gehören das Senden/Lesen von Gruppenchat-Nachrichten, das @Erwähnen von Personen, das Erstellen von Dokumenten und das Festlegen von Dokumentberechtigungen. Verwenden Sie diese Fähigkeit, wenn Vorgänge auf der Feishu-Webseite mit dem WALI-Konto ausgeführt werden müssen.
