Simula operazioni reali sulla versione web di Feishu tramite un browser. Ciò include l'invio/lettura di messaggi di chat di gruppo, la @menzione di persone, la creazione di documenti e l'impostazione delle autorizzazioni per i documenti. Utilizzare questa abilità quando è necessario eseguire operazioni sulla pagina web Feishu utilizzando l'account WALI.
