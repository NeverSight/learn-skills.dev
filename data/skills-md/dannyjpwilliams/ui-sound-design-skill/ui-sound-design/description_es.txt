Diseño de sonido UI programático utilizando Web Audio API y Tone.js. Úselo al crear sonidos de clic, timbres de notificación, comentarios de alternancia, sonidos de desplazamiento, audio de éxito/error, efectos de silbido o al crear una biblioteca de sonidos para interacciones de la interfaz de usuario. Proporciona un flujo de trabajo iterativo para describir, generar, escuchar y refinar con conocimientos de ingeniería de audio traducidos a un inglés sencillo.
