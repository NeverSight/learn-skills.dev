Программный звуковой дизайн пользовательского интерфейса с использованием Web Audio API и Tone.js. Используйте при создании звуков щелчков, звуков уведомлений, обратной связи при переключении, звуков наведения, звука успеха/ошибки, свистящих эффектов или создания библиотеки звуков для взаимодействия с пользовательским интерфейсом. Обеспечивает итеративный рабочий процесс «описать-создать-прослушать-уточнить», используя знания в области аудиотехники, переведенные на простой английский язык.
