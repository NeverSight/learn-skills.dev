Claude Code のアンチスロップ エージェント コーディング ワークフロー。ユーザーがエージェントティック コーディング ワークフローを設定する場合、品質ゲートを適用する場合、技術的負債を防止する場合、正確なタスク仕様を作成する場合、複雑な機能をアトミック単位に分解する場合、または仕様→実行→検証ループを実行する場合には、このスキルを使用します。 「Claude Code 用にプロジェクトをセットアップする」、「これをタスクに分割する」、「仕様を書く」、「手抜きなし」、「品質ゲート」、「コミット前フック」、「エージェント ワークフロー」、「コード ワークフローを出荷」などのフレーズ、または品質とトレーサビリティが重要な場合に、Claude Code を使用して重要な何かを構築するリクエストをトリガーします。
