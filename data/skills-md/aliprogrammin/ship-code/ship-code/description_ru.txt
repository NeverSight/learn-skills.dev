Рабочий процесс агентного кодирования, предотвращающий провалы, для Claude Code. Используйте этот навык всякий раз, когда пользователь хочет настроить рабочий процесс агентного кодирования, обеспечить контроль качества, предотвратить технический долг, написать точные спецификации задач, разложить сложные функции на атомарные единицы или запустить цикл спецификация → выполнение → проверка. Запускайте такие фразы, как «настроить мой проект для Claude Code», «разбить его на задачи», «написать спецификацию», «без помоев», «контроль качества», «перехватчики предварительной фиксации», «агентский рабочий процесс», «рабочий процесс поставки кода» или любой запрос на создание чего-то нетривиального с помощью Claude Code, где качество и отслеживаемость имеют значение.
