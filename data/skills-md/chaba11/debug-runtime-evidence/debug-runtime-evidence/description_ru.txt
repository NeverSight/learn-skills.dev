Отладка на основе гипотез с обязательным инструментированием во время выполнения. Используйте, когда пользователь запрашивает отладку, исследование неожиданного поведения, добавление журналирования, сбор данных во время выполнения или хочет процесс отладки, основанный на гипотезах. Запускается по: «отладка», «исследование», «гипотеза», «добавление журнала», «доказательства времени выполнения», «почему это не работает».
