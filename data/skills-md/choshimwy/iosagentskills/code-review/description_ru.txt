Навыки проверки кода iOS/Swift. Используется, когда просят просмотреть код, изучить различия PR, проверить качество кода или оценить дизайн общедоступного API. Проверьте в порядке правильность → безопасность → память → параллелизм → производительность → ремонтопригодность → согласованность и выведите оценочный отчет.
