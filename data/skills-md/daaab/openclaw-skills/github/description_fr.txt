Opérations GitHub via la CLI `gh` : problèmes, PR, exécutions de CI, révision de code, requêtes API. À utiliser lorsque : (1) vérifier l'état des PR ou des CI, (2) créer/commenter des problèmes, (3) répertorier/filtrer les PR ou les problèmes, (4) consulter les journaux d'exécution. PAS pour : les interactions complexes de l'interface utilisateur Web nécessitant des flux de navigateur manuels (utilisez les outils de navigateur lorsqu'ils sont disponibles), des opérations groupées sur de nombreux dépôts (script avec gh api) ou lorsque gh auth n'est pas configuré.
