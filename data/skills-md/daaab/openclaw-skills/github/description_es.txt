Operaciones de GitHub a través de la CLI `gh`: problemas, relaciones públicas, ejecuciones de CI, revisión de código, consultas de API. Úselo cuando: (1) verifique el estado de PR o CI, (2) cree/comente problemas, (3) enumere/filtre PR o problemas, (4) vea registros de ejecución. NO para: interacciones complejas de la interfaz de usuario web que requieren flujos manuales del navegador (use herramientas del navegador cuando estén disponibles), operaciones masivas en muchos repositorios (script con gh api) o cuando gh auth no está configurado.
