عمليات GitHub عبر `gh` CLI: المشكلات، العلاقات العامة، تشغيل CI، مراجعة التعليمات البرمجية، استعلامات API. يُستخدم عندما: (1) التحقق من حالة العلاقات العامة أو CI، (2) إنشاء/التعليق على المشكلات، (3) سرد/تصفية العلاقات العامة أو المشكلات، (4) عرض سجلات التشغيل. ليس من أجل: تفاعلات واجهة مستخدم الويب المعقدة التي تتطلب تدفقات يدوية للمتصفح (استخدم أدوات المتصفح عندما تكون متاحة)، أو العمليات المجمعة عبر العديد من عمليات إعادة الشراء (البرنامج النصي مع gh api)، أو عندما لا يتم تكوين gh auth.
