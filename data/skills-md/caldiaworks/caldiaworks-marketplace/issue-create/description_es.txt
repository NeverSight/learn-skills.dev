Cree problemas de GitHub a partir de la entrada del usuario o documentos de requisitos de USDM. El modo independiente guía a través del título, el cuerpo y las etiquetas. El modo USDM crea una jerarquía de problemas a partir de requisitos con especificaciones en el cuerpo del problema. Úselo cuando: crear un problema, un problema nuevo, un problema de GitHub, un problema abierto, un problema de archivo, crear problemas a partir de requisitos, USDM a problemas, crear un problema, /crear un problema.
