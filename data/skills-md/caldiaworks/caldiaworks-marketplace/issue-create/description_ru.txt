Создавайте задачи GitHub на основе вводимых пользователем данных или документов с требованиями USDM. Автономный режим позволяет управлять заголовком, текстом и метками. Режим USDM создает иерархию задач из требований со спецификациями в теле задачи. Используйте, когда: создать задачу, новую задачу, задачу GitHub, открыть задачу, проблему с файлом, создать задачу из требований, USDM в задачу, задачу-создать, /issue-create.
