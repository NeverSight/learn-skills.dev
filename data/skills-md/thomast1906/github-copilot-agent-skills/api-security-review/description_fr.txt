Examine les configurations de gestion des API Azure pour les vulnérabilités de sécurité, la conformité OWASP API Security Top 10, la validation du mode interne du réseau virtuel, la vérification du lien privé et l'alignement d'Azure Security Benchmark. À utiliser lors de la réalisation d'audits de sécurité, de validations préalables au déploiement ou d'examens de conformité.
