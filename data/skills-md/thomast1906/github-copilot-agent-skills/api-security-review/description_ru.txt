Проверяет конфигурации управления API Azure на наличие уязвимостей безопасности, соответствие требованиям OWASP API Security Top 10, проверку внутреннего режима виртуальной сети, проверку частной ссылки и соответствие теста безопасности Azure. Используйте при проведении аудита безопасности, проверки перед развертыванием или проверки соответствия.
