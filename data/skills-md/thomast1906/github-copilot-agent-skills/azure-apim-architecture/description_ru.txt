Анализирует и объясняет решения по архитектуре управления API Azure для реализаций корпоративного рынка API с использованием внутреннего режима виртуальной сети, Front Door, гибридной проверки подлинности и стратегий работы с несколькими средами. Используйте при обсуждении выбора компонентов APIM, топологии сети, оптимизации затрат или сравнении альтернатив, таких как рабочие области и экземпляры, внутренний режим виртуальной сети и внешний режим или входная дверь и шлюз приложений.
