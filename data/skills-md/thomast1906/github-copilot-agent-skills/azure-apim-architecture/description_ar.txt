يحلل ويشرح قرارات هندسة Azure API Management لتطبيقات سوق API للمؤسسات باستخدام الوضع الداخلي VNet، والباب الأمامي، والمصادقة المختلطة، واستراتيجيات البيئات المتعددة. يُستخدم عند مناقشة اختيار مكون APIM، أو هيكل الشبكة، أو تحسين التكلفة، أو مقارنة البدائل مثل مساحات العمل مقابل المثيلات، أو وضع VNet الداخلي مقابل الوضع الخارجي، أو الباب الأمامي مقابل بوابة التطبيق.
