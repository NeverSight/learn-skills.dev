如何使用react-i18next和i18next在React应用程序中实现国际化（i18n）。每当用户想要向 React 应用程序添加多语言支持、翻译 UI 内容、设置 i18next 配置、使用 useTranslation 钩子、withTranslation HOC、Trans 组件、处理命名空间、复数、插值或 React 中任何与本地化相关的任务时，请使用此技能。当用户提到“i18n”、“i18next”、“react-i18next”、“本地化”、“翻译”、“多语言”、“国际化”、“区域设置”、“语言切换器”或想要在 React 组件中翻译字符串/JSX 时也会触发 - 即使它们没有明确命名库。也触发 Next.js/Remix/Gatsby i18n 问题，因为react-i18next 是底层解决方案。
