如何使用react-i18next和i18next在React應用程式中實現國際化（i18n）。每當使用者想要為 React 應用程式新增多語言支援、翻譯 UI 內容、設定 i18next 設定、使用 useTranslation 鉤子、withTranslation HOC、Trans 元件、處理命名空間、複數、插值或 React 中任何與在地化相關的任務時，請使用此技能。當使用者提到「i18n」、「i18next」、「react-i18next」、「在地化」、「翻譯」、「多語言」、「國際化」、「區域設定」、「語言切換器」或想要在 React 元件中翻譯字串/JSX 時也會觸發 - 即使它們沒有明確命名庫。也觸發 Next.js/Remix/Gatsby i18n 問題，因為react-i18next 是底層解決方案。
