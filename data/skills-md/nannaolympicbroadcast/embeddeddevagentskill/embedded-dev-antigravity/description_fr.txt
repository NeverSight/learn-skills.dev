Compétence d'agent de développement intégré, conçue exclusivement pour l'IDE Google Antigravity. Doit être déclenché chaque fois que l'utilisateur mentionne le développement intégré, les microcontrôleurs, les cartes de développement, les MCU, RTOS, le flashage du micrologiciel, le débogage série, SSH vers les appareils, ESP32, STM32, Raspberry Pi, Arduino, Zephyr, FreeRTOS, U-Boot, le noyau Linux, les arborescences de périphériques, la compilation croisée ou des sujets connexes. S'applique également à : la lecture des journaux de l'appareil, le flashage du micrologiciel, le débogage des problèmes matériels, la configuration des chaînes d'outils, le portage du code vers les cartes cibles, etc. Même si l'utilisateur demande simplement "comment connecter ma carte en série ?" ou "échec de la compilation", tant qu'il s'agit de matériel embarqué, cette compétence doit être déclenchée.
