مهارة وكيل التطوير المضمنة، المصممة خصيصًا لـ Google Antigravity IDE. يجب أن يتم تشغيله على الفور عندما يذكر المستخدم التطوير المضمن، أو وحدات التحكم الدقيقة، أو لوحات التطوير، أو MCU، أو RTOS، أو وميض البرامج الثابتة، أو التصحيح التسلسلي، أو اتصالات SSH بالأجهزة، أو ESP32، أو STM32، أو Raspberry Pi، أو Arduino، أو Zephyr، أو FreeRTOS، أو U-Boot، أو Linux kernel، أو أشجار الأجهزة، أو التجميع المتقاطع، أو المواضيع ذات الصلة. ينطبق أيضًا على: قراءة سجلات الجهاز، والبرامج الثابتة الوامضة، وتصحيح مشكلات الأجهزة، وتكوين سلاسل الأدوات، ونقل التعليمات البرمجية إلى اللوحات المستهدفة، والسيناريوهات المماثلة. حتى لو سأل المستخدم ببساطة "كيف يمكنني الاتصال بالمنفذ التسلسلي الموجود على اللوحة الخاصة بي" أو "فشل التجميع"، طالما أن الأمر يتضمن أجهزة مضمنة، فيجب تشغيل هذه المهارة.
