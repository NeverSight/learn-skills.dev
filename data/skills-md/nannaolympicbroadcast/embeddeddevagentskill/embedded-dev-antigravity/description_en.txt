Embedded Development Agent Skill, designed exclusively for Google Antigravity IDE. Must be triggered whenever the user mentions embedded development, microcontrollers, dev boards, MCUs, RTOS, firmware flashing, serial debugging, SSH to devices, ESP32, STM32, Raspberry Pi, Arduino, Zephyr, FreeRTOS, U-Boot, Linux kernel, device trees, cross-compilation, or related topics. Also applies to: reading device logs, flashing firmware, debugging hardware issues, configuring toolchains, porting code to target boards, etc. Even if the user simply asks "how do I connect my board to serial?" or "compilation failed," as long as it involves embedded hardware, this Skill must be triggered.
