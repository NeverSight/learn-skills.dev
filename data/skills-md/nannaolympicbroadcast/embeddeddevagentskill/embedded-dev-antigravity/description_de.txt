Eingebetteter Development Agent Skill, exklusiv für Google Antigravity IDE entwickelt. Muss immer dann ausgelöst werden, wenn der Benutzer eingebettete Entwicklung, Mikrocontroller, Entwicklungsplatinen, MCUs, RTOS, Firmware-Flashing, serielles Debugging, SSH zu Geräten, ESP32, STM32, Raspberry Pi, Arduino, Zephyr, FreeRTOS, U-Boot, Linux-Kernel, Gerätebäume, Cross-Compilation oder verwandte Themen erwähnt. Gilt auch für: Lesen von Geräteprotokollen, Flashen von Firmware, Debuggen von Hardwareproblemen, Konfigurieren von Toolchains, Portieren von Code auf Zielplatinen usw. Auch wenn der Benutzer einfach fragt: „Wie verbinde ich meine Platine seriell?“ oder „Kompilierung fehlgeschlagen“, solange es sich um eingebettete Hardware handelt, muss dieser Skill ausgelöst werden.
