يصمم ويبني ويتحقق ويطور أنظمة مهارات مستقلة عن الذكاء الاصطناعي تتألف من مهارات (مع قدرات اختيارية) وأدوار. يتم التنشيط عندما يريد المستخدم: إنشاء مهارة أو قدرة جديدة، أو تحديد دور أو تحسينه، أو إعادة هيكلة التسلسل الهرمي للمهارات/الأدوار، أو الانتقال من المهارات المسطحة إلى نمط جهاز التوجيه + القدرات، أو اتساق نظام مهارات التدقيق، أو سبب كفاءة الرمز المميز ومقايضات قابلية الصيانة. يتم تشغيله أيضًا عندما يذكر المستخدم "نظام المهارات"، أو "المهارة الأساسية"، أو "المنسق"، أو يناقش تنظيم التشغيل الآلي اللاحيادي للذكاء الاصطناعي عبر أدوات متعددة مثل Claude Code، أو Cursor، أو Codex.
