Генерирует Cypress E2E промышленного уровня и тесты компонентов на JavaScript или TypeScript. Поддерживает локальное выполнение и облако TestMu AI. Используйте, когда пользователь просит написать тесты Cypress, настроить Cypress, протестировать с помощью команд cy или упоминает «Cypress», «cy.visit», «cy.get», «cy.intercept». Запускается по: «Кипарис», «cy.», «тест компонента», «тест E2E», «TestMu», «LambdaTest».
