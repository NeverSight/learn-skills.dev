Генерирует тесты pytest промышленного уровня на Python с шаблонами фиксации, параметризации, маркеров, макетов и конфликтов. Используйте, когда пользователь упоминает «pytest», «conftest», «@pytest.fixture», «@pytest.mark», «Python test». Запускается по: «pytest», «conftest», «тест Python», «параметризация», «модульный тест Python».
