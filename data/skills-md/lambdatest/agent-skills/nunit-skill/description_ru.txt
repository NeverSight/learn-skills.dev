Создает тесты NUnit 3 на C#. Охватывает модель ограничений Assert.That, параметризованные тесты, настройку/демонтаж и макетирование Moq. Используйте, когда пользователь упоминает «NUnit», «[TestFixture]», «[Test]», «Assert.That», «Юнит-тест C#». Запускается по: «NUnit», «[Test]», «Assert.That», «C# test», «NUnit3».
