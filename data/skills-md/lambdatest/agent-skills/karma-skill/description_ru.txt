Создает конфигурации средства запуска тестов Karma для тестирования JavaScript на основе браузера. Работает с Jasmine, Mocha или QUnit. Используйте, когда пользователь упоминает «Карма», «karma.conf.js», «запуск тестов браузера». Запускается по: «Карма», «karma.conf», «запуск теста кармы», «тест JS на основе браузера».
