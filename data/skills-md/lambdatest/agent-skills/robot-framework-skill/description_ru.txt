Генерирует тесты Robot Framework в синтаксисе, управляемом ключевыми словами, с помощью Python. Поддерживает SeleniumLibrary, RequestsLibrary и пользовательские ключевые слова. Используйте, когда пользователь упоминает «Robot Framework», «*** Тестовые примеры ***», «SeleniumLibrary», «файл .robot». Запускается по: «Robot Framework», «*** Тестовые примеры ***», «.robot», «SeleniumLibrary», «тест на основе ключевых слов».
