Генерирует тесты TestNG на Java с группами, поставщиками данных, параллельным выполнением, конфигурацией набора XML и прослушивателями. Используйте, когда пользователь упоминает «TestNG», «@DataProvider», «testng.xml», «groups». Запускается по: «TestNG», «@DataProvider», «testng.xml», «Набор TestNG», «параллельные тесты Java».
