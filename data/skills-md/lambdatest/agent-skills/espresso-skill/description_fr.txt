Génère des tests d'interface utilisateur Espresso pour les applications Android en Kotlin ou Java. Espresso s'exécute dans le processus de l'application pour des tests d'interface utilisateur rapides et fiables. Prend en charge les appareils réels locaux et TestMu AI cloud. À utiliser lorsque l'utilisateur mentionne "Espresso", "onView", "ViewMatchers", "Android UI test" ou "instrumentation test". Se déclenche sur : "Espresso", "onView", "ViewMatchers", "Android UI test", "instrumentation", "TestMu".
