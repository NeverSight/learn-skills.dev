Kotlin または Java で Android アプリの Espresso UI テストを生成します。 Espresso はアプリ プロセス内で実行され、高速で信頼性の高い UI テストを実現します。ローカルおよび TestMu AI クラウドの実デバイスをサポートします。ユーザーが「Espresso」、「onView」、「ViewMatchers」、「Android UI テスト」、または「インストルメンテーション テスト」について言及する場合に使用します。トリガー対象: 「Espresso」、「onView」、「ViewMatchers」、「Android UI テスト」、「instrumentation」、「TestMu」。
