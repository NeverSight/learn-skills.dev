Migre et convertit les scripts d'automatisation de test entre Selenium, Playwright, Puppeteer et Cypress. À utiliser lorsque l'utilisateur demande à migrer, convertir ou porter des tests d'un framework à un autre ; réécrire les tests dans un framework différent ; ou passez de Selenium à Playwright, de Playwright à Selenium, de Marionnettiste à Playwright, de Cypress à Playwright, ou vice versa.
