在 Selenium、Playwright、Puppeteer 和 Cypress 之間遷移和轉換測試自動化腳本。當使用者要求將測試從一個框架遷移、轉換或移植到另一個框架時使用；在不同的框架中重寫測試；或從 Selenium 切換到 Playwright，從 Playwright 切換到 Selenium，從 Puppeteer 切換到 Playwright，從 Cypress 切換到 Playwright，反之亦然。觸發：「遷移」、「轉換」、「移植」、「selenium 到劇作家」、「劇作家到 selenium」、「傀儡師到劇作家」、「cypress 到劇作家」、「重寫測試」、「從 [框架] 切換到 [框架]」。
