Migriert und konvertiert Testautomatisierungsskripte zwischen Selenium, Playwright, Puppeteer und Cypress. Verwenden Sie diese Option, wenn der Benutzer Tests von einem Framework auf ein anderes migrieren, konvertieren oder portieren möchte. Tests in einem anderen Framework neu schreiben; oder wechseln Sie von Selenium zu Dramatiker, von Dramatiker zu Selenium, von Puppenspieler zu Dramatiker, von Cypress zu Dramatiker oder umgekehrt. Auslöser bei: „migrieren“, „konvertieren“, „portieren“, „Selenium zu Dramatiker“, „Dramatiker zu Selenium“, „Puppenspieler zu Dramatiker“, „Zypresse zu Dramatiker“, „Tests neu schreiben in“, „Wechsel von [Framework] zu [Framework]“.
