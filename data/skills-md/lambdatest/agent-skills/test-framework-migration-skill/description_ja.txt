Selenium、Playwright、Puppeteer、Cypress の間でテスト自動化スクリプトを移行および変換します。ユーザーがテストをあるフレームワークから別のフレームワークに移行、変換、または移植するように要求する場合に使用します。別のフレームワークでテストを書き直す。または、Selenium から Playwright、Playwright から Selenium、Puppeteer から Playwright、Cypress から Playwright、またはその逆に切り替えることができます。トリガー: 「移行」、「変換」、「ポート」、「Selenium から Playwright」、「Playwright から Selenium」、「Puppeteer から Playwright」、「Cypress から Playwright」、「テストの書き換え」、「[フレームワーク] から [フレームワーク] への切り替え」。
