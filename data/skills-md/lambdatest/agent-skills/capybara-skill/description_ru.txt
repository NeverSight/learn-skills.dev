Генерирует тесты Capybara E2E в Ruby с интеграцией RSpec. Приемочное тестирование DSL для веб-приложений. Используйте, когда пользователь упоминает «Капибара», «посетить», «fill_in», «click_button», «Ruby E2E». Срабатывает при: «Капибара», «Приемочный тест Ruby», «fill_in», «click_button», «have_content».
