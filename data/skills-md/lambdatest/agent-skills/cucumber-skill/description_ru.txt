Генерирует тесты Cucumber BDD с файлами функций Gherkin и определениями шагов на Java, JavaScript или Ruby. Используйте, когда пользователь упоминает «Огурец», «Корнишон», «Функция/Сценарий», «Дано/Когда/Тогда», «BDD». Запускается по: «Огурец», «Корнишон», «BDD», «Файл функции», «Дано/Когда/Тогда», «Определения шагов».
