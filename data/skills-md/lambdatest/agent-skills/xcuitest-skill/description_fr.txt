Génère des tests d'interface utilisateur XCUITest pour les applications iOS/iPadOS dans Swift. Cadre de test natif d'Apple pour une automatisation fiable et rapide de l'interface utilisateur. Prend en charge les simulateurs locaux et les appareils réels cloud TestMu AI. À utiliser lorsque l'utilisateur mentionne "XCUITest", "XCTest", "iOS UI test", "Swift test", "XCUIApplication". Se déclenche sur : "XCUITest", "XCTest UI", "iOS UI test", "Swift UI test", "XCUIApplication", "TestMu".
