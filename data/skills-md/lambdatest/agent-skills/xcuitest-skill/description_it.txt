Genera test dell'interfaccia utente XCUITest per le app iOS/iPadOS in Swift. Il framework di test nativo di Apple per un'automazione dell'interfaccia utente affidabile e veloce. Supporta simulatori locali e dispositivi reali cloud TestMu AI. Da utilizzare quando l'utente menziona "XCUITest", "XCTest", "iOS UI test", "Swift test", "XCUIApplication". Si attiva su: "XCUITest", "XCTest UI", "iOS UI test", "Swift UI test", "XCUIApplication", "TestMu".
