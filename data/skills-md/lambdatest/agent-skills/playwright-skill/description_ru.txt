Создает сценарии автоматизации драматурга производственного уровня и тесты E2E на TypeScript, JavaScript, Python, Java или C#. Поддерживает локальное выполнение и облако TestMu AI для более чем 3000 комбинаций браузера/ОС и реальных мобильных устройств. Используйте, когда пользователь просит написать тесты Драматурга, автоматизировать браузеры, запустить кроссбраузерные тесты, протестировать на реальных устройствах, отладить нестабильные тесты, имитировать API или выполнить визуальную регрессию. Запускается по: «Драматург», «Тест E2E», «Тест браузера», «Запуск в облаке», «Кросс-браузер», «TestMu», «LambdaTest», «Проверить мое приложение», «Проверить на мобильном устройстве», «реальном устройстве».
