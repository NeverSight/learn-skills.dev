Тестируйте локальные веб-приложения с помощью Playwright с Python. Проверяйте функциональность внешнего интерфейса, отлаживайте поведение пользовательского интерфейса, делайте снимки экрана и просматривайте журналы консоли браузера. Поддерживает статические HTML-файлы, динамические веб-приложения с работающими серверами и автоматическое создание тестов.
