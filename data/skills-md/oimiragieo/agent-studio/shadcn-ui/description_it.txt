Esperienza approfondita sulla libreria di componenti shadcn/ui, inclusi modelli di installazione, personalizzazione, temi e accessibilità. Da utilizzare quando si crea un'interfaccia utente con componenti shadcn/ui, si integrano primitive dell'interfaccia utente Radix, si configurano temi Tailwind CSS v4 o si implementa la modalità oscura. Copre le configurazioni del router dell'app Next.js, del router di Pages e di Vite.
