خبرة عميقة في مكتبة مكونات shadcn/ui بما في ذلك التثبيت والتخصيص والتخصيص وأنماط إمكانية الوصول. يُستخدم عند إنشاء واجهة مستخدم باستخدام مكونات shadcn/ui، أو دمج أساسيات Radix UI، أو تكوين سمات Tailwind CSS v4، أو تنفيذ الوضع المظلم. يغطي جهاز توجيه تطبيقات Next.js، وجهاز توجيه الصفحات، وإعدادات Vite.
