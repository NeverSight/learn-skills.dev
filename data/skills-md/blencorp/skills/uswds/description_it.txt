Fornisce una guida completa per la creazione con U.S. Web Design System (USWDS v3) e la libreria di componenti React @trussworks/react-uswds. Utilizzare quando: (1) creare o applicare stili all'interfaccia utente con componenti USWDS, classi di utilità o token di progettazione, (2) implementare layout con il sistema a griglia USWDS, (3) lavorare con token di colore, tipografia o spaziatura USWDS, (4) creare moduli, navigazione, intestazioni, piè di pagina, modali, tabelle, schede o qualsiasi componente USWDS, (5) applicare modelli di accessibilità USWDS, (6) personalizzare le impostazioni del tema USWDS tramite variabili Sass, (7) utilizzare importazioni e oggetti di scena @trussworks/react-uswds, (8) creare interfacce web governative/federali. Trigger su: classi USWDS (usa-*), importazioni @trussworks/react-uswds, token/funzioni USWDS Sass, domande sul sistema di progettazione federale.
