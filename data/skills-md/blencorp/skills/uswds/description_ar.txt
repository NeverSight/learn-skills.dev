يوفر دليلاً شاملاً للبناء باستخدام نظام تصميم الويب الأمريكي (USWDS v3) ومكتبة مكونات React@trussworks/react-uswds. يُستخدم عندما: (1) إنشاء واجهة مستخدم أو تصميمها باستخدام مكونات USWDS أو فئات الأدوات المساعدة أو رموز التصميم، (2) تنفيذ التخطيطات باستخدام نظام شبكة USWDS، (3) العمل مع ألوان USWDS أو الطباعة أو رموز التباعد، (4) إنشاء النماذج أو التنقل أو الرؤوس أو التذييلات أو الوسائط أو الجداول أو البطاقات أو أي مكون USWDS، (5) تطبيق أنماط إمكانية الوصول إلى USWDS، (6) تخصيص إعدادات سمة USWDS عبر Sass المتغيرات، (7) استخدام واردات ودعائم @trussworks/react-uswds، (8) بناء واجهات الويب الحكومية/الفيدرالية. المشغلات على: فئات USWDS (usa-*)، وواردات @trussworks/react-uswds، ورموز/وظائف USWDS Sass، وأسئلة نظام التصميم الفيدرالي.
