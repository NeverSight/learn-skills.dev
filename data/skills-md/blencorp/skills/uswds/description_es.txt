Proporciona una guía completa para crear con el sistema de diseño web de EE. UU. (USWDS v3) y la biblioteca de componentes @trussworks/react-uswds React. Úselo cuando: (1) cree o diseñe una interfaz de usuario con componentes, clases de utilidad o tokens de diseño de USWDS, (2) implemente diseños con el sistema de cuadrícula de USWDS, (3) trabaje con tokens de color, tipografía o espaciado de USWDS, (4) cree formularios, navegación, encabezados, pies de página, modales, tablas, tarjetas o cualquier componente de USWDS, (5) aplique patrones de accesibilidad de USWDS, (6) personalice la configuración del tema de USWDS a través de variables Sass, (7) usando importaciones y accesorios @trussworks/react-uswds, (8) construyendo interfaces web gubernamentales/federales. Activadores de: clases USWDS (usa-*), importaciones @trussworks/react-uswds, tokens/funciones USWDS Sass, preguntas sobre el sistema de diseño federal.
