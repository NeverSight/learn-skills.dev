U.S. Web Design System (USWDS v3) および @trussworks/react-uswds React コンポーネント ライブラリを使用して構築するための包括的なガイドを提供します。次の場合に使用します: (1) USWDS コンポーネント、ユーティリティ クラス、またはデザイン トークンを使用して UI を構築またはスタイル設定する、(2) USWDS グリッド システムを使用してレイアウトを実装する、(3) USWDS カラー、タイポグラフィ、またはスペース トークンを使用する、(4) フォーム、ナビゲーション、ヘッダー、フッター、モーダル、テーブル、カード、または任意の USWDS コンポーネントを作成する、(5) USWDS アクセシビリティ パターンを適用する、(6) Sass を介して USWDS テーマ設定をカスタマイズする変数、(7) @trussworks/react-uswds インポートとプロパティの使用、(8) 政府/連邦 Web インターフェイスの構築。トリガー: USWDS クラス (usa-*)、@trussworks/react-uswds インポート、USWDS Sass トークン/関数、連邦設計システムの質問。
