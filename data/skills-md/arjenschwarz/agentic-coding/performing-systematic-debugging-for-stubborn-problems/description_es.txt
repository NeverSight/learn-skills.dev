Aplica una metodología de inspección Fagan modificada para resolver sistemáticamente errores persistentes y problemas complejos. Úselo cuando varios intentos de reparación anteriores hayan fallado repetidamente, cuando se trate de interacciones complejas del sistema o cuando se necesite un análisis metódico de la causa raíz. No lo utilice para solucionar problemas simples. Se activa después de varios intentos fallidos de depuración del mismo problema complejo.
