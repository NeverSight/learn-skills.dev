Générez la configuration des autorisations Claude Code à partir de l'historique de la session. À utiliser lors de la configuration du mode autonome, de la configuration de .claude/settings.json, de l'évitement de --dangerously-skip-permissions ou de l'analyse des autorisations dont un projet a besoin. Lit les journaux de session pour extraire les commandes Bash et les outils MCP réellement utilisés, puis génère les règles d'autorisation/refus appropriées.
