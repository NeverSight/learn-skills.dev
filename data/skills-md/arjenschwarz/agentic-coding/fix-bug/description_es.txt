Investigación, resolución y documentación sistemática de errores. Úselo para corregir errores que necesiten un análisis exhaustivo, cobertura de pruebas y un informe formal de corrección de errores. Aplica una metodología de depuración sistemática, crea pruebas de regresión y genera un informe estandarizado en specs/bugfixes/<bug-name>/. Para errores complejos, genera agentes de implementación competitivos (incluidos arneses alternativos como Kiro) y selecciona la mejor solución. Se activa en solicitudes como "corregir este error", "depurar y documentar este problema" o cuando un error necesita resolución y documentación.
