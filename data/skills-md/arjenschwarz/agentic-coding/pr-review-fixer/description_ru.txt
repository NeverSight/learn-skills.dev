Получайте неразрешенные PR-комментарии (как на уровне кода, так и на уровне PR), проверяйте проблемы и устраняйте их. Также проверяет статус CI и исправляет неудачные тесты, ошибки lint и проблемы сборки. Используйте при просмотре и рассмотрении отзывов GitHub по связям с общественностью. Отфильтровывает решенные комментарии, сохраняет только последний комментарий claude[bot] для каждой ветки, проверяет проблемы, публикует отчет о проверке как комментарий по связям с общественностью, а затем исправляет проверенные проблемы.
