Шаблоны качества кода и рекомендации по разработке инструментов ToolUniverse. Применяется при написании, исправлении или рефакторинге кода инструмента Python в проекте ToolUniverse. Кодирует уроки из более чем 80 раундов отладки. Используйте вместе с devtu-fix-tool и devtu-self-evolve. Триггеры: внедрение исправлений инструментов, написание новых классов инструментов, проверка качества кода инструментов, проверка правильности схемы, поиск исправлений ошибок, специфичных для API.
