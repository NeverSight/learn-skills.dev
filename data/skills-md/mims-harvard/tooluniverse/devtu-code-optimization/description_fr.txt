Modèles de qualité du code et directives pour le développement d’outils ToolUniverse. Appliquer lors de l'écriture, de la correction ou de la refactorisation du code Python de l'outil dans le projet ToolUniverse. Encode les leçons de plus de 80 cycles de débogage. À utiliser avec devtu-fix-tool et devtu-self-evolve. Déclencheurs : implémentation de correctifs d'outils, écriture de nouvelles classes d'outils, révision de la qualité du code de l'outil, vérification de l'exactitude du schéma, recherche de corrections de bogues spécifiques à l'API.
