Этот навык позволяет взаимодействовать с REST API BitBucket Server для управления запросами на извлечение. Используйте, когда пользователь хочет создавать, просматривать, комментировать, объединять, составлять списки запросов на включение, читать комментарии или управлять задачами на BitBucket Server.
