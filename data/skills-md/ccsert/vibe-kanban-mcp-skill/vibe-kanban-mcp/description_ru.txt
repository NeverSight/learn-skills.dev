Используйте этот навык всякий раз, когда пользователь хочет работать с Vibe Kanban через интерфейс командной строки MCPorter. Охватывает любой рабочий процесс, который создает, обновляет, перечисляет, назначает, помечает, связывает или управляет задачами, проектами, репозиториями, рабочими пространствами или сеансами Vibe Kanban с помощью
 запуск команд `mcporter call vibe_kanban.<tool>` в терминале. Этот навык
— это справочное руководство и руководство по рабочему процессу для разрешения идентификаторов организации/проекта/рабочей области, настройки MCPorter, управления сценариями репозитория, а также запуска или связывания сеансов рабочей области. Срабатывает, даже когда пользователь говорит только такие слова, как «vibe kanban», «kanban», «mcporter», «mcporter call vibe_kanban», «создать задачу», «назначить эту задачу», «начать сеанс рабочей области», «связать рабочую область» или спрашивает, как программно подключить агента, Claude Code, Codex, Cursor, OpenCode, Gemini, Copilot, Amp, Droid или Qwen к Vibe Kanban.
