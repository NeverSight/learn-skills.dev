ユーザーが MCPorter CLI 経由で Vibe Kanban を操作する場合は常に、このスキルを使用します。 Vibe Kanban の問題、プロジェクト、リポジトリ、ワークスペース、またはワークスペース セッションを作成、更新、リスト、割り当て、タグ付け、リンク、または管理するあらゆるワークフローをカバーします。
 ターミナルで「mcporter call vibe_kanban.<tool>」コマンドを実行します。このスキル
は、組織/プロジェクト/ワークスペース ID の解決、MCPorter の構成、リポジトリ スクリプトの管理、およびワークスペース セッションの開始またはリンクに関するリファレンスおよびワークフロー ガイドです。ユーザーが「vibe kanban」、「kanban」、「mcporter」、「mcporter call vibe_kanban」、「問題の作成」、「この問題の割り当て」、「ワークスペース セッションの開始」、「ワークスペースのリンク」などのことを言っただけの場合、またはエージェント、Claude Code、Codex、Cursor、OpenCode、Gemini、Copilot、Amp、Droid、または Qwen をプログラムで Vibe Kanban に接続する方法を尋ねた場合でもトリガーされます。
