Utilisez cette compétence chaque fois que l'utilisateur souhaite travailler avec Vibe Kanban via MCPorter CLI. Couvre tout flux de travail qui crée, met à jour, répertorie, attribue, marque, lie ou gère les problèmes, projets, référentiels, espaces de travail ou sessions d'espace de travail Vibe Kanban en
 exécuter les commandes `mcporter call vibe_kanban.<tool>` dans le terminal. Cette compétence
est le guide de référence et de workflow pour résoudre les ID d'organisation/projet/espace de travail, configurer MCPorter, gérer les scripts de dépôt et démarrer ou lier des sessions d'espace de travail. Déclenchez même lorsque l'utilisateur dit uniquement des choses comme « vibe kanban », « kanban », « mcporter », « mcporter call vibe_kanban », « create issue », « assign this issue », « start workspace session », « link workspace » ou demande comment câbler un agent, Claude Code, Codex, Cursor, OpenCode, Gemini, Copilot, Amp, Droid ou Qwen dans Vibe Kanban par programme.
