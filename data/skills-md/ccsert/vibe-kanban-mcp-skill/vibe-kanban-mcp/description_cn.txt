每当用户想要通过 MCPorter CLI 使用 Vibe Kanban 时，请使用此技能。涵盖通过以下方式创建、更新、列出、分配、标记、链接或管理 Vibe 看板问题、项目、存储库、工作区或工作区会话的任何工作流程
 在终端中运行 `mcporter call vivi_kanban.<tool>` 命令。这个技能
是用于解析组织/项目/工作空间 ID、配置 MCPorter、管理存储库脚本以及启动或链接工作空间会话的参考和工作流程指南。即使用户仅说出“vibe kanban”、“kanban”、“mcporter”、“mcporter call vibe_kanban”、“创建问题”、“分配此问题”、“启动工作区会话”、“链接工作区”等内容，或询问如何以编程方式将代理、Claude Code、Codex、Cursor、OpenCode、Gemini、Copilot、Amp、Droid 或 Qwen 连接到 Vibe 看板，也会触发此操作。
