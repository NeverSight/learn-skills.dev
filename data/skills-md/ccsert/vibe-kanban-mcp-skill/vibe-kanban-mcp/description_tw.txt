每當用戶想要透過 MCPorter CLI 使用 Vibe Kanban 時，請使用此技能。涵蓋透過以下方式建立、更新、列出、指派、標記、連結或管理 Vibe 看板問題、專案、儲存庫、工作區或工作區會話的任何工作流程
 在終端機中執行 `mcporter call vivi_kanban.<tool>` 指令。這個技能
是用於解析組織/專案/工作空間 ID、配置 MCPorter、管理儲存庫腳本以及啟動或連結工作空間會話的參考和工作流程指南。即使用戶只說出「vibe kanban」、「kanban」、「mcporter」、「mcporter call vibe_kanban」、「創建問題」、「分配此問題」、「啟動工作區會話」、「連結工作區」等內容，或詢問如何以程式設計方式將代理程式、Claude Code、Codex、Cursor、OpenCode、CZ、V、Vobe、firbe看板，也會觸發此操作。
