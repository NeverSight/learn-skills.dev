Заказать торговый инструмент. Строгие правила: (1) Все ордера должны выполняться через скрипты узла/exchange.mjs create_order; индивидуальный код для размещения заказов запрещен. (2) create_order — это двухэтапный процесс: первый вызов возвращает предварительный просмотр, показанный пользователю для подтверждения; только после того, как пользователь скажет «подтвердить», будет выполнен второй вызов с подтверждением = true. (3) Автоподтверждение запрещено; пропуск предварительного просмотра запрещен. (4) Чтобы закрыть позицию, используйте close_position; создание взаимозачетного ордера с помощью create_order запрещено. Триггеры: «купить», «продать», «ордер», «длинная позиция», «короткая позиция», «кредитное плечо», «закрыть позицию», «买», «卖», «下单», «做多», «做空», «杠杆», «开仓», «平仓», «平掉», '关仓'.
