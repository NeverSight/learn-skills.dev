Herramienta de negociación de pedidos. Reglas estrictas: (1) Todos los pedidos deben realizarse mediante node scripts/exchange.mjs create_order; Está prohibido escribir código personalizado para realizar pedidos. (2) create_order es un proceso de dos pasos: la primera llamada devuelve una vista previa, que se muestra al usuario para su confirmación; Después de que el usuario dice "confirmar", la segunda llamada se ejecuta con confirmado = verdadero. (3) Están prohibidos la confirmación automática y saltarse la vista previa. (4) Para cerrar una posición, debe utilizar close_position; No se permite crear una orden de compensación con create_order.
