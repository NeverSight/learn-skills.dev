Order execution tool. Strict rules: (1) All orders must be placed via node scripts/exchange.mjs create_order; custom order code is forbidden. (2) create_order is two-step: first returns a preview to show the user; after the user says “confirm”, run again with confirmed=true. (3) Auto-confirmation is forbidden; skipping the preview is forbidden. (4) To close a position you must use close_position; creating an offsetting order with create_order is forbidden.
