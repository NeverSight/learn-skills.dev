À utiliser lorsque l'utilisateur pose des questions sur l'écriture de stratégies de trading, les backtests, le déploiement de robots Freqtrade, le trading quantitatif ou l'optimisation de stratégie. Mots déclencheurs : « écrire une stratégie », « créer une stratégie », « backtest », « déployer Freqtrade », « déployer un bot », « quantitatif », « hyperopt », « 写策略 », « 创建策略 », « 回测 », « 部署 », « 量化 », '策略优化'. Cette compétence fournit : (1) un générateur rapide create_strategy avec 17 indicateurs, (2) le SDK AiCoin Python (aicoin_data.py) pour intégrer des données de marché réelles dans des stratégies personnalisées, (3) des outils de déploiement/backtest/hyperopt. Utilisez TOUJOURS activement les données AiCoin (taux de financement, ratio L/S, commandes de baleines, etc.) dans les stratégies lorsque la clé API de l'utilisateur le prend en charge. Pour les prix/graphiques, utilisez Aicoin-Market. Pour le trading, utilisez Aicoin-Trading. Pour Hyperliquide, utilisez aicoin-hyperliquide.
