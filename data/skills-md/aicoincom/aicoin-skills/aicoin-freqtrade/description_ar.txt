يُستخدم عندما يسأل المستخدم عن كتابة إستراتيجيات التداول، أو الاختبار الخلفي، أو نشر روبوتات Freqtrade، أو التداول الكمي، أو تحسين الإستراتيجية. كلمات التشغيل: "كتابة الإستراتيجية"، "إنشاء إستراتيجية"، "backtest"، "نشر Freqtrade"، "نشر الروبوت"، "كمي"، "hyperopt"، "写策略"، "创建策略"، "回测"، "部署"، "量化"، "الإجابة على هذا السؤال". توفر هذه المهارة: (1) منشئ سريع لـ create_strategy مع 17 مؤشرًا، (2) AiCoin Python SDK (aicoin_data.py) لدمج بيانات السوق الحقيقية في استراتيجيات مخصصة، (3) أدوات النشر/الاختبار الخلفي/hyperopt. استخدم دائمًا بيانات AiCoin (معدل التمويل، ونسبة L/S، وطلبات الحوت، وما إلى ذلك) في الاستراتيجيات عندما يدعمها مفتاح API الخاص بالمستخدم. للأسعار/الرسوم البيانية استخدم aicoin-market. للتداول استخدم aicoin-trading. بالنسبة إلى Hyperliquid، استخدم aicoin-hyperliquid.
