Используйте, когда пользователь спрашивает о написании торговых стратегий, бэктестировании, развертывании ботов Freqtrade, количественной торговле или оптимизации стратегии. Триггерные слова: «написать стратегию», «создать стратегию», «тестирование на истории», «развертывание Freqtrade», «развертывание бота», «количественный», «гиперопт», «写策略», «创建策略», «回测», «部署», «量化», '策略优化'. Этот навык предоставляет: (1) быстрый генератор create_strategy с 17 индикаторами, (2) AiCoin Python SDK (aicoin_data.py) для интеграции реальных рыночных данных в пользовательские стратегии, (3) инструменты развертывания/бэктестирования/гиперопции. ВСЕГДА активно используйте данные AiCoin (ставка финансирования, соотношение L/S, китовые ордера и т. д.) в стратегиях, если API-ключ пользователя поддерживает это. Для цен/графиков используйте aicoin-market. Для торговли используйте aicoin-trading. Для Hyperliquid используйте aicoin-hyperliquid.
