ユーザーが取引戦略の作成、バックテスト、Freqtrade ボットの展開、定量的取引、または戦略の最適化について質問するときに使用します。トリガーワード: 「戦略を書く」、「戦略を作成する」、「バックテスト」、「Freqtrade を展開する」、「ボットを展開する」、「定量的」、「hyperopt」、「写策略」、「创建策略」、「回测」、「配置」、「量化」、「策略优化」。このスキルは、(1) 17 のインジケーターを備えた create_strategy クイック ジェネレーター、(2) 実際の市場データをカスタム戦略に統合するための AiCoin Python SDK (aicoin_data.py)、(3) デプロイ/バックテスト/ハイパーオプト ツールを提供します。ユーザーの API キーがサポートしている場合は、AiCoin データ (資金調達率、L/S 比率、クジラの注文など) を戦略で常に積極的に使用してください。価格/チャートについては、aicoin-market をご利用ください。取引にはaicoin-tradingを使用してください。ハイパーリキッドの場合は、aicoin-hyperliquid を使用します。
