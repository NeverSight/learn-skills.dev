قم بمراجعة مهارات وكيل الذكاء الاصطناعي للمخاطر الأمنية قبل التثبيت أو بشكل دوري. يعمل على Claude Code وOpenClaw وجميع المنصات. اكتشف الحقن الفوري، واستخلاص البيانات، والأوامر الضارة، والتعليمات البرمجية المبهمة، وإساءة استخدام الامتيازات، ومخاطر سلسلة التوريد، وتسمم الذاكرة، واستغلال الثقة، والتلاعب السلوكي. استخدمها قبل تثبيت مهارات الطرف الثالث من أي سوق.
