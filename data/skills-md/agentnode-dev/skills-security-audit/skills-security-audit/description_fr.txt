Auditez les compétences de l’agent IA pour détecter les risques de sécurité avant l’installation ou périodiquement. Fonctionne sur Claude Code, OpenClaw et toutes les plateformes. Détectez l’injection rapide, l’exfiltration de données, les commandes malveillantes, le code obscurci, l’abus de privilèges, les risques liés à la chaîne d’approvisionnement, l’empoisonnement de la mémoire, l’exploitation de la confiance et la manipulation comportementale. À utiliser avant d'installer des compétences tierces à partir de n'importe quel marché.
