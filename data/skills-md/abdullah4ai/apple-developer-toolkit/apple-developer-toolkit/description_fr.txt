Compétence de développeur Apple tout-en-un avec trois outils intégrés livrés sous la forme d'un seul binaire unifié. (1) Recherche de documentation parmi les frameworks Apple, les symboles et 1 267 sessions WWDC de 2014 à 2025. Aucune information d'identification requise. (2) App Store Connect CLI avec plus de 120 commandes couvrant les builds (rechercher/attendre/télécharger), TestFlight, validation de pré-soumission, soumissions, signature, abonnements (partageables en famille), IAP, analyses, Xcode Cloud, workflows de métadonnées, tableau de bord du pipeline de versions, informations, offres de reconquête, achats promus, pages de produits, nominations, déclarations d'accessibilité, précommandes, filtres de tarification, mise à jour des localisations, différences, webhooks avec récepteur local, workflow l'automatisation, et plus encore. Nécessite la clé API App Store Connect. (3) Générateur d'applications multiplateformes (iOS/watchOS/tvOS/iPad/macOS/visionOS) qui génère des applications Swift/SwiftUI complètes à partir du langage naturel avec correction automatique, lancement du simulateur, mode de discussion interactif et ouverture dans Xcode. Nécessite une clé API LLM et Xcode. Comprend 38 règles de développement iOS et 12 guides de bonnes pratiques SwiftUI pour Liquid Glass, la navigation, la gestion des états et les API modernes. Les trois outils sont livrés sous forme d'un seul binaire (appledev). À UTILISER QUAND : documents API Apple, gestion App Store Connect, recherche WWDC ou création d'applications iOS/watchOS/tvOS/macOS/visionOS à partir de zéro. NE PAS UTILISER QUAND : plates-formes non Apple ou codage général.
