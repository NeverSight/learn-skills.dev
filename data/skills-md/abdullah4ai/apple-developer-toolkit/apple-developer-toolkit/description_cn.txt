一体化 Apple 开发人员技能，将三个集成工具作为一个统一的二进制文件提供。 (1) 跨 Apple 框架、符号和 2014 年至 2025 年 1,267 个 WWDC 会议的文档搜索。无需任何凭证。 (2) App Store Connect CLI 包含 120 多个命令，涵盖构建（查找/等待/上传）、TestFlight、预提交验证、提交、签名、订阅（家庭共享）、IAP、分析、Xcode Cloud、元数据工作流程、发布管道仪表板、见解、赢回优惠、促销购买、产品页面、提名、可访问性声明、预订单、定价过滤器、本地化更新、差异、具有本地接收器的 webhooks、工作流程自动化等。需要 App Store Connect API 密钥。 (3) 多平台应用程序构建器 (iOS/watchOS/tvOS/iPad/macOS/visionOS)，可从自然语言生成完整的 Swift/SwiftUI 应用程序，并具有自动修复、模拟器启动、交互式聊天模式和在 Xcode 中打开的功能。需要 LLM API 密钥和 Xcode。包括 38 条 iOS 开发规则和 12 条针对 Liquid Glass、导航、状态管理和现代 API 的 SwiftUI 最佳实践指南。所有这三个工具都作为一个二进制文件 (appledev) 提供。使用时间：Apple API 文档、App Store Connect 管理、WWDC 查找或从头开始构建 iOS/watchOS/tvOS/macOS/visionOS 应用程序。不要在以下情况下使用：非 Apple 平台或一般编码。
