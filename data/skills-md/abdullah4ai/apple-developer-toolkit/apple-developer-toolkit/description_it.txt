Abilità di sviluppatore Apple tutto in uno con tre strumenti integrati forniti come un unico binario unificato. (1) Ricerca di documentazione tra framework, simboli Apple e 1.267 sessioni WWDC dal 2014 al 2025. Non sono necessarie credenziali. (2) CLI di App Store Connect con oltre 120 comandi che coprono build (ricerca/attesa/caricamento), TestFlight, convalida pre-invio, invii, firma, abbonamenti (condivisibili con tutta la famiglia), IAP, analisi, Xcode Cloud, flussi di lavoro di metadati, dashboard della pipeline di rilascio, approfondimenti, offerte di recupero, acquisti promossi, pagine di prodotto, nomination, dichiarazioni di accessibilità, preordini, filtri di prezzo, aggiornamento delle localizzazioni, differenze, webhook con ricevitore locale, automazione del flusso di lavoro e altro ancora. Richiede la chiave API App Store Connect. (3) Generatore di app multipiattaforma (iOS/watchOS/tvOS/iPad/macOS/visionOS) che genera app Swift/SwiftUI complete dal linguaggio naturale con correzione automatica, avvio del simulatore, modalità chat interattiva e apertura in Xcode. Richiede una chiave API LLM e Xcode. Include 38 regole di sviluppo iOS e 12 guide alle best practice SwiftUI per Liquid Glass, navigazione, gestione dello stato e API moderne. Tutti e tre gli strumenti vengono forniti come un unico binario (appledev). DA USARE QUANDO: documenti API Apple, gestione App Store Connect, ricerca WWDC o creazione di app iOS/watchOS/tvOS/macOS/visionOS da zero. NON USARE QUANDO: piattaforme non Apple o codifica generale.
