Habilidad de desarrollador de Apple todo en uno con tres herramientas integradas enviadas como un único binario unificado. (1) Búsqueda de documentación en los marcos, símbolos y 1267 sesiones de la WWDC de Apple entre 2014 y 2025. No se necesitan credenciales. (2) App Store Connect CLI con más de 120 comandos que cubren compilaciones (buscar/esperar/cargar), TestFlight, validación previa al envío, envíos, firmas, suscripciones (compartibles en familia), IAP, análisis, Xcode Cloud, flujos de trabajo de metadatos, panel de canal de lanzamiento, información, ofertas de recuperación, compras promocionadas, páginas de productos, nominaciones, declaraciones de accesibilidad, pedidos anticipados, filtros de precios, actualización de localizaciones, diferencias, webhooks con receptor local, flujo de trabajo. automatización y más. Requiere la clave API de App Store Connect. (3) Creador de aplicaciones multiplataforma (iOS/watchOS/tvOS/iPad/macOS/visionOS) que genera aplicaciones Swift/SwiftUI completas a partir de lenguaje natural con corrección automática, inicio de simulador, modo de chat interactivo y apertura en Xcode. Requiere una clave API LLM y Xcode. Incluye 38 reglas de desarrollo de iOS y 12 guías de mejores prácticas de SwiftUI para Liquid Glass, navegación, administración de estado y API modernas. Las tres herramientas se envían como un binario (appledev). UTILIZAR CUANDO: documentos API de Apple, administración de App Store Connect, búsqueda en WWDC o creación de aplicaciones iOS/watchOS/tvOS/macOS/visionOS desde cero. NO UTILIZAR CUANDO: plataformas que no sean Apple o codificación general.
