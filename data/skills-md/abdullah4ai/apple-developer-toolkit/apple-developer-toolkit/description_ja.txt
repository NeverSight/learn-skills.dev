単一の統合バイナリとして出荷される 3 つの統合ツールを備えたオールインワンの Apple 開発者スキル。 (1) 2014 年から 2025 年までの Apple フレームワーク、シンボル、および 1,267 の WWDC セッションにわたるドキュメント検索。資格情報は必要ありません。 (2) App Store Connect CLI には、ビルド (検索/待機/アップロード)、TestFlight、提出前検証、提出、署名、サブスクリプション (家族で共有可能)、IAP、分析、Xcode Cloud、メタデータ ワークフロー、リリース パイプライン ダッシュボード、洞察、特典、プロモーション購入、製品ページ、指名、アクセシビリティ宣言、事前注文、価格設定フィルター、ローカリゼーション更新、差分、ローカル レシーバーを備えた Webhook、ワークフローの自動化など。 App Store Connect API キーが必要です。 (3) マルチプラットフォーム アプリ ビルダー (iOS/watchOS/tvOS/iPad/macOS/visionOS) は、自動修正、シミュレーター起動、インタラクティブ チャット モード、Xcode で開く機能を備えた自然言語から完全な Swift/SwiftUI アプリを生成します。 LLM API キーと Xcode が必要です。液体ガラス、ナビゲーション、状態管理、最新の API に関する 38 の iOS 開発ルールと 12 の SwiftUI ベスト プラクティス ガイドが含まれています。 3 つのツールはすべて 1 つのバイナリ (appledev) として出荷されます。使用場面: Apple API ドキュメント、App Store Connect 管理、WWDC ルックアップ、または iOS/watchOS/tvOS/macOS/visionOS アプリを最初から構築する場合。次の場合には使用しないでください: Apple 以外のプラットフォームまたは一般的なコーディング。
