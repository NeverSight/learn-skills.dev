Универсальные навыки разработчика Apple с тремя интегрированными инструментами, поставляемыми в виде одного унифицированного двоичного файла. (1) Поиск документации по платформам Apple, символам и 1267 сессиям WWDC за 2014–2025 годы. Никаких учетных данных не требуется. (2) Интерфейс командной строки App Store Connect с более чем 120 командами, охватывающими сборки (поиск/ожидание/загрузка), TestFlight, проверка перед отправкой, отправки, подписание, подписки (для всей семьи), IAP, аналитика, Xcode Cloud, рабочие процессы метаданных, панель мониторинга конвейера релизов, аналитика, предложения возврата, рекламируемые покупки, страницы продуктов, номинации, декларации о доступности, предварительные заказы, ценовые фильтры, обновление локализаций, различия, веб-перехватчики с локальным приемником, автоматизация рабочих процессов и многое другое. Требуется ключ API App Store Connect. (3) Многоплатформенный конструктор приложений (iOS/watchOS/tvOS/iPad/macOS/visionOS), который создает полные приложения Swift/SwiftUI на естественном языке с автоматическим исправлением, запуском симулятора, режимом интерактивного чата и открытием в Xcode. Требуется ключ API LLM и Xcode. Включает 38 правил разработки для iOS и 12 руководств по передовому опыту SwiftUI для Liquid Glass, навигации, управления состоянием и современных API. Все три инструмента поставляются в виде одного двоичного файла (appledev). КОГДА ИСПОЛЬЗУЕТСЯ: документация Apple API, управление App Store Connect, поиск WWDC или создание приложений iOS/watchOS/tvOS/macOS/visionOS с нуля. НЕ ИСПОЛЬЗУЙТЕ КОГДА: платформы, не принадлежащие Apple, или общее кодирование.
