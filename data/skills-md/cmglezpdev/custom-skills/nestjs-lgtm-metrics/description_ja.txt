OpenTelemetry SDK と OTel Collector を使用して、アプリケーション レベルの Prometheus メトリクスを NestJS アプリに追加します。 HTTP RED メトリクス、Node.js ランタイム メトリクス、ビジネス メトリクス、高度な可観測性パターンをカバーします。ユーザーが NestJS アプリケーションにメトリクス、ダッシュボード、またはモニタリングを追加したい場合、Prometheus、Grafana メトリクス、OpenTelemetry メトリクス、OTel Collector について言及する場合、または SLI/SLO、または NestJS コンテキストでのアプリケーション レベルのモニタリングについて質問する場合は、このスキルを使用します。ユーザーが NestJS でカスタム カウンター、ヒストグラム、ゲージ、または概要を必要とする場合にもトリガーします。このスキルは、アプリケーションが発行する必要があるメトリクスのみに焦点を当てています。ロギング、トレース、cAdvisor、node-exporter、postgres-exporter、または redis-exporter からのインフラレベルのメトリクスはカバーされていません。
