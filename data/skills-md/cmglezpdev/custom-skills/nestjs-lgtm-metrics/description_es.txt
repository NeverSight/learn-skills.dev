Agregue métricas de Prometheus a nivel de aplicación a una aplicación NestJS utilizando el SDK de OpenTelemetry y un OTel Collector. Cubre métricas HTTP RED, métricas de tiempo de ejecución de Node.js, métricas comerciales y patrones de observabilidad avanzados. Utilice esta habilidad siempre que el usuario quiera agregar métricas, paneles o monitoreo a una aplicación NestJS, mencione Prometheus, métricas de Grafana, métricas de OpenTelemetry, OTel Collector o pregunte sobre SLI/SLO o monitoreo a nivel de aplicación en un contexto NestJS. También se activa cuando el usuario desea contadores, histogramas, indicadores o resúmenes personalizados en NestJS. Esta habilidad se enfoca exclusivamente en métricas que debe emitir la APLICACIÓN. No cubre el registro, el seguimiento ni las métricas de infranivel de cAdvisor, node-exporter, postgres-exporter o redis-exporter.
