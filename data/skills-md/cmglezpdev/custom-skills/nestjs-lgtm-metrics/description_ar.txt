أضف مقاييس Prometheus على مستوى التطبيق إلى تطبيق NestJS باستخدام OpenTelemetry SDK وOTel Collector. يغطي مقاييس HTTP RED ومقاييس وقت تشغيل Node.js ومقاييس الأعمال وأنماط إمكانية المراقبة المتقدمة. استخدم هذه المهارة عندما يريد المستخدم إضافة مقاييس أو لوحات معلومات أو مراقبة إلى تطبيق NestJS، أو يذكر Prometheus، أو مقاييس Grafana، أو مقاييس OpenTelemetry، أو OTel Collector، أو يسأل عن SLIs/SLOs، أو المراقبة على مستوى التطبيق في سياق NestJS. يتم تشغيله أيضًا عندما يريد المستخدم عدادات أو رسوم بيانية أو مقاييس أو ملخصات مخصصة في NestJS. تركز هذه المهارة حصريًا على المقاييس التي يجب أن يصدرها التطبيق. ولا يغطي التسجيل أو التتبع أو مقاييس المستوى تحت الأحمر من cAdvisor أو Node-exporter أو postgres-exporter أو redis-exporter.
