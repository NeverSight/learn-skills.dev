使用 OpenTelemetry SDK 和 OTel Collector 將應用程式級 Prometheus 指標新增至 NestJS 應用程式。涵蓋 HTTP RED 指標、Node.js 執行時期指標、業務指標和進階可觀察性模式。每當使用者想要在 NestJS 應用程式中新增指標、儀表板或監控、提及 Prometheus、Grafana 指標、OpenTelemetry 指標、OTel Collector，或詢問有關 SLI/SLO 或 NestJS 上下文中的應用程式級監控時，請使用此技能。當使用者想要在 NestJS 中自訂計數器、直方圖、儀表或摘要時也會觸發。該技能專門關注應用程式必須發出的指標。它不涵蓋來自 cAdvisor、node-exporter、postgres-exporter 或 redis-exporter 的日誌記錄、追蹤、基礎等級指標。
