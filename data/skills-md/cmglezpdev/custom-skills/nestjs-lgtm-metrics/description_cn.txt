使用 OpenTelemetry SDK 和 OTel Collector 将应用程序级 Prometheus 指标添加到 NestJS 应用程序。涵盖 HTTP RED 指标、Node.js 运行时指标、业务指标和高级可观察性模式。每当用户想要向 NestJS 应用程序添加指标、仪表板或监控、提及 Prometheus、Grafana 指标、OpenTelemetry 指标、OTel Collector，或询问有关 SLI/SLO 或 NestJS 上下文中的应用程序级监控时，请使用此技能。当用户想要在 NestJS 中自定义计数器、直方图、仪表或摘要时也会触发。该技能专门关注应用程序必须发出的指标。它不涵盖来自 cAdvisor、node-exporter、postgres-exporter 或 redis-exporter 的日志记录、跟踪、基础级别指标。
