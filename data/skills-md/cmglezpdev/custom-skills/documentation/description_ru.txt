Как создавать, обновлять и управлять документацией для любого программного проекта. Используйте этот навык всякий раз, когда вас просят добавить документацию, обновить документацию, создать модуль Runbook, написать ADR, документировать функцию, добавить спецификацию продукта, написать руководства по адаптации или выполнить любую задачу, связанную с документацией. Также срабатывает при внесении изменений кода, изменяющих поведение (функции, исправления, рефакторинг, интеграцию, инфраструктурные изменения), которые требуют сопутствующих обновлений документации — даже если пользователь явно не упоминает «документацию».
