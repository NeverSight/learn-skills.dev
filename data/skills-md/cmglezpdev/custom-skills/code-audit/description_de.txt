Führen Sie eine umfassende Sicherheits- und Codequalitätsprüfung für webbasierte Projekte durch (React, Next.js, NestJS). Verwendet den OWASP Top 10:2025-Standard als primäres Sicherheits-Framework. Erstellt einen detaillierten Markdown-Bericht mit nach Schweregrad kategorisierten Ergebnissen (KRITISCH, HOCH, MITTEL, NIEDRIG). Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer seine Codebasis auf Schwachstellen, Sicherheitsprobleme, Codequalitätsprobleme, fehlerhafte Muster oder potenzielle Fehler überprüfen, überprüfen, scannen oder analysieren möchte. Wird auch ausgelöst, wenn der Benutzer „OWASP“, „Sicherheitsüberprüfung“, „Schwachstellenscan“, „Code-Audit“, „Pentest-Überprüfung“, „Sicherheitsbewertung“, „Code-Gesundheitsprüfung“ erwähnt oder fragt: „Ist mein Code sicher?“ oder „Finde Fehler in meinem Projekt“. Wird auch dann ausgelöst, wenn der Benutzer nur „Überprüfe dies“ oder „Überprüfe meinen Code“ sagt und auf ein Webprojekt zeigt. Dieser Skill unterstützt React-, Next.js- und NestJS-Projekte, einschließlich Monorepos mit mehreren Projekttypen.
