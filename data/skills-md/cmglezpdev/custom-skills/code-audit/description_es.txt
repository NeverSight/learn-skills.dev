Realice una auditoría integral de seguridad y calidad del código en proyectos basados ​​en web (React, Next.js, NestJS). Utiliza el estándar OWASP Top 10:2025 como marco de seguridad principal. Genera un informe de Markdown detallado con hallazgos categorizados por gravedad (CRÍTICO, ALTO, MEDIO, BAJA). Utilice esta habilidad siempre que el usuario solicite auditar, revisar, escanear o analizar su base de código en busca de vulnerabilidades, problemas de seguridad, problemas de calidad del código, patrones incorrectos o errores potenciales. También se activa cuando el usuario menciona "OWASP", "revisión de seguridad", "análisis de vulnerabilidades", "auditoría de código", "revisión de pentest", "evaluación de seguridad", "verificación del estado del código" o pregunta "¿mi código es seguro?" o "encontrar errores en mi proyecto". Se activa incluso si el usuario simplemente dice "auditar esto" o "verificar mi código" apuntando a un proyecto web. Esta habilidad admite proyectos React, Next.js y NestJS, incluidos monorepos que contienen múltiples tipos de proyectos.
