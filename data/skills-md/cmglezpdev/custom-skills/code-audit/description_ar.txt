إجراء تدقيق شامل للأمان وجودة التعليمات البرمجية في المشاريع المستندة إلى الويب (React وNext.js وNestJS). يستخدم معيار OWASP Top 10:2025 كإطار عمل أمني أساسي. يُنشئ تقريرًا مفصلاً عن تخفيض السعر مع النتائج المصنفة حسب الخطورة (حرجة، عالية، متوسطة، منخفضة). استخدم هذه المهارة عندما يطلب المستخدم تدقيق قاعدة التعليمات البرمجية الخاصة به أو مراجعتها أو مسحها أو تحليلها بحثًا عن نقاط الضعف أو المشكلات الأمنية أو مشكلات جودة التعليمات البرمجية أو الأنماط السيئة أو الأخطاء المحتملة. يتم تشغيله أيضًا عندما يذكر المستخدم "OWASP"، أو "مراجعة الأمان"، أو "فحص الثغرات الأمنية"، أو "تدقيق الكود"، أو "مراجعة Pentest"، أو "تقييم الأمان"، أو "التحقق من سلامة الكود"، أو يسأل "هل الكود الخاص بي آمن؟" أو "ابحث عن الأخطاء في مشروعي". يتم تشغيله حتى إذا قال المستخدم فقط "تدقيق هذا" أو "التحقق من الكود الخاص بي" مشيرًا إلى مشروع ويب. تدعم هذه المهارة مشاريع React وNext.js وNestJS، بما في ذلك monorepos التي تحتوي على أنواع متعددة من المشاريع.
