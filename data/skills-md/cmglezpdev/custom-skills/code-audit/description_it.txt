Esegui un controllo completo della sicurezza e della qualità del codice su progetti basati sul Web (React, Next.js, NestJS). Utilizza lo standard OWASP Top 10:2025 come framework di sicurezza principale. Genera un rapporto Markdown dettagliato con risultati classificati in base alla gravità (CRITICO, ALTO, MEDIO, BASSO). Utilizza questa abilità ogni volta che l'utente chiede di verificare, rivedere, scansionare o analizzare la propria base di codice per vulnerabilità, problemi di sicurezza, problemi di qualità del codice, modelli errati o potenziali bug. Si attiva anche quando l'utente menziona "OWASP", "revisione della sicurezza", "scansione delle vulnerabilità", "audit del codice", "revisione pentest", "valutazione della sicurezza", "controllo dello stato del codice" o chiede "il mio codice è sicuro?" o "trova bug nel mio progetto". Si attiva anche se l'utente dice semplicemente "controlla questo" o "controlla il mio codice" indicando un progetto web. Questa competenza supporta progetti React, Next.js e NestJS, inclusi monorepos contenenti più tipi di progetto.
