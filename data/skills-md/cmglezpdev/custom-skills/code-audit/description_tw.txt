對基於 Web 的專案（React、Next.js、NestJS）執行全面的安全性和程式碼品質審核。使用 OWASP Top 10:2025 標準作為主要安全框架。產生詳細的 Markdown 報告，其中包含以嚴重性（嚴重、高、中、低）分類的結果。每當用戶要求審核、審查、掃描或分析其程式碼庫以查找漏洞、安全性問題、程式碼品質問題、不良模式或潛在錯誤時，請使用此技能。當使用者提到「OWASP」、「安全審查」、「漏洞掃描」、「程式碼審計」、「滲透測試審查」、「安全評估」、「程式碼健康檢查」或詢問「我的程式碼安全嗎？」時也會觸發。或“查找我的項目中的錯誤”。即使使用者只是指向 Web 專案說「審核此」或「檢查我的程式碼」也會觸發。此技能支援 React、Next.js 和 NestJS 項目，包括包含多種項目類型的 monorepos。
