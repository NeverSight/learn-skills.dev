Web ベースのプロジェクト (React、Next.js、NestJS) に対して包括的なセキュリティとコード品質の監査を実行します。 OWASP Top 10:2025 標準を主要なセキュリティ フレームワークとして使用します。重大度 (CRITICAL、HIGH、MEDIUM、LOW) ごとに分類された結果を含む詳細な Markdown レポートを生成します。ユーザーがコードベースの脆弱性、セキュリティ問題、コード品質の問題、悪いパターン、または潜在的なバグの監査、レビュー、スキャン、または分析を要求するときは常に、このスキルを使用します。また、ユーザーが「OWASP」、「セキュリティ レビュー」、「脆弱性スキャン」、「コード監査」、「侵入テスト レビュー」、「セキュリティ評価」、「コードの健全性チェック」について言及した場合、または「私のコードは安全ですか?」と尋ねた場合にもトリガーされます。または「プロジェクトのバグを見つける」。ユーザーが Web プロジェクトを指して「これを監査してください」または「コードをチェックしてください」と言っただけでもトリガーされます。このスキルは、複数のプロジェクト タイプを含むモノリポジトリを含む、React、Next.js、および NestJS プロジェクトをサポートします。
