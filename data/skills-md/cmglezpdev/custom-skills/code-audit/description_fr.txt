Effectuer un audit complet de sécurité et de qualité du code sur des projets Web (React, Next.js, NestJS). Utilise la norme OWASP Top 10:2025 comme cadre de sécurité principal. Génère un rapport Markdown détaillé avec les résultats classés par gravité (CRITIQUE, ÉLEVÉ, MOYEN, FAIBLE). Utilisez cette compétence chaque fois que l'utilisateur demande à auditer, examiner, analyser ou analyser sa base de code à la recherche de vulnérabilités, de problèmes de sécurité, de problèmes de qualité du code, de mauvais modèles ou de bogues potentiels. Se déclenche également lorsque l'utilisateur mentionne « OWASP », « examen de sécurité », « analyse de vulnérabilité », « audit de code », « examen pentest », « évaluation de sécurité », « contrôle de santé du code » ou demande « mon code est-il sécurisé ? ou "trouver des bugs dans mon projet". Déclenchez même si l'utilisateur dit simplement « auditer ceci » ou « vérifier mon code » en pointant vers un projet Web. Cette compétence prend en charge les projets React, Next.js et NestJS, y compris les monorepos contenant plusieurs types de projets.
