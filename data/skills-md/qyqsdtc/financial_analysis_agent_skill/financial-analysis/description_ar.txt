مهارات تحليل البيانات المالية المهنية للشركات المدرجة. يستخدم لتحليل التقارير المالية بصيغة PDF (ربع سنوي، نصف سنوي، سنوي)، واستخراج البيانات من البيانات الرئيسية الثلاثة، وحساب المؤشرات المالية متعددة الأبعاد مثل الربحية، والملاءة المالية، والكفاءة التشغيلية، وجودة التدفق النقدي، والنمو، وإجراء تحليل DuPont والاتجاهات، وإنشاء تقييمات وتوصيات الاستثمار. يدعم إنشاء تقارير تحليلية احترافية بتنسيقات Markdown وExcel. استخدم هذه المهارة عندما يطلب المستخدمون تحليل التقارير المالية، أو حساب المؤشرات المالية مثل العائد على حقوق المساهمين/العائد على الأصول، أو تقييم الحالة المالية للشركة، أو تقديم نصائح استثمارية.
