Supporta la generazione di report di analisi professionali nei formati Markdown ed Excel. Utilizza questa competenza quando l'utente richiede l'analisi dei report finanziari, il calcolo del ROE/ROA e altri indicatori, la valutazione della salute finanziaria dell'azienda o la generazione di consigli sugli investimenti.
