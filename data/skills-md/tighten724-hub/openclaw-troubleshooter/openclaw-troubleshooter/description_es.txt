Habilidades de resolución de problemas de OpenClaw. Úselo cuando los usuarios encuentren los siguientes problemas:
1. La puerta de enlace no se inicia, no se puede conectar o no responde
2. Problemas de conexión de canales (WhatsApp, Telegram, Discord, etc.)
3. Problemas de configuración del modelo (clave API, falla en el cambio de modelo)
4. La interfaz de usuario del panel/control no se abre o la conexión falla
5. Problemas de sesión (sesión perdida, no se puede crear una nueva sesión)
6. Errores de configuración (errores en el archivo de configuración)
7. Problemas con los nodos (fallo de emparejamiento, fallo de llamada de herramienta)
8. Problemas con la herramienta del navegador
9. Las tareas cron/heartbeat no se ejecutan
10. Problemas de instalación/actualización

Palabras clave de activación: puerta de enlace, inicio, conexión, error, problema, falla, incapaz, falla, incorrecto, panel, interfaz de usuario, canal, modelo, sesión, nodo, navegador, cron, latido, instalación, actualización

Busque soluciones a pedido en los documentos oficiales y brinde instrucciones paso a paso que incluso un novato puede seguir.
