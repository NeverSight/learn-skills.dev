OpenClaw troubleshooting skills. Use when users encounter:

1. Gateway won’t start, can’t connect, or is unresponsive  
2. Channel connection issues (WhatsApp, Telegram, Discord, etc.)  
3. Model configuration problems (API Key, model-switch failure)  
4. Dashboard/Control UI won’t open or fails to connect  
5. Session issues (lost sessions, can’t create new ones)  
6. Configuration errors (bad config files)  
7. Node problems (pairing failure, tool-call failure)  
8. Browser-tool issues  
9. Cron/heartbeat tasks not running  
10. Install/update problems  

Trigger keywords: Gateway, start, connect, error, problem, fault, unable, fail, failure, error, Dashboard, UI, channel, model, session, node, browser, Cron, heartbeat, install, update  

Find solutions on demand from the official docs and give step-by-step instructions even a beginner can follow.
