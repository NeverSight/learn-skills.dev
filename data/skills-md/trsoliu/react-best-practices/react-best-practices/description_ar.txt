أكمل دليل تطوير مشروع React (بالصينية). يغطي اختيار مكدس التكنولوجيا، وبنية Monorepo (pnpm + Turborepo)، ومعايير التطوير، واتفاقيات الالتزام Git، وإدارة Changelog، واستراتيجيات الاختبار، ومراجعة التعليمات البرمجية، وسير عمل التطوير، وإدارة الوثائق. مصممة لـ React + TypeScript + Zustand + TanStack Query + Vitest + Playwright stack. يتم تنشيطه تلقائيًا عندما يقوم المستخدمون بإنشاء مشروع React، أو كتابة/مراجعة/إعادة صياغة كود React، أو تكوين الاختبار أو CI/CD، أو إنشاء مستندات المتطلبات/التصميم.
