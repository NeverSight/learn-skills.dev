Vollständiger Prozessleitfaden zur React-Projektentwicklung (Chinesisch). Behandelt die Auswahl des Technologie-Stacks, die Monorepo-Architektur (pnpm + Turborepo), Entwicklungsstandards, Git-Commit-Konventionen, Changelog-Management, Teststrategien, Codeüberprüfungen, Entwicklungsworkflows und Dokumentationsmanagement. Anwendbar auf React + TypeScript + Zustand + TanStack Query + Vitest + Playwright-Stack. Wird automatisch aktiviert, wenn Benutzer React-Projekte erstellen, React-Code schreiben/überprüfen/umgestalten, Tests oder CI/CD konfigurieren oder Anforderungen/Designdokumente erstellen.
