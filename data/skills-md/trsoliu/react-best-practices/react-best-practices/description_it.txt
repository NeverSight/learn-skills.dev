Una guida completa allo sviluppo di progetti React (cinese). Copre la selezione dello stack tecnologico, l'architettura Monorepo (pnpm + Turborepo), gli standard di sviluppo, le convenzioni di commit Git, la gestione del changelog, le strategie di test, le revisioni del codice, i flussi di lavoro di sviluppo e la gestione della documentazione. Adatto per lo stack React + TypeScript + Zustand + TanStack Query + Vitest + Playwright. Si attiva automaticamente quando gli utenti creano progetti React, scrivono/revisionano/refactoring il codice React, configurano test o CI/CD o creano documentazione di requisiti/progettazione.
