Guide complet du processus de développement de projet React (chinois). Couvre la sélection de la pile technologique, l'architecture Monorepo (pnpm + Turborepo), les conventions de développement, les conventions de validation Git, la gestion des journaux de modifications, les stratégies de test, les révisions de code, les flux de travail de développement et la gestion de la documentation. Convient pour React + TypeScript + Zustand + TanStack Query + Vitest + Playwright tech stack. S'active automatiquement lorsque les utilisateurs créent des projets React, écrivent/révisent/refactorisent le code React, configurent les tests ou CI/CD, ou créent des exigences/documents de conception.
