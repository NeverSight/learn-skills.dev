Полное руководство по разработке проектов React (на китайском языке). Охватывает выбор технического стека, архитектуру Monorepo (pnpm + Turborepo), стандарты разработки, соглашения о фиксации Git, управление журналом изменений, стратегии тестирования, обзоры кода, рабочие процессы разработки и управление документацией. Специально разработан для стека React + TypeScript + Zustand + TanStack Query + Vitest + Playwright. Автоматически активируется, когда пользователи создают проекты React, пишут/просматривают/рефакторят код React, настраивают тестирование или CI/CD, а также создают требования/проектные документы.
