Guía del proceso completo de desarrollo de proyectos React (chino). Cubre la selección de pila de tecnología, arquitectura Monorepo (pnpm + Turborepo), estándares de desarrollo, convenciones de confirmación de Git, gestión de registros de cambios, estrategias de prueba, revisión de código, flujos de trabajo de desarrollo y gestión de documentación. Adecuado para React + TypeScript + Zustand + TanStack Query + Vitest + Playwright stack. Se activa automáticamente cuando los usuarios crean proyectos de React, escriben/revisan/refactorizan código de React, configuran pruebas o CI/CD, o crean requisitos/documentos de diseño.
