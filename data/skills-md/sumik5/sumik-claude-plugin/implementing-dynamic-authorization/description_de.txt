Dynamisches Autorisierungsdesign, das ABAC-, ReBAC-, PBAC-Modelle und die Cedar-Richtliniensprache umfasst. Verwenden Sie es beim Entwerfen von Zugangskontrollsystemen, beim Auswählen von Autorisierungsmodellen oder beim Schreiben von Cedar-Richtlinien.
