Комплексное руководство по написанию сценариев и автоматизации оболочки Bash, охватывающее основы, поток управления, конвейеры ввода-вывода, управление процессами, автоматизацию системного администрирования, тестирование/отладку, безопасность, шаблоны проектирования и автоматизацию тестирования на проникновение (Nmap, веб-сканирование, эксплуатация, оценка Wi-Fi, отчеты). ДОЛЖЕН загружаться при обнаружении файлов .sh или написании сценариев оболочки. Для шаблонов, специфичных для Docker, вместо этого используйте Managing-Docker. Для более широкой методологии DevOps используйте вместо этого практикующую DevOps.
