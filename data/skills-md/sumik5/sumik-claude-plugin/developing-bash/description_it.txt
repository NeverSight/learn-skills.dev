Guida completa allo scripting e all'automazione della shell Bash che copre i fondamenti, il flusso di controllo, le pipeline di I/O, il controllo dei processi, l'automazione dell'amministrazione del sistema, test/debug, sicurezza, modelli di progettazione e automazione dei test di penetrazione (Nmap, scansione web, sfruttamento, valutazione Wi-Fi, reporting). DEVE essere caricato quando vengono rilevati file .sh o vengono scritti script di shell. Per i modelli specifici di Docker, utilizzare invece Managing-docker. Per una metodologia DevOps più ampia, utilizzare invece practice-devops.
