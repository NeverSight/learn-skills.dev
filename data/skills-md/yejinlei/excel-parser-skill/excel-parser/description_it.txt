Supporta l'estrazione e la scrittura di contenuti per più formati Excel, utilizzando la libreria calamine per gestire .xls, .xlsx, .xlsm e altri formati. Quando gli utenti devono analizzare file Excel, estrarre dati di tabelle, convertire Excel in formato testo, analizzare contenuto Excel, elaborare file Excel in batch, creare nuovi file Excel o aggiornare quelli esistenti, è necessario utilizzare questa competenza. Adatto per scenari quali estrazione di dati, analisi di report, controllo dei contenuti ed esportazione di dati.
