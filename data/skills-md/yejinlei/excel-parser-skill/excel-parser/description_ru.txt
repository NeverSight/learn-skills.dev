Поддерживает извлечение и запись контента для нескольких форматов Excel, используя библиотеку Calamine для обработки .xls, .xlsx, .xlsm и других форматов. Этот навык необходимо использовать, когда пользователям необходимо анализировать файлы Excel, извлекать табличные данные, преобразовывать Excel в текстовый формат, анализировать содержимое Excel, пакетно обрабатывать файлы Excel, создавать новые файлы Excel или обновлять существующие. Подходит для таких сценариев, как извлечение данных, анализ отчетов, аудит контента и экспорт данных.
