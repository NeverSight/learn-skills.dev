Cette compétence doit être utilisée lors de la réalisation de tests de mutation basés sur l'IA pour évaluer et améliorer la qualité des tests unitaires. Il génère des mutants de code ciblés, exécute des tests pour identifier les mutants survivants et renforce ou crée des tests pour les tuer. Accepte un chemin de fichier, un répertoire ou par défaut les fichiers modifiés git diff.
