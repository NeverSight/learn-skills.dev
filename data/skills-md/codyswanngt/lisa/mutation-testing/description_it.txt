Questa abilità dovrebbe essere utilizzata quando si eseguono test di mutazione basati sull'intelligenza artificiale per valutare e migliorare la qualità dei test unitari. Genera codici mutanti mirati, esegue test per identificare i mutanti sopravvissuti e rafforza o crea test per ucciderli. Accetta un percorso file, una directory o i valori predefiniti per i file modificati da git diff.
