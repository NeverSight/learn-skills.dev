Этот навык следует использовать при проведении мутационного тестирования с помощью ИИ для оценки и улучшения качества модульного тестирования. Он генерирует целевые мутанты кода, запускает тесты для выявления выживших мутантов, а также усиливает или создает тесты для их уничтожения. Принимает путь к файлу, каталог или значения по умолчанию для измененных файлов git diff.
