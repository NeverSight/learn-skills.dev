Générateur de matériel de demande d'enregistrement de droits d'auteur de logiciels informatiques en Chine. Utilisé pour générer tous les documents requis pour les applications de droits d'auteur sur les logiciels, y compris le remplissage du formulaire d'informations de base sur le logiciel, le matériel d'identification du programme PDF (surlignage de la syntaxe du code source) et le matériel d'identification du document PDF (manuel d'utilisation). Utilisez cette compétence lorsque les utilisateurs mentionnent « 软著 », « 软件著作权 », « 著作权登记 », « 版权登记 », « 软著申请 » ou « enregistrement des droits d'auteur ».
