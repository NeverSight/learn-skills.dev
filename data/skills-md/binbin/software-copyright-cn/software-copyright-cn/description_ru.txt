Заявка на регистрацию авторских прав на компьютерное программное обеспечение в Китае Генератор материалов. Используется для создания всех материалов, необходимых для подачи заявки на авторские права на программное обеспечение, включая заполнение форм базовой информации о программном обеспечении, PDF-файл с идентификационными материалами программы (подсветка синтаксиса исходного кода) и PDF-файл с идентификационными материалами документов (руководство пользователя).
