Active esta habilidad cuando el usuario desee recopilar datos estructurados, crear formularios o configurar puntos finales de envío. Cubre cualquier tipo de formulario (contacto, comentarios, registro, lista de espera, informe de errores, soporte, captura de clientes potenciales, encuestas, aplicaciones) y cualquier método de recopilación (formularios HTML, scripts, agentes de IA). También activa la gestión de formularios o envíos existentes, la generación de código de integración de formularios de interfaz, la configuración de webhooks o notificaciones de Discord/Slack, la configuración de protección contra spam o funciones de IA (traducción, respuestas inteligentes), la conexión de MCP para la gestión de formularios asistida por IA o cualquier cosa que haga referencia a Postbox o usepostbox.com. Si el usuario necesita que los datos fluyan desde cualquier lugar, se aplica esta habilidad.
