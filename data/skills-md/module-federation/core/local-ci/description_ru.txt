Запустите локальные команды четности CI этого репозитория и задания `pnpm run ci:local`. Используйте при проверке изменения по тем же формам заданий, которые используются в действиях GitHub, выбирая наименьшее локальное задание CI для пакета, рабочего процесса, E2E, Metro, инструментов разработчика, размера пакета или изменения сборки и тестирования.
