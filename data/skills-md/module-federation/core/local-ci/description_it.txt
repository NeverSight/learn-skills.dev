Esegui i comandi di parità CI locale di questo repository e i lavori `pnpm run ci:local`. Da utilizzare quando si convalida una modifica rispetto alle stesse forme di lavoro utilizzate in GitHub Actions, scegliendo il lavoro CI locale più piccolo per un pacchetto, flusso di lavoro, E2E, Metro, devtools, dimensione del bundle o modifica di creazione e test.
