قم بتحديث عنوان GitHub PR الحالي ونص هذا المستودع بحيث يتطابقان مع قالب طلب السحب الخاص بالمستودع ونمط عنوان الالتزام التقليدي. يُستخدم عندما يكون عنوان العلاقات العامة غامضًا أو غير منسق، عندما يفتقد نص العلاقات العامة الأقسام المطلوبة أو عناصر قائمة المراجعة، عندما يحتاج الدستور الغذائي إلى تسوية بيانات تعريف العلاقات العامة قبل المراجعة أو الدمج، أو عندما تشير تعليقات/فحوصات GitHub إلى ضرورة تصحيح عنوان العلاقات العامة أو نص العلاقات العامة.
