Mettez à jour le titre et le corps actuels de GitHub PR pour ce référentiel afin qu'ils correspondent au modèle de demande d'extraction du dépôt et au style de titre de validation conventionnel. À utiliser lorsqu'un titre de PR est vague ou mal formaté, lorsque le corps de la PR ne contient pas les sections requises ou les éléments de la liste de contrôle, lorsque le Codex doit normaliser les métadonnées de la PR avant de les examiner ou de les fusionner, ou lorsque les commentaires/vérifications GitHub indiquent que le titre ou le corps de la PR doit être corrigé.
