Aggiorna il titolo e il corpo PR di GitHub correnti per questo repository in modo che corrispondano al modello di richiesta pull del repository e allo stile del titolo con commit convenzionale. Da utilizzare quando un titolo PR è vago o non formattato correttamente, quando nel corpo PR mancano le sezioni richieste o gli elementi dell'elenco di controllo, quando il Codex deve normalizzare i metadati PR prima della revisione o dell'unione o quando i commenti/controlli di GitHub indicano che il titolo o il corpo PR deve essere corretto.
