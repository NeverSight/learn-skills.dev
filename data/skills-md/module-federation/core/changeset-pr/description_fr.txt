Créez ou mettez à jour un fichier `.changeset/*.md` pour la branche ou PR actuelle dans ce référentiel, choisissez la portée du package et le type de version corrects, et vérifiez le résultat par rapport à la configuration des ensembles de modifications spécifiques au référentiel. À utiliser lorsqu'un package publiable a été modifié, lorsqu'un PR manque un ensemble de modifications, lorsqu'un ensemble de modifications existant doit être corrigé ou lorsque le Codex doit confirmer si une branche doit avoir un ensemble de modifications.
