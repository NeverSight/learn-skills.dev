Erstellen oder aktualisieren Sie eine „.changeset/*.md“-Datei für den aktuellen Zweig oder PR in diesem Repository, wählen Sie den richtigen Paketumfang und Release-Typ aus und überprüfen Sie das Ergebnis anhand der Repo-spezifischen Changesets-Konfiguration. Wird verwendet, wenn sich ein veröffentlichbares Paket geändert hat, wenn einem PR ein Änderungssatz fehlt, wenn ein vorhandener Änderungssatz korrigiert werden muss oder wenn Codex bestätigen muss, ob ein Zweig überhaupt einen Änderungssatz haben sollte.
