Crea o aggiorna un file `.changeset/*.md` per il ramo corrente o PR in questo repository, scegli l'ambito del pacchetto e il tipo di rilascio corretti e verifica il risultato rispetto alla configurazione Changesets specifica del repository. Da utilizzare quando un pacchetto pubblicabile è stato modificato, quando in un PR manca un insieme di modifiche, quando un insieme di modifiche esistente necessita di correzione o quando il Codex deve confermare se un ramo deve avere un insieme di modifiche.
