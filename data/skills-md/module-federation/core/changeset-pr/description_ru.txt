Создайте или обновите файл `.changeset/*.md` для текущей ветки или PR в этом репозитории, выберите правильный объем пакета и тип выпуска и проверьте результат на соответствие конфигурации наборов изменений, специфичных для репозитория. Используйте, когда был изменен публикуемый пакет, когда в PR отсутствует набор изменений, когда существующий набор изменений требует исправления или когда Кодексу необходимо подтвердить, должен ли вообще ветке иметь набор изменений.
