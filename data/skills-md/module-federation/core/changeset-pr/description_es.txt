Cree o actualice un archivo `.changeset/*.md` para la rama o PR actual en este repositorio, elija el alcance del paquete y el tipo de versión correctos y verifique el resultado con la configuración de conjuntos de cambios específica del repositorio. Úselo cuando un paquete publicable cambió, cuando a un PR le falta un conjunto de cambios, cuando un conjunto de cambios existente necesita corrección o cuando Codex necesita confirmar si una rama debería tener un conjunto de cambios.
