Récupérez les métadonnées et les informations de manifeste pour un module de fédération de modules distant : publicPath, remoteEntry, tapez les URL des fichiers et les télécommandes/expositions/partagées du module à partir de son mf-manifest.json. Deux modes : (1) dans un projet consommateur, transmettez uniquement le nom distant et l'URL d'entrée est résolue à partir de mfConfig.remotes ; (2) en dehors d'un projet consommateur, transmettez directement le nom distant ainsi que son URL remoteEntry. Pour inspecter la configuration du projet actuel, utilisez plutôt mf-context.
