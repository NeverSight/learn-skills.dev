Obtenga metadatos e información de manifiesto para un módulo de Federación de módulos remoto: publicPath, entrada remota, URL de archivos de tipo y los controles remotos/expuestos/compartidos del módulo desde su mf-manifest.json. Dos modos: (1) dentro de un proyecto de consumidor, se pasa solo el nombre remoto y la URL de entrada se resuelve desde mfConfig.remotes; (2) fuera de un proyecto de consumidor, pase el nombre remoto más su URL de entrada remota directamente. Para inspeccionar la configuración propia del proyecto actual, utilice mf-context en su lugar.
