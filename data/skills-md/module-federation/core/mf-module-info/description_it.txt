Recupera i metadati e le informazioni sul manifest per un modulo remoto di Module Federation: publicPath, remoteEntry, digita gli URL dei file e remotes/exposes/shared del modulo dal relativo mf-manifest.json. Due modalità: (1) all'interno di un progetto consumer, passa solo il nome remoto e l'URL di ingresso viene risolto da mfConfig.remotes; (2) all'esterno di un progetto consumer, passare direttamente il nome remoto e il relativo URL remoteEntry. Per ispezionare la configurazione del progetto corrente, utilizzare invece mf-context.
