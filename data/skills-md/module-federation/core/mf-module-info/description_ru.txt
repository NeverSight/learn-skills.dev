Получите метаданные и информацию манифеста для удаленного модуля Module Federation — publicPath, RemoteEntry, URL-адреса файлов типа и удаленные/экспонируемые/общие данные модуля из его mf-manifest.json. Два режима: (1) внутри потребительского проекта передается только удаленное имя, а URL-адрес записи разрешается из mfConfig.remotes; (2) вне потребительского проекта напрямую передайте удаленное имя и его URL-адрес RemoteEntry. Чтобы проверить собственную конфигурацию текущего проекта, используйте вместо этого mf-context.
