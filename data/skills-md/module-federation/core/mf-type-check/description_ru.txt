Диагностика проблем с типом Федерации модулей по трем категориям: (1) сбои при создании файла типа производителя (TYPE-001), (2) потребитель не может получить удаленные типы, (3) tsconfig не настроен для использования удаленных типов. Используйте, когда пользователь сталкивается с ошибками типа TS, отсутствует папка @mf-types или не может использовать типы удаленных модулей.
