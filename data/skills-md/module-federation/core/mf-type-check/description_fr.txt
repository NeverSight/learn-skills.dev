Diagnostiquez les problèmes de type de fédération de modules dans trois catégories : (1) échecs de génération de fichiers de type producteur (TYPE-001), (2) échec du consommateur dans l'extraction des types distants, (3) tsconfig non configuré pour consommer des types distants. À utiliser lorsque l'utilisateur rencontre des erreurs de type TS, manque le dossier @mf-types ou ne peut pas consommer de types de modules distants.
