Diagnostica i problemi relativi ai tipi di federazione dei moduli in tre categorie: (1) errori di generazione di file di tipo produttore (TYPE-001), (2) errore del consumatore nel pull di tipi remoti, (3) tsconfig non configurato per utilizzare tipi remoti. Da utilizzare quando l'utente riscontra errori di tipo TS, cartella @mf-types mancante o non può utilizzare tipi di moduli remoti.
