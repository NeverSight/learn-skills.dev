Diagnostique problemas de tipo de federación de módulos en tres categorías: (1) fallas en la generación de archivos de tipo de productor (TYPE-001), (2) el consumidor no puede extraer tipos remotos, (3) tsconfig no está configurado para consumir tipos remotos. Úselo cuando el usuario encuentre errores de tipo TS, falte la carpeta @mf-types o no pueda consumir tipos de módulos remotos.
