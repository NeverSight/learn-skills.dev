تشخيص مشكلات نوع اتحاد الوحدة عبر ثلاث فئات: (1) فشل إنشاء ملف نوع المنتج (TYPE-001)، (2) فشل المستهلك في سحب الأنواع البعيدة، (3) لم يتم تكوين tsconfig لاستهلاك الأنواع البعيدة. يُستخدم عندما يواجه المستخدم أخطاء في نوع TS، أو يفتقد مجلد @mf-types، أو لا يمكنه استهلاك أنواع الوحدة النمطية البعيدة.
