Greifen Sie über die API auf Aikido Security zu, rufen Sie offene Problemgruppen ab, selektieren Sie Ergebnisse und führen Sie First-Pass-Korrekturen in Ihrem Repository durch. Verwenden Sie diese Funktion, wenn Sie aufgefordert werden, Aikido-Warnungen zu überprüfen, Abhängigkeiten oder SAST-Ergebnisse zu beheben oder Sicherheitsprobleme zu schließen.
