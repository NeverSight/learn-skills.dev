Utilizza questa competenza quando scrivi codice Python che interagisce con i sandbox cloud Moru. Ciò include: creazione di sandbox con `Sandbox.create()`, esecuzione di comandi con `sbx.commands.run()`, lettura e scrittura di file con `sbx.files.read()` e `sbx.files.write()`, lavoro con volumi persistenti utilizzando la classe `Volume`, creazione di modelli personalizzati con il builder `Template`, gestione dei processi in background, streaming dell'output dei comandi e operazioni asincrone con `AsyncSandbox`. La competenza copre l'API Python SDK completa, inclusa la gestione degli errori con "TimeoutException", "CommandExitException", "AuthenticationException" e altre eccezioni da "moru.exception". Utilizza questa competenza ogni volta che gli utenti desiderano: eseguire codice in sicurezza in ambienti isolati, creare agenti AI che eseguono codice non affidabile, creare pipeline di elaborazione dati, eseguire web scraper, configurare ambienti di sviluppo o qualsiasi automazione Python che coinvolga sandbox Moru. Si attiva su: "moru python", "from moru import", "Sandbox.create", "pip install moru", "moru sdk python", "python sandbox", "AsyncSandbox", "Volume.create", "Template.build" o qualsiasi codice Python che deve essere eseguito in ambienti cloud isolati. NON utilizzare questa abilità per il codice JavaScript/TypeScript: utilizza invece moru-javascript.
