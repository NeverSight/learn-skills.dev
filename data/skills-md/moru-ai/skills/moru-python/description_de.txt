Nutzen Sie diese Fähigkeit, wenn Sie Python-Code schreiben, der mit Moru-Cloud-Sandboxes interagiert. Dazu gehören: Erstellen von Sandboxen mit „Sandbox.create()“, Ausführen von Befehlen mit „sbx.commands.run()“, Lesen und Schreiben von Dateien mit „sbx.files.read()“ und „sbx.files.write()“, Arbeiten mit persistenten Volumes mithilfe der „Volume“-Klasse, Erstellen benutzerdefinierter Vorlagen mit dem „Template“-Builder, Behandeln von Hintergrundprozessen, Streamen von Befehlsausgaben und asynchrone Vorgänge mit „AsyncSandbox“. Der Skill deckt die komplette Python SDK-API ab, einschließlich der Fehlerbehandlung mit „TimeoutException“, „CommandExitException“, „AuthenticationException“ und anderen Ausnahmen von „moru.Exceptions“. Nutzen Sie diese Fähigkeit, wann immer Benutzer Code sicher in isolierten Umgebungen ausführen, KI-Agenten erstellen möchten, die nicht vertrauenswürdigen Code ausführen, Datenverarbeitungspipelines erstellen, Web Scraper ausführen, Entwicklungsumgebungen einrichten oder eine beliebige Python-Automatisierung mit Moru-Sandboxen durchführen möchten. Wird ausgelöst durch: „moru python“, „from moru import“, „Sandbox.create“, „pip install moru“, „moru sdk python“, „python sandbox“, „AsyncSandbox“, „Volume.create“, „Template.build“ oder jeden Python-Code, der in isolierten Cloud-Umgebungen ausgeführt werden muss. Verwenden Sie diesen Skill NICHT für JavaScript/TypeScript-Code – verwenden Sie stattdessen Moru-Javascript.
