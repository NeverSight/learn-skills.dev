このスキルは、Moru クラウド サンドボックスと対話する Python コードを作成するときに使用します。これには、`Sandbox.create()` によるサンドボックスの作成、`sbx.commands.run()` によるコマンドの実行、`sbx.files.read()` および `sbx.files.write()` によるファイルの読み書き、` Volume` クラスを使用した永続ボリュームの操作、`Template` ビルダーによるカスタム テンプレートの構築、バックグラウンド プロセスの処理、コマンド出力のストリーミング、および非同期操作が含まれます。 「非同期サンドボックス」。このスキルは、`TimeoutException`、`CommandExitException`、`AuthenticationException`、および `moru.Exception` のその他の例外によるエラー処理を含む、完全な Python SDK API をカバーします。ユーザーが、隔離された環境でコードを安全に実行する、信頼できないコードを実行する AI エージェントを構築する、データ処理パイプラインを作成する、Web スクレイパーを実行する、開発環境をセットアップする、または Moru サンドボックスを含む Python 自動化を行う場合は、このスキルをいつでも使用できます。トリガー対象: 「moru python」、「from moru import」、「Sandbox.create」、「pip install moru」、「moru sdk python」、「python Sandbox」、「AsyncSandbox」、「 Volume.create」、「Template.build」、または分離されたクラウド環境で実行する必要がある Python コード。 JavaScript/TypeScript コードにはこのスキルを使用しないでください。代わりに moru-javascript を使用してください。
