استخدم هذه المهارة عند كتابة كود Python الذي يتفاعل مع صناديق الحماية السحابية Moru. يتضمن ذلك: إنشاء صناديق حماية باستخدام `Sandbox.create()`، وتشغيل الأوامر باستخدام `sbx.commands.run()`، وقراءة الملفات وكتابتها باستخدام `sbx.files.read()` و`sbx.files.write()`، والعمل مع وحدات التخزين المستمرة باستخدام فئة `Volume`، وإنشاء قوالب مخصصة باستخدام منشئ `Template`، والتعامل مع العمليات الخلفية، وتدفق مخرجات الأوامر، والعمليات غير المتزامنة باستخدام ``AsyncSandbox``. تغطي المهارة واجهة برمجة تطبيقات Python SDK الكاملة، بما في ذلك معالجة الأخطاء باستخدام "TimeoutException"، و"CommandExitException"، و"AuthenticationException"، والاستثناءات الأخرى من "moru.exceptions". استخدم هذه المهارة عندما يريد المستخدمون: تنفيذ التعليمات البرمجية بأمان في بيئات معزولة، أو إنشاء وكلاء الذكاء الاصطناعي الذين يقومون بتشغيل تعليمات برمجية غير موثوقة، أو إنشاء خطوط أنابيب لمعالجة البيانات، أو تشغيل كاشطات الويب، أو إعداد بيئات التطوير، أو أي أتمتة Python تتضمن صناديق حماية Moru. المشغلات على: 'moru python'، أو 'from moru import'، أو 'Sandbox.create'، أو 'pip install moru'، أو 'moru sdk python'، أو 'python sandbox'، أو 'AsyncSandbox'، أو 'Volume.create'، أو 'Template.build'، أو أي كود Python يحتاج إلى التشغيل في بيئات سحابية معزولة. لا تستخدم هذه المهارة مع تعليمات JavaScript/TypeScript البرمجية - استخدم moru-javascript بدلاً من ذلك.
