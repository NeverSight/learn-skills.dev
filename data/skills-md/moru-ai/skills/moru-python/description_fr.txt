Utilisez cette compétence lors de l'écriture de code Python qui interagit avec les bacs à sable cloud Moru. Cela inclut : la création de bacs à sable avec `Sandbox.create()`, l'exécution de commandes avec `sbx.commands.run()`, la lecture et l'écriture de fichiers avec `sbx.files.read()` et `sbx.files.write()`, l'utilisation de volumes persistants à l'aide de la classe `Volume`, la création de modèles personnalisés avec le générateur `Template`, la gestion des processus en arrière-plan, la diffusion en continu de la sortie des commandes et les opérations asynchrones avec `AsyncSandbox`. La compétence couvre l'API complète du SDK Python, y compris la gestion des erreurs avec `TimeoutException`, `CommandExitException`, `AuthenticationException` et d'autres exceptions de `moru.exceptions`. Utilisez cette compétence chaque fois que les utilisateurs souhaitent : exécuter du code en toute sécurité dans des environnements isolés, créer des agents d'IA qui exécutent du code non fiable, créer des pipelines de traitement de données, exécuter des web scrapers, configurer des environnements de développement ou toute automatisation Python impliquant des bacs à sable Moru. Déclencheurs sur : 'moru python', 'from moru import', 'Sandbox.create', 'pip install moru', 'moru sdk python', 'python sandbox', 'AsyncSandbox', 'Volume.create', 'Template.build' ou tout code Python devant s'exécuter dans des environnements cloud isolés. N'utilisez PAS cette compétence pour le code JavaScript/TypeScript - utilisez plutôt moru-javascript.
