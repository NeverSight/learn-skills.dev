在编写与 Moru 云沙箱交互的 Python 代码时使用此技能。这包括：使用“Sandbox.create()”创建沙箱、使用“sbx.commands.run()”运行命令、使用“sbx.files.read()”和“sbx.files.write()”读取和写入文件、使用“Volume”类处理持久卷、使用“Template”构建器构建自定义模板、处理后台进程、流式传输命令输出以及使用“AsyncSandbox”进行异步操作。该技能涵盖了完整的 Python SDK API，包括“TimeoutException”、“CommandExitException”、“AuthenticationException”和“moru.exceptions”中的其他异常的错误处理。每当用户想要：在隔离环境中安全地执行代码、构建运行不受信任代码的 AI 代理、创建数据处理管道、运行 Web 抓取工具、设置开发环境或任何涉及 Moru 沙箱的 Python 自动化时，都可以使用此技能。触发：“moru python”、“from moru import”、“Sandbox.create”、“pip install moru”、“moru sdk python”、“python sandbox”、“AsyncSandbox”、“Volume.create”、“Template.build”或任何需要在隔离云环境中运行的 Python 代码。不要将此技能用于 JavaScript/TypeScript 代码 - 请改用 moru-javascript。
