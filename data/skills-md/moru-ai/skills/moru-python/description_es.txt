Utilice esta habilidad al escribir código Python que interactúe con los entornos sandbox de la nube de Moru. Esto incluye: crear entornos sandbox con `Sandbox.create()`, ejecutar comandos con `sbx.commands.run()`, leer y escribir archivos con `sbx.files.read()` y `sbx.files.write()`, trabajar con volúmenes persistentes usando la clase `Volume`, crear plantillas personalizadas con el constructor `Template`, manejar procesos en segundo plano, transmitir salida de comandos y operaciones asíncronas con `AsyncSandbox`. La habilidad cubre la API completa del SDK de Python, incluido el manejo de errores con `TimeoutException`, `CommandExitException`, `AuthenticationException` y otras excepciones de `moru.exceptions`. Utilice esta habilidad siempre que los usuarios quieran: ejecutar código de forma segura en entornos aislados, crear agentes de IA que ejecuten código que no sea de confianza, crear canales de procesamiento de datos, ejecutar web scrapers, configurar entornos de desarrollo o cualquier automatización de Python que involucre entornos sandbox de Moru. Activadores en: 'moru python', 'from moru import', 'Sandbox.create', 'pip install moru', 'moru sdk python', 'python sandbox', 'AsyncSandbox', 'Volume.create', 'Template.build' o cualquier código Python que deba ejecutarse en entornos de nube aislados. NO utilice esta habilidad para código JavaScript/TypeScript; utilice moru-javascript en su lugar.
