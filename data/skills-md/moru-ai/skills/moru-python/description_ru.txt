Используйте этот навык при написании кода Python, который взаимодействует с облачными песочницами Moru. Сюда входит: создание песочниц с помощью Sandbox.create(), запуск команд с помощью sbx.commands.run(), чтение и запись файлов с помощью sbx.files.read() и sbx.files.write(), работа с постоянными томами с использованием класса Volume, создание пользовательских шаблонов с помощью построителя Template, обработка фоновых процессов, потоковый вывод команд и асинхронные операции с AsyncSandbox. Навык охватывает весь API Python SDK, включая обработку ошибок с помощью TimeoutException, CommandExitException, AuthenticationException и других исключений из moru.Exceptions. Используйте этот навык всякий раз, когда пользователи хотят: безопасно выполнять код в изолированных средах, создавать агенты ИИ, выполняющие ненадежный код, создавать конвейеры обработки данных, запускать веб-скребки, настраивать среды разработки или любую автоматизацию Python с использованием песочниц Moru. Запускается по: «moru python», «from moru import», «Sandbox.create», «pip install moru», «moru sdk python», «песочница Python», «AsyncSandbox», «Volume.create», «Template.build» или любому коду Python, который необходимо запускать в изолированных облачных средах. НЕ используйте этот навык для кода JavaScript/TypeScript — вместо этого используйте moru-javascript.
