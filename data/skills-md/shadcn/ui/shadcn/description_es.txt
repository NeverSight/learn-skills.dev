Se aplica cuando se trabaja con shadcn/ui, registros de componentes, ajustes preestablecidos, códigos --preset o cualquier proyecto con un archivo componentes.json. También activa "shadcn init", "crear una aplicación con --preset" o "cambiar a --preset".
