Guía de implementación y migración de Effect v4 para las API de Effect actuales. Úselo al escribir o revisar código que depende de Effect v4, migrar API de estilo v3 (Context.Tag, Effect.Service, FiberRef, catchAll/catchSome, fork/forkDaemon), depurar cableado de servicios/capas o validar la compatibilidad con effect-smol/effect@4.0.0-beta.x.
