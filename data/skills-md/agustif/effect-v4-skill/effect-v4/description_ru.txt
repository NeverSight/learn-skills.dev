Руководство по реализации и миграции Effect v4 для текущих API эффектов. Используйте при написании или проверке кода, который зависит от Effect v4, миграции API-интерфейсов в стиле v3 (Context.Tag, Effect.Service, FiberRef, catchAll/catchSome, fork/forkDaemon), отладке подключения службы/уровня или проверке совместимости с effect-smol/effect@4.0.0-beta.x.
