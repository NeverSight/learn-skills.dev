Implementa il monitoraggio della telemetria dell'interfaccia utente utilizzando @nutanix-ui/pulse-telemetry. Utilizzare quando richiesto per: aggiungere telemetria/analisi/tracciamento a un'app Web, implementare requisiti di telemetria da un CSV/elenco, strumenti per clic/passaggi del mouse/visite di pagina, configurare TelemetryProvider o TelemetryManager, creare file di configurazione di telemetria o verificare i payload di telemetria. Copre le basi di codice React e non React.
