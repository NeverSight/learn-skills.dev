Внедрите отслеживание телеметрии пользовательского интерфейса с помощью @nutanix-ui/pulse-telemetry. Используйте, когда вас попросят: добавить телеметрию/аналитику/отслеживание в веб-приложение, реализовать требования к телеметрии из CSV/списка, измерить клики/наведения/посещения страниц, настроить TelemetryProvider или TelemetryManager, создать файлы конфигурации телеметрии или проверить полезные данные телеметрии. Охватывает кодовые базы React и не-React.
