Implémentez le suivi de télémétrie de l'interface utilisateur à l'aide de @nutanix-ui/pulse-telemetry. À utiliser lorsque vous y êtes invité pour : ajouter de la télémétrie/analyse/suivi à une application Web, mettre en œuvre des exigences de télémétrie à partir d'un CSV/liste, des clics/survols/visites de pages d'instruments, configurer TelemetryProvider ou TelemetryManager, créer des fichiers de configuration de télémétrie ou vérifier les charges utiles de télémétrie. Couvre les bases de code React et non-React.
