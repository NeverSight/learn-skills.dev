Инициализация мультиагентного репозитория в один клик из шаблона Антигравитация. Используйте этот навык, когда пользователи хотят быстро создать новый проект («быстрый» режим) или с настройками по умолчанию во время выполнения («полный» режим), включая профиль поставщика LLM, переключатель MCP, контекст предпочтений роя, тип песочницы и необязательную инициализацию git.
