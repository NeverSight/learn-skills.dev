Inizializzazione con un clic di un repository multi-agente dal modello Antigravity. Utilizza questa competenza quando gli utenti desiderano creare un nuovo progetto rapidamente (modalità "rapida") o con impostazioni predefinite di runtime (modalità "completa"), tra cui il profilo del provider LLM, l'attivazione/disattivazione MCP, il contesto delle preferenze swarm, il tipo sandbox e git init facoltativo.
