Inicialización con un clic de un repositorio multiagente desde la plantilla Antigravity. Utilice esta habilidad cuando los usuarios quieran desarrollar un nuevo proyecto rápidamente (modo "rápido") o con valores predeterminados de tiempo de ejecución (modo "completo"), incluido el perfil del proveedor LLM, alternancia de MCP, contexto de preferencia de enjambre, tipo de sandbox y git init opcional.
