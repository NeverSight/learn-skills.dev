Initialisation en un clic d'un référentiel multi-agents à partir du modèle Antigravity. Utilisez cette compétence lorsque les utilisateurs souhaitent créer un nouveau projet rapidement (mode « rapide ») ou avec les paramètres d'exécution par défaut (mode « complet »), notamment le profil du fournisseur LLM, la bascule MCP, le contexte de préférence d'essaim, le type de bac à sable et git init facultatif.
