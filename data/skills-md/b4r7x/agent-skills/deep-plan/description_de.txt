Nimmt eine grobe, unausgereifte Idee und setzt sie selbstständig in einen Umsetzungsplan um. Recherchiert das Projekt eingehend, stellt klärende Fragen, generiert eine präzise interne Eingabeaufforderung und führt diese dann aus, um einen strukturierten Plan mit Aufgaben zu erstellen. Entwickelt für den Planmodus. Verwenden Sie diese Option, wenn der Benutzer eine vage Funktionsanfrage, eine grobe Idee oder eine „schmutzige“ Aufforderung stellt und einen umsetzbaren Implementierungsplan wünscht – z. B. „planen Sie dies“, „tief planen“, „verwandeln Sie dies in einen Plan“, „zaplanuj to“, „zrób planen“.
