Genera mensajes de confirmación de git similares a los humanos basados ​​en cambios preparados o no preparados. Lee git diff, analiza qué cambió y genera 3 opciones de mensajes de confirmación naturales que suenan como si hubieran sido escritas por un desarrollador, no por IA. Esta habilidad debe usarse cuando el usuario desea un mensaje de confirmación, pregunta "qué debo escribir para la confirmación", "generar mensaje de confirmación", "compromiso humano", "wiadomość do commita" o simplemente pide ayuda para confirmar.
