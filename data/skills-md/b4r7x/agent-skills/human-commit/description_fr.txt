Génère des messages de validation git de type humain basés sur des modifications planifiées ou non. Lit git diff, analyse ce qui a changé et génère 3 options de message de validation naturelles qui semblent avoir été écrites par un développeur – et non par l'IA. Cette compétence doit être utilisée lorsque l'utilisateur souhaite un message de validation, demande "que dois-je écrire pour la validation", "générer un message de validation", "un commit humain", "wiadomość do commita", ou demande simplement de l'aide pour la validation.
