Переписывает README.md, чтобы удалить отбросы ИИ — модные словечки, общие вступительные слова, фальшивый энтузиазм и шаблонную структуру — заменив их прямым, честным, человечным текстом. Этот навык следует использовать, когда пользователь хочет очеловечить README, удалить шаблоны написания, созданные ИИ, сделать так, чтобы документация звучала менее так, как будто ее написал ChatGPT, или просит «исправить README», «очеловечить файл readme», «удалить ненужные ИИ», «сделать это человечным».
