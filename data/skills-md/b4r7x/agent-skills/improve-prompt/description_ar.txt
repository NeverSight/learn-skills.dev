يحول فكرة فورية تقريبية غير مصقولة إلى موجه ترميز الذكاء الاصطناعي الدقيق والمنظم. يبحث تلقائيًا عن سياق المشروع الحالي (المكدس، بنية الملف، الاتفاقيات، سجل git) قبل الإنشاء. يجب استخدام هذه المهارة عندما يقدم المستخدم فكرة سريعة غامضة أو "قذرة" ويطلب تحسينها أو تحسينها أو إعادة كتابتها - على سبيل المثال. "تحسين هذه المطالبة" أو "تحسين المطالبة" أو "موجه ulepszony" أو "موجه dopracuj" أو ببساطة يصف ما يريدون القيام به بعبارات تقريبية.
