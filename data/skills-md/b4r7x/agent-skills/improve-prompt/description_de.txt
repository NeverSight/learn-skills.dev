Verwandelt eine grobe, unpolierte Eingabeaufforderungsidee in eine präzise, ​​strukturierte KI-Codierungseingabeaufforderung. Recherchiert vor der Generierung automatisch den aktuellen Projektkontext (Stack, Dateistruktur, Konventionen, Git-Verlauf). Diese Fähigkeit sollte verwendet werden, wenn der Benutzer eine vage oder „schmutzige“ Idee für eine Eingabeaufforderung vorschlägt und darum bittet, diese zu verfeinern, zu verbessern oder neu zu formulieren – z. B. „diese Eingabeaufforderung verbessern“, „meine Eingabeaufforderung verfeinern“, „ulepszony-Eingabeaufforderung“, „dopracuj-Eingabeaufforderung“ oder beschreibt einfach grob, was getan werden soll.
