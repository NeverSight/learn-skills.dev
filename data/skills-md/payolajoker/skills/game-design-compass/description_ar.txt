مهارة تخطيط تصميم الألعاب الشاملة التي توحد سير عمل التصميم القائم على الأدوار والأنواع والأنظمة الأساسية في نظام واحد. يُستخدم عند تخطيط مفاهيم اللعبة، أو كتابة أقسام GDD أو تحسينها، أو تصميم الحلقات الأساسية، أو التقدم، أو التوازن، أو الاقتصاد، أو السرد، أو تدفق المستوى، أو UI/UX، أو تحقيق الدخل، أو تكييف قرارات التصميم للجوال، أو الكمبيوتر الشخصي، أو وحدة التحكم، أو الويب/HTML5، وVR/AR. مفيد بشكل خاص عندما يحتاج الطلب إلى مخرجات تصميم منظمة مع قوائم المراجعة ومؤشرات الأداء الرئيسية والمخاطر وعناصر التسليم الجاهزة للتنفيذ.
