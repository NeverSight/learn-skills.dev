Настройте рабочие деревья git с автоматическим копированием файлов env, связыванием общих служб, исправлением окружения, проверкой базового теста и установкой зависимостей. Используйте, когда пользователь хочет создать конфигурацию .wtsetup для своего проекта, установить функцию оболочки wt или управлять рабочими процессами рабочего дерева. Срабатывает при упоминании рабочих деревьев, .wtsetup или запросах на настройку рабочих деревьев для репозитория.
