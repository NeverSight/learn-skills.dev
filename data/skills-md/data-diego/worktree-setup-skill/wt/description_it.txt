Configura alberi di lavoro git con copia automatica dei file env, collegamento a servizi condivisi, applicazione di patch env, verifica dei test di base e installazione delle dipendenze. Da utilizzare quando l'utente desidera creare una configurazione .wtsetup per il proprio progetto, installare la funzione wt shell o gestire i flussi di lavoro dell'albero di lavoro. Si attiva in caso di menzioni di alberi di lavoro, .wtsetup o richieste di configurazione dell'impostazione dell'albero di lavoro per un repository.
