Configure árboles de trabajo de git con copia automática de archivos env, vinculación de servicios compartidos, parches env, verificación de pruebas de referencia e instalación de dependencias. Úselo cuando el usuario desee crear una configuración .wtsetup para su proyecto, instalar la función wt Shell o administrar los flujos de trabajo del árbol de trabajo. Se activa al mencionar árboles de trabajo, .wtsetup o solicitudes para configurar el árbol de trabajo para un repositorio.
