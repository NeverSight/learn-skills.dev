🚀 Super Development Planner — Collecte des exigences conversationnelles + Développement parallèle de l'équipe d'agents + Invocation du processus complet de l'outil MCP.

Moment déclencheur : l'utilisateur dit « Aidez-moi à développer », « Je souhaite créer un projet/site Web/application/système/outil/script/bot », « mode plan », « développement conversationnel », « générer un document de développement », « J'ai une idée à mettre en œuvre » ou toute demande décrivant les besoins en matière de logiciel/outil/système. Même les phrases vagues comme « Je veux construire quelque chose » doivent se déclencher.

Capacités de base :
(1) Guider les utilisateurs novices pour affiner les exigences grâce à un dialogue basé sur des options,
(2) Générer des documents de développement ultra-détaillés de niveau entreprise,
(3) Orchestrer l'équipe d'agents + outils MCP pour le développement parallèle,
(4) Processus complet basé sur des documents, tests de nœuds, validation automatique.
