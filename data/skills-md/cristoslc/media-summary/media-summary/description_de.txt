Lädt Audio-/Videomedien herunter und fasst sie zusammen – Podcasts, YouTube-Videos, Vorträge, Interviews, Vorträge und Konferenzpräsentationen. Speichert eine strukturierte Markdown-Zusammenfassung lokal, veröffentlicht sie als öffentlichen GitHub Gist und öffnet sie in der Standardanwendung des Systems. Wird verwendet, wenn der Benutzer eine URL zu Audio- oder Videoinhalten bereitstellt (Apple Podcasts, Spotify, YouTube, Konferenzaufzeichnungen usw.).
