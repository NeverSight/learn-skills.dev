Descarga y resume medios de audio/video: podcasts, videos de YouTube, charlas, entrevistas, conferencias y presentaciones de conferencias. Guarda un resumen de rebajas estructurado localmente, lo publica como un GitHub Gist público y lo abre en la aplicación predeterminada del sistema. Se utiliza cuando el usuario proporciona una URL a cualquier contenido de audio o vídeo (Apple Podcasts, Spotify, YouTube, grabaciones de conferencias, etc.).
