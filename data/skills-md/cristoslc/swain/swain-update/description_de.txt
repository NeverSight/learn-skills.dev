Aktualisieren Sie die Swain-Fähigkeiten auf die neueste Version. Verwenden Sie diese Option, wenn der Benutzer „Swain aktualisieren“, „Swain aktualisieren“ oder „Neuesten Swain abrufen“ sagt oder die Installation seiner Swain-Fähigkeiten aktualisieren möchte. Führt den Skills Package Manager (npx) mit einem Git-Clone-Fallback aus und ruft dann swain-config auf, um alle Governance-Änderungen abzugleichen.
