Aide contextuelle et intégration pour l'écosystème de compétences swain. À utiliser lorsque les utilisateurs posent des questions sur Swain : quelles compétences existent, comment les utiliser, quels artefacts sont disponibles, comment les flux de travail se connectent ou lorsqu'ils ont besoin d'une référence rapide. Également invoqué après swain-init pour orienter les nouveaux utilisateurs. Se déclenche sur : « comment puis-je... », « qu'est-ce que... », « aide », « que peut faire Swain », « montre-moi les commandes », « je suis confus », « par où commencer », toute question sur les compétences, les artefacts ou les flux de travail de Swain, et une fois l'intégration du projet terminée.
