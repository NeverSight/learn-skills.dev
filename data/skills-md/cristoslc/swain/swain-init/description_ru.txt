Разовая адаптация проекта для Swain. Переносит существующее содержимое CLAUDE.md в AGENTS.md (с шаблоном включения @AGENTS.md), устанавливает и инициализирует bd (бусинки) для отслеживания задач, очищает автоматически добавленное содержимое AGENTS.md bd и предлагает добавить правила управления Swain. Запустите один раз при внедрении swain в новый проект — используйте swain-doctor для текущих проверок работоспособности каждого сеанса.
