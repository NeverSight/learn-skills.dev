Richiama SEMPRE questa abilità all'INIZIO di ogni sessione prima di svolgere qualsiasi altro lavoro. Questa competenza garantisce che il progetto host disponga di regole di governance dell'agente (instradamento delle competenze, protocollo di pre-implementazione, convenzioni di tracciamento dei problemi) installate nel file di contesto. È idempotente: se le regole sono già presenti, esce silenziosamente. Senza questa abilità eseguita per prima, le altre abilità swain (swain-design, swain-do, swain-release) non saranno instradabili.
