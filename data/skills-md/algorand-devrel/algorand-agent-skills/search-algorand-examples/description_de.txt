Durchsuchen Sie GitHub nach Beispielen und Mustern für intelligente Verträge von Algorand. Verwenden Sie diese Option, wenn Sie Beispielimplementierungen bestimmter Muster (BoxMap, innere Transaktionen) suchen, die Nutzung von Vertragsfunktionen nachschlagen, Algorand-Repositorys ermitteln oder Dateien aus Algorandfoundation-Repositorys abrufen möchten. Zu den starken Auslösern gehören „Finden Sie ein Beispiel für …“, „Zeigen Sie mir, wie man BoxMap verwendet“, „Suchen Sie nach Beispielen für Abstimmungsverträge“, „holen Sie sich den Code aus Puya-ts-Beispielen“ und „finden Sie Algorand NFT-Repositories“.
