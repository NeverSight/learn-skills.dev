Найдите на GitHub примеры и шаблоны смарт-контрактов Algorand. Используйте при поиске примеров реализации конкретных шаблонов (BoxMap, внутренние транзакции), поиске использования функций контракта, обнаружении репозиториев Algorand или извлечении файлов из репозиториев algorandfoundation. Сильные триггеры включают «найти пример...», «покажите мне, как использовать BoxMap», «поиск примеров контрактов для голосования», «получить код из примеров puya-ts», «найти репозитории Algorand NFT».
