أوامر AlgoKit CLI لإنشاء تطبيقات Algorand واختبارها ونشرها. يُستخدم عند إنشاء العقود الذكية أو تجميعها، أو تشغيل مجموعات الاختبار، أو النشر على الشبكة المحلية، أو إدارة شبكة Algorand المحلية، أو فهم سير عمل التطوير. تتضمن المشغلات القوية "إنشاء العقد"، و"تشغيل الاختبارات"، و"بدء الشبكة المحلية"، و"النشر على الشبكة المحلية"، و"تشغيل مشروع algokit"، و"حالة الشبكة المحلية".
