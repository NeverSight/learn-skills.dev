AlgoKit CLI と生成された TypeScript クライアントを使用して、Algorand スマート コントラクトを展開し、操作します。コントラクトをローカルネット/テストネット/メインネットにデプロイする場合、コントラクト メソッドを呼び出す場合、グローバル/ローカル/ボックスの状態を読み取る場合、アプリケーションをオプトインまたは終了する場合、または対話スクリプトを作成する場合に使用します。強力なトリガーには、「コントラクトのデプロイ」、「インクリメント メソッドの呼び出し」、「コントラクトの状態の読み取り」、「アプリへのオプトイン」、「対話するスクリプトの作成」、「getTypedAppClientById」などがあります。
