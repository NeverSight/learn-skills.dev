Bootstraps produktionsreifer AlgoKit-Projekte für Algorand-dApps und Smart Contracts. Verwenden Sie es, wenn Sie neue Algorand-Smart-Contract-Projekte initialisieren, Entwicklungsumgebungen von Grund auf einrichten oder dApps mit vorkonfigurierten Tools gerüsten. Zu den starken Auslösern gehören „Neues Projekt erstellen“, „Neue Algorand-App initialisieren“, „Neuen Smart Contract starten“, „AlgoKit einrichten“, „Gerüst für eine dApp“ und „Algokit-Initialisierung“.
