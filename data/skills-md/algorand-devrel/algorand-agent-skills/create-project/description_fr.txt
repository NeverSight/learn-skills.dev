Démarre les projets AlgoKit prêts pour la production pour les dApps Algorand et les contrats intelligents. À utiliser lors de l'initialisation de nouveaux projets de contrats intelligents Algorand, de la configuration d'environnements de développement à partir de zéro ou de l'échafaudage de dApps avec des outils préconfigurés. Les déclencheurs puissants incluent « créer un nouveau projet », « initialiser une nouvelle application Algorand », « démarrer un nouveau contrat intelligent », « configurer AlgoKit », « échafauder une dApp », « algokit init ».
