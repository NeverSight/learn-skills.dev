Algorand dApps およびスマート コントラクト向けに、本番環境に対応した AlgoKit プロジェクトをブートストラップします。新しい Algorand スマート コントラクト プロジェクトを初期化する場合、開発環境を最初からセットアップする場合、または事前構成されたツールを使用して dApp をスキャフォールディングする場合に使用します。強力なトリガーには、「新しいプロジェクトの作成」、「新しい Algorand アプリの初期化」、「新しいスマート コントラクトの開始」、「AlgoKit のセットアップ」、「dApp のスキャフォールディング」、「algokit init」などがあります。
