Erstellen Sie intelligente Algorand-Verträge mit Algorand TypeScript (PuyaTs) oder Algorand Python (PuyaPy). Verwenden Sie es, wenn Sie neue Smart Contracts von Grund auf erstellen, Funktionen oder Methoden zu bestehenden Verträgen hinzufügen, Algorand-Vertragsentwicklungsmuster verstehen oder Anleitungen zur Vertragsarchitektur erhalten. Zu den starken Auslösern gehören „einen intelligenten Vertrag erstellen“, „einen Vertrag schreiben, der …“, „einen Abstimmungsvertrag erstellen“, „einen NFT-Vertrag implementieren“ und „dem Vertrag eine Methode hinzufügen“.
