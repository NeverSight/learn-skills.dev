Создавайте смарт-контракты Algorand, используя Algorand TypeScript (PuyaTs) или Algorand Python (PuyaPy). Используйте при создании новых смарт-контрактов с нуля, добавлении функций или методов к существующим контрактам, понимании шаблонов разработки контрактов Algorand или получении рекомендаций по архитектуре контрактов. Сильные триггеры включают «создать смарт-контракт», «написать контракт, который…», «создать контракт для голосования», «реализовать контракт NFT», «добавить метод в контракт».
