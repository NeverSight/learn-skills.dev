أنشئ عقود Algorand الذكية باستخدام Algorand TypeScript (PuyaTs) أو Algorand Python (PuyaPy). يُستخدم عند إنشاء عقود ذكية جديدة من البداية، أو إضافة ميزات أو أساليب إلى العقود الحالية، أو فهم أنماط تطوير عقود Algorand، أو الحصول على إرشادات حول بنية العقود. تتضمن المحفزات القوية "إنشاء عقد ذكي"، و"كتابة عقد..."، و"إنشاء عقد تصويت"، و"تنفيذ عقد NFT"، و"إضافة طريقة إلى العقد".
