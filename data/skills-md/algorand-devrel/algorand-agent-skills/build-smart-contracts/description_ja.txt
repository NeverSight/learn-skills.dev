Algorand TypeScript (PuyaTs) または Algorand Python (PuyaPy) を使用して Algorand スマート コントラクトを構築します。新しいスマート コントラクトを最初から作成する場合、既存のコントラクトに機能やメソッドを追加する場合、Algorand コントラクト開発パターンを理解する場合、またはコントラクト アーキテクチャに関するガイダンスを取得する場合に使用します。強力なトリガーには、「スマート コントラクトを作成する」、「...というコントラクトを作成する」、「投票コントラクトを構築する」、「NFT コントラクトを実装する」、「コントラクトにメソッドを追加する」などがあります。
