Créez des interfaces React pour les dApps Algorand avec intégration de portefeuille. À utiliser lors de la création d'interfaces React qui interagissent avec des contrats intelligents, de la configuration de connexions de portefeuille (Pera, Defly, Exodus), de l'intégration de clients d'application typés avec des signataires de portefeuille ou de la création d'interfaces utilisateur dApp qui appellent des méthodes de contrat. Les déclencheurs puissants incluent « créer une interface pour mon contrat », « ajouter une connexion de portefeuille à mon application React », « comment appeler mon contrat depuis l'interface ? », « configurer un portefeuille d'utilisation avec mon client saisi », « connecter le portefeuille Pera à ma dApp », « algorand.setSigner ».
