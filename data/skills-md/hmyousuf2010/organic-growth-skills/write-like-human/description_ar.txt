قم بتحويل النص الذي تم إنشاؤه بواسطة الذكاء الاصطناعي إلى كتابة بشرية أصيلة أو اكتب محتوى جديدًا لا يمكن تمييزه عن الكتابة البشرية. استخدم هذه المهارة عندما يطلب المستخدم كتابة أي محتوى نصي أو إعادة كتابته أو إضفاء الطابع الإنساني عليه أو تحريره - التغريدات أو رسائل البريد الإلكتروني أو السير أو الملفات التمهيدية أو منشورات المدونة أو الوثائق أو النسخ أو أي مخرجات مكتوبة. يُستخدم أيضًا عندما يذكر المستخدم اكتشاف الذكاء الاصطناعي، أو إضفاء الطابع الإنساني على النص، أو جعل النص يبدو طبيعيًا، أو تجنب أنماط الذكاء الاصطناعي.
