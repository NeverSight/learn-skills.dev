Analyse de données exploratoire défensive pour les données scientifiques (CSV, FASTA, etc.). Orienté d'abord vers le contexte, guidé par l'humain ; un tracé à la fois, demandez pourquoi avant d'exécuter, ajout uniquement d'un journal par session, de scripts avec PEP723 et uv run, de tracés WebP. À utiliser lors de l'ouverture de fichiers de données pour EDA ou lorsque l'utilisateur souhaite une exploration guidée des données scientifiques.
