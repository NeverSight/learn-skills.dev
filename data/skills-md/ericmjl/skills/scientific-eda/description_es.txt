Análisis de datos exploratorios defensivos para datos científicos (CSV, FASTA, etc.). Primero el contexto, guiado por el ser humano; un gráfico a la vez, pregunte por qué antes de ejecutar, agregar solo diario por sesión, scripts con PEP723 y ejecución uv, gráficos WebP. Úselo al abrir archivos de datos para EDA o cuando el usuario desee una exploración de datos científicos guiada.
