Используйте интерфейс командной строки GitHub (gh) для распространенных операций, таких как создание запросов на поставку, просмотр журналов действий GitHub, управление проблемами, просмотр заявок на участие и т. д. Используйте это, когда вам нужно взаимодействовать с репозиториями GitHub непосредственно из командной строки.
