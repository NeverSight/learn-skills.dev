تدريب وضبط نماذج الرؤية لاكتشاف الكائنات (D-FINE، وRT-DETR v2، وDETR، وYOLOS)، وتصنيف الصور (نماذج timm - MobileNetV3، وMobileViT، وResNet، وViT/DINOv3 - بالإضافة إلى أي مصنف Transformers)، وتجزئة SAM/SAM2 باستخدام Hugging Face Transformers على وحدات معالجة الرسوميات السحابية Hugging Face Jobs. يغطي إعداد مجموعة البيانات بتنسيق COCO، وزيادة الألبومات، وتقييم mAP/mAR، ومقاييس الدقة، وتجزئة SAM مع مطالبات bbox/point، وفقدان DiceCE، واختيار الأجهزة، وتقدير التكلفة، ومراقبة Trackio، واستمرارية Hub. يُستخدم عندما يذكر المستخدمون التدريب على اكتشاف الكائنات، أو تصنيف الصور، أو SAM، أو SAM2، أو التجزئة، أو تلميع الصور، أو DETR، أو D-FINE، أو RT-DETR، أو ViT، أو timm، أو MobileNet، أو ResNet، أو نماذج المربع المحيط، أو نماذج الرؤية الدقيقة على Hugging Face Jobs.
