Cree, enumere y administre recordatorios de macOS a través de AppleScript. Úselo cuando el usuario solicite crear un recordatorio, agregar una tarea pendiente, realizar una tarea, establecer un recordatorio para algo o cualquier cosa que involucre Apple Reminders en macOS. Activadores de solicitudes como "recuérdame comprar leche", "agregar una tarea para llamar al dentista", "crear un recordatorio para el viernes", "agregar a mi lista de compras", "marcar esto como importante". Solo MacOS.
