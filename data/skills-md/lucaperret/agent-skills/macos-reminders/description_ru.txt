Создавайте, просматривайте и управляйте напоминаниями macOS с помощью AppleScript. Используйте, когда пользователь просит создать напоминание, добавить задачу, создать задачу, установить напоминание о чем-либо или что-либо, связанное с Apple Reminders в macOS. Запускается по таким запросам, как «напомни мне купить молоко», «добавить задачу позвонить стоматологу», «создать напоминание на пятницу», «добавить в список покупок», «отметить как важное». только для MacOS.
