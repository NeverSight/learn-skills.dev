Erstellen, listen und verwalten Sie macOS-Erinnerungen über AppleScript. Verwenden Sie diese Option, wenn der Benutzer eine Erinnerung erstellen, eine Aufgabe hinzufügen, eine Aufgabe erstellen, eine Erinnerung für etwas festlegen oder etwas mit Apple-Erinnerungen unter macOS tun möchte. Wird bei Anfragen wie „Erinnere mich daran, Milch zu kaufen“, „Füge eine Aufgabe hinzu, um den Zahnarzt anzurufen“, „Erstelle eine Erinnerung für Freitag“, „Zu meiner Einkaufsliste hinzufügen“ und „Markiere dies als wichtig“ ausgelöst. Nur macOS.
