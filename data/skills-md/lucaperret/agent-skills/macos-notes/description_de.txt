Erstellen, lesen, durchsuchen und verwalten Sie macOS-Notizen über AppleScript. Verwenden Sie diese Option, wenn der Benutzer eine Notiz machen, etwas aufschreiben, eine Idee speichern, Besprechungsnotizen erstellen, eine Notiz lesen, Notizen durchsuchen oder alles, was mit Apple Notes auf macOS zu tun hat, auffordert. Wird bei Anfragen wie „Notieren Sie sich dies“, „Speichern Sie dies als Notiz“, „Erstellen Sie eine Notiz zu X“, „Zeige meine Notizen“, „Suche meine Notizen nach X“, „Was habe ich über X geschrieben“ ausgelöst. Nur macOS.
