Создавайте, читайте, ищите и управляйте заметками macOS с помощью AppleScript. Используйте, когда пользователь просит сделать заметку, что-то записать, сохранить идею, создать заметки о встрече, прочитать заметку, выполнить поиск в заметках или что-либо еще, связанное с Apple Notes на macOS. Запускается по таким запросам, как «запишите это», «сохраните это как заметку», «создайте заметку о X», «покажите мои заметки», «найдите в моих заметках X», «что я написал о X». только для MacOS.
