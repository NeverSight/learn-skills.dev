Поиск в Интернете и получение веб-контента. Используйте этот навык, когда пользователям нужно искать информацию в Интернете, получать веб-контент, находить данные в реальном времени или выполнять поиск в Интернете. Поддерживает несколько инструментов поиска: WebFetch, навык Firecrawl и навык Tavily.
