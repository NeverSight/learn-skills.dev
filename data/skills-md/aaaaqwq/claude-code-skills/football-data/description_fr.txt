Données de football sur 13 ligues : classements, horaires, statistiques de match, xG, transferts, profils de joueurs. Zéro configuration, pas de clés API. Couvre la Premier League, La Liga, Bundesliga, Serie A, Ligue 1, MLS, Ligue des Champions, Coupe du Monde, Championnat, Eredivisie, Primeira Liga, Serie A Brésil, Championnat d'Europe.

À utiliser lorsque : l'utilisateur pose des questions sur les classements de football, les calendriers, les statistiques de match, le xG, les compositions, les valeurs des joueurs, les transferts, les actualités sur les blessures, les classements, les rencontres quotidiennes ou les profils des joueurs.
Ne pas utiliser lorsque : l'utilisateur pose des questions sur le football américain/NFL (utiliser les données nfl), le football universitaire (utiliser les données cfb), la NBA (utiliser les données nba), la WNBA (utiliser les données wnba), le basket-ball universitaire (utiliser les données cbb), la LNH (utiliser les données nhl), la MLB (utiliser les données mlb), le tennis (utiliser les données tennis), le golf (utiliser les données golf), la Formule 1 (utiliser fastf1) ou les cotes des paris (utiliser polymarket ou kalshi). Ne pas utiliser pour les scores en direct/en temps réel : mises à jour des données après le match. N'utilisez pas get_season_leaders ou get_missing_players pour les ligues non-Premier League (elles reviennent vides). N'utilisez pas get_event_xg pour les ligues en dehors du top 5 (EPL, La Liga, Bundesliga, Serie A, Ligue 1).
