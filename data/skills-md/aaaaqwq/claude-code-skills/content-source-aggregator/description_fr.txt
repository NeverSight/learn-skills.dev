Récolte de sujets d’actualité d’une source d’information unifiée. Contenu tendance gratuit provenant de six plates-formes (X/Twitter, YouTube, Bilibili, GitHub, Reddit, LinuxDo) fourni sous la forme d'un pool de sujets d'actualité standardisé pour les pipelines de création de contenu. Le tout via des API publiques gratuites, aucun paiement requis.
