آلة حاسبة مالية متقدمة تحتوي على جداول القيمة المستقبلية والقيمة الحالية وحسابات الخصم والتسعير الإضافي والفائدة المركبة. يُستخدم عند حساب نمو الاستثمار، أو استراتيجيات التسعير، أو قيم القروض، أو الخصومات، أو مقارنة السيناريوهات المالية عبر معدلات وفترات زمنية مختلفة. يتضمن كلاً من واجهة سطر الأوامر (CLI) وواجهة مستخدم الويب التفاعلية.
