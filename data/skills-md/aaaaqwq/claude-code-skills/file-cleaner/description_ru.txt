Инструмент для очистки системных файлов. Сканируйте и идентифицируйте большие файлы и ненужные файлы (временные файлы, кеши, журналы, резервные копии и т. д.), а также предоставляйте пользователям интерактивный интерфейс очистки, который они могут удалить. Этот навык используется, когда пользователям необходимо очистить дисковое пространство, упорядочить системные файлы, найти большие файлы, удалить ненужные файлы или освободить место для хранения.
