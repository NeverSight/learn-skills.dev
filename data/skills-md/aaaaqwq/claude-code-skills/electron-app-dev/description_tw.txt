Electron桌面應用開發專家。精通electron-vite、TypeScript、React、IPC通信、窗口管理、原生功能集成等Electron全栈开发技术。

適用場景：
- 建立electron-vite + TypeScript + React項目
- 實現安全的IPC通信
- 窗口管理（创建、控制、多窗口、状态持久化）
- 原生功能（系统托盘、菜单、通知、文件对话框）
- electron-builder打包分發
- 自動更新和程式碼簽名

所有程式碼遵循最新Electron安全最佳實務：contextIsolation開啟、nodeIntegration關閉、sandbox模式開啟、contextBridge安全暴露IPC。
