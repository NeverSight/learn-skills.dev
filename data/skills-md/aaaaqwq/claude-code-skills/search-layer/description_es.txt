Herramienta de búsqueda PREDETERMINADA para TODAS las necesidades de búsqueda. Capa de deduplicación y búsqueda de múltiples fuentes con puntuación basada en la intención. Integra Brave Search (web_search), Exa, Tavily y Grok para proporcionar resultados de alta cobertura y calidad. Clasifica automáticamente la intención de la consulta y ajusta la estrategia de búsqueda, las ponderaciones de puntuación y la síntesis de resultados. Úselo para CUALQUIER consulta que requiera una búsqueda en la web: búsquedas objetivas, investigaciones, noticias, comparaciones, búsqueda de recursos, "qué es X", verificaciones de estado, etc. NO use web_search sin formato directamente; Siempre pasa por esta habilidad.
