Инструмент поиска ПО УМОЛЧАНИЮ для ВСЕХ потребностей поиска/поиска. Уровень поиска и дедупликации из нескольких источников с оценкой с учетом намерений. Интегрирует Brave Search (web_search), Exa, Tavily и Grok для обеспечения высокого охвата и высокого качества результатов. Автоматически классифицирует намерение запроса и корректирует стратегию поиска, весовые коэффициенты и синтез результатов. Используйте для ЛЮБОГО запроса, требующего веб-поиска — фактических поисков, исследований, новостей, сравнений, поиска ресурсов, «что такое X», проверки статуса и т. д. НЕ используйте необработанный веб-поиск напрямую; всегда используйте этот навык.
