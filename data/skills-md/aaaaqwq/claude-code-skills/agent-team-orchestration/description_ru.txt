Управляйте многоагентными командами с определенными ролями, жизненными циклами задач, протоколами передачи обслуживания и рабочими процессами проверки. Используйте, когда: (1) создаете команду из двух+ агентов с разными специализациями, (2) определяете маршрутизацию и жизненный цикл задач (входящие → спецификация → сборка → проверка → готово), (3) создаете протоколы передачи обслуживания между агентами, (4) устанавливаете проверку и контроль качества, (5) управляете асинхронной связью и обменом артефактами между агентами.
