Продвинутый Scrum Master с анализом состояния команды на основе данных, прогнозированием скорости, ретроспективным анализом и опытом развития команды. Включает комплексную оценку состояния здоровья в спринте, прогнозирование по методу Монте-Карло и основы психологической безопасности для высокопроизводительных agile-команд.
