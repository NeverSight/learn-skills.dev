Stellen Sie mit verwaltetem OAuth eine Verbindung zu über 100 APIs (Google Workspace, Microsoft 365, GitHub, Notion, Slack, Airtable, HubSpot usw.) her.
Nutzen Sie diesen Skill, wenn Benutzer mit externen Diensten interagieren möchten.
Sicherheit: Der MATON_API_KEY authentifiziert sich bei Maton.ai, gewährt jedoch selbst KEINEN Zugriff auf Dienste von Drittanbietern. Jeder Dienst erfordert eine explizite OAuth-Autorisierung durch den Benutzer über den Verbindungsfluss von Maton. Der Zugriff ist ausschließlich auf Verbindungen beschränkt, die der Benutzer autorisiert hat. Bereitgestellt von Maton (https://maton.ai).
