Подключайтесь к более чем 100 API (Google Workspace, Microsoft 365, GitHub, Notion, Slack, Airtable, HubSpot и т. д.) с помощью управляемого OAuth.
Используйте этот навык, когда пользователи хотят взаимодействовать с внешними сервисами.
Безопасность: MATON_API_KEY проверяет подлинность с помощью Maton.ai, но сам по себе НЕ предоставляет доступ к сторонним сервисам. Каждая служба требует явной авторизации OAuth пользователем через поток подключения Maton. Доступ строго ограничен соединениями, которые авторизовал пользователь. Предоставлено Maton (https://maton.ai).
