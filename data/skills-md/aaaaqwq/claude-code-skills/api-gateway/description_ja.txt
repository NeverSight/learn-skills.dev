マネージド OAuth を使用して 100 以上の API (Google Workspace、Microsoft 365、GitHub、Notion、Slack、Airtable、HubSpot など) に接続します。
ユーザーが外部サービスと対話する場合は、このスキルを使用します。
セキュリティ: MATON_API_KEY は Maton.ai で認証しますが、それ自体ではサードパーティのサービスへのアクセスを許可しません。各サービスには、Maton の接続フローを通じてユーザーによる明示的な OAuth 承認が必要です。アクセスは、ユーザーが許可した接続に厳密に限定されます。まとん（https://maton.ai）より提供。
