Recherchez sur les sites Web officiels comme GitHub et SkillsMP des compétences de haute qualité correspondant à la description de l'utilisateur, présentez-les pour sélection, puis clonez et installez automatiquement la compétence choisie dans le répertoire global ~/.claude/skills/. Déclenchez cette compétence chaque fois que l'utilisateur a besoin de rechercher ou d'installer une nouvelle compétence.
