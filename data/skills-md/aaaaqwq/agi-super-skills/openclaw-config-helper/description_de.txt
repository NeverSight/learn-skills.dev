OpenClaw-Konfigurationsmodifikator. Bevor Sie eine OpenClaw-Konfiguration ändern, müssen Sie die offizielle Dokumentation konsultieren, um eine korrekte Formatierung sicherzustellen und Systemabstürze oder Funktionsstörungen zu vermeiden. Erzwingen Sie den Prozess: Schema prüfen → Dokumentation prüfen → Bestätigen → Ändern.
