Herramienta de sincronización con un clic de Claude Code Environment. Sincronice todas las configuraciones desde un repositorio de GitHub con su máquina local: estilos de salida, mensajes globales CLAUDE.md, configuraciones del servidor MCP. Ideal para unificar entornos en múltiples dispositivos, recuperarse después de cambiar de computadora y compartir configuraciones dentro de los equipos. Utilice esta habilidad cuando necesite sincronizar las configuraciones del entorno de Claude Code desde un repositorio de GitHub.
