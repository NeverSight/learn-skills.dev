Outil de synchronisation en un clic de Claude Code Environment. Synchronisez toutes les configurations d'un référentiel GitHub vers local : styles de sortie, invites globales CLAUDE.md, paramètres du serveur MCP. Idéal pour unifier les environnements sur plusieurs appareils, restaurer après avoir changé d'ordinateur et partager des configurations au sein des équipes. Utilisez cette compétence lorsque les utilisateurs doivent synchroniser les configurations de l'environnement Claude Code à partir d'un référentiel GitHub.
