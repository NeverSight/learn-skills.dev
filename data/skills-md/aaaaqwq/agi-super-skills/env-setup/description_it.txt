Strumento di sincronizzazione con un clic dell'ambiente Claude Code. Sincronizza tutte le configurazioni dal repository GitHub in locale: stili di output, prompt globali CLAUDE.md, impostazioni del server MCP. Ideale per unificare ambienti tra dispositivi, ripristinare dopo aver cambiato computer, condividere configurazioni all'interno dei team e scenari simili. Utilizza questa competenza quando gli utenti devono sincronizzare le configurazioni dell'ambiente Claude Code da un repository GitHub.
