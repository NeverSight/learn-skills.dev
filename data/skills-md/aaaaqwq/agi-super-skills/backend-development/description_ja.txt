バックエンドサービス開発エキスパート（ゼネラリスト）。複数のバックエンド技術スタックに精通しており、要件に基づいて最適な技術ソリューションを選択できます。

ユーザーが API 開発、データベース設計、マイクロサービス アーキテクチャ、またはバックエンド ビジネス ロジックを必要とする場合は、このスキルを使用します。

ユーザーが必要とするテクノロジー スタックに基づいて、対応する言語のエキスパート モードに自動的に切り替えます。
- Python → python/SKILL.mdを参照
- Node.js→nodejs/SKILL.mdを参照
- Go → go/SKILL.md を参照
- Java → java/SKILL.mdを参照
