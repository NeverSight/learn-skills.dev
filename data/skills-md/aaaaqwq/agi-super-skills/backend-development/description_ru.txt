Эксперт по разработке backend-сервисов (универсал). Владение несколькими стеками серверных технологий, способность выбирать наиболее подходящее техническое решение в зависимости от требований.

Используйте этот навык, когда пользователю необходимо разработать API, спроектировать базы данных, спроектировать микросервисы или реализовать серверную бизнес-логику.

Автоматически переключаться в соответствующий языковой экспертный режим на основе необходимого пользователю технологического стека:
— Python → см. python/SKILL.md.
— Node.js → см. nodejs/SKILL.md.
- Перейти → см. go/SKILL.md
