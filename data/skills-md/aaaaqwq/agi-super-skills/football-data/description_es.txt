Datos de fútbol en 13 ligas: clasificaciones, horarios, estadísticas de partidos, xG, transferencias, perfiles de jugadores. Configuración cero, sin claves API. Cubre Premier League, La Liga, Bundesliga, Serie A, Ligue 1, MLS, Champions League, Copa del Mundo, Campeonato, Eredivisie, Primeira Liga, Serie A Brasil, Campeonato de Europa.

Úselo cuando: el usuario pregunta sobre clasificaciones de fútbol, ​​partidos, estadísticas de partidos, xG, alineaciones, valores de jugadores, transferencias, noticias sobre lesiones, tablas de clasificación, partidos diarios o perfiles de jugadores.
No lo use cuando: el usuario pregunta sobre fútbol americano/NFL (use datos de nfl), fútbol americano universitario (use datos de cfb), NBA (use datos de nba), WNBA (use datos de wnba), baloncesto universitario (use datos de cbb), NHL (use datos de nhl), MLB (use datos de mlb), tenis (use datos de tenis), golf (use datos de golf), Fórmula 1 (use fastf1) o probabilidades de apuestas (use polimercado o kalshi). No lo utilices para puntuaciones en vivo o en tiempo real: los datos se actualizan después del partido. No utilices get_season_leaders o get_missing_players para ligas que no sean de la Premier League (devuelven vacíos). No utilices get_event_xg para ligas fuera del top 5 (EPL, La Liga, Bundesliga, Serie A, Ligue 1).
