13 个联赛的足球数据 — 积分榜、赛程表、比赛统计数据、预期进球、转会、球员资料。零配置，无 API 密钥。涵盖英超、西甲、德甲、意甲、法甲、美国职业足球大联盟、欧冠、世界杯、英冠、荷甲、英超、巴西甲级联赛、欧洲杯。

使用场合：用户询问足球排名、赛程、比赛统计数据、预期进球、阵容、球员价值、转会、伤病新闻、联赛表、每日赛程或球员档案。
不要在以下情况下使用：用户询问美式橄榄球/NFL（使用 nfl-data）、大学橄榄球（使用 cfb-data）、NBA（使用 nba-data）、WNBA（使用 wnba-data）、大学篮球（使用 cbb-data）、NHL（使用 nhl-data）、MLB（使用 mlb-data）、网球（使用网球数据）、高尔夫（使用 Golf-data）、一级方程式（使用 fastf1）或投注赔率（使用综合市场或卡尔希）。请勿用于实时/实时比分——赛后数据更新。不要对非英超联赛使用 get_season_leaders 或 get_missing_players （它们返回空）。不要将 get_event_xg 用于前 5 名之外的联赛（EPL、西甲、德甲、意甲、法甲）。
