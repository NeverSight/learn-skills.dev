Administre la configuración de permisos de herramientas globales de Claude Code, agregando automáticamente comandos MCP u otras herramientas a las herramientas permitidas para evitar la aprobación manual cada vez. Flujo de trabajo: confirme los comandos que el usuario desea agregar -> confirme el nivel de adición (global predeterminado ~/.claude.json) -> realice la adición -> verifique y recuerde reiniciar.
