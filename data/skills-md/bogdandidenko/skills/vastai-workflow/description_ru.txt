Запустите общий жизненный цикл API Vast.ai от поиска предложений до демонтажа с проверками безопасности и воспроизводимыми этапами запроса. Используйте, когда пользователям необходимо составить список/фильтровать предложения, создать экземпляры, прикрепить SSH-ключи, проверить готовность, остановить/уничтожить экземпляры или проверить выставление счетов/использование, а также когда необходимые поля времени выполнения (изображение, тип экземпляра, источник ключей API) должны быть собраны в диалоговом окне с предложениями по умолчанию.
