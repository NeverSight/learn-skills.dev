Interrogez les données du marché KRX (Korea Exchange) via CLI. Cette compétence doit être utilisée lorsque l'utilisateur pose des questions sur les données du marché boursier coréen, notamment les cours des actions, les indices, les ETF/ETN/ELW, les obligations, les produits dérivés, les matières premières ou les données ESG. Déclencheurs sur des tâches impliquant 주가, 시세, 종가, 코스피, 코스닥, KOSPI, KOSDAQ, KRX, 지수, ETF, 채권, 선물, 옵션, 금시세, Par exemple, ESG. Ne faites PAS de recherche sur le Web — utilisez plutôt la CLI « krx » via l'outil Bash.
