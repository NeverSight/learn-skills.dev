Fragen Sie KRX-Marktdaten (Korea Exchange) über CLI ab. Diese Fähigkeit sollte verwendet werden, wenn der Benutzer nach Daten zum koreanischen Aktienmarkt fragt, einschließlich Aktienkursen, Indizes, ETF/ETN/ELW, Anleihen, Derivaten, Rohstoffen oder ESG-Daten. Löst auf Aufgaben aus, die 주가, 세세, 종가, 코스피, 코스닥, KOSPI, KOSDAQ, KRX, 지수, ETF, 채권, 선물, 옵션, 코스피, 배출권, ESG. Führen Sie KEINE Websuche durch, sondern verwenden Sie stattdessen die „krx“-CLI über das Bash-Tool.
