Запросите рыночные данные KRX (Корейская биржа) через CLI. Этот навык следует использовать, когда пользователь задает вопросы о данных корейского фондового рынка, включая цены на акции, индексы, ETF/ETN/ELW, облигации, деривативы, сырьевые товары или данные ESG. Запускается по задачам, включающим 주가, 시세, 종가, 코스피, 코스닥, KOSPI, KOSDAQ, KRX, 지수, ETF, 채권, 선물, 옵션, 금시세, Например, ESG. НЕ используйте поиск в Интернете — вместо этого используйте CLI `krx` через инструмент Bash.
