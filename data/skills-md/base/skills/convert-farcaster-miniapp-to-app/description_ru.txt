Преобразует проекты Farcaster miniapp SDK в обычные базовые/веб-приложения. Начинается с интерактивного теста, позволяющего выбрать между преобразованием обычного приложения по умолчанию и узко изолированной поверхностью Farcaster, когда что-то действительно должно оставаться отдельным. Управляет коннекторами wagmi, поставщиками, аутентификацией, действиями SDK, маршрутами манифеста, метатегами, зависимостями и сохранением только для чтения.
