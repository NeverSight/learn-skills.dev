Intégrez les codes de construction de base (ERC-8021) dans les applications Web3 pour l'attribution de transactions en chaîne et le gain de frais de référence. À utiliser lorsqu'un projet doit ajouter un code de constructeur ou un dataSuffix aux transactions sur Base L2, que ce soit en utilisant Wagmi, Viem, Privy, ethers.js ou raw window.ethereum. Couvre des expressions telles que « ajouter des codes constructeur », « intégrer les codes constructeur », « gagner des frais de référence sur les transactions de base », « ajouter un code constructeur à mes transactions », « attribution de transaction », « intégration du code constructeur » ou « attribuer des transactions à mon application ». Gère l'analyse du projet pour détecter les frameworks, localiser les sites d'appels de transactions et les remplacer par des versions attribuées.
