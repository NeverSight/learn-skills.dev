قم بدمج رموز البناء الأساسية (ERC-8021) في تطبيقات web3 لإسناد المعاملات على السلسلة وكسب رسوم الإحالة. يُستخدم عندما يحتاج المشروع إلى إلحاق رمز الإنشاء أو dataSuffix بالمعاملات على Base L2، سواء باستخدام Wagmi أو Viem أو Privy أو ethers.js أو window.ethereum الخام. يغطي عبارات مثل "إضافة رموز المنشئ"، أو "دمج رموز المنشئ"، أو "كسب رسوم الإحالة على المعاملات الأساسية"، أو "إلحاق رمز المنشئ بمعاملاتي"، أو "إسناد المعاملة"، أو "تكامل رمز المنشئ"، أو "إسناد المعاملات إلى تطبيقي". يتعامل مع تحليل المشروع لاكتشاف الأطر وتحديد مواقع استدعاء المعاملات واستبدالها بالإصدارات المنسوبة.
