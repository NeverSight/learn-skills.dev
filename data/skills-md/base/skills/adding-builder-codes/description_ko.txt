온체인 거래 속성 및 추천 수수료 수익을 위해 Base Builder 코드(ERC-8021)를 web3 애플리케이션에 통합합니다. 프로젝트가 Wagmi, Viem, Privy, ethers.js 또는 원시 window.ethereum을 사용하는지 여부에 관계없이 Base L2의 트랜잭션에 빌더 코드 또는 dataSuffix를 추가해야 할 때 사용합니다. "빌더 코드 추가", "빌더 코드 통합", "기본 거래에서 추천 수수료 받기", "내 거래에 빌더 코드 추가", "거래 속성", "빌더 코드 통합" 또는 "내 앱에 트랜잭션 속성 지정"과 같은 문구가 포함됩니다. 프레임워크를 감지하고, 트랜잭션 호출 사이트를 찾고, 이를 속성 버전으로 대체하기 위한 프로젝트 분석을 처리합니다.
