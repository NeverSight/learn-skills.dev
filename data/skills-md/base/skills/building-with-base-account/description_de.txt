Integriert das Base Account SDK für Authentifizierung und Zahlungen. Deckt die Anmeldung mit Base (SIWB), Base Pay, Paymasters, Unterkonten, Ausgabeberechtigungen, Prolinks und Batch-Transaktionen ab. Verwendung beim Erstellen von Apps mit Wallet-Authentifizierung, USDC-Zahlungen, gesponserten Transaktionen, Smart-Wallet-Funktionen, wiederkehrenden Abonnements, gemeinsam nutzbaren Zahlungslinks oder jeder On-Chain-Interaktion auf Base. Umfasst Ausdrücke wie „Anmeldung mit Base hinzufügen“, „SIWB-Schaltfläche“, „USDC-Zahlungen akzeptieren“, „Base Pay“, „Paymaster-Einrichtung“, „Gassponsoring“, „Smart Wallet“, „Unterkonto“, „Ausgabeberechtigungen“ oder „Zahlungslink“.
