Copilot-CLI-Variante. Orchestrieren Sie ein spezialisiertes Softwareentwicklungsagententeam. Empfangen Sie Benutzeranfragen, klassifizieren Sie den Aufgabentyp, wählen Sie den passenden Workflow aus, delegieren Sie jeden Schritt über das Aufgabentool an spezialisierte Agenten und stellen Sie die endgültige Ausgabe zusammen. Verwenden Sie es, wenn der Benutzer eine mehrstufige Softwareentwicklung benötigt, die Architektur, Implementierung, Tests, Sicherheitsüberprüfung oder Codeüberprüfung umfasst. Lösen Sie diese Fähigkeit immer dann aus, wenn eine Aufgabe mehr als ein Anliegen beinhaltet (z. B. „Hinzufügen eines neuen Endpunkts“ erfordert BA + Architekt + Entwickler + QA + Sicherheit), wenn der Benutzer Teamkoordination oder Agentendelegation erwähnt oder wenn die Arbeit eindeutig von mehreren Spezialistenperspektiven profitiert und nicht von einem einzigen Implementierungsdurchgang.
