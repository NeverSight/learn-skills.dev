Synchronisieren Sie die API-Dokumentation aus einer Markdown-Datei mit Confluence-Seiten mithilfe von acli + REST API. Fordert zur Laufzeit zur Eingabe der Confluence-URL, des API-Dokumentdateipfads und der Anmeldeinformationen auf – keine vorkonfigurierten Umgebungsvariablen erforderlich. Verwenden Sie diesen Skill immer dann, wenn Sie API-Dokumente in Confluence hochladen oder synchronisieren, Confluence-Seiten aus einer Markdown-Datei aktualisieren, Dokumentation veröffentlichen, „API-Dokument synchronisieren“, „Dokument an Confluence senden“, „API-Dokument an Confluence senden“, „Confluence-Seiten synchronisieren“, „Confluence-API-Doc“ oder wenn der Benutzer Markdown-basierte Dokumentation in Confluence veröffentlichen oder aktualisieren möchte.
