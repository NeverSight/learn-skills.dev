Синхронизируйте документацию API из файла Markdown со страницами Confluence с помощью acli + REST API. Запрашивает URL-адрес Confluence, путь к файлу документа API и учетные данные во время выполнения — предварительно настроенные переменные среды не требуются. Используйте этот навык всякий раз, когда: загружаете или синхронизируете документацию API с Confluence, обновляете страницы Confluence из файла Markdown, публикуете документацию, «синхронизируете документ API», «перемещаете документ в Confluence», «загружаете API-документ и confluence», «синхронизируете страницы Confluence», «confluence-api-doc» или когда пользователь хочет опубликовать или обновить любой документ на основе Markdown. документация для Confluence.
