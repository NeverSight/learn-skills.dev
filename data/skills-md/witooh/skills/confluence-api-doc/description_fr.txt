Synchronisez la documentation de l'API d'un fichier Markdown vers les pages Confluence à l'aide de l'API acli + REST. Demandes d'URL Confluence, de chemin d'accès au fichier de documentation de l'API et d'informations d'identification au moment de l'exécution : aucune variable d'environnement préconfigurée n'est requise. Utilisez cette compétence à chaque fois : téléchargement ou synchronisation de documents API sur Confluence, mise à jour de pages Confluence à partir d'un fichier Markdown, publication de documentation, "sync api doc", "push doc to confluence", "อัปเดต api doc ไป confluence", "sync confluence pages", "confluence-api-doc", ou lorsque l'utilisateur souhaite publier ou mettre à jour une documentation basée sur Markdown sur Confluence.
