تفاعل مع GitLab عبر glab CLI. يدعم ثلاثة مسارات عمل للواقع المختلط - القراءة (التلخيص)، والمراجعة (الكود الكامل/الأمان/مراجعة ضمان الجودة)، والإصلاح (المراجعة + التنفيذ). يتم تشغيله عندما يقدم المستخدم عنوان URL لـ GitLab MR أو يقول أي شيء مثل "تمامًا MR" أو "ดู MR" أو "check MR" أو "مراجعة MR" أو "مراجعة MR नี้" أو "ตรวจ MR" أو "แก้ตาม MR" أو "إصلاح MR" أو فقط يلصق عنوان URL لـ GitLab MR. يدعم أيضًا إدراج MRs، وعرض حالة MR، والتحقق من خطوط أنابيب CI/CD، والموافقة على MRs، وعمليات الالتقاط الأخرى. قم بتشغيل "التحقق من خط الأنابيب" أو "إدراج MRs المفتوحة" أو أي مهمة متعلقة بـ GitLab. عندما يريد المستخدم إصلاح المشكلات من MR - على سبيل المثال. "แก้ตาม MR нี้"، "fix MR"، "แก้ المشكلة ตาม MR"، "แก้ MR <url>"، "مراجعة แก้ตาม" - قم بتشغيل سير عمل MR Fix (راجع ثم قم بالتسلسل التلقائي إلى /neo-team للتنفيذ).
