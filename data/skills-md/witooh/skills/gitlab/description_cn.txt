通过 glab CLI 与 GitLab 交互。支持三种 MR 工作流程 — 阅读（总结）、审核（完整代码/安全/QA 审核）和修复（审核 + 实施）。每当用户提供 GitLab MR URL 或说出类似“อ่าน MR”、“ดู MR”、“检查 MR”、“审核 MR”、“ช่วย 审核 MR นี้”、“ตรวจ MR”、“แก้ตาม MR”时触发， “修复 MR”，或者仅粘贴 GitLab MR URL。还支持列出 MR、查看 MR 状态、检查 CI/CD 管道、批准 MR 等 glab 操作。触发“检查管道”、“列出开放的 MR”或任何与 GitLab 相关的任务。当用户想要解决 MR 的问题时——例如"แก้ตาม MR นี้"、"fix MR"、"แก้ issues ตาม MR"、"แก้ MR <url>"、"แก้ตาม review" — 运行 MR Fix 工作流程（审核然后自动链接到 /neo-team实施）。
