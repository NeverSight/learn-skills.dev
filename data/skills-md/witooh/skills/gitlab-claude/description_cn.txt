通过 glab CLI 与 GitLab 交互。支持五种 MR 工作流程 - 阅读（总结）、审核（完整代码/安全/QA 审核）、修复（审核 + 实施）、CI 修复（修复管道故障）和反馈（处理审核意见）。每当用户提供 GitLab MR URL 或说出类似“อ่าน MR”、“ดู MR”、“检查 MR”、“审核 MR”、“ช่วย 审核 MR นี้”、“ตรวจ MR”、“แก้ตาม MR”时触发， “修复 MR”、“修复 CI”、“修复管道”、“แก้ 管道”、“แก้ตาม 评论”、“แก้ตาม 反馈”、“地址反馈”，或者只是粘贴 GitLab MR URL。还支持列出 MR、查看 MR 状态、检查 CI/CD 管道、批准 MR 等 glab 操作。触发“检查管道”、“列出打开的 MR”、“管道失败”或任何与 GitLab 相关的任务。
