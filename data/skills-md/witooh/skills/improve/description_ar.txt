قم بتحسين أي مخرجات بشكل متكرر حتى يتم استيفاء معايير قابلة للقياس. يُستخدم عندما يريد المستخدم تحسين العمل الحالي وفقًا لمعايير محددة - سواء كان ذلك رمزًا أو نثرًا أو بيانات أو تكوينًا أو أي شيء آخر. يتم تشغيل عبارات مثل "تحسين هذا"، أو "جعله أفضل"، أو "التكرار"، أو "التحسين"، أو "الاستمرار في التحسين"، أو "ليس جيدًا بما فيه الكفاية بعد"، أو "تحسين هذا"، أو "تحسين هذا"، أو "تشديد هذا"، أو عندما يقدم المستخدم معايير ويريد تحسينًا متكررًا حتى يشعر بالرضا. يُستخدم أيضًا عندما يقدم المستخدم تعليقات على المخرجات ويتوقع منك الاستمرار في التحسين، حتى لو لم يقول "تحسين" بشكل صريح.
