Variante del Código Claude. Orquestar un equipo de agentes de desarrollo de software especializados. Reciba solicitudes de usuarios, clasifique el tipo de tarea, seleccione el flujo de trabajo correspondiente, delegue cada paso a agentes especializados a través de la herramienta Agente y cree el resultado final. Úselo cuando el usuario necesite un desarrollo de software de varios pasos que involucre arquitectura, implementación, pruebas, revisión de seguridad o revisión de código. Active esta habilidad siempre que una tarea involucre más de una preocupación (por ejemplo, "agregar un nuevo punto final" necesita BA + Arquitecto + Desarrollador + Control de calidad + Seguridad), cuando el usuario mencione la coordinación del equipo, la delegación de agentes o cuando el trabajo claramente se beneficie de múltiples perspectivas especializadas en lugar de un solo paso de implementación.
