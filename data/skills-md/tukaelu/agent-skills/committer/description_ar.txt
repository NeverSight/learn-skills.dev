قم بتحليل الملفات التي تم تغييرها بواسطة أمر git وقم بإنشاء التزامات ذات حجم مناسب في وحدات منطقية. يتم تفعيلها فقط عندما يتم تحديد اسم المهارة (الملتزم) بشكل صريح، مثل "/ الملتزم"، أو "استخدام مهارة الالتزام"، أو "الالتزام مع الملتزم". ولا يتم تنشيطه للطلبات العامة مثل "الالتزام" أو "الالتزام بـ git".
