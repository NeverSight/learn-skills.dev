Analysez les fichiers modifiés par les commandes git et créez des commits de taille appropriée en unités logiques. Il est activé uniquement lorsque le nom de la compétence (committer) est explicitement spécifié, tel que « /committer », « en utilisant la compétence committer » ou « commit avec committer ». Il ne s'active pas pour les requêtes générales telles que « commit » ou « git commit ».
