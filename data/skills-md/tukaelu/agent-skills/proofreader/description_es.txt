Habilidad para corregir y revisar textos. Identifica errores tipográficos, gramaticales y áreas para mejorar la expresión, y sugiere correcciones.

Cuando se invoca mediante un comando de barra diagonal (por ejemplo, `/proofreader report.md` o `/proofreader ./docs/readme.txt`), lee el archivo especificado y realiza la revisión.

Cuando se le pide "corregir", "revisar", "revisar el texto", "comprobar si hay errores tipográficos" o "corregir correctamente", corrige el texto generado o presentado más recientemente en la sesión. Si existen varios candidatos o el objetivo no está claro, solicita confirmación.

Maneja tanto japonés como inglés. Puede corregir cualquier tipo de texto (borradores de correo electrónico, documentos, publicaciones de blog, comentarios de código) independientemente del tipo.

Consulte siempre esta habilidad cuando se solicite corrección de pruebas, revisión o revisión de errores tipográficos.
