Skill for proofreading and revising text. It spots typos, grammar mistakes, and areas for better phrasing, then offers corrections.

When invoked via slash command (e.g. `/proofreader report.md` or `/proofreader ./docs/readme.txt`), it reads the specified file and proofreads it.

When asked to “proofread,” “revise,” “check for typos,” “properly proofread,” etc., it corrects the text most recently generated or shown in the session. If multiple or unclear targets exist, it asks for confirmation.

Handles both Japanese and English. Can proofread anything written: email drafts, documents, blog posts, code comments—no restrictions.

Always consult this skill whenever a proofreading, revision, or typo-check is requested.
