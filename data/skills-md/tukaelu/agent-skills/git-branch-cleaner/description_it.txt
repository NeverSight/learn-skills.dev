Elimina tutti i rami locali e remoti che sono stati uniti nel ramo base specificato. Funziona solo quando il nome della competenza (git-branch-cleaner) è esplicitamente menzionato, ad es. “/git-branch-cleaner”, “usando la competenza git-branch-cleaner”, “ripulisci con git-branch-cleaner”. Non si avvia per richieste generiche come “elimina rami” o “rimuovi rami uniti”. L'ambito può essere controllato passando local/remote/entrambi come argomenti (il valore predefinito è local se omesso).
