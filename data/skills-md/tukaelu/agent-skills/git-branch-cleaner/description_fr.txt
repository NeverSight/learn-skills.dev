Supprimez toutes les branches locales et distantes qui ont été fusionnées dans la branche de base spécifiée. Il ne s'exécute que lorsque le nom de la compétence (git-branch-cleaner) est explicitement mentionné, par exemple. "/git-branch-cleaner", "en utilisant la compétence git-branch-cleaner", "nettoyer avec git-branch-cleaner". Il ne démarre pas pour les requêtes génériques telles que « supprimer les branches » ou « supprimer les branches fusionnées ». La portée peut être contrôlée en passant local / distant / les deux comme arguments (par défaut local en cas d'omission).
