Delete all local and remote branches that have been merged into the specified base branch. It runs only when the skill name (git-branch-cleaner) is explicitly mentioned—e.g. “/git-branch-cleaner”, “using git-branch-cleaner skill”, “clean up with git-branch-cleaner”. It does not start for generic requests like “delete branches” or “remove merged branches”. Scope can be controlled by passing local / remote / both as arguments (defaults to local if omitted).
