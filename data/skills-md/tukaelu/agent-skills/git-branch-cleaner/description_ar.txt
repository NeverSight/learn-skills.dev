حذف دفعة واحدة لفروع الميزات المحلية والبعيدة التي تم دمجها بالفعل في الفرع الأساسي المحدد.  
يتم تفعيلها فقط عندما يتم ذكر اسم المهارة (git-branch-cleaner) بشكل صريح، مثل "/ git-branch-cleaner"، أو "استخدام مهارة git-branch-cleaner"، أو "التنظيف باستخدام git-branch-cleaner".  
ولا يتم تفعيله للطلبات العامة مثل "حذف الفروع" أو "إزالة الفروع المدمجة".  
يمكنك التحكم في النطاق عن طريق تمرير محلي/بعيد/كلاهما كوسيطة (الإعداد الافتراضي هو محلي إذا تم حذفه).
