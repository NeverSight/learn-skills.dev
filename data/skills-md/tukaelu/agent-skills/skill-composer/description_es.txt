Apoyo la creación de nuevas habilidades y el mantenimiento de las existentes en japonés. Constantemente llevo a cabo recopilación de información, generación o actualizaciones de habilidades, revisiones de calidad y correcciones automáticas. Se activa mediante solicitudes japonesas como "Crear una habilidad", "Quiero agregar una habilidad que haga XX", "Quiero modificar esta habilidad", "Quiero mejorar una habilidad" o el comando /skill-composer.
