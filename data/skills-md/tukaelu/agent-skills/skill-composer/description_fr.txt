Je soutiens la création de nouvelles compétences et le maintien de celles existantes en japonais. Il gère de manière cohérente la collecte d'informations, la génération ou les mises à jour de compétences, les examens de qualité et les correctifs automatiques. It activates with Japanese requests like “Create a skill,” “I want to add a skill that does 〇〇,” “Please fix this skill,” or “Improve this skill,” as well as via the /skill-composer command.
