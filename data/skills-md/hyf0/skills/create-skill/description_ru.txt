Создает новый навык агента, следуя лучшим практикам структуры, написания описаний и постепенного раскрытия информации. Используйте, когда пользователь хочет создать новый SKILL.md, создать каталог навыков или создать навык многократного использования. НЕ используйте для улучшения существующего навыка — вместо этого используйте оптимизирующий навык.
