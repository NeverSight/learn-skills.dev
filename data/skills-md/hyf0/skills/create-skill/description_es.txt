Crea una nueva habilidad para el agente siguiendo las mejores prácticas de estructura, redacción de descripciones y divulgación progresiva. Úselo cuando el usuario quiera crear un nuevo SKILL.md, desarrollar un directorio de habilidades o crear una habilidad reutilizable. NO lo use para mejorar una habilidad existente; en su lugar, use optimizar-habilidad.
