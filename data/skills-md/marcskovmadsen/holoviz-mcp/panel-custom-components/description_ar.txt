أنشئ مكونات لوحة مخصصة باستخدام JSComponent (vanilla JS، مكونات الويب)، أو ReactComponent (React/JSX)، أو AnyWidgetComponent (مواصفات AnyWidget للأنظمة الأساسية المشتركة)، أو MaterialUIComponent (موضوع واجهة المستخدم المادية). يُستخدم عند تغليف مكتبات JS، أو إنشاء عناصر واجهة مستخدم تفاعلية، أو إنشاء مكونات ذات سمة معينة. يتضمن دليل القرار وأفضل الممارسات وما يجب فعله وما لا يجب فعله وأنماط اختبار واجهة المستخدم الخاصة بالكاتب المسرحي.
