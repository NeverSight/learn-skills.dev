Cree componentes de Panel personalizados utilizando JSComponent (vanilla JS, componentes web), ReactComponent (React/JSX), AnyWidgetComponent (especificación AnyWidget para multiplataforma) o MaterialUIComponent (temática Material UI). Úselo al empaquetar bibliotecas JS, crear widgets interactivos o crear componentes temáticos. Incluye una guía de decisiones, mejores prácticas, lo que se debe y lo que no se debe hacer y patrones de prueba de la interfaz de usuario de Playwright.
