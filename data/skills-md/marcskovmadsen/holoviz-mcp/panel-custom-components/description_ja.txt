JSComponent (バニラ JS、Web コンポーネント)、ReactComponent (React/JSX)、AnyWidgetComponent (クロスプラットフォームの AnyWidget 仕様)、または MaterialUIComponent (マテリアル UI テーマ) を使用して、カスタム パネル コンポーネントを構築します。 JS ライブラリをラップする場合、インタラクティブなウィジェットを作成する場合、またはテーマ別コンポーネントを構築する場合に使用します。意思決定ガイド、ベスト プラクティス、すべきこと/してはいけないこと、Playwright UI テスト パターンが含まれています。
