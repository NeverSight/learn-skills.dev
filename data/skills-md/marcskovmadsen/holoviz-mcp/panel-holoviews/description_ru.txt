Рекомендации по интеграции визуализаций HoloViews и hvPlot в приложения Panel. Используйте при встраивании графиков HoloViews/hvPlot в панели Panel, сохранении состояния масштабирования/панорамирования при обновлении данных с помощью DynamicMap, составлении наложений DynamicMap без ошибок типа, использовании потоков HoloViews (Selection1D, RangeXY, Tap, BoundsXY, Pipe, Buffer) с Panel, перекрестной фильтрации с помощью link_selections, обеспечении отзывчивости графиков HoloViews в макетах Panel или подключении виджетов Panel к Свойства графика боке с помощью jslink.
