Mejores prácticas para desarrollar herramientas, paneles y aplicaciones de datos interactivas con HoloViz Panel. Cree UI reactivas basadas en componentes con widgets, diseños, plantillas y actualizaciones en tiempo real. Úselo al desarrollar herramientas interactivas de exploración de datos, paneles, aplicaciones de datos o cualquier aplicación web interactiva de Python. Admite carga de archivos, transmisión de datos, aplicaciones de varias páginas e integración con HoloViews, hvPlot, Pandas, Polars, DuckDB y el resto de los ecosistemas HoloViz y PyData.
