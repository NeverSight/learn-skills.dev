HoloViz パネルを使用してツール、ダッシュボード、対話型データ アプリを開発するためのベスト プラクティス。ウィジェット、レイアウト、テンプレート、リアルタイム更新を備えたリアクティブなコンポーネントベースの UI を作成します。インタラクティブなデータ探索ツール、ダッシュボード、データ アプリ、またはインタラクティブな Python Web アプリケーションを開発するときに使用します。ファイルのアップロード、ストリーミング データ、マルチページ アプリ、HoloViews、hvPlot、Pandas、Polars、DuckDB、その他の HoloViz および PyData エコシステムとの統合をサポートします。
