يُستخدم عند إنشاء فئات Python باستخدام معلمات مكتوبة تم التحقق من صحتها باستخدام مكتبة Param. تتضمن المشغلات إنشاء فئات التكوين، أو إنشاء مكونات قابلة لإعادة الاستخدام مع الحالة، أو تنفيذ التبعيات التفاعلية بين المعلمات، أو إضافة سمات النوع الآمن بحدود/قيود، أو إنشاء فئات ذات معلمات قابلة للاختبار، أو عندما يذكر المستخدمون param.Parameterized، أو @param.depends، أو param.watch.
