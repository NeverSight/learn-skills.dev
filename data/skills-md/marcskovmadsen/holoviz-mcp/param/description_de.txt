Wird zum Erstellen von Python-Klassen mit validierten, typisierten Parametern mithilfe der Param-Bibliothek verwendet. Zu den Auslösern gehören das Erstellen von Konfigurationsklassen, das Erstellen wiederverwendbarer Komponenten mit Status, das Implementieren reaktiver Abhängigkeiten zwischen Parametern, das Hinzufügen typsicherer Attribute mit Grenzen/Einschränkungen, das Erstellen testbarer parametrisierter Klassen oder wenn Benutzer param.Parameterized, @param.depends oder param.watch erwähnen.
