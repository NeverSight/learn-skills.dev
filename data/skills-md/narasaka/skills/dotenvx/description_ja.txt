dotenvx CLI ツールに関連するタスク (.env ファイルの暗号化、挿入された環境変数を使用したコマンドの実行、環境全体でのシークレットの管理、実行時の復号化など) に使用する必要があります。ユーザーが dotenvx、dotenv 暗号化、DOTENV_PRIVATE_KEY、暗号化された .env ファイル、または dotenvx encrypt/run/set/get/decrypt/keypair コマンドについて言及する場合は常に、このスキルを使用します。また、ユーザーが次のことを行う場合にもトリガーします。 .env ファイルを Git に安全にコミットする、Slack/チャットでのシークレットの共有を停止する、公開鍵暗号で環境変数を暗号化する、複数環境の .env 構成 (本番/ステージング/ci) をセットアップする、-fk フラグを使用してモノリポジトリでシークレットを管理する、python-dotenv またはプレーン dotenv から暗号化された環境に移行する、環境変数を任意のプロセスに挿入する言語 (Node、Python、Ruby、Go、Rust など) を設定したり、暗号化された env ファイルを使用して CI/CD パイプライン (GitHub Actions、Docker) を設定したりできます。このスキルには、権威ある CLI リファレンスが含まれています。これがないと、応答で存在しないコマンドやフラグが幻覚のように表示されます。
