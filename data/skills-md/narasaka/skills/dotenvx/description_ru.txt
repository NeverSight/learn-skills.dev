НЕОБХОДИМО ИСПОЛЬЗОВАТЬ для любой задачи, связанной с инструментом CLI dotenvx — шифрование файлов .env, запуск команд с введенными переменными env, управление секретами в разных средах и расшифровка во время выполнения. Используйте этот навык всякий раз, когда пользователь упоминает dotenvx, шифрование dotenv, DOTENV_PRIVATE_KEY, зашифрованные файлы .env или команды dotenvx encrypt/run/set/get/decrypt/keypair. Также срабатывает, когда пользователь хочет: безопасно зафиксировать файлы .env в git, прекратить делиться секретами через Slack/чат, зашифровать переменные среды с помощью криптографии с открытым ключом, настроить конфигурации .env для нескольких сред (production/staging/ci), управлять секретами в монорепо с флагом -fk, перейти с python-dotenv или простого dotenv на зашифрованные окружения, внедрить переменные окружения в любой процесс на любом языке (Node, Python, Ruby, Go, Rust и т. д.) или настройте конвейеры CI/CD (GitHub Actions, Docker) с зашифрованными файлами env. Этот навык содержит авторитетную ссылку на CLI — без нее ответы будут представлять собой несуществующие команды и флаги.
