CO-STAR UI Creator per React/Next.js: suggerimenti strutturati per creare o integrare l'interfaccia utente (componenti, pagine, sistemi di progettazione) o per creare prompt CO-STAR personalizzati. Da utilizzare quando l'utente desidera aggiungere sezioni hero, dashboard, token di progettazione, integrazione shadcn/Magic UI/Aceternity, sistema di progettazione Newsprint o per redigere il proprio prompt CO-STAR (Contesto, Obiettivo, Stile, Tono, Pubblico, Risposta). Esegui sempre prima le domande conoscitive, poi la pianificazione e infine l'implementazione.
