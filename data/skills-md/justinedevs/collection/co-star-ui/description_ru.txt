CO-STAR UI Creator для React/Next.js: структурированные подсказки для создания или интеграции пользовательского интерфейса (компоненты, страницы, системы дизайна) или для создания пользовательских подсказок CO-STAR. Используйте, когда пользователь хочет добавить главные разделы, информационные панели, жетоны дизайна, интеграцию shadcn/Magic UI/Aceternity, систему дизайна Newsprint или создать собственное приглашение CO-STAR (контекст, цель, стиль, тон, аудитория, реакция). Всегда сначала задавайте вопросы обнаружения, затем планирования, а затем реализации.
