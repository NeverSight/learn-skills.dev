Session-Experten-Managementsystem für Claude-Code-Projekte. Ermöglicht das Verfolgen, Weiterleiten, Aktivieren und Registrieren von Konversationssitzungen als Domänenexperten. Verwenden Sie diese Fähigkeit, wenn:
(1) Einrichten der Sitzungsverwaltung für ein neues Projekt (Initialisierung)
(2) Ein Benutzer fordert eine Funktionsänderung, Fehlerbehebung oder Optimierung an und Sie müssen prüfen, ob ein Sitzungsexperte diese Domäne bereits verwaltet (Routing).
(3) Eine Sitzung hat einen tiefen Domänenkontext angesammelt und sollte sich als Experte registrieren (Registrierung).
(4) Ein bestehender Sitzungsexperte muss seine Registrierung nach Änderungen des Umfangs aktualisieren (Update).
(5) Eine Sitzung ist abgelaufen und muss zur Fortsetzung aktiviert werden (Aktivierung).
(6) Der Benutzer fragt nach der Verwaltung, Organisation oder Optimierung der Sitzungsdokumentation
