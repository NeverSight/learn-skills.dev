Система экспертного управления сессиями для проектов Claude Code. Позволяет отслеживать, маршрутизировать, активировать и регистрировать сеансы разговоров в качестве экспертов предметной области. Используйте этот навык, когда:
(1) Настройка управления сеансами для нового проекта (инициализация)
(2) Пользователь запрашивает изменение функции, исправление ошибок или оптимизацию, и вам необходимо проверить, обрабатывает ли уже эксперт по сеансам этот домен (маршрутизацию).
(3) Сеанс накопил глубокий контекст предметной области и должен зарегистрироваться в качестве эксперта (регистрация).
(4) Существующему эксперту сеанса необходимо обновить свою регистрацию после изменения объема (обновление).
(5) Срок действия сеанса истек, и для возобновления его необходимо активировать (активация).
(6) Пользователь спрашивает об управлении, организации или оптимизации документации сеанса.
