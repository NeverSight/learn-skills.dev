نظام إدارة الخبراء للجلسة لمشاريع كلود كود. يتيح تتبع جلسات المحادثة وتوجيهها وتنشيطها وتسجيلها كخبراء في المجال. استخدم هذه المهارة عندما:
(1) إعداد إدارة الجلسة لمشروع جديد (التهيئة)
(2) يطلب المستخدم تغيير الميزة أو إصلاح الأخطاء أو التحسين وتحتاج إلى التحقق مما إذا كان خبير الجلسة يتعامل بالفعل مع هذا المجال (التوجيه)
(3) تحتوي الجلسة على سياق نطاق عميق متراكم ويجب أن تسجل نفسها كخبير (تسجيل)
(4) يحتاج خبير الجلسة الحالي إلى تحديث تسجيله بعد تغييرات النطاق (التحديث)
(5) انتهت الجلسة وتحتاج إلى تنشيط لاستئنافها (التنشيط)
(6) يسأل المستخدم عن إدارة وثائق الجلسة أو تنظيمها أو تحسينها
