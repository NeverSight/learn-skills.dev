Claude Code プロジェクト用のセッション エキスパート管理システム。ドメインエキスパートとして会話セッションの追跡、ルーティング、アクティブ化、登録を可能にします。このスキルは次の場合に使用します。
(1) 新規プロジェクトのセッション管理の設定（初期化）
(2) ユーザーが機能変更、バグ修正、または最適化をリクエストした場合、セッション エキスパートがすでにそのドメイン (ルーティング) を処理しているかどうかを確認する必要があります。
(3) セッションには深いドメイン コンテキストが蓄積されており、それ自体をエキスパートとして登録する必要があります (登録)
(4) 既存のセッションエキスパートは、スコープ変更後に登録を更新する必要があります（更新）
(5) セッションが期限切れになり、再開するにはアクティブ化が必要です（アクティベーション）
(6) ユーザーはセッション文書の管理、整理、または最適化について尋ねます。
