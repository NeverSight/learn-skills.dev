Sistema di gestione delle sessioni di esperti per progetti Claude Code. Consente il monitoraggio, l'instradamento, l'attivazione e la registrazione delle sessioni di conversazione come esperti di dominio. Usa questa abilità quando:
(1) Impostazione della gestione della sessione per un nuovo progetto (inizializzazione)
(2) Un utente richiede una modifica di funzionalità, correzione di bug o ottimizzazione ed è necessario verificare se un esperto di sessione gestisce già quel dominio (routing)
(3) Una sessione ha accumulato un contesto di dominio profondo e dovrebbe registrarsi come esperto (registrazione)
(4) Un esperto di sessione esistente deve aggiornare la propria registrazione dopo le modifiche dell'ambito (aggiornamento)
(5) Una sessione è scaduta e deve essere attivata per riprendere (attivazione)
(6) L'utente chiede informazioni sulla gestione, organizzazione o ottimizzazione della documentazione della sessione
