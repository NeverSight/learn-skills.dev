Claude Code 프로젝트를 위한 세션 전문가 관리 시스템입니다. 도메인 전문가로서 대화 세션을 추적, 라우팅, 활성화 및 등록할 수 있습니다. 다음과 같은 경우에 이 기술을 사용하십시오.
(1) 새 프로젝트에 대한 세션 관리 설정(초기화)
(2) 사용자가 기능 변경, 버그 수정, 최적화를 요청하고 해당 도메인(라우팅)을 세션 전문가가 이미 처리하고 있는지 확인해야 함
(3) 세션은 깊은 도메인 컨텍스트를 축적했으며 전문가로 등록(등록)해야 합니다.
(4) 기존 세션 전문가는 범위 변경(업데이트) 후 등록을 업데이트해야 합니다.
(5) 세션이 만료되었으며 재개하려면 활성화해야 합니다(활성화).
(6) 사용자가 세션 문서의 관리, 구성 또는 최적화에 대해 문의합니다.
