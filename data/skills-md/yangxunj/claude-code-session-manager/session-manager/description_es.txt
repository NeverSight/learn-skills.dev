Sistema de gestión experto en sesiones para proyectos de Claude Code. Permite rastrear, enrutar, activar y registrar sesiones de conversación como expertos del dominio. Utilice esta habilidad cuando:
(1) Configurar la gestión de sesiones para un nuevo proyecto (inicialización)
(2) Un usuario solicita un cambio de función, corrección de errores u optimización y usted necesita verificar si un experto en sesiones ya maneja ese dominio (enrutamiento).
(3) Una sesión ha acumulado un contexto de dominio profundo y debe registrarse como experto (registro)
(4) Un experto en sesión existente necesita actualizar su registro después de cambios de alcance (actualización)
(5) Una sesión ha caducado y es necesario activarla para reanudarla (activación)
(6) El usuario pregunta sobre la gestión, organización u optimización de la documentación de la sesión.
