Code-Review-Fähigkeiten basierend auf lokalem Git-Diff. Verwenden Sie es zur Selbstüberprüfung, bevor Sie eine PR einreichen, und verwenden Sie dabei strikt die dreidimensionalen Kriterien der PR-Überprüfung (Codequalitätsstandards, FastGPT-Stilrichtlinien, Checkliste für häufige Probleme) und unterstützen Sie die Überprüfung von nicht bereitgestellten Änderungen, bereitgestellten Änderungen, einem bestimmten Commit oder allen Änderungen des aktuellen Zweigs relativ zum Basiszweig.
