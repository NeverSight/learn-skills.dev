Guide d'expert pour créer des composants d'interface utilisateur accessibles et personnalisables avec shadcn/ui, Radix UI et Tailwind CSS. Couvre l'installation, les composants principaux (bouton, entrée, formulaire, carte, boîte de dialogue, sélection, feuille, tableau, toast, graphiques), les formulaires avec validation React Hook Form + Zod, la thématisation avec des variables CSS, l'intégration Next.js et les modèles avancés.
