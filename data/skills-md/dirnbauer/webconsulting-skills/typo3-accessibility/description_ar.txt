تدقيق إمكانية الوصول إلى WCAG 2.2 AA وتنفيذها لـ TYPO3 v12–v14 (يفضل الإصدار 14). يغطي أنماط قوالب السوائل، والبرمجيات الوسيطة لـ PHP، وتحسينات JavaScript، وإمكانية الوصول إلى عناصر المحتوى، وإمكانية الوصول إلى النماذج، وقائمة مرجعية كاملة للبث المباشر. يُستخدم عند العمل مع إمكانية الوصول، أو a11y، أو wcag، أو aria، أو قارئ الشاشة، أو التنقل عبر لوحة المفاتيح، أو إدارة التركيز، أو تباين الألوان، أو النص البديل، أو html الدلالي، أو تخطي الارتباط، أو النماذج التي يمكن الوصول إليها في TYPO3.
