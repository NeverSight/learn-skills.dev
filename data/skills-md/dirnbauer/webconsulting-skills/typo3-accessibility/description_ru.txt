Аудит и реализация доступности WCAG 2.2 AA для TYPO3 v12–v14 (предпочтительно v14). Охватывает шаблоны шаблонов Fluid, промежуточное программное обеспечение PHP, улучшения JavaScript, доступность элементов контента, доступность форм и полный контрольный список для ввода в эксплуатацию. Используйте при работе с специальными возможностями, a11y, wcag, aria, программой чтения с экрана, навигацией с помощью клавиатуры, управлением фокусом, цветовым контрастом, замещающим текстом, семантическим HTML, пропуском ссылок или доступными формами в TYPO3.
