# Book Content Extractor

本技能是专为“深度语义萃取”打造的人工智能专家。它通过标准化的过滤模型，将繁杂的书籍内容重塑为具备极高传播力与逻辑性的核心笔记。

## 核心能力

1. **深度核心萃取 (AI Intelligence)**：严格遵循 `reference/content_extraction_spec.md` 规范，提炼 **50% 的核心高价值内容**。
2. **费曼主线构建**：洞察全书逻辑基石，构建一条“背景 -> 矛盾 -> 方案 -> 路径”的严密讲解主线，确保内容节点顺畅串联。
3. **高价值内容钩子**：强制提取反常识观点、冲击性数据、情绪故事、关键人物及隐性洞察。
4. **行动转化与 SOP**：将理论转化为可执行的步骤、工具表单及检查清单。
5. **语言纯净性**：遵循纯净中文表达规范，杜绝中英混杂句，确保专业交付质量。

## 闭环工作流

1. **[语义分析]**：读取书籍 Markdown 全文及萃取规范。
2. **[逻辑建模]**：寻找“灵魂红线”，构建费曼讲解主线。
3. **[价值萃取]**：执行“结构-逻辑-细节-行动”四层过滤，提炼核心模型。
4. **[质量核检]**：依据规范中的六维度（准确、完整、独立、费曼逻辑、语言纯净、触达力）进行自查。

## 适用场景

- 需要从大量文字中快速提取核心干货时。
- 准备书籍分享、讲稿或深度学习笔记时。
- 希望通过费曼学习法快速掌握一门新领域知识时。
