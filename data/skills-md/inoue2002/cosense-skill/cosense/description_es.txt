Una habilidad para leer, buscar, crear y editar páginas en Cosense (anteriormente Scrapbox). Utiliza las variables de entorno COSENSE_SID y COSENSE_PROJECT para operar la API. Úselo cuando se le solicite "leer una página de Cosense", "buscar en Cosense", "crear una página en Cosense", "editar una página de Cosense", "Sukubo" o "caja de chatarra".
