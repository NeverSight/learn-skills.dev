Lignes directrices et principes universels en matière de style de code pour écrire du code propre et maintenable dans n’importe quel langage de programmation. À utiliser lors de l'écriture ou de la révision de code, de la refactorisation du code existant, de la réalisation de révisions de code ou de l'établissement de normes de codage. Se concentre sur l'abstraction, les principes KISS, les principes SOLID et évite la sur-ingénierie.
