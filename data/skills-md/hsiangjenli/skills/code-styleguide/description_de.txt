Universelle Richtlinien und Prinzipien für den Codestil zum Schreiben von sauberem, wartbarem Code in jeder Programmiersprache. Verwenden Sie es, wenn Sie Code schreiben oder überprüfen, vorhandenen Code umgestalten, Codeüberprüfungen durchführen oder Codierungsstandards festlegen. Konzentriert sich auf Abstraktion, KISS-Prinzipien, SOLID-Prinzipien und die Vermeidung von Over-Engineering.
