Directrices y principios de estilo de código universal para escribir código limpio y fácil de mantener en cualquier lenguaje de programación. Úselo al escribir o revisar código, refactorizar código existente, realizar revisiones de código o establecer estándares de codificación. Se centra en la abstracción, los principios KISS, los principios SOLID y evitar la ingeniería excesiva.
