Utilice esta habilidad para el ciclo de vida de Modal Sandbox y la ejecución aislada interactiva. Se activa cuando el usuario necesita ejecutar código que no es de confianza o proporcionado por el usuario en un entorno limitado, mantener vivo un proceso de controlador de larga duración, transmitir stdout desde llamadas `exec()`, volver a conectarse a un entorno limitado existente, cargar archivos en tiempo de ejecución, exponer un servicio HTTP o TCP tunelizado, o establecer un punto de control y restaurar el estado del entorno limitado. No lo use para implementaciones regulares de `@app.function`, servicio de LLM o flujos de trabajo de capacitación.
