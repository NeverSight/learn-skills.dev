Verwenden Sie diese Fähigkeit für den Modal Sandbox-Lebenszyklus und die interaktive isolierte Ausführung. Wird ausgelöst, wenn der Benutzer nicht vertrauenswürdigen oder vom Benutzer bereitgestellten Code in einer Sandbox ausführen, einen langlebigen Controller-Prozess am Leben halten, Stdout von „exec()“-Aufrufen streamen, eine erneute Verbindung zu einer vorhandenen Sandbox herstellen, Dateien zur Laufzeit hochladen, einen getunnelten HTTP- oder TCP-Dienst verfügbar machen oder einen Checkpoint durchführen und den Sandbox-Status wiederherstellen muss. Verwenden Sie es nicht für reguläre „@app.function“-Bereitstellungen, LLM-Bereitstellung oder Schulungsworkflows.
