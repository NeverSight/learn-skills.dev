Utilizza questa competenza per il ciclo di vita Modal Sandbox e l'esecuzione interattiva isolata. Si attiva quando l'utente deve eseguire codice non attendibile o fornito dall'utente in una sandbox, mantenere attivo un processo del controller di lunga durata, eseguire lo streaming di stdout dalle chiamate "exec()", ricollegarsi a una sandbox esistente, caricare file in fase di runtime, esporre un servizio HTTP o TCP con tunnel o checkpoint e ripristinare lo stato della sandbox. Non utilizzarlo per distribuzioni regolari di `@app.function`, servizi LLM o flussi di lavoro di formazione.
