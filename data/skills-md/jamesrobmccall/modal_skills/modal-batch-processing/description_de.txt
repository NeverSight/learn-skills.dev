Verwenden Sie diesen Skill für die modale Job-Orchestrierung mit „.map“, „.starmap“, „.spawn“, „.spawn_map“ oder „@modal.batched“. Wird ausgelöst, wenn der Benutzer die Arbeit auf modale Container verteilen, Ergebnisse im Prozess sammeln, eine abfragbare Job-ID von einem Webserver zurückgeben, die Parallelität begrenzen, eine Wiederherstellung nach Teilfehlern durchführen oder viele kleine Anforderungen in weniger GPU-Aufrufen zusammenfassen muss. Verwenden Sie es nicht für vLLM- oder SGLang-Bereitstellung, Modell-Feinabstimmung oder Fragen zum Sandbox-Lebenszyklus.
