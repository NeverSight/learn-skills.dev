Utilisez cette compétence pour le LLM de texte ouvert auto-hébergé diffusé sur Modal avec vLLM ou SGLang. Déclenchez lorsque l'utilisateur souhaite un point de terminaison compatible OpenAI pour un modèle qu'il héberge, doit régler les démarrages à froid, la latence, la concurrence, le parallélisme tensoriel ou le débit pour la génération de texte, ou souhaite une inférence de texte par lots hors ligne avec le moteur vLLM. Ne l'utilisez pas pour les intégrations, le service de pipeline générique Hugging Face, la diffusion, le réglage fin, RAG ou les applications qui appellent uniquement l'API hébergée d'OpenAI.
