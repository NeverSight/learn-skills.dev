Verwenden Sie diesen Skill für die selbstgehostete Open-Weight-Text-LLM-Bereitstellung auf Modal mit vLLM oder SGLang. Wird ausgelöst, wenn der Benutzer einen OpenAI-kompatiblen Endpunkt für ein von ihm gehostetes Modell wünscht, Kaltstarts, Latenz, Parallelität, Tensorparallelität oder Durchsatz für die Textgenerierung optimieren muss oder Offline-Batch-Textinferenz mit der vLLM-Engine wünscht. Verwenden Sie es nicht für Einbettungen, generische Hugging Face-Pipeline-Bereitstellung, Diffusion, Feinabstimmung, RAG oder Apps, die nur die gehostete API von OpenAI aufrufen.
