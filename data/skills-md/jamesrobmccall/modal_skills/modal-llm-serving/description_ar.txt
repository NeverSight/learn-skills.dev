استخدم هذه المهارة لعرض LLM النصي مفتوح الوزن المستضاف ذاتيًا على Modal مع vLLM أو SGLang. يتم التشغيل عندما يريد المستخدم نقطة نهاية متوافقة مع OpenAI لنموذج يستضيفه، أو يحتاج إلى ضبط البداية الباردة، أو زمن الوصول، أو التزامن، أو توازي الموتر، أو الإنتاجية لإنشاء النص، أو يريد استنتاج نص دفعة دون اتصال باستخدام محرك vLLM. لا تستخدمه للتضمين، أو خدمة خطوط أنابيب Hugging Face العامة، أو النشر، أو الضبط الدقيق، أو RAG، أو التطبيقات التي تستدعي فقط واجهة برمجة التطبيقات المستضافة لـ OpenAI.
