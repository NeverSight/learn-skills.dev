Используйте этот навык для самостоятельного размещения текста LLM с открытым весом на модальном языке с помощью vLLM или SGLang. Запускается, когда пользователю нужна конечная точка, совместимая с OpenAI, для модели, которую он размещает, ему необходимо настроить холодный запуск, задержку, параллелизм, тензорный параллелизм или пропускную способность для генерации текста или требуется автономный пакетный вывод текста с помощью механизма vLLM. Не используйте его для встраивания, общего обслуживания конвейера Hugging Face, распространения, точной настройки, RAG или приложений, которые вызывают только размещенный API OpenAI.
