Utilice esta habilidad para LLM de texto abierto autohospedado que se publica en Modal con vLLM o SGLang. Se activa cuando el usuario desea un punto final compatible con OpenAI para un modelo que aloja, necesita ajustar los arranques en frío, la latencia, la simultaneidad, el paralelismo tensorial o el rendimiento para la generación de texto, o desea una inferencia de texto por lotes sin conexión con el motor vLLM. No lo utilice para incrustaciones, servicio de canalización genérico de Hugging Face, difusión, ajuste fino, RAG o aplicaciones que solo llaman a la API alojada de OpenAI.
