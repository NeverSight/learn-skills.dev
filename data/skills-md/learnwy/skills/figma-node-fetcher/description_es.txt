Obtenga el JSON del nodo Figma o la imagen del nodo desde una URL de Figma. Úselo cuando los usuarios necesiten una estructura de capas/nodos, exportación de capturas de pantalla de nodos o inspección de tokens de diseño desde Figma. Requerir URL con consulta de ID de nodo. Verifique FIGMA_ACCESS_TOKEN primero y guíe la configuración cuando falte.
