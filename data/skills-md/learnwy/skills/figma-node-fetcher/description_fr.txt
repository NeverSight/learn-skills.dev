Récupérez le nœud Figma JSON ou l'image de nœud à partir d'une URL Figma. À utiliser lorsque les utilisateurs ont besoin d'une structure de couches/nœuds, d'une exportation de capture d'écran de nœud ou d'une inspection de jeton de conception à partir de Figma. Exiger une URL avec une requête d'ID de nœud. Vérifiez d'abord FIGMA_ACCESS_TOKEN et guidez la configuration en cas d'absence.
