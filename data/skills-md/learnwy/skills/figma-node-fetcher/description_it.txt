Recupera il JSON del nodo Figma o l'immagine del nodo da un URL Figma. Utilizzare quando gli utenti necessitano di una struttura di livelli/nodi, di esportazione di screenshot di nodi o di ispezione di token di progettazione da Figma. Richiedi URL con query ID nodo. Controlla prima FIGMA_ACCESS_TOKEN e guida la configurazione quando manca.
