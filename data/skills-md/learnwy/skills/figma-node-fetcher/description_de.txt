Rufen Sie den Figma-Knoten-JSON oder das Knotenbild von einer Figma-URL ab. Wird verwendet, wenn Benutzer eine Ebenen-/Knotenstruktur, einen Knoten-Screenshot-Export oder eine Design-Token-Inspektion von Figma benötigen. URL mit Knoten-ID-Abfrage erforderlich. Überprüfen Sie zuerst FIGMA_ACCESS_TOKEN und führen Sie die Einrichtung durch, wenn es fehlt.
