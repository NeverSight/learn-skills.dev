ТОЛЬКО АВТОМОБИЛЬНЫЙ ОТКАЗ: Комплексный набор методологических инструментов разработки программного обеспечения с 10 проверенными агентами. Используйте только в том случае, если ни один навык, специфичный для проекта (например, рабочий процесс требований), не соответствует. Обеспечивает: определителя проблем (Вайнберг), составителя историй (Паттон), спецификации на примере (Аджик), разработчика предметной области (ДДД/Эванс), разработчика моделей ответственности (CRC/Вирфс-Брок), консультанта по архитектуре (атрибуты качества/Басс), тренера по tdd (Бек), руководства по рефакторингу (Фаулер), хирурга наследия (Физерс), специалиста по стратегии тестирования (Криспин). Срабатывает ТОЛЬКО если навыки проекта не совпадают.
