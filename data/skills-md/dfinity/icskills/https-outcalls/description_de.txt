Stellen Sie HTTPS-Anfragen von Kanistern an externe Web-APIs. Behandelt Transformationsfunktionen für Konsens, Zykluskostenmanagement, Antwortgrößenbeschränkungen und Idempotenzmuster. Wird verwendet, wenn ein Kanister eine externe API aufrufen, Daten aus dem Web abrufen oder HTTP-Anfragen stellen muss. NICHT für EVM/Ethereum-Aufrufe verwenden – verwenden Sie stattdessen evm-rpc.
