Effectuez des requêtes HTTPS depuis des canisters vers des API Web externes. Couvre les fonctions de transformation pour le consensus, la gestion des coûts du cycle, les limites de taille de réponse et les modèles d'idempotence. À utiliser lorsqu'un canister doit appeler une API externe, récupérer des données sur le Web ou effectuer des requêtes HTTP. Ne PAS utiliser pour les appels EVM/Ethereum – utilisez plutôt evm-rpc.
