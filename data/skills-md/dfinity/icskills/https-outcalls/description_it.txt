Effettua richieste HTTPS dai contenitori alle API Web esterne. Copre le funzioni di trasformazione per il consenso, la gestione dei costi del ciclo, i limiti delle dimensioni della risposta e i modelli di idempotenza. Da utilizzare quando un contenitore deve chiamare un'API esterna, recuperare dati dal Web o effettuare richieste HTTP. NON utilizzare per chiamate EVM/Ethereum: utilizzare invece evm-rpc.
