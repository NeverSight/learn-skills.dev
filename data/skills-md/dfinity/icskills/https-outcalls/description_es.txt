Realice solicitudes HTTPS desde contenedores a API web externas. Cubre funciones de transformación para consenso, gestión de costos de ciclo, límites de tamaño de respuesta y patrones de idempotencia. Úselo cuando un recipiente necesita llamar a una API externa, recuperar datos de la web o realizar solicitudes HTTP. NO lo use para llamadas EVM/Ethereum; use evm-rpc en su lugar.
