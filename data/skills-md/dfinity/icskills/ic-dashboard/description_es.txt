Consulta las API REST públicas que impulsan tablero.internetcomputer.org. Cubre metadatos de recipiente, datos del libro mayor del CICR, datos del SNS, libro mayor del ICP y métricas de red con paginación basada en cursor. Úselo cuando obtenga información del recipiente, datos de tokens, propuestas de SNS o estadísticas de red a través de HTTP desde código fuera de la cadena. No se necesitan ciclos ni despliegue del recipiente.
