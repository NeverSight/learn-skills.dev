Interrogez les API REST publiques qui alimentent Dashboard.internetcomputer.org. Couvre les métadonnées du canister, les données du grand livre du CICR, les données SNS, le grand livre ICP et les métriques du réseau avec une pagination basée sur le curseur. À utiliser lors de la récupération d'informations sur la cartouche, de données de jeton, de propositions SNS ou de statistiques réseau via HTTP à partir de code hors chaîne. Aucun déploiement de cartouche ni cycles nécessaires.
