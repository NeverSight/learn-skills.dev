Запросите общедоступные API-интерфейсы REST, которые используются в Dashboard.internetcomputer.org. Охватывает метаданные контейнера, данные реестра МККК, данные социальных сетей, реестр ICP и сетевые показатели с нумерацией страниц на основе курсора. Используйте при получении информации о контейнере, данных токенов, предложений SNS или статистики сети через HTTP из кода вне сети. Никакого развертывания канистры или циклов не требуется.
