Appelez les chaînes Ethereum et EVM à partir des cartouches IC via la cartouche EVM RPC. Couvre les appels JSON-RPC, le consensus multi-fournisseurs, les lectures ERC-20 et l'envoi de transactions pré-signées. À utiliser lors de l'appel d'Ethereum, Arbitrum, Base, Optimism ou de toute chaîne EVM à partir d'une cartouche. Ne PAS utiliser pour les appels HTTPS génériques vers des API non-EVM – utilisez plutôt les appels https-outcalls.
