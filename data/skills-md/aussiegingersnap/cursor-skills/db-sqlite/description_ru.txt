Управление базой данных SQLite с помощью Prisma ORM, типобезопасных запросов и развертывание железной дороги с резервным копированием Litestream. Этот навык следует использовать при создании схем баз данных, написании миграций, управлении томами SQLite on Railway или устранении проблем с базой данных.
