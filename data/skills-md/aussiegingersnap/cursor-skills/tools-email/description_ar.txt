تسليم البريد الإلكتروني باستخدام Resend API. استخدم هذه المهارة عند تنفيذ تدفقات التحقق من البريد الإلكتروني، أو إعادة تعيين كلمة المرور، أو رسائل البريد الإلكتروني الخاصة بالمعاملات، أو تكوين DNS (SPF/DKIM/DMARC)، أو إعداد خادم Resend MCP، أو اتباع أفضل ممارسات البريد الإلكتروني لإمكانية التسليم. يتضمن أنماط وكيل Next.js 16، ومعالجة مستخدم OAuth مقابل كلمة المرور، وأنماط أمان الرمز المميز.
