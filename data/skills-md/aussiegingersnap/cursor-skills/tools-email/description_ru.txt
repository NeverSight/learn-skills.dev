Доставка электронной почты с использованием Resend API. Используйте этот навык при реализации потоков проверки электронной почты, сбросе пароля, транзакционных электронных письмах, настройке DNS (SPF/DKIM/DMARC), настройке сервера повторной отправки MCP или следовании рекомендациям по доставке электронной почты. Включает шаблоны прокси Next.js 16, обработку пользователей OAuth и паролей и шаблоны безопасности токенов.
