shadcn/studio-Komponentenbibliothek mit MCP-Integration, Theme-Generierung und Blockmustern. Diese Fähigkeit sollte beim Erstellen einer Benutzeroberfläche mit Shadcn-Komponenten, beim Auswählen von Dashboard-Layouts oder beim Generieren von Zielseiten verwendet werden. Kanonische Quelle für alle shadcn-basierten Arbeiten.
