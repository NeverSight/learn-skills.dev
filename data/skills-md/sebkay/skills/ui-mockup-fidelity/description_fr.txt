Implémentez l’interface utilisateur du site Web à partir de maquettes avec une haute fidélité visuelle. À utiliser lors de la création ou de la mise à jour de pages/composants à partir de maquettes où la mise en page, l'espacement, les couleurs et les arrière-plans de section doivent correspondre étroitement, en particulier dans les flux de travail CSS axés sur les utilitaires tels que Tailwind.
