تحليل الأوراق المالية، ونماذج العوامل، وبناء محفظة الأسهم. يُستخدم عندما يسأل المستخدم عن الأسهم، أو نسب تقييم الأسهم، أو طرق إنشاء المؤشرات، أو تحليل الأسلوب. يتم تشغيله أيضًا عندما يذكر المستخدمون "نسبة السعر إلى الربحية"، أو "النمو مقابل القيمة"، أو "ترجيح القيمة السوقية"، أو "تخصيص القطاع"، أو "تصنيف GICS"، أو "ربحية السهم"، أو "العوامل الفرنسية-Fama"، أو "CAPM"، أو "عائد الأرباح"، أو "نسبة PEG"، أو "EV/EBITDA"، أو اسأل عن العوامل التي تفسر عوائد الأسهم.
