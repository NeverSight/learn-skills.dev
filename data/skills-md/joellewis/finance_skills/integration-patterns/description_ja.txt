金融システムを接続する統合アーキテクチャ (API、FIX プロトコル、ISO 20022、イベント駆動パターン、バッチ フィード、冪等性、復元力) を設計および実装します。カストディアン統合パイプラインの構築、注文ルーティングのための FIX 接続の実装、ISO 20022 または SWIFT 移行メッセージングの設計、カストディアン フィードまたは EOD 調整のためのバッチ ファイル処理の構築、トランザクション API の冪等性の実装、再試行またはサーキット ブレーカー パターンの設計、異なるスキーマを持つシステム間のデータ マッピング、または偵察中断の原因となる統合エラーのトラブルシューティングを行う場合に使用します。トリガー: FIX プロトコル、ISO 20022、カストディアン フィード、バッチ処理、API 設計、冪等性、サーキット ブレーカー、デッド レター キュー、データ マッピング、統合アーキテクチャ、SWIFT 移行、mTLS、ファイル フィード、イベント駆動型、メッセージ ブローカー。
