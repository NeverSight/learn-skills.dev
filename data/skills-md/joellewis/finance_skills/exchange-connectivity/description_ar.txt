توجيه تصميم وإدارة اتصال مكان التداول والبنية التحتية لبيانات السوق. يُستخدم عند إنشاء جلسات FIX أو استكشاف الأخطاء وإصلاحها لتوجيه الطلب أو النسخ المسقط، ودمج بروتوكولات التبادل مثل OUCH أو ITCH أو PITCH أو Pillar، أو تصميم بنية تغذية بيانات السوق، أو التعامل مع توقف التداول أو قواطع الدائرة أو نطاقات LULD، ورسم خرائط الرموز عبر CUSIP/ISIN/SEDOL/FIGI، والتخطيط للموقع المشترك أو الاستضافة القريبة، وتصميم تجاوز الفشل وDR للاتصال بالتبادل، وتنفيذ القاعدة 15c3-5 ضوابط الوصول إلى السوق، إنشاء جدولة الجلسات لنوافذ ما قبل السوق وما بعده، أو حل فجوات أرقام تسلسل FIX، أو تخطيط البنية التحتية لتقارير CAT.
