تصميم وتشغيل برامج الخصوصية وأمن البيانات للشركات المسجلة في هيئة الأوراق المالية والبورصة (SEC) بموجب توقعات Reg S-P وReg S-ID وSEC للأمن السيبراني. يُستخدم عندما يسأل المستخدم عن إشعارات الخصوصية، أو قاعدة الضمانات، أو برامج منع سرقة الهوية، أو التزامات إشعارات الانتهاك، أو العناية الواجبة لأمان البائع، أو التخطيط للاستجابة للحوادث، أو تصنيف البيانات، أو الامتثال لقانون الخصوصية بالولاية. يتم التشغيل أيضًا عندما يذكر المستخدمون "تم الكشف عن بيانات العميل"، أو "هل نحتاج إلى إخطار العملاء بالانتهاك"، أو "الإعداد لامتحان الأمن السيبراني"، أو "تقييم مخاطر بائعي السحابة"، أو "تشفير بيانات العميل"، أو "سياسة أمان BYOD"، أو "قاعدة الأعلام الحمراء"، أو "متطلبات NY DFS 500"، أو السؤال عن كيفية التعامل مع حادث الأمن السيبراني.
