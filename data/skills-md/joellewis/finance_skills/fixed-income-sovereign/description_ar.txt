تحليل السندات الحكومية بما في ذلك سندات الخزانة الأمريكية ومنحنيات العائد والمدة والتحدب وTIPS. يُستخدم عندما يسأل المستخدم عن سندات الخزانة، أو الديون السيادية، أو إنشاء منحنى العائد، أو مخاطر أسعار الفائدة، أو المدة، أو التحدب، أو TIPS، أو معدلات التضخم التعادلية. يتم تشغيله أيضًا عندما يذكر المستخدمون "أذونات الخزانة"، أو "سندات الخزانة"، أو "تسعير السندات"، أو "العائد حتى تاريخ الاستحقاق"، أو "منحنى العائد المقلوب"، أو "الأسعار الآجلة"، أو "الأسعار الفورية"، أو "DV01"، أو "العوائد الحقيقية"، أو اسأل كيف تتفاعل السندات مع تغيرات أسعار الفائدة.
