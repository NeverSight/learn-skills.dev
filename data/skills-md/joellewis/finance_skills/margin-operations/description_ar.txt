توجيه الإقراض بالهامش ومتطلبات الهامش وعمليات استدعاء الهامش للوساطة والحسابات الاستشارية. يُستخدم عند حساب الهامش الأولي أو القوة الشرائية لـ Reg T، أو تحديد هامش الصيانة أو متطلبات المنزل، أو تقييم أهلية هامش المحفظة بموجب OCC TIMS، أو إنشاء أو حل مكالمات الهامش (مكالمة الاحتياطي الفيدرالي، أو مكالمة المنزل، أو مكالمة التبادل، أو مكالمة التجارة اليومية)، أو تصميم منطق الشلال للتصفية القسرية، أو هيكلة خطوط الائتمان المدعومة بالأوراق المالية (SBLOC)، أو حساب تأثير فائدة الهامش على العوائد، أو تقييم هامش المركز المركز، أو فهم قواعد المتداول اليومي النموذجي، أو مراجعة FINRA 4210 و Reg. متطلبات يو. يغطي أيضًا حسابات SMA وميكانيكا الهامش القصير.
