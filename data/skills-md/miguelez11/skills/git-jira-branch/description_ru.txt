Переключается на ветку git для получения билета Jira. Ищет существующее отделение с префиксом номера билета; если его нет, запускает `jiraSwitchToBranch <TICKET>`, чтобы создать новый и переключиться на него. Используйте, когда пользователь упоминает номер заявки Jira и хочет сменить ветку, начать работу над заявкой, проверить ветку заявки или говорит «переключиться на TICKET-1234».
