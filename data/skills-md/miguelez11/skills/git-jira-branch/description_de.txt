Wechselt zum Git-Zweig für ein Jira-Ticket. Sucht nach einer vorhandenen Filiale mit vorangestellter Ticketnummer; Wenn keine vorhanden ist, wird „jiraSwitchToBranch <TICKET>“ ausgeführt, um eine neue zu erstellen und zu einer neuen zu wechseln. Verwenden Sie diese Option, wenn der Benutzer eine Jira-Ticketnummer angibt und die Filiale wechseln, mit der Arbeit an einem Ticket beginnen, eine Ticketfiliale auschecken möchte oder „zu TICKET-1234 wechseln“ sagt.
