Cambia a la rama git para obtener un ticket de Jira. Busca una sucursal existente con el prefijo número de ticket; si no existe ninguno, ejecuta `jiraSwitchToBranch <TICKET>` para crear y cambiar a uno nuevo. Úselo cuando el usuario mencione un número de ticket de Jira y quiera cambiar de sucursal, comenzar a trabajar en un ticket, consultar una sucursal de ticket o diga "cambiar a TICKET-1234".
