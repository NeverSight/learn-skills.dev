قم بالتبديل إلى فرع git للحصول على تذكرة Jira. البحث عن فرع موجود مسبوقًا برقم التذكرة؛ إذا لم يكن هناك أي شيء، فسيتم تشغيل `jiraSwitchToBranch <TICKET>` لإنشاء واحدة جديدة والتبديل إليها. يُستخدم عندما يذكر المستخدم رقم تذكرة Jira ويريد تبديل الفروع، أو بدء العمل على تذكرة، أو التحقق من فرع التذاكر، أو يقول "التبديل إلى TICKET-1234".
