Schreiben Sie gut strukturierte Jira-Tickets oder verbessern Sie bestehende. Erzwingt eine konsistente Vorlage mit klarem Kontext, klarer Problem-/Anfragedefinition, Akzeptanzkriterien und Definition von erledigt. Verwenden Sie es, wenn der Benutzer ein Jira-Ticket, eine Story, einen Fehlerbericht oder eine Jira-Aufgabe schreiben, erstellen, entwerfen oder verbessern möchte – oder wenn es einem Ticket an Struktur oder Klarheit mangelt.
