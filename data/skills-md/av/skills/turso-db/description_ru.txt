Установите, настройте и работайте с Turso DB — внутрипроцессным SQLite-совместимым ядром реляционной базы данных, написанным на Rust. Используйте, когда пользователю необходимо (1) установить Turso DB, (2) создать или запросить базы данных с помощью оболочки CLI tursodb, (3) использовать Turso из JavaScript/Node.js через @tursodatabase/database, (4) работать с векторным поиском или внедрениями в Turso, (5) настроить полнотекстовый поиск с индексами FTS, (6) настроить транзакции, включая параллельные транзакции MVCC, (7) включить шифрование при хранении или (8) использовать данные изменения Capture (CDC) для ведения журнала аудита.
