Installez, configurez et travaillez avec Turso DB, un moteur de base de données relationnelle en cours compatible SQLite écrit en Rust. À utiliser lorsque l'utilisateur doit (1) installer Turso DB, (2) créer ou interroger des bases de données avec le shell CLI tursodb, (3) utiliser Turso à partir de JavaScript/Node.js via @tursodatabase/database, (4) travailler avec la recherche vectorielle ou les intégrations dans Turso, (5) configurer la recherche en texte intégral avec les index FTS, (6) configurer les transactions, y compris les transactions simultanées MVCC, (7) activer le cryptage au repos ou (8) utiliser la capture de données modifiées. (CDC) pour la journalisation d’audit.
