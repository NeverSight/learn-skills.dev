Instale, configure y trabaje con Turso DB, un motor de base de datos relacional en proceso compatible con SQLite escrito en Rust. Úselo cuando el usuario necesite (1) instalar Turso DB, (2) crear o consultar bases de datos con el shell CLI de tursodb, (3) usar Turso desde JavaScript/Node.js a través de @tursodatabase/database, (4) trabajar con búsqueda vectorial o incrustaciones en Turso, (5) configurar la búsqueda de texto completo con índices FTS, (6) configurar transacciones que incluyen transacciones concurrentes MVCC, (7) habilitar el cifrado en reposo o (8) usar Change Data Capture (CDC) para el registro de auditoría.
