Gestisce l'avvio e l'arresto del servizio Claude Session Dashboard. Questa abilità viene attivata quando l'utente richiede di aprire/chiudere/avviare/interrompere/attivare/disattivare il pannello di sessione di Claude, il dashboard di sessione o il dashboard di sessione di claude. Se il servizio non è installato, le operazioni di installazione e compilazione vengono eseguite automaticamente.
