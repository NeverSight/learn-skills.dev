Gère le démarrage et l'arrêt du service Claude Session Dashboard. Cette compétence est déclenchée lorsque l'utilisateur demande d'ouvrir/fermer/démarrer/arrêter/basculer le panneau de session Claude, le tableau de bord de session ou le tableau de bord de session Claude. Si le service n'est pas installé, les opérations d'installation et de génération sont effectuées automatiquement.
