مهارات احترافية في بناء الصفحات ومعالجة البيانات باستخدام واجهة MingDao Cloud HAP V3. عندما يذكر المستخدم متطلبات مثل "واجهة HAP V3"، و"HAP API"، و"مكالمات API"، و"واجهة البيانات"، و"Appkey"، و"Sign"، و"مصادقة الواجهة"، فيجب استخدام هذه المهارة. توفر هذه المهارة دليلاً كاملاً لاستخدام واجهة HAP V3: تكوين المصادقة، واستدعاءات API، واستخدام عامل التصفية، وعمليات البيانات، وما إلى ذلك. إذا قام المستخدم بالفعل بتكوين HAP MCP، فيجب على الذكاء الاصطناعي استخراج مفتاح المصادقة تلقائيًا من تكوين MCP.
