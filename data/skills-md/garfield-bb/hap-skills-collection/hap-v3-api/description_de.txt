Professionelle Kenntnisse im Erstellen von Seiten und Bearbeiten von Daten mithilfe der MingDao Cloud HAP V3-Schnittstelle. Immer wenn der Benutzer Anforderungen wie „HAP V3-Schnittstelle“, „HAP API“, „API-Aufrufe“, „Datenschnittstelle“, „Appkey“, „Sign“, „Schnittstellenauthentifizierung“ erwähnt, muss dieser Skill verwendet werden. Dieser Skill bietet eine vollständige Anleitung zur Verwendung der HAP V3-Schnittstelle: Authentifizierungskonfiguration, API-Aufrufe, Filternutzung, Datenoperationen usw. Wenn der Benutzer HAP MCP bereits konfiguriert hat, sollte die KI den Authentifizierungsschlüssel automatisch aus der MCP-Konfiguration extrahieren.
