Профессиональные навыки создания страниц и манипулирования данными с использованием интерфейса MingDao Cloud HAP V3. Каждый раз, когда пользователь упоминает такие требования, как «Интерфейс HAP V3», «HAP API», «Вызовы API», «Интерфейс данных», «Ключ приложения», «Подпись», «Аутентификация интерфейса», необходимо использовать этот навык. Этот навык предоставляет полное руководство по использованию интерфейса HAP V3: конфигурация аутентификации, вызовы API, использование фильтров, операции с данными и т. д. Если пользователь уже настроил HAP MCP, AI должен автоматически извлечь ключ аутентификации из конфигурации MCP.
