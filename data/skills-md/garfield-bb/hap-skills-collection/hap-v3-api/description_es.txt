Habilidades profesionales en la creación de páginas y manipulación de datos utilizando la interfaz MingDao Cloud HAP V3. Siempre que el usuario mencione requisitos como "interfaz HAP V3", "API HAP", "llamadas API", "interfaz de datos", "clave de aplicación", "firma", "autenticación de interfaz", se debe utilizar esta habilidad. Esta habilidad proporciona una guía completa para usar la interfaz HAP V3: configuración de autenticación, llamadas API, uso de filtros, operaciones de datos, etc. Si el usuario ya ha configurado HAP MCP, la IA debe extraer automáticamente la clave de autenticación de la configuración de MCP.
