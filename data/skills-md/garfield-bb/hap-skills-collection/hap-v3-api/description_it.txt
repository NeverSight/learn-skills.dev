Competenze professionali nella creazione di pagine e nella manipolazione dei dati utilizzando l'interfaccia MingDao Cloud HAP V3. Ogni volta che l'utente menziona requisiti quali "interfaccia HAP V3", "API HAP", "chiamate API", "interfaccia dati", "Appkey", "Firma", "autenticazione dell'interfaccia", è necessario utilizzare questa competenza. Questa competenza fornisce una guida completa all'utilizzo dell'interfaccia HAP V3: configurazione dell'autenticazione, chiamate API, utilizzo del filtro, operazioni sui dati, ecc. Se l'utente ha già configurato HAP MCP, l'IA dovrebbe estrarre automaticamente la chiave di autenticazione dalla configurazione MCP.
