MingDao Cloud HAP V3 インターフェイスを使用してページを構築し、データを操作する専門的なスキル。ユーザーが「HAP V3 インターフェイス」、「HAP API」、「API 呼び出し」、「データ インターフェイス」、「Appkey」、「署名」、「インターフェイス認証」などの要件に言及するときは常に、このスキルを使用する必要があります。このスキルは、認証構成、API 呼び出し、フィルターの使用法、データ操作など、HAP V3 インターフェイスの使用に関する完全なガイドを提供します。ユーザーがすでに HAP MCP を構成している場合、AI は MCP 構成から認証キーを自動的に抽出します。
