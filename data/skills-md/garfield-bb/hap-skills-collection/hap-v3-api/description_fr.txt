Compétences professionnelles dans la création de pages et la manipulation de données à l'aide de l'interface MingDao Cloud HAP V3. Chaque fois que l'utilisateur mentionne des exigences telles que "interface HAP V3", "API HAP", "appels API", "interface de données", "Appkey", "Sign", "authentification d'interface", cette compétence doit être utilisée. Cette compétence fournit un guide complet d'utilisation de l'interface HAP V3 : configuration de l'authentification, appels API, utilisation du filtre, opérations sur les données, etc. Si l'utilisateur a déjà configuré HAP MCP, l'IA doit automatiquement extraire la clé d'authentification de la configuration MCP.
