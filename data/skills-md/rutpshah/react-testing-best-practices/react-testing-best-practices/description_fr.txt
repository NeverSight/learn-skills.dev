Meilleures pratiques de test React à l'aide de React Testing Library, Vitest et Jest. À utiliser lors de l'écriture, de la révision ou de la génération de tests pour les composants React, les hooks, les fournisseurs de contexte, les interactions asynchrones ou les soumissions de formulaires. Déclencheurs sur des tâches telles que « écrire un test pour ce composant », « ajouter des tests unitaires », « tester ce hook », « se moquer de cet appel d'API », « améliorer la couverture des tests » ou « configurer Vitest ».
