Best Practices für React-Tests mit React Testing Library, Vitest und Jest. Wird zum Schreiben, Überprüfen oder Generieren von Tests für React-Komponenten, Hooks, Kontextanbieter, asynchrone Interaktionen oder Formularübermittlungen verwendet. Wird bei Aufgaben wie „Einen Test für diese Komponente schreiben“, „Komponententests hinzufügen“, „Diesen Hook testen“, „Diesen API-Aufruf verspotten“, „Testabdeckung verbessern“ oder „Vitest einrichten“ ausgelöst.
