Best practice per i test di React utilizzando la libreria di test React, Vitest e Jest. Da utilizzare durante la scrittura, la revisione o la generazione di test per componenti React, hook, provider di contesto, interazioni asincrone o invii di moduli. Si attiva su attività come "scrivi un test per questo componente", "aggiungi test unitari", "prova questo hook", "definisci questa chiamata API", "migliora la copertura del test" o "imposta Vitest".
