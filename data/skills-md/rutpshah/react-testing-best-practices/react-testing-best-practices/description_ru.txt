Лучшие практики тестирования React с использованием библиотеки тестирования React, Vitest и Jest. Используйте при написании, проверке или создании тестов для компонентов React, перехватчиков, поставщиков контекста, асинхронных взаимодействий или отправки форм. Запускается по таким задачам, как «написать тест для этого компонента», «добавить модульные тесты», «проверить этот хук», «имитировать этот вызов API», «улучшить тестовое покрытие» или «настроить Vitest».
