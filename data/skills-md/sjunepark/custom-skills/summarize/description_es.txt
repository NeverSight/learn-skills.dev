Utilice la CLI steipete/summarize para resumir URL, archivos locales, stdin, enlaces de YouTube, podcasts y medios con modelos LLM. Se activa cuando los usuarios solicitan instalar o ejecutar resumen, configurar claves API de modelo/proveedor, ajustar indicadores de salida (longitud/idioma/json/extract/slides), establecer valores predeterminados en ~/.summarize/config.json o solucionar problemas de errores de CLI de resumen.
