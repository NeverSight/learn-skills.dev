Utilisez la CLI steipete/summarize pour résumer les URL, les fichiers locaux, les stdin, les liens YouTube, les podcasts et les médias avec les modèles LLM. Déclenchez lorsque les utilisateurs demandent d'installer ou d'exécuter le résumé, de configurer les clés API du modèle/fournisseur, d'ajuster les indicateurs de sortie (longueur/langue/json/extract/slides), de définir les valeurs par défaut dans ~/.summarize/config.json ou de dépanner les erreurs CLI du résumé.
