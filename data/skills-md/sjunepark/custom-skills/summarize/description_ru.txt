Используйте интерфейс командной строки steipete/summarize для обобщения URL-адресов, локальных файлов, стандартного ввода, ссылок YouTube, подкастов и мультимедиа с помощью моделей LLM. Срабатывает, когда пользователи запрашивают установку или запуск суммирования, настройку ключей API модели/поставщика, настройку выходных флагов (длина/язык/json/extract/слайды), установку значений по умолчанию в ~/.summarize/config.json или устранение ошибок суммирования CLI.
