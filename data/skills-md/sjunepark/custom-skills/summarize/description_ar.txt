استخدم steipete/summarize CLI لتلخيص عناوين URL والملفات المحلية وstdin وروابط YouTube والبودكاست والوسائط باستخدام نماذج LLM. يتم تشغيله عندما يطلب المستخدمون تثبيت أو تشغيل التلخيص، أو تكوين مفاتيح API للنموذج/الموفر، أو ضبط علامات الإخراج (الطول/اللغة/json/الاستخراج/الشرائح)، أو تعيين الإعدادات الافتراضية في ~/.summarize/config.json، أو استكشاف أخطاء تلخيص سطر الأوامر وإصلاحها.
