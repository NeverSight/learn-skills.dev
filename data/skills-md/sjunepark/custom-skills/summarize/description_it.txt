Utilizza la CLI steipete/summarize per riepilogare URL, file locali, stdin, collegamenti YouTube, podcast e contenuti multimediali con i modelli LLM. Si attiva quando gli utenti chiedono di installare o eseguire il riepilogo, configurare le chiavi API del modello/provider, ottimizzare i flag di output (lunghezza/lingua/json/extract/slides), impostare le impostazioni predefinite in ~/.summarize/config.json o risolvere gli errori della CLI di riepilogo.
