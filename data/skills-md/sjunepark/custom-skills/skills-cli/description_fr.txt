Utilisez la CLI des compétences pour découvrir, installer, répertorier, mettre à jour, supprimer et initialiser les compétences pour Codex et Claude Code. À utiliser lorsque les utilisateurs demandent à gérer les compétences à partir de skills.sh, à restaurer à partir de fichiers verrouillés, à synchroniser les compétences à partir de node_modules ou à dépanner la portée de l'agent/de l'installation (projet ou global).
