Administre documentos, carpetas y contenido en DingTalk Cloud Docs. Úselo cuando los usuarios quieran crear documentos, buscar documentos, leer o escribir contenido de documentos o crear carpetas para organizar documentos. También se aplica cuando los usuarios mencionan palabras clave como documentos en la nube, documentos en línea, documentos de DingTalk o documentos de Ding. No active cuando los usuarios necesiten operar tablas multidimensionales, administrar programaciones, enviar mensajes o manejar flujos de trabajo de aprobación.
