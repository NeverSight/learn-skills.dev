Administre documentos, carpetas y contenido en DingTalk Cloud Docs. Úselo cuando el usuario desee crear, buscar, leer o escribir contenido de un documento, o crear carpetas para organizar documentos. También se aplica cuando el usuario menciona palabras clave como documentos en la nube, documentos en línea, documentos de DingTalk o documentos de Ding. No active cuando el usuario necesite operar tablas multidimensionales, administrar programaciones, enviar mensajes o manejar flujos de trabajo de aprobación.
