Verwalten Sie Dokumente, Ordner und Inhalte in DingTalk Cloud Docs. Verwenden Sie diese Option, wenn der Benutzer Dokumentinhalte erstellen, suchen, lesen oder schreiben oder Ordner zum Organisieren von Dokumenten erstellen möchte. Gilt auch, wenn der Benutzer Schlüsselwörter wie Cloud-Dokumente, Online-Dokumente, DingTalk-Dokumente oder Ding-Dokumente erwähnt. Nicht auslösen, wenn der Benutzer mehrdimensionale Tabellen bedienen, Zeitpläne verwalten, Nachrichten senden oder Genehmigungsworkflows bearbeiten muss.
