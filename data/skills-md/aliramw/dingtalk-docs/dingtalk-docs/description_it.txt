Gestisci documenti, cartelle e contenuti in DingTalk Cloud Docs. Utilizzarlo quando l'utente desidera creare, cercare, leggere o scrivere il contenuto del documento oppure creare cartelle per organizzare i documenti. Si applica anche quando l'utente menziona parole chiave come documenti cloud, documenti online, documenti DingTalk o documenti Ding. Non si attiva quando l'utente deve utilizzare tabelle multidimensionali, gestire pianificazioni, inviare messaggi o gestire flussi di lavoro di approvazione.
