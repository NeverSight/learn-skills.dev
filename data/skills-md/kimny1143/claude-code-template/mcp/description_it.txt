Una guida alla creazione e alla gestione dei server MCP (Model Context Protocol). Come aggiungere strumenti personalizzati. Quando utilizzarlo: (1) Quando si desidera aggiungere uno strumento personalizzato a Claude Code (2) Quando si collega a un servizio esterno (3) Quando si desidera creare uno strumento di automazione specifico del progetto (4) Quando si desidera sapere come configurare un server MCP. Esempi di trigger: "Crea server MCP" "Aggiungi strumento personalizzato" "Impostazioni MCP" "Trasforma API esterna in strumento" "Test runner MCP"
