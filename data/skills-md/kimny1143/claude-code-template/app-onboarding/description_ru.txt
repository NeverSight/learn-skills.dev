Навыки разработки и улучшения приложений. Поддерживает четыре формата: контрольный список + отчет об улучшении, дизайн потока (Mermaid), каркасные инструкции по макету экрана и генерацию кода React/SwiftUI. Варианты использования: (1) Разработка онбординга для новых приложений (2) Диагностика и улучшение существующего онбординга (3) Повышение показателей активации и удержания (4) Оптимизация опыта первого запуска. Примеры триггеров: «адаптация», «первый опыт», «первый запуск», «дизайн учебного пособия», «Aha Moment», «коэффициент отказа», «активация», «улучшение удержания», «первый UX», «прохождение», «поток нового пользователя», «поток адаптации», «FTUE», «экран начальной настройки».
