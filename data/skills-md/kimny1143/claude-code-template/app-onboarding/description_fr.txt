Compétences en matière de conception et d’amélioration d’intégration d’applications. Prend en charge quatre formats : liste de contrôle + rapport d'amélioration, document de conception de flux (Sirène), instructions filaires pour la disposition de l'écran et génération de code React/SwiftUI.  
Cas d'utilisation :  
(1) Conception d'intégration pour les nouvelles applications  
(2) Diagnostic et amélioration de l'onboarding existant  
(3) Augmenter les taux d’activation et de rétention  
(4) Optimiser l'expérience de première exécution  

Exemples de déclencheurs : « intégration », « première expérience », « première exécution », « conception de didacticiels », « Aha Moment », « taux d'abandon », « activation », « amélioration de la rétention », « première UX », « procédure pas à pas », « flux de nouveaux utilisateurs », « flux d'intégration », « FTUE », « écran de configuration initiale »
