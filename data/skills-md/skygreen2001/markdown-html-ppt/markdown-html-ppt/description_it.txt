Converti documenti o testo Markdown in presentazioni. Trasformalo prima in un PPT HTML interattivo tramite rivelare.js, quindi facoltativamente in un file PowerPoint. Supporta la ricca sintassi Markdown (grassetto, corsivo, codice, virgolette, blocchi di codice), cinque bellissimi temi e un algoritmo di interruzione di pagina intelligente. Formati di output: HTML (reveal.js) o PowerPoint (.pptx). Versione: v2.4.0
