Convertissez des documents ou du texte Markdown en présentations. Transformez-le d'abord en un HTML PPT interactif via Reveal.js, puis éventuellement convertissez-le en un fichier PowerPoint. Prend en charge la syntaxe Markdown riche (gras, italique, code, guillemets, blocs de code), 5 superbes thèmes et un algorithme de saut de page intelligent. Formats de sortie disponibles : HTML (reveal.js) ou PowerPoint (.pptx). Version : v2.4.0
