Konvertieren Sie Markdown-Dokumente oder -Texte in Präsentationen. Wandeln Sie sie zunächst über reveal.js in interaktive HTML-Diashows um und exportieren Sie sie anschließend optional nach PowerPoint. Unterstützt umfangreiche Markdown-Syntax (Fett, Kursiv, Code, Anführungszeichen, Codeblöcke), 5 elegante Themen und intelligente Paginierung. Ausgabeformate: HTML (reveal.js) oder PowerPoint (.pptx). Version: v2.4.0
