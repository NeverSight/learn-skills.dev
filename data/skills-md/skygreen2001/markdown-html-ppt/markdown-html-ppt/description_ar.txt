تحويل مستندات Markdown أو النص إلى عروض تقديمية. قم أولاً بتحويلها إلى ملفات HTML PPT تفاعلية عبر كشف.js، ثم قم بتحويلها بشكل اختياري إلى ملفات PowerPoint. يدعم بناء جملة Markdown الغني (غامق، مائل، رمز، علامات الاقتباس، كتل التعليمات البرمجية)، 5 سمات أنيقة، وخوارزمية ذكية لفاصل الصفحات. تنسيقات الإخراج: HTML (reveal.js) أو PowerPoint (.pptx). الإصدار: v2.4.0
