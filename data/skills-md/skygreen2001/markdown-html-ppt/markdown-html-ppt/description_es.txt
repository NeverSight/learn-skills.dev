Convierta documentos o texto de Markdown en presentaciones. Primero transfórmelos en HTML PPT interactivo a través de revele.js y luego, opcionalmente, conviértalos a un archivo de PowerPoint. Admite una rica sintaxis de Markdown (negrita, cursiva, código, comillas, bloques de código), 5 hermosos temas y un algoritmo inteligente de salto de página. Formatos de salida: HTML (reveal.js) o PowerPoint (.pptx). Versión: v2.4.0
