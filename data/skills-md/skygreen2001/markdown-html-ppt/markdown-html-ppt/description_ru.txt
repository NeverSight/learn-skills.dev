Преобразуйте документы или текст Markdown в презентации. Сначала преобразуйте их в интерактивные HTML-файлы PPT с помощью show.js, а затем (при необходимости) экспортируйте в PowerPoint. Поддерживает расширенный формат Markdown (жирный шрифт, курсив, код, кавычки, блоки кода), 5 элегантных тем и умные алгоритмы разрыва страниц. Форматы вывода: HTML (reveal.js) или PowerPoint (.pptx). Версия: v2.4.0
