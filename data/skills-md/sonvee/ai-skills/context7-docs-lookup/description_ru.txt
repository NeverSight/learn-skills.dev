Используйте Context7 для получения актуальной документации библиотеки. Активируется, когда пользователь спрашивает о библиотеках, фреймворках, ссылках на API или ему нужны примеры кода — особенно для вопросов по настройке, генерации кода или упоминаний конкретных фреймворков, таких как React, Vue, Next.js, Prisma, Supabase и т. д.
