Diese Fähigkeit wird vom KI-Agenten verwendet, um durch die Analyse von Git-Commit-Protokollen und Verzweigungsstrategien automatisch Pull-Request-Inhalte (PR) zu generieren. Es wird aktiviert, wenn der Benutzer „PR erstellen“, „Änderungen zusammenfassen“ anfordert oder einen Commit-Bereich für die Dokumentation angibt.
