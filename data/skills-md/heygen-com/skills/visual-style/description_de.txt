Erstellen, extrahieren und wenden Sie tragbare visuelle Designsysteme über visual-style.md-Dateien an. Wird verwendet, wenn: (1) ein Visual-Style.md-Designsystem von Grund auf erstellt wird, (2) ein visueller Stil aus einer Website-URL, einem Video oder einem PDF-Markenleitfaden extrahiert wird, (3) ein visueller Stil auf HeyGen-Videos, HTML-Folien, Figma oder paper.design angewendet wird, (4) die Galerie vorgefertigter visueller Stile durchsucht wird (Swiss, Saul Bass, Game Boy usw.), (5) der Benutzer „visueller Stil“, „Designsystem“, „Markenstil“ oder „Stil“ erwähnt Leitfaden“, (6) Gestaltung eines HeyGen-Videos mit einer einheitlichen Designsprache.
