Вывод LLM через платный API: завершение чата, совместимое с OpenAI, через прокси-провайдеров x402. Поддерживает Кими К2.5, МиниМакс М2.5. Использует инструмент x_paying для автоматических микроплатежей в долларах США (0,001–0,003 доллара США за звонок). Используйте, когда: (1) генерируете текст с помощью конкретной модели, (2) запускаете завершение чата через конечную точку LLM с оплатой за запрос, (3) сравниваете результаты разных моделей.
