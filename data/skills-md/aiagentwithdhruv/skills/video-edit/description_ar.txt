مجموعة أدوات كاملة لتحرير الفيديو - إزالة الصمت، والتسميات التوضيحية التلقائية، والاقتصاص العمودي، وقصاصة YouTube، والانتقالات ثلاثية الأبعاد، وضغط الوسائط الاجتماعية. يُستخدم عندما يطلب المستخدم تعديل الفيديو، أو إزالة فترات الصمت، أو إضافة تسميات توضيحية/ترجمات مصاحبة، أو الاقتصاص إلى مقاطع رأسية/شورتات قصيرة، أو تنزيل مقاطع YouTube، أو ضغط الفيديو، أو إنشاء مقاطع فيديو تشويقية.
