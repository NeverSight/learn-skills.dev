Прочтите этот навык, чтобы писать, генерировать или добавлять тестовый код. Срабатывает, когда: (1) пользователь просит написать, добавить, создать или сгенерировать тесты любого типа (юнит-тесты, интеграционные тесты, тесты API, тесты E2E, тесты драматурга, тесты Vitest); (2) у пользователя есть код с отсутствующими или нулевыми тестами, и он хочет получить покрытие; (3) пользователь только что внедрил новую услугу, конечную точку, функцию или модуль и нуждается в ее тестировании; (4) пользователь выполнил рефакторинг кода и хочет проверить с помощью тестов, что ничего не сломалось; (5) пользователь хочет улучшить или расширить существующее тестовое покрытие. Этот навык создает полные наборы тестов в форме пирамиды — читает код, выбирает только необходимые уровни (модуль/интеграция/API/E2E) и генерирует каждый необходимый файл (тесты схемы, сервисные тесты, фабрики, помощники, утилиты очистки, файлы спецификаций) в соответствии со строгими шаблонами Vitest и Playwright. Пропустите этот навык при отладке неудачных тестов, спрашивая, как использовать API или инструменты тестирования, объясняя концепции тестирования, настраивая средства запуска тестов, просматривая существующий тестовый код или мигрируя между платформами тестирования.
