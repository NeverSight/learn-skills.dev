閱讀此技能來編寫、產生或新增測試程式碼。在以下情況下觸發：(1) 使用者要求編寫、新增、建立或產生任何類型的測試（單元測試、整合測試、API 測試、E2E 測試、Playwright 測試、Vitest 測試）； (2) 使用者的程式碼缺少測試或零測試並希望覆蓋； (3) 使用者剛剛實現了新的服務、端點、功能或模組希望並驗證任何問題； (5)使用者希望改善或擴大現有的測試覆蓋範圍。此技能產生完整的金字塔形測試套件 - 讀取程式碼，僅選擇必要的層（單元/整合/API/E2E），並按照嚴格的 Vitest 和 Playwright 模式產生所需的每個檔案（模式測試、服務測試、工廠、說明程式、清理實用程式、規格文件）。在調試失敗的測試、詢問如何使用測試 API 或工具、解釋測試概念、配置測試運行器、檢查現有測試程式碼或在測試框架之間遷移時，請跳過此技能。
