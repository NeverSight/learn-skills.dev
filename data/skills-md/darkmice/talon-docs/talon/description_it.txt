Guida per l'utente del motore dati Talon Multimodal Fusion. Si attiva quando gli utenti devono sviluppare con il database Talon: incluse query SQL, archiviazione KV, ricerca vettoriale, dati di serie temporali, code di messaggi, ricerca full-text, geospaziale, database a grafo, motore AI (Sessione/Contesto/Memoria/RAG/Agente/Traccia). Si applica anche a: selezione dei moduli del motore Talon, utilizzo degli SDK Go/Python/Node.js/Java/.NET, creazione di pipeline RAG, memorizzazione nella cache degli strumenti dell'agente, gestione delle conversazioni, incorporamento della cache, query di fusione tra motori (GraphRAG, ricerca ibrida).
