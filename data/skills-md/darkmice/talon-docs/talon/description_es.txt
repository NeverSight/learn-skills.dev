Guía del usuario del motor de datos Talon Multimodal Fusion. Se activa cuando los usuarios necesitan desarrollar con la base de datos Talon: incluidas consultas SQL, almacenamiento KV, búsqueda vectorial, datos de series temporales, colas de mensajes, búsqueda de texto completo, geoespacial, base de datos de gráficos, motor de IA (Sesión/Contexto/Memoria/RAG/Agente/Trace). También aplicable a: selección de módulos del motor Talon, uso de SDK de Go/Python/Node.js/Java/.NET, creación de canalizaciones RAG, almacenamiento en caché de herramientas de agente, gestión de conversaciones, incrustación de caché, consultas de fusión entre motores (GraphRAG, búsqueda híbrida).
