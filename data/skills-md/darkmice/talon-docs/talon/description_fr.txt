Guide de l'utilisateur du moteur de données Multimodal Fusion Talon. Déclenché lorsque les utilisateurs doivent développer avec la base de données Talon : inclut les requêtes SQL, le stockage KV, la recherche vectorielle, les données de séries chronologiques, les files d'attente de messages, la recherche en texte intégral, la géospatiale, la base de données graphique, le moteur d'IA (Session/Contexte/Mémoire/RAG/Agent/Trace). S'applique également à : la sélection des modules du moteur Talon, l'utilisation des SDK Go/Python/Node.js/Java/.NET, la création de pipelines RAG, la mise en cache de l'outil Agent, la gestion des conversations, l'intégration du cache, les requêtes de fusion inter-moteurs (GraphRAG, recherche hybride).
