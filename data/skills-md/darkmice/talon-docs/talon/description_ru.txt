Руководство пользователя Talon Multimodal Fusion Data Engine. Срабатывает, когда пользователю необходимо разработать базу данных Talon: включая SQL-запросы, хранилище KV, векторный поиск, данные временных рядов, очереди сообщений, полнотекстовый поиск, геопространственную базу данных, графовую базу данных, механизм искусственного интеллекта (сеанс/контекст/память/RAG/агент/трассировка). Также применимо для: выбора модулей движка Talon, использования SDK Go/Python/Node.js/Java/.NET, построения конвейеров RAG, кэширования инструментов агента, управления диалогами, внедрения кэширования, запросов кросс-движка (GraphRAG, гибридный поиск).
