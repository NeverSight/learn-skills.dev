Configura actualizaciones de dependencias automatizadas de Renovate para proyectos de Tuist iOS. Detecta el estilo de integración (basado en Project.swift o Tuist/Package.swift), maneja paquetes de registro frente a paquetes de URL y crea renovate.json además de un flujo de trabajo de acciones de GitHub opcional. Úselo cuando desee automatizar los PR de aumento de dependencia para un proyecto de Tuist.
