Настраивает автоматические обновления зависимостей Renovate для проектов Tuist iOS. Определяет стиль интеграции (на основе Project.swift или Tuist/Package.swift), обрабатывает пакеты реестра и URL-адресов и создает renovate.json плюс дополнительный рабочий процесс GitHub Actions. Используйте, если вы хотите автоматизировать PR-запросы на увеличение зависимостей для проекта Tuist.
