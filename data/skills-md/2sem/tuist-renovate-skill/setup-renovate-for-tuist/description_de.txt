Richtet automatisierte Abhängigkeitsaktualisierungen von Renovate für Tuist-iOS-Projekte ein. Erkennt den Integrationsstil (Project.swift-basiert oder Tuist/Package.swift-basiert), verarbeitet Registrierungs- und URL-Pakete und erstellt renovate.json sowie einen optionalen GitHub Actions-Workflow. Verwenden Sie diese Option, wenn Sie Abhängigkeits-Bump-PRs für ein Tuist-Projekt automatisieren möchten.
