Configure les mises à jour automatiques des dépendances de Renovate pour les projets Tuist iOS. Détecte le style d'intégration (basé sur Project.swift ou Tuist/Package.swift), gère les packages de registre et d'URL et crée renovate.json ainsi qu'un flux de travail GitHub Actions facultatif. À utiliser lorsque vous souhaitez automatiser les PR d'augmentation des dépendances pour un projet Tuist.
