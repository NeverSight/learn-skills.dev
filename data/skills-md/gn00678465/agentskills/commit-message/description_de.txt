Analysieren Sie von Git bereitgestellte Änderungen und generieren Sie automatisch Commit-Nachrichten in traditionellem Chinesisch gemäß der Spezifikation für konventionelle Commits. Geeignet, wenn Sie konforme Commit-Nachrichten generieren müssen, z. B. „Helfen Sie mir, eine Commit-Nachricht zu schreiben“, „Generieren Sie ein Commit“ oder „Änderungen übermitteln“. Es prüft automatisch, ob Sie sich im Haupt-/Master-Zweig befinden und macht entsprechende Vorschläge.
