قم بتحليل التغييرات المرحلية لـ git وإنشاء رسائل الالتزام الصينية التقليدية تلقائيًا وفقًا لمواصفات الالتزامات التقليدية. مناسب عندما تحتاج إلى إنشاء رسائل التزام متوافقة، مثل "ساعدني في كتابة رسالة التزام" أو "إنشاء التزام" أو "إرسال التغييرات". سوف يتحقق تلقائيًا مما إذا كنت في الفرع الرئيسي/الرئيسي ويقدم الاقتراحات المناسبة.
