المساعدة في إنشاء أو تعديل أو عرض أو إدارة طلبات سحب GitHub (PRs). قم بتحليل التزامات الفرع وتغييراته تلقائيًا لإنشاء عناوين وأوصاف علاقات عامة صينية تقليدية موحدة. تتضمن حالات الاستخدام ما يلي: (1) إنشاء علاقات عامة جديدة من الفرع الحالي، (2) تعديل عنوان العلاقات العامة الحالية أو الوصف أو المراجعين أو التصنيفات، (3) التحقق من حالة العلاقات العامة للفرع (مفتوح، مدمج، مغلق)، (4) تلخيص تغييرات متعددة تلقائيًا. ينطبق على طلبات مثل "إنشاء علاقات عامة" أو "إنشاء طلب سحب" أو "ساعدني في فتح علاقات عامة" أو "تعديل محتوى العلاقات العامة". يتم تنفيذ العمليات عبر GitHub CLI (gh).
