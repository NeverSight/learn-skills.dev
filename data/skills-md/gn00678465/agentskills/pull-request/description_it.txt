Assistere nella creazione, modifica, visualizzazione o gestione delle richieste pull GitHub (PR). Analizza automaticamente i commit e le modifiche delle filiali per generare titoli e descrizioni PR tradizionali cinesi conformi. I casi d'uso includono: (1) creazione di un nuovo PR dal ramo corrente, (2) modifica del titolo, della descrizione, dei revisori o delle etichette di un PR esistente, (3) controllo dello stato PR di un ramo (aperto, unito, chiuso), (4) riepilogo automatico di più modifiche. Applicabile alle richieste contenenti frasi come "crea PR", "crea richiesta pull", "aiutami ad aprire una PR" o "modifica contenuto PR". Le operazioni vengono eseguite tramite la CLI GitHub (gh).
