Ayudar a crear, modificar, ver o administrar solicitudes de extracción de GitHub (PR). Analice automáticamente los cambios y las confirmaciones de sucursales para generar títulos y descripciones de relaciones públicas en chino tradicional que cumplan con las normas. Los casos de uso incluyen: (1) crear un nuevo RP a partir de la sucursal actual, (2) editar el título, la descripción, los revisores o las etiquetas de un RP existente, (3) verificar el estado del RP de una sucursal (abierto, fusionado, cerrado), (4) resumir automáticamente múltiples cambios. Aplicable cuando las solicitudes contienen frases como "crear PR", "crear solicitud de extracción", "ayúdame a abrir un PR" o "editar contenido de PR". Las operaciones se realizan a través de GitHub CLI (gh).
