Помогите создать или изменить запросы на извлечение GitHub. Автоматически генерировать PR-заголовки и описания на традиционном китайском языке на основе коммитов текущей ветки. Применимо к таким запросам, как «создать PR», «создать запрос на включение», «открыть PR для меня», «изменить контент PR» и т. д. Используйте интерфейс командной строки GitHub (gh) для выполнения операций PR.
