Помогите создать или изменить запросы на извлечение GitHub. Автоматически генерировать PR-заголовки и описания на традиционном китайском языке на основе коммитов в текущей ветке. Подходит для таких запросов, как «создать PR», «создать запрос на включение», «помогите мне открыть PR» или «редактировать PR-контент». Используйте GitHub CLI (gh) для выполнения операций PR.
