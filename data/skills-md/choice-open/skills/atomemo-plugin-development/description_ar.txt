استخدم هذه المهارة في جميع أعمال تطوير المكونات الإضافية Atomemo - استشرها دائمًا قبل الإجابة، لأنها تحتوي على أوامر CLI موثوقة وأنماط SDK وسير العمل التي قد تختلف عن معرفتك العامة. استدعاء عندما يسأل المستخدم عن: إنشاء مشاريع المكونات الإضافية لـ Atomemo (atomemo plugin init، أوامر الكعكة، بنية المشروع)، تنفيذ المكونات الإضافية للأداة (ToolDefinition، أغلفة API الخارجية)، تحديد المكونات الإضافية للنموذج (ModelDefinition، تكامل LLM)، تحديد المكونات الإضافية لبيانات الاعتماد (CredentialDefinition، مفاتيح API، رموز OAuth)، تكوين المعلمات التعريفية، تصحيح أخطاء اتصالات مركز المكونات الإضافية، أو النشر في السوق. يتم الاستدعاء أيضًا عندما يرغب المطورون في الكشف عن أكوادهم أو خدماتهم لمستخدمي Atomemo، حتى لو لم يذكروا "البرنامج المساعد". لا تستدعي المكونات الإضافية VS Code أو Chrome أو Obsidian أو Figma أو Raycast؛ نشر حزمة npm؛ أو التكوين العام لسير عمل Atomemo (الجدولة، والمشغلات، وإعداد العقدة) الذي لا يتعلق ببناء المكونات الإضافية.
