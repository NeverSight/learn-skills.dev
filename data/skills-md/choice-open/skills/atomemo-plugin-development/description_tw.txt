在所有 Atomemo 外掛程式開發工作中使用此技能 - 在回答之前請務必查閱它，因為它包含可能與您的常識不同的權威 CLI 命令、SDK 模式和工作流程。每當使用者詢問時呼叫：建立 Atomemo 外掛程式項目（atomemo 外掛程式初始化、bun 指令、專案結構）、實作工具外掛程式（ToolDefinition、外部 API 包裝器）、定義模型外掛程式（ModelDefinition、LLM 整合）、定義憑證外掛程式（CredentialDefinition、API 金鑰、OAuth 令牌）、設定值功能到外掛程式當開發人員想要向 Atomemo 用戶公開其程式碼或服務時也調用，即使他們沒有說「插件」。請勿呼叫：VS Code、Chrome、Obsidian、Figma 或 Raycast 外掛程式； npm 套件發佈；或一般 Atomemo 工作流程配置（調度、觸發器、節點設定），與建置外掛程式無關。
