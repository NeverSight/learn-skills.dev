Analysez et nettoyez l'espace disque macOS pour n'importe quel utilisateur. À utiliser lorsque l'utilisateur dit « libérer de l'espace », « disque plein », « nettoyer le stockage », « nettoyer mon Mac », « qu'est-ce qui prend de l'espace », « stockage presque plein », « supprimer les caches », « récupérer de l'espace disque », « vérifier l'utilisation du disque », « stockage faible ». Cibles : corbeille, téléchargements, photos, courrier, iMessage, musique, caches de navigateur, stockage cloud, sauvegardes iOS, caches d'applications, applications volumineuses, installateurs macOS, enregistrements d'écran, données système, outils de développement (Xcode, npm, Docker, Homebrew). Actions : scanner, analyser, identifier, nettoyer, supprimer, élaguer, récupérer, libérer.
