Aplique la inversión y el pensamiento pre-mortem siempre que el usuario solicite evaluar un plan, estrategia, arquitectura, característica o decisión antes de su ejecución, o cuando quiera realizar una prueba de estrés de algo que ya existe. Activadores de frases como "¿es una buena idea?", "¿qué podría salir mal?", "revisar este plan", "¿deberíamos hacer esto?", "¿nos falta algo?", "realizar una prueba de resistencia", "¿cuáles son los riesgos?" o cualquier solicitud para validar una decisión o un diseño. Utilice esta habilidad de manera proactiva: si el usuario está a punto de comprometerse con algo, se debe consultar esta habilidad incluso si no la solicita explícitamente.
