Wenden Sie das Denken zweiter Ordnung immer dann an, wenn der Benutzer eine Entscheidung trifft, eine Änderung vorschlägt oder eine Aktion bewertet – insbesondere eine Aktion, die offensichtlich gut oder offensichtlich sicher erscheint. Auslöser bei Formulierungen wie „Machen wir einfach Wird auch ausgelöst, wenn die Analyse erster Ordnung bereits durchgeführt wurde und der Benutzer tiefer gehen muss. Bevor eine wichtige Entscheidung getroffen wird, sollte das Denken zweiter Ordnung angewendet werden – warten Sie nicht, bis Sie gefragt werden.
