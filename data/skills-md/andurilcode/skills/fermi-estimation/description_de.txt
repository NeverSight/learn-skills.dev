Wenden Sie die Fermi-Schätzung immer dann an, wenn der Benutzer eine quantitative Antwort benötigt, ihm aber präzise Daten fehlen – oder wenn schnell eine Zahl benötigt wird, um eine Entscheidung zu treffen, einen Plan auf Plausibilität zu überprüfen oder die Machbarkeit zu bewerten. Auslöser bei Formulierungen wie „Wie lange wird das dauern?“, „Wie viel wird das kosten?“, „Ist das machbar?“, „Wie viele Dies ist auch ein Auslöser für die Plausibilitätsprüfung bestehender Schätzungen – eine Fermi-Berechnung, die um eine Größenordnung von einer offiziellen Schätzung abweicht, ist ein Signal, das es wert ist, untersucht zu werden. Verweigern Sie die Schätzung nicht, weil Ihnen Daten fehlen. Zerlegen und berechnen.
