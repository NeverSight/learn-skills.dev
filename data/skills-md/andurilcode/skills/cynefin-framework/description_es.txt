Aplique el marco Cynefin siempre que el usuario decida cómo abordar un problema, elija entre soluciones o pregunte por qué una solución anterior no funcionó. Desencadenantes de frases como "¿cómo deberíamos abordar esto?", "¿qué solución es mejor?", "probamos X pero no funcionó", "esto es complicado", "esto parece impredecible", "no sabemos qué hacer", "¿cuál es el proceso correcto aquí?", o al evaluar si aplicar las mejores prácticas, realizar experimentos o llamar a expertos. También se activa cuando el usuario está aplicando un proceso estructurado/rígido a algo que podría ser complejo o caótico; este es un desajuste común que Cynefin detecta. Utilice esta habilidad antes de recomendar cualquier enfoque de solución.
