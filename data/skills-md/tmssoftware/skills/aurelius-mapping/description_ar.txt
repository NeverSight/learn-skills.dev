قم بتعيين فئات دلفي إلى قاعدة بيانات علائقية باستخدام سمات TMS Aurelius ORM. يُستخدم عندما يطلب المستخدم إنشاء فئات كيانات، أو إضافة تعيين Aurelius إلى الفئات الموجودة، أو إصلاح سمات التعيين أو مراجعتها، أو شرح كيفية تعيين فئة ما، أو العمل مع الارتباطات، أو الوراثة، أو التعيين التلقائي، أو الحقول الفارغة، أو النقط الكبيرة، أو المعرفات المركبة. مشغلات لطلبات مثل "إنشاء كيانات Aurelius لـ..."، "تعيين هذه الفئة إلى Aurelius"، "إضافة تعيين ORM"، "إصلاح التعيين على هذه الفئة"، "كيف يمكنني تعيين واحد لأكثر في Aurelius".
