Mappez les classes Delphi à une base de données relationnelle à l'aide des attributs ORM de TMS Aurelius. À utiliser lorsque l'utilisateur demande de créer des classes d'entités, d'ajouter le mappage Aurelius aux classes existantes, de corriger ou de réviser les attributs de mappage, d'expliquer comment une classe est mappée ou de travailler avec des associations, l'héritage, le mappage automatique, les champs nullables, les blobs ou les identifiants composites. Se déclenche sur des requêtes telles que "créer des entités Aurelius pour...", "mapper cette classe à Aurelius", "ajouter un mappage ORM", "corriger le mappage sur cette classe", "comment mapper un un-à-plusieurs dans Aurelius".
