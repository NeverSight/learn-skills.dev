Сохраняйте, загружайте, обновляйте, удаляйте, объединяйте объекты сущностей и перемещайтесь по ним с помощью TMS Aurelius ORM (TObjectManager). Используйте, когда пользователь спрашивает, как сохранять или извлекать данные с помощью Aurelius, управлять временем жизни объекта, работать с ассоциациями или коллекциями во время выполнения, обрабатывать транзакции, использовать кэшированные обновления или управлять параллелизмом. НЕ охватывает запросы DSL или критерии/прогнозы (это отдельный навык). Запускается по таким запросам, как «сохранить объект с помощью Aurelius», «загрузить клиента из базы данных», «добавить элементы в коллекцию», «как обновить объект», «сбросить изменения», «удалить запись с помощью Aurelius», «объединить временный объект», «как транзакции работают в Aurelius», «пакетное обновление Aurelius».
