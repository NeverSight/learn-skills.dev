مهندس هندسة البيانات، والمراجع، وكتاب اللعب لـ Google Cloud Stack (BigQuery، وAirflow/Composer، وPub/Sub، وDataflow، وdbt، وDataForm، وDataplex). يُستخدم عند تصميم خطوط الأنابيب، أو تصميم مخططات BigQuery، أو مراجعة Airflow DAGs، أو تصميم التدفق، أو مراجعة طلبات سحب DE، أو كتابة نماذج dbt، أو تدقيق جودة البيانات، أو تشخيص فشل خطوط الأنابيب، أو تدقيق إنفاق Google Cloud Platform. المشغلات على: تصميم خط الأنابيب، مراجعة DAG، نمذجة BQ، عقد البيانات، دليل التشغيل، تحليل ما بعد الوفاة، بنية التدفق، dbt، جودة البيانات، تدقيق التكلفة، الحادث.
