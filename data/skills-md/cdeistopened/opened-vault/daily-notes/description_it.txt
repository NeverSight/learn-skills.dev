Gestisce le note giornaliere con formato M-G-AAAA (es. 1-1-2026.md). Da utilizzare quando l'utente chiede di visualizzare le note recenti, creare note giornaliere, leggere le note di oggi, riepilogare la settimana o fare riferimento a @note/ o date. Può recuperare le note degli ultimi 7 giorni. La posizione delle note è configurabile in Impostazioni > Archiviazione.
