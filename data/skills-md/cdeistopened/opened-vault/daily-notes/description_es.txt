Gestiona notas diarias con formato M-D-AAAA (por ejemplo, 1-1-2026.md). Úselo cuando el usuario solicite ver notas recientes, crear notas diarias, leer las notas de hoy, resumir la semana o hacer referencia a @notas/ o fechas. Puede recuperar notas de los últimos 7 días. La ubicación de las notas se puede configurar en Configuración > Almacenamiento.
