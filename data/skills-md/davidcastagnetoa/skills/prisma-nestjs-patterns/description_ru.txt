Покровители интеграции Prisma ORM с NestJS для производственных приложений. Используйте ПРОАКТИВНУЮ работу при работе с объектами Prisma, миграциями, PrismaService, атомными транзакциями (включая ВЫБЕРИТЕ ДЛЯ ОБНОВЛЕНИЯ), репозиториями в качестве адаптируемых инфраструктур в чистой архитектуре, оптимизированными запросами (N+1, включает, постраничный), исходными кодами и тестированием с помощью Prisma Mock. Активируйте только те команды, которые могут быть активированы: Prisma, PrismaService, prisma.$transaction, prismamigrate, Schema.prisma, PrismaClient, Prisma.validator, include/select, upsert, createMany или более качественная операция с базой данных с Prisma в проекте NestJS.
