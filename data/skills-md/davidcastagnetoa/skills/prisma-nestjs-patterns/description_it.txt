Patroni di integrazione di Prisma ORM con NestJS per le applicazioni di produzione. Usalo PROATTIVAMENTE quando lavori con schemi Prisma, migrazioni, PrismaService, transazioni atomiche (incluso SELECT FOR UPDATE), repository come adattatori di infrastrutture in Clean Architecture, query ottimizzate (N+1, include, impaginazione), seed e test con Prisma Mock. Attivare ogni volta che vengono visualizzate le parole chiave: Prisma, PrismaService, prisma.$transaction, prisma migrate, schema.prisma, PrismaClient, Prisma.validator, include/select, upsert, createMany o qualsiasi altra operazione di database con Prisma in un progetto NestJS.
