Ermöglicht umfangreiche interaktive UI-Komponenten in Chat-Antworten. Wenn Sie Fragen stellen, die eine strukturierte Eingabe erfordern (Multiple Choice, Richtig/Falsch, Formulare), betten Sie interaktive Blöcke ein, die kompatible Clients als native UI-Elemente rendern. Verwenden Sie diese Option, wenn der Benutzer nach Quizzen, Übungen, Umfragen oder einem anderen strukturierten Eingabeszenario fragt.
