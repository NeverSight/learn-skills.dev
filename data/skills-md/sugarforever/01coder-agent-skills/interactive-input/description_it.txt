Abilita ricchi componenti interattivi dell'interfaccia utente nelle risposte chat. Quando si presentano domande che richiedono input strutturati (scelta multipla, vero/falso, moduli), incorporare blocchi interattivi che i client compatibili visualizzano come elementi dell'interfaccia utente nativi. Da utilizzare quando l'utente richiede quiz, esercizi, sondaggi o qualsiasi scenario di input strutturato.
