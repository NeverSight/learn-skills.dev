Habilita componentes de interfaz de usuario interactivos enriquecidos en las respuestas del chat. Al presentar preguntas que requieren información estructurada (opción múltiple, verdadero/falso, formularios), incorpore bloques interactivos que los clientes compatibles representen como elementos nativos de la interfaz de usuario. Úselo cuando el usuario solicite pruebas, ejercicios, encuestas o cualquier escenario de entrada estructurada.
