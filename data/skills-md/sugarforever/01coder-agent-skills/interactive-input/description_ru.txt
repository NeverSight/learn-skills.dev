Включает богатые интерактивные компоненты пользовательского интерфейса в ответах чата. При представлении вопросов, требующих структурированного ввода (множественный выбор, правда/ложь, формы), встраивайте интерактивные блоки, которые совместимые клиенты отображают как собственные элементы пользовательского интерфейса. Используйте, когда пользователь запрашивает тесты, упражнения, опросы или любой сценарий структурированного ввода.
