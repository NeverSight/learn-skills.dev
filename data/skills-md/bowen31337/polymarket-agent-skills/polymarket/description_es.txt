SDK completo de Python para la plataforma de mercados de predicción Polymarket. Úselo cuando:
(1) Negociar en Polymarket: realizar, cancelar y gestionar órdenes
(2) Investigación de mercados: descubrimiento de eventos, análisis de precios, seguimiento de posiciones.
(3) Creación de robots comerciales: transmisión de WebSocket, fuentes de datos en tiempo real
(4) Gestión de fondos: depósitos y retiros a través de Bridge API
(5) Cualquier tarea que involucre mercados de predicción o API de Polymarket
