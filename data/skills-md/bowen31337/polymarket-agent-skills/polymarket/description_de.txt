Vollständiges Python-SDK für die Polymarket-Prognosemarktplattform. Verwenden Sie, wenn:
(1) Handel auf Polymarket – Platzierung, Stornierung, Verwaltung von Aufträgen
(2) Märkte erforschen – Ereignisse entdecken, Preise analysieren, Positionen verfolgen
(3) Aufbau von Trading-Bots – WebSocket-Streaming, Echtzeit-Datenfeeds
(4) Verwaltung von Geldern – Einzahlungen, Auszahlungen über die Bridge API
(5) Alle Aufgaben, die Prognosemärkte oder Polymarket-APIs betreffen
