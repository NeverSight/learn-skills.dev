SDK Python completo per la piattaforma dei mercati di previsione Polymarket. Utilizzare quando:
(1) Trading su Polymarket: immissione, annullamento, gestione degli ordini
(2) Ricercare i mercati: scoprire eventi, analizzare i prezzi, monitorare le posizioni
(3) Creazione di bot di trading: streaming WebSocket, feed di dati in tempo reale
(4) Gestione dei fondi: depositi, prelievi tramite Bridge API
(5) Qualsiasi attività che coinvolga mercati di previsione o API Polymarket
