كامل Python SDK لمنصة أسواق التنبؤ Polymarket. استخدم عندما:
(1) التداول على Polymarket - تقديم الطلبات وإلغائها وإدارتها
(2) البحث في الأسواق - اكتشاف الأحداث، وتحليل الأسعار، وتتبع المواقف
(3) بناء روبوتات التداول - تدفق WebSocket، وخلاصات البيانات في الوقت الفعلي
(4) إدارة الأموال - الإيداعات والسحوبات عبر Bridge API
(5) أي مهمة تتضمن أسواق التنبؤ أو واجهات برمجة تطبيقات Polymarket
