Полный Python SDK для платформы прогнозирования рынков Polymarket. Используйте, когда:
(1) Торговля на Полимаркете – размещение, отмена, управление ордерами
(2) Исследование рынков – обнаружение событий, анализ цен, отслеживание позиций.
(3) Создание торговых ботов — потоковая передача данных через WebSocket, потоки данных в реальном времени.
(4) Управление средствами – депозиты, снятие средств через Bridge API
(5) Любая задача, связанная с рынками прогнозов или API Polymarket.
