تم إطلاق النار عليه في عام 2017، وتم تنفيذه بواسطة Markdown، دان كو، AI استخدم هذه المهارة عندما يسأل المستخدم عن نشر حساب WeChat الرسمي، أو تحويل Markdown إلى WeChat HTML، أو تحميل الصور إلى WeChat، أو إنشاء مسودات، أو الكتابة بأسلوب Dan Koe، أو إزالة آثار الكتابة بالذكاء الاصطناعي (إضفاء طابع إنساني). يتم تشغيله أيضًا عندما يذكر المستخدم 微信排版، أو 公众号发文، أو 公众号格式، أو 文章排版成微信格式، أو 微信图文، أو 小绿书، أو أي سير عمل لمحتوى WeChat - حتى لو لم يذكروا صراحةً "wechat-studio".
