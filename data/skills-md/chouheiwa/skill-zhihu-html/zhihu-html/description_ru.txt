Преобразование Markdown в HTML, совместимый с редактором Zhihu. Скрипт обрабатывает детерминированное преобразование формата, а ИИ обрабатывает текстовое описание маркеров цитирования. Используйте этот навык, когда: пользователь просит сгенерировать формат Zhihu/HTML Zhihu, преобразовать Markdown в HTML, который можно вставить в редактор Zhihu, упоминает публикацию/набор текста/колонки Zhihu или ему необходимо преобразовать статьи со ссылками в формат поддержки цитат Zhihu.
