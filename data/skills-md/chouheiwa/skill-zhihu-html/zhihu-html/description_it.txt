Converti Markdown in HTML compatibile con l'editor Zhihu. Lo script gestisce la formattazione deterministica e l'intelligenza artificiale elabora le descrizioni del testo e dei dati dei contrassegni di citazione. Utilizza questa abilità quando: l'utente chiede di generare il formato Zhihu/Zhihu HTML, converte Markdown in HTML che può essere incollato nell'editor Zhihu, menziona la pubblicazione su Zhihu/formattazione/colonne o deve convertire articoli con collegamenti nel formato di citazione Zhihu.
