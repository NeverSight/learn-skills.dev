Konvertieren Sie Markdown in Zhihu-Editor-kompatibles HTML. Das Skript übernimmt die deterministische Formatierung und die KI verarbeitet die Datentextbeschreibungen von Zitatmarkierungen. Verwenden Sie diese Fähigkeit, wenn: Der Benutzer möchte das Zhihu-Format/Zhihu-HTML generieren, Markdown in HTML konvertieren, das in den Zhihu-Editor eingefügt werden kann, Zhihu-Beiträge/-Formatierungen/-Spalten erwähnt oder verlinkte Artikel in das Zhihu-Supp-Citation-Format konvertieren muss.
