Convierta Markdown a HTML compatible con el editor Zhihu. El script maneja el formato determinista y la IA procesa las descripciones de texto de datos de los marcadores de citas. Utilice esta habilidad cuando: el usuario solicite generar el formato Zhihu/HTML de Zhihu, convierta Markdown a HTML que se pueda pegar en el editor de Zhihu, mencione publicaciones/composición tipográfica/columnas de Zhihu o necesite convertir artículos vinculados al formato de citación de soporte de Zhihu.
