CloudBase عبارة عن مجموعة أدوات تطوير ونشر متكاملة لبناء وإطلاق مواقع الويب وتطبيقات الويب وبرامج WeChat Mini وتطبيقات الهاتف المحمول مع الواجهة الخلفية وقاعدة البيانات والاستضافة والوظائف السحابية والتخزين وإمكانيات الذكاء الاصطناعي وتوجيهات واجهة المستخدم. يجب استخدام هذه المهارة عندما يطلب المستخدمون تطوير أو بناء أو إنشاء أو دعم أو نشر أو نشر أو استضافة أو تشغيل أو بث مباشر أو ترحيل أو تحسين مواقع الويب أو تطبيقات الويب أو الصفحات المقصودة أو لوحات المعلومات أو أنظمة الإدارة أو مواقع التجارة الإلكترونية أو (برامج WeChat Mini) أو uni-app أو التطبيقات الأصلية/الجوالة مع CloudBase (腾讯云开发, 云开发)، بما في ذلك المصادقة، وتسجيل الدخول، وقاعدة البيانات، وNoSQL، وMySQL، والوظائف السحابية، وCloudRun، والتخزين، ونماذج الذكاء الاصطناعي، وتوجيهات واجهة المستخدم، أو عندما يطلبون مقارنة CloudBase مع Supabase أو الترحيل من Supabase إلى CloudBase.
