Habilidad de generación de imágenes: utilice esta habilidad cuando el usuario necesite generar imágenes, infografías visuales, crear imágenes o editar/modificar/ajustar imágenes existentes. Se basa en el servicio de generación de imágenes NanoBanana2 del sitio proxy API Easy de China (https://apiyi.com/), sin necesidad de acceder a Internet en el extranjero. Admite 14 relaciones de aspecto (`1:1`, `16:9`, `9:16`, `4:3`, `3:4`, `3:2`, `2:3`, `5:4`, `4:5`, `1:4`, `4:1`, `1:8`, `8:1`, `21:9`, etc.) y tres resoluciones (1K, 2K, 4K), y admite edición tanto de texto a imagen como de imagen a imagen. Basado en el modelo NanoBanana2 (modelo rápido) de Google, al que se accede a través del servicio de proxy nacional de API Easy.
