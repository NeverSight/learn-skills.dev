Fertigkeit zur Bildgenerierung: Verwenden Sie diese Fertigkeit, wenn der Benutzer Bilder oder visuelle Infografiken generieren, Bilder erstellen oder vorhandene Bilder bearbeiten/ändern/anpassen muss. Es basiert auf dem NanoBanana2-Bilderzeugungsdienst der chinesischen API Easy-Proxy-Site (https://apiyi.com/) und erfordert keinen externen Internetzugang. Es unterstützt 14 Seitenverhältnisse („1:1“, „16:9“, „9:16“, „4:3“, „3:4“, „3:2“, „2:3“, „5:4“, „4:5“, „1:4“, „4:1“, „1:8“, „8:1“, „21:9“ usw.) und drei Auflösungen (1K, 2K, 4K) und unterstützt sowohl die Text-zu-Bild- als auch die Bild-zu-Bild-Bearbeitung. Unterstützt durch das NanoBanana2-Modell (Schnellmodell) von Google, Zugriff über den inländischen Proxy-Dienst von API Easy.
