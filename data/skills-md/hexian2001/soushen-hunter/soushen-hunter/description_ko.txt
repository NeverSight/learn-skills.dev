고성능 Bing/Google 검색 엔진 스킬 - "검색 신 헌터"
딥 웹 크롤링 및 요소 추출을 위해 Playwright 하위 수준 API를 사용합니다.

특징:
1. Bing/Google 검색 실행 - 구조화된 결과(제목, 링크, 스니펫, 소스)를 반환합니다.
2. 심층 페이지 분석 – 모든 주요 페이지 요소(링크, 양식, 버튼, 스크립트, 메타데이터)를 추출합니다.
3. 구성 가능한 검색 엔진 – Bing과 Google 간 전환

트리거 조건:
- 사용자가 웹 검색을 필요로 하는 경우
- 페이지 구조 정보(링크, 양식 등)가 필요한 경우
- API 비용이 전혀 들지 않는 검색 솔루션이 필요한 경우

사용법:
- 기본 검색: ./soushen "검색 키워드" [--num N] [--engine ENGINE]
- 심층 분석: ./soushen --deep <대상 URL>
- 엔진 설정: ./soushen --set-default-engine bing|google
