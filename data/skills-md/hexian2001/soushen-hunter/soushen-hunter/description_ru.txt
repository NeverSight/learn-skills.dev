Высокоэффективные навыки Bing/поисковой системы Google – «Охотник на поисковых божеств»
Использует низкоуровневый API Playwright для глубокого веб-сканирования и извлечения элементов.

Особенности:
1. Выполнение поиска Bing/Google – возвращает структурированные результаты (заголовок, ссылка, фрагмент, источник).
2. Глубокий анализ страницы – извлекает все ключевые элементы (ссылки, формы, кнопки, скрипты, метаданные).
3. Переключаемая поисковая система — переключение между Bing и Google.

Условия триггера:
- Когда пользователю нужен веб-поиск
- При извлечении информации о структуре страницы (ссылки, формы и т. д.) требуется
- Когда необходимо решение для поиска с нулевой стоимостью API

Использование:
- Базовый поиск: ./soushen "ключевые слова поиска" [--num N] [--engine ENGINE]
- Глубокий анализ: ./soushen --deep <целевой URL>
- Установить движок: ./soushen --set-default-engine bing|google
