High-Performance Bing/Google Search Engine Skill – "Search Deity Hunter"
Uses the Playwright low-level API for deep web crawling and element extraction

Features:
1. Bing/Google search execution – returns structured results (title, link, snippet, source)
2. Deep page analysis – extracts all key elements (links, forms, buttons, scripts, metadata)
3. Configurable engine – switch between Bing and Google

Trigger conditions:
– When the user needs to perform a web search
– When extracting page structure information (links, forms, etc.) is required
– When a search solution without API cost is needed

Usage:
– Basic search: ./soushen "search keywords" [--num N] [--engine ENGINE]
– Deep analysis: ./soushen --deep <target URL>
– Set engine: ./soushen --set-default-engine bing|google
