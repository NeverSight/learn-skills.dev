# 地缘认知探索器

每日探索一个国家、区域或地缘专题，帮助你看懂国际新闻背后的地理逻辑。

## 核心理念

- 地理是理解国际新闻的**底层操作系统**
- 每个国家/区域都有其**地缘"人设"**——理解它，就能预判新闻
- 重点是「关系」和「为什么」，而非「是什么」
- 目标：读完后看到相关新闻能说出「原来如此」

## 语言风格

### ✅ 接地气的表达（推荐）

| 学术表达 | 接地气表达 |
|----------|------------|
| 地缘政治博弈 | 大国掰手腕 |
| 战略缓冲区 | 夹心饼干 |
| 经济依存度高 | 离了它活不了 |
| 多边外交平衡 | 左右逢源/两头通吃 |
| 历史遗留问题 | 历史包袱/老账 |
| 地区霸权 | 区域一哥 |
| 代理人战争 | 借刀砍人 |

### ❌ 避免的表达

- 「从XX视角审视」「基于XX框架分析」
- 「具有重要的战略意义」（太空洞）
- 「不可忽视的地缘价值」（废话）
- 过多使用「XX化」「XX性」等学术后缀

### 语气示例

> ❌ 「土耳其凭借其独特的地理位置，在欧亚地缘博弈中扮演着关键角色。」
>
> ✅ 「土耳其就是那个能同时跟美国、俄罗斯、欧盟讨价还价的'地缘中介'——因为它卡着博斯普鲁斯海峡，谁都得给三分面子。」

## 查重流程

1. 读取 `geo_explorer_history.json`（若不存在则视为空数组）
2. 检查候选主题是否已在历史中
3. 若重复：切换主题（最多3次）
4. 若3次后仍重复：提示用户指定感兴趣的区域
5. 若成功：将主题追加到历史并写回

## 输出保存

1. **保存目录**：`geo_explorer_outputs/`
2. **文件命名**：`{YYYY-MM-DD}_{主题简写}.md`
3. 保存完成后告知用户文件路径

## 主题选择策略（重要！）

### 选题优先级（按顺序尝试）

```
🥇 第一优先：近期热点
     ↓ 若无合适热点
🥈 第二优先：灵感池外的新鲜选题（加分项！）
     ↓ 若需要参考
🥉 第三优先：灵感池内的经典选题
     ↓ 若都覆盖了
🔍 最后：探索冷门未知领域
```

### 选题标准（满足至少2项）

| 标准 | 定义 | 示例 |
|------|------|------|
| 🔥 新闻热点 | 近3个月国际新闻频繁出现 | 加沙、红海危机、台海 |
| 📍 地缘关键 | 战略位置重要或资源/通道价值高 | 霍尔木兹海峡、苏伊士运河 |
| 🤔 被误解 | 大众认知与实际情况有偏差 | 伊朗不是阿拉伯国家 |
| 🇨🇳 与中国相关 | 一带一路、贸易、地缘博弈 | 巴基斯坦、斯里兰卡 |
| ❄️ 冷门但重要 | 很少被讨论但影响深远 | 吉布提、格鲁吉亚 |

### 热点判断流程

执行skill时，**必须**先调用搜索工具获取热点：

```
步骤1：调用 search_web 工具
        查询："本周国际新闻热点 地缘政治" 或 "今日国际新闻"

步骤2：从搜索结果中提取 2-3 个地缘热点话题
        - 识别涉及的国家/区域/通道
        - 识别地缘关键词（冲突、谈判、制裁、军事...）

步骤3：检查是否已在历史记录中
        - 若未讲过：选择该主题
        - 若已讲过：进入第二优先级（池外选题）

步骤4：确定最终选题并记录
```

## 限制性原则

### ✅ 必须做

- 提供「原来如此」的认知增量
- 用大白话解释复杂地缘关系
- 标注争议性观点为「个人解读」
- 新闻事件需真实可查
- 数据需标注来源或注明「据估算」

### ❌ 绝不做

- 使用学术黑话或过于专业的术语
- 在领土争议上表达明确立场（如「XX自古以来属于YY」）
- 编造数据或事件
- 输出超过1500字
- 使用 Mermaid 或其他 Telegram 不支持的格式

### 示例2：专题型

# 🌍 马六甲海峡

> 💡 一句话记住：全球贸易的「咽喉」，中国能源的「命门」

---

## 📍 地理速写

马六甲海峡位于马来半岛和苏门答腊岛之间，连接印度洋和太平洋。全长约1000公里，最窄处仅2.8公里（新加坡海峡段）。

这是全球最繁忙的航运通道之一，每年约10万艘船舶通过，承载全球25%的海运贸易。对中国而言更关键——80%进口石油经此运输。

---

  ↓ 🛡️          ↓ 🛡️           ↓ 💰

## ⚔️ 地缘人设

马六甲海峡是教科书级的「咽喉要道」：

**中国的焦虑**：「马六甲困局」是中国战略界的经典命题——万一美国封锁海峡，中国的能源进口立刻断裂。这也是为什么中国砸重金搞「一带一路」，建中巴经济走廊（瓜达尔港）、中缅油气管道——都是为了绕开马六甲。

**新加坡的崛起**：一个弹丸小国凭什么人均GDP全球前三？答案就在这条海峡。新加坡卡住海峡东端最窄处，靠港口、炼油、金融服务吃透了「过路费」红利。

**大国博弈场**：美国在新加坡樟宜海军基地有驻军，印度在安达曼群岛布防，中国在巴基斯坦和缅甸找替代路线——围绕一条海峡的暗战从未停止。

---

## 🔥 新闻关联

**胡塞武装袭击红海（2024年）**：也门胡塞武装用导弹袭击红海商船，多家船公司改道好望角。为什么这事和马六甲有关？因为它证明了「咽喉要道被卡」的恐怖——中国一直担心的马六甲困局，在红海先上演了一遍预演。

## 🎭 人文冷知识

- 「马六甲」这个名字可能来自马来语「满剌加」，意思是「被放逐者」——传说这里是一位落难王子的避难地
- 海峡因为太浅（平均水深25米），超大型油轮必须绕道龙目海峡
