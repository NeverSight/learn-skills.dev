Cree, desarrolle, depure y publique aplicaciones para la plataforma de hogar inteligente Homey utilizando el SDK de aplicaciones Homey v3. Utilice esta habilidad siempre que el usuario mencione Homey, Homey Pro, Homey Cloud, desarrollo de aplicaciones Homey, Homey CLI, Homey Compose, tarjetas Homey Flow, controladores de dispositivos Homey, capacidades de Homey, widgets de Homey, emparejamiento de Homey o cualquier desarrollo de aplicaciones para el hogar inteligente dirigido a la plataforma Homey. También se activa cuando el usuario desea integrar una marca de dispositivo o un servicio en la nube con Homey, crear automatizaciones de flujo, crear widgets de panel o publicar en la tienda de aplicaciones de Homey. Incluso si el usuario simplemente dice "Quiero crear una aplicación Homey" o menciona un dispositivo que desea controlar con Homey, utilice esta habilidad.
