يمكنك إنشاء تطبيقات ودعمها وتصحيح أخطائها ونشرها لمنصة Homey للمنزل الذكي باستخدام Homey Apps SDK v3. استخدم هذه المهارة عندما يذكر المستخدم Homey، أو Homey Pro، أو Homey Cloud، أو تطوير تطبيق Homey، أو Homey CLI، أو Homey Compose، أو بطاقات Homey Flow، أو برامج تشغيل الأجهزة Homey، أو إمكانيات Homey، أو أدوات Homey، أو إقران Homey، أو أي تطوير لتطبيق منزلي ذكي يستهدف منصة Homey. يتم التشغيل أيضًا عندما يريد المستخدم دمج علامة تجارية للجهاز أو خدمة سحابية مع Homey، أو إنشاء عمليات أتمتة للتدفق، أو إنشاء أدوات لوحة المعلومات، أو النشر على Homey App Store. حتى إذا قال المستخدم فقط "أريد إنشاء تطبيق Homey" أو ذكر جهازًا يريد التحكم فيه باستخدام Homey، فاستخدم هذه المهارة.
