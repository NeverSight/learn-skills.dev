Erstellen, erstellen, debuggen und veröffentlichen Sie Apps für die Homey-Smart-Home-Plattform mit dem Homey Apps SDK v3. Verwenden Sie diesen Skill immer dann, wenn der Benutzer Homey, Homey Pro, Homey Cloud, Homey-App-Entwicklung, Homey CLI, Homey Compose, Homey Flow-Karten, Homey-Gerätetreiber, Homey-Funktionen, Homey-Widgets, Homey-Pairing oder eine beliebige Smart-Home-App-Entwicklung erwähnt, die auf die Homey-Plattform abzielt. Wird auch ausgelöst, wenn der Benutzer eine Gerätemarke oder einen Cloud-Dienst mit Homey integrieren, Flow-Automatisierungen erstellen, Dashboard-Widgets erstellen oder im Homey App Store veröffentlichen möchte. Auch wenn der Benutzer nur sagt „Ich möchte eine Homey-App erstellen“ oder ein Gerät erwähnt, das er mit Homey steuern möchte, nutzen Sie diesen Skill.
