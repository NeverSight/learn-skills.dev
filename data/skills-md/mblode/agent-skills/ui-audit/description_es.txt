Audita la calidad de la interfaz de usuario web en términos de accesibilidad, interacción, formularios, tipografía, navegación, diseño, rendimiento, movimiento y microcopia. Úselo al revisar o perfeccionar la interfaz de usuario antes de fusionarla o lanzarla, o cuando el usuario solicite una auditoría de interfaz de usuario, experiencia de usuario o accesibilidad.
