يُنشئ مخططات خريطة ذهنية لحورية البحر من قواعد التعليمات البرمجية أو المواضيع أو الملفات أو المحادثات. يلخص بصريا المواد المصدر كرسومات بيانية متفرعة. استخدمها عندما يُطلب منك إنشاء خريطة ذهنية لحورية البحر، أو تصور موضوع، أو رسم قاعدة بيانات، أو تلخيص ملف كرسم تخطيطي، أو إنشاء خريطة مفاهيم، أو إنشاء نظرة عامة مرئية.
