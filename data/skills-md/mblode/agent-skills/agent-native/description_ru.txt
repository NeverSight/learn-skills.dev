Разрабатывает приложения, ориентированные на агенты, в которых агенты являются первоклассными гражданами с полной четностью инструментов, атомарными примитивами и явными сигналами завершения. Охватывает проектирование инструментов, внедрение контекста, взаимодействие агента с пользовательским интерфейсом и шаблоны мобильных контрольных точек/возобновления. Используйте при проектировании агентной системы, проектировании инструментальных поверхностей, создании пользовательского интерфейса с поддержкой агентов, реализации шаблонов context.md или при запросе «как мне сделать мое приложение нативным для агента».
