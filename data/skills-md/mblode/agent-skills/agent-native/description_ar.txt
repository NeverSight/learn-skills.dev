يصمم تطبيقات الوكيل الأصلية حيث يكون الوكلاء مواطنين من الدرجة الأولى مع تكافؤ كامل للأداة، وأوليات ذرية، وإشارات إكمال واضحة. يغطي تصميم الأداة، وحقن السياق، والاتصال بين الوكيل وواجهة المستخدم، وأنماط نقاط التفتيش/السيرة الذاتية المتنقلة. يُستخدم عند تصميم نظام وكيل، أو تصميم أسطح الأدوات، أو إنشاء واجهة مستخدم مدركة للوكيل، أو تنفيذ أنماط context.md، أو السؤال "كيف أجعل وكيل التطبيق الخاص بي أصليًا."
