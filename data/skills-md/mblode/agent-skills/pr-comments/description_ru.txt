Сортирует и разрешает комментарии по PR-обзорам от людей и ботов. Извлекает неразрешенные потоки, дедуплицирует выводы ботов, классифицирует их по серьезности, составляет план исправлений для утверждения, затем устраняет реальные проблемы и устраняет шум. Используйте, когда вас просят исправить комментарии по связям с общественностью, разрешить обратную связь по отзывам, сортировать ветки отзывов, обработать комментарии ботов или рассмотреть результаты проверки по связям с общественностью.
