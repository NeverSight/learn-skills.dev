فرز وحل تعليقات مراجعة العلاقات العامة من البشر والروبوتات. جلب المواضيع التي لم يتم حلها، وإلغاء تكرار نتائج الروبوت، والتصنيف حسب الخطورة، وكتابة خطة إصلاح للموافقة عليها، ثم إصلاح المشكلات الحقيقية وحل التشويش. يُستخدم عندما يُطلب منك إصلاح تعليقات العلاقات العامة، أو حل تعليقات المراجعة، أو فرز سلاسل المراجعة، أو التعامل مع تعليقات الروبوت، أو معالجة نتائج مراجعة العلاقات العامة.
