Fornisce la direzione della progettazione e il sistema visivo per l'interfaccia utente Web. Definisce il layout, la tipografia, il colore e l'estetica generale per le esperienze di prodotto (SaaS/amministrazione/con elevato contenuto di dati) o di marketing/marchio. Da utilizzare quando si sceglie la direzione visiva, si selezionano tavolozze di colori e caratteri, si stabiliscono modelli di layout o si avvia un nuovo sistema di progettazione dell'interfaccia utente.
