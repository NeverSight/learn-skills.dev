Écrit ou réécrit les fichiers README.md adaptés au type de projet (CLI, bibliothèque, application, framework, monorepo ou ensemble de compétences). Découvrez le contexte du projet, sélectionne la bonne structure, rédige section par section et valide par rapport aux contrôles de qualité. À utiliser lors de la création d'un README, de l'écriture d'un README à partir de zéro, de la réécriture d'un mauvais README, du démarrage de la documentation d'un projet ou de la demande « d'écrire un README pour ce projet ».
