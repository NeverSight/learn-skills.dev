Записывает или перезаписывает файлы README.md в соответствии с типом проекта (CLI, библиотека, приложение, платформа, монорепозиторий или набор навыков). Обнаруживает контекст проекта, выбирает правильную структуру, записывает раздел за разделом и проверяет его на соответствие проверкам качества. Используйте при создании README, написании README с нуля, переписывании неправильного README, начальной загрузке документации проекта или при запросе «напишите README для этого проекта».
