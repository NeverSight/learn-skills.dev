Arize AX トレースをアプリケーションに追加するときにこのスキルを呼び出します。 Agent-Assisted Tracing の 2 フェーズ フローに従います。コードベース (読み取り専用) を分析し、ユーザーの確認後にインストルメンテーションを実装します。 When the app uses LLM tool/function calling, add manual CHAIN + TOOL spans so traces show each tool's input and output. https://arize.com/docs/ax/alyx/tracing-assistant および https://arize.com/docs/PROMPT.md を活用します。
