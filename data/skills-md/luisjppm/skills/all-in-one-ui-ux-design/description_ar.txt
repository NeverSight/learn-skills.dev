نظام تشغيل موحد لواجهة المستخدم/تجربة المستخدم لتخطيط وتصميم وتنفيذ ومراجعة الواجهات الحديثة عبر مجموعات الويب والأجهزة المحمولة. يُستخدم عندما تتضمن الطلبات إنشاء الصفحات/المكونات أو إعادة تصميمها أو مراجعتها أو إصلاحها أو تحسينها (الصفحات المقصودة ولوحات المعلومات وتطبيقات SaaS ولوحات الإدارة والتجارة الإلكترونية والمحافظ والمدونات وشاشات الهاتف المحمول) وتتطلب توجيهًا مرئيًا قويًا بالإضافة إلى إمكانية الوصول والتفاعل والأداء وجودة الاستجابة.
