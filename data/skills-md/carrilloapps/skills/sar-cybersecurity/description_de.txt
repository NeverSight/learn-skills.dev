Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine Sicherheitsanalyse, Schwachstellenbewertung, Sicherheitsüberprüfung oder irgendeine Form eines Sicherheitsbewertungsberichts (SAR) für eine Codebasis, Infrastruktur, API, Datenbank oder ein System anfordert. Zu den Auslösern gehören: „Meinen Code prüfen“, „Sicherheitsprobleme finden“, „Sicherheitsüberprüfung durchführen“, „SAR generieren“, „Auf Schwachstellen prüfen“, „Ist dieser Code sicher“ oder jede Anfrage, die die Bewertung der Sicherheitslage eines Projekts beinhaltet. Wird auch ausgelöst, wenn der Benutzer Quellcode, Konfigurationsdateien, Umgebungsvariablen oder Architekturdiagramme hochlädt oder darauf verweist und um eine Sicherheitsmeinung bittet. NICHT für generische Codierungsaufgaben, Codeüberprüfungen mit Schwerpunkt auf Qualität statt Sicherheit oder Leistungsoptimierung verwenden, es sei denn, ein Sicherheitsaspekt ist ausdrücklich vorhanden.
