Wenn der Benutzer eine DumplingAI-Funktion oder einen Anbieterendpunkt vom Terminal aus finden, prüfen oder ausführen möchte. Wird auch verwendet, wenn der Benutzer „dumplingai“, „Katalogsuche“, „Katalogdetails“, „dumplingai run“, „balance“, „usage“, „transactions“, „view-config“ erwähnt oder einen DumplingAI-gestützten Workflow testen möchte, ohne ein direktes Anbieter-SDK zu verkabeln. Verwenden Sie dies immer dann, wenn Ausführung, Inspektion oder CLI-Prüfungen auf Kontoebene über DumplingAI statt über eine reine API-Integration erfolgen sollen. Informationen zur Auswahl der richtigen Funktion oder des richtigen Endpunkts finden Sie zunächst unter discovering-dumplingai-apis. Für engere berufsbasierte Einstiegspunkte siehe dumplingai-web-research, dumplingai-scraping-extraction, dumplingai-transcripts-media und dumplingai-seo-data.
