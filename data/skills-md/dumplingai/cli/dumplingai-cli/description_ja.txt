ユーザーが端末から DumplingAI 機能またはプロバイダー エンドポイントを検索、検査、または実行したい場合。また、ユーザーが「dumplingai」、「カタログ検索」、「カタログの詳細」、「dumplingai run」、「balance」、「usage」、「transactions」、「view-config」に言及する場合、または直接ベンダー SDK を接続せずに DumplingAI を利用したワークフローをテストしたい場合にも使用します。実行、検査、またはアカウントレベルの CLI チェックを生の API 統合ではなく DumplingAI を通じて行う必要がある場合は、これを使用します。最初に適切な機能またはエンドポイントを選択するには、discovering-dumplingai-apis を参照してください。より狭いジョブベースのエントリ ポイントについては、dumplingai-web-research、dumplingai-scraping-extraction、dumplingai-transcripts-media、および dumplingai-seo-data を参照してください。
