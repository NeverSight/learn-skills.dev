當使用者想要從終端機尋找、檢查或執行 DumplingAI 功能或提供者端點時。當使用者提到「dumplingai」、「目錄搜尋」、「目錄詳細資料」、「dumplingai run」、「balance」、「usage」、「transactions」、「view-config」時，或是想要測試 DumplingAI 支援的工作流程而不直接連接供應商 SDK 時，也可以使用。每當執行、檢查或帳戶級 CLI 檢查應透過 DumplingAI 而不是原始 API 整合進行時，請使用此選項。若要先選擇正確的功能或端點，請參閱 discovery-dumplingai-apis。對於更窄的基於工作的切入點，請參閱 dumplingai-web-research、dumplingai-scraping-extraction、dumplingai-transcripts-media 和 dumplingai-seo-data。
