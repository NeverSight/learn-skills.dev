يدير العلاقة المنظمة بين كائنات ScriptableObjects (البيانات) والبنية الجاهزة (Logic/Visuals) في Unity 6. ويتبع نمط Bridge "SO-to-Mono" لضمان استقلال المثيل (على سبيل المثال، الصحة الفردية للروبوتات المتطابقة) مع الحفاظ على بنية نظيفة تعتمد على البيانات. يُستخدم عندما: (1) إنشاء أنواع وحدات/عناصر جديدة، (2) توصيل بيانات SO إلى السلوكيات الأحادية الجاهزة، (3) حل أخطاء مشاركة البيانات حيث يؤثر تغيير SO واحد على جميع المثيلات.
