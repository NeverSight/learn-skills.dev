Gestisce la relazione strutturata tra ScriptableObjects (dati) e prefab (logica/visualizzazione) in Unity 6. Segue il modello Bridge "SO-to-Mono" per garantire l'indipendenza dell'istanza (ad esempio, la salute individuale per robot identici) mantenendo un'architettura pulita e basata sui dati. Utilizzare quando: (1) Creare nuovi tipi di unità/articoli, (2) Collegare i dati SO a Prefab MonoBehaviours, (3) Risolvere bug di condivisione dati in cui la modifica di un SO influisce su tutte le istanze.
