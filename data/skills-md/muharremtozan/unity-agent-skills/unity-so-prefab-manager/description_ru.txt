Управляет структурированными отношениями между ScriptableObjects (данные) и префабами (логика/визуальные элементы) в Unity 6. Следует шаблону моста «SO-to-Mono», чтобы обеспечить независимость экземпляров (например, индивидуальную работоспособность идентичных роботов), сохраняя при этом чистую, управляемую данными архитектуру. Используйте при: (1) создании новых типов юнитов/предметов, (2) подключении данных SO к Prefab MonoBehaviours, (3) устранении ошибок совместного использования данных, когда изменение одного SO влияет на все экземпляры.
