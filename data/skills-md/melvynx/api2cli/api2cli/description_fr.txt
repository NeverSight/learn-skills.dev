Générez une CLI + AgentSkill à partir de n'importe quelle API REST. À utiliser lorsque : l'utilisateur dit « créer une CLI pour X », « envelopper cette API », « créer une compétence pour X », « publier ma CLI », « publier sur npm », « pousser vers github ». Gère la découverte, l'échafaudage, la mise en œuvre des ressources, la création, la liaison, la génération de compétences, la publication npm et la publication GitHub.
