تكوين وتشغيل عمليات سير عمل CLI بتنسيق openapi لمستندات OpenAPI/AsyncAPI. يُستخدم عندما تحتاج إلى فرز الحقول، أو تصفية العمليات/العلامات/الأعلام/المحتوى، أو تغيير الحالة، أو إنشاء معرفات العملية، أو تطبيق التراكبات، أو تحويل إصدارات OpenAPI (3.0/3.1 إلى 3.1/3.2)، أو إعادة تسمية العناوين، أو تقسيم المواصفات، أو مراجع الحزمة، أو إدارة خطوط أنابيب التنسيق المدفوعة بـ .openapiformatrc/--configFile مع الحد الأدنى من تجاوزات التكوين.
