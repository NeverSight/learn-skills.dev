既存の MCP サーバーに「コード モード」ツールを追加すると、LLM は、サンドボックス化されたランタイムで大規模な API 応答に対して実行される小さな処理スクリプトを作成できます。スクリプトのコンパクトな出力のみが LLM コンテキスト ウィンドウに入ります。コード モード、コンテキスト削減、スクリプト実行、サンドボックス実行、または LLM で生成されたコード処理を MCP サーバーに追加する場合は、必ずこのスキルを使用してください。また、任意の言語 (TypeScript、Python、Go、Rust など) で、ユーザーがトークン使用量の削減、API 応答の縮小、ユーザー提供のコードの安全な実行、または MCP サーバーへのコード実行ツールの追加について言及したときにもトリガーされます。
