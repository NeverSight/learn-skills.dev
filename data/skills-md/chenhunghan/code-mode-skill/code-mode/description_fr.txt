Ajoutez un outil « mode code » à un serveur MCP existant afin que les LLM puissent écrire de petits scripts de traitement qui s'exécutent sur de grandes réponses API dans un environnement d'exécution en bac à sable : seule la sortie compacte du script entre dans la fenêtre contextuelle LLM. Utilisez cette compétence chaque fois que quelqu'un souhaite ajouter un mode code, une réduction de contexte, une exécution de script, une exécution sandbox ou un traitement de code généré par LLM à un serveur MCP. Se déclenche également lorsque les utilisateurs mentionnent la réduction de l'utilisation des jetons, la réduction des réponses de l'API, l'exécution du code fourni par l'utilisateur en toute sécurité ou l'ajout d'un outil d'exécution de code à leur serveur MCP, dans n'importe quel langage (TypeScript, Python, Go, Rust, etc.).
