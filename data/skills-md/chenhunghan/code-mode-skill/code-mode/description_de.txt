Fügen Sie einem vorhandenen MCP-Server ein „Codemodus“-Tool hinzu, damit LLMs kleine Verarbeitungsskripts schreiben können, die in einer Sandbox-Laufzeit gegen große API-Antworten ausgeführt werden – nur die kompakte Ausgabe des Skripts gelangt in das LLM-Kontextfenster. Verwenden Sie diese Fähigkeit immer dann, wenn jemand einem MCP-Server Codemodus, Kontextreduzierung, Skriptausführung, Sandbox-Ausführung oder LLM-generierte Codeverarbeitung hinzufügen möchte. Wird auch ausgelöst, wenn Benutzer die Reduzierung der Token-Nutzung, die Reduzierung von API-Antworten, die sichere Ausführung von vom Benutzer bereitgestelltem Code oder das Hinzufügen eines Codeausführungstools zu ihrem MCP-Server erwähnen – in einer beliebigen Sprache (TypeScript, Python, Go, Rust usw.).
