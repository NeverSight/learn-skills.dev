Agregue una herramienta de "modo de código" a un servidor MCP existente para que los LLM puedan escribir pequeños scripts de procesamiento que se ejecuten con grandes respuestas de API en un tiempo de ejecución en espacio aislado: solo la salida compacta del script ingresa a la ventana de contexto de LLM. Utilice esta habilidad siempre que alguien quiera agregar modo de código, reducción de contexto, ejecución de script, ejecución de espacio aislado o procesamiento de código generado por LLM a un servidor MCP. También se activa cuando los usuarios mencionan reducir el uso de tokens, reducir las respuestas API, ejecutar código proporcionado por el usuario de forma segura o agregar una herramienta de ejecución de código a su servidor MCP, en cualquier idioma (TypeScript, Python, Go, Rust, etc.).
