أضف أداة "وضع التعليمات البرمجية" إلى خادم MCP موجود حتى يتمكن طلاب LLM من كتابة نصوص معالجة صغيرة تعمل على استجابات واجهة برمجة التطبيقات الكبيرة في وقت تشغيل معزول - فقط الإخراج المضغوط للبرنامج النصي هو الذي يدخل إلى نافذة سياق LLM. استخدم هذه المهارة عندما يريد شخص ما إضافة وضع التعليمات البرمجية، أو تقليل السياق، أو تنفيذ البرنامج النصي، أو تنفيذ وضع الحماية، أو معالجة التعليمات البرمجية التي تم إنشاؤها بواسطة LLM إلى خادم MCP. يتم التشغيل أيضًا عندما يذكر المستخدمون تقليل استخدام الرمز المميز، أو تقليص استجابات واجهة برمجة التطبيقات (API)، أو تشغيل التعليمات البرمجية المقدمة من المستخدم بأمان، أو إضافة أداة تنفيذ تعليمات برمجية إلى خادم MCP الخاص بهم - بأي لغة (TypeScript، وPython، وGo، وRust، وما إلى ذلك).
