Добавьте инструмент «режим кода» к существующему серверу MCP, чтобы LLM могли писать небольшие сценарии обработки, которые выполняются с большими ответами API в изолированной среде выполнения — в контекстное окно LLM попадает только компактный вывод сценария. Используйте этот навык всякий раз, когда кто-то хочет добавить режим кода, сокращение контекста, выполнение сценариев, выполнение в песочнице или обработку кода, сгенерированного LLM, на сервере MCP. Также срабатывает, когда пользователи упоминают о сокращении использования токенов, сокращении ответов API, безопасном запуске предоставленного пользователем кода или добавлении инструмента выполнения кода на свой сервер MCP — на любом языке (TypeScript, Python, Go, Rust и т. д.).
