Aggiungi uno strumento di "modalità codice" a un server MCP esistente in modo che gli LLM possano scrivere piccoli script di elaborazione eseguiti su risposte API di grandi dimensioni in un runtime sandbox: solo l'output compatto dello script entra nella finestra di contesto LLM. Utilizza questa competenza ogni volta che qualcuno desidera aggiungere la modalità codice, la riduzione del contesto, l'esecuzione di script, l'esecuzione sandbox o l'elaborazione del codice generato da LLM a un server MCP. Si attivano anche quando gli utenti menzionano la riduzione dell'utilizzo dei token, la riduzione delle risposte API, l'esecuzione sicura del codice fornito dall'utente o l'aggiunta di uno strumento di esecuzione del codice al proprio server MCP, in qualsiasi linguaggio (TypeScript, Python, Go, Rust, ecc.).
