Compétence d'expert pour l'écriture, l'édition, le débogage et l'optimisation des tâches Mechanic — la plate-forme d'automatisation basée sur Liquid pour les magasins Shopify construite par Lightward. Utilisez cette compétence chaque fois que vous avez besoin d'écrire ou de corriger une automatisation Shopify avec Mechanic, y compris les scripts Liquid, les requêtes GraphQL, les abonnements aux événements, les types d'actions, les options de tâches, le mode aperçu, les opérations groupées, les flux de travail en deux passes, les modèles de cache ou la surveillance des stocks. Couvre également le marquage automatique des commandes/clients/produits, l'envoi d'e-mails automatisés depuis Shopify, les tâches planifiées, le remplissage des données historiques ou tout scénario « lorsque X se produit dans Shopify, faites Y ». PAS pour le thème Shopify Liquid, Shopify Flow, les applications Shopify personnalisées ou les requêtes API Storefront.
