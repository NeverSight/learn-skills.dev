Habilidad experta para escribir, editar, depurar y optimizar tareas de Mechanic: la plataforma de automatización basada en Liquid para tiendas Shopify creada por Lightward. Utilice esta habilidad siempre que necesite escribir o arreglar una automatización de Shopify con Mechanic, incluidas secuencias de comandos Liquid, consultas GraphQL, suscripciones a eventos, tipos de acciones, opciones de tareas, modo de vista previa, operaciones masivas, flujos de trabajo de dos pasos, patrones de caché o monitoreo de inventario. También cubre el etiquetado automático de pedidos/clientes/productos, el envío de correos electrónicos automatizados desde Shopify, las tareas programadas, el reabastecimiento de datos históricos o cualquier escenario de "cuando sucede X en Shopify, haz Y". NO para el tema de Shopify Liquid, Shopify Flow, aplicaciones personalizadas de Shopify o consultas de Storefront API.
