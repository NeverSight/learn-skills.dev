Competenze esperte per la scrittura, la modifica, il debug e l'ottimizzazione delle attività di Mechanic: la piattaforma di automazione basata su Liquid per i negozi Shopify creata da Lightward. Utilizza questa competenza ogni volta che devi scrivere o correggere un'automazione Shopify con Mechanic, inclusi scripting Liquid, query GraphQL, iscrizioni a eventi, tipi di azioni, opzioni di attività, modalità di anteprima, operazioni in blocco, flussi di lavoro a due passaggi, modelli di cache o monitoraggio dell'inventario. Copre anche la codifica automatica di ordini/clienti/prodotti, l'invio di e-mail automatizzate da Shopify, attività pianificate, riempimento di dati storici o qualsiasi scenario "quando X accade in Shopify, fai Y". NON per il tema Shopify Liquid, Shopify Flow, app Shopify personalizzate o query API Storefront.
