Экспертные навыки написания, редактирования, отладки и оптимизации задач Mechanic — платформы автоматизации на основе Liquid для магазинов Shopify, созданной Lightward. Используйте этот навык всякий раз, когда вам нужно написать или исправить автоматизацию Shopify с помощью Mechanic, включая сценарии Liquid, запросы GraphQL, подписки на события, типы действий, параметры задач, режим предварительного просмотра, массовые операции, двухпроходные рабочие процессы, шаблоны кэширования или мониторинг запасов. Также охватывает автоматическую пометку заказов/клиентов/продуктов, отправку автоматических электронных писем из Shopify, запланированные задачи, заполнение исторических данных или любой сценарий «когда X происходит в Shopify, делай Y». НЕ для темы Shopify Liquid, Shopify Flow, пользовательских приложений Shopify или запросов API витрины магазина.
