قم بإنشاء أو تحديث مشكلة GitHub مع توضيح الأسئلة. يتم التشغيل عندما: يطلب المستخدم إنشاء مشكلة GitHub أو تحديثها أو تحريرها أو تحسينها. أمثلة: "إنشاء مشكلة لـ..."، "تحديث الإصدار 42"، "تحسين الوصف في الإصدار 137". لا يتم تشغيله عندما: يريد المستخدم فقط عرض/سرد المشكلات، أو التعليق على العلاقات العامة، أو تنفيذ ميزة.
