Cree un nuevo Registro de Decisión de Arquitectura en doc/adr/. DISPARAR cuando: el usuario solicita crear o registrar una decisión arquitectónica, o menciona ADR. Ejemplos: "crear un ADR para...", "registrar la decisión de utilizar PostgreSQL", "agregar un ADR". NO DISPARAR cuando: el usuario quiera leer/revisar ADR existentes o escribir documentación general.
