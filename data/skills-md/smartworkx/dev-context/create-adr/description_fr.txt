Créez un nouvel enregistrement de décision d'architecture dans doc/adr/. DÉCLENCHEMENT lorsque : l'utilisateur demande à créer ou enregistrer une décision architecturale, ou mentionne ADR. Exemples : "créer un ADR pour...", "enregistrer la décision d'utiliser PostgreSQL", "ajouter un ADR". NE PAS DÉCLENCHER lorsque : l'utilisateur souhaite lire/réviser les ADR existants ou rédiger une documentation générale.
