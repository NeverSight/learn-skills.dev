Utilizzare per il flusso di lavoro PR in pila della CLI di Graphite nei repository con .git/.graphite_repo_config.
Trigger: grafite, PR impilati, PR dipendenti, PR concatenati, stack PR, gt create,
gt modifica, gt invia, gt sincronizza, gt restack, gt log, gt checkout, gt up, gt down,
ribasare il mio stack, risolvere i conflitti di stack, dividere PR, atterrare il mio stack, unire stack,
sincronizza con main/trunk, riordina rami, piega commit, modifica stack, sposta ramo
a un genitore diverso, stack obsoleto, aggiorna il mio stack. Per repository SENZA
.git/.graphite_repo_config, utilizza invece i comandi git standard.
