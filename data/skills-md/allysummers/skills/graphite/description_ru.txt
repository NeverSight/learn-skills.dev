Используйте для рабочего процесса составных PR Graphite CLI в репозиториях с .git/.graphite_repo_config.
Триггеры: графит, сложенные PR, зависимые PR, цепочные PR, стек PR, gt create,
gt изменить, gt submit, gt sync, gt restack, gt log, gt checkout, gt up, gt down,
перебазировать мой стек, исправить конфликты стека, разделить PR, разместить мой стек, объединить стек,
синхронизация с основной/магистралью, изменение порядка ветвей, свертывание коммитов, изменение стека, перемещение ветки
другому родителю, стек устарел, обновите свой стек. Для репозиториев БЕЗ
.git/.graphite_repo_config, вместо этого используйте стандартные команды git.
