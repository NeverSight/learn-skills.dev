Преобразует Markdown в стилизованный HTML с помощью тем, совместимых с WeChat. Поддерживает выделение кода, математические вычисления, PlantUML, сноски, оповещения, инфографику и дополнительные нижние цитаты для внешних ссылок. Используйте, когда пользователь запрашивает «уценку в html», «конвертировать md в html», «md转html», «微信外链转底部引用» или нуждается в стилизованном выводе HTML из уценки.
