Passen Sie Agentenfähigkeiten an, reparieren Sie sie und machen Sie sie umgebungsfest, damit sie in diesem Tresor ordnungsgemäß ausgeführt werden. Verwenden Sie diesen Skill, wenn: (1) ein heruntergeladener oder Drittanbieter-Skill installiert/an die Laufzeit (uv/bun), Ordnerkonventionen und Pfadregeln dieses Tresors angepasst werden muss. (2) Ein vorhandener Skill schlägt zur Laufzeit aufgrund von Umgebungsproblemen fehl oder weist einen Fehler auf – falscher Python/TS-Runner, fehlerhafte Pfade, fehlende Inline-Abhängigkeiten, falsche Ausführungsbefehle oder an den falschen Speicherort geschriebene Artefakte. (3) Die versendeten Dokumente eines Skills (AGENTS.md / SKILL.md) stehen im Widerspruch zu den Standards dieses Tresors und müssen abgeglichen werden. 
Auslöser: „Diesen Skill waschen“, „Diesen Skill reparieren“, „Dieser Skill ist kaputt“, „Skill anpassen von“, „Skill installieren“, „Skill gibt einen Fehler aus“, „Skill kann nicht gefunden werden“, „Skill aktualisieren, damit er hier funktioniert“.
