Ide-IM-Skill : Bridge Cursor (ou Claude Code / Codex) vers Telegram, Discord, Feishu/Lark ou QQ pour que le
l'utilisateur peut discuter avec l'IA depuis son téléphone. Utiliser pour : la configuration, le démarrage, l'arrêt,
ou diagnostiquer le démon du pont ide-im ; IM 桥接、消息推送、连上飞书、手机上看、启动桥接、诊断、查看日志、配置.
Sous-commandes : setup, start, stop, status, logs, reconfigure, doctor.
Ne PAS utiliser pour : créer des robots autonomes, des intégrations de webhooks ou du codage avec des SDK IM.
