Ide-IM-Skill: Bridge Cursor (o Claude Code / Codex) su Telegram, Discord, Feishu/Lark o QQ in modo che
l'utente può chattare con l'intelligenza artificiale dal proprio telefono. Utilizzare per: impostare, avviare, arrestare,
o diagnosticare il demone del bridge ide-im; IM 桥接、消息推送、连上飞书、手机上看、启动桥接、诊断、查看日志、配置.
Sottocomandi: setup, start, stop, status, logs, reconfigure, doctor.
NON utilizzare per: creazione di bot autonomi, integrazioni webhook o codifica con SDK IM.
