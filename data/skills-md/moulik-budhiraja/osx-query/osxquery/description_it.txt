Utilizza OSXQuery quando un agente deve interagire con UI desktop, app Web o flussi di lavoro del browser tramite la CLI "osx". Applicalo per interrogare gli alberi degli elementi dell'interfaccia utente, risolvere riferimenti stabili ed eseguire azioni con verifica; utilizzarlo anche nei test automatizzati in cui è richiesta l'interazione dell'interfaccia utente per convalidare altri sistemi o il comportamento end-to-end. Applica la verifica prima dello screenshot per modifiche di stato significative e azioni non annullabili.
