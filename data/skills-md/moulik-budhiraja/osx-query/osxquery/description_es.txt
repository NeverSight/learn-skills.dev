Utilice OSXQuery cuando un agente deba interactuar con las IU de escritorio, aplicaciones web o flujos de trabajo del navegador a través de la CLI "osx". Aplíquelo para consultar árboles de elementos de la interfaz de usuario, resolver referencias estables y ejecutar acciones con verificación; Úselo también en pruebas automatizadas donde se requiere la interacción de la interfaz de usuario para validar otros sistemas o el comportamiento de un extremo a otro. Aplique la verificación con captura de pantalla primero para cambios de estado significativos y acciones que no se pueden deshacer.
