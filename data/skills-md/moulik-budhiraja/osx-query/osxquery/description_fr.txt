Utilisez OSXQuery lorsqu'un agent doit interagir avec des interfaces utilisateur de bureau, des applications Web ou des flux de travail de navigateur via la CLI « osx ». Appliquez-le pour interroger les arborescences d'éléments de l'interface utilisateur, résoudre les références stables et exécuter des actions avec vérification ; utilisez-le également dans des tests automatisés où une interaction avec l'interface utilisateur est requise pour valider d'autres systèmes ou un comportement de bout en bout. Appliquez la vérification par capture d'écran pour les changements d'état significatifs et les actions irréversibles.
