Les API Google simplifiées : Gmail, Drive, Calendrier, Tâches. Bibliothèque unifiée et CLI de passerelle (go-gmail, go-drive, go-calendar, go-tasks) pour les agents IA. À utiliser lorsque l'utilisateur doit travailler avec Gmail, Google Drive, Google Calendar ou Google Tasks. Remplace gmcli, gdcli, gccli.
