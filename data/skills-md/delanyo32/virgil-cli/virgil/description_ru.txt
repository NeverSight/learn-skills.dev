Разбирайте, исследуйте и понимайте незнакомые кодовые базы с помощью virgil-cli. Охватывает полный рабочий процесс: анализ исходного кода в Parquet, запросы с помощью встроенных команд, навигацию по зависимостям и вызывающим объектам, а также запуск необработанного SQL-кода DuckDB. Используйте, когда вас просят проанализировать базу кода, понять архитектуру, найти символы, отследить зависимости, подключиться к проекту, исследовать ошибки или сопоставить поверхность API любой кодовой базы TypeScript/JavaScript/C/C++/C#/Rust/Python/Go/Java/PHP.
