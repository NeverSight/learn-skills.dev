提供使用 TypeScript 进行 Node.js 开发的特定领域最佳实践，涵盖类型剥离、异步模式、错误处理、流、模块、测试、性能、缓存、日志记录等。在设置具有原生 TypeScript 支持的 Node.js 项目、配置类型剥离 (--experimental-strip-types)、在没有构建步骤的情况下编写 Node 22+ TypeScript 时使用，或者当用户提到“Node 中的原生 TypeScript”、“剥离类型”、“Node 22 TypeScript”、“无需编译的 .ts 文件”、“ts-node 替代方案”时使用，或者需要有关错误处理、正常关闭、片状测试的指导， Node.js 中的分析或环境配置。帮助配置 tsconfig.json 以进行类型剥离、设置 package.json 脚本、处理模块解析和导入扩展，以及在整个 Node.js 堆栈中应用健壮的模式。
