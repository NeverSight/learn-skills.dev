Содержит лучшие практики разработки Node.js с использованием TypeScript для конкретной предметной области, включая удаление типов, асинхронные шаблоны, обработку ошибок, потоки, модули, тестирование, производительность, кэширование, ведение журналов и многое другое. Используйте при настройке проектов Node.js со встроенной поддержкой TypeScript, настройке удаления типов (--experimental-strip-types), написании Node 22+ TypeScript без этапа сборки или когда пользователь упоминает «собственный TypeScript в Node», «типы разделения», «Node 22 TypeScript», «файлы .ts без компиляции», «альтернативу ts-node» или нуждается в руководстве по обработке ошибок, корректному завершению работы, нестабильным тестам, профилирование или настройка среды в Node.js. Помогает настроить tsconfig.json для удаления типов, настроить сценарии package.json, обрабатывать разрешение модулей и импортировать расширения, а также применять надежные шаблоны ко всему стеку Node.js.
