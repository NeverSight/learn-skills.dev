Utilice esta habilidad siempre que el usuario mencione actualizaciones del repositorio de GitHub, notas de la versión, CHANGELOG, comparaciones de versiones, informes diarios/semanales, suscribirse a múltiples fuentes de proyectos, pegar una URL del repositorio de GitHub directamente o integrar actualizaciones de GitHub en cron/automatización.
