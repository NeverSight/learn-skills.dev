📦 Get a summary of GitHub repository updates. Prioritize reading CHANGELOG; fall back to Releases if not found. Use this skill whenever users mention GitHub repo updates, Release Notes, CHANGELOG, version comparison, daily/weekly reports, subscribing to multiple project feeds, pasting a GitHub repo URL directly, or integrating GitHub updates into cron/automation. It outputs stable, structured JSON by default, suitable for scripts and LLM consumption, and reuses the local `gh auth login` session. This skill is designed for small-batch queries—up to 10 repos at a time is recommended.
