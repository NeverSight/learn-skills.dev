📦 Ottieni un riepilogo degli aggiornamenti del repository GitHub. Dai la priorità alla lettura del CHANGELOG, torna a Releases se non trovato. Utilizza questa competenza ogni volta che l'utente menziona aggiornamenti del repository GitHub, note di rilascio, CHANGELOG, confronto delle versioni, report giornalieri/settimanali, sottoscrizione a più feed di progetto, incollando direttamente gli URL del repository GitHub o integrando gli aggiornamenti di GitHub in cron/automazione. Per impostazione predefinita, genera JSON strutturato stabile, adatto a script e modelli di grandi dimensioni e riutilizza la sessione locale `gh auth login`. Questa competenza è per query in batch di piccole dimensioni; si consigliano fino a 10 repository alla volta.
