📦 Obtenez un résumé des mises à jour pour un référentiel GitHub. Prioriser la lecture du CHANGELOG ; revenez aux versions si elles ne sont pas trouvées. Utilisez cette compétence chaque fois que l'utilisateur mentionne les mises à jour du dépôt GitHub, les notes de version, CHANGELOG, les comparaisons de versions, les rapports quotidiens/hebdomadaires, l'abonnement à plusieurs flux de projet, le collage direct d'une URL de dépôt GitHub ou l'intégration des mises à jour GitHub dans cron/automatisation. Produit un JSON stable et structuré par défaut, adapté aux scripts et aux LLM, et réutilise la session locale « gh auth login » Conçu pour les requêtes en petits lots ; recommande d'interroger jusqu'à 10 dépôts à la fois.
