📦 Получите сводку обновлений для репозиториев GitHub. Предпочитаю читать CHANGELOG, если не найден, вернитесь к Releases. Используйте этот навык всякий раз, когда пользователь упоминает обновления репозитория GitHub, примечания к выпуску, журнал изменений, сравнение версий, ежедневные/еженедельные отчеты, подписку на несколько каналов проекта, прямую вставку URL-адресов репозитория GitHub или интеграцию обновлений GitHub в cron/автоматизацию. По умолчанию выводит стабильный структурированный JSON, подходящий для сценариев и использования LLM, и повторно использует локальный сеанс входа в систему `gh auth`. Предназначен для небольших пакетных запросов, рекомендуется использовать до 10 репозиториев одновременно.
