Automatización integral para transcripciones Letterly. Esta habilidad exporta el último CSV de Letterly, procesa notas "mágicas" en Obsidian Markdown con metadatos personalizados, las vincula semánticamente usando una base de datos vectorial y las mueve al directorio final de Transcripciones. Úselo cuando el usuario solicite "procesar nuevas transcripciones de letras", "sincronizar letras" o "importar notas mágicas de letras".
