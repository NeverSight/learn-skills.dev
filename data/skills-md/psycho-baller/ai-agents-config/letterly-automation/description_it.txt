Automazione completa per trascrizioni letterali. Questa abilità esporta l'ultimo CSV da Letterly, elabora le note "magiche" nel markdown di Obsidian con metadati personalizzati, le collega semanticamente utilizzando un database vettoriale e le sposta nella directory Trascrizioni finale. Da utilizzare quando l'utente chiede di "elaborare nuove trascrizioni letterali", "sincronizzazione letterale" o "importare note magiche da letterale".
