Patrones modernos de tidyverse, guía de estilo y guía de migración para el desarrollo de R. Utilice esta habilidad al escribir código R con dplyr, revisar el código tidyverse, actualizar el código R heredado a patrones modernos o aplicar un estilo coherente. Cubre el uso de canalizaciones nativas, sintaxis join_by(), agrupación .by, operaciones de selección/transformación/reencuadre, selección ordenada, patrones de cadena, convenciones de nomenclatura, espaciado y migración desde R base o API tidyverse anteriores.
