Modèles Tidyverse modernes, guide de style et conseils de migration pour le développement R. Utilisez cette compétence lors de l'écriture de code R avec dplyr, de la révision du code Tidyverse, de la mise à jour du code R existant vers des modèles modernes ou de l'application d'un style cohérent. Couvre l'utilisation des tubes natifs, la syntaxe join_by(), le regroupement .by, les opérations de sélection/traversation/recadrage, la sélection soignée, les modèles stringr, les conventions de dénomination, l'espacement et la migration à partir de la base R ou des anciennes API Tidyverse.
