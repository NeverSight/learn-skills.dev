Создание надежных пакетов R с использованием современных шаблонов tidyverse. Используйте этот навык при создании или обслуживании пакетов R, разработке API, выборе зависимостей, реализации проверки входных данных, написании сообщений об ошибках или выборе между внутренними и экспортируемыми функциями. Охватывает стратегию зависимостей, шаблоны проектирования API tidyverse, подходы к проверке, обработку ошибок с помощью cli/rlang, уровни тестирования и приоритеты документации.
