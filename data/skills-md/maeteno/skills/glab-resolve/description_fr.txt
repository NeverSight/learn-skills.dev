Résolvez les fils de révision GitLab MR de manière interactive. À utiliser lorsque l'utilisateur souhaite traiter les commentaires MR, corriger les commentaires de révision, résoudre les fils de discussion ou résoudre les problèmes de révision de code sur une demande de fusion GitLab. Se déclenche sur des requêtes telles que « résoudre les commentaires MR », « corriger les commentaires de révision », « adresser les threads MR », « /glab-resolve » ou « /glab-resolve <MR_ID> ».
