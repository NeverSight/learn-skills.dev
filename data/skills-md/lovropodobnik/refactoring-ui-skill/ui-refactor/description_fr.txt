Guide de conception d'interface utilisateur tactique pour corriger les mises en page, sélectionner les couleurs/polices et créer des interfaces utilisateur professionnelles. À utiliser lorsque l'utilisateur demande à « améliorer l'apparence », a besoin d'aide pour prendre des décisions en matière de CSS/style, souhaite créer un système de conception ou doit réparer une interface encombrée.
