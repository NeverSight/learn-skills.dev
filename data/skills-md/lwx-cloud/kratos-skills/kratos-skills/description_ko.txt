go-kratos 마이크로서비스 프레임워크에 대한 포괄적인 지식 기반입니다.

**다음과 같은 경우에 이 기술을 사용하세요.**
- kratos로 REST/gRPC API 구축 (서비스 → Biz → 데이터 계층 아키텍처)
- DDD 및 클린 아키텍처 패턴으로 마이크로서비스 만들기
- Wire를 이용한 의존성 주입 구현
- 서비스 검색, 로드 밸런싱, 복원력 패턴 구성
- kratos 문제 해결 또는 프레임워크 규칙 이해
- Protobuf를 사용하여 프로덕션에 즉시 사용할 수 있는 마이크로서비스 코드 생성

**특징:**
- ✅ 올바른 예와 ❌ 잘못된 예가 포함된 완전한 패턴 가이드
- DDD/클린 아키텍처 시행
- 생산 모범 사례
- 일반적인 함정 솔루션
