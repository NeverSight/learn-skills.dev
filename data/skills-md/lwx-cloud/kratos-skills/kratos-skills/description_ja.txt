go-kratos マイクロサービス フレームワークの包括的なナレッジ ベース。

**このスキルは次の場合に使用します。**
- kratos を使用した REST/gRPC API の構築 (サービス → ビジネス → データ階層化アーキテクチャ)
- DDD およびクリーン アーキテクチャ パターンを使用したマイクロサービスの作成
- Wire を使用した依存関係の注入の実装
- サービス検出、負荷分散、復元パターンの構成
- Kratos の問題のトラブルシューティング、またはフレームワークの規則の理解
- Protobuf を使用した本番環境に対応したマイクロサービス コードの生成

**特徴:**
- ✅ 正しい例と ❌ 間違った例を含む完全なパターン ガイド
- DDD/クリーンアーキテクチャの実施
- 本番環境のベストプラクティス
- よくある落とし穴の解決策
