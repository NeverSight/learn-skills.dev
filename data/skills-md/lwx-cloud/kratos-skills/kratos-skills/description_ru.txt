Обширная база знаний для инфраструктуры микросервисов go-kratos.

**Используйте этот навык, когда:**
- Создание API REST/gRPC с помощью kratos (Сервис → Бизнес → Многоуровневая архитектура данных)
- Создание микросервисов с использованием шаблонов DDD и чистой архитектуры.
— Реализация внедрения зависимостей с помощью Wire.
- Настройка шаблонов обнаружения служб, балансировки нагрузки и устойчивости.
- Устранение неполадок Кратоса или понимание соглашений фреймворка.
- Генерация готового к использованию кода микросервисов с помощью Protobuf.

**Особенности:**
- Полные руководства по выкройкам с ✅ правильными и ❌ неправильными примерами.
- Применение DDD/Чистой архитектуры
- Лучшие практики производства
- Распространенные решения ошибок
