Base de connaissances complète pour le framework de microservices go-kratos.

**Utilisez cette compétence lorsque :**
- Construction d'API REST/gRPC avec kratos (Service → Biz → Architecture en couches de données)
- Création de microservices avec des modèles DDD et Clean Architecture
- Implémentation de l'injection de dépendances avec Wire
- Configuration des modèles de découverte de services, d'équilibrage de charge et de résilience
- Dépannage des problèmes de kratos ou compréhension des conventions du framework
- Génération de code de microservices prêt pour la production avec Protobuf

**Caractéristiques :**
- Guides de modèles complets avec ✅ exemples corrects et ❌ incorrects
- Application de DDD/Clean Architecture
- Bonnes pratiques de production
- Solutions aux pièges courants
