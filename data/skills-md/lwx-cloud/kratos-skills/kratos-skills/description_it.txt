Base di conoscenza completa per il framework di microservizi go-kratos.

**Usa questa abilità quando:**
- Creazione di API REST/gRPC con kratos (Servizio → Biz → Architettura a livelli di dati)
- Creazione di microservizi con pattern DDD e Clean Architecture
- Implementazione dell'iniezione di dipendenze con Wire
- Configurazione dell'individuazione dei servizi, del bilanciamento del carico e dei modelli di resilienza
- Risoluzione dei problemi di Kratos o comprensione delle convenzioni del framework
- Generazione di codice di microservizi pronti per la produzione con Protobuf

**Caratteristiche:**
- Guide complete sui modelli con esempi ✅ corretti e ❌ errati
- Applicazione DDD/Clean Architecture
- Migliori pratiche di produzione
- Soluzioni comuni alle trappole
