Rufen Sie DingTalk Open Platform APIs auf und unterstützen Sie Benutzersuche/-details/-abfragen, Abteilungsverwaltung (Suche/Details/Unterabteilungen/Benutzerliste/übergeordnete Abteilung), Senden von Roboter-Einzelnachrichten, Senden von Gruppennachrichten, Abfragen von Roboterlisten innerhalb der Gruppe und Abfragen von Rücktrittsdatensätzen. Verwenden Sie diese Funktion, wenn Sie DingTalk-Benutzer oder -Abteilungen durchsuchen, Benutzer-/Abteilungsdetails abrufen, Roboternachrichten senden, Gruppen-Bots auflisten oder gekündigte Mitarbeiter befragen müssen.
