Llame a la API de plataforma abierta DingTalk, que admite búsqueda/detalles/consulta de usuarios, gestión de departamentos (búsqueda/detalles/subdepartamentos/lista de usuarios/departamento principal), envío de mensajes individuales de robots, envío de mensajes grupales, consulta de lista de robots grupales y consulta de registros de renuncia.
