Chiama l'API DingTalk Open Platform, che supporta la ricerca/dettagli/query degli utenti, la gestione del dipartimento (ricerca/dettagli/sottodipartimenti/elenco utenti/dipartimento principale), l'invio di messaggi uno a uno dei robot, l'invio di messaggi di gruppo, la query dell'elenco dei robot nel gruppo e la query del record di dimissioni. Utilizzare quando è necessario cercare utenti o dipartimenti DingTalk, ottenere dettagli utente/dipartimento, inviare messaggi robot, elencare bot di gruppi o interrogare dipendenti dimessi.
