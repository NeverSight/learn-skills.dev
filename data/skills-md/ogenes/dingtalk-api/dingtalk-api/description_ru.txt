Вызов API открытой платформы DingTalk, поддерживающий поиск пользователей/сведения/запросы, управление отделами (поиск/сведения/подотделы/список пользователей/родительский отдел), отправку сообщений робота один на один, отправку групповых сообщений, запрос списка роботов в группе и запрос записи об увольнении. Используйте, когда необходимо найти пользователей или отделы DingTalk, получить сведения о пользователе/отделе, отправить сообщения роботам, составить список групповых ботов или опросить уволившихся сотрудников.
