اتصل بـ DingTalk Open Platform API، الذي يدعم بحث المستخدم/التفاصيل/الاستعلام، وإدارة الأقسام (البحث/التفاصيل/الأقسام الفرعية/قائمة المستخدمين/القسم الأصلي)، وإرسال الرسائل الفردية للروبوت، وإرسال الرسائل الجماعية، والاستعلام عن قائمة الروبوتات في المجموعة، والاستعلام عن سجلات الاستقالة. يُستخدم عند الحاجة إلى البحث عن مستخدمي أو أقسام DingTalk، أو الحصول على تفاصيل المستخدم/القسم، أو إرسال رسائل روبوت، أو قائمة روبوتات المجموعة، أو الاستعلام عن الموظفين المستقيلين.
