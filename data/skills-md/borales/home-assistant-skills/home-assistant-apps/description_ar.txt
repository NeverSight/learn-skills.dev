تطوير وتكوين واختبار ونشر وتأمين تطبيقات Home Assistant (الوظائف الإضافية سابقًا) - الملحقات المستندة إلى Docker والتي يديرها المشرف. يُستخدم عند إنشاء تطبيقات جديدة، أو كتابة ملفات Dockerfiles، أو تكوين config.yaml، أو إعداد Ingress، أو إنشاء المستودعات، أو تنفيذ أفضل ممارسات الأمان، أو النشر إلى سجلات الحاويات.
