Automatisez la vérification de la durée de vie du code de migration (TODO(migration)). Détectez le code de migration expiré ou manquant d’informations et affichez-le sous forme de rapport. Utilisez-le lorsque vous souhaitez vous assurer qu'aucun TODO (migration) ne reste dans la base de code, lors du nettoyage du code de migration expiré ou lors de l'inventaire de la dette technique. Un système pour empêcher que le code « supprimer plus tard » ne soit oublié.
