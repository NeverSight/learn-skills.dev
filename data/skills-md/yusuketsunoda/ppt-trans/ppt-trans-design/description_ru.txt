Система дизайна, посвященная этому проекту (перевод xPPT, редактирование SaaS). Он определяет принципы UX продукта (различное представление/поток утверждения/надежность/согласованность макета) и оригинальную цветовую палитру на основе токенов MD3. Всегда обращайтесь к нему при реализации пользовательского интерфейса для этого проекта, включая новые компоненты пользовательского интерфейса, поддержку темного режима, перевод пользовательского интерфейса и минимальное приложение для дизайна. Используйте его для проектных решений, специфичных для проекта, а не для общего дизайна материалов.
