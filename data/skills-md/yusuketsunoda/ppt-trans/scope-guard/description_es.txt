Antes de comprometer o crear un PR, detecta "diferencias no deseadas" y advierte si se desvían del propósito de la rama. Incluso ofrece comandos para descartar cambios no relacionados que se hayan introducido. Úselo para verificaciones de alcance previas a la confirmación, para verificar que no se mezclen archivos adicionales en la diferencia y para garantizar que el alcance de un PR no haya crecido demasiado. Siempre que se pregunte: "¿Hay algún cambio no relacionado con el objetivo de esta rama?", ejecútelo.
