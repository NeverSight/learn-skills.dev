Erstellt reaktive Webanwendungen mit StarHTML, einem Python-First-Framework über Datastar. Wird beim Schreiben von StarHTML-Komponenten, Signalen, Ereignishandlern, reaktiven Attributen, bedingten Helfern, CSS-Klassen, berechneten Signalen, HTTP-Aktionen, SSE-Endpunkten oder Plugins (Persist, Scroll, Resize, Drag, Canvas, Motion, Markdown, Katex, Mermaid, Split, Nodegraph) verwendet. Verwenden Sie für UI-Komponenten StarUI (shadcn/ui für Python). Nachdem Sie eine StarHTML-Datei generiert haben, führen Sie zur Validierung „starhtml-check <file.py>“ aus.
