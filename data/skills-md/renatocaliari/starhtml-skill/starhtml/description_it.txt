Costruisce applicazioni web reattive con StarHTML, un framework basato su Python su Datastar. Da utilizzare durante la scrittura di componenti StarHTML, segnali, gestori di eventi, attributi reattivi, helper condizionali, classi CSS, segnali calcolati, azioni HTTP, endpoint SSE o plug-in (persist, scroll, resize, drag, canvas, motion, markdown, katex, mermaid, split, nodegraph). Per i componenti dell'interfaccia utente, utilizzare StarUI (shadcn/ui per Python). Dopo aver generato qualsiasi file StarHTML, esegui `starhtml-check <file.py>` per convalidare.
