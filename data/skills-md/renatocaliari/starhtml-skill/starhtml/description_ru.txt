Создает реактивные веб-приложения с помощью StarHTML, фреймворка, основанного на Python, поверх Datastar. Используйте при написании компонентов StarHTML, сигналов, обработчиков событий, реактивных атрибутов, условных помощников, классов CSS, вычисляемых сигналов, действий HTTP, конечных точек SSE или плагинов (сохранение, прокрутка, изменение размера, перетаскивание, холст, движение, уценка, катекс, русалка, разделение, нодграф). Для компонентов пользовательского интерфейса используйте StarUI (shadcn/ui для Python). После создания любого файла StarHTML запустите `starhtml-check <file.py>` для проверки.
