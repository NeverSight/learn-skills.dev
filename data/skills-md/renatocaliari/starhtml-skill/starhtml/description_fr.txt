Crée des applications Web réactives avec StarHTML, un framework Python sur Datastar. À utiliser lors de l'écriture de composants StarHTML, de signaux, de gestionnaires d'événements, d'attributs réactifs, d'assistances conditionnelles, de classes CSS, de signaux calculés, d'actions HTTP, de points de terminaison SSE ou de plugins (persister, faire défiler, redimensionner, glisser, canevas, mouvement, démarque, katex, sirène, fractionnement, noeudgraphe). Pour les composants de l'interface utilisateur, utilisez StarUI (shadcn/ui pour Python). Après avoir généré un fichier StarHTML, exécutez `starhtml-check <file.py>` pour valider.
