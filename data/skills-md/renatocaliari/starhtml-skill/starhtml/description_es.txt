Crea aplicaciones web reactivas con StarHTML, un marco de Python basado en Datastar. Úselo al escribir componentes StarHTML, señales, controladores de eventos, atributos reactivos, ayudantes condicionales, clases CSS, señales calculadas, acciones HTTP, puntos finales SSE o complementos (persistir, desplazarse, cambiar el tamaño, arrastrar, lienzo, movimiento, rebajas, katex, sirena, dividir, nodegraph). Para los componentes de la interfaz de usuario, utilice StarUI (shadcn/ui para Python). Después de generar cualquier archivo StarHTML, ejecute `starhtml-check <file.py>` para validar.
