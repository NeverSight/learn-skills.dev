Datastar を超える Python ファーストのフレームワークである StarHTML を使用してリアクティブ Web アプリケーションを構築します。 StarHTML コンポーネント、シグナル、イベント ハンドラー、リアクティブ属性、条件付きヘルパー、CSS クラス、計算シグナル、HTTP アクション、SSE エンドポイント、またはプラグイン (永続、スクロール、サイズ変更、ドラッグ、キャンバス、モーション、マークダウン、katex、マーメイド、分割、ノードグラフ) を作成するときに使用します。 UI コンポーネントの場合は、StarUI (Python の場合は shadcn/ui) を使用します。 StarHTML ファイルを生成した後、「starhtml-check <file.py>」を実行して検証します。
