Datastar를 통한 Python 우선 프레임워크인 StarHTML을 사용하여 반응형 웹 애플리케이션을 구축합니다. StarHTML 구성 요소, 신호, 이벤트 처리기, 반응 속성, 조건부 도우미, CSS 클래스, 계산된 신호, HTTP 작업, SSE 끝점 또는 플러그인(지속, 스크롤, 크기 조정, 드래그, 캔버스, 모션, 마크다운, katex, mermaid, 분할, nodegraph)을 작성할 때 사용합니다. UI 구성 요소의 경우 StarUI(Python의 경우 shadcn/ui)를 사용합니다. StarHTML 파일을 생성한 후 `starhtml-check <file.py>`를 실행하여 유효성을 검사하세요.
