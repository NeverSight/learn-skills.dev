إنشاء تطبيقات ويب تفاعلية باستخدام StarHTML، وهو إطار عمل Python-first عبر Datastar. يُستخدم عند كتابة مكونات StarHTML، أو الإشارات، أو معالجات الأحداث، أو السمات التفاعلية، أو المساعدين الشرطيين، أو فئات CSS، أو الإشارات المحسوبة، أو إجراءات HTTP، أو نقاط نهاية SSE، أو المكونات الإضافية (الاستمرار، أو التمرير، أو تغيير الحجم، أو السحب، أو اللوحة القماشية، أو الحركة، أو تخفيض السعر، أو katex، أو mermaid، أو Split، أو Nodegraph). بالنسبة لمكونات واجهة المستخدم، استخدم StarUI (shadcn/ui for Python). بعد إنشاء أي ملف StarHTML، قم بتشغيل `starhtml-check <file.py>` للتحقق من صحته.
