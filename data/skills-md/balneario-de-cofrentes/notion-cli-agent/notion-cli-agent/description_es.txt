Utilice la CLI de Notion local (notion-cli-agent) para consultar, crear, actualizar y administrar páginas y bases de datos de Notion a través del shell. Úselo al interactuar con espacios de trabajo de Notion, consultar bases de datos, crear o actualizar páginas, administrar tareas, leer bloques de contenido o ejecutar operaciones masivas/por lotes en datos de Notion. Prefiere las llamadas Notion MCP o API.
