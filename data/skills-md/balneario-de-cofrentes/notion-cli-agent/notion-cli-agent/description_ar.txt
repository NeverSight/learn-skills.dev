استخدم Notion CLI المحلي (notion-cli-agent) للاستعلام عن صفحات وقواعد بيانات Notion وإنشائها وتحديثها وإدارتها عبر Shell. يُستخدم عند التفاعل مع مساحات عمل Notion، أو الاستعلام عن قواعد البيانات، أو إنشاء الصفحات أو تحديثها، أو إدارة المهام، أو قراءة كتل المحتوى، أو تشغيل عمليات مجمعة/مجمعة على بيانات Notion. تفضل على مكالمات Notion MCP أو API.
