Utilizza la CLI Notion locale (notion-cli-agent) per eseguire query, creare, aggiornare e gestire pagine e database Notion tramite shell. Da utilizzare quando si interagisce con gli spazi di lavoro di Notion, si eseguono query su database, si creano o si aggiornano pagine, si gestiscono attività, si leggono blocchi di contenuto o si eseguono operazioni in blocco/batch sui dati di Notion. Preferisci le chiamate Notion MCP o API.
