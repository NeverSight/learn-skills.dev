دليل لإنشاء تطبيقات CarPlay للطلب السريع (طلب الطعام، واستلامه، وما إلى ذلك) باستخدام إطار عمل CarPlay. استخدم هذه المهارة عندما يريد المستخدم إنشاء تطبيق طلب CarPlay، أو دمج قوالب CarPlay لطلب الطعام/الشراب/الالتقاط، أو العمل مع CPPointOfInterestTemplate، أو CPListTemplate، أو CPTabBarTemplate، أو CPInterfaceController لطلب سير العمل. يتم التشغيل أيضًا عندما يذكر المستخدم استحقاقات CarPlay، أو الأنشطة المباشرة مع طلب CarPlay، أو دفع تحديثات الإشعارات لحالة الطلب. يغطي دورة الحياة الكاملة: إعداد الاستحقاق، والتسلسل الهرمي للقالب، واختيار النقاط المهمة على أساس الخريطة، ووضع الطلب، وتكامل النشاط المباشر، وإدارة الرمز المميز.
