Руководство по созданию приложений быстрого заказа CarPlay (заказ еды, самовывоз и т. д.) с использованием платформы CarPlay. Используйте этот навык всякий раз, когда пользователь хочет создать приложение для заказа CarPlay, интегрировать шаблоны CarPlay для заказа еды/напитков/самовывоза, работать с CPPointOfInterestTemplate, CPListTemplate, CPTabBarTemplate или CPInterfaceController для рабочих процессов заказа. Также срабатывает, когда пользователь упоминает права CarPlay, интерактивные действия с заказом CarPlay или обновления push-уведомлений о статусе заказа. Охватывает полный жизненный цикл: настройку прав, иерархию шаблонов, выбор POI на основе карты, размещение заказов, интеграцию Live Activity и управление push-токенами.
