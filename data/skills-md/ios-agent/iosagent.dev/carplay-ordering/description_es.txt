Guía para crear aplicaciones de pedidos rápidos de CarPlay (pedidos de comida, recogida, etc.) utilizando el marco de CarPlay. Utilice esta habilidad siempre que el usuario quiera crear una aplicación de pedidos de CarPlay, integrar plantillas de CarPlay para pedidos de comida/bebida/recogida, trabajar con CPPointOfInterestTemplate, CPListTemplate, CPTabBarTemplate o CPInterfaceController para flujos de trabajo de pedidos. También se activa cuando el usuario menciona derechos de CarPlay, actividades en vivo con pedidos de CarPlay o actualizaciones de notificaciones automáticas para el estado del pedido. Cubre el ciclo de vida completo: configuración de derechos, jerarquía de plantillas, selección de puntos de interés basada en mapas, realización de pedidos, integración de actividades en vivo y administración de tokens push.
