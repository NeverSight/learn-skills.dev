Guía completa del marco de Mastra. Enseña cómo encontrar documentación actual, verificar firmas de API y crear agentes y flujos de trabajo. Cubre estrategias de búsqueda de documentación (documentos integrados, documentos remotos), conceptos básicos (agentes frente a flujos de trabajo, herramientas, memoria, RAG), requisitos de TypeScript y patrones comunes. Utilice esta habilidad para todo el desarrollo de Mastra para asegurarse de que está utilizando las API actuales de la versión instalada o la documentación más reciente.
