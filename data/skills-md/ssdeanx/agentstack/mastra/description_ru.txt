Полное руководство по платформе Matra. Учит, как находить текущую документацию, проверять подписи API, а также создавать агенты и рабочие процессы. Охватывает стратегии поиска документации (встроенные документы, удаленные документы), основные концепции (агенты и рабочие процессы, инструменты, память, RAG), требования TypeScript и общие шаблоны. Используйте этот навык для всей разработки Mastra, чтобы убедиться, что вы используете текущие API из установленной версии или последней версии документации.
