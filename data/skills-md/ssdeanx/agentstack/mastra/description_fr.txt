Guide complet du cadre Mastra. Apprenez à trouver la documentation actuelle, à vérifier les signatures API et à créer des agents et des flux de travail. Couvre les stratégies de recherche de documentation (documents intégrés, documents distants), les concepts de base (agents vs workflows, outils, mémoire, RAG), les exigences TypeScript et les modèles courants. Utilisez cette compétence pour tous les développements Mastra afin de vous assurer que vous utilisez les API actuelles de la version installée ou de la dernière documentation.
