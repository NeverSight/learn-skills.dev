Flujo de trabajo de revisión sistemática de código utilizando la herramienta Codex de Zen MCP. Utilice esta habilidad cuando el usuario solicite explícitamente "usar el códice para verificar el código", "verificar si el código generado recientemente tiene algún problema" o "verificar el código después de cada generación". La habilidad realiza ciclos de revisión iterativos: verifica la calidad del código, presenta problemas al usuario para su aprobación, aplica correcciones y vuelve a verificar hasta que no quedan problemas o se alcanza el máximo de iteraciones (5).
