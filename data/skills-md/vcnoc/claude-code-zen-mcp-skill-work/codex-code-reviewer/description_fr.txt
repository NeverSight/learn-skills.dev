Workflow de révision systématique du code à l'aide de l'outil codex de zen mcp. Utilisez cette compétence lorsque l'utilisateur demande explicitement « utiliser le codex pour vérifier le code », « vérifier si le code récemment généré présente des problèmes » ou « vérifier le code après chaque génération ». La compétence effectue des cycles de révision itératifs : vérifier la qualité du code, présenter les problèmes à l'utilisateur pour approbation, appliquer les correctifs et revérifier jusqu'à ce qu'il ne reste plus de problème ou que le nombre maximum d'itérations (5) soit atteint.
