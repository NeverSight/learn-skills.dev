Systematischer Code-Review-Workflow mit dem Codex-Tool von zen mcp. Verwenden Sie diese Fähigkeit, wenn der Benutzer ausdrücklich anfordert: „Verwenden Sie Codex, um den Code zu überprüfen“, „Überprüfen Sie, ob der kürzlich generierte Code Probleme aufweist“ oder „Überprüfen Sie den Code nach jeder Generierung“. Der Skill führt iterative Überprüfungszyklen durch – prüft die Codequalität, stellt dem Benutzer Probleme zur Genehmigung vor, wendet Korrekturen an und prüft erneut, bis keine Probleme mehr vorhanden sind oder die maximale Anzahl an Iterationen (5) erreicht ist.
