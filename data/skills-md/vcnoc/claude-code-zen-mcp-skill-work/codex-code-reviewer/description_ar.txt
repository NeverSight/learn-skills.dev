سير عمل المراجعة المنهجية للتعليمات البرمجية باستخدام أداة Codex الخاصة بـ Zen mcp. استخدم هذه المهارة عندما يطلب المستخدم صراحةً "استخدام المخطوطة للتحقق من الكود" أو "التحقق مما إذا كان الكود الذي تم إنشاؤه مؤخرًا به أي مشكلات" أو "التحقق من الكود بعد كل جيل". تنفذ المهارة دورات مراجعة متكررة - التحقق من جودة التعليمات البرمجية، وعرض المشكلات على المستخدم للموافقة عليها، وتطبيق الإصلاحات، وإعادة التحقق حتى لا تبقى أية مشكلات أو يتم الوصول إلى الحد الأقصى من التكرارات (5).
