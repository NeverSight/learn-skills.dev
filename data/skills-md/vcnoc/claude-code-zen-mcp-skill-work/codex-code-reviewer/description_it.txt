Flusso di lavoro di revisione sistematica del codice utilizzando lo strumento codex di zen mcp. Utilizza questa abilità quando l'utente richiede esplicitamente "usa il codice per controllare il codice", "controlla se il codice generato di recente presenta problemi" o "controlla il codice dopo ogni generazione". La competenza esegue cicli di revisione iterativi: verifica della qualità del codice, presentazione dei problemi all'utente per l'approvazione, applicazione delle correzioni e ricontrollo fino a quando non rimangono più problemi o viene raggiunto il numero massimo di iterazioni (5).
