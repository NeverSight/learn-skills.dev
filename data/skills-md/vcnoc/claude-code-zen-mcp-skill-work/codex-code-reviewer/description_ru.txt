Систематический рабочий процесс проверки кода с использованием инструмента codex zen mcp. Используйте этот навык, когда пользователь явно просит «использовать кодекс для проверки кода», «проверить, есть ли проблемы в недавно сгенерированном коде» или «проверять код после каждого поколения». Навык выполняет циклы итеративной проверки — проверку качества кода, представление проблем пользователю на утверждение, применение исправлений и повторную проверку до тех пор, пока проблемы не исчезнут или не будет достигнуто максимальное количество итераций (5).
