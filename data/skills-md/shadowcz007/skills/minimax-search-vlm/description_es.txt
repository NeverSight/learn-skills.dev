Utilice la API del Plan de codificación MiniMax para búsqueda web (web_search) y comprensión de imágenes (understand_image). Requiere configuración de MINIMAX_CP_API_KEY (Clave API MiniMax, generalmente comienza con sk-cp-). Ejemplos de activación: "búsqueda MiniMax", "comprensión de imágenes", "búsqueda web", "comprender imágenes", "búsqueda MiniMax", "describir esta imagen".
