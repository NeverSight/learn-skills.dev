Verwenden Sie die MiniMax Coding Plan API für die Websuche (web_search) und das Bildverständnis (understand_image). Sie müssen MINIMAX_CP_API_KEY (MiniMax-API-Schlüssel, normalerweise beginnend mit sk-cp-) konfigurieren. Trigger-Beispiele: „MiniMax-Suche“, „Bildverständnis“, „Websuche“, „Bild verstehen“, „MiniMax-Suche“, „Dieses Bild beschreiben“.
