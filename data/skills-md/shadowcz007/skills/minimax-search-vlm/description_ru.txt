Используйте API плана кодирования MiniMax для веб-поиска (web_search) и понимания изображений (understand_image). Вам необходимо настроить MINIMAX_CP_API_KEY (ключ API MiniMax, обычно начинается с sk-cp-). Примеры триггеров: «Поиск MiniMax», «Понимание изображения», «Поиск в Интернете», «Понять изображение», «Поиск MiniMax для меня», «Описать это изображение».
