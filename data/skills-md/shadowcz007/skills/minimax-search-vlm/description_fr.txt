Utilisez l'API MiniMax Coding Plan pour la recherche sur le Web (web_search) et la compréhension des images (understand_image). Vous devez configurer MINIMAX_CP_API_KEY (clé API MiniMax, commençant généralement par sk-cp-). Exemples de déclencheurs : "Recherche MiniMax", "Compréhension de l'image", "Recherche sur le Web", "Comprendre l'image", "Recherche MiniMax", "Décrire cette image".
