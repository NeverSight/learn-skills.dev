Utilizza l'API MiniMax Coding Plan per la ricerca sul Web (web_search) e la comprensione delle immagini (understand_image). È necessario configurare MINIMAX_CP_API_KEY (chiave API MiniMax, che di solito inizia con sk-cp-). Esempi di trigger: "Ricerca MiniMax", "Comprensione dell'immagine", "ricerca sul web", "comprendi l'immagine", "ricerca MiniMax", "Descrivi questa immagine".
