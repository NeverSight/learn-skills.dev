Baidu Maps JavaScript API UI-Kit-Komponentenbibliotheks-Entwicklungshandbuch. Stellt Verwendungsreferenzen für die Komponenten PlaceAutocomplete, PlaceSearch, PlaceDetail und RoutePlan bereit. Verwenden Sie diese Fähigkeit, wenn Benutzer: (1) die automatische Vervollständigung für Eingabefelder für die Standortsuche implementieren müssen, (2) die Standortsuchfunktion auf Baidu Maps implementieren, (3) POI-Detailinformationen anzeigen, (4) die Routenplanungsfunktionalität implementieren müssen, (5) die API-Verwendung von @baidumap/jsapi-ui-kit verstehen müssen, (6) Rückrufe von Komponentenereignissen verarbeiten müssen.
