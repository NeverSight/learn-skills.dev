Baidu Map JavaScript API UI-Kit 구성 요소 라이브러리 개발 가이드. PlaceAutocomplete, PlaceSearch, PlaceDetail 및 RoutePlan 구성 요소에 대한 사용 참조를 제공합니다. 사용자가 (1) 위치 검색 입력 상자에 대한 자동 완성 구현, (2) Baidu 지도에 위치 검색 기능 구현, (3) POI 세부 정보 표시, (4) 운전 경로 계획 기능 구현, (5) @baidumap/jsapi-ui-kit의 API 사용 이해, (6) 구성 요소 이벤트 콜백 처리가 필요할 때 이 기술을 사용합니다.
