Guía de desarrollo de la biblioteca de componentes del kit de interfaz de usuario de la API JavaScript de Baidu Maps. Proporciona referencias de uso para los componentes PlaceAutocomplete, PlaceSearch, PlaceDetail y RoutePlan. Utilice esta habilidad cuando los usuarios necesiten: (1) implementar el autocompletado para los cuadros de entrada de búsqueda de ubicación, (2) implementar la funcionalidad de búsqueda de ubicación en Baidu Maps, (3) mostrar información detallada de PDI, (4) implementar la funcionalidad de planificación de rutas de conducción, (5) comprender el uso de API de @baidumap/jsapi-ui-kit, (6) manejar devoluciones de llamadas de eventos de componentes.
