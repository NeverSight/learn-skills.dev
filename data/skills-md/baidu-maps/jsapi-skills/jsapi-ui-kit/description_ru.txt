Руководство по разработке библиотеки компонентов пользовательского интерфейса Baidu Maps JavaScript API ui-kit. Предоставляет ссылки на использование компонентов PlaceAutocomplete, PlaceSearch, PlaceDetail и RoutePlan. Используйте этот навык, когда пользователям необходимо: (1) реализовать автозаполнение полей ввода для поиска местоположения, (2) реализовать функцию поиска местоположения на картах Baidu, (3) отобразить подробную информацию о POI, (4) реализовать функцию планирования маршрута вождения, (5) понять использование API @baidumap/jsapi-ui-kit, (6) обрабатывать обратные вызовы событий компонента.
