Baidu Maps JavaScript API UI-Kit コンポーネント ライブラリ開発ガイド。 PlaceAutocomplete、PlaceSearch、PlaceDetail、および RoutePlan コンポーネントの使用方法のリファレンスを提供します。このスキルは、ユーザーが次のことを行う必要がある場合に使用します: (1) 位置検索入力ボックスのオートコンプリートを実装する、(2) Baidu Maps に位置検索機能を実装する、(3) POI の詳細情報を表示する、(4) 運転ルート計画機能を実装する、(5) @baidumap/jsapi-ui-kit の API 使用法を理解する、(6) コンポーネント イベント コールバックを処理する。
