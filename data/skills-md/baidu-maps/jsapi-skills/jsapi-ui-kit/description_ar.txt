دليل تطوير مكتبة مكونات Baidu Maps JavaScript API UI-Kit. يوفر مراجع الاستخدام لمكونات PlaceAutocomplete وPlaceSearch وPlaceDetail وRoutePlan. استخدم هذه المهارة عندما يحتاج المستخدمون إلى: (1) تنفيذ الإكمال التلقائي لمربعات إدخال البحث عن الموقع، (2) تنفيذ وظيفة البحث عن الموقع على خرائط Baidu، (3) عرض معلومات تفصيلية عن النقاط المهمة، (4) تنفيذ وظيفة تخطيط مسار القيادة، (5) فهم استخدام واجهة برمجة التطبيقات @baidumap/jsapi-ui-kit، (6) التعامل مع عمليات الاسترجاعات لحدث المكون.
