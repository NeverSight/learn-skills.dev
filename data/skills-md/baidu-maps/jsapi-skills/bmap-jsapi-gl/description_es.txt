Guía de desarrollo de Baidu Maps JSAPI WebGL (BMapGL). Aplique esta habilidad al escribir, revisar o depurar código que utilice la API de Baidu Maps. Adecuado para tareas que implican inicialización de mapas, representación de superposiciones, gestión de capas, manejo de eventos, interacción de controles u optimización del rendimiento. Se activa automáticamente cuando los usuarios mencionan BMapGL, Baidu Maps, jsapi-gl o necesidades de desarrollo de mapas relacionados.
