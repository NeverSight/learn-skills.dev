دليل تطوير خرائط بايدو JSAPI WebGL (BMapGL). قم بتطبيق هذه المهارة عند كتابة أو مراجعة أو تصحيح التعليمات البرمجية التي تستخدم Baidu Maps API. مناسب للمهام التي تتضمن تهيئة الخريطة، أو عرض التراكب، أو إدارة الطبقة، أو معالجة الأحداث، أو تفاعل التحكم، أو تحسين الأداء. يتم تشغيله تلقائيًا عندما يذكر المستخدمون BMapGL أو Baidu Maps أو jsapi-gl أو احتياجات تطوير الخرائط ذات الصلة.
