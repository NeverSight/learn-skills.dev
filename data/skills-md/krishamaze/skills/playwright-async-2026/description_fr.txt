Modèles d'API Python asynchrones du dramaturge pour 2026. Utilisez cette compétence chaque fois que vous écrivez du code d'automatisation de navigateur en Python : navigation dans les pages, interaction entre les éléments, sélecteurs, contexte persistant, état de stockage et gestion des événements. Il s'agit de l'API SOUS-JACENTE enveloppée par Camoufox. Sachez-le pour comprendre ce qu’expose Camoufox. Utilisez TOUJOURS async_api (et non sync_api) lors de l'intégration avec FastAPI ou tout service Python asynchrone.
