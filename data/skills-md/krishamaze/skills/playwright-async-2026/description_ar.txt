أنماط Playwright async Python API لعام 2026. استخدم هذه المهارة عند كتابة تعليمات برمجية لأتمتة المتصفح في Python - التنقل بين الصفحات، وتفاعل العناصر، والمحددات، والسياق المستمر، وحالة التخزين، ومعالجة الأحداث. هذه هي واجهة برمجة التطبيقات الأساسية التي يغلفها Camoufox. اعرف هذا لتفهم ما يكشفه Camoufox. استخدم دائمًا async_api (وليس sync_api) عند التكامل مع FastAPI أو أي خدمة Python غير متزامنة.
