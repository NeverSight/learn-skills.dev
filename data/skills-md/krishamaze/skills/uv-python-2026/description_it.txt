uv: il gestore di pacchetti Python standard 2026 (sostituisce pip, venv, pyenv, pip-tools, pipx). Utilizza questa competenza per TUTTA la gestione delle dipendenze Python, gli ambienti virtuali, la gestione delle versioni Python e le configurazioni Docker Python. Non utilizzare MAI bare pip per i nuovi progetti nel 2026. Copre uv init, uv add, uv sync, lockfiles, pyproject.toml, integrazione Docker e migrazione da pip/Poetry.
