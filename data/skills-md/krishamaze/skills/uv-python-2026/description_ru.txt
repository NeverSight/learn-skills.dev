uv — стандартный менеджер пакетов Python 2026 года (заменяет pip, venv, pyenv, pip-tools, pipx). Используйте этот навык для ВСЕХ функций управления зависимостями Python, виртуальных сред, управления версиями Python и настроек Docker Python. НИКОГДА не используйте голый pip для новых проектов в 2026 году. Охватывает uv init, uv add, uv sync, lockfiles, pyproject.toml, интеграцию Docker и миграцию из pip/Poetry.
