أنماط FastAPI async Python API لعام 2026. استخدم هذه المهارة عند إنشاء طبقات API لوكلاء التشغيل الآلي - تعريفات المسار، والمعالجات غير المتزامنة، ومهام الخلفية، وWebSocket، ونماذج Pydantic v2، وأحداث العمر، وCORS، ونشر الإنتاج باستخدام عمال Gunicorn + Uvicorn. استخدم دائمًا تعريف غير متزامن للمسارات المرتبطة بالإدخال/الإخراج. لا تستخدم أبدًا مكالمات قاعدة البيانات المتزامنة في المسارات غير المتزامنة.
