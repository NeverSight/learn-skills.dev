Modèles d'API Python asynchrone FastAPI pour 2026. Utilisez cette compétence lors de la création de couches API pour les agents d'automatisation : définitions de routes, gestionnaires asynchrones, tâches en arrière-plan, WebSocket, modèles Pydantic v2, événements de durée de vie, CORS et déploiement de production avec les travailleurs Gunicorn + Uvicorn. Utilisez TOUJOURS async def pour les routes liées aux E/S. N'utilisez JAMAIS d'appels de base de données de synchronisation dans les routes asynchrones.
