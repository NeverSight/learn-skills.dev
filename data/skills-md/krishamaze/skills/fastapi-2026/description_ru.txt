Шаблоны асинхронного API Python FastAPI для 2026 года. Используйте этот навык при создании уровней API для агентов автоматизации — определений маршрутов, асинхронных обработчиков, фоновых задач, моделей WebSocket, Pydantic v2, событий продолжительности жизни, CORS и производственного развертывания с рабочими процессами Gunicorn + Uvicorn. ВСЕГДА используйте async def для маршрутов, связанных с вводом-выводом. НИКОГДА не используйте вызовы синхронизации БД в асинхронных маршрутах.
