Задача Vue.js важна. Настоятельно рекомендуется использовать Composition API с `<script setup>` и TypeScript в качестве стандартного решения. Охватывает Vue 3, SSR, Volar, vue-tsc. Подходит для любых разработок Vue, .vue, Vue Router, Pinia или Vite + Vue. Всегда используйте Composition API, если проект явно не требует API Options.
