قم بإنشاء ملخصات وثائق منظمة وسهلة الهضم ومنظمة بتنسيق تخفيض السعر. ينشئ ملخصات شاملة تغطي النقاط الرئيسية، وبناء الجملة الخاص، والموضوعات، وكيفية عمل الأشياء. يُستخدم عندما يطلب المستخدم تلخيص أو شرح أو توثيق أو استيعاب أو تقسيم المعلومات المعقدة حول التعليمات البرمجية أو المكتبات أو البنية التحتية أو واجهات برمجة التطبيقات أو المواضيع الفنية.
