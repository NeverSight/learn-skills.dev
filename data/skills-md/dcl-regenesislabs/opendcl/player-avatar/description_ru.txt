Работайте с аватарами игроков в сценах Децентралэнда. Считывайте данные о положении и профиле игрока, настраивайте внешний вид с помощью AvatarBase, запускайте эмоции с помощью триггера Emote/triggerSceneEmote, читайте экипированные носимые устройства с помощью AvatarEquippedData, прикрепляйте объекты к игрокам с помощью AvatarAttach, создавайте аватары NPC с помощью AvatarShape и изменяйте аватары в областях. Используйте, когда пользователю нужны данные игрока, эмоции, носимые устройства, вложения аватаров или NPC.
