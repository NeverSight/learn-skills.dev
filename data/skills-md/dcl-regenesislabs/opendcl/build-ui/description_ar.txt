أنشئ واجهات مستخدم ثنائية الأبعاد لمشاهد Decentraland باستخدام React-ECS. قم بإنشاء HUDs والقوائم والأشرطة الصحية ولوحات النتائج ومربعات الحوار والأزرار ونماذج الإدخال. يُستخدم عندما يريد المستخدم إضافة واجهة مستخدم أو شاشة عرض معلوماتية (HUD) أو أزرار أو تراكبات نصية أو قوائم أو عناصر تظهر على الشاشة.
