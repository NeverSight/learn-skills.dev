게임 상태를 제어하고 변경 사항을 검증하며 부정 행위를 방지하는 헤드리스 권한 서버로 멀티플레이어 장면을 구축하세요. @dcl/sdk@auth-server를 설치하고 hammurabi-server로 실행합니다. isServer()를 사용하여 논리를 분기하고, RegisterMessages()를 클라이언트-서버 통신에 사용하고, 서버 전용 구성 요소에 verifyBeforeChange()를 사용하고, 지속성을 위해 Storage를, 구성을 위해 EnvVar를 사용합니다. 사용자가 신뢰할 수 있는 서버, 치트 방지, 서버 측 유효성 검사, 영구 저장소, 환경 변수 또는 서버 메시지를 원할 때 사용합니다.
