Создавайте многопользовательские сцены с помощью автономного авторитетного сервера, который контролирует состояние игры, проверяет изменения и предотвращает мошенничество. Установите @dcl/sdk@auth-server и запустите с помощью Hammurabi-server. Используйте isServer() для логики ветвления, RegisterMessages() для связи клиент-сервер, validateBeforeChange() для серверных компонентов, Storage для постоянства и EnvVar для конфигурации. Используйте, когда пользователю нужен авторитетный сервер, защита от мошенничества, проверка на стороне сервера, постоянное хранилище, переменные среды или сообщения сервера.
