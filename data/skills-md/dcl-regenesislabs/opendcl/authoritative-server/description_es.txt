Cree escenas multijugador con un servidor autorizado sin cabeza que controla el estado del juego, valida los cambios y evita las trampas. Instale @dcl/sdk@auth-server y ejecútelo con hammurabi-server. Utilice isServer() para bifurcar la lógica, RegisterMessages() para la comunicación cliente-servidor, validarBeforeChange() para componentes solo de servidor, Almacenamiento para persistencia y EnvVar para configuración. Úselo cuando el usuario desee un servidor autorizado, antitrampas, validación del lado del servidor, almacenamiento persistente, variables de entorno o mensajes del servidor.
