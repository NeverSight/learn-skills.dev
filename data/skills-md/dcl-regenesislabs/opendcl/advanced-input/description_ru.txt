Расширенная обработка ввода в сценах Децентралэнда. Используйте PointerLock для определения состояния захвата курсора, InputModifier для фиксации или ограничения движения игрока, PrimaryPointerInfo для положения курсора и мирового луча, inputSystem.getInputCommand для опроса ввода для каждого объекта и шаблонов клавиатуры для управления движением WASD. Используйте, когда пользователю нужен индивидуальный ввод, управление курсором, ограничение движения, обработка клавиатуры, управление частотой кадров или опрос ввода.
