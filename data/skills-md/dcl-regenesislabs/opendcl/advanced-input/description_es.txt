Manejo avanzado de entradas en escenas de Decentraland. Utilice PointerLock para detectar el estado de captura del cursor, InputModifier para congelar o restringir el movimiento del jugador, PrimaryPointerInfo para la posición del cursor y el rayo mundial, inputSystem.getInputCommand para sondeo de entrada por entidad y patrones de teclado para controles de movimiento WASD. Úselo cuando el usuario desee entrada personalizada, control del cursor, restricción de movimiento, manejo del teclado, controles FPS o sondeo de entrada.
