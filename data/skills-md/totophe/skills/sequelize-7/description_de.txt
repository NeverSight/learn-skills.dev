Umfassende Referenz für Sequelize v7 (Alpha), das TypeScript-first Node.js ORM, das PostgreSQL, MySQL, MariaDB, SQLite, MSSQL, DB2 und Snowflake unterstützt. Verwendung bei der Arbeit mit dem @sequelize/core-Paket: Definieren von Modellen mit Dekoratoren, Schreiben von Datenbankabfragen (CRUD, Assoziationen, Roh-SQL), Verwalten von Transaktionen, Hooks, Verbindungspools, Einrichten von Assoziationen oder Migration von Sequelize v6 auf v7.
