مرجع شامل لـ Sequelize v7 (alpha)، وTypeScript-first Node.js ORM الذي يدعم PostgreSQL، وMySQL، وMariaDB، وSQLite، وMSSQL، وDB2، وSnowflake. يُستخدم عند العمل مع الحزمة @sequelize/core: تحديد النماذج باستخدام أدوات الديكور، أو كتابة استعلامات قاعدة البيانات (CRUD، أو الارتباطات، أو SQL الخام)، أو إدارة المعاملات، أو الخطافات، أو تجمعات الاتصال، أو إعداد الارتباطات، أو الترحيل من Sequelize v6 إلى v7.
