Riferimento completo per Sequelize v7 (alpha), il primo ORM Node.js di TypeScript che supporta PostgreSQL, MySQL, MariaDB, SQLite, MSSQL, DB2 e Snowflake. Da utilizzare quando si lavora con il pacchetto @sequelize/core: definizione di modelli con decoratori, scrittura di query di database (CRUD, associazioni, SQL non elaborato), gestione di transazioni, hook, pool di connessioni, impostazione di associazioni o migrazione da Sequelize v6 a v7.
