Referencia completa para Sequelize v7 (alfa), el primer ORM Node.js de TypeScript que admite PostgreSQL, MySQL, MariaDB, SQLite, MSSQL, DB2 y Snowflake. Úselo cuando trabaje con el paquete @sequelize/core: defina modelos con decoradores, escriba consultas de bases de datos (CRUD, asociaciones, SQL sin formato), administre transacciones, enlaces, grupos de conexiones, configure asociaciones o migre de Sequelize v6 a v7.
