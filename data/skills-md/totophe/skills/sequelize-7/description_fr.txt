Référence complète pour Sequelize v7 (alpha), le premier ORM Node.js TypeScript prenant en charge PostgreSQL, MySQL, MariaDB, SQLite, MSSQL, DB2 et Snowflake. À utiliser lorsque vous travaillez avec le package @sequelize/core : définition de modèles avec des décorateurs, écriture de requêtes de base de données (CRUD, associations, SQL brut), gestion des transactions, des hooks, des pools de connexions, configuration d'associations ou migration de Sequelize v6 vers v7.
