Aiuta l'utente a ribasare in modo sicuro un ramo di funzionalità su "main", "develop" o su un altro ramo di destinazione preservando il comportamento delle funzionalità in caso di conflitti. Utilizza questa abilità ogni volta che l'utente chiede di rebase o aggiornare un ramo di lunga durata, riprodurre una funzionalità su una base più recente o risolvere conflitti di rebase dopo refactoring, API rinominate, file spostati, modifiche di digitazione o nuove convenzioni sul ramo di destinazione.
