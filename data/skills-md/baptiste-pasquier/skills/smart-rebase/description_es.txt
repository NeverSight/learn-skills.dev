Ayude al usuario a cambiar de forma segura una rama de función en "principal", "desarrollo" u otra rama de destino mientras preserva el comportamiento de la función a través de conflictos. Utilice esta habilidad siempre que el usuario solicite cambiar la base o actualizar una rama de larga duración, reproducir una característica en una base más nueva o resolver conflictos de base después de refactorizaciones, API renombradas, archivos movidos, cambios de escritura o nuevas convenciones en la rama de destino.
