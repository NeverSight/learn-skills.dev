Créez, exécutez et déployez des agents IA à l'aide du kit de développement d'agents (ADK) de Google pour Python. Utilisez cette compétence chaque fois que l'utilisateur souhaite : créer un agent ADK, écrire des outils pour les agents ADK, créer des pipelines multi-agents (séquentiels, parallèles, en boucle), gérer l'état de session ou la mémoire, configurer des rappels, déployer sur Vertex AI Agent Engine ou Cloud Run, configurer des modèles LLM (Gemini, Claude, LiteLLM, Ollama), utiliser des outils MCP avec ADK, diffuser des réponses d'agent ou intégrer ADK avec des API externes. Déclenchez des expressions telles que "agent ADK", "google adk", "kit de développement d'agent", "créer un agent avec ADK", "outils adk", "déploiement adk" ou toute requête impliquant une orchestration multi-agents avec le framework de Google. Utilisez toujours cette compétence lorsque ADK Python est impliqué, même pour des questions en une seule étape, car elle contient des modèles d'API critiques, des exigences en matière de structure de code et des connaissances en matière de déploiement qui ne font pas partie des données de formation générales.
