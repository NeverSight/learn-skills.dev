Erstellen, führen und implementieren Sie KI-Agenten mit dem Agent Development Kit (ADK) für Python von Google. Verwenden Sie diese Fähigkeit, wann immer der Benutzer Folgendes möchte: einen ADK-Agenten erstellen, Tools für ADK-Agenten schreiben, Multi-Agent-Pipelines (sequentiell, parallel, Schleife) erstellen, den Sitzungsstatus oder den Speicher verwalten, Rückrufe einrichten, auf Vertex AI Agent Engine oder Cloud Run bereitstellen, LLM-Modelle konfigurieren (Gemini, Claude, LiteLLM, Ollama), MCP-Tools mit ADK verwenden, Agentenantworten streamen oder ADK mit externen APIs integrieren. Lösen Sie auf Phrasen wie „ADK-Agent“, „Google Adk“, „Agent Development Kit“, „Build an Agent with ADK“, „Adk Tools“, „Adk Deployment“ oder jeder Anfrage aus, die die Orchestrierung mehrerer Agenten mit dem Google-Framework beinhaltet. Verwenden Sie diese Fertigkeit immer, wenn es um ADK Python geht – auch bei einstufigen Fragen –, da sie wichtige API-Muster, Anforderungen an die Codestruktur und Bereitstellungswissen enthält, die nicht in allgemeinen Trainingsdaten enthalten sind.
