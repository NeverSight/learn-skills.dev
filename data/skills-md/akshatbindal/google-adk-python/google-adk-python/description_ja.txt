Google の Python 用エージェント開発キット (ADK) を使用して、AI エージェントを構築、実行、デプロイします。ユーザーが必要なときにこのスキルを使用します。ADK エージェントの作成、ADK エージェント用のツールの作成、マルチエージェント パイプライン (シーケンシャル、並列、ループ) の構築、セッション状態またはメモリの管理、コールバックの設定、Vertex AI Agent Engine または Cloud Run へのデプロイ、LLM モデル (Gemini、Claude、LiteLLM、Ollama) の構成、ADK での MCP ツールの使用、エージェント応答のストリーミング、または ADK と外部 API の統合。 「ADK エージェント」、「Google adk」、「エージェント開発キット」、「ADK でエージェントを構築」、「adk ツール」、「adk デプロイ」などのフレーズ、または Google のフレームワークを使用したマルチエージェント オーケストレーションを含むリクエストでトリガーされます。 ADK Python が関係する場合は、たとえシングルステップの質問であっても、常にこのスキルを使用してください。これには、一般的なトレーニング データには含まれない重要な API パターン、コード構造要件、デプロイメントの知識が含まれているためです。
