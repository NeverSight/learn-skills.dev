Создавайте, запускайте и развертывайте агенты ИИ с помощью комплекта разработки агентов Google (ADK) для Python. Используйте этот навык всякий раз, когда пользователь хочет: создать агент ADK, написать инструменты для агентов ADK, построить многоагентные конвейеры (последовательные, параллельные, циклические), управлять состоянием сеанса или памятью, настроить обратные вызовы, развернуть в Vertex AI Agent Engine или Cloud Run, настроить модели LLM (Gemini, Claude, LiteLLM, Ollama), использовать инструменты MCP с ADK, транслировать ответы агента или интегрировать ADK с внешними API. Запускайте такие фразы, как «агент ADK», «google adk», «комплект разработки агента», «создайте агент с помощью ADK», «инструменты adk», «развертывание adk» или любой запрос, включающий оркестровку нескольких агентов с помощью платформы Google. Всегда используйте этот навык, когда используется ADK Python — даже для одношаговых вопросов — поскольку он содержит важные шаблоны API, требования к структуре кода и знания по развертыванию, а не общие данные для обучения.
