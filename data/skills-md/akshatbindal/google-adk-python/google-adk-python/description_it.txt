Crea, esegui e distribuisci agenti IA utilizzando l'Agent Development Kit (ADK) di Google per Python. Utilizza questa competenza ogni volta che l'utente desidera: creare un agente ADK, scrivere strumenti per agenti ADK, creare pipeline multi-agente (sequenziali, parallele, in loop), gestire lo stato o la memoria della sessione, impostare callback, eseguire la distribuzione su Vertex AI Agent Engine o Cloud Run, configurare modelli LLM (Gemini, Claude, LiteLLM, Ollama), utilizzare strumenti MCP con ADK, trasmettere in streaming le risposte degli agenti o integrare ADK con API esterne. Attiva frasi come "agente ADK", "google adk", "kit di sviluppo agente", "crea un agente con ADK", "strumenti adk", "distribuzione adk" o qualsiasi richiesta che coinvolga l'orchestrazione multi-agente con il framework di Google. Utilizza sempre questa competenza quando è coinvolto ADK Python, anche per domande a passaggio singolo, poiché contiene modelli API critici, requisiti della struttura del codice e conoscenze di distribuzione non nei dati di formazione generali.
