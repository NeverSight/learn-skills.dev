يمكنك إنشاء وكلاء الذكاء الاصطناعي وتشغيلهم ونشرهم باستخدام Google Agent Development Kit (ADK) لـ Python. استخدم هذه المهارة عندما يريد المستخدم: إنشاء وكيل ADK، أو كتابة أدوات لوكلاء ADK، أو إنشاء خطوط أنابيب متعددة الوكلاء (متسلسلة، متوازية، حلقة)، أو إدارة حالة الجلسة أو الذاكرة، أو إعداد عمليات رد الاتصال، أو النشر إلى Vertex AI Agent Engine أو Cloud Run، أو تكوين نماذج LLM (Gemini، Claude، LiteLLM، Ollama)، استخدام أدوات MCP مع ADK، أو تدفق استجابات الوكيل، أو دمج ADK مع واجهات برمجة التطبيقات الخارجية. قم بتشغيل عبارات مثل "وكيل ADK"، أو "google adk"، أو "مجموعة تطوير الوكيل"، أو "إنشاء وكيل باستخدام ADK"، أو "أدوات adk"، أو "نشر adk"، أو أي طلب يتضمن تنسيقًا متعدد الوكلاء مع إطار عمل Google. استخدم هذه المهارة دائمًا عندما يتعلق الأمر بـ ADK Python - حتى بالنسبة للأسئلة ذات الخطوة الواحدة - لأنها تحتوي على أنماط واجهة برمجة التطبيقات المهمة ومتطلبات بنية التعليمات البرمجية ومعرفة النشر وليس في بيانات التدريب العامة.
