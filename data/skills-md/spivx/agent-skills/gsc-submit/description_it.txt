Invia URL a Google per la scansione tramite l'API di indicizzazione: supporta elenchi di URL manuali, importazioni di file (txt, CSV, sitemap.xml) e rilevamento automatico del percorso dalla codebase. Genera riepilogo del terminale + report HTML. Da utilizzare quando l'utente desidera inviare URL a Google, richiedere l'indicizzazione, notificare a Google pagine nuove o aggiornate, reindicizzare pagine, rimuovere URL da Google, "inviare la mia mappa del sito", "dire a Google di eseguire la scansione di queste pagine", "indicizzare le mie nuove pagine", "inviare percorsi a Google" o qualsiasi cosa relativa all'invio di URL di Google e alle richieste di indicizzazione.
