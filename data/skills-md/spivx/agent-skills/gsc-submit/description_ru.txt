Отправка URL-адресов в Google для сканирования через API индексирования — поддерживает списки URL-адресов вручную, импорт файлов (txt, CSV, sitemap.xml) и автоматическое определение маршрутов из базы кода. Генерирует сводку терминала + отчет HTML. Используйте, когда пользователь хочет отправить URL-адреса в Google, запросить индексацию, уведомить Google о новых или обновленных страницах, переиндексировать страницы, удалить URL-адреса из Google, «отправить мою карту сайта», «попросить Google просканировать эти страницы», «индексировать мои новые страницы», «отправить маршруты в Google» или что-либо, связанное с запросами на отправку и индексирование URL-адресов Google.
