Indexing API 経由でクロールする URL を Google に送信します。手動 URL リスト、ファイル インポート (txt、CSV、sitemap.xml)、コードベースからの自動ルート検出をサポートします。端末概要 + HTML レポートを生成します。ユーザーが Google への URL の送信、インデックス作成のリクエスト、新しいページまたは更新されたページの Google への通知、ページの再インデックス作成、Google からの URL の削除、「サイトマップを送信する」、「これらのページをクロールするように Google に指示する」、「新しいページをインデックスに登録する」、「Google にルートを送信する」、または Google URL の送信およびインデックス作成リクエストに関連するあらゆる場合に使用します。
