Monitorare le azioni GitHub CI/CD viene eseguito dopo il push. Caricamento automatico dopo qualsiasi comando "jj git push" o "git push". Da utilizzare anche quando si controlla lo stato dell'elemento della configurazione o si osserva una pipeline. Rileva automaticamente le filiali, deduplica i monitor, segnala l'esito positivo/negativo con i registri non riusciti.
