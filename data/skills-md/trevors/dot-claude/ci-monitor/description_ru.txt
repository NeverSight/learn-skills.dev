Мониторинг действий GitHub CI/CD запускается после отправки. Автоматическая загрузка после любой команды jj git push или git push. Также используйте при проверке статуса CI или просмотре конвейера. Автоматическое обнаружение ветвей, дедупликация мониторов, отчеты об успешном/неудачном выполнении с журналами ошибок.
