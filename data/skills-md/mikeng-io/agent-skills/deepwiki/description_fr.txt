Compétence facultative d'intelligence de base de code : enveloppe le serveur DeepWiki MCP de Devin pour fournir une recherche de documentation structurée, une découverte de sujets et des questions-réponses en langage naturel sur tout référentiel GitHub indexé par Devin. Prend en charge les référentiels publics et privés (le privé nécessite la clé API Devin). Revient gracieusement à Glob/Grep/Read + QMD s'il n'est pas configuré. Invocable de manière autonome ou appelé par d'autres compétences en tant que source de données.
