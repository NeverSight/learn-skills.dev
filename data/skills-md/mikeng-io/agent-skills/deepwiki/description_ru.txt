Дополнительный навык анализа базы кода — включает сервер DeepWiki MCP Devin для обеспечения структурированного поиска документации, обнаружения тем и вопросов и ответов на естественном языке о любом репозитории GitHub, проиндексированном Devin. Поддерживает общедоступные и частные репозитории (для частного требуется ключ API Devin). Грамотно возвращается к Glob/Grep/Read + QMD, если не настроено. Вызывается автономно или вызывается другими навыками в качестве источника данных.
