Эталонный адаптер для отправки Claude (Anthropic). Чтение любым навыком оркестровки с помощью инструмента «Чтение». Определяет, как вызывать субагенты Claude или Anthropic API с проверкой доступности. Может использоваться любым оркестратором ИИ — Claude Code, OpenCode, Codex, Gemini или пользовательскими агентами.
