Многомодельный обзорный совет. Отправляет задачи проверки всем доступным мостовым адаптерам (Claude, Gemini, Codex, OpenCode), каждый из которых запускает внутренних субагентов. Синтезирует результаты по семействам моделей, используя протокол дебатов. Можно использовать автономно или в качестве второго прохода после глубокой проверки. Независимость от оркестратора — может вызываться с помощью Claude Code, OpenCode или Codex CLI.
