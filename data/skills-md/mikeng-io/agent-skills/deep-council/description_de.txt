Multi-Modell-Überprüfungsrat. Versendet Überprüfungsaufgaben an alle verfügbaren Bridge-Adapter (Claude, Gemini, Codex, OpenCode), von denen jeder interne Subagenten ausführt. Synthetisiert Ergebnisse über Modellfamilien hinweg mithilfe des Debattenprotokolls. Kann eigenständig oder als zweiter Durchgang von Deep-Verify verwendet werden. Orchestrator-unabhängig – kann von Claude Code, OpenCode oder Codex CLI aufgerufen werden.
