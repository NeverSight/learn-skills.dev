Enriquecimiento de búsqueda web opcional a través de Brave Search MCP. Proporciona búsqueda web, de noticias y local con un índice independiente (sin dependencia de Google). Se puede invocar de forma independiente o mediante una investigación profunda, un explorador profundo o cualquier habilidad que necesite una búsqueda en la web. Sin bloqueo: recurre a otras herramientas de búsqueda si no están disponibles.
