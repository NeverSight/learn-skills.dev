Легкое уточнение области применения. Вызывается навыками deep-*, когда контекст разговора слишком скуден, чтобы определить, что анализировать. Задает 1–3 целевых вопроса по одному, затем возвращает структурированный блокscope_clarification для использования вызывающим навыком. Адаптировано на основе навыка мозгового штурма сверхспособностей (https://github.com/obra/superpowers) — тот же принцип: задавать вопросы, прежде чем действовать, но направлен на проверку/анализ намерений, а не на разработку функций.
