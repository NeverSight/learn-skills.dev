Эталонный адаптер для OpenCode — мост с несколькими поставщиками, не зависящий от модели. Чтение любым навыком оркестровки с помощью инструмента «Чтение». Охватывает предполетные проверки с интерактивными рекомендациями, путь к серверу HTTP API (предпочтительно), путь запуска CLI в качестве резервного варианта, встроенные правильные флаги и формат модели. Подходит для глубокого совета, глубокого анализа, глубокого аудита или любого будущего навыка, требующего многомодельной проверки.
