Adaptateur de référence pour OpenCode — pont multi-fournisseur indépendant du modèle. Lecture par n'importe quelle compétence d'orchestration via l'outil de lecture. Couvre les vérifications avant vol avec des conseils interactifs, le chemin du serveur API HTTP (de préférence), le chemin d'exécution CLI comme solution de secours, les indicateurs corrects et le format de modèle intégré. Utilisable par un conseil approfondi, un examen approfondi, un audit approfondi ou toute compétence future nécessitant un examen multi-modèle.
