Adaptador de referencia para OpenCode: puente multiproveedor independiente del modelo. Lea con cualquier habilidad de orquestación a través de la herramienta Leer. Cubre comprobaciones previas al vuelo con aviso interactivo, ruta del servidor API HTTP (preferida), ruta de ejecución de CLI como alternativa, indicadores correctos y formato de modelo integrado. Utilizable por consejo profundo, revisión profunda, auditoría profunda o cualquier habilidad futura que necesite una revisión de múltiples modelos.
