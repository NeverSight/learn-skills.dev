المحول المرجعي لـ OpenCode — جسر متعدد الموفرين لا يعتمد على النموذج. اقرأ بأي مهارة تنسيقية عبر أداة القراءة. يغطي فحوصات ما قبل الرحلة مع الاستشارات التفاعلية، ومسار خادم HTTP API (المفضل)، ومسار تشغيل CLI كبديل، والأعلام الصحيحة وتنسيق النموذج المضمن. يمكن استخدامها من قبل المجلس العميق، أو المراجعة العميقة، أو التدقيق العميق، أو أي مهارة مستقبلية تحتاج إلى مراجعة متعددة النماذج.
