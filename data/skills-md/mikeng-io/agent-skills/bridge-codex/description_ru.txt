Эталонный адаптер для мультиагентной проверки Кодекса. Чтение любым навыком оркестровки с помощью инструмента «Чтение». Путь к серверу MCP (предпочтительный) с возможностью автоматической настройки, путь CLI в качестве резервного варианта, интерактивные предполетные рекомендации, если они не настроены, встроены правильные флаги. Может использоваться в рамках глубокого совета, глубокого анализа, глубокого аудита или любого будущего навыка, требующего проверки на основе Кодекса.
