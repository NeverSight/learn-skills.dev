Workleap の共有 Web 構成パッケージ (@workleap/eslint-configs、typescript-configs、rsbuild-configs、rslib-configs、stylelint-configs、browserslist-config)。

次の場合に使用します。
(1) Workleap プロジェクトでの共有 Web ツール構成のセットアップまたはカスタマイズ
(2) プロジェクトタイプ別のESLintの設定(Webアプリ、Reactライブラリ、TSライブラリ、モノリポジトリ)
(3) プロジェクト タイプ別の TypeScript の設定 (Web アプリケーション、ライブラリ、モノリポジトリ ワークスペース)
(4) Rsbuild または Rslib バンドルの構成 (dev、build、Storybook)
(5) Stylelint、Browserslist、またはモノリポジトリ (Turborepo) とポリリポジトリの戦略の構成
(6) 共有構成の拡張またはカスタマイズ、ESM/ESNext 制約のトラブルシューティング
