Guía para @workleap/logging, una biblioteca de registro estructurada para aplicaciones TypeScript frontend de Workleap.

Utilice esta habilidad cuando:
(1) Instalar o configurar registradores @workleap/logging (BrowserConsoleLogger, CompositeLogger)
(2) Uso de la API @workleap/logging: niveles de registro, segmentos encadenados, alcances, salida con estilo
(3) Componer o filtrar registradores @workleap/logging para registros multidestino
(4) Integración de @workleap/logging con LogRocket a través de CompositeLogger
(5) Revisar las relaciones públicas o solucionar problemas de uso de @workleap/registro
