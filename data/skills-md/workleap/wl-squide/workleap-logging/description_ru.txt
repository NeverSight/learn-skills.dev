Руководство по @workleap/logging, библиотеке структурированного ведения журналов для интерфейсных приложений Workleap TypeScript.

Используйте этот навык, когда:
(1) Установка или настройка регистраторов @workleap/журналов (BrowserConsoleLogger, CompositeLogger)
(2) Использование API @workleap/logging: уровни журналирования, цепочки сегментов, области действия, стилизованный вывод.
(3) Составление или фильтрация журналов @workleap/logging для ведения журналов по нескольким адресатам.
(4) Интеграция @workleap/logging с LogRocket через CompositeLogger.
(5) Просмотр запросов на запросы или устранение неполадок @workleap/регистрации использования
