Guide pour @workleap/logging, une bibliothèque de journalisation structurée pour les applications TypeScript frontales Workleap.

Utilisez cette compétence lorsque :
(1) Installation ou configuration des enregistreurs @workleap/logging (BrowserConsoleLogger, CompositeLogger)
(2) Utilisation de l'API @workleap/logging : niveaux de journalisation, segments chaînés, portées, sortie stylisée
(3) Composition ou filtrage des enregistreurs @workleap/logging pour la journalisation multi-destination
(4) Intégration de @workleap/logging avec LogRocket via CompositeLogger
(5) Examen des PR ou dépannage de l'utilisation de @workleap/logging
