Workleap フロントエンド TypeScript アプリケーション用の構造化ログ ライブラリである @workleap/logging のガイド。

このスキルは次の場合に使用します。
(1) @workleap/logging ロガー (BrowserConsoleLogger、CompositeLogger) のセットアップまたは構成
(2) @workleap/logging API の使用: ログ レベル、連鎖セグメント、スコープ、スタイル付き出力
(3) 複数宛先ロギング用の @workleap/logging ロガーの構成またはフィルタリング
(4) CompositeLogger を介した @workleap/logging と LogRocket の統合
(5) PR の確認または @workleap/logging の使用法のトラブルシューティング
