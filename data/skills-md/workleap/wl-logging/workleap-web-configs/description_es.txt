Paquetes de configuración web compartidos de Workleap (@workleap/eslint-configs, typescript-configs, rsbuild-configs, rslib-configs, stylelint-configs, browserslist-config).

Úselo cuando:
(1) Configurar o personalizar configuraciones de herramientas web compartidas en un proyecto de Workleap
(2) Configurar ESLint por tipo de proyecto (aplicación web, biblioteca React, biblioteca TS, monorepo)
(3) Configuración de TypeScript por tipo de proyecto (aplicación web, biblioteca, espacio de trabajo monorepo)
(4) Configuración del paquete Rsbuild o Rslib (dev, build, Storybook)
(5) Configuración de Stylelint, Browserslist o monorepo (Turborepo) frente a estrategias de polirepo
(6) Ampliar o personalizar configuraciones compartidas, solucionar problemas de restricciones de ESM/ESNext
