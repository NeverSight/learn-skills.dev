Packages de configuration Web partagés de Workleap (@workleap/eslint-configs, typescript-configs, rsbuild-configs, rslib-configs, stylelint-configs, browserslist-config).

À utiliser lorsque :
(1) Configuration ou personnalisation des configurations d'outils Web partagés dans un projet Workleap
(2) Configuration d'ESLint par type de projet (application web, bibliothèque React, bibliothèque TS, monorepo)
(3) Configuration de TypeScript par type de projet (application Web, bibliothèque, espace de travail monorepo)
(4) Configuration du regroupement Rsbuild ou Rslib (dev, build, Storybook)
(5) Configuration de Stylelint, Browserslist ou monorepo (Turborepo) vs stratégies polyrepo
(6) Extension ou personnalisation des configurations partagées, dépannage des contraintes ESM/ESNext
