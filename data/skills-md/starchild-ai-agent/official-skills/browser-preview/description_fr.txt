Connaissance du panneau d'aperçu du navigateur : l'onglet Navigateur basé sur iframe dans le panneau de droite. À utiliser lorsque l'utilisateur mentionne le navigateur, l'aperçu, l'onglet disparu, la page cassée, l'écran vide, l'écran blanc, le navigateur, l'onglet 不见了, l'aperçu ne se charge pas ou pose des questions sur les services en cours d'exécution.
