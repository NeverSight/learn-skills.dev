Frontend- und Fullstack-Entwicklung mit Live-Vorschau. Verwenden Sie diese Option, wenn der Benutzer eine Webseite, eine Frontend-App, ein Fullstack-Projekt oder eine beliebige Web-Benutzeroberfläche erstellen möchte – einschließlich React, Vue, Vite, statisches HTML, Express, FastAPI oder ein beliebiges Framework, das ein im Browser sichtbares Ergebnis liefert. Verwenden Sie dies auch, wenn der Benutzer eine Vorschau im öffentlichen Internet bereitstellen, veröffentlichen oder freigeben möchte (Community-Veröffentlichung).
