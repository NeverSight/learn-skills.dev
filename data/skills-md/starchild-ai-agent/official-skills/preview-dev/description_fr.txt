Développement frontend et fullstack avec aperçu en direct. À utiliser lorsque l'utilisateur souhaite créer une page Web, une application frontale, un projet fullstack ou toute autre interface utilisateur Web, notamment React, Vue, Vite, HTML statique, Express, FastAPI ou tout autre framework produisant un résultat visible dans le navigateur. À utiliser également lorsque l'utilisateur souhaite déployer, publier ou partager un aperçu sur l'Internet public (publication communautaire).
