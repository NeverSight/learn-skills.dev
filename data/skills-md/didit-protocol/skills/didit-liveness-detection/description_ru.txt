Определяет живость одного селфи-изображения с помощью автономного API Didit. Используйте при проверке физического присутствия человека, обнаружении спуфинга или презентационных атак, реализации мер по борьбе со спуфингом или выполнении пассивной проверки активности. Возвращает оценку живости, качество лица и показатели яркости. Точность 99,9%.
