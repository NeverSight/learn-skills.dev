Комплексная проверка личности KYC (Знай своего клиента) для привлечения реальных пользователей. Используйте, когда кому-то необходимо выполнить KYC, зарегистрировать пользователей с проверкой личности, подтвердить личность человека с помощью сканирования удостоверения личности и селфи, запустить полный процесс KYC, создать сеанс проверки для пользователя, настроить идентификатор + живучесть + проверку соответствия лица или реализовать регистрацию пользователя с помощью проверки документов и биометрических данных. Создает рабочий процесс KYC, генерирует URL-адрес проверки и получает решение.
