Intégrez l'API Didit Database Validation pour vérifier les données personnelles par rapport aux bases de données gouvernementales. À utiliser lorsque l'utilisateur souhaite valider son identité par rapport aux documents gouvernementaux, vérifier les numéros d'identification nationaux, vérifier les numéros CPF/CURP/DNI/cedula, effectuer des recherches dans la base de données d'identité, valider les documents d'identité par rapport à des sources officielles ou mettre en œuvre la vérification de la base de données pour les documents d'identité latino-américains ou espagnols à l'aide de Didit. Prend en charge 18 pays avec les méthodes de correspondance 1x1 et 2x2.
