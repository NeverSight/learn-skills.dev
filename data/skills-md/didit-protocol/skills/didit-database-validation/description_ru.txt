Интегрируйте API проверки базы данных Didit для проверки личных данных по государственным базам данных. Используйте, когда пользователь хочет проверить личность по государственным записям, проверить национальные идентификационные номера, проверить номера CPF/CURP/DNI/cedula, выполнить поиск в базе данных идентификационных данных, проверить документы, удостоверяющие личность, на соответствие официальным источникам или реализовать проверку базы данных для документов, удостоверяющих личность в Латинской Америке или Испании, с помощью Didit. Поддерживает 18 стран с методами сопоставления 1x1 и 2x2.
