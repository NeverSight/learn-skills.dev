Úselo cuando el usuario desee validar la identidad con registros gubernamentales, verificar números de identificación nacional, verificar números CPF/CURP/DNI/cédula, realizar búsquedas en bases de datos de identidad, validar documentos de identidad con fuentes oficiales o implementar verificación de bases de datos para documentos de identidad latinoamericanos o españoles utilizando Didit. Admite 18 países con métodos de coincidencia 1x1 y 2x2.
