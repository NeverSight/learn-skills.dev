Integrieren Sie die eigenständige API zur E-Mail-Verifizierung von Didit, um E-Mail-Adressen über OTP zu überprüfen. Verwenden Sie diese Option, wenn der Benutzer E-Mails verifizieren, E-Mail-OTP-Codes senden, E-Mail-Bestätigungscodes überprüfen, gehackte oder entfernbare E-Mails erkennen, prüfen möchte, ob eine E-Mail unzustellbar ist, oder eine E-Mail-basierte Identitätsüberprüfung mit Didit implementieren möchte. Unterstützt Betrugssignale (IP, Gerät, Benutzeragent), konfigurierbare Codelänge, alphanumerische Codes und richtlinienbasierte automatische Ablehnung für riskante E-Mails.
