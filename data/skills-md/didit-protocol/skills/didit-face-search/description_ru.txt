Интегрируйте автономный API Didit Face Search для выполнения поиска по лицу 1:N по всем ранее проверенным сеансам. Используйте, когда пользователь хочет обнаружить повторяющиеся учетные записи, выполнить поиск совпадающих лиц, проверить, существует ли лицо уже в системе, предотвратить повторную регистрацию, выполнить поиск в черном списке или реализовать дедупликацию лиц с помощью Didit. Возвращает ранжированные совпадения с процентом сходства.
