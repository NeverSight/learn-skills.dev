Durchgängige KYC-Identitätsüberprüfung (Know Your Customer) für das Onboarding echter Benutzer. Verwenden Sie diese Funktion, wenn jemand KYC durchführen, Benutzer mit Identitätsüberprüfung einbinden, die Identität einer Person mit ID-Scan und Selfie überprüfen, einen vollständigen KYC-Ablauf ausführen, eine Überprüfungssitzung für einen Benutzer erstellen, ID + Liveness + Gesichtsübereinstimmungsüberprüfung einrichten oder Benutzer-Onboarding mit Dokumenten- und biometrischen Prüfungen implementieren muss. Erstellt einen KYC-Workflow, generiert eine Verifizierungs-URL und ruft die Entscheidung ab.
