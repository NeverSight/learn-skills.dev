Интегрируйте автономный API Didit AML Screening для проверки отдельных лиц или компаний на соответствие глобальным спискам наблюдения. Используйте, когда пользователь хочет выполнить проверки AML, проверку на соответствие санкционным спискам, проверить статус политически значимого лица, обнаружить вредоносные СМИ, обеспечить соблюдение требований KYC/AML, проверить на соответствие спискам наблюдения OFAC/UN/EU, рассчитать оценки риска или выполнить проверку на предмет отмывания денег с помощью Didit. Поддерживает более 1300 баз данных, нечеткое сопоставление имен, настраиваемые веса оценок и непрерывный мониторинг.
