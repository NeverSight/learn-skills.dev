Интегрируйте автономный API Didit Email Verification для проверки адресов электронной почты через OTP. Используйте, когда пользователь хочет проверить электронную почту, отправить OTP-коды электронной почты, проверить коды проверки электронной почты, обнаружить взломанные или одноразовые электронные письма, проверить, невозможно ли доставить электронное письмо, или реализовать проверку личности по электронной почте с помощью Didit. Поддерживает сигналы мошенничества (IP-адрес, устройство, пользовательский агент), настраиваемую длину кода, буквенно-цифровые коды и автоматическое отклонение на основе политик для опасных писем.
