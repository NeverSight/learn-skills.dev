Интегрируйте автономный API Didit Phone Verification для проверки номеров телефонов через OTP. Используйте, когда пользователь хочет подтвердить телефоны, отправить SMS или коды WhatsApp или Telegram, проверить коды проверки телефона, обнаружить одноразовые номера или номера VoIP или реализовать проверку личности по телефону с помощью Didit. Поддерживает несколько каналов доставки (SMS, WhatsApp, Telegram, голосовая связь), сигналы мошенничества и автоматическое отклонение на основе политик.
