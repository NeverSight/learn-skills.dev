Schätzt das Alter einer Person anhand eines Gesichtsbilds über die eigenständige Didit-API. Verwenden Sie diese Funktion, wenn Sie eine Altersbegrenzung implementieren, überprüfen, ob jemand über 18 oder 21 Jahre alt ist, eine Altersüberprüfung zur Einhaltung der Vorschriften durchführen oder minderjährige Benutzer erkennen. Inklusive passiver Lebendigkeitsprüfung. Unterstützt konfigurierbare Schwellenwerte, adaptives Fallback zur ID-Überprüfung und länderspezifische Einschränkungen.
