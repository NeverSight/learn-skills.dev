Оценивает возраст человека по изображению лица с помощью автономного API Didit. Используйте при внедрении ограничения по возрасту, проверке того, есть ли кто-то старше 18 или 21 года, проверке возраста на соответствие требованиям или обнаружении несовершеннолетних пользователей. Включает пассивную проверку работоспособности. Поддерживает настраиваемые пороговые значения, адаптивный переход к проверке удостоверения личности и ограничения для каждой страны.
