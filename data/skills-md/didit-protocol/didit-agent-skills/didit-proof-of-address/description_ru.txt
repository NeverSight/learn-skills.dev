Интегрируйте автономный API Didit Proof of Address для проверки адресных документов. Используйте, когда пользователь хочет подтвердить подтверждение адреса, проверить счета за коммунальные услуги, банковские выписки, правительственные документы, извлечь адрес из документов, проверить адрес проживания, реализовать проверку адреса или выполнить проверку PoA с помощью Didit. Поддерживает извлечение OCR, геокодирование, сопоставление имен и многостраничные документы.
