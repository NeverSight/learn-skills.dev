Verifica i documenti di identità tramite l'API autonoma Didit. Da utilizzare quando si verifica un passaporto, una carta d'identità, una patente di guida o un permesso di soggiorno, si esegue l'estrazione OCR, l'analisi MRZ, i controlli di autenticità del documento o la convalida del documento KYC. Supporta oltre 4000 tipi di documenti in oltre 220 paesi.
