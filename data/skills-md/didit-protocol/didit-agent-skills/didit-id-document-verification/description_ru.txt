Проверяет документы, удостоверяющие личность, через автономный API Didit. Используйте при проверке паспорта, удостоверения личности, водительских прав или вида на жительство, извлечения OCR, анализа MRZ, проверки подлинности документов или проверки документов KYC. Поддерживает более 4000 типов документов в более чем 220 странах.
