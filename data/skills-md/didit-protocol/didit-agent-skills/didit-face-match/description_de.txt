Integrieren Sie die eigenständige Didit Face Match-API, um zwei Gesichtsbilder zu vergleichen. Verwenden Sie diese Option, wenn der Benutzer mit Didit Gesichter vergleichen, die Gesichtsidentität überprüfen, einen biometrischen Vergleich, eine Gesichtserkennung oder einen Selfie-Dokument-Abgleich durchführen möchte. Gibt einen Ähnlichkeitswert (0–100) mit konfigurierbarem Rückgangsschwellenwert zurück. Unterstützt Bilddrehung und Erkennung mehrerer Gesichter.
