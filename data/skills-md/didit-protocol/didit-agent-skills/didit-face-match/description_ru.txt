Интегрируйте автономный API Didit Face Match для сравнения двух изображений лица. Используйте, когда пользователь хочет сравнить лица, подтвердить личность лица, реализовать биометрическое сравнение, распознавание лиц или сопоставление селфи с документом с помощью Didit. Возвращает оценку сходства (0–100) с настраиваемым порогом отклонения. Поддерживает поворот изображения и обнаружение нескольких лиц.
