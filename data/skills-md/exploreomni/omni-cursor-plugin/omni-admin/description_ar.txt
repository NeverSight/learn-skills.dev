إدارة مثيل Omni Analytics - إدارة الاتصالات والمستخدمين والمجموعات وسمات المستخدم والأذونات والجداول الزمنية وتحديثات المخطط عبر REST API. استخدم هذه المهارة عندما يريد شخص ما إدارة المستخدمين أو المجموعات، أو إعداد الأذونات على لوحة معلومات أو مجلد، أو تكوين سمات المستخدم، أو إنشاء الجداول الزمنية أو تعديلها، أو إدارة اتصالات قاعدة البيانات، أو تحديث المخطط، أو إعداد عناصر التحكم في الوصول، أو توفير المستخدمين، أو أي متغير من "إضافة مستخدم"، أو "منح الوصول إلى"، أو "إعداد الأذونات"، أو "من لديه حق الوصول"، أو "تكوين الاتصال"، أو "تحديث المخطط"، أو "جدولة التسليم".
