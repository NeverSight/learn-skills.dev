Administrer une instance Omni Analytics : gérez les connexions, les utilisateurs, les groupes, les attributs utilisateur, les autorisations, les planifications et les actualisations de schéma via l'API REST. Utilisez cette compétence chaque fois que quelqu'un souhaite gérer des utilisateurs ou des groupes, configurer des autorisations sur un tableau de bord ou un dossier, configurer des attributs utilisateur, créer ou modifier des planifications, gérer des connexions à une base de données, actualiser un schéma, configurer des contrôles d'accès, provisionner des utilisateurs ou toute variante de « ajouter un utilisateur », « donner accès à », « configurer des autorisations », « qui a accès », « configurer la connexion », « actualiser le schéma » ou « planifier une livraison ».
