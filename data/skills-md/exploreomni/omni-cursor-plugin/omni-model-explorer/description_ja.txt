Omni REST API を使用して、Omni Analytics のモデル、トピック、ビュー、フィールド、ディメンション、メジャー、および関係を検出および検査します。 Omni で利用可能なデータを理解したい場合、セマンティック モデルを探索したい場合、特定のフィールドやビューを検索したい場合、テーブルが結合する方法を確認したい場合、どのようなトピックが存在するかを確認したい場合、または「何をクエリできるか」、「どのフィールドが利用可能か」、「モデルを見せて」、「どのようなデータがあるか」、「このデータはどのようにモデル化されているか」などの質問をしたい場合には、このスキルを使用してください。何かを構築または変更する前に、Omni モデルの構造を理解する必要がある場合にも使用します。
