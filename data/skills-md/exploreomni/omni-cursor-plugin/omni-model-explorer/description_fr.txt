Découvrez et inspectez les modèles, sujets, vues, champs, dimensions, mesures et relations Omni Analytics à l'aide de l'API Omni REST. Utilisez cette compétence chaque fois que quelqu'un souhaite comprendre quelles données sont disponibles dans Omni, explorer son modèle sémantique, rechercher des champs ou des vues spécifiques, vérifier comment les tables se réunissent, voir quels sujets existent ou demande n'importe quelle variante de « que puis-je interroger », « quels champs sont disponibles », « montre-moi le modèle », « de quelles données disposons-nous » ou « comment ces données sont-elles modélisées ». À utiliser également lorsque vous avez besoin de comprendre la structure du modèle Omni avant de créer ou de modifier quoi que ce soit.
