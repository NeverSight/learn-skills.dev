Entdecken und prüfen Sie Omni Analytics-Modelle, -Themen, -Ansichten, -Felder, -Dimensionen, -Kennzahlen und -Beziehungen mithilfe der Omni-REST-API. Verwenden Sie diese Fähigkeit immer dann, wenn jemand verstehen möchte, welche Daten in Omni verfügbar sind, sein semantisches Modell erkunden, bestimmte Felder oder Ansichten finden, überprüfen möchte, wie Tabellen zusammengefügt werden, sehen, welche Themen vorhanden sind, oder eine beliebige Variante von „Was kann ich abfragen“, „Welche Felder sind verfügbar“, „Zeige mir das Modell“, „Welche Daten haben wir“ oder „Wie werden diese Daten modelliert“ fragt? Verwenden Sie es auch, wenn Sie die Struktur des Omni-Modells verstehen müssen, bevor Sie etwas erstellen oder ändern.
