Descubra e inspeccione modelos, temas, vistas, campos, dimensiones, medidas y relaciones de Omni Analytics utilizando la API REST de Omni. Utilice esta habilidad siempre que alguien quiera comprender qué datos están disponibles en Omni, explorar su modelo semántico, encontrar campos o vistas específicos, verificar cómo se unen las tablas, ver qué temas existen o preguntar cualquier variante de "qué puedo consultar", "qué campos están disponibles", "muéstreme el modelo", "qué datos tenemos" o "cómo se modelan estos datos". Úselo también cuando necesite comprender la estructura del modelo Omni antes de construir o modificar algo.
