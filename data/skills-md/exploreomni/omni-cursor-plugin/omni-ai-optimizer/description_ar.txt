قم بتحسين نموذج Omni Analytics الخاص بك لـ Blobby، مساعد الذكاء الاصطناعي الخاص بـ Omni - قم بتكوين ai_context، وai_fields، وsample_queries، وإنشاء ملحقات موضوعات خاصة بالذكاء الاصطناعي. استخدم هذه المهارة عندما يريد شخص ما تحسين دقة الذكاء الاصطناعي في Omni، أو جعل Blobby أكثر ذكاءً، أو تكوين سياق الذكاء الاصطناعي، أو إضافة أمثلة للأسئلة، أو ضبط استجابات الذكاء الاصطناعي، أو إعداد نماذج استعلامات، أو تنظيم الحقول للذكاء الاصطناعي، أو إنشاء موضوعات محسّنة للذكاء الاصطناعي، أو استكشاف الأخطاء وإصلاحها بسبب إعطاء Blobby إجابات خاطئة، أو أي شكل من أشكال "جعل الذكاء الاصطناعي أفضل"، أو "Blobby لا يجيب بشكل صحيح"، أو "إضافة سياق للذكاء الاصطناعي"، أو "تحسين الذكاء الاصطناعي"، أو "تعليم الذكاء الاصطناعي حول بياناتنا".
