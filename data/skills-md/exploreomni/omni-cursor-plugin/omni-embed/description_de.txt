Betten Sie Omni Analytics-Dashboards in externe Anwendungen ein – URL-Signierung, benutzerdefinierte Designs, Iframe-Ereignisse, Entitätsarbeitsbereiche und berechtigungsbewusste Inhalte – mithilfe des @omni-co/embed SDK und der REST-API. Verwenden Sie diese Fertigkeit immer dann, wenn jemand ein Dashboard einbetten, eine Einbettungs-URL signieren, das eingebettete Design anpassen, Einbettungsereignisse verarbeiten, auf Klicks oder Übungen im Iframe warten, Filter an ein eingebettetes Dashboard senden, Entitätsarbeitsbereiche einrichten, eingebettete Benutzer suchen, eine berechtigungsbewusste Inhaltsliste erstellen, ein eingebettetes Dashboard mit einem White-Label versehen oder eine beliebige Variante von „Dieses Dashboard einbetten“, „das Iframe-Design anpassen“, „Klickereignisse aus der Einbettung verarbeiten“, „das eingebettete Dashboard filtern“ oder „Einbettung einrichten“ möchte. oder „Welche Dashboards kann dieser Benutzer sehen“.
