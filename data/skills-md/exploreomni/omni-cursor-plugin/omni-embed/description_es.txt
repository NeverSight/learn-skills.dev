Integre paneles de Omni Analytics en aplicaciones externas (firma de URL, temas personalizados, eventos de iframe, espacios de trabajo de entidades y contenido con reconocimiento de permisos) utilizando el SDK @omni-co/embed y la API REST. Utilice esta habilidad siempre que alguien quiera incrustar un panel, firmar una URL de inserción, personalizar el tema incrustado, manejar eventos de incrustación, escuchar clics o ejercicios en el iframe, enviar filtros a un panel incrustado, configurar espacios de trabajo de entidades, buscar usuarios incrustados, crear una lista de contenido con permisos, etiquetar un panel incrustado o cualquier variante de "incrustar este panel", "personalizar el tema del iframe", "manejar eventos de clic desde la inserción", "filtrar el panel incrustado", "configurar incrustación" o "qué paneles puede ver este usuario".
