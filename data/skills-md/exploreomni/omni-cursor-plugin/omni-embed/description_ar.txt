قم بتضمين لوحات معلومات Omni Analytics في التطبيقات الخارجية - توقيع URL، والموضوعات المخصصة، وأحداث iframe، ومساحات عمل الكيان، والمحتوى المدرك للأذونات - باستخدام @omni-co/embed SDK وREST API. استخدم هذه المهارة عندما يريد شخص ما تضمين لوحة معلومات، أو التوقيع على عنوان URL للتضمين، أو تخصيص السمة المضمنة، أو التعامل مع أحداث التضمين، أو الاستماع للنقرات أو التدريبات في iframe، أو إرسال عوامل التصفية إلى لوحة معلومات مضمنة، أو إعداد مساحات عمل الكيان، أو البحث عن المستخدمين المضمنين، أو إنشاء قائمة محتوى مدركة للأذونات، أو وضع علامة بيضاء على لوحة معلومات مضمنة، أو أي نوع مختلف من "تضمين لوحة المعلومات هذه"، أو "تخصيص سمة iframe"، أو "التعامل مع أحداث النقر من التضمين"، أو "تصفية المضمنة" لوحة المعلومات"، أو "إعداد التضمين"، أو "ما هي لوحات المعلومات التي يمكن لهذا المستخدم رؤيتها".
