@omni-co/embed SDK と REST API を使用して、Omni Analytics ダッシュボードを外部アプリケーション (URL 署名、カスタム テーマ、iframe イベント、エンティティ ワークスペース、権限対応コンテンツ) に埋め込みます。ダッシュボードの埋め込み、埋め込み URL の署名、埋め込みテーマのカスタマイズ、埋め込みイベントの処理、iframe でのクリックまたはドリルのリッスン、埋め込みダッシュボードへのフィルターの送信、エンティティ ワークスペースの設定、埋め込みユーザーの検索、権限を認識したコンテンツ リストの構築、埋め込みダッシュボードのホワイト ラベル、または「このダッシュボードの埋め込み」、「iframe テーマのカスタマイズ」、「埋め込み」、「埋め込みダッシュボードのフィルター」、「埋め込みの設定」、または「このユーザーに表示できるダッシュボード」。
