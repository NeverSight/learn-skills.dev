Créez et modifiez les définitions de modèles sémantiques Omni Analytics (vues, sujets, dimensions, mesures, relations et vues de requête) à l'aide de YAML via l'API REST Omni. Utilisez cette compétence chaque fois que quelqu'un souhaite ajouter un champ, créer une nouvelle dimension ou mesure, définir un sujet, configurer des jointures entre des tables, modifier le modèle de données, créer une nouvelle vue, ajouter un champ calculé, créer une relation, modifier YAML, travailler sur une branche, promouvoir des modifications de modèle ou toute variante de « modéliser ces données », « ajouter cette métrique », « créer une vue pour » ou « configurer une jointure entre ». Également utilisé pour migrer des modèles de modélisation, car le YAML d'Omni est conceptuellement similaire aux autres définitions de couche sémantique.
