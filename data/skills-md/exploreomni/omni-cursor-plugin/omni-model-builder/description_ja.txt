Omni REST API を通じて YAML を使用して、Omni Analytics セマンティック モデル定義 (ビュー、トピック、ディメンション、メジャー、リレーションシップ、クエリ ビュー) を作成および編集します。フィールドの追加、新しいディメンションまたはメジャーの作成、トピックの定義、テーブル間の結合の設定、データ モデルの変更、新しいビューの構築、計算フィールドの追加、リレーションシップの作成、YAML の編集、ブランチでの作業、モデルの変更の促進、または「このデータのモデル化」、「このメトリクスの追加」、「ビューの作成」、または「間の結合の設定」の変形を行う場合には、このスキルを使用します。 Omni の YAML は概念的に他のセマンティック レイヤー定義と似ているため、モデリング パターンの移行にも使用します。
