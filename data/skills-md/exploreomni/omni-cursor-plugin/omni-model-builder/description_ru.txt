Создавайте и редактируйте определения семантической модели Omni Analytics — представления, темы, измерения, меры, связи и представления запросов — с помощью YAML через Omni REST API. Используйте этот навык всякий раз, когда кто-то хочет добавить поле, создать новое измерение или меру, определить тему, настроить соединения между таблицами, изменить модель данных, построить новое представление, добавить вычисляемое поле, создать связь, отредактировать YAML, поработать с ветвью, продвигать изменения модели или любой вариант «моделировать эти данные», «добавить эту метрику», «создать представление для» или «настроить соединение между». Также используйте для миграции шаблонов моделирования, поскольку YAML Omni концептуально аналогичен определениям других семантических слоев.
