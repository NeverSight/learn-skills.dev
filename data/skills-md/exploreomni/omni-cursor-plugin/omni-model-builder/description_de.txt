Erstellen und bearbeiten Sie semantische Modelldefinitionen von Omni Analytics – Ansichten, Themen, Dimensionen, Kennzahlen, Beziehungen und Abfrageansichten – mithilfe von YAML über die Omni REST API. Verwenden Sie diese Fähigkeit immer dann, wenn jemand ein Feld hinzufügen, eine neue Dimension oder Kennzahl erstellen, ein Thema definieren, Verknüpfungen zwischen Tabellen einrichten, das Datenmodell ändern, eine neue Ansicht erstellen, ein berechnetes Feld hinzufügen, eine Beziehung erstellen, YAML bearbeiten, an einem Zweig arbeiten, Modelländerungen fördern oder eine beliebige Variante von „diese Daten modellieren“, „diese Metrik hinzufügen“, „eine Ansicht erstellen für“ oder „eine Verknüpfung zwischen einrichten“ möchte. Wird auch zum Migrieren von Modellierungsmustern verwendet, da Omnis YAML konzeptionell anderen semantischen Ebenendefinitionen ähnelt.
