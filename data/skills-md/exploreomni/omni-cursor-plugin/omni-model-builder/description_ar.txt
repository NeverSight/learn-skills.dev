قم بإنشاء وتحرير تعريفات النماذج الدلالية لـ Omni Analytics - طرق العرض والموضوعات والأبعاد والمقاييس والعلاقات وطرق عرض الاستعلام - باستخدام YAML من خلال Omni REST API. استخدم هذه المهارة عندما يريد شخص ما إضافة حقل، أو إنشاء بُعد أو قياس جديد، أو تعريف موضوع، أو إعداد الصلات بين الجداول، أو تعديل نموذج البيانات، أو إنشاء طريقة عرض جديدة، أو إضافة حقل محسوب، أو إنشاء علاقة، أو تحرير YAML، أو العمل على فرع، أو الترويج لتغييرات النموذج، أو أي متغير من "نموذج هذه البيانات"، أو "إضافة هذا المقياس"، أو "إنشاء طريقة عرض لـ"، أو "إعداد صلة بين". يُستخدم أيضًا لترحيل أنماط النمذجة نظرًا لأن YAML الخاص بـ Omni يشبه من الناحية المفاهيمية تعريفات الطبقة الدلالية الأخرى.
