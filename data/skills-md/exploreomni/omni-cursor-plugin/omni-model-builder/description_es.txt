Cree y edite definiciones de modelos semánticos de Omni Analytics (vistas, temas, dimensiones, medidas, relaciones y vistas de consultas) utilizando YAML a través de la API REST de Omni. Utilice esta habilidad siempre que alguien quiera agregar un campo, crear una nueva dimensión o medida, definir un tema, configurar uniones entre tablas, modificar el modelo de datos, crear una nueva vista, agregar un campo calculado, crear una relación, editar YAML, trabajar en una rama, promover cambios de modelo o cualquier variante de "modelar estos datos", "agregar esta métrica", "crear una vista para" o "configurar una unión entre". Úselo también para migrar patrones de modelado, ya que YAML de Omni es conceptualmente similar a otras definiciones de capas semánticas.
