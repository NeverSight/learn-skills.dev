Omni REST API를 통해 YAML을 사용하여 Omni Analytics 의미 체계 모델 정의(보기, 주제, 차원, 측정값, 관계 및 쿼리 보기)를 생성하고 편집합니다. 필드 추가, 새 차원 또는 측정값 생성, 주제 정의, 테이블 간 조인 설정, 데이터 모델 수정, 새 뷰 구축, 계산된 필드 추가, 관계 생성, YAML 편집, 분기 작업, 모델 변경 승격 또는 "이 데이터 모델링", "이 메트릭 추가", "에 대한 뷰 생성" 또는 "간 조인 설정"의 변형을 원할 때마다 이 기술을 사용하세요. Omni의 YAML은 개념적으로 다른 의미 계층 정의와 유사하므로 모델링 패턴 마이그레이션에도 사용됩니다.
