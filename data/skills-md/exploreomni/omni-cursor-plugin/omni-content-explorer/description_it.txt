Trova, sfoglia e organizza i contenuti in Omni Analytics (pannelli di controllo, cartelle di lavoro, cartelle ed etichette) utilizzando l'API REST. Utilizza questa abilità ogni volta che qualcuno desidera trovare un dashboard esistente, cercare contenuti, elencare cartelle di lavoro, sfogliare cartelle, vedere quali dashboard esistono, trovare report popolari, scaricare un dashboard come PDF o PNG, contenuti preferiti, gestire le etichette sui documenti o qualsiasi variante di "trova il dashboard su", "quali report abbiamo", "mostrami i nostri dashboard", "dov'è il report sulle vendite" o "scarica questo dashboard".
