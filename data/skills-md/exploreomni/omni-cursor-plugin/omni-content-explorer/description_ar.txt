يمكنك البحث عن المحتوى وتصفحه وتنظيمه في Omni Analytics - لوحات المعلومات والمصنفات والمجلدات والتسميات - باستخدام REST API. استخدم هذه المهارة عندما يريد شخص ما العثور على لوحة معلومات موجودة، أو البحث عن المحتوى، أو إدراج المصنفات، أو تصفح المجلدات، أو الاطلاع على لوحات المعلومات الموجودة، أو العثور على التقارير الشائعة، أو تنزيل لوحة معلومات بصيغة PDF أو PNG، أو المحتوى المفضل، أو إدارة التسميات على المستندات، أو أي شكل من أشكال "البحث عن لوحة المعلومات حول"، أو "ما هي التقارير المتوفرة لدينا"، أو "إظهار لوحات المعلومات الخاصة بنا"، أو "أين يوجد تقرير المبيعات"، أو "تنزيل لوحة المعلومات هذه".
