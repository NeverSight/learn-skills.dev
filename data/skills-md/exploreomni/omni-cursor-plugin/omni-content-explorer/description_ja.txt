REST API を使用して、Omni Analytics のコンテンツ (ダッシュボード、ワークブック、フォルダー、ラベル) を検索、参照、整理します。既存のダッシュボードの検索、コンテンツの検索、ワークブックの一覧表示、フォルダーの参照、存在するダッシュボードの確認、人気のあるレポートの検索、PDF または PNG としてのダッシュボードのダウンロード、お気に入りのコンテンツ、ドキュメントのラベルの管理、または「ダッシュボードの検索」、「どのようなレポートがあるか」、「ダッシュボードを見せて」、「売上レポートはどこにある」、または「このダッシュボードをダウンロード」の変形を行う場合には、このスキルを使用します。
