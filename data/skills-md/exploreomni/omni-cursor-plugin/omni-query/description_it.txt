Esegui query sul livello semantico di Omni Analytics utilizzando l'API REST, interpreta i risultati e concatena le query per l'analisi in più passaggi. Utilizza questa abilità ogni volta che qualcuno desidera eseguire query sui dati tramite Omni, eseguire un report, ottenere metriche, estrarre numeri, analizzare dati, chiedere "quanti", "qual è la tendenza", "mostrami i dati", recuperare i risultati delle query del dashboard o eseguire qualsiasi recupero di dati tramite il motore di query di Omni. Utilizzalo anche quando qualcuno desidera estrarre dati a livello di codice da una dashboard o una cartella di lavoro Omni esistente.
