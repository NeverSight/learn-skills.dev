Exécutez des requêtes sur la couche sémantique d'Omni Analytics à l'aide de l'API REST, interprétez les résultats et chaînez les requêtes pour une analyse en plusieurs étapes. Utilisez cette compétence chaque fois que quelqu'un souhaite interroger des données via Omni, exécuter un rapport, obtenir des mesures, extraire des chiffres, analyser des données, demander « combien », « quelle est la tendance », « me montrer les données », récupérer les résultats d'une requête de tableau de bord ou effectuer toute récupération de données via le moteur de requête d'Omni. À utiliser également lorsque quelqu'un souhaite extraire par programme des données d'un tableau de bord ou d'un classeur Omni existant.
