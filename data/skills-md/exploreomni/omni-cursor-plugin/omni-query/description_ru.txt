Выполняйте запросы к семантическому слою Omni Analytics с помощью REST API, интерпретируйте результаты и объединяйте запросы в цепочку для многоэтапного анализа. Используйте этот навык всякий раз, когда кто-то хочет запросить данные через Omni, запустить отчет, получить показатели, получить цифры, проанализировать данные, спросить «сколько», «какая тенденция», «показать мне данные», получить результаты запроса на панели мониторинга или выполнить любой поиск данных через механизм запросов Omni. Также используйте, когда кто-то хочет программно извлечь данные из существующей информационной панели или книги Omni.
