REST API を使用して Omni Analytics のセマンティック レイヤーに対してクエリを実行し、結果を解釈して、複数ステップの分析のためにクエリをチェーンします。 Omni を介してデータをクエリしたり、レポートを実行したり、メトリクスを取得したり、数値を取得したり、データを分析したり、「数」、「傾向は何ですか」、「データを見せて」と質問したり、ダッシュボードのクエリ結果を取得したり、Omni のクエリ エンジンを介してデータ取得を実行したりする場合は、このスキルをいつでも使用できます。既存の Omni ダッシュボードまたはワークブックからプログラムでデータを抽出したい場合にも使用します。
