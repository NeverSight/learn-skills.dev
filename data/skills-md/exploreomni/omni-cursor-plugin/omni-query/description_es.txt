Ejecute consultas en la capa semántica de Omni Analytics utilizando la API REST, interprete resultados y encadene consultas para análisis de varios pasos. Utilice esta habilidad siempre que alguien quiera consultar datos a través de Omni, ejecutar un informe, obtener métricas, extraer números, analizar datos, preguntar "cuántos", "cuál es la tendencia", "muéstreme los datos", recuperar resultados de consultas del panel o realizar cualquier recuperación de datos a través del motor de consultas de Omni. Úselo también cuando alguien quiera extraer datos mediante programación de un panel o libro de trabajo de Omni existente.
