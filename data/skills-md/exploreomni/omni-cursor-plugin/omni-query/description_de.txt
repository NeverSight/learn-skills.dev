Führen Sie mithilfe der REST-API Abfragen für die semantische Ebene von Omni Analytics aus, interpretieren Sie Ergebnisse und verketten Sie Abfragen für eine mehrstufige Analyse. Verwenden Sie diese Fähigkeit immer dann, wenn jemand Daten über Omni abfragen, einen Bericht erstellen, Metriken abrufen, Zahlen abrufen, Daten analysieren, fragen möchte, „wie viele“, „was ist der Trend“, „mir die Daten zeigen“, Dashboard-Abfrageergebnisse abrufen oder einen Datenabruf über die Abfrage-Engine von Omni durchführen möchte. Verwenden Sie es auch, wenn jemand Daten programmgesteuert aus einem vorhandenen Omni-Dashboard oder einer Arbeitsmappe extrahieren möchte.
