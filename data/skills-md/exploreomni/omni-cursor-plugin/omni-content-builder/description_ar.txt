قم بإنشاء وتحديث وإدارة مستندات Omni Analytics ولوحات المعلومات برمجيًا - دورة حياة المستند، والتجانبات، والمرئيات، والمرشحات، والتخطيطات - باستخدام REST API. استخدم هذه المهارة عندما يريد شخص ما إنشاء لوحة معلومات، أو إنشاء مصنف، أو إضافة تجانبات أو مخططات، أو تكوين عوامل تصفية لوحة المعلومات، أو تحديث نموذج لوحة معلومات موجودة، أو إعداد عرض مؤشرات الأداء الرئيسية، أو إنشاء تصورات، أو تخطيط لوحة معلومات، أو إنشاء مستند، أو إعادة تسمية مصنف، أو حذف لوحة معلومات، أو نقل مستند إلى مجلد، أو تكرار لوحة معلومات، أو أي نوع مختلف من "إنشاء لوحة معلومات لـ"، "إنشاء تقرير يظهر"، "إضافة مخطط إلى"، "إنشاء لوحة معلومات"، "تحديث تخطيط لوحة المعلومات"، "إعادة تسمية هذا المستند" أو "الانتقال إلى المجلد" أو "حذف لوحة المعلومات هذه". استخدمه أيضًا عند تعديل تخصيصات النموذج على مستوى لوحة المعلومات مثل الصلات أو الحقول الخاصة بالمصنف.
