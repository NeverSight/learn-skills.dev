REST API を使用して、Omni Analytics ドキュメントとダッシュボード (ドキュメント ライフサイクル、タイル、ビジュアライゼーション、フィルター、レイアウト) をプログラムで作成、更新、管理します。ダッシュボードの構築、ワークブックの作成、タイルまたはグラフの追加、ダッシュボード フィルターの構成、既存のダッシュボード モデルの更新、KPI ビューのセットアップ、ビジュアライゼーションの作成、ダッシュボードのレイアウト、ドキュメントの作成、ワークブックの名前変更、ダッシュボードの削除、フォルダーへのドキュメントの移動、ダッシュボードの複製、または「ダッシュボードの構築」、「表示するレポートの作成」、「グラフの追加」、「 「ダッシュボード」、「ダッシュボードのレイアウトを更新」、「このドキュメントの名前を変更」、「フォルダーに移動」、または「このダッシュボードを削除」。ワークブック固有の結合やフィールドなどのダッシュボード レベルのモデルのカスタマイズを変更する場合にも使用します。
