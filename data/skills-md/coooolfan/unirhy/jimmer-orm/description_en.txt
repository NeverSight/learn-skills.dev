Jimmer ORM development assistant, helping to write entity mapping code and DSL query statements. Applicable to: (1) defining or modifying Jimmer entities (interfaces annotated with @Entity); (2) writing Jimmer DSL query code; (3) configuring entity associations (@ManyToOne, @OneToMany, @ManyToMany); (4) using dynamic predicates, dynamic table joins, or implicit subqueries; (5) editing Java/Kotlin files containing Jimmer-related code.
