Diese Fähigkeit sollte verwendet werden, wenn der Benutzer möchte, dass Claude Code lediglich als Manager/Architekt fungiert, während Gemini CLI die gesamte Codierungsarbeit erledigt. Claude Code leitet Gemini wie ein Praktikant – er gibt Aufgaben aus, überprüft die Ausgabe, fordert Korrekturen an – schreibt aber nie selbst Code. Verwenden Sie diese Option, wenn der Benutzer „Gemini verwalten“, „Architektenmodus“ oder „Gemini steuern“ sagt oder die gesamte Implementierung an Gemini delegieren möchte.
