يجب استخدام هذه المهارة عندما يريد المستخدم أن يعمل Claude Code كمدير/مهندس معماري فقط بينما يقوم Gemini CLI بجميع أعمال البرمجة. يقود كلود كود برج الجوزاء مثل المتدرب - حيث يصدر المهام، ويراجع المخرجات، ويطلب الإصلاحات - لكنه لا يكتب الكود نفسه أبدًا. يُستخدم عندما يقول المستخدم "إدارة الجوزاء"، أو "الوضع المعماري"، أو "قيادة الجوزاء"، أو يريد تفويض كل عمليات التنفيذ إلى الجوزاء.
