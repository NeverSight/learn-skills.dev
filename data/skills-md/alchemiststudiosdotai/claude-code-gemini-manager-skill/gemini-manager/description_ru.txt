Этот навык следует использовать, когда пользователь хочет, чтобы Claude Code действовал исключительно как менеджер/архитектор, в то время как Gemini CLI выполнял всю работу по кодированию. Клод Код управляет Gemini как стажером — выдает задания, проверяет выходные данные, запрашивает исправления — но никогда не пишет код сам. Используйте, когда пользователь говорит «управление Gemini», «режим архитектора», «управление Gemini» или хочет делегировать всю реализацию Gemini.
