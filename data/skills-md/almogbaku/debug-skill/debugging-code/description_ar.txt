تصحيح أخطاء التعليمات البرمجية المصدر بشكل تفاعلي — تعيين نقاط التوقف، والخطوة خلال التنفيذ سطرًا تلو الآخر، وفحص حالة المتغير المباشر، وتقييم التعبيرات مقابل البرنامج قيد التشغيل، والتنقل في مكدس الاستدعاءات لتتبع الأسباب الجذرية. يُستخدم عندما يتعطل برنامج ما، أو يثير استثناءات غير متوقعة، أو ينتج مخرجات خاطئة، أو عندما تحتاج إلى فهم كيفية وصول التنفيذ إلى حالة معينة، أو عندما لا يكشف تصحيح أخطاء بيان الطباعة بشكل كافٍ.
