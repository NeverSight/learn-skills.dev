Выполняйте интерактивную отладку исходного кода — устанавливайте точки останова, выполняйте пошаговое выполнение, проверяйте состояние действующих переменных, сравнивайте выражения с работающей программой и перемещайтесь по стеку вызовов для отслеживания основных причин. Используйте, когда программа выходит из строя, вызывает неожиданные исключения, выдает неправильный вывод, когда вам нужно понять, как выполнение достигло определенного состояния, или когда отладка операторов печати недостаточно информативна.
