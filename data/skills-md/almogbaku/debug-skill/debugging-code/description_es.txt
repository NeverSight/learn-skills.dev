Depure el código fuente de forma interactiva: establezca puntos de interrupción, avance la ejecución línea por línea, inspeccione el estado de las variables en vivo, evalúe expresiones con respecto al programa en ejecución y navegue por la pila de llamadas para rastrear las causas fundamentales. Úselo cuando un programa falla, genera excepciones inesperadas, produce resultados incorrectos, cuando necesita comprender cómo la ejecución alcanzó un cierto estado o cuando la depuración de declaraciones impresas no es lo suficientemente reveladora.
