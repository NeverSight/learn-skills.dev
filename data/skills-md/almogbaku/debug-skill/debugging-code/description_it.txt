Esegui il debug in modo interattivo del codice sorgente: imposta punti di interruzione, esegui l'esecuzione riga per riga, ispeziona lo stato delle variabili in tempo reale, valuta le espressioni rispetto al programma in esecuzione e naviga nello stack di chiamate per tracciare le cause principali. Da utilizzare quando un programma si arresta in modo anomalo, solleva eccezioni impreviste, produce output errato, quando è necessario comprendere come l'esecuzione ha raggiunto un determinato stato o quando il debug dell'istruzione di stampa non è sufficientemente rivelatore.
