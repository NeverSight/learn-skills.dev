يجب استخدام هذه المهارة عندما يطلب المستخدم إنشاء "استحقاق الرمز المميز"، أو "تدفقات استحقاق الرمز المميز"، أو "استحقاق onchain"، أو "استحقاق Ethereum"، أو "استحقاق EVM"، أو "استحقاق Solana"، أو "استحقاق ERC-20"، أو "استحقاق ERC20"، أو "استحقاق BEP-20"، أو "استحقاق BEP20" مع Sablier Lockup، ويريد إنشاء جداول استحقاق لرمز مميز أو رموز مميزة على Ethereum. تحتاج السلاسل المتوافقة مع EVM، أو BNB Chain، أو Solana، إلى وكيل لتشغيل معاملات إنشاء الاستحقاق على السلسلة نيابةً عنها.
