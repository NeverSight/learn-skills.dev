Interagissez avec le protocole Jupiter Lend. SDK en lecture seule (@jup-ag/lend-read) pour interroger les pools de liquidités, les marchés de prêt (jlTokens) et les coffres-forts. Écrivez un SDK (@jup-ag/lend) pour les opérations de prêt (dépôt/retrait) et de coffre-fort (dépôt de garantie, emprunt, remboursement, gestion des positions). Aucune clé API nécessaire pour les interactions en chaîne.
