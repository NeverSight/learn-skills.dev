Interagisci con il protocollo Jupiter Lend. SDK di sola lettura (@jup-ag/lend-read) per interrogare pool di liquidità, mercati di prestito (jlToken) e casseforti. Scrivi l'SDK (@jup-ag/lend) per le operazioni di prestito (deposito/prelievo) e di deposito (deposito di garanzie, prestito, rimborso, gestione delle posizioni). Non sono necessarie chiavi API per le interazioni sulla catena.
