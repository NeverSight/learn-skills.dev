Взаимодействуйте с протоколом Юпитера Ленда. SDK только для чтения (@jup-ag/lend-read) для запроса пулов ликвидности, рынков кредитования (jlTokens) и хранилищ. Напишите SDK (@jup-ag/lend) для кредитования (ввод/вывод средств) и операций с хранилищем (внесение залога, заимствование, погашение, управление позициями). Для взаимодействия в сети не требуются ключи API.
