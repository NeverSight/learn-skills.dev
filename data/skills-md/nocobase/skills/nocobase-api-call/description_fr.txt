Exécutez des requêtes API NocoBase authentifiées via un script shell réutilisable. À utiliser lorsque les utilisateurs demandent à appeler directement les points de terminaison NocoBase (GET/POST/PUT/PATCH/DELETE), à inspecter les réponses brutes de l'API ou à exécuter des opérations de collecte/champ avec un point de terminaison et une charge utile spécifiques.
