Комплексная оценка и анализ качества Java-репозиториев. Сочетает в себе существующие инструменты статического анализа (PMD, SpotBugs, Checkstyle, JaCoCo, JDepend) с криминалистической экспертизой истории Git (горячие точки, карты знаний, коэффициент шины, временная связь) для создания подробного отчета о работоспособности. Все инструменты вызываются через полные координаты плагина Maven — pom.xml проекта НИКОГДА не изменяется. Используйте этот навык всякий раз, когда пользователь хочет проанализировать, оценить, проверить или просмотреть Java-проект, базу кода или репозиторий, включая запросы на отчеты о качестве кода, обзоры архитектуры, оценки технического долга, расстановку приоритетов рефакторинга, анализ командных знаний или оценки ремонтопригодности. Также срабатывает, когда пользователь упоминает работоспособность кода, анализ горячих точек, коэффициент шины, владение кодом или анализ зависимостей в контексте Java.
