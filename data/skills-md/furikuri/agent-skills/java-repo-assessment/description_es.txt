Evaluación integral y análisis de calidad de repositorios Java. Combina herramientas de análisis estático establecidas (PMD, SpotBugs, Checkstyle, JaCoCo, JDepend) con análisis forense del historial de Git (puntos de acceso, mapas de conocimiento, factor de bus, acoplamiento temporal) para producir un informe de salud detallado. Todas las herramientas se invocan a través de las coordenadas del complemento totalmente calificado de Maven: el pom.xml del proyecto NUNCA se modifica. Utilice esta habilidad siempre que el usuario desee analizar, evaluar, auditar o revisar un proyecto, base de código o repositorio de Java, incluidas solicitudes de informes de calidad del código, revisiones de arquitectura, evaluaciones de deuda tecnológica, priorización de refactorización, análisis de conocimiento del equipo o evaluaciones de mantenibilidad. También se activa cuando el usuario menciona el estado del código, el análisis de puntos de acceso, el factor de bus, la propiedad del código o el análisis de dependencia en un contexto de Java.
