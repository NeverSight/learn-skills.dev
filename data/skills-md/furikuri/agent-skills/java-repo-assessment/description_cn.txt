Java 存储库的综合评估和质量分析。将已建立的静态分析工具（PMD、SpotBugs、Checkstyle、JaCoCo、JDepend）与 Git 历史取证（热点、知识图、总线因子、时间耦合）相结合，生成详细的运行状况报告。所有工具都是通过 Maven 完全限定的插件坐标调用的——项目的 pom.xml 永远不会被修改。每当用户想要分析、评估、评价、审核或审查 Java 项目、代码库或存储库时，都可以使用此技能，包括请求代码质量报告、架构审查、技术债务评估、重构优先级、团队知识分析或可维护性评估。当用户在 Java 上下文中提及代码运行状况、热点分析、总线因素、代码所有权或依赖性分析时也会触发。
