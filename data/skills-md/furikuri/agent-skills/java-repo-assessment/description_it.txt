Valutazione completa e analisi della qualità dei repository Java. Combina strumenti di analisi statica consolidati (PMD, SpotBugs, Checkstyle, JaCoCo, JDepend) con la cronologia forense di Git (hotspot, mappe della conoscenza, fattore bus, accoppiamento temporale) per produrre un report dettagliato sullo stato. Tutti gli strumenti vengono richiamati tramite le coordinate del plugin completo Maven: il pom.xml del progetto non viene MAI modificato. Utilizza questa competenza ogni volta che l'utente desidera analizzare, valutare, valutare, controllare o rivedere un progetto Java, una base di codice o un repository, comprese richieste di report sulla qualità del codice, revisioni dell'architettura, valutazioni del debito tecnologico, definizione delle priorità di refactoring, analisi delle conoscenze del team o valutazioni di manutenibilità. Si attiva anche quando l'utente menziona l'integrità del codice, l'analisi degli hotspot, il fattore bus, la proprietà del codice o l'analisi delle dipendenze in un contesto Java.
