Referencia completa de la documentación de la API de OpenRouter. Úselo cuando trabaje con OpenRouter, una plataforma de enrutamiento de modelos de IA que proporciona una API unificada para acceder a más de 300 LLM (OpenAI, Anthropic, Google, Meta, Mistral, etc.). Activadores: integración de API de OpenRouter, enrutamiento de modelo, selección de proveedores, respaldos de múltiples modelos, SDK de OpenRouter (TypeScript/Python), finalización de chat de OpenRouter, incrustaciones de OpenRouter, salidas estructuradas de OpenRouter, llamadas de herramientas de OpenRouter, transmisión de OpenRouter, BYOK (traiga su propia clave), OpenRouter OAuth, barreras de seguridad de OpenRouter, almacenamiento en caché de avisos a través de OpenRouter, facturación/créditos de OpenRouter, variantes de modelo (:free, :extended, :nitro, :thinking, :online), integraciones de observabilidad de transmisión, retención de datos cero o cualquier tarea que involucre puntos finales de API openrouter.ai.
