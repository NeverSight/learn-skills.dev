Référence complète de la documentation de l'API OpenRouter. À utiliser lorsque vous travaillez avec OpenRouter — une plateforme de routage de modèles d'IA qui fournit une API unifiée pour accéder à plus de 300 LLM (OpenAI, Anthropic, Google, Meta, Mistral, etc.). Déclencheurs : intégration de l'API OpenRouter, routage du modèle, sélection du fournisseur, solutions de secours multimodèles, SDK OpenRouter (TypeScript/Python), complétion de chat OpenRouter, intégrations OpenRouter, sorties structurées OpenRouter, appel de l'outil OpenRouter, streaming OpenRouter, BYOK (Bring Your Own Key), OpenRouter OAuth, garde-fous OpenRouter, mise en cache des invites via OpenRouter, facturation/crédits OpenRouter, variantes de modèle (: gratuit, :extended, :nitro, :thinking, :online), intégrations d'observabilité de diffusion, zéro rétention de données ou toute tâche impliquant des points de terminaison de l'API openrouter.ai.
