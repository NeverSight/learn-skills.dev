Полная справка по документации OpenRouter API. Используйте при работе с OpenRouter — платформой маршрутизации моделей AI, предоставляющей унифицированный API для доступа к более чем 300 LLM (OpenAI, Anthropic, Google, Meta, Mistral и т. д.). Триггеры: интеграция API OpenRouter, маршрутизация модели, выбор поставщика, резервные варианты нескольких моделей, OpenRouter SDK (TypeScript/Python), завершение чата OpenRouter, встраивание OpenRouter, структурированные выходные данные OpenRouter, вызов инструментов OpenRouter, потоковая передача OpenRouter, BYOK (принесите свой собственный ключ), OpenRouter OAuth, защитные ограждения OpenRouter, кэширование запросов через OpenRouter, выставление счетов/кредитов OpenRouter, варианты модели (:free, :extended, :nitro, :thinking, :online), интеграция наблюдения за трансляцией, нулевое сохранение данных или любые задачи, связанные с конечными точками API openrouter.ai.
