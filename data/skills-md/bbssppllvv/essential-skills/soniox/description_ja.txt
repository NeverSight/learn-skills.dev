リアルタイムおよび非同期文字起こしのために、Soniox speech-to-text API を統合します。 Soniox WebSocket ストリーミング、Soniox REST トランスクリプション、Soniox SDK (Python、Node、Web) を操作する場合、または Soniox を使用して音声/スピーチ機能を実装する場合に使用します。 Soniox API、soniox パッケージ、@soniox/node、@soniox/speech-to-text-web、Soniox を介したリアルタイム文字起こし、音声からテキストへの統合、または Soniox による音声文字起こしへの言及をトリガーします。
