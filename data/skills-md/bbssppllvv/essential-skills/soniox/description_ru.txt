Интегрируйте API преобразования речи в текст Soniox для асинхронной транскрипции в реальном времени. Используйте при работе с потоковой передачей Soniox WebSocket, транскрипцией Soniox REST, SDK Soniox (Python, Node, Web) или реализацией функций голоса/речи с помощью Soniox. Запускается при упоминании Soniox API, пакета soniox, @soniox/node, @soniox/speech-to-text-web, транскрипции в реальном времени через Soniox, интеграции речи в текст или транскрипции аудио с помощью Soniox.
