Руководство по использованию интерфейса командной строки Pinecone (ПК) для управления ресурсами Pinecone с терминала. CLI поддерживает ВСЕ типы индексов (стандартные, интегрированные, разреженные) и все векторные операции — в отличие от MCP, который поддерживает только интегрированные индексы. Используйте для пакетных операций, векторного управления, резервного копирования, пространств имен, автоматизации CI/CD и полного контроля над ресурсами Pinecone.
