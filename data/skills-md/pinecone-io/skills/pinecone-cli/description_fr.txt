Guide d'utilisation de Pinecone CLI (pc) pour gérer les ressources Pinecone depuis le terminal. La CLI prend en charge TOUS les types d'index (standard, intégrés, clairsemés) et toutes les opérations vectorielles, contrairement au MCP qui ne prend en charge que les index intégrés. À utiliser pour les opérations par lots, la gestion des vecteurs, les sauvegardes, les espaces de noms, l'automatisation CI/CD et le contrôle total des ressources Pinecone.
