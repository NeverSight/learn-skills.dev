Referenz für die Pinecone MCP-Server-Tools. Dokumentiert alle verfügbaren Tools – Listenindizes, Beschreibungsindex, Beschreibungsindexstatistiken, Erstellungsindex für Modell, Upsert-Datensätze, Suchdatensätze, kaskadierende Suche und Dokumente zum Neuordnen. Verwenden Sie diese Option, wenn ein Agent verstehen muss, welche Pinecone MCP-Tools verfügbar sind, wie sie verwendet werden oder welche Parameter sie akzeptieren.
