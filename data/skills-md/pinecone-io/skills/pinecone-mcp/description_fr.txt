Référence pour les outils du serveur Pinecone MCP. Documente tous les outils disponibles : index de liste, index de description, statistiques d'index de description, création d'index pour le modèle, insertion d'enregistrements, enregistrements de recherche, recherche en cascade et reclassement des documents. À utiliser lorsqu'un agent a besoin de comprendre quels outils Pinecone MCP sont disponibles, comment les utiliser ou quels paramètres ils acceptent.
