Interaktiver Pinecone-Schnellstart für neue Entwickler. Wählen Sie zwischen zwei Pfaden: Datenbank (einen integrierten Index erstellen, Daten hochladen und mit Pinecone MCP + Python abfragen) oder Assistent (einen Pinecone-Assistenten für Fragen und Antworten zu Dokumenten erstellen). Verwenden Sie diese Option, wenn ein Benutzer zum ersten Mal mit Pinecone beginnen möchte oder eine Führung durch die Tools von Pinecone erhalten möchte.
