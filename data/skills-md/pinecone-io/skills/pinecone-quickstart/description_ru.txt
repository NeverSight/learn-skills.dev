Интерактивное краткое руководство по сосновой шишке для новых разработчиков. Выберите один из двух путей: база данных (создание интегрированного индекса, обновление данных и запрос с помощью Pinecone MCP + Python) или помощник (создание помощника Pinecone для вопросов и ответов по документам). Используйте, когда пользователь хочет впервые начать работу с Pinecone или хочет получить экскурсию по инструментам Pinecone.
