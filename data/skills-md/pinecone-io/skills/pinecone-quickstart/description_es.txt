Inicio rápido interactivo de Pinecone para nuevos desarrolladores. Elija entre dos rutas: Base de datos (cree un índice integrado, inserte datos y realice consultas usando Pinecone MCP + Python) o Asistente (cree un Asistente de Pinecone para preguntas y respuestas sobre documentos). Úselo cuando un usuario quiera comenzar con Pinecone por primera vez o quiera una visita guiada por las herramientas de Pinecone.
