Производительность веб-приложений и оптимизация UX посредством автоматического анализа. Используйте, когда разработчикам необходимо (1) измерить и диагностировать проблемы с производительностью с помощью Lighthouse и Core Web Vitals, (2) определить основные причины проблем LCP, CLS, INP, (3) проанализировать болевые точки UX, такие как изменения макета или медленная интерактивность, (4) создать действенные планы улучшения с конкретными исправлениями на уровне кода. Запускается по таким запросам, как «проанализировать производительность», «почему моя страница работает медленно», «исправить проблемы CLS», «оптимизировать веб-жизненные показатели», «запустить Lighthouse», «аудит производительности» или «улучшить скорость страницы».
