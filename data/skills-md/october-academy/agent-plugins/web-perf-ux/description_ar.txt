أداء تطبيقات الويب وتحسين تجربة المستخدم من خلال التحليل الآلي. يُستخدم عندما يحتاج المطورون إلى (1) قياس مشكلات الأداء وتشخيصها باستخدام Lighthouse وCore Web Vitals، (2) تحديد الأسباب الجذرية لمشاكل LCP وCLS وINP، (3) تحليل نقاط الضعف في تجربة المستخدم مثل تغييرات التخطيط أو التفاعل البطيء، (4) إنشاء خطط تحسين قابلة للتنفيذ مع إصلاحات ملموسة على مستوى التعليمات البرمجية. مشغلات لطلبات مثل "تحليل الأداء"، أو "لماذا تكون صفحتي بطيئة"، أو "إصلاح مشكلات CLS"، أو "تحسين مؤشرات الويب الحيوية"، أو "تشغيل Lighthouse"، أو "تدقيق الأداء"، أو "تحسين سرعة الصفحة".
