Performances des applications Web et optimisation UX grâce à une analyse automatisée. À utiliser lorsque les développeurs doivent (1) mesurer et diagnostiquer les problèmes de performances à l'aide de Lighthouse et Core Web Vitals, (2) identifier les causes profondes des problèmes LCP, CLS, INP, (3) analyser les problèmes UX tels que les changements de disposition ou l'interactivité lente, (4) générer des plans d'amélioration exploitables avec des correctifs concrets au niveau du code. Se déclenche sur des requêtes telles que « analyser les performances », « pourquoi ma page est-elle lente », « résoudre les problèmes CLS », « optimiser Web Vitals », « exécuter Lighthouse », « audit de performances » ou « améliorer la vitesse de la page ».
