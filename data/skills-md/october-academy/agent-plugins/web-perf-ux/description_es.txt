Rendimiento de aplicaciones web y optimización de UX mediante análisis automatizado. Úselo cuando los desarrolladores necesiten (1) medir y diagnosticar problemas de rendimiento utilizando Lighthouse y Core Web Vitals, (2) identificar las causas fundamentales de los problemas de LCP, CLS e INP, (3) analizar los puntos débiles de UX, como cambios de diseño o interactividad lenta, (4) generar planes de mejora procesables con correcciones concretas a nivel de código. Activadores de solicitudes como "analizar el rendimiento", "por qué mi página es lenta", "solucionar problemas de CLS", "optimizar web vitals", "ejecutar Lighthouse", "auditoría de rendimiento" o "mejorar la velocidad de la página".
