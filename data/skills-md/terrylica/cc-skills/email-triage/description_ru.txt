Запланированный дайджест электронной почты через Agent SDK Haiku. Распределяет электронные письма по 3 категориям, отправляет уведомления Telegram + голос подкаста. ТРИГГЕРЫ — дайджест электронной почты, сортировка электронных писем, дайджест, дайджест запуска, сводка электронной почты, голосовой брифинг.
