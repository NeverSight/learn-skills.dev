Создавайте хорошо отформатированные задачи GitHub с помощью интеллектуальных предложений меток на основе искусственного интеллекта и определения типа контента. Используйте его всякий раз, когда пользователь хочет создать отчет об ошибке, запрос функции, вопрос или проблему с документацией на GitHub или говорит «сообщить о проблеме», «создать проблему» или «создать проблему». НЕ используйте для управления существующими проблемами, организации иерархии проблем (вместо этого используйте Issue-workflow) или для создания связей с общественностью.
