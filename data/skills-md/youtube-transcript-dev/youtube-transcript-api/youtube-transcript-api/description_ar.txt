قم باستخراج ونسخ وترجمة نصوص فيديو YouTube باستخدام YouTubeTranscript.dev V2 API. يدعم التسميات التوضيحية، والنسخ الصوتي ASR، ومعالجة الدفعات (ما يصل إلى 100 مقطع فيديو)، والترجمة إلى أكثر من 100 لغة، وتنسيقات الإخراج المتعددة. يُستخدم عند العمل مع مقاطع فيديو YouTube أو الترجمات المصاحبة أو التسميات التوضيحية أو تحويل الفيديو إلى نص.
