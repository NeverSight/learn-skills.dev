Извлекайте, расшифровывайте и переводите расшифровки видео YouTube с помощью API YouTubeTranscript.dev V2. Поддерживает субтитры, транскрипцию аудио ASR, пакетную обработку (до 100 видео), перевод на более чем 100 языков и несколько форматов вывода. Используйте при работе с видео YouTube, субтитрами, подписями или преобразованием видео в текст.
