Estrai, trascrivi e traduci le trascrizioni dei video di YouTube utilizzando l'API YouTubeTranscript.dev V2. Supporta sottotitoli, trascrizione audio ASR, elaborazione batch (fino a 100 video), traduzione in oltre 100 lingue e più formati di output. Da utilizzare quando si lavora con video, sottotitoli, didascalie o conversioni da video a testo di YouTube.
