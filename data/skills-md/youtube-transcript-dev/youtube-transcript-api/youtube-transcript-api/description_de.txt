Extrahieren, transkribieren und übersetzen Sie YouTube-Videotranskripte mit der YouTubeTranscript.dev V2-API. Unterstützt Untertitel, ASR-Audiotranskription, Stapelverarbeitung (bis zu 100 Videos), Übersetzung in über 100 Sprachen und mehrere Ausgabeformate. Verwenden Sie diese Option, wenn Sie mit YouTube-Videos, Untertiteln, Bildunterschriften oder der Video-zu-Text-Konvertierung arbeiten.
