Conseils d’experts pour la configuration et le déploiement d’OpenTelemetry Collector. À utiliser lors de la configuration d'un pipeline Collector, de la configuration de récepteurs, d'exportateurs ou de processeurs, du déploiement d'un collecteur sur Kubernetes ou Docker, ou du transfert de télémétrie vers Dash0. Se déclenche sur les demandes impliquant la configuration d'un collecteur, d'un pipeline, d'un récepteur OTLP, d'un exportateur ou d'un collecteur Dash0.
