Fachkundige Anleitung zur Konfiguration und Bereitstellung des OpenTelemetry Collector. Verwenden Sie diese Option, wenn Sie eine Collector-Pipeline einrichten, Empfänger, Exporteure oder Prozessoren konfigurieren, einen Collector für Kubernetes oder Docker bereitstellen oder Telemetriedaten an Dash0 weiterleiten. Wird bei Anfragen ausgelöst, die die Einrichtung von Collector, Pipeline, OTLP-Empfänger, Exporter oder Dash0-Collector betreffen.
