Эксперт по языку преобразования OpenTelemetry (OTTL). Используйте при написании или отладке выражений OTTL для любого компонента OpenTelemetry Collector, который поддерживает OTTL (процессоры, соединители, приемники, экспортеры). Запускается при выполнении задач, связанных с преобразованием телеметрии, фильтрацией, манипулированием атрибутами, редактированием данных, политиками выборки, маршрутизацией или конфигурацией коллектора. Охватывает синтаксис, контексты, функции, обработку ошибок и производительность.
