Experte für OpenTelemetry Transformation Language (OTTL). Wird zum Schreiben oder Debuggen von OTTL-Ausdrücken für jede OpenTelemetry Collector-Komponente verwendet, die OTTL unterstützt (Prozessoren, Konnektoren, Empfänger, Exporter). Wird bei Aufgaben ausgelöst, die Telemetrietransformation, Filterung, Attributmanipulation, Datenschwärzung, Sampling-Richtlinien, Routing oder Collector-Konfiguration umfassen. Behandelt Syntax, Kontexte, Funktionen, Fehlerbehandlung und Leistung.
