خبير في الاتفاقيات الدلالية للقياس عن بعد. يُستخدم عند تحديد أو تطبيق أو مراجعة سمات القياس عن بعد أو أسماء الامتدادات أو أنواع الامتدادات أو رموز حالة الامتداد. تشغيل المهام التي تتضمن اختيار السمات، أو قياس البيانات عن بعد، أو الامتثال للاصطلاحات الدلالية، أو ترحيل السمات، أو قرارات السمات المخصصة. يغطي تسجيل السمات، وأنماط التسمية، وتعيين الحالة، وموضع السمة، والإصدار.
