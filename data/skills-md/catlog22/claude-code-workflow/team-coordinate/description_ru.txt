Универсальные навыки координации команды с динамическим генерированием ролей. Использует архитектуру агента «командный работник» с файлами спецификаций ролей. Встроенный только координатор — все рабочие роли генерируются во время выполнения как спецификации ролей и создаются через агента коллективного рабочего. Модель ритма/каденции для оркестровки. Триггеры по «Координате команды».
