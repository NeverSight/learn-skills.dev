Навыки гибридной команды для планирования и реализации конвейера. Волна CSV является основной для планирования и выполнения. Планировщик разбивает требования на проблемы и решения, а затем исполнитель реализует каждое из них с помощью инструментов CLI. Поддерживает идентификаторы задач, ввод текста и ввод файла плана.
