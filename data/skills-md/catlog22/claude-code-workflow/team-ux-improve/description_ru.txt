Единые командные навыки для улучшения UX. Систематически обнаруживает и устраняет проблемы взаимодействия UI/UX, включая неотвечающие кнопки, отсутствие обратной связи и проблемы с обновлением состояния. Использует архитектуру агента «команда-работник» с ролями/ для логики предметной области. Координатор управляет конвейером, работники — это агенты командной работы. Запускается по команде «улучшение пользовательского интерфейса».
