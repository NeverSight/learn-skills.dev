Compétence d’équipe unifiée pour un flux de travail de développement basé sur une feuille de route. Le coordinateur discute de la feuille de route avec l'utilisateur, puis distribue le pipeline d'exécution par étapes (planifier -> exécuter -> vérifier). Tous les rôles invoquent cette compétence avec --role arg. Déclencheurs sur "team roadmap-dev".
