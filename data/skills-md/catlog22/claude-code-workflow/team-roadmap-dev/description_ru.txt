Единые командные навыки для рабочего процесса разработки на основе дорожной карты. Координатор обсуждает план действий с пользователем, затем отправляет конвейер поэтапного выполнения (план -> выполнение -> проверка). Все роли вызывают этот навык с помощью --role arg. Запускается по команде «team roadmap-dev».
