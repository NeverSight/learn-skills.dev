Generatore completo di documentazione di progetto basato su onde con scomposizione dinamica delle attività. Analizza la struttura del progetto e genera attività di documentazione appropriate, calcola le onde di esecuzione ottimali tramite l'ordinamento topologico, produce una suite di documentazione completa che include architettura, metodi, teoria, caratteristiche, utilizzo e filosofia di progettazione.
