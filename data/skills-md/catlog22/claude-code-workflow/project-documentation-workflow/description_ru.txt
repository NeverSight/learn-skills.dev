Комплексный генератор проектной документации на волновой основе с динамической декомпозицией задач. Анализирует структуру проекта и генерирует соответствующие задачи по документации, вычисляет оптимальные волны выполнения посредством топологической сортировки, создает полный пакет документации, включая архитектуру, методы, теорию, функции, использование и философию проектирования.
