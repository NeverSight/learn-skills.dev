Wave-basierter umfassender Projektdokumentationsgenerator mit dynamischer Aufgabenzerlegung. Analysiert die Projektstruktur und generiert entsprechende Dokumentationsaufgaben, berechnet optimale Ausführungswellen über topologische Sortierung und erstellt eine vollständige Dokumentationssuite einschließlich Architektur, Methoden, Theorie, Funktionen, Verwendung und Designphilosophie.
