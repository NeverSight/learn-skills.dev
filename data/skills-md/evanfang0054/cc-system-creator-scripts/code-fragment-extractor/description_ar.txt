قم بقراءة واستكشاف أدلة/ملفات التعليمات البرمجية التي يقدمها المستخدم بعناية ودقة، وتحليلها وتقسيمها إلى مقتطفات تعليمات برمجية مقابلة للرجوع إليها، مع الدقة وصولاً إلى المستوى الوظيفي. قم باستخراج المقتطفات فقط دون تغيير محتوى التعليمات البرمجية أو بنية تخطيط الصفحة، واحفظها في الدليل ./fragment-extractor. قم بإنشاء ملفات قالب مقتطف التعليمات البرمجية وفقًا لتنسيق القالب المحدد، مما يضمن الاكتمال الوظيفي والتطبيق العملي لمقتطفات التعليمات البرمجية.
