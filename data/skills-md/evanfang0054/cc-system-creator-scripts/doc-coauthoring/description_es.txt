Guíe a los usuarios a través de un flujo de trabajo estructurado para escribir documentos de forma colaborativa. Utilice este flujo de trabajo cuando el usuario quiera escribir documentos, propuestas, especificaciones técnicas, documentos de decisión o contenido estructurado similar. Ayuda a los usuarios a transmitir contexto de manera eficiente, refinar el contenido a través de iteraciones y verificar que el documento sea adecuado para el público objetivo. Active este flujo de trabajo cada vez que el usuario mencione escribir un documento, crear una propuesta, redactar una especificación o tareas documentales similares.
