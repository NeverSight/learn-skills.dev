Помогите пользователям совместно писать документы с помощью структурированного рабочего процесса. Используется, когда пользователь хочет написать документ, предложение, техническую спецификацию, документ решения или аналогичный структурированный контент. Этот рабочий процесс помогает пользователям эффективно предоставлять контекст, уточнять контент посредством итераций и проверять, подходят ли документы для читателей. Срабатывает, когда пользователь упоминает написание документа, создание предложения, составление спецификации или аналогичную задачу документирования.
