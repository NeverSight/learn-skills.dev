Направляйте пользователей через структурированный рабочий процесс для совместной написания документов. Используйте этот рабочий процесс, когда пользователь хочет написать документы, предложения, технические спецификации, документы решений или аналогичный структурированный контент. Это помогает пользователям эффективно передавать контекст, уточнять контент посредством итераций и проверять, подходит ли документ целевой аудитории. Запускайте этот рабочий процесс всякий раз, когда пользователь упоминает написание документа, создание предложения, составление спецификации или аналогичные задачи по документированию.
