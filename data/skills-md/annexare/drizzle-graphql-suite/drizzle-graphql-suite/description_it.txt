Crea API GraphQL da schemi Drizzle PostgreSQL con CRUD generati automaticamente, client indipendenti dai tipi e hook React Query. Da utilizzare durante la creazione di server GraphQL da tabelle Drizzle ORM, la creazione di client GraphQL sicuri per i tipi, l'aggiunta di hook di recupero dati React con TanStack Query o la generazione di tipi/SDL GraphQL da schemi Drizzle.
