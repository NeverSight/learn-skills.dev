أنشئ واجهات برمجة تطبيقات GraphQL من مخططات Drizzle PostgreSQL باستخدام CRUD التي يتم إنشاؤها تلقائيًا وعملاء النوع الآمن وخطافات React Query. يُستخدم عند إنشاء خوادم GraphQL من جداول Drizzle ORM، أو إنشاء عملاء GraphQL آمني النوع، أو إضافة خطافات جلب بيانات React باستخدام TanStack Query، أو إنشاء أنواع/SDL لـ GraphQL من مخططات Drizzle.
