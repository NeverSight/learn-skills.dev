Создавайте API-интерфейсы GraphQL на основе схем Drizzle PostgreSQL с помощью автоматически создаваемых CRUD, типобезопасных клиентов и перехватчиков React Query. Используйте при создании серверов GraphQL из таблиц Drizzle ORM, создании типобезопасных клиентов GraphQL, добавлении перехватчиков получения данных React с помощью TanStack Query или создании SDL/типов GraphQL из схем Drizzle.
