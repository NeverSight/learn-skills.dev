Семантический поиск кода и понимание кодовой базы с использованием инструментов git-ai MCP. Используйте, когда: (1) Поиск символов, функций или семантических концепций, (2) Понимание архитектуры проекта, (3) Анализ графиков вызовов и взаимосвязей кода. Триггеры: «найти X», «поиск X», «кто звонит X», «где находится X», «понять эту кодовую базу».
