Recherche de code sémantique et compréhension de la base de code à l'aide des outils git-ai MCP. À utiliser lorsque : (1) Recherche de symboles, de fonctions ou de concepts sémantiques, (2) Compréhension de l'architecture du projet, (3) Analyse des graphiques d'appels et des relations de code. Déclencheurs : "trouver X", "rechercher X", "qui appelle X", "où est X", "comprendre cette base de code".
