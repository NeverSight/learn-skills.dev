Z.AI CLI fournissant :
- Vision : analyse d'image/vidéo, OCR, UI-to-code, diagnostic d'erreur (GLM-4.6V)
- Recherche : recherche Web en temps réel avec filtrage de domaine/récence
- Reader : page Web à extraire
- Repo : recherche et lecture de code GitHub via ZRead
- Outils : découverte d'outils MCP et appels bruts
- Code : chaînage d'outils TypeScript
À utiliser pour l'analyse de contenu visuel, la recherche sur le Web, la lecture de pages ou l'exploration GitHub. Nécessite Z_AI_API_KEY.
