Z.AI CLI que proporciona:
- Visión: análisis de imagen/vídeo, OCR, UI-to-code, diagnóstico de errores (GLM-4.6V)
- Búsqueda: búsqueda web en tiempo real con filtrado de dominio/reciente
- Lector: página web para extracción de rebajas.
- Repo: búsqueda y lectura de código GitHub a través de ZRead
- Herramientas: descubrimiento de herramientas MCP y llamadas sin procesar
- Código: encadenamiento de herramientas TypeScript
Úselo para análisis de contenido visual, búsqueda web, lectura de páginas o exploración de GitHub. Requiere Z_AI_API_KEY.
