Z.AI CLI che fornisce:
- Visione: analisi immagini/video, OCR, UI-to-code, diagnosi errori (GLM-4.6V)
- Ricerca: ricerca web in tempo reale con filtro dominio/recency
- Lettore: pagina web per l'estrazione del markdown
- Repo: ricerca e lettura del codice GitHub tramite ZRead
- Strumenti: rilevamento degli strumenti MCP e chiamate grezze
- Codice: concatenamento degli strumenti TypeScript
Utilizzare per l'analisi visiva dei contenuti, la ricerca sul Web, la lettura di pagine o l'esplorazione di GitHub. Richiede Z_AI_API_KEY.
