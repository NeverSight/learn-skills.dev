Z.AI CLI обеспечивает:
- Vision: анализ изображений/видео, оптическое распознавание текста, преобразование пользовательского интерфейса в код, диагностика ошибок (GLM-4.6V)
- Поиск: веб-поиск в реальном времени с фильтрацией по домену/давности.
- Reader: веб-страница для извлечения уценки.
- Репо: поиск и чтение кода GitHub через ZRead.
- Инструменты: обнаружение инструментов MCP и необработанные вызовы.
— Код: цепочка инструментов TypeScript.
Используйте для визуального анализа контента, веб-поиска, чтения страниц или исследования GitHub. Требуется Z_AI_API_KEY.
