توفير Z.AI CLI:
- الرؤية: تحليل الصور/الفيديو، التعرف الضوئي على الحروف، واجهة المستخدم إلى الكود، تشخيص الأخطاء (GLM-4.6V)
- البحث: بحث الويب في الوقت الفعلي مع تصفية المجال/الحداثة
- القارئ: صفحة ويب لاستخراج تخفيض السعر
- الريبو: البحث عن كود GitHub وقراءته عبر ZRead
- الأدوات: اكتشاف أداة MCP والمكالمات الأولية
- الكود: تسلسل أداة TypeScript
يُستخدم لتحليل المحتوى المرئي أو بحث الويب أو قراءة الصفحة أو استكشاف GitHub. يتطلب Z_AI_API_KEY.
