Obtenga subtítulos de videos de YouTube utilizando la API de subtítulos de YouTube de Felo. Úselo cuando los usuarios solicitan obtener subtítulos de YouTube, extraer subtítulos de un video, obtener una transcripción por ID de video o URL, o cuando se usan comandos explícitos como /felo-youtube-subtitling. Admite idioma y marcas de tiempo opcionales.
