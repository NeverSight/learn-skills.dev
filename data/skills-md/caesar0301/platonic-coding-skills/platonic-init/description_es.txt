Inicialice el sistema de codificación platónica para cualquier proyecto. Andamia la infraestructura de especificaciones (.platonic.yml, plantillas RFC, directorio de guía impl) y escanea las bases de código existentes para recuperar las especificaciones de diseño conceptual y arquitectónico que faltan como borradores de RFC. Úselo al adoptar codificación platónica para un proyecto nuevo o existente.
