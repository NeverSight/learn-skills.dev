Initialisez le système de codage platonique pour n’importe quel projet. Les échafaudages spécifient l'infrastructure (.platonic.yml, modèles RFC, répertoire de guides impl) et analysent les bases de code existantes pour récupérer les spécifications de conception conceptuelle et architecturale manquantes sous forme de projets de RFC. À utiliser lors de l’adoption du codage platonique pour un projet nouveau ou existant.
