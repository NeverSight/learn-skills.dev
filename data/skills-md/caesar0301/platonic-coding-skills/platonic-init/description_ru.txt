Инициализируйте систему Platonic Coding для любого проекта. Создает спецификации инфраструктуры (.platonic.yml, шаблоны RFC, каталог руководств по реализации) и сканирует существующие базы кода для восстановления недостающих спецификаций концептуального и архитектурного проектирования в виде черновиков RFC. Используйте при принятии платонического кодирования для нового или существующего проекта.
