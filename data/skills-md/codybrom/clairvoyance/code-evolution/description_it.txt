Valuta se le modifiche al codice esistente mantengono o riducono la qualità della progettazione. Da utilizzare quando si esaminano le modifiche al codice esistente (diff, PR o file modificati di recente) per valutare se ogni modifica sembra progettata o integrata. Non per confrontare un elenco di controllo di odori di progettazione (usare segnali d'allarme) o valutare l'investimento complessivo nella progettazione (usare una mentalità strategica).
