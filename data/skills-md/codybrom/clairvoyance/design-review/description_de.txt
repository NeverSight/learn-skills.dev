Orchestriert eine strukturierte Designüberprüfung, indem vorhandene Fähigkeiten in einem Diagnosetrichter ausgeführt werden, von der Komplexitätstriage über Struktur-, Schnittstellen- und Oberflächenprüfungen bis hin zu einer vollständigen Überprüfung der Warnsignale. Verwenden Sie diese Option, wenn Sie eine Datei, ein Modul oder eine PR auf Gesamtqualität des Designs überprüfen und eine umfassende, priorisierte Bewertung anstelle einer Einzelprüfung wünschen. Nicht für die Anwendung einer bestimmten Linse (verwenden Sie diese Fähigkeit direkt) oder für die evolutionäre Analyse, wie sich Code im Laufe der Zeit verändert hat (verwenden Sie Code-Evolution).
