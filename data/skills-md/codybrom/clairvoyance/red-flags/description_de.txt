Scannt Code anhand von 17 benannten Design-Gerüchen und erstellt einen strukturierten Diagnosebericht. Verwenden Sie diese Funktion, wenn Sie eine PR auf Designqualität überprüfen, unbekannten Code anhand einer umfassenden Checkliste bewerten oder wenn der Benutzer einen Scan auf Warnsignale anfordert. Nicht zur Diagnose, warum sich Code komplex anfühlt (verwenden Sie Komplexitätserkennung) oder zur Bewertung, ob ein PR die Design-Trajektorie beibehält (verwenden Sie Code-Evolution).
