Analyse le code par rapport à 17 odeurs de conception nommées et produit un rapport de diagnostic structuré. À utiliser lors de l'examen d'un PR pour la qualité de la conception, de l'évaluation d'un code inconnu par rapport à une liste de contrôle complète ou lorsque l'utilisateur demande une analyse des signaux d'alarme. Pas pour diagnostiquer pourquoi le code semble complexe (utiliser la reconnaissance de la complexité) ou évaluer si un PR maintient la trajectoire de conception (utiliser l'évolution du code).
