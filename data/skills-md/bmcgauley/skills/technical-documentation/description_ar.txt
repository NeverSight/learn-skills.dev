مهارة التوثيق الفني الشاملة لإنشاء وثائق واضحة ودقيقة ومرتكزة على المستخدم عبر جميع التنسيقات بما في ذلك مستندات API وأدلة المستخدم ووثائق المطورين وقواعد المعرفة ووثائق النظام. يجب استخدام هذه المهارة عند إنشاء أو تنظيم أو صيانة الوثائق الفنية التي تشرح الأنظمة أو الإجراءات أو المفاهيم المعقدة لمختلف الجماهير التقنية وغير الفنية.
