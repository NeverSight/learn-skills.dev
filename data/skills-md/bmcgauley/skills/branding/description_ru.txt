Комплексная стратегия бренда и навыки разработки индивидуальности для создания целостной системы бренда во всех точках взаимодействия. Этот навык следует использовать при разработке фирменного стиля, систем визуального дизайна, систем обмена сообщениями или реализации руководящих принципов бренда для любого проекта или организации.
