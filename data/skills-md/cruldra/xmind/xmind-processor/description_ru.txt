Программно создавайте, читайте и изменяйте файлы карт связей XMind (.xmind). Предоставляет утилиты Python для анализа файлов xmind, вставки тем, установки цветов фона и создания новых интеллект-карт на основе шаблонов. Используйте при работе с файлами XMind для автоматизации, пакетной обработки или программного создания интеллект-карт.
