Crea, leggi e modifica a livello di codice i file delle mappe mentali XMind (.xmind). Fornisce utilità Python per l'analisi di file xmind, l'inserimento di argomenti, l'impostazione dei colori di sfondo e la creazione di nuove mappe mentali da modelli. Da utilizzare quando si lavora con file XMind per l'automazione, l'elaborazione batch o la generazione di mappe mentali a livello di codice.
