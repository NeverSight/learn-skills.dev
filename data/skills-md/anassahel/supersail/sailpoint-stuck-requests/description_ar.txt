قم بتشخيص وحل طلبات الوصول إلى SailPoint المتوقفة أو المنتظرة أو المعلقة باستخدام واجهة سطر الأوامر الخاصة بالشراع. استخدم هذه المهارة عندما يذكر المستخدم طلب وصول عالقًا، أو طلبًا معلقًا لن يكتمل، أو مرحلة توفير معلقة، أو يريد إلغاء طلب وصول أو إغلاقه بالقوة، أو يسأل عن طلبات الوصول في حالة الانتظار/التنفيذ في SailPoint Identity Security Cloud. يتم تشغيله أيضًا عندما يقول المستخدم أشياء مثل "طلبي عالق"، أو "لن يتم تنفيذ طلب الوصول"، أو "معلق لعدة أيام"، أو "لا يمكنني رؤية طلبي".
