Diagnostizieren und lösen Sie blockierte, wartende oder blockierte SailPoint-Zugriffsanfragen mithilfe der Sail-CLI. Verwenden Sie diesen Skill immer dann, wenn der Benutzer eine blockierte Zugriffsanforderung, eine ausstehende Anforderung, die nicht abgeschlossen werden kann, eine blockierte Bereitstellungsphase erwähnt, eine Zugriffsanforderung abbrechen oder erzwingen möchte oder nach Zugriffsanforderungen fragt, die sich in der SailPoint Identity Security Cloud im Warte-/Ausführungsstatus befinden. Wird auch ausgelöst, wenn der Benutzer Dinge wie „Meine Anfrage bleibt hängen“, „Zugriffsanfrage wird nicht bearbeitet“, „Seit Tagen ausstehend“ oder „Ich kann meine Anfrage nicht sehen“ sagt.
