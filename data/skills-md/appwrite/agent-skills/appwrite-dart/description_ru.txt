Appwrite навык Dart SDK. Используйте при создании приложений Flutter (мобильных, веб-, настольных) или серверных приложений Dart с помощью Appwrite. Охватывает аутентификацию на стороне клиента (электронная почта, OAuth), запросы к базе данных, загрузку файлов с собственной обработкой файлов, подписки в реальном времени и администрирование на стороне сервера с помощью ключей API для управления пользователями, администрирования базы данных, хранения и функций.
