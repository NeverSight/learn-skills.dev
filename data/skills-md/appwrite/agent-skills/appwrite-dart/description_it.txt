Competenza Appwrite Dart SDK. Da utilizzare durante la creazione di app Flutter (dispositivi mobili, Web, desktop) o applicazioni Dart lato server con Appwrite. Copre l'autenticazione lato client (e-mail, OAuth), query sul database, caricamenti di file con gestione nativa dei file, abbonamenti in tempo reale e amministrazione lato server tramite chiavi API per la gestione degli utenti, l'amministrazione del database, l'archiviazione e le funzioni.
