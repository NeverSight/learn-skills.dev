Abilità Appwrite Kotlin SDK. Da utilizzare durante la creazione di app Android native o backend Kotlin/JVM lato server con Appwrite. Copre l'autenticazione lato client (e-mail, OAuth con integrazione attività), query di database, caricamenti di file, abbonamenti in tempo reale con supporto coroutine e amministrazione lato server tramite chiavi API per la gestione degli utenti, l'amministrazione del database, l'archiviazione e le funzioni.
