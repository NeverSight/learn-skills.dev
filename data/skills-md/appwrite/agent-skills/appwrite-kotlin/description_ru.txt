Appwrite навык Kotlin SDK. Используйте при создании собственных приложений для Android или серверных серверных частей Kotlin/JVM с помощью Appwrite. Охватывает аутентификацию на стороне клиента (электронная почта, OAuth с интеграцией действий), запросы к базе данных, загрузку файлов, подписки в реальном времени с поддержкой сопрограмм и администрирование на стороне сервера через ключи API для управления пользователями, администрирования базы данных, хранения и функций.
