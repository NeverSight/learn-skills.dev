Appwrite مهارة Kotlin SDK. يُستخدم عند إنشاء تطبيقات Android الأصلية أو واجهات Kotlin/JVM الخلفية من جانب الخادم باستخدام Appwrite. يغطي المصادقة من جانب العميل (البريد الإلكتروني، OAuth مع تكامل النشاط)، واستعلامات قاعدة البيانات، وتحميل الملفات، والاشتراكات في الوقت الفعلي مع دعم coroutine، والمسؤول من جانب الخادم عبر مفاتيح API لإدارة المستخدم، وإدارة قاعدة البيانات، والتخزين، والوظائف.
