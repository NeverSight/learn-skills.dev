Навыки написания приложений Swift SDK. Используйте при создании собственных приложений iOS, macOS, watchOS или tvOS или серверных приложений Swift с помощью Appwrite. Охватывает аутентификацию на стороне клиента (электронная почта, OAuth), запросы к базе данных, загрузку файлов, подписки в реальном времени с асинхронным/ожиданием и администрирование на стороне сервера с помощью ключей API для управления пользователями, администрирования базы данных, хранения и функций.
