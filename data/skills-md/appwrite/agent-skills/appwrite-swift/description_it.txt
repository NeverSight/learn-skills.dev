Competenza Appwrite Swift SDK. Da utilizzare durante la creazione di app native per iOS, macOS, watchOS o tvOS o applicazioni Swift lato server con Appwrite. Copre l'autenticazione lato client (e-mail, OAuth), query di database, caricamenti di file, abbonamenti in tempo reale con async/await e amministrazione lato server tramite chiavi API per la gestione degli utenti, l'amministrazione del database, l'archiviazione e le funzioni.
