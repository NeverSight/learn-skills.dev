Habilidad Appwrite Swift SDK. Úselo al crear aplicaciones nativas de iOS, macOS, watchOS o tvOS, o aplicaciones Swift del lado del servidor con Appwrite. Cubre la autenticación del lado del cliente (correo electrónico, OAuth), consultas de bases de datos, carga de archivos, suscripciones en tiempo real con async/await y administración del lado del servidor mediante claves API para gestión de usuarios, administración de bases de datos, almacenamiento y funciones.
