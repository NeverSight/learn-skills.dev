Отправьте USDC из кошелька агента t2000 на другой адрес Sui. Используйте, когда вас просят заплатить кому-то, перевести средства, отправить деньги, дать чаевые создателю или совершить платеж на определенный адрес Sui. НЕ используйте для платежей через API — используйте t2000-pay для сервисов, защищенных x402.
