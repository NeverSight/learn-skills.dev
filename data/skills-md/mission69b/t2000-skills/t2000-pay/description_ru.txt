Оплатите услугу API, защищенную x402, с помощью кошелька t2000. Используйте, когда API возвращает ответ 402 Payment Required, когда его просят «вызвать этот платный API», «оплатить данные из», «получить доступ к службе x402» или при получении ресурса, требующего микроплатежей. Автоматически обрабатывает полное рукопожатие x402.
