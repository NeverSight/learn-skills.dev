Paga un servizio API protetto x402 utilizzando il portafoglio t2000. Da utilizzare quando un'API restituisce una risposta 402 di pagamento richiesto, quando viene chiesto di "chiamare l'API a pagamento", "pagare i dati da", "accedere al servizio x402 su" o quando si recupera una risorsa che richiede un micropagamento. Gestisce automaticamente l'handshake x402 completo.
