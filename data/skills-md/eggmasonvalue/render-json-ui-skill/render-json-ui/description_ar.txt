قم بتحويل عناصر JSON إلى واجهة مستخدم قابلة للقراءة من خلال سير عمل الفحص أولاً والحقائق أولاً. يُستخدم عندما يحتاج Codex إلى تحويل ملفات JSON، أو أوامر shell المنتجة لـ JSON، أو عناصر إخراج CLI، أو الحمولات المنظمة غير المعروفة إلى مواصفات واجهة مستخدم تعريفية يمكن عرضها محليًا بواسطة أداة التسخير أو من خلال عارض مرجعي طرفي أصلي، بما في ذلك الحالات ذات السجلات الفرعية المتكررة المشفرة كمصفوفات محاذية.
