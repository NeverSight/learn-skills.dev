Rendern Sie JSON-Artefakte in einer lesbaren Benutzeroberfläche mit einem Arbeitsablauf, bei dem zuerst die Prüfung und die Fakten im Vordergrund stehen. Wird verwendet, wenn Codex JSON-Dateien, JSON-produzierende Shell-Befehle, CLI-Ausgabeartefakte oder unbekannte strukturierte Nutzlasten in eine deklarative UI-Spezifikation umwandeln muss, die nativ vom Harness oder über einen terminal-nativen Referenzrenderer gerendert werden kann, einschließlich Fällen mit wiederholten untergeordneten Datensätzen, die als ausgerichtete Arrays kodiert sind.
