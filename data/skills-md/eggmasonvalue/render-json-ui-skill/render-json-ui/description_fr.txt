Restituez les artefacts JSON dans une interface utilisateur lisible avec un flux de travail axé d'abord sur l'inspection et sur les faits. À utiliser lorsque le Codex doit transformer des fichiers JSON, des commandes shell produisant du JSON, des artefacts de sortie CLI ou des charges utiles structurées inconnues en une spécification d'interface utilisateur déclarative qui peut être restituée de manière native par le harnais ou via un moteur de rendu de référence natif du terminal, y compris les cas avec des enregistrements enfants répétés codés sous forme de tableaux alignés.
