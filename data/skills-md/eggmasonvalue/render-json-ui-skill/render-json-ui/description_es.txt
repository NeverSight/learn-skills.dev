Represente artefactos JSON en una interfaz de usuario legible con un flujo de trabajo que prioriza los hechos y la inspección. Úselo cuando Codex necesite convertir archivos JSON, comandos de shell que producen JSON, artefactos de salida de CLI o cargas útiles estructuradas desconocidas en una especificación de interfaz de usuario declarativa que se puede representar de forma nativa mediante el arnés o mediante un renderizador de referencia nativo de terminal, incluidos los casos con registros secundarios repetidos codificados como matrices alineadas.
