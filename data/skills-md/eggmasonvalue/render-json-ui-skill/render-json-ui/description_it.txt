Esegui il rendering degli artefatti JSON in un'interfaccia utente leggibile con un flusso di lavoro incentrato sull'ispezione e sui fatti. Da utilizzare quando Codex deve trasformare file JSON, comandi shell che producono JSON, artefatti di output della CLI o payload strutturati sconosciuti in una specifica dell'interfaccia utente dichiarativa di cui è possibile eseguire il rendering in modo nativo dal cablaggio o tramite un renderer di riferimento nativo del terminale, inclusi i casi con record secondari ripetuti codificati come array allineati.
