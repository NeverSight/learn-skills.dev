oRPC (v1.12+) طبقة API الآمنة لـ TypeScript. يغطي تطوير العقد أولاً، وتعريف الإجراء، والبرمجيات الوسيطة، وأجهزة التوجيه، والسياق، ومعالجة الأخطاء، وتكامل Hono، واستعلام TanStack، ومكررات الأحداث، والمكونات الإضافية، وتحميل الملفات، ومحول WebSocket، وأفضل الممارسات. يُستخدم عند إنشاء أو تعديل إجراءات واجهة برمجة التطبيقات (API)، أو خطافات جلب البيانات، أو معالجات الخادم، أو ميزات الوقت الفعلي. تشغيل المهام التي تتضمن oRPC، أو واجهات برمجة التطبيقات الآمنة للطباعة، أو إجراءات RPC، أو معالجات مسار الخادم.
