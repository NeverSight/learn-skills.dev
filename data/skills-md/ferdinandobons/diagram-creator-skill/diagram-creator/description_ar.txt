قم بإنشاء مخططات معمارية جميلة وجاهزة للإنتاج كملفات HTML قائمة بذاتها من أي إدخال. استخدم هذه المهارة عندما يريد المستخدم إنشاء رسم تخطيطي، أو مخطط، أو مخطط انسيابي، أو رسم تخطيطي للشبكة، أو تصور النظام، أو الجدول الزمني، أو المخطط التنظيمي، أو أي تمثيل مرئي للمفاهيم - تقنية أو غير فنية. يتم تشغيله عندما يذكر المستخدم "مخططًا" أو "مخططًا" أو "هندسة معمارية" أو "تدفق" أو "هيكل" أو "خط أنابيب" أو "خط زمني" أو "تصور" أو "رسم بياني"، أو يطلب تحويل ملف إلى رسم تخطيطي مرئي. يقبل أي نوع ملف كمدخل (.md، .txt، .json، .yaml، .csv، .pdf، .py، .ts، .toml، .env، .tf، .dockerfile، وما إلى ذلك) - يكتشف كلود التنسيق تلقائيًا ويستخرج البنية ذات الصلة للرسم التخطيطي. حتى الجملة البسيطة مثل "رسم تخطيطي لكيفية عمل OAuth" أو "تصور هذا الملف" تؤدي إلى تفعيل هذه المهارة.
