Genere hermosos diagramas de arquitectura listos para producción como archivos HTML independientes a partir de cualquier entrada. Utilice esta habilidad siempre que el usuario desee crear un diagrama, esquema, diagrama de flujo, diagrama de red, visualización del sistema, línea de tiempo, organigrama o cualquier representación visual de conceptos, técnicos o no técnicos. Se activa cuando el usuario menciona "diagrama", "esquema", "arquitectura", "flujo", "topología", "canalización", "línea de tiempo", "visualizar", "gráfico" o solicita convertir un archivo en un diagrama visual. Acepta CUALQUIER tipo de archivo como entrada (.md, .txt, .json, .yaml, .csv, .pdf, .py, .ts, .toml, .env, .tf, .dockerfile, etc.): Claude detecta automáticamente el formato y extrae la estructura relevante para diagramar. Incluso una frase simple como "diagrama de cómo funciona OAuth" o "visualizar este archivo" activa esta habilidad.
