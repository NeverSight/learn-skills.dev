Générez de superbes diagrammes d'architecture prêts pour la production sous forme de fichiers HTML autonomes à partir de n'importe quelle entrée. Utilisez cette compétence chaque fois que l'utilisateur souhaite créer un diagramme, un schéma, un organigramme, un diagramme de réseau, une visualisation du système, une chronologie, un organigramme ou toute représentation visuelle de concepts - techniques ou non techniques. Déclenchement lorsque l'utilisateur mentionne « diagramme », « schéma », « architecture », « flux », « topologie », « pipeline », « chronologie », « visualiser », « graphique » ou demande de transformer un fichier en diagramme visuel. Accepte TOUT type de fichier en entrée (.md, .txt, .json, .yaml, .csv, .pdf, .py, .ts, .toml, .env, .tf, .dockerfile, etc.) — Claude détecte automatiquement le format et extrait la structure pertinente pour le diagramme. Même une simple phrase comme « schéma du fonctionnement d'OAuth » ou « visualiser ce fichier » déclenche cette compétence.
