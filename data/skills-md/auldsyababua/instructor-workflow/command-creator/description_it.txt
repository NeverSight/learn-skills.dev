Definire comandi barra personalizzati del codice Claude per gli agenti nel framework di applicazione di Traycer. Questa capacità deve essere utilizzata durante la creazione o l'aggiornamento degli agenti e la necessità di specificare prompt riutilizzabili che gli agenti possono eseguire come comandi barra. I comandi sono file Markdown archiviati in .claude/commands/ e a cui si fa riferimento nei file config.yaml dell'agente. Questo è per i comandi slash di Claude Code (/command-name), non per i comandi bash/CLI.
