Defina comandos de barra diagonal personalizados de Claude Code para los agentes en el marco de aplicación de Traycer. Esta habilidad se debe utilizar al crear o actualizar agentes y cuando sea necesario especificar mensajes reutilizables que los agentes puedan ejecutar como comandos de barra diagonal. Los comandos son archivos Markdown almacenados en .claude/commands/ y a los que se hace referencia en los archivos agent config.yaml. Esto es para los comandos de barra diagonal de Claude Code (/nombre-comando), no para los comandos bash/CLI.
