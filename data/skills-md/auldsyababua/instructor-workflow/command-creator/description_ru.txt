Определите специальные команды косой черты Claude Code для агентов в системе обеспечения соблюдения Traycer. Этот навык следует использовать при создании или обновлении агентов, а также при необходимости указать многократно используемые подсказки, которые агенты могут выполнять в виде команд с косой чертой. Команды — это файлы Markdown, хранящиеся в .claude/commands/, на которые есть ссылки в файлах config.yaml агента. Это для команд косой черты Claude Code (/имя-команды), а не для команд bash/CLI.
