Извлекайте, консолидируйте и приоритезируйте все комментарии из запросов на извлечение GitHub для систематической проверки кода. Получает как встроенные комментарии к обзору, так и общие PR-обсуждения, затем анализирует и систематизирует их по приоритету (критические ошибки/безопасность, улучшения дизайна, придирки к стилю). Используйте при работе с PR-обзорами, консолидации отзывов от нескольких рецензентов или создании планов действий на основе комментариев к обзору.
