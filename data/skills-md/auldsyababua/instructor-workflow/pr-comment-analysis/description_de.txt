Extrahieren, konsolidieren und priorisieren Sie alle Kommentare aus GitHub Pull Requests für eine systematische Codeüberprüfung. Ruft sowohl Inline-Bewertungskommentare als auch allgemeine PR-Gespräche ab und analysiert und organisiert sie dann nach Priorität (kritische Fehler/Sicherheit, Designverbesserungen, Stilfehler). Verwenden Sie diese Funktion, wenn Sie mit PR-Bewertungen arbeiten, das Feedback mehrerer Prüfer konsolidieren oder Aktionspläne aus Bewertungskommentaren erstellen.
