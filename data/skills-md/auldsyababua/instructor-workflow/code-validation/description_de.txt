Automatisierte Codevalidierung für Diff-Überprüfung, Änderungshygiene und Erkennung von Warnsignalen. Verwenden Sie diese Option, wenn Sie Git-Diffs, PRs oder geänderte Dateien überprüfen, um Deaktivierungsmuster, Geheimnisse, Pfadportabilitätsprobleme, Sicherheitsflags und große Löschungen zu testen. Unterstützt Python, TypeScript, JavaScript, HTML und CSS.
