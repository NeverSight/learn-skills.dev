Автоматическая проверка кода для проверки различий, гигиены изменений и обнаружения тревожных сигналов. Используйте при просмотре различий git, PR или измененных файлов для проверки шаблонов отключения, секретов, проблем переносимости путей, флагов безопасности и крупных удалений. Поддерживает Python, TypeScript, JavaScript, HTML и CSS.
