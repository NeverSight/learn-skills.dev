Convalida automatizzata del codice per la revisione delle differenze, l'igiene delle modifiche e il rilevamento dei segnali di allarme. Da utilizzare durante la revisione di git diff, PR o file modificati per testare modelli di disabilitazione, segreti, problemi di portabilità del percorso, flag di sicurezza ed eliminazioni di grandi dimensioni. Supporta Python, TypeScript, JavaScript, HTML e CSS.
