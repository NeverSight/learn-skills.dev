يجب استخدام هذه المهارة عند إنشاء مستندات متطلبات المنتج (PRDs) أو التحقق من صحتها أو تحويلها إلى تنسيق LLM الأصلي. استخدم هذه المهارة عندما يطلب المستخدم كتابة PRD، أو مراجعة PRD للتأكد من امتثاله، أو تحويل PRD موجود إلى تنسيق يمكن قراءته آليًا، أو ضمان جودة PRD لاستهلاك وكيل الذكاء الاصطناعي. تم تحسين هذه المهارة للمستخدمين الذين يعملون مع أطر عمل الترميز الوكيل الذين يحتاجون إلى مواصفات متطلبات صارمة لمنع تسمم السياق والهلوسة وتحسين الميسا.
