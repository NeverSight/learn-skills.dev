Diese Fähigkeit sollte beim Erstellen, Validieren oder Konvertieren von Produktanforderungsdokumenten (PRDs) in das LLM-native Format verwendet werden. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine PRD schreiben, eine PRD auf Konformität überprüfen, eine vorhandene PRD in ein maschinenlesbares Format konvertieren oder die PRD-Qualität für die Nutzung von KI-Agenten sicherstellen möchte. Diese Fertigkeit ist für Benutzer optimiert, die mit Agenten-Coding-Frameworks arbeiten und strenge Anforderungsspezifikationen benötigen, um Kontextvergiftung, Halluzination und Mesa-Optimierung zu verhindern.
