Этот навык следует использовать при создании, проверке или преобразовании документов с требованиями к продукции (PRD) в собственный формат LLM. Используйте этот навык, когда пользователь просит написать PRD, проверить его на соответствие требованиям, преобразовать существующий PRD в машиночитаемый формат или обеспечить качество PRD для использования агентом ИИ. Этот навык оптимизирован для пользователей, работающих с агентными средами кодирования, которым необходимы строгие спецификации требований для предотвращения контекстного отравления, галлюцинаций и меза-оптимизации.
