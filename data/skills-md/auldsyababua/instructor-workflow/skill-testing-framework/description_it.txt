Fornisce casi di test e strumenti di convalida per le competenze. Da utilizzare quando si creano test per una nuova competenza, si aggiungono test di regressione dopo gli aggiornamenti delle competenze, si eseguono suite di test per verificare la funzionalità delle competenze o si convalida che gli output delle competenze corrispondano ai risultati previsti. Supporta test unitari, test di integrazione e test di regressione con convalida della coppia input/output.
