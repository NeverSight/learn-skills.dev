Cree definiciones de agentes de infraestructura especializados para la gestión de plataformas/servicios (Grafana, Prometheus, Traefik, ERPNext, etc.). Se utiliza cuando el usuario solicita la creación de un agente para una plataforma tecnológica o componente de infraestructura específico. Esta habilidad produce indicaciones completas para los agentes con investigación integrada, SOP, referencias de herramientas y protocolos de transferencia siguiendo el marco de flujo de trabajo de Linear-First Agentic.
