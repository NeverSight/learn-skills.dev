Создайте определения специализированных агентов инфраструктуры для управления платформой/сервисами (Grafana, Prometheus, Traefik, ERPNext и т. д.). Используйте, когда пользователь запрашивает создание агента для конкретной технологической платформы или компонента инфраструктуры. Этот навык создает полные подсказки для агентов с интегрированными исследованиями, СОПами, справочниками по инструментам и протоколами передачи обслуживания в соответствии с линейной структурой агентского рабочего процесса.
