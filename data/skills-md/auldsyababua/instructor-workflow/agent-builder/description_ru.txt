Создайте полные агенты Claude Code для системы обеспечения соблюдения Traycer. Этот навык следует использовать при создании новых агентов или обновлении существующих агентов. Создает все компоненты агента, включая системные приглашения, перехватчики, навыки, команды, сценарии и справочную документацию. Также обновляются координационные документы, в которых перечислены доступные агенты. На основе лучших практик разработки официальных агентов Anthropic.
