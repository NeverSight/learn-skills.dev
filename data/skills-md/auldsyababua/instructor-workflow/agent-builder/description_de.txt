Erstellen Sie vollständige Claude-Code-Agenten für das Traycer-Enforcement-Framework. Diese Fähigkeit sollte beim Erstellen neuer Agenten oder beim Aktualisieren vorhandener Agenten verwendet werden. Erstellt alle Agentenkomponenten, einschließlich Systemeingabeaufforderung, Hooks, Fähigkeiten, Befehle, Skripts und Referenzdokumente. Aktualisiert außerdem Koordinationsdokumente, in denen verfügbare Agenten aufgeführt sind. Basierend auf den offiziellen Best Practices für die Agentenentwicklung von Anthropic.
