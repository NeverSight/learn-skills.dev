قم بإنشاء وكلاء Claude Code الكاملين لإطار تنفيذ Traycer. يجب استخدام هذه المهارة عند إنشاء وكلاء جدد أو تحديث الوكلاء الحاليين. يقوم بإنشاء جميع مكونات الوكيل بما في ذلك موجه النظام والخطافات والمهارات والأوامر والبرامج النصية والمستندات المرجعية. يقوم أيضًا بتحديث وثائق التنسيق التي تسرد الوكلاء المتاحين. استنادًا إلى أفضل ممارسات تطوير الوكيل الرسمي لشركة Anthropic.
