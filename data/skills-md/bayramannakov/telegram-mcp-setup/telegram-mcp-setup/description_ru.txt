Интерактивный мастер настройки подключения Telegram к Claude Code через telegram-mcp.
Проводит пользователей через весь процесс: получение учетных данных API, создание строки сеанса,
безопасное хранение учетных данных и регистрация сервера MCP.

Триггеры: «настройка телеграммы», «подключение телеграммы к Клоду», «настройка телеграммы mcp»,
«установить телеграмму-MCP», «интеграция телеграммы», «телеграмма для Клода»
