Assistant de configuration interactif pour connecter Telegram à Claude Code via telegram-mcp.
Guide les utilisateurs tout au long du processus : obtention des informations d'identification de l'API, génération de la chaîne de session,
stocker les informations d'identification en toute sécurité et enregistrer le serveur MCP.

"installer telegram-mcp", "intégration de télégramme", "télégramme pour Claude"
