Auditer la conformité de l'accessibilité dans le code frontend. À utiliser lorsque l'utilisateur demande de « vérifier l'accessibilité », « /a11y », « audit a11y », « vérifier les WCAG » ou souhaite rechercher des problèmes d'accessibilité. Ne pas utiliser pour le code backend, les fichiers non-UI ou les projets sans sortie HTML/JSX.
