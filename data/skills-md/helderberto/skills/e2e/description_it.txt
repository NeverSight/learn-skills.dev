Scrivi test end-to-end per i flussi utente utilizzando Cypress. Da utilizzare quando l'utente chiede di "scrivere test e2e", "/e2e", "aggiungere test Cypress" o desidera testare un flusso utente end-to-end. Non utilizzare per test unitari, test di componenti o progetti che utilizzano Playwright, Puppeteer o altri framework non Cypress.
