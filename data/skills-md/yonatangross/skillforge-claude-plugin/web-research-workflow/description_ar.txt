شجرة قرارات موحدة لأبحاث الويب ومراقبة المنافسة. التحديد التلقائي لـ WebFetch أو Tavily أو متصفح الوكيل بناءً على خصائص الموقع المستهدف ومفاتيح API المتاحة. يتضمن تتبع صفحات المنافسين وتمييز اللقطات وتنبيهات التغيير. يُستخدم عند البحث عن محتوى الويب، أو الاستخراج، أو استخراج تخفيض السعر الخام، أو التقاط الوثائق، أو مراقبة تغييرات المنافسين.
