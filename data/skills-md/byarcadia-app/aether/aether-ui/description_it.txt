Conoscenza completa dell'API per la creazione di schermate con @byarcadia-app/aether: una libreria di componenti React Native solo per iOS basata su NativeWind/Tailwind. Utilizza questa abilità ogni volta che un utente desidera utilizzare componenti etere, creare uno schermo con etere, aggiungere un pulsante/scheda/elenco/modulo, creare un'interfaccia utente React Native con aether, utilizzare i componenti @byarcadia-app/aether, creare un modulo/schermata/layout utilizzando etere o lavorare con la libreria dei componenti aether NativeWind. DEVE UTILIZZARE per qualsiasi attività che coinvolga componenti aether-ui, stili, ganci o gettoni di colore.
