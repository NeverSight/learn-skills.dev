Umfassende API-Kenntnisse zum Erstellen von Bildschirmen mit @byarcadia-app/aether – einer nur für iOS verfügbaren React Native-Komponentenbibliothek, die auf NativeWind/Tailwind basiert. Verwenden Sie diese Fähigkeit immer dann, wenn ein Benutzer Aether-Komponenten verwenden, einen Bildschirm mit Aether erstellen, eine Schaltfläche/Karte/Liste/Formular hinzufügen, eine React Native-Benutzeroberfläche mit Aether erstellen, @byarcadia-app/aether-Komponenten verwenden, ein Formular/einen Bildschirm/ein Layout mit Aether erstellen oder mit der aether NativeWind-Komponentenbibliothek arbeiten möchte. MUSS für jede Aufgabe verwendet werden, die Aether-UI-Komponenten, Styling, Hooks oder Farbtokens betrifft.
