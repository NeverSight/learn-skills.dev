اكتشف واتصل بخدمات API المدفوعة التابعة لجهات خارجية من خلال سوق Nightmarket. استخدم هذه المهارة عندما يحتاج المستخدم إلى واجهة برمجة تطبيقات تابعة لجهة خارجية، أو يريد العثور على خدمات واجهة برمجة التطبيقات المتاحة، أو عندما تواجه استجابة 402 "الدفع المطلوب" من عنوان URL الخاص بـnightmarket.ai. يُستخدم أيضًا عندما يذكر المستخدم Nightmarket، أو يتصفح واجهات برمجة التطبيقات (APIs) لوكيله، أو يدفع مقابل مكالمات واجهة برمجة التطبيقات (API) مع USDC. حتى إذا لم يذكر المستخدم Nightmarket بالاسم، فاستخدم هذه المهارة إذا كان بحاجة إلى بيانات خارجية أو تحليلات أو أتمتة أو أي خدمة API مدفوعة يمكن لوكيله الاتصال بها.
