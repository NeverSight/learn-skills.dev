Scopri e chiama servizi API di terze parti a pagamento attraverso il marketplace Nightmarket. Utilizza questa abilità ogni volta che l'utente ha bisogno di un'API di terze parti, desidera trovare servizi API disponibili o quando incontri una risposta 402 Pagamento richiesto da un URL nightmarket.ai. Da utilizzare anche quando l'utente menziona Nightmarket, la navigazione nelle API per il proprio agente o il pagamento delle chiamate API con USDC. Anche se l'utente non menziona Nightmarket per nome, utilizza questa competenza se ha bisogno di dati esterni, analisi, automazione o qualsiasi servizio API a pagamento che il suo agente possa chiamare.
