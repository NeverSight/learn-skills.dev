Находите и вызывайте платные сторонние API-сервисы через торговую площадку Nightmarket. Используйте этот навык всякий раз, когда пользователю нужен сторонний API, он хочет найти доступные сервисы API или когда вы встречаете ответ 402 «Требуется оплата» с URL-адреса nightmarket.ai. Также используйте, когда пользователь упоминает Nightmarket, просматривает API своего агента или оплачивает вызовы API с помощью USDC. Даже если пользователь не упоминает Nightmarket по имени, используйте этот навык, если ему нужны внешние данные, аналитика, автоматизация или любая платная служба API, которую может вызвать его агент.
