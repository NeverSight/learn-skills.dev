Entdecken und rufen Sie kostenpflichtige API-Dienste von Drittanbietern über den Nightmarket-Marktplatz an. Verwenden Sie diesen Skill immer dann, wenn der Benutzer eine Drittanbieter-API benötigt, nach verfügbaren API-Diensten suchen möchte oder wenn Sie von einer nightmarket.ai-URL auf die Antwort „402 Zahlung erforderlich“ stoßen. Verwenden Sie es auch, wenn der Benutzer Nightmarket erwähnt, APIs für seinen Agenten durchsucht oder API-Aufrufe mit USDC bezahlt. Auch wenn der Benutzer Nightmarket nicht namentlich erwähnt, nutzen Sie diese Fähigkeit, wenn er externe Daten, Analysen, Automatisierung oder einen kostenpflichtigen API-Dienst benötigt, den sein Agent aufrufen könnte.
