Découvrez et appelez des services API tiers payants via le marché Nightmarket. Utilisez cette compétence chaque fois que l'utilisateur a besoin d'une API tierce, souhaite rechercher des services API disponibles ou lorsque vous rencontrez une réponse 402 Paiement requis provenant d'une URL nightmarket.ai. À utiliser également lorsque l'utilisateur mentionne Nightmarket, parcourt les API de son agent ou paie des appels d'API avec USDC. Même si l'utilisateur ne mentionne pas Nightmarket par son nom, utilisez cette compétence s'il a besoin de données externes, d'analyses, d'automatisation ou de tout service API payant que son agent pourrait appeler.
