Nightmarket マーケットプレイスを通じて、有料のサードパーティ API サービスを見つけて呼び出します。このスキルは、ユーザーがサードパーティ API を必要とする場合、利用可能な API サービスを見つけたい場合、または nightmarket.ai URL から 402 Payment Required 応答が発生した場合に使用します。ユーザーが Nightmarket について言及したり、エージェントの API を閲覧したり、USDC で API 呼び出しの料金を支払ったりする場合にも使用します。ユーザーが Nightmarket の名前を言及していない場合でも、外部データ、分析、自動化、またはエージェントが呼び出すことができる有料 API サービスが必要な場合は、このスキルを使用します。
