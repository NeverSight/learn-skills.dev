Agentenzahlungsservice über CrowPay – gibt Ihrem Agenten eine Geldbörse, mit der er für APIs und Dienste bezahlen kann. Verwenden Sie diese Fähigkeit immer dann, wenn der Agent Antworten auf HTTP 402-Zahlungsanforderungen verarbeiten, für eine API bezahlen, Credits oder Abonnements kaufen, einen Händler mit einer Kreditkarte bezahlen, eine Agenten-Wallet einrichten oder Ausgabenregeln verwalten muss. Verwenden Sie es auch, wenn der Benutzer CrowPay, Crow, Agentenzahlungen, USDC, x402 oder den Kauf von Gegenständen im Internet für den Agenten erwähnt – auch wenn er „Crow“ nicht namentlich nennt.
