خدمة دفع الوكيل عبر CrowPay - تمنح وكيلك محفظة للدفع مقابل واجهات برمجة التطبيقات والخدمات. استخدم هذه المهارة عندما يحتاج الوكيل إلى التعامل مع استجابات الدفع المطلوبة لـ HTTP 402، أو الدفع مقابل واجهة برمجة التطبيقات، أو شراء أرصدة أو اشتراكات، أو الدفع للتاجر باستخدام بطاقة ائتمان، أو إعداد محفظة وكيل، أو إدارة قواعد الإنفاق. يُستخدم أيضًا عندما يذكر المستخدم CrowPay، أو Crow، أو مدفوعات الوكيل، أو USDC، أو x402، أو شراء أي شيء على الإنترنت للوكيل - حتى لو لم يذكر "Crow" بالاسم.
