Servicio de pago de agentes a través de CrowPay: le brinda a su agente una billetera para pagar las API y los servicios. Utilice esta habilidad siempre que el agente necesite manejar respuestas HTTP 402 Pago requerido, pagar una API, comprar créditos o suscripciones, pagarle a un comerciante con una tarjeta de crédito, configurar una billetera de agente o administrar reglas de gasto. Úselo también cuando el usuario mencione CrowPay, Crow, pagos de agentes, USDC, x402 o comprar algo en Internet para el agente, incluso si no dice "Crow" por su nombre.
