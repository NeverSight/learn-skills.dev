CrowPay によるエージェント支払いサービス — エージェントに API やサービスの支払い用のウォレットを提供します。エージェントが HTTP 402 Payment Required 応答の処理、API の支払い、クレジットまたはサブスクリプションの購入、クレジット カードによる販売者への支払い、エージェント ウォレットの設定、または支出ルールの管理を行う必要がある場合には、このスキルを使用します。また、ユーザーが名前を「Crow」と言わなくても、CrowPay、Crow、エージェント支払い、USDC、x402、またはエージェントのためにインターネット上で何かを購入することに言及する場合にも使用します。
