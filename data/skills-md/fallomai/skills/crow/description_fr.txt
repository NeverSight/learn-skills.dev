Service de paiement d'agent via CrowPay : donne à votre agent un portefeuille pour payer les API et les services. Utilisez cette compétence chaque fois que l'agent doit gérer les réponses HTTP 402 Payment Required, payer une API, acheter des crédits ou des abonnements, payer un commerçant avec une carte de crédit, configurer un portefeuille d'agent ou gérer des règles de dépenses. À utiliser également lorsque l'utilisateur mentionne CrowPay, Crow, les paiements d'agent, USDC, x402 ou achète quoi que ce soit sur Internet pour l'agent, même s'il ne dit pas « Crow » par son nom.
