Служба оплаты агентов через CrowPay — предоставляет вашему агенту кошелек для оплаты API и услуг. Используйте этот навык всякий раз, когда агенту необходимо обрабатывать ответы HTTP 402 «Требуется оплата», платить за API, покупать кредиты или подписки, платить продавцу с помощью кредитной карты, настраивать кошелек агента или управлять правилами расходов. Также используйте, когда пользователь упоминает CrowPay, Crow, агентские платежи, USDC, x402 или покупает что-либо в Интернете для агента, даже если он не произносит «Crow» по имени.
