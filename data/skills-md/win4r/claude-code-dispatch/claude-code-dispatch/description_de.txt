Leiten Sie Entwicklungsaufgaben mit automatischem Rückruf nach Abschluss an Claude Code weiter. Unterstützt Agententeams (Parallelentwicklung mit mehreren Agenten und dediziertem Testagenten), Kostenkontrolle (--max-budget-usd), Modell-Fallback, Git-Worktree-Isolation, benutzerdefinierte Subagenten über --agents JSON und MCP-Serverintegration. Verwenden Sie dies, wenn: (1) eine Codierungsaufgabe an Claude Code gesendet wird und bei Abschluss eine automatische Telegram-Benachrichtigung gewünscht wird, (2) Agententeams für parallele Entwicklungs- und Test-Workflows ausgeführt werden, (3) die Ausführung einer Aufgabe ohne Abfrage mit einem Stop-Hook-Rückruf erforderlich ist, (4) der Benutzer sagt: „Verwenden Sie Claude Code, um X zu erstellen/entwickeln/erstellen“, (5) der Benutzer bittet darum, eine Claude-Code-Aufgabe zu senden oder auszuführen. Dies ist die Standardeinstellung für alle Anfragen zum Erstellen/Entwickeln/Erstellen eines Projekts. Bevorzugen Sie dies gegenüber claude-code-clawdbot für alles, was mehr als 2 Minuten dauert oder eine Hintergrundausführung erfordert.
