Invia attività di sviluppo a Claude Code con richiamata automatica al completamento. Supporta team di agenti (sviluppo parallelo multi-agente con agente di test dedicato), controlli dei costi (--max-budget-usd), fallback del modello, isolamento git worktree, agenti secondari personalizzati tramite --agents JSON e integrazione del server MCP. Da utilizzare quando: (1) si invia un'attività di codifica a Claude Code e si desidera una notifica automatica di Telegram al completamento, (2) si eseguono team di agenti per flussi di lavoro paralleli di sviluppo+test, (3) si necessita l'esecuzione di attività con polling zero con richiamata Stop Hook, (4) l'utente dice "usa Claude Code per costruire/sviluppare/creare X", (5) l'utente chiede di inviare o eseguire un'attività di Claude Code. QUESTO È IL PREDEFINITO per qualsiasi richiesta di "costruisci/sviluppa/crea un progetto": preferisci questo a claude-code-clawdbot per tutto ciò che richiede più di 2 minuti o necessita di esecuzione in background.
