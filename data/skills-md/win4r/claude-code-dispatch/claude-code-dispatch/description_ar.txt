إرسال مهام التطوير إلى Claude Code مع رد الاتصال التلقائي عند الانتهاء. يدعم فرق الوكلاء (تطوير متوازي متعدد الوكلاء مع وكيل اختبار مخصص)، وضوابط التكلفة (--max-budget-usd)، والنموذج الاحتياطي، وعزل شجرة عمل git، والوكلاء الفرعيين المخصصين عبر --agents JSON، وتكامل خادم MCP. يُستخدم عندما: (1) إرسال مهمة برمجة إلى Claude Code والرغبة في إشعار Telegram تلقائيًا عند الانتهاء، (2) تشغيل Agent Teams لسير عمل التطوير + الاختبار المتوازي، (3) الحاجة إلى تنفيذ مهمة بدون استقصاء مع رد اتصال Stop Hook، (4) يقول المستخدم "استخدم Claude Code لبناء/تطوير/إنشاء X"، (5) يطلب المستخدم إرسال أو تشغيل مهمة Claude Code. هذا هو الخيار الافتراضي لأي طلب "إنشاء/تطوير/إنشاء مشروع" - تفضل هذا على claude-code-clawdbot لأي شيء يستغرق أكثر من دقيقتين أو يحتاج إلى تنفيذ في الخلفية.
