Отправляйте задачи разработки в Claude Code с автоматическим обратным вызовом по завершении. Поддерживает команды агентов (многоагентная параллельная разработка с выделенным агентом тестирования), контроль затрат (--max-budget-usd), резервную модель, изоляцию рабочего дерева git, настраиваемые подагенты через --agents JSON и интеграцию с сервером MCP. Используйте, когда: (1) отправка задачи кодирования в Claude Code и требуется автоматическое уведомление Telegram о завершении, (2) запуск команд агентов для параллельных рабочих процессов разработки и тестирования, (3) необходимость выполнения задачи с нулевым опросом с обратным вызовом Stop Hook, (4) пользователь говорит: «Используйте Claude Code для сборки/разработки/создания X», (5) пользователь просит отправить или запустить задачу Claude Code. ЭТО УСТАНОВКА ПО УМОЛЧАНИЮ для любого запроса «создать/разработать/создать проект» — предпочтите его, а не claude-code-clawdbot для всего, что занимает >2 минуты или требует выполнения в фоновом режиме.
