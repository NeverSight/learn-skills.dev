Envoyez les tâches de développement à Claude Code avec rappel automatique à la fin. Prend en charge les équipes d'agents (développement parallèle multi-agents avec agent de test dédié), les contrôles de coûts (--max-budget-usd), le modèle de secours, l'isolation de l'arbre de travail git, les sous-agents personnalisés via --agents JSON et l'intégration du serveur MCP. À utiliser lorsque : (1) vous envoyez une tâche de codage à Claude Code et souhaitez une notification Telegram automatique à la fin, (2) exécutez des équipes d'agents pour des workflows de développement et de test parallèles, (3) avez besoin d'une exécution de tâche sans interrogation avec rappel Stop Hook, (4) l'utilisateur dit « utiliser Claude Code pour construire/développer/créer X », (5) l'utilisateur demande de distribuer ou d'exécuter une tâche Claude Code. C'EST LA VALEUR PAR DÉFAUT pour toute requête « construire/développer/créer un projet » – préférez-la à claude-code-clawdbot pour tout ce qui prend plus de 2 minutes ou nécessite une exécution en arrière-plan.
