완료 시 자동 콜백을 사용하여 개발 작업을 Claude Code에 전달합니다. 에이전트 팀(전용 테스트 에이전트가 있는 다중 에이전트 병렬 개발), 비용 제어(--max-budget-usd), 모델 대체, git 작업 트리 격리, --agents JSON을 통한 사용자 정의 하위 에이전트 및 MCP 서버 통합을 지원합니다. 다음의 경우에 사용하십시오: (1) 코딩 작업을 Claude Code에 디스패치하고 완료 시 자동 텔레그램 알림을 원할 때, (2) 병렬 개발+테스트 워크플로를 위해 Agent Teams를 실행할 때, (3) Stop Hook 콜백을 통해 제로 폴링 작업 실행이 필요할 때, (4) 사용자가 'Claude Code를 사용하여 X를 구축/개발/생성하세요'라고 말할 때, (5) 사용자가 Claude Code 작업을 디스패치하거나 실행할 것을 요청하는 경우. 이는 모든 '프로젝트 빌드/개발/생성' 요청에 대한 기본값입니다. 2분 이상 걸리거나 백그라운드 실행이 필요한 모든 작업에는 claude-code-clawdbot보다 이 옵션을 선호합니다.
