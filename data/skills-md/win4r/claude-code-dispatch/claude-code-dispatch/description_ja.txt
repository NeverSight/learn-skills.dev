完了時に自動コールバックを使用して、開発タスクを Claude Code にディスパッチします。エージェント チーム (専用のテスト エージェントを使用したマルチエージェントの並列開発)、コスト管理 (--max-budget-usd)、モデル フォールバック、git ワークツリーの分離、--agents JSON を介したカスタム サブエージェント、および MCP サーバーの統合をサポートします。次の場合に使用します: (1) コーディング タスクを Claude Code にディスパッチし、完了時に自動 Telegram 通知が必要な場合、(2) 並行開発 + テスト ワークフローのためにエージェント チームを実行する場合、(3) Stop Hook コールバックによるゼロポーリング タスクの実行が必要な場合、(4) ユーザーが「Claude Code を使用して X を構築/開発/作成する」と指示した場合、(5) ユーザーが Claude Code タスクをディスパッチまたは実行するように要求した場合。これは、「プロジェクトのビルド/開発/作成」リクエストのデフォルトです。2 分以上かかる場合やバックグラウンドでの実行が必要な場合は、claude-code-clawdbot よりもこれを優先します。
