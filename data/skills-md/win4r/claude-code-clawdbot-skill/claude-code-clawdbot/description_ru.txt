Запустите Claude Code (Anthropic) с этого хоста через Claude CLI (Agent SDK) в безголовом режиме (-p) для анализа кодовой базы, рефакторинга, исправления тестов и структурированного вывода. Используйте, когда пользователь запрашивает использование кода Claude, запускает `claude -p`, использует режим планирования, автоматически утверждает инструменты с помощью --allowedTools, генерирует выходные данные JSON или интегрирует код Claude в рабочие процессы/cron Clawdbot.
