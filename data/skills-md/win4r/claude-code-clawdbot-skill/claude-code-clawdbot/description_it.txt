Esegui Claude Code (Anthropic) da questo host tramite la CLI `claude` (Agent SDK) in modalità headless (`-p`) per l'analisi della base di codice, i refactoring, la correzione dei test e l'output strutturato. Da utilizzare quando l'utente richiede di utilizzare Claude Code, eseguire `claude -p`, utilizzare la modalità Plan, approvare automaticamente gli strumenti con --allowedTools, generare output JSON o integrare Claude Code nei flussi di lavoro/cron di Clawdbot.
