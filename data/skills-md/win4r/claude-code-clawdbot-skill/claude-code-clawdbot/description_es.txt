Ejecute Claude Code (Anthropic) desde este host a través de la CLI `claude` (Agent SDK) en modo sin cabeza (`-p`) para análisis de base de código, refactorizaciones, corrección de pruebas y salida estructurada. Úselo cuando el usuario solicite usar Claude Code, ejecutar `claude -p`, usar el modo Plan, aprobar automáticamente herramientas con --allowedTools, generar resultados JSON o integrar Claude Code en los flujos de trabajo/cron de Clawdbot.
