Exécutez Claude Code (Anthropic) à partir de cet hôte via la CLI `claude` (Agent SDK) en mode sans tête (`-p`) pour l'analyse de la base de code, les refactors, la correction des tests et la sortie structurée. À utiliser lorsque l'utilisateur demande à utiliser Claude Code, à exécuter `claude -p`, à utiliser le mode Plan, à approuver automatiquement les outils avec --allowedTools, à générer une sortie JSON ou à intégrer Claude Code dans les workflows/cron de Clawdbot.
