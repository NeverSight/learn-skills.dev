قم بتشغيل Claude Code (Anthropic) من هذا المضيف عبر `claude` CLI (Agent SDK) في الوضع بدون رأس (`-p`) لتحليل قاعدة التعليمات البرمجية، وإعادة البناء، وإصلاح الاختبار، والإخراج المنظم. يُستخدم عندما يطلب المستخدم استخدام Claude Code، أو تشغيل `clude -p`، أو استخدام وضع الخطة، أو أدوات الموافقة التلقائية باستخدام --allowedTools، أو إنشاء مخرجات JSON، أو دمج Claude Code في سير عمل Clawdbot/cron.
