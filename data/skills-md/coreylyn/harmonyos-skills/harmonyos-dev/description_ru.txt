Помощник по разработке приложений HarmonyOS на основе фреймворков ArkTS и ArkUI. Перед запуском подтвердите версию API (API по умолчанию 20+) и тип целевого устройства (ПК по умолчанию). Поддерживает разработку компонентов пользовательского интерфейса, управление данными, сетевую связь и интеграцию системных возможностей. Применимо для разработки новых проектов, повторения существующих проектов и проектирования архитектуры. Ссылки на официальную документацию Huawei, включая примечания к выпуску, руководства по разработке, ссылки на API, лучшие практики и часто задаваемые вопросы.
