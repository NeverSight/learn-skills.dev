Навыки проверки кода HarmonyOS для проверки проектов ArkTS на соответствие официальным рекомендациям разработки Huawei и передовым практикам безопасности. Используйте при проверке приложений HarmonyOS на предмет: (1) соответствия требованиям безопасности (жестко закодированные учетные данные, шифрование, проверка ввода), (2) языковых стандартов ArkTS (использование hilog, безопасность типов, магические числа), (3) управления жизненным циклом компонентов (очистка ресурсов, обработка подписки на события), (4) управления состоянием (согласованность декораторов V1/V2), (5) операций с базой данных (обработка ResultSet, управление транзакциями, шифрование), (6) управления разрешениями (официальные шаблоны разрешений), (7) проблем с производительностью (асинхронность). forEach, утечки ресурсов), (8) совместимость версий API, (9) лучшие практики использования комплекта. Создает подробные отчеты об уценках с рекомендациями по приоритетным исправлениям.
