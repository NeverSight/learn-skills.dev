Fournit un guide de développement complet pour le moteur de jeu Cocos Creator 3.8, couvrant le système de composants (_decorator, Component), les rappels du cycle de vie, le système d'événements (EventTarget, input), la gestion des ressources, le système d'interpolation, les pools d'objets, le système d'interface utilisateur, les collisions physiques et l'optimisation des publicités jouables. Déclenché lorsque les utilisateurs écrivent ou refactorisent le code TypeScript Cocos Creator 3.x, implémentent des fonctionnalités de jeu, gèrent le chargement et la publication des ressources, optimisent les performances/taille du package, révisent les modifications de code ou mettent en place une architecture de projet publicitaire jouable. Applicable également lorsque les utilisateurs mentionnent des importations depuis « cc », composant, nœud, ressources, interpolation, directeur et autres API 3.x.
