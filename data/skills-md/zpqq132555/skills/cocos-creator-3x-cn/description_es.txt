Guía de desarrollo integral para el motor de juego Cocos Creator 3.8, que cubre el sistema de componentes (_decorator, Component), devoluciones de llamadas del ciclo de vida, sistema de eventos (EventTarget, entrada), gestión de activos de recursos, sistema de animación de interpolación, grupos de objetos, sistema de interfaz de usuario, física y colisión, y optimización de anuncios reproducibles. Se activa cuando los usuarios escriben o refactorizan código TypeScript de Cocos Creator 3.x, implementan funciones del juego, manejan la carga y liberación de activos, optimizan el rendimiento/tamaño del paquete, revisan cambios de código o configuran una arquitectura de proyecto publicitario reproducible. También se aplica cuando los usuarios mencionan la importación desde 'cc', componente, nodo, recursos, interpolación, director y otras API 3.x.
