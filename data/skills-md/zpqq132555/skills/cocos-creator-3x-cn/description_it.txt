Fornisce una guida completa allo sviluppo per il motore di gioco Cocos Creator 3.8, coprendo il sistema di componenti (_decorator, Component), callback del ciclo di vita, sistema di eventi (EventTarget, input), gestione delle risorse, sistema di animazione interpolata, pool di oggetti, sistema di interfaccia utente, collisioni fisiche e ottimizzazione degli annunci riproducibili. Si attiva quando gli utenti scrivono o effettuano il refactoring del codice TypeScript di Cocos Creator 3.x, implementano funzionalità di gioco, gestiscono il caricamento e il rilascio delle risorse, ottimizzano le prestazioni/dimensioni del pacchetto, esaminano le modifiche del codice o impostano l'architettura del progetto di annunci riproducibili. Applicabile anche quando gli utenti menzionano l'importazione da "cc", componente, nodo, risorse, interpolazione, direttore e altre API 3.x.
