Помощник по созданию изображений Nano Banana. Оптимизирует запросы на китайском языке на английский язык и вызывает API Gemini для создания изображений. Активируйте только тогда, когда пользователь явно упоминает нанобанан/нанобанана или использует команду/нанобанана. НЕ активируйте общие фразы для создания изображений, такие как «生成图片» или «画一个». Типичные триггеры: команда /nanobanana, «用 нанобанан 画», «нанобанан 生图», «нанобанан 生成», «нанобанан 优化提示词».
