Démarrez un projet complet axé d'abord sur l'agent OU modernisez un projet existant avec le framework Harness — AGENTS.md, CLAUDE.md, plans d'exécution, PRD et scripts d'automatisation, tous basés sur les principes d'ingénierie Harness. Utilisez cette compétence chaque fois que l'utilisateur souhaite démarrer un nouveau projet, initialiser une base de code, créer un échafaudage de projet, configurer un monorepo ou dire quelque chose comme « nouveau projet », « bootstrap », « init », « échafaudage », « démarrer la construction », « configuration du projet », « créer une application », « configurer un dépôt ». Déclenchez AUSSI lorsque l'utilisateur souhaite ajouter un harnais à un projet existant, dit "rénovation", "ajouter AGENTS.md à mon projet", "configurer un harnais pour mon dépôt", "convertir mon projet", ou mentionne AGENTS.md, ingénierie de harnais, développement axé sur l'agent. Même si l'utilisateur décrit simplement une idée et souhaite « se lancer », c'est la compétence à utiliser. Utilisez-le également lorsque l'utilisateur souhaite qu'une introduction au projet ou un contenu de documentation prêt pour GitBook soit produit parallèlement au travail réel du projet.
