Starten Sie ein komplettes Agent-First-Projekt ODER rüsten Sie ein bestehendes Projekt mit dem Harness-Framework nach – AGENTS.md, CLAUDE.md, Ausführungspläne, PRD und Automatisierungsskripte, alle basierend auf Harness Engineering-Prinzipien. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer ein neues Projekt starten, eine Codebasis initialisieren, ein Projektgerüst erstellen, ein Monorepo einrichten oder etwas sagen möchte wie „Neues Projekt“, „Bootstrap“, „Init“, „Gerüst“, „Bau beginnen“, „Projekteinrichtung“, „App erstellen“, „Repo einrichten“. Wird AUCH ausgelöst, wenn der Benutzer einen Kabelbaum zu einem bestehenden Projekt hinzufügen möchte, „Nachrüsten“, „AGENTS.md zu meinem Projekt hinzufügen“, „Kabelbaum für mein Repo einrichten“, „Mein Projekt konvertieren“ sagt oder AGENTS.md, Kabelbaum-Engineering, Agent-First-Entwicklung erwähnt. Auch wenn der Benutzer nur eine Idee beschreibt und „starten“ möchte, ist dies die Fähigkeit, die es zu nutzen gilt. Verwenden Sie es auch, wenn der Benutzer neben der eigentlichen Projektarbeit eine GitBook-fähige Projekteinführung oder Dokumentationsinhalte erstellen möchte.
