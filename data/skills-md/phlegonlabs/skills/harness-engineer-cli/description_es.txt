Arranque un proyecto completo en el que el agente primero O modernice un proyecto existente con el marco de arnés: AGENTS.md, CLAUDE.md, planes de ejecución, PRD y scripts de automatización, todos basados ​​en los principios de Harness Engineering. Utilice esta habilidad siempre que el usuario quiera iniciar un nuevo proyecto, inicializar una base de código, crear un andamio de proyecto, configurar un monorepo o decir algo como "nuevo proyecto", "bootstrap", "init", "scaffold", "comenzar a construir", "configurar proyecto", "crear una aplicación", "configurar un repositorio". TAMBIÉN se activa cuando el usuario desea agregar un arnés a un proyecto existente, dice "adaptarse", "agregar AGENTS.md a mi proyecto", "configurar un arnés para mi repositorio", "convertir mi proyecto" o menciona AGENTS.md, ingeniería de arnés, desarrollo de agente primero. Incluso si el usuario simplemente describe una idea y quiere "empezar", esta es la habilidad que debe utilizar. Úselo también cuando el usuario desee una introducción al proyecto lista para GitBook o contenido de documentación producido junto con el trabajo real del proyecto.
