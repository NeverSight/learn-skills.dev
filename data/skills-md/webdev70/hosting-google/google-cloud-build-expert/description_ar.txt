معرفة الخبراء بخطوط أنابيب Google Cloud Build CI/CD بما في ذلك بناء جملة cloudbuild.yaml وخطوات البناء والمنشئين ومتغيرات الاستبدال والمشغلات والأسرار ومعالجة العناصر والنشر إلى Cloud Run. يُستخدم عند العمل مع تكوينات Cloud Build، أو استكشاف أخطاء مسارات البناء وإصلاحها، أو النشر إلى Google Cloud Platform.
