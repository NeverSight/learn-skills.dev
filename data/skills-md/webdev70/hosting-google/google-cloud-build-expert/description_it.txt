Conoscenza approfondita delle pipeline CI/CD di Google Cloud Build, tra cui sintassi cloudbuild.yaml, passaggi di creazione, builder, variabili di sostituzione, trigger, segreti, gestione degli artefatti e distribuzione su Cloud Run. Da utilizzare quando si lavora con le configurazioni Cloud Build, si risolvono i problemi delle pipeline di build o si esegue la distribuzione su Google Cloud Platform.
