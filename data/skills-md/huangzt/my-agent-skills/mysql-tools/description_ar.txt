مجموعة أدوات قاعدة بيانات MySQL للاتصال بمثيلات قاعدة البيانات، وسرد جميع الجداول، وعرض هياكل الجدول، وتنفيذ استعلامات SQL، والعمليات الأخرى. استخدم هذه المهارة عندما تحتاج إلى تشغيل قواعد بيانات MySQL أو الاستعلام عن البيانات أو تحليل هياكل الجدول. يدعم تحديد معلومات الاتصال مثل المضيف والمنفذ والمستخدم وكلمة المرور وقاعدة البيانات عبر معلمات سطر الأوامر، ويمكن أن يحل محل خدمة mysql mcp. متوافق مع أنظمة التشغيل Windows وmacOS وLinux.
