Kit de herramientas de base de datos MySQL para conectarse a instancias de bases de datos, enumerar todas las tablas, ver estructuras de tablas, ejecutar consultas SQL y más. Utilice esta habilidad cuando necesite operar bases de datos MySQL, consultar datos o analizar estructuras de tablas. Admite la especificación de información de conexión, como host, puerto, usuario, contraseña y base de datos mediante parámetros de línea de comandos, y puede reemplazar el servicio mysql mcp. Compatible con plataformas Windows, macOS y Linux.
