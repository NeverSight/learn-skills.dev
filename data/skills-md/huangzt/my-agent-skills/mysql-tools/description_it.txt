Toolkit di database MySQL per la connessione alle istanze del database, l'elenco di tutte le tabelle, la visualizzazione delle strutture delle tabelle, l'esecuzione di query SQL e altro ancora. Utilizza questa competenza quando devi operare su un database MySQL, interrogare dati o analizzare strutture di tabelle. Supporta la specifica delle informazioni di connessione come host, porta, utente, password e database tramite parametri della riga di comando e può sostituire il servizio mysql mcp. Compatibile con piattaforme Windows, macOS e Linux.
