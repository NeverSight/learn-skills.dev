Набор инструментов базы данных MySQL для подключения к экземплярам базы данных, составления списка всех таблиц, просмотра структур таблиц, выполнения запросов SQL и многого другого. Используйте этот навык, когда вам нужно работать с базами данных MySQL, запрашивать данные или анализировать структуры таблиц. Поддерживает указание информации о соединении, такой как хост, порт, пользователь, пароль и база данных, через параметры командной строки и может заменить службу mysql mcp. Совместимость с платформами Windows, macOS и Linux.
