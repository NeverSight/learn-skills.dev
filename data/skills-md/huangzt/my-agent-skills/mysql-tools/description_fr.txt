Boîte à outils de base de données MySQL pour se connecter aux instances de base de données, répertorier toutes les tables, afficher les structures des tables, exécuter des requêtes SQL et d'autres opérations. Utilisez cette compétence lorsque vous devez exploiter des bases de données MySQL, interroger des données ou analyser des structures de tables. Prend en charge la spécification des informations de connexion telles que l'hôte, le port, l'utilisateur, le mot de passe et la base de données via des paramètres de ligne de commande, et peut remplacer le service mysql mcp. Compatible avec les plateformes Windows, macOS et Linux.
