Lösen Sie PR-Rezensionskommentare automatisch auf, indem Sie ungelöste Kommentare abrufen, Korrekturen anwenden, pushen und wiederholen, bis alle Kommentare gelöst sind. Funktioniert mit jeder Codebasis, mit erweiterter Unterstützung für Laravel- und PHP-Projekte. Extrahiert AI-Fix-Eingabeaufforderungen aus Codespect-io-Bot-Reviews für präzise Fixes. Wird verwendet, wenn: PR auflösen, PR-Kommentare korrigieren, PR-Resolver, Bewertungskommentare korrigieren, PR-Feedback ansprechen, Überprüfung auflösen, Laravel PR korrigieren, PHP-Überprüfung durchführen.
