Se utiliza al crear una nueva habilidad o modificar una existente en el repositorio de DaleStudy/skills. Se activa en las siguientes situaciones: (1) cuando se solicita crear una nueva habilidad, (2) cuando se escribe o edita el archivo SKILL.md, (3) cuando se hacen preguntas sobre la estructura de la habilidad o el frontmatter, (4) cuando se trabaja con palabras clave como 'skill', 'SKILL.md' o 'frontmatter'.
