Utilizzato quando si crea una nuova abilità o si modifica una esistente nel repository DaleStudy/skills. Attivato nelle seguenti situazioni: (1) quando si richiede di creare una nuova abilità, (2) quando si scrive o si modifica il file SKILL.md, (3) quando si pongono domande sulla struttura delle abilità o sul frontmatter, (4) quando si lavora con parole chiave come "skill", "SKILL.md" o "frontmatter".
