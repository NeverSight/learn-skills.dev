Навыки создания рабочих процессов GitHub Actions, обеспечения безопасности и управления версиями. Используйте в следующих ситуациях: (1) при создании нового файла рабочего процесса (.yml), (2) при изменении существующего рабочего процесса, (3) при просмотре или обновлении версий действий, (4) при аудите безопасности CI/CD и (5) при работе с ключевыми словами «actions/», «uses:», «workflow» и «.github/workflows».
