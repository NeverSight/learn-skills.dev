Guía de GitHub CLI (gh) para interactuar con la plataforma GitHub. Úselo en las siguientes situaciones: (1) al crear, ver o modificar problemas de GitHub; (2) al crear, revisar o fusionar solicitudes de extracción; (3) al crear y administrar versiones de GitHub; (4) al gestionar proyectos, incluidas etiquetas e hitos; (5) al ejecutar flujos de trabajo de GitHub Actions y verificar los resultados; (6) cuando se trabaja con palabras clave como 'gh', 'problema', 'solicitud de extracción', 'PR', 'lanzamiento', 'etiqueta', 'flujo de trabajo', 'ejecutar'
