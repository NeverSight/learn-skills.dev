GitHub CLI (gh) Руководство по взаимодействию с платформой GitHub. Используйте в следующих ситуациях: (1) при создании, просмотре или изменении задач GitHub; (2) при создании, просмотре или объединении запросов на включение; (3) при создании выпусков GitHub и управлении ими; (4) при управлении проектами, включая метки и этапы; (5) при запуске рабочих процессов GitHub Actions и проверке результатов; (6) при работе с такими ключевыми словами, как «gh», «проблема», «запрос на включение», «PR», «релиз», «ярлык», «рабочий процесс», «запуск».
