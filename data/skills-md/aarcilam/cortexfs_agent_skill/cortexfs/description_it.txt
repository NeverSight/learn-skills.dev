CortexFS Memory Skill: livello di memoria cognitiva persistente supportato dalla CLI CortexFS. Consente di salvare, leggere, ricercare conoscenze e stati di sessione, nell'ambito del contesto del progetto. Include l'installazione CLI automatica e comandi per OpenCode e Cursor. Tutti i comandi sono BASH/CLI che devono essere eseguiti sul TERMINAL/SHELL.
