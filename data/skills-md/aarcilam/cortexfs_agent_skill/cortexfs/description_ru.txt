CortexFS Memory Skill — постоянный уровень когнитивной памяти, поддерживаемый интерфейсом командной строки CortexFS. Позволяет сохранять, читать, искать знания и состояния сеанса в зависимости от контекста проекта. Включает автоматическую установку CLI и команды для OpenCode и Cursor. Все команды из BASH/CLI, которые должны быть запущены в TERMINAL/SHELL.
