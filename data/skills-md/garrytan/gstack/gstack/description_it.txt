Navigazione web veloce per Claude Code tramite il demone Chromium headless persistente. Passare a qualsiasi URL,
leggere il contenuto della pagina, fare clic sugli elementi, compilare moduli, eseguire JavaScript, acquisire screenshot,
ispeziona CSS/DOM, acquisisci log di console/rete e altro ancora. ~100ms per comando dopo
prima chiamata. Da utilizzare quando è necessario controllare un sito Web, verificare una distribuzione, leggere documenti,
o interagire con qualsiasi pagina web. Nessun MCP, nessuna estensione Chrome: solo CLI veloce.
