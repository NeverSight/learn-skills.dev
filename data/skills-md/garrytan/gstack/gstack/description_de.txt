Schnelles Surfen im Internet für Claude Code über einen persistenten, kopflosen Chromium-Daemon. Navigieren Sie zu einer beliebigen URL,
Seiteninhalte lesen, Elemente anklicken, Formulare ausfüllen, JavaScript ausführen, Screenshots machen,
Überprüfen Sie CSS/DOM, erfassen Sie Konsolen-/Netzwerkprotokolle und mehr. ~100 ms pro Befehl danach
Erster Anruf. Verwenden Sie diese Option, wenn Sie eine Website überprüfen, eine Bereitstellung überprüfen, Dokumente lesen müssen.
oder mit einer beliebigen Webseite interagieren. Kein MCP, keine Chrome-Erweiterung – nur schnelles CLI.
