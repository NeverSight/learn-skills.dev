Быстрый просмотр веб-страниц для Claude Code с помощью постоянного безголового демона Chromium. Перейдите к любому URL-адресу,
читать содержимое страницы, щелкать элементы, заполнять формы, запускать JavaScript, делать снимки экрана,
проверять CSS/DOM, собирать журналы консоли/сети и многое другое. ~100 мс на команду после
первый звонок. Используйте, когда вам нужно проверить веб-сайт, проверить развертывание, прочитать документацию,
или взаимодействовать с любой веб-страницей. Никакого MCP, никакого расширения Chrome — только быстрый CLI.
