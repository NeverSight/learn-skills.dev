MindOS Knowledge Base Chinese Operation Guide for Agent tasks on local markdown/csv knowledge bases.  
Whenever the task involves note files, SOP/workflow documents, Profile/context documents, CSV tables, knowledge-base organization, cross-Agent handoff or decision synchronization, and is executed via MindOS MCP tools, it should be proactively triggered.  
Typical requests include “update notes,” “search knowledge base,” “organize files,” “execute SOP,” “review per team standards,” “hand task to another Agent,” “sync decisions,” “append CSV,” “review this conversation,” “extract key lessons,” “adaptively update review results into the corresponding document,” “route this information to the relevant file,” “sync-update all related documents,” etc.;  
Trigger even if the user does not explicitly mention MindOS.
