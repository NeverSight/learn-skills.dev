Guide d'utilisation en chinois de la base de connaissances MindOS pour les tâches d'agent sur les bases de connaissances locales markdown/csv.  
Chaque fois que la tâche implique des fichiers de notes, des documents SOP/workflow, des documents de profil/contexte, des tables CSV, une organisation de base de connaissances, un transfert entre agents ou une synchronisation de décision, et qu'elle est exécutée via les outils MindOS MCP, elle doit être déclenchée de manière proactive.  
Les requêtes typiques incluent « mettre à jour les notes », « rechercher dans la base de connaissances », « organiser les fichiers », « exécuter les SOP », « réviser selon les normes de l'équipe », « confier la tâche à un autre agent », « synchroniser les décisions », « ajouter le CSV », « réviser cette conversation », « extraire les leçons clés », « mettre à jour de manière adaptative les résultats de l'examen dans le document correspondant », « acheminer ces informations vers le fichier approprié », « synchroniser et mettre à jour tous les documents associés », etc. ;  
Déclenchez même si l'utilisateur ne mentionne pas explicitement MindOS.
