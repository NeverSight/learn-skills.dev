Test des composants, des hooks et des interactions de React avec la bibliothèque de tests React. À utiliser lors de l'écriture de nouveaux tests, du débogage de tests ayant échoué, de la refactorisation du code de test, de la configuration de l'infrastructure de test, de la simulation d'API dans les tests React ou lorsque l'utilisateur mentionne Jest/Vitest avec React, `render`, `screen`, `userEvent`, `waitFor`, `findBy`, MSW ou des modèles de test. À utiliser également lorsque l'utilisateur demande comment tester un composant ou un hook React spécifique, même s'il ne mentionne pas explicitement « bibliothèque de tests ».
