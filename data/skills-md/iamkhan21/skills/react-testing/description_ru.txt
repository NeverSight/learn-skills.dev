Тестирование компонентов React, перехватчиков и взаимодействий с помощью библиотеки тестирования React. Используйте при написании новых тестов, отладке неудачных тестов, рефакторинге тестового кода, настройке тестовой инфраструктуры, имитации API в тестах React или когда пользователь упоминает Jest/Vitest с React, `render`, `screen`, `userEvent`, `waitFor`, `findBy`, MSW или шаблоны тестирования. Также используйте, когда пользователь спрашивает, как протестировать конкретный компонент или хук React, даже если он явно не упоминает «библиотеку тестирования».
