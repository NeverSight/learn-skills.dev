React コンポーネント、フック、および React Testing Library との相互作用をテストします。新しいテストの作成、失敗したテストのデバッグ、テスト コードのリファクタリング、テスト インフラストラクチャのセットアップ、React テストでの API のモック、またはユーザーが React、`render`、`screen`、`userEvent`、`waitFor`、`findBy`、MSW、またはテスト パターンで Jest/Vitest について言及したときに使用します。また、「ライブラリのテスト」について明示的に言及していない場合でも、ユーザーが特定の React コンポーネントまたはフックをテストする方法を尋ねた場合にも使用します。
