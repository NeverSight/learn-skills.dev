اختبار مكونات React والخطافات والتفاعلات مع مكتبة اختبار React. يُستخدم عند كتابة اختبارات جديدة، أو تصحيح أخطاء الاختبارات الفاشلة، أو إعادة بناء كود الاختبار، أو إعداد البنية التحتية للاختبار، أو السخرية من واجهات برمجة التطبيقات في اختبارات React، أو عندما يذكر المستخدم Jest/Vitest مع React، أو `render`، أو `screen`، أو `userEvent`، أو `waitFor`، أو `findBy`، أو MSW، أو أنماط الاختبار. يُستخدم أيضًا عندما يسأل المستخدم عن كيفية اختبار مكون أو رابط React محدد، حتى لو لم يذكر "اختبار المكتبة" بشكل صريح.
