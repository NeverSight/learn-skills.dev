Estandariza una aplicación web TypeScript frontal ya estructurada con herramientas, estructura y convenciones básicas. Úselo al configurar o alinear pnpm + TypeScript estricto + alias de ruta, Vitest, Oxc (oxlint/oxfmt) y un diseño de carpeta de estilo FSD progresivo, o cuando el usuario pregunta sobre las reglas iniciales del proyecto y las opciones de la biblioteca de UI.
