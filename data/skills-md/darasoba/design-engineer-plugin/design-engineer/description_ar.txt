قم ببناء واجهات باستخدام حرفة هندسة التصميم - تقاطع التصميم والتعليمات البرمجية. استخدم هذه المهارة عندما يطلب المستخدم إنشاء مكونات واجهة المستخدم أو العناصر التفاعلية أو الرسوم المتحركة أو التفاعلات الدقيقة أو تجارب الويب المصقولة. مشغلات عند الإشارة إلى "مهندس التصميم"، أو "تصميم التفاعل"، أو "التفاعلات الدقيقة"، أو "الحرفية"، أو "التلميع"، أو "البهجة"، أو "الرسوم المتحركة"، أو "تصميم الحركة"، أو "اللمس"، أو "الخبز المحمص"، أو "لوحة الأوامر"، أو "مكونات نظام التصميم"، أو طلبات الواجهات التي تبدو حية وسريعة الاستجابة ومفصلة بدقة. يجب أيضًا استخدام هذه المهارة عندما يريد المستخدم إضافة تفاصيل غير مرئية، أو تحسين التفاعلات، أو جعل شيء ما "يبدو على ما يرام". يكمل مهارة تصميم الواجهة الأمامية من خلال التركيز بشكل خاص على جودة التفاعل والحركة والحرف الهندسية.
