Cree interfaces con la ingeniería de diseño: la intersección del diseño y el código. Utilice esta habilidad cuando el usuario solicite crear componentes de interfaz de usuario, elementos interactivos, animaciones, microinteracciones o experiencias web pulidas. Se desencadenan menciones de "ingeniero de diseño", "diseño de interacción", "microinteracciones", "artesanía", "pulido", "deleite", "animación", "diseño de movimiento", "háptica", "brindis", "paleta de comandos", "diseño de componentes del sistema" o solicitudes de interfaces que se sientan vivas, receptivas y meticulosamente detalladas. Esta habilidad también debe usarse cuando el usuario desea agregar detalles invisibles, ajustar las interacciones o hacer que algo "se sienta bien". Complementa la habilidad de diseño frontend enfocándose específicamente en la calidad de la interacción, el movimiento y la ingeniería.
