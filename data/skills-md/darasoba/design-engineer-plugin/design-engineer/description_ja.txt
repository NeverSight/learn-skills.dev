設計とコードの交差点である設計エンジニアリング技術を使用してインターフェイスを構築します。このスキルは、ユーザーが UI コンポーネント、インタラクティブ要素、アニメーション、マイクロインタラクション、または洗練された Web エクスペリエンスを構築するように要求した場合に使用します。 「デザイン エンジニア」、「インタラクション デザイン」、「マイクロ インタラクション」、「クラフト」、「磨き」、「喜び」、「アニメーション」、「モーション デザイン」、「ハプティクス」、「トースト」、「コマンド パレット」、「デザイン システム コンポーネント」などの言及、または生き生きとして応答性が高く、細心の注意を払って作られていると感じられるインターフェイスのリクエストをトリガーします。このスキルは、ユーザーが目に見えない詳細を追加したり、インタラクションを微調整したり、何かを「適切に感じさせる」場合にも使用する必要があります。特にインタラクションの品質、モーション、エンジニアリング技術に重点を置くことで、フロントエンド設計スキルを補完します。
