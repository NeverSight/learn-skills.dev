Orquestador de flujo de trabajo multiagente para proyectos de software. Úselo cuando el usuario diga spidy, implemente usando spidy, planifique con spidy, nuevo proyecto spidy, depure spidy o solicite ejecutar cualquier flujo de trabajo spidy. Úselo también cuando el usuario diga confirmar, enviar, crear pr, flujo de trabajo git, enviarlo o solicite ramificar/confirmar/PR sus cambios. Orquesta agentes especializados para: configuración de nuevos proyectos, planificación de fases, ejecución de fases, verificación, depuración, mapeo de base de código y flujo de trabajo de git.
