Generieren Sie Songs und Musik mit KI (ACE-Step, lokal). Wird verwendet, wenn Benutzer Folgendes wünschen: Singen Sie ein Lied, erstellen Sie Musik, erstellen Sie einen Beat, schreiben und spielen Sie ein Lied, generieren Sie Hintergrundmusik usw. Deckt Anfragen ab wie „Singen Sie ein Lied“, „Schreiben Sie mir ein Lied“, „Singen Sie mir ein Lied“, „Erstellen Sie einen Rap über Codierung“, „Machen Sie ein Lied über Katzen“.
