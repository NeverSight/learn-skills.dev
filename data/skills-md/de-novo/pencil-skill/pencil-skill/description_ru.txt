Создавайте и редактируйте файлы дизайна Pencil (.pen) программным способом. Создавайте проекты пользовательского интерфейса в виде файлов .pen на основе JSON с темами, переменными, компонентами и макетами. Включает инструменты CLI для чтения, поиска, редактирования и проверки файлов .pen без необходимости использования настольного приложения Pencil. Требуется Node.js 18+. Схема версии 2.8.
