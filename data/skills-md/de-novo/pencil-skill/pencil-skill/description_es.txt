Cree y edite archivos de diseño Pencil (.pen) mediante programación. Genere diseños de interfaz de usuario como archivos .pen basados ​​en JSON con temas, variables, componentes y diseños. Incluye herramientas CLI para leer, buscar, editar y validar archivos .pen sin necesidad de la aplicación de escritorio Pencil. Requiere Node.js 18+. Versión del esquema 2.8.
