Nutzen Sie diese Fähigkeit, wenn Sie mit ost-tools arbeiten – einer CLI, die Obsidian-Markdown-Frontmatter anhand von JSON-Schemas für strukturierte Wissensräume validiert und in andere Formate konvertiert. Wird ausgelöst, wenn: (1) ein Bereich nach Inhaltsbearbeitungen validiert wird, (2) eine Schemadatei oder Regeln geschrieben oder aktualisiert werden, (3) ein Bereich konfiguriert oder ein Schema (neu oder vorhanden) entworfen wird, (4) Validierungsfehler behoben werden, (5) „ost-tools“-Befehle ausgeführt werden.
