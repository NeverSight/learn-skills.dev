Utilice esta habilidad cuando trabaje con herramientas ost: una CLI que valida el frontmatter de rebajas de Obsidian frente a esquemas JSON para "espacios" de conocimiento estructurado y convierte a otros formatos. Se activa cuando: (1) valida un espacio después de editar contenido, (2) escribe o actualiza un archivo de esquema o reglas, (3) configura un espacio o diseña un esquema (nuevo o existente), (4) soluciona errores de validación, (5) ejecuta comandos `ost-tools`.
