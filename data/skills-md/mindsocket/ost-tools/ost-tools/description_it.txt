Utilizza questa competenza quando lavori con ost-tools: una CLI che convalida il frontmatter del markdown di Obsidian rispetto agli schemi JSON per "spazi" di conoscenza strutturata e lo converte in altri formati. Si attiva quando: (1) convalida di uno spazio dopo la modifica del contenuto, (2) scrittura o aggiornamento di un file di schema o di regole, (3) configurazione di uno spazio o progettazione di uno schema (nuovo o esistente), (4) risoluzione dei problemi di errori di convalida, (5) esecuzione dei comandi "ost-tools".
