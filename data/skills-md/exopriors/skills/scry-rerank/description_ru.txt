Многоатрибутное переранжирование наборов кандидатов на основе LLM посредством парного сравнения. Поддерживает канонические атрибуты (ясность, техническая глубина, понимание), пользовательские подсказки, выбор уровня модели и конфигурацию TopK. Используйте, когда задача включает в себя: переранжирование, ранжирование по ясности, ранжирование по содержательности, ранжирование по глубине, лучшие элементы, уровень качества, оценку LLM, парное сравнение, ранжирование по множеству атрибутов, повторное ранжирование из SQL или списка. НЕ для: простой сортировки SQL (ORDER BY date/upvotes/score — используйте scry) или семантического поиска и алгебры внедрения (используйте scry-векторы).
