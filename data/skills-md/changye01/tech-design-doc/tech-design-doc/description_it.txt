Strumento per la generazione di documenti di progettazione tecnica. Utilizzato per: (1) progettazione della soluzione prima dello sviluppo di nuove funzionalità (2) generazione di un documento di progettazione completo tramite domande guidate (3) output in formato Markdown. Si attiva quando l'utente dice "scrivi documento di progettazione", "TDD" o "proposta tecnica".
