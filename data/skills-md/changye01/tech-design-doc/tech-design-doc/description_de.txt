Tool zur Erstellung technischer Designdokumente. Wird verwendet für: (1) Planung vor der Entwicklung neuer Funktionen, (2) Generierung eines vollständigen Designdokuments durch geführte Fragen, (3) Ausgabe im Markdown-Format. Wird ausgelöst, wenn der Benutzer „Designdokument schreiben“, „TDD“ oder „Technischer Plan“ sagt.
