Генератор документов технического проектирования. Используется для: (1) проектирования решения перед разработкой новых функций (2) создания полного проектного документа с помощью направляющих вопросов (3) вывода в формате Markdown. Срабатывает, когда пользователь говорит «написать проектную документацию», «TDD» или «техническое решение».
