ワークフロー自動化を構築するための包括的な n8n v2.0 リファレンス。 n8n ワークフローを操作する場合、特に (1) インポート用のワークフロー JSON ファイルの作成、(2) 待機ノードを使用した人間参加型承認ワークフローの実装、(3) サブワークフロー実行ノードの問題のトラブルシューティング、(4) API 資格情報の設定 (LinkedIn、Anthropic、NewsAPI)、(5) サブワークフローによるオーケストレーター パターンの構築、(6) ノードでの式の使用、(7) 理解などの場合にこのスキルを使用します。 MCP サーバーの制限、または (8) n8n ワークフロー開発タスク。
