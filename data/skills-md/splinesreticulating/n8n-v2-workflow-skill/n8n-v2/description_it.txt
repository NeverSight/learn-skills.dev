Riferimento completo n8n v2.0 per la creazione di automazione del flusso di lavoro. Utilizzare questa competenza quando si lavora con flussi di lavoro n8n, in particolare per (1) creare file JSON del flusso di lavoro per l'importazione, (2) implementare flussi di lavoro di approvazione human-in-the-loop con nodi Wait, (3) risolvere problemi relativi al nodo Execute Sub-Workflow, (4) impostare credenziali API (LinkedIn, Anthropic, NewsAPI), (5) creare modelli di orchestrazione con flussi di lavoro secondari, (6) utilizzare espressioni nei nodi, (7) comprendere le limitazioni del server MCP o (8) qualsiasi attività di sviluppo del flusso di lavoro n8n.
