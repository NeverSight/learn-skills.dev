Комплексный справочник по n8n v2.0 для автоматизации рабочих процессов. Используйте этот навык при работе с рабочими процессами n8n, особенно для (1) создания файлов JSON рабочего процесса для импорта, (2) реализации рабочих процессов утверждения человеком в цикле с узлами ожидания, (3) устранения неполадок узлов выполнения подпроцесса рабочего процесса, (4) настройки учетных данных API (LinkedIn, Anthropic, NewsAPI), (5) построения шаблонов оркестратора с подпроцессами рабочих процессов, (6) использования выражений в узлах, (7) понимания ограничений сервера MCP, или (8) любая задача разработки рабочего процесса n8n.
