Planifiez des voyages, vérifiez les départs en temps réel et trouvez des arrêts sur le réseau de transports publics de Sydney (trains, bus, ferries, train léger sur rail, métro). Utilisez cette compétence lorsque l'utilisateur pose des questions sur les trains, les bus, les ferries, le train léger sur rail, le métro, les tarifs Opal, les déplacements domicile-travail, "comment puis-je m'y rendre", le prochain départ, les horaires, le quai ou toute question sur les transports publics de Sydney. Fonctionne sans clés API (fournit des liens Google Maps/TfNSW) mais mieux avec une clé API TfNSW gratuite pour les données en temps réel.
