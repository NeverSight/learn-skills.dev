Управление Nextcloud через OCS API и WebDAV: файлы, общий доступ, пользователи, группы, приложения. Используйте этот навык для ЛЮБЫХ задач, связанных с Nextcloud: загрузка/скачивание файлов, создание папок, управление доступом (общие ресурсы, общедоступные ссылки), администрирование пользователей и групп, управление приложениями. Триггеры: nextcloud, общий доступ к файлам, облачное хранилище, WebDAV, OCS API, общедоступная ссылка на файл, управление пользователями Nextcloud, загрузка файла в облако, загрузка из облака, общий доступ к файлу.
