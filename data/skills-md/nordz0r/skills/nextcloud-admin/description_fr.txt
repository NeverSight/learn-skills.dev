Déclencheurs : nextcloud, partage de fichiers, stockage cloud, WebDAV, API OCS, lien de fichier public, gestion des utilisateurs nextcloud, téléchargement de fichier vers le cloud, téléchargement depuis le cloud, partage de fichier.
