Une référence détaillée en russe sur Open WebUI : architecture, autorisation, fonctions, pipelines, API, RAG, mise à l'échelle, débogage et fonctionnalités cachées. Utilisez cette compétence pour toute question sur Open WebUI : comment il est structuré, comment le déployer, configurer l'authentification (OAuth, LDAP, JWT), écrire une fonction ou un pipeline, connecter un modèle (Ollama, OpenAI), configurer RAG/base de connaissances, passer à la production, déboguer les problèmes. Utilisez-le également lors de l'écriture de code pour Open WebUI : fonctions (filtre, canal, action), pipelines, configurations, docker-compose.
