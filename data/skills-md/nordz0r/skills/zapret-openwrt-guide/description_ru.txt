Полная русскоязычная документация проекта zapret-openwrt: утилита Anti-DPI для роутеров OpenWrt. Используйте этот навык для решения любых вопросов, касающихся zapret-openwrt — архитектуры проекта, конфигурации, стратегий обхода DPI, веб-интерфейса LuCI, сценариев, обновлений, диагностики, перехватчиков custom.d, списков хостов, параметров nfqws, управления службами через procd. Также используйте во время разработки: добавление новых стратегий, изменение пользовательского интерфейса, написание скриптов custom.d, отладка проблем. Срабатывает по словам: zapret, nfqws, DPI desync, обход блоков, антиблокировка, стратегии zapret, список хостов, автохостлист.
