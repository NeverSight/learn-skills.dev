Documentazione completa in lingua russa per il progetto zapret-openwrt: un'utilità Anti-DPI per router OpenWrt. Utilizza questa competenza per qualsiasi domanda su zapret-openwrt: architettura del progetto, configurazione, strategie di bypass DPI, interfaccia web LuCI, script, aggiornamenti, diagnostica, hook custom.d, elenchi di host, parametri nfqws, gestione del servizio tramite procd. Usalo anche durante lo sviluppo: aggiunta di nuove strategie, modifica dell'interfaccia utente, scrittura di script custom.d, debug di problemi. Innescato dalle parole: zapret, nfqws, desync DPI, bypass blocchi, anti-blocco, strategie zapret, hostlist, autohostlist.
