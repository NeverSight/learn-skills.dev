Documentation complète en russe pour le projet zapret-openwrt : un utilitaire Anti-DPI pour les routeurs OpenWrt. Utilisez cette compétence pour toute question sur zapret-openwrt : architecture du projet, configuration, stratégies de contournement DPI, interface Web LuCI, scripts, mises à jour, diagnostics, hooks personnalisés.d, listes d'hôtes, paramètres nfqws et gestion des services procd. Utilisez-le également pendant le développement : ajout de nouvelles stratégies, modification de l'interface utilisateur, écriture de scripts personnalisés, problèmes de débogage. Déclenché par les mots : zapret, nfqws, DPI desync, bypass block, anti-blocking, zapret stratégies, hostlist, autohostlist.
