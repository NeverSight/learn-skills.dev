Speichern und organisieren Sie Zielartefakte (Dokumente, Präsentationen, Markdown, Bilder, SVGs) in ~/projects/artifacts-viewer. Verwenden Sie diese Option, wenn der Benutzer Dokumente, PDFs, PPTs, Markdown-Dateien, Bilder (PNG, JPG, SVG) oder beliebige visuelle/Dokumentartefakte an einem zentralen Viewer-Speicherort speichern oder organisieren möchte. Wird auch verwendet, um das Git-Repo für den Artefakt-Viewer zu initialisieren und an GitHub zu übertragen.
