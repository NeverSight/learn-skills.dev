Salva e organizza gli artefatti di destinazione (documenti, presentazioni, markdown, immagini, SVG) in ~/projects/artifacts-viewer. Da utilizzare quando l'utente desidera salvare o organizzare documenti, PDF, PPT, file di markdown, immagini (PNG, JPG, SVG) o qualsiasi elemento visivo/documentario in una posizione di visualizzazione centrale. Utilizzalo anche per inizializzare il repository git per il visualizzatore di artefatti e inviarlo a GitHub.
