Guarde y organice artefactos de destino (documentos, presentaciones, rebajas, imágenes, SVG) en ~/projects/artifacts-viewer. Úselo cuando el usuario desee guardar u organizar documentos, PDF, PPT, archivos de rebajas, imágenes (PNG, JPG, SVG) o cualquier artefacto visual/documental en una ubicación de visor central. También se utiliza para inicializar el repositorio de git para el visor de artefactos y enviarlo a GitHub.
