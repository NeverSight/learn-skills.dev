Référence de la documentation OpenClaw. OpenClaw est une passerelle open source (MIT) auto-hébergée qui connecte les applications de chat (WhatsApp, Telegram, Discord, iMessage, Slack, etc.) aux agents IA. À utiliser lorsque l'utilisateur pose des questions sur l'installation, la configuration, les canaux, l'automatisation, le déploiement ou toute autre fonctionnalité OpenClaw.
