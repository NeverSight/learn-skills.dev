Da utilizzare quando Claude Code necessita di una seconda opinione, verifica o ricerca più approfondita su questioni tecniche. Ciò include la ricerca sul funzionamento di una libreria o di un'API, la conferma degli approcci di implementazione, la verifica dei presupposti tecnici, la comprensione di modelli di codice complessi o l'acquisizione di prospettive alternative sulle decisioni relative all'architettura. L'agente sfrutta la CLI Codex per fornire analisi e convalida indipendenti.
