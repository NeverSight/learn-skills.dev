À utiliser lorsque Claude Code a besoin d'un deuxième avis, d'une vérification ou d'une recherche plus approfondie sur des questions techniques. Cela inclut la recherche du fonctionnement d'une bibliothèque ou d'une API, la confirmation des approches de mise en œuvre, la vérification des hypothèses techniques, la compréhension de modèles de code complexes ou l'obtention de perspectives alternatives sur les décisions architecturales. L'agent exploite la CLI Codex pour fournir une analyse et une validation indépendantes.
