استخدمه عندما يحتاج Claude Code إلى رأي ثانٍ، أو التحقق، أو بحث أعمق حول الأمور الفنية. يتضمن ذلك البحث عن كيفية عمل المكتبة أو واجهة برمجة التطبيقات (API)، أو تأكيد أساليب التنفيذ، أو التحقق من الافتراضات الفنية، أو فهم أنماط التعليمات البرمجية المعقدة، أو الحصول على وجهات نظر بديلة بشأن القرارات المعمارية. يستفيد الوكيل من Codex CLI لتوفير التحليل والتحقق المستقلين.
