OBBLIGATORIO prima di iniziare qualsiasi attività. Applica il ciclo di esecuzione GPA che impedisce la proliferazione delle chiamate allo strumento. G: la fase GATHER combina query di rilevamento + letture di memoria + letture di file in un'unica fase. P: Pianifica nel testo senza chiamate a strumenti. R: APPLICA tutte le scritture/modifiche/verifica in un'unica fase. Una chiamata per tipo di utensile per fase: raggruppa insieme tutte le operazioni dello stesso tipo. Copre l'analisi delle dipendenze, le opportunità batch, la stima dell'ambito e i trigger di loopback.
