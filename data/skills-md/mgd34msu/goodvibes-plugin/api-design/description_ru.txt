Загружайте ПРОАКТИВНО, когда задача включает в себя создание или изменение конечных точек API. Используйте, когда пользователь говорит «создать API», «добавить конечную точку», «создать маршрут REST», «настроить GraphQL» или «добавить процедуры tRPC». Охватывает проектирование маршрутов и организацию файлов, проверку запросов с помощью Zod, форматирование ответов, шаблоны обработки ошибок, композицию промежуточного программного обеспечения, средства защиты аутентификации, ограничение скорости, нумерацию страниц и создание документации API.
