Chargez de manière PROACTIVE lorsque la tâche implique la création ou la modification de points de terminaison d'API. À utiliser lorsque l'utilisateur dit « créer une API », « ajouter un point de terminaison », « créer une route REST », « configurer GraphQL » ou « ajouter des procédures tRPC ». Couvre la conception des itinéraires et l'organisation des fichiers, la validation des demandes avec Zod, le formatage des réponses, les modèles de gestion des erreurs, la composition du middleware, les gardes d'authentification, la limitation du débit, la pagination et la génération de documentation API.
