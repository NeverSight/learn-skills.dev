Cargue PROACTIVAMENTE cuando la tarea implique crear o modificar puntos finales de API. Úselo cuando el usuario diga "crear una API", "agregar un punto final", "crear una ruta REST", "configurar GraphQL" o "agregar procedimientos tRPC". Cubre el diseño de rutas y organización de archivos, validación de solicitudes con Zod, formato de respuesta, patrones de manejo de errores, composición de middleware, protecciones de autenticación, limitación de velocidad, paginación y generación de documentación API.
