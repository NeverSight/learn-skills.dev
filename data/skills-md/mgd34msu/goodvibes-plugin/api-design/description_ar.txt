قم بالتحميل بشكل استباقي عندما تتضمن المهمة إنشاء نقاط نهاية واجهة برمجة التطبيقات أو تعديلها. يُستخدم عندما يقول المستخدم "إنشاء واجهة برمجة التطبيقات" أو "إضافة نقطة نهاية" أو "إنشاء مسار REST" أو "إعداد GraphQL" أو "إضافة إجراءات tRPC". يغطي تصميم المسار وتنظيم الملفات، والتحقق من صحة الطلب باستخدام Zod، وتنسيق الاستجابة، وأنماط معالجة الأخطاء، وتكوين البرامج الوسيطة، وحراس المصادقة، وتحديد المعدل، وترقيم الصفحات، وإنشاء وثائق API.
