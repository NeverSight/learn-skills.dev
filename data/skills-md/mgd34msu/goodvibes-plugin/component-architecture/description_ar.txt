قم بالتحميل بشكل استباقي عندما تتضمن المهمة تصميم أو بناء مكونات واجهة المستخدم. يُستخدم عندما يقول المستخدم "إنشاء مكون"، أو "إنشاء نموذج"، أو "إضافة نموذج"، أو "تصميم التخطيط"، أو "إعادة بناء هذه الصفحة". يغطي تكوين المكونات وتسلسلها الهرمي، وتصميم الدعامات وكتابتها، وتحسين العرض (المذكرة، وuseMemo، وuseCallback)، وأنماط المكونات المركبة، والنماذج الخاضعة للرقابة مقابل النماذج غير الخاضعة للرقابة، وتنظيم الملفات، وإمكانية الوصول إلى React وVue وSvelte.
