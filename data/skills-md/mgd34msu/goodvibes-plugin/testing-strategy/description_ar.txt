قم بالتحميل بشكل استباقي عندما تتضمن المهمة كتابة الاختبارات أو تحسين التغطية أو البنية التحتية للاختبار. يُستخدم عندما يقول المستخدم "أضف اختبارات"، أو "اكتب اختبارات الوحدة"، أو "إعداد اختبار E2E"، أو "تحسين التغطية"، أو "أضف نماذج وهمية". يغطي اختبارات وحدة Vitest/Jest، واختبارات مكونات مكتبة React Testing، واختبارات Playwright E2E، وسخرية MSW API، وتنظيم الاختبار وتسميته، وإدارة التركيبات، وأهداف التغطية وعتباتها، واختبار اللقطة، وتكامل CI.
