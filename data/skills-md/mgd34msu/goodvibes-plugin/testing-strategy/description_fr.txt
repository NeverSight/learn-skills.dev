Chargez de manière PROACTIVE lorsque la tâche implique l'écriture de tests, l'amélioration de la couverture ou l'infrastructure de test. À utiliser lorsque l'utilisateur dit « ajouter des tests », « écrire des tests unitaires », « configurer les tests E2E », « améliorer la couverture » ou « ajouter des simulations ». Couvre les tests unitaires Vitest/Jest, les tests des composants de la bibliothèque de tests React, les tests Playwright E2E, la simulation d'API MSW, l'organisation et la dénomination des tests, la gestion des appareils, les cibles et seuils de couverture, les tests d'instantanés et l'intégration CI.
