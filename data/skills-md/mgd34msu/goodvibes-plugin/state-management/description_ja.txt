タスクにアプリケーションの状態、データのフェッチ、またはフォームの処理が含まれる場合は、積極的にロードします。ユーザーが「状態を管理する」、「データ取得を追加する」、「Zustand をセットアップする」、「フォーム検証を処理する」、または「React Query を追加する」と言うときに使用します。サーバー状態 (キャッシュを備えた TanStack クエリ、オプティミスティック更新)、クライアント状態 (Zustand ストア)、フォーム状態 (Zod 検証を備えた React Hook Form)、URL 状態 (検索パラメータ、ルーティング)、および状態ソリューションの選択について説明します。
