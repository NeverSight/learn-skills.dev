قم بالتحميل بشكل استباقي عندما تتضمن المهمة حالة التطبيق أو جلب البيانات أو معالجة النموذج. يُستخدم عندما يقول المستخدم "إدارة الحالة"، أو "إضافة جلب البيانات"، أو "إعداد Zustand"، أو "التعامل مع التحقق من صحة النموذج"، أو "إضافة استعلام React". يغطي حالة الخادم (استعلام TanStack مع التخزين المؤقت والتحديثات المتفائلة)، وحالة العميل (مخازن Zustand)، وحالة النموذج (نموذج React Hook مع التحقق من Zod)، وحالة URL (معلمات البحث، والتوجيه)، والاختيار بين حلول الحالة.
