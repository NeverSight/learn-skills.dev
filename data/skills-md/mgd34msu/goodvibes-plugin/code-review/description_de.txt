Laden Sie PROAKTIV, wenn die Aufgabe die Überprüfung von Code, die Prüfung der Qualität oder die Validierung von Implementierungen umfasst. Wird verwendet, wenn der Benutzer sagt: „Diesen Code überprüfen“, „diese PR prüfen“, „die Codebasis prüfen“ oder „diese Implementierung bewerten“. Deckt die 10-dimensional gewichtete Bewertungsrubrik (Korrektheit, Sicherheit, Leistung, Architektur, Tests, Fehlerbehandlung, Typsicherheit, Wartbarkeit, Zugänglichkeit, Dokumentation), die automatische Mustererkennung für Anti-Patterns und die strukturierte Überprüfungsausgabe mit umsetzbaren Ergebnissen ab.
