Cargue PROACTIVAMENTE cuando la tarea implique revisar código, auditar la calidad o validar implementaciones. Úselo cuando el usuario diga "revisar este código", "verificar este PR", "auditar la base de código" o "calificar esta implementación". Cubre la rúbrica de puntuación ponderada de 10 dimensiones (corrección, seguridad, rendimiento, arquitectura, pruebas, manejo de errores, seguridad de tipos, mantenibilidad, accesibilidad, documentación), detección automatizada de patrones para antipatrones y resultados de revisión estructurados con hallazgos procesables.
