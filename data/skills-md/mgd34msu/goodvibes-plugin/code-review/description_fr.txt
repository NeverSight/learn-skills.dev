Chargez de manière PROACTIVE lorsque la tâche implique la révision du code, l’audit de la qualité ou la validation des implémentations. À utiliser lorsque l'utilisateur dit « examiner ce code », « vérifier ce PR », « auditer la base de code » ou « évaluer cette implémentation ». Couvre la rubrique de notation pondérée en 10 dimensions (exactitude, sécurité, performances, architecture, tests, gestion des erreurs, sécurité des types, maintenabilité, accessibilité, documentation), la détection automatisée des modèles pour les anti-modèles et la sortie d'examen structuré avec des résultats exploitables.
