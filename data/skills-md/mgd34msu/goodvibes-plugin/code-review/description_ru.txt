Загружайте ПРОАКТИВНО, когда задача включает проверку кода, аудит качества или проверку реализации. Используйте, когда пользователь говорит «проверьте этот код», «проверьте этот PR», «проверьте кодовую базу» или «оцените эту реализацию». Охватывает 10-мерную взвешенную оценку (правильность, безопасность, производительность, архитектура, тестирование, обработка ошибок, безопасность типов, удобство сопровождения, доступность, документация), автоматическое обнаружение шаблонов для антишаблонов и структурированные результаты проверки с практическими выводами.
