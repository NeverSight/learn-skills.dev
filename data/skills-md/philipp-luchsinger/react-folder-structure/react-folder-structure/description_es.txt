Estructura del proyecto, convenciones y guía de configuración de React + Vite + TypeScript + shadcn/ui. Utilice esta habilidad al iniciar un nuevo proyecto, crear una nueva característica, conectar llamadas API con TanStack Query + Axios o administrar el estado con Zustand. Aplica funciones autónomas, una capa API escrita y un estilo Tailwind v4 basado en temas a través de variables CSS.
