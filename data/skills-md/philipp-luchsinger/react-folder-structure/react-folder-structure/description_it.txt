Opinione React + Vite + TypeScript + struttura del progetto shadcn/ui, convenzioni e guida all'installazione. Utilizza questa competenza durante l'avvio di un nuovo progetto, la creazione di una nuova funzionalità, il collegamento di chiamate API con TanStack Query + Axios o la gestione dello stato con Zustand. Applica funzionalità autonome, un livello API digitato e uno stile Tailwind v4 sensibile al tema tramite variabili CSS.
