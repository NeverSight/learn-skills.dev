Упрямая структура проекта React + Vite + TypeScript + shadcn/ui, соглашения и руководство по настройке. Используйте этот навык при загрузке нового проекта, создании новой функции, подключении вызовов API с помощью TanStack Query + Axios или управлении состоянием с помощью Zustand. Обеспечивает автономные функции, типизированный уровень API и стиль Tailwind v4 с учетом тем с помощью переменных CSS.
