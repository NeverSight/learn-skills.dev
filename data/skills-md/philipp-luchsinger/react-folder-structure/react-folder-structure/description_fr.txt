Opinionated React + Vite + TypeScript + structure du projet shadcn/ui, conventions et guide de configuration. Utilisez cette compétence lors du démarrage d'un nouveau projet, de la création d'une nouvelle fonctionnalité, du câblage des appels d'API avec TanStack Query + Axios ou de la gestion de l'état avec Zustand. Applique des fonctionnalités autonomes, une couche API typée et un style Tailwind v4 sensible au thème via des variables CSS.
