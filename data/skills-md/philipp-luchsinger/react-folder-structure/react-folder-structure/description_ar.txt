هيكل مشروع React + Vite + TypeScript + shadcn/ui، واتفاقياته، ودليل الإعداد. استخدم هذه المهارة عند تمهيد مشروع جديد، أو إنشاء ميزة جديدة، أو توصيل مكالمات API باستخدام TanStack Query + Axios، أو إدارة الحالة باستخدام Zustand. يفرض ميزات قائمة بذاتها، وطبقة API مكتوبة، وتصميم Tailwind v4 المدرك للموضوع عبر متغيرات CSS.
