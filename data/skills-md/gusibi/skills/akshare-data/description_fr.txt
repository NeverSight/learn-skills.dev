Interrogez des données financières chinoises et mondiales à l'aide de la bibliothèque AKShare Python. À utiliser lorsqu'on vous le demande pour (1) récupérer des cotations boursières, des prix historiques ou des états financiers pour les actions A/HK/américaines, (2) interroger des données macroéconomiques telles que PIB/IPC/PMI, (3) obtenir des données sur les contrats à terme/options/obligations/forex/fonds, (4) rechercher des données d'indice ou (5) récupérer des données alternatives comme le sentiment de l'actualité. Couvre plus de 500 interfaces de données pour les actions, les contrats à terme, les options, les obligations, le forex, les fonds, les données macro, les indices et les données alternatives.
