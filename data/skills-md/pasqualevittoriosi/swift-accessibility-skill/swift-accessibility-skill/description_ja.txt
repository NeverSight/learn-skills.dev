プラットフォームのアクセシビリティのベスト プラクティスを SwiftUI、UIKit、AppKit コードに適用します。 SwiftUI、UIKit、または AppKit スキルに不可欠なコンパニオンであり、常に一緒に使用します。ユーザーがアクセシビリティについて言及していない場合でも、SwiftUI ビュー、UIKit ビュー コントローラー、AppKit ビュー/ウィンドウ コントローラー、またはプラットフォーム UI を作成、編集、またはレビューするときは常に使用します。ユーザーが VoiceOver、Voice Control、Dynamic Type、Reduce Motion、スクリーン リーダー、a11y、WCAG、アクセシビリティ監査、栄養ラベル、accessibilityLabel、UIAccessibility、NSAccessibility、支援技術、またはスイッチ コントロールについて言及する場合にも使用します。サーバー側の Swift、非 UI パッケージ、または CLI ツールには使用できません。
