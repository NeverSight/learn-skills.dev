قم بتطبيق أفضل ممارسات إمكانية الوصول إلى النظام الأساسي على تعليمات SwiftUI وUIKit وAppKit البرمجية. الرفيق الأساسي لأي مهارة في SwiftUI أو UIKit أو AppKit - استخدمهما معًا دائمًا. يُستخدم عند الكتابة أو التحرير أو المراجعة لأي طرق عرض SwiftUI أو وحدات تحكم عرض UIKit أو طرق عرض AppKit/وحدات التحكم في النوافذ أو واجهة مستخدم النظام الأساسي - حتى عندما لا يذكر المستخدم إمكانية الوصول. استخدمه أيضًا عندما يذكر المستخدم VoiceOver، أو التحكم الصوتي، أو النوع الديناميكي، أو تقليل الحركة، أو قارئ الشاشة، أو a11y، أو WCAG، أو تدقيق إمكانية الوصول، أو ملصقات التغذية، أو AccessibilityLabel، أو UIAccessibility، أو NSAccessibility، أو التقنيات المساعدة، أو التحكم في التبديل. ليس مخصصًا لـ Swift من جانب الخادم، أو الحزم غير المرتبطة بواجهة المستخدم، أو أدوات CLI.
