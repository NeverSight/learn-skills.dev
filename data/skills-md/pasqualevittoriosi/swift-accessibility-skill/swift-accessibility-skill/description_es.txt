Aplique las mejores prácticas de accesibilidad de la plataforma al código SwiftUI, UIKit y AppKit. Compañero esencial para cualquier habilidad SwiftUI, UIKit o AppKit: utilícelos siempre juntos. Úselo siempre que escriba, edite o revise CUALQUIER vista de SwiftUI, controladores de vista UIKit, vistas/controladores de ventana de AppKit o interfaz de usuario de plataforma, incluso cuando el usuario no mencione la accesibilidad. Úselo también cuando el usuario mencione VoiceOver, Control de voz, Tipo dinámico, Reducir movimiento, lector de pantalla, a11y, WCAG, auditoría de accesibilidad, etiquetas nutricionales, etiqueta de accesibilidad, UIAccessibility, NSAccessibility, tecnologías de asistencia o control por interruptor. No para Swift del lado del servidor, paquetes que no sean UI o herramientas CLI.
