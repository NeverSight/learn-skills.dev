Быстро создайте и внедрите играбельную игру — без ресурсов, дизайна, звука, развертывания и монетизации. Быстро выведите что-нибудь на экран. Используйте, когда пользователь говорит «быстрая игра», «быстрый прототип», «просто получите что-нибудь играбельное» или хочет игру без полного конвейера. Для полного конвейера вместо этого используйте make-game. НЕ используйте для производственных игр (используйте make-game для полного конвейера).
