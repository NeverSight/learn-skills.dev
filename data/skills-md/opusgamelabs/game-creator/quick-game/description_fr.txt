Élaborez et implémentez rapidement un jeu jouable – sans ressources, conception, audio, déploiement ou monétisation. Obtenez rapidement quelque chose à l'écran. À utiliser lorsque l'utilisateur dit « jeu rapide », « prototype rapide », « obtenez simplement quelque chose de jouable » ou souhaite un jeu sans le pipeline complet. Pour le pipeline complet, utilisez plutôt make-game. Ne PAS utiliser pour les jeux de production (utilisez make-game pour le pipeline complet).
