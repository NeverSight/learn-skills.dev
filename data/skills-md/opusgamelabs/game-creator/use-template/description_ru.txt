Клонируйте шаблон игры из галереи в качестве отправной точки. Используйте, когда пользователь говорит «использовать шаблон», «начать с шаблона», «клонировать Flappy-bird», «использовать шаблон платформера» или хочет быстро загрузить игру из существующего примера. НЕ используйте для создания игры с нуля (используйте make-game).
