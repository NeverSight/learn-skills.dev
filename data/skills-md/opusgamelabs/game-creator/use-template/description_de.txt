Klonen Sie als Ausgangspunkt eine Spielvorlage aus der Galerie. Verwenden Sie diese Option, wenn der Benutzer sagt: „Eine Vorlage verwenden“, „Mit einer Vorlage beginnen“, „Flappy-Bird klonen“, „Die Platformer-Vorlage verwenden“ oder ein Spiel schnell aus einem vorhandenen Beispiel booten möchte. NICHT zum Erstellen eines Spiels von Grund auf verwenden (verwenden Sie make-game).
