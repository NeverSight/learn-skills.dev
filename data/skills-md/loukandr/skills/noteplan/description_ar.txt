معرفة شاملة لبرنامج NotePlan، وهو تطبيق الإنتاجية القائم على Markdown والذي يجمع بين تدوين الملاحظات وإدارة المهام وتكامل التقويم. استخدم هذه المهارة عندما يذكر المستخدم NotePlan، ويسأل عن بناء جملة NotePlan، وجدولة المهام، وملاحظات التقويم، والملاحظات اليومية/الأسبوعية/الشهرية/الربع سنوية/السنوية، وملاحظات المشروع، وحظر الوقت، والقوالب، والموضوع الأمامي، ومرشحات البحث، وروابط wiki، والروابط الخلفية، والعلامات، والإشارات، ونظام URL لـ NotePlan (noteplan://)، والخطوط المتزامنة، وتكامل تقويم Apple أو التذكيرات داخل NotePlan، وموضوعات NotePlan، وكانبان/مجلد طرق عرض، أو يريد إنشاء/تعديل/تصحيح مكونات NotePlan الإضافية (plugin.json، script.js، JavaScript API مع DataStore، Editor، CommandBar، HTMLView، Calendar، Clipboard object).
