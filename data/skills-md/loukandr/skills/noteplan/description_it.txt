Conoscenza completa di NotePlan, l'app di produttività basata su Markdown che combina la presa di appunti, la gestione delle attività e l'integrazione del calendario. Utilizza questa abilità ogni volta che l'utente menziona NotePlan, chiede informazioni sulla sintassi del markdown di NotePlan, sulla pianificazione delle attività, sulle note del calendario, sulle note giornaliere/settimanali/mensili/trimestrali/annuali, sulle note di progetto, sul blocco temporale, sui modelli, sul frontmatter, sui filtri di ricerca, sui collegamenti wiki, sui backlink, sui tag, sulle menzioni, sullo schema URL di NotePlan (noteplan://), sulle linee sincronizzate, sull'integrazione di Apple Calendar o Promemoria all'interno di NotePlan, sui temi NotePlan, sulle visualizzazioni Kanban/cartelle o se desidera costruire/modificare/debug dei plugin NotePlan (plugin.json, script.js, API JavaScript con DataStore, Editor, CommandBar, HTMLView, Calendario, oggetti Appunti).
