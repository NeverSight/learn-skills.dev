Úselo al analizar capturas de pantalla o diseños de la interfaz de usuario para extraer y documentar elementos del sistema de diseño. Analiza imágenes para identificar colores, tipografía, espaciado, componentes y patrones, luego genera un sistema de diseño estructurado con tokens de diseño y fragmentos de CSS. Ideal para diseños de ingeniería inversa, creación de guías de estilo o documentación de interfaces de usuario existentes.
