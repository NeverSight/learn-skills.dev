يُستخدم عند تحليل لقطات شاشة واجهة المستخدم أو تصميماتها لاستخراج وتوثيق عناصر نظام التصميم. يحلل الصور لتحديد الألوان والطباعة والمسافات والمكونات والأنماط، ثم ينشئ نظام تصميم منظم باستخدام رموز التصميم ومقتطفات CSS. مثالي لتصميمات الهندسة العكسية، أو إنشاء أدلة الأنماط، أو توثيق واجهات المستخدم الموجودة.
