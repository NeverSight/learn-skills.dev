Générez un échafaudage de plugins Orderly SDK avec un identifiant unique, un passe-partout d'enregistrement et des intercepteurs typés. À utiliser lorsque l'utilisateur mentionne la création, le développement, l'enregistrement, l'implémentation ou la génération d'un plugin (par exemple, « créer un plugin », « générer un plugin », « développer un plugin », « enregistrer un plugin », « implémenter un plugin », « construire un plugin », « ajouter un plugin », « élaborer un plugin », « nouveau plugin »).
