يرشد المطورين والمختبرين في كتابة سيناريوهات اختبار تحميل جاتلينج وإصلاحها وهيكلتها. استخدم هذه المهارة عندما يذكر المستخدم اختبار التحميل، أو اختبار الأداء، أو اختبار التحمل، أو Gatling، أو المستخدمين الظاهريين، أو VUs، أو التعزيز، أو ملفات تعريف الحقن، أو عمليات المحاكاة، أو ترحيل JMeter، أو ترحيل k6، أو الإنتاجية، أو اتفاقيات مستوى الخدمة لوقت الاستجابة، أو يريد قياس واجهة برمجة التطبيقات أو الخدمة - حتى لو لم يذكر صراحة "Gatling" أو "اختبار الأداء".
