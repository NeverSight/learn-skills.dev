يرشد المطورين والمختبرين في كتابة البرامج النصية لاختبار تحميل k6 وإصلاحها وبنيتها باستخدام JavaScript أو TypeScript. استخدم هذه المهارة عندما يذكر المستخدم k6، أو Grafana k6، أو اختبار التحميل باستخدام JavaScript، أو المستخدمين الظاهريين، أو VUs، أو السيناريوهات، أو المنفذين، أو الحدود، أو عمليات التحقق، أو SharedArray، أو مراحل التكثيف، أو معدل الوصول، أو نموذج عبء العمل المفتوح/المغلق، أو يريد قياس أداء واجهة برمجة التطبيقات (API)، أو WebSocket، أو خدمة gRPC باستخدام k6 - حتى لو لم يذكروا صراحة "k6" أو "اختبار التحميل".
