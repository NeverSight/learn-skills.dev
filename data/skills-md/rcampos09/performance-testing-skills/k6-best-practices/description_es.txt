Guía a los desarrolladores y evaluadores a escribir, corregir y estructurar scripts de prueba de carga k6 utilizando JavaScript o TypeScript. Utilice esta habilidad siempre que el usuario mencione k6, Grafana k6, pruebas de carga con JavaScript, usuarios virtuales, VU, escenarios, ejecutores, umbrales, comprobaciones, SharedArray, etapas de aceleración, tasa de llegada, modelo de carga de trabajo abierta/cerrada o quiera comparar un servicio API, WebSocket o gRPC usando k6, incluso si no dice explícitamente "k6" o "prueba de carga".
