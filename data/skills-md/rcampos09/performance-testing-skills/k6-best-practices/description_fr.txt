Guide les développeurs et les testeurs dans l'écriture, la correction et la structuration de scripts de test de charge K6 à l'aide de JavaScript ou TypeScript. Utilisez cette compétence chaque fois que l'utilisateur mentionne k6, Grafana k6, les tests de charge avec JavaScript, les utilisateurs virtuels, les VU, les scénarios, les exécuteurs, les seuils, les contrôles, SharedArray, les étapes de montée en puissance, le taux d'arrivée, le modèle de charge de travail ouvert/fermé, ou souhaite comparer un service API, WebSocket ou gRPC à l'aide de k6, même s'il ne dit pas explicitement « k6 » ou « test de charge ».
