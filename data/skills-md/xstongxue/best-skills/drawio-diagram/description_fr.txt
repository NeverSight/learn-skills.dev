Générer des visualisations au format Draw.io (.drawio) standard pour les modèles d'apprentissage en profondeur, les architectures de réseau, les flux d'algorithmes, etc. ; prend en charge à la fois la génération à partir de zéro et le transfert de style. À partir de zéro : diagrammes d'architecture, organigrammes, schémas de champ récepteur, etc. Transfert de style : image de référence + description du contenu/projet → nouvelle image dans le style de référence. Assurez-vous que le XML est correct qui s'ouvre et se modifie directement dans Draw.io.
