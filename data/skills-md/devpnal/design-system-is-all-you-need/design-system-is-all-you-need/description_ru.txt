Этот навык предоставляет тщательно подобранный набор рекомендаций по кросс-платформенному проектированию для создания согласованных высококачественных пользовательских интерфейсов. Каждое руководство определяет общие правила, руководства для конкретных компонентов и автоматически создаваемую систему токенов — и все это основано на минимальном вмешательстве пользователя.
