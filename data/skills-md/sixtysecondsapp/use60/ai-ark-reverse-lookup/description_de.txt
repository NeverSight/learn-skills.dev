Bereichern Sie einen bekannten Kontakt mit vollständigen Profildaten mithilfe der Reverse-Lookup-Funktion von AI Ark.
Akzeptiert einen Namen, eine E-Mail-Adresse oder eine LinkedIn-URL und gibt die aktuelle Berufsbezeichnung, das Unternehmen usw. zurück.
Abteilung, Dienstalter, bestätigte E-Mail-Adresse, Telefonnummer und soziale Profile.
Wird verwendet, wenn ein Benutzer sagt: „Diesen Kontakt anreichern“, „Diese Person suchen“
„Vollständiges Profil erhalten für“ oder stellt eine LinkedIn-URL zur Anreicherung bereit.
Kann im Batch-Modus über alle Zeilen in einer Tabellenspalte ausgeführt werden.
