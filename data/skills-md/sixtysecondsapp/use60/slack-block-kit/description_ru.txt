Создавайте насыщенные интерактивные сообщения Slack с помощью Block Kit для рабочих процессов продаж.
Содержит брифинги совещаний, коучинговые дайджесты, оповещения о сделках, сводки конвейеров,
Утверждения HITL и все типы уведомлений. Использует наши типизированные функции построения
из slackBlocks.ts с безопасностью автоматического усечения. Обрабатывает полное сообщение
жизненный цикл: формирование блоков, доставка через send-slack-message, обработка взаимодействий.
Используйте при создании любого уведомления, оповещения, дайджеста или интерактивного сообщения Slack.
НЕ используйте для содержимого электронной почты или компонентов пользовательского интерфейса приложения.
