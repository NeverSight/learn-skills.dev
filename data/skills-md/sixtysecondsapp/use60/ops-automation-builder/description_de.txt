Erstellen und verwalten Sie Automatisierungsregeln für Operationstabellen. Regeln lösen Aktionen aus, wenn sie spezifisch sind
Es treten Ereignisse auf – etwa das Aktualisieren einer Zelle, wenn die Anreicherung abgeschlossen ist, oder das Übertragen von Zeilen an HubSpot
wenn ein Tag hinzugefügt wird, oder Senden von Benachrichtigungen, wenn eine neue Zeile erstellt wird.
Wird verwendet, wenn ein Benutzer sagt: „Eine Regel erstellen“, „Diese Tabelle automatisieren“, „Wenn sich eine Zelle ändert, führen Sie X aus“.
„Automatisierung einrichten“, „Mich benachrichtigen, wenn“ oder möchte eine Wenn-dies-dann-das-Stillogik erstellen
auf ihren Operationstischen. Listet auch vorhandene Regeln in einer Tabelle auf.
