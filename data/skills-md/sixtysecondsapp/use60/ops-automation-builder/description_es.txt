Cree y administre reglas de automatización para tablas de operaciones. Las reglas desencadenan acciones cuando son específicas
ocurren eventos, como actualizar una celda cuando se completa el enriquecimiento, enviar filas a HubSpot
cuando se agrega una etiqueta, o enviar notificaciones cuando se crea una nueva fila.
Úselo cuando un usuario dice "crear una regla", "automatizar esta tabla", "cuando una celda cambie, haga X",
"configurar la automatización", "notificarme cuando" o quiere crear una lógica de estilo si-esto-entonces-aquello
en sus mesas de operaciones. También enumera las reglas existentes en una tabla.
