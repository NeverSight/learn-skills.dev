Bietet einen systematischen Anforderungsanalyse-Workflow zum Verständnis von Anforderungen, zur Erkundung der Codebasis, zur Klärung von Problemen, zur Verwendung einer detaillierten Analyse und zur Präsentation eines Implementierungsplans vor dem Codieren. Anwendbar auf Szenarien wie Feature-Entwicklung, API-Design, Datenbankdesign, Modulentwicklung und Feature-Änderung. Dieser Skill wird automatisch ausgelöst, wenn Benutzer Anforderungen für die Funktionsentwicklung, das API-Design, das Datenbankdesign usw. vorschlagen. Interagiert entsprechend den Benutzereinstellungen und der Eingabesprache.
