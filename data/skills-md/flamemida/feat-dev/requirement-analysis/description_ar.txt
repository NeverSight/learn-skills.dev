يوفر سير عمل تحليل المتطلبات المنهجي لفهم الاحتياجات، واستكشاف قاعدة التعليمات البرمجية، وتوضيح المشكلات، واستخدام التحليل المتعمق، وتقديم خطة التنفيذ قبل البرمجة. ينطبق على سيناريوهات مثل تطوير الميزات وتصميم واجهة برمجة التطبيقات وتصميم قاعدة البيانات وتطوير الوحدة وتعديل الميزات. يتم تشغيل هذه المهارة تلقائيًا عندما يقترح المستخدمون متطلبات تطوير الميزات، وتصميم واجهة برمجة التطبيقات، وتصميم قاعدة البيانات، وما إلى ذلك. ويتفاعل وفقًا لإعدادات المستخدم ولغة الإدخال.
