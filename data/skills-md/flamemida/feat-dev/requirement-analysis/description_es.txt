Proporciona un flujo de trabajo sistemático de análisis de requisitos para comprender las necesidades, explorar la base del código, aclarar problemas, utilizar un análisis en profundidad y presentar un plan de implementación antes de codificar. Aplicable a escenarios como desarrollo de funciones, diseño de API, diseño de bases de datos, desarrollo de módulos y modificación de funciones. Esta habilidad se activa automáticamente cuando los usuarios proponen requisitos para el desarrollo de funciones, diseño de API, diseño de bases de datos, etc. Interactúa según la configuración del usuario y el idioma de entrada.
