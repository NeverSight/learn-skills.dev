Обеспечивает систематический 9-этапный анализ требований и рабочий процесс реализации (понимание требований, исследование кода, исследование внешних ресурсов, уточняющие вопросы, углубленный анализ, планирование презентации, разработка реализации, проверка кода, резюме). Подходит для разработки сложных функций, сравнения нескольких вариантов, исследования новых технологических стеков и других сценариев, требующих глубокого планирования и полной реализации. Срабатывает, когда пользователи предлагают сложную разработку функций, проектирование API или проектирование базы данных, требующее углубленного анализа и исследования внешних ресурсов.
