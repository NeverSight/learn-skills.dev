Compétences en analyse de données de bout en bout prenant en charge le traitement des données tabulaires (CSV/Excel), de documents (PDF/DOCX/MD) et d'images. Le flux de travail principal suit strictement : 1. Exploration sécurisée : lecture uniquement des métadonnées et d'un petit échantillon, le chargement complet est interdit pour empêcher le débordement de jeton ; 2. Contrôle qualité : inspectez automatiquement les valeurs manquantes, les doublons et la cohérence ; 3. Exécution de code : nettoyer, analyser et visualiser à l'aide de Python (Pandas/Matplotlib) ; 4. Validation des résultats et génération de rapports. Scénarios applicables : analyse exploratoire (EDA), extraction de données multimodales, tracé de graphiques et génération de rapports d'analyse des données.
