مهارات تحليل البيانات ذات الارتباط الكامل، ودعم معالجة الجداول (CSV/Excel)، والمستندات (PDF/DOCX/MD)، وبيانات الصور. يتبع سير العمل الأساسي بدقة ما يلي: 1. الاستكشاف الآمن: اقرأ فقط البيانات الوصفية وعينة صغيرة، ولا تحميل كامل أبدًا لتجنب تجاوز سعة الرمز المميز؛ 2. فحص الجودة: الكشف التلقائي عن القيم المفقودة والتكرارات والاتساق؛ 3. تنفيذ التعليمات البرمجية: التنظيف والتحليل والتصور باستخدام Python (Pandas/Matplotlib)؛ 4. التحقق من صحة النتيجة وإنشاء التقرير. ينطبق على: تحليل البيانات الاستكشافية (EDA)، واستخراج البيانات متعددة الوسائط، والرسوم البيانية، وإنشاء تقارير رؤى البيانات.
