مهارات تحليل البيانات الشاملة التي تدعم الجداول (CSV/Excel)، والمستندات (PDF/DOCX/MD)، ومعالجة بيانات الصور. يتبع سير العمل الأساسي بدقة ما يلي: 1. الاستكشاف الآمن: قراءة البيانات الوصفية وعينة صغيرة فقط، ويُحظر التحميل الكامل لمنع تجاوز الرمز المميز؛ 2. فحص الجودة: فحص القيم المفقودة والتكرارات والاتساق تلقائيًا؛ 3. تنفيذ التعليمات البرمجية: التنظيف والتحليل والتصور باستخدام Python (Pandas/Matplotlib)؛ 4. التحقق من صحة النتيجة وإنشاء التقرير. السيناريوهات القابلة للتطبيق: التحليل الاستكشافي (EDA)، واستخراج البيانات متعددة الوسائط، ورسم المخططات، وإنشاء تقارير رؤية البيانات.
