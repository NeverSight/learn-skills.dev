End-to-end data analysis skills supporting tabular (CSV/Excel), document (PDF/DOCX/MD), and image data processing. Core workflow strictly follows: 1. Safe exploration: read only metadata and a small sample, full loading is forbidden to prevent token overflow; 2. Quality check: automatically inspect missing values, duplicates, and consistency; 3. Code execution: clean, analyze, and visualize using Python (Pandas/Matplotlib); 4. Result validation and report generation. Applicable scenarios: exploratory analysis (EDA), multimodal data extraction, chart plotting, and generating data insight reports.
