Навыки комплексного анализа данных с поддержкой обработки табличных (CSV/Excel), документов (PDF/DOCX/MD) и изображений изображений. Основной рабочий процесс строго соблюдается: 1. Безопасное исследование: метаданные только для чтения и небольшая выборка, полная загрузка запрещена во избежание переполнения токена; 2. Проверка качества: автоматическая проверка отсутствующих значений, дубликатов и согласованности; 3. Выполнение кода: очистка, анализ и визуализация с помощью Python (Pandas/Matplotlib); 4. Проверка результатов и составление отчета. Применимые сценарии: исследовательский анализ (EDA), мультимодальное извлечение данных, построение диаграмм и создание отчетов с аналитикой данных.
