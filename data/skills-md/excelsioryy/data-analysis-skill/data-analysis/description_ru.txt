Навыки полноканального анализа данных с поддержкой обработки табличных (CSV/Excel), документов (PDF/DOCX/MD) и изображений изображений. Основной рабочий процесс строго следующий: 1. Безопасное исследование: только чтение метаданных и небольшой выборки; полная загрузка запрещена во избежание переполнения токена; 2. Проверка качества: автоматическое обнаружение пропущенных значений, дубликатов и несоответствий; 3. Выполнение кода: очистка, анализ и визуализация с помощью Python (Pandas/Matplotlib); 4. Проверка результатов и составление отчета. Применяется для исследовательского анализа данных (EDA), мультимодального извлечения данных, построения диаграмм и создания аналитических отчетов.
