フルチェーンのデータ分析機能、テーブル (CSV/Excel)、ドキュメント (PDF/DOCX/MD)、および画像データ処理をサポートします。コア ワークフローは厳密に次のとおりです。 1. 安全な探索: メタデータと小さなサンプルのみを読み取ります。トークン オーバーフローを避けるために完全な読み込みは禁止されています。 2. 品質チェック: 欠損値、重複、一貫性を自動的に検出します。 3. コードの実行: Python (Pandas/Matplotlib) 経由でクリーンアップ、分析、視覚化します。 4. 結果の検証とレポートの生成。適用可能なシナリオ: 探索的データ分析 (EDA)、マルチモーダル データ抽出、グラフのプロット、およびデータ インサイト レポートの生成。
