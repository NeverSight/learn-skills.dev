表形式（CSV/Excel）、ドキュメント（PDF/DOCX/MD）、画像データ処理をサポートするエンドツーエンドのデータ分析スキル。コア ワークフローは次のとおりです。 1. 安全な探索: 読み取り専用のメタデータと小さなサンプル。トークンのオーバーフローを防ぐために完全な読み込みは禁止されています。 2. 品質チェック: 欠損値、重複、一貫性を自動的に検査します。 3. コードの実行: Python (Pandas/Matplotlib) を使用してクリーンアップ、分析、視覚化します。 4. 結果の検証とレポートの生成。適用可能なシナリオ: 探索的分析 (EDA)、マルチモーダル データ抽出、チャート プロット、およびデータ インサイト レポートの生成。
