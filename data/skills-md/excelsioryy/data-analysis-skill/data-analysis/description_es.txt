Habilidades de análisis de datos de cadena completa, manejo de tablas (CSV/Excel), documentos (PDF/DOCX/MD) e imágenes. El flujo de trabajo principal sigue estrictamente lo siguiente: 1. Exploración segura: solo lectura de metadatos y una pequeña muestra, nunca datos completos para evitar el desbordamiento del token; 2. Control de calidad: detección automática de faltantes, duplicados y coherencia; 3. Ejecución de código: limpiar, analizar y visualizar con Python (Pandas/Matplotlib); 4. Validación de resultados y generación de informes. Casos de uso: análisis de datos exploratorios (EDA), extracción multimodal, gráficos e informes de información.
