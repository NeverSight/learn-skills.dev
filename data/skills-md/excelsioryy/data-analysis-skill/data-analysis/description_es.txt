Habilidades de análisis de datos de un extremo a otro que admiten el procesamiento de datos tabulares (CSV/Excel), documentos (PDF/DOCX/MD) e imágenes. El flujo de trabajo principal sigue estrictamente lo siguiente: 1. Exploración segura: metadatos de solo lectura y una pequeña muestra, está prohibida la carga completa para evitar el desbordamiento del token; 2. Control de calidad: inspecciona automáticamente los valores faltantes, los duplicados y la coherencia; 3. Ejecución de código: limpiar, analizar y visualizar usando Python (Pandas/Matplotlib); 4. Validación de resultados y generación de informes. Escenarios aplicables: análisis exploratorio (EDA), extracción de datos multimodal, trazado de gráficos y generación de informes de conocimiento de datos.
