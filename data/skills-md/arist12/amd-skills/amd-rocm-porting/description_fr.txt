Portez les bases de code NVIDIA CUDA vers les GPU AMD ROCm. À utiliser pour faire fonctionner des modèles PyTorch sur des GPU AMD, remplacer les bibliothèques spécifiques à NVIDIA par des équivalents AMD, corriger les échecs de construction/d'exécution ROCm ou porter les noyaux C/C++ CUDA sur HIP. Couvre également le débogage des dépendances et la configuration de l'environnement sur les images ROCm Docker.
