Используйте этот навык, когда вам нужна документация для сторонней библиотеки, SDK или API, прежде чем писать код, который ее использует — например, «использовать OpenAI API», «вызвать Stripe API», «использовать Anthropic SDK», «запросить Pinecone» или в любой момент, когда пользователь просит вас написать код для внешней службы, и вам нужна текущая ссылка на API. Прежде чем ответить, получите документы с помощью голавля, а не полагайтесь на знания, полученные в результате обучения.
