Verwenden Sie diese Fähigkeit, wenn Sie Dokumentation für eine Bibliothek, ein SDK oder eine API eines Drittanbieters benötigen, bevor Sie Code schreiben, der diese verwendet – zum Beispiel „verwenden Sie die OpenAI-API“, „rufen Sie die Stripe-API auf“, „verwenden Sie das Anthropic SDK“, „fragen Sie Pinecone ab“ oder immer dann, wenn der Benutzer Sie auffordert, Code für einen externen Dienst zu schreiben, und Sie eine aktuelle API-Referenz benötigen. Rufen Sie die Dokumente mit Chub ab, bevor Sie antworten, anstatt sich auf Schulungswissen zu verlassen.
