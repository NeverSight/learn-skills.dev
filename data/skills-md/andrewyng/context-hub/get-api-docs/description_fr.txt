Utilisez cette compétence lorsque vous avez besoin de documentation pour une bibliothèque, un SDK ou une API tiers avant d'écrire du code qui l'utilise — par exemple, « utilisez l'API OpenAI », « appelez l'API Stripe », « utilisez le SDK Anthropic », « interrogez Pinecone » ou chaque fois que l'utilisateur vous demande d'écrire du code sur un service externe et que vous avez besoin d'une référence API actuelle.
