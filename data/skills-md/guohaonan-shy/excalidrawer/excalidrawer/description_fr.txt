Génération de diagrammes Excalidraw basée sur le code avec exportation SVG et PNG. À utiliser lorsqu'on vous demande de « créer un diagramme », « dessiner un organigramme », « générer un diagramme d'architecture », « exporter un diagramme vers SVG », « exporter un diagramme vers PNG » ou « visualiser un processus ». Produit des fichiers .excalidraw, .svg et .png à l'aide du package excalidrawer npm — aucun navigateur requis.
