قم بمعالجة كميات كبيرة من طلبات الاستدلال بشكل غير متزامن بتكلفة أقل بنسبة تصل إلى 50% عبر Batch API الخاص بـ Together AI. يدعم ما يصل إلى 50 ألف طلب لكل دفعة، بحد أقصى 100 ميجابايت لحجم الملف. يُستخدم عندما يحتاج المستخدمون إلى معالجة الدفعات، أو الاستدلال دون الاتصال بالإنترنت، أو تصنيف البيانات المجمعة، أو إنشاء البيانات الاصطناعية، أو أعباء عمل LLM واسعة النطاق ومُحسّنة من حيث التكلفة.
