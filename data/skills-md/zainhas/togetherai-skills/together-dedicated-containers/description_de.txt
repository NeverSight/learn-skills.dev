Stellen Sie mithilfe von Dedicated Container Inference (DCI) benutzerdefinierte Docker-Inferenz-Workloads auf der verwalteten GPU-Infrastruktur von Together AI bereit. Zu den Tools gehören Jig CLI zum Erstellen/Bereitstellen, Sprocket SDK zur Anforderungsbearbeitung und eine private Container-Registrierung. Verwenden Sie diese Option, wenn Benutzer benutzerdefinierte Modellbereitstellung, containerisierte Inferenz, Docker-basierte GPU-Workloads oder Workloads über Standardmodellendpunkte hinaus benötigen.
