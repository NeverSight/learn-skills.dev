الدردشة بدون خادم واستدلال إكمال النص عبر واجهة برمجة التطبيقات المتوافقة مع OpenAI الخاصة بـ Together AI. يمكنك الوصول إلى أكثر من 100 نموذج مفتوح المصدر مع تسعير الدفع لكل رمز. يتضمن استدعاء الوظائف (استخدام الأداة) مع 6 أنماط استدعاء، ومخرجات منظمة (وضع JSON، وjson_schema، وregex)، ونماذج التفكير/الاستدلال (DeepSeek R1، وQwen3 Thinking، وKimi K2). يُستخدم عند إنشاء تطبيقات الدردشة، أو إنشاء النص، أو المحادثات متعددة الأدوار، أو استدعاء الوظائف، أو مخرجات JSON المنظمة، أو الاستدلال/سلسلة الأفكار، أو تبديل وضع التفكير، أو أي مهمة استدلال LLM باستخدام Together AI.
