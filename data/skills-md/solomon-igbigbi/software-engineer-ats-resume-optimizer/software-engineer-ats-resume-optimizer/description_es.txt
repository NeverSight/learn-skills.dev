Analiza los currículums de ingeniería de software (backend, frontend o full-stack) para determinar la compatibilidad con ATS, las lagunas de palabras clave y la calidad de las viñetas, generando una puntuación ATS (0–100%), una lista de lagunas de palabras clave, viñetas reescritas y correcciones priorizadas. Utilice esta habilidad de manera proactiva para cualquier ingeniero de software que pregunte cualquier cosa sobre su currículum, incluyendo: compartir un currículum para revisión o comentarios, preguntar qué palabras clave o habilidades agregar para un rol específico, preguntar cómo expresar o enumerar una tecnología o experiencia en su currículum, querer reescribir o mejorar viñetas, solicitar una puntuación ATS o una verificación de aprobado/reprobado, verificar si un currículum coincide con una descripción de trabajo, dirigirse a empresas o roles específicos con un currículum, o decir algo como "¿qué piensas?", "¿qué falta?", "puntuar mi currículum", o "¿puedes ayudarme a mejorar esto?" Invoque esta habilidad incluso cuando aún no se haya pegado ningún currículum; si la pregunta es sobre cómo optimizar un currículum para solicitudes de empleo de ingeniería, esta habilidad se aplica. NO lo use para cartas de presentación, preparación de entrevistas, negociación salarial, perfiles de LinkedIn, portafolios de GitHub ni hojas de ruta de aprendizaje.
