Controlla i repository software per verificarne l'eleganza/pulizia/organizzazione e spiega perché sembrano disordinati o raffinati. Da utilizzare quando un utente chiede se un repository è carino/elegante/pulito, desidera una revisione dell'estetica della base di codice o ha bisogno di passaggi attuabili per migliorare la struttura, la denominazione, la coesione, l'igiene del repository o la documentazione.
