قم بإنشاء مستودعات GitHub احترافية جيدة التنظيم مع أفضل ممارسات الصناعة. يُستخدم عندما: (1) إنشاء مستودعات GitHub جديدة، (2) إعداد بنية المستودع، (3) إضافة الملفات الأساسية (README، LICENSE، .gitignore)، (4) تكوين سير عمل CI/CD، (5) إعداد إرشادات المساهمة، (6) إضافة قوالب المشكلات/العلاقات العامة، (7) تكوين قواعد حماية الفرع، (8) إعداد الوصف والمواضيع للاكتشاف، (9) تحديد مكدس التكنولوجيا والتطبيق المناسب القوالب (React، Next.js، Django، FastAPI، Go، MERN، T3، إلخ.)
