Verwenden Sie diesen Skill, wenn Sie ein VMOS Edge-Android-Cloud-Telefon über die Android Control API steuern. Wenn die aktuelle Sitzung auch „vmos-edge-container-api“ hat, verwenden Sie „host_ip“ und lösen Sie zuerst die Ziel-„db_id“ über die Container-APIs auf. Wenn dieser Skill alleine installiert wird, bevorzugen Sie „cloud_ip“; Wenn „host_ip“ und „db_id“ bereits bekannt sind, kann auch die Host-Route verwendet werden. Wenn keine IP angegeben ist und lokal „cbs_go“ ausgeführt wird, ist „host_ip“ standardmäßig auf „127.0.0.1“ eingestellt.
