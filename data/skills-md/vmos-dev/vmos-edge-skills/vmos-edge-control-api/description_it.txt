Utilizza questa abilità quando controlli un telefono cloud Android VMOS Edge tramite l'API di controllo Android. Se la sessione corrente ha anche `vmos-edge-container-api`, utilizza `host_ip` e risolvi prima il `db_id` di destinazione tramite le API del contenitore. Se questa abilità è installata da sola, preferisci `cloud_ip`; se `host_ip` e `db_id` sono già noti, è possibile utilizzare anche la rotta host. Se non viene fornito alcun IP e `cbs_go` locale è in esecuzione, il valore predefinito di `host_ip` è `127.0.0.1`.
