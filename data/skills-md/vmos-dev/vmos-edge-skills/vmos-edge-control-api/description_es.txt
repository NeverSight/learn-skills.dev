Utilice esta habilidad cuando controle un teléfono en la nube Android VMOS Edge a través de la API de control de Android. Si la sesión actual también tiene `vmos-edge-container-api`, use `host_ip` y resuelva primero el `db_id` de destino a través de las API del contenedor. Si esta habilidad se instala sola, prefiera `cloud_ip`; Si ya se conocen `host_ip` y `db_id`, también se puede utilizar la ruta del host. Si no se proporciona ninguna IP y se está ejecutando `cbs_go` local, el `host_ip` predeterminado es `127.0.0.1`.
