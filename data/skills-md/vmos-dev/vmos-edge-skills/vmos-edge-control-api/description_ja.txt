Android Control API を通じて VMOS Edge Android クラウド フォンを制御する場合は、このスキルを使用します。現在のセッションにも「vmos-edge-container-api」がある場合は、「host_ip」を使用し、最初にコンテナ API を通じてターゲットの「db_id」を解決します。このスキルが単独でインストールされている場合は、「cloud_ip」を優先します。 「host_ip」と「db_id」がすでにわかっている場合は、ホスト ルートも使用できます。 IP が指定されておらず、ローカルの「cbs_go」が実行されている場合、デフォルトの「host_ip」は「127.0.0.1」になります。
