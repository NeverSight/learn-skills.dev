Используйте этот навык при управлении облачным телефоном Android VMOS Edge через Android Control API. Если текущий сеанс также имеет `vmos-edge-container-api`, используйте `host_ip` и сначала разрешите целевой `db_id` через API-интерфейсы контейнера. Если этот навык установлен отдельно, выберите `cloud_ip`; если `host_ip` и `db_id` уже известны, также можно использовать маршрут хоста. Если IP-адрес не указан и локальный `cbs_go` запущен, для `host_ip` по умолчанию используется значение `127.0.0.1`.
