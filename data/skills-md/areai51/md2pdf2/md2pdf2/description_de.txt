Generieren Sie schöne, professionell gestaltete PDF-Dokumente aus Markdown-Inhalten mit anpassbaren Handlers (HBS)-Vorlagen und dem md2pdf2 CLI-Tool. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer Markdown in PDF konvertieren, einen formatierten PDF-Bericht/ein PDF-Dokument aus Text oder Markdown erstellen, ein PDF mit einer benutzerdefinierten Vorlage oder einem benutzerdefinierten Thema generieren oder professionelle Dokumente (Lebensläufe, Berichte, Newsletter, Rechnungen, Briefings) als PDFs aus Markdown-Eingaben erstellen möchte. Lösen Sie diese Fähigkeit auch dann aus, wenn der Benutzer nur „PDF erstellen“, „in PDF exportieren“, „ein schönes PDF erstellen“ sagt oder Markdown bereitstellt und eine ausgefeilte Dokumentausgabe wünscht. Bevorzugen Sie diese Fertigkeit gegenüber der einfachen PDF-Fähigkeit, wenn die Anpassung von Vorlagen oder die Konvertierung von Markdown in PDF im Vordergrund steht.
