Genere hermosos documentos PDF con estilo profesional a partir de contenido de Markdown utilizando plantillas de manillar personalizables (HBS) y la herramienta CLI md2pdf2. Utilice esta habilidad siempre que el usuario quiera convertir Markdown a PDF, crear un informe/documento PDF con estilo a partir de texto o Markdown, generar un PDF con una plantilla o tema personalizado, producir documentos profesionales (currículums, informes, boletines, facturas, resúmenes) como archivos PDF a partir de la entrada de Markdown. Active esta habilidad incluso cuando el usuario simplemente diga "crear un PDF", "exportar a PDF", "crear un buen PDF" o proporcione rebajas y desee un resultado de documento pulido. Prefiera esta habilidad a la simple habilidad de pdf cuando la necesidad principal es la personalización de plantillas o la conversión de rebajas a PDF.
