使用可自訂的 Handlebars (HBS) 範本和 md2pdf2 CLI 工具從 Markdown 內容產生美觀、專業風格的 PDF 文件。每當使用者想要將 Markdown 轉換為 PDF、從文字或 Markdown 建立樣式化的 PDF 報告/文件、使用自訂範本或主題產生 PDF、從 Markdown 輸入將專業文件（履歷、報告、新聞通訊、發票、簡報）產生為 PDF 時，請使用此技能。即使使用者只是說「建立 PDF」、「匯出為 PDF」、「製作漂亮的 PDF」或提供 Markdown 並想要精美的文件輸出，也可以觸發此技能。當模板定製或 Markdown 到 PDF 轉換是核心需求時，優先選擇此技能而不是純 pdf 技能。
