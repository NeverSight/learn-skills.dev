Referencia de PostgreSQL 16+ para escribir consultas, diseñar esquemas, administrar índices y optimizar el rendimiento. Úselo cuando: (1) escriba consultas SQL (SELECCIONAR, INSERTAR, ACTUALIZAR, ELIMINAR, CTE, funciones de ventana, subconsultas), (2) diseñar o modificar esquemas de tablas (CREAR TABLA, restricciones, secuencias, tipos de datos), (3) elegir o crear índices (árbol B, GIN, GiST, BRIN, parcial, expresión), (4) administrar transacciones y niveles de aislamiento (BEGIN/COMMIT, puntos de guardado, MVCC), (5) trabajar con datos JSONB y jsonpath, (6) analizar el rendimiento de las consultas con EXPLAIN ANALYZE o (7) usar la CLI psql.
