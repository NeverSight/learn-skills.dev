À utiliser lorsque vous travaillez avec le package R ggplot2, en particulier les fonctionnalités de ggplot2 4.0+. Couvre la migration S7 (@ accès aux propriétés), les valeurs par défaut du thème avec encre/papier/accent, element_geom(), from_theme(), les raccourcis de thème (theme_sub_*), les thèmes de palette, les étiquettes avec dictionnaire/attributs, les améliorations d'échelle discrètes (palette,continu.limits, minor_breaks, sec.axis), l'esthétique de la position (nudge_x/nudge_y, order), facet_wrap dir/space/layout, boxplot/violin/label style, stat_manual(), stat_connect(), inversion des coordonnées.
