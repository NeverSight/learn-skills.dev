Diseñe sistemas distribuidos, defina la arquitectura para lograr escalabilidad y confiabilidad, o cree documentos de diseño de sistemas. Úselo cuando necesite diagramas de componentes, análisis de flujo de datos, planificación de capacidad, estrategias de fragmentación de bases de datos, diseño de contratos API, análisis de modos de falla, compensaciones de teoremas CAP, migración de monolito a microservicio o registros de decisiones de arquitectura para sistemas nuevos o existentes.
