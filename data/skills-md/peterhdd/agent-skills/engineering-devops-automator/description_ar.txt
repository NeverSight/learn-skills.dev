أتمتة توفير البنية التحتية، وخطوط أنابيب CI/CD، والعمليات السحابية لعمليات نشر موثوقة. استخدمه عندما تحتاج إلى البنية التحتية لـ Terraform كرمز، أو حاوية Docker، أو عمليات النشر باللونين الأزرق والأخضر أو ​​الكناري، أو إعداد المراقبة والتنبيه، أو تجميع السجلات، أو التخطيط للتعافي من الكوارث، أو إدارة الأسرار، أو تحسين التكلفة، أو التكوين متعدد البيئات باستخدام أدوات مثل Vault، أو ELK، أو Loki، أو AWS.
