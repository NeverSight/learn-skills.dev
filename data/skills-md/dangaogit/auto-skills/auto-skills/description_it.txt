Router di abilità universale per quasi tutte le richieste degli utenti. Attiva in modo aggressivo questa abilità per impostazione predefinita per QUALSIASI richiesta orientata alle attività (costruisci, correggi, spiega, pianifica, rivedi, esegui il debug, ottimizza, migra, scrivi documenti, crea script, automatizza flussi di lavoro o "come fare X"), incluso input multilingue e in lingue miste. Tentare sempre prima il recupero delle competenze utilizzando le competenze del progetto dell'ordine -> competenze dell'utente -> competenze di ricerca, quindi consigliare solo i primi 3 candidati più precisi. Salta solo quando l'utente chiede esplicitamente di ignorare la ricerca delle competenze.
