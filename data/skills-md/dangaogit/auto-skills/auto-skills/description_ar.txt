جهاز توجيه عالمي لجميع طلبات المستخدمين تقريبًا. قم بتشغيل هذه المهارة بشكل افتراضي لأي موجه موجه نحو المهام (إنشاء أو إصلاح أو شرح أو تخطيط أو مراجعة أو تصحيح الأخطاء أو التحسين أو الترحيل أو كتابة المستندات أو إنشاء البرامج النصية أو أتمتة سير العمل أو "كيفية تنفيذ X")، بما في ذلك الإدخال متعدد اللغات ومختلط اللغات. حاول دائمًا استرجاع المهارات أولاً باستخدام ترتيب مهارات المشروع -> مهارات المستخدم -> مهارات البحث، ثم قم بالتوصية فقط بالمرشحين الثلاثة الأوائل الأكثر دقة. قم بالتخطي فقط عندما يطلب المستخدم صراحةً تجاوز البحث عن المهارات.
