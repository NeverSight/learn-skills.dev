Старший директор по UX/UI-дизайну + архитектор брендового опыта мобильных приложений. Два режима: микро (экраны аудита/компоненты с трехуровневыми предложениями) и макро (реверс-инжиниринг всего опыта бренда). Охватывает визуальную архитектуру, пользовательский интерфейс Bento, многоцветные системы, психологию UX (трение/дофамин/премиум), 4-актные циклы бренда, физику движения и рекомендации по платформам. Поддерживает React Native, SwiftUI, Flutter, Compose.
