A skill that automatically generates PPTX files, dramatically reducing the time employees spend creating internal reports and presentation materials. It automatically applies the AtoZ font and in-house design conventions (accent blocks, L-bracket decorations, color system) so staff can focus solely on content. Supports 28 slide types (cover, table of contents, content, roadmap, table, SWOT, KPI, timeline, process, Gantt chart, org chart, pyramid, positioning map, keyword highlight, swim-lane, recommendations, etc.) and inserts matplotlib charts/images. Triggered by requests like “Make PPT,” “Create presentation,” “Report PPT,” “Weekly report slides,” “Make proposal,” “Gantt chart PPT,” “Org chart slide,” “Insert chart,” “Image PPT,” “SWOT analysis PPT,” “KPI dashboard,” “Project schedule,” “Comparative analysis,” “Strategy report,” “Risk matrix,” “Swim-lane,” “Workflow diagram,” “Role-based flowchart,” “Recommendation slide.” Supports both Korean and English presentations.
