Используйте агент автоматического запуска для автоматизации анализа, оценки, проектирования и оптимизации целевого проекта. Полный процесс включает в себя: изучение текущего состояния проекта → выявление проблем → составление списка задач → создание среды выполнения → запуск автономного выполнения. Подходит для любого проекта, требующего пакетной оптимизации.
