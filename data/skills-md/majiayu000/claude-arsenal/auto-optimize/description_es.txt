Utilice el agente de ejecución automática para automatizar el análisis, la evaluación, el diseño y la optimización del proyecto objetivo. El flujo de trabajo completo incluye: explorar el estado actual del proyecto → identificar problemas → diseñar la lista de tareas → crear un entorno de ejecución → iniciar la ejecución autónoma. Adecuado para cualquier proyecto que requiera optimización por lotes.
