Analysiert Forschungsarbeiten (PDF/arXiv-URL) und wandelt sie in ausführbaren Code um.
Wird automatisch aktiviert, wenn eine Papiervervielfältigung, Algorithmusimplementierung oder Forschungsreproduktion angefordert wird.
Reagiert auf Anfragen wie „Implementieren Sie dieses Papier“, „paper2code“, „Konvertieren Sie Papier in Code“.
