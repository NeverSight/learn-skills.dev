Рекомендации Nuxt по SEO-оптимизации для приложений, развернутых с помощью Cloudflare. Охватывает генерацию динамических OG-изображений на CF Worker, составных компонентах SEO страниц, структурированных данных JSON-LD, внешних параметрах SSR и конфигурации Nitro. Запускается при выполнении задач, связанных с SEO, метатегами, изображениями OG, Open Graph, Satori, социальным обменом или развертыванием Cloudflare Workers.
