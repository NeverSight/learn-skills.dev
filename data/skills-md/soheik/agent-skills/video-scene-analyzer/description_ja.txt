シーンの境界を検出し、構造化された制作メタデータを JSON として生成することでビデオ ファイルを分析します。ビデオ内のシーンの分析、シーンの変更の検出、シーンレベルのメタデータ (カメラ アングル、色調、テンポ、トランジション、テキスト オーバーレイ) の生成、ビジュアル アセットのカタログ化、またはビデオ コンテンツの構造化された JSON 内訳の生成を求められた場合に使用します。フレーム抽出用のバンドルされた videx スクリプトが含まれています (ffmpeg/ffprobe が必要)。シーンごとのコンテンツの概要、プロダクション/ディレクション タグ、およびクロスシーン アセット ライブラリを含む Analysis.json を出力します。
