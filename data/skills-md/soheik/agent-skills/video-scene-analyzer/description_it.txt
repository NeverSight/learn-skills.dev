Analizza i file video rilevando i confini della scena e generando metadati di produzione strutturati come JSON. Da utilizzare quando richiesto per analizzare le scene in un video, rilevare cambiamenti di scena, generare metadati a livello di scena (angolazioni di ripresa, tonalità di colore, ritmo, transizioni, sovrapposizioni di testo), catalogare risorse visive o produrre una suddivisione JSON strutturata dei contenuti video. Include script videx in bundle per l'estrazione dei frame (richiede ffmpeg/ffprobe). Genera analisi.json con riepiloghi dei contenuti per scena, tag di produzione/direzione e una libreria di risorse multiscena.
