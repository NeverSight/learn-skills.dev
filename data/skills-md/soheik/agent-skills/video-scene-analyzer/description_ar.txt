تحليل ملفات الفيديو عن طريق الكشف عن حدود المشهد وإنشاء بيانات تعريف الإنتاج المنظمة مثل JSON. يُستخدم عندما يُطلب منك تحليل المشاهد في مقطع فيديو، أو اكتشاف تغييرات المشهد، أو إنشاء بيانات تعريف على مستوى المشهد (زوايا الكاميرا، ودرجة اللون، والإيقاع، والانتقالات، وتراكبات النص)، أو كتالوج الأصول المرئية، أو إنتاج تحليل JSON منظم لمحتوى الفيديو. يتضمن برنامج Videx النصي المجمع لاستخراج الإطار (يتطلب ffmpeg/ffprobe). يقوم بإخراج ملف Analysis.json مع ملخصات المحتوى لكل مشهد، وعلامات الإنتاج/الاتجاه، ومكتبة الأصول عبر المشهد.
