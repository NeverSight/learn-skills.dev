Créez et déployez des applications Next.js (App Router) qui se connectent à TiDB. Couvre les gestionnaires de routes (app/api/*/route.ts), la sélection du runtime Node vs Edge pour l'accès à la base de données, la gestion des variables d'environnement et les modèles de base de données sécurisés pour la production sur Vercel/sans serveur. Préférez l'intégration Prisma/Kysely, avec mysql2 en option pour des exemples minimaux. Comprend des guides et des modèles TypeScript.
