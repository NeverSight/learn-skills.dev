Cree e implemente aplicaciones Next.js (App Router) que se conecten a TiDB. Cubre controladores de ruta (app/api/*/route.ts), selección de tiempo de ejecución de nodo frente a borde para acceso a bases de datos, manejo de variables de entorno y patrones de base de datos seguros para producción en Vercel/sin servidor. Prefiera la integración Prisma/Kysely, con mysql2 opcional para ejemplos mínimos. Incluye guías y plantillas de TypeScript.
