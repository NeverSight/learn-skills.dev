Подключитесь к TiDB из JavaScript/Node.js с помощью драйвера mysql2 (mysql2/promise). Используйте для настройки соединения TiDB (TCP), конфигурации TLS/CA для общедоступных конечных точек TiDB Cloud, создания пулов, транзакций и безопасных параметризованных запросов (заполнители выполнения/?), а также небольших исполняемых шаблонов соединений/CRUD.
