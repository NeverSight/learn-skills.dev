Connectez-vous à TiDB depuis JavaScript/Node.js à l'aide du pilote mysql2 (mysql2/promise). À utiliser pour la configuration de la connexion TiDB (TCP), la configuration TLS/CA pour les points de terminaison publics TiDB Cloud, le pooling, les transactions et les requêtes paramétrées sécurisées (exécuter/espaces réservés) ainsi que de petits modèles de connexion/CRUD exécutables.
