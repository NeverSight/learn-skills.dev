Conéctese a TiDB desde JavaScript/Node.js usando el controlador mysql2 (mysql2/promise). Úselo para la configuración de conexión TiDB (TCP), configuración TLS/CA para puntos finales públicos de TiDB Cloud, agrupación, transacciones y consultas parametrizadas seguras (ejecutar/? marcadores de posición), además de pequeñas plantillas CRUD/conexión ejecutables.
