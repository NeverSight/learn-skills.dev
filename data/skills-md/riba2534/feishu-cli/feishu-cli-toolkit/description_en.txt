Feishu Toolbox: spreadsheets, calendar, task management, group chat management, whiteboard actions, PlantUML diagrams, file management, asset upload/download, doc comments, knowledge base, search, user directory. Use when the user requests operations on Feishu sheets, viewing calendars, creating tasks, managing group chats, working on whiteboards, generating PlantUML, managing files, uploading assets, viewing comments, accessing knowledge base, searching messages, looking up users, or querying departments. Covers all feishu-cli functions except document read/write/import/export and permission management.
