Операции со слайдами общедоступного API (X-API-KEY). Создание контуров, обновление плана, получение колоды, пакетное создание слайдов, создание стенограммы/изображения/контента, отмена, тема, стенограмма, дублирование раздела. Типы соответствуют PublicApiSlideDeckController (/api/v2/projects/{projectId}/slide-deck/{slideDeckId}).
