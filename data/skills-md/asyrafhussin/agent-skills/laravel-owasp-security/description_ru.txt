Топ-10 рекомендаций OWASP по аудиту безопасности и безопасному кодированию для приложений Laravel + React/Inertia.js. Используйте при аудите уязвимостей («запустите аудит OWASP», «проверка безопасности», «проверьте безопасность моего приложения») или при написании безопасного кода Laravel, включающего аутентификацию, платежи, загрузку файлов или разработку API. Запускается при выполнении задач, связанных с безопасностью, обработке платежей, аутентификации или любом запросе на аудит базы кода Laravel.
