OWASP أفضل 10 إرشادات للتدقيق الأمني ​​والتشفير الآمن لتطبيقات Laravel + React/Inertia.js. يُستخدم عند التدقيق بحثًا عن الثغرات الأمنية ("تشغيل تدقيق OWASP"، أو "مراجعة الأمان"، أو "التحقق من أمان تطبيقي") أو كتابة تعليمات برمجية Laravel آمنة تتضمن المصادقة، أو المدفوعات، أو تحميل الملفات، أو تصميم واجهة برمجة التطبيقات. يتم تفعيلها عند تنفيذ المهام المتعلقة بالأمان، أو التعامل مع الدفع، أو المصادقة، أو أي طلب لتدقيق قاعدة تعليمات Laravel البرمجية.
