CLI de automatización del navegador antidetección para agentes de IA. Utilícelo cuando el usuario necesite interactuar con sitios web con detección de bots, CAPTCHA o bloques anti-bots, lo que incluye navegar por páginas, completar formularios, hacer clic en botones, tomar capturas de pantalla, extraer datos, probar aplicaciones web o automatizar cualquier tarea del navegador que requiera eludir las comprobaciones de huellas dactilares.
