CLI d'automatisation du navigateur anti-détection pour les agents IA. À utiliser lorsque l'utilisateur a besoin d'interagir avec des sites Web avec détection de robots, CAPTCHA ou blocages anti-bots, notamment pour naviguer dans des pages, remplir des formulaires, cliquer sur des boutons, prendre des captures d'écran, extraire des données, tester des applications Web ou automatiser toute tâche de navigateur nécessitant de contourner les contrôles d'empreintes digitales.
