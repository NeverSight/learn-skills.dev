CLI di automazione del browser anti-rilevamento per agenti AI. Da utilizzare quando l'utente deve interagire con siti Web dotati di rilevamento bot, CAPTCHA o blocchi anti-bot, tra cui navigazione nelle pagine, compilazione di moduli, clic su pulsanti, acquisizione di schermate, estrazione di dati, test di app Web o automazione di qualsiasi attività del browser che richieda il superamento dei controlli delle impronte digitali.
