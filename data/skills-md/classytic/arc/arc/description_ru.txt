@classytic/arc — Ресурсно-ориентированная серверная среда для Fastify.
Используйте при создании REST API с помощью Fastify, ресурса CRUD, defineResource, createApp,
разрешения, пресеты, адаптеры базы данных, перехватчики, события, QueryCache, аутентификация,
мультитенантное SaaS, OpenAPI, очереди заданий, WebSocket или производственное развертывание.
Триггеры: дуга, ресурс fastify, defineResource, createApp, BaseController, предустановка дуги,
аутентификация дуги, события дуги, задания дуги, веб-сокет дуги, плагин дуги, тестирование дуги, Arc Cli,
разрешения дуг, перехватчики дуг, конвейер дуг, фабрика дуг, кэш дуг, QueryCache дуги.
