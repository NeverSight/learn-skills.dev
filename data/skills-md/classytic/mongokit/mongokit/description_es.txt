@classytic/mongokit: patrón de repositorio MongoDB de nivel de producción para Node.js/TypeScript.
Úselo al crear MongoDB CRUD, API REST con Mongoose 9, patrón de repositorio,
paginación, almacenamiento en caché, eliminación temporal, seguimiento de auditoría, multiinquilino, generación de ID personalizada o análisis de consultas.
Activadores: modelo mangosta, patrón de repositorio, mongokit, mongo crud, paginación,
eliminación temporal, seguimiento de auditoría, multiinquilino, identificación personalizada, analizador de consultas, complemento de caché, BaseController.
