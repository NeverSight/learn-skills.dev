@classytic/mongokit: modello di repository MongoDB di livello produttivo per Node.js/TypeScript.
Da utilizzare durante la creazione di MongoDB CRUD, API REST con Mongoose 9, pattern di repository,
impaginazione, memorizzazione nella cache, eliminazione temporanea, traccia di controllo, multi-tenant, generazione di ID personalizzati o analisi di query.
Trigger: modello mongoose, modello di repository, mongokit, mongo crud, impaginazione,
eliminazione temporanea, traccia di controllo, multi-tenant, ID personalizzato, parser di query, plug-in di cache, BaseController.
