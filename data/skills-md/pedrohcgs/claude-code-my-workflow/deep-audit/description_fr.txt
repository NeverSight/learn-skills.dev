Audit de cohérence approfondi de l’ensemble de l’infrastructure du référentiel.
Lance 4 agents spécialisés parallèles pour rechercher des erreurs factuelles, des bugs de code,
les incohérences de comptage et les incohérences entre documents. Puis résout tous les problèmes
et des boucles jusqu'à ce qu'elles soient propres.
À utiliser quand : après avoir apporté des modifications importantes, avant les versions ou lorsque l'utilisateur le demande
"auditer", "trouver les incohérences", "tout vérifier".
