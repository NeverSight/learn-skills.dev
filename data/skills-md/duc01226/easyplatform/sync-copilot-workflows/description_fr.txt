[IA et outils] Synchronisez le catalogue de workflows de workflows.json avec les instructions GitHub Copilot. Exécutez après avoir ajouté/supprimé/modifié des flux de travail pour maintenir copilot-instructions.md à jour. Copilot n'a pas de hooks, cette synchronisation manuelle remplace donc l'injection automatique que Claude Code obtient de workflow-router.cjs.
