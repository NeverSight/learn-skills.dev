[KI & Tools] Synchronisieren Sie den Workflow-Katalog von Workflows.json mit GitHub Copilot-Anweisungen. Wird nach dem Hinzufügen/Entfernen/Ändern von Arbeitsabläufen ausgeführt, um copilot-instructions.md auf dem neuesten Stand zu halten. Copilot hat keine Hooks, daher ersetzt diese manuelle Synchronisierung die automatische Injektion, die Claude Code von Workflow-router.cjs erhält.
