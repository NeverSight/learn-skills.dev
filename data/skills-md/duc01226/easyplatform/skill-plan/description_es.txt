[Herramientas y meta] Guía para crear habilidades efectivas, agregar referencias de habilidades, guiones de habilidades u optimizar habilidades existentes. Esta habilidad debe usarse cuando los usuarios desean crear una nueva habilidad (o actualizar una habilidad existente) que amplíe las capacidades de Claude con conocimiento especializado, flujos de trabajo, marcos, bibliotecas o uso de complementos, o integraciones de API y herramientas.
