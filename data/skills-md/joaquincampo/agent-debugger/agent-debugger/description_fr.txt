À utiliser lorsqu'un programme plante, qu'un test échoue ou que le code produit des résultats erronés et que la lecture de la source ne suffit pas pour comprendre pourquoi. Vous permet de suspendre l'exécution sur n'importe quelle ligne et d'inspecter l'état d'exécution réel, les valeurs des variables, les types, les piles d'appels, pour trouver ce qui n'a pas fonctionné.
