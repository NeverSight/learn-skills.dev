Verwenden Sie es, wenn ein Programm abstürzt, ein Test fehlschlägt oder Code falsche Ergebnisse liefert und das Lesen der Quelle nicht ausreicht, um herauszufinden, warum. Ermöglicht Ihnen, die Ausführung in einer beliebigen Zeile anzuhalten und den tatsächlichen Laufzeitstatus, Variablenwerte, Typen und Aufrufstapel zu überprüfen, um herauszufinden, was schief gelaufen ist.
