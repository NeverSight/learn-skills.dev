창의적인 브레인스토밍, 사용자 피드백, 생태계를 통해 새로운 기능 기회를 찾아보세요.
동향 및 프로젝트 간 연구. .feature-radar/opportunities/에 결과를 씁니다.
사용자가 기존 아이디어를 평가하는 것이 아니라 새로운 아이디어를 생성하고 싶을 때 이 기술을 사용해야 합니다.
브레인스토밍, 탐색, 발견 또는 새로운 기능 아이디어 찾기에 대한 요청을 트리거합니다.
"우리가 또 무엇을 할 수 있을지 궁금합니다" 또는 "아이디어를 주세요"와 같은 캐주얼한 것입니다.
사용자가 다음과 같은 경우에 사용하세요.
- "우리가 또 무엇을 만들 수 있을까요?", "기능 아이디어를 주세요", "우리가 놓치고 있는 것이 무엇입니까?"라고 묻습니다.
- 브레인스토밍하고, 새로운 방향을 탐색하고, 기회 백로그를 새로 고치고 싶은 분
- "생태계 스캔", "기회 스캔", "새로운 기능 찾기"라고 말합니다.
- 영감을 얻기 위해 GitHub 문제, 커뮤니티 피드백 또는 인접 도구를 검토하도록 요청합니다.
- 기능 맥락에서 '탐색', '발견' 또는 '새로운 방향'을 언급합니다.
- 막연한 생각이 있음: "나한테 아이디어가 있어", "우리가...", "내가 생각하고 있던 건..."
기존 기능을 평가/우선순위로 지정하는 데 사용하지 마십시오. 이는 기능 레이더의 작업입니다.
