Usar cuando el usuario:
- Dice "recordar este enfoque", "documentar esta decisión", "guardar este patrón"
- Reflexiona: “eso funcionó bien”, “lecciones aprendidas”, “qué aprendimos”
- Quiere capturar: "no olvides esto", "este fue un buen patrón"
- Llegar a un callejón sin salida: "esto no funcionó porque...", "evite este enfoque"
- Tomó una decisión técnica que vale la pena registrar para la posteridad.
- Dice "extraer aprendizajes" o "capturar lo que aprendimos"
