사용자가 다음과 같은 경우에 사용하세요.
- "이 접근법을 기억하세요", "이 결정을 문서화하세요", "이 패턴을 저장하세요"라고 말합니다.
- 반영: "잘 작동했습니다.", "배운 교훈", "우리는 무엇을 배웠습니까?"
- 캡쳐하고 싶은 말 : "잊지 마세요", "좋은 패턴이었어요"
- 막다른 골목에 도달: "이것은 작동하지 않았습니다. 왜냐하면...", "이 접근 방식을 피하십시오"
- 후세에 기록할 가치가 있는 기술적 결정을 내렸습니다.
- "학습 내용 추출" 또는 "배운 내용 캡처"라고 말합니다.
