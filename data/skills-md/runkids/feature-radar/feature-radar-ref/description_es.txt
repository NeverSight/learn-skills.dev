Incluso las menciones casuales como "Vi algo interesante en X" deberían activar esta habilidad.
Usar cuando el usuario:
- Dice "Vi algo interesante en la API de X", "mira cómo X maneja esto".
- Comparte una URL, un artículo, una charla o un hallazgo de investigación con información relevante
- Observa una tendencia del ecosistema: nuevas herramientas, estándares, patrones comunitarios.
- Menciona un proyecto relacionado que incluye una característica notable.
- Quiere marcar inspiración externa como favorito: "enfoque interesante", "guardar esto"
- Dice "agregar referencia", "registrar observación", "seguir este proyecto"
NO lo use para aprendizajes/patrones internos; ese es el trabajo de feature-radar-learn.
NO lo use para priorizar funciones; ese es el trabajo del radar de funciones.
