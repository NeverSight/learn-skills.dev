將外部觀察、生態系統趨勢和創意靈感記錄到
.feature-radar/references/。當使用者提到某事時必須使用此技能
來自專案之外的有趣內容——其他工具、文章、方法或趨勢。
即使是隨意提及「我在 X 中看到了一個很酷的東西」也應該觸發此技能。
當使用者出現以下情況時使用：
- 說“我在 X 的 API 中看到了這個很酷的東西”，“看看 X 如何處理這個”
- 分享具有相關見解的 URL、文章、談話或研究結果
- 注意到生態系趨勢：新工具、標準、社群模式
- 提到一個具有顯著功能的相關項目
- 想要為外部靈感添加書籤：“有趣的方法”、“保存這個”
- 說“添加參考”、“記錄觀察”、“跟踪該項目”
不要用於內部學習/模式—這是功能雷達學習的工作。
不要用於確定功能的優先順序——這是功能雷達的工作。
