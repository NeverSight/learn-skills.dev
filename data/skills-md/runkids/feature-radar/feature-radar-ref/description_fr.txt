Enregistrez les observations externes, les tendances de l'écosystème et l'inspiration créative dans
.feature-radar/references/. DOIT utiliser cette compétence lorsque l'utilisateur mentionne quelque chose
intéressant de l’extérieur de leur projet – d’autres outils, articles, approches ou tendances.
Même des mentions occasionnelles comme « J'ai vu un truc sympa dans X » devraient déclencher cette compétence.
À utiliser lorsque l'utilisateur :
- Dit "J'ai vu ce truc sympa dans l'API de X", "regardez comment X gère ça"
- Partage une URL, un article, une discussion ou un résultat de recherche avec des informations pertinentes
- Note une tendance de l'écosystème : nouveaux outils, normes, modèles de communauté
- Mentionne un projet connexe proposant une fonctionnalité notable
- Veut mettre en favoris une inspiration externe : "approche intéressante", "enregistrer ceci"
- Dit "ajouter une référence", "enregistrer l'observation", "suivre ce projet"
Ne PAS utiliser pour des apprentissages/modèles internes - c'est le travail de feature-radar-learn.
Ne PAS utiliser pour hiérarchiser les fonctionnalités - c'est le travail de Feature-Radar.
