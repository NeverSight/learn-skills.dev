将外部观察、生态系统趋势和创意灵感记录到
.feature-radar/references/。当用户提到某事时必须使用此技能
来自项目之外的有趣内容——其他工具、文章、方法或趋势。
即使是随意提及“我在 X 中看到了一个很酷的东西”也应该触发此技能。
当用户出现以下情况时使用：
- 说“我在 X 的 API 中看到了这个很酷的东西”，“看看 X 如何处理这个”
- 分享具有相关见解的 URL、文章、谈话或研究结果
- 注意到生态系统趋势：新工具、标准、社区模式
- 提到一个具有显着功能的相关项目
- 想要为外部灵感添加书签：“有趣的方法”、“保存这个”
- 说“添加参考”、“记录观察”、“跟踪该项目”
不要用于内部学习/模式——这是功能雷达学习的工作。
不要用于确定功能的优先级——这是功能雷达的工作。
