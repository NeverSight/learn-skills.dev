.feature-radar/references/. 사용자가 무언가를 언급할 때 이 스킬을 사용해야 합니다.
프로젝트 외부의 흥미로운 도구, 기사, 접근 방식 또는 동향.
"X에서 멋진 걸 봤어"와 같은 일상적인 언급도 이 기술을 발동해야 합니다.
사용자가 다음과 같은 경우에 사용하세요.
- "X의 API에서 이 멋진 것을 봤어요", "X가 이것을 어떻게 처리하는지 확인해 보세요"라고 말합니다.
- 관련 통찰력이 있는 URL, 기사, 대화 또는 연구 결과를 공유합니다.
- 생태계 동향 기록: 새로운 도구, 표준, 커뮤니티 패턴
- 주목할 만한 기능을 제공하는 관련 프로젝트를 언급합니다.
- 외부 영감을 북마크에 추가하고 싶음: "흥미로운 접근 방식", "이것을 저장"
- "참조 추가", "로그 관찰", "이 프로젝트 추적"이라고 말합니다.
내부 학습/패턴에는 사용하지 마십시오. 이는 feature-radar-learn의 작업입니다.
기능의 우선순위를 지정하는 데 사용하지 마십시오. 이는 기능 레이더의 작업입니다.
