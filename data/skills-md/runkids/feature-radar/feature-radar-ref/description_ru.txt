Записывайте внешние наблюдения, тенденции экосистем и творческое вдохновение в
.feature-радар/ссылки/. ДОЛЖЕН использовать этот навык, когда пользователь что-то упоминает
интересен за пределами их проекта — другие инструменты, статьи, подходы или тенденции.
Даже случайное упоминание типа «Я видел классную вещь в X» должно активировать этот навык.
Используйте, когда пользователь:
- Говорит: «Я видел эту классную штуку в API X», «посмотрите, как X с этим справляется».
- Делится URL-адресом, статьей, докладом или результатами исследования с соответствующей информацией.
- Отмечает тенденцию экосистемы: новые инструменты, стандарты, модели сообщества.
- Упоминает связанный проект с примечательной особенностью.
- Хочет добавить в закладки внешнее вдохновение: «интересный подход», «сохраните это».
- Говорит "добавить ссылку", "записать в журнал наблюдения", "отследить этот проект".
НЕ используйте для внутреннего обучения/шаблонов — это работа Feature-Radar-Learn.
НЕ используйте для определения приоритета функций — это работа радара функций.
