Riepiloga gli AIP di Google per la progettazione e l'utilizzo di buffer di protocollo, RPC, servizi, messaggi, campi e denominazione. Da utilizzare durante la creazione o la revisione di file .proto, la progettazione di API orientate alle risorse, l'allineamento con google.aip.dev e API Linter o il controllo della denominazione e dei modelli di richiesta/risposta per metodi standard.
