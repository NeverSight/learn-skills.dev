Загрузите локальные изображения в PR GitHub и вставьте их в описание или комментарии. Используйте, когда вас просят «прикрепить скриншоты к PR», «добавить изображения в PR», «загрузить результаты теста в PR», «встроить скриншоты в описание PR», «добавить изображения до/после в PR», «прикрепить скриншоты пользовательского интерфейса», «показать результаты теста в PR», «добавить визуальные доказательства в PR» или при любом запросе, связанном с изображениями и PR. Всегда используйте этот навык, когда пользователь хочет визуально задокументировать изменения в запросе на включение, даже если он не использует слово «загрузить» — такие фразы, как «поместить скриншот в PR» или «показать изображение в PR» должны активировать этот навык. Поддерживает Playwright MCP / Chrome DevTools MCP / агент-браузер в качестве серверной части автоматизации браузера.
