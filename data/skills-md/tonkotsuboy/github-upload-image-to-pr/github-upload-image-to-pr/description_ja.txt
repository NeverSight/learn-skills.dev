ローカル画像を GitHub PR にアップロードし、説明またはコメントに埋め込みます。 「PR にスクリーンショットを添付する」、「PR に画像を追加する」、「PR にテスト結果をアップロードする」、「PR の説明にスクリーンショットを埋め込む」、「PR に前後の画像を追加する」、「UI スクリーンショットを添付する」、「PR にテスト結果を表示する」、「PR に視覚的証拠を追加する」、または画像と PR に関連するリクエストを求められた場合に使用します。ユーザーがプル リクエストの変更を視覚的に文書化したい場合は、「アップロード」という単語を使用しない場合でも、常にこのスキルを使用してください。「PR にスクリーンショットを入れる」または「PR に画像を表示する」などのフレーズがこのスキルをトリガーする必要があります。ブラウザ自動化バックエンドとして Playwright MCP / Chrome DevTools MCP / エージェント ブラウザをサポートします。
