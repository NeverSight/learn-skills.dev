Téléchargez des images locales sur un PR GitHub et intégrez-les dans la description ou les commentaires. À utiliser lorsqu'on vous demande de « joindre des captures d'écran au PR », « ajouter des images au PR », « télécharger les résultats des tests dans le PR », « intégrer des captures d'écran dans la description du PR », « ajouter des images avant/après au PR », « joindre des captures d'écran de l'interface utilisateur », « afficher les résultats des tests dans le PR », « ajouter des preuves visuelles au PR » ou toute demande impliquant des images et des PR. Utilisez toujours cette compétence lorsque l'utilisateur souhaite documenter visuellement les modifications dans une demande d'extraction, même s'il n'utilise pas le mot « télécharger » — des expressions telles que « mettre la capture d'écran dans le PR » ou « afficher l'image dans le PR » devraient déclencher cette compétence. Prend en charge Playwright MCP / Chrome DevTools MCP / agent-browser en tant que backends d'automatisation du navigateur.
