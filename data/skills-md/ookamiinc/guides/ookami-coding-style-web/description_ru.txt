Соглашения о стиле кодирования веб-интерфейса команды ookami. Используйте этот навык при написании, изменении или просмотре JavaScript, JSX, SCSS, компонентов React, перехватчиков, селекторов, контейнеров или файлов тестирования внешнего интерфейса. Также используйте, когда пользователь спрашивает о структуре компонента React, соглашениях об атомарном дизайне, модулях CSS, форматировании Prettier, использовании руководства по стилю Airbnb, именовании файлов для .jsx/.js/.scss или о том, как структурировать тесты внешнего интерфейса с помощью описания/it.
