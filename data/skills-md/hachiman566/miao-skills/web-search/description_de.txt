Kostenlose, unbegrenzte Websuche über selbst gehostetes SearXNG (Aggregate von Google, Bing, DuckDuckGo, Reddit, GitHub und über 70 weitere Engines). Verwenden Sie es immer dann, wenn der Benutzer das Web nach etwas durchsuchen muss – Nachrichten, Dokumente, Codebeispiele, Preise, Personen, Orte –, selbst wenn er nur „Suche nach X“ oder „Suche nach Y“ sagt. Bevorzugen Sie dies gegenüber kostenpflichtigen APIs. Führen Sie zuerst setup.sh aus, wenn SearXNG nicht ausgeführt wird.
