Ricerca web illimitata gratuita tramite SearXNG self-hosted (aggrega Google, Bing, DuckDuckGo, Reddit, GitHub e oltre 70 motori). Utilizzalo ogni volta che l'utente ha bisogno di cercare qualcosa sul Web (notizie, documenti, esempi di codici, prezzi, persone, luoghi), anche se dice semplicemente "cerca X" o "cerca Y". Preferisci questo alle API a pagamento. Esegui prima setup.sh se SearXNG non è in esecuzione.
