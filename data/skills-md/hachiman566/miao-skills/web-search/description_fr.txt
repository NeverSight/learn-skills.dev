Recherche Web gratuite et illimitée via SearXNG auto-hébergé (regroupe Google, Bing, DuckDuckGo, Reddit, GitHub et plus de 70 moteurs supplémentaires). À utiliser chaque fois que l'utilisateur a besoin de rechercher quoi que ce soit sur le Web (actualités, documents, exemples de code, prix, personnes, lieux), même s'il dit simplement « rechercher X » ou « rechercher Y ». Préférez cela aux API payantes. Exécutez d'abord setup.sh si SearXNG n'est pas en cours d'exécution.
