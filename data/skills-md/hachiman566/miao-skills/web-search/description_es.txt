Búsqueda web gratuita e ilimitada a través de SearXNG autohospedado (agrega Google, Bing, DuckDuckGo, Reddit, GitHub y más de 70 motores más). Úselo siempre que el usuario necesite buscar cualquier cosa en la web (noticias, documentos, ejemplos de código, precios, personas, lugares), incluso si solo dice "buscar X" o "buscar Y". Prefiere esto a las API pagas. Ejecute setup.sh primero si SearXNG no se está ejecutando.
