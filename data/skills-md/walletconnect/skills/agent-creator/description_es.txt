Guía para crear subagentes personalizados de Claude Code. Úselo cuando el usuario desee crear un nuevo agente (o actualizar un agente existente) que maneje tipos específicos de tareas con indicaciones personalizadas, restricciones de herramientas y permisos. Se activa ante solicitudes para crear agentes, subagentes, agentes personalizados o cuando el usuario desea asistentes de IA especializados para flujos de trabajo específicos.
