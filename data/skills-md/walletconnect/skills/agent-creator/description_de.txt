Leitfaden zum Erstellen benutzerdefinierter Claude Code-Subagenten. Verwenden Sie diese Option, wenn der Benutzer einen neuen Agenten erstellen (oder einen vorhandenen Agenten aktualisieren) möchte, der bestimmte Arten von Aufgaben mit benutzerdefinierten Eingabeaufforderungen, Tooleinschränkungen und Berechtigungen verarbeitet. Wird bei Anfragen zum Erstellen von Agenten, Subagenten oder benutzerdefinierten Agenten ausgelöst oder wenn der Benutzer spezielle KI-Assistenten für bestimmte Arbeitsabläufe benötigt.
