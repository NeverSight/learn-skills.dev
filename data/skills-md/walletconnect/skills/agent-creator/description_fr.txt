Guide de création de sous-agents Claude Code personnalisés. À utiliser lorsque l'utilisateur souhaite créer un nouvel agent (ou mettre à jour un agent existant) qui gère des types spécifiques de tâches avec des invites personnalisées, des restrictions d'outils et des autorisations. Se déclenche sur les demandes de création d'agents, de sous-agents, d'agents personnalisés ou lorsque l'utilisateur souhaite des assistants IA spécialisés pour des flux de travail spécifiques.
