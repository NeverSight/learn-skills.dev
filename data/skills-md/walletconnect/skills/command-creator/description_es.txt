Guía para crear comandos de barra personalizados de Claude Code. Úselo cuando el usuario quiera crear un nuevo comando (o actualizar un comando existente) que proporcione un fragmento de mensaje, un flujo de trabajo o una automatización reutilizables. Se activa en solicitudes para crear comandos /, comandos de barra diagonal, comandos personalizados o cuando el usuario desea definir mensajes de uso frecuente como comandos reutilizables.
