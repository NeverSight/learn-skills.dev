Anleitung zum Erstellen benutzerdefinierter Claude Code-Slash-Befehle. Verwenden Sie diese Option, wenn der Benutzer einen neuen Befehl erstellen (oder einen vorhandenen Befehl aktualisieren) möchte, der ein wiederverwendbares Eingabeaufforderungs-Snippet, einen wiederverwendbaren Workflow oder eine wiederverwendbare Automatisierung bereitstellt. Wird bei Anfragen zum Erstellen von /commands, Slash-Befehlen, benutzerdefinierten Befehlen oder wenn der Benutzer häufig verwendete Eingabeaufforderungen als wiederverwendbare Befehle definieren möchte, ausgelöst.
