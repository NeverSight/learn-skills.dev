Guide pour créer des commandes slash Claude Code personnalisées. À utiliser lorsque l'utilisateur souhaite créer une nouvelle commande (ou mettre à jour une commande existante) qui fournit un extrait d'invite, un flux de travail ou une automatisation réutilisable. Se déclenche sur les demandes de création de commandes /, de commandes slash, de commandes personnalisées ou lorsque l'utilisateur souhaite définir des invites fréquemment utilisées en tant que commandes réutilisables.
