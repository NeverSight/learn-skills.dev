دليل لإنشاء أوامر شرطة مائلة مخصصة لـ Claude Code. يُستخدم عندما يريد المستخدم إنشاء أمر جديد (أو تحديث أمر موجود) يوفر مقتطف مطالبة قابلاً لإعادة الاستخدام أو سير عمل أو أتمتة. يتم تشغيله عند طلبات إنشاء /commands، أو أوامر الشرطة المائلة، أو الأوامر المخصصة، أو عندما يريد المستخدم تحديد المطالبات المستخدمة بشكل متكرر كأوامر قابلة لإعادة الاستخدام.
