Руководство по созданию пользовательских слэш-команд Claude Code. Используйте, когда пользователь хочет создать новую команду (или обновить существующую команду), которая предоставляет повторно используемый фрагмент приглашения, рабочий процесс или автоматизацию. Запускается по запросам на создание /commands, косой черты, пользовательских команд или когда пользователь хочет определить часто используемые запросы как команды многократного использования.
