واجهة برمجة تطبيقات الطريقة الكاملة وأدلة القرار والمزالق والأنماط الخاصة بمجموعة Laravel وLazyCollection. يُستخدم عند اختيار طريقة التجميع الصحيحة، أو كتابة خطوط التجميع أو مراجعتها، أو العمل مع مجموعات بيانات كبيرة عبر المؤشر/الكسل. المشغلات عند التجميع ()، أو التجميع، أو LazyCollection، أو الخريطة، أو التصفية، أو التصغير، أو الالتقاط، أو groupBy، أو كل مؤشر، أو تحويلات المصفوفة بطلاقة في Laravel.
