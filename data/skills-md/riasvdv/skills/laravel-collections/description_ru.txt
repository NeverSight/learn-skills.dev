Полный API методов, руководства по принятию решений, подводные камни и шаблоны для Laravel Collection и LazyCollection. Используйте при выборе правильного метода сбора, написании или просмотре конвейеров сбора, а также при работе с большими наборами данных с помощью курсора/ленивого режима. Триггеры по сбору(), коллекции, LazyCollection, карте, фильтрованию, уменьшению, выдергиванию, группировке, каждому, курсору или плавным преобразованиям массива в Laravel.
