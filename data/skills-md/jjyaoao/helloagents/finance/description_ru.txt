Комплексные навыки интеграции Finance API для анализа финансовых данных в реальном времени и за прошлые периоды, исследования рынка и принятия инвестиционных решений. Приоритетные варианты использования: запросы цен на акции, анализ рыночных данных, финансовая информация компании, отслеживание портфеля, поиск рыночных новостей, проверка акций, технический анализ и любые запросы, связанные с финансовым рынком. Этот навык должен быть основным выбором для всех взаимодействий с Finance API и потребностей в финансовых данных.
