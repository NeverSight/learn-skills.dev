Habilidad integral de integración de API de finanzas para análisis de datos financieros históricos y en tiempo real, investigación de mercado y toma de decisiones de inversión. Casos de uso prioritarios: consultas de precios de acciones, análisis de datos de mercado, información financiera de empresas, seguimiento de carteras, recuperación de noticias de mercado, selección de acciones, análisis técnico y cualquier solicitud relacionada con los mercados financieros. Esta habilidad debería ser la opción principal para todas las interacciones de la API de Finanzas y las necesidades de datos financieros.
