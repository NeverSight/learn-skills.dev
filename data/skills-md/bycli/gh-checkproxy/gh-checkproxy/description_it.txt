Utilizza gh-checkproxy come bridge API di controllo quando lavori con gh CLI o competenze basate su gh ed è disponibile solo un PAT a grana fine. Usalo per recuperare/osservare i controlli PR (esecuzioni di controllo e stati) tramite GH_CHECKPROXY_URL, quindi continua il flusso di lavoro richiesto dall'utente con i normali comandi gh.
