Используйте gh-checkproxy в качестве моста API проверок при работе с gh CLI или навыками на основе gh, и доступен только детальный PAT. Используйте его для получения/просмотра проверок PR (проверок и статусов) через GH_CHECKPROXY_URL, а затем продолжайте запрошенный пользователем рабочий процесс с помощью обычных команд gh.
