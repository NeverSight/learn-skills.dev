Utilice gh-checkproxy como puente API de comprobaciones cuando trabaje con gh CLI o habilidades basadas en gh y solo esté disponible una PAT detallada. Úselo para buscar/ver verificaciones de relaciones públicas (ejecuciones de verificación y estados) a través de GH_CHECKPROXY_URL, luego continúe con el flujo de trabajo solicitado por el usuario con los comandos normales de gh.
