Полный рабочий процесс для преобразования корейского/английского академического опроса Markdown в PDF-файлы LaTeX публикационного качества. Управляет очисткой артефактов Obsidian (вики-ссылки, русалка), компиляцией XeLaTeX с поддержкой корейских шрифтов (ucharclasses + Noto Sans CJK KR), цитатами BibTeX, генерацией фигур TikZ, блоками примеров tcolorbox и итеративным просмотром формата. Поддерживает структуру папок на основе разделов с оркестратором main.tex и управлением BibTeX refs.bib. Используйте этот навык всякий раз, когда пользователь хочет создать PDF-файл из академического файла уценки, составить обзорный документ LaTeX или исправить проблемы с форматированием в существующем академическом PDF-файле.
