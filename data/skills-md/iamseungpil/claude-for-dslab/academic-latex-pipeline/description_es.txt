Flujo de trabajo completo para convertir Markdown de encuestas académicas en coreano/inglés en archivos PDF LaTeX con calidad de publicación. Maneja la limpieza de artefactos de Obsidian (wikilinks, mermaid), compilación de XeLaTeX con soporte de fuentes coreanas (ucharclasses + Noto Sans CJK KR), citas de BibTeX, generación de figuras TikZ, cuadros de ejemplo de tcolorbox y revisión de formato iterativo. Admite una estructura de carpetas basada en secciones con el orquestador main.tex y la administración BibTeX refs.bib. Utilice esta habilidad siempre que el usuario quiera crear un PDF a partir de un archivo de rebajas académico, compilar un trabajo de encuesta LaTeX o solucionar problemas de formato en un PDF académico existente. DISPARADORES OBLIGATORIOS: encuesta LaTeX, PDF académico, 서베이 PDF, 논문 PDF 빌드, xelatex, build_latex.py, artículo académico coreano, compilación de encuestas
