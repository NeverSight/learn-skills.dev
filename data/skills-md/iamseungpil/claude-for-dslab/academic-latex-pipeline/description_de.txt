Vollständiger Workflow zum Konvertieren von Markdown für akademische Umfragen auf Koreanisch/Englisch in LaTeX-PDFs in Publikationsqualität. Behandelt die Bereinigung von Obsidian-Artefakten (Wikilinks, Meerjungfrau), die XeLaTeX-Kompilierung mit Unterstützung für koreanische Schriftarten (ucharclasses + Noto Sans CJK KR), BibTeX-Zitate, die Generierung von TikZ-Figuren, Tcolorbox-Beispielboxen und die iterative Formatüberprüfung. Unterstützt abschnittsbasierte Ordnerstruktur mit main.tex Orchestrator und BibTeX refs.bib-Verwaltung. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine PDF-Datei aus einer akademischen Markdown-Datei erstellen, eine LaTeX-Umfragearbeit zusammenstellen oder Formatierungsprobleme in einer vorhandenen akademischen PDF-Datei beheben möchte.
