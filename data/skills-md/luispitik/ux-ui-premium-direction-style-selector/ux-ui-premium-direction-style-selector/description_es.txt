Implemente 8 demostraciones de dirección visual radicalmente diferentes para un sitio web o proyecto de aplicación web. Crea páginas Next.js independientes en /demo/ con identidades distintas (tipografía, paleta de colores, diseño, animación, accesibilidad) para que el usuario pueda comparar y elegir. Después de la selección, genera tokens de tema listos para producción, configuración de Tailwind, variables CSS y componentes base que coinciden con la dirección elegida. Úselo cuando: iniciar un nuevo proyecto web, rediseñar un sitio, elegir la dirección visual, el usuario dice "selector de estilo", "elegir estilos", "alternativas visuales", "elegir una dirección de diseño" o quiere comparar diferentes enfoques estéticos antes de comprometerse con uno. Funciona con proyectos Next.js + Tailwind CSS (páginas de destino, aplicaciones web, SaaS, paneles).
