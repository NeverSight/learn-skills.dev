Web サイトまたは Web アプリ プロジェクト用に、8 つの根本的に異なるビジュアル ディレクション デモを展開します。ユーザーが比較して選択できるように、明確なアイデンティティ (タイポグラフィ、カラー パレット、レイアウト、アニメーション、アクセシビリティ) を持つ自己完結型 Next.js ページを /demo/ の下に作成します。選択後、選択した方向に一致する実稼働対応のテーマ トークン、Tailwind 構成、CSS 変数、および基本コンポーネントを生成します。新しい Web プロジェクトの開始、サイトの再デザイン、視覚的な方向の選択、ユーザーが「スタイル セレクター」、「スタイルの選択」、「視覚的な代替案」、「デザインの方向性の選択」と言う場合、または 1 つにコミットする前にさまざまな美的アプローチを比較したい場合に使用します。 Next.js + Tailwind CSS プロジェクト (ランディング ページ、Web アプリ、SaaS、ダッシュボード) で動作します。
