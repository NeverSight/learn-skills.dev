Distribuisci 8 demo di direzione visiva radicalmente diverse per un sito web o un progetto webapp. Crea pagine Next.js autonome in /demo/ con identità distinte (tipografia, tavolozza dei colori, layout, animazione, accessibilità) in modo che l'utente possa confrontare e scegliere. Dopo la selezione, genera token del tema pronti per la produzione, configurazione Tailwind, variabili CSS e componenti di base corrispondenti alla direzione scelta. Utilizzare quando: avviando un nuovo progetto web, riprogettando un sito, scegliendo la direzione visiva, l'utente dice "selettore di stile", "scegli stili", "alternative visive", "scegli una direzione di progettazione" o desidera confrontare diversi approcci estetici prima di impegnarsi in uno. Funziona con progetti Next.js + Tailwind CSS (pagine di destinazione, app Web, SaaS, dashboard).
