انشر 8 عروض توضيحية للاتجاهات المرئية مختلفة جذريًا لموقع ويب أو مشروع تطبيق ويب. ينشئ صفحات Next.js قائمة بذاتها ضمن /demo/ بهويات مميزة (الطباعة ولوحة الألوان والتخطيط والرسوم المتحركة وإمكانية الوصول) حتى يتمكن المستخدم من المقارنة والاختيار. بعد التحديد، يتم إنشاء رموز مميزة جاهزة للإنتاج وتكوين Tailwind ومتغيرات CSS والمكونات الأساسية المطابقة للاتجاه المختار. يُستخدم عندما: بدء مشروع ويب جديد، أو إعادة تصميم موقع، أو اختيار الاتجاه المرئي، أو عندما يقول المستخدم "محدد النمط"، أو "اختر الأنماط"، أو "البدائل المرئية"، أو "اختر اتجاه التصميم"، أو عندما يريد مقارنة الأساليب الجمالية المختلفة قبل الالتزام بأحدها. يعمل مع مشاريع Next.js + Tailwind CSS (الصفحات المقصودة وتطبيقات الويب وSaaS ولوحات المعلومات).
