Déployez 8 démos de direction visuelle radicalement différentes pour un projet de site Web ou d'application Web. Crée des pages Next.js autonomes sous /demo/ avec des identités distinctes (typographie, palette de couleurs, mise en page, animation, accessibilité) afin que l'utilisateur puisse comparer et choisir. Après la sélection, génère des jetons de thème prêts pour la production, une configuration Tailwind, des variables CSS et des composants de base correspondant à la direction choisie. À utiliser lorsque : démarrer un nouveau projet Web, repenser un site, choisir une direction visuelle, l'utilisateur dit "sélecteur de style", "choisir des styles", "alternatives visuelles", "choisir une direction de conception" ou souhaite comparer différentes approches esthétiques avant de s'engager dans une. Fonctionne avec les projets CSS Next.js + Tailwind (pages de destination, applications Web, SaaS, tableaux de bord).
