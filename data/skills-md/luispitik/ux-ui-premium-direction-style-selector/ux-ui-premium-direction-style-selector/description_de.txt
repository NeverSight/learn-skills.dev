Stellen Sie 8 völlig unterschiedliche visuelle Richtungsdemos für ein Website- oder Webanwendungsprojekt bereit. Erstellt eigenständige Next.js-Seiten unter /demo/ mit unterschiedlichen Identitäten (Typografie, Farbpalette, Layout, Animation, Barrierefreiheit), damit der Benutzer vergleichen und auswählen kann. Nach der Auswahl werden produktionsbereite Theme-Tokens, Tailwind-Konfiguration, CSS-Variablen und Basiskomponenten generiert, die der gewählten Richtung entsprechen. Verwenden Sie diese Option, wenn Sie ein neues Webprojekt starten, eine Website neu gestalten, eine visuelle Richtung wählen, der Benutzer „Stilauswahl“, „Stile auswählen“, „visuelle Alternativen“, „eine Designrichtung auswählen“ sagt oder verschiedene ästhetische Ansätze vergleichen möchte, bevor er sich auf einen festlegt. Funktioniert mit Next.js + Tailwind CSS-Projekten (Landingpages, Web-Apps, SaaS, Dashboards).
