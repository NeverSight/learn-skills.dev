Разверните 8 демонстраций совершенно разных визуальных направлений для проекта веб-сайта или веб-приложения. Создает автономные страницы Next.js в /demo/ с различными идентичностями (типографика, цветовая палитра, макет, анимация, доступность), чтобы пользователь мог сравнивать и выбирать. После выбора генерирует готовые к использованию токены темы, конфигурацию Tailwind, переменные CSS и базовые компоненты, соответствующие выбранному направлению. Используйте, когда: начинаете новый веб-проект, меняете дизайн сайта, выбираете визуальное направление, пользователь говорит «выбор стиля», «выберите стили», «визуальные альтернативы», «выберите направление дизайна» или хочет сравнить различные эстетические подходы, прежде чем выбрать один. Работает с CSS-проектами Next.js + Tailwind (целевые страницы, веб-приложения, SaaS, информационные панели).
