Используйте этот навык всякий раз, когда пользователь хочет установить, настроить или использовать инструмент Gemini CLI (gemini-cli). Активируйте этот навык для таких задач, как: установка gemini-cli через npm/npx/brew, настройка аутентификации (ключ API, Google OAuth, Vertex AI), запуск неинтерактивных/безголовых запросов с флагом -p, настройка settings.json, создание контекстных файлов GEMINI.md, написание пользовательских косых команд (файлы .toml), подключение серверов MCP, создание расширений, автоматизация задач с помощью сценариев оболочки, использование --output-format json/stream-json, управление сеансами чата, использование команд /memory, режим --auto-approve, учетные данные приложения по умолчанию (ADC) или любые сценарии/автоматизация с использованием Gemini-cli. Также срабатывает, когда пользователь спрашивает об интеграции моделей Gemini в рабочие процессы CLI, конвейеры CI/CD или о программном использовании API Gemini с помощью инструмента CLI.
