Utilice esta habilidad siempre que el usuario desee instalar, configurar o utilizar la herramienta Gemini CLI (gemini-cli). Active esta habilidad para tareas como: instalar gemini-cli a través de npm/npx/brew, configurar la autenticación (clave API, Google OAuth, Vertex AI), ejecutar mensajes no interactivos/sin cabeza con el indicador -p, configurar settings.json, crear archivos de contexto GEMINI.md, escribir comandos de barra personalizados (archivos .toml), conectar servidores MCP, crear extensiones, automatizar tareas con scripts de shell, usar --output-format json/stream-json, administrar sesiones de chat, usar comandos /memory, modo de aprobación automática, credenciales predeterminadas de la aplicación (ADC) o cualquier secuencia de comandos/automatización que involucre gemini-cli. También se activa cuando el usuario pregunta sobre la integración de modelos Gemini en flujos de trabajo CLI, canales de CI/CD o el uso programático de la API Gemini a través de la herramienta CLI.
