استخدم هذه المهارة عندما يريد المستخدم تثبيت أو تكوين أو استخدام أداة Gemini CLI (gemini-cli). قم بتفعيل هذه المهارة لمهام مثل: تثبيت Gemini-cli عبر npm/npx/brew، وإعداد المصادقة (مفتاح API، وGoogle OAuth، وVertex AI)، وتشغيل المطالبات غير التفاعلية/مجهولة الرأس باستخدام علامة -p، وتكوين settings.json، وإنشاء ملفات سياق GEMINI.md، وكتابة أوامر شرطة مائلة مخصصة (ملفات .toml)، وتوصيل خوادم MCP، وإنشاء الامتدادات، وأتمتة المهام باستخدام نصوص shell، باستخدام --output-format json/stream-json، أو إدارة جلسات الدردشة، أو استخدام أوامر / الذاكرة، أو وضع الموافقة التلقائية، أو بيانات اعتماد التطبيق الافتراضية (ADC)، أو أي برمجة نصية/أتمتة تتضمن الجوزاء-cli. يتم التشغيل أيضًا عندما يسأل المستخدم عن دمج نماذج Gemini في مهام سير عمل CLI، أو خطوط أنابيب CI/CD، أو الاستخدام البرمجي لـ Gemini API من خلال أداة CLI.
