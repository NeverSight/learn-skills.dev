Verwenden Sie diesen Skill immer dann, wenn der Benutzer das Gemini CLI-Tool (gemini-cli) installieren, konfigurieren oder verwenden möchte. Lösen Sie diesen Skill für Aufgaben aus wie: Installation von gemini-cli über npm/npx/brew, Einrichten der Authentifizierung (API-Schlüssel, Google OAuth, Vertex AI), Ausführen nicht interaktiver/kopfloser Eingabeaufforderungen mit der Flagge -p, Konfigurieren von Settings.json, Erstellen von GEMINI.md-Kontextdateien, Schreiben benutzerdefinierter Slash-Befehle (.toml-Dateien), Verbinden von MCP-Servern, Erstellen von Erweiterungen, Automatisieren von Aufgaben mit Shell-Skripten, Verwenden von --output-format json/stream-json, Verwalten von Chat-Sitzungen, Verwenden von /memory-Befehlen, --auto-approve-Modus, Application Default Credentials (ADC) oder jegliche Skripterstellung/Automatisierung mit gemini-cli. Wird auch ausgelöst, wenn der Benutzer nach der Integration von Gemini-Modellen in CLI-Workflows, CI/CD-Pipelines oder der programmgesteuerten Verwendung der Gemini-API über das CLI-Tool fragt.
