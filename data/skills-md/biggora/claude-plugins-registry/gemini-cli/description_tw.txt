每當使用者想要安裝、設定或使用 Gemini CLI (gemini-cli) 工具時，請使用此技能。觸發此技能的任務包括：透過 npm/npx/brew 安裝 gemini-cli、設定身份驗證（API 金鑰、Google OAuth、Vertex AI）、使用 -p 標誌執行非互動式/無頭提示、設定 settings.json、建立 GEMINI.md 上下文檔案、擴充自訂斜線指令（.toml-form）、使用執行任務）、執行任務 -- json/stream-json、管理聊天會話、使用 /memory命令、--auto-approve 模式、應用程式預設憑證 (ADC) 或任何涉及 gemini-cli 的腳本/自動化。當使用者詢問將 Gemini 模型整合到 CLI 工作流程、CI/CD 管道或透過 CLI 工具以程式設計方式使用 Gemini API 時也會觸發。
