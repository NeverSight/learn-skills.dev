ユーザーが Gemini CLI (gemini-cli) ツールをインストール、構成、または使用する場合は、このスキルを使用します。このスキルは、npm/npx/brew 経由の gemini-cli のインストール、認証 (API キー、Google OAuth、Vertex AI) のセットアップ、-p フラグによる非インタラクティブ/ヘッドレス プロンプトの実行、settings.json の構成、GEMINI.md コンテキスト ファイルの作成、カスタム スラッシュ コマンド (.toml ファイル) の作成、MCP サーバーの接続、拡張機能の作成、シェル スクリプトによるタスクの自動化、--output-format json/stream-json の使用、などのタスクに対してトリガーします。 /memory コマンド、--auto-approve モード、アプリケーションのデフォルト認証情報 (ADC)、または gemini-cli に関連するスクリプト/自動化を使用して、チャット セッションを管理します。また、ユーザーが Gemini モデルを CLI ワークフロー、CI/CD パイプラインに統合すること、または CLI ツールを介した Gemini API のプログラムによる使用について尋ねたときにもトリガーされます。
