Utilizza questa competenza ogni volta che l'utente desidera installare, configurare o utilizzare lo strumento Gemini CLI (gemini-cli). Attiva questa abilità per attività quali: installazione di gemini-cli tramite npm/npx/brew, impostazione dell'autenticazione (chiave API, Google OAuth, Vertex AI), esecuzione di prompt non interattivi/headless con flag -p, configurazione di settings.json, creazione di file di contesto GEMINI.md, scrittura di comandi slash personalizzati (file .toml), connessione di server MCP, creazione di estensioni, automazione di attività con script di shell, utilizzo di --output-format json/stream-json, gestione della chat sessioni, utilizzando i comandi /memory, la modalità --auto-approve, le credenziali predefinite dell'applicazione (ADC) o qualsiasi scripting/automazione che coinvolga gemini-cli. Si attiva anche quando l'utente chiede informazioni sull'integrazione dei modelli Gemini nei flussi di lavoro CLI, nelle pipeline CI/CD o sull'uso programmatico dell'API Gemini tramite lo strumento CLI.
