Fertigkeit zum automatisierten Testen mobiler Anwendungen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine mobile App (Android oder iOS) testen, Testfälle schreiben, die App-Struktur analysieren, automatisierte UI-Tests über einen Emulator ausführen oder Testberichte erstellen möchte. Wird ausgelöst, wenn der Benutzer Folgendes erwähnt: „Meine App testen“, „Tests ausführen“, „UI-Tests“, „Testfälle schreiben“, „App-Funktionalität prüfen“, „Auf Emulator testen“, „Mobile QA“, „Testabdeckung“, „Anwendungsfalltests“, „Benutzerszenariotests“ oder eine beliebige Kombination aus Mobil + Testen/Prüfen/Verifizieren/Validieren. Wird auch ausgelöst, wenn der Benutzer ein APK, eine IPA-Datei oder einen mobilen Projektordner (React Native, Flutter, Android, iOS) hochlädt oder referenziert und fragt, was als Nächstes zu tun ist. Verwenden Sie diese Fähigkeit immer für alle QA-Aufgaben für mobile Apps – auch für Teilaufgaben wie „Schreiben Sie einfach einige Anwendungsfälle“ oder „Zeigen Sie mir, welche Tests ich ausführen soll“.
