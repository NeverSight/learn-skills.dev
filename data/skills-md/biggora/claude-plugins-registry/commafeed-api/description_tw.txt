每當使用者想要透過其 REST API 與 CommaFeed RSS 閱讀器實例互動時，請使用此技能。觸發器包括：列出、訂閱、取消訂閱 RSS 來源；閱讀、加註星標或標記提要條目；管理類別；將條目標記為已讀/未讀；導入或匯出 OPML；管理使用者設定或個人資料；查看未讀計數；清爽的飼料；透過 URL 取得 Feed 資訊；管理 CommaFeed 實例上的使用者。也可用於「顯示我的提要」、「訂閱 RSS 提要」、「列出未讀文章」、「為該條目加註星標」、「將我的提要匯出為 OPML」、「將所有提要標記為已讀」、「取得未讀計數」或「刷新我的提要」等請求。始終在任何 CommaFeed API 互動中使用此技能。
