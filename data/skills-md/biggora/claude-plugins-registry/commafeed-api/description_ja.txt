ユーザーが REST API 経由で CommaFeed RSS リーダー インスタンスと対話する場合は常に、このスキルを使用します。トリガーには、RSS フィードのリスト、購読、購読解除が含まれます。フィード エントリの読み取り、スター付け、またはタグ付け。カテゴリの管理。エントリを既読/未読としてマークします。 OPML のインポートまたはエクスポート。ユーザー設定またはプロファイルの管理。未読数の表示。フィードを更新します。 URL によってフィード情報を取得します。 CommaFeed インスタンスでのユーザーの管理。 「フィードを表示する」、「RSS フィードを購読する」、「未読の記事をリストする」、「このエントリにスターを付ける」、「フィードを OPML としてエクスポートする」、「すべてを既読としてマークする」、「未読数を取得する」、または「フィードを更新する」などのリクエストにも使用します。 CommaFeed API インタラクションには常にこのスキルを使用してください。
