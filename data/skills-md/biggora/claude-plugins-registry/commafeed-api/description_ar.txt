استخدم هذه المهارة عندما يريد المستخدم التفاعل مع مثيل قارئ CommaFeed RSS عبر REST API الخاص به. تتضمن المشغلات: الإدراج والاشتراك وإلغاء الاشتراك في خلاصات RSS؛ قراءة إدخالات الخلاصة أو تمييزها بنجمة أو وضع علامة عليها؛ إدارة الفئات؛ وضع علامة على الإدخالات كمقروءة/غير مقروءة؛ استيراد أو تصدير OPML؛ إدارة إعدادات المستخدم أو الملف الشخصي؛ عرض الأعداد غير المقروءة؛ خلاصات منعشة؛ جلب معلومات الخلاصة عن طريق URL؛ إدارة المستخدمين على مثيل CommaFeed. استخدمه أيضًا لطلبات مثل "إظهار خلاصاتي" أو "الاشتراك في موجز RSS" أو "إدراج المقالات غير المقروءة" أو "تمييز هذا الإدخال بنجمة" أو "تصدير خلاصاتي كـ OPML" أو "وضع علامة على الكل كمقروءة" أو "الحصول على عدد غير مقروء" أو "تحديث خلاصاتي". استخدم هذه المهارة دائمًا لأي تفاعل مع CommaFeed API.
