Use this skill whenever the user wants to interact with a CommaFeed RSS reader instance via its REST API. Triggers include: listing, subscribing, unsubscribing from RSS feeds; reading, starring, or tagging feed entries; managing categories; marking entries as read/unread; importing or exporting OPML; managing user settings or profile; viewing unread counts; refreshing feeds; fetching feed info by URL; administering users on a CommaFeed instance. Also use for requests like "show my feeds", "subscribe to RSS feed", "list unread articles", "star this entry", "export my feeds as OPML", "mark all as read", "get unread count", or "refresh my feeds". Always use this skill for any CommaFeed API interaction.
