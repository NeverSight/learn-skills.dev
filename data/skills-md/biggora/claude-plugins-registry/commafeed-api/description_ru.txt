Используйте этот навык всякий раз, когда пользователь хочет взаимодействовать с экземпляром средства чтения RSS CommaFeed через его REST API. Триггеры включают: листинг, подписку, отказ от подписки на RSS-каналы; чтение, пометка пометок или пометка записей канала; управление категориями; пометка записей как прочитанных/непрочитанных; импорт или экспорт OPML; управление настройками пользователя или профилем; просмотр количества непрочитанных сообщений; обновление каналов; получение информации о канале по URL-адресу; администрирование пользователей в экземпляре CommaFeed. Также используйте для таких запросов, как «показать мои каналы», «подписаться на RSS-канал», «составить список непрочитанных статей», «пометить эту запись», «экспортировать мои каналы как OPML», «отметить все как прочитанные», «получить счетчик непрочитанных» или «обновить мои каналы». Всегда используйте этот навык для любого взаимодействия с API CommaFeed.
