Utilisez cette compétence chaque fois que l'utilisateur souhaite interagir avec une instance de lecteur RSS CommaFeed via son API REST. Les déclencheurs incluent : le référencement, l'abonnement, le désabonnement des flux RSS ; lire, mettre en vedette ou marquer les entrées de flux ; gérer les catégories ; marquer les entrées comme lues/non lues ; importer ou exporter OPML ; gérer les paramètres ou le profil utilisateur ; afficher les décomptes non lus ; aliments rafraîchissants; récupérer les informations du flux par URL ; administrer les utilisateurs sur une instance CommaFeed. Utilisez également pour des requêtes telles que « afficher mes flux », « s'abonner au flux RSS », « lister les articles non lus », « mettre en vedette cette entrée », « exporter mes flux au format OPML », « tout marquer comme lu », « obtenir le nombre de non lus » ou « actualiser mes flux ». Utilisez toujours cette compétence pour toute interaction avec l'API CommaFeed.
