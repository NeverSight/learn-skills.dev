每当用户想要通过其 REST API 与 CommaFeed RSS 阅读器实例交互时，请使用此技能。触发器包括：列出、订阅、取消订阅 RSS 源；阅读、加注星标或标记提要条目；管理类别；将条目标记为已读/未读；导入或导出 OPML；管理用户设置或个人资料；查看未读计数；清爽的饲料；通过 URL 获取 Feed 信息；管理 CommaFeed 实例上的用户。还可用于“显示我的提要”、“订阅 RSS 提要”、“列出未读文章”、“为该条目加注星标”、“将我的提要导出为 OPML”、“将所有提要标记为已读”、“获取未读计数”或“刷新我的提要”等请求。始终在任何 CommaFeed API 交互中使用此技能。
