Verwenden Sie diesen Skill immer dann, wenn der Benutzer über seine REST-API mit einer CommaFeed-RSS-Reader-Instanz interagieren möchte. Zu den Auslösern gehören: Auflisten, Abonnieren und Abbestellen von RSS-Feeds; Feedeinträge lesen, markieren oder markieren; Verwalten von Kategorien; Einträge als gelesen/ungelesen markieren; OPML importieren oder exportieren; Verwalten von Benutzereinstellungen oder Profilen; ungelesene Zählungen anzeigen; erfrischende Futtermittel; Feed-Informationen per URL abrufen; Verwalten von Benutzern auf einer CommaFeed-Instanz. Auch für Anfragen wie „Meine Feeds anzeigen“, „RSS-Feed abonnieren“, „Ungelesene Artikel auflisten“, „Diesen Eintrag markieren“, „Meine Feeds als OPML exportieren“, „Alle als gelesen markieren“, „Anzahl der Ungelesenen abrufen“ oder „Meine Feeds aktualisieren“ verwenden. Verwenden Sie diesen Skill immer für jede CommaFeed-API-Interaktion.
