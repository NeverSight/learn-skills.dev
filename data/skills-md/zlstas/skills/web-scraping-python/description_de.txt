Wenden Sie Web Scraping mit Python-Praktiken an (Ryan Mitchell). Behandelt erste Scraper (Kap. 1: urllib, BeautifulSoup), HTML-Parsing (Kap. 2: find, findAll, CSS-Selektoren, Regex, Lambda), Crawling (Kap. 3-4: Single-Domain, Cross-Site, Crawl-Modelle), Scrapy (Kap. 5: Spiders, Items, Pipelines, Regeln), Speichern von Daten (Kap. 6: CSV, MySQL, Dateien, E-Mail), Lesen von Dokumenten (Kap. 7: PDF, Word, Codierung), Bereinigen von Daten (Kap 8: Normalisierung, OpenRefine), NLP (Kapitel 9: n-Gramm, Markov, NLTK), Formulare und Anmeldungen (Kapitel 10: POST, Sitzungen, Cookies), JavaScript (Kapitel 11: Selenium, Headless, Ajax), APIs (Kapitel 12: REST, undokumentiert), Bild/OCR (Kapitel 13: Pillow, Tesseract), Vermeidung von Fallen (Kapitel 14: Header, Honeypots), Testen (Kapitel 15: Unittest, Selenium), Parallel (Kapitel 16: Threads, Prozesse), Remote (Kapitel 17: Tor, Proxys), Legalitäten (Kapitel 18: robots.txt, CFAA, Ethik). Auslöser für „Web Scraping“, „BeautifulSoup“, „Scrapy“, „Crawler“, „Spider“, „Scraper“, „Parse HTML“, „Selenium Scraping“, „Datenextraktion“.
