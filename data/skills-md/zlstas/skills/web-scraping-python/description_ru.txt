Примените парсинг веб-страниц с помощью методов Python (Райан Митчелл). Охватывает первые парсеры (глава 1: urllib, BeautifulSoup), анализ HTML (глава 2: find, findAll, селекторы CSS, регулярное выражение, лямбда), сканирование (главы 3–4: модели однодоменного, межсайтового сканирования, модели сканирования), Scrapy (глава 5: пауки, элементы, конвейеры, правила), хранение данных (глава 6: CSV, MySQL, файлы, электронная почта), чтение документов (глава 7: PDF, Word, кодирование), очистка данных (глава 8: нормализация, OpenRefine), NLP (глава 9: n-граммы, Markov, NLTK), формы и логины (глава 10: POST, сеансы, файлы cookie), JavaScript (глава 11: Selenium, headless, Ajax), API (глава 12: REST, недокументированные), Image/OCR (глава 13: Pillow, Tesseract), избежание ловушек (глава 14: заголовки, приманки), тестирование (глава 15: unittest, Selenium), параллельное (глава 16: потоки, процессы), удаленное (глава 17: Tor, прокси), законность (глава 18: robots.txt, CFAA, этика). Запуск по «парсингу веб-страниц», «BeautifulSoup», «Scrapy», «сканеру», «пауку», «скребку», «анализу HTML», «очистке селена», «извлечению данных».
