تطبيق تجريف الويب باستخدام ممارسات بايثون (ريان ميتشل). يغطي أدوات الكشط الأولى (السلسلة 1: urllib، BeautifulSoup)، تحليل HTML (السلسلة 2: العثور على، findAll، محددات CSS، regex، lambda)، الزحف (السلسلة 3-4: النطاق الفردي، المواقع المشتركة، نماذج الزحف)، Scrapy (السلسلة 5: العناكب، العناصر، خطوط الأنابيب، القواعد)، تخزين البيانات (السلسلة 6: CSV، MySQL، الملفات، البريد الإلكتروني)، قراءة المستندات (السلسلة 7: PDF، Word، التشفير)، تنظيف البيانات (8 سلسلة: التطبيع، OpenRefine)، البرمجة اللغوية العصبية (9 سلسلة: n-grams، Markov، NLTK)، النماذج وتسجيلات الدخول (10 سلسلة: POST، الجلسات، ملفات تعريف الارتباط)، JavaScript (11 سلسلة: السيلينيوم، مقطوعة الرأس، Ajax)، واجهات برمجة التطبيقات (12 سلسلة: REST، غير موثقة)، صورة/التعرف الضوئي على الحروف (13 سلسلة: وسادة، تسراكت)، تجنب الفخاخ (14 سلسلة: الرؤوس، مصائد مخترقي الشبكات)، الاختبار (15 سلسلة: Unittest، السيلينيوم)، Parallel (16 سلسلة: الخيوط، العمليات)، Remote (17 سلسلة: Tor، الوكلاء)، الجوانب القانونية (18 سلسلة: robots.txt، CFAA، الأخلاقيات). تشغيل "تجريد الويب"، "BeautifulSoup"، "Scrapy"، "الزاحف"، "العنكبوت"، "مكشطة"، "تحليل HTML"، "تجريف السيلينيوم"، "استخراج البيانات".
