Appliquez le Web Scraping avec les pratiques Python (Ryan Mitchell). Couvre les premiers Scrapers (Ch 1 : urllib, BeautifulSoup), l'analyse HTML (Ch 2 : find, findAll, sélecteurs CSS, regex, lambda), l'exploration (Ch 3-4 : domaine unique, intersite, modèles d'analyse), Scrapy (Ch 5 : araignées, éléments, pipelines, règles), le stockage de données (Ch 6 : CSV, MySQL, fichiers, courrier électronique), la lecture de documents (Ch 7 : PDF, Word, encodage), le nettoyage des données. (Ch 8 : normalisation, OpenRefine), NLP (Ch 9 : n-grams, Markov, NLTK), Formulaires et connexions (Ch 10 : POST, sessions, cookies), JavaScript (Ch 11 : Selenium, sans tête, Ajax), API (Ch 12 : REST, non documenté), Image/OCR (Ch 13 : Pillow, Tesseract), Éviter les pièges (Ch 14 : en-têtes, pots de miel), Tests (Ch 15 : unittest, Selenium), Parallèle (Ch 16 : threads, processus), Remote (Ch 17 : Tor, proxys), Légalités (Ch 18 : robots.txt, CFAA, éthique).
