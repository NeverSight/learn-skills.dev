Utilisez csesh pour gérer les sessions Claude Code. Déclenchez lorsque l'utilisateur mentionne des sessions, la gestion de session, le nettoyage de session, l'historique de session, la reprise de sessions ou souhaite analyser son utilisation de Claude Code. Les commandes incluent la liste csesh, l'analyse csesh, les statistiques csesh, le nettoyage csesh, la reprise csesh, le web csesh, la balise csesh, le titre csesh, l'exportation csesh.
