當使用者編寫從「@yo-protocol/core」匯入的 TypeScript 或 JavaScript 程式碼、使用「createYoClient」或透過 SDK 以程式設計方式與 Yo Protocol 金庫（yoETH、yoUSD、yoBTC、yoEUR、yoGOLD、yoUSDT）互動時，請務必使用此技能。此技能涵蓋了完整的「@yo-protocol/core」 SDK：客戶端初始化（createYoClient、多鏈publicClients、partnerId）、金庫讀取（getVaultState、previewDeposit、previewRedeem、isPaused）、使用者讀取（getUserPosition、getShareBalance、getTokenBalance、getAllowance）、準備好的交易（prepareDepositWithApproval、preparey查詢（getVaultSnapshot、getVaultYieldHistory、getVaultTvlHistory、getUserHistory、getUserPerformance）、Merkl 獎勵聲明（getClaimableRewards、prepareClaimMerklRewards）和公用程式（parseTokenAmount、formatTokenAmount、VAatToTSAmount、VAULTS 登錄位址）。觸發 Node.js 腳本、後端服務、機器人、Safe/AA 批次交易建構器或與 Yo Protocol 互動的任何非 React TypeScript 程式碼。當用戶提到 Yo 網關、PreparedTransaction、特定於 Yo 的 ERC-4626 金庫交互，或詢問每個鏈的金庫地址和代幣地址時也會觸發。請勿用於 React hooks（使用 yo-protocol-react）或 CLI shell 指令（使用 yo-protocol-cli）。
