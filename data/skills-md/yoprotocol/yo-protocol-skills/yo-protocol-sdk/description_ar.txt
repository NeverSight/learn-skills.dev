استخدم هذه المهارة دائمًا عندما يكتب المستخدم كود TypeScript أو JavaScript الذي يستورد من `@yo-protocol/core`، أو يستخدم `createYoClient`، أو يتفاعل برمجيًا مع خزائن Yo Protocol (yoETH، yoUSD، yoBTC، yoEUR، yoGOLD، yoUSDT) عبر SDK. تغطي هذه المهارة مجموعة SDK `@yo-protocol/core` الكاملة: تهيئة العميل (createYoClient، multi-chain publicClients، PartnerId)، قراءات المخزن (getVaultState، PreviewDeposit، PreviewRedeem، isPaused)، قراءات المستخدم (getUserPosition، getShareBalance، getTokenBalance، getAllowance)، المعاملات المعدة (prepareDepositWithApproval، PreparRedeemWithApproval، PreparationApprove)، استعلامات REST API (getVaultSnapshot، getVaultYieldHistory، getVaultTvlHistory، getUserHistory، getUserPerformance)، مطالبات مكافآت Merkl (getClaimableRewards، PreparClaimMerklRewards)، والأدوات المساعدة (parseTokenAmount، formatTokenAmount، تسجيل VAULTS، عنوان البوابة). مشغل البرامج النصية Node.js، أو خدمات الواجهة الخلفية، أو الروبوتات، أو منشئي المعاملات الدفعية Safe/AA، أو أي كود غير تابع لـ React TypeScript يتفاعل مع بروتوكول Yo. يتم التشغيل أيضًا عندما يذكر المستخدم بوابة Yo، وPreparedTransaction، وتفاعلات قبو ERC-4626 الخاصة بـ Yo، أو يسأل عن عناوين المخزن وعناوين الرمز المميز لكل سلسلة. لا تستخدمه مع خطافات React (استخدم yo-protocol-react) أو أوامر shell CLI (استخدم yo-protocol-cli).
