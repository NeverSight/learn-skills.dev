Entdecken und installieren Sie Agent-Skills von Drittanbietern aus dem offenen Skills-Ökosystem (skills.sh). Dieser Skill sollte verwendet werden, wenn Benutzer externe Skills für Claude Code oder andere KI-Agenten finden, installieren, aktualisieren oder verwalten möchten, oder wenn sie skills.sh, npx-Skills erwähnen oder ihren Agenten mit Community-Skills erweitern möchten.
