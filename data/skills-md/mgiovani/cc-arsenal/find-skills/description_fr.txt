Découvrez et installez des compétences d'agent tiers à partir de l'écosystème de compétences ouvert (skills.sh). Cette compétence doit être utilisée lorsque les utilisateurs souhaitent rechercher, installer, mettre à jour ou gérer des compétences externes pour Claude Code ou d'autres agents IA, ou lorsqu'ils mentionnent des compétences skills.sh, npx, ou lorsqu'ils souhaitent étendre leur agent avec des compétences communautaires.
