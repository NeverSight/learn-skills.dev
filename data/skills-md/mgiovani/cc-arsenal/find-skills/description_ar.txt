اكتشف مهارات وكيل الطرف الثالث وقم بتثبيتها من النظام البيئي للمهارات المفتوحة (skills.sh). يجب استخدام هذه المهارة عندما يريد المستخدمون العثور على مهارات خارجية أو تثبيتها أو تحديثها أو إدارتها لـ Claude Code أو عملاء الذكاء الاصطناعي الآخرين، أو عندما يذكرون Skills.sh أو مهارات npx أو يريدون توسيع وكيلهم بمهارات المجتمع.
