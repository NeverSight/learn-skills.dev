Refactoriza código existente complejo. Usa ist eine Fähigkeit, die den Benutzer vereinfacht, umschreibt, den Code mit größeren Funktionen löscht oder besser macht, den Code dupliziert, ihn tiefgreifend animiert, übermäßige Parameter, eine clevere logische Logik, injizierte Abhängigkeiten oder so viel Einfallsreichtum. Anwendung KISS, DRY, YAGNI und die Philosophie von Linus Torvalds.
