Zod — проверка схемы на основе TypeScript со статическим выводом типа. Используйте при сборке с помощью Zod или при запросе об определениях схемы, выводе типов, синтаксическом анализе, преобразованиях, уточнениях, принуждении, обработке ошибок или интеграции с формами, API или tRPC. Получите актуальную документацию для получения актуальной информации.
