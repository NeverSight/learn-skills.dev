Meilisearch - محرك بحث سريع ومفتوح المصدر مع إمكانية التسامح مع الأخطاء المطبعية والبحث متعدد الأوجه والبحث المختلط المدعوم بالذكاء الاصطناعي. يُستخدم عند البناء باستخدام Meilisearch أو السؤال عن تكوين الفهرس أو معلمات البحث أو عوامل التصفية أو الجوانب أو مفاتيح API أو البحث الجغرافي أو قواعد التصنيف أو التكامل مع عملاء JavaScript/TypeScript. جلب الوثائق الحية للحصول على تفاصيل محدثة.
