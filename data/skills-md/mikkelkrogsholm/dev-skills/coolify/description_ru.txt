Coolify — самостоятельная PaaS с открытым исходным кодом для развертывания приложений, баз данных и сервисов через Docker. Используйте при сборке с помощью Coolify или при запросе конфигурации его развертывания, переменных среды, настройки обратного прокси-сервера, проверок работоспособности, постоянного хранилища или интеграции CI/CD на основе Git. Получите актуальную документацию для получения актуальной информации.
