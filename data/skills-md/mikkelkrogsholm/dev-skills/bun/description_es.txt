Bun: rápido tiempo de ejecución todo en uno de JavaScript/TypeScript, administrador de paquetes, agrupador y ejecutor de pruebas. Úselo al compilar con Bun, ejecutar TypeScript, administrar paquetes con bun install, escribir pruebas con bun test o preguntar sobre las API de Bun, la configuración o la migración de Node.js. Obtenga documentación en vivo para obtener detalles de API actualizados.
