Bun — быстрая универсальная среда выполнения JavaScript/TypeScript, менеджер пакетов, сборщик и средство запуска тестов. Используйте при сборке с помощью Bun, запуске TypeScript, управлении пакетами с помощью Bun install, написании тестов с помощью Bun Test или при запросе об API-интерфейсах Bun, конфигурации или миграции Node.js. Получите актуальную документацию для получения актуальной информации об API.
