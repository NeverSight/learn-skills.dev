Bun – environnement d'exécution JavaScript/TypeScript tout-en-un rapide, gestionnaire de packages, bundler et exécuteur de tests. À utiliser lors de la création avec Bun, de l'exécution de TypeScript, de la gestion de packages avec bun install, de l'écriture de tests avec bun test ou de questions sur les API Bun, la configuration ou la migration Node.js. Récupérez la documentation en direct pour obtenir des détails à jour sur l'API.
