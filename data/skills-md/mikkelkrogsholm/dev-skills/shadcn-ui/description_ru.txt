shadcn/ui — библиотека компонентов React, принадлежащая копиям, построенная на основе пользовательского интерфейса Radix и CSS Tailwind. Используйте при сборке с помощью shadcn/ui или при запросе о его компонентах, интерфейсе командной строки, темах, конфигурации или интеграции с Next.js, Vite, Remix или другими платформами. Получите актуальную документацию для получения актуальной информации.
