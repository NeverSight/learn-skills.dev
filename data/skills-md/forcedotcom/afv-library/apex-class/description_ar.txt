قم بإنشاء فئات Apex جاهزة للإنتاج لـ Salesforce باتباع أفضل ممارسات المؤسسة. يغطي فئات الخدمة، والمحددات، وفئات المجال، والدُفعات/القابلة للانتظار/الجدولة، وDTOs، والأدوات المساعدة، والواجهات، والفئات المجردة، والاستثناءات المخصصة. ليس للمشغلات أو اختبارات الوحدة. استخدم هذه المهارة عندما يطلب المستخدم إنشاء أو كتابة أو بناء أو إنشاء أو دعم أي فئة Apex - بما في ذلك عندما يصف الوظيفة التي تتطلب فئة Apex دون قول "فئة Apex" صراحةً. قم بتشغيل عبارات مثل "إنشاء خدمة من أجل" أو "إنشاء معالج" أو "أحتاج إلى فصل دراسي" أو "أكتب Apex إليه" أو "دعم مهمة دفعية" أو "إنشاء قائمة انتظار" أو أي طلب يتضمن منطق جانب خادم Salesforce. يُستخدم أيضًا عندما يطلب المستخدم إعادة بناء فئات Apex الحالية أو تحسينها أو إعادة هيكلتها. حتى إذا كان الطلب غامضًا مثل "أحتاج إلى شيء يعالج الفرص ليلاً"، فاستخدم هذه المهارة إذا كان الحل يتضمن فئة Apex.
