Genere clases Apex listas para producción para Salesforce siguiendo las mejores prácticas empresariales. Cubre clases de servicio, selectores, clases de dominio, lotes/en cola/programables, DTO, utilidades, interfaces, clases abstractas y excepciones personalizadas. No para desencadenadores o pruebas unitarias. Utilice esta habilidad siempre que el usuario solicite crear, escribir, construir, generar o aplicar scaffolding a cualquier clase de Apex, incluso cuando describa una funcionalidad que requeriría una clase de Apex sin decir explícitamente "clase de Apex". Active frases como "crear un servicio para", "crear un controlador", "necesito una clase que", "escribir Apex en", "desarrollar un trabajo por lotes", "crear una cola" o cualquier solicitud que involucre la lógica del lado del servidor de Salesforce. Utilícelo también cuando el usuario solicite refactorizar, mejorar o reestructurar clases de Apex existentes. Incluso si la solicitud es vaga como "Necesito algo que procese Oportunidades todas las noches", use esta habilidad si la solución implicaría una clase de Apex.
