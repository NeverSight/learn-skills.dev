Führt gründliche Codeüberprüfungen für Flutter/Dart-Pull-Requests und Merge-Requests durch. Verwenden Sie diese Option, wenn Sie aufgefordert werden, eine PR, MR, einen Zweig oder eine Reihe geänderter Dateien zu überprüfen. Folgt einer strukturierten Checkliste, die Korrektheit, Sicherheit, Stil, Tests und Dokumentation umfasst.
