Bietet DeFi-Liquiditätspooldaten, Tresorverwaltung, Benutzerpositionen und LP-Transaktionsaufbau über die Krystal-API. Verwendung für Pool-Analysen (TVL/Volumen/APR), Auflistung von Positionen (einschließlich Einzelpositionssuche nach PositionId) und LP-Aktionen wie Zap-In, Zap-Out (einschließlich Gebührenforderung über liquidityPercent=0), Swap-and-Increase, Adjust-Range (Rebalancing) und Compound über Ketten hinweg wie Ethereum, BSC, Polygon, Arbitrum, Base und Solana.
