Proporciona datos del fondo de liquidez de DeFi, gestión de bóvedas, posiciones de usuarios y creación de transacciones LP a través de la API de Krystal. Úselo para análisis de grupos (TVL/volumen/APR), listado de posiciones (incluida la búsqueda de una sola posición por ID de posición) y acciones de LP como zap-in, zap-out (incluida la reclamación de tarifas a través de liquidityPercent=0), intercambio y aumento, ajuste de rango (reequilibrio) y compuestos en cadenas como Ethereum, BSC, Polygon, Arbitrum, Base y Solana.
