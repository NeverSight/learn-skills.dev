يوفر بيانات مجمع سيولة DeFi وإدارة الخزينة ومواقع المستخدمين وبناء معاملات LP عبر Krystal API. يُستخدم لتحليلات المجمع (TVL/الحجم/APR)، ومواضع الإدراج (بما في ذلك البحث عن موضع واحد بواسطة PositionId)، وإجراءات LP مثل الانطلاق والانطلاق (بما في ذلك المطالبة بالرسوم عبر نسبة السيولة = 0)، والمبادلة والزيادة، وضبط النطاق (إعادة التوازن)، والمركب عبر سلاسل مثل Ethereum، وBSC، وPolygon، وArbitrum، وBase، وSolana.
