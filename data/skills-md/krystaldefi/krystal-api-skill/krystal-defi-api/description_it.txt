Fornisce dati sul pool di liquidità DeFi, gestione del deposito, posizioni degli utenti e creazione di transazioni LP tramite l'API Krystal. Utilizzare per l'analisi del pool (TVL/volume/APR), elencare le posizioni (inclusa la ricerca di una singola posizione per positionId) e azioni LP come zap-in, zap-out (inclusa la richiesta di commissioni tramite liquidityPercent=0), swap-and-increase, aggiustamento dell'intervallo (ribilanciamento) e compound tra catene come Ethereum, BSC, Polygon, Arbitrum, Base e Solana.
