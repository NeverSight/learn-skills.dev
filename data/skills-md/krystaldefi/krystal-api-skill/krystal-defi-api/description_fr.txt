Fournit des données sur le pool de liquidité DeFi, la gestion des coffres-forts, les positions des utilisateurs et la création de transactions LP via l'API Krystal. À utiliser pour l'analyse de pool (TVL/volume/APR), la liste des positions (y compris la recherche d'une position unique par positionId) et les actions LP telles que le zap-in, le zap-out (y compris la réclamation de frais via liquidPercent=0), l'échange et l'augmentation, l'ajustement de la plage (rééquilibrage) et le composé sur des chaînes comme Ethereum, BSC, Polygon, Arbitrum, Base et Solana.
