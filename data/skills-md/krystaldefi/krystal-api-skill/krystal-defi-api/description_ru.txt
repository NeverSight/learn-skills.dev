Предоставляет данные пула ликвидности DeFi, управление хранилищем, позиции пользователей и построение транзакций LP через Krystal API. Используйте для аналитики пула (TVL/объем/APR), листинговых позиций (включая поиск одной позиции по идентификатору позиции) и действий LP, таких как вход, выход (включая требование комиссии через LiquidityPercent = 0), обмен и увеличение, корректировка диапазона (ребалансировка) и соединение между такими цепочками, как Ethereum, BSC, Polygon, Arbitrum, Base и Solana.
