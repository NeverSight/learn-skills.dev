Umfassender Sicherheitsprüfer für Claude Skills- und MCP-Server. Analysiert Code auf bösartige Muster,
verdächtiges Verhalten und Sicherheitslücken. Bietet detaillierte Risikobewertung und Empfehlungen.
Verwenden Sie diese Funktion, wenn Sie neue Fertigkeiten bewerten, MCP-Server prüfen, heruntergeladenen Code überprüfen oder die Sicherheit von Fertigkeiten überprüfen möchten.
Aktivieren bei: „Sicherheitsaudit“, „Ist dieser Skill sicher“, „Diesen Skill prüfen“, „MCP prüfen“, „Skillsicherheit überprüfen“,
„guvenli mi“, „skill analiz“, „guvenlik kontrolu“.
