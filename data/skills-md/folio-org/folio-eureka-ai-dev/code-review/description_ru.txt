Используйте, когда пользователь просит выполнить проверку кода, просмотреть изменения кода, проанализировать различия или проверить качество кода. Выполняет структурированный анализ выходных данных git diff, охватывающий безопасность, корректность, производительность, удобство сопровождения и стиль. Создает отчет об уценке, сохраненный в виде файла .md, названного в честь текущей ветки.
