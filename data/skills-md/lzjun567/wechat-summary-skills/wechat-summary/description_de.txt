Fassen Sie die Artikel zum offiziellen WeChat-Konto zusammen. Verwenden Sie diesen Skill, wenn ein Benutzer einen Link zu einem Artikel des offiziellen WeChat-Kontos (mp.weixin.qq.com) bereitstellt und eine Zusammenfassung, einen Überblick, eine Zusammenfassung oder ein schnelles Verständnis des Inhalts anfordert. Dies gilt auch, wenn der Benutzer Dinge sagt wie „Erzählen Sie mir, worum es in diesem Artikel geht“, „Was sagt dieser Artikel zum offiziellen Konto“ oder „Fassen Sie diesen Link zusammen.“ Immer wenn ein mp.weixin.qq.com-Link gesehen wird und der Benutzer seinen Inhalt wissen möchte, sollte diese Fähigkeit genutzt werden.
