دليل اختيار وتكوين وتخصيص وحدات Terraform الخاصة بـ Google Cloud Foundation Fabric (CFF) ومراحل منطقة الهبوط السريعة. يُستخدم عند العمل مع Cloud-foundation-fabric، أو Fabric FAST، أو مناطق الهبوط GCP، أو وحدات CFF، أو مصنع المشروع، أو مصنع VPC، أو عندما يذكر المستخدم وحدات Google Cloud Terraform من GoogleCloudPlatform/cloud-foundation-fabric. يُستخدم أيضًا عند تصميم البنية التنظيمية لـ Google Cloud Platform، أو مراحل الشبكات، أو تكوين أنماطfactors_config.
