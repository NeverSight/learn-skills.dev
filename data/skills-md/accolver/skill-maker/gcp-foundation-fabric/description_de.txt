Anleitung zur Auswahl, Konfiguration und Anpassung der Terraform-Module und FAST-Landezonenstufen von Google Cloud Foundation Fabric (CFF). Wird verwendet, wenn Sie mit Cloud-Foundation-Fabric, Fabric FAST, GCP-Landezonen, CFF-Modulen, Projektfabrik, VPC-Fabrik arbeiten oder wenn der Benutzer Google Cloud Terraform-Module von GoogleCloudPlatform/cloud-foundation-fabric erwähnt. Wird auch beim Entwerfen der GCP-Organisationsstruktur, der Netzwerkphasen oder beim Konfigurieren von „factorys_config“-Mustern verwendet.
