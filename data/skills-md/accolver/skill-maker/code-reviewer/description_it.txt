Esegue revisioni strutturate del codice analizzando differenze o file per individuare bug, vulnerabilità della sicurezza, problemi di prestazioni, violazioni di stile e problemi di manutenibilità. Produce risultati categorizzati con livelli di gravità e suggerimenti per soluzioni attuabili. Da utilizzare durante la revisione del codice, le richieste pull, le richieste di unione, le modifiche al codice o quando l'utente richiede feedback sul proprio codice.
