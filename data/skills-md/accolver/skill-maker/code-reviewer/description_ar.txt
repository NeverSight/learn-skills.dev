يقوم بإجراء مراجعات للتعليمات البرمجية المنظمة لتحليل الاختلافات أو الملفات بحثًا عن الأخطاء ونقاط الضعف الأمنية ومشكلات الأداء وانتهاكات الأسلوب ومخاوف الصيانة. ينتج نتائج مصنفة بمستويات خطورة واقتراحات إصلاح قابلة للتنفيذ. يُستخدم عند مراجعة التعليمات البرمجية، أو سحب الطلبات، أو طلبات الدمج، أو تغييرات التعليمات البرمجية، أو عندما يطلب المستخدم تعليقات على التعليمات البرمجية الخاصة به.
