Führt strukturierte Codeüberprüfungen durch und analysiert Unterschiede oder Dateien auf Fehler, Sicherheitslücken, Leistungsprobleme, Stilverstöße und Wartbarkeitsprobleme. Erstellt kategorisierte Ergebnisse mit Schweregraden und umsetzbaren Lösungsvorschlägen. Wird zum Überprüfen von Code, Pull-Requests, Merge-Requests, Codeänderungen oder wenn der Benutzer um Feedback zu seinem Code bittet, verwendet.
