Genera messaggi di commit convenzionali da modifiche git organizzate analizzando diff, classificando i tipi di modifica (feat, fix, refactor, docs, test, chore, ci, perf, style, build) e componendo messaggi di commit strutturati con ambito facoltativo e notazione di modifica di rilievo. Da utilizzare durante il commit del codice, la scrittura di messaggi di commit, la preparazione di commit git o quando l'utente chiede aiuto con i messaggi di commit.
