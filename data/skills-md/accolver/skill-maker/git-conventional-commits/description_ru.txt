Генерирует обычные сообщения о фиксации из поэтапных изменений git, анализируя различия, классифицируя типы изменений (подвиг, исправление, рефакторинг, документация, тест, работа, ci, perf, стиль, сборка) и составляя структурированные сообщения о фиксации с дополнительной областью действия и нотацией критических изменений. Используйте при фиксации кода, написании сообщений о фиксации, подготовке коммитов git или когда пользователь запрашивает помощь с сообщениями о фиксации.
