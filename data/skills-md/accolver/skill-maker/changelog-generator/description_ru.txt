Создает ориентированные на аудиторию журналы изменений на основе истории git с классификацией воздействия SemVer, инструкциями по миграции для критических изменений, сгруппированными категориями (функции, исправления, производительность, зависимости) и руководствами по обновлению для основных версий. Используйте при создании журналов изменений, написании примечаний к выпуску, подготовке обновлений версий или когда пользователю необходимо задокументировать, что изменилось между выпусками.
