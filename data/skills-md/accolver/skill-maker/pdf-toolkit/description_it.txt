Estrai testo, tabelle e immagini da PDF, PDF scansionati con OCR, crea PDF da testo/immagini/markdown e unisci o dividi file PDF. Da utilizzare quando si lavora con documenti PDF, quando l'utente menziona PDF, estrazione di documenti, OCR, scansione, unione di PDF, divisione, combinazione o creazione di report PDF. Tutte le operazioni utilizzano script Bun TypeScript in bundle con output JSON strutturato.
