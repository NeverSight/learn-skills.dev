يكتب عمليات ترحيل آمنة وقابلة للعكس لقاعدة البيانات باستخدام خطط التراجع، وأوامر النسخ الاحتياطي للبيانات، وملاحظات النشر بدون توقف، وتحليل تأثير الفهرس. يُنتج ملفات الترحيل لأعلى ولأسفل مع استعلامات التحقق. يُستخدم عند كتابة عمليات ترحيل قاعدة البيانات، أو تغييرات المخطط، أو عبارات ALTER TABLE، أو إضافة الفهارس، أو عندما يحتاج المستخدم إلى تعديل بنية قاعدة البيانات.
