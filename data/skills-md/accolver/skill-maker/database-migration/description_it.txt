Scrive migrazioni di database sicure e reversibili con piani di rollback, comandi di backup dei dati, note di distribuzione senza tempi di inattività e analisi dell'impatto dell'indice. Produce file di migrazione verso l'alto e verso il basso con query di verifica. Da utilizzare durante la scrittura di migrazioni di database, modifiche dello schema, istruzioni ALTER TABLE, aggiunta di indici o quando l'utente deve modificare la struttura del database.
