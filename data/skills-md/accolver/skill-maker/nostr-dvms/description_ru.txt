Создавайте службы и клиенты Nostr Data Vending Machine (DVM) с помощью NIP-90. Используйте при реализации поставщиков ИИ/вычислительных услуг (типы 5000–5999 запросов заданий, 6000–6999 результатов заданий, тип 7000 отзывов), создании клиентов запросов заданий DVM с обработкой платежей, объединении заданий DVM в цепочку, обработке зашифрованных параметров DVM или публикации событий обнаружения поставщиков услуг NIP-89 для DVM.
