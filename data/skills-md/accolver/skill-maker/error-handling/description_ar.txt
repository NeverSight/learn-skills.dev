توحيد معالجة الأخطاء عبر قاعدة التعليمات البرمجية باستخدام تصنيف موحد للأخطاء، وأكواد أخطاء متسقة، وسلاسل نشر مناسبة، وفصل بين الأخطاء الداخلية ومواجهة المستخدم، والتسجيل المنظم بمعرفات الارتباط. يُستخدم عند إضافة معالجة الأخطاء، أو تنفيذ فئات الأخطاء، أو توحيد الاستثناءات، أو إنشاء استجابات للأخطاء، أو عندما يحتاج المستخدم إلى أنماط خطأ متسقة.
