Estandariza el manejo de errores en una base de código con una taxonomía de errores unificada, códigos de error consistentes, cadenas de propagación adecuadas, separación de errores internos y de cara al usuario y registro estructurado con ID de correlación. Úselo al agregar manejo de errores, implementar clases de error, estandarizar excepciones, crear respuestas de error o cuando el usuario necesite patrones de error consistentes.
