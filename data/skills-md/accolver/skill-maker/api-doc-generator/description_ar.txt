ينشئ وثائق API شاملة من التعليمات البرمجية المصدر عن طريق تحليل تعريفات نقطة النهاية ومخططات الطلب/الاستجابة ومتطلبات المصادقة وأكواد الخطأ. يُنتج مستندات منظمة بتنسيق متوافق مع OpenAPI مع أمثلة وأوصاف المعلمات وأدلة الاستخدام. يُستخدم عند توثيق واجهات برمجة التطبيقات، أو إنشاء مستندات مرجعية لواجهة برمجة التطبيقات، أو إنشاء وثائق نقطة النهاية، أو عندما يحتاج المستخدم إلى مستندات واجهة برمجة التطبيقات من قاعدة التعليمات البرمجية الخاصة به.
