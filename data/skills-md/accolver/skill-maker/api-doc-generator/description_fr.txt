Génère une documentation API complète à partir du code source en analysant les définitions de points de terminaison, les schémas de demande/réponse, les exigences d'authentification et les codes d'erreur. Produit des documents structurés au format compatible OpenAPI avec des exemples, des descriptions de paramètres et des guides d'utilisation. À utiliser lors de la documentation des API, de la génération de documents de référence d'API, de la création de documentation sur les points de terminaison ou lorsque l'utilisateur a besoin de documents d'API à partir de sa base de code.
