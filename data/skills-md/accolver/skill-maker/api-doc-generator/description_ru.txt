Создает исчерпывающую документацию по API из исходного кода путем анализа определений конечных точек, схем запросов/ответов, требований аутентификации и кодов ошибок. Создает структурированную документацию в формате, совместимом с OpenAPI, с примерами, описаниями параметров и руководствами по использованию. Используйте при документировании API, создании справочной документации API, создании документации по конечным точкам или когда пользователю нужны документы API из его базы кода.
