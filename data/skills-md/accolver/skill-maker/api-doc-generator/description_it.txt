Genera una documentazione API completa dal codice sorgente analizzando le definizioni degli endpoint, gli schemi di richiesta/risposta, i requisiti di autenticazione e i codici di errore. Produce documenti strutturati in formato compatibile con OpenAPI con esempi, descrizioni dei parametri e guide all'utilizzo. Da utilizzare quando si documentano le API, si generano documenti di riferimento API, si crea documentazione sugli endpoint o quando l'utente ha bisogno di documenti API dalla propria codebase.
