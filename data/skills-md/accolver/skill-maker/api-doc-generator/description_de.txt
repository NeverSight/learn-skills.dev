Erstellt eine umfassende API-Dokumentation aus Quellcode durch Analyse von Endpunktdefinitionen, Anforderungs-/Antwortschemata, Authentifizierungsanforderungen und Fehlercodes. Erstellt strukturierte Dokumente im OpenAPI-kompatiblen Format mit Beispielen, Parameterbeschreibungen und Nutzungshandbüchern. Verwenden Sie es, wenn Sie APIs dokumentieren, API-Referenzdokumente erstellen, Endpunktdokumentation erstellen oder wenn der Benutzer API-Dokumente aus seiner Codebasis benötigt.
