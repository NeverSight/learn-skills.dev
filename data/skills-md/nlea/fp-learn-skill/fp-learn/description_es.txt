Úselo cuando el usuario solicite "planificar con objetivos de aprendizaje", "aprender mientras construye", "fp-learn", "agregar aprendizaje al plan" o quiere realizar un seguimiento de lo que está aprendiendo durante un proyecto. Incluye fp-plan/fp-implement/fp-review para clasificar tareas como bricolaje o delegar, realizar un seguimiento del progreso por fase y generar cuestionarios.
