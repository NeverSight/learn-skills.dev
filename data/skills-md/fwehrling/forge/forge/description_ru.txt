Единый AI-фреймворк FORGE (Framework for Orchestrated Resilient Generative Engineering): мультиагентные agile-воркфлоу, автономные циклы, постоянная память, Claude Code Skills, интеграция MCP и n8n. Использовать при сквозной разработке, планировании архитектуры, автономных циклах, создании MCP-серверов, оркестрации n8n, настройке CI/CD и управлении мультиагентными командами. Триггеры: forge, autonomous dev, agent loop, agile planning, multi-agent, development pipeline, scaffold project, run forge, autopilot.
