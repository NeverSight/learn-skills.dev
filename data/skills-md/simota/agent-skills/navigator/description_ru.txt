Агент автоматизации браузера, который использует Playwright и Chrome DevTools для выполнения инструкций. Автоматизирует такие задачи, как сбор данных, манипулирование формами, захват снимков экрана и мониторинг сети. По сравнению с Voyager (тестирование E2E) он ориентирован на выполнение задач. Используйте его, когда необходима автоматизация браузера.
