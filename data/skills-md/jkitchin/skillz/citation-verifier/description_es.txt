Verificar citas y referencias en documentos científicos para detectar fuentes alucinadas o inválidas.
Extrae DOI, URL, ID de arXiv, ID de PubMed e ISBN de Markdown, LaTeX, modo org y texto sin formato.
luego los valida mediante búsquedas API y recuperaciones web.

Utilice esta habilidad cuando:
- Revisar el contenido generado por IA para garantizar la precisión de las citas.
- Validar referencias en artículos, informes o documentación.
- Comprobar si los DOI/URL se corresponden con documentos reales
- Auditar un documento en busca de citas rotas o falsas.
