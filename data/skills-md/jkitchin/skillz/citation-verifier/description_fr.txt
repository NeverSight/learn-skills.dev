Vérifiez les citations et les références dans les documents scientifiques pour détecter les sources hallucinées ou invalides.
Extrait les DOI, les URL, les identifiants arXiv, les identifiants PubMed et les ISBN de Markdown, LaTeX, du mode org et du texte brut,
puis les valide à l'aide de recherches API et de récupérations Web.

Utilisez cette compétence lorsque :
- Examen du contenu généré par l'IA pour vérifier l'exactitude des citations
- Valider les références dans les articles, rapports ou documentations
- Vérifier si les DOI/URL correspondent à des articles réels
- Audit d'un document pour les citations cassées ou fausses
