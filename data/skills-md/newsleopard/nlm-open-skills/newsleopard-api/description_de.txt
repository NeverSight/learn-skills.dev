API-Referenz und Codegenerierung für die NewsLeopard EDM-API (api.newsleopard.com) und die SureNotify-Transaktions-API (mail.surenotifyapi.com). Verwenden Sie diese Option, wenn der Benutzer Code schreiben, Integrationen erstellen, REST-Endpunkte aufrufen oder API-Endpunktdetails und Feldspezifikationen für diese spezifischen Dienste nachschlagen möchte. Deckt Kontakte, Kampagnen, A/B-Tests, Berichte, Vorlagen, Automatisierung, E-Mail, SMS, Webhooks und Domain-Verifizierung ab. NICHT auslösen für: (1) allgemeine E-Mail-/SMS-Aufgaben ohne Erwähnung von NewsLeopard oder SureNotify, (2) betriebliche Anfragen wie das Senden von E-Mails, das Überprüfen von Kontoständen oder das Pausieren von Kampagnen (auch wenn darin Nachrichten/NewsLeopard erwähnt werden – das sind MCP-Server-Aufgaben, keine API-Codierungsaufgaben), (3) MCP-Server-Vorgänge.
