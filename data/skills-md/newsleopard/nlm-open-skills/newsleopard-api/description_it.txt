Riferimento API e generazione di codice per l'API NewsLeopard EDM (api.newsleopard.com) e l'API transazionale SureNotify (mail.surenotifyapi.com). Da utilizzare quando l'utente chiede di scrivere codice, creare integrazioni, chiamare endpoint REST o cercare i dettagli dell'endpoint API e le specifiche dei campi per questi servizi specifici. Copre contatti, campagne, test A/B, report, modelli, automazione, e-mail, SMS, webhook e verifica del dominio. NON attivare per: (1) attività generiche di posta elettronica/SMS senza menzionare NewsLeopard o SureNotify, (2) richieste operative come l'invio di e-mail, il controllo dei saldi o la sospensione delle campagne (anche se menzionano 電子豹/NewsLeopard: si tratta di attività del server MCP, non attività di codifica API), (3) operazioni del server MCP.
