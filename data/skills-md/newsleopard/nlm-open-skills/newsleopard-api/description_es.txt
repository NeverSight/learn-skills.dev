Referencia de API y generación de código para la API NewsLeopard EDM (api.newsleopard.com) y la API transaccional SureNotify (mail.surenotifyapi.com). Úselo cuando el usuario solicite escribir código, crear integraciones, llamar a puntos finales REST o buscar detalles de puntos finales API y especificaciones de campo para estos servicios específicos. Cubre contactos, campañas, pruebas A/B, informes, plantillas, automatización, correo electrónico, SMS, webhooks y verificación de dominio. NO active para: (1) tareas genéricas de correo electrónico/SMS sin mencionar NewsLeopard o SureNotify, (2) solicitudes operativas como enviar correos electrónicos, verificar saldos o pausar campañas (incluso si mencionan 電子豹/NewsLeopard; esas son tareas del servidor MCP, no tareas de codificación API), (3) operaciones del servidor MCP.
