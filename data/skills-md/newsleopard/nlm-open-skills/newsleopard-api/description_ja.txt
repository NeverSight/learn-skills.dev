NewsLeopard EDM API (api.newsleopard.com) および SureNotify トランザクション API (mail.surenotifyapi.com) の API リファレンスとコード生成。ユーザーがコードの作成、統合の構築、REST エンドポイントの呼び出し、またはこれらの特定のサービスの API エンドポイントの詳細とフィールド仕様の検索を要求する場合に使用します。連絡先、キャンペーン、A/B テスト、レポート、テンプレート、自動化、電子メール、SMS、Webhook、ドメイン検証をカバーします。次の場合はトリガーしないでください: (1) NewsLeopard または SureNotify について言及していない一般的な電子メール/SMS タスク、(2) 電子パンの送信、残高の確認、キャンペーンの一時停止などの操作リクエスト (電子パン/NewsLeopard について言及している場合でも、これらは MCP サーバーのタスクであり、API コーディング タスクではありません)、(3) MCP サーバーの操作。
