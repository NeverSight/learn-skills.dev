NewsLeopard EDM API(api.newsleopard.com) 및 SureNotify 트랜잭션 API(mail.surenotifyapi.com)에 대한 API 참조 및 코드 생성입니다. 사용자가 코드 작성, 통합 구축, REST 엔드포인트 호출 또는 이러한 특정 서비스에 대한 API 엔드포인트 세부 정보 및 필드 사양 조회를 요청할 때 사용합니다. 연락처, 캠페인, A/B 테스트, 보고서, 템플릿, 자동화, 이메일, SMS, 웹훅 및 도메인 확인을 다룹니다. 다음의 경우에는 트리거하지 마십시오: (1) NewsLeopard 또는 SureNotify를 언급하지 않은 일반 이메일/SMS 작업, (2) 이메일 보내기, 잔액 확인 또는 캠페인 일시 중지와 같은 운영 요청(전자豹/NewsLeopard를 언급하더라도 — API 코딩 작업이 아닌 MCP 서버 작업임), (3) MCP 서버 작업.
