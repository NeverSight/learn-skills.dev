Référence API et génération de code pour l'API NewsLeopard EDM (api.newsleopard.com) et l'API transactionnelle SureNotify (mail.surenotifyapi.com). À utiliser lorsque l'utilisateur demande à écrire du code, à créer des intégrations, à appeler des points de terminaison REST ou à rechercher des détails sur les points de terminaison d'API et les spécifications de champs pour ces services spécifiques. Couvre les contacts, les campagnes, les tests A/B, les rapports, les modèles, l'automatisation, les e-mails, les SMS, les webhooks et la vérification de domaine. Ne déclenchez PAS pour : (1) les tâches génériques de courrier électronique/SMS sans mentionner NewsLeopard ou SureNotify, (2) les demandes opérationnelles telles que l'envoi d'e-mails, la vérification des soldes ou la suspension des campagnes (même si elles mentionnent 電子豹/NewsLeopard - ce sont des tâches du serveur MCP, pas des tâches de codage d'API), (3) les opérations du serveur MCP.
