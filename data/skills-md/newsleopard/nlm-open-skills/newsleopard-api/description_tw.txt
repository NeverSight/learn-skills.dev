NewsLeopard EDM API (api.newsleopard.com) 和 SureNotify 事務 API (mail.surenotifyapi.com) 的 API 參考和程式碼產生。當使用者要求編寫程式碼、建置整合、呼叫 REST 端點或查找這些特定服務的 API 端點詳細資訊和欄位規格時使用。涵蓋聯絡人、活動、A/B 測試、報告、範本、自動化、電子郵件、簡訊、網路鉤子和網域驗證。請勿觸發：(1) 未提及 NewsLeopard 或 SureNotify 的通用電子郵件/簡訊任務，(2) 發送電子郵件、檢查餘額或暫停活動等操作請求（即使提及電子豹/NewsLeopard — 這些是 MCP 伺服器任務，而不是 API 編碼任務），(3) MCP 伺服器操作。
