Ссылка на API и создание кода для API NewsLeopard EDM (api.newsleopard.com) и транзакционного API SureNotify (mail.surenotifyapi.com). Используйте, когда пользователь просит написать код, построить интеграцию, вызвать конечные точки REST или найти сведения о конечных точках API и спецификации полей для этих конкретных служб. Охватывает контакты, кампании, A/B-тестирование, отчеты, шаблоны, автоматизацию, электронную почту, SMS, веб-перехватчики и проверку домена. НЕ запускайте для: (1) общих задач электронной почты/SMS без упоминания NewsLeopard или SureNotify, (2) рабочих запросов, таких как отправка электронных писем, проверка баланса или приостановка кампаний (даже если в них упоминается 電子豹/NewsLeopard — это задачи MCP-сервера, а не задачи кодирования API), (3) операции MCP-сервера.
