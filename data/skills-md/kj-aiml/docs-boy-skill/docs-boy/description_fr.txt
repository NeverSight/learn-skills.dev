Générateur et mainteneur de documentation de projet complète. Se déclenche lorsque l'utilisateur demande de créer des documents, d'écrire de la documentation, de mettre à jour des documents, de générer un journal des modifications, de créer un jalon, une documentation d'audit, une documentation de configuration pour un nouveau projet ou avant de s'engager pour générer UPDATE-SUMMARY et un journal des modifications. Utilisez-le de manière proactive lorsque des modifications importantes du code sont apportées ou que de nouvelles fonctionnalités sont implémentées.
