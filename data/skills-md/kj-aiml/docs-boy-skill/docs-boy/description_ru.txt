Комплексный генератор проектной документации и сопровождающий. Срабатывает, когда пользователь запрашивает создание документации, запись документации, обновление документации, создание журнала изменений, создание контрольной точки, документации по аудиту, настройке документации для нового проекта или перед фиксацией создания UPDATE-SUMMARY и журнала изменений. Используйте активно, когда вносятся значительные изменения в код или реализуются новые функции.
