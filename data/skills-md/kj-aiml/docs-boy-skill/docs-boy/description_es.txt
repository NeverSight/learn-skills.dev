Generador y mantenedor integral de documentación de proyectos. Se activa cuando el usuario solicita crear documentos, escribir documentación, actualizar documentos, generar un registro de cambios, crear un hito, auditar documentación, configurar documentación para un nuevo proyecto o antes de confirmar para generar ACTUALIZACIÓN-RESUMEN y registro de cambios. Úselo de forma proactiva cuando se realicen cambios importantes en el código o se implementen nuevas funciones.
