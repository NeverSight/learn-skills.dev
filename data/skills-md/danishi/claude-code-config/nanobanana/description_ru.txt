Создавайте и редактируйте изображения с помощью Google Gemini 3 Pro Image (Nano Banana Pro). Поддерживает преобразование текста в изображение, редактирование изображений с использованием эталонных изображений, настраиваемые соотношения сторон, вывод 1K/2K/4K, заземление поиска Google и пакетную генерацию. Работает как с Gemini Developer API, так и с Vertex AI.
