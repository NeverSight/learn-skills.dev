Genere y edite imágenes usando Google Gemini 3 Pro Image (Nano Banana Pro). Admite generación de texto a imagen, edición de imágenes con imágenes de referencia, relaciones de aspecto configurables, salida de 1K/2K/4K, conexión a tierra de la Búsqueda de Google y generación por lotes. Funciona tanto con Gemini Developer API como con Vertex AI.
