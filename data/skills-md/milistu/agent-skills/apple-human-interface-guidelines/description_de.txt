Referenz zu den Apple Human Interface Guidelines (HIG) zum Entwerfen von iOS-, iPadOS-, macOS-, tvOS-, visionOS- und watchOS-Apps. Behandelt UI-Komponenten, Layout, Barrierefreiheit, Typografie, Navigation, Eingaben und Plattformtechnologien. Verwenden Sie es beim Entwerfen von Apple-Plattform-Benutzeroberflächen, beim Überprüfen von SwiftUI/UIKit-Mustern oder beim Anwenden von HIG-Designprinzipien auf eine beliebige App.
