Комплексная команда сквозного тестирования. Запускает параллельные субагенты для исследования базы кода (структура, схема базы данных, потенциальные ошибки), затем использует интерфейс командной строки Vercel Agent Browser для тестирования каждого взаимодействия пользователя — создания снимков экрана, проверки пользовательского интерфейса и взаимодействия с пользователем и запроса к базе данных для проверки записей. Запустите после реализации, чтобы проверить все перед проверкой кода.
