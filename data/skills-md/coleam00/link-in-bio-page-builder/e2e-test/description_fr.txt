Commande de test complète de bout en bout. Lance des sous-agents parallèles pour rechercher la base de code (structure, schéma de base de données, bogues potentiels), puis utilise la CLI du navigateur d'agent Vercel pour tester chaque parcours utilisateur - en prenant des captures d'écran, en validant l'interface utilisateur/UX et en interrogeant la base de données pour vérifier les enregistrements. Exécutez après la mise en œuvre pour tout valider avant la révision du code.
