Úselo al planificar o implementar proyectos de Vue 3: ayuda a diseñar la estructura de los componentes, planificar la implementación de funciones y aplicar los primeros patrones de TypeScript con la API de composición, definir el modelo para enlaces, biblioteca de pruebas para pruebas de comportamiento del usuario y MSW para simulaciones de API. Especialmente útil en la fase de planificación para guiar los patrones adecuados antes de escribir el código.
