Accédez à 97 outils de service cloud via le serveur Sapiom MCP : grattez des sites Web, exécutez du code dans des bacs à sable, stockez des données (Redis/Vector/Postgres), recherchez sur le Web, appelez des LLM, générez des images et de l'audio, envoyez des messages et gérez la gouvernance des agents. À utiliser lorsque l'utilisateur dit « utiliser sapiom », « gratter cette page », « rechercher sur le Web », « exécuter ce code dans le cloud », « stocker des données », « générer une image », « créer une base de données », « configurer une file d'attente de messages » ou a besoin d'un service cloud payant sans configuration de compte fournisseur. Ne PAS utiliser lorsque les compétences sapiom-deploy ou sapiom-agent-builder sont chargées (celles-ci sont plus spécifiques).
