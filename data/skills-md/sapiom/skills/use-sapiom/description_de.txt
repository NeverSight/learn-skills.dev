Greifen Sie über den Sapiom MCP-Server auf 97 Cloud-Service-Tools zu – durchsuchen Sie Websites, führen Sie Code in Sandboxes aus, speichern Sie Daten (Redis/Vector/Postgres), durchsuchen Sie das Web, rufen Sie LLMs auf, generieren Sie Bilder und Audio, senden Sie Nachrichten und verwalten Sie die Agenten-Governance. Wird verwendet, wenn der Benutzer sagt: „Sapiom verwenden“, „Diese Seite scannen“, „im Internet suchen“, „Diesen Code in der Cloud ausführen“, „Daten speichern“, „ein Bild generieren“, „eine Datenbank erstellen“, „eine Nachrichtenwarteschlange einrichten“ oder einen kostenpflichtigen Cloud-Dienst ohne Einrichtung eines Anbieterkontos benötigt. NICHT verwenden, wenn die Fertigkeiten sapiom-deploy oder sapiom-agent-builder geladen sind (diese sind spezifischer).
