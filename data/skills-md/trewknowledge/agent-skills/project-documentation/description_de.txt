Erstellen Sie nichttechnische Dokumentation für Projektfunktionen im Ordner docs/. Verwenden Sie diese Funktion, wenn Sie abgeschlossene Funktionen dokumentieren, eine Funktionsdokumentation schreiben, erklären, wie Funktionen funktionieren, oder wenn der Benutzer darum bittet, das Projekt zu dokumentieren oder Dokumente für Nicht-Entwickler zu erstellen.
