Habilidad de automatización del navegador de uso general para Playwright. Utilice esta habilidad cuando el usuario desee automatizar tareas del navegador, como probar flujos de inicio de sesión, extraer datos, tomar capturas de pantalla, completar formularios, hacer clic en elementos o cualquier automatización web interactiva. Esta habilidad proporciona utilidades inteligentes para la gestión de sesiones, el manejo de errores y la detección de servidores de desarrollo. NO para monitoreo programado (use web-monitor-bot en su lugar).
