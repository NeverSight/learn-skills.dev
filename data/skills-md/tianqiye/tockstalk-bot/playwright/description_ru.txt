Общие навыки автоматизации браузера для драматурга. Используйте этот навык, когда пользователь хочет автоматизировать задачи браузера, такие как тестирование потоков входа в систему, сбор данных, создание снимков экрана, заполнение форм, нажатие элементов или любую интерактивную веб-автоматизацию. Этот навык предоставляет интеллектуальные утилиты для управления сеансами, обработки ошибок и обнаружения серверов разработки. НЕ для запланированного мониторинга (вместо этого используйте веб-монитор-бот).
