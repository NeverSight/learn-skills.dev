Esta habilidad debe usarse cuando el usuario desea monitorear un sitio web en busca de cambios, disponibilidad o condiciones específicas. Utilice esta habilidad para tareas como rastrear el stock de productos, monitorear los espacios para citas, buscar boletos para eventos, verificar reservas en restaurantes o cualquier escenario que requiera verificaciones web periódicas automatizadas con análisis de patrones. La habilidad crea un sistema de monitoreo completo con el bot Playwright, programación cron, panel de análisis y notificaciones inteligentes.
