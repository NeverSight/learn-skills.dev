Publicación automática de Codebase.blog a través de MCPorter + OAuth. Palabras desencadenantes: 자동포스팅, 자동포스팅해, 포스팅해줘, 블로그 포스팅해줘, 글 발행해줘, crear publicación, publicar publicación. Regla de ruta: habilidad -> mcporter (/mcp-remote), mcp -> MCP directo (/mcp), valor predeterminado ambiguo para habilidad.
