Cette compétence doit être utilisée lors de la configuration ou du dépannage de la ligne d'état de Claude Code à l'aide de l'outil ccstatusline. Il fournit des conseils d'installation, de configuration et de personnalisation pour afficher les informations sur le modèle, l'état Git, l'utilisation du contexte et d'autres informations utiles dans la ligne d'état Claude Code.
