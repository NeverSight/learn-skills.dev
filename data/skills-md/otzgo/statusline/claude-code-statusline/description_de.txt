Diese Fähigkeit sollte bei der Konfiguration oder Fehlerbehebung der Statuszeile von Claude Code mit dem ccstatusline-Tool verwendet werden. Es bietet Installations-, Konfigurations- und Anpassungsanleitungen für die Anzeige von Modellinformationen, Git-Status, Kontextverwendung und anderen nützlichen Informationen in der Claude Code-Statuszeile.
