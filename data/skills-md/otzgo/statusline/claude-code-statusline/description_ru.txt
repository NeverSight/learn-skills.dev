Этот навык следует использовать при настройке или устранении неполадок строки состояния Claude Code с помощью инструмента ccstatusline. Он предоставляет рекомендации по установке, настройке и настройке для отображения информации о модели, состояния Git, использования контекста и другой полезной информации в строке состояния кода Claude.
