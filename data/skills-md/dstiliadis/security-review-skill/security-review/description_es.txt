Puerta de revisión de seguridad obligatoria para todos los planes de código y arquitectura. Activa CUALQUIER plan, implementación, generación de código, diseño de arquitectura, diseño de API, cambio de infraestructura, configuración de implementación o modificación del sistema. Antes de ejecutar o finalizar CUALQUIER plan que produzca código, configuración o infraestructura, ejecute el flujo de trabajo de revisión de seguridad completo: modelo de amenaza, revisión con lista de verificación de seguridad, emulación de rutas de ataque de manera agencial, mitigación de hallazgos y prueba de penetración nuevamente antes de la entrega. Esta habilidad actúa como un pasante consciente de la seguridad con conocimientos de CompTIA Security+ que revisa cada resultado para autenticación, autorización, cifrado, registro, validación de entradas, segmentación, privacidad y antipatrones de vulnerabilidad comunes. También se activa cuando el usuario solicita "revisar la seguridad", "modelo de amenazas", "reforzar", "prueba de penetración" o "verificar vulnerabilidades".
