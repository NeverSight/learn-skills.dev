Recherchez la documentation officielle de l'écosystème Laravel à l'aide de la recherche de vecteurs sémantiques. Utilisez cette compétence UNIQUEMENT lorsque des informations spécifiques à la version sont nécessaires pour un package de l'écosystème Laravel (par exemple, vérifier les modifications de l'API entre les versions, confirmer la syntaxe actuelle pour une version spécifique ou vérifier les fonctionnalités disponibles dans une version particulière). N'utilisez PAS cette compétence pour des connaissances générales sur Laravel que l'agent connaît déjà, uniquement lorsqu'une documentation précise et spécifique à la version est requise. Couvre laravel/framework, livewire, inertia, filament et d'autres packages d'écosystème. IMPORTANT – Utilisez des requêtes techniques de 2 à 4 mots (par exemple, "hasMany appartient à"), PAS des questions en langage naturel (par exemple, "comment créer une relation").
