Verwenden Sie diese Fähigkeit NUR, wenn versionspezifische Informationen für ein Laravel-Ökosystempaket benötigt werden (z. B. Überprüfung von API-Änderungen zwischen Versionen, Bestätigung der aktuellen Syntax für eine bestimmte Version oder Überprüfung der in einer bestimmten Version verfügbaren Funktionen). Verwenden Sie diesen Skill NICHT für allgemeines Laravel-Wissen, das der Agent bereits kennt – nur, wenn eine genaue, versionenspezifische Dokumentation erforderlich ist. Deckt Laravel/Framework-, Livewire-, Inertia-, Filament- und andere Ökosystempakete ab. WICHTIG – Verwenden Sie technische Abfragen mit 2 bis 4 Wörtern (z. B. „hasMany gehört zu“), KEINE Fragen in natürlicher Sprache (z. B. „Wie erstellt man eine Beziehung“).
