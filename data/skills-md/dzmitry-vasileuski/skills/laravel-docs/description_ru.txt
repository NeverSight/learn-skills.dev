Найдите официальную документацию экосистемы Laravel, используя поиск семантического вектора. Используйте этот навык ТОЛЬКО тогда, когда для пакета экосистемы Laravel необходима информация, специфичная для версии (например, проверка изменений API между версиями, подтверждение текущего синтаксиса для конкретной версии или проверка функций, доступных в конкретном выпуске). НЕ используйте этот навык для общих знаний Laravel, которые уже знает агент — только тогда, когда требуется точная документация для конкретной версии. Охватывает laravel/framework, livewire, inertia, filament и другие пакеты экосистемы. ВАЖНО. Используйте технические запросы из 2–4 слов (например, «hasMany ownTo»), а НЕ вопросы на естественном языке (например, «как создать отношения»).
