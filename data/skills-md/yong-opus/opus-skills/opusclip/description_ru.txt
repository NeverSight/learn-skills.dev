Превратите длинные видео в короткие клипы с помощью API OpusClip. Используйте, когда пользователь хочет вырезать видео на YouTube, загрузить локальное видео для вырезания, управлять коллекциями клипов, составить список шаблонов брендов, публично поделиться проектами, подвергнуть цензуре ненормативную лексику или выполнить любую задачу, связанную с OpusClip. Запускается по таким фразам, как «вырезать это видео», «создать короткометражки», «opusclip», «сделать клипы из видео», «загрузить в opusclip».
