通用产品需求文档 (PRD) 是常用交互对话的一部分。我们将根据问题的“问题”进行调整，对问题进行分类，并总结谈判的必要条件。 El 输出是 un .md orientado a Problemas de negocio y requisitos (NO técnico)。 Usar cuando el usuario quiere create un PRD, documentar requisitos de Producto, definir un Problema a Solutionr, o dice "quiero un PRD", "documentar requisitos", "definir qué vamos a construir".
