Verwalten Sie die Sicherung und Synchronisierung der KI-Agent-Skills über die Skills-Manager-CLI. Verwenden Sie diese Option, wenn der Benutzer darum bittet, Fertigkeiten auf GitHub zu übertragen, Fertigkeiten von GitHub abzurufen, Fertigkeiten mit Agenten zu verknüpfen/symlinken, Fertigkeiten maschinenübergreifend zu synchronisieren, Fertigkeiten auf Projektebene einzurichten oder das Verzeichnis ~/.agents/ zu verwalten. Löst auf „Push-Skills“, „Pull-Skills“, „Link-Skills“, „Sync-Skills“, „Backup-Skills“, „Restore-Skills“, „Skills-Manager“, „Link --Project“ und „Projekt-Skills-Setup“ aus.
