Управляйте резервным копированием и синхронизацией навыков агентов ИИ с помощью интерфейса командной строки менеджера навыков. Используйте, когда пользователь просит отправить навыки на GitHub, извлечь навыки из GitHub, связать или символически связать навыки с агентами, синхронизировать навыки между компьютерами, настроить навыки на уровне проекта или управлять каталогом ~/.agents/. Запускается по «навыкам push», «навыкам извлечения», «навыкам связи», «навыкам синхронизации», «навыкам резервного копирования», «навыкам восстановления», «менеджеру навыков», «связыванию --project», «настройке навыков проекта».
