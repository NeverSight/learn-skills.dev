سير عمل تطوير منظم يفصل بين البحث والتخطيط والتنفيذ في مراحل متميزة مع عناصر تخفيض الأسعار المستمرة. يُستخدم عند بدء أي ميزة غير تافهة، أو إعادة البناء، أو التحقيق في الأخطاء، أو تغيير قاعدة التعليمات البرمجية. قم بتشغيل: "العمل العميق"، "البحث والتخطيط"، "التخطيط قبل البرمجة"، "اكتب خطة"، "ابحث في قاعدة التعليمات البرمجية هذه"، "لا تبرمج بعد"، "افهم ثم نفذ"، أو عندما يريد المستخدم اتباع نهج منضبط لمهمة معقدة. استخدمه أيضًا عندما يقول المستخدم "بحث"، أو "خطة"، أو "تعليق توضيحي"، أو "تنفيذ الخطة"، أو يشير إلى عناصر Research.md/plan.md.
