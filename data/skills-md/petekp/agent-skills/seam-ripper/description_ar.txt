قم بتحليل الطبقات المعمارية بلا رحمة - الواجهات والحدود والعقود بين مكونات النظام - لكشف مشاكل الاقتران، وتسريبات التجريد، وفشل التصميم. يُستخدم عندما يُطلب منك مراجعة البنية، أو تحليل الاقتران، أو البحث عن مشكلات في الواجهة، أو تحسين حدود الوحدة النمطية، أو تدقيق التبعيات، أو إعادة تصميم بنية النظام. يقدم مقترحات إعادة تصميم لا هوادة فيها والتي تعطي الأولوية للصحة على التوافق مع الإصدارات السابقة.
