Diese Fähigkeit sollte beim Bereinigen von Codebasen verwendet werden, in denen sich toter Code, redundante Implementierungen und verwaiste Artefakte angesammelt haben – insbesondere Codebasen, die von Codierungsagenten verwaltet werden. Wird ausgelöst bei „toten Code finden“, „unbenutzten Code bereinigen“, „überflüssigen Code entfernen“, „diese Codebasis bereinigen“, „toten Code entfernen“, „Codebereinigung“ oder wenn eine Codebasis mehrere agentengesteuerte Refaktorierungen durchlaufen hat und wahrscheinlich übersehene Reste enthält. Identifiziert Cruft systematisch, kategorisiert Ergebnisse und entfernt bestätigten toten Code mit Zustimmung des Benutzers.
