Questa abilità dovrebbe essere utilizzata durante la pulizia delle basi di codice che hanno accumulato codice morto, implementazioni ridondanti e artefatti orfani, in particolare le basi di codice gestite da agenti di codifica. Si attiva su "trova codice morto", "ripulisci codice inutilizzato", "rimuovi codice ridondante", "elimina questa base di codice", "spazzamento di codice morto", "pulizia del codice" o quando una base di codice è passata attraverso più refactoring gestiti da agenti e probabilmente contiene resti trascurati. Identifica sistematicamente il cruft, classifica i risultati e rimuove il codice morto confermato con l'approvazione dell'utente.
