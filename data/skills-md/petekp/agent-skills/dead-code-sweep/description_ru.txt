Этот навык следует использовать при очистке кодовых баз, в которых накопился мертвый код, избыточные реализации и потерянные артефакты, особенно кодовых баз, поддерживаемых агентами кодирования. Срабатывает при «найти мертвый код», «очистить неиспользуемый код», «удалить избыточный код», «обрезать эту кодовую базу», «очистку мертвого кода», «очистку кода» или когда база кода прошла через несколько рефакторингов, управляемых агентом, и, вероятно, содержит пропущенные остатки. Систематически выявляет ошибки, классифицирует обнаруженные ошибки и удаляет подтвержденный мертвый код с одобрения пользователя.
