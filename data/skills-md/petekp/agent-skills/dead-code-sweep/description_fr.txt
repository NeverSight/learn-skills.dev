Cette compétence doit être utilisée lors du nettoyage des bases de code qui ont accumulé du code mort, des implémentations redondantes et des artefacts orphelins, en particulier les bases de code gérées par les agents de codage. Déclencheurs sur "trouver le code mort", "nettoyer le code inutilisé", "supprimer le code redondant", "élaguer cette base de code", "balayer le code mort", "nettoyer le code", ou lorsqu'une base de code a traversé plusieurs refactors pilotés par des agents et contient probablement des restes négligés. Identifie systématiquement les erreurs, catégorise les résultats et supprime le code mort confirmé avec l'approbation de l'utilisateur.
