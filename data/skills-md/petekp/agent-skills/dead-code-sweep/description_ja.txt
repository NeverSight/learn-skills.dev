このスキルは、デッド コード、冗長な実装、孤立したアーティファクトが蓄積されたコードベース、特にコーディング エージェントによって保守されているコードベースをクリーンアップするときに使用する必要があります。 「デッド コードの検索」、「未使用のコードのクリーンアップ」、「冗長コードの削除」、「このコードベースのプルーニング」、「デッド コード スイープ」、「コード クリーンアップ」、またはコードベースがエージェント主導の複数のリファクタリングを経て、見落とされた残骸が含まれている可能性がある場合にトリガーされます。不正なコードを体系的に特定し、結果を分類し、確認されたデッドコードをユーザーの承認を得て削除します。
