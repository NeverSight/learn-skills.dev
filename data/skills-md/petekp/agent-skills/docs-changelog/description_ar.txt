اكتب إدخالات سجل التغيير لمواقع التوثيق مفتوحة المصدر باستخدام الاحتفاظ بتنسيق سجل التغيير. يُستخدم عندما يُطلب منك "كتابة سجل التغيير"، أو "تحديث سجل التغيير"، أو "إضافة إدخال سجل التغيير"، أو "توثيق التغييرات الأخيرة"، أو بعد إصدار/مجموعة من التغييرات التي يجب تسجيلها. تُنفذ المراجعات git منذ آخر إدخال في سجل التغيير وتنتج إدخالاً مصنفًا يمكن قراءته بواسطة الإنسان.
