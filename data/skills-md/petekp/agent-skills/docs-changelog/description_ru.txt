Создавайте записи журнала изменений для сайтов документации с открытым исходным кодом, используя формат Keep a Changelog. Используйте, когда вас просят «написать журнал изменений», «обновить журнал изменений», «добавить запись в журнал изменений», «задокументировать недавние изменения» или после выпуска/набора изменений, которые следует записать. Просматривает коммиты git с момента последней записи в журнале изменений и создает категоризированную, удобочитаемую запись.
