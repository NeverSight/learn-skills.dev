Rédigez des entrées du journal des modifications pour les sites de documentation open source en utilisant le format Keep a Changelog. À utiliser lorsqu'on vous demande de « rédiger un journal des modifications », « mettre à jour le journal des modifications », « ajouter une entrée du journal des modifications », « documenter les modifications récentes » ou après une version/un ensemble de modifications qui doivent être enregistrées. Examine les commits de git depuis la dernière entrée du journal des modifications et produit une entrée catégorisée et lisible par l'homme.
