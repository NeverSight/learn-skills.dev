Scrivi voci del registro delle modifiche per i siti di documentazione open source utilizzando il formato Mantieni un registro delle modifiche. Da utilizzare quando viene richiesto di "scrivere un registro delle modifiche", "aggiornare il registro delle modifiche", "aggiungere una voce al registro delle modifiche", "documentare le modifiche recenti" o dopo un rilascio/insieme di modifiche che dovrebbero essere registrate. Controlla i commit di git dall'ultima voce del log delle modifiche e produce una voce classificata e leggibile dall'uomo.
