إطار عمل قائم على الشرائح ومبني على الأدلة لتدقيق وترحيل قواعد التعليمات البرمجية المثقلة بالديون. يُستخدم لإعادة البناء الكبيرة، أو دمج قاعدة التعليمات البرمجية، أو سداد ديون التكنولوجيا، أو توحيد التبعية، أو تنظيف البنية، أو أي ترحيل متعدد الجلسات. استخدم أيضًا عندما يُطلب منك تدقيق قاعدة التعليمات البرمجية، أو الترحيل من X إلى Y، أو توحيد نهج واحد، أو دمج التعليمات البرمجية المكررة، أو تشغيل إعادة بناء منهجية مع التنظيف المضمن، وعمليات مسح البقايا، وبوابة الاستعداد النهائية للسفينة. تأكد من استخدام هذه المهارة عندما يريد المستخدم ترحيلًا متحكمًا به باستخدام الشرائح أو حواجز الحماية أو عمليات التسليم أو التنظيف أو إغلاق الإصدار، حتى لو لم يذكر "الترحيل" صراحةً. يجب أن تؤدي الإشارات إلى السقاطة أو الشرائح أو أدلة اللعب الخاصة بالهجرة أو الانضباط المضاد للآثار أو التقارب أو تنظيف الإصدار إلى إثارة ذلك بقوة.
