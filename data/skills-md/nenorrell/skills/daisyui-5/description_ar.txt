أفضل الممارسات والأنماط والاستخدامات لمكتبة مكونات DaisyUI 5 لـ Tailwind CSS 4. استخدمها عند إنشاء واجهة مستخدم بأسماء فئات daisyUI، أو إنشاء تخطيطات قائمة على daisyUI، أو تصميم HTML باستخدام مكونات daisyUI، أو إنشاء مكونات غلاف React لـ daisyUI، أو أي مهمة تتضمن ملفات daisyUI (ملفات .html، .jsx، .tsx، .vue، .svelte). المشغلات على: مكونات daisyUI (btn، البطاقة، المشروطة، الدرج، شريط التنقل، وما إلى ذلك)، أسماء ألوان daisyUI (الأساسية، الثانوية، الأساسية 100، وما إلى ذلك)، تكوين daisyUI (@plugin "daisyui")، daisy-meta.ts، generator-daisy-safelist، المكونات المركبة التي تغلف daisyUI، أو أي مهمة واجهة مستخدم في مشروع يستخدم daisyUI/Tailwind CSS 4.
