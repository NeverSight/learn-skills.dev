Рекомендации, шаблоны и использование библиотеки компонентов DaisyUI 5 для Tailwind CSS 4. Используйте при создании пользовательского интерфейса с именами классов daisyUI, создании макетов на основе daisyUI, стилизации HTML с помощью компонентов daisyUI, создании компонентов оболочки React для daisyUI или любых задачах, связанных с daisyUI (файлы .html, .jsx, .tsx, .vue, .svelte). Запускается по: компонентам daisyUI (кнопка, карта, модальное окно, ящик, панель навигации и т. д.), именам цветов daisyUI (основной, вторичный, базовый 100 и т. д.), конфигурации daisyUI (@plugin "daisyui"), daisy-meta.ts,generate-daisy-safelist, составным компонентам, обертывающим daisyUI, или любой задаче пользовательского интерфейса в проекте, использующей daisyUI/Tailwind CSS 4.
