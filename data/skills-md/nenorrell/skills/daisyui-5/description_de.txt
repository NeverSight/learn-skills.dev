Best Practices, Muster und Verwendung der DaisyUI 5-Komponentenbibliothek für Tailwind CSS 4. Verwendung beim Erstellen einer Benutzeroberfläche mit daisyUI-Klassennamen, beim Erstellen von daisyUI-basierten Layouts, beim Stylen von HTML mit daisyUI-Komponenten, beim Erstellen von React-Wrapper-Komponenten für daisyUI oder bei jeder Aufgabe, die daisyUI betrifft (.html-, .jsx-, .tsx-, .vue-, .svelte-Dateien). Löst aus: daisyUI-Komponenten (BTN, Karte, Modal, Schublade, Navigationsleiste usw.), daisyUI-Farbnamen (primär, sekundär, Basis-100 usw.), daisyUI-Konfiguration (@plugin „daisyui“), daisy-meta.ts, generic-daisy-safelist, zusammengesetzte Komponenten, die daisyUI umschließen, oder jede UI-Aufgabe in einem Projekt mit daisyUI/Tailwind CSS 4.
