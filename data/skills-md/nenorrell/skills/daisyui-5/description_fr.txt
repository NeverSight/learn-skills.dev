Meilleures pratiques, modèles et utilisation de la bibliothèque de composants DaisyUI 5 pour Tailwind CSS 4. À utiliser lors de la création d'une interface utilisateur avec des noms de classe DaisyUI, de la création de mises en page basées sur DaisyUI, du style HTML avec des composants DaisyUI, de la création de composants wrapper React pour DaisyUI ou de toute tâche impliquant DaisyUI (fichiers .html, .jsx, .tsx, .vue, .svelte). Déclencheurs sur : les composants DaisyUI (btn, card, modal, Drawer, navbar, etc.), les noms de couleurs DaisyUI (primaire, secondaire, base-100, etc.), la configuration DaisyUI (@plugin "daisyui"), Daisy-meta.ts, generate-daisy-safelist, les composants composés enveloppant DaisyUI ou toute tâche d'interface utilisateur dans un projet utilisant DaisyUI/Tailwind CSS 4.
