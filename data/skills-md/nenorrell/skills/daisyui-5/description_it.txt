Best practice, modelli e utilizzo della libreria di componenti DaisyUI 5 per Tailwind CSS 4. Da utilizzare durante la creazione di un'interfaccia utente con nomi di classi daisyUI, la creazione di layout basati su daisyUI, l'applicazione di stili HTML con componenti daisyUI, la creazione di componenti wrapper React per daisyUI o qualsiasi attività che coinvolga daisyUI (file .html, .jsx, .tsx, .vue, .svelte). Si attiva su: componenti daisyUI (btn, card, modal, drawer, navbar, ecc.), nomi dei colori daisyUI (primario, secondario, base-100, ecc.), configurazione daisyUI (@plugin "daisyui"), daisy-meta.ts, generate-daisy-safelist, componenti composti che racchiudono daisyUI o qualsiasi attività dell'interfaccia utente in un progetto utilizzando daisyUI/Tailwind CSS 4.
