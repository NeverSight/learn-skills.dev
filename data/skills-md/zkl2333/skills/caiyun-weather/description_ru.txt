Получите данные о погоде через API Цайюнь (彩云天气) через MCP. Используйте, когда: пользователь спрашивает о погоде, температуре, прогнозах, качестве воздуха или оповещениях о погоде для любого места в Китае. Поддерживает режим реального времени, ежечасно (72 часа), ежедневно (7 дней), исторические (24 часа) и оповещения. Триггеры: «天气», «погода», «气温», «空气质量», «AQI», «降水», «预报». НЕ для: мест за пределами Китая (вместо этого используйте навыки погоды).
