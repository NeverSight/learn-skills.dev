Этот навык следует использовать, когда пользователь хочет увидеть все установленные плагины, агенты, навыки и серверы MCP, а также проверить текущий репозиторий на наличие локальных агентов, навыков и конфигурации MCP. Сканирует среду и представляет структурированный каталог доступных ресурсов. Работает на всех платформах AI CLI (Claude Code, GitHub Copilot, Gemini CLI, OpenCode, OpenAI Codex).
