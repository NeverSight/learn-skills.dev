Esta habilidad debe usarse cuando el usuario necesita convertir archivos entre formatos (documentos, imágenes, video, audio, PDF, presentaciones), realizar operaciones de PDF (OCR, fusionar, dividir, rotar, cifrar, descifrar) o procesar archivos que pandoc y docling no admiten. Se activa cuando el usuario dice "convertir este archivo", "transformar a PDF", "extraer texto con OCR", "fusionar archivos PDF", "comprimir video", "convertir pptx a pdf", "convertir imagen a webp" o menciona CloudConvert explícitamente. Utilice esta habilidad siempre que sea necesaria la conversión de formato o la manipulación de PDF, incluso si el usuario no nombra la herramienta.
