يجب استخدام هذه المهارة عندما يحتاج المستخدم إلى تحويل الملفات بين التنسيقات (المستندات والصور والفيديو والصوت وملفات PDF والعروض التقديمية)، أو إجراء عمليات PDF (التعرف الضوئي على الحروف، أو الدمج، أو الانقسام، أو التدوير، أو التشفير، أو فك التشفير)، أو معالجة الملفات التي لا يدعمها pandoc وdocling. يتم تشغيله عندما يقول المستخدم "تحويل هذا الملف" أو "التحويل إلى PDF" أو "استخراج النص باستخدام التعرف الضوئي على الحروف" أو "دمج ملفات PDF" أو "ضغط الفيديو" أو "تحويل pptx إلى pdf" أو "تحويل الصورة إلى webp" أو ذكر CloudConvert صراحةً. استخدم هذه المهارة عندما تكون هناك حاجة إلى تحويل التنسيق أو معالجة PDF، حتى لو لم يذكر المستخدم اسم الأداة.
