Esta habilidad debe usarse cuando el usuario necesita realizar un seguimiento de diferentes versiones de currículum, mantener un currículum maestro y administrar versiones personalizadas para múltiples aplicaciones. Úselo cuando haga malabarismos con múltiples objetivos de trabajo, realice un seguimiento de qué versión se envió y dónde o mantenga una fuente de verdad centralizada del currículum.
