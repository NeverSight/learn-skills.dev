Diese Fähigkeit sollte verwendet werden, wenn der Benutzer verschiedene Lebenslaufversionen verfolgen, einen Master-Lebenslauf verwalten und maßgeschneiderte Versionen für mehrere Anwendungen verwalten muss. Verwenden Sie es, wenn Sie mehrere Jobziele unter einen Hut bringen, nachverfolgen möchten, welche Version wohin gesendet wurde, oder eine zentrale Informationsquelle für Lebensläufe pflegen.
