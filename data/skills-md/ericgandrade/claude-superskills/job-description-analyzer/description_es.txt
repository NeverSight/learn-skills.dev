Esta habilidad debe usarse cuando el usuario necesita analizar una oferta de trabajo, calcular puntuaciones de coincidencia entre el currículum y el trabajo, identificar brechas de habilidades y crear una estrategia de solicitud. Úselo al evaluar la idoneidad para un puesto específico, extraer requisitos clave o preparar materiales específicos para currículums y cartas de presentación.
