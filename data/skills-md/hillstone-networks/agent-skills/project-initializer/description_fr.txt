Élabore de nouveaux projets avec README.md, AGENTS.md et CI/CD (GitLab CI, GitHub Actions). Gère le type de projet (générique / backend Flask / frontend React / mini-application Taro), la pile technologique, les normes de codage, le niveau de qualité et le SDD (OpenSpec, SpecKit, GSD). Tous les flux d'initialisation (Flask, React, Taro) et conventions (backend-python-cicd, frontend-codegen, flask-backend-codegen, QA/testing, agents-rôles/sous-agents) sont intégrés ; pas de compétences distinctes. Les documents sont en chinois par défaut. À utiliser lors de la création d'un projet, de l'initialisation d'un dépôt ou de la configuration de CI/CD/SDD.
