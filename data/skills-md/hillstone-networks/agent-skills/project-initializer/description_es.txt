Da soporte a nuevos proyectos con README.md, AGENTS.md y CI/CD (GitLab CI, GitHub Actions). Maneja el tipo de proyecto (genérico/backend de Flask/frontend de React/miniaplicación Taro), pila tecnológica, estándares de codificación, nivel de calidad y SDD (OpenSpec, SpecKit, GSD). Todos los flujos de inicio (Flask, React, Taro) y las convenciones (backend-python-cicd, frontend-codegen, flask-backend-codegen, control de calidad/pruebas, roles de agente/subagentes) están integrados; sin habilidades separadas. Los documentos están predeterminados en chino. Úselo al crear un proyecto, inicializar un repositorio o configurar CI/CD/SDD.
