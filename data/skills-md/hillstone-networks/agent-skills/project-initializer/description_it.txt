Supporta nuovi progetti con README.md, AGENTS.md e CI/CD (GitLab CI, GitHub Actions). Gestisce il tipo di progetto (generico/backend Flask/frontend React/miniapp Taro), stack tecnologico, standard di codifica, livello di qualità e SDD (OpenSpec, SpecKit, GSD). Tutti i flussi di init (Flask, React, Taro) e le convenzioni (backend-python-cicd, frontend-codegen, flask-backend-codegen, QA/testing, agent-roles/subagents) sono integrati; nessuna abilità separata. Per impostazione predefinita, i documenti sono in cinese. Da utilizzare durante la creazione di un progetto, l'inizializzazione di un repository o la configurazione di CI/CD/SDD.
