أنشئ تطبيقات Next.js التي تعمل بالذكاء الاصطناعي باستخدام Giselle Agent SDK. يُستخدم عندما يريد المطورون (1) إضافة وكيل AI إلى تطبيق Next.js، (2) إنشاء وكيل مدعوم بأداة متصفح يتفاعل مع الصفحة، (3) إعداد Giselle Agent SDK باستخدام أدوات useChat وstreamText والمتصفح، (4) تعريف وكيل السقالة ومسار الدردشة وواجهة المستخدم باستخدام معرف أداة متصفح البيانات. مشغلات على "وكيل giselle"، و"وكيل البناء"، و"إضافة وكيل إلى تطبيقي"، و"وكيل AI في Next.js"، و"وكيل أدوات المتصفح"، و"وكيل جداول البيانات".
