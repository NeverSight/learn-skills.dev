Erstellen Sie KI-Agent-gestützte Next.js-Apps mit dem Giselle Agent SDK. Verwenden Sie diese Option, wenn Entwickler (1) einen KI-Agenten zu einer Next.js-App hinzufügen, (2) einen browsertoolbasierten Agenten erstellen möchten, der mit der Seite interagiert, (3) das Giselle Agent SDK mit useChat, streamText und Browsertools einrichten möchten, (4) Gerüstagentendefinition, Chatroute und Benutzeroberfläche mit data-browser-tool-id. Löst auf „Giselle-Agent“, „Build-Agent“, „Agent zu meiner App hinzufügen“, „KI-Agent in Next.js“, „Browser-Tool-Agent“ und „Tabellenkalkulationsagent“ aus.
