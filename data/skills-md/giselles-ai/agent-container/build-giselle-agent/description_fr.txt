Créez des applications Next.js basées sur un agent IA avec le SDK Giselle Agent. À utiliser lorsque les développeurs souhaitent (1) ajouter un agent IA à une application Next.js, (2) créer un agent alimenté par un outil de navigateur qui interagit avec la page, (3) configurer le SDK Giselle Agent avec useChat, streamText et les outils de navigateur, (4) la définition de l'agent d'échafaudage, l'itinéraire de discussion et l'interface utilisateur avec l'ID de l'outil de navigateur de données. Déclencheurs sur "agent giselle", "agent de construction", "ajouter un agent à mon application", "agent IA dans Next.js", "agent de l'outil de navigateur", "agent de feuille de calcul".
