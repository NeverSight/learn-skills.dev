Создавайте приложения Next.js на основе AI-агента с помощью Giselle Agent SDK. Используйте, когда разработчики хотят (1) добавить агент AI в приложение Next.js, (2) создать агент на базе браузера, который взаимодействует со страницей, (3) настроить Giselle Agent SDK с помощью useChat,streamText и инструментов браузера, (4) определение агента scaffold, маршрут чата и пользовательский интерфейс с идентификатором инструмента браузера данных. Запускается по «агенту Жизель», «агенту сборки», «добавить агент в мое приложение», «агенту AI в Next.js», «агенту инструмента браузера», «агенту электронных таблиц».
