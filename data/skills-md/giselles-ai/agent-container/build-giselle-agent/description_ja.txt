Giselle Agent SDK を使用して、AI エージェントを利用した Next.js アプリを構築します。開発者が、(1) Next.js アプリに AI エージェントを追加する、(2) ページと対話するブラウザー ツールを利用したエージェントを作成する、(3) useChat、streamText、およびブラウザー ツールを使用して Giselle Agent SDK をセットアップする、(4) data-browser-tool-id を使用してスキャフォールド エージェント定義、チャット ルート、UI を設定する場合に使用します。 「giselle エージェント」、「ビルド エージェント」、「アプリへのエージェントの追加」、「Next.js の AI エージェント」、「ブラウザ ツール エージェント」、「スプレッドシート エージェント」でトリガーされます。
