Erstellen Sie atomare Git-Commits mit konventionellen Commits und Gitmoji. Verwenden Sie diese Option, wenn der Benutzer nach einem Commit für Änderungen fragt, einen Commit erstellt oder „/commit“ sagt. Überprüfen Sie bereitgestellte und nicht bereitgestellte Unterschiede, teilen Sie Änderungen in logische Gruppen auf und schreiben Sie gezielte Commit-Nachrichten, die den lokalen Konventionen des Repositorys folgen.
