Создавайте атомарные коммиты git с помощью обычных коммитов и Gitmoji. Используйте, когда пользователь просит зафиксировать изменения, создать фиксацию или говорит «/commit». Просматривайте проиндексированные и непроиндексированные различия, разбивайте изменения на логические группы и пишите целевые сообщения о фиксации, соответствующие локальным соглашениям репозитория.
