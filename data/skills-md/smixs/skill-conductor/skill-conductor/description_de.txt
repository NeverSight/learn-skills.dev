Erstellen, bearbeiten, bewerten und bündeln Sie Agentenfähigkeiten. Verwenden Sie diese Funktion, wenn Sie eine neue Fertigkeit von Grund auf erstellen, eine vorhandene Fertigkeit verbessern, Evaluierungen zum Testen einer Fertigkeit durchführen, die Fertigkeitsleistung bewerten, die Beschreibung einer Fertigkeit für eine bessere Auslösung optimieren, Fertigkeiten von Drittanbietern auf Qualität überprüfen oder Fertigkeiten für die Verteilung verpacken. Nicht für den Einsatz von Fertigkeiten oder allgemeine Codierungsaufgaben.
