의사 계획 모드 아키텍처를 사용한 대화형 코드 멘토링.
AI는 시각적 청사진(다이어그램 + 시퀀스 흐름)을 심층적으로 분석하고 설계하며 제시합니다.
인간이 계획을 승인한 후 AI 지침에 따라 모든 코드를 작성합니다.
멘토는 선택적으로 내구성 있는 스캐폴딩으로 승인된 대상 파일에 제한된 지원 댓글을 삽입할 수 있습니다.
"완료" 후에 AI는 행동의 정확성(구현 적합성이 아님)을 확인합니다.

"이해력 부채" 해결 - 개발자가 자신이 작성한 모든 줄을 이해할 수 있도록 보장합니다.

주요 사용 사례(기존 코드베이스):
- 대규모 코드베이스의 기능 수정
- 기존 프로젝트에 새로운 기능 추가
- 전체 상황 이해를 통한 버그 수정
- 패턴 이해를 통한 리팩토링

보조 사용 사례:
- 실습 코딩을 통해 새로운 기술 스택 학습

사용 시기: 기존 코드 수정, 기능 추가, 버그 수정,
리팩토링, 새로운 프레임워크 학습, 실습 코딩 안내를 원함,
"썼지만 이해하지 못함" 증후군을 피하세요.

키워드: 멘토, 안내 코딩, 수정, 기능, 버그 수정, 리팩터링,
기존 코드베이스, 실습, 이해, 소유권, 학습, 계획 모드
