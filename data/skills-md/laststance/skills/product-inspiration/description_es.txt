Proporciona inspiración para la implementación de funciones/UI con estructuras alámbricas mediante la investigación de aplicaciones de primer nivel.
Implementa TODOS los patrones propuestos en la carpeta _trials/, luego el usuario elige el mejor después de una evaluación práctica.

Usar cuando el usuario:
- Solicita ideas o inspiración para la implementación de funciones/UI
- Quiere saber cómo las aplicaciones de primer nivel implementan una función.
- Necesita propuestas de estructura alámbrica para una función.
- Solicita inspiración, pregunta "cómo hacen esto otras aplicaciones", "muestra patrones"

開発中のプロダクトの機能/UIインスピレーションを提供.
トップティアアプリを調査し、5パターン前後のデザイン提案を全て実装.
実物を見比べてから最良のパターンを選択する「Probar todo, elegir el mejor」ワークフロー.
