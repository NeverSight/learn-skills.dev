Fournit une inspiration pour la mise en œuvre de l'interface utilisateur/des fonctionnalités avec des wireframes en recherchant des applications de premier plan.
Implémente TOUS les modèles proposés dans le dossier _trials/, puis l'utilisateur choisit le meilleur après une évaluation pratique.

À utiliser lorsque l'utilisateur :
- Demande des idées ou de l'inspiration pour la mise en œuvre de l'interface utilisateur/des fonctionnalités
- Veut savoir comment les applications de premier plan implémentent une fonctionnalité
- A besoin de propositions filaires pour une fonctionnalité
- Demande l'inspiration, demande "comment les autres applications font cela", "affiche les modèles"
