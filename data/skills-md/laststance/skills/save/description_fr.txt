Enregistrez le contexte de session dans la mémoire Serena MCP pour la persistance entre sessions.
Analyse les réalisations, conserve les apprentissages sous forme de souvenirs structurés,
crée un point de contrôle pour la restauration de session future.
Portable sur tous les agents compatibles Serena (Claude Code, Cursor, Windsurf, etc.).

À utiliser lorsque : mettre fin à une session, interrompre le travail, enregistrer la progression ou préserver les apprentissages.
Mots-clés : session, sauvegarde, persistance, point de contrôle, mémoire, stockage, contexte, fin
