Сохраните контекст сеанса в памяти Serena MCP для обеспечения устойчивости между сеансами.
Анализирует достижения, сохраняет полученные знания в виде структурированных воспоминаний,
создает контрольную точку для будущего восстановления сеанса.
Переносимость для всех агентов с поддержкой Serena (Claude Code, Cursor, Windsurf и т. д.).

Используйте, когда: завершение сеанса, приостановка работы, сохранение прогресса или сохранение результатов обучения.
Ключевые слова: сеанс, сохранение, сохранение, контрольная точка, память, сохранение, контекст, конец.
