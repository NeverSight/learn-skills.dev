Standard-Implementierungsworkflow für alle Codierungsaufgaben.
Führt einen systematischen 5-Phasen-Zyklus aus: Untersuchen → Planen → Implementieren → Überprüfen → Abschließen.
Integriert Serena Think Checkpoints, Selbstbeobachtungsmarkierungen und Qualitätstore.
Unterstützt das Flag --frontend-verify für die visuelle Überprüfung von Browser/App/CLI.

Verwenden Sie, wenn:
– Der Benutzer bittet darum, eine Funktion zu implementieren, einen Fehler zu beheben oder Code umzugestalten
– Der Benutzer stellt eine Aufgabe bereit, die Codeänderungen erfordert
- Der Benutzer sagt „Tu dies“, „Erstelle dies“, „Repariere dies“, „Füge dies hinzu“
- Jegliche Implementierungsarbeiten, die Codebearbeitung beinhalten

Schlüsselwörter: Aufgabe, implementieren, erstellen, reparieren, hinzufügen, erstellen, umgestalten, aktualisieren, ändern
