Trazador de ruta de ejecución de código interactivo que explica cómo fluye el código desde el punto de entrada hasta la salida.
Utiliza navegación paso a paso con AskUserQuestion para explorar ramas condicionales y llamadas a funciones.

Úselo cuando:
- El usuario pregunta "¿Cómo funciona X en este código base?"
- El usuario quiere entender el flujo de solicitud/respuesta HTTP.
- El usuario pregunta sobre el orden de ejecución del middleware.
- El usuario quiere rastrear una cadena de llamadas a funciones.
- El usuario pregunta "¿Qué sucede cuando..."?
- El usuario quiere aprender cómo se conectan las rutas de código.

Palabras clave: rastreo, flujo, ejecución, ruta, cadena de llamadas, middleware, manejo de solicitudes,
qué sucede, cómo sucede, avanza, sigue el código
