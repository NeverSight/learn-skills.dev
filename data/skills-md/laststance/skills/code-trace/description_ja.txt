コードがエントリ ポイントから出力までどのように流れるかを説明する対話型のコード実行パス トレーサー。
AskUserQuestion によるステップバイステップのナビゲーションを使用して、条件分岐と関数呼び出しを探索します。

次の場合に使用します。
- ユーザーが「X はこのコードベースでどのように動作しますか?」と尋ねます。
- ユーザーは HTTP リクエスト/レスポンス フローを理解したいと考えています
- ユーザーはミドルウェアの実行順序について尋ねます
- ユーザーは関数呼び出しチェーンをトレースしたい
- ユーザーは「次の場合はどうなりますか?」という質問をします。
- ユーザーはコードパスがどのように接続されるかを知りたいと考えています

キーワード: トレース、フロー、実行、パス、コール チェーン、ミドルウェア、リクエスト処理、
何が起こるか、どのように行われるか、ステップ実行し、コードに従います
