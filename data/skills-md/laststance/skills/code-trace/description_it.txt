Tracciatore interattivo del percorso di esecuzione del codice che spiega come il codice scorre dal punto di ingresso all'output.
Utilizza la navigazione dettagliata con AskUserQuestion per esplorare rami condizionali e chiamate di funzione.

Utilizzare quando:
- L'utente chiede "Come funziona X in questa codebase?"
- L'utente desidera comprendere il flusso di richiesta/risposta HTTP
- L'utente chiede informazioni sull'ordine di esecuzione del middleware
- L'utente desidera tracciare una catena di chiamate di funzione
- L'utente pone domande "Cosa succede quando...".
- L'utente desidera sapere come si connettono i percorsi del codice

Parole chiave: traccia, flusso, esecuzione, percorso, catena di chiamate, middleware, gestione delle richieste,
cosa succede, come procedere, seguire il codice
