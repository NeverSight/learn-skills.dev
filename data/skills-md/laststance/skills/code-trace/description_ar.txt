متتبع مسار تنفيذ التعليمات البرمجية التفاعلي الذي يشرح كيفية تدفق التعليمات البرمجية من نقطة الإدخال إلى الإخراج.
يستخدم التنقل خطوة بخطوة مع AskUserQuestion لاستكشاف الفروع الشرطية واستدعاءات الوظائف.

استخدم عندما:
- يسأل المستخدم "كيف يعمل X في قاعدة التعليمات البرمجية هذه؟"
- يريد المستخدم فهم تدفق طلب/استجابة HTTP
- يسأل المستخدم عن أمر تنفيذ البرامج الوسيطة
- يريد المستخدم تتبع سلسلة استدعاء دالة
- يسأل المستخدم أسئلة "ماذا يحدث عندما...".
- يريد المستخدم معرفة كيفية اتصال مسارات التعليمات البرمجية

الكلمات المفتاحية: التتبع، التدفق، التنفيذ، المسار، سلسلة الاتصال، البرامج الوسيطة، معالجة الطلب،
ماذا يحدث، كيف، الخطوة، اتباع التعليمات البرمجية
