Traceur interactif du chemin d'exécution du code qui explique comment le code circule du point d'entrée à la sortie.
Utilise la navigation étape par étape avec AskUserQuestion pour explorer les branches conditionnelles et les appels de fonction.

À utiliser lorsque :
- L'utilisateur demande "Comment X fonctionne-t-il dans cette base de code ?"
- L'utilisateur souhaite comprendre le flux de requête/réponse HTTP
- L'utilisateur pose des questions sur l'ordre d'exécution du middleware
- L'utilisateur souhaite tracer une chaîne d'appel de fonction
- L'utilisateur pose des questions "Que se passe-t-il quand..."
- L'utilisateur souhaite savoir comment les chemins de code se connectent

Mots clés : trace, flux, exécution, chemin, chaîne d'appels, middleware, traitement des requêtes,
que se passe-t-il, comment, parcourez-vous, suivez le code
