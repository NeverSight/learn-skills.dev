Interaktiver Code-Ausführungspfad-Tracer, der erklärt, wie der Code vom Einstiegspunkt zur Ausgabe fließt.
Verwendet die schrittweise Navigation mit AskUserQuestion, um bedingte Verzweigungen und Funktionsaufrufe zu erkunden.

Verwenden Sie, wenn:
– Der Benutzer fragt: „Wie funktioniert X in dieser Codebasis?“
– Der Benutzer möchte den HTTP-Anfrage-/Antwortfluss verstehen
– Der Benutzer fragt nach der Ausführungsreihenfolge der Middleware
– Der Benutzer möchte eine Funktionsaufrufkette verfolgen
- Der Benutzer stellt „Was passiert, wenn...“-Fragen
– Der Benutzer möchte erfahren, wie Codepfade verbunden sind

Schlüsselwörter: Ablaufverfolgung, Ablauf, Ausführung, Pfad, Aufrufkette, Middleware, Anforderungsbearbeitung,
Was passiert, wie funktioniert es, gehen Sie durch, befolgen Sie den Code
