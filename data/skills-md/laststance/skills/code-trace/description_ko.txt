코드가 진입점에서 출력까지 어떻게 흐르는지 설명하는 대화형 코드 실행 경로 추적기입니다.
AskUserQuestion과 함께 단계별 탐색을 사용하여 조건부 분기 및 함수 호출을 탐색합니다.

다음과 같은 경우에 사용하세요:
- 사용자가 "이 코드베이스에서 X는 어떻게 작동하나요?"라고 묻습니다.
- 사용자는 HTTP 요청/응답 흐름을 이해하고 싶어합니다.
- 사용자가 미들웨어 실행 순서를 묻는 경우
- 사용자가 함수 호출 체인을 추적하고 싶어합니다.
- 사용자가 "..."라고 묻는 질문
- 사용자가 코드 경로가 어떻게 연결되는지 알고 싶어 합니다.

키워드: 추적, 흐름, 실행, 경로, 호출 체인, 미들웨어, 요청 처리,
무슨 일이 일어나는가, 어떻게 하는가, 단계별로 진행하다, 코드를 따르다
