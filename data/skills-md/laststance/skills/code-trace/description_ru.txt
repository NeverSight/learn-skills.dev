Интерактивный трассировщик пути выполнения кода, который объясняет, как код передается от точки входа к выходу.
Использует пошаговую навигацию с помощью AskUserQuestion для изучения условных ветвей и вызовов функций.

Используйте, когда:
- Пользователь спрашивает: «Как X работает в этой кодовой базе?»
- Пользователь хочет понять поток HTTP-запросов/ответов.
- Пользователь спрашивает о порядке выполнения промежуточного программного обеспечения.
- Пользователь хочет отследить цепочку вызовов функций.
- Пользователь задает вопросы «Что произойдет, когда…»
- Пользователь хочет узнать, как соединяются пути кода.

Ключевые слова: трассировка, поток, выполнение, путь, цепочка вызовов, промежуточное программное обеспечение, обработка запросов,
что происходит, как происходит, проходите, следуйте коду
