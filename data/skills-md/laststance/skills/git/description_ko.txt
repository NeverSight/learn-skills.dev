지능형 커밋 메시지 및 워크플로 최적화를 통한 Git 작업.
변경 사항을 분석하여 기존 커밋 메시지를 자동으로 생성합니다.
상태, 커밋, 푸시, 풀, 분기 및 병합 작업을 다룹니다.

다음과 같은 경우에 사용하세요:
- 사용자가 git 상태를 커밋, 푸시 또는 확인하도록 요청합니다.
- 사용자가 브랜치를 생성하거나 전환하려는 경우
- 사용자가 병합 또는 충돌 해결에 도움이 필요함
- 사용자가 "이것을 커밋", "푸시", "변경된 내용", "브랜치 만들기"라고 말합니다.
- 모든 Git 워크플로 작업

키워드: git, 커밋, 푸시, 풀, 상태, 분기, 병합, 리베이스, 숨김, 로그, diff
