Операции Git с интеллектуальными сообщениями о фиксации и оптимизацией рабочего процесса.
Анализирует изменения для автоматического создания сообщений об обычной фиксации.
Охватывает операции статуса, фиксации, отправки, извлечения, ветвления и слияния.

Используйте, когда:
- Пользователь просит зафиксировать, отправить или проверить статус git.
- Пользователь хочет создать или переключить ветки
- Пользователю нужна помощь в слиянии или разрешении конфликтов.
- Пользователь говорит «зафиксировать это», «отправить», «что изменилось», «создать ветку».
- Любая операция рабочего процесса git.

Ключевые слова: git, коммит, push, pull, статус, ветка, слияние, перебазирование, тайник, журнал, разница
