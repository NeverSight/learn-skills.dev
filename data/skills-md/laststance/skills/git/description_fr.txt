Opérations Git avec messages de validation intelligents et optimisation du flux de travail.
Analyse les modifications pour générer automatiquement des messages de validation conventionnelle.
Couvre les opérations d’état, de validation, de poussée, d’extraction, de branchement et de fusion.

À utiliser lorsque :
- L'utilisateur demande de valider, de pousser ou de vérifier le statut de git
- L'utilisateur souhaite créer ou changer de branche
- L'utilisateur a besoin d'aide pour la fusion ou la résolution de conflits
- L'utilisateur dit « valider ceci », « pousser », « ce qui a changé », « créer une branche »
- Toute opération de workflow git

Mots-clés : git, commit, push, pull, status, branch, merge, rebase, stash, log, diff
