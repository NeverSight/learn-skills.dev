Operazioni Git con messaggi di commit intelligenti e ottimizzazione del flusso di lavoro.
Analizza le modifiche per generare automaticamente messaggi di commit convenzionale.
Copre le operazioni di stato, commit, push, pull, branch e merge.

Utilizzare quando:
- L'utente chiede di eseguire il commit, il push o il controllo dello stato di git
- L'utente desidera creare o cambiare ramo
- L'utente ha bisogno di aiuto con l'unione o la risoluzione dei conflitti
- L'utente dice "commetti questo", "push", "cosa è cambiato", "crea un ramo"
- Qualsiasi operazione del flusso di lavoro git

Parole chiave: git, commit, push, pull, status, branch, merge, rebase, stash, log, diff
