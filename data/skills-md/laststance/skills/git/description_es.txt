Operaciones de Git con mensajes de confirmación inteligentes y optimización del flujo de trabajo.
Analiza los cambios para generar mensajes de confirmación convencional automáticamente.
Cubre operaciones de estado, confirmación, inserción, extracción, bifurcación y fusión.

Úselo cuando:
- El usuario solicita confirmar, enviar o verificar el estado de git
- El usuario quiere crear o cambiar de sucursal.
- El usuario necesita ayuda con la fusión o la resolución de conflictos
- El usuario dice "confirmar esto", "empujar", "qué cambió", "crear una rama"
- Cualquier operación de flujo de trabajo de git

Palabras clave: git, commit, push, pull, estado, rama, fusión, rebase, stash, log, diff
