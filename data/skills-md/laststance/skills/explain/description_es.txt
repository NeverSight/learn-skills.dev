Explicación profunda y sistemática del código, los conceptos y el comportamiento del sistema.
Opera siempre a nivel avanzado con marcadores de introspección y validación.
Analiza el objetivo minuciosamente antes de explicarlo, verifica la precisión después.

Úselo cuando:
- El usuario pregunta "¿qué hace esto?", "cómo funciona esto", "explica esto"
- El usuario quiere comprender el código, la arquitectura o un concepto en profundidad.
- El usuario señala un archivo, función o sistema y pide una explicación.
- El usuario dice "explicar", "guiarme", "enseñarme", "desglosar"

Palabras clave: explicar, comprender, cómo, qué, por qué, recorrer, enseñar, desglosar, aclarar.
