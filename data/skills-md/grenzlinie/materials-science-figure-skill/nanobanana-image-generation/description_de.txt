Verwenden Sie diese Option, wenn der Benutzer Bilder mit den Nanobanana/Gemini-Bildmodellen von Google mithilfe der offiziellen Gemini-API-Form generieren oder bearbeiten möchte. Bevorzugen Sie diese Fähigkeit für die Text-zu-Bild-Bearbeitung, die Bild-zu-Bild-Bearbeitung, Arbeitsabläufe mit mehreren Bildreferenzen, auf Anhängen basierende Nachbildungen oder wenn der Benutzer Abbildungen im Publikationsstil wie z. B. Schaltpläne für materialwissenschaftliche Arbeiten wünscht. Verwenden Sie es, wenn der Benutzer eine materialwissenschaftliche Abbildung, eine wissenschaftliche Illustration im Zeitschriftenstil, eine grafische Zusammenfassung, ein Mechanismusdiagramm oder eine papierfertige Materialabbildung wünscht.
