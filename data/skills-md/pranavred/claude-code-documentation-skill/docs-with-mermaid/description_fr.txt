Créez une excellente documentation technique avec les diagrammes Mermaid. À utiliser pour documenter l'architecture de code, les flux d'API, les schémas de base de données, les machines à états, la conception de systèmes ou tout autre concept technique bénéficiant de diagrammes visuels. Utilisez également lorsqu'on vous demande d'expliquer le code, de créer de la documentation, d'écrire des fichiers README ou de documenter le fonctionnement des systèmes.
