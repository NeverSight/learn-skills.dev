Un vocabulario UX compartido para descripciones conversacionales precisas de interfaces. Utilice esta habilidad siempre que el usuario describa una interfaz de usuario, un flujo de UX, una interacción o un diseño usando palabras del glosario de UX, como transformar, crecer, producir, carril, seleccionable, superposición, esqueleto, ágil, marcar y deshacer, etc. También se activa cuando el usuario describe cualquier comportamiento de la interfaz y necesita resolver la ambigüedad sobre lo que significa. Esta habilidad define ~40 palabras con significados exactos para que el usuario pueda describir UX en inglés sencillo sin dejar de ser preciso. Cuando vea estas palabras, NO las interprete a la ligera: tienen definiciones específicas acordadas.
