Ein gemeinsames UX-Vokabular für präzise Konversationsbeschreibungen von Schnittstellen. Verwenden Sie diese Fertigkeit immer dann, wenn der Benutzer eine Benutzeroberfläche, einen UX-Fluss, eine Interaktion oder ein Layout mit Wörtern aus dem UX-Glossar beschreibt – etwa „Morph“, „Grow“, „Yield“, „Rail“, „Selectable“, „Overlay“, „Skelett“, „Snappy“, „Mark-and-Undo“ usw.). Wird auch ausgelöst, wenn der Benutzer ein Schnittstellenverhalten beschreibt und Sie Unklarheiten darüber beseitigen müssen, was sie bedeuten. Diese Fähigkeit definiert etwa 40 Wörter mit genauen Bedeutungen, sodass der Benutzer UX in einfachem Englisch beschreiben und dabei präzise bleiben kann. Wenn Sie diese Wörter sehen, interpretieren Sie sie NICHT unverbindlich – sie haben spezifische, vereinbarte Definitionen.
