مهارة الإصدار 2.1 من تصميم شرائح الميزات الرسمية (FSD) لتطبيق المنهجية على مشاريع الواجهة الأمامية. يُستخدم عندما تتضمن المهمة تنظيم بنية المشروع باستخدام طبقات FSD، أو تحديد المكان الذي تنتمي إليه التعليمات البرمجية، أو تحديد واجهات برمجة التطبيقات العامة وحدود الاستيراد، أو حل عمليات الاستيراد المتبادل أو تقييم نمط @x، أو تحديد ما إذا كان يجب أن يظل المنطق محليًا أو يتم استخراجه، أو الترحيل من FSD v2.0 أو قاعدة تعليمات برمجية غير FSD، أو دمج FSD مع أطر العمل، أو تنفيذ أنماط شائعة مثل المصادقة ومعالجة API وRedux وReact Query داخل FSD.
