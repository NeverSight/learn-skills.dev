Официальный навык Feature-Sliced ​​Design (FSD) v2.1 для применения методологии во внешних проектах. Используйте, когда задача включает в себя организацию структуры проекта с помощью слоев FSD, принятие решения о принадлежности кода, определение общедоступных API и границ импорта, разрешение перекрестного импорта или оценку шаблона @x, принятие решения о том, должна ли логика оставаться локальной или ее следует извлекать, миграцию из FSD v2.0 или кодовой базы, отличной от FSD, интеграцию FSD с платформами или реализацию общих шаблонов, таких как аутентификация, обработка API, Redux и React Query в FSD.
