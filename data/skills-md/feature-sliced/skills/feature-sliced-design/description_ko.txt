프론트엔드 프로젝트에 방법론을 적용하기 위한 공식 FSD(Feature-Sliced ​​Design) v2.1 기술입니다. 작업에 FSD 레이어를 사용하여 프로젝트 구조 구성, 코드가 속하는 위치 결정, 공개 API 및 가져오기 경계 정의, 교차 가져오기 해결 또는 @x 패턴 평가, 논리를 로컬로 유지해야 하는지 또는 추출해야 하는지 결정, FSD v2.0 또는 비FSD 코드베이스에서 마이그레이션, FSD를 프레임워크와 통합 또는 FSD 내에서 인증, API 처리, Redux 및 React Query와 같은 공통 패턴 구현이 포함되는 경우에 사용합니다.
