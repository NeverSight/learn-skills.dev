Suite d'outils permettant de créer des artefacts HTML claude.ai élaborés et multi-composants à l'aide des technologies Web frontales modernes. Prend en charge l'initialisation du projet React 18 + TypeScript + Tailwind CSS + shadcn/ui et Vue 3 + TypeScript + Tailwind CSS, puis regroupe les applications dans un seul artefact HTML. À utiliser pour les artefacts complexes nécessitant une gestion d'état, un routage ou des systèmes de composants - et non pour de simples extraits HTML/JSX/Vue d'un seul fichier.
