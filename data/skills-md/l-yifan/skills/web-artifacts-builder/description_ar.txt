مجموعة من الأدوات لإنشاء عناصر HTML معقدة ومتعددة المكونات claude.ai باستخدام تقنيات الويب الأمامية الحديثة. يدعم React 18 + TypeScript + Tailwind CSS + shadcn/ui وVue 3 + TypeScript + Tailwind CSS، ثم يجمع التطبيقات في قطعة HTML واحدة. يُستخدم مع العناصر المعقدة التي تتطلب إدارة الحالة أو التوجيه أو أنظمة المكونات - وليس لمقتطفات HTML/JSX/Vue البسيطة ذات الملف الواحد.
