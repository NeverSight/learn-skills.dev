Suite di strumenti per la creazione di elaborati artefatti HTML claude.ai multicomponente utilizzando moderne tecnologie web frontend. Supporta React 18 + TypeScript + Tailwind CSS + shadcn/ui e Vue 3 + TypeScript + Tailwind CSS inizializzazione del progetto, quindi raggruppa le app in un singolo artefatto HTML. Utilizzare per artefatti complessi che richiedono gestione dello stato, routing o sistemi di componenti, non per semplici snippet HTML/JSX/Vue a file singolo.
