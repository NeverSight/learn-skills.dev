Conjunto de herramientas para crear artefactos HTML claude.ai elaborados y de múltiples componentes utilizando tecnologías web frontend modernas. Admite la inicialización de proyectos React 18 + TypeScript + Tailwind CSS + shadcn/ui y Vue 3 + TypeScript + Tailwind CSS, luego agrupa aplicaciones en un único artefacto HTML. Úselo para artefactos complejos que requieren administración de estado, enrutamiento o sistemas de componentes, no para fragmentos simples de HTML/JSX/Vue de un solo archivo.
