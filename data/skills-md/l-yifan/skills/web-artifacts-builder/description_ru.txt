Набор инструментов для создания сложных многокомпонентных HTML-артефактов claude.ai с использованием современных веб-технологий внешнего интерфейса. Поддерживает инициализацию проекта React 18 + TypeScript + Tailwind CSS + shadcn/ui и Vue 3 + TypeScript + Tailwind CSS, а затем объединяет приложения в один HTML-артефакт. Используйте для сложных артефактов, требующих управления состоянием, маршрутизации или систем компонентов, а не для простых однофайловых фрагментов HTML/JSX/Vue.
