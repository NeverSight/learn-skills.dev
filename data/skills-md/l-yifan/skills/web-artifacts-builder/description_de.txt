Suite von Tools zum Erstellen komplexer, mehrkomponentiger claude.ai-HTML-Artefakte unter Verwendung moderner Frontend-Webtechnologien. Unterstützt die Projektinitialisierung von React 18 + TypeScript + Tailwind CSS + shadcn/ui und Vue 3 + TypeScript + Tailwind CSS und bündelt dann Apps in einem einzigen HTML-Artefakt. Wird für komplexe Artefakte verwendet, die Zustandsverwaltung, Routing oder Komponentensysteme erfordern – nicht für einfache einzelne HTML-/JSX-/Vue-Snippets.
