BettaFish (Micro-Public-Opinion) Multi-Agent Sentiment Analysis System – built on a three-engine parallel architecture of QueryAgent, MediaAgent, and InsightAgent, with ForumEngine enabling inter-agent collaborative discussion to generate dual-format reports: Word/PDF + high-design-quality HTML.

This skill is triggered when users need:
- Analysis of a brand/company/product’s social-media reputation and word-of-mouth
- Tracking of hot public-opinion events (e.g., an automaker under fire, a celebrity controversy)
- Mining content and influence data of specific social accounts
- Monitoring competitor sentiment dynamics for multi-brand comparisons
- Analyzing public emotion and attitude toward a topic
- Generating sentiment-monitoring reports or crisis early-warning analysis
- Formal reports in Word/PDF format
- High-design-quality interactive HTML visualizations
- Deep sentiment analysis based on real data

The skill employs a QueryAgent + MediaAgent + InsightAgent parallel architecture; ForumEngine coordinates inter-agent discussion, executing three reflective loops to refine results, finally outputting:
1. **Word/PDF documents** – generated via docx/pdf subskill, suitable for formal presentations, printing, archiving
2. **High-design-quality HTML reports** – via frontend-design subskill, unique editorial-magazine style with interactive visuals

**No databases or mock data are used**; all data is fetched in real time through WebSearch/WebFetch/Browser/Curl.

Always use this skill for complex multi-step analyses, multi-source integration, or professional-format sentiment reports.
