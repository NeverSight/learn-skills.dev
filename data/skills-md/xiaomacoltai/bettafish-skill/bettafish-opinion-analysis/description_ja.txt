BettaFish (Micro-Public) マルチエージェント世論分析システム – QueryAgent、MediaAgent、InsightAgent の並列 3 エンジン アーキテクチャに基づいて構築されており、ForumEngine を使用してエージェント間の共同ディスカッションを可能にし、Word/PDF とデザイン品質の高い HTML 形式の両方でレポートを生成できます。

このスキルは、ユーザーが以下を必要とするときにトリガーされます。
- ブランド/企業/製品のソーシャルメディアでの評判と口コミの分析
- 注目を集めている世論イベントの追跡 (例: 自動車メーカーへの批判、有名人の論争)
- 特定のソーシャルメディアアカウントのコンテンツと影響力データのマイニング
- 競合他社の世論動向を監視し、複数ブランドの比較を実施する
- トピックに対する国民の感情と態度を分析する
- 世論監視レポートまたは危機早期警戒分析の作成
- Word/PDF形式の正式なレポートが必要
- デザイン品質の高いインタラクティブな HTML ビジュアル レポートが必要
- 実際のデータに基づいた深い世論分析が必要

このスキルは、QueryAgent + MediaAgent + InsightAgent の並列アーキテクチャを採用しており、ForumEngine によってエージェント間のディスカッションとコラボレーションが可能になり、結果を最適化するために 3 ラウンドのリフレクション ループを実行して、最終的に次の出力が行われます。
1. **Word/PDF ドキュメント** – docx/pdf サブスキルによって生成され、正式なプレゼンテーション、印刷、アーカイブに適しています
2. **高デザイン品質の HTML レポート** – フロントエンド デザインのサブスキル、インタラクティブなビジュアルを備えた独特の編集雑誌スタイルによって生成

**データベースやモックデータは使用しません**。すべてのデータは、WebSearch/WebFetch/Browser/Curl を介してリアルタイムで取得されます。

複雑な複数ステップの分析ニーズ、複数のデータ ソースの統合、または専門的な形式の世論レポートの生成の場合でも、常にこのスキルを使用してください。
