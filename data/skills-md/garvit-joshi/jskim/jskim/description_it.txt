Lettore di file Java con risparmio di token. Da utilizzare quando si lavora con file Java (.java) per ridurre l'utilizzo dei token. Si attiva automaticamente durante l'esplorazione, la lettura o la comprensione di classi, servizi, controller, entità Java o qualsiasi file .java. Esegui jskim prima di utilizzare lo strumento Leggi sui file Java per comprendere prima la struttura, quindi leggi solo le righe necessarie.
