Lecteur de fichiers Java économisant les jetons. À utiliser lorsque vous travaillez avec des fichiers Java (.java) pour réduire l'utilisation des jetons. Se déclenche automatiquement lors de l'exploration, de la lecture ou de la compréhension des classes, services, contrôleurs, entités Java ou de tout fichier .java. Exécutez jskim avant d'utiliser l'outil de lecture sur les fichiers Java pour comprendre d'abord la structure, puis lisez uniquement les lignes dont vous avez besoin.
