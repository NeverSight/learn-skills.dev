Convierta páginas web públicas en Markdown limpio con markdown.new para flujos de trabajo de IA. Úselo cuando las tareas requieran una conversión de URL a Markdown para resumen, ingesta de RAG, extracción, archivado o reducción de tokens, incluida la selección del método de conversión (automático/ai/navegador), la habilitación de la retención de imágenes y el manejo de límites de tasa o fallas de conversión.
