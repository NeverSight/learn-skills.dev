Unendlich laufendes KI-Entwicklungssystem, das komplette Softwareprojekte über unbegrenzte Kontextfenster hinweg erstellt. Verwenden Sie diese Fähigkeit, wenn der Benutzer ein großes Projekt autonom erstellen, einen lang laufenden Codierungsagenten ausführen, eine Auto-Entwicklungsschleife einrichten, eine komplexe App Feature für Feature implementieren möchte oder „unendliche Entwicklung“, „autonome Codierung“, „automatische Entwicklung“, „lang laufenden Agent“ erwähnt oder möchte, dass Claude sitzungsübergreifend weiter baut. Verwenden Sie dies auch, wenn der Benutzer eine feature_list.json oder claude-progress.txt in seinem Projekt hat oder erwähnt, dass er ein großes Projekt in inkrementelle Features aufteilen möchte.
