Guida completa per lo styling dei componenti React con Vanilla Extract (integrazione Vite, stili digitati, temi, composizione, API globali, ricette, temi dinamici di runtime e sprinkles opzionali). Utilizza questa competenza quando le attività dell'utente implicano l'implementazione o il refactoring dello stile dei componenti React con Vanilla Extract, in particolare quando si enfatizza la generazione di CSS a runtime zero, la forte ergonomia di TypeScript, varianti scalabili e modelli di temi sicuri.
