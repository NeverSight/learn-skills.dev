Комплексное руководство по стилизации компонентов React с помощью Vanilla Extract (интеграция с Vite, типизированные стили, темы, композиция, глобальные API, рецепты, динамическое оформление тем во время выполнения и дополнительные элементы разбрызгивания). Используйте этот навык, когда пользовательские задачи включают реализацию или рефакторинг стилей компонентов React с помощью Vanilla Extract, особенно когда особое внимание уделяется созданию CSS с нулевым временем выполнения, сильной эргономике TypeScript, масштабируемым вариантам и безопасным шаблонам тем.
