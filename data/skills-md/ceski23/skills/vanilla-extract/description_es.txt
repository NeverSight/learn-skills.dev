Guía completa para diseñar componentes de React con Vanilla Extract (integración de Vite, estilos escritos, temas, composición, API globales, recetas, temas de tiempo de ejecución dinámico y elementos opcionales). Utilice esta habilidad cuando las tareas del usuario impliquen implementar o refactorizar el estilo de los componentes de React con Vanilla Extract, especialmente cuando se enfatice la generación de CSS en tiempo de ejecución cero, una sólida ergonomía de TypeScript, variantes escalables y patrones de temas seguros.
