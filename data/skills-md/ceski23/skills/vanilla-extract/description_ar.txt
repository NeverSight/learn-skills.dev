دليل شامل لتصميم مكونات React باستخدام Vanilla Extract (تكامل Vite، والأنماط المكتوبة، والسمات، والتركيب، وواجهات برمجة التطبيقات العامة، والوصفات، والسمات الديناميكية في وقت التشغيل، والرشات الاختيارية). استخدم هذه المهارة عندما تتضمن مهام المستخدم تنفيذ أو إعادة تصميم تصميم مكون React باستخدام Vanilla Extract، خاصة عند التركيز على إنشاء CSS بدون وقت تشغيل، وبيئة عمل TypeScript القوية، والمتغيرات القابلة للتطوير، وأنماط السمات الآمنة.
