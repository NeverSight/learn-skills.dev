A deep research and integration analysis tool for GitHub repositories. Supports single/multiple repository studies, comparison with local projects, and heuristic integration suggestions. Features topic-driven search mode: automatically searches for related repositories, clones, analyzes, and generates reports. Clones remote repositories to a local research/ directory, performs deep code analysis, architecture evaluation, dependency resolution, and generates structured research reports. Trigger conditions: when a user provides a GitHub URL requesting research/analysis/integration/comparison, or provides topic keywords to search and study related repositories.
