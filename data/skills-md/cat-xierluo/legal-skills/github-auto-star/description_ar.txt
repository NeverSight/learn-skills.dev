قم باستخراج مستودعات GitHub تلقائيًا من المحتوى (المقالات ولقطات الشاشة والنصوص) وتمييزها بنجمة. حالات الاستخدام: (1) يطلب المستخدمون تحليل المحتوى وتمييز مشاريع GitHub ذات الصلة بنجمة، (2) معالجة المقالات/المدونات التي تشير إلى الأدوات أو المكتبات، (3) تحليل لقطات الشاشة التي تحتوي على مراجع فنية، (4) جدولة المهام الدورية لاكتشاف المستودعات وتمييزها تلقائيًا بنجمة.
