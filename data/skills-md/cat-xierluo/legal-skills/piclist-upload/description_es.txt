Utilice el servidor HTTP PicList para cargar imágenes locales en archivos Markdown a un servicio de alojamiento de imágenes y reemplazarlas con enlaces en la nube. Esta habilidad debe usarse cuando el usuario necesita cargar imágenes en Markdown, procesar Markdown que contiene referencias de imágenes locales, procesar por lotes varios archivos o directorios de Markdown o reemplazar rutas locales con enlaces a la nube para acceso entre dispositivos.
