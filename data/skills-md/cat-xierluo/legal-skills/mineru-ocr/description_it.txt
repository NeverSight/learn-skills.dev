Converti documenti PDF nel formato Markdown utilizzando l'API MinerU, che supporta il riconoscimento OCR, tabelle e formule. Questa capacità deve essere utilizzata quando l'utente deve convertire documenti PDF, Word, PPT o immagini in Markdown o menziona PDF in Markdown, OCR, riconoscimento del testo, riconoscimento delle tabelle, riconoscimento delle formule, conversione di documenti, immagine in testo o conversione di file scansionati.
