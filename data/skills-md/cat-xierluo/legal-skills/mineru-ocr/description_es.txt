Convierta documentos PDF al formato Markdown utilizando la API MinerU, que admite reconocimiento de OCR, tablas y fórmulas. Esta habilidad debe usarse cuando el usuario necesita convertir documentos PDF, Word, PPT o imágenes a Markdown, o menciona PDF a Markdown, OCR, reconocimiento de texto, reconocimiento de tablas, reconocimiento de fórmulas, conversión de documentos, imagen a texto o conversión de archivos escaneados.
