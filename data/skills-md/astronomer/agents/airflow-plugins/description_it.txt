Crea plug-in Airflow 3.1+ che incorporano app FastAPI, pagine dell'interfaccia utente personalizzate, componenti React, middleware, macro e collegamenti operatore direttamente nell'interfaccia utente di Airflow. Utilizza questa competenza ogni volta che l'utente desidera creare un plug-in Airflow, aggiungere una pagina dell'interfaccia utente personalizzata o una voce di navigazione ad Airflow, creare endpoint supportati da FastAPI all'interno di Airflow, fornire risorse statiche da un plug-in, incorporare un'app React nell'interfaccia utente di Airflow, aggiungere middleware al server API Airflow, creare collegamenti aggiuntivi personalizzati per l'operatore o chiamare l'API REST di Airflow dall'interno di un plug-in. Si attiva anche quando l'utente menziona AirflowPlugin, fastapi_apps, external_views, react_apps, registrazione plugin o incorpora un'app Web in Airflow 3.1+. Se qualcuno sta creando qualcosa di personalizzato all'interno di Airflow 3.1+ che coinvolga Python e un'interfaccia rivolta al browser, questa abilità quasi sicuramente si applica.
