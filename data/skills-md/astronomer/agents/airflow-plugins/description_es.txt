Cree complementos de Airflow 3.1+ que incorporen aplicaciones FastAPI, páginas de interfaz de usuario personalizadas, componentes de React, middleware, macros y enlaces de operadores directamente en la interfaz de usuario de Airflow. Utilice esta habilidad siempre que el usuario desee crear un complemento de Airflow, agregar una página de interfaz de usuario personalizada o una entrada de navegación a Airflow, crear puntos finales respaldados por FastAPI dentro de Airflow, servir activos estáticos desde un complemento, incrustar una aplicación React en la interfaz de usuario de Airflow, agregar middleware al servidor API de Airflow, crear enlaces adicionales de operador personalizados o llamar a la API REST de Airflow desde dentro de un complemento. También se activa cuando el usuario menciona AirflowPlugin, fastapi_apps, external_views, react_apps, registro de complementos o integración de una aplicación web en Airflow 3.1+. Si alguien está creando algo personalizado dentro de Airflow 3.1+ que involucre Python y una interfaz orientada al navegador, es casi seguro que esta habilidad se aplica.
