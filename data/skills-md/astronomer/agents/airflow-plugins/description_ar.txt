أنشئ مكونات إضافية لـ Airflow 3.1+ تتضمن تطبيقات FastAPI وصفحات واجهة المستخدم المخصصة ومكونات React والبرامج الوسيطة ووحدات الماكرو وروابط المشغل مباشرةً في واجهة مستخدم Airflow. استخدم هذه المهارة عندما يريد المستخدم إنشاء مكون إضافي لـ Airflow، أو إضافة صفحة واجهة مستخدم مخصصة أو إدخال تنقل إلى Airflow، أو إنشاء نقاط نهاية مدعومة بـ FastAPI داخل Airflow، أو تقديم أصول ثابتة من مكون إضافي، أو تضمين تطبيق React في واجهة مستخدم Airflow، أو إضافة برامج وسيطة إلى خادم Airflow API، أو إنشاء روابط إضافية لمشغل مخصص، أو الاتصال بـ Airflow REST API من داخل مكون إضافي. يتم التشغيل أيضًا عندما يذكر المستخدم AirflowPlugin، أو fastapi_apps، أو External_views، أو React_apps، أو تسجيل البرنامج المساعد، أو تضمين تطبيق ويب في Airflow 3.1+. إذا كان شخص ما يبني أي شيء مخصص داخل Airflow 3.1+ يتضمن لغة Python وواجهة متصفح، فمن المؤكد تقريبًا أن هذه المهارة تنطبق.
