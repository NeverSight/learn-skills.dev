Документация Markdown экспертного уровня НАВЫКИ для создания доработанных файлов README, журналов изменений, руководств по участию и технической документации. Варианты использования: (1) README со значками и разделами, (2) автоматический журнал изменений из истории git, (3) оглавление, (4) рекомендации по участию, (5) форматирование технической документации, (6) документация по коду с подсветкой синтаксиса.
