Direction créative et intelligence de conception pour les vidéos graphiques animées Remotion. À utiliser lorsqu'un agent doit (1) convertir un script ou un SRT en un document de conception visuelle plan par plan, (2) créer des graphiques animés avec une intention éditoriale pour chaque scène, (3) produire un dossier de conception pour l'approbation de l'utilisateur avant d'écrire le code Remotion. Utilisez toujours avant de générer un code vidéo Remotion à partir d'un script.
