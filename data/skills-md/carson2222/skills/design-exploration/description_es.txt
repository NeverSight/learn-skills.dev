Genera múltiples variantes de diseño distintas de un componente o página, cada una con una dirección visual completamente diferente, y luego implementa la elegida. Úselo cuando el usuario solicite rediseñar, cambiar el estilo, explorar opciones de diseño, crear múltiples direcciones visuales o comparar enfoques de diseño para cualquier elemento de la interfaz de usuario: componentes, páginas, secciones, paneles, páginas de destino o diseños completos.
