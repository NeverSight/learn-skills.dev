Этот навык следует использовать, когда пользователь просит «создать проектный документ», «создать схему базы данных», «спроектировать интерфейс API», «нарисовать блок-схему», «создать диаграмму последовательности», «спроектировать архитектуру системы», «спроектировать структуру модуля», «создать диаграмму потока данных» или упоминает техническую проектную документацию. Создает комплексную техническую проектную документацию со схемами баз данных, спецификациями API и диаграммами Mermaid.
