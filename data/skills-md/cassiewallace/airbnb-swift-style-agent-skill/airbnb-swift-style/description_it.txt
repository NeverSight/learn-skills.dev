Applica la Guida allo stile Swift di Airbnb per iOS sul codice Swift. Utilizza questa competenza ogni volta che l'utente desidera: rivedere il codice Swift per verificarne la conformità allo stile, eseguire l'intreccio o controllare una base di codice iOS, applicare le convenzioni Swift di Airbnb al codice nuovo o esistente, capire perché il codice viola le regole di stile di Airbnb o ottenere riscritture corrette degli snippet Swift. Si attiva su qualsiasi richiesta che menziona la qualità del codice Swift, la revisione del codice iOS, lo stile Airbnb, l'linting dei file Swift o quando l'utente incolla il codice Swift e chiede feedback, correzioni o una revisione. Da utilizzare anche quando l'utente chiede di "ripulire" o "formattare" il codice Swift oppure chiede "è buono Swift?" – anche se non menzionano Airbnb per nome.
