يفرض دليل Airbnb iOS Swift Style Guide على رمز Swift. استخدم هذه المهارة في أي وقت يريد فيه المستخدم: مراجعة كود Swift للتأكد من توافق النمط، أو فحص أو تدقيق قاعدة بيانات iOS، أو تطبيق اصطلاحات Airbnb Swift على التعليمات البرمجية الجديدة أو الموجودة، أو فهم سبب انتهاك التعليمات البرمجية لقواعد نمط Airbnb، أو الحصول على عمليات إعادة كتابة مصححة لمقتطفات Swift. يتم تشغيله عند أي طلب يذكر جودة كود Swift، أو مراجعة كود iOS، أو نمط Airbnb، أو فحص ملفات Swift، أو عندما يلصق المستخدم كود Swift ويطلب التعليقات أو الإصلاحات أو المراجعة. يُستخدم أيضًا عندما يطلب المستخدم "تنظيف" أو "تنسيق" كود Swift، أو يسأل "هل هذا Swift جيد؟" - حتى لو لم يذكروا Airbnb بالاسم.
