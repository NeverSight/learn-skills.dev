Erzwingt den Airbnb iOS Swift Style Guide für Swift-Code. Verwenden Sie diese Fähigkeit jederzeit, wenn der Benutzer Swift-Code auf Stilkonformität überprüfen, eine iOS-Codebasis linten oder prüfen, Airbnb-Swift-Konventionen auf neuen oder vorhandenen Code anwenden, verstehen möchte, warum Code gegen Airbnb-Stilregeln verstößt, oder korrigierte Neufassungen von Swift-Snippets erhalten möchte. Wird bei jeder Anfrage ausgelöst, die Swift-Codequalität, iOS-Codeüberprüfung, Airbnb-Stil, Linting von Swift-Dateien erwähnt oder wenn der Benutzer Swift-Code einfügt und um Feedback, Korrekturen oder eine Überprüfung bittet. Wird auch verwendet, wenn der Benutzer Swift-Code „bereinigen“ oder „formatieren“ möchte oder fragt: „Ist das gutes Swift?“ – auch wenn Airbnb nicht namentlich erwähnt wird.
