Aplica la Guía de estilo Swift de iOS de Airbnb en el código Swift. Utilice esta habilidad cada vez que el usuario desee: revisar el código Swift para verificar el cumplimiento del estilo, filtrar o auditar una base de código de iOS, aplicar las convenciones de Airbnb Swift a código nuevo o existente, comprender por qué el código viola las reglas de estilo de Airbnb u obtener reescrituras corregidas de fragmentos de Swift. Se activa cualquier solicitud que mencione la calidad del código Swift, la revisión del código de iOS, el estilo de Airbnb, la eliminación de archivos Swift o cuando el usuario pega el código Swift y solicita comentarios, correcciones o una revisión. Úselo también cuando el usuario solicite "limpiar" o "formatear" el código Swift, o pregunte "¿es este un buen Swift?" – incluso si no mencionan a Airbnb por su nombre.
