Комплексные принципы проектирования пользовательского интерфейса и UX для разработки программного обеспечения. Этот навык следует использовать при проектировании интерфейсов, реализации взаимодействия с пользователем, обеспечении доступности, оптимизации производительности и создании адаптивных макетов. Запускается при выполнении задач, связанных с компонентами пользовательского интерфейса, потоками взаимодействия с пользователем, требованиями доступности, адаптивным дизайном или реализацией системы дизайна.
