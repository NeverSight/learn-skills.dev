مبادئ تصميم واجهة المستخدم وتجربة المستخدم الشاملة لتطوير البرمجيات. يجب استخدام هذه المهارة عند تصميم الواجهات، وتنفيذ تفاعلات المستخدم، وضمان إمكانية الوصول، وتحسين الأداء، وإنشاء تخطيطات سريعة الاستجابة. تشغيل المهام التي تتضمن مكونات واجهة المستخدم، أو تدفقات تجربة المستخدم، أو متطلبات إمكانية الوصول، أو التصميم سريع الاستجابة، أو تنفيذ نظام التصميم.
