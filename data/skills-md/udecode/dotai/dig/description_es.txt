Busque documentación y código fuente para bibliotecas y paquetes. Úselo cuando el usuario haga una pregunta sobre una biblioteca, necesite comprender la API de una biblioteca o cuando necesite información sobre una biblioteca que no conoce. Se activan preguntas como "¿Cómo uso la biblioteca X?", "¿Cuál es la API para Y?", "Muéstreme cómo maneja esto la biblioteca Z" o cuando se encuentra con un uso de biblioteca desconocido.
