Suchen Sie nach Dokumentation und Quellcode für Bibliotheken und Pakete. Verwenden Sie diese Option, wenn der Benutzer eine Frage zu einer Bibliothek stellt, die API einer Bibliothek verstehen muss oder wenn Sie Informationen über eine Bibliothek benötigen, von der Sie nichts wissen. Wird ausgelöst bei Fragen wie „Wie verwende ich die X-Bibliothek?“, „Was ist die API für Y?“, „Zeigen Sie mir, wie die Z-Bibliothek damit umgeht“ oder wenn Sie auf eine unbekannte Bibliotheksnutzung stoßen.
