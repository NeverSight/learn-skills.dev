Senden Sie eine proaktive Nachricht an einen gekoppelten Benutzer über seinen Kanal (Discord, Telegram oder WhatsApp). Verwenden Sie diese Option, wenn der Agent einen Benutzer benachrichtigen, warnen oder ihm eine unerwünschte Nachricht senden muss – insbesondere bei Heartbeat-Aufrufen, geplanten Aufgaben oder wenn der Agent ohne vorherige Benutzernachricht in der aktuellen Konversation Kontakt aufnehmen möchte. Auslöser: „Nachricht an Benutzer senden“, „Benutzer benachrichtigen“, „Benutzer benachrichtigen“, „dem Benutzer eine Nachricht auf Discord/Telegram/WhatsApp senden“ oder jede Notwendigkeit, proaktiv mit einem gekoppelten Absender zu kommunizieren.
