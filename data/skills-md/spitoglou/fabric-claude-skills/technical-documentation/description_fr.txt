Créez et améliorez la documentation technique, y compris les conceptions d'architecture, les PRD, les documents API, les explications de code et les diagrammes visuels. À utiliser lorsqu'on vous demande de documenter les systèmes, d'expliquer les bases de code, de créer des documents de conception, d'écrire des PRD, de générer des diagrammes Mermaid/PlantUML, de réviser l'architecture, d'écrire des user stories ou de créer des descriptions de demandes d'extraction. Les déclencheurs incluent « documenter ceci », « créer un document de conception », « expliquer ce code », « écrire un PRD », « créer un diagramme », « présentation de l'architecture », « user story pour », « description PR ».
