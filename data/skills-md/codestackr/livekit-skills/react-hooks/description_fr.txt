Créez des interfaces utilisateur React personnalisées avec les hooks LiveKit de @livekit/components-react. Utilisez cette compétence lorsque vous avez besoin d'un contrôle de bas niveau sur l'état de l'agent, les participants, les pistes, le chat et les canaux de données. Pour les composants d’interface utilisateur prédéfinis, utilisez plutôt la compétence livekit-agents-ui.
