Создавайте собственные пользовательские интерфейсы React с помощью хуков LiveKit из @livekit/comComponents-react. Используйте этот навык, когда вам нужен низкоуровневый контроль над состоянием агента, участниками, треками, чатом и каналами данных. Для предварительно созданных компонентов пользовательского интерфейса используйте вместо этого навык livekit-agents-ui.
