Создавайте серверные части агента LiveKit на TypeScript или JavaScript. Используйте этот навык при создании голосовых агентов искусственного интеллекта, голосовых помощников или любого приложения искусственного интеллекта реального времени с помощью SDK агентов Node.js Agents SDK LiveKit (@livekit/agents-js). Охватывает AgentSession, класс агента, функциональные инструменты с zod, модели STT/LLM/TTS, обнаружение поворотов и модели реального времени.
