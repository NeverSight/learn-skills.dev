أنشئ واجهات React الأمامية لعملاء الذكاء الاصطناعي الصوتي LiveKit. استخدم هذه المهارة عند إنشاء واجهات ويب للمساعدين الصوتيين باستخدام مكونات Agents UI المستندة إلى shadcn الخاصة بـ LiveKit. يغطي AgentSessionProvider، والمرئيات الصوتية، وعناصر التحكم في الوسائط، ونصوص الدردشة، والتخصيص باستخدام Tailwind CSS.
