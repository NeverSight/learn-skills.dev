Crea frontend React per gli agenti di intelligenza artificiale vocale LiveKit. Utilizza questa competenza durante la creazione di interfacce Web per gli assistenti vocali utilizzando i componenti dell'interfaccia utente degli agenti basati su shadcn di LiveKit. Copre AgentSessionProvider, visualizzatori audio, controlli multimediali, trascrizioni di chat e personalizzazione con Tailwind CSS.
