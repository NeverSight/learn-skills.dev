Créez des interfaces React pour les agents d'IA vocale LiveKit. Utilisez cette compétence lors de la création d'interfaces Web pour les assistants vocaux à l'aide des composants d'interface utilisateur des agents basés sur shadcn de LiveKit. Couvre AgentSessionProvider, les visualiseurs audio, les contrôles multimédias, les transcriptions de discussion et la personnalisation avec Tailwind CSS.
