Создавайте интерфейсы React для агентов голосового искусственного интеллекта LiveKit. Используйте этот навык при создании веб-интерфейсов для голосовых помощников с использованием компонентов пользовательского интерфейса агентов LiveKit на основе Shadcn. Охватывает AgentSessionProvider, аудиовизуализаторы, элементы управления мультимедиа, расшифровки чатов и настройку с помощью Tailwind CSS.
