Создайте серверную часть агента LiveKit на Python. Используйте этот навык при создании голосовых агентов искусственного интеллекта, голосовых помощников или любого приложения искусственного интеллекта реального времени с помощью SDK Python Agents SDK LiveKit (livekit-agents). Охватывает AgentSession, класс агента, функциональные инструменты, модели STT/LLM/TTS, обнаружение поворотов и рабочие процессы с несколькими агентами.
