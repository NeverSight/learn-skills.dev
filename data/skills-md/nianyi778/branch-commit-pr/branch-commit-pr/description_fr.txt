Flux de travail complet de développement git : détectez la branche par défaut, créez des branches de fonctionnalités avec une dénomination intelligente, effectuez des validations atomiques en suivant les conventions de dépôt et ouvrez des demandes d'extraction avec des problèmes GitHub ou des liens Jira. À utiliser lorsqu'on vous demande de « commencer à travailler sur », « créer une branche », « valider », « ouvrir un PR », « soumettre des modifications » ou toute tâche complète du cycle de développement.
