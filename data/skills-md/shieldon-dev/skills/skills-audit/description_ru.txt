Сканер безопасности для навыков ИИ-агента. Перед установкой проверяет файлы навыков на предмет кражи учетных данных, утечки данных, опасных команд и запутывания. Используйте перед установкой любого нового навыка, чтобы получить структурированный отчет о рисках с оценкой (0–100), уровнем серьезности и практическими выводами. Запускается локально без внешних зависимостей — по сети ничего не отправляется.
