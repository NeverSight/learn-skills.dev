Automatisieren Sie Headless-Browser-Interaktionen mithilfe der Agent-Browser-CLI. Verwenden Sie es zum Scrapen von Webseiten, zum Automatisieren von Formularübermittlungen, zum Testen von Web-Apps oder zum Durchführen von Browser-Automatisierungsaufgaben. Funktioniert mit Elementreferenzen (@e1, @e2), die für die Argumentation von KI-Agenten optimiert sind.
