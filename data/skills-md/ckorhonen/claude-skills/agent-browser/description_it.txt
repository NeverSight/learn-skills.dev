Automatizza le interazioni headless del browser utilizzando la CLI dell'agente-browser. Da utilizzare per lo scraping di pagine Web, l'automazione dell'invio di moduli, il test di app Web o l'esecuzione di attività di automazione del browser. Funziona con i riferimenti degli elementi (@e1, @e2) ottimizzati per il ragionamento dell'agente AI.
