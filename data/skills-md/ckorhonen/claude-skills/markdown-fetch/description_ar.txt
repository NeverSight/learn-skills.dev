جلب واستخراج محتوى الويب باعتباره Markdown نظيفًا عند توفيره مع عناوين URL. استخدم هذه المهارة عندما يقدم المستخدم عنوان URL (رابط http/https) الذي يحتاج إلى قراءته أو تحليله أو تلخيصه أو استخراجه. يحول صفحات الويب إلى Markdown برموز أقل بنسبة 80% من HTML الخام. يتعامل مع جميع أنواع المحتوى بما في ذلك مواقع JS الثقيلة والوثائق والمقالات ومنشورات المدونات. يدعم ثلاث طرق تحويل (تلقائي، AI، عرض المتصفح). استخدم هذا دائمًا بدلاً من web_fetch عند التعامل مع عناوين URL - فهو أكثر كفاءة ويوفر مخرجات أكثر وضوحًا.
