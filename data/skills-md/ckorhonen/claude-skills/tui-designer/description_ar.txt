تصميم وتنفيذ واجهات مستخدم طرفية على طراز الرجعية/السايبربانك/القراصنة. يغطي أساليب React (Tuimorphic)، وSwiftUI (التظليل المعدني)، وCSS. يُستخدم عند إنشاء عناصر جمالية طرفية، أو تأثيرات CRT، أو توهج النيون، أو خطوط المسح الضوئي، أو شاشات العرض باللون الأخضر الفوسفوري، أو الواجهات المستقبلية القديمة.
