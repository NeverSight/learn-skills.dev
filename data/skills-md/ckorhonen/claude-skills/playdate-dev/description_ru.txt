Разработка игр Playdate на Lua с помощью Playdate SDK, включая игровой цикл, спрайты, графику, ввод (рукоятка, кнопки, акселерометр), пользовательский интерфейс, производительность, метаданные (pdxinfo) и рабочий процесс симулятора/устройства. Используйте, когда вас просят создать игру для Playdate, реализовать механику, специфичную для Playdate, или применить рекомендации по дизайну и доступности Playdate.
