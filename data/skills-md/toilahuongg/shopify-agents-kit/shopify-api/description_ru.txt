Подробное руководство по API Shopify в приложениях Remix. Охватывает Admin GraphQL/REST, API витрины, все ресурсы (продукты, заказы, клиенты, инвентарь, коллекции, скидки, выполнение, метаполя, файлы), массовые операции, веб-перехватчики, средства выбора ресурсов и шаблоны TypeScript. Используйте при запросе/изменении данных Shopify или построении интеграции.
