دليل شامل لـ Shopify APIs في تطبيقات Remix. يغطي Admin GraphQL/REST وواجهة برمجة تطبيقات Storefront وجميع الموارد (المنتجات والطلبات والعملاء والمخزون والمجموعات والخصومات والوفاءات وحقول التعريف والملفات) والعمليات المجمعة وخطافات الويب ومنتقي الموارد وأنماط TypeScript. يُستخدم عند الاستعلام/تعديل بيانات Shopify أو إنشاء عمليات تكامل.
