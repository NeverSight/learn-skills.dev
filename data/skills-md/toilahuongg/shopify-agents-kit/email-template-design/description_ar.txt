قم بتصميم وإنشاء قوالب بريد إلكتروني احترافية بتنسيق HTML باستخدام CSS مضمّنة لتحقيق توافق واسع النطاق مع عميل البريد الإلكتروني. استخدم هذه المهارة عندما يطلب المستخدم إنشاء أو تصميم أو إنشاء قوالب بريد إلكتروني أو رسائل إخبارية أو رسائل بريد إلكتروني للمعاملات (تأكيدات الطلب أو الإيصالات أو إشعارات الشحن أو إعادة تعيين كلمة المرور) أو رسائل البريد الإلكتروني التسويقية أو سلسلة الترحيب أو رسائل البريد الإلكتروني الخاصة بالإعداد أو رسائل البريد الإلكتروني لعربة التسوق المهجورة أو حملات التنقيط أو أي تخطيط بريد إلكتروني بتنسيق HTML. يغطي التصميم سريع الاستجابة، ودعم الوضع المظلم، والتوافق مع Gmail، وOutlook (سطح المكتب + الويب)، وApple Mail، وYahoo، وعملاء الأجهزة المحمولة.
