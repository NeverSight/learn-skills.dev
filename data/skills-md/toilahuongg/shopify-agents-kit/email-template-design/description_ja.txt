インライン CSS を使用してプロフェッショナルな HTML 電子メール テンプレートを設計および構築し、幅広い電子メール クライアントとの互換性を実現します。このスキルは、ユーザーがメール テンプレート、ニュースレター、トランザクション メール (注文確認、受領書、出荷通知、パスワード リセット)、マーケティング メール、ウェルカム シリーズ、オンボーディング メール、カート放棄メール、ドリップ キャンペーン、または HTML メール レイアウトの作成、デザイン、構築を要求する場合に使用します。レスポンシブ デザイン、ダーク モードのサポート、Gmail、Outlook (デスクトップ + Web)、Apple Mail、Yahoo、およびモバイル クライアントとの互換性について説明します。
