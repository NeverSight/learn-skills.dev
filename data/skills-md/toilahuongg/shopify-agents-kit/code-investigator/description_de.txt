Umfassendes Code-Untersuchungs- und Audit-Tool. Erkennt alle Projektfunktionen und entsendet dann parallele Subagenten, um Probleme, Risiken, toten Code, fehlende Funktionalität und Redundanzen zu analysieren. Erstellt einen priorisierten Risikobericht. Verwenden Sie diese Fähigkeit, wenn der Benutzer „Code untersuchen“, „Projekt prüfen“, „Risiken finden“, „Codequalität prüfen“, „Codebasis analysieren“, „Was stimmt mit diesem Code nicht“, „Projektzustandsprüfung“, „Codeüberprüfung des gesamten Projekts“, „toten Code finden“, „redundanten Code finden“ oder eine gründliche Codebasisanalyse anfordert.
