Комплексный инструмент для исследования и аудита кода. Обнаруживает все функции проекта, а затем отправляет параллельные субагенты для анализа проблем, рисков, мертвого кода, недостающих функций и дублирования. Создает отчет о приоритетных рисках. Используйте этот навык, когда пользователь просит «исследовать код», «аудит проекта», «найти риски», «проверить качество кода», «проанализировать кодовую базу», «что не так с этим кодом», «проверить работоспособность проекта», «проверить код всего проекта», «найти мертвый код», «найти избыточный код» или любой запрос на тщательный анализ кодовой базы.
