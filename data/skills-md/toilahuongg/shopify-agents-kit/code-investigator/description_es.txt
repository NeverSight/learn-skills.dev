Herramienta integral de auditoría e investigación de código. Descubre todas las características del proyecto y luego envía subagentes paralelos para analizar problemas, riesgos, código inactivo, funcionalidad faltante y redundancias. Elabora un informe de riesgos priorizados. Utilice esta habilidad cuando el usuario solicite "investigar código", "auditar proyecto", "encontrar riesgos", "verificar la calidad del código", "analizar la base del código", "qué hay de malo en este código", "verificar el estado del proyecto", "revisar el código de todo el proyecto", "encontrar código inactivo", "encontrar código redundante" o cualquier solicitud para un análisis exhaustivo de la base del código.
