Outil complet d’investigation et d’audit du code. Détecte toutes les fonctionnalités du projet, puis envoie des sous-agents parallèles pour analyser les problèmes, les risques, le code mort, les fonctionnalités manquantes et les redondances. Produit un rapport sur les risques hiérarchisés. Utilisez cette compétence lorsque l'utilisateur demande « enquêter sur le code », « auditer le projet », « trouver des risques », « vérifier la qualité du code », « analyser la base de code », « quel est le problème avec ce code », « vérifier l'état du projet », « réviser le code dans son ensemble », « trouver du code mort », « trouver du code redondant » ou toute demande d'analyse approfondie de la base de code.
