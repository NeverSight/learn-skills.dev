Guía para crear extensiones de interfaz de usuario de Shopify POS (punto de venta) que integran funcionalidad personalizada en la interfaz minorista de Shopify. Utilice esta habilidad cuando el usuario necesite crear mosaicos, modales, bloques o acciones de POS para la red inteligente, el carrito, los detalles del cliente, los detalles del pedido, los detalles del producto o las pantallas posteriores a la compra. Cubre objetivos, componentes, API y flujo de trabajo de desarrollo para extensiones de aplicaciones POS.
