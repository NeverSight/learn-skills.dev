Руководство по созданию расширений пользовательского интерфейса Shopify POS (точка продаж), которые интегрируют пользовательские функции в розничный интерфейс Shopify. Используйте этот навык, когда пользователю необходимо создать плитки POS, модальные окна, блоки или действия для интеллектуальной сетки, корзины, сведений о клиенте, сведений о заказе, сведений о продукте или экранов после покупки. Охватывает цели, компоненты, API и рабочий процесс разработки расширений POS-приложений.
