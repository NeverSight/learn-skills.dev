Fachkundige Anleitung zum Erstellen von KI-Agenten mit dem Claude Agent SDK in Python. Wird verwendet, wenn der Benutzer Python-Code mit dem SDK schreibt, claude_agent_sdk importiert, query(), ClaudeSDKClient, ClaudeAgentOptions, AgentDefinition verwendet, benutzerdefinierte @tool-Funktionen erstellt, Hooks einrichtet, MCP-Server integriert, Sitzungen verwaltet oder SDK-Probleme in Python debuggt. Wird auch ausgelöst bei: „claude_agent_sdk“, „ClaudeAgentOptions“, „from claude_agent_sdk import“, „pip install claude-agent-sdk“, „asyncio.run“ oder „async for message in query“.
