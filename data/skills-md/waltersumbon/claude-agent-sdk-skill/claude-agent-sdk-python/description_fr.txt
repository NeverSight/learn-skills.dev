Conseils d'experts pour créer des agents IA avec le SDK Claude Agent en Python. À utiliser lorsque l'utilisateur écrit du code Python avec le SDK, importe claude_agent_sdk, utilise query(), ClaudeSDKClient, ClaudeAgentOptions, AgentDefinition, crée des fonctions @tool personnalisées, configure des hooks, intègre des serveurs MCP, gère des sessions ou débogue les problèmes du SDK en Python. Se déclenche également sur : "claude_agent_sdk", "ClaudeAgentOptions", "from claude_agent_sdk import", "pip install claude-agent-sdk", "asyncio.run" ou "async for message in query".
