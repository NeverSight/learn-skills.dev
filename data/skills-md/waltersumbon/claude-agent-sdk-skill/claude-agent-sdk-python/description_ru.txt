Экспертное руководство по созданию агентов ИИ с помощью Claude Agent SDK на Python. Используйте, когда пользователь пишет код Python с помощью SDK, импортирует claude_agent_sdk, использует query(), ClaudeSDKClient, ClaudeAgentOptions, AgentDefinition, создает пользовательские функции @tool, настраивает перехватчики, интегрирует серверы MCP, управляет сеансами или отлаживает проблемы SDK в Python. Также срабатывает при: «claude_agent_sdk», «ClaudeAgentOptions», «из импорта claude_agent_sdk», «pip install claude-agent-sdk», «asyncio.run» или «async для сообщения в запросе».
