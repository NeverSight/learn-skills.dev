Python에서 Claude Agent SDK를 사용하여 AI 에이전트를 구축하기 위한 전문가 지침입니다. 사용자가 SDK로 Python 코드를 작성하고, claude_agent_sdk를 가져오고, query(), ClaudeSDKClient, ClaudeAgentOptions, AgentDefinition을 사용하고, 사용자 정의 @tool 함수를 생성하고, 후크를 설정하고, MCP 서버를 통합하고, 세션을 관리하거나 Python에서 SDK 문제를 디버그할 때 사용합니다. 또한 "claude_agent_sdk", "ClaudeAgentOptions", "from claude_agent_sdk 가져오기", "pip install claude-agent-sdk", "asyncio.run" 또는 "쿼리의 메시지에 대한 비동기"에서도 트리거됩니다.
