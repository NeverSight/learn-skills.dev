Python の Claude Agent SDK を使用して AI エージェントを構築するための専門家によるガイダンス。ユーザーが SDK を使用して Python コードを作成する場合、claude_agent_sdk をインポートする場合、query()、ClaudeSDKClient、ClaudeAgentOptions、AgentDefinition を使用する場合、カスタム @tool 関数を作成する場合、フックをセットアップする場合、MCP サーバーを統合する場合、セッションを管理する場合、または Python で SDK の問題をデバッグする場合に使用します。また、「claude_agent_sdk」、「ClaudeAgentOptions」、「from claude_agent_sdk import」、「pip install claude-agent-sdk」、「asyncio.run」、または「async for message in query」でもトリガーされます。
