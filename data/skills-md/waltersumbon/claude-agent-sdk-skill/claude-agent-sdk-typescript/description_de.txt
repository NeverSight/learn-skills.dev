Fachkundige Anleitung zum Erstellen von KI-Agenten mit dem Claude Agent SDK in TypeScript/JavaScript. Wird verwendet, wenn der Benutzer TypeScript- oder JavaScript-Code mit dem SDK schreibt, aus „@anthropic-ai/claude-agent-sdk“ importiert, query(),allowedTools,permissionMode verwendet, benutzerdefinierte Tools mit createSdkMcpServer erstellt, Hooks einrichtet, MCP-Server integriert, Sitzungen verwaltet oder SDK-Probleme in TypeScript/Node.js debuggt. Wird auch ausgelöst bei: „@anthropic-ai/claude-agent-sdk“, „allowedTools“, „permissionMode“, „npm install“, „for waiting“, „AgentDefinition“ oder „settingSources“.
