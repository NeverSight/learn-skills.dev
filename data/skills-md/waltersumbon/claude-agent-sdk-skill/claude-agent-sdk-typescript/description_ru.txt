Экспертное руководство по созданию агентов ИИ с помощью Claude Agent SDK на TypeScript/JavaScript. Используйте, когда пользователь пишет код TypeScript или JavaScript с помощью SDK, импортирует из «@anthropic-ai/claude-agent-sdk», использует query(), AllowTools, PermissionMode, создает собственные инструменты с помощью createSdkMcpServer, настраивает перехватчики, интегрирует серверы MCP, управляет сеансами или отлаживает проблемы SDK в TypeScript/Node.js. Также срабатывает при: «@anthropic-ai/claude-agent-sdk», «allowedTools», «permissionMode», «npm install», «for await», «AgentDefinition» или «settingSources».
