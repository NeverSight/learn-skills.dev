Conseils d'experts pour créer des agents IA avec le SDK Claude Agent en TypeScript/JavaScript. À utiliser lorsque l'utilisateur écrit du code TypeScript ou JavaScript avec le SDK, importe depuis « @anthropic-ai/claude-agent-sdk », utilise query(), AllowedTools, permissionMode, crée des outils personnalisés avec createSdkMcpServer, configure des hooks, intègre des serveurs MCP, gère des sessions ou débogue les problèmes du SDK dans TypeScript/Node.js. Se déclenche également sur : "@anthropic-ai/claude-agent-sdk", "allowedTools", "permissionMode", "npm install", "for wait", "AgentDefinition" ou "settingSources".
