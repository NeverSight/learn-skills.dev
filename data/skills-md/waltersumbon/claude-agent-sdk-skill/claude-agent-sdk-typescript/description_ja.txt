TypeScript/JavaScript で Claude Agent SDK を使用して AI エージェントを構築するための専門家によるガイダンス。ユーザーが SDK を使用して TypeScript または JavaScript コードを作成する場合、「@anthropic-ai/claude-agent-sdk」からインポートする場合、query()、allowedTools、permissionMode を使用する場合、createSdkMcpServer を使用してカスタム ツールを作成する場合、フックを設定する場合、MCP サーバーを統合する場合、セッションを管理する場合、TypeScript/Node.js で SDK の問題をデバッグする場合に使用します。また、「@anthropic-ai/claude-agent-sdk」、「allowedTools」、「permissionMode」、「npm install」、「for await」、「AgentDefinition」、または「settingSources」でもトリガーされます。
