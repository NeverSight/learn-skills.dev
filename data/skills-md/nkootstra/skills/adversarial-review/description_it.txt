Configura pipeline di verifica contraddittoria multi-agente (ricerca bug → avversario → arbitro) per la revisione del codice ad alta fedeltà e il rilevamento dei bug. Si adatta sia alla CLI di Claude Code (sottoagenti reali) che a Claude.ai (simulazione sequenziale). Da utilizzare quando l'utente richiede una revisione contraddittoria, una verifica multi-agente, una ricerca di bug ad alta affidabilità, una pipeline di revisione degli agenti o una revisione automatizzata approfondita oltre un singolo passaggio.
