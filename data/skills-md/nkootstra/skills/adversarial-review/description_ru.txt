Настройте многоагентные конвейеры состязательной проверки (поиск ошибок → противник → проверяющий) для высокоточной проверки кода и обнаружения ошибок. Адаптируется как к Claude Code CLI (реальные субагенты), так и к Claude.ai (последовательное моделирование). Используйте, когда пользователь запрашивает состязательную проверку, многоагентную проверку, надежный поиск ошибок, конвейер проверки агента или тщательную автоматическую проверку за один проход.
