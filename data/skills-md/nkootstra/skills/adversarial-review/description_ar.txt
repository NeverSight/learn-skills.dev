قم بإعداد مسارات التحقق من الخصومة متعددة الوكلاء (مكتشف الأخطاء → الخصم → الحكم) لمراجعة التعليمات البرمجية عالية الدقة واكتشاف الأخطاء. يتكيف مع كل من Claude Code CLI (الوكلاء الفرعيين الحقيقيين) وClaude.ai (المحاكاة التسلسلية). يُستخدم عندما يطلب المستخدم إجراء مراجعة تعارضية، أو التحقق متعدد الوكلاء، أو اكتشاف الأخطاء بثقة عالية، أو مسار مراجعة الوكيل، أو مراجعة تلقائية شاملة تتجاوز تمريرة واحدة.
