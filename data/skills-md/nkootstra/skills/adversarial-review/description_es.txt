Configure canales de verificación adversarios de múltiples agentes (buscador de errores → adversario → árbitro) para revisión de código de alta fidelidad y detección de errores. Se adapta tanto a Claude Code CLI (subagentes reales) como a Claude.ai (simulación secuencial). Úselo cuando el usuario solicite una revisión adversaria, una verificación de múltiples agentes, una búsqueda de errores de alta confianza, un proceso de revisión de agentes o una revisión automatizada exhaustiva más allá de una sola pasada.
