قم بتوجيه ترحيل موقع الويب دون فقدان التصنيفات - نقل النطاق، أو تبديل CMS، أو إعادة هيكلة عنوان URL، أو HTTP إلى HTTPS، أو إعادة التصميم. يُستخدم عندما يسأل المستخدم عن ترحيل الموقع، أو تغيير النطاق، أو ترحيل نظام إدارة المحتوى (CMS)، أو إعادة هيكلة عنوان URL، أو إعادة تصميم تأثير تحسين محركات البحث (SEO)، أو إعادة توجيه التعيين، أو كيفية نقل موقع بأمان.
