À utiliser lorsque vous travaillez avec les services de trousseau iOS/macOS (requêtes SecItem, kSecClass, erreurs OSStatus), l'authentification biométrique (LAContext, Face ID, Touch ID), CryptoKit (AES-GCM, ChaChaPoly, ECDSA, ECDH, HPKE, ML-KEM), Secure Enclave, stockage sécurisé des informations d'identification (jetons OAuth, clés API), épinglage de certificat (SecTrust, SPKI), partage de trousseau entre applications/extensions, migration de secrets depuis UserDefaults ou plists, ou conformité mobile OWASP MASVS/MASTG sur les plateformes Apple.
