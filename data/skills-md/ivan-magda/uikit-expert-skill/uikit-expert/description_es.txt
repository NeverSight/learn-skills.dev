Escriba, revise o mejore el código UIKit siguiendo las mejores prácticas para el ciclo de vida del controlador de vistas, diseño automático, vistas de colección, navegación, animación, administración de memoria y API modernas de iOS 18–26. Úselo al crear nuevas funciones de UIKit, refactorizar vistas o controladores de vistas existentes, revisar la calidad del código, adoptar patrones UIKit modernos (fuentes de datos diferenciables, diseño compositivo, configuración de celdas) o unir UIKit con SwiftUI. No cubre el código exclusivo de SwiftUI.
