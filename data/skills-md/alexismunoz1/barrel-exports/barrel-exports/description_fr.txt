Analyse et migre les modèles d'exportation de barils dans les projets TypeScript/React pour réduire la taille du bundle. Détecte les fichiers barils (réexportations index.ts), vérifie les effets secondaires, mesure l'impact du bundle et réécrit en toute sécurité les importations vers des chemins directs. Il a été prouvé qu'il réduisait les fragments de page jusqu'à 19 fois dans les applications Next.js. À utiliser lorsque l'utilisateur mentionne « analyser les barils », « vérifier les exportations de barils », « migrer les barils », « corriger les exportations de barils », « taille du paquet trop grande », « le tremblement d'arbre ne fonctionne pas », « réexportations index.ts » ou « optimiser les importations ». Ne PAS utiliser pour des questions générales d’importation/exportation ou de migration ESM/CJS.
