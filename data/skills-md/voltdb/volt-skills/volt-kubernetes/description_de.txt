Hilft bei der Bereitstellung des VoltDB-Clusters auf Kubernetes. Verwenden Sie diese Option, wenn der Benutzer Skripts zum Erstellen von Clustern auf Kubernetes mit den Volt-Versionen erstellen möchte. Verwenden Sie diese Option, wenn der Benutzer Terraform, Helmfile oder Helmskripte erstellen möchte. Verwenden Sie diese Option, wenn der Benutzer eine Konfiguration als Code erstellen möchte.
