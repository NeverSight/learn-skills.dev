Configuración del servidor web Caddy para proxy inverso y HTTPS automático. Úselo cuando:
(1) Escribir configuraciones de Caddyfile para configuraciones de proxy inverso
(2) Configurar HTTPS automático con Let's Encrypt
(3) Configuración de certificados comodín con desafío DNS
(4) Integración de Caddy con contenedores Docker (misma red o servicios de host)
(5) Configuración de coincidencias de solicitudes, bloques de identificadores y patrones de enrutamiento
(6) Configuración de equilibrio de carga y controles de estado
(7) Solución de problemas de TLS, conectividad ascendente o problemas de configuración
(8) Configurar la manipulación del encabezado (header_up, header_down)
