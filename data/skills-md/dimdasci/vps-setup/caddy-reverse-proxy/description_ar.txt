تكوين خادم الويب Caddy للوكيل العكسي وHTTPS التلقائي. استخدم عندما:
(1) كتابة تكوينات Caddyfile لإعدادات الوكيل العكسي
(2) تكوين HTTPS التلقائي باستخدام Let's Encrypt
(3) إعداد شهادات أحرف البدل مع تحدي DNS
(4) تكامل Caddy مع حاويات Docker (نفس الشبكة أو خدمات المضيف)
(5) تكوين مُطابقات الطلب وكتل المقبض وأنماط التوجيه
(6) إعداد موازنة التحميل والفحوصات الصحية
(7) استكشاف أخطاء TLS وإصلاحها أو الاتصال الأولي أو مشكلات التكوين
(8) تكوين معالجة الرأس (header_up، header_down)
