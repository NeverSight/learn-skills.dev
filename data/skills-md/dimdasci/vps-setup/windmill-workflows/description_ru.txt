Развертывание механизма рабочего процесса Windmill и создание сценариев. Используйте, когда:
(1) Настройка автономного хостинга Windmill с помощью Docker Compose
(2) Создание пользовательских рабочих изображений с помощью инструментов обработки PDF/изображений.
(3) Написание сценариев для извлечения, манипулирования или преобразования текста PDF.
(4) Написание скриптов для обработки изображений с помощью libvips.
(5) Настройка рабочих процессов, масштабирование и ограничения ресурсов.
(6) Устранение неполадок при выполнении заданий или проблем с работниками
(7) Интеграция Windmill с обратным прокси-сервером Caddy
