Zitadel ID プロバイダーのセットアップ、構成、および OIDC 統合。次の場合に使用します。
(1) Zitadel を OIDC/OAuth2 ID プロバイダーとして設定する
(2) Caddy リバースプロキシを使用した Zitadel の構成
(3) サービス用のOAuth2/OIDCアプリケーションの作成
(4) ユーザー、組織、サービスアカウントの管理
(5) Zitadel SSO とアプリケーションの統合 (Nextcloud、Windmill など)
(6) 認証、トークン、または接続の問題のトラブルシューティング
(7) 自動化のための Zitadel API の使用
