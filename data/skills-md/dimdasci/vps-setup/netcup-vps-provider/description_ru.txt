Управление Netcup VPS и DNS API. Используйте, когда:
(1) Управление записями DNS через Netcup DNS API
(2) Настройка VPS через панель управления сервером (SCP)
(3) Настройка правил брандмауэра и обратного DNS (PTR)
(4) Получение учетных данных Netcup API и управление ими.
(5) Устранение неполадок распространения DNS
(6) Управление доменами, зарегистрированными в Netcup
(7) Настройка внешних доменов для использования Netcup DNS
