Gestión de Netcup VPS y DNS API. Úselo cuando:
(1) Administrar registros DNS a través de la API DNS de Netcup
(2) Configuración de VPS a través del Panel de control del servidor (SCP)
(3) Configuración de reglas de firewall y DNS inverso (PTR)
(4) Obtener y administrar las credenciales de la API de Netcup
(5) Solución de problemas de propagación de DNS
(6) Administrar dominios registrados en Netcup
(7) Configurar dominios externos para usar Netcup DNS
