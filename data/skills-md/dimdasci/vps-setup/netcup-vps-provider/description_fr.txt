Gestion des API Netcup VPS et DNS. À utiliser lorsque :
(1) Gestion des enregistrements DNS via l'API DNS Netcup
(2) Configuration du VPS via le panneau de configuration du serveur (SCP)
(3) Configuration des règles de pare-feu et du DNS inversé (PTR)
(4) Obtention et gestion des informations d'identification de l'API Netcup
(5) Dépannage des problèmes de propagation DNS
(6) Gestion des domaines enregistrés sur Netcup
(7) Configuration de domaines externes pour utiliser Netcup DNS
