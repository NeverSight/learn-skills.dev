Gestione Netcup VPS e API DNS. Utilizzare quando:
(1) Gestione dei record DNS tramite Netcup DNS API
(2) Configurazione VPS tramite il Pannello di controllo del server (SCP)
(3) Impostazione delle regole del firewall e DNS inverso (PTR)
(4) Ottenere e gestire le credenziali API Netcup
(5) Risoluzione dei problemi di propagazione DNS
(6) Gestione dei domini registrati presso Netcup
(7) Configurazione di domini esterni per utilizzare Netcup DNS
