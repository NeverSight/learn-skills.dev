Instalación, configuración y administración del servidor de correo electrónico Mox. Úselo cuando:
(1) Configurar el servidor de correo Mox en un VPS
(2) Configuración de registros DNS para correo electrónico (MX, SPF, DKIM, DMARC, MTA-STS)
(3) Integración de Mox con proxy inverso Caddy y Docker
(4) Administrar cuentas de correo electrónico, dominios y alias
(5) Solución de problemas de entrega de correo electrónico, TLS o problemas de conectividad
(6) Sincronización de certificados TLS de Caddy a Mox
