Diese Fähigkeit sollte verwendet werden, wenn ein Benutzer automatisierte VORHER/NACHHER-Screenshots für eine React Native-Web-App benötigt und Bilder an einen Feature-Branch übergeben und aktualisieren oder eine GitHub-Pull-Anfrage oder GitLab-Merge-Anfrage erstellen möchte, indem er den Anbieter automatisch vom Ursprung her erkennt.
