Проверка кода безопасности на экспертном уровне, которая находит реальные уязвимости на любом языке или платформе. Выявляет топ-10 OWASP, проблемы, выявленные CWE, недостатки внедрения, обходы аутентификации, раскрытие секретов, небезопасные настройки по умолчанию и риски цепочки поставок, а затем предоставляет конкретные исправления методом копирования. Используйте этот навык всякий раз, когда пользователь просит проверить код на предмет безопасности, проверить кодовую базу, проверить наличие уязвимостей, усилить защиту приложения, выполнить оценку безопасности, найти ошибки безопасности или задать что-либо, связанное с безопасностью приложений, AppSec, моделированием угроз или методами безопасного кодирования. Также срабатывает при проверке запросов на включение, если упоминается безопасность, или когда пользователь говорит такие вещи, как «это безопасно?», «можно ли это использовать?» или «пентестировать этот код».
