Revisión de código de seguridad a nivel de expertos que encuentra vulnerabilidades reales en cualquier lenguaje o marco. Identifica los 10 principales problemas de OWASP, problemas mapeados por CWE, fallas de inyección, omisiones de autenticación, exposición de secretos, valores predeterminados inseguros y riesgos de la cadena de suministro; luego proporciona correcciones específicas de copiar y pegar. Utilice esta habilidad siempre que el usuario solicite revisar el código por seguridad, auditar una base de código, verificar vulnerabilidades, reforzar una aplicación, realizar una evaluación de seguridad, encontrar errores de seguridad o preguntar cualquier cosa relacionada con la seguridad de la aplicación, AppSec, modelado de amenazas o prácticas de codificación segura. También se activa al revisar las solicitudes de extracción si se menciona la seguridad o cuando el usuario dice cosas como "¿es esto seguro?", "¿se puede explotar esto?" o "probar este código".
