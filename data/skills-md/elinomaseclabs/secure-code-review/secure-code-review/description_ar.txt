مراجعة رمز الأمان على مستوى الخبراء والتي تكتشف نقاط الضعف الحقيقية عبر أي لغة أو إطار عمل. يحدد أهم 10 مشاكل في OWASP، ومشكلات CWE، وعيوب الحقن، وتجاوزات المصادقة، والكشف عن الأسرار، والافتراضيات غير الآمنة، ومخاطر سلسلة التوريد - ثم يوفر إصلاحات محددة للنسخ واللصق. استخدم هذه المهارة عندما يطلب المستخدم مراجعة التعليمات البرمجية للأمان، أو تدقيق قاعدة التعليمات البرمجية، أو التحقق من الثغرات الأمنية، أو تقوية التطبيق، أو إجراء تقييم أمني، أو العثور على أخطاء أمنية، أو يسأل أي شيء يتعلق بأمان التطبيق، أو AppSec، أو نمذجة التهديدات، أو ممارسات الترميز الآمن. يتم تشغيله أيضًا عند مراجعة طلبات السحب إذا تم ذكر الأمان، أو عندما يقول المستخدم أشياء مثل "هل هذا آمن؟"، أو "هل يمكن استغلال هذا؟"، أو "اخترق هذا الرمز".
