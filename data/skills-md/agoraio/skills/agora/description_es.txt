Escriba código utilizando Agora SDK (agora.io) para comunicación en tiempo real. Cubre RTC (llamadas de video/voz, transmisión en vivo, uso compartido de pantalla), RTM (señalización, mensajería, presencia), IA conversacional (agentes de IA de voz), grabación en la nube, puerta de enlace del servidor y generación de tokens del lado del servidor. Úselo cuando el usuario desee crear aplicaciones de audio/video en tiempo real, integrar SDK de Agora (Web JS/TS, React, iOS Swift, Android Kotlin/Java, Go, Python), administrar canales, pistas, tokens, usar RTM para mensajería/señalización, grabar sesiones de RTC o crear IA conversacional con el kit de herramientas del agente. Se activan menciones de Agora, agora.io, RTC, RTM, videollamadas, llamadas de voz, comunicación en tiempo real, pantalla compartida, pantalla compartida, grabación de sesión, grabación de llamadas, grabación en la nube, Server Gateway, SDK de medios de Linux, agora-rtc-sdk-ng, agora-rtc-react, agora-rtm, IA conversacional con Agora, generación de tokens de Agora, autenticación de Agora, kit de herramientas de cliente-agente-agora, agora-agent-client-toolkit-react, agora-agent-server-sdk, AgoraVoiceAI, AgoraClient, useConversationalAI, useTranscript, useAgentState, transcripción del agente, enlace de estado del agente.
