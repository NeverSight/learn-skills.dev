Единый рабочий процесс документации для написания, проверки и обновления технической документации в разных репозиториях. Используйте, когда пользователи просят написать или отредактировать документы, проверить качество документов, сопоставить изменения кода с влиянием документа, подготовить новую документацию, обновить руководства README/API или улучшить документы в файлах `/docs` и `*.md`/`*.mdx`. Поддерживает автономное поведение на основе профилей для общих документов, документов Next.js и документов Angular.
