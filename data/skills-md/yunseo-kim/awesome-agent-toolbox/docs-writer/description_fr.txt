Flux de travail de documentation unifié pour la rédaction, la révision et la mise à jour de documents techniques dans tous les référentiels. À utiliser lorsque les utilisateurs demandent à écrire ou à modifier des documents, à vérifier la qualité d'un document, à mapper les modifications du code sur l'impact du document, à créer une nouvelle documentation, à mettre à jour les guides README/API ou à améliorer les documents dans les fichiers `/docs` et `*.md`/`*.mdx`. Prend en charge le comportement autonome basé sur le profil pour les documents génériques, les documents Next.js et les documents Angular.
