ДОЛЖЕН ИСПОЛЬЗОВАТЬСЯ для ЛЮБЫХ операций git. Атомарные фиксации, перебазирование/сжатие, выбор вишни, рабочее дерево, восстановление повторного журнала и поиск по истории (обвинение, разделение пополам, журнал -S). Триггеры: «коммит», «перебазирование», «сквош», «выбор вишни», «рабочее дерево», «рефлог», «кто написал», «когда был добавлен X», «найти этот коммит».
