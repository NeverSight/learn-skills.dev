DOIT être utilisé chaque fois que l'utilisateur souhaite ajouter, installer, rechercher ou rechercher une compétence, un serveur MCP ou un plugin. À utiliser également lors de la configuration de la synchronisation des agents multi-outils entre Claude Code, OpenCode, Codex et Cursor. Cette compétence garantit que tous les outils restent identiques en les installant d'abord sur Claude Code et en les synchronisant via vsync. Utilisez de manière PROACTIVE lorsque l'utilisateur dit des choses comme « ajouter un serveur MCP », « trouver une compétence pour X », « installer un plugin », « ajouter un outil », « rechercher des compétences », « configurer MCP », « J'ai besoin d'une compétence qui fait X », « ajouter un MCP svelte » ou toute demande d'ajout de fonctionnalités à l'agent. Se déclenche également sur : vsync, parité entre outils, synchronisation des agents, évaluation des outils d'IA, /setup-agent-sync, /find-skill, /add-skill-submodule, /find-mcp, /update-agent-tools.
