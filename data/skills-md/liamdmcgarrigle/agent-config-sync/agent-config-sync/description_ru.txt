ДОЛЖЕН использоваться всякий раз, когда пользователь хочет добавить, установить, найти или выполнить поиск навыка, сервера MCP или плагина. Также используйте при настройке синхронизации агентов между инструментами между Claude Code, OpenCode, Codex и Cursor. Этот навык гарантирует, что все инструменты останутся идентичными, поскольку сначала они устанавливаются на Claude Code и синхронизируются через vsync. Используйте ПРОАКТИВНО, когда пользователь говорит такие вещи, как «добавить сервер MCP», «найти навык для X», «установить плагин», «добавить инструмент», «поиск навыков», «настроить MCP», «Мне нужен навык, который делает X», «добавить svelte MCP» или любой запрос на добавление возможностей агенту. Также срабатывает при: vsync, проверке четности между инструментами, синхронизации агентов, оценке инструментов AI, /setup-agent-sync, /find-skill, /add-skill-submodule, /find-mcp, /update-agent-tools.
