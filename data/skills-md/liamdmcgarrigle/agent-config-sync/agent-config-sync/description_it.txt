DEVE essere utilizzato ogni volta che l'utente desidera aggiungere, installare, trovare o cercare una competenza, un server MCP o un plug-in. Da utilizzare anche durante la configurazione della sincronizzazione dell'agente cross-tool tra Claude Code, OpenCode, Codex e Cursor. Questa abilità garantisce che tutti gli strumenti rimangano identici installandoli prima su Claude Code e sincronizzandoli tramite vsync. Utilizzare in modo PROATTIVO quando l'utente dice cose come "aggiungi un server MCP", "trova una competenza per X", "installa un plugin", "aggiungi uno strumento", "cerca competenze", "configura MCP", "ho bisogno di una competenza che faccia X", "aggiungi MCP snello" o qualsiasi richiesta per aggiungere funzionalità all'agente. Si attiva anche su: vsync, parità tra strumenti, sincronizzazione dell'agente, valutazione degli strumenti AI, /setup-agent-sync, /find-skill, /add-skill-submodule, /find-mcp, /update-agent-tools.
