يجب استخدامه عندما يريد المستخدم إضافة أو تثبيت أو العثور على أو البحث عن مهارة أو خادم MCP أو مكون إضافي. استخدمه أيضًا عند إعداد مزامنة الوكيل عبر الأدوات عبر Claude Code وOpenCode وCodex وCursor. تضمن هذه المهارة بقاء جميع الأدوات متطابقة عن طريق التثبيت على Claude Code أولاً والمزامنة عبر vsync. استخدم بشكل استباقي عندما يقول المستخدم أشياء مثل "أضف خادم MCP"، أو "ابحث عن مهارة لـ X"، أو "تثبيت مكون إضافي"، أو "أضف أداة"، أو "ابحث عن مهارات"، أو "إعداد MCP"، أو "أحتاج إلى مهارة تؤدي X"، أو "أضف MCP Svelte"، أو أي طلب لإضافة إمكانات إلى الوكيل. يتم تشغيله أيضًا على: vsync، والتكافؤ عبر الأدوات، ومزامنة الوكيل، وتقييم أدوات الذكاء الاصطناعي، و/setup-agent-sync، و/find-skill، و/add-skill-submodule، و/find-mcp، و/update-agent-tools.
