يتم تشغيله في نهاية جلسة البرمجة، أو قبل التسليم، أو في أي وقت يريد المستخدم تسجيل ما تم تعلمه. قم بمراجعة المحادثة وتغييرات git الأخيرة، وحدد التصحيحات والقرارات وسير العمل الجديد ومهام المتابعة وفرص الأتمتة والمشاكل غير الواضحة، ثم قم بتحديث مستندات المشروع الصحيحة وملفات تعليمات الوكيل (AGENTS.md، CLAUDE.md، تعليمات Copilot، قواعد المؤشر، أو ما يعادلها من المشروع) واستمر في الدروس الدائمة للجلسات المستقبلية.
استخدم هذه المهارة عندما يريد المستخدم إنهاء العمل، أو تسليمه، أو حفظ ما تم تعلمه، أو تحديث المستندات أو تعليمات الوكيل استنادًا إلى الجلسة، أو التقاط الدروس، أو تسجيل الأخطاء، أو الحفاظ على القرارات، أو ملاحظة الخطوات التالية، أو السؤال عما يجب الالتزام به، أو توثيق الأخطاء المتكررة لتجنبها في المرة القادمة، حتى لو لم يذكر صراحةً "اختتام الجلسة".
