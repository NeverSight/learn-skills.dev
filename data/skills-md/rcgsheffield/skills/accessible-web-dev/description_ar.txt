بناء تطبيقات الويب المتوافقة مع WCAG 2.1 AA لجامعة شيفيلد. يغطي أنماط HTML وARIA الدلالية وإمكانية الوصول إلى النماذج والتنقل عبر لوحة المفاتيح وتباين الألوان والنص البديل والتسميات التوضيحية والاختبار الآلي. يُستخدم عند إنشاء مواقع الويب، أو تطبيقات الويب، أو النماذج، أو المكونات التفاعلية، أو إمكانية الوصول إلى التدقيق من أجل امتثال WCAG، أو قارئات الشاشة، أو الوصول إلى لوحة المفاتيح، أو التصميم الشامل.
