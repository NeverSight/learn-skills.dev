Créez des applications Web conformes aux WCAG 2.1 AA pour l'Université de Sheffield. Couvre le HTML sémantique, les modèles ARIA, l'accessibilité des formulaires, la navigation au clavier, le contraste des couleurs, le texte alternatif, les légendes et les tests automatisés. À utiliser lors de la création de sites Web, d'applications Web, de formulaires, de composants interactifs ou lors de l'audit de l'accessibilité pour la conformité WCAG, les lecteurs d'écran, l'accès au clavier ou la conception inclusive.
