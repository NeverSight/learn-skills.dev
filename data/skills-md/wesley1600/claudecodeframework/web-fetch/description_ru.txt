Извлекает и анализирует контент из предоставленных пользователем URL-адресов или результатов поиска с помощью инструмента WebFetch. Управляет фильтрацией доменов и ограничениями max_uses для предотвращения кражи данных. Используйте, когда пользователи предоставляют URL-адреса для загрузки, при анализе веб-страниц или при отслеживании результатов поиска.
