Créez et gérez des compartiments Amazon S3 avec l'authentification EKS Pod Identity. À utiliser lorsque : (1) création de compartiments S3 pour tout service exécuté sur EKS, (2) configuration de l'identité du pod EKS afin que les pods puissent accéder à S3 sans informations d'identification statiques, (3) configuration des rôles/politiques IAM pour l'accès S3, (4) connexion des services Kubernetes au stockage S3 sur EKS, (5) dépannage de l'accès S3 à partir des pods Kubernetes. Nécessite : aws CLI v2, kubectl, eksctl.
