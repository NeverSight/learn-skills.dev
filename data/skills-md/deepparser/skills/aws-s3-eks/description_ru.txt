Создавайте корзины Amazon S3 и управляйте ими с помощью аутентификации EKS Pod Identity. Используйте при: (1) создании корзин S3 для любой службы, работающей в EKS, (2) настройке идентификации модуля EKS, чтобы модули могли получать доступ к S3 без статических учетных данных, (3) настройке ролей/политик IAM для доступа к S3, (4) подключении служб Kubernetes к хранилищу S3 в EKS, (5) устранении неполадок доступа S3 из модулей Kubernetes. Требуется: aws CLI v2, kubectl, eksctl.
