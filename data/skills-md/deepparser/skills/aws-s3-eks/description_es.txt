Cree y administre depósitos de Amazon S3 con autenticación EKS Pod Identity. Úselo cuando: (1) cree depósitos de S3 para cualquier servicio que se ejecute en EKS, (2) configure la identidad del pod de EKS para que los pods puedan acceder a S3 sin credenciales estáticas, (3) configure roles/políticas de IAM para el acceso a S3, (4) conecte servicios de Kubernetes al almacenamiento de S3 en EKS, (5) solucione problemas de acceso a S3 desde pods de Kubernetes. Requiere: AWS CLI v2, kubectl, eksctl.
