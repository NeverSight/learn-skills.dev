Инструмент CLI, похожий на Curl, для выполнения аутентифицированных запросов к API X (Twitter). Используйте этот навык, когда вам нужно публиковать твиты, отвечать, цитировать, искать, читать сообщения, управлять подписчиками, отправлять личные сообщения, загружать медиафайлы или взаимодействовать с любой конечной точкой X API v2. Поддерживает несколько приложений, OAuth 2.0, OAuth 1.0a и аутентификацию только для приложений.
