أداة CLI تشبه الضفيرة لتقديم طلبات مصادق عليها إلى X (Twitter) API. استخدم هذه المهارة عندما تحتاج إلى نشر تغريدات أو الرد أو الاقتباس أو البحث أو قراءة المنشورات أو إدارة المتابعين أو إرسال رسائل مباشرة أو تحميل الوسائط أو التفاعل مع أي نقطة نهاية X API v2. يدعم تطبيقات متعددة، OAuth 2.0، وOAuth 1.0a، ومصادقة التطبيق فقط.
