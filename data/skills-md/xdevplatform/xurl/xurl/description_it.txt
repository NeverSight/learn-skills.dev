Uno strumento CLI simile a un ricciolo per effettuare richieste autenticate all'API X (Twitter). Utilizza questa competenza quando devi pubblicare tweet, rispondere, citare, cercare, leggere post, gestire follower, inviare messaggi diretti, caricare contenuti multimediali o interagire con qualsiasi endpoint X API v2. Supporta più app, OAuth 2.0, OAuth 1.0a e autenticazione solo app.
