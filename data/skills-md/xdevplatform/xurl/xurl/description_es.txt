Una herramienta CLI similar a curl para realizar solicitudes autenticadas a la API X (Twitter). Utilice esta habilidad cuando necesite publicar tweets, responder, citar, buscar, leer publicaciones, administrar seguidores, enviar mensajes directos, cargar medios o interactuar con cualquier punto final de X API v2. Admite múltiples aplicaciones, OAuth 2.0, OAuth 1.0a y autenticación solo de aplicaciones.
