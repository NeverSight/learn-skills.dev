Revise la calidad de las pruebas y la cobertura de las pruebas de auditoría para cualquier módulo. Esta habilidad debe usarse al revisar pruebas existentes, auditar las lagunas en las pruebas, redactar nuevas pruebas o cuando se le solicite evaluar el estado de las pruebas. Incorpora estándares de prueba al flujo de trabajo de auditoría para producir un informe de brechas priorizadas. El resultado es un informe, no un código; no escriba implementaciones de prueba hasta que se revise el informe.
