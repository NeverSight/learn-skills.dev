Настраивает TraceKit APM в приложениях React с границами ошибок, отслеживанием производительности компонентов и распределенной трассировкой. Используйте, когда пользователь просит добавить TraceKit, добавить возможность наблюдения, инструментировать приложение React или настроить APM в проекте React/TypeScript.
