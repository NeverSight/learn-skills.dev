Lens Studio TypeScript コンポーネント システムのリファレンス ガイド — @component、@input、@hint、@allowUnknown、および @label デコレータ、BaseScriptComponent ライフサイクル (onAwake と OnStartEvent、UpdateEvent、DelayedCallbackEvent のワンショットおよび反復タイマー、TurnOnEvent/TurnOffEvent、onDestroy)、getComponent によるコンポーネントへのアクセス (さらに、「null のプロパティを読み取れない」を修正するための null チェック パターンをカバー)エラー)、getTypeName() による TypeScript 間インポート、NativeLogger と印刷、プレハブのインスタンス化 (同期および非同期)、SceneObject 階層クエリ、オブジェクトの有効化/無効化。このスキルは、Lens Studio TypeScript スクリプトの作成やデバッグ、シーン オブジェクトの接続、または「これは未定義です」エラーや null 参照エラーの修正を行うときに常に使用します。プラットフォームに依存しません (眼鏡や電話のレンズに機能します)。
