Справочное руководство по пониманию мира и оценке в Lens Studio, охватывающее WorldQueryModule HitTestSession (HitTestSessionOptions.filter для сглаживания дрожания, семантическую классификацию поверхностей для обнаружения пола/стены/потолка/стола, обработку нулевых результатов, покадровую производительность), SIK InteractionManager, нацеленный на шаблон интерактивных лучей, Physics.createGlobalProbe().rayCast для попаданий коллайдера сцены с фильтрацией слоев столкновений, выравнивание объектов по поверхности нормальные значения с использованием quat.lookAt и LeaderboardModule (создание/получение с помощью TTL и OrderingType, submitScore, getLeaderboardInfo с помощью UsersType.Global/Friends). Используйте этот навык при обнаружении реальных полов/стен/столов для размещения AR-контента, raycasting для наведения курсора или взаимодействия с объектами сцены или добавления глобальной таблицы лидеров внутри линзы — это отличие от очков-линз-основных (физика/SIK) и от очков-облака (постоянство Supabase).
