قم بكتابة أو تحديث الوثائق الخاصة بالتغييرات الهندسية - سواء المتعلقة بالمنتج (مستندات المستخدم، ومرجع واجهة برمجة التطبيقات، والأدلة) والداخلية (مستندات البنية، وكتيبات التشغيل، ومستندات التعليمات البرمجية المضمنة). يبني نموذجًا عالميًا لما تغير ويتتبع عواقب التوثيق العابر عبر جميع الأسطح المتأثرة. يكتشف ويستخدم مهارات التوثيق الخاصة بالريبو وأدلة الأسلوب والاتفاقيات. مستقل أو قابل للتركيب مع /ship. المشغلات: المستندات، الوثائق، كتابة المستندات، تحديث المستندات، توثيق التغييرات، مستندات المنتج، المستندات الداخلية، سجل التغيير، دليل الترحيل.
