Напишите или обновите документацию для сайта документации Inkeep (пакет Agents-Docs). Запускается при: документах, документации, MDX, документах агентов, написании документов, обновлении документов, добавлении страницы, новом руководстве, справочнике по API, руководстве по интеграции.
