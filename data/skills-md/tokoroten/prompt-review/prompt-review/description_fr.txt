Cette compétence est utilisée lorsqu'un utilisateur demande « examiner l'invite », « analyser l'historique du dialogue » ou « évaluer ma compréhension », ou lorsqu'il est invoqué via /prompt-review. Il lit les journaux de conversation passés avec les agents IA (Claude Code, GitHub Copilot Chat, Cline, Roo Code, Windsurf, Antigravity, OpenCode), estime la compréhension technique, les modèles d'invite et la dépendance à l'IA, et génère un rapport.
