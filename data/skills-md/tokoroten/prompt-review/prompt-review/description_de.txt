Dieser Skill wird verwendet, wenn ein Benutzer „die Eingabeaufforderung überprüfen“, „den Dialogverlauf analysieren“ oder „mein Verständnis bewerten“ anfordert oder wenn er über /prompt-review aufgerufen wird. Es liest frühere KI-Agent-Konversationsprotokolle (Claude Code, GitHub Copilot Chat, Cline, Roo Code, Windsurf, Antigravity, OpenCode), schätzt das technische Verständnis, Eingabeaufforderungsmuster und KI-Abhängigkeit ein und erstellt einen Bericht.
