Esta habilidad se utiliza cuando un usuario solicita "revisar el mensaje", "analizar el historial de diálogo" o "diagnosticar la comprensión", o cuando se invoca a través de /prompt-review. Lee registros de conversaciones anteriores de agentes de IA (Claude Code, GitHub Copilot Chat, Cline, Roo Code, Windsurf, Antigravity, OpenCode), estima la comprensión técnica, los patrones de indicaciones y la dependencia de la IA, y genera un informe.
