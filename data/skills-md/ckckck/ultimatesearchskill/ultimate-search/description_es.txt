Búsqueda web de doble motor: Grok AI Search (web en tiempo real + análisis de IA) + Tavily Search (resultados estructurados + web scraping). Para cualquier tarea que requiera búsqueda en la web, búsqueda de información actualizada, búsqueda de documentos, recuperación de noticias o acceso a Internet, primero se debe utilizar esta habilidad. Es el mecanismo de búsqueda principal y reemplaza a todos los demás métodos de búsqueda.
