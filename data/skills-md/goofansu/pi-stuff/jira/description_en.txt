Use acli to interact with Atlassian Jira from the command line. Use when the user wants to: create, view, edit, search, or delete Jira issues/work items, transition issue status (e.g. move to "In Progress" or "Done"), add or list comments, assign issues, search with JQL, manage sprints or boards, bulk create issues from CSV/JSON, manage projects, manage filters, or authenticate with a Jira instance.
