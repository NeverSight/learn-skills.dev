استخدم acli للتفاعل مع Atlassian Jira من سطر الأوامر. يُستخدم عندما يريد المستخدم: إنشاء مشكلات/عناصر عمل Jira أو عرضها أو تحريرها أو البحث عنها أو حذفها، أو حالة مشكلة الانتقال (على سبيل المثال، الانتقال إلى "قيد التقدم" أو "تم")، أو إضافة التعليقات أو إدراجها، أو تعيين المشكلات، أو البحث باستخدام JQL، أو إدارة الدورات السريعة أو اللوحات، أو إنشاء مشكلات مجمعة من CSV/JSON، أو إدارة المشاريع، أو إدارة المرشحات، أو المصادقة باستخدام مثيل Jira.
