يُستخدم عند العمل على مشاريع TanStack Start - يفرض قواعد البنية (الطبقات، والمسارات، والخطافات، ووظائف الخادم، والاصطلاحات) مع التحقق الإلزامي قبل أي تغيير في التعليمات البرمجية. مشغلات عند إنشاء الملف، أو مسار العمل، أو كتابة وظيفة الخادم، أو أنماط الخطاف، أو أي تغيير هيكلي في قاعدة بيانات TanStack Start.
