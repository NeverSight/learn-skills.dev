À utiliser lorsque vous travaillez sur des projets TanStack Start - applique les règles d'architecture (couches, routes, hooks, fonctions serveur, conventions) avec validation obligatoire avant toute modification de code. Déclencheurs lors de la création de fichiers, du travail d'itinéraire, de l'écriture de fonctions de serveur, des modèles de hook ou de tout changement structurel dans une base de code TanStack Start.
