Используйте интерфейс командной строки Glassnode (gn) для составления списка активов и показателей, получения сетевых и рыночных данных из API Glassnode и управления конфигурацией. Используйте, когда пользователь спрашивает о Glassnode, данных в цепочке, криптометриках, командах gn или ему необходимо вызвать API Glassnode с терминала.
