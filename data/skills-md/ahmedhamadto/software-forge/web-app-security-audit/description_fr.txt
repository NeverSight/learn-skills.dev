À utiliser lors du test d'une application Web pour détecter les vulnérabilités de sécurité, avant le déploiement ou pendant l'examen de la sécurité : guide à travers une méthodologie structurée de test d'intrusion en 10 phases couvrant le mappage, l'authentification, la gestion de session, les contrôles d'accès, l'injection, les failles logiques et la configuration du serveur.
