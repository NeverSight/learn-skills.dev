GitLab CLI (glab) per richieste di unione, problemi e pipeline CI/CD. Da utilizzare quando si lavora con repository GitLab per la creazione/revisione di MR, la gestione dei problemi, il debug della pipeline o qualsiasi operazione API GitLab. Trigger su URL GitLab, menzioni di "richiesta di unione" o "MR" (non "PR"), gitlab.com o comandi glab.
