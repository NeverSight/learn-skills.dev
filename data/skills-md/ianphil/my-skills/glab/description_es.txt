GitLab CLI (glab) para solicitudes de fusión, problemas y canalizaciones de CI/CD. Úselo cuando trabaje con repositorios de GitLab para la creación/revisión de MR, gestión de problemas, depuración de canalizaciones o cualquier operación de API de GitLab. Activadores en URL de GitLab, menciones de "solicitud de fusión" o "MR" (no "PR"), gitlab.com o comandos glab.
