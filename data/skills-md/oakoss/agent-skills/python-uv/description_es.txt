Desarrollo moderno de Python con uv, el rápido paquete Python y administrador de proyectos. Cubre la gestión de proyectos (uv init, uv add, uv sync, uv lock), entornos virtuales, gestión de versiones de Python (uv python install/pin), ejecutores de scripts (uv run), gestión de herramientas (uvx), soporte de espacio de trabajo para monorepos y publicación en PyPI. Incluye patrones de Python para herramientas FastAPI, Pydantic, async/await, verificación de tipos, pytest, structlog y CLI.

Úselo al inicializar proyectos de Python, administrar dependencias con uv, configurar pyproject.toml, configurar entornos virtuales, ejecutar scripts, administrar versiones de Python, crear monorepos con espacios de trabajo, contener aplicaciones de Python o escribir Python moderno con sugerencias de tipo.
