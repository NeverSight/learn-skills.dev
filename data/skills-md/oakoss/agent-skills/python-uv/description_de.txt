Moderne Python-Entwicklung mit uv, dem schnellen Python-Paket und Projektmanager. Umfasst Projektmanagement (uv init, uv add, uv sync, uv lock), virtuelle Umgebungen, Python-Versionsverwaltung (uv python install/pin), Skriptläufer (uv run), Toolverwaltung (uvx), Workspace-Unterstützung für Monorepos und Veröffentlichung auf PyPI. Enthält Python-Muster für FastAPI, Pydantic, Async/Await, Typprüfung, Pytest, Structlog und CLI-Tools.

Verwenden Sie es, wenn Sie Python-Projekte initialisieren, Abhängigkeiten mit UV verwalten, pyproject.toml konfigurieren, virtuelle Umgebungen einrichten, Skripts ausführen, Python-Versionen verwalten, Monorepos mit Arbeitsbereichen erstellen, Python-Apps containerisieren oder modernes Python mit Typhinweisen schreiben.
