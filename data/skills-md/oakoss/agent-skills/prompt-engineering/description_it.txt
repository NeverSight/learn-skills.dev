Modelli di ingegneria tempestiva e orchestrazione degli agenti. Da utilizzare quando si creano prompt per modelli di ragionamento, si implementa la catena di pensiero o l'albero dei pensieri, si progettano cicli ReAct, si creano esempi di poche riprese, si ottimizzano le prestazioni dei prompt, si strutturano prompt di sistema, si utilizza il pensiero esteso o si progettano flussi di lavoro per l'utilizzo di strumenti. Utilizzare per modelli di prompt, flussi di lavoro degli agenti in più fasi, protocolli di pensiero strutturato e prompt multimodali.
