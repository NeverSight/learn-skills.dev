Tabelle dati senza testa TanStack Table v8 per React. Copre definizioni di colonne, ordinamento, filtraggio (fuzzy/sfaccettato), impaginazione lato server con TanStack Query, scorrimento infinito, virtualizzazione (TanStack Virtual), blocco di colonne/righe, espansione/raggruppamento di righe, ridimensionamento di colonne e componenti riutilizzabili in stile Shadcn. Previene 15 errori documentati tra cui infiniti rendering, incompatibilità del compilatore React e mancate corrispondenze di stato lato server.

Da utilizzare durante la creazione di tabelle di dati, la correzione delle prestazioni delle tabelle, l'implementazione dell'impaginazione lato server, l'aggiunta di filtri/ordinamenti o il debug di problemi relativi allo stato delle tabelle.
