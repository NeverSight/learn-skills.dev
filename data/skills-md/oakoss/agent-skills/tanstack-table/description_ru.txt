Таблицы безголовых данных TanStack Table v8 для React. Охватывает определения столбцов, сортировку, фильтрацию (нечеткую/фасетную), разбиение на страницы на стороне сервера с помощью TanStack Query, бесконечную прокрутку, виртуализацию (TanStack Virtual), закрепление столбца/строки, расширение/группировку строк, изменение размера столбца и повторно используемые компоненты в стиле Shadcn. Предотвращает 15 задокументированных ошибок, включая бесконечные повторные рендеринги, несовместимость компилятора React и несоответствие состояний на стороне сервера.

Используйте при построении таблиц данных, исправлении производительности таблиц, реализации разбиения на страницы на стороне сервера, добавлении фильтрации/сортировки или отладке проблем с состоянием таблицы.
