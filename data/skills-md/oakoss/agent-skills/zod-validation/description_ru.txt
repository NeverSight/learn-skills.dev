Проверка схемы Zod v4 для TypeScript. Охватывает примитивы, строковые форматы, объекты, массивы, объединения, приведение, преобразования, уточнения, синтаксический анализ, вывод типов, настройку ошибок, схему JSON, проверку файлов и метаданные. Используйте при написании схем, проверке входных данных, анализе данных, выводе типов или преобразовании схем с помощью Zod.
