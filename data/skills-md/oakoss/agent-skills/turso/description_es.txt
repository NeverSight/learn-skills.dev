Plataforma SQLite alojada en el borde de Turso construida sobre libSQL (bifurcación SQLite de código abierto). Cubre @libsql/client SDK, réplicas integradas con sincronización local, arquitectura de múltiples bases de datos por inquilino, API de plataforma para aprovisionamiento de bases de datos, migraciones de esquemas, búsqueda vectorial con F32_BLOB, operaciones por lotes, transacciones interactivas y cifrado en reposo.

Úselo al conectarse a bases de datos de Turso, configurar réplicas integradas, aprovisionar bases de datos a través de la API de la plataforma, implementar aislamiento de bases de datos por inquilino, realizar búsquedas de similitud de vectores o integrar libSQL con Drizzle ORM.
