Piattaforma SQLite edge-hosted Turso basata su libSQL (fork SQLite open source). Copre l'SDK @libsql/client, repliche incorporate con sincronizzazione locale, architettura multi-database per tenant, API della piattaforma per il provisioning del database, migrazioni di schemi, ricerca vettoriale con F32_BLOB, operazioni batch, transazioni interattive e crittografia dei dati inattivi.

Da utilizzare durante la connessione ai database Turso, la configurazione di repliche incorporate, il provisioning di database tramite l'API della piattaforma, l'implementazione dell'isolamento del database per tenant, l'esecuzione di ricerche di similarità vettoriale o l'integrazione di libSQL con Drizzle ORM.
