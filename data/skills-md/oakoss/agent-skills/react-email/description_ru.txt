Создавайте адаптивные электронные письма в формате HTML с помощью компонентов React Email. Охватывает структуру электронного письма (Html, заголовок, тело, контейнер), компоненты контента (текст, заголовок, кнопка, ссылка, изображение), макет (раздел, строка, столбец), стили с помощью встроенного CSS и Tailwind, пользовательские шрифты, предварительный просмотр текста и рендеринг в HTML.
Используйте при создании транзакционных электронных писем, маркетинговых шаблонов или систем дизайна электронной почты. Используйте для рендеринга электронной почты, предварительного просмотра, оформления электронной почты Tailwind и интеграции поставщика с Resend, Nodemailer или SendGrid.
