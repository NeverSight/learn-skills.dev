API tRPC v11 de bout en bout de type sécurisé pour TypeScript. Couvre les définitions de routeur et de procédure, la validation des entrées avec Zod, le middleware et le contexte, les abonnements, la gestion des erreurs avec TRPCError, les hooks client React via @trpc/react-query, les appelants côté serveur avec createCallerFactory, la configuration de l'adaptateur (autonome, Express, Fastify, Hono) et les modèles de test.

À utiliser lors de la création de routeurs tRPC, de la définition de procédures, de la configuration de middleware, de la configuration d'adaptateurs, de l'intégration de tRPC avec React, de la gestion des erreurs, de la mise en œuvre d'abonnements ou du test des procédures tRPC.
