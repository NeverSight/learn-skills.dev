API indipendenti dai tipi end-to-end tRPC v11 per TypeScript. Copre definizioni di router e procedure, convalida dell'input con Zod, middleware e contesto, abbonamenti, gestione degli errori con TRPCError, hook client React tramite @trpc/react-query, chiamanti lato server con createCallerFactory, configurazione dell'adattatore (autonomo, Express, Fastify, Hono) e modelli di test.

Da utilizzare durante la creazione di router tRPC, la definizione di procedure, l'impostazione del middleware, la configurazione degli adattatori, l'integrazione di tRPC con React, la gestione degli errori, l'implementazione di abbonamenti o il test delle procedure tRPC.
