Workflows GitHub CLI pour les référentiels, les problèmes, les demandes d'extraction, les actions, les versions, les projets et les appels d'API. À utiliser lors de la création de PR, de la révision du code, du tri des problèmes, du déclenchement de flux de travail, de la publication de versions, de la gestion de projets, de la vérification des attestations ou de l'interrogation de l'API GitHub. Utiliser pour gh cli, l'automatisation de github, la révision du code, la gestion des versions.
