Framework Tauri v2 per la creazione di applicazioni desktop e mobili con frontend web e backend Rust. Copre la configurazione del progetto, i comandi e gli eventi IPC, la gestione delle finestre, la barra delle applicazioni, l'accesso al file system, il sistema di plug-in (appunti, finestra di dialogo, notifica, aggiornamento, collegamento profondo), app multi-finestra, file binari sidecar, protocolli personalizzati, firma delle app, distribuzione, aggiornamenti automatici e modello di sicurezza basato sulle funzionalità.

Da utilizzare durante la creazione di un progetto Tauri, la definizione di comandi Rust, l'invocazione della logica di backend dal frontend, la configurazione di autorizzazioni e funzionalità, l'impostazione di plug-in, la gestione dello stato dell'applicazione, la creazione di menu nella barra delle applicazioni, la creazione di layout multi-finestra, il raggruppamento di file binari sidecar, l'implementazione di aggiornamenti automatici o la firma di app per la distribuzione.
