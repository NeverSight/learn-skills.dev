Marco Tauri v2 para crear aplicaciones móviles y de escritorio con interfaces web y backends Rust. Cubre la configuración del proyecto, comandos y eventos de IPC, administración de ventanas, bandeja del sistema, acceso al sistema de archivos, sistema de complementos (portapapeles, diálogo, notificación, actualizador, enlace profundo), aplicaciones de múltiples ventanas, archivos binarios complementarios, protocolos personalizados, firma de aplicaciones, distribución, actualizaciones automáticas y el modelo de seguridad basado en capacidades.

Úselo al desarrollar un proyecto Tauri, definir comandos de Rust, invocar la lógica de backend desde el frontend, configurar permisos y capacidades, configurar complementos, administrar el estado de la aplicación, crear menús de la bandeja del sistema, crear diseños de ventanas múltiples, agrupar binarios secundarios, implementar actualizaciones automáticas o firmar aplicaciones para su distribución.
