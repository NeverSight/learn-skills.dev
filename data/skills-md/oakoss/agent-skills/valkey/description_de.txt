Valkey (Redis-kompatibler) In-Memory-Datenspeicher für Caching, Sitzungsspeicher, Pub/Sub, Warteschlangen und verteilte Sperren. Verwenden Sie diese Option, wenn Sie Valkey- oder Redis-kompatibles Caching konfigurieren, Räumungsrichtlinien auswählen, Ratenbegrenzung implementieren, Pub/Sub oder Streams einrichten, Docker Compose-Dienste schreiben, von Redis migrieren oder die Leistung optimieren. Verwendung für Valkey, Redis, Cache, Sitzung, Pub/Sub, Streams, verteilte Sperre, Ratenbegrenzung, sortierte Menge.
