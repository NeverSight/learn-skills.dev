Создавайте интерфейсы, совместимые с WCAG 2.2 AA, с семантическим HTML, ARIA, навигацией с помощью клавиатуры, управлением фокусом, цветовым контрастом и поддержкой чтения с экрана. Охватывает формы, диалоговые окна, вкладки, динамические области, ссылки пропуска, замещающий текст и таблицы данных.

Используйте при реализации доступных пользовательских интерфейсов, аудите соответствия WCAG, устранении проблем чтения с экрана, навигации с помощью клавиатуры, ловушек фокуса или устранения неполадок «отсутствует контур фокуса», «требуется aria-метка», «недостаточный контраст», «отсутствует замещающий текст», «иерархия заголовков».
