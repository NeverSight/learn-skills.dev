Diseña y organiza ecosistemas de protocolo de contexto modelo (MCP), incluido el desarrollo de servidores, el diseño de herramientas y la configuración de transporte. Utilícelo al crear servidores MCP, diseñar interfaces de herramientas agentes, configurar transportes MCP, implementar seguridad OAuth o solucionar problemas de conectividad MCP. Úselo para herramientas orientadas a resultados, validación de Zod, recursos de divulgación progresiva, obtención y tareas asincrónicas.
