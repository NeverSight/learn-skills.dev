Modelli ed ecosistema del linguaggio Rust che coprono proprietà, prestito, durata, gestione degli errori (Risultato/Opzione, questo errore, comunque), asincronia con Tokio, tratti, generici, serializzazione di serde, aree di lavoro Cargo e test. Include framework web axum, interfaccia utente del terminale ratatui, analisi CLI clap, benchmarking dei criteri, sicurezza della catena di fornitura cargo-deny, distribuzione binaria cargo-dist e automazione del changelog release-plz.

Da utilizzare durante la scrittura di codice Rust, la progettazione di modelli di proprietà, la creazione di servizi asincroni con Tokio/axum, la creazione di UI terminali con ratatui, l'impostazione di applicazioni CLI con clap, la configurazione di pipeline CI/CD per Rust, il benchmarking con criterion, il controllo delle dipendenze con cargo-deny o la distribuzione di file binari con cargo-dist.
