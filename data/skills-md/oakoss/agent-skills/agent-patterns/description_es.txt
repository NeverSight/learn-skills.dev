Patrones de diseño multiagente para Claude Code y sistemas de desarrollo de IA. Cubre subagentes personalizados, subagentes integrados (Explorar/Planificar/Propósito general), agentes fungibles versus especializados, delegación, orquestación, flujos de trabajo por lotes, higiene de contexto y recuperación de fallas.

Úselo cuando: diseñe sistemas de múltiples agentes, cree subagentes personalizados, delegue operaciones masivas, orqueste trabajo paralelo, elija entre agentes fungibles y especializados, configure el front-matter del agente o depure problemas de coordinación de agentes.
