Modelli di progettazione multi-agente per Claude Code e sistemi di sviluppo AI. Copre agenti secondari personalizzati, agenti secondari integrati (Esplora/Pianifica/Scopo generale), agenti fungibili e specializzati, delega, orchestrazione, flussi di lavoro batch, igiene del contesto e ripristino degli errori.

Da utilizzare quando si progettano sistemi multi-agente, si creano agenti secondari personalizzati, si delegano operazioni in blocco, si orchestra il lavoro parallelo, si sceglie tra agenti fungibili e specializzati, si configura il frontmatter dell'agente o si esegue il debug di problemi di coordinamento dell'agente.
