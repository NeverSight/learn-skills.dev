Modèles de conception multi-agents pour les systèmes de développement Claude Code et IA. Couvre les sous-agents personnalisés, les sous-agents intégrés (Explorer/Planifier/À usage général), les agents fongibles ou spécialisés, la délégation, l'orchestration, les flux de travail par lots, l'hygiène du contexte et la récupération après échec.

À utiliser lors de : la conception de systèmes multi-agents, la création de sous-agents personnalisés, la délégation d'opérations groupées, l'orchestration de travaux parallèles, le choix entre des agents fongibles et spécialisés, la configuration de la façade d'agent ou le débogage de problèmes de coordination d'agent.
