Turborepo-Monorepo-Build-System und Orchestrierung. Behandelt Aufgabenpipelines, Abhängigkeitssyntax, Caching-Konfiguration, Remote-Cache, Filterung, CI-Optimierung, Umgebungsvariablen, Arbeitsbereichsverwaltung, Überwachungsmodus, Paketgrenzen und Codegenerierung.

Wird zum Konfigurieren von Aufgaben, zum Erstellen von Paketen, zum Einrichten von Monorepo, zum Teilen von Code zwischen Apps, zum Ausführen geänderter Pakete, zum Debuggen des Caches, zum Optimieren von CI, zum Auflösen von Arbeitsbereichsabhängigkeiten, zum Durchsetzen von Paketgrenzen oder zum Generieren von Code verwendet.
