Configuración y ecosistema de la herramienta de compilación de Vite. Cubre la configuración de vite.config.ts, la creación de complementos y complementos populares, la configuración del servidor de desarrollo (proxy, HMR, HTTPS), la optimización de la compilación (fragmentación, agitación de árboles, fragmentos manuales, rollupOptions), modo de biblioteca para publicar paquetes, configuración SSR, variables de entorno (manejo de .env), aplicaciones de varias páginas, manejo de CSS (PostCSS, módulos CSS, preprocesadores) y manejo de activos.

Úselo al configurar proyectos de Vite, crear complementos de Vite, optimizar compilaciones, configurar servidores proxy de desarrollo, configurar SSR, manejar variables de entorno o solucionar problemas de Vite.
