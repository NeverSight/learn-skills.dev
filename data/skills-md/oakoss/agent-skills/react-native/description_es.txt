React Native con el marco Expo para crear aplicaciones móviles nativas en Android, iOS y web desde una única base de código TypeScript. Cubre la navegación basada en archivos de Expo Router (pila, pestañas, cajón), EAS Build/Submit/Update para actualizaciones de CI/CD y OTA, expo-dev-client para compilaciones de desarrollo personalizadas, Expo Modules API para módulos nativos Swift/Kotlin, patrones principales de RN (StyleSheet, FlatList, código específico de la plataforma), API de dispositivos nativos (cámara, notificaciones, hápticos) y NativeWind para estilo CSS Tailwind.

Úselo al crear aplicaciones React Native con Expo, configurar la navegación de Expo Router, configurar canalizaciones de EAS Build, implementar actualizaciones OTA, crear módulos nativos o integrar API de dispositivos como cámara y notificaciones push.
