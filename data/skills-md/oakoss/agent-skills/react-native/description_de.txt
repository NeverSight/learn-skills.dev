React Native mit Expo-Framework zum Erstellen nativer mobiler Apps für Android, iOS und Web aus einer einzigen TypeScript-Codebasis. Behandelt die dateibasierte Navigation des Expo Routers (Stack, Tabs, Schublade), EAS Build/Submit/Update für CI/CD- und OTA-Updates, Expo-Dev-Client für benutzerdefinierte Entwicklungs-Builds, Expo Modules API für native Swift/Kotlin-Module, Kern-RN-Muster (StyleSheet, FlatList, plattformspezifischer Code), native Geräte-APIs (Kamera, Benachrichtigungen, Haptik) und NativeWind für Tailwind CSS-Stil.

Verwenden Sie es, wenn Sie React Native-Apps mit Expo erstellen, die Expo-Router-Navigation konfigurieren, EAS Build-Pipelines einrichten, OTA-Updates implementieren, native Module erstellen oder Geräte-APIs wie Kamera- und Push-Benachrichtigungen integrieren.
