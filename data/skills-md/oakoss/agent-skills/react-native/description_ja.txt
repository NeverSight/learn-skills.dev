単一の TypeScript コードベースから Android、iOS、Web 上でネイティブ モバイル アプリを構築するための Expo フレームワークを使用した React Native。 Expo Router のファイルベースのナビゲーション (スタック、タブ、ドロワー)、CI/CD および OTA 更新用の EAS ビルド/送信/更新、カスタム開発ビルド用の expo-dev-client、ネイティブ Swift/Kotlin モジュール用の Expo モジュール API、コア RN パターン (スタイルシート、フラットリスト、プラットフォーム固有のコード)、ネイティブ デバイス API (カメラ、通知、ハプティクス)、Tailwind CSS スタイル用の NativeWind をカバーします。

Expo を使用した React Native アプリの構築、Expo Router ナビゲーションの構成、EAS Build パイプラインのセットアップ、OTA アップデートの実装、ネイティブ モジュールの作成、またはカメラやプッシュ通知などのデバイス API の統合を行うときに使用します。
