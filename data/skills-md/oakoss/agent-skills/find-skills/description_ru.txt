Обнаруживает и устанавливает навыки агентов из открытой экосистемы. Дополняет результаты поиска описаниями, полученными из файлаkills.sh.

Используйте, когда пользователь спрашивает: «Как мне сделать X», «найти навык для X», «есть ли навык, который может...» или хочет расширить возможности агента. Используйте для открытия навыков, поиска навыков, установки навыков, поиска инструментов.
