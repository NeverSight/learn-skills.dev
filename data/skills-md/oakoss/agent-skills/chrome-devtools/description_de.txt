Browser-Automatisierung über Chrome-Erweiterung und DevTools MCP. Zur Steuerung eines angemeldeten Chrome-Browsers, zur Automatisierung von Arbeitsabläufen, zum Ausfüllen von Formularen, zum Extrahieren von Daten, zum Debuggen von Web-Apps, zum Lesen von Konsolenprotokollen oder zum Ausführen geplanter Browseraufgaben. Verwendung für Browsertests, Formularautomatisierung, Datenextraktion, Web Scraping und authentifiziertes Surfen.
