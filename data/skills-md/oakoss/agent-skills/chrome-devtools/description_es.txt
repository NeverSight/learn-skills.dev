Automatización del navegador a través de la extensión Chrome y DevTools MCP. Úselo para controlar un navegador Chrome conectado, automatizar flujos de trabajo, completar formularios, extraer datos, depurar aplicaciones web, leer registros de la consola o ejecutar tareas programadas del navegador. Úselo para pruebas de navegador, automatización de formularios, extracción de datos, web scraping y navegación autenticada.
