Vercel-Bereitstellungskonfiguration, Vorschauumgebungen und Edge-Funktionen. Behandelt die Konfiguration von vercel.json und vercel.ts, Umschreibungen, Weiterleitungen, Header, Umgebungsvariablen in Produktion/Vorschau/Entwicklung, benutzerdefinierte Domänen, DNS-Setup, Edge- und serverlose Funktionen, Vercel CLI, GitHub Actions CI/CD und Monorepo-Bereitstellungen.

Zur Verwendung bei der Bereitstellung auf Vercel, beim Konfigurieren von Builds, beim Einrichten von Vorschauzweigen, beim Verwalten von Umgebungsvariablen, beim Konfigurieren benutzerdefinierter Domänen oder beim Verwenden von Edge-/serverlosen Funktionen. Verwendung für Vercel, Bereitstellung, Vorschau, Edge-Funktion, Serverlos, Vercel-JSON, Umgebungsvariablen, Domänen, Monorepo.
