Configuration du déploiement Vercel, environnements de prévisualisation et fonctions Edge. Couvre la configuration vercel.json et vercel.ts, les réécritures, les redirections, les en-têtes, les variables d'environnement dans la production/préversion/développement, les domaines personnalisés, la configuration DNS, les fonctions Edge et sans serveur, la CLI Vercel, les actions CI/CD GitHub et les déploiements monorepo.

À utiliser lors du déploiement sur Vercel, de la configuration des builds, de la configuration des branches d'aperçu, de la gestion des variables d'environnement, de la configuration de domaines personnalisés ou de l'utilisation de fonctions Edge/sans serveur. À utiliser pour vercel, déployer, prévisualiser, fonction Edge, sans serveur, vercel-json, variables d'environnement, domaines, monorepo.
