Création de workflow GitHub Actions pour les pipelines CI/CD. Couvre la syntaxe des flux de travail, les déclencheurs, les tâches, les étapes, les stratégies matricielles, la mise en cache, les artefacts, les flux de travail réutilisables, les actions composites, la gestion des secrets, l'authentification OIDC et le renforcement des autorisations.
À utiliser lors de la création de flux de travail, de la configuration de tâches, de la configuration de la mise en cache ou de l'automatisation des versions. À utiliser pour les actions github, le workflow, ci, cd, les actions, les tâches, les étapes, la matrice, le cache, l'artefact, les secrets, le workflow réutilisable.
