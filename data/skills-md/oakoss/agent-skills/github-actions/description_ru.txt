Разработка рабочего процесса GitHub Actions для конвейеров CI/CD. Охватывает синтаксис рабочего процесса, триггеры, задания, шаги, матричные стратегии, кэширование, артефакты, многократно используемые рабочие процессы, составные действия, управление секретами, аутентификацию OIDC и усиление разрешений.
Используйте при создании рабочих процессов, настройке заданий, настройке кэширования или автоматизации выпусков. Используйте для действий github, рабочего процесса, ci, cd, действий, заданий, шагов, матрицы, кеша, артефакта, секретов, рабочего процесса многократного использования.
