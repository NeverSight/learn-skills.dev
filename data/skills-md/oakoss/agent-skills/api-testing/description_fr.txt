Modèles de test API avec supertest, MSW et Vitest. Couvre les tests d'intégration pour les API REST, la simulation de requêtes HTTP avec Mock Service Worker v2, les assertions de réponse, la validation de schéma, l'organisation des tests et les modèles spécifiques au framework pour Express, Fastify et Hono.

À utiliser lors de l'écriture de tests d'intégration pour les API REST, de requêtes HTTP simulées ou de tests de points de terminaison d'API. À utiliser pour api-test, supertest, msw, mock-service-worker, Integration-test, http-mock, endpoint-test, request-test.
