Шаблоны тестирования API с помощью Supertest, MSW и Vitest. Охватывает интеграционное тестирование для REST API, имитацию HTTP-запросов с помощью Mock Service Worker v2, утверждения ответов, проверку схемы, организацию тестирования и специфичные для платформы шаблоны для Express, Fastify и Hono.

Используйте при написании интеграционных тестов для REST API, имитации HTTP-запросов или тестировании конечных точек API. Используйте для API-теста, супертеста, MSW, Mock-Service-Worker, Integration-Test, HTTP-Mock, Endpoint-Test, Request-Test.
