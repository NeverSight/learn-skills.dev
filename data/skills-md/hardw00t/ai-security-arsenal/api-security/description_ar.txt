مهارة اختبار أمان واجهة برمجة التطبيقات الشاملة لواجهات برمجة تطبيقات REST وGraphQL وgRPC وWebSocket. يجب استخدام هذه المهارة عند إجراء اختبار اختراق واجهة برمجة التطبيقات، واختبار أهم 10 نقاط ضعف لواجهة برمجة تطبيقات OWASP، وتشويش نقاط نهاية واجهة برمجة التطبيقات، واختبار المصادقة/الترخيص، وتحليل مواصفات واجهة برمجة التطبيقات. يتم تشغيل الطلبات لاختبار أمان واجهة برمجة التطبيقات، أو اختبار واجهات برمجة تطبيقات REST، أو اختبار نقاط نهاية GraphQL، أو تحليل مواصفات OpenAPI/Swagger، أو العثور على ثغرات أمنية في واجهة برمجة التطبيقات.
