Compétence complète en matière de tests de sécurité des API pour les API REST, GraphQL, gRPC et WebSocket. Cette compétence doit être utilisée lors de la réalisation de tests d'intrusion d'API, de tests des 10 principales vulnérabilités de l'API OWASP, de fuzzing de points de terminaison d'API, de tests d'authentification/autorisation et d'analyse des spécifications d'API. Se déclenche sur les requêtes pour tester la sécurité des API, tester les API REST, tester les points de terminaison GraphQL, analyser les spécifications OpenAPI/Swagger ou rechercher des vulnérabilités d'API.
