Competenze complete di test sulla sicurezza delle API per API REST, GraphQL, gRPC e WebSocket. Questa competenza dovrebbe essere utilizzata durante l'esecuzione di test di penetrazione dell'API, test delle 10 principali vulnerabilità dell'API OWASP, fuzzing degli endpoint API, test di autenticazione/autorizzazione e analisi delle specifiche API. Attiva le richieste per testare la sicurezza dell'API, eseguire il pentest delle API REST, testare gli endpoint GraphQL, analizzare le specifiche OpenAPI/Swagger o individuare le vulnerabilità dell'API.
