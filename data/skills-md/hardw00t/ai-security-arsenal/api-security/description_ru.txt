Навыки комплексного тестирования безопасности API для API REST, GraphQL, gRPC и WebSocket. Этот навык следует использовать при выполнении тестирования на проникновение API, тестировании уязвимостей API из списка OWASP Top 10, фаззинге конечных точек API, тестировании аутентификации/авторизации и анализе спецификаций API. Запускается по запросам на проверку безопасности API, пентест REST API, тестирование конечных точек GraphQL, анализ спецификаций OpenAPI/Swagger или поиск уязвимостей API.
