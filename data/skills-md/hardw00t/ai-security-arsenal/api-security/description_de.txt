Umfassende Fähigkeiten zum Testen der API-Sicherheit für REST-, GraphQL-, gRPC- und WebSocket-APIs. Diese Fertigkeit sollte beim Durchführen von API-Penetrationstests, beim Testen auf OWASP-API-Top-10-Schwachstellen, beim Fuzzing von API-Endpunkten, beim Testen der Authentifizierung/Autorisierung und beim Analysieren von API-Spezifikationen eingesetzt werden. Wird bei Anfragen zum Testen der API-Sicherheit, zum Pentest von REST-APIs, zum Testen von GraphQL-Endpunkten, zum Analysieren von OpenAPI/Swagger-Spezifikationen oder zum Auffinden von API-Schwachstellen ausgelöst.
