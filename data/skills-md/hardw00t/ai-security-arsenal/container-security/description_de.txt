Fähigkeiten zur Container- und Kubernetes-Sicherheitsbewertung für Docker-, Kubernetes- und Container-Orchestrierungsplattformen. Diese Fähigkeit sollte beim Scannen von Container-Images auf Schwachstellen, beim Überprüfen der Kubernetes-Cluster-Sicherheit, beim Testen von Container-Escape-Szenarien, beim Überprüfen von Docker-Konfigurationen oder beim Durchführen einer Analyse der Container-Laufzeitsicherheit verwendet werden. Wird bei Anfragen zum Scannen von Docker-Images, zur Prüfung der Kubernetes-Sicherheit, zum Testen von Containerkonfigurationen oder zur Bewertung der Sicherheit der Container-Orchestrierung ausgelöst.
