Habilidad de evaluación de seguridad de contenedores y Kubernetes para Docker, Kubernetes y plataformas de orquestación de contenedores. Esta habilidad debe usarse al escanear imágenes de contenedores en busca de vulnerabilidades, auditar la seguridad del clúster de Kubernetes, probar escenarios de escape de contenedores, revisar configuraciones de Docker o realizar análisis de seguridad en tiempo de ejecución de contenedores. Activa solicitudes para escanear imágenes de Docker, auditar la seguridad de Kubernetes, probar configuraciones de contenedores o evaluar la seguridad de la orquestación de contenedores.
