Навыки оценки безопасности контейнеров и Kubernetes для Docker, Kubernetes и платформ оркестрации контейнеров. Этот навык следует использовать при сканировании образов контейнеров на наличие уязвимостей, аудите безопасности кластера Kubernetes, тестировании сценариев выхода из контейнера, проверке конфигураций Docker или выполнении анализа безопасности во время выполнения контейнера. Запускается по запросам на сканирование образов Docker, аудит безопасности Kubernetes, тестирование конфигураций контейнеров или оценку безопасности оркестрации контейнеров.
