مهارة تقييم أمان الحاويات وKubernetes لمنصات Docker وKubernetes ومنصات تنسيق الحاويات. يجب استخدام هذه المهارة عند فحص صور الحاوية بحثًا عن الثغرات الأمنية، أو تدقيق أمان مجموعة Kubernetes، أو اختبار سيناريوهات الهروب من الحاوية، أو مراجعة تكوينات Docker، أو إجراء تحليل أمان وقت تشغيل الحاوية. تشغيل طلبات فحص صور Docker، أو تدقيق أمان Kubernetes، أو اختبار تكوينات الحاوية، أو تقييم أمان تنسيق الحاوية.
