YouTube ビデオからトランスクリプトを取得し、構造化 Markdown ファイルとして ~/yt_transcripts/ に保存します。ユーザーが YouTube URL またはビデオ ID を共有し、トランスクリプト、キャプション、または字幕を抽出したい場合は、このスキルを使用します。また、ユーザーが YouTube ビデオでメモを取ったり、ビデオ コンテンツを要約したり、ビデオ内で話された内容を分析したり、メモ内で YouTube コンテンツを参照したりする場合にもトリガーされます。たとえ「トランスクリプト」と明示的に言わなくても。ユーザーが他のコンテキストなしで YouTube リンクを貼り付けた場合、デフォルトでトランスクリプトが取得されます。
