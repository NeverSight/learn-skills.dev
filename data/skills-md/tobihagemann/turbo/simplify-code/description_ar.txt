قم بإجراء مراجعة متعددة الوكلاء للملفات التي تم تغييرها فيما يتعلق بمشكلات إعادة الاستخدام والجودة والكفاءة والوضوح متبوعة بإصلاحات تلقائية. يُستخدم عندما يطلب المستخدم "تبسيط التعليمات البرمجية"، أو "مراجعة التعليمات البرمجية التي تم تغييرها"، أو "التحقق من إعادة استخدام التعليمات البرمجية"، أو "مراجعة جودة التعليمات البرمجية"، أو "مراجعة الكفاءة"، أو "تبسيط التغييرات"، أو "تنظيف التعليمات البرمجية"، أو "تغييرات إعادة التصنيع"، أو "تشغيل التبسيط".
