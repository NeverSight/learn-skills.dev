يمكنك العثور على التعليمات البرمجية الميتة باستخدام تحليل الوكيل الفرعي المتوازي وأدوات CLI الاختيارية، والتعامل مع التعليمات البرمجية المشار إليها فقط من الاختبارات على أنها ميتة. يُستخدم عندما يطلب المستخدم "البحث عن الكود الميت"، أو "البحث عن الكود غير المستخدم"، أو "البحث عن الصادرات غير المستخدمة"، أو "البحث عن وظائف غير مرجعية"، أو "تنظيف الكود الميت"، أو "ما هو الكود غير المستخدم". التحليل فقط - لا يقوم بتعديل أو حذف التعليمات البرمجية.
