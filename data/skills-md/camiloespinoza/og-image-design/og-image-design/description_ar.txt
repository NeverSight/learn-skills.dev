تصميم وإنشاء صور Open Graph (OG) لمشاركة الوسائط الاجتماعية. استخدم هذه المهارة عندما يذكر المستخدم صور OG، أو صور المشاركة الاجتماعية، أو البطاقات الاجتماعية، أو بطاقات Twitter، أو صور معاينة الرابط، أو الصور التعريفية، أو og:image، أو الصور المصغرة الاجتماعية، أو يريد إنشاء صور تظهر عند مشاركة الروابط على Facebook، أو Twitter/X، أو LinkedIn، أو Discord، أو Slack، أو iMessage، أو أي منصة اجتماعية. يُستخدم أيضًا عندما يسأل المستخدم عن og: العلامات الوصفية للصورة، أو تحسين المعاينة الاجتماعية، أو ربط العناصر المرئية.
