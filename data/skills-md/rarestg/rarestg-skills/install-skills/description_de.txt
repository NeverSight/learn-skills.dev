Installieren, ermitteln, entfernen und aktualisieren Sie Agent-Skills mithilfe der Npx-Skills-CLI. Verwenden Sie diese Option, wenn Sie aufgefordert werden, einen Skill zu installieren, einen Skill aus einem Repo hinzuzufügen, nach Skills zu suchen oder zu suchen, installierte Skills aufzulisten, einen Skill zu entfernen oder zu deinstallieren, Skills zu aktualisieren oder nach Updates zu suchen. Wird ausgelöst bei: „X-Skill installieren“, „Y-Skill hinzufügen“, „Skills für Z finden“, „Welche Skills sind verfügbar“, „Skill entfernen“, „Meine Skills aktualisieren“, „Nach Skill-Updates suchen“, „Nach einem Skill suchen, der X ausführt“.
