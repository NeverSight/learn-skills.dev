Управляйте интерфейсом командной строки Codex и Claude Code в качестве фоновых помощников через tmux. Думайте о них как о своей команде — делегируйте свободно. Используйте, если хотите: получить второе мнение о своей работе; делегировать исследовательские или контекстно-зависимые задачи; параллельное распределение работы (сокращение карты между агентами); запустить одну и ту же инструкцию для многих элементов; исправлять проблемы параллельно с рабочими деревьями; или построить что-то комплексное (исследования → план → поэтапное внедрение → обзор). Срабатывает при: «запустить кодекс», «использовать Клод», «создать агента», «второе мнение», «делегировать», «разветвить», «просмотреть мою работу», «создать с помощью агентов».
