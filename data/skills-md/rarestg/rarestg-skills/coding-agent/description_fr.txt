Orchestrez Codex CLI et Claude Code en tant qu'assistants d'arrière-plan via tmux. Considérez-les comme votre équipe – déléguez librement. Utilisez-le lorsque vous souhaitez : obtenir un deuxième avis sur votre travail ; déléguer des tâches de recherche ou à forte intensité contextuelle ; répartir le travail en parallèle (réduire la carte entre les agents) ; exécuter la même instruction sur de nombreux éléments ; résoudre les problèmes en parallèle avec les arbres de travail ; ou construire quelque chose de bout en bout (recherche → plan → mise en œuvre progressive → examen). Se déclenche sur : "exécuter le codex", "utiliser Claude sur", "engendrer un agent", "deuxième avis", "déléguer", "distribuer", "réviser mon travail", "construire avec des agents".
