Orchestrieren Sie Codex CLI und Claude Code als Hintergrundassistenten über tmux. Betrachten Sie sie als Ihr Team – delegieren Sie frei. Verwenden Sie es, wenn Sie: eine zweite Meinung zu Ihrer Arbeit einholen möchten; Recherche oder kontextlastige Aufgaben delegieren; Arbeit parallel auffächern (Map-Reduction über Agenten hinweg); Führen Sie dieselbe Anweisung für viele Elemente aus. Behebung von Problemen parallel zu Arbeitsbäumen; oder etwas End-to-End aufbauen (Forschung → Plan → schrittweise Implementierung → Überprüfung). Wird ausgelöst bei: „Codex ausführen“, „Claude verwenden“, „Agenten erzeugen“, „zweite Meinung“, „Delegieren“, „Auffächern“, „Meine Arbeit überprüfen“, „Mit Agenten erstellen“.
