قم بتنسيق Codex CLI وClaude Code كمساعدين في الخلفية عبر tmux. فكر فيهم كفريقك – قم بالتفويض بحرية. استخدمه عندما تريد: الحصول على رأي ثانٍ في عملك؛ تفويض البحث أو المهام الثقيلة السياق؛ توزيع العمل بشكل متوازٍ (تقليل الخريطة عبر الوكلاء)؛ تشغيل نفس التعليمات عبر العديد من العناصر؛ إصلاح المشكلات بالتوازي مع أشجار العمل؛ أو بناء شيء شامل (بحث ← خطة ← تنفيذ مرحلي ← مراجعة). تشغيل: "تشغيل المخطوطة"، "استخدام كلود"، "إنشاء وكيل"، "الرأي الثاني"، "التفويض"، "الخروج"، "مراجعة عملي"، "البناء مع الوكلاء".
