Codex CLI と Claude Code を tmux 経由でバックグラウンド アシスタントとして調整します。彼らをあなたのチームと考えてください。自由に委任してください。次のような場合に使用してください。自分の仕事についてセカンドオピニオンを得る。研究やコンテキストが重要なタスクを委任します。ファンアウト作業は並行して行われます (エージェント間のマップリデュース)。多くの項目に対して同じ命令を実行します。ワークツリーと並行して問題を修正します。または、エンドツーエンドで何かを構築します (調査 → 計画 → 段階的実装 → レビュー)。トリガー: 「コーデックスを実行する」、「クロードを使用する」、「エージェントを生成する」、「セカンドオピニオン」、「デリゲート」、「ファンアウト」、「作業をレビューする」、「エージェントを使用して構築する」。
