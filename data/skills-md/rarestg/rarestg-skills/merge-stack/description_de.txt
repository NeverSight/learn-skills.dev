Führen Sie einen linearen Stapel von GitHub-PRs nacheinander in die Hauptversion zusammen. Verwenden Sie diese Option, wenn der Benutzer mehrere PRs hat, die eine Kette bilden (jeder zielt auf den vorherigen Zweig ab, wobei der unterste auf den Hauptzweig abzielt) und sie alle nacheinander in den Hauptzweig zusammenführen möchte. Wird bei Phrasen wie „Stack zusammenführen“, „meine PRs zusammenführen“, „gestapelten PRs zusammenführen“ ausgelöst oder wenn der Benutzer eine nummerierte Zweigserie (z. B. Feature-1, Feature-2, Feature-3) hat, die er zusammenführen möchte.
