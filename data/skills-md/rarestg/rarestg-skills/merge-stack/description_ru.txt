Объедините линейный стек PR GitHub в основной один за другим. Используйте, когда у пользователя есть несколько PR, которые образуют цепочку (каждый из которых нацелен на предыдущую ветку, а нижний — на основную) и он хочет последовательно объединить их все в основную. Запускается по таким фразам, как «объединить стек», «объединить мои PR», «объединить составные PR» или когда у пользователя есть пронумерованная серия ветвей (например, функция-1, функция-2, функция-3), которую он хочет объединить.
