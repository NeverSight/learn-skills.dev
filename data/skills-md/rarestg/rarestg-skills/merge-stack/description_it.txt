Unisci uno stack lineare di PR GitHub nel principale uno per uno. Da utilizzare quando l'utente ha più PR che formano una catena (ciascuno destinato al ramo precedente, con quello inferiore indirizzato al principale) e desidera unirli tutti nel principale in sequenza. Si attiva su frasi come "unisci lo stack", "unisci i miei PR", "unisci PR in pila" o quando l'utente ha una serie di rami numerati (ad esempio funzione-1, funzione-2, funzione-3) che desidera unire.
