Fusione una pila lineal de relaciones públicas de GitHub en la principal, una por una. Úselo cuando el usuario tiene varios PR que forman una cadena (cada uno dirigido a la rama anterior, con el de abajo dirigido a la rama principal) y desea fusionarlos todos en la rama principal de forma secuencial. Se activa con frases como "fusionar la pila", "fusionar mis relaciones públicas", "fusionar relaciones públicas apiladas" o cuando el usuario tiene una serie de ramas numeradas (por ejemplo, característica-1, característica-2, característica-3) que desea fusionar.
