Assistente per newapi (new-api), una piattaforma gateway AI unificata open source (https://github.com/QuantumNous/new-api). Da utilizzare quando l'utente chiede informazioni sulla nuova API, sulla gestione di modelli, gruppi, saldo o token oppure sulla copia sicura delle chiavi, applicandole ai file di configurazione o utilizzandole nei comandi senza esporre i segreti.
