タスク完了後にエージェントの振り返りと学習した教訓を実行、書き込み、生成、記録、保存、保持、および合成します。取得検証ループを使用して、セッションの品質を自己評価し、間違いを特定し、再利用可能なパターンを抽出し、永続的な LESSONS.md ファイルを更新します。自動トリガーのセットアップ: OpenCode プラグイン フック、Claude Code Stop フック、および Cline TaskComplete フックをインストールして、すべての Todo が完了したときにレトロが自動的に起動されるようにします。 session_read (OpenCode)、コンテキスト ウィンドウの再構築 (Claude Code、Cursor、Gemini CLI)、および圧縮パイプラインで動作します。タスクを終了するとき、セッションの終了時、反省、レビュー、報告書を書くように求められたとき、または自動レトロトリガーを設定するときに使用します。
