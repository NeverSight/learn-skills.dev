Ausführen, Schreiben, Generieren, Aufzeichnen, Speichern, Beibehalten und Synthetisieren von Agentenrückblicken und gewonnenen Erkenntnissen nach Abschluss von Aufgaben. Bewerten Sie die Sitzungsqualität selbst, identifizieren Sie Fehler, extrahieren Sie wiederverwendbare Muster und aktualisieren Sie persistente LESSONS.md-Dateien mithilfe der Retrieval-Verify-Schleife. Auto-Trigger-Setup: Installieren Sie OpenCode-Plugin-Hooks, Claude Code Stop-Hooks und Cline TaskComplete-Hooks, damit Retros automatisch ausgelöst werden, wenn alle Aufgaben erledigt sind. Funktioniert mit session_read (OpenCode), Kontextfenster-Rekonstruktion (Claude Code, Cursor, Gemini CLI) und Komprimierungspipelines. Verwenden Sie diese Funktion, wenn Sie eine Aufgabe erledigen, am Ende einer Sitzung, wenn Sie zum Nachdenken, Überprüfen, Nachbesprechen oder Verfassen einer Obduktion aufgefordert werden oder wenn Sie automatische Retro-Auslöser einrichten.
