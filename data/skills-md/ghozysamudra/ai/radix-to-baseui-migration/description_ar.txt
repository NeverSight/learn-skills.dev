قم بترحيل مكونات shadcn وبدائل Radix UI المخصصة إلى Base UI من خلال سير عمل يدوي منخفض المخاطر. يُستخدم عندما يطلب المستخدمون الانتقال من Radix UI إلى Base UI، أو استبدال أنماط فرعية، أو إصلاح أخطاء الترحيل في TypeScript، أو تحديث الدعائم/الأنواع المعاد تسميتها، أو تكييف موضع القائمة المنسدلة/المنبثقة/مربع الحوار مع أنماط محدد موضع واجهة المستخدم الأساسية.
