Migre componentes shadcn y primitivas personalizadas de Radix UI a Base UI con un flujo de trabajo manual de bajo riesgo. Úselo cuando los usuarios soliciten pasar de la interfaz de usuario de Radix a la interfaz de usuario base, reemplazar patrones asChild, corregir errores de migración de TypeScript, actualizar accesorios/tipos renombrados o adaptar el posicionamiento del menú desplegable/emergente/diálogo a los patrones del posicionador de la interfaz de usuario base.
