Routeur unifié pour 7 bases de connaissances canoniques en génie logiciel. Achemine les requêtes vers les compétences sous-jacentes appropriées : gof-patterns (23 modèles de conception), algorithmes clrs (40 structures de données), code propre (SOLID + pratiques), ddia (systèmes distribués), programmeur pragmatique (savoir-faire), ddd (modélisation de domaine), sicp (fondamentaux CS). S'active automatiquement pour les décisions d'architecture, la sélection de modèles, le choix d'algorithmes et la conception du système.
