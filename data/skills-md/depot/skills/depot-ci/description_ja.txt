Depot CI を構成および管理します。これは、ワークフローを完全に Depot 内で実行する GitHub Actions のドロップイン代替品です。 GitHub Actions ワークフローを Depot CI に移行する場合、「depot ci merge」を実行する場合、Depot CI のシークレットと変数を管理する場合、「depot ci run」を使用してワークフローを実行する場合、Depot CI の実行をデバッグする場合、ワークフローの互換性を確認する場合、または Depot CI の現在のベータ制限を理解する場合に使用します。ユーザーが .depot/ ディレクトリ、depot ci コマンドについて言及したり、GitHub でホストされるランナーを使用せずに Depot のインフラストラクチャで GitHub Actions ワークフローを実行することについて尋ねたりする場合にも使用します。注: Depot CI は現在ベータ版であり、利用可能性は限られています。
