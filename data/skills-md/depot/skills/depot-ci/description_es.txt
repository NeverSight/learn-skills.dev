Configura y administra Depot CI, un reemplazo directo de GitHub Actions que ejecuta flujos de trabajo completamente dentro de Depot. Úselo al migrar flujos de trabajo de GitHub Actions a Depot CI, ejecutar `depot ci migrar`, administrar secretos y variables de Depot CI, ejecutar flujos de trabajo con `depot ci run`, depurar ejecuciones de Depot CI, verificar la compatibilidad del flujo de trabajo o comprender las limitaciones beta actuales de Depot CI. Úselo también cuando el usuario mencione el directorio .depot/, los comandos depot ci o pregunte sobre la ejecución de flujos de trabajo de GitHub Actions en la infraestructura de Depot sin ejecutores alojados en GitHub. NOTA: Depot CI se encuentra actualmente en versión beta con disponibilidad limitada.
