Konfiguriert und verwaltet Depot CI, einen Drop-in-Ersatz für GitHub Actions, der Workflows vollständig innerhalb von Depot ausführt. Verwenden Sie diese Option, wenn Sie GitHub Actions-Workflows zu Depot CI migrieren, „depot ci migrate“ ausführen, Depot CI-Geheimnisse und -Variablen verwalten, Workflows mit „depot ci run“ ausführen, Depot CI-Ausführungen debuggen, Workflow-Kompatibilität überprüfen oder die aktuellen Beta-Einschränkungen von Depot CI verstehen. Verwenden Sie es auch, wenn der Benutzer das Verzeichnis .depot/ oder Depot-CI-Befehle erwähnt oder nach der Ausführung von GitHub Actions-Workflows auf der Depot-Infrastruktur ohne von GitHub gehostete Läufer fragt. HINWEIS: Depot CI befindet sich derzeit in der Betaversion mit begrenzter Verfügbarkeit.
