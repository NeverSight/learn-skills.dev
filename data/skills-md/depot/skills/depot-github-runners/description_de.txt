Konfiguriert Depot-verwaltete GitHub Actions-Läufer als Ersatz für von GitHub gehostete Läufer. Verwenden Sie diese Option beim Einrichten oder Migrieren von GitHub Actions-Workflows zur Verwendung von Depot-Runnern, beim Auswählen von Runner-Größen (CPU/RAM), beim Konfigurieren von Run-on-Bezeichnungen, beim Einrichten von ARM- oder Windows- oder macOS-Runnern, beim Beheben von Problemen mit GitHub Actions-Runnern, beim Konfigurieren der Ausgangsfilterung, bei der Verwendung von Depot Cache mit GitHub Actions oder beim Ausführen von Dagger/Dependabot auf Depot-Runnern. Wird auch verwendet, wenn der Benutzer die Läuferbezeichnungen depot-ubuntu, depot-windows, depot-macos erwähnt oder nach schnelleren/günstigeren GitHub Actions-Läufern fragt.
