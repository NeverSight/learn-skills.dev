Depot で管理される GitHub Actions ランナーを、GitHub でホストされるランナーのドロップイン代替として構成します。デポ ランナーを使用するための GitHub Actions ワークフローのセットアップまたは移行、ランナー サイズ (CPU/RAM) の選択、ランオン ラベルの構成、ARM または Windows または macOS ランナーのセットアップ、GitHub Actions ランナーの問題のトラブルシューティング、出力フィルタリングの構成、GitHub Actions でのデポ キャッシュの使用、またはデポ ランナーでの Dagger/Dependabot の実行を行うときに使用します。ユーザーが depot-ubuntu、depot-windows、depot-macos ランナー ラベルについて言及したり、より高速/安価な GitHub Actions ランナーについて質問したりする場合にも使用します。
