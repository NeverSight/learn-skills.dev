Configura los ejecutores de GitHub Actions administrados por Depot como un reemplazo directo para los ejecutores alojados en GitHub. Úselo al configurar o migrar flujos de trabajo de GitHub Actions para usar ejecutores de Depot, elegir tamaños de ejecutores (CPU/RAM), configurar etiquetas de ejecución, configurar ejecutores ARM o Windows o macOS, solucionar problemas de ejecutores de GitHub Actions, configurar filtrado de salida, usar Depot Cache con GitHub Actions o ejecutar Dagger/Dependabot en ejecutores de Depot. Úselo también cuando el usuario mencione etiquetas de ejecutor depot-ubuntu, depot-windows, depot-macos o pregunte sobre ejecutores de GitHub Actions más rápidos/baratos.
