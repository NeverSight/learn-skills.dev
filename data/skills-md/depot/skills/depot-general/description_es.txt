Maneja la instalación, autenticación, inicio de sesión, configuración de proyectos, administración de organizaciones y acceso a API de Depot CLI. Úselo al instalar la CLI de Depot, iniciar sesión con "inicio de sesión de depósito", crear o administrar proyectos de Depot, configurar tokens de API o relaciones de confianza OIDC, configurar depot.json, administrar organizaciones, restablecer cachés de compilación o usar las API/SDK de Depot. Utilícelo también cuando el usuario pregunte sobre los métodos de autenticación de Depot, los tipos de tokens, las variables de entorno o la configuración general de la plataforma de Depot que no es específica de las compilaciones de contenedores, los ejecutores de GitHub Actions o el CI de Depot.
