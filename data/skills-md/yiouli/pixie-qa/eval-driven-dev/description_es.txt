Instrumente aplicaciones Python LLM, cree conjuntos de datos valiosos, escriba pruebas basadas en evaluaciones, ejecútelas y solucione fallas raíz, cubriendo todo el ciclo de desarrollo impulsado por evaluaciones. Asegúrese de utilizar esta habilidad siempre que un usuario esté desarrollando, probando, controlando la calidad, evaluando o comparando un proyecto de Python que llame a un LLM, incluso si no dice "evaluaciones" explícitamente. Úselo para asegurarse de que una aplicación de IA funcione correctamente, detectar regresiones después de cambios rápidos, depurar por qué un agente comenzó a comportarse de manera diferente o validar la calidad de la salida antes del envío.
