Instrumentez les applications Python LLM, créez des ensembles de données fiables, rédigez des tests basés sur l'évaluation, exécutez-les et identifiez les causes profondes des échecs, couvrant ainsi le cycle complet de développement basé sur l'évaluation. Assurez-vous d'utiliser cette compétence chaque fois qu'un utilisateur développe, teste, contrôle qualité, évalue ou compare un projet Python qui appelle un LLM, même s'il ne dit pas explicitement « évaluations ». Utilisez-le pour vous assurer qu'une application d'IA fonctionne correctement, détecter les régressions après des modifications rapides, déboguer pourquoi un agent a commencé à se comporter différemment ou valider la qualité de la sortie avant l'expédition.
