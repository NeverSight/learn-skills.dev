README スタイルの出力 (`<picture>` + ライト/ダーク SVG) を使用して、mermint で Markdown Mermaid ブロックを処理します。
`--keep-mermaid` を使用してソース ブロックを保存し、生成された SVG アセットを安全に更新します。

このスキルは次の場合に使用します。
1. ユーザーは Markdown ファイル内の Mermaid コード ブロックを変換するように要求します
2. ユーザーは、このリポジトリのように README 図をレンダリング/更新するように要求します
3. ユーザーは `svgs/*-light.svg` および `svgs/*-dark.svg` アセットを再生成するように要求します
4. ユーザーは、保存された Mermaid ソースを使用して冪等マークダウンの再処理を実行するように要求します。
5. ユーザーは、AWS サービス アイコンを使用して Mermaid の「アーキテクチャ ベータ」図を作成または更新するように要求します。
