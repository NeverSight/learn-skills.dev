README 스타일 출력(`<picture>` + light/dark SVG)을 사용하여 mermint로 Markdown Mermaid 블록을 처리합니다.
`--keep-mermaid`를 사용하여 소스 블록을 보존하고 생성된 SVG 자산을 안전하게 새로 고칠 수 있습니다.

다음과 같은 경우에 이 기술을 사용하십시오.
1. 사용자가 Markdown 파일의 Mermaid 코드 블록을 변환하도록 요청합니다.
2. 사용자가 이 저장소와 같은 README 다이어그램을 렌더링/업데이트하도록 요청합니다.
3. 사용자가 `svgs/*-light.svg` 및 `svgs/*-dark.svg` 자산을 재생성하도록 요청합니다.
4. 사용자가 보존된 Mermaid 소스를 사용하여 멱등성 마크다운 재처리를 실행하도록 요청합니다.
5. 사용자가 AWS 서비스 아이콘을 사용하여 Mermaid '아키텍처-베타' 다이어그램을 생성하거나 업데이트하도록 요청합니다.
