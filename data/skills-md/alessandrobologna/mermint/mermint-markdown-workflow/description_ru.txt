Обрабатывать блоки Markdown Mermaid с помощью mermint, используя вывод в стиле README (`<picture>` + светлые/темные SVG),
сохраняйте исходные блоки с помощью `--keep-mermaid` и безопасно обновляйте сгенерированные ресурсы SVG.

Используйте этот навык, когда:
1. Пользователь просит конвертировать блоки кода Mermaid в файлы Markdown.
2. Пользователь просит отобразить/обновить диаграммы README, как в этом репозитории.
3. Пользователь просит восстановить ресурсы `svgs/*-light.svg` и `svgs/*-dark.svg`.
4. Пользователь просит запустить идемпотентную повторную обработку уценки с сохраненным исходным кодом Mermaid.
5. Пользователь просит создать или обновить диаграммы бета-версии архитектуры Mermaid с помощью значков сервисов AWS.
