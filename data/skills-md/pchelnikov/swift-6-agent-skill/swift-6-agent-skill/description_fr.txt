Générez, révisez ou refactorisez le code Swift à l'aide des fonctionnalités de langage moderne de Swift 6.0 à 6.2.3. À utiliser lors de l'écriture d'un nouveau code Swift, de la révision du code existant à la recherche de modèles obsolètes, de la migration de Swift 5.x vers 6.x, du débogage des avertissements de concurrence ou de la réponse à des questions sur les idiomes Swift actuels. Couvre la concurrence, les lancements typés, les types non copiables, l'observation, les tests, etc.
