Développement piloté par les spécifications vérifiées (VSDD) : une méthodologie d'ingénierie rigoureuse native de l'IA qui fusionne le développement piloté par les spécifications, le développement piloté par les tests et la vérification contradictoire en un seul pipeline. À utiliser lorsque l'utilisateur dit "vsdd", "basé sur les spécifications", "spécifications vérifiées", "examen contradictoire", "zéro pente", "vérification complète", "développement axé sur les spécifications", "construire ceci correctement avec les spécifications", ou souhaite un workflow rigoureux → test → implémentation → vérification. À utiliser également lorsque cela est demandé pour une vérification formelle, des flux de travail de tests de mutation ou une révision contradictoire du code dans le cadre d'un processus de développement.
