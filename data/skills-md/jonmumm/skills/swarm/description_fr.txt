Un workflow de développement d'IA multi-agents. Charge en amont les questions de planification (grill-me), puis lance des agents parallèles dans les arbres de travail Git : un agent de fonctionnalités construit à partir d'un backlog via TDD, tandis que les agents CRAP, Mutation et Acceptance renforcent continuellement la base de code à l'aide de métriques déterministes. À utiliser lorsque l'utilisateur dit « essaim », « exécuter l'essaim », « lancer un essaim » ou souhaite planifier puis exécuter une exécution multi-agent de longue durée.
