Escriba y refactorice componentes de React para que sean pequeños, componibles y personalizables, haciendo una cosa bien. Mejore el rendimiento de renderizado, la mantenibilidad y la reutilización aprovechando los componentes compuestos, la distribución de accesorios y la fusión de clases de servicios públicos. Aplique estos patrones al crear nuevos componentes o descomponer componentes monolíticos grandes.
