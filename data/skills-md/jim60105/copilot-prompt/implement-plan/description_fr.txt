Implémentez un plan de développement à partir de GitHub Issues avec un flux de travail DevOps complet comprenant le branchement, les tests et la création de demandes d'extraction. À utiliser lorsque l'utilisateur souhaite implémenter un problème GitHub, travailler sur un élément du backlog ou corriger un bogue à l'aide du flux de travail GitHub DevOps avec suivi des problèmes, gestion des succursales et soumission des relations publiques.
