Implementa un piano di sviluppo da GitHub Issues con un flusso di lavoro DevOps completo che include ramificazioni, test e creazione di richieste pull. Da utilizzare quando l'utente desidera implementare un problema GitHub, lavorare su un elemento del backlog o correggere un bug utilizzando il flusso di lavoro GitHub DevOps con monitoraggio dei problemi, gestione delle filiali e invio PR.
