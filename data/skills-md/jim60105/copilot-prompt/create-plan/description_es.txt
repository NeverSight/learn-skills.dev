Cree un plan de desarrollo detallado y problemas de GitHub para un proyecto. Úselo cuando el usuario desee planificar una nueva característica o solución, crear problemas de GitHub para trabajos pendientes o errores, o redactar un plan de desarrollo integral con desgloses de tareas, dependencias y estrategias de prueba. Los planes están escritos en chino tradicional con ejemplos de código en inglés.
