Erstellen Sie einen detaillierten Entwicklungsplan, der lokal in „.github/plans/“ gespeichert ist. Verwenden Sie diese Option, wenn der Benutzer eine neue Funktion planen, einen lokalen Rückstand oder Fehlerplan erstellen oder einen detaillierten Entwicklungsplan entwerfen möchte, ohne GitHub-Probleme zu verursachen. Die Pläne sind in traditionellem Chinesisch mit englischen Codebeispielen verfasst.
