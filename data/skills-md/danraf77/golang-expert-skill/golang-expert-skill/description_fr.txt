Documentation complète et spécifications du langage pour le langage de programmation Go. Utilisez cette compétence lorsqu'on vous demande d'écrire ou de déboguer du code Go, d'expliquer la syntaxe ou les structures de données Go, de comprendre les modèles de concurrence (goroutines, canaux), les modules et les configurations de l'espace de travail. Il contient une référence exhaustive sur tout ce qui concerne Go.
