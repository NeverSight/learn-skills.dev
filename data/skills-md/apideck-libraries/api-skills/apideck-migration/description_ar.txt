دليل للانتقال من عمليات تكامل واجهة برمجة التطبيقات المباشرة التابعة لجهات خارجية إلى واجهة برمجة التطبيقات الموحدة لـ Apidec. يُستخدم عندما يريد المستخدم استبدال عمليات التكامل الحالية لـ Salesforce أو HubSpot أو QuickBooks أو Xero أو غيرها من عمليات التكامل المباشرة لواجهة برمجة التطبيقات (APIdeck)، أو عند دمج عمليات التكامل المتعددة في طبقة موحدة واحدة.
