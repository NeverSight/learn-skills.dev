Шаблоны интеграции Apideck Unified API для TypeScript и Node.js. Используйте при интеграции с бухгалтерским программным обеспечением (QuickBooks, Xero, NetSuite), CRM (Salesforce, HubSpot, Pipedrive), платформами HRIS (Workday, BambooHR), хранилищами файлов (Google Drive, Dropbox, Box), системами ATS (Greenhouse, Lever), электронной коммерцией или любым из более чем 200 соединителей Apideck. Охватывает @apideck/unify SDK, аутентификацию, операции CRUD, разбиение на страницы, фильтрацию, веб-перехватчики и управление подключениями к Vault.
