Apideck Modelli di integrazione API unificati per TypeScript e Node.js. Da utilizzare quando si creano integrazioni con software di contabilità (QuickBooks, Xero, NetSuite), CRM (Salesforce, HubSpot, Pipedrive), piattaforme HRIS (Workday, BambooHR), archiviazione di file (Google Drive, Dropbox, Box), sistemi ATS (Greenhouse, Lever), e-commerce o uno qualsiasi degli oltre 200 connettori Apideck. Copre l'SDK @apideck/unify, l'autenticazione, le operazioni CRUD, l'impaginazione, il filtraggio, i webhook e la gestione delle connessioni di Vault.
