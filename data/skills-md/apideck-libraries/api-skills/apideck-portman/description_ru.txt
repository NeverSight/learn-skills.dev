Контрактное тестирование API с Portman от Apideck. Используйте при создании коллекций Postman на основе спецификаций OpenAPI, написании контрактных тестов, вариационных тестов, интеграционных тестов, нечеткого тестирования или настройке конвейеров тестирования API CI/CD. Портман преобразует спецификации OpenAPI 3.x в коллекции Postman с помощью автоматически создаваемых наборов тестов.
