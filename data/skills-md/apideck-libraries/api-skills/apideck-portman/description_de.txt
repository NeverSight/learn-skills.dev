API-Vertragstests mit Portman von Apideck. Verwenden Sie diese Option, wenn Sie Postman-Sammlungen aus OpenAPI-Spezifikationen generieren, Vertragstests, Variationstests, Integrationstests, Fuzz-Tests schreiben oder CI/CD-API-Testpipelines einrichten. Portman konvertiert OpenAPI 3.x-Spezifikationen in Postman-Sammlungen mit automatisch generierten Testsuiten.
