Test de contrat API avec Portman par Apideck. À utiliser lors de la génération de collections Postman à partir de spécifications OpenAPI, de l'écriture de tests contractuels, de tests de variations, de tests d'intégration, de tests fuzz ou de la configuration de pipelines de test d'API CI/CD. Portman convertit les spécifications OpenAPI 3.x en collections Postman avec des suites de tests générées automatiquement.
