اختبار عقد واجهة برمجة التطبيقات (API) مع Portman بواسطة Apidec. يُستخدم عند إنشاء مجموعات Postman من مواصفات OpenAPI، أو كتابة اختبارات العقود، أو اختبارات التباين، أو اختبارات التكامل، أو اختبار الزغب، أو إعداد مسارات اختبار CI/CD API. يقوم Portman بتحويل مواصفات OpenAPI 3.x إلى مجموعات Postman مع مجموعات اختبار يتم إنشاؤها تلقائيًا.
