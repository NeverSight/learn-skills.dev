適用於任何語言的 Apideck 統一 REST API 參考。在使用直接 HTTP 呼叫與會計軟體（QuickBooks、Xero、NetSuite）、CRM（Salesforce、HubSpot、Pipedrive）、HRIS 平台（Workday、BambooHR）、文件儲存（Google Drive、Dropbox、Box）、ATS 系統（Greenhouse、Lever）、電子商務或任何 Apideck 的 200 個連接器整合時使用多個連接器。涵蓋身份驗證標頭、CRUD 操作、基於遊標的分頁、過濾、排序、錯誤處理、速率限制、傳遞參數和 Webhooks。與語言無關－適用於curl、fetch、axios、httpx或任何HTTP客戶端。
