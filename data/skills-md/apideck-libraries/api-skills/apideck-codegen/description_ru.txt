Создавайте типизированные клиенты API на основе спецификаций Apideck OpenAPI с помощью генераторов кода. Используйте, когда пользователь хочет создать собственные клиенты SDK, типизированные модели, заглушки API или серверные каркасы на основе опубликованных спецификаций OpenAPI Apideck. Охватывает рабочие процессы импорта openapi-generator, Speakeasy и Postman.
