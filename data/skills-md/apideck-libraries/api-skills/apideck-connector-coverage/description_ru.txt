Прежде чем создавать интеграцию, проверьте покрытие API коннектора Apideck. Используйте при определении того, какие операции поддерживает соединитель, сравнении возможностей соединителя или диагностике сбоя вызова API с конкретным соединителем. Учит агентов запрашивать API соединителя для получения данных о покрытии в реальном времени.
