تحقق من تغطية واجهة برمجة تطبيقات موصل Apieck قبل إنشاء عمليات التكامل. يُستخدم عند تحديد العمليات التي يدعمها الموصل، أو مقارنة إمكانيات الموصل، أو تشخيص سبب فشل استدعاء واجهة برمجة التطبيقات (API) مع موصل معين. يعلم الوكلاء كيفية الاستعلام عن واجهة API للموصل للحصول على بيانات التغطية في الوقت الفعلي.
