C# および .NET 用の Apideck 統合 API 統合パターン。会計ソフトウェア (QuickBooks、Xero、NetSuite)、CRM (Salesforce、HubSpot、Pipedrive)、HRIS プラットフォーム (Workday、BambooHR)、ファイル ストレージ (Google Drive、Dropbox、Box)、ATS システム (Greenhouse、Lever)、電子商取引、または .NET を使用する Apideck の 200 以上のコネクタとの統合を構築する場合に使用します。 ApideckUnifySdk NuGet パッケージ、認証、CRUD 操作、ページネーション、エラー処理、および Vault 接続管理について説明します。
