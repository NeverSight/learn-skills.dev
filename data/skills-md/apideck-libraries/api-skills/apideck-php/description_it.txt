Apideck Modelli di integrazione API unificati per PHP. Da utilizzare quando si creano integrazioni con software di contabilità (QuickBooks, Xero, NetSuite), CRM (Salesforce, HubSpot, Pipedrive), piattaforme HRIS (Workday, BambooHR), archiviazione di file (Google Drive, Dropbox, Box), sistemi ATS (Greenhouse, Lever), e-commerce o qualsiasi degli oltre 200 connettori Apideck che utilizzano PHP. Copre il pacchetto apideck-libraries/sdk-php Composer, l'autenticazione, le operazioni CRUD, l'impaginazione, la gestione degli errori e la gestione della connessione Vault.
