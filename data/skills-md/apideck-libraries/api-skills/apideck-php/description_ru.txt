Шаблоны интеграции Apideck Unified API для PHP. Используйте при создании интеграции с бухгалтерским программным обеспечением (QuickBooks, Xero, NetSuite), CRM (Salesforce, HubSpot, Pipedrive), платформами HRIS (Workday, BambooHR), хранилищами файлов (Google Drive, Dropbox, Box), системами ATS (Greenhouse, Lever), электронной коммерцией или любым из более чем 200 соединителей Apideck с использованием PHP. Охватывает пакет apideck-libraries/sdk-php Composer, аутентификацию, операции CRUD, нумерацию страниц, обработку ошибок и управление подключениями к Vault.
