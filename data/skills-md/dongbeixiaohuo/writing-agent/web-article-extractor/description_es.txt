Utilice Chrome DevTools MCP para extraer y analizar el contenido de la página web. Úselo cuando los usuarios soliciten obtener contenido de una página web, leer artículos en línea, extraer texto de sitios web, capturar instantáneas de páginas web o analizar la estructura de una página web. Admite múltiples formatos de extracción, incluidos texto sin formato, HTML y contenido estructurado. Optimizado específicamente para sitios web con restricciones de seguridad, como las cuentas oficiales de WeChat.
