Comprehensive stock technical analysis tool, beginner-friendly. Uses hybrid data sources (Yahoo Finance + Alpha Vantage MCP), provides interactive HTML visual reports (Lightweight Charts candlesticks + technical overlays + swing labels + trendlines + S/R bands) and easy-to-understand analysis text, buy/sell suggestions, Excel position management. Supports Hong Kong stocks local calculation, ATR dynamic stop-loss, KDJ stochastic, MACD/RSI divergence detection, OBV volume analysis, Fibonacci support/resistance, candlestick pattern recognition (hammer/engulfing/doji), trend pattern recognition (double bottom/head-and-shoulders/triangle). Supports TradingView jump, Feishu multi-dimensional table sync. Activated when user requests stock analysis, technical indicators, trading advice, or position analysis.
