Patrones de interfaz de usuario reutilizables para aplicaciones de utilidad macOS/iOS creadas con SwiftUI y Liquid Glass. Incluye implementaciones completas para: paneles de acceso rápido estilo Spotlight con navegación por teclado, barra lateral NavigationSplitView + diseños de detalles, paneles de editor flotantes, paletas de cuadrícula compactas con transformación al pasar el cursor, formularios de configuración con secciones de vidrio, superposiciones de notificaciones/tostos, sistemas de marcadores de posición variables (detección, interfaz de usuario de entrada, resaltado de sintaxis, autocompletar), selectores de plantillas, copia enriquecida del portapapeles multiformato (normal + HTML + RTF) y vistas de diferencias. También cubre antipatrones (sobrecarga de vidrio, vidrio sobre fondos oscuros). Úselo al crear administradores de mensajes, utilidades de portapapeles, expansores de texto, lanzadores, herramientas de fragmentos o cualquier utilidad macOS controlada por teclado. Úselo junto con vidrio líquido para el sistema de diseño y appkit-bridge para NSPanel/NSStatusItem.
