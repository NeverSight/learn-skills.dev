Многоразовые шаблоны пользовательского интерфейса для служебных приложений macOS/iOS, созданные с помощью SwiftUI и Liquid Glass. Включает в себя полные реализации для: панелей быстрого доступа в стиле Spotlight с навигацией с помощью клавиатуры, боковой панели NavigationSplitView + макетов деталей, плавающих панелей редактора, компактных сеточных палитр с морфингом при наведении, форм настроек со стеклянными секциями, наложений всплывающих окон/уведомлений, систем переменных заполнителей (обнаружение, пользовательский интерфейс ввода, подсветка синтаксиса, автозаполнение), средств выбора шаблонов, многоформатного копирования в буфер обмена (обычный + HTML + RTF) и представлений различий. Также охватывает анти-паттерны (перегрузка стекла, стекло на темном фоне). Используйте при создании менеджеров подсказок, утилит буфера обмена, расширителей текста, средств запуска, инструментов фрагментов или любой утилиты macOS, управляемой с клавиатуры. Используйте вместе с жидким стеклом для системы дизайна и appkit-bridge для NSPanel/NSStatusItem.
