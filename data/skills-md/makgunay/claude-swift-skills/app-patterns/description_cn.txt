使用 SwiftUI 和 Liquid Glass 构建的 macOS/iOS 实用程序应用程序的可重用 UI 模式。包括完整的实现：带键盘导航的 Spotlight 式快速访问面板、NavigationSplitView 侧边栏 + 详细布局、浮动编辑器面板、带悬停变形的紧凑网格调色板、带玻璃部分的设置表单、Toast/通知覆盖、可变占位符系统（检测、输入 UI、语法突出显示、自动完成）、模板选择器、丰富的多格式剪贴板副本（普通 + HTML + RTF）和差异视图。还涵盖反模式（玻璃过载、深色背景上的玻璃）。在构建提示管理器、剪贴板实用程序、文本扩展器、启动器、代码片段工具或任何键盘驱动的 macOS 实用程序时使用。与液体玻璃一起用于设计系统，并与 NSPanel/NSStatusItem 的 appkit-bridge 一起使用。
