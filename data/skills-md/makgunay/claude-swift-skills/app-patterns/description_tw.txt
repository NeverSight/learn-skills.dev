使用 SwiftUI 和 Liquid Glass 构建的 macOS/iOS 实用程序应用程序的可重用 UI 模式。包括完整的實作：帶有鍵盤導航的 Spotlight 式快速存取面板、NavigationSplitView 側邊欄 + 詳細佈局、浮動編輯器面板、帶懸停變形的緊湊網格調色板、帶玻璃部分的設定表單、Toast/通知覆蓋、可變佔位符系統（檢測、編號 HTML + HTML + RTF）、功能完成模板、普通完成的副本：普通完成器格式。还涵盖反模式（玻璃过载、深色背景上的玻璃）。在建立提示管理器、剪貼簿實用程式、文字擴充器、啟動器、程式碼片段工具或任何鍵盤驅動的 macOS 實用程式時使用。与液体玻璃一起用于设计系统，并与 NSPanel/NSStatusItem 的 appkit-bridge 一起使用。
