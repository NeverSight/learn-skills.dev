SwiftUI と Liquid Glass で構築された macOS/iOS ユーティリティ アプリの再利用可能な UI パターン。キーボード ナビゲーションを備えた Spotlight スタイルのクイック アクセス パネル、NavigationSplitView サイドバー + 詳細レイアウト、フローティング エディター パネル、ホバー モーフィングを備えたコンパクトなグリッド パレット、ガラス セクションを備えた設定フォーム、トースト/通知オーバーレイ、可変プレースホルダー システム (検出、入力 UI、構文ハイライト、オートコンプリート)、テンプレート ピッカー、リッチ マルチフォーマット クリップボード コピー (プレーン + HTML + RTF)、および差分ビューの完全な実装が含まれています。アンチパターン (ガラスの過負荷、暗い背景上のガラス) もカバーします。プロンプト マネージャー、クリップボード ユーティリティ、テキスト エキスパンダー、ランチャー、スニペット ツール、またはキーボード駆動の macOS ユーティリティを構築するときに使用します。デザイン システムの液体ガラスと NSPanel/NSStatusItem の appkit-bridge と一緒に使用します。
