أنماط واجهة مستخدم قابلة لإعادة الاستخدام لتطبيقات macOS/iOS المساعدة المبنية باستخدام SwiftUI وLiquid Glass. يتضمن تطبيقات كاملة لـ: لوحات الوصول السريع بنمط Spotlight مع التنقل عبر لوحة المفاتيح، والشريط الجانبي NavigationSplitView + تخطيطات التفاصيل، ولوحات المحرر العائمة، ولوحات الشبكة المدمجة مع التحوير، ونماذج الإعدادات مع أقسام زجاجية، وتراكبات الخبز المحمص/الإشعارات، وأنظمة العناصر النائبة المتغيرة (الكشف، وواجهة مستخدم الإدخال، وتمييز بناء الجملة، والإكمال التلقائي)، ومنتقيات القوالب، ونسخة حافظة غنية متعددة التنسيق (عادي + HTML + RTF)، وطرق عرض مختلفة. يغطي أيضًا الأنماط المضادة (الحمل الزائد للزجاج، الزجاج على الخلفيات الداكنة). يُستخدم عند إنشاء مديري الطلبات، أو الأدوات المساعدة للحافظة، أو موسعات النص، أو المشغلات، أو أدوات القصاصات، أو أي أداة مساعدة لنظام التشغيل macOS تعتمد على لوحة المفاتيح. يُستخدم جنبًا إلى جنب مع الزجاج السائل لنظام التصميم وجسر التطبيقات لـ NSPanel/NSStatusItem.
