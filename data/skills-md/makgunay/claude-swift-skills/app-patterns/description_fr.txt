Modèles d'interface utilisateur réutilisables pour les applications utilitaires macOS/iOS créées avec SwiftUI et Liquid Glass. Comprend des implémentations complètes pour : panneaux d'accès rapide de style Spotlight avec navigation au clavier, barre latérale NavigationSplitView + dispositions détaillées, panneaux d'éditeur flottants, palettes de grille compactes avec morphing en survol, formulaires de paramètres avec sections en verre, superpositions de toast/notification, systèmes d'espaces réservés variables (détection, interface utilisateur de saisie, coloration syntaxique, saisie semi-automatique), sélecteurs de modèles, copie de presse-papiers multiformat riche (simple + HTML + RTF) et vues différentielles. Couvre également les anti-motifs (surcharge de verre, verre sur fonds sombres). À utiliser lors de la création de gestionnaires d'invites, d'utilitaires de presse-papiers, d'extenseurs de texte, de lanceurs, d'outils d'extraits de code ou de tout utilitaire macOS piloté par clavier. À utiliser avec le verre liquide pour le système de conception et l'appkit-bridge pour NSPanel/NSStatusItem.
