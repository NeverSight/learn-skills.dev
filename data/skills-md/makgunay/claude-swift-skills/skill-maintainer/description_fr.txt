Méta-compétence pour ingérer la documentation des développeurs Apple (transcriptions WWDC, documents API, guides de migration, notes de version) et l'utiliser pour créer, mettre à jour ou améliorer les compétences de développement Swift/macOS. Déclenchement lorsque l'utilisateur télécharge une nouvelle documentation Apple, demande de mettre à jour les compétences avec de nouvelles informations API, demande la création de compétences à partir de la documentation, dit « ingérer ceci », « mettre à jour les compétences avec ceci » ou fournit des fichiers markdown/texte contenant la documentation du framework Apple. Se déclenche également lorsque l'utilisateur demande à vérifier l'obsolescence des compétences existantes, à fusionner les compétences qui se chevauchent ou à vérifier les compétences par rapport aux nouvelles versions du système d'exploitation.
