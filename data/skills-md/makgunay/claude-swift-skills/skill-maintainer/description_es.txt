Metahabilidad para incorporar documentación para desarrolladores de Apple (transcripciones de la WWDC, documentos API, guías de migración, notas de la versión) y usarla para crear, actualizar o mejorar las habilidades de desarrollo de Swift/macOS. Se activa cuando el usuario carga nueva documentación de Apple, solicita actualizar habilidades con nueva información de API, solicita la creación de habilidades a partir de la documentación, dice "ingerir esto", "actualizar habilidades con esto" o proporciona archivos de texto/rebajas que contienen documentación del marco de Apple. También se activa cuando el usuario solicita auditar las habilidades existentes para detectar obsolescencias, fusionar habilidades superpuestas o comparar habilidades con nuevas versiones del sistema operativo.
