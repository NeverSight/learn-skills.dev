Apple 開発者ドキュメント (WWDC トランスクリプト、API ドキュメント、移行ガイド、リリース ノート) を取り込み、それを使用して Swift/macOS 開発スキルを作成、更新、または改善するためのメタスキル。ユーザーが新しい Apple ドキュメントをアップロードしたとき、新しい API 情報でスキルを更新するように要求したとき、ドキュメントからのスキルの作成を要求したとき、「これを取り込む」、「これでスキルを更新する」と指示したとき、または Apple フレームワークのドキュメントを含むマークダウン/テキスト ファイルを提供したときにトリガーされます。また、ユーザーが既存のスキルの古さの監査、重複するスキルのマージ、または新しい OS リリースに対するスキルのチェックを要求したときにもトリガーされます。
