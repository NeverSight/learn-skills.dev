Meta-competenza per l'acquisizione della documentazione per sviluppatori Apple (trascrizioni WWDC, documenti API, guide alla migrazione, note di rilascio) e il suo utilizzo per creare, aggiornare o migliorare le competenze di sviluppo Swift/macOS. Si attiva quando l'utente carica nuova documentazione Apple, chiede di aggiornare le competenze con nuove informazioni API, richiede la creazione di competenze dalla documentazione, dice "importa questo", "aggiorna competenze con questo" o fornisce file di markdown/testo contenenti la documentazione del framework Apple. Si attiva anche quando l'utente chiede di controllare se le competenze esistenti non sono obsolete, di unire competenze sovrapposte o di verificare le competenze rispetto alle nuove versioni del sistema operativo.
