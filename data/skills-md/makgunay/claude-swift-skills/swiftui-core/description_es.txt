Patrones principales de SwiftUI para el desarrollo de macOS e iOS, incluida la navegación (NavigationSplitView, NavigationStack), administración de estado (@State, @Binding, @Environment, @Bindable con @Observable), el nuevo sistema de barra de herramientas personalizable (ID de barra de herramientas, ToolbarSpacer, DefaultToolbarItem, searchToolbarBehavior, matchedTransitionSource,sharedBackgroundVisibility), edición de texto con estilo (TextEditor con AttributedString, AttributedTextSelection, transformAttributes, textFormattingDefinition) y patrones de diseño. Úselo al crear cualquier vista de SwiftUI, implementar la navegación, administrar el estado, crear barras de herramientas o crear editores de texto enriquecido. Corrige errores comunes de LLM, como el uso de NavigationView obsoleto, envoltorios de estado incorrectos y API de barra de herramientas obsoletas.
