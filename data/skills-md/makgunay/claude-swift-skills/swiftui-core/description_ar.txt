أنماط SwiftUI الأساسية لتطوير نظامي التشغيل macOS وiOS، بما في ذلك التنقل (NavigationSplitView وNavigationStack)، وإدارة الحالة (@State، @Binding، @Environment، @Bindable مع @Observable)، ونظام شريط الأدوات الجديد القابل للتخصيص (معرفات شريط الأدوات، ToolbarSpacer، DefaultToolbarItem، searchToolbarBehavior، matchedTransitionSource، SharedBackgroundVisibility)، وتحرير النص المصمم. (TextEditor مع AttributedString، وAttributedTextSelection، وtransformAttributes، وtextFormattingDefinition)، وأنماط التخطيط. يُستخدم عند إنشاء أي عرض SwiftUI، أو تنفيذ التنقل، أو إدارة الحالة، أو إنشاء أشرطة الأدوات، أو إنشاء برامج تحرير نص منسقة. يصحح أخطاء LLM الشائعة مثل استخدام NavigationView المهمل وأغلفة الحالة الخاطئة وواجهات برمجة تطبيقات شريط الأدوات القديمة.
