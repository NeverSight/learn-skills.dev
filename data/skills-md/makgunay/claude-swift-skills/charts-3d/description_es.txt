Visualización 3D de Swift Charts con contenedor Chart3D, SurfacePlot para representación de superficies 3D a partir de funciones o datos, Chart3DPose para control de ángulo de visualización (posturas predefinidas y acimut/inclinación personalizados), Chart3DSurfaceStyle para coloración de degradado, rotación interactiva mediante enlace @State, personalización de ejes (etiquetas, líneas de cuadrícula, rangos de valores) y representación volumétrica de visionOS. Úselo al crear visualización de datos con superficies 3D, gráficos científicos o experiencias de gráficos inmersivo.
