3D-визуализация Swift Charts с контейнером Chart3D, SurfacePlot для 3D-рендеринга поверхностей на основе функций или данных, Chart3DPose для управления углом просмотра (предопределенные позы и настраиваемый азимут/наклон), Chart3DSurfaceStyle для градиентной окраски, интерактивного вращения с помощью привязки @State, настройки осей (метки, линии сетки, диапазоны значений) и объемного рендеринга VisionOS. Используйте при создании визуализации данных с помощью трехмерных поверхностей, научных графиков или иммерсивных диаграмм.
