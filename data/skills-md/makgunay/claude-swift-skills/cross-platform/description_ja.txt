SwiftUI アプリで macOS と iOS の間でコードを共有するためのパターン。プロジェクト構造 (共有 70% / macOS 15% / iOS 15%)、プロトコルおよび #if os() 条件付きコンパイルによるプラットフォームの抽象化、アダプティブ ナビゲーション (Mac/iPad の NavigationSplitView → iPhone の NavigationStack)、プラットフォーム スタイルを備えた共有コンポーネント、iOS 固有の拡張機能 (カスタム キーボード拡張機能、インタラクティブ ウィジェット、共有拡張機能、アクション拡張機能、コントロール センター ウィジェット、ロック画面ウィジェット)、拡張機能とのデータ共有のためのアプリ グループをカバーします。 CloudKit 同期モニタリング、JSON エクスポート/インポート、スキーマのバージョン管理と移行、URL スキームのディープ リンク、完全な macOS→iOS 移行チェックリスト。 macOS と iOS の両方をターゲットとするアプリを構築する場合、macOS アプリに iOS サポートを追加する場合、ウィジェットやキーボード拡張機能を構築する場合、または SwiftData との iCloud 同期を設定する場合に使用します。
