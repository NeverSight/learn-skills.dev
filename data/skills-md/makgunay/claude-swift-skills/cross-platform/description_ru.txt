Шаблоны для совместного использования кода между macOS и iOS в приложениях SwiftUI. Охватывает структуру проекта (70 % общего доступа / 15 % macOS / 15 % iOS), абстракцию платформы с помощью протоколов и условную компиляцию #if os(), адаптивную навигацию (NavigationSplitView на Mac/iPad → NavigationStack на iPhone), общие компоненты со стилем платформы, расширения для iOS (настраиваемое расширение клавиатуры, интерактивные виджеты, расширение общего доступа, расширение действий, виджет Центра управления, виджет экрана блокировки), группы приложений для обмена данными с расширениями, мониторинг синхронизации CloudKit, Экспорт/импорт JSON, управление версиями и миграцией схем, глубокие ссылки на схемы URL-адресов и полный контрольный список миграции с macOS на iOS. Используйте при создании приложений, предназначенных как для macOS, так и для iOS, при добавлении поддержки iOS в приложение macOS, при создании виджетов или расширений клавиатуры или при настройке синхронизации iCloud со SwiftData.
