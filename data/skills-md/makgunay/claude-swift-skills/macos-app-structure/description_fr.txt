Modèles d'architecture d'application macOS couvrant le protocole d'application (@main), les types de scènes (WindowGroup, Window, Settings, MenuBarExtra), la gestion multi-fenêtres, NSApplicationDelegateAdaptor pour les hooks de cycle de vie AppKit, la configuration Info.plist (LSUIElement pour les applications de barre de menus, NSAccessibilityUsageDescription), les droits pour le sandbox/runtime renforcé et les conventions de structure de projet. À utiliser pour créer une nouvelle application macOS, configurer des scènes et des fenêtres, configurer des applications de barre de menus ou résoudre des problèmes de cycle de vie spécifiques à macOS. Corrige l'erreur courante du LLM consistant à générer des structures d'applications iOS uniquement.
