Шаблоны архитектуры приложений macOS, охватывающие протокол приложения (@main), типы сцен (WindowGroup, Window, Settings, MenuBarExtra), управление несколькими окнами, NSApplicationDelegateAdaptor для перехватчиков жизненного цикла AppKit, конфигурацию Info.plist (LSUIElement для приложений строки меню, NSAccessibilityUsageDescription), права на изолированную среду/защищенную среду выполнения и соглашения о структуре проекта. Используйте при создании нового приложения macOS, настройке сцен и окон, настройке приложений строки меню или решении проблем жизненного цикла, специфичных для macOS. Исправляет распространенную ошибку LLM, связанную с созданием структур приложений только для iOS.
