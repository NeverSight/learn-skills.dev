Patrones de arquitectura de aplicaciones macOS que cubren el protocolo de la aplicación (@main), tipos de escenas (WindowGroup, Window, Settings, MenuBarExtra), administración de múltiples ventanas, NSApplicationDelegateAdaptor para enlaces de ciclo de vida de AppKit, configuración de Info.plist (LSUIElement para aplicaciones de barra de menú, NSAccessibilityUsageDescription), derechos para sandbox/tiempo de ejecución reforzado y convenciones de estructura de proyecto. Úselo al crear una nueva aplicación de macOS, configurar escenas y ventanas, configurar aplicaciones de la barra de menús o resolver problemas del ciclo de vida específicos de macOS. Corrige el error común de LLM de generar estructuras de aplicaciones solo para iOS.
