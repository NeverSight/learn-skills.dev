Integrando componentes de AppKit en aplicaciones SwiftUI macOS. Cubre los protocolos NSViewRepresentable y NSViewControllerRepresentable para alojar vistas de AppKit en SwiftUI, NSHostingView/NSHostingController para alojar SwiftUI en AppKit, NSPanel para ventanas flotantes, configuración de NSWindow (styleMask, nivel, collectionBehavior), integración de cadena de respuesta, monitoreo de NSEvent (global y local), NSAnimationContext para animaciones de AppKit, NSPopover, NSStatusItem para barra de menú y NSGlassEffectView para vidrio líquido AppKit. Úselo cuando SwiftUI carezca de un equivalente nativo, creando paneles flotantes, ventanas cromadas personalizadas o integrando componentes heredados de AppKit.
