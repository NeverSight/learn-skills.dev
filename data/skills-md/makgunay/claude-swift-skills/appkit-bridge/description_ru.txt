Объединение компонентов AppKit с приложениями SwiftUI для macOS. Охватывает протоколы NSViewRepresentable и NSViewControllerRepresentable для размещения представлений AppKit в SwiftUI, NSHostingView/NSHostingController для размещения SwiftUI в AppKit, NSPanel для плавающих окон, конфигурацию NSWindow (styleMask, level, CollectionBehavior), интеграцию цепочки ответов, мониторинг NSEvent (глобальный и локальный), NSAnimationContext для анимации AppKit, NSPopover, NSStatusItem для строки меню и NSGlassEffectView для AppKit Liquid Glass. Используйте, когда SwiftUI не имеет собственного эквивалента, для создания плавающих панелей, пользовательского хромирования окон или интеграции устаревших компонентов AppKit.
