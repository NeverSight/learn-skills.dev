أنماط إطار عمل SwiftData المستمر لتطبيقات macOS وiOS. يغطي تعريفات @Model، @Query، والعلاقات (مع قواعد الحذف)، وتكوين ModelContainer/ModelContext، وراثة الفئة مع فئات @Model الفرعية، والمسندات المستندة إلى النوع (#Predicate مع is/as؟ casting)، والعلاقات متعددة الأشكال، وخيارات @Attribute (.preserveValueOnDeletion)، وترحيلات المخطط، والاستيراد/التصدير. يُستخدم عند تنفيذ ثبات البيانات، أو تحديد النماذج، أو الاستعلام عن البيانات، أو العمل مع SwiftData. يمنع أخطاء LLM الشائعة مثل إنشاء بنية البيانات الأساسية، أو اختراع واجهات برمجة تطبيقات SwiftData غير الموجودة، أو إساءة استخدام @Query.
