macOS 및 iOS 앱용 SwiftData 지속성 프레임워크 패턴. @Model 정의, @Query, 관계(삭제 규칙 포함), ModelContainer/ModelContext 구성, @Model 하위 클래스를 사용한 클래스 상속, 유형 기반 조건자(is/as? 캐스팅을 사용한 #Predicate), 다형성 관계, @Attribute 옵션(.preserveValueOnDeletion), 스키마 마이그레이션 및 가져오기/내보내기를 다룹니다. 데이터 지속성 구현, 모델 정의, 데이터 쿼리 또는 SwiftData 작업 시 사용합니다. 핵심 데이터 구문 생성, 존재하지 않는 SwiftData API 개발 또는 @Query 오용과 같은 일반적인 LLM 오류를 방지합니다.
