Patrones de marco de persistencia de SwiftData para aplicaciones macOS e iOS. Cubre definiciones de @Model, @Query, relaciones (con reglas de eliminación), configuración de ModelContainer/ModelContext, herencia de clases con subclases de @Model, predicados basados ​​en tipos (#Predicate con is/as? casting), relaciones polimórficas, opciones de @Attribute (.preserveValueOnDeletion), migraciones de esquemas e importación/exportación. Úselo al implementar la persistencia de datos, definir modelos, consultar datos o trabajar con SwiftData. Previene errores comunes de LLM, como generar la sintaxis de Core Data, inventar API SwiftData inexistentes o hacer un mal uso de @Query.
