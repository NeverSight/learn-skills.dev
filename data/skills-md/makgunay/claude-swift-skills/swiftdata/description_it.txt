Modelli del framework di persistenza SwiftData per app macOS e iOS. Copre le definizioni @Model, @Query, relazioni (con regole di eliminazione), configurazione ModelContainer/ModelContext, ereditarietà delle classi con sottoclassi @Model, predicati basati sul tipo (#Predicate con cast is/as?), relazioni polimorfiche, opzioni @Attribute (.preserveValueOnDeletion), migrazioni di schemi e importazione/esportazione. Da utilizzare quando si implementa la persistenza dei dati, si definiscono modelli, si eseguono query sui dati o si lavora con SwiftData. Previene gli errori LLM comuni come la generazione della sintassi Core Data, l'invenzione di API SwiftData inesistenti o l'uso improprio di @Query.
