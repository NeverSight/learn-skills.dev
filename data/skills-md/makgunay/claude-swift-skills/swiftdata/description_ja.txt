macOS および iOS アプリ用の SwiftData 永続フレームワーク パターン。 @Model 定義、@Query、リレーションシップ (削除ルールを使用)、ModelContainer/ModelContext 構成、@Model サブクラスによるクラス継承、タイプベースの述語 (is/as? キャストを使用した #Predicate)、ポリモーフィックなリレーションシップ、@Attribute オプション (.preserveValueOnDeletion)、スキーマの移行、インポート/エクスポートについて説明します。データ永続性の実装、モデルの定義、データのクエリ、または SwiftData の操作を行うときに使用します。コアデータ構文の生成、存在しない SwiftData API の作成、@Query の誤用などの一般的な LLM エラーを防止します。
