Modèles de framework de persistance SwiftData pour les applications macOS et iOS. Couvre les définitions @Model, @Query, les relations (avec règles de suppression), la configuration ModelContainer/ModelContext, l'héritage de classe avec les sous-classes @Model, les prédicats basés sur le type (#Predicate avec is/as? casting), les relations polymorphes, les options @Attribute (.preserveValueOnDeletion), les migrations de schéma et l'importation/exportation. À utiliser lors de la mise en œuvre de la persistance des données, de la définition de modèles, de l'interrogation de données ou de l'utilisation de SwiftData. Empêche les erreurs LLM courantes telles que la génération de la syntaxe Core Data, l'invention d'API SwiftData inexistantes ou l'utilisation abusive de @Query.
