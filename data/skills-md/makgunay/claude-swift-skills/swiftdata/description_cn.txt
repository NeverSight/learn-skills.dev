适用于 macOS 和 iOS 应用程序的 SwiftData 持久性框架模式。涵盖 @Model 定义、@Query、关系（带有删除规则）、ModelContainer/ModelContext 配置、带有 @Model 子类的类继承、基于类型的谓词（#Predicate 带有 is/as? 转换）、多态关系、@Attribute 选项 (.preserveValueOnDeletion)、模式迁移和导入/导出。在实现数据持久性、定义模型、查询数据或使用 SwiftData 时使用。防止常见的 LLM 错误，例如生成 Core Data 语法、发明不存在的 SwiftData API 或滥用 @Query。
