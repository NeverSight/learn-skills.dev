Modèles de langage Swift 6.2 et corrections critiques pour la génération de code LLM. Couvre le changement de paradigme de la concurrence (isolation MainActor par défaut, @concurrent pour le travail en arrière-plan, conformités isolées, désinscription non isolée), @Observable vs ObservableObject, la concurrence structurée, la gestion moderne des erreurs, les wrappers de propriétés, InlineArray, Span et les génériques de valeur. À utiliser chaque fois que vous générez du code Swift, en particulier du code asynchrone/simultané, une conception de classe/structure ou des chemins critiques en termes de performances. Cette compétence évite les erreurs LLM les plus courantes dans la génération de code Swift.
