Языковые шаблоны Swift 6.2 и критические исправления для генерации кода LLM. Охватывает смену парадигмы параллелизма (изоляция MainActor по умолчанию, @concurrent для фоновой работы, изолированные соответствия, неизолированный отказ), @Observable и ObservableObject, структурированный параллелизм, современная обработка ошибок, оболочки свойств, InlineArray, Span и универсальные значения. Используйте при создании кода Swift, особенно асинхронного/параллельного кода, проектирования классов/структур или путей, критически важных для производительности. Этот навык предотвращает наиболее распространенные ошибки LLM при генерации кода Swift.
