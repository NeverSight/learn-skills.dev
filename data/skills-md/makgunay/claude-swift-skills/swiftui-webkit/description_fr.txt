Intégration native de SwiftUI WebKit avec la nouvelle structure WebView et la classe observable WebPage. Couvre la création de WebView à partir d'URL, la page Web pour le contrôle de navigation et la gestion de l'état, l'exécution de JavaScript (callJavaScript avec des arguments et des mondes de contenu), les gestionnaires de schémas d'URL personnalisés, la gestion de la navigation (chargement, rechargement, arrière/suivant), les décisions de navigation, la recherche de texte (findNavigator), la capture de contenu (instantanés, génération de PDF, archives Web) et la configuration (magasins de données, agents utilisateurs, autorisations JS). À utiliser lors de l'intégration de contenu Web dans des applications SwiftUI au lieu de l'ancien modèle de pont WKWebView + UIViewRepresentable/NSViewRepresentable. Il s'agit d'une toute nouvelle API — n'utilisez PAS l'ancienne approche d'encapsulation WKWebView.
