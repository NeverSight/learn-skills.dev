ネイティブ SwiftUI WebKit と新しい WebView 構造体および WebPage 監視可能クラスとの統合。 URL からの WebView の作成、ナビゲーション制御と状態管理のための WebPage、JavaScript の実行 (引数とコンテンツ ワールドを指定した callJavaScript)、カスタム URL スキーム ハンドラー、ナビゲーション管理 (ロード、リロード、戻る/進む)、ナビゲーションの決定、テキスト検索 (findNavigator)、コンテンツ キャプチャ (スナップショット、PDF 生成、Web アーカイブ)、構成 (データ ストア、ユーザー エージェント、JS 権限) をカバーします。古い W​​KWebView + UIViewRepresentable/NSViewRepresentable ブリッジ パターンの代わりに、SwiftUI アプリに Web コンテンツを埋め込むときに使用します。これはまったく新しい API です。古い WKWebView ラッピング アプローチは使用しないでください。
