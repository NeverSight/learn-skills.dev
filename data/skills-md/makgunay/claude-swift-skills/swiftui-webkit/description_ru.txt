Встроенная интеграция SwiftUI WebKit с новой структурой WebView и наблюдаемым классом WebPage. Охватывает создание WebView из URL-адресов, WebPage для управления навигацией и управления состоянием, выполнение JavaScript (вызов JavaScript с аргументами и мирами контента), настраиваемые обработчики схем URL-адресов, управление навигацией (загрузка, перезагрузка, назад/вперед), решения по навигации, текстовый поиск (findNavigator), захват контента (снимки, создание PDF-файлов, веб-архивы) и настройку (хранилища данных, пользовательские агенты, разрешения JS). Используйте при встраивании веб-контента в приложения SwiftUI вместо старого шаблона моста WKWebView + UIViewRepresentable/NSViewRepresentable. Это совершенно новый API — НЕ используйте старый подход к упаковке WKWebView.
