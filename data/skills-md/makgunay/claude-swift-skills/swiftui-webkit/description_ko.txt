새로운 WebView 구조체 및 WebPage 관찰 가능 클래스와 기본 SwiftUI WebKit 통합. URL에서 WebView 생성, 탐색 제어 및 상태 관리를 위한 웹 페이지, JavaScript 실행(인수 및 콘텐츠 세계가 포함된 callJavaScript), 사용자 정의 URL 체계 처리기, 탐색 관리(로드, 다시 로드, 뒤로/앞으로), 탐색 결정, 텍스트 검색(findNavigator), 콘텐츠 캡처(스냅샷, PDF 생성, 웹 아카이브) 및 구성(데이터 저장소, 사용자 에이전트, JS 권한)을 다룹니다. 이전 WKWebView + UIViewRepresentable/NSViewRepresentable 브리지 패턴 대신 SwiftUI 앱에 웹 콘텐츠를 삽입할 때 사용합니다. 이것은 완전히 새로운 API입니다. 기존 WKWebView 래핑 접근 방식을 사용하지 마세요.
