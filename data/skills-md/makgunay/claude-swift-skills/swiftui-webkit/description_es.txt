Integración nativa de SwiftUI WebKit con la nueva estructura WebView y la clase observable WebPage. Cubre la creación de WebView a partir de URL, página web para control de navegación y administración de estado, ejecución de JavaScript (callJavaScript con argumentos y mundos de contenido), controladores de esquemas de URL personalizados, administración de navegación (carga, recarga, retroceso/adelante), decisiones de navegación, búsqueda de texto (findNavigator), captura de contenido (instantáneas, generación de PDF, archivos web) y configuración (almacenes de datos, agentes de usuario, permisos JS). Úselo al incrustar contenido web en aplicaciones SwiftUI en lugar del antiguo patrón de puente WKWebView + UIViewRepresentable/NSViewRepresentable. Esta es una API completamente nueva: NO utilice el antiguo enfoque de ajuste WKWebView.
