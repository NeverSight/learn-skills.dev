Das FoundationModels-Framework von Apple für die LLM-Integration auf dem Gerät in Apps. Behandelt die SystemLanguageModel-Verfügbarkeitsprüfung, die Erstellung von LanguageModelSessions mit Anweisungen, das Prompt-Engineering, das @Generable-Makro für strukturierte Ausgabe mit @Guide-Einschränkungen, das Snapshot-Streaming mit PartiallyGenerated-Typen, das Tool-Protokoll für benutzerdefinierte Tool-Aufrufe, GenerationOptions (Temperatur), Kontextfenstergrenzen (4096 Token) und Fehlerbehandlung. Verwenden Sie es, wenn Sie die Textgenerierung von Apple Intelligence integrieren, KI-gestützte Funktionen mithilfe von Modellen auf dem Gerät erstellen oder die Generierung strukturierter Daten implementieren. Wichtiger Hinweis: Verwenden Sie immer „response.content“ (NICHT „response.output“), wenn Sie auf generierte Werte zugreifen.
