クリップボード シミュレーション、CGEvent キーストローク タイピング、および Accessibility API の直接テキスト フィールド アクセスを介して、他の macOS アプリケーションにテキストを挿入します。クリップボードの保存/復元を伴う NSPasteboard の読み取り/書き込み、CGEvent キーストローク シミュレーション (Cmd+V ペースト、文字ごとの入力)、直接テキスト フィールド操作のための AXUIElement、変数/プレースホルダー展開 ({{clipboard}}、{{date}}、{{cursor}})、ユーザー プロンプト変数 ({{?name:prompt:default}})、挿入後のカーソル位置、挿入メソッドのフォールバック チェーンをカバーします。プロンプト マネージャー、テキスト エキスパンダー、スニペット ツール、または他のアプリケーションにテキストを挿入する必要があるアプリを構築するときに使用します。
