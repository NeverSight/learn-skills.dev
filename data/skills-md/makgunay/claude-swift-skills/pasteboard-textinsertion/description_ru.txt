Вставка текста в другие приложения macOS с помощью моделирования буфера обмена, набора текста с помощью клавиш CGEvent и прямого доступа к текстовым полям через API специальных возможностей. Охватывает чтение/запись NSPasteboard с сохранением/восстановлением буфера обмена, симуляцию нажатия клавиш CGEvent (вставка Cmd+V, посимвольный ввод), AXUIElement для прямого манипулирования текстовыми полями, расширение переменных/заполнителей ({{clipboard}}, {{date}}, {{cursor}}), запрашиваемые пользователем переменные ({{?name:prompt:default}}), позиционирование курсора после вставки и резервные цепочки метода вставки. Используйте при создании менеджеров подсказок, расширителей текста, инструментов фрагментов или любого приложения, которому необходимо вставлять текст в другие приложения.
