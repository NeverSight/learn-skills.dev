قم بإعداد شجرة عمل git جديدة للتطوير الموازي. يقوم بإنشاء شجرة العمل، ونسخ ملفات .env، وتثبيت التبعيات، وإنشاء عميل Prisma، وبدء تشغيل التطبيق اختياريًا (مع حل تعارض المنافذ) أو تشغيل الاختبارات. يتم تشغيله عندما يطلب المستخدم إعداد شجرة عمل، أو العمل على فرع بمعزل عن الآخر، أو يحتاج إلى بيئة منفصلة لفرع أو علاقات عامة.
