Richten Sie einen neuen Git-Arbeitsbaum für die parallele Entwicklung ein. Erstellt den Arbeitsbaum, kopiert .env-Dateien, installiert Abhängigkeiten, generiert den Prisma-Client und startet optional die App (mit Portkonfliktlösung) oder führt Tests aus. TRIGGER, wenn der Benutzer einen Arbeitsbaum einrichten möchte, isoliert an einem Zweig arbeiten möchte oder eine separate Umgebung für einen Zweig oder PR benötigt.
