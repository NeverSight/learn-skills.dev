Configure un nuevo árbol de trabajo de git para el desarrollo paralelo. Crea el árbol de trabajo, copia archivos .env, instala dependencias, genera el cliente Prisma y, opcionalmente, inicia la aplicación (con resolución de conflictos de puertos) o ejecuta pruebas. DISPARA cuando el usuario solicita configurar un árbol de trabajo, trabajar en una rama de forma aislada o necesita un entorno separado para una rama o PR.
