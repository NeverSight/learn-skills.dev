Руководство по использованию Polymarket CLI — инструмента командной строки на основе Rust для просмотра рынков прогнозов, размещения сделок, управления кошельками и взаимодействия с ончейн-контрактами на Polymarket. Используйте этот навык всякий раз, когда пользователь упоминает Polymarket, CLI рынков прогнозирования, Polymarket-cli, торговлю с терминала, рыночные шансы, книгу заказов CLOB, условные токены (CTF) или хочет просматривать рынки прогнозирования и торговать программными средствами. Также срабатывает, когда пользователь хочет создать сценарий на основе данных Polymarket, проверить рыночные цены, разместить лимитные или рыночные ордера, управлять позициями или взаимодействовать с API Polymarket из терминала или автоматизированного рабочего процесса.
