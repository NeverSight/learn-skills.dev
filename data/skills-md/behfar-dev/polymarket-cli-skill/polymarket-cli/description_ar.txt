دليل استخدام Polymarket CLI - أداة سطر أوامر قائمة على Rust لتصفح أسواق التنبؤ، وإجراء الصفقات، وإدارة المحافظ، والتفاعل مع العقود على السلسلة في Polymarket. استخدم هذه المهارة عندما يذكر المستخدم Polymarket، أو CLI لأسواق التنبؤ، أو polymarket-cli، أو التداول من المحطة، أو احتمالات السوق، أو دفتر طلبات CLOB، أو الرموز المميزة المشروطة (CTF)، أو يريد تصفح/تداول أسواق التنبؤ برمجيًا. يتم التشغيل أيضًا عندما يريد المستخدم كتابة نص برمجي مقابل بيانات Polymarket، أو التحقق من أسعار السوق، أو وضع أوامر الحد أو السوق، أو إدارة المراكز، أو التفاعل مع واجهة برمجة التطبيقات الخاصة بـ Polymarket من محطة طرفية أو سير عمل آلي.
