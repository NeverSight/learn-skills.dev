Leitfaden zur Verwendung der Polymarket-CLI – einem Rust-basierten Befehlszeilentool zum Durchsuchen von Prognosemärkten, zum Platzieren von Trades, zum Verwalten von Wallets und zum Interagieren mit On-Chain-Verträgen auf Polymarket. Verwenden Sie diesen Skill immer dann, wenn der Benutzer Polymarket, Prognosemärkte CLI, Polymarket-CLI, Handel vom Terminal, Marktquoten, CLOB-Orderbuch, bedingte Token (CTF) erwähnt oder Prognosemärkte programmgesteuert durchsuchen/handeln möchte. Wird auch ausgelöst, wenn der Benutzer Skripts mit Polymarket-Daten erstellen, Marktpreise überprüfen, Limit- oder Marktaufträge erteilen, Positionen verwalten oder über ein Terminal oder einen automatisierten Workflow mit der API von Polymarket interagieren möchte.
