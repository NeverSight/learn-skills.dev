Guía para usar Polymarket CLI: una herramienta de línea de comandos basada en Rust para explorar mercados de predicción, realizar operaciones, administrar billeteras e interactuar con contratos en cadena en Polymarket. Utilice esta habilidad siempre que el usuario mencione Polymarket, mercados de predicción CLI, polymarket-cli, operaciones desde terminal, probabilidades de mercado, libro de órdenes CLOB, tokens condicionales (CTF) o desee explorar/negociar mercados de predicción mediante programación. También se activa cuando el usuario desea realizar secuencias de comandos con datos de Polymarket, verificar precios de mercado, realizar órdenes limitadas o de mercado, administrar posiciones o interactuar con la API de Polymarket desde una terminal o un flujo de trabajo automatizado.
