グローバル指標、恐怖/貪欲、指数、トレンドトピック、チャートを含む、CoinMarketCap市場全体のエンドポイントのAPIリファレンス。
ユーザーが市場 API について言及する場合、恐怖/貪欲指数について質問する場合、グローバル指標または BTC ドミナンス データが必要な場合、k 折れ線グラフが必要な場合、または市場センチメントを扱う場合には、このスキルを使用してください。これは、CMC 市場全体の API に関する質問の完全なリファレンスです。
