مرجع واجهة برمجة التطبيقات (API) لنقاط نهاية تبادل CoinMarketCap بما في ذلك معلومات التبادل والحجم وأزواج السوق والأصول.
استخدم هذه المهارة عندما يذكر المستخدم واجهة برمجة تطبيقات التبادل، أو يسأل عن أحجام التبادل، أو يريد بيانات زوج التداول، أو يحتاج إلى معلومات إثبات الاحتياطيات، أو يقوم ببناء عمليات تكامل التبادل. هذا هو المرجع الرسمي لأسئلة CMC Exchange API.
