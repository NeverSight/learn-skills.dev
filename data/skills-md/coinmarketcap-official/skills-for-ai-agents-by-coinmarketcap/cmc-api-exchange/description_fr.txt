Référence API pour les points de terminaison d'échange CoinMarketCap, y compris les informations d'échange, le volume, les paires de marché et les actifs.
Utilisez cette compétence chaque fois que l'utilisateur mentionne l'API d'échange, pose des questions sur les volumes d'échange, souhaite des données sur les paires de trading, a besoin d'informations de preuve de réserves ou construit des intégrations d'échange. Il s'agit de la référence faisant autorité pour les questions relatives à l'API d'échange CMC.
