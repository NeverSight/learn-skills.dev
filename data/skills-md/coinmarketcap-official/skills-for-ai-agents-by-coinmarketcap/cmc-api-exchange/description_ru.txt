Справочник API для конечных точек обмена CoinMarketCap, включая информацию об обмене, объем, рыночные пары и активы.
Используйте этот навык всякий раз, когда пользователь упоминает API биржи, спрашивает об объемах обмена, хочет получить данные о торговых парах, нуждается в информации о подтверждении резервов или строит интеграцию биржи. Это авторитетный справочник по вопросам API обмена CMC.
