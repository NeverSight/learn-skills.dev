أنشئ صفحات مقصودة بتنسيق HTML جميلة وجاهزة للإنتاج ومتحركة بملف واحد. استخدم هذه المهارة عندما يطلب المستخدم إنشاء صفحة مقصودة، أو صفحة تسويق، أو صفحة منتج، أو الصفحة الرئيسية لبدء التشغيل، أو SaaS، أو صفحة إطلاق التطبيق، أو صفحة قائمة الانتظار، أو أي موقع ترويجي من صفحة واحدة. يتم تشغيله أيضًا عندما يقول المستخدم "إنشاء صفحة مقصودة لـ..."، أو "إنشاء صفحة مقصودة"، أو "إنشاء صفحة رئيسية لي"، أو "إنشاء صفحة منتج"، أو "صفحة مقصودة لبدء التشغيل الخاص بي"، أو يطلب تحسين/إعادة تصميم صفحة مقصودة موجودة. يغطي كلاً من إنشاء مسارات العمل "من الصفر" و"تحسين/تحويل". يُخرج ملف HTML واحدًا مستقلاً باستخدام Tailwind CSS ورسوم Vanilla JS المتحركة وتصميم سريع الاستجابة ووضع داكن/فاتح وإعدادات مسبقة للنمط المرئي. استخدم هذه المهارة دائمًا حتى مع طلبات الوصول البسيطة - فهي تضمن إنتاجًا متميزًا وغير عام.
