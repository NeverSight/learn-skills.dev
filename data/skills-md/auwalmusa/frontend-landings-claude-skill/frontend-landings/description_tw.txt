產生漂亮的、可用於生產的動畫單檔 HTML 登陸頁面。每當使用者要求建立登陸頁面、行銷頁面、產品頁面、啟動主頁、SaaS 登陸、應用程式啟動頁面、候補清單頁面或任何促銷單頁網站時，請使用此技能。當使用者說「為...登陸」、「產生登陸頁面」、「為我建立一個主頁」、「建立產品頁面」、「為我的新創公司登陸頁面」或要求改進/重新設計現有登陸頁面時也會觸發。涵蓋「從頭開始」產生和「改進/轉換」工作流程。輸出一個獨立的 HTML 文件，其中包含 Tailwind CSS、vanilla JS 動畫、響應式設計、暗/亮模式和精選的視覺樣式預設。即使對於簡單的著陸請求也始終使用此技能 - 它可以確保優質的非通用輸出。
