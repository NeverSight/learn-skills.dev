Genera archivos PDF de informes corporativos/comerciales profesionales a partir de texto o Markdown a través de HTML. Utiliza ECharts para visualizaciones avanzadas (mapas de calor, radar, indicadores, sankey), sistema de diseño enriquecido (cubiertas de degradado, tarjetas de estadísticas, cuadros de llamadas, barras de progreso, líneas de tiempo) y tipografía profesional. Úselo cuando el usuario desee informes PDF de calidad empresarial con narración de datos moderna.
