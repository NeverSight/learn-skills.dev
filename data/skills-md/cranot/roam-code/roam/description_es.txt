Comprensión de la base de código a través de CLI de código móvil. Úselo al explorar bases de código, planificar modificaciones, depurar fallas, evaluar el riesgo de relaciones públicas o verificar el estado de la arquitectura. Activadores de: comprensión de la estructura del proyecto, comprobaciones de seguridad previas al cambio, búsqueda de símbolos/archivos, análisis del radio de explosión, pruebas afectadas, puntuación de estado, orientación de refactorización, revisión de código. Requiere tener instalado el código de roaming (`pip install roam-code`) y un proyecto indexado (`roam init`).
