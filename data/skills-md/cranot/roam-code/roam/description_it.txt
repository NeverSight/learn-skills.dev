Comprensione della base di codice tramite CLI del codice roaming. Da utilizzare durante l'esplorazione delle codebase, la pianificazione delle modifiche, il debug degli errori, la valutazione del rischio PR o il controllo dell'integrità dell'architettura. Trigger su: comprensione della struttura del progetto, controlli di sicurezza prima della modifica, ricerca di simboli/file, analisi del raggio di esplosione, test interessati, punteggio di integrità, linee guida per il refactoring, revisione del codice. Richiede il codice roam installato (`pip install roam-code`) e un progetto indicizzato (`roam init`).
