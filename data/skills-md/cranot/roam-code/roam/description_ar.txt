فهم قاعدة التعليمات البرمجية عبر رمز التجوال CLI. يُستخدم عند استكشاف قواعد التعليمات البرمجية، أو تخطيط التعديلات، أو تصحيح الأخطاء، أو تقييم مخاطر العلاقات العامة، أو التحقق من سلامة البنية. المحفزات: فهم هيكل المشروع، وفحوصات السلامة قبل التغيير، والعثور على الرموز/الملفات، وتحليل نصف قطر الانفجار، والاختبارات المتأثرة، وتسجيل الصحة، وإرشادات إعادة البناء، ومراجعة الكود. يتطلب تثبيت رمز التجوال (`pip install roam-code`) ومشروعًا مفهرسًا (`roam init`).
