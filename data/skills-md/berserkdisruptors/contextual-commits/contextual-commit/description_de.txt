Schreiben Sie kontextbezogene Commits, die neben Codeänderungen auch Absichten, Entscheidungen und Einschränkungen erfassen. Verwenden Sie diese Option, wenn Sie Code festschreiben, eine Aufgabe abschließen oder wenn der Benutzer eine Festschreibung anfordert. Erweitert herkömmliche Commits um strukturierte Aktionszeilen im Commit-Körper, die beibehalten, WARUM Code geschrieben wurde, und nicht nur, WAS sich geändert hat.
