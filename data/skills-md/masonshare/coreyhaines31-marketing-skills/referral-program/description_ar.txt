عندما يريد المستخدم إنشاء أو تحسين أو تحليل برنامج إحالة أو برنامج تابع أو استراتيجية شفهية. يُستخدم أيضًا عندما يذكر المستخدم "الإحالة"، أو "الشركة التابعة"، أو "السفير"، أو "الأحاديث الشفهية"، أو "الحلقة الفيروسية"، أو "إحالة صديق"، أو "برنامج الشريك"، أو "حافز الإحالة"، أو "كيفية الحصول على الإحالات"، أو "العملاء الذين يحيلون العملاء"، أو "عائدات الشركات التابعة". استخدم هذا عندما يريد شخص ما من المستخدمين أو الشركاء الحاليين جلب عملاء جدد. لمعرفة مدى الانتشار الخاص بالإطلاق، راجع استراتيجية الإطلاق.
