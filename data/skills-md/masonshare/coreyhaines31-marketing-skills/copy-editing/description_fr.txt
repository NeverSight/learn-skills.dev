Lorsque l'utilisateur souhaite modifier, réviser ou améliorer une copie marketing existante. À utiliser également lorsque l'utilisateur mentionne « modifier cette copie », « réviser ma copie », « copier les commentaires », « relire », « peaufiner ceci », « améliorer cela », « balayer la copie », « resserrer cela », « cela se lit mal », « nettoyer ce texte », « trop verbeux » ou « affiner le message ». Utilisez-le lorsque l'utilisateur possède déjà une copie et souhaite l'améliorer plutôt que la réécrire à partir de zéro. Pour rédiger une nouvelle copie, voir rédaction.
