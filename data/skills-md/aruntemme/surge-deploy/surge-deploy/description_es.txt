Esta habilidad debe usarse cuando el usuario solicita implementar para aumentar, publicar en surge.sh o dice "despliegue estático", "implementar estático", "implementar para aumentar", "/despliegue estático". Implementa cualquier resultado de compilación estática: React, Vite, Vue, exportación estática Next.js, Astro, SvelteKit, Hugo, HTML simple o cualquier marco que produzca una carpeta de compilación. Esto es específicamente para implementaciones de surge.sh, no para Vercel o Netlify.
