Cette compétence doit être utilisée lorsque l'utilisateur demande un déploiement sur surge, publie sur surge.sh ou dit « déploiement statique », « déploiement statique », « déploiement sur surge », « /static-deploy ». Déploie n'importe quelle sortie de build statique - React, Vite, Vue, exportation statique Next.js, Astro, SvelteKit, Hugo, HTML brut ou tout framework qui produit un dossier de build. Ceci est spécifiquement destiné aux déploiements surge.sh, pas à Vercel ou Netlify.
