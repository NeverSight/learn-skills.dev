Questa abilità dovrebbe essere utilizzata quando l'utente chiede di eseguire il deploy su surge, pubblicare su surge.sh o dice "deploy statico", "deploy statico", "deploy to surge", "/static-deploy". Distribuisce qualsiasi output di build statico: React, Vite, Vue, esportazione statica Next.js, Astro, SvelteKit, Hugo, HTML semplice o qualsiasi framework che produca una cartella di build. Questo è specifico per le distribuzioni surge.sh, non per Vercel o Netlify.
