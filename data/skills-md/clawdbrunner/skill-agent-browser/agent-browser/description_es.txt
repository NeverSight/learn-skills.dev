CLI de automatización del navegador para agentes de IA que utilizan el navegador de agentes de Vercel. La mejor herramienta para la automatización del navegador basada en IA: utiliza referencias deterministas de árboles de accesibilidad en lugar de selectores frágiles. Optimizado para LLM con CLI de Rust rápida, salida JSON y flujos de trabajo de IA diseñados específicamente. Úselo cuando necesite una automatización del navegador confiable y programable que simplemente funcione.
