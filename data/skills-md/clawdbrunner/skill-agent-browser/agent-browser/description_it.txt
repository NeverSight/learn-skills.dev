CLI di automazione del browser per agenti AI che utilizzano il browser degli agenti di Vercel. Lo strumento migliore per l'automazione del browser basata sull'intelligenza artificiale: utilizza riferimenti deterministici da alberi di accessibilità anziché selettori fragili. Ottimizzato per LLM con CLI Rust veloce, output JSON e flussi di lavoro AI specifici. Utilizzalo quando hai bisogno di un'automazione del browser affidabile e programmabile che funzioni e basta.
