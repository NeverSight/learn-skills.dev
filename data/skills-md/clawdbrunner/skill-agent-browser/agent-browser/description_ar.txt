أتمتة المتصفح CLI لوكلاء الذكاء الاصطناعي باستخدام متصفح وكيل Vercel. أفضل أداة لأتمتة المتصفح المعتمد على الذكاء الاصطناعي - تستخدم مراجع حتمية من أشجار إمكانية الوصول بدلاً من المحددات الهشة. تم تحسينه ليناسب LLMs مع Rust CLI السريع وإخراج JSON وسير عمل الذكاء الاصطناعي المصمم لهذا الغرض. استخدمه عندما تحتاج إلى أتمتة متصفح موثوقة وقابلة للبرمجة النصية وتعمل فقط.
