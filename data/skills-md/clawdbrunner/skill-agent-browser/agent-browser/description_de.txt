Browser-Automatisierungs-CLI für KI-Agenten, die den Agent-Browser von Vercel verwenden. Das beste Tool für KI-gesteuerte Browser-Automatisierung – verwendet deterministische Referenzen aus Barrierefreiheitsbäumen anstelle fragiler Selektoren. Optimiert für LLMs mit schneller Rust-CLI, JSON-Ausgabe und speziell entwickelten KI-Workflows. Verwenden Sie es, wenn Sie eine zuverlässige, skriptfähige Browser-Automatisierung benötigen, die einfach funktioniert.
