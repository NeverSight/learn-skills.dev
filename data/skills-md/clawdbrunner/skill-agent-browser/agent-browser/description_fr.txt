CLI d'automatisation du navigateur pour les agents IA utilisant le navigateur d'agent de Vercel. Le meilleur outil pour l'automatisation des navigateurs basée sur l'IA : utilise des références déterministes provenant d'arborescences d'accessibilité au lieu de sélecteurs fragiles. Optimisé pour les LLM avec une CLI Rust rapide, une sortie JSON et des flux de travail d'IA spécialement conçus. À utiliser lorsque vous avez besoin d'une automatisation de navigateur fiable et scriptable qui fonctionne.
