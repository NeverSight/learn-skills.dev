Интерфейс командной строки для автоматизации браузера для агентов ИИ, использующих агент-браузер Vercel. Лучший инструмент для автоматизации браузера на основе искусственного интеллекта — использует детерминированные ссылки из деревьев доступности вместо хрупких селекторов. Оптимизирован для LLM благодаря быстрому интерфейсу командной строки Rust, выводу JSON и специально разработанным рабочим процессам искусственного интеллекта. Используйте, когда вам нужна надежная автоматизация браузера с поддержкой сценариев, которая просто работает.
