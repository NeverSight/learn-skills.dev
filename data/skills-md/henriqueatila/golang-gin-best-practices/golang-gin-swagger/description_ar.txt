قم بإنشاء وثائق Swagger/OpenAPI لواجهات برمجة تطبيقات Go Gin باستخدام swaggo/swag. يغطي التعليقات التوضيحية الكاملة لنقطة النهاية CRUD + auth، ووثائق النموذج، وإعداد Swagger UI، وتعريفات الأمان، وتكامل CI/CD. يُستخدم عند إضافة مستندات API، أو Swagger UI، أو مواصفات OpenAPI، أو وثائق نقطة النهاية، أو مرجع API الذي تم إنشاؤه تلقائيًا إلى تطبيق Gin.
