PostgreSQL DBA ومهارة المهندس المعماري لواجهات برمجة تطبيقات Go Gin. يغطي قرارات تصميم المخطط، وتحليل تأثير الترحيل، واستراتيجية الفهرس، وتحسين الاستعلام، والتقسيم، وتغيير حجم تجمع الاتصال، والنظام البيئي لامتداد PostgreSQL (ParadeDB، وpgvector، وPostGIS، وTimescaleDB). يُستخدم عند تصميم المخططات، أو تحليل سلامة الترحيل، أو اختيار الفهارس، أو تحسين الاستعلامات، أو تحديد الامتدادات، أو اتخاذ أي قرار بشأن بنية PostgreSQL. هذا هو "كيفية التفكير" المكمل لقاعدة بيانات جولانج جين "كيفية الاتصال". يتم التنشيط أيضًا عندما يذكر المستخدم شرح التحليل، أو مستويات القفل، أو ترحيل وقت التوقف الصفري، أو البحث عن نص كامل، أو تضمينات المتجهات، أو الاستعلامات الجغرافية المكانية، أو بيانات السلاسل الزمنية، أو ضبط أداء قاعدة البيانات.
