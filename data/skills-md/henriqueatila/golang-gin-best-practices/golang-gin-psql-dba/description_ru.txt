Администратор базы данных PostgreSQL и навыки архитектора API Go Gin. Охватывает решения по проектированию схемы, анализ влияния миграции, стратегию индексирования, оптимизацию запросов, секционирование, определение размера пула соединений и экосистему расширений PostgreSQL (ParadeDB, pgvector, PostGIS, TimescaleDB). Используйте при проектировании схем, анализе безопасности миграции, выборе индексов, оптимизации запросов, выборе расширений или принятии любого решения по архитектуре PostgreSQL. Это дополнение «как думать» к «как подключаться» в базе данных golang-gin. Также активируйте, когда пользователь упоминает EXPLAIN ANALYZE, уровни блокировки, миграцию с нулевым временем простоя, полнотекстовый поиск, векторные внедрения, геопространственные запросы, данные временных рядов или настройку производительности базы данных.
