Навыки архитектора программного обеспечения для API Go Gin. Охватывает проектирование системы, оценку сложности, шаблоны масштабируемости, разработку API, сквозные проблемы, ADR, управление техническим долгом и оркестрацию навыков джинго. Используйте при принятии архитектурных решений, оценке сложности, проектировании систем, выборе шаблонов, планировании развития API или координации различных навыков джина. Также активируйте, когда пользователь упоминает микросервисы вместо монолита, CQRS, источники событий, моделирование предметной области, ограниченные контексты, управление версиями API, стратегию кэширования, наблюдаемость или технический долг. Это «мозг», который управляет всеми остальными навыками джина.
