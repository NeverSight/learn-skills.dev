مهارة مهندس البرمجيات لواجهات برمجة تطبيقات Go Gin. يغطي تصميم النظام، وتقييم التعقيد، وأنماط قابلية التوسع، وتصميم واجهة برمجة التطبيقات (API)، والمخاوف الشاملة، وتسويات النزاعات، وإدارة ديون التكنولوجيا، وتنسيق مهارات جينغو. يُستخدم عند اتخاذ قرارات التصميم، أو تقييم التعقيد، أو تصميم الأنظمة، أو اختيار الأنماط، أو التخطيط لتطوير واجهة برمجة التطبيقات، أو التنسيق عبر مهارات الجن المتعددة. يتم التنشيط أيضًا عندما يذكر المستخدم الخدمات الصغيرة مقابل المتراصة، أو CQRS، أو مصادر الأحداث، أو نمذجة النطاق، أو السياقات المحدودة، أو إصدارات واجهة برمجة التطبيقات، أو استراتيجية التخزين المؤقت، أو إمكانية المراقبة، أو الديون التقنية. هذا هو "العقل" الذي ينظم جميع مهارات الجن الأخرى.
