Testez les API REST Go Gin avec des tests unitaires, d'intégration, e2e, de référence et fuzz. Couvre les modèles de test http, les tests basés sur des tables, les instantanés de fichiers Golden, les référentiels moqueurs, les conteneurs de test, les seuils de couverture, la détection de race et l'intégration CI/CD. À utiliser lors de l'écriture de tests pour les gestionnaires, les services, les middlewares Gin ou la configuration d'une infrastructure de test pour une API Go.
