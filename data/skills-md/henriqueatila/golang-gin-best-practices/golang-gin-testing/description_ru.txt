Тестируйте REST API Go Gin с помощью модульных, интеграционных, e2e, эталонных и нечетких тестов. Охватывает шаблоны httptest, табличные тесты, золотые снимки файлов, макеты репозиториев, тестовые контейнеры, пороговые значения покрытия, обнаружение гонок и интеграцию CI/CD. Используйте при написании тестов для обработчиков Gin, сервисов, промежуточного программного обеспечения или при настройке тестовой инфраструктуры для Go API.
