Implémentez l'authentification et l'autorisation dans les API Go Gin. Couvre le middleware JWT (liste noire jti, rotation d'actualisation), les gestionnaires de connexion/enregistrement, le hachage de mot de passe bcrypt, RBAC, la protection CSRF, la limitation de débit sur les points de terminaison d'authentification, les cookies sécurisés et les gardes d'usurpation d'identité d'administrateur. À utiliser lors de l'ajout d'authentification, de connexion, d'inscription, de jetons JWT, de sessions utilisateur, d'autorisations ou de vérifications de rôle à une application Gin.
