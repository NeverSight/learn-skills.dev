قم بدمج قواعد بيانات PostgreSQL مع واجهات برمجة تطبيقات Go Gin باستخدام GORM أو sqlx. يغطي نمط المستودع، وإعادة محاولة الاتصال مع التراجع، وترقيم صفحات المؤشر/مجموعة المفاتيح، والمعاملات المستندة إلى السياق، ووضع TLS/sslmode، والترحيلات، وحقن التبعية. يُستخدم عند إضافة دعم قاعدة البيانات، أو إنشاء النماذج، أو كتابة الاستعلامات، أو تنفيذ المستودعات، أو إعداد عمليات الترحيل، أو توصيل طبقات قاعدة البيانات إلى مشروع Gin. يتم التنشيط أيضًا عندما يذكر المستخدم GORM، أو sqlx، أو اتصال قاعدة البيانات، أو استعلامات SQL، أو نمط المستودع، أو عمليات ترحيل قاعدة البيانات في سياق Go/Gin.
