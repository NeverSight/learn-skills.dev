GORM または sqlx を使用して、PostgreSQL データベースを Go Gin API と統合します。リポジトリ パターン、バックオフを使用した接続の再試行、カーソル/キーセットのページネーション、コンテキストベースのトランザクション、TLS/SSL モード、移行、依存関係の注入について説明します。データベース サポートの追加、モデルの作成、クエリの作成、リポジトリの実装、移行のセットアップ、またはデータベース レイヤーを Jin プロジェクトに接続するときに使用します。ユーザーが Go/Gin コンテキストで GORM、sqlx、データベース接続、SQL クエリ、リポジトリ パターン、またはデータベース移行に言及した場合にもアクティブ化します。
