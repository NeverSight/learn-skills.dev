Чистая архитектура для API Go Gin. Охватывает разделение слоев (сущность, вариант использования, репозиторий, доставка), внедрение зависимостей, шаблон репозитория, обработку ошибок, очистку входных данных и тестируемую структуру проекта. Используйте при организации проектов Go/Gin, реализации чистой/шестиугольной архитектуры, настройке внедрения зависимостей или структурировании кода для обеспечения тестируемости.
