Arquitectura limpia para las API de Go Gin. Cubre la separación de capas (entidad, caso de uso, repositorio, entrega), inyección de dependencia, patrón de repositorio, manejo de errores, desinfección de entradas y estructura de proyecto comprobable. Úselo al organizar proyectos Go/Gin, implementar una arquitectura limpia/hexagonal, configurar la inyección de dependencias o estructurar código para la capacidad de prueba.
