Создает элегантные коммиты git, используя обычный формат коммитов emoji. Выполняет проверку перед фиксацией, автоматическую подготовку, анализирует различия для логической группировки, предлагает разделить большие изменения на целевые фиксации и пишет четкие императивные сообщения о фиксации. Используйте этот навык, когда пользователь хочет зафиксировать изменения, промежуточные файлы или создать сообщение о фиксации. Поддерживает --no-verify для пропуска проверок перед фиксацией.
