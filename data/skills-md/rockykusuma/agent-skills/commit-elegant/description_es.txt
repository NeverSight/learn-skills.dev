Crea elegantes confirmaciones de git utilizando el formato de confirmación convencional de emoji. Maneja la verificación previa a la confirmación, la preparación automática, analiza diferencias para agrupaciones lógicas, sugiere dividir cambios grandes en confirmaciones enfocadas y escribe mensajes de confirmación imperativos claros. Utilice esta habilidad cuando el usuario quiera confirmar cambios, preparar archivos o crear un mensaje de confirmación. Admite --no-verify para omitir comprobaciones previas a la confirmación.
