Crea eleganti commit git utilizzando il formato commit convenzionale emoji. Gestisce la verifica pre-commit, l'auto-staging, analizza le differenze per il raggruppamento logico, suggerisce di suddividere grandi modifiche in commit mirati e scrive messaggi di commit chiari e imperativi. Utilizza questa abilità quando l'utente desidera confermare modifiche, eseguire stage sui file o creare un messaggio di commit. Supporta --no-verify per saltare i controlli pre-commit.
