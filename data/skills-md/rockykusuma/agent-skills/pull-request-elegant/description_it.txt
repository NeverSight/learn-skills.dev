Crea richieste pull professionali su Azure DevOps usando l'interfaccia della riga di comando di Azure ("az repos pr"). Gestisce il push dei rami, la generazione di titoli e descrizioni PR da differenze e cronologia dei commit, il collegamento di elementi di lavoro, l'aggiunta di revisori, l'impostazione del completamento automatico e l'apertura del PR nel browser. Usare questa competenza quando l'utente vuole creare, aggiornare o gestire una richiesta pull in Azure DevOps.
