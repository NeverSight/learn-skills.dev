Создает профессиональные запросы на включение в Azure DevOps с помощью Azure CLI (az repos pr). Управляет отправкой веток, созданием заголовков и описаний PR на основе различий и истории коммитов, связыванием рабочих элементов, добавлением рецензентов, настройкой автозаполнения и открытием PR в браузере. Используйте этот навык, когда пользователь хочет создать, обновить или управлять запросом на включение в Azure DevOps.
