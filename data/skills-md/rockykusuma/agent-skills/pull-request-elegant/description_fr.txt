Crée des demandes de tirage professionnelles sur Azure DevOps à l'aide d'Azure CLI (`az repos pr`). Gère le poussée des branches, la génération de titres et de descriptions de PR à partir des différences et de l'historique des validations, la liaison des éléments de travail, l'ajout de réviseurs, la configuration de la saisie semi-automatique et l'ouverture du PR dans le navigateur. Utilisez cette compétence lorsque l’utilisateur souhaite créer, mettre à jour ou gérer une pull request sur Azure DevOps.
