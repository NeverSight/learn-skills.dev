Crea solicitudes de extracción profesionales en Azure DevOps mediante la CLI de Azure (`az repos pr`). Se encarga de impulsar ramas, generar títulos y descripciones de relaciones públicas a partir de diferencias y del historial de confirmaciones, vincular elementos de trabajo, agregar revisores, configurar la función de autocompletar y abrir las relaciones públicas en el navegador. Utilice esta habilidad cuando el usuario quiera crear, actualizar o administrar una solicitud de extracción en Azure DevOps.
