Erstellt professionelle Pull-Anfragen auf Azure DevOps mithilfe der Azure CLI („az repos pr“). Behandelt das Pushen von Zweigen, das Generieren von PR-Titeln und -Beschreibungen aus Diffs und dem Commit-Verlauf, das Verknüpfen von Arbeitselementen, das Hinzufügen von Prüfern, das Festlegen der automatischen Vervollständigung und das Öffnen des PR im Browser. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine Pull-Anfrage in Azure DevOps erstellen, aktualisieren oder verwalten möchte.
