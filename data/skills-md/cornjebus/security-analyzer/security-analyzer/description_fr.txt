Analyse complète des vulnérabilités de sécurité pour les bases de code et l'infrastructure. Analyse les dépendances (npm, pip, gem, go, cargo), les conteneurs (Docker, Kubernetes), le cloud IaC (Terraform, CloudFormation) et détecte l'exposition de secrets. Récupère les données CVE en direct à partir d'OSV.dev, calcule les scores de risque et génère des plans de remédiation par étapes avec des tests de validation TDD. À utiliser lorsque les utilisateurs mentionnent une analyse de sécurité, une vulnérabilité, un CVE, un exploit, un audit de sécurité, un test d'intrusion, OWASP, un renforcement, un audit de dépendance, la sécurité des conteneurs ou lorsqu'ils souhaitent améliorer la posture de sécurité.
