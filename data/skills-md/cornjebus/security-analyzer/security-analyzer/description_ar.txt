تحليل شامل للثغرات الأمنية لقواعد التعليمات البرمجية والبنية التحتية. يقوم بمسح التبعيات (npm، pip، Gem، go، Cargo)، والحاويات (Docker، Kubernetes)، Cloud IaC (Terraform، CloudFormation)، ويكتشف كشف الأسرار. جلب بيانات CVE المباشرة من OSV.dev، وحساب درجات المخاطر، وإنشاء خطط معالجة مرحلية باستخدام اختبارات التحقق من صحة TDD. يُستخدم عندما يذكر المستخدمون الفحص الأمني، أو الثغرات الأمنية، أو CVE، أو الاستغلال، أو تدقيق الأمان، أو اختبار الاختراق، أو OWASP، أو التعزيز، أو تدقيق التبعية، أو أمان الحاوية، أو عندما يرغبون في تحسين الوضع الأمني.
