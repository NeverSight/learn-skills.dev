دليل تشغيل عقد شبكة Ika - أدوات التحقق من الصحة، والعقد الكاملة، والمنبهات. يُستخدم عند نشر البنية التحتية لـ Ika أو تكوينها أو مراقبتها أو استكشاف أخطاء البنية التحتية لـ Ika وإصلاحها. تشغيل المهام التي تتضمن عقدة ika، أو ika-validator، أو ika-fullnode، أو ika-notifier، أو NodeConfig، أو إعداد الإجماع، أو شبكة P2P، أو تسجيل المدقق، أو عمليات عقدة Ika.
