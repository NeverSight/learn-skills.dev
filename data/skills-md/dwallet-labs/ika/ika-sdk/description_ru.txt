Руководство по сборке с помощью Ika TypeScript SDK (@ika.xyz/sdk) на Mysten Sui v2. Используйте при создании dWallet, подписании межсетевых транзакций, управлении ключами шифрования или интеграции с сетью Ika из TypeScript/JavaScript. Запускается при выполнении задач, связанных с @ika.xyz/sdk, операциями dWallet, IkaClient, IkaTransaction или межсетевым подписанием Ika.
