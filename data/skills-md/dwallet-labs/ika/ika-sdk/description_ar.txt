دليل البناء باستخدام Ika TypeScript SDK (@ika.xyz/sdk) على Mysten Sui v2. يُستخدم عند إنشاء dWallets، أو توقيع المعاملات عبر السلسلة، أو إدارة مفاتيح التشفير، أو التكامل مع شبكة Ika من TypeScript/JavaScript. تشغيل المهام التي تتضمن @ika.xyz/sdk، أو عمليات dWallet، أو IkaClient، أو IkaTransaction، أو التوقيع عبر السلسلة Ika.
