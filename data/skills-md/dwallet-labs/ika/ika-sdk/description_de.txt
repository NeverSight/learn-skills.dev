Anleitung zum Erstellen mit dem Ika TypeScript SDK (@ika.xyz/sdk) auf Mysten Sui v2. Verwenden Sie es, wenn Sie dWallets erstellen, kettenübergreifende Transaktionen signieren, Verschlüsselungsschlüssel verwalten oder über TypeScript/JavaScript in das Ika-Netzwerk integrieren. Wird bei Aufgaben ausgelöst, die @ika.xyz/sdk, dWallet-Vorgänge, IkaClient, IkaTransaction oder Ika-Cross-Chain-Signierung umfassen.
