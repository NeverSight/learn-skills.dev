Руководство по передовым практикам языка Go, охватывающее фреймворки (Gin/Go-Kratos), ORM (GORM), миграцию баз данных (Goose), структуру проекта DDD, обработку ошибок, шаблоны параллелизма, тестирование (целевой охват 99%) и инструменты Lint. Этот навык следует использовать при написании, проверке или рефакторинге кода Go, чтобы обеспечить соблюдение лучших практик. Запускается при разработке проекта Go, проверке кода, оптимизации производительности или проектировании архитектуры.
