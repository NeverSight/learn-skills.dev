Fragen Sie ein Bitpanda-Konto über die Bitpanda-API mithilfe einer gebündelten Bash-CLI ab. Deckt alle schreibgeschützten Endpunkte ab: Salden, Trades, Transaktionen, Vermögensinformationen und Live-Preise. Verwenden Sie diese Option, wenn der Benutzer sein Bitpanda-Portfolio, seine Guthaben, seine Handelshistorie, Einzahlungen, Auszahlungen, Vermögenspreise oder andere Kontoinformationen von Bitpanda überprüfen möchte. Verwendbar für Erwähnungen von Bitpanda, Krypto-Portfolio auf Bitpanda, Preisüberprüfungen oder Anfragen zu Bitpanda-Wallet-/Handelsdaten. Dieser Skill ist NUR LESEN – er unterstützt KEINEN Handel, das Aufgeben von Aufträgen, Einzahlungen, Auszahlungen oder Schreibvorgänge.
