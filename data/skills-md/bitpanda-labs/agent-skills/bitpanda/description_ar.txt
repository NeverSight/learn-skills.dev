استعلم عن حساب Bitpanda عبر Bitpanda API باستخدام bash CLI المجمعة. يغطي جميع نقاط النهاية للقراءة فقط: الأرصدة والصفقات والمعاملات ومعلومات الأصول والأسعار المباشرة. يُستخدم عندما يريد المستخدم التحقق من محفظة Bitpanda أو الأرصدة أو تاريخ التداول أو الودائع أو عمليات السحب أو أسعار الأصول أو أي معلومات حساب من Bitpanda. يمكن استخدامها عند ذكر Bitpanda، أو محفظة العملات المشفرة على Bitpanda، أو التحقق من الأسعار، أو الطلبات التي تتضمن بيانات محفظة/تداول Bitpanda. هذه المهارة للقراءة فقط — فهي لا تدعم التداول أو وضع الأوامر أو الإيداع أو السحب أو أي عمليات كتابة.
