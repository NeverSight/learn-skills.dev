Запросите учетную запись Bitpanda через API Bitpanda, используя встроенный CLI bash. Охватывает все конечные точки, доступные только для чтения: балансы, сделки, транзакции, информацию об активах и текущие цены. Используйте, когда пользователь хочет проверить свой портфель Bitpanda, балансы, историю торговли, депозиты, снятие средств, цены активов или любую информацию об учетной записи из Bitpanda. Используется для упоминаний Bitpanda, крипто-портфеля на Bitpanda, проверки цен или запросов, связанных с данными кошелька/торговых операций Bitpanda. Этот навык ТОЛЬКО ДЛЯ ЧТЕНИЯ — он НЕ поддерживает торговлю, размещение ордеров, депозиты, снятие средств или любые операции записи.
