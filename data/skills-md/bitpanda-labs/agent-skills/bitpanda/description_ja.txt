バンドルされている bash CLI を使用して、Bitpanda API 経由で Bitpanda アカウントをクエリします。すべての読み取り専用エンドポイント (残高、取引、トランザクション、資産情報、ライブ価格) をカバーします。ユーザーが Bitpanda のポートフォリオ、残高、取引履歴、入出金、資産価格、または Bitpanda のアカウント情報を確認したい場合に使用します。 Bitpanda、Bitpanda 上の暗号ポートフォリオ、価格チェック、または Bitpanda ウォレット/取引データに関するリクエストについての言及に使用できます。このスキルは読み取り専用です。取引、注文、入金、出金、または書き込み操作はサポートされていません。
