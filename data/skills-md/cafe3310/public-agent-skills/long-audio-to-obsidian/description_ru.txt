Используется для организации и объединения сложной файловой структуры проектов преобразования речи в текст в документы Markdown, подходящие для архивирования Obsidian. Этот навык основан на когнитивных способностях агента идентифицировать логику файлов и использовать общие сценарии для выполнения слияния.
