Используется для аудита кодовых баз, чтобы убедиться, что их наименование соответствует установленной терминологии и стандартам. Этот навык следует использовать при обеспечении соблюдения «вездесущего языка» проекта, выявлении отклонений в именах методов/переменных/параметров и предложении исправлений.
