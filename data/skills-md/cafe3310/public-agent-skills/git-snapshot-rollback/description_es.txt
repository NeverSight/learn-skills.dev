Se utiliza en el desarrollo de Git cuando es necesario abandonar el intento actual y volver a una confirmación histórica. Archiva automáticamente el estado actual en el archivo/rama y crea enlaces bidireccionales (origen y destino) en ARCHIVE.md, lo que garantiza la trazabilidad del flujo de decisiones de desarrollo. Adecuado para escenarios donde se requiere una reversión segura y retención del contexto del intento fallido.
