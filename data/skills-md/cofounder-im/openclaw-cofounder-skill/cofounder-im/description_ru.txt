Получите данные о стартап-проекте и спецификации сборки, созданные ИИ, с сайта CoFounder.im. Используйте, когда пользователь хочет создать проект, который был проверен и запланирован на CoFounder.im — перечисляет проекты, получает полную спецификацию сборки (технический стек, план MVP, UI/UX, план реализации, выходные данные компоновщика OpenClaw) и помогает организовать субагенты для его сборки. Триггерные фразы — «создать мой проект CoFounder», «вытащить мой проект из cofounder.im», «использовать мою спецификацию сборки CoFounder», «начать сборку из CoFounder», «получить мой план запуска», «создать из сооснователя».
