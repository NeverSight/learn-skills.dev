Comprensione del linguaggio naturale degli intenti di query delle risorse Aliyun, pianificazione e generazione di piani di esecuzione CLI Aliyun standardizzati (formato JSON). Utilizza questa competenza quando gli utenti interrogano le risorse Aliyun (ECS, RDS, VPC, SLB, OSS, ecc.), necessitano di ottenere lo stato delle risorse, visualizzare i dettagli delle risorse o analizzare le relazioni tra le risorse. Supporta query semplici, query di associazione, query composte, query diagnostiche e altri scenari.
