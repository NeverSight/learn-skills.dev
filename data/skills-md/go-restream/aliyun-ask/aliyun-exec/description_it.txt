Analizza la configurazione dell'analisi dei requisiti utente (formato JSON), genera e genera l'elenco delle attività TODO. Utilizza questa competenza quando l'utente fornisce una configurazione di analisi dei requisiti della query sulle risorse Alibaba Cloud e deve convertire la configurazione JSON in un elenco di attività eseguibili. Supporta solo operazioni di query sulle risorse Alibaba Cloud; rifiuta qualsiasi richiesta di operazione relativa alla modifica. Utilizza lo strumento da riga di comando Aliyun CLI per eseguire query e ottenere assistenza sui parametri tramite la guida di aliyun.
