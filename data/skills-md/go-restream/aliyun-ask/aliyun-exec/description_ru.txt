Анализировать конфигурацию анализа требований пользователя (формат JSON), генерировать и выводить список задач TODO. Используйте этот навык, когда пользователь предоставляет конфигурацию анализа требований к ресурсам Alibaba Cloud и ему необходимо преобразовать конфигурацию JSON в список исполняемых задач. Поддерживает только операции запроса ресурсов Alibaba Cloud; отклоняет любые запросы операций, связанных с изменениями. Используйте инструмент командной строки Aliyun CLI для выполнения запросов и получения справки по параметрам через справку aliyun.
