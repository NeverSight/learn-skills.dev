Analiza la configuración del análisis de requisitos del usuario (formato JSON), genera y genera una lista de tareas TODO. Utilice esta habilidad cuando el usuario proporcione una configuración de análisis de requisitos de consulta de recursos de Alibaba Cloud y necesite convertir la configuración JSON en una lista de tareas ejecutable. Solo admite operaciones de consulta de recursos de Alibaba Cloud; rechaza cualquier solicitud de operación relacionada con cambios. Utilice la herramienta de línea de comandos Aliyun CLI para realizar consultas y obtener ayuda sobre parámetros a través de la ayuda de aliyun.
