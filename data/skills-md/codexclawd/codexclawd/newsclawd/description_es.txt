Habilidad de monitoreo de noticias y alertas para OpenClaw. Supervisa cuentas de X/Twitter, precios de criptomonedas, canales RSS de noticias web y envía alertas cuando ocurren eventos importantes. Úselo cuando el usuario desee realizar un seguimiento de las noticias, recibir alertas de precios, monitorear cuentas específicas de Twitter o configurar notificaciones automáticas para temas que le interesan.
