Controle teléfonos Android reales a través de la API Mobilerun. Admite tocar, deslizar, escribir, tomar capturas de pantalla, leer el árbol de accesibilidad de la interfaz de usuario y administrar aplicaciones. Úselo cuando el usuario desee automatizar o controlar de forma remota un dispositivo Android, interactuar con aplicaciones móviles o ejecutar tareas de agente de IA en un teléfono. Requiere una clave API de Mobilerun (con el prefijo dr_sk_) y un dispositivo conectado (teléfono personal a través de Portal APK o dispositivo en la nube).
