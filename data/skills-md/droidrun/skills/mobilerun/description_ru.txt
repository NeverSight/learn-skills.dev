Управляйте реальными телефонами Android через API Mobilerun. Поддерживает касание, пролистывание, ввод текста, создание снимков экрана, чтение дерева специальных возможностей пользовательского интерфейса и управление приложениями. Используйте, когда пользователь хочет автоматизировать или удаленно управлять устройством Android, взаимодействовать с мобильными приложениями или запускать задачи агента ИИ на телефоне. Требуется ключ API Mobilerun (с префиксом dr_sk_) и подключенное устройство (личный телефон через Portal APK или облачное устройство).
