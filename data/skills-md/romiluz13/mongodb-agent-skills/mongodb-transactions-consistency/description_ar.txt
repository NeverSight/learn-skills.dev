صحة معاملة MongoDB واتساقها وأمان إعادة المحاولة. يُستخدم عند تنفيذ عمليات كتابة مستندات متعددة، أو تصحيح أخطاء المعاملات الفاشلة، أو اختيار readConcern/writeConcern، أو التعامل مع TransientTransactionError أو UnknownTransactionCommitResult، أو تحديد متى تكون المعاملات مطلوبة. مشغلات "المعاملة"، و"مع المعاملة"، و"الجلسة"، و"قراءة القلق"، و"كتابة القلق"، و"الاتساق السببي"، و"اللقطة"، و"إعادة محاولة الالتزام"، و"ACID"، و"TransientTransactionError"، و"UnknownTransactionCommitResult".
