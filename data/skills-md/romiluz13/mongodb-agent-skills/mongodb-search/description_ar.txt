بنية محرك البحث MongoDB وتكوين الاستعلام وعمليات البحث. يُستخدم عند إنشاء فهارس البحث، أو كتابة خطوط أنابيب $search أو $searchMeta، أو اختيار المحللين، أو ضبط الملاءمة المعجمية، أو التعامل مع تنبيهات/مقاييس البحث، أو نشر البحث في المجتمع، أو تنسيق البحث المختلط باستخدام $rankFusion/$scoreFusion. مشغلات على "بحث MongoDB"، "بحث Atlas"، "$search"، "$searchMeta"، "الإكمال التلقائي"، "المرادفات"، "الوجه"، "المركب"، "نقاط البحث"، "تسليط الضوء"، "storedSource"، "returnStoredSource"، "returnScope"، "mongot"، "searchCoordinator"، "البحث في الحقول القصوى المفهرسة"، "حقول nGram"، "$rankFusion"، "$scoreFusion" و"البحث المختلط".
