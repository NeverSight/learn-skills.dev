MongoDB 搜尋引擎架構、查詢組合和搜尋操作。在建立搜尋索引、編寫 $search 或 $searchMeta 管道、選擇分析器、調整詞彙相關性、處理搜尋警報/指標、在社群中部署搜尋或使用 $rankFusion/$scoreFusion 編排混合搜尋時使用。在「MongoDB Search」、「Atlas Search」、「$search」、「$searchMeta」、「autocomplete」、「synonyms」、「facet」、「compound」、「search Score」、「highlight」、「storedSource」、「returnStoredSource Max」、「物件」 fields」、「$rankFusion」上觸發「$scoreFusion」和「混合搜尋」。
