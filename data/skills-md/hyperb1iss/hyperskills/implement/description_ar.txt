استخدم هذه المهارة عند كتابة التعليمات البرمجية — إنشاء الميزات، أو إصلاح الأخطاء، أو إعادة البناء، أو أي أعمال تنفيذ متعددة الخطوات. يتم التنشيط عند الإشارة إلى تنفيذ، أو بناء هذا، أو ترميز هذا، أو بدء البرمجة، أو إصلاح هذا الخطأ، أو إعادة البناء، أو إجراء تغييرات، أو تطوير هذه الميزة، أو خطة التنفيذ، أو مهمة الترميز، أو كتابة الكود، أو البدء في البناء.
