Используйте этот навык для сложных операций git, включая перебазирование, разрешение конфликтов слиянием, выборку вишен, управление ветвями или археологию репозитория. Активируется при упоминании git rebase, конфликта слияния, выбора вишни, истории git, очистки ветвей, git bisect, рабочего дерева, принудительной отправки или сложных операций git.
