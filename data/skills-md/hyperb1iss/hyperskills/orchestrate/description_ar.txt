استخدم هذه المهارة عند تنسيق العمل متعدد الوكلاء على نطاق واسع - أسراب البحث، أو إنشاءات الميزات المتوازية، أو الإرسال المستند إلى الموجة، أو خطوط أنابيب البناء والمراجعة والإصلاح، أو أي مهمة تتطلب أكثر من 3 وكلاء. يتم التنشيط عند ذكر السرب، أو الوكلاء المتوازيين، أو وكلاء متعددين، أو منسقين، أو متباعدين، أو إرسال موجة، أو جيش بحث، أو إطلاق العنان، أو وكلاء إرسال، أو عمل موازي.
