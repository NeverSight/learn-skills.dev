قم بتشغيل خط أنابيب ملخص المحتوى اليومي الذي يجلب خلاصات Podcast RSS وقنوات YouTube، ويسجل الصوت، وينشئ ملخصات الذكاء الاصطناعي باللغة الصينية التقليدية، ويرسل الإشعارات بشكل اختياري عبر Telegram. استخدم هذه المهارة عندما يريد المستخدم: إعداد أو تشغيل ملخص يومي، أو تلخيص ملفات podcast أو مقاطع فيديو YouTube، أو إنشاء ملخص محتوى، أو تجميع محتوى RSS/YouTube، أو إنشاء نظام تلخيص تلقائي. مشغلات أيضًا لـ: "الملخص اليومي" أو "البودكاست" أو "YouTube 整理" أو "المحتوى" أو "البودكاست" أو "تلخيص اشتراكاتي" أو أي طلب لجمع الوسائط وتلخيصها بشكل دوري المحتوى. حتى لو قال المستخدم "ملخص" أو "摘要" فقط، تحقق مما إذا كانت هذه المهارة تنطبق.
