Cette compétence analyse les sites Web de documentation pour générer automatiquement des organigrammes interactifs Mermaid montrant les flux de travail et les processus des utilisateurs. Utilisez cette compétence lorsque les utilisateurs demandent des diagrammes de flux, une visualisation de processus ou une documentation de flux de travail à partir de la documentation produit existante, des documents API ou des sites de didacticiels.
