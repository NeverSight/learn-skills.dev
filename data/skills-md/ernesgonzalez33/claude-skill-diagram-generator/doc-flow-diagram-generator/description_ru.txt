Этот навык анализирует веб-сайты с документацией для автоматического создания интерактивных блок-схем Mermaid, показывающих рабочие процессы и процессы пользователя. Используйте этот навык, когда пользователи запрашивают блок-схемы, визуализацию процессов или документацию рабочих процессов из существующей документации по продукту, документации API или обучающих сайтов.
