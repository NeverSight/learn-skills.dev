Questa competenza analizza i siti Web di documentazione per generare automaticamente diagrammi di flusso interattivi della Sirena che mostrano i flussi di lavoro e i processi degli utenti. Utilizza questa competenza quando gli utenti richiedono diagrammi di flusso, visualizzazione di processi o documentazione del flusso di lavoro dalla documentazione di prodotto esistente, dalla documentazione API o dai siti di tutorial.
