Esta habilidad analiza sitios web de documentación para generar automáticamente diagramas de flujo interactivos de Mermaid que muestran los flujos de trabajo y los procesos de los usuarios. Utilice esta habilidad cuando los usuarios soliciten diagramas de flujo, visualización de procesos o documentación de flujo de trabajo de documentación de productos, documentos API o sitios de tutoriales existentes.
