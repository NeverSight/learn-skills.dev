وكيل تحرير فيديو كامل يعمل بنظام FFmpeg. استخدم هذه المهارة عندما يريد المستخدم القيام بأي شيء بملفات الفيديو أو الصوت - التشذيب، أو القطع، أو تحويل الترميز، أو تغيير الحجم، أو التسلسل، أو إضافة ترجمات، أو تراكب الصور أو النص، أو تطبيق درجات الألوان أو جداول البحث، أو تطبيع مستويات الصوت، أو إنشاء انتقالات، أو تركيب الشاشة الخضراء، أو استخراج الإطارات، أو عروض الشرائح، أو التصدير لوسائل التواصل الاجتماعي (TikTok، Reels، YouTube). استخدمه أيضًا لطرح أسئلة برامج الترميز (H.264، وH.265، وVP9، وAV1، وProRes، وFFV1)، والتشفير المسرع بواسطة الأجهزة (NVIDIA NVENC، وIntel QSV، وVAAPI، وApple VideoToolbox)، وأي إنشاء أوامر ffmpeg أو ffprobe. حتى لو قال المستخدم فقط "تحويل هذا الفيديو" أو "تصغيره" أو "أضف شعاري" - يمكنك الوصول إلى هذه المهارة. المستندات المرجعية في المراجع/مرشحات الغلاف وبرامج الترميز والانتقالات وتسريع الأجهزة. البرامج النصية المساعدة في البرامج النصية / التعامل مع الفحص والتسلسل والتطبيع.
