Recherchez sur le Web à l’aide d’une instance SearXNG auto-hébergée. À utiliser lorsque les utilisateurs demandent à effectuer une recherche avec SearXNG, ou lorsqu'une recherche sur le Web est nécessaire et qu'une instance SearXNG est configurée. Prend en charge les catégories, les moteurs, la plage horaire et les filtres de langue.
