Sorgen Sie dafür, dass die Agentenfähigkeiten mit dauerhaftem Feedback in Einklang gebracht werden, indem Sie Kommentare, Korrekturen und wiederholte Anleitungen in der engsten vorhandenen Fähigkeit zusammenfassen. Verwenden Sie es auf Schritt und Tritt, insbesondere wenn das Feedback eine wiederverwendbare Regel, einen wiederholten Fehler oder eine bessere Abstraktion offenbart, die ein aufgabenspezifisches Beispiel ersetzen sollte.
