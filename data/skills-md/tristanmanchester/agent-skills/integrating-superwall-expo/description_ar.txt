يدمج نظام حظر الاشتراك غير المدفوع لـ Superwall وبوابة الاشتراك في تطبيقات React Native Expo باستخدام حزمة تطوير البرامج (SDK) لـ expo-superwall. يُستخدم عند إضافة Superwall إلى مشروع Expo (SDK 53+)، أو مواضع الأسلاك/بوابات الميزات، أو إدارة المستخدمين وحالة الاشتراك، أو التعامل مع الروابط العميقة/الخروج من الويب، أو دمج RevenueCat، أو تصحيح أخطاء الوحدة الأصلية.
