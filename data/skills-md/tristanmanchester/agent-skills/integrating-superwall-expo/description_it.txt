Integra i paywall Superwall e il gate degli abbonamenti nelle app React Native Expo utilizzando l'SDK expo-superwall. Da utilizzare quando si aggiunge Superwall a un progetto Expo (SDK 53+), si cablano posizionamenti/gating di funzionalità, si gestiscono utenti e stato dell'abbonamento, si gestiscono collegamenti diretti/checkout Web, si integra RevenueCat o si esegue il debug di problemi del modulo nativo.
