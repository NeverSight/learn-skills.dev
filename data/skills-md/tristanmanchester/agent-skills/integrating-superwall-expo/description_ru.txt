Интегрирует платный доступ Superwall и ограничение подписки в приложениях React Native Expo с помощью expo-superwall SDK. Используйте при добавлении Superwall в проект Expo (SDK 53+), подключении мест размещения/контроля функций, управлении пользователями и статусом подписки, обработке глубоких ссылок/веб-оформлении, интеграции RevenueCat или отладке проблем, связанных с собственными модулями.
