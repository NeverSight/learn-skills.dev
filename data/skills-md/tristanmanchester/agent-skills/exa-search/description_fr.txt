Effectue une recherche sur le Web via l'API de recherche d'Exa et renvoie les URL sources (éventuellement avec les surlignages, le texte intégral, les résumés et les sous-pages). À utiliser lorsque l'utilisateur demande « rechercher avec Exa », « utiliser Exa », « trouver des sources/URL », « faire des recherches sur le Web », « récupérer le texte d'une page Web », « obtenir des faits saillants/résumés », « filtrer par domaine/date/catégorie » ou a besoin de nouveaux résultats (actualités, recherches en temps réel).
