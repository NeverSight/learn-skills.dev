Выполняет поиск в Интернете через API поиска Exa и возвращает исходные URL-адреса (необязательно с выделенными фрагментами, полным текстом, сводками и подстраницами). Используйте, когда пользователь просит «поиск с помощью Exa», «использовать Exa», «найти источники/URL-адреса», «провести веб-исследование», «получить текст веб-страницы», «получить основные моменты/сводки», «фильтровать по домену/дате/категории» или ему нужны свежие результаты (новости, поиск в реальном времени).
