Configure et fait évoluer les backends Convex pour les applications Expo / React Native : créez des déploiements, configurez EXPO_PUBLIC_CONVEX_URL (et les variables d'environnement EAS), connectez ConvexProvider + ConvexReactClient, utilisez les types d'API générés et implémentez des requêtes/mutations/actions, schémas/index, authentification et téléchargements de fichiers. À utiliser lorsque l'utilisateur mentionne Convex avec Expo/React Native, create-expo-app, expo-router, useQuery/useMutation, ConvexReactClient, convex dev, URL de déploiement, EAS builds ou demande une base de données/backend en temps réel dans une application Expo.
