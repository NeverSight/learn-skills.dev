Настраивает и развивает серверные части Convex для приложений Expo/React Native: создает развертывания, настраивает EXPO_PUBLIC_CONVEX_URL (и переменные среды EAS), подключает ConvexProvider + ConvexReactClient, использует сгенерированные типы API и реализует запросы/мутации/действия, схемы/индексы, аутентификацию и загрузку файлов. Используйте, когда пользователь упоминает Convex с Expo/React Native, create-expo-app, expo-router, useQuery/useMutation, ConvexReactClient, convex dev, URL-адрес развертывания, сборки EAS или запрашивает базу данных/бэкэнд в реальном времени в приложении Expo.
