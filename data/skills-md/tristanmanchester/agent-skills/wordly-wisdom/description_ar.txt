يوفر تحليلًا معايرًا للقرارات باستخدام النماذج العقلية المتعددة بأسلوب تشارلي مونجر، والانعكاس، ورسم خرائط الحوافز، وفحوصات دائرة الكفاءة، وعمليات تدقيق سوء التقدير، وتأثيرات الدرجة الثانية، وتحديثات التنبؤ. يُستخدم عندما يطلب المستخدم الحصول على رأي أوراكل، أو مكالمة صعبة، أو مذكرة قرار، أو تقرير ما قبل الوفاة، أو رؤية خارجية، أو فريق أحمر، أو فحص سلامة العقل، أو ما الذي أفتقده، أو التفكير في ذلك، أو يريد تحليل استراتيجية أو توظيف أو استثمار أو خطة أو منتج أو شراكة أو اختيار رئيسي للحياة. تجنب البحث عن الحقائق البسيطة أو الأسئلة القانونية أو الطبية أو السوقية الحساسة للوقت دون أدلة جديدة.
