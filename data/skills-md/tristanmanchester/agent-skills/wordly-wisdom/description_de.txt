Bietet kalibrierte Entscheidungsanalysen mit mehreren mentalen Modellen im Charlie-Munger-Stil, Inversion, Anreizzuordnung, Kompetenzkreisprüfungen, Fehleinschätzungsprüfungen, Effekten zweiter Ordnung und Prognoseaktualisierungen. Verwenden Sie diese Option, wenn der Benutzer eine Orakelaufnahme, einen harten Anruf, eine Entscheidungsnotiz, ein Prämortem, eine Außenansicht, ein Red-Team, eine Überprüfung der geistigen Gesundheit, „Was habe ich übersehen“ anfordert, dies durchdenkt oder eine Analyse einer Strategie, Einstellung, Investition, eines Plans, eines Produkts, einer Partnerschaft oder einer wichtigen Lebensentscheidung wünscht. Vermeiden Sie es, nach einfachen Fakten zu suchen oder zeitkritische rechtliche, medizinische oder marktbezogene Fragen ohne neue Beweise zu stellen.
