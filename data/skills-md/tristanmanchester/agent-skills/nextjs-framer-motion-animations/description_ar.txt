يضيف حركة آمنة للإنتاج لرسوم React أو Framer Motion المتحركة إلى تطبيقات Next.js، بما في ذلك التفاعلات الدقيقة للكشف والتمرير والنقر، وانتقالات whileInView، وstagger، وAnimatePresence، وانتقالات التخطيط وLayoutId، وإعادة الترتيب، وواجهة المستخدم المرتبطة بالتمرير، والانتقالات خفيفة الوزن لمحتوى المسار. يُستخدم عندما يطلب المستخدم إضافة أو إعادة بناء أو تصحيح قواعد تعليمات Motion أو Framer Motion في App Router أو Pages Router، خاصة حول حدود الخادم/العميل، أو الحركة المنخفضة، أو LazyMotion، أو حجم الحزمة، أو الترطيب، أو انتقالات المسار. تجنب استخدام المخططات الزمنية بنمط GSAP، أو المشاهد WebGL أو المشاهد ثلاثية الأبعاد، أو رواية القصص ذات التمرير الثقيل، أو تأثيرات CSS فقط ما لم يتم طلب الحركة صراحةً.
