在 Next.js 應用程式中新增生產安全的 React Motion 或 Framer Motion 動畫，包括顯示、懸停和點擊微互動、whileInView、交錯、AnimatePresence、佈局和佈局 ID 轉換、重新排序、滾動連結 UI 和輕量級路由內容轉換。當使用者要求在 App Router 或 Pages Router 程式碼庫中新增、重構或偵錯 Motion 或 Framer Motion 時使用，特別是在伺服器/用戶端邊界、減少運動、LazyMotion、套件大小、水合作用或路由轉換周圍。避免使用 GSAP 風格的時間軸、WebGL 或 3D 場景、大量滾動故事講述或純 CSS 效果，除非明確要求 Motion。
