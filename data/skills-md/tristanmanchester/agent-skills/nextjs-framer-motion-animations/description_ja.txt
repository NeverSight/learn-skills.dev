Next.js アプリに実稼働環境で安全な Motion for React または Framer Motion アニメーションを追加します。これには、リビール、ホバー、タップのマイクロインタラクション、whileInView、stagger、AnimatePresence、レイアウトとlayoutId トランジション、並べ替え、スクロールリンクされた UI、軽量のルートコンテンツトランジションが含まれます。ユーザーが App Router または Pages Router のコードベースでモーションまたはフレーマー モーションの追加、リファクタリング、またはデバッグを要求する場合に使用します。特にサーバー/クライアント境界付近、モーションの削減、LazyMotion、バンドル サイズ、ハイドレーション、ルート遷移などです。モーションが明示的に要求されない限り、GSAP スタイルのタイムライン、WebGL または 3D シーン、重いスクロールのストーリーテリング、または CSS のみのエフェクトの場合は避けてください。
