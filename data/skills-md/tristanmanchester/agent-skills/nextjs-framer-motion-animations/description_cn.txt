向 Next.js 应用程序添加生产安全的 React Motion 或 Framer Motion 动画，包括显示、悬停和点击微交互、whileInView、交错、AnimatePresence、布局和布局 ID 转换、重新排序、滚动链接 UI 和轻量级路由内容转换。当用户要求在 App Router 或 Pages Router 代码库中添加、重构或调试 Motion 或 Framer Motion 时使用，特别是在服务器/客户端边界、减少运动、LazyMotion、包大小、水合作用或路由转换周围。避免使用 GSAP 风格的时间轴、WebGL 或 3D 场景、大量滚动故事讲述或纯 CSS 效果，除非明确请求 Motion。
