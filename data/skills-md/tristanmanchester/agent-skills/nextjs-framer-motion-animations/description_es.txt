Agrega animaciones Motion for React o Framer Motion seguras para producción a las aplicaciones Next.js, incluidas microinteracciones de revelación, desplazamiento y toque, mientras que InView, stagger, AnimatePresence, transiciones de diseño y diseñoId, reordenamiento, interfaz de usuario con desplazamiento vinculado y transiciones ligeras de contenido de ruta. Úselo cuando el usuario solicite agregar, refactorizar o depurar Motion o Framer Motion en las bases de código de App Router o Pages Router, especialmente en torno a límites de servidor/cliente, movimiento reducido, LazyMotion, tamaño de paquete, hidratación o transiciones de ruta. Evite líneas de tiempo estilo GSAP, escenas WebGL o 3D, narraciones con desplazamiento pesado o efectos solo CSS a menos que se solicite explícitamente Motion.
