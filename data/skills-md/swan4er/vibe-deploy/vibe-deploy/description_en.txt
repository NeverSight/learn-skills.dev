Automatic deployment of any projects (frontend, backend, fullstack) to Timeweb Cloud. Supports two paths — App Platform (PaaS) and cloud server (VPS). Includes domain, DNS, SSL, database, Docker, and CI/CD (GitHub Actions) setup. Use this skill when the user wants to deploy, spin up, or publish a project, site, or app on Timeweb Cloud, or when timeweb, twc, or deployment is mentioned.
