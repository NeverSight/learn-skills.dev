Навыки автоматического анализа выбора продуктов Amazon по категориям. Проводит углубленное исследование рынка по категориям Amazon с помощью пятимерной модели оценки и создает отчет об анализе Markdown. Запускается, когда пользователь использует команду /category-selection или такие запросы, как «анализ категории XX», «исследования рынка категории XX», «выбор продукта категории XX» и т. д. Поддерживается настраиваемое количество анализа, по умолчанию — «Топ-20».
