Esegui un'analisi di penetrazione completa degli elenchi dei concorrenti di Amazon, coprendo la logica di copia, l'analisi delle revisioni, l'analisi delle parole chiave, le dinamiche di mercato, ecc. Dopo l'analisi, salva automaticamente come documento di report Markdown nella directory report/. Richiama quando l'utente utilizza il comando /amazon-analyse con un ASIN del prodotto.
