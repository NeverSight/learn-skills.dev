OpenClaw のクロード コードの統合。このスキルは、次のインターフェースを提供します。
- https://code.claude.com/docs からクロード コードのドキュメントをクエリします
- spec-kit でスペック駆動開発を使用する
- サブエージェントとコーディングタスクを管理する
- AI支援コーディングワークフローの実行
- ベスト プラクティスと一般的なワークフローにアクセスする
このスキルは、ユーザーが次のことを行う場合に使用します。
- コーディング作業のサポートを受ける
- クロード コードのドキュメントのクエリ
- 構造化された開発にはスペックキット ワークフローを使用する
- AI支援開発ワークフローを管理
- 複雑なプログラミングタスクを実行する
