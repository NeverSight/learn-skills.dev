Integración de Claude Code para OpenClaw. Esta habilidad proporciona interfaces para:
- Consultar la documentación de Claude Code desde https://code.claude.com/docs
- Utilice el desarrollo basado en especificaciones con el kit de especificaciones
- Gestionar subagentes y tareas de codificación.
- Ejecutar flujos de trabajo de codificación asistidos por IA
- Acceda a las mejores prácticas y flujos de trabajo comunes
Utilice esta habilidad cuando los usuarios quieran:
- Obtenga ayuda con tareas de codificación
- Consultar la documentación del Código Claude
- Utilice el flujo de trabajo del kit de especificaciones para un desarrollo estructurado.
- Gestionar flujos de trabajo de desarrollo asistidos por IA
- Ejecutar tareas de programación complejas.
