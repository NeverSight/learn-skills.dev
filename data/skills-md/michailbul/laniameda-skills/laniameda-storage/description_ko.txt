기둥 태그가 지정된 프롬프트와 참조 자산(Instagram 또는 모든 소스)을 Laniameda Prompt Storage 대시보드(Convex KB)에 저장하세요.
사용자가 "이것을 저장하세요", "이것을 저장하세요", "볼트에 추가하세요"라고 말하거나 프롬프트/레시피/워크플로를 공유하거나 재사용할 만한 참조 미디어를 보낼 때 사용하세요.
규칙: 단일 항목 = 하나의 프롬프트 또는 하나의 자산. 배치 = JSON 배열. 변형은 동일한 PromptText + PromptIngestKey를 공유합니다. 서로 다른 프롬프트를 연결하지 마십시오.
ownerUserId는 항상 KB_OWNER_USER_ID env var에서 자동으로 읽혀집니다(호출자가 전달하지 않음).
