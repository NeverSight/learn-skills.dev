يُستخدم عند تحليل كيفية اتصال الخدمات أو تصميمها أو إصلاحها - خاصةً عندما يكون الاتصال غير مستقر، أو أحادي الاتجاه، أو ينسى، أو يفتقر إلى الضغط الخلفي، أو لا يوجد به مسار استرداد. يغطي المزامنة بين الخدمات، ودورة حياة الاتصال، وتحليل معدل التجزئة، ورسم خرائط وضع الفشل، وتصميم البروتوكول لأنظمة الإصلاح الذاتي. يُستخدم عندما يذكر المستخدم عدم الاستقرار بين الخدمات، أو مشكلات المزامنة، أو مشكلات إعادة الاتصال، أو عندما يطلب مراجعة كيفية تواصل نظامين مع بعضهما البعض.
