Abilità nella generazione di immagini tramite intelligenza artificiale. Utilizza l'API Gemini per creare immagini di prodotti, poster, miniature, ecc. di alta qualità (risoluzione fino a 4K).

Scenari scatenanti:
- Necessità di generare immagini promozionali di prodotti o immagini di visualizzazione di e-commerce
- Necessità di creare poster, banner o grafica per social media
- È necessaria l'intelligenza artificiale per generare o modificare immagini ad alta definizione
- Necessità di produrre in batch variazioni di immagini

Parole trigger: generazione di immagini, immagini AI, immagini di prodotto, poster, miniature, 4K, HD, creazione di immagini, /immagine, modifica di immagini

[Non si attiva] Attività di puro testo, generazione di codice, discussioni di progettazione che non coinvolgono immagini
[Chiedi prima] Se manca: descrizione dell'immagine, preferenza di risoluzione, proporzioni: chiedi di completare.
