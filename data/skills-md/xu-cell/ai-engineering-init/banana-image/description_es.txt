Habilidades de generación de imágenes de IA. Utilice la API de Gemini para crear imágenes de productos, carteles, miniaturas, etc. de alta calidad (hasta una resolución de 4K).

Escenarios desencadenantes:
- Necesidad de crear imágenes promocionales de productos, imágenes de visualización de comercio electrónico.
- Necesidad de diseñar carteles, pancartas y gráficos para redes sociales.
- Necesita IA para generar o editar imágenes de alta resolución
- Necesidad de producir variantes de imágenes por lotes.

Palabras desencadenantes: generar imagen, imagen AI, imagen de producto, póster, miniatura, 4K, HD, crear imagen, /image, editar imagen

[Sin disparador] Tareas de texto puro, generación de código, discusiones de diseño que no involucran imágenes.
[Preguntar primero] Si falta: descripción de la imagen, preferencia de resolución, relación de aspecto: solicite completar.
