Backend utility usage guide. Covers selection and usage rules for common utilities such as object conversion, string handling, collection operations, tree construction, date-time, JSON, and parameter validation.

Trigger scenarios:
- Object conversion (BO/VO/Entity)
- String handling, collection stream operations
- Tree structure building
- JSON serialization/deserialization
- Date-time processing
- Parameter validation

Trigger words: utility, object conversion, BeanUtils, StringUtils, collection operations, tree structure, DateUtils, JsonUtils, convert, string handling, date-time

Notes:
- Object conversion tools must be uniformly selected according to the project; mixing multiple solutions is prohibited.
- This guide is a general Java backend utility reference, applicable to any Spring Boot project.
