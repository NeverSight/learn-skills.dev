バックエンド ユーティリティ クラスの使用ガイド  
一般的なユーティリティの選択と使用基準をカバーします: オブジェクト変換、文字列処理、コレクション操作、ツリー構築、日時、JSON、パラメータ検証など。

トリガーシナリオ:
- オブジェクト変換(BO/VO/Entity)
- 文字列処理、コレクションストリーム操作
- ツリー構造の建物
- JSONのシリアル化/逆シリアル化
- 日時の処理
- パラメータの検証

トリガーワード: ユーティリティクラス、オブジェクト変換、BeanUtils、StringUtils、コレクション操作、ツリー構造、DateUtils、JsonUtils、変換、文字列処理、日時

注:
- オブジェクト変換ユーティリティはプロジェクトごとに均一に選択する必要があります。複数の溶液を混合することは禁止されています。
- このガイドは、あらゆる Spring Boot プロジェクトに適した、一般的な Java バックエンド ユーティリティのリファレンスです。
