後端實用類使用指南
涵蓋常見實用程式的選擇和使用標準，例如物件轉換、字串處理、集合操作、樹建置、日期時間、JSON 和參數驗證。

觸發場景：
- 物件轉換（BO/VO/Entity）
- 字串處理、集合流操作
- 樹結構建築
- JSON序列化/反序列化
- 日期時間處理
- 參數驗證

觸發詞：實用程式類別、物件轉換、BeanUtils、StringUtils、集合運算、樹狀結構、DateUtils、JsonUtils、轉換、字串處理、日期時間

注意：
- 每個項目必須一致地選擇和使用物件轉換實用程式；禁止混合多種溶液。
- 本指南是通用 Java 後端實用程式參考，與任何 Spring Boot 專案相容。
