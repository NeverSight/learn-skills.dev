백엔드 유틸리티 클래스 사용 가이드  
객체 변환, 문자열 처리, 수집 작업, 트리 작성, 날짜-시간, JSON, 매개변수 검증 등 일반적인 유틸리티에 대한 선택 및 사용 표준을 다룹니다.

트리거 시나리오:  
- 객체 변환(BO/VO/엔티티)  
- 문자열 처리, 수집 스트림 작업  
- 트리 구조 구축  
- JSON 직렬화/역직렬화  
- 날짜-시간 처리  
- 매개변수 검증  

트리거 단어: 유틸리티 클래스, 객체 변환, BeanUtils, StringUtils, 컬렉션 작업, 트리 구조, DateUtils, JsonUtils, 변환, 문자열 처리, 날짜-시간  

참고:  
- 객체 변환 유틸리티는 프로젝트별로 일관되게 선택하고 사용해야 합니다. 여러 솔루션을 혼합하는 것은 금지되어 있습니다.  
- 본 가이드는 모든 종류의 Spring Boot 프로젝트에 적용 가능한 일반적인 Java 백엔드 유틸리티 클래스 가이드입니다.
