Guida all'utilizzo delle classi di utilità backend. Copre gli standard di selezione e utilizzo per utilità comuni come conversione di oggetti, gestione di stringhe, operazioni di raccolta, creazione di alberi, data/ora, JSON e convalida dei parametri.

Scenari scatenanti:
- Conversione di oggetti (BO/VO/Entità)
- Elaborazione di stringhe, operazioni di flusso di raccolta
- Costruzione di strutture ad albero
- Serializzazione/deserializzazione JSON
- Gestione data-ora
- Convalida dei parametri

Parole trigger: classe di utilità, conversione di oggetti, BeanUtils, StringUtils, operazione di raccolta, struttura ad albero, DateUtils, JsonUtils, conversione, elaborazione di stringhe, data-ora

Nota:
- Gli strumenti di conversione degli oggetti devono essere selezionati in modo uniforme in base al progetto; è vietato mescolare più soluzioni.
- Questa guida è una guida generale della classe di utilità backend Java compatibile con tutti i tipi di progetti Spring Boot.
