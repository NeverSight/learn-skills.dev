Запрашивайте онлайн-журналы через Grafana Loki, поддерживая переключение между несколькими средами, отслеживание идентификаторов трассировки, запросы путей API и поиск по ключевым словам.

Триггерные сценарии:
- Необходимо проверять онлайн-журналы для устранения проблем.
- Запросить полную цепочку запросов по идентификатору трассировки.
- Запросить все журналы API по его пути.
- Поиск конкретных журналов ошибок или стеков исключений.
- Переключение между различными средами обслуживания журналов.
