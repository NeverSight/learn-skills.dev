Сотрудничайте с Google Gemini CLI. Делегируйте Gemini задачи по кодированию для создания прототипов внешнего интерфейса, дизайна пользовательского интерфейса и проверки кода.

Триггерные сценарии:
- Нужны прототипы интерфейса/UI/стиля.
- Нужен дизайн компонентов CSS/React/Vue.
- Нужен обзор кода и анализ ошибок.
- Пользователь явно запрашивает сотрудничество с Gemini.
- Прототипирование сложной логики интерфейса.

Триггерные слова: Gemini, совместная работа, мультимодель, интерфейсный прототип, дизайн пользовательского интерфейса, CSS, стиль, Gemini-collab.

Предпосылки:
- Установлен Gemini CLI (npm install -g @google/gemini-cli)
- Ключ Google API настроен.

Примечание. Gemini имеет ограничения в понимании внутренней логики; расставить приоритеты Кодекса для внутренних задач.
