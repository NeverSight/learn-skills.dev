Collaborate with Google Gemini CLI on development. Delegate coding tasks to Gemini for front-end prototyping, UI design, and code review.

Trigger scenarios:
- Front-end/UI/styling prototyping needed
- CSS/React/Vue component design needed
- Code review and bug analysis needed
- User explicitly requests Gemini collaboration
- Complex front-end logic prototyping

Trigger words: Gemini, collaborate, multi-model, front-end prototype, UI design, CSS, styling, gemini-collab

Prerequisites:
- Gemini CLI installed (npm install -g @google/gemini-cli)
- Google API Key configured

Note: Gemini has limited understanding of backend logic; prioritize Codex for backend tasks.
