التعاون مع Google Gemini CLI. تفويض مهام البرمجة إلى Gemini للحصول على النماذج الأولية للواجهة الأمامية وتصميم واجهة المستخدم ومراجعات التعليمات البرمجية.

سيناريوهات الزناد:
- يلزم وجود نماذج أولية للواجهة الأمامية/واجهة المستخدم/التصميم
- مطلوب تصميم مكونات CSS/React/Vue
- يلزم مراجعة الكود وتحليل الأخطاء
- يطلب المستخدم صراحة تعاون الجوزاء
- النماذج الأولية لمنطق الواجهة الأمامية المعقد

الكلمات المحفزة: الجوزاء، التعاون، النماذج المتعددة، النموذج الأولي للواجهة الأمامية، تصميم واجهة المستخدم، CSS، التصميم، التعاون الجوزاء

المتطلبات الأساسية:
- تثبيت Gemini CLI (npm install -g @google/gemini-cli)
- تم تكوين مفتاح Google API

ملحوظة: الجوزاء لديه قيود في فهم المنطق الخلفي؛ إعطاء الأولوية للمخطوطة للمهام الخلفية.
