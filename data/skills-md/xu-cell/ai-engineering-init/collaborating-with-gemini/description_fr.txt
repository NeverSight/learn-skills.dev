Collaborez avec Google Gemini CLI pour le développement. Déléguez les tâches de codage à Gemini pour le prototypage frontal, la conception de l'interface utilisateur et la révision du code.

Scénarios de déclenchement :
- Prototypage front-end/UI/style nécessaire
- Conception de composants CSS/React/Vue requise
- Revue de code et analyse de bugs nécessaires
- L'utilisateur demande explicitement la collaboration de Gemini
- Prototypage de logiques front-end complexes

Mots déclencheurs : Gemini, collaborer, multi-modèle, prototype front-end, conception d'interface utilisateur, CSS, style, collaboration Gemini

Conditions préalables :
- Gemini CLI installé (npm install -g @google/gemini-cli)
- Clé API Google configurée

Remarque : Gemini a des limites dans la compréhension de la logique backend ; donner la priorité au Codex pour les tâches backend.
