Codesarrollo con Google Gemini CLI. Delegue tareas de codificación a Gemini para la creación de prototipos de interfaz de usuario, diseño de interfaz de usuario y revisión de código.

Escenarios desencadenantes:
- Necesita prototipos de front-end/UI/estilo
- Necesita diseño de componentes CSS/React/Vue
- Necesita revisión de código y análisis de errores.
- El usuario solicita explícitamente la colaboración de Gemini.
- Creación de prototipos de lógica front-end compleja

Palabras desencadenantes: Gemini, colaboración, multimodelo, prototipo de interfaz de usuario, diseño de interfaz de usuario, CSS, estilo, colaboración gemini

Requisitos previos:
- Gemini CLI instalada (npm install -g @google/gemini-cli)
- Clave API de Google configurada

Nota: Gemini tiene una comprensión limitada de la lógica de back-end; priorizar el Codex para las tareas de back-end.
