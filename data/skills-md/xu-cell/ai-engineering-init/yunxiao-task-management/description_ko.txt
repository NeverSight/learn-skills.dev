Alibaba Cloud Yunxiao 프로젝트 협업 및 작업 관리 기술. Yunxiao Open API를 통해 작업 항목을 직접 운영합니다: 작업 쿼리, 상태 변경, 설명 편집, 댓글 추가, 작업 시간 기록, 하위 작업 생성, 프로젝트 검색 등.
전체 개발 워크플로를 지원합니다. 작업 읽기 → 상위 요구 사항 찾기 → 요구 사항 세부 정보 읽기 → 테스트 요청 티켓 생성 → 테스트 요청 티켓 편집 → 테스트에 전달.
지능형 테스트 요청 티켓 강화 지원: git에서 자동으로 분기/커밋 정보를 가져오고, pom.xml에서 버전 번호를 읽고, 사용자 작업을 찾고, 테스트 요청 템플릿을 채웁니다.

트리거 시나리오:
- Yunxiao 작업 상태 쿼리/수정
- 테스트 요청 티켓 콘텐츠 강화(Git 정보 자동 수집)
- 테스트 요청 하위 작업 생성
- 개인 대기 중인 작업 보기
- 요구사항 세부정보 읽기

트리거 단어: Yunxiao, 작업 관리, 작업 항목, 상태 변경, 테스트 요청, 테스트 요청 티켓, 강화 테스트 요청 티켓, SARW, EZML, IXXP, Yunxiao 작업
