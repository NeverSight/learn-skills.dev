Alibaba Cloud Yunxiao プロジェクト コラボレーション タスク管理スキル。 Yunxiao Open API 経由で作業項目を直接操作します: タスクのクエリ、ステータスの変更、説明の編集、コメントの追加、作業時間の記録、サブタスクの作成、プロジェクトの検索など。
完全な開発ワークフローをサポートします: タスクの読み取り → 親要件の検索 → 要件の詳細の読み取り → テスト リクエストの作成 → テスト リクエストの編集 → テストに引き渡す。
インテリジェントなテスト要求の完了をサポートします。git からブランチ/コミット情報を自動的に取得し、pom.xml からバージョン番号を読み取り、ユーザー タスクを見つけてテスト要求テンプレートに入力します。

トリガーシナリオ:
- Yunxiaoタスクステータスのクエリ/変更
- 完全なテストリクエストコンテンツ (git 情報の自動収集)
- テスト要求サブタスクの作成
- 個人の保留中のタスクを表示する
- 要件の詳細を読む
