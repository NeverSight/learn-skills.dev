Compétences en débogage de requêtes de base de données MySQL. Interrogez la base de données MySQL locale directement via la CLI mysql et résolvez les bogues avec les journaux.
Évaluez automatiquement si une requête de base de données est nécessaire lors des corrections de bogues, extrayez le nom de la base de données (ID de locataire) des journaux et prenez en charge la spécification manuelle de la base de données.

Scénarios de déclenchement :
- Interroger la base de données pour vérifier les données lors du dépannage des bogues avec les journaux
- Vérifier la valeur réelle d'un enregistrement dans la base de données
- Vérifier si les résultats de l'exécution SQL répondent aux attentes
- Enquêter sur les incohérences, les pertes de données et les anomalies de données
- Besoin d'afficher la structure des tables, les index et la distribution des données

Mots déclencheurs : vérifier la base de données, vérifier la table, exécuter SQL, vérifier l'enregistrement, requête MySQL, dépannage de la base de données, vérifier les données, vérifier les données, requête de base de données, débogage de la base de données
