Habilidades de depuración de consultas de bases de datos MySQL. Consulte la base de datos MySQL local directamente a través de la CLI de MySQL y solucione errores con la ayuda de registros.
Evalúe automáticamente si es necesario consultar la base de datos al corregir errores, extraiga el nombre de la base de datos (ID del inquilino) de los registros y admita la especificación manual de la base de datos.

Escenarios desencadenantes:
- Al solucionar errores con registros, consulte la base de datos para verificar los datos.
- Verificar el valor real de un registro en la base de datos.
- Verificar si los resultados de la ejecución de SQL cumplen con las expectativas.
- Solucionar problemas de inconsistencia de datos, pérdida de datos y anomalías de datos.
- Necesidad de ver la estructura de la tabla, los índices y la distribución de datos.

Palabras desencadenantes: verificar base de datos, verificar tabla, ejecutar SQL, verificar registro, consulta mysql, solución de problemas de base de datos, verificar datos, verificar datos, consulta de base de datos, depuración de base de datos
