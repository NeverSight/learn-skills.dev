MySQL データベース クエリのデバッグ スキル。 mysql CLI を介してローカル MySQL データベースに直接クエリを実行し、バグ調査にログを使用します。
バグ修正時にデータベース クエリが必要かどうかを自動的に評価し、ログからデータベース名 (テナント ID) を抽出し、手動でのデータベース指定をサポートします。

トリガーシナリオ:
- ログのバグを調査するときにデータベースにクエリを実行してデータを検証する必要がある
- データベース内のレコードの実際の値を確認する
- SQLの実行結果が期待どおりかどうかを検証する
- データの不整合、データ損失、または異常なデータの問題を調査します。
- テーブル構造、インデックス、データ分布を表示する必要がある
