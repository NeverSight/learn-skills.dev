مهارات تصحيح استعلام قاعدة بيانات MySQL. استعلم عن قاعدة بيانات MySQL المحلية مباشرة عبر MySQL CLI، باستخدام السجلات لتتبع الأخطاء.  
أثناء إصلاحات الأخطاء، قم تلقائيًا بتقييم ما إذا كانت قاعدة البيانات بحاجة إلى الاستعلام عنها، واستخرج اسم قاعدة البيانات (معرف المستأجر) من السجلات، ودعم المواصفات اليدوية لقاعدة البيانات.

سيناريوهات الزناد:
- الاستعلام عن قاعدة البيانات للتحقق من البيانات عند تصحيح الأخطاء باستخدام السجلات  
- التحقق من القيمة الفعلية للسجل في قاعدة البيانات  
- التحقق مما إذا كانت نتائج تنفيذ SQL تلبي التوقعات  
- التحقيق في حالات عدم اتساق البيانات أو فقدان البيانات أو الحالات الشاذة  
- الحاجة إلى عرض هياكل الجدول والفهارس وتوزيع البيانات  

كلمات التشغيل: 查数据库، 查表، 执行SQL، 查记录، mysql查询، 数据库排查، 验证数据، 查数据، db查询، شكرا جزيلا
