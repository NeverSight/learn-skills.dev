Starten Sie eine neue OpenSpec-Änderung mithilfe des experimentellen Artefakt-Workflows. Verwenden Sie diese Option, wenn der Benutzer eine neue Funktion, Korrektur oder Änderung mit einem strukturierten Schritt-für-Schritt-Ansatz erstellen möchte.

Auslöseszenarien:
– Es muss eine neue OpenSpec-Änderung erstellt werden
- Starten Sie einen strukturierten Workflow für eine neue Funktion oder Korrektur
– Initialisieren Sie das Änderungsverzeichnis und zeigen Sie die erste Artefaktvorlage an

Auslösewörter: neue Änderung, neue Änderung, opsx:new, OpenSpec New, Änderung erstellen, neue Funktion starten, Änderung hinzufügen
