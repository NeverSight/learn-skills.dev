Especificación de desarrollo de interfaz API del proyecto leniu-yunshitang-core. Guía de desarrollo de API basada en una arquitectura de cuatro capas.

Escenarios desencadenantes:
- Diseño de interfaces API del proyecto leniu.
- Escribir código de capa del controlador leniu
- Configuración de permisos de interfaz y validación de parámetros.
- Seleccionar tipos de valores de retorno de interfaz
- Planificación de enrutamiento multiextremo (Web/Móvil/Android)

Proyecto aplicable: leniu-tengyun-core (proyecto Cloud Canteen)

Palabras desencadenantes: interfaz API, controlador, LeResult, LeResponse, LeRequest, desarrollo de interfaz, prefijo de ruta, consulta de paginación, permiso de interfaz
