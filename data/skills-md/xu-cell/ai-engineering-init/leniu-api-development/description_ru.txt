Спецификация разработки API проекта leniu-yunshitang-core. Руководство по разработке API на основе четырехуровневой архитектуры.

Сценарии запуска:
- Проектирование API-интерфейсов проекта leniu.
- Написание кода слоя контроллера leniu.
- Настройка разрешений интерфейса и проверка параметров.
- Выбор типов возвращаемых значений интерфейса.
- Планирование многосторонней маршрутизации (Интернет/Мобильный/Android)

Применимый проект: leniu-tengyun-core (проект «Облачная столовая»)

Триггерные слова: интерфейс API, контроллер, LeResult, LeResponse, LeRequest, разработка интерфейса, префикс маршрута, запрос на нумерацию страниц, разрешение интерфейса.
