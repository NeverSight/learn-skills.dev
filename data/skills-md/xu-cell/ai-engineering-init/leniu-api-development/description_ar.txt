مواصفات تطوير واجهة برمجة تطبيقات المشروع leniu-yunshitang-core. دليل تطوير واجهة برمجة التطبيقات (API) استنادًا إلى بنية رباعية الطبقات.

سيناريوهات التشغيل:
- تصميم واجهات API لمشروع leniu
- كتابة كود طبقة تحكم leniu
- تكوين أذونات الواجهة والتحقق من صحة المعلمة
- حدد أنواع قيمة إرجاع الواجهة
- تخطيط التوجيه متعدد الأطراف (الويب/الجوال/Android)

المشروع القابل للتطبيق: leniu-tengyun-core (مشروع المقصف السحابي)

كلمات التشغيل: واجهة API، وحدة التحكم، LeResult، LeResponse، LeRequest، تطوير الواجهة، بادئة المسار، استعلام ترقيم الصفحات، إذن الواجهة
