leniu-yunshitang-core Specifiche di sviluppo dell'API del progetto. Guida allo sviluppo dell'API basata su un'architettura a quattro livelli.

Scenari scatenanti:
- Progettazione delle API del progetto leniu
- Scrittura del codice del livello controller leniu
- Configurazione delle autorizzazioni dell'interfaccia e della convalida dei parametri
- Selezione dei tipi di valore restituito dall'API
- Pianificazione del routing multi-end (Web/Mobile/Android)

Progetto applicabile: leniu-tengyun-core (progetto Cloud Canteen)

Parole trigger: interfaccia API, controller, LeResult, LeResponse, LeRequest, sviluppo interfaccia, prefisso percorso, query di impaginazione, autorizzazione interfaccia
