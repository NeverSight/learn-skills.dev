Spécification de développement de l'interface API du projet leniu-yunshitang-core. Guide de développement d'API basé sur une architecture à quatre couches.

Scénarios de déclenchement :
- Concevoir les interfaces API du projet Leniu
- Écrire le code de la couche du contrôleur leniu
- Configurer les autorisations de l'interface et la validation des paramètres
- Choisissez les types de valeurs de retour d'interface
- Planification de routage multi-end (Web/Mobile/Android)

Projet applicable : leniu-tengyun-core (projet Cloud Canteen)

Mots déclencheurs : interface API, Controller, LeResult, LeResponse, LeRequest, développement d'interface, préfixe de route, requête de pagination, autorisation d'interface
