Utilisez cette compétence automatiquement lorsqu'une nouvelle fonctionnalité doit être créée à partir de zéro et que le flux de travail de développement complet est requis.

Scénarios de déclenchement :
- Une nouvelle fonctionnalité doit être développée à partir de zéro
- Les tables de base de données doivent être conçues et le code généré
- Un guide complet du workflow de développement est nécessaire
- Les générateurs de code doivent être configurés et le code back-end généré

Mots déclencheurs : développer une fonctionnalité, dev, nouvelle fonctionnalité, développement de fonctionnalités, développer à partir de zéro, développement complet, développer un nouveau module
