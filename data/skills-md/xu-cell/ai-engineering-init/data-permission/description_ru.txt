Руководство по созданию универсальных разрешений на данные на уровне строк. Обеспечьте изоляцию данных с помощью шаблона перехватчика AOP + MyBatis, поддерживая детализацию разрешений на уровне отдела, собственных и настраиваемых разрешений.  
Триггерные сценарии:  
- Добавлена фильтрация данных на уровне строк для бизнес-модулей.  
- Изоляция данных на уровне проектного отдела/собственности  
- Расширение типов разрешений настраиваемых данных.  
- Временно обойти разрешения на данные для запроса полных данных.  
- Устранение неполадок, связанных с неэффективностью разрешений на данные.  

Триггерные слова: разрешение на данные, разрешение на уровне строки, изоляция данных, разрешение отдела, собственное разрешение, настраиваемое разрешение, DataScope, DataPermission, фильтрация данных.  
Примечание. Если в проекте есть специальный навык (например, «leniu-data-permission»), сначала используйте выделенную версию.
