Используется для отладки точки останова **Node.js stdio MCP Server** в VS Code (прикрепите к Node Inspector). Всякий раз, когда пользователь упоминает: MCP stdio, @modelcontextprotocol/inspector, node --inspect/--inspect-brk, отладку присоединения VS Code, не достигнутые точки останова, исходную карту, отладку артефактов сборки/распространения, конфликт/занят порт 9229, конфигурацию launch.json и т. д., всегда используйте этот навык. Выходные данные должны напрямую содержать готовый к копированию фрагмент файла launch.json, работоспособные команды запуска инспектора (включая версии с измененным портом) и контрольный список устранения неполадок с сортировкой по приоритету.
