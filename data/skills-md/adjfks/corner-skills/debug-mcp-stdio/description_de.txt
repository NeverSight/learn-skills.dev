Used for breakpoint debugging of the **Node.js stdio MCP Server** in VS Code (attach to Node Inspector). Immer wenn der Benutzer Folgendes erwähnt: MCP stdio, @modelcontextprotocol/inspector, node --inspect/--inspect-brk, VS-Code-Attach-Debugging, Haltepunkte nicht erreicht, Sourcemap, Debuggen von Build-/Dist-Artefakten, Port 9229-Konflikt/belegt, launch.json-Konfiguration usw., muss diese Fähigkeit verwendet werden. Die Ausgabe sollte direkt ein zum Kopieren und Einfügen bereites launch.json-Snippet, ausführbare Inspektor-Startbefehle (einschließlich Port-Change-Versionen) und eine nach Priorität geordnete Checkliste zur Fehlerbehebung bereitstellen.
