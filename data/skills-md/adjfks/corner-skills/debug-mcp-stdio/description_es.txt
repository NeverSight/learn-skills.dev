Se utiliza para la depuración de puntos de interrupción del **servidor MCP estándar Node.js** en VS Code (adjuntar al inspector de nodos). Siempre que el usuario mencione: MCP stdio, @modelcontextprotocol/inspector, nodo --inspect/--inspect-brk, depuración de adjuntos de VS Code, puntos de interrupción no alcanzados, mapa fuente, depuración de artefactos de compilación/dist, conflicto/ocupado del puerto 9229, configuración de launch.json, etc., se debe utilizar esta habilidad. La salida debe proporcionar directamente fragmentos de launch.json listos para copiar y pegar, comandos de inicio del inspector ejecutables (incluidas las versiones de cambio de puerto) y una lista de verificación de solución de problemas ordenada por prioridad.
