Utilizzato per il debug dei punti di interruzione del **server MCP stdio Node.js** in VS Code (Collega a Node Inspector). Ogni volta che l'utente menziona: MCP stdio, @modelcontextprotocol/inspector, nodo --inspect/--inspect-brk, debugging di collegamento VS Code, punti di interruzione non raggiunti, sourcemap, debug di artefatti build/dist, conflitto/occupata porta 9229, configurazione launch.json e così via, utilizza sempre questa abilità. L'output dovrebbe fornire direttamente uno snippet launch.json copia-incollabile, comandi di avvio dell'ispettore eseguibili (incluse le versioni con porta modificata) e un elenco di controllo per la risoluzione dei problemi in ordine di priorità.
