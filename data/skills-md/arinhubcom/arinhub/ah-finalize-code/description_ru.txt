Используйте этот навык, чтобы завершить изменения кода перед созданием PR при использовании префикса «ах». Используйте, когда вас просят «ах завершить код» или «ах завершить изменения». Запускает многоэтапный рабочий процесс, который упрощает код, создает ретроспективу, добавляет тесты, обновляет ссылки JSDoc, обновляет документацию, оптимизирует спецификации, выполняет проверку кода и создает запрос на включение — фиксация после каждого шага.
