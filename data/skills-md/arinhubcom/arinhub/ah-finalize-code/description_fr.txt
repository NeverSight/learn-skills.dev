Utilisez cette compétence pour finaliser les modifications du code avant de créer un PR lorsque vous utilisez le préfixe « ah ». À utiliser lorsqu'on vous demande de « ah finaliser le code » ou « ah finaliser les modifications ». Exécute un flux de travail en plusieurs étapes qui simplifie le code, crée une rétrospective, ajoute des tests, met à jour les références JSDoc, met à jour la documentation, optimise les spécifications, effectue une révision du code et crée une pull request - en s'engageant après chaque étape.
