استخدم هذه المهارة لإنهاء تغييرات التعليمات البرمجية قبل إنشاء PR عند استخدام البادئة "ah". استخدمه عندما يُطلب منك "آه إنهاء الكود"، أو "آه إنهاء التغييرات". تشغيل سير عمل متعدد الخطوات يبسط التعليمات البرمجية، وينشئ بأثر رجعي، ويضيف الاختبارات، ويحدث مراجع JSDoc، ويحدث الوثائق، ويحسن المواصفات، وينفذ مراجعة التعليمات البرمجية، وينشئ طلب سحب - الالتزام بعد كل خطوة.
