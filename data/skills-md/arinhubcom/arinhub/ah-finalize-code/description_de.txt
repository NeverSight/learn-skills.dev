Verwenden Sie diesen Skill, um Codeänderungen abzuschließen, bevor Sie eine PR erstellen, wenn Sie das Präfix „ah“ verwenden. Verwenden Sie diese Option, wenn Sie nach „Ah Code finalisieren“ oder „Änderungen finalisieren“ gefragt werden. Führt einen mehrstufigen Workflow aus, der Code vereinfacht, eine Retrospektive erstellt, Tests hinzufügt, JSDoc-Referenzen aktualisiert, Dokumentation aktualisiert, Spezifikationen optimiert, eine Codeüberprüfung durchführt und einen Pull-Request erstellt – der nach jedem Schritt festgeschrieben wird.
