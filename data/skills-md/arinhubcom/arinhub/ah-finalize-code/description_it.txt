Utilizza questa competenza per finalizzare le modifiche al codice prima di creare un PR quando si utilizza il prefisso "ah". Da utilizzare quando viene richiesto "ah finalizzare il codice" o "ah finalizzare le modifiche". Esegue un flusso di lavoro in più fasi che semplifica il codice, crea una retrospettiva, aggiunge test, aggiorna i riferimenti JSDoc, aggiorna la documentazione, ottimizza le specifiche, esegue una revisione del codice e crea una richiesta pull, impegnandosi dopo ogni passaggio.
