Utilice esta habilidad para finalizar los cambios de código antes de crear un PR cuando utilice el prefijo "ah". Úselo cuando se le solicite "finalizar el código" o "finalizar los cambios". Ejecuta un flujo de trabajo de varios pasos que simplifica el código, crea una retrospectiva, agrega pruebas, actualiza las referencias de JSDoc, actualiza la documentación, optimiza las especificaciones, realiza una revisión del código y crea una solicitud de extracción, comprometiéndose después de cada paso.
