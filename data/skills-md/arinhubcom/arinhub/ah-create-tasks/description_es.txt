Utilice esta habilidad para crear tareas desde un PRD y ADR usando el prefijo "ah". Úselo cuando se le solicite "crear tareas". Esta habilidad ejecuta todo el proceso del kit de especificaciones (especificar, aclarar, planificar, investigar, verificar la complejidad, lista de verificación y generación de tareas) con pases de análisis de coherencia y comprometiéndose después de cada paso importante.
