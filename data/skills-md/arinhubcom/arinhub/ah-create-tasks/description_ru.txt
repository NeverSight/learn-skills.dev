Используйте этот навык для создания задач из PRD и ADR с использованием префикса «ах». Используйте, когда вас просят «ах создать задачи». Этот навык запускает весь конвейер Spec Kit — определение, уточнение, планирование, исследование, проверку сложности, контрольный список и создание задач — с проходами анализа согласованности и фиксацией после каждого основного шага.
