Быстрый локальный анализ PDF-файлов с помощью PyMuPDF (fitz) для вывода Markdown/JSON и дополнительных изображений/таблиц. Используйте, когда скорость важнее надежности, или в качестве запасного варианта, когда более тяжелые парсеры недоступны. По умолчанию анализируется один PDF-файл с выходными папками для каждого документа.
