À utiliser lorsque vous travaillez avec C#, F#, les bibliothèques .NET, ASP.NET Core, Blazor, Entity Framework Core et l'écosystème .NET plus large.
UTILISER POUR : fonctionnalités du langage .NET, choix des bibliothèques et des frameworks, structure du projet, sélection des packages, décisions d'architecture
NE PAS UTILISER POUR : les détails de configuration spécifiques de la bibliothèque (utiliser les sous-compétences : web, données, tests, événements, cloud, etc.)
