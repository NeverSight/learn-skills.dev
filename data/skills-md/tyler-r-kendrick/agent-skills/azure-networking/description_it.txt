Servizi di rete di Azure tra cui reti virtuali, endpoint privati, bilanciatori del carico, gateway applicazione, front door e DNS. Copre la topologia hub-spoke, i modelli di endpoint privati ​​e i livelli di sicurezza della rete.
UTILIZZARE PER: configurazione della rete virtuale, configurazione dell'endpoint privato, topologia hub-spoke, peering VNet, gruppi di sicurezza di rete, regole del firewall di Azure, configurazione del servizio di bilanciamento del carico, configurazione del gateway applicazione, porta anteriore di Azure, configurazione del collegamento privato, configurazione DNS, isolamento della rete
NON UTILIZZARE PER: configurazione delle risorse di calcolo (usare azure-create-app), rete di Funzioni di Azure (usare funzioni di Azure), rete di database all'interno di Postgres/cosmos (usare le rispettive competenze), analisi dei costi (usare azure-cost-optimization), distribuzione delle risorse (usare azure-deploy)
