Azure-Netzwerkdienste einschließlich virtueller Netzwerke, privater Endpunkte, Lastausgleichsfunktionen, Application Gateway, Front Door und DNS. Deckt Hub-Spoke-Topologie, private Endpunktmuster und Netzwerksicherheitsebenen ab.
VERWENDUNG FÜR: Einrichtung virtueller Netzwerke, Konfiguration privater Endpunkte, Hub-Spoke-Topologie, VNet-Peering, Netzwerksicherheitsgruppen, Azure-Firewall-Regeln, Einrichtung des Lastenausgleichs, Konfiguration des Application Gateway, Azure Front Door, Einrichtung privater Links, DNS-Konfiguration, Netzwerkisolation
NICHT VERWENDEN FÜR: Computing-Ressourcenkonfiguration (Azure-Create-App verwenden), Azure-Funktionsnetzwerke (Azure-Funktionen verwenden), Datenbanknetzwerke innerhalb von Postgres/Cosmos (entsprechende Fähigkeiten verwenden), Kostenanalyse (Azure-Kostenoptimierung verwenden), Ressourcenbereitstellung (Azure-Deploy verwenden)
