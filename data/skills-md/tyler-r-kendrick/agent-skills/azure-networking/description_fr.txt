Services de mise en réseau Azure, notamment réseaux virtuels, points de terminaison privés, équilibreurs de charge, Application Gateway, Front Door et DNS. Couvre la topologie hub-spoke, les modèles de points de terminaison privés et les couches de sécurité réseau.
UTILISER POUR : configuration de réseau virtuel, configuration de point de terminaison privé, topologie hub-spoke, peering de réseau virtuel, groupes de sécurité réseau, règles de pare-feu Azure, configuration de l'équilibreur de charge, configuration d'Application Gateway, Azure Front Door, configuration de liaison privée, configuration DNS, isolation du réseau
NE PAS UTILISER POUR : la configuration des ressources de calcul (utilisez azure-create-app), la mise en réseau Azure Functions (utilisez azure-functions), la mise en réseau de bases de données dans postgres/cosmos (utilisez les compétences respectives), l'analyse des coûts (utilisez azure-cost-optimisation), le déploiement de ressources (utilisez azure-deploy)
