Сетевые службы Azure, включая виртуальные сети, частные конечные точки, балансировщики нагрузки, шлюз приложений, входную дверь и DNS. Охватывает звездчатую топологию, шаблоны частных конечных точек и уровни сетевой безопасности.
ИСПОЛЬЗОВАНИЕ ДЛЯ: настройки виртуальной сети, конфигурации частной конечной точки, звездообразной топологии, пиринга виртуальных сетей, групп сетевой безопасности, правил брандмауэра Azure, настройки балансировщика нагрузки, конфигурации шлюза приложений, Azure Front Door, настройки частного канала, конфигурации DNS, изоляции сети.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: конфигурации вычислительных ресурсов (используйте azure-create-app), сетевых функций Azure (используйте azure-functions), сетевых баз данных в postgres/cosmos (используйте соответствующие навыки), анализа затрат (используйте azure-cost-optimization), развертывания ресурсов (используйте azure-deploy).
