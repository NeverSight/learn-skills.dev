仮想ネットワーク、プライベート エンドポイント、ロード バランサー、アプリケーション ゲートウェイ、フロント ドア、DNS などの Azure ネットワーキング サービス。ハブスポーク トポロジ、プライベート エンドポイント パターン、ネットワーク セキュリティ層をカバーします。
用途: 仮想ネットワークのセットアップ、プライベート エンドポイントの構成、ハブスポーク トポロジ、VNet ピアリング、ネットワーク セキュリティ グループ、Azure ファイアウォール ルール、ロード バランサーのセットアップ、Application Gateway の構成、Azure Front Door、プライベート リンクのセットアップ、DNS 構成、ネットワーク分離
次の用途には使用しないでください: コンピューティング リソース構成 (azure-create-app を使用)、Azure Functions ネットワーキング (azure-functions を使用)、postgres/cosmos 内のデータベース ネットワーキング (それぞれのスキルを使用)、コスト分析 (azure-cost-optimization を使用)、リソースのデプロイ (azure-deploy を使用)
