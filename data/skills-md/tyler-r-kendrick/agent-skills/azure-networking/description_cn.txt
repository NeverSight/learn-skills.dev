Azure 网络服务，包括虚拟网络、专用终结点、负载均衡器、应用程序网关、前门和 DNS。涵盖中心辐射型拓扑、专用端点模式和网络安全层。
用于：虚拟网络设置、专用终结点配置、中心辐射型拓扑、VNet 对等互连、网络安全组、Azure 防火墙规则、负载均衡器设置、应用程序网关配置、Azure 前门、专用链接设置、DNS 配置、网络隔离
请勿用于：计算资源配置（使用 azure-create-app）、Azure Functions 网络（使用 azure-functions）、postgres/cosmos 中的数据库网络（使用相应的技能）、成本分析（使用 azure-cost-optimization）、资源部署（使用 azure-deploy）
