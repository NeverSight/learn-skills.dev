가상 네트워크, 개인 끝점, 부하 분산 장치, 응용 프로그램 게이트웨이, 전면 도어 및 DNS를 포함한 Azure 네트워킹 서비스입니다. 허브-스포크 토폴로지, 개인 끝점 패턴 및 네트워크 보안 계층을 다룹니다.
사용 대상: 가상 네트워크 설정, 프라이빗 엔드포인트 구성, 허브-스포크 토폴로지, VNet 피어링, 네트워크 보안 그룹, Azure 방화벽 규칙, 부하 분산 장치 설정, Application Gateway 구성, Azure Front Door, 프라이빗 링크 설정, DNS 구성, 네트워크 격리
