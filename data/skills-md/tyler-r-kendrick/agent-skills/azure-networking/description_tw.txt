Azure 網路服務，包括虛擬網路、專用終端、負載平衡器、應用程式閘道、前門和 DNS。涵蓋中心輻射型拓樸、專用端點模式和網路安全層。
用於：虛擬網路設定、專用終點配置、中心輻射型拓撲、VNet 對等互連、網路安全群組、Azure 防火牆規則、負載平衡器設定、應用程式閘道配置、Azure 前門、專用連結設定、DNS 設定、網路隔離
請勿用於：計算資源配置（使用 azure-create-app）、Azure Functions 網路（使用 azure-functions）、postgres/cosmos 中的資料庫網路（使用對應的技能）、成本分析（使用 azure-cost-optimization）、資源部署（使用 azure-deploy）
