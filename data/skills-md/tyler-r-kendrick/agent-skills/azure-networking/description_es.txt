Servicios de red de Azure que incluyen redes virtuales, puntos de conexión privados, equilibradores de carga, puerta de enlace de aplicaciones, puerta principal y DNS. Cubre topología radial, patrones de puntos finales privados y capas de seguridad de red.
USO PARA: configuración de red virtual, configuración de punto final privado, topología radial de centro, emparejamiento de VNet, grupos de seguridad de red, reglas de firewall de Azure, configuración del balanceador de carga, configuración de Application Gateway, Azure Front Door, configuración de vínculo privado, configuración de DNS, aislamiento de red
NO UTILIZAR PARA: configuración de recursos informáticos (use azure-create-app), redes de Azure Functions (use azure-functions), redes de bases de datos dentro de postgres/cosmos (use las habilidades respectivas), análisis de costos (use azure-cost-optimization), implementación de recursos (use azure-deploy)
