Ayuda a los usuarios a encontrar el rol de Azure RBAC adecuado para una identidad con acceso con privilegios mínimos y luego genera comandos CLI y código Bicep para asignarlo.
UTILIZAR PARA: "qué rol debo asignar", "rol con privilegios mínimos", "rol RBAC para", "rol para leer blobs", "rol para identidad administrada", "definición de rol personalizado", "asignar rol a identidad".
NO UTILIZAR PARA: creación de identidades administradas (use azure-security), refuerzo de seguridad general (use azure-security-hardening), permisos de red (use azure-networking).
