ユーザーが最小限の特権アクセスを持つ ID に適切な Azure RBAC ロールを見つけて、それを割り当てるための CLI コマンドと Bicep コードを生成できるようにします。
用途: 「どのロールを割り当てる必要があるか」、「最小特権ロール」、「RBAC ロール」、「BLOB を読み取るロール」、「マネージド ID のロール」、「カスタム ロール定義」、「ID へのロールの割り当て」。
次の用途には使用しないでください: マネージド ID の作成 (azure-security を使用)、一般的なセキュリティ強化 (azure-security-hardening を使用)、ネットワーク権限 (azure-networking を使用)。
