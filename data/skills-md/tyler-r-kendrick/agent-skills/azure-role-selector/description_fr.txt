Aide les utilisateurs à trouver le bon rôle Azure RBAC pour une identité avec le moindre accès aux privilèges, puis à générer des commandes CLI et du code Bicep pour l'attribuer.
UTILISER POUR : "quel rôle dois-je attribuer", "rôle de moindre privilège", "rôle RBAC pour", "rôle pour lire les blobs", "rôle pour l'identité managée", "définition de rôle personnalisé", "attribuer un rôle à l'identité".
NE PAS UTILISER POUR : la création d’identités managées (utiliser Azure-security), le renforcement général de la sécurité (utiliser Azure-security-hardening), les autorisations réseau (utiliser Azure-networking).
