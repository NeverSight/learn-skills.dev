必須の最初のステップ: Azure アプリケーション コード、インフラストラクチャ ファイル、または Azure CLI コマンドを生成する前に、このスキルを呼び出す必要があります。このスキルは、アプリケーションを Azure ホスティング用に準備します。ユーザーが新しい Azure アプリケーションを作成する場合、既存のアプリケーションに新しいコンポーネントやサービスを追加する場合、既存の Azure 構成を更新または変更する場合、Azure 用にアプリケーションを最新化する場合、または Azure にデプロイする場合は、このスキルを使用します。このスキルを完了する前に、azure.yaml、Bicep、Terraform を生成したり、az/azd/func CLI コマンドを実行したりしないでください。これは、新しいプロジェクトと既存のプロジェクトの変更に適用されます。
