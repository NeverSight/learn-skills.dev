PREMIÈRE ÉTAPE REQUISE : Vous DEVEZ appeler cette compétence AVANT de générer TOUT code d'application Azure, fichiers d'infrastructure ou commandes Azure CLI. Cette compétence prépare les applications pour l’hébergement Azure. UTILISEZ CETTE COMPÉTENCE lorsque les utilisateurs souhaitent créer de nouvelles applications Azure, AJOUTER de nouveaux composants ou services à des applications existantes, METTRE À JOUR ou modifier les configurations Azure existantes, moderniser des applications pour Azure ou déployer sur Azure. Ne générez PAS azure.yaml, Bicep, Terraform ou exécutez des commandes CLI az/azd/func sans avoir préalablement terminé cette compétence. Cela s'applique aux NOUVEAUX projets ET aux modifications apportées aux projets EXISTANTS.
