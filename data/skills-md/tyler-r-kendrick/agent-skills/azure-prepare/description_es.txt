PRIMER PASO OBLIGATORIO: DEBE invocar esta habilidad ANTES de generar CUALQUIER código de aplicación de Azure, archivos de infraestructura o comandos de la CLI de Azure. Esta habilidad prepara aplicaciones para alojamiento de Azure. USE ESTA HABILIDAD cuando los usuarios quieran crear nuevas aplicaciones de Azure, AGREGAR nuevos componentes o servicios a aplicaciones existentes, ACTUALIZAR o modificar configuraciones de Azure existentes, modernizar aplicaciones para Azure o implementarlas en Azure. NO genere azure.yaml, Bicep, Terraform ni ejecute comandos CLI az/azd/func sin completar primero esta habilidad. Esto se aplica a proyectos NUEVOS Y cambios en proyectos EXISTENTES.
