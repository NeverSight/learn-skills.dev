Conseils pour le développement de logiciels Python dans l’ensemble de l’écosystème. Couvre la configuration du projet, la gestion des packages, le développement CLI et les bibliothèques populaires.
UTILISER POUR : configuration d'un projet Python, choix des outils Python, compréhension des fonctionnalités du langage Python, environnements virtuels, astuces de type, programmation asynchrone, stratégie de packaging
NE PAS UTILISER POUR : des analyses approfondies d'outils spécifiques (utiliser les sous-compétences : système de projet, gestion de packages, cli, packages)
