Guida per lo sviluppo di software Python nell'ecosistema. Copre la configurazione del progetto, la gestione dei pacchetti, lo sviluppo della CLI e le librerie più diffuse.
UTILIZZO PER: configurazione del progetto Python, scelta degli strumenti Python, comprensione delle funzionalità del linguaggio Python, ambienti virtuali, suggerimenti sul tipo, programmazione asincrona, strategia di packaging
NON UTILIZZARE PER: approfondimenti su strumenti specifici (utilizzare le sottocompetenze: sistema di progetto, gestione dei pacchetti, cli, pacchetti)
