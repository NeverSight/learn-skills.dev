Instrumente aplicaciones web para enviar datos de telemetría a Azure Application Insights para su observabilidad y monitoreo. USO PARA: instrumentar la aplicación con información de la aplicación, agregar instrumentación de la aplicación, configurar la información de la aplicación, configurar el monitoreo de telemetría, habilitar la instrumentación automática de la información de la aplicación, agregar observabilidad a la aplicación web Azure, instrumentar la aplicación web para enviar datos a la información de la aplicación, configurar la telemetría para el servicio de la aplicación. NO UTILIZAR PARA: monitoreo que no sea de Azure (use CloudWatch para AWS, Datadog para terceros), análisis de registros (use azure-kusto), monitoreo de costos (use azure-cost-optimization), monitoreo de seguridad (use azure-security).
