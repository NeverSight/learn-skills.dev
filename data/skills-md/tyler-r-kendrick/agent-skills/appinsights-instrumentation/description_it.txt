Strumentare le applicazioni Web per inviare dati di telemetria ad Azure Application Insights per l'osservabilità e il monitoraggio. UTILIZZARE PER: strumentare l'app con approfondimenti dell'app, aggiungere strumentazione appinsights, configurare approfondimenti dell'applicazione, impostare il monitoraggio della telemetria, abilitare la strumentazione automatica di approfondimenti dell'app, aggiungere osservabilità all'app Web di Azure, strumentare l'app web per inviare dati agli approfondimenti dell'app, configurare la telemetria per il servizio app. NON UTILIZZARE PER: monitoraggio non Azure (utilizzare CloudWatch per AWS, Datadog per terze parti), analisi dei log (utilizzare azure-kusto), monitoraggio dei costi (utilizzare azure-cost-optimization), monitoraggio della sicurezza (utilizzare azure-security).
