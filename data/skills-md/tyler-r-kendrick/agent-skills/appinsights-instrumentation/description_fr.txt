Instrumentez les applications Web pour envoyer des données de télémétrie à Azure Application Insights à des fins d’observabilité et de surveillance. UTILISER POUR : instrumenter une application avec des informations sur les applications, ajouter une instrumentation Appinsights, configurer des informations sur les applications, configurer la surveillance de la télémétrie, activer l'auto-instrumentation des informations sur les applications, ajouter de l'observabilité à l'application Web Azure, instrumenter une application Web pour envoyer des données aux informations sur les applications, configurer la télémétrie pour le service d'application. NE PAS UTILISER POUR : la surveillance non-Azure (utilisez CloudWatch pour AWS, Datadog pour les tiers), l'analyse des journaux (utilisez azure-kusto), la surveillance des coûts (utilisez azure-cost-optimization), la surveillance de la sécurité (utilisez azure-security).
