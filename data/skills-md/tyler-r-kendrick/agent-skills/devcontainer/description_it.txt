Da utilizzare durante la configurazione dei contenitori di sviluppo o di GitHub Codespace. Copre lo schema devcontainer.json, le funzionalità, gli hook del ciclo di vita, il port forwarding e le personalizzazioni.
UTILIZZARE PER: configurazione devcontainer.json, configurazione di GitHub Codespaces, hook del ciclo di vita, port forwarding, personalizzazioni VS Code, funzionalità del contenitore di sviluppo
NON UTILIZZARE PER: impostazione di un linguaggio specifico (utilizzare le competenze secondarie: dotnet, python, typescript), configurazione Docker-in-Docker (utilizzare docker-in-docker), servizi sidecar (utilizzare spazi di lavoro multi-contenitore)
