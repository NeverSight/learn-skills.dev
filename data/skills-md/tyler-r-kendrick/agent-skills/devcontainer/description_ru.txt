Используйте при настройке контейнеров разработки или кодовых пространств GitHub. Охватывает схему, функции, функции жизненного цикла devcontainer.json, переадресацию портов и настройки.
ИСПОЛЬЗОВАНИЕ ДЛЯ: конфигурации devcontainer.json, настройки GitHub Codespaces, перехватчиков жизненного цикла, переадресации портов, настройки VS Code, функций контейнера разработки.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: настройки конкретного языка (используйте дополнительные навыки: dotnet, python, typescript), конфигурации Docker-in-Docker (используйте docker-in-docker), дополнительных сервисов (используйте рабочие области с несколькими контейнерами).
