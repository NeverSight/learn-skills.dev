À utiliser lors de la configuration des conteneurs de développement ou des espaces de codes GitHub. Couvre le schéma devcontainer.json, les fonctionnalités, les hooks de cycle de vie, la redirection de port et les personnalisations.
UTILISER POUR : configuration devcontainer.json, configuration des espaces de codes GitHub, hooks de cycle de vie, redirection de port, personnalisations VS Code, fonctionnalités du conteneur de développement
NE PAS UTILISER POUR : la configuration d'un langage spécifique (utiliser les sous-compétences : dotnet, python, typescript), la configuration Docker-in-Docker (utiliser docker-in-docker), les services side-car (utiliser des espaces de travail multi-conteneurs)
