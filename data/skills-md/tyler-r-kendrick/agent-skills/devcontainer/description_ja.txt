用途: devcontainer.json 構成、GitHub コードスペースのセットアップ、ライフサイクル フック、ポート転送、VS Code のカスタマイズ、開発コンテナー機能
次の用途には使用しないでください: 特定の言語セットアップ (サブスキル: dotnet、python、typescript を使用)、Docker-in-Docker 構成 (docker-in-docker を使用)、サイドカー サービス (マルチコンテナー ワークスペースを使用)
