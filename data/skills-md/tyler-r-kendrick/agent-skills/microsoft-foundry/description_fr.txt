Utilisez cette compétence pour travailler avec Microsoft Foundry (Azure AI Foundry) : déployez des modèles d'IA à partir du catalogue, créez des applications RAG avec des index de connaissances, créez et évaluez des agents d'IA.
UTILISER POUR : Microsoft Foundry, AI Foundry, déployer un modèle, un catalogue de modèles, RAG, un index de connaissances, créer un agent, évaluer un agent, surveiller un agent.
NE PAS UTILISER POUR : Azure Functions (utilisez azure-functions), App Service (utilisez azure-create-app).
