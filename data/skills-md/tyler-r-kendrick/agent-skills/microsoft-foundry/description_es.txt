Utilice esta habilidad para trabajar con Microsoft Foundry (Azure AI Foundry): implemente modelos de IA desde el catálogo, cree aplicaciones RAG con índices de conocimiento, cree y evalúe agentes de IA.
USO PARA: Microsoft Foundry, AI Foundry, modelo de implementación, catálogo de modelos, RAG, índice de conocimiento, creación de agente, evaluación de agente, monitoreo de agente.
NO UTILIZAR PARA: Azure Functions (use azure-functions), App Service (use azure-create-app).
