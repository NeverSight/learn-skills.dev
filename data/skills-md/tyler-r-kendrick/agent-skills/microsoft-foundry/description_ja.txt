このスキルを使用して Microsoft Foundry (Azure AI Foundry) を操作します。カタログから AI モデルをデプロイし、ナレッジ インデックスを使用して RAG アプリケーションを構築し、AI エージェントを作成して評価します。
用途: Microsoft Foundry、AI Foundry、モデルの展開、モデル カタログ、RAG、ナレッジ インデックス、エージェントの作成、エージェントの評価、エージェントの監視。
次の用途には使用しないでください: Azure Functions (azure-functions を使用)、App Service (azure-create-app を使用)。
