Используйте этот навык для работы с Microsoft Foundry (Azure AI Foundry): развертывайте модели ИИ из каталога, создавайте приложения RAG с индексами знаний, создавайте и оценивайте агенты ИИ.
ИСПОЛЬЗОВАНИЕ ДЛЯ: Microsoft Foundry, AI Foundry, модели развертывания, каталога моделей, RAG, индекса знаний, создания агента, оценки агента, мониторинга агента.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: Функций Azure (используйте функции azure), Службы приложений (используйте azure-create-app).
