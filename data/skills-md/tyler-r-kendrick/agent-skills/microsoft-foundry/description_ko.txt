이 기술을 사용하여 Microsoft Foundry(Azure AI Foundry)와 작업할 수 있습니다. 카탈로그에서 AI 모델을 배포하고, 지식 인덱스를 사용하여 RAG 애플리케이션을 구축하고, AI 에이전트를 생성 및 평가합니다.
사용 대상: Microsoft Foundry, AI Foundry, 모델 배포, 모델 카탈로그, RAG, 지식 인덱스, 에이전트 생성, 에이전트 평가, 에이전트 모니터링.
사용하지 마십시오: Azure Functions(azure-functions 사용), App Service(azure-create-app 사용).
