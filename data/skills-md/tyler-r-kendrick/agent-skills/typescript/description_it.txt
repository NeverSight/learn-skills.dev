Da utilizzare quando si lavora con progetti, strumenti ed ecosistemi TypeScript. Copre il sistema di tipi, la configurazione del progetto, la gestione dei pacchetti, lo sviluppo della CLI e i pacchetti delle librerie.
UTILIZZO PER: funzionalità del linguaggio TypeScript, scelta degli strumenti di creazione, gestori di pacchetti, struttura del progetto, guida al sistema di tipi, selezione del runtime
NON UTILIZZARE PER: dettagli specifici sulla configurazione dello strumento (utilizzare le sottocompetenze: sistema di progetto, gestione dei pacchetti, cli, pacchetti)
