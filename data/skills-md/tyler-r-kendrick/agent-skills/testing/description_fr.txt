À utiliser pour choisir une stratégie de test, dimensionner correctement la couverture de test ou comprendre les catégories de test. Couvre le modèle Test Trophy, les compromis entre les types de tests et des conseils sur l'équilibrage des tests d'analyse statique, d'unité, d'intégration et de bout en bout.
UTILISER POUR : stratégie de test, Test Trophy, sélection du type de test, dimensionnement approprié de la couverture de test, équilibrage des catégories de test, choix des outils de test, architecture d'automatisation des tests
NE PAS UTILISER POUR : l'implémentation d'une catégorie de test spécifique (utiliser l'analyse statique, les tests unitaires, les tests d'intégration, les tests e2e, etc.), la rédaction de spécifications BDD (utiliser les spécifications/documentation/cornichon)
