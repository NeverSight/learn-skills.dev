Verwenden Sie es, wenn Sie eine Teststrategie auswählen, die Testabdeckung richtig dimensionieren oder Testkategorien verstehen. Behandelt das Test Trophy-Modell, Testtyp-Kompromisse und Anleitungen zum Ausgleich von statischen Analysen, Unit-, Integrations- und End-to-End-Tests.
VERWENDUNG FÜR: Teststrategie, Testtrophäe, Auswahl des Testtyps, richtige Dimensionierung der Testabdeckung, Ausbalancieren von Testkategorien, Auswahl von Testtools, Testautomatisierungsarchitektur
NICHT VERWENDEN FÜR: spezifische Testkategorieimplementierung (verwenden Sie statische Analyse, Unit-Testing, Integrationstests, E2E-Testing usw.), BDD-Spezifikationserstellung (verwenden Sie specs/documentation/gherkin).
