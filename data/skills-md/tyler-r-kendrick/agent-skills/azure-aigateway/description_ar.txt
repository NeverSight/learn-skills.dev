قم بتكوين Azure API Management (APIM) كبوابة AI لتأمين نماذج الذكاء الاصطناعي وخوادم MCP والوكلاء ومراقبتها والتحكم فيها. يساعد في تحديد المعدل والتخزين المؤقت الدلالي وأمان المحتوى وموازنة التحميل.
الاستخدام لـ: بوابة AI، APIM، بوابة الإعداد، بوابة التكوين، إضافة بوابة، بوابة النموذج، خادم MCP، حد المعدل، حد الرمز المميز، ذاكرة التخزين المؤقت الدلالية، أمان المحتوى، توازن التحميل، استيراد OpenAPI، تحويل API إلى MCP.
لا تستخدم من أجل: نشر النماذج (استخدم microsoft-foundry)، ووظائف Azure (استخدم وظائف azure)، وقواعد البيانات (استخدم azure-postgres).
