**重要**: Azure リソースをデプロイする前に azure-validate を実行します。 Azure の展開の準備が整っているかを検証します。用途: アプリケーションを Azure にデプロイする準備ができているかどうかの評価、azure.yaml または Bicep 構成ファイルの検証、Azure プリフライト チェックの実行、デプロイ プレビューと what-if 分析のレビュー、Azure インフラストラクチャ構成の確認、Azure デプロイ エラーのトラブルシューティング。新しいアプリケーションの作成 (azure-prepare を使用)、デプロイメントの実行 (azure-deploy を使用) には使用しないでください。
