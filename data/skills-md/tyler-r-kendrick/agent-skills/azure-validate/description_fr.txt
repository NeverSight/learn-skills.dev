**CRITIQUE** : exécutez azure-validate avant de déployer des ressources Azure. Valide la préparation au déploiement Azure. UTILISER POUR : évaluer si l'application est prête à être déployée sur Azure, valider les fichiers de configuration azure.yaml ou Bicep, exécuter des vérifications en amont Azure, examiner les aperçus de déploiement et l'analyse de simulation, vérifier la configuration de l'infrastructure Azure, résoudre les erreurs de déploiement Azure. NE PAS UTILISER POUR : créer de nouvelles applications (utiliser azure-prepare), exécuter des déploiements (utiliser azure-deploy).
