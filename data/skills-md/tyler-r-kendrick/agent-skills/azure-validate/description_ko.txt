**중요**: Azure 리소스를 배포하기 전에 azure-validate를 실행하세요. Azure 배포 준비 상태를 확인합니다. 사용 대상: 애플리케이션이 Azure에 배포할 준비가 되었는지 평가하고, azure.yaml 또는 Bicep 구성 파일의 유효성을 검사하고, Azure 실행 전 검사를 실행하고, 배포 미리 보기 및 가상 분석을 검토하고, Azure 인프라 구성을 확인하고, Azure 배포 오류 문제를 해결합니다. 사용하지 마십시오: 새 애플리케이션 생성(azure-prepare 사용), 배포 실행(azure-deploy 사용).
