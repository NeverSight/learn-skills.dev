**KRITISCH**: Führen Sie „azure-validate“ aus, bevor Sie Azure-Ressourcen bereitstellen. Validiert die Bereitschaft zur Azure-Bereitstellung. VERWENDUNG FÜR: Bewerten, ob die Anwendung für die Bereitstellung in Azure bereit ist, Validieren von azure.yaml- oder Bicep-Konfigurationsdateien, Ausführen von Azure-Preflight-Prüfungen, Überprüfen von Bereitstellungsvorschauen und Was-wäre-wenn-Analysen, Überprüfen der Azure-Infrastrukturkonfiguration, Beheben von Azure-Bereitstellungsfehlern. NICHT VERWENDEN FÜR: Erstellen neuer Anwendungen (verwenden Sie azure-prepare), Ausführen von Bereitstellungen (verwenden Sie azure-deploy).
