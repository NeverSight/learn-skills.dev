**CRITICO**: eseguire azure-validate prima di distribuire le risorse di Azure. Convalida la disponibilità della distribuzione di Azure. UTILIZZO PER: valutare se l'applicazione è pronta per la distribuzione in Azure, convalidare file di configurazione azure.yaml o Bicep, eseguire controlli preliminari di Azure, esaminare anteprime di distribuzione e analisi what-if, verificare la configurazione dell'infrastruttura di Azure, risolvere gli errori di distribuzione di Azure. NON UTILIZZARE PER: creazione di nuove applicazioni (usare azure-prepare), esecuzione di distribuzioni (usare azure-deploy).
