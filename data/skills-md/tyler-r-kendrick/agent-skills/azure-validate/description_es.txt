**CRÍTICO**: Ejecute azure-validate antes de implementar recursos de Azure. Valida la preparación para la implementación de Azure. UTILIZAR PARA: evaluar si la aplicación está lista para implementarse en Azure, validar los archivos de configuración azure.yaml o Bicep, ejecutar comprobaciones previas de Azure, revisar vistas previas de implementación y análisis hipotéticos, verificar la configuración de la infraestructura de Azure, solucionar errores de implementación de Azure. NO UTILIZAR PARA: crear nuevas aplicaciones (use azure-prepare), ejecutar implementaciones (use azure-deploy).
