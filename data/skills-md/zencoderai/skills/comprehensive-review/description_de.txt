Umfassende Codeüberprüfung mithilfe paralleler spezialisierter Subagenten. Verwenden Sie diese Option, wenn Sie eine gründliche Codeüberprüfung wünschen, die Architektur, Sicherheit, Leistung, Codequalität, Anforderungskonformität und Fehler abdeckt. Funktioniert mit GitHub PR-Links ODER lokalen Branch-Änderungen. Wenn eine PR-URL angegeben wird, werden PR-Details abgerufen und Kommentare gepostet. Wenn keine PR bereitgestellt wird, wird der Unterschied zwischen dem aktuellen Zweig und seinem Basiszweig sowie allen nicht festgeschriebenen Änderungen überprüft. WICHTIG: Diese Fähigkeit ist kostspielig. Verwenden Sie sie nicht, es sei denn, der Benutzer fordert ausdrücklich dazu auf.
