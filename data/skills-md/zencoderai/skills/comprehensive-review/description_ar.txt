مراجعة شاملة للكود باستخدام وكلاء فرعيين متخصصين متوازيين. استخدمه عندما تريد مراجعة شاملة للتعليمات البرمجية تغطي البنية والأمان والأداء وجودة التعليمات البرمجية والامتثال للمتطلبات والأخطاء. يعمل مع روابط GitHub PR أو تغييرات الفرع المحلي. إذا تم توفير عنوان URL للعلاقات العامة، فإنه يجلب تفاصيل العلاقات العامة ويمكنه نشر التعليقات. إذا لم يتم توفير علاقات عامة، قم بمراجعة الفرق بين الفرع الحالي وفرعه الأساسي بالإضافة إلى أي تغييرات غير ملتزم بها. هام: هذه المهارة مكلفة، لا تستخدمها إلا إذا طلب المستخدم صراحةً استخدام هذه المهارة.
