Révision complète du code à l’aide de sous-agents spécialisés parallèles. À utiliser lorsque vous souhaitez une révision approfondie du code couvrant l'architecture, la sécurité, les performances, la qualité du code, la conformité aux exigences et les bogues. Fonctionne avec les liens GitHub PR OU les modifications de branche locale. Si une URL PR est fournie, récupère les détails du PR et peut publier des commentaires. Si aucun PR n'est fourni, examine la différence entre la branche actuelle et sa branche de base ainsi que toutes les modifications non validées. IMPORTANT : cette compétence est coûteuse, ne l'utilisez pas à moins que l'utilisateur ne le demande explicitement.
