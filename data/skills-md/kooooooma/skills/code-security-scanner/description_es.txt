Escanee repositorios de códigos en busca de amenazas a la seguridad, incluida la filtración de datos, puertas traseras, inyección de códigos maliciosos, riesgos de la cadena de dependencia y acceso a archivos confidenciales. Utilice esta habilidad cuando los usuarios quieran auditar una base de código (especialmente proyectos TypeScript/JavaScript/Node.js) en busca de vulnerabilidades de seguridad, detectar malware oculto, revisar las dependencias de npm para detectar ataques a la cadena de suministro, comprobar si hay fugas de credenciales o realizar una revisión de seguridad previa a la implementación. Activadores de solicitudes como "buscar código malicioso", "auditoría de seguridad", "comprobar puertas traseras", "revisar dependencias en busca de vulnerabilidades", "detectar filtración de datos".
