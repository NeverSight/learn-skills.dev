コード リポジトリをスキャンして、データ漏洩、バックドア、悪意のあるコード インジェクション、依存関係チェーン リスク、機密ファイル アクセスなどのセキュリティ脅威を検出します。このスキルは、ユーザーがコードベース (特に TypeScript/JavaScript/Node.js プロジェクト) のセキュリティ脆弱性を監査する場合、隠れたマルウェアを検出する場合、サプライ チェーン攻撃に対する npm の依存関係を確認する場合、資格情報の漏洩を確認する場合、または展開前のセキュリティ レビューを実行する場合に使用します。 「悪意のあるコードのスキャン」、「セキュリティ監査」、「バックドアのチェック」、「依存関係の脆弱性の確認」、「データ漏洩の検出」などのリクエストをトリガーします。
