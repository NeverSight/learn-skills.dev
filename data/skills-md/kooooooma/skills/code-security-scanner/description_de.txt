Scannen Sie Code-Repositorys auf Sicherheitsbedrohungen, einschließlich Datenexfiltration, Hintertüren, Einschleusung von bösartigem Code, Abhängigkeitskettenrisiken und Zugriff auf vertrauliche Dateien. Verwenden Sie diese Fähigkeit, wenn Benutzer eine Codebasis (insbesondere TypeScript-/JavaScript-/Node.js-Projekte) auf Sicherheitslücken prüfen, versteckte Malware erkennen, NPM-Abhängigkeiten auf Lieferkettenangriffe überprüfen, auf Anmeldeinformationslecks prüfen oder eine Sicherheitsüberprüfung vor der Bereitstellung durchführen möchten. Wird bei Anfragen wie „Auf schädlichen Code scannen“, „Sicherheitsüberprüfung“, „Auf Hintertüren prüfen“, „Abhängigkeiten auf Schwachstellen überprüfen“ und „Datenexfiltration erkennen“ ausgelöst.
