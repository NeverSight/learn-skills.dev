Analysez les référentiels de code à la recherche de menaces de sécurité, notamment l'exfiltration de données, les portes dérobées, l'injection de code malveillant, les risques de chaîne de dépendance et l'accès aux fichiers sensibles. Utilisez cette compétence lorsque les utilisateurs souhaitent auditer une base de code (en particulier les projets TypeScript/JavaScript/Node.js) pour détecter les vulnérabilités de sécurité, détecter les logiciels malveillants cachés, examiner les dépendances npm pour les attaques de la chaîne d'approvisionnement, vérifier les fuites d'informations d'identification ou effectuer un examen de sécurité avant le déploiement. Se déclenche sur des requêtes telles que "rechercher du code malveillant", "audit de sécurité", "vérifier les portes dérobées", "examiner les dépendances pour les vulnérabilités", "détecter l'exfiltration de données".
