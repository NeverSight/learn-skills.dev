Сканируйте репозитории кода на предмет угроз безопасности, включая кражу данных, бэкдоры, внедрение вредоносного кода, риски цепочки зависимостей и доступ к конфиденциальным файлам. Используйте этот навык, когда пользователи хотят проверить базу кода (особенно проекты TypeScript/JavaScript/Node.js) на наличие уязвимостей безопасности, обнаружить скрытое вредоносное ПО, просмотреть зависимости npm на предмет атак в цепочке поставок, проверить наличие утечек учетных данных или выполнить проверку безопасности перед развертыванием. Запускается по таким запросам, как «сканирование на наличие вредоносного кода», «аудит безопасности», «проверка на наличие бэкдоров», «проверка зависимостей на наличие уязвимостей», «обнаружение утечки данных».
