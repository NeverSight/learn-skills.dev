エージェント スキルを完全な GitHub リポジトリにパッケージ化し、skills.sh 経由で配布できるようにします。 README、LICENSE、plugin.json、marketplace.json、.gitignore、および適切なディレクトリ構造を生成します。必要に応じて、git リポジトリを初期化し、GitHub リポジトリを作成します。このスキルは、スキルを公開するとき、配布用にスキルをパッケージ化するとき、スキル リポジトリを準備するとき、またはユーザーが「スキルの公開」、「スキルのパッケージ化」、「スキルのリリース」、「発行スキル」、「打包スキル」と言うときに使用する必要があります。
