Empaquete una habilidad de agente en un repositorio completo de GitHub listo para su distribución a través de skills.sh. Genera README, LICENSE, plugin.json, Marketplace.json, .gitignore y la estructura de directorio adecuada. Opcionalmente, inicializa un repositorio de git y crea un repositorio de GitHub. Esta habilidad debe usarse al publicar una habilidad, empaquetar una habilidad para su distribución, preparar un repositorio de habilidades o cuando el usuario dice 'publicar habilidad', 'empaquetar habilidad', 'liberar habilidad', '发布技能', '打包 habilidad'.
