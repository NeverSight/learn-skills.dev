Упакуйте навык агента в полный репозиторий GitHub, готовый к распространению черезkills.sh. Создает README, ЛИЦЕНЗИЮ, плагин.json, marketplace.json, .gitignore и соответствующую структуру каталогов. При необходимости инициализирует репозиторий git и создает репозиторий GitHub. Этот навык следует использовать при публикации навыка, упаковке навыка для распространения, подготовке репозитория навыка или когда пользователь говорит «публиковать навык», «упаковать навык», «выпустить навык», «发布技能», «打包 навык».
