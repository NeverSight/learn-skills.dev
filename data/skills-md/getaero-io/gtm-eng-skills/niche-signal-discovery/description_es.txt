Descubra señales propias de nicho que diferencian las cuentas cerradas ganadas de las cerradas perdidas para el análisis de ICP. Úselo cuando el usuario proporciona listas de dominios de clientes ganados/perdidos y desea señales diferenciales (contenido del sitio web, ofertas de trabajo, pila tecnológica, marcadores de madurez) para crear modelos de puntuación de cuentas y criterios de prospección. Activadores: análisis de ICP, señales de nicho, análisis de ganancias y pérdidas, señales diferenciales, descubrimiento de señales, informe de señales de ICP, señales de puntuación de cuentas, puntuación de clientes potenciales, señales propias, señales de compradores. Antes de leer este archivo, primero lea gtm-meta-skill para comprender la herramienta Deepline CLI y cómo usarla. Luego lea este archivo para obtener orientación sobre la tarea.
