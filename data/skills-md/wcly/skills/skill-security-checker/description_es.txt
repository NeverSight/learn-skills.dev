检查 Skill/代码仓库是否存在安全风险。Cuando el usuario quiere comprobar si es seguro descargar o usar una habilidad, un repositorio de GitHub, un paquete npm o un código local. Esto incluye detectar código malicioso, malware, robo de claves, modificación de variables de entorno, comportamiento sospechoso de la red y evaluar la reputación del repositorio (estrellas, bifurcaciones, contribuyentes, antigüedad). Utilice esta habilidad siempre que el usuario mencione verificar habilidades para detectar riesgos de seguridad, escanear repositorios en busca de malware, verificar la seguridad del código, verificar paquetes npm en busca de amenazas o preguntar si una descarga es segura.
