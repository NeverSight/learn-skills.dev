检查 Skill/代码仓库是否存在安全风险.Wenn der Benutzer überprüfen möchte, ob ein Skill, ein GitHub-Repository, ein npm-Paket oder ein lokaler Code sicher heruntergeladen oder verwendet werden kann. Dazu gehört die Erkennung von bösartigem Code, Malware, Schlüsseldiebstahl, Änderung von Umgebungsvariablen, verdächtigem Netzwerkverhalten und die Bewertung der Repository-Reputation (Sterne, Forks, Mitwirkende, Alter). Verwenden Sie diese Fertigkeit immer dann, wenn der Benutzer die Überprüfung von Fertigkeiten auf Sicherheitsrisiken, das Scannen von Repositorys auf Malware, die Überprüfung der Codesicherheit, die Überprüfung von NPM-Paketen auf Bedrohungen oder die Frage, ob ein Download sicher ist, erwähnt.
