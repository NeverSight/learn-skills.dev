检查 Skill/代码仓库是否存在安全风险。Lorsque l'utilisateur souhaite vérifier si une compétence, un référentiel GitHub, un package npm ou un code local peut être téléchargé ou utilisé en toute sécurité. Cela inclut la détection des codes malveillants, des logiciels malveillants, du vol de clés, de la modification des variables d'environnement, des comportements réseau suspects et l'évaluation de la réputation du référentiel (étoiles, forks, contributeurs, âge). Utilisez cette compétence chaque fois que l'utilisateur mentionne vérifier les compétences pour les risques de sécurité, analyser les référentiels à la recherche de logiciels malveillants, vérifier la sécurité du code, vérifier les packages npm pour les menaces ou demander si un téléchargement est sûr.
