عندما يريد المستخدم التحقق مما إذا كانت إحدى المهارة أو مستودع GitHub أو حزمة npm أو الكود المحلي آمنة للتنزيل أو الاستخدام. يتضمن ذلك اكتشاف التعليمات البرمجية الضارة، والبرامج الضارة، وسرقة المفاتيح، وتعديل متغيرات البيئة، وسلوك الشبكة المشبوه، وتقييم سمعة المستودع (النجوم، والشوكات، والمساهمين، والعمر). استخدم هذه المهارة عندما يذكر المستخدم مهارات التحقق من المخاطر الأمنية، أو فحص المستودعات بحثًا عن البرامج الضارة، أو التحقق من سلامة التعليمات البرمجية، أو التحقق من حزم npm بحثًا عن التهديدات، أو السؤال عما إذا كان التنزيل آمنًا.
