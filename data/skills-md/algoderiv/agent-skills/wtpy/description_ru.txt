WonderTrader/wtpy Комплексное руководство по разработке количественной торговли, объединяющее официальную документацию, заметки сообщества и практические примеры. Охватывает полный рабочий процесс разработки стратегии, тестирования на исторических данных, моделирования, торговли в реальном времени, управления данными, а также мониторинга и операций для платформы wtpy Python и ядра WonderTrader C++.
