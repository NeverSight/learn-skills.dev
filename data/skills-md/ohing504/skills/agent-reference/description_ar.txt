قم بإنشاء تقارير فحص مرجعية موضوعية حول المستخدم من بيانات تعاون الذكاء الاصطناعي الحقيقية - سجل الجلسة، وسجلات git، وملف تعريف GitHub، وملفات الذاكرة. مثل زميل يكتب مرجعًا احترافيًا، ولكنه يرتكز على العمل المشترك الفعلي. يُستخدم عندما يطلب المستخدم تقييمه كمطور، أو يريد خطابًا مرجعيًا، أو تحليلًا لأسلوب العمل، أو مقدمًا من خلال محتوى وكلائي، أو الإعداد للمقابلة من سجل التعاون، أو موضوعات المدونة من المناقشات السابقة. المحفزات: كتابة مرجع، تحليل أنماط عملي، ما رأيك فيي، أو اختياري، أو اختياري. ليس للمراجعة العامة للتعليمات البرمجية، أو مستندات الهندسة المعمارية، أو خطابات التقديم، أو تحليل قاعدة التعليمات البرمجية فقط.
