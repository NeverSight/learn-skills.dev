Создавайте объективные отчеты о проверке рекомендаций о пользователе на основе реальных данных совместной работы ИИ — истории сеансов, журналов git, профиля GitHub и файлов памяти. Как коллега, пишущий профессиональную характеристику, но основанный на реальной совместной работе. Используйте всякий раз, когда пользователь просит оценить его как разработчика, хочет получить рекомендательное письмо, анализ стиля работы, контент, представленный моими агентами, подготовку к собеседованию на основе истории сотрудничества или темы блога из прошлых обсуждений. Запускается: написать рекомендацию, проанализировать мою работу, что вы обо мне думаете, 나에 대한 레퍼런스 써줘, 내 작업 스타일 분석해줘. Не для общего обзора кода, документации по архитектуре, сопроводительных писем или анализа только кодовой базы.
