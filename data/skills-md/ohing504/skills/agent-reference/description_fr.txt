Générez des rapports objectifs de vérification des références sur l'utilisateur à partir de données réelles de collaboration IA : historique de session, journaux Git, profil GitHub et fichiers mémoire. Comme un collègue rédigeant une référence professionnelle, mais ancré dans un véritable travail partagé. À utiliser chaque fois que l'utilisateur demande à l'évaluer en tant que développeur, souhaite une lettre de référence, une analyse du style de travail, un contenu présenté par mes agents, une préparation d'entretien à partir de l'historique de collaboration ou des sujets de blog issus de discussions passées. Déclencheurs : rédiger une référence, analyser mes habitudes de travail, que pensez-vous de moi, 나에 대한 레퍼런스 써줘, 내 작업 스타일 분석해줘. Pas pour la révision générale du code, les documents d'architecture, les lettres de motivation ou l'analyse de la base de code uniquement.
