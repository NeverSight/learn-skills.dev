Genere un sitio de cartera personalizado a partir de informes de referencia de agentes e impleméntelo en GitHub Pages. El sitio refleja el estilo de trabajo del usuario observado por sus colaboradores de IA: la IA analiza los informes, propone un concepto de diseño, estructura un sitio de Astro con temas basados ​​en conceptos y lo implementa en {nombre de usuario}.github.io. Utilice esta habilidad siempre que el usuario solicite "crear mi portafolio", "crear un sitio de portafolio", "crear un sitio a partir de mis informes", "implementar en páginas de github", "sitio github.io" o diga cosas como "포트폴리오 사이트 만들어줘", "사이트 배포해줘", o quiera activar informes de referencia de agentes en un sitio web en vivo. También se activa cuando el usuario tiene informes de referencia de agentes listos y desea publicarlos como un sitio, desea un sitio personal generado a partir de datos de colaboración de IA o solicita actualizar o volver a implementar una cartera de agentes existente. NO lo utilice para el desarrollo general de Astro, la creación de sitios web genéricos, el análisis de referencias de agentes sin generación de sitios ni la redacción de currículums que no impliquen la implementación de un sitio.
