Générez un site de portefeuille personnalisé à partir de rapports de référence d'agent et déployez-le sur les pages GitHub. Le site reflète le style de travail de l'utilisateur tel qu'observé par ses collaborateurs IA : l'IA analyse les rapports, propose un concept de conception, étaye un site Astro avec une thématique basée sur le concept et se déploie sur {username}.github.io. Utilisez cette compétence chaque fois que l'utilisateur demande "construire mon portfolio", "créer un site de portfolio", "créer un site à partir de mes rapports", "déployer sur des pages github", "site github.io", ou dit des choses comme "포트폴리오 사이트 만들어줘", "사이트 배포해줘", ou veut activer rapports de référence d'agent sur un site Web en direct. Se déclenche également lorsque l'utilisateur dispose de rapports de référence d'agent prêts et souhaite les publier en tant que site, souhaite qu'un site personnel soit généré à partir de données de collaboration IA ou demande de mettre à jour/redéployer un portefeuille d'agents existant. Ne PAS utiliser pour le développement Astro général, la création de sites Web génériques, l'analyse de référence d'agent sans génération de site ou la rédaction de CV n'impliquant pas le déploiement d'un site.
