Generieren Sie eine personalisierte Portfolio-Site aus Agentenreferenzberichten und stellen Sie sie auf GitHub Pages bereit. Die Site spiegelt den Arbeitsstil des Benutzers wider, wie er von seinen KI-Mitarbeitern beobachtet wird – KI analysiert die Berichte, schlägt ein Designkonzept vor, baut eine Astro-Site mit konzeptbasiertem Thema auf und stellt sie auf {username}.github.io bereit. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer „mein Portfolio erstellen“, „Portfolio-Site erstellen“, „aus meinen Berichten eine Site erstellen“, „auf Github-Seiten bereitstellen“, „github.io-Site“ auffordert oder Dinge wie „포트폴리오 사이트 만들어줘“, „사이트 배포해줘“ sagt oder sich umdrehen möchte Agentenreferenzberichte in eine Live-Website. Wird auch ausgelöst, wenn der Benutzer Agentenreferenzberichte bereithält und diese als Site veröffentlichen möchte, eine persönliche Site aus KI-Zusammenarbeitsdaten erstellen möchte oder die Aktualisierung/Neubereitstellung eines vorhandenen Agentenportfolios anfordert.
