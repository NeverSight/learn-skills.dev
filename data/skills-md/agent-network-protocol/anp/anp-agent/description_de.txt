Das ANP-Protokoll ermöglicht den agentenübergreifenden Skill-Aufruf. Über die dezentrale Identität did:wba ruft es jeden Agenten im ANP-Netzwerk an (z. B. Amap, Hotelbuchung, Express-Lieferanfrage usw.). Diese Fähigkeit wird ausgelöst, wenn Benutzer ANP erwähnen, einen Agenten beauftragen, ein Hotel buchen, Expresslieferungen prüfen, nach Karten suchen oder Routen planen.
