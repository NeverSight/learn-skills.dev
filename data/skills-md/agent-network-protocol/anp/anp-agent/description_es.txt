El protocolo ANP permite la invocación de habilidades entre agentes. Usando la identidad descentralizada did:wba, puede llamar a cualquier Agente en la red ANP (como Amap, reserva de hotel, seguimiento exprés, etc.). Esta habilidad se activa cuando el usuario menciona ANP, invoca a un Agente, reserva un hotel, realiza un seguimiento exprés, consulta mapas o planifica una ruta.
