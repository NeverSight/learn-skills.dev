Le protocole ANP permet l’invocation de compétences entre agents. En utilisant l'identité décentralisée did:wba, il appelle n'importe quel agent du réseau ANP (comme Amap, réservation d'hôtel, demande de livraison express, etc.). Cette compétence est déclenchée lorsque les utilisateurs mentionnent ANP, appellent un agent, réservent un hôtel, vérifient une livraison express, recherchent des cartes ou planifient des itinéraires.
