Il protocollo ANP consente il richiamo delle competenze tra agenti. Utilizzando l'identità decentralizzata did:wba, chiama qualsiasi agente nella rete ANP (ad esempio Amap, prenotazione di hotel, tracciamento del corriere). Si attiva quando gli utenti menzionano ANP, richiamano l'agente, prenotano un hotel, seguono il pacchetto, cercano una mappa o pianificano il percorso.
