يتيح بروتوكول ANP إجراء مكالمات المهارات عبر الوكلاء. باستخدام الهوية اللامركزية فعلت:wba، فإنه يستدعي أي وكيل في شبكة ANP (مثل Amap، وحجز الفنادق، وتتبع البريد السريع، وما إلى ذلك). يتم تفعيل هذه المهارة عندما يذكر المستخدمون ANP، أو يتصلون بالوكيل، أو يحجزون فندقًا، أو يتتبعون البريد السريع، أو يفحصون الخرائط، أو يخططون للطرق.
