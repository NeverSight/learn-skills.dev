単一ファイルの HTML ツールを構築します。ビルド手順を行わずに特定の問題を解決する自己完結型の HTML+JS+CSS アプリケーションです。ユーザーがユーティリティ、コンバータ、ビューア、デバッガ、または小さな対話型 Web ツールの構築を要求するときは常に、このスキルを使用します。また、ユーザーが「... 用のツールを構築してください」、「... 用の簡単なアプリを作成してください」、「... 用の単一ファイルの HTML ページが必要です」、または静的ファイルとしてホストできるものを作成したい場合にもトリガーされます。このスキルは、実用的でユーティリティに重点を置いたツール (ランディング ページ、ポートフォリオ、マーケティング サイトではなく、これらにはフロントエンド デザインを使用します) に特化しています。
