Cree herramientas HTML de un solo archivo: aplicaciones HTML+JS+CSS independientes que resuelven un problema específico sin un paso de compilación. Utilice esta habilidad siempre que el usuario solicite crear una utilidad, un convertidor, un visor, un depurador o cualquier pequeña herramienta web interactiva. También se activa cuando el usuario dice "constrúyeme una herramienta que...", "crea una aplicación rápida para...", "Necesito una página HTML de un solo archivo que...", o quiere crear algo que pueda alojarse como un archivo estático. Esta habilidad es específicamente para herramientas prácticas y centradas en utilidades (no para páginas de destino, carteras o sitios de marketing; utilice el diseño de interfaz para ellos).
