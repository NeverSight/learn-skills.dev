Создавайте однофайловые инструменты HTML — автономные приложения HTML+JS+CSS, которые решают конкретную проблему без этапа сборки. Используйте этот навык всякий раз, когда пользователь просит создать утилиту, конвертер, средство просмотра, отладчик или любой небольшой интерактивный веб-инструмент. Также срабатывает, когда пользователь говорит: «Создайте мне инструмент, который...», «Создайте быстрое приложение для...», «Мне нужна однофайловая HTML-страница, которая...» или хочет создать что-то, что можно было бы разместить как статический файл. Этот навык специально предназначен для практических, ориентированных на утилиты инструментов (не целевых страниц, портфолио или маркетинговых сайтов — для них используйте дизайн интерфейса).
