Crea strumenti HTML a file singolo: applicazioni HTML+JS+CSS autonome che risolvono un problema specifico senza una fase di creazione. Utilizza questa abilità ogni volta che l'utente chiede di creare un'utilità, un convertitore, un visualizzatore, un debugger o qualsiasi piccolo strumento web interattivo. Si attiva anche quando l'utente dice "creami uno strumento che...", "crea un'app veloce per...", "Ho bisogno di una pagina HTML a file singolo che..." o desidera creare qualcosa che possa essere ospitato come file statico. Questa competenza è specifica per strumenti pratici e incentrati sull'utilità (non pagine di destinazione, portfolio o siti di marketing: utilizza il design del frontend per quelli).
