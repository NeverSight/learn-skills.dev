Erstellen Sie Einzeldatei-HTML-Tools – eigenständige HTML+JS+CSS-Anwendungen, die ein bestimmtes Problem ohne einen Build-Schritt lösen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer ein Dienstprogramm, einen Konverter, einen Viewer, einen Debugger oder ein anderes kleines interaktives Webtool erstellen möchte. Wird auch ausgelöst, wenn der Benutzer sagt: „Erstelle mir ein Tool, das ...“, „Erstelle eine schnelle App für ...“, „Ich benötige eine HTML-Seite mit einer einzelnen Datei, die ...“ oder etwas erstellen möchte, das als statische Datei gehostet werden könnte. Diese Fertigkeit ist speziell für praktische, nützlichkeitsorientierte Tools gedacht (nicht für Landingpages, Portfolios oder Marketingseiten – verwenden Sie dafür Frontend-Design).
