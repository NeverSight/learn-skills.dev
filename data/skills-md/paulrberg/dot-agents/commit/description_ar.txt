يجب استخدام هذه المهارة عندما يطلب المستخدم "إجراء تغييرات"، أو "صياغة رسالة التزام"، أو "المرحلة والالتزام"، أو "إجراء تعديلات الجلسة فقط"، أو تشغيل سير عمل التزام باستخدام علامات مثل --all، أو --deep، أو --push. ينشئ التزامات git ذرية باستخدام التحليل الإرشادي وتنسيق الالتزام التقليدي وقواعد التدريج والتحليل العميق الاختياري والدفع الاختياري.
