Administre túneles SSH de forma declarativa. Úselo cuando el usuario desee crear, iniciar, detener, habilitar, deshabilitar o verificar el estado de los túneles de reenvío de puertos SSH. Úselo también cuando el usuario mencione túneles SSH, reenvío de puertos, hosts bastión o conexión a bases de datos/servicios remotos a través de SSH.
