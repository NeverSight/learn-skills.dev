Преобразуйте проекты кода в базы знаний Obsidian. Активируется, когда пользователь упоминает обсидиан, проектную документацию, базу знаний, проект анализа, проект преобразования.

[Необходимо выполнить после активации]:
1. Сначала полностью прочитайте этот файл SKILL.md.
2. Понимание правил написания AI (по умолчанию 00_Inbox/AI/, добавление, унифицированная схема)
3. Выполните ШАГ 0: используйте AskUserQuestion, чтобы запросить у пользователя подтверждение.
4. Запустите сканирование проекта STEP 1 после подтверждения пользователя.
5. Строго следуйте порядку ШАГОВ 0 → 1 → 2 → 3 → 4.

[Запрещенное поведение]:
- Запрещено начинать анализ проекта без прочтения SKILL.md.
- Отключить пропуск подтверждения пользователя ШАГА 0.
- Запретить создание напрямую в 30_Resources (сначала до 00_Inbox/AI/)
- Запрещено самостоятельно определять место вывода
