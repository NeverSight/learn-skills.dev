دليل شامل لتطوير واجهة Python الخلفية وهندسة البرمجيات على نطاق واسع. يُستخدم عند تصميم واجهات برمجة التطبيقات، أو إنشاء خدمات خلفية، أو إنشاء خدمات صغيرة، أو هيكلة مشاريع Python، أو تنفيذ أنماط قاعدة البيانات، أو كتابة تعليمات برمجية غير متزامنة، أو أي مهمة تطوير لواجهة Python الخلفية/جانب الخادم. يغطي الهندسة المعمارية النظيفة، والتصميم المستند إلى المجال، والهندسة المستندة إلى الأحداث، وأنماط FastAPI/Django، وتصميم قاعدة البيانات، واستراتيجيات التخزين المؤقت، وقابلية الملاحظة، والأمن، واستراتيجيات الاختبار، وأنماط النشر لأنظمة الإنتاج واسعة النطاق.
