Guide complet pour le développement backend Python et l'architecture logicielle prêts pour la production à grande échelle. À utiliser lors de la conception d'API, de la création de services backend, de la création de microservices, de la structuration de projets Python, de la mise en œuvre de modèles de base de données, de l'écriture de code asynchrone ou de toute tâche de développement Python backend/côté serveur. Couvre l'architecture propre, la conception pilotée par domaine, l'architecture pilotée par événements, les modèles FastAPI/Django, la conception de bases de données, les stratégies de mise en cache, l'observabilité, la sécurité, les stratégies de test et les modèles de déploiement pour les systèmes de production à grande échelle.
