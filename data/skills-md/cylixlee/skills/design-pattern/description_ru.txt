Применяет принципы объектно-ориентированного проектирования и шаблоны проектирования для создания удобного в сопровождении расширяемого кода. Используйте при создании кода, который требует правильного архитектурного многоуровневого построения, принципов SOLID и соответствующих шаблонов проектирования для решения повторяющихся проблем проектирования программного обеспечения.
