Навык развития Eino Agent Development Kit. Для создания приложений AI Agent, включая ChatModelAgent, рабочие процессы (последовательные/параллельные/циклические), многоагентные системы (Supervisor/PlanExecute), управление человеком в цикле (прерывание/утверждение). Используйте, когда пользователям необходимо создавать агентов, использовать Runner для выполнения, управлять вызовами инструментов, создавать многоагентные системы.
