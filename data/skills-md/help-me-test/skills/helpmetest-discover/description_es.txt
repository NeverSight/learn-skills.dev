Utilice esta habilidad cuando el usuario aún no sepa qué probar. Este es el paso "primero conocer el sitio": para sitios web desconocidos, proyectos nuevos o cualquier situación en la que los artefactos de características/personas aún no existan. Úselo cuando el usuario: proporciona una URL sin ninguna prueba específica en mente, pregunta qué características o flujos tiene un sitio, quiere explorar o recorrer un sitio, es nuevo en un proyecto o dice "explorar antes de probar". Úselo también para comandos simples de "prueba [URL]" sin más contexto. No lo utilice cuando ya existan artefactos de funciones o cuando el usuario haga referencia a pruebas o errores conocidos específicos.
