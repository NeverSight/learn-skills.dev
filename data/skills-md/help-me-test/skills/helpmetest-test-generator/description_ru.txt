Используйте этот навык, когда пользователю нужны тесты, написанные для определенной функции или потока. Срабатывает при: «написать тесты для X», «создать тесты для оформления заказа», «создать тесты для входа в систему», «добавить тесты для регистрации», «у нас есть сценарии — теперь напишите тесты» или любой запрос на создание автоматизированного тестового покрытия для известной функции. Также срабатывает, когда обнаружение завершено и пользователь готов перейти от документирования сценариев к их фактическому тестированию. Не для: изучения сайта, чтобы узнать, что тестировать, оценки качества существующего теста или отладки неудачного теста.
