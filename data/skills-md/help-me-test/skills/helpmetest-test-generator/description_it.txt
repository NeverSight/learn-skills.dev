Utilizza questa competenza quando l'utente desidera che i test siano scritti per una funzionalità o un flusso specifico. Si attiva su: "scrivi test per X", "genera test per il checkout", "crea test per l'accesso", "aggiungi test per la registrazione", "abbiamo scenari: ora scrivi i test" o qualsiasi richiesta di produrre una copertura di test automatizzata per una funzionalità nota. Si attiva anche quando l'individuazione viene completata e l'utente è pronto a passare dalla documentazione degli scenari al test effettivo. Non per: esplorare un sito per scoprire cosa testare, giudicare se un test esistente è buono o eseguire il debug di un test fallito.
