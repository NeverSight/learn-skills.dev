Utilisez cette compétence lorsque l'utilisateur souhaite écrire des tests pour une fonctionnalité ou un flux spécifique. Déclencheurs sur : "écrire des tests pour X", "générer des tests pour le paiement", "créer des tests pour la connexion", "ajouter des tests pour l'enregistrement", "nous avons des scénarios - écrivez maintenant les tests", ou toute demande visant à produire une couverture de test automatisée pour une fonctionnalité connue. Se déclenche également lorsque la découverte est terminée et que l'utilisateur est prêt à passer de la documentation des scénarios à leur test. Pas pour : explorer un site pour découvrir ce qu'il faut tester, juger si un test existant est bon ou déboguer un test qui a échoué.
