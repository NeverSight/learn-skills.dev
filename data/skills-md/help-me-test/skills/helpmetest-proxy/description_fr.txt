Configurez les tunnels proxy HelpMeTest pour les tests de développement local. À utiliser lorsque l'utilisateur doit tester localhost, souhaite remplacer les URL de production par des ports locaux ou doit acheminer plusieurs services. À utiliser lorsque l'utilisateur dit « configurer le proxy », « tester localhost », « tunnel vers local » ou avant d'exécuter des tests sur des serveurs de développement locaux.
