Richten Sie HelpMeTest-Proxy-Tunnel für lokale Entwicklungstests ein. Verwenden Sie diese Option, wenn der Benutzer localhost testen muss, Produktions-URLs durch lokale Ports ersetzen möchte oder mehrere Dienste weiterleiten muss. Verwenden Sie diese Option, wenn der Benutzer sagt: „Proxy einrichten“, „Localhost testen“, „Tunnel zu Local“ oder bevor Tests auf lokalen Entwicklungsservern ausgeführt werden.
