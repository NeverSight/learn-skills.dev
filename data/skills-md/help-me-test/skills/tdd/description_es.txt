Ejecutor de desarrollo basado en pruebas utilizando HelpMeTest. Úselo cuando el usuario desee implementar una nueva función, refactorizar código o corregir errores. Fuerza el flujo de trabajo de prueba primero: escriba TODAS las pruebas integrales de un extremo a otro antes de CUALQUIER implementación, use las pruebas como barreras de seguridad durante el desarrollo y asegúrese de que todas las pruebas se aprueben antes de afirmar que están terminadas. Úselo cuando el usuario diga "implementar X", "agregar función Y", "refactorizar Z", "corregir error" o comience a codificar sin pruebas.
