Outil de développement piloté par les tests utilisant HelpMeTest. À utiliser lorsque l'utilisateur souhaite implémenter une nouvelle fonctionnalité, refactoriser le code ou corriger des bogues. Force le flux de travail axé sur les tests : écrivez TOUS les tests complets de bout en bout avant TOUTE implémentation, utilisez les tests comme garde-fous pendant le développement, assurez-vous que tous les tests réussissent avant de prétendre qu'ils sont terminés. À utiliser lorsque l'utilisateur dit « implémenter X », « ajouter une fonctionnalité Y », « refactoriser Z », « corriger un bug » ou commence à coder sans tests.
