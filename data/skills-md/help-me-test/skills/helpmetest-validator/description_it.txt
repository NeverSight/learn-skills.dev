Richiama questa abilità quando un utente condivide il codice del test e si chiede se funziona effettivamente come previsto, non per eseguire o correggere il test, ma per valutare se il test ha un valore reale. Si attiva: "questo test è buono?", "rileverebbe un vero bug?", "questo test passa sempre - è normale?", "rivedi questi test prima di impegnarmi" o "questo test verifica qualcosa di significativo?". Si attiva anche quando qualcuno sospetta che un test sia inutile, desidera un controllo di qualità pre-commit o non è sicuro che valga la pena conservare un test generato automaticamente. La domanda principale a cui risponde questa abilità è: "Questo test fallirebbe se la funzionalità si rompesse?" In caso contrario, il test viene rifiutato. NON utilizzare per generare nuovi test, correggere test falliti o esplorare le funzionalità dell'applicazione.
