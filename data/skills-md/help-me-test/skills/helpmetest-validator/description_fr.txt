Invoquez cette compétence lorsqu'un utilisateur partage le code de test et se demande s'il fonctionne réellement comme prévu – non pas pour exécuter ou corriger le test, mais pour évaluer si le test a une valeur réelle. Se déclenche sur : "Ce test est-il bon ?", "Est-ce que cela détecterait un vrai bug ?", "Ce test réussit toujours - est-ce normal ?", "Examinez ces tests avant de m'engager" ou "Ce test vérifie-t-il quelque chose de significatif ?". Se déclenche également lorsque quelqu'un soupçonne qu'un test est inutile, souhaite une porte de qualité avant la validation ou ne sait pas si un test généré automatiquement vaut la peine d'être conservé. La question principale à laquelle répond cette compétence : « Ce test échouerait-il si la fonctionnalité tombait en panne ? » Dans le cas contraire, le test est rejeté. Ne PAS utiliser pour générer de nouveaux tests, corriger des tests défaillants ou explorer les fonctionnalités de l'application.
