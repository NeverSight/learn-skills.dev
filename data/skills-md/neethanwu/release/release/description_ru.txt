Управляет рабочими процессами выпуска пакетов для экосистем npm, Python и Go. Определяет тип проекта по файлам манифеста, обновляет версии, преобразует журналы изменений (сохранять формат журнала изменений), создает git-коммиты и теги и, при необходимости, генерирует рабочие процессы публикации CI. Используйте при выпуске пакетов, вырезании версий, публикации в реестрах, обновлении версий или настройке инфраструктуры выпуска.
