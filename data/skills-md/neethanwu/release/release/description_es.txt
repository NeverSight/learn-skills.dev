Gestiona los flujos de trabajo de lanzamiento de paquetes para los ecosistemas npm, Python y Go. Detecta el tipo de proyecto a partir de archivos de manifiesto, modifica versiones, transforma registros de cambios (mantener un formato de registro de cambios), crea confirmaciones y etiquetas de git y, opcionalmente, genera flujos de trabajo de publicación de CI. Úselo al publicar paquetes, cortar versiones, publicar en registros, actualizar versiones o configurar la infraestructura de lanzamiento.
