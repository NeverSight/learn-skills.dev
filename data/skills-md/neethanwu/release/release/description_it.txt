Gestisce i flussi di lavoro di rilascio dei pacchetti per gli ecosistemi npm, Python e Go. Rileva il tipo di progetto dai file manifest, esegue il bump delle versioni, trasforma i log delle modifiche (formato Keep a Changelog), crea commit e tag git e, facoltativamente, genera flussi di lavoro di pubblicazione CI. Da utilizzare durante il rilascio di pacchetti, il taglio di versioni, la pubblicazione su registri, il bumping di versioni o l'impostazione dell'infrastruttura di rilascio.
