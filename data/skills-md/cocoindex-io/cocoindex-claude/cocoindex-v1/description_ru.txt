Этот навык следует использовать при построении конвейеров обработки данных с помощью CocoIndex v1, библиотеки Python для поэтапного преобразования данных. Используйте, когда задача включает обработку файлов/данных в базы данных, создание векторных внедрений, построение графиков знаний, рабочие процессы ETL или любой конвейер данных, требующий автоматического обнаружения изменений и дополнительных обновлений. CocoIndex v1 является родным для Python (поддерживает любые типы Python), не имеет DSL и в настоящее время находится на стадии предварительной версии (версия 1.0.0a1 или новее).
