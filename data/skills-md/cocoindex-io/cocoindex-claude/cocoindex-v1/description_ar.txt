يجب استخدام هذه المهارة عند إنشاء خطوط معالجة البيانات باستخدام CocoIndex v1، وهي مكتبة Python لتحويل البيانات المتزايدة. يُستخدم عندما تتضمن المهمة معالجة الملفات/البيانات في قواعد البيانات، أو إنشاء عمليات تضمين متجهة، أو إنشاء رسوم بيانية للمعرفة، أو سير عمل ETL، أو أي خط أنابيب بيانات يتطلب الكشف التلقائي عن التغيير والتحديثات المتزايدة. CocoIndex v1 هو إصدار Python الأصلي (يدعم أي أنواع Python)، ولا يحتوي على DSL، وهو حاليًا قيد الإصدار المسبق (الإصدار 1.0.0a1 أو أحدث).
