Esta habilidad debe usarse al crear canales de procesamiento de datos con CocoIndex v1, una biblioteca de Python para la transformación de datos incremental. Úselo cuando la tarea implique procesar archivos/datos en bases de datos, crear incrustaciones de vectores, crear gráficos de conocimiento, flujos de trabajo ETL o cualquier canal de datos que requiera detección automática de cambios y actualizaciones incrementales. CocoIndex v1 es nativo de Python (admite cualquier tipo de Python), no tiene DSL y actualmente se encuentra en versión preliminar (versión 1.0.0a1 o posterior).
