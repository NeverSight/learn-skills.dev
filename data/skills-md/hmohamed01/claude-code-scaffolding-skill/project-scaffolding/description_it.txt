Procedura guidata per impalcature di progetto di livello IDE per la creazione di nuovi progetti con configurazione completa. Supporta oltre 70 tipi di progetti: siti Web HTML/CSS, React, Next.js, Vue, Astro, Remix, React Native, Flutter, Expo, FastAPI, Django, Express, NestJS, Go/Gin, Rust/Axum, Spring Boot, Hono, Elysia, Chrome Extensions, VS Code Extensions, app desktop Tauri, funzioni serverless e altro ancora. Fornisce la creazione di progetti a livello di WebStorm/PyCharm con selezione interattiva dell'SDK, configurazione del framework, impostazione del database e strumenti DevOps. Utilizzare quando: creazione di un nuovo progetto, configurazione di un'applicazione framework, inizializzazione di una codebase, scaffolding boilerplate, creazione di estensioni, creazione di app mobili/desktop/web, configurazione di monorepos o creazione di siti Web/pagine di destinazione statici.
