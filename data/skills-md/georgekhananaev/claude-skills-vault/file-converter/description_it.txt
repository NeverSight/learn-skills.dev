Converti e trasforma file: immagini (ridimensionamento, formato, HEIC), markdown (PDF/HTML), dati (CSV/JSON/YAML/TOML/XML), SVG, base64, codifica testo. Modalità multipiattaforma, singola e batch. Questa capacità dovrebbe essere utilizzata durante la conversione di formati di file, il ridimensionamento di immagini, la generazione di PDF dal markdown o la trasformazione di dati tra formati.
