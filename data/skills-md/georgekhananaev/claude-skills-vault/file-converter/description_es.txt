Convierta y transforme archivos: imágenes (cambiar tamaño, formato, HEIC), rebajas (PDF/HTML), datos (CSV/JSON/YAML/TOML/XML), SVG, base64, codificación de texto. Modo multiplataforma, único y por lotes. Esta habilidad debe usarse al convertir formatos de archivos, cambiar el tamaño de imágenes, generar archivos PDF a partir de rebajas o transformar datos entre formatos.
