Bootstrap Python MCP proyectos de servidor y espacios de trabajo en macOS usando uv y FastMCP con valores predeterminados consistentes. Úselo al crear un nuevo servidor MCP desde cero, crear un único proyecto uv MCP, crear un espacio de trabajo uv con miembros de paquete/servicio, personalizar los valores predeterminados del scaffold a través de perfiles YAML en capas, inicializar los valores predeterminados de pytest+ruff+mypy, crear README.md, inicializar git, ejecutar comprobaciones de validación iniciales o comenzar desde OpenAPI/FastAPI con guía de mapeo MCP.
