Amorcez les projets de serveur et les espaces de travail Python MCP sur macOS à l'aide de uv et FastMCP avec des valeurs par défaut cohérentes. À utiliser lors de la création d'un nouveau serveur MCP à partir de zéro, de l'échafaudage d'un seul projet uv MCP, de l'échafaudage d'un espace de travail uv avec des membres de package/service, de la personnalisation des valeurs par défaut de l'échafaudage via des profils YAML en couches, de l'initialisation des valeurs par défaut de pytest+ruff+mypy, de la création de README.md, de l'initialisation de git, de l'exécution de contrôles de validation initiaux ou du démarrage à partir d'OpenAPI/FastAPI avec les conseils de mappage MCP.
