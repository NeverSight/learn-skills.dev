Avvia progetti server e aree di lavoro Python MCP su macOS utilizzando uv e FastMCP con impostazioni predefinite coerenti. Da utilizzare durante la creazione di un nuovo server MCP da zero, l'impalcatura di un singolo progetto uv MCP, l'impalcatura di uno spazio di lavoro uv con membri del pacchetto/servizio, la personalizzazione delle impostazioni predefinite dello scaffold tramite profili YAML a più livelli, l'inizializzazione delle impostazioni predefinite pytest+ruff+mypy, la creazione di README.md, l'inizializzazione di git, l'esecuzione di controlli di convalida iniziali o l'avvio da OpenAPI/FastAPI con guida alla mappatura MCP.
