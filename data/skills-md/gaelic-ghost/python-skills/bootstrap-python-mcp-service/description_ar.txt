مشاريع خادم Bootstrap Python MCP ومساحات العمل على macOS باستخدام uv وFastMCP مع إعدادات افتراضية متسقة. يُستخدم عند إنشاء خادم MCP جديد من البداية، أو دعم مشروع uv MCP واحد، أو دعم مساحة عمل uv مع أعضاء الحزمة/الخدمة، أو تخصيص إعدادات السقالة الافتراضية من خلال ملفات تعريف YAML ذات الطبقات، أو تهيئة إعدادات pytest+ruff+mypy الافتراضية، أو إنشاء README.md، أو تهيئة git، أو تشغيل فحوصات التحقق الأولية، أو البدء من OpenAPI/FastAPI مع توجيهات تعيين MCP.
