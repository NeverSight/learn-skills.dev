Richten Sie Komponententests für Python-UV-Projekte und UV-Arbeitsbereiche mit Pytest ein und führen Sie sie aus. Verwenden Sie diese Option, wenn Sie die Pytest-Konfiguration in pyproject.toml erstellen oder aktualisieren, Pytest-Entwicklungsabhängigkeiten mit uv installieren, Tests in einem Workspace-Mitgliedspaket über „uv run --package“ ausführen, Pytest-Workflow-Standardeinstellungen über mehrschichtige YAML-Profile anpassen, Tests mit Fixtures/Markern/parametrisieren organisieren oder Testerkennungs- und Importfehler beheben.
