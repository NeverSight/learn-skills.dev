قم بإعداد وتشغيل اختبارات الوحدة لمشاريع Python uv ومساحات عمل uv باستخدام pytest. يُستخدم عند إنشاء أو تحديث تكوين pytest في pyproject.toml، أو تثبيت تبعيات pytest dev باستخدام uv، أو تشغيل الاختبارات في حزمة أعضاء مساحة العمل عبر `uv run --package`، أو تخصيص الإعدادات الافتراضية لسير عمل pytest من خلال ملفات تعريف YAML ذات الطبقات، أو تنظيم الاختبارات باستخدام التركيبات/العلامات/المعلمات، أو استكشاف أخطاء اكتشاف الاختبار وفشل الاستيراد وإصلاحها.
