Configure y ejecute pruebas unitarias para proyectos uv de Python y espacios de trabajo uv con pytest. Úselo al crear o actualizar la configuración de pytest en pyproject.toml, instalar dependencias de desarrollo de pytest con uv, ejecutar pruebas en un paquete miembro del espacio de trabajo a través de `uv run --package`, personalizar los valores predeterminados del flujo de trabajo de pytest a través de perfiles YAML en capas, organizar pruebas con accesorios/marcadores/parametrizar o solucionar problemas de descubrimiento de pruebas e importación.
