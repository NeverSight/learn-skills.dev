pytest を使用して、Python UV プロジェクトと UV ワ​​ークスペースの単体テストをセットアップして実行します。 pyproject.toml で pytest 構成を作成または更新する場合、uv を使用して pytest dev の依存関係をインストールする場合、「uv run --package」を使用してワークスペース メンバー パッケージでテストを実行する場合、階層化された YAML プロファイルを使用して pytest ワークフローのデフォルトをカスタマイズする場合、フィクスチャ/マーカー/パラメーター化を使用してテストを整理する場合、またはテストの検出とインポートの失敗のトラブルシューティングを行う場合に使用します。
