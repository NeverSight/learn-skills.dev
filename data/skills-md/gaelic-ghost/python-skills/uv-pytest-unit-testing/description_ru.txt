Настраивайте и запускайте модульные тесты для UV-проектов Python и рабочих пространств UV с помощью pytest. Используйте при создании или обновлении конфигурации pytest в pyproject.toml, установке зависимостей pytest dev с помощью uv, запуске тестов в пакете члена рабочей области с помощью `uv run --package`, настройке параметров рабочего процесса pytest по умолчанию с помощью многоуровневых профилей YAML, организации тестов с помощью фиксаторов/маркеров/параметризации или устранения неполадок при обнаружении тестов и сбоях импорта.
