Configura ed esegui unit test per progetti uv Python e aree di lavoro uv con pytest. Da utilizzare durante la creazione o l'aggiornamento della configurazione pytest in pyproject.toml, l'installazione delle dipendenze di sviluppo pytest con uv, l'esecuzione di test in un pacchetto membro dell'area di lavoro tramite `uv run --package`, la personalizzazione delle impostazioni predefinite del flusso di lavoro pytest tramite profili YAML a più livelli, l'organizzazione di test con dispositivi/marcatori/parametrizzazione o la risoluzione dei problemi relativi al rilevamento dei test e agli errori di importazione.
