Загрузите службы Python FastAPI в macOS с использованием uv и согласованных шаблонов проекта и рабочей области. Используйте при создании новой внутренней службы/службы API с нуля, создании шаблонов для одного проекта службы uv, создании рабочей области uv с элементами пакета/службы, настройке параметров по умолчанию для шаблонов с помощью многоуровневых профилей YAML, инициализации значений по умолчанию pytest+ruff+mypy, создания README.md, инициализации git и выполнения команд начальной проверки.
