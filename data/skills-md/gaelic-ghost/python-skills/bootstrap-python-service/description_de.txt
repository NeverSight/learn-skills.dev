Bootstrap-Python-FastAPI-Dienste unter macOS mithilfe von UV mit konsistenten Projekt- und Arbeitsbereichsgerüsten. Wird verwendet, wenn Sie einen neuen Backend-/API-Dienst von Grund auf erstellen, ein Gerüst für ein einzelnes UV-Dienstprojekt erstellen, einen UV-Arbeitsbereich mit Paket-/Dienstmitgliedern aufbauen, Gerüststandards durch mehrschichtige YAML-Profile anpassen, pytest+ruff+mypy-Standards initialisieren, README.md erstellen, Git initialisieren und anfängliche Validierungsbefehle ausführen.
