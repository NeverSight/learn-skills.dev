Avvia i servizi Python FastAPI su macOS utilizzando uv con scaffold di progetto e area di lavoro coerenti. Da utilizzare durante la creazione di un nuovo servizio backend/API da zero, lo scaffolding di un singolo progetto di servizio uv, lo scaffolding di uno spazio di lavoro uv con membri del pacchetto/servizio, la personalizzazione delle impostazioni predefinite dello scaffold tramite profili YAML a più livelli, l'inizializzazione delle impostazioni predefinite pytest+ruff+mypy, la creazione di README.md, l'inizializzazione di git e l'esecuzione dei comandi di convalida iniziali.
