خدمات Bootstrap Python FastAPI على نظام التشغيل macOS باستخدام الأشعة فوق البنفسجية مع سقالات متسقة للمشروع ومساحة العمل. يُستخدم عند إنشاء خدمة خلفية/واجهة برمجة تطبيقات جديدة من البداية، ودعم مشروع خدمة uv واحد، ودعم مساحة عمل uv مع أعضاء الحزمة/الخدمة، وتخصيص الإعدادات الافتراضية للسقالة من خلال ملفات تعريف YAML ذات الطبقات، وتهيئة إعدادات pytest+ruff+mypy الافتراضية، وإنشاء README.md، وتهيئة git، وتشغيل أوامر التحقق الأولية.
