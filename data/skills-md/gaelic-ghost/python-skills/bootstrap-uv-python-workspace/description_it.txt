Avvia nuovi progetti Python e aree di lavoro multipacchetto con uv su macOS utilizzando script deterministici e impostazioni predefinite coerenti. Da utilizzare durante la creazione di un nuovo progetto Python uv, l'impalcatura di un monorepo/workspace uv, l'impostazione di profili di pacchetti o servizi, la personalizzazione delle impostazioni predefinite dello scaffold tramite profili YAML a più livelli, l'inizializzazione degli strumenti di sviluppo (pytest, ruff, mypy), la creazione di scaffold README o l'inizializzazione di git con un primo commit opzionale.
