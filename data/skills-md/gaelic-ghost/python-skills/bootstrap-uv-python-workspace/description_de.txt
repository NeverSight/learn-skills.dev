Bootstrapieren Sie neue Python-Projekte und Arbeitsbereiche mit mehreren Paketen mit uv unter macOS unter Verwendung deterministischer Skripte und konsistenter Standardeinstellungen. Verwenden Sie diese Option, wenn Sie ein neues UV-Python-Projekt erstellen, ein UV-Monorepo/einen UV-Arbeitsbereich aufbauen, Paket- oder Serviceprofile einrichten, Gerüststandards durch mehrschichtige YAML-Profile anpassen, Entwicklungstools (Pytest, Ruff, Mypy) initialisieren, README-Gerüste erstellen oder Git mit einem optionalen ersten Commit initialisieren.
