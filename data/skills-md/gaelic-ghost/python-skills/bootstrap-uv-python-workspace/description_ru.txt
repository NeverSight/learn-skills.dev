Загружайте новые проекты Python и рабочие пространства с несколькими пакетами с помощью uv на macOS, используя детерминированные сценарии и согласованные настройки по умолчанию. Используйте при создании нового проекта uv Python, создании шаблонов uv монорепо/рабочего пространства, настройке профилей пакетов или служб, настройке параметров шаблона по умолчанию с помощью многоуровневых профилей YAML, инициализации инструментов разработки (pytest, ruff, mypy), создании шаблонов README или инициализации git с необязательной первой фиксацией.
