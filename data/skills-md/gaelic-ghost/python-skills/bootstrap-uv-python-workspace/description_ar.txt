قم بتشغيل مشاريع Python الجديدة ومساحات العمل متعددة الحزم باستخدام الأشعة فوق البنفسجية على نظام التشغيل macOS باستخدام نصوص برمجية حتمية وافتراضيات متسقة. يُستخدم عند إنشاء مشروع uv Python جديد، أو سقالات uv monorepo/مساحة عمل، أو إعداد ملفات تعريف الحزمة أو الخدمة، أو تخصيص الإعدادات الافتراضية للسقالة من خلال ملفات تعريف YAML ذات الطبقات، أو تهيئة أدوات التطوير (pytest، أو ruff، أو mypy)، أو إنشاء سقالات README، أو تهيئة git بالالتزام الأول الاختياري.
