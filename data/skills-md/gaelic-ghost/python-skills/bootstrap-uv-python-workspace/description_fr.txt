Démarrez de nouveaux projets Python et des espaces de travail multi-packages avec uv sur macOS à l'aide de scripts déterministes et de valeurs par défaut cohérentes. À utiliser lors de la création d'un nouveau projet uv Python, de l'échafaudage d'un monorepo/espace de travail uv, de la configuration de profils de package ou de service, de la personnalisation des paramètres par défaut de l'échafaudage via des profils YAML en couches, de l'initialisation des outils de développement (pytest, ruff, mypy), de la création d'échafaudages README ou de l'initialisation de git avec un premier commit facultatif.
