Arranque nuevos proyectos de Python y espacios de trabajo de paquetes múltiples con uv en macOS utilizando scripts deterministas y valores predeterminados consistentes. Úselo al crear un nuevo proyecto uv Python, crear un scaffolding de un monorepo/espacio de trabajo uv, configurar perfiles de paquetes o servicios, personalizar los valores predeterminados del scaffold a través de perfiles YAML en capas, inicializar herramientas de desarrollo (pytest, ruff, mypy), crear scaffolds README o inicializar git con una primera confirmación opcional.
