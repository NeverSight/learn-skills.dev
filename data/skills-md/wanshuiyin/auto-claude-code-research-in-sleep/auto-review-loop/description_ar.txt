حلقة مراجعة بحثية مستقلة ومتعددة الجولات. قم بالمراجعة بشكل متكرر عبر Codex MCP، وقم بتنفيذ الإصلاحات، وإعادة المراجعة حتى يتم الوصول إلى التقييم الإيجابي أو الوصول إلى الحد الأقصى للجولات. يُستخدم عندما يقول المستخدم "حلقة المراجعة التلقائية"، أو "المراجعة حتى تمر"، أو يريد تحسينًا تكراريًا مستقلاً.
