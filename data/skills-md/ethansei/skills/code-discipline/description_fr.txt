Qualité générale du code et discipline d’ingénierie. Utilisez-le sur n’importe quelle tâche de code pour appliquer des modifications minimales, propres et de niveau production. Suivez ces règles lors de l’écriture, de la modification ou de la révision du code. S'active sur : coder, implémenter, corriger, construire, refactoriser, fonctionnalité, bogue, changer, modifier, ajouter, créer, développer, écrire, éditer, améliorer, optimiser, mettre à jour, supprimer, supprimer, renommer, déplacer, extraire, intégrer, migrer, convertir, remplacer, réécrire.
