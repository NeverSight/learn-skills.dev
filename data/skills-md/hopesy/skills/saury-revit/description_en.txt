Create a Revit plug-in project based on Saury.Revit.Template. Triggered when the user requests to create a Revit plug-in, start a new Revit project, initialize a Revit development environment, or use the saury-revit template. Covers scenarios: (1) creating a new Revit plug-in project, (2) installing/updating the saury-revit dotnet template, (3) adding new features (Command/View/ViewModel/Service) to an existing project. Keywords: Revit, plug-in, plugin, saury-revit, dotnet new.
