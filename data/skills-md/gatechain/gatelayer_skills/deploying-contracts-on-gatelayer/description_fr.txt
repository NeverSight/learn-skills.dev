Déploie des contrats intelligents sur GateLayer à l'aide de Foundry. Couvre les commandes de création de forge, la vérification des contrats et la configuration des clés API GateScan. À utiliser lors du déploiement de contrats Solidity sur GateLayer Mainnet ou Testnet. Couvre des expressions telles que « déployer le contrat sur GateLayer », « créer une forge sur GateLayer », « vérifier le contrat sur GateScan », « obtenir testnet GT », « robinet GateLayer », « comment déployer sur GateLayer » ou « publier mon contrat ».
