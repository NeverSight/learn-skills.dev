Produkt-Denktool, das dabei hilft, vage Ideen vor und während der Entwicklung in klare, strukturierte Designentscheidungen umzuwandeln. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer /shape aufruft oder darum bittet, eine Produktidee „durchzudenken“, zu „planen“, was gebaut werden soll, „Anforderungen zu definieren“, eine Funktion „zu spezifizieren“, eine Idee „zu formen“ oder Hilfe bei der Entscheidung benötigt, was erstellt werden soll, bevor Code geschrieben wird. Wird auch ausgelöst, wenn Benutzer Dinge sagen wie „Ich möchte X erstellen, habe aber die Details noch nicht herausgefunden“, „Hilf mir, das durchzudenken“, „Was sollte ich vor dem Erstellen beachten“ oder „Lassen Sie mich das planen“. Bei dieser Fähigkeit geht es um das Denken auf Produktebene (was gebaut werden soll und warum), nicht um die Implementierungsplanung (wie man es baut) – Letzteres wird im Planmodus erledigt.
