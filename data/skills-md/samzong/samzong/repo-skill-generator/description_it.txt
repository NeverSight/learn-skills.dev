Genera capacità di codifica e revisione locale del repository dalla cronologia git di un repository e dall'attività di revisione PR di GitHub. Da utilizzare quando viene richiesto di creare, aggiornare o aggiornare le competenze di codifica/revisione specifiche del repository, apprendere dai PR, analizzare la cronologia dei commit, estrarre modelli di codifica o generare linee guida di codifica per il repository corrente o un modulo specifico.
