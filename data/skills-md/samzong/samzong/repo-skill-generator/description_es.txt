Genere habilidades de revisión y codificación local de repositorio a partir del historial de git de un repositorio y la actividad de revisión de relaciones públicas de GitHub. Úselo cuando se le solicite crear, actualizar o actualizar habilidades de codificación/revisión específicas del repositorio, aprender de los RP, analizar el historial de confirmaciones, extraer patrones de codificación o generar pautas de codificación para el repositorio actual o un módulo específico.
