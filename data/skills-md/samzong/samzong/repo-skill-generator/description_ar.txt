أنشئ مهارات الترميز والمراجعة المحلية من خلال سجل git الخاص بالمستودع ونشاط مراجعة GitHub PR. استخدمه عندما يُطلب منك إنشاء أو تحديث أو تحديث مهارات الترميز/المراجعة الخاصة بالمستودع، أو التعلم من العلاقات العامة، أو تحليل سجل الالتزام، أو استخراج أنماط الترميز، أو إنشاء إرشادات الترميز للريبو الحالي أو وحدة معينة.
