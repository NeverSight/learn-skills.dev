Analice los problemas del repositorio de GitHub verificando el estado de los asignados, las relaciones públicas vinculadas y las señales de etiquetas, luego filtre y clasifique los problemas contribuyentes por prioridad (P0-P5). Genera informes JSON legibles por máquina y Markdown legibles por humanos. Úselo cuando busque problemas en los que se pueda colaborar, encuentre oportunidades de contribución, busque el primer problema bueno, encuentre problemas que requieran ayuda o descubra tareas fáciles de usar para principiantes.
