Analizza i problemi del repository GitHub controllando lo stato dell'assegnatario, i PR collegati e i segnali delle etichette, quindi filtra e classifica i problemi che possono contribuire in base alla priorità (P0-P5). Genera report JSON leggibili dalla macchina e Markdown leggibili dall'uomo. Da utilizzare quando si cercano problemi a cui è possibile contribuire, si trovano opportunità di contributo, si cerca il primo problema valido, si trovano problemi che richiedono aiuto o si scoprono attività adatte ai principianti.
