قم بتحليل مشكلات مستودع GitHub عن طريق التحقق من حالة المُعيَّن، والمستفيدين الرئيسيين المرتبطين، وإشارات التسمية، ثم قم بتصفية المشكلات القابلة للمساهمة وتصنيفها حسب الأولوية (P0-P5). يُخرج تقارير JSON التي يمكن قراءتها آليًا وتقارير Markdown التي يمكن قراءتها بواسطة الإنسان. يُستخدم عند البحث عن المشكلات القابلة للمساهمة، أو العثور على فرص المساهمة، أو البحث عن العدد الأول الجيد، أو البحث عن المشكلات المطلوبة للمساعدة، أو اكتشاف المهام المناسبة للمبتدئين.
