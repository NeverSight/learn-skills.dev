Используйте API эластичного развертывания частного облака AutoDL для пакетного планирования и управления контейнерами графического процессора. Поддерживает создание развертываний (ReplicaSet/Job/Container), запрос состояния контейнера, настройку количества реплик, остановку/удаление развертываний и другие операции. Используется, когда пользователи упоминают AutoDL, эластичное развертывание, планирование контейнера графического процессора, пакетное создание контейнера или развертывание частного облака.
