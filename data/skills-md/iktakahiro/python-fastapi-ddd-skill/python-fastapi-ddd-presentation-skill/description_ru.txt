Управляет уровнем представления FastAPI в приложении Python DDD + Onion Architecture (структура обработчика маршрутов, схемы запросов/ответов Pydantic, сопоставление исключений домена с ошибками HTTP и документация по ошибкам OpenAPI) на основе справочника dddpy. Используйте при добавлении/рефакторинге конечных точек, которые вызывают UseCases и преобразуют примитивы ↔ Value Objects/Entities.
