Leitet die FastAPI-Präsentationsschicht in einer Python-DDD + Onion-Architektur-App (Route-Handler-Struktur, Pydantic-Anforderungs-/Antwortschemata, Zuordnung von Domänenausnahmen zu HTTP-Fehlern und OpenAPI-Fehlerdokumentation) basierend auf der dddpy-Referenz. Wird beim Hinzufügen/Umgestalten von Endpunkten verwendet, die UseCases aufrufen und Grundelemente ↔ Wertobjekte/-entitäten konvertieren.
