Guide les tests unitaires pour les applications Python DDD + Onion Architecture (entités de domaine/objets de valeur et cas d'utilisation) à l'aide de pytest et de simulations de référentiel, basées sur la référence dddpy. À utiliser lors de l'ajout de tests, du choix des éléments à simuler ou de la structuration de dossiers de tests pour un projet DDD FastAPI.
