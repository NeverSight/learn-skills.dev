Guide la conception backend FastAPI à l’aide de Domain-Driven Design (DDD) et d’Onion Architecture en Python. À utiliser lors de la structuration d'une application FastAPI (routes/gestionnaires, schémas Pydantic, DI basée sur les dépendances), la modélisation d'entités/objets de valeur de domaine, la définition d'interfaces de référentiel, l'implémentation d'adaptateurs d'infrastructure SQLAlchemy ou l'écriture de cas d'utilisation, basés sur la référence dddpy.
