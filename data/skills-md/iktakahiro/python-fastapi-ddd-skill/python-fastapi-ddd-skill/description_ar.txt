يرشد تصميم الواجهة الخلفية لـ FastAPI باستخدام التصميم المستند إلى المجال (DDD) والبنية البصلية في Python. يُستخدم عند إنشاء تطبيق FastAPI (المسارات/المعالجات، مخططات Pydantic، DI القائم على التبعيات)، أو نمذجة كيانات المجال/كائنات القيمة، أو تحديد واجهات المستودع، أو تنفيذ محولات البنية الأساسية لـ SQLAlchemy، أو كتابة حالات الاستخدام، استنادًا إلى مرجع dddpy.
