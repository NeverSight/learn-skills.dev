Guida la progettazione del backend FastAPI utilizzando Domain-Driven Design (DDD) e Onion Architecture in Python. Da utilizzare durante la strutturazione di un'app FastAPI (percorsi/gestori, schemi Pydantic, DI basati su Depends), la modellazione di entità di dominio/oggetti valore, la definizione di interfacce repository, l'implementazione di adattatori dell'infrastruttura SQLAlchemy o la scrittura di casi d'uso, in base al riferimento dddpy.
