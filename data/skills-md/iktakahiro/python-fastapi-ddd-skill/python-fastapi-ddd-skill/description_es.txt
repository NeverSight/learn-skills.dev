Guía el diseño de backend de FastAPI utilizando el diseño controlado por dominio (DDD) y la arquitectura Onion en Python. Úselo al estructurar una aplicación FastAPI (rutas/controladores, esquemas Pydantic, DI basada en dependencias), modelar entidades de dominio/objetos de valor, definir interfaces de repositorio, implementar adaptadores de infraestructura SQLAlchemy o escribir casos de uso, según la referencia dddpy.
