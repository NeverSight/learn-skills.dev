Руководствуется проектированием серверной части FastAPI с использованием доменно-ориентированного проектирования (DDD) и луковой архитектуры на Python. Используйте при структурировании приложения FastAPI (маршруты/обработчики, схемы Pydantic, DI на основе зависимостей), моделировании объектов предметной области/объектов значений, определении интерфейсов репозитория, реализации адаптеров инфраструктуры SQLAlchemy или написании вариантов использования на основе ссылки dddpy.
