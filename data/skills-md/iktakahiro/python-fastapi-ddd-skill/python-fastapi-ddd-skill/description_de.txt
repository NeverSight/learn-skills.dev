Leitet das FastAPI-Backend-Design mit Domain-Driven Design (DDD) und Onion-Architektur in Python. Wird zum Strukturieren einer FastAPI-App (Routen/Handler, Pydantic-Schemas, Depends-basiertes DI), zum Modellieren von Domänenentitäten/Wertobjekten, zum Definieren von Repository-Schnittstellen, zum Implementieren von SQLAlchemy-Infrastrukturadaptern oder zum Schreiben von Anwendungsfällen basierend auf der dddpy-Referenz verwendet.
