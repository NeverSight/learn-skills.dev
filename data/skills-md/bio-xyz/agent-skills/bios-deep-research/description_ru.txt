Выполните глубокие исследовательские запросы по BIOS. Поддерживает аутентификацию по ключу API (api.ai.bio.xyz) и криптографические платежи x402 с использованием USDC на базе (x402.ai.bio.xyz). Используйте при отправке заданий на исследование BIOS, проведении переговоров по оплате, опросе статуса разговора, отправке последующих действий или записи отзывов ERC-8004.
