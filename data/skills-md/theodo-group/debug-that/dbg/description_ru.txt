Отлаживайте приложения с помощью отладчика CLI dbg. Поддерживает Node.js (V8/CDP), Bun (WebKit/JSC) и собственный код через LLDB (DAP). Используйте, когда: (1) исследование ошибок во время выполнения путем пошагового выполнения кода, (2) проверка значений переменных в определенных точках выполнения, (3) установка точек останова и условных точек останова, (4) оценка выражений в приостановленном контексте, (5) горячее исправление кода без перезапуска (JS/TS), (6) отладка сбоев теста путем подключения к работающему процессу, (7) отладка C/C++/Rust/Swift с помощью LLDB, (8) любая задача, где для понимания поведения во время выполнения требуется отладчик. Триггеры: «отладить это», «установить точку останова», «перейти», «проверить переменные», «почему это неправильное значение», «отследить выполнение», «подключить отладчик», «ошибка выполнения», «segfault», «дамп ядра».
