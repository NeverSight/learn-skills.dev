Eseguire il debug delle applicazioni utilizzando il debugger CLI dbg. Supporta Node.js (V8/CDP), Bun (WebKit/JSC) e codice nativo tramite LLDB (DAP). Utilizzare quando: (1) investigare i bug di runtime esaminando il codice, (2) ispezionare i valori delle variabili in punti di esecuzione specifici, (3) impostare punti di interruzione e punti di interruzione condizionali, (4) valutare espressioni in un contesto in pausa, (5) applicare patch a caldo al codice senza riavviare (JS/TS), (6) eseguire il debug degli errori di test collegandosi a un processo in esecuzione, (7) eseguire il debug di C/C++/Rust/Swift con LLDB, (8) qualsiasi attività dove la comprensione del comportamento in fase di esecuzione richiede un debugger.
