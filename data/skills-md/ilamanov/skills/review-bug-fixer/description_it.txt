Correggere i risultati validi della revisione del codice da uno o più file di markdown della revisione. Da utilizzare quando l'utente dispone di file di revisione (ad esempio, review1.md, review2.md) contenenti risultati, problemi o suggerimenti della revisione del codice e desidera correggere quelli validi nel ramo corrente. Si attiva su richieste come "correzione dei risultati della revisione", "correzione dei problemi dalla revisione", "applicazione delle correzioni della revisione del codice" o qualsiasi menzione della correzione di bug/problemi dai file di markdown della revisione.
