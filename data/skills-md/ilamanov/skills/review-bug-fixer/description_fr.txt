Corrigez les résultats de révision de code valides à partir d’un ou plusieurs fichiers de démarques de révision. À utiliser lorsque l'utilisateur dispose de fichiers de révision (par exemple, review1.md, review2.md) contenant des résultats, des problèmes ou des suggestions de révision de code et qu'il souhaite corriger ceux valides dans la branche actuelle. Se déclenche sur des requêtes telles que « corriger les résultats de la révision », « réparer les problèmes liés à la révision », « appliquer les correctifs de révision du code » ou toute mention de correction de bogues/problèmes à partir des fichiers de démarque de révision.
