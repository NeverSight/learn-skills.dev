Rhino Health Python SDK を使用してフェデレーション分析ワークフローを計画し、実行します。ユーザーが生存分析、メトリクス、データの調和、モデルのトレーニング、または複数ステップの SDK ワークフローを実行する場合に使用します。大まかな研究目標を取り上げ、実行可能なコードを含む段階的な実行計画を作成します。 SDK の質問、rhino_health エラーのデバッグ、メトリクスの選択も処理します。トリガー対象: rhino-health、rhino_health、RhinoSession、FCP、フェデレーション分析、OMOP、FHIR、ハーモナイゼーション、または 40 以上のフェデレーション メトリクスのいずれか。
