Этот навык следует использовать, когда пользователь сталкивается с ошибкой или обратной трассировкой из SDK Rhino Health Python, нуждается в отладке кода rhino_health, видит NotAuthenticatedError, ValidationError, TypeError, ImportError или любое исключение, упоминающее rhino_health, или спрашивает, почему его код SDK не работает.
