Cette compétence doit être utilisée lorsque l'utilisateur décrit un objectif de recherche ou d'analyse de haut niveau à l'aide du SDK Python de Rhino Health, souhaite planifier un flux de travail en plusieurs étapes, dit « planifier », « concevoir un flux de travail », « comment dois-je aborder », « quelles sont les étapes à suivre », « architecter », « mettre en place un pipeline », souhaite combiner l'analyse avec des objets de code ou une harmonisation, ou a besoin d'aide pour décomposer une tâche informatique fédérée complexe en opérations SDK ordonnées.
