「gws」 CLI を介して Google Workspace を管理します。ドライブ、Gmail、カレンダー、スプレッドシート、ドキュメント、チャット、管理者、タスク、Meet、スライド、フォーム、連絡先、その他すべての Workspace API を管理します。次の場合に使用します: (1) Google ドライブ上のファイルの一覧表示、アップロード、ダウンロード、または共有、(2) Gmail メッセージの読み取り、送信、ラベル付け、またはフィルタリング、(3) Google カレンダーのイベントの作成、更新、またはクエリ、(4) Google スプレッドシート データの読み取りまたは書き込み、(5) Google ドキュメントの作成または編集、(6) Google Chat メッセージの送信、(7) Google Tasks の管理、(8) その他の Google Workspace 操作。すべての Workspace API を動的に検出する公式の「gws」 CLI をラップします。エージェントパイプラインに適した構造化された JSON を出力します。
