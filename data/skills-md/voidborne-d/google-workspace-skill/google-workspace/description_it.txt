Gestisci Google Workspace tramite l'interfaccia a riga di comando `gws`: Drive, Gmail, Calendario, Fogli, Documenti, Chat, Amministrazione, Attività, Meet, Presentazioni, Moduli, Contatti e ogni altra API di Workspace. Da utilizzare per: (1) elencare, caricare, scaricare o condividere file su Google Drive, (2) leggere, inviare, etichettare o filtrare messaggi Gmail, (3) creare, aggiornare o interrogare eventi di Google Calendar, (4) leggere o scrivere dati di Fogli Google, (5) creare o modificare documenti Google, (6) inviare messaggi di Google Chat, (7) gestire Google Task, (8) qualsiasi altra operazione di Google Workspace. Comprende la CLI ufficiale "gws" che rileva dinamicamente tutte le API di Workspace. Genera JSON strutturato adatto per pipeline di agenti.
