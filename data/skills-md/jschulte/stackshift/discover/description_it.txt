Scoperta dell'ecosistema da un unico repository iniziale. Esegue la scansione dei segnali di integrazione (pacchetti npm, docker compose, env vars, chiamate API, trigger CI/CD, configurazioni dell'area di lavoro, code di messaggi, riferimenti all'infrastruttura), cerca GitHub per repository correlati, esegue la scansione del file system locale, quindi presenta una mappa dell'ecosistema con punteggio di confidenza e un grafico delle dipendenze della sirena. Non toccare i repository confermati su /stackshift.batch o /stackshift.reimagine.
