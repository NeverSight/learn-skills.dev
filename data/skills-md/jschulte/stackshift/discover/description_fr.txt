Découverte de l'écosystème à partir d'un seul dépôt de départ. Analyse les signaux d'intégration (packages npm, docker compose, variables d'environnement, appels d'API, déclencheurs CI/CD, configurations d'espace de travail, files d'attente de messages, références d'infrastructure), recherche sur GitHub les dépôts associés, analyse le système de fichiers local, puis présente une carte de l'écosystème avec un score de confiance et un graphique de dépendance Mermaid. Ne confiez pas les dépôts confirmés à /stackshift.batch ou /stackshift.reimagine.
