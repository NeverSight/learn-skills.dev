Преобразует 11 документов реверс-инжиниринга из Gear 2 в спецификации GitHub Spec Kit с тремя уровнями тщательности (только спецификации, спецификации+планы, спецификации+планы+задачи). Создает каталог .specify/ с конституцией, спецификациями функций, планами реализации и разбивкой задач. Шестерня 3 из 6 в конвейере реверс-инжиниринга. Запускается по: «создать спецификации из документов реверс-инжиниринга», «создать комплект спецификаций из документации», «запустить механизм 3», «преобразовать документы в комплект спецификаций».
