Создайте агентов искусственного интеллекта для платформы Codebolt, используя @codebolt/agent. При создании агента сначала решите, какой тип вам нужен — (1) оболочка CodeboltAgent для простых агентов на основе подсказок без настраиваемого цикла, (2) базовые компоненты для настройки агентного цикла или (3) Core API для полного расширенного управления. Также рассматриваются агенты Remix (без кода), ActionBlocks, инструменты, рабочие процессы и процессоры. Разрешения обрабатываются приложением, а не агентом.
