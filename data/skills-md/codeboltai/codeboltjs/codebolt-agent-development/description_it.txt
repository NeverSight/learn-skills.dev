Crea agenti AI per la piattaforma Codebolt utilizzando @codebolt/agent. Quando crei un agente, decidi innanzitutto il tipo di cui hai bisogno: (1) wrapper CodeboltAgent per agenti semplici basati su prompt senza loop personalizzato, (2) componenti di base per personalizzare il loop agenti o (3) API core per un controllo avanzato completo. Copre anche Remix Agents (senza codice), ActionBlock, strumenti, flussi di lavoro e processori. Le autorizzazioni vengono gestite dall'applicazione, non dall'agente.
