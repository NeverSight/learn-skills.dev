Cree agentes de IA para la plataforma Codebolt usando @codebolt/agent. Al crear un agente, primero decida qué tipo necesita: (1) contenedor CodeboltAgent para agentes simples basados ​​en mensajes sin bucle personalizado, (2) componentes base para personalizar el bucle agente o (3) API principal para un control avanzado total. También cubre Remix Agents (sin código), ActionBlocks, herramientas, flujos de trabajo y procesadores. Los permisos los maneja la aplicación, no el agente.
