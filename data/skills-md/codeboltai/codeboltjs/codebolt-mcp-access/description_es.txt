Habilidad básica para que los agentes de Codebolt interactúen con la API Codebolt MCP (Model Context Protocol). Úselo cuando los agentes necesiten ejecutar herramientas para operaciones de archivos, git, automatización del navegador, terminal, memoria, orquestación, planificación, pruebas, colaboración o cualquier funcionalidad de la plataforma Codebolt. Activa solicitudes que involucran codebolt.tools.executeTool(), operaciones del sistema de archivos, comandos git, control del navegador, orquestación de agentes, administración de tareas o cualquier operación del espacio de nombres MCP.
