المهارة الأساسية لوكلاء Codebolt للتفاعل مع واجهة برمجة تطبيقات Codebolt MCP (بروتوكول سياق النموذج). يُستخدم عندما يحتاج الوكلاء إلى تنفيذ أدوات لعمليات الملفات، أو git، أو أتمتة المتصفح، أو المحطة الطرفية، أو الذاكرة، أو التنسيق، أو التخطيط، أو الاختبار، أو التعاون، أو أي وظيفة من وظائف منصة Codebolt. تشغيل الطلبات التي تتضمن codebolt.tools.executeTool()، أو عمليات نظام الملفات، أو أوامر git، أو التحكم في المستعرض، أو تنسيق الوكيل، أو إدارة المهام، أو أي عمليات مساحة اسم MCP.
