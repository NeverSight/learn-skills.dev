Основные навыки агентов Codebolt по взаимодействию с API Codebolt MCP (Model Context Protocol). Используйте, когда агентам необходимо запускать инструменты для файловых операций, git, автоматизации браузера, терминала, памяти, оркестрации, планирования, тестирования, совместной работы или любых функций платформы Codebolt. Запускается по запросам, включающим codebolt.tools.executeTool(), операции с файловой системой, команды git, управление браузером, оркестровку агентов, управление задачами или любые операции с пространством имен MCP.
