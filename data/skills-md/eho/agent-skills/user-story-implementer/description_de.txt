Implementieren Sie eine einzelne User Story oder Aufgabe aus einem GitHub Issue Backlog. Führt eine einzelne Ralph-Loop-Iteration aus, indem das nächste offene Problem abgerufen, zugewiesen, der Code implementiert, eine Verzweigung und PR erstellt und fortgefahren wird. Sie MÜSSEN diese Fähigkeit verwenden, wenn Sie aufgefordert werden, „eine User Story zu implementieren“, „eine Iteration auszuführen“, „die nächste Aufgabe auszuführen“, „eine Ralph-Loop-Iteration auszuführen“ oder „eine Aufgabe aus dem Backlog abzuschließen“.
