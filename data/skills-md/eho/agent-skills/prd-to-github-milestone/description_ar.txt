تحليل مستند متطلبات المنتج (PRD) لاستخراج قصص المستخدمين وإنشاء مشكلات GitHub المقابلة. ويمكن ربطها بشكل اختياري بـ GitHub Milestone. تعمل هذه المهارة كمرحلة إعداد لتتبع مشكلات GitHub الأصلية. تأكد من استخدام هذه المهارة عندما يطلب المستخدم "إرسال PRD إلى GitHub"، أو "إنشاء مشكلات من PRD"، أو "إعداد الحدث الرئيسي"، أو يذكر تحويل المتطلبات إلى مشكلات GitHub قابلة للتنفيذ.
