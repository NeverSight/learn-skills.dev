Используйте, когда пользователь явно запрашивает «инициализировать проект», «создать файл инструкций проекта» или «создать AGENTS.md». Полностью автоматизированный: автоматически определяет язык ОС по умолчанию, анализирует структуру каталогов проекта (поддерживает проекты Python/Web/Rust/Go/Java/data-science/documentation и т. д.), определяет тип и цель проекта и генерирует стандартизированный документ с инструкциями по проекту в один клик. Создаваемые файлы включают: AGENTS.md (кросс-платформенные универсальные инструкции проекта, единый источник истины), CLAUDE.md (адаптация кода Claude, ссылка на которую осуществляется через @./AGENTS.md), README.md (введение и использование проекта), CHANGELOG.md (журнал изменений проекта), .gitignore (Git игнорирует правила, безопасность превыше всего).
