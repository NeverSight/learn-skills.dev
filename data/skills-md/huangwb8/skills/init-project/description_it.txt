Da utilizzare quando l'utente richiede esplicitamente "inizializza progetto", "crea file di istruzioni del progetto" o "genera AGENTS.md". Completamente automatizzato: rileva automaticamente la lingua predefinita del sistema operativo, analizza la struttura della directory del progetto (supporta progetti Python/Web/Rust/Go/Java/data-science/documentazione, ecc.), deduce il tipo e lo scopo del progetto e genera un documento di istruzioni di progetto standardizzato con un clic. I file generati includono: AGENTS.md (istruzioni di progetto universali multipiattaforma, Single Source of Truth), CLAUDE.md (adattamento specifico del codice Claude, a cui si fa riferimento tramite @./AGENTS.md), README.md (introduzione e utilizzo del progetto), CHANGELOG.md (registro delle modifiche del progetto), .gitignore (Git ignora le regole, la sicurezza innanzitutto).
