Úselo cuando el usuario solicite explícitamente "iniciar proyecto", "crear archivo de instrucciones del proyecto" o "generar AGENTS.md". Totalmente automatizado: detecta automáticamente el idioma predeterminado del sistema operativo, analiza la estructura del directorio del proyecto (admite proyectos Python/Web/Rust/Go/Java/data-science/documentation, etc.), infiere el tipo y el propósito del proyecto y genera un documento de instrucciones del proyecto estandarizado con un solo clic. Los archivos generados incluyen: AGENTS.md (instrucciones de proyecto universales multiplataforma, fuente única de la verdad), CLAUDE.md (adaptación específica del Código Claude, referenciada a través de @./AGENTS.md), README.md (introducción y uso del proyecto), CHANGELOG.md (registro de cambios del proyecto), .gitignore (Git ignora las reglas, la seguridad primero).
