Используйте, когда пользователю необходимо провести систематический обзор/обзор литературы/связанную работу: ИИ автоматически генерирует условия поиска, поиск из нескольких источников → дедупликация → ИИ читает и оценивает каждую статью (1–10 по семантической релевантности и группировке по подтемам) → пропорционально выбирает статьи с высокими оценками → автоматически генерирует бюджет слов для «обзора» (70 % цитируется + 30 % нецитируется, усреднено по трем выборкам) → старший эксперт в предметной области пишет свободно (фиксировано) аннотация/введение/подтемы/обсуждение/прогноз/заключение), с жесткой проверкой окончательного количества слов и ссылок, принудительным экспортом в PDF и Word. Поддерживает многоязычный перевод и интеллектуальную компиляцию (en/zh/ja/de/fr/es).
