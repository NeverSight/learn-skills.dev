Когда пользователь явно запрашивает «написать/доработать реферат предложения NSFC», «создать реферат на китайском и английском языках» или «перевести реферат с китайского на английский».

Выведите как китайскую, так и английскую версии: английский должен быть точным переводом китайского (без дополнительной информации, без упущения ключевых ограничений).
По умолчанию аннотация на китайском языке составляет не более 400 символов (включая знаки пунктуации), аннотация на английском языке — не более 4000 символов (включая знаки препинания); окончательные ограничения следуют за `skills/nsfc-abstract/config.yaml:limits`.

Метод вывода: запишите результаты в файл `NSFC-ABSTRACTS.md` в **рабочем каталоге** (имя файла следует за `config.yaml:output.filename`), с одним заголовком `#` для китайского языка и одним заголовком `#` для английского языка.

⚠️ Не применимо в следующих случаях:
- Пользователь хочет перевести только общий текст, не связанный с предложением (перевести напрямую)
- Пользователь хочет написать только обоснование/содержание исследования/текст фундамента исследования (используйте соответствующий навык серии nsfc)
