Meilleures pratiques de Software Mansion pour la production d'applications React Native sur la nouvelle architecture. À utiliser lors de l'écriture, de la révision ou du débogage du code React Native dans les animations, les gestes, le SVG, l'haptique, l'IA sur l'appareil, le multimédia, le texte enrichi, le multithreading ou l'audio. Déclenchez-vous sur n'importe quelle question d'implémentation de React Native ou lorsqu'une sous-compétence plus spécifique ci-dessous correspond à la tâche.
