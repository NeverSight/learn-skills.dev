Surveillez les pages Web pour détecter les modifications de contenu et recevez des alertes. Suivez les URL, détectez les mises à jour, affichez les différences. À utiliser lorsqu'on vous demande de consulter un site Web, de suivre les modifications sur une page, de surveiller les nouveaux messages/contenus, de configurer des alertes de changement de page ou de vérifier si un site a été mis à jour. Prend en charge les sélecteurs CSS pour une surveillance ciblée.
