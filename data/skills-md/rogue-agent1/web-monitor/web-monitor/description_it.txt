Monitora le pagine Web per le modifiche ai contenuti e ricevi avvisi. Tieni traccia degli URL, rileva gli aggiornamenti, visualizza le differenze. Utilizzare quando viene richiesto di guardare un sito Web, tenere traccia delle modifiche su una pagina, monitorare nuovi post/contenuti, impostare avvisi di modifica della pagina o verificare se un sito è stato aggiornato. Supporta i selettori CSS per il monitoraggio mirato.
