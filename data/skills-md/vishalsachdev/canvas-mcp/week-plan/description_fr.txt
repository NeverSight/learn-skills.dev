Planificateur de devoirs hebdomadaire pour étudiants. Affiche toutes les dates d'échéance, le statut de soumission, les notes et les évaluations par les pairs pour tous les cours. À utiliser lorsque l'étudiant dit « ce qui est dû », « planifier ma semaine », « contrôle hebdomadaire » ou souhaite organiser ses cours.
