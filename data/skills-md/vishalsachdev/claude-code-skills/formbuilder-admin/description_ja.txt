イリノイ大学 ATLAS FormBuilder Admin アプリケーション (appserv7.admin.uillinois.edu/FormBuilderAdmin) でフォームを作成、編集、管理します。ユーザーが FormBuilder フォームを操作する必要がある場合に使用します。これには、フォームの質問とセクションの作成または編集、イベント セッション カテゴリとイベント セッションの構成、支払品目と CFOAPAL コードの設定、条件付きステートメントによるワークフロー フェーズとルーティング トリガーの管理、ローカル ユーザーの認証と自己登録の構成、フォーム期間の編集、待機リストと登録制限の管理、条件付き表示のための質問トリガーの設定、テキスト差し込みフィールドの操作、フォームの応答とレポートの処理、フォーム動作のトラブルシューティングなどがあります。キーワード: FormBuilder、ATLAS、UIUC、フォーム ビルダー、イベント登録、アンケート、フォームの質問、支払い構成、ワークフロー、ルーティング トリガー、フェーズ、イベント セッション、iPay、CFOAPAL。
