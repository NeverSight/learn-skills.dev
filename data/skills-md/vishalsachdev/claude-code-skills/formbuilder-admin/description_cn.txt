在伊利诺伊大学 ATLAS FormBuilder 管理应用程序 (appserv7.admin.uillinois.edu/FormBuilderAdmin) 中创建、编辑和管理表单。当用户需要使用 FormBuilder 表单时使用，包括：创建或编辑表单问题和部分、配置事件会话类别和事件会话、设置付款行项目和 CFOAPAL 代码、使用条件语句管理工作流程阶段和路由触发器、为本地用户配置身份验证和自我注册、编辑表单周期、管理候补名单和注册限制、设置条件可见性的问题触发器、使用文本合并字段、处理表单响应和报告或对表单行为进行故障排除。关键词：FormBuilder、ATLAS、UIUC、表单生成器、事件注册、调查、表单问题、支付配置、工作流程、路由触发器、阶段、事件会话、iPay、CFOAPAL。
