Cree, edite y administre formularios en la aplicación ATLAS FormBuilder Admin de la Universidad de Illinois (appserv7.admin.uillinois.edu/FormBuilderAdmin). Úselo cuando el usuario necesite trabajar con formularios de FormBuilder, lo que incluye: crear o editar preguntas y secciones de formularios, configurar categorías de sesiones de eventos y sesiones de eventos, configurar líneas de pago y códigos CFOAPAL, administrar fases de flujo de trabajo y enrutar activadores con declaraciones condicionales, configurar autenticación y autorregistro para usuarios locales, editar períodos de formularios, administrar listas de espera y límites de registro, configurar activadores de preguntas para visibilidad condicional, trabajar con campos de combinación de texto, manejar respuestas e informes de formularios o solucionar problemas de comportamiento de formularios. Palabras clave: FormBuilder, ATLAS, UIUC, creador de formularios, registro de eventos, encuesta, preguntas de formulario, configuración de pagos, flujo de trabajo, activadores de enrutamiento, fases, sesiones de eventos, iPay, CFOAPAL.
