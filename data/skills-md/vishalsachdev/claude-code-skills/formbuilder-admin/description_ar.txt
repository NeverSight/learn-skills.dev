قم بإنشاء النماذج وتحريرها وإدارتها في تطبيق ATLAS FormBuilder Admin بجامعة إلينوي (appserv7.admin.uillinois.edu/FormBuilderAdmin). يُستخدم عندما يحتاج المستخدم إلى العمل مع نماذج FormBuilder بما في ذلك: إنشاء أسئلة وأقسام النموذج أو تحريرها، وتكوين فئات جلسات الأحداث وجلسات الأحداث، وإعداد عناصر سطر الدفع وأكواد CFOAPAL، وإدارة مراحل سير العمل وتوجيه المشغلات باستخدام البيانات الشرطية، وتكوين المصادقة والتسجيل الذاتي للمستخدمين المحليين، وتحرير فترات النموذج، وإدارة قوائم الانتظار وحدود التسجيل، وإعداد مشغلات الأسئلة للرؤية المشروطة، والعمل مع حقول دمج النص، أو التعامل مع استجابات النماذج والتقارير، أو استكشاف الأخطاء وإصلاحها سلوك النموذج. الكلمات الرئيسية: FormBuilder، ATLAS، UIUC، منشئ النماذج، تسجيل الأحداث، المسح، أسئلة النموذج، تكوين الدفع، سير العمل، مشغلات التوجيه، المراحل، جلسات الأحداث، iPay، CFOAPAL.
