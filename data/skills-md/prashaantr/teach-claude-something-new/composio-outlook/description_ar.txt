البريد الإلكتروني في Outlook/Microsoft 365 عبر Composio API. استخدم عندما:
(1) إرسال رسائل البريد الإلكتروني عبر Outlook/Microsoft 365
(2) إدراج رسائل البريد الوارد في Outlook أو البحث عنها أو قراءتها
(3) الرد على رسائل البريد الإلكتروني في Outlook
(4) إنشاء المسودات أو إدارة المجلدات
استخدم Composio HTTP API فقط.
