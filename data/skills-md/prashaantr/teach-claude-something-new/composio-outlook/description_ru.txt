Электронная почта Outlook/Microsoft 365 через Composio API. Используйте, когда:
(1) Отправка электронной почты через Outlook/Microsoft 365.
(2) Просмотр, поиск или чтение входящих сообщений Outlook
(3) Ответ на электронные письма Outlook
(4) Создание черновиков или управление папками
Используйте только HTTP API Composio.
