E-mail Outlook/Microsoft 365 tramite API Composio. Utilizzare quando:
(1) Invio di e-mail tramite Outlook/Microsoft 365
(2) Elencare, cercare o leggere i messaggi della posta in arrivo di Outlook
(3) Rispondere alle e-mail di Outlook
(4) Creazione di bozze o gestione di cartelle
Utilizza solo l'API HTTP Composio.
