Correo electrónico Outlook/Microsoft 365 a través de la API de Composio. Úselo cuando:
(1) Envío de correos electrónicos a través de Outlook/Microsoft 365
(2) Listar, buscar o leer mensajes de la bandeja de entrada de Outlook
(3) Responder a correos electrónicos de Outlook
(4) Crear borradores o administrar carpetas
Utilice únicamente la API HTTP de Composio.
