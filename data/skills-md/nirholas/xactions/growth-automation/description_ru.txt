Автоматизирует рост X/Twitter с помощью скриптов консоли браузера. Автоматически лайкает твиты с помощью фильтров по ключевым словам/пользователям, автоматически комментирует целевых пользователей, следит за ними с помощью поиска по ключевым словам или взаимодействия, следит за аудиторией целевых учетных записей с помощью богатой фильтрации, обучает алгоритм для ниш и запускает комбинированные пакеты роста. Во всех случаях сначала необходимо вставить core.js. Используйте при автоматизации роста, вовлеченности, подписки или создания аудитории в Твиттере.
