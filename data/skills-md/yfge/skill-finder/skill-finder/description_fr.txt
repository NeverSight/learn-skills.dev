Recherchez, découvrez et installez les compétences d'agent OpenClaw à partir de plusieurs registres. À utiliser lorsque : l'utilisateur demande "trouver une compétence pour X", "existe-t-il une compétence qui peut...", "comment puis-je faire X" (où X pourrait avoir une compétence existante), souhaite parcourir les compétences disponibles ou demande d'installer/ajouter une compétence. Se déclenche également sur « compétences de recherche », « explorer les compétences », « pour quelles compétences existent » ou « étendre les capacités ».
