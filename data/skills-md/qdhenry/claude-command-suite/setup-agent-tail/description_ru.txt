Настройте агрегирование журналов агента для текущего проекта. Автоматически определяет инфраструктуру (Vite, Next.js, обычный Node, monorepo) и настраивает средство запуска CLI, плагины журналов браузера и места назначения вывода. Используйте при настройке хвоста агента, настройке ведения журналов сервера разработки или конвейерных журналов для использования агента AI.
