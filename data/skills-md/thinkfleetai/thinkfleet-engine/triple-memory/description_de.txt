Komplettes Speichersystem, das den automatischen Rückruf von LanceDB, den strukturierten Git-Notes-Speicher und die dateibasierte Arbeitsbereichssuche kombiniert. Verwenden Sie diese Option, wenn Sie einen umfassenden Agentenspeicher einrichten, wenn Sie dauerhaften Kontext über Sitzungen hinweg benötigen oder wenn Sie Entscheidungen/Einstellungen/Aufgaben mit mehreren zusammenarbeitenden Speicher-Backends verwalten.
