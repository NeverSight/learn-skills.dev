Orchestrieren Sie komplexe Arbeiten über eine phasengesteuerte Multi-Agenten-Schleife (Prüfung → Entwurf → Implementierung → Überprüfung → Validierung → Bereitstellung). Verwenden Sie es, wenn Sie Arbeit in Subsysteme aufteilen, unabhängige Audits durchführen, Ergebnisse in einer Liste bestätigter Probleme abgleichen, Korrekturen in Clustern delegieren, PASS/FAIL-Überprüfungs-Gates durchsetzen und eine durchgängig validierte Bereitstellung vorantreiben müssen. Nicht für kleine Aufgaben mit nur einer Datei verwenden.
