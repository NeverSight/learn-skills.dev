Scanner de vulnérabilités automatisé pour les plateformes d'agents. Effectue une analyse des dépendances (audit npm, pip-audit), une recherche CVE multi-bases de données (OSV, NVD, GitHub Advisory), une analyse SAST (Semgrep, Bandit) et des tests d'exécution de hook DAST spécifiques à l'agent pour les hooks OpenClaw.
