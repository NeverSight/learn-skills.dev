Автоматический сканер уязвимостей для агентских платформ. Выполняет сканирование зависимостей (аудит npm, pip-audit), поиск CVE в нескольких базах данных (OSV, NVD, GitHub Advisory), анализ SAST (Semgrep, Bandit) и тестирование выполнения перехватчиков DAST для конкретного агента для перехватчиков OpenClaw.
