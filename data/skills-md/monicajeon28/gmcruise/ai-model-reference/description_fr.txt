Noms d’appel de l’API du modèle AI et guide de référence des prix. Fournit le nom exact du modèle (chaîne de modèle) et les tarifs à jour lors de l'appel de modèles IA avec une clé API. À utiliser lorsque : (1) vous avez besoin du nom de modèle correct pour les appels vers OpenAI, Anthropic, Google, DeepSeek, etc. ; (2) vous souhaitez comparer les coûts/prix des jetons ; (3) vous devez choisir le modèle de raisonnement le plus récent, RAPIDE ou rentable ; (4) vous souhaitez optimiser les coûts d'une mise en cache rapide ou d'un traitement par lots.
