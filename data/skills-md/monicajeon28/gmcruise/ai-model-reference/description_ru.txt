Справочное руководство по названию и ценам API модели AI. При вызове модели AI с помощью ключа API предоставляется точное имя модели (строка модели) и последняя информация о ценах. Используйте его, когда: (1) вам нужно точное имя модели для вызовов API к OpenAI, Anthropic, Google, DeepSeek и т. д.; (2) вам необходимо сравнить стоимость/цену токенов; (3) вам нужно выбрать новейшую модель рассуждения, модель FAST или экономичную модель; (4) вам необходимо оптимизировать затраты на оперативное кэширование или пакетную обработку.
