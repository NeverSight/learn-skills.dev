AI Model API اسم الاتصال والدليل المرجعي للسعر. يوفر أسماء نماذج دقيقة (سلاسل نماذج) وأسعارًا محدثة عند استدعاء نماذج الذكاء الاصطناعي باستخدام مفتاح واجهة برمجة التطبيقات. حالات الاستخدام: (1) عندما تكون أسماء النماذج مطلوبة لاستدعاءات واجهة برمجة التطبيقات (API) إلى OpenAI وAnthropic وGoogle وDeepSeek وما إلى ذلك؛ (2) عند مقارنة تكاليف/أسعار الرمز المميز؛ (3) عند اختيار أحدث النماذج المنطقية أو السريعة أو ذات القيمة الأفضل؛ (4) عند تحسين تكاليف التخزين المؤقت السريع أو معالجة الدُفعات
