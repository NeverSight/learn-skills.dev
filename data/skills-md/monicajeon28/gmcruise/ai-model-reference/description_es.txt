Guía de referencia de precios y nombres de llamadas de API del modelo AI. Proporciona el nombre exacto del modelo (cadena de modelo) y la información de precios más reciente al llamar a modelos de IA con una clave API. Casos de uso: (1) Cuando se necesita un nombre de modelo para llamadas API a OpenAI, Anthropic, Google, DeepSeek, etc., (2) Cuando se requiere una comparación de costo/precio del token, (3) Cuando se selecciona el último modelo de razonamiento/modelo FAST/modelo rentable, (4) Cuando se necesita una rápida optimización de costos de almacenamiento en caché/procesamiento por lotes
